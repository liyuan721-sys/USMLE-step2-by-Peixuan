# 📋 Daily Summary 工作流 — Prompt 速查卡

> [!info] 用法
> 把这张卡放在 vault 根目录 / 工作流.md 里。
> 每次需要触发对应操作时，**复制对应 prompt 粘贴到聊天框**即可。

---

## 🎯 工作流总览

```
学科 X 聊天框 1（前半）
  └─ 结束前 → 用 Prompt A（场次速记）→ 复制保存

学科 X 聊天框 2（后半）
  └─ 开始时 → 用 Prompt B（导入前文）+ 粘贴 part 1 速记
  └─ 学习中
  └─ 结束前 → 用 Prompt C（学科日总结）→ 保存到 vault

第二天早上 → 按学科日总结的 "30 分钟复盘清单" 执行

每周末 → 用 Prompt D（Claude Code）整合本周所有文件
```

---

## 📝 Prompt A：场次速记（每个聊天框结束用）

```
出本场速记，格式：

# [学科] 速记 — 2026-MM-DD part [X]
## 做了什么（Q#### 列表）
## 核心知识点（一行一条，不超过 5 条）
## 暴露的卡点（一行一句，不超过 3 条）
## 产出文件清单

不用 emoji 标记，用 callout 区分严重度。
```

**用途**：
- 同学科多聊天框时，给下一个聊天框做上下文
- 保留每场的快照，方便溯源

**保存位置**：复制到剪贴板 / 临时文件，待粘贴到下一场

---

## 📝 Prompt B：导入前文（同学科聊天框 2 开头用）

```
我今天在另一个聊天框已经做了一些[学科]学习，下面是那场速记。
请记住这些内容，今天后续学习要在这个基础上累积：

[粘贴 Prompt A 输出的速记]

接下来继续做题，保持累积。
```

**用途**：让聊天框 2 的 Claude 有完整学习上下文

---

## 📝 Prompt C：学科日总结（学科最后一个聊天框结束时用）

```
基于本场聊天 + 我之前贴的 part 1 速记，
出今天的 [学科] 日总结，格式：

# [学科] 日总结 — 2026-MM-DD

## 📍 今日完成（跨所有聊天框）
表格：| 来源 | 内容 | 状态 |

## 💡 核心知识点（一行一条，合并所有 part）
按主题分组

## ⚠️ 暴露的卡点（合并所有 part）
每个卡点：
- 错因（一句话）
- 修正（一句话）
- 状态（✅ 已建工具 / ⚠️ 需二次复习 / ❌ 未解决）

## 🎯 明早 30 分钟复盘清单
按 10 分钟 × 3 part 设计：
- Part 1：默写口诀 / 公式
- Part 2：默写关键对比表
- Part 3：应用题验证

## 📁 产出文件清单
所有聊天框产出的文件路径

## 🌱 待生成专题笔记
本学科积累中的素材进度（X/Y）

文件名：daily-summary/2026-MM-DD_[学科].md
存到 /mnt/user-data/outputs/
```

**用途**：生成第二天复盘用的核心文件

---

## 📝 Prompt D：周整合（Claude Code 用，每周末）

```bash
请读取 daily-summary/ 下本周所有文件（2026-MM-DD 到 2026-MM-DD），
执行以下任务：

1. 跨学科分析本周重复出现的卡点（=真薄弱）
2. 检查"待生成专题笔记"素材是否足够（≥3 道）
3. 如足够，生成对应的 完整笔记/专题笔记/[专题名].md
4. 更新 薄弱点.md（标记本周新发现的薄弱点）
5. 更新 复习入口_Dashboard.md（本周学习数据）
6. 输出一份本周总结报告到 daily-summary/2026-MM-DD_周报.md

注意：
- 不要重复创建已存在的专题笔记
- 关联部分按双链路径 v2 规则
- 标签按 v2 规则（只加学科分类）
```

**用途**：周末批量整合，自动驱动专题笔记生成

## 🎯 实际使用例子（今天的场景）

### 场景 1：Biostats 一天用了 2 个聊天框
```
早上 Biostats 聊天 1：做了 Q1, Q2
  → 结束前用 Prompt A → 生成速记 A1
  → 保存到记事本

下午 Biostats 聊天 2（本对话）：
  → 开头用 Prompt B + 粘贴速记 A1
  → 做 Q20435, Q4121
  → 结束前用 Prompt C → 生成 2026-05-12_Biostats.md
```

### 场景 2：Neuro 也用了 2 个聊天框
```
晚上 Neuro 聊天 1 + 2
  → 同样的 Prompt A / B / C 操作
  → 生成 2026-05-12_Neuro.md
```

### 第二天早上
```
打开 daily-summary/2026-05-12_Biostats.md → 30 分钟复盘
打开 daily-summary/2026-05-12_Neuro.md → 30 分钟复盘
（或合并为 60 分钟跨学科复盘）
```

---

## ⚠️ 注意事项

1. **Prompt A 必须保存**：不保存就没法贴给下一个聊天框
2. **学科日总结只在最后一场出**：避免重复生成
3. **文件名严格按格式**：YYYY-MM-DD_学科.md，方便 Claude Code 周末批量读取
4. **同学科多聊天框时**：part 编号必须连续（part 1, part 2, part 3...）
5. **跨学科不混用**：Biostats 和 Neuro 各自独立总结，不要合并

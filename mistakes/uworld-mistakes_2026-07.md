> [!example]- [2026-07-03] 皮肤 / 慢性瘢痕溃疡恶变 Marjolin ulcer → SCC (Q3456)
> ^Q3456
> 
> ## Stem 模式
> 50 岁男左小腿慢性不愈溃疡 2 月，10 年前该处**全层烧伤 + 植皮致瘢痕**；溃疡起于旧瘢痕、红色肉芽易出血、始终不愈。
> 
> ## 核心概念
> **Marjolin ulcer 马乔林溃疡** = 在**慢性伤口 / 瘢痕**（烧伤瘢痕、静脉淤积溃疡等）基础上恶变的 **Squamous cell carcinoma 鳞状细胞癌 (SCC)**。
> 
> ## 普适规则
> - **慢性不愈溃疡 / 伤口 = SCC 直到证伪 → 皮肤活检 (skin biopsy)**。
> - 潜伏期平均约 30 年，但可早（甚至伤后 1 年内）。
> - 形态：**增大、不愈、边缘卷起 / 外翻 (rolled/everted margins)**、肉芽过度易出血。
> - 侵袭性强，**复发 / 局部侵犯 / 转移风险高**；治疗宽切缘切除 / Mohs / 必要时截肢。
> 
> ## 题干线索→临床意义
> - "10 年前烧伤瘢痕上新发不愈溃疡" → Marjolin ulcer / 瘢痕恶变 SCC 的经典钩子。
> - "红色肉芽、触之易出血" → SCC（低分化牛肉红易出血）。
> - "已 2 月不愈" → 慢性不愈 = 恶性信号，非普通伤口。
> 
> ## 易混陷阱（普适）
> - **Cutaneous T-cell lymphoma 皮肤 T 细胞淋巴瘤（mycosis fungoides 蕈样肉芽肿）**：瘙痒**湿疹样斑片 (eczematous patches)**，不起于旧瘢痕。
> - **Basal cell carcinoma 基底细胞癌**：珍珠样丘疹 + 毛细血管扩张；慢性伤口少数也可发生，但远不如 SCC。
> - **Venous stasis ulcer 静脉淤积溃疡**：内踝区 + **下肢水肿 / 淤积征**（本例双下肢无水肿）。
> - **Delayed graft failure 植皮延迟失败**：植皮后**第一周**内坏死，非数年后。
> 
> ## 我为什么错
> - 选了：B（Cutaneous T-cell lymphoma 皮肤 T 细胞淋巴瘤）
> - 错因：知识（同主题第 2 次错，见 Q4319 下唇 SCC）
> - 核心陷阱：没抓"慢性瘢痕上不愈溃疡 = 恶变 SCC"这条 buzzword，被淋巴瘤干扰项带走。
> 
> ## Memory Hook
> **「烧伤瘢痕 / 慢性溃疡上冒出不愈溃疡 = Marjolin ulcer = SCC，先活检。」**

> [!example]- [2026-07-03] 皮肤 / 花斑糠疹 Tinea versicolor (Q16963)
> ^Q16963
> 
> ## Stem 模式
> 21 岁男数月来上背 / 肩 / 胸融合斑，**夏季加重**；职业为**游泳池救生员**，问最佳治疗。
> 
> ## 核心概念
> **Tinea versicolor 花斑糠疹 / 花斑癣（pityriasis versicolor）** = **Malassezia 马拉色菌**（M. globosa / furfur）过度增殖，产色素异常斑 + 细鳞屑。
> 
> ## 普适规则
> - 好发**上躯干 / 上肢**；**热湿环境（夏季、出汗、游泳池）促发**——共生酵母转丝状致病型。
> - 皮损可**hypopigmented 色素减退 / hyperpigmented 色素沉着 / erythematous 红斑**（故名"versicolor 变色"）+ **细鳞（evoked scale 刮之更明显）**。
> - 诊断多临床；KOH（potassium hydroxide 氢氧化钾）镜检 = **"spaghetti-and-meatballs 意面加肉丸"**（短粗菌丝 + 厚壁芽生孢子）。
> - 治疗：**topical selenium sulfide 二硫化硒** 或 **ketoconazole 酮康唑 / terbinafine 特比萘芬**；色素恢复需数月。
> 
> ## 题干线索→临床意义
> - "夏季加重 + 游泳池 / 出汗" → Malassezia 转致病型的诱因。
> - "上背 / 肩 / 胸融合斑 + 细鳞" → 花斑癣分布与形态。
> 
> ## 易混陷阱（普适）
> - **Topical corticosteroid 外用激素**：会**促进 Malassezia 生长、加重 TV**（本次错选）。
> - **IM penicillin G**：治二期梅毒（掌跖 + 全身斑丘疹 + 淋巴结）。
> - **仅避免日晒**：只让皮损不明显，**不治愈**感染。
> - **Oral antihistamine 口服抗组胺**：治荨麻疹（风团，24h 消退）。
> 
> ## 我为什么错
> - 选了：A（Corticosteroid ointment 外用激素）
> - 错因：知识
> - 核心陷阱：把真菌感染当炎症性皮病用激素；激素反而"喂"真菌、加重。
> 
> ## Memory Hook
> **「上背融合色素斑 + 夏季 / 出汗加重 = 花斑癣（Malassezia）：抗真菌（硒 / 酮康唑），激素会喂它。」**
>

> [!example]- [2026-07-03] 皮肤 / 大疱性类天疱疮初始治疗 Bullous pemphigoid (Q2751)
> ^Q2751
> 
> ## Stem 模式
> 65 岁女双大腿内侧 3 周红斑瘙痒 → 出现多个**紧张大疱**，无口腔 / 眼 / 黏膜受累；既往**早期 Parkinson 病**；问活检结果前的**初始治疗**。
> 
> ## 核心概念
> **Bullous pemphigoid 大疱性类天疱疮 (BP)** = 自身免疫**表皮下 (subepidermal)** 大疱，抗**hemidesmosome 半桥粒**抗原；老年（**>60**）、**神经系统病（Parkinson / 痴呆）**关联。
> 
> ## 普适规则
> - **前驱期**瘙痒性湿疹 / 荨麻疹样疹（数周-数月）→ 出现**紧张大疱 (tense bullae)**；黏膜**罕见**，**瘙痒**（非疼痛）。
> - 诊断：皮肤活检 + **DIF 直接免疫荧光（基底膜线状 IgG + C3）**；血清抗 BP 抗原抗体支持。
> - **一线治疗 = 高效价外用糖皮质激素 clobetasol 氯倍他索（即使广泛病变也有效）**；需系统治疗时用 **oral doxycycline 多西环素** 或口服激素（后者不比外用更有效）。
> 
> ## 题干线索→临床意义
> - ">60 岁 + Parkinson 病" → BP 的年龄与神经关联。
> - "先瘙痒红斑数周 → 紧张大疱" → BP 前驱 + 紧张疱。
> - "无黏膜受累 + 瘙痒非疼痛" → 支持 BP、排除 PV（天疱疮累黏膜 + 疼痛 + 松弛疱）。
> 
> ## 易混陷阱（普适）
> - **Oral dapsone 氨苯砜 + 无麸质饮食**：是 **Dermatitis herpetiformis 疱疹样皮炎（DH）**（乳糜泻，肘 / 膝 / 臀剧痒小疱，无前驱大疱）的治疗——本次错选。
> - **Oral valacyclovir**：治带状疱疹（单侧皮节 grouped vesicles + 神经痛，非瘙痒）。
> - **Topical clotrimazole 克霉唑**：治体癣（环形鳞屑斑）。
> - **Pemphigus vulgaris 寻常型天疱疮**：松弛易破疱 + **黏膜必累（口腔常首发）** + Nikolsky +；本例无黏膜。
> 
> ## 我为什么错
> - 选了：B（Oral dapsone 氨苯砜）
> - 错因：知识（治疗一线）
> - 核心陷阱：把 DH 的药（dapsone）用到 BP 上；BP 一线是**高效价外用激素 clobetasol**。
> 
> ## Memory Hook
> **「>60 + Parkinson + 紧张大疱 + 瘙痒无黏膜 = BP：一线高效价外用激素（clobetasol）；dapsone 是 DH 的。」**
>

> [!example]- [2026-07-03] 皮肤 / 汗疱疹 Dyshidrotic eczema (Q16841)
> ^Q16841
> 
> ## Stem 模式
> 43 岁男 1 周手部**水疱疹 + 剧痒**；前 2 月内类似发作 2 次、均以**手部脱屑**收场；无家用 / 职业化学暴露，问诊断。
> 
> ## 核心概念
> **Dyshidrotic eczema 汗疱疹（pompholyx / acute palmoplantar eczema 急性掌跖湿疹）** = 复发性、**手足深在瘙痒性水疱**。
> 
> ## 普适规则
> - **复发性急性发作**；**deep-seated 深在小疱 / 大疱**于**掌、跖、指侧**。
> - 并发症：脱屑 (desquamation)、慢性皮炎、继发感染。
> - 病因不明、多因素（可与刺激 / 变应性接触性皮炎、UV、远处真菌反应相关）。
> - 诊断临床；治疗 **emollients 润肤剂 + 高 / 超高效价外用激素（betamethasone 倍他米松等）**。
> 
> ## 题干线索→临床意义
> - "2 月内复发 + 手部深在痒疱 + 每次脱屑愈" → 汗疱疹的**复发刻板模式 (stereotyped recurrences)**。
> - "无职业 / 家用化学暴露" → 削弱刺激性 / 接触性皮炎，倾向内源性汗疱疹。
> 
> ## 易混陷阱（普适）
> - **Nummular eczema 钱币状湿疹**：圆形（硬币状）**鳞屑斑块**、干燥区、多在**下肢**——**非水疱**（本次错选）。
> - **Herpetic whitlow 疱疹性瘭疽**：局部痛性水疱 / 脓疱，**不痒**，通常单指 / 单手。
> - **Psoriasis 银屑病**：界清鳞屑斑块，无水疱、无剧痒。
> - **Scabies 疥疮**：指缝 + 夜间剧痒 + 隧道。
> - **Tinea manuum 手癣**：常**单侧** + 慢性鳞屑 + 多合并足癣，非复发急性疱。
> 
> ## 我为什么错
> - 选了：C（Nummular eczema 钱币状湿疹）
> - 错因：pattern / 干扰项
> - 核心陷阱：把复发**水疱**疹误当 nummular；nummular 是**鳞屑斑块**不是水疱。
> 
> ## Memory Hook
> **「手足复发深在痒疱 + 每次脱屑 = 汗疱疹（pompholyx）；钱币状湿疹是鳞屑斑不是疱。」**

> [!example]- [2026-07-03] 皮肤 / 婴儿血管瘤 Infantile hemangioma (Q4404)
> ^Q4404
> 
> ## Stem 模式
> 5 月龄女婴臀部皮损；**生后数周**现小红斑（被当虫咬），外用氢化可的松无效，**近 2 月进行性增大**；图 = 鲜红隆起斑块。
> 
> ## 核心概念
> **Infantile hemangioma 婴儿血管瘤（strawberry hemangioma 草莓状血管瘤）** = 良性**毛细血管肿瘤**，内皮细胞异常增殖。
> 
> ## 普适规则
> - 自然史：**生后数天-数周出现** → **1 岁内快速增殖**（鲜红、软、隆起）→ **1 岁后自行消退**（数年）。
> - 多数**仅观察**；高危 → **beta blocker β 受体阻滞剂 propranolol 普萘洛尔**：大 / 面部 / 节段 / 快速增长 / **溃疡**、**眶周（视力）**、**肝（高输出心衰）**、**声门下（气道梗阻）**。
> - 特殊评估：**≥5 个皮肤病灶 → 肝超声**；**面部节段 → PHACE 综合征筛查**（Posterior fossa 后颅窝畸形、Hemangioma 血管瘤、Arterial 动脉异常、Cardiac 心脏异常、Eye 眼异常；查超声心动 + 头 MRI）；颈面"胡须"分布 → 喉镜；**腰骶 → 脊髓超声**。
> 
> ## 题干线索→临床意义
> - "生后数周出现 + 近月快速增大 + 鲜红隆起" → 婴儿血管瘤**增殖期**自然史。
> - "外用激素无效、继续长大" → 非虫咬 / 皮炎，是血管肿瘤增殖。
> 
> ## 易混陷阱（普适）
> - **Cherry angioma 樱桃状血管瘤**：**成人**躯干小红点，随年龄增多；非婴儿快速增大（本次错选）。
> - **Arteriovenous malformation 动静脉畸形**：紫红斑 + 可见血管 + **震颤 / 杂音**、温热。
> - **Nevus flammeus 鲜红斑痣（port-wine stain）**：**出生即有、扁平、不消退**（可伴 Sturge-Weber）。
> - **Nevus simplex（鲑鱼斑 / 鹳吻痕 stork bite）**：眼睑 / 眉间 / 颈项，褪色、多 1-2 岁自消。
> - **Pyogenic granuloma 化脓性肉芽肿**：小、易出血、圆顶丘疹，**不消退**。
> 
> ## 我为什么错
> - 选了：B（Cherry angioma 樱桃状血管瘤）
> - 错因：pattern
> - 核心陷阱：忽略"婴儿 + 生后数周现 + 快速增大"的年龄 / 自然史，把它当成人 cherry angioma。
> 
> ## Memory Hook
> **「婴儿生后数周现 + 快速增大鲜红隆起 = 婴儿血管瘤（草莓状），会自消；cherry angioma 是成人的。」**

> [!success]- [2026-07-03] ✅做对 皮肤 / 变应性接触性皮炎 ACD（记忆 T 细胞）(Q20937)
> ^Q20937
> 
> ## Stem 模式
> 21 岁女露营 / 徒步（科罗拉多）后**上肢 + 躯干痒性红疹 → 小疱**；自述**跌入灌木 + 数处虫咬**；问皮损最可能的**病因机制**。
> 
> ## 核心概念
> **Allergic contact dermatitis 变应性接触性皮炎 (ACD)** = **IV 型迟发型超敏**，由**记忆 T 细胞激活 (activation of memory T cells)** 介导；常见触发 **Rhus 毒漆植物（poison ivy / oak）**、镍、染料、外用药。
> 
> ## 普适规则
> - 两相：
>   - **致敏相 (sensitization)**：初次暴露，naive T 细胞在区域淋巴结分化为致敏 T，**不出皮损**。
>   - **激发相 (elicitation)**：再暴露，**记忆 T 细胞增殖**释放细胞因子（CD8 介导 + 巨噬细胞募集），**48-72h** 出疹。
> - 急性 = 痒、红斑、水疱 / 大疱（常**线状**，接触植物处）；慢性 = 苔藓化 / 皲裂。
> - 治疗：**避敏 + 外用 / 系统糖皮质激素**；顽固者斑贴试验。
> 
> ## 做对要点
> - 抓"**接触植物（灌木 / poison ivy）后线状痒疱**" = IV 型迟发超敏 = **记忆 T 细胞**（激发相）。
> 
> ## 易混陷阱（普适）
> - **Autoantibody formation 自身抗体**：BP（bullous pemphigoid 大疱性类天疱疮）等自身免疫大疱，非 ACD。
> - **Bacterial toxin absorption 细菌毒素吸收**：TSS（toxic shock syndrome 中毒性休克综合征；高热 + 低血压 + 多系统），非局部痒疱。
> - **Drug-induced keratinocyte necrosis 药物致角质细胞坏死**：SJS / TEN（Stevens-Johnson syndrome 史蒂文斯-约翰逊综合征 / toxic epidermal necrolysis 中毒性表皮坏死松解症；疼痛 + 黏膜 + 剥脱）。
> - **Rickettsial infection 立克次体感染**：RMSF（Rocky Mountain spotted fever 落基山斑疹热；腕踝起斑丘疹 + 发热 / 头痛 / 肌痛）。
> 
> ## Memory Hook
> **「接触 poison ivy 后线状痒疱 = ACD = IV 型迟发超敏 = 记忆 T 细胞（激发相 48-72h）。」**

# 📝 错题处理工作流

> 本目录用于 **UWorld 错题的结构化整理 + 通用医学知识可视化**。
> 所有内容均基于公开教科书重新整合，**严格不复制 UWorld 原文**。

---

## 📁 目录约定

```
mistakes/
├── README.md                      # 本文件 — 工作流说明
├── _template.md                   # 单题错题模板
├── _weekly_review_template.md     # 周总结模板
├── 2026-05-09.md                  # 每日错题记录（按日期命名）
├── 2026-05-10.md                  # ...
├── weekly-2026-W19.md             # 周总结（按 ISO 周）
└── uworld-mistakes.md             # 旧版总索引（保留作历史归档）
```

### 命名规则
- **每日错题文件**：`YYYY-MM-DD.md`，**一天一个文件**，当天所有错题写在同一个文件里
- **周总结文件**：`weekly-YYYY-Www.md`（ISO 周编号，如 `weekly-2026-W19.md`）
- 下划线开头的 `_template.md` / `_weekly_review_template.md` 是**模板**，不要直接编辑

## 🚀 快速开始：如何用模板创建新错题

### 方式 A — 让 Claude Code 帮你建（推荐）
直接对话：

> "我做错了 UW#12345，[一句话题型]，帮我用 mistakes/_template.md 套个错题笔记到 mistakes/2026-05-09.md"

Claude 会自动：
1. 检查今天是否已有文件，没有就新建
2. 套用 `_template.md` 结构
3. 基于通用医学知识填充病机、鉴别表、决策图
4. 在文末加双链关联到对应 `notes/*.md`

### 方式 B — 手动复制
1. 打开 `_template.md`，全选复制
2. 在 `mistakes/YYYY-MM-DD.md` 末尾粘贴（如果是第一道题，新建该文件）
3. 把所有 `[占位符]` 替换成你的内容

```

请完成：
1. 扫描 `mistakes/` 下本周所有 `YYYY-MM-DD.md` 错题文件

3. 自动统计：
   - 做题总量、错题数、错题率
   - 按系统分布（含 Mermaid pie chart 数据）
   - 错因分类频次
4. 识别我"反复犯的"思维错误模式（同一类错因 ≥ 3 次的标记出来）
5. 推荐下周重点复习的 `notes/*.md` 文件清单（按错题密度 + 优先级排序）

🚫 总结里**不复制任何 UW 原文**，只引用错题号 + 我自己写的提炼。

完成后告诉我：
- 这周最值得周末重点回看的 3 个 notes/*.md
- 一句最重要的下周提醒
```

## ⚠️ 版权重要说明（必读）

### ❌ 不要做的事

- 直接复制 UWorld **题干** 到笔记里
- 直接复制 UWorld **解析（explanation）** 段落到笔记里
- 截图 UWorld 的 **原图、原表格** 保存到仓库
- 把 UWorld 内容传到 GitHub 公开仓库 / 任何第三方平台

### ✅ 正确做法

- 让 Claude Code 基于 **通用医学知识** 重新整合：
  - **First Aid for USMLE Step 2 CK**
  - **Master the Boards Step 2 CK**
  - **UpToDate**（临床决策）
  - **AMBOSS**（公开学习库）
  - **Pathoma / Goljan**（病理基础）
  - **CDC / NIH / 各专科指南**（公共领域）
- 这种 **"通用医学知识可视化"** 既能帮你深入理解，又完全合法
- 你的笔记应该是 **"消化后的知识"**，而不是 "UWorld 内容存档"

### 💡 为什么这样做反而更好

1. **学习更深入**：UW 解析是"为这道题量身定制"的，覆盖面窄；教科书内容是"系统性"的，能让你举一反三
2. **可视化更通用**：Claude 画的鉴别表 / 决策图覆盖整个疾病谱，不只是这一道题
3. **法律安全**：UW 用户协议禁止复制传播；公开教科书的事实性医学知识不受版权限制（具体表达受版权保护，所以让 Claude **重写** 而不是引用原文）
4. **复习效率高**：消化后的笔记 = 你大脑里的 schema，比原文摘录更容易记忆

### 🔒 仓库保护机制

- `.gitignore` 已忽略 `*.png / *.jpg / *.pdf` 等文件 — 即使你不小心把 UW 截图保存到仓库目录，也不会被 git 推送
- 如果不小心把 UW 文字内容 commit 了，**立刻**用 `git reset --soft HEAD~1` 撤销（推送前）
- 如果已经推送，让 Claude Code 帮你处理（涉及 force push，要谨慎）

## 🔗 与其他文件的关系

```
mistakes/YYYY-MM-DD.md  ──[[双链]]──>  notes/<system>.md
                                            │
                                            ↓
                                   系统笔记里加一条 "易错点"
                                   并反向链接回错题
```

**双向关联示例**：

`mistakes/2026-05-09.md`：
```markdown
## 🔗 关联笔记
- [[notes/cardiovascular#1. 缺血性心脏病]]
```

`notes/cardiovascular.md` 第 1 节末尾：
```markdown
**易错点**（来自错题）：
- ACS 中 STEMI 与 dissection 鉴别 → [[mistakes/2026-05-09]]
```

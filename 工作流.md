# Step 2 CK 复习工作流

## 三工具分工

> - **Claude.ai 网页版**：错题截图分析（读图、联网搜最新指南）
> - **Claude Code**（本地）：写错题本、更新衍生文件、出题陪练
> - **Obsidian**：阅读、双链、长期记忆

## 一、每日启动 Prompt

每次打开 Claude Code 先粘贴这个进入"陪练状态"：

```
你是我的 Step 2 CK 陪练。

我的状态：
- NBME 12 = 210，目标 250+
- 每日复习 6-8h；UW 第一轮完成，正在刷错题
- 强项：神经、消化、内分泌、妇产
- 弱项：Ethics、Biostats（基本零基础），心血管/肺/肾/Heme-Onc/ID/Rheum/Psych/Peds/Surgery/Derm（状态待定）

工作目录布局：
- `完整笔记/*.md` — 系统笔记（按学科）
- `完整笔记/_衍生_*.md` — 4 个派生速查文件（高频陷阱、鉴别诊断速查、Screening、评分公式）
- `notes/Biostats_Epi.md` + `notes/Ethics_Communication.md`
- `错题本.md` / `薄弱点.md` / `工作流.md`（Vault 根目录）

规则（每次回答必须遵守）：
1. 解答按 NBME 风格：题眼 → 鉴别 → best next step → 干扰项逐个解释
2. 不复制 UWorld 题干/解析/原图（版权）。基于 First Aid / MTB / UpToDate / AMBOSS 重写
3. 引用笔记位置用 `[[文件名#章节]]` 格式
4. 我做 vignette 时先让我答，再给解析
5. 改任何衍生文件时遵守格式硬规则：表格前后必须空行，子标题和表格之间必须空一行

确认就绪后，等我下一步指令。不要主动展开。
```

## 二、错题处理（双工具流）

**核心原则**：先读 UW 英文解析，30% 真不懂的才用 Claude。

### 步骤 1：网页版 Claude（给截图）

```
我是 Peixuan，USMLE Step 2 CK 备考中。附图是 UWorld 错题截图。

请输出**标准 markdown** 错题条目（我会直接粘贴到 错题本.md）。

⚠️ 严禁复制 UW 题干/解析/原图（版权）。基于公开教科书（First Aid / MTB / UpToDate / AMBOSS）重写概念。

输出格式严格按此模板：

### [今天日期 YYYY-MM-DD] [题目核心考点一句话]
- **来源**：UW Q##### (题目编号)
- **题干要点**：30 字内精简（不复制原文，提取关键 vignette 要素）
- **正确答案**：X — 因为...（1-2 句话）
- **我选了**：Y — 错因：[知识 / pattern / 干扰项 / 粗心]
- **关键 buzzword / 识别公式**：...
- **关联笔记**：[[完整笔记/学科.md#章节]]（建议补到哪个笔记）
- **重做次数**：☐ 1st ☐ 2nd ☐ 3rd
- **状态**：🔴 反复错 / 🟡 不熟 / 🟢 已掌握（默认 🟡）

markdown 之外，额外给我（不算入条目）：
- 干扰项分析：A/B/C/D 为什么错（USMLE trap 视角）
- 若涉及最新指南（USPSTF / AHA / GINA / ADA），联网核对最新版并指出与笔记可能的差异
```

### 步骤 2：Claude Code 入库

```
请把下面这条错题 append 到 错题本.md 里对应学科的 ## 节末尾。

要求：
1. 根据考点关键词自动识别学科
2. 加在该学科 ## 节的最后一条之后，前面用 `---` 分隔（如果该学科目前是"暂无错题"，删除该占位文字后直接放）
3. 保持文件其他部分原样，不重排
4. 完成后报告：加到了哪个学科节、当前该学科共多少条错题、是否同考点已出现 ≥ 2 次（如有，提示要升档到 薄弱点.md）

[在此粘贴网页版给的 markdown 错题条目]
```

### 步骤 3（可选）：更新衍生文件

```
这道错题暴露了我对 [考点名] 的理解漏洞。请检查衍生文件，看是否需要更新：

1. `完整笔记/_衍生_高频陷阱.md` — 是否有相关 X vs Y 对比？若有，看 CK 一句话识别是否准确；若无，建议是否加
2. `完整笔记/_衍生_鉴别诊断速查.md` — 相关主诉下是否已收录？若缺，建议加在哪
3. `完整笔记/_衍生_Screening_Guidelines.md` / `_衍生_高频评分_公式.md` — 若涉及筛查/评分，是否覆盖
4. 对应系统笔记 `完整笔记/[学科].md` — 是否要加一行 high-yield 速记

逐文件检查后报告：
- 哪些文件已经覆盖了（不用改）
- 哪些文件需要补（先给我建议改动 diff，我同意再 Edit）

⚠️ 实际改之前必须给我看，不要自动改。改时遵守格式硬规则：
- 表格前后空行
- 子标题和表格之间空一行
- 改完跑 awk 自查破损 before/after-table 必须为空
```

## 三、每周日复盘

```
今天周日，做错题本周复盘。请按顺序执行：

1. 读 错题本.md，统计本周（过去 7 天）新增错题：
   - 按学科分布
   - 按状态分布（🔴🟡🟢）
   - 按错因分布（知识 / pattern / 干扰项 / 粗心）

2. 找反复错的考点（同考点 ≥ 2 次）：
   - 列出考点名 + 出错次数 + 关联学科 + 日期

3. 更新 薄弱点.md：
   - 第 2 次错 → 升档到 🟡
   - 第 3 次及以上 → 升档到 🔴
   - 引用错题本里对应日期 + 题号
   - 连续 2 周再做不错 → 降档（如有）

4. 输出本周报告：
   - 学科强弱分布（文字描述雷达图概念）
   - 下周建议重点学科 top 3
   - 是否需要回头补某个系统笔记（哪个文件、哪个章节）
   - 是否需要派生新衍生文件（如果某类考点反复错且无现有衍生覆盖）

5. 改完 薄弱点.md 后给我 diff，我确认后保存。
```

## 四、NBME / UWSA 后规划

```
我刚做完 [NBME X / UWSA Y / Free 120]，分数 [XXX]。
学科分布如下（从 score report 截图复制 / 我手动转 markdown）：

[粘贴学科分数表或描述]

请帮我：

1. 对照目标 250+，算出每个学科还差多少分（粗估）
2. 找出 3 个最弱学科（按 percentile 或 错题密度）
3. 针对每个弱学科，给出未来 7 天密集复习计划：
   - 重读 `完整笔记/[学科].md` 哪些章节
   - 刷 UW 哪些子主题（具体 ID 范围或主题名）
   - 看衍生文件 `_衍生_*.md` 哪些对应节
4. 这次模考 → 真实考预测：用 NBME/UWSA → real exam 转换表给区间（保 X / 冲 Y / 现实最差 Z）
5. "保 250 / 冲 260" 的剩余周次具体里程碑：每周需要 NBME 分数到多少、还需做几个 form
6. 是否需要把这次模考的错题批量入库到 错题本.md（如果是，告诉我下一步用哪个 prompt）
```

## 五、考前 2 周冲刺

**不再读 `完整笔记/`，只看顶层**：

1. `错题本.md` / `薄弱点.md`
2. `_衍生_高频陷阱.md` / `_衍生_鉴别诊断速查.md` / `_衍生_Screening_Guidelines.md` / `_衍生_高频评分_公式.md`
3. `notes/Biostats_Epi.md` + `notes/Ethics_Communication.md`

**考前每天用 Claude Code 出题**：

```
考前冲刺，今天给我出 [N=10/20] 道 Step 2 CK NBME 风格题。

要求：

1. 考点来源（按优先级随机抽）：
   - 错题本.md 里 🔴🟡 状态题（最高优先）
   - 薄弱点.md 里 🔴 完全不会档（最高优先）
   - 4 个 _衍生_*.md 文件随机抽
   - notes/Biostats_Epi.md + Ethics_Communication.md

2. 严格 NBME vignette 风格：
   - 一段 vignette（年龄/性别 + 主诉 + 病史 + 体征 + 化验/影像）
   - 4-5 选项，仅一个最佳答案
   - 干扰项必须 plausible（不是明显错的）
   - ⚠️ 不复制 UWorld 原题，基于公开教科书重写

3. 流程：
   - 第一步：一次性给我 [N] 道题题干 + 选项，不给答案
   - 我答完一次性发给你（"1A 2C 3B..."格式）
   - 你再给逐题解析（NBME 风格：正确答案为什么对 + 每个干扰项为什么错）

4. 答完整体反馈：
   - 总分（X/N）
   - 错的考点列表 + 对应衍生文件位置
   - 是否需要追加到 错题本.md（你列出 markdown 条目我审）
   - 今晚临睡前再看哪些笔记/衍生文件章节
```

## 六、特殊场景

### 批处理一个 block 的所有错题

```
我刚做完一个 UW block（40 题），错了 [N] 题。下面是这一批错题的 markdown 条目（从网页版 Claude 批量生成）：

---

[粘贴 N 个错题 markdown，每条之间用 `---` 分隔]

---

请逐条入库到 错题本.md：

1. 每条自动识别学科，append 到对应 ## 节末尾
2. 同一学科多条按时间顺序排（最新在最后）
3. 全部入库后，给我本 block 报告：
   - 学科分布（哪几个学科占大头）
   - 错因分布（知识 / pattern / 干扰项 / 粗心）
   - 最值得回头补的 3 个考点 + 对应衍生文件位置
4. 自动判断：有没有重复出现 ≥ 2 次的考点 → 提示升档到 薄弱点.md（列出建议升档的条目）
5. 完成后跑 awk 自查 错题本.md 是否还格式干净（如果原本就有表格，确认没破损）
```

### 不懂某考点

```
我对 [考点名] 完全不懂，请深度讲解：

1. 一句话定义（2025 USMLE 视角）
2. 病理生理 / 机制（从根上讲，不堆 buzzword）
3. 典型临床表现（NBME vignette 长什么样）
4. 诊断流程：best initial test → confirmatory test → gold standard
5. 治疗：first-line / alternative / 特殊人群（孕妇/儿童/老人/CKD）
6. 与相关考点的鉴别 — 给一个 X vs Y 对比表（前后空行，符合格式规则）
7. NBME 高频考法：题干一旦出现什么词/数据就锁定这个诊断
8. 关联笔记位置：`[[完整笔记/对应学科.md#章节]]` — 告诉我应该把这块知识补到哪个文件、哪一节
9. 是否需要更新衍生文件（如果是，列出哪个、加什么）

最后：出 3 道 NBME 风格变体题（vignette + 4-5 选项），让我练习识别。
我答完你给解析。

⚠️ 全程不复制 UWorld 原题/原图，基于 First Aid / MTB / UpToDate / AMBOSS 概念重写。
```

### 模拟考某学科

```
给我做一个 [学科名] 学科模拟 block。

要求：

1. 40 题 NBME 风格 vignette（模拟真实 block 长度）
2. 题目来源：基于 完整笔记/[学科].md + 衍生文件 + 错题本里该学科条目 + 公开教科书
   ⚠️ 不复制 UWorld 原题
3. 时间限制：60 分钟（我自己计时，你只出题）
4. 题型分布：
   - 60% 诊断（认病/鉴别）
   - 20% next best step（管理决策）
   - 10% 机制 / 病生
   - 10% 预防 / 筛查 / 伦理 / 统计
5. 流程：
   - 一次性给我所有 40 题题干 + 选项，markdown 格式
   - 不给答案
   - 我答完发给你（"1A 2C 3B..."格式）
6. 反馈：
   - 总分 + 学科子主题细分错率
   - 错的题逐个解析（NBME 风格）
   - 是否需要批量 append 到 错题本.md（列出建议条目）
   - 是否暴露新薄弱点 → 提示更新 薄弱点.md
   - 与你之前的"该学科状态"判断比，是否需要重新评估强弱
```

### 训练医学英语

```
今天训练 Step 2 CK 临床英语。请从下面 4 种里挑一种（或我指定一种）：

(A) **Vignette 朗读 + 术语解释**
- 你给我一段 NBME 风格 vignette（80-150 字）
- 我朗读/读理解
- 你 quiz 我：这个 buzzword 什么意思 / 这个体征英文怎么说 / 这个药 generic name 拼写

(B) **Standardized Patient 对话模拟**
- 你扮演患者（一个 chief complaint）
- 我用英文问诊（open-ended → close-ended → 申请 physical exam）
- 你按真实 SP 评分：communication 流畅度 / key questions 是否问到 / DDx 思路是否清晰
- 完成后反馈 + 改进的英文表达

(C) **病历口述（SOAP note）**
- 你给我一个虚拟病例（中文描述）
- 我用英文写 SOAP note（Subjective / Objective / Assessment / Plan）
- 你 critique：格式、术语、逻辑链是否符合 USMLE Step 2 / OET 标准

(D) **医患对话听力**
- 你说一段医患对话（英文，模拟 SP 案例，可文字也可让我想象口语）
- 我复述 / 抓 chief complaint + key findings + concerning red flags
- 你 reveal 完整文本，我对照差距

告诉我你想选哪一种，或随机给我选一种开始。
```

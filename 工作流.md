# Step 2 CK 复习工作流（Prompt 库）

> **此文件是 Prompt 库 + 操作手册。** 规则手册见 [[CLAUDE.md]]。
> 
> 日常使用：从下面找到对应场景 → 复制 prompt → 粘贴到 Claude Code 或 Claude.ai 网页版。

---

## 三工具分工

> - **Claude.ai 网页版**：错题截图分析（读图、联网搜最新指南、长对话学习）
> - **Claude Code**（本地）：写错题本、更新衍生文件、出题陪练、整合主笔记
> - **Obsidian**：阅读、双链、长期记忆、**Dataview 聚合视图**

详细分工见 [[CLAUDE.md]] § 四。

## 一、每日启动 Prompt

每次打开 Claude Code 先粘贴这个进入"陪练状态"：

```
你是我的 Step 2 CK 陪练。

我的状态：
- NBME 12 = 210，目标 250+
- 每日复习 6-8h；UW 第一轮完成，正在刷错题
- 强项：神经、消化、内分泌、妇产
- 弱项：Ethics、Biostats（基本零基础），心血管/肺/肾/Heme-Onc/ID/Rheum/Psych/Peds/Surgery/Derm（状态待定）

工作目录布局（详细见 CLAUDE.md）：
- `完整笔记/Peixuan分科笔记/*.md` — 系统笔记（按学科）
- `完整笔记/专题笔记/_衍生_*.md` — 派生速查文件
- `notes/Biostats_Epi.md` + `notes/Ethics_Communication.md`
- `mistakes/uworld-mistakes.md` — 错题原始数据（按日期 ## 分节）
- `错题本.md` — 错题 Dataview 聚合视图（按学科/状态/错因看）
- `薄弱点.md` / `工作流.md` / `CLAUDE.md`
- `草稿/` — 待整合主笔记草稿

规则（每次回答必须遵守）：
1. 解答按 NBME 风格：题眼 → 鉴别 → best next step → 干扰项逐个解释
2. 不复制 UWorld 题干/解析/原图（版权）。基于 First Aid / MTB / UpToDate / AMBOSS 重写
3. 引用笔记位置用 `[[完整笔记/Peixuan分科笔记/学科名]]` 或 `[[完整笔记/专题笔记/笔记名]]` 格式
4. 我做 vignette 时先让我答，再给解析
5. 改任何衍生文件时遵守格式硬规则：表格前后必须空行，子标题和表格之间必须空一行
6. 生成错题卡时必须按三段式结构：知识区（Stem/核心/规则/陷阱/我为什么错/Memory Hook）→ `---` → 学习记录区（提问/关联/复盘）→ `---` → 4 个 inline fields（学科/主题/状态/错因）沉底

确认就绪后，等我下一步指令。不要主动展开。
```

## 二、错题处理（双工具流）

**核心原则**：先读 UW 英文解析，30% 真不懂的才用 Claude。

### 步骤 1：网页版 Claude.ai（给截图，对话式深度分析）

```
我是 Peixuan，USMLE Step 2 CK 备考中。附图是 UWorld 错题截图。

请按 9 部分模板分析（详细学习版，用于对话讨论）：
1. Stem 关键词识别（双语表）
2. 机制理解（ASCII 流程图）
3. 正确答案 + 核心概念
4. 干扰项逐项击破
5. 我为什么错（个性化反思）
6. High-Yield Summary Table
7. Memory Hook（记忆挂钩）
8. 关联（按 v2 规则 4 类齐全）
9. 复盘行动清单

⚠️ 严禁复制 UW 题干/解析/原图（版权）。基于公开教科书重写概念。

分析完后，如果我说"给入库版"或"给浓缩 breakdown"，再输出 30-50 行的精简 callout 折叠版本（用于贴到错题本）。

入库版必须严格按三段式结构：

> [!example]- [YYYY-MM-DD] 学科 / 主题 (Q#####)
> 
> [知识区]
> ## Stem 模式
> ## 核心概念
> ## 普适规则
> ## 易混陷阱（普适）
> ## 我为什么错
> ## Memory Hook
> 
> ---
> 
> [学习记录区]
> ## 🤔 我的提问 / 卡点
> ## 🔗 关联（4 类齐全：🔁 同主题错题 / 📚 主笔记 / 🏥 跨学科 / 🌱 TODO）
> ## ✅ 复盘行动
> 
> ---
> 
> [元数据沉底]
> 学科:: [Biostats / Ethics / 心血管 / 肺 / 肾 / Heme-Onc / 感染 / GI / 内分泌 / GYN / OB / 神经 / Psych / 儿科 / 风湿 / 皮肤 / 眼科 / 外科 / 骨科]
> 主题:: [一句话功能性考点名]
> 状态:: 🟡 (默认；同主题第 2+ 次错升 🔴)
> 错因:: [知识 / pattern / 干扰项 / 粗心 / 计算]
```

### 步骤 2：Claude Code 入库（callout 折叠 + 校验三段式 + inline fields）

```
请把下面这条错题 append 到 mistakes/uworld-mistakes.md 里今天日期（YYYY-MM-DD）对应的 ## 节末尾。

要求：
1. 找到 ## YYYY-MM-DD（今天日期）的 section
   - 没有就在文件末尾新建一个 ## YYYY-MM-DD section
2. 在该 section 已有错题之后加 `---` 分隔，然后追加新卡
3. 校验卡的三段式结构：
   - 知识区（Stem/核心/规则/陷阱/我为什么错/Memory Hook）→ `---` → 学习记录区（提问/关联/复盘）→ `---` → inline fields
   - 4 个 inline fields 在卡的最底部（学科/主题/状态/错因）
   - 缺则提示我补
   - 字段值范围见 CLAUDE.md § 七
4. 保持文件其他部分原样，不重排
5. 完成后报告：
   - 加到了哪个日期节
   - 该错题学科 + 主题 + 状态
   - 是否同主题已出现 ≥ 2 次（如有，提示要把该卡 状态 改成 🔴 并升档到 薄弱点.md）
   - 提示我刷新 错题本.md 看更新后的聚合视图
6. 改完跑 awk 自查 mistakes/uworld-mistakes.md 是否还格式干净

[在此粘贴网页版给的 callout 折叠错题卡]
```

### 步骤 3（可选）：更新衍生文件

```
这道错题暴露了我对 [考点名] 的理解漏洞。请检查衍生文件，看是否需要更新：

1. `完整笔记/专题笔记/_衍生_高频陷阱.md` — 是否有相关 X vs Y 对比？若有，看 CK 一句话识别是否准确；若无，建议是否加
2. `完整笔记/专题笔记/_衍生_鉴别诊断速查.md` — 相关主诉下是否已收录？若缺，建议加在哪
3. `完整笔记/专题笔记/_衍生_高频评分_公式.md` — 若涉及评分/公式，是否覆盖
4. 对应学科主笔记 `完整笔记/Peixuan分科笔记/[学科].md` — 是否要加一行 high-yield 速记

逐文件检查后报告：
- 哪些文件已经覆盖了（不用改）
- 哪些文件需要补（先给我建议改动 diff，我同意再 Edit）

⚠️ 实际改之前必须给我看，不要自动改。改时遵守格式硬规则：
- 表格前后空行
- 子标题和表格之间空一行
- 改完跑 awk 自查破损 before/after-table 必须为空
```

## 三、每周日复盘（用 Dataview 视图）

```
今天周日，做错题本周复盘。请按顺序执行：

1. 让我先在 Obsidian 打开 错题本.md 刷新 Dataview，截图给你看：
   - § 1 按学科分布
   - § 3 按错因分布
   - § 4 反复错考点（≥ 2 次）
   - § 7 本周新增错题
   - § 8 "知识"类错因专项

2. 基于截图分析：
   - 本周哪个学科错题最多 → 是否需要补该学科主笔记/草稿
   - 错因分布告诉我什么短板（知识 / pattern / 干扰项 / 粗心）
   - 反复错考点 → 哪些升档到 🔴 + 进 薄弱点.md
   - "知识"类错因多的学科 → 提示对应草稿可能需要升级

3. 更新 薄弱点.md：
   - 反复错 ≥ 2 次 → 升档到 🟡 或 🔴
   - 连续 2 周再做不错 → 降档（如有）
   - 引用错题本里对应日期 + 题号

4. 输出本周报告：
   - 学科强弱总结
   - 下周建议重点学科 top 3
   - 是否需要让 Claude Code 整合某个章节主笔记（草稿已积累够）
   - 是否需要派生新衍生文件

5. 改完 薄弱点.md 后给我 diff，我确认后保存。
```

> [!tip] 复盘升档自动化
> Dataview § 4 的"反复错考点"会自动列出 count ≥ 2 的考点。复盘时只需要扫一眼这张表，不需要手动统计。

## 四、NBME / UWSA 后规划

```
我刚做完 [NBME X / UWSA Y / Free 120]，分数 [XXX]。
学科分布如下（从 score report 截图复制 / 我手动转 markdown）：

[粘贴学科分数表或描述]

请帮我：

1. 对照目标 250+，算出每个学科还差多少分（粗估）
2. 找出 3 个最弱学科（按 percentile 或 错题密度）
3. 针对每个弱学科，给出未来 7 天密集复习计划：
   - 重读 `完整笔记/Peixuan分科笔记/[学科].md` 哪些章节
   - 刷 UW 哪些子主题（具体 ID 范围或主题名）
   - 看衍生文件 `完整笔记/专题笔记/_衍生_*.md` 哪些对应节
4. 这次模考 → 真实考预测：用 NBME/UWSA → real exam 转换表给区间（保 X / 冲 Y / 现实最差 Z）
5. "保 250 / 冲 260" 的剩余周次具体里程碑：每周需要 NBME 分数到多少、还需做几个 form
6. 是否需要把这次模考的错题批量入库到 mistakes/uworld-mistakes.md（如果是，告诉我下一步用哪个 prompt）
```

## 五、考前 2 周冲刺

**不再读 `完整笔记/Peixuan分科笔记/`，只看顶层**：

1. `错题本.md` Dataview 视图（看 § 5 🔴 反复错档 + § 9 学科 × 状态交叉表）
2. `薄弱点.md`
3. `完整笔记/专题笔记/_衍生_*.md`（所有衍生速查文件）
4. `notes/Biostats_Epi.md` + `notes/Ethics_Communication.md`

**考前每天用 Claude Code 出题**：

```
考前冲刺，今天给我出 [N=10/20] 道 Step 2 CK NBME 风格题。

要求：

1. 考点来源（按优先级随机抽）：
   - 错题本.md Dataview § 5 显示的所有 🔴 反复错卡片对应主题（最高优先）
   - 薄弱点.md 里 🔴 完全不会档（最高优先）
   - 完整笔记/专题笔记/_衍生_*.md 文件随机抽
   - notes/Biostats_Epi.md + Ethics_Communication.md

2. 严格 NBME vignette 风格：
   - 一段 vignette（年龄/性别 + 主诉 + 病史 + 体征 + 化验/影像）
   - 4-5 选项，仅一个最佳答案
   - 干扰项必须 plausible（不是明显错的）
   - ⚠️ 不复制 UWorld 原题，基于公开教科书重写

3. 流程：
   - 第一步：一次性给我 [N] 道题题干 + 选项，不给答案
   - 我答完一次性发给你（"1A 2C 3B..."格式）
   - 你再给逐题解析（NBME 风格：正确答案为什么对 + 每个干扰项为什么错）

4. 答完整体反馈：
   - 总分（X/N）
   - 错的考点列表 + 对应衍生文件位置
   - 是否需要追加到 mistakes/uworld-mistakes.md（你列出 callout 折叠卡片我审，按三段式结构 + 含 4 个 inline fields）
   - 今晚临睡前再看哪些笔记/衍生文件章节
```

```
我刚做完一个 UW block（40 题），错了 [N] 题。下面是这一批错题的 callout 折叠卡片（从网页版 Claude 批量生成，按三段式结构）：

---

[粘贴 N 个错题 callout 卡片，每条之间用 `---` 分隔]

4. 每张卡之间用 `---` 分隔
5. 全部入库后，给我本 block 报告：
   - 学科分布（看 inline fields 学科:: 字段）

8. 提示我刷新 错题本.md 看更新后的聚合视图
```

### 不懂某考点

```
我对 [考点名] 完全不懂，请深度讲解：

1. 一句话定义（2025 USMLE 视角）
2. 病理生理 / 机制（从根上讲，不堆 buzzword）
3. 典型临床表现（NBME vignette 长什么样）
4. 诊断流程：best initial test → confirmatory test → gold standard
5. 治疗：first-line / alternative / 特殊人群（孕妇/儿童/老人/CKD）
6. 与相关考点的鉴别 — 给一个 X vs Y 对比表（前后空行，符合格式规则）
7. NBME 高频考法：题干一旦出现什么词/数据就锁定这个诊断
8. 关联笔记位置：`[[完整笔记/Peixuan分科笔记/学科名]]` 或 `[[完整笔记/专题笔记/笔记名]]` — 告诉我应该把这块知识补到哪个文件、哪一节
9. 是否需要更新衍生文件（如果是，列出哪个、加什么）

最后：出 3 道 NBME 风格变体题（vignette + 4-5 选项），让我练习识别。
我答完你给解析。

⚠️ 全程不复制 UWorld 原题/原图，基于 First Aid / MTB / UpToDate / AMBOSS 概念重写。
```

### 模拟考某学科

```

要求：

1. 40 题 NBME 风格 vignette（模拟真实 block 长度）

4. 题型分布：
   - 60% 诊断（认病/鉴别）
   - 20% next best step（管理决策）
   - 10% 机制 / 病生
   - 10% 预防 / 筛查 / 伦理 / 统计
5. 流程：
   - 一次性给我所有 40 题题干 + 选项，markdown 格式

6. 反馈：
   - 总分 + 学科子主题细分错率
   - 错的题逐个解析（NBME 风格）
   - 是否需要批量 append 到 mistakes/uworld-mistakes.md（列出建议 callout 卡片，按三段式结构 + 含 4 个 inline fields）
   - 是否暴露新薄弱点 → 提示更新 薄弱点.md
```

### 训练医学英语

```
今天训练 Step 2 CK 临床英语。请从下面 4 种里挑一种（或我指定一种）：

(A) **Vignette 朗读 + 术语解释**
- 你给我一段 NBME 风格 vignette（80-150 字）
- 我朗读/读理解
- 你 quiz 我：这个 buzzword 什么意思 / 这个体征英文怎么说 / 这个药 generic name 拼写

(B) **Standardized Patient 对话模拟**
- 你扮演患者（一个 chief complaint）
- 我用英文问诊（open-ended → close-ended → 申请 physical exam）
- 你按真实 SP 评分：communication 流畅度 / key questions 是否问到 / DDx 思路是否清晰
- 完成后反馈 + 改进的英文表达

(C) **病历口述（SOAP note）**
- 你给我一个虚拟病例（中文描述）
- 我用英文写 SOAP note（Subjective / Objective / Assessment / Plan）
- 你 critique：格式、术语、逻辑链是否符合 USMLE Step 2 / OET 标准

(D) **医患对话听力**
- 你说一段医患对话（英文，模拟 SP 案例，可文字也可让我想象口语）
- 我复述 / 抓 chief complaint + key findings + concerning red flags
- 你 reveal 完整文本，我对照差距

告诉我你想选哪一种，或随机给我选一种开始。
```

## 七、主笔记生成工作流（2026-05-12 确立）⭐

### 分工

| 阶段 | 谁负责 | 产出 | 存放位置 |

### 触发 Claude Code 整合的 prompt 模板

```
请整合 [章节名，如 Biostats Bias / Hypothesis testing] 主笔记。

执行步骤：
1. 先读 CLAUDE.md
2. 读草稿：草稿/[草稿文件名，如 2026-05-12_Hypothesis_testing主笔记草稿_v2.md]
3. 读 mistakes/uworld-mistakes.md 里所有 [章节关键词] 相关的错题卡
   - 可用 grep "学科:: Biostats" 或 grep "主题:: [关键词]" 检索
4. 按 CLAUDE.md § 五的笔记规则 + § 六的关联规则 v2 整合
5. 输出到 完整笔记/专题笔记/_衍生_[具体名].md
6. 错题只在易混点处精引用（不全挂）
7. 整合完后在草稿文件开头加 "[已整合到正式笔记 YYYY-MM-DD]"
8. 不修改错题本里的卡片
9. 不直接修改 完整笔记/Peixuan分科笔记/ 下的学科主笔记

完成后给我整合摘要：
- 整合了哪些素材
- 输出文件路径
- 有哪些章节是占位（🚧 待填充）
- 有哪些 TODO 留给未来
```

### Claude Code 执行规则
1. 收到"整合 XX 章节主笔记"指令时，按上面步骤执行
2. 草稿中标记 🚧 待填充的章节 → 若错题本里有相关素材，整合进来；没有则保留占位
3. **绝不修改**：错题本卡片、学科主笔记（`Peixuan分科笔记/` 下）、`错题本.md` 里的 Dataview query

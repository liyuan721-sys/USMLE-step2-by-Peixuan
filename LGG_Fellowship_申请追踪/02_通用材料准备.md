# LGG Fellowship 通用材料准备清单

> 返回 [[00_总览_行动清单]] · 项目细节 [[01_项目详情卡]]
> 缩写：**LOR** = Letter of Recommendation（推荐信）；**PS** = Personal Statement（个人陈述）；**CV** = Curriculum Vitae（简历）

LGG 申请多为各项目独立邮件投递，但材料高度重叠。下面这套备齐后，可"一稿多投、按项目微调"。

---

## 一、核心材料（一次做好，反复用）

- [ ] **CV / 简历**
  - 突出：科研产出（论文/abstract）、实验室技术（NGS / Sanger / 核型 / FISH / 微阵列 / 变异解读等）、博后/科研年限
  - 部分项目要求注明**居留身份**（如 Wisconsin）→ 写明"Permanent Resident / Green Card"
- [ ] **Personal Statement（PS）通用版**
  - 讲清：为什么转 LGG、科研背景如何衔接临床分子诊断、职业目标（当 lab director）
  - 留好可替换段落，按项目改（如 UCLA 要呼应其 mission）
- [ ] **3 封推荐信（LOR）**
  - [ ] 推荐人 1：________
  - [ ] 推荐人 2：________
  - [ ] 推荐人 3：________
  - [ ] **第 4 位推荐人**（**Penn/CHOP 要 4 封**）：________
  - 提示：多数项目要求推荐人**直接发**到项目邮箱；提前 2–4 周打招呼
- [ ] **成绩单**（研究生 / 医学院 official transcript；部分项目要）
- [ ] **2023 ABMGG 资格审核信**（外国学位 credential review）
  - [ ] ⚠️ **先确认到 2027 入学仍有效**（见下方"二"）
  - 适用并明确要求的项目：UCLA / UW / Columbia / Brigham / Rutgers / Emory / Children's Mercy / Cincinnati / Henry Ford / Mayo / Michigan / Chicago / VCU / NHGRI 等
- [ ] **一篇发表论文 / 学术写作样本**（**Children's Mercy 必交**，其他可作附加亮点）

---

## 二、最优先：确认 2023 ABMGG 资格信是否仍有效

> [!warning] 这步影响全部申请，先做
> 你 2023 拿到的是 **eligibility / 资格审核通过**，不是认证（你还没考板考、没读 fellowship）。这类资格审核**通常有有效期窗口**，需确认到 2027 入学时是否仍有效、要不要更新。

- [ ] 发邮件问 ABMGG，建议措辞：
  > "I was approved for credential/eligibility review by ABMGG in 2023 (as a foreign medical graduate) but have not yet started an LGG fellowship. Is my eligibility approval still valid for programs starting July 2027, and is there a time limit by which I must begin training? Do I need to update or re-submit anything?"
- [ ] 记录回复 + 有效期：________

---

## 三、绿卡优势（写进 cover letter）

绿卡 = 无需签证 sponsorship，对下列对签证敏感的项目是**实打实加分**，cover letter 里点明"U.S. Permanent Resident, no visa sponsorship required"：

- **University of Colorado** — 不能 sponsor 工作签证
- **University of Pittsburgh** — 要求公民/永久居民/合法工作身份
- **University of Wisconsin** — 非公民须出示永久居留
- **Rutgers NJMS** — 须公民或永久居民
- **Nationwide Children's** — 页面默认申 J-1，**邮件说明绿卡免签证**
- **Baylor / Children's Mercy** — 涉及签证逻辑，绿卡省事

> [!tip] 投递节奏
> 1. **6 月内**：备齐 CV + PS 通用版，约好推荐人，问 ABMGG
> 2. **7 月**：投 UCLA（7/31 首轮）、UW（8/1）、Greenwood（8/1）
> 3. **8 月上中旬**：Nationwide（8/14）、Michigan（8/15）
> 4. **8 月底集中投**：Penn/CHOP、Indiana、Chicago、Baylor、Pittsburgh、Wisconsin
> 5. **9 月批**：Emory、Utah、Cincinnati、Henry Ford、Case Western、Columbia
> 6. **10 月批**：Children's Mercy、UCLA 终极

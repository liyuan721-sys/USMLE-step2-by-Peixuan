“**<span class="mark">游走性关节炎</span>**”：风湿热 \> 播散性淋球菌感染 \> 感染性心内膜炎 / Lyme / SLE / 病毒性关节炎

**1. Acute rheumatic fever（急性风湿热）**

> **最经典、最爱考**
>
> 典型是 **migratory polyarthritis**
>
> 常累及 **大关节**：膝、踝、肘、腕
>
> 常见题干：**孩子/青少年，前几周有咽痛或链球菌感染史，随后发热 + 游走性大关节炎 ± 心脏杂音/舞蹈症/皮疹**
>
> CDC 明确写到风湿热的关节炎通常是 **migratory**。

**2. Disseminated gonococcal infection, DGI（播散性淋球菌感染）**

> Step 2 也非常常考
>
> 更准确常写成 **migratory polyarthralgia**
>
> 经典三联：**关节痛 + 腱鞘炎 + 皮肤脓疱/出血性小皮疹**
>
> 常见题干：**年轻性活跃女性，发热，手腕/手指/踝疼，皮疹，可能泌尿生殖道症状不明显**
>
> CDC 指出 DGI 常见 **asymmetric polyarthralgia, tenosynovitis, skin lesions**。

**3. Infective endocarditis（感染性心内膜炎）**

> 也可出现 **migratory arthralgia/arthritis**
>
> 常见题干：**发热 + 新发杂音 + 栓塞体征/Osler nodes/Janeway lesions/IVDU/人工瓣膜**
>
> 不是最经典第一选择，但在鉴别题里会出现。

**4. Lyme disease（莱姆病）**

> 可有迁移性的关节表现，晚期更常见反复性或间歇性关节炎
>
> 题干常有：**蜱虫暴露、erythema migrans、面瘫、AV block、膝关节炎**
>
> 在“migratory arthritis”鉴别里常被提到。

**5. SLE（系统性红斑狼疮）**

> 可有 migratory pattern，但 Step 2 里通常不会把它当成最经典答案
>
> 题干更多会给：**女性、皮疹、口腔溃疡、蛋白尿、细胞减少、ANA/dsDNA**
>
> 作为鉴别记住即可。

**6. Viral arthritis（病毒性关节炎）**

> 例如 **parvovirus B19、hepatitis B/C**
>
> 更常是急性对称性多关节痛，但也可进入游走性关节炎鉴别
>
> 题干常有：**病毒前驱症状、皮疹、肝炎线索**。

**7. Reactive arthritis（反应性关节炎）**

> 严格说更典型的是 **asymmetric oligoarthritis**，不是最标准“游走性”
>
> 但考试里有时会跟“关节到处疼”混在鉴别中
>
> 题干抓：**腹泻/尿道炎后，关节炎 + 结膜炎 + 尿道炎**
>
> 所以它要记，但不是“游走性关节炎”最经典代表。

**Gonorrhea, chlamydia后续管理 (Partner Management)**

> **告知患者：** 在治疗完成前（通常 7 天）禁止性行为。
>
> **伴侣治疗 (EPT)：** 必须通知并治疗过去 60 天内的性伴侣，即使他们没有症状。
>
> **进一步筛查：** 既然有不洁性行为，应同时筛查 **HIV、梅毒和乙肝**。

**避坑指南：** 看到**年轻男性**有尿痛，第一反应应该是 **STI** 而非普通 UTI。普通大肠杆菌感染在无解剖畸形的年轻男性中极少见。

 **Fitz-Hugh-Curtis Syndrome：** 如果这个病人在尿痛的同时伴有右上腹痛，要想到衣原体可能引起的肝周炎。

 **Partner Treatment：** 一旦 NAAT 阳性，不仅要治疗患者，还必须通知并治疗其性伴侣，以防止乒乓感染（Ping-pong infection）。

替硝唑 (Tinidazole)： 属于硝基咪唑类 。其原理是在寄生虫体内被还原成活性自由基，直接破坏其 DNA 螺旋结构或抑制其合成，导致寄生虫死亡 。相比甲硝唑，其半衰期更长，通常只需单次给药 。

尼他索隆 (Nitazoxanide)： 是一种广谱抗寄生虫药，主要通过抑制寄生虫能量代谢中的关键酶（丙酮酸：铁氧还蛋白氧化还原酶，PFOR），使滋养体无法获得能量

巴龙霉素 (Paromomycin)，属于氨基糖苷类，不经肠道吸收。它只在肠腔内杀虫，不进入系统血液循环，对胎儿更安全。

**<u>Giardia治疗后的“假复发”</u>**

如果患者完成治疗后腹泻改善，但摄入奶制品后再次出现腹胀和水样泻，这通常不是药物失败，而是：**<span class="mark"><u>继发性乳糖不耐受</u></span>**： 由于贾第鞭毛虫破坏了小肠刷状缘，导致乳糖酶暂时性缺失 。

处理： 这种情况下不需要再次使用抗寄生虫药物，而是**建议暂时禁食奶制品，等待肠粘膜完全修复** 。

| 机制类型 | 常见诱因 | 临床特征 |
|:---|:---|:---|
| 抗生素相关性腹泻 (AAD) | 几乎所有广谱抗生素（如阿莫西林） | 急性、水样便；通常在服药期间发生 。 |
| 艰难梭菌感染 (CDI) | 克林霉素、氟喹诺酮、广谱青霉素 | 黏液便/血便、发热、白细胞显著升高、腹痛剧烈 |

乙肝血清学5 个核心标志：

**<u>HBsAg（surface antigen，表面抗原）</u>**

表示：体内现在有病毒。也就是“**现在感染着/有传染性**”。但打完疫苗后短期内也可能一过性 HBsAg 阳性，所以要结合时间和其他指标看。

**<u>anti-HBs（surface antibody，表面抗体）</u>**

表示：**有保护性免疫**，是中和抗体。来源有两种：

> 打疫苗后产生
>
> 自然感染后恢复产生

**<u>anti-HBc（core antibody，核心抗体）</u>**

表示：**曾经接触过真正的 HBV 病毒**。

> 疫苗不会让你产生 core antibody
>
> 所以 anti-HBc 阳性 = 过去或现在感染过 HBV。而且 total anti-HBc 通常会持续很久，常可终生存在。

**<u>IgM anti-HBc</u>**

表示：**近期感染 / 急性感染**。是急性期最有价值的 core antibody。

**<u>HBeAg / anti-HBe</u>**

这是“复制活跃度/传染性”的线索：

> HBeAg 阳性 → 通常提示 **病毒复制活跃、传染性高**
>
> anti-HBe 阳性 → 通常提示复制下降、传染性较前降低，但不代表病毒没了。

**Serology pearls**

> **<span class="mark">Core antibody only = Window period</span>**

所谓 window period，就是：

> HBsAg 已经下降消失
>
> anti-HBs 还没来得及升起来

这段时间最能抓到的标志就是：anti-HBc，尤其 **IgM anti-HBc**

**Window 里“表面都没了，只剩核心”**。也就是：

> **没 surface antigen**
>
> **没 surface antibody**
>
> **只剩 core antibody**
>
> **<span class="mark">Core antibody + surface antibody = Recovered</span>**

**Core = 真感染过；Surface antibody = 现在恢复有保护**

两个一起出现，就是：**Recovered after natural infection**

> **<span class="mark">Surface antibody only = Vaccinated</span>**
>
> **<span class="mark">E antigen = High infectivity</span>**

**E = Easy to spread**

> **<span class="mark">E antibody = Lower infectivity but still infected</span>**

**Everything positive except surface antibody**

> **HBsAg 阳性**
>
> **anti-HBc 阳性**
>
> anti-HBs 阴性

这意味着：现在仍是乙肝感染中。因为只要 HBsAg 还在，就说明病毒还在。

然后再靠 IgM vs IgG/total anti-HBc 区分急性和慢性：

**表面抗原还在 = 还没好**

> **IgM → 刚感染，急性**
>
> **没有 IgM（以 IgG/total anti-HBc 为主） → 变慢性**
>
> **Acute:**

**IgM**

> **Chronic:**

**IgG**

> **Treat:**

**Chronic hepatitis B**

总结：

1\. Surface Ag = Sick now；HBsAg = 现在体内有病毒

2\. Surface Ab = Safe；anti-HBs = 有保护

3\. Core Ab = Contacted real virus；anti-HBc = 真正感染过，不是疫苗

4\. E Ag = easy to spread

| 组合                  | 解释                       |
|:----------------------|:---------------------------|
| HBsAg +               | 现在感染                   |
| anti-HBs only         | 打疫苗                     |
| anti-HBc + anti-HBs + | 既往感染后恢复             |
| anti-HBc only         | window period 最经典       |
| HBeAg +               | 高传染性                   |
| anti-HBe +            | 传染性下降，但仍可能感染中 |

**2️⃣ Bilateral / interstitial pattern（双侧弥漫，非典型 CAP）**

| **项目** | **记忆点** |
|:---|:---|
| **Etiology** | **Mycoplasma pneumoniae, Chlamydia pneumoniae, 各种病毒** |
| **Clinical** | **起病 缓慢：发热、乏力、头痛、咽痛；长期干咳；儿童常 还能正常活动；体检：bilateral crackles 或 wheeze** |
| **Treatment** | **Macrolide（如 azithromycin）** |

**👉 关键词：\
“双侧+慢性小咳嗽+还能玩 → Mycoplasma/Chlamydia → Azithro。”**

**四、和前面成人用药原则合并记忆的小总表**

> **成人 CAP/HAP 用药：**

**門診健康：macrolide / doxycycline**

**門診有共病：β-lactam + macrolide 或 resp FQ**

**住院非 ICU：resp FQ 或 ceftriaxone + macrolide**

**ICU：β-lactam + macrolide / FQ**

**HAP：Vanc + Zosyn**

> **特殊影像 & 人群：**

**Cavity → TB / fungus / abscess**

**Abscess → Clinda ± I&D**

**HIV + atypical → PCP：BAL + TMP-SMX + steroids**

**儿科 lobar vs bilateral：S. pneumo → Amox；Mycoplasma/Chlamydia → Macrolide**

**🎯 USMLE & 临床秒选逻辑**

**1️⃣ amoxicillin-clavulanate（amox-clav）**

**关键词：门诊、口服、混合感染**

> 门诊 **CAP + comorbidities**：→ **amox-clav + macrolide**
>
> 急性鼻窦炎、中耳炎、牙科/牙龈感染
>
> 人咬伤 / 狗咬伤（口腔厌氧 + 皮肤 G(+)）

**2️⃣ ampicillin-sulbactam（amp-sulbactam, Unasyn）**

**关键词：住院 IV + aspiration + 口腔/腹盆腔混合感染**

> **Aspiration pneumonia / lung abscess**（住院、无 Pseudomonas 风险）
>
> 轻中度腹腔、盆腔、妇科感染
>
> Ludwig’s angina、深部颌面部感染
>
> 中度糖足 / 皮软组织感染（不太像 ICU 级别）

“hospitalized, aspiration event, poor dentition, RLL consolidation, afebrile or low-grade fever, foul-smelling sputum”→ 很典型 **aspiration pneumonia** → **ampicillin-sulbactam** 或 **ceftriaxone + metronidazole**。

**3️⃣ piperacillin-tazobactam（pip-tazo, Zosyn）**

**关键词：ICU、重症、Pseudomonas、HAP/VAP**

> **HAP / VAP**：→ 标准搭档：**pip-tazo + vancomycin**（抗绿脓 + 抗 MRSA）
>
> 重度腹腔感染、胆道感染、复杂腹膜炎
>
> Neutropenic fever 的 G(−) 一线
>
> 糖足 / 坏死性筋膜炎的组合治疗之一（再配 clindamycin, vanc 等）

题干关键词：“ICU, ventilated patient, new infiltrate after 5 days of intubation, fever, purulent secretions”→ VAP →✅ **vancomycin + piperacillin-tazobactam**

“septic shock, hypotension, intra-abdominal source suspected”→ 最常见经验性：**pip-tazo ± vanc ± fluconazole**

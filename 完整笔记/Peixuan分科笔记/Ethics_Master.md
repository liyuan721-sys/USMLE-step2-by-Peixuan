## ⭐ 本文件独有的高价值内容(个人深度笔记精华)

速查表里**没有**或**很简略**,但本文件给出了详细历史、法律依据和 vignette 解析的考点:

1. **Schloendorff v. Society of New York Hospital(1914)** — 确立"未经同意治疗 = 殴打罪(assault and battery)"的判例,是 informed consent 的法律根基
2. **Tarasoff 案** — 治疗师对潜在受害者的**警告义务(duty to warn)**
3. **Malpractice 4 要素** — Duty / Breach / Damage / Causation,缺一不可;**No harm = no malpractice**
4. **Doctrine of Double Effect** — 缓痛意图下,即使加速死亡也允许(吗啡 vs 安乐死的关键区别)
5. **Brain death = death** — 法律和伦理上的等同性
6. **Organ donation 三原则** — 自愿、不可买卖、家属有最终决定权
7. **IRB 在急诊研究中的特殊地位 — Exception from Informed Consent (EFIC)** — 21 CFR 50.24,**研究合法性 vs 代理授权的优先级**
8. **"研究 vs 救命"的判题口诀** — 题目提到"研究/评估/试验"→ IRB;"昏迷大出血找不到家属"→ 直接救命

## 一、4 大伦理原则

| 原则 | 含义 | CK 倾向 |
|---|---|---|
| **Autonomy(自主)** | 病人对自己身体有完全控制权 | **几乎永远胜出** |
| **Beneficence(行善)** | 为病人最大利益行事 | 次要 |
| **Non-maleficence(不伤害)** | 不造成伤害(Do no harm) | 次要 |
| **Justice(公正)** | 公平分配资源 | 罕见考 |

> **CK 黄金法则:** 选项里出现"**尊重病人意愿**" — **大概率是答案**。

### Schloendorff v. Society of New York Hospital(1914)⭐

- **里程碑判例**,确立病人自主权的法律基础
- 核心论断:**"未经病人同意进行治疗 = 殴打罪(assault and battery)"**
- 这是现代 informed consent 制度的法律根基

## 二、Capacity vs Competence

| 概念 | 谁判断 | 性质 |
|---|---|---|
| **Capacity** | **主管医生**(任何医生都可) | **临床判断** |
| **Competence** | **法官** | **法律概念**,法庭裁定 |

### Capacity 4 要素评估法 ⚠️补充

1. 病人能**理解(understand)** 信息
2. 能**表达(express)** 选择
3. 能**理性推理(reason)**
4. 能**理解后果(appreciate consequences)** — 最容易被忽略的一条

### 关键判断

- **Suicidal patients lack capacity**(自杀念头的患者无 capacity)
- **看似"非理性"的决定 ≠ 没 capacity**
  - 例:**Jehovah's Witness 拒输血**、**老人拒绝化疗** — 只要符合 4 要素就**尊重**

### 🔥 CK 高频陷阱(Capacity)

- **陷阱 1:** 病人选了你觉得"不对"的方案 → 不要立刻判 incapacity,**先评估 4 要素**
- **陷阱 2:** Capacity 评估默认由**主管医生**做,**不需要精神科**(精神科只在复杂 case 介入)

## 三、Informed Consent(知情同意)

### 4 要素

1. **Capacity**(有判断能力)
2. **Disclosure**(医生告知信息:**风险、获益、替代方案**)
3. **Understanding**(病人理解,用大白话)
4. **Voluntariness**(自愿,无胁迫)

### 不需要 informed consent 的例外

| 例外 | 说明 |
|---|---|
| **Emergency**(紧急情况) | 病人无意识或生命危急时,**implied consent** |
| **Therapeutic privilege** ⚠️补充 | 极罕见,告知会严重伤害病人 — **CK 几乎不选这个** |
| **病人 waive** | 病人主动放弃知情权(可指定家属代收信息) |
| **Public health emergency** ⚠️补充 | 强制隔离/治疗某些传染病 |

### 特殊要求

- 必须包含 **风险 + 获益 + 替代方案**
- 必须是**病人能理解**的语言
- 病人主动询问时,**医生必须披露自己的经验水平**

### 🔥 CK 高频陷阱(Informed Consent)

- **陷阱 1:** 家属说"不要告诉病人诊断" → **拒绝**,病人有知情权(除非病人主动放弃)
- **陷阱 2:** 病人主动说"我不想知道" → **尊重**,可指定家属代收信息
- **陷阱 3:** 急诊室昏迷大出血,家属联系不到 → **直接救命**(implied consent),不需要等任何审批

## 四、Surrogate Decision-Making(代理决策)⚠️补充

### 决策顺序(由高到低)

当病人**没有 capacity 且无 advance directive** 时,按此顺序找代理人:

1. **配偶(Spouse)**
2. **成年子女(Adult children)**
3. **父母(Parents)**
4. **成年兄弟姐妹(Adult siblings)**
5. **其他亲属(Other relatives)**

### 决策标准(由高到低)

| 标准 | 优先级 | 含义 |
|---|---|---|
| **Substituted judgment**(替代判断) | **优先** | 病人有 capacity 时**会怎么选**?基于其价值观 |
| **Best interest**(最佳利益) | 次之 | 没有依据时,选**对病人最好的** |

### Advance Directives(预先指示)

| 类型 | 含义 |
|---|---|
| **Living will** | 提前写明特定情境下的医疗意愿 |
| **Health-care proxy / DPOA** | 指定一个人**替自己做决定**;**proxy 表达的是病人的意愿,不是自己的** |
| **DNR(Do Not Resuscitate)** | **只**意味着不复苏;**其他治疗继续**(常考陷阱) |

### 决策冲突标准答案

| 场景 | 答案 |
|---|---|
| 病人有 capacity,**家属反对** | **病人意愿优先** |
| 病人无 capacity,**有 advance directive** | **按 directive 执行** |
| 病人无 capacity,**无 directive,有家属** | **按 surrogate 顺序** |
| 病人无 capacity,**无 directive,无家属** | 医生 + 伦理委员会按 best interest |

## 五、Minors(未成年人同意)

### 一般规则

- Minor 一般医疗 → 需要**父母或监护人同意**
- **Emancipated minor**(已婚 / 参军 / 经济独立 / 法院裁定 / 有孩子) → **自己全权决定**

### 部分解放(Partial Emancipation)— 不需要家长同意的领域

- **STDs / STI 检查与治疗**
- **避孕(Contraception)** — 所有州
- **怀孕相关护理(Pregnancy care)**
- **物质成瘾治疗(Substance abuse)** — 多数州
- **精神健康** — 部分州

### 特殊情境

| 场景 | 标准答案 |
|---|---|
| Minor 紧急情况 | **立即治疗**,事后通知家长 |
| **家长拒绝救命治疗**(如输血给孩子) | **法院介入,治疗** |
| 家长拒绝**非紧急但有益**治疗(如疫苗) | **教育沟通**,通常尊重家长 |
| Minor 堕胎 | 因州而异,CK 选"按州法律" |
| 14 岁少女要求避孕,母亲反对 | **给避孕药,保密** |

### 🔥 CK 高频陷阱(Minors)

- **陷阱 1:** 父母替孩子拒绝救命治疗(如 JW 拒输血给婴儿)→ **法院介入,强制输血**
- **陷阱 2:** Minor 来要避孕药/STI 检查 → **不需要家长同意,且必须保密**
- **陷阱 3:** Minor 已婚或有孩子 → **emancipated**,自己决定

## 六、Confidentiality 与强制上报

### 一般原则

**未经病人同意,不得披露** — 但有以下例外:

### 强制上报(必报,不需病人同意)

| 类别 | 备注 |
|---|---|
| **儿童虐待(Child abuse)** | 强制 |
| **老人虐待(Elder abuse)** | 多数州强制 |
| **传染病(Reportable diseases)** | 公共卫生上报(TB、某些 STI) |
| **枪伤(Gunshot wounds)** | 强制 |
| **法院命令(Court order)** | 强制 |

### 不强制(需谨慎处理)

| 场景 | 标准答案 |
|---|---|
| 怀疑**家暴(成年伴侣)** | **不强制**,提供资源,尊重病人选择(因州而异) |
| **HIV+ 不告诉伴侣** ⚠️补充 | **两步走:** ① 先**劝病人告知** → ② 不行**通过 public health 通知** |

### 危险行为类(必须介入)

| 场景 | 标准答案 |
|---|---|
| **病人威胁要伤害特定他人**(Tarasoff) | **警告潜在受害者 + 通知警方** |
| **病人有自杀风险** | **不可让其离开**,精神科评估,必要时 **involuntary hold**（详见 [[mistakes/uworld-mistakes#^Q15895]] — 青少年 passive SI 实战场景：未成年人 SI 时打破保密通知父母 + duty to protect）|
| **司机有癫痫 / 视力问题不能开车** ⚠️补充 | **劝其停止 + 报告 DMV**(因州而异,多数选报告) |

### 同事问题(Impaired Physician)⚠️补充

| 场景 | 标准答案 |
|---|---|
| **同事酗酒上班** | **报告主管 / 医院** |
| **同事偷药** | **报告** |

### Tarasoff Case ⭐

- 加州判例,确立治疗师对**可识别的潜在受害者** 的**警告义务(duty to warn)**
- 核心:**保密原则不能凌驾于他人生命安全之上**

### 🔥 CK 高频陷阱(Confidentiality)

- **陷阱 1:** 配偶虐待你成年女病人,你立刻报警 → **错!** 成年家暴**非强制上报**,提供资源,尊重选择
- **陷阱 2:** HIV+ 不告诉伴侣,你立刻打电话给伴侣 → **错!** 必须先**劝病人自己告知**
- **陷阱 3:** 同事酗酒,你心里有数但不报 → **错!** **Impaired physician 必须上报**

## 七、Disclosure of Medical Errors

| 场景 | 标准答案 |
|---|---|
| **自己犯了医疗错误** | **主动告知病人 + 上报系统** |
| **同事犯错** | **先和同事本人谈** → 不行再上报 |
| 病人问"是不是出错了" | **诚实告知** |
| 错误被掩盖了想披露 | **披露**,无论后果 |

### 🔥 CK 高频陷阱(Disclosure)

- **陷阱 1:** 错误未造成 harm,觉得"没必要说" → **错!** **诚实披露**是绝对原则,与后果无关
- **陷阱 2:** 同事犯错你立刻去找院长 → **错!** **先与同事本人沟通**

## 八、End-of-Life Care

### 撤治疗 vs 不上治疗

> **法律和伦理上完全等同** — 撤呼吸机 = 不上呼吸机 = 撤水/食(在适当情况下)

### DNR 关键陷阱

- **DNR 只意味着不复苏(CPR)**
- **其他所有治疗继续**(抗生素、营养、止痛、手术等)

### Doctrine of Double Effect ⭐

- **核心:** 给吗啡缓痛**可能加速死亡**,只要**意图是缓痛**就**允许**
- 区分:
  - 给吗啡 **意图缓痛**(可能加速死亡)→ ✅ 允许
  - 给大剂量吗啡 **意图致死** → ❌ 安乐死,违法

### Physician-Assisted Suicide(PAS) vs Euthanasia

| 行为 | 谁执行 | 合法性 |
|---|---|---|
| **PAS** | **医生提供手段**(如开处方),**病人自己用** | 仅在合法州(俄勒冈等);**CK 默认拒绝** |
| **Euthanasia** | **医生亲自执行**(如注射) | **大部分州非法**,即使病人要求也**拒绝** |

### Brain Death ⭐

- **Brain death = death**(法律和伦理上等同)
- 一旦确诊 brain death,即可撤治疗(无需家属同意撤治),也可摘取器官

### 撤水/食(无 capacity 的植物人)

- 按 **advance directive 或 surrogate** 决定
- 与撤呼吸机伦理上等同

### 🔥 CK 高频陷阱(End-of-Life)

- **陷阱 1:** "撤呼吸机算杀人" → **错!** 撤治疗 = 不上治疗,**伦理等同**
- **陷阱 2:** DNR 病人感染了肺炎,觉得"DNR 就不治了" → **错!** **DNR ≠ 不治疗**,该用抗生素就用
- **陷阱 3:** 给吗啡缓痛怕"加速死亡" → 只要**意图是缓痛** = 双重效应原则,**可以做**
- **陷阱 4:** 病人要求安乐死,你帮忙 → **错!** 默认**拒绝**(除非 PAS 合法州 + 严格条件)

## 九、Malpractice(医疗事故)⭐

### 4 要素 — 必须**全部满足**

1. **Duty(义务)** — 医患关系成立,医生有照护义务
2. **Breach(违反)** — 医生未达到 standard of care
3. **Damage(损害)** — 病人受到实际伤害
4. **Causation(因果)** — 损害**直接由**违反义务造成

### 关键认知

> **No harm = No malpractice**
> 即使医生明显违规,只要**没造成损害**,就**不构成 malpractice**(但可能构成其他职业过失追责)。

## 十一、Communication(沟通题万能套路)

### 核心原则:**先 acknowledge 情绪,再给信息**

### SPIKES 法 — 告知坏消息 ⚠️补充

| 字母 | 步骤 | 做什么 |
|---|---|---|
| **S** — Setting | 设置环境 | 私密空间、坐下、关门 |
| **P** — Perception | 了解病人认知 | "您对自己情况了解多少?" |
| **I** — Invitation | 询问意愿 | "您想知道详情吗?" |
| **K** — Knowledge | 告知信息 | 用大白话,**先 warning shot**("我有不太好的消息要告诉您") |
| **E** — Emotion | 回应情绪 | **acknowledge,不打断** |
| **S** — Strategy/Summary | 制定计划 | 总结 + 下一步 |

### "正确选项"语言模式 ⚠️补充

CK communication 题的**正确答案通常包含**:

- "I can see this is very difficult for you..."
- "Tell me more about your concerns."
- "What worries you most?"
- "How are you coping with this?"
- "Let's talk about what this means for you."

### "错误选项"语言模式 ⚠️补充

- "Don't worry, everything will be fine." ❌(空洞 reassurance)
- "You shouldn't feel that way." ❌(否定情绪)
- "Let me refer you to..." ❌(太早转介)
- "You need to..." ❌(命令式)
- 立刻给一堆**医学信息**(没先回应情绪) ❌

### 高频沟通场景表

| 场景 | 标准答案 |
|---|---|
| 病人**哭 / 愤怒 / 恐惧** | **先 acknowledge 情绪**,不立刻给信息 |
| **"Don't worry, it'll be fine"** 这种选项 | **永远是错的** |
| **病人不会英语** | **用专业翻译**,**不用家属(尤其不能用孩子)** |
| 病人听不懂医学术语 | 用大白话重说,**不要让家属代翻译** |
| **病人不依从治疗** | **探究原因**,不直接换医生/责怪 |
| 病人要求做不必要检查 | **教育解释**,不直接拒绝或满足 |
| 病人否认明显问题(如酗酒) | **共情 + 开放式问题**,不对抗 |
| **病人想换医生** | **尊重**,提供帮助 |

### 🔥 CK 高频陷阱(Communication)

- **陷阱 1:** 选项里有"reassurance"(don't worry / everything will be fine) → **永远是错的**
- **陷阱 2:** 选项让你立刻"refer to specialist" → **太早转介**,通常错
- **陷阱 3:** 病人不会英语,选"让 ta 8 岁儿子翻译" → **绝对错**!**必须用专业翻译**
- **陷阱 4:** 病人哭着进来,你先讲诊断 → **错!** **先 acknowledge 情绪**

## 十二、Gifts and Boundaries

| 场景 | 标准答案 |
|---|---|
| 病人送**小礼物**(贺卡、烘焙) | **可接受** |
| 病人送**贵重礼物** | **拒绝** |
| 病人邀请约会(治疗中) | **拒绝**,继续治疗 |
| 病人邀请约会(治疗结束后) | **拒绝**(**精神科病人永远禁止**) |
| **给家人朋友看病** | **避免**,除非紧急 |
| **Pharma rep 请吃饭 / 送礼** ⚠️补充 | **拒绝** |
| 病人提供**商业机会**(投资等) | **拒绝** |

## 十三、Cultural and Religious Considerations

| 场景 | 标准答案 |
|---|---|
| **成年 Jehovah's Witness 拒输血** | **尊重**(有 capacity) |
| **父母 JW 拒给孩子输血** | **法院介入,输血** |
| 病人坚持**传统/替代医学** | **不评判,提供医学信息**,共同决策 |
| **文化原因家属要求隐瞒诊断** | 先**问病人**愿不愿意知道,按病人意愿 |
| **男医生 vs 女病人(伊斯兰文化)** ⚠️补充 | **尊重偏好**,如可能换医生 |

### 🔥 CK 高频陷阱(文化)

- **陷阱 1:** "他们文化就这样,所以听家属的不告诉病人" → **错!** 必须**先问病人**愿不愿意知道
- **陷阱 2:** 成年 JW 自己拒输血 vs 父母替孩子拒 — **完全不同处理**(成人尊重,孩子法院)

## 十四、研究伦理 — IRB & EFIC

> **本节是 Biostats / Ethics 交叉考点;研究设计本身见 [[Biostats_Master#五研究设计6-2-种必须秒认]]**

### IRB(Institutional Review Board)

- **任何研究**(包括第一个受试者)**之前**,研究方案**必须先获 IRB 批准**
- 没 IRB 批准,任何形式的知情同意**法律上都无效**
- **研究合法性 vs 代理授权的优先级:**
  - **IRB 批准 = 研究的"准入证"**(行政审批)
  - **家属/病人同意 = 执行环节**(临床执行)
  - 必须先有 IRB,才能谈知情同意

### Exception from Informed Consent(EFIC)— 急诊研究 ⭐

- 法规依据:**FDA 21 CFR 50.24**
- 适用于无法获得即时同意的**急诊危重症研究**(家属也不在现场,情况紧急)
- **由 IRB 决定**:
  - "家属同意是否足够"
  - "是否可以先救人后补签(deferred consent)"
- **不是医生个人决定**

### 一般研究伦理规则

| 场景 | 标准答案 |
|---|---|
| **研究招募** | 需要 **IRB 批准 + 知情同意** |
| **弱势群体**(囚犯、儿童、孕妇)研究 | **额外保护**(extra protection) |
| 治疗中发现新有效药 | **告知所有受试者**,可能终止试验 |
| **Placebo 控制** ⚠️补充 | **仅当无现有有效治疗时**可用 |

### 🔥 CK 高频陷阱(研究伦理)— 关键判题口诀 ⭐

- **判题口诀 1 — 研究 vs 救命:**
  - 题目提到"研究 / 评估效果 / 新药试验" → 第一步永远是 **"IRB Approval"**
  - 题目问"昏迷大出血找不到家属下一步" → **"Proceed with emergency treatment"**(直接救命,不等 IRB 不等法院)

- **判题口诀 2 — 选项识别:**
  - 选项里出现 **"Apply to the IRB"** 或 **"Consult the Ethics Committee"**,且研究尚未启动 → **优先级最高**

- **陷阱:** 急诊研究找家属同意 → **错!** 家属同意能解决"代理授权",但解决不了"研究合法性",只有 **IRB 批准** 才是法定第一步

## 十五、考试套路 + 终极心法

### 题型 1:Capacity 判断

- 病人做"非理性"决定 → 检查是否符合 **capacity 4 要素**
- 符合就**尊重**,不符合才介入

### 题型 2:Surrogate 决策

- 按顺序找:**配偶 → 成年子女 → 父母 → 兄弟姐妹**
- **Substituted judgment 优先,best interest 次之**

### 题型 3:Minor 同意

- 涉及**避孕 / STI / 孕产 / 成瘾 / 精神** → 通常**不需家长**
- 一般医疗 → 需家长(emancipated 例外)
- 紧急 → 立即治疗

### 题型 4:Disclosure / 强制上报

- **虐待儿童 / 老人** → 强制
- **传染病** → 强制
- **威胁伤人(具体目标)** → Tarasoff(警告 + 报警)
- **自伤 / 自杀** → 紧急介入
- **家暴(成年伴侣)** → **不强制**
- **HIV 不告诉伴侣** → 两步走

### 题型 5:Communication

- **永远先 acknowledge feeling + open-ended question**
- 永远不是 reassurance、转介、命令

### 题型 6:End-of-life

- 撤治疗 = 不上治疗(伦理等同)
- 安乐死 / PAS → 默认拒绝
- 双重效应 → 可以(意图是缓痛)
- **DNR ≠ 不治疗**

### 题型 7:研究 vs 救命

- "研究 / 试验 / 评估" → **IRB 优先**
- "急诊救命" → **直接治疗**

## 十六、终极心法(6 条搞定 90% 的题)

CK 的 Ethics:

1. **病人有 capacity → 听病人的**
2. **病人没 capacity → 找 directive,再找 surrogate**
3. **保密原则 → 除非:虐待、传染、威胁他人、自伤、impaired physician**
4. **沟通题 → 先承认情绪,再说话**
5. **错误 → 诚实披露**
6. **拿不准 → 选最尊重病人 + 最像"开放式对话" + 不空洞 reassure 的选项**

> **90% 的题用上述 6 条就能选出来。**
> 剩下 10% 是研究伦理 / 文化冲突 / minor 边界 / impaired physician — 见上面对应章节。

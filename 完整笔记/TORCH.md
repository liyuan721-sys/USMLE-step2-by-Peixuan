**【TORCH】（会流产/IUFD + 先天异常的核心组）**

**T — Toxoplasma gondii**

> **题眼**：生肉/未熟肉、猫粪/园艺土；孕期原发感染
>
> **机制**：经胎盘 → 胎儿中枢/眼受累，可流产/IUFD
>
> **三联征**：**脉络膜视网膜炎 + 脑积水 + 弥漫颅内钙化**
>
> **检查**：母体/新生儿 **IgM/IgG**，PCR（羊水）

**O — Others（最常考的“其他”）**

下面单列在“其他导致IUFD”里（如：**梅毒、VZV、HBV/HIV、Listeria、Parvo B19**等）。

**R — Rubella（风疹）**

> **题眼**：未接种疫苗、孕早期脸开始的**斑丘疹** + **耳后淋巴结** + 关节痛
>
> **机制**：胎盘感染 → 流产/IUFD + 器官发育异常
>
> 致畸风险**与孕周强相关**：**孕早期（尤其\<12周）最高**；**\>16周后CRS风险很低（安慰，对症）**
>
> **先天风疹**：**PDA（或肺动脉狭窄）+ 白内障 + 感音神经性耳聋**（± “blueberry muffin”）
>
> **检查**：孕妇血清学 IgM，胎儿评估
>
> **风疹（Rubella）典型三联题眼**
>
> **瘙痒性斑丘疹**：**从脸开始向下蔓延**（face → down）
>
> **掌跖通常不累及**（spares palms/soles）
>
> **耳后/枕后淋巴结肿大**（postauricular/posterior cervical LAD）
>
> **关节痛/关节炎**（成人女性常见：polyarthritis）

**C — CMV（巨细胞病毒）**

> **题眼**：托儿所接触、类似单核细胞增多症；多数母体轻症
>
> **机制**：经胎盘 → 胎儿脑损伤，严重可IUFD
>
> **典型**：**感音神经性听力损失**（最常考）、**室周钙化**、癫痫、智力障碍、IUGR、肝脾大
>
> **检查**：PCR（羊水/尿液），新生儿尿/唾液PCR

**H — HSV（单纯疱疹）**

> **题眼**：围产期暴露（分娩时有活动性病灶最关键）
>
> **机制**：多为**分娩时接触传播** → 新生儿播散感染/脑炎；IUFD不是最常考但可发生于重症/上行感染
>
> **要点**：孕晚期复发史 → **36周起抑制治疗（acyclovir/valacyclovir）**；分娩时有病灶→剖宫产
>
> **检查**：PCR（新生儿/母体病灶）

**二、非TORCH但特别爱考的 IUFD 感染（务必背）**

**Parvovirus B19（第五病）**

> **题眼**：接触学龄儿童/“slapped cheek”；孕妇可轻症
>
> **机制**：感染红系前体 → **重度胎儿贫血** → **hydrops fetalis** → IUFD
>
> **检查**：母体血清学、**胎儿MCA多普勒**评估贫血（考试高频）

**Listeria monocytogenes**

> **题眼**：孕妇“**水样腹泻/发热/肌痛**” + 冷藏熟食(deli meats)、未巴氏奶/软奶酪
>
> **机制**：食源性 → 菌血症 → **经胎盘** → 胎盘炎/胎儿败血症 → IUFD/早产
>
> **治疗要点**：**ampicillin（± gentamicin）**；预防饮食回避

**Syphilis（梅毒，Treponema pallidum）**

> **题眼**：无产前保健/高危性史；母体可无症状或有二期皮疹史
>
> **机制**：经胎盘 → 胎儿感染 → **流产/IUFD**、早产、先天梅毒
>
> **检查**：RPR/VDRL（筛查）+ FTA-ABS（确证）
>
> **治疗**：**青霉素**（孕期唯一推荐的确证治疗）

**Varicella（VZV）**

> **题眼**：孕妇水痘；早孕感染风险更大
>
> **机制**：胎儿水痘综合征/流产；围产期感染可致重症新生儿水痘
>
> **典型**：皮肤瘢痕、肢体发育不良、眼/神经异常（IUFD可见但不如上面三项常考）

**三、上行宫内感染（破水/产程相关）→ 可胎死**

**Chorioamnionitis（绒毛膜羊膜炎，多菌群）**

> **题眼**：发热 + 子宫压痛 + 臭羊水 + 母胎心动过速（常PROM/产程中）
>
> **机制**：上行感染 → 胎儿败血症/缺氧 → 早产/胎死
>
> **管理**：**ampicillin + gentamicin + prompt delivery**（剖产再加clinda/metro）

**最快“题眼定位”口诀**

> **腹泻/冷藏熟食 + IUFD** → **Listeria**
>
> **接触小孩/第五病 + 胎儿水肿(hydrops)** → **Parvo B19**
>
> **室周钙化 + 听力损失** → **CMV**
>
> **PDA+白内障+耳聋** → **Rubella**
>
> **弥漫颅内钙化+脑积水+脉络膜视网膜炎** → **Toxo**
>
> **破水/产程发热臭羊水** → **Chorioamnionitis**
>
> **无产检+梅毒筛查阳性** → **Syphilis（Penicillin）**

**会导致宫内死胎（IUFD）的感染总结（按“最爱考”分组）**

**1）胎盘/胎儿直接感染（经典：TORCH + Listeria）TORCH**

> **Listeria monocytogenes**：食源性胃肠炎后 → 经胎盘 → IUFD/早产/新生儿早发败血症
>
> **Parvovirus B19**：感染红系前体 → **严重胎儿贫血** → **hydrops fetalis** → IUFD
>
> **Syphilis（梅毒）**：胎盘感染/胎儿感染 → IUFD、早产、先天梅毒表现
>
> **CMV**：可致胎儿感染、IUGR、胎死（更常考“室周钙化/听力损失”）
>
> **Toxoplasma**：可致流产/死胎（更常考“弓形虫三联征：脉络膜视网膜炎+脑积水+颅内钙化”）
>
> **Rubella**：可致流产/死胎（更常考“先天风疹：白内障/耳聋/PDA”）
>
> **Varicella（VZV）**：早孕可导致胎儿综合征/流产；围产期可严重新生儿感染
>
> **Zika**：可导致严重胎儿脑发育异常与胎死（更偏公共卫生题）

**2）上行感染 → 宫内感染/早产/胎死（常与破水相关）**

> **Chorioamnionitis（多菌群）**：母体发热、子宫压痛、臭羊水、胎儿心动过速 → 可致胎儿败血症/胎死\
> 常见菌：**GBS、E. coli、厌氧菌**等混合

**3）母体严重系统性感染/高热缺氧（间接导致胎死）**

> 严重 **疟疾、败血症、重症流感/肺炎** 等（USMLE偶尔提“高热/缺氧”导致胎儿不良结局）

**速记小表（你刷题用）**

> **腹泻 + 冷藏熟食/未巴氏奶 + IUFD** → **Listeria**
>
> **“小孩第五病接触/老师” + 胎儿水肿** → **Parvo B19**
>
> **无产前保健 + 无痛性溃疡史/皮疹** → **梅毒**
>
> **破水/产程中发热臭羊水** → **Chorioamnionitis（混合菌）**

**宫内感染超声题眼对照表（USMLE高频）**

| **病原** | **母体/暴露题眼** | **产前超声最典型题眼（最有区分度）** | **进一步提示/记忆点** |
|:---|:---|:---|:---|
| **CMV** | daycare幼儿接触；母体轻症/无症状 | **室周钙化（periventricular）** + **脑室扩大** ± **小头** + **FGR**；常伴 **肝脾大/肝内钙化** | 远期：**感音神经性耳聋**（最常考） |
| **Toxoplasma** | 生肉/未熟肉；猫粪/园艺土 | **弥漫颅内钙化（diffuse）** + **脑积水**（三联征核心） | 记忆：Toxo=“**Diffuse** calcifications” |
| **Parvovirus B19** | 接触小孩“第五病” | **胎儿贫血 → hydrops fetalis**：皮下水肿、**腹水/胸水**、心衰表现；可见胎盘水肿 | 关键检查：**MCA Doppler↑**（提示贫血） |
| **Rubella** | 未免疫；孕早期发热出疹 | 超声不如上面典型；可见 **FGR**、心脏结构异常线索 | 新生儿：**PDA/白内障/耳聋** |
| **Syphilis** | 无产检/高危性史；RPR阳性 | **肝脾大**、**胎盘增厚**、**羊水过多**、可 **hydrops**；（有时骨异常） | 影像不如血清学典型；确诊看 **treponemal test** |
| **Listeria** | 冷藏熟食/未巴氏奶；孕妇短暂胃肠炎 | 更偏：**IUFD/早产**；有时提示**胎盘/胎儿弥漫感染**（granulomatosis infantiseptica概念） | 题眼常是“**腹泻后IUFD**” |
| **HSV** | 既往生殖器疱疹 | 产前超声不固定；更关键是分娩时有无病灶 | 考点核心：**36周起抑制治疗**；分娩时病灶→剖 |
| **VZV** | 孕期水痘接触/出疹 | **肢体发育不良**、皮肤瘢痕样改变线索、眼/脑异常 | “先天水痘综合征”更偏综合表现 |
| **Zika** | 流行区旅行/蚊媒 | **严重小头** + **皮质下/灰白质交界处钙化**+皮质发育异常 | 常和“旅行史”绑定 |

> **宫内感染相关眼病**
>
> **1) Toxoplasma gondii**
>
> **典型眼病**：**脉络膜视网膜炎（chorioretinitis）**（最经典）
>
> **题眼**：猫粪/园艺土、生肉；三联征 **脉络膜视网膜炎 + 脑积水 + 弥漫颅内钙化**
>
> **后果**：视力下降、复发性视网膜炎

**2) CMV**

> **典型眼病**：**脉络膜视网膜炎（chorioretinitis）**（也可有），视网膜炎/视网膜瘢痕
>
> **题眼**：daycare幼儿接触；**室周钙化 + 脑室扩大 + FGR**；最常考远期是 **感音神经性耳聋**
>
> **区分点**：CMV的“主打”是听力，但眼也会考到 chorioretinitisCMV的“主打”是听力，但眼也会考到

**3) Rubella（先天风疹）**

> **典型眼病**：**白内障（cataracts）** ± 视网膜病变（“salt-and-pepper retinopathy”）
>
> **题眼**：三联征 **PDA/肺动脉狭窄 + 白内障 + 感音神经性耳聋**；孕早期感染风险最高

**4) HSV（新生儿疱疹）**

> **典型眼病**：**角结膜炎/角膜炎（keratoconjunctivitis, keratitis）**
>
> **题眼**：分娩时接触最关键；皮肤-眼-口(SEM)病变、脑炎；孕36周起抑制治疗；分娩时有活动病灶→剖宫产SEM

**5) Syphilis（先天梅毒）**

> **典型眼病**：**间质性角膜炎（interstitial keratitis）**（晚期表现，Hutchinson三联征之一）
>
> **题眼**：Hutchinson triad：**间质性角膜炎 + Hutchinson牙 + 神经性耳聋**（常在儿童期/青少年出现）

**6) VZV（先天/新生儿水痘）**

> **典型眼病**：**脉络膜视网膜炎、白内障、视神经萎缩**（可考）
>
> **题眼**：孕早期感染→先天水痘综合征（皮肤瘢痕、肢体发育不良、眼/中枢异常）

**7) Zika**

> **典型眼病**：**脉络膜视网膜萎缩/瘢痕、视神经发育不良、黄斑异常**（总体是“视网膜-脉络膜结构异常”）pigmentary retinal mottling
>
> **题眼**：旅行/流行区；**重度小头** + 皮质发育异常；钙化多在**皮质下/灰白质交界**

**一句话速记（考试最常用）**

> **Toxo/CMV：chorioretinitis**（Toxo更“经典三联征”）
>
> **Rubella：cataracts**
>
> **Syphilis：interstitial keratitis**
>
> **HSV：keratitis/keratoconjunctivitis（SEM）**

**10秒“定位秒杀”口诀**

> **室周钙化** + 脑室扩大→ **CMV**（再加：耳聋）
>
> **弥漫钙化 + 脑积水** → **Toxo**
>
> **白内障** → **Rubella**
>
> **角膜炎/皮肤眼口(SEM)** → **HSV**
>
> **重度小头 + 皮质下钙化** → **Zika**
>
> **Hydrops（胸/腹水+水肿）** → 先想 **Parvo B19（贫血）**

**暴露/母体**：daycare幼儿接触；母体常轻症或无症状\
**超声**：**室周钙化 + 脑室扩大/小头 + FGR + 肝脾大/肝内钙化 ± hydrops**\
**新生儿**：**“blueberry muffin”/紫癜、黄疸、肝脾大、小头、脉络膜视网膜炎**\
**远期**：**感音神经性耳聋（最常考）+ 癫痫/发育迟缓**\
**确诊**：胎儿 **羊水PCR**（或新生儿尿/唾液PCR）

# 模块 1：下肢血管缺血

## 1.1 Peripheral Artery Disease (PAD)

### Stem 锁定钩子
- 老人/中年男 + smoker / DM / HTN + 走路时小腿/大腿/臀部疼，**休息缓解**
- **ABI ≤ 0.9** → 诊断
- **ABI > 1.3 或不可压缩** → DM/CKD 钙化血管 → **TBI** (toe-brachial index/趾肱指数)
- Rest pain / ulcer / gangrene → critical limb ischemia
 #### 临床意义

> [!tip] USMLE 钩子 Stem 写："**DM patient with leg pain, ABI = 1.4**" → 别认为 ABI 高 = 没 PAD！ → ABI > 1.3 = **钙化血管假性升高** → **改用 TBI**



### Buerger sign vs Buerger disease（必混淆陷阱）
- **Buerger sign** = 体征（抬高变白，下垂变红）= severe PAD
- **Buerger disease (TAO)** = 独立病 = 年轻男 + 重度吸烟 + 远端肢体缺血 + 节段性血管炎症 → **戒烟唯一有效**

### PAD 治疗优先级 ⭐ USMLE 必考顺序
1. **戒烟** > 一切
2. **Antiplatelet (ASA/clopidogrel) + Statin** = 降 MI/Stroke 死亡率（不是改善走路）
3. **HbA1c < 7%、BP 控制**
4. Claudication 无 rest pain → **Supervised Exercise Therapy** ⭐ 首选
5. 二线：**Cilostazol**（PDE3 抑制剂；**HF 禁用**）
6. Critical limb ischemia / 顽固症状 → **Revascularization**
   - PTA = 短局灶 / 微创 / 易再狭窄
   - Bypass = 长/多灶 / 持久 / 严重弥漫

## 1.2 Leriche Syndrome（主髂闭塞）

### 三联征 ⭐
```
Aortoiliac 闭塞（主动脉分叉处）
        ↓
┌───────┴───────┐
↓               ↓
Hypogastric/    Common iliac
Internal iliac  → 大腿/臀肌缺血
→ 阴茎海绵体    
   供血↓
   
临床三联：
1. Buttock + thigh claudication（双侧）
2. Erectile dysfunction（阳痿）⭐
3. ↓ Femoral pulses（股动脉搏动减弱/消失）
```

### USMLE 锁定句
"老年男性 + 双侧大腿/臀部走路痛 + ED（sildenafil 无效）+ 双侧股动脉摸不到" → Leriche

### Memory Hook
"主动脉分叉处堵车，三条路同时瘫痪"

## 1.3 Vascular vs Neurogenic Claudication ⭐ 金表

| 特征 | Vascular | Neurogenic |
|---|---|---|
| 病因 | 动脉粥样硬化 | 椎管狭窄 |
| 走路加重 | ✅ | ✅ |
| **休息缓解** | **站着就缓解** | **需坐下/弯腰** |
| 站立不动 | 缓解 | ❌ 不缓解 |
| 骑车 | 加重 | ✅ 不加重 |
| Pulse | ↓ | 正常 |
| 影像 | CTA/MRA | MRI 腰椎 |

⭐ **黄金口诀**：站着缓解 = Vascular；弯腰缓解 = Neurogenic

## 1.4 下肢溃疡 3 类鉴别 ⭐

| 特征 | Arterial | Venous | Neuropathic |
|---|---|---|---|
| 位置 | 远端、足趾、外踝、压力点 | **内踝上方**（gaiter） | 足底压力点 |
| 疼痛 | **剧痛** | 轻/中，**抬高缓解** | **无痛** ⭐ |
| 边缘 | 锐利"打孔样" | 不规则浅 | 锐利 + callus |
| 周围 | 冷、苍白、无毛、亮、肌萎缩 | **棕色色素沉着 + 静脉曲张 + 水肿** | 干、callus、Charcot 足 |
| Pulse | ↓ / 无 | 正常 | 正常 |
| 病因 | PAD | CVI | DM 神经病变（大纤维丢失）|

### Venous ulcer 必查 ⭐
**Compression therapy 前必查 ABI**（排除合并 PAD；ABI < 0.5 禁用 compression）

### DM 神经病变深度细节
- **小纤维**（Aδ + C + 自主）：burning pain（早期）、痛温↓、自主功能
- **大纤维**（Aα + Aβ）：振动 / 本体觉↓、踝反射↓、Romberg+、**10g monofilament**
- **Neuropathic ulcer = 大纤维 + 运动（足畸形）+ 自主（皮肤干）三位一体**
- "小纤维先死你先痛，大纤维后死你才烂脚"

# 模块 2：脊髓不全损伤综合征 ⭐

## 2.1 三大束交叉时机（USMLE 必懂解剖）

```
纤维束             在哪里交叉？
─────────────────────────────────
Corticospinal     延髓 (medulla)
(运动)            → 进入脊髓时已经在对侧
                  → 脊髓损伤 = **同侧瘫**
─────────────────────────────────
Dorsal column     延髓 (medulla)
(本体/振动/       → 在脊髓内一路同侧上行
 轻触)            → 脊髓损伤 = **同侧↓**
─────────────────────────────────
Spinothalamic     **脊髓内即刻交叉**
(痛温)            （进入脊髓 1-2 节段就过对侧）
                  → 脊髓损伤 = **对侧↓**
```

⭐ **口诀**："运动、本体感觉在上面交叉，痛温觉在脊髓里就交叉"

**Cuneate vs Gracile**：

- **Gracile** (内侧) = below T6 = 下肢
- **Cuneate** (外侧) = above T6 = 上肢
- Memory: "**Gracile = legs (below T6)**, Cuneate = arms" — 字母 G 在前但管下肢，记反向

## 2.2 4 大不全脊髓损伤综合征对比 ⭐⭐

| 综合征 | 位置 | 临床 | 典型 stem |
|---|---|---|---|
| **Central Cord** | 中央灰质 + 周围 | 上肢瘫 > 下肢、披肩痛温↓、老人过伸伤 | 老人楼梯摔 + 上肢瘫 |
| **Anterior Cord** | 前 2/3（前角+CST+STT）| **双侧** 瘫 + 双侧痛温↓，**本体觉保留** | 主动脉手术 / 屈曲伤 |
| **Brown-Séquard** | 单侧半切 | 同侧运动↓ + 同侧本体↓ + **对侧痛温↓** | 穿透伤 / 刀刺 |
| **Posterior Cord** | 后索 | 双侧本体/振动↓，sensory ataxia | B12 / Tabes / MS |
| **Cauda Equina** | L2 以下神经根 | 不对称下肢瘫 + saddle 麻木 + **早期尿便障碍** + LMN | 中央椎间盘突出 |
| **Conus Medullaris** | S1-S5 脊髓节段 | 对称下肢瘫 + saddle + 早期尿便 + **UMN+LMN 混合** | 脊髓终丝病 |

⚠️ **Brown-Séquard 的对侧 pain/temp 丧失水平比病变低 1-2 节段** 因为 spinothalamic 进脊髓后要爬升 1-2 节段才交叉，所以损伤后**对侧症状起始平面**比 dorsal column 同侧症状平面**低 1-2 节段**。 NBME 考过：让你判断病变到底在 T8 还是 T10。

## 2.3 Central Cord Syndrome（CCS）⭐ 老人创伤后

### 经典 stem
**老人 + 过伸性颈部外伤（楼梯摔/挥鞭样/跌倒撞下巴）+ 上肢瘫 > 下肢 + 手部痛温觉↓**

### 解剖逻辑
- 皮质脊髓束**手部纤维位于内侧**（靠近中央），腿部纤维位于外侧 → 中央水肿先压手
- 脊髓丘脑束在中央**交叉**通过 → 中央水肿就切断交叉纤维 → 披肩样痛温↓

### 影像首选
**MRI cervical spine WITHOUT contrast** ⭐
- 骨 → CT
- 脊髓本身 → MRI no contrast（外伤）
- 脊髓 + 怀疑肿瘤/感染 → MRI with contrast
- 血管 → CTA/MRA

⚠️ **CT with contrast 用于颈椎创伤 = 错** ⭐ 血液和造影剂 Hounsfield 接近 → 掩盖出血

## 2.4 Syringomyelia 5 幕剧 ⭐⭐ 必背

颈髓中央管空洞向外扩展，按距离中央的远近损伤：

幕 1：前白连合（最近）
       ↓
       双侧痛温丢失（披肩样）⭐ "painless burns" 来源
       
幕 2：外侧角（T1-T2）
       ↓
       同侧 Horner 综合征（ptosis + miosis + anhidrosis）
       
幕 3：前角细胞
       ↓
       上肢 LMN 征（萎缩 + 肌力↓ + reflex↓）
       
幕 4：外侧 CST
       ↓
       下肢 UMN 征（hyperreflexia + Babinski +）
       
幕 5：Sacral CST 最外侧（最远）
       ↓
       Sacral sparing 最后破（屁股麻最晚）

```
1️⃣ 前白连合（最先）→ 双侧披肩痛温↓ -- “烫到手不知道”
2️⃣ 外侧角（T1-T2）→ Horner 综合征 ⭐ -- “眼皮垂下来”
3️⃣ 前角 → 上肢 LMN 征（biceps reflex ↓ 不是 ↑）-- “胳膊变软”
4️⃣ 外侧 CST → 下肢 UMN 征 -- “腿变硬，反射强”
5️⃣ Sacral sparing 最后破（CST 最外侧）-- “屁股最后才麻”

```
[!success] **"Painless burns on hands"** = 病人**手被烫了但没感觉痛温** + **触觉正常**
这种"**分离性感觉障碍**"（dissociated sensory loss）= **痛温觉丢失 + 触觉/振动/本体觉保留** = Syringomyelia 的**病理生理招牌**
因为脊髓中央空洞**先损伤前白连合**（spinothalamic 交叉纤维）→ **痛温觉双侧丧失** + 其他感觉束（dorsal column / CST）**完好**


脊髓中央管区域 → 空洞形成 → 逐渐扩大
        ↓
┌─────────────────────────────────────────┐
│ 阶段 1（最先受损）⭐                      │
│ Anterior white commissure（前白连合）    │
│ ← 脊髓丘脑束交叉的纤维在此通过           │
│ → 双侧痛温觉↓（披肩样）                  │
│ → 触觉、本体觉、运动、反射 全部正常       │
└─────────────────────────────────────────┘
        ↓ 空洞扩大
┌─────────────────────────────────────────┐
│ 阶段 2（再扩大）                          │
│ Lateral horn（外侧角，T1-T2）            │
│ ← 一级交感神经元（去往面部的下行交感）    │
│ → 同侧 Horner 综合征（ptosis/miosis/    │
│    anhidrosis）⭐ 本题答案                │
└─────────────────────────────────────────┘
        ↓ 空洞再扩大
┌─────────────────────────────────────────┐
│ 阶段 3（再扩大）                          │
│ Anterior horn（前角细胞）                │
│ → 上肢 LMN 征（hyporeflexia / 萎缩 /    │
│   flaccid paresis）                     │
│ → biceps reflex ↓（不是 ↑！）⭐          │
└─────────────────────────────────────────┘
        ↓ 空洞累及外侧
┌─────────────────────────────────────────┐
│ 阶段 4（晚期）                            │
│ Lateral corticospinal tract             │
│ → 损伤平面以下 UMN 征                    │
│ → 下肢 hyperreflexia + Babinski +       │
└─────────────────────────────────────────┘

### Syringomyelia 病因（USMLE 必背）

| 病因                                 | 特点                             |
| ---------------------------------- | ------------------------------ |
| **Chiari I malformation** ⭐ 最常见原发性 | 小脑扁桃体疝下 → 阻碍 CSF 流动 → 中央管扩张    |
| **创伤后**                            | 数年后形成（"post-traumatic syrinx"） |
| **脊髓肿瘤**（ependymoma）               | 肿瘤分泌液体 / 阻塞                    |
| **蛛网膜炎** / 感染                      | CSF 流动异常                       |
| **脊髓栓系**（tethered cord）            | 与 spina bifida 相关              |

### ⭐ Syringomyelia 关键鉴别（USMLE 高频）

| 鉴别          | Syringomyelia                           | Central Cord Syndrome               |
| ----------- | --------------------------------------- | ----------------------------------- |
| **起病**      | **慢性进展**（数年）                            | **急性**（创伤即刻）                        |
| **典型 stem** | "Painless burns on hands"，年轻人，缓慢加重      | "Fell, weak arms, neck pain"，老人，过伸伤 |
| **病理**      | 中央管周围**空洞**                             | 中央灰质水肿/出血                           |
| **常见关联**    | **Chiari I 畸形** (最常见) / whiplash / 脊髓栓系 | Cervical spondylosis + 过伸伤          |
| **影像**      | MRI 见 T2 高信号"空洞"                        | MRI 见水肿/挫伤                          |
| **运动**      | 缓慢进展的上肢 LMN                             | 急性上肢瘫（> 下肢）                         |

⭐ **黄金鉴别词**：
缓慢进展 + "Painless burns on hands" → Syringomyelia
"**Fell + acute weakness**" + 老人 + 过伸 → Central cord



> [!success] **"分离性感觉障碍" = 某些感觉丢失 + 其他感觉保留**（一个广义术语）
> 
> 两种主要表现形式：
> 
> |类型|谁丢|谁留|代表病|
> |---|---|---|---|
> |**"痛温丢 + 触振留"**|Spinothalamic|Dorsal column|**Syringomyelia + ACS** ⭐ 都是这种|
> |**"触振丢 + 痛温留"**|Dorsal column|Spinothalamic|**SCD / Tabes dorsalis**|
> 
> ⭐ Syringomyelia 和 ACS 的"分离性感觉障碍" = **同一种类型**（都是痛温丢 + 触振留），但**机制完全不同**。

### ⭐ Syringomyelia vs ACS 的"分离性感觉障碍"对比

虽然都是"痛温丢 + 触振留"，但**临床细节完全不同**：

|维度|**Syringomyelia**|**Anterior Cord Syndrome**|
|---|---|---|
|**起病**|**慢性**（数月-年）|**急性**（数分钟-小时）⭐|
|**机制**|中央空洞向外扩|**前脊髓动脉梗死**|
|**诱因**|**Chiari I** / whiplash 数年后 / 肿瘤|**AAA 术后** ⭐ / 主动脉夹层 / 脊髓动脉栓塞 / 低血压 watershed|
|**痛温丢分布**|**披肩样**（节段性 C5-T1）|**病变以下所有节段**（双侧）|
|**触振保留**|✅ Dorsal column 完好|✅ Dorsal column 完好（后脊髓动脉保护）|
|**运动**|**早期保留**（CST 在外侧未波及）|**❌ 双侧瘫痪** ⭐（CST + 前角全废）|
|**症状关键差异**|仅感觉异常 + 慢性|**感觉 + 运动同时** + 急性|
|**影像**|MRI 颈髓见**空洞**（高 T2 信号管）|MRI 见**前 2/3 梗死信号**|
|**关联**|Horner（外侧角扩展）/ 上肢 LMN|Adamkiewicz 动脉损伤（T9-T12 左）|

### ⭐ 临床场景对比

#### Syringomyelia 典型 stem

> "42 岁女 + Chiari I 史 + **6 个月**反复双手无痛烫伤 + **披肩样**痛温↓ + 触振正常 + **肌力 5/5** + 反射正常"

⭐ 关键：**慢性 + 节段性 + 无运动症状**

#### ACS 典型 stem

> "68 岁男 + **AAA 修复术后 24 小时** + **突发双下肢瘫痪** + 双下肢痛温↓ + 触振/位置觉正常 + 尿潴留"

⭐ 关键：**急性 + 病变以下 + 双侧瘫痪**

```
病人有"分离性感觉障碍"
        ↓
哪些感觉丢？哪些留？
        ↓
┌──────────────────────┬──────────────────────┐
**痛温丢 + 触振留**     **触振丢 + 痛温留**
（Spinothalamic 受累）  （Dorsal column 受累）
↓                      ↓
慢性 + 披肩样 +         慢性 + 振动↓ + 位置↓
无运动？                + Romberg+？
↓                      ↓

                       ┌──────────────┐
```

### ⭐ 为什么这个对比是 USMLE 高频陷阱

> [!warning] **USMLE 经常并排考 Syringomyelia 和 ACS** —— 因为两个病：
> 
> 1. **感觉表现一模一样**（痛温丢 + 触振留）
> 2. **机制完全不同**（空洞 vs 梗死）
> 3. **临床急/慢性差异巨大**
> 4. **治疗完全不同**
> 
> 考生容易在感觉描述上**混淆**，但只要看**急/慢性 + 是否有运动症状**就能区分。

### ⭐ 4 大脊髓感觉障碍类型 ⭐⭐⭐ 终极对比

|类型|痛温|触振|代表病|Stem 钩子|
|---|---|---|---|---|
|**痛温丢 + 触振留**（前型）|❌|✅|**Syringomyelia / ACS**|"Painless burns" / "AAA 术后双瘫"|
|**触振丢 + 痛温留**（后型）|✅|❌|**B12 SCD / Tabes dorsalis**|"振动↓ + Romberg+ + 巨幼贫"|
|**同侧触振 + 对侧痛温**（半切）|对侧❌|同侧❌|**Brown-Séquard**|"刀刺 / 脊椎手术"|
|**完全感觉丢失**（横断）|❌|❌|**完全脊髓损伤**|"外伤性脊髓横断"|

⭐ **记忆钩子**：

- **前面的束（spinothalamic + CST）** 一起死 = **前型**（ACS / Syringomyelia）
- **后面的束（dorsal column）** 单独死 = **后型**（B12 / 梅毒）
- **一边的束全死** = **半切**（Brown-Séquard）
- **全部束死** = **横断**

---

### ⭐ 顺手延伸：B12 SCD 的"反向分离"（USMLE 经典）

```
B12 缺乏 → 髓鞘合成异常 → 髓鞘脱失
        ↓
哪些束髓鞘需求高？髓鞘脱失最严重？
        ↓
- **Dorsal column** ⭐ 髓鞘最厚 → 最先废
- **Lateral CST** ⭐ 髓鞘也厚 → 也废
- **Spinocerebellar** → 共济失调
- **但 Spinothalamic（无髓 / 薄髓 C 纤维）→ 保留** ⭐
        ↓
临床：
- 振动 / 位置觉↓ ⭐
- UMN 征（CST）
- 共济失调
- **痛温觉保留** ← 反向分离！
- + 巨幼贫血 / glossitis / 神经精神症状
        ↓
治疗：B12 IM 注射
```

⭐ **反向记忆法**：B12 SCD = "**振动死，痛温活**" = Syringomyelia / ACS 的**完全镜像**

---

### ⭐ Memory Hook ⭐

> [!success] **"分离性感觉障碍 = 看哪个束死"**
> 
> |哪个束死|临床|病|
> |---|---|---|
> |**Spinothalamic 死**|痛温丢 + 触振留|Syringomyelia (中央 内→外) / **ACS (前 2/3 梗死)**|
> |**Dorsal column 死**|触振丢 + 痛温留|B12 SCD / Tabes dorsalis|
> |**两侧/全部死**|看具体位置|Brown-Séquard / 完全横断|

> [!tip] **USMLE 经典 4 道题型**
> 
> 1. 慢性 + 披肩样痛温丢 + Chiari → **Syringomyelia**
> 2. 急性 + 双瘫 + 痛温丢 + AAA 术后 → **ACS**
> 3. 慢性 + 振动丢 + Romberg+ + 巨幼贫 → **B12 SCD**
> 4. 急性 + 单侧 + "刀刺" → **Brown-Séquard**

---

### ⭐ 一个 USMLE 同选项对比题型（你可能会遇到）

> "68 岁男 + 主动脉瘤修复术后 12 小时 + 突发**双下肢瘫痪** + **双下肢痛温觉↓** + 振动位置觉**正常** + 尿潴留"
> 
> "最可能的诊断？"
> 
> A. Syringomyelia B. **Anterior Cord Syndrome** ⭐ 答案 C. Brown-Séquard D. Posterior Cord (SCD) E. Central Cord Syndrome

**为什么 A（Syringomyelia）错**：

- Syringomyelia = **慢性**（数月-年），不是 12 小时
- Syringomyelia 早期无运动症状
- Syringomyelia 是**披肩样**，不是**双下肢**
- Syringomyelia 病因是 Chiari I / whiplash，不是 AAA 术后

**为什么 B（ACS）对**：

- AAA 术后 ⭐ Adamkiewicz 动脉损伤经典诱因
- 急性双瘫 + 痛温丢 + dorsal column 保留 = 完美 ACS

⭐ **题眼 = "AAA 术后" + "急性" + "运动 + 感觉同时"** → ACS（不是 Syringomyelia）

---

### ⭐ 一句话总结

> [!success] **Syringomyelia 和 ACS 都有"痛温丢 + 触振留"，但来源不同**：
> 
> - **Syringomyelia** = **中央空洞向外扩**，先切断 spinothalamic 交叉纤维（前白连合）→ 慢性、披肩样、无运动
> - **ACS** = **前脊髓动脉梗死**，前 2/3 全废 dorsal column 保留 → 急性、病变以下、双侧瘫痪
> 
> ⭐ **共同点**：都是脊髓**前部**结构受累，**后方 dorsal column 保留** ⭐ **区别**：Syringomyelia 是**结构性破坏**（空洞），ACS 是**血管性梗死**

## 2.5 HY 补充 — AMBOSS 没明说但 NBME 爱考 ⭐⭐

> [!tip] 来源
> 这一节是 AMBOSS HY200 没明说、但 NBME 反复埋点的补充考点。重点配合 § 2.2 / § 2.3 的脊髓综合征表使用。

### 2.5.1 Artery of Adamkiewicz 的考点 ⭐

- **最常见起源**：T9-T12 **左侧肋间动脉**
- **供应**：脊髓下 2/3 的**前 2/3**（即前脊髓动脉 (anterior spinal artery, ASA) 的主供血源）
- **高危手术**：胸主动脉瘤修复 (TAA repair) / AAA repair / 主动脉夹层 (aortic dissection)
- **经典 stem 组合**："胸主动脉瘤修复" + "thoracolumbar 平面" → **Adamkiewicz 缺血 → Anterior cord syndrome (ACS)**

| ACS 病因分类 | 机制                                             | 经典 stem                                               |
| -------- | ---------------------------------------------- | ----------------------------------------------------- |
| **血管性**  | Anterior spinal a. 或 Adamkiewicz 闭塞/损伤         | AAA repair、aortic dissection、severe hypotension/shock |
| **外伤性**  | **Vertebral body burst fracture** → 骨片向后挤压脊髓前部 | 高处坠落、车祸、跳水撞击                                          |
| **压迫性**  | 椎管内肿瘤、脓肿（罕见纯 ACS）                              | 慢性病程                                                  |

> [!info]- 📷 课件截图：Anterior Cord Syndrome + Artery of Adamkiewicz 解剖（UW）
> ![[anterior_cord_syndrome_detail_UW.png.png]]

### 2.5.2 Watershed area（分水岭）⭐⭐⭐

- **T4-T8 是脊髓 watershed**（前后循环交界区）
- 全身低灌注（hypotension / severe hemorrhage / shock）→ **优先打击此区**
- **NBME 套路**："心脏术后 / 大出血 / shock 后 → 截瘫" = **watershed infarct** → 双侧 ACS 样表现
- 这也是为什么 ACS 经典 stem 不止"AAA 术后"，**任何低灌注**都能选

### 2.5.3 急性 vs 慢性表现：spinal shock ⭐

> [!warning] NBME 反直觉钩子
> 急性脊髓损伤**早期看不到 UMN 征**，反而像 LMN！

- **急性期（数小时-数天）**：spinal shock → **flaccid paralysis + areflexia**（hyporeflexia）
- **后期（数天-数周）**：才变成 **spastic + hyperreflexia + Babinski (+)**
- ⚠️ Stem 写 "1/5 muscle strength" + "1 小时后" = **还在 spinal shock 期**
  - 此时**不能因为没有 hyperreflexia 就排除 UMN / CST 损伤**
  - 同理 anal wink / bulbocavernosus reflex 也可暂时消失

**HY 三连考**：#薄弱点 
1. ⭐ Adamkiewicz **最常起源左侧 T9-T12 肋间动脉**
2. ⭐ Watershed = **T4-T8**（低血压时最先死的区域）
3. ⭐ Adamkiewicz 损伤 → **下半身 ACS**（thoracolumbar 平面以下）

### 1️⃣ Adamkiewicz 动脉（Artery of Adamkiewicz, AKA）

**解剖锚点**：最常起源于**左侧 T9-T12** 肋间动脉（约 75%），供应脊髓**下 2/3** 的前部。

**Stem 关键词识别**：

- **AAA repair**（腹主动脉瘤修复术）/ **thoracic aortic surgery** / **aortic dissection**
- **Cross-clamping of aorta**（主动脉钳夹）
- 术后 / 术中突发 → "patient cannot move legs"
- 也见于：severe hypotension during cardiac surgery、aortic coarctation repair

**核心症状**（= Anterior Spinal Artery Syndrome 下半身版）：

- ✅ **下肢瘫痪**（flaccid → 后期 spastic paraplegia）
- ✅ **痛温觉丧失**（spinothalamic，双侧、损伤平面以下）
- ✅ **大小便失禁**（autonomic）
- ❌ **本体感觉/振动觉保留**（dorsal column 由后脊髓动脉供应，不受累）
- 上肢、面部完全正常（因为损伤在 thoracolumbar 平面）

**一句话锁定**：

> AAA 术后患者下肢动不了 + 大小便失禁 + 但能感觉到脚趾被动位置 → **Adamkiewicz 损伤 / ASA syndrome**

---

### 2️⃣ Watershed Zone（分水岭区）T4-T8

**解剖锚点**：脊髓 T4-T8 是**两套供血系统的交界**（上方颈段供血 vs 下方 Adamkiewicz 供血），侧支循环最差 → **全身低灌注时最先缺血**。

**Stem 关键词识别**：

- **Prolonged hypotension** / **shock**（任何原因：sepsis、cardiac arrest、massive hemorrhage）
- **Cardiac surgery with prolonged bypass**
- **Resuscitation after cardiac arrest**（"successfully resuscitated but..."）
- 关键词："blood pressure dropped to 60/40 during surgery"

**核心症状**（中胸段脊髓损伤）：

- ✅ **下肢无力 / 瘫痪**（皮质脊髓束）
- ✅ **痛温觉丧失**（spinothalamic，约 T4-T8 平面以下）
- ✅ **本体感觉保留或部分保留**（后柱由后脊髓动脉供血，分水岭效应较轻）
- ✅ 上肢正常（因为是胸段而非颈段）
- 感觉平面定位：**乳头水平 = T4，剑突 = T6，肋缘 = T8**

**一句话锁定**：

> 大手术中血压长时间过低 / 心脏骤停复苏后 → 醒来发现腿不能动、胸部以下痛温觉消失 → **T4-T8 watershed infarction**



看到 stem 描述脊髓损伤，按这个顺序读：
#### Step 1：找"感觉消失的最高皮节"→ 锁定平面

| 感觉消失最高点 | 损伤平面      |
| ------- | --------- |
| 锁骨下     | C4        |
| 上臂外侧    | C5        |
| 拇指      | C6        |
| 中指      | C7        |
| 小指      | C8        |
| **乳头**  | **T4** ⭐  |
| **剑突**  | **T6** ⭐  |
| **肋缘**  | **T8** ⭐  |
| **肚脐**  | **T10** ⭐ |
| **腹股沟** | **L1** ⭐  |
| 膝前      | L3-L4     |
| 足背      | L5        |
| 足底外侧    | S1        |
| 肛周（鞍区）  | S2-S5     |
#### Step 2：看运动→ 判断束（哪个动脉/哪种综合征）

- 下肢瘫痪 + 痛温觉丧失 + 振动觉保留 → **ASA 综合征**（前 2/3）
- 同侧运动 + 对侧痛温 → **Brown-Séquard**（半切）
- 上肢重于下肢 → **Central cord**（中央型，老人摔跤过伸）
- 振动/本体觉单独丧失 → **后柱**（**梅毒、B12、MS**）

### ⭐ 一句话总结

> **"痛温定平面，运动定严重，振动定束别"**

> 题干说"瘫痪"——只能告诉你是脊髓问题 
> 题干说"乳头以下痛温没了"——告诉你 T4，是 watershed 
> 题干说"振动觉好"——告诉你不是全横断，是 ASA
**感觉平面（尤其痛温觉）= 定损伤平面的金标准
** **运动平面 = 不能定位，而是用于判断严重程度和预后** 

---

### 3️⃣ Adamkiewicz 损伤 vs Watershed 鉴别表

|特征|Adamkiewicz (T9-T12)|Watershed (T4-T8)|
|---|---|---|
|**典型病因**|AAA 修复、主动脉手术、夹层|持续休克、心脏骤停后复苏|
|**机制**|直接动脉损伤/结扎|全身低灌注|
|**损伤平面**|Thoracolumbar（约 T10 以下）|中胸段（T4-T8）|
|**感觉平面**|肚脐（T10）/腹股沟（L1）以下|乳头（T4）/剑突（T6）以下|
|**上肢**|正常|正常|
|**下肢运动**|瘫痪|瘫痪|
|**痛温觉**|丧失|丧失|
|**本体感觉**|保留|保留（或轻度受累）|
|**大小便**|失禁|失禁|

**两者共同特征 = ASA 综合征**：运动 + 痛温觉受累、本体感觉保留

---

### ⭐ Memory Hook

> **"AAA 钳一夹，T10 以下全趴下；血压塌一晚，乳头以下没感觉"**

- AAA 手术 → Adamkiewicz → 肚脐（T10）以下挂
- 长时间低血压 → Watershed → 乳头（T4）以下挂
- **共性**：都是 ASA 综合征（前 2/3 脊髓）→ 运动 + 痛温觉死，本体感觉活

## ACS 经典临床情景与 Stem 套路（增量补充）

> [!info] 适用范围 此段为 Anterior Cord Syndrome（ACS）三大考点 —— Adamkiewicz、Watershed、ASA occlusion —— 在 NBME / UWorld 题目里的**具体临床情景模板**。可直接对照 stem 关键词识别。

---

### 三大经典临床情景（NBME 高频）

#### 情景 1：胸/腹主动脉手术后（最高频）

**Stem 关键词识别**：

|Key Stem Clue（英文）|Why It Matters|中文解析|
|---|---|---|
|Status post **thoracic aortic aneurysm repair**|经典触发词|胸主动脉瘤修复术后|
|Following **AAA repair**|AAA 修复术后|腹主动脉瘤修复|
|After **aortic cross-clamping**|术中主动脉阻断|阻断期间 Adamkiewicz 缺血|
|Within hours of aortic surgery|时间线急性|数小时内出现神经症状|

> [!example] 经典 stem 模板 "70 岁男性，**胸主动脉瘤修复术后 1 小时**，突发剧烈背痛，双下肢肌力 1/5，**T10 平面以下痛温觉丧失，振动觉/位置觉正常**。最可能诊断？"

**考点指向**：

- 病因：**Artery of Adamkiewicz** 在术中被结扎/损伤
- 诊断：Anterior Cord Syndrome
- 平面：thoracolumbar（Adamkiewicz 供应下 2/3）

---

#### 情景 2：主动脉夹层（Aortic Dissection）

**Stem 关键词识别**：

|Key Stem Clue（英文）|Why It Matters|中文解析|
|---|---|---|
|**Tearing chest pain radiating to the back**|夹层典型疼痛|撕裂样胸背痛|
|**Aortic dissection** on CT/echo|影像确诊|CT/超声示夹层|
|**Widened mediastinum** on CXR|胸片线索|纵隔增宽|
|Marfan / Ehlers-Danlos / uncontrolled HTN|危险因素|结缔组织病 / 高血压|

> [!example] 经典 stem 模板 "55 岁高血压男性，突发**撕裂样胸背痛**，随即**双下肢瘫 + 痛温觉丧失**。CT 示主动脉夹层。脊髓损伤机制？"

**考点指向**：

- 机制：夹层延伸到 **intercostal artery 开口** → Adamkiewicz 血流中断
- 诊断：ACS
- 陷阱：让你以为是 stroke，但要识别**夹层间接造成脊髓梗死**

---

#### 情景 3：严重低血压 / 大出血 / 心脏术后（Watershed 梗死）

**Stem 关键词识别**：

|Key Stem Clue（英文）|Why It Matters|中文解析|
|---|---|---|
|**Cardiopulmonary bypass / cardiac surgery**|体外循环|CABG 等心脏手术|
|**Prolonged hypotension** during surgery|术中长时间低血压|MAP < 50 mmHg 持续|
|**Hemorrhagic shock / massive blood loss**|失血性休克|大出血复苏后|
|**Cardiac arrest** 复苏后|心跳骤停|ROSC 后出现下肢瘫|

> [!example] 经典 stem 模板 "65 岁女性，**CABG 术中曾有 30 分钟 MAP < 50 mmHg**。术后清醒发现**双下肢瘫 + 痛温觉丧失，振动觉正常**。神经查体定位中胸段。最可能机制？"

**考点指向**：

- 机制：**T4–T8 watershed area 低灌注**（不是特定动脉闭塞，而是分水岭区缺血）
- 诊断：ACS（局限在 T4–T8）

> [!warning] Watershed vs Adamkiewicz 鉴别（高频陷阱）
> 
> - **Watershed infarct = mid-thoracic (T4–T8)**：由整体低灌注引起
> - **Adamkiewicz 损伤 = thoracolumbar (T8 以下)**：由特定动脉闭塞引起
> 
> Stem 给"心脏术后"+"中胸段定位" → watershed Stem 给"主动脉手术后"+"thoracolumbar 定位" → Adamkiewicz

---

### 第二梯队情景（中频）

|情景|Stem 特征|机制|
|---|---|---|
|**动脉粥样硬化栓塞**|老年人 + DM/HTN/吸烟/PAD + 突发 ACS 无外伤无手术|ASA 原位血栓 / 栓塞|
|**脊髓血管畸形（AVM）**|年轻人 + 反复发作下肢无力 + 脊髓血管异常影像|AVM 盗血 / 出血压迫|
|**椎间盘突出压迫 ASA**|急性巨大椎间盘突出 + 双下肢瘫 + 痛温觉丧失|椎间盘碎片压迫 ASA|
|**Vertebral burst fracture**|高处坠落 / 车祸 / 跳水 + 急性 ACS|骨片向后挤压脊髓前部|

---

### NBME 常见题型变体

#### 题型 A：定位题

> "病变最可能在以下哪个解剖位置？"

正确答案：

- Anterior spinal artery territory
- Anterior 2/3 of the spinal cord

干扰项：

- Posterior spinal artery territory（错，那是 dorsal column）
- Lateral corticospinal tract bilaterally（错，只是部分病变区域）

#### 题型 B：动脉识别题

> "Which artery is most likely affected?"

答案：**Artery of Adamkiewicz**（若 stem 提到 thoracolumbar 平面 + 主动脉手术）

#### 题型 C：机制题

> "What is the most likely mechanism of his neurological deficit?"

按 stem 选：

- **Watershed infarction of the spinal cord**（若是低灌注 stem）
- **Anterior spinal artery occlusion**（若是手术 / 夹层）

#### 题型 D：保留功能题（反向考解剖）

> "Which sensory modality is most likely **preserved**?"

答案：**Vibration and proprioception**（dorsal column 由 PSA 供血，保留）

### 一句话锁定表

|Stem 关键词|一句话锁定|
|---|---|
|胸主动脉术后 + 双下肢瘫 + dorsal column 保留|Adamkiewicz 损伤 → ACS|
|心脏术后 / shock / cardiac arrest 后 + 中胸段 ACS|Watershed (T4–T8) infarct|
|撕裂样胸痛 + 双下肢瘫|主动脉夹层延伸到肋间动脉 → ACS|
|老年 PAD 危险因素 + 突发 ACS 无外伤|ASA 动脉粥样硬化栓塞|
|Vertebral burst fracture + ACS 表现|骨片直接压迫 ASA|

### 鉴别陷阱（NBME 挖坑模式）

> [!danger] 坑 1：Leriche syndrome 干扰
> 
> - Leriche = aortoiliac 慢性阻塞，**无**急性脊髓症状
> - 区分：Leriche 是**慢性** claudication + ED + 股动脉消失；ACS 是**急性**瘫痪 + 感觉障碍

> [!danger] 坑 2：Brown-Séquard 干扰
> 
> - 半切 = **单侧** motor + **对侧** pain/temp
> - ACS = **双侧**都丧失
> - 看 stem 是否写 "bilateral"

> [!danger] 坑 3：Cauda equina 干扰
> 
> - Cauda equina = LMN 表现（areflexia）+ saddle 麻木 + bowel/bladder
> - 病变在 L1 以下
> - ACS 病变在更高位（thoracic）→ UMN 表现（除急性 spinal shock 期）

> [!danger] 坑 4：Spinal epidural hematoma 干扰
> 
> - 抗凝 / 外伤后急性背痛 + 神经定位
> - 但**所有束都受累**（不像 ACS 选择性保留 dorsal column）

**2. 6 Brown-Séquard Syndrome**
> [!success] BS 三步识别法
> 
> 1. **找穿透伤 / 半切机制** ✅
> 2. **症状一侧一种** —— 不是"全废"，是"分工"
> 3. **运动 + 本体觉永远同一边**；**痛温觉永远在另一边** —— 就是 Brown-Séquard

下次看到 stem 描述 "左手不能动 + 左手不知道位置 + 右手感觉不到热" → **0.5 秒锁定 BS，不再蒙**。

> [!tip] 一句话锁定 **穿透伤 + 同侧运动+本体↓ + 对侧痛温↓（损伤平面以下）** → **Brown-Séquard syndrome**

---

### ⭐ 为什么"同侧运动 + 同侧本体觉 + 对侧痛温"？解剖必懂

这是 USMLE Neuro 最经典的解剖逻辑题，**搞懂一次永远不再蒙**：

```
脊髓三大束的"交叉时机":

┌─────────────────────────────────────────────┐
│  纤维束             在哪里交叉？             │
├─────────────────────────────────────────────┤
│  Corticospinal     延髓 (medulla)            │
│  (运动)            → 进入脊髓时已经在对侧    │
│                    → 脊髓损伤 = 同侧瘫       │
├─────────────────────────────────────────────┤
│  Dorsal column     延髓 (medulla)            │
│  (本体/振动/       → 在脊髓内一路同侧上行    │
│   轻触)            → 脊髓损伤 = 同侧↓        │
├─────────────────────────────────────────────┤
│  Spinothalamic     脊髓内即刻交叉            │
│  (痛温)            （进入脊髓 1-2 节段就过对侧）│
│                    → 脊髓损伤 = 对侧↓        │
└─────────────────────────────────────────────┘
```

#### 一句话记忆口诀

> [!success] **"两个在上面交叉，一个在脊髓里就交叉"**
> 
> - **运动 + 后索** = 延髓交叉 → 脊髓里都在**同侧**走 → 半切伤 = **同侧↓**
> - **痛温** = 进脊髓就交叉 → 脊髓里走**对侧** → 半切伤 = **对侧↓**
> 
> **画面感**：刺右半边脊髓 → 切断右侧运动 + 右侧本体（都是同侧） + 切断"从左侧爬过来的"痛温纤维（来自左半身）→ **右侧动不了 + 右侧没本体 + 左侧没痛温**

---

### 详细 BS 损伤示意（T10 半切，右侧）

```
T10 水平脊髓横截面（右半切）：

        后索（被切）
        ↓
   ┌────╳────┐
   │  XXX   │  ← 右侧后索：本体/振动/部分轻触
   │  XXX   │     → 右下肢同侧↓
   │  XXX   │
   │XX   ▮▮│  ← 左侧脊髓丘脑束 完整（来自右下肢的痛温）
   │XX   ▮▮│     → 右下肢痛温觉相对保留？
   │XX   ▮▮│
   │  XXX   │  ← 右侧外侧 CST（被切）
   │  XXX   │     → 右下肢同侧瘫
   │XX      │  ← 右侧脊髓丘脑束（被切）：来自左下肢的痛温
   │XX      │     → 左下肢痛温觉↓（损伤平面以下 1-2 节段开始）
   └────────┘

X = 被切断的纤维
▮ = 保留的纤维

T10 = 脐部水平 → 所以"脐以下"指 T10 损伤平面以下
```

⚠️ **细节高频考点**：

- 对侧痛温觉↓ 一般**从损伤平面下 1-2 节段开始**（因为脊髓丘脑束交叉斜行 1-2 节段才完全过对侧）
- 损伤同侧**那一节段**可以有同侧痛温↓（因为后角细胞被破坏）+ LMN 征（前角受损）

|鉴别要点|Anterior Spinal Artery 综合征|Brown-Séquard|
|---|---|---|
|**诱因**|**主动脉手术**、动脉栓塞、低灌注|**穿透伤**（刀刺、枪伤）、肿瘤、MS（脱髓鞘半切）|
|**对称性**|**双侧**|**单侧**|
|**运动**|双侧瘫（CST）|同侧瘫|
|**痛温觉**|双侧↓|**对侧**↓|
|**本体/振动**|**保留** ⭐（后索完整）|同侧↓|
|**膀胱/肠**|**常受累**（autonomic）|通常正常|
|**典型 stem**|"Post-aortic surgery, can't move legs, no pain/temp, but can feel vibration"|"Stab wound to back, weak on one side, can't feel temp on the other"|

⭐ **ASA 综合征的灵魂特征**：**"运动 + 痛温都没了，但本体觉/振动还在"** → 病人能感觉腿的位置但动不了、感觉不到痛。这是 USMLE 的标志性描述。


# 模块 3：痴呆完整谱系 ⭐⭐⭐ 最重磅模块

## 3.1 DSM-5 分类

```
正常老化  →  MCI (Mild NCD)  →  Dementia (Major NCD)
                ↓                          ↓
        IADL 不受影响              IADL 受损（金标准！）
        能独立生活                 不能独立
        每年 5-15% 进展 → 痴呆      不可逆
```

### MMSE 数字标准 ⭐⭐
| 分数 | 意义 |
|---|---|
| **28-30** | 正常老化（"benign senility"）|
| 24-27 | MCI |
| **< 24** | 痴呆诊断阈值 |
| **20-24** | Mild AD → **Donepezil** |
| **10-19** | Mod-Severe AD → **Memantine** (+/- Donepezil) |
| **< 10** | Severe AD |

⭐ **MMSE 18 = Donepezil vs Memantine 分水岭**

### 3 个认知量表
- **MMSE**（30 分制，24/18 cut-off）→ 临床决策
- **MoCA**（30 分制，< 26 = 异常）→ 比 MMSE 敏感（抓 MCI）
- **Mini-Cog**（3 词回忆 + 画钟）→ 快速筛查

### ADL 6 项 vs IADL 8 项 ⭐
- **ADL** = DEATH-B：Dressing, Eating, Ambulating, Toileting, Hygiene, Bowel/Bladder continence
- **IADL** = SHAFT-MT：Shopping, Housekeeping, Accounting, Food preparation, Transportation, Telephone, Medication
- **受损顺序**：IADL 先（理财→用药→做饭）→ ADL 后（洗澡→穿衣→吃饭）

## 3.2 Alzheimer Disease 完整模块 ⭐

### 风险因素（早发 vs 晚发对比 ⭐ USMLE 必考）

| 类型 | 最强 risk |
|---|---|
| **早发 AD (< 65)** | **Family history** ⭐ |
| **晚发/散发 AD (> 65)** | **Age** ⭐ |

### 早发家族性 AD 3 基因（常染色体显性）
- **APP** (chr 21) ⭐ Down 关联
- **PSEN1** (chr 14) - 最常见
- **PSEN2** (chr 1)

### APOE 风险修饰
- **ε4 → ↑ risk**（"4 for AD"）
- **ε2 → protective**（"pro-2-ctive"）

### APP 切割通路 ⭐
- **α + γ-secretase** = 安全
- **β + γ-secretase** = 致病 → Aβ peptide → 斑块
- PSEN1/PSEN2 突变 = γ-secretase 异常 → Aβ42↑

### 病理 5 特征
1. **Senile plaques**（Aβ，细胞外）
2. **NFT**（tau，细胞内，**数量与痴呆程度相关** ⭐）
3. **Hirano bodies**（嗜酸性，海马）
4. **Granulovacuolar degeneration**
5. **海马 + 颞顶萎缩 + 脑沟增宽**
- 并发：**CAA**（Aβ 血管壁沉积 → 脑叶出血）

### 临床（5 大域）
- 记忆（顺行性遗忘）⭐ 早期 / 首发
- 视空间（迷路）
- 执行（不会理财）
- 语言（找词难，poverty of speech）
- 运动（晚期 gait apraxia → 跌倒）

### Sundowning（夜间恶化）
- 非药物 first：**降噪 + 减刺激 + 规律作息**
- ❌ "Bright illumination 24/7"（NBME 经典错误答案）
- ❌ BZD（加重）

### 治疗分层
```
MMSE > 18 → Donepezil（AChE 抑制剂）
MMSE ≤ 18 → Memantine 加用（NMDA 拮抗，不替换）
早期 MCI/AD → Lecanemab（抗 Aβ 单抗，ARIA 风险）
```

⚠️ **机制陷阱**：
- **AChE inhibitor ≠ ACEi**（别混）
- **Memantine ≠ 抗 Aβ**，是 NMDA 拮抗（兴奋毒性下游）
- **所有 AD 药物本质都是对症**，只有抗 Aβ 单抗"针对病理"

### AD 三层楼用药模型 ⭐
```
病因层：Aβ 沉积 ← 抗 Aβ 单抗 "剪源头"
   ↓
毒性层：谷氨酸兴奋毒性 ← Memantine "踩刹车"
   ↓
症状层：ACh↓ ← AChE 抑制剂 "打补丁"
```

### AD 行为症状（BPSD）金原则
**非药物 first**！按症状选药：
- 抑郁 → SSRI
- 严重激越/幻觉 → Quetiapine 短期低剂量（黑框警告↑死亡）
- **失眠 → Trazodone 首选**（不是 BZD）
- ❌ 禁用：BZD / 抗胆碱（Benztropine/Diphenhydramine）/ Haloperidol 高剂量

### Beers Criteria（老年人禁忌药 5 类）
1. **抗胆碱**：一代抗组胺、TCA、Oxybutynin
2. **心血管**：alpha-1 blockers、中枢 alpha-2、多种抗心律失常药
3. **CNS**：TCAs、抗精神病、Barbiturates、BZDs
4. **内分泌**：长效磺脲、Sliding scale insulin
5. **疼痛**：长期非选择 NSAIDs、骨骼肌松弛药

### Down 综合征早发 AD
- APP 在 21 号 → 3 拷贝 → APP↑ 50% → Aβ↑
- 40 岁脑内 AD 病理几乎 100%
- 50-60 岁约 50-70% 临床痴呆
- 进展更快（3-7 年 vs 散发 8-10 年）

## 3.3 Vascular Dementia (VaD)

### 4 大金钩 (CASF)
- **C**V 风险因素（HTN/DM/CAD/AFib）
- **A**brupt/Stepwise decline ⭐
- **S**ystem-localized focal neuro signs ⭐⭐
- **F**indings on MRI（多发腔隙 / 白质病变）

### 3 亚型
- **Multi-infarct dementia** - 多次大血管卒中累积
- **Strategic infarct dementia** - 单个关键位置（丘脑/内囊膝/海马/扣带）→ 急性
- **Subcortical vascular encephalopathy (Binswanger)** - 小血管病 + 弥漫白质 + 尿失禁/步态异常

### 治疗 = 二级预防（不是 Donepezil！）
- **Aspirin + Statin** + 控 BP/DM + 戒烟 + 抗凝（AFib → DOAC）

### Pronator drift（旋前肌漂移）⭐
- 闭眼 + 双臂前平举 + 掌朝上
- 阳性：一侧手臂缓慢下漂 + 内旋
- **= UMN 征**（比 hyperreflexia/Babinski 更敏感）

## 3.4 Lewy Body Dementia (LBD) ⭐

### 四联
1. **视幻觉** (visual hallucinations) - "看到小孩/动物"
2. **波动性认知** - "Good days and bad days"
3. **自发性帕金森** - bradykinesia/rigidity/tremor
4. **REM 睡眠行为障碍 (RBD)** - 梦中拳打脚踢

### LBD vs PD-D 1 年规则 ⭐
- **认知 vs 运动症状间隔 < 1 年** = LBD
- **运动数年后认知** (> 1 年) = PD-D

### 致命药物陷阱 ⭐⭐⭐
**LBD 患者极度敏感于抗精神病药** → Haloperidol/Risperidone **致命 NMS 样反应**
- 视幻觉首选：**Rivastigmine**（AChE-i）
- 必须时：低剂量 **Quetiapine**（D2 亲和力低）

## 3.5 Frontotemporal Dementia (FTD)

### bvFTD 6 大核心
1. **去抑制**（Disinhibition）⭐ 不当评论/性行为/偷窃
2. **淡漠 / 惯性**
3. **同情心丧失**
4. **强迫 / 仪式化行为**
5. **饮食改变 / Hyperorality** ⭐ 偏甜偏碳水 + 增重 + pica
6. **执行功能↓**

### USMLE 招牌
- **MMSE 正常或近正常** ⭐（记忆相对保留）
- **50-60 岁**起病
- **额颞叶萎缩**（MRI）
- **家族性 20-40%**（C9orf72 / MAPT / GRN）

### FTD 病理 3 类
- **FTLD-tau**（Pick bodies，MAPT 突变）
- **FTLD-TDP**（C9orf72, GRN）⭐ 与 **ALS 谱系**重叠
- **FTLD-FUS**

### 治疗
- SSRI / Trazodone（对症）
- **AChE-i 可能加重** ❌

## 3.6 Normal Pressure Hydrocephalus (NPH) ⭐ 可治愈

### 金三联
- **Wet** = 尿失禁（最先出现）
- **Wobbly** = 磁性步态
- **Wacky** = 痴呆

### 影像 vs AD（金标准鉴别）
- **AD** = 脑室扩大 + **sulci 增宽**（ex vacuo）
- **NPH** = 脑室扩大 + **sulci 正常**（true hydrocephalus）

### 治疗
- **LP tap test**（诊断 + 部分治疗）
- **VP shunt**（首选）⭐
- 早期 + 三联齐全 → 症状可逆 ⭐

## 3.7 Creutzfeldt-Jakob Disease (CJD)

### 4 大金标准
1. **快速进展性痴呆**（数周-月，< 12 月致死）
2. **Startle myoclonus**（响声诱发）⭐
3. **EEG 周期性尖波**（1Hz triphasic）
4. **CSF 14-3-3 蛋白 / RT-QuIC**

### 病理
- **PrP^C → PrP^Sc** β-折叠错构
- 抗蛋白酶 → 海绵状改变（spongiform）

### 类型
- 散发（85%）
- 家族性（PRNP 突变）
- 医源性（角膜移植、神经外科器械、生长激素提取）
- 变异型 vCJD（疯牛病食用）

### 治疗 = 无（致死）

## 3.8 HIV-Associated Dementia (HIV-D)

- CD4 < 200 触发
- 皮质下功能障碍 + 共济失调 + 抑郁 + 锥体束征
- MRI: 弥漫灰质 + 皮层下萎缩 + ex vacuo
- 治疗：**ART**

### HIV+ 痴呆鉴别
- HIV-D / PML (JC) / CMV / CNS 淋巴瘤 / Crypto meningitis

## 3.9 Pseudodementia（抑郁假性痴呆）⭐

### 5 大钩子
1. **起病数周-月**（快）
2. **病人自己关心**（自知力存在）⭐
3. **回答"我不知道"**（放弃式）
4. **抑郁征核心**（5/9 MDD：抑郁 / 兴趣丧失 / 睡眠 / 食欲 / PMR / 疲乏 / 无价值 / 注意力 / 自杀）
5. **抑郁先于认知**

### 治疗 = SSRI ⭐
- Sertraline / Citalopram (≤ 20mg/d 老人，QT↑)
- 4-6 周认知应改善 → 不改善 = 重新查 dementia

### USMLE 黄金鉴别口诀
**"Pseudodementia patients complain; AD patients are brought in"**

## 3.10 Subacute Combined Degeneration (B12 SCD)

### 累及的 3 个束 + 1 个不累及（反向记忆法）⭐
- 累及：**Dorsal columns**（振动/本体）+ **Lateral CST**（UMN）+ **Spinocerebellar**（共济失调）
- **Spinothalamic 不累及**（痛温觉保留！）⭐ 反向记忆法

### 高危人群
- Vegan / "tea and toast diet" / **Pernicious anemia** / Crohn / **Metformin 长期** / PPI / **N2O 滥用**

### 诊断
- B12 ↓（但可正常）/ **MMA + homocysteine 更敏感**
- 神经症状可早于贫血

### 治疗 = B12 IM 注射

|---|---|---|
| Pseudodementia | concerned + lost interest + 数周 | SSRI |
| B12 SCD | + 后索征 + macrocytic anemia | B12 注射 |

| **神经梅毒** | + Argyll-Robertson + tabes + (+)VDRL | IV PCN |
| Neurolyme | 流行区 + 蜱 + EM + 颜面瘫史 | Doxy / Ceftriaxone |
| **NPH** | Wet/Wobbly/Wacky | VP shunt |
| **慢性 SDH** | 老人 + 跌倒 + 抗凝 + 波动认知 | 手术引流 |
| 药物毒性 | 抗胆碱 / BZD / 阿片 | 停药 |

### 痴呆必查 workup
- **TSH + Vit B12 + RPR/VDRL + CBC + CMP + Folate** + 抑郁筛 + MRI

## 3.12 Huntington Disease (HD) ⭐

### 核心事实
- **常染色体显性** / **HTT 基因** chr 4
- **CAG 重复 > 40** + **anticipation**（父系更早 + 更重）
- 发病 ~40 岁
- 几乎必有家族史（< 3% 散发）

### 三联
- **精神症状**（早期 + 自杀风险↑）
- **Chorea**（舞蹈样）⭐
- 痴呆

### 病理
- **尾状核 (caudate) 萎缩** + 脑室代偿性扩大（"box-like"）
- mutant huntingtin

### 治疗
- **Chorea**：VMAT2 抑制剂（**Tetrabenazine / Deutetrabenazine / Valbenazine**）
- 抑郁：SSRI
- 精神病性：非典型抗精神病
- **无疾病修饰治疗**

## 3.13 Wilson Disease

### 核心事实
- **常染色体隐性** / **ATP7B 基因** chr 13
- **< 35 岁** 发病 ⭐

### 三系统受累
1. **肝**：肝炎 → 肝硬化（年轻人肝硬化想 Wilson）
2. **脑**：基底节铜沉积 → **拍翅样震颤** + 精神症状 + 痴呆
3. **眼**：**Kayser-Fleischer 环**（Descemet 膜铜沉积）

### 诊断
- 裂隙灯（K-F 环）
- **Ceruloplasmin ↓**
- 24h 尿铜 ↑
- 肝活检（金标准）

### 治疗
- 低铜饮食
- **Penicillamine**（首选铜螯合剂；副作用 SLE 样反应）
- Trientine（二线）
- Zinc 盐（维持）
- 肝移植（暴发性 / 失代偿）

|---|---|---|---|---|---|---|---|

# 模块 4：头痛 5 大类型 + 进阶 ⭐

## 4.1 头痛 5 大全谱速查 ⭐

| 类型 | 性别 | 部位 | 性质 | 持续 | 招牌伴随 |
|---|---|---|---|---|---|
| **Migraine** | 女 > 男 (3:1) | 单侧 | **搏动性** | 4-72 hr | 畏光 / 畏声 / 恶心 / ± aura |
| **Tension** | 女 > 男 | **双侧 band-like** | 紧绷 / 钝 | 30 min - 7 天 | 颈肩肌紧 / 应激 |
| **Cluster** | **男 >> 女** | **单侧眶周/眼后** | 钻刺 / 灼烧 | **15-180 min** | **自主神经征 + Horner** |
| **Trigeminal neuralgia** | 女 > 男 | V2/V3 单侧 | **闪电样** | **< 1 min** | 触发点 |
| **GCA** | 女 > 男 + **> 50 岁** | 颞部 | 钝痛 | 持续 | **颌跛行 + 视力↓ + ESR↑** |

### 病人行为差异 ⭐ NBME 钩子
- **Migraine** = 躲暗室、闭眼
- **Cluster** = **踱步、撞墙**
- **TN** = 避免触面 / 不敢刷牙
- **GCA** = 摸头时头皮痛，咀嚼时下颌痛

## 4.2 Migraine 完整

### POUND 诊断
- **P**ulsatile / **O**ne-day / **U**nilateral / **N**ausea / **D**isabling
- 4/5+ = migraine

### 4 阶段
- Prodrome（数小时-天，食欲改变）
- Aura（5-60 min，视觉/感觉/运动）
- Headache（4-72h）
- Postdrome（疲乏）

### 治疗（急性 vs 预防）

| 急性 | 预防（≥4 次/月）|
|---|---|
| NSAIDs / Acetaminophen | **β-blocker** ⭐（+ HTN 双赢）|
| **Triptans** ⭐ (HTN/CAD/妊娠禁忌) | **Topiramate**（+ 肥胖 / 癫痫）|
| Ergots (HTN/CAD 禁忌) | **Amitriptyline TCA**（+ 抑郁/失眠）|
| CGRP gepants (rimegepant) | Valproate（**育龄女避免**）|
| Antiemetics（metoclopramide）| CGRP 单抗 (Erenumab) |

### 共病匹配预防药 ⭐ USMLE 秒选法
- + **HTN** → **Propranolol**
- + 抑郁 → Amitriptyline
- + 肥胖 / 癫痫 → Topiramate
- + 妊娠 → 一线 **Acetaminophen**（禁 Triptan/Ergot/Valproate/Topiramate）
- + 育龄女 → 避免 Valproate（致畸）

### 病理生理
- Cortical Spreading Depression + Trigeminovascular system + **CGRP 释放**

### 妊娠期治疗阶梯
1. 非药物（休息/补水/热敷）
2. **Acetaminophen** ⭐ 首选
3. Antiemetics
4. NSAIDs（仅 2nd trimester，3rd 禁忌→PDA 早闭+羊水少）
5. Opioids 最后

### ⚠️ 经典陷阱
- **OCP + Migraine with aura → 卒中风险↑ 6-8 倍** → 雌激素 OCP 禁忌
- Triptans 用前查 HTN / CAD / 妊娠

## 4.3 Cluster Headache ⭐

### 招牌
- 男性 20-40，**半夜痛醒**，**踱步躁动**
- 单侧眶周 / 眼后，15-180 分钟
- **昼夜节律 + 季节性聚集**（连续 6-12 周每天）
- 自主神经征（同侧）：流泪 + 流涕 + 鼻塞 + 结膜充血 + **Horner** + 出汗

### 治疗 ⭐ USMLE 必考
- **急性首选：100% O₂ 12-15 L/min**（非 Triptan / 非 NSAID）⭐
- 二线急性：Sumatriptan SC / INH
- **预防：Verapamil** ⭐

## 4.4 Trigeminal Neuralgia

### 招牌
- **"Lancinating / knife-like / electric shock-like"** < 1 min
- V2/V3 分布（V1 罕见 → V1 痛先想 cluster）
- 触发点（刷牙 / 嚼 / 风 / 触面 / 冷水）
- 机制：**小脑上动脉**压迫 CN V

### 治疗
- **Carbamazepine** ⭐ 首选
- Oxcarbazepine / Baclofen / Gabapentin
- 微血管减压术

### USMLE NBME 钩子
- "哪部分大脑受累？" → **Pons（脑桥，CN V 起源）**
- 年轻人单侧 TN → 查 **MS**

## 4.5 Giant Cell Arteritis (GCA / Temporal Arteritis) ⭐ 急症

### 5 大标志
1. **> 50 岁** ⭐ 必备
2. 新发颞部头痛
3. **颌跛行 (jaw claudication)** ⭐
4. **视力变化**（一过性失明 / 复视 / 永久失明）
5. **ESR > 50** + CRP↑
- ± **PMR**（近端肢体痛 + 晨僵 + ESR↑，无头痛）
- ± 颞动脉压痛 + 搏动消失

### 黄金规则 ⭐⭐
**"Treat first, biopsy later"**
- 怀疑 → **立刻 IV 类固醇**（不等 biopsy）
- IV Methylprednisolone（视力受累）/ 口服泼尼松（无视力症状）
- Biopsy 1-2 周内做仍准确
- 长期口服 + 加 ASA + 必要时 Tocilizumab

### 并发症
- **缺血性视神经病变 (ION)** → 永久失明
- 主动脉瘤

### PMR vs Polymyositis
- **PMR**：近端肢体痛 + 晨僵 + ESR↑ + **CK 正常**（治疗：低剂量泼尼松 15-20mg）
- **Polymyositis**：肌无力 + **CK↑** + EMG/MRI/biopsy

## 4.6 进阶头痛（占位 + 已讲细节）

### SAH（蛛网膜下腔出血）
- "**Worst headache of my life**" + thunderclap + 项强直
- 病因：Saccular (berry) aneurysm 破裂
- 诊断：非增强 CT 首选 → 阴性高度怀疑 → LP（xanthochromia）
- 并发症：**Vasospasm (3-14 天)** → **Nimodipine 预防**
- 治疗：神经外科夹闭 / 弹簧圈 + 控 BP

### Preeclampsia / Eclampsia
- 妊娠 > 20 周 + BP > 140/90 + 蛋白尿
- 严重：BP > 160/110 / 头痛 / 视觉变化 / 上腹痛 / 抽搐 = eclampsia
- 治疗：**Magnesium sulfate** + labetalol/hydralazine/nifedipine + **分娩**

### IIH (Idiopathic Intracranial Hypertension / Pseudotumor)
- 年轻肥胖女 + 头痛 + papilledema + CN VI 麻痹（外展受限/复视）
- 触发：维 A 酸 / OCP / 四环素 / 类固醇撤药
- LP 见压力 > 25 cmH₂O
- 治疗：**Acetazolamide** ⭐ + 减重 + 视神经鞘开窗 / VP shunt
- 不治 → 永久失明

### Increased ICP / 占位
- 早晨重 + Valsalva 加重 + projectile vomiting
- Cushing's triad（晚期）：HTN + bradycardia + 不规则呼吸
- MRI 增强首选

### Medication Overuse Headache (MOH)
- 急性药 > 10-15 天/月 持续 > 3 月
- 治疗：停药/替换 + 开始预防药

### Headache SNOOP red flags
- **S**ystemic / **N**eurologic / **O**nset sudden / **O**lder / **P**rogressive / **P**recipitated by Valsalva / **P**regnancy

# 模块 5：跨学科 Schema（高 ROI 衍生笔记素材）

## 5.1 Horner 综合征 3 级定位 ⭐⭐⭐（已建议立即建主笔记）

| 级别 | 损伤部位 | 病因 | 伴随征 |
|---|---|---|---|
| **一级（中枢）** | 下丘脑 → T1-T2 lateral horn | **Syringomyelia** / **Wallenberg** / 脑干卒中 / MS | 脊髓 / 脑干征 |
| **二级（节前）** | T1-T2 → stellate → SCG | **Pancoast tumor** / 颈部手术 / 颈胸创伤 / 主动脉瘤 | 手内在肌萎缩 (C8-T1) / 肺尖肿块 |
| **三级（节后）** | SCG → ICA → 眼 | **Carotid artery dissection** / 海绵窦病变 / Cluster | 同侧颈痛/头痛、可无 anhidrosis (Partial Horner) |

### 鉴别 Horner ptosis vs CN III ptosis
- **CN III ptosis**：提上睑肌瘫 → 完全下垂 + mydriasis + "down and out"
- **Horner ptosis**：Müller's muscle 瘫 → 轻度下垂（1-2mm）+ **miosis** + anhidrosis

## 5.2 不自主运动 5 种鉴别 ⭐⭐

| 运动 | 特点 | 标志病 |
|---|---|---|
| Tremor | 节律性来回 | PD / 小脑 / ET |
| **Chorea** | 不规则流畅快速 | **Huntington** / Sydenham / SLE |
| Athetosis | 缓慢扭转 | HD 晚期 / 脑瘫 |
| **Myoclonus** | **闪电样、瞬间** | **CJD** / AD 晚期 / 缺氧 / 癫痫 |
| Tic | 可短暂抑制 | Tourette |
| Dystonia | 持续扭转 | 急性 EPS / 肌张力障碍 |

## 5.3 EPS (Extrapyramidal Symptoms) 4 类 + 治疗

| EPS | 起病 | 表现 | 治疗 |
|---|---|---|---|
| Acute dystonia | 小时-天 | 颈/口/舌/眼肌突发持续收缩 | **Benztropine / Diphenhydramine** |
| Akathisia | 天 | 主观坐立不安 | **β-blocker (Propranolol)** ⭐ / BZD / Benztropine |
| Parkinsonism | 天-周 | 震颤 + 僵直 + bradykinesia | Benztropine / Amantadine |
| **TD (Tardive dyskinesia)** | **长期用药 > 6 月** | 不自主舞蹈样运动（口面躯干肢）| **Valbenazine / Deutetrabenazine** ⭐（不是 Benztropine！抗胆碱反加重 TD）|

⭐ **EPS 起病时间口诀**："急 dystonia，烦 akathisia，慢 parkinsonism，老 TD"

## 5.4 DM 神经病变小纤维 vs 大纤维

| 特征 | 小纤维（Aδ/C/自主）| 大纤维（Aα/Aβ） |
|---|---|---|
| 时间 | 先损伤 | 后损伤 |
| 临床 | Burning pain / 痛温↓ / autonomic | 振动/本体↓ / 踝反射↓ / Romberg+ |
| 查体 | 针刺 / 冷热觉 | **10g monofilament** ⭐ |
| 临床意义 | **Painful diabetic neuropathy** | **Painless ulcer 风险** |

⭐ "小纤维先死你先痛，大纤维后死你才烂脚"

## 一、妊娠期 CV 决策快查

### 1.1 绝对妊娠禁忌

```
PPHM-CASC：
P - PAH
P - PPCM with residual LV dysfunction
H - HF (LVEF <30% or NYHA III-IV)
M - Marfan/aortic dilation (>45 mm; Turner ASI >25)
C - Coarctation (severe)
A - AS (severe symptomatic)
S - MS (severe symptomatic)
C - Cyanotic CHD (Eisenmenger)
```

### 1.2 妊娠生理变化（理解所有妊娠 CV 的基础）

```
Blood volume ↑ 50%
CO ↑ 30-50%
HR ↑ 10-20 bpm
SVR ↓
```

### 1.3 妊娠安全 vs 禁忌药速查

| 类别 | ✅ 安全 | ❌ 禁忌 |
|------|--------|---------|
| **HTN** | Labetalol, Nifedipine ER, Methyldopa, Hydralazine | ACEI, ARB, ARNI, MRA, Atenolol, Nitroprusside |
| **Anticoagulation** | LMWH, UFH | Warfarin*, DOACs (* = 除机械瓣) |
| **HF** | Hydralazine + Isosorbide, Furosemide, Metoprolol, Digoxin | ACEI/ARB/ARNI, MRA, SGLT-2i, Atenolol |
| **Arrhythmia** | Adenosine, Digoxin, Cardioversion | Amiodarone (除非难治), Ivabradine |
| **PAH 药** | — | Endothelin antagonists (-sentan), Riociguat |
| **降脂** | — | Statins, Fibrates, Ezetimibe |

### 1.4 急症 HTN（≥160/110）IV 用药决策

```
HR 正常/快, 无呕吐 → IV Labetalol
HR <60 (bradycardia) → IV Hydralazine
持续呕吐 → IV (避免口服 nifedipine)
急症避免：Methyldopa（起效慢）、Nitroprusside（cyanide 毒性）
```

## 二、妊娠期 HTN 谱系

### 2.1 4 种类型 1 秒鉴别

| 类型 | 时间 | 蛋白尿/器官损伤 | 抽搐 |
|------|------|------------------|------|
| Chronic HTN | 孕前或 **<20 周** | ❌ | ❌ |
| Gestational HTN | **≥20 周新发** | ❌ | ❌ |
| **Preeclampsia** | ≥20 周新发 | ✅ | ❌ |
| **Eclampsia** | + | ✅ | ✅ |
| Chronic + superimposed preeclampsia | 孕前 HTN + | ≥20 周新发蛋白尿 | — |

> [!warning] **<20 周 HTN + proteinuria** = **Molar pregnancy**
> 不是 preeclampsia

### 2.2 Preeclampsia 诊断要点

**HTN**: BP ≥140/90 (2 次, 间隔 4h) OR BP ≥160/110 (1 次重复)
**+ 蛋白尿** (任一): 24h ≥300 mg / UPCR ≥0.3 / 试纸 1+ (仅前两者不可用时)
**OR 器官损伤** (任一 = severe features):
- PLT <100K / Cr >1.1 / LFT >2× / 肺水肿 / 头痛 / 视觉异常

### 2.3 治疗

| 病 | 处理 |
|---|------|
| Gestational HTN | NMHL + 37-39 周分娩 |
| Preeclampsia 轻 | NMHL + MgSO4 + 37 周分娩 |
| **Preeclampsia 重** | NMHL + **MgSO4** + ≥34 周分娩 |
| **Eclampsia** | **MgSO4 + immediate delivery** |
| **HELLP** | **Immediate delivery** (可无 HTN/蛋白尿) |
| MgSO4 中毒 | **IV Calcium gluconate** |

### 2.4 Cushing Reflex 警报

**HTN + Bradycardia + 头痛/呕吐** = 颅内压↑ → 即将抽搐/ICH
- 选 **IV Hydralazine**（不用 labetalol，已 bradycardia）
- + MgSO4 + 紧急分娩

## 三、APS（抗磷脂综合征）

### 3.1 1 秒识别 APS

```
反复早孕流产 (≥2-3 次 <10 周)
+ 长 aPTT (不被混合纠正)
+ 假阳性 VDRL/RPR (FTA-ABS 阴性)
+ 轻度血小板减少
± Livedo reticularis ⭐
± Cryptogenic stroke in young
```

### 3.2 治疗（按情境）

| 情境 | 方案 |
|------|------|
| Non-pregnant + 既往血栓 | **终身 Warfarin (INR 2-3)** |
| **Pregnant + 既往血栓** | **ASA + 治疗剂量 LMWH** |
| **Pregnant + 仅妊娠丢失** | **ASA + 预防剂量 LMWH** |
| Pregnant + 仅抗体阳性 | ASA |
| **CAPS（灾难性）** | Heparin + steroids + plasma exchange |
| **+ SLE** | + Hydroxychloroquine |

> [!warning] APS 时间窗
> - **36 周停 ASA**（避新生儿 PDA 早闭）
> - 产后继续 LMWH **6 周**
> - **DOACs 不用于 APS**（即使非孕）

### 3.3 关键关联

- **Libman-Sacks endocarditis**（MV 无菌赘生物）→ embolic stroke
- **SLE 合并率 30-40%**
- **Stroke in young + recurrent miscarriage + livedo** → APS until proven otherwise

## 四、Peripartum Cardiomyopathy (PPCM)

### 4.1 诊断 4 条（必须全满足）

1. HF 妊娠最后 1 月 ~ 产后 5 月
2. **LVEF <0.45**
3. 无其他 HF 原因
4. 既往无心脏病

### 4.2 危险因素

- Maternal age >30, Multiparity, Multiple gestation, **Preeclampsia/eclampsia**, **African ancestry**

### 4.3 再孕咨询

| LVEF | 建议 |
|------|------|
| **≥0.5 + 持续 ≥6 月** | 谨慎尝试 |
| **<0.5 持续** | **⛔ 强烈避孕** |

### 4.4 治疗

- **Stable**: HID + Metoprolol（妊娠版 ABCD）
- **Unstable**: 紧急 C-section
- LVEF <0.35 → Anticoagulation (LMWH)

## 五、Mitral Stenosis in Pregnancy

### 5.1 1 秒识别

```
Young woman + Childhood strep throat / tonsillectomy
+ Immigrant from developing country (印度/东南亚/非洲)
+ Pregnancy 中突发 dyspnea / orthopnea / dry cough
+ New-onset AF with RVR
```

### 5.2 严重度分级

| MS 严重度 | MVA |
|----------|-----|
| Mild | >1.5 cm² |
| Moderate | 1.0-1.5 |
| **Severe** | **<1.0** |

### 5.3 急性失代偿管理

```
1. Rate control: IV Metoprolol / Digoxin
2. Diuretic: Furosemide
3. Anticoagulation: LMWH（AF）
4. Cardioversion if 不稳定
5. **PBMV**（根本治疗，妊娠期可做）
```

### 5.4 PBMV 适应症

- Symptomatic moderate-severe MS (MVA ≤1.5)
- **Wilkins ≤8**（pliable, minimal thickening/calcification, no subvalvular fusion）
- 无 LA 血栓（TEE 排除）
- 无中-重度 MR

### 5.5 体征 ⭐

- Diastolic rumble + opening snap + loud S1
- S2-OS 间距越短 → MS 越重
- P mitrale + AF
- RVH + RAD（长期 → PAH）
- Malar flush

### 5.6 Mechanical vs Bioprosthetic Valve

| 患者 | 推荐 |
|------|------|
| **育龄女性想生孩子** | **Bioprosthetic**（避 Warfarin） |
| 年轻男性 / 不计划怀孕 | Mechanical（一次到位） |
| 老年 | Bioprosthetic（寿命匹配） |

## 六、ECG 速查

### 6.1 P 波（房增大）

| P 波 | 病变 | 经典病因 |
|------|------|----------|
| **P mitrale** (II 双峰, V1 双相) | LAE | MS |
| **P pulmonale** (II tall, V1 initial up) | RAE | PAH, COPD, MS late |

### 6.2 电轴（口对口 vs 尖对尖）

```
口对口 (I+, aVF-) → LAD
尖对尖 (I-, aVF+) → RAD
都(+) → Normal
都(-) → Extreme
病理 LAD 确认 → II(-)
```

| 电轴 | 病因 |
|------|------|
| **LAD** | LVH, inferior MI, **LAFB** |
| **RAD** | RVH, **PAH, COPD, MS, PE**, lateral MI, **LPFB** |
| **Extreme** | **VT, hyperkalemia, lead misplacement** |

### 6.3 S2 Splitting

| 类型 | 病因 |
|------|------|
| **Wide** | RBBB, PS, PE |
| **Fixed** ⭐ | **ASD** |
| **Paradoxical** | LBBB, severe AS, HCM |

### 6.4 LVH / RVH 标志

- **LVH**: S(V1) + R(V5/V6) >35; S in V3 + R in aVL; ± LAD + strain
- **RVH**: RAD + **V1 R>S** + V5/V6 deep S; ± P pulmonale

### 6.5 PE ECG

| 征象 | 备注 |
|------|------|
| **Sinus tachycardia** | **最敏感** ⭐ |
| **RAD** | Acute RV strain |
| **T inversion V1-V4** | RV strain |
| **S1Q3T3** | Classic 但 sens 仅 15% |

## 七、PE 诊断与抗凝

### 7.1 诊断流程

```
Stable:
  Wells score
   ├─ <6 + D-dimer (-) → excluded
   ├─ <6 + D-dimer (+) → CTA
   └─ >6 → 直接 CTA

Unstable:
  Bedside TTE
   ├─ RV dysfunction → Massive PE → tPA
   └─ Normal RV → 找其他 shock 原因
```

### 7.2 抗凝选择

| 情境 | 首选 |
|------|------|
| General | **DOACs** |
| Cancer (non-GI/GU) | **LMWH** (或 DOAC) |
| **Pregnancy** | **LMWH only** |
| **APS** | **Warfarin** |
| **Mechanical valve** | **Warfarin** |

### 7.3 抗凝时长

- Provoked → 3 月
- Unprovoked → ≥3 月，常 indefinite
- **APS → Lifelong**
- Pregnancy → throughout + **6 wk postpartum**

## 八、抗血小板 vs 抗凝

| 维度 | Antiplatelet | Anticoagulation |
|------|--------------|-----------------|
| 适用 | 动脉（高剪切）→ 血小板血栓 | 静脉/低流量 → 纤维蛋白血栓 |
| 病变 | CAD, PCI, stroke (atherosclerotic), PAD, TIA | DVT, PE, **AF**, mechanical valve, APS, LV thrombus |
| ACS | 联合使用（DAPT + 短期肝素） | — |

## 九、先心病 — "3D + 5T"

### 9.1 总框架

```
Acyanotic L→R "3D":
   ├─ ASD (Fixed split S2)
   ├─ VSD (Holosystolic LLSB ± thrill)
   ├─ PDA (Continuous machinery murmur)
   └─ + Coarctation

Cyanotic R→L "5T":
   1 Truncus arteriosus (single S2)
   2 Transposition (egg on string)
   3 Tricuspid atresia (LAD in newborn)
   4 Tetralogy of Fallot (boot-shaped, tet spell)
   5 TAPVR (snowman sign)
```

### 9.2 Duct-Dependent → PGE1 救命

| 类型 | 病变 |
|------|------|
| **Pulmonary flow** | Severe TOF, Tricuspid atresia, Pulmonary atresia |
| **Systemic flow** | **HLHS, Critical coarctation**, critical AS, IAA |
| **Mixing** | **TGA** |

### 9.3 紫绀型 CHD Buzzwords

| Buzzword | 诊断 |
|----------|------|
| **Boot-shaped heart + tet spell + squatting relief** | **TOF** |
| **Egg on a string + maternal DM** | **TGA** |
| **Snowman sign** | **TAPVR** |
| **Single S2 + DiGeorge** | **Truncus arteriosus** |
| **LAD in newborn** | **Tricuspid atresia** |

### 9.4 关键药物对比

| 药 | 用途 |
|----|------|
| **PGE1** | 保持 DA 开放（duct-dependent → 救命） |
| **Indomethacin** | 关闭 DA（早产儿 PDA） |
| **iNO** | PPHN（选择性肺血管扩张） |

## 十、Coarctation 速查

### 10.1 临床表现按年龄

```
新生儿 Critical Coarctation:
   ├─ 前 1-2 天无症状（DA 代偿）
   ├─ DA 关闭后突发 shock + acidosis + 上下肢 BP 差异
   └─ → **PGE1 + 紧急手术**

年长儿/成人 Mild Coarctation:
   ├─ Upper extremity HTN（头痛、鼻出血）
   ├─ Lower extremity claudication
   ├─ Brachial-femoral pulse delay
   ├─ 上下肢 BP 差 >20 mmHg
   └─ Left interscapular continuous murmur
```

### 10.2 CXR 经典征象

- **"3 sign"**：aortic knob 凹陷 + post-stenotic dilation
- **Rib notching**：3rd-8th 肋下缘扇贝样（年长儿，侧支建立后）
- **Cardiomegaly + pulmonary edema**：新生儿 critical

### 10.3 关联

- **Bicuspid AV (50-85%)** ⭐
- **Turner syndrome** ⭐
- **Berry aneurysms**（颅内）
- VSD, PDA

### 10.4 5 大并发症

1. Refractory HTN
2. HF
3. **Cerebral berry aneurysm rupture → SAH**
4. Aortic dissection/rupture
5. **IE / endarteritis**（湍流 + 合并 bicuspid AV）

### 10.5 PGE1 机制（USMLE 陷阱）

```
PGE1 ≠ "扩张主动脉"
PGE1 ≠ "降 LV afterload"
PGE1 = **保持 DA 开放**
   ↓
DA 开放后，PA 压 > 缩窄远端主动脉压
   ↓
PA → DA → 降主动脉 (**R→L through DA**)
   ↓
下半身重新获得灌注（脱氧血也比无血强）
```

## 十一、PPHN 速查

### 11.1 病理 + 危险因素

```
出生后 PVR 未下降 → R→L shunt across DA + PFO
   ↓
危险因素：
   ├─ Lung hypoplasia (CDH)
   ├─ Meconium aspiration
   └─ Neonatal pneumonia / sepsis
```

### 11.2 体征

- **Pre > Post SpO2** (differential cyanosis)
- Respiratory distress + central cyanosis
- **Loud P2**
- **对 O2 部分改善**（可逆）

### 11.3 治疗

```
1. O2 + Ventilation
2. iNO ⭐（选择性肺血管扩张）
3. Sildenafil
4. ECMO（难治）
```

## 十二、PPHN vs Coarctation 关键鉴别

| 维度 | PPHN | Coarctation |
|------|------|-------------|
| **本质** | 肺血管痉挛（可逆） | 主动脉机械狭窄 |
| **SpO2 差异** | **显著 (Pre>Post)** | 对称 |
| **BP 差异** | **基本无** | **Upper >> Lower** ⭐ |
| **股动脉脉搏** | **正常** | **弱** |
| **CXR** | 心影正常 | "3 sign", rib notching, cardiomegaly |
| **O2 反应** | **改善** | 无关 |
| **救命药** | **iNO** | **PGE1** |
| **侧支循环** | ❌ | ✅（年长儿） |
| **可逆性** | 可逆 | 不可逆（需手术） |

## 十三、Hyperoxia Test

**方法**：100% O2 10-15 min → 测 PaO2

| PaO2 反应 | 诊断 |
|-----------|------|
| **>250 mmHg** | Pulmonary disease |
| 100-250 | PPHN（部分可逆） |
| **<100 mmHg** ⭐ | **Cyanotic CHD → PGE1** |

## 十四、PFO vs ASD

| 特征 | **PFO** | **ASD** |
|------|---------|---------|
| 本质 | Flap valve（两层 septum 重叠） | 真正缺损 |
| 频率 | **25-30% 人群** | 罕见 |
| 分流 | **Transient R→L on Valsalva/cough** ⭐ | **Persistent L→R** |
| S2 | Physiologic split | **Fixed split S2** ⭐ |
| 处理 | 多数无需处理 | 多数需封堵 |
| 关键并发症 | **Paradoxical embolism**（cryptogenic stroke） | Eisenmenger（晚期） |

### 14.1 PFO 处理指征

仅 4 种情境：
1. Paradoxical embolism 已发生
2. Large PFO >2.5 cm
3. 特殊职业（divers）
4. 长期抗凝禁忌

## 十五、儿童杂音 Benign vs Pathologic

### 15.1 4 大 Benign 杂音

| 杂音 | 位置 | 特征 |
|------|------|------|
| **Still's** | LLSB | Vibratory/musical, mid-systolic, supine ↑ |
| **Pulmonic flow** | Upper LSB | Soft ejection |
| **PPS** | Upper LSB + **radiates to back** | 新生儿期，自然消退 |
| **Venous hum** | Supra/infraclavicular | **Continuous, 站立 ↑** ⭐ |

### 15.2 Pathologic 红旗（任一即 Echo）

- **Diastolic** ❌
- **Holosystolic / late systolic**
- **Loud (≥3/6) / harsh / thrill**
- **Standing/Valsalva ↑**（除 venous hum）
- Central cyanosis / Fixed split S2 / Weak femoral / Hepatomegaly / FTT

## 十六、Holosystolic Murmur 鉴别

| 杂音 | 位置 | 加重 | 典型病因 |
|------|------|------|----------|
| **MR** | Apex → axilla | Handgrip ↑ | MVP, rheumatic, **Libman-Sacks** |
| **TR** | LLSB | **Inspiration ↑ (Carvallo)** | IVDU IE, PAH |
| **VSD** | LLSB | Handgrip ↑ | 先天 / post-MI rupture |

| Valsalva strain | **↓** | — |
| Standing | ↓ | — |
| Squatting | ↑ | ↑ |
| Handgrip | — | **↑↑** |
| Inspiration | ↑ right-sided | — |

## 十九、Acrocyanosis vs Central Cyanosis

| 特征 | Acrocyanosis（良性） | Central（病理） |
|------|---------------------|----------------|
| 位置 | 手脚 + 口周 | 全身 + **黏膜** |
| 黏膜颜色 | **粉红** | **紫绀** |
| SpO2 | 正常 | ↓ |
| 肢端温度 | **冰冷** | 可温 |
| 处理 | Reassurance | Workup |

## 二十、PAH 三大类药

| 通路 | 药物 | 字尾 |
|------|------|------|
| **NO–cGMP** | Sildenafil, Tadalafil (PDE-5i); **Riociguat** (sGC) | **-afil** |
| **Prostacyclin** | Epoprostenol IV, Iloprost inhaled, Selexipag oral | **-prost-** |
| **Endothelin** | **Bosentan, Ambrisentan, Macitentan** | **-sentan** |

**关键副作用**:
- Sildenafil + Nitrates = **致死性低血压**
- Bosentan = **肝毒性 + 致畸**
- Epoprostenol = **下颌痛 + 中断反跳危象**

**CTEPH 首选**：**Riociguat + PEA (pulmonary endarterectomy)**

## 二十一、HFrEF "ABCD" 四联

| 字母 | 类别 | 代表药 |
|------|------|--------|
| **A** | ARNI > ACEI > ARB | Sacubitril-valsartan / Lisinopril / Losartan |

## 二十五、复习问题 (Spaced Repetition) ⭐

> [!question]- Q1: 妊娠绝对禁忌 8 种？
> PPHM-CASC：PAH, PPCM+LV dysfunction, HF LVEF<30%, Severe coarct, Severe MS, Severe AS, Severe aortic dilation (Marfan/Turner), Eisenmenger

> [!question]- Q2: 妊娠 HTN 一线药？
> **NMHL**: Nifedipine ER, Methyldopa, Hydralazine, Labetalol（首选）

> [!question]- Q3: 妊娠 HF 治疗方案？
> **HID + Metoprolol**

> [!question]- Q4: APS 妊娠期治疗？
> **ASA + LMWH 联合**（不是单药）

> [!question]- Q5: PPCM 4 条诊断标准？
> 1月-产后5月 / LVEF<0.45 / 无其他原因 / 既往无心脏病

> [!question]- Q6: 严重 preeclampsia features？
> BP≥160/110, PLT<100K, Cr>1.1, LFT>2×, 肺水肿, 头痛/视觉

> [!question]- Q7: 蛋白尿三方法层级？
> 24h ≥300mg > UPCR ≥0.3 > 试纸 1+（仅前两者不可用时）

> [!question]- Q8: <20 周 HTN + 蛋白尿？
> **Molar pregnancy**（不是 preeclampsia）

> [!question]- Q9: Cushing reflex 三联征？
> HTN + Bradycardia + 不规则呼吸/头痛 → 颅内压↑ → 即将抽搐/ICH

> [!question]- Q10: MgSO4 中毒解毒？
> **IV Calcium gluconate**

> [!question]- Q11: HFrEF ABCD 死亡率获益？
> ABCD ✅（SGLT-2i 部分）；Loop diuretic + Digoxin ❌

> [!question]- Q12: PE 最敏感 ECG？
> **Sinus tachycardia**（不是 S1Q3T3）

> [!question]- Q13: PFO vs ASD 关键鉴别？
> PFO = Flap valve, transient R→L on cough, physiologic S2
> ASD = 真正缺损, persistent L→R, **fixed split S2**

> [!question]- Q14: TOF "PROVe"？
> Pulmonary stenosis + RVH + Overriding aorta + VSD
> Tet spell → squatting 缓解（↑SVR）

> [!question]- Q15: Coarctation 5 大并发症？
> HTN, HF, **Cerebral berry aneurysm SAH**, Aortic dissection, IE

> [!question]- Q16: PGE1 vs iNO vs Indomethacin？
> PGE1 = 保 DA（救命）
> iNO = 扩肺血管（PPHN）
> Indomethacin = 关 DA（早产 PDA）

> [!question]- Q17: PPHN vs Coarctation 鉴别？
> PPHN = SpO2 差异 + BP 对称 + iNO
> Coarctation = BP 差异 + 股弱 + PGE1

> [!question]- Q18: 反常 Valsalva 增强杂音？
> **MVP 和 HCM**

> [!question]- Q19: Holosystolic 三大病？
> MR (apex→axilla), TR (LLSB, inspiration ↑), VSD (LLSB ± thrill)

> [!question]- Q20: APS 三大实验室？
> 长 aPTT + 假阳性 VDRL/RPR + 血小板减少

> [!question]- Q21: PAH 三大类药？
> NO/PDE-5i (-afil) + Prostacyclin (-prost-) + Endothelin (-sentan)

> [!question]- Q22: CTEPH 治疗？
> **PEA + Riociguat**

> [!question]- Q23: 4 大儿童 benign 杂音？
> Still's, Pulmonic flow, PPS, **Venous hum**（唯一站立增强）

> [!question]- Q24: Pre/Post-ductal SpO2 解剖？
> Pre = 右手, Post = 足部
> Pre>Post >3% = Differential cyanosis

> [!question]- Q25: AR 关联综合征？
> Marfan, Loeys-Dietz, Ankylosing spondylitis, Syphilis, Bicuspid AV

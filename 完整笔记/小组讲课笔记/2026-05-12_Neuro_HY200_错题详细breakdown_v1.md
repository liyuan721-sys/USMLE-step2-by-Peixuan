### 修复方法（强制流程）

> [!warning] 从今天起强制执行的扫描流程

```
看完 stem 第一段
    ↓
不要立刻定诊断 ⛔
    ↓
扫 stem 所有细节 → 列出"症状清单"
    ↓
做"全症状交集" → 哪个病能解释最多？
    ↓
排除"次要解释"病（覆盖症状少的）
    ↓
看治疗选项 → 匹配最强解释力的病
```

### Trigger Words（看到下列词必须暂停扫描）

| Trigger word | 提示 | 为什么 |
|---|---|---|
| "Over the past X weeks/months" | 速度词 | 触发病程分类（急/亚急/慢）|
| "Patient is concerned about..." | 自知力 | 鉴别 Pseudodementia vs AD |
| "Lost interest in [hobby]" | 抑郁标志 | DSM-5 MDD 标准 |
| "Provoked by..." | 修饰语 | 锁定 startle/postural/exertional 等 |
| "Sometimes..." / "Occasionally..." | 频率词 | 区分波动 vs 持续 |
| "Current medications include..." | 用药史 | 解释症状（如 bradycardia 来自 β-blocker）|
| "History of..." | 基础病 | 风险因素谱 |
| 影像/EEG/CSF 描述 | 金标准证据 | 通常是诊断锁定钥匙 |

### 今天的 USMLE 元技能产出（6 条）

1. **元技能 #1 - 分母重设词识别**：年龄/基础病/遗传综合征/用药史 → 重设症状意义
2. **元技能 #2 - 慢性病管理 "add 不 switch"**：升级 = 加药，不是换药
3. **元技能 #3 - 症状交集 → 最强解释力的病**：不被单一症状带偏
4. **元技能 #4 - "下一步"题流程图答法**：流程图 + 排除已做 + 下一站
5. **元技能 #5 - 强制扫全 stem 细节**：看到主线不要反射性诊断
6. **元技能 #6 - 扫描 4 步算法 + 5 类关键词**：45 秒读题（详见专门手册）

## 🧠 错题 1：Pseudodementia 误选（Q-78yo Forgetfulness 5 weeks）

### 1. Stem 关键词识别

| Key Stem Clue（英文） | Why It Matters | 中文解析 |
|---|---|---|
| 78-year-old man | 老年男性，AD/抑郁高发 | 老年痴呆鉴别人群 |
| **5 weeks** of forgetfulness ⭐ | 速度太快 → 排除 AD | 时间词触发分母重设 |
| **Left front door wide open + tap water running** | 危险行为 / IADL 失能 | 严重认知问题 |
| **Wakes up 6-7 times/night** ⭐ | 睡眠紊乱 | MDD 标准 #3 |
| **Too tired to leave room** ⭐ | 疲乏 | MDD 标准 #5 |
| **Lost interest in piano** (months ago) ⭐ | 兴趣丧失 + 抑郁先于认知 | MDD 标准 #2 |
| Wandering at night | 严重定向障碍 | |
| MMSE not specified（题 stem 没明确给）| 但提示认知受损 | |
| **Psychomotor retardation** ⭐ | PMR | MDD 标准 #5 |
| **Blunt affect** ⭐ | 情感平淡 | 抑郁招牌 |
| Memory + attention + concentration all impaired | 多领域认知 | |
| Labs all normal (TSH/B12/folate) | 排除可治疗病因后还在 | |
| **Patient is concerned about memory lapses** ⭐⭐ | **有自知力** | Pseudo vs AD 金标准鉴别 |

> [!tip] 一句话锁定
> **78 岁 + 5 周（快）+ 病人自己关心 + 5 项 MDD 标准 + labs 正常** → **Pseudodementia (MDD)** → **SSRI (Citalopram)**

### 2. 机制理解

```
Major Depressive Disorder
        ↓
神经递质失衡（5-HT, NE, DA↓）
        ↓
前额叶 + 海马功能受影响
        ↓
执行功能↓ + 注意↓ + 记忆"看起来"↓
        ↓
**伪装成 Dementia** ← Pseudodementia
        ↓
但本质不是神经退行性
        ↓
治疗抑郁 → 认知 reverse ⭐
```

⭐ **核心病理生理**：抑郁导致**前额叶皮层活性↓**和**海马灰质萎缩**（应激相关）→ 临床表现像痴呆，但**可逆**。

### 3. 正确答案 + 核心概念

**Answer**: B - Citalopram

**Core Concept (EN)**: Late-life depression frequently presents as pseudodementia. Unlike true dementia, cognitive deficits in MDD are reversible with antidepressant treatment (typically SSRIs).

**核心概念**：老年期抑郁常表现为假性痴呆。与真性痴呆不同，**MDD 的认知缺陷在抗抑郁治疗后可逆**（首选 SSRI）。

**为什么是 Citalopram**：
1. **病人满足 5/9 MDD 诊断标准**（anhedonia + sleep + 疲乏 + PMR + 认知↓）
2. **病人有自知力**（concerned about memory）= 抑郁招牌
3. **病程 5 周**（快）= 不可能是 AD
4. **labs 正常**（排除可治疗的甲减/B12/叶酸）
5. **SSRI 是 MDD 一线**，Citalopram 老年人 ≤ 20mg/d（QT 延长警告）

### 4. 干扰项逐项击破

| 选项 | 是什么 | 为什么错 | USMLE 陷阱 |
|---|---|---|---|
| **A. Donepezil** ⭐ 我选的 | AChE 抑制剂，AD 一线 | AD 起病数年（不是 5 周）+ 病人缺乏自知力（不是自己担心）+ 抑郁征不是 AD 特征 | "老人健忘 = 反射性 AD" |
| **B. Citalopram** ✅ | SSRI | 完美匹配 MDD + 老年人 ≤ 20mg/d | |
| **C. Aspirin** | 抗血小板 | VaD 治疗，但 stem 无 CV 风险 + 无局灶征 + 无 MRI 白质病变 | "老人 + 健忘 = 反射性 VaD" |
| **D. VP shunt** | 治 NPH | NPH 三联（wet/wobbly/wacky）一个都没（没尿失禁、没磁性步态、没脑室扩大）| |
| **E. Memantine** | NMDA 拮抗剂，中重度 AD | 根本诊断错（不是 AD）+ Memantine 是 add-on | |

### 5. 我为什么错（个性化反思）⭐ 最关键

**我选了**：A. Donepezil（基于 AD 诊断）

**我的思路**：
"78 岁老人 + 5 周健忘" → 反射性想到"老人健忘 = AD" → 选 Donepezil（AD 一线）

**根本错因**（多重）：
- ☑ **"分母重设词"识别失败**：**5 周**（不是数年）应触发"AD？太快了"的怀疑 — 元技能 #1 失效
- ☑ **单一症状导向 + 没做症状交集**：只抓了"健忘"，**忽略了 5 个抑郁标志**（PMR/blunt affect/兴趣丧失/睡眠/疲乏）— 元技能 #3 失效
- ☑ **没识别"病人态度"陷阱**："**病人自己关心**"是 pseudodementia 金标准，AD 通常**缺乏**自知力
- ☑ **没用"可治疗痴呆 8 大" schema**：今天**早些时候**才学的 schema，这道题应该启用 → "labs 都正常 + 抑郁征 + 速度快" = 直指 Pseudodementia
- ☑ **没考虑"反射性诊断"的元问题**：看到主线（老人 + 健忘）就反射选 AD 治疗，没暂停扫全 stem — 元技能 #5 失效

**核心陷阱**：**老人 + 健忘 ≠ 自动 AD**——必须先排除 8 大可治疗病因，pseudodementia 是 #1 高频。

### 6. High-Yield Summary Table — Pseudodementia vs AD 终极鉴别

| 特征 | **Pseudodementia (MDD)** ⭐ | **AD** |
|---|---|---|
| 起病速度 | **数周-月**（如本题 5 周）| 数年缓慢 |
| 主诉来源 | **病人自己**关心 | 家属带来（病人缺乏自知力）|
| 自知力 | **过度自责 + 担心** | 缺乏（"我没问题"）|
| 回答模式 | "**我不知道**"（放弃式）| 编造（confabulation）/ 错答 |
| 抑郁征 | **核心**（≥ 5/9 MDD）| 可有但非核心 |
| PMR / blunt affect | **典型** | AD 早期通常无 |
| 睡眠 | **失眠 / 早醒** | 通常保留至晚期 sundowning |
| 食欲 / 体重 | **↓**（典型）| 通常保留 |
| 症状演变 | **抑郁先于认知** | 认知首发 |
| 认知 testing 配合 | 配合差，努力少 | 努力尝试，答错 |
| MMSE | 可低，random pattern | 系统性下降 |
| 影像 | MRI 通常无明显异常 | 海马 + 颞顶萎缩 |
| 治疗 | **SSRI** 4-6 周 | Donepezil / Memantine |
| 预后 | **可逆** ⭐ | 不可逆 |
| 关键钩子 | "Concerned about memory + lost interest" | "Got lost in own neighborhood" |

### 7. Memory Hook（记忆挂钩）⭐

> [!success] 
> **"Pseudodementia 5 个'特别'"**
> 
> 1. **起病特别快**（数周-月）
> 2. **病人特别担心**（主动抱怨 + 自知力存在）
> 3. **回答特别消极**（"我不知道"放弃式）
> 4. **抑郁特别明显**（PMR / blunt / 兴趣丧失 / 睡眠 / 疲乏）
> 5. **特别可治**（SSRI 4-6 周 → 认知改善）

> [!tip] 
> **看到 "concerned about memory" + "lost interest in [hobby]" + "sleep disturbance"** → **0.5 秒锁 Pseudodementia → SSRI**

> [!tip] DSM-5 MDD 9 大标准记忆（≥ 5 项 + ≥ 2 周）
> **"SIG E CAPS + 抑郁情绪"** 助记符：
> - **S**leep disturbance（睡眠↓/↑）
> - **I**nterest loss (anhedonia)
> - **G**uilt / worthlessness
> - **E**nergy loss / fatigue
> - **C**oncentration↓
> - **A**ppetite/weight change
> - **P**sychomotor agitation/retardation
> - **S**uicidal ideation
> - + Depressed mood

## 🧠 错题 2：CJD 误选 I-FTD（Q-57yo 2 months mental decline）

### 1. Stem 关键词识别

| Key Stem Clue（英文） | Why It Matters | 中文解析 |
|---|---|---|
| 57-year-old man | 中老年 | |
| Worsening mental status **over past 2 months** ⭐ | **快速进展** (RPD) | 数月 = CJD 速度 |
| Initially memory lapses → 3 weeks ADL impaired ⭐ | **极快**（3 周内不能 ADL）| RPD 关键证据 |
| Yesterday: lost going to nearby post office | 急性认知恶化 | |
| HTN treated with HCTZ | 干扰项 | |
| Strength normal, reflexes 2+ symmetric | 排除 stroke/UMN | |
| **Diffuse involuntary muscle jerking** ⭐ | "**Jerking**" = myoclonus（不是震颤！）| 易混淆词 |
| **Provoked by loud noises** ⭐⭐ | **Startle myoclonus** = CJD 招牌 | 修饰语必看 |
| Verbally uncommunicative + blunt affect | Akinetic mutism（CJD 晚期）| |
| Labs normal | 排除代谢性 | |
| **EEG: Triphasic periodic sharp-wave complexes** ⭐⭐⭐ | **CJD EEG 金标准** | 题 stem 最后旁白直接给 |

> [!tip] 一句话锁定
> **快速进展（2 月）+ Startle myoclonus + EEG 周期性尖波** = **CJD**（三个金标准齐全）

### 2. 机制理解

```
PrP^C (正常 prion 蛋白, α-helix)
        ↓ 突变 / 自发 / 接触污染
PrP^Sc (异常 prion, β-pleated sheet)
        ↓
抗蛋白酶（不被降解）
        ↓
聚集成淀粉样斑块
        ↓
神经元 spongiform 改变（海绵状空泡）
        ↓
↓↓↓ 快速进展性痴呆 ↓↓↓
        ↓
皮层 + 基底节 + 丘脑 + 小脑都受累
        ↓
临床三联：
- RPD（认知下降）
- Startle myoclonus（皮层兴奋性↑）
- Cerebellar ataxia
        ↓
12 月内死亡
```

### 3. 正确答案 + 核心概念

**Answer**: D - Creutzfeldt-Jakob Disease

**Core Concept (EN)**: CJD is a fatal prion disease characterized by rapidly progressive dementia (weeks-months), startle-provoked myoclonus, and characteristic EEG findings (triphasic periodic sharp-wave complexes). CSF 14-3-3 protein and RT-QuIC support diagnosis.

**核心概念**：CJD 是致死性 prion 病，三联特征：**RPD + startle myoclonus + EEG 周期性尖波**。

### 4. 干扰项逐项击破

| 选项 | 适用病 | 为什么错（精确鉴别点）|
|---|---|---|
| **A. AD** | 阿尔茨海默 | AD 进展**数年**（不是 2 月）+ AD 早期无 myoclonus（晚期才有，且不是 startle）|
| **B. PD** | 帕金森 | PD 必备 bradykinesia + rigidity + tremor，本题**一项都没有**；PD 震颤是节律性来回 ≠ jerking |
| **C. HD** | 亨廷顿 | HD 标志 = **chorea**（舞蹈）≠ myoclonus；HD 数年进展（不是 2 月）；HD myoclonus 极少 |
| **D. CJD** ✅ | 朊蛋白病 | 完美匹配 |
| **E. ALS** | 肌萎缩侧索硬化 | ALS 必备 UMN + LMN + fasciculations + 肌力↓，本题"strength normal" |
| **F. MS** | 多发性硬化 | MS = R&R 病程 + 视神经炎 + Lhermitte + UMN 征，本题无 |
| **G. Wilson** | 肝豆状核变性 | Wilson **< 35 岁** + K-F 环 + 肝病 + 拍翅震颤，本题 57 岁全不符 |
| **H. NPH** | 正常压力脑积水 | NPH 必备 Wet/Wobbly/Wacky + 脑室扩大，本题无 gait + 无 incontinence + 无 EEG 改变 |
| **I. FTD** ⭐ **我选的** | 额颞叶痴呆 | **FTD 数年进展**（不是 2 月）+ FTD 招牌"行为/语言改变"（不是 mutism + EEG 异常）+ FTD 无 myoclonus + 无 EEG 周期性尖波 |

⚠️ **I 选项陷阱**（你为什么选这个）：
- 看到 "diffuse involuntary muscle jerking" → 误读为"肢体不自主动作"→ 联想帕金森方向
- 看到 "blunt affect" → 进一步强化帕金森方向（masked facies）
- 帕金森不在选项 → 选"最近邻" = FTD（也能解释情感改变）

### 5. 我为什么错（个性化反思）⭐

**我选了**：I. FTD

**我的思路**（双因子结构）：
- 因子 1：看到"**diffuse involuntary muscle jerking**" → 联想帕金森样症状（震颤/僵直）
- 因子 2：看到"**blunt affect**" → 进一步强化帕金森方向（误以为是 masked facies）
- 帕金森不在选项 → 选"邻近"的 FTD（错误的"最近邻"匹配）

**根本错因**：
- ☑ **症状词识别错误**：myoclonus（闪电样、瞬间）≠ 震颤（节律性来回）≠ 舞蹈（流畅连续）
- ☑ **没注意修饰语**："**provoked by loud noises**" 是 startle myoclonus = CJD 招牌，被忽略
- ☑ **单一症状导向**：被 "blunt affect" 和 "jerking" 两个信号带偏方向，没做症状交集
- ☑ **忽略 EEG 描述**：stem 旁白 "triphasic periodic sharp wave complexes" = CJD 锁定金标准，被漏看
- ☑ **没识别 "2 months" 速度词**：太快 = 不可能是 AD/FTD/PD/HD（这些都是数年）

**核心陷阱**：
- **"jerking" ≠ 震颤 / 舞蹈** = **myoclonus**（独立症状概念，必须建立）
- **"Blunt affect" 是多义症状**（PD/抑郁/精分/FTD/CJD/AD 晚期都可能），不能单独锁定方向

### 6. High-Yield Summary Table — 5 种不自主运动鉴别 ⭐

| 不自主运动 | 英文 | 特点 | 标志性疾病 |
|---|---|---|---|
| **震颤** | Tremor | **节律性、来回振荡** | 静止性→PD / 意向性→小脑 / 姿势性→ET |
| **舞蹈** | **Chorea** | **不规则、无目的、流畅快速** | **Huntington** / Sydenham / SLE |
| **手足徐动** | Athetosis | **缓慢扭转、蛇行** | HD 晚期 / 脑瘫 |
| **肌阵挛** ⭐ | **Myoclonus** | **突发、短促、闪电样** | **CJD** ⭐ / AD 晚期 / 缺氧后 / 癫痫 |
| **抽搐** | Tic | **可短暂抑制**的刻板动作 | Tourette |
| **肌张力障碍** | Dystonia | **持续性、扭转性收缩** | 急性 EPS / 肌张力障碍综合征 |

### CJD vs FTD 镜像对照表（这两题选项一样，答案不同）

| 维度 | **CJD（本题，我错选 FTD）** | **FTD（下一道我选对的）** |
|---|---|---|
| 年龄 | 50-60+ | 50-60 岁（典型）|
| 病程 | **数周-月**（< 12 月致死）| **数年**（6-9 年生存）|
| 首发 | 快速进展认知 + 神经体征 | **行为/性格改变** |
| 记忆 | 严重受累 | **相对保留**（MMSE 正常）|
| 运动 | **Startle myoclonus**（招牌）| 晚期 parkinsonism |
| 饮食 | 无特殊 | **Hyperorality + 偏甜**（招牌）|
| 社交 | 进展为 mutism | **去抑制 / 不当社交** |
| EEG | **周期性尖波**（标志）| 非特异 |
| MMSE | 可低，但执行功能为先 | **正常或近正常** |

### 7. Memory Hook ⭐

> [!success] 
> **"5 种不自主运动 5 个画面"**
> 
> - Tremor (震颤) = **钟摆来回**（节律性）
> - Chorea (舞蹈) = **跳舞流畅**（不规则）

## 🧠 错题 3：Syringomyelia + Horner（Q-62yo Cape-like sensory loss）

### 1. Stem 关键词识别

| Key Stem Clue（英文） | Why It Matters | 中文解析 |
|---|---|---|
| 62-year-old woman with **T2DM** | 干扰项（让你想 DM neuropathy）| 分母重设陷阱 |
| **1-year progressive paresthesias in upper limbs** | 慢性进展 | 排除急性创伤 |
| **Accidentally burned fingers on hot dishes several times** ⭐ | **Painless burns** = 痛温觉↓ + 触觉保留 | "分离性感觉障碍" |
| **MVC 3 years ago (whiplash)** ⭐ | 数年前创伤 → 创伤后空洞 | 病因 |
| **Absent temperature sensation + normal fine touch over upper extremities + upper chest** ⭐⭐ | **披肩样（cape-like）分离性感觉障碍** | Syringomyelia 招牌 |

> [!tip] 一句话锁定
> **车祸数年后 + 慢性进展 + 手部无痛烫伤 + 上肢/上胸披肩痛温↓ + 触觉正常** → **Syringomyelia**

**题目实际问的**：Further evaluation most likely to show? → **病程演变下一步**

### 2. 机制理解

```
颈髓中央管区域 → 空洞形成 → 逐渐扩大（从内向外）

⭐ Syringomyelia 5 幕剧（从内向外烂）：

阶段 1（最早）⭐
Anterior white commissure（前白连合）
← 脊髓丘脑束交叉的纤维在此通过
→ 双侧痛温觉↓（**披肩样 cape-like**）
→ 触觉、本体觉、运动、反射 全部正常

阶段 2（再扩大）
Lateral horn（外侧角，T1-T2）
← 一级交感神经元（去往面部的下行交感）
→ 同侧 **Horner 综合征**（ptosis/miosis/anhidrosis）⭐ 题目答案

阶段 3（再扩大）
Anterior horn（前角细胞）
→ 上肢 **LMN 征**（hyporeflexia / 萎缩 / flaccid paresis）
→ biceps reflex ↓（不是 ↑！）

阶段 4（晚期）
Lateral corticospinal tract
→ 损伤平面以下 UMN 征
→ 下肢 hyperreflexia + Babinski +

阶段 5（最晚）
Sacral CST 最外侧
→ Sacral sparing 才丢
```

⭐ **核心解剖逻辑**：**空洞从中央向外扩展 → 谁离中央近，谁先废**

### 3. 正确答案 + 核心概念

**Answer**: E - Drooping of the eyelid (Horner syndrome)

**Core Concept (EN)**: Syringomyelia expands centrifugally from the central canal. As the syrinx enlarges from anterior white commissure (stage 1) to lateral horn (stage 2), it damages first-order sympathetic neurons (T1-T2), producing ipsilateral Horner syndrome.

**核心概念**：脊髓空洞**从中央向外扩张**。从前白连合（阶段 1）扩到外侧角（阶段 2）时，损伤 T1-T2 一级交感神经元 → 同侧 Horner。

**为什么是 E（眼睑下垂）**：
- Stem 现状 = 阶段 1 完成（披肩痛温↓）
- "Further evaluation most likely to show" = 病程**下一步** = 阶段 2 征象
- 阶段 2 = Horner 综合征 = **ptosis（眼睑下垂）** + miosis + anhidrosis

### 4. 干扰项逐项击破

| 选项 | 适用情境 | 为什么错（病程时机）|
|---|---|---|
| **A. Exaggerated biceps reflex（↑）** | 颈髓 UMN 征 | **方向反了**！Syrinx 在 C5-T1 损伤**前角细胞** → biceps reflex ↓（LMN），不是 ↑。UMN 征出现在**损伤平面以下**（下肢），不在上肢 |
| **B. ↓ MMSE** | 大脑皮层 | Syringomyelia **不影响认知**。MMSE↓ 见于 **B12 SCD / Tabes paralytic dementia** |
| **C. ↓ Anal wink reflex** | S2-S4 → pudendal n. | **Sacral sparing** 是 Syringomyelia / Central cord 的特征！骶段在 CST 最外侧，**最后被累及** |
| **D. Jaw deviation** | CN V 运动核（pons）| CN V 运动核在脑桥，syrinx 颈段累及不到。只有 spinal trigeminal nucleus（感觉核）能延伸到 C2 |
| **E. Drooping of the eyelid** ✅ | T1-T2 一级交感 | 完美匹配（阶段 2）|

### 5. 我为什么错（个性化反思）⭐

**我的知识缺口**（你的原话）："以前只知道披肩征，从来没听过 Horner 跟这个有关系"

**根本错因**：**真实知识缺口型**（不是答题方法问题）
- ☑ 我对 Syringomyelia 的知识结构是**"一个症状的疾病"**（披肩样痛温↓）
- ☑ 没有建立"**时间轴疾病，会发展**"的模型
- ☑ 没有意识到 **T1-T2 lateral horn = 一级交感** 这个关键解剖关联
- ☑ "Horner" 在我脑中只关联 PanCoast / 颈动脉夹层 / Wallenberg，没关联 Syringomyelia

**核心陷阱**：
**单症状疾病模型 vs 病程演变疾病模型**——很多疾病不是"一个症状的疾病"，而是"按时间发展的多症状疾病"

### 6. High-Yield Summary Table — Horner 3 级定位 ⭐

| 级别 | 损伤部位 | 病因（USMLE 高频） | 伴随征 |
|---|---|---|---|
| **一级（中枢）** | 下丘脑 → T1-T2 lateral horn | **Syringomyelia** ⭐ / **Lateral medullary (Wallenberg)** / 脑干卒中 / MS | 中枢神经系统其他症状（脊髓/脑干征）|
| **二级（节前）** | T1-T2 → stellate → SCG | **Pancoast tumor**（肺尖癌）⭐ / 颈部手术 / 颈胸创伤 / 主动脉瘤 | 上肢痛、手内在肌萎缩（C8-T1）、肺尖肿块 |
| **三级（节后）** | SCG → ICA → 眼 | **Carotid artery dissection** ⭐ / 海绵窦病变 / Cluster headache | 同侧颈痛/头痛、可无 anhidrosis（节后纤维分支不全）|

⭐ **节后 Horner 可无 anhidrosis** = "Partial Horner"（汗腺纤维跟颈外动脉分支走了）

### Syringomyelia vs Central Cord Syndrome（高频鉴别）

| 特征 | **Syringomyelia** | **Central Cord Syndrome** |
|---|---|---|
| 起病 | **慢性**（数月-年）| **急性**（创伤即刻）|
| 诱因 | **Chiari I**（最常见）/ whiplash | Cervical spondylosis + 过伸伤 |
| 年龄 | 年轻人为主 | 老人为主 |
| 典型 stem | "Painless burns on hands" | "Fell, weak arms, neck pain" |
| 病理 | 中央**空洞**（cyst）| 中央灰质水肿/出血 |
| 影像 | MRI 见**空腔**（高 T2 信号）| MRI 见**水肿/挫伤** |
| 共同点 | 都是手 > 腿、披肩痛温↓、上肢 LMN | （所以课件放对照图）|

### 7. Memory Hook ⭐

> [!success] 
> **"Syringomyelia 五幕剧"**（从内向外烂）
> 
> 1️⃣ **披肩痛温**（前白连合）— "烫到手不知道"
> 2️⃣ **Horner**（外侧角）— "眼皮垂下来"
> 3️⃣ **上肢 LMN**（前角）— "胳膊变软"

## 🧠 错题 4：Leriche 综合征（Q-49yo Erectile dysfunction + claudication）

### 1. Stem 关键词识别

| Key Stem Clue（英文） | Why It Matters | 中文解析 |
|---|---|---|
| 49-year-old man + **35 PY smoking** + **BMI 34** + **BP 169/98** + 父冠心病 + 母 SAH | 全身动脉粥样硬化高危画像 | 经典 CV 高危 |
| **6 个月 ED + sildenafil 无效** ⭐ | ED 不是血管平滑肌 cGMP 问题，是**动脉灌注不够** | PDE5i 没血流可扩 |
| **腰背 + 大腿痛 + 走路加重 + 休息缓解** | Claudication 模式 | |
| **建筑工人 + 长距离走时下肢无力** | Exertional → 缺血性 | |
| HbA1c 6.2%（< 6.5%）| **排除 DM**（题目埋的反陷阱）| 分母重设 |
| Cr 正常 | 排除 CKD 钙化血管 | |

> [!tip] 一句话锁定
> **男性 + 双侧大腿/臀 claudication + ED + sildenafil 无效 + 全身动脉粥样硬化高危** → **Leriche Syndrome**（主髂分叉闭塞）→ 答 **↓ 双侧股动脉搏动 (Decreased bilateral femoral pulses)**

### 2. 机制理解

```
吸烟 35 PY + HTN + 肥胖 + 家族 CAD
        ↓
全身动脉粥样硬化
        ↓
**Aortoiliac 分叉处**闭塞（Leriche）
        ↓
┌─────────────┬──────────────┐
↓             ↓              ↓
内髂动脉↓     臀肌/股深↓     双侧股动脉↓
（阴茎海绵    （臀 + 大腿       （PE 体征 ← 答案）
体灌注↓）     claudication）
↓
**ED**（sildenafil 无效，
   因为不是 cGMP 问题，
   是上游血流问题）
```

### 3. 正确答案 + 核心概念

**Answer**: A - Decreased bilateral femoral pulses

**Core Concept (EN)**: Leriche syndrome = aortoiliac occlusive disease. Triad: (1) bilateral buttock/thigh claudication, (2) erectile dysfunction (internal iliac → cavernosal artery insufficiency), (3) decreased/absent femoral pulses.

**核心概念**：Leriche 综合征 = 主髂闭塞。三联：
1. **双侧大腿/臀部 claudication**（臀肌 + 大腿肌缺血）
2. **ED**（内髂动脉 → 海绵体灌注↓）
3. **股动脉搏动↓/消失**（PE 体征）

### 4. 干扰项逐项击破

| 选项 | 是什么 | 为什么错 | USMLE 陷阱 |
|---|---|---|---|
| **A. ↓ Bilateral femoral pulses** ✅ | Leriche 体征 | 完美匹配 | |
| **B. INO (Internuclear ophthalmoplegia)** | MLF 病变 → MS / stroke | MS 是 young female + R&R；此题双侧慢性 + 强血管危险因素 | "看到 ED 就跳 MS" 经典误区 |
| **C. Palmar/plantar rash** | Secondary syphilis | 4 性伴 + 套不规律是诱饵；二期梅毒不引起 claudication | **性史是 red herring** |
| **D. JVD** | 右心衰征 | 此人 CAD 风险因素 ≠ 当前 HF；无水肿/dyspnea/啰音 | **风险因素 ≠ 当前疾病** |
| **E. ↓ Anal tone** | 脊髓压迫 / 马尾 | Spinal stenosis 是**坐下/弯腰缓解**；本题**站立时缓解**（不需要弯腰）| Vascular vs Neurogenic 鉴别 |
| **F. Painless foot ulcer** | DM autonomic neuropathy | **HbA1c 6.2% < 6.5%** 排除 DM；DM 溃疡 painless，**PAD 溃疡 painful** | HbA1c 数字埋的陷阱 |

### 5. 我为什么错（个性化反思）⭐

**我选了**："没思路"

**我的状态**（你的原话）：**"完全不认识 Leriche 这个名词"**

**根本错因**：**Schema 缺失型**（不是答题方法问题）
- ☑ 没有建立"PAD 三联征"的 schema：ED + 大腿痛 + sildenafil 无效 → 不能 cluster 成一个诊断
- ☑ 没有"aortoiliac 闭塞同时影响 3 条下游血管 → 3 个症状" 的解剖映射
- ☑ Stem 信号无法组成 cohesive picture → 任何选项都解释不掉全部信息 → 选不出来

**核心陷阱**：**"看见 ED 不要只想泌尿/心理"** → 想血管/神经/激素四大方向，**sildenafil 无效**强烈提示血管源

### 6. High-Yield Summary Table — Vascular vs Neurogenic Claudication ⭐ 金表

| 特征 | **Vascular (PAD/Leriche)** | **Neurogenic (Lumbar spinal stenosis)** |
|---|---|---|
| 病因 | 动脉粥样硬化闭塞 | 椎管狭窄压迫马尾/神经根 |
| 走路加重 | ✅ | ✅（两者都加重，区分不了）|
| 休息缓解 | ✅ **站着就缓解** | ✅ **需要坐下/弯腰**才缓解 |
| 站立不动 | 缓解 | ❌ 不缓解，甚至加重 |
| 弯腰/坐下/上坡 | 无关 | ✅ 缓解（增大椎管）|
| 骑自行车 | 加重（耗氧↑）| ✅ **不加重**（弯腰位）|
| Pulse | ↓ | 正常 |
| ED 来源 | 海绵体缺血 | 神经根受压（少见）|
| 影像 | CTA / MRA | MRI 腰椎 |

⭐ **黄金鉴别口诀**：**"站立不动就缓解 = Vascular；坐下/弯腰才缓解 = Neurogenic"**

### PAD 治疗优先级（USMLE 必考顺序）

```
所有 PAD 患者（无论症状）：
1. 戒烟 > 一切（最重要 risk factor 控制）
2. Antiplatelet (ASA 或 clopidogrel) + Statin
   = 降低 MI/Stroke 死亡率（不是改善走路）
3. HbA1c < 7%、BP 控制

Claudication（无 rest pain）：
- 首选：Supervised exercise therapy ⭐ 高频考点
- 二线：Cilostazol（PDE3 抑制剂；HF 患者禁用）

Critical limb ischemia / 顽固症状：
- Revascularization（PTA 或 bypass）
```

### 7. Memory Hook ⭐

> [!success] 
> **"主动脉分叉处堵车，三条路同时瘫痪"**
> 
> - **内髂** → ED
> - **臀肌** → 屁股痛
> - **股总** → 腿痛 + 脉弱
> 
> 看到男性 + 双侧大腿臀痛 + ED + 双股动脉摸不到 → **0.5 秒锁 Leriche**

> [!tip] PAD 锁定钥匙
> "**Sildenafil 无效的 ED**" → 想**血管源**（不是 cGMP 问题）
> "**走路加重 + 站立缓解 + 脉↓**" → **Vascular** claudication
> "**坐下/弯腰缓解**" → **Neurogenic** claudication

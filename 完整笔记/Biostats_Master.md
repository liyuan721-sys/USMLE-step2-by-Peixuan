## 一、2×2 表 — 所有公式的根

记忆口诀:**"行是测试,列是疾病"**

```
                Disease (+)     Disease (−)
  Test (+)          a               b
  Test (−)          c               d
```

- **a** = 真阳性(TP) — 有病且测阳
- **b** = 假阳性(FP) — 无病却测阳
- **c** = 假阴性(FN) — 有病却测阴
- **d** = 真阴性(TN) — 无病且测阴

→ **做任何题前先确认行/列方向**,UW 常把方向反过来骗你。

### 核心公式(必背)tag: #薄弱点 

| 公式                                | 写法                               |
| --------------------------------- | -------------------------------- |
| **Sensitivity**                   | `a / (a + c)`                    |
| **Specificity**                   | `d / (b + d)`                    |
| **PPV**                           | `a / (a + b)`                    |
| **NPV**                           | `d / (c + d)`                    |
| **Prevalence**                    | `(a + c) / total`                |
| **Accuracy**                      | `(a + d) / total`                |
| **Relative Risk (RR)**            | `[a/(a+b)] / [c/(c+d)]`          |
| **Odds Ratio (OR)**               | `(a × d) / (b × c)`              |
| **Attributable Risk (AR)**        | `a/(a+b) − c/(c+d)`              |
| **Absolute Risk Reduction (ARR)** | `c/(c+d) − a/(a+b)`(对照组率 − 治疗组率) |

## 二、诊断试验 — Sn / Sp / PPV / NPV / LR / Accuracy / ROC

### Sensitivity(灵敏度,真阳性率)

- 在所有**确实有病**的人中,测试结果为**阳性**的比例
- **SnNout** — Sn 高,**阴性**结果可用来**排除**疾病(用于**筛查**)
- 高 Sn → 假阴性少,但假阳性多
- 比喻:像**渔网**,抓得全(包括杂物)

### Specificity(特异度,真阴性率)

- 在所有**确实无病**的人中,测试结果为**阴性**的比例
- **SpPin** — Sp 高,**阳性**结果可用来**确诊**(用于**确诊**)
- 高 Sp → 假阳性少,但假阴性多

### 核心规则:**看阳性组变大还是变小**

- **阳性组变大** → Sn 增加(放进更多人,漏掉的更少)
- **阳性组变小** → Sp 增加(更挑剔,误诊更少)

### 阈值变化(以糖尿病诊断空腹血糖为例)

| 阈值动作 | 阳性组 | Sn | Sp |
|---|---|---|---|
| **升高阈值**(126 → 130) | 变小 | ↓ | ↑ |
| **降低阈值**(126 → 100) | 变大 | ↑ | ↓ |

### PPV / NPV(预测值)

- **PPV** — 测阳的人里,**真有病**的概率
- **NPV** — 测阴的人里,**真没病**的概率

| 影响因素 | PPV | NPV |
|---|---|---|
| **Prevalence ↑** | **↑**(同向) | **↓**(反向) |
| 假阳性 ↑ | ↓ | — |
| 假阴性 ↓ | — | ↑ |

> **关键认知:** Sn / Sp 是检测的**固有属性**,不受患病率影响;PPV / NPV 跟着患病率"摇摆"。

### 检后概率(Post-test Probability)

- **测阴个体的患病概率 = 1 − NPV**
- **检前概率(Prevalence)越高 → NPV 越低**
- **临床应用:** 高危患者的阴性结果**不足以排除诊断**,需要更精准的"金标准"检查

### Likelihood Ratio(似然比) — 不受患病率影响 tag: #薄弱点 

似然比将**检前概率**转化为**检后概率**,衡量"病人比健康人更容易出现这种检测结果几倍"。

- **LR+ = Sn / (1 − Sp)** — 阳性结果的诊断价值
这个检测在**有病人**中阳性的概率，**除以**在**无病人**中阳性的概率
LR+ = P(test+ | disease+) / P(test+ | disease−)
    = Sensitivity / (1 − Specificity)
    = 真阳性率 / 假阳性率
  - **LR+ 越大 → 阳性结果越能"确诊"**
  - **LR+ > 10** → highly specific test → 强诊断价值 → 可确诊
  - **LR+ = 10** ≈ 患病概率提升 ~45%
- LR+ 5-10  → 中等价值
- LR+ 2-5   → 弱价值
- LR+ < 2   → 几乎无用
- LR+ = 1 → 检测毫无价值（阳性 ≡ 阴性概率）
  
- **LR− = (1 − Sn) / Sp** — 阴性结果的诊断价值
  - **LR− < 0.1** → highly sensitive test → 强阴性结果,可排除
  - **LR− = 0.1** ≈ 患病概率降低 ~45%
- **LR = 1** → 测试完全没用,做了白做

### 一句话区分(死记)

| 已知 | 求 | 看 |
|---|---|---|
| **患病状态** | 检测结果概率 | **Sn / Sp** |
| **检测结果** | 真实患病概率 | **PPV / NPV** |

## 🔑 诊断指标核心 — 2 句话搞定一切

### 核心 2 句话（必背）

> **Sensitivity = 有病阳性**
> **Specificity = 无病阴性**

### "1 减"逻辑（推出 4 格）

| | Disease (+) | Disease (−) |
|---|-------------|-------------|
| **Test (+)** | Sens（有病阳性）| 1 − Spec（无病阳性）|
| **Test (−)** | 1 − Sens（有病阴性）| Spec（无病阴性）|

### LR 公式（同行相除）

> **LR+ = 阳性行**的两组比值相除
> **LR− = 阴性行**的两组比值相除

```
LR+ = (有病阳性) / (无病阳性) = Sens / (1−Spec)
LR− = (有病阴性) / (无病阴性) = (1−Sens) / Spec
```

### 受 prevalence 影响吗？

| 指标 | 是否受 prevalence 影响 |
|------|---------------------|
| Sens, Spec, LR+, LR− | ❌ **不受影响** |
| PPV, NPV | ✅ **受影响** |

**记忆**：S 和 L 不受影响，P 和 N 受影响

### 经典数字气质

| 选项 | 可能是什么 |
|------|-----------|
| 0.1-0.9 之间，小数字 | LR−（< 1 表示阴性可排除）|
| 5-100 | LR+（> 10 是强诊断）|
| ~75%, ~85% | Sens, Spec, PPV, NPV |

#考前必看 #核心概念

### Accuracy(准确度)

- `Accuracy = (TP + TN) / 总人数 = (a + d) / total`
- 受患病率加权影响,CK 较少单独考

### ROC 曲线 + AUC

ROC 曲线绘制:**纵轴 = Sensitivity**,**横轴 = 1 − Specificity**(不是 Specificity 本身!)

- **左上角 (0, 1)** = 100% Sn + 100% Sp = 完美试验(诊断界的"神")
- **右下角对角线** = 随机瞎猜
- **最佳切点(约登指数最大点)** = 曲线左上角最"凸"的点

**AUC(曲线下面积)解读:**

| AUC 范围 | 诊断价值 |
|---|---|
| **= 0.5** | 完全没用(瞎猜) |
| 0.5 – 0.7 | 较低 |
| 0.7 – 0.9 | 中等 |
| **> 0.9** | 极高 |
| **= 1.0** | 完美(现实不存在) |

### 🔥 CK 高频陷阱(诊断试验)

- **陷阱 1:** ROC 横轴是 **1 − Sp**,不是 Sp 本人 — UW 常画错坐标骗你
- **陷阱 2:** 认为 AUC 跟患病率有关 — 错!AUC 是**试验固有属性**,**不受患病率影响**
- **陷阱 3:** 两条 ROC 曲线交叉 — AUC 即使相等,临床选择不同(筛查选左侧陡,确诊选右侧高)
- **陷阱 4:** 题目让你"调阈值",别纠结升降,直接看**阳性组变大还是变小**

## 三、流行病学指标

| 指标 | 定义 | 适用场景 |
|---|---|---|
| **Prevalence(患病率)** | 某时间点现存病例数 / 总人口 | 描述疾病负担(横断面研究) |
| **Incidence(发病率)** | 某期间新发病例数 / 风险人口 | 描述风险(队列研究) |
| **Mortality rate** | 死亡数 / 人群 | 人群死亡风险 |
| **Case-fatality rate** | 病死数 / 病例数 | 疾病的致死性 |
| **Standardized rate** | 校正人口结构差异后的率 | 跨人群比较 |
| **Prevalence Ratio (PR)** | 暴露组患病率 / 非暴露组患病率 | 比较人群分布 |

### 关键关系(必背)

**Prevalence ≈ Incidence × Duration**

- **慢性病** → duration 长 → prevalence 高(如糖尿病)
- **急性致死病** → duration 极短 → prevalence 低(如狂犬病)
- **治愈率提升** → duration 短 → prevalence 下降(并不意味着 incidence 下降!)

## 四、风险与效应指标(RR / OR / ARR / RRR / NNT / NNH)

### 概念分清:风险 vs 比值

| 概念 | 定义 | 例子(10 人吸烟,2 人得病) |
|---|---|---|
| **Risk(风险/概率)** | 事件人数 / **总人数** | 2/10 = 20% |
| **Odds(比值)** | 事件人数 / **未发生人数** | 2/8 = 0.25 |

### RR vs OR — 设计决定指标

| 指标 | 公式 | 适用研究 | 含义 |
|---|---|---|---|
| **RR(相对危险度)** | 暴露组风险 / 非暴露组风险 | **Cohort、RCT** | 暴露后患病风险变成原来几倍 |
| **OR(比值比)** | 暴露组 Odds / 非暴露组 Odds = `(a×d)/(b×c)` | **Case-control**(及 logistic regression 直接产出) | 通过比值比例**估算**关联强度 |

记忆:**Cohort → Count risk → RR**;**Case-control → OR**(口诀:"you take the **case** to the **OR**")。

> **罕见病例外:** 如果疾病在人群中**罕见**(发病率 <10%),case-control 算出的 **OR ≈ RR**。

### 风险变化指标(RCT 必懂) tag: #薄弱点

| 缩写     | 全称                              | 中文            |
| ------ | ------------------------------- | ------------- |
| **Re** | **R**isk in **e**xposed group   | **暴露组**的疾病风险  |
| **Ru** | **R**isk in **u**nexposed group | **不暴露组**的疾病风险 |
| **Rc** | **R**isk in **c**ontrol group   | **对照组**的疾病风险  |
| **Rt** | **R**isk in **t**reated group   | **治疗组**的疾病风险  |

| 暴露情境                           | 治疗情境                         |
| ------------------------------ | ---------------------------- |
| **Re** (Risk in **e**xposed)   | **Rt** (Risk in **t**reated) |
| **Ru** (Risk in **u**nexposed) | **Rc** (Risk in **c**ontrol) |

ARP ↔ RRR  （公式镜像：百分比形式）
ARI ↔ ARR  （公式镜像：绝对差值）
NNH ↔ NNT  （公式镜像：人数）

3 对镜像 × 2 个情境（暴露 vs 治疗）= 6 个指标

## 流行病学指标"一家人"速查

| 场景 | 问什么 | 人话例子 | 缩写 | 全称 | 公式 |
|------|--------|----------|------|------|------|
| 害人 | 增加多少风险？ | "吸烟者比不吸烟者多 18% 得癌" | **ARI** | Absolute Risk Increase | Re − Ru |
| 害人 | 害了多大比例？ | "吸烟者的癌，90% 是吸烟害的" | **ARP** | Attributable Risk Percentage | (Re−Ru) / Re |
| 害人 | 多少人害 1 个？ | "6 个人吸烟，多 1 个得癌" | **NNH** | Number Needed to Harm | 1 / ARI |
| 救人 | 减少多少风险？ | "吃药比不吃药少 3% 心梗" | **ARR** | Absolute Risk Reduction | Rc − Rt |
| 救人 | 减少多大比例？ | "吃药相对少 37% 心梗" | **RRR** | Relative Risk Reduction | ARR / Rc = 1 − RR |
| 救人 | 多少人救 1 个？ | "33 人吃药，多救 1 个" | **NNT** | Number Needed to Treat | 1 / ARR |

> [!tip] 字母拆解
> A = Absolute（绝对）/ Attributable（归因）
> R = Risk（风险）/ Relative（相对）/ Reduction（减少）
> I = Increase（增加，害人信号）
> P = Percentage（百分比）
> N = Number Needed（所需人数）
> T = Treat（治疗，救人信号）
> H = Harm（伤害，害人信号）
> 

## 💡 公式分母规律（自己总结）

**核心**：百分比题（ARP / RRR），分母总是**大的那个数**

| 公式 | 分母 | 原因 |
|------|------|------|
| ARP = (Re−Ru) / **Re** | Re（暴露组，大）| 问"暴露组里多少 %" |
| RRR = (Rc−Rt) / **Rc** | Rc（对照组，大）| 问"baseline 减少多少 %" |

**通用规律**：百分比 = 差值 / 那个总参照（大的）

#难记 #生物统计

### 暴露 / 危害情境（5 个指标）

| 缩写                 | 全称                                         | 公式                            | 含义           | 单位  |
| ------------------ | ------------------------------------------ | ----------------------------- | ------------ | --- |
| **RR**             | Relative Risk                              | Re/Ru                         | 相对风险倍数       | 倍数  |
| **AR / ARI**       | Attributable Risk / Absolute Risk Increase | Re−Ru                         | 绝对风险增加       | 比例差 |
| **ARP / AF / AR%** | Attributable Risk Percentage / Fraction    | (Re−Ru)/Re                    | 暴露组中归因比例     | 百分比 |
| **PAF / PAR%**     | Population Attributable Fraction           | p(e)×(RR−1) / [1+p(e)×(RR−1)] | 全人群归因比例      | 百分比 |
| **NNH**            | Number Needed to Harm                      | 1/ARI                         | 多害 1 人所需暴露人数 | 整数  |

### 治疗 / 获益情境（4 个指标）

| 缩写 | 全称 | 公式 | 含义 | 单位 |
|------|------|------|------|------|
| **RR** | Relative Risk | Rt/Rc | 相对风险（<1）| 倍数 |
| **ARR** | Absolute Risk Reduction | Rc−Rt | 绝对风险减少 | 比例差 |
| **RRR** | Relative Risk Reduction | ARR/Rc = 1−RR | 相对风险减少 | 百分比 |
| **NNT** | Number Needed to Treat | 1/ARR | 救 1 人所需治疗人数 | 整数 |

RR > 1：暴露/治疗组**增加**风险（危险）
RR = 1：暴露/治疗**无影响**
RR < 1：暴露/治疗组**减少**风险（保护）

例：
吸烟肺癌 RR = 10 → 危险因素
疫苗肺炎 RR = 0.2 → 保护因素

### Case-Control 研究（专用）

| 缩写 | 全称 | 公式 | 含义 |
|------|------|------|------|
| **OR** | Odds Ratio | (a×d)/(b×c) | 优势比（罕见病时 ≈ RR）|

### 镜像对应关系

| 暴露害人         | 治疗救人         |
| ------------ | ------------ |
| ARI = Re−Ru  | ARR = Rc−Rt  |
| ARP = ARI/Re | RRR = ARR/Rc |
| NNH = 1/ARI  | NNT = 1/ARR  |
### 互补关系(深度理解)

- **RR + RRR = 1** — RR 是"剩多少风险",RRR 是"少了多少风险"
  - 比喻:**RR = 折扣率(打 7 折)**;**RRR = 优惠额度(少花 30%)**
- **ARR = 对照组发生率 × (1 − RR)** — ARR 同时取决于"风险降幅"和"基线严重程度"

### 解读规则(选择题速判)

| RR / OR 数值 | 含义 |
|---|---|
| **= 1** | 治疗无效 / 无关联 |
| **< 1** | 治疗有效(保护因素);**离 1 越远,RRR 越大,疗效越好** |
| **> 1** | 暴露增加风险(危险因素;如吸烟肺癌 RR > 1) |
| **CI 跨过 1**(对 RR/OR) | 不显著 |
| **CI 跨过 0**(对差值/ARR/MD) | 不显著 |

### 临床视角对比

- **医生偏爱 ARR / NNT** — 直接告诉你"治多少人能省一笔开销 / 救一条命"
- **药企偏爱 RRR** — 数值看起来惊艳(如 2% → 1%,ARR 仅 1% 但 RRR = 50%)

#### 各指标的**典型数量级**

| 指标                          | 典型范围                          | 单位/性质      |
| --------------------------- | ----------------------------- | ---------- |
| **RR**                      | 通常 **0.1 - 10**（罕见超过 100）     | 倍数         |
| **OR**                      | 类似 RR                         | 倍数         |
| **ARI / ARR / RD**          | 通常 **0.01 - 0.5**（即 1% - 50%） | 比例（小数 或 %） |
| **ARP / AF / PAF**          | 通常 **20% - 95%**（接近 100% 时）   | 百分比（大数）    |
| **RRR**                     | 通常 **20% - 80%**              | 百分比        |
| **NNT / NNH**               | 通常 **2 - 1000**（整数）           | 人数         |
| **Risk** (本身)               | 通常 **<50%**（除非极端情况）           | 百分比        |
| **Sensitivity/Specificity** | **0% - 100%**                 | 百分比        |
| **PPV / NPV**               | **0% - 100%**                 | 百分比        |


### 🔥 CK 高频陷阱(效应指标)

- **陷阱 1:** 看到 RRR = 50% 觉得很厉害 — 检查 ARR,如果只有 1%,临床获益其实很小
- **陷阱 2:** Case-control 让你算 RR — 不行!回顾性研究**不能算真实发病率**,只能算 OR
- **陷阱 3:** NNT 算出小数 — **向上取整**(10.2 → 11)
- **陷阱 4:** 报 NNT 不带"对照、结局、时间" — 完整描述必须包含**人数 + 对照 + 结局 + 时间窗**
- **陷阱 5:** P < 0.05 ≠ 强关联 — **关联强度只看 RR/OR 偏离 1.0 的程度**,P 值只代表"差异是真的"

## 五、研究设计(6+2 种,必须秒认)

| 设计 | 一句话识别 | 报告指标 | 优点 | 缺点 |
|---|---|---|---|---|
| **Cross-sectional** | "在某时点同时测暴露和疾病" | Prevalence | 快、便宜 | 无因果 |
| **Case-control** | "已有病的 vs 没病的,回头查暴露" | **OR** | 适合**罕见病** | Recall bias |
| **Cohort(prospective)** | "暴露 vs 不暴露,往前追踪" | **RR、Incidence** | 可推因果 | 慢、贵 |
| **Cohort(retrospective)** | "用既往记录追踪" | RR | 较快 | 数据质量差 |
| **RCT** | "随机分组干预" | RR、ARR、NNT | **最高证据等级** | 贵、伦理限制 |
| **Crossover trial** | "受试者先 A 后 B,自己做自己对照" | 同 RCT | 控制个体差异 | **残留效应(carry-over)** |
| **Meta-analysis** | "汇总多个研究" | 综合 RR/OR | **增加 power、缩窄 CI** | 各研究异质性、发表偏倚 |
| **Case series / report** | "几例描述" | 无 | 发现新病 | 无对照,不能推因果 |

### Meta-analysis 的核心机制

- **Pooling 小样本研究 → 总体样本量增加**
- **样本量 ↑** → 标准误 ↓ → **CI 变窄 → 精度 ↑** → **Power ↑(β ↓,即降低 II 型错误)**
- 但**不直接降低 α**(I 型错误)

### Twin / Adoption Studies(独有考点)

| 设计 | 控制变量 | 研究目的 |
|---|---|---|
| **Twin studies** | 基因相同(同卵) | 看**环境**因素影响 |
| **Adoption studies** | 环境共享(养父母家) | 看**遗传**因素影响 |

### Clinical Trial Phases(0 – 4)

| Phase | 重点 | 受试者 |
|---|---|---|
| **0** | 微剂量人体探索,决定是否继续 | 极少数(< 15) |
| **1** | **安全性 + 毒性** | 健康志愿者(20–80) |
| **2** | **有效性 + 副作用** | 患者(几百) |
| **3** | **与现行标准对比**(关键 RCT) | 大量患者(几千) |
| **4** | **上市后监测**(real-world、依从性、罕见 ADR) | 广泛人群 |

> **研究伦理 / 知情同意 / IRB 详见 [[Ethics_Master#十四研究伦理-irb--efic]]**(任何 phase 都需先有 IRB 批准)

## 六、Hill 因果判定标准(Hill's Criteria) ⭐独有

光有"统计学显著"**不能证明因果**;Hill 给出了 9 条判定标准,CK 最常考其中 3 条:

### 三大核心(必懂)

1. **Temporality(时序)**→ 唯一"必须满足"的硬性条件,因在果前
2. **关联强度(Strength of Association)** — RR / OR 偏离 1.0 越远,关联越强
3. **剂量-反应关系(Dose-Response)** — 暴露剂量↑ → 风险呈**单调阶梯式**↑
   - 例:吸烟量 1→2→3 PPD,癌症 RR 2.4→3.0→4.1(明显阶梯)

**### 加分项(偶尔考) - 
1. **生物学合理性 (Plausibility)** — 与已知机制一致 
- **一致性 (Consistency)** — 多个研究在不同人群得到相同结论 
- **可逆性 (Reversibility / Experimental evidence)** — 撤除暴露后风险下降 = 最强实验证据 
- **特异性 (Specificity)** — 一个原因对应一个结局(现已不太强调) 
- 
- ### ⚠️ 不要混淆
- **Hill's specificity**(一因一果) ≠ **诊断 specificity**(test 排除疾病能力) 
- **Temporality 是"必要"不"充分"** — 只有时序不够证明因果,选项里写"because A preceded B, A causes B" 是错的(USMLE 绝对化陷阱)
- 
1. 可逆性(Reversibility)** — 撤除暴露后风险下降 = **最强实验证据**

### 其他常考

- **时序性(Temporality)** — 因在前,果在后(必要条件)
- **生物学合理性(Plausibility)** — 与已知机制一致
- **一致性(Consistency)** — 多个研究在不同人群得到相同结论
- **特异性(Specificity)** — 一个原因对应一个结局(现已不太强调)

### 🔥 CK 高频陷阱(因果)

- **陷阱 1:** "P < 0.05 ≠ 因果"。它只代表关联是**真的**,不代表关联是**强的**,更不代表是**因果**。
- **陷阱 2:** "RR 大 ≠ 剂量-反应"。如果不同剂量下 RR 都差不多(都是 3.0),则**没有**剂量-反应关系。
- **陷阱 3:** Cross-sectional / case-control 永远**不能直接推因果**(只有 cohort 和 RCT 能)。
- **陷阱 4:** 相关 ≠ 因果 — 即使 r = 0.99,也不能说谁引起了谁。

## 七、Bias(高频前 12)

> 关键:**记情境,不是记定义**。CK 给场景让你认。

### 选样 / 抽样

| Bias | 典型情境 | 防范 |
|---|---|---|
| **Selection bias** | 研究对象不能代表总体(non-response、自选) | 随机抽样 |
| **Berkson bias** | 用**住院患者**作样本造成偏倚 | 用**人群对照** |

### 数据收集

| Bias | 典型情境 | 防范 |
|---|---|---|
| **Recall bias** | Case-control 中,**病人比对照记得清楚** | 用客观记录,缩短回忆窗口 |
| **Measurement bias** | 工具不一致 | 统一工具 |

### 研究执行

| Bias | 典型情境 | 防范 |
|---|---|---|
| **Attrition bias** | 失访造成两组不均 | **Intent-to-treat 分析** |
| **Procedure bias** | 不同组得到不同对待 | 统一流程 + 盲法 |

### 观察者效应

| Bias | 典型情境 | 防范 |
|---|---|---|
| **Hawthorne effect** | 受试者**因被观察而改变行为** | 盲法、安慰剂 |
| **Observer / Pygmalion(Expectancy)** | 研究者预期**影响测量** | 双盲、三盲 |

### 筛查特有

| Bias | 典型情境 | 防范 |
|---|---|---|
| **Lead-time bias** | 早发现 ≠ 真活更久,只是知道得早 | 用**疾病特异死亡率**比较 |
| **Length-time bias** | 筛查抓到的多是**惰性病变** | RCT 校正 |
| **Overdiagnosis bias** ⚠️补充 | 发现了**不会进展**的病变 | 长期随访验证 |

### 文献 / 出版

| Bias | 典型情境 | 防范 |
|---|---|---|
| **Publication bias** ⚠️补充 | 阳性结果更易发表 | **Funnel plot**(漏斗图)检测;不对称提示存在 |

### 变量层面

| 概念 | 典型情境 | 处理 |
|---|---|---|
| **Confounding bias** | **第三变量**同时影响暴露和结局(经典:咖啡-肺癌-吸烟) | **设计阶段:** 随机化、配对、crossover<br>**分析阶段:** 分层、多元回归 |
| **Effect modification** | **不同亚组效应不同**(不是 bias,但常考) | **分层报告**,不要校正掉 |

### Confounding vs Effect Modification 终极对比 ⭐

| 特性 | **混杂(Confounding)** | **效应修饰(Effect Modification)** |
|---|---|---|
| 与暴露/结局关系 | 均有关联,**非中间环节**;产生虚假相关 | 改变暴露与结局**关联的大小** |
| 统计学表现 | **校正前后**效应值显著改变 | **分层后**各层效应值彼此**不等** |
| 处理目标 | **消除/控制**(Control) | **分析/描述**(Analyze) |
| 口诀 | **总分不等,层间相等**(大家都被同一因素忽悠) | **层间不等,各显神通**(不同人不同反应) |
| 例子 | 总 RR = 4.0,分层后两层都 = 2.0 | 总 RR = 2.0,男 RR = 4.0,女 RR = 1.0 |
| 临床用途 | 必须校正,否则结论被"污染" | **亚组分析(精准医疗)** — 找出最获益人群 |

### 🔥 CK 高频陷阱(Bias)

- **陷阱 1:** Lead-time vs Length-time 区分点 — **lead-time = 时间提前**(早诊断假象),**length-time = 疾病性质**(惰性病变多)
- **陷阱 2:** Effect modification **不要校正掉** — 它本身就是研究价值,要分层报告
- **陷阱 3:** 看到"住院病例"作样本 → 必选 **Berkson bias**
- **陷阱 4:** Case-control 看到"病人记得更清楚" → 必选 **Recall bias**

## 八、统计检验选择(题目给场景,选检验)

> 口诀:**双定量选回归,双定性选卡方,自定性因定量选 t/ANOVA。**

### 主表

| 自变量(组别) | 因变量(结局) | 推荐检验 |
|---|---|---|
| 分类(2 组) | **定量(均值)** | **t-test**(均值视为连续) |
| 分类(>2 组) | **定量(均值)** | **ANOVA** |
| 分类(2 组) | **分类(频率)** | **Chi-square** |
| **定量(连续)** | **定量(连续)** | **Correlation / Linear regression** |

### 速查表补充 ⚠️

| 数据特殊情况 | 检验 |
|---|---|
| **小样本分类**(单元格 < 5) | **Fisher's exact test** |
| **配对数据**(同人前后测量) | **Paired t-test** |
| **等级 / 非正态相关** | **Spearman correlation** |
| **正态、连续、双向相关** | **Pearson correlation** |

### 卡方 vs 逻辑回归 ⭐

| 特性 | **卡方(Chi-square)** | **逻辑回归(Logistic)** |
|---|---|---|
| 自变量(X)要求 | **必须是分类** | 分类、定量皆可 |
| 主要功能 | **比较比例 / 检验独立性**(p 值) | **风险预测 / 计算 OR** |
| 处理变量数 | 通常 2 个(单因素) | **可多个**(多因素校正混杂) |
| 产出 | p 值 | **p 值 + OR** |

> **注意:** 即便原始数据是连续的(如血压),只要研究者**按门槛划分为"是/否"**两类,统计分析就**必须用卡方**。

### 线性回归 vs 逻辑回归 ⭐

| 特性 | **线性回归(Linear)** | **逻辑回归(Logistic)** |
|---|---|---|
| **因变量(Y)** | 连续数值 / 定量(身高、血压) | **二分类**结局(生 / 死) |
| **主要产出** | 系数(β) | **比值比(OR)** |
| **图形形状** | 直线 | **S 型曲线(Sigmoid)** |

### 决策流程速记

- 只问"两者是否有关联" → **卡方**(最简便)
- 要"算 OR" 或"控制混杂" → **必须用逻辑回归**

### 🔥 CK 高频陷阱(统计检验)

- **陷阱 1:** 看到"分类变量"立刻想到卡方 — 但若**单元格 < 5** 必须用 **Fisher's exact**
- **陷阱 2:** 同人测两次的数据用普通 t-test — 错!必须用 **Paired t-test**
- **陷阱 3:** 数据偏态 / 等级用 Pearson — 错!应用 **Spearman**

## 九、假设检验(α / β / Power / p / CI)

### 错误类型矩阵

| | 实际无差异 | 实际有差异 |
|---|---|---|
| **检验"无差异"** | ✅ 正确 | ❌ **Type II(β)** — 假阴性,漏掉真实差异 |
| **检验"有差异"** | ❌ **Type I(α)** — 假阳性,误认为有差异 | ✅ **Power = 1 − β** |

### 关键公式

- **CI = (1 − α) %** — 95% CI 对应 α = 0.05
- **Power = 1 − β** — 通常希望 ≥ 80%(即 β ≤ 0.20)
- **样本量 N ↑** → 标准误 ↓ → CI 变窄 → 精度 ↑ → Power ↑(β ↓)

### 如何提高 Power

1. **↑ 样本量(最有效,Meta-analysis 也是这条路)**
2. ↑ 效应大小
3. ↑ α(放宽显著标准,代价是更易假阳性)
4. ↓ 变异度(更精确测量)

### p 值的"硬核定义"

> **在 H₀(零假设)为真的前提下,观察到当前样本结果或更极端结果的概率。**

p 值的"三大要素":

1. **前提:** 假设 H₀ 是真的(两组没区别)
2. **内容:** 看到目前这个差异(或更极端的)
3. **结果:** 发生的概率(小概率事件)

**例:** 新药比二甲双胍多降 0.5% HbA1c,P = 0.02
→ 含义:**如果两药真的没区别**,仅凭运气出现 ≥ 0.5% 优势的概率只有 2%。

### 置信区间(CI)宽度原理 ⭐

- **置信水平越高(95% > 90%) → CI 越宽**(相同数据下)
- **样本量越大 → CI 越窄**
- **想要的方向不一致**:**Power ↑(越大越好)** vs **CI ↓(越窄越好)**

### CI 包含的"无效值"

| 研究指标 | 计算方式 | **无效点** | 接近该点的意义 |
|---|---|---|---|
| **均值差(MD) / ARR** | 减法 | **0** | 差异极小,临床获益微弱 |
| **OR / RR / HR** | 除法 | **1** | 风险改变极小,关联极弱 |

记忆:
- **差值看 0** — 想象拔河,两边力气一样 → 位移 = 0
- **比率看 1** — 想象分蛋糕,两边分得一样 → 比例 = 1:1

> **核心结论:** CI 包含无效点 = **无统计学意义**;CI 紧贴无效点 = **临床意义可能不足**。
> CI 必须以**样本观察值为中心**,且其是否包含无效值必须与 p 值显著性**绝对一致**。

### 🔥 CK 高频陷阱(假设检验)

- **陷阱 1:** "P 值 = 错误率" — **错**!P 值是 H₀ 为真时观察到现状的概率,不是研究犯错的概率
- **陷阱 2:** "样本量太小" 问缺点 → 选 **"Low statistical power"** 或 **"High Type II error risk"**
- **陷阱 3:** "样本量太小" 问改进 → 选 **"Increase sample size"** 或 **"Perform a meta-analysis"**
- **陷阱 4:** CI 越宽越好 — **错**!**Narrower CI = More precise = Better**
- **陷阱 5:** P = 0.06 vs 0.04 临床意义差异巨大 — **错**!统计学是非黑即白的,但**临床显著性看效应大小**

## 十、统计描述(集中趋势 + 偏态 + 正态分布)

### 集中趋势

| 指标 | 特点 |
|---|---|
| **Mean(均值)** | 加总除以个数;**对极端值/outliers 敏感** |
| **Median(中位数)** | 排序后中间值;**偏态分布优选** |
| **Mode(众数)** | 出现最多的值;**唯一可用于分类变量** |
| **Bimodal(双峰)** | 两个峰 → 提示**两个不同人群**(如儿童 + 老人) |

### 偏态分布(口诀:**Median 永远在中间**)

| 类型 | 排序(低 → 高) | 现象 |
|---|---|---|
| **正偏(右偏,right-tailed)** | **Mode → Median → Mean** | 尾巴在右;均值被高值拉高(如收入) |
| **负偏(左偏,left-tailed)** | **Mean → Median → Mode** | 尾巴在左;均值被低值拉低 |

### 正态分布 68 / 95 / 99.7% 规则 ⚠️补充

| 范围 | 覆盖比例 |
|---|---|
| **±1 SD** | **68%** |
| **±2 SD**(精确 1.96) | **95%** |
| **±3 SD** | **99.7%** |

## 十一、Screening Test 完整概念

### 设计原则

- **筛查 test** → 要求 **Sn 高**(漏诊代价大,如癌症筛查)
- **确诊 test** → 要求 **Sp 高**(误诊代价大)

### Cutoff 调整

| 动作 | Sn | Sp |
|---|---|---|
| **Cutoff 调低** | ↑ | ↓ |
| **Cutoff 调高** | ↓ | ↑ |

### 评估筛查项目的 4 大偏倚(必考)

1. **Selection bias** — 愿意被筛查的人本身更健康
2. **Lead-time bias** — 早发现 ≠ 真的活更久
3. **Length-time bias** — 筛查抓到的多是慢病
4. **Overdiagnosis bias** ⚠️ — 发现了不会进展的病变(过度治疗)

### 🔥 CK 高频陷阱(筛查)

- **陷阱:** 题干说"自从开始筛查,平均生存期从 3 年延长到 5 年" → 别急着说"筛查有效",**先排除 lead-time bias**(用疾病特异死亡率才能判断真实获益)

## 十四、Aging — High-Yield 生理变化 ⭐独有

| 系统 | 老年改变 |
|---|---|
| **心血管** | **收缩压 ↑、舒张压 ↓ → 脉压 ↑** |

| **体成分** | 肌肉 ↓、脂肪 ↑(影响药物分布) |
| **肺** | **残气量(RV)↑、总肺活量(TLC)不变** |
| **认知** | **智商保持稳定**(不要轻易诊断痴呆) |

## 十五、考试套路总结

### 题型 1:给 2×2 表算指标
- 看清问的是 Sn / Sp / PPV / NPV / 哪个
- **小心:** 它常给 prevalence 让你算 PPV(贝叶斯)

### 题型 2:识别研究设计
- "回顾性看暴露" → case-control → OR
- "前瞻追踪" → cohort → RR
- "随机分组" → RCT
- 看到"在某时点测量了..." → cross-sectional

### 题型 3:识别 bias
- 关键词识别:"病人记得更清楚" → recall;"住院患者样本" → Berkson;"早发现等于活更久?" → lead-time

### 题型 4:统计检验选择
- 看变量类型(连续 vs 分类)+ 组数 → 套表;**单元格 < 5** 必用 Fisher's exact;**同人前后** 必用 paired

### 题型 5:p 值/CI 解读
- "CI 跨过 1" → 不显著(RR/OR)
- "CI 跨过 0" → 不显著(差值/MD/ARR)
- 90% 的题不需要算,只需要解读

### 题型 6:Hill 因果 / 剂量-反应
- 看 RR 偏离 1 的程度(强度)+ 剂量是否阶梯式增加(剂量-反应)+ 撤除是否好转(可逆性)

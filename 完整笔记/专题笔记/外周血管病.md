# A. 缩写速查 + 分类树

## A.1 血管病大分类

```
血管病大谱
│
├── 动脉病
│   ├── 慢性
│   │   ├── PAD (Peripheral Artery Disease) 外周动脉病 ⭐
│   │   ├── PVD (Peripheral Vascular Disease) - 旧术语（包含动+静脉，避免用）
│   │   ├── CAD (Coronary Artery Disease) 冠心病
│   │   ├── CVD (Cerebrovascular Disease) 脑血管病
│   │   ├── RAS (Renal Artery Stenosis) 肾动脉狭窄
│   │   ├── MAD (Mesenteric Artery Disease) 肠系膜动脉病
│   │   ├── Leriche syndrome（主髂闭塞）
│   │   └── CLI (Critical Limb Ischemia) 重度肢体缺血 ⭐
│   ├── 急性
│   │   ├── ALI (Acute Limb Ischemia) 急性肢体缺血 ⭐
│   │   ├── ACS (Acute Coronary Syndrome) 急性冠脉综合征
│   │   └── Stroke (Acute Cerebrovascular Event) 急性脑卒中
│   ├── 炎症性
│   │   ├── GCA / TA (Giant Cell Arteritis / Temporal Arteritis)
│   │   ├── Takayasu Arteritis 大动脉炎（< 40 岁亚洲女）
│   │   ├── PAN (Polyarteritis Nodosa) 结节性多动脉炎
│   │   ├── Buerger disease / TAO (Thromboangiitis Obliterans) ⭐
│   │   └── Kawasaki Disease 川崎病
│   ├── 痉挛性
│   │   └── Raynaud Phenomenon 雷诺现象
│   └── 结构异常
│       ├── AAA (Abdominal Aortic Aneurysm) 腹主动脉瘤
│       ├── TAA (Thoracic Aortic Aneurysm) 胸主动脉瘤
│       └── Aortic Dissection 主动脉夹层
│
└── 静脉病
    ├── 急性
    │   ├── DVT (Deep Vein Thrombosis) 深静脉血栓 ⭐
    │   ├── PE (Pulmonary Embolism) 肺栓塞 ⭐
    │   ├── VTE (Venous Thromboembolism) = DVT + PE 总称
    │   ├── SVT (Superficial Vein Thrombosis) 浅静脉血栓
    │   └── Phlegmasia Cerulea Dolens
    └── 慢性
        ├── CVI (Chronic Venous Insufficiency) 慢性静脉功能不全 ⭐
        ├── Varicose veins 静脉曲张
        ├── PTS (Post-Thrombotic Syndrome) 血栓后综合征
        └── Stasis dermatitis 淤积性皮炎
```

## A.2 5 组容易混淆名词

| 易混 | 区别 |
|---|---|
| **PAD vs PVD** | PAD = 仅动脉（现代术语）；**PVD = 旧术语**（动+静脉），避免用 |
| **CLI vs ALI** | CLI = **慢性**重症（rest pain/溃疡/坏疽）；ALI = **急性**突发（6 P's, 6h 窗口）|
| **DVT vs SVT** | DVT = **深静脉**（危险 → PE）；SVT = **浅静脉**（多自限）|
| **GCA vs Takayasu** | GCA = **> 50 岁** + 颞动脉 + 大血管；Takayasu = **< 40 岁亚洲女** + 主动脉弓 |
| **Buerger sign vs Buerger disease** | Sign = 体征（抬高变白）；Disease = TAO（年轻男 + 重度吸烟）|

## A.3 命名规律

| 前缀 | 含义 |
|---|---|
| **A**- | Acute → ALI, ACS, AAA |
| **C**- | Chronic / Critical → CLI, CVI |
| **D**- | Deep → DVT |
| **P**- | Peripheral → PAD, PE |
| **S**- | Superficial → SVT |
| **T**- | Thoracic → TAA |
| **V**- | Venous → VTE |

## A.4 VTE 总称概念

```
VTE (Venous Thromboembolism) = 静脉血栓栓塞症
   ↓
分两个表现：
   ├── DVT（血栓在静脉）
   └── PE（血栓栓塞到肺动脉）
   ↓
70-90% PE 来源于 DVT
```

⭐ **临床上 DVT 和 PE 是同一谱系的两个阶段**。

# B. 评分系统集合

> [!info] 设计原则
> 评分**单独整理**，不混在疾病讲解里。讲疾病时只**链接**到这里。

## B.1 全部评分速查表

| 评分 | 用于 | 关键 cut-off |
|---|---|---|
| **ABI** | 诊断 PAD | ≤ 0.9 = PAD；> 1.3 = 钙化 → 改 TBI |
| **TBI** | ABI 不可信时 | < 0.7 = PAD |
| **Rutherford** | PAD 严重度（北美）| 0-6（4-6 = CLI）|
| **Fontaine** | PAD 严重度（欧洲）| I-IV（III-IV = CLI）|
| **Wells DVT** | DVT 临床概率 | ≥ 2 = likely / < 2 = unlikely |
| **Wells PE** | PE 临床概率 | > 4 = likely / ≤ 4 = unlikely |
| **PERC** | 低风险 PE 排除 | 8 项全阴 → 排除 PE |
| **PESI** | PE 死亡率分层 | I-V，决定门诊 vs ICU |
| **CHA₂DS₂-VASc** | AFib 卒中风险（决定抗凝）| ≥ 2 (男) / ≥ 3 (女) → 抗凝 |
| **HAS-BLED** | AFib 抗凝出血风险 | ≥ 3 = 高出血风险 |

## B.2 ABI（Ankle-Brachial Index 踝肱指数）

```
ABI = 踝部 SBP / 上臂 SBP
        ↓
≥ 1.0  正常
0.9-1.0  临界
≤ 0.9   PAD（诊断阈值）⭐
0.4-0.9  Mild-moderate PAD
< 0.4   Severe PAD / CLI ⭐
> 1.3   钙化血管（DM/CKD）→ 不可信 → 改 TBI ⭐
```

## B.3 TBI（Toe-Brachial Index 趾肱指数）

⚠️ **注意缩写歧义**：TBI 在 Neuro 是 Traumatic Brain Injury，这里是 Toe-Brachial。

```
TBI = 大拇趾 SBP / 上臂 SBP
        ↓
TBI < 0.7 = PAD（趾动脉很少钙化，结果可靠）
```

适用：DM / CKD / 钙化血管 / ABI > 1.3 时

## B.4 Rutherford 分级（PAD 严重度）

| Category | 临床 | 治疗方向 |
|---|---|---|
| **0** | 无症状 | Risk factor 控制 |
| **1** | 轻度 claudication | Exercise + 药物 |
| **2** | 中度 claudication | Exercise + 药物 |
| **3** | 重度 claudication (< 1 block 痛) | Exercise + 药物 → 顽固→revasc |
| **4** | **Rest pain** ⭐ | **Revasc** |
| **5** | 小溃疡 / 组织丧失 | **Revasc + 伤口护理** |
| **6** | 大溃疡 / 坏疽 | **Revasc 或 截肢** |

⭐ **Rutherford 4-6 = Critical Limb Ischemia (CLI)**

## B.5 Fontaine 分级（PAD 严重度，欧洲）

| Stage | 临床 |
|---|---|
| **I** | 无症状 |
| **IIa** | 轻度 claudication（> 200m）|
| **IIb** | 重度 claudication（< 200m）|
| **III** | **Rest pain** ⭐ → CLI |
| **IV** | 溃疡 / 坏疽 → CLI |

⭐ **Fontaine III-IV = CLI**

## B.6 Wells DVT 评分 ⭐⭐ 9 项

```
活动性癌症（治疗中或 6 月内）          +1
肢体瘫痪 / 近期石膏制动               +1
卧床 > 3 天 或 大手术 < 12 周         +1
深静脉走向压痛                       +1
**整条腿肿胀**                       +1
小腿肿胀 > 对侧 3 cm                  +1
凹陷性水肿（症状侧重）                +1
非静脉曲张的侧支浅静脉                +1
**有其他更可能诊断 (-2)** ⭐ 减分项！
─────────────────────────────────
≥ 2 = DVT likely → Duplex US
< 2 = DVT unlikely → D-dimer
   阳性 → US；阴性 → 排除
```

## B.7 Wells PE 评分 ⭐⭐ 7 项

```
临床 DVT 征象（单侧腿肿+压痛）          +3 ⭐
**PE 是最可能诊断**                    +3 ⭐⭐ 主观项
HR > 100                              +1.5
近期 < 4 周制动 / 手术                  +1.5
既往 DVT/PE                            +1.5
咯血                                  +1
活动性癌症                            +1
─────────────────────────────────
> 4 = PE likely → CTPA
≤ 4 = PE unlikely → D-dimer
   阳性 → CTPA；阴性 → 排除
```

⚠️ **Wells DVT vs Wells PE 是两个评分**！同名（同一个 Wells 学者），但项目不同。
- 看**腿** → DVT 评分（9 项，≥ 2 likely）
- 看**胸** → PE 评分（7 项，> 4 likely）
- **PE 评分独有"PE 是最可能诊断"主观项 +3 分**

## B.8 PERC（PE Rule-out Criteria）

**适用**：临床低风险 PE + 想避免 D-dimer 时

```
8 项必须**全部阴性**：
1. 年龄 < 50
2. HR < 100
3. SpO₂ > 94%
4. 无单侧腿肿
5. 无咯血
6. 近期无手术 / 创伤
7. 既往无 DVT/PE
8. 无激素使用（OCP / HRT）
        ↓
全阴 + 临床低概率 → **直接排除 PE**（无需 D-dimer）
任一阳性 → 走 D-dimer 路径
```

## B.9 PESI（PE Severity Index）

**用于已确诊 PE 后的死亡率分层 + 治疗场所决策**

```
PESI 分级：
Class I-II（低风险）→ 死亡率 < 3% → **可门诊治疗**
Class III（中风险）→ 住院抗凝
Class IV-V（高风险）→ ICU 监测，考虑溶栓
```

## B.10 CHA₂DS₂-VASc（AFib 卒中风险）

```
**C**HF (+1)
**H**TN (+1)
**A**ge ≥ 75 (+2)
**D**M (+1)
**S**troke / TIA 既往 (+2)
**V**ascular disease (PAD/MI) (+1)
**A**ge 65-74 (+1)
**S**ex female (+1)
─────────────────────
≥ 2 (男) / ≥ 3 (女) → **抗凝必须**（DOAC 首选）
```

## B.11 HAS-BLED（AFib 抗凝出血风险）

```
**H**TN (uncontrolled)
**A**bnormal renal/liver function
**S**troke history
**B**leeding history
**L**abile INR (Warfarin)
**E**lderly > 65
**D**rugs/alcohol (NSAIDs/抗板)
─────────────────────
≥ 3 = 高出血风险（不是抗凝禁忌，但需要更严密监测）
```

# C. 动脉病

## C.1 PAD（Peripheral Artery Disease）⭐ 必考

### Stem 锁定钩子
- 老人/中年男 + smoker / DM / HTN + **走路时小腿/大腿/臀部疼，休息缓解**
- **ABI ≤ 0.9** → 诊断
- Rest pain / ulcer / gangrene → CLI

### 临床表现谱
- **Claudication**（运动诱发痛 / 休息缓解）⭐
- **Rest pain**（夜间足趾痛，**dangling 缓解**）⭐ → CLI
- 肢端冷 / 苍白 / 毛发↓ / 肌萎缩 / **Buerger sign**（抬高变白）
- 远端溃疡 / 坏疽

### 治疗按 Rutherford 决策树 ⭐⭐

```
所有 PAD 病人（无论分级）：
  ① 戒烟 > 一切（最重要）
  ② Antiplatelet (ASA/clopidogrel) + Statin
     → 降 MI/Stroke 死亡率（不是为腿）
  ③ HbA1c < 7%、BP < 130/80
        ↓
分症状决策：
        ↓
┌────────────────────────────────────────┐
│ 无症状 / Mild-Moderate Claudication    │
│ (Rutherford 0-3, Fontaine I-IIb)       │
│ → Supervised Exercise Therapy ⭐ 首选   │
│ → 二线 Cilostazol（HF 禁用！）          │
│ → 顽固 + 严重影响生活 → 考虑 revasc    │
├────────────────────────────────────────┤
│ Critical Limb Ischemia (CLI) ⭐⭐       │
│ Rutherford 4-6, Fontaine III-IV        │
│ 定义：                                  │
│   • Rest pain > 2 周 + ABI < 0.4       │
│   • Ischemic ulcer + ABI < 0.4         │
│   • Gangrene + ABI < 0.4               │
│ 招牌钩子：                              │
│   "Dangles leg over bed at night" ⭐    │
│   (重力辅助灌注缓解疼痛)                │
│ → Revascularization **必须**            │
│   流程：CTA/MRA 找可 revasc 血管        │
│   → PTA / Stent / Bypass / 截肢         │
└────────────────────────────────────────┘
```

### USMLE 高频陷阱
1. claudication 立刻 stent → 错。先 Exercise + 药物
2. Cilostazol HF 禁用（PDE3 → 心律失常）
3. Rest pain ≠ "claudication 加重版"，是 **CLI**（阶段质变）
4. DM 病人 ABI 看起来正常（钙化假性升高）→ 改 TBI
5. 所有 rest pain 都立刻 revasc？→ 不准确，需要符合 CLI 标准 + 可耐受手术 + 有可 revasc 血管

### Buerger sign vs Buerger disease（必混淆）
- **Buerger sign** = 体征（抬高变白，下垂变红）= severe PAD
- **Buerger disease** = 独立病（TAO）= 年轻男 + 重度吸烟（详见 [[#C.4 Buerger Disease (TAO)]]）

## C.2 Leriche Syndrome（主髂闭塞）

### 三联征 ⭐
```
Aortoiliac 闭塞（主动脉分叉处）
        ↓
┌───────┴───────┐
↓               ↓
Internal iliac   Common iliac
→ 阴茎海绵体    → 大腿/臀肌缺血
   供血↓        
   ↓
临床三联：
1. **Buttock + thigh claudication**（双侧）
2. **Erectile dysfunction**（阳痿，sildenafil 无效）⭐
3. **↓ Femoral pulses**（股动脉搏动减弱/消失）
```

### USMLE 锁定句
"老年男性 + 双侧大腿/臀部走路痛 + ED（sildenafil 无效）+ 双侧股动脉摸不到"

### 治疗
同 PAD（保守 + ASA + Statin + 戒烟）→ 严重 → **Aortobifemoral bypass**

### Memory Hook
"**主动脉分叉处堵车，三条路同时瘫痪**"

## C.3 急性肢体缺血 (Acute Limb Ischemia, ALI) ⭐ 急症

### 6 P's 必背
- **P**ain（剧烈）
- **P**allor（苍白）
- **P**ulselessness（无脉）
- **P**aresthesias（感觉异常）
- **P**oikilothermia（皮温变冷）
- **P**aralysis（瘫痪 - 晚期，**6 小时窗口**！）

### 病因（修正：不只是栓塞）
```
ALI 病因：
├── **栓塞** (~30%)
│   └── AFib → 左房血栓（最常见 ⭐）
│   └── 心室壁血栓（post-MI）
│   └── 主动脉夹层
└── **原位血栓** (~70%) ⭐ 实际更常见
    └── 既往 PAD 急性恶化
    └── 动脉粥样硬化斑块破裂
    └── Bypass 移植物血栓
    └── 创伤
```

### 治疗：黄金 6 小时窗口 ⭐
```
1. **立刻 IV Heparin** ⭐
   - 防止血栓**继续扩展**
   - 防止远端继发血栓
   - 维持侧支血管通畅
2. **紧急 revascularization**：
   - Surgical embolectomy / thrombectomy
   - Catheter-directed thrombolysis（tPA）
   - Bypass
3. > 6 小时 = 不可逆肌肉损伤 → **截肢**
4. Reperfusion 后注意：
   - **Compartment syndrome** → fasciotomy
   - 横纹肌溶解 → 监测 CK / 尿色
   - **高钾** → 警惕心律失常
5. 找病因：
   - 栓塞性 → 长期抗凝（如 AFib → DOAC 终身）
   - 原位血栓 → 抗板 + Statin + Revasc 维持
```

## C.4 Buerger Disease (TAO, Thromboangiitis Obliterans)

### 经典画像
- 年轻男（20-40 岁）
- **重度吸烟史** ⭐
- 远端肢体缺血（手指 / 足趾）
- 节段性血管炎症 + 血栓

### 治疗
- ⭐ **戒烟唯一有效**（不戒烟 → 截肢）
- 抗血小板 / 抗凝**无效**

## C.5 Raynaud 现象

### 三相色变
**白 → 蓝 → 红**（小动脉痉挛 → 缺血 → 反应性充血）

### 治疗
1. 避免诱因（冷 / 应激）+ 保暖手套
2. **CCB（Nifedipine / Amlodipine）⭐** 一线药物
3. α-blocker / SSRI / PDE5i / Prostacyclin（severe）
4. 严重溃疡 → IV Iloprost / 手术

### Primary vs Secondary Raynaud
- Primary = 单纯 Raynaud（年轻女，对称，无 ANA）
- Secondary = 合并 CTD（**Scleroderma / SLE / MCTD**，可不对称）

## C.6 急性动脉血栓事件简表

| 病 | 受累动脉 | 治疗钥匙 |
|---|---|---|
| ALI | 肢体动脉 | IV Heparin + 紧急手术（详见 [[#C.3 急性肢体缺血]]）|
| MI/ACS | 冠脉 | DAPT + PCI（详见 [[完整笔记/Peixuan分科笔记/心内]]）|
| Ischemic Stroke | 脑动脉 | tPA + 24h 后 ASA（详见 [[完整笔记/Peixuan分科笔记/neuro]] - 待章节）|
| Mesenteric ischemia | 肠系膜动脉 | 抗凝 + 手术（详见 [[完整笔记/Peixuan分科笔记/外科]]）|

⭐ **同样栓子（如 AFib）栓到不同脏器，治疗完全不同** → 详见 G 节《急性血栓事件治疗哲学对比》。

# D. 静脉病

## D.1 DVT（深静脉血栓）⭐⭐ 必考急症

### 诊断流程
```
怀疑 DVT（单侧腿肿胀/疼痛/红/Homans）
        ↓
Wells DVT 评分（详见 [[#B.6 Wells DVT 评分]]）
        ↓
┌─────────────┬─────────────────┐
< 2 (unlikely)  ≥ 2 (likely)
↓             ↓
D-dimer       **Duplex US** ⭐ 首选
↓             ↓
阴性→排除     找到 DVT → 确诊
阳性→US       未找到 → serial US 1 周
```

### 治疗按位置
```
近端 DVT（腘静脉以上） → **抗凝必须**
   ↓
首选：**DOAC**（Rivaroxaban / Apixaban）⭐ 现在一线
   - 无需 bridging / 无需 INR 监测
替代：LMWH (Enoxaparin) → Warfarin (INR 2-3)
   
远端 DVT（小腿） → 监测 (serial US 2 周) 或 抗凝（如症状重）

特殊情况：
- **妊娠** → **LMWH**（不能 DOAC/Warfarin）⭐
- **癌症相关** → DOAC > LMWH > Warfarin
- **严重肾衰** → UFH (IV)
- **抗凝禁忌** → **IVC filter**（不替代抗凝）
- **Phlegmasia cerulea dolens** → CDT/取栓（详见 [[#D.5 Phlegmasia Cerulea Dolens]]）
```

### 疗程
- **诱发因素**（术后/制动）→ **3 个月**
- **无诱发因素** → ≥ 3 个月，考虑长期
- **复发 / 易栓症 / 癌症** → **终身**

## D.2 PE（肺栓塞）⭐⭐ 必考急症

### 诊断流程（看血流动力学）

```
怀疑 PE
        ↓
血流动力学稳定？
        ↓
┌──────────────────────┬──────────────────────┐
**不稳定**             稳定
(BP < 90/休克)        ↓
↓                      Wells PE 评分
床旁回声立刻 ⭐         (详见 [[#B.7 Wells PE 评分]])
看右心：               ↓
- RV dilation           ┌──────────┬──────────┐
- RV hypokinesis        ≤ 4         > 4
- D-shaped LV           ↓           ↓
- McConnell sign        D-dimer     CTPA 直接
↓                      ↓
经验性 tPA              阳性→CTPA
+ 抗凝桥接              阴性→排除
```

### Massive PE 完整体征（USMLE 高频）⭐

```
床旁体征：
- SBP < 90 / 休克
- **JVD（颈静脉怒张）** ⭐
- Right-sided S3 / S4 奔马律
- Pulsus paradoxus（吸气 SBP 下降 > 10）
- Hepatojugular reflux
- 肢端冰冷 / 苍白 / 意识改变

影像 / 实验室：
- **床旁回声** ⭐⭐：
  • **RV dilation**（新发右心扩张）
  • **RV hypokinesis**（运动减弱）
  • **McConnell sign**（RV free wall 不动，apex 仍动）= 急性 PE 特异
  • **D-shaped LV**（室间隔向左移）
  • TR 加重 + PASP 升高
- **CTPA**：
  • **RV/LV 比 > 0.9** ⭐ 提示 RV strain
  • 主肺动脉鞍状血栓（saddle embolus）= massive
- **Troponin ↑**（RV 微梗死）
- **BNP / NT-proBNP ↑**（心室壁应力）
- **ECG**：
  • 窦速（最常见）
  • **S1Q3T3**（仅 20% 出现）⭐
  • **新发 RBBB** ⭐ ← RV 扩张牵拉右束支
  • 右心轴偏 / T 倒置 V1-V4
```

⭐ 看到 **新发 RBBB + 呼吸困难 + 低血压** → 0.5 秒锁 Massive PE → tPA 准备

### PE 风险分层 ⭐ 决定治疗

| 类型 | 定义 | 死亡率 | 治疗 |
|---|---|---|---|
| **Massive (高风险)** | SBP < 90 ≥ 15 min / 休克 / 心搏骤停 | > 30% | **tPA 必须** / 取栓 |
| **Submassive (中风险)** | BP 正常 + RV 障碍标志（≥ 1 项）| 3-15% | 抗凝为主 + 个体化考虑溶栓 |
| **Low-risk** | 稳定 + 无 RV 受累 | < 1% | 抗凝（DOAC），可门诊 |

**Submassive PE 的 RV 障碍标志**：
- 回声：RV dilation / hypokinesis
- CTPA：RV/LV > 0.9
- BNP / NT-proBNP ↑
- Troponin ↑（RV 微损伤）
- ECG：新发 RBBB / S1Q3T3

### 治疗按风险
```
Low-risk → DOAC，门诊或短住
Submassive → 住院 + DOAC/LMWH + 监测右心 / 心肌酶
              部分病例 → 评估溶栓（个体化）
Massive → ICU + **tPA**（溶栓禁忌 → 取栓）+ 抗凝桥接
```

### USMLE 经典 stem
> "65 岁女 + 大手术后 3 天 + 突发呼吸困难 + BP 78/40 + HR 130 + **JVD** + 床旁回声 **RV dilation + hypokinesis** + ECG **new RBBB**"
> 
> → **Massive PE** → **tPA 立刻**

## D.3 CVI（Chronic Venous Insufficiency）

### 机制
```
深静脉瓣膜失功 / 血栓后综合征
   ↓
血液淤滞 → 静脉高压
   ↓
铁质渗出 → **棕色色素沉着** (hemosiderin)
脂肪硬化 → **lipodermatosclerosis**
   ↓
**Venous ulcer**（内踝上方，gaiter area）
```

### 临床
- 久站加重 / **抬高缓解** ⭐
- 凹陷性水肿
- 棕色色素 / 静脉曲张
- 内踝上方溃疡（不规则、浅、渗液多、轻中度痛）

### 治疗
```
Step 1: 保守治疗
  ① **Compression therapy** ⭐⭐⭐ 核心（30-40 mmHg）
     ⚠️ **必须先查 ABI** 排除合并 PAD
     - ABI > 0.8 → 安全 compression
     - ABI 0.5-0.8 → 低压 compression
     - ABI < 0.5 → **禁用**（动脉缺血优先治）
  ② **抬高下肢**（休息时高于心脏）
  ③ 减重 / 运动（小腿肌肉泵）
  ④ 皮肤护理

Step 2: 溃疡处理
  - Compression（70% 4-6 月愈合）
  - Wound care（保湿敷料）
  - 感染 → 抗生素

Step 3: 顽固 / 严重
  - 静脉手术（valve repair / ligation）
  - 射频 / 激光消融（RFA / EVLA）
  - 硬化剂注射
```

⭐ **USMLE 黄金钩子**：**Compression therapy 前必查 ABI**

## D.4 Varicose Veins（静脉曲张）

### 治疗阶梯
1. 一线：弹力袜 + 抬高 + 减重 + 运动
2. 二线：硬化剂注射 / RFA / 激光消融
3. 三线：手术（stripping / ligation）

## D.5 Phlegmasia Cerulea Dolens ⭐ 罕见但必考

### 名词拆解
- Phlegmasia = 炎症
- Cerulea = 蓝色
- Dolens = 疼痛
- = **"蓝色 + 疼痛性的静脉炎"**

### 病理生理（核心）
```
极严重的髂股 DVT
   ↓
血栓**完全闭塞**深静脉 + **大部分侧支静脉**
   ↓
静脉回流**完全停止**
   ↓
组织间质压力↑↑↑
   ↓
动脉灌注被 venous hypertension 压迫
   ↓
**继发性动脉缺血** ⭐ ← 这才是危险所在
   ↓
肢体青紫 + 剧痛 + 巨大水肿
   ↓
Compartment syndrome / 肢体坏死 / 截肢
   ↓
**死亡率 20-40%**（多脏器衰竭）
```

⭐ **关键概念**：**静脉病引起的动脉缺血**——静脉淤血严重到压迫动脉。

### 临床钩子

> [!warning] USMLE Stem 招牌写法
> 
> "65 岁女 + 卵巢癌 + 化疗中 + 突发左下肢**剧痛 + 巨大肿胀 + 紫色**，足部冰冷，**足背动脉脉搏微弱**，足趾发紫"

| Stem 特征 | 解析 |
|---|---|
| **极度肿胀 + 整条腿** | 完全静脉回流障碍 |
| **紫色 / 青紫** | venous congestion → 还原血红蛋白↑ |
| **剧痛**（远超普通 DVT）| 张力 + 缺血 |
| **足背动脉脉搏弱 / 冰冷** ⭐ | 继发动脉灌注差 |
| **可有水疱 / 起泡** | 严重组织损伤 |
| **诱因**：癌症 / 易栓症 / IVC filter 血栓 / May-Thurner 综合征 | 必有 |

### Phlegmasia Alba vs Cerulea
```
Phlegmasia ALBA dolens（白色）
- 早期 / 程度较轻
- 血栓闭塞深静脉但**侧支还工作**
- 反射性动脉痉挛 → 肢体**苍白**
- 仍然剧痛 + 水肿
   ↓
进展 →
   ↓
Phlegmasia CERULEA dolens（蓝色）⭐
- 严重晚期
- 侧支也闭塞 → 静脉淤血 → cyanosis
- **继发动脉缺血** ← 急症
```

### 治疗 ⭐ USMLE 必考
```
急症处理：
1. 抬高肢体（**有限**——平衡缺血风险）
2. **IV Heparin 立刻**
3. **取栓 + 溶栓**（catheter-directed thrombolysis, CDT）⭐
   或手术取栓（CDT 禁忌时）
4. **Fasciotomy**（如 compartment syndrome）
5. 找诱因：**癌症筛查必须**
6. 后续：IVC filter 评估 + 长期抗凝
```

⚠️ **关键**：普通 DVT → 抗凝足够；Phlegmasia → **抗凝 + 溶栓/取栓**（罕见的 DVT 溶栓适应症）

### 锁定句
> "DVT 史 / 癌症 + 突发整条腿剧痛 + 青紫 + 巨大水肿 + **足脉弱**" → **Phlegmasia Cerulea Dolens** → IV Heparin + CDT/取栓

## D.6 SVT（Superficial Vein Thrombosis）

- 通常**自限**
- NSAIDs + 局部热敷 + compression
- **延伸到大隐静脉近端** → 抗凝（低剂量 LMWH）

# E. 动脉病 vs 静脉病治疗对比 ⭐⭐⭐ USMLE 必背

## E.1 核心原则
- **动脉病** = 缺血 → 治疗目标 **改善供血**
- **静脉病** = 淤血 → 治疗目标 **促进回流**
- ⚠️ **动脉病禁忌压迫**（加重缺血 = 截肢风险）

## E.2 5 大操作对比

| 操作 | 动脉病 | 静脉病 |
|---|---|---|
| **抬高肢体** | ❌ 加重疼痛（Buerger sign）| ✅ 缓解症状 |
| **压迫 (Compression)** | ❌ **绝对禁忌** ⭐⭐⭐ | ✅ **核心治疗** |
| **抗血小板** | ✅ 首选（ASA/Clopidogrel）| ❌ 几乎无效 |
| **抗凝** | 仅特殊情况（ALI / AFib + PAD）| ✅ 首选 |
| **运动** | ✅ Supervised Exercise（核心）| 辅助 |
| **溶栓** | ALI 偶用 | **不稳定 PE 救命** ⭐ |

## E.3 血栓性质（机制基础）

- **动脉血栓** = **白血栓**（血小板为主）→ 抗血小板
- **静脉血栓** = **红血栓**（纤维蛋白 + RBC）→ 抗凝

## E.4 USMLE 5 大经典陷阱

1. **Venous ulcer 直接 compression** → 错，先查 **ABI** 排除 PAD
2. **DVT 妊娠用 Warfarin / DOAC** → 错，**只能 LMWH**
3. **PAD claudication 直接 stent** → 错，先 **戒烟 + ASA + Statin + Exercise**
4. **DVT 抗凝禁忌就放弃** → 错，放 **IVC filter** 防 PE
5. **ALI 慢慢做影像确诊** → 错，**立刻 IV Heparin + 手术**（6h 窗口）

## E.5 Memory Hook
> "**动脉怕压，静脉爱压**"
> "**动脉抬了更痛，静脉抬了就好**"
> "**动脉缺血静脉淤，treatment 反着搞**"

# F. DVT vs PE 治疗区别原理

## F.1 谱系关系

```
DVT 和 PE 是同一谱系（VTE）的两个阶段
        ↓
DVT = 血栓在深静脉
        ↓
脱落 → 下腔静脉 → 右心 → 肺动脉
        ↓
PE = 血栓嵌顿肺动脉
        ↓
轻中度 = 缺氧 / 心率↑
        ↓
**Massive PE**：
  RV 急性扩张 → 室间隔向左移
  → LV 充盈↓ → 心输出量↓
  → 休克 / 死亡 ⭐
```

⭐ **核心区别**：
- **DVT** = 血栓在**外周静脉** → 主要风险 = **未来 PE**
- **PE** = 血栓在**肺动脉** → **当下心肺衰竭风险**
- 所以 PE 多了一个考量：**血流动力学**

## F.2 治疗对比表

| 维度 | DVT | PE |
|---|---|---|
| **核心治疗** | 抗凝 | 抗凝 + 评估稳定性 |
| **首选药** | DOAC | 同 DVT |
| **疗程** | 诱发 3 月 / 无诱发 ≥ 3 月 / 复发终身 | 同 DVT |
| **溶栓 (tPA)** | **不常规用** ⚠️ | **不稳定 PE 必用** ⭐ |
| **取栓** | 严重髂股 DVT (phlegmasia) | 溶栓禁忌的不稳定 PE |
| **IVC filter** | 抗凝禁忌时 | 抗凝禁忌时 |
| **额外评估** | 找诱因（癌症/易栓症）| **PESI 评分** → 决定门诊 vs 住院 |

## F.3 关键差异

### 差异 1：溶栓角色不同
- **DVT** 几乎不溶栓（血栓自然机化 + 出血风险大）
  - **例外**：Phlegmasia / 严重髂股 DVT 年轻患者（减少 PTS）
- **PE** 看血流动力学：
  - 稳定 → 抗凝
  - **Submassive**（RV strain）→ 个体化
  - **Massive**（休克）→ **tPA 必须** ⭐

### 差异 2：PE 有死亡风险分层（PESI）
PE 直接致死 → 需要 PESI 分级 → 决定门诊 vs ICU
DVT 不直接致死 → 没有这种分层

### 差异 3：监测重点
| 项 | DVT | PE |
|---|---|---|
| 监测 | Duplex US 复查 / 找诱因 | **回声**（评估 RV）/ Troponin / BNP |
| 急诊指标 | 一般无 | **HR / BP / SpO₂ / Shock index** |

## F.4 一句话总结

> [!success] 
> **DVT** = 慢性威胁（"防止它变 PE"）→ **稳态抗凝**
> **PE** = 即时威胁（"现在能死人"）→ **抗凝 + 看血流动力学决定要不要溶栓**

# G. 急性血栓事件治疗哲学（5 大急症终极对比）⭐⭐⭐

## G.1 核心认知

> [!warning] 关键反直觉
> 
> "急性血栓 = 都用抗凝" ❌ **错觉**
> 
> 5 个急症虽然都是血栓，但治疗哲学**完全不同**：
> 
> - **静脉系统**（PE/DVT）→ 抗凝核心
> - **冠脉**（MI）→ DAPT + PCI 核心（Heparin 辅助）
> - **脑动脉**（Stroke）→ tPA / 取栓核心（**急性不抗凝/不抗板**）⭐ 最特殊
> - **肢体动脉**（ALI）→ Heparin + 手术

## G.2 5 大急症完整对比

### 诊断流程对比

| 急症 | 第一步 / 关键检查 | 第二步 | 风险分层 |
|---|---|---|---|
| **PE** | 稳定吗？ | 不稳定→床旁回声+tPA；稳定→Wells PE→D-dimer/CTPA | PESI |
| **DVT** | Wells DVT 评分 | < 2→D-dimer；≥ 2→Duplex US | 位置（近端 vs 远端）|
| **MI** | **ECG 10 min + Troponin** | ECG 分 STEMI/NSTEMI | TIMI / GRACE |
| **Stroke** | **非增强 CT 排除出血** | 时间窗 + NIHSS + CTA（如考虑取栓）| 时间窗（4.5h/24h）|
| **ALI** | **临床（6 P's）** | 立刻 IV Heparin + CTA | Rutherford 急性 |

### 急性治疗对比

| 急症 | 核心治疗 | 时间窗 | 抗凝？ | 抗板？ | 溶栓？ |
|---|---|---|---|---|---|
| **PE** | 抗凝 | 立刻 | ✅ **核心** | ❌ | 仅 Massive |
| **DVT** | 抗凝 | 立刻 | ✅ **核心** | ❌ | 仅 phlegmasia |
| **MI** | **DAPT + PCI** | **PCI 90 min / tPA 12h** | ✅ 辅助 | ✅ **核心 (DAPT)** | tPA 仅无 PCI 时 |
| **Stroke** | **tPA / 取栓** | **tPA 4.5h / 取栓 24h** | ❌ **急性禁用** | ❌ **24h 后才用** | ✅ **核心 (tPA)** |
| **ALI** | **Heparin + 紧急手术** | **6 h** | ✅ **核心** | ❌ | 偶尔 (CDT) |

### "第一件事"对比 ⭐ 最容易混

| 急症 | 立刻做的第一件事 |
|---|---|
| **PE 稳定** | 抗凝（即使没确诊也可经验性开始）|
| **PE 不稳定** | **tPA** + 抗凝 |
| **DVT** | 抗凝（DOAC 或 LMWH）|
| **MI** | **ECG + ASA 嚼服** |
| **Stroke** | **CT 排除出血** ⭐（不是给药！）|
| **ALI** | **IV Heparin** + 紧急联系血管外科 |

## G.3 为什么 Stroke 不立即抗凝？（深度理解）

```
缺血性卒中 → 脑组织缺血损伤
   ↓
血脑屏障受损 → 受损脑组织血管脆弱
   ↓
此时**抗凝** → **大量脑出血** ⭐ 致命
   ↓
病人可能死于"治疗"而非原发病
```

⭐ 这与肢体动脉栓塞**完全不同**：
- 肢体动脉栓塞 → Heparin **保住肢体**
- 脑动脉栓塞 → Heparin **杀死病人**（出血风险）

### Stroke 抗血栓药时间点

| 时间 | 操作 |
|---|---|
| **0-4.5 h** | tPA（如符合 + 排除出血）|
| **0-24 h** | LVO → 机械取栓 |
| **24 h 内** | ❌ 禁用抗凝 + 抗板 |
| **24-48 h** | 启动 **ASA** |
| **数天-周后**（稳定）| AFib → 启动 **DOAC**（小卒中 3-5 天 / 中 5-7 天 / 大 2 周）|

⚠️ **特殊例外**：颈/椎动脉夹层可早期抗凝（但指南首选仍是抗板）

## G.4 MI 治疗澄清：MONA-BASH 现代化版

### 经典 MONA-BASH 拆解

| 字母 | 名称 | 作用 | 是否"立刻"给 |
|---|---|---|---|
| **M** | Morphine | 镇痛 + 静脉扩张 | ⚠️ 仅剧痛（有研究显示↑死亡率，避免常规）|
| **O** | Oxygen | 改善缺氧 | ⚠️ **仅 SpO₂ < 90%** |
| **N** | Nitroglycerin | 冠脉扩张 + 前负荷↓ | ⚠️ **下壁 MI / RV 梗死禁用** |
| **A** | Aspirin | 抗血小板 | ✅ **立刻 162-325 mg 嚼服** ⭐ |
| **B** | β-blocker | 心率↓ + 耗氧↓ | 24h 内若稳定（不是立刻）|
| **A** | ACEi / ARB | 心肌重塑预防 | 24h 内若稳定 |
| **S** | Statin | 高强度（Atorvastatin 80）| 入院后尽快 |
| **H** | Heparin | 抗凝辅助 | ✅ 急性期辅助 |

### 现代实践：3 核心 + 4 辅助 + 2 按需 ⭐

```
立刻必给（"核心"）：
- ASA 162-325 mg 嚼服 ⭐
- P2Y12 抑制剂（Ticagrelor / Clopidogrel / Prasugrel）⭐
- Heparin 或 Bivalirudin（PCI 辅助）⭐

按需给（"按需"）：
- Morphine - 仅剧痛
- Oxygen - 仅 SpO₂ < 90%

数小时-24h 内（"稳定后"）：
- Nitroglycerin（胸痛 + BP 正常 + 非下壁）
- β-blocker（24h 内稳定 + EF 可）
- ACEi / ARB（24h 内稳定 + LVEF < 40%）
- Statin 高强度
```

⭐ **真正"必须立刻"的只有 A（ASA）+ P2Y12 + Heparin** —— 其他都有条件。

### MONA-BASH USMLE 经典陷阱

1. **常规给氧**：SpO₂ ≥ 90% 不给氧（无益甚至有害）
2. **下壁 MI 给硝酸甘油**：可能合并 RV 梗死，依赖前负荷 → 低血压休克
3. **所有 MI 都给 β-blocker**：急性失代偿 HF / 心源性休克 / 心动过缓 = 禁忌
4. **把吗啡当一线**：首选 ASA + P2Y12 + Heparin + PCI，吗啡仅在 NTG 无效时考虑

## G.5 时间窗数字记忆 ⭐

```
"90-4.5-6-24"
- 90 min → MI PCI 窗口
- 4.5 h → Stroke tPA 窗口  
- 6 h → ALI 不可逆肌肉损伤窗口
- 24 h → Stroke 取栓 / 24h 后 ASA
```

## G.6 同一栓子，不同脏器 = 不同治疗 ⭐ 经典 USMLE 对比题

### 对比题 1：AFib + Stroke
> "65 岁男 + AFib，未抗凝。突发左侧偏瘫 2 小时，CT 无出血。下一步？"
> 
> **答案：IV tPA**（不是抗凝！即使是 AFib 栓塞性 stroke）

### 对比题 2：AFib + ALI
> "65 岁男 + AFib，未抗凝。突发左下肢剧痛 + 苍白 + 无脉 2 小时。下一步？"
> 
> **答案：IV Heparin + 紧急手术**

⭐ **同 AFib 栓塞，治疗截然相反** —— 取决于**栓到哪**。

## G.7 Memory Hook（终极版）

> [!success] 
> **"血栓 5 急症，看脏器决定治"**
> 
> ```
> 静脉血栓（PE, DVT）→ 抗凝（核心）
> 动脉血栓（ALI, MI, Stroke）→ **各不相同**！
>    ├── ALI（肢体）→ Heparin + 手术
>    ├── MI（心）→ DAPT + PCI
>    └── Stroke（脑）→ tPA（**不抗凝/不抗板**）⭐ 最特殊
> ```
> 
> 记住：**"脑子最娇贵，急性不抗凝不抗板"**

## G.8 分 3 步学习建议

> [!tip] 不要试图一次记住全部

**Step 1（今天/明天）**：核心治疗记 5 个
- PE = 抗凝；DVT = 抗凝
- MI = DAPT + PCI
- Stroke = tPA / 取栓
- ALI = Heparin + 手术

**Step 2（后天）**：第一步记 5 个
- PE 稳定吗？DVT Wells？
- MI ECG！Stroke CT！
- ALI 6 P's → Heparin！

**Step 3（一周后）**：时间窗记 4 个
- MI PCI 90 min
- Stroke tPA 4.5h
- Stroke 取栓 24h
- ALI 6h

# H. 易混陷阱合集

## H.1 缩写歧义

| 缩写 | 含义 1 | 含义 2 |
|---|---|---|
| **TBI** | Toe-Brachial Index（PAD）| Traumatic Brain Injury（Neuro）|
| **AD** | Alzheimer Disease | Autosomal Dominant |
| **MS** | Multiple Sclerosis | Mitral Stenosis |
| **PE** | Pulmonary Embolism | Physical Exam / Preeclampsia |
| **HR** | Heart Rate | Hazard Ratio |
| **PVD** | 旧术语（含动+静脉），避免使用 | |

## H.2 同人不同评分

| 评分 | 区别 |
|---|---|
| **Wells DVT** | 9 项，≥ 2 likely → US |
| **Wells PE** | 7 项，> 4 likely → CTPA。**独有"PE 是最可能诊断" +3 主观项** |

⚠️ 看到 stem 写"Wells score = X" 不说清楚 → 看上下文：腿 → DVT；胸 → PE。

## H.3 治疗哲学陷阱

1. "动脉病用 compression" → ❌ 致命
2. "DVT 妊娠用 DOAC" → ❌ 只能 LMWH
3. "Ischemic Stroke 急性立刻抗凝" → ❌ 24h 后才用 ASA
4. "MI 立刻溶栓" → ❌ 首选 PCI 90 min，tPA 仅无 PCI 时
5. "Massive PE 等 CTPA 确诊" → ❌ 不稳定 → tPA 经验性给
6. "ALI 等影像" → ❌ 立刻 IV Heparin，6h 窗口
7. "Cilostazol 用于 HF 患者的 PAD" → ❌ 禁忌
8. "Phlegmasia 抗凝就够" → ❌ 必须 CDT/取栓
9. "Buerger disease 抗血小板" → ❌ 唯一有效 = 戒烟
10. "所有 rest pain 都立刻 revasc" → ❌ 看 CLI 标准 + 可耐受 + 可 revasc 血管

## H.4 时间窗

| 急症 | 时间窗 | 错过后果 |
|---|---|---|
| MI - PCI | **90 min**（door-to-balloon）| 心肌坏死扩大 |
| MI - tPA | < 12 h | 再灌注无益 |
| Stroke - tPA | **4.5 h** | 出血风险↑↑↑ |
| Stroke - 取栓 | **24 h**（LVO 选择性）| 神经损伤不可逆 |
| ALI | **6 h** | 肌肉坏死不可逆 |
| Phlegmasia | 数小时-天 | 截肢 / 死亡 |

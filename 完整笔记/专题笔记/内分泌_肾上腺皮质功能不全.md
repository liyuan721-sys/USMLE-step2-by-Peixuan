# _衍生_肾上腺皮质功能不全决策树（Adrenal Insufficiency & Crisis）

> [!info] 本笔记定位
> 跨学科整合：**PAI 五联实验室指纹** → **三类 AI 鉴别**（原发/继发/三发）→ **急性危象识别 + 处理** → **围手术期 stress dose** → **难治性休克 / 嗜酸增高完整鉴别**。
> 
> **核心思维**：症状不特异时回到实验室。**PAI 五联 (Na↓/K↑/HCO3↓/Glu↓/Eos↑) 是 USMLE 内分泌最特征的指纹组合**。
> 
> **来源**：错题 Q17169（2026-05-14）+ 后续 AI / Cushing 撤药 / etomidate AI / 儿童 CAH 题积累

---

## Part 1. 核心识别框架（指纹思维 + 红圈扫读）

> [!tip] 一句话总览
> **AI 的症状极不特异，必须用"实验室指纹 + 阴性体征 + 慢性病程"三层锁定。** 跳过症状迷雾，直接看实验室异常组合。

### 1.1 PAI 五联实验室指纹 ⭐⭐⭐ 必背锚点

| 指标 | 改变 | 一句话机制 |
|---|---|---|
| **Na** | ↓（130-ish）| Aldo↓ 排 Na + Cortisol↓ 致 ADH 反向↑ |
| **K** | ↑（5.0-5.5）| Aldo↓ → K 潴留 |
| **HCO3** | ↓（20-ish）| Aldo↓ → H+ 排泌↓（正常 AG 代酸，HARDASS 的 A）|
| **Glucose** | ↓（< 70）| Cortisol↓ → 糖异生↓ + 外周糖摄取↑ |
| **Eos** | ↑（> 5%, 常 8-15%）| Cortisol↓ → 解除 Eos 抑制 |

**判读规则**：
- 见任意 **3 项** → 应该想 PAI
- 见 **4 项** → 高度怀疑
- 见 **5 项** → 几乎确诊

### 1.2 阴性体征红圈扫读法

> [!warning] 读题流程
> ```
> 第一遍：找阳性核心症状 → 形成 1-2 个候选诊断
> 第二遍：扫所有 no / without / negative / normal / unremarkable → 红圈标记
> 第三遍：每个候选诊断 × 阴性体征逐一核对
> 反推法：如果答案是 X，stem 应该写什么？现在写了什么相反的？
> ```

**高频"阴性触发语言"**：

| 表达 | 含义 |
|---|---|
| No X / Without X / Denies X | 主动排除 X |
| Negative for X | X 阴性 |
| X is normal / Unremarkable X | 排除 X 系统疾病 |
| No evidence of X | 排除 X |
| **No improvement with X** | ⭐ 关键线索：排除对 X 敏感的病 |
| Imaging / Lab unrevealing | 排除该系统器质性病 → 多提示功能性 / 内分泌 |

## Part 2. AI 三类完整鉴别 ⭐⭐⭐

```
HPA 轴解剖（自上而下）：
┌──────────────┐
│ Hypothalamus │ → CRH ↓ → 三发 AI（外源激素停药 / 下丘脑病变）
└──────┬───────┘
       ▼
┌──────────────┐
│  Pituitary   │ → ACTH ↓ → 继发 AI（Sheehan / 垂体瘤 / apoplexy）
└──────┬───────┘
       ▼
┌──────────────┐
│  Adrenal     │ → cortisol / aldo / androgens ↓ → 原发 AI (PAI / Addison's)
└──────────────┘
```

### 2.1 三类 AI 对比表

| 维度 | **Primary (PAI)** | **Secondary** | **Tertiary** |
|---|---|---|---|
| 病变位置 | 肾上腺本身 | 垂体 | 下丘脑 / 长期外源激素 |
| **Cortisol** | ↓↓ | ↓↓ | ↓↓ |
| **Aldosterone** | **↓↓** | 正常 | 正常 |
| **Androgens** | ↓ | ↓ | ↓ |
| **ACTH** | **↑↑**（反馈无）| ↓ | ↓ |
| **CRH** | ↑ | ↑ | ↓ |
| Na | ↓↓ | ↓（cortisol↓→ADH↑致）| ↓（轻）|
| **K** | **↑↑** | **正常** | 正常 |
| Glucose | ↓ | ↓ | ↓ |
| Eos | ↑ | ↑ | ↑ |
| **皮肤色素沉着** | **有**（ACTH↑→MSH 效应）| **无** | **无** |
| 危象严重度 | **最重** | 中 | 中 |
| 响应升压药 | refractory | 部分有效 | 部分有效 |

> [!success] 一句话记忆
> **"原发缺三样（cortisol + aldo + androgens），继发 / 三发只缺一样（仅 cortisol）。"** 这就是 PAI 危象最凶 + refractory to fluids / pressors 的根本原因。

### 2.2 各类典型病因

| 类型 | 常见病因 |
|---|---|
| **PAI**（原发）| ① 自身免疫破坏（80%，最常见，Addison's）② 结核 ③ 转移癌 ④ 真菌感染 ⑤ Adrenal hemorrhage（Waterhouse-Friderichsen 脑膜炎球菌败血症 / 抗凝过量）⑥ AIDS-related ⑦ 先天性 **CAH**（儿童；**21-OH 缺乏** = 经典"新生儿低钠 + 高钾 + 低血糖 + 休克 + 17-OHP 飙升"，皮质醇 / 醛固酮双缺；**11β-OH 缺乏** = 反向表现 = **高血压 + 低钾**，因为 11-deoxycorticosterone 类似 aldo）⑧ APS-I / II 多腺体综合征 |
| **Secondary**（垂体）| ① 垂体瘤（含术后）② Sheehan 综合征 ③ Pituitary apoplexy ④ 头部外伤 ⑤ 放疗 ⑥ Empty sella |
| **Tertiary**（下丘脑 / 外源）| ① **长期糖皮质激素 > 3 周突然停药**（最常见）② 下丘脑肿瘤 / 外伤 / 放疗 |

## Part 3. Adrenal Crisis（急性危象）

### 3.1 三大病因

| 病因 | 典型情境 |
|---|---|
| **Adrenal hemorrhage / infarction** | Waterhouse-Friderichsen / 抗凝过量 / 严重应激致肾上腺梗死 |
| **Illness / injury / surgery in chronic AI patient** | 已知 PAI 患者遇应激 / **未诊断 PAI 患者首次失代偿（如 Q17169）** |
| **Pituitary apoplexy** | 垂体卒中 → 急性 ACTH↓ → 急性继发 AI |

### 3.2 危象临床特征

| 系统 | 表现 |
|---|---|
| 循环 | 严重低血压、心动过速、**refractory to fluids and pressors** |
| 代谢 | 低血糖 / 代谢性酸中毒 |
| 电解质 | Na↓↓ / K↑↑（仅 PAI）|
| GI | 严重 N/V / 腹痛（可被误为急腹症）|
| 神经 | 嗜睡、谵妄、昏迷 |
| 实验室 | PAI 五联指纹完整出现 |

### 3.3 危象处理流程

```
怀疑 Adrenal Crisis（不必等检验确诊）：
┌────────────────────────────────────────────────┐
│ Step 1: Hydrocortisone 100 mg IV bolus         │
│         （不能延迟，怀疑即给）                  │
├────────────────────────────────────────────────┤
│ Step 2: 大量补液（NS or D5NS 1-2 L 快速）       │
│         同时补糖（如低血糖）                    │
├────────────────────────────────────────────────┤
│ Step 3: 维持 Hydrocortisone 200 mg / 24h 输注  │
│         或 50 mg IV q6h                         │
├────────────────────────────────────────────────┤
│ Step 4: 治疗诱因（感染 → 抗生素；手术 → 围术期）│
├────────────────────────────────────────────────┤
│ Step 5: 病情稳定后 → 减量 + 加 fludrocortisone │
│         （仅 PAI 需要，继发 / 三发不必）        │
├────────────────────────────────────────────────┤
│ Step 6: 完善诊断检验                            │
│         - ACTH stim test（不被 dexa 干扰）      │
│         - 血浆 ACTH / cortisol / aldo / renin   │
│         - 抗肾上腺抗体（自免）                  │
│         - CT 肾上腺（如怀疑肿瘤 / 出血）        │
└────────────────────────────────────────────────┘
```

### 3.4 关键药物选择

| 糖皮质激素 | 糖皮质活性 | **盐皮质活性** | 用途 |
|---|---|---|---|
| **Hydrocortisone**（cortisol）| 1× | **有**（大剂量时显著）| **首选** — 同时部分替代 aldosterone |
| **Dexamethasone** | 25-30× | 无 | 次选；用于**不干扰 cortisol 测定**时（可同时做诊断）|
| **Prednisone** | 4× | 弱 | 不首选 |
| **Methylprednisolone** | 5× | 弱 | 不首选 |
| **Fludrocortisone** | 弱（10×）| **极强**（150-200×）| **后续维持**用，急性危象不够强糖皮质 |

> [!warning] 为什么不在急性危象用 fludrocortisone？
> ① 仅口服（急性休克吸收差）② 糖皮质活性不够 ③ Hydrocortisone 大剂量已提供足够盐皮质效应

## Part 4. 慢性 AI 维持治疗

### 4.1 替代治疗方案

| AI 类型 | 糖皮质 | 盐皮质 |
|---|---|---|
| **PAI** | Hydrocortisone 15-25 mg/天分次（早 2/3 + 下午 1/3）或 prednisone 5 mg | **Fludrocortisone 0.05-0.2 mg/天** ⭐ 必须加 |
| **Secondary** | Hydrocortisone 同上 | 通常不需要（aldo 正常）|
| **Tertiary** | Hydrocortisone 同上 / 渐减原激素 | 通常不需要 |

### 4.2 患者教育（USMLE 高频）

- 永久戴 **medical alert bracelet**（识别身份用）
- 常备 **emergency hydrocortisone 注射剂**（家用）
- **生病时加倍剂量规则**（sick day rules）：
  - 发烧 / 感染 → 剂量加倍 × 3 天
  - 严重呕吐 / 腹泻 → 急诊静脉注射
  - 手术前 → 通知麻醉医生给 stress dose

## Part 5. 围手术期应激剂量激素 ⭐⭐⭐

> [!danger] 临床场景
> **慢性 AI 患者**或**长期口服激素 > 3 周的患者**做手术 → 必须给 stress dose，否则术中 crisis。

### 5.1 应激分级速查

| 级别 | 典型情境 | Stress Dose |
|---|---|---|
| **轻度** | 牙科 / 局部小手术 / 内镜（无镇静）/ 轻感染 | 维持原剂量 或 hydrocortisone 25 mg IV 单次 |
| **中度** | **内镜 + 镇静（Q17169 档次）** / 疝 / 胆囊 / 关节镜 / 肺炎 / 外伤 | Hydrocortisone **50 mg IV** 术前 + 25 mg q8h × 24h |
| **重度** | 大手术（心脏 / 腹 / 胸）/ 重症脓毒症 / 严重创伤 / 烧伤 | Hydrocortisone **100 mg IV** + 50 mg q8h × 48-72h |
| **危象** | 已发生 adrenal crisis | **100 mg IV bolus** + 200 mg/24h infusion |

> [!example] 高频 stem："长期激素 + 急诊外科"
> RA / IBD / 哮喘 / 移植患者长期口服 prednisone 5-20 mg/d > 3 周 → 阑尾炎 / 胆囊炎 / 肠梗阻 急诊开腹手术。
> 术中或术后突发顽固性低血压、对补液和升压药无反应 → **adrenal crisis（HPA 轴被外源激素长期抑制，无法应对应激）**。
> **First step：IV hydrocortisone 100 mg bolus**，然后维持。不要把这种"长期吃药患者"的低血压当成出血或脓毒症在补血补液。

### 5.2 "应激"完整清单

| 类别 | 应激源 |
|---|---|
| 手术 / 操作 | 任何手术 / 内镜 + 镇静 / 介入操作 / 拔牙 / 活检 |
| 感染 | 肺炎 / UTI / 蜂窝织炎 / 胃肠炎 / 流感 / COVID |
| 创伤 | 骨折 / 车祸 / 烧伤 / 头部外伤 |
| 代谢 | DKA / 甲危 / 严重低血糖 |
| 心血管 | MI / 卒中 / 大出血 / 休克 |
| 妊娠 | 分娩 / 剖宫产 / 妊娠剧吐 |
| **药物** | **Etomidate**（11β-羟化酶抑制）★ / Ketoconazole / Mitotane / **Rifampin**（加速类固醇代谢）|
| 撤药 | 长期糖皮质激素 > 3 周突然停药 |
| **甲状腺急症联动** | **Thyroid storm**：steroids（hydrocortisone）是 5 步治疗"4P + S"的一员 — 双重作用：① 抑制 T4 → T3 外周转化 ② 预防相对性 AI；**Myxedema coma**：**必须先给 glucocorticoid 再给 levothyroxine**（防潜在 AI 被 T4 加速代谢皮质醇而触发 crisis） |

> [!warning] Etomidate 冷知识
> **单次诱导剂量都能让 cortisol 合成停 24+ 小时**。Sepsis / 危重患者用 etomidate 后 24-48h 内出现 refractory hypotension → 考虑 etomidate 诱发的 relative AI。

## Part 6. 核心鉴别诊断

### 6.1 Adrenal Crisis vs Anaphylaxis

| 维度 | Adrenal Crisis | Anaphylaxis |
|---|---|---|
| 病程 | 慢性背景 + 急性触发 | 纯急性，分钟内 |
| 诱因 | 任何应激 | 特定过敏原（药 / 食 / 虫咬）|
| 皮肤 | 慢性色素沉着 / 无急性皮疹 | **荨麻疹 / 皮疹 / 血管性水肿** |
| 气道 | 正常 | 喉头水肿 / 喘鸣 |
| **SaO2** | 正常 | **多 < 90%** |
| HR | 心动过速 | 心动过速 |
| Na / K | Na↓ / K↑ | 正常 |
| Glucose | 低 | 正常 |
| HCO3 | 低 | 正常 / 轻乳酸升 |
| Eosinophils | **稳定升高** | 升高（**延迟数小时**，急性早期不一定升）|
| 对补液反应 | **Refractory** | 部分有效 |
| 第一步处理 | **Hydrocortisone IV + 补液** | **IM epinephrine** + 补液 + H1 / H2 + 类固醇 |

### 6.2 Eos↑ 完整鉴别决策树

```
见到 Eos↑ → 看伴随线索：
├─ 伴 Na↓ / K↑ / Glu↓ ────→ ★ PAI / Addison's
├─ 伴皮疹 / wheezing / 喉水肿 → Anaphylaxis
├─ 伴用药 + 皮疹 + 发热 + 器官损伤 → DRESS
├─ 伴旅行 / 移民史 ──────→ 寄生虫（Strongyloides / 钩虫 / 血吸虫）
├─ 伴哮喘 + 鼻炎 + 鼻息肉 ──→ AERD / EGPA (Churg-Strauss)
├─ 伴食管 / 食物过敏 ────→ 嗜酸性食管炎 / 胃肠炎
├─ 伴 AKI + 用药 + 皮疹 → AIN（嗜酸尿）
└─ 单纯 Eos↑↑ + B 症状 → 嗜酸性白血病 / HES
```

### 6.3 难治性休克鉴别（与 Q4212 共建）

| 类型 | 特征 |
|---|---|
| Adrenal crisis | 慢性背景 + 应激 + Na↓ / K↑ / Glu↓ + Eos↑ |
| 心源性 | 肺水肿 + JVD + S3 + 心电图改变 + Troponin↑ |
| 失血性（如 ruptured AAA）| 失血源 + Hb↓ + 凉性肢端 |
| 神经源性 | 脊髓损伤 + 心动**过缓**（与其他休克不同）+ 暖性肢端 |
| 脓毒症 | 发热 + WBC↑ / ↓ + 感染源 + 暖性肢端（早期）|
| 张力性气胸 | 单侧呼吸音消失 + 气管偏移 + SaO2↓ |
| 大块 PE | SaO2↓ + tachypnea + 右心负荷 + DVT 风险 |

### 6.4 PAI 反面镜像：Cushing 与 Conn 速查（指纹对照）

> [!info] 一句话
> PAI 是"皮质醇 + 醛固酮全缺"；Cushing 是"皮质醇过剩"；Conn 是"醛固酮过剩"。**三者实验室指纹完全镜像**，对照记忆效率最高。

| 维度 | **PAI** | **Cushing**（皮质醇↑↑） | **Conn / PA**（醛固酮↑↑） |
|---|---|---|---|
| **Na** | ↓↓ | 正常 / 略 ↑ | 正常 / 略 ↑ |
| **K** | ↑↑ | ↓ | ↓ |
| **HCO3** | ↓（代酸）| ↑（代碱）| ↑（代碱）|
| **Glucose** | ↓ | ↑ | 正常 |
| **BP** | 低 → 危象 refractory | **高** | **顽固高血压** |
| **Eos** | ↑ | ↓ | 正常 |
| 皮肤 | 色素沉着（PAI 独有） | 紫纹 / 痤疮 / 多毛 / moon face | — |
| **筛查** | 8 AM cortisol + ACTH | **24h UFC 或 LDDST 或 深夜唾液 cortisol（任 1 阳性即可）** | **ARR（早晨 PAC/PRA）** |
| **确诊** | **Cosyntropin stim test** | **ACTH → HDDST**（高 ACTH 时再做：能抑制 = 垂体；不抑 = 异位）| **Saline suppression test** |
| **定位** | 抗 21-OH 抗体 + 肾上腺 CT | ACTH 低 → 肾上腺 CT；ACTH 高 → MRI 垂体；HDDST 不抑 → 找异位（肺 SCLC）| 肾上腺 CT → 不明 / 双侧 → **AVS** |

> [!warning] ARR 筛查的 4 大适应证（高血压人群中）
> ① 顽固性 HTN（3 药联控失败）② HTN + 低钾（**自发或利尿剂诱发**）③ 年轻 HTN（< 30 岁）④ **意外瘤 (incidentaloma)** 患者
>
> **意外瘤三联必筛**：Pheo（**metanephrines**）+ Cushing（**LDDST 或 24h UFC**）+ 仅 HTN/低钾 患者加 PA（**ARR**）— 这是 endocrine 高频 next-step 题。

### 6.5 嗜铬细胞瘤作为 AI / Crisis 家族鉴别

> [!warning] 为什么放在这里
> Pheochromocytoma（嗜铬细胞瘤）属于**肾上腺髓质**肿瘤（而 PAI 是肾上腺**皮质**）。但它和 AI 共享"肾上腺起源 + 危象表现 + 围手术期高死亡率"特点，是 endocrine crisis 家族鉴别的关键反面。

| 维度 | **Adrenal Crisis (PAI)** | **Pheochromocytoma Crisis** |
|---|---|---|
| 来源 | 肾上腺**皮质** | 肾上腺**髓质** |
| 激素 | Cortisol / Aldo↓↓ | Catecholamines↑↑ |
| **BP** | **低 refractory** | **极高 + 阵发** |
| **HR** | 心动过速（失代偿）| **阵发性心悸 + 头痛 + 出汗**（三联）|
| 血糖 | **低** | **高**（catecholamines 抑制 insulin + 升糖）|
| 触发 | 应激 + 慢性 AI 背景 | 体位 / 手术 / 麻醉 / 造影 |
| 筛查 | 8 AM cortisol + ACTH | **血浆 / 24h 尿 metanephrines**（首选，比 catecholamines 灵敏）|
| 关联综合征 | APS-I / II | **MEN 2A / 2B**（与 MTC + PTH 三联）、**NF1**、**VHL** |
| 处理 | Hydrocortisone IV + 补液 | **先 α-blocker（phenoxybenzamine）后 β-blocker** 控压；最终手术切除 |

> [!danger] 高频陷阱
> - **MTC + Pheo**（MEN 2 患者）→ **必须先做嗜铬切除，再切甲状腺**（甲状腺手术应激可诱发未处理的嗜铬高血压危象 → 术中死亡）
> - **意外瘤**患者必查 metanephrines —— 即便没有典型三联症（可能阵发性分泌或亚临床）
> - **顽固 HTN + 不明原因高血糖 + 头痛 / 出汗 / 心悸** → 抽血查 metanephrines
> - 嗜铬危象单用 β-blocker = 致命（α 受体无对抗 → 高血压危象加重）

## Part 7. 呼吸 / 氧合数据解读训练

### 7.1 SaO2 判读

| SaO2 (on room air) | 解读 |
|---|---|
| 95-100% | 正常 |
| 92-94% | 临界 / 轻降，老年 / 慢病可接受 |
| < 90% | 明确低氧 |
| < 88% | 需补氧 |

### 7.2 SaO2 陷阱清单

| Stem 写法 | 真实含义 |
|---|---|
| SaO2 95% **on room air** | 真正正常 |
| SaO2 95% **on 2L NC** | ⚠️ 隐性低氧（吸氧才达 95%）|
| SaO2 95% **on 6L NC** | 严重低氧 |
| SaO2 正常但 **A-a gradient widened** | 早期 PE / ILD / 右-左分流 |
| SaO2 正常但 **carboxyhemoglobin↑** | CO 中毒（SaO2 假性正常）|
| SaO2 88% 但 patient comfortable | 可能 COPD baseline |

### 7.3 休克 × SaO2 速查

| 休克类型 | SaO2 |
|---|---|
| Anaphylaxis | **< 90%** |
| 大块 PE | **< 90%** |
| 心源性（左衰）| 多 < 92% |
| 张力性气胸 | < 90% |
| 脓毒症 | 正常或轻降 |
| 失血性（早）| 正常 |
| **Adrenal crisis** | **正常** ★ |
| 神经源性 | 正常 |

> [!success] 核心结论
> **严重休克 + SaO2 正常 = 非肺源性休克**。提示血管 / 容量 / 内分泌 / 神经源性。Adrenal crisis 是这一类的高频代表。

## Part 8. 诊断检验

### 8.1 确诊流程

```
怀疑 AI：
┌────────────────────────────────────────────────┐
│ Step 1: 8 AM Cortisol + ACTH（同时抽）         │
│   - Cortisol < 3 μg/dL → 几乎确诊 AI            │
│   - Cortisol > 18 μg/dL → 排除 AI               │
│   - 中间值 → 进一步刺激试验                    │
├────────────────────────────────────────────────┤
│ Step 2: ACTH stim test (Cosyntropin 250 μg IV) │
│   - 测 0 / 30 / 60 min cortisol                │
│   - Cortisol 应升至 > 18 μg/dL                  │
│   - 不升 → AI 确诊                             │
├────────────────────────────────────────────────┤
│ Step 3: 区分原发 vs 继发                       │
│   - ACTH 高 → Primary                           │
│   - ACTH 低或正常 → Secondary / Tertiary       │
├────────────────────────────────────────────────┤
│ Step 4: 病因检查                                │
│   - Primary: 抗 21-羟化酶抗体（自免）+ CT 肾上腺│
│   - Secondary: MRI 垂体 + 其他垂体激素          │
└────────────────────────────────────────────────┘
```

### 8.2 检验陷阱

> [!warning] Hydrocortisone vs Dexamethasone 选择
> **危象处理**：可以用 dexamethasone（**不干扰 cortisol 测定**），同时做 ACTH stim test 诊断。但理论首选仍是 hydrocortisone（含盐皮质活性）。
> **稳定后转换**：dexamethasone → hydrocortisone（更生理）

## Part 9. 高频陷阱总结

| 陷阱 | 正确认知 |
|---|---|
| ❌ 体重↓ / GI 症状 → 不会想 PAI（太不特异）| ✓ 必须靠"五联指纹 + 慢性病程"锁定，跳过症状迷雾 |
| ❌ Eos↑ = 过敏 | ✓ Eos↑ 在 PAI 是 cortisol 缺乏指纹；走 6 路决策树 |
| ❌ "GI 症状几个月"是来就诊原因 | ✓ USMLE 不写废话，慢性病程是诊断核心 |
| ❌ Refractory hypotension 多补液即可 | ✓ PAI 危象 refractory 因 α₁ 受体无 cortisol 不工作；**必须给 hydrocortisone** |
| ❌ PAI 危象主要因 cortisol 缺乏 | ✓ **主要由 aldosterone 缺乏驱动**（容量 + 电解质），cortisol 致升压药无效 |
| ❌ 急性危象用 fludrocortisone | ✓ Fludro 仅口服 + 糖皮质弱；急性用 hydrocortisone |
| ❌ 长期激素停药 = 安全 | ✓ > 3 周突然停 = 三发 AI 危象风险，必须渐减 |
| ❌ SaO2 95% = 低氧 | ✓ 正常下限；必看"on what FiO2"；on RA 时 95% 排除呼吸源性休克 |
| ❌ Etomidate 安全无副作用 | ✓ 单次剂量都能抑制 cortisol 合成 24h+；危重患者慎用 |
| ❌ Secondary AI 也有色素沉着 | ✓ 仅 PAI 有（ACTH↑→MSH 效应）；Secondary / Tertiary 无 |

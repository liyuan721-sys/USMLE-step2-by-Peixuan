## 一、分流第 0 步总钩 ⭐（看到题型，先跑这一步，别先认病原）

> [!tip] 共同失败模式
> 昨天 10 张全栽在**没跑"第 0 步分流"就被干扰项的病原名 / 旅行史 / 高热数字钩走**。下面每类题，先做左→右的第 0 步动作，病原自己会浮出来。

| 触发题型 | 第 0 步先扫什么（别先报病原） | 锁定 |
|---|---|---|
| **蜱传发热** | 有无 **血管内溶血 + 无脾（asplenia）** | 有 → Babesiosis（巴贝虫）→ 薄血涂片 Maltese cross |
| **肝占位 / 囊肿** | 先分 **囊 vs 脓肿** → 看 旅行腹泻 + 嗜酸 + 终末器官 | 光滑囊+子囊=包虫；单发低回声+旅行腹泻+嗜酸不高=阿米巴 |
| **生殖器溃疡** | 先摸 **痛 / 无痛 + 成簇否** | 痛 + 成簇 + 全身症状 → HSV（herpes simplex virus 单纯疱疹病毒）|
| **渗出性咽炎** | 先扫 **淋巴结范围 + 病程 + 乏力** | 全身淋巴结 + ≥2 周 + 显著乏力 → IM（infectious mononucleosis 传染性单核细胞增多症，EBV）→ Monospot |
| **新生儿 / 婴儿感染** | 先按 **月龄分层** | ≤28 天 → GBS（Group B Streptococcus B 族链球菌）（晚发与 IAP 产时抗生素预防无关）|
| **上呼吸道（鼻窦炎 / 中耳炎）** | 默认 **三联病原固定** | 不可分型 H. flu + 肺炎链 + 卡他莫拉 |
| **慢性消耗 + 肺空洞** | 记住 **TST（tuberculin skin test 结核菌素试验）在免疫低下不可信** | 老年 + CKD/DM + 空洞 + TST 阴 → TB（肺结核，pulmonary tuberculosis）（改 IGRA / 痰）|
| **皮肤栓塞 / 免疫现象** | 见 **Osler / Janeway** 直接定病 | → 感染性心内膜炎（IE）→ echo + 3 套血培 |
| **"选下一步 / 顺序"题** | **救命 > 确诊**（发热+瘀点+颈抵抗）| 立即 vancomycin（万古霉素）+ ceftriaxone（头孢曲松），不降温 / 不等 LP（腰穿 lumbar puncture）·CT |

## 二、10 张反复错 · 指纹默写（先默后展开）

### 🔴🔴🔴 第 3 次还错 — 天天默（已 `#考前必看`）

> [!question]- ① 无脾 + 美国东北 Ixodes 蜱 + 血管内溶血 = ？（Babesia 第 3 次错）
> **指纹**：**无脾（asplenia/脾切除）+ 美东北蜱（Ixodes scapularis）+ 血管内溶血（贫血 + 网织↑ + 黄疸/深色尿 + LDH↑）= Babesiosis（巴贝虫病，Babesia microti）**。
> **确诊**：薄血涂片见红细胞内 **Maltese cross（马耳他十字）**。**治**：阿托伐醌 atovaquone + 阿奇霉素 azithromycin。
> **陷阱**：Lyme（莱姆病，伯氏疏螺旋体，招牌 EM（erythema migrans 游走性红斑）、**无溶血**）/ RMSF（Rocky Mountain spotted fever 落基山斑疹热）—— 共感染常见 ≠ 本题确诊；**有溶血 + 无脾就是 Babesia，查血涂片不查 Lyme 血清**。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q3104]] ｜ [[完整笔记/专题笔记/感染/感染_蜱传疾病鉴别]]

> [!question]- ② 发热 + 瘀点/紫癜 + 颈抵抗 + 数小时恶化 → 第一步做什么？（流脑 第 3 次错）
> **指纹**：疑暴发 **脑膜炎球菌脑膜炎（meningococcal，发热 + 瘀点/紫癜 + 颈抵抗 + 数小时恶化 + 青少年聚集）→ 唯一第一步 = 抽血培养后立即 vancomycin（万古霉素）+ ceftriaxone（头孢曲松）（± 地塞米松 dexamethasone）**。**绝不为 LP / CT 推迟抗生素**。
> **陷阱**：被"高热 40℃"勾去想热射病而选**降温**——降温只用于**劳力性热射病（>40℃ + 脑病、无瘀点/颈抵抗）**；**有瘀点 + 颈抵抗 = 脓毒症，不是热射病**。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q18591]]

### 🔴 第 2 次错 — 隔天轮默

> [!question]- ③ ≤28 天新生儿 败血/脑膜炎 首位病原 = ？（GBS）
> **指纹**：**新生儿（≤1 月）败血/脑膜炎首位 = GBS（Group B Streptococcus，B 族链球菌 / 无乳链球菌 S. agalactiae）**（+ 大肠杆菌等 G⁻ 杆菌、李斯特菌 Listeria、HSV（herpes simplex virus 单纯疱疹病毒））。**≥1 月才是肺炎链球菌 / 脑膜炎奈瑟菌**。
> **要点**：晚发 GBS（>1 周，本例 12 天）= 水平接触传播，**IAP（intrapartum antibiotic prophylaxis 产时抗生素预防）不改变晚发率** → "母亲没做 IAP" ≠ 排除 GBS。新生儿败血凶兆：**低体温 + 白细胞减少 + 核左移**（不一定发热）。
> **陷阱**：Hib（新生儿罕见，母传抗体）/ 脑膜炎奈瑟菌（>1 月）。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q3671]] ｜ [[完整笔记/专题笔记/儿科/儿科_新生儿感染病原谱]]

> [!question]- ④ 发展中国家旅行 + 腹泻史 + 肝右叶单发低回声 + 嗜酸不高 = ？（阿米巴肝脓肿）
> **指纹**：**旅行 + 腹泻史 + 肝右叶单发低回声 + 嗜酸不高（原虫不致嗜酸↑）= 阿米巴肝脓肿（Entamoeba histolytica）→ 甲硝唑 metronidazole + 巴龙霉素 paromomycin**。
> **陷阱**：包虫（犬/牧区 + 蛋壳钙化大囊 + 嗜酸↑、忌穿刺）/ 化脓性脓肿（老年/中毒貌 + DM + 胆道·GI 源）。**嗜酸 2% 是支持阿米巴、反对蠕虫/包虫的硬钉**（只有侵袭性蠕虫才嗜酸↑）。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q2968]] ｜ [[完整笔记/专题笔记/感染/感染_肝脏占位囊肿鉴别]]

> [!question]- ⑤ 肝光滑囊肿 + 内含子囊（daughter cysts）+ 牧羊区/犬接触 = ？（包虫病 — 注意与④是镜像）
> **指纹**：**肝光滑囊肿 + 子囊 + 牧羊区/犬密切接触（吞虫卵）= 包虫病（Echinococcus granulosus，犬为终宿主、羊为中间宿主）→ albendazole 阿苯达唑 ± PAIR 经皮（>5cm/分隔）；禁穿刺活检（外溢→过敏性休克）**。
> **终末器官定病原**：**肝子囊 = 犬绦虫（包虫）**；**脑 = 猪带绦虫（Taenia solium → 脑囊虫病）**。
> **陷阱**：两次同向错选猪带绦虫（生猪肉）——猪带绦虫**肝囊罕见**，见"绦虫 + 囊"**先定器官**再答。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q3789]] ｜ [[完整笔记/专题笔记/感染/感染_肝脏占位囊肿鉴别]]

> [!question]- ⑥ 儿童 ≥10 天浓涕 + 咳无改善，病原三联 = ？（ABRS）
> **指纹**：**儿童 ABRS（acute bacterial rhinosinusitis 急性细菌性鼻窦炎，≥10 天浓涕+咳无改善）三联 = 不可分型流感嗜血杆菌 nontypeable H. influenzae（首位）+ 肺炎链球菌 + 卡他莫拉菌（同急性中耳炎 AOM, acute otitis media）→ 阿莫西林 amoxicillin ± 克拉维酸 clavulanate**。PCV13（13 价肺炎球菌结合疫苗，pneumococcal conjugate vaccine）后肺炎链占比↓ → H. flu 升首位。
> **陷阱**：金葡（金黄色葡萄球菌 S. aureus）= **慢性**鼻窦炎（>12 周）；GAS（group A Streptococcus 化脓性链球菌）= **咽炎**，极少鼻窦炎。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q3286]] ｜ [[完整笔记/专题笔记/感染/感染_抗生素高频20场景秒杀]]

> [!question]- ⑦ 痛 + 成簇浅溃疡 + 痛性淋巴结 + 全身症状 = ？（生殖器溃疡先分痛/无痛）
> **指纹**：**生殖器溃疡先分痛/无痛：痛 + 成簇浅溃疡 + 痛性区域淋巴结 + 全身症状（发热乏力）= 原发 HSV（herpes simplex virus 单纯疱疹病毒）生殖器疱疹 → 阿昔洛韦 acyclovir**。
> **陷阱**：梅毒（syphilis，T. pallidum 苍白密螺旋体）= **单个无痛硬下疳 chancre + 无痛淋巴结**（与本题正相反）；LGV（lymphogranuloma venereum 性病淋巴肉芽肿，衣原体 Chlamydia L1-3）= 初无痛小溃疡 → 数周后痛性化脓 buboes（腹股沟脓肿，时序不同）。**旅行史是干扰项**，别带去热带病。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q10068]] ｜ [[完整笔记/专题笔记/感染/感染_生殖器溃疡鉴别]]

> [!question]- ⑧ 老年 + 共病 + 数月消瘦 + 颈/肺门淋巴结 + 空洞 + TST 阴 = ？（老年 TB）
> **指纹**：**老年 + 共病（CKD 慢性肾病 chronic kidney disease / DM 糖尿病 diabetes mellitus）+ 数月厌食消瘦 + 颈/肺门淋巴结 + 空洞 + TST 阴 = 肺结核 TB（tuberculosis）**。**CKD（及 HIV/激素/重症 TB/营养不良/老年）→ 细胞免疫↓ → TST 假阴（anergy 无反应性）→ 阴性不能排除 TB** → 改 IGRA（interferon-γ release assay γ干扰素释放试验）或痰 AFB（acid-fast bacilli 抗酸杆菌）/ NAAT（nucleic acid amplification test 核酸扩增试验）。
> **陷阱**：GPA（granulomatosis with polyangiitis 肉芽肿性多血管炎，c-ANCA + 鼻窦破坏 + 血尿 + 多发结节，单空洞+肺门淋巴结不典型）/ 传单（IM 传染性单核细胞增多症）。**老年 TB 伪装成慢性消耗病**。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q19418]] ｜ [[完整笔记/专题笔记/感染/感染_TB与NTM肺病鉴别路径]]

> [!question]- ⑨ 青年 + ≥2 周咽痛 + 渗出扁桃体 + 全身淋巴结 + 显著乏力 = ？（IM/EBV）
> **指纹**：**青年 + ≥2 周迁延发热咽痛 + 渗出性扁桃体 + 全身淋巴结（颈+腋+腹股沟）压痛 + 显著乏力 ± 脾大 = IM（infectious mononucleosis 传染性单核细胞增多症，EBV EB 病毒）→ 嗜异性抗体 heterophile / Monospot**（首周 25% 假阴；<4 岁用 EBV 血清学）。处理：**避接触性运动 ≥4 周防脾破裂**。
> **陷阱**：GAS（group A Streptococcus 化脓性链球菌）咽炎（仅**颈部**淋巴结 + <1 周 + 无显著乏力）；急性 HIV（人类免疫缺陷病毒，**无痛**淋巴结 + 皮疹 + 腹泻，渗出少——仍应低门槛查 HIV，但非本题"最可能确诊"）。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q4354]] ｜ [[完整笔记/专题笔记/感染/感染_单核样综合征鉴别]]

> [!question]- ⑩ 亚急性发热 + Osler 结节 + 肾小球肾炎 + 贫血 + ESR↑ → 确诊做什么？（IE）
> **指纹**：**亚急性发热 + Osler 结节（指尖痛性紫色结节）+ GN（glomerulonephritis 肾小球肾炎，血尿 / RBC red blood cell 红细胞管型）+ 正细胞贫血 + ESR（erythrocyte sedimentation rate 红细胞沉降率）↑ = 感染性心内膜炎（IE, infective endocarditis）→ 确诊 = 超声心动图 echocardiography + 3 套血培养（改良 Duke 标准）**。
> **口诀**：**Osler = Ouch（痛、指尖、免疫）；Janeway = 平、掌跖、无痛、栓塞**。
> **陷阱**：IE 也可 RF（rheumatoid factor 类风湿因子）阳（免疫现象），**但那是诊断后伴随发现，不是确诊手段**（选 RF 是陷阱）；PSGN（post-streptococcal glomerulonephritis 链球菌感染后肾小球肾炎）前驱咽炎/脓皮病、**无 Osler 结节**。别被"GN/关节痛"牵走。
> 🔗 [[mistakes/uworld-mistakes_2026-06#^Q4668]] ｜ [[完整笔记/专题笔记/心内/心内_感染性心内膜炎]]

# AIDS 机会性感染按 CD4 谱

> [!info] 笔记定位
> **HIV 机会性感染（opportunistic infection, OI）的"CD4 钟表"。**
> Step 2 CK 的 HIV-OI 题 90% 靠两步秒杀：① **CD4 落在哪一档** → 锁定病原候选 → ② **再用一条鉴别钥匙**（血便 vs 水样 / 单灶 vs 多灶 / 溃疡形态）定到唯一病原 → 对口药。
> 用法：先死记 §二 CD4 分层总表 + §三预防阈值，再用 §四-六三张鉴别速查表收尾。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q2268]]（MAC）+ [[mistakes/uworld-mistakes_2026-06#^Q3917]]（CMV 结肠炎）。

---

## 一、设计哲学 — 两步定位法

```
Step 1：看 CD4 落哪一档 → 缩小病原范围
  <500  口腔念珠菌 / OHL / KS / 细菌性肺炎 / TB / 带状疱疹
  <200  PCP（卡氏肺孢菌肺炎）→ 此线起 TMP-SMX 预防
  <150  播散性组织胞浆菌（Histoplasma）
  <100  弓形虫脑炎 / 隐球菌脑膜炎 / 食管念珠菌
  <50   MAC / CMV / 原发 CNS 淋巴瘤 / PML
        │
Step 2：用一条鉴别钥匙定唯一病原（见 §四-六）
  腹泻：血便→CMV / 水样+高热→MAC / 重度水样→Crypto
  CNS：多发环形→Toxo / 单发→淋巴瘤 / 不强化无占位→PML
  食管：白斑→Candida / 大线状溃疡→CMV / 小簇溃疡→HSV
```

> [!tip] 核心心法
> CD4 是"能不能得"，鉴别钥匙是"到底是哪个"。两题（MAC vs CMV）同在 CD4<50，正因如此**只能靠 Step 2 的钥匙（血 vs 水）分开**——这就是 UW 反复设的陷阱。

## 二、CD4 分层总表 ⭐⭐⭐（必背）

> [!danger] 这张表是本笔记的核心
> 每个 CD4 阈值都是一道高频题的钥匙。**默写到全对为止。**

| CD4 阈值 | 新增风险病原 | 经典画像 | 对口治疗 |
|---|---|---|---|
| **<500** | 口腔念珠菌（thrush）；口腔毛状白斑 OHL（EBV）；Kaposi 肉瘤（HHV-8）；细菌性肺炎（肺炎链球菌）；带状疱疹（VZV）；肺结核（任何 CD4，此线风险升） | thrush 白膜可刮；OHL 侧舌**不可刮**；KS 紫红斑块 | Candida → 局部/氟康唑；KS → 优化 ART |
| **<200** | **PCP（Pneumocystis jirovecii 肺炎）** ⭐ | 亚急性干咳 + 劳力性呼吸困难 + **双肺弥漫间质浸润** + **LDH↑** + 运动后血氧↓ | **TMP-SMX**（高剂量）+ 激素（PaO₂<70 或 A-a>35）|
| **<150** | 播散性组织胞浆菌（Histoplasma capsulatum） | 发热 + 体重↓ + 肝脾大 + 全血细胞↓；尿/血抗原 | Amphotericin B → Itraconazole |
| **<100** | **弓形虫脑炎（Toxoplasma）** ⭐；**隐球菌脑膜炎（Cryptococcus）** ⭐；食管念珠菌 | Toxo：多发环形强化灶（基底节）；Crypto：亚急性头痛 + ↑颅压；食管炎吞咽痛 | Toxo → 乙胺嘧啶 + 磺胺嘧啶 + 亚叶酸；Crypto → Ampho B + Flucytosine → Fluconazole；食管 Candida → Fluconazole |
| **<50** | **MAC（播散性）** ⭐；**CMV（视网膜炎/结肠炎/食管炎）** ⭐；原发 CNS 淋巴瘤（EBV）；PML（JC 病毒）；Bartonella（杆菌性血管瘤病）| MAC：发热 + 体重↓ + 腹泻 + ALP↑ + 血培 NTM；CMV 视网膜：pizza-pie 出血渗出 | MAC → **大环内酯 + ethambutol**；CMV → Ganciclovir；淋巴瘤 → 化疗/放疗 + ART；PML → 优化 ART |

> [!success] CD4 阈值口诀
> - **500 → 黏膜 + 皮肤**（thrush / OHL / KS / 带状疱疹）
> - **200 → PCP**（"200 喘不上气"，此线起 TMP-SMX 预防）
> - **150 → 组织胞浆菌**
> - **100 → 脑 + 食管**（Toxo / Crypto / 食管 Candida）
> - **50 → 钱花光了**（MAC / CMV / 淋巴瘤 / PML —— 最深的坑）

## 三、一级预防阈值 + 何时停 ⭐（高频）

> [!warning] 预防 = "在病原能感染之前就开始吃药"
> Step 2 高频问"CD4 多少开始/停止预防"。

| CD4 阈值 | 预防对象 | 药 | 备注 |
|---|---|---|---|
| **<200** | **PCP** | **TMP-SMX** | 也顺带覆盖弓形虫；CD4<200 或有口腔念珠菌/AIDS 定义病即起 |
| **<100**（+ Toxo IgG ⊕） | **弓形虫** | **TMP-SMX** | 通常已因 PCP 在吃 TMP-SMX，自动覆盖 |
| **<50** | **MAC** | （阿奇霉素，每周）| ⚠️ **现已不再常规推荐**——若及时启动 ART，因成本/低风险/耐药而取消 |

> [!tip] 何时停预防（同样高频）
> ART 有效后 **CD4 回升 >200 持续 ≥3 个月 → 停 PCP 预防**。核心原则：**免疫重建 = 撤预防**。

> [!info] 与 PEP 区别
> 本表是**长期一级预防**（CD4 阈值驱动），与暴露后短程 [[完整笔记/专题笔记/感染/感染_暴露后预防PEP速查]] 是两套逻辑，别混。

## 四、HIV 腹泻病因鉴别 ⭐（血 vs 水）

> [!danger] CD4<50 双雷区：MAC vs CMV
> 两者都 CD4<50 + 腹泻 + 体重↓，**靠大便性状分**：CMV = 血便（小量频繁）；MAC = 水样 + 高热。错题 [[mistakes/uworld-mistakes_2026-06#^Q3917]] 正是此陷阱。

| 病原 | CD4 | 腹泻特征 | 鉴别钥匙 |
|---|---|---|---|
| **Cryptosporidium** | <180 | **重度水样**、低热、体重↓ | 大量水样、慢性 |
| **Microsporidium / Isosporidium** | <100 | 水样 + 腹绞痛 | **罕发热** |
| **MAC** | <50 | **水样 + 高热(>39°C)** + 体重↓ | 高热 + 血培 NTM + ALP↑ |
| **CMV 结肠炎** | <50 | **频繁、小量、血便 + 腹痛** | **血便（hematochezia）** = 唯一血性 |

> [!question]- Q：AIDS + CD4<50 + 间断血性腹泻 + 腹痛，最可能病原？
> **CMV 结肠炎**。CD4<50 的腹泻里只有 CMV 是**血性、小量频繁**；MAC/Crypto/Microsporidia 都是水样。诊断结肠镜活检（核内嗜酸 + 胞浆嗜碱包涵体），治 ganciclovir + ART，**活动性 CMV 必查视网膜炎**。

## 五、HIV 颅内病变鉴别 ⭐（影像 + 血清）

| 病变 | 病原 | 影像特征 | 关键线索 | 治疗 |
|---|---|---|---|---|
| **弓形虫脑炎** | Toxoplasma | **多发**环形强化、基底节、有占位水肿 | **Toxo IgG ⊕**；经验治疗 2 周无改善 → 活检 | 乙胺嘧啶 + 磺胺嘧啶 + 亚叶酸 |
| **原发 CNS 淋巴瘤** | EBV | **单发**、弱环形强化 | CSF EBV ⊕；thallium SPECT ⊕；Toxo IgG 常阴 | 化疗/放疗 + ART |
| **PML** | JC 病毒 | **白质**多灶、**不强化、无占位** | 亚急性局灶神经缺损 + 认知下降 | **优化 ART**（无特异抗病毒）|
| **隐球菌脑膜炎** | Cryptococcus | 多无占位（脑膜为主）；↑颅压 | India ink / **隐球菌抗原（CrAg）**⊕；亚急性头痛 | Ampho B + Flucytosine → Fluconazole；**反复 LP 降颅压** |

> [!success] 颅内三连记忆
> **多发环形 = 虫（Toxo）；单发 = 瘤（淋巴瘤）；不强化白质 = 病毒（PML）。** 隐球菌看脑膜 + CrAg + 高颅压。

## 六、HIV 食管炎鉴别 ⭐

| 病原 | 内镜形态 | 伴随 | 治疗 |
|---|---|---|---|
| **Candida** | 白色斑块/伪膜 | 多伴口腔 thrush；最常见 | **经验氟康唑**；无效再 EGD |
| **CMV** | **大的、线状/单个深溃疡** | CD4<50 | Ganciclovir |
| **HSV** | **小而多发、簇状、punched-out 溃疡** | 口唇疱疹史 | Acyclovir |

> [!tip] 食管炎策略
> AIDS + 吞咽痛 + 口腔 thrush → **先经验氟康唑**（最常见 Candida），**不先内镜**；氟康唑无效才 EGD 活检区分 CMV（大线状溃疡）vs HSV（小簇溃疡）。

## 七、病原 → 对口药配对表（治疗速查）

| 病原 | 一线治疗 |
|---|---|
| PCP（Pneumocystis jirovecii） | **TMP-SMX** + 激素（PaO₂<70 / A-a>35）|
| 弓形虫脑炎 | **乙胺嘧啶 + 磺胺嘧啶 + 亚叶酸** |
| 隐球菌脑膜炎 | **Amphotericin B + Flucytosine** → Fluconazole 巩固 |
| 播散性 MAC | **Macrolide（克拉/阿奇）+ Ethambutol** |
| CMV（视网膜/结肠/食管） | **Ganciclovir / Valganciclovir** |
| 食管/口腔念珠菌 | **Fluconazole**（口腔轻症可局部）|
| 播散性组织胞浆菌 | **Amphotericin B** → Itraconazole |
| PML（JC 病毒） | **优化 ART**（无特异药）|
| 原发 CNS 淋巴瘤（EBV） | 化疗/放疗 + ART |
| Kaposi 肉瘤（HHV-8） | 优化 ART（重病加化疗）|

> [!danger] 反复出现的"对口药"陷阱
> - **pyrimethamine（乙胺嘧啶）= 弓形虫**，不是 MAC（错题 [[mistakes/uworld-mistakes_2026-06#^Q2268]] 即误选 pyrimethamine 给 MAC）
> - **MAC = 大环内酯 + ethambutol**，不是 INH（那是 TB）
> - 血培循环里是**非结核分枝杆菌（NTM）= MAC**，不是 TB

## 八、Memory Hook（救命口诀总集）

```
CD4 钟表
  500 → 黏膜皮肤（thrush/OHL/KS/带状疱疹）
  200 → PCP（起 TMP-SMX 预防）
  150 → 组织胞浆菌
  100 → 脑 + 食管（Toxo/Crypto/食管Candida）
  50  → 最深坑（MAC/CMV/淋巴瘤/PML）

鉴别钥匙
  腹泻：血便→CMV / 水样+高热→MAC / 重度水样→Crypto
  CNS：多发环形→Toxo / 单发→淋巴瘤 / 不强化白质→PML / 脑膜+CrAg→隐球菌
  食管：白斑→Candida（先经验氟康唑）/ 大线溃疡→CMV / 小簇溃疡→HSV

对口药陷阱
  pyrimethamine = 弓形虫（不是 MAC）
  MAC = 大环内酯 + ethambutol（不是 INH）
  血里分枝杆菌 = NTM = MAC（不是 TB）

预防阈值
  <200 PCP（TMP-SMX，顺带 Toxo）
  CD4>200×3 月 → 停 PCP 预防
  <50 MAC 预防已不再常规推荐
```

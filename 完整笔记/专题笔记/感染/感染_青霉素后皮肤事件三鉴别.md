# 青霉素后皮肤事件三鉴别（过敏 / 赫氏 / 传单）

> [!info] 笔记定位
> "**用了青霉素 (penicillin) / 氨基青霉素 (amoxicillin·ampicillin) 之后皮肤出事**"是 Step 2 CK 高频陷阱——三件事**机制、时相、皮肤性质、要不要停药全不同**，却都"非典型药疹"易被一锅烩成"青霉素过敏"。
> 核心钥匙 = **时相**：分钟–1h（真过敏）/ 数小时（赫氏）/ 数天（传单）。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q3256]]（一期梅毒治疗→赫氏）+ [[mistakes/uworld-mistakes_2026-06#^Q4354]]（传单→氨基青霉素疹）。

---

## 一、三鉴别总表 ⭐⭐⭐

| | **IgE 真过敏** | **赫氏反应（Jarisch-Herxheimer）** | **传单疹（IM + 氨基青霉素）** |
|---|---|---|---|
| **时相** | **速发：分钟–1h** | **数小时（2–12h），<24h 自限** | **迟发：给药数天后** |
| **机制** | **I 型变态反应（IgE）** | **螺旋体被杀→菌体崩解释放类内毒素/脂蛋白→细胞因子风暴（TNF-α/IL-6）** | **病毒（EBV）感染期免疫活化 + 药物**，T 细胞介导倾向（**非 IgE、非过敏**）|
| **皮肤** | **荨麻疹/风团 + 血管性水肿**，可过敏性休克 | **不是新疹——原有皮损（梅毒疹/下疳）一过性加重变红** | **新发、泛发、对称麻疹样（morbilliform）斑丘疹** |
| **全身** | 喉头水肿/喘鸣/低血压（速发过敏）| **急性发热、寒战、肌痛、头痛、心动过速、低血压**（一过性）| 皮疹本身不急；全身来自底层 IM（咽炎/淋巴结/脾大/乏力，迁延）|
| **背景** | 任何 β-内酰胺暴露 | **螺旋体病**（梅毒/莱姆/钩体/回归热）首剂治疗 | 青少年咽炎 + **后颈淋巴结 + 脾大 + 异型淋巴 + 嗜异性抗体⊕** |
| **是否真过敏** | **是** | **否** | **否** |
| **要停青霉素吗** | **停 + 标记过敏**（未来避 β-内酰胺 / 脱敏）| **不停**，支持治疗（退热/补液）| 不是过敏；当时多停换药，**痊愈后再用青霉素通常不再出疹** |

> [!danger] 一句话定生死
> **别把"赫氏反应"和"传单疹"误标成青霉素过敏** → 会让病人**终身白白避开 β-内酰胺**，梅毒/莱姆等还可能因此停掉唯一有效治疗。**只有速发荨麻疹/血管水肿/休克才是真过敏。**

## 二、逐个拆解

### 2.1 IgE 真过敏（type I hypersensitivity）

> [!warning] 速发 = 真过敏
> 暴露后**分钟–1 小时**：**荨麻疹 (urticaria) / 血管性水肿 (angioedema) / 支气管痉挛 / 低血压 → 过敏性休克 (anaphylaxis)**。
> - **机制**：预存 IgE → 肥大细胞脱颗粒。
> - **处理**：**立即停药 + 标记青霉素过敏**；急性过敏按肾上腺素 (epinephrine) 抢救；未来需用则**脱敏 (desensitization)**。
> - ⚠️ 与"迟发斑丘疹"区分：**速发风团 = 真过敏**；迟发斑丘疹多良性/非 IgE。

### 2.2 赫氏反应（Jarisch-Herxheimer reaction）

> [!danger] 螺旋体首剂青霉素后数小时 = 赫氏，不是过敏
> - **机制**：青霉素杀死大量**螺旋体 (spirochete)** → 菌体崩解**释放类内毒素样脂蛋白** → **细胞因子风暴（TNF-α/IL-6/IL-8）**。
> - **时相**：首剂后**2–12h**，峰值 6–8h，**24h 内自限**。
> - **表现**：**发热、寒战、肌痛、头痛、心动过速、低血压** + **原有皮损（梅毒玫瑰疹/下疳）一过性加重**。
> - **处理**：**支持治疗（退热药、补液）+ 继续完成青霉素疗程**，不停不换。
> - **见于**：**梅毒（最经典，一/二期治疗高发）** / 莱姆病 (Lyme) / 钩端螺旋体病 (leptospirosis) / **回归热 (relapsing fever，反应最剧烈可致命)**。
> - ⚠️ **孕妇梅毒**：赫氏反应可致**发热 + 宫缩 → 早产/胎儿窘迫**，但**仍需治疗**（监测下进行）。

### 2.3 传单疹（IM + 氨基青霉素 morbilliform rash）

> [!warning] 传单给氨基青霉素 → 90%+ 麻疹样疹（非过敏）
> - **机制**：**EB 病毒 (EBV) 传染性单核细胞增多症 (IM)** 被误当链球菌咽炎 → 给**阿莫西林/氨苄西林 (amoxicillin/ampicillin)** → **几乎普遍出泛发麻疹样斑丘疹**；病毒期免疫活化 + 药物诱发，**非 IgE 过敏**。
> - **时相**：给药**数天后**（迟发型斑丘疹）。
> - **为何非过敏（判别钥匙）**：传单疹的全身不适（咽炎/淋巴结/脾大/乏力）来自**底层 IM 本身**，皮疹**不伴喉头水肿/喘鸣/低血压**；只有**速发气道/循环危象**才是 IgE 真过敏 → 故传单疹**不是过敏**。（⚠️ 总表"全身"行：喉头水肿/休克属 IgE 真过敏列，传单列是 IM 的咽炎/脾大，别看反）
> - **要点**：**痊愈后再用青霉素通常不再出疹** → **不应据此终身标"青霉素过敏"**。
> - **stem**：青少年咽炎渗出 + 后颈/全身淋巴结 + 脾大 + 乏力 + **异型淋巴细胞 + 嗜异性抗体⊕** → 答案查 **Monospot 排 IM**，别只想"过敏"。
> - 也见于 **巨细胞病毒 (CMV)** 等其他病毒感染期 + 氨基青霉素；同服 **别嘌醇 (allopurinol)** 增加该疹。

## 三、怎么三秒分开（考场决策树）

```
青霉素类用药后皮肤出事
├─ 分钟–1h：风团/血管水肿/喘鸣/低压 → IgE 真过敏 → 停+标记
├─ 数小时(<24h自限)：发热寒战+原皮损加重 + 螺旋体病(梅毒/莱姆/钩体/回归热)
│        → 赫氏反应 → 不停，支持治疗
└─ 数天后：新发泛发麻疹样斑丘疹 + 单核样(咽炎/脾大/异型淋巴/Monospot⊕)
         → 传单疹(EBV+氨基青霉素) → 非过敏，查Monospot
```

> [!tip] 三个维度一对就分开
> **时相（分钟 / 小时 / 天）× 皮肤（风团 / 原疹加重 / 新发斑丘疹）× 背景（任何暴露 / 螺旋体病 / 传单）**。
> 三件事**只有"真过敏"要停药并终身标记**；另两件都**不是过敏**。

## 四、Memory Hook

```
三个"青霉素后皮肤事件" = 三个时相：
  分钟–1h 风团 = IgE 真过敏（停+标记）        机制:IgE肥大细胞
  数小时 发热+原疹加重 = 赫氏(螺旋体,不停)     机制:菌死放毒→细胞因子
  数天后 新发斑丘疹 = 传单(EBV+氨基青霉素)      机制:病毒期免疫+药(非过敏)
只有真过敏要停药终身标记；赫氏/传单都不是过敏
赫氏见于:梅毒/莱姆/钩体/回归热;孕妇梅毒→可早产仍要治
传单:再用青霉素多半不再出疹,别白标过敏
```

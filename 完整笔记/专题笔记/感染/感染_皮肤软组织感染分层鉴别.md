# 皮肤软组织感染分层鉴别（SSTI）

> [!info] 笔记定位
> SSTI 题的解法 = **先定"感染到哪一层"→ 锁病原 → 选药**。把表浅到深部、毒素介导、特殊暴露串成一张分层速查。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q3642]] 脓疱疮 / [[mistakes/uworld-mistakes_2026-06#^Q3263]] 丹毒 / [[mistakes/uworld-mistakes_2026-06#^Q3530]] Ludwig / [[mistakes/uworld-mistakes_2026-06#^Q12168]] 创伤弧菌 / [[mistakes/uworld-mistakes_2026-06#^Q4317]] 咬伤。

---

## 一、按解剖层次分层 ⭐⭐⭐（核心）

```
表皮 epidermis          → 脓疱疮 Impetigo / SSSS(毒素)
  ↓
上层真皮+浅淋巴 dermis   → 丹毒 Erysipelas（隆起锐界、累及耳）
  ↓
深层真皮+皮下脂肪        → 蜂窝织炎 Cellulitis（平坦模糊、起病慢）
  ↓
毛囊                    → 毛囊炎/疖 Furuncle/痈 Carbuncle（化脓）
  ↓
筋膜 fascia             → 坏死性筋膜炎（剧痛>体征、捻发感、中毒）★外科急症
  ↓
肌肉 muscle            → 肌炎/气性坏疽（Clostridium）
```

| 层次 | 病 | 招牌 | 病原 |
|---|---|---|---|
| **表皮** | **脓疱疮 Impetigo** | 蜜黄色结痂（非大疱）/ 大疱（金葡毒素）| **S. aureus**、GAS |
| **表皮（毒素）** | **SSSS** | 弥漫剥脱、Nikolsky⊕、婴幼儿 | S. aureus 表皮剥脱毒素 |
| **上层真皮+淋巴** | **丹毒 Erysipelas** | **隆起锐界 + 起病快发热早 + 累及耳廓（Milian 征）** | **GAS（S. pyogenes）** |
| **深真皮+皮下** | **蜂窝织炎 Cellulitis** | **平坦模糊边界 + 起病慢发热晚** | GAS + **金葡**（化脓型含 MRSA）|
| **毛囊→脓肿** | **疖 Furuncle/痈 Carbuncle** | 化脓、脓液、波动感 | **S. aureus（含 MRSA）** |
| **筋膜** | **坏死性筋膜炎** | **剧痛>体征 + 捻发感 + 大疱 + 中毒休克 + 进展快** | **GAS（II 型）/ 多微生物（I 型）/ Vibrio / Clostridium** |
| **肌肉** | **气性坏疽** | 外伤后剧痛 + 捻发 + 肌坏死 | **Clostridium perfringens** |

## 二、丹毒 vs 蜂窝织炎（最高频对子）⭐

| | **丹毒 Erysipelas** | **蜂窝织炎 Cellulitis** |
|---|---|---|
| **层次** | 上层真皮 + 浅淋巴管 | 深层真皮 + 皮下脂肪 |
| **边界** | **隆起、锐利清晰** | **平坦、模糊** |
| **起病/发热** | **快；发热早** | 隐匿（数天）；发热晚 |
| **病原** | **GAS** | GAS + **金葡**（化脓型 = 金葡含 MRSA）|
| **特殊钥匙** | **累及外耳廓 = Milian 耳征**（耳无深层真皮 → 蜂窝织炎长不到）| 化脓（脓液/疖/痈）= 金葡 |

> 错题：[[mistakes/uworld-mistakes_2026-06#^Q3263]]（累及耳廓 → 丹毒 GAS）。

## 三、用药速查 ⭐

| 病 | 经验用药 |
|---|---|
| **脓疱疮·局限非大疱** | **外用 mupirocin**（[[mistakes/uworld-mistakes_2026-06#^Q3642]]）|
| **脓疱疮·广泛/大疱** | 口服抗葡萄球菌（cephalexin / dicloxacillin；疑 MRSA → clinda/doxy/TMP-SMX）|
| **丹毒** | IV（**ceftriaxone/cefazolin**）；轻症口服 **amoxicillin**（覆 GAS）|
| **非化脓性蜂窝织炎** | 覆 GAS+MSSA：**cephalexin / cefazolin**（链球菌为主）|
| **化脓性蜂窝织炎/脓肿** | **切开引流 + 覆 MRSA**（clindamycin / TMP-SMX / doxycycline；重症 vancomycin）|
| **坏死性筋膜炎** | **急诊外科清创 + 广谱（vanc + pip-tazo）± clindamycin（抑毒素）** ★不能只靠药 |
| **气性坏疽（Clostridium）** | 清创 + **penicillin + clindamycin** |
| **Ludwig angina（口底深部蜂窝织炎）** | **amp-sulbactam / clindamycin + 拔病牙 + 盯气道**（[[mistakes/uworld-mistakes_2026-06#^Q3530]]）|

> [!tip] 化脓 vs 非化脓决定要不要覆 MRSA
> **非化脓性蜂窝织炎 = 链球菌为主 → cephalexin/cefazolin（不必常规覆 MRSA）**；**化脓（脓液/脓肿/疖痈）= 金葡含 MRSA → 引流 + 覆 MRSA**。

## 四、特殊暴露 → 特定病原 ⭐

| 暴露钩子 | 病原 | 用药 |
|---|---|---|
| **猫/狗咬抓** | **Pasteurella multocida** | **amox-clavulanate**（[[mistakes/uworld-mistakes_2026-06#^Q4317]]）|
| **人咬** | Eikenella + 口腔厌氧 | amox-clav |
| **海水伤口/生牡蛎 + 肝病/血色病** | **Vibrio vulnificus**（出血性大疱、数小时暴发）| **ceftazidime + doxycycline** + 清创（[[mistakes/uworld-mistakes_2026-06#^Q12168]]）|
| **淡水伤口** | **Aeromonas hydrophila** | FQ / 三代头孢 + doxy |
| **处理鱼/肉/兽（手部）** | **Erysipelothrix rhusiopathiae** | penicillin |
| **玫瑰刺/园艺（结节性淋巴管）** | **Sporothrix**（孢子丝菌）| itraconazole |
| **热水浴缸/泡澡** | **Pseudomonas**（毛囊炎）| 多自限；重症抗铜绿 |

## 五、红旗：识别坏死性筋膜炎（别当普通蜂窝织炎）⭐

> [!danger] 看到这些 = 外科急症，别只开抗生素
> - **疼痛与体征不成比例（剧痛 >> 表面所见）**
> - **皮肤捻发感（crepitus）/ 影像示软组织气体**
> - **出血性大疱 / 皮肤坏死/麻木**
> - **全身中毒：休克、多器官衰竭、进展以小时计**
> - 危险宿主：糖尿病、肝病/血色病（Vibrio）、免疫抑制
>
> 处理：**立即外科清创 + 广谱抗生素（vanc + pip-tazo ± clindamycin 抑毒素）** —— 抗生素**不能替代手术**。

## 六、Memory Hook

```
分层定病：表皮=脓疱疮/SSSS(金葡毒素) → 上真皮淋巴=丹毒(GAS,隆起锐界累及耳)
        → 深真皮皮下=蜂窝织炎(GAS+金葡) → 毛囊脓肿=金葡 → 筋膜=坏死性筋膜炎(急症)
化脓与否定 MRSA：非化脓→链球菌(cephalexin)；化脓→金葡MRSA(引流+覆MRSA)
暴露定特殊菌：猫狗咬=Pasteurella(amox-clav)；海水+肝病=Vibrio(ceftaz+doxy)；
            淡水=Aeromonas；鱼肉=Erysipelothrix；玫瑰刺=Sporothrix
红旗坏死筋膜炎：剧痛>体征+捻发+大疱+中毒 → 手术清创(药不够)
```

# 暴露后预防 PEP 速查（密接给谁、用什么）

> [!info] 笔记定位
> 回答"**哪些传染病要给密切接触者预防用药 + 给什么**"。分两类：① **化学预防**（抗生素/抗病毒）② **暴露后免疫预防**（Ig/疫苗）。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q3906]]（百日咳 PEP）+ [[mistakes/uworld-mistakes_2026-06#^Q3905]]（百日咳免疫衰减）。

---

## 一、密接化学预防（抗生素/抗病毒）⭐⭐⭐

| 病 | 给密接什么 | 关键要点 |
|---|---|---|
| **脑膜炎球菌** | **Rifampin / Ciprofloxacin / Ceftriaxone（孕妇首选 IM）** | 密接**不论疫苗史**；飞沫；指示病例若用 penicillin 治也要补一剂（不清除鼻咽携带）|
| **百日咳 Pertussis** | **大环内酯（azithromycin 首选）** | **所有密接，不论疫苗史**（疫苗管未来、不管即时风险）|
| **Hib（b 型流感嗜血杆菌）** | **Rifampin** | **仅当家中有 <4 岁未全程接种 或 免疫缺陷儿童**（≠ 所有密接）|
| **白喉 Diphtheria** | **Erythromycin 或 Penicillin** + 加强疫苗 | + 培养随访 |
| **肺鼠疫 Pneumonic plague** | **Doxycycline / Ciprofloxacin** | 飞沫人传人 |
| **结核 TB（密接 LTBI）** | **INH（或 rifampin / 短程方案）** | 密接筛 TST/IGRA，阳性 + 无活动病 → 治潜伏（[[mistakes/uworld-mistakes_2026-06#^Q3246]]）|
| **流感 Influenza** | **Oseltamivir** | **高危密接 / 养老院等机构暴发** |
| **侵袭性 GAS（重症 STSS/坏死筋膜炎）** | Penicillin ±（选择性，有争议）| 仅严重侵袭性病例的高危家庭密接 |

> [!tip] 记忆框架
> **"飞沫/呼吸道人传人 + 高危 → 密接抗生素"**：脑膜炎球菌、百日咳、Hib、白喉、肺鼠疫；外加结核(治潜伏)、流感(抗病毒)。
> - **两个"不论疫苗史一律给"**：**百日咳 + 脑膜炎球菌**（所有密接）。
> - **一个"有条件才给"**：**Hib → 家里有 <4 岁易感儿/免疫缺陷才给 rifampin**。

## 二、暴露后免疫预防（Ig / 疫苗，非抗生素）

| 暴露 | 预防 |
|---|---|
| **HIV**（职业针刺/性暴露/性侵）| **3 药 × 28 天**（tenofovir-emtricitabine + dolutegravir 或 raltegravir）|
| **乙肝 HBV**（针刺/性/围产）| **HBIG + 疫苗** |
| **狂犬 Rabies**（咬伤）| **RIG + 疫苗**（伤口清洗）|
| **水痘 Varicella**（易感高危密接）| **VariZIG**（免疫抑制/孕妇/新生儿）或疫苗（健康者 3-5 天内）|
| **麻疹 Measles**（易感密接）| **MMR（72h 内）** 或 免疫球蛋白 |
| **甲肝 HepA**（密接）| **疫苗 ± 免疫球蛋白** |
| **破伤风**（伤口）| **Tdap ± TIG**（按接种史 + 伤口性质）|

## 三、高频陷阱

- **百日咳**：家人"疫苗齐全 + 无症状"**不豁免** PEP（[[mistakes/uworld-mistakes_2026-06#^Q3906]]）——反直觉点；疫苗不降即时风险，密接一律 azithromycin。
- **Hib ≠ 一律给**：与脑膜炎球菌/百日咳的"所有密接都给"不同，**Hib 只在家有 <4 岁易感儿/免疫缺陷时给 rifampin**。
- **脑膜炎球菌密接范围**：同住/日托/接吻/无防护暴露分泌物的医护（如插管）；**普通同事/同教室 casual contact 不给**。
- **脑膜炎球菌指示病例**：若用 **penicillin** 治（不清除鼻咽携带）→ 出院前也要补一剂预防；**ceftriaxone/cefotaxime 能清除携带**则不必。
- **结核**：密接不是"预防性给所有人"，而是**筛查 → LTBI 阳性者治潜伏**。

## 四、Memory Hook

```
密接给抗生素(化学预防)：
  一律给(不论疫苗) → 百日咳(azithro) + 脑膜炎球菌(rifampin/cipro/ceftriaxone)
  有条件给         → Hib(家有<4岁易感儿→rifampin)
  其他            → 白喉(erythro/PCN) / 肺鼠疫(doxy/cipro) / 流感(oseltamivir高危) / TB(LTBI→INH)

暴露后免疫预防(Ig/疫苗)：
  HIV(3药28天) / HBV(HBIG+疫苗) / 狂犬(RIG+疫苗) / 水痘(VariZIG) / 麻疹(MMR72h) / 甲肝(疫苗±Ig) / 破伤风(Tdap±TIG)
```

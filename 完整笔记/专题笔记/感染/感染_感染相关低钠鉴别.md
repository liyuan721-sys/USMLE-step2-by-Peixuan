# 感染相关低钠鉴别（哪些感染 → 低钠）

> [!info] 笔记定位
> 低钠（hyponatremia）是感染题的高频**暗钩**——尤其 **军团菌（Legionella）**和 **RMSF**。本笔记按**机制**（SIADH / 低容量 / 肾上腺功能不全）归类"哪些感染会低钠 + 怎么和其他线索一起秒杀"。
> 通用低钠诊断算法（血渗/尿渗/容量轴）见 [[完整笔记/专题笔记/肾脏/肾脏_低钠血症诊疗SOP]]；本笔记是其**感染切面**。

---

## 一、机制总览（4 条通路）⭐⭐

```
感染 → 低钠
├─ SIADH（ADH 不当分泌·等容量）⭐ 最常见
│    肺部感染（军团菌⭐/肺炎/TB/肺脓肿/脓胸）
│    中枢感染（脑膜炎/脑炎/脑脓肿）
├─ 低血容量性（容量丢失 → ADH 代偿↑）
│    胃肠炎/霍乱（腹泻呕吐）、RMSF（血管渗漏）、脓毒症第三间隙
├─ 肾上腺功能不全（皮质醇/醛固酮↓）→ 低钠 + 高钾
│    Waterhouse-Friderichsen（脑膜炎球菌→肾上腺出血）
│    TB 肾上腺炎（Addison）、CMV 肾上腺炎（AIDS）
└─ 假性/稀释（少见）：高血糖转移性低钠
```

---

## 二、高频感染 → 低钠对照表 ⭐⭐⭐

| 感染 | 机制 | 低钠之外的招牌（一起考） |
|---|---|---|
| **军团菌肺炎 Legionella** ⭐最经典 | **SIADH** | **低钠 + 腹泻 + 转氨酶↑ + 相对缓脉 + 水/空调暴露**；尿抗原 |
| **落基山斑疹热 RMSF** | 低容量→ADH↑（血管内皮渗漏）| **疹腕踝→掌跖向心 + 血小板↓ + 转氨酶↑**；doxy 别等 |
| **细菌性脑膜炎 / 脑炎** | **SIADH** 或 **脑性盐耗 CSW** | 头痛/颈强直/意识改变（SIADH 等容量 vs CSW 低容量）|
| **肺结核 TB** | **SIADH**（肺）/ **Addison**（肾上腺 TB → 低钠 + **高钾**）| 慢性咳/消瘦；肾上腺型有色素沉着 |
| **肺炎 / 肺脓肿 / 脓胸** | SIADH | 发热 + 咳脓痰 |
| **脑膜炎球菌血症** | **Waterhouse-Friderichsen → 肾上腺出血 → 急性肾上腺危象** | **暴发性紫癜 + 休克 + 低钠 + 高钾** |
| **胃肠炎 / 霍乱** | **低血容量性**（腹泻呕吐丢钠水）| 水样泻、脱水 |
| **AIDS（CMV 肾上腺炎等）** | 肾上腺功能不全 / SIADH | CD4 低、多病因叠加 |

---

## 三、两个最该秒杀的 ⭐

> [!danger] Legionella（军团菌）= "低钠的代名词"
> **非典型肺炎 + 低钠 + 腹泻 + 转氨酶↑ + 相对缓脉 + 水源/空调/旅馆暴露 = 军团菌**。USMLE 一见"**肺炎 + 低钠**"几乎条件反射 Legionella。机制 = **SIADH**。诊断：**尿抗原**。治疗：**大环内酯 / 呼吸喹诺酮**（β-内酰胺无效，胞内菌）。

> [!warning] 低钠 + 高钾 = 想肾上腺功能不全（别只想 SIADH）
> **SIADH = 低钠但血钾正常**；若**低钠 + 高钾**同现 → **肾上腺功能不全（醛固酮↓）**：
> - **脑膜炎球菌血症 → Waterhouse-Friderichsen**（肾上腺出血 + 暴发性紫癜 + 休克）
> - **TB / CMV 肾上腺炎 → Addison**
> 这是把"感染性低钠"从 SIADH 拨向肾上腺的关键钥匙。详见 [[完整笔记/专题笔记/内分泌/内分泌_肾上腺皮质功能不全]]。

---

## 四、SIADH vs 脑性盐耗 CSW（CNS 感染时易混）⭐

> [!tip] 都尿钠高，靠容量状态分 — 搞反会害人
> 
> | | **SIADH** | **脑性盐耗 CSW** |
> |---|---|---|
> | 容量 | **等容量 / 轻高** | **低容量（脱水）** |
> | 尿钠 | 高 | 高 |
> | 处理 | **限水** | **补盐 + 补容量** |
> | 鉴别钥匙 | 容量状态正常 | **临床脱水 + 体重↓** |
> 
> CNS 感染/出血两者都可能——**SIADH 限水、CSW 反而要补**，分错方向会加重病情。

---

## 五、Memory Hook

```
感染低钠先想机制：
  肺炎/CNS感染 + 等容量 → SIADH（军团菌是头牌！）
  腹泻/呕吐/RMSF渗漏 → 低血容量性
  低钠+高钾 → 肾上腺(脑膜炎球菌W-F / TB·CMV Addison)
军团菌五联：低钠+腹泻+转氨酶↑+相对缓脉+水暴露
"肺炎+低钠" → 条件反射 Legionella
"低钠+高钾+紫癜休克" → Waterhouse-Friderichsen
```

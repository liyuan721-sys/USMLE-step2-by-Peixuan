# RIPE 抗结核四联 — 副作用一页对照

> [!info] 笔记定位
> 活动性结核 (active TB, tuberculosis) 一线 **RIPE** 四联的**副作用 / 机制 / 监测 / 对策**一页速查。USMLE Step 2 高频考"加什么 / 查什么 / 停不停"。
> RIPE = **R**ifampin（利福平, RIF）+ **I**soniazid（异烟肼, INH）+ **P**yrazinamide（吡嗪酰胺, PZA）+ **E**thambutol（乙胺丁醇, EMB）。
> 疗程：强化期 2 月四联 → 巩固期 4 月（INH+RIF），共 **6 月**。鉴别 / 诊断路径见 [[完整笔记/专题笔记/感染/感染_TB与NTM肺病鉴别路径]]。

---

## 一、四药招牌副作用主表 ⭐⭐

| 药 | 招牌副作用 | 机制 | 监测 / 对策 |
|---|---|---|---|
| **R — Rifampin（利福平）** | ① **体液橙红色**（尿/泪/汗/唾液，**无害**，染隐形眼镜）② **强 CYP450 诱导剂** ③ 肝毒（胆汁淤积型）④ 流感样综合征 / 血小板减少（间断给药）| 诱导肝药酶 → 加速代谢 | 提前告知橙色**不用停**；查药物相互作用 |
| **I — Isoniazid（异烟肼）** | ① **周围神经病** (peripheral neuropathy) ② 肝炎 ③ **药物性狼疮** (drug-induced lupus, DILE) ④ 过量 → 难治性癫痫 | 耗竭维生素 B6（吡哆醇 pyridoxine）；慢乙酰化者风险↑ | **常规加维生素 B6 (pyridoxine 吡哆醇)**；过量用大剂量 B6 解 |
| **P — Pyrazinamide（吡嗪酰胺）** | ① **高尿酸血症 / 痛风** (hyperuricemia/gout) ② **肝毒性最强** ③ 关节痛 | 抑制肾小管尿酸排泄 | 监测尿酸 + 肝功 (LFT) |
| **E — Ethambutol（乙胺丁醇）** | **视神经炎** (optic neuritis) → **红绿色盲 / 视力↓**（剂量相关，早停可逆）| 视神经毒性 | **基线 + 定期查色觉 / 视力** |

> [!danger] 共性：肝毒性
> 四药中 **INH / RIF / PZA 肝毒性最相关**（EMB 几乎不肝毒）。**所有人起始查基线 LFT（肝功）**；出现症状性肝炎或转氨酶 >3×ULN（正常上限 upper limit of normal）伴症状 / >5×ULN 无症状 → 停药。

## 二、按字母锁定（一句话钩子）

```
R — Rifampin   → Red/oRange 橙红体液（无害）+ CYP450 诱导（降华法林/OCP/PI）
I — Isoniazid  → Injury to nerves 神经病（加 B6）+ 肝 + 狼疮
P — Pyrazinamide → uric acid ↑ 尿酸/痛风 + 肝毒最强
E — Ethambutol  → Eyes 眼（视神经炎 → 红绿盲，查色觉）
```

> [!tip] 三个最爱考的"配对反射"
> - **INH → 加 B6（吡哆醇 pyridoxine）防周围神经病**（高危：孕妇 / 糖尿病 / 酗酒 / 营养不良 / HIV / 尿毒症）
> - **EMB → 用前查基线色觉 + 视力**（Ethambutol = Eyes）
> - **RIF 橙色体液 = 无害，照常用药、不停药**（题干给"尿变橙吓到了"= 安抚 + 继续）

## 三、特殊人群 / 高频陷阱

> [!warning] 妊娠 (pregnancy)
> - **INH（+B6）/ RIF / EMB 孕期可用**，是孕期活动性 TB 主力。
> - **链霉素 streptomycin（氨基糖苷）孕期禁用**（胎儿耳毒性 → 先天性耳聋）。
> - PZA 在美国孕期常**谨慎/回避**（安全数据有限），WHO 则常规使用——题干若强调"美国孕妇"倾向不含 PZA。

> [!warning] 药物相互作用 — RIF 是主角
> Rifampin 强诱导 CYP450 → **降低**以下浓度：**华法林 warfarin（INR 国际标准化比值↓需加量）、口服避孕药 OCP (oral contraceptive pill)（避孕失败 → 改非激素法）、蛋白酶抑制剂 PI / 美沙酮 methadone / 糖皮质激素 / 部分降糖药**。题干给"TB 治疗中口服避孕药失败 / INR 掉" → 想 RIF。

> [!tip] "出现副作用 → 停哪个药"快速决策
> | 新发症状 | 元凶 | 处理 |
> |---|---|---|
> | 视物模糊 / 分不清红绿 | **EMB** | 立即停 EMB（早停可逆）|
> | 足袜套样麻木刺痛 | **INH** | 加 B6（多不必停 INH）|
> | 急性单关节红肿热痛 + 尿酸↑ | **PZA** | 评估，必要时停 PZA |
> | 尿/泪橙红 | **RIF** | 安抚，**不停药** |
> | 转氨酶↑↑ + 症状 | INH/PZA/RIF | 停可疑肝毒药，查 LFT |

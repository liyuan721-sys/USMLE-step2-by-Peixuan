> [!info] 总原则
> 感染题不是先背所有病原，而是先问：**病人稳不稳 → 感染部位在哪里 → 社区/医院/免疫抑制吗 → 是否需要先取培养 → 经验覆盖到哪里 → 何时降阶梯。**

```
感染题主轴
unstable / sepsis?
  ├─ yes → cultures if quick + broad empiric antibiotics + fluids/source control
  └─ no  → localize syndrome → targeted workup → empiric or await cultures

部位决定菌谱
  lung       → S. pneumoniae + atypicals
  CNS        → S. pneumoniae / N. meningitidis ± Listeria
  urine      → E. coli and other G− rods
  abdomen    → G− rods + anaerobes
  skin       → Strep / Staph ± MRSA
  valve      → Staph / Strep / Enterococcus by risk factor
```

## 一、急症优先级

> [!danger] Sepsis / septic shock
> 发热或感染可疑 + hypotension / altered mental status / hypoperfusion / lactate elevation → **1 小时内 broad-spectrum antibiotics**。可以先抽血培养，但不能为了结果延迟抗生素。

常见经验覆盖：

| 场景 | 经验方案 |
|---|---|
| 未明来源 sepsis | Vancomycin + piperacillin-tazobactam / cefepime |
| Neutropenic fever | Cefepime 单药；低血压/导管/皮肤感染/MRSA 风险才加 vancomycin |
| HAP/VAP | Cefepime or pip-tazo + vancomycin |
| Intra-abdominal sepsis | Pip-tazo；或 ceftriaxone + metronidazole |
| Meningitis unstable | Blood culture → immediate antibiotics + dexamethasone |

> [!warning] Source control
> 抗生素不是全部。脓肿 abscess、坏死性筋膜炎 necrotizing fasciitis、感染性假体、胆管炎、腹腔污染、IE 瓣周脓肿，都要考虑 drainage / debridement / surgery。

## 二、抗生素考场核心

| 场景 | 首选/核心 |
|---|---|
| Surgical prophylaxis | Cefazolin，切皮前 30-60 min |
| Nonpurulent cellulitis | Cephalexin / dicloxacillin |
| Purulent SSTI / MRSA risk | TMP-SMX / doxycycline；重症 IV vancomycin |
| Inpatient CAP non-ICU | Ceftriaxone + azithromycin，或 respiratory fluoroquinolone |
| HAP/VAP | Cefepime or pip-tazo + vancomycin |
| Bacterial meningitis age 2 mo-50 yr | Vancomycin + ceftriaxone |
| Meningitis >50/pregnant/alcohol use | Vancomycin + ceftriaxone + ampicillin |
| Immunocompromised meningitis | Vancomycin + ampicillin + cefepime |
| Neonatal meningitis | Ampicillin + cefotaxime; avoid ceftriaxone |
| Gonorrhea | Ceftriaxone IM; add doxycycline only if Chlamydia not excluded |
| Chlamydia | Doxycycline; pregnancy uses azithromycin |
| PID outpatient | Ceftriaxone + doxycycline + metronidazole |
| C. difficile | Fidaxomicin preferred or PO vancomycin; fulminant adds IV metronidazole |
| Latent TB | Rifampin 4 mo / INH-rifapentine 3 mo / INH 9 mo |
| Active TB | RIPE x2 mo → INH + rifampin x4 mo |

> [!danger] 药物禁忌与毒性反射
> - Neonate + ceftriaxone → kernicterus / calcium precipitation
> - Aminoglycoside + loop diuretic → ototoxicity
> - TMP-SMX + ACEi/ARB/spironolactone → hyperkalemia
> - Vancomycin rapid infusion → infusion reaction, not IgE allergy
> - Daptomycin for pneumonia → wrong; surfactant inactivates it
> - Linezolid + SSRI → serotonin syndrome
> - Fluoroquinolone → tendon rupture, QT prolongation, C. difficile, aortic risk

## 三、CNS 感染

> [!danger] Meningitis next step
> 成人或急重：blood cultures → immediate empiric antibiotics + dexamethasone → CT if indicated → LP。  
> 稳定儿童/婴儿且无 ICP risk：blood culture + LP sampling → immediate antibiotics。  
> 共同点：抗生素不等 Gram stain / culture / PCR 结果。

LP 前先 CT 指征：

- Altered mental status
- Focal neurologic deficit
- Papilledema
- New seizure
- Immunocompromised state

CSF pattern：

| 类型 | Glucose | Protein | Cells |
|---|---|---|---|
| Bacterial meningitis | Low | High | Neutrophils |
| Viral meningitis | Normal | Mild high | Lymphocytes |
| TB meningitis | Low | High | Lymphocytes/mononuclear |
| Cryptococcus | Low | High | Lymphocytes |
| HSV encephalitis | Often normal glucose | High | Lymphocytes + RBC possible |

> [!warning] Meningitis vs encephalitis
> **Mental status normal** + meningismus → meningitis。  
> Confusion / personality change / seizure / focal deficit → encephalitis，先想 HSV-1 temporal lobe，立即 IV acyclovir，不等 PCR。

## 四、肺部感染

CAP 判断：

| 场景 | 常见病原/处理 |
|---|---|
| Typical CAP | S. pneumoniae；lobar consolidation, productive cough |
| Atypical CAP | Mycoplasma, Chlamydophila, Legionella；dry cough/systemic symptoms |
| Legionella | Pneumonia + diarrhea + hyponatremia; azithromycin or levofloxacin |
| Aspiration pneumonia/abscess | Anaerobes; amp-sulbactam, amox-clav, or clindamycin in select settings |
| PCP | HIV CD4 <200, dyspnea, dry cough, bilateral ground-glass; TMP-SMX ± steroids |

> [!tip] CAP 用药分层
> Healthy outpatient：amoxicillin high dose OR doxycycline OR macrolide if resistance low。  
> Comorbid outpatient：amox-clav/cephalosporin + macrolide or doxy；或 respiratory fluoroquinolone。  
> Inpatient non-ICU：ceftriaxone + azithromycin；或 respiratory fluoroquinolone。

TB 高考点：

- Active TB：cough, fever, night sweats, weight loss, apical cavitation
- PPD/IGRA positive → next step CXR
- Active suspected → sputum AFB smear/culture/NAAT + airborne isolation
- Latent TB：positive test + normal CXR + asymptomatic

## 五、泌尿与腹腔感染

UTI：

| 场景 | 处理 |
|---|---|
| Uncomplicated cystitis | Nitrofurantoin / TMP-SMX / fosfomycin |
| Pyelonephritis outpatient | Fluoroquinolone or TMP-SMX if susceptible |
| Pyelonephritis inpatient | Ceftriaxone IV |
| Pregnancy asymptomatic bacteriuria | Treat: nitrofurantoin, beta-lactam options |
| Enterococcus UTI clue | Nitrite negative but culture positive; ampicillin or vancomycin depending susceptibility |

> [!warning] Nitrofurantoin
> 只适合下尿路 cystitis；不进肾实质，所以不能治 pyelonephritis。

腹腔感染：

- Appendicitis / diverticulitis / perforation / abscess → cover G− rods + anaerobes
- Ceftriaxone + metronidazole or pip-tazo
- Abscess usually needs drainage
- SBP：cirrhosis + ascites + PMN ≥250 → cefotaxime/ceftriaxone

## 六、皮肤软组织与骨关节

SSTI 分层：

| 场景 | 想法 |
|---|---|
| Nonpurulent cellulitis | Strep > MSSA; cephalexin |
| Purulent abscess | S. aureus/MRSA; incision & drainage ± MRSA coverage |
| Necrotizing fasciitis | Severe pain out of proportion, bullae, crepitus, toxicity; surgery now |
| Diabetic foot severe infection | Broad coverage + source control; consider osteomyelitis |
| Osteomyelitis | MRI best imaging; bone biopsy culture before long antibiotics if stable |

> [!danger] Necrotizing fasciitis
> 不要等 imaging。临床高度怀疑 → emergent surgical debridement + broad antibiotics。

## 七、感染性心内膜炎

IE 三个入口：

- Fever + new murmur
- IVDU + fever + pulmonary septic emboli
- Prosthetic valve + fever

第一步：

> [!danger] Suspected IE
> 先抽 **3 sets blood cultures**，再抗生素。Echo 一般 TTE 起步；人工瓣、TTE 阴但高度怀疑、疑瓣周脓肿时 TEE。

病原匹配：

| 场景 | 病原 |
|---|---|
| IVDU | S. aureus, often right-sided tricuspid |
| Dental / damaged valve | Viridans streptococci |
| Early prosthetic valve | S. epidermidis |
| Colon cancer association | S. gallolyticus, Clostridium septicum |
| GU procedure / nitrite negative UTI | Enterococcus |
| Culture negative | HACEK, Coxiella, Bartonella, Brucella |

IE 手术指征：

- Heart failure from valve dysfunction
- Perivalvular abscess / new AV block
- Persistent bacteremia despite antibiotics
- Fungal or resistant organism
- Large vegetation with recurrent emboli

IE prophylaxis：

- High-risk heart condition + dental gingival/apical manipulation
- Amoxicillin first line
- PCN allergy: azithromycin/clarithromycin/doxycycline; cephalexin if non-severe allergy
- No routine prophylaxis for GI/GU procedures without active infection

## 八、HIV/AIDS 机会感染

CD4 阈值：

| CD4 | 感染 |
|---|---|
| <200 | PCP |
| <100 | Toxoplasma, Cryptococcus |
| <50 | MAC, CMV |

Classic patterns：

- PCP：dry cough, hypoxemia, diffuse bilateral infiltrates → TMP-SMX; add steroids if PaO2 <70 or A-a gradient high
- Toxoplasmosis：ring-enhancing brain lesions → pyrimethamine + sulfadiazine
- Cryptococcus：subacute meningitis + high opening pressure → amphotericin B + flucytosine, serial LP
- CMV retinitis：floaters, retinal hemorrhage → ganciclovir/valganciclovir
- MAC：fever, night sweats, weight loss, anemia, diarrhea → azithromycin + ethambutol

> [!tip] ART timing
> Most OIs: start ART after acute stabilization. Cryptococcal meningitis and TB meningitis often require delayed ART due to IRIS risk.

## 九、STI 与生殖器综合征

Urethritis/cervicitis：

| Clue | Diagnosis | Treatment |
|---|---|---|
| Purulent discharge, intracellular G− diplococci | Gonorrhea | Ceftriaxone |
| Mucopurulent discharge, NAAT positive | Chlamydia | Doxycycline; pregnancy azithromycin |
| PID | Polymicrobial GC/CT/anaerobes | Ceftriaxone + doxy + metronidazole |

Genital ulcer：

| Clue | Diagnosis |
|---|---|
| Painful grouped vesicles/ulcers | HSV |
| Painless chancre | Primary syphilis |
| Painful soft chancre + tender nodes | Chancroid |
| Painless ulcer then painful buboes | LGV |

Syphilis：

- Diagnosis: nontreponemal RPR/VDRL for screening/activity; treponemal test confirms
- Treatment: penicillin G
- Pregnant + PCN allergy → desensitization then penicillin

## 十、腹泻与食源性感染

先分 watery vs inflammatory：

| Pattern | Think |
|---|---|
| Watery, no blood | Viral, ETEC, cholera, Giardia |
| Bloody/inflammatory | Shigella, Campylobacter, Salmonella, EHEC, Entamoeba |
| Antibiotic exposure | C. difficile |
| Rice-water stool | Vibrio cholerae |
| Undercooked poultry | Campylobacter |
| Reptiles/eggs/poultry | Salmonella |
| Undercooked beef + HUS | EHEC; avoid antibiotics/antimotility |

> [!warning] HUS clue
> Bloody diarrhea followed by anemia, thrombocytopenia, AKI → EHEC. Supportive care; avoid antibiotics.

Typhoid fever：

- Stepwise fever, relative bradycardia, abdominal pain, rose spots
- Constipation early, pea-soup diarrhea later
- Complication: intestinal hemorrhage/perforation around week 3-4
- Treatment: ceftriaxone or azithromycin depending resistance context

## 十一、蜱传与人畜共患

| Clue | Diagnosis | Treatment |
|---|---|---|
| Erythema migrans, facial palsy, AV block, arthritis | Lyme | Doxy; pregnancy/early localized amoxicillin; neuro/carditis ceftriaxone |
| Fever + headache + rash wrists/ankles to trunk | RMSF | Doxycycline, including children |
| Hemolysis + asplenia + Maltese cross | Babesiosis | Atovaquone + azithromycin |
| Leukopenia/thrombocytopenia, transaminitis | Ehrlichiosis/Anaplasmosis | Doxycycline |
| Rabbit/tick + ulceroglandular disease | Tularemia | Gentamicin |
| Cat scratch + regional LAD | Bartonella henselae | Often self-limited; azithromycin can shorten |

> [!danger] Tick-borne fever
> If RMSF suspected, give doxycycline immediately. Do not wait for serology.

## 十二、发热出疹与单核样综合征

| Clue | Diagnosis |
|---|---|
| Fever + cough/coryza/conjunctivitis + Koplik + cephalocaudal rash | Measles |
| Low fever + posterior auricular LAD + rash | Rubella |
| Slapped cheek | Parvovirus B19 |
| Diffuse sandpaper rash + strawberry tongue | Scarlet fever |
| Fever then rash after defervescence | Roseola |
| Vesicles in different stages | Varicella |
| Exudative pharyngitis + posterior cervical LAD + splenomegaly | EBV |
| Mononucleosis-like but heterophile negative | CMV, acute HIV, toxoplasmosis |

> [!warning] Ampicillin rash
> EBV infectious mononucleosis + ampicillin/amoxicillin → rash; not true penicillin allergy.

## 十三、疫苗与暴露后预防

High-yield PEP：

| Exposure | PEP |
|---|---|
| Meningococcal close contact | Rifampin / ciprofloxacin / ceftriaxone |
| Pertussis close contact | Azithromycin regardless vaccination status |
| Rabies high-risk exposure | Wound cleaning + vaccine + rabies immunoglobulin if not previously vaccinated |
| Tetanus-prone wound | Tdap/Td ± TIG depending immunization history |
| HIV occupational exposure | Start PEP ASAP, ideally within hours |
| HBV exposure | Vaccine ± HBIG depending vaccination/immune status |

Asplenia vaccines:

- Pneumococcal
- Hib
- Meningococcal ACWY and B
- Annual influenza

## 十四、复习题

> [!question]- 发热 + 低血压 + suspected infection，下一步？
> 先抽培养如果不延误，然后 1 小时内 broad-spectrum antibiotics + fluids；不要等培养结果。

> [!question]- 成人疑细菌性脑膜炎，意识改变，需要 CT。抗生素在 CT 前还是后？
> 血培养后立刻给 empiric antibiotics + dexamethasone；CT/LP 不能延迟抗生素。

> [!question]- 稳定 8 月龄婴儿，疑脑膜炎，无局灶神经征，前囟开放，下一步？
> Blood culture + LP sampling，然后立即 empiric antibiotics。重点是可以等采样，不等结果。

> [!question]- IVDU 发热 + 三尖瓣赘生物，经验治疗？
> Vancomycin 覆盖 MRSA；若重症/医院菌风险可加 cefepime 覆 Pseudomonas。

> [!question]- Gonorrhea 治疗是不是永远 ceftriaxone + doxy？
> 不是。Gonorrhea 本身 ceftriaxone IM；Chlamydia 未排除才加 doxycycline。

> [!question]- HIV CD4 80，头痛高颅压，CSF India ink/CrAg positive？
> Cryptococcal meningitis：amphotericin B + flucytosine，serial LP 降颅压，ART 延后以降低 IRIS 风险。

> [!question]- Bloody diarrhea 后 Hgb 下降、platelet 下降、Cr 上升，最可能？抗生素吗？
> EHEC-associated HUS；supportive care，避免 antibiotics 和 antimotility。

> [!question]- Lyme pregnant patient with erythema migrans 怎么治？
> Amoxicillin。Doxycycline 避免；ceftriaxone 留给 neuro/carditis 等严重病。

## 十五、考前 10 分钟默写清单

- Sepsis 1-hour antibiotics
- Meningitis: blood culture → antibiotics timing; CT before LP indications
- CSF 四型：bacterial / viral / TB / cryptococcus
- CAP outpatient vs inpatient vs HAP/VAP
- UTI cystitis vs pyelonephritis: nitrofurantoin 不能治肾盂肾炎
- IE: blood cultures first; TTE vs TEE; surgery indications
- HIV CD4: 200 / 100 / 50
- STI: gonorrhea vs chlamydia; genital ulcer differential
- TB: latent vs active
- Tick-borne: RMSF and Lyme treatment
- C. diff and EHEC treatment traps

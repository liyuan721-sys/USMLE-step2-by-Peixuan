# 感染_抗假单胞药一览（覆盖 vs 不覆盖铜绿 + 场景 + 陷阱）

> [!danger] 为什么建这张（反复错触发）
> NOE（坏死性外耳道炎）= 铜绿，**两次错都绕铜绿**：[[mistakes/uworld-mistakes_2026-06#^Q2828]]（错病原选曲霉）+ [[mistakes/uworld-mistakes_2026-06#^Q4282]]（错治疗选**氨苄西林-舒巴坦，不覆铜绿**）。把"哪些药抗铜绿、哪些是坑"一次焊死。

---

## 一、覆盖 vs 不覆盖铜绿 ⭐⭐⭐（核心）

> [!success] ✅ 抗假单胞（覆盖铜绿）
> **β-内酰胺类**
> - 青霉素：**哌拉西林-他唑巴坦 piperacillin-tazobactam**（替卡西林 ticarcillin）
> - 头孢：**头孢他啶 ceftazidime（3 代里唯一）**、**头孢吡肟 cefepime（4 代）**（头孢洛扎-他唑巴坦 ceftolozane-tazobactam）
> - 碳青霉烯：**美罗培南 meropenem、亚胺培南 imipenem、多尼培南 doripenem**（⚠️ **厄他培南 ertapenem 除外**）
> - 单环：**氨曲南 aztreonam**（**β-内酰胺严重过敏唯一可用的 β-内酰胺**——单环结构与青霉素 / 头孢 / **碳青霉烯 carbapenem 均无交叉**；⚠️ 其余 β-内酰胺都共享 β-内酰胺环、**碳青霉烯交叉率低但真实 → 重度过敏一并避开**。仅覆 G⁻/铜绿、**无 G⁺ 无厌氧**；唯一小坑：与头孢他啶 ceftazidime 共享侧链可交叉）
>
> **氟喹诺酮**
> - **环丙沙星 ciprofloxacin（抗铜绿最强）**、**左氧氟沙星 levofloxacin**（⚠️ **莫西沙星 moxifloxacin 除外**）
>
> **氨基糖苷**（多作联合，少单用）
> - **庆大霉素 gentamicin、妥布霉素 tobramycin、阿米卡星 amikacin**
>
> **多黏菌素**：黏菌素 colistin（多重耐药 MDR 补救）

> [!warning] ❌ 不抗铜绿（高频陷阱）
> - **氨苄西林-舒巴坦 ampicillin-sulbactam**（[[mistakes/uworld-mistakes_2026-06#^Q4282]] 错点）、阿莫西林-克拉维酸 amoxicillin-clavulanate
> - **头孢曲松 ceftriaxone / 头孢噻肟 cefotaxime**（3 代但只有 ceftazidime 抗铜绿）、头孢唑林 cefazolin（1 代）、头孢呋辛 cefuroxime（2 代）
> - **厄他培南 ertapenem**（碳青霉烯里**唯一**不抗铜绿）
> - **莫西沙星 moxifloxacin**（喹诺酮里的例外）
> - 绝大多数口服药、复方新诺明 TMP-SMX

---

## 二、各类里的"例外"——最爱考的三不抗 ⭐⭐

| 类别 | 大多抗铜绿 | **例外（不抗）** |
|---|---|---|
| 碳青霉烯 | 美罗/亚胺/多尼培南 | **厄他培南 ertapenem** |
| 抗铜绿喹诺酮 | 环丙 / 左氧氟 | **莫西沙星 moxifloxacin** |
| 3 代头孢 | **仅** 头孢他啶 ceftazidime | 头孢曲松 / 头孢噻肟 |

> 记："**厄他、莫西、头孢曲松**——三个看着广谱，偏偏漏铜绿。"

---

## 三、何时需要抗铜绿（高危场景）⭐⭐

```
想到铜绿 = 想到"湿 + 院内/免疫低 + 器械/特殊暴露"：
  中性粒细胞减少发热 neutropenic fever
  HAP / VAP（医院 / 呼吸机肺炎）
  囊性纤维化 CF 肺恶化
  烧伤 burns 创面
  坏死性外耳道炎 NOE（老年糖尿病）
  留置导管 / 器械（CLABSI）、静脉吸毒 IE
  穿运动鞋扎伤足 → 假单胞骨髓炎；隐形眼镜 → 假单胞角膜炎
  泡热水浴后毛囊炎 hot-tub folliculitis；绿甲 / 绿脓
```

## 四、经验方案速记 ⭐⭐

| 场景 | 经验首选 | 备注 |
|---|---|---|
| **中性粒缺发热** | **头孢吡肟 cefepime 单药** | 或 哌拉西林他唑巴坦 piperacillin-tazobactam / 美罗培南 meropenem；**不常规加万古霉素 vancomycin——除非：休克/低血压 · 皮肤软组织感染 · 留置导管感染 · 已知 MRSA 定植 · 严重黏膜炎** |
| **HAP / VAP** | **头孢吡肟 cefepime + 万古霉素 vancomycin**（双覆铜绿 + MRSA） | 或 哌拉西林他唑巴坦 piperacillin-tazobactam + 万古霉素 vancomycin；**别用 头孢曲松 ceftriaxone**（漏铜绿 + MRSA） |
| **坏死性外耳道炎 NOE** | **IV 环丙沙星 ciprofloxacin** 6–8 周 | 见 [[完整笔记/专题笔记/感染/感染_外耳道炎谱鉴别]] |
| **重症 / MDR 风险双覆盖** | β-内酰胺 β-lactam（抗铜绿）+ 氟喹诺酮 fluoroquinolone **或** 氨基糖苷 aminoglycoside | 多数非重症单药 β-内酰胺即可 |
| **CF 急性恶化** | 抗铜绿 β-内酰胺 β-lactam + 妥布霉素 tobramycin | 见 [[完整笔记/专题笔记/儿科/儿科_囊性纤维化CF气道病原速查]] |

## 🎣 Memory Hook

> [!success] 一句话锁定
> **抗铜绿 = "哌拉西林他唑 + 头孢他啶/头孢吡肟 + 美罗培南 + 氨曲南 + 环丙/左氧氟 + 氨基糖苷"。**
> **三大坑（看着广谱却漏铜绿）：氨苄西林-舒巴坦、头孢曲松、厄他培南**（再加喹诺酮里的莫西沙星）。

# 头孢菌素分代速查（Cephalosporins by Generation）

> [!info] 笔记定位
> **覆盖谱 + 临床场景 + 选药对照 + 易混陷阱**一站速查。
> USMLE 头孢题 90% 考的是"哪代覆盖什么 / 哪个场景选哪代"，**机制/化学结构基本不考**。

---

## 一、5 代核心对比表 ⭐⭐⭐

| 代 | 代表药 | G⁺（含 MSSA/Strep） | G⁻（含肠杆菌科） | Pseudomonas | MRSA | 厌氧菌 | CNS 穿透 |
|---|---|---|---|---|---|---|---|
| **1 代** | **Cefazolin** (IV) / **Cephalexin** (PO) | **强** ⭐ | 弱（**PEcK**：Proteus / E. coli / Klebsiella） | ❌ | ❌ | ❌ | ❌ |
| **2 代** | **Cefuroxime** / **Cefoxitin** / **Cefotetan** | 中 | 中（**HEN-PEcK**：+ H. influenzae / Enterobacter / Neisseria） | ❌ | ❌ | **Cefoxitin / Cefotetan ✅** | ❌（cefuroxime 弱穿透）|
| **3 代** | **Ceftriaxone** / **Cefotaxime** / **Ceftazidime** | 弱 → 中 | **强** ⭐ | **仅 Ceftazidime ✅** | ❌ | ❌ | **✅**（首选脑膜炎）|
| **4 代** | **Cefepime** | **强**（捡回 G⁺） | **强** | ✅ | ❌ | ❌ | ✅ |
| **5 代** | **Ceftaroline** | **强 + MRSA** ⭐ | 中 | ❌ | **✅ 唯一** | ❌ | 中 |

> [!tip] 一句话总结演化逻辑
> **1 代 G⁺ 强 G⁻ 弱 → 2 代各加一点（少数加厌氧）→ 3 代 G⁻ 王 + 进 CNS（Ceftaz 单挑 Pseudo）→ 4 代 G⁺ 回归 + Pseudo 全包 → 5 代终于打 MRSA**

> [!warning] 所有代头孢 **不覆盖** 的三大类（必背）
> - ❌ **Enterococcus**（5 代也不行 — 用 ampicillin / vanc）
> - ❌ **Listeria**（用 ampicillin）
> - ❌ **Atypicals**（Mycoplasma / Chlamydia / Legionella — 用 macrolide / FQ / doxy）
>
> 助记 **"LAME"**：**L**isteria / **A**typicals / **M**RSA(仅 5 代例外) / **E**nterococci → 头孢盲区

> [!danger] 盲区 E = Enterococcus 治疗速记（高频陷阱）⭐ #难记
> **肠球菌虽是 G⁺ 球菌，但所有头孢都不覆盖它**（天然耐药）——这是 USMLE 高频陷阱（看到"GU 操作后 + GPC 血培养 + nitrite⊖ UTI"别反射头孢）。
>
> | 情境 | 方案 |
> |---|---|
> | **一般感染** | **Ampicillin**（或 vanc）± 氨基糖苷协同 |
> | **IE（心内膜炎）** | **Ampicillin + Gentamicin** 或 **Ampicillin + Ceftriaxone**（双 β-lactam 协同，替代氨基糖苷） |
> | **VRE（耐万古）** | **Linezolid / Daptomycin** |
>
> ⚠️ **ampicillin + ceftriaxone 里的 ceftriaxone 是"协同"角色，不是"头孢能覆盖肠球菌"**——别因此推翻"头孢不覆盖肠球菌"的铁律。
> 📍 肠球菌完整一页纸（身份/nitrite 机制/头孢耐药/治疗）见 [[完整笔记/专题笔记/感染/感染_肠球菌Enterococcus]]（[[完整笔记/专题笔记/感染/感染_感染性心内膜炎完整体系]] §6.2 为 IE 语境）。

## 二、各代细看

### 2.1 第 1 代 — G⁺ 之王 + 外科预防

| 药 | 给药 | 主要场景 |
|---|---|---|
| **Cefazolin** | IV | **外科预防首选**（手术前 1h，剂量按 BMI 调）/ MSSA 菌血症 / 心内膜炎 NV |
| **Cephalexin** | PO | 皮肤软组织感染（蜂窝织炎，门诊）/ 单纯 UTI / 链球菌咽炎备选 |

**覆盖速记**：**MSSA / Strep / PEcK G⁻**（Proteus mirabilis / E. coli / Klebsiella 社区菌株）

**临床钥匙**：
- 看到"手术前 30-60 min 给抗生素" → **Cefazolin**（无 MRSA 风险）/ Vancomycin（已知 MRSA）
- "Skin abscess + cellulitis 门诊治疗" → **Cephalexin**（无 MRSA 怀疑时）
- **PCN 过敏 cross-reactivity ~1%**（不是早期教的 10%）→ 大部分 PCN 过敏可安全用头孢

### 2.2 第 2 代 — 中间过渡 + 子分组

> 第 2 代是**唯一有内部分化**的代 — 分两个子家族：

| 子家族 | 药 | 特点 |
|---|---|---|
| **True cephalosporins** | **Cefuroxime** | 仍走 G⁺/G⁻ 平衡谱，**无厌氧** |
| **Cephamycins**（结构略不同） | **Cefoxitin** / **Cefotetan** | **覆盖厌氧** ⭐（B. fragilis）→ PID / 腹腔混合感染 |

**覆盖速记**：**HEN-PEcK**（H. influenzae / Enterobacter / Neisseria + 1 代的 PEcK）

**临床钥匙**：
- **PID（Pelvic Inflammatory Disease）住院** → **Cefoxitin / Cefotetan + Doxycycline**（cefoxitin/cefotetan 覆盖厌氧；doxy 覆盖 Chlamydia/Mycoplasma）
- **Cefotetan 副作用**：双硫仑反应（避免酒精）+ 凝血障碍（含 NMTT 侧链 → ↓ 维 K → PT 延长 / 出血）
- Cefuroxime 口服剂型用于轻度社区肺炎 / 鼻窦炎 / 中耳炎

### 2.3 第 3 代 — G⁻ 王 + 进 CNS

| 药 | 关键身份 |
|---|---|
| **Ceftriaxone** ⭐⭐⭐ | "社区感染万金油"：**社区肺炎 / 脑膜炎 / 淋病 / Lyme / SBP / 急性肾盂肾炎住院 / Typhoid** 首选/重要选项 |
| **Cefotaxime** | Ceftriaxone 替代（**新生儿首选** — ceftriaxone 在新生儿 displace bilirubin → 核黄疸 + 与含钙 IV 沉淀致死）|
| **Ceftazidime** | **3 代里唯一覆盖 Pseudomonas** — 但 G⁺ 弱（很少单用） |

**覆盖速记**：**强 G⁻**（含 Neisseria / Hib / Enterobacteriaceae 社区株）+ **进入 CNS**

**临床钥匙**：
- **细菌性脑膜炎经验治疗**（年龄 >1 月 ~ 50 岁）：**Vancomycin + Ceftriaxone**（+ Ampicillin 若 >50 岁 / 免疫低 / 新生儿，覆盖 Listeria）
- **淋病（GC）** ：**Ceftriaxone 500 mg IM × 1** + **Doxy 100 mg PO BID × 7d**（covering co-infection Chlamydia，2021 起停用 azithro 联用）
- **Lyme disease 神经/心脏受累 / 严重关节炎**：**Ceftriaxone IV 14-28d**（早期局限 = Doxy PO）
- **SBP（自发性细菌性腹膜炎）**：**Cefotaxime / Ceftriaxone**（确诊后立即；预防用 ciprofloxacin / TMP-SMX）
- **新生儿 sepsis**：**Ampicillin + Cefotaxime / Gentamicin**（**绝不 ceftriaxone**）
- **SCD 骨髓炎**：**抗葡萄球菌(clinda/vanc) ＋ ceftriaxone**。Ceftriaxone 一个药同时覆 **肺炎链球菌 + Salmonella**，但**不可靠覆金葡(MRSA 不覆)** → 缺口在金葡，故加 clinda/vanc。对照 **cefazolin(1 代)覆金葡却漏 Salmonella** → 单一头孢都不能通吃。详见 [[完整笔记/专题笔记/血液/血液_镰刀细胞病SCD]] §2.2

### 2.4 第 4 代 — Cefepime 三全 + AmpC 杀手

> **Cefepime 是 4 代独苗**（其他 4 代药如 cefpirome 美国基本不用）

**覆盖速记**：**G⁺**（强，捡回 1 代水平）+ **G⁻**（强，含 ESBL/AmpC 难搞菌）+ **Pseudomonas** ✅ + **进入 CNS**

**为什么 4 代能搞定 AmpC？**
- 1-3 代 β-lactam 易被 **AmpC β-lactamase**（Enterobacter / Serratia / Citrobacter / Pseudomonas 产）水解
- 4 代结构上对 AmpC 稳定 → 这些 "**S**erratia / **P**seudomonas / **A**cinetobacter / **C**itrobacter / **E**nterobacter" 才用 cefepime

**临床钥匙**：
- **Neutropenic fever**（中性粒细胞绝对值 <500 + 发热 ≥38.3°C 一次 / ≥38°C 持续 1h）→ **Cefepime 单药** ⭐（或 piperacillin-tazobactam / meropenem）
- **HAP / VAP**（医院获得性 / 呼吸机相关肺炎）→ **Cefepime ± Vanc**（覆盖 MRSA 风险时加 vanc）
- **IVDU IE 经验治疗** → **Vancomycin + Cefepime**（MRSA + Pseudomonas）
- **AmpC G⁻ 感染**（Enterobacter / Serratia 培养出）→ **Cefepime**（不要降阶梯到 3 代 — AmpC 诱导耐药）
- **复杂院内 UTI / 腹腔感染（怀疑 Pseudomonas）** → Cefepime

**副作用警告 ⚠️**：**Cefepime 神经毒性**（脑病 / 肌阵挛 / non-convulsive status epilepticus）— **CKD 患者剂量未调** 是高危场景

### 2.5 第 5 代 — 终于打 MRSA 的头孢

| 药 | 关键身份 |
|---|---|
| **Ceftaroline** | **唯一覆盖 MRSA 的头孢** ⭐ — 也覆盖 VRSA / VISA / 顽固 G⁺ |

**覆盖速记**：**G⁺（含 MRSA / Strep / Enterococcus faecalis 部分株）+ 中 G⁻**（**无 Pseudomonas**）

**临床钥匙**：
- **MRSA 菌血症 / 复杂皮肤软组织感染**（vanc 失败 / 不能用 vanc）
- **MRSA 社区肺炎**
- ⚠️ **Enterococcus faecium 不覆盖**（只覆盖部分 faecalis）→ Enterococcus 仍首选 amp / vanc / linezolid / dapto
- ⚠️ **不覆盖 Pseudomonas** — 不能用于 HAP/VAP 经验治疗
- ⚠️ **不覆盖厌氧**

## 三、临床场景 → 选哪代速查（USMLE 高频）

| 场景 | 首选 | 代 | 备注 |
|---|---|---|---|
| **外科手术预防** | Cefazolin | 1 | MRSA 风险 → Vanc 替代 |
| **门诊蜂窝织炎 / 单纯 UTI** | Cephalexin | 1 | MRSA → TMP-SMX / doxy |
| **PID 住院** | Cefoxitin / Cefotetan + Doxy | 2 | 覆盖厌氧 + Chlamydia |
| **社区肺炎住院（非 ICU）** | Ceftriaxone + Azithro | 3 + macrolide | atypical 覆盖 |
| **细菌性脑膜炎经验**（1 月-50 岁） | **Vanc + Ceftriaxone** | 3 | >50 / 免疫低 / 新生儿 → 加 Amp 覆盖 Listeria |
| **新生儿 sepsis** | Amp + **Cefotaxime** / Gent | 3 | **禁 Ceftriaxone**（核黄疸 + Ca 沉淀） |
| **淋病** | Ceftriaxone 500 mg IM × 1 + Doxy 7d | 3 | 2021 后单药 ceftriaxone 高剂量 |
| **Lyme 神经/心脏/严重关节** | Ceftriaxone IV 14-28d | 3 | 早期局限 → Doxy PO |
| **SBP（自发性细菌性腹膜炎）** | Cefotaxime / Ceftriaxone | 3 | 预防：cipro / TMP-SMX |
| **Typhoid fever** | Ceftriaxone | 3 | 或 azithro / FQ |
| **SCD 骨髓炎** | **Ceftriaxone + clinda/vanc** | 3 + 抗葡药 | ceftriaxone 覆 Salmonella，加抗葡药补金葡(MRSA) |
| **急性肾盂肾炎住院** | Ceftriaxone | 3 | 门诊轻 → cipro / TMP-SMX |
| **Pseudomonas 感染（非中性粒）** | Ceftazidime / Cefepime / Pip-Tazo | 3/4 | 看复杂度 + G⁺ 共感染风险 |
| **HAP / VAP** | **Cefepime ± Vanc**（或 Pip-Tazo） | 4 | 覆盖 Pseudomonas + MRSA 风险 |
| **Neutropenic fever** | **Cefepime** 单药 | 4 | 或 Pip-Tazo / Meropenem |
| **IVDU IE 经验** | Vanc + Cefepime | 4 | MRSA + Pseudo |
| **MRSA 菌血症 / 复杂 SSTI**（vanc 替代） | **Ceftaroline** | 5 | 或 dapto / linezolid |
| **Enterococcus IE** | Amp + Gent / Amp + Ceftri | (3 协同) | 头孢单药无效 |

## 四、易混陷阱

> [!danger] 高频陷阱（USMLE 必出）
>
> **1. "所有头孢覆盖 enterococcus / Listeria"** → ❌ 错
> - **所有代头孢都不覆盖 enterococcus 和 Listeria**（5 代也不行）
> - 助记：**LAME**（Listeria / Atypicals / MRSA / Enterococci → 头孢盲区，5 代只破 M 那一项）
>
> **2. "Cefepime 覆盖 MRSA"** → ❌ 错
> - 只有 **Ceftaroline（5 代）** 覆盖 MRSA
> - Cefepime 强 G⁺ 是 MSSA / Strep，不是 MRSA
>
> **3. "Ceftazidime 是 4 代"** → ❌ 错
> - **Ceftazidime 是 3 代**，但**特殊地覆盖 Pseudomonas**（3 代里唯一）
> - 别把"覆盖 Pseudo"和"4 代"画等号
>
> **4. "Cephalosporins 不能用于 PCN 过敏者"** → ❌ 过度规避
> - 真实交叉反应率 **~1%**（不是早期教的 10%）
> - **严重 PCN 过敏**（anaphylaxis / SJS / TEN）→ 避免；**轻度皮疹** → 头孢可用（特别 2-5 代）
>
> **5. "新生儿用 Ceftriaxone"** → ❌ 致死陷阱
> - **绝对禁忌**：① displaces bilirubin → 核黄疸；② 与含 Ca IV 液沉淀致死
> - 新生儿用 **Cefotaxime** 替代
>
> **6. "AmpC 菌（Enterobacter / Serratia）用 Ceftriaxone"** → ❌ 错
> - 即使 in vitro 敏感，治疗中诱导 AmpC 产生 → 临床失败
> - 必须 **Cefepime** 或 meropenem
>
> **7. "Cefoxitin / Cefotetan 是 3 代"** → ❌ 错
> - **2 代 cephamycins**（结构上特殊，覆盖厌氧）
>
> **8. "Cefepime CKD 不用调剂量"** → ❌ 神经毒性陷阱
> - **CKD + cefepime 不调剂量** → 脑病 / 肌阵挛 / non-convulsive SE（USMLE 经常作为干扰诊断）
> - 见到老年 + CKD + 新发意识改变 + 在用头孢 → 想 cefepime 脑病
>
> **9. "Cefotetan 喝酒没事"** → ❌ 错
> - **NMTT 侧链** → 双硫仑反应 + ↓ 维 K → PT 延长 / 出血
> - 治疗期间 + 治疗后 72h 禁酒；出血风险高者监测 PT
>
> **10. "覆盖厌氧 = 3 代头孢"** → ❌ 错
> - 覆盖厌氧的是 **2 代 cephamycins**（cefoxitin / cefotetan）；其他头孢 + metronidazole 才覆盖

## 五、Memory Hook

### 5.1 主框架口诀

- **"1 代 G⁺ 强 G⁻ 弱；2 代各加一点（cefoxi/cefote 加厌氧）；3 代 G⁻ 王 + 进 CNS（Ceftaz 单挑 Pseudo）；4 代 G⁺ 回归 + Pseudo 全包；5 代终于打 MRSA"**

### 5.2 代表药记忆

- **Cef**a**z**olin = **z** = "外科预防 **z**ero risk"（最便宜 + 最常用预防）
- **Cef**ox**itin** / Cef**ote**tan = **2 代厌氧**（"**ox**ygen-avoiding bugs" 联想 Cefoxitin → 厌氧）
- **Ceft**ria**xone** = "3 代 social"（社区感染万金油：肺炎 / 脑膜炎 / 淋病 / Lyme / SBP / Typhoid）
- **Ceft**az**idime** = "3 代 **Az**ure = 远方 / 海外 Pseudomonas"（**Az** 像 Australia）
- **Cef**epime** = "4 代 = For **E**verything"（除了 MRSA + Enterococcus + 厌氧 + Listeria）
- **Cef**ta**R**oline = "5 代 ta**R**oline 打 M**R**SA"（双 R 锁定）

### 5.3 盲区口诀

- **LAME**（头孢盲区）：**L**isteria / **A**typicals / **M**RSA（5 代例外）/ **E**nterococci
- **PEcK**（1 代 G⁻）：**P**roteus / **E**. coli / **K**lebsiella
- **HEN-PEcK**（2 代 G⁻ = 1 代 + **H**aemophilus / **E**nterobacter / **N**eisseria）
- **SPACE 菌**（4 代专攻 AmpC）：**S**erratia / **P**seudomonas / **A**cinetobacter / **C**itrobacter / **E**nterobacter

### 5.4 致死陷阱钩子

- **新生儿 + Ceftriaxone = 核黄疸 / Ca 沉淀致死** → 新生儿永远用 **Cefotaxime**
- **CKD + 老人 + Cefepime 不调量 = 脑病** → 新发意识改变想到药源
- **Cefotetan + 酒 = 双硫仑 + 出血** → 治疗期 + 72h 禁酒

## 六、Penicillin 过敏的处理（简表）

| 过敏严重度 | 头孢可用性 |
|---|---|
| **轻度皮疹 / 不明确过敏史** | **可用**（特别 2-5 代，避开 1 代如 cefazolin / cephalexin 因结构最相似 PCN） |
| **严重过敏**（anaphylaxis / SJS / TEN / 血管性水肿 / 急性间质性肾炎） | **避免所有 β-lactam** → 换 vanc / FQ / clinda / aztreonam 等 |
| **真实交叉反应率** | **~1%**（不是早期教科书的 10%） |

> [!info] Aztreonam = β-lactam 但**无交叉过敏**（PCN/cephalosporin 严重过敏唯一可用 β-lactam）— 仅覆盖 G⁻

# 感染 High-Yield 速查总览（Infectious Disease — 决策导向）

> [!info] 用途
> 整科"**识别钥匙 buzzword → 经验治疗 empiric Tx → 确诊方法 Dx**"决策速查，派生自 [[完整笔记/Peixuan分科笔记/感染]]（分科主笔记，只读）的高频提炼。
> 感染各深挖主题已有大量独立专题笔记 → 本卡只放**决策骨架 + 钥匙**，深挖点 🧩 指向。
> 阈值 / 数字 / 经验用药均按分科笔记原文，未自行补充。
> 一句话整科记忆：**先血培养 + 经验抗生素覆盖到位，再按部位锁病原换窄谱；脓毒症 1 小时内广谱不等结果。**

## 一、脓毒症与按部位经验抗生素总框架（Sepsis & empiric backbone）

> [!danger] 脓毒症铁律（最高频"下一步"）
> 疑脓毒症 sepsis（发热 + 低血压 / 终末器官低灌注）→ **1 小时内** 给**经验广谱抗生素**（先抽血培养但不为等结果而延误）+ 液体复苏。糖尿病足 / 中性粒缺等高危 stem 出现"先查别的"干扰项时，永远选**先广谱抗生素**。
> 详见 [[mistakes/uworld-mistakes_2026-06#^Q19283]]（糖尿病足脓毒症 1h 广谱）。

缩写：CAP = community-acquired pneumonia 社区获得性肺炎；HAP = hospital-acquired pneumonia 医院获得性肺炎；VAP = ventilator-associated pneumonia 呼吸机相关性肺炎；MRSA = methicillin-resistant S. aureus 耐甲氧西林金葡。

```
整科经验治疗"由社区→医院"升级逻辑（肺炎为缩影）
社区 community ── 头孢曲松 ceftriaxone + 阿奇霉素 azithromycin（覆盖典型 + 非典型）
   ↓ 越往医院 / 呼吸机走，越要防"耐药医院菌"
医院 HAP ────── 抗假单胞 β-内酰胺（哌拉西林-他唑巴坦 pip-tazo / 头孢吡肟 cefepime）± MRSA 万古霉素
   ↓
呼吸机 VAP ──── 3 药（双抗假单胞 + MRSA 覆盖）
```

## 二、CNS 感染（脑膜炎 / 脑炎 — CSF 判读是核心）

> [!warning] 脑膜炎 meningitis 初始管理顺序（铁律）
> ① 血培养 → ② **经验抗生素**（勿等腰穿 LP, lumbar puncture）→ ③ 决定 LP vs 先 CT（computed tomography 计算机断层）。
> **LP 前先 CT 的指征**：神志改变 AMS (altered mental status) / 局灶缺损 / 癫痫 / 视乳头水肿 papilledema。
> "先腰穿才给药"是经典陷阱 → 见 [[mistakes/uworld-mistakes_2026-06#^Q18591]]（脑膜炎球菌：先经验抗生素勿等腰穿，反复错）。

按年龄经验治疗 + 病原（源文原文）：

| 人群 | 主要病原 | 经验治疗 |
|---|---|---|
| 新生儿 <1 周 | 大肠杆菌 E. coli / B 族链球菌 GBS / 李斯特菌 Listeria（尤危）| 氨苄西林 ampicillin + 庆大霉素 gentamicin |
| 青少年 | 脑膜炎奈瑟菌 N. meningitidis / 肺炎链球菌 S. pneumoniae / 埃可病毒 Echovirus | 头孢噻肟 cefotaxime + 万古霉素 vancomycin |
| 成人 / 老年 | 肺炎链球菌 S. pneumoniae / 李斯特菌 Listeria | 基础方案 + 加覆盖 Listeria |
| 社区获得性首位 | **S. pneumoniae 肺炎链球菌**（±肺炎）| —— |

CSF（cerebrospinal fluid 脑脊液）鉴别（源文原文）：

| 类型 | 葡萄糖 | 蛋白 | 细胞 |
|---|---|---|---|
| 病毒 viral | 正常 | 正常 / 轻高 | 淋巴 |
| 细菌 bacterial | **低** | 高 | 中性粒 |
| 真菌 fungal（Cryptococcus）| 低 | 高 | 淋巴 |
| 结核 TB | 低 | 高 | 单核 |
| 疱疹 herpes（脑炎重叠）| —— | —— | **CSF 见 RBC** |

> [!tip] CNS 经验治疗 + 确诊钥匙速记
> - **脑膜炎球菌血症 + 紫癜** → 警惕 DIC（disseminated intravascular coagulation 弥散性血管内凝血），查纤维蛋白原 fibrinogen（[[mistakes/uworld-mistakes_2026-06#^Q18596]]）。
> - **隐球菌脑膜炎 cryptococcal meningitis**（HIV + 鹅口疮 + 亚急性↑颅压头痛）→ Dx = **CSF 隐球菌抗原 / 印度墨汁**，Tx = 两性霉素 B amphotericin B（[[mistakes/uworld-mistakes_2026-06#^Q9885]]）。
> - **弓形虫脑炎 toxoplasma encephalitis**（HIV + 环形强化灶 ring-enhancing）→ 乙胺嘧啶 pyrimethamine + 磺胺嘧啶 sulfadiazine。
> - **PML（JC 病毒）**：晚期 AIDS + 无占位效应白质灶，**无强化**（[[mistakes/uworld-mistakes_2026-06#^Q2277]]）。
> - 疑感染性鼻窦炎 + 晨起呕吐 / 嗜睡 → 颅内并发症（脑脓肿）先头颅 CT（[[mistakes/uworld-mistakes_2026-06#^Q15725]]）。

## 三、肺部感染（CAP / HAP / 非典型 / TB / 真菌）

### 3.1 CAP 住院决策 + 经验治疗（源文原文）

**CURB-65**：Confusion / Uremia / 呼吸频率↑ / 低血压 / 年龄 ≥65；**1 条即可考虑住院**（本框架）。

| 场景 | 经验治疗（源文）|
|---|---|
| 门诊·年轻健康 | 阿莫西林 amoxicillin 或 多西环素 doxycycline（覆盖非典型）|
| 门诊·有 COPD（慢阻肺）/DM（糖尿病）/CHF（心衰）| 阿莫西林-克拉维酸 amox-clav + 阿奇霉素 azithromycin 或 左氧氟沙星 levofloxacin（典型+非典型）|
| 住院 | **静脉 头孢曲松 ceftriaxone + 阿奇霉素 azithromycin** 或 静脉 左氧氟沙星/莫西沙星 levo/moxifloxacin |
| HAP | **2 药**，含抗假单胞广谱 β-内酰胺（哌拉西林-他唑巴坦 pip-tazo / 头孢吡肟 cefepime / 头孢他啶 ceftazidime / 碳青霉烯 / 氨曲南 aztreonam）+ MRSA |
| VAP | **3 药** |

> [!tip] 肺部"识别钥匙"
> - **支原体 Mycoplasma 行走肺炎 walking pneumonia**：仍需经验抗生素，非"不用药"（[[mistakes/uworld-mistakes_2026-06#^Q19837]]）。
> - **军团菌 Legionella**：呼吸喹诺酮 fluoroquinolone / 大环内酯 macrolide；**克林霉素 clindamycin 无效**（[[mistakes/uworld-mistakes_2026-06#^Q3054]]）。
> - **PCP（Pneumocystis jirovecii pneumonia 耶氏肺孢子菌肺炎）**：干咳 + 气促 + HIV → 复方磺胺甲噁唑 TMP-SMX；**PaO2 ≤70 → 加糖皮质激素 corticosteroid**（[[mistakes/uworld-mistakes_2026-06#^Q3938]]）。
> - **流感后原发流感肺炎 influenza pneumonia**：骤起全身症状 + 双侧弥漫间质浸润（非 PCP，反复错点 [[mistakes/uworld-mistakes_2026-06#^Q4418]]）。
> - **CF（cystic fibrosis 囊性纤维化）肺炎病原**：幼儿 = 金葡 S. aureus；成人 = 铜绿假单胞 Pseudomonas。
> - **Nocardia 诺卡菌**：免疫抑制 + 肺结节 + 脑脓肿 + 部分抗酸分枝丝状 → 复方磺胺甲噁唑 TMP-SMX（[[mistakes/uworld-mistakes_2026-06#^Q4417]]）。

### 3.2 结核 TB（tuberculosis 结核病）— PPD 判读阈值（源文原文）

PPD = purified protein derivative 结核菌素皮试（= TST, tuberculin skin test）。

```
PPD 阳性阈值（mm）
≥5  : HIV / 免疫抑制 / 活动 TB 密切接触
≥10 : 医务 / 囚犯 / 5 年内移民 / 军人 / 尘肺 / 无家可归·酗酒
≥15 : 一般人群
```

- PPD 阳 → **下一步 CXR（chest X-ray 胸片）**：阴 → 潜伏 TB（LTBI, latent TB infection 潜伏结核感染）治疗（异烟肼 INH + 维生素 B6 ×9 月 / 利福平 RIF + INH ×3 月非 HIV / RIF ×4 月若 INH 耐药）；阳 → 启 RIPE（利福平+异烟肼+吡嗪酰胺+乙胺丁醇）+ 留 **3 份痰**。
- **TST 假阴（anergy 无反应性）**：CKD（慢性肾病）/HIV/播散 TB/营养不良/老年 → 改 IGRA（interferon-γ release assay 干扰素-γ 释放试验）或痰 AFB（acid-fast bacilli 抗酸杆菌）/NAAT（nucleic acid amplification test 核酸扩增）；老年 TB 多表现数月乏力消瘦（非典型，[[mistakes/uworld-mistakes_2026-06#^Q19418]]）。
- 新诊断 HIV + LTBI → 先治潜伏结核（[[mistakes/uworld-mistakes_2026-06#^Q3010]]）；婴儿多器官粟粒结节 + 慢咳接触史 = 粟粒性 TB（陷阱：误选 Listeria，[[mistakes/uworld-mistakes_2026-06#^Q19349]]）。

### 3.3 地方性双相真菌（钥匙地理）

- 组织胞浆菌 Histoplasma：俄亥俄/密西西比河谷、鸟/蝙蝠粪、洞穴。
- 球孢子菌 Coccidioides：西南沙漠（亚利桑那）、结节性红斑 erythema nodosum。
- 芽生菌 Blastomyces：中西部、疣状紫红皮损 + 痰镜见宽基出芽酵母（[[mistakes/uworld-mistakes_2026-06#^Q2999]]）。

## 四、泌尿道感染（UTI, urinary tract infection 尿路感染 / 肾盂肾炎）

UA = urinalysis 尿常规；WBC = white blood cell 白细胞。

| 类型 | best initial / most accurate | 经验治疗（源文）|
|---|---|---|
| 膀胱炎·健康女性·非复杂 | UA 尿常规 / 尿培养 | 复方磺胺甲噁唑 TMP-SMX ×**3 天**（或呋喃妥因 nitrofurantoin）|
| 复杂膀胱炎（男 / 孕 / 糖尿病）| UA / 尿培养 | ×**7 天**（TMP-SMX / 阿莫西林 amoxicillin / 头孢 / 呋喃妥因）|
| 复发·与性交无关 | —— | 每日低剂量 TMP-SMX |
| 复发·与性交相关 | —— | 事后排尿 + 事后 TMP-SMX |
| 孕妇无症状菌尿（产检查出）| —— | **必治**：阿莫西林或呋喃妥因 |
| 肾盂肾炎 pyelonephritis | UA 见 **WBC 管型 cast** / 尿培养 | —— |

- 复发定义：6 个月 ≥2 次 或 1 年 ≥3 次。
- WBC 无管型 + 无病原 → 想 **乳头坏死 papillary necrosis**（镰贫 sickle cell / 镇痛药 / 糖尿病 DM）。
- 婴儿 UTI 经验抗生素（[[mistakes/uworld-mistakes_2026-06#^Q15829]]）；UTI 病原与亚硝酸盐 nitrites 判读（[[mistakes/uworld-mistakes_2026-06#^Q4196]]）。

## 五、皮肤软组织（SSTI, skin and soft tissue infection 皮肤软组织感染）/ 骨关节 / 头颈

> [!tip] SSTI 与头颈"识别钥匙"
> - **急性单侧颈淋巴结炎 lymphadenitis**：病原 金葡 S. aureus / 化脓性链球菌 GAS（group A Streptococcus A 族链球菌）→ 经验 **克林霉素 clindamycin**；陷阱是误选猫抓病 cat-scratch disease（反复错 [[mistakes/uworld-mistakes_2026-06#^Q4843]]、[[mistakes/uworld-mistakes_2026-06#^Q4353]]）。
> - **Ludwig angina 路德维希咽峡炎（口底蜂窝织炎）**：源自牙根（非扁桃体）（[[mistakes/uworld-mistakes_2026-06#^Q3530]]）。
> - **Lemierre 综合征 Lemierre syndrome**：咽炎后颈静脉血栓 + 空洞肺结节 = 坏死梭杆菌 Fusobacterium（[[mistakes/uworld-mistakes_2026-06#^Q11863]]）。
> - **猫咬伤**：阿莫西林-克拉维酸 amox-clav 预防（多杀巴斯德菌 Pasteurella，[[mistakes/uworld-mistakes_2026-06#^Q4317]]）。
> - **骨髓炎 osteomyelitis**：镰状细胞病 SCD（sickle cell disease）→ 抗葡萄球菌 + 三代头孢覆盖 沙门菌 Salmonella（[[mistakes/uworld-mistakes_2026-06#^Q3571]]）；假体关节感染 PJI（prosthetic joint infection）看时相与病原（[[mistakes/uworld-mistakes_2026-06#^Q9111]]）。
> - **麻风 leprosy 麻风病**：无痛低色素麻木斑 + 周围神经粗大 → 皮肤活检（[[mistakes/uworld-mistakes_2026-06#^Q3425]]）。

## 六、感染性心内膜炎（IE, infective endocarditis）

> [!warning] IE 核心钥匙
> **发热 + 新杂音 = 心内膜炎**。
> - 急性 = **金葡 S. aureus**，三尖瓣，健康瓣（右心赘生物 → 肺栓塞）。
> - 亚急性 = **草绿色链球菌 S. viridans / mutans**，二尖瓣，受损瓣。
> - SLE（systemic lupus erythematosus 系统性红斑狼疮）= Libman-Sacks 利-萨心内膜炎（无菌赘生物，瓣两侧）。
> - 血培养长出 **牛链球菌 S. gallolyticus / 败毒梭菌 C. septicum → 结肠镜 colonoscopy**（查结肠癌）。

- Dx 顺序：疑 IE → **先抽血培养**（勿先做超声 / 先给药，[[mistakes/uworld-mistakes_2026-06#^Q3012]]）；确诊 = 超声心动（best initial = TEE, transesophageal echo 经食道超声）+ 血培养（[[mistakes/uworld-mistakes_2026-06#^Q4668]]）。
- **IVDU（intravenous drug user 静脉吸毒者）三尖瓣 IE**：金葡 S. aureus 脓毒肺栓（非 PCP，[[mistakes/uworld-mistakes_2026-06#^Q4388]]）→ 经验 **万古霉素 vancomycin**（[[mistakes/uworld-mistakes_2026-06#^Q3013]]）。
- 脾脓肿三联 + MVP（mitral valve prolapse 二尖瓣脱垂）→ 想 IE 脓毒栓子（[[mistakes/uworld-mistakes_2026-06#^Q4714]]）。

## 七、腹腔 / 肝脏 / 感染性腹泻

- **血性腹泻病原**：弯曲杆菌 Campylobacter（→ 吉兰-巴雷 GBS, Guillain-Barré syndrome）/ 肠出血性大肠杆菌 EHEC（enterohemorrhagic E. coli，→ 溶血尿毒综合征 HUS, hemolytic uremic syndrome，未熟牛肉）/ 志贺菌 Shigella（儿童惊厥，[[mistakes/uworld-mistakes_2026-06#^Q12458]]）/ 沙门菌 Salmonella / 耶尔森菌 Yersinia / 弧菌 Vibrio。
- **<8 小时起病的肠胃炎**：金葡 S. aureus / 蜡样芽孢杆菌 Bacillus cereus（炒饭）= 预成毒素。
- **贾第虫 Giardia**：旅行 / 露营 / 水样泻 → 治后摄奶再泻 = 继发乳糖不耐受（非治疗失败）。
- **肝占位钥匙**：旅行 + 腹泻史 + 肝右叶单发 = **阿米巴肝脓肿 amebic liver abscess**（非化脓性，[[mistakes/uworld-mistakes_2026-06#^Q2968]]）；肝囊肿含子囊 + 犬/牧区 = **包虫病 echinococcosis**（[[mistakes/uworld-mistakes_2026-06#^Q3789]]）；胆道支架后发热 RUQ（right upper quadrant 右上腹）+ ALP（alkaline phosphatase 碱性磷酸酶）↑ = 化脓性肝脓肿 → 腹部 CT（[[mistakes/uworld-mistakes_2026-06#^Q17223]]）。

## 八、STI 与生殖器综合征（溃疡 / 分泌物）

### 8.1 生殖器溃疡（按"痛 vs 不痛 + 淋巴结"）

- **HSV（herpes simplex virus 单纯疱疹病毒）生殖器疱疹**：成簇浅溃疡 + 触痛淋巴结 + 全身症状（[[mistakes/uworld-mistakes_2026-06#^Q10068]]）。
- **梅毒 syphilis 一期硬下疳 chancre**：无痛硬下疳 → Tx 苄星青霉素 benzathine penicillin；重度青霉素过敏 → 多西环素 doxycycline（[[mistakes/uworld-mistakes_2026-06#^Q3256]]）。
- 软下疳 chancroid（痛）/ 性病性淋巴肉芽肿 LGV（lymphogranuloma venereum）/ 腹股沟肉芽肿 granuloma inguinale 见深挖卡。

### 8.2 阴道分泌物 / 宫颈炎（源文经验治疗）

| 病因 | 钥匙 / Dx | 治疗 |
|---|---|---|
| 念珠菌 Candida（念珠菌病）| 痒 + 凝乳样；复发查 HIV/糖尿病 | 氟康唑 fluconazole |
| 加德纳菌 Gardnerella（细菌性阴道病 BV, bacterial vaginosis）| 鱼腥味 + 线索细胞 clue cells | 甲硝唑 metronidazole |
| 滴虫 Trichomonas（滴虫病）| 动滴虫（湿片 wet mount）+ 多性伴 | 甲硝唑 metronidazole（**夫妻同治**）|
| 淋病 Gonorrhea | —— | **头孢曲松 ceftriaxone** |
| 衣原体 Chlamydia | —— | 阿奇霉素 azithromycin（孕）或 多西环素 doxycycline |

### 8.3 梅毒血清学

## 九、HIV / AIDS — 按 CD4 机会性感染

HIV = human immunodeficiency virus 人类免疫缺陷病毒；AIDS = acquired immunodeficiency syndrome 获得性免疫缺陷综合征；CD4 = CD4+ T 淋巴细胞计数。

- Dx：best initial = ELISA（酶联免疫吸附）→ 确诊 Western blot（蛋白印迹）；婴儿 = PCR RNA。
- Tx：2 NRTI（核苷类逆转录酶抑制剂）+ 整合酶抑制剂 integrase inhibitor（整科主流框架）。
- **急性 HIV** 可致无菌性脑膜炎（[[mistakes/uworld-mistakes_2026-06#^Q19266]]）；急性逆转录综合征 + 单核样表现。

```
CD4 阈值 → 机会性感染（钥匙）
CD4 <200 ── PCP 耶氏肺孢子菌肺炎（干咳气促）→ TMP-SMX；念珠菌食管炎
CD4 <100 ── 弓形虫 Toxoplasma（环形强化）；隐球菌 Cryptococcus 脑膜炎
CD4 <50  ── MAC 鸟分枝杆菌复合群（持续咳+贫血+反复感染）；CMV 巨细胞病毒（视网膜炎/结肠炎）
其他      ── PML 进行性多灶性白质脑病（JC 病毒，白质灶无强化）
```

> 缩写：MAC = Mycobacterium avium complex 鸟分枝杆菌复合群；CMV = cytomegalovirus 巨细胞病毒；PML = progressive multifocal leukoencephalopathy 进行性多灶性白质脑病。

- AIDS 血性腹泻 → CMV 结肠炎（[[mistakes/uworld-mistakes_2026-06#^Q2323]]）；CD4<50 播散性 MAC 治疗（[[mistakes/uworld-mistakes_2026-06#^Q2267]]）；念珠菌血症 candidemia → 抗真菌 + 拔管（[[mistakes/uworld-mistakes_2026-06#^Q2182]]）。

🧩 AIDS 机会性感染按 CD4 谱 → [[完整笔记/专题笔记/感染/感染_AIDS机会性感染按CD4谱]]

## 十、蜱传 / 人畜共患 / 寄生虫

> [!tip] 蜱传与人畜共患"识别钥匙"
> - **莱姆病 Lyme disease**（硬蜱 Ixodes，游走性红斑 ECM, erythema chronicum migrans）：成人 多西环素 doxycycline；**孕妇 / 儿童轻症 → 阿莫西林 amoxicillin**；心阻滞 / 神经型 → 头孢曲松 ceftriaxone。孕妇早期 Lyme 用阿莫西林是反复错点（[[mistakes/uworld-mistakes_2026-06#^Q3257]]）。
> - **落基山斑疹热 RMSF**（Rocky Mountain spotted fever，立克次体 Rickettsia，腕踝向心性瘀点疹 + 血小板↓/Na↓）→ 多西环素 doxycycline（小孩也用）。
> - **埃立克体病 Ehrlichiosis**（[[mistakes/uworld-mistakes_2026-06#^Q2596]]）/ **巴贝虫病 Babesiosis**（无脾 + 血管内溶血 → 血涂片 blood smear，[[mistakes/uworld-mistakes_2026-06#^Q2777]]）。
> - **钩端螺旋体病 Leptospirosis**：夏威夷/洪水/鼠尿 + 结膜充血无渗出 conjunctival suffusion + 腓肠肌痛 + 黄疸 + 肌酐 Cr↑ → 多西环素 doxycycline。
> - **兔热病 tularemia**：溃疡 + 化脓区域淋巴结 + 野生动物接触（[[mistakes/uworld-mistakes_2026-06#^Q106792]]）。

## 十一、发热出疹 / 单核样综合征（儿科病毒疹）

- **麻疹 Measles**：Koplik 斑 + 3C（咳/卡他/结膜炎）+ 离心性疹。
- **风疹 Rubella**：耳后淋巴结 + 先天"风吹过耳，心眼瞎"（动脉导管未闭 PDA, patent ductus arteriosus / 耳聋/白内障）。
- **传染性单核细胞增多症 mononucleosis（EBV, Epstein-Barr virus EB 病毒）**：迁延咽炎 + 全身淋巴结 + 显著乏力 → 嗜异性抗体 heterophile（[[mistakes/uworld-mistakes_2026-06#^Q4354]]）；忌氨苄西林 ampicillin 出疹、忌运动防脾破。
- **猩红热 Scarlet fever**：砂纸样疹 + 皮褶 Pastia 线（[[mistakes/uworld-mistakes_2026-06#^Q3141]]）。
- **水痘 varicella（VZV, varicella-zoster virus 水痘-带状疱疹病毒）**：不同阶段水疱共存 + 强烈瘙痒（含疫苗后突破，[[mistakes/uworld-mistakes_2026-06#^Q2768]]）。
- **传染性软疣 molluscum contagiosum**：中央脐凹肉色丘疹 → 观察安慰（非虐待上报，[[mistakes/uworld-mistakes_2026-06#^Q19310]]）。

## 十二、医院感染 / 暴露后预防 / 疫苗

- **CLABSI（central line-associated bloodstream infection 中心导管相关血流感染）**：bundle = 最大无菌屏障 + 氯己定 chlorhexidine 皮肤消毒 + 首选锁骨下静脉 + scrub the hub。
- **PEP（post-exposure prophylaxis 暴露后预防）**：百日咳 pertussis 暴露 → 大环内酯 macrolide（[[mistakes/uworld-mistakes_2026-06#^Q3906]]）；脑膜炎球菌密接 → 利福平 rifampin / 环丙沙星 ciprofloxacin / 头孢曲松 ceftriaxone。
- **疫苗**：百日咳成人发病主因 = 疫苗免疫衰减（[[mistakes/uworld-mistakes_2026-06#^Q3905]]）。

## 复习题（自测，先不看答案）

> [!question]- 脑膜炎患者 next step 顺序是什么？什么时候 LP 前先做 CT？
> 顺序：① 血培养 → ② **经验抗生素**（勿等腰穿）→ ③ 决定 LP vs CT。
> LP 前先 CT 指征：神志改变 / 局灶缺损 / 癫痫 / 视乳头水肿。"先腰穿才给药"是陷阱。

> [!question]- PPD 三档阳性阈值（5/10/15 mm）各对应哪些人群？阳性后下一步？
> ≥5：HIV / 免疫抑制 / 活动 TB 密接；≥10：医务/囚犯/5 年内移民/军人/尘肺/无家可归·酗酒；≥15：一般人群。
> 阳性 → 下一步 CXR；阴 → 潜伏 TB 治（INH+B6×9 月等）；阳 → RIPE + 3 份痰。

> [!question]- 非复杂膀胱炎（健康女性）vs 复杂膀胱炎，疗程与药物各是什么？
> 非复杂：TMP-SMX ×3 天（或呋喃妥因）。复杂（男/孕/糖尿病）：×7 天。孕妇无症状菌尿必治（阿莫西林/呋喃妥因）。

> [!question]- IE 疑诊时第一步做什么？IVDU 右心 IE 经验抗生素是什么？
> 疑 IE → **先抽血培养**（勿先超声 / 先给药）。确诊 = 超声心动（TEE）+ 血培养。
> IVDU 三尖瓣 IE（S. aureus）→ 经验 **万古霉素**（陷阱是误选头孢曲松）。

> [!question]- 孕妇早期 Lyme（游走性红斑）经验治疗选什么？为什么不是 doxycycline？
> **amoxicillin**。doxycycline 孕妇 + 儿童禁用（牙着色/骨）；心阻滞/神经型才用 ceftriaxone。此为我反复错点（×3）。

> [!question]- 按 CD4 排：PCP / Toxoplasma / Cryptococcus / MAC / CMV 各在什么阈值以下？
> <200 PCP；<100 Toxoplasma + Cryptococcus；<50 MAC + CMV。

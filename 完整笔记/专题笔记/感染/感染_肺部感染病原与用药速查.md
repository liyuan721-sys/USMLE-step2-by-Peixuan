# 肺部感染病原 → 用药速查

> [!info] 笔记定位
> 把"**看到这个情景 → 想这个病原 → 用这个药**"的肺部感染题压成一张速查。两条主线：① **CAP 经验治疗分层**（门诊/住院/ICU）② **"看人下菜"特殊场景**（暴露/宿主 → 特定病原 + 对口药）。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q19837]] 支原体 / [[mistakes/uworld-mistakes_2026-06#^Q3054]] 军团菌 / [[mistakes/uworld-mistakes_2026-06#^Q4388]] 三尖瓣 IE 脓毒肺栓 / [[mistakes/uworld-mistakes_2026-06#^Q11863]] Lemierre / [[mistakes/uworld-mistakes_2026-06#^Q20113]] 脓胸 / [[mistakes/uworld-mistakes_2026-06#^Q3905]] 百日咳。

## 一、CAP 经验治疗分层 ⭐⭐⭐

| 场景 | 经验方案 | 覆盖逻辑 |
|---|---|---|
| **门诊·健康无合并症** | **Amoxicillin**（高剂量）或 doxycycline 或大环内酯 | 主覆肺炎链球菌 |
| **门诊·有合并症**（心肺肝肾病/DM/酗酒/免疫抑制） | **β-lactam（amox-clav/头孢）+ 大环内酯** 或 **呼吸喹诺酮单药** | 加非典型 + 耐药菌 |
| **住院·非 ICU** | **Ceftriaxone + 大环内酯（azithromycin）** 或 **呼吸喹诺酮（levofloxacin）单药** | 链球菌 + 非典型 |
| **ICU 重症** | **β-lactam（ceftriaxone/ampi-sulbactam）+（大环内酯 或 呼吸喹诺酮）** | 不可单用喹诺酮以外的单药 |
| **疑 MRSA**（坏死/流感后/重症） | 加 **vancomycin / linezolid** | clindamycin **不替代主力**（诱导耐药 iMLSb + 抑菌）；仅坏死/毒素介导时**加上去抑毒素**或用于 CA-MRSA 皮肤软组织 |
| **疑 Pseudomonas**（结构肺病/近期住院/抗生素） | **抗铜绿 β-lactam（cefepime / pip-tazo / meropenem / ceftazidime）± 第二抗铜绿药（氨基糖苷 或 抗铜绿 FQ：cipro/levo）** | 重症/脓毒/高耐药风险 → **双覆盖**；稳定低风险 → 单药即可 |

> [!tip] 一句话
> **住院 CAP = "β-lactam + 大环内酯" 或 "呼吸喹诺酮单药"** —— 都把"典型链球菌 + 非典型"一次覆盖。

## 二、非典型肺炎（Mycoplasma / Legionella / Chlamydophila）⭐⭐⭐

> [!danger] 铁律：非典型/胞内病原**只认大环内酯 / 呼吸喹诺酮 / 多西环素**
> **β-内酰胺（amox/头孢）、clindamycin、vancomycin 一概无效**：
> - **Mycoplasma 无细胞壁** → β-内酰胺没靶点
> - **Legionella / Chlamydophila 胞内** → 需药进细胞且有活性；β-内酰胺进不去
> - **clindamycin 虽机制像大环内酯（50S + MLSb 交叉耐药），但对非典型固有无活性** → 也不行（[[mistakes/uworld-mistakes_2026-06#^Q3054]]）

| 病原 | 招牌线索 | 用药 |
|---|---|---|
| **Mycoplasma pneumoniae** | 年轻 + 聚集（学校/军营）+ 缓起轻"行走"肺炎 + 双侧网状浸润 + **冷凝集素** | **Azithromycin** 首选 / 呼吸喹诺酮（[[mistakes/uworld-mistakes_2026-06#^Q19837]]）|
| **Legionella pneumophila** | 污染水（**邮轮/酒店/医院空调**）+ 高热 + **腹泻** + **意识混乱** + **低钠** + 相对缓脉；尿抗原确诊 | **呼吸喹诺酮（levofloxacin）** 或 azithromycin；住院偏喹诺酮（[[mistakes/uworld-mistakes_2026-06#^Q3054]]）|
| **Chlamydophila pneumoniae** | 青少年/年轻、声嘶、缓起 | 大环内酯 / 多西环素 |
| **C. psittaci（鹦鹉热）** | **鸟类接触** + 头痛高热 | **多西环素** |
| **Coxiella（Q fever）** | 牲畜/产羔接触 | 多西环素 |

> [!warning] "walking pneumonia" ≠ 不用药
> 行走肺炎只是症状轻，**有浸润就是真肺炎 → 需经验抗生素**（防人际传播 + 肺外并发如脑炎）。别选"无需抗生素"（[[mistakes/uworld-mistakes_2026-06#^Q19837]]）。

## 三、看人下菜：暴露/宿主 → 病原 → 用药 ⭐⭐⭐

| 情景钩子 | 病原 | 用药 / 处理 |
|---|---|---|
| **IVDU + 多发空洞肺结节** | **三尖瓣 IE → S. aureus**（脓毒肺栓）| 抗葡萄球菌：**MSSA→nafcillin/cefazolin；MRSA/重过敏→vanc**（[[mistakes/uworld-mistakes_2026-06#^Q4388]]）|
| **咽炎后 + 胸锁乳突肌颈肿 + 空洞肺结节** | **Lemierre → Fusobacterium**（颈静脉脓毒栓）| IV 抗生素覆厌氧（β-lactam/BLI、metronidazole）（[[mistakes/uworld-mistakes_2026-06#^Q11863]]）|
| **肺炎 + loculated 胸腔积液（脓胸）** | 肺炎链球菌 + **S. aureus（含 MRSA）** | **ceftriaxone + vancomycin + 引流**（[[mistakes/uworld-mistakes_2026-06#^Q20113]]）|
| **阵发咳 + 咳后呕吐（成人疫苗衰减）** | **Bordetella pertussis** | **大环内酯（azithromycin）** + 密接 PEP（[[mistakes/uworld-mistakes_2026-06#^Q3905]] / [[mistakes/uworld-mistakes_2026-06#^Q3906]]）|
| **SCD 发热脓毒症（无骨痛）** | **肺炎链球菌**（功能性无脾）| ceftriaxone（[[mistakes/uworld-mistakes_2026-06#^Q3640]]）|
| **SCD 急性胸综合征（ACS）** | 感染/脂肪栓塞 | **ceftriaxone + azithromycin** + O₂ + 重症换血（[[完整笔记/专题笔记/血液/血液_镰刀细胞病SCD]] §3.3）|
| **酒精/糖尿病 + "currant jelly"痰 + 上叶空洞** | Klebsiella pneumoniae | 三代头孢/抗铜绿 β-lactam 据药敏 |
| **误吸（酗酒/意识障碍）+ 下垂肺段/腐臭痰** | 口腔厌氧菌 | **ampicillin-sulbactam / clindamycin** |
| **HIV CD4<200 + 双侧间质 + 亚急性** | **PCP（Pneumocystis）** | **TMP-SMX**（±激素若 PaO₂<70）；非空洞结节（[[完整笔记/专题笔记/感染/感染_AIDS机会性感染按CD4谱]]）|
| **数周咳 + 盗汗体重↓ + 上叶空洞** | **TB** | RIPE；潜伏 → INH/利福（[[mistakes/uworld-mistakes_2026-06#^Q3246]]）|
| **流感后继发细菌肺炎** | S. pneumoniae / **S. aureus**（含 MRSA）| 加 vancomycin/linezolid |

## 四、"多发空洞性肺结节"= 脓毒性肺栓子（高频鉴别）⭐

> [!tip] 看到多发外周结节 ± 空洞 → 先想脓毒栓子两源
> | 源 | 病原 | 钥匙 |
> |---|---|---|
> | **右心三尖瓣 IE** | S. aureus | **IVDU** + 针痕 |
> | **Lemierre** | Fusobacterium | **咽炎后 + 胸锁乳突肌颈肿 + 颈静脉血栓** |
> 
> 再排：TB（亚急性盗汗）/ 真菌 / GPA（肉芽肿性多血管炎）/ 鳞癌。

## 五、高频陷阱

- ❌ **非典型肺炎用 β-lactam / clindamycin / vancomycin** → 无效；只认大环内酯/喹诺酮/多西环素
- ❌ **"walking pneumonia 轻 → 不用药"** → 有浸润就治
- ❌ **HIV 肺病灶反射 PCP** → 先看 **CD4 分层（<200 才高危）+ 影像形态**（PCP 双侧间质，非空洞结节）
- ❌ **脓胸只升头孢不加 MRSA + 不引流** → 中毒貌脓胸要 vanc + 引流
- ❌ **Legionella 用头孢/克林** → 无效；尿抗原确诊 + 喹诺酮/大环内酯
- ❌ **clindamycin 当大环内酯替代治非典型** → 机制像但谱不覆盖胞内非典型

## 六、Memory Hook

```
住院 CAP = β-lactam + 大环内酯 / 呼吸喹诺酮单药
非典型(支原体/军团/衣原体) = 大环内酯/喹诺酮/多西环素（β-lactam·克林·万古 全无效）
空洞肺结节 = 脓毒栓子（IVDU三尖瓣IE金葡 / Lemierre咽炎后Fusobacterium）
脓胸 = ceftriaxone + vanc + 引流
百日咳/SCD-ACS = 含 azithromycin
HIV 肺病 = 先看 CD4（<200→PCP→TMP-SMX；双侧间质非空洞）
```

# 咽炎鉴别与 GAS 诊治

> [!info] 笔记定位
> **咽炎（pharyngitis）的"病原鉴别 + GAS 检测流程 + 并发症"三件套。**
> Step 2 CK 咽炎题三向：① 病原（GAS vs EBV vs 病毒）② **儿童 RADT 阴 → 咽培养**的检测流程 ③ 链球菌后并发症（ARF / PSGN / 猩红热）。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q2193]]（儿童 RADT 阴 → 咽培养）+ [[mistakes/uworld-mistakes_2026-06#^Q3969]]（猩红热）。

---

## 一、咽炎病原鉴别 ⭐

| 病原 | 钥匙 | 处理 |
|---|---|---|
| **GAS（S. pyogenes）** | **无病毒症状** + 扁桃体渗出 + **前颈压痛淋巴结** + 软腭瘀点 + 发热；**无咳嗽/鼻涕** | RADT/培养 → **penicillin/amoxicillin** |
| **EBV 单核细胞增多症** | **青少年** + **后颈淋巴结** + 脾大 + 渗出 + 疲乏；**嗜异性抗体⊕** | 支持；**避 amox（致疹）+ 避剧烈运动（脾破裂）** |
| **病毒性（腺病毒等）** | **有病毒症状**（咳嗽/鼻涕/结膜炎/口腔溃疡）| 支持 |
| **疱疹性咽峡炎** | Coxsackie A；**后口腔（软腭/腭垂）灰疱溃疡** + 夏末初秋 | 支持 |

> [!warning] GAS vs EBV 关键分水岭（你栽过）
> 渗出性扁桃体炎两者都可有；分**年龄 + 淋巴结位置 + 病毒症状**：**GAS = 无病毒症状 + 前颈**；**EBV = 青少年 + 后颈 + 脾大**。别一看渗出就反射 mono（[[mistakes/uworld-mistakes_2026-06#^Q2193]]）。

---

## 二、GAS 检测流程（儿童 vs 成人）⭐⭐⭐

```
有病毒症状（咳嗽/结膜炎/口腔溃疡/鼻涕）？
  ├─ 有 → 病毒性 → 支持治疗（不查 GAS）
  └─ 无 → RADT 快速抗原（特异高、敏感有限）
           ├─ 阳 → GAS → penicillin / amoxicillin
           └─ 阴 → 【儿童】咽拭子培养确认 ⭐
                     ├─ 阳 → 治疗
                     └─ 阴 → 病毒性 → 支持
           ※【成人】RADT 阴无需培养确认（ARF 风险低）
```

> [!danger] 儿童 RADT 阴必做培养
> **儿童未治 GAS 的 ARF（急性风湿热）风险远高于成人** → **RADT 阴性用咽培养兜底**（敏感性更高）。成人阴性不必。

> [!tip] 治疗
> **一线 penicillin V / amoxicillin**（PCN 过敏 → cephalexin / macrolide）。治疗目的：缩短病程 + 减传播 + **预防 ARF**（PSGN 不被抗生素预防）。

## 三、链球菌后并发症 ⭐

| 并发症 | 特点 | 抗生素能否预防 |
|---|---|---|
| **猩红热 Scarlet fever** | GAS 咽炎 + 红疹毒素 → **砂纸样疹 + 皮褶 + 草莓舌 + 脱皮** | 同 GAS 治疗（见 [[完整笔记/专题笔记/儿科/儿科_发热出疹病鉴别]]）|
| **急性风湿热 ARF** | **咽炎后** 2-4 周；Jones 标准（心炎/关节/舞蹈/皮下结节/边缘红斑）| **能预防**（及时治 GAS 咽炎）|
| **PSGN** | 咽炎**或脓皮病**后；血尿/红细胞管型/水肿/HTN/低补体 | **不被抗生素预防** |

> [!info] ARF vs PSGN 来源差异
> **ARF 只跟咽炎**；**PSGN 咽炎或皮肤感染（脓疱疮）都可**。抗生素**只防 ARF，不防 PSGN**。

## 四、Memory Hook

```
病原
  无病毒症状+前颈+渗出 = GAS → penicillin
  青少年+后颈+脾大+嗜异性⊕ = EBV mono（避amox/避运动）
  有病毒症状（咳嗽鼻涕）= 病毒 → 支持
  后口腔灰疱（夏秋）= 疱疹性咽峡炎(Coxsackie)

流程
  有病毒症状 → 支持；无 → RADT
  儿童 RADT 阴 → 咽培养确认（ARF风险）；成人不必

并发
  猩红热（砂纸疹）/ ARF（咽炎后,能防）/ PSGN（咽or皮,不能防）
```

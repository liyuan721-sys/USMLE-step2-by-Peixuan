# 抗生素高频 20 场景秒杀 + 致死 5

> [!info] 笔记定位
> **抗生素的"题型字典"，不是教科书。**
> USMLE Step 2 CK 抗生素题 80% 集中在 20 个场景里，5 个致死陷阱必考。
> 用法：**默写整张速查表 + 致死 5 → 反复回炉 → 错题驱动扩展**。
> 不要试图覆盖所有抗生素，**够用就行**。

## 一、设计哲学 — 为什么这样组织

### 1.1 抗生素题型本质

USMLE Step 2 抗生素题 95% 是 **"Stem 给场景 → 选药"**，不是 "给药问适应症"。
所以记忆方向必须**反过来**：**场景 = key，药 = value**。

### 1.2 三层认知

| 层 | 内容 | 记忆方式 |
|---|---|---|
| **L1 — 高频 20 场景** | 第一选择必须 < 3 秒答出 | 死记 + 默写 |
| **L2 — 致死 5 陷阱** | 错 = 直接送命 | 反向 Anki 卡 |
| **L3 — 修改条件** | PCN 过敏 / 孕妇 / 儿童 / CKD / 老人 | 触发词 → 替换药 |

### 1.3 使用顺序

```
模考前 24h
  ↓
默写 §二 速查表（20 行 5 列）— 30 min
  ↓
默写 §三 致死 5（5 个陷阱）— 10 min
  ↓
扫 §四 修改条件触发词表 — 10 min
  ↓
不会的标记 → §六 错题驱动扩展
```

## 二、高频 20 场景秒杀速查表 ⭐⭐⭐（必背）

> [!danger] 这张表是本笔记的核心
> 每一行都是一道经常出现的 USMLE 题。**默写到全对为止。**

| # | 场景识别钥匙 | 首选药 | 干扰项 / 不选什么 | 修改条件 |
|---|---|---|---|---|
| **1** | **外科手术预防**（切皮前 30-60 min） | **Cefazolin** IV | 不用 vanc（除非已知 MRSA / β-lactam 过敏） | MRSA 已知 → Vanc；结肠手术 → 加 Metronidazole |
| **2** | **门诊蜂窝织炎 / 单纯 SSTI**（无脓 = non-purulent） | **Cephalexin** PO 或 **Dicloxacillin** PO | 不用 TMP-SMX / Doxy（这两个用于有脓 / MRSA 怀疑） | 有脓 / MRSA 怀疑 → **TMP-SMX** 或 **Doxy** |
| **3** | **PID 住院**（pelvic inflammatory disease） | **Cefoxitin / Cefotetan IV + Doxy** | 不用 Ceftriaxone 单药（不覆盖厌氧）| 门诊轻 → Ceftriaxone IM + Doxy PO + Metronidazole |
| **4** | **社区肺炎住院 — 非 ICU** | **Ceftriaxone + Azithromycin** | 不单用 ceftriaxone（漏 atypical）；不单用 azithro（漏 strep pneumo 耐药）| 呼吸 FQ（levofloxacin / moxifloxacin）单药也可；ICU 加 vanc |
| **5** | **细菌性脑膜炎经验**（>1 月 ~ 50 岁） | **Vancomycin + Ceftriaxone** | 不单用 ceftri（漏耐药肺炎链球菌）；不单用 vanc（漏 G⁻） | >50 / 免疫低 / 酒精 / 孕妇 → **加 Ampicillin**（cover Listeria）<br>新生儿 → **Amp + Cefotaxime / Gent**（禁 ceftri）|
| **6** | **淋病** | **Ceftriaxone 500 mg IM × 1 + Doxy 100 mg PO BID × 7d** | 2021 后不再用 azithro 联用；不单用 doxy | 同时治 partner |
| **7** | **SBP（自发性细菌性腹膜炎）** | **Cefotaxime** / Ceftriaxone IV | 不用 vanc（G⁻ 为主） | 预防：**Ciprofloxacin** PO 或 **TMP-SMX** |
| **8** | **急性肾盂肾炎住院** | **Ceftriaxone IV** | 不用 nitrofurantoin（不进肾实质）；不用 fosfomycin（仅膀胱炎）| 门诊轻 → **Cipro PO** 或 TMP-SMX |
| **9** | **Lyme — 神经 / 心脏 / 严重关节** | **Ceftriaxone IV 14-28d** | 早期局限（仅 EM 皮疹）→ **Doxy PO**（不用 ceftri） | 孕妇 → Amoxicillin（替代 doxy） |
| **10** | **Neutropenic fever**（中性粒 <500 + 发热 ≥38.3°C / ≥38°C × 1h） | **Cefepime 单药** | 不加 vanc（除非：低血压 / 皮肤软组织感染 / 留置导管感染 / 已知 MRSA 定植）| Pip-Tazo / Meropenem 也可作单药 |
| **11** | **HAP / VAP**（医院 / 呼吸机肺炎） | **Cefepime + Vancomycin**（或 Pip-Tazo + Vanc） | 不用 ceftriaxone（漏 Pseudo + MRSA） | MRSA 风险低 → 可不加 vanc |
| **12** | **IVDU IE 经验** | **Vancomycin + Cefepime** | 不用 ceftri（漏 MRSA + Pseudo） | 培养出 MSSA → 降阶梯 Nafcillin |
| **13** | **自然瓣 IE 经验**（非 IVDU） | **Vancomycin + Ceftriaxone** | 不加 cefepime（无 Pseudo 风险） | MSSA 培养 → Nafcillin / Cefazolin |
| **14** | **Enterococcus IE** | **Ampicillin + Gentamicin** 或 **Ampicillin + Ceftriaxone** | 头孢单药无效；vanc 仅 PCN 过敏 | VRE → Linezolid / Daptomycin |
| **15** | **MRSA 菌血症 / 复杂 SSTI** | **Vancomycin** IV（首选） | 不用任何头孢 1-4 代；不用 dapto 治肺炎（surfactant 灭活） | Vanc 失败 → **Daptomycin**（非肺炎）/ **Linezolid** / **Ceftaroline** |
| **16** | **单纯下尿路感染（女性膀胱炎）** | **Nitrofurantoin** 5d / **TMP-SMX** 3d / **Fosfomycin** 1d | 不用 cipro / amox / cephalexin 当首选（耐药 / 副作用）| 孕妇 → Nitrofurantoin（孕中期）/ Amox-Clav / Cephalexin |
| **17** | **C. difficile** | **首发 / 非重症**：**Fidaxomicin** PO 或 **Vancomycin PO** | 绝不用 **Metronidazole PO** 作首选（2018 指南后降级，仅在 Vanc/Fidax 不可用时备选）；不用 vanc IV（不进肠腔）| 重症 / fulminant → Vanc PO + IV Metronidazole；复发 → Fidaxomicin |
| **18** | **Atypical CAP**（Mycoplasma / Chlamydophila / Legionella） | **Azithromycin** 或 **Doxycycline** 或 **呼吸 FQ**（levo/moxi） | 不用任何 β-lactam（atypicals 无细胞壁 — Mycoplasma；细胞内 — Chlamydophila/Legionella） | Legionella 重症 → Levo > Azithro |
| **19** | **Latent TB**（IGRA / PPD ⊕ + CXR 阴性 + 无症状） | **Rifampin 4 月** 或 **INH + Rifapentine 3 月（3HP，每周一次）** 或 **INH 9 月** | 不用 RIPE 4 药（那是 active TB）| 选 4R / 3HP > 9H（短疗程依从好）|
| **20** | **Active TB** | **RIPE × 2 月 → INH + Rifampin × 4 月** | 不单药（必耐药）| **R**ifampin / **I**NH / **P**yrazinamide / **E**thambutol；INH 加 **B6 防神经病** |

> - **手术 → Cef**a**z**olin（外科一线 z = zero risk）
> - **门诊皮肤 → Ce**ph**alexin**（无脓）/ **TMP-SMX**（有脓 / MRSA）
> - **社区肺炎 → Cef**tri**axone + Azithro**（β-lactam + atypical 双覆盖）
> - **脑膜炎 → Vanc + Ceftri**（>50/免疫低 +Amp；新生儿换 Cefotaxime）
> - **淋病 → Ceftri 500 IM + Doxy 7d**（2021 后单药 ceftri）
> - **Neutropenic → Cef**epime** 单药**
> - **HAP/VAP → Cef**epime + Vanc**
> - **IVDU IE → Vanc + Cef**epime**
> - **自然瓣 IE → Vanc + Ceftri**
> - **Entero IE → Amp + Gent**（or Amp+Ceftri）
> - **MRSA → Vanc**
> - **膀胱炎 → Nitro / TMP-SMX / Fosfo**
> - **C. diff → Fidax / Vanc PO**（不再 metro）
> - **Atypical CAP → Azithro / Doxy / 呼吸 FQ**
> - **Latent TB → 4R / 3HP / 9H**
> - **Active TB → RIPE**

## 三、致死 5 陷阱 ⭐⭐⭐（错 = 送命）

> [!danger] 这 5 个陷阱是 USMLE 经典"杀手题"
> 每个都是反向 Anki 卡候选 — **看到触发词立即排除错误选项**。

### 致死 1 — **新生儿 + Ceftriaxone = 致死**

**触发词**：新生儿（<28d）/ 婴儿 / "premature infant" / "newborn"

**机制**：
- Ceftriaxone **displace bilirubin** from albumin → **核黄疸 (kernicterus)**
- Ceftriaxone + **含钙 IV 液**（Lactated Ringer / TPN）→ 沉淀 → 微栓塞 → **致死性肺栓塞**

**正确替代**：
- 新生儿 sepsis 经验：**Ampicillin + Cefotaxime / Gentamicin**
- 新生儿脑膜炎：**Ampicillin + Cefotaxime**（绝不用 ceftri）

**反射弧**：看到 "neonate / newborn / preterm" + "ceftriaxone" 选项 → **立即排除**

---

### 致死 2 — **Aminoglycoside + Loop diuretic = 耳聋**

**触发词**：Gentamicin / Tobramycin / Amikacin / Streptomycin + Furosemide / Bumetanide / Torsemide

**机制**：两者**独立耳毒性**叠加 → 不可逆感音神经性聋（尤其老年 / CKD / 长疗程）

**额外叠加**：Aminoglycoside + **Vanc** = 肾毒性叠加（不是耳毒）

**反射弧**：见 "elderly on furosemide" + "starting gentamicin" → **选监测听力 / 换药**

---

### 致死 3 — **FQ（氟喹诺酮）+ 特定人群 = 多种灾难**

**触发词**：Ciprofloxacin / Levofloxacin / Moxifloxacin

| 人群 | 灾难 |
|---|---|
| **儿童 < 18 岁** | 软骨损伤 / 关节病变（除非 anthrax / CF 等绝对必须）|
| **孕妇** | 软骨损伤（替代：Amox-Clav / Cephalexin）|
| **老人（特别 + 类固醇）** | **肌腱断裂**（跟腱最常见，可双侧）|
| **任何人** | **QT 延长 → torsades**（避免 + 其他 QT 延长药）|
| **任何人** | **大动脉夹层 / 主动脉瘤破裂**（FDA 警告）|
| **任何人** | **周围神经病**（不可逆）|
| **任何人** | **C. diff**（高风险）+ **2-7d 后腹泻**|

**反射弧**：见 "elderly + FQ + 跟腱痛" / "FQ + 突发胸背痛" → **立即停药 + 替代方案**

---

### 致死 4 — **TMP-SMX + ACEi/ARB/MRA = 高钾炸弹**

**触发词**：TMP-SMX / Bactrim + lisinopril / losartan / spironolactone / 老年 / CKD

**机制**：**TMP 阻断 ENaC**（类 amiloride 机制）→ 远端集合管不排 K → 与 ACEi/MRA 叠加 → 急性高钾（可 sine wave / 心搏停）

**额外**：TMP-SMX 也会引起：
- 高钾（上述）
- **AKI**（间质性肾炎 / 假性 Cr 升高 — TMP 阻肾小管分泌）
- **SJS / TEN**（磺胺过敏）
- **G6PD 缺乏症溶血**
- **Hyperkalemia + Hyponatremia**（罕见 SIADH 样）

**反射弧**：见"老年 / HFrEF / CKD + 已在 ACEi/spironolactone + 新开 TMP-SMX" → **K 监测 / 换药（如 doxy / cephalexin 替代）**

详见 [[完整笔记/Peixuan分科笔记/肾脏]] 高钾章节 + [[草稿/2026-05-17_高钾血症完整体系_v1]] §Trigger vs Permissive

---

### 致死 5 — **Vanc 输注过快 = Red Man Syndrome / 低血压**

**触发词**：Vancomycin 1 g IV 静推 / 30 min 内输注

**机制**：直接刺激 mast cell **非 IgE 介导**释放组胺（**不是真过敏**）→ 红斑 / 瘙痒 / 低血压

**正确做法**：
- **≥1 g 必须 ≥1 h 慢推**（每 500 mg 至少 30 min）
- 发生时：**慢推 + 苯海拉明** —— **不必停药 / 不是过敏**
- 如真过敏（IgE，皮疹 + bronchospasm + anaphylaxis）→ 换药（Linezolid / Dapto / Ceftaroline）

**额外 vanc 副作用**：**Nephrotoxicity**（特别 + aminoglycoside / 高谷浓度 / CKD）；耳毒性较少

**反射弧**：见"vanc 输注 20 min 后红斑 / 低血压"→ **不是过敏，慢推 + 抗组胺**

---

### 致死陷阱补充（次频但仍要会）

| 陷阱 | 触发词 | 灾难 |
|---|---|---|
| **Daptomycin 治肺炎** | "dapto for pneumonia" | **肺表面活性剂灭活 dapto → 治疗失败**（dapto 不能用于肺炎）|
| **Linezolid + SSRI / MAOI** | 抑郁 + linezolid | **5-HT 综合征**（linezolid 是弱 MAO-I） |
| **Cefepime + CKD 不调量** | 老年 CKD + cefepime | **脑病 / 肌阵挛 / non-convulsive SE** |
| **Cefotetan + 酒** | 治疗期 + 酒 | **双硫仑反应 + ↓ 维 K → 出血**（NMTT 侧链）|
| **Metronidazole + 酒** | 治疗期 + 72h 内 | **双硫仑反应**（潮红 / 头痛 / 呕吐 / 心悸） |
| **Tetracycline / Doxy + 孕妇 / <8 岁儿童** | — | **牙黄染 + 骨发育抑制**（doxy 短疗程 <21d 风险较低）|
| **Aminoglycoside + 孕妇** | — | **胎儿耳毒性**（D 类） |
| **Carbapenem + Valproate** | 癫痫患者 + meropenem | **VPA 浓度急降 → 癫痫发作** |
| **INH 单用 + 不加 B6** | TB 治疗 | **周围神经病**（INH ↓ B6 → 必须加吡哆醇）|

## 四、修改条件触发词速查表 ⭐

> [!tip] 这张表帮你识别 "标准方案 → 必须改" 的触发词
> 看到这些词，**默认方案不再适用**，必须替换。

| 触发词 | 必须避免 | 替代方案 |
|---|---|---|
| **PCN allergic — 轻度皮疹 / 不明确** | 严重 PCN 过敏避所有 β-lactam | 轻度可用头孢（2-5 代尤其安全）/ aztreonam |
| **PCN allergic — anaphylaxis / SJS / TEN** | 所有 PCN + 多数头孢 | **Vanc** / **Linezolid** / **FQ** / **Macrolide** / **Doxy** / **Clinda** / **Aztreonam**（仅 G⁻） |
| **Pregnancy** | **FQ** / **Doxy/Tet** / **TMP-SMX**（孕早 + 孕末）/ **Aminoglycoside** / **Metronidazole 孕早期** | **β-lactam（PCN / Cephalo）**安全 / **Erythromycin / Azithro / Clinda**（Macrolide 安全）/ **Nitrofurantoin（孕中期）** |
| **Lactation** | **FQ** / **Tet** / **Sulfas**（G6PD / 黄疸风险）| β-lactam / Macrolide 安全 |
| **<18 岁 / 儿童** | **FQ**（软骨）/ **Tet / Doxy <8 岁**（牙）| β-lactam / Macrolide |
| **Neonate (<28d)** | **Ceftriaxone**（核黄疸 + Ca 沉淀）/ **Sulfas** | **Cefotaxime** / Amp / Gent |
| **CKD / Renal failure** | 多数抗生素需调剂量 | 调剂量；**cefepime / aminoglycoside / vanc** 尤其需监测；**Nitrofurantoin CrCl <30 禁用** |
| **Hepatic failure** | Macrolide / FQ / Linezolid / RIPE 部分 | 调剂量 / 避免 |
| **G6PD 缺乏症** | **TMP-SMX / Sulfas** / **Nitrofurantoin** / **Primaquine** / **Dapsone** | 替代品种 |
| **Elderly + steroid** | **FQ**（肌腱断裂）| Macrolide / β-lactam |
| **Heart failure / 长 QT** | **FQ / Macrolide**（QT 延长）| β-lactam |
| **On warfarin** | **TMP-SMX / Metronidazole / Macrolide**（CYP 抑制 → INR ↑↑）| 监测 INR / 换 β-lactam |
| **On methotrexate** | **TMP-SMX**（↓ MTX 排泄 → 骨髓抑制致命）| β-lactam |
| **On SSRI / MAOI** | **Linezolid**（5-HT 综合征）| Vanc / Dapto |

## 五、特殊场景小补丁（非主表 20 但常见）

| 场景 | 首选 |
|---|---|
| **MSSA 菌血症** | **Nafcillin / Cefazolin** IV（不要 vanc — vanc 治 MSSA 不如 nafcillin）|
| **Strep pyogenes 咽炎 / 蜂窝织炎** | **PCN V / Amoxicillin** PO（PCN 过敏 → cephalexin / macrolide）|
| **GBS 产前预防**（GBS ⊕ + 临产）| **PCN G** IV（PCN 过敏 → cefazolin / clinda / vanc）|
| **GAS necrotizing fasciitis** | **PCN + Clindamycin**（clinda 抑毒素合成 → "Eagle effect"）|
| **梅毒（任何期）** | **PCN G**（早期 1 剂 IM，late 3 剂周间）/ 孕妇 PCN 过敏 → **脱敏后 PCN**（doxy 致畸 + 不替代）|
| **百日咳（治疗 + 暴露后预防 PEP）** | **Azithromycin**；**PEP 给所有密切接触者，不论疫苗接种史**（<1 月龄首选 azithro；高危有限暴露者——孕妇/婴儿/免疫缺陷——也给）|
| **动物咬伤（猫/狗/人）预防 + 治疗** | **Amox-Clav**（覆盖 *Pasteurella multocida* + 口腔厌氧 + 皮肤菌）；**猫咬深穿刺一律预防**；clinda 不覆盖 Pasteurella 不可单用 |
| **急性细菌性颈淋巴结炎（acute bacterial cervical lymphadenitis）** ⭐ | **克林霉素（clindamycin）**（覆 *S. aureus*含MRSA + GAS）；**单侧·急性·红肿热痛·可波动化脓** → 金葡/化脓链球菌；⚠️**绝不 azithromycin** |
| **猫抓病（cat-scratch disease，*Bartonella henselae* 巴尔通体）** | **阿奇霉素（azithromycin）**；**亚急性·无痛或轻痛·区域淋巴结肿大 + 猫(尤幼猫)抓咬史 + 接种处丘疹** → 多自限，azithro 仅缩短病程 |
| **无脾 / 脾切除 + 发热脓毒症（OPSI 暴发性）** ⭐ | **Ceftriaxone（头孢曲松）**（± **vancomycin 万古霉素** 覆耐药肺炎链球菌）；荚膜菌第一 = **肺炎链球菌（S. pneumoniae）** > Hib > 脑膜炎球菌；**先抽血培养立即打药别等**（OPSI 数小时暴发）；⚠️**无骨痛=脓毒症（荚膜菌）；有骨痛=骨髓炎（金葡 + Salmonella 双覆盖）**——别因"SCD=Salmonella"反射漏金葡。详见 [[完整笔记/专题笔记/血液/血液_镰刀细胞病SCD]] |
| **白喉** | **Erythromycin** + 抗毒素 |
| **Anthrax**（暴露后预防 / 治疗）| **Cipro / Doxy**（儿童特例可用 FQ）|
| **HSV 脑炎** | **Acyclovir IV**（其实是抗病毒，但常出现在脑膜炎经验治疗题里）|
| **Cryptococcal 脑膜炎** | **Amphotericin B + Flucytosine**（诱导）→ Fluconazole（巩固）|
| **PJP（卡氏肺孢菌肺炎）** | **TMP-SMX**（高剂量）+ Steroid（PaO₂ <70 / A-a >35）|
| **Trichomoniasis** | **Metronidazole** 单剂 PO（伴侣同治）|
| **BV (bacterial vaginosis)** | **Metronidazole** PO / 阴道用药 |
| **Helicobacter pylori** | **三联 / 四联**（PPI + Amox + Clarithro / PPI + Bismuth + Tet + Metro）|
| **Acute otitis media** | **Amoxicillin 高剂量**（PCN 过敏 → cephalosporin / macrolide）|
| **Strep throat** | **PCN V / Amoxicillin** |
| **Rocky Mountain Spotted Fever** | **Doxycycline**（包括儿童 — 致畸风险 < 不治死风险）|
| **Endometritis 产后** | **Clinda + Gent**（覆盖厌氧 + G⁻）|
| **单纯膀胱炎 (cystitis, 下尿路)** | **呋喃妥因 (nitrofurantoin) / TMP-SMX / 磷霉素 (fosfomycin)**（**仅下尿路** — 肾实质浓度不足）|
| **肾盂肾炎 (pyelonephritis, 上尿路)** ⭐ | 门诊口服 **氟喹诺酮（环丙沙星 ciprofloxacin）/ TMP-SMX**；重症 IV **三代头孢（头孢曲松 ceftriaxone）**；**呋喃妥因不行**（无肾实质穿透）；**已有听力↓ → 避氨基糖苷耳毒性** |

> [!danger] 肾盂肾炎选药 = 部位穿透 + 患者毒性规避（NBME11 Q145）
> 培养给"多药敏感"时别只挑敏感药，要同时过三关：① **感染部位**（肾盂肾炎需肾实质浓度 → 排除呋喃妥因 nitrofurantoin）；② **患者个体毒性**（**已有听力下降 → 避开氨基糖苷 aminoglycoside 耳毒性**，呼应"致死 2：氨基糖苷 + 袢利尿剂 = 耳聋"）；③ **过敏史**（磺胺 sulfa 过敏 → 避 TMP-SMX）。三关筛完 → **环丙沙星 (ciprofloxacin)**。

> [!tip] 为什么明知 72 岁老人有 FQ 肌腱风险，还是选环丙沙星？= 相对禁忌 vs 绝对禁忌
> 这是这道题最反直觉的点：FQ（氟喹诺酮 fluoroquinolone）肌腱断裂 (tendon rupture) 在老人（尤其 + 类固醇 corticosteroid）确实是真风险（见"致死 3"），但它是**相对禁忌 / 需告知监测 (relative caution)**，不是绝对禁忌。
> - **绝对枪毙**：磺胺过敏 → TMP-SMX 出局；氨基糖苷耳毒性 + 已有听力↓ → 庆大/阿米卡星/妥布出局。
> - **疗效枪毙**：呋喃妥因治不了上尿路（给了≈没治）。
> - **相对风险**：cipro 肌腱风险 → 知情 + 避免合用激素 + 嘱跟腱痛立即停药即可，**不能因此否掉唯一有效药**。
> - **权衡天平**：肾盂肾炎是会进展成脓毒症 (sepsis) / 菌血症 (bacteremia) 的严重感染，"治不好/出大事" > "可能肌腱痛"。当"有效且无绝对禁忌"的选项只剩一个时，相对风险不优先。
> - **旋钮**：若 stem 改成"该老人正在吃泼尼松 prednisone"，cipro 肌腱风险权重被抬高，答案可能重新掂量——这是出题人调权重的常用手法。
> 通用反射：**绝对禁忌（过敏/已有该药靶器官损伤）和疗效不足 = 直接出局；相对风险 = 知情监测、不否决唯一有效药。**

## 五-补、青霉素家族分类 + β-内酰胺过敏/孕期选药逻辑 ⭐

> 触发：Q3256（梅毒 + 青霉素过敏 → 多西环素）+ Q3257（孕妇 Lyme → 阿莫西林）暴露"阿莫西林 vs 多西环素能不能互换"困惑。

### 5补.1 青霉素家族（penicillins）= 都是 β-内酰胺，共享交叉过敏

| 亚类 | 代表药 | 主要用途 |
|---|---|---|
| **天然青霉素 natural** | **青霉素 G（penicillin G，IM/IV）**、青霉素 V（PO） | **梅毒**、A 组链球菌（GAS）、 |
| **氨基青霉素 amino** | **阿莫西林（amoxicillin）**、氨苄西林（ampicillin） | **莱姆（Lyme）**、李斯特（Listeria）、肠球菌、中耳炎、Hp |
| **抗葡萄球菌 antistaph** | 萘夫西林（nafcillin）、双氯西林（dicloxacillin） | MSSA |
| **抗假单胞 antipseudomonal** | 哌拉西林（piperacillin） | 铜绿假单胞菌 |

> [!warning] 关键
> **阿莫西林就是青霉素类（氨基青霉素）** → **青霉素过敏者对阿莫西林也过敏（交叉）**。所以"青霉素过敏"时，**整个 β-内酰胺类（含阿莫西林）都要避开** → 逃向**不同类**（四环素/大环内酯等）。

### 5补.2 阿莫西林 vs 多西环素：不是通用替代，按"病 + 禁忌"选 ⭐⭐

> [!danger] 两者属不同类，能否互换**完全取决于该病原对谁敏感**
> | | 病原对什么敏感 | 约束 | 选谁 | 为什么 |
> |---|---|---|---|---|
> | **莱姆（Lyme，早期 EM）** | doxy / **阿莫西林** / 头孢呋辛 **都行** | **孕妇/儿童<8 → 禁四环素** | **阿莫西林** | Borrelia 对氨基青霉素也敏感；阿莫西林孕期安全 |
> | **梅毒（syphilis）** | **几乎只认青霉素 G**；过敏者退而求其次 doxy | **青霉素过敏（含阿莫西林）** | **多西环素** | 跳出整个 β-内酰胺类；阿莫西林既过敏又非梅毒循证方案 |
>
> **镜像逻辑**：
> - **孕妇** → 禁四环素（doxy）→ 被推向 **β-内酰胺（阿莫西林）**
> - **青霉素过敏** → 禁 β-内酰胺（含阿莫西林）→ 被推向 **多西环素（四环素）**
> - 两个约束相反 → 选出的药相反；**但前提是该病对这两类都（或分别）有效**。

### 5补.3 为什么梅毒"同是青霉素却只用 penicillin G、不用阿莫西林"

- 苍白密螺旋体（Treponema pallidum）需**长效低浓度维持数周** → 只有**苄星青霉素 G（benzathine penicillin G，IM 长效储库）** 能稳定做到。
- 阿莫西林虽同类，但**无梅毒循证方案**（口服浓度/依从不可靠）→ 不用。**不是因为它不是青霉素**。

> [!danger] ⚠️ 高频陷阱：孕妇 + 梅毒 + 青霉素过敏 → **必须脱敏（desensitization）后打青霉素**
> 孕妇禁多西环素（致畸）+ 阿莫西林对梅毒不管用且过敏 + 大环内酯耐药 → **唯一正确 = 青霉素脱敏**。别因"青霉素过敏"就反射多西环素而忘了**孕妇连 doxy 也不能用**。

| 猫咬深穿刺 | **Amox-Clav**（Pasteurella；一律预防，不是"观察"）|
| **急性红肿热痛颈淋巴结**（化脓性）| **Clindamycin**（金葡/GAS；**不是 azithro**）|
| **猫抓 + 无痛/亚急性区域淋巴结**（Bartonella）| **Azithromycin** |
| 百日咳家庭接触者 | **Azithro PEP**（不论疫苗史，全给）|

> [!danger] 反复错陷阱 — 颈淋巴结炎 ≠ 猫抓病（你已栽 3 次）
> **看见颈部淋巴结肿大别条件反射猫抓病/azithromycin。先分急慢 + 痛不痛：**
> - **急性 + 红肿热痛 + 化脓波动 = 细菌性淋巴结炎（金葡/GAS）→ clindamycin**（覆 MRSA + GAS）。
> - **亚急性 + 无痛或轻痛 + 猫(幼猫)抓咬史 + 接种处丘疹 = 猫抓病（Bartonella）→ azithromycin**。
> **azithromycin 只属猫抓病那一栏**；化脓性急性淋巴结炎用 azithro = 漏覆金葡。错题：[[mistakes/uworld-mistakes_2026-06#^Q4843]] / [[mistakes/uworld-mistakes_2026-06#^Q4843-R0609]]（第 3 次又选 azithromycin）。

> [!tip] 用药原理 — 为什么是这两个药（病原在哪 → 谁能打到那里）
> **两个药同结合 50S 亚基抑制蛋白合成，但谱 + 穿透完全不同 → 不可互换。**
> 
> | | 急性细菌性颈淋巴结炎 | 猫抓病 |
> |---|---|---|
> | **病原** | 金葡(含CA-MRSA) + GAS | *Bartonella henselae*（**胞内**苛养 G⁻）|
> | **选药** | **Clindamycin（克林霉素）** | **Azithromycin（阿奇霉素）** |
> | **药理关键** | 覆 **G+ 含 MRSA + 厌氧 + 抑外毒素**（Eagle effect）| **脂溶→巨噬细胞内浓集 10–100×**，打躲在细胞里的菌 |
> | **为什么不互换** | 大环内酯**漏覆金葡/完全不覆 MRSA**（耐药率高）| β-内酰胺/clinda **穿透差、打不到胞内苛养菌** |
> 
> - **Clindamycin = "G+ 含 MRSA 的实心拳头"** → 体表化脓金葡/链球菌。
> - **Azithromycin = "钻进细胞的渗透弹"** → 胞内 Bartonella（同理 Legionella/Mycoplasma/Chlamydia 等胞内/无壁菌都靠大环内酯胞内穿透）。

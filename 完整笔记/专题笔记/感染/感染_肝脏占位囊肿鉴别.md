# 肝脏占位/囊肿鉴别

> [!info] 笔记定位
> 肝脏"占位/囊肿"题的解法 = **先分囊性 vs 实性 → 看宿主/暴露/影像钥匙 → 定病因**。重点是**感染性（阿米巴 vs 化脓 vs 包虫）**三角，附实性占位速览。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q2968]]（阿米巴）+ [[mistakes/uworld-mistakes_2026-06#^Q17223]]（化脓性）+ [[mistakes/uworld-mistakes_2026-06#^Q2970]]（包虫）。

---

## 一、一图总览 ⭐⭐⭐

```
肝脏病灶
├─ 脓肿（发热+RUQ痛+ALP↑）
│   ├─ 阿米巴   年轻+发展中国家旅行+腹泻史+右叶单发+嗜酸不高     → metronidazole+paromomycin
│   ├─ 化脓性   老年/中毒貌/合并症(DM)+胆道或GI源+高热白细胞↑↑   → 引流+IV抗生素
│   └─ 真菌微脓肿 中性粒缺/化疗恢复期+肝脾多发小"靶征"          → fluconazole/echinocandin
├─ 囊肿（真性）
│   ├─ 包虫 granulosus  犬/牧羊区+蛋壳钙化+子囊+嗜酸↑+IgG⊕   → albendazole±PAIR，别穿刺
│   ├─ 包虫 multilocularis 泡型=多房+侵袭性+似恶性
│   └─ 单纯囊肿(先天)   无症状+无钙化+薄壁无分隔(anechoic)     → 不处理
├─ 胆道/吸虫  Fasciola(实质隧道+嗜酸↑) / Clonorchis(胆管炎→胆管癌)
└─ 实性占位（见§五）海绵状血管瘤/FNH/腺瘤/HCC/转移
```

## 二、感染性：脓肿 + 囊肿三角 ⭐⭐⭐

| | **阿米巴脓肿** | **化脓性脓肿（pyogenic）** | **包虫囊肿（Echinococcus）** |
|---|---|---|---|
| **病原** | **Entamoeba histolytica（原虫）** | **多微生物**（E.coli/Klebsiella/厌氧/链球菌）| **E. granulosus（犬绦虫）** |
| **宿主/暴露** | 年轻 + **发展中国家旅行 + 腹泻史** | **老年/中毒貌/合并症（DM）** | **犬接触 + 牧羊区**（南美/中东）|
| **来源** | 结肠 → **门脉 → 肝**（右叶单发）| **胆道（梗阻/胆管炎）最常见** / 门脉(阑尾炎/憩室) / 血行 | 吞虫卵 |
| **影像** | **右叶单发低回声** | 单/多发；可分隔积气 | **蛋壳/弧形钙化 + 子囊/内分隔** |
| **嗜酸** | **不高**（原虫）| 不高 | **↑**（蠕虫）|
| **诊断** | 粪 PCR/抗原 + **血清学⊕**（孤立脓肿）| **血培养 + 脓液培养** | 影像 + **IgG 血清学** |
| **治疗** | **Metronidazole/tinidazole + 腔内 paromomycin**（清肠道包囊）| **经皮引流 + IV 抗生素** | **Albendazole ±PAIR(>5cm)**；破裂→手术；**禁穿刺活检** |

> [!warning] 共同表现 + 并发
> 三者都可 **RUQ 痛 + 肝大 + ALP↑ > 转氨酶**；**肝顶病灶 → 膈刺激 → 右侧反应性胸腔积液**（呼吸音↓）。

## 三、阿米巴 vs 化脓性（最高频对子）⭐

> [!tip] 看宿主 + 旅行/腹泻史分
> | | 阿米巴 | 化脓性 |
> |---|---|---|
> | 宿主 | **年轻、不显中毒貌** | **老年/中毒貌/合并症（DM）** |
> | 钩子 | **发展中国家旅行 + 腹泻史** | **胆道/GI 感染源**（胆管炎/阑尾炎/憩室）|
> | 病灶 | **右叶单发** | 单/多发 |
> | 嗜酸 | 不高 | 不高 |
> | 抽吸 | "**anchovy paste**"（无菌、无中性）| 脓性、培养⊕ |
>
> 错题：[[mistakes/uworld-mistakes_2026-06#^Q2968]]（旅行+腹泻→阿米巴，误选化脓）/ [[mistakes/uworld-mistakes_2026-06#^Q17223]]（胆道支架→化脓，首选腹部 CT）。

## 四、囊性病变

> [!info] 包虫囊肿（Echinococcus granulosus）→ [[mistakes/uworld-mistakes_2026-06#^Q2970]]
> 犬绦虫、羊中间宿主；人吞卵偶然感染。**蛋壳钙化 + 子囊** = 高度提示；**嗜酸↑ + IgG⊕**。治 **albendazole ± 经皮 PAIR（>5cm/分隔）**；破裂→手术。
> **⚠️ 禁贸然穿刺活检 → 囊液外溢致过敏性休克（anaphylaxis）。**
> **E. multilocularis（泡型）** = **多房、侵袭性、似恶性**；granulosus = 单房。

> [!info] 单纯肝囊肿（先天）
> **无症状、薄壁、无分隔、无钙化、anechoic**；偶然发现 → 随访，不处理。

> [!info] 真菌微脓肿 = 肝脾念珠菌病（hepatosplenic candidiasis）⭐
> **念珠菌（Candida）经门脉播散**；典型宿主 = **血液恶性肿瘤/化疗后中性粒细胞缺乏"恢复期"**（中性粒回升时显现）。影像 **肝脾多发小病灶（"牛眼/靶征 bull's-eye"）**；持续发热 + ALP↑。诊断活检/培养（血培养常阴）。治 **氟康唑（fluconazole）/ 棘白菌素（echinocandin）**。

> [!info] 肝吸虫（多在胆道，非实质真囊）
> - **肝片吸虫（Fasciola hepatica）**：实质"隧道"样病灶 + **嗜酸↑**（生水芹/污染水）。
> - **华支睾/后睾吸虫（Clonorchis / Opisthorchis）**：**胆道**（胆管炎）→ **胆管癌（cholangiocarcinoma）**；生淡水鱼。

## 五、实性占位速览（非感染，鉴别完整性）

| 病 | 钥匙 |
|---|---|
| **海绵状血管瘤** | **最常见良性肝肿瘤**；女性；增强**周边结节状不连续强化 → 向心填充** |
| **局灶结节增生 FNH** | 年轻女；**中央星状瘢痕**；良性不处理 |
| **肝腺瘤** | **口服避孕药/合成类固醇/妊娠**；**破裂出血 + 恶变**风险 → 停 OCP，>5cm 切 |
| **HCC（肝细胞癌）** | **肝硬化（HBV/HCV/酒精/NASH）**；**AFP↑**；动脉期强化 + 门脉期廓清 |
| **转移瘤** | **最常见恶性肝病灶**（GI/乳腺/肺）；多发 |

## 六、关键钥匙（速记）

- **嗜酸 ↑ = 蠕虫（包虫）；嗜酸不高 = 原虫（阿米巴）/ 细菌（化脓）。**
- **蛋壳/弧形钙化 = 包虫**（且**别穿刺**）。
- **旅行 + 腹泻 + 右叶单发 = 阿米巴**；**老年/中毒貌/胆道源 = 化脓**。
- **肝硬化 + AFP↑ + 动脉强化廓清 = HCC**；**OCP + 出血 = 腺瘤**；**中央瘢痕 = FNH**。
- 肝脓肿/囊肿 **ALP↑ > 转氨酶**；肝顶灶 → **右胸积液**。

## 七、Memory Hook

```
脓肿：阿米巴(旅行+腹泻+右叶单发+嗜酸低→metro+paromomycin) vs 化脓(老年/中毒/胆道源→引流+抗生素)
囊肿：包虫(蛋壳钙化+嗜酸↑+IgG，禁穿刺→albendazole±PAIR) vs 单纯(无钙化薄壁→随访)
实性：血管瘤(向心填充) / FNH(中央瘢痕) / 腺瘤(OCP+出血) / HCC(肝硬化+AFP) / 转移(最常见恶性)
```

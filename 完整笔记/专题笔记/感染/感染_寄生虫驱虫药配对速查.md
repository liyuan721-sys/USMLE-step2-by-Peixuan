# 寄生虫 & 驱虫药配对速查

> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q3612]]（蛲虫 → 噻嘧啶/阿苯达唑，被湖游史钩向血吸虫）。
> 用法：vignette 给寄生虫感染场景让你选"最合适治疗"时，先**定虫的大类（线虫/吸虫/绦虫/原虫/体外）→ 再选药**。
> 覆盖角度：§一–§四 = 药↔虫配对 + 用药机制（治疗导向）；§五 = 现有药表**外**的高频虫种（识别导向）；§六 = 嗜酸/诊断方法/系统并发症（"这是什么虫 + 会损伤什么器官"导向）。
> 版权：基于公开教科书概念（First Aid / CDC 公开摘要）重新整理，未复制 UWorld 题干。

## 一、一张总表（药 ↔ 虫）⭐

> [!tip] 三句话先记住主干
> - **肠道线虫（蛲/蛔/钩/鞭）** → **阿苯达唑 / 甲苯达唑 / 噻嘧啶**（"-bendazole" 三兄弟 + 噻嘧啶）
> - **吸虫 + 绦虫（成虫）** → **吡喹酮（praziquantel）**
> - **原虫（阿米巴/贾第/滴虫）** → **甲硝唑（metronidazole）**

| 药 | 中文 | 主治 |
|---|---|---|
| albendazole / mebendazole | 阿苯达唑 / 甲苯达唑 | **肠道线虫**：蛲虫、蛔虫、钩虫、鞭虫；**囊虫病（cysticercosis）+ 包虫病（echinococcus）** |
| pyrantel pamoate | 双羟萘酸噻嘧啶 | 蛲虫、蛔虫、钩虫（孕妇相对安全选择） |
| ivermectin | 伊维菌素 | **粪类圆线虫病（strongyloidiasis）、盘尾丝虫病（onchocerciasis 河盲）**；疥疮/虱备选 |
| diethylcarbamazine (DEC) | 乙胺嗪 | **淋巴丝虫病（Wuchereria）、罗阿丝虫（Loa loa）** |
| praziquantel | 吡喹酮 | **血吸虫病（schistosomiasis）、吸虫（肝/肺吸虫）、绦虫成虫** |
| metronidazole / tinidazole | 甲硝唑 / 替硝唑 | **原虫**：阿米巴、贾第虫（Giardia）、滴虫（Trichomonas） |
| nitazoxanide | 硝唑尼特 | 隐孢子虫（Cryptosporidium）、贾第虫 |
| pyrimethamine + sulfadiazine | 乙胺嘧啶 + 磺胺嘧啶 | 弓形虫病（toxoplasmosis） |
| atovaquone + azithromycin | 阿托伐醌 + 阿奇霉素 | **巴贝虫病（babesiosis）** |
| 氯喹 / 青蒿素类 ± 伯氨喹 | chloroquine / artemisinin ± primaquine | **疟疾（malaria）**；伯氨喹清肝内休眠子（P. vivax/ovale） |
| permethrin | 氯菊酯 | **疥疮（scabies）、虱（lice）** |

## 二、按虫的大类定药（决策主干）

```
寄生虫
├─ 蠕虫 Helminths（组织移行 → 嗜酸↑）
│   ├─ 线虫 Nematodes（圆虫）
│   │   ├─ 肠道：蛲/蛔/钩/鞭 ── 阿苯达唑·甲苯达唑·噻嘧啶
│   │   ├─ 组织：粪类圆·盘尾丝虫 ── 伊维菌素
│   │   ├─ 淋巴丝虫·Loa loa ── 乙胺嗪 DEC
│   │   └─ 幼虫移行：皮肤 CLM(犬钩虫)/内脏 VLM(Toxocara) ── 伊维菌素·阿苯达唑
│   ├─ 吸虫 Trematodes（血吸/肝/肺吸虫）── 吡喹酮
│   └─ 绦虫 Cestodes
│       ├─ 成虫（肠道带绦虫）── 吡喹酮
│       └─ 幼虫：囊虫·包虫 ── 阿苯达唑（± 手术）
└─ 原虫 Protozoa（一般不致嗜酸↑）
    ├─ 肠道：阿米巴·贾第·隐孢子 ── 甲硝唑 / 硝唑尼特
    ├─ 泌尿生殖：滴虫 ── 甲硝唑（性伴同治）
    ├─ 血液：疟原虫·巴贝虫 ── 氯喹/青蒿素 · 阿托伐醌+阿奇
    ├─ 组织：弓形虫 ── 乙胺嘧啶+磺胺嘧啶；利什曼 ── 两性霉素B/锑剂
    ├─ 鞭毛虫(虫媒)：锥虫(Chagas/非洲睡眠病) ── 苄硝唑/喷他脒等
    └─ 自由生活阿米巴：Naegleria(鼻→脑)·Acanthamoeba(隐形眼镜角膜炎) ── 两性霉素B / PHMB+氯己定
```

### 线虫 Nematodes 招牌 → 药

| 虫 | 招牌 vignette | 首选 |
|---|---|---|
| **蛲虫 Enterobius（pinworm）** | **夜间肛周瘙痒**，学龄儿童；透明胶带试验 | **噻嘧啶 pyrantel / 阿苯达唑 albendazole**；全家同治、2 周后重复 |
| 蛔虫 Ascaris | 粪口；可肠梗阻/胆道/Löffler 肺浸润 | 阿苯达唑 albendazole / 甲苯达唑 mebendazole |
| 钩虫 Hookworm | 赤脚接触土壤；**缺铁性贫血** | 阿苯达唑 albendazole / 甲苯达唑 mebendazole |
| 鞭虫 Trichuris | **直肠脱垂**（重度儿童） | 阿苯达唑 albendazole / 甲苯达唑 mebendazole |
| **粪类圆线虫 Strongyloides** | 自体感染；免疫抑制 → **播散/高度感染**；荨麻疹+腹痛+咳喘 | **伊维菌素 ivermectin** |
| 盘尾丝虫 Onchocerca | 黑蝇；**河盲（river blindness）**+ 皮肤结节 | **伊维菌素 ivermectin** |
| 旋毛虫 Trichinella | 生猪肉；**肌痛 + 眶周水肿 + 嗜酸↑** | 阿苯达唑 albendazole（± 激素） |

### 吸虫 / 绦虫 → 吡喹酮（幼虫例外）

| 虫 | 招牌 | 首选 |
|---|---|---|
| **血吸虫 Schistosoma** | **淡水游泳（swimmer's itch）**→ 后期血尿（埃及血吸虫）/ 肝脾门脉高压（曼氏） | **吡喹酮 praziquantel** |
| 肝/肺吸虫 Clonorchis/Paragonimus | 生鱼/生蟹 | 吡喹酮 praziquantel |
| 带绦虫 Taenia（成虫） | 生牛/猪肉 | 吡喹酮 praziquantel |
| **囊虫病 Neurocysticercosis** | 猪带绦虫卵 → **脑囊肿 + 癫痫** | **阿苯达唑 albendazole**（± 激素/抗癫痫） |
| 包虫病 Echinococcus | 犬/羊；**肝包虫囊（蛋壳钙化）** | **阿苯达唑 albendazole** + 手术/PAIR（勿穿破致过敏） |

### 原虫 Protozoa

| 虫 | 招牌 | 首选 |
|---|---|---|
| 阿米巴 Entamoeba | 旅行；血性腹泻 + **肝脓肿（RUQ 痛）** | **甲硝唑 metronidazole** + 巴龙霉素 paromomycin（清肠腔包囊） |
| 贾第虫 Giardia | 野营/溪水；**脂肪泻 + 胀气**，无血 | **甲硝唑 metronidazole**（或硝唑尼特 nitazoxanide） |
| 滴虫 Trichomonas | **黄绿泡沫白带 + 草莓样宫颈** | **甲硝唑 metronidazole**（**性伴同治**） |
| 隐孢子虫 Cryptosporidium | **AIDS（CD4<100）水样泻** | 硝唑尼特 nitazoxanide（核心是恢复免疫/ART） |
| 弓形虫 Toxoplasma | **AIDS 环形强化脑病灶**；先天 TORCH | 乙胺嘧啶 pyrimethamine + 磺胺嘧啶 sulfadiazine（+ 亚叶酸 leucovorin） |
| 疟原虫 Plasmodium | 旅行 + 周期热 + 溶血 | 氯喹 chloroquine/青蒿素类 artemisinin；**伯氨喹 primaquine 清 vivax/ovale 肝休眠子** |
| **巴贝虫 Babesia** | **无脾 + 东北蜱 + 血管内溶血**；血涂片 **Maltese cross** | **阿托伐醌 atovaquone + 阿奇霉素 azithromycin** |

## 三、高频干扰项 & buzzword（最容易错的地方）⭐

> [!warning] 干扰项黑名单
> - **吡喹酮 ≠ 蛲虫**：吡喹酮治**血吸虫/吸虫/绦虫成虫**；"淡水游泳"把你往血吸虫（吡喹酮）带，但**夜间肛周痒是蛲虫**（噻嘧啶/阿苯达唑）。← [[mistakes/uworld-mistakes_2026-06#^Q3612]] 错点
> - **囊虫/包虫（幼虫）用阿苯达唑，不是吡喹酮**：脑囊虫癫痫、肝包虫囊 → 阿苯达唑（吡喹酮治的是肠道**成虫**绦虫）。
> - **克霉唑（clotrimazole）/ 制霉菌素是抗真菌**（念珠菌），**不是驱虫药**——出现在选项里是纯干扰。
> - **甲硝唑只打原虫（阿米巴/贾第/滴虫）+ 厌氧菌**，对线虫/吸虫无效。
> - **疥疮 vs 蛲虫都"夜间痒"**：疥疮在**指缝/腕/隧道 burrows**（permethrin 氯菊酯）；蛲虫在**肛周**（噻嘧啶/阿苯达唑）。

> [!tip] 招牌 → 立刻反射
> | Buzzword | 虫 | 药 |
> |---|---|---|
> | 夜间**肛周**痒 + 学龄儿 | 蛲虫 Enterobius | 噻嘧啶 pyrantel/阿苯达唑 albendazole |
> | **淡水游泳** + 血尿/肝脾 | 血吸虫 Schistosoma | 吡喹酮 praziquantel |
> | **河盲** + 皮下结节 + 黑蝇 | 盘尾丝虫 Onchocerca | 伊维菌素 ivermectin |
> | 免疫抑制 + **播散性高度感染** | 粪类圆线虫 Strongyloides | 伊维菌素 ivermectin |
> | **脑环形强化**（AIDS） | 弓形虫 Toxoplasma | 乙胺嘧啶 pyrimethamine+磺胺嘧啶 sulfadiazine |
> | **无脾 + 东北蜱 + 溶血 + Maltese cross** | 巴贝虫 Babesia | 阿托伐醌 atovaquone+阿奇霉素 azithromycin |
> | **草莓样宫颈 + 泡沫白带** | 滴虫 Trichomonas | 甲硝唑 metronidazole（性伴同治） |
> | **赤脚 + 缺铁性贫血** | 钩虫 Hookworm | 阿苯达唑 albendazole/甲苯达唑 mebendazole |

## 四、几个易考机制点

- **阿苯达唑/甲苯达唑（albendazole/mebendazole）**：抑制微管（β-tubulin）→ 虫不能摄取葡萄糖。孕期慎用（致畸，尤其早孕）→ 孕妇肠道线虫可考虑**噻嘧啶（pyrantel）**或推迟治疗。
- **吡喹酮（praziquantel）**：增加虫体膜对 Ca²⁺ 通透 → 痉挛麻痹。**脑囊虫先评估眼/脑负荷 + 备激素**（杀虫致炎症反应）。
- **伊维菌素（ivermectin）**：开虫体谷氨酸门控 Cl⁻ 通道。**Loa loa 高负荷者禁伊维菌素**（脑病风险），改 DEC（乙胺嗪 diethylcarbamazine）前需先评估微丝蚴血症。
- **甲硝唑（metronidazole）**：双硫仑样反应（disulfiram-like）→ 用药期间**禁酒**。

## 五、现有药表外的高频虫种（识别导向）⭐

> [!info] 为什么单列
> 这些虫 §一药表没收，但 Step 2 常考——它们的治疗**跳出"-bendazole / 吡喹酮 / 甲硝唑"三主干**（多用两性霉素 B、锑剂、苄硝唑、内镜取虫），靠"招牌 vignette"直接秒杀。

### 5.1 自由生活阿米巴（free-living amebae）——非粪口、来自水/隐形眼镜

| 虫（中英） | 招牌 vignette | 诊断 | 治疗 |
|---|---|---|---|
| **福氏耐格里 Naegleria fowleri** | **温暖淡水（湖/温泉/未氯化泳池）游泳，水经鼻入**→数日内暴发 **PAM（primary amebic meningoencephalitis，原发性阿米巴脑膜脑炎）**；死亡率 >97% | 脑脊液（CSF, cerebrospinal fluid）**似细菌性化脓**（PMN（polymorphonuclear，多形核中性粒细胞）↑/**糖↓**/蛋白↑），湿片见**活动滋养体** | **两性霉素 B（amphotericin B）** IV+鞘内 ± **米替福新（miltefosine）** |
| **棘阿米巴 Acanthamoeba** | ① **隐形眼镜（contact lens）+ 自来水冲洗 → 角膜炎（keratitis）**：单眼剧痛 + 角膜溃疡，常规抗菌无效 ② 免疫低下 → 慢性 **GAE（granulomatous amebic encephalitis，肉芽肿性阿米巴脑炎）** | 角膜刮片/培养；染色见包囊 | 角膜炎：**PHMB（polyhexamethylene biguanide，聚六亚甲基双胍）+ 氯己定（chlorhexidine）** + 清创 |

### 5.2 虫媒鞭毛原虫（锥虫·利什曼）——治疗特殊

| 虫（中英） | 招牌 vignette | 治疗 |
|---|---|---|
| **美洲锥虫 Trypanosoma cruzi（Chagas 病，恰加斯病）** | **锥蝽（reduviid / "kissing bug"）**，中南美洲土房；急性 **Romaña 征（单侧眶周水肿）**→ 慢性**扩张型心肌病 + 传导阻滞 + 心尖室壁瘤**、**巨食管/巨结肠（megaesophagus/megacolon）** | **苄硝唑（benznidazole）/ 硝呋替莫（nifurtimox）**（仅早期有效；器官损害**不可逆**）|
| **非洲锥虫 T. brucei（睡眠病 sleeping sickness）** | **采采蝇（tsetse fly）**；间歇热 + **后颈淋巴结（Winterbottom 征）**→ 晚期入脑 → 嗜睡/昏迷 | 早期 **喷他脒（pentamidine）/ 苏拉明（suramin）**；CNS 期 **美拉胂醇（melarsoprol）/ 依氟鸟氨酸（eflornithine）** |
| **利什曼原虫 Leishmania** | **白蛉（sandfly）**；皮肤型=无痛**中央溃疡**；**黏膜皮肤型（espundia）=鼻中隔/口鼻毁损**（NBME 爱考）；内脏型=**黑热病（kala-azar，L. donovani）**：发热 + 肝脾大 + **全血细胞减少** | **脂质体两性霉素 B（liposomal amphotericin B）/ 葡萄糖酸锑钠（sodium stibogluconate）** |

### 5.3 食源/接触性线虫补遗

| 虫（中英） | 招牌 vignette | 诊断/治疗 |
|---|---|---|
| **内脏幼虫移行症 VLM（visceral larva migrans）= 犬弓蛔虫 Toxocara canis** | **幼儿食土（geophagia）+ 养狗**→ **嗜酸↑↑ + 肝脾大 + 喘**；眼型 **OLM（ocular larva migrans）= 单眼白瞳**（需与**视网膜母细胞瘤 retinoblastoma** 鉴别）| 血清学（Toxocara IgG）；阿苯达唑 ± 激素 |
| **皮肤幼虫移行症 CLM（cutaneous larva migrans）= 犬/猫钩虫 Ancylostoma braziliense** | **沙滩/赤脚**接触猫狗粪污土 → **蜿蜒（serpiginous）迁移性剧痒红线**，多在足/臀 | 临床诊断；**伊维菌素（ivermectin）/ 阿苯达唑（albendazole）** |
| **异尖线虫 Anisakis** | **生鱼/寿司/刺身**后数小时**上腹剧痛**（似急腹症/消化性溃疡）| **胃镜直视取虫**（多自限）|

## 六、嗜酸增多 + 系统并发症 + 诊断方法（深化）⭐

> [!tip] 嗜酸性粒细胞增多（eosinophilia）只认"蠕虫组织移行"
> - **蠕虫（线虫/吸虫/绦虫）组织移行期** → 嗜酸↑；纯管腔内成虫（肠道带绦虫/蛲虫）未必↑。
> - **原虫（阿米巴/贾第/疟原虫/巴贝/弓形虫）一般不致嗜酸↑**（少数例外：Cystoisospora/Cyclospora）。
> - 考场反射：**"嗜酸↑ + 肺浸润" = Löffler 综合征**（蛔虫/钩虫/粪类圆**幼虫过肺**）；**"嗜酸↑ + 肌痛 + 双侧眶周水肿 + CK（creatine kinase，肌酸激酶）↑" = 旋毛虫**。

### 6.1 系统并发症（"这个虫损伤什么器官"）

| 虫 | 系统并发症 | 钩子 |
|---|---|---|
| **蛔虫 Ascaris** | 幼虫过肺 → **Löffler（一过性游走肺浸润 + 嗜酸↑）**；成虫团 → **肠梗阻/胆道梗阻/胰腺炎** | 大量虫体 |
| **粪类圆线虫 Strongyloides** | **自身感染（autoinfection）**→ 用**激素/HTLV-1（human T-lymphotropic virus 1）/免疫抑制**时 **播散性高度感染（hyperinfection）+ 革兰阴性菌血症/脑膜炎** | 用激素或移植前**先筛查根除**（伊维菌素）|
| **埃及血吸虫 S. haematobium** | 膀胱壁卵肉芽肿 → 无痛**终末血尿** → **膀胱鳞癌（SCC, squamous cell carcinoma）** | 中东/非洲；**鳞癌非腺癌** |
| **曼氏/日本血吸虫 S. mansoni/japonicum** | 肝周 **pipestem 纤维化** → **门脉高压/食管静脉曲张**；卵入肺 → **肺动脉高压** | 肝大但肝细胞功能早期保留 |
| **旋毛虫 Trichinella** | 肌肉期：**双侧眶周水肿 + 肌痛 + 甲下裂片状出血 + CK↑/嗜酸↑**；重者心肌炎 | 生猪/熊肉；**双侧**（vs Chagas Romaña **单侧**）|
| **神经囊虫病 Neurocysticercosis** | 脑实质囊 → **成人新发癫痫**（发展中国家首位病因）；影像多发囊/钙化 | 猪带绦虫**卵**（粪口），非吃肉 |
| **美洲锥虫 Chagas** | 慢性扩张型心肌病 + 传导阻滞、巨食管/巨结肠 | 见 §5.2 |

### 6.2 诊断方法 → 查什么虫

| 方法 | 主要查 |
|---|---|
| **透明胶带试验（scotch tape test）** | 蛲虫（晨起肛周卵）|
| **粪便 O&P（ova & parasites）/ 抗原 / PCR** | 肠道线虫卵；**Giardia/Entamoeba 抗原·PCR 优于镜检**（镜检敏感度低）|
| **厚薄血涂片（Giemsa 染色）** | 疟原虫（环状体）、**巴贝虫（Maltese cross，无色素）**、锥虫、利什曼（无鞭毛体）|
| **血清学** | 弓形虫、囊虫/包虫、Toxocara（VLM）、血吸虫、阿米巴肝脓肿 |
| **影像** | 神经囊虫（囊+钙化）、包虫（蛋壳钙化/子囊）、阿米巴肝脓肿（**anchovy paste**）|
| **内镜** | 异尖线虫（直视取虫）|

> [!warning] 两个"治疗后腹泻不退"别误判失败
> - **Giardia 治后脂肪泻迁延** → 多为**继发性乳糖不耐受**（绒毛刷状缘破坏，乳糖酶丢失）→ **禁奶制品**等绒毛恢复，**不是驱虫失败**。
> - **阿米巴 Entamoeba** → 甲硝唑杀组织滋养体后，须加 **巴龙霉素（paromomycin）清肠腔包囊**，否则带囊复发。

# 寄生虫 & 驱虫药配对速查

> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q3612]]（蛲虫 → 噻嘧啶/阿苯达唑，被湖游史钩向血吸虫）。
> 用法：vignette 给寄生虫感染场景让你选"最合适治疗"时，先**定虫的大类（线虫/吸虫/绦虫/原虫/体外）→ 再选药**。
> 版权：基于公开教科书概念（First Aid / CDC 公开摘要）重新整理，未复制 UWorld 题干。

---

## 一、一张总表（药 ↔ 虫）⭐

> [!tip] 三句话先记住主干
> - **肠道线虫（蛲/蛔/钩/鞭）** → **阿苯达唑 / 甲苯达唑 / 噻嘧啶**（"-bendazole" 三兄弟 + 噻嘧啶）
> - **吸虫 + 绦虫（成虫）** → **吡喹酮（praziquantel）**
> - **原虫（阿米巴/贾第/滴虫）** → **甲硝唑（metronidazole）**

| 药 | 中文 | 主治 |
|---|---|---|
| albendazole / mebendazole | 阿苯达唑 / 甲苯达唑 | **肠道线虫**：蛲虫、蛔虫、钩虫、鞭虫；**囊虫病（cysticercosis）+ 包虫病（echinococcus）** |
| pyrantel pamoate | 双羟萘酸噻嘧啶 | 蛲虫、蛔虫、钩虫（孕妇相对安全选择） |
| ivermectin | 伊维菌素 | **粪类圆线虫病（strongyloidiasis）、盘尾丝虫病（onchocerciasis 河盲）**；疥疮/虱备选 |
| diethylcarbamazine (DEC) | 乙胺嗪 | **淋巴丝虫病（Wuchereria）、罗阿丝虫（Loa loa）** |
| praziquantel | 吡喹酮 | **血吸虫病（schistosomiasis）、吸虫（肝/肺吸虫）、绦虫成虫** |
| metronidazole / tinidazole | 甲硝唑 / 替硝唑 | **原虫**：阿米巴、贾第虫（Giardia）、滴虫（Trichomonas） |
| nitazoxanide | 硝唑尼特 | 隐孢子虫（Cryptosporidium）、贾第虫 |
| pyrimethamine + sulfadiazine | 乙胺嘧啶 + 磺胺嘧啶 | 弓形虫病（toxoplasmosis） |
| atovaquone + azithromycin | 阿托伐醌 + 阿奇霉素 | **巴贝虫病（babesiosis）** |
| 氯喹 / 青蒿素类 ± 伯氨喹 | chloroquine / artemisinin ± primaquine | **疟疾（malaria）**；伯氨喹清肝内休眠子（P. vivax/ovale） |
| permethrin | 氯菊酯 | **疥疮（scabies）、虱（lice）** |

## 二、按虫的大类定药（决策主干）

```
寄生虫
├─ 蠕虫 Helminths
│   ├─ 线虫 Nematodes（圆虫）
│   │   ├─ 肠道：蛲/蛔/钩/鞭 ── 阿苯达唑·甲苯达唑·噻嘧啶
│   │   ├─ 组织：粪类圆·盘尾丝虫 ── 伊维菌素
│   │   └─ 淋巴丝虫·Loa loa ── 乙胺嗪 DEC
│   ├─ 吸虫 Trematodes（血吸/肝/肺吸虫）── 吡喹酮
│   └─ 绦虫 Cestodes
│       ├─ 成虫（肠道带绦虫）── 吡喹酮
│       └─ 幼虫：囊虫·包虫 ── 阿苯达唑（± 手术）
└─ 原虫 Protozoa
    ├─ 肠道：阿米巴·贾第·隐孢子 ── 甲硝唑 / 硝唑尼特
    ├─ 泌尿生殖：滴虫 ── 甲硝唑（性伴同治）
    ├─ 血液：疟原虫·巴贝虫 ── 氯喹/青蒿素 · 阿托伐醌+阿奇
    └─ 组织：弓形虫 ── 乙胺嘧啶+磺胺嘧啶
```

### 线虫 Nematodes 招牌 → 药

| 虫 | 招牌 vignette | 首选 |
|---|---|---|
| **蛲虫 Enterobius（pinworm）** | **夜间肛周瘙痒**，学龄儿童；透明胶带试验 | **噻嘧啶 pyrantel / 阿苯达唑 albendazole**；全家同治、2 周后重复 |
| 蛔虫 Ascaris | 粪口；可肠梗阻/胆道/Löffler 肺浸润 | 阿苯达唑 albendazole / 甲苯达唑 mebendazole |
| 钩虫 Hookworm | 赤脚接触土壤；**缺铁性贫血** | 阿苯达唑 albendazole / 甲苯达唑 mebendazole |
| 鞭虫 Trichuris | **直肠脱垂**（重度儿童） | 阿苯达唑 albendazole / 甲苯达唑 mebendazole |
| **粪类圆线虫 Strongyloides** | 自体感染；免疫抑制 → **播散/高度感染**；荨麻疹+腹痛+咳喘 | **伊维菌素 ivermectin** |
| 盘尾丝虫 Onchocerca | 黑蝇；**河盲（river blindness）**+ 皮肤结节 | **伊维菌素 ivermectin** |
| 旋毛虫 Trichinella | 生猪肉；**肌痛 + 眶周水肿 + 嗜酸↑** | 阿苯达唑 albendazole（± 激素） |

### 吸虫 / 绦虫 → 吡喹酮（幼虫例外）

| 虫 | 招牌 | 首选 |
|---|---|---|
| **血吸虫 Schistosoma** | **淡水游泳（swimmer's itch）**→ 后期血尿（埃及血吸虫）/ 肝脾门脉高压（曼氏） | **吡喹酮 praziquantel** |
| 肝/肺吸虫 Clonorchis/Paragonimus | 生鱼/生蟹 | 吡喹酮 praziquantel |
| 带绦虫 Taenia（成虫） | 生牛/猪肉 | 吡喹酮 praziquantel |
| **囊虫病 Neurocysticercosis** | 猪带绦虫卵 → **脑囊肿 + 癫痫** | **阿苯达唑 albendazole**（± 激素/抗癫痫） |
| 包虫病 Echinococcus | 犬/羊；**肝包虫囊（蛋壳钙化）** | **阿苯达唑 albendazole** + 手术/PAIR（勿穿破致过敏） |

### 原虫 Protozoa

| 虫 | 招牌 | 首选 |
|---|---|---|
| 阿米巴 Entamoeba | 旅行；血性腹泻 + **肝脓肿（RUQ 痛）** | **甲硝唑 metronidazole** + 巴龙霉素 paromomycin（清肠腔包囊） |
| 贾第虫 Giardia | 野营/溪水；**脂肪泻 + 胀气**，无血 | **甲硝唑 metronidazole**（或硝唑尼特 nitazoxanide） |
| 滴虫 Trichomonas | **黄绿泡沫白带 + 草莓样宫颈** | **甲硝唑 metronidazole**（**性伴同治**） |
| 隐孢子虫 Cryptosporidium | **AIDS（CD4<100）水样泻** | 硝唑尼特 nitazoxanide（核心是恢复免疫/ART） |
| 弓形虫 Toxoplasma | **AIDS 环形强化脑病灶**；先天 TORCH | 乙胺嘧啶 pyrimethamine + 磺胺嘧啶 sulfadiazine（+ 亚叶酸 leucovorin） |
| 疟原虫 Plasmodium | 旅行 + 周期热 + 溶血 | 氯喹 chloroquine/青蒿素类 artemisinin；**伯氨喹 primaquine 清 vivax/ovale 肝休眠子** |
| **巴贝虫 Babesia** | **无脾 + 东北蜱 + 血管内溶血**；血涂片 **Maltese cross** | **阿托伐醌 atovaquone + 阿奇霉素 azithromycin** |

## 三、高频干扰项 & buzzword（最容易错的地方）⭐

> [!warning] 干扰项黑名单
> - **吡喹酮 ≠ 蛲虫**：吡喹酮治**血吸虫/吸虫/绦虫成虫**；"淡水游泳"把你往血吸虫（吡喹酮）带，但**夜间肛周痒是蛲虫**（噻嘧啶/阿苯达唑）。← [[mistakes/uworld-mistakes_2026-06#^Q3612]] 错点
> - **囊虫/包虫（幼虫）用阿苯达唑，不是吡喹酮**：脑囊虫癫痫、肝包虫囊 → 阿苯达唑（吡喹酮治的是肠道**成虫**绦虫）。
> - **克霉唑（clotrimazole）/ 制霉菌素是抗真菌**（念珠菌），**不是驱虫药**——出现在选项里是纯干扰。
> - **甲硝唑只打原虫（阿米巴/贾第/滴虫）+ 厌氧菌**，对线虫/吸虫无效。
> - **疥疮 vs 蛲虫都"夜间痒"**：疥疮在**指缝/腕/隧道 burrows**（permethrin 氯菊酯）；蛲虫在**肛周**（噻嘧啶/阿苯达唑）。

> [!tip] 招牌 → 立刻反射
> | Buzzword | 虫 | 药 |
> |---|---|---|
> | 夜间**肛周**痒 + 学龄儿 | 蛲虫 Enterobius | 噻嘧啶 pyrantel/阿苯达唑 albendazole |
> | **淡水游泳** + 血尿/肝脾 | 血吸虫 Schistosoma | 吡喹酮 praziquantel |
> | **河盲** + 皮下结节 + 黑蝇 | 盘尾丝虫 Onchocerca | 伊维菌素 ivermectin |
> | 免疫抑制 + **播散性高度感染** | 粪类圆线虫 Strongyloides | 伊维菌素 ivermectin |
> | **脑环形强化**（AIDS） | 弓形虫 Toxoplasma | 乙胺嘧啶 pyrimethamine+磺胺嘧啶 sulfadiazine |
> | **无脾 + 东北蜱 + 溶血 + Maltese cross** | 巴贝虫 Babesia | 阿托伐醌 atovaquone+阿奇霉素 azithromycin |
> | **草莓样宫颈 + 泡沫白带** | 滴虫 Trichomonas | 甲硝唑 metronidazole（性伴同治） |
> | **赤脚 + 缺铁性贫血** | 钩虫 Hookworm | 阿苯达唑 albendazole/甲苯达唑 mebendazole |

## 四、几个易考机制点

- **阿苯达唑/甲苯达唑（albendazole/mebendazole）**：抑制微管（β-tubulin）→ 虫不能摄取葡萄糖。孕期慎用（致畸，尤其早孕）→ 孕妇肠道线虫可考虑**噻嘧啶（pyrantel）**或推迟治疗。
- **吡喹酮（praziquantel）**：增加虫体膜对 Ca²⁺ 通透 → 痉挛麻痹。**脑囊虫先评估眼/脑负荷 + 备激素**（杀虫致炎症反应）。
- **伊维菌素（ivermectin）**：开虫体谷氨酸门控 Cl⁻ 通道。**Loa loa 高负荷者禁伊维菌素**（脑病风险），改 DEC（乙胺嗪 diethylcarbamazine）前需先评估微丝蚴血症。
- **甲硝唑（metronidazole）**：双硫仑样反应（disulfiram-like）→ 用药期间**禁酒**。

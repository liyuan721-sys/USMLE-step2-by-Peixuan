# 肠球菌 Enterococcus 一页纸

## 零、一图总览 ⭐⭐⭐

```
肠球菌 Enterococcus（G⁺ 球菌）
├─ 身份：成对/短链、catalase⊖、耐胆汁 + 6.5% NaCl、PYR⊕、group D
├─ nitrite⊖ 机制：无硝酸盐还原酶（≠ 无菌性脓尿，培养仍⊕）
├─ 头孢天然耐药 = LAME 的 E（所有头孢单药都无效）
└─ 治疗：一般 amp±氨基糖苷 / IE 协同(amp+gent 或 amp+ceftriaxone) / VRE→linezolid·dapto
```

---

## 一、身份（微生物学）

| 项目 | 内容 |
|---|---|
| **染色/形态** | **革兰阳性球菌（G⁺ cocci）**，成对或**短链**（像链球菌）|
| **分类** | 原属 **group D 链球菌**，后独立成属；常考 **E. faecalis（最常见）/ E. faecium（更耐药、VRE 多）** |
| **关键生化** | **catalase 阴性**（与葡萄球菌区分）；**PYR 阳性**；**耐胆汁 + 6.5% NaCl 生长**（group D strep 鉴别招牌）|
| **常见感染** | UTI（尤导尿/器械后）、**IE（GU 操作后）**、胆道/腹腔感染、导管相关菌血症 |

---

## 二、nitrite 阴性机制 ⭐

> [!tip] 为什么肠球菌 UTI 是 nitrite 阴性
> 尿试纸 **nitrite 阳性**依赖细菌的**硝酸盐还原酶**（把尿中 nitrate→nitrite）。
> - **G⁻ 肠杆菌科（E. coli / Klebsiella / Proteus）有此酶** → **nitrite 阳性**（典型膀胱炎）。
> - **肠球菌 + 其他 G⁺ 尿路菌（如 Staph saprophyticus）无此酶** → **nitrite 阴性**（即使真有 UTI）。
>
> **"脓尿/菌尿 + nitrite 阴性" = 把 G⁺ 尿路菌（肠球菌）和 G⁻（E. coli）分开的钥匙。**

> [!warning] nitrite 阴性 ≠ 无菌性脓尿
> - **肠球菌 UTI：尿培养仍阳性**，只是 nitrite 试纸阴。
> - **真正"无菌性脓尿"（WBC⁺ / 培养⊖ / nitrite⊖）** → 指向**衣原体/淋球菌尿道炎**或 **TB**。
> - 分水岭 = **尿培养阳 vs 阴**。

---

## 三、头孢天然耐药（LAME 的 E）⭐

> [!danger] 肠球菌虽是 G⁺，但所有头孢都不覆盖它（天然耐药）— 高频陷阱
> 看到 **"GU 操作后 + GPC 血培养 + nitrite⊖ UTI"** 别反射头孢。
> 头孢盲区 **LAME**：**L**isteria / **A**typicals / **M**RSA（仅 5 代 ceftaroline 例外）/ **E**nterococci。

---

## 四、治疗 ⭐⭐⭐

| 情境 | 方案 |
|---|---|
| **一般感染** | **Ampicillin**（或 vancomycin）± 氨基糖苷协同 |
| **IE（心内膜炎）** | **Ampicillin + Gentamicin** 或 **Ampicillin + Ceftriaxone**（双 β-lactam 协同，替代氨基糖苷以避肾毒/耳毒）|
| **PCN 过敏** | **Vancomycin** |
| **VRE（耐万古肠球菌，多为 E. faecium）** | **Linezolid / Daptomycin** |

> [!warning] 最易自我推翻的点
> **IE 方案里的 ceftriaxone 是"协同"角色（amp 抑 PBP + ceftriaxone 占其他 PBP），不代表头孢能单独覆盖肠球菌。** 别因为方案里有头孢就推翻"头孢不覆盖肠球菌"的铁律。

---

## 五、Memory Hook

> **「肠球菌 = G⁺ 球菌（catalase⊖、耐胆汁+盐、PYR⊕）；UTI nitrite⊖（无还原酶，但培养⊕）；头孢一概不覆盖（LAME 的 E）；治 = amp±氨基糖苷，IE 协同（amp+gent / amp+ceftriaxone），VRE→linezolid/dapto。」**

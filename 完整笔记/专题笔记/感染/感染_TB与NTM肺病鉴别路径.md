# TB 与 NTM 肺病鉴别路径

> [!info] 笔记定位
> 慢性咳嗽 + 消瘦盗汗 + 抗酸阳性 (AFB+, acid-fast bacilli 抗酸杆菌) 这一组，要在 **TB（结核 tuberculosis）vs NTM（非结核分枝杆菌 nontuberculous mycobacteria，代表 = 肺 MAC 鸟分枝杆菌复合群 Mycobacterium avium complex）** 之间分流。
> 本卡 = **分流 + 诊断路径 + 治疗对照**决策骨架；TB 原发 vs 再激活影像详解 + MAC 人物模板（含 X 线图）见 [[完整笔记/专题笔记/感染/感染_肺部感染病原与用药速查]] §三（不在此重复，只反链）。
> 错题驱动：[[NBME11_错题本#^Q070]]（肺 MAC，先 AFB）+ [[mistakes/uworld-mistakes_2026-06#^Q19417]]（原发型 TB，抗肺炎无效）+ [[mistakes/uworld-mistakes_2026-06#^Q3246]]（潜伏 TB 治疗）。

## 一、一眼分流：TB vs 肺 MAC/NTM ⭐⭐⭐

| 维度 | **结核 TB** | **肺 MAC / NTM** |
|---|---|---|
| 人物模板 | 暴露史（流行区 / 监狱 / 收容 / HIV）、各年龄 | **瘦小、年长、不吸烟女性**（Lady Windermere）；或 COPD（慢性阻塞性肺疾病 chronic obstructive pulmonary disease）男（纤维空洞型）|
| 传播 | **人传人**（飞沫，需隔离）| **环境获得（水/土），不人传人**（无需隔离）|
| 经典影像 | 再激活 = **上叶空洞**；原发 = 叶浸润 + **单侧肺门淋巴结** ± 胸水 | **中下叶支气管扩张 (bronchiectasis) + 散在小结节**（结节-支扩型）|
| 全身症状 | 盗汗 / 消瘦 / 发热（常更显著）| 慢性咳痰 + 消瘦 / 盗汗（更隐袭）|
| 抗酸染色 | **AFB 阳性**（两者都阳，染色分不开）| **AFB 阳性** |
| 怎么最终分 | **培养 + 分子（NAAT）**鉴定菌种 | 同左 —— **靠培养/分子定种** |
| PPD/IGRA | 多阳（反映感染/暴露）| **可弱阳/假阳**（交叉反应），不用于诊断 |

> [!danger] 核心分水岭
> **AFB 阳性 ≠ 一定是 TB**。抗酸染色把 TB 和 NTM 一起染红，**最终定种靠培养 + 分子检测 (NAAT)**。一眼初分靠**人物 + 影像**：上叶空洞 + 暴露史 + 人传人风险 → TB；瘦老不吸烟女 + 中下叶支扩结节 + 环境获得 → MAC。

## 二、诊断路径（处置顺序）⭐

```
慢性咳 + 消瘦盗汗 + 疑分枝杆菌
  │
  ① 无创病原学先行：痰 AFB 涂片 ×3 + 分枝杆菌培养（金标准，慢，4–6 周）
  │      + NAAT/PCR（核酸扩增检测 nucleic acid amplification test，快速定 TB 并查耐药 rpoB/利福平）
  │      （间歇血丝痰 ≠ 大咯血 → 不跳栓塞/活检/肺叶切除）
  │
  ② 疑活动性 TB → 同时呼吸隔离（负压病房）+ 报告公共卫生
  │
  ③ 培养/分子出菌种：
  │      M. tuberculosis → 活动性 TB → RIPE（见 §四）
  │      NTM（M. avium 等）→ 套 NTM 诊断三条件（见下）再决定治不治
  │
  └ 若问"筛查既往感染"（无症状）→ 走 PPD / IGRA（见下），不是痰培养
```

> [!warning] NTM 诊断三条件（ATS）—— 为什么不能"一次阳性就治" ⭐
> NTM 是**环境定植菌**，单次痰阳可能只是污染/定植。诊断需**临床 + 影像 + 微生物**三者齐全：
> 1. **临床**：呼吸/全身症状（且排除其他诊断）。
> 2. **影像**：结节-支扩 或 空洞。
> 3. **微生物**：**≥2 次独立咳痰培养阳性**，或 **1 次 BAL（支气管肺泡灌洗 bronchoalveolar lavage）/ 经支气管活检阳性**。
> → 对比 TB：**一次确认 M. tuberculosis 即按活动性 TB 治 + 隔离**（TB 永远是致病菌，无"定植"一说）。

> [!info] PPD vs IGRA（查既往/潜伏感染，不查"是 TB 还是 NTM"）
> | | PPD（结核菌素皮试 tuberculin skin test）| IGRA（γ干扰素释放试验 interferon-γ release assay）|
> |---|---|---|
> | 读法 | 48–72h 看**硬结**直径（非红斑）| 抽血 |
> | BCG（卡介苗）影响 | **会假阳**（接种过 BCG 者）| **不受 BCG 干扰** ⭐ |
> | NTM 交叉反应 | **可假阳** | 较少 |
> | 适用 | 基层/儿童 | **BCG 接种史 / 复诊依从差**首选 |
>
> **PPD 三档阳性阈值**（阳性后下一步 = CXR 胸片；阳 → 痰 + 评活动性，阴 → 按潜伏 TB 治）：
>
> | 阈值 | 人群 |
> |---|---|
> | **≥5 mm** | HIV / 免疫抑制 / 器官移植 / 活动 TB 密接 / CXR 陈旧 TB 灶 |
> | **≥10 mm** | 近 5 年移民（高发区）/ 医务 / 监狱 / 无家可归·静脉吸毒 / 糖尿病·尘肺等高危病 / <4 岁儿童 |
> | **≥15 mm** | 无任何危险因素的一般人群 |

## 三、影像分型速记（详图见肺部感染速查 §三）

```
TB
  再激活型 reactivation → 上叶尖后段空洞（最经典"消瘦盗汗 + 上叶空洞"）
  原发型 primary        → 叶浸润 + 单侧肺门淋巴结肿 ± 胸水（似普通肺炎、抗肺炎无效）
                          ※ 单侧肺门 = 感染(TB)/肿瘤；双侧对称 = 结节病/淋巴瘤

NTM / 肺 MAC
  结节-支扩型 nodular-bronchiectatic → 瘦老不吸烟女、中下叶（Lady Windermere）
  纤维空洞型 fibrocavitary           → 既往肺病/COPD 男、上叶空洞（似 TB，更易混 → 靠培养切）
```

> [!tip] 最易混点
> **MAC 纤维空洞型**也长上叶空洞，和再激活 TB 影像几乎一样 → **影像分不开时，人物（瘦老不吸烟女 vs 暴露史）+ 培养定种**才是终局。原发型 TB 则常被当"治不好的肺炎"（[[mistakes/uworld-mistakes_2026-06#^Q19417]]）。

## 四、治疗对照 ⭐⭐

| | **活动性 TB** | **潜伏 TB (LTBI)** | **肺 MAC（需治时）** |
|---|---|---|---|
| 方案 | **RIPE** 四联 | 单/双药（见下）| **大环内酯 + 利福平 + 乙胺丁醇**三联 |
| 药 | 异烟肼 isoniazid (INH) + 利福平 rifampin (RIF) + 吡嗪酰胺 pyrazinamide (PZA) + 乙胺丁醇 ethambutol (EMB) | INH×9 月 / RIF×4 月 / INH+利福喷丁 rifapentine 每周×3 月 | 阿奇霉素 azithromycin（或克拉霉素 clarithromycin）+ RIF + EMB |
| 疗程 | 强化 2 月（4 药）→ 巩固 4 月（INH+RIF），共 **6 月** | 3–9 月 | **培养转阴后 ≥12 月**（长） |
| 前提 | 已确认/高度疑活动性 + 隔离 | **先排除活动性**（CXR 阴、无症状）才单药 | 满足 NTM 三条件 |

> [!danger] 用药铁律
> - **潜伏 TB 治疗前必须先排除活动性 TB**（CXR ± 痰）——对活动性病只给单药 INH = **诱导耐药**（[[mistakes/uworld-mistakes_2026-06#^Q3246]] 考点）。
> - **MAC 用单药大环内酯 = 诱导大环内酯耐药** → 必须三联，别图省事。

> [!warning] 抗结核药四毒性（高频"加什么/查什么"）
> | 药 | 招牌毒性 | 对策 |
> |---|---|---|
> | **INH 异烟肼** | **周围神经病**（耗 B6）、肝炎、药物性狼疮 | **加维生素 B6 (pyridoxine 吡哆醇)** |
> | **RIF 利福平** | **体液橙红色**（泪/尿/汗）、肝、**强 CYP450 诱导剂**（降华法林 warfarin / 口服避孕药 OCP / 蛋白酶抑制剂浓度）| 提前告知橙色；查相互作用 |
> | **PZA 吡嗪酰胺** | **高尿酸血症/痛风**、肝 | 监测尿酸/肝功 |
> | **EMB 乙胺丁醇** | **视神经炎 → 红绿色盲 / 视力↓** | 基线 + 定期**查色觉/视力** |
>
> 共性：四药多有肝毒性 → 基线 + 监测 LFT（肝功）；INH/RIF/PZA 肝毒性最相关。

## 五、Memory Hook

```
AFB 阳性两条路 → 培养定种才分家
  上叶空洞 + 暴露/人传人 + 各年龄 → TB → RIPE 6 月（隔离）
  中下叶支扩结节 + 瘦老不吸烟女 + 环境不传人 → MAC → 大环内酯+利福平+乙胺丁醇 ≥12 月

诊断顺序：先无创 AFB 痰涂片+培养（别跳活检/手术）
NTM 要三条件（环境定植 → 一次阳性不算）；TB 一次确认就治+隔离
潜伏 TB 单药前先排活动性；MAC 不用单药（都防耐药）

四毒性：INH→神经(加B6)/RIF→橙色+酶诱导/PZA→尿酸/EMB→视神经红绿盲
PPD 三档 5/10/15；BCG 史用 IGRA
```

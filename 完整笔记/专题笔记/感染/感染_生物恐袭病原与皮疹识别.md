# 感染_生物恐袭病原与皮疹识别（Bioterrorism agents & rash recognition）

> [!tip] 一句话核心
> USMLE 生物恐袭题的题眼往往是**皮疹形态学**或**暴露/前驱模式**。天花（smallpox / variola）与水痘（varicella）的最强判别词是 **"same stage（同一阶段）" vs "different stages（多阶段并存）"** + 分布（离心 centrifugal vs 躯干为主 central）。

---

## 一、天花 vs 水痘（NBME11 Q034 原型）⭐

| 特征 | 天花 Smallpox (Variola) | 水痘 Varicella (VZV) |
|---|---|---|
| 前驱期 prodrome | **明显**（高热、乏力、头痛、腹痛，退热后出疹） | 通常轻或无 |
| 皮疹阶段 | **所有病灶同一阶段**（synchronous）⭐ | **多阶段并存**（斑丘疹 + 水疱 + 结痂同现） |
| 分布 distribution | **离心性 centrifugal**：面部、口腔、四肢 > 躯干 | **向心性 / 躯干为主 central**：躯干 > 四肢 |
| 手掌足底 | 常受累 | 少受累 |
| 生物恐袭关联 | **强**（已根除，现实中出现即疑生物恐袭） | 弱（常见儿童病） |
| 接触者预防 prophylaxis | **牛痘 vaccinia**（ring vaccination 环状接种）⭐ | 水痘疫苗 varicella vaccine / VZIG |

> [!warning] 记忆点
> - **"Same stage = smallpox"**（同阶段 = 天花）。
> - 天花是 **centrifugal（离心）**；⚠️ 麻疹 measles 是自发际/耳后**头向足蔓延 cephalocaudal**，勿把 "centrifugal" 张冠李戴到麻疹。
> - 接触者预防不是打水痘疫苗，而是 **vaccinia（牛痘）** 做 ring vaccination——这正是历史上根除天花的核心策略。

```
天花 smallpox 处理逻辑
variola 暴露
   │ 潜伏
   ▼
febrile prodrome（明显前驱：热退后出疹）
   │
   ▼
synchronous vesicular-pustular rash（同阶段疱疹→脓疱，离心分布）
   │
   ▼
concern for person-to-person spread（人传人风险）
   │
   ▼
ring vaccination of contacts with vaccinia（接触者环状接种牛痘）
```

## 二、Category A 生物恐袭病原速查

| 病原 | 疾病 | 招牌线索 | 一线处理/预防 |
|---|---|---|---|
| **Variola virus 天花病毒** | Smallpox 天花 | 同阶段离心疱疹 + 明显前驱 | Vaccinia 牛痘（接触者环状接种） |
| **Bacillus anthracis 炭疽杆菌** | Anthrax 炭疽 | **无痛黑焦痂 painless black eschar**（皮肤型）；吸入型 → 纵隔增宽 widened mediastinum + 出血性纵隔炎 | 环丙沙星 ciprofloxacin / 多西环素 doxycycline |
| **Yersinia pestis 鼠疫耶尔森菌** | Plague 鼠疫 | **淋巴结肿大化脓 bubo**（腺鼠疫）；肺鼠疫 → 咯血 | 链霉素 streptomycin / 庆大霉素 gentamicin |
| **Francisella tularensis 土拉热杆菌** | Tularemia 兔热病 | 溃疡 + 局部淋巴结肿（ulceroglandular）、猎兔/蜱暴露 | 链霉素 streptomycin / 庆大霉素 gentamicin |
| **Clostridium botulinum 肉毒梭菌** | Botulism 肉毒中毒 | **对称下行性弛缓性麻痹 descending flaccid paralysis** + 瞳孔散大、复视，感觉正常 | 抗毒素 antitoxin + 支持 |

> [!info] 皮肤线索一眼分
> - **无痛黑焦痂 (painless black eschar)** → 炭疽（皮肤型），别与糖尿病足坏死焦痂混（后者有慢性溃疡背景）。
> - **同阶段离心疱疹** → 天花。
> - **淋巴结肿大化脓 (bubo)** → 鼠疫。

---

## 三、易混陷阱（普适）

> [!danger] 高频坑
> - **看到疱疹就选水痘** ❌ → 先扫"同阶段 vs 多阶段"和"离心 vs 躯干"两条线。
> - **炭疽黑焦痂 vs 天花疱疹** —— 炭疽是无痛焦痂性丘疹，不是同步疱疹。
> - **接触者天花预防选水痘疫苗** ❌ → 是 vaccinia（牛痘）。

## 四、复习自测

> [!question]- 疑似生物恐袭暴露后：先高热、乏力、头痛、腹痛，热退后出疹；疱疹集中在面部和四肢、口腔，且**全部处于同一发展阶段**。诊断？接触者如何预防？
> **天花 smallpox (variola)**。判别词：明显前驱期 + **同阶段病灶（synchronous）** + **离心分布（面部四肢 > 躯干）**。接触者预防用 **vaccinia（牛痘）** 做环状接种，不是水痘疫苗。

> [!question]- 农业工人手背出现无痛性黑色焦痂，周围水肿，无明显疼痛。最可能病原？
> **Bacillus anthracis（炭疽杆菌）**，皮肤型炭疽的无痛黑焦痂。一线环丙沙星 ciprofloxacin 或多西环素 doxycycline。

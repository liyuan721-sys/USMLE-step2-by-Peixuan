# 免疫抑制宿主肺浸润鉴别（Pulmonary infiltrates in the immunocompromised host）

> [!info] 为什么建这张
> 整合错题簇：[[mistakes/uworld-mistakes_2026-06#^Q2997]]（侵袭性肺曲霉病 halo 征）+ [[mistakes/uworld-mistakes_2026-06#^Q18899]]（移植后 CMV 肺炎，在吃 TMP-SMX → 排 PCP）+ [[mistakes/uworld-mistakes_2026-06#^Q4417]]（Nocardia 肺结节 + 脑脓肿）+ [[mistakes/uworld-mistakes_2026-06#^Q7494]]（移植 <1 月 = 细菌）。
> **解题三步**：① 免疫缺陷类型 → 病原谱 → ② 移植/暴露时间线 → ③ 影像模式 → 锁病原 → 选确诊 + 经验治疗。
> 区别于 [[完整笔记/专题笔记/免疫/免疫_原发性免疫缺陷鉴别]]（原发遗传缺陷谱）——本篇针对**获得性免疫抑制**（化疗粒缺 / 移植 / HIV / 长期激素）的肺浸润。

---

## 一、第一步：免疫缺陷类型 → 病原谱 ⭐⭐⭐

> [!danger] "缺什么 → 长什么菌"
> 先问"宿主缺哪一层免疫"，就能把病原圈到一类。

| 缺陷类型 | 典型情境 | 高危病原（肺）|
|---|---|---|
| **中性粒细胞减少 (neutropenia, <500/μL)** | 化疗、白血病、再障 | **革兰阴性杆菌（铜绿假单胞 Pseudomonas）+ 肠杆菌；侵袭性霉菌（曲霉 Aspergillus / 毛霉 Mucor）+ 念珠菌 Candida** |
| **细胞免疫缺陷（T 细胞）** | **HIV CD4<200、实体器官/造血干细胞移植、长期激素（泼尼松 ≥20 mg）、钙调磷酸酶抑制剂、抗 TNF 生物制剂** | **耶氏肺孢子菌 (PCP/PJP)、巨细胞病毒 (CMV)、结核 TB / 非结核分枝杆菌、隐球菌 (Cryptococcus) / 地方性真菌、Nocardia、Listeria、弓形虫** |
| **体液 / 脾切 / 补体** | 多发性骨髓瘤、CVID、脾切除、利妥昔单抗 (rituximab) | **有荚膜菌**：肺炎链球菌 (S. pneumoniae)、流感嗜血杆菌 (H. influenzae)、奈瑟菌 |

> [!tip] 钥匙
> **粒缺 → 假单胞 + 霉菌；细胞免疫 → 机会感染（PCP/CMV/真菌/Nocardia）；体液/脾/补体 → 有荚膜菌。**

---

## 二、第二步：实体器官移植时间线（高频考"什么时候长什么"）⭐⭐

```
移植后时间轴：
│
├─ <1 月  ── 院内/手术相关细菌（吻合口/导管/吸入性 HAP）、供者来源、HSV 再激活
│            ※ "机会性感染窗"还没开 → 别选 CMV/PCP/曲霉（Q7494）
│
├─ 1–6 月 ── 机会性感染高峰：PCP、CMV、曲霉 Aspergillus、Nocardia、TB 再激活
│            （预防药一旦停/漏 → 立刻冒头）
│
└─ >6 月  ── ① 社区获得（肺炎链球菌、流感）为主
             ② 慢性排斥/持续强免疫抑制者 → 仍可迟发机会感染（PCP/CMV/隐球/Nocardia）
             ③ EBV → 移植后淋巴增殖病 (PTLD)
```

> [!warning] 移植 <1 月发热脓毒 ≠ 机会感染
> **<1 月窗口主因是普通院内细菌**（手术部位、导管、吸入），**不是 CMV/PCP/曲霉**——这些"机会性"病原通常 1 月后才登场（Q7494）。

## 三、第三步：影像模式 → 病原 ⭐⭐

| CT/胸片模式 | 首先想 | 备选 |
|---|---|---|
| **结节 + 晕征 (halo sign，结节周围磨玻璃)** | **侵袭性肺曲霉病**（血管侵袭性霉菌）| 其他血管侵袭霉菌、Nocardia、脓毒栓 |
| **反晕征 (reverse halo)** | **毛霉病 (mucormycosis)** | 机化性肺炎 |
| **双肺弥漫磨玻璃 / 间质浸润** | **PCP（HIV CD4<200）、CMV、病毒性肺炎** | 弥漫肺泡出血 DAH（非感染）、药物/放射性肺炎 |
| **大叶实变 (lobar consolidation)** | 典型细菌（肺炎链球菌等）| — |
| **空洞 (cavitary)** | **结核 TB、Nocardia、真菌** | 脓毒栓、GPA（非感染）|
| **树芽征 (tree-in-bud)** | 非结核分枝杆菌、病毒/细菌细支气管炎 | — |

## 四、重点病原速查（钥匙 → 确诊 → 治疗）⭐⭐⭐

> [!info] 表内缩写：BAL = 支气管肺泡灌洗 bronchoalveolar lavage；GM = 半乳甘露聚糖 galactomannan；CrAg = 隐球菌抗原 cryptococcal antigen；TMP-SMX = 复方磺胺甲噁唑 trimethoprim-sulfamethoxazole。

| 病原 | 宿主/钥匙 | 影像 | 确诊 | 治疗 |
|---|---|---|---|---|
| **侵袭性肺曲霉病 (invasive aspergillosis)** | 粒缺/移植；**发热+胸膜痛+咯血**三联 | 结节 + **halo 征**、空洞 | 血清 **GM + β-D-葡聚糖**、BAL 培养/活检 | **伏立康唑 (voriconazole) ± 棘白菌素（卡泊芬净 caspofungin）** |
| **耶氏肺孢子菌 (PCP/PJP)** | **HIV CD4<200**，或移植/激素**未用 TMP-SMX 预防**；**亚急性进行性气短 + 干咳 + 显著低氧 / 运动后血氧骤降** | **双肺弥漫磨玻璃**（肺门周围）| **↑LDH + β-D-葡聚糖+**；诱导痰/BAL 银染（不能培养）| **TMP-SMX**（PaO₂<70 或 A-a>35 加**糖皮质激素**）|
| **巨细胞病毒 (CMV) 肺炎** | 移植 **1–6 月**（尤 供者+/受者−）；发热+干咳 ± **其他器官**（结肠炎/视网膜炎）| 双肺弥漫磨玻璃 | **CMV PCR（病毒载量）**、活检 **"猫头鹰眼"包涵体** | **更昔洛韦 / 缬更昔洛韦 (ganciclovir / valganciclovir)** |
| **诺卡菌 (Nocardia)** | 长期激素 / 细胞免疫缺陷；**肺结节/空洞 ± 脑脓肿** | 结节、空洞、肿块 | **弱/部分抗酸 + 革兰阳性分枝串珠丝状菌** | **TMP-SMX**（重症 + 亚胺培南；累 CNS 时联合）|
| **毛霉病 (mucormycosis)** | **糖尿病酮症酸中毒 (DKA)、铁过载/去铁胺、粒缺** | **反晕征**、血管侵袭、鼻-眶-脑 | 活检见**宽大无隔菌丝直角分支** | **两性霉素 B (amphotericin B) + 手术清创** |
| **隐球菌 (Cryptococcus)** | HIV/T 细胞缺陷；肺结节 ± **脑膜炎** | 结节、肿块 | **血清/脑脊液 CrAg** | 两性霉素 B + 氟胞嘧啶 → 氟康唑 |
| **结核再激活 (TB)** | 免疫抑制；**上叶空洞** | 上叶空洞、粟粒 | 抗酸 + 培养 + 核酸 NAAT | 四联抗结核 (RIPE) |

## 五、高频鉴别钥匙（CK 最爱设的镜像题）⭐

> [!danger] 钥匙 1：在吃 TMP-SMX 预防 + 双肺磨玻璃 → 排 PCP，想 CMV
> TMP-SMX 同时预防 PCP，所以**正在规律服 TMP-SMX 的移植患者出现弥漫磨玻璃肺炎 → PCP 概率极低 → 转向 CMV**（Q18899）。反之**没用预防 + HIV CD4<200 + 运动后血氧骤降 + LDH↑ = PCP**。

> [!danger] 钥匙 2：结节 + halo + 咯血 + 粒缺 = 曲霉；反晕 + DKA = 毛霉
> 都是血管侵袭性霉菌：**曲霉 = halo 征 + 粒缺/移植 + 咯血**（Q2997）；**毛霉 = 反晕征 + 糖尿病酮症酸中毒/铁过载 + 鼻眶脑**，治疗必加**手术**。

> [!danger] 钥匙 3：肺结节 + 脑脓肿 + 部分抗酸分枝丝状 = Nocardia → TMP-SMX
> 长期激素 + 肺部结节/空洞**同时有 CNS 占位** → Nocardia（弱抗酸、革兰阳性串珠分支）（Q4417）。

> [!danger] 钥匙 4：移植 <1 月发热脓毒 = 普通院内细菌（不是机会感染）
> 机会性感染窗 1 月后才开；<1 月先查/治院内细菌源（吻合口/导管/吸入）（Q7494）。

> [!tip] CMV vs PCP 快速分流
> | | PCP | CMV |
> |---|---|---|
> | 经典宿主 | HIV CD4<200 / 未预防 | 移植 1–6 月 |
> | 关键钩子 | 运动后血氧骤降、LDH↑、β-D-葡聚糖+ | 累及他器官（结肠炎/视网膜炎）、CMV PCR+ |
> | 在吃 TMP-SMX? | 几乎排除 PCP | 仍可能 |
> | 治疗 | TMP-SMX (±激素) | 更昔洛韦 |

## 六、别漏：非感染性"肺浸润"拟态 (mimics) ⭐

免疫抑制宿主肺浸润**不一定是感染**，鉴别要带上：

- **药物性肺炎 (drug-induced pneumonitis)**：博来霉素 bleomycin、甲氨蝶呤 methotrexate、免疫检查点抑制剂等。
- **放射性肺炎 (radiation pneumonitis)**：照射野内、放疗后数周–数月。
- **弥漫肺泡出血 (DAH, diffuse alveolar hemorrhage)**：血降 + 弥漫磨玻璃 + BAL 渐血性。
- **移植物抗宿主病 (GVHD)**（造血干细胞移植）：肺 = 闭塞性细支气管炎；**植入综合征 / 特发性肺炎综合征 (IPS)**。
- **移植后淋巴增殖病 (PTLD, EBV 驱动)**：肿块/结节 + 淋巴结。
- **肺水肿 / ARDS / 白血病细胞浸润**。

## 七、Memory Hooks

> [!success] 一句话锁定
> - **三步**：缺陷类型（粒缺=假单胞+霉菌 / 细胞免疫=机会感染 / 体液脾补体=有荚膜菌）→ 移植时间线（<1 月细菌 / 1–6 月机会感染 / >6 月社区+迟发）→ 影像（halo=曲霉 / 反晕=毛霉 / 弥漫磨玻璃=PCP·CMV / 空洞=TB·Nocardia）。
> - **在吃 TMP-SMX + 弥漫磨玻璃 → 排 PCP 想 CMV。**
> - **halo+咯血+粒缺=曲霉；反晕+DKA=毛霉；结节+脑脓肿+弱抗酸分支=Nocardia。**
> - **移植 <1 月脓毒 = 院内细菌，不是机会感染。**

## 八、复习题（可折叠 Q&A）

> [!question]- Q1. 肾移植 3 月、规律服 TMP-SMX 的患者，发热 + 干咳 + 双肺弥漫磨玻璃，最可能病原 + 确诊？
> **CMV 肺炎**（在吃 TMP-SMX → PCP 概率极低）。确诊：**CMV PCR / 病毒载量**，活检见"猫头鹰眼"包涵体。治疗更昔洛韦。

> [!question]- Q2. 急性白血病化疗后粒缺，发热 + 胸膜性胸痛 + 咯血，CT 结节伴 halo 征。诊断 + 治疗？
> **侵袭性肺曲霉病**。血清半乳甘露聚糖 + BAL；**伏立康唑 ± 棘白菌素**。

> [!question]- Q3. 长期泼尼松患者，肺多发结节 + 头痛，影像见脑占位；痰部分抗酸、革兰阳性分枝串珠丝状菌。诊断 + 首选药？
> **诺卡菌病 (Nocardia)**；**TMP-SMX**（累 CNS 联合用药）。

> [!question]- Q4. 糖尿病酮症酸中毒患者肺部反晕征 + 鼻窦/眶受累，活检宽大无隔直角分支菌丝。诊断 + 治疗？
> **毛霉病 (mucormycosis)**；**两性霉素 B + 手术清创**。

> [!question]- Q5. HIV 未治、CD4 80，亚急性气短 + 干咳 + 运动后血氧骤降 + LDH 升高，双肺弥漫磨玻璃。诊断 + 治疗（含何时加激素）？
> **PCP（耶氏肺孢子菌肺炎）**；**TMP-SMX**；**PaO₂<70 mmHg 或 A-a 梯度>35 → 加糖皮质激素**。

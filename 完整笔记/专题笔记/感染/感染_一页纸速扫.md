# 感染一页纸速扫

> [!success] 用法
> **感染杂的是知识，不杂的是这一页。** 每天/考前扫 5 分钟，只记"**触发词 → 反射答案**"；要细节点对应双链下钻。
> 感染题 90% 是 pattern 识别——认出钥匙就够，不用背全。

## 1 · 抗生素 场景→药（[[完整笔记/专题笔记/感染/感染_抗生素高频20场景秒杀]]）

```
手术预防     → Cefazolin
门诊蜂窝织炎 → Cephalexin(无脓) / TMP-SMX·Doxy(有脓/MRSA)
社区肺炎住院 → Ceftriaxone + Azithro
脑膜炎       → Vanc + Ceftri (+Amp 若 >50/免疫低/孕妇)
淋病         → Ceftri 500 IM + Doxy
Neutropenic  → Cefepime 单药
HAP/VAP      → Cefepime + Vanc
MRSA         → Vanc
膀胱炎       → Nitro / TMP-SMX / Fosfo
C. diff      → Fidax / Vanc PO（不用 metro 首选）
Latent TB    → 4R / 3HP / 9H ；Active TB → RIPE
动物咬伤     → Amox-Clav（猫咬必给）
百日咳 PEP   → Azithro（全家，不论疫苗史）
```

> [!danger] 致死 5（看触发词立即排除）
> 新生儿+Ceftriaxone=死（用 Cefotaxime）｜氨基糖苷+Lasix=聋｜FQ+老人/类固醇=肌腱断｜TMP-SMX+ACEi/MRA=高钾炸弹｜Vanc 快推=Red man（**非过敏**，慢推+苯海拉明）

## 2 · AIDS 机会性感染 看 CD4（[[完整笔记/专题笔记/感染/感染_AIDS机会性感染按CD4谱]]）

```
CD4: 500黏膜皮肤 | 200 PCP(起TMP-SMX预防) | 150组织胞浆菌
     100 脑+食管(Toxo/隐球菌/食管Candida) | 50 最深坑(MAC/CMV/淋巴瘤/PML)
腹泻: 血便→CMV  |  水样+高热→MAC  |  重度水样→Crypto
CNS:  多发环形→Toxo | 单发→淋巴瘤 | 不强化白质→PML | 脑膜+CrAg→隐球菌
食管: 白斑→Candida(先经验氟康唑) | 大线溃疡→CMV | 小簇溃疡→HSV
陷阱: pyrimethamine=Toxoplasma(不是MAC) ; MAC=大环内酯+ethambutol
```

## 3 · 蜱传病（[[完整笔记/专题笔记/感染/感染_蜱传疾病鉴别]]）

```
溶血+无脾        → Babesia → 血涂片 Maltese cross → atovaquone+azithro
白↓板↓LFT↑无疹   → 埃立克(单核morulae)/无形体(粒细胞morulae) → doxy
疹腕踝→掌跖向心  → RMSF → doxy（疑诊即起）
EM 游走红斑      → Lyme → doxy（神经/心脏 ceftriaxone）
皮肤溃疡+淋巴结+兔 → 兔热病 → 氨基糖苷
铁律: 蜱传一律 doxy；唯 Babesia + Lyme播散 例外
```

## 4 · 脑膜炎 CSF（[[完整笔记/专题笔记/感染/感染_脑膜炎CSF鉴别]]）

```
先看糖!
  糖低 → 细菌(中性,WBC极高) / TB(淋巴,数周前驱) / 隐球菌(AIDS,CrAg)
  糖正常 → 病毒(肠病毒/HSV) / 急性HIV
钥匙: 瘀点紫癜→脑膜炎球菌→DIC查纤维蛋白原
      斑丘疹+全身LAD+肠病毒阴→急性HIV(查RNA)
      蛋白↑↑细胞正常→GBS(不是脑膜炎)
```

## 5 · 梅毒血清学（[[完整笔记/专题笔记/感染/感染_梅毒分期诊疗与血清学]]）

```
非螺旋体 VDRL/RPR = 定量·不特异·可转阴 → 筛查 + 监测疗效
螺旋体 FTA-ABS    = 定性·特异·终身阳 → 确诊
非阳+螺阴 = 假阳性(孕/IVDU/肝/HIV/自免)
非阴+螺阳 = 既往已治疗(最常见)
治疗: 任何期 PCN G；孕妇过敏→脱敏；打针后几小时发热=Jarisch-Herxheimer(继续打)
```

## 6 · 生殖器溃疡（[[完整笔记/专题笔记/感染/感染_生殖器溃疡鉴别]]）

```
无痛硬+洁净底+无痛淋巴结 = 梅毒(chancre)
剧痛软+脏污+化脓buboes   = 软下疳(ducreyi→you cry)
多发痛性簇水疱           = HSV
一过小疹→痛性buboes+groove sign = LGV
牛肉红易出血+无淋巴结    = 腹股沟肉芽肿(Donovan小体)
```

## 7 · STI 分泌物（[[完整笔记/专题笔记/感染/感染_STI分泌物综合征鉴别]]）

```
阴道炎看 pH:
  pH正常+豆腐渣+痒 = 念珠菌(非STI)
  pH↑+clue cells+whiff+不红 = BV(非STI)
  pH↑+泡沫黄绿+草莓宫颈 = Trichomonas(STI→治伴侣)
宫颈/尿道: 脓性急=淋病(Ceftri500+doxy) | 黏液脓/无症状=衣原体(doxy)
  性活跃女+无菌脓尿 = 尿道炎(非普通UTI)
并发: 淋病播散DGI(迁徙关节+腱鞘+脓疱) | 衣原体反应性关节炎(看不了+尿不了+爬不了)
```

## 8 · 咽炎 / GAS（[[完整笔记/专题笔记/感染/感染_咽炎鉴别与GAS诊治]]）

```
无病毒症状+前颈+渗出 = GAS → penicillin
青少年+后颈+脾大+嗜异性⊕ = EBV mono(避amox/避运动)
有咳嗽鼻涕 = 病毒 → 支持
流程: 儿童 RADT 阴 → 咽培养(ARF风险) ；成人不必
并发: ARF(只咽炎,能防) | PSGN(咽or皮,不能防)
```

## 9 · 儿童发热出疹（[[完整笔记/专题笔记/儿科/儿科_发热出疹病鉴别]]）

```
时间钥匙: 热中出疹 vs 退热后出疹(=Roseola)
砂纸+皮褶+草莓舌 = 猩红热(GAS→penicillin)
掌掴颊+网状     = 第五病(parvovirus B19)
退热后玫瑰疹    = 幼儿急疹(HHV-6)
Koplik+3C       = 麻疹
手足+口溃疡     = HFMD(Coxsackie)
各期并存        = 水痘
≥5天热+4项      = Kawasaki
```

# STI 分泌物综合征鉴别

> [!info] 笔记定位
> **非溃疡性 STI / 分泌物综合征的"两层鉴别"**（与姊妹篇 [[完整笔记/专题笔记/感染/感染_生殖器溃疡鉴别]] 互补：溃疡 vs 分泌物）。
> 第一层：**阴道炎三联**（BV / Trichomonas / 念珠菌）—— 靠 **pH + whiff + 镜检**秒分。
> 第二层：**宫颈炎/尿道炎**（淋病 GC / 衣原体 CT）—— 上行 → **PID**、播散 → **DGI / 反应性关节炎 / 新生儿**。
> Step 2 高频："给分泌物性状 + pH + 镜检 → 选病原/选药"。

---

## 一、阴道炎三联鉴别 ⭐⭐⭐（必背）

| | **细菌性阴道病 BV** | **Trichomonas（滴虫）** | **念珠菌 Candida** |
|---|---|---|---|
| 病原 | Gardnerella（菌群失衡、乳酸杆菌↓，**非 STI**）| **Trichomonas vaginalis**（原虫，**是 STI**）| **Candida albicans**（真菌，**非 STI**）|
| 分泌物 | **稀薄灰白**、量多 | **黄绿、泡沫状、恶臭** | **白色稠厚、豆腐渣/cottage cheese** |
| 气味 | **鱼腥味** | 恶臭 | 无明显异味 |
| **pH** | **>4.5** | **>4.5** | **正常 <4.5** ⭐ |
| 瘙痒/炎症 | **无炎症**（不红不痒为主）| 瘙痒 + 红 + dysuria | **明显瘙痒 + 红肿** |
| **镜检** | **Clue cells**（线索细胞）| **活动带鞭毛滋养体**（wet mount）| **假菌丝/芽生孢子**（KOH）|
| 特征体征 | **Whiff test ⊕**（KOH 鱼腥）| **草莓宫颈 strawberry cervix**（点状出血）| 外阴红斑 + 卫星灶 |
| 治疗 | **Metronidazole** | **Metronidazole**（**伴侣同治**）| **Fluconazole** / 局部唑类 |

> [!success] 三联秒杀钥匙
> - **pH 正常（<4.5）+ 豆腐渣 + 痒 = 念珠菌**（唯一不升 pH）
> - **pH↑ + clue cells + whiff⊕ + 不痒不红 = BV**
> - **pH↑ + 泡沫黄绿 + 草莓宫颈 + 活动Trichomonas = Trichomonas**

> [!danger] STI 与否决定"治不治伴侣"
> - **Trichomonas = STI → 伴侣同治 + 查其他 STI**
> - **BV / 念珠菌 ≠ STI → 不治伴侣**（BV 是菌群失衡；Candida 诱因 DM/抗生素/孕/免疫低）
> - 念珠菌诱因（[[完整笔记/Peixuan分科笔记/endocrine]] DM / 抗生素 / 孕期 / 免疫抑制）

> [!tip] 诊断标准小补丁
> - **BV — Amsel 标准**（≥3/4）：稀薄白带 + clue cells + pH>4.5 + whiff⊕
> - Trichomonas wet mount 敏感性一般，**NAAT 最敏感**；孕期Trichomonas与早产/PROM 相关
> - 念珠菌 pH 正常是与另两个最快的区分点

## 二、宫颈炎 / 尿道炎（GC vs CT）⭐⭐⭐

| | **淋病 Gonorrhea** | **衣原体 Chlamydia** |
|---|---|---|
| 病原 | *Neisseria gonorrhoeae*（G− 双球菌）| *Chlamydia trachomatis* D-K（细胞内）|
| 分泌物 | **脓性、量多**、急 | **黏液脓性**、常**无症状/轻** |
| 镜检/诊断 | G− 双球菌；**NAAT** | **NAAT**（细胞内、不易培养）|
| 治疗 | **Ceftriaxone 500 mg IM ×1** | **Doxycycline** ×7d（孕期 → azithromycin）|
| **联用** | **常规同时治衣原体**（共感染常见）| — |
| 男性 | 尿道炎（脓性）+ **附睾炎/前列腺炎(prostatitis)** | **非淋菌性尿道炎 NGU 最常见因** + **附睾炎(epididymitis)** |

> [!warning] 关键临床要点
> - **性活跃女性 + 尿痛 + 无菌性脓尿（WBC⊕ / 培养− / 亚硝酸盐−）= 尿道炎（CT/GC）**，不是普通膀胱炎 → 查 NAAT，别只按 UTI 治。
> - 淋病治疗 **2021 后单用 ceftriaxone 500 IM**（不再 azithro 联用），但**若未排除衣原体仍加 doxycycline**。
> - 见 [[完整笔记/专题笔记/感染/感染_抗生素高频20场景秒杀]] §二 #6（淋病）/ #3（PID）。

> [!tip] 附睾炎 Epididymitis 的年龄分水岭 ⭐（FA 增量）
> 阴囊痛 + 附睾压痛 + **Prehn 征⊕**（抬高阴囊**疼痛缓解**）→ 附睾炎；病原**按年龄/性史二分**：
> - **<35 岁 + 性活跃 = 衣原体 Chlamydia / 淋病 Gonorrhea**（CT 最常见）→ **Ceftriaxone 头孢曲松 IM + Doxycycline 多西环素**（与宫颈炎/尿道炎同方案）
> - **>35 岁 / 非性活跃 / 留置导尿/器械 = 肠道革兰阴性菌（E. coli 大肠埃希菌等 coliforms，同 UTI 尿路感染病原）→ Fluoroquinolone 氟喹诺酮**（如 levofloxacin 左氧氟沙星）
> - ⚠️ **必须排除睾丸扭转（testicular torsion）**：扭转 = **急骤、提睾反射消失、Prehn 征−（抬高不缓解）**，是**外科急症**（多普勒超声血流↓）；附睾炎起病较缓、提睾反射存在、Prehn⊕。别把扭转误当附睾炎用抗生素拖延。

## 三、上行 + 播散并发症 ⭐

```
宫颈炎（GC/CT）
   │ 上行
   ▼
PID（盆腔炎）── Fitz-Hugh-Curtis（肝周炎，RUQ 痛 + 肝包膜"琴弦"粘连）
   │                              └─ 远期：不孕 / 异位妊娠 / 慢性盆腔痛
   │ 播散
   ├─ 淋病 → DGI（播散性淋球菌感染）
   │     两型：① 关节炎-皮炎综合征（迁徙性多关节痛 + 腱鞘炎 + 脓疱疹）
   │           ② 化脓性单关节炎（滑液 WBC 常 >50,000）
   └─ 衣原体 → 反应性关节炎（Reactive arthritis）
         "不能看、不能尿、不能爬"= 结膜炎 + 尿道炎 + 关节炎（HLA-B27）
```

> [!info] 与假体关节感染对照（接你 [[mistakes/uworld-mistakes_2026-06#^Q9111]]）
> Q9111 的干扰项正是这些：**Neisseria gonorrhoeae** = DGI（迁徙性多关节 + 腱鞘炎 + 皮疹；化脓性单关节滑液 >50,000）；**Chlamydia / Salmonella** = 反应性关节炎（非化脓、HLA-B27）。这些都**不是**假体关节感染（PJI = 低毒力 coagulase-negative staph）。

> [!tip] PID 治疗（住院经验）
> **Cefoxitin/Cefotetan IV + Doxycycline**（覆盖厌氧）；门诊轻症 → Ceftriaxone IM + Doxy PO ± Metronidazole。详见抗生素秒杀 §二 #3。

## 四、新生儿相关（垂直传播）

| 病原 | 新生儿病 | 时间 / 处理 |
|---|---|---|
| **淋病 GC** | 新生儿结膜炎（**超急性、化脓**，可穿孔）| 生后 **2-5 天**；产时获得；**预防 = erythromycin眼膏** |
| **衣原体 CT** | 新生儿结膜炎 + **衣原体肺炎**（staccato cough，无热，嗜酸↑）| 结膜炎 **5-14 天**；肺炎 **数周**；治 **口服 erythromycin/azithromycin**（眼膏不防 CT）|

> [!success] 新生儿结膜炎按时间钟
> 生后 <24h = 化学性（硝酸银）；**2-5d = 淋球菌**（最凶）；**5-14d = 衣原体**。

## 五、Memory Hook 总集

```
阴道炎三联（pH 是分水岭）
  pH 正常 + 豆腐渣 + 痒 = 念珠菌（fluconazole，非STI）
  pH↑ + clue cells + whiff⊕ + 不红 = BV（metro，非STI）
  pH↑ + 泡沫黄绿 + 草莓宫颈 + 活动Trichomonas = Trichomonas（metro，STI→治伴侣）

宫颈炎/尿道炎
  脓性急 = 淋病 → ceftriaxone 500 IM（+doxy 盖衣原体）
  黏液脓/无症状 = 衣原体 → doxycycline（孕→azithro）
  性活跃女 + 无菌性脓尿 = 尿道炎(CT/GC)，不是普通UTI
  附睾炎 <35性活跃=CT/GC(ceftriaxone+doxy)；>35=肠道菌(fluoroquinolone)；先排扭转(Prehn−/提睾消失)

并发症
  上行 = PID（→不孕/异位/Fitz-Hugh-Curtis）
  淋病播散 = DGI（迁徙关节+腱鞘+脓疱 / 化脓单关节>50k）
  衣原体 = 反应性关节炎（看不了+尿不了+爬不了，HLA-B27）

新生儿结膜炎钟
  <24h 化学；2-5d 淋球菌（erythromycin眼膏预防）；5-14d 衣原体(+肺炎，口服erythromycin)
```

# 蜱传疾病鉴别

> [!info] 笔记定位
> **蜱传发热病的"一表定病原"。**
> Step 2 CK 蜱传题套路固定：**蜱咬史 + 发热 + 一条实验室/皮损招牌 → 选病原/选诊断/选药**。
> 一条主钥匙：**看血象——溶血（Babesia）vs 减少 cytopenia（埃立克/无形体）vs 皮疹（RMSF/Lyme）**。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q3247]]（埃立克体）+ [[mistakes/uworld-mistakes_2026-06#^Q3104]]（Babesia）。

---

## 一、两步定位法

```
Step 1：血象/皮损招牌？
  血管内溶血（贫血+网织↑+LDH↑+深色尿+黄疸）→ Babesia
  白↓ + 板↓ + 转氨酶↑（无皮疹）           → 埃立克 / 无形体
  皮疹（腕踝→掌跖→向心）                   → RMSF
  游走性红斑 EM（环形扩大中心清）           → Lyme

Step 2：地区/蜱佐证
  东北（Ixodes）：Babesia / 无形体 / Lyme（常共感染）
  东南·中南（Amblyomma=Lone Star）：埃立克
  东南·中南（Dermacentor）：RMSF
```

> [!tip] 共感染（高频）
> **Ixodes scapularis 同时传 Babesia + Anaplasma + Lyme** → 三者**共感染常见**；一个东北蜱咬患者可同时中招。

## 二、蜱传 6 病鉴别总表 ⭐⭐⭐

| 病 | 病原 | 蜱 / 地区 | 招牌 | 诊断 | 治疗 |
|---|---|---|---|---|---|
| **Babesiosis（巴贝虫病）** | *Babesia microti*（原虫 protozoa）| **Ixodes** / 东北 | **血管内溶血** + **无脾→重症** | 血涂片 **Maltese cross / 红细胞内环** | **Atovaquone + azithromycin**（重症 quinine + clinda）|
| **埃立克 Ehrlichiosis** | *E. chaffeensis* | **Amblyomma** / 东南·中南 | **无皮疹**（"RMSF without spots"）+ 白↓板↓LFT↑ | 单核细胞 **morulae** / PCR | **Doxycycline**（经验性）|
| **无形体 Anaplasmosis** | *A. phagocytophilum* | **Ixodes** / 东北·中西北 | 无皮疹 + 白↓板↓LFT↑ | **粒细胞** morulae / PCR | **Doxycycline** |
| **落基山斑疹热 RMSF** | *Rickettsia rickettsii* | **Dermacentor** / 东南·中南 | **皮疹腕踝→掌跖→向心**（90%）+ 低钠 | 临床 + 血清学（回顾）| **Doxycycline**（疑诊即起，儿童也用）|
| **Lyme** | *Borrelia burgdorferi* | **Ixodes** / 东北 | **游走性红斑 EM**；晚期神经/心脏/关节 | 血清学（两步 ELISA→WB）| 早期 EM **Doxy**；神经/心脏 **Ceftriaxone** |
| **兔热病 Tularemia** | *Francisella tularensis* | Dermacentor / 兔接触 | **溃疡腺型**（皮肤溃疡 + 区域淋巴结）| 血清学（高危培养）| **Streptomycin / gentamicin** |

> [!success] 一句话钥匙
> - **溶血 + 无脾 = Babesia**（血涂片 Maltese cross）
> - **无皮疹 + 白↓板↓ = 埃立克/无形体**（morulae；单核=埃立克 / 粒细胞=无形体）
> - **皮疹掌跖向心 = RMSF**
> - **EM 游走红斑 = Lyme**（神经/心脏才 ceftriaxone）
> - **皮肤溃疡 + 淋巴结 + 兔 = 兔热病**（氨基糖苷）

## 二·五、落基山斑疹热 RMSF 专项 ⭐⭐⭐（致命，疑诊即治）

> [!danger] 机制 = 攻击血管内皮 → 全身毛细血管渗漏
> **立克次体（*Rickettsia rickettsii*，G⁻ 专性胞内菌）侵袭血管内皮细胞** → **进行性毛细血管通透性↑ → 液体外渗 → 低血容量 → 灌注不足/休克 + 多器官损伤**。所有表现（疹/低钠/血小板↓/肺水肿/休克）都是**这一条血管损伤主线**的下游。

> [!info] 时间线 + 皮疹演变（高频陷阱多）
> - **第 1 阶段（3-4 天）**：**非特异病毒样**——发热、剧烈头痛、肌痛、恶心呕吐（**像普通病毒病** → 早期易漏诊）。
> - **第 3-5 天出疹**：**斑丘疹起于腕/踝 → 累及掌跖（palms/soles）→ 向心蔓延（向躯干）→ 逐渐变瘀点（petechial）**。
> - ⚠️ **陷阱**：① **疹初为斑丘疹、可尚未累及掌跖** → 别因"没到掌跖"排除；② **常不记得蜱咬史**；③ 早期甚至无疹（"spotless RMSF"）→ **别等典型疹才治**。

> [!warning] 实验室三件套 + 儿童专属陷阱
> - **血小板↓（thrombocytopenia）**：血管内血小板破坏消耗。
> - **低钠（hyponatremia）**：低血容量 → **ADH↑**（抗利尿激素）。
> - **转氨酶↑（AST/ALT）**：肝血管内皮损伤（晚期）。
> - **白细胞常正常**（可白↓或白↑，不定）。
> - **⭐ 儿童 RMSF：剧烈腹痛**——可被误当**阑尾炎 / 肠梗阻**（高频干扰）。

> [!danger] 与脑膜炎球菌脑膜炎鉴别（都"瘀点疹 + 发热 + 脓毒症"）
> 意识改变 + 瘀点疹时常做 LP 排脑膜炎球菌：
> | | **RMSF** | **脑膜炎球菌脑膜炎** |
> |---|---|---|
> | **CSF 白细胞** | **轻度↑（<100/mm³）** | **显著↑（>1000/mm³）** |
> | **CSF 蛋白** | 轻微↑ | 高 |
> | **CSF 糖** | **正常** | **低** |
> 两者都要**经验性覆盖**（RMSF→doxy；脑膜炎球菌→ceftriaxone），不等结果。

> [!success] 处理铁律
> **疑 RMSF → 立即经验性多西环素（doxycycline），不等血清学确诊**（确诊靠血清学**回顾性** / 皮肤活检）。
> - **儿童也用 doxy**：短疗程（5-7 天）**牙染色风险极小 ≪ 不治的死亡风险** → 仍是首选（不是大环内酯/氯霉素）。
> - 休克者 + **静脉补液**支持。
> - 反射：**发热头痛 + 疹腕踝→掌跖向心（或儿童剧烈腹痛）+ 板↓低钠 = RMSF → doxy 别等**。

## 三、治疗主线 ⭐

> [!danger] 一律 doxycycline，除两个例外
> **埃立克 / 无形体 / RMSF / 早期 Lyme → Doxycycline**（RMSF/埃立克**疑诊即经验性起、不等确诊**，儿童也用——不治死风险 > 牙黄风险）。
> **两个例外**：
> - **Babesia = atovaquone + azithromycin**（原虫，doxy 无效）
> - **Lyme 播散（神经/心脏）= ceftriaxone**（早期 EM 仍 doxy）

> [!tip] Atovaquone（阿托伐醌）三用途 ⭐
> 抗原虫/真菌药，**抑制线粒体电子传递（cytochrome bc1 / 复合体 III）→ 阻断嘧啶合成**；口服、**高脂餐增吸收**、耐受好。
> - **巴贝虫 Babesia**：**atovaquone + azithromycin**（本病轻症一线）
> - **PCP（卡氏肺孢菌）**：**磺胺过敏 / G6PD 缺乏的替代**（一线仍 TMP-SMX）
> - **疟疾 Malaria**：**atovaquone-proguanil（Malarone）** 预防 + 治疗
> 记忆："**原虫线粒体毒 bc1：巴贝虫(+azithro) / PCP(磺胺替代) / 疟疾(Malarone)**"。

## 四、Memory Hook

```
血象/皮损定病原
  溶血+无脾        → Babesia → 血涂片 Maltese cross → atovaquone+azithro
  白↓板↓LFT↑无疹   → 埃立克(单核morulae)/无形体(粒细胞morulae) → doxy
  疹腕踝→掌跖向心(变瘀点)/儿童剧烈腹痛+板↓低钠 → RMSF → doxy（疑诊即起,儿童也用）
  EM游走红斑       → Lyme → doxy（神经/心脏 ceftriaxone）
  皮肤溃疡+淋巴结+兔 → 兔热病 → 氨基糖苷

铁律
  蜱传一律 doxy；唯Babesia(atovaquone+azithro) + Lyme播散(ceftriaxone) 例外
  Ixodes 一蜱三毒：Babesia + Anaplasma + Lyme 共感染
```

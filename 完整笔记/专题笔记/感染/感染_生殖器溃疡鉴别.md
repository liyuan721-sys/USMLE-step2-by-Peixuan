# 生殖器溃疡鉴别

> [!info] 笔记定位
> **生殖器溃疡（genital ulcer disease, GUD）的"两轴秒杀"。**
> Step 2 CK 几乎只考一件事：**给溃疡形态 + 淋巴结特点 → 选病原 / 选药**。
> 两轴定位：① **痛 vs 无痛** ② **淋巴结：痛性化脓（buboes）vs 无痛 vs 无肿大**。两轴一交叉 → 病原唯一。
> 相关错题：[[mistakes/uworld-mistakes_2026-06#^Q20553]]（梅毒血清学）。详见 [[完整笔记/专题笔记/感染/感染_梅毒分期诊疗与血清学]]。

---

## 一、两轴定位法 ⭐⭐⭐

```
轴一：溃疡痛不痛？
  无痛 → 梅毒（硬、洁净底）/ LGV 原发疹（一过常漏）/ 腹股沟肉芽肿（牛肉红、易出血）
  痛   → HSV（多发小水疱/簇）/ 软下疳（软、脏污底、边缘潜行）

轴二：淋巴结？
  无痛 → 梅毒
  痛性、化脓、可破溃（buboes） → 软下疳 / LGV（groove sign 沟槽征）
  双侧痛性 → HSV（初发常伴全身症状）
  无真性肿大（假性 pseudobuboes = 皮下肉芽肿）→ 腹股沟肉芽肿
```

> [!success] 一句话钥匙
> - **无痛溃疡 + 无痛淋巴结 = 梅毒（chancre）**
> - **痛溃疡 + 痛性化脓淋巴结 = 软下疳（"ducreyi → you cry"）**
> - **多发痛性小水疱/簇 = HSV**
> - **一过小疹 → 痛性腹股沟 buboes + groove sign = LGV**
> - **无痛牛肉红、易出血、无淋巴结 = 腹股沟肉芽肿（Donovanosis）**

## 二、五病鉴别总表 ⭐⭐⭐（必背）

| 病 | 病原 | 溃疡痛 | 溃疡形态 | 淋巴结 | 诊断 | 治疗 |
|---|---|---|---|---|---|---|
| **梅毒（一期）** | *Treponema pallidum*（螺旋体）| **无痛** | **硬下疳**：质硬、边界清、**基底洁净** | **无痛**、橡皮样 | 暗视野 / 血清学（[[完整笔记/专题笔记/感染/感染_梅毒分期诊疗与血清学]]）| **Benzathine penicillin G** IM |
| **软下疳 Chancroid** | *Haemophilus ducreyi*（G− 球杆菌）| **剧痛** | **软**、边缘潜行/参差、**脓性污秽底** | **痛性、化脓、可破溃（buboes）** | 临床 + 培养（"school of fish"）| **Azithromycin** PO ×1 / **Ceftriaxone** IM ×1 |
| **生殖器疱疹 HSV** | HSV-2 > HSV-1 | **痛** | **多发、簇状小水疱 → 浅溃疡** | **双侧痛性**、初发伴发热 | **PCR**（首选）/ Tzanck（多核巨细胞）| **Acyclovir / Valacyclovir** |
| **LGV（Lymphogranuloma venereum 性病性淋巴肉芽肿）** | *Chlamydia trachomatis* **L1-L3** | 原发**无痛**小疹（一过、常漏）| 小丘疹/溃疡，自愈 | **痛性腹股沟 buboes + groove sign 沟槽征**；可直肠炎 | NAAT | **Doxycycline** ×21d |
| **腹股沟肉芽肿（Donovanosis）** | *Klebsiella granulomatis* | **无痛** | **牛肉红、肉芽样、易出血、进行性** | **无真性肿大**（pseudobuboes 皮下肉芽）| 活检 **Donovan 小体** | **Doxycycline**（≥3 周至愈合）|

> [!danger] 最高频的两组对照
> - **梅毒 vs 软下疳**：无痛硬洁净（梅毒）↔ **剧痛软脏污 + 化脓淋巴结**（软下疳）。**"ducreyi → you cry"** 记痛。
> - **软下疳 vs LGV**（都有痛性腹股沟肿大）：软下疳是**溃疡本身痛 + 淋巴结化脓**；LGV 是**原发疹无痛/已消** + 之后才出**痛性 buboes + groove sign**（腹股沟韧带上下分隔成"沟槽"）。

## 三、各病加分细节

> [!tip] 梅毒
> 无痛 + 自愈（即使不治下疳也消，易误以为好了）。下疳期 RPR 可能尚未转阳 → **暗视野**或复查。详见专题笔记。

> [!tip] 软下疳 Chancroid
> 热带/资源有限地区多见；**HIV 共感染常见**——见生殖器痛性溃疡要顺手查 HIV/梅毒。革兰染色"**school of fish / railroad track**"（链状排列）。

> [!tip] HSV
> 初发最重（多发痛性簇状水疱 + 发热 + 双侧痛性淋巴结），复发渐轻、有前驱刺痛。诊断首选 **PCR**（比 Tzanck 敏感特异）。无根治，发作期/抑制疗法用 acyclovir 类。

> [!tip] LGV
> **三期演变**：① 一过无痛小疹（常漏）② **痛性腹股沟/股淋巴结 buboes + groove sign**③ 慢性 → 直肠结肠炎/瘘/狭窄（MSM 直肠 LGV）。治 **doxycycline 21 天**（比衣原体常规 7 天长）。

> [!tip] 腹股沟肉芽肿 / Donovanosis
> **无痛、牛肉红（beefy-red）、易出血、缓慢扩大**；**无真性淋巴结肿大**（pseudobuboes = 皮下肉芽肿块，非淋巴结）。活检见胞浆内 **Donovan 小体**。热带地区，美国罕见。

## 四、非 STI / 鉴别提醒

> [!warning] 别忘非感染性溃疡
> - **Behçet 白塞病**：**复发性、痛性**口腔 + 生殖器**无菌**阿弗他溃疡 + 葡萄膜炎 + pathergy；无病原、不靠抗生素 → 见 [[完整笔记/Peixuan分科笔记/免疫，疫苗]] / 风湿
> - **固定性药疹（fixed drug eruption）**：同一部位反复、有用药史
> - **外伤 / 拉链伤 / 阿弗他溃疡**：病史导向

## 五、Memory Hook 总集

```
两轴
  痛？   无痛=梅毒/LGV原发/Donovan；痛=HSV/软下疳
  淋巴？ 无痛=梅毒；痛性化脓buboes=软下疳/LGV；双侧痛=HSV；无肿(假buboes)=Donovan

五病钥匙
  梅毒    无痛硬洁净 + 无痛淋巴结 → Benzathine PCN
  软下疳  剧痛软脏 + 化脓buboes（you cry）→ azithro/ceftriaxone
  HSV     多发痛性簇水疱 → acyclovir（PCR诊断）
  LGV     一过小疹→痛性buboes+groove sign → doxy 21d
  Donovan 牛肉红易出血、无淋巴结、Donovan小体 → doxy

诊断招牌
  暗视野/血清 = 梅毒
  school of fish = 软下疳
  Tzanck 多核巨细胞 / PCR = HSV
  groove sign = LGV
  Donovan 小体 = 腹股沟肉芽肿
```

# 感染性腹泻：血性 vs 水样 + 病原 + 何时用抗生素

> [!info] 笔记定位
> 感染性腹泻题三步：① **血性 vs 水样（watery）** → ② **暴露/宿主/潜伏期 + 并发症定病原** → ③ **要不要抗生素（多数自限；STEC 禁）**。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q12458]]（志贺·儿童惊厥）+ [[mistakes/uworld-mistakes_2026-06#^Q17637]]（轻症弯曲菌·仅支持）+ [[mistakes/uworld-mistakes_2026-06#^Q105673]]（病毒·非血性 ORS）+ [[mistakes/uworld-mistakes_2026-06#^Q3917]]（CMV·AIDS 血性）+ [[mistakes/uworld-mistakes_2026-06#^Q105597]]（志贺·炎症性血性腹泻 / MSM 肠炎 vs 直肠炎·✅做对补鉴别）。

---

## 一、第一步：血性 vs 水样（watery）⭐⭐⭐

```
急性腹泻
├─ 水样（watery，分泌性/渗透性/病毒）→ 多自限 → 补液为主
│    病毒(诺如/轮状)⭐最常见 / ETEC(旅行) / 霍乱弧菌 /
│    金葡·蜡样·产气荚膜(食物中毒毒素) / 贾第虫·隐孢子(迁延)
└─ 血性/黏液（炎症性/侵袭性）→ 按"暴露 + 并发症"定病原
     STEC(O157) / 志贺 Shigella / 弯曲菌 Campylobacter /
     非伤寒沙门 / 阿米巴 / EIEC / 艰难梭菌 C. diff
```

> [!tip] 一眼分流
> **水样 + 无高热/无里急后重** → 多病毒/毒素 → 补液（多不查不给药）。
> **血便/黏液 + 高热/里急后重/腹绞痛** → 侵袭性 → 按暴露+并发症定病原（见 §三）。

## 二、水样（非血性）泻 ⭐⭐⭐（新增）

### 2.1 病原速查（暴露 → 线索）

| 病原 | 暴露/史 | 招牌 | 处理 |
|---|---|---|---|
| **诺如病毒 norovirus** ⭐ | 邮轮/聚集/家庭暴发，各年龄**最常见** | 呕吐 + 水样，24–48h 自限 | 补液 |
| **轮状病毒 rotavirus** | **≤2 岁未接种**、冬季 | 婴幼儿水样泻 + 脱水 | ORS；疫苗预防 |
| **产肠毒素大肠杆菌 ETEC** | **旅行者腹泻 #1**（发展中国家） | 水样，无血无高热（LT/ST 毒素↑cAMP/cGMP） | 补液 ±（中重度）阿奇霉素 |
| **霍乱弧菌 Vibrio cholerae** | 污染水、流行区 | **米泔水样便（rice-water）**大量 → 重度脱水/低钾 | **积极补液（ORS 靠钠-葡萄糖共转运 SGLT1）** ± 多西环素 |
| **金黄色葡萄球菌 S. aureus** | **预成毒素**：蛋黄酱/奶/肉室温放置 | **1–6h 骤起呕吐**为主 | 支持（毒素病，无需抗生素） |
| **蜡样芽孢杆菌 Bacillus cereus** | **炒饭/复热米饭**（呕吐型）/ 肉菜（腹泻型） | 呕吐型 1–6h；腹泻型 8–16h | 支持 |
| **产气荚膜梭菌 C. perfringens** | **加热不当的肉/肉汁、自助餐** | **8–16h** 水样泻 + 绞痛，无呕吐为主 | 支持 |
| **副溶血弧菌 V. parahaemolyticus** | **生海鲜/牡蛎** | 水样（偶血性） | 支持 |
| **贾第虫 Giardia** | **野营/井水/溪水**、男男性行为 | **脂肪泻 + 恶臭 + 胀嗝**，可迁延数周 | 甲硝唑（metronidazole）/替硝唑 / 硝唑尼特 |
| **隐孢子虫 Cryptosporidium** | **AIDS（CD4<100）**、水源 | 慢性大量水样泻 | 硝唑尼特（nitazoxanide）+ **恢复免疫（ART）** |
| **环孢子虫 Cyclospora** | **进口浆果/生菜** | 迁延水样泻 | **复方磺胺甲噁唑（TMP-SMX）** |

### 2.2 食物中毒按潜伏期（高频秒杀）⭐

| 潜伏期 | 机制 | 病原 | 食物 | 主征 |
|---|---|---|---|---|
| **1–6 h** | **预成毒素 preformed** | **金葡 / 蜡样（呕吐型）** | 蛋黄酱·奶·肉 / **炒饭** | **呕吐为主** |
| **8–16 h** | 体内产毒 | **C. perfringens / 蜡样（腹泻型）** | 加热不当肉·肉汁 / 米饭菜 | **水样泻** |
| **>16 h** | 感染/产毒 | ETEC / 病毒 / 弧菌 | 旅行水食 / 海鲜 | 水样泻 |

> [!tip] 潜伏期口诀
> **"吐得快（1–6h）= 预成毒素（金葡/蜡样炒饭）；泻得稍慢（8–16h）= 产气荚膜/蜡样腹泻型；更久 = ETEC/病毒/弧菌"**。侵袭性血性（志贺/弯曲/沙门/STEC）潜伏期更长（数天）→ 见 §三。

### 2.3 旅行者腹泻（traveler's diarrhea）

- **首位 = ETEC**（其次弯曲/志贺/诺如）；多**水样自限**。
- **处理**：补液为主；**中重度 → 阿奇霉素（azithromycin）**（东南亚弯曲菌对氟喹诺酮耐药高，首选阿奇）；**loperamide（洛哌丁胺）** 仅在**无血便、无高热**时辅助。
- **预防**：饮食卫生（"煮沸、削皮、不吃生冷"）；高危者 rifaximin / 铋剂。

> [!warning] 水样泻处理大原则
> **核心是补液**（多数 2–3 天自限）：轻中度 **ORS**、重度 **IV**。**ORS 即使在霍乱也有效**（毒素不破坏钠-葡萄糖共转运）。**特异抗微生物**只用于：霍乱重症、贾第虫/隐孢子/环孢子、旅行者中重度。

### 2.4 海洋生物毒素（marine biotoxins）——胃肠 + 神经/过敏样 ⭐

> [!warning] 两个高频"吃海鲜后"毒素综合征（非感染、不需抗生素）
> 这两类不是病原感染，而是**预成毒素 (preformed toxin)**，潜伏期短、支持治疗为主，常和肠炎/过敏混淆。
>
> | 综合征 | 毒素/来源 | 时间轴 | 招牌表现 | 处理 |
> |---|---|---|---|---|
> | **雪卡毒中毒 (ciguatera poisoning)** | ciguatoxin，富集于**大型珊瑚礁鱼 (reef fish)**：grouper 石斑鱼 / barracuda 梭鱼 / snapper 红鲷 / moray eel 海鳗 | 进食后数分钟–数小时先 GI，**神经症状持续数周–数月** | **先恶心呕吐腹泻 → 口周四肢感觉异常 (paresthesia) + 冷热感觉倒错 (cold allodynia / temperature reversal「冷水摸着像烫」)**；可有反射异常、心动过缓/低血压 | 支持治疗（补液+对症）；恢复期避鱼/酒/坚果（可诱发复发） |
> | **组胺鱼中毒 (scombroid poisoning)** | 鱼肉储存不当致细菌分解组氨酸生**组胺 (histamine)**，常见金枪鱼 tuna / 鲭鱼 mackerel | **进食后数分钟**（极快） | **潮红 flushing + 头痛 + 荨麻疹 + 心悸 + 腹泻**，酷似 IgE 过敏但是组胺直接作用 | **抗组胺药 (antihistamines)**；多自限 |

> [!tip] 一眼分流
> - **「reef fish + 冷热颠倒/口周麻」→ ciguatera 雪卡毒**（神经为主、持续久）
> - **「金枪鱼/鲭鱼放久 + 立刻潮红荨麻疹」→ scombroid 组胺**（似过敏、抗组胺）
> - 雪卡毒的 areflexia 是 **GBS (Guillain-Barré syndrome, 吉兰-巴雷综合征) 最强诱饵**——但雪卡毒**无上行性无力 (ascending weakness)**，靠这条 + 暴露史分流。

> [!note] 关联错题：NBME11A Q151（吃 grouper 后 GI → 冷热倒错 + 口周四肢麻、反射消失但肌力正常 = ciguatera；误判 GBS 是高频坑）→ [[NBME11_错题本#^Q151]]

## 三、血性（炎症性）泻：病原 → 暴露 → 并发症 ⭐⭐⭐

| 病原 | 暴露/史 | 治疗 | **标志性并发症** |
|---|---|---|---|
| **产志贺毒素大肠杆菌 STEC（E. coli O157:H7）** | **未熟牛肉**、**无高热** | **仅支持（禁抗生素！）** | **HUS（溶血尿毒综合征）** |
| **志贺菌 Shigella** | 污染食/水、**日托暴发**（低感染剂量）| 支持；**重症抗生素（不增 HUS）** | **儿童惊厥** + 里急后重；HUS（较少）|
| **弯曲菌 Campylobacter** | **生/未熟禽肉** | 支持；重症抗生素（首选 azithromycin）| **吉兰-巴雷综合征 GBS**；反应性关节炎 |
| **沙门菌 Salmonella**（非伤寒，可非血性）| **未熟鸡肉、爬行动物** | 支持；重症/高危抗生素 | **菌血症**（婴儿/老年/免疫低/镰刀贫）|
| **阿米巴 Entamoeba histolytica** | **发展中国家旅行** | metronidazole + paromomycin | **肝脓肿** |
| **艰难梭菌 C. difficile** | **近期抗生素/住院** | 停诱因 + 口服 vancomycin / fidaxomicin | 中毒性巨结肠 |

> [!danger] 按并发症反推病原（高频秒杀）
> **HUS → STEC（O157，禁抗生素）/ 惊厥 → 志贺（儿童）/ GBS → 弯曲菌 / 菌血症 → 沙门（鸡-爬行）/ 肝脓肿 → 阿米巴 / 近期抗生素 → C. diff。**

> [!warning] ⚠️ 沙门菌两张面孔：**伤寒（非血性肠热症）vs 非伤寒（可血性肠炎）** —— 别混
> 上表里的"沙门菌"指**非伤寒沙门菌**。**伤寒沙门菌不在血性腹泻表里**——它是**全身性肠热症**，不以血便为主。
>
> | 维度 | **非伤寒沙门菌 non-typhoidal**（S. enteritidis/typhimurium） | **伤寒沙门菌 Salmonella Typhi（肠热症 enteric fever）** |
> |---|---|---|
> | 暴露 | 未熟鸡蛋/禽肉、爬行动物 | 南亚/印度次大陆**旅行**、污染水 |
> | 起病 | **急性肠炎**（数天） | **隐袭全身病**（1–3 周） |
> | 腹泻 | **炎症性、可血性/黏液** | **早期反而便秘（成人）**；后期"**豌豆汤样**（pea-soup）"**黄绿稀便·非血性** |
> | 全身招牌 | 多无 | **阶梯热 + 相对缓脉（Faget sign）+ 玫瑰疹（rose spots）+ 肝脾大** |
> | 血便何时出现 | 肠炎期即可 | **仅第 3–4 周并发症**：Peyer 斑坏死 → **肠出血/穿孔**（黑便/血便，晚期，非就诊腹泻） |
> | 诊断 | 粪培养 | **血培养**（粪培养早期常阴） |
> | 治疗 | 多自限；重症/高危才抗生素 | **必须抗生素**：头孢曲松（ceftriaxone）/ 氟喹诺酮 |
>
> **一句话**：看到"**鸡蛋/禽肉 + 急性血性/炎症性肠炎**"= 非伤寒沙门菌；看到"**旅行 + 阶梯热 + 缓脉 + 玫瑰疹 + 便秘/豌豆汤泻**"= 伤寒（肠热症，**非血便**）。
>
> 🧩 **伤寒五钩子速记 + 热型（fever pattern）对照表**（阶梯热 vs 间歇/稽留/波状/Pel-Ebstein）→ [[完整笔记/专题笔记/感染/感染_HighYield速查总览]] §十三。

## 四、何时用抗生素（核心决策）⭐⭐⭐

> [!warning] 大原则：多数感染性腹泻自限 → 支持治疗；抗生素仅重症/高危
> **抗生素指征**（弯曲/志贺/沙门等）：**重症（>7 天 / 高热 / 血便重 / 中毒貌）/ 高危宿主（婴儿、老年、孕妇、免疫抑制、无脾）/ 防控（医护、食品、日托暴发）**。

| 情形 | 处理 |
|---|---|
| **轻症、非血性、自限** | **仅支持（补液）**——别因粪培养阳就给药（[[mistakes/uworld-mistakes_2026-06#^Q17637]]）|
| **STEC（E. coli O157:H7）** ⭐ | **禁抗生素 + 禁止泻药**（↑毒素释放 → **↑HUS**）；仅支持 |
| **志贺 重症** | 抗生素（**不增 HUS**，与 STEC 相反）|
| **弯曲菌 重症** | **Azithromycin**（环丙耐药率高）|
| **沙门 重症/高危** | 抗生素（**普通轻症反延长携带** → 一般不给）|
| **伤寒（肠热症）** | **一律抗生素**：头孢曲松 / 氟喹诺酮 |
| **霍乱 重症** | 积极补液 ± 多西环素/阿奇霉素 |
| **旅行者腹泻 中重度** | 阿奇霉素（± loperamide，无血无热时）|
| **贾第虫 / 阿米巴** | metronidazole（阿米巴 + paromomycin 清肠腔包囊）|
| **隐孢子 / 环孢子** | 硝唑尼特（恢复免疫）/ TMP-SMX |
| **C. difficile** | 停诱因 + 口服 vancomycin / fidaxomicin |

> [!danger] STEC 抗生素铁律
> **产志贺毒素大肠杆菌（STEC，O157:H7）= 唯一"血性腹泻禁抗生素"**——抗生素 + 止泻药都**↑志贺毒素释放 → ↑HUS**。其他血性腹泻（志贺/弯曲/沙门）**重症可给**。别把"血性腹泻禁抗生素"一刀切，**只有 STEC 禁**。

## 五、何时查粪（粪培养/镜检）

- **指征**：血/黏液便、病程 >7 天、高热 >40℃、**旅行/农场/爬行动物暴露**、免疫抑制、日托/食品从业（防控）。
- **不必查**：急性非血性短病程（多病毒）→ 临床诊断 + 补液（[[mistakes/uworld-mistakes_2026-06#^Q105673]]）。
- **培养阳 ≠ 要治**：用途是"重症选药 + 防控溯源"，**判断核心是临床轻重**（[[mistakes/uworld-mistakes_2026-06#^Q17637]]）。

## 六、Memory Hook

```
先分血性 vs 水样：
  水样 → 病毒(诺如/轮状)最常见 / 食物中毒按潜伏期 / 旅行=ETEC → 补液为主
  血性/黏液 → 按"暴露+并发症"定病原

水样食物中毒(潜伏期)：
  1-6h 吐 = 预成毒素(金葡 / 蜡样炒饭)
  8-16h 泻 = 产气荚膜 / 蜡样腹泻型
  米泔水样大量 = 霍乱 → 猛补液
  旅行水样 = ETEC → 阿奇(中重度)
  脂肪泻恶臭胀气(野营井水) = 贾第虫 → 甲硝唑
  AIDS慢性水样 = 隐孢子 → 恢复免疫

血性4病原(并发症秒杀)：
  STEC(O157,未熟牛肉,无高热) → HUS → 禁抗生素+禁止泻
  志贺(日托暴发) → 儿童惊厥 → 重症抗生素(不增HUS)
  弯曲(生禽肉) → GBS → azithromycin
  沙门(鸡/爬行,非伤寒) → 菌血症

沙门两面孔：非伤寒=急性可血便；伤寒=全身肠热症非血便(旅行+阶梯热+缓脉+玫瑰疹)

抗生素：多数自限只支持；重症/高危才给；STEC 永远禁；伤寒一律给
```

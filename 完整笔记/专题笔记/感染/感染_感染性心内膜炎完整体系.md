# 感染性心内膜炎完整体系（Infective Endocarditis, IE）

> [!info] 笔记定位
> **跨学科（感染 ↔ 心内）整合**：把感染主笔记的"病原 + 治疗 + 经典体征"和心内主笔记的"瓣膜并发症 + ECG 警报 + 秒杀技巧 + 手术指征"两条主线整合到一个查询入口。
> **设计原则**：诊断（Duke）→ 经验治疗 → 每日扫并发症 → 手术决策点贯穿全程 → 预防是闭环。

---

## 一、核心定义

**Infective Endocarditis (IE)**：心内膜（含瓣膜、心室壁、人工材料）的感染，特征是 **vegetation**（赘生物 = 纤维蛋白 + 血小板 + 微生物 + 炎症细胞）形成。

经典三联触发题干 → 想到 IE：
- **Fever + new murmur**（最经典）
- **IVDU + 发热 + 杂音 / 栓塞征**
- **人工瓣 + 发热**

## 二、危险因素 + 病原速查 ⭐

### 2.1 病原 vs 危险因素对应表

| 危险因素 / 患者群 | 最常见病原 | 注意点 |
|---|---|---|
| **IVDU**（静脉吸毒者） | **S. aureus**（含 MRSA），其次 strep、念珠菌、铜绿假单胞菌 | 偏右心（**tricuspid**），但左心也常见 |
| **天然瓣 + 急性起病 / 健康瓣** | **S. aureus** | 急性，破坏快；典型 tricuspid（IVDU 关联） |
| **天然瓣 + 牙科/口腔操作 / 受损瓣**（多为 mitral） | **Viridans group strep**（S. mutans, S. sanguinis） | 亚急性 |
| **人工瓣 < 2 个月**（早期 PVE） | **S. epidermidis** (CoNS) | 院内获得 |
| **人工瓣 > 12 个月**（晚期 PVE） | 与天然瓣类似（strep, staph） | |
| **结肠癌 / GI 病变** | **S. gallolyticus**（旧名 S. bovis）、**Clostridium septicum** | ⭐ **必做 colonoscopy** 找肠癌 |
| **泌尿生殖道操作 / 老年男性 / 前列腺操作后** | **Enterococcus**（E. faecalis） | **临床钥匙**：GU 操作后发热 + new murmur + GPC 血培养 + nitrite negative UTI → Enterococcus |
| **培养阴性 IE** | **HACEK**、**Coxiella burnetii** (Q fever)、**Bartonella**、**Brucella**、**Tropheryma**（Whipple）、Legionella、Chlamydia | 需特殊培养基 / 血清学 |
| **免疫抑制 / 长期住院 / 全肠外营养** | **Fungi**（Candida, Aspergillus） | 大赘生物 + 高栓塞风险 → **直接手术指征** |
| **SLE / 恶性肿瘤** | **NBTE / Libman-Sacks endocarditis** | 培养阴性、Sterile vegetations、**瓣膜两面都长**（vegetations on both sides）、免疫复合物 / 恶液质相关 |

### 2.2 助记

> **HACEK**：**H**aemophilus, **A**ggregatibacter, **C**ardiobacterium, **E**ikenella, **K**ingella

> **病原"看人下菜"** —
> - IVDU → S. aureus（右心 tricuspid）
> - 牙医 → Viridans strep
> - 早期人工瓣 → S. epidermidis
> - 老人 + 肠 → S. gallolyticus（**找肠癌**）
> - GU 操作 + UTI nitrite⊖ → Enterococcus
> - 培养阴性 → HACEK + Q + Bartonella + Brucella
> - 免疫抑制 → 真菌

## 三、Duke Criteria（诊断框架）

> [!danger] 疑 IE 的 best next step = 血培养（唯一硬规则：先于抗生素 + 先于影像）⭐ #难记
> 题问"best next step"且尚无微生物诊断时 → **先抽 3 套不同部位血培养**。**只有"血培养第一"是固定的**；抗生素与 echo 的相对顺序看病人**急不急**（见下）。
> - **血培养永远先于抗生素**：药后阳性率 >90% → 骤降 → 丢失病原诊断（这是硬规则）。
> - **echo 绝不拖延血培养/抗生素**：TTE/TEE 要数小时安排；**直奔 TEE 是高频错选**（[[mistakes/uworld-mistakes_2026-06#^Q3012]] 牙科操作后 IE 误选 TEE）。
>
> ```
> 疑 IE → ① 血培养 ×3（不同部位、抗生素前）★唯一固定第一步
>        │
>        ├─ 急重 / 高度疑 IE → ② 经验抗生素（培养后立刻给，不等 echo）→ ③ Echo
>        │
>        └─ 亚急性 / 稳定（本例）→ ② Echo 安排 → ③ 抗生素（可延到培养回报）
>
>   Echo：TTE 起步 → 人工瓣 / TTE 阴 / 疑脓肿 → TEE
> ```
> > [!warning] 别把本例顺序当通用
> > 本题病人**亚急性、稳定**（27 岁、2 周乏力），所以抗生素可延后 → 顺序看似"血培→echo→抗生素"。但**急重病人是"血培→抗生素→echo"**（抗生素必须在 echo 前就给，不能等影像）。两者唯一共同点 = **血培养第一**。
>
> 同一条"取材定病原优先"反射也见于糖尿病足骨髓炎（[[mistakes/uworld-mistakes_2026-06#^Q7522]] 骨活检培养先于经验抗生素）。

### 3.1 总体判读

**Definite IE** = 2 Major / 1 Major + 3 Minor / 5 Minor
**Possible IE** = 1 Major + 1 Minor / 3 Minor

### 3.2 Major Criteria（2 条）

1. **典型 IE 病原阳性血培养**
   - 两次独立血培养阳性（病原为 viridans strep / S. gallolyticus / HACEK / S. aureus / 社区获得性 enterococcus 等）
   - 或持续菌血症（≥2 套间隔 12h 阳性 / ≥3 套大部分阳性）
   - 或单次 Coxiella burnetii 阳性 / phase I IgG ≥1:800
2. **心内膜受累证据**
   - Echo: vegetation / abscess / 新发瓣膜 dehiscence
   - **新发瓣膜反流**（不只是杂音变化）

### 3.3 Minor Criteria（5 条）— 助记 **"FROM JANE"**

1. **Fever** ≥38°C
2. **Predisposition**（IVDU / 既往心脏病变 / 人工瓣）
3. **Vascular phenomena**：动脉栓塞、septic pulmonary infarct、mycotic aneurysm、ICH、结膜出血、**Janeway lesions**（无痛、手掌脚掌）
4. **Immunologic phenomena**：肾小球肾炎、**Osler nodes**（痛、指尖）、**Roth spots**（视网膜）、RF+
5. **微生物学证据**（不满足 major 的血培养阳性 / 血清学）

> [!tip] Janeway vs Osler 助记
> - **J**aneway = **J**ust spots（无痛、手掌、栓塞性）
> - **O**sler = **O**uch!（痛、指尖、免疫性）

### 3.4 影像选择

| 情境 | 影像选择 |
|---|---|
| 怀疑 IE（任何） | **TTE 先做** |
| 高度怀疑 / 人工瓣 / TTE 阴性但临床高度怀疑 / 找瓣周脓肿 / 找瓣周漏 | **TEE**（敏感性远高于 TTE，TTE 仅 60-70%） |
| 已诊断 IE + 新发 AV block | **TEE 找 perivalvular abscess** |

## 四、经典外周体征（Classic Peripheral Findings）

| 体征 | 部位 | 性质 | 机制 | 类别 |
|---|---|---|---|---|
| **Roth spots** | 视网膜 | Hemorrhagic lesions w/ pale centers | 微栓塞 → 免疫复合物 | Immunologic |
| **Splinter hemorrhages** | 指甲床 | Linear hemorrhagic lesions | 微栓塞 | Vascular |
| **Janeway lesions** ⭐ | 手掌、脚掌 | **无痛**、平坦、红斑样 | **栓塞性**（败血性微栓塞）| Vascular |
| **Osler nodes** ⭐ | 指尖、趾尖 pads | **痛！**、隆起、紫色 | **免疫复合物沉积** | Immunologic |
| **结膜出血** | 球结膜 | Hemorrhagic | 微栓塞 | Vascular |

> [!success] 助记
> **Janeway = Just spots**（无痛, 栓塞 → Vascular）
> **Osler = Ouch!**（痛, 免疫 → Immunologic）

> [!danger] 反向陷阱警告（来自 2026-05-18 自测 Q14 错题）
> USMLE 高频"方向反向"选项：把 Janeway 写成免疫复合物 / Osler 写成栓塞 — 凭"我脑子里知道两者不同"会选反。**必须把字母-性质映射刻成条件反射**。
>
> **双字母锁定**：
> - **J**aneway = **J**ust spots → 字母 **J** 直线感 → 平 + 栓塞（**V**ascular，V 像血管直线）
> - **O**sler = **O**uch! → 字母 **O** 圆形 → 隆起 + 免疫（**I**mmunologic，I 像免疫复合物聚成的球）
>
> **位置配字母**：J = surface（手掌足底，J 像平的）；O = digital pulp（O 像凸起的）
>
> **3 秒判断**：题里看到 "**Painful** nodule on **finger pad**" → Osler → Immunologic；"**Painless** lesion on **palm/sole**" → Janeway → Vascular。**不要在考场临场推理。**
>
> 详见 [[mistakes/uworld-mistakes_2026-05#^Q14-2026-05-18|Q14 自测错题]]。

### 4.1 IE 相关 GN vs PSGN ⭐（都"链球菌 + 免疫复合物 GN"，但不同病）

> [!warning] 高频鉴别陷阱
> IE 的免疫复合物肾小球肾炎和 PSGN 长得像（都低补体 + 血尿 + 红细胞管型），但**病原不同、确诊路不同**。题干给"暗尿 + GN"易诱去查 ASO（PSGN），但**有 Osler 结节/瓣膜 = IE**。

| | 亚急性 IE 的 GN | PSGN |
|---|---|---|
| **病原** | **Viridans 链球菌**（口腔菌、α-溶血；牙科操作/坏损瓣膜）| **A 组链球菌 GAS / S. pyogenes**（β-溶血、致肾炎株）|
| **GN 机制** | 心内膜炎免疫复合物沉积肾（IE **并发症**）| GAS 感染后免疫复合物 GN（**不累及心脏**）|
| **前驱/伴随** | 持续菌血症 + **Osler 结节 + 瓣膜/呼吸困难 + 赘生物** | **咽炎或脓皮病后 2-4 周**，无心脏体征 |
| **共同点** | 免疫复合物 + **低补体** + 血尿/红细胞管型 | 同左（易混点）|
| **确诊** | **Duke：3 套血培 + 超声心动（echo）** | 临床 + **ASO/抗 DNase B**↑ + 低 C3 |

> [!tip] 一句话
> **Viridans(IE) ≠ pyogenes(PSGN)；有 Osler/瓣膜 = IE → 走 Duke（血培 + echo），不是 ASO。** 错题驱动 [[mistakes/uworld-mistakes_2026-06#^Q4668]]。

## 五、临床三秒锁定（考场秒杀技巧）⭐⭐⭐（来自心内主笔记）
#难记

> [!danger] 三句锁定 — 默写到考前
> 1. **IE + New Murmur** → **Echo（TEE is better）**
> 2. **IE + New Heart Block** → **Perivalvular Abscess**（看心电图 + TEE）
> 3. **IE + Acute SOB + Crackles** → **Acute Valve Destruction**（急性 AR/MR）→ Emergency Surgery may be needed

### 5.1 IE 并发症 ECG / 临床警报对照

| 看到这个体征 | 想到这个并发症 | 后续动作 |
|---|---|---|
| **新发 PR 间期延长** / 新发 AV block | **主动脉瓣周脓肿**（Perivalvular abscess） | **TEE** + 外科咨询 |
| 突发呼吸困难 + 新发杂音 + S3 | **急性瓣膜毁损 / 急性心衰** | Echo + 外科咨询（emergency） |
| 新发神经系统缺失 / 偏瘫 | **脓毒性栓塞**（septic emboli → 脑） | 头部 MRI + 评估瓣膜 |
| 腰痛 + 发热 | **椎体骨髓炎**（vertebral osteomyelitis） | 脊柱 MRI |

> [!danger] "Stable but progressing" 假象（来自 2026-05-18 自测 Q12 错题）
> **IE + 新发任何传导异常（PR 延长 / Mobitz / 束支 / CHB）= TEE 不过夜**，不管血流动力学是否稳定、不管传导异常等级。
>
> **致命陷阱**：题干给"hemodynamically stable" → 触发"先观察"反射。但脓肿可在**数小时内**从 1° → Mobitz I → Mobitz II → CHB → 心源性休克 → 心源性死亡。"Stable" 是**进展前的窗口期**，不是"病情稳定"。
>
> **错误选项排除**：
> - ❌ 仅观察 / 继续抗生素 → 不解决结构问题
> - ❌ Transcutaneous pacing → 只治症状不治源（脓肿）
> - ❌ 换更广谱抗生素 → 脓肿是 source control 问题，不是 antibiotic coverage 问题
> - ❌ Lyme 血清学 → 病人已在 IE 治疗，是并发症识别题不是新诊断题
>
> **正确路径**：**TEE 立即** → 确认脓肿 → 外科会诊（valve surgery + abscess drainage）。
>
> 详见 [[mistakes/uworld-mistakes_2026-05#^Q12-2026-05-18|Q12 自测错题]]。

## 六、经验性抗生素 + 目标治疗

### 6.1 经验性方案（while awaiting cultures）

| 患者群 | 经验方案 |
|---|---|
| **天然瓣，无 IVDU** | Vancomycin + Ceftriaxone（cover staph + strep + HACEK） |
| **IVDU** | **Vancomycin**（cover MRSA）± 加 Cefepime（cover pseudomonas） |
| **人工瓣 < 12 个月** | Vancomycin + Gentamicin + Rifampin（cover MRSE） |
| **人工瓣 > 12 个月** | 类似天然瓣方案 |

- 治疗周期：**4-6 周**（依瓣膜类型 / 病原 / 并发症）
- 人工瓣 IE 通常 6 周 + Rifampin（穿透 biofilm）

### 6.2 目标治疗 — by 病原（培养出结果后降阶梯）

#### S. aureus

| 瓣膜 | 方案 | 备注 |
|---|---|---|
| **Native valve（MSSA）** | **Nafcillin / Cefazolin** | 主力杀 MSSA，抗青霉素酶；现代 native valve staph 不常规加 gentamicin |
| **Prosthetic valve** | **Nafcillin + Gentamicin + Rifampin** | Rifampin 针对 biofilm |
| **PCN 过敏 / MRSA** | **Vancomycin** | |

#### Strep viridans / mutans（敏感性通常好）

| 严重度 | 方案 |
|---|---|
| Mild | **Penicillin** |
| More aggressive | **Ceftriaxone + Gentamicin**（协同杀菌） |

> 备注：天然瓣 vs 人工瓣对此框架影响不大。

#### Enterococcus（β-lactam 单用常不够）⭐

> **核心**：必须协同（synergy）。

| 严重度 | 方案 |
|---|---|
| Mild | Penicillin |
| Aggressive | **Ampicillin + Gentamicin** 或 **Ampicillin + Ceftriaxone**（替代部分 aminoglycoside） |
| PCN 过敏 | Vancomycin |

> **Enterococcus 临床场景钥匙**：刚做完前列腺操作 / 导尿 / 泌尿系手术 → 发热 → 新 murmur → 血培养 gram-positive cocci → **UTI + nitrite negative** → Enterococcus endocarditis

#### S. gallolyticus / Clostridium septicum

抗生素同 strep；但**必做 colonoscopy** 找肠癌。

## 七、IE 患者管理总流程 ⭐（一图总览）

```
怀疑 IE（发热 + 危险因素 / 杂音 / 栓塞征象）
   │
   ▼
Step 1: 诊断
─────────────────────────────
• 血培养 × 3（不同部位、间隔 ≥1h）
• Echo: TTE 先做 → 高度怀疑 / 人工瓣 / TTE 阴性 → TEE
• 应用 Duke criteria
   │
   ▼
Step 2: 经验性广谱抗生素（不等培养结果）
─────────────────────────────
• 天然瓣无 IVDU → Vancomycin + Ceftriaxone
• IVDU / MRSA 风险 → Vancomycin ± Cefepime
• 人工瓣 <12 月 → Vancomycin + Gentamicin + Rifampin
• 培养出结果后 → 降阶梯到敏感方案
   │
   ▼
Step 3: 每日评估并发症（决定是否手术）⭐
─────────────────────────────
   │
   ├─── 心衰？(HF from valve dysfunction) ──────→ 外科会诊（Emergent）⭐ 最常见 + 最致死
   │
   ├─── 新发 AV block / 心音变化？─────────────→ TEE 找 perivalvular abscess → 外科
   │
   ├─── 持续发热 ≥5-7d / 持续菌血症？──────────→ 排除其他感染源 → 重复 BC → 考虑外科
   │
   ├─── 难治菌种？(fungi / MDR / Brucella) ────→ 早期外科
   │
   └─── Vegetation >10 mm + 既往/反复栓塞？────→ 外科会诊
   │
   ▼
Step 4: 完成 4-6 周抗生素 + 长期随访
─────────────────────────────
• 天然瓣 4-6 周；人工瓣 6 周 + Rifampin
• 完成治疗后 echo 复查
• S. gallolyticus → 必做 colonoscopy
• IVDU → addiction medicine 转介
• 预防：高危心脏 + 牙科操作前 Amox 2g
```

> **流程口诀**：**诊断 → 抗生素 → 每日扫并发症 → 完成疗程**
> 抗生素是底盘，**外科决策点贯穿全程**（不是治疗末期才考虑）。

## 八、5 大手术指征 ⭐⭐⭐（HEROIC 口诀）

| 指征 | 临床判断标准 | 紧迫性 |
|---|---|---|
| **1. Heart failure** ⭐⭐⭐ | 急性 valve regurg (MR/AR) → pulm edema / 低血压 | **Emergent**（数小时-数天）— 最常见死因 + 最常见手术指征 |
| **2. Extension of infection** ⭐⭐⭐ | Perivalvular abscess / fistula / **新发 heart block**（提示瓣环脓肿蔓延到传导系统）| Urgent（数天） |
| **3. Resistant organism** ⭐⭐ | **Fungi**（Candida, Aspergillus）、**MDR**、Brucella、Coxiella | Urgent |
| **4. Ongoing bacteremia** ⭐⭐ | 抗生素 ≥5-7 天仍发热 / 血培养阳性，排除其他感染源 | Urgent |
| **5. Immense embolism risk** ⭐⭐ | Vegetation **>10 mm** + 既往栓塞 / 反复栓塞 in spite of ATB | Semi-urgent |

> [!success] 助记口诀 "HEROIC"
> **H**eart failure / **E**xtension / **R**esistant organism / **O**ngoing bacteremia / **I**mmense embolism / **C**onsultation early
>
> 或 **"3D + 2E"**：
> **D**ysfunction(HF) / **D**ifficult organism / **D**estruction extends / **E**mbolism / **E**ndless bacteremia

## 九、并发症速查（局部 + 远端）

### 9.1 总览表

| 类别 | 并发症 | 机制 | 处理 |
|---|---|---|---|
| **局部（心脏）** | **急性心衰** | Valve destruction → 急性 MR/AR | **手术**（最常见死因） |
| | **Perivalvular abscess** | 感染从瓣叶蔓延到瓣环 | **新发 heart block** = 强烈警示；TEE 看；**手术** |
| | **Pericarditis** | 局部蔓延 | 内科支持 |
| | **Intracardiac fistula** | 脓肿穿破入相邻腔室 | 手术 |
| | **Conduction block** | Abscess 蔓延到 AV node 区域 | 新发 AV block = 红旗 → 手术 |
| **远端（栓塞）** | **Septic emboli** | Vegetation 脱落 | 左心 IE → 脑/脾/肾/肢体；右心 IE → 肺（septic pulmonary infarct） |
| | **Metastatic abscess** | 栓子定植远端组织 | 抗生素 ± 引流 |
| | **Mycotic aneurysm** | 栓子嵌入血管壁 → 感染性扩张 | 脑动脉最危险（破裂 = ICH），**抗凝禁忌**！ |
| | **Organ infarction**（脾、肾） | 栓塞 | 支持 |
| | **Glomerulonephritis** | 免疫复合物沉积 | 血尿、Cr↑；支持 |

## 十、主动脉瓣 vs 二尖瓣 IE 并发症倾向（关键鉴别）

| 受累瓣膜 | 并发症倾向 | 对传导系统的影响 |
|---|---|---|
| **主动脉瓣（Aortic）** | 更易形成**瓣周脓肿** | **极易导致新发传导阻滞（AV block）** → 看到 AV block 就找瓣周脓肿 |
| **二尖瓣（Mitral）** | 更易**腱索断裂 → 急性 MR → 急性心衰** | 较少直接引起传导阻滞（除非脓肿极大） |

> [!warning] 一句话锁定
> **IE + AV block = Aortic IE + 瓣周脓肿** → TEE + 外科。

## 十一、急性 vs 慢性瓣膜反流（关键概念）

| 特征 | **急性 MR/AR**（如 IE / dissection / 乳头肌断裂） | **慢性 MR/AR**（如风心、退行性、MVP） |
|---|---|---|
| 起病 | 数小时-数天 | 数月-数年 |
| LV 大小 | **正常**（无代偿时间） | **扩大** |
| LV 功能 | 看似正常或亢进 | EF 逐渐↓ |
| 临床 | **急性肺水肿 + 心源性休克** | 渐进劳力性呼吸困难 |
| 杂音 | 短促、可能不明显（高 LA 压 → 梯度↓） | 典型 holosystolic（MR）/ decrescendo diastolic（AR） |
| 处理 | **紧急手术** | 药物 + 适时择期手术 |

> [!info] 急性反流题干触发词
> "bounding pulses" / "wide pulse pressure" / "endocarditis" / "aortic dissection (acute AR)"

## 十二、IE 预防性抗生素 — 牙科 / 操作前用药指征

### 12.1 高危心脏条件（满足才用预防）

| 高危心脏条件 |
|---|
| 人工瓣（包括 TAVI） |
| 既往 IE 史 |
| 未修复的 cyanotic 先心病 |
| 修复的先心病 < 6 个月内 |
| 修复后残余缺陷 |
| 心脏移植 + 瓣膜病变 |

### 12.2 高危 vs 低危操作分类（来自心内主笔记）

| 类别 | 操作 | 预防方案 |
|---|---|---|
| **高危**（需预防） | Gingival/apical **牙科操作**（穿透牙龈 / 牙根周围 / 牙髓） | **Amoxicillin 2g PO** 操作前 30-60 min（PCN 过敏 → Clindamycin / Macrolide / Cephalosporin） |
| | **呼吸道黏膜穿透**手术 | Amoxicillin |
| | **感染皮肤 / 肌肉骨骼**组织手术 | **Vancomycin** |
| | **GI / GU 操作** + 活动感染 | Ampicillin |
| **低危**（不需预防） | GI / GU 内镜（无活动感染） | — |
| | 多数阴道 / 剖宫产分娩 | — |
| | **机械瓣 + routine screening colonoscopy** | — |

### 12.3 常见误区

- ❌ 风心病、二尖瓣脱垂、ASD/VSD（修复后无残余）、单纯 MR — **不再推荐预防**
- ❌ 胃肠/泌尿系统操作 — **不再推荐预防**（除非有活动感染）
- ❌ 机械瓣患者去做 **routine screening colonoscopy** → 也不需要 IE 预防

### 12.4 历史方案反向陷阱 ⚠️（来自 2026-05-18 自测 Q13 错题）

> [!danger] AHA 2007 之前的旧方案 = USMLE 干扰项
> 2007 年前 GI/GU 高风险操作曾用 **IV ampicillin + gentamicin** 做 IE 预防。**该方案现已完全废除**，但 USMLE 干扰项里高频出现。
>
> **看到下列选项立即识别为历史 / 错误方案**：
> - ❌ **IV ampicillin + gentamicin**（GI/GU 操作） → 2007 前老方案，已废除
> - ❌ **IV vancomycin**（任何操作） → 不在任一 IE 预防方案
> - ❌ **Clindamycin 600 mg PO**（非 dental） → 仅 dental + PCN 过敏备选
> - ❌ **Amoxicillin 2 g PO**（非 dental） → 仅 dental 方案，不能套到 GI/GU/skin
>
> **唯二记忆**：
> 1. **"D + R only"**：IE prophylaxis 仅给 **D**ental + **R**espiratory invasive 操作
> 2. **"GI / GU is OUT since 2007"**：已被踢出预防名单 18+ 年
>
> **特殊情况**：GI/GU 操作 + 活动性感染 → 给抗生素治疗感染本身（按感染源选药，**不是 IE 预防剂量**）。
>
> 详见 [[mistakes/uworld-mistakes_2026-05#^Q13-2026-05-18|Q13 自测错题]]。

## 十三、IE 易混陷阱

- ❌ "Vegetation = 血栓 → 抗凝" → **错**：IE 不常规抗凝，**反增 ICH 风险**（mycotic aneurysm 破裂）；仅在已有其他抗凝指征时维持
- ❌ "急性心衰先 β-blocker" → **错**：急性失代偿期禁用 β-blocker，先 surgical + 利尿 + 抗生素
- ❌ "IVDU 发热 = IE 立即做 echo" → **顺序**：先血培养 × 3 + 经验抗生素 + TTE → 必要时 TEE
- ❌ "TTE 阴性就排除 IE" → **错**：TTE 敏感性 60-70%，**人工瓣 / 高度怀疑必须 TEE**
- ❌ "S. bovis 是普通菌" → **错**：S. gallolyticus = **结肠癌信号**，必须 colonoscopy
- ❌ "等抗生素治好再考虑手术" → **错**：手术指征出现就要做，不等"杀完菌"
- ❌ "S. aureus IE 都是急性" → **错**：MSSA 急性多见，但也可亚急性表现
- ❌ "机械瓣去做 screening colonoscopy 要预防性抗生素" → **错**：低危操作，不需要

## 十四、Memory Hook

- **"IE 死于 HF，HF 救于刀"** — 心衰是 IE 最常死因 + 最强手术指征
- **HEROIC** 口诀（5 大手术指征）：HF / Extension / Resistant / Ongoing / Immense embolism / Consultation
- **三句秒杀**：
  - IE + New Murmur → **Echo（TEE better）**
  - IE + New Heart Block → **Perivalvular Abscess**
  - IE + Acute SOB + Crackles → **Acute Valve Destruction → Emergency Surgery**
- **类比**：IE = "房子着火 + 房梁坏了"
  - 抗生素 = 消防员（灭火）
  - 手术 = 工程队（修房梁）
  - **光灭火房子还是塌**
- **病原"看人下菜"**：
  - IVDU → S. aureus（右心 tricuspid）
  - 牙医 → Viridans strep
  - 早期人工瓣 → S. epidermidis
  - 老人 + 肠 → S. gallolyticus（**找肠癌**）
  - GU 操作 + UTI nitrite⊖ → Enterococcus
  - 培养阴性 → HACEK + Q + Bartonella + Brucella
  - 免疫抑制 → 真菌
- **Janeway = Just spots（无痛, 栓塞 / Vascular）; Osler = Ouch!（痛, 免疫 / Immunologic）**
  - 反向陷阱：USMLE 必出至少一道选项故意写反，看到 "**Painful** + **finger pad**" → Osler→Immune；"**Painless** + **palm/sole**" → Janeway→Vascular
- **瓣膜 → 并发症倾向**：Aortic IE → 瓣周脓肿 + AV block；Mitral IE → 腱索断裂 + 急性心衰
- **"Stable but progressing in 12h" 不是 stable** — IE + 新发任何传导异常 → TEE 不过夜
- **"D + R only"** — IE prophylaxis 仅 **D**ental + **R**espiratory，GI/GU since 2007 = OUT

## 十五、跨科鉴别与关联场景

### 15.1 IE 也属于"游走性关节炎"鉴别（来自感染主笔记）

> 题干"migratory arthralgia/arthritis"鉴别：
> 1. 风湿热（最经典）
> 2. **播散性淋球菌感染 (DGI)** — 年轻性活跃女性 + 关节痛 + 腱鞘炎 + 皮疹三联
> 3. **IE** — 发热 + 新发杂音 + 栓塞体征 / Osler / Janeway / IVDU / 人工瓣
> 4. Lyme — 蜱虫 + erythema migrans + AV block
> 5. SLE / 病毒（parvo B19, HBV/HCV）/ 反应性关节炎

### 15.2 心脏传导阻滞 + 发热 / 感染 — "LSD loves company"（来自感染主笔记）

> AV block 引起原因的"心脏传导阻滞 bugs"鉴别：
> - **L**yme disease
> - **S**almonella typhi
> - **D**iphtheria
> - **L**egionella
> - **C**hagas disease
>
> **但**：如果患者已确诊 IE → 新发 AV block **不要往上面这些跳**，要先想 **Aortic IE + 瓣周脓肿** → TEE。

# HIV 初始管理与 ART

> [!info] 笔记定位
> 回答"**刚诊断 HIV (human immunodeficiency virus, 人类免疫缺陷病毒) 的人，第一次就诊该做什么**"。覆盖三大块：
> ① **初始评估清单**（基线 CD4 / 病毒载量 / 耐药 / 合并感染 + 疫苗）
> ② **ART (antiretroviral therapy, 抗逆转录病毒治疗) 何时启动 + 首选方案**
> ③ **IRIS (immune reconstitution inflammatory syndrome, 免疫重建炎症综合征)**
> 派生自 [[NBME11_错题本#^Q093]]（HIV 初治"双动作"：人人 ART + CD4<200 加 PCP 预防）。
>
> **分工**：本笔记 = 初治"做什么/何时做"；**机会性感染 (OI) 病原谱 + CD4 分层 + 对口药**详见姊妹篇 [[完整笔记/专题笔记/感染/感染_AIDS机会性感染按CD4谱]]（§二 CD4 钟表 + §三预防阈值）。两篇互补，预防阈值表两边都放是为各自自洽，认准其一默写即可。

## 一、HIV 初治"双动作"⭐⭐⭐（最高频考点）

> [!danger] 别只做一半（派生自 Q093）
> 新诊断 HIV 的处理永远是**两件事一起**：
> 1. **所有人立即启动三药联合 ART** —— 无论有无症状、无论 CD4 高低、无论病毒载量。"无症状"不是观望理由。
> 2. **按 CD4 阈值加机会性感染一级预防** —— CD4 <200/mm³ → PCP (Pneumocystis jirovecii pneumonia, 耶氏肺孢子菌肺炎，旧称 PCP / 现规范 PJP) 预防，一线 **TMP-SMX (trimethoprim-sulfamethoxazole, 复方磺胺甲噁唑)**。
>
> ⚠️ 当题目把"ART"和"PCP 预防"拆成**独立选项**，正确答案常是"**两个都要**"：只选 ART（漏预防）或只选预防（漏 ART）都是设计好的陷阱。**两药 ART (two-drug) 也错** —— 增加耐药风险，初始标准是**三药**（来自至少两个不同药物类别）。

## 二、新诊断 HIV 初始评估清单 ⭐（第一次就诊做什么）

> [!info] 框架：确诊后第一步是"建立基线 + 排合并感染 + 补疫苗"，与 ART 同步推进，不互相等待
> ART 不必等任何一项检查回报（确诊即启），但下面这些基线检查应在初诊一并开出。

| 类别 | 具体项目 | 临床意义 |
|---|---|---|
| **基线免疫状态** | **CD4 计数** | 决定 OI 预防阈值（<200 PCP / <100 弓形虫 / <50 MAC 评估）+ 判断免疫损伤程度 |
| **病毒学基线** | **HIV RNA 病毒载量 (viral load)** | ART 疗效随访的基线；治疗目标 = 测不到 (undetectable)；也是传染性指标（U=U, undetectable=untransmittable, 测不到=不传播） |
| **耐药基线** | **基因型耐药检测 (genotypic resistance testing)** | 启 ART 前查，发现传播性耐药突变 → 指导方案选择 |
| **合并感染筛查** | **结核 TB (tuberculosis, 结核)**（IGRA 或 TST；阳性 + 无活动病 → 治 LTBI (latent TB infection, 潜伏结核感染)） | HIV 是 TB 强危险因素；漏诊潜伏 TB → ART 后 IRIS 激活活动 TB |
| | **乙肝 HBV**（HBsAg / 抗-HBs / 抗-HBc） | HBV 共感染影响 ART 选药（tenofovir 同时覆盖 HBV）；非免疫者补种 HBV 疫苗 |
| | **丙肝 HCV**（抗-HCV） | 共感染常见（共用针具/性传播）；阳性 → 评估直接抗病毒 (DAA, direct-acting antiviral, 直接抗病毒药) 治疗 |
| | **其他 STI (sexually transmitted infection, 性传播感染)**（梅毒 RPR/VDRL、淋病/衣原体 NAAT） | 性传播途径共病高发；梅毒尤其常并存 |
| | **弓形虫 Toxoplasma IgG** | 阳性 + CD4<100 → 起弓形虫一级预防（TMP-SMX 顺带覆盖） |
| **疫苗** | **灭活疫苗优先**：肺炎球菌 (PCV/PPSV)、流感（每年）、HBV、HAV、Tdap、HPV、（按 CD4）脑膜炎球菌、带状疱疹（重组 RZV 灭活，可用） | 免疫损伤前尽早补；CD4 越高免疫应答越好 |
| **疫苗禁忌** | **CD4 <200 禁活疫苗** (MMR / 水痘 VZV / 黄热)；CD4 ≥200 可谨慎接种 MMR/水痘 | 活疫苗在低 CD4 可致疫苗株播散 |

> [!tip] 评估清单记忆
> **"4-Vis + TB + 肝炎 + STI + 疫苗"**：4 个一查到底 —— **CD4 / 病毒载量 (Viral load) / 耐药基因型 / 弓形虫 IgG**；合并感染三筛 —— **TB / HBV + HCV / 梅毒等 STI**；最后**补灭活疫苗，CD4<200 禁活疫苗**。

> [!info] 与分科笔记呼应
> [[完整笔记/Peixuan分科笔记/感染]] 把 HIV 感染风险管理分两块：**A. CD4 还高 → 主做疫苗预防；B. CD4 明显下降 → 才加抗感染预防药**。本清单 = 这两块的具体落地项。

## 三、ART 何时启动 + 首选方案 ⭐

### 3.1 何时启动 —— 现行指南：确诊即启

> [!danger] 现指南："Treat all, treat now"——确诊即启，不等 CD4
> **所有 HIV 感染者，一经确诊即应尽早启动 ART**，**不再以 CD4 阈值作为启动门槛**（旧观念"CD4 掉到某值才开始"已废弃）。
> - **无症状不延迟**，CD4 高也不延迟，病毒载量低也不延迟。
> - 越早抑制病毒载量 → 越早重建免疫 + 降低传播（U=U）+ 减少非 AIDS 并发症。
> - **少数需略缓的情形**（CK 偶考，了解即可）：某些 OI 活动期需先稳住再启（如**隐球菌脑膜炎 (cryptococcal meningitis)** 常先抗真菌、推迟 ART 数周以减重症 IRIS；**TB 脑膜炎**亦个体化），但**绝大多数情况"确诊即启"**。

### 3.2 首选方案概念 —— INSTI 为基础

> [!info] 首选骨架：INSTI + 2 NRTI（"整合酶 + 两核苷"）
> 现代一线初始方案 = **以 INSTI (integrase strand transfer inhibitor, 整合酶链转移抑制剂) 为核心** + 2 个 NRTI (nucleoside reverse transcriptase inhibitor, 核苷类逆转录酶抑制剂) 骨架。INSTI 起效快、耐受好、耐药屏障高，已取代蛋白酶抑制剂 (PI) 成为首选。

| 角色 | 代表药（中英双语） | 备注 |
|---|---|---|
| **INSTI 核心**（首选） | **比克替拉韦 bictegravir** / **多替拉韦 dolutegravir** / **拉替拉韦 raltegravir** | 单片复方常用 bictegravir 或 dolutegravir |
| **NRTI 骨架** | **替诺福韦 tenofovir** + **恩曲他滨 emtricitabine**（或 **拉米夫定 lamivudine** / **阿巴卡韦 abacavir**） | tenofovir 同时覆盖 HBV；abacavir 用前查 **HLA-B*5701**（阳性禁用，防超敏） |

> [!tip] 概念记忆（CK 不要求记完整三联组合）
> 题目通常考"**新诊断 → 上三药 ART**"这一动作本身，或考"**两药 / 单药错（耐药）**"、"**确诊即启不等 CD4**"。具体药名作为**类别识别**：见 **多替拉韦 / 比克替拉韦** → 认出 INSTI 首选骨架即可。
> - 经典干扰："只给 ART 不给预防" / "等 CD4 降到某值再启 ART" / "两药联合" —— 全错。

> [!warning] 药名真实性核验
> bictegravir 比克替拉韦 / dolutegravir 多替拉韦 / raltegravir 拉替拉韦 / tenofovir 替诺福韦 / emtricitabine 恩曲他滨 / lamivudine 拉米夫定 / abacavir 阿巴卡韦 —— 均为真实在用药物（INSTI 后缀 -tegravir；NRTI）。

## 四、按 CD4 阈值的机会性感染一级预防"铁律"⭐

> [!warning] 与姊妹篇同源（认一处默写即可）
> 详细病原谱 + 何时停预防见 [[完整笔记/专题笔记/感染/感染_AIDS机会性感染按CD4谱]] §三。此处为初治清单自洽保留核心铁律。

| CD4 阈值 | 预防对象 | 一线药 | 关键备注 |
|---|---|---|---|
| **<200** | **PCP / PJP（耶氏肺孢子菌肺炎）** | **TMP-SMX**（复方磺胺甲噁唑） | 也顺带覆盖弓形虫；CD4<200 或已有口腔念珠菌 / AIDS 定义病即起 |
| **<100**（且 **Toxo IgG ⊕**） | **弓形虫 Toxoplasma** | **TMP-SMX** | 通常已因 PCP 在吃 TMP-SMX，自动覆盖 |
| **<50** | **MAC (Mycobacterium avium complex, 鸟分枝杆菌复合群)** | （azithromycin 每周）| ⚠️ **现已不再常规推荐** —— 若及时启动有效 ART，因成本/低风险/耐药取消 |

```
CD4 阈值 → 一级预防决策树
─────────────────────────────
确诊 HIV
   │
   ├─ 立即启动三药 ART（人人，不等 CD4）
   │
   └─ 查 CD4，按阈值加预防：
        CD4 < 200  ── 起 ── PCP/PJP 预防（TMP-SMX）
        CD4 < 100  ── 且 Toxo IgG⊕ ── 弓形虫预防（TMP-SMX 顺带覆盖）
        CD4 < 50   ── MAC 评估（现多不再常规预防，靠及时 ART）
                          │
   ART 起效、CD4 回升 >200 持续 ≥3 个月
                          └─→ 停 PCP/PJP 预防（免疫重建 = 撤预防）
```

> [!tip] 何时停预防（高频）
> ART 有效后 **CD4 回升 >200 持续 ≥3 个月 → 停 PCP/PJP 预防**。核心原则：**免疫重建 = 撤预防**。

## 五、IRIS 免疫重建炎症综合征 ⭐

> [!info] IRIS (immune reconstitution inflammatory syndrome, 免疫重建炎症综合征) 是什么
> **启动 ART 后免疫系统快速恢复**，对**原本潜伏/亚临床的病原（或其残余抗原）**发起过度炎症反应，导致**已有感染"看似恶化"或新暴露出来**。本质是**免疫好转的副产物**，不是 ART 失败、不是新感染、不是耐药。

| 维度 | 要点 |
|---|---|
| **机制** | ART 抑制病毒 → CD4 快速回升 → 对潜伏抗原（TB、MAC、隐球菌、CMV、JC 病毒等）的炎症应答**重新出现/放大** |
| **时间** | 多在**启动 ART 后数周内**（常 2–8 周；CD4 基线越低、回升越快、风险越高） |
| **典型场景** | 既往潜伏/未治 **TB**、**MAC**、**隐球菌脑膜炎**、**CMV 视网膜炎**、**PML** 在 ART 后"突然冒头"或原有病灶炎症加重（如淋巴结肿大、发热、原 OI 部位炎症恶化） |
| **关键鉴别** | 不是 ART 无效（病毒载量在降、CD4 在升，与"治疗失败"相反）；不是新发 OI 进展（是免疫攻击已存在抗原） |

> [!danger] IRIS 处理铁律：**继续 ART + 治疗/对症，不停 ART** ⭐
> - **不要停 ART**（停了反而前功尽弃）—— 这是最高频考点。
> - **继续/启动针对潜在病原的特异治疗**（如 IRIS 暴露出 TB → 抗结核）。
> - **对症 + 抗炎**：症状重时加**糖皮质激素 (corticosteroids)** 抑制炎症。
> - **预防角度**：启 ART 前先筛/治潜伏感染（尤其 TB），可降低 IRIS 风险；个别重症 OI（隐球菌脑膜炎）会**适当推迟 ART** 数周以减重症 IRIS（见 §3.1）。

> [!tip] IRIS 一句话锁定
> **"上了 ART 后反而更糟" + 病毒载量在降/CD4 在升 → IRIS**；答案是**继续 ART + 对症（重则激素）**，**不是停 ART、不是换方案、不是诊断治疗失败**。

## 六、Memory Hook（救命口诀总集）

```
HIV 初治两步走
  ① 人人立即三药 ART（不等 CD4、不管症状、不管病载）
  ② 看 CD4 门槛加预防（200 PCP / 100 弓形虫+IgG⊕ / 50 MAC评估）
  选项拆开 → 常是"两个都要"；两药/单药 = 耐药错

初始评估清单
  4 查：CD4 / 病毒载量 / 耐药基因型 / 弓形虫 IgG
  3 筛合并感染：TB / HBV+HCV / 梅毒等 STI
  补疫苗：灭活优先；CD4<200 禁活疫苗(MMR/水痘/黄热)

ART 方案
  首选骨架 = INSTI(多替拉韦/比克替拉韦) + 2 NRTI(替诺福韦+恩曲他滨)
  确诊即启 "treat all treat now"

IRIS（上 ART 后反而更糟）
  时间：ART 后数周（2–8w）
  本质：免疫重建打潜伏抗原（TB/MAC/隐球菌/CMV）
  处理：继续 ART + 治病原 + 重则激素 → 千万别停 ART
```

## 七、复习题

> [!question]- Q1：37 岁男，3 周前新诊断 HIV，无症状，CD4 160，病毒载量 25,000。最佳治疗？
> **三药联合 ART + 同时启动 PCP/PJP 一级预防（TMP-SMX）**。两个动作都要：确诊即启 ART（不因无症状/CD4 而等），且 CD4<200 跨过 PCP 预防阈值。只选 ART 漏预防 = 错（即 Q093 错点）。

> [!question]- Q2：新诊断 HIV，CD4 480，无症状。是否现在启动 ART？
> **是，立即启动**。现指南"treat all, treat now"——确诊即启，**不以 CD4 为门槛**；CD4 高不是延迟理由。此 CD4>200 暂不需 PCP 预防。

> [!question]- Q3：HIV 患者启动 ART 后 4 周，出现发热 + 颈部淋巴结肿大 + 原有肺部病灶炎症加重；复查病毒载量较基线下降、CD4 上升。最可能机制 + 处理？
> **IRIS（免疫重建炎症综合征）**。病毒载量降 + CD4 升 = ART 有效，排除治疗失败；是免疫重建对潜伏病原（此处提示 TB/分枝杆菌）的炎症反应。处理：**继续 ART + 针对病原治疗 + 重症加糖皮质激素**，**不停 ART**。

> [!question]- Q4：新诊断 HIV 初诊应开哪些基线合并感染筛查？
> **TB（IGRA/TST）、HBV、HCV、梅毒等 STI、弓形虫 IgG**；加基线 **CD4 / 病毒载量 / 基因型耐药**；补**灭活疫苗**（CD4<200 禁活疫苗）。

# 感染_HCV 相关疾病（Hepatitis C Virus, 丙型肝炎病毒）

> [!info] 一句话定位
> HCV 是一个 **RNA 病毒（Flaviviridae 黄病毒科）**，血源传播为主。Step 2 考点分两大块：
> ① **肝内**——慢性化率高（75–85%）→ 肝硬化 → 肝细胞癌（HCC, hepatocellular carcinoma）；
> ② **肝外表现（extrahepatic manifestations）**——这是 HCV 区别于其他肝炎、也是 CK（Clinical Knowledge, 临床知识）出题最爱的部分：**混合型冷球蛋白血症（MCS）、膜增生性肾炎（MPGN）、B 细胞淋巴瘤、迟发性皮肤卟啉症（PCT）、扁平苔藓**。
> 核心记忆锁：**看到"IVDU / 输血史 + 皮疹 / 肾炎 / 淋巴瘤 / 光敏水疱" → 查 anti-HCV。**

---

## 一、病毒学 + 传播（Virology & Transmission）

- **HCV = 单链 RNA 病毒（ssRNA）**，无 DNA 中间体（对比 **HBV（乙型肝炎病毒 hepatitis B virus）** 是部分双链 DNA、有逆转录）；**无疫苗**（高变异性，包膜糖蛋白 E1/E2 易突变）。
- **主要传播 = 血液（parenteral）**：
  - **IV 吸毒（IVDU, injection drug use, 静脉注射吸毒）共用针具** = 美国当前首要途径 ⭐
  - 1992 年前输血（现已筛查，罕见）
  - 针刺伤 / 血透 / 文身
- **性传播 + 母婴传播（vertical）效率低**（远低于 HBV / HIV）——但 HIV 共感染时垂直传播率升高。

> [!tip] HCV vs HBV 传播记忆
> **HBV = 三条路都高效（血 / 性 / 母婴），有疫苗**；**HCV = 主要靠血（IVDU），性/母婴低效，无疫苗**。stem 里"男男性行为 / 多性伴"更指向 HBV/HIV；"IVDU / 输血"更指向 HCV。

---

## 二、自然史（Natural History）⭐

```
急性 HCV 感染（Acute, 多数无症状/亚临床）
   │  ~15–25% 自发清除
   ├──────────────► 痊愈
   │
   │  ~75–85% 慢性化 ⭐（慢性化率远高于 HBV 成人的 <5%）
   ▼
慢性 HCV（Chronic, 常无症状带毒 20–30 年）
   │  ~20–30% 在 20–30 年内进展
   ▼
肝硬化（Cirrhosis）
   │  每年 ~1–4%
   ▼
肝细胞癌（HCC, hepatocellular carcinoma）
```

> [!warning] 高频对比：慢性化率
> - **HCV 成人急性感染 → 慢性化 ~75–85%**（高！）
> - **HBV 成人急性感染 → 慢性化 <5%**；但**围产期 HBV 感染慢性化 >90%**（年龄倒挂）
> - 记忆：**"HCV 静悄悄变慢性，HBV 成人多痊愈"**。

- **HCV 促 HCC 通常需先经肝硬化**（对比 **HBV 可不经肝硬化直接致 HCC**，因 HBV DNA 整合入宿主基因组）。这是 HBV vs HCV 致癌机制的经典鉴别点。

## 三、诊断与血清学（Diagnosis & Serology）⭐

> [!info] 两步法（USMLE 必考流程）
> **① 筛查 = anti-HCV 抗体（酶免疫法）** → 阳性说明"曾暴露"（不能区分现症 vs 既往已清除）。
> **② 确诊 / 判断现症 = HCV RNA（PCR，定量）** → RNA 阳性 = **现症活动性感染**；RNA 阴性 = 已清除（自发或治疗后）。

| 结果组合 | 解读 |
|---|---|
| anti-HCV (+) + HCV RNA (+) | **现症感染**（急性或慢性）|
| anti-HCV (+) + HCV RNA (−) | 既往感染**已清除**（自愈或治愈）；抗体可终身阳性 |
| anti-HCV (−) + HCV RNA (+) | **急性早期窗口期** 或免疫抑制宿主（抗体未产生）|
| anti-HCV (−) + HCV RNA (−) | 无感染 |

- **抗体阳性不代表有保护性免疫**——HCV 抗体**非中和性**，可反复再感染（对比乙肝表面抗体 = 保护性）。
- 治愈的判定 = **SVR（sustained virologic response, 持续病毒学应答）** = 停药后 12 周 HCV RNA 检测不到。

---

## 四、筛查（Screening）⭐

> [!success] USPSTF（美国预防服务工作组 US Preventive Services Task Force）/ CDC（美国疾控中心 Centers for Disease Control）现行建议（高频）
> - **所有成人 18–79 岁一生至少筛查一次**（不再局限于"婴儿潮 baby boomer 1945–1965 出生"这一旧标准，但后者仍是重点人群）。
> - **每次妊娠都筛查**（每次 pregnancy）。
> - **高危人群反复筛查**：现/既往 IVDU、血透、HIV 感染、1992 年前输血/器官移植、职业暴露、HCV 母亲所生婴儿。

## 五、治疗（Treatment）

- **直接抗病毒药（DAAs, direct-acting antivirals）** = 现代标准：
  - **NS5A 抑制剂（-asvir，如 ledipasvir 来迪帕韦、velpatasvir 维帕他韦）**
  - **NS5B 聚合酶抑制剂（-buvir，如 sofosbuvir 索磷布韦）**
  - **NS3/4A 蛋白酶抑制剂（-previr，如 glecaprevir 格卡瑞韦）**
  - 常用固定复方：**sofosbuvir/velpatasvir、glecaprevir/pibrentasvir**。
- **治愈率 >95%**，疗程 8–12 周，口服，副作用小（**已淘汰旧的 interferon 干扰素 + ribavirin 利巴韦林方案**——后者副作用大：干扰素致流感样/抑郁、利巴韦林致溶血性贫血 + 致畸）。
- **治疗前必查 HBV 血清学** ⚠️：HCV/HBV 共感染者用 DAA 治 HCV 可致 **HBV 再激活（reactivation）**——需同时评估是否抗 HBV 治疗。
- **成功清除 HCV（SVR）可显著改善甚至逆转部分肝外表现**（尤其冷球蛋白血症、部分 PCT）。

## 六、肝外表现（Extrahepatic Manifestations）⭐⭐⭐

> [!danger] 这是 HCV 在 CK 的"灵魂考点"
> HCV 是**慢性抗原刺激 → B 细胞克隆增殖 + 免疫复合物沉积**的典型病毒。任何"多系统 + 病毒血清学"题里出现皮疹/肾炎/淋巴瘤/光敏，都要想到 HCV。

```
HCV 慢性感染
   │
   ├─ 免疫复合物 / B 细胞激活
   │     ├─ 混合型冷球蛋白血症 MCS ⭐（最经典）
   │     │      ├─ 皮肤：可触性紫癜（palpable purpura）
   │     │      ├─ 肾：膜增生性肾炎 MPGN（RBC 管型 + 蛋白尿）
   │     │      ├─ 神经：周围神经病
   │     │      └─ 关节痛 + 低补体
   │     └─ B 细胞非霍奇金淋巴瘤（尤其边缘区 / 脾）
   │
   ├─ 皮肤
   │     ├─ 迟发性皮肤卟啉症 PCT（光暴露部位水疱/脆性）⭐
   │     └─ 扁平苔藓 Lichen planus（口腔/皮肤）
   │
   └─ 内分泌 / 代谢
         ├─ 桥本样甲状腺炎 / 甲状腺自身抗体
         └─ 2 型糖尿病 / 胰岛素抵抗
```

### 6.1 混合型冷球蛋白血症（Mixed Cryoglobulinemia Syndrome, MCS）⭐ 最高频

> [!danger] 指纹三联（记死）
> **抗 HCV 阳性 + 类风湿因子（RF）阳性 + 低补体（尤其 C4↓↓）** = 混合型冷球蛋白血症。

- **机制**：HCV → 慢性 B 细胞激活 → 产生**冷球蛋白 = 多克隆 IgG + 带 RF 活性的单克隆 IgM**（<37°C 沉淀）→ 免疫复合物**激活并消耗补体（→ 低补体）** → 沉积于**小血管** → 小血管炎（small vessel vasculitis, SVV）。
- **临床五联**（Meltzer 三联 = 紫癜 + 关节痛 + 乏力）扩展：
  - **可触性紫癜（palpable purpura，下肢）** ⭐
  - **膜增生性肾小球肾炎（MPGN）**：血尿 + RBC 管型 + 蛋白尿 + Cr↑
  - **周围神经病（peripheral neuropathy）**
  - **关节痛 + 全身乏力**
  - **肝大**
- **关键鉴别锁**：
  - **RF 阳性 ≠ 类风湿关节炎**——MCS 的 RF 来自冷球蛋白里带 RF 活性的 IgM（结合 IgG 的 Fc），不是 RA。
  - **补体分流**（SVV 病因判断黄金第一步）：
    - **低补体 = 免疫复合物型** → MCS / SLE（系统性红斑狼疮 systemic lupus erythematosus）/ 链球菌后肾炎（PSGN, poststreptococcal glomerulonephritis）/ 膜增生性肾炎（GN = glomerulonephritis, 肾小球肾炎）
    - **正常补体 = pauci-immune（寡免疫）** → **ANCA（抗中性粒细胞胞浆抗体 anti-neutrophil cytoplasmic antibody）** 相关（GPA 肉芽肿性多血管炎 / MPA 显微镜下多血管炎 / EGPA 嗜酸性肉芽肿性多血管炎）/ IgA 血管炎 / 抗 GBM（肾小球基底膜 glomerular basement membrane）
- **治疗**：**根治病因 = 抗 HCV（DAA）**；重症血管炎/肾炎加免疫抑制（糖皮质激素 ± 利妥昔单抗 rituximab ± 血浆置换）。
- 详见错题卡 [[mistakes/uworld-mistakes_2026-06#^Q2232]] 与 [[完整笔记/专题笔记/免疫/免疫_血管炎大小血管谱系]]（补体分水岭）。

### 6.2 膜增生性肾小球肾炎（Membranoproliferative GN, MPGN）

- HCV 是 **MPGN 最常见的可识别病因**（多经冷球蛋白介导）。
- 表现：**肾病范围蛋白尿 + 血尿 + RBC 管型 + 低补体（C3、C4 均可↓）**；可呈肾炎-肾病重叠综合征。
- 光镜"**双轨征（tram-track，基底膜双层化）**"。
- 与 [[完整笔记/专题笔记/肾脏/肾脏_肾炎肾病综合征对照]] 交叉。

### 6.3 B 细胞非霍奇金淋巴瘤（B-cell Non-Hodgkin Lymphoma）

- 慢性 HCV → 持续 B 细胞克隆刺激 → **边缘区淋巴瘤 / 脾边缘区淋巴瘤**风险升高。
- 部分低级别淋巴瘤在 **HCV 清除（DAA 治愈）后可缓解**——病因治疗即抗病毒。

### 6.4 迟发性皮肤卟啉症（Porphyria Cutanea Tarda, PCT）⭐

> [!warning] "光暴露部位水疱 + 皮肤脆性↑ + 多毛" → 查 HCV（及铁、酒精、雌激素、HIV）
> - 机制：肝脏 **尿卟啉原脱羧酶（UROD）活性↓** → 卟啉堆积 → 光敏。
> - **诱因（"HCV 是最常见感染性诱因"）**：HCV、**铁过载/血色病 HFE 突变**、酒精、雌激素、HIV。
> - 表现：**手背等曝光部位无痛性水疱/大疱、糜烂、瘢痕、多毛、皮肤脆性增加**；尿在 Wood 灯下呈**粉红/橙红色荧光**。
> - 处理：戒酒/停雌激素、**放血疗法（phlebotomy）降铁**、低剂量羟氯喹、**治疗 HCV**、防晒。
> - 与 [[完整笔记/专题笔记/皮肤/皮肤_水疱大疱病鉴别]] 交叉。

### 6.5 扁平苔藓（Lichen Planus）

- **6 个 P**：Pruritic, Purple, Polygonal, Planar, Papules/Plaques（+ 口腔 **Wickham 纹**网状白纹）。
- 与 HCV 有流行病学关联——CK 里"扁平苔藓患者"可作为筛查 HCV 的线索。

### 6.6 内分泌 / 代谢

- **甲状腺自身免疫**（桥本样甲状腺炎、甲状腺抗体阳性）——尤其旧的干扰素治疗可诱发/加重甲状腺功能异常。
- **2 型糖尿病 / 胰岛素抵抗**风险升高。

## 七、肝硬化并发症（Cirrhosis Complications）——HCV 是主要病因之一

HCV 肝硬化失代偿后的并发症谱（详见 GI 专题）：

| 并发症 | 关键点 | 交叉笔记 |
|---|---|---|
| **门脉高压 / 食管静脉曲张** | 非选择性 β 阻滞剂一级预防 + 内镜套扎 | [[完整笔记/专题笔记/GI/GI_门脉高压治疗]] |
| **肝肾综合征（HRS, hepatorenal syndrome）** | 容量复苏（NS 生理盐水 normal saline + albumin 白蛋白）**无改善** + UA（尿常规 urinalysis）bland（无 cells/casts/蛋白）；缩血管药（terlipressin 特利加压素/去甲肾上腺素 norepinephrine）+ albumin | [[完整笔记/专题笔记/GI/GI_HRS肝肾综合征]]、错题 [[mistakes/uworld-mistakes_2026-05#^Q4752]] |
| **自发性细菌性腹膜炎（SBP, spontaneous bacterial peritonitis）** | 腹水 PMN（多形核中性粒细胞 polymorphonuclear）≥250；头孢噻肟 cefotaxime；SBP 后加 albumin 防 HRS | [[完整笔记/专题笔记/感染/感染_感染性腹泻]]（相邻主题参考）|
| **肝性脑病（HE, hepatic encephalopathy）** | 乳果糖 lactulose + 利福昔明 rifaximin；找诱因（感染/出血/电解质）| — |
| **低钠血症（稀释性）** | 无症状肝硬化低钠 → **限水即可，不需 HCTZ（氢氯噻嗪 hydrochlorothiazide）**（噻嗪 thiazide 反而加重）| 错题 [[mistakes/uworld-mistakes_2026-05#^Q106381]] |
| **肝细胞癌（HCC）筛查** | 肝硬化 → **超声 ± AFP（甲胎蛋白 alpha-fetoprotein）每 6 个月** | [[完整笔记/专题笔记/GI/GI_HighYield速查总览]] §9.4 |

> [!tip] HRS 一句话锚
> **肝硬化 + 进行性 Cr↑ + UA 干净（无 cells/casts/protein）+ 补液无效** = 肝肾综合征（功能性肾衰，肾本身正常）。

## 八、高频陷阱（Traps）

> [!warning] Step 2 常见坑
> - **anti-HCV 阳性 ≠ 现症感染** → 必须查 **HCV RNA** 确认活动性（区分已清除）。
> - **anti-HCV 阳性 ≠ 有免疫力** → 抗体非中和性，可再感染（≠ 乙肝表面抗体）。
> - **RF 阳性 + 低补体 + 紫癜** → 先想 **HCV 冷球蛋白血症**，别一头扎进 RA/SLE。
> - **光暴露部位水疱 + 皮肤脆** → 查 HCV / 铁 / 酒精（PCT），不是单纯晒伤或 PCT 只想到血色病。
> - **HCV 致 HCC 通常需先有肝硬化**；HBV 可不经肝硬化直接致癌（DNA 整合）。
> - **DAA 治 HCV 前查 HBV** → 防 HBV 再激活。
> - **慢性化率**：HCV 成人 ~75–85%（高）；HBV 成人 <5%、围产期 >90%。

## 九、复习题（先思考，答案自查）

> [!question]- Q1. 50 岁男性，既往 IVDU 史，数周下肢可触性紫癜 + 关节痛 + 乏力，UA 见血尿 + RBC 管型，化验示低补体 + RF 阳性 + ANCA 阴性。最可能诊断？下一步确诊病因查什么？
> **诊断**：混合型冷球蛋白血症综合征（MCS，HCV 相关小血管炎）。
> **病因确认**：查 **anti-HCV → HCV RNA**（指纹三联：抗 HCV + RF + 低补体）；肾累及即冷球蛋白介导的 **MPGN**。治疗 = 抗 HCV（DAA）± 利妥昔单抗/激素/血浆置换。

> [!question]- Q2. 患者 anti-HCV 抗体阳性，但 HCV RNA 检测不到。如何解读？
> **既往感染已清除**（自发清除或治疗后 SVR）。抗体可终身阳性且**无保护性**（非中和抗体，可再感染）。无需抗病毒治疗，但需评估既往肝损/纤维化。

> [!question]- Q3. 中年男性手背等曝光部位反复无痛性水疱、糜烂、瘢痕 + 皮肤脆性增加 + 面部多毛，尿液 Wood 灯下橙红荧光。诊断？应筛查哪些诱因？
> **迟发性皮肤卟啉症（PCT）**，UROD 活性↓。筛查诱因：**HCV、铁过载/血色病、酒精、雌激素、HIV**。处理：放血降铁 + 戒酒/停雌激素 + 低剂量羟氯喹 + 治 HCV + 防晒。

> [!question]- Q4. 为什么慢性 HCV 相比 HBV 更"沉默"却更容易变成慢性？两者致 HCC 机制差异？
> HCV 急性感染多无症状、慢性化率 **75–85%**（HBV 成人 <5%）。**HCV 致 HCC 通常需经肝硬化**；**HBV 可不经肝硬化直接致 HCC**（病毒 DNA 整合入宿主基因组 + 直接致癌）。

> [!question]- Q5. HCV 肝硬化患者因 HE + SBP 入院治疗后神志改善，但 Cr 从 1.1 进行性升至 3.9，UA 无 cells/casts/protein，肾脏 US 正常，IV NS + albumin 无改善。最可能机制？
> **肝肾综合征（HRS）**——内脏血管扩张 → 有效循环容量↓ → 肾血管收缩性低灌注（功能性，肾实质正常）。治疗：缩血管药（terlipressin/去甲肾上腺素）+ albumin；确定性治疗 = 肝移植。

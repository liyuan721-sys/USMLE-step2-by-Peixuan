# 地方性双相真菌鉴别（Endemic Dimorphic Fungi）

> [!info] 一句话框架
> **地方性双相真菌（dimorphic fungi）= 环境中霉菌（mold，25℃）、体内酵母（yeast，37℃）**。吸入孢子 → **肺**（多自限）→ 免疫缺陷或高暴露 → **血行播散**。CK 三步：**① 地区/暴露 → ② 形态钥匙（镜下）→ ③ 宿主（免疫正常 vs 抑制）定播散风险 + 选药**。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q2999]]（芽生菌·疣状皮损，误选曲霉）+ [[mistakes/uworld-mistakes_2026-06#^Q4938]]（播散性组织胞浆菌·AIDS → 尿抗原，误选淋巴结活检）。

---

## 一、四菌总表（CK 高频）⭐⭐

| 真菌 | 地区 / 暴露 | 形态钥匙（37℃ 组织相）| 招牌临床 | 高危播散 |
|---|---|---|---|---|
| **组织胞浆菌 Histoplasma capsulatum** | **密西西比 / 俄亥俄河谷**；**鸟 / 蝙蝠粪**、洞穴、拆迁 | **巨噬细胞内小酵母**（intracellular，2-4μm）| 多无症 / 肺；播散 = 发热 + 肝脾大 + **全血细胞↓** + 黏膜溃疡 | **AIDS（CD4<150）**、TNF-α 抑制剂 |
| **芽生菌 Blastomyces dermatitidis** | **中西部 / 大湖 / 密西西比-俄亥俄**；户外 / 河岸土壤 | **宽基出芽酵母**（broad-based budding，大）| 肺（可似细菌肺炎/肿块）+ **疣状紫红皮损**（verrucous）+ 骨 / 前列腺 | **免疫正常也可播散** ⭐ |
| **球孢子菌 Coccidioides** | **西南沙漠 / 加州中央谷 / 亚利桑那**；沙尘、地震 | **球囊 spherule**（内含内孢子 endospores）| **Valley fever**：肺 + **结节红斑 / 多形红斑 + 关节痛**（免疫反应）| 黑人 / 菲律宾裔、**孕晚期**、免疫抑制 → 播散（脑膜炎）|
| **孢子丝菌 Sporothrix schenckii** | **玫瑰刺 / 园艺 / 苔藓**（"rose gardener"）| 雪茄形酵母（cigar-shaped）| **淋巴皮肤型**：接种处溃疡 + **沿淋巴链串珠结节**（lymphocutaneous）| 少播散（免疫抑制 / 酗酒肺型）|

> [!tip] 第五个（少考）：副球孢子菌 Paracoccidioides
> **中南美洲**；**舵轮样多芽酵母（"pilot's wheel"）**；口腔黏膜溃疡 + 肺 + 淋巴结。北美 CK 少见。

---

## 二、形态钥匙 = 一眼定位（镜下最爱考）⭐

```
组织胞浆菌 → 巨噬细胞内 小酵母（intracellular）
芽生菌     → 宽基出芽 broad-based budding（大酵母）
球孢子菌   → 球囊 spherule + 内孢子（endospores）
孢子丝菌   → 雪茄形 cigar-shaped 酵母 + 淋巴链串珠
副球孢子菌 → 舵轮 pilot's wheel（多芽）
```

> [!warning] 双相真菌（酵母）vs 霉菌（mold）的分流 — Q2999 的坑
> **题里写"湿片 / 痰镜见酵母（yeast）" → 立刻排霉菌（曲霉 Aspergillus / 毛霉 Mucor）**（霉菌见**菌丝 hyphae / 分生孢子**）。曲霉/毛霉是**霉菌、侵袭性、主攻免疫抑制**；双相真菌组织相是**酵母**（球孢子菌例外 = 球囊）。**见酵母 + 地方性流行病学 = 双相真菌，不是曲霉。**

---

## 三、诊断 — 选检查（确诊路径高频）⭐

| 真菌 | 首选/最快 | 备注 |
|---|---|---|
| **组织胞浆菌** | **尿 / 血清抗原**（敏感性 >95%，快；播散性尤佳）| 培养确诊但慢 4-6 周；**活检/慢培养留备选**（[[mistakes/uworld-mistakes_2026-06#^Q4938]] 误选淋巴结活检）|
| **芽生菌** | 镜检/培养见**宽基出芽** + 尿抗原（与组织胞浆交叉反应）| 抗原特异性较低 → 结合形态/培养 |
| **球孢子菌** | **血清学（IgM/IgG）** + 镜检见**球囊** | 沙漠地区 CAP（community-acquired pneumonia 社区获得性肺炎）不缓解 → 查血清学 |
| **孢子丝菌** | **培养**（金标准）| 镜检常阴；按典型淋巴皮肤型 + 园艺史临床诊断 |

> [!info] 通法
> **"地方性真菌 → 抗原 / 血清学 + 镜检形态，优先于侵入活检与慢培养"**。播散性组织胞浆菌尤其：**尿抗原高敏无创 = 首选**。

---

## 四、治疗（按严重度 / 宿主）⭐

> [!tip] 统一原则
> **轻-中度 / 免疫正常 → 唑类（itraconazole 伊曲康唑为主）；重度 / 播散 / 免疫抑制 / CNS（central nervous system 中枢神经系统）/ 孕妇 → 两性霉素 B（amphotericin B）起始 → 唑类维持。**

| 真菌 | 轻中度 | 重度 / 播散 / 免疫抑制 |
|---|---|---|
| **组织胞浆菌** | itraconazole（伊曲康唑）| **amphotericin B**（两性霉素 B）→ itraconazole 维持（AIDS）|
| **芽生菌** | itraconazole | amphotericin B（重度 / CNS / 免疫抑制 / 孕妇）|
| **球孢子菌** | 多数自限（观察）；持续/重症 fluconazole（氟康唑）| amphotericin B；**脑膜炎 = 终身 fluconazole** |
| **孢子丝菌** | **itraconazole**（淋巴皮肤型首选）| amphotericin B（播散 / 肺）|

## 五、一眼定位决策（考试反射）⭐

```
见"地方性真菌"题 → 先抓 3 个维度：
  ① 地区/暴露
     鸟/蝙蝠粪·密西西比俄亥俄 → 组织胞浆
     大湖/中西部·户外土壤      → 芽生菌
     西南沙漠/加州中央谷·沙尘  → 球孢子菌
     玫瑰刺/园艺               → 孢子丝菌
  ② 形态（镜下）
     胞内小酵母 / 宽基出芽 / 球囊 / 雪茄形（见上 §二）
  ③ 宿主 + 招牌
     AIDS<150 + 肝脾大 + 全血细胞↓ + 黏膜溃疡 → 组织胞浆（→尿抗原）
     免疫正常 + 疣状紫红皮损                  → 芽生菌
     结节红斑 + 关节痛（valley fever）         → 球孢子菌
     沿淋巴链串珠结节                          → 孢子丝菌
```

> [!warning] 高频陷阱集合
> - **见酵母排曲霉/毛霉（霉菌）**（[[mistakes/uworld-mistakes_2026-06#^Q2999]]）。
> - **播散组织胞浆菌 → 尿抗原，别先活检/等培养**（[[mistakes/uworld-mistakes_2026-06#^Q4938]]）。
> - **芽生菌免疫正常也播散** — 别因"免疫正常"排除真菌。
> - **球孢子菌孕晚期 + 黑人/菲律宾裔 → 播散风险高**；脑膜炎要终身氟康唑。
> - **结节红斑 erythema nodosum = 球孢子菌的免疫反应（好预兆）**，不是直接感染皮损（区别于芽生菌疣状皮损 = 真感染）。

# 单核样综合征鉴别（EBV 真传单 vs 嗜异性阴性模仿者）

> [!info] 笔记定位
> **"单核样综合征 (mononucleosis syndrome)"= 发热 + 咽炎 + 淋巴结 + 乏力 ± 脾大 + 血涂片异型淋巴细胞**这一套**临床+血象**，**不是某一病原专利**。
> Step 2 CK 核心：① **传单 (IM) 专指 EBV**（异嗜性抗体⊕）② **Monospot ⊖ 不能排除** → 想 4 个"嗜异性阴性模仿者"③ **急性 HIV 绝不能漏**（抗体窗口期 → 查 RNA）。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q4354]]（EBV 传单 → 嗜异性抗体）。

---

## 一、术语先理清 ⭐⭐⭐

> [!warning] "传单"是专有名词，"单核样综合征"是大伞
> - **传染性单核细胞增多症 (infectious mononucleosis, IM) = 专指 EB 病毒 (EBV)** 引起的综合征 → **异嗜性抗体⊕（Monospot⊕，~90%）**。**只有这个叫"传单"。**
> - **巨细胞病毒 (CMV) / 急性 HIV / 弓形虫 (Toxoplasma gondii) / HHV-6** 引起的是**长得很像、但异嗜性抗体阴性**的 **"类传单 / 单核样综合征"**（heterophile-negative mononucleosis）——**不叫"传单"，是模仿者**。
> - **异型淋巴细胞 (atypical/reactive lymphocytes) = 反应性 T 细胞**，对上述**任何**病原都可出现 → **看见异型淋巴别直接 = EBV**，要靠 **Monospot + 各自特异检测**定病原。

---

## 二、鉴别总表 ⭐⭐⭐

| 病原 | 是否"传单" | 异嗜性抗体 | 鉴别钩子 | 确诊 |
|---|---|---|---|---|
| **EB 病毒 (EBV)** | **✅ 真传单 (IM)** | **⊕** | **渗出咽炎重 + 后颈淋巴结 + 脾大 + 乏力迁延**；青少年；**氨基青霉素→皮疹** | Monospot / **VCA-IgM** |
| **巨细胞病毒 (CMV)** | ❌ 类传单 | **⊖** | **最常见的嗜异性阴性 mono**；**咽炎/淋巴结较轻、肝炎/转氨酶↑突出**；免疫低下更重 | CMV PCR / IgM |
| **急性 HIV (acute retroviral syndrome)** | ❌ 类传单 | **⊖** | **发热 + 咽炎 + 皮疹 + 口腔/生殖器黏膜溃疡 + 淋巴结** + 高危史 | **HIV RNA（病毒载量）**，非抗体（窗口期⊖）|
| **弓形虫 (Toxoplasma gondii)** | ❌ 类传单 | **⊖** | **颈淋巴结肿大为主、全身症状轻**；免疫低下→脑弓形虫 | Toxo 血清学 (IgM/IgG) |
| **HHV-6** | ❌ 类传单 | **⊖** | **婴幼儿 = 幼儿急疹 (roseola)**；成人偶见类传单 | 临床 / PCR |

> [!tip] 一句话区分模仿者
> **CMV = 肝炎重咽炎轻**；**急性 HIV = 皮疹 + 黏膜溃疡 + 高危史（查 RNA）**；**弓形虫 = 淋巴结为主**；**HHV-6 = 婴儿玫瑰疹**。

---

## 三、EBV 真传单（IM）要点

> [!info] 经典三联 + 高频陷阱
> - **临床**：青少年 + **渗出性咽炎 + 后颈/全身淋巴结 + 脾大 + 显著乏力（迁延数周）**。
> - **血象**：淋巴细胞增多 + **异型淋巴细胞**；可一过性肝酶↑。
> - **诊断**：**异嗜性抗体 (Monospot)**；早期/<4 岁假阴性 → 查 **EBV VCA-IgM**。
> - **陷阱 1**：误当链球菌咽炎给**阿莫西林/氨苄西林 (amoxicillin/ampicillin) → 90%+ 麻疹样疹**（非过敏，详见 [[完整笔记/专题笔记/感染/感染_青霉素后皮肤事件三鉴别]]）。
> - **陷阱 2**：**脾大 → 避免对抗性运动 3–4 周（防脾破裂）**。
> - **并发症**：脾破裂、自身免疫性溶血、气道梗阻（扁桃体肿）、罕见 EBV 相关淋巴增殖。

---

## 四、异嗜性抗体阳性/阴性谱（与本主题强相关）

> [!info] Monospot 判读速查
> - **⊕**：**EBV 真传单**（绵羊/马红细胞凝集，Paul-Bunnell 抗体）。
> - **假阴性**：**病程早期(<1 周) / <4 岁幼儿** → 复查或查 **VCA-IgM**。
> - **⊖ 但单核样**：想 **CMV / 急性 HIV / 弓形虫 / HHV-6**（本笔记 §二）。
> - **假阳性**：偶见淋巴瘤 (lymphoma) / 系统性红斑狼疮 (SLE) / 病毒性肝炎。
> - ⚠️ **另一概念**：实验室"异嗜性抗体干扰"（HAMA）→ **幻影 hCG (phantom hCG)**（血⊕尿⊖ hCG 不一致），与本主题无关但同名易混。

---

## 五、为什么必须分清（考点意义）⭐

> [!danger] 三个"别漏 / 别误判"
> 1. **Monospot ⊖ ≠ 排除单核样综合征** → 阴性时主动想 4 个模仿者。
> 2. **急性 HIV 绝不能漏**——表现最像传单（发热咽炎皮疹溃疡），但**抗体窗口期可⊖ → 必须查 HIV RNA**。这是高频"伪装成传单"的题。
> 3. **氨基青霉素疹也可见于 CMV 等**（不只 EBV）——同样**不是过敏**，别终身标"青霉素过敏"。

## 六、Memory Hook

```
"传单 = EBV 专属（嗜异性⊕）；其余 4 个 = 类传单（嗜异性⊖）"
共享:发热+咽炎+淋巴结+乏力±脾大+异型淋巴细胞(反应性T,非EBV专利)

Monospot⊖ + 单核样 → 想四模仿者:
  CMV  = 肝炎重咽炎轻（最常见嗜异性阴性）
  急性HIV = 皮疹+黏膜溃疡+高危史 → 查HIV RNA（别漏！）
  弓形虫 = 颈淋巴结为主
  HHV-6 = 婴儿玫瑰疹roseola

EBV陷阱:氨基青霉素→皮疹(非过敏);脾大→停运动防破裂
早期/<4岁 Monospot假阴 → 查VCA-IgM
```

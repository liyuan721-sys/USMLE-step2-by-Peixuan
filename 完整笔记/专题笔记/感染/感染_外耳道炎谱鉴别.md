# 感染_外耳道炎谱鉴别（耳痛/流脓鉴别）

> [!danger] 为什么建这张表
> 2026-06-10 ENT 块**两道耳道感染全错**：急性 OE 误选胆脂瘤（[[mistakes/uworld-mistakes_2026-06#^Q15993]]）、坏死性 NOE 误选曲霉（[[mistakes/uworld-mistakes_2026-06#^Q2828]]）。
> 把"耳痛 + 流脓"五个鉴别一次理清。病原细节见 [[完整笔记/专题笔记/感染/感染_常见病原菌总结_按部位综合征]]；主笔记 [[完整笔记/Peixuan分科笔记/感染]]。

> [!info] 缩写速查
> - **OE = Otitis Externa 外耳道炎**（外耳道弥漫炎症的统称）
> - **AOE = Acute Otitis Externa 急性外耳道炎**（= swimmer's ear 游泳者耳，本表所称"急性 OE"）
> - **NOE = Necrotizing Otitis Externa 坏死性外耳道炎** = **MOE = Malignant Otitis Externa 恶性外耳道炎**（旧称"malignant/恶性"指**死亡率高、侵颅底骨髓炎**，**非肿瘤**）

---

## 一、招牌钥匙速查 ⭐

> [!tip] 一句话锁定
> - **急性耳道肿痛 + 脓 + 鼓膜清亮 + 无肉芽** = **急性外耳道炎 OE**（铜绿）
> - **老年糖尿病 + 夜间剧痛 + 耳道底肉芽 ± 面瘫** = **坏死性(恶性)外耳道炎 NOE**（铜绿，颅底骨髓炎）
> - **痒为主 + 白屑上黑色孢子** = **真菌性外耳道炎**（曲霉 Aspergillus）
> - **慢性流脓 + 鼓膜内陷/角化上皮 + 可侵蚀骨** = **胆脂瘤 Cholesteatoma**
> - **鼓膜后积液/穿孔流脓 + 耳道不红肿** = **浆液性中耳炎 / 中耳炎**

## 二、五病大对照表 ⭐⭐

| 病 | 病程 | 招牌体征 | 鼓膜 | 病原/本质 | 处理 | 错题 |
|---|---|---|---|---|---|---|
| **急性外耳道炎 OE** | **急性（数天）** | 耳道**红肿 + 脓**、**牵拉痛** | **清亮（不累及）** | **铜绿假单胞菌 Pseudomonas**（+金葡）| **局部氟喹诺酮滴耳** ± 局部激素 | [[mistakes/uworld-mistakes_2026-06#^Q15993]] |
| **坏死性(恶性)外耳道炎 NOE** | 进展性 | **夜间剧痛 + 耳道底肉芽 ± 面瘫(CN VII)** | 可清亮 | **铜绿**（颅底骨髓炎）| **IV 抗假单胞（环丙沙星 ciprofloxacin）6–8 周** | [[mistakes/uworld-mistakes_2026-06#^Q2828]] |
| **真菌性外耳道炎** | 亚急性 | **痒为主** + 白屑上**黑色孢子** | 清亮 | **曲霉 Aspergillus**（少致 NOE）| 清创 + 局部抗真菌 | — |
| **胆脂瘤 Cholesteatoma** | **慢性（月-年）** | 慢性恶臭流脓、**角化上皮 / 鼓膜内陷囊袋**、侵蚀骨 | **内陷/异常** | 角化鳞状上皮陷入中耳 | 手术 | — |
| **中耳炎 / 浆液性 OM** | 急/慢 | 耳道**不红肿**；鼓膜后积液或穿孔流脓 | **积液/膨隆/穿孔** | 不可分型 H. flu / 肺炎链球菌 / 卡他莫拉菌 | 阿莫西林±克拉维酸 | — |

> [!warning] 危险因素 → OE
> 进水（游泳者耳）、创伤（棉签/挖耳）、**异物（助听器/耳机）**、湿疹/接触性皮炎。
> [[mistakes/uworld-mistakes_2026-06#^Q15993]] 就是**助听器裂耳模刮耳道**（异物+创伤+潮湿）。

## 三、决策树（耳痛/流脓快速分流）

```
耳痛 + 流脓
├─ 耳道红肿 + 牵拉痛 + 鼓膜清亮
│   ├─ 急性、无肉芽 ─────────────→ 急性 OE（铜绿，局部氟喹诺酮）
│   ├─ 老年糖尿病 + 夜间剧痛 + 肉芽 ± 面瘫 → NOE（铜绿，IV 环丙沙星）
│   └─ 痒为主 + 黑色孢子 ─────────→ 真菌性 OE（曲霉）
├─ 慢性恶臭流脓 + 鼓膜内陷/角化上皮 ──→ 胆脂瘤（手术）
└─ 耳道不红 + 鼓膜后积液/穿孔 ───────→ 中耳炎 / 浆液性 OM
```

> [!danger] OE vs NOE 的硬分水岭 = 肉芽 + 全身/颅底征
> 都是铜绿、都耳道病变，但：
> - **OE**：局限软组织、鼓膜清亮、**无肉芽**、CN 正常 → 局部治疗。
> - **NOE**：**耳道底肉芽**、夜间剧痛、**面瘫/发热**（侵颅底骨髓炎）→ **IV 全身抗假单胞 6–8 周**。
> 见肉芽 + 面瘫 + 老年糖尿病 = NOE，别停在 OE。

> [!warning] "糖尿病 + 坏死感染"病原别认错
> - **耳道（NOE）= 铜绿假单胞菌**（不是真菌！[[mistakes/uworld-mistakes_2026-06#^Q2828]] 错点）
> - **鼻脑（毛霉病）= 毛霉 Rhizopus**（鼻甲**黑焦痂**）
> - 真菌性 OE（痒+黑孢子）才是曲霉，且**极少**致 NOE。

## 四、耳痛但耳道 + 鼓膜都正常 → 牵涉性耳痛 referred otalgia

> [!info] 耳全正常的耳痛要换思路
> 上面五病都是**耳源性**（耳道/鼓膜有病）。若**耳道、鼓膜、Weber/Rinne 全正常**却耳痛 → 多为**牵涉性耳痛 referred otalgia**（痛源在别处，经共享神经 CN V/IX/X 投射到耳）：
> - **最常见 = 颞下颌关节紊乱 TMD**：**咀嚼/张口加重的耳痛** + **磨牙 bruxism**（牙面磨损光滑、无龋）+ 焦虑/抗抑郁药史；关节压痛、咔哒声。处理：教育 + **软食 / 避诱因** + 夜间咬合垫 + NSAID（萘普生 naproxen）；**诊断靠临床、不需影像**。（UW Q2194）
> - **牙病**（龋齿/牙周脓肿）：另一大常见牵涉耳痛源。
> - 其他：咽炎/扁桃体、舌咽神经痛（CN IX）、**老年吸烟者警惕头颈鳞癌**牵涉痛（红旗）。

> [!tip] 一句话钥匙
> **耳痛 + 耳道鼓膜全正常 + 咀嚼加重 + 磨牙 → TMD（牵涉性耳痛），不是耳病。**

## 五、急性中耳炎 (AOM) 并发症 → 乳突炎 (mastoiditis)：耳后痛 + 耳廓外移 → 颞骨 CT ⭐（NBME11 Q149）

> [!danger] 何时"不能再只开阿莫西林"
> **急性中耳炎 (acute otitis media, AOM)** 大多口服抗生素即可，但一旦出现下列**乳突炎 / 骨膜下脓肿 (subperiosteal abscess)** 升级信号，需**颞骨 CT (CT scan of temporal bone)** + IV 抗生素 ± 手术：
> - **耳后压痛 / 红肿 (postauricular tenderness/swelling)**
> - **耳廓向外/下移位 (auricle displaced laterally)** ⭐ 高辨识度词
> - 高热 + 脓性分泌 + 耳道水肿

```text
急性中耳炎 (AOM)
   ↓ 感染扩散入乳突气房 (mastoid air cells)
化脓性乳突炎 (suppurative mastoiditis)
   ↓ 耳后红肿压痛 + 耳廓外移
骨膜下脓肿 (subperiosteal abscess)
   → 颞骨 CT 定范围/并发症 + IV 抗生素（± 鼓膜切开/乳突切除）
```

> [!info] 病原与并发症
> 最常见菌：**肺炎链球菌 (S. pneumoniae) / 化脓链球菌 (S. pyogenes) / 金黄色葡萄球菌 (S. aureus)**。严重并发症：脑膜炎、脑脓肿、骨膜下脓肿、**Bezold 脓肿**（脓沿胸锁乳突肌下行的颈部脓肿）、面神经麻痹、永久性听力丧失。

> [!warning] 干扰项识别（NBME11 Q149）
> - **前颈部 1-cm 可动无痛淋巴结** = 反应性淋巴结，**干扰项**，不是主线（别选 Monospot 查 EBV）。
> - Tympanography（鼓室图）查中耳积液 → 远不够评估乳突炎。
> - 氢化可的松/多黏菌素/新霉素滴耳 (hydrocortisone/polymyxin/neomycin drops) = 外耳道炎用，覆盖不了乳突炎。
> - **口服阿莫西林 (oral amoxicillin)** = 单纯 AOM 够，乳突炎不够 → 本题错点。

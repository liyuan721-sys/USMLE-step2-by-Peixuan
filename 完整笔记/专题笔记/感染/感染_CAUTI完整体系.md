# CAUTI 完整体系 — Catheter-Associated Urinary Tract Infection（导尿管相关尿路感染）

> [!info] 定位与边界
> 本笔记是 **CAUTI 的 ID（感染）+ patient safety 视角主笔记**：机制 / 预防 / **神经原性膀胱 CIC 决策** / **无症状菌尿不治** / 诊断 / 干扰项。
> - 它是 [[完整笔记/专题笔记/感染/感染_CLABSI完整体系]] 的**姊妹篇**（同属 device-associated infection 器械相关感染；HAI/HAC 家族）。
> - 导尿管**置入操作 / 禁忌（尿道损伤）/ 操作并发症** → [[完整笔记/专题笔记/USMLE/USMLE_侵入操作安全流程]]（Foley 段），本篇不复制。
> - 派生自 [[mistakes/uworld-mistakes_2026-06#^Q4167]]（神经原性膀胱长期导尿 → CIC）。

> [!note]- 缩写对照（全称 中英）
> - **CAUTI** = catheter-associated urinary tract infection 导尿管相关尿路感染
> - **CIC** = clean intermittent catheterization 清洁间歇导尿
> - **HAI** = healthcare-associated infection 医疗相关（院内）感染
> - **HAC** = hospital-acquired condition 院内获得性情况（CMS 不报销列表）
> - **ASB** = asymptomatic bacteriuria 无症状菌尿
> - **UA** = urinalysis 尿常规
> - **CVC** = central venous catheter 中心静脉导管

## 一、机制：为什么导尿管易感染 ⭐⭐⭐

```
留置导尿管 → 破坏天然屏障 + 提供细菌"高速路"
  │
  ├─ 腔外 extraluminal（最常见）── 细菌沿导管【外壁】上行，24h 内可达膀胱
  │
  └─ 腔内 intraluminal（较少）──── 引流不畅 / 集尿袋污染 → 逆行
  │
  ▼
Biofilm 生物膜沿导管形成 → 屏蔽抗生素 + 难根除
  │
  ▼
风险随【留置天数】线性↑（每留置日菌尿率 ~3-7%；留置 ≥30 天几乎必有菌尿）
```

| 病原体 | 关键点 |
|---|---|
| **E. coli 大肠埃希菌** | 最常见 |
| **Enterococcus 肠球菌 / Pseudomonas 铜绿假单胞菌** | 院内 / 长留置 |
| **Candida 念珠菌** | 长留置 + 长抗生素 + 糖尿病/免疫低 |
| **Proteus mirabilis 奇异变形杆菌** ⭐ | **产尿素酶(urease) → 尿碱化 → struvite 鸟粪石 + 导管结壳(encrustation)/堵管** |

> [!info] CAUTI 在 HAI/HAC 家族里的位置
> 与 CLABSI / VAP（呼吸机相关肺炎）/ SSI（手术部位感染）/ C. diff 同属 **HAI**；也在 **CMS 2008 起 HAC 不报销列表**内 → 医院主动推"减少不必要置管"checklist。详见 [[完整笔记/专题笔记/感染/感染_CLABSI完整体系]] §一/§二。

## 二、预防核心 = 注意事项 ⭐⭐⭐（最有效全在"少用 / 早拔"）

```
最硬道理：减少 + 缩短导管使用（best prevention）
  1. 避免不必要置管 + 每天评估必要性、尽早拔除 ⭐（单一最有效）
  2. 无菌置入(aseptic insertion) + 闭式引流系统(closed drainage system)
  3. 集尿袋始终 < 膀胱水平（防尿液回流）+ 管路不打折(no kinking) + 不触地
  4. 手卫生(hand hygiene) + 妥善固定导管减少移动牵拉
```

> [!success] 神经原性膀胱(neurogenic bladder)需长期排空 → 首选 CIC ⭐
> 当**长期**需排空膀胱（如脊髓损伤 spinal cord injury / 神经原性膀胱），**清洁间歇导尿(CIC)是首选**：
> - 每 **4-6 小时**插入-排空-拔除一次清洁导管；**患者本人或陪护可操作**。
> - **CAUTI 风险显著低于长期留置导管**（不给细菌持续"常驻高速路"，中断腔外+腔内两条路径）。
> - 次选：**留置导管(indwelling，每月更换)**——不能 CIC 时；代价 = UTI / 尿道狭窄 / 膀胱痉挛↑。
> - 备选：**耻骨上膀胱造瘘(suprapubic tube)**。
>
> **本题 Q4167 钥匙**：40 岁脊髓损伤 T10 平面 + 神经原性膀胱长期导尿 → 降 CAUTI 最佳 = **CIC**。

> [!danger] 没用甚至有害的措施（= Q4167 四个干扰项，UWorld 反复考）
> | 措施 | 为什么错 |
> |---|---|
> | 膀胱抗菌冲洗(bladder irrigation with antibacterial) | 杀正常菌群 → **诱导耐药**，不降 UTI |
> | 尿道口抗菌膏(meatal antibacterial application) | **无效** |
> | 外阴/外生殖器抗菌清洗 | **无效** |
> | 预防性全身抗生素(prophylactic systemic antibiotics) | **耐药↑**；抗生素只用于**证实的有症状 UTI** |
>
> **记法**：降 CAUTI 唯一靠谱 = **别插 / 早拔 + 长期用 CIC**；一切**局部消毒、膀胱冲洗、预防性抗生素都没用（反增耐药）**。

## 三、无症状菌尿不治 ⭐⭐（CAUTI 高频配套陷阱）

> [!warning] 留置导管者菌尿"几乎必然"——但不筛、不治
> - 留置导管患者 **asymptomatic bacteriuria (ASB) 极常见**，但 **不常规筛尿培养、不治疗**（治了只会选出耐药菌）。
> - **治疗指征 = 有症状 CAUTI**：发热 / 耻骨上或腰区痛 / 新发谵妄 / 脊髓损伤者**自主神经反射异常(autonomic dysreflexia)** / 脓毒症等，且无其他来源。
> - **唯二要治 ASB 的例外**：**孕妇** + **泌尿外科操作前**（详见 [[完整笔记/专题笔记/OB_GYN/OB_GYN_HighYield速查总览]] 孕妇 UTI）。

> [!tip] 诊断 CAUTI 的套路
> 1. **先换管或拔管，再从【新管/中段尿】留取培养**（旧管定植菌干扰结果）。
> 2. 症状 + **脓尿(pyuria)** + 尿培养阳性（导管尿常 ≥10³ CFU/mL）。
> 3. **脓尿本身不等于感染**——无症状的脓尿/菌尿仍不治。
> 4. 评估并排除其他发热源（血流 / 肺 / 伤口）。

## 四、CK 经典 Stem 模式

> [!question]- 模式 1（Q4167 原型）：神经原性膀胱长期导尿 → 怎样最有效降 CAUTI？
> ✅ **清洁间歇导尿 CIC**；❌ 膀胱冲洗 / 尿道口抗菌膏 / 外阴消毒 / 预防性抗生素（全无效/有害）。

> [!question]- 模式 2：ICU 留置 Foley 第 5 天，尿培养阳性但**无症状**，下一步？
> ✅ **不治（不用抗生素）**；评估是否仍需导管 → **能拔就拔**。例外只有孕妇 / 泌尿术前。

> [!question]- 模式 3：住院患者反复发热查不到源，关键预防/排查动作？
> ✅ **每天评估并尽早拔除不必要的导管**（"导管还在吗？还需要吗？"）——降 CAUTI 单一最有效措施。

> [!question]- 模式 4：长期留置导管 + 反复**碱性尿 + 结石/导管结壳堵管**，想哪个菌？
> ✅ **Proteus mirabilis**（产尿素酶 → struvite 鸟粪石）。

## 五、高频陷阱 ⭐（CK 钓鱼选项）

| 陷阱 | 为什么错 |
|---|---|
| **膀胱抗菌冲洗能防 CAUTI** | ❌ 诱导耐药，不推荐 |
| **尿道口抗菌膏 / 外阴消毒能防 CAUTI** | ❌ 无效 |
| **预防性全身抗生素能防 CAUTI** | ❌ 耐药↑，仅治证实的有症状 UTI |
| **导管患者尿培养阳性 → 立即上抗生素** | ❌ 无症状菌尿**不治**（除孕妇/泌尿术前）|
| **神经原性膀胱长期 → 留置导管最好** | ❌ **CIC 优于留置**（CAUTI 风险更低）|
| **脓尿 = 一定要治** | ❌ 无症状脓尿/菌尿不治 |

## 六、Memory Hooks（一句话锁定）

- **"降 CAUTI 唯一硬道理 = 别插 / 早拔 + 长期用 CIC"** ⭐⭐⭐
- **"神经原性膀胱长期排空 → 清洁间歇导尿(CIC) > 留置导管"** ⭐⭐⭐
- **"膀胱冲洗 / 尿道口抗菌膏 / 外阴消毒 / 预防性抗生素 = 四大无效干扰项"** ⭐⭐
- **"导管患者无症状菌尿/脓尿 → 不治（例外只有孕妇 + 泌尿术前）"** ⭐⭐⭐
- **"留置越久菌尿越必然 → 每天问：导管还需要吗？"** ⭐
- **"长期导管 + 碱性尿 + 结石/结壳堵管 = Proteus（尿素酶 → struvite）"** ⭐
- **"腔外上行(沿外壁)是最常见入侵路；biofilm 屏蔽抗生素"**

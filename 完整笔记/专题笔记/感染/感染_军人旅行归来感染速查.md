# 军人 / 旅行归来感染速查 (Military & Returning-Service-Member ID)

> [!info] 用途与来源
> CK 高频"**军人 / 旅行归来 (returning service member, RSM) + 地理线索 + 暴露史 → 病原**"题型集中一页。出题套路固定：**地点 + 媒介/动物接触 + 一个标志性体征** → 反射诊断 + 治疗。源：Demi (Divine Intervention) RF/Screening/Vaccine 大全 §Military。地名/治疗按 CDC 复核。

> [!tip] 解题三要素
> **① 去过哪（地理）② 接触了什么（媒介/动物/水/尸体）③ 标志体征**——三者任凑两个即可锁病原。

---

## 一、旅行归来感染：地理 + 暴露 → 病原速查表

| 关键线索（地理 / 暴露 / 体征）| 病原 / 诊断 | 媒介 | 一线治疗 |
|---|---|---|---|
| 中东/南美/北非回来 + 上肢**无痛紫色溃疡性丘疹** | **皮肤利什曼病 (cutaneous Leishmaniasis 利什曼病)** | 白蛉 (sandfly) | 两性霉素 B (amphotericin B) 或 巴龙霉素 (paromomycin)；活检见无鞭毛体 amastigotes |
| 热带回来 + 发热/消瘦/**全血细胞减少/肝脾大/皮肤变黑** | **内脏利什曼病 (visceral, kala-azar 黑热病, L. donovani)** | 白蛉 | 葡萄糖酸锑钠 (sodium stibogluconate) 或 amphotericin B |
| 埃及 + **血尿** | **血吸虫 (Schistosoma haematobium 埃及血吸虫)** → 膀胱 SCC | 钉螺/疫水 | 吡喹酮 (praziquantel) |
| 埃及/非洲 + **门脉高压/肝脾大** | **血吸虫 (S. mansoni 曼氏血吸虫)** | 疫水 | praziquantel |
| 南美 + **超高热 + 骨痛 (break-bone) + 出血 + 止血带试验阳性 + 血小板↓** | **登革出血热 (Dengue hemorrhagic fever)** | 伊蚊 (Aedes) | 支持治疗（毛细血管通透性↑）|
| 同登革但**无出血** | **基孔肯雅热 (Chikungunya)** | 伊蚊 | 支持（关节痛突出，别与登革混）|
| 孕妇/胎儿**小头畸形 + 颅内钙化** | **寨卡 (Zika)** | 伊蚊 | 支持 |
| **黄疸 + 肝大**（登革没有）| **黄热病 (Yellow fever)** | 伊蚊 | 支持 + 补液（有疫苗，蛋过敏禁）|
| 俄/中/韩 或 美西南（犹他/新墨/亚利桑那/科罗拉多）+ **接触鼠（鹿鼠/谷仓）** + **双侧间质性肺水肿** | **汉坦病毒 (Hantavirus)** | 鼠排泄物气溶胶 | 支持（无特效药）|
| 夏威夷/海军 + **结膜充血 + 黄疸** | **钩端螺旋体病 (Leptospirosis)** | 疫水/鼠尿 | 多西环素 (doxycycline)；重型 = **Weil 病**（贫血+高肌酐+黄疸，攻击肝肾）|
| **咯血 + 纵隔增宽** ± 颈部坏死溃疡 | **炭疽 (Anthrax, 出血性纵隔炎)** | 吸入/皮肤 | doxycycline 或 氟喹诺酮 (fluoroquinolone)；有军用疫苗 |
| 军人上肢**无痛黑痂 (eschar)** | **皮肤炭疽 (cutaneous anthrax)** —— 军人炭疽最常见型 | 皮肤接触 | doxycycline / fluoroquinolone |
| 接触**动物/动物产品/未灭菌奶** + **波状热 (undulant fever) + 大汗** | **布鲁氏菌病 (Brucellosis)** | 乳/动物 | doxycycline + 利福平 (rifampin)；复发加氨基糖苷 |
| 牛羊接生 + 肺炎/胸痛（培养阴性心内膜炎）| **Q 热 (Q fever, Coxiella burnetii)** | 气溶胶 | doxycycline（**肺炎是 Q 热区别布病的点**）|
| 接触狗/猫/猪 + **血性腹泻** | **空肠弯曲菌 (Campylobacter jejuni)** | 粪口 | 支持（重症 azithromycin）；关联 **吉兰-巴雷综合征 (GBS, Guillain-Barré syndrome) + 反应性关节炎** |
| 接触龟/禽/大量蛋 + **血性腹泻 + 中性粒↑** | **非伤寒沙门菌 (nontyphoidal Salmonella)** | 粪口 | **不治疗**（抗生素延长带菌）|
| 发热/颈强/头痛 + LP：压力不高、WBC↑(淋巴)、蛋白不高、糖不低 | **西尼罗病毒 (West Nile virus)** | 蚊 | 支持 |
| 非洲维和 + 出血/瘀点 + 接触**野生动物/尸体** | **埃博拉 / 马尔堡 (Ebola / Marburg)** | 体液/污染物 | 支持（无特效）|
| 吞咽困难(贲门失弛缓) + 慢性便秘(巨结肠) + S3(扩张型心肌病) | **Chagas 病 (锥虫)** | 锥蝽 (reduviid)；Romaña 征=眶周水肿 | 硝呋替莫 (nifurtimox) / 苄硝唑 (benznidazole) |
| 慢性咳/发热/咯血/盗汗 + 上叶浸润/空洞 | **结核 (TB, tuberculosis)** | 呼吸道 | **RIPE**（利福平 Rifampin/异烟肼 Isoniazid/吡嗪酰胺 Pyrazinamide/乙胺丁醇 Ethambutol）2 月 → 利福平+异烟肼续（别忘 **维生素 B6 吡哆醇**）|
| 军中腹泻最常见 | **诺如病毒 (Norovirus)** | 粪口 | 支持 |

## 二、疟疾预防 (Malaria PPX) — 亚/非/中东

> [!warning] 疟疾分型 + 预防
> - **预防药**：阿托伐醌+氯胍 (atovaquone-proguanil) / 甲氟喹 (mefloquine) / 多西环素 (doxycycline) / 氯喹 (chloroquine，耐药多)。
> - **伯氨喹 (primaquine)** 清除 **P. vivax 间日疟 / P. ovale 卵形疟** 的肝内**休眠子 (hypnozoite)** —— 用前必查 **G6PD 缺乏**（X-连锁，男性，会溶血）。
> - **热型**：间日/卵形 = 隔日热 (tertian)；P. malariae 三日疟 = 四日热 (quartan)；**P. falciparum 恶性疟 = 不规则热**，最凶（**脑型疟、肺水肿、肾衰**）。

## 三、军人疫苗 / 防护 (Military Vaccines & Prophylaxis)

> [!info] 军人常规接种
> **MMRV、Tdap、伤寒、流感、脊灰、甲肝**；按驻地加 **黄热、炭疽、天花**。
> - **医务兵 / 性活跃** → 加 **乙肝 (Hep B)**。
> - **防虫**：皮肤涂 **DEET 避蚊胺**、衣物喷 **氯菊酯 (permethrin)**。
> - **脑膜炎暴发 (N. meningitidis)** → 治疗用头孢曲松 (ceftriaxone) 或环丙沙星；密接 **利福平 (rifampin) 预防**。
> - **去高发钩体病地区** → doxycycline 预防。

## 四、Memory Hooks

> [!success] 一句话锁定
> - **无痛紫溃疡 + 中东归来 = 利什曼**；**血尿 + 埃及 = 血吸虫(haematobium)**。
> - **骨痛热 + 出血 + 止血带阳性 = 登革**；**无出血的登革 = 基孔肯雅**。
> - **鼠 + 双肺间质水肿 = 汉坦**；**黄疸 + 结膜充血 + 海军/夏威夷 = 钩体(Weil)**。
> - **波状热 + 大汗 + 奶/动物 = 布病**；**+ 肺炎 → 改 Q 热**。
> - **龟/禽 + 血性腹泻 = 沙门菌 → 不治**；**咯血 + 纵隔增宽 = 炭疽**。

## 五、附：军人非感染高频（指向专科笔记）

> [!note] TBI / PTSD（军人题常考，机制见神经/精神笔记）
> - **创伤性脑损伤 (TBI, traumatic brain injury)**：**GCS (Glasgow Coma Scale 格拉斯哥昏迷评分)** 分级（≥13 轻=脑震荡 / 9–12 中 / ≤8 重）；首选 **MRI**；病理标志 **弥漫性轴索损伤 (DAI, diffuse axonal injury)**；**禁用激素**；**颅内压 (ICP, intracranial pressure)** ↑ = Cushing 反射（心动过缓+高血压+不规则呼吸），急降 ICP 首选**过度通气**（短期）、甘露醇 (mannitol)（CHF 禁）、高渗盐水。
> - **创伤后应激障碍 (PTSD, post-traumatic stress disorder)**：**禁苯二氮䓬 (benzodiazepine)**（恶化）+ 禁单纯 debriefing；一线 **SSRI/SNRI**；噩梦用 **哌唑嗪 (prazosin)**；首选心理治疗 = **暴露疗法 (exposure therapy)**（PTSD/强迫症 OCD/恐惧症通用）；最大风险因素 = 军中性创伤 (military sexual trauma)。
> → 详见 [[完整笔记/Peixuan分科笔记/neuro]]（TBI/ICP）/ [[完整笔记/Peixuan分科笔记/psych]]（PTSD）。

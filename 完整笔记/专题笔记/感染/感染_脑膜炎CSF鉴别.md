# 脑膜炎 CSF 鉴别

> [!info] 笔记定位
> **拿到 CSF 四件套（WBC / 糖 / 蛋白 / 分类）→ 三步定脑膜炎类型与病因。**
> Step 2 CK 脑膜炎题核心：**先看糖切两半（糖低=细菌/TB/隐球菌；糖正常=病毒/无菌）**，再用分类 + 临床钥匙定病因。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q18596]]（脑膜炎球菌 DIC）+ [[mistakes/uworld-mistakes_2026-06#^Q19266]]（急性 HIV 无菌性脑膜炎）+ [[mistakes/uworld-mistakes_2026-06#^Q18591]]（经验抗生素优先、勿等 LP）。

---

## 零、急救处理顺序（疑细菌性脑膜炎）⭐⭐⭐

> [!danger] 最高频救命陷阱（你栽过 → [[mistakes/uworld-mistakes_2026-06#^Q18591]]）
> **临床高度怀疑细菌性脑膜炎时，经验抗生素绝不能为 LP / CT 延迟。** 延迟用药显著增加死亡。CSF 判读重要，但**治疗优先于诊断操作**。

```
疑细菌性脑膜炎（发热 + 头痛 + 颈抵抗 / 瘀点 / 意识↓）
  │
  ① 抽血培养（不耽误用药）
  │
  ② 立刻经验抗生素 + 地塞米松   ← 一刻不能等
  │     Vancomycin + Ceftriaxone（成人/青少年）
  │     >50/免疫低/孕妇 + Ampicillin（覆 Listeria）
  │
  ③ 需先 CT 再 LP 的 5 指征 → 先 CT（但抗生素已先行）
  │     免疫抑制 / 局灶神经体征 / 视乳头水肿 / 新发抽搐 / 意识严重↓
  │     （怕脑疝；无这些指征则直接 LP）
  │
  ④ 腰穿 LP 取 CSF → 按下方四型判读
```

> [!warning] 地塞米松时机
> 疑肺炎链球菌脑膜炎：**地塞米松在首剂抗生素前或同时**给（减少听力损失/神经后遗症）；确认非肺炎链球菌可停。

> [!tip] 急救 vs 判读两阶段
> 本节（§零）= **先救命**（打药顺序）；§一–§三 = 拿到 CSF 后**再判病因**。考试两类问法都高频：①"best next step" → 选**经验抗生素**（不是 LP）；②给了 CSF 四件套 → 判**类型/病因**。

## 一、三步判读法 ⭐

```
① 看糖（CSF glucose，正常 40-70 或 ≥血糖 2/3）
     糖低（<40-45）→ 细菌 / TB / 隐球菌（耗糖病原）
     糖正常        → 病毒 / 无菌（含急性 HIV）
        │
② 看分类（WBC 主导细胞）
     中性为主 + 糖低 + WBC 极高 → 细菌
     淋巴为主 + 糖低           → TB / 隐球菌
     淋巴为主 + 糖正常         → 病毒 / 急性 HIV
        │
③ 临床钥匙定具体病因（见 §三）
```

> [!danger] 最高频陷阱（你栽过）
> **淋巴细胞性脑膜炎 ≠ 一定 TB**。先看糖：**糖正常 = 病毒/急性 HIV**（[[mistakes/uworld-mistakes_2026-06#^Q19266]] 糖 50 正常却误选 TB）；**糖低才是 TB/隐球菌**。

## 一·五、脑膜炎 (meningitis) vs 脑炎 (encephalitis) 临床鉴别 ⭐⭐（你栽过 → [[NBME11_错题本#^Q063]]）

> [!danger] 核心：分界靠"脑实质有没有受累"，**不靠 CSF**
> **病毒性脑膜炎与病毒性脑炎的 CSF 可以完全雷同**（都淋巴为主 + 糖正常 + 蛋白轻升）→ **CSF 区分不了二者**。真正分水岭 = **临床上有没有脑功能障碍**：
> - 意识改变 (AMS, altered mental status) / 抽搐 / 局灶神经征 / 人格·行为改变 = **脑炎**（脑实质）。
> - 只有脑膜刺激征、而**神志清、查体正常** = **脑膜炎**（脑膜）。

| 维度 | **脑膜炎 meningitis** | **脑炎 encephalitis** |
|---|---|---|
| 炎症部位 | 脑膜 / 蛛网膜下腔 meninges | **脑实质 brain parenchyma** |
| 招牌 | **脑膜刺激征**：颈强直 + Kernig 征 + Brudzinski 征 | **脑功能障碍**：AMS / 抽搐 / 局灶征 / 人格·行为改变 |
| 精神状态 | **正常**（关键分界）| **异常**（hallmark）|
| 头痛 / 发热 / 畏光 | 有 | 有（**重叠**，不能用来分两者）|
| 最常病原 | 病毒（肠病毒 enterovirus 最多）/ 细菌 | **HSV-1**（herpes simplex virus type 1, 1 型单纯疱疹病毒；散发致死性脑炎首位，颞叶）|
| 影像 | 多正常 | **颞叶异常**（HSV，MRI / EEG）|
| CSF | 按四型判读（见 §二）| 淋巴为主、糖正常、蛋白可↑、**可有红细胞**（出血坏死性）|
| 处理钥匙 | 疑细菌 → 经验抗生素（勿等 LP）| 疑 HSV → **经验静脉阿昔洛韦，勿等 PCR** |

> [!warning] HSV 脑炎 (HSV-1 encephalitis) — 脑炎最高频考点
> - **颞叶 (temporal lobe)** 受累 → 人格 / 行为改变、嗅幻觉 (olfactory hallucination)、失语、局灶抽搐。
> - CSF：淋巴细胞增多 + **红细胞 / 黄变**（出血坏死）、糖正常；**HSV PCR** 确诊（最敏感特异）。
> - MRI 颞叶水肿、EEG 颞叶周期性放电 (PLEDs)。
> - **治疗：立即经验性静脉阿昔洛韦 (IV acyclovir, 阿昔洛韦)，不等 PCR 结果**——延误显著增加死亡 / 后遗症。

> [!info] 重叠地带：脑膜脑炎 (meningoencephalitis)
> 两者可并存（脑膜 + 脑实质同时炎症）= **meningoencephalitis（脑膜脑炎）**；常见于李斯特菌 (Listeria monocytogenes, 单核细胞增生性李斯特菌)、虫媒病毒 (arbovirus，如 West Nile virus 西尼罗病毒)、阿米巴 (Naegleria fowleri, 福氏耐格里阿米巴)。**既有脑膜刺激征又有 AMS / 局灶 → 按脑膜脑炎处理**（抗生素 + acyclovir 双覆盖，直到排除）。

> [!tip] 解题钥匙（Q063 教训）
> - 题干"**mental status normal + 神经查体正常 + 无局灶 / 抽搐**" → **脑膜炎**（Q063 = 无菌性）。
> - 题干"**confusion / seizure / focal deficit / 人格改变**" → **脑炎**（先想 HSV）。
> - Q063 把"病毒 + 淋巴性 CSF"直接当脑炎 = 漏了"**精神状态正常**"这条铁分界。

## 二、CSF 四型总表 ⭐⭐⭐

| 类型 | WBC (/mm³) | 主导细胞 | 糖 | 蛋白 | 经验线索 |
|---|---|---|---|---|---|
| **正常** | 0-5 | — | 40-70 | <40 | — |
| **细菌** | **>1000** | **中性** | **↓<40** | **↑>250** | 急、休克、瘀点（脑膜炎球菌）|
| **结核 TB** | 100-500 | 淋巴 | **↓<45** | ↑100-500 | **数周前驱** + 进行性 |
| **病毒 / 无菌** | 10-500 | **淋巴** | **正常** | 轻↑<150 | 肠病毒最常见；HSV 脑炎 |
| **真菌（隐球菌）** | 淋巴 | 淋巴 | **↓** | ↑ | AIDS CD4<100、**↑颅压**、CrAg ⊕ |
| **急性 HIV** | 淋巴 | 淋巴 | **正常** | 轻↑ | **发热+全身LAD+斑丘疹** + 肠病毒阴 |

> [!tip] GBS 对照（防混）
> **Guillain-Barré：CSF 白细胞正常（0-5）+ 蛋白显著↑（albuminocytologic dissociation 蛋白细胞分离）** —— 这是脱髓鞘不是脑膜炎，但同列在 CSF 表里常一起考。

## 三、各病因临床钥匙

> [!info] 细菌性（糖低 + 中性 + WBC 极高）
> 经验治疗 **Vancomycin + Ceftriaxone**（>50/免疫低/孕妇 +Ampicillin 覆 Listeria；新生儿换 Cefotaxime）。详见 [[完整笔记/专题笔记/感染/感染_抗生素高频20场景秒杀]] §二 #5。
> - **脑膜炎球菌**（G− 双球菌）：瘀点/紫癜 → **DIC**（查纤维蛋白原↓）→ [[mistakes/uworld-mistakes_2026-06#^Q18596]]；并发 Waterhouse-Friderichsen。
> - **肺炎链球菌**：最常见**社区获得性**细菌性脑膜炎（~70%，各年龄段）；**常继发于肺炎链球菌肺炎/菌血症**（肺 + 脑同一菌）→ [[mistakes/uworld-mistakes_2026-06#^Q12240]]。

> [!warning] 脑膜炎球菌（N. meningitidis）易感情景 ⭐（荚膜菌：靠补体 + 脾清除）
> | 情景 | 临床意义 |
> |---|---|
> | **青少年/年轻成人**（峰值）+ 婴儿 | 年龄是最大流行病学线索；中老年反而少 |
> | **聚集居住**（大学宿舍/兄弟会、军营、日托）| 飞沫密切传播 → 暴发 |
> | **终末补体缺陷（C5–C9 / MAC）** ⭐ | 不能形成膜攻击复合物 → **反复奈瑟菌感染（脑膜炎球菌 + 淋球菌）** → 查 **CH50** |
> | **备解素缺陷（properdin，X 连锁）** | 暴发性脑膜炎球菌血症 |
> | **Eculizumab（抗 C5）** ⭐ | 药物性补体阻断 → 风险剧增 → **用药前必接种脑膜炎球菌疫苗** |
> | **无脾/功能性无脾**（SCD/脾切）| 荚膜菌清除障碍（同肺炎链球菌、Hib）|
> 
> **钩子**：反复奈瑟菌感染 → 查终末补体（CH50）；eculizumab → 先打疫苗。**有瘀点 + 青少年 + 聚集 = 脑膜炎球菌**（[[mistakes/uworld-mistakes_2026-06#^Q4265]]）；**无瘀点 + 中老年 + 肺炎 = 肺炎链球菌**（[[mistakes/uworld-mistakes_2026-06#^Q12240]]）。

> [!info] 结核性（糖低 + 淋巴 + 数周前驱）
> 亚急性数周、进行性神经下降；糖低、蛋白高。与隐球菌都糖低，靠 CrAg/临床分。

> [!info] 隐球菌（糖低 + 淋巴 + AIDS）
> **CD4<100**、亚急性头痛、**↑开放压**；**India ink / CrAg ⊕**；治 Ampho B + flucytosine → fluconazole + **反复 LP 降颅压**。详见 [[完整笔记/专题笔记/感染/感染_AIDS机会性感染按CD4谱]] §五。

> [!info] 病毒/无菌（糖正常 + 淋巴）
> 肠病毒最常见（夏秋）；**HSV 脑炎**（颞叶、PCR、acyclovir）。

> [!info] 急性 HIV（糖正常 + 淋巴 + 全身表现）
> **急性逆转录病毒综合征**：发热 + **全身淋巴结肿大** + **斑丘疹** + 无菌性脑膜炎（~25%）；性活跃者低门槛查 **HIV RNA 病毒载量**。→ [[mistakes/uworld-mistakes_2026-06#^Q19266]]

## 三·五、按年龄 → 经验病原 + 治疗 ⭐⭐⭐（细菌性）

| 年龄/宿主 | 经验病原 | 经验治疗 |
|---|---|---|
| **新生儿 <1 月** | **GBS** ⭐、E. coli/G⁻ 杆菌、**Listeria**、HSV | **Ampicillin + Cefotaxime**（±acyclovir 若疑 HSV；**不用 ceftriaxone**——核黄疸/钙沉淀）|
| **2 月 – 50 岁** | **肺炎链球菌、脑膜炎球菌** | **Vancomycin + 三代头孢（ceftriaxone）** |
| **>50 / 孕妇 / 酗酒** | 上述 + **Listeria** | **Vancomycin + Ceftriaxone + Ampicillin**（加 amp 覆 Listeria）|
| **免疫抑制** ⭐ | 上述 + **Listeria + G⁻ 杆菌（含 Pseudomonas）** | **Vancomycin + Ampicillin + Cefepime**（注意：**不是 ceftriaxone 而是 cefepime**——要抗铜绿）|
| **神经外科术后 / 穿透伤 / VP 分流** | **G⁻ 杆菌（含 Pseudomonas）、MRSA、凝固酶阴性葡萄球菌（CoNS）** | **Vancomycin + Cefepime**（无社区 Listeria 暴露 → 不加 amp）|
| **辅助** | 疑肺炎链球菌 → **地塞米松**（首剂抗生素前/同时）| — |

> [!info] 用药原理：基础方案 + 两个"开关" ⭐
> **每个药的分工**：
> - **Vancomycin（所有组都有）** → 覆**耐青霉素肺炎链球菌（PRSP）+ MRSA**；单靠头孢可能漏耐药株 → 万古兜底。
> - **三代头孢（ceftriaxone/cefotaxime）** → 覆敏感肺炎链球菌 + **脑膜炎球菌** + Hib + 常见 G⁻；CSF 穿透好（社区"骨架"）。
> - **Ampicillin** → **唯一覆 Listeria**（头孢盲区 LAME 的 L）。
> - **Cefepime（4 代）** → **抗铜绿 + G⁻ 杆菌**（免疫抑制/神外怕 Pseudomonas，ceftriaxone 不覆铜绿）。
>
> **两个开关**：
> 1. **有 Listeria 风险？**（>50 / 新生儿 / 孕妇 / **免疫抑制**）→ **加 ampicillin**（PCN 过敏 → TMP-SMX）。
> 2. **有铜绿/G⁻ 风险？**（**免疫抑制** / 神外·穿透·分流）→ **β-lactam 从 ceftriaxone 升级成 cefepime**（替代：ceftazidime / meropenem）。
>
> 所以**免疫抑制 = 两个开关都打开** → **vanc + ampicillin + cefepime**（这是与 ">50" 组唯一区别：>50 用 ceftriaxone，免疫抑制用 cefepime）。**神外**只开第 2 个开关（无社区 Listeria）→ vanc + cefepime。

> [!danger] 三条贯穿全程的脑膜炎铁律
> 1. **治疗优先于诊断操作**：疑暴发细菌性脑膜炎 → **血培养后立刻经验抗生素 + 地塞米松，勿等 LP/CT**（[[mistakes/uworld-mistakes_2026-06#^Q18591]] 反复错×2：误选腰穿/降温）。
> 2. **瘀点/紫癜 = 脑膜炎球菌**：青少年 + 聚集（露营/宿舍）+ 瘀点 + 中性 CSF → N. meningitidis（[[mistakes/uworld-mistakes_2026-06#^Q4265]]）；瘀点进展 = **DIC**（查纤维蛋白原↓，[[mistakes/uworld-mistakes_2026-06#^Q18596]]）。
> 3. **新生儿首位仍 GBS**：**晚发 GBS（>1 周）水平传播、IAP 不防**，"没做 IAP"不能排除 GBS（[[mistakes/uworld-mistakes_2026-06#^Q3671]]）；新生儿败血凶兆 = 低体温 + 白细胞减少 + 核左移。

## 三·六、脑膜炎球菌（N. meningitidis）专项考点 ⭐⭐（表外高频）

> 关联 [[mistakes/uworld-mistakes_2026-06#^Q18591]]（经验治疗·反复错×2）+ [[mistakes/uworld-mistakes_2026-06#^Q4265]]（诊断）+ [[mistakes/uworld-mistakes_2026-06#^Q18596]]（DIC）。

**① 经验 vs 目标治疗**
- **经验（病原未明）= Vancomycin + Ceftriaxone**（vanc 覆耐药肺炎链球菌，因还没排除肺炎链球菌）。
- **确诊脑膜炎球菌 → 降阶 Ceftriaxone 单药**（奈瑟菌很少耐药）。⚠️ 别一上来只给 ceftriaxone。

**② 补体缺陷 / Eculizumab**（最高频"独家"考点）
- **反复奈瑟菌感染（脑膜炎球菌 ± 播散性淋球菌）→ 查终末补体（C5–C9/MAC）缺陷 → CH50**。
- **Eculizumab/ravulizumab（抗 C5）→ 风险剧增 → 用药前必接种脑膜炎球菌疫苗**。
- 反直觉：补体缺陷者脑膜炎球菌病**反复但死亡率较低**（缺 MAC → 内毒素释放少）。

**③ Waterhouse-Friderichsen 综合征**（= 表里"adrenal hemorrhage"）
- **双侧肾上腺出血 → 急性肾上腺功能不全 → 难治性休克**。
- 钩子：脑膜炎球菌血症 + **补液升压无效的低血压** ± 低血糖/低钠 → **加应激剂量糖皮质激素**。

**④ 瘀点/紫癜 → DIC**
- petechiae → purpura → **purpura fulminans（皮肤坏死）**；紫癜 = DIC → **查纤维蛋白原↓ + 板↓ + PT/PTT↑**（[[mistakes/uworld-mistakes_2026-06#^Q18596]]）。

**⑤ 化学预防（chemoprophylaxis）**

| 项 | 内容 |
|---|---|
| **给谁** | **密切接触者**（同住/日托/接吻/无防护暴露分泌物的医护如插管）；普通 casual contact **不给** |
| **用药** | **Rifampin / Ciprofloxacin（成人单剂）/ Ceftriaxone（单剂 IM，孕妇首选）** |
| **指示病例自己** | 若治疗药**不清除鼻咽携带**（如 penicillin）→ 出院前也要一剂；**ceftriaxone/cefotaxime 能清除**则不必 |
| **时机** | 越早越好（理想 24h 内）+ **飞沫隔离** |

**⑥ 疫苗高危人群**：无脾/SCD、**补体缺陷、eculizumab**、微生物学家、新兵、**住宿大学新生**、**旅行者去脑膜炎带或朝觐（Hajj）**。MenACWY 常规 11–12 岁 + 16 岁加强；MenB 高危/暴发/大学生。
> - **脑膜炎带（Meningitis Belt）**：撒哈拉以南非洲，从塞内加尔横跨到埃塞俄比亚，**全球脑膜炎球菌发病率最高**（旱季高发；A>C/W/X 群）→ 去旅行/工作要接种。
> - **朝觐（Hajj）**：去麦加的伊斯兰朝圣，**数百万人聚集** → 易暴发（著名 2000 年 W135 群）；**沙特强制要求 MenACWY 接种证明**才发签证。
> - 题干钩子：见"**撒哈拉以南非洲旅行**"或"**去麦加朝觐**"→ 想脑膜炎球菌疫苗。

## 四、Memory Hook

```
先看糖
  糖低 → 细菌(中性,WBC极高) / TB(淋巴,数周) / 隐球菌(淋巴,AIDS,CrAg)
  糖正常 → 病毒(肠病毒/HSV) / 急性HIV(发热+全身LAD+斑丘疹)

钥匙
  瘀点紫癜 → 脑膜炎球菌 → DIC查纤维蛋白原
  数周前驱+糖低 → TB
  AIDS+↑颅压+CrAg → 隐球菌
  斑丘疹+全身淋巴结+肠病毒阴 → 急性HIV(查RNA)
  蛋白↑↑细胞正常 → GBS(不是脑膜炎)
```

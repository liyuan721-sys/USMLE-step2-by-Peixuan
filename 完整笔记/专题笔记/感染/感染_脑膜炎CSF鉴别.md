# 脑膜炎 CSF 鉴别

> [!info] 笔记定位
> **拿到 CSF 四件套（WBC / 糖 / 蛋白 / 分类）→ 三步定脑膜炎类型与病因。**
> Step 2 CK 脑膜炎题核心：**先看糖切两半（糖低=细菌/TB/隐球菌；糖正常=病毒/无菌）**，再用分类 + 临床钥匙定病因。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q18596]]（脑膜炎球菌 DIC）+ [[mistakes/uworld-mistakes_2026-06#^Q19266]]（急性 HIV 无菌性脑膜炎）+ [[mistakes/uworld-mistakes_2026-06#^Q18591]]（经验抗生素优先、勿等 LP）。

---

## 零、急救处理顺序（疑细菌性脑膜炎）⭐⭐⭐

> [!danger] 最高频救命陷阱（你栽过 → [[mistakes/uworld-mistakes_2026-06#^Q18591]]）
> **临床高度怀疑细菌性脑膜炎时，经验抗生素绝不能为 LP / CT 延迟。** 延迟用药显著增加死亡。CSF 判读重要，但**治疗优先于诊断操作**。

```
疑细菌性脑膜炎（发热 + 头痛 + 颈抵抗 / 瘀点 / 意识↓）
  │
  ① 抽血培养（不耽误用药）
  │
  ② 立刻经验抗生素 + 地塞米松   ← 一刻不能等
  │     Vancomycin + Ceftriaxone（成人/青少年）
  │     >50/免疫低/孕妇 + Ampicillin（覆 Listeria）
  │
  ③ 需先 CT 再 LP 的 5 指征 → 先 CT（但抗生素已先行）
  │     免疫抑制 / 局灶神经体征 / 视乳头水肿 / 新发抽搐 / 意识严重↓
  │     （怕脑疝；无这些指征则直接 LP）
  │
  ④ 腰穿 LP 取 CSF → 按下方四型判读
```

> [!warning] 地塞米松时机
> 疑肺炎链球菌脑膜炎：**地塞米松在首剂抗生素前或同时**给（减少听力损失/神经后遗症）；确认非肺炎链球菌可停。

> [!tip] 急救 vs 判读两阶段
> 本节（§零）= **先救命**（打药顺序）；§一–§三 = 拿到 CSF 后**再判病因**。考试两类问法都高频：①"best next step" → 选**经验抗生素**（不是 LP）；②给了 CSF 四件套 → 判**类型/病因**。

## 一、三步判读法 ⭐

```
① 看糖（CSF glucose，正常 40-70 或 ≥血糖 2/3）
     糖低（<40-45）→ 细菌 / TB / 隐球菌（耗糖病原）
     糖正常        → 病毒 / 无菌（含急性 HIV）
        │
② 看分类（WBC 主导细胞）
     中性为主 + 糖低 + WBC 极高 → 细菌
     淋巴为主 + 糖低           → TB / 隐球菌
     淋巴为主 + 糖正常         → 病毒 / 急性 HIV
        │
③ 临床钥匙定具体病因（见 §三）
```

> [!danger] 最高频陷阱（你栽过）
> **淋巴细胞性脑膜炎 ≠ 一定 TB**。先看糖：**糖正常 = 病毒/急性 HIV**（[[mistakes/uworld-mistakes_2026-06#^Q19266]] 糖 50 正常却误选 TB）；**糖低才是 TB/隐球菌**。

## 二、CSF 四型总表 ⭐⭐⭐

| 类型 | WBC (/mm³) | 主导细胞 | 糖 | 蛋白 | 经验线索 |
|---|---|---|---|---|---|
| **正常** | 0-5 | — | 40-70 | <40 | — |
| **细菌** | **>1000** | **中性** | **↓<40** | **↑>250** | 急、休克、瘀点（脑膜炎球菌）|
| **结核 TB** | 100-500 | 淋巴 | **↓<45** | ↑100-500 | **数周前驱** + 进行性 |
| **病毒 / 无菌** | 10-500 | **淋巴** | **正常** | 轻↑<150 | 肠病毒最常见；HSV 脑炎 |
| **真菌（隐球菌）** | 淋巴 | 淋巴 | **↓** | ↑ | AIDS CD4<100、**↑颅压**、CrAg ⊕ |
| **急性 HIV** | 淋巴 | 淋巴 | **正常** | 轻↑ | **发热+全身LAD+斑丘疹** + 肠病毒阴 |

> [!tip] GBS 对照（防混）
> **Guillain-Barré：CSF 白细胞正常（0-5）+ 蛋白显著↑（albuminocytologic dissociation 蛋白细胞分离）** —— 这是脱髓鞘不是脑膜炎，但同列在 CSF 表里常一起考。

## 三、各病因临床钥匙

> [!info] 细菌性（糖低 + 中性 + WBC 极高）
> 经验治疗 **Vancomycin + Ceftriaxone**（>50/免疫低/孕妇 +Ampicillin 覆 Listeria；新生儿换 Cefotaxime）。详见 [[完整笔记/专题笔记/感染/感染_抗生素高频20场景秒杀]] §二 #5。
> - **脑膜炎球菌**（G− 双球菌）：瘀点/紫癜 → **DIC**（查纤维蛋白原↓）→ [[mistakes/uworld-mistakes_2026-06#^Q18596]]；并发 Waterhouse-Friderichsen。
> - **肺炎链球菌**：最常见**社区获得性**细菌性脑膜炎（~70%，各年龄段）；**常继发于肺炎链球菌肺炎/菌血症**（肺 + 脑同一菌）→ [[mistakes/uworld-mistakes_2026-06#^Q12240]]。

> [!warning] 脑膜炎球菌（N. meningitidis）易感情景 ⭐（荚膜菌：靠补体 + 脾清除）
> | 情景 | 临床意义 |
> |---|---|
> | **青少年/年轻成人**（峰值）+ 婴儿 | 年龄是最大流行病学线索；中老年反而少 |
> | **聚集居住**（大学宿舍/兄弟会、军营、日托）| 飞沫密切传播 → 暴发 |
> | **终末补体缺陷（C5–C9 / MAC）** ⭐ | 不能形成膜攻击复合物 → **反复奈瑟菌感染（脑膜炎球菌 + 淋球菌）** → 查 **CH50** |
> | **备解素缺陷（properdin，X 连锁）** | 暴发性脑膜炎球菌血症 |
> | **Eculizumab（抗 C5）** ⭐ | 药物性补体阻断 → 风险剧增 → **用药前必接种脑膜炎球菌疫苗** |
> | **无脾/功能性无脾**（SCD/脾切）| 荚膜菌清除障碍（同肺炎链球菌、Hib）|
> 
> **钩子**：反复奈瑟菌感染 → 查终末补体（CH50）；eculizumab → 先打疫苗。**有瘀点 + 青少年 + 聚集 = 脑膜炎球菌**（[[mistakes/uworld-mistakes_2026-06#^Q4265]]）；**无瘀点 + 中老年 + 肺炎 = 肺炎链球菌**（[[mistakes/uworld-mistakes_2026-06#^Q12240]]）。

> [!info] 结核性（糖低 + 淋巴 + 数周前驱）
> 亚急性数周、进行性神经下降；糖低、蛋白高。与隐球菌都糖低，靠 CrAg/临床分。

> [!info] 隐球菌（糖低 + 淋巴 + AIDS）
> **CD4<100**、亚急性头痛、**↑开放压**；**India ink / CrAg ⊕**；治 Ampho B + flucytosine → fluconazole + **反复 LP 降颅压**。详见 [[完整笔记/专题笔记/感染/感染_AIDS机会性感染按CD4谱]] §五。

> [!info] 病毒/无菌（糖正常 + 淋巴）
> 肠病毒最常见（夏秋）；**HSV 脑炎**（颞叶、PCR、acyclovir）。

> [!info] 急性 HIV（糖正常 + 淋巴 + 全身表现）
> **急性逆转录病毒综合征**：发热 + **全身淋巴结肿大** + **斑丘疹** + 无菌性脑膜炎（~25%）；性活跃者低门槛查 **HIV RNA 病毒载量**。→ [[mistakes/uworld-mistakes_2026-06#^Q19266]]

## 三·五、按年龄 → 经验病原 + 治疗 ⭐⭐⭐（细菌性）

| 年龄/宿主 | 经验病原 | 经验治疗 |
|---|---|---|
| **新生儿 <1 月** | **GBS** ⭐、E. coli/G⁻ 杆菌、**Listeria**、HSV | **Ampicillin + Cefotaxime**（±acyclovir 若疑 HSV；**不用 ceftriaxone**——核黄疸/钙沉淀）|
| **1 月 – 50 岁** | **肺炎链球菌、脑膜炎球菌** | **Vancomycin + Ceftriaxone** |
| **>50 / 免疫抑制 / 孕妇 / 酗酒** | 上述 + **Listeria** | **Vancomycin + Ceftriaxone + Ampicillin**（覆 Listeria）|
| **神经外科术后 / 穿透伤 / VP 分流** | 葡萄球菌（含 MRSA）、G⁻（含 Pseudomonas）| **Vancomycin + Cefepime/Ceftazidime/Meropenem** |
| **辅助** | 疑肺炎链球菌 → **地塞米松**（首剂抗生素前/同时）| — |

> [!danger] 三条贯穿全程的脑膜炎铁律
> 1. **治疗优先于诊断操作**：疑暴发细菌性脑膜炎 → **血培养后立刻经验抗生素 + 地塞米松，勿等 LP/CT**（[[mistakes/uworld-mistakes_2026-06#^Q18591]] 反复错×2：误选腰穿/降温）。
> 2. **瘀点/紫癜 = 脑膜炎球菌**：青少年 + 聚集（露营/宿舍）+ 瘀点 + 中性 CSF → N. meningitidis（[[mistakes/uworld-mistakes_2026-06#^Q4265]]）；瘀点进展 = **DIC**（查纤维蛋白原↓，[[mistakes/uworld-mistakes_2026-06#^Q18596]]）。
> 3. **新生儿首位仍 GBS**：**晚发 GBS（>1 周）水平传播、IAP 不防**，"没做 IAP"不能排除 GBS（[[mistakes/uworld-mistakes_2026-06#^Q3671]]）；新生儿败血凶兆 = 低体温 + 白细胞减少 + 核左移。

## 四、Memory Hook

```
先看糖
  糖低 → 细菌(中性,WBC极高) / TB(淋巴,数周) / 隐球菌(淋巴,AIDS,CrAg)
  糖正常 → 病毒(肠病毒/HSV) / 急性HIV(发热+全身LAD+斑丘疹)

钥匙
  瘀点紫癜 → 脑膜炎球菌 → DIC查纤维蛋白原
  数周前驱+糖低 → TB
  AIDS+↑颅压+CrAg → 隐球菌
  斑丘疹+全身淋巴结+肠病毒阴 → 急性HIV(查RNA)
  蛋白↑↑细胞正常 → GBS(不是脑膜炎)
```

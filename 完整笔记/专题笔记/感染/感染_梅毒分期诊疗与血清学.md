# 梅毒分期诊疗与血清学

> [!info] 笔记定位
> **梅毒（Treponema pallidum）的"分期 + 两类血清学"双主线。**
> Step 2 CK 梅毒题三大考向：① 分期临床识别（一期下疳 / 二期手掌脚掌疹 / 三期）② **血清学判读**（非螺旋体 vs 螺旋体 + 筛查/确诊算法 + 假阳/假阴）③ 治疗（PCN G 按期；孕妇脱敏；Jarisch-Herxheimer）。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q20553]]（血清学方向判读：非螺旋体阴 + 螺旋体阳 = 既往治疗）。

---

## 一、分期临床特征 ⭐⭐⭐

| 期 | 时间 | 标志表现 | 关键钥匙 |
|---|---|---|---|
| **一期 Primary** | 暴露后 ~3 周 | **无痛性硬下疳（painless chancre）**：质硬、边界清、基底洁净；伴**无痛性**区域淋巴结 | **无痛** + 自愈（即使不治也消）|
| **二期 Secondary** | 下疳后 4-10 周 | **弥漫斑丘疹累及手掌足底（palms & soles）** ⭐；**扁平湿疣 condyloma lata**（潮湿处灰白疣状、**传染性极强**）；黏膜斑；全身淋巴结；斑片状脱发；发热乏力 | **手掌脚掌疹** + **滴度最高** |
| **潜伏 Latent** | 无症状 | 仅血清学阳 | **早潜伏 <1 年** vs **晚潜伏 >1 年/不详** |
| **三期 Tertiary** | 数年后 | **树胶肿 gummas**（肉芽肿，皮肤/骨/脏器）；**心血管梅毒**（升主动脉炎 → 胸主动脉瘤 + 主动脉瓣反流，vasa vasorum 闭塞性动脉内膜炎）；**神经梅毒**（见 §四）| 慢性破坏性 |
| **先天 Congenital** | 见 §六 | 早期（snuffles 鼻炎 + 肝脾大 + 手足底疹）/ 晚期（Hutchinson 三联）| 孕期筛查可防 |

![[{53D13795-98B5-49AB-B9F5-E6C7DC250E02}.png]]

> [!success] 分期口诀（你主笔记的"采棉花"延伸）
> - **一期 = 一个无痛下疳**（painless chancre）
> - **二期 = 满身疹 + 手掌脚掌 + 扁平湿疣**（最传染、滴度最高）
> - **三期 = 树胶肿 + 主动脉 + 神经**（"采棉花" gumma / aorta / neuro）

> [!warning] 鉴别：无痛 vs 痛性生殖器溃疡
> - **无痛 + 硬 + 洁净底 = 梅毒下疳（chancre）**
> - **痛 + 软 + 脏污底 + 痛性淋巴结化脓 = 软下疳（chancroid, Haemophilus ducreyi）**——"**ducreyi → you cry**（痛）"
> - 无痛 + 一过性丘疹后**痛性腹股沟 buboes** = LGV（Chlamydia L1-3）

> [!tip] 滑车上淋巴结肿大（epitrochlear）鉴别 ⭐——二期梅毒经典体征
> 滑车上淋巴结（肘内侧）引流**前臂/手尺侧**，**平时摸不到、一摸到就有意义**。两大 buzzword + 两类机制：
> - **二期梅毒（secondary syphilis）**⭐：**滑车上淋巴结肿大是二期经典体征**（配掌跖疹 + 扁平湿疣 + 全身淋巴结）。
> - **猫抓病（Bartonella）**：手/前臂抓咬接种 → 滑车上/腋窝**痛性区域结**（[[mistakes/uworld-mistakes_2026-06#^Q20206]]）。
> 
> | 机制 | 触痛 | 病因 |
> |---|---|---|
> | **局部引流**（手前臂尺侧门户）| **痛 / 触痛**（可化脓）| 猫抓病 / 兔热病溃疡腺型 / 孢子丝菌病（玫瑰刺，结节性淋巴管炎）/ 手皮肤感染 |
> | **全身性**（多处淋巴结之一）| **无痛 / 非触痛** | **二期梅毒**（梅毒一律无痛！）/ HIV / EBV 传单 / 淋巴瘤 / 结节病 / RA |
> 
> **钥匙**：**痛 + 单侧 + 手有接种灶 = 局部引流（猫抓/兔热）**；**无痛 + 全身多处 = 系统病（梅毒/淋巴瘤/HIV）**；**滑车上 + 掌跖疹 = 二期梅毒（无痛，查 RPR）**。
> ⚠️ **梅毒淋巴结一律无痛/非触痛**（一期区域 + 二期全身都是）；触痛 → 想化脓性（猫抓/兔热），不是梅毒。

## 二、血清学诊断 ⭐⭐⭐（本笔记核心）

| | 非螺旋体（VDRL / RPR） | 螺旋体（FTA-ABS / TPPA） |
|---|---|---|
| 抗原 | cardiolipin-lecithin-cholesterol（**非特异**）| **螺旋体特异**抗原 |
| 结果 | **定量（滴度 titer）** | **定性（阳/阴）** |
| 特异性 | **低**（假阳性多）| **高** |
| 治疗后 | **滴度下降 → 可转阴** → **监测疗效** | **终身阳性**（不能判活动性）|

### 筛查 → 确诊算法（传统，CK 默认）

```
① 筛查：VDRL / RPR（便宜、定量、可量产筛查）
        │ 阳性
        ▼
② 确诊：FTA-ABS / TPPA（特异，排假阳性）
        │ 两者皆阳
        ▼
   现症或既往梅毒（结合分期 + 滴度判活动性）
```

> [!tip] 为什么筛查用非螺旋体？
> **便宜 + 可定量**——量大筛查成本低，阳性后还能用**滴度追踪疗效**（治疗成功 → 滴度↓≥4 倍 = 2 个稀释度）。FTA-ABS 贵、终身阳无法监测，故放确诊。
> ※ 实验室常用**反向算法**（先螺旋体 EIA/CIA → 阳再做 RPR），但 **CK 默认考传统算法**。

### Discordant 判读（[[mistakes/uworld-mistakes_2026-06#^Q20553]] 核心）

| 非螺旋体(VDRL) | 螺旋体(FTA-ABS) | 解释 |
|---|---|---|
| 阳 | 阳 | 现症感染 |
| **阳** | **阴** | **假阳性**：孕期 / IVDU / 肝病 / HIV / 自身免疫（SLE、抗磷脂综合征）|
| **阴** | **阳** | **既往已治疗（最常见）**；或晚潜伏/神经梅毒（滴度衰减）；或极早期一期 |

> [!danger] 两个方向别搞反
> **自身免疫/孕期/IVDU/肝病/HIV → "非螺旋体假阳 + 螺旋体阴"**（上面第 2 行）；
> 而 **"非螺旋体阴 + 螺旋体阳" = 既往治疗**（第 3 行）。Q20553 即误把这两行配错。

> [!warning] 非螺旋体（VDRL/RPR）会假阴性 —— 3 大场景
> 1. **极早期一期**：下疳刚出，非螺旋体尚未转阳 → 几周后复查转阳
> 2. **晚期潜伏/三期**：滴度随年衰减
> 3. **Prozone 现象**：抗体**过多**封锁凝集 → RPR 假阴（**二期梅毒 / 孕妇**高抗体时）→ **稀释血清后转阳**
> → 高度怀疑而 RPR 阴时，仍要做螺旋体试验。

> [!tip] 疗效监测
> 随访**非螺旋体滴度**：治疗后 6-12 月 **↓≥4 倍 = 成功**；**↑≥4 倍 = 再感染或治疗失败**。

## 三、暗视野显微镜（早期补充）

一期/二期病灶渗出液 **dark-field microscopy** 可直接看到活动螺旋体（早期 RPR 可能还没转阳时有用）。CK 偶考"下疳 + RPR 阴 → 下一步"= 暗视野 / 直接检测。

## 四、神经梅毒（Neurosyphilis）⭐

> [!danger] 神经梅毒可发生在任何分期，不限于三期
> HIV 合并感染者更易、更早出现神经梅毒。

| 类型 | 表现 | 高频钥匙 |
|---|---|---|
| **早期** | 梅毒性脑膜炎；**眼梅毒（葡萄膜炎）**；耳梅毒 | 任何期都可早累及 |
| **脊髓痨 Tabes dorsalis** | 后索变性 → **感觉性共济失调 + Romberg ⊕**、闪电样痛、本体觉/振动觉丧失、腱反射消失、**Charcot 关节** | 后索 = 振动/本体 |
| **全身轻瘫 General paresis** | 进行性痴呆、人格改变、精神症状 | 晚期"麻痹性痴呆" |
| **Argyll Robertson 瞳孔** ⭐ | **调节存在但对光不反应**（accommodates but doesn't react）| "**娼妓瞳孔**：近反射在、光反射失" |

> [!info] 何时做 LP（腰穿）查神经梅毒
> 神经/眼/耳症状、三期、治疗失败（滴度不降/复升）、（HIV + 晚潜伏——有争议）。
> **CSF 判读**：**CSF-VDRL** = 特异不敏感（阳即确诊）；CSF-FTA-ABS = 敏感不特异（阴可排除）；伴 CSF 淋巴细胞↑ + 蛋白↑。

## 五、治疗 ⭐⭐⭐

> [!danger] 梅毒任何期一线 = Penicillin G（无替代地位）

| 期 | 方案 |
|---|---|
| **一期 / 二期 / 早潜伏(<1y)** | **Benzathine PCN G 2.4 MU IM × 1 剂** |
| **晚潜伏(>1y) / 三期（非神经）** | **Benzathine PCN G 2.4 MU IM 每周 × 3 剂** |
| **神经梅毒 / 眼梅毒 / 耳梅毒** | **Aqueous crystalline PCN G IV × 10-14 天**（住院静脉，不是肌注 benzathine）|

> [!warning] PCN 过敏怎么办
> - **非孕**：**Doxycycline**（替代）
> - **孕妇**：**必须脱敏后用 PCN**（**doxycycline 致畸 + 不替代**；erythromycin 不可靠不穿胎盘充分）—— 见 [[完整笔记/专题笔记/感染/感染_抗生素高频20场景秒杀]] §6.5

> [!danger] Jarisch-Herxheimer 反应 ⭐（超高频陷阱）
> 治疗后 **24h 内**发热、寒战、肌痛、头痛、皮疹加重——**死亡螺旋体释放内毒素样物质**所致，尤其**早期梅毒**。
> **关键：这不是 penicillin 过敏！**
> → **支持治疗 + 退热药**，**继续 PCN，不停药**。
> 反射弧：「**梅毒打针后几小时发热加重 = Jarisch-Herxheimer，不是过敏**」（你主笔记 §四已强调）。
> 与"传单疹""IgE 真过敏"三鉴别见 [[完整笔记/专题笔记/感染/感染_青霉素后皮肤事件三鉴别]]（时相：分钟过敏 / 小时赫氏 / 数天传单）。

## 六、孕期 + 先天梅毒

> [!info] 孕期
> **所有孕妇首次产检常规筛查梅毒**（RPR/VDRL）。孕妇梅毒 → **只用 PCN**；过敏 → **脱敏**。治疗可防先天梅毒（垂直传播）。

| 先天梅毒 | 表现 |
|---|---|
| **早期（<2 岁）** | **鼻炎 snuffles（含螺旋体，传染）**、肝脾大、**手足底斑丘疹**、黄疸、长骨干骺端异常、假性瘫 |
| **晚期** | **Hutchinson 三联** = **Hutchinson 齿（楔形门牙）+ 间质性角膜炎 + CN VIII 耳聋**；桑椹臼齿、**马鞍鼻 saddle nose**、**军刀胫 saber shins**、额部隆起 |

> [!success] 先天梅毒记忆
> 你主笔记里"**风吹过耳，心眼瞎**"——CN VIII 聋（耳）+ 间质性角膜炎（眼）等多系统受累。Hutchinson 三联 = **齿 + 角膜 + 聋**。

## 七、Memory Hook 总集

```
分期
  一期 = 1 个无痛硬下疳（painless chancre，自愈）
  二期 = 满身疹 + 手掌脚掌 + 扁平湿疣（最传染、滴度最高）
  三期 = 树胶肿 + 升主动脉炎 + 神经梅毒

血清学（核心）
  非螺旋体 VDRL/RPR = 定量、不特异、可转阴 → 筛查 + 监测疗效
  螺旋体 FTA-ABS    = 定性、特异、终身阳 → 确诊
  非阳+螺阴 = 假阳性（孕/IVDU/肝/HIV/自免）
  非阴+螺阳 = 既往治疗（最常见）
  RPR 假阴 = 极早期 / 晚期 / prozone（稀释后转阳）

神经梅毒
  Argyll Robertson = 调节在、对光失（娼妓瞳孔）
  脊髓痨 = 后索（振动/本体）+ Romberg⊕ + Charcot 关节
  确诊 CSF-VDRL（特异不敏感）

治疗
  任何期 = PCN G
  一期/二期/早潜伏 = Benzathine PCN ×1；晚潜伏/三期 = ×3 周；神经 = IV ×10-14d
  孕妇过敏 = 脱敏（doxy 致畸不替代）
  打针后几小时发热 = Jarisch-Herxheimer（继续打，退热）

监测
  滴度↓4倍=好；↑4倍=再感染/失败
```

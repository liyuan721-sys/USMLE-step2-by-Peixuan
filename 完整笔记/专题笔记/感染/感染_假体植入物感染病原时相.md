# 假体 / 植入物感染 — 病原按时相（Prosthetic & Implanted Device Infection — Pathogens by Time Phase）

> [!info] 用途
> 把所有"人工材料 + 感染"题统一到**一条主线**：植入物 = 生物膜 biofilm 温床 → 平时的"污染菌"凝固酶阴性葡萄球菌 CoNS（coagulase-negative staphylococci，以表皮葡 Staphylococcus epidermidis 为主）翻身成真凶；**发病距手术的时间**决定是**围手术期种植（早发）**还是**血行播散（晚发）**。
> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q9111]]（假体关节感染 PJI 时相）+ 跨器械整合（人工瓣 / CIED / 脑脊液分流 / 血管移植物）。
> 时相阈值口径与 [[完整笔记/专题笔记/感染/感染_感染性心内膜炎完整体系]] 保持一致。

> [!danger] 一条主线（先背这个）
> **植入物感染 = 生物膜病 biofilm disease**：
> 1. **CoNS（S. epidermidis 表皮葡）是植入物头号真凶** —— 产黏液生物膜定植；它平时是血培养**头号污染菌**，但**有假体 / 导管 / 人工瓣 / CIED = 当真感染**（生物膜温床）。
> 2. **时间钟 time clock**：离手术越近 = **围手术期种植（早发）**，毒力高 + 院内菌；离手术越远 = **血行播散（晚发）**，菌谱似天然部位。
> 3. **治疗双支柱**：① 多数需**取出 / 更换植入物**（生物膜里抗生素打不进、清不掉）；② 葡萄球菌生物膜加**利福平 rifampin**（唯一能穿透生物膜的抗葡菌药，**绝不单用** —— 单用快速耐药，必须联用）。

## 一、各器械时相对照（核心表）

### 1.1 假体关节感染 PJI（prosthetic joint infection 假体关节感染）⭐锚点

| 时相 | 术后时间 | 临床 | 最常见病原 |
|---|---|---|---|
| **Early 早发** | <3 月 | 急性痛、伤口感染 / 裂开、**发热** | **金葡 S. aureus**、革兰阴性杆菌 G− bacilli、厌氧菌 anaerobes |
| **Delayed 延迟** | 3–12 月 | 慢性关节痛、**假体松动 loosening**、窦道；常**无发热 / 白细胞不升** | **CoNS（表皮葡 S. epidermidis）**、痤疮丙酸杆菌 Cutibacterium (Propionibacterium) acnes、肠球菌 Enterococcus |
| **Late 晚发** | >12 月 | 既往无症状关节**急性**发作 + 近期远处感染（血行播散） | 金葡 S. aureus、革兰阴性杆菌 G−、β-溶血链球菌 β-hemolytic strep |

> [!tip] PJI 读题钥匙
> - **滑液白细胞 synovial WBC**：PJI 常 **>1,000/μL** 但**远低于天然关节化脓性关节炎 septic arthritis（>50,000/μL）** —— 低度炎症 + 假体在场 = 想 PJI，别套天然关节阈值。
> - **慢性低度炎症反而是"低毒力"菌**：高毒力菌（金葡 / 铜绿 Pseudomonas）感染急、有发热 → 多在早发期现形；低毒力 CoNS 产生物膜隐匿定植 → 3–12 月才慢慢现形，常无发热与白细胞升高。
> - **肩关节置换 shoulder arthroplasty** → **Cutibacterium acnes 痤疮丙酸杆菌特别高发**，病程极隐匿、培养需**延长孵育至 14 天**（否则假阴）。

### 1.2 人工瓣心内膜炎 PVE（prosthetic valve endocarditis 人工瓣感染性心内膜炎）

| 时相 | 换瓣后时间 | 最常见病原 | 备注 |
|---|---|---|---|
| **早期 PVE** | <2 月（院内获得 nosocomial） | **CoNS（表皮葡 S. epidermidis）**、金葡 S. aureus、革兰阴性杆菌 G−、念珠菌 Candida、白喉样杆菌 diphtheroids | 围手术期种植 |
| **中期** | 2–12 月 | 过渡型（CoNS + 金葡 S. aureus 仍居多） | —— |
| **晚期 PVE** | >12 月 | **似天然瓣**：草绿色链球菌 viridans strep、金葡 S. aureus、肠球菌 Enterococcus、HACEK 群 | 社区获得 / 血行 |

> [!question]- 低毒力 CoNS 为何在 PVE 是"早期"主角、在 PJI 却是"延迟"？（口径不同，非矛盾）⭐
> CoNS（表皮葡 S. epidermidis）身份没变——两处都是**围手术期种植的低毒力生物膜菌**。绊人的是两套"早 / 晚"轴定义不一样：
> - **PVE 按"获得途径 acquisition"切线**：早期 = **院内 / 围手术期获得 nosocomial**（换瓣时从皮肤种到缝合环上）；手术时最常见的皮肤污染菌就是 CoNS → 所以它自然是"早期"主角，**与毒力高低无关**。晚期才转成社区 / 血行的似天然瓣菌（草绿链等）。
> - **PJI 按"现形速度 declare time"切线**：早发(<3 月)留给 declare 快的高毒力菌（金葡 / G−）；CoNS 毒力低、靠生物膜隐匿 → 拖到 **延迟(3–12 月)** 才现形（同样是围手术期种的）。
> - **为何瓣膜 2 月内就发病、关节却要 3–12 月**：瓣膜**在血流里**（缝合环有血供 + 持续浸血）→ 低毒力菌也能较快出**菌血症 + 新杂音 + 栓塞**；关节假体腔隙**乏血供** → CoNS 生物膜只隐匿钝痛 / 假体松动，拖久才够响被诊断。
> - **一句话**：同一个"围手术期低毒力生物膜 CoNS"，PVE 叫它"早期"（按院内获得 + 瓣膜在血流发病快），PJI 叫它"延迟"（按慢现形 + 关节乏血供）。**两套时间线画法不同，不是生物学矛盾。**

> [!info] PVE 经验治疗（详见 [[完整笔记/专题笔记/感染/感染_感染性心内膜炎完整体系]]）
> 人工瓣 IE 经验 = **万古霉素 vancomycin + 庆大霉素 gentamicin + 利福平 rifampin**（cover 耐甲氧西林表皮葡 MRSE）；疗程 **≥6 周**。**rifampin 针对生物膜**，是人工瓣方案区别于天然瓣的关键药。

### 1.3 心脏植入式电子装置感染 CIED（cardiac implantable electronic device 心脏植入电子装置：起搏器 pacemaker / 除颤器 ICD）

- **病原**：多 **CoNS（表皮葡 S. epidermidis）+ 金葡 S. aureus**（生物膜）。
- **表现**：囊袋红肿 / 破溃 / 窦道（pocket infection 囊袋感染）；或导线相关心内膜炎 lead endocarditis + 菌血症。
- **处理**：**完整移除整套系统（装置 + 所有导线）** + 抗生素 —— 留任何导线都清不掉生物膜。
- 钥匙：**金葡 S. aureus 菌血症 + 在位 CIED → 查导线赘生物（经食道超声 TEE, transesophageal echocardiography）**。

### 1.4 脑脊液分流管感染 CSF shunt（VP shunt, ventriculoperitoneal shunt 脑室腹腔分流）

- **病原**：**CoNS（表皮葡 S. epidermidis）最常见** > 金葡 S. aureus > 革兰阴性杆菌 G−（多见于腹腔端 / 修订术后）。
- **时间**：多在置管后 **<1–2 月**（围手术期皮肤菌种植）。
- **表现**：**分流功能障碍 shunt malfunction + 低热 + 头痛**；脑膜刺激征常**不明显**（低毒力菌）。
- **处理**：**取出 / 外置分流（externalization）** + 脑室内 intraventricular / 全身抗生素。

### 1.5 血管移植物 / 人工血管感染 vascular graft infection（人工血管感染）

| 时相 | 术后时间 | 病原 |
|---|---|---|
| **早期** | <4 月 | 金葡 S. aureus、革兰阴性杆菌 G−（铜绿假单胞 Pseudomonas） |
| **晚期** | >4 月 | **CoNS（表皮葡 S. epidermidis）**（生物膜、隐匿） |

- **处理**：移除 / 替换移植物 + 长程抗生素。

## 二、跨器械"时间钟"统一逻辑

```
植入物感染 = 生物膜病（CoNS 表皮葡是共通主角）
            │
       离手术多久？
            │
   ┌────────┴─────────┐
  早（围手术期种植）        晚（血行播散）
  院内菌 + 高毒力           菌谱似天然部位
  金葡 / G− / 念珠菌        草绿链 / 金葡 / 肠球
  （但 CoNS 始终在场）       （CoNS 仍可隐匿）
            │
   ── 共通处理 ──────────────────────────
   ① 取出 / 更换植入物（生物膜清不掉）
   ② 葡萄球菌生物膜 → 加 rifampin（穿生物膜，勿单用）
   ③ 长程抗生素（数周～6 周+）
```

> [!warning] 注意"时间窗阈值"各器械不同，别混
> - **PJI**：早 <3 月 / 延迟 3–12 月 / 晚 >12 月
> - **PVE**：早 <2 月 / 晚 >12 月（治疗用利福平的界 = <12 月）
> - **血管移植物**：早 <4 月 / 晚 >4 月
> 共性只是"早=围手术种植高毒力、晚=血行/隐匿低毒力 + CoNS 全程在场"，**具体月数各表为准**。

## 三、高频陷阱 / 钥匙速记

> [!tip] 必记 5 条
> - **血培养长 CoNS 的判读**：无植入物 = 多半污染（重抽 2 套）；**有假体 / 导管 / 人工瓣 / CIED = 当真感染**（治 + 拔 / 换装置）。详见 [[mistakes/uworld-mistakes_2026-06#^Q14664]] / [[完整笔记/专题笔记/感染/感染_常见病原菌总结_按部位综合征]]。
> - **肩置换 + 隐匿痛 / 假体松动** → **Cutibacterium acnes 痤疮丙酸杆菌**（培养延长孵育 14 天，否则假阴）。
> - **起搏器 / ICD + 金葡 S. aureus 菌血症** → 整套系统移除 + TEE 找导线赘生物。
> - **利福平 rifampin** 是抗葡萄球菌生物膜唯一利器，但**绝不单用**（单用快速耐药），必须联用。
> - **念珠菌 Candida** 是**早期 PVE / 长期静脉营养 / 静脉吸毒 IVDU** 的隐匿病原 → 拔管 / 换瓣 + 抗真菌（如棘白菌素 echinocandin）。

## 复习题（自测，先不看答案）

> [!question]- PJI 三时相的时间窗 + 各自最常见病原？
> Early <3 月 = 金葡 S. aureus / G− 杆菌 / 厌氧菌（急性 + 发热）；Delayed 3–12 月 = CoNS（表皮葡）/ Cutibacterium acnes / 肠球菌（慢性、假体松动、常无发热）；Late >12 月 = 金葡 / G− / β-溶血链球菌（血行播散，急性发作 + 近期远处感染）。

> [!question]- 65 岁，全膝置换术后 6 个月，3 周隐痛，无发热，滑液白细胞 10,000/μL（中性为主）。最可能病原？为何不是铜绿假单胞？
> **delayed-onset PJI → CoNS（表皮葡 S. epidermidis）**。6 月 + 低度炎症 + 无发热 = 低毒力生物膜菌；铜绿 / 金葡是早发（<3 月）高毒力急性病原，不对应亚急性慢性病程。滑液 10,000 远低于天然关节化脓性关节炎（>50,000），符合 PJI。

> [!question]- 换瓣后 1 个月出现发热 + 新杂音，最可能病原是什么？人工瓣 IE 经验三药？
> 早期 PVE（<2 月）→ **CoNS（表皮葡 S. epidermidis）**为首（含金葡 / G− / 念珠菌）。经验 = **vancomycin 万古 + gentamicin 庆大 + rifampin 利福平**（cover MRSE），≥6 周。

> [!question]- 血培养长出 S. epidermidis，怎么判断是污染还是真感染？真感染时除抗生素外还要做什么？
> 看有无"当真信号"：**有假体 / 导管 / 人工瓣 / CIED**（生物膜温床）、多套均阳、免疫抑制、脓毒 → 当真。真感染（在植入物上）→ 抗生素 + **取出 / 更换植入物** + 葡萄球菌生物膜加 rifampin（勿单用）。

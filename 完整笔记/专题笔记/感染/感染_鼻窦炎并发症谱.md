# 鼻窦炎并发症谱（ABRS + 眶/颅内/骨）

> [!info] 笔记定位
> 急性细菌性鼻窦炎（acute bacterial rhinosinusitis, ABRS）的**基础（诊断/病原/危险因素/治疗）+ 并发症（邻近蔓延：眶 / 颅内 / 骨）**一站速查。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q4850]]（危险因素=病毒 URI）+ [[mistakes/uworld-mistakes_2026-06#^Q3286]]（病原=不可分型 H. flu）+ [[mistakes/uworld-mistakes_2026-06#^Q15725]]（颅内脑脓肿→CT）+ [[mistakes/uworld-mistakes_2026-06#^Q3326]]（隔前 vs 眶蜂窝织炎）。

---

## 一、ABRS 基础 ⭐⭐

> [!info] 诊断标准（3 选 1）+ 病原 + 治疗
> **诊断（任一）**：① 症状 **≥10 天**无改善 ② **重症**（高热 ≥39℃ + 脓涕 ≥3 天）③ **双相恶化**（好转后再恶化 double-sickening）。
> **病原（同急性中耳炎 AOM）**：
> - **不可分型流感嗜血杆菌（nontypeable *Haemophilus influenzae*）≈ 40–50%** — **PCV13 后首位** ⭐
> - **肺炎链球菌（*S. pneumoniae*）≈ 20–25%**
> - **卡他莫拉菌（*Moraxella catarrhalis*）≈ 25%**（儿童占比更高）
> 
> **PCV13 疫苗效应**：13 价肺炎球菌结合疫苗普及后 → **肺炎链球菌占比↓ → 不可分型 H. flu 升为首位**（H. flu 是"不可分型"，**不在 PCV/Hib 疫苗覆盖范围**，故相对占比上升）。
> **最常见危险因素 = 病毒性上呼吸道感染（viral URI）** > 过敏性鼻炎（[[mistakes/uworld-mistakes_2026-06#^Q4850]]）。
> **治疗**：**阿莫西林（amoxicillin）± 克拉维酸（clavulanate）**。
> ⚠️ **GAS（S. pyogenes）= 咽炎，极少鼻窦炎**；**真菌（曲霉/毛霉）鼻窦炎 = 免疫抑制/控制差糖尿病**（鼻出血/鼻甲破坏/腭部焦痂）（[[mistakes/uworld-mistakes_2026-06#^Q3286]]）。

> [!tip] "PCV13 后病原洗牌"= ABRS / AOM 共享高频钩子 ⭐
> **ABRS（急性细菌性鼻窦炎）与 AOM（急性中耳炎 acute otitis media）病原谱相同**，受同一疫苗效应影响：
> - **PCV13 普及 → 肺炎链球菌占比↓ → 不可分型 H. flu 升首位**（H. flu "不可分型" **不在 PCV / Hib 覆盖**，Hib 疫苗只防 b 型荚膜株）。
> - **解题反射**：题干强调**"疫苗接种完整的儿童" + 中耳炎/鼻窦炎** → 优先想 **不可分型 H. flu**（不是肺炎链球菌）。
> - **治疗两病一致**：**阿莫西林（amoxicillin）高剂量 ± 克拉维酸（clavulanate）**（产 β-内酰胺酶的 H.flu / 卡他莫拉菌 → 加 clavulanate）。

> [!warning] 但"一样"只限急性社区——这些情况两者分叉 ⭐
> 
> | 情境 | 中耳炎（AOM）| 鼻窦炎（ABRS）|
> |---|---|---|
> | **占比微差** | **卡他莫拉菌在儿童占比更高** | 三者相近，不可分型 H.flu 居首 |
> | **新生儿/小婴儿** | 可加 **GBS / G⁻ 杆菌 / 金葡**（少）| 婴儿鼻窦未发育，**ABRS 罕见** |
> | **慢性/复发** | **慢性化脓性中耳炎 → 铜绿假单胞（Pseudomonas）+ 金葡 + 厌氧** | **慢性鼻窦炎 → 多菌 + 厌氧 + 金葡** |
> | **免疫低/糖尿病** | — | **真菌（毛霉 mucor / 曲霉）→ 鼻脑型，致死急症** |
> 
> 一句话：**急性社区 AOM = ABRS（同三菌 + 同 PCV13 效应 + 同 amoxicillin±clav）；慢性两者都转铜绿/金葡/厌氧；免疫低鼻窦炎防真菌。**

> [!danger] 别把"外耳道炎"混进来 — 病原是铜绿，不是上呼吸道三菌
> - **急性外耳道炎（otitis externa，"游泳者耳" swimmer's ear）= 铜绿假单胞（Pseudomonas aeruginosa）+ 金葡**（潮湿/游泳）→ **局部抗生素滴耳**（耳屏牵拉痛、外耳道肿）。
> - **恶性外耳道炎（malignant otitis externa）= 铜绿假单胞**，见于**糖尿病/老年**，可侵颅底骨髓炎 → **全身抗假单胞（如环丙沙星 ciprofloxacin）**。
> - 反射：**"游泳 / 糖尿病 + 耳痛 + 牵拉痛" → 外耳道炎（铜绿）**，与中耳炎/鼻窦炎那套**完全两码事**，别张冠李戴。

## 二、变应性 vs 非变应性 vs 药物性鼻炎（ABRS 鉴别）⭐⭐

> [!info] 为什么放这里
> 慢性/反复鼻塞流涕先分"是不是感染"——这三类**非感染性鼻炎**最常和 ABRS 混；分型错 → 治疗方向全错（尤其**药物性鼻炎要停药、不是加药**）。

| 特征 | 变应性鼻炎 Allergic | 非变应性（血管运动性）Vasomotor | 药物性 Rhinitis medicamentosa |
|---|---|---|---|
| 机制 | **IgE 介导** I 型超敏、肥大细胞 → 组胺 | **非 IgE**；自主神经失调、黏膜高反应 | 局部减充血剂滥用 → **反跳性充血** |
| 诱因 | 过敏原（花粉/尘螨/皮屑）、季节性 or 常年 | 温度变化/气味/烟/酒/辛辣（味觉性）/情绪 | 鼻用 α-激动剂 **>3–5 天**（羟甲唑啉 oxymetazoline / 去氧肾上腺素 phenylephrine）；可卡因 |
| 痒/喷嚏/眼痒 | **突出**（鼻痒、阵发喷嚏、眼痒 + 结膜炎） | **少**（以鼻塞 + 流涕为主） | 无 |
| 鼻黏膜 | 苍白/淡蓝、水肿（boggy）、清水样涕 | 红/正常、清涕 | 红肿/糜烂、顽固性鼻塞 |
| 起病/家族史 | 多儿童青少年、过敏/特应史 + | 成人起病、无过敏史 | 任意；有鼻喷雾滥用史 |
| 辅助检查 | 鼻涂片**嗜酸 +**、皮肤点刺/特异性 IgE + | 皮试 -、嗜酸 -（NARES 亚型嗜酸 +） | 病史即可 |
| 一线治疗 | **鼻用激素**（一线、最有效）+ 二代抗组胺 <br> 避敏；难治 → 免疫治疗 | **鼻用激素 / 鼻用抗组胺**（氮卓斯汀 azelastine） <br> 避诱因；流涕重 → 异丙托溴铵 ipratropium | **停减充血剂**（核心！） <br> 鼻用激素仅过渡缓解反跳 |

> [!tip] 三个 stem 钥匙
> - **鼻痒 + 眼痒 + 阵发喷嚏 + 季节性/过敏史 + 苍白水肿鼻甲** → **变应性**（一线鼻用激素）
> - **气味/温度/辛辣诱发、成人、无过敏史、皮试阴性** → **非变应性血管运动性**（鼻用激素/抗组胺；口服抗组胺效果差）
> - **越用鼻喷越堵、停药加重、有 OTC 减充血剂滥用史** → **药物性鼻炎**（**停药**才是答案）

> [!warning] 最坑点
> - 药物性鼻炎考点 = **停药**；选"再开/换一个鼻用减充血剂"必错。
> - 变应性一线是**鼻用糖皮质激素**（比口服抗组胺更有效），别反射选口服抗组胺。
> - 非变应性鼻炎：**口服抗组胺无效**（非组胺/IgE 驱动），但**鼻用抗组胺（氮卓斯汀 azelastine）有效**——局部除抗 H1 外还**抗炎 + 降鼻黏膜神经高反应性**，作用不依赖组胺。即"**口服没用、鼻用有用**"，别混。

> [!success] 一线治疗一句话（共同点 + 例外）
> - **变应性 + 非变应性 → 一线首选都是【鼻用糖皮质激素 INCS】**（对鼻塞最有效）；鼻用抗组胺 azelastine 是两者通用的同级替代。
> - 分歧在第二步：**变应性**加口服/鼻用二代抗组胺（组胺驱动有效）；**非变应性**靠鼻用激素/azelastine + 流涕加鼻用异丙托溴铵 ipratropium + 避诱因（口服抗组胺无效）。
> - **例外 = 药物性鼻炎**：一线不是激素，是**停减充血剂**（激素只过渡缓解反跳）。

## 三、并发症总览（邻近蔓延）⭐⭐⭐

```
ABRS 未治/进展 → 邻近蔓延
├─ 眶（最常见，尤儿童筛窦炎）
│   ├─ 隔前蜂窝织炎(preseptal)  眶隔前·仅睑肿·无眶内征 → 口服抗生素
│   ├─ 眶蜂窝织炎(orbital)      眶隔后·眼动痛+突出+视力↓ → 静脉抗生素+眶CT±手术
│   └─ 骨膜下/眶脓肿
├─ 颅内
│   ├─ 脑脓肿(额窦→额叶)  头痛+晨起呕吐(↑颅压)+意识改变+局灶 → CT/MRI+引流
│   ├─ 海绵窦血栓        双眼眶内征+CN III/IV/VI麻痹 → MRI/MRV+抗凝
│   └─ 脑膜炎/硬膜外·硬膜下脓肿
└─ 骨
    └─ Pott puffy tumor  额窦炎→额骨骨髓炎→前额肿(骨膜下脓肿)
```

## 四、眶并发症：隔前 vs 眶蜂窝织炎 ⭐⭐⭐

| | **隔前蜂窝织炎（preseptal）** | **眶蜂窝织炎（orbital）** |
|---|---|---|
| **部位/源** | 眶隔**前**；皮肤破损/虫咬 | 眶隔**后**；多**鼻窦炎（筛窦）蔓延** |
| **眶内征** | **无**（眼动正常无痛、无突出、视力正常）| **有**：**眼动痛/眼肌麻痹（ophthalmoplegia）+ 眼球突出（proptosis）+ 视力变化** |
| **病原** | 金葡/化脓性链球菌（皮肤）| 鼻窦菌（H.flu/肺炎链球菌等）|
| **处理** | **口服抗生素**（克林霉素 clindamycin 覆 GAS+MRSA）| **眶 CT + 静脉抗生素 ± 手术** |

> [!danger] 分水岭 = 有无眶内征
> **眼动痛/眼肌麻痹 + 眼球突出 + 视力变化 → 眶（急、要 CT+静脉抗生素）；全无 → 隔前（口服）**（[[mistakes/uworld-mistakes_2026-06#^Q3326]]）。

## 五、颅内并发症 ⭐⭐

> [!danger] 脑脓肿（额窦 → 额叶）→ 急诊影像
> **红旗**：持续头痛 + **晨起/卧位呕吐（↑颅压）** + 意识改变（嗜睡）+ 局灶神经/颈痛。→ **急诊头颅 CT（快、检早期脑炎；环形强化 = 脓肿）**；治疗 IV 抗生素 + 外科引流。**↑颅压时影像先于 LP**（[[mistakes/uworld-mistakes_2026-06#^Q15725]]）。

> [!warning] 海绵窦血栓（cavernous sinus thrombosis, CST）
> **通路**：鼻窦（蝶/筛窦）/ 面部危险三角感染经**无瓣面静脉 → 眼上下静脉**逆行入海绵窦。
> **不总是双侧！** 起病多为**单侧**（同感染源侧），经**海绵间窦（intercavernous sinus）**蔓延 → **双侧化**。"双侧眶征 + V2 麻木"是甩开眶蜂窝织炎（永远单侧）的两张王牌，但**不必一上来就对称**。
> **定位金律**：① **CN VI 最先 / 最常受累**（在窦腔中央裸奔、紧贴颈内动脉 ICA）→ 外展不能 / 水平复视常首发；② **V1 + V2 受累但 V3 保留**、**视力早期多保留**（视神经 CN II 不在窦内）→ 锁定病变在海绵窦（非眶尖 / 非整条三叉神经）。
> **眶征**：眼球突出 proptosis + 结膜水肿 chemosis + 眶周肿；眼肌麻痹（CN III/IV/VI）；瞳孔可**散大**（CN III）或 **Horner**（ICA 旁交感）→ 瞳孔表现可矛盾，别被单一瞳孔带偏。伴 **高热脓毒、剧烈头痛**（比眶蜂窝织炎更"毒"）。
> **病原**：最常 **金黄色葡萄球菌 Staphylococcus aureus**；糖尿病 / 免疫低下警惕 **毛霉病 mucormycosis**（黑色坏死焦痂）。
> **诊断** MRI/MRV（或 CT 静脉造影）；**治疗** 大剂量 IV 抗生素覆盖 MRSA（万古霉素 vancomycin + 头孢曲松 ceftriaxone + 甲硝唑 metronidazole）+ **抗凝**（防血栓蔓延）+ 清除感染源；若毛霉 → 两性霉素 B amphotericin B + 急诊清创 + 纠正 DKA。

> [!info] 其他颅内
> 脑膜炎、硬膜外/硬膜下脓肿（empyema）——均需神经影像 + IV 抗生素 ± 引流。

## 六、骨并发症：Pott puffy tumor

> [!info] Pott puffy tumor（额骨骨髓炎）
> **额窦炎 → 额骨骨髓炎 + 骨膜下脓肿 → 前额局限肿胀（"puffy"）**；多见青少年。可向后蔓延致颅内（硬膜外脓肿）。诊断影像（CT/MRI）；治疗 IV 抗生素 + 外科引流。

---

## 七、红旗 + 何时影像

- **眶红旗**（→ 眶 CT + 静脉抗生素）：**眼动痛/眼肌麻痹、眼球突出、视力变化、双侧眼征**。
- **颅内红旗**（→ 头颅 CT/MRI）：**晨起呕吐（↑颅压）、意识改变、局灶神经、抽搐、颈强直、剧烈头痛**。
- **骨红旗**：**前额局限肿胀（Pott puffy）**。
- 单纯 ABRS 无红旗 → 临床诊断 + 抗生素，无需影像。

## 八、Memory Hook

```
ABRS：≥10天/重症/双相恶化(3选1) → 不可分型H.flu+肺炎链球菌+卡他莫拉菌 → amoxicillin±clav
      危险因素=病毒URI；GAS是咽炎非鼻窦炎；真菌要免疫抑制/糖尿病
并发症蔓延：
  眶：隔前(无眶内征→口服) vs 眶(眼动痛+突出+视力↓→眶CT+静脉±手术)
  颅内：脑脓肿(晨起呕吐↑颅压→CT+引流) / 海绵窦血栓(双眼CN III-IV-VI→MRV+抗凝)
  骨：Pott puffy tumor(额骨骨髓炎→前额肿)
```

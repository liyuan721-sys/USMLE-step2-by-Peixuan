# 鼻窦炎并发症谱（ABRS + 眶/颅内/骨）

> [!info] 笔记定位
> 急性细菌性鼻窦炎（acute bacterial rhinosinusitis, ABRS）的**基础（诊断/病原/危险因素/治疗）+ 并发症（邻近蔓延：眶 / 颅内 / 骨）**一站速查。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q4850]]（危险因素=病毒 URI）+ [[mistakes/uworld-mistakes_2026-06#^Q3286]]（病原=不可分型 H. flu）+ [[mistakes/uworld-mistakes_2026-06#^Q15725]]（颅内脑脓肿→CT）+ [[mistakes/uworld-mistakes_2026-06#^Q3326]]（隔前 vs 眶蜂窝织炎）。

---

## 一、ABRS 基础 ⭐⭐

> [!info] 诊断标准（3 选 1）+ 病原 + 治疗
> **诊断（任一）**：① 症状 **≥10 天**无改善 ② **重症**（高热 ≥39℃ + 脓涕 ≥3 天）③ **双相恶化**（好转后再恶化 double-sickening）。
> **病原（同急性中耳炎 AOM）**：
> - **不可分型流感嗜血杆菌（nontypeable *Haemophilus influenzae*）≈ 40–50%** — **PCV13 后首位** ⭐
> - **肺炎链球菌（*S. pneumoniae*）≈ 20–25%**
> - **卡他莫拉菌（*Moraxella catarrhalis*）≈ 25%**（儿童占比更高）
> 
> **PCV13 疫苗效应**：13 价肺炎球菌结合疫苗普及后 → **肺炎链球菌占比↓ → 不可分型 H. flu 升为首位**（H. flu 是"不可分型"，**不在 PCV/Hib 疫苗覆盖范围**，故相对占比上升）。
> **最常见危险因素 = 病毒性上呼吸道感染（viral URI）** > 过敏性鼻炎（[[mistakes/uworld-mistakes_2026-06#^Q4850]]）。
> **治疗**：**阿莫西林（amoxicillin）± 克拉维酸（clavulanate）**。
> ⚠️ **GAS（S. pyogenes）= 咽炎，极少鼻窦炎**；**真菌（曲霉/毛霉）鼻窦炎 = 免疫抑制/控制差糖尿病**（鼻出血/鼻甲破坏/腭部焦痂）（[[mistakes/uworld-mistakes_2026-06#^Q3286]]）。

> [!tip] "PCV13 后病原洗牌"= ABRS / AOM 共享高频钩子 ⭐
> **ABRS（急性细菌性鼻窦炎）与 AOM（急性中耳炎 acute otitis media）病原谱相同**，受同一疫苗效应影响：
> - **PCV13 普及 → 肺炎链球菌占比↓ → 不可分型 H. flu 升首位**（H. flu "不可分型" **不在 PCV / Hib 覆盖**，Hib 疫苗只防 b 型荚膜株）。
> - **解题反射**：题干强调**"疫苗接种完整的儿童" + 中耳炎/鼻窦炎** → 优先想 **不可分型 H. flu**（不是肺炎链球菌）。
> - **治疗两病一致**：**阿莫西林（amoxicillin）高剂量 ± 克拉维酸（clavulanate）**（产 β-内酰胺酶的 H.flu / 卡他莫拉菌 → 加 clavulanate）。

## 二、并发症总览（邻近蔓延）⭐⭐⭐

```
ABRS 未治/进展 → 邻近蔓延
├─ 眶（最常见，尤儿童筛窦炎）
│   ├─ 隔前蜂窝织炎(preseptal)  眶隔前·仅睑肿·无眶内征 → 口服抗生素
│   ├─ 眶蜂窝织炎(orbital)      眶隔后·眼动痛+突出+视力↓ → 静脉抗生素+眶CT±手术
│   └─ 骨膜下/眶脓肿
├─ 颅内
│   ├─ 脑脓肿(额窦→额叶)  头痛+晨起呕吐(↑颅压)+意识改变+局灶 → CT/MRI+引流
│   ├─ 海绵窦血栓        双眼眶内征+CN III/IV/VI麻痹 → MRI/MRV+抗凝
│   └─ 脑膜炎/硬膜外·硬膜下脓肿
└─ 骨
    └─ Pott puffy tumor  额窦炎→额骨骨髓炎→前额肿(骨膜下脓肿)
```

## 三、眶并发症：隔前 vs 眶蜂窝织炎 ⭐⭐⭐

| | **隔前蜂窝织炎（preseptal）** | **眶蜂窝织炎（orbital）** |
|---|---|---|
| **部位/源** | 眶隔**前**；皮肤破损/虫咬 | 眶隔**后**；多**鼻窦炎（筛窦）蔓延** |
| **眶内征** | **无**（眼动正常无痛、无突出、视力正常）| **有**：**眼动痛/眼肌麻痹（ophthalmoplegia）+ 眼球突出（proptosis）+ 视力变化** |
| **病原** | 金葡/化脓性链球菌（皮肤）| 鼻窦菌（H.flu/肺炎链球菌等）|
| **处理** | **口服抗生素**（克林霉素 clindamycin 覆 GAS+MRSA）| **眶 CT + 静脉抗生素 ± 手术** |

> [!danger] 分水岭 = 有无眶内征
> **眼动痛/眼肌麻痹 + 眼球突出 + 视力变化 → 眶（急、要 CT+静脉抗生素）；全无 → 隔前（口服）**（[[mistakes/uworld-mistakes_2026-06#^Q3326]]）。

## 四、颅内并发症 ⭐⭐

> [!danger] 脑脓肿（额窦 → 额叶）→ 急诊影像
> **红旗**：持续头痛 + **晨起/卧位呕吐（↑颅压）** + 意识改变（嗜睡）+ 局灶神经/颈痛。→ **急诊头颅 CT（快、检早期脑炎；环形强化 = 脓肿）**；治疗 IV 抗生素 + 外科引流。**↑颅压时影像先于 LP**（[[mistakes/uworld-mistakes_2026-06#^Q15725]]）。

> [!warning] 海绵窦血栓（cavernous sinus thrombosis）
> 鼻窦/面部感染经**无瓣面静脉**逆行 → **双侧眶内征**（眼球突出 + CN **III/IV/VI** 麻痹眼肌麻痹 + V1/V2 感觉异常）+ 头痛高热。诊断 **MRI/MRV**；治疗 IV 抗生素 + **抗凝**。

> [!info] 其他颅内
> 脑膜炎、硬膜外/硬膜下脓肿（empyema）——均需神经影像 + IV 抗生素 ± 引流。

---

## 五、骨并发症：Pott puffy tumor

> [!info] Pott puffy tumor（额骨骨髓炎）
> **额窦炎 → 额骨骨髓炎 + 骨膜下脓肿 → 前额局限肿胀（"puffy"）**；多见青少年。可向后蔓延致颅内（硬膜外脓肿）。诊断影像（CT/MRI）；治疗 IV 抗生素 + 外科引流。

---

## 六、红旗 + 何时影像

- **眶红旗**（→ 眶 CT + 静脉抗生素）：**眼动痛/眼肌麻痹、眼球突出、视力变化、双侧眼征**。
- **颅内红旗**（→ 头颅 CT/MRI）：**晨起呕吐（↑颅压）、意识改变、局灶神经、抽搐、颈强直、剧烈头痛**。
- **骨红旗**：**前额局限肿胀（Pott puffy）**。
- 单纯 ABRS 无红旗 → 临床诊断 + 抗生素，无需影像。

## 七、Memory Hook

```
ABRS：≥10天/重症/双相恶化(3选1) → 不可分型H.flu+肺炎链球菌+卡他莫拉菌 → amoxicillin±clav
      危险因素=病毒URI；GAS是咽炎非鼻窦炎；真菌要免疫抑制/糖尿病
并发症蔓延：
  眶：隔前(无眶内征→口服) vs 眶(眼动痛+突出+视力↓→眶CT+静脉±手术)
  颅内：脑脓肿(晨起呕吐↑颅压→CT+引流) / 海绵窦血栓(双眼CN III-IV-VI→MRV+抗凝)
  骨：Pott puffy tumor(额骨骨髓炎→前额肿)
```

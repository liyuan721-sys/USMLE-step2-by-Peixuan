# A. 5 大急性血栓事件对比 ⭐⭐⭐

## A.1 核心认知 — 不都用抗凝

> [!warning] 关键反直觉
> 
> "急性血栓 = 都用抗凝" ❌ **错觉**
> 
> 5 个急症虽然都是血栓，但治疗哲学**完全不同**：
> 
> - **静脉系统**（PE/DVT）→ 抗凝核心
> - **冠脉**（MI）→ DAPT + PCI 核心（Heparin 辅助）
> - **脑动脉**（Stroke）→ tPA / 取栓核心（**急性不抗凝/不抗板**）⭐ 最特殊
> - **肢体动脉**（ALI）→ Heparin + 手术

### 诊断流程对比

| 急症         | 第一步 / 关键检查                | 第二步                                   | 风险分层          |
| ---------- | ------------------------- | ------------------------------------- | ------------- |
| **PE**     | 稳定吗？                      | 不稳定→床旁超声+tPA；稳定→Wells PE→D-dimer/CTPA | PESI          |
| **DVT**    | Wells DVT 评分              | < 2→D-dimer；≥ 2→Duplex US             | 位置（近端 vs 远端）  |
| **MI**     | **ECG 10 min + Troponin** | ECG 分 STEMI/NSTEMI                    | TIMI / GRACE  |

| **ALI**    | **临床（6 P's）**             | 立刻 IV Heparin + CTA                   | Rutherford 急性 |

### 急性治疗对比

| 急症 | 核心治疗 | 时间窗 | 抗凝？ | 抗板？ | 溶栓？ |
|---|---|---|---|---|---|
| **PE** | 抗凝 | 立刻 | ✅ **核心** | ❌ | 仅 Massive |
| **DVT** | 抗凝 | 立刻 | ✅ **核心** | ❌ | 仅 phlegmasia |
| **MI** | **DAPT + PCI** | **PCI 90 min / tPA 12h** | ✅ 辅助 | ✅ **核心 (DAPT)** | tPA 仅无 PCI 时 |
| **Stroke** | **tPA / 取栓** | **tPA 4.5h / 取栓 24h** | ❌ **急性禁用** | ❌ **24h 后才用** | ✅ **核心 (tPA)** |
| **ALI** | **Heparin + 紧急手术** | **6 h** | ✅ **核心** | ❌ | 偶尔 (CDT) |

### "第一件事"对比 ⭐⭐⭐ 最容易混（精确版）

> [!warning] "第一件事"要区分"诊断步骤"vs"治疗步骤"
> 
> 5 大急症的"第一件事"不完全一样：
> - **诊断驱动型**（PE 稳定 / DVT / Stroke）：先**确诊**再治疗
> - **临床驱动型**（PE 不稳定 / ALI）：边治疗边确诊
> - **混合型**（MI）：ASA 立刻 + ECG 并行

| 急症 | **诊断第一步** ⭐ | **治疗第一步** |
|---|---|---|
| **PE 稳定（低-中度可疑）** | Wells PE 评分 → D-dimer / CTPA | 确诊后抗凝 |
| **PE 稳定（高度可疑）** | + **评估抗凝禁忌** ⭐ | **经验性抗凝 + CTPA**（ACCP 推荐）⭐ |
| **PE 不稳定** | **床旁 TTE 看右心** ⭐⭐ | 见右心负荷 → **tPA 经验性** + 抗凝 |
| **DVT** | Wells DVT → **Duplex US** ⭐⭐ | 确诊后抗凝 |
| **MI** | **ECG 10 min** + Troponin | **ASA 嚼服立刻**（不等 ECG 完成）+ PCI |
| **Stroke** | **非增强 CT 排除出血** ⭐⭐ | < 4.5h → tPA |
| **ALI** | **临床（6 P's）即诊** ⭐⭐ | **IV Heparin 立刻**（不等影像）+ 紧急手术 |

#### 关键临床解释

**PE 稳定但高度可疑 → 经验性抗凝（ACCP 指南推荐）⭐**

```
临床现实：
- CTPA 完成需要 30 分钟 - 数小时
- 这段时间病人**血栓可能继续脱落**
- 新栓子 → 更多 PE → 死亡率↑↑
   ↓
**ACCP 2016 推荐**：
"高度 / 中度临床怀疑 PE → 等待诊断检查期间开始抗凝"
   ↓
所以：
**高度可疑 + 无禁忌 → 不等结果，先抗凝**
- 标准：Heparin / LMWH / Fondaparinux
- 如后续证明不是 PE → 停药即可
```

⭐ **核心权衡**：**"漏诊 PE" 死亡风险 > "经验性抗凝几小时" 出血风险**

**经验性抗凝前必评的禁忌** ⭐

```
绝对禁忌：
- 活动性出血（GI / 颅内）
- 近期颅内出血（< 3 月）
- 严重血小板减少（< 30k）
- 严重凝血障碍（INR > 4 无逆转）
- 近期大手术 / 创伤（< 14 天）
- 主动脉夹层（怀疑）⚠️
- 已知 HIT → 不能 Heparin/LMWH

相对禁忌：
- 近期手术（14-30 天）
- 血小板减少（30-50k）
- 严重肝 / 肾衰
- 既往严重出血
- 妊娠期硬膜外置管
```

⭐ **抗凝禁忌 + 确诊 PE → IVC filter** ⭐ 经典适应症

**PE 稳定完整决策流程（精确版）**

```
PE 稳定（SBP > 90）
        ↓
Step 1：临床概率评估（Wells / Geneva）
Step 2：**抗凝禁忌评估** ⭐
        ↓
┌─────────────────────────────────────┐
│  Wells > 6（**高度可疑**）+ 无禁忌    │
│  → **经验性抗凝立刻** ⭐               │
│  + CTPA 确诊                         │
│  → 确诊→继续 / 未确诊→停              │
├─────────────────────────────────────┤
│  Wells 2-6（**中度可疑**）            │
│  → D-dimer 或直接 CTPA               │
│  → 确诊后抗凝（如无禁忌）              │
├─────────────────────────────────────┤
│  Wells < 2（**低度可疑**）            │
│  → PERC 排除？                      │
│  → D-dimer 阴性 → 排除               │
│  → D-dimer 阳性 → CTPA              │
├─────────────────────────────────────┤
│  **任何可疑 PE + 抗凝禁忌**            │
│  → 紧急 CTPA 确诊                    │
│  → 确诊 → **IVC filter** ⭐           │
└─────────────────────────────────────┘
```

**PE 不稳定为什么不直接 CTPA？**
```
不稳定病人 ≠ 直接 CTPA
   ↓
1. 病人无法搬到 CT 室（休克 / 持续抢救）
2. CTPA 需造影剂 + 时间（数十分钟）
3. 移动不稳定病人 → 心搏骤停风险
   ↓
**床旁 TTE 第一步** ⭐
- 5-10 分钟出结果
- 不用搬动
- 看 RV 负荷标志（dilation / hypokinesis / McConnell / D-shaped LV）
- 经验性 tPA 决策依据
```

⭐ **TTE 是"穷人的 CTPA"**——床旁可做、看到右心负荷就能决定 tPA。

**DVT 为什么不直接抗凝？**
```
怀疑 DVT ≠ 直接抗凝
   ↓
1. **抗凝有出血风险**（误诊代价大）
2. **必须确诊**（Duplex US 或 D-dimer + US）
3. 标准流程：Wells → 分诊 → 确诊 → 抗凝
   ↓
**例外**（罕见）：
- 临床高度怀疑 + 影像延迟（如夜间）→ 经验性 LMWH + 次日晨 US
- Massive PE 不稳定 → 保命第一，经验性抗凝同时确诊
```

⭐ **USMLE 默认流程**：**Wells → 确诊（D-dimer/US）→ 抗凝**

**ALI 为什么不等影像？**
```
ALI 6 P's = 临床即诊 ⭐
   ↓
**6 小时窗口** —— 等影像 = 截肢
   ↓
**IV Heparin 立刻** + 紧急联系血管外科
+ 同时 CTA（不延误治疗）
```

## A.3 为什么 Stroke 不立即抗凝？（深度理解）

```
缺血性卒中 → 脑组织缺血损伤
   ↓
血脑屏障受损 → 受损脑组织血管脆弱
   ↓
此时**抗凝** → **大量脑出血** ⭐ 致命
   ↓
病人可能死于"治疗"而非原发病
```

⭐ 这与肢体动脉栓塞**完全不同**：
- 肢体动脉栓塞 → Heparin **保住肢体**
- 脑动脉栓塞 → Heparin **杀死病人**（出血风险）

### Stroke 抗血栓药时间点

| 时间 | 操作 |
|---|---|
| **0-4.5 h** | tPA（如符合 + 排除出血）|
| **0-24 h** | LVO → 机械取栓 |
| **24 h 内** | ❌ 禁用抗凝 + 抗板 |
| **24-48 h** | 启动 **ASA** |
| **数天-周后**（稳定）| AFib → 启动 **DOAC**（小卒中 3-5 天 / 中 5-7 天 / 大 2 周）|

⚠️ **特殊例外**：颈/椎动脉夹层可早期抗凝（但指南首选仍是抗板）

## A.4 MI 治疗 MONA-BASH 现代化

### 经典 MONA-BASH 拆解

| 字母 | 名称 | 作用 | 是否"立刻"给 |
|---|---|---|---|
| **M** | Morphine | 镇痛 + 静脉扩张 | ⚠️ 仅剧痛（有研究显示↑死亡率，避免常规）|
| **O** | Oxygen | 改善缺氧 | ⚠️ **仅 SpO₂ < 90%** |
| **N** | Nitroglycerin | 冠脉扩张 + 前负荷↓ | ⚠️ **下壁 MI / RV 梗死禁用** |
| **A** | Aspirin | 抗血小板 | ✅ **立刻 162-325 mg 嚼服** ⭐ |
| **B** | β-blocker | 心率↓ + 耗氧↓ | 24h 内若稳定（不是立刻）|
| **A** | ACEi / ARB | 心肌重塑预防 | 24h 内若稳定 |
| **S** | Statin | 高强度（Atorvastatin 80）| 入院后尽快 |
| **H** | Heparin | 抗凝辅助 | ✅ 急性期辅助 |

### 现代实践：3 核心 + 4 辅助 + 2 按需 ⭐

```
立刻必给（"核心"）：
- ASA 162-325 mg 嚼服 ⭐
- P2Y12 抑制剂（Ticagrelor / Clopidogrel / Prasugrel）⭐
- Heparin 或 Bivalirudin（PCI 辅助）⭐

按需给（"按需"）：
- Morphine - 仅剧痛
- Oxygen - 仅 SpO₂ < 90%

数小时-24h 内（"稳定后"）：
- Nitroglycerin（胸痛 + BP 正常 + 非下壁）
- β-blocker（24h 内稳定 + EF 可）
- ACEi / ARB（24h 内稳定 + LVEF < 40%）
- Statin 高强度
```

⭐ **真正"必须立刻"的只有 A（ASA）+ P2Y12 + Heparin** —— 其他都有条件。

### MONA-BASH USMLE 经典陷阱

1. **常规给氧**：SpO₂ ≥ 90% 不给氧（无益甚至有害）
2. **下壁 MI 给硝酸甘油**：可能合并 RV 梗死，依赖前负荷 → 低血压休克
3. **所有 MI 都给 β-blocker**：急性失代偿 HF / 心源性休克 / 心动过缓 = 禁忌
4. **把吗啡当一线**：首选 ASA + P2Y12 + Heparin + PCI，吗啡仅在 NTG 无效时考虑

## A.5 时间窗数字记忆

```
"90-4.5-6-24"
- 90 min → MI PCI 窗口
- 4.5 h → Stroke tPA 窗口  
- 6 h → ALI 不可逆肌肉损伤窗口
- 24 h → Stroke 取栓 / 24h 后 ASA
```

## A.6 同栓子不同脏器 = 不同治疗 ⭐ 经典 USMLE 对比题

### 对比题 1：AFib + Stroke
> "65 岁男 + AFib，未抗凝。突发左侧偏瘫 2 小时，CT 无出血。下一步？"
> 
> **答案：IV tPA**（不是抗凝！即使是 AFib 栓塞性 stroke）

### 对比题 2：AFib + ALI
> "65 岁男 + AFib，未抗凝。突发左下肢剧痛 + 苍白 + 无脉 2 小时。下一步？"
> 
> **答案：IV Heparin + 紧急手术**

⭐ **同 AFib 栓塞，治疗截然相反** —— 取决于**栓到哪**。

## A.7 Memory Hook（终极版）

### 版本 2：脏器决定治疗（树状版）

> [!success] 
> **"血栓 5 急症，看脏器决定治"**
> 
> ```
> 静脉血栓（PE, DVT）→ 抗凝（核心）
> 动脉血栓（ALI, MI, Stroke）→ **各不相同**！
>    ├── ALI（肢体）→ Heparin + 手术
>    ├── MI（心）→ DAPT + PCI
>    └── Stroke（脑）→ tPA（**不抗凝/不抗板**）⭐ 最特殊
> ```
> 
> 记住：**"脑子最娇贵，急性不抗凝不抗板"**

## A.8 分 3 步学习建议

> [!tip] 不要试图一次记住全部

**Step 1（今天/明天）**：核心治疗记 5 个
- PE = 抗凝；DVT = 抗凝
- MI = DAPT + PCI
- Stroke = tPA / 取栓
- ALI = Heparin + 手术

**Step 2（后天）**：第一步记 5 个
- PE 稳定吗？DVT Wells？
- MI ECG！Stroke CT！
- ALI 6 P's → Heparin！

**Step 3（一周后）**：时间窗记 4 个
- MI PCI 90 min
- Stroke tPA 4.5h
- Stroke 取栓 24h
- ALI 6h

# B. 抗凝药完整对比 ⭐⭐⭐

## B.1 抗凝药分类树

```
抗凝药分类（按机制）：
│
├── A. 间接因子 Xa / IIa 抑制（通过 ATIII）
│   ├── **UFH**（普通肝素）— Xa + IIa 都抑
│   ├── **LMWH**（Enoxaparin / Dalteparin）— 主要 Xa
│   └── **Fondaparinux** — 选择性 Xa
│
├── B. 维生素 K 拮抗剂（VKA）
│   └── **Warfarin** — 抑制 II/VII/IX/X + 蛋白 C/S 合成
│
├── C. 直接凝血酶抑制剂（DTI）
│   ├── **Dabigatran**（口服，DOAC）
│   ├── **Bivalirudin**（IV，PCI 用）
│   ├── **Argatroban**（IV，HIT 用）
│   └── **Lepirudin**（IV）
│
└── D. 直接 Xa 因子抑制剂（口服，DOAC）
    ├── **Rivaroxaban**（"-xa-ban"）
    ├── **Apixaban**
    └── **Edoxaban**
```

⭐ **DOAC = Direct Oral AntiCoagulants** = 直接口服抗凝药
- 包括 Dabigatran（凝血酶抑制剂）+ 3 个 Xa 抑制剂

## B.2 UFH（普通肝素，Unfractionated Heparin）

### 机制
```
UFH 与 Antithrombin III (ATIII) 结合
   ↓
ATIII 活性 ↑↑↑（1000 倍）
   ↓
抑制：**因子 Xa + IIa（凝血酶）**
```

### 关键特征

| 特征 | 详情 |
|---|---|
| **给药** | **IV 持续输注**（首选）或 SC |
| **起效** | **立刻**（IV）/ 1-2h（SC）|
| **半衰期** | **1-2 小时**（最短）⭐ |
| **监测** | **aPTT**（目标 1.5-2.5x 基线）⭐ |
| **拮抗剂** | **Protamine sulfate** ⭐ |
| **过胎盘** | ❌ 不过（妊娠可用）|
| **肾排泄** | 部分 → **严重肾衰首选 UFH** ⭐ |

### 临床应用 ⭐
- **PCI 术中抗凝**（首选，因为半衰期短便于调整）
- **严重肾衰** + 需要抗凝
- **不稳定病人**（可快速调整 / 停止）
- **高出血风险**（短半衰期 → 出血时停药快）
- 桥接 Warfarin（旧方案，现已少用）

### 并发症 ⭐ USMLE 必考
1. **出血**（最常见）
2. **HIT (Heparin-Induced Thrombocytopenia)** ⭐
   - Type I：< 100k，前 2 天，非免疫，可继续用
   - **Type II**：> 5 天，免疫性（PF4-肝素抗体），**血栓而非出血** ⭐
   - 处理：**立刻停 UFH/LMWH** + 改用 **Argatroban / Bivalirudin / Fondaparinux**
3. **骨质疏松**（长期用）
4. **高钾**（醛固酮抑制）

⚠️ **HIT 经典 stem**：术后 5-10 天 + 血小板↓ > 50% + 新血栓（不是出血）→ HIT

## B.3 LMWH（低分子肝素）

### 代表药
- **Enoxaparin** (Lovenox) — 最常用
- Dalteparin
- Tinzaparin

### 机制
```
LMWH 分子较小 → 与 ATIII 结合
   ↓
**主要抑 Xa**（不抑 IIa）
   ↓
更选择性 + 更可预测
```

### 关键特征

| 特征 | 详情 |
|---|---|
| **给药** | **SC 注射**（每日 1-2 次）|
| **起效** | 1-2 小时 |
| **半衰期** | **4-6 小时** |
| **监测** | 通常**不需要监测** ⭐（剂量按体重）|
| 特殊监测 | **Anti-Xa**（肥胖 / 妊娠 / 肾衰）|
| **拮抗剂** | Protamine（**部分逆转**，~60%）|
| **过胎盘** | ❌ 不过 |
| **肾排泄** | 是 → **CrCl < 30 慎用，需调整** |
SC=subcutaneous

### 临床应用 ⭐
- **VTE 治疗 + 预防**（最常用）
- **妊娠期抗凝** ⭐ 首选（不过胎盘）
- **癌症相关 VTE** ⭐ 首选（CLOT 试验）
- **桥接 Warfarin**（外科前后）
- **DVT/PE 急性期**（DOAC 替代之一）

### LMWH vs UFH 优势
- ✅ 不需要常规监测
- ✅ 可门诊使用（SC 注射）
- ✅ HIT 风险低（但仍可能）
- ❌ 半衰期长（出血时不易快速逆转）
- ❌ 肾衰需调整

## B.4 Warfarin（华法林）

### 机制
```
Warfarin 抑制 **VKORC1** 酶
   ↓
维生素 K 不能再循环
   ↓
不能合成：
- 凝血因子：**II, VII, IX, X**（"1972"）
- 抗凝蛋白：**Protein C, S**
   ↓
**初始** Protein C/S 先下降（半衰期短）
   ↓
**初期反而 → 高凝状态** ⭐
   ↓
所以需要 **bridge** with Heparin（前 5 天）⭐
```

### 关键特征

| 特征 | 详情 |
|---|---|
| **给药** | **PO**（每日 1 次）|
| **起效** | **3-5 天**（达稳态）⭐ |
| **半衰期** | **40 小时** |
| **监测** | **INR**（目标 2-3，部分 2.5-3.5）⭐ |
| **拮抗剂** | **Vitamin K**（口服 / IV，慢）+ **FFP / PCC**（快，紧急）⭐ |
| **过胎盘** | ✅ **过胎盘** ⚠️（致畸 — 妊娠禁忌）|
| **代谢** | CYP2C9（多药相互作用）|

### Warfarin 致畸（妊娠禁忌）⭐
- **Trimester 1 暴露**：胚胎病（鼻发育不全 / 软骨钙化点）
- **任何时期**：CNS 异常 / 出血
- 妊娠**绝对禁忌** → 用 LMWH 替代

### 临床应用
- **AFib + 高 CHA₂DS₂-VASc**（部分情境）
- **机械瓣膜** ⭐ 必须 Warfarin（DOAC 在机械瓣失败！）
- **APS（抗磷脂综合征）** ⭐ DOAC 失败率高
- 严重肾衰（DOAC 禁忌时）
- VTE 长期（DOAC 替代之一）

### Warfarin 食物 / 药物相互作用 ⭐
**↑ INR（出血风险）**：
- 食物 ↓ Vit K：禁食 / 腹泻
- 药物：抗生素（破坏肠菌）/ 胺碘酮 / Cimetidine / 甲状腺素

**↓ INR（血栓风险）**：
- 食物 ↑ Vit K：**绿叶蔬菜**（菠菜 / 西兰花）⚠️ 不是不能吃，是要**稳定**摄入
- 药物：Phenytoin / Rifampin / 巴比妥（CYP 诱导）

⭐ **关键教育**：病人**不需要避免绿叶蔬菜**，要**保持稳定摄入量**

## B.5 DOAC（直接口服抗凝药）

![[{B34BE628-0DEB-460D-B8C1-C23FB59D4438}.png]]
### 直接凝血酶抑制剂（DTI）家族对比

|药物|途径|是否 DOAC|适应症|
|---|---|---|---|
|**Dabigatran**|**PO（口服）**|✅ **是 DOAC**|房颤、DVT/PE|
|**Bivalirudin**|**IV**|❌ 不是 DOAC|PCI 术中抗凝|
|**Argatroban**|**IV**|❌ 不是 DOAC|**HIT**（肝素诱导血小板减少）|
|**Desirudin / Lepirudin**|**SC / IV**|❌ 不是 DOAC|HIT（基本退市）|

**它们机制都一样**（直接结合凝血酶 IIa 活性位点），但**只有 Dabigatran 是口服的，所以只有它算 DOAC**。
### 4 个 DOAC

| 药 | 机制 | 给药 | 拮抗剂 |
|---|---|---|---|
| **Dabigatran** | 直接 IIa（凝血酶）抑制 | BID | **Idarucizumab** ⭐ |
| **Rivaroxaban** | 直接 Xa 抑制 | QD | **Andexanet alfa** ⭐ |
| **Apixaban** | 直接 Xa 抑制 | BID | **Andexanet alfa** ⭐ |
| **Edoxaban** | 直接 Xa 抑制 | QD | **Andexanet alfa** ⭐ |

### 共同特征

| 特征 | 详情 |
|---|---|
| **给药** | PO（每日 1-2 次）|
| **起效** | **1-3 小时**（快）⭐ |
| **半衰期** | 8-15 小时 |
| **监测** | **不需要监测** ⭐（最大优势）|
| **过胎盘** | ⚠️ 妊娠数据少，**避免使用** |
| **代谢** | 部分肾排泄（需按 CrCl 调整）|

### DOAC 优势 vs Warfarin
- ✅ 不需要 INR 监测
- ✅ 起效快（不需 bridging）
- ✅ 食物 / 药物相互作用少
- ✅ 颅内出血风险**低**
- ✅ 剂量固定
- ❌ 价格贵
- ❌ **机械瓣禁忌** ⭐
- ❌ **APS 失败率高** ⭐
- ❌ **妊娠避免**
- ❌ 严重肾衰禁忌（CrCl < 15-30）

### DOAC 适应症 ⭐
- **AFib（非瓣膜性）** + CHA₂DS₂-VASc ≥ 2（男）/ 3（女）
- **VTE 治疗**（DVT / PE）首选
- **VTE 长期预防**（无诱因 / 复发）
- **POSTTHROMBOTIC SYNDROME 预防**（部分情境）

### DOAC 禁忌 ⭐ USMLE 必背
1. **机械瓣膜** ⚠️（RE-ALIGN 试验显示失败）
2. **APS**（抗磷脂综合征）— 失败率高
3. **妊娠 / 哺乳**
4. **严重肾衰**（CrCl < 15-30 看具体药）
5. **严重肝衰**（Child-Pugh C）
6. 风湿性二尖瓣狭窄

## B.6 Fondaparinux + 间接抗凝家族深度对比

> [!danger] 一句话锁定 ⭐
> **Fondaparinux ≠ 最小的 Heparin**
> 它是**人工合成的五糖 (pentasaccharide)**，只模仿了 Heparin 上结合 AT III 的那 5 个糖单位 — **不是 Heparin 家族成员**。
> 结构、来源、作用机制都不一样。

### 机制
- **选择性 Xa 抑制**（合成五糖）
- 依赖 AT III，但**只抑制 Xa，不抑制 IIa**（链太短桥不到凝血酶）

### B.6.1 三者本质区别 — UFH vs LMWH vs Fondaparinux

| 特征 | **UFH（普通肝素）** | **LMWH**（Enoxaparin）| **Fondaparinux** |
|---|---|---|---|
| **来源** | 猪肠 / 牛肺**天然提取**糖胺聚糖 | UFH **化学 / 酶解**切短 | **完全人工合成**五糖 |
| **大小** | 长链 5,000-30,000 Da（平均 ~15,000）| 中等 ~4,500 Da | 小 **1,728 Da**（固定 5 糖）|
| **结构** | 多种糖单位混合（长度不一）| 多种糖单位混合（较短）| **单一明确**五糖序列 |
| **抑制 Xa** | ✅✅ | ✅✅✅（主要）| ✅✅✅（**唯一作用**）|
| **抑制 IIa（凝血酶）** | ✅✅（同等强）| ✅（弱，链太短）| ❌ **完全不抑制** |
| **依赖 AT III** | ✅ | ✅ | ✅ |

### B.6.2 机制差异的核心 — 链长决定能否桥接 IIa

```
要抑制 Xa：只需要 AT III 结合到药物上即可
        药物 ── AT III ── Xa  ✅（短链即可）

要抑制 IIa（凝血酶）：需要药物同时"桥接" AT III 和 IIa
        药物长链 ───────────
         │            │
        AT III       IIa     ← 需要 ≥ 18 个糖单位才够长
```

> [!tip] 一句话推理
> - **UFH 长** → 能桥接 → IIa 和 Xa 都抑（**1 : 1**）
> - **LMWH 较短** → 桥接能力 ↓ → 偏 Xa（**Xa : IIa ≈ 3-4 : 1**）
> - **Fondaparinux 只有 5 糖** → 桥不到 IIa → **纯 Xa 抑制剂**

### B.6.3 USMLE 高频对比表 ⭐⭐⭐

| 项目 | UFH | LMWH | Fondaparinux |
|---|---|---|---|
| **给药** | IV / SC | SC | SC（**QD** 每日 1 次）|
| **半衰期** | 1-2 h | 4-6 h | **17-21 h**（最长）|
| **监测** | **aPTT** | 一般不监测（高危查 anti-Xa）| 不监测 |
| **肾清除** | 弱（网状内皮系统）| 中等 | **几乎完全肾排** |
| **肾衰能用？** | ✅ **首选** | ⚠️ 减量 | ❌ **CrCl < 30 禁用** |
| **HIT 风险** | 高（~3%）| 低（~0.5%）| 极低 / 几乎无 |
| **HIT 病史能用？** | ❌ 禁用 | ❌ 禁用 | ⚠️ **理论可用但 USMLE 非首选**（不和 PF4 抗体交叉，但 Argatroban 才是首选）|
| **逆转** | **Protamine** 完全逆转 | Protamine 部分（~60%）| **Protamine 无效 ❌**（rFVIIa / Andexanet alfa off-label）|
| **致畸** | 不致畸（不过胎盘）| 不致畸 | 理论不过胎盘，**数据少** |

### B.6.4 临床适应症速记

| 药 | 主战场 |
|---|---|
| **UFH** | 急性血栓需快速起效 / 可逆转、**PCI**、CRRT、**孕妇**、**肾衰**、围手术期桥接 |
| **LMWH**（Enoxaparin）| VTE 预防 / 治疗、ACS、**孕妇首选！** |
| **Fondaparinux** | VTE 预防 / 治疗、HIT 既往史（非急性 HIT 首选）、不愿用 Heparin 类 |

### B.6.5 4 大 USMLE 经典陷阱

> [!danger] 陷阱 1：HIT 患者抗凝
> - **首选**：**Argatroban** / Bivalirudin（DTI）
> - Fondaparinux 理论可用（不和 PF4 抗体交叉），但 **USMLE 不选 Fondaparinux 当首选** → 选 **Argatroban**
> - 详见 [[#J.1 陷阱 1：HIT 患者的抗凝选择]]

> [!danger] 陷阱 2：肾衰患者抗凝
> - ❌ **Fondaparinux**（CrCl < 30 禁用 — 几乎完全肾排）
> - ❌ LMWH 需减量
> - ✅ **UFH 首选**（非肾代谢）
> - 详见 [[#J.3 陷阱 3：肾衰患者抗凝选药（代谢分级）]]

> [!danger] 陷阱 3：Protamine 逆转效力差异
> 
> | 药 | Protamine 效力 | 备选 |
> |---|---|---|
> | **UFH** 大出血 | ✅ **完全逆转**（1 mg : 100 U Heparin，限最近 1 小时给）| — |
> | **LMWH** 大出血 | ⚠️ 部分逆转（~60%，因链短结合差）| 严重时考虑 rFVIIa |
> | **Fondaparinux** 大出血 | ❌ **无效** | **rFVIIa** / **Andexanet alfa**（off-label）|

> [!danger] 陷阱 4：作用机制选择题（铁律回答）
> - "通过 AT III **只抑制 Xa**"？ → **Fondaparinux**
> - "抑制 Xa 和 IIa **比例相等**"？ → **UFH**
> - "抑制 **Xa > IIa**"？ → **LMWH**

### B.6.6 完整"间接抗凝药"家族树（依赖 AT III）

```
依赖 AT III 的抗凝药
│
├── UFH (5,000-30,000 Da)
│   └── 抑制 Xa = IIa (1:1)
│
├── LMWH (~4,500 Da)
│   ├── Enoxaparin
│   ├── Dalteparin
│   └── 抑制 Xa >> IIa (3-4:1)
│
└── Fondaparinux (1,728 Da, 合成 5 糖)
    └── 只抑制 Xa（纯 Xa）

         分子量 ↓  →  Xa 选择性 ↑  →  IIa 抑制 ↓
```

### B.6.7 Memory Hook

- **"Fonda = Found a 5-mer"** — 人工合成的五糖
- **"链越短，越偏 Xa"**：
  - UFH（长）→ Xa + IIa
  - LMWH（中）→ 偏 Xa
  - Fondaparinux（最短）→ 纯 Xa
- **"-parin 结尾 ≠ Heparin"** — Fonda**parin**ux 名字带 parin 但**不是 Heparin**
- **"Protamine 救肝素，救不了 Fondaparinux"**

> [!success] 终极 trade-off 口诀 ⭐
> - **链短 = 干净、稳定、长效、不可逆**
>   - 干净 = HIT 少、出血少
>   - 稳定 = 不用监测
>   - 长效 = 每日 1 针
>   - 不可逆 = Protamine 救不回来
> - **链长 = 全面、灵活、可救、麻烦**
>   - 全面 = Xa + IIa 双抑制
>   - 灵活 = 起效快、可滴定
>   - 可救 = Protamine 完全逆转
>   - 麻烦 = 必须 aPTT、HIT 风险高
> 
> **核心 trade-off**：灵活性 vs 便捷性 = 短期 vs 长期 = 急性 vs 稳定 = **UFH vs LMWH / Fondaparinux**

### B.6.8 凝血级联 + "链短偏 Xa" 的 6 大临床后果（机制层面）

> [!info] 为什么链长决定一切？先看 Xa / IIa 在级联中的位置

```
内源性（aPTT）        外源性（PT/INR）
   ↓                    ↓
   XII → XI → IX        VII
              ↓          ↓
             VIII       组织因子（TF）
              ↓          ↓
              └──→ X ──→ Xa ←┐
                       ↓     │
                  凝血酶原 (II) → IIa（凝血酶）
                       ↓
                  纤维蛋白原 → 纤维蛋白 → 血栓

放大效应：1 个 Xa → 生成 ~1,000 个 IIa
        1 个 IIa → 切 ~1,000 个纤维蛋白原
```

> [!tip] 两个关键位置
> - **Xa 在"上游放大点"** — 抑制它 = 提前掐源头（高效但温和）
> - **IIa 在"下游执行点"** — 抑制它 = 拦最后一道（强力但出血）+ 还会激活血小板 + 反馈激活 V/VIII/XI

#### "链短偏 Xa" 的 6 个机制结论

| # | 临床后果 | **机制 Why** |
|---|---|---|
| 1️⃣ | **出血风险 ↓** | IIa 不只形成血栓，还**激活血小板 + 维持初级止血**。抑 IIa = 双重抗凝（凝血 + 血小板）→ 出血↑；只抑 Xa = "只挡上游"，血小板和已生成的 IIa 仍能完成止血 |
| 2️⃣ | **药效可预测性 ↑** | UFH 长链不均一（5,000-30,000 Da）+ 与白蛋白 / PF4 / 内皮 / 巨噬细胞**广泛结合** → 个体差异大；短链结合↓ → 剂量-反应**线性** → 按体重给药即可 |
| 3️⃣ | **半衰期 ↑** | UFH 长链被**网状内皮系统（巨噬细胞）**快速吞噬清除；短链主要靠**肾排**，清除慢 → Fondaparinux 半衰期 17-21h（QD 给药） |
| 4️⃣ | **HIT 风险 ↓** | HIT 机制 = Heparin **长链** + PF4 → 形成**大免疫复合体** → IgG 激活血小板。短链**形不成大复合体** → Fondaparinux 几乎无 HIT |
| 5️⃣ | **Protamine 逆转效力 ↓** ⭐ | Protamine = **碱性多肽**，与 Heparin 长链**负电荷**静电结合中和。短链 = 结合面积↓ → LMWH 只 60%；Fondaparinux 5 糖太短 → **完全无效** |
| 6️⃣ | **起效速度 ↓ + 可滴定性 ↓** | UFH IV = 几分钟即时调整；LMWH SC = 1-2h；Fondaparinux SC = 2-3h → 急诊 / 围手术期需"秒级调控"只能选 UFH |

> [!warning] 关键数据点（USMLE 可能直接考百分比）
> - **大出血率**：UFH ~3-5% > LMWH ~1-2% > Fondaparinux ~1% > Apixaban ~0.6%
> - **HIT 发生率**：UFH ~3% > LMWH ~0.5% > Fondaparinux ≈ 0

### B.6.9 Heparin 类内部决策树 ⭐

> [!info] 与 E 节互补
> E.1-E.3 = 按"血栓类型"和"特殊人群"选大类药；本节 = **Heparin 三兄弟内部**怎么选

```
临床场景 → 选哪种 Heparin？
│
├── 急性、不稳定、可能要手术 → UFH
│   ├── 起效快（几分钟 IV）✅
│   ├── 可监测 aPTT ✅
│   ├── Protamine 完全逆转 ✅
│   └── 肾衰也能用 ✅
│
├── 稳定、门诊、长期 → LMWH
│   ├── 每日 1-2 针 SC ✅
│   ├── 不用监测 ✅
│   ├── 出血风险低 ✅
│   └── 孕妇首选 ✅
│
├── 不愿打 LMWH / HIT 既往（非急性） → Fondaparinux
│   ├── 每日 1 针 SC ✅
│   ├── 无 HIT 风险 ✅
│   ├── 肾衰禁用 ❌（CrCl<30）
│   └── 出血无解药 ❌（Protamine 无效）
│
└── 急性 HIT → Argatroban / Bivalirudin
    └── 直接凝血酶抑制剂（DTI），不属 Heparin 家族
```

> [!tip] 一句话选药逻辑
> - **要灵活**（急性 / 围手术期 / 肾衰）→ UFH
> - **要便捷**（门诊 / 孕妇 / 长期）→ LMWH
> - **HIT 既往 + 稳定**（非急性）→ Fondaparinux
> - **急性 HIT** → 跳出 Heparin 家族 → Argatroban

## B.7 抗凝药终极对比表 ⭐⭐⭐

| 药 | 给药 | 起效 | 监测 | 拮抗剂 | 妊娠 | 肾衰 | 关键应用 |
|---|---|---|---|---|---|---|---|
| **UFH** | IV/SC | 立刻 | **aPTT** | **Protamine** | ✅ 可用 | ✅ **首选** | PCI / 不稳定 / 严重肾衰 |
| **LMWH** | SC | 1-2h | 通常无（Anti-Xa）| Protamine(部分) | ✅ **首选** | ⚠️ 调整 | VTE / 妊娠 / 癌症 |
| **Warfarin** | PO | 3-5 天 | **INR** | **Vit K + FFP/PCC** | ❌ 致畸 | ✅ | **机械瓣** / **APS** |
| **DOAC** | PO | 1-3h | **不需要** | Idarucizumab/Andexanet | ❌ 避免 | ⚠️ 调整 / 禁忌 | AFib / VTE 首选 |
| **Fondaparinux** | SC | 2-3h | 无 | ❌ 无 | 慎用 | ⚠️ 禁忌 | **HIT** |

⭐ **5 大记忆钩子**：
1. **机械瓣** = **只能 Warfarin** ⭐
2. **妊娠** = **只能 LMWH** ⭐
3. **HIT** = **不能用 Heparin/LMWH** → **Argatroban / Bivalirudin**（USMLE 首选 DTI）；Fondaparinux 理论可用但**非首选** ⭐
4. **严重肾衰** = **首选 UFH**（不靠肾排泄）⭐
5. **AFib / VTE 一般** = **DOAC 首选** ⭐

## B.8 特殊人群选药

| 人群 | 首选 | 避免 | 原因 |
|---|---|---|---|
| **妊娠** | **LMWH** ⭐ | Warfarin（致畸）/ DOAC（数据少）| 不过胎盘 |
| **机械瓣** | **Warfarin** ⭐ | DOAC（失败）| RE-ALIGN 试验 |
| **APS** | **Warfarin** ⭐ | DOAC（失败率高）| TRAPS 试验 |
| **癌症 + VTE** | **LMWH** ⭐ | Warfarin | CLOT 试验；DOAC 部分情境可 |
| **严重肾衰**（CrCl < 30）| **UFH** | DOAC（除 Apixaban） | 不靠肾排泄 |
| **肥胖（BMI > 40）** | LMWH（按体重）/ Warfarin | DOAC 数据少 | 剂量不确定 |
| **HIT 病人** | **Argatroban / Bivalirudin** ⭐ | UFH / LMWH | 抗体交叉 |
| **老年（出血风险高）** | 低剂量 Apixaban | Warfarin（不稳定）| 半衰期短 + 拮抗剂可用 |
| **PCI 术中** | **UFH / Bivalirudin** ⭐ | DOAC | 短半衰期便于调整 |

# C. 抗血小板药完整对比 ⭐⭐⭐

## C.1 抗血小板药分类树

```
抗血小板药分类（按靶点）：
│
├── A. **COX 抑制剂**
│   └── **Aspirin (ASA)** — 不可逆 COX-1 抑制
│
├── B. **P2Y12 受体抑制剂**（ADP 受体）
│   ├── **Clopidogrel**（Plavix）— 前药，不可逆
│   ├── **Prasugrel**（Effient）— 前药，不可逆，强效
│   ├── **Ticagrelor**（Brilinta）— 直接作用，可逆 ⭐
│   └── Ticlopidine（旧药，少用）
│
├── C. **GP IIb/IIIa 抑制剂**（IV 用，PCI 术中）
│   ├── **Abciximab**（ReoPro）
│   ├── **Eptifibatide**
│   └── **Tirofiban**
│
└── D. **PDE 抑制剂**
    ├── **Cilostazol** — PDE3，用于 PAD claudication
    └── **Dipyridamole** — PDE5，组合用于 Stroke 二级预防
```

## C.2 ASA（阿司匹林）

### 机制
```
ASA 不可逆乙酰化 **COX-1**
   ↓
**TXA2（血栓素 A2）合成 ↓**
   ↓
血小板聚集 ↓
   ↓
**血小板寿命内**（7-10 天）持续作用
```

### 关键特征

| 特征 | 详情 |
|---|---|
| **给药** | PO（嚼服更快吸收）|
| **抗血小板剂量** | **81 mg/d**（低剂量）或急性 162-325 mg 嚼服 |
| **起效** | 30-60 分钟（嚼服更快）|
| **作用时间** | **7-10 天**（血小板生命周期）|
| **拮抗剂** | ❌ 无（停药 + 等血小板更新）|
| 严重出血 → | 输血小板 |

### 临床应用 ⭐
- **MI 急性**：**162-325 mg 嚼服** ⭐⭐⭐
- **MI 二级预防**：81 mg 终身
- **稳定 PAD**：81 mg 终身
- **Stroke 二级预防**（非心源性）：81 mg
- **Stroke 急性（24h 后）**：81 mg
- **AFib + DAPT 后**：通常停 ASA 保 P2Y12 + DOAC

### ASA 副作用
- **GI 出血** ⭐
- **Reye 综合征**（儿童病毒感染时）⚠️
- 哮喘加重（AERD - Samter's triad）
- 肾损伤（长期高剂量）

## C.3 P2Y12 抑制剂

### 3 个常用药对比

| 药 | 类型 | 给药 | 起效 | 强度 | 关键 |
|---|---|---|---|---|---|
| **Clopidogrel** | 前药 / 不可逆 | PO | **2-6 小时** | 中等 | CYP2C19 多态性 → 部分人无效 |
| **Prasugrel** | 前药 / 不可逆 | PO | **30 分钟** | 最强 | ⚠️ Stroke 史 / 75+ 禁忌 |
| **Ticagrelor** | 直接作用 / **可逆** ⭐ | PO BID | **30 分钟** | 强 | 副作用：**呼吸困难** ⭐ |

⭐ **USMLE 高频**：
- **Clopidogrel** 在 CYP2C19 慢代谢者效果差 → 可基因检测
- **Prasugrel** 强但出血风险高 → 禁用于既往 stroke / TIA / 75+ / 体重 < 60 kg
- **Ticagrelor** 副作用独特：**呼吸困难 / 心动过缓**（腺苷再摄取）

### 临床应用
- **MI 急性 + DAPT**：ASA + P2Y12（首选 Ticagrelor）
- **PCI / Stent 后**：DAPT 至少 6-12 月
- **Stroke 二级预防**：Clopidogrel 单药或 + ASA（CHANCE / POINT 试验）
- **PAD 二级预防**：替代 ASA（如 ASA 不耐受）

## C.4 GP IIb-IIIa 抑制剂

### 机制
- 直接阻断血小板表面 GP IIb/IIIa 受体
- 阻止**纤维蛋白原桥接**血小板
- 强效抗血小板

### 3 个药
- **Abciximab** — 单克隆抗体，半衰期长
- **Eptifibatide** — 短半衰期（可逆）
- **Tirofiban** — 短半衰期（可逆）

### 临床应用
- **PCI 术中**（高风险病人）
- 急性冠脉综合征（部分情境）
- ⚠️ 仅 IV，**仅住院使用**

### 副作用
- 出血（特别穿刺部位）
- **血小板减少**（急性，可严重）⭐

## C.5 PDE 抑制剂

### Cilostazol
- 机制：**PDE3 抑制** → cAMP ↑ → 抗血小板 + 血管扩张
- 适应：**PAD claudication**（改善步行距离）⭐
- ⚠️ **HF 禁忌** ⭐（PDE3 抑制剂在 HF 增加死亡率）

### Dipyridamole
- 机制：PDE5 抑制 → cGMP ↑ → 抗血小板 + 血管扩张
- 适应：**Stroke 二级预防**（与 ASA 联合，Aggrenox）

## C.6 抗板药终极对比表 ⭐⭐⭐

| 药 | 机制 | 给药 | 拮抗 | 关键应用 | 注意 |
|---|---|---|---|---|---|
| **ASA** | COX-1 不可逆 | PO | 输血小板 | MI / PAD / Stroke 二级预防 | GI 出血 / Reye |
| **Clopidogrel** | P2Y12 不可逆 | PO | 输血小板 | DAPT (MI/PCI) / Stroke | CYP2C19 多态性 |
| **Prasugrel** | P2Y12 不可逆 | PO | 输血小板 | DAPT（强效）| Stroke/75+/<60kg 禁忌 |
| **Ticagrelor** | P2Y12 **可逆** | PO BID | 输血小板 | DAPT（首选 ACS）| **呼吸困难** |
| **GP IIb/IIIa** | GP IIb/IIIa 阻断 | IV | 时间 / 血小板 | PCI 术中 | 血小板减少 |
| **Cilostazol** | PDE3 | PO | - | PAD claudication | **HF 禁忌** |
| **Dipyridamole** | PDE5 | PO | - | Stroke 二级预防（+ASA）| 头痛 |

## C.7 DAPT 单药 vs 双抗决策

### 单药 vs DAPT 决策

```
单药抗血小板（ASA 或 Clopidogrel）：
- 稳定 PAD
- 稳定 CAD（无 PCI / MI 1 年以上）
- Stroke 二级预防（非心源性）

DAPT（ASA + P2Y12）：
- **ACS / MI 后** ≥ 1 年 ⭐
- **PCI 后**（药物涂层支架 12 月 / 裸支架 6 月）
- 高风险病人（CHD-PAD）
- Stroke 急性期短期（21 天 - CHANCE 试验）

三联（ASA + P2Y12 + DOAC）：
- AFib + 近期 PCI
- 高出血风险 → 短期使用（1-3 月）→ 转双联
```

### DAPT 时长指南

| 情境 | DAPT 时长 |
|---|---|
| **STEMI / NSTEMI 后** | **至少 12 月** ⭐ |
| **PCI + 药物涂层支架** | **6-12 月** |
| **PCI + 裸支架** | **1 月**（最短）|
| **稳定 CAD + Stent** | **6 月** |
| **Stroke 急性 24h 后** | **21 天 DAPT** → 单药（CHANCE / POINT）|
| **PAD revasc** | **1-6 月** → 单药 |
| **PAD + AFib** | **慎用 DAPT**（DOAC + ASA 短期）|

⭐ **DAPT 决策核心矛盾**：
- 时间长 → **防血栓更好**
- 时间长 → **出血风险更高**
- 必须个体化平衡

# D. 溶栓药（Fibrinolytics）

## D.1 溶栓药分类

```
溶栓药分类：
│
├── 1. **第二代 tPA 衍生物**（现代主流）⭐
│   ├── **Alteplase**（rtPA，重组组织型纤溶酶原激活物）
│   ├── **Tenecteplase**（TNK，长半衰期）
│   └── **Reteplase**
│
└── 2. 第一代（已少用）
    └── **Streptokinase**（链激酶）— 异源蛋白，过敏 + 抗原
```

## D.2 tPA 衍生物对比

| 药 | 给药 | 半衰期 | 用法 | 应用 |
|---|---|---|---|---|
| **Alteplase** (tPA) | IV bolus + 输注 | **5 分钟** | 多次给药 | **Stroke / STEMI / Massive PE** ⭐ |
| **Tenecteplase** (TNK) | **单次 IV bolus** ⭐ | **20 分钟** | 一次完成 | STEMI（首选）/ Stroke（新兴）|
| **Reteplase** | 2 次 IV bolus | 18 分钟 | 2 次给药 | STEMI |

⭐ Tenecteplase 因为**单次注射 + 长半衰期** → 救护车上给药很方便

## D.3 溶栓适应症 + 禁忌

### 临床适应症

| 病 | 用法 |
|---|---|
| **Ischemic Stroke** | **Alteplase < 4.5 h** ⭐ |
| **STEMI**（无 PCI 条件）| Alteplase / TNK < 12h |
| **Massive PE** | Alteplase ⭐ |
| **极严重 DVT**（Phlegmasia / 髂股 + 年轻）| **CDT**（导管定向） |
| **ALI** | CDT 偶用 |

### 绝对禁忌 ⭐ USMLE 必背

```
出血风险绝对禁忌：
- **既往任何颅内出血** ⭐
- 缺血性 stroke < 3 月
- **CNS 肿瘤**（恶性 / 血管畸形 / 动脉瘤）
- **活动性出血**（非月经）
- 近期严重头部创伤（3 月内）⭐
- 近期颅内 / 脊柱手术
- **可疑主动脉夹层** ⚠️
- 严重未控制 HTN（> 185/110）⭐ Stroke tPA 特别注意
```

### 相对禁忌

- 严重 HTN（> 180/110）
- 近期 GI 出血（< 1 月）
- 近期大手术（< 14 天）
- 妊娠
- 抗凝中 INR > 1.7

### 并发症
- **颅内出血** ⭐ 最严重（1-7% 看情境）
- 系统性出血
- 过敏（Streptokinase 多）

## D.4 出血并发症处理

```
溶栓后大出血处理：
1. 立刻停溶栓
2. **Cryoprecipitate**（含纤维蛋白原）⭐
3. **FFP** 补充凝血因子
4. **抗纤溶药**：**Aminocaproic acid / Tranexamic acid**
5. 输血小板（如血小板↓）
6. 颅内出血 → 神外评估
```

# E. 临床应用决策树 ⭐⭐⭐

## E.1 急性血栓 → 选什么药

```
急性血栓事件
        ↓
什么部位？
        ↓
┌──────────────────┬──────────────────┐
**静脉**             **动脉**
↓                  ↓
DVT/PE              位置？
↓                  ↓
**抗凝核心**         ┌──────────────────┐
- DOAC              心 → MI
- LMWH              脑 → Stroke
- (Warfarin)        肢 → ALI
↓                  ↓
± tPA              **各自不同**
（不稳定PE / Phlegmasia） ↓
                  详见下方分支
```

### 静脉血栓（PE / DVT）决策

```
DVT 急性：
- 稳定 + 一般 → DOAC
- 妊娠 → LMWH
- 癌症 → LMWH (首选) / DOAC
- 严重肾衰 → UFH
- 抗凝禁忌 → IVC filter

PE 急性：
- 稳定 + 低 PESI → DOAC（可门诊）
- 稳定 + Submassive → 抗凝 + 监测
- 不稳定 (Massive) → tPA + 抗凝

机械瓣 / APS → Warfarin
```

### 动脉血栓决策

```
MI 急性：
- ASA + P2Y12 + Heparin (核心 3 件套) ⭐
- + PCI < 90 min（首选）
- 无 PCI 条件 → tPA < 12h

Stroke 急性：
- **CT 排除出血** ⭐ 第一步
- < 4.5h → tPA
- < 24h LVO → 取栓
- **24h 内禁抗凝/抗板** ⭐
- 24h 后 → ASA / DOAC（看病因）

ALI 急性：
- **IV Heparin 立刻** ⭐
- + 紧急手术 / CDT
- 6h 窗口
```

## E.2 长期预防 → 选什么药

### AFib 长期抗凝
```
CHA₂DS₂-VASc 评分：
≥ 2 (男) / ≥ 3 (女) → 抗凝
        ↓
非瓣膜性 AFib：**DOAC 首选** ⭐
机械瓣膜性 AFib：**Warfarin** ⭐
风湿性 MS + AFib：**Warfarin**
```

### MI / PCI 后
```
ACS / PCI 急性：DAPT（ASA + P2Y12）
        ↓
1 年内：维持 DAPT
        ↓
1 年后：ASA 单药终身
+ Statin 终身
+ β-blocker（如 EF < 40%）
+ ACEi（如 EF < 40%）
```

### 稳定 PAD
```
稳定 PAD（无近期 revasc）：
- **单药**：ASA 81mg 或 Clopidogrel 75mg
- ❌ 不是 DAPT
        ↓
近期下肢 revasc / stent：
- DAPT 1-6 月 → 单药
        ↓
高 CV 风险 + 低出血：
- ASA + Rivaroxaban 2.5mg BID（COMPASS）
```

### VTE 长期
```
VTE 治疗后：
- 诱发因素（术后）→ 3 月
- 无诱发因素 → ≥ 3 月，考虑终身
- 复发 / 癌症 / 易栓症 → 终身
        ↓
首选：DOAC
妊娠 / 癌症 / APS / 机械瓣 → 看 B.8
```

### Stroke 二级预防
```
非心源性 Stroke：
- ASA / Clopidogrel 单药
- 高风险短期 DAPT 21 天（CHANCE）
        ↓
心源性 Stroke（AFib / 心室壁血栓）：
- DOAC（按时机启动）
        ↓
颈/椎动脉夹层：
- 抗板（首选）/ 短期抗凝
```

## E.3 特殊人群决策树

```
特殊人群 → 选药
        ↓
妊娠 → LMWH
        ↓
机械瓣 → Warfarin
        ↓
APS → Warfarin
        ↓
癌症 → LMWH（首选）/ DOAC
        ↓
严重肾衰（CrCl < 30）→ UFH / Warfarin
        ↓
HIT → Argatroban / Bivalirudin（首选）；Fondaparinux 非首选
        ↓
老年 + 出血风险 → 低剂量 Apixaban
        ↓
PCI 术中 → UFH / Bivalirudin
```

# F. 出血并发症 + 紧急逆转

## F.1 抗血栓药出血紧急处理流程

```
病人在抗血栓药 + 急性出血
        ↓
1. 立刻停用所有抗血栓药
        ↓
2. 评估出血严重度：
   - 轻度 → 局部 + 观察
   - 中度 → 输血 + 拮抗剂
   - 重度（颅内 / 大出血 / 血流动力学不稳）→ 紧急拮抗 ⭐
        ↓
3. 选拮抗剂（按当前药物）：
   - UFH → Protamine
   - LMWH → Protamine（部分）
   - Warfarin → PCC（紧急） / Vit K（慢）
   - Dabigatran → Idarucizumab
   - Xa-DOAC → Andexanet alfa
   - Antiplatelet → 输血小板 + 时间
   - 溶栓 → Cryoprecipitate + FFP + 抗纤溶药
        ↓
4. 输血支持（PRBC / FFP / 血小板）
        ↓
5. 找出血源 + 处理（内镜 / 手术 / 介入）
```

## F.2 抗凝药 vs 出血风险

| 药 | 颅内出血风险 | 总出血风险 |
|---|---|---|
| **Warfarin** | **最高** ⭐ | 高 |
| **DOAC** | **较低** ⭐⭐ | 中 |
| LMWH | 低 | 中 |
| UFH | 低（但变异大）| 看监测 |
| ASA | 低 | 中 |
| DAPT | 中 | 高 |
| 三联 | 高 | 最高 |

⭐ **DOAC 取代 Warfarin 的核心理由**：**颅内出血风险显著降低**

## F.3 HAS-BLED 评分（评估出血风险）

```
**H**TN (uncontrolled)
**A**bnormal renal/liver function
**S**troke history
**B**leeding history
**L**abile INR (Warfarin)
**E**lderly > 65
**D**rugs/alcohol (NSAIDs/抗板)
─────────────────────
≥ 3 = 高出血风险（不是抗凝禁忌，但需要更严密监测）
```

⭐ **临床现实**：HAS-BLED 高不阻止抗凝，是**提示需要加强监测**

# G. USMLE 经典题型集合

## G.1 急症识别题

> "65 岁男 + AFib，未抗凝。突发左侧偏瘫 2 小时，CT 无出血。下一步？"
> → **IV tPA** ⭐（不是抗凝！即使是 AFib 栓塞）

> "65 岁男 + AFib。突发左下肢剧痛 + 苍白 + 无脉 2 小时。下一步？"
> → **IV Heparin + 紧急手术**

> "55 岁男 + 胸痛 30 分钟 + ECG ST 抬高。下一步？"
> → **ASA 嚼服 + ECG 反复 + 准备 PCI < 90 min**

## G.1B "第一步"陷阱题 ⭐⭐⭐（高频混淆）

> [!warning] 看 stem 区分"诊断"还是"治疗"

**题 1：稳定 PE 高度可疑（经验性抗凝）**
> "65 岁女 + 大手术 5 天前 + 突发胸痛 + HR 120 + 单侧腿肿胀 + Wells PE = 7.5 + SBP 110 + 无禁忌"
> 
> ❌ A. 等 CTPA 结果再决定
> ✅ **B. 经验性抗凝 + CTPA** ⭐（ACCP 推荐）

**题 2：稳定 PE 低度可疑**
> "30 岁女 + OCP + 轻微胸痛 + Wells PE = 1.5"
> 
> ✅ **D-dimer**（如阴性可排除）
> ❌ 不直接抗凝

**题 3：不稳定 PE**
> "BP 70/40 + 严重呼吸困难 + 怀疑 PE"
> 
> ✅ **床旁 TTE 评估 RV** ⭐（不是直接 CTPA 或 tPA）
> 见右心负荷 → 经验性 tPA

**题 4：DVT 高 Wells**
> "Wells DVT = 3 + 单侧腿肿"
> 
> ✅ **Duplex US 确诊** ⭐（不是直接抗凝）

**题 5：抗凝禁忌的 PE**
> "70 岁男 + 高度可疑 PE + 3 天前 GI 大出血"
> 
> ❌ A. 经验性抗凝
> ✅ **B. 紧急 CTPA + IVC filter** ⭐

**题 6：ALI**
> "突发左下肢剧痛 + 6 P's"
> 
> ✅ **IV Heparin 立刻**（不等影像）⭐

## G.2 药物选择题

> "32 岁女 + 妊娠 28 周 + 新发 DVT。最佳抗凝？"
> → **LMWH（Enoxaparin）** ⭐（不能 Warfarin / DOAC）

> "68 岁男 + 机械二尖瓣 + AFib。最佳抗凝？"
> → **Warfarin INR 2.5-3.5** ⭐（不能 DOAC）

> "45 岁女 + SLE + Lupus anticoagulant + 第三次 DVT 在 Apixaban 中复发。改用什么？"
> → **Warfarin 或 LMWH**（APS DOAC 失败率高）

> "70 岁男 + AFib + CrCl 25 + 严重肾衰。最佳抗凝？"
> → **Warfarin** 或调整剂量 **Apixaban**

## G.3 出血紧急处理题

> "78 岁女 + Warfarin + 颅内出血 + INR 4.5。下一步？"
> → **4F-PCC + Vit K** ⭐（不是单独 FFP）

> "55 岁男 + Dabigatran + 严重消化道出血。最佳拮抗剂？"
> → **Idarucizumab** ⭐

> "60 岁男 + Apixaban + 颅内出血。拮抗剂？"
> → **Andexanet alfa** ⭐

## G.4 HIT 经典题

> "65 岁男 + 术后 7 天 + 血小板从 250k 降到 90k + 新发下肢血栓。诊断？治疗？"
> → **HIT Type II** → **立刻停 UFH/LMWH** + **改用 Argatroban / Bivalirudin**（USMLE 首选；Fondaparinux 理论可用但非首选）⭐

## G.5 DAPT 时长题

> "STEMI + PCI + 药物涂层支架。DAPT 最短时长？"

# H. Memory Hook 集合

## H.1 5 大急症一句话

> [!success] 
> | 急症 | 钥匙 |
> |---|---|
> | **PE 不稳定** | tPA |
> | **DVT** | DOAC |
> | **MI** | DAPT + PCI (90 min) |
> | **Stroke** | tPA (4.5h) + 24h 后 ASA |
> | **ALI** | Heparin + 手术 (6h) |

## H.2 特殊人群抗凝 5 件事

> [!tip] 
> 1. **机械瓣** = **只能 Warfarin** ⭐
> 2. **妊娠** = **只能 LMWH** ⭐
> 3. **HIT** = **不能用 Heparin/LMWH** → **Argatroban / Bivalirudin**（USMLE 首选 DTI）；Fondaparinux 理论可用但**非首选** ⭐
> 4. **严重肾衰** = **首选 UFH**（不靠肾排泄）⭐
> 5. **AFib / VTE 一般** = **DOAC 首选** ⭐

## H.3 拮抗剂"4 个 i" 助记

> [!tip] 
> - UFH → Prota**m**ine
> - Warfarin → P**C**C + Vit K
> - Dabigatran → **I**darucizumab
> - Xa-DOAC → A**n**dexanet alfa

## H.4 时间窗 "90-4.5-6-24"

> [!success] 
> - **90 min** → MI PCI 窗口
> - **4.5 h** → Stroke tPA 窗口
> - **6 h** → ALI 不可逆肌肉损伤
> - **24 h** → Stroke 取栓 / 24h 后 ASA

## H.5 "脑子最娇贵" 终极口诀

> [!success] 
> **"血栓 5 急症，看脏器决定治"**
> 
> ```
> 静脉血栓（PE, DVT）→ 抗凝（核心）
> 动脉血栓（ALI, MI, Stroke）→ 各不相同！
>    ├── ALI（肢体）→ Heparin + 手术
>    ├── MI（心）→ DAPT + PCI
>    └── Stroke（脑）→ tPA（不抗凝/不抗板）⭐ 最特殊
> ```
> 
> **"脑子最娇贵，急性不抗凝不抗板"**

## H.6 抗凝药"5 大类"分类

> [!tip] 
> 1. **Heparin 类**（UFH / LMWH / Fondaparinux）— 间接 / IV+SC
> 2. **Warfarin** — VKA，需 INR
> 3. **DOAC** — 直接 IIa / Xa，PO
> 4. **直接凝血酶抑制剂**（IV：Argatroban / Bivalirudin；PO：Dabigatran）
> 5. 其他（Fondaparinux 独立）

# J. USMLE 高频陷阱补充 ⭐⭐⭐

> [!info] 来源
> 2026-05-14 复盘错题学习补充：HIT 治疗双相陷阱 + PCI 术中选药 + 肾衰代谢分级 + 词根速记。
> 与 B 节互补，不替代 B；本节专治"易混 + 应用"陷阱。

## J.1 陷阱 1：HIT 患者的抗凝选择

> [!danger] HIT 四大铁律
> 1. **禁用所有 Heparin**（UFH / LMWH / Fondaparinux 也避免——尽管理论上 Fondaparinux 交叉反应低）
> 2. **首选 DTI**：**Argatroban**（肝代谢，肾衰也能用）/ **Bivalirudin**
> 3. **不能立刻用 Warfarin** ⭐ — 早期会**加重血栓 + 皮肤坏死风险**
>    - 机制：Warfarin 首先抑制 Protein C / S（短半衰期）→ 短暂高凝状态
> 4. **桥接转换**：等血小板恢复 > 150 后，重叠 DTI 与 Warfarin / DOAC ≥ 5 天 + INR 达标

### HIT 选药时间线

| 时机 | 药物 | 关键点 |
|---|---|---|
| **急性期**（血小板 < 150） | **Argatroban**（首选）or **Bivalirudin** | DTI，IV 给药 |
| **桥接转换**（血小板 ≥ 150） | 启动 Warfarin（重叠 ≥ 5 天 + INR 达标）or DOAC | **不可单用 Warfarin 早启动** |
| 长期 | Warfarin / DOAC | 终身病史标记，避免 Heparin |

> [!tip] Memory Hook
> "**A**rgatroban → **A**cute HIT" — A 开头 = 急性期首选

---

## J.2 陷阱 2：PCI 术中抗凝选择

| 方案 | 组成 | 适用 |
|---|---|---|
| **传统** | Heparin + GP IIb/IIIa 抑制剂（Abciximab / Eptifibatide / Tirofiban） | 标准 PCI |
| **新方案** ⭐ | **Bivalirudin**（DTI，IV） | **出血风险更低**；HIT 史 / 高出血风险患者首选 |

> [!tip] Memory Hook
> "**Bi**valirudin → **Bi** = 双向" — 同时结合凝血酶的**催化位点 + 阴离子结合位点** → 精准抗凝 + 出血风险低 = PCI 术中专用

---

## J.3 陷阱 3：肾衰患者抗凝选药（代谢分级）

| 药物 | 代谢途径 | 肾衰能用？ |
|---|---|---|
| **Argatroban** | **肝**（100%） | ✅ **肾衰最佳**（包括 ESRD / 透析） |
| **Bivalirudin** | 80% 蛋白酶 + 20% 肾 | ⚠️ CrCl < 30 减量 |
| **Dabigatran** | **80% 肾** | ❌ 慎用，**CrCl < 30 禁用** |
| **Apixaban** | 主要肝（25% 肾排泄） | ✅ **肾衰最佳 DOAC**（CrCl < 25 低剂量 2.5 mg BID） |
| **Rivaroxaban / Edoxaban** | 35-50% 肾 | ⚠️ CrCl < 15 禁用 |
| **Warfarin** | 肝（CYP2C9） | ✅ 任何 CrCl 都安全 |
| **UFH** | 网状内皮系统（非肾） | ✅ 紧急可用，但需 PTT 监测 |
| **LMWH（Enoxaparin）** | **肾** | ❌ CrCl < 30 减半剂量 / 改 UFH |

> [!warning] USMLE 关键题眼
> - "老人 + CrCl 25 + AFib 需长期抗凝" → **Warfarin 或低剂量 Apixaban**（不是 UFH，不是 Dabigatran）
> - "ESRD 透析 + 新发 HIT" → **Argatroban**（肝代谢，肾衰也行）
> - "妊娠 + DVT" → **LMWH**（肾正常时；肾衰时改 UFH）

## J.4 抗凝药全家族分类树（增强版）

> [!info] 与 B.1 互补
> B.1 按机制分；本节按**给药途径 + DOAC 判定**重排，专治"DOAC 范围"陷阱

```
抗凝药
├── 维生素 K 拮抗剂
│   └── Warfarin (PO)
│
├── 间接（通过 AT III）
│   ├── Heparin / UFH (IV/SC) → 抑制 IIa + Xa
│   ├── LMWH (SC) → 主要抑制 Xa
│   │   └── Enoxaparin, Dalteparin
│   └── Fondaparinux (SC) → 纯 Xa
│
├── 直接凝血酶（IIa）抑制剂 = DTI
│   ├── Dabigatran (PO) ⭐ DOAC
│   ├── Bivalirudin (IV) ❌ 非 DOAC
│   ├── Argatroban (IV) ❌ 非 DOAC
│   └── Desirudin (SC) ❌ 非 DOAC
│
└── 直接 Xa 抑制剂
    ├── Rivaroxaban (PO) ⭐ DOAC
    ├── Apixaban (PO) ⭐ DOAC
    ├── Edoxaban (PO) ⭐ DOAC
    └── Betrixaban (PO) ⭐ DOAC
```

> [!danger] DOAC 判定铁律
> **DOAC 的 O = Oral** → 看到 **IV / SC** 的"直接"抑制剂，再厉害也**不是 DOAC**
> - DTI 家族里：只有 **Dabigatran 是 DOAC**（PO）
> - **Bivalirudin / Argatroban / Desirudin** 都是 DTI 但**不是 DOAC**（IV / SC）
> - 4 个 Xa 抑制剂（Riva / Api / Edo / Betr）全是 DOAC（都 PO）

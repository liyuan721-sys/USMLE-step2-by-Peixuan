# 烧伤管理速查（Burn Management）

> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q19256]]（侵袭性烧伤创面感染 → 抗假单胞 + 万古）。
> 用法：烧伤 vignette 按 **气道 → 液体复苏 → 创面 → 感染 → 转诊** 五步走。
> 液体复苏公式细节见 [[完整笔记/Peixuan分科笔记/外科]]（Parkland / 4-2-1 / 前8后16）；本笔记补全谱。
> 版权：基于公开教科书（First Aid / ABA 公开指南摘要）重新整理，未复制 UWorld 题干。

## 一、五步框架（ABCDE 之后）

```
严重烧伤
├─ ① 气道 Airway ── 吸入伤征象 → 早期插管（水肿会进展，越早越好）
│        └─ 闭室火灾 → 查 CO（一氧化碳）+ 氰化物（cyanide）中毒
├─ ② 液体 Fluid ── Parkland（LR 乳酸林格）；只算部分+全层；滴定尿量
├─ ③ 创面 Wound ── 深度分级 + 清创 + 外用抗菌（磺胺嘧啶银）；环周全层 → 焦痂切开
├─ ④ 感染 Infection ── 定植≠感染；侵袭性 BWI → 抗假单胞β-内酰胺 + 万古
└─ ⑤ 转诊 + 支持 ── 烧伤中心指征 / 破伤风 / 营养 / 应激性溃疡预防
```

## 二、TBSA 估算 + 烧伤深度

### 九分法（Rule of 9s，成人）

> [!tip] 只算**部分层 + 全层**烧伤（**浅表/一度不计入** TBSA 与补液）
> | 部位 | 成人 % |
> |---|---|
> | 头颈 | 9 |
> | 每侧上肢 | 9 |
> | 每侧下肢 | 18 |
> | 躯干前 / 后 | 各 18 |
> | 会阴 | 1 |
>
> **患者手掌（含指）≈ 1%**（小面积/散在烧伤估算）。**儿童头大腿小** → 用 **Lund-Browder 图** 更准（儿童头 ~18%、每腿 ~14%）。

### 深度分级

| 深度 | 累及 | 表现 | 痛觉 | 计入 TBSA |
|---|---|---|---|---|
| 浅表（一度 superficial） | 表皮 | 红、干、**无水疱**（晒伤） | 痛 | **否** |
| 浅部分层（浅二度） | 真皮浅层 | **水疱、湿润、发红、压之褪色** | **剧痛** | 是 |
| 深部分层（深二度） | 真皮深层 | 苍白/花斑、压之褪色差 | 减弱 | 是 |
| 全层（三度 full thickness） | 全真皮 | **皮革样、白/焦、干、不褪色** | **无痛**（神经毁） | 是 |
| 四度 | 至肌/骨 | 焦化 | 无 | 是 |

## 三、液体复苏（要点，公式见外科主笔记）

> [!info] Parkland 速记
> - **公式**：4 mL × 体重(kg) × TBSA(%) 的 **LR（乳酸林格液 lactated Ringer's）**，**前 8 h 给一半、后 16 h 给另一半**（从**受伤时刻**计时，不是入院时刻）。
> - **复苏成功的金标准 = 尿量（urine output）**：成人 **0.5 mL/kg/h（~30–50 mL/h）**，儿童 **1 mL/kg/h** → **按尿量滴定**，公式只是起始估算。
> - **指征**：成人 >20% TBSA / 儿童 >10–15% TBSA 才正式复苏。

> [!tip] 为什么 LR 优于 NS（机制）⭐
> - **LR 含乳酸 lactate → 肝代谢成 HCO₃⁻** → **缓冲烧伤休克的代谢性酸中毒**（而非加重）。
> - **大量 NS（生理盐水）→ 高氯 Cl⁻ → 高氯性代谢性酸中毒**（NAGMA，稀释性 + Cl⁻ 顶替 HCO₃⁻）。
> - **PRBC（浓缩红细胞）不能与 LR 同管输**：LR 的 **钙 Ca²⁺ 会与血制品的枸橼酸 citrate 螯合 → 促凝/凝块**（PRBC 用 NS 冲）。

> [!info] 为什么烧伤要这么大量补液（第三间隙 + 毛细血管漏）⭐
> 烧伤释放炎症介质 → **毛细血管通透性↑（capillary leak）→ 血浆蛋白 + 水一起漏入组织间隙（第三间隙 third-spacing）**：**"外面肿、里面干"**（组织水肿但血管内容量不足/低血容量休克）。→ 这就是 Parkland 需大量晶体的原因。同机制见胰腺炎/脓毒症/肠梗阻/腹膜炎。

> [!tip] 儿童液体复苏 = Parkland（烧伤丢失）+ 4-2-1（基础维持，叠加）
> - **Parkland** 算烧伤额外丢失（同公式，只算部分层+全层 TBSA）。
> - **4-2-1 基础维持（每小时）**：前 **10 kg ×4 mL/kg** + 次 **10 kg ×2 mL/kg** + **>20 kg 部分 ×1 mL/kg**。
>   - 例：25 kg 儿童维持 = 40 + 20 + 5 = **65 mL/h**。
> - 儿童复苏液常用**含糖**维持（防低血糖，糖原储备少）。

## 四、气道 / 吸入伤 + 燃烧产物中毒 ⭐

> [!danger] 吸入伤（inhalation injury）→ 早期插管
> **征象**：面部烧伤、**鼻毛烧焦**、**碳质痰（carbonaceous sputum）**、声嘶/喘鸣、闭室火灾、口咽部烟灰。
> **气道水肿进行性加重** → **趁早插管**（等到喘鸣/缺氧再插就困难/危险）。

| 毒物 | 来源 | 线索 | 解毒 |
|---|---|---|---|
| **一氧化碳 CO（carbon monoxide）** | 任何燃烧 | 头痛/意识改变；**SpO₂ 假性正常**（脉氧仪测不出 COHb）；查**碳氧血红蛋白（COHb）**；"樱桃红"多为晚期/尸检 | **100% 纯氧**；重症/神经症状/孕妇 → **高压氧（hyperbaric O₂）** |
| **氰化物 cyanide** | 合成材料（塑料/泡沫）燃烧 | **乳酸性酸中毒 + 静脉血氧↑（SvO₂↑，组织用不了氧）** | **羟钴胺（hydroxocobalamin）** ± 硫代硫酸钠（sodium thiosulfate） |

## 五、创面感染（BWI）⭐（核心错题来源）

> [!warning] 定植 ≠ 感染
> 创面伤后快速被菌**定植（colonization）不损伤愈合**；**感染（>10⁵ 菌/g 组织活检）才损伤**。
> **临床疑诊** = **创面外观改变 / 存活植皮丢失 / 周围未烧伤组织出现感染征（红斑）**。

| 维度 | 侵袭性 invasive | 非侵袭性 noninvasive |
|---|---|---|
| 全身表现 | **有**（意识模糊/心动过速/发热） | 无 / 极少 |
| 侵入未烧伤组织 | 有 | 无 |
| 两大病原 | **铜绿假单胞菌（Pseudomonas aeruginosa）+ 金黄色葡萄球菌（含 MRSA）** | 同 |
| 经验抗生素 | **哌拉西林-他唑巴坦（piperacillin-tazobactam）或 碳青霉烯（美罗培南 meropenem）** + **万古霉素（vancomycin）** | 窄谱：头孢唑林（cefazolin）/ 克林霉素（clindamycin） |
| 加抗真菌 | **活检见真菌侵袭** → 加两性霉素 B（amphotericin B） | — |

**外用抗菌剂**：
- **磺胺嘧啶银（silver sulfadiazine）**：常用、不痛，**不穿透焦痂**；**避面部 + 新生儿/G6PD（核黄疸/溶血风险）**。
- **醋酸磺胺米隆（mafenide acetate）**：**穿透焦痂**（适合厚焦痂/软骨如耳），但**痛 + 碳酸酐酶抑制 → 代谢性酸中毒**。

## 六、转诊 + 支持治疗

> [!info] 转烧伤中心指征（ABA，记大类）
> - 部分层 **>10% TBSA**；**任何全层**烧伤
> - **特殊部位**：面/手/足/生殖器/会阴/大关节
> - **电烧伤（含雷击）、化学烧伤、吸入伤**
> - 合并创伤 / 共病重 / 需特殊社会心理支持（含可疑虐待）

- **环周全层烧伤（四肢/胸）→ 焦痂切开术（escharotomy）**：焦痂无弹性 → 肢体筋膜室综合征 / 胸廓限制通气 → 切开减张。
- **破伤风（tetanus）预防**：按免疫史给 Tdap ± TIG。
- **应激性溃疡预防（Curling ulcer，柯林溃疡）**：大面积烧伤 → PPI/H₂ 阻滞剂。
- **营养**：烧伤高代谢 → 早期高热卡高蛋白肠内营养。
- **电烧伤（含高压电击）⭐**：**表面轻、深部重**（骨阻抗最高 → 产热最多 → 骨周骨骼肌坏死 > 可见体表损伤）。骨骼肌坏死 → **三件套：① 急性筋膜室综合征（肌肿胀，需筋膜切开）② 横纹肌溶解 rhabdomyolysis（肌红蛋白尿）③ 血红素色素性 AKI**。处理 = **积极 IV 补液 + 监测 CK/K⁺/尿色/心律**；因横纹肌溶解高钾风险 → **避免含钾液（如 LR 慎用，按 K⁺ 调整）**。**"最大风险"看电流路径 + 现有体征**：手→足路径不过脑 + 清醒 → 排癫痫/脑疝；ECG 正常 → 排心律失常（电击最常见心脏并发症仍是心律失常/骤停，非心包填塞）。详见 [[mistakes/uworld-mistakes_2026-06#^Q19263]]。

> [!danger] 烧伤/电烧伤的麻醉：禁琥珀胆碱 succinylcholine ⭐（高频考点）
> - **机制**：烧伤/挤压/去神经损伤后 **肌膜乙酰胆碱受体（ACh-R）上调（upregulation）+ 横纹肌溶解细胞裂解** → 琥珀胆碱（去极化肌松剂）触发 **大量 K⁺ 外漏 → 致命性高钾血症 / 心搏骤停**。
> - **时间窗**：**伤后约 24–48 h 起风险显著上升**，持续数月至创面愈合（受体上调需时间，故超急性期相对安全，但临床多直接避用）。
> - **替代**：用**非去极化肌松剂（如罗库溴铵 rocuronium）**。
> - 同理适用于：大面积创伤、长期制动/去神经、横纹肌溶解 → 都怕琥珀胆碱诱发高钾（见 [[完整笔记/专题笔记/肾脏/肾脏_高钾血症完整体系]]）。

## 七、高频陷阱

> [!warning] 易错点黑名单
> - **浅表（一度）烧伤不计入 TBSA / 不算补液**（只算部分层 + 全层）。
> - **Parkland 从受伤时刻计时**，不是入院时刻；**尿量才是复苏终点**，别死守公式数字。
> - **CO 中毒 SpO₂ 假性正常** → 必须查 COHb；治疗是 100% O₂ / 高压氧（不是单纯吸 21% 空气）。
> - **吸入伤要早插管**（等水肿封气道就晚了）；面部烧伤 + 碳质痰 + 鼻毛烧焦 = 红旗。
> - **烧伤创面：定植 ≠ 感染**；看到"烧伤发热"先想**细菌（铜绿/金葡）**，**真菌仅活检见才加两性霉素 B**。
> - **首选 LR 不用 NS**（NS → 高氯酸中毒）；**LR 不与 PRBC 同管**（钙螯合枸橼酸→凝块）。
> - **烧伤/电烧伤麻醉禁琥珀胆碱**（伤后 >24–48 h ACh-R 上调 → 致命高钾）→ 用罗库溴铵。
> - **复苏不足/呕吐丢液时容量性低钠 + 代碱**：尿氯 **UCl 低（<20）= 容量反应性**（盐水反应性代碱）→ 补 LR/NS 即纠；别误当 SIADH（SIADH 容量正常、UNa 高）。

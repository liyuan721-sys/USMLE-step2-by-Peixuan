# 术后肺部并发症与肺不张（Postoperative pulmonary complications & atelectasis）

> [!info] 用途
> 术后第 2–5 天"新发气短/低氧"高频 = 肺不张。本卡锁：① 机制 + ABG ② 预防（含戒烟时间窗陷阱）③ **影像纵隔移位方向**鉴别（反复考）④ 处理阶梯。
> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q4931]]（ABG=呼碱）+ [[mistakes/uworld-mistakes_2026-06#^Q4932]]（预防）+ [[mistakes/uworld-mistakes_2026-06#^Q3734]]（黏液栓肺不张影像）。

---

## 一、肺不张 atelectasis —— 机制 + 表现

```
两条路致肺泡塌陷
① 低潮气量：术后疼痛限制胸廓扩张 + 麻醉残留 + 卧位 → 基底肺泡塌陷（最常见，术后 2–5 天高峰）
② 气道阻塞：黏液栓 / 肿瘤 / 异物 → 远端气体吸收入血 → 肺泡塌陷
```

- 表现：**呼吸困难 + 心动过速 + 低氧**；患区**叩浊 + 呼吸音消失**。
- **ABG = 低氧（↓PaO₂）→ 过度通气 → 原发呼吸性碱中毒（↑pH + ↓PaCO₂）**（[[mistakes/uworld-mistakes_2026-06#^Q4931]]）。
  - ❌ 别因"术后用了吗啡"就选呼吸性酸中毒——**呼吸频率↑是过度通气，不是低通气**（麻醉药过量低通气才呼酸）。

---

## 二、影像：不透光 + 纵隔移位方向（鉴别钥匙）⭐（Q3734）

| 病 | 患侧影像 | 纵隔 | 机制 |
|---|---|---|---|
| **肺不张 / 全肺切除** | 不透光（白）| **移向患侧** | 容积↓ 牵拉 |
| **大量胸腔积液** | 不透光（白）| **推离患侧** | 占位 |
| **张力气胸** | **透亮（黑）** | **推离患侧** | 占位 |
| 大叶肺炎 | 不透光（白）| 不移位 | 实变无容积变化 |

> [!danger] 一秒判读"白肺 + 纵隔移位"
> **移向患侧 = 容积减少（肺不张 / 肺切除）；推离患侧 = 占位（积液 / 张力气胸）。张力气胸是"黑"不是"白"。** 肺不张还伴**患侧肋间隙变窄**（容积↓）。

---

## 三、预防（围术期）⭐（Q4932）

| 措施 | 要点 |
|---|---|
| **深呼吸训练 + 激励性肺量计 incentive spirometry** | **最有效**，促高潮气量、保持肺泡张开 |
| **充分镇痛** | 减少疼痛限制胸廓扩张 |
| **戒烟** | **必须术前 >4–8 周**才有获益（气道炎症/黏液需数周改善）|
| 控制基础肺病 | 哮喘/慢性阻塞性肺病 COPD 术前优化 |

> [!warning] 戒烟时间窗陷阱
> **术前 1 周戒烟 = 无预防获益**（甚至短期分泌物增多）。要 **>4–8 周**。高危因素：年龄>50、吸烟、基础心肺病、急诊/手术>3h。

> [!tip] 不预防肺不张的"诱饵"
> 广谱抗生素（无感染证据不用）/ 术前激素+沙丁胺醇（只优化已知哮喘 COPD）/ 低分子肝素（防静脉血栓栓塞 VTE 非肺不张）——都不是肺不张预防答案。

---

## 四、处理阶梯

```
轻度       ：激励性肺量计 + 深呼吸 + 镇痛 + 早期活动
分泌物多   ：胸部物理治疗 + 吸痰
仍塌陷     ：CPAP（持续气道正压）开肺
大容量/黏液栓：支气管镜 bronchoscopy 清除栓子（[[mistakes/uworld-mistakes_2026-06#^Q3734]]）
```

---

## 五、Memory Hook

> [!success] 一句话
> **术后 2–5 天新发低氧 = 肺不张 → 低氧 + 呼碱（别选呼酸）。预防 = 激励性肺量计 + 镇痛；戒烟要 >4–8 周。影像：白肺 + 纵隔【移向】患侧 + 肋间窄；大容量/黏液栓 → 支气管镜。**

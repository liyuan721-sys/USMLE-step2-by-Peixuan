# 腹部外伤速查（Blunt Abdominal Trauma, BAT — 影像决策 + 器官定位 + 隐匿伤陷阱）

> [!info] 用途
> 钝性腹部外伤 (Blunt Abdominal Trauma, BAT) stem 高频考"**下一步**"。本卡锁：① 按血流动力学稳定性走的**影像决策树**（FAST / CT / 剖腹）② 器官定位（脾/肝/胰/空腔）+ 腹膜内 vs 腹膜后 ③ 反复考的**隐匿/延迟伤陷阱**（外伤性胰腺炎、腹膜后器官早期体征阴性、延迟脾破裂）。
> 派生自 [[NBME11_错题本#^Q008]]（外伤性胰腺炎，本卡建卡触发）。

---

## 一、BAT 影像决策树（按血流动力学稳定性分流）⭐

> [!danger] BAT 铁律：先看"稳不稳"，再决定去哪
> **不稳定患者绝不送 CT 室。** 补液无效的不稳定 + 腹胀 → **直接进手术室剖腹 (laparotomy)**。

| 患者状态 | 下一步 |
|---|---|
| **稳定** | **增强 CT (CT with IV contrast)** — 看清实质器官 + 分级 |
| **不稳定 + FAST 阳性** | **立即剖腹 laparotomy** |
| **不稳定 + FAST 阴性/可疑** | 诊断性腹腔灌洗 (DPL, diagnostic peritoneal lavage) |
| 稳定 + 血尿 (hematuria) | 腹盆 CT（评估肾损伤，见 [[完整笔记/专题笔记/外科/外科_GU创伤处理体系]]）|

```
                  钝性腹部外伤 (BAT)
                        |
              血流动力学稳定吗？
              /                  \
           稳定                  不稳定
            |                       |
       增强 CT               床旁 FAST
   (实质器官分级)            /        \
                        阳性          阴性/可疑
                         |              |
                    立即剖腹          DPL
                   (laparotomy)
```

> [!tip] 影像各自的能与不能
> - **FAST (focused assessment with sonography in trauma)**：床旁、快、找游离液体（腹腔出血）；**只看腹膜内**（肝/脾/膀胱周/Morrison 窝），**看不清腹膜后**（肾/胰/主动脉/十二指肠）；operator-dependent，**不能定位具体器官损伤**。
> - **增强 CT**：细节最好、可对实质器官损伤分级；前提是**患者稳定**到能离开复苏区。
> - **DPL**：现在少用，留给"不稳定 + FAST 不可靠"的兜底。

## 二、器官定位 + 腹膜内 vs 腹膜后

| 器官 | 典型机制 | 诊断关键 | 临床特征 |
|---|---|---|---|
| **脾 (spleen)** | 左季肋部受撞 | CT 可确诊 | **BAT 最常受损器官**；可伴 **Kehr 征（左肩牵涉痛）** |
| **肝 (liver)** | 右季肋部受撞 | CT 可确诊 | 次于脾；常引起血腹 (hemoperitoneum) |
| **胰 (pancreas)** | 上腹中心挤压（车把伤/脊柱挤压） | **延迟表现，CT 早期易漏** | 胰周积液、淀粉酶↑ → 见 §四 |
| **十二指肠 (duodenum)** | 上腹中心挤压（车把/脊柱/虐童） | 腹膜后，**FAST 阴**，CT / UGI 造影 | 壁内血肿 → **延迟胆汁性呕吐**（高位梗阻）→ 见 §五 |
| **空腔脏器 (hollow viscus)** | 突然腔内压升高 | 膈下游离气体 | 肠破裂/肠系膜撕裂 → 腹膜炎 |

> [!warning] 腹膜内 (intraperitoneal) vs 腹膜后 (retroperitoneal) — 决定 FAST 灵不灵
> - **腹膜内**：肝、脾、胃、空回肠 → 活动度大，损伤后血液易入腹腔 → **FAST 能抓**。
> - **腹膜后**：**肾、胰腺（除尾部）、十二指肠降部** → 位置固定、**早期体征可不明显**（血/消化液被限制在腹膜后，不立即刺激壁层腹膜）→ **FAST 早期阴性，需 CT 确诊**。

> [!tip] 肋骨骨折定位 → 关联器官
> **下位肋骨 (9–12) 骨折** → 警惕下方**腹腔实质器官**（脾/肝/肾）损伤 → 腹部 CT。详见 [[完整笔记/专题笔记/外科/外科_胸部创伤速查]] 肋骨定位表 + [[mistakes/uworld-mistakes_2026-06#^Q16455]]。

## 三、隐匿 / 延迟伤陷阱（NBME 最爱）⭐

> [!danger] 腹膜后器官伤 = 早期"假阴性"
> 钝性挤压伤后**初期 CT 阴性、体征轻**，但数日后出现持续腹痛、呕吐、发热 + 大量胰周积液 → 高度怀疑**未诊断出的胰腺（胰管）损伤**。腹膜后器官（十二指肠、胰腺）早期可无板状腹，别被"首查阴性"骗了。

> [!warning] 脾破裂的两个陷阱
> - **延迟破裂 (delayed rupture)**：被膜下血肿初始 CT 正常，数天后突然破裂 → 休克。
> - **非创伤性脾破裂 (atraumatic splenic rupture)**：不一定要有外伤。高危：血液肿瘤（慢性淋巴细胞白血病 CLL / 慢性粒细胞白血病 CML / 淋巴瘤 lymphoma）、**传染性单核细胞增多症 (mononucleosis 传单)**、系统性红斑狼疮 (SLE, systemic lupus erythematosus) / 胰腺炎、抗凝药 (anticoagulants)。三联：腹痛（放射左肩 = **Kehr 征**）+ 休克体征 + 急性贫血。

> [!tip] Kehr 征 (Kehr sign)
> 膈肌受血液/积液刺激 → **左肩牵涉痛**（C3–5 膈神经）。脾破裂多见，肝破裂也可出现。

## 四、外伤性胰腺炎（Traumatic pancreatitis）— 源 Q008 深挖 ⭐

> [!info] 机制
> **钝性上腹外伤**（脊柱把胰腺挤压 / 车把伤 / 儿童自行车把手伤 / 被人跳压上腹）→ 胰腺挫裂 → **外伤性胰腺炎**。儿童 BAT 后胰腺炎尤其经典。
> 📚 急性胰腺炎诊断/病因/酶判读的知识主源见 [[完整笔记/专题笔记/GI/GI_HighYield速查总览]]（§10.1 急性胰腺炎，已补外伤病因 + 酶判读陷阱）。

**急性胰腺炎诊断（3 取 2）**：
1. 典型上腹痛放射至背
2. **脂肪酶 / 淀粉酶 ≥ 3× ULN (upper limit of normal, 正常上限)**（脂肪酶更特异、持续更久）
3. 影像符合（CT/US）

**病因谱**：胆石 / 酒精 / **外伤（钝性）** / 高甘油三酯 / 高钙 / 药物 / post-ERCP（内镜逆行胰胆管造影 endoscopic retrograde cholangiopancreatography 术后）。

> [!danger] 酶判读陷阱（[[NBME11_错题本#^Q008]] 错点）⚠️
> **"脂肪酶在正常范围 + 只有淀粉酶轻升" ≠ 排除胰腺炎。** 早期或外伤性胰腺炎酶可不典型；NBME 仍据**机制（明确钝伤）+ 临床（上腹痛呕吐）**判外伤性胰腺炎。教训：**别因脂肪酶不高就排除胰腺。**

> [!info] 反过来:淀粉酶↑ 也不特异 — 别只凭它就钉死胰腺
> 高淀粉酶血症 (hyperamylasemia) 来源很杂:唾液腺(腮腺炎 mumps)、**肠缺血/梗阻/穿孔**(钝伤肠挫伤也可轻升)、异位妊娠 (ectopic pregnancy 宫外孕)、糖尿病酮症酸中毒 (DKA, diabetic ketoacidosis)、肾衰、巨淀粉酶血症 (macroamylasemia)。∴ 单纯淀粉酶↑ 是**"嫌疑线索"不是"定罪证据"**。完整来源表(主源)见 [[完整笔记/专题笔记/GI/GI_HighYield速查总览]] §10.1。

**钝性上腹伤后横向鉴别**：

| 诊断 | 钩子 | 关键 |
|---|---|---|
| **外伤性胰腺炎** (traumatic pancreatitis) | 上腹挤压 + 上腹痛 + **淀粉酶↑** | 支持治疗（补液/禁食/镇痛）；CT 评估分级 |
| **十二指肠血肿** (duodenal hematoma) | 上腹挤压 + **延迟胆汁性呕吐**（高位梗阻） | CT / UGI 造影（"coiled spring"）；多保守 NG + TPN → 见 §五 |
| 肠挫伤 (intestinal contusion) | 钝伤后腹痛 | 早期酶/影像常**不特异**，可隐匿 |
| 肝/脾裂伤 (laceration) | 钝伤 + **低血压/腹腔积血** | FAST/CT，可需手术 |

**下一步**：血流动力学稳定 + 疑外伤性胰腺炎 → **腹部 CT**（评估胰腺损伤/分级 + 排其他脏器伤）+ 支持治疗。

> [!tip] 钝性上腹伤如何把"淀粉酶↑"翻译成定位反射（排除逻辑）
> 淀粉酶↑ 只告诉你"上腹有事"，定位靠**逐个排掉别人的专属指纹**：
> - **幅度 >3× ULN** → 更指胰腺（肠源/梗阻多只轻升）
> - **无腹膜炎征 (rebound/guarding) / 无膈下游离气 (free air)** → 排**肠穿孔**
> - **无延迟胆汁性呕吐** → 排**十二指肠血肿**（§五）
> - **不是 LUQ + Kehr 征 + 低血压** → 排**脾破裂**（§二/§三）
> - 剩下「上腹挤压 + 上腹痛放射背 + 淀粉酶↑ + ileus（肠鸣音↓）」= **外伤性胰腺炎**。

## 五、十二指肠血肿（Duodenal hematoma）— 胰腺的"孪生伤"⭐

> [!info] 机制（和外伤性胰腺炎同源）
> **钝性上腹中央挤压**（自行车车把伤 handlebar injury / 脊柱把十二指肠压扁 / **被踢被打**）→ 十二指肠壁内出血形成**壁内血肿 (intramural hematoma)**（黏膜下层）→ 撑窄肠腔 → **高位肠梗阻 / 胃出口梗阻 (gastric outlet obstruction)**。十二指肠是**腹膜后 (retroperitoneal)** 固定器官，和胰腺一样常被同一下钝伤一起伤，**两者可并存**。

> [!danger] USMLE 红旗：幼儿 + 十二指肠血肿（或胰腺伤）+ 病史对不上 = 虐童
> 腹中线钝伤在能自主活动的儿童**意外少见**；幼儿出现十二指肠血肿/胰腺损伤而**外伤史不符、就诊延迟** → 高度怀疑 **非意外创伤 / 虐童 (non-accidental trauma / child abuse)**。这是 NBME 反复埋的伦理 + 创伤交叉点。

**临床（关键鉴别点 — 和胰腺炎区分就靠这个）**：

| 特征 | 十二指肠血肿 | 外伤性胰腺炎 |
|---|---|---|
| 主症 | **延迟（数小时~数天）胆汁性呕吐 (bilious, non-bloody vomiting)** = 高位梗阻 | 持续上腹痛放射至背 |
| 淀粉酶 | 多正常或轻升 | 可 ≥3× ULN（也可不典型）|
| 本质 | **机械性梗阻** | 炎症 |

> [!tip] 诊断
> - **腹部增强 CT (computed tomography, 计算机断层)**：首选 — 看壁内血肿 + 肠壁增厚，同时排胰腺伤 / 穿孔。
> - **上消化道造影 (Upper GI series, UGI 上消化道钡剂/水溶造影)**："**coiled spring**（螺旋弹簧）/ **stacked coins**（叠硬币）/ picket-fence" 征 = 血肿把黏膜皱襞撑开的经典像。
> - 必须排除**十二指肠穿孔 (duodenal perforation)**：腹膜后游离气体 / 造影剂外渗 → 转手术。

> [!success] 处理：先保守，多数自行吸收
> - **一线 = 保守**：禁食 (NPO, nil per os) + **鼻胃管减压 (nasogastric, NG tube decompression)** + **全肠外营养 (total parenteral nutrition, TPN)**；壁内血肿多在 **1–2 周自行吸收**、梗阻缓解。
> - **手术指征**：保守 ~2 周不缓解、出现**穿孔 / 腹膜炎 (peritonitis)**、或合并需修补的脏器伤 → 手术（血肿清除 ± 引流）。
> - 对比记忆：十二指肠血肿**保守为主**；外伤性胰腺炎也以**支持治疗**为主（仅胰管断裂等才手术）。

## 六、穿透伤 + 复苏反应

> [!tip] 穿透伤路径（钝 / 枪 / 刺）
> - **钝伤**：轻（仅皮下）→ ER 观察复查；较重但未崩 → CT/FAST；不稳定不应液 → 剖腹。
> - **枪伤 (gunshot)**：稳定 → X 线定位；不稳定 → 剖腹（穿透腹膜多需探查）。
> - **刺伤 (stab)**：稳定 → CT；不稳定 → 剖腹。**决定开腹的是有没有穿透腹膜 + 腹内损伤**，不是伤口深浅。

> [!info] 创伤复苏后反应评估（液体试验）
> | 反应类型 | 表现 | 下一步 |
> |---|---|---|
> | **快速反应 (rapid)** | 生命体征恢复并维持 | 完善影像 (CT) |
> | **一过性 (transient)** | 补液时改善、停液恶化 | 备**输血** + 快速评估 (FAST/DPL) |
> | **无反应 (minimal/none)** | 补液无改善 | **立即手术（剖腹/开胸）** |

## 七、Memory Hook

> [!success] 一句话锁定
> - **稳不稳定决定去哪**：稳 → 增强 CT；不稳 + FAST 阳 → 剖腹；不稳不应液 + 腹胀 → 直接手术室（**别去 CT**）。
> - **FAST 看腹膜内、看不清腹膜后** → 肾/胰/十二指肠靠 CT。
> - **脾 = BAT 最常伤**（Kehr 征左肩痛）；延迟破裂 + 非创伤性破裂（传单/CLL）别漏。
> - **上腹被撞 + 淀粉酶↑ = 外伤性胰腺炎**（儿童车把伤经典）；**脂肪酶正常不排除**。
> - **上腹被撞 + 延迟胆汁性呕吐 = 十二指肠血肿**（壁内血肿 → 高位梗阻）；保守 NG + TPN，多 1–2 周吸收。
> - **幼儿 + 十二指肠血肿/胰腺伤 + 病史对不上 = 想虐童**。

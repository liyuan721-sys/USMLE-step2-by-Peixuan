# 孤立肺结节 SPN 管理决策树（Solitary pulmonary nodule）

> [!info] 用途
> 偶发肺结节"下一步"高频。本卡锁：① 定义 ② 恶性概率因素（大小是核心）③ 钙化模式良恶性 ④ **大小 + 概率 → 监测 / 活检 / 切除**决策树。
> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q4252]]（2cm + 不规则 + 重度吸烟 → 手术切除，误选 3 月复查 CT）。

---

## 一、定义

**孤立肺结节 (solitary pulmonary nodule, SPN)** = **≤3cm** 圆形阴影 + 周围肺实质 + **无**淋巴结肿大/肺不张/胸腔积液。
- **>3cm = 肿块 (mass)**（恶性概率更高，直接组织诊断）。

---

## 二、恶性概率↑因素 ⭐

| 维度 | 高恶性 | 低恶性（良性倾向）|
|---|---|---|
| **大小** ⭐ | **>2cm（独立 >50% 恶性）** | **<0.6cm 极少恶性** |
| 年龄 | 高龄 | 年轻 |
| 吸烟 | 现/既往吸烟 | 不吸烟 |
| 边缘 | **不规则/毛刺 spiculated** | 光滑 |
| 位置 | 上叶 | — |
| 病史 | 肺癌家族/个人史 | — |
| 性别 | 女性 | — |
| 生长 | **系列影像增大** | **2 年无变化 = 良性** |

---

## 三、钙化模式（calcification）—— 良恶性钥匙

| 良性钙化（不用慌）| 可疑/恶性 |
|---|---|
| **中央型 central** | 偏心型 eccentric |
| **同心层状 laminated/concentric**（肉芽肿）| 点状 stippled |
| **爆米花样 popcorn**（错构瘤 hamartoma）| **无钙化** |
| 弥漫致密 diffuse | — |

> [!tip] 一句话
> **中央/层状/爆米花/弥漫 = 良性钙化**；偏心/点状/无钙化 = 别放心。

---

## 四、决策树：大小 + 概率 → 处理 ⭐

> [!danger] 第一步永远是「调旧片对比」
> **先找既往影像**：**稳定 >2 年 = 良性**（停）；新发/增大 = 按下走。

```
SPN（已对比旧片，新发或无旧片）
│
├─ <0.6 cm ──────→ 一般不随访
│
├─ 0.6–0.8 cm ───→ 低危监测（※ Fleischner 补充；UWorld/NBME 只考 <0.6 与 >0.8 两点）
│
└─ >0.8 cm ──→ 评估恶性概率（大小>2cm + 老 + 烟 + 不规则缘 + 上叶…）
       │
       ├─ 低概率（<5%）────→ 监测：系列 CT 复查
       │
       └─ 中/高概率（≥5%）─→ 组织诊断：活检 or 手术切除
                              · 需不需组织诊断「不确定」时 → 先 PET（正电子发射断层 positron emission tomography；高代谢→倾向恶性→再活检/切除）
                              · 周围病灶=经胸穿刺；中央=支气管镜；或 VATS（电视胸腔镜手术）切除
```

> [!tip] 两个数 = 两个职责（别混）⭐
> - **0.8 cm = 动作闸门**：决定**要不要 work up**。<0.6 无视；>0.8 才进概率评估。
> - **2 cm = 概率砝码**（不是大小档）：>2cm 单独一项就 >50% 恶性 → 把概率推高 → 从"观察"升级到"切除"。
> 一句话：**0.8 把你从"无视"切到"开查"；2 把你从"观察"推到"切掉"。**

> [!warning] Q4252 焊死：高危大结节 ≠ 复查 CT 观察
> **2cm + 不规则缘 + 重度吸烟 + 65 岁 = 中-高概率 → expedient 组织诊断（手术切除）**。
> **3–6 月复查 CT 只给低概率**（如 1cm + 50 岁 + 不吸烟）。别把高危结节"观察等待"。
> 干扰项：结核皮试（A，结核结节多发+小）/ 痰细胞学（B，敏感度低）/ 单纯安抚（C，高危不可）。

---

## 五、Memory Hook

> [!success] 一句话
> **先对比旧片（稳定 >2 年=良性）。0.8 是闸门、2 是砝码。>0.8cm 看概率：低(<5%) → 复查 CT；中/高(≥5%，大+老+烟+不规则缘) → 组织诊断/手术切除（仅"不确定"时先 PET）。>2cm 独立 >50% 恶性，别观察。良性钙化=中央/层状/爆米花/弥漫。**

## 一、一图总览（咬蜇伤鉴别）

```
咬/蜇伤
├─ 蜘蛛 Spider
│   ├─ 棕色隐居蛛 Loxosceles（提琴背）── 灼痛→水疱→黑焦痂（局部坏死）
│   │       关键：坏死稳定前不清创；冷敷 + 支持；高压氧/激素无获益
│   └─ 黑寡妇 Black widow（红沙漏）──── 腹肌强直/腹痛 + 全身肌痉挛（神经毒）
│           关键：钙剂/苯二氮䓬止痛解痉；重症抗毒血清
├─ 膜翅目 Hymenoptera（蜂/黄蜂/蚂蚁）
│   ├─ 局部小反应（1–5 cm）──────────── 自限
│   ├─ 大局部反应 LLR（≥10 cm，连续）── IgE 局部；冷敷+激素+抗组胺
│   ├─ 全身性过敏 anaphylaxis ────────── 肾上腺素 IM（首位）
│   └─ 毒性反应 toxic reaction（多次蜇）─ 血管活性物质直接毒（呕吐/抽搐/发热）
├─ 蛇 Snake（响尾蛇等 pit viper）──────── 数小时内剧痛/肿胀/瘀斑 + 凝血病/血红蛋白尿
│           关键：制动+抬高+尽快抗蛇毒血清；不切开吸吮、不上止血带
└─ 蜱 Tick（Lyme 等）──────────────── 靶形游走性红斑（中心透亮、不溃疡）；蜱需附着~48h
            关键：是干扰项——"中心透亮"≠隐居蛛坏死灶
```

## 二、蜘蛛咬伤（Spider bites）⭐

### 2.1 棕色隐居蛛 Loxosceles reclusa（brown recluse，loxoscelism）

| 维度 | 要点 |
|---|---|
| 情境 | 美国中南部；**温暖干燥处**（阁楼 attic / 木堆 / 地窖）；穿衣/打扫时被咬；**咬时常无痛、不被发现** |
| 皮损演变 | **灼烧痛 → 数小时后红斑 → 水疱（可蓝色变）→ 黑色焦痂 eschar**；数天扩大、数周愈合；重力依赖方向蔓延 |
| 机制 | venom **磷脂酶 D (sphingomyelinase)** → 局部缺血性皮肤坏死（dermonecrosis，约 10% 咬伤）|
| 系统性 loxoscelism | **罕见**（发热、溶血 hemolysis、DIC）；**与皮损大小无关** |
| 处理 ⭐ | **清洁 + 冷敷（↓磷脂酶活性）+ 止痛 + 破伤风更新**；**坏死稳定前不清创**（早清创→创面扩大、愈合更差，稳定后清创/植皮才有助）|
| **无确切获益**（别选）| **高压氧 hyperbaric O₂**、dapsone 氨苯砜、四环素、糖皮质激素 —— 均未证实有效；抗毒血清美国未批准 |

> [!warning] 隐居蛛两大考点
> - **已成焦痂 → 只做支持护理**（[[mistakes/uworld-mistakes_2026-06#^Q20068]]）；别上高压氧/激素/抗生素（无感染征时）。
> - **"未来并发症" → 局部坏死 + 焦痂**（[[mistakes/uworld-mistakes_2026-06#^Q3399]]），不是系统病。
> - **中心透亮**是**坏死灶**，**不是** Lyme 的靶形红斑（erythema migrans）。

### 2.2 黑寡妇 Latrodectus（black widow）

| 维度 | 要点 |
|---|---|
| 标志 | 黑色 + **腹部红色沙漏**；α-latrotoxin → 大量 ACh/NE 释放（神经毒）|
| 表现 | 咬后数小时内 **腹肌强直/剧烈腹痛**（可像急腹症）+ **全身肌痉挛/痛**（>60%）+ 出汗、呕吐、高血压、心动过速 |
| 处理 | 止痛 + **苯二氮䓬/钙剂解痉**；重症 → 抗毒血清 |
| 对比隐居蛛 | 黑寡妇 = **神经毒/肌痉挛**（无皮肤坏死）；隐居蛛 = **皮肤坏死**（无肌痉挛）|

## 三、膜翅目蜇伤（Hymenoptera：蜂 / 黄蜂 / 蚂蚁）⭐

> [!info] 四种反应分层（按局部 → 系统）
> 同样"被蜂蜇 + 红肿"，**靠范围 + 时间 + 有无系统症状**区分：

| 反应 | 范围/特点 | 时间线 | 机制 | 处理 |
|---|---|---|---|---|
| **局部小反应** | 1–5 cm 红斑肿胀，**限于蜇点** | 数小时 | 局部 | 冷敷、对症 |
| **大局部反应 LLR** ⭐ | **≥10 cm，与蜇点连续**的肿胀红斑硬结 | **24–48 h 高峰**，数天退 | **IgE 介导**（局部）| **冷敷 + 口服/外用激素 + 抗组胺 + NSAID** |
| **全身性过敏 anaphylaxis** | 荨麻疹/喉头水肿/喘鸣/低血压/呕吐 | 数分钟内 | IgE（全身）| **肾上腺素 IM（首位）** + 支持 |
| **毒性反应 toxic reaction** | 多次蜇（大量毒液）；呕吐/腹泻/抽搐/发热 | 数小时 | 毒液血管活性物质**直接毒**（非 IgE）| 支持 |

> [!warning] LLR 易被误判（[[mistakes/uworld-mistakes_2026-06#^Q20066]]）
> LLR ≠ 感染。**别误选淋巴管炎 / 蜂窝织炎**：
> - **淋巴管炎 lymphangitis**：细菌沿淋巴管 → **线状红色条纹向近端走 + 发热**（LLR 是以蜇点为中心的**连续片状**、无发热）。
> - **蜂窝织炎 cellulitis**：蜇后**感染需 3–5 天**才起（LLR 24–48 h 即高峰）；昆虫咬后早期超感染少见。
> - LLR 患者未来全身过敏风险略升（<3–16%），是否常规开肾上腺素笔仍有争议。

### 局部红肿"四鉴别"（一句话钥匙）

| 病 | 钥匙 |
|---|---|
| **LLR** | 蜇点为中心**连续大片** + **无发热** + 24–48 h 高峰（IgE）|
| **蜂窝织炎** | 咬后 **3–5 天**起 + 发热 + 边界欠清 |
| **淋巴管炎** | **线状红条纹** + 发热（细菌沿淋巴播散）|
| **筋膜室综合征** | 张力性肿 + **被动牵拉剧痛 / 灌注或感觉↓**（罕见，需急诊）|

## 四、蛇咬伤（Snakebite，pit viper 响尾蛇/铜头蛇）

| 维度 | 要点 |
|---|---|
| 表现 | **数小时内**咬处剧痛、肿胀、瘀斑/变色 + 进行性蔓延；可 **凝血病、血红蛋白尿、血小板↓、肌肉麻痹/系统毒** |
| 处理 ⭐ | **制动 + 抬高患肢 + 尽快抗蛇毒血清（antivenom）** + 监测肿胀范围/凝血 + 破伤风 |
| **别做** | ❌ 切开吸吮、❌ 止血带、❌ 冰敷过度（加重缺血）|
| 对比隐居蛛 | 蛇咬**数小时内**重度局部痛+肿+系统毒；隐居蛛初常无痛、数天才坏死 |

## 五、蜱 / Lyme —— 经典"干扰项"（不是咬蜇坏死）

> [!tip] 为什么放进来：它是隐居蛛/蜘蛛咬题的高频干扰项
> - **游走性红斑 erythema migrans**：**靶形（牛眼）、中心透亮、不溃疡、不痛/微痒**；蜱需**附着 ~48 h** 才传病。
> - **Lyme 晚期**（数周—数月）：关节炎、**心脏炎（高度 AV 阻滞）**、**面神经麻痹 Bell palsy**、神经 Lyme。
> - ❌ 蜘蛛咬题里看到"关节炎/心律失常/面瘫"= Lyme 晚期诱导项（[[mistakes/uworld-mistakes_2026-06#^Q3399]]）——**蜘蛛咬的并发症是坏死焦痂，不是 Lyme**。
> - 治疗见 [[完整笔记/专题笔记/感染/感染_蜱传疾病鉴别]]（早期 doxycycline；孕妇 amoxicillin；神经/心脏 ceftriaxone）。

## 六、高频陷阱黑名单

> [!warning] 易错点
> - **隐居蛛已成焦痂 = 只支持护理**，别上高压氧 / 激素 / 抗生素（无感染征时）；**坏死稳定前不清创**。
> - **隐居蛛"中心透亮" = 坏死灶**，不是 Lyme 靶心；蜘蛛咬并发症 = 坏死焦痂，不是 Lyme 关节/心脏/面瘫。
> - **蜂蜇大片肿（连续 + 无发热 + 24–48 h）= LLR**，不是淋巴管炎（红条纹+发热）/蜂窝织炎（3–5 天后）。
> - **anaphylaxis = 肾上腺素 IM 首位**（不是抗组胺/激素首位）。
> - **黑寡妇 = 肌痉挛/腹痛（神经毒）**；**隐居蛛 = 皮肤坏死** —— 两种蜘蛛别混。
> - **蛇咬别切开吸吮/上止血带**；制动 + 抬高 + 抗蛇毒血清。

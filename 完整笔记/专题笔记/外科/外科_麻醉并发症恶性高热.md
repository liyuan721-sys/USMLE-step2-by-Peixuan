# 恶性高热 (Malignant Hyperthermia, MH) 及麻醉相关高热/僵硬急症鉴别

> [!danger] 一句话核心
> 全身麻醉诱导后**分钟级**出现咬肌/全身**肌僵硬 + 高碳酸血症（对增加通气无反应）+ 心动过速**，随后**高热**——就是 **恶性高热 (MH)**。立即停用致病麻醉药 + 静推 **dantrolene 丹曲林** 是救命第一步，高热是晚期表现，别等它出现才诊断。

---

## 一、恶性高热 (Malignant Hyperthermia, MH)

- **本质**：罕见但**致命**的骨骼肌高代谢危象。
- **遗传/机制**：常染色体显性遗传，最常见为 **RYR1 (ryanodine receptor 1, 兰尼碱受体) 基因突变** → 骨骼肌肌浆网**钙 (Ca²⁺) 调控失常** → 触发药暴露后细胞内钙大量、持续释放 → 肌肉持续收缩 + 高代谢（产热、耗氧、产 CO₂）。
- **触发药 (triggers)**：
	- **挥发性吸入麻醉药 (volatile anesthetics)**：如 **halothane 氟烷 / sevoflurane 七氟烷 / isoflurane / desflurane**。
	- **succinylcholine 琥珀胆碱**（去极化型肌松剂）。

> [!info] 记忆钩子
> 触发 = "挥发麻醉药 + 琥珀胆碱"。局麻药、非去极化肌松剂（如 rocuronium 罗库溴铵）、丙泊酚 propofol、氧化亚氮 nitrous oxide **不触发**，可安全用于易感者。

### 临床表现（麻醉诱导后分钟级出现）

- **Masseter/generalized rigidity 咬肌/全身肌僵硬**：常为**最早**体征（尤其给琥珀胆碱后咬肌痉挛）。
- **Sinus tachycardia 窦性心动过速**：常为**最早的循环征象之一**。
- **Hypercarbia 高碳酸血症**：呼气末 CO₂ (EtCO₂) 迅速升高，**对增加分钟通气量无反应**（关键鉴别点）+ 通气困难。
- **Rhabdomyolysis 横纹肌溶解**：CK 升高、肌红蛋白尿（酱油色尿）。
- **Hyperkalemia 高钾血症**：肌细胞破坏释放钾 → 可致致命性心律失常。
- **Hyperthermia 高热**：**晚期**表现，体温可 **>40℃**，进展迅猛。
- 后续可有代谢性酸中毒、DIC、多器官衰竭。

> [!warning] 高热是"晚期 + 命名误导"
> 名字叫"恶性高热"，但**高热往往是后到的表现**。等到体温飙升才想到 MH 会错过黄金抢救窗。真正的早期信号是**不明原因的 EtCO₂ 升高 + 肌僵硬 + 窦速**。

### 治疗

> [!danger] 抢救顺序
> 1. **立即停用所有触发性麻醉药**（关挥发麻醉药、停琥珀胆碱），换非触发方案，100% 氧气过度通气。
> 2. **Dantrolene 丹曲林**（特效药）：直接作用于骨骼肌 ryanodine 受体，抑制肌浆网钙释放 → 终止肌肉高代谢。尽快静脉给药并按需重复。
> 3. **主动降温**：冰袋、冷盐水灌注、体表降温。
> 4. **支持 + 纠正内环境**：治疗**高钾**（胰岛素+葡萄糖、钙剂、碳酸氢钠）、纠正**酸中毒**、处理心律失常、监测/防治横纹肌溶解与 DIC，转 ICU。

## 二、核心鉴别：MH vs NMS vs 5-HT 综合征 vs 镁中毒

四者都可有"高热 + 神经肌肉异常"，但**触发因素、起病速度、关键体征、解药各不相同**，是 USMLE 高频对比点。

| 综合征 | 触发 | 起病速度 | 关键特征 | 解药/处理 |
| --- | --- | --- | --- | --- |
| **MH 恶性高热** | 挥发性麻醉药 / succinylcholine 琥珀胆碱 | 麻醉后**分钟级** | 肌僵硬（咬肌/全身）+ 高热 + 高碳酸（增加通气无效）+ 难通气 + 高钾 | **停触发药 + dantrolene 丹曲林** |
| **NMS 神经阻滞剂恶性综合征** | 抗精神病药（haloperidol 氟哌啶醇等）/ 突然撤停多巴胺能药 | **数天**（缓） | 铅管样强直 (lead-pipe rigidity) + 高热 + 自主神经不稳 + 意识改变 + CK↑ | **停药 + dantrolene / bromocriptine 溴隐亭** |
| **Serotonin syndrome 5-HT 综合征** | 5-羟色胺能药（SSRI / MAOI / 曲马多 tramadol / 林奈唑胺等，常为联用） | **数小时**（急） | **阵挛 clonus（下肢显著）+ 反射亢进 (hyperreflexia) + 激越 + 高热** | 停药 + 支持 + **cyproheptadine 赛庚啶** |
| **Magnesium toxicity 硫酸镁中毒** | 静脉硫酸镁（子痫前期/子痫治疗） | 输注中，随血镁上升 | 肌**无力**、**深腱反射消失** (areflexia)、呼吸抑制、传导延长/心脏骤停（**非**僵硬、**非**高热） | 停镁 + **calcium gluconate 葡萄糖酸钙** |

> [!tip] 一步分开的钥匙
> - **反射方向**：MH/NMS/5-HT 是"肌张力↑"，镁中毒是"反射消失、肌无力"——方向相反，最好用来排镁中毒。
> - **起病时间轴**：分钟（MH）< 小时（5-HT）< 天（NMS）。
> - **僵硬 vs 阵挛**：NMS = 铅管样"全程僵"；5-HT = **clonus/阵挛 + 反射亢进**（下肢明显）。
> - **解药记忆**：MH 与 NMS 都可用 **dantrolene**；5-HT 用 **cyproheptadine**；镁中毒用 **calcium gluconate**。

```
麻醉/用药后 出现「肌肉+体温」异常？
│
├─ 分钟级，全麻中，EtCO₂↑难通气 + 肌僵硬 ── MH ──► 停触发药 + dantrolene
│
├─ 数天，抗精神病药/撤多巴胺药，铅管强直 ── NMS ──► dantrolene / bromocriptine
│
├─ 数小时，5-HT 能药，clonus + 反射亢进 ── 5-HT 综合征 ──► cyproheptadine
│
└─ 输镁中，肌无力 + 反射消失 + 呼吸抑制 ── 镁中毒 ──► calcium gluconate
```

## 三、产科情境钩子（高频陷阱）

> [!warning] 经典 stem 模式
> 子痫前期 (preeclampsia) 产妇正在静脉输**硫酸镁 (magnesium sulfate)**，因胎儿窘迫行**急诊剖宫产全身麻醉**；插管（挥发麻醉药 + 琥珀胆碱）后**突发全身僵硬 + 高热 + EtCO₂ 升高、通气困难**。
>
> - 陷阱：题干先铺垫"正在用镁"，诱导你选**镁中毒**。
> - 破题：**镁中毒是肌无力 + 反射消失**，绝不会是**僵硬 + 高热 + 高碳酸**。出现"僵硬 + 高热 + 难通气"= **MH**（触发来自全麻的挥发麻醉药/琥珀胆碱）。
> - 下一步：停触发麻醉药 + **dantrolene 丹曲林**。

## 四、自测题

> [!question]- Q1. 32 岁子痫前期产妇静脉输注硫酸镁中，因胎心减速行全麻剖宫产。插管（七氟烷 + 琥珀胆碱）后数分钟内出现全身肌肉僵硬、窦速、EtCO₂ 快速升高且增加分钟通气量无改善，随后体温升至 40.2℃。最可能诊断及首要处理？
> **答案**：恶性高热 (MH)。分钟级起病 + 全麻触发（挥发麻醉药 + 琥珀胆碱）+ 肌僵硬 + 对通气无反应的高碳酸 + 高热 = MH，**不是**镁中毒（镁中毒是肌无力、反射消失、呼吸抑制，绝无僵硬高热）。首要处理：**立即停用触发性麻醉药 + 静推 dantrolene 丹曲林**，同时降温、纠正高钾与酸中毒。

> [!question]- Q2. 如何用"起病速度 + 关键体征"一步区分 MH、NMS 与 5-HT 综合征？各自解药是什么？
> **答案**：
> - **MH**：麻醉后**分钟**级，肌僵硬 + 高碳酸 + 高热 → **dantrolene**。
> - **NMS**：抗精神病药后**数天**，**铅管样强直** + 高热 + 自主神经不稳 + 意识改变 → **dantrolene / bromocriptine**。
> - **5-HT 综合征**：5-HT 能药后**数小时**，**clonus 阵挛（下肢显著）+ 反射亢进 + 激越** + 高热 → **cyproheptadine**。
> 时间轴：分钟 < 小时 < 天；僵（NMS）vs 阵挛/反射亢进（5-HT）是关键分野。

> [!question]- Q3. 一名子痫产妇静脉输硫酸镁后出现全身无力、膝反射消失、呼吸变浅。这与恶性高热在体征上最本质的区别是什么？如何处理？
> **答案**：**镁中毒表现为肌无力 + 深腱反射消失 (areflexia) + 呼吸抑制**，肌张力和反射是**下降**方向；而 MH 是**肌僵硬 + 反射/张力升高 + 高热 + 高碳酸**，方向相反。处理：**停镁 + 静推 calcium gluconate 葡萄糖酸钙**（钙拮抗镁），支持通气。

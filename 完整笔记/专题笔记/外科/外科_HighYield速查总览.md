## 一、术前评估（Preoperative evaluation）

### 1.1 心脏风险（最高频）
**核心问题**：非心脏手术会不会发生 MACCE（major adverse cardiac & cerebrovascular events，心梗/卒中/心脏骤停/心源性死亡）？

**5 步算法**（Hook：「急诊直接做，活动性心脏病必须等；低危手术不用管，高危人衰 METs<4 才压测」）

```
① 急诊手术？           → 是：直接手术
② 活动性心脏病(急性冠脉综合征 ACS 等)？→ 是：推迟，先治心脏
③ 低风险手术？         → 是：直接手术
④ RCRI 评分           → 0–1：直接手术 ; ≥2：评 METs
⑤ METs               → ≥4：直接手术 ; <4 且检查会改变管理：药物负荷压测
```

- **手术风险分级**：高危主要心脏不良事件 (MACE, major adverse cardiac events) >5%（大血管/胸腔内/腹腔内开放）；中危 1–5%（头颈/全髋膝置换/前列腺）；低危 <1%（白内障/内镜/乳腺/浅表）。
- **RCRI (Revised Cardiac Risk Index 改良心脏风险指数) 6 因子**：①高风险手术 ②缺血性心脏病史 ③充血性心衰 (CHF) 史 ④脑血管病 (卒中/短暂性脑缺血发作 TIA) ⑤需胰岛素的糖尿病 ⑥术前肌酐 (Cr) >2 mg/dL。
- **METs (metabolic equivalents 代谢当量)**：<4 = 差（仅吃饭穿衣室内慢走）；≥4 = 可（爬一层楼/园艺/高尔夫）。

> [!tip] 药物负荷压测选哪种（按合并症分流）
> - **多巴酚丁胺负荷超声 (dobutamine stress echo)**：增心肌耗氧；**哮喘/慢阻肺 (COPD) 首选**。
> - **腺苷/双嘧达莫 (adenosine/dipyridamole) 心肌灌注显像 (MPI)**：冠脉扩张窃血；**严重哮喘/COPD 禁忌**（支气管痉挛）。

> [!warning] 药物红线
> β 受体阻滞剂 (β-blocker) 平时用则继续，**禁手术当天新启动高剂量**；直接口服抗凝药 (DOAC) 术前停 1–3 天无需桥接。

### 1.2 抗凝术前管理（房颤 AF 常考）
**多数房颤停抗凝不需桥接 (bridging)**，仅**华法林 (warfarin，维生素 K 拮抗剂 VKA) + 中/高血栓风险**才桥接；DOAC 一律不桥。

### 1.3 瓣膜病术前
**有症状的重度主动脉狭窄 (severe AS, aortic stenosis)** → **择期手术前先处理瓣膜**；重度二尖瓣反流 (MR)/主动脉反流 (AR) + 差射血分数 (EF, ejection fraction)/肺高压 → 同理先处理瓣膜。

### 1.4 肺/肾/肝/内分泌
- **肺**：戒烟 **6–8 周**（可用贴片）。
- **肾**：术前停二甲双胍 (metformin)；透析患者末次透析安排在**术前 24h**。
- **肝**：严重肝病≈手术近乎绝对禁忌。
- **糖尿病 (DM)**：停口服降糖，应激下改胰岛素。

## 二、创伤（Trauma）

### 2.1 Primary survey（ABCDE）
A 气道（+颈椎保护，不能维持 → 插管）→ B 呼吸（张力气胸/大量血胸/肺挫伤/连枷胸）→ C 循环 → D 神志（格拉斯哥昏迷评分 GCS, Glasgow Coma Scale）→ E 暴露全身查。
**原则**：缺乏明确诊断不能延误指征明确的救命处理。

### 2.2 腹部创伤影像 + BAT 决策
**FAST (focused assessment with sonography in trauma) 只看腹膜内（肝/脾/腹水），看不清腹膜后（肾/胰/主动脉）。**

| 状态 | 处理 |
|---|---|
| 不稳定 + FAST 阳性 | **立即剖腹 laparotomy** |
| 稳定 + 高危机制/血尿 | 增强 CT |
| 不稳定 + FAST 阴性/可疑 | 诊断性腹腔灌洗 (DPL) |
| 弥漫性腹膜炎 | 急诊剖腹 |

> [!danger] 钝性腹部创伤 (BAT) 铁律
> 补液 2L 后仍不稳 + 腹胀 → **直接进手术室开腹，绝不去做 CT**。

### 2.3 钝伤器官定位
**脾 = BAT 最常受损**（左季肋，Kehr 征 = 左肩牵涉痛，膈刺激 C3–5）；胰 = 上腹中心挤压（车把伤），**CT 早期易漏，数日后胰周积液 → 疑胰管损伤**。
**非创伤性脾破裂**高危：慢性淋巴细胞白血病 (CLL)/慢性粒细胞白血病 (CML)/淋巴瘤、传染性单核细胞增多症 (mononucleosis 传单)、系统性红斑狼疮 (SLE)；**延迟破裂** = 被膜下血肿初 CT 正常、数天后突发休克。

### 2.4 泌尿生殖 (GU, genitourinary) 创伤决策
| 钥匙 | 部位 | 下一步 |
|---|---|---|
| 侧腹痛 + 下肋骨折 + 血尿 + 休克 | 肾 | 增强 CT |
| 骨盆骨折 + 尿道口流血 + 前列腺高浮 + 插不进尿管 | 尿道 | 逆行尿道造影 (RUG) |
| 骨盆骨折 + 耻骨上压痛 + 大量肉眼血尿 | 膀胱 | 逆行膀胱造影 (cystography) |

🧩 GU 创伤深挖 → [[完整笔记/专题笔记/外科/外科_GU创伤处理体系]]

### 2.5 复苏反应评估
快速反应 → 完善 CT；**一过性**反应（停液恶化）→ 备输血 + FAST/DPL；**无反应** → 立即剖腹/开胸。

### 2.6 创伤后胸痛（72h "心脏风暴"）
不明胸痛 + 脉搏绝对不齐（房颤）+ 低血压 → **先做心电图 (ECG, electrocardiogram)**；证实快速房颤 + 低血压 → 同步电复律（但复律前必须先有 ECG）。钝性心脏损伤 = 胸骨骨折 + 新发心律失常 → ECG + 超声心动图。

## 三、胸部急症（Thoracic emergencies）

| 急症 | 钥匙 | 颈静脉 | 处理 |
|---|---|---|---|
| 张力气胸 tension PTX | 患侧呼吸音↓ + 气管推向健侧 | 怒张 | **立即针刺减压 → 胸管**（别等 X 线）|
| 自发气胸 spontaneous PTX | 年轻高瘦男、突发、气管居中 | 多平 | 针刺减压 ± 胸管 |
| 大量血胸 massive hemothorax | 患侧呼吸音/叩浊↓ + 颈静脉**扁平**（失血）| 平 | **直接胸管**（不先针刺）|

> [!danger] 针刺减压 (needle decompression) 操作 —— ICS 别混
> 进针点二选一：**第 2–3 肋间隙 (ICS, intercostal space 肋间隙) 锁骨中线** 或 **第 5 肋间隙腋前线**（成人首选，胸壁薄）；**贴下位肋骨上缘进针**避开肋骨下缘的神经血管束 (VAN)。
> ⚠️ 此处 ICS = 肋间隙；慢阻肺里的 ICS = 吸入糖皮质激素 inhaled corticosteroid。
> 颈静脉**怒张找"挤"**（气胸/压塞）、**平找"漏"**（出血）。

## 四、液体与血制品（Fluids & blood products）

- **等渗（生理盐水 NS, 0.9% saline）**：只扩细胞外液 (ECF)；扩容/休克首选。重度脱水 → **20 mL/kg 等渗 bolus**。
- **低渗（0.45% saline）**：水进细胞。
- **D5W (5% dextrose in water)**：葡萄糖代谢后 = 自由水（最终低渗），**非扩容液**；用于高钠血症补自由水。
- **高渗（3% saline）**：水拉出细胞；用于**严重症状性低钠 / 脑水肿 / 颅内压 (ICP) ↑**。
- **乳酸林格液 (LR, lactated Ringer's)**：**烧伤复苏首选**（近 ECF，乳酸→碳酸氢盐缓冲；不用 NS 防高氯性代酸）。

> [!tip] 血制品要点
> - 浓缩红细胞 (PRBC)：**不与 LR 同输**（LR 的钙 + 枸橼酸 citrate → 凝血并发症）；每单位升红细胞压积 (Hct, hematocrit) 3–4。
> - 新鲜冰冻血浆 (FFP)：含凝血因子，纠正凝血酶原时间/部分凝血活酶时间 (PT/PTT)（等不及维生素 K 时）。
> - 冷沉淀 (cryoprecipitate)：纤维蛋白原 + 因子 VIII（血管性血友病 vWD/血友病 A/弥散性血管内凝血 DIC）。
> - 凝血酶原复合物 (PCC)：因子 II/VII/IX/X（血友病 B）。
> - **大量输血 PRBC : 血浆 : 血小板 = 1 : 1 : 1**。

🧩 烧伤 Parkland（4 mL×kg×全身体表面积 TBSA%, total body surface area）/ 4-2-1 维持 / 前 8h 给半 → [[完整笔记/专题笔记/外科/外科_烧伤管理速查]]

## 五、术后并发症（Postoperative complications）

### 5.1 术后发热 5W 时间线（经典）

| 术后日 | 病因 | 处理 |
|---|---|---|
| Day 1 | 肺不张 atelectasis (Wind) | 激励性肺量计/深呼吸 |
| Day 3 | 肺炎 pneumonia | CXR + 痰培养 + 抗生素 |
| Day 4–5 | 尿路感染 UTI (Water) | 尿培养 + 抗生素 |
| Day 5–7 | 深静脉血栓/肺栓塞 DVT/PE (Walking) | 多普勒/CT；肝素桥接华法林 |
| Day 7–8 | 伤口感染 wound (Wound) | 培养 + 抗生素 + 引流 |
| Day 9–15 | 脓肿/药物热 (Wonder drug) | CT + 切开引流 (I&D) |

### 5.2 术后神志改变（关键分流）
> [!danger] "AMS（意识改变）+ 呼吸频率"= 一秒分流
> - **AMS + 呼吸慢 (<10/min)** → 阿片诱发呼吸抑制 → 查**动脉血气 (ABG)** → 纳洛酮 (naloxone)。
> - **AMS + 呼吸快** → 肺栓塞 / 脓毒症 / 糖尿病酮症酸中毒 (DKA)。

- **戒酒综合征 (AWS)**：6–24h 震颤出汗 → 12–48h 戒断癫痫 → 48–96h 震颤谵妄 (DT)；一线**静脉苯二氮䓬 (benzodiazepine)**，苯妥英 (phenytoin) 无效。
- 🔴 **恶性高热 (MH, malignant hyperthermia)**：氟烷 halothane / 琥珀胆碱 succinylcholine 诱发（全麻场景）→ **早征 = 高碳酸（加大通气不改善）+ 心动过速 + 咬肌/全身肌僵 + 棕尿（肌红蛋白尿）**；**⚠️ 高热是晚期表现，初期体温可不高——别因"没发热"排除 MH**。处理 = 停触发药 + **丹曲林 (dantrolene)** + 降温。**反复错**（Q15653 两错）：麻醉场景先想 MH，不是 NMS（抗精神病药 + 数天）。→ [[完整笔记/专题笔记/外科/外科_麻醉并发症恶性高热]]
- **术后非血栓性心肌损伤**（术中低血压致冠脉灌注↓）→ 治疗 = **补液，禁溶栓**。

### 5.3 镇痛与导管
- **术后高血压 + 剧痛** → 高血压继发于痛，首选**强化止痛（患者自控镇痛 PCA 吗啡）**而非降压药。PCA **呼吸频率 (RR, respiratory rate) <10 停药**；**严禁家属代按 (PCA by proxy)**。
- **腹腔脓肿**：术后 1–2 周再发热腹痛；膈下脓肿 = 右上腹痛 + 打嗝 + 右胸水 → CT → **经皮穿刺引流** + 抗生素。
- **中心导管相关血流感染 (CLABSI) bundle**：最大无菌屏障、皮肤消毒首选**氯己定 (chlorhexidine)**、首选**锁骨下静脉**、scrub the hub。

🧩 术后肺不张/PE 深挖 → [[完整笔记/专题笔记/外科/外科_术后肺部并发症与肺不张]]

## 六、血管病（Vascular）

- **主动脉夹层 (aortic dissection)**：撕裂样胸痛**放射到背**；**升主动脉** → 先降压（拉贝洛尔 labetalol + 硝普钠 nitroprusside）再**手术**；**降主动脉** → 药物降压。
- **腹主动脉瘤 (AAA, abdominal aortic aneurysm 腹主动脉瘤)**：初查**超声**，确诊后 CT 定大小；**手术阈值：男 5.5 cm / 女 5.0 cm / 半年增 >0.5 cm**；不稳定 + 已知 AAA → 直接急诊修补。

![[外科_AAA不稳定评估流程.png]]
*疑似**不稳定 AAA** 评估流程：有症状（腹/胁/腹股沟痛 + 搏动性肿块 + 胁腹瘀斑 + 肢体缺血）先判**血流动力学是否稳定**。稳定 → 腹部 CT 确认 → 修补；**不稳定** → 已知 AAA 直接**急诊修补 (emergency repair)**，未知 AAA 先做**床旁腹部超声 (focused abdominal ultrasound)** 快速确认再急诊修补。核心钥匙：**不稳定者绝不为影像耽误手术**（超声在手术室/床旁即可，不去 CT 室）。*
- **锁骨下动脉盗血 vs 胸廓出口综合征 (TOS)**：引起**脑症状（晕厥/意识改变）的是盗血**（两臂收缩压差 >15–20 mmHg + 锁骨上杂音 + 椎动脉血流反向）；TOS 是年轻人举臂臂痛（结构压迫如颈肋），治疗切肋骨。
- **外周血管病 (PVD)**：间歇性跛行；初查**踝肱指数 (ABI)** → 运动康复 + 西洛他唑 (cilostazol)。
- **动脉 vs 静脉溃疡**：动脉=**痛 + 外侧**（吸烟/DM）；静脉=**不痛 + 内侧**（抬腿 + 加压）。

## 七、颅内出血 + 骨科（Neuro-trauma & Ortho）

| 出血 | 钥匙 | 处理 |
|---|---|---|
| 硬膜外 epidural | 侧方外伤 + **清醒间期 lucid interval** | 急诊开颅 craniotomy |
| 硬膜下 subdural | **桥静脉**、延迟（数日–周）| CT；有中线移位 → 开颅，无 → 抬头/甘露醇/过度通气 |
| 蛛网膜下 SAH | 霹雳样头痛 + 畏光 | 非增强 CT → 阴性疑诊查腰穿 LP (黄变 xanthochromia) → 磁共振血管造影 MRA 定位；**硫酸镁防癫痫 + 钙通道阻滞剂 (CCB) 防血管痉挛 + 夹闭** |

**髋/股骨**：车祸 (MVA, motor vehicle accident) 后腿短缩 + **内旋** = 髋脱位（闭合复位）；老年跌倒 + **外旋** = 髋骨折；股骨干骨折 → 并发**脂肪栓塞**（详见 [[完整笔记/专题笔记/pulmonary/pulmonary_肺栓塞PE]] §四）。

## 八、普外 GI（General surgery）

- **消化性溃疡手术指征 IHOP**：Intractable（顽固痛）/ Hemorrhage（出血）/ Obstruction（梗阻）/ Perforation（穿孔）。
- **胆道四病**：
  - 餐后间歇右上腹 (RUQ, right upper quadrant) 痛（40 岁肥胖）= 胆绞痛 → 超声 → 择期胆囊切除。
  - 持续 RUQ 痛放射肩胛 ± 发热 = 胆囊炎 → 超声（看不到→肝胆显像 HIDA scan）；**<72h** 早期胆囊切除，**>72h** 保守延迟。
  - 痛 + 发热 + 黄疸 = **上行性胆管炎 (ascending cholangitis)** → 抗生素 + **内镜逆行胰胆管造影 (ERCP)**。
  - 痛 + 黄疸 = **胆总管结石 (choledocholithiasis)** → ERCP。
- **肠系膜缺血 (mesenteric ischemia)**：房颤 + 餐后突发剧痛 = 急性 → CT 血管造影 → 取栓；老年 + 数月餐后痛 = 慢性。
- **阑尾炎 (appendicitis)**：无并发症 → 抗生素 + 手术；**穿孔 → 先保守**（抗生素），改善后手术。
  - **妊娠期阑尾炎 (appendicitis in pregnancy)**：增大的子宫将阑尾**向头侧 (cephalad) 推移**，压痛点随孕周上移（中晚孕可位于右中腹/右上腹而非经典 McBurney 点）→ 易误诊、穿孔风险高；孕妇疑阑尾炎首选 **超声 (ultrasound)**，不确定时 **MRI**（避辐射）；确诊仍 **阑尾切除术 (appendectomy)**，孕期不因妊娠延误手术。

![[L37062.png]]

*妊娠 32 周子宫将阑尾推向头侧 (cephalad displacement) — 解释中晚孕阑尾炎疼痛定位上移*
- **憩室炎 (diverticulitis)**：左下腹痛 → CT + 抗生素；**憩室病 (diverticulosis)** = 无痛便血 → 结肠镜 + 高纤维。
- **肠梗阻**：小肠梗阻 (SBO) 最常见**粘连** → 禁食 (NPO, nil per os)/胃肠减压/观察至需手术。

---

## 九、泌尿生殖肿瘤（GU oncology）

- **肾细胞癌 (RCC)**：血尿 + 侧腹痛 + 可触及肿块（± 红细胞生成素 EPO↑）；右侧更需警惕（直接入下腔静脉 IVC）→ 根治性肾切除。
- **膀胱癌**：**无痛血尿**（吸烟/血吸虫）→ 经尿道膀胱肿瘤切除 (TURBT)。
- **前列腺癌**：多无症状或似良性前列腺增生 (BPH)。
- **睾丸癌**：20–30 岁、**无痛单侧肿块**；标志物 乳酸脱氢酶 (LDH) / β-人绒毛膜促性腺激素 (β-hCG) / 甲胎蛋白 (AFP)；高危 = 隐睾。

---

## 十、手术部位感染预防（SSI prophylaxis）

> [!tip] 抗生素核心记忆
> **皮肤 → 头孢唑林 (cefazolin)**；**膈肌以下 → 覆盖革兰阴性杆菌**；**穿透肠管/口腔 → 必加甲硝唑 (metronidazole) 覆盖厌氧**。
> - 清洁手术 (clean)：cefazolin（过敏/耐甲氧西林金葡菌 MRSA → 万古霉素 vancomycin / 克林霉素 clindamycin）；疝修补放补片 (mesh) 才需预防。
> - **结直肠**：cefazolin + metronidazole。
> - **剖宫产/子宫切除**：cefazolin，**切皮前 60 分钟内**给。

## 十一、感染速查（识别钥匙 → 经验治疗）

- **感染性心内膜炎**：发热 + 新杂音；急性 = 金黄色葡萄球菌 (S. aureus，常三尖瓣)；亚急性 = 草绿色链球菌 (S. viridans)；Dx = 超声 + 血培养；经验 = 万古霉素 + 庆大霉素。
- **膀胱炎/肾盂肾炎**：E. coli/Proteus/Klebsiella；best initial = 尿常规，most accurate = 尿培养。
- **脑膜炎**：先血培养 + 经验抗生素；**腰穿前先 CT 指征** = 意识改变/视乳头水肿/癫痫/颅压高体征。
- **血性腹泻**：弯曲杆菌 (Campylobacter → 吉兰-巴雷 GBS) / 肠出血性大肠杆菌 (EHEC → 溶血尿毒综合征 HUS，未熟牛肉)。

# 外科_腮腺肿大鉴别（疼痛不可靠 + 面瘫=恶性）

> [!danger] 为什么建这张表
> 2026-06-10 ENT 块由两道腮腺题 + 两个追问串成体系：
> - [[mistakes/uworld-mistakes_2026-06#^Q4599]] 腮腺**肿瘤** + 面瘫 = 恶性（我误选"无疼痛"）
> - [[mistakes/uworld-mistakes_2026-06#^Q4544]] 化脓性腮腺炎（肿到张不开嘴**却无面瘫**）
> - 追问 ①"炎症肿大不会压迫神经吗？" ②"腮腺炎可以不疼吗？"
> 核心结论两条：**疼痛不可靠（判别力≈0）**；**面瘫才是恶性红旗**。主笔记散见 [[完整笔记/Peixuan分科笔记/感染]]（Mumps）/ [[完整笔记/Peixuan分科笔记/消化腺]]（唾液腺淀粉酶/结石）。

---

## 一、按"疼不疼"分类（炎症≠一定疼）⭐

> [!warning] "炎症 = 疼"不绝对
> 急性细菌/病毒性腮腺炎**疼**；慢性自身免疫性、肉芽肿性、非炎症性肥大**可以不疼**。

| 类型 | 疼吗 | 招牌 |
|---|---|---|
| **化脓性腮腺炎 Suppurative parotitis**（金葡 S. aureus）| **剧痛 + 压痛** | 老年术后脱水、发热、WBC↑、咀嚼加重、血淀粉酶↑（无胰腺炎）|
| **流行性腮腺炎 Mumps**（腮腺炎病毒 paramyxovirus）| **痛/压痛**（多双侧）| 未接种、发热头痛、**睾丸炎 orchitis**、血淀粉酶↑ |
| **涎石症 Sialolithiasis**（导管结石）| **进食时痛**（meal-time pain）| 反复肿胀、可继发感染 |
| **干燥综合征 Sjögren syndrome**（自身免疫）| **常无痛 / 轻压痛** | 慢性反复腮腺肿 + 口眼干、抗 SSA/SSB |
| **结节病 / IgG4 相关病 Sarcoidosis / IgG4-RD** | **多无痛** | 慢性、双侧、其他器官受累 |
| **涎腺病 Sialadenosis**（非炎症性肥大）| **无痛** | 酗酒/糖尿病/贪食症/营养不良，双侧软肿 |
| **腮腺肿瘤（良/恶）** | 良性多无痛；恶性疼痛**无规律** | 良性最常见多形性腺瘤；**恶性看面瘫不看疼** |

## 二、良恶性红旗（判别权重压在面神经）⭐⭐

> [!danger] 腮腺肿物恶性红旗
> **疼痛/无疼痛都不可靠（双向）** → 改看下面这些：

```
腮腺肿物 → 找恶性红旗：
  ① 面神经麻痹（CN VII 面瘫）   ← 最强信号（神经侵犯）
  ② 面部麻木（CN V）
  ③ 固定 / 快速增大 / 皮肤侵犯
  ④ 颈部淋巴结肿大
  ⑤ 放射史 / 既往恶性
  —— 任一阳性 → 影像(CT/MRI) + 细针穿刺，按恶性查
良性提示：可活动、边界清、缓慢生长、无脑神经症状
```

> [!tip] 一句话
> **腮腺肿物 + 面瘫(CN VII)/面麻(CN V) = 恶性 until proven otherwise**，不管腺体肿多大、疼不疼。

## 三、面神经：挤开 vs 侵犯（为什么炎症/良性肿不出面瘫）⭐

> [!info] 解剖前提
> 面神经 CN VII **穿行腮腺**（分浅叶/深叶）。但神经对两种损害方式耐受**完全不同**：

| 损害方式 | 面神经结局 | 对应病 |
|---|---|---|
| **占位/炎症"挤开"（displacement）** | 神经被推移但**仍工作 → 无面瘫** | 良性瘤、化脓性腮腺炎、囊肿 |
| **肿瘤"侵犯破坏"（perineural invasion）** | 轴突被浸润摧毁 → **面瘫** | 腮腺恶性肿瘤 |

> [!success] 活教材就在错题本
> [[mistakes/uworld-mistakes_2026-06#^Q4544]] 化脓性腮腺炎**肿到 trismus（张口受限）+ 剧痛发热**，比 Q4599 肿得还狠 —— **却完全没有面瘫**。证明：**炎症再肿也只是"挤"，不会"瘫"**。出现面瘫 = 神经被侵犯（癌）。

> [!warning] 别混淆
> 单纯**面神经本身炎症 = Bell 麻痹 Bell palsy**，是另一回事（不是腺体压出来的）。

## 四、高频腮腺病速认

> [!question]- 老年术后 + 痛性耳前肿 + 发热 WBC↑ + 张口受限 → ?
> **化脓性腮腺炎**（金葡逆行；脱水/口腔卫生差/NPO → 唾液淤滞）。预防=补液+口腔卫生。详见 [[mistakes/uworld-mistakes_2026-06#^Q4544]]。

> [!question]- 未接种 + 发热 + 双侧腮腺痛肿 + 男孩睾丸痛 + 淀粉酶↑ → ?
> **流行性腮腺炎 Mumps**（疫苗可预防，MMR）。

> [!question]- 慢性反复腮腺肿（不太痛）+ 口干眼干 + 抗 SSA/SSB → ?
> **干燥综合征 Sjögren**（注意长期有腮腺**淋巴瘤 MALT**风险）。

> [!question]- 双侧无痛软肿 + 酗酒/糖尿病/贪食 → ?
> **涎腺病 Sialadenosis**（非炎症性，对因处理）。

> [!question]- 腮腺/下颌下肿物 + 面瘫 / 面麻 / 放射史 → ?
> **恶性肿瘤**（神经侵犯红旗）。影像 + 细针穿刺。详见 [[mistakes/uworld-mistakes_2026-06#^Q4599]]。

> [!tip] 淀粉酶小钩
> 腮腺源性淀粉酶↑见于 **mumps / 化脓性腮腺炎**；别一见淀粉酶高就只想胰腺（唾液腺也产，特异性低 → 见 [[完整笔记/Peixuan分科笔记/消化腺]]）。

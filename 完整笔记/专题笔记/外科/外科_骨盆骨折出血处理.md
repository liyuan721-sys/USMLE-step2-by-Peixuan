# 外科 骨盆骨折出血处理：骨盆带 → 血管栓塞 → 外固定阶梯

> [!info] 用途与来源
> 派生自 Q16331（不稳定骨盆骨折出血 → 先上骨盆带，🔴 反复错）。骨盆骨折是**钝性创伤失血性休克的重要隐匿源**，出血可高达数升却"看不见"。
> **本篇专攻**：血流动力学不稳时的**出血控制阶梯 + 各步指征 + 与 GU（泌尿）损伤评估的先后**。骨盆骨折的**尿道 / 膀胱损伤**评估见姊妹篇 [[完整笔记/专题笔记/外科/外科_GU创伤处理体系]]，本篇不重复。

> [!note]- 缩写对照（全称 中英）
> - **DCO** = damage control orthopedics 损伤控制骨科
> - **ex-fix** = external fixation 外固定
> - **MTP** = massive transfusion protocol 大量输血方案
> - **FAST** = focused assessment with sonography for trauma 创伤超声重点评估
> - **CTA** = CT angiography CT 血管造影
> - **IR** = interventional radiology 介入放射
> - **angioembolization** = 血管栓塞
> - **preperitoneal packing** = 腹膜外骨盆填塞
> - **retrograde urethrography** = 逆行尿道造影

---

## 一、为什么骨盆骨折会休克（机制）

> [!danger] 出血源不止一个，但大头是静脉丛 + 骨面
> **开书样 (open-book) / 垂直剪切 (vertical shear)** 骨折 → 骨盆环破裂、容积增大 → 失去填塞效应 → 出血灌进扩大的盆腔。

```
骨盆骨折出血来源（按占比）
├─ 静脉丛 (venous plexus) 出血  ── 最常见（大头）★
├─ 骨折骨面 (cancellous bone) 渗血 ── 常见
└─ 动脉 (arterial, 如髂内动脉分支) ── 约 10–15%，但来势最凶
```

> [!tip] 这个占比决定了阶梯逻辑
> - 大多数是**静脉 / 骨面出血** → **骨盆带缩小容积 + 压迫**就能止住（所以 binder 是第一步、对多数人有效）。
> - 少数**动脉出血** → binder 压不住 → 需**血管栓塞**。这就是"binder 后仍不稳 → 升级"的原因。

---

## 二、出血控制阶梯（核心）

> [!danger] 铁律：不稳定骨盆骨折 + 血流动力学不稳 → 立即骨盆带，先于任何需转运的检查
> 骨盆带 (pelvic binder) 永远是**第一步**；之后按"对复苏是否有反应 + 出血源"升级。

```
高能创伤 + 不稳定骨盆骨折（开书样/剪切）+ 血流动力学不稳
│
① 立即上骨盆带 pelvic binder（大转子 greater trochanter 水平，不是髂嵴）
│    作用: 缩小骨盆容积 + 稳定骨折 + 压迫静脉丛止血
│    同时: 启动大量输血 MTP；FAST + 胸片 找其他出血源
│
├─ 对 binder + 复苏「有反应」→ 稳定 → CT/CTA 定损伤性质 + 后续处理
│
└─ 对复苏「无反应 / 一过性反应」→ 找持续出血源, 升级:
      ├─ FAST 阳性（腹腔积血）────────► 剖腹 laparotomy（先处理腹腔源）
      ├─ CTA 见动脉外渗/假性动脉瘤 ───► 血管栓塞 angioembolization（IR）
      └─ 静脉/骨面为主 或 无 IR/来不及 ► 腹膜外骨盆填塞 preperitoneal packing + 外固定 ex-fix
```

> [!warning] 三个易混点
> - **外固定 ex-fix ≠ 止动脉血**：它提供**机械稳定 + 减容**（常和 packing 一起在手术室做），但**控制不了动脉出血** —— 动脉源仍要栓塞。
> - **栓塞 vs 填塞的选择**取决于**出血源（动脉→栓塞 / 静脉→填塞）+ 资源（有无 IR、能否转运）**：动脉源 + IR 可及 → 栓塞；持续不稳 / 无 IR → 直接进手术室 packing + ex-fix。
> - **binder 是临时措施**，不是终点；它争取时间到达栓塞 / 填塞 / 手术。

## 三、各步指征速查

| 步骤 | 时机 / 指征 | 作用 | 关键点 |
|---|---|---|---|
| **骨盆带 pelvic binder** | 不稳定骨盆骨折 + 血流动力学不稳，**立即**，先于转运检查 | 缩容 + 压迫静脉丛止血 | 放**大转子**水平；先于 CT / 尿道造影 |
| **大量输血 MTP** | 同时启动 | 复苏（红细胞:血浆:血小板 ≈ 1:1:1）| 控制"致死三联"：低体温 / 酸中毒 / 凝血病 |
| **剖腹 laparotomy** | **FAST 阳性**（腹腔出血）| 处理腹腔内出血源 | 腹腔源优先于盆腔 |
| **血管栓塞 angioembolization** | binder + 复苏后仍不稳 **且 动脉源**（CTA 外渗）| 止动脉血 | 需 IR + 患者能转运 |
| **腹膜外骨盆填塞 preperitoneal packing** | 持续不稳、静脉源、无 IR / 来不及 | 压迫静脉 + 骨面 | 手术室快速完成 |
| **外固定 ex-fix** | 不稳定骨盆需机械稳定（常与 packing 同做）| 稳定骨折 + 减容 | **不控制动脉出血** |

## 四、与 GU（泌尿）损伤评估的先后 —— Q16331 陷阱 ⭐

> [!danger] 血流动力学不稳 → 先止血复苏，GU 造影放稳定后
> 骨盆骨折常伴**尿道损伤**（尿道口血 / high-riding prostate 高位前列腺 / 会阴瘀斑）—— 评估重要，但**不是救命步骤**。
> - **不稳定患者**：先 **骨盆带 + 复苏**；**逆行尿道造影 (retrograde urethrography)** 等需转运 / 非救命的检查**放到稳定之后**。
> - ❌ Q16331 经典错点：被"尿道口血"带去先做**逆行尿道造影**，忽略了**血流动力学不稳者先止血**。
> - 尿道损伤本身的处理（造影先于 Foley、后尿道 → 造瘘延期修）见 [[完整笔记/专题笔记/外科/外科_GU创伤处理体系]]。
> 源：[[mistakes/uworld-mistakes_2026-07#^Q16331-R0718]]

> [!tip] 一句话分层
> **止血 > 诊断**：不稳定 = 先 binder + 输血把命保住；等血流动力学稳了，再做 CT / 尿道造影去"看清楚"。

## 五、记忆钥匙 + 自测

> [!success] 记忆钥匙
> - **不稳定骨盆骨折 + 休克 → 先 binder（大转子水平），先于一切转运检查。**
> - **升级看两件事**：对复苏有无反应 + 出血源（动脉→栓塞 / 静脉·骨面→填塞 + 外固定 / 腹腔→剖腹）。
> - **ex-fix 稳骨折不止动脉血；binder 只是临时。**
> - **止血永远先于 GU 造影。**

> [!question]- Q1. 高能车祸，开书样骨盆骨折，BP 84/52、HR 130，尿道口见血。最佳下一步？
> **先上骨盆带 (pelvic binder) + 启动大量输血**。血流动力学不稳 → 先止血复苏，先于逆行尿道造影（尿道损伤评估重要但非救命，放稳定后）。

> [!question]- Q2. 上骨盆带 + 输血后仍持续低血压，FAST 阴性，CTA 见盆腔动脉造影剂外渗。下一步？
> **血管栓塞 angioembolization（IR）** —— 有动脉源（外渗）+ FAST 阴性排除腹腔源 + binder 压不住 → 栓塞止动脉血。

> [!question]- Q3. 为什么单靠外固定 (ex-fix) 不足以处理持续动脉出血？
> ex-fix 提供**机械稳定 + 减小骨盆容积**（帮静脉 / 骨面止血），但**不能封闭动脉破口** → 动脉出血仍需**栓塞**或手术，常与**腹膜外填塞**联合。

# 钝性创伤脊柱影像指征（Blunt Trauma — Spine Imaging）

> [!info] 用途
> 钝性创伤"**要不要影像 / 影像哪一段 / 平片还是 CT**"的决策速查。派生自 UWorld Q18658 / Q18659（同一 vignette 两问）。
> 缩写约定（首次给全称）：**MVC** = motor vehicle collision（机动车碰撞）；**CT** = computed tomography（计算机断层）；**NEXUS** = National Emergency X-Radiography Utilization Study（国家急诊 X 线应用研究，颈椎低危判定标准）；**C-spine** = cervical spine（颈椎）；**TLS** = thoracolumbar spine（胸腰椎）；**BCVI** = blunt cerebrovascular injury（钝性脑血管损伤）；**LMN/UMN** = 下/上运动神经元。

---

## 一、颈椎：要不要影像？（NEXUS 低危 5 项）

> [!danger] 核心规则
> **NEXUS 五项低危标准必须全部满足才可不做颈椎影像**；**任何一项阳性 → 需颈椎影像**。此外**高能机制**（high-energy mechanism）本身即为影像指征。

```
钝性创伤，评估颈椎
   │
   ▼
NEXUS 5 项低危标准（全阴才可临床清除，不影像）：
   ① 无后正中线脊柱压痛 (no midline spinal tenderness)
   ② 无局灶神经功能缺损 (no focal neurologic deficit)
   ③ 意识状态正常 (normal alertness / mental status)
   ④ 无中毒 (no intoxication)
   ⑤ 无分散注意力的疼痛性损伤 (no distracting injury)
   │
   ├─ 5 项全阴 + 无高能机制 → 临床清除，不影像
   └─ 任一阳性  或  高能机制 → 颈椎 CT（首选）
```

- **高能机制阈值**：高速 MVC、坠落 ≥ 3 m（10 ft）、造成合并闭合性颅脑损伤的创伤等。
- **distracting injury（分散注意力的损伤）** 的典型例子：**长骨骨折（尤其开放性）**、大面积烧伤、内脏损伤——疼痛掩盖颈痛，故不能靠"患者不喊颈痛"排除。

> [!tip] 加拿大颈椎准则（Canadian C-spine rule）
> 与 NEXUS 平行的另一套决策规则，强调高危因素（年龄 ≥ 65、危险机制、肢体麻刺）→ 影像；低危 + 能主动左右转头 45° → 临床清除。CK 常考 NEXUS，二者理念一致：**高危/不满足低危 → 影像**。

## 二、平片还是 CT？

> [!success] 结论
> **颈椎 CT 是创伤筛查首选**（灵敏度 ~98% vs 平片 ~52%），更快更准。
> 影像应在**紧急手术前**完成——因术中可能需经口气管插管 / 颈部操作，须先除外颈椎不稳。

| 手段 | 灵敏度 | 地位 |
|---|---|---|
| **颈椎 CT** | ~98% | **首选筛查**（有任一指征即做）|
| 平片 X 线 | ~52% | 已基本被 CT 取代 |
| MRI | — | 查韧带 / 脊髓损伤（神经缺损但 CT 阴性时补充）|

## 三、发现一处椎骨骨折之后：查全脊柱

> [!danger] 硬规则
> **钝性创伤中发现单节椎体骨折（尤其颈椎）= 影像评估整个脊柱的指征**——**非连续（noncontiguous）第二处椎体骨折风险高达 ~20%**。

- 发现 C 段骨折 → 加做**胸腰椎 (TLS) CT**（反之亦然）。
- 其他 TLS 影像指征：局灶背痛 / step-off、相应神经缺损、意识改变、中毒、distracting injury、高能机制。
- **CT 同样优于平片**。

## 四、创伤性单神经根病 vs 血管损伤（干扰鉴别）

| 表现 | 定位 / 病因 | 处理 |
|---|---|---|
| C5-C6 小关节半脱位 → **腕伸无力 + 拇指感觉丧失** | **C6 神经根病**（根受压，已被骨折/半脱位解释）| 无需额外神经传导 / 血管检查 |
| **偏瘫 / Horner 综合征 / 头颈痛** | 疑 **BCVI**（如颈动脉夹层）→ CT 血管造影 | 有对应症状才查，不能凭根性症状去查颈动脉 |

> [!warning] 易错点
> 局灶根性症状（腕伸弱 / 拇指麻）若已被**椎骨骨折 + 小关节半脱位**解释，就**不再加颈动脉超声 / 神经传导**；颈动脉影像仅在有 BCVI 症状（偏瘫、Horner、头颈痛）时。

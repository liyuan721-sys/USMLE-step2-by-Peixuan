# 外科_头颈鳞癌 HNSCC 总结（HPV 型 vs 烟酒型 + 颈块红旗）

> [!danger] 为什么建这张（反复错触发）
> 2026-06-10 **一周内 2 次没认出头颈鳞癌 HNSCC**：[[mistakes/uworld-mistakes_2026-06#^Q16127]]（耳正常的耳痛 + 烟酒 + 颈淋巴结 → 选了鼓室图）+ [[mistakes/uworld-mistakes_2026-06#^Q16201]]（溃疡扁桃体 + 固定淋巴结 + 多性伴 → 选了放线菌）。
> **共同盲点**：看到"**慢性 + 溃疡 / 固定无痛颈淋巴结**"没有反射到头颈癌。本张把这个盲点焊死。

---

## 一、两型病因对比 ⭐⭐⭐（核心）

| | **HPV(+) 型** | **烟酒型（HPV−）** |
|---|---|---|
| 好发部位 | **口咽**（扁桃体、舌根） | 口腔 / 喉 / 下咽 |
| 危险因素 | **HPV-16**、多性伴、口交 | **烟草 + 酒精（协同 multiplicative）**、槟榔 betel nut、牙列差 |
| 年龄 | **较年轻**（无烟酒史） | 较年长 |
| 标志物 | **p16 阳性**（HPV 替代标志） | p16 阴性 |
| 预后 | **较好**（对放化疗反应好） | 较差 |
| 预防 | **HPV 疫苗**可防 | 戒烟戒酒 |

> [!tip] 一句话分型
> **无烟酒 + 较年轻 + 多性伴 + 口咽（扁桃体/舌根）= HPV 型**；**老烟枪 + 酗酒 + 口腔/喉 = 烟酒型**。HPV 型发病率上升、预后反而好。

---

## 二、红旗与表现 ⭐⭐⭐（最该反射的）

> [!warning] 这些 = 头颈鳞癌，别当感染/良性
> - **颈部肿块：硬、固定、无痛** → **转移淋巴结**（成人此征 = 排癌 until proven otherwise）
> - **单侧扁桃体增大 + 溃疡**（[[mistakes/uworld-mistakes_2026-06#^Q16201]]）
> - **声嘶 > 2 周**（喉癌）
> - **持续咽痛 / 吞咽痛 / 吞咽困难 > 2–3 周**
> - **牵涉性耳痛 referred otalgia**：耳检查**完全正常**的耳痛（经 CN IX/X 投射）（[[mistakes/uworld-mistakes_2026-06#^Q16127]]）
> - 口腔不愈溃疡 / 白斑 leukoplakia / 红斑 erythroplakia（癌前）
> - 口臭 halitosis、咯血、不明体重下降

> [!danger] 成人颈部无痛固定肿块 = 转移癌，先想癌再想其他
> 炎症淋巴结：**软、痛、活动**；恶性转移：**硬、固定、无痛**。**别当良性炎症随访拖延**。

---

## 三、颈部肿块 / 可疑病灶的处理（next step）⭐⭐

```
可见黏膜病灶（扁桃体/口腔/喉溃疡肿物）
   → 直接活检 biopsy + p16/HPV 检测 + 颈影像(CT/MRI) + 全内镜 panendoscopy

成人颈部肿块疑转移（硬/固定/无痛）
   → 细针穿刺 FNA（首选）+ 颈影像 + 喉咽镜找原发
   ✘ 忌首选"切开 / 切除活检 excisional biopsy"
     （播散 + 影响后续颈清扫分期；除非疑淋巴瘤且 FNA 不能定性）
```

> [!tip] next step 钥匙
> - **看得见的黏膜灶** → **活检**（+ 影像 + 内镜找范围）。
> - **只摸到颈块** → **FNA + 找原发**，**不要先切开**。
> - 牵涉性耳痛（耳正常）+ 危险因素 → **柔性喉咽镜**找原发。

## 四、鉴别（甩开骗过你的干扰项）

| 病 | 招牌 | 与 HNSCC 的分水岭 |
|---|---|---|
| **放线菌 Actinomyces**（Q16201 选错） | 慢性无痛肿块 + **黄色硫磺颗粒**窦道、**越组织平面** | 区域淋巴结**少见**；HNSCC 有溃疡 + 固定淋巴结 |
| EBV 单核增多 / 淋巴瘤 | **双侧**淋巴结、发热、较软 | HNSCC **单侧 + 溃疡 + 固定** |
| Lemierre（Fusobacterium necrophorum） | **急性高热** + 颈痛 + IJV 血栓 + 脓毒栓 | HNSCC 慢性渐进、无脓毒 |
| 一期梅毒（Treponema pallidum） | **无痛 chancre**，暴露后数周 | 扁桃体 chancre 罕见、无固定淋巴结 |
| 反应性淋巴结 | **软、痛、活动**、感染后 | 恶性 = 硬、固定、无痛 |

## 五、特例：鼻咽癌 NPC（EBV 相关，别和 HPV 口咽癌混）

> [!info] 鼻咽癌 Nasopharyngeal carcinoma
> - 病因 **EBV**（非 HPV）；**华南 / 东南亚**高发，亚硝胺饮食。
> - 三联表现：**颈部肿块（常首发，上颈后）+ 鼻塞 / 鼻出血 + 单侧浆液性中耳炎**（肿瘤堵咽鼓管 → 成人单侧中耳积液要查鼻咽！）。
> - 可有颅神经受累（侵颅底）。诊断鼻咽镜 + 活检；治疗以**放疗**为主（对放疗敏感）。

---

## 六、治疗速览

> [!info] 分期决定
> - **早期**：手术 **或** 放疗（单一即可）。
> - **进展期**：放化疗（**顺铂 cisplatin** 为基础）± 手术；靶向 **西妥昔单抗 cetuximab（抗 EGFR）**。
> - **HPV(+) 型**预后好，临床试验在探索"降强度"治疗。
> - 鼻咽癌 NPC：**放疗**为主。

# 外科_软骨膜下血肿先切排（耳廓 + 鼻中隔）

> [!danger] 为什么建这张卡
> 2026-06-10 同一个"软骨无血供"盲点**犯了两次**：耳廓血肿选加压（[[mistakes/uworld-mistakes_2026-06#^Q20314]]）、鼻中隔血肿选冰敷（[[mistakes/uworld-mistakes_2026-06#^Q16961]]）。
> 主笔记 [[完整笔记/Peixuan分科笔记/外科]] 此前**无耳鼻外伤内容**，故此卡为新建核心法则。

---

## 一、通用法则（一条原理统两病）⭐⭐

```
软骨【无直接血供】 → 营养全靠软骨膜向软骨【扩散】
   外伤 → 软骨膜与软骨之间积血（血肿）
      血肿物理性【隔断】扩散通路
         → 软骨【缺血坏死】 + 纤维增生
            → 永久畸形

∴ 处理 = 【先切开引流/抽吸清除血肿】 恢复扩散通路
        → 再加压/填塞防复积
   只加压 / 只冰敷 / 只抗生素 = 不够 → 软骨坏死
```

> [!tip] 一句话锁定
> **软骨膜下血肿（耳廓 or 鼻中隔）→ 第一步永远是「切开引流」**；加压/填塞只是引流**之后**的辅助，冰敷 NSAIDs 只减炎性水肿不能替代引流。

---

## 二、耳廓血肿 vs 鼻中隔血肿（平行对照）⭐

| | 耳廓血肿 Auricular hematoma | 鼻中隔血肿 Septal hematoma |
|---|---|---|
| **机制** | 接触运动击打（摔跤/拳击/武术）| 鼻外伤（击面/骨折）|
| **体征** | 耳廓红肿、波动性积血 | 鼻中隔**软、波动性**肿胀（双侧）+ 鼻塞 |
| **触诊钥匙** | 波动 = 血肿 | **软波动 = 血肿；硬 = 中隔偏曲** |
| **不治后果** | 软骨缺血坏死 → **菜花耳 cauliflower ear**（纤维软骨增生）| 软骨坏死 → **中隔穿孔 / 鞍鼻 saddle nose / 鼻瓣塌陷** |
| **处理** | **切开引流/抽吸 + 加压包扎 + 口服抗生素** | **切开引流 + 前鼻填塞 + 抗生素 + ENT** |
| **抗生素覆盖** | 皮肤菌群 + 铜绿假单胞菌 | 皮肤菌群（继发脓肿/感染）|
| **错题** | [[mistakes/uworld-mistakes_2026-06#^Q20314]] | [[mistakes/uworld-mistakes_2026-06#^Q16961]] |

> [!warning] 时间窗
> 血肿可在 **2–3 天内**继发感染/脓肿，且软骨坏死进行性 → **及时（prompt）引流**，并 3–5 天密切随访评估复积/感染。

## 三、易混选项陷阱（两题共用）

> [!danger] 为什么这些"看似合理"的选项错
> - **只加压包扎 / 前鼻填塞（不先引流）**：加压/填塞是引流**后**压迫软骨膜贴软骨用的；血肿没清 → 软骨仍缺血 → 坏死。
> - **冰敷 + NSAIDs**：只减**炎性水肿**，血肿仍在 → 软骨坏死。
> - **只口服抗生素 + 出院**：防继发感染，但**血肿必须先清除**。
> - **观察/自行吸收**：软骨膜下血肿**不会**安全自行吸收 → 畸形。

---

## 四、和"非血肿"区分

| 表现 | 处理 |
|---|---|
| 软、**波动** = 血肿 | **切开引流**（本卡法则）|
| **硬** = 中隔偏曲 / 鼻骨折（无血肿）| 不需急诊切排；按骨折/偏曲处理 |
| 活动性鼻衄 | 止血（填塞 / 必要时栓塞蝶腭动脉），非切排 |

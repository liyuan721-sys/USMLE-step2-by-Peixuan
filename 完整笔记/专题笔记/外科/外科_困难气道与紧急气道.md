# 外科_困难气道与紧急气道（升级阶梯 + CICV → 环甲膜切开）

> [!danger] 为什么建这张
> 兑现 [[mistakes/uworld-mistakes_2026-06#^Q15962]]（HIV 会厌炎插管失败 → 环甲膜切开）的困难气道 TODO。困难气道升级是**通用反射**——会厌炎、血管性水肿、过敏、面颈创伤都用得上，一张焊死。

---

## 一、气道升级阶梯（核心算法）⭐⭐⭐

```
危及生命的气道 / 严重低氧：
① 100% O₂ 球囊面罩 BVM 通气（目标 SpO₂ ≥ 88–90%）
② 气管插管（首选；困难气道用视频喉镜 video laryngoscope 提高可视）
③ 单次插管失败 + 持续严重低氧 / 病情恶化
        ↓ = CICV（Can't Intubate, Can't Ventilate/Oxygenate）
④ 立即外科气道：环甲膜切开 surgical cricothyrotomy
        （由最有经验者操作——优先 ENT / 普外）
```

> [!tip] 关键决策点
> - **困难气道首选视频喉镜插管**（比直接喉镜可视化好）。
> - **单次视频喉镜插管失败 + 严重低氧的恶化患者 → 别反复试 → 直接外科气道**（重复插管只会拖延氧合）。
> - 例外：若失败的是**无视频的直接喉镜** → 可换**视频喉镜**再试一次；已用视频喉镜失败则升级外科。

---

## 二、插管失败后为什么不选这些 ⭐⭐（高频干扰项）

| 选项 | 为什么不行 |
|---|---|
| **重复插管 reattempt** | 恶化、严重低氧者**反复试只拖延氧合** → 单次（视频喉镜）失败即升级外科 |
| **喉罩 LMA（声门上气道）** | LMA 坐在**声门上方** → **绕不过声门上梗阻**（会厌炎、声门上肿胀）；仅在**非梗阻性**插管失败时作通气补救 |
| **无创正压 BiPAP / NIV** | 正压**克服不了固定机械梗阻**（肿胀会厌）；且意识不清/不能保护气道者禁用 |
| **盲探经鼻插管 blind nasotracheal** | 会厌炎肿胀的会厌**必须可视**才能过管；盲插过不去、还可能加重 |

> [!danger] 一句话钥匙
> **声门上梗阻（会厌炎）插管失败 → 只有"绕到梗阻下方"的环甲膜切开有用**；LMA/BiPAP/盲鼻插都在梗阻**上方或之外**，无效。

---

## 三、环甲膜切开要点（成人 vs 儿童）⭐⭐

- **外科环甲膜切开 surgical cricothyrotomy**：切开**环甲膜 cricothyroid membrane** → 在声门/声门上梗阻**下方**建立气道；CICV 的最终手段。
- **儿童（约 <10–12 岁）**：解剖小、环状软骨/喉易损 → **改用针刺环甲膜切开 needle cricothyrotomy + 喷射通气**（不做外科环甲膜切开）。
- **气管切开 tracheostomy**：更确定但**更慢**，不用于"crash"急症（紧急用环甲膜切开，稳定后可转气切）。
- **禁忌/慎用**：喉部断裂/严重喉外伤 → 环甲膜切开可能加重 → 直接气切。

## 四、按情境（同一升级逻辑，触发不同）

| 情境 | 要点 |
|---|---|
| **会厌炎 epiglottitis** | 三脚架/流涎/声嘶 + pooled 分泌物 + stridor；**别压舌激惹**；BVM→视频喉镜→失败+低氧→环甲膜切开（[[mistakes/uworld-mistakes_2026-06#^Q15962]]）|
| **血管性水肿 angioedema**（ACEI / 遗传性 C1-INH） | 进行性唇舌咽喉肿 → **早期建立气道**（纤维支气管镜/清醒插管）；不能插 → 环甲膜切开。遗传性用 C1-INH / 缓激肽拮抗剂 |
| **过敏性休克 anaphylaxis** | **先肌注肾上腺素 epinephrine**；气道将闭 → 早插管；失败 → 外科气道 |
| **面/颈部创伤** | 气道常困难 + 出血 → 视频喉镜/纤支镜；不能插 → 环甲膜切开（喉断裂例外 → 气切）|
| **颈椎损伤** | **轴线固定 in-line stabilization** + 视频喉镜/纤支镜，避免颈部过伸 |

## 五、插管 vs NPPV/HFNC —— 该不该升级到插管 ⭐（气道保护）

> 上面 §一–四是"已决定要插管、但插不进"的升级；本节是另一决策：**低氧/呼吸窘迫时，选无创（HFNC/NPPV）还是直接插管**。派生自 [[mistakes/uworld-mistakes_2026-06#^Q20184]]（溺水嗜睡，我误选 NPPV）。

> [!danger] 插管硬指征（满足任一即插，别试 NPPV）
> - **不能保护气道**：意识↓/嗜睡/昏迷/无 gag → **误吸风险**（NPPV 不防误吸）。
> - **NPPV/HFNC 仍难治的低氧或高碳酸**（氧合/通气达不到目标）。
> - **气道将闭/梗阻**（血管性水肿、会厌炎、严重面颈伤）。
> - 需要气道保护去做有风险转运/操作（如恶化中去 CT）。

| 选项 | 适用 | 禁忌/局限 |
|---|---|---|
| **高流量鼻导管 HFNC / NPPV（CPAP/BiPAP）** | **清醒、能配合、能保护气道**的低氧（减肺不张/V-Q 失配，如溺水轻症、心源性肺水肿、COPD/CHF）| **意识↓/不能保护气道/误吸风险/固定机械梗阻** → 禁用 |
| **气管插管 + 机械通气** | 上述任一插管硬指征 | — |

### 5.1 无创 vs 有创 —— 概念对照 ⭐

| 维度 | **无创 Noninvasive** | **有创 Invasive** |
|---|---|---|
| 含义 | **不突破气道**，经面罩/鼻接口给气 | **导管进入气管**（过声门）|
| 代表 | NPPV：CPAP / BiPAP；HFNC（高流量鼻导管）；普通面罩吸氧 | 气管插管 endotracheal intubation；气管切开/环甲膜切开 |
| 保护气道? | **不保护**（不防误吸）| **保护**（气囊封闭气道，防误吸）|
| 适用 | 清醒、能配合、能保护气道的低氧/高碳酸 | 意识↓/不能护气道、无创失败、需确定性气道 |

> [!info] "有创/invasive" 指的是进气道，不是"开刀"⭐（易误解）
> **划线标准 = 有没有把人工装置放进气道（越过声门），不是切不切皮。**
> - **气管插管**：经口/鼻 → 声门 → 气管的**天然通道**，**不切皮**，但导管进入气道、气囊封闭、绕过咳嗽/声门反射 → 算 **invasive（确定性气道，能防误吸）**。
> - **真正"开刀"的只有外科气道**：环甲膜切开 / 气管切开（颈前切口）。
> - 两条独立的轴：**侵不侵入气道**（面罩否 < 插管是 < 气切是）/ **切不切皮**（面罩否 < 插管否 < 气切是）。气管插管 = **侵入气道了，但还没到切皮**。

> [!warning] 钥匙
> **"能不能保护气道"是分水岭**：嗜睡/昏迷 = 插管；清醒能配合 = 可试 NPPV。NPPV 不保护气道、克服不了固定梗阻。**气道 > 影像/复温**：先稳气道再转运。

## 六、插管后确认位置：capnography 波形判读 ⭐（派生自 Q20145）

> 上面 §一–五是"怎么把管放进去"；本节是"放进去后怎么确认管在气管、没移位"。**Capnography（二氧化碳波形图 capnogram）= 确认气管插管 (ETT, endotracheal tube) 位置最可靠的方法**——测呼气末 CO₂ (ETCO₂, end-tidal CO₂)。

![[{BCE6510A-1172-45C6-95E0-9225061B0F81}.png]]

> [!info] 图说（UWorld 7 种波形，源卡 [[mistakes/uworld-mistakes_2026-06#^Q20145]]）
> Normal 矩形 4 相（α/β + phase 0–3）/ Hyperventilation（ETCO₂↓）/ Hypoventilation（ETCO₂↑）/ Bronchospasm（失 α 角鲨鱼鳍）/ **Tube displacement 管移位（全相骤降/消失）** / CPR（低灌注 ETCO₂↓）/ Apnea（全相消失=平线）。

> [!info] 传感器到底在气道哪个位置测？——ETT 最外端接口（hub / 15 mm 接头）
> **不是插到气管/肺泡里去测，而是在 ETT 最外端接口处（气道开口 airway opening，与呼吸机回路或球囊 BVM 相接处）测"从管子里呼出来的那口气"的 CO₂。**
> 
> ```
> 肺泡 ←→ 气管 ←→ [ETT 管身] ←→ ★接口(hub)★ ←→ 回路/球囊
>                                  ↑ CO₂ 传感器/采样口在这里
> ```
> 
> - **主流式 (mainstream)**：红外测量小室直接卡在接口上，实时无延迟（插管病人）。
> - **旁流式 (sidestream)**：接口接细采样管抽气回监护仪内测，有几秒延迟（可用于未插管者，接鼻氧管采样口）。
> 
> **机制推论**：传感器只"看"经过接口的气体——管尖只要不在通气的肺里（进食管 / 脱管 / 回路脱开 / 无灌注），出口就没 CO₂ → 立刻平线。**这正是"平线 = ETT 在食管"逻辑成立的根本**（胃肠不产 CO₂）。

> [!danger] ⭐ 关键局限：#5 管移位（完全）与 #7 Apnea 在波形上**无法区分**
> capnography 只回答"接口处有没有 CO₂"，**不告诉你为什么没有**。所以**平线（零 CO₂）= 6 种病因，波形完全相同**：
> 
> | 平线 6 因 | |
> |---|---|
> | ① 食管插管 / 脱管（导管移位）| ② 呼吸暂停 / 无通气（Apnea）|
> | ③ 回路脱开 (disconnection) | ④ 完全气道梗阻 |
> | ⑤ 长时间心脏骤停（CO₂ 停产/无灌注）| ⑥ 设备 / 传感器故障 |
> 
> **靠波形之外的信息区分**（这正是 UWorld 出题逻辑——给你生命体征/情境替你定因）：
> - **有脉搏 / 应声 / 自主活动** → 排除骤停 → 锁导管移位入食管（= [[mistakes/uworld-mistakes_2026-06#^Q20145]]）。
> - 呼吸机在送气 + 胸廓不起伏 + 无 CO₂ 返回 → 导管移位 / 回路脱开 / 完全梗阻。
> - 完全无呼吸动作、无机械通气 → 真正 Apnea。
> - **确认动作**：脱开看回路 → 球囊通气看胸廓 → **直接喉镜 (direct laryngoscopy) 看管在不在声门内** → 听诊双肺/上腹。
> 
> **可区分的两种**：① 部分移位（reduction）波形**变矮但还在** ≠ 全平；② CPR/低灌注 ETCO₂ **低但有波形** ≠ 平线（故 CPR 中 ETCO₂ 突跳升 = ROSC 能看出）。
> 
> 一句话：**完全管移位 = 完全 Apnea = 平线，波形等价；谁是谁永远靠生命体征 + 查体 + 直接喉镜定，不靠曲线本身。**

### 6.1 波形 → 含义速查

| 波形 | 含义 |
|---|---|
| 正常矩形 4 相 | ETT 在气管 (trachea)，位置正确、通气良好 |
| **平线 (flat / 零 CO₂)** | **食管插管 / 管移位入食管 / 脱管**；或**长时间心脏骤停**（CO₂ 停产）|
| 失 α 角 + 2/3 相上斜（鲨鱼鳍 shark fin）| 支气管痉挛 (bronchospasm) |
| ETCO₂ 突然↓ | 心输出↓/灌注↓（如心肺复苏 CPR 中）、肺栓塞 (PE) |
| ETCO₂ 突然↑ | 自主循环恢复 (ROSC, return of spontaneous circulation) |

> [!danger] 平线钥匙：零 CO₂ = 食管 / 脱管 / 骤停
> **平线 = 末端完全测不到 CO₂**。胃肠/食管 (esophagus) 不产 CO₂ → 食管插管经典平线。**有生命体征（应声、自主活动、有脉搏）→ 排除骤停 → 锁 ETT 移入食管**。
> 
> **易移位高危**：**无气囊 ETT (uncuffed ETT)** + 儿童（气管到食管距离短）→ 转运颠簸最易滑入食管。

> [!warning] 别混："平线" vs "波形变了但仍有 CO₂"
> - **右主支气管插管 (right mainstem intubation)**：仍能测到 CO₂ → 波形可变但**不会平线**；靠胸片/查体（单侧呼吸音）诊断，不是 capnography。
> - **平线只指向"零 CO₂"= 食管/脱管/骤停**，不要用平线解释右主支气管。

> [!success] 一句话锁定
> **困难气道：BVM(100% O₂) → 视频喉镜插管 → 单次失败 + 低氧 = CICV → 环甲膜切开（成人）/ 针刺(儿童)。**

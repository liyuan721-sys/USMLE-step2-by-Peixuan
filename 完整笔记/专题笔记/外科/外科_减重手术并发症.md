# 减重手术并发症 (Bariatric Surgery Complications)

> 相关：[[完整笔记/专题笔记/外科/外科_HighYield速查总览]]、[[完整笔记/专题笔记/GI/GI_HighYield速查总览]]、[[完整笔记/Peixuan分科笔记/外科]]

## 一、Roux-en-Y 胃旁路解剖机制 (Roux-en-Y Gastric Bypass, RYGB)

RYGB 通过两条机制减重：

- 限制摄入 (restriction)：外科制造一个约 15–30 mL 的小胃袋 (gastric pouch)，早饱、进食量骤减。
- 吸收不良 (malabsorption)：绕过大部分胃、十二指肠及近端空肠，食物与胆胰液分流，仅在 common limb (共同肠段) 汇合后才充分吸收营养。

两个关键吻合口 (anastomoses)——很多并发症就发生在此：

```
食管
  │
[小胃袋 gastric pouch]
  │
  ├─ GJ 吻合口 (gastrojejunal, 胃空肠吻合)  ← 狭窄 / 溃疡好发处
  │
[Roux limb 空肠 (食物臂, 无胆胰液)]
  │
  ├─ JJ 吻合口 (jejunojejunal, 空肠空肠吻合)  ← 胆胰液在此汇入
  │
[Common limb 共同肠段]  ← 营养主要吸收段
```

被绕过的十二指肠是铁、钙吸收主场；胃酸/内因子减少影响 B12——这解释了下文的营养缺乏谱。

## 二、并发症一览对比表

> 记忆主线：进行性机械性梗阻症状（先固体、后连液体都吐）几乎总指向 GJ 吻合口出口问题 → 下一步 EGD（食管胃十二指肠镜 esophagogastroduodenoscopy）。

| 并发症 | 机制 | 时相 | 典型症状 | 下一步 / 处理 |
|---|---|---|---|---|
| 吻合口/胃口狭窄 (stomal / anastomotic stenosis) | GJ 吻合口进行性瘢痕狭窄（局部缺血、溃疡愈合挛缩） | 术后数周至 1 年内 | 进行性恶心、呕吐、反流、吞咽困难；先固体不耐受、后连液体都吐 | EGD（诊断 + 内镜球囊扩张 balloon dilation）；扩张失败 → 手术修整 |
| 边缘溃疡 (marginal ulcer) | GJ 吻合口（空肠侧黏膜）溃疡，可致狭窄样表现或出血/穿孔 | 术后数月内（吸烟、非甾体抗炎药 NSAID、幽门螺杆菌 Helicobacter pylori、大胃袋致酸多为危险因素） | 上腹痛、恶心呕吐、呕血/黑便 | EGD 确诊 → 质子泵抑制剂 (proton pump inhibitor, PPI) + 去除诱因；不做经验性治疗 |
| 倾倒综合征 (dumping syndrome) | 高渗单糖不经胃调控快速倾入小肠，液体内移 + 肠激素释放 | 进食后（早发 15–30 min，或晚发 1–3 h 反应性低血糖） | 餐后腹痛、腹泻 + 低血压、心动过速、出汗、头晕、甚至晕厥 | 饮食调整：避免单糖/精制碳水，少量多餐、干湿分离；顽固者 octreotide (奥曲肽) |
| 胆石病 (cholelithiasis, 胆囊结石) | 快速减重期胆汁淤积、胆固醇过饱和 | 术后数月内快速减重期 | 右上腹绞痛、脂餐后加重 | 右上腹超声 (RUQ ultrasound)；有症状 → 胆囊切除 |
| 内疝 (internal hernia) | 肠系膜缺损处小肠疝入（术后减重、系膜脂肪消失使缺损增大） | 迟发，常术后数月至数年 | 间歇性/进行性腹痛，可进展为小肠梗阻、绞窄 | CT（计算机断层 computed tomography，可为阴性，高度怀疑仍需诊断性腹腔镜探查） |
| 营养缺乏 (nutritional deficiencies) | 十二指肠/近端空肠被绕过 + 胃酸内因子减少 | 慢性，术后数月至数年 | 缺铁性贫血、巨幼贫、骨病、周围神经病等 | 常规补充 + 监测；见下节 |

## 三、营养缺乏谱 (绕过近端小肠的后果)

- 铁 (iron)：十二指肠吸收被绕过 → 缺铁性贫血 (小细胞) 最常见。
- 维生素 B12 / 钴胺素 (cobalamin)：胃酸与内因子减少 → 巨幼细胞性贫血、亚急性联合变性。
- 钙 (calcium) + 维生素 D (vitamin D)：十二指肠/空肠吸收受损 → 继发性甲旁亢、代谢性骨病。
- 硫胺素 / 维生素 B1 (thiamine)：术后频繁呕吐 + 摄入骤减 → Wernicke 脑病风险（补糖前先补 B1）。

## 四、关键鉴别陷阱

> [!danger] 进行性「先固体、后连液体都吐」= 机械性出口梗阻 → 首选 EGD，不要点胃排空扫描
> - 症状学是关键：真正的机械性出口狭窄，患者从固体不耐受进展到连液体都吐、伴反流——这是结构性梗阻的语言。
> - 胃排空扫描 (gastric emptying scan / scintigraphy) 是胃轻瘫 (gastroparesis, 功能性排空障碍) 的首选检查，但：
>   1. 近期 RYGB 已彻底改变胃解剖，扫描结果不可靠、无法定位；
>   2. 未先排除机械性梗阻就做功能性检查，属于顺序错误。
> - 正确路径：可视化出口 → EGD 既能直接看到吻合口狭窄/溃疡，又能当场球囊扩张。诊断 + 治疗一步到位。

> [!warning] 狭窄 vs 边缘溃疡
> 两者都在 GJ 吻合口、都可致呕吐，EGD 都是下一步——但溃疡确诊后加 PPI + 去诱因（戒烟、停 NSAID、查 Hp），狭窄则行内镜扩张。别对疑似溃疡做经验性 PPI 而跳过内镜。

> [!tip] 时相帮你缩小鉴别
> 餐后即刻症状群（腹泻 + 血管运动症状）想倾倒综合征；进行性吞咽困难/呕吐想狭窄；迟发腹痛 + 梗阻想内疝；快速减重期右上腹痛想胆石。

## 五、复习自测

> [!question]- 45 岁女性，Roux-en-Y 胃旁路术后 4 个月，进行性呕吐——起初只吐固体食物，现在连液体也吐，伴反流。下一步最合适的处理是？
> 上消化道内镜 (EGD)。进行性从固体到液体的呕吐 + 反流 = GJ 吻合口机械性出口梗阻（吻合口/胃口狭窄 stomal stenosis）。EGD 既能诊断，又能行球囊扩张治疗。胃排空扫描是胃轻瘫首选，但近期旁路改变解剖使其不可靠，且未排除机械梗阻前不应做功能性检查。

> [!question]- 为什么 RYGB 术后特别容易缺铁和 B12？
> RYGB 绕过十二指肠与近端空肠（铁的主要吸收部位），同时小胃袋使胃酸和内因子分泌减少（B12 吸收依赖胃酸释放蛋白结合的 B12 及内因子）。故缺铁性（小细胞）贫血最常见，B12 缺乏致巨幼贫。

> [!question]- 减重术后患者餐后 20 分钟出现腹痛、腹泻、心悸、出汗、头晕，机制与处理？
> 倾倒综合征 (dumping syndrome)。高渗单糖快速倾入小肠引起液体内移与肠激素释放。处理为饮食调整：避免单糖/精制碳水、少量多餐、干湿分离；顽固病例可用 octreotide。

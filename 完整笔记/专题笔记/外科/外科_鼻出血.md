# 鼻出血（Epistaxis）

> [!info] 一句话定位
> 本卡是**鼻出血 (epistaxis) 的完整专题**：① **解剖分类**（前部 anterior vs 后部 posterior）② **急性处理流程**（压迫 → 血管收缩 → 烧灼 → 填塞 → 后鼻球囊/栓塞）③ **全病因鉴别**（局部解剖 / 系统性出血病 / 血管全身病）。
> Step 2 两条主线：**一边是"现在怎么止血"（管理）**，**一边是"为什么反复出血"（病因）**。本卡两条都覆盖。

> [!note]- 本篇缩写速查（首次全称中英）
> - **HHT** = hereditary hemorrhagic telangiectasia 遗传性出血性毛细血管扩张症（= Osler-Weber-Rendu 综合征）
> - **vWD** = von Willebrand disease 血管性血友病（血友病 von Willebrand 因子缺陷）
> - **ITP** = immune thrombocytopenic purpura 免疫性血小板减少性紫癜
> - **GPA** = granulomatosis with polyangiitis 肉芽肿性多血管炎（旧称 Wegener 韦格纳肉芽肿）
> - **JNA** = juvenile nasopharyngeal angiofibroma 青少年鼻咽纤维血管瘤
> - **NPC** = nasopharyngeal carcinoma 鼻咽癌
> - **vWF** = von Willebrand factor 血管性血友病因子
> - **PT/PTT** = prothrombin time / partial thromboplastin time 凝血酶原时间 / 部分凝血活酶时间
> - **INR** = international normalized ratio 国际标准化比值
> - **NSAID** = nonsteroidal anti-inflammatory drug 非甾体抗炎药
> - **DOAC** = direct oral anticoagulant 直接口服抗凝药
> - **AVM** = arteriovenous malformation 动静脉畸形
> - **ANCA** = antineutrophil cytoplasmic antibody 抗中性粒细胞胞浆抗体（c-ANCA / anti-PR3 = 抗蛋白酶 3 抗体）
> - **EBV** = Epstein-Barr virus EB 病毒
> - **DDAVP** = desmopressin 去氨加压素（促血管内皮释放储存的 vWF）
> - **CKD** = chronic kidney disease 慢性肾病
> - **TSS** = toxic shock syndrome 中毒性休克综合征
> - **SPA** = sphenopalatine artery 蝶腭动脉

## 一、解剖分类：前部 vs 后部 ⭐⭐⭐

> [!tip] 第一刀分前后——决定处理路径与风险
> **前部（anterior，约 90%）** 多良性、好压迫；**后部（posterior）** 量大、有气道风险，需后鼻填塞/住院。

| | **前部 anterior（~90%）** | **后部 posterior（少见但凶）** |
|---|---|---|
| 责任血管 | **Kiesselbach 丛（Little 区，前鼻中隔）** | **蝶腭动脉 SPA 分支** |
| 典型人群 | 儿童 / 年轻人；抠鼻、干燥、外伤 | 老年 + 高血压 + 抗凝药 |
| 出血特点 | 单侧、可见前鼻出血点、易压迫止 | 量大、**血流入咽部**、可双侧、难定位 |
| 风险 | 低 | **误吸 / 气道阻塞、失血** |
| 处理倾向 | 压迫 + 烧灼 ± 前鼻填塞 | **后鼻填塞/球囊 + 住院 + ENT** |

## 二、急性鼻出血处理流程 ⭐⭐⭐

```
急性鼻出血 (acute epistaxis)
│
├─ ① 评估：气道/血流动力学 + 出血量 + 前/后 + 抗凝药·出血病史
│
├─ ② 一线（多数前部）：坐位身体前倾 + 捏住鼻翼(alae)持续压迫 10–15 min + 冰敷
│      （前倾防误吸；压迫 Kiesselbach 丛）
│
├─ ③ 局部血管收缩：oxymetazoline 羟甲唑啉（±利多卡因 lidocaine）
│
├─ ④ 仍出血 + 前鼻可见出血点：化学烧灼 silver nitrate 硝酸银 或 电凝
│      （单侧烧灼，勿双侧 → 防鼻中隔穿孔）
│
├─ ⑤ 烧灼无效 → 前鼻填塞 (anterior packing)：可吸收材料 / 纱条 / 鼻球囊
│
└─ ⑥ 后部出血 / 前鼻填塞失败 → 后鼻填塞或球囊导管
       → 收入院 + 气道监测 + ENT 会诊
       → 难治：蝶腭动脉栓塞 (embolization) 或结扎 (ligation)
```

> [!warning] 三个高频处理陷阱
> - **体位**：让患者**坐位前倾**（不是仰头）——仰头会把血咽下/误吸，且无法判断出血量。
> - **化学烧灼只烧单侧**：双侧硝酸银烧灼 → **鼻中隔穿孔 (septal perforation)** 风险。
> - **填塞留置**：传统会给**预防性抗生素**防 **TSS（中毒性休克综合征）/ 鼻窦炎**（证据有争议，但考点常出 TSS 预防）。

> [!tip] 全身层面别忘
> 控制血压、**复查并调整抗凝/抗血小板药**、纠正凝血异常（必要时维生素 K / 鱼精蛋白 / 血制品）。**反复或难止 → 转入下面的病因鉴别找根因。**

## 三、病因第一刀分诊框架 ⭐⭐⭐

```
鼻出血 (epistaxis)
│
├─ 单侧 + 局部症状(鼻塞/恶臭分泌物/可见肿块)
│   └─► 解剖/局部原因 → 异物 / JNA / NPC / 外伤 / 可卡因
│
├─ 双侧 + 其他部位出血(牙龈/瘀点/月经过多/拔牙渗血)
│   └─► 系统性出血病 → vWD / ITP / 白血病 / 血友病 / 抗凝药
│
├─ 老年 + 大量难止(后部 posterior)
│   └─► 高血压 Hypertension
│
└─ 鼻血 + 鞍鼻 + 肺(咯血)+ 肾(血尿)
    └─► GPA(肉芽肿性多血管炎)
```

> [!tip] USMLE 反射口诀
> **单侧看解剖,双侧看血液;老人想血压,鞍鼻想 GPA;自幼 + 毛细血管扩张 + 家族史 = HHT。**

## 四、局部 / 解剖原因（Local causes，多为**单侧**）

| 病因 | Stem 钩子 | 关键处理 / 陷阱 |
|---|---|---|
| **抠鼻 digital trauma + 干燥空气** | 儿童最常见;冬天暖气房;**Kiesselbach 丛(前部)** | 局部压迫 + 保湿;最常见、最良性 |
| **鼻腔异物 nasal foreign body** | 幼儿**单侧**鼻出血 + **恶臭脓性单侧分泌物** | 经典儿科题;取出异物 |
| **青少年鼻咽纤维血管瘤 JNA** | **青春期男孩** + 反复鼻出血 + 单侧鼻塞 | 良性但血供极丰富;**禁活检/盲检(大出血)**;增强 CT/MRI 诊断 |
| **鼻咽癌 NPC** | **华南/EBV 相关**;鼻血 + 颈部包块 + **单侧浆液性中耳炎(传导性聋)** | 颈部转移淋巴结常为首发;鼻内镜 + 活检 |
| **可卡因 cocaine 吸入** | 反复鼻血 + **鼻中隔穿孔(septal perforation)** | 血管收缩 → 缺血坏死;见 [[完整笔记/专题笔记/物质单药/psych_物质_Cocaine]] |

> [!warning] 单侧 + 进行性 = 排肿瘤
> 青春期男孩单侧反复鼻血 + 鼻塞 → **JNA**(别盲目活检);成人单侧鼻血 + 颈块 + 单耳积液 → **NPC**。"单侧 + 进行性 + 局部占位症状"必须排肿瘤,不能只当抠鼻。

## 五、系统性出血病（Systemic — 多为**双侧、反复 + 其他部位出血**）⭐⭐⭐

> [!danger] 核心反射
> 鼻出血 + **其他黏膜皮肤出血**（牙龈渗血、瘀点 petechiae、瘀斑、月经过多 menorrhagia、拔牙/术后渗血）→ 想**血液系统**,不是局部!这类出血叫 **mucocutaneous bleeding(黏膜皮肤型出血)** = 血小板/vWF 问题的招牌。

### 5.1 黏膜皮肤型出血（platelet / vWF 缺陷）

| 病 | Stem 钩子 | 实验室 |
|---|---|---|
| **血管性血友病 vWD** | **最常见遗传性出血病**;鼻血 + 月经过多 + 拔牙渗血 + 家族史(常显) | **PTT 可↑(vWF 载 VIII)** + 出血时间↑;血小板数正常;Ristocetin 试验异常 |
| **免疫性血小板减少 ITP** | 儿童**病毒感染后**孤立性血小板减少;成人慢性 | **孤立性血小板↓**;PT/PTT 正常;余血象正常 |
| **白血病 leukemia** | 鼻血 + **疲乏 + 发热 + 骨痛** + 全血细胞异常 | 血小板↓ + 母细胞;需骨髓检查 |
| **药物**：阿司匹林 aspirin / NSAID | 抗血小板 → 黏膜出血倾向 | 血小板数正常、功能↓ |

### 5.2 凝血因子型出血（coagulation factor 缺陷，**深部/关节出血为主**）

| 病 | Stem 钩子 | 实验室 |
|---|---|---|
| **血友病 Hemophilia A/B** | **男孩 + 关节出血(hemarthrosis)/深部血肿**;鼻血非主征 | **PTT↑, PT 正常**;X 连锁隐性 |
| **肝病 liver disease** | 凝血因子合成↓ + 脾亢血小板↓ | PT↑(先) → PT/PTT 均↑ |
| **维生素 K 缺乏 vitamin K deficiency** | 新生儿/吸收不良/抗生素 | PT↑ 为主 |
| **华法林 warfarin / DOAC 过量** | 抗凝药史 | warfarin → PT/INR↑ |

> [!tip] 黏膜 vs 深部:一眼分两类
> **黏膜皮肤出血(鼻血/牙龈/瘀点/月经多)= 血小板 or vWD**;**深部/关节出血(hemarthrosis/肌肉血肿)= 凝血因子(血友病)**。vWD 横跨两者(因 vWF 既助血小板黏附又载 VIII 因子),所以 vWD 既有黏膜出血**又可能 PTT↑**——这是高频考点。

### 5.3 易混陷阱:黏膜出血但**三项凝血都正常** ⭐

> [!warning] 尿毒症出血(uremic bleeding)——黏膜出血却查不出异常
> **CKD(慢性肾病 chronic kidney disease)** 的尿素毒素抑制血小板功能 → 黏膜皮肤出血(鼻血/瘀斑),但 **血小板数正常 + PT 正常 + PTT 正常 + 仅出血时间(bleeding time)↑**。看到"有出血、凝血四项却正常 + 肾衰病史"别漏 CKD。治疗:**去氨加压素 DDAVP**(±透析、雌激素)。

> [!tip] DDAVP(去氨加压素 desmopressin)的"三用"——统一记忆钩子
> 机制 = 促血管内皮释放储存的 **vWF** → 升 vWF + VIII 因子。三个适应证:① **轻型 vWD(Type 1)** ② **轻型血友病 A(hemophilia A)** ③ **尿毒症出血(CKD 血小板功能障碍)**。三个"出血时间相关"病共用一把钥匙。

## 六、血管 / 全身病

| 病 | Stem 钩子 | 分水岭 |
|---|---|---|
| **遗传性出血性毛细血管扩张症 HHT** | **自幼反复鼻血 + 口唇/指端毛细血管扩张 + 内脏 AVM + 家族史(常显)** | 三联 + 缺铁贫血;详见 [[完整笔记/专题笔记/pulmonary/pulmonary_咯血鉴别与HHT]] |
| **高血压 Hypertension** | **老年 + 大量后部(posterior)鼻出血、难止** | 后鼻孔出血、量大;控压 + 后鼻填塞 |
| **肉芽肿性多血管炎 GPA** | 反复鼻血 + **鞍鼻畸形(saddle nose)** + 鼻黏膜溃疡/结痂 + **肺(咯血)+ 肾(血尿/新月体肾炎)** | **c-ANCA / anti-PR3 阳性**;肺-肾受累是分水岭 |

> [!warning] 高频陷阱:HHT vs GPA(都有鼻血 + 肺出血)
> **HHT**:自幼鼻血 + 毛细血管扩张 + 连续性肺杂音 + 无钙化光滑肺结节 + **尿正常**。
> **GPA**:破坏性鼻窦炎/鞍鼻 + 弥漫肺泡出血 + **镜下血尿(肺-肾综合征)** + c-ANCA。
> **尿正常基本否定 GPA**。详细对照见 HHT 专题卡 §三。

## 七、Memory Hook

```
鼻出血两条线：①怎么止血(管理) ②为什么出血(病因)
解剖：前部(Kiesselbach丛,90%,好压迫) vs 后部(蝶腭动脉,老年/高压,气道险)
止血阶梯：坐位前倾压鼻翼10-15min → 羟甲唑啉 → 硝酸银烧灼(单侧!) → 前鼻填塞 → 后鼻球囊+住院 → 栓塞
填塞陷阱：单侧烧灼防穿孔 ; 填塞留置防TSS给抗生素
病因第一刀 = 单侧/双侧 + 有无其他部位出血
单侧+局部症状 → 解剖(异物/JNA/NPC/可卡因/外伤)
双侧+黏膜出血 → 血液(vWD最常见 / ITP / 白血病)
黏膜出血=血小板/vWD ; 深部关节出血=凝血因子(血友病)
vWD横跨两类:既黏膜出血又PTT↑(vWF载VIII因子)
黏膜出血但PT/PTT/血小板三项全正常 = 尿毒症出血(CKD),治DDAVP
DDAVP三用:轻型vWD / 轻型血友病A / 尿毒症出血
老年+大量后鼻血 = 高血压
鞍鼻+肺+肾(血尿) = GPA(c-ANCA) ; 尿正常→排GPA
自幼鼻血+毛细血管扩张+AVM+家族史 = HHT
青春期男孩单侧鼻血+鼻塞 = JNA(禁活检) ; 成人鼻血+颈块+单耳积液 = NPC
```

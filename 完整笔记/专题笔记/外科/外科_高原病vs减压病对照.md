# 高原病 HAI（High Altitude Illness）vs 减压病 DCS（Decompression Sickness）对照（环境压力急症）

> [!info] 用途
> 两个都和"压力变化"有关、都可能用**高压氧 HBOT（Hyperbaric Oxygen Therapy，hyperbaric O₂）**，但**方向相反**、机制不同、根本治疗不同 —— 最容易在 stem 里混。本卡一张表锁死 + 各自细分。
> 同类环境急症见 [[完整笔记/专题笔记/外科/外科_低体温分级与复温]]。

---

## §零 一句话总分（先锁大方向）⭐

| 维度 | 高原病 HAI | 减压病 DCS |
|---|---|---|
| **诱因** | **上升到高海拔**（>2500m）| **从深处上升**（潜水后 / 飞行）|
| **核心物理** | 低压性缺氧 hypobaric hypoxia | Henry 定律：溶解的 **氮气 N₂ 析出成气泡** |
| **罪魁气体** | **缺 O₂**（氧不够）| **N₂ 气泡**（氮气太多） |
| **起病** | 数小时–数天（登高后）| 上浮后 **数分钟–数小时** |
| **根本治疗** | **下降 (descent) + O₂** ⭐ | **高压氧 (HBOT) 再加压** ⭐ |
| **特异药** | 乙酰唑胺 acetazolamide / 地塞米松 dexamethasone / 硝苯地平 nifedipine | 无特异药（纯 O₂ + 再加压）|

> [!danger] 两个最快钥匙
> - **登山 / 高海拔 + 缺氧症状** → HAI → **下降 + 氧**
> - **潜水上浮后 + 关节痛/神经症状** → DCS → **100% O₂ + 高压氧**

---

## §一 高原病 HAI —— 低压缺氧三兄弟

### 1.1 三型谱（由轻到重）

| 型 | 全称 | 关键表现 | 钥匙 |
|---|---|---|---|
| **AMS** | 急性高原病 Acute Mountain Sickness | **头痛** + 恶心/厌食/乏力/头晕/失眠 | 登高 6–12h，自限 |
| **HACE** | 高原脑水肿 High-Altitude Cerebral Edema | AMS 进展 → **共济失调 ataxia** + 意识改变 + 昏迷 | **ataxia = 早期红旗** ⭐ |
| **HAPE** | 高原肺水肿 High-Altitude Pulmonary Edema | **静息呼吸困难** + 咳嗽 + 粉红泡沫痰 + 湿啰音 + 低氧 | **高原死亡最常见原因** ⭐ |

### 1.2 机制

```
低压性缺氧 (hypobaric hypoxia)
   ├─ 脑：缺氧 → 脑血管扩张 → ICP(颅内压 intracranial pressure)↑ + 血管源性水肿 → AMS / HACE
   └─ 肺：缺氧 → 不均匀肺血管收缩 → 局部肺毛细血管压↑ → 渗漏 → HAPE
                                              （非心源性肺水肿 noncardiogenic）
```

### 1.3 预防 + 治疗 ⭐

| 场景 | 处理 |
|---|---|
| **预防** | **缓慢登高** + **乙酰唑胺 acetazolamide**（碳酸酐酶抑制剂 → 代谢性酸中毒 → 刺激通气）|
| **AMS** | 停止继续登高 + 乙酰唑胺 + 对症；恶化则下降 |
| **HACE** | **立即下降** + **地塞米松 dexamethasone** + O₂ + 高压氧袋 (Gamow bag) |
| **HAPE** | **下降** + O₂ + **硝苯地平 nifedipine**（降肺动脉压）±**西地那非 sildenafil**；保暖 |
| **所有型通用根本** | **下降 (descent) + 吸氧** ⭐ |

> [!tip] 药物分工记忆
> **乙酰唑胺管"高"（脑/AMS 预防+治）；地塞米松管"脑"（HACE）；硝苯地平管"肺"（HAPE）。** 但**任何严重型,下降永远是第一位**。

> [!danger] 利尿剂（呋塞米 furosemide）是 HACE/HAPE 的陷阱选项（Q20203）
> HACE / HAPE 的水肿都是**非心源性 / 非容量过载**：
> - **HACE** = 缺氧诱导脑血管扩张 + 毛细血管渗漏（血管源性 vasogenic）。
> - **HAPE** = 不均匀肺血管收缩 → 局部毛细血管压↑ → 渗漏（非心源性）。
>
> → **呋塞米 furosemide 无效甚至有害**。高原患者反而常**容量不足**（碳酸氢盐利尿 bicarbonate diuresis + 液体向血管外转移 + 醛固酮 aldosterone 抑制）。**别把脑/肺水肿当"水多了"上利尿** —— 答案是地塞米松（HACE）/ 硝苯地平（HAPE）+ 下降。

> [!info]- 乙酰唑胺 acetazolamide 治高原反应的机制（通气悖论 + 解刹车）
> **它是碳酸酐酶抑制剂（carbonic anhydrase inhibitor），靠制造代谢性酸中毒，解除身体给呼吸踩的刹车。**
>
> **① 先懂"通气悖论"**
> ```
> 登高 → 低压性缺氧 hypobaric hypoxia
>    → 反射性过度通气 hyperventilation 想多吸 O₂
>    → 呼出大量 CO₂ → 呼吸性碱中毒 respiratory alkalosis（pH↑）
>    → 化学感受器感到碱中毒，反而【抑制呼吸】← 自己踩刹车
>    → 通气上不去 → 缺氧纠正不足 → AMS
> ```
> 矛盾：你**需要**拼命喘对抗缺氧，但喘出的碱中毒**反过来抑制呼吸**。
>
> **② 乙酰唑胺解开这个结**
> ```
> 抑制肾近端小管碳酸酐酶 → HCO₃⁻（碳酸氢根）重吸收↓ → 经尿排出 bicarbonaturia
>    → 代谢性酸中毒 metabolic acidosis（pH↓）→ 抵消呼吸性碱中毒
>    → 松开"碱中毒刹车" → 允许持续过度通气 → 通气↑ → 氧合↑ → 加速习服 acclimatization
> ```
> **机制链一句话**：抑碳酸酐酶 → 肾排 HCO₃⁻ → 代谢性酸中毒 → 解除碱中毒对呼吸的抑制 → 通气↑。**不是直接给氧，是让你自己的呼吸驱动恢复工作。**
>
> **③ 附带好处**：改善夜间周期性呼吸（periodic breathing）/ 轻度利尿（减体液潴留）/ 降脑脊液（CSF）生成。
>
> > [!warning] 考试要点
> > - 用途：AMS **预防 + 治疗**（登高前 1 天开始预防最佳）；重型仍靠下降 + 地塞米松/硝苯地平
> > - 副作用：**手脚麻刺 paresthesias**、碳酸饮料变"没气怪味"、多尿；**磺胺过敏 sulfa allergy 慎用**（乙酰唑胺是磺胺衍生物）
> > - 迷惑项：别选"补碳酸氢钠 sodium bicarbonate" —— 会加重碱中毒，方向相反

## §二 减压病 DCS —— 氮气冒泡

### 2.1 机制（Henry 定律：装载 loading vs 析出 release）⭐

```
① 装载 loading ← 只在【深处高压】发生
   潜水到水下 → 环境压↑ → Henry 定律：更多 N₂ 溶进组织/血液

② 析出 release ← 任何【压力下降】触发
   压力骤降 → 溶解的 N₂ 来不及经肺排出 → 组织/血管内【析出成气泡】
      → 阻塞微血管 + 机械压迫 + 炎症
   两种压降都算：a) 潜水快速上浮  b) 潜水后飞行/登高（舱压更低，再逼出残余 N₂）
```

> [!warning] 关键：飞行不是独立的 N₂ 来源
> **N₂ 是潜水（深处高压）时装进去的；飞行只是再踩一脚"减压"。**
> - **健康人单纯坐飞机（没潜水）一般不 DCS** —— 体内无多余 N₂ 负荷，正常客舱压降逼不出气泡。
> - 危险的是 **"潜水后过早飞行"**：刚装一身 N₂ 没排净 + 飞行压降 → 逼出残余 N₂ → 诱发/加重 DCS。
> - → **"潜水后 18–24h 勿飞行"**铁律即源于此。
> - 一句话：**压力下降 = 析出的唯一开关；但要先有潜水装载。**

### 2.2 分型

| 型 | 累及 | 表现 |
|---|---|---|
| **I 型（轻）** | 关节 / 皮肤 / 淋巴 | **关节痛 "the bends"**、皮肤花斑 (cutis marmorata)、瘙痒 |
| **II 型（重）** | 神经 / 肺 / 内耳 | **脊髓**（截瘫、膀胱功能障碍）、**"the chokes"**（肺,胸痛呼吸困难）、内耳眩晕 |

> [!warning] 鉴别近亲：动脉气体栓塞 AGE（Arterial Gas Embolism）
> **屏气快速上浮 → 肺气压伤 → 气体直接进肺静脉 → 体循环栓塞**（脑卒中样、即刻发作）。比 DCS 更急更重；处理同 —— **立即高压氧**。

### 2.3 治疗 ⭐

```
① 100% O₂（加速 N₂ 洗出 washout）
② 高压氧 (HBOT) 再加压 —— 确证根本治疗（缩小气泡 + 供氧）
③ 仰卧位 + 补液
④ 潜水后 18–24h 内勿飞行（防诱发）
```

> [!danger] DCS 没有"特异药"
> 治疗核心是**物理性的：吸氧 + 高压氧再加压**,不是某个药。别在选项里找"某某药" —— 答案是 **hyperbaric oxygen**。

## §三 易混点对照（考试陷阱）

| 看到... | 想 | 别选 |
|---|---|---|
| 登山者 4000m + 头痛 + **走路不稳** | **HACE** → 下降+地塞米松 | 别只当普通 AMS |
| 登山者 + **静息气促 + 粉红泡沫痰** | **HAPE** → 下降+O₂+硝苯地平 | 别当心源性肺水肿上利尿 |
| 潜水员上浮 1h 后 **关节痛 + 皮肤花斑** | **DCS I 型** → O₂+高压氧 | — |
| 潜水员**屏气快速上浮后即刻 LOC/抽搐** | **AGE 动脉气栓** → 立即高压氧 | — |
| 潜水后**当天坐飞机**发病 | DCS（低压诱发 N₂ 析出）| — |

> [!success] Memory Hook
> **HAI = 缺氧(上山)→ 下来吸氧；DCS = 冒泡(上浮)→ 进舱加压。** 一个"往下走"、一个"进高压舱";一个缺 O₂、一个多 N₂。高压氧两边都可能用,但**只有 DCS/AGE 是确证根本**,HAI 里高压氧只是辅助、下降才是王道。

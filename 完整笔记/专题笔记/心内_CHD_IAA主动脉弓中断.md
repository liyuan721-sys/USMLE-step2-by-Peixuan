# IAA — Interrupted Aortic Arch（主动脉弓中断）

> [!info] 一句话定位
> 升主动脉与降主动脉**完全断开**（不是狭窄）→ 下半身血流完全依赖 PDA
> **22q11.2 / DiGeorge 三大 conotruncal 心脏畸形之一**（Type B 关联 50-90%，最高）⭐⭐

---

## 1. 黄金诊断三联

> [!tip] Stem 模式
> **新生儿 PDA 关闭后突发休克 + 下肢脉搏消失 + 差异性紫绀 + DiGeorge 特征（腭裂 / 低钙 / 反复感染）** = IAA-B
>
> 救命：**PGE₁ 维持 PDA** → 一期主动脉弓重建 + 修 VSD

---

## 2. 解剖核心 — "弓断了"

> [!info] 定义
> **升主动脉与降主动脉之间完全断开**（连续性中断）
> 不是狭窄（那是 Coarctation），是**真正断开**
> 下半身血流完全依赖 PDA

### 三种分型（按断开位置）

```
正常主动脉弓：
                 ┌── 头臂干 → 右锁骨下 + 右颈总
升主动脉 ── 弓 ──┤── 左颈总
                 └── 左锁骨下 ── 降主动脉

IAA 三型：

Type A：断在左锁骨下**远端**（~30%）
     头臂干 ─ 左颈总 ─ 左锁骨下 ┊ 降主动脉

Type B：断在左颈总和左锁骨下之间 ⭐ 最常见 (~70%)
     头臂干 ─ 左颈总 ┊ 左锁骨下 ─ 降主动脉
                    ↑
                  这里断

Type C：断在头臂干和左颈总之间（罕见）
     头臂干 ┊ 左颈总 ─ 左锁骨下 ─ 降主动脉
```

| 分型 | 频率 | 22q11.2 关联 | 临床特征 |
|---|---|---|---|
| **Type A** | ~30% | 弱 | 左锁骨下远端中断 |
| **Type B** ⭐ | **~70%** | **50-90%** ⭐⭐ | **左右上肢 SpO₂ 不一致** |
| **Type C** | < 5% | 中 | 罕见 |

---

## 3. 病理生理 — 为什么 PDA 关闭就死

```
胎儿期：
PDA 开放 + 弓中断
        ↓
RV → 肺动脉 → PDA → 降主动脉 → 下半身（OK）

出生后（PDA 还开着）：
LV → 升主动脉 → 头 + 上肢（OK）
RV → PDA → 降主动脉 → 下半身（OK）
        ↓
临床看似稳定

PDA 关闭（生后 24-72h）：
RV → PDA（关）✗ → 下半身**完全无血**
        ↓
下半身急性缺血 + 代谢性酸中毒
        ↓
下肢脉搏消失 + 灰白色 + 休克 + 死亡
```

> [!danger] 致命时间窗
> 出生时**可能看起来"完全正常"**（PDA 开）
> 24-72 小时后 PDA 关闭 → **突发休克 + 下半身脉搏消失**

## 4. 临床表现 — Differential Cyanosis & BP Discrepancy ⭐⭐

### 上下半身血源不同的解剖图（Type B）

```
         LV → 升主动脉
              ↓
         ┌────┴────┐
         ↓         ↓
       头臂干    左颈总     ← 氧合血（SpO₂ 高）
       ↓          ↓
   右上肢       左颈
   右脑         + 头
                            
              （Type B 断点）
                ┊
         RV → PA → PDA → 降主动脉
                          ↓
                  ┌───────┴───────┐
                  ↓               ↓
              左上肢            下半身
            (左锁骨下)         (双下肢)
              ↓                  ↓
           缺氧血             缺氧血（SpO₂ 低）
```

### 关键体征

> [!warning] Differential Cyanosis（差异性紫绀）⭐⭐
> **上半身红润 + 下半身紫绀**
> 上下肢 SpO₂ 差异 > 10%

| 部位 | Type A | Type B |
|---|---|---|
| 右上肢 SpO₂ | 高 | 高 |
| 左上肢 SpO₂ | 高 | **低** ⭐ |
| 下肢 SpO₂ | 低 | 低 |

> [!tip] Type B 鉴别神技
> **左右上肢 SpO₂ 不一致**（左低右高）→ Type B IAA
> 因为左锁骨下断在 PDA 侧

### BP & 脉搏体征

| 体征 | 表现 |
|---|---|
| 右上肢 BP | **高**（LV → 头臂干）|
| 下肢 BP | **极低 / 测不到** |
| 下肢脉搏 | **弱 / 消失** ⭐ |
| 股动脉搏动 | **消失** ⭐ |

## 5. Stem 关键词识别

| Key Stem Clue | 中文 |
|---|---|
| "Newborn looks **well at birth**" | 出生时看似正常 |
| "**Sudden shock at day 2-3**"（PDA 关闭）| 2-3 日龄突发休克 ⭐ |
| "**Absent femoral pulses**" | 股动脉搏动消失 ⭐ |
| "**Differential cyanosis**" | 差异性紫绀（上红下紫）|
| "**Upper > lower extremity BP**" | 上肢血压 >> 下肢 |
| "**Cleft palate + hypocalcemia + recurrent infections**" | DiGeorge 三联 ⭐⭐ |
| "**FISH probe for 22q11.2 deletion**" | DiGeorge 确诊 |
| "Often with **VSD** + **bicuspid aortic valve**" | 常合并 VSD + BAV |

## 6. 诊断流程

```
新生儿突发休克 + 下肢脉弱
        ↓
①BP 4 肢测量
   → 上肢 > 下肢 → 主动脉弓水平病变
        ↓
②上下肢 SpO₂
   → 差异 > 10% → 差异性紫绀（PDA 右→左分流）
        ↓
③Echocardiography（确诊）⭐
   → 主动脉弓中断 + 分型
        ↓
④染色体检查
   → FISH for 22q11.2 deletion
        ↓
⑤代谢检查
   → 血钙（低）+ T 细胞亚群（缺陷）
```

---

## 7. 治疗 — 三步走 ⭐⭐

### Step 1：PGE₁ 救命

> [!danger] 急救第一步
> **PGE₁ IV** 立即维持 PDA 开放 → 恢复下半身灌注 → 纠正休克

### Step 2：稳定 + DiGeorge 评估

- 机械通气
- 纠正酸中毒
- **补钙**（DiGeorge → 低钙）⭐
- **输血必须照射血液制品**（防 GVHD）⭐⭐
- **避免活疫苗**

### Step 3：一期主动脉弓重建

- 主动脉弓重建（端端吻合 / 补片）
- 同时修补 VSD
- 结扎 PDA

**预后**：手术死亡率 5-15%，长期生存率 70-80%

## 9. ⭐⭐ Conotruncal 畸形三巨头 vs IAA

| 病 | 22q11.2 关联 | 数字法 | 主要特征 |
|---|---|---|---|
| **Truncus arteriosus** | ~50% | 1 | 共干 + VSD + **轻紫绀重心衰** + 洪脉 |
| **TOF** | 15-20% | 4 | **Tet spells** + 蹲下缓解 + Boot heart |
| **IAA (B 型)** | **50-90%** ⭐⭐ | — | **PDA 关闭后突发休克** + **差异性紫绀** + 下肢脉消失 |

> [!success] DiGeorge 三大心脏病
> **TOF + Truncus + IAA** = 22q11.2 缺失三巨头
> 都是"圆锥动脉干（conotruncal）"发育异常的结果
> **IAA-B 关联最强（50-90%）**

## 10. IAA vs Coarctation 鉴别

| 项目 | Coarctation | IAA |
|---|---|---|
| **解剖** | 主动脉缩窄（**狭窄但相连**）| 完全**断开** |
| **严重程度** | 变化大 | 极重 |
| **症状出现** | 婴儿 / 儿童 / 成人都可 | **新生儿期 PDA 关闭后立刻** |
| **PDA 依赖** | Critical Coarctation 是 | **绝对依赖** ⭐ |
| **CXR** | "3" sign + 肋骨切迹（年长儿）| 心增大 + 肺血 ↑ |
| **DiGeorge 关联** | 弱 | **强** ⭐ |
| **治疗** | 球囊扩张 / 手术 | PGE₁ + 急诊手术 |

> [!tip] 一句话鉴别
> **Coarctation = 狭窄**（血流减少但相连）
> **IAA = 断开**（血流完全靠 PDA 绕过）

---

## 11. IAA vs HLHS 鉴别（都是 PDA 依赖体循环）

| 维度 | HLHS | IAA |
|---|---|---|
| **核心问题** | LV 发育不良 | 主动脉弓断开 |
| **依赖 PDA** | ✓（全部体循环）| ✓（下半身）|
| **上肢 BP** | 弱 / 全身弱 | **强**（上肢正常）|
| **下肢 BP** | 弱 | **极低** |
| **Differential cyanosis** | 无（全身灰白）| **有**（上红下紫）⭐ |
| **DiGeorge 关联** | 无 | **强**（50-90%）|
| **CXR** | Globular heart + 肺水肿 | 心增大 + 肺血 ↑ |

> [!tip] 一句话鉴别
> **HLHS**：全身休克 + 4 肢全弱 → "全部灌注差"
> **IAA**：上半身正常 + 下半身休克 + 差异性紫绀 → "上下半身分裂"

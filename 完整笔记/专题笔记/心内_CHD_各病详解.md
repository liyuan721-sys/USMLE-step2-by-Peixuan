## §1. TGA（D-Transposition of Great Arteries）⭐⭐

> [!tip] 一句话
> **主动脉接 RV，肺动脉接 LV** → 两套循环**完全平行**不混合 → 出生即重度紫绀

### 解剖
- 主动脉错位起自 **RV**（输出全身缺氧血）
- 肺动脉错位起自 **LV**（输出已氧合血回肺）
- **必须有 PDA / ASD / VSD** 混合两套循环才能活

### 临床
- 出生 **数小时内**重度紫绀（无前驱期）
- **Single loud S2 + 无杂音**（除非合并 VSD）
- 不靠 PDA 即死
- 100% O₂ 无效（hyperoxia test 阴性 — 紫绀不缓解）

### CXR ⭐
- **Egg on string** 🥚（蛋挂线上）— 大血管前后排列变窄纵隔

### 诊断
- 紧急 Echocardiography 确诊
- 显示主动脉起自 RV / 肺动脉起自 LV

### 治疗
1. **PGE₁** IV 维持 PDA（**必须立即**）
2. **Balloon atrial septostomy (Rashkind)** — 急诊扩大 ASD 增加心房水平混合
3. **Arterial Switch (Jatene)** 根治 — 出生 2 周内

### 高危关联
- **糖尿病母亲**（gestational DM）⭐ 经典考点

### 鉴别要点
- vs Truncus：TGA single S2 + 无杂音，Truncus 洪脉 + 心衰重
- vs TOF：TGA 出生即重度紫绀，TOF 多在数月后 Tet spell

> [!success] Memory Hook
> "糖妈生鸡蛋" — 糖尿病母亲 + Egg on string = **TGA**

## §2. TOF（Tetralogy of Fallot）⭐⭐

> [!info] 深度详解
> 本节为短版索引（8 小节速查）；**体征↔机制对照 / Tet spell SOP / 5 张鉴别表 / 6 条 Memory Hooks / 自测**详见 [[完整笔记/专题笔记/心内_CHD_ToF法洛四联症]]。

> [!tip] 一句话
> 四联：**PS + VSD + 主动脉骑跨 + RVH**；严重程度由 **RVOT 梗阻多重**决定

### 解剖（PROVe）
1. **P**ulmonary stenosis (RVOT obstruction)
2. **R**VH (right ventricular hypertrophy)
3. **O**verriding aorta (主动脉骑跨)
4. **V**SD

### 病理 — 右向左分流
- RVOT 梗阻 → RV 压力↑ → 血经 VSD 右→左到主动脉 → 紫绀
- 梗阻越重 → 紫绀越重；极重时**ductal-dependent**

### 临床
- 经典：**Tet spells**（hypercyanotic）— 哭闹/排便/进食诱发，突发严重紫绀
- **蹲位（squatting）缓解** — 增加 SVR → 减少右→左分流
- LUSB 收缩期 harsh 杂音（PS 杂音；右向左分流时杂音**反而减弱**）
- 单一 S2

### CXR ⭐
- **Boot-shaped heart** 👢（靴形心 — RVH + 肺动脉凹陷）

### 诊断
- Echo

### 治疗
**Tet spell 急性**：
1. **Knee-to-chest 体位**（同蹲位机制）
2. 100% O₂
3. Morphine（镇静 + 降低 SVR 反矛盾？实际通过降低呼吸驱动减少 Tet spell）
4. IV fluids（容量负荷↑右心）
5. **Phenylephrine**（α-1 → SVR↑ → 减少右→左分流）
6. β-blocker（缓解 RVOT 痉挛）

**根治**：6-12 个月手术（VSD 修补 + RVOT 流出道扩开）

### 高危关联
- **DiGeorge / 22q11.2**（15-20%）

### 鉴别要点
- vs Truncus：TOF tet spells + 蹲位，Truncus 早期心衰
- vs Eisenmenger：TOF 先天 / Eisenmenger 是大 VSD 演变

> [!success] Memory Hook
> "DiGeorge 出靴" — 22q11.2 + 靴形心 = TOF
> Tet spell 4 步：**Knee-chest → O₂ → Morphine → Phenylephrine**

## §3. Truncus Arteriosus（共同动脉干）⭐⭐

> [!tip] 一句话
> 主肺动脉**不分隔** → 一个共同大血管同时供应体循环 + 肺循环 + 冠脉 → **轻紫绀 + 重心衰 + 洪脉**

### 解剖
- 单一大血管骑跨在 VSD 上
- 共同动脉干分出主动脉 + 肺动脉 + 冠脉
- **必有 VSD**

### 病理
- 体循环 + 肺循环血 在共干水平混合
- PVR 下降后 → 肺血流↑↑ → <font color="#ff0000">**早期严重心衰**</font>
- 紫绀**轻**（充分混合）但心衰**重**

### 临床
- 生后数周早期心衰：**呼吸急促 / 喂养困难 / 出汗 / 体重不增**
- **Bounding pulses（洪脉）+ 脉压宽** — 共干舒张期回流到肺
- **Single S2**（只有一个半月瓣）
- 心尖收缩期杂音 + 舒张早期 click

### CXR
- <font color="#ff0000">心增大 + 肺血流↑</font>

### 诊断
- Echo

### 治疗
- 早期手术（**Rastelli 修补**）— 分出主动脉 + 人工肺动脉导管
- **不依赖 PDA**（共干已混合）→ **不需 PGE₁** ⭐

### 高危关联
- **DiGeorge / 22q11.2**（~50%）⭐

### 鉴别要点
- vs TGA：Truncus 洪脉 + 早心衰 + 不需 PGE₁；TGA single S2 + 无杂音 + 必须 PGE₁
- vs TOF：Truncus 心衰为主，TOF 紫绀 / Tet spell 为主

> [!success] Memory Hook
> "DiGeorge 出共干" + **单一大血管 = 单一 S2 + 洪脉**

## §4. Tricuspid Atresia（三尖瓣闭锁）

> [!tip] 一句话
> 三尖瓣缺如 → RA 血进不去 RV → RV 发育不全 → 必依赖 **ASD/PFO + VSD/PDA** 让血混合 + 到肺

### 解剖
- 三尖瓣完全闭锁，无 RA→RV 通路
- RV **hypoplastic**（发育不良）
- 必有：ASD/PFO（RA→LA）+ VSD/PDA（让血到肺）

### 病理
- RA 血经 ASD/PFO 到 LA → LV 是唯一泵
- 肺血流靠 VSD（RV 残余 → 肺）或 PDA（主动脉 → 肺动脉）

### 临床
- 早期紫绀（生后即可显现）
- 单一 S2 + 可能 VSD holosystolic murmur
- 肺血少则 PDA 依赖

### CXR
- 肺血流减少 + 心轴左偏（**LAD in newborn 罕见！⭐**，大部分新生儿是 RAD）

### ECG ⭐
- **Left axis deviation**（RV 小，电活动以 LV 为主）— 新生儿罕见，强力提示

### 诊断
- Echo

### 治疗
- **PGE₁**（如肺血流依赖 PDA）
- 分期手术：**BT shunt → Glenn → Fontan**（单心室循环）

### 鉴别要点
- 新生儿紫绀 + **LAD** = Tricuspid atresia（其他紫绀型多 RAD）

> [!success] Memory Hook
> "三尖闭锁 → 左偏 + 单心室"
> 新生儿 LAD 几乎等同 Tricuspid atresia

## §5. TAPVR（Total Anomalous Pulmonary Venous Return）

> [!tip] 一句话
> 肺静脉**全部不回左房**，回到右侧（SVC / RA / 冠状窦 / 膈下门静脉）→ 必靠 **ASD/PFO** 维持体循环

### 解剖 4 种类型
- Supracardiac（→ SVC，最常见）
- Cardiac（→ 冠状窦或 RA）
- Infracardiac（→ 门静脉，**最易梗阻**）
- Mixed

### 病理
- 全部肺静脉回流到右心 → 完全混合（饱和差，紫绀）
- LA 仅靠 ASD/PFO 获得血液 → **绝对依赖 ASD**
- Infracardiac 型常**梗阻**（穿过膈肌）→ 肺静脉高压 + 肺水肿

### 临床
- **Obstructed type**（梗阻型）：生后即重度紫绀 + **呼吸窘迫 + 肺水肿** → 急症
- **Non-obstructed**：数周内心衰 + 紫绀

### CXR ⭐
- **Snowman / 雪人形 / 8 字形** ⛄（Supracardiac 型典型）
- Obstructed 型：心不大 + 肺水肿

### 诊断
- Echo + CT angiography

### 治疗
- **不靠 PDA！靠 ASD/PFO**
- **PGE₁ 无效** ⭐⭐ 经典 USMLE 陷阱
- Obstructed 型 → **急诊手术**（重新吻合肺静脉）
- Non-obstructed → 早期手术修补

### 鉴别要点
- vs TGA：TAPVR Snowman + 呼吸窘迫，TGA Egg on string + 无肺水肿
- vs HLHS：TAPVR 紫绀 + 肺水肿，HLHS 全身灰白休克

> [!success] Memory Hook
> "雪人 + PGE₁ 无效" = **TAPVR**
> Infracardiac 必梗阻

## §6. HLHS（Hypoplastic Left Heart Syndrome）⭐

> [!tip] 一句话
> **左心系统全部发育不良**（LV / 二尖瓣 / 主动脉瓣 / 升主动脉）→ 体循环必靠 **RV → PDA → 主动脉**（逆灌）

### 解剖
- LV 严重发育不良
- 二尖瓣 / 主动脉瓣狭窄或闭锁
- 升主动脉 hypoplastic

### 病理
- 肺静脉血进 LA → 经 ASD/PFO 到 RA → RV 同时泵肺循环 + 体循环
- 体循环全部靠 RV → PA → **PDA 逆灌**主动脉
- **PDA 关闭即死**

### 临床
- 出生时正常（PDA 开）
- PDA 关闭后（24-72h）：**全身灰白 + 4 肢均弱脉 + 严重休克 + 代酸**
- Single S2 + 无典型杂音

### CXR
- **Globular heart**（球形心）+ 肺水肿

### 诊断
- Echo

### 治疗
- **PGE₁** IV 立即
- 分期手术：**Norwood → Glenn → Fontan**（单心室循环）
- 或心脏移植

### 鉴别要点
- vs IAA：HLHS **全身**休克 + 4 肢均弱；IAA **下半身**休克 + 差异性紫绀 + 上肢 BP 正常
- vs Critical AS：HLHS LV 不存在；Critical AS LV 大但流出受阻

> [!success] Memory Hook
> "PDA 关闭 → 全身灰白 + 4 肢弱" = **HLHS**

## §7. IAA（Interrupted Aortic Arch）⭐⭐

> [!tip] 一句话
> 升降主动脉**完全断开** → 下半身完全依赖 PDA → DA 关闭后**下半身**休克 + **差异性紫绀**

详见专题：[[完整笔记/专题笔记/心内_CHD_IAA主动脉弓中断]]

### 速查
- Type B 最常见（~70%）+ **22q11.2 关联 50-90%** ⭐⭐
- 上肢 BP 高 + 下肢 BP 极低 + 股动脉脉消
- 上肢 SpO₂ 高 + 下肢 SpO₂ 低（差>10%）
- 治疗：PGE₁ + 一期弓重建 + 修 VSD（80-90% 合并）

### 关键鉴别
- vs Coarctation：IAA **断开**；Coarc **狭窄但相连**
- vs HLHS：IAA 上下分裂；HLHS 全身灰白

## §8. Coarctation of the Aorta（主动脉缩窄）⭐

> [!tip] 一句话
> 主动脉**狭窄但相连**（多在动脉导管对面 juxtaductal）→ 上下肢 BP 悬殊

### 解剖 2 型
- **Preductal / Infantile**：导管前狭窄 → 重度，新生儿期发病，PDA 依赖
- **Postductal / Adult**：导管后狭窄 → 轻度，年长儿/成人发现

### 临床
- **Critical Coarctation**（新生儿）：PDA 关闭后下肢休克 + 弱脉 + 代酸
- 年长儿/成人：**上肢高血压 + 下肢低血压**（差异 BP）+ 股动脉延迟 / 弱
- 体格检查：股动脉脉搏与肱动脉脉搏不同步

### CXR ⭐
- **"3" sign** — 主动脉狭窄前后两段扩张呈 3 字形（年长儿）
- **Rib notching** — 肋骨切迹（侧支循环导致肋骨下缘骨吸收，年长儿）

### 诊断
- Echo + CT/MR angiography

### 治疗
- Critical Coarc：**PGE₁** + 紧急手术
- 年长儿：**球囊扩张 ± 支架** 或外科修补

### 高危关联
- **Turner syndrome (45,X)** ⭐ — 45,X 女性矮小 + 蹼颈 + 性腺发育不良 + Coarc
- Bicuspid aortic valve（80% 合并）

### 鉴别要点
- vs IAA：Coarctation 狭窄相连；IAA 完全断开
- vs HLHS：Coarc 上肢 BP 正常或高；HLHS 全身弱脉

> [!success] Memory Hook
> "Turner → Coarc"；**3 字 + 肋骨切迹**

## §9. Critical AS（Critical Aortic Stenosis）

> [!tip] 一句话
> 主动脉瓣**重度狭窄** → LV 流出严重受阻 → 体循环必靠 **肺动脉 → PDA → 主动脉**

### 解剖
- 主动脉瓣严重狭窄（通常 unicuspid 或 dysplastic）

### 病理
- LV 几乎泵不出血到主动脉
- 体循环依赖 PDA 逆灌（类似 HLHS）

### 临床
- PDA 关闭后：低血压 + 低灌注 + 休克
- 4 肢均弱脉（vs IAA 上肢强 / 下肢弱）
- 主动脉瓣区收缩期 harsh 杂音
- LV 可能扩大（vs HLHS LV 不存在）

### 诊断
- Echo（评估瓣口面积 / 跨瓣压差）

### 治疗
- **PGE₁** 维持 PDA
- **急诊球囊瓣膜成形术**（balloon valvotomy）

### 鉴别要点
- vs HLHS：Critical AS 有 LV 但流出受阻；HLHS LV 几乎不存在
- vs Coarc：Critical AS 4 肢均弱；Coarc 上下肢悬殊

## §10. Pulmonary Atresia / Critical PS

> [!tip] 一句话
> 肺动脉瓣**闭锁或重度狭窄** → RV 出不去 → 肺血流靠 **主动脉 → PDA → 肺动脉**（逆灌）

### 解剖
- Pulm atresia：肺动脉瓣完全闭锁（可合并室间隔完整 IVS 或 VSD）
- Critical PS：肺动脉瓣重度狭窄

### 病理
- 肺血流绝对依赖 PDA

### 临床
- 重度紫绀（生后即发病）
- 单一 S2
- PDA 关闭后立即失代偿

### 诊断
- Echo

### 治疗
- **PGE₁** 维持 PDA
- **急诊球囊瓣膜成形**（PS）或外科手术（PA-IVS）
- PA-VSD 处理同 severe TOF

### 高危关联
- 部分合并 22q11.2

### 鉴别要点
- vs TOF：Pulm atresia 更重（全闭），TOF 部分梗阻

## §11. VSD（Ventricular Septal Defect）⭐

> [!tip] 一句话
> 室间隔缺损 → 左→右分流 → **PVR 下降后**（6-8 周）显症

### 解剖 4 型
- Perimembranous（最常见，~80%）
- Muscular
- Inlet（合并 AV canal）
- Outlet / Supracristal

### 病理
- 出生时 PVR 还高 → 分流量小 → 无症状
- 6-8 周 PVR↓ → 左→右分流量↑↑ → CHF

### 临床
- **Holosystolic murmur, LLSB**（**左下胸骨缘**）— 杂音越**响**孔越**小**（流量小但湍流强）
- 大 VSD：早期 CHF（喂养差 / 出汗 / 体重不增）
- 小 VSD（多数）：常自闭 / 无症状
- Eisenmenger 演变：大 VSD 未修复 → 肺血管病变 → 右→左分流 → 紫绀

### 诊断
- Echo

### 治疗
- 小 VSD：观察（大部分 2 岁内自闭）
- 大 VSD + CHF：利尿 + ACEi → 手术修补（4-6 月）

### 高危关联
- Down / Trisomy 13 / Trisomy 18 / IDM

### 鉴别要点
- vs ASD：VSD holosystolic LLSB；ASD fixed split S2 + LUSB 喷射音
- vs Tricuspid regurgitation：VSD 不变化；TR 吸气增强

> [!success] Memory Hook
> "**全 holo / 左下 / 小响大软**"

## §12. ASD（Atrial Septal Defect）

> [!tip] 一句话
> 房间隔缺损 → 左→右分流 → **Fixed split S2** + 长期晚发并发症（PAH / 房颤）

### 解剖 4 型
- **Ostium secundum**（最常见，~70%）
- Ostium primum（合并 AV canal）
- Sinus venosus
- Coronary sinus

### 病理
- 左→右分流 → RV 容量负荷
- 一般不引起新生儿/儿童心衰
- 成人期：肺动脉高压 / 房颤 / 矛盾性栓塞（cryptogenic stroke）

### 临床
- 儿童期常无症状
- **Fixed split S2** ⭐⭐（不随呼吸变化）— 右心容量过载使 P2 始终延迟
- LUSB 喷射音（**相对**性肺动脉流量增加）
- 大 ASD 可有 LSB 舒张早期流量音

### 诊断
- Echo

### 治疗
- 小 ASD：观察（部分自闭）
- 大 ASD（成人）：经皮封堵或外科修补

### 高危关联
- **Holt-Oram syndrome** ⭐ — TBX5 基因 + **拇指 / 上肢畸形** + ASD
- 母亲糖尿病 / Down syndrome

### 鉴别要点
- vs VSD：ASD fixed split S2 + 杂音弱；VSD holosystolic 杂音响
- vs 正常生理 split S2：生理 split 随呼吸变化；ASD fixed

> [!success] Memory Hook
> "**Fixed split + 拇指畸形 = Holt-Oram + ASD**"

---

## §13. PDA（Patent Ductus Arteriosus）

> [!tip] 一句话
> 动脉导管出生后**未关闭** → 主动脉 → 肺动脉 左→右分流 → **连续机器样**杂音

### 病理
- 正常出生后 PDA 12-24h 功能性关闭，2-3 周解剖关闭
- 早产儿 / 低氧 / 高原 → 关闭延迟
- 左→右分流 → 容量过载 + CHF（大 PDA）

### 临床
- **连续机器样**杂音（continuous machine-like murmur）— 收缩 + 舒张连续
- **左锁骨下区**（infraclavicular）最响
- **Wide pulse pressure + bounding pulses**（舒张期主动脉血漏到肺动脉）
- 大 PDA：CHF / 喂养差 / 体重不增

### 诊断
- Echo

### 治疗
- **早产儿**：**Indomethacin / Ibuprofen** 关闭（COX 抑制 → PG 下降 → 关 PDA）
- **足月儿**：通常不会自闭 → **经皮封堵** 或手术结扎
- **PDA 依赖型 CHD**（特殊）：**PGE₁ 维持开放**（详见 [[完整笔记/专题笔记/心内_CHD_PDA依赖型]] §6-§7）

### 高危关联
- **先天性 Rubella** ⭐ — PDA + 先天白内障 + 耳聋（"3 联"）
- 早产
- 高海拔出生

### 鉴别要点
- vs VSD：PDA continuous（收缩+舒张）；VSD 只 holosystolic
- vs Venous hum：PDA infraclavicular + 不随颈静脉压消失；venous hum 颈部 + 压颈消失
- vs Truncus：PDA 心尖以外 + 不一定有早心衰；Truncus 早心衰为主

> [!success] Memory Hook
> "**机器样 + 早产 / Rubella + 洪脉 / 宽脉压**"

## §14. AV Canal Defect / CAVSD（Atrioventricular Canal Defect）⭐

> [!tip] 一句话
> 心内膜垫缺损 → **大 ASD + 大 VSD + 共同房室瓣** → Down 综合征**经典**

### 解剖
- Ostium primum ASD + Inlet VSD + Common AV valve
- "Complete" AVSD：全部 3 个缺损 + 单瓣
- "Partial" AVSD：仅 ASD primum + 二尖瓣裂

### 病理
- 早期严重 L→R 分流（既 ASD 又 VSD）
- 早期肺血流过多 + CHF
- 早期 Eisenmenger 风险

### 临床
- 早期严重 CHF（生后数周内）
- Holosystolic murmur + Fixed split S2

### 诊断
- Echo

### 治疗
- 利尿 + ACEi 桥接
- **早期手术修补**（3-6 月）— 防 Eisenmenger

### 高危关联
- **Down syndrome (Trisomy 21)** ⭐⭐ — Down 心脏畸形最常见就是 CAVSD
- Heterotaxy

### 鉴别要点
- vs 单纯 VSD：CAVSD 早期 CHF + Down 容貌
- vs ASD：CAVSD 严重 CHF；secundum ASD 多年后才显症

> [!success] Memory Hook
> "**Down → CAVSD**"，AV 瓣 + VSD + ASD 三合一

## §15. IDM-HCM（Hypertrophic Cardiomyopathy of Infant of Diabetic Mother）

> [!tip] 一句话
> 糖妈儿的**功能性 LVH**（高胰岛素 → 心肌肥厚）→ **1 岁内自愈** + **禁 inotrope / 利尿剂** ⭐

### 病理
- 母亲糖尿病 → 胎儿高血糖 → 胎儿高胰岛素 → 心肌细胞肥大（肌节增加）
- 室间隔不对称肥厚（类似 HOCM）+ LV 流出道动态梗阻

### 临床
- 巨大儿 + 母亲糖尿病史
- 生后数周内呼吸急促 / 喂养差 / 心衰样表现
- LV 收缩期喷射杂音（流出道动态梗阻）

### 诊断
- Echo（室间隔肥厚 + SAM of mitral valve + LVOT 梗阻）

### 治疗 ⭐ 反直觉
- **β-blocker**（降心率，让 LV 充盈，降低动态梗阻）
- **静脉补液**（增加前负荷）
- ❌ **禁 inotrope**（如 dopamine）— 会加重动态梗阻 ⭐
- ❌ **禁利尿剂**（如 furosemide）— 减少前负荷加重梗阻 ⭐
- 自然病程：**1 岁内自愈**

### 高危关联
- 糖尿病母亲（pregestational + gestational）

### 鉴别要点
- vs HOCM（基因性）：IDM-HCM 自愈，HOCM 终生 / 家族史 / 突死风险
- vs HLHS：IDM-HCM LV 大但功能好；HLHS LV 几乎不存在

> [!success] Memory Hook
> "糖妈 + LV 肥厚 → **β-blocker + 补液**，**禁 inotrope / 利尿**"

## 🎯 法庭审判 — 一图记牢所有概念

```
H₀ = "被告无罪"（默认：两组无差异 / 新药无效）
H₁ = "被告有罪"（你想证明的：有差异 / 药有效）
```

|  | **真相：无罪**（H₀ 真）| **真相：有罪**（H₀ 假）|
|---|---|---|
| **判决：无罪**（接受 H₀）| ✅ 正确释放 | ❌ **放过坏人** = Type **II** (β) |
| **判决：有罪**（拒绝 H₀）| ❌ **冤枉好人** = Type **I** (α) | ✅ 正确定罪 = **Power** (1−β) |

---

## 🔑 核心口诀

### 口诀 1：记住 Type I / II 是什么
> **"I 冤 α，II 放 β"**
> - **Type I** = 冤枉好人 = 假阳性 = α（一般 0.05）
> - **Type II** = 放过坏人 = 假阴性 = β（一般 0.20）
> - **Power** = 1−β = 真有罪时成功定罪 = 真阳性率（一般 0.80）

### 口诀 2：α / β / Power 之间的关系
> **"α 和 Power 同向，α 和 β 反向"**
> - **α ↑ → Power ↑**（同向，因为门槛低，真假都容易宣布"有差异"）
> - **α ↑ → β ↓**（反向，因为 Power = 1−β）
> - **n ↑ → Power ↑, β ↓**（n 不影响 α）

---

## ⚠️ 最反直觉的点：α ↑ → Power ↑（不是 ↓！）

### 错误直觉（很多人卡这里）
```
"α 大 = 标准松 = 研究质量差 = Power 小"  ❌
```

### 正确逻辑
```
α = 拒绝 H₀ 的"门槛"
│
├─ α ↑（门槛低）→ 容易拒绝 H₀
│   ├─ 真有差异时 → 容易宣布"有差异" → Power ↑ ✅
│   └─ 真无差异时 → 容易误判"有差异" → Type I error 也 ↑ ❌
│
└─ α ↓（门槛高）→ 难拒绝 H₀
    ├─ 真有差异时 → 容易漏掉 → Power ↓ ❌
    └─ 真无差异时 → 不会误判 → Type I error ↓ ✅
```

### 数字示例：同一个研究，p = 0.04

| α 设定 | 判定 | 结论 |
|---|---|---|
| α = 0.10（松）| 0.04 < 0.10 ✅ | "有差异" → Power ↑ |
| α = 0.05（标准）| 0.04 < 0.05 ✅ | "有差异" |
| α = 0.01（严）| 0.04 > 0.01 ❌ | "无差异" → 漏掉真差异 → Power ↓ |

**关键**：同一个数据，改 α 就改结论！

## 🎪 大门比喻（永久记忆）

| α 状态 | 大门 | Power | Type I (α) |
|---|---|---|---|
| α ↑（**松**，如 0.10）| 🚪 敞开，谁都能进 | **↑** 真坏人容易抓 | **↑** 也容易误抓好人 |
| α = 0.05（标准）| 🚪 半开，常规查 | 标准 | 0.05 |
| α ↓（**严**，如 0.01）| 🚪 紧闭，严查 | **↓** 漏掉真坏人 | **↓** 不冤枉好人 |

---

## ⚖️ 为什么实践中 α 固定 0.05？

> [!warning] α 和 Power 都想要"大"，但 α 必须"小"
> - 想要 Power **大**（真有差异能检测出）
> - 但**不想** Type I error 大（不要假阳性药物上市）
> - 两个需求冲突 → **α 固定 0.05** 是学术共识
> 
> **实际调控 Power 的合法手段**：
> - ✅ **增加 sample size (n)** — 最实用，不影响 α
> - ✅ **优化测量** → 减少 variability
> - ❌ **不能靠提高 α**（增加假阳性）

## 🎯 应用层强化（针对易错点）⭐⭐⭐

### 🔑 锁点 1：方向感 — 反比时要真的反过来
看到 α 变化，**先画箭头**：
```
α ↑ → β ↓（反向陪走）→ Power ↑（同向陪走）
α ↓ → β ↑（反向陪走）→ Power ↓（同向陪走）
```
**陷阱**：理论记住了"反比"，应用时方向搞反。每次应用必须**画箭头验证**。

### 🔑 锁点 2：n 是"独立变量"，不碰 α

| 调整 n | 对 α | 对 β | 对 Power |
|---|---|---|---|
| n ↑ | ❌ **不变** | ✅ ↓ | ✅ ↑ |
| n ↓ | ❌ **不变** | ✅ ↑ | ✅ ↓ |

**核心**：α 是**研究者主动设定**的（默认 0.05），不是算出来的。n 多大都不会改 α。
**口诀**：**"n 只动 β 和 Power，不碰 α 和 Effect"**

### 🔑 锁点 3：USMLE 题干约束词必读

| 看到这些词 | 排除的选项类型 |
|---|---|
| "**不增加假阳性**" / "without increasing Type I" | 排除"调高 α"的选项 |
| "**不增加假阴性**" / "without increasing Type II" | 排除"调低 α"的选项 |
| "**在 α 固定的前提下**" | 只能选改 n / effect / variability |
| "**仅增加 Power**" | 找单独提升 Power（即 C 类调整）|

**陷阱**：只看后半句"增加 Power"，忽略前半句的约束 → 错选调 α 的选项。

### 黄金答题套路
> 问"如何提升 Power" → **优先选"增加 sample size"**（唯一不动 α 的合法手段）
> 问"如何降低 Type I error" → 选"调低 α"（但代价是 Power 也降）
> 问"为何 p > 0.05 但仍想确认无差异" → 检查"是否 underpowered"（n 是否足够）

---



### 题型 1：错误识别
**Q**: 研究说"咖啡致癌"（p = 0.04），实际咖啡不致癌 → ?
**A**: Type I error（冤枉好人 = 假阳性）

### 题型 2：错误识别（反向）
**Q**: 研究说"疫苗无效"（p = 0.15），实际疫苗有效 → ?
**A**: Type II error（放过坏人 = 假阴性）

### 题型 3：减少错误
**Q**: 怎么减少 Type II error？
**A**: 增加 n → Power ↑ → β ↓ → Type II error ↓

### 题型 4：α 调整影响
**Q**: 把 α 从 0.05 改成 0.01，Power 会？
**A**: Power ↓（同向变化）

### 题型 5：药厂操纵
**Q**: 药厂把 α 从 0.05 调到 0.10，效果？
**A**: Power ↑ **同时** Type I error ↑（不可接受，所以学术界禁止）

## 🎯 终极速记口诀（背下来这一段就够）

> [!success] 黄金五句话
> 1. **"I 冤 α，II 放 β"** — 类型 I 冤枉好人（α），类型 II 放过坏人（β）
> 2. **"Power = 1−β = 真有罪时成功定罪"**
> 3. **"α ↑ → Power ↑"**（同向，门槛低则真假都容易"定罪"）
> 4. **"α ↑ → Type I error ↑"**（代价：假阳性也变多）
> 5. **"实践中 α 固定 0.05，调 Power 只能靠 n"**

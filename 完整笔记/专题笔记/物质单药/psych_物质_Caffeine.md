# 物质单药 — 咖啡因（Caffeine）

> [!info] 一句话定位
> 咖啡因 = **腺苷受体拮抗剂**，产生拟交感效应。中毒 = **HTN / 心动过速 + 失眠 / 焦虑 / 震颤**；**过量 >1 g 可致 seizure**；戒断真实存在 = **头痛 + 嗜睡 + 易怒**。是"日常物质里被低估的兴奋剂"。

---

## 机制

| 维度 | 内容 |
|---|---|
| **核心机制** | **拮抗腺苷（adenosine）受体** ⭐ —— 腺苷本是促镇静 / 扩血管的"刹车"，被阻断 → CNS 兴奋 |
| **下游效应** | 间接 ↑ 儿茶酚胺 / cAMP → 拟交感（心率↑ / BP↑ / 警觉↑）|
| **来源** | 咖啡 / 茶 / 能量饮料 / 减肥药 / OTC 提神片 —— 注意"隐形剂量叠加" |
| **戒断机制** | 长期用 → 腺苷受体上调；停用 → 腺苷作用反弹 → 头痛（脑血管扩张）/ 嗜睡 |

---

## 中毒表现 Intoxication

| 系统 | 表现 |
|---|---|
| **心血管** | **高血压 + 心动过速** + 心悸；高剂量可致心律失常 |
| **神经 / 精神** | **失眠 + 焦虑 / 紧张（jitteriness）+ 震颤** + 烦躁 + 思维奔逸；重度可达 psychosis |
| **胃肠** | 恶心 / 利尿 |
| **重度过量** | **摄入 > 1 g → seizure** ⭐；可伴心律失常、低钾、横纹肌溶解 |

> [!danger] 高频考点 —— 咖啡因也会 seizure
> 大剂量咖啡因（如能量饮料 + 减肥药 + 提神片叠加，或药片过量）→ **seizure**。
> stem 里"年轻人大量能量饮料 / 减肥药 + 心动过速 + 焦虑 + 震颤 ± 抽搐" → 想咖啡因中毒。

---

## 戒断 Withdrawal

> [!info] 咖啡因戒断真实存在 —— 但不致死
> DSM 已正式收录咖啡因戒断。**难受、影响功能，但不危险**。

戒断表现：

- **头痛**（最经典 —— 脑血管反弹性扩张）⭐
- **嗜睡 / 疲乏 / 困倦**（drowsiness）
- **易怒 / 情绪低落 / 注意力↓**
- 流感样不适（肌痛 / 恶心）—— 与"opioid / SSRI 停药 / 咖啡因戒断"同属 flu-like，需靠用药史区分

> [!tip] 戒断 stem 线索
> 周末不喝咖啡就头痛 / 困倦、术前禁食后头痛 → 咖啡因戒断（常被误当其他病因）。

---

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **轻度中毒** | 停用 + 支持治疗（多数自愈）|
| **焦虑 / 激越 / 心动过速** | 支持治疗；重者短期 BZD |
| **seizure** | BZD 控制 + 支持 |
| **戒断** | 支持 + 宣教；逐步减量而非骤停可避免头痛 |

---

## UW Stem 模式

- 学生考试季大量咖啡 / 能量饮料后**失眠 + 焦虑 + 震颤 + 心动过速** → 咖啡因中毒
- 大量咖啡因摄入（药片 / 叠加来源）后**抽搐** → 咖啡因过量 seizure
- 重度焦虑 / 心悸患者 —— 问诊揪出"隐形咖啡因来源"（能量饮料 / OTC）→ 病因
- 周末 / 术前禁咖啡后**头痛 + 嗜睡 + 易怒** → 咖啡因戒断（非偏头痛 / 抑郁）

---

## Memory Hook

> [!tip] 锁定
> **"咖啡因 = 拔掉腺苷的刹车"** —— 腺苷踩刹车（镇静），咖啡因拆刹车（兴奋）。
> 中毒 = **"抖、慌、睡不着"**；过量 = **>1 g 抽搐**。
> 戒断 = **"头痛 + 犯困"** —— 拔走兴奋剂，腺苷反扑。

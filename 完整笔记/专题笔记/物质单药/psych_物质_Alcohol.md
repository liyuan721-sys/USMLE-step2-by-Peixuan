# 物质单药 — 酒精（Alcohol / Ethanol）

> [!info] 一句话定位
> 酒精 = **GABA-A 激动 + NMDA 拮抗**的 CNS 抑制剂。中毒看"共济失调 + 构音障碍 + 呼吸抑制"，戒断是**镜像兴奋 + 可致死**（AWS / DT）。**急性戒断时间轴 / 维持期 / 停药综合征已在 hub 笔记深挖**——本页只做单药考点 + 链回。

---

## 机制

| 维度 | 内容 |
|---|---|
| **急性作用** | **↑ GABA-A**（抑制）+ **↓ NMDA**（谷氨酸兴奋）→ 双重 CNS 抑制 |
| **慢性适应** | 长期饮酒 → GABA-A 受体下调 + NMDA 受体上调（代偿）|
| **戒断时** | 突然停酒 → GABA 不足 + NMDA 过度兴奋 → **过度兴奋状态**（震颤 / seizure / DT）|
| **代谢** | Ethanol →(ADH)→ 乙醛 →(ALDH)→ 乙酸；**Disulfiram 抑制 ALDH** → 乙醛蓄积 |

> [!tip] 镜像逻辑
> 慢性饮酒把大脑"调成抑制基线"；突然撤酒 = 抑制消失但代偿仍在 → **净兴奋** → 这就是为什么酒精戒断会 seizure / DT。

---

## 中毒表现 Intoxication

| 系统 | 表现 |
|---|---|
| **神经** | **构音障碍（slurred speech）+ 共济失调（ataxia）**+ 眼震 + 判断力下降 + 反应迟钝 |
| **意识** | 嗜睡 → 昏睡 → 昏迷（剂量依赖）|
| **生命体征** | **呼吸抑制**（高剂量）+ 低体温 + 低血糖（肝糖原耗竭）|
| **瞳孔** | 正常 / 略大（非诊断性）|

> [!danger] Wernicke encephalopathy（必记）
> 慢性酗酒者 **thiamine（B1）缺乏** → 三联征 **眼肌麻痹 + 共济失调 + 意识混乱**。
> ⚠️ **先给 thiamine 再给葡萄糖** —— 先给糖会耗尽残余 B1 → 诱发/加重 Wernicke。

## 戒断 Withdrawal

> [!danger] 酒精戒断会致死 ⭐
> 酒精与 BZD 是仅有的**两种会致死的物质戒断**（同 GABA-A 通路）。机制 = GABA 不足 + NMDA 过兴奋 → seizure + 自主神经风暴（DT）。**Delirium tremens 死亡率不可忽视**。

**AWS 4 阶段时间轴口诀 "6-12-12-48"**：6h 颤 → 12h 抽 → 12h 幻 → 48h 谵。

- 6-24h 轻度戒断（震颤 / 焦虑 / 失眠）
- 12-48h withdrawal seizure（全身强直阵挛）+ alcoholic hallucinosis（**幻觉但定向力正常 + 生命体征稳定**）
- 48-96h **Delirium tremens**（定向力丧失 + 自主神经风暴 + 可致死）

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **急性中毒** | 支持治疗 + **保护气道**（呼吸抑制 / 误吸风险）；监测血糖；慢性者补 thiamine |
| **急性戒断** | **Benzodiazepines** 一线（lorazepam / chlordiazepoxide / diazepam）—— 替代 GABA，防进展到 seizure / DT |
| **DT** | ICU + 大剂量 BZD |
| **维持期 AUD** | 见下方一句话 |

> [!success] AUD 维持期 3 药一句话（详见 hub §15.2）
> - **Naltrexone**：craving 强 + 仍在喝 → 首选（边喝边启动 OK）
> - **Acamprosate**：已戒断 + 防复饮 + **肝病也能用**（肾排泄）
> - **Disulfiram**：二线 —— 需 first-line 失败 + 高动机 + 监督，三者缺一不选

> [!warning] 急性戒断 BZD ≠ 维持期治疗
> 急性期 BZD 只是短期脱毒桥接；长期戒酒靠 Naltrexone / Acamprosate / Disulfiram。

## UW Stem 模式

- 慢性饮酒者 stop 后 **6-48h 出现震颤 → 后续 seizure / DT** → AWS
- 酗酒者 + **眼肌麻痹 + 共济失调 + 意识混乱** → Wernicke → 先 thiamine
- 急诊昏睡 + 构音障碍 + 共济失调 + 呼吸抑制 → 急性酒精中毒，保护气道
- "幻觉 + 定向力正常 + 生命体征稳定" vs "幻觉 + 定向力丧失 + 自主神经风暴" → hallucinosis vs DT
- 戒酒者问"哪个维持药" → 看 craving / 肝病 / 动机分流（§15.2 决策树）

## Memory Hook

> [!tip] 锁定
> **"喝下去松软，撤下来抽搐"** —— 中毒 = 软（共济失调 / 构音障碍 / 呼吸抑制）；戒断 = 硬（震颤 / seizure / DT）。
> 致命戒断 = **GABA 三兄弟**（Alcohol / BZD / Barbiturate）。
> Wernicke = **"COA"**（Confusion + Ophthalmoplegia + Ataxia），先 B1 后糖。

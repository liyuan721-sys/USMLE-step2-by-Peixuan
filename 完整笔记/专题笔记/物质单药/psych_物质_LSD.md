# 物质单药 — LSD / 致幻剂（LSD / Hallucinogens）

> [!info] 一句话定位
> LSD（lysergic acid diethylamide）= **5-HT2A 受体激动**的经典致幻剂。指纹 = **视/听幻觉 + synesthesia（联觉）+ 散瞳 + "bad trip"**。**无 withdrawal 综合征**；远期可遗留 **HPPD（持续性知觉障碍 / "闪回"）**。

---

## 机制

| 维度 | 内容 |
|---|---|
| **作用位点** | 主要 **5-HT2A 受体激动剂**（部分激动）|
| **效应** | 强烈知觉改变：幻觉、感知扭曲、**synesthesia**（"看见声音 / 听见颜色"）|
| **同类** | psilocybin（蘑菇）、mescaline（仙人掌）—— 同属经典致幻剂，机制类似 |
| **检测** | 常规尿药筛查通常**不含 LSD**（剂量极低 + 结构特殊）|

---

## 中毒表现 Intoxication

| 系统 | 表现 |
|---|---|
| **知觉** | **视幻觉 + 听幻觉** + 感知扭曲（物体变形 / 拖影）|
| **特异线索** | **Synesthesia 联觉** ⭐ —— 感官交叉（看到声音的颜色）—— LSD 的经典指纹 |
| **瞳孔** | **散瞳（mydriasis）** |
| **精神** | 欣快 / 灵性体验 ↔ **焦虑 / 偏执 / 惊恐（"bad trip"）** —— 可致冲动自伤 |
| **自主神经** | 心动过速 + 高血压 + 出汗 + 震颤（轻度拟交感）|
| **意识** | 通常**清醒 + 定向力保留**（与谵妄性物质不同）|

> [!warning] "Bad trip" —— 高频 stem
> 不愉快的致幻体验 = 强烈恐惧 / 偏执 / 失控感 → 可致**冲动性危险行为 / 自伤 / 跳楼**。
> 处理 = **安静低刺激环境 + 安抚（"talk down"）**；激越重时短期 **BZD**。

> [!danger] HPPD（Hallucinogen Persisting Perception Disorder）⭐
> 致幻剂使用**很久之后**，在未再用药情况下反复出现**知觉障碍 / "闪回（flashbacks）"**（如拖影、几何幻视）。
> - 提示远期后遗效应，与急性中毒分开。
> - 需与精神病性障碍鉴别（HPPD 患者**自知力保留**，知道这是知觉异常）。

---

## 戒断 Withdrawal

> [!success] LSD / 致幻剂无戒断综合征 ⭐
> 致幻剂**不产生生理依赖**，停药**无 withdrawal**。这是与酒精 / BZD / 阿片 / 兴奋剂的关键区别 —— stem 问"戒断表现"时，致幻剂 = 无。
> （耐受可快速产生，但停药不出现戒断症状。）

---

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **急性中毒 / bad trip** | **安静低刺激环境 + 言语安抚（reassurance / "talk down"）**为主 |
| **重度激越 / 焦虑** | 短期 **Benzodiazepines** |
| **精神病性激越（环境 + BZD 无效）** | 可加抗精神病药 |
| **HPPD** | 停用致幻剂；症状显著时可试 BZD / 部分病例用其他药；以宣教 + 随访为主 |

---

## UW Stem 描述模式

- 年轻人聚会后**视幻觉 + "听见颜色 / 看见声音"（synesthesia）+ 散瞳** → LSD 中毒
- 出现强烈恐惧 / 偏执 + 冲动危险行为 → "bad trip" → 安静环境 + 安抚 ± BZD
- 数月前用过致幻剂、现无用药却反复"闪回 / 拖影"且**有自知力** → HPPD
- 问"停药后戒断症状" → 致幻剂**无戒断**（陷阱题）

---

## Memory Hook

> [!tip] 锁定
> **"看见声音 = LSD"** —— synesthesia 联觉是 LSD 最独特的指纹。
> **散瞳 + 清醒 + 幻觉** —— 区别于谵妄性物质（意识混乱）。
> **无戒断、有闪回** —— LSD 不上瘾（无 withdrawal），但可留 HPPD。
> Bad trip 处理 = "talk down"，先环境安抚再考虑 BZD。

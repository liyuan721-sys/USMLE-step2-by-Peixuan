# 物质单药 — 甲基苯丙胺（Methamphetamine）

> [!info] 一句话定位
> Methamphetamine（甲基苯丙胺、"meth"）= **长效拟交感兴奋剂**，强力促单胺释放。USMLE 招牌：**极度躁动 + 精神病 + 舞蹈样动作 + 烂牙（meth mouth）/ 磨牙**；作用持续 **8-24 小时**，远比可卡因长。

---

## 机制

> [!tip] 核心机制 — Releaser（促释放剂）
> 与可卡因（reuptake blocker）不同，meth 是**苯丙胺类促释放剂**：
> - 进入突触前神经元，**逆转单胺转运体**，把胞质内 **DA / NE / 5-HT 大量泵出**到突触间隙。
> - 同时抑制再摄取 + 抑制 MAO 降解 → 单胺效应被三重放大、且持久。
> - **DA ↑↑** → 强烈欣快、精神病、刻板/舞蹈样动作、强成瘾。
> - **NE ↑↑** → 交感风暴。
> - 高脂溶性 + 代谢慢 → **作用 8-24h（长效）⭐**。

```
Methamphetamine
   │  逆转转运体 → 强制释放胞内单胺
   ▼
突触间隙 DA/NE/5-HT ↑↑↑（+ 抑制 reuptake + 抑制 MAO）
   ├─ DA ↑↑ → 精神病 / 刻板·舞蹈样动作 / 强成瘾
   └─ NE ↑↑ → 交感风暴
作用持续：8-24h（长效）⭐ —— vs 可卡因 ~30 min
```

---

## 中毒表现 Intoxication

> [!warning] Stimulant intoxication（agitation 突出）
> 通用兴奋剂表现 + meth 特有的**极端躁动与慢性使用体征**。

| 表现 | 说明 |
|---|---|
| **极度躁动 Agitation** ⭐ | 比可卡因更剧烈、更持久的精神运动性激越 |
| **精神病 Psychosis** | 偏执妄想、幻觉（含触幻觉 "meth bugs"、抓挠致皮肤损伤）|
| **舞蹈样 / 刻板动作** ⭐ | choreoathetoid movements、反复无目的动作（"punding"）|
| **磨牙 Bruxism / Grinding teeth** ⭐ | 下颌紧咬、磨牙 |
| **烂牙 "Meth mouth"** ⭐ | 慢性使用招牌 — 严重广泛龋齿 / 牙齿脱落（口干 + 磨牙 + 糖食 + 忽视口腔卫生）|
| 生命体征 | HTN、tachycardia、hyperthermia、散瞳、出汗、seizure |
| 并发症 | rhabdomyolysis、缺血性卒中 / MI、心肌病 |

> [!tip] 一眼锁定 meth
> 慢性使用 + **烂牙 + 磨牙 + 舞蹈样动作 + 长时间躁动/精神病** → meth（不是可卡因 —— 可卡因短效、招牌是冠脉痉挛/鼻中隔穿孔）。

---

## 戒断 Withdrawal

> [!info] Severe stimulant withdrawal — 镜像 "crash"
> 中毒兴奋 → 戒断抑制。生理不致命，**情绪致命**。

| 维度 | 戒断表现 |
|---|---|
| 情绪 | **严重抑郁（severe depression）⭐ + 快感缺失 + 自杀念头** |
| 渴求 | **强烈渴求（strong drug craving）⭐** |
| 睡眠 / 食欲 | 嗜睡（hypersomnia）、生动不愉快梦境、食欲 ↑ |
| 体力 | 极度疲乏、精神运动迟滞、注意力差 |

> [!warning] 与可卡因戒断的关系
> meth 与 cocaine 戒断本质相同（兴奋剂 crash），meth 因长效 + 强力，**抑郁程度更重、渴求更强**。处理同样是支持治疗 + 严密**自杀风险监测**，无 FDA 批准药。

---

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **激越 / 精神病（首选）** | **Benzodiazepine（lorazepam IV/IM）一线** —— 镇静、降 HR/BP、控 seizure |
| 激越顽固 / 明显精神病 | 加**抗精神病药（haloperidol / 二代）** 作二线 |
| **高血压** | BZD 一线；顽固 → phentolamine / 硝酸酯 / CCB（同可卡因，**避开纯 β-blocker**）|
| **Hyperthermia** | 主动降温 + 补液（重症需积极处理）|
| **Rhabdomyolysis** | 大量补液、监测 K⁺ / 肾功能 |
| **戒断** | 支持治疗 + **自杀风险评估**；无特效药 |

> [!tip] 激越管理通用原则
> 兴奋剂 / PCP 致激越 → **BZD 永远是第一线**（既镇静又降生命体征又抗 seizure）。抗精神病药作二线（注意降低癫痫阈值、可致高热）。

## UW Stem 模式

> [!example]- 典型 Stem 模式（中文重述，非 UW 原文）
> - 患者**严重躁动 / 攻击性 / 偏执**就诊，体检见**广泛龋齿（烂牙）、磨牙、皮肤抓挠伤、不自主舞蹈样动作** → 指向 methamphetamine。
> - 兴奋剂中毒急诊，问首选镇静药 → 答 **benzodiazepine**。
> - 长期使用者停药数日后**严重抑郁 + 嗜睡 + 强烈渴求 + 食欲增加** → meth withdrawal；处理 = 支持 + 评估自杀。
> - 鉴别陷阱：题干强调"短效、胸痛、鼻中隔穿孔"→ 选 cocaine；强调"长效、烂牙、舞蹈样动作"→ 选 meth。

## Memory Hook

> [!success] 一句话锁定
> **"逆转泵把多巴胺全泼出来 —— 烂牙磨牙、舞着跳着、躁上 24 小时。"**
>
> 口诀：
> - **中毒**："长效兴奋剂 → 烂牙 + 磨牙 + 舞蹈样 + 久躁久精神病"。
> - **戒断**："严重抑郁 + 强烈渴求" = meth crash。
> - **激越**："先 BZD，后抗精神病药"。

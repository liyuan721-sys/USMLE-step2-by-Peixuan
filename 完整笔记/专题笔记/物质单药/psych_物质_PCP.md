# 物质单药 — PCP（Phencyclidine / "Angel Dust"）

> [!info] 一句话定位
> PCP（苯环利定，"天使粉"）= **NMDA 受体拮抗剂**的解离性致幻剂。USMLE 招牌：**多方向眼震（nystagmus）⭐ + 极端暴力 / 攻击性 + 痛觉减退 + 缩瞳（miosis）**。激越管理首选 **BZD**。

---

## 机制

> [!tip] 核心机制 — NMDA 受体拮抗
> PCP 是 **NMDA 谷氨酸受体的非竞争性拮抗剂**（与 ketamine 同类，效力更强、半衰期更长）：
> - 阻断 NMDA → **解离状态**：感觉与意识脱节、痛觉与情绪迟钝。
> - 同时**增强儿茶酚胺活性**（抑制再摄取）→ HTN、tachycardia、出汗（拟交感成分）。
> - 中脑 DA 系统受扰 → 精神病、暴力、躁狂样行为。
> - 对运动 / 眼动通路的影响 → 招牌的**多方向眼震**。

```
PCP
   │  非竞争性拮抗 NMDA (谷氨酸) 受体  +  增强儿茶酚胺
   ▼
解离状态 ── 痛觉↓ / 感知与意识脱节
   ├─► 暴力 / 攻击性 / 精神病 / 躁狂样
   ├─► 多方向眼震（水平+垂直+旋转）⭐ 招牌
   └─► 拟交感：HTN / 心率↑ / 高热 / 出汗
```

---

## 中毒表现 Intoxication

> [!warning] PCP intoxication — "暴力 + 眼震 + 不怕疼"
> 与其他兴奋剂的关键差异：**攻击性极端、痛觉显著减退、瞳孔反而缩小**。

| 表现 | 说明 | 一眼线索 |
|---|---|---|
| **多方向眼震 Nystagmus** ⭐⭐⭐ | **水平 + 垂直 + 旋转**眼震 —— PCP 招牌 | 见到多方向眼震基本锁 PCP |
| **暴力 / 攻击性 Violence** ⭐ | 极端 combative、冲动、精神运动性激越；可裸奔、自伤 | "为何如此暴力" → 想 PCP |
| **痛觉减退 Pain insensitivity** ⭐ | 对疼痛 / 创伤无反应 → 可承受重伤而不察觉 | 受伤却不喊痛 |
| **缩瞳 Miosis** ⭐ | 瞳孔可缩小（与多数兴奋剂散瞳相反，易混陷阱）| — |
| 精神病 / 躁狂 | 妄想、幻觉、解离、躁狂样行为 | — |
| 生命体征 | HTN、tachycardia、hyperthermia、肌强直 | — |
| 重症 | seizure、横纹肌溶解、昏迷 | — |

> [!danger] 安全第一
> PCP 患者**极度暴力且不感知疼痛** → 对自己和医护都危险。处理首要 = **环境安全 + 镇静**，再做评估。

## 戒断 Withdrawal

> [!info] PCP 戒断 — 无明确致命戒断综合征
> PCP 没有像酒精 / BZD 那样的明确躯体戒断综合征。
> - 停用后可有情绪低落、嗜睡、渴求、注意力 / 记忆问题（PCP 高脂溶性，可在脂肪组织蓄积、缓慢释放，效应迁延）。
> - 处理 = 支持治疗；无特效戒断药。

---

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **激越 / 暴力（首选）** | **Benzodiazepine（lorazepam IV/IM）一线** —— 镇静、降生命体征、控 seizure |
| 激越顽固 / 明显精神病 | 二线加**抗精神病药（haloperidol）**（注意降癫痫阈值 / 高热风险）|
| 环境 | **低刺激、安静、安全环境**；保护患者与医护安全 |
| **高血压 / 高热** | BZD 一线；顽固 HTN 用 phentolamine / 硝酸酯；主动降温 |
| **Rhabdomyolysis** | 大量补液、监测 K⁺ / 肾功能 |
| 重症 | 气道保护、控 seizure；尿酸化已不推荐 |

> [!tip] 激越首选 BZD
> PCP 致激越**首选 benzodiazepine**（既镇静又降 HR/BP 又抗 seizure）。抗精神病药仅作二线（haloperidol 会降低癫痫阈值、可加重高热，单用不理想）。

## UW Stem 模式

> [!example]- 典型 Stem 模式（中文重述，非 UW 原文）
> - 患者**极端暴力 / 攻击性**被带来急诊，体检发现**水平 + 垂直眼震**、对疼痛刺激反应迟钝 → 锁定 PCP。
> - 题干描述**裸奔 + 暴力 + 多方向眼震** → PCP 经典三联。
> - 激越 PCP 患者，问首选镇静处理 → 答 **benzodiazepine**（非单用 haloperidol）。
> - 鉴别陷阱：瞳孔**缩小**别误判成阿片 —— 阿片是呼吸抑制 + 嗜睡，PCP 是高血压 + 暴力 + 眼震。
> - 尿筛陷阱：服 **NSAID / DXM / DPH / 文拉法辛**者 PCP 免疫筛查可假阳性 → 意外阳性先想假阳性（详见主笔记 §13）。

## Memory Hook

> [!success] 一句话锁定
> **"NMDA 一关，人就解离 —— 眼珠到处乱跳、暴力不怕疼、瞳孔还缩小；镇住他，先给 BZD。"**
>
> 口诀：
> - **PCP 三联**：**多方向眼震 + 暴力 + 痛觉减退**（+ 缩瞳易混陷阱）。
> - **眼震是指纹**：见 nystagmus（水平+垂直+旋转）几乎专属 PCP。
> - **激越**：BZD 第一线，haloperidol 第二线。

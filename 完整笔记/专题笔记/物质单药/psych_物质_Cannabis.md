# 物质单药 — 大麻（Cannabis / Marijuana）

> [!info] 一句话定位
> 大麻 = **CB1/CB2 受体激动**（THC）。中毒指纹 = **红眼（结膜充血）+ 口干 + 瞳孔正常 + 食欲↑ + 心动过速**；高剂量可致焦虑/偏执；慢性重度用者警惕 **cannabinoid hyperemesis（热水澡缓解）**。

---

## 机制

| 维度 | 内容 |
|---|---|
| **活性成分** | **THC**（δ-9-tetrahydrocannabinol）= 精神活性主成分；CBD 不致 high |
| **受体** | 激动 **CB1**（CNS — 欣快 / 认知 / 食欲）+ **CB2**（免疫）|
| **效应** | 欣快 + 放松 + 感知改变 + 食欲↑（"munchies"）+ 时间感扭曲 |
| **检测** | 脂溶性 → 尿筛**慢性使用者可阳性数周**（蓄积于脂肪）|

---

## 中毒表现 Intoxication

| 系统 | 表现 |
|---|---|
| **眼** | **结膜充血 / 红眼（conjunctival injection）** ⭐ —— 经典指纹 |
| **瞳孔** | **正常**（不散不缩 —— 重要鉴别点）|
| **口腔** | **口干（dry mouth）** |
| **食欲** | **↑↑（增加）** —— 与兴奋剂戒断的食欲↑不同语境 |
| **心血管** | **心动过速（tachycardia）**+ 体位性低血压 |
| **精神** | 欣快 / 放松；**高剂量 → 焦虑 / 偏执 / 惊恐 / 急性精神病样反应** |
| **体温** | 正常（**不致 hyperthermia / myoclonus** —— 区别于兴奋剂 / SS）|

> [!warning] Cannabinoid Hyperemesis Syndrome（CHS）⭐
> 慢性重度大麻使用者出现**反复剧烈呕吐 + 腹痛**，**热水澡 / 热水淋浴可缓解**（强迫性洗热水澡是经典 stem 线索）。
> - 机制：长期 THC 致内源性大麻素系统紊乱
> - 治疗：**停用大麻**（根治）+ 急性期 haloperidol / 局部 capsaicin 乳膏
> - 易误诊为 cyclic vomiting —— 用药史 + 热水澡线索区分。

---

## 戒断 Withdrawal

> [!info] 大麻戒断真实存在但不致死
> 慢性重度使用者停药后可出现戒断综合征 —— **难受但不危险**，支持治疗即可。

戒断表现（"兴奋剂戒断的轻量版"）：

- **易怒 / 焦虑 / 攻击性**
- **失眠 + 怪梦**
- **食欲↓**（与中毒时食欲↑相反 —— 镜像）
- 不安、抑郁情绪、躯体不适（头痛 / 出汗 / 腹痛）

---

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **急性中毒 / 焦虑惊恐反应** | **支持治疗** + 安静环境 + 安抚；重度激越可短期 BZD |
| **急性精神病样反应** | 支持 + 观察；多数随药效消退自愈 |
| **CHS** | **停用大麻**（根治）+ 急性期 haloperidol / capsaicin |
| **戒断** | **支持治疗**（无 FDA 批准药物）|

---

## UW Stem 模式

- 年轻人 **红眼 + 口干 + 食欲↑ + 心动过速 + 瞳孔正常** → 大麻中毒
- 大学生吸食后**焦虑 / 偏执 / 惊恐发作** → 大麻高剂量反应（非原发焦虑障碍）
- 慢性大麻使用者**反复呕吐 + 强迫洗热水澡缓解** → Cannabinoid hyperemesis
- 戒大麻者出现**易怒 + 失眠 + 食欲↓** → 大麻戒断（不致死，支持治疗）

---

## Memory Hook

> [!tip] 锁定
> **"红眼干嘴大胃口，心跳快但瞳孔正"** —— 大麻中毒五件套。
> **瞳孔正常**是关键 —— 兴奋剂散瞳、阿片缩瞳、大麻不动瞳。
> **CHS = 抱着热水器吐** —— 慢性大麻 + 反复呕吐 + 洗热水澡缓解。
> 戒断食欲↓ vs 中毒食欲↑ —— 完美镜像。

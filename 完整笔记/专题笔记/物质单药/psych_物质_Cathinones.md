# 物质单药 — 合成卡西酮（Synthetic Cathinones / "Bath Salts"）

> [!info] 一句话定位
> 合成卡西酮 = **苯丙胺类似物（amphetamine analog）**，俗称"浴盐 bath salts"。指纹 = **极端 combative / 暴力 + 精神病 / 谵妄 + 病程最长（数天–1 周）+ 常规尿筛测不出**。是物质中毒里"最疯最久查不到"的那个。

---

## 机制

| 维度 | 内容 |
|---|---|
| **化学本质** | 卡西酮（cathinone）衍生物 —— **结构类似苯丙胺**的合成兴奋剂 |
| **作用** | 促单胺（**多巴胺 / 去甲肾上腺素 / 5-HT**）释放 + 再摄取抑制 → 强拟交感 + 精神兴奋 |
| **效力 / 病程** | 比典型兴奋剂**更强、作用更久** —— 这是"病程长达 1 周"的根源 |
| **检测** | **设计药物（designer drug）—— 常规尿免疫筛查针对 d-amphetamine，测不出 cathinone 衍生物** ⭐ |

> [!warning] 为什么尿筛阴性 ≠ 排除
> 常规 UDS = 免疫法，靠抗体识别特定结构。合成卡西酮结构改造过 → 抗体不识别 → **假阴性**。临床表现永远优先于尿筛结果。确证需 GC-MS / LC-MS。

---

## 中毒表现 Intoxication

| 系统 | 表现 |
|---|---|
| **精神 / 行为** | **极端 combative / 暴力 / 攻击性** + 精神病（幻觉 / 偏执）+ **谵妄 / 激越** ⭐ |
| **病程** | **数天–1 周**（物质中毒里**持续最久** —— 关键鉴别点）|
| **瞳孔** | **散大（mydriasis）** |
| **体温** | **Hyperthermia（高热）** —— 拟交感风暴 |
| **电解质** | **低钠血症（hyponatremia）** —— 类似 MDMA |
| **其他** | 心动过速 + 高血压 + 出汗 + 肌阵挛 + seizure；可致横纹肌溶解 / 肾衰 |

> [!danger] "最疯 + 最久 + 查不到"三连
> 合成卡西酮 stem 的核心识别 = **极端暴力行为 + 病程拖到数天–1 周 + 尿筛阴性**。看到"暴力激越但 tox screen negative"且病程超长 → 想 bath salts。

---

## 戒断 Withdrawal

> [!info] 戒断同其他兴奋剂 —— 不致死
> 类似苯丙胺戒断 = "crash"：**疲乏 + 嗜睡 + 抑郁 + 食欲↑ + 强烈渴求**。生理上不危险，但需监测抑郁 / 自杀念头。

---

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **激越 / 暴力** | **Benzodiazepines** 一线（控制激越 + 降交感）；重度精神病可加抗精神病药 |
| **Hyperthermia** | 积极降温 + 补液 |
| **低钠 / 横纹肌溶解** | 纠正电解质 + 监测肾功能 / CK |
| **环境** | 安全约束 + 低刺激环境（保护患者与医护）|

> [!tip] 兴奋剂中毒激越通用原则
> 物质诱发激越首选 **BZD**（不是直接上抗精神病药）—— BZD 同时降交感、抗 seizure、不降 seizure 阈值。

---

## UW Stem 模式

- 急诊**极度暴力 / combative + 精神病性激越**，约束都压不住 → 合成卡西酮
- "症状持续数天到 1 周"（病程异常长）→ bath salts（vs 可卡因数小时、苯丙胺约 1 天）
- **尿药筛查阴性**但临床明显兴奋剂中毒 → 合成卡西酮 / 合成大麻等设计药物
- hyperthermia + mydriasis + 低钠 + 激越 → bath salts（与 MDMA 高度相似，靠"暴力 + 病程长 + 尿筛阴性"区分）

---

## Memory Hook

> [!tip] 锁定
> **"浴盐 = 最疯、最久、最隐形"** —— 最暴力的激越、最长的病程（≈1 周）、常规尿筛最测不到。
> 它是苯丙胺的"加强隐身版"：拟交感（散瞳 / 高热 / 低钠）全都在，只是更狂更久更难抓。

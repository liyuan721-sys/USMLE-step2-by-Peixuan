# 物质单药 — MDMA（MDMA / Ecstasy / "Molly"）

> [!info] 一句话定位
> MDMA（3,4-亚甲二氧基甲基苯丙胺，"摇头丸"）= **5-HT 主导的兴奋剂 + 致幻剂**。USMLE 招牌：**派对场景 + hyperthermia + 低钠血症（SIADH + 过量饮水）+ serotonin syndrome（+SSRI 放大）+ 磨牙**。

---

## 机制

> [!tip] 核心机制 — 5-HT 主导的促释放剂
> MDMA 是苯丙胺衍生物，但与 meth（DA/NE 主导）不同，它**强烈促进 5-HT 释放并抑制其再摄取**：
> - **5-HT ↑↑↑（主导）** → 共情/亲密感（"entactogen"）、欣快、体温调节失衡、**血清素毒性风险**。
> - DA / NE ↑（次要）→ 兴奋剂样交感激活。
> - 刺激下丘脑 → **ADH 释放 ↑（SIADH）**；叠加派对剧烈运动出汗 + 大量饮水 → **稀释性低钠血症**。

```
MDMA
   │  促 5-HT 释放（主导）+ 抑制 reuptake；DA/NE 次要
   ▼
5-HT ↑↑↑ ──┬─► 欣快 / 共情感
           ├─► 体温失调 → hyperthermia
           └─► +其他 5-HT 药 → Serotonin Syndrome ⭐
   │
   └─► 刺激 ADH → SIADH ＋ 派对大量饮水 → 低钠血症 ⭐
```

---

## 中毒表现 Intoxication

> [!warning] Stimulant intoxication（euphoria + agitation）+ MDMA 特异
> 通用兴奋剂表现：散瞳、HTN、tachycardia、hyperthermia、出汗、激越、欣快。

MDMA **特异并发症**（USMLE 高频）：

| 并发症 | 机制 / 表现 | 一眼线索 |
|---|---|---|
| **Hyperthermia（高热）** ⭐ | 5-HT 致体温调节失衡 + 闷热拥挤舞池环境 | 可极重，致命 |
| **低钠血症 Hyponatremia** ⭐ | **SIADH（ADH↑）+ 过量饮水** | 头痛、恶心、意识改变、**癫痫、脑水肿** |
| **Serotonin syndrome（血清素综合征）** ⭐ | 5-HT 过载；**合用 SSRI/SNRI/MAOI/triptan 时被放大** | 反射亢进 + 肌阵挛 + clonus + 高热 + 自主神经不稳 |
| **磨牙 Bruxism / Grinding teeth** ⭐ | 下颌紧咬、磨牙、牙关紧闭（trismus）| 派对常见线索 |
| 其他 | rhabdomyolysis、肝损、DIC（重症高热）| — |

> [!danger] 低钠 + 高热是 MDMA 两大致命路径
> 年轻人在 rave/派对后**意识改变 / 抽搐**，查血钠低 → 想 MDMA 致 SIADH + 水中毒。**纠正低钠要慎重**（避免渗透性脱髓鞘），症状性者用高渗盐水。

> [!warning] 尿筛陷阱
> 常规 amphetamine 免疫筛查对 MDMA **交叉反应不稳定（可阴可阳）**，阴性不能排除。临床表现优先（详见主笔记 §13）。

## 戒断 Withdrawal

> [!info] MDMA 戒断 — 5-HT 耗竭后的低落
> 无致命躯体戒断综合征。使用后数天因 **5-HT 储备耗竭**出现：
> - 抑郁、易怒、焦虑、疲乏（俗称 "Tuesday/midweek blues"）。
> - 注意力差、食欲改变、睡眠紊乱。
> - 处理 = 支持治疗 + 监测情绪 / 自杀风险；无特效药。

---

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **激越** | **Benzodiazepine 一线**（镇静 + 降温 + 控 seizure）|
| **Hyperthermia** | **主动降温（核心）** + 补液；极重难治高热 → 考虑 dantrolene |
| **低钠血症** | 限水；症状性（抽搐/意识改变）→ **高渗盐水（3% NaCl）**，缓慢纠正 |
| **Serotonin syndrome** | **停所有 5-HT 药** + 支持 + BZD；中-重度 → **cyproheptadine（5-HT2A 拮抗）** |
| **戒断** | 支持治疗 + 情绪监测 |

> [!tip] 三件事别忘
> MDMA 急诊三查：**查体温（高热）、查血钠（低钠）、查 5-HT 体征（反射亢进/肌阵挛 → SS）**。激越统一先上 BZD。

## UW Stem 模式

> [!example]- 典型 Stem 模式（中文重述，非 UW 原文）
> - 年轻人在**音乐节 / 通宵派对**后送急诊，**高热 + 激越 + 散瞳 + 磨牙**，问物质 → MDMA。
> - 派对青年**抽搐 / 意识模糊**，化验**血钠明显偏低** → MDMA 致 SIADH + 过量饮水（水中毒）。
> - 长期服 **SSRI** 的人聚会吸 MDMA 后**反射亢进 + 肌阵挛 + clonus + 高热 + 出汗** → serotonin syndrome；处理 = 停药 + 支持 + cyproheptadine。
> - 鉴别：题干强调"低钠 / SIADH / 派对喝水"几乎专属 MDMA。

## Memory Hook

> [!success] 一句话锁定
> **"血清素摇头丸 —— 烧得发烫、咬牙磨牙、猛灌水把钠冲低；遇上 SSRI，烧成 serotonin syndrome。"**
>
> 口诀：
> - **MDMA 四联**：**Hyper**thermia + **Hypo**natremia + **S**erotonin syndrome + 磨牙。
> - **放大键**：MDMA + SSRI/SNRI/MAOI = SS 风险倍增。
> - **激越**：先 BZD；SS → cyproheptadine；高热 → 主动降温。

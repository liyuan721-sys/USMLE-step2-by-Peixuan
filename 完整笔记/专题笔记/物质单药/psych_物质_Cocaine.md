# 物质单药 — 可卡因（Cocaine）

> [!info] 一句话定位
> Cocaine（可卡因）= **短效拟交感兴奋剂**，封锁单胺再摄取 → 交感风暴。USMLE 招牌考点：**年轻人胸痛 / 急性 MI**（冠脉痉挛）+ **鼻中隔穿孔**，且 **β-blocker 禁用**（unopposed α）。

---

## 机制

> [!tip] 核心机制 — Reuptake blocker
> Cocaine 阻断突触前 **DA / NE / 5-HT 再摄取转运体** → 三种单胺在突触间隙堆积。
> - **NE ↑** → 交感风暴：HTN、tachycardia、散瞳、出汗、血管收缩。
> - **DA ↑** → 欣快、激越、精神病性症状、强化成瘾。
> - 同时有**局部麻醉作用**（阻断 Na⁺ 通道）→ 大剂量可致心肌传导异常 / 宽 QRS。

```
Cocaine
   │  阻断 DA/NE/5-HT reuptake transporter
   ▼
突触间隙单胺 ↑↑
   ├─ NE ↑ → 交感风暴（HTN / 心率↑ / 散瞳 / 血管痉挛）
   └─ DA ↑ → 欣快 / 激越 / 成瘾强化
作用持续：吸食/静注约 30 min（短效）⭐
```

---

## 中毒表现 Intoxication

> [!warning] Stimulant intoxication 通用 + 可卡因特异
> 通用兴奋剂表现：散瞳（mydriasis）、HTN、tachycardia、hyperthermia、出汗、激越、seizure、精神病。

可卡因**特异并发症**（USMLE 高频）：

| 并发症 | 机制 / 表现 | 一眼线索 |
|---|---|---|
| **冠脉痉挛 Coronary vasospasm** ⭐ | NE 致冠脉收缩 → 胸痛 / 急性 MI（即使冠脉本身正常）| **年轻人胸痛 + 用药史** |
| **Rhabdomyolysis（横纹肌溶解）** | 激越 + 高热 + 血管收缩缺血 → 肌肉坏死 | CK ↑↑↑、肌红蛋白尿、AKI |
| **Pulmonary hemorrhage（肺出血）** | "crack lung" — 吸入 crack 损伤肺泡毛细血管 | 咯血、低氧、弥漫浸润影 |
| **Perforated nasal septum（鼻中隔穿孔）** ⭐ | 鼻吸长期使用 → 鼻黏膜血管收缩缺血坏死 | 慢性鼻吸史的招牌体征 |

> [!danger] 可卡因相关急性胸痛 / MI
> 年轻人、无传统冠心病危险因素、突发胸痛 + HTN + tachycardia → 想 cocaine。机制是**冠脉痉挛 + 促血栓 + 心肌需氧↑**三重打击，不一定有固定狭窄。

## 戒断 Withdrawal

> [!info] Stimulant withdrawal — "Crash"，痛苦但不致死
> 中毒兴奋 → 戒断抑制（镜像规律）。生理不致命，**情绪致命（自杀风险）**。

| 维度 | 戒断表现 |
|---|---|
| 情绪 | **严重抑郁（"crash"）⭐ + 强烈快感缺失（anhedonia）+ 自杀念头** |
| 睡眠 | 嗜睡 + **生动、不愉快的噩梦（vivid unpleasant dreams）⭐** |
| 食欲 | 食欲 ↑（hyperphagia）|
| 瞳孔 | **缩瞳（constricted pupils）⭐**（与中毒散瞳相反）|
| 体力 | 极度疲乏、精神运动迟滞 + 强烈渴求（craving）|

> [!tip] 戒断三联记忆
> Cocaine crash = **抑郁 + 生动噩梦 + 缩瞳**。无致命躯体危象，处理 = 支持治疗 + **监测自杀风险**（无 FDA 批准药）。

---

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **激越 / 精神病** | **Benzodiazepine（lorazepam）首选** —— 同时降 HR/BP、控 seizure、镇静 |
| **高血压 / 胸痛** | BZD 一线；顽固 HTN → **phentolamine（α 阻滞）** 或 nitroglycerin / CCB |
| **急性 MI / 缺血** | 阿司匹林 + BZD + 硝酸酯 + CCB；溶栓/PCI 按指征 |
| **Hyperthermia** | 主动降温 + 补液 |
| **Rhabdomyolysis** | 大量补液、监测 K⁺ 与肾功能 |
| **戒断** | 支持治疗 + **自杀风险评估**（无特效药）|

> [!danger] 绝对禁忌 — β-blocker（unopposed α）⭐⭐⭐
> 单纯 β 阻断 → β2 介导的血管舒张被切断，**α 介导血管收缩失去对抗（unopposed α-agonism）→ 冠脉痉挛加重、血压反跳升高**。
> - **不用**纯 β-blocker（如 metoprolol、propranolol）。
> - **用** BZD（一线）/ phentolamine / 硝酸酯 / CCB。
> - labetalol（兼 α/β）存在争议，考试默认选 BZD + phentolamine 最安全。

## UW Stem 模式

> [!example]- 典型 Stem 模式（中文重述，非 UW 原文）
> - **年轻人（20-40 岁）突发胸痛**，BP 升高、心率快，ECG 可见缺血改变，肌钙蛋白可升高 → 问最可能病因 / 最佳处理 → 答 cocaine 相关；处理选 **BZD + 硝酸酯，避开 β-blocker**。
> - **慢性鼻吸毒史 + 鼻中隔穿孔 / 反复鼻出血** → 指向 cocaine。
> - 激越患者 + 散瞳 + 高热 + CK 显著升高 + 肌红蛋白尿 → cocaine 致 rhabdomyolysis。
> - 停用兴奋剂数日后**严重抑郁 + 嗜睡 + 噩梦 + 食欲增加 + 缩瞳** → cocaine withdrawal（"crash"），评估自杀。
> - 急诊医生想给可卡因胸痛患者用 metoprolol → 选"**不应使用 β-blocker**"。

## Memory Hook

> [!success] 一句话锁定
> **"短效兴奋剂，封锁回收泵 → 冠脉一痉挛，胸痛要人命；β 一上去更要命，BZD 才保命。"**
>
> 口诀：
> - **中毒**："散瞳火热、胸痛抽搐、鼻中隔一个洞"。
> - **戒断**："抑郁 + 噩梦 + 缩瞳" = crash 三联。
> - **禁忌**："Cocaine + β-blocker = unopposed α" —— 见到就排除。

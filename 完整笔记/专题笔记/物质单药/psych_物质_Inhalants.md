# 物质单药 — 吸入剂（Inhalants）

> [!info] 一句话定位
> 吸入剂 = 挥发性溶剂 / 气雾剂（胶水、油漆、打火机油、清洁剂、气雾喷罐等）。**青少年线索**最关键 —— 廉价易得。指纹 = **口周皮疹 + 心律失常猝死（"sudden sniffing death"，禁用肾上腺素）**；**无明显 withdrawal**。

---

## 机制

| 维度 | 内容 |
|---|---|
| **种类** | 挥发性溶剂（甲苯 / 汽油 / 胶水）、气雾推进剂、硝酸盐（"poppers"）、笑气（N₂O）|
| **作用** | 多数为弥漫性 **CNS 抑制**（增强 GABA / 抑制 NMDA 等），起效**极快、持续短**（数分钟–半小时）|
| **使用方式** | "huffing"（布上倒溶剂吸）/ "bagging"（袋中吸）/ "sniffing"|
| **人群** | **青少年 / 低龄** —— 便宜、合法、家庭随手可得（重要 stem 线索）⭐ |

---

## 中毒表现 Intoxication

| 系统 | 表现 |
|---|---|
| **神经** | 短暂欣快 → 头晕 / **眼震 nystagmus** / 共济失调 ataxia / 构音障碍 dysarthria / 意识混乱 → 嗜睡 / 昏迷；高剂量 seizure |
| **特异体征** | **口周 / 鼻周皮疹（perioral rash，"glue sniffer's rash"）** ⭐ + 衣物 / 呼气溶剂气味 |
| **心血管** | **心律失常** —— 致命核心机制（见下）|
| **自主神经** | **体位性低血压 orthostatic hypotension（orthostasis，直立性低血压）** + 头晕 —— CNS 抑制 + 血管扩张所致 |
| **病程** | 起效快、**作用时间极短** —— 急诊时可能已大部分消退 |
| **慢性** | 神经损害（白质病 / 认知下降 / 周围神经病）、肝肾损害、骨髓抑制 |

> [!danger] Sudden Sniffing Death Syndrome ⭐⭐⭐
> 吸入剂使心肌**对儿茶酚胺敏感化（catecholamine sensitization）** → 一次惊吓 / 运动 / 情绪激动引发儿茶酚胺激增 → **致命性室性心律失常 → 猝死**。
> - 可发生在**首次使用**的健康青少年。
> - ⚠️ **禁用肾上腺素 / 拟交感药** —— 会进一步敏感化心肌、诱发恶性心律失常。
> - 心律失常处理优先 **β-blocker**（如 esmolol）+ 支持，**不上 epinephrine**。

---

## 戒断 Withdrawal

> [!success] 吸入剂无明显戒断综合征 ⭐
> 吸入剂**不产生显著的生理戒断**。停用可有轻微烦躁 / 渴求，但**无明确 withdrawal 综合征**。
> stem 问"戒断表现"时，吸入剂 ≈ 无（与致幻剂同属"无戒断"组）。

---

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **急性中毒** | **支持治疗** + 保护气道 + 监测；脱离吸入环境 |
| **心律失常** | **β-blocker（esmolol）** + 支持；**禁肾上腺素 / 拟交感药** ⚠️ |
| **激越** | 安静环境；必要时短期 BZD |
| **长期** | 行为治疗 + 戒断咨询（无 FDA 批准药物）；筛查神经 / 肝肾损害 |

> [!warning] 最高频陷阱
> 吸入剂中毒心律失常 → 答案选项里如有 **epinephrine** 一定是陷阱。机制 = 心肌已被溶剂敏感化，再给儿茶酚胺 = 火上浇油。

---

## UW Stem 模式

- **青少年** + **口周皮疹** + 衣物溶剂气味 + 行为改变 → 吸入剂滥用
- 健康青少年吸入剂后受惊吓 / 跑动 → **突然心搏骤停 / 猝死** → sudden sniffing death
- 吸入剂中毒心律失常，选治疗 → β-blocker（**排除 epinephrine**）
- 短暂欣快 + 共济失调 + 很快消退 + 低龄患者 → 吸入剂（起效快、作用短）
- **青少年 + 眼震 nystagmus + 体位性低血压 orthostasis + 嗜睡** → 吸入剂（CNS 抑制谱）；**误选 LSD 是经典陷阱**（见下鉴别）
- 问"停用后戒断" → 吸入剂**无明显 withdrawal**

> [!warning] 吸入剂 vs LSD 鉴别（Q028 翻车点）⭐
> 两者都"青少年 + 精神状态改变"，但谱系相反：
> - **吸入剂 = CNS 抑制**：**眼震 + 体位性低血压 + 共济失调 + 嗜睡**，起效快作用短，口周疹/溶剂味；可猝死。
> - **LSD (lysergic acid diethylamide 麦角酸二乙酰胺) = 拟交感 + 致幻**：**瞳孔散大 + 心动过速 + 高血压 + 生动幻觉/感知扭曲**，**无眼震、无体位性低血压**。
> - 钥匙：**"眼震 + 直立性低血压 + 嗜睡（抑制相）" → 吸入剂，不是 LSD**（LSD 是兴奋/拟交感，不会低血压嗜睡）。源 [[NBME11_错题本#^Q028]]。

---

## Memory Hook

> [!tip] 锁定
> **"青少年 + 口周疹 + 一惊就猝死"** —— 吸入剂三件套。
> **CNS 抑制谱**：眼震 nystagmus + 体位性低血压 orthostasis + 共济失调 + 嗜睡 → 别误当 LSD（LSD 是拟交感兴奋：散大瞳孔 + 高血压 + 幻觉，不低血压不嗜睡）。
> **Sudden sniffing death** = 心肌被溶剂"调成易燃"，肾上腺素是火柴 → **禁 epinephrine**。
> 致幻剂、吸入剂 = "无戒断双子" —— 停了不抽不闹。

# 物质单药 — 苯二氮䓬（Benzodiazepines, BZD）

> [!info] 一句话定位
> BZD = **增强 GABA-A 开放频率**的 CNS 抑制剂。**单用很少致死**，但 **+ 酒精/阿片 → 呼吸抑制致死**；戒断与酒精同属"会 seizure 致死"。**戒断 / 停药综合征已在 hub 笔记 §9/§16 深挖**——本页精简 + 链回。

---

## 机制

| 维度 | 内容 |
|---|---|
| **作用位点** | GABA-A 受体 → BZD **增加氯离子通道开放频率**（巴比妥 = 增加开放时长）|
| **净效应** | CNS 抑制：镇静 / 抗焦虑 / 抗惊厥 / 肌松 / 顺行性遗忘 |
| **半衰期分组** | 短效（alprazolam / lorazepam / midazolam）vs 长效（diazepam / chlordiazepoxide / clonazepam）|
| **慢性适应** | 同酒精 —— GABA-A 下调 → 撤药后净兴奋 |

> [!tip] BZD vs 巴比妥
> BZD = 频率↑（有"封顶效应"，单用安全窗大）；巴比妥 = 时长↑（无封顶，单用即可致死）。这就是为什么 BZD 单用很少致命。

---

## 中毒表现 Intoxication

| 系统 | 表现 |
|---|---|
| **神经** | **镇静 / 嗜睡 + 共济失调 + 言语含糊** + 顺行性遗忘 |
| **生命体征** | **正常**（关键鉴别点 —— 区别于阿片的呼吸抑制）|
| **瞳孔** | 正常（非诊断性）|

> [!danger] 单用 vs 联用 —— 致死的关键
> - **BZD 单用过量** → 嗜睡 / 共济失调，**很少致命**（呼吸驱动相对保留）
> - **BZD + 酒精**（或 + 阿片）→ **协同呼吸抑制 → 致死** ⭐
> - UW 高频陷阱：见"昏迷 + 呼吸抑制"别只想阿片 —— 问联用史。

---

## 戒断 Withdrawal

> [!danger] BZD 戒断会致死 ⭐
> 与酒精同 GABA-A 通路 —— 戒断可致 **seizure**、自主神经风暴。**必须缓慢减量，不能骤停**。

> [!warning] 半衰期决定 withdrawal 风险与发作时机 ⭐
> - **短效 BZD**（alprazolam / lorazepam）→ 戒断**发作快、症状重**（"断崖式"血药下降）
> - **长效 BZD**（diazepam / chlordiazepoxide）→ 戒断**发作慢、症状缓**（自带"渐减"缓冲）
> - 因此换药脱毒：用**长效 BZD 替代后缓慢递减**。

戒断表现：焦虑、震颤、失眠、心慌、知觉过敏 → 重者 **seizure / 谵妄**。

> [!info] 半衰期 → withdrawal 风险完整表 / SSRI discontinuation 鉴别
> → 见 hub 笔记 **§16.3 BZD 专题** + **§9.3**（[[完整笔记/专题笔记/psych/psych_物质急诊速查]]）。

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **急性中毒（单用）** | **支持治疗** + 监测气道；多数自愈 |
| **急性中毒（+ 酒精/阿片）** | 气道管理 / 通气支持优先 |
| **Flumazenil** | BZD 受体拮抗剂 —— **慎用** ⚠️（见下）|
| **戒断** | 重启 BZD（长效）后**缓慢减量** |

> [!warning] Flumazenil 为什么慎用 ⚠️
> - **慢性 BZD 使用者用 Flumazenil → 诱发急性戒断 → seizure**（解除了 GABA 增强 → 突发净兴奋）
> - 若患者同时服用降低 seizure 阈值的药（如 TCA）→ 更危险
> - **适应症仅限：单纯医源性过镇静、明确无慢性 BZD 依赖、无 seizure 风险**
> - 多数 BZD 过量 → **支持治疗即可，不常规用 Flumazenil**。

## UW Stem 模式

- "嗜睡 + 共济失调 + 言语含糊"但**生命体征正常** → BZD 中毒（vs 阿片有呼吸抑制）
- 昏迷 + 呼吸抑制 + 派对/饮酒史 → BZD **+ 酒精**协同致死
- 长期服 alprazolam 患者骤停后焦虑 / 震颤 / seizure → BZD 戒断
- 急诊给 Flumazenil 后患者 seizure → 慢性 BZD 用者被诱发急性戒断
- "选哪个药脱毒" → 长效 BZD 替代后缓慢减量

---

## Memory Hook

> [!tip] 锁定
> **"单飞安全，结伴致命"** —— BZD 单用过量少死人，配酒/阿片就压呼吸。
> **半衰期 = 戒断脾气** —— 短效药戒断"暴脾气"（快而重），长效药戒断"温和"（慢而缓）。
> **Flumazenil = 拔插头** —— 慢性依赖者被拔 → seizure。

# 物质单药 — 阿片类（Opioids）

> [!info] 一句话定位
> Opioids（阿片类：heroin / fentanyl / oxycodone / morphine 等）= **μ 受体激动剂**。USMLE 招牌：中毒**三联 = 针尖瞳 + 呼吸抑制 + CNS 抑制**（致命的是呼吸抑制）；戒断"反向阿片"**痛苦但不致死**。
>
> ⚠️ 本页只做**单药考点速记 + 重链回主笔记**。阿片**戒断细节 / 处方监测 / 维持期 3 药对比**已在主笔记 [[完整笔记/专题笔记/psych_物质急诊速查]] §9 / §14 / §15 深挖，本页不重复其大表格。

---

## 机制

> [!tip] 核心机制 — μ 阿片受体激动
> 阿片激动 **μ（主要）/ κ / δ** 阿片受体（Gi 偶联）：
> - **μ 激动** → 镇痛、欣快、镇静、**呼吸抑制（致死机制）**、缩瞳、便秘。
> - 抑制脑干呼吸中枢对 CO₂ 的反应 → **低通气 → 高碳酸血症 → 呼吸停止**。
> - 作用于 Edinger-Westphal 核 → **缩瞳（miosis）**。
> - 抑制肠道蠕动 → 便秘（耐受不产生 → 长期用便秘持续）。
> - 中脑边缘 DA 通路被去抑制 → 欣快、强成瘾。

---

## 中毒表现 Intoxication

> [!danger] 阿片中毒三联（USMLE 最高频）⭐⭐⭐
> **① 针尖瞳（pinpoint miosis）+ ② 呼吸抑制（致命）+ ③ CNS 抑制（嗜睡 → 昏迷）**
> 见到"针尖瞳 + 呼吸慢/浅 + 意识抑制" → 立即想阿片过量。

| 表现 | 说明 |
|---|---|
| **针尖瞳 Miosis** ⭐ | 招牌体征（注意：合并缺氧 / 合用其他药时瞳孔可不典型）|
| **呼吸抑制** ⭐ | 致死机制 —— 呼吸频率↓、潮气量↓、发绀、呼吸停止 |
| **CNS 抑制** | 嗜睡、反应迟钝 → 昏迷 |
| 其他 | 便秘、低血压、心率↓、肠鸣音↓、欣快（早期）|
| 并发症 | 非心源性肺水肿、误吸、低氧脑损伤 |

> [!warning] 易混
> 瞳孔缩小也见于 organophosphate / clonidine / 桥脑出血 / PCP；但"针尖瞳 + **呼吸抑制** + 意识抑制"组合最指向阿片。

## 戒断 Withdrawal

> [!info] 阿片戒断 = "反向阿片" —— 极度痛苦，但不致死
> 中毒抑制 → 戒断兴奋（镜像规律）。**致命戒断只有 alcohol / BZD / barbiturate；阿片不在其中。**

阿片戒断"反向阿片"指纹（与中毒处处相反）：

| 中毒 | → 戒断（镜像）|
|---|---|
| 缩瞳 | **散瞳 Mydriasis / dilated pupils** |
| 便秘、肠鸣音↓ | **腹泻、N/V、绞痛、肠鸣音亢进** |
| 镇静 | 激越、焦虑、失眠 |
| —— | **竖毛 Piloerection（"cold turkey"鸡皮疙瘩）** |
| —— | **流泪 Lacrimation + 流涕 Rhinorrhea + 打哈欠 Yawning** |
| —— | 肌痛 / 关节痛、出汗、HTN / 心率↑（拟交感样）|

> [!tip] 流感样 + 鸡皮疙瘩 = 阿片戒断
> "myalgia + 流涕 + 流泪 + 打哈欠 + 鸡皮疙瘩 + 腹泻" = flu-like 戒断画像。**难受但成人通常不致死** —— 戒断管理细节见主笔记 §9。

## 治疗 Management

| 场景 | 处理 |
|---|---|
| **急性过量（中毒）** | **Naloxone**（μ 拮抗剂）—— 逆转呼吸抑制，鼻喷 / IM / IV；保护气道、通气支持 |
| **激越管理** | 阿片相关激越罕见（阿片是镇静性）；戒断期激越 → 对症 + 戒断管理；一般物质性激越仍循 BZD 原则 |
| **戒断管理** | 见主笔记 §9 —— **Methadone / Buprenorphine**（替代）或 **Clonidine**（对症抑交感）|
| **维持期 / 防复发** | 见主笔记 §15 —— Methadone / Buprenorphine / Naltrexone |

> [!warning] 4 个 "N/M/B" 药一句话角色（细节见主笔记 §15）
> - **Naloxone** = 急性过量**救命**（短效，30 min 代谢完，可能把患者推入急性戒断）。→ §15
> - **Methadone** = μ **完全激动剂**，戒断 + 维持，OTP 诊所专管。→ §15
> - **Buprenorphine** = μ **部分激动剂**（有"天花板效应"，过量风险低），门诊可开；体内有阿片时给会诱发 precipitated withdrawal。→ §15
> - **Naltrexone** = μ **拮抗剂**，**防复发**（不治戒断），需完全脱毒后才能用。→ §15

> [!danger] 三个临床阶段别混（详见主笔记 §15.7）
> **过量（→ Naloxone 救命）≠ 戒断（→ Methadone/Buprenorphine/Clonidine 管症状）≠ 维持（→ Naltrexone 防复发）**。

## UW Stem 模式

> [!example]- 典型 Stem 模式（中文重述，非 UW 原文）
> - 患者**昏睡 / 无反应，呼吸浅慢，瞳孔针尖样** → 阿片过量；处理 = **naloxone** + 通气支持。
> - 停用海洛因后**散瞳 + 流涕 + 流泪 + 打哈欠 + 鸡皮疙瘩 + 腹泻 + 肌痛** → 阿片戒断；问"哪条体征支持戒断"→ 选 piloerection / 散瞳等"反向阿片"项。
> - 鉴别陷阱：把阿片戒断（散瞳）误当阿片中毒（缩瞳）—— 中毒缩瞳、戒断散瞳，方向相反。
> - 戒断 vs 致命：题干问"哪种戒断可致死" → 阿片戒断**不致死**（致死的是 alcohol / BZD）。
> - 体内仍有阿片时给 buprenorphine / naltrexone → 诱发 **precipitated withdrawal**（细节见 §15）。

## Memory Hook

> [!success] 一句话锁定
> **"μ 一激动 → 针尖瞳、不喘气、睡过去（致命）；一停药 → 瞳孔散、鸡皮疙瘩、又拉又流泪（难受不致命）。"**
>
> 口诀：
> - **中毒三联**：**针尖瞳 + 呼吸抑制 + CNS 抑制** → naloxone 救命。
> - **戒断**："反向阿片" = 散瞳 / 腹泻 / 竖毛 / 流泪 / 打哈欠，**痛苦不致死**。
> - **4 药**：Nalox**one** 救命、**M**ethadone/**B**uprenorphine 替代、**N**altrexone 防复发。

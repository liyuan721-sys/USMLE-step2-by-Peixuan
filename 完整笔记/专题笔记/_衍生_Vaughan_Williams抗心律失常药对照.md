# _衍生_Vaughan Williams 抗心律失常药对照（Antiarrhythmic Drugs Library）

> [!info] 本笔记定位
> **跨章节药理 + 电生理整合枢纽**：USMLE Step 2 CK 抗心律失常药最容易混的领域。本笔记把 **4 大类 + 14 个必背药 + 5 急救场景 + 3 禁忌组合 + QT 延长病因 + LQTS 三亚型** 全部铺平到一处。
> 
> **核心信念**：抗心律失常药 = 心肌动作电位 4 个时相对应 4 个 class。**先记框架，再填药；先记 QT 效应，再记副作用**。
> 
> **来源**：错题 Q3910（J&L-N LQTS）+ 后续 Brugada / CPVT / 获得性 QT 延长 / TdP / WPW / SVT 等错题积累

## Part 1. 核心框架 — 心肌动作电位 → 4 Class

```
心肌动作电位 5 个时相：

         Phase 1 (K out 短暂)
        ↗     Phase 2 (Ca in 平台)
Phase 0 →     ↘
(Na in)       Phase 3 (K out 复极) ←← Class III
       ↘                            ↘
        Phase 4 (静息 + SA/AV 起搏) ←← Class II / IV
        ↑↑
   Class I (Na 通道阻滞)
```

| Class | 阻滞通道 | 影响时相 | 心电图改变 |
|---|---|---|---|
| **I** | **Na** | Phase 0 ↓ | QRS 增宽（IC 最明显）|
| **II** | β-AR → ↓ cAMP → ↓ Ca | Phase 4 起搏速度↓ | HR↓ + PR↑ |
| **III** | **K** | Phase 3 延长 | **QT 延长** |
| **IV** | **Ca**（非 DHP）| Phase 0/2 ↓ | HR↓ + PR↑（不影响 QRS）|

> [!success] 终极框架口诀
> **"1 钠 2 β 3 钾 4 钙"** + **"1A 长，1B 短，2β 4 钙是真朋友，3 全长（除阿米奥），1C 不动 QRS 宽"**

## Part 2. Class I 详解（Na 通道阻滞）

### 2.1 三亚型对比

| | **IA** | **IB** | **IC** |
|---|---|---|---|
| **结合 / 解离速度** | 中速 | 快速 | 慢速 |
| **代表药** | Quinidine, Procainamide, Disopyramide | **Lidocaine**（IV）, **Mexiletine**（PO）| Flecainide, Propafenone |
| **QT 效应** | **延长 QT** ⭐ | **缩短 QT** | ↑QRS（不长 QT）|
| **QRS** | 轻度延长 | 不变 | **明显延长** |
| **临床应用** | AFib 转复（罕用）、WPW + AFib（Procainamide）| **室速急救**（Lidocaine）+ LQT3（Mexiletine）| AFib 转复（**无结构性心脏病**）|
| **LQTS** | ❌ 禁用 | ✓ 安全；**LQT3 特别用** | 谨慎 |
| **副作用记忆点** | **Cinchonism**（Quinidine）/ **狼疮**（Procainamide）/ TTP | CNS 毒性（震颤 / 抽搐）| **CAST 试验：MI 后增死亡率** ❌ |

### 2.2 IA — 必考副作用记忆点

| 药 | 关键副作用 |
|---|---|
| **Quinidine** | **Cinchonism**（耳鸣 / 头痛 / 头晕 / 视觉异常）+ TTP + 血小板↓ + 延长 QT → TdP |
| **Procainamide** | **药物性狼疮**（最常见诱发狼疮的药）+ 粒细胞缺乏 + 延长 QT |
| **Disopyramide** | **强抗胆碱**（口干 / 尿潴留 / 视力模糊）+ 负性肌力（HF 禁用）|

### 2.3 IB — 唯一缩短 QT 的 Class I

> [!tip] 为什么 IB 缩短 QT？
> IB 阻滞**晚期 Na 内流**（INa late）→ Phase 2/3 缩短 → 动作电位时长 (APD) ↓ → QT ↓
> **LQT3 = INa 持续异常增加** → Mexiletine 正好对症

| 药 | 给药 | 特殊用途 |
|---|---|---|
| **Lidocaine** | **IV only**（口服首过代谢高）| 室速急救（尤其 MI 急性期）|
| **Mexiletine** | **PO** | **LQT3 治疗**（唯一对 LQT3 有效的抗心律失常药）|

### 2.4 IC — CAST 试验阴影

> [!danger] CAST 试验（必考）
> **冠心病 / MI 后患者用 Flecainide → 死亡率↑**（致命性心律失常）
> 
> **铁律**：IC 类**仅用于结构性心脏病阴性**患者的 AFib 转复（"pill-in-the-pocket"）

| 药 | 用途 |
|---|---|
| **Flecainide** | 健康心脏的 AFib 转复 |
| **Propafenone** | 同上 |

## Part 3. Class II 详解（β-blocker）

### 3.1 抗心律失常用的 β-blocker

| 药 | 选择性 | 特点 |
|---|---|---|
| **Propranolol** | 非选择性 | LQTS 经典首选；穿血脑屏障 |
| **Nadolol** | 非选择性 | LQTS 首选（半衰期长，QD 给药，依从性好）⭐ |
| **Metoprolol** | β1 选择 | 心衰 / MI 后 |
| **Esmolol** | β1 + **超短半衰期** | **IV 输注，急救可滴定**（甲危 / 主动脉夹层 / 围术期）|

### 3.2 Class II 适应症

- **LQTS 一线**（90% 的 LQTS 题选 β-blocker）
- AFib / 房扑：**控心率**
- AVNRT：终止 + 长期预防
- VT（部分场景，慢性预防）
- MI 后 / HFrEF：减少猝死（Carvedilol / Metoprolol succinate / Bisoprolol 三大）

### 3.3 禁忌

> [!warning] β-blocker 禁忌
> - 严重哮喘 / COPD 急性（用 β1 选择如 Metoprolol，但慎用）
> - 急性失代偿心衰
> - 2-3 度 AV block（无起搏器）
> - 严重心动过缓 / 病窦
> - 急性可卡因中毒（unopposed α → HTN 危象，慎用）
> - 嗜铬细胞瘤未先 α-block（同上）

## Part 4. Class III 详解（K 通道阻滞）

### 4.1 4 个药对比

| 药 | 特点 | 延长 QT | TdP 风险 |
|---|---|---|---|
| **Amiodarone** | **多通道效应**（I/II/III/IV 都有）+ 半衰期 60 天 | 是 | **罕见**（保护性多通道平衡）|
| **Sotalol** | **K + β-blocker 双效**，**K 占主导** | **强** | **高** ⭐ |
| **Ibutilide** | IV 急性转复 AFib | 强 | 高 |
| **Dofetilide** | 严格剂量 / 住院启用 | 强 | 高 |

### 4.2 Amiodarone 的 5 大副作用 ⭐⭐⭐

> [!danger] Amiodarone "五脏六腑"副作用
> 1. **肺纤维化**（最致命，长期最严重）
> 2. **甲状腺**：可甲亢 or 甲减（Amio 含 39% 碘）
> 3. **肝毒性**：转氨酶↑
> 4. **角膜微沉积**（多数无症状，视力变化）
> 5. **皮肤蓝灰色变**（"Smurf 皮肤"，长期高剂量）
> 
> **必查基线 + 定期监测**：PFT / TFT / LFT / 眼科

### 4.3 Sotalol vs Amiodarone（USMLE 高频鉴别）

| | Sotalol | Amiodarone |
|---|---|---|
| 主导效应 | K 阻滞（III）| 多通道 |
| TdP 风险 | **高** | 罕见 |
| 长期副作用 | 少 | 多（肺 / 甲 / 肝 / 角膜 / 皮肤）|
| LQTS | ❌ **禁忌** | 可用（多通道平衡）但谨慎 |
| 适应症 | AFib / VT | AFib / VT / VF |

## Part 5. Class IV 详解（Ca 通道阻滞 — 非 DHP）

| 药 | 用途 | 特点 |
|---|---|---|
| **Verapamil** | AVNRT 终止、AFib 控率 | **强负性肌力**（HF 禁用）+ 便秘 |
| **Diltiazem** | AFib 控率（HF 较 Verapamil 安全）| 心率控制首选 |

> [!warning] Class IV 禁忌
> - 失代偿心衰（HFrEF）
> - 2-3 度 AV block
> - **WPW + 房颤**（与 AV 结阻滞剂禁忌组合，致室颤）⭐
> - 不与 β-blocker 联用（双重 AV 抑制）

---

## Part 6. 其他必备药

| 药 | 类别 | 用途 | 关键点 |
|---|---|---|---|
| **Adenosine** | AV 结超短抑制 | **SVT（AVNRT）一线急救** | IV push + 半衰期 < 10 秒；**WPW + 房颤禁用** |
| **Magnesium** | — | **TdP 一线**（不论血 Mg）| 1-2g IV bolus + 维持 |
| **Digoxin** | Na/K-ATPase 抑制 | HFrEF + AFib 控率（罕用）| 治疗指数窄 + 低 K 增毒 + 黄绿视 |
| **Ivabradine** | If 通道（窦房结）| HFrEF 窦速 / 不耐 β-blocker | 不影响传导 / 收缩力 |

## Part 7. 5 急救场景 ⭐⭐⭐

| 急救 | 一线 | 二线 / 特殊 |
|---|---|---|
| **SVT 急性**（稳定）| Vagal 动作 → **Adenosine IV push** | Verapamil / Diltiazem / β-blocker；不稳定 → 电复律 |
| **室速稳定** | **Amiodarone IV** / Lidocaine IV | 不稳定 → 电复律 |
| **室颤 / 无脉室速** | **电除颤**（不等药）| 复苏后 Amiodarone |
| **Torsades de Pointes** | **Magnesium IV**（不论血 Mg）| 反复发作 → 加快心率（Isoproterenol / overdrive pacing） |
| **WPW + AFib** | **Procainamide / 电复律** | **绝对禁忌 AV 结阻滞剂**（β / Ca / Adenosine / Digoxin → 致 VF）|

## Part 8. 3 禁忌组合 ⭐⭐⭐

| 组合 | 后果 | 替代 |
|---|---|---|
| **LQTS + Class IA / III**（Quinidine / Sotalol / Ibutilide / Dofetilide）| 延长 QT → TdP | **β-blocker**（II）✓ |
| **WPW + AFib + AV 结阻滞剂**（β / Ca / Adenosine / Digoxin）| 旁路下传↑ → **VF** | **Procainamide / 电复律** ✓ |
| **结构性心脏病 / MI 后 + Class IC**（Flecainide / Propafenone）| 致命性心律失常 ↑（CAST 试验）| Amiodarone / β-blocker ✓ |

## Part 10. QT 延长完整病因（UWorld 官方表）⭐⭐⭐

### 10.1 完整双层分类

| 大类 | 子类 | 具体 |
|---|---|---|
| **Acquired**（获得性）| **药物** | • Macrolides & Fluoroquinolones |
| | | • Antiemetics（**Ondansetron**）|
| | | • Azoles（**Fluconazole**）|
| | | • Antipsychotics / TCAs / **SSRIs**（尤 Citalopram）|
| | | • Some opioids（**Methadone, Oxycodone**）|
| | | • **Class IA**（Quinidine）|
| | | • **Class III**（Dofetilide, Sotalol）|
| | **电解质** | ↓ Magnesium / ↓ Potassium / ↓ Calcium |
| | **其他** | 心动过缓 / 颅内出血 / 甲减 / 低温 |
| **Congenital**（先天性）| Romano-Ward | AD，仅心脏 |
| | **Jervell & Lange-Nielsen** | AR + **感音神经性聋** ★ |

### 10.2 Peixuan 5 行口诀（原样保留）

> [!tip] QT 延长病因 5 行助记
> 1. **"IA，III 类 QT 长"**（抗心律失常 IA + III）
> 2. **"大环三环喹诺酮"**（Macrolides + TCA + Fluoroquinolones）
> 3. **"抗精抗吐昂丹司琼"**（抗精神病 + Ondansetron）
> 4. **"氯喹噻嗪阿唑钠"**（Chloroquine + Thiazide 致低 K 间接 + Azoles + Na 类）
> 5. **"哪位鸦片镁钾钙（降低）"**（Methadone + opioids + ↓ Mg / K / Ca）

## Part 11. LQTS 三亚型 + 治疗

### 11.1 三亚型对比 ⭐⭐⭐

| 维度 | **LQT1** | **LQT2** | **LQT3** |
|---|---|---|---|
| 基因 | KCNQ1 | KCNH2 (hERG) | SCN5A |
| 通道 | IKs↓ | IKr↓ | INa 持续 |
| **触发** | **运动 / 游泳** ★ | **听铃 / 惊吓 / 产后 / 低 K** | **睡眠 / 安静** |
| ECG T 波 | **宽底** | **切迹 / 双相** | **长 ST + 晚出 T** |
| 猝死率 | 中 | 中-高 | **最高** ⭐ |
| β-blocker 反应 | **极好** ⭐⭐⭐ | 中等 ⭐⭐ | **较差** ⭐ |
| **特殊治疗** | 严格避游泳 | **补 K + Mg** | **Mexiletine (IB)** ⭐ |
| ICD 阈值 | 高 | 中 | **低** |
| 运动限制 | 严格 + 避游泳 | 中度 | **不严格**（运动反保护）|
| J&L-N 可能 | **可能（纯合）** | 极罕见 | 不出现 |

### 11.2 Romano-Ward vs Jervell & Lange-Nielsen

| | Romano-Ward | **J&L-N** |
|---|---|---|
| 遗传 | **AD**（杂合）| **AR**（纯合）|
| 受累 | 仅心脏 | **心脏 + 双侧感音神经性聋** ★ |
| 基因 | LQT1/2/3/...17 任一杂合 | **必是 KCNQ1 纯合（= LQT1 范畴）** |

### 11.3 LQT 治疗哲学

> [!success] "避诱因 + β-blocker + ICD" = 90% 答案
> - 90% 题选 β-blocker
> - 10% 题选 ICD（已发生心脏骤停 / 反复事件）
> - LQT3 加 Mexiletine
> - 几乎不选其他抗心律失常药（IA / III 都禁忌）

### 12.3 LQT 触发口诀

> **"游泳 1，听铃 2，睡觉 3"**

### 12.4 LQT ECG T 波口诀

> **"1 宽 2 双 3 晚"**

### 12.5 LQT 治疗个性化口诀

> **"1 β 救命，2 补 K 防铃，3 美西律救场"**

### 12.6 QT 延长病因 5 行（见 Part 10.2）

### 12.7 急救场景一句话

> **"SVT 找腺苷，室速找胺碘，室颤先电击，扭转找镁离，WPW 找普鲁卡因"**

### 12.8 类比理解

- **LQTS 心脏 = 充电慢的手机**：复极慢 → 高心率时来不及充满 → 触发崩溃（TdP）
- **β-blocker = 让手机用慢点**：减慢心率 → 给心脏足够时间复极
- **游泳 / 听铃 / 惊吓 = 突然高负荷使用**：手机崩溃
- **Quinidine / Sotalol = 把充电速度调更慢**：火上浇油
- **Mexiletine = 快充贴**：缩短充电时间（缩短 QT）

## Part 13. 跨学科陷阱

| 学科 | 陷阱 | 关键点 |
|---|---|---|
| **感染** | 开 Macrolides / Fluoroquinolones / Azoles 不查 QT | 老人 / 心脏病 / 多重用药患者 → 查 ECG 基线 + 复查 |
| **精神科** | 抗精神病 / TCAs / SSRIs（**Citalopram > 40 mg / d 黑框**）| 老人 ≤ 20 mg / d |
| **疼痛科** | Methadone 维持治疗 | 监测 QTc + 警惕骤死 |
| **GI** | Ondansetron 反复用 | 高剂量 / 多重用药 → 查 ECG |
| **肿瘤** | 化疗药（Anthracycline + Cisplatin + 5-FU + TKI 多数）| 监测 ECG + Echo |
| **内分泌** | 甲减 → 长 QT；甲亢治疗后突然甲减 | 替代 thyroxine |
| **肾** | CKD + 低 K / 低 Mg | 补充 + 避免延长 QT 药 |
| **儿科** | 儿童游泳后晕厥 / 兄弟猝死 | 立刻想 LQTS |

## Part 14. 高频陷阱总结

| 陷阱 | 正确认知 |
|---|---|
| ❌ 遗传性离子通道病 → 用抗心律失常药 | ✓ LQTS 反而禁用大多数；β-blocker 是基础 |
| ❌ Quinidine 是 III 类 | ✓ Quinidine 是 **IA** |
| ❌ Sotalol 是 β-blocker 所以 LQTS 可用 | ✓ Sotalol K 通道阻滞主导，仍延长 QT，**禁忌** |
| ❌ Amiodarone 延长 QT 所以禁忌 | ✓ Amiodarone 致 TdP 罕见（多通道平衡），LQTS 可用但谨慎 |
| ❌ β-blocker 缩短 QT | ✓ β-blocker 不直接缩短 QT，通过抑交感预防 TdP 触发 |
| ❌ Torsades 用 β-blocker 急救 | ✓ 急性期加重；用 **Magnesium** + 加快心率 |
| ❌ Adenosine 治所有 SVT | ✓ **WPW + AFib 禁用**！会致 VF |
| ❌ Verapamil 治所有快速心律 | ✓ **WPW + AFib 禁用 + HF 禁用** |
| ❌ Flecainide 健康人都能用 | ✓ **结构性心脏病 / MI 后禁用**（CAST）|
| ❌ Lidocaine 用于所有心律失常 | ✓ 仅 **室速 / 室颤** + IV 给药（口服首过代谢高）|
| ❌ Mexiletine 是新型抗心律失常 | ✓ IB 类，**LQT3 特别用**（缩短 QT）|

## 📋 快速锁定：Stem 关键词 → 综合征

| Stem 关键词 | 立刻想到 |
|---|---|
| **重叠指 + 摇椅足 + 突出枕骨** | **Edwards (T18)** |
| **唇腭裂 + 多指 + 皮肤缺损 + 小眼** | **Patau (T13)** |
| **单掌纹 + 上斜眼 + 舌大 + 肌张力低** | **Down (T21)** |
| **新生儿手足淋巴水肿 + 颈蹼 + 宽间距乳头** | **Turner (45,XO)** |
| **高个 + 男性乳房 + 学习障碍 + 小睾** | **Klinefelter (47,XXY)** |
| **低钙抽搐 + 反复感染 + 主动脉弓异常** | **DiGeorge (22q11del)** |
| **精灵脸 + 高钙 + 主动脉瓣上狭窄 + 外向** | **Williams (7q11del)** |
| **新生儿严重低张 + 喂养困难 → 后期肥胖暴食** | **Prader-Willi (15q11 父源缺失)** |
| **快乐木偶 + 共济失调 + 癫痫 + 笑** | **Angelman (15q11 母源缺失)** |

| **大睾 + 长脸 + 大耳 + 智障 + 自闭** | **Fragile X (FMR1 三核苷酸重复)** |

## 🔬 三大常染色体三体综合征对比

| 项目 | **Trisomy 21 (Down)** | **Trisomy 18 (Edwards)** | **Trisomy 13 (Patau)** |
|---|---|---|---|
| **发病率** | 1/700（最常见活产染色体病） | 1/6000 | 1/10000 |
| **母亲年龄** | 强相关（>35岁风险↑） | 相关 | 相关 |
| **预后** | 寿命 ~60 岁 | **>50% 1月内死亡，<10% 活过 1 岁** | **80% 1月内死亡** |
| **面容** | 上斜眼、内眦赘皮、扁脸、舌大、低耳 | **小下颌、突出枕骨**、小眼 | **唇腭裂、小眼/无眼、前额倾斜** |
| **手** | **单掌纹 (simian crease)**、第 5 指内弯 | **重叠指**（clenched fist）、掌纹缺失 | **多指 (polydactyly)** |
| **足** | 拇趾分离（sandal gap） | **摇椅足 (rocker-bottom)** | **摇椅足 + 多趾** |
| **心脏** ⭐ | **AVSD** > VSD/ASD | **VSD** > ASD/PDA | **VSD / PDA / ASD（混合）** |
| **神经** | 智力障碍、肌张力低、**早发 Alzheimer (40+)** | 严重发育迟缓、张力异常 | **Holoprosencephaly（前脑无裂）**、严重智障 |
| **消化** | **十二指肠闭锁 (double-bubble sign)**、Hirschsprung | 脐膨出、肠旋转不良 | 脐膨出 |
| **泌尿** | 一般正常 | 马蹄肾 | 多囊肾 |
| **血液/肿瘤** | **ALL / AML 风险↑**、新生儿一过性骨髓增殖 | — | — |
| **内分泌** | **甲减风险↑** | — | — |
| **喂养** | 喂养差（肌张力低 + 舌大 + 心脏疲劳） | **严重喂养困难**（小下颌 + 张力差） | 严重喂养困难（唇腭裂 + 中枢异常） |
| **其他** | 第一/二颈椎不稳（atlantoaxial instability）→ 运动前查 X 线 | 多脏器畸形 | **皮肤缺损 (cutis aplasia)** |
| **产前筛查** | quad screen: ↓AFP, ↑hCG, ↓uE3, ↑inhibin A | quad screen: **全部↓** | 通常 quad 正常，需 NIPT/超声 |

## 👫 性染色体异常

| 项目 | **Turner (45,XO)** | **Klinefelter (47,XXY)** | **47,XYY** | **Triple X (47,XXX)** |
|---|---|---|---|---|
| **性别** | 女 | 男 | 男 | 女 |
| **发病率** | 1/2500 女婴 | 1/500-1000 男婴 | 1/1000 男婴 | 1/1000 女婴 |
| **身高** | **矮小**（SHOX 基因缺失） | **高瘦**（长肢） | 高 | 高 |
| **性发育** | **卵巢发育不全（streak ovary）**、原发闭经、不育 | **小睾、男性乳房发育、不育**、testosterone↓ | 正常 | 一般正常 |
| **特征体征** | **颈蹼、宽间距乳头、肘外翻、低发际、新生儿手足淋巴水肿、马蹄肾** | 智力可正常或轻度低、社交学习问题 | 高大、轻度学习/行为问题 | 大多无症状 |
| **心脏** ⭐ | **Bicuspid aortic valve > Coarctation of aorta** | — | — | — |
| **激素** | **FSH/LH↑, estrogen↓**（高促性腺低性腺） | **FSH/LH↑, testosterone↓**（高促性腺低性腺） | 正常 | 正常 |
| **认知** | 智力多正常，**空间/数学弱** | 轻度学习障碍，语言弱 | 轻度学习障碍 | 轻度学习障碍 |
| **关键考点** | 心脏 + 不孕 + 矮小 | 男性不育 + 男乳 + 小睾 | 高大 + 行为问题 | 多数漏诊 |

## 🧬 微缺失 / 微重复综合征
#难记

| 综合征 | 染色体 | 心脏 | 关键体征 | 高频考点 |
|---|---|---|---|---|
| **DiGeorge (22q11.2 del)** | 22q11.2 缺失 | **Truncus arteriosus / ToF / 主动脉弓中断** | **CATCH-22**: <br>**C**ardiac defect <br>**A**bnormal facies <br>**T**hymic aplasia (→ T 细胞缺陷) <br>**C**left palate <br>**H**ypocalcemia (甲状旁腺发育不全) <br>22q11 | 新生儿低钙抽搐 + 反复病毒/真菌感染 + 心脏异常 |
| **Williams (7q11.23 del, 含 elastin)** | 7q11.23 缺失 | **Supravalvular aortic stenosis** | 三联：FTT + 高钙 + SVAS<br>+ 精灵脸 + 外向（"cocktail party"）+ 星状虹膜 | 高钙 + 主动脉瓣上 AS + 过度友好 |
| **Cri-du-chat (5p-)** | 5p 缺失 | VSD (一些) | **猫叫样高调哭声 (cat-like cry)** + 小头 + 严重智障 + 圆脸 | 婴儿哭声特征 |
| **Prader-Willi** | 15q11-13 **父源**缺失 (or 母源 UPD) | — | **新生儿期严重 hypotonia + 喂养困难** → 2-3 岁后 **hyperphagia + 肥胖** + 小手小脚 + 性腺发育不全 + 智障 | 双相表现（饿 → 暴食）+ 印记疾病 |
| **Angelman** | 15q11-13 **母源**缺失 (or 父源 UPD) | — | "Happy puppet"：**频繁大笑** + 共济失调 + 严重智障 + 癫痫 + 小头 | 笑 + 共济失调 + 印记疾病 |

## 🌱 三核苷酸重复 + 印记 + X 连锁

| 综合征 | 机制 | 关键体征 |
|---|---|---|
| **Fragile X** | FMR1 基因 **CGG 重复 >200**（X 连锁） | **男性**：长脸、大耳、**大睾（macroorchidism，青春期后）**、智障、自闭谱系、关节松弛<br>**女性**：携带者，可能轻度受累<br>**第二常见智障原因（仅次于 Down）** |
| **Huntington** | HTT **CAG 重复**（常染显性，**遗传早现 anticipation**） | 中年起病：舞蹈 + 痴呆 + 精神症状 |
| **Myotonic dystrophy** | DMPK **CTG 重复** | 肌强直 + 面肌无力 + 白内障 + 秃顶 + 心律失常 |
| **Friedreich ataxia** | FXN **GAA 重复**（常染**隐性**） | 共济失调 + **HCM** + 糖尿病 + 足弓高 |

## 🩺 多系统受累总览（横向对比）

### 💗 心脏畸形（最常见 → 第二）

| 综合征 | #1 心脏畸形 | #2 心脏畸形 |
|---|---|---|
| Down | **AVSD** (endocardial cushion) | VSD, ASD |
| Edwards | **VSD** | ASD, PDA |
| Patau | VSD / PDA / ASD | 混合 |
| Turner | **Bicuspid aortic valve** | Coarctation of aorta |
| DiGeorge | **Truncus arteriosus** | ToF, 主动脉弓中断 |
| Williams | **Supravalvular AS** | 周围肺动脉狭窄 |
| Marfan | **Aortic root 扩张** | MVP |
| Friedreich ataxia | **HCM** | — |
| Noonan | **Pulmonic stenosis** (瓣膜发育异常) | HCM |

### 🍼 喂养困难

| 综合征 | 喂养困难 | 主要机制 |
|---|---|---|
| **Prader-Willi** | ⭐⭐⭐ 极严重，需鼻饲 | 严重 hypotonia |
| **Edwards** | ⭐⭐⭐ 严重 | 小下颌 + 张力异常 + 中枢异常 + 先心病 |
| **Patau** | ⭐⭐⭐ 严重 | 唇腭裂 + 中枢异常 |
| **Williams** | ⭐⭐ 婴儿期严重（FTT） | 高钙 → 呕吐厌食 + 心脏疲劳 |
| **DiGeorge** | ⭐⭐ | 腭裂 + 低钙 + 心脏疲劳 |
| **Down** | ⭐ 中等 | 肌张力低 + 舌大 + 心脏疲劳 |
| **Cri-du-chat** | ⭐⭐ | 喉异常 + 张力低 + 小下颌 |
| **Angelman** | ⭐ 轻 | 早期吸吮差、GERD |
| **Turner** | — 新生儿期一般正常 | — |

### 🧠 智力 / 神经发育

| 程度 | 综合征 |
|---|---|
| **严重智障**（IQ <35） | Patau、Edwards（活过婴儿期者）、Cri-du-chat、Angelman、Prader-Willi |
| **中度智障**（IQ 35-50） | Down（多数）、Fragile X（男）、Williams |
| **轻度智障或正常智力**（IQ 50-85） | Turner（多正常但空间弱）、Klinefelter（多正常或轻度）、47,XYY |
| **正常智力** | Marfan、Noonan（多数） |

### 📏 身高

| 矮小 | 高大 |
|---|---|
| Turner、Down、Prader-Willi（成人期）、Williams、Noonan | Klinefelter、Marfan、47,XYY、Homocystinuria |

### 🎯 肿瘤风险

| 综合征 | 肿瘤 |
|---|---|
| **Down** | **ALL, AML**（白血病风险 ↑10-20 倍）、睾丸生殖细胞瘤 |
| **Klinefelter** | **乳腺癌**（↑20-50 倍）、纵隔生殖细胞瘤 |
| **Turner** | **性腺母细胞瘤**（如有 Y 物质需切除条索状性腺） |
| **Fragile X** | — |
| **Li-Fraumeni (TP53)** | 多种早发癌（乳腺、肉瘤、脑、肾上腺皮质） |

### 🧪 实验室关键值

| 综合征 | 关键化验 |
|---|---|
| Down | quad screen: ↓AFP, ↑hCG, ↓uE3, ↑inhibin A；甲减筛查 |
| Edwards | quad screen: **全部↓** |
| Turner | FSH/LH↑, estrogen↓ |
| Klinefelter | FSH/LH↑, testosterone↓ |
| Williams | **血钙↑** |
| DiGeorge | **血钙↓**, PTH↓, **T 细胞↓**（CD3, CD4, CD8）|
| Prader-Willi | — (依赖 DNA 甲基化检测) |

## 🎯 USMLE 高频出题模式

> [!danger] 经典 stem 套路

### 套路 1：新生儿多发畸形 → 问诊断 / 心脏畸形
- 重叠指 + 摇椅足 → Edwards → **VSD**
- 唇腭裂 + 多指 → Patau → **混合心脏畸形**
- 单掌纹 + 舌大 + 上斜眼 → Down → **AVSD**

### 套路 2：婴儿期表现 → 问诊断
- **新生儿 floppy + 吃不下** + 后期暴食肥胖 → **Prader-Willi**
- **FTT + 高钙 + 心杂音** → **Williams**
- **猫叫样哭声** → **Cri-du-chat**
- **新生儿手足肿 + 颈蹼** → **Turner**

### 套路 3：青少年/成人表现
- 矮小女孩 + 原发闭经 → **Turner**（查 FSH↑、超声看流出道梗阻）
- 高瘦男 + 男乳 + 小睾 + 不育 → **Klinefelter**
- 中年发病 + 舞蹈 + 痴呆 + 家族史 → **Huntington**
- 男孩 + 长脸 + 大耳 + 智障 + 大睾 → **Fragile X**

### 套路 4：免疫/感染问题
- 新生儿低钙抽搐 + 反复**病毒/真菌**感染 → **DiGeorge**（T 细胞缺陷）

### 套路 5：印记疾病鉴别 ⭐
- **15q 父源缺失** = Prader-Willi（"父亲走了 → 孩子饿"）
- **15q 母源缺失** = Angelman（"妈妈走了 → 孩子笑"）

## 💡 Memory Hooks 汇总

> [!success] 一句话锁定

- **Down → AVSD**："Down's heart cushioned" (endocardial cushion)
- **Edwards → VSD**："Edwards' V" (18 倒过来像 V)
- **Patau → 混合**：13 三体太严重，啥都坏
- **Turner → Bicuspid AV / Coarctation**："Turner 上下不通"（上肢高压下肢弱脉）
- **DiGeorge → Truncus**："22 个真没胸腺"（CATCH-22）
- **Williams → Supravalvular AS**："精灵脸瓣上窄"
- **Williams 三联**："W-AS-Ca-FTT"（瓣上窄 + 高钙 + 长不大）
- **Prader-Willi**："父走子饿 → 后期暴食"（15q 父源缺失）
- **Angelman**："母走子笑"（15q 母源缺失，快乐木偶）
- **Klinefelter**："多 X 男乳大乳癌"（XXY → 男乳 + 乳腺癌风险）
- **Fragile X**："长脸大耳大睾丸"（青春期后 macroorchidism）

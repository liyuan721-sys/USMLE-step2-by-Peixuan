# 免疫_超敏反应 4 型分类（Hypersensitivity Types I–IV）

> [!info] 用途
> 超敏反应（hypersensitivity）4 型分类是免疫学高频骨架。UWorld 常考"某药/某暴露后出现 XX 表现 = 第几型超敏 / 什么机制"。本笔记承载分类骨架 + **血清病（III 型）** 重点，来源错题 [[mistakes/uworld-mistakes_2026-06#^Q20700]]。

---

## 一、4 型超敏总表（Gell-Coombs 分类）

| 型 | 介质 | 时相 | 机制一句话 | 代表病 |
|---|---|---|---|---|
| **I 型** | **IgE**（+ 肥大细胞/嗜碱粒细胞）| **数分钟**（速发）| 抗原交联 IgE → 脱颗粒释放组胺 | 过敏性休克、荨麻疹/血管性水肿、过敏性鼻炎/哮喘、食物/药物速发过敏 |
| **II 型** | **抗体（IgG/IgM）对细胞表面/基质抗原** | 数小时–数天 | 抗体结合细胞 → 补体/ADCC/调理吞噬 → 细胞毁损 | 自身免疫溶血性贫血、ITP、Goodpasture、急性溶血性输血反应、Graves/重症肌无力（刺激/阻断型）|
| **III 型** | **免疫复合物 immune complex（抗原-抗体）** | **5–14 天** | 可溶性 IC 沉积组织 → 激活经典补体 → 中性粒浸润损伤 | **血清病 serum sickness**、SLE、链球菌感染后肾炎 PSGN、过敏性紫癜 HSP、Arthus 反应 |
| **IV 型** | **T 细胞（无抗体）** | **迟发 24–72 小时** | 致敏 T 细胞 → 细胞因子/CTL 直接杀伤 | 接触性皮炎、PPD/结核菌素试验、移植排斥、SJS/TEN、DRESS |

> [!tip] 记忆口诀
> **ACID**：**A**llergic/**A**naphylaxis(I) · **C**ytotoxic(II) · **I**mmune complex(III) · **D**elayed(IV)。
> 时相：I 型分钟级 → II/III 型天级 → IV 型 24–72h 迟发。

## 二、III 型重点：血清病 Serum Sickness ⭐

> [!danger] 招牌钥匙
> **「异源蛋白（单抗/抗毒素/抗蛇毒血清）用药后 5–14 天，发热 + 荨麻疹样疹 + 关节痛（+ 轻度蛋白尿）= 血清病 = III 型免疫复合物」**

### 机制

```
异源蛋白（foreign protein）
   ↓ 机体产生高亲和力抗体
形成大量免疫复合物（IC）
   ↓ 饱和吞噬系统 → IC 聚集
沉积组织（皮肤 / 关节 / 肾小球）
   ↓ 激活经典补体通路
中性粒浸润 → 组织损伤
   ↓
发热 + 荨麻疹 + 关节痛 + 轻度蛋白尿（自限）
```

### 常见诱因（异源蛋白）

- **单克隆抗体 monoclonal antibody**：英夫利昔单抗 infliximab（嵌合单抗，含鼠源蛋白）、利妥昔单抗 rituximab 等
- **抗毒素 antitoxin / 抗蛇毒血清 antivenom**
- **抗原性疫苗**

### 临床表现

- 发热（高热，因 IC 滴度高 + 补体大量消耗）
- 荨麻疹样疹 urticarial rash（躯干四肢）
- 关节痛 arthralgia
- 轻度蛋白尿（IC 沉积肾小球）
- **时间窗**：暴露后 **5–14 天**；再次给药后可更快复发
- **转归**：多**自限**（IC 清除后自行缓解）

## 三、SS vs SSLR（血清病 vs 血清病样反应）

> [!warning] 高频对照
> UWorld 给的表常并列这两者——区别在**诱因是异源蛋白还是药物**，以及**补体消耗程度**。

| 项目 | **SS 血清病** | **SSLR 血清病样反应** |
|---|---|---|
| **常见诱因** | **异源蛋白**：抗蛇毒血清、抗毒素、**单抗（infliximab）** | **药物**：头孢克洛 cefaclor、青霉素 penicillin、复方磺胺甲噁唑 TMP-SMX |
| **免疫复合物** | **高滴度** | 轻或无 |
| **补体激活/消耗** | **广泛（明显消耗）** | 极少 |
| **发热** | **高热** | 低热 |
| **关节痛 / 荨麻疹** | 均有 | 均有 |
| **起病** | 暴露后 5–14 天 | 暴露后 5–14 天 |
| **处理** | 支持治疗，自限 | **停药即缓解** |

> [!tip] SSLR 诱因的年龄差异 + EBV 陷阱
> - **儿科 SSLR 高频诱因 = 头孢克洛 cefaclor**；成人多为青霉素 penicillin / 复方磺胺甲噁唑 TMP-SMX。
> - **EBV 单核细胞增多症患者用阿莫西林/氨苄西林 → 麻疹样皮疹**，是**药物-病毒交互的"超敏样反应"，并非真正 IgE 过敏**，日后仍可用青霉素类。

## 四、易混陷阱

> [!warning] 生物制剂副作用 ≠ 只想感染
> 见 infliximab 别条件反射"结核 TB 再激活"。infliximab 既可因**异源蛋白**触发**血清病（III 型，本身病机）**，也可增加 TB/机会感染风险（启动前应筛查潜伏结核 LTBI）。**急性 + 荨麻疹三联**指向血清病，而非感染。

- **I 型 vs III 型**：均可有荨麻疹，但 I 型在**数分钟内**发作（IgE 速发），III 型在**5–14 天**（IC 迟发）+ 有关节痛/蛋白尿。
- **IV 型（SJS/TEN）vs III 型血清病**：IV 型为 **T 细胞介导**、**晒伤样弥漫疹 + 黏膜受累 + 高热**；血清病为荨麻疹 + 关节痛、黏膜正常。
- **PSGN / SLE** 也是 III 型——同属免疫复合物病，机制同源。

## 五、各型超敏病谱扩展（高频补充）

### I 型（IgE / 肥大细胞，分钟级）

- **过敏性休克 anaphylaxis**：药物（amoxicillin）/食物（花生/牛奶/海鲜）/虫咬后**分钟级 ≥2 器官系统**（皮肤+呼吸/心血管/GI）→ **IM 肾上腺素**（详见 [[mistakes/uworld-mistakes_2026-06#^Q17055]]）
- **食物过敏 / 嗜酸性食管炎 EoE**：食物抗原诱发；EoE 内镜见**纵行裂沟 grooves / 环状食管**，活检嗜酸粒↑
- 过敏性鼻炎/哮喘、荨麻疹/血管性水肿

### III 型（免疫复合物）扩展

- **过敏性紫癜 HSP / IgA 血管炎**：**儿童最常见系统性血管炎**；可触性紫癜（下肢/臀）+ 关节痛 + 腹痛 + 血尿（IgA 免疫复合物沉积小血管）
- **川崎病 Kawasaki**：≥5 天发热 + ≥4 项（双侧结膜 / 口唇舌 / 肢端 / 多形疹 / 颈淋巴结）；**冠脉瘤风险 → IVIG + 阿司匹林**
- 链球菌感染后肾炎 PSGN、SLE、血清病（见 §二）

### IV 型（T 细胞迟发，24–72h）扩展

- **PPD / 结核菌素试验**：经典迟发型超敏教学范例；**免疫抑制 / 细胞免疫低下时假阴性**
- 接触性皮炎（镍 / 毒藤）、SJS/TEN、DRESS、移植排斥

> [!warning] 时间窗 = 分型快捷键
> **分钟级 = I 型（IgE）｜5–14 天 = III 型（免疫复合物）｜接触后 24–72h = IV 型（T 细胞）**。stem 给的"起病速度"常是分型第一钥匙。

## 六、Anaphylaxis 急救 SOP（I 型重点）

> [!danger] 诊断 3 标准（迅速起病，满足任一即可）
> 1. **皮肤/黏膜受累**（荨麻疹/唇舌肿）**+ 低血压或呼吸窘迫**
> 2. 暴露可疑过敏原后 **≥2 器官系统**受累（皮肤/呼吸/心血管/GI）
> 3. 暴露**已知过敏原**后**低血压**
> ⚠️ 低血压非必需；皮肤受累不明显时易误诊为哮喘急性发作。

| 步骤 | 内容 | 关键细节 |
|---|---|---|
| **第一步** | **IM 肾上腺素 epinephrine** | **1:1000**，**大腿外侧肌 vastus lateralis**，每 **5–15 分钟**重复，最多 3 剂 |
| 第二步 | 气道 + 快速补液 | 大剂量晶体液（生理盐水）|
| 辅助 | 抗组胺（H1/H2）+ 糖皮质激素 | **仅缓解皮肤 / 防晚发，不替代肾上腺素** |
| 特殊 | 用 β 阻滞剂且肾上腺素无效 | 加 **胰高血糖素 Glucagon** |

> [!tip] 诱因 + 预防
> - 诱因：食物（坚果/海鲜）、药物（β-内酰胺）、虫咬、**NSAID / 运动诱发**、造影剂（类过敏 anaphylactoid，非 IgE）
> - **双相反应 biphasic**：症状缓解后 4–12h 可复发 → 严重者留观
> - **预防金标准 = 识别 + 规避过敏原**（数周后做 SPT / 体外测，急性期假阴性）；**毒液过敏 → 免疫治疗（诱导 IgG 阻断抗体）**；出院带 **EpiPen**

## 七、血管性水肿鉴别（缓激肽 vs 组胺）⭐

| 类型 | 介质 | **C4** | 诱因 / 特征 | 处理 |
|---|---|---|---|---|
| **遗传性 HAE** | 缓激肽 | **↓** | AD 家族史、青少年起病、**拔牙/外伤诱发**、**非痒性**水肿 + 阵发腹痛 + 喉头水肿 | 急性 **C1-INH 浓缩物 / Icatibant / Ecallantide**；预防 达那唑 Danazol；**禁 ACEI** |
| **ACEI 诱导** | 缓激肽 | **正常** | 唇舌肿、**无荨麻疹无痒**、用药任何时间（数年也可）| **停药换 ARB** |
| **获得性 C1-INH 缺乏** | 缓激肽 | **极低 + C1q↓** | 伴 **B 细胞淋巴瘤 / 自身免疫** | 治原发病 |
| **过敏性（组胺）** | **组胺** | 正常 | **瘙痒 + 荨麻疹**、对肾上腺素反应好 | 肾上腺素 / 抗组胺 |

> [!warning] 缓激肽介导的关键
> **缓激肽性水肿对肾上腺素 / 激素 / 抗组胺反应差** → 重点**气道保护**；筛查首选 **C4↓**，确诊测 C1-INH 浓度/功能。HAE 属补体**调节蛋白**缺陷（见 [[完整笔记/专题笔记/免疫/免疫_原发性免疫缺陷鉴别]] §一补体两端）。

## 八、皮肤测试：点刺 SPT vs 斑贴 Patch（I 型 vs IV 型诊断）

| 特性 | **点刺 SPT (skin prick)** | **斑贴 Patch test** |
|---|---|---|
| 超敏型 | **I 型（IgE）** | **IV 型（T 细胞迟发）** |
| 用途 | 食物 / 花粉 / 哮喘 | **接触性皮炎** |
| 结果时间 | **15–20 分钟** | **48–72 小时** |
| 阳性表现 | **风团 wheal** | 红斑 / 水疱 / 硬结 |
| 部位 | 前臂 | 背部 |

> [!tip] 一句话
> **点刺 = 即刻 IgE 过敏（食物/花粉）｜斑贴 = 迟发 T 细胞接触性皮炎（金属/香料）**——正好对应 I 型 vs IV 型。

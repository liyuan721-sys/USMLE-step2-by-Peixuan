# 血管炎大/中/小血管谱系（Vasculitis）

> [!info] 笔记定位
> 血管炎 USMLE 高频但易混。本笔记目标：**先按血管口径（大/中/小）定位 → 小血管炎再用"ANCA + 补体 + 免疫荧光"三轴分型**。来源：[[mistakes/uworld-mistakes_2026-06#^Q2232]]（HCV → 混合型冷球蛋白血症 MCS，做对但 SVV 鉴别表不熟 → 早建 v1）。

---

## Part 1. 按血管口径总览 ⭐⭐⭐（第一刀）

```
血管炎按受累血管口径分三层：
├─ 大血管：主动脉 + 主要分支 ── GCA / Takayasu
├─ 中血管：内脏中等动脉    ── PAN / Kawasaki / Buerger
└─ 小血管：小动脉/毛细血管/小静脉 ── ANCA 相关(GPA/MPA/EGPA) + 免疫复合物(IgA/MCS/anti-GBM/药物)
```

| 口径 | 病 | 一句话指纹 |
|---|---|---|
| **大** | **GCA（巨细胞/颞动脉炎）** | >50 岁 + 头痛 + 颌跛行 + 视力骤失 + PMR + ESR↑ |
| **大** | **Takayasu（多发性大动脉炎）** | <40 岁亚洲女 + "无脉症" + 双臂血压差 + bruit |
| **中** | **PAN（结节性多动脉炎）** | 多系统 + 单神经多发 + 肾性 HTN + **HBV 相关** + **肺不受累** + ANCA 阴性 |
| **中** | **Kawasaki（川崎）** | <5 岁 + 发热≥5 天 + 结膜炎/草莓舌/皮疹/手足 + **冠脉瘤** |
| **中** | **Buerger（血栓闭塞性脉管炎）** | 年轻**男烟民** + 肢端缺血/溃疡 + Raynaud → **戒烟** |
| **小** | **ANCA 相关**（GPA/MPA/EGPA）| ANCA+ / pauci-immune / **补体正常** |
| **小** | **免疫复合物**（IgA/MCS/anti-GBM/药物）| 免疫荧光阳性 / 多数**低补体** |

---

## Part 2. 大血管炎

> [!danger] GCA（巨细胞/颞动脉炎）—— 怕失明，先治不等活检 ⭐
> **>50 岁** + 新发**颞侧头痛** + **颌跛行（jaw claudication，最特异）** + 头皮压痛 + 一过性黑朦/视力骤失（前部缺血性视神经病）+ 常合并 **PMR（风湿性多肌痛）**。ESR/CRP↑↑。
> - **处理**：**立即大剂量糖皮质激素**（疑诊即上，**别等颞动脉活检**）→ 防不可逆失明；活检（granulomatous + 跳跃性病灶）可在数日内做。
> - 远期：胸主动脉瘤风险↑。

> [!info] Takayasu（多发性大动脉炎）
> **<40 岁亚洲年轻女**；主动脉弓 + 分支炎症狭窄 → **"无脉症"**、双上肢**血压不对称**、肢体跛行、血管 bruit；ESR↑。处理：糖皮质激素。

## Part 3. 中血管炎

| 病 | 人群 | 招牌 | 关键点 |
|---|---|---|---|
| **PAN** | 中年 | 发热/消瘦/肌痛 + **单神经多发** + 肾性 HTN + 肠系膜缺血(腹痛) + 皮肤(网状青斑/结节/溃疡) | **HBV 相关(~30%)** / **肺不受累** / **ANCA 阴性** / 血管造影**串珠样微动脉瘤** / 活检确诊；治疗激素±环磷酰胺，HBV 相关者抗病毒 |
| **Kawasaki** | <5 岁 | 发热≥5 天 + 结膜炎 + 草莓舌/皲裂唇 + 多形皮疹 + 手足肿脱皮 + 颈淋巴结 | **冠状动脉瘤**（最怕）→ **IVIG + 阿司匹林** |
| **Buerger** | 年轻男烟民 | 肢端缺血/静息痛/溃疡/坏疽 + Raynaud + 游走性浅静脉炎 | 造影**螺旋样侧支(corkscrew)**；**唯一有效治疗 = 戒烟** |

> [!warning] PAN 反 pattern
> **PAN 肺不受累 + ANCA 阴性** —— 与小血管 ANCA 相关血管炎（常累肺 + ANCA+）正好相反。题干"中年 + 多系统 + HBV + 肾 + 神经 + 不咳血 + ANCA 阴性" → 想 PAN。

## Part 4. 小血管炎 ⭐⭐⭐（最易混，三轴分型）

### 4.1 三轴速判

```
小血管炎 → 三轴定型：
① ANCA 阳性？ → 阳性 = ANCA 相关（pauci-immune，补体正常）
② 补体？     → 低补体 = 免疫复合物型（MCS / 狼疮 / 药物）
③ 免疫荧光？ → IgA 沉积=HSP / 线性 IgG=anti-GBM / 颗粒状=免疫复合物
```

### 4.2 ANCA 相关（pauci-immune，**补体正常**）

| 病 | ANCA | 招牌 |
|---|---|---|
| **GPA（肉芽肿性多血管炎/Wegener）** | **c-ANCA / PR3** | **上呼吸道**（鼻窦炎/鞍鼻/鼻溃疡）+ **下呼吸道**（肺结节/空洞/咯血/**气管狭窄+溃疡 tracheal narrowing & ulceration，高度特异**）+ GN；有肉芽肿 |
| **MPA（显微镜下多血管炎）** | **p-ANCA / MPO** | 肺毛细血管炎(咯血) + GN；**无肉芽肿、无上呼吸道突出** |
| **EGPA（嗜酸性肉芽肿性多血管炎/Churg-Strauss）** | **p-ANCA / MPO** | **哮喘 + 外周嗜酸↑** + GN + 神经病；有肉芽肿 + 嗜酸 |

> [!warning] GPA 高频干扰项：职业暴露钩子 ≠ 职业病（派生自 Q3049）
> 题干给"**石材切割工 / 矿工（硅尘 silica 暴露）**" + 空洞性肺结节 → 诱导选**硅肺 + 结核 (silicosis with tuberculosis)**。硅暴露确实增 TB 与空洞病灶风险，**但不解释气管狭窄溃疡 + 快进展性肾衰 (GN) + 慢性鼻窦炎**——这三联是 GPA 招牌。
> **钥匙：见职业暴露别自动跳职业病，要全器官对账**——上呼吸道 + 气管溃疡 + 肾 GN 同时出现 = GPA（c-ANCA/PR3），不是硅肺/TB。

### 4.3 免疫复合物型

| 病 | 免疫荧光/血清 | 补体 | 招牌 |
|---|---|---|---|
| **IgA 血管炎（HSP）** | **IgA 沉积** | **正常** | 儿童；URI 后；可触性紫癜（臀/下肢）+ 腹痛 + 关节痛 + IgA 肾病 |
| **冷球蛋白血症（MCS）** ⭐ | **冷球蛋白 + RF + 抗 HCV** | **低** | **HCV** 最常见；可触性紫癜 + 膜增生性 GN + 周围神经病 + 肝大 |
| **Anti-GBM（Goodpasture）** | **线性 IgG（抗 GBM 抗体）** | 正常 | GN + 肺出血（咯血）|
| **药物性** | ± ANCA / 抗组蛋白 | 低 | 最常见、常局限单器官 |

> [!tip] 血管炎性紫癜的共性 + 与"血小板减少性紫癜"的第一刀分水岭
> 小血管炎（HSP / MCS / ANCA 相关等）皮损 = **可触性紫癜 (palpable purpura) + 血小板正常**（血管壁炎症漏血，非血小板问题）；而 **血小板减少性紫癜（ITP 免疫性血小板减少 immune thrombocytopenia / TTP 血栓性血小板减少性紫癜 thrombotic thrombocytopenic purpura）= 血小板↓ + 紫癜平坦不可触**。
> 钥匙：**「可触 + 血小板正常 = 血管炎；平坦 + 血小板低 = 血小板病。」** 紫癜三机制框架 + HSP vs ITP/TTP 完整对比见 [[完整笔记/专题笔记/儿科/儿科_IgA血管炎HSP完整体系]] §七。

## Part 5. 补体分水岭 ⭐⭐（小血管炎最高产鉴别轴）

> [!danger] 一刀切：低补体 vs 正常补体
> - **低补体（消耗）= 免疫复合物型**：**MCS（冷球蛋白）/ 狼疮性 / 链球菌后 GN（PSGN）/ 药物性**
> - **补体正常**：**ANCA 相关（pauci-immune）/ IgA 血管炎（HSP）/ anti-GBM / PAN**
>
> 题干"小血管炎 + **ANCA 阴性 + 低补体 + 抗 HCV + RF**" → 锁 **MCS**（排除补体正常的 ANCA 相关如 MPA）。

> [!tip] RF 阳性 ≠ 类风湿
> MCS 的 RF 来自冷球蛋白里**带 RF 活性的 IgM**（结合 IgG 的 Fc，<37°C 沉淀）→ 免疫复合物激活、消耗补体。不是 RA。

## Part 6. 病毒/继发关联（高频钩子）

| 病 | 关联 |
|---|---|
| **PAN** | **HBV**（乙肝）~30% |
| **MCS（冷球蛋白血症）** | **HCV**（丙肝）最常见；亦 HIV / 自免（SLE/RA）|
| **GPA/MPA/EGPA** | 原发（ANCA）；药物（hydralazine/PTU/minocycline 可诱发 ANCA 血管炎）|
| **IgA 血管炎(HSP)** | URI / 链球菌后 |

## Part 7. Memory Hooks ⭐

> [!tip] 一句话锁定
> - **先分口径**：大(GCA/Takayasu) → 中(PAN/Kawasaki/Buerger) → 小(ANCA/免疫复合物)
> - **GCA：>50 + 颌跛行 + 视力 → 立即激素别等活检**
> - **PAN：HBV + 多系统 + 肾/神经 + 肺不受累 + ANCA 阴性 + 串珠微动脉瘤**
> - **Buerger：年轻男烟民肢端坏疽 → 戒烟**
> - **小血管补体分流**：**低补体 = 免疫复合物（MCS/狼疮/PSGN/药物）；正常 = ANCA/IgA/anti-GBM**
> - **MCS = HCV + 可触性紫癜 + 低补体 + RF+**；冷球蛋白 = IgM(RF)抱 IgG
> - **ANCA 类型**：GPA = c-ANCA/PR3（上下呼吸道+肾）；MPA/EGPA = p-ANCA/MPO（EGPA 加哮喘+嗜酸）

## Part 8. 复习题（可折叠 Q&A）

> [!question]- Q1. 50yo + IVDU 史 + 可触性紫癜 + GN(RBC 管型) + 周围神经病 + 肝大 + 低补体 + RF+ + 抗 HCV+ + ANCA 阴性 → 诊断？
> **混合型冷球蛋白血症（MCS / 冷球蛋白血症性血管炎）**，HCV 致。三联：抗 HCV + RF + **低补体**；ANCA 阴性 + 低补体排除 ANCA 相关（如 MPA，补体正常）。

> [!question]- Q2. 小血管炎里哪些低补体、哪些补体正常？
> **低补体（免疫复合物）**：MCS / 狼疮性 / 链球菌后 GN / 药物性。
> **补体正常**：ANCA 相关（GPA/MPA/EGPA）/ IgA 血管炎（HSP）/ anti-GBM。

> [!question]- Q3. PAN 与小血管 ANCA 相关血管炎最关键的 3 点区别？
> ① **PAN 肺不受累**（ANCA 相关常累肺）；② **PAN ANCA 阴性**；③ **PAN 与 HBV 相关**（血管造影串珠样微动脉瘤）。

> [!question]- Q4. >50 岁颞侧头痛 + 颌跛行 + 一过性黑朦，下一步处理？
> **立即大剂量糖皮质激素**（疑诊即上，不等颞动脉活检）防失明；随后数日内做活检（granulomatous + 跳跃病灶）。

> [!question]- Q5. 三个 ANCA 相关血管炎的 ANCA 类型 + 招牌？
> **GPA** = c-ANCA/PR3（上呼吸道鞍鼻 + 肺结节 + GN）；**MPA** = p-ANCA/MPO（肺毛细血管炎 + GN，无肉芽肿）；**EGPA** = p-ANCA/MPO（**哮喘 + 嗜酸** + GN + 神经病）。

> [!question]- Q6. 年轻男烟民足趾缺血溃疡 + Raynaud，诊断 + 唯一有效治疗？
> **Buerger（血栓闭塞性脉管炎）**；**戒烟**是唯一有效治疗（造影见 corkscrew 侧支）。

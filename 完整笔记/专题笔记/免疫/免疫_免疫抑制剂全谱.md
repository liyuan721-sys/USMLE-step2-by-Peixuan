# 免疫抑制剂全谱 — 5 大类 + 通路 + 副作用 + 监测

> [!info] 这一篇定位

> CNI 急性肾毒 / Transplant RAS 鉴别 → 详见 [[完整笔记/专题笔记/肾脏/肾脏_移植后AKI鉴别SOP]]（不重复）

## 一、宏观分类：5 大类 + 一句话定位

| 类别 | 代表药 | 一句话机制 | 主战场 |
|---|---|---|---|
| **Calcineurin Inhibitors (CNI)** | Cyclosporine / **Tacrolimus** | 阻 calcineurin → 阻 NFAT → **阻 IL-2 转录** | 实体器官移植（首选）|
| **mTOR Inhibitors** | **Sirolimus** (Rapamycin) / Everolimus | 阻 mTOR → 阻 IL-2 信号下游 | 肾移植排斥预防 ★ |
| **Antimetabolites** | **Azathioprine** / MMF / MTX | 阻嘌呤/嘧啶合成 → 阻 B/T 细胞增殖 | 自身免疫病 + 移植辅助 |
| **Biologics** | **Basiliximab** (IL-2R) / TNF-α (infliximab/adalimumab) | 单抗精准阻断 | 移植诱导 / RA / AS / IBD |
| **Glucocorticoids** | Prednisone / Methylpred | 阻 NF-κB → 广谱免疫抑制 | 万能桥接，长期副作用大 |

## 二、T 细胞激活通路 + 药物作用位点

```
APC（抗原递呈细胞）
   │ MHC II + 抗原
   ▼
TCR (T 细胞受体)
   │
   ▼
┌──────────────────────────────────────────────────────────┐
│  Cyclosporine ──┐      Tacrolimus ──┐                    │
│     绑 Cyclophilin    绑 FKBP-12                         │
│        └──┬──────────────┴──┐                           │
│           ▼                                              │
│       Calcineurin (被阻)                                  │
│           │                                              │
│           ▼                                              │
│       NFAT 活化失败                                       │
│           │                                              │
│           ▼                                              │
│       ★ IL-2 基因转录 ↓ ★                                 │
└──────────────────────────────────────────────────────────┘
           │
           ▼
       IL-2 分泌（被 CNI 阻）
           │
           ▼
       IL-2 → IL-2R    ◄── Basiliximab 在这里阻
                │
                ▼
           mTOR（被 Sirolimus 阻）
                │
                ▼
       ★ 增殖基因转录 ↑ ★（被 mTOR 抑制剂阻）
                │
                ▼
       T 细胞克隆扩增

           ↓ 整个过程的"原料"（核苷酸合成）
```

> [!tip] 通路记忆 — 4 层打击
> - **CNI** 阻 IL-2 **生产**（最早期）
> - **Basiliximab** 阻 IL-2 **接收**
> - **mTOR-i** 阻 IL-2 **下游信号**
> - **Antimetabolites** 阻**原料供给**
> - **Glucocorticoids** 在 NF-κB 层面广谱阻

## 三、Calcineurin Inhibitors — Cyclosporine vs Tacrolimus

> [!danger] 机制几乎一样，副作用细节考个不停 ★★★

| | **Cyclosporine** | **Tacrolimus (FK506)** |
|---|---|---|
| **绑哪个免疫亲和素** | **Cyclophilin** | **FKBP**（FK506 binding protein）|
| **机制** | 复合物阻 Calcineurin → IL-2 转录 ↓ | 同上 |
| **指征** | Psoriasis / RA / 移植 | **实体器官移植首选** ★ |
| **共同毒性** | 肾毒 ★ / HTN / 神经毒 (tremor) / 高血糖 / 高血脂 | 同上 + **糖尿病风险 ↑** + 神经毒 ↑ |
| **独特毒性** | **牙龈增生 (gingival hyperplasia)** ★ + **多毛 (hirsutism)** ★ | **无**牙龈增生 + **无**多毛 |
| **长期肿瘤风险** | **皮肤鳞癌 (cutaneous SCC)** ★ + PTLD (EBV-driven 淋巴瘤) | 同 |
| **感染谱** | URI + UTI 最常见 | 同 |
| **代谢** | CYP3A4 | CYP3A4 |

> [!danger] "见牙龈增生想三个药"
> 1. **Cyclosporine**（免疫抑制）★
> 2. **Phenytoin**（抗癫痫）
> 3. **CCB**（dihydropyridines: nifedipine, amlodipine, verapamil）
>
> Tacrolimus **没有**这条副作用 — 看到牙龈增生别选 tacro

> [!warning] CNI 肾毒 + CYP3A4 互动深读
> CNI 急性肾毒 + Transplant RAS 鉴别 + CYP3A4 抑制/诱导剂列表 → 详见 [[完整笔记/专题笔记/肾脏/肾脏_移植后AKI鉴别SOP]]
> 关键陷阱：**新加 azole / macrolide / 葡萄柚 / diltiazem-verapamil → CNI 血药↑↑ → 急性肾毒**

## 四、mTOR Inhibitor — Sirolimus（Rapamycin）

> [!success] "**Sir-vives Kidney**" 助记
> Sirolimus = **肾移植排斥预防 specifically** 的药 — "Sir Basil's Kidney"
> **不肾毒**（与 CNI 截然不同）★ → 给肾移植病人特别合适

| 项 | 内容 |
|---|---|
| 机制 | 绑 **FKBP**（与 tacro 同一蛋白）→ 但抑制的是 **mTOR**（不是 calcineurin）→ 阻 IL-2 信号下游 → 阻增殖基因 |
| 指征 | **肾移植排斥预防** ★ / 药物涂层支架 (drug-eluting stents, DES) |
| 毒性 | **Pancytopenia** ★（全血减少）/ Insulin resistance / 高血脂 / 伤口愈合差 / **non-infectious pneumonitis** |
| 反陷阱 | **NOT 肾毒** ★（vs CNI 的肾毒）|
| 协同 | 可与 Cyclosporine 联用（IL-2 生产 + IL-2 信号双阻）|

## 五、Antimetabolites — 三段拆讲

### 5.1 Azathioprine → 6-MP（嘌呤合成阻断）

> [!info] 机制链
> Azathioprine → 体内代谢成 **6-MP** → 经 HGPRT 活化 → **阻 PRPP amidotransferase** → 嘌呤合成 ↓ → B/T 细胞增殖 ↓

| 项 | 内容 |
|---|---|
| 指征 | **IBD (Crohn / UC)** ★ / RA / 自身免疫 GN / 移植辅助 / **MG 维持** |
| 毒性 | **Pancytopenia** / 肝毒 / 胰腺炎 / 感染 |
| 关键代谢 | **6-MP 经 xanthine oxidase (XO) 降解** |
| 用药前必查 | **TPMT or NUDT15 酶活性**（防严重骨髓抑制）+ CBC + LFT |

> [!danger] ★★★ CK 必考陷阱：Azathioprine + Allopurinol
> Allopurinol / Febuxostat = **XO 抑制剂**（治痛风）
> 联用 → 6-MP 不能被 XO 降解 → **血药 ↑↑ → 致命骨髓抑制**
> 必须 **azathioprine 减量 75%** 或换药
>
> **AMBOSS 经典 stem 模式**（Q7）：
> MG 2 个月，prednisone + azathioprine 1 个月 → **基础合并痛风（用 allopurinol）** → 1 个月后全血减少 → "What would have prevented this near miss?" → **Assessment of potential drug interactions**
>
> 替代药：**Cyclosporine**（不走 XO，对骨髓影响小）

> [!warning] 反陷阱（同 Q7 干扰项）
> 题干给 prednisone + azathioprine 的 MG 病人查出 pancytopenia，选项里"folate supplement" / "withhold prednisone" / "CT chest" 都是干扰：
> - **Folate** = 是 MTX 用，不是 aza
> - **Prednisone 不致 pancytopenia** — 反而 **leukocytosis + ↑ Hb/RBC**（demargination）★
> - **CT chest 查 thymoma** = MG 标准检查，但 thymoma → **PRCA (pure red cell aplasia)** 不是 pancytopenia

### 5.2 Mycophenolate Mofetil (MMF)

| 项 | 内容 |
|---|---|
| 机制 | 抑 **IMP dehydrogenase** → 阻 GMP 合成 → B/T 细胞**特异性**嘌呤合成阻（淋巴细胞依赖 de novo 嘌呤通路，其他细胞有 salvage 代偿）|
| 指征 | **实体器官移植辅助** ★（与 tacrolimus 联用是标准）/ RA / SLE / lupus nephritis |
| 毒性 | **GI（腹泻 / 腹痛 / 呕吐）** ★ / **Pancytopenia** / HTN / **invasive CMV** 感染风险 ★ / 高血糖 |
| 反陷阱 | **不肾毒 + 不神经毒**（区别 CNI）|
| 妊娠 | **致畸**（用药前妊娠测试 + 避孕）|

> [!danger] MMF "三联" pattern → 锁定
> **"全血减少 + 高血糖 + GI（腹泻/呕吐）"** 同时见 = **MMF** ★
> CNI（tacro / cyclo）= 高血糖 ✓，**全血减少 ✗**
> → AMBOSS Q4 的考点

### 5.3.5 Cyclophosphamide（CK 高频补充）

> [!info] 虽不严格属"免疫抑制剂 5 大类"，但作为细胞毒类经常出现在 SLE / 血管炎题里

| 项 | 内容 |
|---|---|
| 机制 | **烷化剂** — DNA 交联 → 阻 DNA 复制（B/T 细胞 + 肿瘤细胞）|
| 指征 | **重症 SLE**（lupus nephritis / CNS lupus）★ / **ANCA 血管炎**（GPA / MPA）/ 某些化疗（乳腺癌 / 淋巴瘤 / 卵巢癌）|
| 急性毒性 | **Hemorrhagic cystitis** ★（acrolein 损伤膀胱黏膜）→ 加 **Mesna** 预防 / **SIADH**（低钠）|
| 长期毒性 | **Bladder cancer** ★（多年后）/ **不孕**（卵巢早衰 + 精子缺失）/ 骨髓抑制 |
| 解毒剂 | **Mesna**（结合 acrolein → 经尿排出）|

> [!danger] CK 经典 stem
> 重症 SLE 病人 cyclophosphamide 治疗后 → 血尿 → **Hemorrhagic cystitis**（漏 Mesna）
> 10 年后膀胱镜见鳞癌 → **cyclophosphamide-induced bladder cancer**

详细化疗药谱见 [[完整笔记/Peixuan分科笔记/hematology oncology]]

### 5.3 Methotrexate (MTX)

| 项 | 内容 |
|---|---|
| 机制 | 抑 **DHFR (Dihydrofolate Reductase)** → THF ↓ → dTMP 合成 ↓ → DNA 合成 ↓ |
| 指征 | RA / Psoriasis / 异位妊娠 / 化疗（ALL / 淋巴瘤 / 乳腺癌）/ 终止妊娠 |
| 毒性 | **骨髓抑制** ★ / **肝毒** / 间质性肺纤维化 / **粘膜炎** / 致畸 ★ / 巨幼贫血 |
| 解救 | **Leucovorin (folinic acid)** — 见 §六 |
| 用药前必查 | **CBC + LFT + Cr** + 妊娠测试 |
| 监测 | CBC q月 / LFT q3 月 / 出现呼吸症状 → PFT 评估 pneumonitis |

## 六、Leucovorin 的"双面人格" ★⭐⭐

> [!danger] 同一个药，在 MTX 和 5-FU 中**作用完全相反**
> CK 必考反直觉点

| 配伍 | Leucovorin 角色 | 机制 |
|---|---|---|
| **MTX + Leucovorin** | **解救正常细胞**（骨髓 / 黏膜）| MTX 阻 DHFR → THF 没了 → leucovorin = **已经活化的叶酸（folinic acid）** → 跳过 DHFR 直接补 THF → **救骨髓但不救肿瘤**（肿瘤细胞跨膜运输弱）|
| **5-FU + Leucovorin** | **增强 5-FU 抗肿瘤** | 5-FU → 5-FdUMP；leucovorin → 增加细胞内 THF → **三元复合物 (5-FdUMP + TS + THF) 更稳定** → 更强抑制 thymidylate synthase → **抗肿瘤 ↑** |

> [!warning] CK 出题模式
> - "MTX + leucovorin 24-48 h 内" → **救援防骨髓抑制**（AMBOSS Q3）
> - "结直肠癌 FOLFOX / FOLFIRI" → **5-FU + Leucovorin 增效**
> - 别搞反！

## 八、Glucocorticoids — 万能 + 万恶

| 项 | 内容 |
|---|---|
| 机制 | 抑制 **NF-κB** → 抑多种细胞因子转录 → 抑 B/T + 诱导 T 细胞凋亡 |
| 指征 | 几乎所有自身免疫病 / 移植 / 哮喘 / IBD / 肾上腺功能不全 / CLL / NHL 等 |
| 急性毒性 | 高血糖 / 失眠 / 精神症状 / HTN / 感染易感 |
| 慢性毒性 ★ | **Cushing 综合征** / 骨质疏松 / **白内障** / 青光眼 / 消化道溃疡 / 股骨头无菌坏死 / 紫纹 / 痤疮 / 月经异常 / 儿童身高停滞 |
| **致命陷阱** ★★★ | **突然停药 → 肾上腺危象 (Adrenal Crisis)** — 长期用必须**逐渐减量** |
| 反直觉 | **Demargination of WBCs → "伪白细胞增多"** + ↑ Hb / RBC（不代表感染，也不致 pancytopenia）★ |

> [!danger] CK 经典 stem
> 长期类固醇病人突然停药 → 低血压 + 低血糖 + 低钠 + 高钾 + 休克 → **Adrenal crisis**
> 治疗：**IV hydrocortisone 100 mg q8h + 大量补液**

## 九、嘌呤 / 嘧啶合成通路（整合）

```
                     PRPP (核糖磷酸焦磷酸)
                          │
        ┌─────────────────┼─────────────────┐
        ▼                                   ▼
    嘌呤通路                           嘧啶通路
    (Purine)                          (Pyrimidine)
        │                                   │
        ▼  ★ 6-MP / Azathioprine             ▼
   PRPP amidotransferase 被阻         Carbamoyl phosphate
        │                                   │ (需 aspartate)
        ▼                                   ▼ ★ Leflunomide 阻
       IMP                              Orotic acid
        │ ★ MMF / Ribavirin                  │
        ▼  阻 IMP dehydrogenase              ▼
   AMP & GMP                              UMP → UDP
                                            │ ★ Hydroxyurea 阻
                                            ▼ Ribonucleotide Reductase
                                          dUDP → dUMP
                                            │ ★ 5-FU 阻
                                            ▼ Thymidylate Synthase
                                           dTMP → DNA

        ★ MTX / TMP / Pyrimethamine 阻 DHFR
        → THF↓ → dTMP↓（间接抑制嘧啶 + 嘌呤双通路）
```

| 药 | 作用位点 | 用途 |
|---|---|---|
| **Azathioprine / 6-MP** | PRPP amidotransferase | IBD / ALL / 移植 |
| **MMF / Ribavirin** | IMP dehydrogenase | 移植 / HCV / RSV |
| **Leflunomide** | Dihydroorotate dehydrogenase | RA |
| **Hydroxyurea** | Ribonucleotide reductase | CML / 镰刀贫 |
| **5-FU** | Thymidylate synthase | 结直肠癌 / 乳腺癌 |
| **MTX / TMP / Pyrimethamine** | DHFR | RA / 抗菌 / 弓形虫 |

> [!tip] 通路口诀
> "**P**RPP → **I**MP → **A**MP / **G**MP" — **嘌呤** Tree
> "**O**rotic → **U**MP → **dU**DP → **dT**MP" — **嘧啶** Tree

## 十、★ 用药前 / 用药中"基线检查"必查表（AMBOSS 高频考点）

> [!success] CK 反复考的"开药前查什么"
>
> | 药 | 用药前必查 | 用药中监测 |
> |---|---|---|
> | **TNF-α inhibitor** (infliximab / adalimumab) | **PPD or IGRA**（latent TB）★ + **Hep B / Hep C 血清** ★ + 更新疫苗 + （心衰史 → echo）| 感染征象 / 病毒重激活 |
> | **MTX** | **CBC + LFT** + Cr + 妊娠测试 | CBC q月 / LFT q3 月 / 出现呼吸症状 → PFT |
> | **Azathioprine** | **TPMT or NUDT15 酶活性** + CBC + LFT + **筛查现用药 (尤其 allopurinol)** ★ | CBC q月 + 避免 allopurinol |
> | **Mycophenolate** | CBC + 妊娠测试（致畸）| CBC + GI 症状 + CMV 监测 |
> | **CNIs (tacro / cyclo)** | Cr + 血压 + 血糖 + 血脂 | **血药浓度** ★ + Cr + 电解质 |
> | **Sirolimus** | CBC + 血脂 + 血糖 | CBC + 脂 + 伤口 + 肺症状 (pneumonitis) |
> | **Hydroxychloroquine** | **基线眼检（视网膜）** ★ | **每年眼检 retinopathy** |
> | **长期 Glucocorticoids** | 血糖 / DEXA / 血压 / Hep B | 血糖 + 骨密度 q1-2 年 + 渐撤药防 adrenal crisis |
> | **Cyclophosphamide** | UA + 妊娠测试 | **+ Mesna 防 hemorrhagic cystitis** ★ |
> | **Cisplatin** | Cr + 听力 | **+ Amifostine 防肾毒** |
> | **Doxorubicin** | Echo (EF) | **+ Dexrazoxane 防心毒** |

### TNF-α inhibitor 用前 4 大筛查 ★

| 项目 | 原因 |
|---|---|
| **PPD / IGRA** | TNF 维持肉芽肿对 TB 的隔离；抑制 TNF → 肉芽肿崩解 → latent TB 重激活 ★ |
| **Hep B 血清 (HBsAg + Anti-HBc)** | TNF-i 可致 HBV 重激活 → **fulminant 肝衰** ★ |
| **Hep C 血清** | 类似机制（相对较少）|
| **心脏评估**（HF 史时）| TNF-i **中-重度心衰 NYHA III/IV 禁用** |
| **疫苗** | 开始 biologics 前更新；治疗中**禁活疫苗** |

### ★ 免疫抑制者疫苗规则（CK 高频）

> [!danger] 通用规则
> **所有免疫抑制剂使用者**（CNI / mTOR-i / Antimetabolites / Biologics / 长期高剂量 Glucocorticoids）：
> - ❌ **禁活疫苗**：MMR / Varicella / 鼻喷流感 (LAIV) / 黄热病 / BCG / 口服脊灰
> - ✅ **可灭活疫苗**：灭活流感 / Tdap / HPV / Hep A&B / PCV13 + PPSV23 / Meningococcal / 灭活脊灰 / 新冠 mRNA
> - **理想时机**：免疫抑制开始**前 ≥ 4 周**更新所有疫苗

> [!warning] 类固醇 "高剂量" 阈值
> - **Prednisone ≥ 20 mg/d × ≥ 14 天** = 免疫抑制 → 同上规则
> - 短疗程（< 14 天）/ 低剂量 / 局部用 = **不算**免疫抑制，活疫苗可用

> [!info] 移植病人疫苗
> 实体器官移植**前** = 抓紧补所有活疫苗（≥ 4 周缓冲）
> 移植**后** = 终生**禁活疫苗**（即使停免疫抑制后也是）
> 详细 HIV / CD4 < 200 疫苗策略 → [[完整笔记/Peixuan分科笔记/免疫，疫苗]] line 1951-1980

## 十一、副作用对照速查表（5 维度）

| 药 | **肾毒** | **骨髓抑制** | **代谢** | **特征性副作用** |
|---|---|---|---|---|
| **Cyclosporine** | ★ 强 | + | ↑ 糖 / 脂 | **牙龈增生 + 多毛 + 皮肤鳞癌** |
| **Tacrolimus** | ★ 强 | + | ↑ 糖 (尤其) | **无**牙龈/多毛；**神经毒强** + 糖尿病 |
| **Sirolimus** | **✗ 不肾毒** ★ | **★ Pancytopenia** | ↑ 脂 / Insulin R | 间质肺炎 / 伤口差 |
| **Azathioprine** | — | + | — | **+ Allopurinol = 致命**（XO 互动） |
| **MMF** | — | + | + 高血糖 | **GI 三联（腹泻/呕吐）+ CMV 感染** |
| **MTX** | + (高剂量) | ★ | — | 肝毒 + 肺纤维化 + 致畸 + 巨幼贫血 |
| **Basiliximab** | — | — | — | 轻 (edema / 震颤) |
| **TNF-α inhibitor** | — | + (轻) | — | **TB / HBV 重激活** + 脱髓鞘 + 心衰恶化 |
| **Glucocorticoids** | — | **反向 → 白细胞↑** | ↑↑ 糖 + Cushing | **突停 → adrenal crisis** |

## 十二、移植 vs 自身免疫场景速记

### 实体器官移植"标准三联"

```
诱导期（术后即时）：
   Basiliximab IV 1-2 剂（阻 IL-2R 短期）
   ± 大剂量 methylprednisolone

维持期（长期）：
   Tacrolimus  (CNI 主力)
   + MMF       (抗增殖)
   + 低剂量 prednisone  (广谱)

肾移植 specifically:
   可换 / 加 Sirolimus（不肾毒，保护移植肾）

急性排斥反应:
   大剂量 IV methylpred 冲击 / 加 ATG / OKT3
```

### 自身免疫病一线

| 病 | 一线 / 二线 |
|---|---|
| **RA** | MTX 一线 → +/− leflunomide / sulfasalazine / hydroxychloroquine → biologics (TNF-i) |
| **AS (强直性脊柱炎)** | NSAIDs + 运动 → **TNF-α inhibitor**（infliximab / adalimumab）★ |
| **IBD (Crohn / UC)** | 5-ASA → 类固醇桥接 → **Azathioprine / 6-MP** 维持 → biologics |
| **SLE** | Hydroxychloroquine 全员；严重 → MMF / azathioprine / cyclophosphamide / belimumab |
| **Psoriasis** | 局部 → MTX / Cyclosporine / biologics |
| **MS** | β-interferon / glatiramer / 单抗 (natalizumab, ocrelizumab) |
| **MG** | Pyridostigmine → Prednisone + **Azathioprine** 维持 / 胸腺切除 |

## 十三、3 个反直觉双向陷阱 ⭐⭐⭐

> [!danger] CK 最爱反复考 — 看到 stem 千万别想反

### ① "高血糖 + 全血减少" → MMF（不是 CNI）

- CNI（tacro / cyclo）= **高血糖 ✓，全血减少 ✗**
- MMF = **高血糖 ✓，全血减少 ✓**
- **同时见两个 = MMF** ★（AMBOSS Q4）

### ② "肾毒 + 神经毒 + 高血糖" → 想 CNI（不是 MMF）

- CNI = **三联齐**
- MMF = 不肾毒、不神经毒
- 反过来分流

### ③ "牙龈增生 + 多毛" = Cyclosporine（不是 Tacrolimus）

- Tacrolimus 比 Cyclosporine 更"现代"用得多，但**这两条特征副作用 cyclosporine 独有**
- 见牙龈增生 → 想三个药：**Cyclosporine / Phenytoin / CCB (nifedipine, verapamil)**

## 十四、CK 高频陷阱合集

> [!question]- Q1: 牙龈增生 + 多毛的免疫抑制剂？
> A: **Cyclosporine**（不是 tacrolimus）

> [!question]- Q2: IBD 病人用 azathioprine 维持，痛风发作给 allopurinol → 2 周后全血减少。机制？
> A: **Allopurinol 抑 XO → 6-MP 不能被代谢 → 血药 ↑↑ → 致命骨髓抑制**
> 必须 azathioprine 减量 75% 或换 **cyclosporine**

> [!question]- Q3: MG 病人 prednisone + azathioprine 1 mo，**痛风共病用 allopurinol** → 全血减少。本可如何避免？
> A: **Assessment of potential drug interactions** (AMBOSS Q7)
> 干扰项警示：
> - Folate = MTX 用，不是 aza
> - Prednisone 致 leukocytosis + ↑Hb/RBC，不致 pancytopenia
> - CT chest 查 thymoma → 致 **PRCA**（不是 pancytopenia）

> [!question]- Q4: 肾移植后要选**不肾毒**的免疫抑制剂，选哪个？
> A: **Sirolimus**（mTOR-i，pancytopenia 但不肾毒）— "Sir-vives kidney"

> [!question]- Q5: 长期 prednisone 病人感冒自行停药 → 第二天低 BP + 嗜睡 + 低 Na + 高 K。诊断 + 治疗？
> A: **Adrenal crisis** → **IV hydrocortisone 100 mg q8h + IV NS**

> [!question]- Q6: 实体器官移植 tacrolimus 维持，治真菌感染加 fluconazole → 2 周后 Cr 升。机制？
> A: **Fluconazole 抑 CYP3A4 → tacrolimus 血药 ↑ → 急性肾毒**（同样 erythromycin / 葡萄柚 / verapamil/diltiazem）
> 详见 [[完整笔记/专题笔记/肾脏/肾脏_移植后AKI鉴别SOP]] + [[mistakes/uworld-mistakes#^Q16485]]

> [!question]- Q7: 移植病人长期 MMF + 突发胃肠不适伴肺部异常。警惕什么感染？
> A: **CMV**（MMF 特征性增加 CMV 感染风险）

> [!question]- Q8: MTX 高剂量化疗，给 leucovorin。leucovorin 作用？
> A: **Rescue** — 跳过 DHFR 补 THF，**救正常细胞骨髓 + 黏膜**，不影响 MTX 抗肿瘤

> [!question]- Q9: 结直肠癌化疗给 5-FU + leucovorin。leucovorin 作用？
> A: **增强 5-FU 抗肿瘤**（与 MTX 用法**相反**）— 三元复合物 (5-FdUMP + TS + THF) 更稳定

> [!question]- Q10: 长期类固醇病人查血 WBC 22000，无发热感染征象。机制？
> A: **Demargination of WBCs** = **artificial leukocytosis**，不代表感染

> [!question]- Q11: RA 病人 MTX 治疗中肝酶升 2 倍 + 干咳 4 周。最警惕？
> A: **MTX 肝毒 + 间质性肺纤维化** — 停药 + 影像评估

> [!question]- Q12: AS 病人 NSAIDs + 运动 6 周无改善，开始 TNF-α inhibitor。开始前查什么？
> A: **PPD / IGRA**（latent TB 筛查）★（AMBOSS Q5）

> [!question]- Q13: RA 病人 MTX + HCQ 失败要换 TNF-α inhibitor。除 PPD 外还要查？
> A: **Hep B + Hep C 血清** ★（AMBOSS Q6）— TNF-i 致 HBV 重激活 → fulminant 肝衰

> [!question]- Q14: 肾移植 1 mo 后 + 腹泻 + 呕吐 + 全血减少 + 高血糖。最可能的免疫抑制剂副作用？
> A: **Mycophenolate mofetil**（MMF "三联"：全血减少 + 高血糖 + GI）★（AMBOSS Q4）
> CNI 高血糖但**不全血减少** → 排除

> [!question]- Q15: 肝移植后牙龈增生 + 多毛 + HTN + 高血糖 + 黑色粗毛。最可能的药？
> A: **Cyclosporine** ★（AMBOSS Q1）
> Tacrolimus 高血糖 + HTN 都有，但**无牙龈 / 多毛**

> [!question]- Q16: 类风湿病人 hydroxychloroquine 长期用，定期需查？
> A: **眼底检查 retinopathy** 每年

> [!question]- Q17: 化疗用 cyclophosphamide，预防什么副作用要加什么药？
> A: **Hemorrhagic cystitis** → 加 **Mesna**（结合 acrolein）
> 长期还要监测 **bladder cancer**

> [!question]- Q18: 5 yo 儿童因 JIA 用 etanercept (TNF-α inhibitor) 中。学校要求打 MMR。能打吗？
> A: **不能打 MMR**（活疫苗）★
> 同样禁：水痘 / 鼻喷流感 / BCG / 黄热
> 可打：灭活流感 / Tdap / Hep B / HPV / PCV / 灭活脊灰

> [!question]- Q19: 25 yo F 慢性哮喘，急性发作打了 prednisone 60 mg × 5 天。下个月想打鼻喷流感疫苗，能打吗？
> A: **能打**（短疗程 < 14 天不算免疫抑制）
> 如果是 prednisone ≥ 20 mg/d × ≥ 14 天 = 算免疫抑制 → 禁活疫苗

## 十五、Pattern 速查："看到 XX → 想 YY 药"

| 症状 / 检查异常 | 优先想到的药 |
|---|---|
| 牙龈增生 + 多毛 | **Cyclosporine** |
| 全血减少 + 高血糖 + GI | **MMF** |
| 移植后 Cr 升 + 加了 azole / CCB / 葡萄柚 | **CNI 中毒**（CYP3A4 互动）|
| RA 新药后骨髓抑制 | **MTX**（没给 leucovorin）|
| IBD / MG 病人加 allopurinol 后骨髓抑制 | **Azathioprine + Allopurinol** |
| 移植后 invasive CMV | **MMF** |
| 长期类固醇病人突停 + 休克 | **Adrenal crisis** |
| 移植病人新发皮肤鳞癌 | **Cyclosporine**（长期免疫抑制 + UV）|
| TNF-α 开始后发热 + 咳嗽 + 盗汗 | **Latent TB 重激活**（没筛查）|
| TNF-α 开始后黄疸 + ALT ↑↑ | **HBV 重激活 → fulminant 肝衰**（没筛查）|
| 视网膜病变 | **Hydroxychloroquine** 长期 |
| 胸闷干咳 + 间质改变 | **MTX 或 Amiodarone** 肺毒 |
| Sirolimus 病人胸痛干咳 | Sirolimus **non-infectious pneumonitis** |
| Cyclophosphamide 后血尿 | **Hemorrhagic cystitis**（没给 mesna）|
| 长期类固醇 WBC ↑ 但无感染征象 | **Demargination**（假白细胞增多）|

## 十六、记忆口诀

> [!tip] CNI 副作用三联 "**NHL**"
> **N**ephrotoxicity / **H**TN / **L**ipid + Neuro
> Cyclosporine 额外 = "**Gum + Hair**"

> [!tip] "**Sir-vives** kidney"
> **Sir**olimus 不肾毒 + 肾移植 specifically

> [!tip] "**Aza + Allo = ⚠️**"
> 首字母都 A，但合体会爆炸 — XO 被抑制 → 6-MP 累积 → 骨髓死

> [!tip] Leucovorin 双面
> "**MTX 救正常 / 5-FU 救肿瘤**"（同药相反作用）

> [!tip] Glucocorticoid 三大魔咒
> "**升糖 / 升压 / 突停 = 死**"
> 反直觉：**WBC ↑ + Hb ↑**（不是 ↓）

> [!tip] MMF 双特征
> "**腹泻 + CMV**"（移植病人腹泻 + 发热警惕 CMV colitis）
> "**全血减少 + 高血糖 + GI**" 三联锁定 MMF

> [!tip] TNF-α 用前 "**TB + HBV**" 双筛
> PPD/IGRA + Hep B/C — 漏一个就死人

# 免疫_疫苗禁忌与接种规则（Vaccine Contraindications & Schedule）

> [!info] 用途
> 疫苗**禁忌 vs 注意事项 vs 非禁忌** + 各疫苗特异禁忌（DTaP/轮状/活疫苗）+ 蛋类过敏 + 水痘疫苗株皮疹 + 破伤风伤口矩阵 + 儿童时间表 + 疫苗犹豫沟通。来源：Demi 讲课。时间表按 CDC 核对。

---

## 一、禁忌 vs 注意事项 vs 非禁忌（总原则）

> [!success] 非禁忌（照常接种）
> - **轻度急性病**（URI、中耳炎、**低烧**）
> - **正在服抗生素**
> - 既往接种后发热、单纯热性惊厥史

> [!danger] 唯一例外
> **口服活伤寒 / OPV 口服脊灰**——正在服抗生素时活菌/活病毒**被灭活**，须推迟。

> [!warning] 注意事项 Precaution（评估利弊、可延迟）
> 中/重度急性病、进展性神经病、未控制癫痫。

---

## 二、各疫苗特异禁忌/注意事项

### DTaP

| 绝对禁忌 | 注意事项（延迟至稳定）|
|---|---|
| 对成分**严重过敏 anaphylaxis** | **进展性神经疾病** |
| 接种 **7 天内不明原因脑病** | **未控制癫痫**（婴儿痉挛/进展性脑病）|
| | 接种后 6 周内 GBS；中/重度急性病 |

> [!info] DTaP 神经事项原理
> 接种后发热降低癫痫阈值 / 难与基础病进展区分 → 脑子"活跃"（未控发作）时先等，稳定后再种；担心发热可预防性对乙酰氨基酚。

### 轮状病毒 Rotavirus（口服减毒活）

| 绝对禁忌 | 注意事项 |
|---|---|
| **肠套叠 intussusception 史** | 脊柱裂、膀胱外翻 |
| **SCID** | |
| 严重过敏 | |

- 年龄窗：**15 周前**首剂，**8 月龄前**末剂。CF 本身**不是**禁忌。
- 肠套叠经典：阵发腹痛 + **果酱样便** + 香肠样肿块 → **空气/盐水灌肠**（诊断+治疗）。

### 活疫苗通用禁忌
**孕妇** + **严重免疫抑制**（移植受者、**HIV CD4 <200**，见 [[mistakes/uworld-mistakes_2026-06#^Q4762]] SCID 禁活疫苗）。

## 三、蛋类过敏与流感疫苗

> [!tip]
> - **仅局部荨麻疹** → **不禁**流感疫苗，正常接种。
> - 仅**严重过敏反应 anaphylaxis** 才是禁忌。
> - **DTaP / Hib / PCV 不含蛋**——蛋过敏与这些无关。

> [!danger] 蛋过敏唯一真正要避开的疫苗 = 黄热 (yellow fever)
> 鸡蛋过敏 → **只需避开黄热病疫苗 (yellow fever vaccine)**；**其余全部照打——包括 MMR！**（MMR 在鸡胚成纤维细胞培养，含蛋白极微，蛋过敏非禁忌）。流感疫苗历史上也提，但现行指南蛋过敏仍可接种（必要时观察）。

---

## 四、水痘疫苗株皮疹（vaccine-strain vs wild-type）

> [!info] 机制
> 水痘疫苗 = 减毒活，接种后病毒有限复制；3–5% 出现轻微皮疹 = **疫苗株复制，非超敏反应**。

| 特征 | 疫苗株 | 野毒 |
|---|---|---|
| 时间 | **接种后 1–3 周** | 无近期接种/有接触 |
| 皮疹数 | **少（<10）** | 多（>100）|
| 传染性 | 极低 | 高 |

> [!tip] Memory Hook
> **"Few and late"——1–3 周、<10 个皮疹 = 疫苗株（非超敏）**；通常无需治疗、不强制隔离。

## 五、破伤风伤口预防矩阵

> [!info] 破伤风疫苗 = **类毒素 toxoid**（主动免疫）；**TIG = 被动免疫**中和游离毒素。

| 接种史 | 清洁小伤口 | 脏/重伤口 |
|---|---|---|
| **未接种 / <3 剂** | **仅疫苗** | **疫苗 + TIG** |
| ≥3 剂，末次 **<5 年** | 不需要 | 不需要 |
| ≥3 剂，末次 **5–10 年** | 不需要 | **疫苗** |
| ≥3 剂，末次 **>10 年** | **疫苗** | **疫苗** |

> [!example] 速记
> 从未接种 + 脏伤口 → **TIG + 疫苗**；7 年前接种 + 脏伤口 → 只疫苗；3 年前接种 → 不需要。

> [!info] 破伤风临床/治疗（背景）
> tetanospasmin 逆行轴突运输 → 抑制 GABA/甘氨酸 → **牙关紧闭 + 角弓反张 + 自主神经亢进**。诊断=临床。治疗：清创 + **甲硝唑 metronidazole** + **TIG** + Tdap。

## 六、儿童接种时间表（CDC 简表 + 助记）

```
出生      HBV #1（早产 <2kg 且母 HBsAg 阴性 → 推迟至 1 月龄/出院）
2/4/6 月  "DR HIP" = DTaP · Rotavirus · Hib · IPV · PCV（6 月再加 HBV#3）
6 月起    流感 Influenza 每年（10–4 月季）
12–15 月  MMR#1 · Varicella#1 · Hib · PCV · HepA#1
15–18 月  DTaP（第 4 剂）
4–6 岁    DTaP#5 · IPV#4 · MMR#2 · Varicella#2
11–12 岁  Tdap · HPV · MenACWY
```

> [!tip] 要点
> **常规疫苗剂量不按体重调整**；HBV 出生即种；活疫苗（MMR/水痘）满 12 月才种。

## 七、减毒活疫苗清单（CK 必考）

> [!warning] Live-Attenuated
> 病毒：**MMR、水痘 Varicella、带状疱疹活疫苗 Zostavax、鼻喷流感 (intranasal influenza, LAIV)、口服脊灰 OPV、黄热 yellow fever、轮状 Rotavirus**。细菌：**BCG 卡介苗、口服伤寒**。
> 禁忌人群：孕妇、严重免疫缺陷（HIV CD4 <200、移植）→ 改用**灭活疫苗**。
> ⚠️ 注意：带状疱疹现行美国首选 **Shingrix（重组灭活，非活）**；Zostavax（活）已基本淘汰但 NBME 旧题仍可能列为活疫苗。

## 八、疫苗犹豫沟通（伦理/行为科学）

> [!info] 核心：建"同盟"非对立 —— Empathy → Open-ended Q → Educate → Document

| 步骤 | 技巧 | 示例 |
|---|---|---|
| 认可 | 同理心 | "我理解您想保护孩子" |
| **探索** | **开放式提问** | "能说说您具体担心什么吗？" |
| 教育 | 事实 | "婴儿每天接触的环境抗原远超疫苗" |
| 记录 | 最后手段 | 持续拒绝才签拒绝书 |

> [!warning] 错误应对
> 闭合式提问 ✗｜过早提签拒绝书 ✗｜未倾听就纠正 ✗｜不当折中延误免疫 ✗。
> 拒绝后：**维持医患关系不逐出诊所**，**每次就诊重启讨论**，病历记录。**Listen first, Educate second, Document always.**

## 九、成人特殊人群疫苗 — 慢性肝病/肝硬化（CK 高频）

> [!info] 原则
> **慢性肝病（chronic liver disease）/ 肝硬化（cirrhosis）= 常规成人疫苗 + 疾病特异疫苗（甲肝 + 乙肝 + 肺炎球菌）**。理由：慢性肝病者一旦感染病毒性肝炎 → **肝功能失代偿（hepatic decompensation）风险↑**。

**慢性肝病推荐疫苗**

| 疫苗 | 频率 / 时机 |
|---|---|
| **破伤风 Tetanus（Td / Tdap）** | 每 **10 年**；**Tdap 一生一次**（未打过 Tdap → 用 Tdap 替一次 Td，不论距上次多久）|
| **流感 Influenza** | **每年**；**肌注灭活**优于鼻喷减毒活疫苗（LAIV, live attenuated influenza vaccine）|
| **肺炎球菌 Pneumococcus** | **诊断时即种**：PCV20 单针，或 PCV15 → PPSV23 |
| **甲肝 Hepatitis A** | **无免疫证据则全程接种** |
| **乙肝 Hepatitis B** | **无免疫证据则全程接种** |

> [!tip] 肺炎球菌成人接种适应证（PCV20 或 PCV15→PPSV23）
> - **所有 ≥65 岁**；
> - **<65 岁 + 高危共病**：**慢性肝/肺/心病、糖尿病（DM）、吸烟**、无脾（asplenia）、免疫抑制、CSF 漏 / 耳蜗植入。
> - **PCV20** = 20 价肺炎球菌结合疫苗（pneumococcal conjugate vaccine）；**PPSV23** = 23 价多糖疫苗；**Td** = 破伤风-减量白喉；**Tdap** = 破伤风-减量白喉-无细胞百日咳。

> [!warning] 易错点（[[mistakes/uworld-mistakes_2026-06#^Q4476]]）
> - **血清学阴 = 无免疫 → 甲肝 + 乙肝都要种**（本题 HAV IgG 阴 + HBsAb 阴 → 两个都补）。
> - 上次破伤风 **≥10 年前** → 本次补 Td/Tdap（本题 31 岁打过、现 42 岁 = 11 年 → 该补；未打过 Tdap → 用 Tdap）。
> - 慢性肝病加 **肺炎球菌**（疾病特异），别只给常规三联（破伤风/流感/甲乙肝）漏掉它。

## 十、疫苗类型 → 免疫应答 + 是否要加强针（CK 概念题）

| 类型 | 免疫应答 | 加强针 | 例子 |
|---|---|---|---|
| **减毒活 Live-attenuated** | **T 细胞 + 强而持久** | 一般**不需**（终身） | MMR / 水痘 / Zostavax / 鼻喷流感 / OPV / 黄热 / 轮状 / BCG / 口服伤寒 |
| **灭活 Killed/Inactivated** | **体液 (humoral) 为主** | **需要加强** | 甲肝 / 乙肝 / 肌注流感 / 注射脊灰 (IPV) / 狂犬 / **Shingrix** |
| **类毒素 Toxoid** | 抗毒素抗体（体液）| 需要（如 Td q10y）| 破伤风 / 白喉 |
| **多糖 Polysaccharide** | **体液、无 T 细胞、<2 岁无效** | — | **PPSV23** |
| **结合 Conjugate** | 多糖 + 蛋白载体 → **B + T 细胞、婴儿有效** | 见儿童程序 | **PCV13/PCV15/PCV20** / Hib / MenACWY |

> [!tip] Memory Hook
> **活 = T 细胞、不用加强；灭活/类毒素/多糖 = 体液、要加强。** 多糖 (PPSV23) <2 岁无效 → 婴儿用**结合 (PCV)**。

## 十一、肺炎球菌 PCV/PPSV 接种时序 + 无脾 (Pneumococcal Sequencing & Asplenia)

> [!info] 现行 vs 经典
> **现行简化（2024 CDC）**：≥65 岁或高危 → **PCV20 单针**，或 PCV15 → 1 年后 PPSV23。
> **经典 NBME 版（仍考时序逻辑）**：**先 PCV13 → 再 PPSV23**（"13 在 23 前"）。下表保留经典逻辑，遇 PCV20 题按单针处理。

| 人群 | 经典方案 |
|---|---|
| **<65 岁 + 免疫抑制 / 无脾 / 镰状(功能无脾) / HIV / CKD / 肾病综合征 / 移植 / 血液恶性 / 耳蜗植入 / 脑脊液漏** | **PCV13 → 8 周后 PPSV23**；5 年后再补 1 针 PPSV23（≥65 岁后再终生 1 针）|
| **<65 岁 + 慢性病（糖尿病 / 吸烟 / 心 / 肺 / 肝病 / 酗酒）** | **仅 PPSV23** |
| **≥65 岁（健康）** | **PPSV23 单针**，或 PCV13 → 1 年后 PPSV23 |

> [!warning] 无脾 / 功能性无脾 (asplenia / functional asplenia) → 防荚膜菌三联
> **脾切 / 镰状细胞 (sickle cell)** 患者必须接种 **3 种荚膜菌 (encapsulated organisms) 疫苗**：
> - **肺炎链球菌 (S. pneumoniae)** = PCV + PPSV23
> - **流感嗜血杆菌 b 型 (H. influenzae type b, Hib)**
> - **脑膜炎奈瑟菌 (N. meningitidis)** = MenACWY + MenB
> - 另：脾切/镰状儿童加 **青霉素 (penicillin) 或阿莫西林 (amoxicillin) 预防至 5 岁**。
> - 择期脾切 → 术前 **≥2 周**接种（保证应答）。

## 十二、成人特殊疫苗：HPV / 带状疱疹 / 脑膜炎球菌

> [!info] HPV 疫苗（人乳头瘤病毒 human papillomavirus）
> - 常规 **11–12 岁**起（可早至 9 岁），补种至 **26 岁**。
> - Divine 旧分法：女 9–26 / 男 11–21 / **MSM（男男性行为者）至 26**。
> - **<15 岁 2 剂；≥15 岁或免疫抑制 3 剂。** 孕期不打（非活疫苗但数据有限，推迟至产后）。

> [!info] 带状疱疹 (Herpes Zoster)
> - **Shingrix（重组灭活）= 现行首选**：**≥50 岁，2 剂间隔 2–6 个月**；免疫抑制者也可用（因非活）。
> - Zostavax（活）已淘汰，免疫抑制禁用。

> [!info] 脑膜炎球菌 (Meningococcal, MenACWY / MenB)
> 适应证：**11–12 岁（16 岁加强）、大学新生住宿、军队新兵、无脾、补体末端缺陷 (terminal complement deficiency, C5-9)、用依库珠单抗 (eculizumab，抗 C5 单抗) 者、旅行至流行带**。
> - 暴发流行 → 治疗用头孢曲松 (ceftriaxone)/环丙沙星，密接 **利福平 (rifampin) 预防**。

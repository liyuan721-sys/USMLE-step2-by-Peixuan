## 一、免疫球蛋白时相（Immunoglobulin timing）

> [!tip] 核心钥匙
> **IgM = 首次应答（current/新发感染）**；**IgG = 长期应答（既往感染或免疫 + 长期保护）**；**加强针 booster = 诱导 IgG 记忆**。

```
IgM:      出 3 天 → 峰 2 周 → 退 2 月
IgG:      出 2 周 → 峰 2 月 → 退 1 年
IgG 记忆:  出 3 天 → 峰 5 年 → 退 10 年
```

- **婴儿免疫发育**：约 **6 月前**主靠 IgM → **6 月**开始自产 IgG → **18 月**（= 6 月 + 1 年）诱导 IgG 记忆（活疫苗 MMR/水痘进场）。
- **IgM↑** = 当前新发感染；**IgG↑** = 既往感染 / 已免疫 / 长期保护。

## 二、疫苗程序与活/灭活（Vaccine schedule & live/killed）⭐

### 2.1 儿童程序骨架

> 缩写：**Hib = b 型流感嗜血杆菌疫苗 (Haemophilus influenzae type b)**；**PCV = 肺炎球菌结合疫苗 (pneumococcal conjugate vaccine)**；**DTaP = 白喉-破伤风-无细胞百日咳 (diphtheria-tetanus-acellular pertussis，大写=儿童剂量)**；**Tdap = 成人加强版 (tetanus-diphtheria-acellular pertussis，小写=低剂量)**；**MMR = 麻疹-腮腺炎-风疹 (measles-mumps-rubella)**；**HPV = 人乳头瘤病毒疫苗 (human papillomavirus)**。

```
出生:       Hep B（乙肝 hepatitis B）
2 月:       Hep B / Hib / 肺炎球菌 PCV / 轮状 Rotavirus(活,口服) / 脊灰 Polio / DTaP
4 月:       同 2 月，但无 Hep B（Hib/PCV/Rotavirus/Polio/DTaP）
6 月:       Hep B / 流感 Influenza(起) / 轮状(末剂) / DTaP / Polio / Hib / PCV
18 月:      + 活疫苗 MMR + 水痘 Varicella（诱导 IgG 记忆）
4–6 岁:     小学加强（DTaP booster）
~9 岁:      HPV 可起
16 岁:      青少年加强（把免疫带进 20 岁出头）
```

> [!warning] 轮状病毒 Rotavirus 红线
> **活、口服疫苗；6 月后停**（年龄窗严格，与**肠套叠 intussusception** 风险管理相关，不能无限延后）。

### 2.2 活疫苗清单（超高频——免疫抑制 / 孕妇禁用）

> [!danger] 活疫苗 = 体液 + 细胞免疫；免疫抑制 & 孕妇慎用 / 禁用
> **MMR / 水痘 Varicella / 口服脊灰 (Sabin) / 轮状 Rotavirus / 黄热 Yellow fever / 卡介苗 BCG / 天花 Smallpox / 鼻喷流感 (intranasal)**。
> 灭活 = 仅体液：**IM 脊灰 (Salk) / 流感针剂 (flu shot)**。

> [!tip] 孕前 & 免疫抑制决策
> - **备孕女性**：孕前补 **MMR / 水痘**（活疫苗，孕后不能随便补），理想 **孕前 2–3 月**。
> - **免疫抑制患者**：避所有活疫苗（MMR / 水痘 / 鼻喷流感）。

### 2.3 特定疫苗钥匙

| 疫苗 | 钥匙 / 阈值 | 决策点 |
|---|---|---|
| Hep B（新生儿） | 妈妈 **HBsAg 阳性** | 出生后尽快疫苗 **+ HBIG（乙肝免疫球蛋白）**；可母乳（只要婴儿完成免疫） |
| 流感 Influenza | **≥6 月所有人，每年一次（10–11 月）** | 鼻喷 = 活；针剂 = 灭活；**父亲鸡蛋过敏不影响给婴儿** |
| 肺炎球菌 Pneumococcal | **≥65（或高危 ≥50）+ 免疫抑制 + 无脾** | PCV 结合(记忆好) vs **PPSV23（23 价肺炎球菌多糖疫苗 pneumococcal polysaccharide vaccine）** 多糖(覆盖广)；详见 🧩 |
| Tdap | **孕 28 周每次妊娠 / 脏伤口 / 旅行** | "大写 DTaP 给小孩，小写 Tdap 给大人" |
| 水痘 / 带状疱疹 Zoster | **Shingrix（重组灭活）≥50 岁打 2 剂**（现行首选）；旧 Zostavax（活疫苗，已淘汰）≥60 岁 1 剂 | 别把剂次记反：新苗 50 岁 2 剂 |
| MMR | 育龄期 / 医护；**孕前 2–3 月** | 孕期❌ 免疫抑制❌ |
| 脑膜炎球菌 Meningococcal | **大学新生 / 无脾 / 军人 / 旅行至流行区** | — |
| HPV / Gardasil 9 | **9–45 岁**（常规推荐至 26 岁；27–45 岁共同决策），覆盖 6/11/16/18 +31/33/45/52/58（9 价） | — |
| Hep A | **出国旅行 / Hep C / 肝病或凝血问题** | 粪-口传播，注意同住接触者 |
| 狂犬 Rabies | 不明动物咬 / **蝙蝠·臭鼬暴露（即使无明显咬痕）** | **RIG 免疫球蛋白 + 疫苗** |

> [!warning] 肺炎球菌序贯钥匙（PCV vs PPSV23）
> - **PCV20 或 PCV21 打完 → 通常完成，不补 PPSV23**。
> - **PCV15 打完 → 后面要接 PPSV23**（一般 1 年后）。
> - **免疫抑制 / CSF 漏 / 人工耳蜗 → 最短 8 周间隔**。
> - **高危（无脾 / 免疫抑制）**：PCV13 → **间隔 8 周** → PPSV23（缩短间隔尽快全覆盖）。

### 2.4 破伤风脏伤口逻辑

> [!danger] "Clean 十，dirty 五，无免 dirty TIG"
> - **干净伤口**：距上次 **>10 年** → 补疫苗。
> - **脏伤口**：距上次 **>5 年** → 补疫苗。
> - **从未免疫 / 免疫史不明 + 脏伤口** → **TIG（破伤风免疫球蛋白）+ 疫苗**（毒素病，疫苗来不及，TIG 给即时被动免疫）。
> - 反例（不需补）：疫苗 3 年前 + 现旅行 / 干净小伤 → 不需要。

## 三、超敏反应 4 型（Hypersensitivity）

> [!tip] 一句话锁 4 型（时相是关键鉴别）
> **I 型 = IgE / 肥大细胞，数分钟（速发）**；**II 型 = 抗体对细胞表面抗原**；**III 型 = 免疫复合物（IC），5–14 天**；**IV 型 = T 细胞，迟发（48–72h）**。

| 型 | 机制 | 代表病 / 钥匙 |
|---|---|---|
| **I（速发）** | IgE → 肥大细胞脱颗粒 | 过敏性休克 anaphylaxis / 过敏性鼻炎 / 荨麻疹 / 食物·药物速发过敏 |
| **II（抗体）** | 抗体直接打细胞表面抗原 | 自身免疫性溶血性贫血 / Goodpasture（anti-GBM）/ 急性溶血性输血反应 / Graves |
| **III（免疫复合物）** | IC（免疫复合物 immune complex）沉积 → 激活补体 | **血清病 serum sickness** / 系统性红斑狼疮 SLE / 链球菌后肾小球肾炎 PSGN (poststreptococcal glomerulonephritis) |
| **IV（迟发）** | T 细胞介导 | 接触性皮炎 / PPD 结核菌素试验 / SJS-TEN（Stevens-Johnson 综合征–中毒性表皮坏死松解症）/ 移植排斥 |

> [!warning] I 型 vs III 型时相（错题反复考）
> **给药后数分钟 + 皮肤风团 + 腹痛 + 咽紧（多系统）= I 型 anaphylaxis** → 第一步 **IM 肾上腺素 epinephrine**。
> **给药后 5–14 天 + 发热 + 荨麻疹 + 关节痛 (+ 轻蛋白尿) = III 型血清病**（异源蛋白：抗蛇毒 / 抗毒素 / 单抗如 infliximab）；多自限。
> **SS（真血清病）= 异源蛋白，IC 多 + 补体大量消耗 + 高热**；**SSLR（血清病样反应）= 药物（头孢克洛 cefaclor / 青霉素 / TMP-SMX），IC 轻、补体极少消耗、低热**，停药即可。

## 四、原发性免疫缺陷（PID — 反复感染模式定位）⭐

> [!tip] 反复感染病原 → 缺陷层次（最高频四分法）
> - **有荚膜菌反复呼吸道 / 鼻窦 / 中耳（肺炎链球菌等）= 体液（抗体 / B 细胞）**：CVID（常见变异型免疫缺陷 common variable immunodeficiency）、XLA（X 连锁无丙种球蛋白血症 X-linked agammaglobulinemia）。
> - **机会感染（PJP 肺孢子菌肺炎 Pneumocystis pneumonia / 真菌 / 病毒）+ FTT（生长发育停滞 failure to thrive）= T 细胞 / CD4**：SCID（重症联合免疫缺陷 severe combined immunodeficiency）、DiGeorge、HIV。
> - **catalase⁺（过氧化氢酶阳性）菌（金葡/Nocardia 诺卡菌/Serratia 沙雷菌/Burkholderia 伯克霍尔德菌）+ 曲霉 Aspergillus + 脓肿 / 肉芽肿 = 吞噬细胞**：CGD（慢性肉芽肿病 chronic granulomatous disease）。
> - **奈瑟菌 Neisseria 反复 = 补体 C5–C9（膜攻击复合物 MAC, membrane attack complex）**。

### 4.1 抗体缺陷三兄弟 + SCID + THI（查什么）

| 病 | 钥匙 | 第一步检查 |
|---|---|---|
| **XLA（Bruton 无丙球）** | 男婴、6 月后反复中耳炎/肺炎、扁桃体缺如、**↓↓所有 Ig + CD19 B 细胞极低** | 定量 Ig + 流式 CD19 |
| **CVID** | **成人 20–45 岁**反复鼻窦肺炎 + GI 感染 + 自免 + 淋巴瘤风险↑、**B 细胞正常** | **定量血清免疫球蛋白**（↓IgG + ↓IgA/IgM）+ 疫苗无应答 |
| **THI（一过性婴儿低丙球）** | 婴儿**孤立 IgG↓ + 疫苗应答正常 + 发育正常 + 血小板正常**、自限 | 定量 Ig + 疫苗特异 IgG（应答正常） |
| **SCID** | 婴儿**机会感染 + FTT + 淋巴减少、CD3 缺**、活疫苗致命 | 淋巴细胞分型（CD3 absent）；根治 = **干细胞移植** |
| **CGD** | 婴儿 **Nocardia/曲霉空洞肺炎 + 脓肿/肉芽肿、Ig 正常** | **DHR123 流式（二氢罗丹明 dihydrorhodamine）/ NBT 试验（硝基蓝四唑 nitroblue tetrazolium）测氧化爆发** |

> [!danger] 两组高频鉴别钥匙
> - **XLA vs CVID**：CD19（B 细胞）**没了选 XLA，正常选 CVID**；XLA 婴儿男、CVID 成人散发。
> - **THI（一过性婴儿低丙种球蛋白血症 transient hypogammaglobulinemia of infancy）vs Wiskott**：婴儿湿疹 + 反复感染 → **血小板正常 + 疫苗应答正常 = THI**；**血小板↓（小血小板）+ 湿疹 + 感染 = Wiskott-Aldrich 综合征（Wiskott-Aldrich syndrome）**。

## 五、自身免疫与血管炎（Vasculitis — 补体分水岭）

> [!tip] 小血管炎找病因：先查 ANCA + 补体
> **补体是分水岭**：
> - **低补体 → 免疫复合物型**：混合型冷球蛋白血症 MCS（mixed cryoglobulinemia syndrome）/ 系统性红斑狼疮 SLE / 链球菌后肾小球肾炎 GN / 药物性。
> - **补体正常 → ANCA（抗中性粒细胞胞浆抗体 antineutrophil cytoplasmic antibody）相关（pauci-immune 寡免疫）/ IgA 血管炎 / anti-GBM（抗肾小球基底膜抗体 anti–glomerular basement membrane）**。

| 类型 | 关键血清学 | 补体 | 钩子 |
|---|---|---|---|
| **ANCA 相关（GPA 肉芽肿性多血管炎/MPA 显微镜下多血管炎/EGPA 嗜酸性肉芽肿性多血管炎）** | ANCA（PR3/MPO）阳性，pauci-immune | **正常** | 肺 + 肾 + 鼻窦；无免疫复合物 |
| **混合型冷球蛋白血症 MCS** | 冷球蛋白 + **RF（类风湿因子 rheumatoid factor）阳性 + 抗 HCV（丙型肝炎病毒 hepatitis C virus）** | **低** | **HCV** 史、可触性紫癜 + 肾小球肾炎 GN + 周围神经病 + 肝大 |
| **IgA 血管炎（HSP 过敏性紫癜 Henoch-Schönlein purpura）** | 无血清学；活检 IgA 沉积 | 正常 | 儿童、腹痛 |

> [!warning] MCS 指纹三联
> **抗 HCV 阳性 + RF 阳性 + 低补体**（冷球蛋白 = 带 RF 活性的 IgM 抱 IgG，<37°C 沉淀 → IC 激活补体）。**RF 阳性 ≠ 类风湿**。

## 六、免疫抑制剂（Immunosuppressants — 移植）

> [!tip] 各药"人设"反射（看副作用反推药）
> - **钙调磷酸酶抑制剂 CNI（calcineurin inhibitor）（他克莫司 tacrolimus / 环孢素 cyclosporine）**：阻钙调磷酸酶 calcineurin → 阻 NFAT（活化 T 细胞核因子 nuclear factor of activated T cells）→ 阻 IL-2（白介素-2 interleukin-2）。多系统毒性五联 = **肾毒 + 高血压 + 神经毒（震颤）+ 糖耐量受损（↑胰岛素，tacro 尤甚）+ 高钾**。
> - **环孢素独有**：牙龈增生 + 多毛。
> - **霉酚酸酯 MMF（mycophenolate mofetil）**：腹泻（GI 胃肠道）+ 骨髓抑制。
> - **西罗莫司 sirolimus**：间质性肺炎。

> [!warning] 移植后多系统毒性钥匙 + 药物互动
> "**移植后 震颤 + 血糖↑ + 高血压 + 高钾 + 肾毒 = CNI（tacrolimus）**"。
> **CYP3A4（细胞色素 P450 3A4）抑制剂（地尔硫卓 diltiazem / 唑类抗真菌 azoles / 红霉素 erythromycin / 葡萄柚）→ 他克莫司血药↑ → 急性肾毒**。

## 七、过敏与血管性水肿（Allergy & angioedema）

> [!danger] Anaphylaxis 过敏性休克
> 诊断 = **暴露后急性发作 + ≥2 器官系统**（皮肤黏膜 / 呼吸 / 心血管 / GI）。机制 = **IgE 介导肥大细胞脱颗粒（I 型）**；首次暴露致敏（无症状）→ 再暴露交联 IgE → 脱颗粒。
> **第一步 = 肌注肾上腺素 IM epinephrine**。

> [!warning] 血管性水肿鉴别（无风团时）
> - **遗传性血管性水肿 hereditary angioedema, HAE（C1 酯酶抑制剂 C1 esterase inhibitor 缺乏）**：补体介导（缓激肽），**无荨麻疹**、反复面/喉/肠壁肿胀。
> - **ACEI 诱发血管性水肿**：缓激肽蓄积，**无风团**，停药。
> - 两者均**无荨麻疹**（区别于 I 型 IgE 过敏的风团）。

> [!tip] 青霉素过敏 → 抗生素选择（错题高频）
> - **轻度青霉素过敏（单纯皮疹）** → 头孢可用。
> - **严重过敏（anaphylaxis / SJS）** → **连头孢都避**（β-内酰胺交叉风险）→ **大环内酯（azithromycin）或克林霉素 clindamycin**。

## 八、补体通路 & 免疫球蛋白替代（Complement & Ig replacement）

- **补体缺陷感染线索**：**C5–C9（膜攻击复合物 MAC）缺 → 反复奈瑟菌 Neisseria 感染**（脑膜炎球菌 / 淋球菌）；**C1 酯酶抑制剂缺 → 遗传性血管性水肿 HAE**。
- **低补体疾病速记**（消耗补体）：MCS / SLE / 链球菌后 GN / 药物性血管炎 / 胆固醇结晶栓塞。
- **免疫球蛋白替代 IVIG（静脉注射免疫球蛋白 intravenous immunoglobulin）适应症**：抗体缺陷（**CVID / XLA** 低丙球血症）→ 反复荚膜菌感染的长期替代治疗。

## 九、HIV 与无脾特殊人群（HIV / asplenia）

> [!warning] HIV 疫苗原则——先看 CD4 计数
> | 疫苗 | 规则 |
> |---|---|
> | **肺炎球菌（最重要）** | PCV13 → PPSV23（8 周后）→ **PPSV23（5 年后复种）**；65 岁再一剂 |
> | **脑膜炎球菌** | MenACWY（四价脑膜炎球菌结合疫苗）**每 5 年加强** |
> | **HBV（乙型肝炎病毒 hepatitis B virus）** | 初始 3 剂；监测 Anti-HBs（乙肝表面抗体），**<10 mIU/mL 补种** |
> | **活疫苗（MMR / 水痘）** | **CD4 <200 时禁用** |

> [!tip] HIV 药物预防（按 CD4）
> - **CD4 <200**：预防肺孢子菌肺炎 PJP → **复方磺胺甲噁唑 TMP-SMX（甲氧苄啶-磺胺甲噁唑 trimethoprim-sulfamethoxazole）**。
> - **CD4 <100**：预防弓形虫病 Toxoplasmosis → **TMP-SMX**。
> - **CD4 <50**：MAC（鸟分枝杆菌复合群 Mycobacterium avium complex）预防——已接受 ART（抗逆转录病毒治疗 antiretroviral therapy）+ 病毒载量低则通常**不再常规预防**。

- **无脾 asplenia**：高荚膜菌（肺炎链球菌 / 脑膜炎球菌 / Hib）风险 → **三联荚膜菌疫苗**（肺炎球菌 + 脑膜炎球菌 + Hib）；肺炎球菌高危按 **8 周缩短间隔**。

## 复习自测

> [!question]- Q1. 成人 32 岁，3 年内 3 次肺炎 + 重度鼻窦炎 + 感染性血性腹泻，HIV 阴性、无家族史。最能揭示潜在病因的检查？
> **定量血清免疫球蛋白**（→ CVID，↓IgG + ↓IgA/IgM，B 细胞正常）。别被"咳嗽气短"骗去做哮喘激发试验。

> [!question]- Q2. 给药后 30 分钟出现眶周肿 + 咽紧 + 腹绞痛 + 躯干风团。机制？第一步？
> **I 型超敏（IgE / 肥大细胞活化）= anaphylaxis**；第一步 **IM 肾上腺素**。对比血清病（III 型，5–14 天）。

> [!question]- Q3. 单抗（infliximab）用药后 7–10 天发热 + 荨麻疹 + 关节痛 + 轻蛋白尿。机制？
> **III 型免疫复合物（血清病 serum sickness）**——异源蛋白触发；自限。别反射"infliximab → 结核"。

> [!question]- Q4. 男婴 9 月，6 月后反复中耳炎/肺炎，扁桃体缺如，↓↓所有 Ig + CD19 极低。诊断？怎么和 CVID 分？
> **XLA（Bruton）**；CD19 没了选 XLA，CD19 正常选 CVID。

> [!question]- Q5. 婴儿 7 月，孤立 IgG↓、疫苗应答正常、发育正常、血小板正常、有湿疹。诊断？为何不是 Wiskott？
> **THI（一过性婴儿低丙球）**，自限观察；Wiskott 必有血小板↓（小血小板）。

> [!question]- Q6. 50 岁男，HCV 史、可触性紫癜 + GN + 周围神经病、低补体 + RF 阳性 + ANCA 阴性。诊断？指纹？
> **混合型冷球蛋白血症 MCS**；三联 = 抗 HCV + RF 阳性 + 低补体。低补体看免疫复合物，正常补体看 ANCA。

> [!question]- Q7. 肾移植 6 月，动作性震颤 + 胰岛素需求↑ + 血压 152/90 + 高钾。哪个免疫抑制药？
> **钙调磷酸酶抑制剂他克莫司 tacrolimus**（多系统毒性五联）。MMF = 腹泻 + 骨髓抑制；环孢素 = 牙龈增生 + 多毛。

> [!question]- Q8. 脏伤口，从未接种破伤风疫苗。处理？干净伤口 vs 脏伤口的年限阈值？
> **TIG + 疫苗**；干净伤口 >10 年补，脏伤口 >5 年补。

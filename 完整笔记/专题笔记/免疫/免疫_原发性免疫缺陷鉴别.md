# 免疫_原发性免疫缺陷鉴别（Primary Immunodeficiency, PID）

> [!info] 用途 + 阅读路线
> 解题核心 = **从"反复感染的病原谱 + 起病年龄"反推缺陷层次（B / T / 吞噬 / 补体）**，再选对确诊检查 + 治疗。
> 路线：**§一 框架**（病原谱→层次）→ **§二 按机制·疾病全谱**（一站式，所有病按四象限归类）→ **§三 CVID 深入**（错题源）→ **§四 年龄窗** → **§五 治疗原理** → **§六 体征反查**。来源错题 [[mistakes/uworld-mistakes_2026-06#^Q4301]]。

---

## 一、解题主轴：感染模式 → 缺陷层次 ⭐

> [!danger] 第一钥匙
> **看反复感染的"病原种类"定位免疫缺陷的"层次"，比记每个病名更高效。**

| 缺陷层次 | 机制（为什么这类感染）| 反复感染特征 | 病原谱 | 代表病 | 确诊检查 |
|---|---|---|---|---|---|
| **体液（B 细胞 / 抗体）** | **抗体缺乏 → 不能调理（opsonize）有荚膜菌 + 不能中和毒素/肠道病毒** | **反复鼻窦肺部感染**（sinopulmonary）+ GI 感染 | **有荚膜菌**（肺炎链球菌/流感嗜血/沙门菌）、贾第虫、肠道病毒 | **CVID**、Bruton XLA、选择性 IgA 缺乏、高 IgM 综合征 | **定量免疫球蛋白** + 疫苗应答 |
| **T 细胞（细胞免疫）** | **细胞免疫缺陷 → 不能清除胞内 / 病毒 / 真菌**（CD8/Th 协同失效）| **机会性感染** | PJP、念珠菌/真菌、病毒（CMV）、胞内菌 | DiGeorge（22q11）、SCID、HIV（继发）| CD4 计数、流式 |
| **吞噬细胞** | **中性粒数量↓或杀菌缺陷 → 不能氧化爆发杀 catalase⁺ 菌 → 脓肿 + 肉芽肿** | 反复**皮肤/深部脓肿** + 慢性肉芽肿 | **过氧化氢酶阳性菌**（金葡菌、Serratia、曲霉、Nocardia）| 慢性肉芽肿病 CGD、LAD、Chediak-Higashi | DHR / NBT 试验 |
| **补体** | **末端补体缺 → 无膜攻击复合物 MAC → 不能溶解奈瑟菌**（其薄壁尤赖 MAC）| **反复奈瑟菌感染**（脑膜炎/淋球菌）| 奈瑟菌 Neisseria | 末端补体缺陷 C5–C9 | CH50 |

> [!tip] 一句话定位
> **荚膜菌反复呼吸道 = 抗体（B）｜机会感染 = T 细胞｜脓肿+肉芽肿 = 吞噬｜奈瑟菌反复 = 补体。**

> [!tip] 病原谱口诀（First Aid 助记）
> - **有荚膜菌**（抗体 / 补体 / 无脾宿主反复感染）——"**SHiNE my SKiS**"：**S**trep pneumoniae 肺炎链球菌、**H**aemophilus influenzae b 流感嗜血杆菌、**N**eisseria 奈瑟菌、**E**. coli 大肠杆菌、**S**almonella 沙门菌、**K**lebsiella 克雷伯菌、group **B** **S**trep B 族链球菌（+ Pseudomonas 铜绿假单胞）→ 抗体负责调理这些荚膜。
> - **catalase⁺ 菌**（吞噬 / CGD 反复）——"**Some Bacteria Produce No Serious granules**"：**S**taph aureus 金葡菌、**B**urkholderia cepacia 洋葱伯克霍尔德菌、**P**seudomonas 铜绿假单胞、**N**ocardia 诺卡菌、**S**erratia 沙雷菌（+ 真菌 Aspergillus 曲霉 / Candida 念珠菌 / Mucor 毛霉）。

> [!info] 继发性 / 特殊宿主的感染定位（同一逻辑外延）
> 同样"病原谱→缺陷"的思路也用于**继发性免疫低下**，UWorld 常混在感染题里：
> - **无脾 / 脾功能低下**（脾切除、镰状细胞病）→ **有荚膜菌暴发性感染（OPSI）**（肺炎链球菌/Hib/脑膜炎奈瑟菌）+ **巴贝虫 Babesia / 疟疾**重症 → 需荚膜菌疫苗
> - **中性粒细胞减少**（化疗后）→ **铜绿假单胞 + 肠杆菌 + 真菌（曲霉/念珠菌）**
> - **补体末端缺陷（C5–C9）**→ **反复奈瑟菌**（脑膜炎/播散性淋球菌）
> - **细胞免疫低下**（HIV/移植/长期类固醇）→ 机会感染按 CD4 谱（PJP/CMV/隐球菌）

## 二、按机制分类·PID 疾病全谱 ⭐⭐（一站式）

> [!info] 用法
> 全部 PID 按 §一 四象限归类，**同机制一张表**。每行带齐：机制 / 招牌钥匙（含记忆词）/ 确诊·实验室 / 治疗。CVID 另有 §三 深入。
> **各子表内按严重度↓排（≈ 治疗强度↓：致命/需 HSCT → IVIG 替代 → 抗菌预防 → 观察）**；唯 2.4 补体按级联位置排（早期 → 末端 → 调节蛋白）。

### 2.1 抗体 / 体液（B 细胞）缺陷

*机制共性：Ig 产生或类别转换障碍 → 反复鼻窦肺部（荚膜菌）+ GI 感染。**首选确诊 = 定量免疫球蛋白 + 疫苗接种后抗体应答**（± 流式 CD19 分 XLA vs CVID）。治疗主线 = IVIG（IgA 缺乏除外）。*

| 病（遗传）| 机制 | 起病 · 招牌钥匙 / 特征 | 确诊 / 实验室 | 治疗 |
|---|---|---|---|---|
| **XLA（Bruton 无丙球血症）**（X 连锁隐性，男婴）| **BTK 突变 → pre-B 不成熟 → 无成熟 B 细胞** | **〔起病 6 月–2 岁〕** 男婴（母源 IgG 6 月耗竭后）反复荚膜菌 + **肠道病毒（脑膜脑炎 / 疫苗株脊灰）** + **扁桃体 / 淋巴结缺如** | **↓↓全部 Ig + 流式 CD19 极低 + 生发中心缺如**；BTK 基因 | **IVIG**（禁活疫苗）|
| **高 IgM 综合征**（CD40L 缺陷，X 连锁）| **T 细胞 CD40L 缺陷 → 不能结合 B 细胞 CD40 → 类别转换失败**（只产 IgM）；根因 T、表型抗体 | **〔起病 婴儿 <1–2 岁〕** 反复鼻窦肺 + **机会感染：PJP / Cryptosporidium 隐孢子虫（慢性腹泻、胆管炎）/ CMV** | **↓IgG/IgA/IgE + IgM 正常或↑ + 生发中心缺如** | **IVIG + TMP-SMX 预防 PJP**；CD40L 型可 HSCT |
| **CVID（普通变异型免疫缺陷）** | **B 细胞分化为浆细胞障碍 → 低丙球**（散发，常无家族史）| **〔起病 成人 20–45 岁〕** 反复鼻窦肺 + 感染性（血性）腹泻 + **自免 / 非霍奇金淋巴瘤↑** | **↓IgG + ↓IgA/IgM + B 细胞数正常 + 疫苗无应答** | **IVIG**（详见 §三）|
| **选择性 IgA 缺乏**（最常见 PID，多无症状）| **B 终末分化障碍 → 不产分泌型 IgA**（黏膜防御↓）| **〔起病 任何年龄·多无症状〕** 反复黏膜感染 + **慢性贾第虫 Giardia 腹泻** + **输血 anaphylaxis（抗 IgA）** + 自免（ITP / 乳糜泻）+ **假阳性血清妊娠试验** | **↓IgA，IgG/IgM 正常**；乳糜泻 **tTG-IgA 假阴 → 改测 tTG-IgG** | **❌不给 IVIG**；对症 + 输血用**洗涤红细胞 / IgA 缺乏供者血** |
| **THI（一过性婴儿低丙球，非真缺陷）** | **婴儿自产 IgG 延迟 → 生理 nadir 延长**（B/T 正常）| **〔起病 婴 3–6 月，2–4 岁缓解〕** 婴儿**孤立 IgG↓** + **疫苗应答正常 + 血小板正常**（借此排除 XLA/CVID/Wiskott）| 定量 Ig + 疫苗特异抗体（**正常**）| **观察**（自限）|

### 2.2 T 细胞 / 联合缺陷

*机制共性：T 细胞（或 T-helper 亚群）缺陷 → 机会感染 + 常累及抗体。**首选确诊 = 淋巴细胞亚群流式（CD3/CD4 计数）+ 各病特异检查**（SCID 加新生儿 TREC；IL-12R 测 IFN-γ）。根治常需 HSCT（造血干细胞移植）。*

| 病（遗传）| 机制 | 起病 · 招牌钥匙 / 特征 | 确诊 / 实验室 | 治疗 |
|---|---|---|---|---|
| **SCID（重症联合免疫缺陷）**（IL2RG 最常见 X 连锁 / ADA AR / RAG）| **T 细胞发育失败 + B 失功能**（IL2RG / **ADA 腺苷脱氨酶** / **RAG → V(D)J 重组缺陷**）| **〔起病 婴 <6 月〕** 机会感染 + **鹅口疮 + 慢性腹泻 + FTT** + 淋巴减少 | **淋巴亚群 CD3 缺 + 生发中心缺 + 胸腺影缺**；新生儿 **TREC 筛查** | **HSCT 根治**；ADA 型酶替代 / 基因治疗；三禁（活疫苗 / 非辐照血 / 母乳）+ TMP-SMX |
| **Wiskott-Aldrich 综合征 WAS**（X 连锁隐性，男婴）| **WASp 突变 → 肌动蛋白细胞骨架重排缺陷 → T 细胞 + 血小板障碍** | **〔起病 婴儿期〕** **WATER**：湿疹（Eczema）+ **血小板↓（小血小板）+ 出血点** + 反复感染；自免 / 淋巴瘤↑ | **↓IgM、↑IgA/IgE、IgG 正常 + 小而少血小板**；基因 | **HSCT**；IVIG 支持 |
| **DiGeorge（胸腺发育不全，22q11 微缺失）** | **第 3/4 咽囊发育失败 → 无胸腺 + 无甲状旁腺** | **〔起病 新生儿〕** **CATCH-22**：圆锥动脉干心畸形（TOF 法洛四联症 / truncus 永存动脉干）+ **低钙抽搐（PTH↓）** + 腭裂面容 + 无胸腺 → T 细胞↓机会感染 | **↓T 细胞 + ↓PTH + ↓Ca²⁺ + 胸腺影胸片（CXR）缺如** | 补钙；重型 → **胸腺移植（非 HSCT）**；禁活疫苗 |
| **共济失调-毛细血管扩张 Ataxia-telangiectasia**（AR，ATM）| **ATM 激酶缺陷 → DNA 双链断裂修复障碍**（放射敏感 + 淋巴瘤↑）| **〔起病 幼儿 1–2 岁（会走后）〕** **小脑共济失调 + 眼/皮肤毛细血管扩张 + 反复鼻窦肺 + 淋巴瘤** | **↓IgA/IgG/IgE + AFP↑ + 淋巴细胞减少** | 支持 + **避免辐射**；抗感染 |
| **Hyper-IgE / Job 综合征**（AD，STAT3）| **STAT3 突变 → Th17 分化↓ → 中性粒趋化障碍**（根因 T-helper，表型呈吞噬募集障碍 = "冷"脓肿）| **〔起病 婴 / 儿童早期〕** **FATED**：粗面容 + **冷脓肿（无炎症征，金葡）** + 保留乳牙 + IgE↑↑ + 湿疹 + 肺大疱 + **轻伤骨折** | **血清 IgE↑↑ + 嗜酸↑** | **抗葡菌预防 TMP-SMX** ± 抗真菌 |
| **IL-12 受体缺陷（IL-12 receptor deficiency）**（AR）| **IL-12 受体缺陷 → Th1↓ → IFN-γ↓ → 巨噬细胞活化不足** | **〔起病 儿童（常 BCG 后）〕** **播散性分枝杆菌（结核/非结核）+ 真菌**；**卡介苗 BCG 接种后发病** | **↓IFN-γ**；**MSMD（孟德尔遗传性分枝杆菌易感症）最常见病因** | 抗分枝杆菌 + **IFN-γ（干扰素-γ）** |
| **慢性皮肤黏膜念珠菌病（Chronic mucocutaneous candidiasis, CMC）**（AIRE 等）| **T 细胞对念珠菌的特异性细胞免疫缺陷**（仅对 Candida 选择性无反应）| **〔起病 儿童期〕** **持续非侵袭性皮肤 / 指甲 / 黏膜念珠菌**（不入血、不播散）| **体外 T 对念珠菌抗原无增殖 + 皮肤迟发试验无反应**；AIRE → **APECED / APS-1** | 抗真菌（唑类 azoles）|

> [!danger] ⭐ Wiskott-Aldrich（WAS）专题深入 — 自测两错点专攻
> #考前必看
> **一句话**：X 连锁隐性（**仅男婴**）· **WASp 蛋白突变 → 肌动蛋白（actin）细胞骨架重排缺陷 → T 细胞 + 血小板双功能障碍**。属**联合免疫缺陷**（既体液又细胞）。
>
> **① 三联征 "WATER"**：**W**AS = 血小板减少（**T**hrombocytopenia，**小血小板** microthrombocytopenia）+ 湿疹（**E**czema）+ 反复感染（**R**ecurrent，荚膜菌鼻窦肺/中耳炎）。**独家钥匙 = 小而少的血小板 + 出血/瘀点**——别的湿疹病血小板都正常。
> **⭐ 出血常为首发（高频钩子）**：WAS 常以**出血**而非感染最先就诊——**新生儿期**即可表现为**血性腹泻 / 包皮环切后出血不止 / 瘀点瘀斑**，**比反复感染出现更早**（感染要等母源抗体耗竭后才显）。stem 给"男婴 + 出生不久出血 + 湿疹"别只想凝血病，查血小板（小）。
>
> **② Ig 指纹（为什么 IgM↓）**：**IgM↓**、IgA / IgE↑、IgG 正常或↓。机制 = WASp 缺陷砸的是**对 T 非依赖性 II 型抗原（细菌荚膜多糖 polysaccharide）的应答**，而多糖抗原**不经类别转换、主力产物就是 IgM** → 抗多糖塌 = **IgM 首当其冲↓**。可考标志：**同种血凝素（isohemagglutinins，抗 A/B 血型多糖的 IgM 抗体）↓或缺如**。临床 = 对有荚膜菌调理差 → 反复鼻窦肺/中耳炎。（IgE↑ 配特应/湿疹，IgA↑ 配慢性黏膜刺激；对照多数抗体缺陷以 IgG↓ 为主，WAS 的"IgM↓为主"是指纹。）
>
> **③ 湿疹三病别再撞车**（错点 1 = Q3 误判为 WAS）：先看血小板，再看伴随征。
> ```
> 湿疹 + 反复感染（男婴）
>   ├─ 小血小板↓ + 出血/瘀点 ──────────────→ Wiskott-Aldrich（IgM↓）
>   ├─ 血小板正常 + 冷脓肿 + 保留乳牙 + IgE↑↑ → Hyper-IgE / Job
>   └─ + 小脑共济失调 + 毛细血管扩张 + AFP↑ ─→ Ataxia-telangiectasia
> ```
> > 反推：Q3 stem 有"冷脓肿+乳牙+IgE↑↑"且**无血小板↓** = Job，不是 WAS。
>
> **④ 治疗 = HSCT（造血干细胞移植）根治，不是 IVIG**（错点 2 = Q5）：WAS 缺陷在 **WASp → T 细胞 + 血小板的细胞骨架**，是**细胞功能本身**的缺陷 → 必须**换细胞**。**IVIG 只是支持**（补部分抗体，修不了血小板/T 细胞）；脾切除可升血小板但不根治、反增感染风险。对照 §五"能不能补成品 → 缺细胞就换细胞"。
>
> **⑤ 并发症 / 死因**：**出血**（颅内/消化道）、反复感染、**自身免疫**（溶贫/血管炎/IgA 肾病）、**淋巴瘤（EBV，Epstein-Barr 病毒 / EB 病毒 相关）风险↑**。
>
> **⑥ 干扰项陷阱**（见 [[NBME11_错题本#^Q038]]）：泛化选项 **"Combined immunodeficiency"** 会诱你——WAS 确属联合缺陷，但题要**最具体诊断**；见"小血小板+湿疹+出血"就锁 **WAS**。

### 2.3 吞噬细胞缺陷

*机制共性：中性粒数量 / 黏附 / 杀菌缺陷 → 脓肿 + 特殊菌谱。**首选确诊按病分**：CGD → DHR123 流式 / NBT；LAD-1 → 流式 CD18（CD11/CD18 整合素）；Chediak → 外周血涂片看巨大颗粒。治疗以抗菌预防为主，重症 HSCT。*

| 病（遗传）| 机制 | 起病 · 招牌钥匙 / 特征 | 确诊 / 实验室 | 治疗 |
|---|---|---|---|---|
| **Chediak-Higashi（契-东综合征）**（AR）| **LYST 突变 → 溶酶体转运 / 吞噬体-溶酶体融合缺陷**（不能杀菌）| **〔起病 婴幼儿〕** **PLAIN**：进行性神经退化 + 淋巴组织细胞增生（HLH）+ **部分白化** + 反复化脓菌 + 周围神经病 | **中性粒 / 血小板巨大溶酶体颗粒 + 全血细胞减少 + 轻度凝血缺陷** | 抗感染 + **HSCT**；HLH 按方案 |
| **白细胞黏附缺陷 I 型（Leukocyte adhesion deficiency type 1, LAD-1）**（AR）| **LFA-1 整合素 β2 亚基（CD18）缺陷 → 中性粒不能黏附 / 迁移 / 趋化**（到不了感染灶）| **〔起病 新生儿〕** **脐带延迟脱落（>30 天）+ 反复皮肤黏膜细菌 + 伤口不愈 + 脓肿"无脓"** | **外周血中性粒↑↑（但感染灶内缺如）** | 抗感染；重症 **HSCT** |
| **CGD（慢性肉芽肿病）**（X 连锁最常见 / AR）| **NADPH 氧化酶缺陷 → 无氧化爆发 → 不能杀 catalase⁺ 菌** | **〔起病 婴–5 岁〕** **catalase⁺ 菌**（金葡 / Serratia / Burkholderia / Nocardia）/ **真菌（曲霉）空洞肺炎 + 脓肿 + 肉芽肿** | **DHR123 流式（↓绿色荧光）/ NBT（阴性，过时）** | 终身 **TMP-SMX + 伊曲康唑 itraconazole + IFN-γ**；重症 HSCT |

### 2.4 补体缺陷

*机制共性：**首选确诊 / 共同筛查 = CH50（总补体活性，缺任一成分即↓）**，再按表型分早期（C4/C2↓）vs 末端（C1–C4 正常）。两端表型截然不同。*

| 病 | 机制 | 起病 · 招牌钥匙 / 特征 | 确诊 / 实验室 | 治疗 |
|---|---|---|---|---|
| **早期补体缺乏（C1q / C2 / C4）** | **免疫复合物 + 凋亡细胞清除障碍** | **〔起病 儿童期〕** **早发 SLE（尤童年，C1q 最强 SLE 关联）** + 反复细菌（荚膜菌）| **CH50↓ + C4/C2↓** | 治 SLE + 荚膜菌疫苗 |
| **末端补体缺乏（C5–C9）** | **无膜攻击复合物 MAC → 不能溶解奈瑟菌** | **〔起病 青少年–青年〕** **反复奈瑟菌**（脑膜炎 / 播散性淋球）| **CH50↓，C1–C4 正常** | **脑膜炎奈瑟菌 + 肺炎链球菌 + Hib 疫苗** + 抗生素 |
| **C1 酯酶抑制物（C1-INH）缺乏 = 遗传性血管性水肿 HAE**（调节蛋白缺陷，非感染表型）| **C1-INH 缺 → 缓激肽 bradykinin↑** | **〔起病 青春期起·渐重〕** **非痒性血管性水肿 + 阵发腹痛 + 喉头水肿**；**禁 ACEI** | **C4↓（发作间期也低）+ C1-INH 水平/功能↓** | 急性：**icatibant（艾替班特，缓激肽拮抗）/ C1-INH 浓缩剂**；预防：**danazol（达那唑，雄激素）** |

> [!warning] IgA 缺乏的超敏交叉点
> 选择性 IgA 缺乏患者体内可产生**抗 IgA 抗体** → 输含 IgA 血制品时发生**过敏 / anaphylaxis（I 型超敏）** → 需用洗涤红细胞 / IgA 缺乏供者血。这是"免疫缺陷 × 超敏反应"的交叉考点。

> [!success] 一句话配对反射（确诊导向）
> **↓↓全 Ig + CD19 缺 → XLA｜↓Ig + CD19 正常 + 疫苗无应答 → CVID｜CD3 缺 + FTT + TREC↓ → SCID｜低钙 + 心畸形 + 胸腺影缺 → DiGeorge｜catalase⁺ 菌 + DHR → CGD｜反复奈瑟菌 + CH50↓ → 末端补体｜湿疹 + 小血小板 → Wiskott｜婴儿孤立 IgG↓ + 疫苗应答正常 + 血小板正常 → THI｜湿疹 + 冷脓肿 + 乳牙 + IgE↑↑ → Job｜早发 SLE + CH50↓ → C1q（早期补体）。**

## 三、CVID 深入（成人最常见有症状 PID，原错题源）⭐

> [!danger] 招牌钥匙
> **「成人（20–45 岁）反复肺炎/鼻窦炎 + 感染性（血性）腹泻 + 无家族史 → 查定量免疫球蛋白（↓↓IgG）= CVID」**

| 项目 | 内容 |
|---|---|
| **机制** | **B 细胞分化为浆细胞障碍 → 低丙种球蛋白血症 hypogammaglobulinemia**；多为散发突变，**常无家族史** |
| **起病年龄** | 典型 **20–45 岁确诊**（可早至青春期），区别于儿童期发病的 Bruton/SCID |
| **感染谱** | 反复**呼吸道**（肺炎/鼻窦炎/中耳炎）+ **GI**（沙门菌/弯曲菌/贾第虫）；抗生素有效但清除慢 |
| **非感染表现** | **自身免疫**（溶血性贫血/RA/恶性贫血）、类 IBD 肠病、肉芽肿、**非霍奇金淋巴瘤风险↑** |
| **诊断** | **定量血清免疫球蛋白：↓↓IgG + ↓IgA 和/或 IgM** + **疫苗接种后抗体无应答**（B 细胞数正常）|
| **治疗** | **免疫球蛋白替代 IVIG** |

> [!warning] CVID stem 常见干扰（别被单个急性主诉带偏）
> CVID stem 常以"持续咳嗽气短"等单一急性主诉开头，真正主线藏在**反复感染史**里（多次肺炎 + 鼻窦炎 + 感染性腹泻）。

| 干扰检查/诊断 | 指向什么 | 为何不是本病 |
|---|---|---|
| **CD4 计数** | DiGeorge / HIV | 这两者引起**机会性感染**；DiGeorge 生后即现；CVID 患者 HIV 阴性、感染谱是细菌 |
| **α1-抗胰蛋白酶** | α1-AT 缺乏（肺气肿+肝病）| **不增加感染风险** |
| **美沙胆碱激发** | 哮喘 | 哮喘**不引起反复细菌性肺炎/鼻窦炎/血性腹泻** |
| **汗液氯试验** | 囊性纤维化 CF | CF 多**儿童期**确诊；血性腹泻不典型（CF 为脂肪泻/胰腺功能不全）|

## 四、年龄速记（PID 起病窗）

```
出生~6月内起病 → T 细胞/联合缺陷（SCID、DiGeorge）
  （母源 IgG 6 月后耗竭）
6月~2岁起病   → 抗体缺陷（Bruton XLA，男孩）
20~45岁成人   → CVID（成人最常见有症状 PID）
```

> [!tip] 为什么 Bruton 在 6 月后才发病
> 出生头 6 个月有**母源 IgG**保护；耗竭后抗体缺陷才暴露 → 反复化脓菌感染。CVID 则因发病晚 + 表现多样 + 诊断延迟，常到成年才确诊。

> [!warning] THI（一过性婴儿低丙球）的两把鉴别钥匙 ⭐
> 婴儿**孤立 IgG↓**别急着诊断 XLA/CVID/Wiskott：
> - **疫苗应答正常**（破伤风/白喉特异 IgG 升高）→ 排除 CVID/XLA（二者无应答）→ 指向 **THI**
> - **血小板正常** → 排除 Wiskott（其必有血小板↓+小血小板）
> THI = 生理性 IgG nadir 延长 >6 月，B/T 正常，常伴 atopy（湿疹/食物过敏），**自限**（儿童早期正常化），**观察为主**。见错题 [[mistakes/uworld-mistakes_2026-06#^Q19628]]。

## 五、治疗原理：缺什么补什么 ⭐（横向对比）

> [!danger] 治疗第一钥匙
> **治疗逻辑跟着缺陷层次走：B → 补抗体（IVIG）｜T / 联合 → 换免疫系统（HSCT 造血干细胞移植）｜吞噬 → 防感染（抗菌 / 抗真菌预防 ± IFN-γ）｜补体 → 打疫苗（奈瑟菌为主）。**

> [!tip] 一条原理贯穿：「能不能补成品」决定策略 ⭐
> 缺的是**可溶蛋白终产物**（抗体 / 酶）→ 能外源补就**替代**；缺的是**细胞功能本身**（T 细胞 / 造血）→ 无法瓶装只能**换细胞**；既难补又非致命 → 转**防御**。

```
PID 治疗策略：先问「缺的是什么」
│
├─ 缺【可溶蛋白终产物】(抗体/酶) ── 能外源补 ──→ 替代
│    ├─ 缺 IgG (XLA / CVID / 高IgM) ────────→ IVIG (每 3-4 周, 卡 IgG 半衰期)
│    │    └─ ✗ 选择性 IgA 缺乏「不补」: 无效(补不进分泌型IgA) + 抗IgA过敏
│    └─ 缺 ADA 酶 (ADA-SCID) ──────────────→ PEG-ADA(聚乙二醇化腺苷脱氨酶) / 基因治疗
│
├─ 缺【细胞功能本身】不能瓶装 ── 只能换细胞 ──→ HSCT 造血干细胞移植
│    ├─ 缺陷源在造血干细胞 (SCID / WAS / 重症CGD·LAD) → HSCT
│    └─ 缺陷在胸腺基质 (DiGeorge 重型, 造血干细胞正常) → 胸腺移植(非HSCT)
│
├─ 缺【吞噬功能】难补又非致命 ── 降感染负荷 ──→ 抗菌 + 抗真菌预防
│    └─ CGD 额外 IFN-γ(干扰素-γ) 增强残余杀菌; 重症 → HSCT
│
└─ 缺【补体蛋白】半衰期短不可替代 ── 转防御 ──→ 疫苗(奈瑟菌/荚膜菌) + 抗生素
     └─ 早期补体(C1q)缺 = SLE 表型 → 治 SLE(非感染)
```

| 缺陷层次 | 治疗主线 | 具体方案（代表病）| 关键例外 / 急症处理 |
|---|---|---|---|
| **体液（B / 抗体）** | **免疫球蛋白替代** | **IVIG（静脉免疫球蛋白）/ SCIG（皮下免疫球蛋白）每 3–4 周**（XLA、CVID、高 IgM）| ⭐ **选择性 IgA 缺乏 ❌不给 IVIG**（多数无症状 + 制品含微量 IgA → 可诱发抗 IgA 过敏）；需输血时用**洗涤红细胞 / IgA 缺乏供者血** |
| **T 细胞 / 联合** | **重建免疫 = HSCT（造血干细胞移植）** | **SCID / 重症联合 → HSCT 根治**；**ADA-SCID（腺苷脱氨酶 adenosine deaminase 缺陷型）** 可酶替代 / 基因治疗；**DiGeorge** 重型 → 胸腺移植 + 补钙（低钙抽搐）| ⭐ **SCID 三禁**：❌活疫苗（BCG / 口服脊灰 / MMR / 水痘 / 轮状）❌非辐照血 ❌生母母乳（CMV）；+ **TMP-SMX（复方磺胺甲噁唑）预防 PJP（耶氏肺孢子菌肺炎 Pneumocystis jirovecii pneumonia）**；新生儿 **TREC（T 细胞受体切除环 T-cell receptor excision circles）筛查**抢救窗 |
| **吞噬细胞** | **终身抗菌 + 抗真菌预防** | **TMP-SMX + 伊曲康唑 itraconazole（抗曲霉）**；**CGD 加 IFN-γ（干扰素-γ）**；重症 → HSCT | 脓肿需**切开引流**；靶向 **catalase⁺ 菌**；LAD / 重症 CGD 根治也靠移植 |
| **补体** | **疫苗 + 必要时抗生素** | **脑膜炎奈瑟菌 + 肺炎链球菌 + Hib 疫苗**；爆发性奈瑟菌感染早用抗生素 | ⭐ **早期补体缺（C1q）→ 治 SLE**（非感染表型）；用 **eculizumab（依库珠单抗，C5 抑制剂）者必先打脑膜炎疫苗** |

> [!warning] IVIG「给谁 / 不给谁」高频对比
> - **给**：XLA、CVID、高 IgM 综合征（抗体产生 / 类别转换缺陷 → 替代缺失的 IgG）
> - **不给**：**选择性 IgA 缺乏**（绝大多数无症状，且制品含微量 IgA 反而触发抗 IgA 抗体 → anaphylaxis）；THI（自限，观察即可）
> - 记法：**IVIG 补的是 IgG**；IgA 缺乏缺的是 IgA、补 IVIG 无益反有害。

> [!warning] 活疫苗禁忌的 PID（高频安全考点）⭐
> **抗体缺陷（XLA / CVID）+ T 细胞 / 联合缺陷（SCID / 高 IgM / DiGeorge 重型）→ 禁所有活疫苗**：口服脊灰 OPV、MMR、水痘、轮状、卡介苗 BCG、口服伤寒。
> - **XLA**：尤禁 **OPV / 肠道病毒**（无中和抗体 → 疫苗株 / 野生肠道病毒可致**慢性脑膜脑炎**）
> - **SCID / IL-12 受体缺陷**：**BCG 接种后播散性分枝杆菌感染** = 揭示潜在缺陷的钩子
> - 原理：活疫苗 = 减毒活病原，需**完整抗体（中和）+ T 细胞（清除胞内）**才安全；缺哪层就被那层本该挡住的活病原反噬。

## 六、按【临床体征 / 实验室指纹】反查 ⭐⭐（看到 X → 想到谁）

> [!danger] 逆向钥匙
> §一·§二 是"病 → 表现"，本节是**反方向**：stem 抛一个招牌体征 / 实验室模式，直接锁病 + 给近邻区分钥匙。

### 6.1 招牌体征 → 锁定病

| stem 招牌体征 | 第一反应（病）| 与近邻区分钥匙 |
|---|---|---|
| **湿疹 + 血小板↓（小血小板）+ 出血点**（男婴）| **Wiskott-Aldrich** | 看血小板：WAS **小而少**；Job 血小板正常 |
| **湿疹 + 冷脓肿 + 保留乳牙 + 粗面容 + IgE↑↑** | **Hyper-IgE / Job** | 冷脓肿（无红肿热）+ 面容 + 乳牙 + IgE↑↑ |
| **反复"冷"脓肿**（cold abscess，无炎症征）| **Hyper-IgE / Job** | 对比 CGD = **热**脓肿 + 肉芽肿 |
| **反复化脓脓肿 + 肉芽肿 + catalase⁺ 菌** | **CGD（慢性肉芽肿病）** | **DHR / NBT 阳性**确诊 |
| **脐带延迟脱落（>30d）+ 伤口不愈 + 无脓 + WBC↑↑** | **LAD（白细胞黏附缺陷）** | **"无脓"**是钥匙（中性粒到不了感染灶）|
| **部分白化（银发）+ 反复化脓 + 周围神经病** | **Chediak-Higashi** | 中性粒**巨大溶酶体颗粒** |
| **眼 / 皮肤毛细血管扩张 + 小脑共济失调 + AFP↑** | **Ataxia-telangiectasia（共济失调-毛细血管扩张症）** | 放射敏感 + 淋巴瘤风险↑ |
| **低钙抽搐 + 圆锥动脉干心畸形（TOF / truncus）+ 腭裂 + 无胸腺** | **DiGeorge（迪格奥尔格，22q11）** | "**CATCH-22**"；**胸腺影胸片（CXR）缺如 + PTH↓ + T 细胞↓** |
| **输血 / 血制品过敏 anaphylaxis** | **选择性 IgA 缺乏** | 抗 IgA 抗体；常伴慢性贾第虫 |
| **扁桃体 / 淋巴结 / 腺样体缺如**（男婴）| **XLA（Bruton）** | 无 B → 无生发中心；**CD19 极低** |
| **早发 SLE（尤童年）+ 反复细菌** | **早期补体缺（C1q > C4 > C2）** | **CH50↓ + C4 / C2↓** |
| **反复奈瑟菌**（脑膜炎 / 播散性淋球）| **末端补体缺 C5–C9** | CH50↓，C1–C4 正常 |
| **非痒性血管性水肿 + 阵发腹痛 + 喉头水肿** | **遗传性血管性水肿 HAE（C1-INH 缺）** | C4↓ + 禁 ACEI；非感染表型 |
| **慢性水样 / 脂肪泻 + 贾第虫 Giardia** | **IgA 缺乏 / CVID / XLA** | 抗体缺陷共性（黏膜 + 调理障碍）|
| **成人反复肺 / 鼻窦 + 自免 + 淋巴瘤↑** | **CVID** | **Ig↓ + B 细胞数正常 + 疫苗无应答** |

### 6.2 免疫球蛋白指纹 → 病（实验室反查）

| Ig / 实验室模式 | 锁定病 |
|---|---|
| **全部 Ig↓↓ + B 细胞（CD19）极低 / 缺** | **XLA（Bruton）** |
| **全部 Ig↓ + B 细胞数正常 + 疫苗无应答** | **CVID** |
| **IgM 正常 / ↑，IgG / IgA / IgE↓** | **高 IgM 综合征（CD40L）** |
| **孤立 IgA↓，IgG / IgM 正常** | **选择性 IgA 缺乏** |
| **IgM↓ + IgA / IgE↑ + IgG 正常** | **Wiskott-Aldrich** |
| **IgE↑↑ + 嗜酸↑** | **Hyper-IgE / Job** |
| **IgA / IgG↓ + AFP↑** | **Ataxia-telangiectasia** |
| **婴儿孤立 IgG↓ + 疫苗应答正常 + 血小板正常** | **THI（一过性婴儿低丙球）** |

> [!warning] 三大高频"撞车"体征鉴别（看到就要分叉）
> 湿疹 / 脓肿 / 血小板↓ 是最爱混的三组，按下树分叉：
> ```
> 湿疹 + 反复感染（男婴）：
>   ├─ 血小板↓（小血小板）+ 出血点 ──→ Wiskott-Aldrich（IgM↓, IgA/IgE↑）
>   └─ 血小板正常 + 冷脓肿 + 保留乳牙 + 粗面容 ──→ Hyper-IgE/Job（IgE↑↑）
>
> 反复脓肿：
>   ├─ "冷"脓肿（无红肿热）+ IgE↑↑ ──────────→ Hyper-IgE/Job
>   ├─ 热脓肿 + 肉芽肿 + catalase⁺菌 + DHR阳 ──→ CGD
>   └─ 无脓 + 脐带延迟脱落 + WBC↑↑ ──────────→ LAD（白细胞黏附缺陷）
> ```

> [!success] 逆向反射一句话
> **小血小板+湿疹→Wiskott｜乳牙+冷脓肿+IgE↑↑→Job｜无脓+脐带晚脱→LAD｜银发+神经病→Chediak｜毛细血管扩张+共济失调→A-T｜低钙+心畸形+腭裂→DiGeorge｜输血过敏→IgA缺乏｜扁桃体缺如→XLA｜早发SLE→早期补体｜反复奈瑟菌→末端补体｜非痒水肿+腹痛→HAE。**

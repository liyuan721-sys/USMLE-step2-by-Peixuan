# 免疫_原发性免疫缺陷鉴别（Primary Immunodeficiency, PID）

> [!info] 用途
> 原发性免疫缺陷的解题核心 = **从"反复感染的病原谱 + 起病年龄"反推缺陷层次（B / T / 吞噬 / 补体）**，再选对确诊检查。本笔记承载鉴别骨架 + **CVID** 重点，来源错题 [[mistakes/uworld-mistakes_2026-06#^Q4301]]。

---

## 一、解题主轴：感染模式 → 缺陷层次 ⭐

> [!danger] 第一钥匙
> **看反复感染的"病原种类"定位免疫缺陷的"层次"，比记每个病名更高效。**

| 缺陷层次 | 机制（为什么这类感染）| 反复感染特征 | 病原谱 | 代表病 | 确诊检查 |
|---|---|---|---|---|---|
| **体液（B 细胞 / 抗体）** | **抗体缺乏 → 不能调理（opsonize）有荚膜菌 + 不能中和毒素/肠道病毒** | **反复鼻窦肺部感染**（sinopulmonary）+ GI 感染 | **有荚膜菌**（肺炎链球菌/流感嗜血/沙门菌）、贾第虫、肠道病毒 | **CVID**、Bruton XLA、选择性 IgA 缺乏、高 IgM 综合征 | **定量免疫球蛋白** + 疫苗应答 |
| **T 细胞（细胞免疫）** | **细胞免疫缺陷 → 不能清除胞内 / 病毒 / 真菌**（CD8/Th 协同失效）| **机会性感染** | PJP、念珠菌/真菌、病毒（CMV）、胞内菌 | DiGeorge（22q11）、SCID、HIV（继发）| CD4 计数、流式 |
| **吞噬细胞** | **中性粒数量↓或杀菌缺陷 → 不能氧化爆发杀 catalase⁺ 菌 → 脓肿 + 肉芽肿** | 反复**皮肤/深部脓肿** + 慢性肉芽肿 | **过氧化氢酶阳性菌**（金葡菌、Serratia、曲霉、Nocardia）| 慢性肉芽肿病 CGD、LAD、Chediak-Higashi | DHR / NBT 试验 |
| **补体** | **末端补体缺 → 无膜攻击复合物 MAC → 不能溶解奈瑟菌**（其薄壁尤赖 MAC）| **反复奈瑟菌感染**（脑膜炎/淋球菌）| 奈瑟菌 Neisseria | 末端补体缺陷 C5–C9 | CH50 |

> [!tip] 一句话定位
> **荚膜菌反复呼吸道 = 抗体（B）｜机会感染 = T 细胞｜脓肿+肉芽肿 = 吞噬｜奈瑟菌反复 = 补体。**

> [!info] 继发性 / 特殊宿主的感染定位（同一逻辑外延）
> 同样"病原谱→缺陷"的思路也用于**继发性免疫低下**，UWorld 常混在感染题里：
> - **无脾 / 脾功能低下**（脾切除、镰状细胞病）→ **有荚膜菌暴发性感染（OPSI）**（肺炎链球菌/Hib/脑膜炎奈瑟菌）+ **巴贝虫 Babesia / 疟疾**重症 → 需荚膜菌疫苗
> - **中性粒细胞减少**（化疗后）→ **铜绿假单胞 + 肠杆菌 + 真菌（曲霉/念珠菌）**
> - **补体末端缺陷（C5–C9）**→ **反复奈瑟菌**（脑膜炎/播散性淋球菌）
> - **细胞免疫低下**（HIV/移植/长期类固醇）→ 机会感染按 CD4 谱（PJP/CMV/隐球菌）

> [!warning] 补体缺陷分两端（高频鉴别）⭐
> - **早期补体 C1q / C1–C4 缺乏** → 免疫复合物 + 凋亡细胞清除障碍 → **>90% 早发 SLE**（CH50↓ + C4/C2↓）；**C1q 是最强 SLE 关联**
> - **末端补体 C5–C9 缺乏** → **反复奈瑟菌**（脑膜炎/播散性淋球菌），C1–C4 正常
> - 共同筛查 = **CH50**（总补体活性，缺任一成分即↓）
> - 注：**C1 酯酶抑制物 C1-INH 缺乏 = 遗传性血管性水肿 HAE**（缓激肽介导、C4↓、非痒性水肿 + 阵发腹痛 + 喉头水肿，**禁 ACEI**）——属补体**调节蛋白**缺陷，非感染表型，单独考。

## 二、CVID 重点（成人最常见有症状 PID）⭐

> [!danger] 招牌钥匙
> **「成人（20–45 岁）反复肺炎/鼻窦炎 + 感染性（血性）腹泻 + 无家族史 → 查定量免疫球蛋白（↓↓IgG）= CVID」**

| 项目 | 内容 |
|---|---|
| **机制** | **B 细胞分化为浆细胞障碍 → 低丙种球蛋白血症 hypogammaglobulinemia**；多为散发突变，**常无家族史** |
| **起病年龄** | 典型 **20–45 岁确诊**（可早至青春期），区别于儿童期发病的 Bruton/SCID |
| **感染谱** | 反复**呼吸道**（肺炎/鼻窦炎/中耳炎）+ **GI**（沙门菌/弯曲菌/贾第虫）；抗生素有效但清除慢 |
| **非感染表现** | **自身免疫**（溶血性贫血/RA/恶性贫血）、类 IBD 肠病、肉芽肿、**非霍奇金淋巴瘤风险↑** |
| **诊断** | **定量血清免疫球蛋白：↓↓IgG + ↓IgA 和/或 IgM** + **疫苗接种后抗体无应答** |
| **治疗** | **免疫球蛋白替代 IVIG** |

## 三、易混鉴别（CVID stem 常见干扰）

> [!warning] 别被单个急性主诉带偏
> CVID stem 常以"持续咳嗽气短"等单一急性主诉开头，真正的诊断主线藏在**反复感染史**里（多次肺炎 + 鼻窦炎 + 感染性腹泻）。

| 干扰检查/诊断 | 指向什么 | 为何不是本病 |
|---|---|---|
| **CD4 计数** | DiGeorge / HIV | 这两者引起**机会性感染**；DiGeorge 生后即现；CVID 患者 HIV 阴性、感染谱是细菌 |
| **α1-抗胰蛋白酶** | α1-AT 缺乏（肺气肿+肝病）| **不增加感染风险** |
| **美沙胆碱激发** | 哮喘 | 哮喘**不引起反复细菌性肺炎/鼻窦炎/血性腹泻** |
| **汗液氯试验** | 囊性纤维化 CF | CF 多**儿童期**确诊；血性腹泻不典型（CF 为脂肪泻/胰腺功能不全）|

## 四、年龄速记（PID 起病窗）

```
出生~6月内起病 → T 细胞/联合缺陷（SCID、DiGeorge）
  （母源 IgG 6 月后耗竭）
6月~2岁起病   → 抗体缺陷（Bruton XLA，男孩）
20~45岁成人   → CVID（成人最常见有症状 PID）
```

> [!tip] 为什么 Bruton 在 6 月后才发病
> 出生头 6 个月有**母源 IgG**保护；耗竭后抗体缺陷才暴露 → 反复化脓菌感染。CVID 则因发病晚 + 表现多样 + 诊断延迟，常到成年才确诊。

> [!warning] THI（一过性婴儿低丙球）的两把鉴别钥匙 ⭐
> 婴儿**孤立 IgG↓**别急着诊断 XLA/CVID/Wiskott：
> - **疫苗应答正常**（破伤风/白喉特异 IgG 升高）→ 排除 CVID/XLA（二者无应答）→ 指向 **THI**
> - **血小板正常** → 排除 Wiskott（其必有血小板↓+小血小板）
> THI = 生理性 IgG nadir 延长 >6 月，B/T 正常，常伴 atopy（湿疹/食物过敏），**自限**（儿童早期正常化），**观察为主**。见错题 [[mistakes/uworld-mistakes_2026-06#^Q19628]]。

## 五、其他重点 PID 速览（补全谱系）

| 病 | 遗传 | 发病机制 | 钥匙三联 / 特征 | 实验室 | 记忆点 |
|---|---|---|---|---|---|
| **Wiskott-Aldrich 综合征 WAS** | X 连锁隐性（男婴）| **WASp 蛋白突变 → 肌动蛋白细胞骨架重排缺陷 → T 细胞 + 血小板功能障碍**（血小板小而少）| **湿疹 + 血小板减少（小血小板）+ 反复感染** | **↓IgM**、↑IgA/IgE、IgG 正常 | "**WATER**"：WAS + 血小板↓ + 湿疹（Eczema）+ 反复感染 |
| **选择性 IgA 缺乏** | 最常见 PID（多数无症状）| **B 细胞终末分化障碍 → 不能产生分泌型 IgA**（黏膜免疫防御↓）| 反复黏膜感染（鼻窦/肺/GI）+ **慢性贾第虫 Giardia 腹泻** + **输血过敏反应（抗 IgA → 可致 anaphylaxis）** + 自身免疫（**ITP** / 乳糜泻↑）| **↓IgA**、IgG/IgM 正常 | 乳糜泻 **tTG-IgA 假阴性 → 改测 tTG-IgG** |
| **高 IgM 综合征（CD40L 缺陷）** | X 连锁 | **T 细胞 CD40L 缺陷 → 无法结合 B 细胞 CD40 → 免疫球蛋白类别转换失败**（只产 IgM）| 反复鼻窦肺部 + **PJP**（类 T 细胞表现）| **↓IgG/IgA/IgE + IgM 正常或↑** | 不能 class-switch |
| **Chediak-Higashi** | AR | **LYST 基因突变 → 溶酶体转运 / 吞噬体-溶酶体融合缺陷**（不能杀菌）| **部分白化 + 反复化脓菌感染 + 周围神经病** | 中性粒**巨大溶酶体颗粒** | 溶酶体转运缺陷 |
| **Hyper-IgE / Job 综合征** | AD（STAT3）| **STAT3 突变 → Th17 分化↓ → 中性粒趋化障碍**（脓肿无炎症征 = "冷"脓肿）| **湿疹 + 反复"冷"脓肿（cold abscess，金葡）+ 肺炎伴肺大疱 + 保留乳牙 + 特殊面容** | **血清 IgE↑↑** + 嗜酸↑ | "**FATED**"：面容/脓肿/乳牙/IgE↑/皮炎 |
| **共济失调-毛细血管扩张 Ataxia-telangiectasia** | AR（ATM）| **ATM 激酶缺陷 → DNA 双链断裂修复障碍**（放射敏感 + 淋巴瘤↑）| **小脑共济失调 + 眼/皮肤毛细血管扩张 + 反复鼻窦肺感染 + 淋巴瘤风险↑** | **↓IgA/IgG + AFP↑** | DNA 修复缺陷、放射敏感 |

> [!warning] IgA 缺乏的超敏交叉点
> 选择性 IgA 缺乏患者体内可产生**抗 IgA 抗体** → 输含 IgA 血制品时发生**过敏/anaphylaxis（I 型）** → 需用洗涤红细胞 / IgA 缺乏供者血。这是"免疫缺陷 × 超敏反应"的交叉考点。

## 六、PID 诊断检查配对速查 ⭐⭐（UWorld 高频）

| 怀疑病 | 发病机制 | 招牌线索 | 确诊 / 首选检查 | 治疗第一步 |
|---|---|---|---|---|
| **XLA（Bruton）** | **BTK 突变 → pre-B 不成熟 → 无成熟 B 细胞** | 男婴 6 月后反复荚膜菌 + 扁桃体缺如 | **定量 Ig（↓↓全部）+ 流式 CD19（极低）**；BTK 突变 | IVIG |
| **CVID** | **B 细胞分化为浆细胞障碍 → 低丙球** | 成人反复鼻窦肺 + 慢性肺病 / 自免 | **定量 Ig（↓IgG + ↓IgA/IgM）+ 疫苗无应答**（B 细胞数正常）| IVIG |
| **SCID** | **T 细胞发育失败（IL2RG / ADA）+ B 失功能** | 婴儿机会感染 + FTT + 淋巴减少 | **淋巴细胞亚群（CD3 缺）**；新生儿筛查 TREC | **干细胞移植** |
| **CGD** | **NADPH 氧化酶缺陷 → 无氧化爆发** | 婴儿 catalase⁺ 菌 / 真菌空洞肺炎 + 脓肿肉芽肿 | **DHR123 流式 / NBT**（测氧化爆发）| 抗感染 + IFN-γ；移植 |
| **补体末端缺陷** | **C5–C9 缺 → 无膜攻击复合物 MAC** | 反复奈瑟菌 | **CH50** | 疫苗 + 抗生素 |
| **Wiskott-Aldrich** | **WASp 突变 → 细胞骨架缺陷 → T 细胞 + 血小板障碍** | 湿疹 + 血小板↓ + 感染 | 血小板（小）+ Ig（↓IgM）+ 基因 | 移植 |
| **THI（婴儿一过性）** | **婴儿自产 IgG 延迟 → 生理 nadir 延长** | 婴儿孤立 IgG↓ + **疫苗应答正常 + 血小板正常** | 定量 Ig + 疫苗特异抗体（正常）| 观察（自限）|
| **Hyper-IgE/Job** | **STAT3 突变 → Th17↓ → 中性粒趋化障碍** | 湿疹 + 冷脓肿 + 保留乳牙 | **血清 IgE↑↑** + 嗜酸↑ | 抗葡菌预防（TMP-SMX）|
| **早期补体（C1q）** | **免疫复合物 / 凋亡细胞清除障碍** | 早发 SLE + 反复细菌 | **CH50↓ + C4/C2↓** | 治 SLE + 疫苗 |

> [!success] 一句话配对反射
> **↓↓全 Ig + CD19 缺 → XLA｜↓Ig + CD19 正常 → CVID｜CD3 缺 + FTT → SCID｜catalase⁺ 菌 + DHR → CGD｜反复奈瑟菌 + CH50 → 末端补体｜湿疹+血小板↓ → Wiskott｜婴儿孤立 IgG↓ + 疫苗应答正常 + 血小板正常 → THI｜湿疹+冷脓肿+乳牙+IgE↑↑ → Job｜早发 SLE + CH50↓ → C1q（早期补体）。**

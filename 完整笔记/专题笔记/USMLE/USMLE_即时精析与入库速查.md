# _衍生_即时精析与入库速查（Post-Block Review SOP）

> [!info] 本笔记定位
> **流程枢纽**：不讲具体疾病，只讲 **做完一个 UWorld block 后怎么复盘**。
>
> 核心信念：**精析对所有 🔴🟡 做（思考免费）；入库只挑一部分（写卡很贵）**。全入库 = 错题本臃肿到不会再翻 = 白写。
>
> 配套：读题/选项类错因见 [[完整笔记/专题笔记/USMLE/USMLE_读题策略]]、[[完整笔记/专题笔记/USMLE/USMLE_答案选项策略]]；入库规则与状态码见 [[工作流]]、[[错题本]]。

---

## Part 0. 流程总览（做完 block 对着走）

```
做完 40 题 block
   │
   ① 分流 2×2 ───────────► 只对 🔴🟡 往下走（约 15–20 题）
   │
   ② 单题精析 5 步（盖答案自问 → 抓考点+干扰项 → 标错因 → 钩子句）
   │
   ③ 入库 yes/no 过滤器 ──► "换个 stem 同考点还会错吗？"
   │        会 → 开卡       不会 → 走人
   │
   ④ 不开卡的也别丢价值（粗心汇总 / 衍生加反链计数）
   │
   ⑤ 改答案题单独记一笔（第一直觉 vs 改后）
```

> [!tip] 时间硬上限
> 一个 40 题 block 精析 **≤ 50 分钟**。超了 = 你在被动重读解析，停下抓重点。

---

## Part 1. 分流 2×2（决定每题花多久）

按 **「对错 × 有没有把握」** 把 40 题分四格：

|              | 做对                              | 做错 / omit              |
| ------------ | ------------------------------- | --------------------- |
| **有把握**  | 🟢 **10 秒**：扫一眼确认不是蒙对 → 跳    | 🔴 **全力精析**（看错题 / 知识空白） |
| **没把握**  | 🟡 **全力精析**（蒙对 = 隐藏窟窿，答对 ≠ 掌握） | 🔴 **全力精析**           |

> [!warning] 🟡 是最容易漏的金矿
> 你蒙对了，UWorld 算你对，但你**根本不会**。不揪出来，考试就是它们坑你。
> （这也是正确率数字 > 真实掌握度的原因。）

🔴🟡 大约占一个 block 的 **15–20 题**，只对这些走 Part 2。

## Part 2. 单题精析 5 步协议（对 🔴🟡 用）

```
① 先盖答案自问（~30 秒）—— 别急着读解析
   "考点是什么？正确逻辑链？我当时怎么想的？"
   → 自己先重推，再对解析。直接读 = 被动，无记忆增益

② 读解析只抓三样
   • Educational objective（UW 加粗那句 = 考点本体）
   • 我选的那个为什么错
   • 其他干扰项为什么错（USMLE 视角）

③ 标错因（一题只归一类）
   知识空白 / 推理错 / 看错题·粗心 / 改答案改错 / 不会取舍
   → 类型决定怎么补：空白=补笔记；推理=练逻辑；粗心=改读题习惯

④ 逼出一条【可迁移规则】，不是记这道题
   ✗ "这个病人是 PE"
   ✓ "术后/制动 + 单侧腿肿 + 低氧 + 心动过速 → 先想 PE，先 CTA"
   写成 "看到 X → 想 Y / 先做 Z" 的钩子句

⑤ 决定去向：进错题本？还是更新某张衍生？（见 Part 3）
```

## Part 3. 入库 yes/no 过滤器

> [!question]- 一句话判据
> **"换个 stem、同一个考点再考一次，我还会错吗？"**
> - **会** → ✅ 入库（有可迁移价值，值得再测）
> - **不会**（纯手滑 / 已经真懂） → ❌ 精析完走人

### 🔴 错题分情况

| 错因                              | 入库？               |
| ------------------------------- | ----------------- |
| 知识空白（尤其弱科：心内/肾/Heme/感染/儿科/风湿/皮/眼） | ✅ 入库，带钩子句         |
| 推理错 / 不会取舍                      | ✅ 入库（最容易重犯）       |
| **看错题·粗心**（懂，就是手滑）            | ❌ 单题不入库 → Part 4① |
| 超低频冷知识（这辈子不会再考第二次）           | ❌ 顶多笔记加一行，不开卡     |

### 🟡 蒙对题先分两种

- 蒙对 + 一看解析**真不会机制** → ✅ 入库，标 **🟡/知识**（答对 ≠ 掌握）
- 蒙对 + 其实你会、只是**不自信乱怀疑** → ❌ 别开卡，这是信心问题 → 归到 Part 5

---

## Part 4. 不入库的，也别丢价值

> [!success] 不开卡 ≠ 不管
> 1. **粗心错**：别给每道开卡 → 攒成**一张「读题习惯」元笔记**，记模式（"老漏看 EXCEPT / least likely"、"没看完选项就选"）。错的是**习惯**，不是 40 个知识点。详见 [[完整笔记/专题笔记/USMLE/USMLE_读题策略]]。
> 2. **已有笔记/衍生完美覆盖的**：别新建卡 → 去那张衍生**加反链 + 计数**（"这点又错一次"）。错够 2 次自动升 🔴 + 建议 #考前必看，走反复错归纳（见 [[工作流]]）。

---

## Part 5. 改答案追踪（盯你的 Correct→Incorrect）

> [!danger] 你的已知漏分点
> UWorld 统计里 **Correct→Incorrect = 188**（想太多改错）。
>
> 精析时遇到**改过答案的题**，单独记一笔：第一直觉 vs 改后，哪个对。
> 累积几周看命中率 → 大概率"改错 ≫ 改对" → 以后训练自己**压住手别改**。这是能白捡的分。

## Part 6. 一个 block 下来的产出（参考量）

```
🔴🟡 ~15–20 题精析
  ├─ 真入库:    可能就 6–10 张卡（弱科空白 + 推理错 + 🟡真不会）
  ├─ 粗心  →   汇进 1 张读题元笔记
  ├─ 已覆盖 →   衍生加反链计数
  └─ 冷知识/已真懂 → 走人
```

> [!tip] 入库率高不是好事
> 若 20 题里 18 题都想入库 → 要么**弱科太集中**（该回去补该科主笔记/衍生，不是堆卡），要么**门槛太松**。

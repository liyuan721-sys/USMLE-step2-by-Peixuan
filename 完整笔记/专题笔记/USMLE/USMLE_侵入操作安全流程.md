## Part 1. 核心方法论

### 1.1 三个统一安全原则 ⭐⭐⭐

> [!success] 适用所有侵入操作
> 1. **知情同意**（除非救命急）
> 2. **Time-out 三方核对**（患者 + 操作 + 位点）
> 3. **影像 / 波形确认位置 → 才能使用**
>    - 位置未确认前 → 不给药、不补液、不输血、不喂食

### 1.2 USMLE 程序题识别（重要方法论）

> [!warning] 题型识别 4 信号
> 看到以下 4 个信号 = **程序性安全题**，不是临床决策题：
> 1. Stem 详细描述某个操作过程（CVC / LP / 胸管 / 内镜 / 手术）
> 2. 问 "best next step" 但操作刚结束
> 3. 选项里"确认 / 影像"和"治疗"并存
> 4. 临床背景看似紧急（让你想跳步骤）
> 
> **答题原则：永远先选"确认 / 安全检查"，再选"治疗"**

## Part 3. 气道类

### 3.1 Endotracheal Tube (ET, 气管插管)

| 维度 | 详情 |
|---|---|
| **目的** | 气道保护 + 机械通气 + 防误吸 |
| **适应症** | ① 气道保护（GCS ≤ 8）② 呼吸衰竭（I 型低氧 / II 型高碳酸）③ 大手术全麻 ④ 严重外伤 ⑤ 气道梗阻风险（烧伤吸入 / 喉水肿）|
| **路径** | 经口（多数）/ 经鼻（少用）|
| **目标位置** | tip 在 **carina 上方 2-4 cm**（CXR 显示 T2-T4 水平）|
| **置入后确认** | ⭐ **Capnography（end-tidal CO2）= 金标准**；CXR 看深度；双肺呼吸音对称 |
| **常见并发症** | ① **食管插管**（最严重，capnography 识别）② **右主支气管插管**（tube 太深 → 左肺塌陷 / 左侧呼吸音消失）③ 牙齿损伤 ④ 声带损伤 ⑤ 长期 → VAP |

> [!warning] ET 管确认陷阱
> ❌ 仅听呼吸音 → 食管插管也可能听到
> ❌ SaO2 → 延迟反应
> ✅ **Capnography 5-6 次稳定 ETCO2 波 = 金标准**
> ✅ CXR 确认深度

> [!info] ET 太深处理
> tip 越过 carina → 通常入**右主支气管**（右支气管更直）→ 左肺塌陷 → 左侧呼吸音消失
> **处理：回撤 1-2 cm**

### 3.2 Tracheostomy（气切）

| 维度 | 详情 |
|---|---|
| **适应症** | ① ET 插管 > 10-14 天 ② 上气道梗阻 ③ 神经肌肉病长期通气 ④ 大型颌面手术 |
| **位置** | 第 2-3 气管软骨之间，颈前正中 |
| **并发症** | **早期**：出血 / 气胸 / 气管后壁穿孔；**晚期**：气管狭窄 / **Tracheo-innominate fistula**（致命）/ 气管-食管瘘 |

> [!danger] Tracheo-Innominate Fistula
> 气切术后数天-数周突发气切口大出血（先有 "sentinel bleed" → 大出血）
> 紧急处理：**手指压迫止血 + 紧急胸外科**

## Part 4. 消化道类

### 4.1 Nasogastric Tube (NG)

| 维度 | 详情 |
|---|---|
| **目的** | 胃肠减压 / 胃内容物采集 / 短期喂养 / 给药 |
| **路径** | 鼻 → 食管 → 胃 |
| **目标位置** | tip 越过膈下，在胃内 |
| **置入后确认** | ⭐ **CXR = 金标准**；抽出胃内容物 pH < 5.5 |
| **绝对禁忌** | ⭐ **基底骨折**（NG 可能入颅）→ 改 OG |
| **并发症** | ① 气管误入 ② 鼻黏膜出血 ③ **颅内置管**（基底骨折时）④ 误吸（位置错给食）⑤ 食管 / 胃穿孔（罕见）|

> [!danger] 基底骨折禁忌（USMLE 必考）⭐
> **三联征**（任一即怀疑 basilar skull fracture）：
> - **Raccoon eyes**（双眶周淤血）— 前颅窝骨折
> - **Battle sign**（耳后乳突淤血）— 中颅窝骨折
> - **CSF 鼻漏 / 耳漏**（halo sign / glucose+）— 硬膜撕裂
> - + Hemotympanum（鼓室积血）/ 颅底骨折影像
> 
> → **绝对禁忌"所有经鼻管路"**（不只是 NG 一种），必须改对应**经口**路径：
> 
> | ❌ 禁忌（经鼻）| ✅ 替代（经口）| 备注 |
> |---|---|---|
> | **NGT**（鼻胃管）| **OGT**（口胃管）| 减压 / 短期给药 |
> | **Dobhoff**（鼻肠喂养管）⚠️ | OGT（短期）/ PEG（择期 > 4-6 wk）| **常被误以为"特殊安全管"** — 路径完全相同（鼻 → 食管），同样禁忌！|
> | **Nasotracheal intubation**（经鼻气管插管）| **Orotracheal intubation**（经口气管插管 + capnography 确认）| 急救建立气道 |
> | **NPA**（鼻咽通气道，nasopharyngeal airway）| **OPA**（口咽通气道，oropharyngeal airway）| 短期开放气道 |
> 
> **机制**：颅底硬膜缺损 → 任何经鼻细管都可能**经筛板（cribriform plate）进入颅内** → 颅内置管 / 颅内感染（脑膜炎）/ 直接脑损伤
> 
> **临床还要避免**：
> - 经鼻吸引（nasal suctioning）
> - 用力擤鼻（valsalva → 气颅）
> - 经鼻 BiPAP / CPAP
> - 任何经鼻药物 / 喂养
> 
> **USMLE 高频陷阱**：选项给 "Dobhoff tube" 让你以为能绕过 NG 禁忌 → **错**！Dobhoff 也是经鼻 → **唯一对的答案 = OGT**。

### 4.2 Orogastric Tube (OG)

| 维度 | 详情 |
|---|---|
| **适应症** | ① 基底骨折 ② 颜面外伤 ③ 鼻腔阻塞 ④ 短期（手术中）|
| **缺点** | 清醒患者难耐受 → 多用于插管 / 意识障碍患者 |

### 4.3 Dobhoff / NJ Tube（鼻肠喂养管）

| 维度 | 详情 |
|---|---|
| **目的** | 长期肠内营养（绕过胃）|
| **适应症** | ① 严重 GERD / 误吸风险 ② 胃排空障碍 ③ **急性胰腺炎**（绕过胰腺刺激）④ ICU 营养 |
| **路径** | 鼻 → 食管 → 胃 → 幽门 → 十二指肠 / 空肠 |
| **置入** | X 线或内镜引导（细 + 软）|

### 4.4 PEG（经皮内镜下胃造瘘）

| 维度 | 详情 |
|---|---|
| **适应症** | 长期肠内营养（> 4-6 周）：卒中后吞咽障碍 / ALS / 痴呆终末 / 头颈癌 |
| **并发症** | **即刻**：穿孔 / 误置入结肠；**晚期**：**Buried bumper syndrome**（内固定埋入胃壁）/ 堵管 |

## Part 5. 血管类

### 5.1 PIV (Peripheral IV, 外周静脉)

| 维度 | 详情 |
|---|---|
| **限制** | 不能给：血管活性药 / TPN / 3% saline / 化疗腐蚀药 |
| **并发症** | 静脉炎 / 渗液（infiltration）/ **外渗（extravasation）**腐蚀药致坏死 |

### 5.2 CVC (Central Venous Catheter) ⭐⭐⭐

#### 5.2.1 三大穿刺点对比

| 维度 | **IJV（内颈）** | **Subclavian** | **Femoral** |
|---|---|---|---|
| 默认地位 | **首选**（超声引导）| 长期通路 / 患者舒适 | 急诊 / CPR |
| 解剖标志 | 胸锁乳突肌两头三角 + 颈动脉外侧 | 锁骨中外 1/3 下方朝胸骨上凹 | 腹股沟韧带下 2-3 cm，股动脉内侧（NAVEL）|
| 超声引导 | ★ 强烈推荐 | 通常无（盲穿）| 推荐 |
| **气胸风险** | 低 | **最高** | 无 |
| **动脉穿刺** | 颈动脉（可压）| 锁骨下动脉（**不可压**）| 股动脉（可压）|
| 可压迫止血 | ✓ | **✗** | ✓ |
| **感染风险** | 中 | **最低** | **最高** |
| DVT 风险 | 低 | 低 | **最高** |
| 患者舒适 | 中 | **最好** | 差 |
| CPR 中可用 | ✗ | ✗ | **✓** |
| 置管深度 | 右 15-17 cm / 左 18-20 cm | 14-16 cm | 20-25 cm |

#### 5.2.2 CVC 选位决策树

```
急诊 / CPR? → Femoral
凝血障碍? → 避 SC（不可压迫）→ IJV / Femoral
COPD? → 避 SC（气胸代偿差）→ IJV
拟血透? → 避 SC（中心静脉狭窄影响 fistula）→ IJV / Femoral
长期 + 患者要活动? → SC / PICC
默认 → IJV 超声引导 ★
```

#### 5.2.3 CVC 标准流程

```
1. 知情同意 + Time-out
2. Trendelenburg（IJV / SC）+ 严格无菌 + Chlorhexidine
3. 超声 / 标志定位
4. Seldinger 技术
   ① 针穿入血管 → 血回
   ② 送 J-tip 导丝
   ③ 拔针留丝
   ④ 扩张器扩张
   ⑤ 沿丝送导管
   ⑥ 拔丝
   ⑦ 各管腔抽血回 + 冲管
5. 缝合固定 + 无菌敷料
6. ⭐ Portable CXR（IJV / SC 必做；Femoral 可省）
   确认：tip 在 lower SVC (T4-T6) + 无气胸 + 无血胸 + 无异位
7. ⭐ CXR 确认后才能使用 ⭐
```

#### 5.2.3a CLABSI 预防 Bundle ⭐⭐⭐（USMLE 高频）

> [!success] 完整 CLABSI（Central Line-Associated BSI）预防 Bundle
> CDC / IHI 标准 5 条 — 任一条单独问到都要能识别：

| 步骤 | 措施 | 关键点 |
|---|---|---|
| **① 最大无菌屏障** | 操作者：帽 + 口罩 + **无菌手术衣 + 无菌手套**；患者：从头到脚**全身大无菌单** | 不是只盖局部 |
| **② 皮肤消毒** | **Chlorhexidine（氯己定）> 0.5% 酒精溶液**首选 | **优于** povidone-iodine（碘伏）/ alcohol 单用 |
| **③ 穿刺点选择** | **首选锁骨下（Subclavian）** > 颈内（IJV）> **股静脉最差**（最高感染率） | **感染风险**：股 > 颈 > 锁骨下 |
| **④ 每日评估必要性** | 不再需要 → **尽早拔管** | 留置时间是 CLABSI 最强独立危险因素 |
| **⑤ Scrub the hub** | 每次使用接口前 **70% 酒精擦拭 ≥ 15 s** | + 严格手卫生 + 每日氯己定擦浴 |

> [!warning] 长期 CVC 维护三大并发症
>
> | 并发症 | 预防 |
> |---|---|
> | **感染（CLABSI）** | 严格手卫生 + 氯己定接口消毒 + 定期更换敷料 |
> | **血栓（CRT）** | 定期肝素 / 生理盐水**冲管 (flushing)** |
> | **脱出 (Dislodgement)** | 妥善固定 + 避免剧烈运动 |
>
> **核心口诀**："手部清洁，导管安全" — 触碰接口前洗手永远是第一步。

> [!info] CLABSI 穿刺点选择悖论
> CLABSI 预防 **首选 SC**（感染最低），但 SC **气胸风险最高 + 不可压迫**。所以 USMLE 选位题要看权衡：
> - 感染最关心（长期 / 免疫低下） → SC
> - 凝血障碍 / 急诊 → 避 SC（不可压迫）→ IJV
> - CPR / 紧急 → Femoral（可接受短期高感染风险）

#### 5.2.4 CVC 并发症（按时序）

| 时间 | 并发症 | 关键处理 |
|---|---|---|
| **即刻** | 气胸（SC 最高）| 小→观察；大→胸管 |
| | 动脉穿刺（颈 / 锁骨下 / 股）| 压迫；已置管→血管外科 |
| | **空气栓塞**（"machinery murmur"）| **左侧卧位 + Trendelenburg + 100% O2 + 抽气** |
| | 心律失常（tip 入右房）| 回撤 1-2 cm |
| | 心包填塞（穿心房）| 紧急心包穿刺 |
| | 胸导管损伤（左侧）→ chylothorax | 引流 + 禁脂 |
| | 神经损伤 | 多自愈 |
| **早期** | 导管闭塞 | tPA 溶栓 |
| | 延迟气胸（SC）| 复查 CXR |
| **晚期** | **CRBSI** | 拔管 + 培养 + 抗生素 |
| | 血栓（DVT）| 抗凝 ± 拔管 |
| | 中心静脉狭窄（长期 SC）| 介入扩张 |

#### 5.2.5 CVC stem 识别快表

| Stem 关键词 | 诊断 |
|---|---|
| SC 后突发胸痛 + 单侧呼吸音↓ | 气胸 |
| IJV 高压鲜红血回 | 颈动脉穿刺 |
| 置入后低血压 + 缺氧 + 杂音 | **空气栓塞** |
| tip 入右房后 VT | 导管诱发心律失常 |
| 突发休克 + Beck 三联 | 心包填塞 |
| 留置 > 48h 新发发热 + 阳培养 | CRBSI |
| 左侧 SC 后乳糜样积液 | Chylothorax |
| 长期 SC 后上臂 / 面部水肿 | 中心静脉狭窄 / SVC 综合征 |

### 5.3 PICC

| 维度 | 详情 |
|---|---|
| **适应症** | 长期 IV 抗生素 / 化疗 / TPN |
| **路径** | 上肢贵要静脉 / 肱静脉 → SC → SVC |
| **优点** | 置入风险低（无气胸 / 大动脉风险）/ 留置长（数周-数月）|
| **缺点** | **血栓风险高于 CVC** / 流量低 / 不能快速大量复苏 |
| **置入后确认** | CXR |

### 5.4 Arterial Line (A-line)

| 维度 | 详情 |
|---|---|
| **目的** | 连续 BP 监测 + 频繁 ABG + 升压药患者 |
| **位置** | **桡动脉**（首选）/ 股动脉 / 肱动脉 / 足背动脉 |
| **置入前** | Allen test（历史性，证据有限）|
| **置入后确认** | ⭐ **动脉波形** + 鲜红高压血气 |
| **并发症** | ① **远端缺血**（血栓 / 血管痉挛→手指坏死）② 出血 / 血肿 ③ 感染 ④ **误注药致远端坏死** |

> [!danger] A-line 致命错误
> **A-line 严禁注药！** 误注 IV 药 → 直接到达远端组织 → 化学性损伤 + 血管痉挛 + 坏死
> 预防：A-line **物理隔离 + 清楚标记**

### 5.5 Swan-Ganz / PA Catheter

| 维度 | 详情 |
|---|---|
| **目的** | 监测 RA 压 / PA 压 / **PCWP** / CO |
| **路径** | CVC → 右房 → 右室 → 肺动脉 → 楔入肺小动脉 |
| **现状** | 使用大幅减少（无预后获益），被 echo / 无创替代 |
| **致命并发症** | **肺动脉破裂**（罕见）|

> [!info] PCWP 临床意义
> PCWP 反映左房压：
> - 心源性肺水肿 → PCWP↑
> - ARDS / 非心源 → PCWP 正常
> - 鉴别心源 vs 非心源休克

### 5.6 IO (Intraosseous, 骨内通路)

| 维度 | 详情 |
|---|---|
| **适应症** | CPR 中 / 儿童急救 / 烧伤创伤 IV 困难 |
| **位置** | **胫骨近端**（首选）/ 肱骨头 / 胫骨远端 |
| **优点** | 60-90 秒置入；几乎任何药都可给 |
| **限制** | 最多留置 **24 小时**，之后改 IV / CVC |

## Part 6. 泌尿类

### 6.1 Foley Catheter

| 维度 | 详情 |
|---|---|
| **适应症** | 尿潴留 / 尿量监测 / 长期失禁（最后选）|
| **置入后确认** | 尿液回流 + 球囊充水 5-10 mL |
| **绝对禁忌** | ⭐ **泌尿生殖道损伤**（骨盆骨折 + 血尿 + 尿道口血 + 阴囊血肿）|
| **并发症** | ① **CAUTI**（最常见院感）② **尿道破裂**（球囊在尿道内充气）③ 尿道狭窄 ④ 血尿 |

> [!danger] 尿道损伤禁忌（USMLE 必考）
> 骨盆骨折患者出现：血尿 / 尿道口血 / 阴囊血肿 / 高骑前列腺 → 怀疑尿道损伤
> **绝对禁忌 Foley**！强行置 = 部分撕裂变完全断裂
> 正确：**先 retrograde urethrogram** → 完整再 Foley；急需引流 → **耻骨上膀胱造瘘**

### 6.2 Suprapubic Catheter

| 维度 | 详情 |
|---|---|
| **适应症** | 尿道损伤 / 长期尿潴留 / Foley 失败 |
| **置入** | 耻骨上方 2 横指经皮，需膀胱充盈 |

## Part 7. 胸腔类

### 7.1 Chest Tube（胸管）⭐⭐⭐

| 维度 | 详情 |
|---|---|
| **目的** | 引流胸腔：气 / 液 / 血 / 脓 |
| **适应症** | ① 气胸（大 / 有症状 / 张力性 / 双侧）② 胸腔积液 / 血胸 / 脓胸 ③ 术后引流 |
| **位置** | ⭐ **"Triangle of Safety"** 安全三角：腋中线后方 + 腋前线前方 + 第 5 肋间上方 |
| **方向** | **气胸 → 前上**（气向上）；**液 / 血胸 → 后下**（液向下）|
| **置入后确认** | ⭐ **CXR** + 引流物对应（气泡 / 液 / 血）|
| **并发症** | ① 肺穿刺 / 膈肌肝脾损伤（位置太低）② 漏气 / 引流不畅 ③ **复张性肺水肿**（>1.5L 快速引流）④ 感染 |

> [!danger] 张力性气胸 — 临床诊断不靠 CXR
> **不等影像！** 单侧呼吸音消失 + 气管偏**对侧** + JVD + 低血压 + 缺氧
> 立即处理：**针刺减压**（第 2 肋间锁骨中线 / 第 4-5 肋间腋前线）→ **正式胸管**

> [!warning] 复张性肺水肿
> 大量胸腔积液一次性 > 1.5L → 肺快速复张 → 毛细血管损伤 → 肺水肿
> 预防：每次 ≤ 1.5 L，分次引流

#### 7.1.1 解剖定位精确版（USMLE 必考）⭐

> [!danger] 三种胸腔急症 — 操作位置 + 进针顺序对比

| 情境 | 操作 | 位置 | 进针 | 关键陷阱 |
|---|---|---|---|---|
| **张力性气胸**（立刻减压）| **针刺减压（Needle decompression）** | **第 2-3 肋间，锁骨中线 (midclavicular line)** | **肋骨上缘**（above the rib，避肋间神经 / 血管）| 临床诊断 → **不等 CXR**；针刺只是过渡，仍需正式胸管 |
| **自发气胸 / 一般气胸** | **胸管 (Chest tube)** | **第 5-6 肋间，腋中线 (midaxillary line)**（"Triangle of Safety"内）| **肋骨上缘** | 方向：气胸 → 前上 |
| **Hemothorax / 血胸**（创伤）| **直接胸管，不针刺**| 同上 5-6 肋间腋中线 | **肋骨上缘** | **flat / 扁颈静脉**（失血致回心↓，与张力性气胸 JVD 相反）;一次引流 > 1500 mL 或持续 > 200 mL/h 需开胸 |

> [!info] 张力气胸 vs 血胸 颈静脉对比
> - **张力性气胸**：胸内压↑↑→压迫上腔静脉 → **JVD（颈静脉怒张）** + 气管偏对侧
> - **Hemothorax**：失血致回心血量↓ → **Flat neck veins（扁颈静脉）** + 气管居中
> - 这是 USMLE 创伤题快速分流的关键体征

> [!warning] "Above the rib"原则
> 肋间神经血管束沿**肋骨下缘**走行（V-A-N 自上而下）→ 进针必须**贴下一肋上缘**避免损伤。所有胸腔操作（针刺减压 / 胸管 / thoracentesis）通用。

### 7.2 Thoracentesis（胸腔穿刺）

| 维度 | 详情 |
|---|---|
| **目的** | 诊断（Light's criteria）/ 治疗（缓解症状）|
| **位置** | 第 7-9 肋间，肩胛线 / 腋后线，**肋骨上缘**（避肋间神经血管）|
| **置入后** | CXR 排除气胸 |
| **并发症** | ① **气胸**（最常见 5-10%）② 血胸 ③ 肝 / 脾损伤 ④ 复张性肺水肿 |

## Part 8. 腹腔 / 腰椎类

### 8.1 Paracentesis（腹腔穿刺）

| 维度 | 详情 |
|---|---|
| **目的** | 诊断（SBP 筛查 / SAAG 病因）/ 治疗（大量腹水）|
| **位置** | ⭐ **左下腹**（McBurney 镜像，避盲肠）/ 脐下中线 |
| **置入前** | 叩诊定位 + **超声引导**（首选）|
| **大量放液（> 5L）** | ⭐ **补充白蛋白 6-8 g/L removed**（防 PICD - paracentesis-induced circulatory dysfunction）|
| **并发症** | 肠穿孔（罕见）/ 出血 / 持续渗漏 / SBP |

### 8.2 Lumbar Puncture (LP) ⭐⭐

| 维度 | 详情 |
|---|---|
| **目的** | 诊断脑膜炎 / SAH / GBS / MS；化疗鞘内；测 ICP |
| **位置** | **L3-L4 或 L4-L5**（脊髓 L1-L2 终止）|
| **体位** | 侧卧屈膝（最佳测压）/ 坐位前屈 |
| **置入前必须** | ① 神经系统检查 ② **以下指征先 CT**（排除占位防脑疝）|
| **并发症** | ① **Post-LP headache**（最常见 5-30%，体位性）② **脑疝**（颅压高未先 CT，致命）③ 出血（凝血障碍致硬膜外血肿）④ 感染 |

> [!danger] LP 前必须 CT 的指征（USMLE 必背）
> ① 免疫低下（HIV / 移植）
> ② 新发癫痫
> ③ 局灶神经体征
> ④ 意识改变（GCS < 15）
> ⑤ 视乳头水肿
> ⑥ > 60 岁
> 
> **怀疑脑膜炎 → 先经验性抗生素 → CT → LP**（不能因等 CT 延误抗生素）

> [!info] Post-LP Headache 处理
> 平卧 + 大量补液 + 咖啡因 + NSAIDs
> 顽固性 → **Epidural blood patch**（自体血注入硬膜外腔封漏）

## Part 11. USMLE 题型识别 4 类

### 11.1 选位 / 选管题（Selection）

> "Which is the most appropriate access for this patient?"

**答题原则**：根据患者情况（凝血 / 急诊 / 感染 / 长期 / CPR）选位
- CPR → Femoral / IO
- 凝血障碍 → 避 SC
- 长期 IV → PICC / SC
- 默认 → IJV 超声

### 11.2 置入后下一步题（Confirmation）⭐ Q3504

> "Following placement, what is the best next step?"

**答题原则**：永远先选**位置确认**
- ET → Capnography
- NG / CVC / 胸管 → CXR
- A-line → 动脉波形

### 11.3 并发症识别题（Complication）

> "Following placement, patient develops [症状]. Most likely cause?"

**答题原则**：时间 + 位点 + 症状 → 推并发症
- 即刻 + SC + 胸痛单侧呼吸音↓ → 气胸
- 即刻 + 低血压 + 缺氧 + 杂音 → 空气栓塞
- 数天 + 发热 + 阳培养 → CRBSI

### 11.4 禁忌识别题（Contraindication）

> "Which patient should NOT receive [procedure]?"

**答题原则**：识别绝对禁忌
- 基底骨折 + NG = ❌
- 尿道损伤 + Foley = ❌
- 颅压高 + LP = ❌

## Part 12. 速记口诀总结

> [!success] "三个统一安全原则"
> 1. 知情同意（除非急救）
> 2. Time-out 三方核对
> 3. 影像 / 波形确认 → 才能用

> [!success] "三个必做 CXR"
> CVC（IJV / SC）/ NG 管 / 胸管

> [!success] "三个特殊确认"
> ET → Capnography
> A-line → 动脉波形
> LP → CSF 流出

> [!success] "三个绝对禁忌"
> 基底骨折 + NG = ❌（改 OG）
> 尿道损伤 + Foley = ❌（改 suprapubic）
> 颅压高 + LP = ❌（先 CT）

> [!success] "三个最常见并发症"
> SC CVC → 气胸
> Foley → CAUTI
> LP → Post-LP headache

> [!success] "三个最致命并发症"
> ET 食管插管 → 缺氧死亡
> LP（颅压高）→ 脑疝
> 张力气胸（不等 CXR）→ 立即针刺减压

> [!success] "CVC 三大穿刺点选择"
> 默认 → IJV
> 急诊 / CPR → Femoral
> 长期 / 舒适 → SC / PICC

> [!success] Peixuan CVC 口诀（自创）⭐
> **"CVC 尖端放 SVC，carina 附近最合适；太浅穿静脉，太深穿心肌；锁骨下怕气胸，颈内静脉靠超声。"**
>
> - 尖端目标位置：**lower SVC ≈ carina level**
> - **太浅**（颈静脉 / 锁骨下静脉本段）→ 易致 venous perforation
> - **太深**（入右房 / 右室）→ 致心肌穿孔 → tamponade + 心律失常
> - **SC（锁骨下）**：解剖标志法常用，**最怕气胸**
> - **IJV（颈内）**：**超声引导首选**

## Part 13. 高频陷阱总结

| 陷阱 | 正确认知 |
|---|---|
| ❌ 临床优先级 = 跳过安全检查 | ✓ 紧迫感不能省略位置确认（Q3504）|
| ❌ 听呼吸音 = ET 在气管 | ✓ Capnography 才是金标准 |
| ❌ 抽血回 = CVC 在静脉 | ✓ 动脉穿刺也有血回；要看 CXR + 波形 |
| ❌ Femoral 是劣等位点 | ✓ 急诊 / CPR / 凝血障碍时首选 |
| ❌ A-line 可给药 | ✓ 严禁！会致远端坏死 |
| ❌ 张力气胸要等 CXR 确诊 | ✓ 临床诊断，立即针刺减压 |
| ❌ 大量胸水 / 腹水一次抽完 | ✓ 胸水 ≤ 1.5L / 次；腹水 > 5L 补白蛋白 |
| ❌ 怀疑脑膜炎要等 CT 才用抗生素 | ✓ 先经验抗生素 → CT → LP |
| ❌ 骨盆骨折随意上 Foley | ✓ 怀疑尿道损伤先 RUG，必要时 suprapubic |
| ❌ 基底骨折可以 NG | ✓ 绝对禁忌！改 OG |

# USMLE — 种族族裔指纹速查（Race/Ethnicity Stem Triggers）

> [!info] 笔记定位
> USMLE stem 写出 race/ethnicity（如 "African American" / "Ashkenazi Jewish" / "Mediterranean"）几乎都是**有意的诊断线索**。本笔记按"族裔 × 高发疾病 × 机制 × Stem 模式"组织，是 USMLE 的"族裔指纹"速查。来源：[[mistakes/uworld-mistakes_2026-05#^Q4436]]（5-28 SCT 错题 — 反射"AA + 多尿 + 家族 sickle"未建立 → 早建 v1）。

---

## Part 1. 7 大族裔指纹速查表 ⭐⭐⭐

### 1.1 African American (AA / Black) — 最高频
| 高发疾病 | 机制 / 关键考点 | Stem 反射 |
|---|---|---|
| **Sickle cell (SCT/SCD)** ⭐⭐⭐ | HbS 基因 AA 携带率 ~8% / SCT 1/13 / SCD 1/365 出生 | "AA + 多尿稀尿 + 家族 sickle" → SCT hyposthenuria |
| **G6PD deficiency** ⭐ | X-linked / 男性多发 / **fava bean / sulfa / 抗疟药 / 感染** 诱发 | "AA 男 + 用 TMP-SMX 后溶血 + bite cells / Heinz body" |
| **Sarcoidosis** ⭐⭐ | 非干酪样肉芽肿 / **1,25-VitD↑→高钙** / ACE↑ / 肺门淋巴结 | "AA 女 + 干咳 + bilateral hilar adenopathy + erythema nodosum" |
| **早发 HTN + CKD** | RAAS 敏感 + 盐敏感 | "AA + 50yo + HTN 控制差" → 早开始 thiazide/CCB（**ACEi 单用效果差**）|
| **SLE** | 4-9 倍 vs 白人 | "AA 育龄女 + 多脏器" |
| **Keloid** | 真皮纤维过度增生 | 耳穿洞 / 切口处 |
| **Prostate ca** | 1.7 倍 vs 白人 + 更早 | 50yo 起筛查 PSA |
| **TB 易复发** | — | 移民史 + AA → TB 高危 |

### 1.2 Mediterranean（希腊 / 意大利 / 中东 / 北非）
| 高发疾病 | 机制 / 关键考点 | Stem 反射 |
|---|---|---|
| **β-thalassemia** ⭐⭐ | β-globin 突变 → microcytic 贫血 + target cells + HbF↑ | "意大利 / 希腊 + 小细胞贫血 + 输血依赖 + 铁过载" |
| **G6PD severe variant** | Mediterranean 变体比 AA 变体严重 | "希腊男 + fava bean + 溶血" |
| **FMF (Familial Mediterranean Fever)** ⭐ | AR / MEFV 基因 / **colchicine 预防** | "Sephardic Jewish / Armenian / Turkish + 反复发热腹膜炎关节炎 + 淀粉样变 (AA amyloidosis)" |

### 1.3 Ashkenazi Jewish（东欧犹太）⭐
| 高发疾病 | 机制 / 关键考点 | Stem 反射 |
|---|---|---|
| **Tay-Sachs disease** ⭐⭐ | AR / Hexosaminidase A↓ / GM2 ganglioside 积 | "Ashkenazi 婴儿 + cherry-red spot + 神经退化 + 无 hepatosplenomegaly" |
| **Gaucher disease** | AR / Glucocerebrosidase↓ / 巨噬细胞充塞 | "Ashkenazi + hepatosplenomegaly + 骨痛 + Gaucher 细胞" |
| **Niemann-Pick** | AR / Sphingomyelinase↓ | "Ashkenazi 婴儿 + cherry-red spot + **hepatosplenomegaly**"（与 Tay-Sachs 区别在肝脾）|
| **Crohn disease** | 2-4 倍 vs 一般 | 慢性腹泻 + 跳跃节段 + 肉芽肿 |
| **BRCA1/BRCA2** ⭐ | 创始突变 / 乳腺 + 卵巢癌风险↑↑ | 40yo 前乳腺癌 + 卵巢癌 + Ashkenazi 家族史 → 基因检测 |
| **Factor XI deficiency** | AR / 轻 - 中度出血 | 罕考但 Ashkenazi 高频背景 |

### 1.4 Northern European（北欧 / 苏格兰 / 凯尔特）
| 高发疾病 | 机制 / 关键考点 | Stem 反射 |
|---|---|---|
| **Cystic fibrosis (CF)** ⭐⭐⭐ | AR / CFTR 突变 (Δ508 最常见) / 北欧后裔 1/3000 | "白人婴儿 + 反复肺感染 + 胰功能不全 + 男不育 + 汗 Cl⁻↑ > 60" |
| **Hemochromatosis (HFE)** ⭐ | AR / C282Y 突变 / 铁过载 | "苏格兰 / 凯尔特后裔 + 中年男 + 皮肤铜色 + DM + 肝硬化 + 心衰" |
| **α1-antitrypsin deficiency** | AR / Pi*ZZ / 肺气肿 + 肝病 | "白人非吸烟者 + 早发 panacinar 肺气肿 + 肝硬化" |

### 1.5 Asian / Southeast Asian（东亚 / 东南亚）
| 高发疾病 | 机制 / 关键考点 | Stem 反射 |
|---|---|---|
| **α-thalassemia** ⭐ | α-globin 缺失 / 东南亚 + 中国南方 | "越南 / 老挝 / 中国南方 + microcytic 贫血 + 4 基因型不同表型（silent / trait / HbH / Hb Barts hydrops fetalis）" |
| **G6PD (Asian variant)** | 中间程度严重 | "中国男 + 蚕豆 / 抗疟药 + 溶血" |
| **Lactose intolerance** | 乳糖酶持续性表型缺失（成年酶↓）| 喝奶后腹泻腹胀 |
| **鼻咽癌 (NPC)** | **EBV 关联** + 广东 / 香港高发 | "中国南方男 + 颈淋巴 + 鼻塞 + EBV+" |
| **胃癌** | H. pylori + 高盐饮食 | 日本 / 韩国老年 |

### 1.6 Hispanic / Latino + Native American
| 高发疾病 | 机制 / 关键考点 | Stem 反射 |
|---|---|---|
| **T2DM** ⭐ | 胰岛素抵抗 + 肥胖 + 西化饮食 | 30-40yo + 肥胖 + HbA1c↑ + acanthosis nigricans |
| **NAFLD / NASH** | T2DM + 肥胖 + 西化 | 转氨酶轻↑ + US fatty liver |
| **胆囊癌 / 胆石** | Native American 女性最高发 | RUQ 痛 + 胆囊壁钙化 (porcelain GB) |

### 1.7 Scandinavian
| 高发疾病 | 机制 / 关键考点 | Stem 反射 |
|---|---|---|
| **Hemochromatosis** | 同 1.4 北欧 | "瑞典 / 挪威后裔 + 铁过载" |
| **PSC (Primary sclerosing cholangitis)** | + UC 联病 / 男性多 | 北欧男 + UC + 胆汁淤积 + 串珠样胆管 |

## Part 2. 反 pattern 警示 ⭐

| 反 pattern | 真相 |
|---|---|
| ❌ "AA = 一定 SCT" | SCT 只 1/13，AA 也可能其他病；要看 stem 配套指纹（家族史 / 多尿 / 血涂片）|
| ❌ "白人 = 一定 CF" | CF 也有非白人；要看 stem 配套（反复肺感染 / 汗 Cl⁻）|
| ❌ "Ashkenazi = 一定 Tay-Sachs" | 也可能 Gaucher / Niemann-Pick / Crohn / BRCA；要靠区分性指纹（cherry-red + 无肝脾 = Tay-Sachs / cherry-red + 肝脾大 = Niemann-Pick）|
| ❌ "Mediterranean = 一定 β-thal" | 也可能 G6PD / FMF；β-thal 看 target cells + HbF↑，FMF 看反复发热腹膜炎 |
| ✅ **Race 仅是"先验概率提升"** | 不能单凭 race 锁病，必须配 stem 其他指纹 |

### USMLE 出题哲学 ⭐
- Race 是**有意线索**（不是 PC 装饰）— 出题人写就有理由
- **但不是排他诊断** — 必须配病史 / 体检 / 实验室
- 反向：**stem 没写 race 也不排除某病**（如非 AA 也可有 SCT）

## Part 4. 进化机制（为什么这些族裔有这些病？）⭐

| 疾病 | 进化保护 | 携带优势区 |
|---|---|---|
| **Sickle cell (HbS)** | 杂合对 **恶性疟** 保护 | 撒哈拉以南非洲 / 地中海 / 印度 / 中东 |
| **β-thalassemia** | 杂合对 **疟疾** 保护 | 地中海 / 中东 / 印度 / 东南亚 |
| **α-thalassemia** | 杂合对 **疟疾** 保护 | 东南亚 / 中国南方 / 非洲 |
| **G6PD deficiency** | 对 **疟疾** 保护 | 非洲 / 地中海 / 中东 / 亚洲 |
| **Tay-Sachs / Gaucher** | 可能对 **TB** 保护（争议）| Ashkenazi 创始者效应 |
| **CFTR 杂合** | 可能对 **霍乱 / 伤寒** 保护 | 北欧后裔 |
| **HFE 杂合** | 可能对 **铁缺乏环境** 适应 | 凯尔特 / 维京后裔 |

**核心规律**：**疟疾流行区 → 红细胞遗传病高发**（5 大都是疟疾防御）⭐

## Part 5. Memory Hook

> **"Race 是先验概率提升，不是排他诊断"** ⭐
>
> **AA 三大金刚**：Sickle / G6PD / Sarcoidosis
>
> **Ashkenazi 三大金刚**：Tay-Sachs / Gaucher / BRCA
>
> **Mediterranean 三大金刚**：β-thal / G6PD (severe) / FMF
>
> **Northern European 三大金刚**：CF / HFE / α1-AT
>
> **东南亚 / 东亚**：α-thal / G6PD / 鼻咽癌 (EBV)
>
> **Hispanic / Native**：T2DM / NAFLD / 胆囊癌
>
> **进化规律**：疟疾区 → 红细胞遗传病（Sickle / β-thal / α-thal / G6PD）

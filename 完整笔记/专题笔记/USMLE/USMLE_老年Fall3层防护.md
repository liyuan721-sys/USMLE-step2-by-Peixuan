# USMLE_老年Fall 3 层防护

> [!info] 笔记定位
> 整合 **Q107151 + Q107152 + Q107154 + Q107171** 老年 Fall 4 题成体系（2026-05-26 ~ 2026-05-27）。
> 核心目标：建立 **3 层防御 + 7 维评估 + 反 pattern 4 联** 反射框架，并锁死 5 大铁律。
> 关键易踩坑：alarm 类设备无效（Q107152 + Q107154 同源反 pattern 2 次踩）。

---

## §1 流行病学与考点定位

- **Falls = 老年 (age >65) injury 主要原因** — trauma / prolonged immobilization / functional decline
- 1 次 fall → 增 subsequent fall 风险（**prior history of fall** 是最强 predictor）
- 严重后果 = **hip fracture**（leading serious injury → mortality 高）
- USMLE CK 考点：

```
Fall 题 4 大角度
├─ 风险评估（multifactorial 7 维）       — Q107171
├─ Screening tool 评价（PPV 陷阱）       — Q107151
├─ 干预策略选择（tailored vs universal）— Q107152
└─ Secondary 预防（bone density）        — Q107154
```

---

## §2 3 层防御核心框架 ⭐⭐⭐

> [!success] **见到"reduce fall" → 必须先问：减发生 (primary) / 减 severity (secondary) / 还是 recovery (tertiary)？**

| 层级 | 目标 | 关键干预 | Evidence |
|---|---|---|---|
| **Primary**（防 fall 发生）| 减 fall 频率 | Exercise + home safety + medication review + vision-hearing + vitamin D | 强 |
| **Secondary**（减 fall 后 serious injury）⭐ | 减 hip fracture | **FRAX → DEXA → bisphosphonate** | 强 |
| Tertiary（fall 后 recovery）| 恢复功能 | PT / OT rehab | — |

### 2.1 PT vs OT 分工速查 ⭐（Fall 防护中的角色对照）

| | **PT — Physical Therapy（物理治疗）** | **OT — Occupational Therapy（作业治疗）** |
|---|---|---|
| 主要训练 | 大动作 — gait / balance / strength / range of motion | 精细动作 + ADL — 穿衣 / 洗澡 / 做饭 / 上厕所 |
| Fall 防护中角色 | Balance training / supervised exercise / cane-walker 训练 | **Home safety assessment** ⭐ / grab bars / 环境改造 / adaptive equipment |
| 在 3 层中位置 | Primary（exercise）+ Tertiary（rehab）| **Primary（home safety）** + Tertiary（ADL rehab）|
| 经典 stem 触发词 | "improve strength and balance" / "post-stroke gait rehab" / "supervised exercise" | "evaluate ADL" / "home modifications" / "fall prevention home visit" |

> [!tip] 记忆要点
> - **PT 管"身体怎么动"**（gait + balance + strength）
> - **OT 管"做事情的能力"**（ADL + 环境适应）
> - **Home safety assessment = OT 招牌**（CK stem 看到"OT 来家访" → home safety / ADL 评估）

> [!tip] 思维切换原则
> Stem 已 mention "primary 已实施"（exercise + home safety done）→ 思路**必须切到 secondary**。
> 选 primary 重复项（再 home safety / 再 exercise）= 错。
> 选 "serious injury" 关键词 = 直接锁 **bone density 优化**（hip fracture 是 fall 后最严重）。

## §3 Multifactorial Risk Assessment 7 维矩阵 ⭐⭐⭐

> [!info] 所有 fallers (post-fall follow-up) 必须做 multifactorial assessment

| 维度 | 评估工具 | 干预 |
|---|---|---|
| **Vision** ⭐ | Visual acuity + dilated eye exam（cataracts / macular degen / dark adaptation 减）| Corrective eyewear + cataract surgery + macular degen nutritional supplementation |
| **Hearing** | Audiometry | Hearing aids |
| **Proprioception / Balance** | Get-up-and-go test（**asks patient to rise from chair & walk briskly to end of room**）| PT balance training + supervised exercise |
| **Strength** | Manual muscle testing + slow rise from chair | Supervised exercise + PT |
| **Medication review** | Polypharmacy + 高 risk drugs（neuroleptics / antidepressants / vasodilators / sedatives）| Deprescribe / dose adjust |
| **Home safety** | OT 评估 | 修缮（grab bars / lighting / 移除 throw rugs） |
| **Bone density** | FRAX → DEXA | Bisphosphonate（secondary injury prevention）|

### 3.1 Vision 关键 contributor 机制（Q107171 核心）

```
Age-related decline
  ├─ Proprioception ↓
  └─ Balance ↓
       ↓
依赖 visual input ↑
       ↓
若 visual acuity ↓（cataract / macular degen）
       ↓
Fall risk substantially elevated
```

- **暗处 / 夜起厕所 fall 高发** ← cataract / macular degen 减 dark adaptation
- Vision 是 **可纠正性强**的 contributor — corrective eyewear / cataract surgery 效益高
- USMLE 反射：post-fall + 老人 + 没做 vision screening → **优先 visual acuity testing**

### 3.2 Get-up-and-Go Test 细节（Q107153 + Q107154 双锚）⭐⭐

- Patient rises from chair → walks briskly to end of room → 计时
- **Time + 主观 difficulty 双指标**预测 fall 风险

#### 3.2.1 Get-Up-Go 4 档解读 ⭐（v1.3 增量 — Q107153 锚）

```
让 patient 从 chair 起立 + walk briskly 到房间另一端
    │
    ├─ 自己起立 + 不 ataxic + 走得稳 → 低 risk
    │
    ├─ Pushes with arms when rising ⭐⭐⭐
    │     → 下肢 strength / muscle imbalance 阳性
    │     → 高 fall risk → supervised strength training 必备
    │     → （即使 walks briskly without ataxia 仍获益）
    │
    ├─ Walks slowly / unsteady gait / 需 cane | walker → 高 risk
    │
    └─ Cannot complete / 中途坐下 → 极高 risk
```

#### 3.2.2 Q107153 经典 Stem 解构

| Stem 细节 | 解读 | 干扰 vs decisive |
|---|---|---|
| HTN + T2DM + post-MI 2 yr | Chronic disease 多 = fall risk ↑ | **Decisive**（识别 高 risk 患者）|
| Independent ADL + walks dog 15 min/day | 仍 ambulatory → 不需 SNF / hospital | Decisive（排除 institution）|
| **Pushes with arms when rising** ⭐ | **Gait/muscle imbalance 阳性** | **Decisive** → primary 答案是 supervised strength training |
| Walks briskly without ataxia | 不矛盾 — strength 差 ≠ ataxia | 不解锁 "不需 strength training"（陷阱）|
| 1 glass wine 6 days/wk | 在 women 推荐限内（≤ 1 drink/day）| 干扰（不增 fall risk）|
| BMI 20 / BP 129/78 无 orthostasis | 排除 vit D deficiency 暗示 + 排除 ACE-I taper | 干扰 |

### 3.3 Primary Prevention 反射链 — Supervised Strength Training ⭐⭐⭐（v1.3 增量 — Q107153 锚）

> [!success] **Supervised exercise / strength training 是减 fall risk 最 evidence-based 干预**
> 适用所有：① 高 risk elderly ② previously fallen ③ Get-Up-Go 异常（pushes with arms / slow gait）

##### Program 必含 3 元素

```
Supervised exercise training =
    ├─ Strength training（下肢力量）⭐
    ├─ Balance training（Tai Chi / single-leg stance）
    └─ Cardiovascular（walking / 自行车）
```

##### 反射钩

| Stem 信号 | 0.5 秒锁 |
|---|---|
| Elderly + chronic disease + **pushes with arms when rising** | **Supervised strength training** ⭐ |
| Elderly + prior fall + 仍 ambulatory | Supervised strength training |
| Elderly + 仅 walks regularly + 无 strength training | **Add supervised strength training**（walking 单独 ≠ strength） |

##### 与 §2 3 层防御联动

| 层级 | Q107153 角色 |
|---|---|
| **Primary** ⭐ | Q107153 = 防 fall 发生 — supervised strength training（本题答）|
| Secondary | Q107154 = 减 fall 后 hip fracture — bone density 优化 |
| Tertiary | PT / OT rehab 已 fallen 后 |

---

### 3.4 Vitamin D Supplementation 严格条件 ⭐⭐（v1.3 增量 — Q107153 反 pattern 钓饵）

| 条件 | 是否补 Vit D for fall prevention |
|---|---|
| **Vitamin D insufficiency < 10 ng/mL**（测过低）| ✓ 补 |
| **Prior fall history** | ✓ 补 |
| **Empiric for all elderly**（无以上条件）| ❌ **不**补（无证据 fall risk 减少）|

> [!danger] 反 pattern：Empiric Vit D for elderly
> "老年人都要补 vit D 防 fall" = ❌ **错** — 只有 **insufficiency < 10 或 prior falls** 才补。
> Q107153 stem 没测 vit D 水平 + 无 prior fall → ❌ 选 A empiric vit D 错。

---

### 3.5 Antihypertensive Taper in Fall Risk ⭐⭐（v1.3 增量 — Q107153 ACE-I 边界）

##### Antihypertensive 处理决策

| 情况 | 是否 taper |
|---|---|
| **Orthostasis 阳性** + 高 fall risk | ✓ 考虑 taper / 换药 |
| **正常 BP + 无 orthostasis** + 高 fall risk | ❌ **不 taper**（除非有 specific 适应症）|
| **Post-MI / HFrEF / CKD** 适应症 + 无 orthostasis | ❌ **绝不 taper**（ACE-I 是 mortality benefit 必备）⭐ |

##### Q107153 ACE-I 决策

- BP 129/78 + 无 orthostasis + **post-MI 2 yr**（ACE-I mortality benefit 强适应症）→ ❌ C 错（taper ACE-I）
- Even 高 fall risk → ACE-I 在 post-MI 不停（mortality risk > fall risk）

##### 反 pattern：见 elderly + fall → 反射 taper 所有 antihypertensive ⚠️

错 — 要看 **3 个 gate**：
1. 有 orthostasis 阳性吗？（无 → 不 taper）
2. 有 post-MI / HFrEF / CKD 适应症吗？（有 → 即使 fall risk 高仍不 taper）
3. BP 是否 well-controlled within target？（well-controlled + 无 orthostasis → 不 taper）

---

### 3.6 Education vs Exercise 边界（v1.3 增量 — Q107153 反 pattern）

> [!warning] **Active exercise > Passive education**
> "Patient handout describing fall prevention strategies" = **less effective** than supervised exercise training。
> Education 仅作为 exercise + multifactorial intervention 的**辅助**，不是单独 primary 答案。

## §4 Screening Tools 的 PPV 陷阱（Q107151 ⭐）

> [!warning] 高 pretest probability + 80% classified as high-risk → 工具失效

| 现象 | 解读 |
|---|---|
| 高 pretest probability（住院老人 fall rate 已高）| 任何 screening tool 的 **PPV 天然低** |
| 工具把 **80%** 患者 classify 为 high-risk | universal alert 失去 discrimination |
| Actual fall rate **11.5 / 1000** | base rate 很低 |
| 结果 | tool 无法区分谁真的需要 tailored intervention |

### 4.1 Bayes 视角（为什么 PPV 低）

```
PPV = TP / (TP + FP)

高 pretest probability + 80% classify high-risk
→ FP 极大（很多 not-fall 患者也被 classify）
→ PPV ↓↓
```

### 4.2 临床含义

- **当 80% 都贴 high-risk 腕带 → 等于谁都没 high-risk**（alert fatigue）
- 解决 = **tailored individual risk factor 干预**（不是 universal label）
- ⭐ Q107151 (Item 1) 的 80% 数据是 Q107152 (Item 2) 的 decisive context

## §5 Tailored Prevention Strategy（Q107152 ⭐）

> [!success] **个体化 risk factor 干预 > universal protocol**

| 反 pattern | 为什么错 |
|---|---|
| Bed alarm / call alarm（universal）| 80% 患者贴标签 = 失效；严重 injury 患者无法激活 alarm |
| Universal bed rest | 反而 ↑ deconditioning + ↑ fall risk |
| 全部贴 high-risk wristband | alert fatigue + lose discrimination |
| AMBOSS 推荐通用 fall prevention bundle | 个性化才有效（每患者 risk factors 不同）|

### 5.1 Tailored Approach 实例

| 患者 risk factor | 个性化干预 |
|---|---|
| 视觉障碍 | Visual acuity testing + corrective eyewear |
| 步态不稳 | PT balance training |
| 药物 (BZD / antihypertensive) | Deprescribe / dose adjust |
| 家居障碍 | OT home safety assessment |
| Bone density 低 | Bisphosphonate |
| 夜间起厕所 | Bedside commode + night light（个性化 home safety）|

## §6 Secondary Prevention = Bone Density 优化（Q107154 ⭐⭐⭐）

> [!danger] **Reduce SERIOUS fall-related INJURY = Reduce hip fracture = Optimize bone density**

### 6.1 Osteoporosis 评估 + 治疗阶梯

| 工具 | 用途 | 触发 |
|---|---|---|
| **FRAX** | 10-year major osteoporotic fracture risk 评估（基于 clinical risk factors）| Postmenopausal women / men 50+ + fall risk |
| **DEXA** | 骨密度 T-score 测量；osteoporosis 诊断（T ≤ -2.5）| **全 65+ women = USPSTF 明确推荐**；70+ men 属 NOF/ISCD（USPSTF 对男性未明确推荐，别在"USPSTF 立场"题上踩） |
| **Bisphosphonate**（alendronate / risedronate / zoledronic acid）| Osteoporosis 一线治疗 | **10 年髋部骨折 ≥3% 或 major osteoporotic 骨折 ≥20%**（FRAX）或 T ≤ -2.5 |

### 6.2 决策树

```
Post-fall 老人
  ├─ Primary prevention 已实施（exercise + home safety + medication review）
  └─ 问"额外 reduce serious injury"
       ↓
  Bone density 优化（FRAX → DEXA → bisphosphonate if 髋≥3% / major≥20%）
       ↓
  减 hip fracture
       ↓
  减 fall 后 mortality + morbidity
```

## §7 反 Pattern 黑名单（4 联 + 扩展）⭐⭐⭐

> [!danger] **见到这些选项 = 0.3 秒排除**

| 反 pattern | 为什么无效 / 反向 |
|---|---|
| **Call alarm to summon assistance** | 严重 fall injuries（LOC / limb injury / fracture）患者**无法激活** alarm → 不减 outcomes（Q107154 ⚠️ 已踩 — 升 🔴）|
| **Bed alarm + universal** | 80% 都贴 = 失效；不减 fall outcomes（Q107152 已踩）|
| **Wearable hip protector** | Multiple studies 显示**无 fracture risk 减少**（依从性极差，不舒服不愿穿）|
| **Discontinue anticoagulation 防 SDH** | 错！Subdural hematoma from fall 较 rare；AF 不抗凝的 stroke 风险 ≫ fall-related head injury 风险 |
| **Cane / walker for fear of fall** | 反而 **higher fall risk**（被自己 device 绊倒）— 除非有明确 mobility 限制 + PT 训练后使用 |
| **Motorized scooter** | Assistive device 应在 multifactorial assessment **之后** identify；无 training 增 fall risk |
| **Direct to assisted living / facility** | 跳过 correctable contributors；**保 independence 是首要** |

## §8 关键铁律（5 大）⭐⭐⭐

### 8.1 AF + Fall = 不停抗凝 ⭐
- **Default**：继续抗凝（stroke 风险 ≫ fall 出血风险）
- **例外**：反复 fall + 明确 head trauma history → 重新评估（罕见停药）
- CK stem 看到 "AF + fall risk + anticoagulant" → **不要选 stop anticoagulation**

### 8.2 Orthostasis 定义 ⭐
- Sitting → Standing → BP 差 **≥20 mmHg systolic 或 ≥10 mmHg diastolic**
- Q107171 stem: 142/87 → 141/79 = **差 1/8** = **NOT orthostasis** → 不停 ACE
- 反 pattern：见 "老人 + 抗压药 + fall" 自动选 stop ACE → 错；先查 orthostasis 定义

### 8.3 保 Independence 老人 Disposition 原则
- **首要**：preservation of independence + autonomy
- **顺序**：先 screen correctable contributors → 修正 → 失败才升级 disposition
- 反 pattern：直接送 assisted living / facility = 跳过 correctable + 违反 autonomy

### 8.4 Vision Priority ⭐
- Multifactorial 7 维中 vision 是**最可纠正**且 evidence 最强
- 反射：post-fall 没做过 visual acuity → 优先 vision testing

### 8.5 Bone Density = Serious Injury 唯一答案 ⭐
- "Reduce SERIOUS fall-related injury" = 永远是 **bone density 优化**
- 不是 call alarm / 不是 hip protector / 不是 stop anticoag

## §9 解题反射速查 ⭐

```
Stem 关键词 → 0.3 秒锁的答案
─────────────────────────────────────────────
"Reduce SERIOUS fall-related injury"     → Bone density (FRAX/DEXA/bisphosphonate)
"Multifactorial fall risk assessment"    → Vision / Hearing / Proprioception / etc
"Post-fall, no orthostasis"              → 不停 ACE，继续 multifactorial
"老人 + 80% high-risk classify"          → tool PPV 低，反 alert fatigue
"老人 + AF + fall + anticoagulant"       → 不停抗凝
"老人 + 历史 multiple falls + 独居"      → 先 screen correctable，不送 facility
"老人 + 夜起 / 暗处 fall"                → cataract / macular degen → vision
"Bed alarm / call alarm / hip protector" → 0.3 秒排除（反 pattern）
"Universal protocol (80% all high-risk)" → tailored 优于 universal
```

## §11 高频干扰项分析速查表

| 干扰项类型 | 例子 | 为什么错 |
|---|---|---|
| Alarm 类设备 | call alarm / bed alarm / wristband | 严重 injury 无法激活 + universal 失效 |
| 限制患者活动 | bed rest / restraint | 反而 ↑ deconditioning + fall risk |
| 设备无 assessment | motorized scooter / cane 默认配 | 应在 multifactorial 后；无 training ↑ fall |
| 升级 disposition | assisted living / SNF | 跳过 correctable；违 autonomy |
| 停必要药物 | stop ACE / stop anticoag | 没 orthostasis / stroke 风险 ≫ SDH |
| 通用 bundle | universal protocol / 80% all high-risk | tailored 才有效 |
| Hip protector | wearable padding | 依从性差 → no evidence |
| Holter / Carotid duplex | average-risk faller | 无 syncope / palpitations 不 indicate |

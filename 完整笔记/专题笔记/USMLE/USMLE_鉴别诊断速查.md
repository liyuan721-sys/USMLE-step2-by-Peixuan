## Chest Pain 胸痛

**最常见病因**（按频率）：
1. GERD / esophageal spasm（最常见门诊主诉）
2. Musculoskeletal / costochondritis（按压可复制）
3. Stable angina / ACS
4. Anxiety / panic
5. Pericarditis

**不能漏诊（life-threatening / can't miss）**：
- STEMI / NSTEMI
- Aortic dissection
- Pulmonary embolism (PE)
- Tension pneumothorax
- Cardiac tamponade
- Esophageal rupture (Boerhaave)

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| STEMI | 压榨痛 + 放射至左臂/下颌 + 大汗 + ST↑ | 12-lead ECG + troponin | PCI < 90 min（无 PCI 则溶栓 < 30 min） |
| Aortic dissection | 撕裂样痛背放射 + **双臂 BP 差 ≥ 20 mmHg** + 纵隔增宽 | CT angiogram（稳定）/ TEE（不稳定） | A 型→手术；B 型→β-blocker 降 HR/BP |
| PE | 突发胸痛 + SOB + 长途飞行/术后/OCP + tachycardia + 低氧 | CTPA（Wells 高）/ D-dimer（Wells 低） | LMWH / DOAC；不稳定→thrombolytics |
| Tension PTX | 单侧呼吸音消失 + 气管偏移 + JVD + 低血压 | 临床诊断（不等 CXR） | 立即针刺减压第 2 肋间锁中线 |
| Cardiac tamponade | Beck 三联 (低血压 + JVD + 心音遥远) + pulsus paradoxus | Bedside echo | Pericardiocentesis |
| Pericarditis | 锐痛随**前倾减轻、卧位加重** + 弥漫 ST↑ + PR↓ | ECG + echo | NSAIDs + colchicine |
| GERD | 烧灼感 + 餐后/平卧加重 + 缓解于 PPI 试验 | 临床；难治→EGD | PPI 8 周 + 生活方式 |
| Boerhaave | 剧吐后剧烈胸痛 + 皮下气肿 + 纵隔气肿 | Gastrografin esophagram | 手术修补 + 广谱抗生素 |

**CK 高频陷阱**：
- BP 双臂差 ≥ 20 mmHg + 撕裂痛 + 纵隔宽 → **dissection**（先 CT，不要先给抗血小板！）
- MI 后 2-7 天突发新心尖收缩期杂音 + 急性肺水肿 → 急性 MR（papillary muscle rupture），不是 reinfarct
- 胸痛 + ST↑且不局限于某一冠脉支配区 + PR 压低 → pericarditis，不要送 cath lab
- 年轻瘦高男性突发单侧胸痛 + 呼吸音↓ → 自发性 PTX；不需要张力性那套抢救

## Shortness of Breath / Dyspnea 呼吸困难

**最常见病因**：
1. CHF exacerbation
2. COPD / asthma 急性加重
3. Pneumonia
4. PE
5. Anxiety / hyperventilation

**不能漏诊**：
- PE
- Tension PTX
- Acute pulmonary edema (cardiogenic / flash)
- Anaphylaxis / upper airway obstruction
- ARDS

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| 急性 CHF | 端坐呼吸 + S3 + 双肺啰音 + JVD + Kerley B | CXR + BNP + echo | IV furosemide + nitrates + O2；CPAP |
| COPD 急性加重 | 长期吸烟 + 桶状胸 + 呼气延长 + 痰↑变脓 | ABG + CXR | O2 (目标 SpO2 88-92%) + 支扩 + 类固醇 + 抗生素 |
| Asthma 急性 | 喘鸣 + PEF↓ + 哮鸣 + 助呼肌 | PEF / Spirometry | SABA + ipratropium + 全身类固醇 |
| PE | 突发 SOB + 单侧腿肿 + tachycardia + 低氧 | CTPA / V-Q | LMWH/DOAC；不稳→thrombolytics |
| Pneumonia | 发热 + 咳脓痰 + 单侧啰音 + 实变 | CXR + CBC | 经验性 abx (CAP 用 amox/macrolide 或氟喹) |
| ARDS | 危重病人 + PaO2/FiO2 ≤ 300 + 双侧 infiltrate + 非心源 | CXR + 排除 CHF | 低潮气量通气 6 mL/kg + PEEP |
| Anaphylaxis | 蜂叮/食/药后数分钟 + 喉头水肿 + 风团 + 低血压 | 临床 | IM epinephrine 0.3-0.5 mg 大腿外侧 |

**CK 高频陷阱**：
- COPD 给 O2 不要追求 SpO2 ≥ 95%（去 hypoxic drive 致 CO2 潴留），目标 88-92%
- 突发 SOB + 单侧腿肿 + tachycardia 即使 D-dimer 阴性也要警惕 PE 临床概率
- 心衰 EF 正常≠没有 HF（HFpEF）；老高血压女性 SOB + S4 + LVH 想到

## Cyanosis 紫绀

**最常见病因**：
1. 肺源性（肺炎 / COPD / asthma 急性 / PE / ARDS）
2. 心源性（R→L shunt：ToF / Eisenmenger / 重度肺高压）
3. 静脉淤滞（休克 / 低心排）
4. **异常 Hb**（MetHb / CarboxyHb / Sulfhemoglobin）

**不能漏诊**：
- Methemoglobinemia（婴儿井水 / Benzocaine / Dapsone）
- CO 中毒（樱桃红 + 假性高 SpO₂）
- 心源性 shunt 反转（Eisenmenger）

**关键鉴别要点**：

| 类型 | SpO₂ 表现 | 吸 100% O₂ | 血液颜色 | Best initial test | Next step |
|:---|:---|:---|:---|:---|:---|
| 肺源性 | **真性↓** | **改善** | 暗红 | CXR + ABG | 治原发病 |
| 心源性（R→L shunt） | **真性↓** | **不改善** | 暗红 | Echo + bubble study | 心导管 / 修补 |
| **MetHb** | **卡 ~85%**（不动） | **不改善** | **巧克力棕** ⭐ | **Co-oximetry MetHb%** | Methylene Blue IV（G6PD → Vit C） |
| **CO 中毒** | **假性 > 95%** | 无意义 | **樱桃红** | **Co-oximetry CO-Hb%** | 100% O₂ → 重症 HBO |

**CK 高频陷阱**：
- 紫绀 + SpO₂ 顽固 85% + 吸 O₂ 不改善 + 巧克力棕血 → **MetHb**；下一步 **co-oximetry**，治疗 **Methylene Blue**（G6PD → Vit C）
- 紫绀 + SpO₂ "正常" + 樱桃红 + 共室人同病 → **CO 中毒**，普通 SpO₂ 会骗你
- **Saturation gap**（SpO₂ vs co-oximetry SaO₂ 差 > 5%）= MetHb / CarboxyHb / Sulfhemoglobin 三选一
- 婴儿 4 月龄 + 农村井水冲奶 + 口周紫绀 = 经典 MetHb stem，机制 = 高 NO₃⁻ + 胃酸低 + 肠菌还原
- Benzocaine 咽喉局麻（TEE / 内镜）术中突发紫绀 = 医源性 MetHb（高频）
- **CO 反直觉**：CO 中毒患者通常**不紫绀**反樱桃红（CO-Hb 比 Oxy-Hb 还红），SpO₂ 假性 > 95% — 别因为 SpO₂ 正常或皮肤不紫就排除 CO。冬季取暖 / 火灾 + 全家症状 = 立刻查 co-oximetry CO-Hb%。
- **火灾双中毒**：CO + Cyanide 同时考虑；首选 Hydroxocobalamin + 100% O₂，**避免单用 Sodium nitrite**（会再造 MetHb 雪上加霜）
- 详细机制 / 阈值 / G6PD 例外 → [[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全]] 5.6 Methemoglobinemia / 5.7 异常 Hb 双毒终极对照

## Epigastric Abdominal Pain 上腹痛

**最常见病因**：
1. PUD / gastritis
2. GERD
3. Gallbladder disease (cholelithiasis, cholecystitis)
4. Pancreatitis
5. MI（下壁 MI 可仅表现上腹痛）

**不能漏诊**：
- Inferior wall MI
- Pancreatitis (severe / necrotizing)
- Perforated peptic ulcer
- Ruptured AAA
- Mesenteric ischemia

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| PUD | 餐后痛 (gastric) / 空腹夜间痛 (duodenal) + NSAID/H. pylori | EGD + H. pylori test | PPI + 根除 H. pylori (三联) |
| Acute cholecystitis | RUQ/上腹痛餐后 + Murphy 征 + 发热 | RUQ US（胆囊壁厚、周围液） | NPO + IV fluids + abx + cholecystectomy |
| Pancreatitis | 持续上腹痛放射至背 + 屈曲减轻 + 酒/胆 + 脂酶↑ ≥ 3× | Lipase + CT 仅评严重度 | NPO + IV fluids（积极）+ 镇痛；ERCP 若胆源 |
| Perforated PU | 突发剧烈"刀割"上腹痛 + 板状腹 + 膈下游离气 | 站立 CXR / CT | 手术修补 + abx |
| Inferior MI | 上腹痛 + 大汗 + 恶心 + ST↑ II/III/aVF | ECG + troponin | 同 STEMI 流程 |
| Mesenteric ischemia | 老年 + AF + "pain out of proportion" + 乳酸↑ | CT angiogram | 抗凝 + 紧急血运重建 |

**CK 高频陷阱**：
- 任何 ≥ 50 岁上腹痛，先做 ECG，别一上来就 CT —— 下壁 MI 经常被当胃炎处理
- 胰腺炎 → CT 不是初始检查，初始是 lipase；CT 仅用于评估 necrosis / 并发症 ≥ 48-72h
- 上腹痛 + 黄疸 + 寒战 = Charcot 三联 → 化脓性胆管炎，需 ERCP 紧急减压

## Lower Abdominal Pain 下腹痛（广义）

**最常见病因**：
1. UTI / cystitis
2. Gastroenteritis
3. Constipation
4. PID / endometriosis（生育期女性）
5. IBS

**不能漏诊**：
- Ectopic pregnancy（任何生育期女腹痛先 β-hCG！）
- Ovarian torsion
- Testicular torsion（男）
- AAA rupture（老年）
- Appendicitis / diverticulitis
- Mesenteric ischemia

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Ectopic | 停经 + 单侧下腹痛 + 阴道点滴出血 + β-hCG (+) | β-hCG + TVUS | Methotrexate（稳定/小）/ surgery（破裂） |
| PID | 性活跃 + 双侧下腹痛 + 宫颈摇摆痛 + 发热 + 分泌物 | 临床诊断 + GC/CT NAAT | Ceftriaxone + doxy + metronidazole |
| UTI | 尿频尿急尿痛 + 耻骨上压痛 + 无系统症状 | UA + dipstick | 经验性 nitrofurantoin / TMP-SMX 3-5 天 |
| Constipation | 排便习惯改变 + 硬便 + 腹胀 | 临床 + KUB | 纤维 + osmotic laxative |
| IBS | 腹痛 + 排便缓解 + 排便习惯改变 ≥ 3 个月，无报警征 | Rome IV 临床 | 饮食 + 抗痉挛药 / SSRI |

**CK 高频陷阱**：
- 生育期女腹痛三件套：β-hCG → UA → US。漏 β-hCG 是经典扣分点。
- 老 + 突发剧烈腰背痛 + 搏动包块 + 低血压 → 直接 OR / bedside US，不要 CT

## RLQ Abdominal Pain 右下腹痛

**最常见病因**：
1. Acute appendicitis
2. Ovarian / tubal pathology（cyst, torsion, PID, ectopic）
3. Mesenteric adenitis（儿童病毒后）
4. Crohn's ileitis
5. Renal colic / UTI

**不能漏诊**：
- Appendicitis with perforation
- Ectopic pregnancy（女）
- Ovarian torsion
- Testicular torsion referred（男）
- Cecal volvulus / perforation

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Appendicitis | 脐周→RLQ 迁移 + 厌食 + McBurney 压痛 + Rovsing | CT abdomen / pelvis（孕 → US 或 MRI） | Appendectomy |
| Ovarian torsion | 突发单侧锐痛 + 恶心 + 卵巢囊肿史 + 可摸包块 | TVUS + Doppler | 紧急 detorsion 手术 |
| Ectopic | 停经 + 点滴出血 + β-hCG (+) | β-hCG + TVUS | MTX 或 surgery |
| Crohn ileitis | 慢性腹泻 + 体重↓ + 瘘 + RLQ 包块 | Colonoscopy + biopsy | 5-ASA / 生物制剂 |
| Renal colic | 阵发腰腹痛放射腹股沟 + 血尿 + 无压痛 | CT non-contrast / US | 镇痛 + α-blocker + 滤尿 |

**CK 高频陷阱**：
- 孕妇 RLQ 痛 → 首选 US 或 MRI，不要 CT（辐射）
- 儿童病毒前驱 + RLQ 软痛 + 无 rebound + 已发热 → mesenteric adenitis（自限）, 不立即开刀
- 阑尾炎 CT 已确诊就开刀，不要等 "复查"

## LLQ Abdominal Pain 左下腹痛

**最常见病因**：
1. Diverticulitis（最经典）
2. Constipation
3. UTI / cystitis
4. IBS
5. Renal colic

**不能漏诊**：
- Perforated diverticulitis with peritonitis
- Sigmoid volvulus
- Colon cancer with obstruction / perforation
- Ovarian torsion / ectopic（左）
- AAA rupture

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Diverticulitis | ≥ 50 岁 + LLQ 痛 + 发热 + 便秘 + WBC↑ | CT abdomen + contrast | 单纯型：abx (cipro + metro) + 流食；脓肿→引流；穿孔→手术 |
| Sigmoid volvulus | 老人 + 腹胀 + KUB 显示"咖啡豆征" | KUB / CT | Endoscopic detorsion；复发→sigmoidectomy |
| Colon cancer | 老 + 体重↓ + 便血 + 缺铁性贫血 + 排便改变 | Colonoscopy + biopsy | 手术 + 分期 |
| Ischemic colitis | 老 + AF/动脉粥样 + LLQ 痛 + 便血 | CT + colonoscopy | 支持 + abx；坏死→手术 |

**CK 高频陷阱**：
- 首次诊断的 diverticulitis 抗生素治愈后 6 周复查 colonoscopy（排除穿孔时漏诊的 cancer）
- Diverticulitis 急性期**不做 colonoscopy**（穿孔风险）—— 先 CT

## Headache 头痛

**最常见病因**：
1. Tension-type headache（最常见）
2. Migraine
3. Medication-overuse headache
4. Cluster headache
5. Sinus headache

**不能漏诊**：
- SAH (subarachnoid hemorrhage)
- Bacterial meningitis
- Brain tumor / mass with ICP↑
- Giant cell arteritis (GCA, > 50 岁)
- Acute angle-closure glaucoma
- Pre-eclampsia / eclampsia（孕妇）
- CO 中毒

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Migraine | 单侧搏动 + 畏光畏声 + 恶心 + ± aura | 临床 (POUND) | 急性：triptan + NSAID；预防：β-blocker / topiramate |
| Tension-type | 双侧"带状"压迫 + 无恶心 + 应激相关 | 临床 | NSAID + 解压 |
| Cluster | 单侧眶周剧痛 + 流泪鼻塞 + 男 + 季节性 | 临床 | 100% O2 + sumatriptan SC；预防 verapamil |
| SAH | "Thunderclap" 最严重一生 + 颈强 + LOC | Non-contrast CT 头颅 → 阴性做 LP（黄变） | Nimodipine + 神经外科 (clip/coil) |
| Bacterial meningitis | 发热 + 颈强 + AMS + Kernig/Brudzinski | LP（先给 abx + 类固醇若延迟） | Ceftriaxone + vancomycin + dexamethasone |
| GCA | > 50 + 颞动脉痛压痛 + 颌跛行 + 视觉变化 + ESR↑↑ | ESR + temporal artery biopsy | **立即** 大剂量 prednisone，不等活检 |
| 急性闭角型青光眼 | 单侧眶痛 + 红眼 + 瞳孔中扩 + 看灯虹晕 | 测眼压 (IOP) | Acetazolamide + 局部 timolol + pilocarpine |
| Pre-eclampsia | 孕 > 20 周 + 头痛 + 上腹痛 + BP↑ + 蛋白尿 | BP + 尿蛋白 + LFT/Plt | Mg sulfate + 控压 + 终止妊娠 |

**CK 高频陷阱**：
- 老人 + 新发头痛 + 颌跛行 + ESR > 50 → GCA。**先给类固醇再活检**（否则失明！）
- "一生最严重的头痛" 即使 CT 阴性也要 LP；超过 6h 非增强 CT 敏感度↓
- 孕 > 20 周头痛 + BP↑ → pre-eclampsia / eclampsia，不是 migraine
- 单侧红眼 + 瞳孔半扩 + 头痛 + 恶心 → 别误诊偏头痛，做眼压

## Syncope 晕厥

**最常见病因**：
1. Vasovagal / neurocardiogenic（最常见）
2. Orthostatic hypotension
3. Cardiogenic (arrhythmia, structural)
4. Situational (cough, micturition, defecation)
5. Carotid sinus hypersensitivity

**不能漏诊**：
- VT / VF / heart block
- HCM（年轻运动员）
- Aortic stenosis（老人劳力性晕厥）
- PE
- Aortic dissection
- SAH
- Long QT / Brugada

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Vasovagal | 站立久/抽血/疼痛 + 前驱恶心潮热 + 快速恢复 | 临床 + ECG (排) | 教育 + 避免诱因 + 反压动作 |
| Orthostatic | 起身后晕 + BP 立位下降 ≥ 20/10 + 药物/脱水 | Orthostatic VS | 补液 + 调药 + 弹力袜 |
| Cardiogenic | 无前驱 + 劳力中晕 + 心脏病史 + 异常 ECG | ECG → echo → telemetry/Holter | 治原因（pacemaker / ICD / 瓣膜修） |
| HCM | 年轻运动员 + 家族猝死史 + 劳力性晕 + Valsalva 杂↑ | Echo (室壁 ≥ 15 mm) | β-blocker；高危→ICD；避免剧运 |
| AS | 老 + crescendo-decrescendo 杂音放颈 + slow-rising pulse | Echo | 有症状重度 AS → AVR / TAVR |
| PE | 长途/术后/OCP + SOB + 单腿肿 | CTPA | 抗凝 |

**CK 高频陷阱**：
- 劳力中晕厥 = 红旗（HCM / AS / 致心律失常），不是 vasovagal
- ECG 看 QT 间期、δ wave (WPW)、Brugada 形态 —— 这些是 syncope 第一道筛
- 起身晕 + Parkinson 样症状 + 自主神经表现 → MSA / 多系统萎缩，不只是普通 orthostatic

## Altered Mental Status 神志改变

**最常见病因**：
1. Infection (UTI / pneumonia / sepsis 老人特别多)
2. Medication / intoxication
3. Metabolic（hypoglycemia, hypoNa, uremia, hepatic）
4. Stroke / TIA
5. Postictal

**不能漏诊**：
- Hypoglycemia（任何 AMS 先测指血糖！）
- Bacterial meningitis / encephalitis
- ICH / SAH / stroke
- Opioid overdose (RR↓, pinpoint)
- Wernicke encephalopathy（thiamine 缺）
- DKA / HHS
- Toxic ingestion (CO, salicylate, TCA)

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Hypoglycemia | 糖尿病 + 胰岛素/磺脲 + 大汗心悸 | 指血糖 | D50W IV / glucagon IM |
| Hepatic encephalopathy | 肝硬化 + asterixis + 触发(GIB/感染/利尿过) + 氨↑ | 临床 + NH3 | Lactulose + rifaximin + 找诱因 |
| HypoNa | Na < 125 + 头痛恶心 + 抽搐 | 血 Na + 渗透压 + 尿渗 | 慢纠（< 8 mEq/24h，防 ODS）；急/严重→3% saline |
| Wernicke | 酗酒 / 营养差 + 三联：眼麻+共济+精神 | 临床 | **IV thiamine 先于葡萄糖** |
| Opioid OD | RR ↓ + 针尖瞳孔 + 昏迷 | 临床 | Naloxone IV / IM |
| TCA OD | 抗胆 + 宽 QRS + 抽搐 + 低血压 | ECG | NaHCO3 IV |
| Delirium 老人 | 注意力↓ + 急性波动 + 感染/术后/药 | 临床 + UA + CBC | 找原因，非药物，避免 benzodiazepine（除 EtOH 戒断） |

**CK 高频陷阱**：
- AMS 急救 ABCDE：A 气道 / B 呼吸 / C 循环 / D **dextrose + naloxone + thiamine** / E 暴露体温
- Wernicke：先 thiamine 再葡萄糖（顺序错可加重）
- 老人术后 AMS 第一想 delirium，不是 dementia；先找诱因（UTI、痛、药、电解质）
- HypoNa 纠正过快 → osmotic demyelination（数日后下肢瘫）

## Hematuria 血尿

**最常见病因**：
1. UTI / cystitis
2. Nephrolithiasis
3. BPH（老男）
4. Glomerulonephritis (IgA, post-strep)
5. Bladder / RCC（老 + 无痛肉眼血尿）

**不能漏诊**：
- Bladder cancer（老 + 吸烟 + 无痛肉眼血尿）
- RCC（侧腹痛 + 包块 + 血尿三联）
- RPGN（crescentic GN，肾功能数天内崩）
- Goodpasture（肺出血 + 肾炎）

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| UTI | 尿痛尿频 + 耻骨上痛 + 脓尿 + 亚硝盐 + | UA | 经验性 abx |
| Stone | 阵发腰腹痛放射腹股沟 + 血尿 + 不能安静 | CT non-contrast (gold) | 镇痛 + α-blocker (tamsulosin) + 滤尿；> 10 mm → 内窥/ESWL |
| IgA 肾病 | 感染**同时**或 1-3 天后肉眼血尿 + 年轻男 | UA + IgA + 肾活检 | ACEI + 控压；急→类固醇 |
| Post-strep GN | 咽痛/皮肤感染**1-3 周后** + 肿 + HTN + 茶色尿 + 补体↓ | ASO + C3↓ | 支持治疗（自限）+ 利尿降压 |
| Bladder cancer | > 50 + 吸烟 + 苯胺暴露 + **无痛肉眼血尿** | Cystoscopy + 尿细胞学 | TURBT + BCG 或 cystectomy |
| RCC | 三联（侧腹痛 + 包块 + 血尿，< 10%）+ paraneoplastic | CT abdomen + contrast | 部分 / 根治性肾切除 |

**关键判别（肾小球 vs 非肾小球）**：
- **RBC casts + 蛋白尿 + 畸形 RBC + 茶色/可乐色** → 肾小球
- **凝血块 + 单一形 RBC + 鲜红血尿** → 输尿管下游

**CK 高频陷阱**：
- ≥ 35 岁无痛肉眼血尿，UA 阳性 → **cystoscopy** 不能省
- 感染**同时**肉眼血尿 = IgA；感染**后 1-3 周** = post-strep。时间窗考点
- Hematuria + 脱发 + 神经性耳聋 + 家族 → Alport（X-linked）

## Upper GI Bleeding 上消化道出血

**最常见病因**：
1. Peptic ulcer disease (PUD)（最常见，NSAID / H. pylori）
2. Esophageal / gastric varices（肝硬化）
3. Mallory-Weiss tear（剧吐后）
4. Gastritis / esophagitis
5. Dieulafoy lesion / vascular malformation

**不能漏诊**：
- Variceal bleed（肝硬化）
- Aortoenteric fistula（既往 AAA 修补 → 大量呕血）
- Bleeding gastric cancer

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| PUD bleed | 上腹痛 + NSAID/H. pylori + 黑便/呕血 | EGD | IV PPI + EGD + 内镜止血 |
| Varices | 肝硬化（蜘蛛痣、腹水、splenomeg）+ 大量呕血 | EGD | Octreotide + ceftriaxone + EGD 套扎；TIPS |
| Mallory-Weiss | 剧呕之后**变成**呕血 (黄→红 → 棕) | EGD | 多自限；内镜止血 |
| Aortoenteric fistula | 既往腹主动脉手术 + "herald bleed" + 大出血 | 紧急 EGD / CTA | 紧急手术 |

**初始管理通用**：
1. ABC + 两条粗静脉 + IV 晶体 + 输血（Hb < 7，活动出血 < 8）
2. PPI 80 mg IV bolus 再 8 mg/h
3. 肝硬化者**经验性**加 octreotide + ceftriaxone（即使未确认 varices）
4. 24h 内 EGD

**CK 高频陷阱**：
- 肝硬化 UGIB 即使不确定是 variceal，也要先 octreotide + abx
- BUN/Cr 比例 ↑↑（> 30）+ 黑便 = 上消化道出血典型
- "Coffee-ground" 呕吐物 = 已止血的旧血

## Lower GI Bleeding 下消化道出血

**最常见病因**：
1. Diverticulosis（最常见大量 LGIB）
2. Angiodysplasia（老 + ESRD / AS 关联）
3. Hemorrhoids（最常见少量便血）
4. Colorectal cancer
5. IBD（UC > Crohn）
6. Ischemic colitis

**不能漏诊**：
- Massive diverticular bleed
- Colon cancer
- Mesenteric ischemia
- Brisk upper GI source 伪装为 LGIB（NG 抽吸阴性不能完全排）

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Diverticular bleed | 老 + **无痛** 大量鲜红/暗红便血 + 多自限 | Colonoscopy | 多保守；持续→clip/电凝；不止→血管栓塞或切除 |
| Angiodysplasia | 老 + ESRD / AS + **慢性少量** + 缺铁性贫血 | Colonoscopy + 可能胶囊内镜 | 内镜烧灼；AS → 换瓣可治愈 (Heyde) |
| Hemorrhoid | 排便后**鲜红**血滴/纸上 + 痒/痛 | 视诊 / 肛门镜 | 纤维 + sitz bath + 局部药 |
| Colon cancer | 体重↓ + 缺铁 + 排便改变 + 老 | Colonoscopy + biopsy | 手术 + 分期 |
| Ischemic colitis | 老 + AF/动脉硬化 + LLQ 痛 + 便血 | CT + colonoscopy | 支持 + abx；坏死→切除 |
| UC | 年轻 + 黏液脓血便 + 里急后重 + ESR↑ | Colonoscopy | 5-ASA → 类固醇 → 生物制剂 |

**CK 高频陷阱**：
- 鲜红便血 + 不稳定生命体征 → 先排除快速上消化道源（NG 抽吸或 EGD）
- ≥ 50 岁任何便血 → 一定要 colonoscopy
- 老 + 主动脉狭窄 + GI 出血 = Heyde syndrome (acquired vWD)

## Monoarticular Joint Pain 单关节痛

**最常见病因**：
1. Gout / pseudogout (CPPD)
2. Osteoarthritis 急性发作
3. Septic arthritis
4. Trauma / hemarthrosis
5. Avascular necrosis (AVN)

**不能漏诊**：
- Septic arthritis（关节破坏 in 24-48h！）
- Gonococcal arthritis（年轻性活跃，先 migratory）
- AVN（股骨头，类固醇/酒/镰状）

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Synovial fluid | Next step |
|:---|:---|:---|:---|
| Septic arthritis | 急性 + 红热肿 + 发热 + 不能负重 | **WBC > 50k**（PMN > 75%）+ Gram stain ± | **关节穿刺 + 培养** → IV abx (vanc + ceftriaxone) + 引流 |
| Gout | 第一跖趾关节（podagra）+ 突发夜间 + 红肿热 | WBC 2-50k + **针状负折光晶体** | NSAID / colchicine / 类固醇；预防：allopurinol（**不在急性期开始**） |
| Pseudogout (CPPD) | 大关节（膝/腕）+ 老 + 软骨钙化 | **菱形正折光晶体** | NSAID / colchicine / 类固醇 |
| OA | 长期、活动加重、休息减轻 | WBC < 2k（非炎症） | 减重 + PT + 对乙酰氨基酚 / NSAID |
| AVN 股骨头 | 类固醇 / 酒 / 镰状 / SLE + 腹股沟痛 | — | MRI；治：减负重，晚期 THA |
| Hemarthrosis | 凝血病 / 抗凝过量 / 血友病 | 出血性 | 压迫 + 抗凝逆转 + 因子替代 |

**Synovial fluid 速查**：

| 类别 | WBC | PMN | 例子 |
|:---|:---|:---|:---|
| 非炎症 | < 2,000 | < 25% | OA, 创伤 |
| 炎症 | 2,000-50,000 | 50% | Gout, RA, SLE |
| 化脓 | > 50,000 | > 75% | Septic |
| 出血 | RBC++ | — | 创伤, 凝血病 |

**CK 高频陷阱**：
- 急性单关节红热肿 + 发热 → **关节穿刺优先**，crystals + septic 可并存
- 急性痛风发作时**不要启动 allopurinol**（会加重）；已用者继续别停
- 年轻人 + 多关节迁移痛 + 腱鞘炎 + 皮疹脓疱 → 淋球菌（DGI），先 ceftriaxone

## Polyarticular Joint Pain 多关节痛

**最常见病因**：
1. Rheumatoid arthritis (RA)
2. SLE
3. Viral arthritis (parvovirus B19, hepatitis B/C)
4. Psoriatic arthritis / spondyloarthropathies
5. Polyarticular gout（晚期）

**不能漏诊**：
- SLE with 肾炎 / CNS / 心肺累及
- Endocarditis with reactive arthritis
- Rheumatic fever（小儿）

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| RA | 对称 + 小关节 (MCP/PIP/wrist) + 晨僵 > 1h + 累 | RF + anti-CCP + ESR/CRP + 手 X 线 | MTX 首选；联合生物制剂 |
| OA | 不对称 + DIP/PIP/第一 CMC + 活动加重 | X 线（关节间隙↓+骨赘） | 减重 + acetaminophen / NSAID |
| SLE | 年轻女 + 颊红 + 光敏 + 关节痛 + 多系统 | ANA → anti-dsDNA + anti-Sm | Hydroxychloroquine + 类固醇 |
| Psoriatic arthritis | 银屑病 + 指甲点蚀 + DIP + 腊肠指 | 临床 + 影像 | NSAID → DMARD / 生物 |
| Reactive arthritis | "Can't see / pee / climb a tree" 尿道 / 结膜 / 关节，HLA-B27 | 临床 + HLA-B27 | NSAID；持续→DMARD |
| Parvovirus B19 | 儿童病例后大人对称小关节痛 + 颊红儿 | IgM anti-parvo | 自限，NSAID |

**CK 高频陷阱**：
- DIP 受累通常**不是 RA**（RA 累 MCP/PIP，**spares DIP**）—— 想 OA 或 psoriatic
- RA 颈椎 C1-C2 半脱位 → 全麻术前**必须**颈椎 X 线评估
- "颊红 / 颧部蝶形" 不与鼻唇沟相连 = SLE；与之相连 = rosacea / 脂溢

## Back Pain 背痛

**最常见病因**：
1. Lumbar muscle strain（最常见）
2. Degenerative disc / OA
3. Disc herniation with radiculopathy
4. Spinal stenosis（老人 + 走路缓解坐位）
5. Osteoporotic compression fracture

**不能漏诊**：
- Cauda equina syndrome
- Spinal epidural abscess（IVDU / DM / 免疫抑制）
- Vertebral osteomyelitis
- Metastatic cancer（乳/前列腺/肺/肾/甲状腺）
- AAA rupture
- MI (referred)
- Pyelonephritis / nephrolithiasis (referred)

**红旗 (red flags)**：
- 年龄 > 50 或 < 20 新发
- 发热、夜痛、体重↓、癌史
- 鞍区麻木、二便失禁、双下肢无力（cauda equina）
- IVDU、近期感染
- 类固醇 / 骨质疏松 + 微外伤
- 神经损伤进展

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Mechanical strain | 抬重物后 + 活动加重 + 无红旗 | 临床 (无影像 < 6 周) | 保持活动 + NSAID + 不卧床 |
| Disc herniation | 单侧坐骨神经痛 + 直腿抬高 + + 皮节感觉↓ | 临床；持续 6 周→MRI | NSAID + PT；术后保守 6 周 |
| Spinal stenosis | 老 + 双下肢"假性跛行"+ **前倾/坐位缓解** | MRI 腰椎 | PT + epidural；严重→减压手术 |
| Cauda equina | 鞍区麻 + 二便失禁 + 双下肢无力 | **紧急 MRI** | 紧急减压手术（数小时窗） |
| Epidural abscess | IVDU/DM + 发热 + 局灶背痛 + 进展神经损 | MRI + ESR/CRP + 血培养 | 紧急减压 + IV vanc + ceftriaxone |
| Vertebral osteo | 慢性局灶痛 + 发热 + ESR↑ + 危险因素 | MRI + 血/活检 | 6 周 IV abx |
| Compression fx | 老 + 类固醇/骨质疏松 + 突发剧痛 | 脊柱 X 线 → MRI | 镇痛 + 抗骨质疏松；vertebroplasty |
| 骨转移 | 已知癌史 + 夜痛 + 体重↓ + 高钙 | MRI + bone scan | 放疗 + bisphosphonate + 治原发 |
| AAA rupture | 老男 + 突发腰/腹痛 + 搏动包块 + 低血压 | Bedside US / CT 若稳定 | 紧急手术 |

**CK 高频陷阱**：
- 急性背痛 + 无红旗 → 不做影像 4-6 周（80% 6 周内好转）
- 鞍区麻 + 二便变化 → 立即 MRI，不要等到第二天
- IVDU + 发热 + 局灶背痛 + WBC↑ → 想 epidural abscess，需要 MRI 不是 CT
- "缺铁性贫血 + 60 岁男背痛 + 体重↓" → 别只查背，做 colonoscopy（colon ca 转移）

## Acute Cough 急性咳嗽（< 3 周）

**最常见病因**：
1. URI / 普通感冒（最常见）
2. Acute bronchitis
3. Pneumonia
4. Postnasal drip
5. Acute exacerbation of asthma / COPD

**不能漏诊**：
- Pneumonia (especially 老/免疫抑制)
- PE（突发 SOB + 咳嗽 + 咯血）
- Foreign body aspiration（儿童 + 突发）
- Anaphylaxis
- CHF / pulmonary edema

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| 病毒性 URI | 鼻塞 + 咽痛 + 低烧 + 自限 | 临床 | 对症（NSAID, 多喝水），**不给 abx** |
| Acute bronchitis | URI 后咳 1-3 周 + 痰 + 无实变 | 临床 | **不给 abx**（除非百日咳/COPD/老） |
| CAP | 发热 + 痰脓 + 实变啰音 + WBC↑ | CXR | Amox/macrolide（健康年轻）；CURB-65 评估 |
| FB aspiration | 儿童 + 突发咳 + **进食/玩玩具时** + 单侧哮鸣 | CXR（呼气片）/ 胸 CT | Rigid bronchoscopy |
| Pertussis | 严重阵发咳 + "whoop" + 呕吐 + < 2 月未疫苗 | PCR / 培养 | Macrolide（azithromycin） |

**CK 高频陷阱**：
- 健康成人急性支气管炎**不给抗生素**，痰黄不代表细菌
- 儿童突发咳 + 进食中 → 异物吸入，做呼气相 CXR 看不对称气体潴留
- 老人肺炎可能 afebrile，只有 confusion + 食欲↓

## Chronic Cough 慢性咳嗽（> 8 周）

**最常见病因**：
1. Upper airway cough syndrome (postnasal drip)
2. Asthma / cough-variant asthma
3. GERD
4. ACE inhibitor
5. Smoker / COPD
6. Bronchiectasis

**不能漏诊**：
- Lung cancer（> 40 + 吸烟 + 咯血 / 体重↓）
- TB（HIV / 移民 / 监狱 / 接触史 + 夜汗 + 体重↓）
- Bronchiectasis
- ILD / IPF
- CHF（"心脏咳嗽"）

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| UACS / postnasal drip | 鼻后流 + 咽部异物感 + 反复清嗓 | 临床 | 1st-gen antihistamine + 鼻喷类固醇 |
| Cough-variant asthma | 干咳 + 夜间 + 运动诱发 + 无哮鸣 | Spirometry + methacholine | ICS + SABA |
| GERD | 烧心 + 平卧加重 + 餐后 | 临床 → PPI 试验 | PPI 8-12 周 + 生活方式 |
| ACEI 咳嗽 | 服 ACEI 数周后开始干咳 | 停药试验 | 改 ARB |
| Lung cancer | 吸烟 + 咯血 + 体重↓ + 锁骨上 LN | CXR → CT 胸 | 活检 + 分期 |
| TB | 接触史 + 夜汗 + 咯血 + 上叶空洞 | CXR + 3 次痰 AFB + NAAT | RIPE × 2m → RI × 4m |

**CK 高频陷阱**：
- ACE inhibitor 咳嗽**与剂量无关**，发生在用药数日到数月，停药 1-4 周缓解
- 慢性咳嗽**初始评估**：CXR + 停 ACEI；再按 UACS → asthma → GERD 顺序经验性治疗
- 体重↓ + 咯血 + 吸烟 → 不要先 PPI，直接 CT 胸

## Fatigue / Weight Loss 乏力/消瘦

**最常见病因**：
1. 抑郁 / 焦虑 / 睡眠不足
2. 甲状腺疾病（亢/减）
3. 贫血（缺铁、慢病、B12/folate）
4. 糖尿病（未控/新发 T1）
5. 慢性感染（TB, HIV, 心内膜炎）

**不能漏诊**：
- Cancer（GI, lung, lymphoma, 胰）
- HIV / TB / 慢性感染
- Addison disease
- 心衰
- Eating disorder（年轻女）

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Hypothyroidism | 怕冷 + 便秘 + 皮肤干 + 体重↑ + 反射延长 | TSH | Levothyroxine |
| Hyperthyroidism | 怕热 + 心悸 + 体重↓ + 多汗 + 反射快 | TSH↓ + free T4↑ | Methimazole / β-blocker；RAIU 定病因 |
| DM (new T1) | Polyuria + polydipsia + 体重↓ + 年轻 | 空腹/随机/A1c | 胰岛素 |
| Anemia | 疲乏 + 苍白 + 心悸 + 黑便/月经多 | CBC + 网织 + 铁四件 | 找原因；缺铁 → 口服铁 + 找 GI source ≥ 50 |
| Addison | 疲乏 + 色素沉着 + 低 Na 高 K + 站立晕 | 8AM cortisol + ACTH; cosyntropin 刺激 | Hydrocortisone + fludrocortisone |
| Depression | 情绪低 + 兴趣丧失 + 睡眠 + 注意↓ + ≥ 2 周 | PHQ-9 | SSRI + CBT |
| Malignancy | 体重↓ > 10% / 6 月 + 局部症状 + 老 | 按系统评估 | 定向影像 / 活检 |

**CK 高频陷阱**：
- 不能解释的体重↓ + 缺铁性贫血 + ≥ 50 → **colonoscopy** 是第一步
- 年轻女疲乏 + 大量月经 → 缺铁，先 ferritin 不是 Hb 单看
- 怕冷 + 便秘 + 反射延长 + 大舌 → 直接 TSH，别先 CBC

## Fever of Unknown Origin (FUO) 不明原因发热

**定义**：体温 > 38.3°C 持续 > 3 周 + 1 周住院评估未明（经典定义）

**最常见病因**：
1. 感染（30-40%，TB、心内膜炎、脓肿、HIV、CMV）
2. 恶性肿瘤（20-30%，淋巴瘤、白血病、RCC、肝转）
3. 非感染炎症（10-20%，GCA / PMR、Still、SLE、IBD）
4. 杂项 + 未明

**不能漏诊**：
- TB（移民/HIV/接触史）
- Endocarditis（杂音 + 栓子）
- Intra-abdominal / pelvic abscess
- Lymphoma
- GCA（> 50 + 颞动脉 + ESR > 50）
- Drug fever
- HIV seroconversion / opportunistic

**初始评估**：
- CBC + diff、CMP、ESR/CRP、UA、血培养 × 3、CXR、HIV、PPD/IGRA
- 影像：CT 胸腹盆
- 进阶：echo（如怀疑 IE）、骨髓活检、PET-CT、temporal artery biopsy

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Endocarditis | 杂音 + Janeway/Osler + IVDU/瓣膜病 | 血培养 × 3 + TTE → TEE | 经验性 vanc + ceftriaxone |
| TB | 接触/移民/HIV + 夜汗 + 体重↓ + 上叶空洞 | CXR + 痰 AFB + NAAT | RIPE 治疗 |
| Lymphoma | 无痛淋巴结↑ + B 症状（汗、热、瘦）+ ↑LDH | LN 活检 | 化疗（R-CHOP / ABVD） |
| GCA | > 50 + 颞痛 + 颌跛 + ESR↑↑ | ESR + 活检 | 立即类固醇 |
| Drug fever | 用新药后 + 嗜酸↑ + 相对缓脉 + 一般状况好 | 停药试验 | 停可疑药 |
| 腹腔脓肿 | 术后 / 阑尾穿孔后 + 持续热 | CT 腹盆 | 引流 + abx |

**CK 高频陷阱**：
- 相对缓脉 (pulse-temp dissociation) → 想 typhoid / drug fever / 中枢热 / 钩端螺旋体
- IVDU FUO → 即使 echo 阴性也要 TEE（右心 IE 可漏诊）
- > 50 + 头痛 + ESR > 50 → 类固醇 first 再活检

## Edema 水肿

**最常见病因**：
1. CHF
2. 肝硬化
3. 肾脏（nephrotic / nephritic）
4. 静脉功能不全 (chronic venous insufficiency)
5. 药物（CCB、NSAID、TZD、激素）
6. 妊娠

**不能漏诊**：
- DVT（单侧）
- Compartment syndrome（创伤后）
- Cellulitis（红热痛）
- 急性 CHF / pulmonary edema
- 心包填塞 / 缩窄性心包炎
- Nephrotic syndrome with PE / 感染

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| CHF | 双下肢凹陷 + JVD + S3 + 端坐 + 啰音 | BNP + echo + CXR | 利尿 + ACEI + β-blocker；减盐 |
| 肝硬化 | 腹水 + 蜘蛛痣 + 黄疸 + 静脉曲张 + Plt↓ | LFT + albumin + US 腹 | 限钠 + spironolactone + furosemide |
| Nephrotic | 蛋白尿 > 3.5 g/d + 低 albumin + 高脂 + 凹陷肿 | 24h 尿蛋白 / UPCR + lipid + albumin | ACEI + 类固醇（按病理）+ 抗凝考虑 |
| Nephritic | 血尿 + RBC casts + HTN + 轻度水肿 | UA + 补体 + ANA / ANCA | 治原发 GN |
| DVT | 单侧腿肿痛 + 危险因素 (术/制动/OCP) | 多普勒 US 下肢 | LMWH/DOAC |
| CCB induced | 服 amlodipine + 双侧踝肿 + 无 HF 证据 | 临床 | 换 / 加 ACEI |
| Lymphedema | 慢性单侧 + 非凹陷 + 癌/术后/丝虫 | 临床 + 淋巴显影 | 压迫 + PT |
| Hypothyroidism | 全身**非凹陷**黏液水肿 + 怕冷 + 反射延长 | TSH | Levothyroxine |

**CK 高频陷阱**：
- Pitting vs non-pitting：非凹陷 → 想 hypothyroid (myxedema)、lymphedema
- CCB 引起踝水肿 → 加 ACEI/ARB 是经典解，单换 CCB 也可
- 突发单侧腿肿 + 危险因素 → DVT，不能仅用 D-dimer 排（高 pretest 直接 US）

## Vaginal Bleeding in Pregnancy 孕期阴道出血

**1st trimester (< 20 周)**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Spontaneous abortion | 阴道出血 ± 痉挛痛 + β-hCG 不升 | TVUS + β-hCG 趋势 | 期待 / misoprostol / D&C |
| Ectopic | 单侧腹痛 + 点滴出血 + β-hCG (+) 但 < discriminatory | TVUS + β-hCG | MTX / 手术 |
| Molar pregnancy | β-hCG > 100k + 子宫**大于孕周** + 葡萄状 + 早期重 preeclampsia | TVUS（雪暴）+ β-hCG | D&E + 系列 β-hCG 监测 1 年；避孕 |
| Implantation bleed | 4-6 周少量 + 无痛 | 临床 + β-hCG | 期待 |

**3rd trimester (≥ 20 周)**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| **Placenta previa** | **无痛**鲜红血 + 软子宫 + 胎盘覆盖颈口 | **TVUS** | **避免阴道检查**！剖宫产 36-37 周 |
| **Placental abruption** | 创伤/HTN/可卡因 + **痛性**血 + 硬张子宫 + 胎心异常 | 临床；US 仅 50% 敏 | 紧急分娩（若母胎不稳） |
| Vasa previa | 破膜后**鲜红**少量 + 胎心快速 deceleration | 产前 TVUS + Doppler | 36 周计划性剖宫产 |
| Uterine rupture | 既往剖宫产 VBAC + 突发剧痛 + 胎心丢 + 胎位↑ | 临床 | 立即开腹 + 复苏 |

**CK 高频陷阱**：
- 妊娠中晚期阴道出血**先 US 再做阴道检查**（previa 一查就大出血）
- 隐性 abruption（无显血但腹痛 + 胎心异常）—— 不能用血量否定
- Rh(–) 母亲 + 任何孕期出血/创伤 → **anti-D 免疫球蛋白**

## Vaginal Bleeding (Non-pregnant) 非孕期阴道出血

**按生命阶段**：

| 阶段 | 最常见 | 不能漏 |
|:---|:---|:---|
| Prepubertal | 异物 / 创伤 / 性虐 / 早熟 | 性虐、肿瘤 |
| Reproductive age | Anovulation (PCOS, 极端体重, 应激)、纤维瘤、息肉、激素避孕 | 妊娠相关、宫颈/子宫癌、coagulopathy |
| Perimenopausal | 排卵障碍 + 息肉 + 纤维瘤 + 早期内膜增生 | 子宫内膜癌 |
| Postmenopausal | **子宫内膜癌（必排）**、萎缩、息肉、HRT | **子宫内膜癌** |

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| PCOS | 月经稀发 + 多毛 + 痤疮 + BMI↑ + LH/FSH > 2 | 临床（Rotterdam 2/3）+ 激素 | 减重 + COC + metformin；想孕→letrozole |
| Fibroids | 经量多 + 痛 + 子宫不规则增大 + 缺铁 | TVUS | 期待 / OCP / LNG-IUD / 切除 |
| Endometrial polyp | 经间出血 + 老/HRT | TVUS + saline-infused sonohysterography | Hysteroscopic polypectomy |
| **Endometrial cancer** | **绝经后出血** + 肥胖/DM/无对抗雌激素 | **TVUS 测 endometrial thickness > 4 mm → 内膜活检** | 全子宫双附件切除 + 分期 |
| Cervical cancer | 性交后出血 + 阴道分泌物 + HPV | Pap + colposcopy | 按分期手术/放化疗 |
| Coagulopathy | 青少年初潮即重月经 + 易瘀 + 家族 | CBC + PT/PTT + vWF panel | 治原发；DDAVP / vWF concentrate |

**CK 高频陷阱**：
- **绝经后任何出血 = endometrial ca 直到证伪**，初始 TVUS + 必要时活检
- 青少年重月经 + 易瘀 → 想 vWD，不只是青春期功能性
- AUB 反复 + 肥胖 → 内膜活检即使年轻（无对抗雌激素 → 增生）

## Pelvic Pain 盆腔痛

**急性**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Ectopic | 停经 + 单侧痛 + 点滴血 + β-hCG (+) | β-hCG + TVUS | MTX / 手术 |
| Ovarian torsion | 突发锐痛 + 恶心 + 囊肿史 + Doppler 血流↓ | TVUS Doppler | 紧急 detorsion 手术 |
| Ruptured ovarian cyst | 排卵期突发 + 自限 + 少量盆腔液 | TVUS | 镇痛 + 期待；大量出血→手术 |
| PID / TOA | 性活跃 + 发热 + 宫颈摇摆 + 包块 | NAAT + 临床 + US | TOA → 住院 IV abx（cefoxitin + doxy） |
| Appendicitis | 脐周→RLQ + 厌食 + WBC↑ | CT/US | Appendectomy |

**慢性**：

| 病因 | 典型 vignette 特征 | 诊断 | 治疗 |
|:---|:---|:---|:---|
| Endometriosis | 经期前后渐重 + 性交痛 + 不孕 + 子宫骶韧带结节 | 临床；金标准 laparoscopy + 活检 | NSAID + COC / GnRH agonist + 手术切除 |
| Adenomyosis | 经量多 + 痛经 + **均匀**增大软子宫 + 围绝经 | TVUS / MRI | NSAID + LNG-IUD；明确→全子宫切除 |
| Fibroids | 经量多 + 不规则增大子宫 + 压迫症状 | TVUS | 同前 |
| Chronic PID / adhesions | 慢性盆腔痛 + STI 史 | 临床 + 必要时 laparoscopy | 镇痛 + 抗生素 + 粘连松解 |
| IC / 痛性膀胱综合征 | 尿频 + 充盈痛 + 排尿缓解 + UA 阴 | 排除 + 膀胱镜 | 行为 + amitriptyline + pentosan |

**CK 高频陷阱**：
- TVUS Doppler 显示卵巢血流不能完全排除 torsion（双供血）—— 临床高度怀疑 → 手术
- Endometriosis 不需要 laparoscopy 才能治疗；首选 NSAID + 激素经验性
- PID 任何包块（TOA）→ 住院 IV abx + 影像随访

## Amenorrhea 闭经

**Primary（≥ 15 岁仍未月经 或 ≥ 13 岁无第二性征）**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Constitutional delay | 家族晚熟 + 身材矮 + 骨龄 < 年龄 + 第二性征发育 | 骨龄 | 期待 |
| Turner syndrome | 矮 + 蹼颈 + 主动脉缩窄 / 二叶 + streak 卵巢 + 高 FSH | 核型 45,X | HRT + 心脏 / 听觉随访 |
| Androgen insensitivity | 表型女 + 无毛 + 盲端阴道 + 高 T + 46,XY | 核型 + T | 切除性腺 + HRT |
| Mullerian agenesis | 正常第二性征 + 无 / 短阴道 + 无子宫 + 46,XX + 正常 T | MRI / US | 阴道扩张 / 手术 |
| Imperforate hymen | 周期性下腹痛 + 蓝色突起 + 无月经 | 体检 | 手术切开 |

**Secondary（已有月经 ≥ 3 月不来 / 不规律 ≥ 6 月）**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| **妊娠** | 任何性活跃女继发闭经 | **β-hCG** | 产检 |
| PCOS | 多毛 + 痤疮 + BMI↑ + LH/FSH > 2 | 临床 + 激素 | 同 PCOS 处理 |
| Hyperprolactinemia | 溢乳 + 闭经 + 头痛 + 视野缺损 | Prolactin + MRI sella | Cabergoline / 治原因（停药、甲减） |
| Hypothalamic | 极端运动 / 体重↓ / 应激 + 低 FSH/LH/E2 | 临床 + 激素 | 营养康复 + 心理；HRT 防骨质流失 |
| 早期卵巢功能减退 (POI) | < 40 + 闭经 + 潮热 + FSH↑ | 两次 FSH > 40 | HRT 至自然绝经年龄 |
| Asherman | 子宫粘连后（D&C / 内膜炎）+ 无月经 + 周期感 | Hysteroscopy | 粘连松解 + 雌激素 |

**评估算法**：
1. **β-hCG**（永远第一）
2. TSH + prolactin
3. 若正常 → progestin challenge（撤药出血 → anovulation；不出血 → 雌激素低或流出道）
4. 若 progestin 阴 → 雌孕激素试验
5. 若阴 → 流出道（Asherman、流出道梗阻）
6. 若阳 → FSH/LH 区分中央 vs 卵巢

**CK 高频陷阱**：
- 任何继发闭经**永远先 β-hCG**
- 闭经 + 头痛 + 视野缺损 + 溢乳 → MRI 看 prolactinoma
- 青少年闭经 + 蓝色处女膜膨出 + 周期性下腹痛 → imperforate hymen

## Pediatric Fever 小儿发热

**按年龄分层**：

**< 28 天（neonate, 任何发热都是急症）**：
- 全血、尿、CSF 培养 + 经验性 abx（**ampicillin + gentamicin** 或 + cefotaxime）+ 入院
- 不能漏：GBS, E. coli, Listeria, HSV

**28-90 天**：
- 完整 sepsis workup；按 Philadelphia / Rochester criteria 决定门诊 vs 住院

**3-36 月**：
- 大多病毒；评估 toxic / non-toxic
- UTI 高发；不明发热 → UA + 培养（尤其女、未割包皮男）

**关键不能漏诊**：
- Meningitis（任何年龄）
- UTI（婴儿期 occult）
- Bacteremia / sepsis
- Kawasaki disease (KD)
- Osteomyelitis / septic arthritis
- Appendicitis（学龄）

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| Viral URI | 鼻塞咳嗽 + 自限 + active | 临床 | 对症 |
| AOM | 发热 + 哭闹拉耳 + 鼓膜膨隆/红 | Otoscopy | < 6 月 / 重 → amoxicillin；观察策略可选 |
| UTI | < 2 岁不明发热 + 喂养差 + 呕吐 | 导尿 UA + 培养 | Abx + 影像（RBUS, VCUG 按指征） |
| Kawasaki | **发热 ≥ 5 天** + 结膜不渗出 + 草莓舌 + 颈 LN + 皮疹 + 手足红/脱皮 | 临床 + echo（冠脉瘤） | **IVIG + 高剂量 aspirin**，发病 10 天内 |
| Meningitis | 发热 + 嗜睡 + 颈强（婴儿可不明显）+ 喷射呕吐 + 囟门膨 | LP（先 abx 若延迟）+ 血培 | 经验性 ceftriaxone + vanc + dex（≥ 1 月） |
| Septic arthritis | 关节红肿热痛 + 不能负重 + WBC/CRP↑ | 关节穿刺 | IV abx + 引流 |
| Osteomyelitis | 局灶骨痛 + 发热 + ESR/CRP↑ + Staph aureus 最多 | MRI + 血培 | IV abx 4-6 周 |
| Pneumonia | 发热 + 咳 + 呼吸快 + 实变 | CXR | Amoxicillin（典型）/ azithromycin（非典） |

**CK 高频陷阱**：
- 新生儿（< 28 天）发热**没有 "low-risk"** —— 全部 full septic workup + 入院
- 婴儿 occult UTI 表现仅为不明发热 + 喂养差 → 不查 UA 就是失分
- Kawasaki 必须 echo 看冠脉瘤；IVIG 必须 10 天内（最好 7 天内）
- 阿司匹林**唯一**可以给儿童的情况 = Kawasaki（其他禁，怕 Reye）

## Pediatric Vomiting 小儿呕吐

**按年龄分层最常见**：

**新生儿（0-28 天）**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| **Malrotation + volvulus** | 突发**胆汁性**呕 + 喂养差 + 远端腹胀少 | **Upper GI series**（鸟嘴 / 螺旋） | **紧急手术（Ladd procedure）** |
| Duodenal atresia | 21-三体 + 出生数小时胆汁性呕 + **双泡征** | 腹部 X 线 | 手术 |
| Pyloric stenosis | **3-6 周** + **非胆汁** + **喷射性** + 喂奶后饿 + 橄榄包块 + 低 Cl/低 K 代碱 | US 幽门 | 补液纠 → pyloromyotomy |
| Necrotizing enterocolitis | 早产 + 喂养不耐受 + 血便 + **pneumatosis intestinalis** | 腹平片 | NPO + NG 减压 + abx + 必要时手术 |
| Hirschsprung | 24-48h 不排胎便 + 腹胀 + 呕吐 | 钡灌肠（移行带）+ 直肠活检（无神经节细胞） | 切除受累段 |

**婴儿期 (1 月-2 岁)**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| GERD（physiologic） | 婴儿 + 频繁吐奶 + 生长好 + 无并发症 | 临床 | 增稠喂养 + 抬上半身；自限 |
| **Intussusception** | 6-36 月 + 阵发剧哭 + 抬腿屈膝 + **果酱便** + 腊肠包块 | US（**bull's eye / target sign**） | **空气灌肠**复位；失败/穿孔→手术 |
| Viral gastroenteritis | 呕吐 + 腹泻 + 接触史 + 自限 | 临床 | ORS / 静脉补液 |

**儿童（> 2 岁）**：
- Gastroenteritis、appendicitis、DKA、增加 ICP、毒物、psychogenic

**关键不能漏诊**：
- Malrotation with volvulus（**胆汁性呕的新生儿 → 直奔 upper GI**）
- Pyloric stenosis
- Intussusception
- DKA（新发 T1）
- Increased ICP（VP shunt 故障、肿瘤）
- 中毒

**CK 高频陷阱**：
- **新生儿胆汁性呕吐 = malrotation/volvulus 直到证伪** —— 不要先 US，做 upper GI series
- Pyloric 是**非胆汁性**喷射 + 喂后饿；电解质典型为低 Cl 低 K 代碱
- 阵发哭闹 + 蜷腿 + 果酱便 (晚) → intussusception，US 优先

## Pediatric Rash 小儿皮疹

**伴发热**：

| 疾病 | 典型 vignette 特征 | 标志 / 并发 | Next step |
|:---|:---|:---|:---|
| Measles | 高热 + 3 C (cough, coryza, conjunctivitis) + **Koplik spots** → 头脸→躯干红疹 | SSPE 远期 | 支持 + Vit A；隔离；MMR 暴露后预防 |
| Rubella | 低热 + **耳后/枕 LN** + 粉色 maculopapular 从面→下 | **先天性风疹综合征**（孕） | 支持；孕妇避免接触 |
| Roseola (HHV-6) | 6 月-2 岁 + **3-5 天高热**骤退 → **退热同时**躯干粉疹 | 热性惊厥 | 支持 |
| Erythema infectiosum (parvo B19) | 学龄 + "slapped cheek" + 蕾丝状躯干疹 | 孕妇→ hydrops fetalis；镰状→ aplastic crisis | 支持 |
| Hand-foot-mouth (Coxsackie A) | 口腔溃疡 + 手足红丘疹/水疱 + 低热 | 自限 | 支持 |
| Varicella | "**Dewdrops on rose petal**" 各期皮疹同时 + 痒 + 发热 | Bacterial superinfection、肺炎、脑炎 | 支持；高危→acyclovir |
| Scarlet fever | A 组链感染 + **砂纸样**疹 + 草莓舌 + 环口苍白 | 风湿热、PSGN | Penicillin / amoxicillin |
| Kawasaki | 发热 ≥ 5 天 + 5 条标准 | **冠脉瘤** | IVIG + ASA + echo |
| Meningococcemia | **紫癜/瘀点**快速进展 + 发热 + 休克 | DIC、Waterhouse-Friderichsen | 立即 ceftriaxone + ICU；接触者预防 rifampin/cipro |
| RMSF | 蜱 + 头痛发热 + **手腕脚踝起疹向心** | 致命若漏 | **不等血清学** → doxycycline (任何年龄) |
| HSP (IgA vasculitis) | **可触瘀点下肢/臀** + 关节痛 + 腹痛 + 肾炎 | 肾炎、肠套叠 | 支持 + 监肾；类固醇若严重 |

**无发热常见**：
- Atopic dermatitis（屈侧 + 痒 + 个人/家族特应）
- Contact dermatitis
- Scabies（指间 / 腋下 + 夜痒）
- Tinea / pityriasis

**CK 高频陷阱**：
- 蜱叮 + 发热 + 手腕脚踝起疹 → **直接 doxycycline，不等检验**（RMSF 致命）
- 紫癜瘀点 + 发热不退 → 想 meningococcemia，**先 abx 再 LP**
- 可触瘀点 + 双下肢 + 腹痛 + 蛋白尿 → HSP（IgA），监测肾 6-12 月
- Varicella 同一时间见各期皮疹（vs smallpox 同步）

## Failure to Thrive 生长发育迟缓

**定义**：体重 < 第 3-5 百分位 或 **跨越两条百分位下移**

**病因分类**：

| 类别 | 例子 |
|:---|:---|
| **摄入不足**（最常见）| 喂养困难 / 喂养方式不当 / 心理社会忽视 / 贫困 |
| **吸收不良** | Celiac、囊性纤维化、慢性腹泻、IBD、奶蛋白过敏 |
| **代谢需求↑** | 心衰、慢性肺病、肾衰、甲亢、慢性感染、恶性肿瘤 |
| **遗传 / 内分泌** | 染色体异常、GH 缺乏、先天代谢病、Turner |

**关键评估**：

| 步骤 | 内容 |
|:---|:---|
| 1 | 详细喂养日记 + 心理社会 + 生长曲线（HC/W/L/W-for-L）|
| 2 | 体检（脱水、心音、肝大、特殊面容、神经发育）|
| 3 | 基础实验：CBC、CMP、UA、TSH、celiac（≥ 12 月）、汗 Cl 若 GI/肺症状 |
| 4 | 卡路里量化干预 1-2 周 + 随访体重 |

**CK 高频陷阱**：
- 大多 FTT 是**非器质性**（喂养 / 心理社会）——先详细喂养评估再做实验
- 慢性腹泻 + 油脂便 + 反复肺炎 → 汗氯试验排 CF
- 高体重新生儿急速消瘦 + 喂养差 → 想代谢病 / 先心 / 甲亢

## Limp in a Child 小儿跛行

**按年龄分层**：

**Toddler (1-3 岁)**：toddler's fracture（胫骨螺旋）、transient synovitis、DDH、osteomyelitis
**School age (4-10)**：transient synovitis、Legg-Calvé-Perthes、septic arthritis、osteomyelitis、JIA
**Adolescent (11-16)**：SCFE（肥胖男）、Osgood-Schlatter、stress fracture、osteosarcoma、JIA

**关键鉴别要点**：

| 病因 | 典型 vignette 特征 | Best initial test | Next step |
|:---|:---|:---|:---|
| **Transient synovitis** | 3-10 岁 + URI 后 + 髋痛 + **低烧 / 无 / 轻** + 可负重 | US / X 线 + CBC + CRP | NSAID + 休息；4 周内消 |
| **Septic arthritis** | 急性 + 红热肿 + **不能负重** + 高热 + WBC/CRP/ESR↑↑ | 关节穿刺 | 紧急 abx + 引流 |
| Legg-Calvé-Perthes | 4-10 岁男 + 渐进髋/膝痛 + 跛 + **股骨头缺血坏死** | 髋 X 线 / MRI | 支具 / 手术（保护股骨头） |
| **SCFE** | 肥胖青春期男 + 髋/膝痛 + **外旋畸形** + 屈髋时外旋 | **双髋 frog-leg X 线** | **不负重** + 紧急 in situ 螺钉固定 |
| DDH | 新生儿 + Barlow / Ortolani + 大腿皮纹不对称 | < 6 月 US；> 6 月 X 线 | < 6 月 Pavlik harness；> 6 月 closed reduction |
| Osteomyelitis | 局灶骨痛 + 发热 + ESR/CRP↑ + Staph aureus | MRI + 血培 | IV abx 4-6 周 |
| Osgood-Schlatter | 跳跃运动 + 胫骨结节痛肿 + 青春期 | 临床 | 休息 + ice + NSAID + 拉伸 |

**Kocher 标准**（鉴别 septic vs transient synovitis 髋）：
- 发热 > 38.5°C
- 不能负重
- ESR > 40
- WBC > 12k
- 4/4 = > 99% septic；3/4 ~ 93%；≤ 1 = transient 可能性高

**CK 高频陷阱**：
- 跛行 + 不能负重 + 发热 + ESR/CRP↑ → **关节穿刺**优先，不要先 MRI
- 肥胖青少年膝痛 → 别忘了**髋部**（SCFE 经常 referred 膝痛）→ frog-leg X 线
- Transient synovitis 是排除诊断；高度怀疑 septic 一律穿刺

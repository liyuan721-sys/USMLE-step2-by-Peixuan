# Social Determinants of Health & Health Equity — SDOH 全景

> [!info] 定位
> SDOH = USMLE Step 2 CK / Step 3 **横向高频主题**，跨 Ethics / Biostats / Pediatrics / OB / Family Medicine。本笔记把 5 大 Domain + 4 种 CK stem 模式 + 与 implicit bias 的边界 + interpreter / cross-cultural care 规则一次性扫平。
>
> 与 [[完整笔记/专题笔记/USMLE/USMLE_患者安全与质量改进工具]] §13.3.3（Patient-Associated Risk Factors 含 SDOH 简版）+ §13.4.x（Implicit Bias 3 步管理 + 4 反 pattern）形成 patient-side 与 provider-side 互补。

---

## 一、定义与反直觉关键事实

**SDOH (Social Determinants of Health, 健康的社会决定因素)** = 人**出生 / 成长 / 工作 / 生活 / 衰老**的**非医疗社会环境因素**，对健康结果的影响**占 80%+**。

> [!danger] 反直觉关键事实（公共卫生转向的根本 logic）
> 医疗服务本身只占健康结果的 **~10-20%**；**SDOH + 个人行为占 80%+**。
> 这就是为什么 Healthy People 2030 / CMS / Joint Commission 把 SDOH 提到 health policy 中心 —— "医院里能改的有限，社区里改的多"。

**WHO 定义**：people are born, grow, work, live, and age 的 conditions + 塑造这些 conditions 的更广泛 forces and systems（economic policies / social norms / political systems）。

---

## 二、5 大 Domain — Healthy People 2030 框架 ⭐⭐⭐

USMLE 最常考就是这 5 类。**记忆**：**E·E·H·N·S** — 经济 / 教育 / 医疗 / 邻里 / 社会。

| Domain | 中文 | 内容 | 经典 stem 信号 |
|---|---|---|---|
| ① **Economic Stability** | 经济稳定 | 收入 / 就业 / poverty / 食品安全 / 住房稳定 | "lives below poverty line" / "unemployed" / "food insecurity" / "cannot afford medication" |
| ② **Education Access & Quality** | 教育 | 受教育水平 / **health literacy** / 早教 / language | "did not complete high school" / "limited English proficiency (LEP)" / "low health literacy" |
| ③ **Health Care Access & Quality** | 医疗 access | 保险 / 是否有 PCP / 距离 / 语言服务 / 文化能力 | "uninsured" / "no PCP" / "rural area" / "no translator available" |
| ④ **Neighborhood & Built Environment** | 邻里 / 物理环境 | 住房质量 / 安全 / 交通 / 食物沙漠 / 空气污染 | "food desert" / "unsafe neighborhood" / "no transportation" / "lead paint" / "mold" |
| ⑤ **Social & Community Context** | 社会 / 社区 | 社会支持 / 凝聚力 / discrimination / 监禁史 / **IPV** | "lives alone" / "incarcerated parent" / "social isolation" / "history of discrimination" / IPV |

> [!tip] 记忆 Hook
> **E·E·H·N·S** → "Easy Educational Health Needs Society"
> 一句话锁：经济（钱）→ 教育（识字）→ 医疗（access）→ 邻里（住）→ 社会（关系网）

## 三、SDOH 在 Public Health 框架里的位置

```
Health Outcomes（健康结果）
    ↑
    ├── 医疗服务（~10-20%）
    │     ├── 诊断准确性
    │     └── 治疗 quality
    │
    └── 非医疗（80%+）⭐⭐⭐
          ├── SDOH ⭐⭐⭐
          │     ├── ① 经济稳定
          │     ├── ② 教育
          │     ├── ③ 医疗 access（系统层）
          │     ├── ④ 邻里 / 环境
          │     └── ⑤ 社会 / 社区
          │
          ├── 个人行为（饮食 / 运动 / 烟酒）
          │   ↑ 但行为很大程度被 SDOH 决定
          │
          └── 遗传 / 生物
```

> SDOH 与个人行为不是平行 —— 个人行为很大程度**由 SDOH 决定**（住 food desert → 吃 fast food；上夜班 → 睡眠不足；childcare 困境 → skip preventive care）。所以单独 counsel patient 改变行为 = blame the victim。

## 四、4 种 CK 经典 Stem 模式 ⭐⭐⭐

### 模式 1：SDOH 暴露 → 选 social work / resource referral

> 50 yo 糖尿病患者 HbA1c 反复超标，自述"经常 skip insulin doses 因为**买不起**" → next step?

| 选项 | 评价 |
|---|---|
| ✅ **Social work referral** / financial assistance / patient assistance program | 解决 root cause |
| ❌ "Intensify insulin regimen" | 加药不解决"买不起" |
| ❌ "Counsel on importance of compliance" | **Blame the victim**（CK 高频陷阱）|
| ❌ "Discharge with same regimen" | 不 address SDOH = 必然 readmission |

### 模式 2：Compliance / readmission 问题 → 看 SDOH 根因

> 哮喘儿童反复 ED visits，妈妈"忘记给药" → 进一步问发现住 **mold-infested apartment** + **没有 PCP**

✅ Address housing + 帮忙找 PCP / FQHC（多因素干预）
❌ "Educate mother on medication adherence"（root cause 不是教育）

### 模式 3：Routine screening 决策

> Well-child visit / prenatal visit — 题干给 social history 暴露 SDOH risk → 选 **screen + refer**。

CK 反射：见到题干给 social history 详细信息（住房 / 食物 / IPV / 财务）→ 答案多半涉及 **screen** 该 domain + connect to resource。

### 模式 4：Health disparity 题（最高频）

> 数据显示 minority group readmission ↑ 30% / cancer screening 率低 / opioid prescribing 率差异

CK 答案路径：**Multifactorial** mechanism
- **Patient-side** = SDOH（low literacy / no insurance / transportation barrier）
- **Provider-side** = implicit bias（详见 [[完整笔记/专题笔记/USMLE/USMLE_患者安全与质量改进工具]] §13.4.x）
- **System-side** = structural racism / policy gaps

❌ 单一原因归因 / 单一干预

## 五、SDOH vs Implicit Bias 区分 ⭐⭐⭐（核心边界）

最容易混的两个概念 —— 但工具完全不同。

| 维度 | **SDOH** | **Implicit Bias** |
|---|---|---|
| **在哪一侧** | **Patient-side**（病人所处环境）| **Provider-side**（医生 unconscious bias）|
| **性质** | 社会经济环境因素 | Provider 的 unconscious 偏见 |
| **Stem 信号** | "lives in poverty / food desert / no insurance / LEP" | "providers endorse equity but data shows disparity" |
| **干预** | Social work / community resource / system reform | **IAT** + **group debriefing** + metacognition |
| **错误干预** | Counsel patient / 加药 | Anonymous portal / zero-tolerance / punitive |

> [!warning] 两者经常共存
> 同一道 health disparity 题可能有两条 mechanism。题干暗示哪边，看主语：
> - "**Patient** lives in..." → **SDOH**
> - "**Provider survey** shows..." → **Implicit bias**
> - 两个都给 → multifactorial 答案
>
> Implicit Bias 完整管理（Joint Commission 3 步 + 4 反 pattern）见 [[完整笔记/专题笔记/USMLE/USMLE_患者安全与质量改进工具]] §13.4.x（Q21366 锚定）。

## 六、SDOH 高频干预答案库 ⭐⭐（CK 答案反射）

| Domain | 经典干预 | CK 关键词 |
|---|---|---|
| **① 经济** | **Social work referral** / SNAP / Medicaid enrollment / 食物银行 / housing assistance / patient assistance program | "social work" / "case manager" / "financial counseling" |
| **② 教育** | **Teach-back method** / 简化文字 / 视觉材料 / health literacy assessment / **professional medical interpreter**（不是家属） | "low literacy materials" / "pictures" / "professional interpreter" |
| **③ 医疗 access** | **Patient navigator** / **FQHC** (Federally Qualified Health Center) / community health center / sliding scale clinic / telehealth | "navigator" / "FQHC" / "sliding scale" |
| **④ 邻里** | **Lead testing**（旧房 + 儿童）/ 哮喘 trigger 评估 / community resource referral / housing remediation | "environmental assessment" / "lead screening" |
| **⑤ 社会** | **IPV screen** / depression screen / community support group / peer support | "support group" / "screen for IPV" |

### 答案选择反射钩

> 题干给 SDOH 暴露 + 5 个选项 → 优先级：
>
> **资源 / referral（social work / navigator / FQHC）> 系统层 screening > 简化教育 > 加药 > 责怪患者 ❌**

## 七、LEP & Interpreter Rules ⭐⭐（独立高频考点）

**LEP (Limited English Proficiency)** 是 Education + Health care access 双重 SDOH。

> [!danger] 铁律 — Professional Interpreter
> **LEP 患者必用 professional medical interpreter**（电话 / 现场 / 视频均可）。
>
> **❌ 不能用家属 / 朋友 / 同事翻译**，原因：
> 1. **隐私 / confidentiality 风险**
> 2. **准确性差**（家属常 filter / soften / omit 信息 — 特别是 bad news / sex / mental health）
> 3. **儿童尤其禁止**（认知 + emotional burden + 角色错位）
> 4. **违反 standard of care + Title VI（Civil Rights Act）联邦法律**

| 场景 | 正确做法 |
|---|---|
| 急诊 + 患者 LEP + 家属在场 | Call **phone interpreter line** —— 不用家属 |
| LEP 患者 + 双语医生在场（但非翻译身份） | 用 professional interpreter —— 双语医生角色不同 |
| LEP 患者 + 自带儿童翻译 | ❌ 禁止 —— call professional |
| 患者**拒绝** professional interpreter，坚持用家属 | 仍尝试 + document refusal + 在 emergency 涉及 bad news / consent 时强烈劝告 professional |

## 八、Cross-Cultural Care 3 大支柱 ⭐⭐⭐（Q21242 锚定）

CK 常考"hesitant minority/immigrant patient → first step?"

| 支柱 | 内容 |
|---|---|
| ① **Patient-centeredness** | Understanding + aligning care with patient values + priorities |
| ② **Cultural sensitivity** | Exploring + responding to patient's cultural perspective（**避免 stereotyping** / 主动 ask questions）+ 理解 sociocultural influences |
| ③ **Enhanced communication** | 解决 language barriers（professional interpreter）+ tailor communication styles（eye contact / addressing family）to **individual** preferences |

### Cross-Cultural Counseling 决策树

```
Minority / immigrant patient hesitant for screening / treatment
    │
    ▼
① 首选 — Identify patient's perceptions + beliefs about disease / 病因 / prevention ⭐
   "What do you think causes cancer?"
   "How do people in your community view screening?"
    │
    ▼
② Based on perceptions → tailor counseling
   ├─ Spirituality / fate-based → 用 patient framework 内 reframe
   ├─ Misunderstanding of mechanism → 科学 explanation
   └─ Stigma / fear → empathic 解决 specific concern
    │
    ▼
③ Family-centered discussion（仅在 certain cultures + patient prefer 时）
    │
    ▼
④ Culturally appropriate materials（**adjunct**，不是 first step）
```

### 非白人 patient 常见 cancer screening perceptions

| Perception | 文化根源 | 处理方向 |
|---|---|---|
| **"I'm less susceptible to cancer"** | Prevention messaging primarily features White American communities → disconnect | Community-specific data + culturally relevant examples |
| **"No symptoms = no cancer"** | 早期 cancer 多无症状的概念不普及 | 解释 screening rationale + 早期发现 = 治愈率高 |
| **"Tempting fate to screen"** | 部分文化认为没事别去找事 | Empathic acknowledge + reframe 为 "protecting future health" |
| **"Cervical / breast cancer 与 sexual activity / 怀孕等 stigma"** | 文化 stigma | Confidentiality 重申 + private setting |
| **"Religious / spiritual cause of illness"** | Non-Western health framework | Acknowledge + 在 patient framework 内 reframe（不否定）|

> [!warning] First step 永远是 "**Identify perceptions**"
> ❌ 不是讲科学（否定 patient framework）
> ❌ 不是 fear-based messaging（加剧 hesitancy）
> ❌ 不是直接发 culturally appropriate materials（adjunct）
> ❌ 不是反射推家庭 discussion（要看 individual preference）

详见 [[mistakes/uworld-mistakes#^Q21242]] 完整拆解。

## 九、Routine SDOH Screening — 当代 Guidelines

| 学会 | 推荐 | 频率 |
|---|---|---|
| **USPSTF** | Routine screening for **IPV** / depression / unhealthy substance use | Primary care 常规 |
| **AAP**（儿科）| Routine SDOH screening — food insecurity / housing instability / parental depression | **每次 well-child visit** ⭐ |
| **ACOG**（OB）| 每次 prenatal visit 筛 **IPV** / food insecurity / housing | 每次 prenatal |
| **CMS** | 要求 hospital 收集 SDOH ICD-10 Z codes（Z55-Z65） | Systematic |
| **Joint Commission** | Health equity 是 National Patient Safety Goal | 长期 |

|---|---|

| **Z56** | 就业 / unemployment |
| **Z57** | 职业暴露 |
| **Z58** | 物理环境 |

## 十、Cultural Competency vs Cultural Humility（容易混）

| 概念 | 定义 | 重点 |
|---|---|---|
| **Cultural competency** | Provider 获得 specific cultural 知识 / skills | "学会"某文化 — 容易 stereotype |
| **Cultural humility** ⭐ | **Lifelong** self-reflection + 承认自身 cultural limitations + 与 patient 学习 | 谦逊 + 持续 + 个体化 |

> **当代倾向**：Cultural humility > Cultural competency
> 原因：Competency 暗示"学会就 done"，容易形成 stereotype（"所有 X 文化都这样"）；Humility 强调每个病人是 individual + provider 永远在 learning。

## 十一、高频陷阱 ⭐⭐（CK 钓鱼选项库）

| 陷阱 | 为什么错 | 正确反应 |
|---|---|---|
| **"Counsel on compliance importance"**（病人买不起药）| Blame the victim | Social work / financial assistance referral |
| **"Intensify medication"**（HbA1c 不达标 + SDOH 暴露）| 加药不解决"买不起" | Address SDOH first |
| **"Educate mother on adherence"**（mold-infested home）| Root cause 不是教育 | Address housing |
| **用家属当翻译**（LEP 患者）| 违反 standard + Title VI | **Professional medical interpreter** |
| **用儿童当翻译** | 认知 + emotional burden + 角色错位 | Professional interpreter（任何情境）|
| **把 SDOH 当 individual behavior** | SDOH 是结构性因素 | 系统层干预（resource + policy）|
| **Disparity 归单一原因** | Disparity 多因素 | SDOH **+** implicit bias **+** structural |
| **Cultural competency = SDOH** | 不等价 | Competency 是 provider skill；SDOH 是 patient 环境 |
| **Cross-cultural first step = 给 materials** | Materials 是 adjunct | **Identify perceptions first** |
| **Cross-cultural first step = 讲科学** | 否定 patient framework | Understand patient framework first |
| **Cross-cultural 反射 = family-centered** | 要看 individual preference | 先 ask patient preferences |

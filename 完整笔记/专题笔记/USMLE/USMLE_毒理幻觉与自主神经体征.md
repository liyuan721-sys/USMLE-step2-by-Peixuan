# 毒理幻觉与自主神经体征（Hallucination-type → toxin 反查 + 自主神经签名确认）

## 〇、核心原则：幻觉类型本身就是诊断线索 ⭐

> [!success] 三条 Step 2 高频反射
> 1. **视幻觉 (visual hallucination) / 触幻觉 (tactile hallucination) → 优先想"器质 / 中毒 / 戒断 / 谵妄 (delirium)"**，不是原发精神分裂症 (schizophrenia)。
> 2. **纯听幻觉 (auditory hallucination) + 神志清 + 无自主神经体征 → 原发精神病 (primary psychosis)** 更可能（精分的典型幻觉是听觉）。
> 3. **幻觉 + 自主神经亢进（瞳孔 / 出汗 / 心率 / 血压 / 体温异常）→ 一定是中毒或戒断**，自主神经签名就是把"哪个毒"钉死的第二轴。

> [!warning] 关键例外：酒精性幻觉症 (alcoholic hallucinosis) —— 毒源性却以"听幻觉"为主
> 戒酒 12–24 h 出现，可为**听 / 视 / 触幻觉**，但**神志清醒 (clear sensorium) + 定向力正常 + 生命体征稳定** → 别因为"有听幻觉"就反射精分。机制是戒断，但还没到震颤谵妄 (delirium tremens, DT) 的自主神经风暴。详见 §三。

```
一句话决策：
有幻觉
 ├─ 视 / 触 / 小人 / 联觉 → 中毒 or 戒断 or 谵妄（看自主神经签名定毒，见 §二+§四）
 └─ 纯听 ──┬─ 神志清 + 无自主神经征 → 原发精神病（精分）
           └─ 戒酒 12–24h + 神志清 + 生命体征稳 → 酒精性幻觉症（毒源例外）
```

## 一、幻觉类型 → 毒物/状态反查表 ⭐⭐（本页核心增量）

> 表前先记：**触幻觉 formication 几乎专属"拟交感 + 戒酒"；联觉 synesthesia 专属 LSD；小人幻觉 lilliputian 想抗胆碱 + 戒酒。**

| 幻觉类型 | 最高频指向 | 配套硬钩子 | 链接深挖 |
|---|---|---|---|
| **触幻觉 / 蚁走虫爬感 formication**（"虫子在皮下爬"+ 反复抓挠致抓痕 excoriations）| **可卡因 cocaine（"coke bugs"，Magnan 征 Magnan sign）/ 甲基苯丙胺 methamphetamine·苯丙胺 amphetamine（"meth mites"）**；**戒酒 DT / 苯二氮䓬 benzodiazepine (BZD) 戒断** | 散瞳 + 大汗 + 心率↑ + 高热（拟交感）；戒断则有震颤 + 自主神经风暴 | [[完整笔记/专题笔记/物质单药/psych_物质_Cocaine]] / [[完整笔记/专题笔记/物质单药/psych_物质_Methamphetamine]] |
| **联觉 synesthesia**（"看见声音的颜色 / 听见颜色"）+ 知觉扭曲 / 拖影 | **LSD（麦角酸二乙酰胺 lysergic acid diethylamide）/ 裸盖菇素 psilocybin** | **散瞳 + 轻度交感（心率/血压/体温轻↑）+ insight 保留**（知道是药物所致）| [[完整笔记/专题笔记/物质单药/psych_物质_LSD]] |
| **视幻觉 visual**（生动、动物 / 昆虫 / 火光）| **抗胆碱 anticholinergic（阿托品 atropine / 苯海拉明 diphenhydramine / 曼陀罗 Jimson weed）**；戒酒；谵妄；左旋多巴 levodopa / 帕金森病 (Parkinson disease, PD) 用药 | 抗胆碱 = 干红热散瞳 + 肠鸣↓ + 尿潴留；谵妄 = 注意力波动 | [[完整笔记/专题笔记/USMLE/USMLE_毒物综合征toxidrome速查]] |
| **小人幻觉 lilliputian**（看见缩小的人/物）+ 在空中抓摸 / 摘线头 (carphologia)、拔毛样动作 | **抗胆碱谵妄**（"mad as a hatter"）/ **戒酒 DT** | 抗胆碱 = 全干无汗；DT = 全湿 + 定向力丧失 | 见 §四 |
| **纯听幻觉 auditory** + 神志清 + **无**自主神经征 | **原发精神分裂症 schizophrenia**（非毒理）| 无瞳孔/出汗/生命体征异常、长病程 | [[完整笔记/Peixuan分科笔记/psych]] |
| **听/视/触幻觉 + 神志清 + 生命体征稳 + 戒酒 12–24h** | **酒精性幻觉症 alcoholic hallucinosis** | 与 DT 区别 = 定向力**正常** + **无**自主神经风暴 | [[完整笔记/专题笔记/物质单药/psych_物质_Alcohol]] |
| **黄视 / 绿视 xanthopsia / chromatopsia**（"看东西发黄绿、灯周光晕"）| **地高辛中毒 digoxin toxicity** | 恶心呕吐 + 视觉异常 + 心律失常 + 高钾（急性）| [[完整笔记/专题笔记/USMLE/USMLE_毒物综合征toxidrome速查]] |

> [!tip] 反查口诀
> **"虫爬拟交感（meth/coke）、看声色 = LSD、干热视幻觉 = 抗胆碱、纯听清醒 = 精分、戒酒可全套（清醒幻觉症 / 风暴 DT）、黄绿光晕 = 地高辛。"**

## 二、触幻觉 formication 专攻（最高频的"幻觉作为入口"题）⭐

> [!danger] Stem 模式
> 反复出现"**患者感觉皮肤下有虫子在爬 (bugs crawling under the skin) + 反复抓挠 → 皮肤抓痕 / 溃疡 (excoriations)**"，逼你从触幻觉反推毒物。

| 病因 | 神志 / 自主神经 | 钥匙 |
|---|---|---|
| **可卡因 / 甲基苯丙胺 / 苯丙胺中毒（拟交感 sympathomimetic）** ⭐ | 激越 + **散瞳 + 大汗 + 心率↑ + 高热** | 急性中毒场景 + 抓痕 + **"coke bugs"（Magnan 征）/ "meth mites"**；冰毒还有 **烂牙 meth mouth + 磨牙** |
| **戒酒震颤谵妄 DT (delirium tremens) / 苯二氮䓬 benzodiazepine (BZD) 戒断** | 戒断 48–96h（酒）+ **定向力丧失 + 自主神经风暴**（心率/血压/体温↑）| 酗酒/停药时间线 + 全湿 + 谵妄（同 GABA 撤退谱，见 §三）|
| **寄生虫妄想 delusional parasitosis (Ekbom syndrome)**（非毒理）| 神志清 + **无自主神经征** + 长期 | 老年 / 单一妄想；常带"虫子标本"来诊 (matchbox sign)；可继发可卡因长期使用 |
| **疥疮 scabies（真实病，非幻觉）** | 无中枢/自主神经征 | **指缝 / 腕屈侧夜间剧痒 + 隧道 burrow**；家人同痒 → 这是真痒不是幻觉 |

> [!success] 一句话
> **"虫爬感 + 抓痕 + 急性激越散瞳大汗 = 拟交感（冰毒/可卡因）"** 是默认答案；神志清 + 无自主神经征 + 慢性单一妄想才退到寄生虫妄想；真有隧道夜痒家人同患 = 疥疮。

## 三、致幻毒物的自主神经签名配对（确认轴 —— 精简版，细节链回母笔记）⭐

> [!info] 本表只列"会致幻的毒/状态"的自主神经一行签名，用来在 §一锁定类型后做**第二轴确认**。完整受体机制 / 解毒剂见 [[完整笔记/专题笔记/USMLE/USMLE_毒物综合征toxidrome速查]]，床旁五轴分流见 [[完整笔记/专题笔记/USMLE/USMLE_中毒昏迷五轴床旁鉴别]]。

| 致幻毒/状态 | 瞳孔 | 皮肤 | 心率/血压/体温 | 一行签名 |
|---|---|---|---|---|
| **抗胆碱 anticholinergic** | 散大、对光差 | **干、红、热（无汗）** | ↑ / ↑ / ↑ | 干热散瞳 + 肠鸣↓ + 尿潴留 |
| **拟交感 sympathomimetic**（可卡因/冰毒/MDMA 亚甲二氧基甲基苯丙胺 / 摇头丸 ecstasy）| 散大、对光保留 | **湿（大汗）** | ↑↑ / ↑ / ↑ | 湿热散瞳 + 肠鸣存 |
| **LSD / 致幻剂** | 散大 | 偏湿/正常 | 轻↑ / 轻↑ / 轻↑ | **轻度交感 + insight 保留**（最"轻"的自主神经） |
| **戒酒幻觉症 hallucinosis** | 正常/轻散 | 轻度交感 | 轻↑ | **神志清 + 生命体征稳**（自主神经轻） |
| **戒酒 DT** | 散大 | **大汗** | ↑↑ / ↑↑ / ↑↑ | **定向力丧失 + 自主神经风暴**（最重，可致死）|
| **地高辛 digoxin** | — | — | **缓脉/心律失常** + 高钾 | 黄绿视 + 恶心呕吐 + 房室阻滞 |

> [!warning] 两对最爱考的"同幻觉不同自主神经"
> - **抗胆碱 vs 拟交感**：都散瞳 + 高热 + 心率快 + 可视幻觉 → **靠"干 vs 湿 + 肠鸣"切**（抗胆碱干、肠鸣↓、尿潴留；拟交感湿、肠鸣存）。详 [[完整笔记/专题笔记/psych/psych_物质急诊速查]] §8.4。
> - **酒精性幻觉症 vs DT**：都戒酒 + 幻觉 → **靠"定向力 + 自主神经风暴"切**（幻觉症神志清、生命体征稳；DT 定向力丧失、风暴、致死）。

## 四、高频鉴别对（幻觉 + 自主神经组合题）

> [!question]- 都散瞳 + 视幻觉 + 心率快：抗胆碱 vs 拟交感怎么切？
> **看皮肤干湿 + 肠鸣音。** 抗胆碱 = 皮肤干红热 + 黏膜干 + 肠鸣↓ + 尿潴留（"干红热瞎疯"）；拟交感 = 大汗 + 肠鸣存 + 常有冠脉痉挛/心肌梗死 (myocardial infarction, MI) 线索。两者都"散瞳火热"，**"出汗"是分水岭**（拟交感湿、抗胆碱干）。

> [!question]- 戒酒患者出现幻觉 —— 是 alcoholic hallucinosis 还是 DT？
> **看定向力 + 自主神经 + 时间线。**
> - **Alcoholic hallucinosis**：戒酒 **12–24h**，幻觉（视/听/触）+ **定向力正常 + 生命体征稳定** → BZD（苯二氮䓬 benzodiazepine）+ 支持。
> - **DT (delirium tremens)**：戒酒 **48–96h**，幻觉 + **定向力丧失 + 自主神经风暴（HR↑/BP↑/T↑/大汗）** → ICU + 大剂量 BZD，**可致死**。
> 口诀："清醒稳 = 幻觉症；糊涂风暴 = DT。"完整酒精戒断综合征 (alcohol withdrawal syndrome, AWS) 4 阶段见 [[完整笔记/专题笔记/psych/psych_物质急诊速查]] §9.2。

> [!question]- 年轻人聚会后视幻觉 + "看见声音的颜色" + 散瞳 —— 与急性精神病如何区分？
> **LSD 中毒**：联觉 synesthesia（LSD 经典指纹）+ 散瞳 + 轻度交感 + **insight 保留**（患者多能意识到"是药物让我这样"）；持续数小时自限。**原发精神病**：无联觉、无明确用药时间线、**insight 缺失**、病程长、听幻觉为主。

> [!info] 药源性 / 中毒性视幻觉的其它常考来源（非"街头毒品"）
> | 来源 | 线索 |
> |---|---|
> | **左旋多巴 / 多巴胺激动剂 levodopa / dopamine agonist**（PD 治疗）| 帕金森病 (Parkinson disease) 患者加药后**生动视幻觉**（常先于其它精神症状）|
> | **路易体痴呆 Lewy body dementia**（非药，但高频同框）| **反复生动视幻觉 + 波动认知 + 帕金森征**三联 |
> | **谵妄 delirium**（任何病因：感染/代谢/药物）| 注意力波动 + 意识水平改变 + 视幻觉，**急性起病、昼夜波动** |
> | **抗胆碱负荷**（老人多药：苯海拉明 diphenhydramine / 奥昔布宁 oxybutynin 等）| 干红热散瞳 + 视幻觉 + 谵妄 |

## 五、Memory Hook

> [!success] 一句话锁定
> **先问"哪种幻觉"再看"自主神经":**
> - **触幻觉（虫爬）= 拟交感（meth/coke）or 戒酒** → 看散瞳大汗(中毒) vs 风暴(DT)。
> - **联觉（看见声音颜色）= LSD** → insight 保留 + 散瞳轻交感。
> - **视幻觉 + 干热散瞳 = 抗胆碱**；湿则拟交感。
> - **纯听 + 清醒 + 无自主神经 = 精分**；唯一毒源听幻觉例外 = 戒酒幻觉症（清醒稳）。
> - **黄绿视 + 光晕 = 地高辛**。
> 第二轴永远是自主神经签名（干/湿·瞳孔·HR/BP/T）—— 它把"哪种幻觉"收窄到"哪个毒"。

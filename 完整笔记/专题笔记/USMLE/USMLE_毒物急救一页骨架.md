## ① 认综合征：决策树（先看 瞳孔 + 干湿 + 呼吸）

```
中毒/误服 → 看【瞳孔 · 皮肤干湿 · 呼吸 · 肠鸣】
│
├─ 针尖瞳 pinpoint
│   ├─ 呼吸↓ + 干（无湿征）            → 阿片 opioid     → 纳洛酮 naloxone
│   └─ 湿（大汗/流涎）+ 肌束颤 + 农药   → 有机磷 organophos → 阿托品+解磷定（先脱衣冲洗去污）
│
├─ 散瞳 mydriasis
│   ├─ 干 + 无汗 + 高热 + 尿潴留 + 肠鸣↓ → 抗胆碱 anticholinergic（苯海拉明）
│   └─ 湿（出汗）+ 肠鸣存 + 欣快/躁动    → 拟交感 sympathomimetic（可卡因/苯丙胺）
│
├─ 瞳孔多正常 + 嗜睡 + 生命体征基本正常  → 镇静催眠 BZD → 氟马西尼（慎用）
│
└─ 瞳孔不主导 → 看场景/实验室指纹：
     火灾 + 樱桃红 + 乳酸↑↑（SpO₂不掉）  → 氰化物 cyanide  → 羟钴胺 hydroxocobalamin
     火灾 + SpO₂假高 + 樱桃红            → CO 一氧化碳     → 100% O₂ / 高压氧
     草酸钙结晶 + 高 AG(阴离子间隙) + 低钙 → 乙二醇 ethylene glycol → 甲吡唑 fomepizole
     视力模糊 + 高 AG                    → 甲醇 methanol   → 甲吡唑
     宽 QRS + 抗胆碱征                   → TCA(三环抗抑郁 tricyclic antidepressant) → 碳酸氢钠 sodium bicarbonate
```

> 钥匙：**出汗区分不了拟交感 vs 胆碱能（都湿）→ 看"其他分泌 + 瞳孔"；只有抗胆碱"无汗"。** **有机磷/尼古丁的 HR/BP 不可靠**（毒蕈碱降 vs 烟碱升拔河）。

## ② 配解毒剂：高频对子表

| 毒物 | 解毒剂 |
|---|---|
| 对乙酰氨基酚 acetaminophen | N-乙酰半胱氨酸 NAC |
| 阿片 opioid | 纳洛酮 naloxone |
| 苯二氮䓬 benzodiazepine | 氟马西尼 flumazenil（慎）|
| 有机磷 organophosphate | 阿托品 atropine + 解磷定 pralidoxime |
| TCA / 钠通道阻断（宽 QRS）| 碳酸氢钠 sodium bicarbonate |
| β阻滞剂 β-blocker（低糖）| 胰高血糖素 glucagon |
| 钙通道阻滞剂 CCB（高糖）| 钙 + 胰岛素-葡萄糖 HIET |
| 铁 iron | 去铁胺 deferoxamine |
| 甲醇/乙二醇 methanol/ethylene glycol | 甲吡唑 fomepizole（+透析）|
| 一氧化碳 CO | 100% O₂ / 高压氧 |
| 高铁血红蛋白 methemoglobin | 亚甲蓝 methylene blue |
| 氰化物 cyanide | 羟钴胺 hydroxocobalamin |
| 磺脲类低血糖 sulfonylurea | 葡萄糖 + 奥曲肽 octreotide |
| 抗胆碱 anticholinergic | 毒扁豆碱 physostigmine |
| 铅/砷汞 lead/arsenic-mercury | 螯合剂（EDTA/DMSA/BAL）|

## ③ 处理顺序铁律

```
① ABC 稳定（气道/呼吸/循环；意识差先插管）
② 去污 decontamination（脱衣冲洗皮肤 / 活性炭<1h）
③ 解毒 antidote（见对子表）
④ 增强清除（碱化尿 / 血透）
```

> 活性炭**无效**毒物口诀 **PHAILS**：农药 / 烃类 / 酸碱腐蚀剂 / 铁 / 锂 / 醇类（各有专属解法）。

## ④ 五句焊死（扛 80%）

> [!success]
> 1. **针尖瞳 + 呼吸停** = 阿片 → 纳洛酮
> 2. **湿 + 缩瞳 + 肌束颤 + 农药** = 有机磷 → 阿托品+解磷定（先脱衣冲洗）
> 3. **干 + 散瞳 + 高热 + 尿潴留 + 无汗** = 抗胆碱
> 4. **火灾 + 樱桃红 + 乳酸爆表** = 氰化物 → 羟钴胺
> 5. **草酸钙结晶 + 高 AG + 低钙** = 乙二醇 → 甲吡唑

---

## ⑤ 反常识陷阱速记（容易丢分）

- **碳酸氢钠**：TLS（肿瘤溶解综合征 tumor lysis syndrome）/ DKA（糖尿病酮症酸中毒 diabetic ketoacidosis）**别反射给**；高钾单用降钾差（一线钙+胰岛素-葡萄糖）。
- **冻伤**复温不活 → **血管造影/溶栓**救肢，**不是立刻截肢**（截肢等数周分界）。
- **HACE / HAPE**（高原脑/肺水肿）→ 地塞米松/硝苯地平 + 下降，**别利尿**（非容量过载）。
- **有机磷**插管后**先脱衣冲洗去污**，再阿托品（碳酸氢钠是 TCA 的，方向反）。
- **抗胆碱 = 唯一不出汗的高热**；其余高热（拟交感/胆碱能/中暑）都出汗。

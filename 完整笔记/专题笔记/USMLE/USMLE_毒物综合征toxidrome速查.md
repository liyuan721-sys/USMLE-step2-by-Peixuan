# 毒物综合征 Toxidrome 速查（误服/中毒 pattern 识别）

> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q3856]]（腐蚀剂误服，做对）+ [[mistakes/uworld-mistakes_2026-06#^Q3827]]（铁中毒）。
> 用法：见"中毒/误服"，先看 **意识 + 瞳孔 + 皮肤干湿 + 心率 + 肠鸣** 五轴 → 锁定 toxidrome。铁中毒指纹另见 [[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全]] §8.3。

## 一、五大经典 toxidrome ⭐

| Toxidrome | 代表毒物 | 瞳孔 | 皮肤 | 心率/血压 | 神志 | 特征 |
|---|---|---|---|---|---|---|
| **抗胆碱能 anticholinergic** | 阿托品 atropine、抗组胺药 antihistamine、TCA（三环类抗抑郁药 tricyclic antidepressant）、莨菪类 belladonna alkaloids | **散大** | **干、红、热** | **↑** | 谵妄、躁动 | **"红如甜菜、干如骨、热如兔、盲如蝠、狂如帽匠"**；尿潴留、肠鸣↓ |
| **胆碱能 cholinergic** | **有机磷 organophosphate**、氨基甲酸酯 carbamate、毒蕈 muscarinic mushroom | **缩小（针尖）** | **湿（大汗 + 流涎）** | **↓（缓脉）** | 意识混乱/抽搐 | **DUMBELS / SLUDGE**（流涎泪、尿便、支气管痉挛、缓脉）；解毒 = **阿托品 atropine + 解磷定 pralidoxime** |
| **拟交感 sympathomimetic** | 可卡因 cocaine、苯丙胺 amphetamine、MDMA（亚甲二氧基甲基苯丙胺 / 摇头丸 ecstasy）| **散大** | **湿（出汗）** | **↑↑** | 躁动、欣快 | 高热、抽搐；vs 抗胆碱 = **出汗 + 肠鸣存**（抗胆碱皮肤干、肠鸣↓）|
| **阿片 opioid** | 海洛因 heroin、吗啡 morphine、芬太尼 fentanyl | **针尖** | 正常 | **↓ + 呼吸↓** | **昏睡/昏迷** | **呼吸抑制三联（昏迷 + 针尖瞳 + 呼吸↓）**；解毒 = **纳洛酮 naloxone** |
| **镇静催眠 sedative-hypnotic** | 苯二氮䓬 benzodiazepine (BZD)、巴比妥 barbiturate、酒精 alcohol | 多正常 | 正常 | ↓ | 嗜睡/昏迷 | 呼吸抑制（BZD 单用轻）；BZD 解毒 = **氟马西尼 flumazenil**（慎用）|

## 二、易混轴：流涎 salivation 四鉴别 ⭐（Q3856 关键）

| 病 | 流涎 + ? | 钥匙 |
|---|---|---|
| **腐蚀剂误服 caustic** | 流涎 + **口咽溃疡 + 意识清** | **局部接触损伤**，不改变意识；碱→液化坏死（穿孔/狭窄/远期鳞癌）|
| **有机磷 organophosphate** | 流涎 + **缩瞳 + 缓脉 + 多尿/腹泻** | **胆碱能全身过量**（DUMBELS）|
| **抗胆碱 anticholinergic** | **口干（反向！）** + 散瞳 + 谵妄 | 与流涎相反，鉴别点 |
| **甲醇/乙二醇 methanol/ethylene glycol** | 无特异流涎 | 延迟视力模糊（甲醇）/ 草酸钙肾损（乙二醇）+ **高 AG 代酸 + 渗透压间隙↑** |

> [!danger] 我做对但要焊死（Q3856）
> **口咽溃疡 + 流口水 + 人清醒 = 腐蚀剂**（局部损伤，不改变意识）。同是流涎，**有机磷一定伴缩瞳 + 缓脉 + 腹泻**；抗胆碱反而**口干**。

## 二-bis、易混轴：针尖瞳（miosis / pinpoint pupil）三鉴别 ⭐

> 昏迷 + 针尖瞳是高频 stem。三个病因都针尖,**靠伴随征 + 纳洛酮反应区分**,别一看针尖就反射阿片。

| 病因 | 针尖瞳机制 | 伴随征（鉴别钥匙）| 处理 |
|---|---|---|---|
| **阿片中毒 opioid** | 兴奋 E-W 核（Edinger-Westphal nucleus）→ 副交感 | **呼吸抑制 + 昏睡**，无胆碱能湿征、无局灶脑干征 | **纳洛酮 naloxone**（逆转）|
| **有机磷 organophosphate** | 毒蕈碱受体过度激动 → 瞳孔括约肌收缩 | **湿性胆碱能 DUMBELS**：大汗/流涎 + 缓脉 + 腹泻 + **肌束颤 fasciculations** | **阿托品 atropine + 解磷定 pralidoxime** |
| **脑桥出血 pontine hemorrhage** | 破坏下行交感通路 → 副交感失对抗（瞳孔**对光仍有反应**）| **突发昏迷 + 四肢瘫 + 高热 + 其他脑干征**（凝视麻痹 / 眼球浮动 ocular bobbing）；**高血压史** | **头 CT** 确诊 → 神外 / 支持；**纳洛酮无效** |

> [!danger] 三秒分流
> 昏迷 + 针尖瞳 → ① **先给纳洛酮**：醒 = 阿片；不醒往下走。② 找**胆碱能湿征**（大汗/流涎/缓脉/肌束颤）= 有机磷。③ 有**突发 + 局灶脑干征 + 高血压史** → **脑桥出血 → 立即 CT**（纳洛酮无效、无胆碱能征）。
> 钥匙:**阿片"干"(无汗)、有机磷"湿"(大汗)、脑桥出血"瘫+热+突发"**。

## 三、特殊：腐蚀剂 caustic ingestion 要点

- **碱性 alkali**（管道清洁剂）→ **液化坏死**，穿透深 → 食管穿孔/纵隔炎、**远期食管狭窄 + 数年后鳞癌**；胃损伤较轻（胃酸中和）。
- **酸性 acid** → **凝固坏死**，胃损伤相对重。
- 处理：**不催吐、不中和、不洗胃、不放 NG（盲插）**；评估气道 + **内镜分级**（24 h 内）+ 支持。

## 四、中毒解毒剂 antidote 速查表 ⭐
#难记

| 毒物 | 解毒剂 antidote |
|---|---|
| 对乙酰氨基酚 acetaminophen | **N-乙酰半胱氨酸 N-acetylcysteine (NAC)** |
| 阿片 opioid | **纳洛酮 naloxone** |
| 苯二氮䓬 benzodiazepine | 氟马西尼 flumazenil（慎用，可致癫痫）|
| 有机磷 organophosphate / 氨基甲酸酯 carbamate（胆碱能）| **阿托品 atropine + 解磷定 pralidoxime** |
| TCA / 钠通道阻断（可卡因 cocaine / IC 类）| **碳酸氢钠 sodium bicarbonate** |
| β受体阻滞剂 β-blocker | **胰高血糖素 glucagon** |
| 钙通道阻滞剂 CCB | 钙 calcium + 胰岛素-葡萄糖 |
| 地高辛 digoxin | 地高辛免疫 Fab (digoxin-specific Fab) |
| 华法林 warfarin | 维生素 K vitamin K + FFP（新鲜冰冻血浆 fresh frozen plasma）/ PCC（凝血酶原复合物 prothrombin complex concentrate）|
| 肝素 heparin | 鱼精蛋白 protamine |
| 铁 iron | 去铁胺 deferoxamine |
| 铅 lead | EDTA / 琥珀酸 succimer (DMSA) / 二巯丙醇 dimercaprol (BAL)；脑病先 BAL |
| 砷/汞/金 arsenic/mercury/gold | 二巯丙醇 BAL / DMSA |
| 铜 copper（Wilson 病）| 青霉胺 penicillamine |
| 甲醇/乙二醇 methanol/ethylene glycol | 甲吡唑 fomepizole（±乙醇 ethanol）+ 透析 |
| 一氧化碳 CO (carbon monoxide) | 100% O₂ / 高压氧 HBO |
| 高铁血红蛋白血症 methemoglobinemia | 亚甲蓝 methylene blue |
| 氰化物 cyanide | 羟钴胺 hydroxocobalamin / 硫代硫酸钠 sodium thiosulfate |
| 磺脲类低血糖 sulfonylurea | 葡萄糖 dextrose + 奥曲肽 octreotide |
| 抗胆碱能 anticholinergic | 毒扁豆碱 physostigmine |

| 水杨酸 salicylate | 碳酸氢钠 + 血液透析 hemodialysis |
| 局麻药/脂溶性药过量 (LAST) | 脂肪乳 lipid emulsion |

> [!danger] 焊死陷阱：CCB 中毒的"钙 + 胰岛素-葡萄糖" ≠ 高钾的"钙 + 胰岛素-葡萄糖"
> 同样两味药，**给的理由完全相反**：
>
> | | 高钾血症 hyperkalemia | CCB 中毒（钙通道阻滞剂中毒）|
> |---|---|---|
> | **钙 calcium** | **稳膜防室颤**（提高阈电位）；**不降钾** | **克服通道阻滞**：堆高细胞外钙，从残余通道硬挤入 → 复收缩/传导 |
> | **胰岛素-葡萄糖** | 胰岛素**把 K⁺ 赶进细胞**（真正降血钾）；葡萄糖防低糖 | **HIET（高胰岛素-正常血糖疗法 hyperinsulinemia-euglycemia therapy）**：胰岛素是**正性肌力 + 心肌摄糖供能**，与钾无关；葡萄糖维持血糖 |
> | **剂量** | 常规胰岛素 ~10 U + 葡萄糖（一次 shift）| **1 U/kg 弹丸 + 0.5–1 U/kg/h 泵**（10 倍↑，盯血糖）|
> | **目标** | 降钾 | 升心输出 |
>
> 一句话：高钾"钙护膜 + 胰岛素**移钾**"；CCB 中毒"钙**补底物** + 大剂量胰岛素**强心 HIET**"。

> [!info] 顺带：β阻滞剂 vs CCB 中毒解毒对比
> - **β受体阻滞剂 β-blocker 中毒** → 首选 **胰高血糖素 glucagon**（绕过 β 受体直接升 cAMP，环磷酸腺苷 cyclic AMP）；常伴**低血糖**。
> - **CCB 中毒** → **钙 calcium + HIET**（±胰高血糖素）；常伴**高血糖**（胰岛素分泌受抑）。
> - 钥匙：**低糖 → β阻滞剂（glucagon）；高糖 → CCB（钙 + HIET）**。

> [!warning] 磺脲类 sulfonylurea 低血糖：奥曲肽 octreotide 为什么是关键（不止给糖）
> **磺脲类机制**：关闭 β 细胞 K-ATP 通道 (KATP channel) → 去极化 → Ca²⁺ 内流 → **葡萄糖非依赖地持续促胰岛素 insulin 释放**（药不走，β 细胞停不下来）。
>
> - ❌ **光给葡萄糖 dextrose = 陷阱**：升高的血糖**反过来再刺激**那个被磺脲类卡开的 β 细胞 → 更多胰岛素 → **反跳性/顽固性低血糖 rebound hypoglycemia**（给糖→更多胰岛素→再低糖的恶性循环）。
> - ✅ **奥曲肽 octreotide = 生长抑素类似物 (somatostatin analog)**：结合 β 细胞生长抑素受体 (SSTR) → **抑制 Ca²⁺ 内流 → 直接关掉胰岛素分泌** → 打断循环、防反跳。
> - 一句话：**葡萄糖治症状（升糖），奥曲肽治原因（关掉不当分泌）**；顽固/反复低血糖（D50 给了又掉）→ 加奥曲肽。
>
> | 低血糖原因 | 处理钥匙 |
> |---|---|
> | **磺脲类 sulfonylurea** | 葡萄糖 + **奥曲肽**（抑制内源胰岛素分泌）|
> | **外源性胰岛素 exogenous insulin 过量** | 葡萄糖**支持**（奥曲肽无效——已是外来胰岛素，无分泌可抑）|
> | **β阻滞剂中毒**伴低血糖 | 胰高血糖素 glucagon |

> [!tip] 今日错题覆盖
> 本表 NAC(Q3136) / 碳酸氢钠(Q4487·Q3138) / 胰高血糖素(Q19181) / 纳洛酮(Q4510) / 水杨酸→碳酸氢钠+血透(Q18925·Q19677) 均来自今天的题。

## 五、阿片中毒 intoxication vs 戒断 withdrawal（纳洛酮的两面）⭐

> 错题地基：[[mistakes/uworld-mistakes_2026-06#^Q4510]]（中毒→纳洛酮 ✓）+ [[mistakes/uworld-mistakes_2026-06#^Q4509]]（戒断→误给纳洛酮 ✗）。

| 维度 | 中毒 intoxication | 戒断 withdrawal |
|---|---|---|
| 本质 | 阿片**过量（抑制）** | 阿片**撤药（反跳亢进）** |
| 神志/呼吸 | 昏睡 + **呼吸↓**（最可靠指标）| 清醒烦躁 + 呼吸正常 |
| 瞳孔 | **针尖瞳**（可被合用拟交感抵消）| **散瞳 mydriasis** |
| 自主神经 | 低血压/缓脉/肠鸣↓/低体温 | **↑BP↑脉** + 流涕流泪 + 腹泻呕吐 + 肌痛 + 立毛 |
| 起病 | 用药后 | 末次用药 **4–12h** 后，**24–48h** 峰 |
| 处理 | **纳洛酮 naloxone（拮抗剂）** | **丁丙诺啡 buprenorphine / 美沙酮 methadone（激动剂）** + 可乐定 clonidine |

> [!danger] 纳洛酮的两面
> **中毒给纳洛酮救命 ✓；戒断给纳洛酮 = 诱发/加重急性戒断 ✗。** 一个"抑制要拮抗"，一个"撤药要补激动剂"。别一看阿片就反射纳洛酮。

> [!tip] 别只锁阿片：呼吸抑制完整鉴别
> 见"呼吸↓/低通气"还要想**可乐定（阿片李鬼）/ 镇静药合用 / 高镁血症（产妇 MgSO₄）/ GBS / 肉毒**等 → 见 [[完整笔记/专题笔记/USMLE/USMLE_呼吸抑制低通气鉴别]]。

### 5.1 四药用药原理（受体药理）⭐

> [!info] 总纲：全部围绕 μ 阿片受体 (μ-opioid receptor, MOR) 做文章
> **中毒 = 激动剂太多 → 给拮抗剂；戒断 = 激动剂被撤 → 补回激动剂 或 压住"去甲肾上腺素风暴"。**
>
> ```
> 中毒：μ 受体过度激动 → 脑干呼吸中枢抑制 → 呼吸↓（致死主因）
> 戒断：长期占据的 μ 受体突然空出 → 蓝斑 (locus coeruleus) 去甲肾上腺素能
>       神经元失去阿片抑制 → 交感暴发（流涕泪/腹泻/立毛/↑BP↑HR/肌痛）
> ```

| 药 | 受体药理 | 打哪个环节 | 关键考点 |
|---|---|---|---|
| **纳洛酮 naloxone** | 纯 μ 竞争性拮抗剂 | 中毒：把阿片挤下受体 | **半衰期短（~30–60min）< 多数阿片** → 可能"醒了又昏"，需重复/泵注；依赖者**催戒断** → 滴定到**呼吸频率恢复**即可，别追求完全清醒 |
| **美沙酮 methadone** | 完全 μ 激动剂（长效）| 戒断/维持：稳补激动剂 | 长半衰期 + 缓起效 → 平稳维持 + 抑渴求 + 封顶欣快；**QT 延长**风险、须 OTP（阿片类治疗项目 opioid treatment program）诊所发药 |
| **丁丙诺啡 buprenorphine** | **部分** μ 激动 + κ 拮抗 | 戒断/维持：补激动剂（更安全）| 高亲和力 + 低内在活性 → **呼吸抑制有封顶效应 (ceiling effect)**，过量致死风险低 |
| **可乐定 clonidine** | 中枢 α2 激动剂 | 戒断：压交感风暴 | 作用于**蓝斑**抑制中枢去甲肾上腺素输出 → 只缓解**自主神经症状**，**不解决渴求、不替代阿片**；副作用低血压 + 镇静 |

> [!danger] 部分激动剂的"沉降性戒断"陷阱（必考）
> **丁丙诺啡 buprenorphine 亲和力极高**，会把还占在受体上的完全激动剂（海洛因）**挤掉**，但自己只给**部分**激动 → 净激动↓ → **若在患者尚未进入戒断时就给 = 诱发"沉降性戒断 (precipitated withdrawal)"**。
> ✅ 所以**必须等患者已出现戒断征象**再用丁丙诺啡。

> [!tip] Suboxone（丁丙诺啡 + 纳洛酮复方）的防滥用设计
> 舌下含服时**纳洛酮几乎不吸收（无效）**，靠丁丙诺啡起治疗作用；但**若有人拿去静脉注射，纳洛酮立刻起效 → 催戒断** → 从药学上**劝退静脉滥用**。

> [!question]- 可乐定是 α2 **激动剂**，怎么"压"交感风暴？（破直觉）
> **激动剂激动的是一个"抑制性"受体——踩的不是油门，是刹车。**
> - **α2 = 负反馈自身受体 (autoreceptor)**，位于**突触前膜** + **中枢蓝斑 (locus coeruleus)/延髓**；偶联 **Gi → ↓cAMP（环磷酸腺苷 cyclic AMP）→ 关掉去甲肾上腺素 norepinephrine (NE) 释放**。
> - 所以激动 α2 = **从源头少放 NE → 抗交感 (sympatholytic)**，不是兴奋。受体激动剂 ≠ 一定让系统兴奋，要看那个受体本身干什么。
> - **对比"下游拦截"思路**：想压交感也可用 **α1 拮抗剂**（酚妥拉明 phentolamine / 哌唑嗪 prazosin）阻断突触后 α1；可乐定管"上游少放"，α1 阻滞剂管"下游不应答"。
> - **串回阿片戒断**：阿片（μ）和可乐定（α2）**都偶联 Gi、都抑制蓝斑放电** → 可乐定能顶替阿片压住 NE 暴发（自主神经症状），但不碰 μ → 不解渴求。
> - ⚠️ **配套考点**：长期可乐定骤停 → 中枢交感反弹 → **反跳性高血压 rebound hypertension** → 须**逐渐减停**（与"压交感"是一枚硬币两面）。
> - 同类（α2 激动 = 抗交感 + 镇静）：α-甲基多巴 methyldopa、右美托咪定 dexmedetomidine。

## 六、Memory Hook

> [!success] 一句话
> **干红热散瞳狂 = 抗胆碱；湿缩瞳缓脉拉 = 有机磷（胆碱能）；针尖瞳 + 呼吸停 = 阿片（纳洛酮）。** 流涎 + 口咽溃疡 + 清醒 = 腐蚀剂；流涎 + 缩瞳缓脉 = 有机磷。

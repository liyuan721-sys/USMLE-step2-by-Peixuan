# 易混药物对照（Confusable Drug Pairs）

> [!info] 本笔记定位
> 收集 USMLE 高频「名字像 / 听着像 / 同场景出现」却**容易混淆**的药物对。
> 每对破解思路统一：**先问一个鉴别问题 → 立刻分开**。
> 跨学科收录（心内 / 精神 / 急诊 / …），遇到新的就追加。

## 1. Adenosine（腺苷）vs Atropine（阿托品）⭐

> [!danger] 一句话锚定
> 两药**作用相反、适应症相反**：Adenosine 治**心动过速**（踩刹车），Atropine 治**心动过缓**（踩油门）。
> 鉴别只需问一句：**“这题心跳太快还是太慢？”** —— 快 → 腺苷，慢 → 阿托品。

### 对照表

| 维度 | **Adenosine（腺苷）** | **Atropine（阿托品）** |
|---|---|---|
| 对心率 | **↓ 减慢 / 阻断**（刹车）| **↑ 加快**（油门）|
| 机制 | 激动 A1 受体 → ↑K⁺ 外流 → **短暂阻断 AV 结** | **抗毒蕈碱（抗胆碱）** → 阻断迷走 → ↑SA 结放电 + ↑AV 传导 |
| 适应症 | **阵发性 SVT / AVNRT**（终止）| **症状性心动过缓**（窦缓、部分 AV 阻滞）|
| ACLS 角色 | SVT：迷走动作无效 → adenosine | 症状性心动过缓**一线** |
| 半衰期 | **超短（~10 秒）** → 快速 IV 推 + 立即冲管 | 较长 |
| 招牌副作用 | 一过性潮红 / 胸闷 / 濒死感 / 短暂窦性停搏；**哮喘慎用**（支气管痉挛）| 抗胆碱毒性 —— 口干、散瞳、尿潴留、谵妄（“干红怒热”）|
| 另一用途 | 药理负荷试验（冠脉扩张剂）| **有机磷 / 胆碱能中毒解药** |

### 机制 —— 为什么相反

```
            心脏的“刹车” = 迷走神经（胆碱能 / M2 受体）
                          │
   ┌──────────────────────┴──────────────────────┐
   ▼                                             ▼
Adenosine                                    Atropine
强化“刹车” → 直接压住 AV 结                   阻断“刹车” → 迷走失灵
   │                                             │
   ▼                                             ▼
心率 ↓、终止 SVT                              心率 ↑、治心动过缓
```

### UW 经典 stem

- **Adenosine**：年轻人突发心悸，**窄 QRS + 规则 + ~180/min + 无 P 波** → 阵发性 SVT → vagal maneuvers 无效 → adenosine 快速 IV 推。（亦可短暂阻 AV 结 → 露出 flutter 波助诊。）
- **Atropine**：老年人 HR 42 + 头晕 + 低血压 → 症状性心动过缓 → atropine 一线 → 无效 / infranodal 阻滞（Mobitz II、三度）→ 经皮起搏。

### Memory Hook

> [!success] 锁定口诀
> - **腺苷（adeno​Sine）→ S​VT** —— 都有 S；“腺苷压住 AV 结掐断 SVT”
> - **阿托品 → 抗胆碱 → 解迷走刹车 → 心率飙** —— 治“慢”；与抗胆碱毒性“心动过速”是同一机制
> - 一句话：**“快用腺苷压，慢用阿托提”**

## 2. Sound-alike Drug Trap（名字像 / 外观像，但完全不同药）⭐⭐⭐

> [!danger] 一句话锚定
> Sound-alike drug = USMLE Patient Safety 高频考点。**最经典对**：**Glucovance（含 sulfonylurea → 致 hypoglycemia）vs Glucophage（metformin → 不致 hypoglycemia）**。
> 入手 strategy：**Teach-back 强制**（让患者用自己话 describe changes → detect miscomprehension）。

### 2.1 Glucovance vs Glucophage ⭐⭐⭐（Q19788 经典 stem）

| 维度 | **Glucovance** | **Glucophage** |
|---|---|---|
| 通用名 | **Metformin + Glyburide**（combination）| **Metformin only** |
| 类别 | Biguanide + **Sulfonylurea** | Biguanide |
| **Hypoglycemia 风险** | **有**（sulfonylurea 致命 hypo）⚠️ | **无**（metformin 不致 hypo）|
| 老人风险 | 极高 — sulfonylurea 在老年 hypo 高风险 + prolonged hypo | 低 |
| 适应症 | T2DM combo therapy | T2DM monotherapy / 一线 |
| 主要副作用 | Hypoglycemia + lactic acidosis（罕）| GI（diarrhea / nausea）+ lactic acidosis（罕）|

### 2.2 Prednisone vs Prednisolone

| 维度 | **Prednisone** | **Prednisolone** |
|---|---|---|
| 类别 | Prodrug（需肝代谢激活）| 活性形式 |
| 用法 | 口服一线 GC | 严重肝病 / 儿科 / 局部用 |
| 临床效力 | 经肝转 prednisolone 后等效 | 直接活性 |
| Sound-alike 风险 | 名字几乎一样 → 处方混淆 | 同上 |

### 2.3 Hydroxyzine vs Hydralazine

| 维度 | **Hydroxyzine** | **Hydralazine** |
|---|---|---|
| 类别 | **H1 抗组胺**（first-gen sedating）| 直接 **vasodilator**（动脉为主）|
| 适应症 | 焦虑 + 瘙痒 + 失眠（短期）| HTN（孕期 safe）+ HF |
| 主要副作用 | 嗜睡 + 抗胆碱效应（口干 / 谵妄老人）| 反射性 tachycardia + lupus-like syndrome（slow acetylators）|
| Sound-alike 风险 | 名字头 "Hydr" + 尾 "-zine/-zine" 极易混 | 同上 |

### 2.4 防 Sound-alike Error 5 大工具（Patient-Centered Counseling）⭐⭐⭐

> [!success] **Teach-back 是最强工具 — 让患者用自己话 describe changes → 直接 detect miscomprehension**

| 方法 | 主动/被动 | 效力 |
|---|---|---|
| **Teach-back** ⭐⭐⭐ | 主动 | 最强 — patient describes their understanding |
| **Plain-language**（drug names + indication + dose + frequency）| 主动 | 强 |
| **Medication reconciliation**（multi-provider checkpoints）| 主动 | 强 — 防 omission |
| **Tall-man lettering**（如 "predniSONE" vs "predniSOLONE"）| 系统级 | 中 |
| "Ask if any questions" | **被动** | 弱（患者不知该问什么；通常答 "no"）|

### 2.5 ADE Risk Factors 3 大维度（sound-alike 是其中之一）

| 维度 | Risk Factors |
|---|---|
| **Patient** | Polypharmacy / Low health literacy / Very young or advanced age / Communication barriers / Nonadherence |
| **Provider** | Vague Rx / Incorrect med reconciliation / Absence of counseling / Illegible handwriting |
| **Medication** ⭐ | **Look-/sound-alike drugs**（本节核心）|

### 2.6 UW 经典 stem（Q19788）

- 83M + HTN + 高血脂 + admitted symptomatic hypoglycemia → 既往 **Glucovance**（metformin + glyburide）→ 出院 stop combined + 改 **Glucophage**（metformin only）→ 6 天后 mental status change + glucose 40 → 患者**仍服 combined pill**
- **关键陷阱**：药名 "Gluco-" 前缀 + 老人 + low health literacy → 没 detect change
- **正解**：teach-back（"Ask patients to describe the changes made in their treatment"）
- **反 pattern**：
  - "Ask if questions" = 被动 → 弱
  - Avoid combination drugs = 错（combination 增 adherence + convenience）
  - Family member present = 该患者 cognitively intact + 独立 → direct engagement 优先
  - Printed discharge summary = passive

### 2.7 Memory Hook

> [!success] 锁定口诀
> - **"Gluco-vance = 含 vance（=glyburide）→ 致 hypo"** ⭐
> - **"Gluco-phage = phage 吃糖（metformin 减糖）→ 不致 hypo"** ⭐
> - **"看到 sound-alike → 强制 teach-back"**（不是"ask if questions"）
> - **"Cognitively intact 患者 → direct engagement，不靠 family"**

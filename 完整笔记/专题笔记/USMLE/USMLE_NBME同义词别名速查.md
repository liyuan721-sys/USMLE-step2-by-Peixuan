# NBME 同义词 / 别名速查 (Alternative NBME Vocabulary)

> [!info] 用途与来源
> NBME / UWorld 故意**换说法**考你——把熟悉的病用一个生僻同义词（旧名 / 学名 / 描述性命名）写出来，认不出就丢分。这页把高频"同义词 ⇄ 常用名"按系统集中，**复习时反向自测：看到左列能不能立刻反应右列**。源：Demi (Divine Intervention) RF/Screening/Vaccine 大全 §Alternative NBME Vocabulary。

> [!tip] 用法
> 这页是**纯认词反射**工具，不展开机制——机制点到对应分科/专题笔记。盖住右列默写。

## 一、影像 / 操作 (Imaging & Procedures)

| NBME 说法 | = 常用名 / 含义 |
|---|---|
| Virtual colonoscopy 虚拟结肠镜 | **CT colonography** CT 结肠成像 |
| Laparoscopy 腹腔镜 | 微创"钥匙孔"手术（小切口）|
| Laparotomy / celiotomy 剖腹术 | 腹部大切口（exploratory laparo**to**my 剖腹探查）|
| Defibrillation 除颤 | **非同步**电复律 (unsynchronized cardioversion) |
| Direct current cardioversion / countershock | **同步**电复律 (synchronized cardioversion) |
| Polysomnography 多导睡眠图 | **Sleep study** 睡眠监测 |
| Spiral CT scan 螺旋 CT | **CT angiography** CTA（注意："pulmonary angiography 肺血管造影" ≠ CTA）|

## 七、神经 (Neurology)

| NBME 说法 | = 常用名 |
|---|---|
| Embolic stroke of undetermined source (ESUS) | **隐源性卒中 (cryptogenic stroke)**（旧名）|
| Spinal dysraphism 脊柱裂畸形 | **神经管缺陷 (neural tube defects)**（含 spina bifida occulta）|
| Simple partial seizure 单纯部分性发作 | **局灶性发作不伴意识丧失 (focal seizure without loss of awareness)** |
| Acute inflammatory demyelinating polyneuropathy (AIDP) | **Guillain-Barré 综合征** |
| Werdnig-Hoffmann disease | **脊髓性肌萎缩 1 型 (SMA type 1)** |
| Multi-system atrophy (MSA) / Shy-Drager | 帕金森叠加 + 自主神经衰竭（直立性低血压/无汗/阳痿）|
| Hereditary motor and sensory neuropathy (HMSN) | **Charcot-Marie-Tooth 病** |
| Hereditary hemorrhagic telangiectasia (HHT) | **Osler-Weber-Rendu 综合征** |
| Multi-infarct dementia 多发梗死性痴呆 | **血管性痴呆 (vascular dementia)** |
| Encephalotrigeminal angiomatosis | **Sturge-Weber 综合征** |
| Von Recklinghausen disease | **神经纤维瘤病 1 型 (NF1)** |
| Reflex sympathetic dystrophy 反射性交感神经营养不良 | **复杂区域疼痛综合征 (CRPS)** |

# 呼吸抑制 / 低通气鉴别（Respiratory depression / Hypoventilation DDx）

> 派生自 [[完整笔记/专题笔记/USMLE/USMLE_毒物综合征toxidrome速查]] §五（阿片中毒）延伸：见"呼吸↓/低通气"别只锁阿片。
> 用法：先抓 **神志 + 瞳孔 + 肌力 + 自主神经线索** → 一刀切分**中枢抑制 / 神经肌肉 / 代谢**三类。跨毒理 + 神经 + OB，high-yield。

---

## 〇、总纲：一句话三分类 ⭐

> [!info] 呼吸抑制 = "驱动↓" 还是 "推不动气" 还是 "代谢慢"
> ```
> ① 中枢驱动↓（CNS 抑制）   ："不想呼吸" → 频率↓ + 神志↓        （阿片/可乐定/镇静药/脑干）
> ② 神经肌肉无力           ："想呼吸但推不动气" → 频率可↑但浅弱 + 神志清 （GBS/MG/肉毒/有机磷/高镁）
> ③ 代谢/内分泌           ：驱动钝化或呼吸肌弱                （黏液水肿/低钾低磷/肥胖低通气）
> ```
> **第一刀 = 看神志清不清**：神志↓ → 中枢抑制；神志清 + 肌无力 → 神经肌肉。

---

## 一、中枢驱动↓（CNS 抑制——"不想呼吸"）⭐

呼吸频率↓ + 神志↓，脑干呼吸中枢被压。

| 病因 | 区分钥匙 | 处理 |
|---|---|---|
| **阿片 opioid**（标杆）| **针尖瞳 + 昏睡 + 呼吸↓** 三联 | 纳洛酮 naloxone 逆转 + 通气支持 |
| **可乐定 clonidine 中毒**（α2 激动）⭐ | **最像阿片的李鬼**：缩瞳 + 缓脉 + 低压 + CNS 抑制 + 呼吸↓；儿童误服一片可致命 | 支持为主；**可部分对纳洛酮有反应** |
| **镇静催眠药 sedative-hypnotic** | 苯二氮䓬 benzodiazepine（**单用少致命，合用酒精/阿片才重**）、巴比妥 barbiturate、酒精 alcohol、GHB（γ-羟基丁酸）；瞳孔多正常 | 支持；BZD → 氟马西尼 flumazenil（慎，可致癫痫）|
| **脑干卒中/出血**（脑桥出血 pontine hemorrhage）、**颅压↑ (↑ICP) / 脑疝** | 针尖瞳但**对光有反应** + 突发 + 局灶脑干征 + 高血压史 | **头 CT**；纳洛酮无效 |
| **CO₂ 麻醉 (CO₂ narcosis)** | 慢性高碳酸血症（COPD 慢阻肺）吸高浓度氧后驱动消失 | 控制性氧疗（目标 SpO₂ 88–92%）|

> [!danger] 两大阿片"李鬼"（别一看缩瞳+呼吸↓就只想海洛因）
> **① 可乐定 clonidine**（缩瞳 + 缓脉 + 呼吸↓，几乎复刻阿片，且部分应答纳洛酮）；**② 镇静药 + 酒精/阿片合用**（BZD 单用很少致命呼吸抑制）。

## 二、神经肌肉无力（"想呼吸但推不动气"）⭐⭐

驱动正常，但呼吸肌/膈肌没力 → 通气衰竭。**神志通常清醒**（与中枢抑制的关键区分）。**评估指标 = FVC（用力肺活量）/ NIF（最大吸气负压）**，别等 SpO₂ 掉（晚）或血气恶化才插管。

| 病因 | 区分钥匙 | 处理要点 |
|---|---|---|
| **吉兰-巴雷综合征 Guillain-Barré (GBS)** | **上行性**对称无力 + **腱反射消失** + 近期感染史（弯曲菌 Campylobacter / 病毒）| 查 **FVC < 20 mL/kg 或 NIF** → 插管；IVIG（静脉注射免疫球蛋白 intravenous immunoglobulin）或血浆置换 |
| **重症肌无力危象 myasthenic crisis**（MG = 重症肌无力 myasthenia gravis）| **波动性**无力 + 易疲劳 + 上睑下垂/复视，**活动后加重** | 插管支持；IVIG / 血浆置换；避免诱因药 |
| **肉毒中毒 botulism**（含婴儿肉毒 infant botulism）| **下行性**麻痹 + 瞳孔散大固定 + 便秘 + 脑神经先受累（复视/吞咽难）| 抗毒素；婴儿 → 肉毒免疫球蛋白 BIG-IV |
| **有机磷 organophosphate 中毒** | 胆碱能 DUMBELS：大汗/流涎/**缩瞳**/缓脉 + 肌束颤 fasciculations + **支气管分泌物淹肺** | 阿托品 atropine + 解磷定 pralidoxime |
| **高镁血症 hypermagnesemia** ⭐（OB 高频）| **子痫前期产妇用硫酸镁 MgSO₄** → 呼吸抑制 + **深腱反射消失（先消失，早期预警）** | 停镁 + **葡萄糖酸钙 calcium gluconate**（解药）|
| **高位脊髓损伤**（C3–5 膈神经 phrenic nerve）、肌萎缩侧索硬化 ALS、蜱麻痹 tick paralysis | 外伤史 / 进行性运动神经元病 / 找蜱 | 病因处理 + 通气支持 |

> [!warning] 高镁血症是 OB 必考李鬼 ⭐
> 子痫前期/子痫 (preeclampsia/eclampsia) 用 **硫酸镁 MgSO₄** 防抽搐时，过量 → **腱反射消失（先）→ 呼吸抑制 → 心脏停搏**（剂量递增顺序）。
> - **监测**：用药中查**膝反射 + 呼吸频率 + 尿量**；反射消失 = 第一预警。
> - **解药 = 葡萄糖酸钙 calcium gluconate** IV（钙拮抗镁）。

> [!info]- 附图：膈神经走行（C3-5 → 膈肌；"C3,4,5 keep the diaphragm alive"）
> ![[{B6DF0989-3B26-4744-AE74-475A9BA54DF9}.png]]
> 高位脊髓损伤 (C3-5) / 膈神经 (phrenic nerve) 损伤（纵隔肿瘤、心脏术后、肺癌侵犯）→ 膈肌麻痹 → 通气衰竭（§二 神经肌肉无力一类）。

## 三、代谢 / 内分泌

| 病因 | 区分钥匙 | 处理 |
|---|---|---|
| **黏液水肿昏迷 myxedema coma**（严重甲减 hypothyroidism）| 低通气 + 低体温 + 低钠 + 反应迟钝 + 甲减史 | 左甲状腺素 levothyroxine + 氢化可的松 hydrocortisone |
| **严重低钾 / 低磷血症** | 呼吸肌无力；查电解质 | 纠正电解质（见 [[完整笔记/专题笔记/肾脏/肾脏_电解质急症速查]]）|
| **肥胖低通气综合征 OHS (obesity hypoventilation syndrome) / OSA（阻塞性睡眠呼吸暂停 obstructive sleep apnea）** | 肥胖 + 日间嗜睡 + 高碳酸血症 | 减重 + 夜间无创通气（CPAP 持续气道正压 / BiPAP 双水平气道正压）|

## 四、床旁分流思路 ⭐

```
呼吸抑制 → ① 查 神志 + 瞳孔
   缩瞳 + 昏睡        → 阿片？可乐定？      → 先试纳洛酮（醒=阿片/部分=可乐定）
   神志清 + 肌无力     → 神经肌肉(GBS/MG/肉毒/高镁) → 查 FVC/NIF + 腱反射 + 病史
   突发 + 局灶脑干征   → 脑干出血           → 头 CT（纳洛酮无效）
   低体温+甲减史      → 黏液水肿昏迷
② 别漏"可纠正解药"：
   阿片→纳洛酮 naloxone  ;  BZD→氟马西尼 flumazenil(慎)
   高镁→葡萄糖酸钙 calcium gluconate  ;  有机磷→阿托品 atropine + 解磷定 pralidoxime
```

## 五、Memory Hook

> [!success] 一句话
> **"压驱动 vs 推不动气 vs 代谢慢"三类。** 压驱动里**可乐定 + 镇静药合用**是阿片两大李鬼（都缩瞳+呼吸↓）；推不动气里**高镁血症（产妇 MgSO₄ → 反射先消失）+ GBS（上行）/ 肉毒（下行）**最常考；**神志清不清**一把分中枢 vs 神经肌肉。
>
> 钥匙：**缩瞳+昏睡=中枢（阿片/可乐定）；神志清+反射消失=神经肌肉（GBS/高镁）；上行 GBS / 下行肉毒。**

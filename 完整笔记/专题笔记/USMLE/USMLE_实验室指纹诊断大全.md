## Part 0. 指纹思维核心方法论

> [!success] 三步识别法
> 1. **跳过症状迷雾**：当 stem 给"体重↓ / 疲劳 / GI 症状"等不特异线索时，**先扫实验室**
> 2. **识别指纹组合**：找 3+ 项异常聚集 → 触发"哪个病的指纹长这样？"
> 3. **反推验证**：用慢性病程 / 应激触发 / 阴性体征做最终锁定

### 0.1 指纹的 4 种形态

| 形态 | 例子 | USMLE 出题套路 |
|---|---|---|
| **N 联指标组合** | PAI 五联（Na↓ / K↑ / HCO3↓ / Glu↓ / Eos↑）| 给完整表，问诊断 |
| **单一超特异性指标** | M-spike → MM；5-HIAA → 类癌 | 给一个金标准，问下一步 |
| **比值 / 间隙** | AG / 渗透压间隙 / TTKG / FENa | 给计算，问机制 |
| **形态学指纹** | 镜下血涂片（schistocytes / blasts / 棘细胞）| 给图，问诊断 |

### 0.2 三层锁定法（PAI 模板，可推广）

```
第 1 层：实验室指纹（必有）
       ↓ 看到 N 项异常聚集
第 2 层：慢性病程 / 应激触发（背景舞台）
       ↓ "数月体重↓"+"突发事件"
第 3 层：阴性体征（排除竞争诊断）
       ↓ "no rash, no wheezing"排除 Anaphylaxis
最终：锁定诊断 → 选第一步处理
```

## Part 1. 内分泌 / 肾上腺指纹

### 1.1 PAI / Addison's ⭐⭐⭐（指纹病祖师爷）

| 指纹 | Na↓ + K↑ + HCO3↓ + Glu↓ + Eos↑（**5 联**）|
|---|---|
| **机制核心** | Aldo↓（→ Na↓ / K↑ / HCO3↓）+ Cortisol↓（→ Glu↓ / Eos↑）|
| **加分指纹** | 皮肤色素沉着（ACTH↑→MSH 效应，仅原发）|
| **危象触发** | 应激（手术 / 感染 / 创伤 / etomidate）|
| **详见** | [[完整笔记/专题笔记/内分泌/内分泌_肾上腺皮质功能不全]] |

### 1.2 Cushing 综合征

| 指纹 | K↓ + HCO3↑ + Glu↑ + Na 正常↑ + HTN + 中央型肥胖（满月脸 / buffalo hump / 紫纹）|
|---|---|
| **机制核心** | Cortisol↑↑ → 糖异生↑（高糖）+ 大剂量 cortisol 占用盐皮质受体（低钾 / 代碱）|
| **诊断金标准** | 24h 尿游离 cortisol / late-night salivary cortisol / 1-mg dexamethasone 抑制试验 |
| **病因鉴别** | ACTH 依赖（垂体腺瘤 = Cushing disease；异位 ACTH = 小细胞肺癌）vs ACTH 独立（肾上腺腺瘤 / 癌）|
| **详见** | [[完整笔记/Peixuan分科笔记/endocrine]] |

### 1.3 Conn 综合征（原发性醛固酮增多症）

| 指纹 | Na 正常↑ + K↓ + HCO3↑ + HTN（**难治性**，常 < 50 岁）|
|---|---|
| **机制核心** | Aldo↑ → Na 潴留（容量↑→HTN）+ K 排泌↑+ H+ 排泌↑（代碱）|
| **诊断三部曲** | ① **筛查**：晨起 ARR（PAC/PRA）> 20 + PAC > 15 ng/dL ② **确诊**：生理盐水负荷试验（盐水后 aldo 不被抑制）③ **定位**：肾上腺 CT → 不明 / 双侧增生 → **AVS**（肾上腺静脉采样） |
| **vs Cushing** | Conn 不胖 / 不糖尿病 / 无满月脸；Cushing 胖 + 糖尿病 + 紫纹 |
| **病因** | 单侧腺瘤（Conn's tumor，手术治愈）/ 双侧增生（spironolactone）|
| **ARR 筛查 4 适应证** | ① 顽固性 HTN（3 药联控失败）② HTN + 低钾（**自发或利尿剂诱发**，"小剂量利尿剂出现不成比例严重低钾"= 经典 stem）③ 年轻 HTN（< 30 岁）④ **意外瘤**患者（incidentaloma，≥ 1 cm，体检偶然发现） |
| **"意外瘤"三联必筛** | Pheo（**metanephrines**）+ Cushing（**LDDST 或 24h UFC**）+ 仅 HTN/低钾 者加 PA（**ARR**） |

### 1.4 嗜铬细胞瘤（Pheochromocytoma）

| 指纹 | **24h 尿 metanephrines + normetanephrines ↑↑**（金标准）|
|---|---|
| **临床三联** | 阵发性头痛 + 出汗 + 心悸（+ HTN paroxysm）|
| **机制核心** | 肾上腺髓质 chromaffin 细胞分泌 catecholamines |
| **MEN 2A/2B 关联** | 双侧嗜铬 + 甲状腺髓样癌 + 甲旁亢（2A）/ 黏膜瘤（2B）|
| **治疗顺序** | **α-blocker（phenoxybenzamine）先**，再 β-blocker，再手术（不能先 β，会致 unopposed α）|

### 1.5 DKA（糖尿病酮症酸中毒）⭐

| 指纹 | **Glu↑↑** (300-800) + **HCO3↓↓** (<18) + **AG↑↑** + 酮↑ + pH↓ + Kussmaul 呼吸 |
|---|---|
| **机制核心** | 胰岛素绝对不足 → 脂肪分解产酮（β-OHB + AcAc）+ 糖异生失抑 → 高糖渗透性利尿 + AG 代酸 |
| **典型患者** | **年轻 + 1 型 DM**（可首次起病 stem）|
| **触发 5I** | **I**nfection / **I**nsulin 漏打 / **I**nfarct (MI) / **I**schemia / **I**nitial（首次起病）|
| **核心钾指纹** | **Pseudohyperkalemia**：血清 K **表面正常 / ↑**（酸 → H+ 进细胞、K+ 出细胞 + 渗透性利尿丢 K），实际**全身 K 极度缺乏**。治疗中 K 必崩。 |
| **AG 关闭判停点** | 治疗目标 = AG 8-12 + HCO3 ≥ 15 + 能吃饭 + Glu < 200 → 才换皮下胰岛素（**且 IV / SQ 重叠 1-2 h**）|

> [!danger] DKA 管理顺序（必背 SOP）
> ```
> Step 1: ABC + 抽血 + 查 K
> Step 2: IV NS 0.9% 大量补液（1-2 h 内 1-2 L）
> Step 3: 查 K → 决策点
>     - K < 3.3 → 暂缓 insulin，先补 K（否则胰岛素会让 K 暴跌到致命）
>     - K 3.3-5.3 → 同时补 K + 上 insulin
>     - K ≥ 5.3 → 暂不补 K，直接上 insulin
> Step 4: IV regular insulin（持续输注）
> Step 5: Glu 降到 200 → **液体换为 D5W**（加葡萄糖防低血糖，
>          继续打 insulin 直到 AG 关闭 — 关键不是降糖,是解决酮症)
> Step 6: 监测 AG / HCO3 → AG 8-12 即"resolved"
> Step 7: 转皮下 insulin + 喂饭 + 转 ICU
>          ★ IV insulin 与 SQ overlap 1-2 h 才能停 IV
> Step 8: pH ≤ 6.9 才考虑 NaHCO3；Phos < 1.0 / 心衰 / 呼衰才补 Phos
> Step 9: 找诱因（感染 / MI / 妊娠）
> ```

> [!warning] DKA vs Type 4 RTA 钾问题对比
> - **DKA**：全身缺 K，但血清 K 假性正常 / 高（H+ 内移驱赶 K+ 外移 + 胰岛素缺乏 K 跑出细胞）→ 治疗中 K 暴跌
> - **Type 4 RTA**：醛固酮抵抗，肾不能排 K → **真性高 K**

### 1.6 HHS（高血糖高渗状态）⭐

| 指纹 | **Glu↑↑↑** (>600，常 >1000) + **血浆渗透压↑↑** (>320) + **HCO3 正常** + **无 / 极少酮** + 意识改变 / 局灶神经体征 |
|---|---|
| **机制核心** | 残余胰岛素足以抑制脂解（**无酮**），但不足以控糖 → 渗透性利尿 → 极度脱水 + 高渗 |
| **典型患者** | **老人 + 2 型 DM**（常合并 dementia / 卒中 / 感染，无法主动饮水加重脱水）|
| **vs DKA** | HHS 老 / 2 型 / 无酮 / 无 AG / 渗透压 ↑↑ / Glu 更高；DKA 年轻 / 1 型 / 有酮 / 有 AG |
| **Na 校正公式** | **校正 Na = 测得 Na + 1.6 × ((Glu − 100) / 100)** — HHS 常见"稀释性低钠"，但校正后多正常或高（提示真性脱水）|
| **K 问题** | 比 DKA 缺得更多（更长时间渗透性利尿）|

> [!danger] HHS 管理（与 DKA 区别）
> - **补液更激进**：常需 **9 L+**（比 DKA 多得多，因为脱水更重）
> - **胰岛素剂量小 / 慢**：避免快速降糖致脑水肿；目标 Glu 50-70 mg/dL/h
> - **同样监测 K**，规则同 DKA
> - **找诱因**（感染最常见）

### 1.7 SIADH ⭐

| 指纹（5 项）| **Na↓** + **血浆渗透压↓** (<275) + **尿 Osm↑**（不适当浓缩，>100）+ **euvolemic** + **尿 Na↑**（>30-40）|
|---|---|
| **机制核心** | ADH 不适当分泌 → 自由水潴留 → 稀释性低钠 |
| **euvolemic 但尿 Na > 30 的机制** | 体内多出来的一点水**轻度抑制 RAAS + 促进排钠**，所以**肾在留水（尿 Osm↑）但不留钠（尿 Na↑）** — 这是 SIADH 与"低容量低钠"的关键鉴别 |
| **病因 5 大类** | ① **异位 ADH**：SCLC（最经典）② **CNS**：stroke / 出血 / 创伤 / 肿瘤 ③ **肺**：肺炎 / TB / abscess ④ **药物**：carbamazepine / SSRIs / NSAIDs / cyclophosphamide / 鼻吸 DDAVP ⑤ 疼痛 / 恶心 / 术后应激 |
| **治疗阶梯** | ① **限水**（根本：水多不是钠少）② 难治 → **demeclocycline**（诱导肾性 DI，让 ADH 不灵）或 **vaptans**（V2 拮抗 → aquaresis）③ Na < 120 或神经症状 → **3% NS slowly**（防 ODS）④ 神经症状 + 容量过载 → **3% NS + furosemide** |

> [!danger] Demeclocycline 致 nephrogenic DI 的生理意义
> Demeclocycline 是**四环素类抗生素 + V2 受体后效应抑制剂** → 肾对 ADH 不敏感 → "诱导出"肾性 DI → 排出多余自由水。是 SIADH 难治时的"以毒攻毒"策略，但本身也会致**真正的 nephrogenic DI**（USMLE 鉴别中常作为 nephrogenic DI 的药物病因之一）。

> [!warning] 低钠纠正速度
> - **慢性低钠**：纠正速度 < 8 mmol/L/24h，否则 **ODS**（osmotic demyelination syndrome / 渗透性脱髓鞘）
> - **急性低钠 + 重症神经症状**：可在 1-2 h 内 3% NS 推注小剂量提 4-6 mmol/L
> - **Na < 120 + 无症状** ≠ 必须 3% NS（限水即可）

### 1.8 中枢性 vs 肾性 DI ⭐

> [!tip]+ 渗透压正常值速查（也适用 §1.7 SIADH / §1.5 HHS）⭐
>
> | 项目 | 正常范围 | 关键阈值 |
> |---|---|---|
> | **血浆 Osm (Posm)** | **275–295 mOsm/kg**（中点 ~285）| < 275 = SIADH / 水中毒；> 295 = DI / 脱水 / 高钠 / 高血糖；**> 320 = HHS** |
> | **尿 Osm (Uosm)** | **50–1200 mOsm/kg**（跨度大）| 最稀 50 / **等渗 300** / 最浓 1200（年轻人 ~1400）|
> | **Posm 计算式** | 2×Na + Glu/18 + BUN/2.8 | 实测 − 计算 = **Osm gap**（>10 → 甲醇 / 乙二醇 / 异丙醇）|
>
> **DI / SIADH 锁定 Uosm 阈值（反直觉考点）**：
> - **DI**：Posm ↑（>295）但 **Uosm < 300**（应浓缩没浓缩）★
> - **SIADH**：Posm ↓（<275）但 **Uosm > 100**（应稀释没稀释）★
> - **等渗尿 Uosm ≈ 300** = 严重 CKD / ATN 失浓缩能力
> - **正常脱水** Uosm > 800 / **正常水合** Uosm 200-500
>
> **记忆口诀**：血浆 **275-295**（中点 285 ≈ 正常 Na × 2）；尿三锚点 **50 / 300 / 1200**（最稀 / 等渗 / 最浓）

| 指纹 | **多尿（>3 L/d，稀尿）+ 稀释尿 Osm↓**（<300）+ **USG < 1.006** + 高 / 正常血 Na（"dry inside, soaked outside"）|
|---|---|
| **机制鉴别金标准** | **水剥夺试验 + DDAVP**：① 水剥夺后尿 Osm 仍低（不浓缩）= DI；② 再给 DDAVP → 尿 Osm 升 > 50% = **中枢性**；升 < 10% = **肾性** |
| **中枢性病因** | 头部外伤 / 垂体手术 / 鞍区肿瘤 / 缺血性脑病 / 特发；治疗：MRI 垂体 + **DDAVP** |
| **肾性病因** | ① **Lithium**（最经典）② **Demeclocycline**（SIADH 长期治疗后）③ **Amphotericin B** ④ **Hypercalcemia** → 损害肾髓质浓缩功能 → "DI-like" 表现 ⑤ 遗传（AVPR2 / aquaporin 2）|
| **肾性治疗** | **Thiazide**（反常抗利尿：轻度容量收缩 → 近端重吸收↑→远端 delivery↓→ 尿量↓）+ 限钠 + **amiloride**（对 lithium DI 特别有用：阻 ENaC → 减少 lithium 进主细胞）|

> [!info] 高钙致肾性 DI 的生理
> 高血钙 → 集合管 CaSR 激活 → 拮抗 ADH 信号通路 → aquaporin-2 表达↓ → 尿无法浓缩。"vitamin D 过量 / 维生素 D 中毒 / 甲旁亢患者"多尿题 → 要查 Ca。

> [!warning] 妊娠期多尿 3 鉴别
> - **妊娠期 DI**：胎盘 **vasopressinase** 降解 ADH → USG **< 1.006** + > 3 L/d，分娩后消退
> - **妊娠期 DM**：渗透性利尿，USG **> 1.020**
> - **生理性尿频**：子宫压迫膀胱，24h < 3 L，USG 正常

### 1.9 1° HPT（原发性甲旁亢）⭐

| 指纹 | **Ca↑ + PO₄↓ + PTH↑ + Cl/PO₄ ratio > 33**（4 联锁定）|
|---|---|
| **机制核心** | 腺瘤 / 增生**自主分泌 PTH** → ① 骨吸收↑（释 Ca）② 远端肾小管 Ca 重吸收↑ ③ 近端肾小管 PO₄ 排泌↑（→ ↓PO₄）④ 1,25-(OH)₂D↑（肠 Ca 吸收↑）|
| **典型患者** | 老年女性 / 体检偶发高钙 / 反复肾结石 / "stones, bones, abdominal groans, psychic moans" |
| **诊断流程** | 血 Ca + iPTH + 24h 尿 Ca（**排 FHH**，FHH 尿 Ca < 100）+ DXA + 肾超声 |
| **手术指征**（无症状者满足 1 项即手术）| ① 年龄 < 50 / ② Ca > 上限 1 mg/dL / ③ T-score ≤ -2.5 或椎体骨折 / ④ CrCl < 60 / ⑤ 24h 尿 Ca > 400 或结石 / 肾钙化 |
| **详见** | [[完整笔记/Peixuan分科笔记/endocrine]] / [[完整笔记/Peixuan分科笔记/骨科]] |

> [!tip] Cl/PO₄ ratio 的鉴别用途（**核心高频陷阱**）
> 当 PTH borderline / 鉴别不清时，**Cl/PO₄ 比值帮助区分真 HPT vs PTHrP 介导**：
> - **1° HPT**：**Cl/PO₄ > 33**（常见 35-40）
> - **Malignancy (PTHrP)**：**Cl/PO₄ < 33**
> 
> **机制**：PTH → 远端肾小管 ↑Cl 重吸收 + 抑制近端 HCO3 重吸收 → **轻度高氯性代谢性酸中毒** + ↓PO₄ 重吸收 → 比值升高。
> **PTHrP 虽然模仿 PTH 的 Ca / PO₄ 效应，但不引起高氯性酸中毒**，所以 Cl/PO₄ < 33 — 这是 PTH vs PTHrP 的本质差异。

> [!warning] ↑Ca 4 因鉴别一张表（PTH + Cl/PO₄ 双锚定）
> 
> | 病因 | Ca | PO₄ | PTH | Cl/PO₄ | 鉴别要点 |
> |---|---|---|---|---|---|
> | **1° HPT** | ↑ | ↓ | **↑** | **> 33** | 老年女 / 偶发 / 肾结石 |
> | **Malignancy (PTHrP)** | ↑↑ | ↓ | **↓**（受抑）| **< 33** | 鳞癌 / 乳腺 / 肾癌 / 头颈 |
> | **Granulomatous (1,25-VD↑)** | ↑ | ↑ | ↓ | < 33 | Sarcoid / TB / 肺门淋巴 / ACE↑ |
> | **FHH** | ↑（轻）| 正常 | 正常 / 轻↑ | < 33 | **24h 尿 Ca < 100** + Ca/Cr clearance < 0.01 |

> [!info] 与 1.8 DI 的隐藏联动
> 1° HPT 严重时 → 高血钙 → **肾性 DI**（CaSR 激活拮抗 ADH → AQP2↓ → 尿无法浓缩）→ 多尿 + 多饮 stem。"多尿 + 高 Ca" 出现在同一题 → 优先想 HPT / VitD 中毒。

### 1.10 内源性高胰岛素血症 / 低血糖鉴别 ⭐

| 指纹 | **Whipple 三联征**（症状 + 实测 Glu↓ + 补糖缓解）+ **Insulin** / **C-peptide** / **Proinsulin** / **磺脲类血药筛查** 四联化验 |
|---|---|
| **机制核心** | C-peptide = 自产胰岛素的"身份证"（与 insulin 等摩尔从 proinsulin 剪切而来）；外源针剂无 C-peptide → C-pep 是内源 vs 外源的分水岭 |
| **典型患者** | 见下表 4 行 stem 关键词 |
| **诊断 SOP** | ① 先确认 Whipple 三联征（不是化验偶发低糖）② 补糖前抽 insulin + C-pep + proinsulin + 磺脲类筛查 + β-hydroxybutyrate ③ 按下表锁分支 |

**「一家人」四联对比表**：

| 病因 | Insulin | C-peptide | Proinsulin | 磺脲类筛查 | Stem 关键词 |
|---|---|---|---|---|---|
| **Insulinoma** | ↑ | ↑ | ↑ | 阴性 | >40 岁、孤立肿瘤、MEN1 关联 |
| **磺脲类偷用**（Surreptitious sulfonylurea）| ↑ | ↑ | ↑ | **阳性** ⭐ | 年轻、家属 DM、医护/有药渠道 |
| **外源注射胰岛素** | ↑ | **↓** ⭐ | ↓ | 阴性 | 医护、抑郁、Munchausen、C-pep 低 |
| **酒精/饥饿/败血症/肾上腺/垂体功能↓** | ↓/正常 | ↓/正常 | — | 阴性 | 酗酒、营养不良、危重、Addison |

> [!success] 一句话区分本质 — C-peptide 是身份证
> 自产胰岛素带身份证（C-pep 与 insulin 等摩尔成对↑）；外面打进来的针剂没身份证（insulin↑ 但 **C-pep↓**）。**两都高 = 内源**；磺脲筛查阳性 = 偷药；C-peptide↓ = 注射胰岛素。

> [!warning] 高频陷阱（必看）
> - **Insulinoma vs 磺脲偷用化验完全一样**（都两高 + proinsulin 高）→ 唯一区别 = **磺脲类血药筛查**；年轻 + 有药物渠道 → 概率上先想磺脲。
> - **C-peptide↓ 反向题**：insulin↑ + C-pep↓ 直接锁"外源注射胰岛素"，不要被"insulin 高"骗去想 insulinoma。
> - **化验题眼 > 人口学画像**：UW 经典套路用"年轻女学生 + 期末压力 + 偏瘦"诱导你选进食障碍/催吐（[[mistakes/uworld-mistakes#^Q20217|Q20217]] 触发题）；化验值已锁死分支，人口学只是钓饵。
> - **酒精 / 饥饿 / 呕吐性低血糖**：insulin 不高（非胰岛素介导）；呕吐还会有 OH + 代碱 + 低氯（一同审）。
> - **"家属有 DM" / "医护身份"** = UW 固定的**药物获取渠道**关键词 → 立刻怀疑 surreptitious。

### 2.3 RTA 四型对比 ⭐

| 类型 | K | 尿 pH | AG | 机制 | 经典病因 |
|---|---|---|---|---|---|
| **Type 1**（远端）| **↓** | **> 5.5** | 正常 | DCT 排 H+ 障碍 | Sjögren / 自免 / amphotericin |
| **Type 2**（近端）| **↓** | < 5.5（早期）| 正常 | PCT 重吸收 HCO3 障碍 + Fanconi | MM / Wilson / 重金属 / acetazolamide |
| **Type 4**（醛固酮抵抗）⭐ | **↑** | < 5.5 | **正常** | Aldo 缺乏 / 抵抗 → K 不能排 + H+ 排泌↓ | 见下表 |

> [!danger] 入口反射：「低钾 + 高氯」先想 NAGMA，再用血钾切 Type 4 ⭐
> **看到「低 K + 高 Cl + 低 HCO3」第一反应 = NAGMA（正常 AG 代酸）**，别急着算 AG 也已八九不离十。
> **关键捷径——先用血钾把 Type 4 切出去**：Type 4 是四病因里唯一**高钾**的（aldo 抵抗 → 排不出 K）。
> - **低钾 + 高氯 → 基本排除 Type 4**，只剩 **腹泻 / Type 1 / Type 2**（再走下面两步）
> - **高钾 + 高氯 → 直奔 Type 4**（老糖人 + 糖尿病肾病，见下方完整指纹）
> 一句话：**「钾低看远近端 RTA / 腹泻，钾高直接 Type 4」**。

> [!tip] NAGMA（正常 AG 代酸）两步定位 + 尿 AG ⭐（Q18881 触发增量）
> NAGMA = 低 HCO3 + **正常 AG** + 高 Cl（hyperchloremic）。四病因 = **腹泻 / Type 1 / Type 2 / Type 4 RTA**。
>
> **第 1 步——看尿能否酸化（代酸时尿 pH）**：
> - 尿 pH **> 5.5（不恰当偏碱）= 远端 Type 1 RTA**（DCT 泌 H⁺ 障碍，尿酸化不了）⭐
> - 尿 pH **< 5.5（肾正常代偿）= 腹泻 / Type 2 / Type 4**
>
> **第 2 步——分腹泻 vs 肾性，算尿 AG = U_Na + U_K − U_Cl**：
> - **负值 = 腹泻**（"**neGUT-ive**"，肠丢 HCO3，肾代偿排 NH4Cl 把 Cl 拉高 → 公式变负）
> - **正值 = RTA**（肾排不出 NH4⁺/H⁺，无足够未测阴离子 Cl）
>
> **配 Winter 公式确认无叠加呼吸紊乱**：预期 PaCO2 = 1.5×HCO3 + 8 (±2)。例：HCO3 13 → 预期 ≈ 27.5；实测 26 = **单纯代酸的恰当呼吸代偿**（不是原发呼酸/呼碱）。
> 易错点：代酸患者 PaCO2 低（如 26）是代偿，**不是呼吸肌障碍/呼酸**（呼酸 PaCO2 应 > 40）。

**NAGMA 四病因一表分（含腹泻，K + 尿 pH + 尿 AG + Stem 线索）⭐**

| 病因 | K | 尿 pH | 尿 AG | Stem 线索 |
|---|---|---|---|---|
| **腹泻**（GI 丢 HCO3）| **↓** | 酸 < 5.5 | **负**（neGUTive）| 肠丢碱，肾正常酸化 |
| **Type 1 远端 RTA** | **↓** | **碱 > 5.5** ⭐ | 正 | Sjögren / 自免 / 肾结石 / amphotericin |
| **Type 2 近端 RTA** | **↓** | 多变（早 < 5.5）| 正 | Fanconi / MM / Wilson / acetazolamide |
| **Type 4 RTA** | **↑** | < 5.5 | 正 | 老糖人 + 糖尿病肾病 + 低肾素低醛 |

> 读表顺序：**先血钾**（高钾 → Type 4；低钾 → 余三者）→ **尿 pH**（>5.5 = Type 1）→ **尿 AG**（负 = 腹泻，正 = RTA）。

> [!info]- 为什么这样分？三条机制原理（能现推，不用背）⭐
> **① 为什么 NAGMA 一定"高氯"——电中性填空**
> 血浆电中性：Na ≈ Cl + HCO3 + 未测阴离子（AG）。**直接丢 HCO3⁻**（腹泻丢肠液 / RTA 肾丢碱）时，HCO3 腾出的位置被 **Cl⁻ 顶上来填**（肾代偿多重吸收 Cl）→ HCO3↓ + Cl↑ 但 **AG 不变**。所以 **hyperchloremic = normal AG 是同义**，看到"低 HCO3 + 高 Cl"不算 AG 也知道是 NAGMA。
> 对比：高 AG 代酸（DKA/乳酸）加进来的是**新阴离子**（酮酸/乳酸根）占了空位 → **AG 撑大，Cl 不涨**。
>
> **② 为什么血钾能切 Type 4——醛固酮统一轴（灵魂）⭐**
> 醛固酮在集合管**一手排 K⁺（principal cell）一手排 H⁺（α-intercalated cell）**，两件事绑定。
> - **腹泻 / Type 1 / Type 2**：多伴容量丢失 → RAAS 触发 **继发性醛固酮↑（系统在加班）** → K 被排掉 → **低钾**
> - **Type 4**：本质就是**醛固酮缺乏或抵抗**（老糖人 JG 细胞坏 = 低肾素低醛；或 ACEI/ARB/螺内酯/肝素阻断）→ K 排不出 → **高钾**
> > 钉死：**"前三个醛固酮在干活(低钾)，Type 4 醛固酮罢工(高钾)；血钾就是醛固酮系统的体检表。"**
>
> **③ 为什么 Type 4 也酸、尿 pH 还 < 5.5——双重机制**
> - 醛固酮也驱动 H⁺ 排泌，它少了 → H⁺ 排不出 → 酸。
> - **高钾本身抑制近端氨生成（ammoniagenesis）** → 尿 NH4⁺（排酸主力载体）产量↓ → 排酸能力↓ → 酸中毒。
> - **关键区别**：Type 4 的 H⁺ 泵本身是好的（尿能酸化 → 尿 pH < 5.5），坏在**缓冲不够**（没足够 NH4⁺ 携带 H⁺）；Type 1 是 **H⁺ 泵真坏了**（DCT 泌 H⁺ 障碍）→ 尿酸化不了 → **尿 pH > 5.5**。这就是表里 Type 1 vs Type 4 尿 pH 分道扬镳的机制根。

#### Type 4 RTA 完整指纹（"低肾素低醛 + 老糖人 + 高钾酸中毒"）

| 项目 | Type 4 RTA |
|---|---|
| **指纹组合** | **高 K + 正常 AG 代酸 + 尿 pH < 5.5 + 低肾素 + 低醛固酮** |
| **典型患者** | **老年 + 长病程糖尿病 + 糖尿病肾病** |
| **机制** | 糖尿病 / 间质肾炎 → JG 细胞受损 → 肾素↓ → 醛固酮↓ → 远端 K 排泌↓ + H+ 排泌↓ |
| **诱因 / 加重药** | **ACEI / ARB / 螺内酯 / 肝素 / NSAID / 钙调磷酸酶抑制剂**（这些药都进一步降低 aldo 或拮抗其效应）|
| **治疗** | ① 停止上述药 ② **Fludrocortisone**（补盐皮质）③ Loop diuretic（促 K 排）④ 限钾饮食 ⑤ NaHCO3 |

#### 儿科版「婴儿低钠 + 高钾 + 代酸」鉴别 ⭐（成人三镜像的新生儿镜像）

> [!info] 共同表现 = 新生儿失盐危象
> 1-4 周龄婴儿 **低钠 + 高钾 + 正常 AG 代酸 + 脱水 / 喂养差 / 体重不增**。先分两步：**① 皮质醇够不够（有无低血糖 / 低血压休克）② 醛固酮是低（缺乏）还是高（抵抗）**。

| 病因 | 醛固酮 | 皮质醇 | 雄激素 / 生殖器 | 一锤定音 |
|---|---|---|---|---|
| **21-羟化酶缺乏 CAH 失盐型** ⭐ | **↓** | **↓** | **↑**（女婴外生殖器男性化）| **17-OHP↑↑**；最常见；1-2 周龄危象 |
| **PHA1**（假性醛固酮减少症）| **↑↑（抵抗）** | 正常 | 正常 / 正常 | **醛固酮高是关键**；补盐，AD 型随龄改善 |
| **醛固酮合成酶缺乏** | **↓（孤立）** | 正常 | 正常 / 正常 | 单纯失盐，皮质醇 / 雄激素都正常 |
| **原发肾上腺功能不全**（新生儿 Addison）| **↓** | **↓** | 视病因 | 少见：肾上腺出血 / 发育不全 / ALD（X 连锁，男婴）|

> [!tip] 解题三步钩
> - **女婴外生殖器男性化 + 失盐 + 17-OHP↑↑ = 21-OH CAH** ⭐（治疗：hydrocortisone + fludrocortisone + 补盐 / NaCl）
> - **醛固酮 ↑↑（不是↓）→ PHA1**（受体抵抗，越补越高；治疗只能补盐 + 限钾，不是补盐皮质）
> - **皮质醇低（休克 / 低血糖）= 肾上腺源（CAH / Addison）；皮质醇正常 = 肾 / 受体源（PHA1 / 醛固酮合成酶缺乏 / Type 4）**
>
> 镜像对照：成人三剑客 = DKA（甜）/ Addison（黑）/ Type 4（老）；婴儿三剑客 = **CAH（生殖器）/ PHA1（醛固酮反高）/ 肾上腺危象**。逻辑同源：**醛固酮缺乏 vs 抵抗**定生死。

#### PHA（假性醛固酮减少症）专题：真性 vs 假性 + PHA1/PHA2 镜像 ⭐

> [!info]- PHA = Pseudohypoaldosteronism（点开看两型镜像）
> **"假性"= 表现像醛固酮不够（失盐 / 高钾 / 代酸），但醛固酮本身正常或↑↑，问题在肾小管抵抗。**
> | 维度 | **真性醛固酮减少**（Addison / CAH / 合成酶缺乏）| **PHA（假性）** |
> |---|---|---|
> | 醛固酮水平 | **↓（真低）** | **正常 / ↑↑** |
> | 病灶 | 肾上腺（产不出）| 肾小管（受体 / 通道抵抗）|
> | Fludrocortisone | **有效** | **无效**（受体不听）→ 只能补盐 / 限钾 |
>
> **PHA 两型长得相反，唯一共性 = 高钾 + NAGMA：**
> | | **PHA1**（经典型，儿科）| **PHA2 = Gordon 综合征**（成人 / 青少年）|
> |---|---|---|
> | 基因 / 机制 | ENaC（SCNN1，AR 重）/ MR（NR3C2，AD 轻）失活 → 醛固酮抵抗 | WNK1/4、KLHL3、CUL3 → **NCC（噻嗪靶点）活性过强** |
> | 钠 | **保不住 → 失盐** | **吸太多** |
> | 血压 | **低 / 正常** | **高** ⭐ |
> | 醛固酮 / 肾素 | 醛固酮 **↑↑** | **低肾素** |
> | 典型 stem | 新生儿失盐 + 高钾 + 醛固酮↑↑ + 皮质醇正常 | 高血压 + 高钾 + 低肾素 + NAGMA |
> | 治疗 | 补盐 + 限钾（AD 型随龄改善）| **噻嗪特效** ⭐ |
>
> 一句话：**"PHA1 = 抵抗失盐 → 低钠高钾低血压（婴儿，补盐）；PHA2/Gordon = 钠吸太多 → 高钾高血压低肾素（噻嗪特效）。"** Gordon 正好是 **Gitelman 的镜像**（Gitelman = NCC 失活 → 低钾低血压代碱；Gordon = NCC 过强 → 高钾高血压代酸）。

### 2.4 TLS（肿瘤溶解综合征）

| 指纹 | **K↑ + Phos↑ + UA↑ + Ca↓**（4 联）+ AKI |
|---|---|
| **机制核心** | 大量细胞死亡释放内容物（K / Phos / 核酸 → UA）+ Phos 结合 Ca |
| **触发** | 化疗后 24-72h，高级别肿瘤（Burkitt / 白血病 / SCLC）|

### 2.5.1 Urinary Casts 指纹族系 ⭐⭐⭐（任何 AKI 题先扫）

> [!info] 由 [[mistakes/uworld-mistakes#^Q4007]] AIN by allopurinol + [[mistakes/uworld-mistakes#^Q4004]] Analgesic nephropathy CTIN 触发整合 — 两道均因没扫到 UA WBC casts 选错（一个反射 DIL，一个反射 nephrolithiasis）。

| Casts 类型 | 锁定家族 | 典型病 | 关键鉴别 |
|---|---|---|---|
| **RBC casts**（红细胞管型 + dysmorphic RBC）| **Glomerular** ⭐ | GN（IgA / postinfectious / RPGN / Lupus nephritis / Anti-GBM）| 必伴**中重度蛋白尿** |
| **WBC casts**（白细胞管型）| **间质性家族** ⭐ | **AIN / CTIN / pyelonephritis** | 伴**外周嗜酸**（AIN）/ 急慢性区分（AIN 急 / CTIN 慢 + 双小肾） |
| **Muddy brown casts**（颗粒褐色 / "renal tubular epithelial" casts）| **ATN** | 缺血 / 药毒（aminoglycoside / contrast / cisplatin）/ 横纹肌溶解（尿肌红蛋白）| FENa > 2% |
| **Waxy casts**（蜡样管型）| **慢性肾衰** | 终末期 CKD | 配双侧小肾 |
| **Fatty casts + oval fat bodies / Maltese cross**（脂肪管型）| **肾病综合征** | MCD / FSGS / MN / DM 肾病 / Amyloid | 必伴**重度蛋白尿 > 3.5 g/d + 低白蛋白 + edema** |
| **Granular casts**（颗粒管型）| 非特异 | ATN 早期 / 慢性肾病 / 多种 | 单独无诊断价值 |
| **Hyaline casts**（透明管型）| 非特异（生理性）| 脱水 / 运动后 / 利尿剂后 | 通常无临床意义 |
| **Broad casts**（宽管型）| 集合管扩张 | 严重 CKD / ESRD | 预后差 |

> [!success] AKI 题做题 SOP（0.5 秒先扫 casts）⭐⭐⭐
> ```
> AKI + UA → 先看 casts：
>     ├─ RBC casts → GN（→ 看蛋白尿 + 补体 + ANCA + ASO + dsDNA）
>     ├─ WBC casts → 间质性家族 → 病程急 = AIN（嗜酸 + 新药）/ 慢 = CTIN（双小肾 + 慢镇痛史）
>     │              / 反复 dysuria-fever = pyelonephritis
>     ├─ Muddy brown casts → ATN（→ 看 FENa > 2% + 触发：缺血 / 药毒 / 横纹肌）
>     ├─ Fatty casts + Maltese cross → 肾病综合征（→ 看蛋白 > 3.5 + 低白 + edema）
>     ├─ Waxy / broad casts → 慢性肾衰（终末期）
>     ├─ Hyaline casts only → 非特异（脱水 / 运动 / 利尿剂后）
>     └─ UA bland（无 casts / cells / protein）→ Pre-renal / HRS / Transplant RAS / CNI 毒
>             （后 4 个需结合 stem 病史 + 扩容反应分鉴别）
> ```

> [!warning] UA bland 5 鉴别（无 casts 不等于"无病"）
> | 病 | 区分 |
> |---|---|
> | **Pre-renal volume depletion** | 扩容**改善** + FENa < 1% |
> | **HRS** ⭐ | 肝硬化 + 扩容**无改善** + 肾正常大小 |
> | **Transplant RAS** | post-tx 1-2 yr + ACEI 后 Cr↑ + resistant HTN + Doppler 见狭窄 |
> | **CNI 急性毒性** | 血药 ↑ + 加 CYP3A4 抑制剂史 + HTN |
> | **早期 GN（轻症）** | UA 可"接近 bland"但仔细看仍有 dysmorphic RBC + 轻蛋白 |
> 详见 [[完整笔记/专题笔记/肾脏/肾脏_移植后AKI鉴别SOP]]（CNI / RAS）+ [[mistakes/uworld-mistakes#^Q4752]] HRS

> [!danger] 反 pattern：把 casts 名字读对很重要
> - **RBC cast ≠ red cell 在 cast 里（炎症 dysmorphic RBC 提示肾小球破损）**
> - **WBC cast ≠ 单纯有 WBC（必须 WBC 嵌入管型 = 间质来源；游离 WBC 可能 UTI）**
> - **Muddy brown = 肾小管上皮坏死脱屑 + 颗粒变性的 cast**，不是 hemoglobin / myoglobin 染色（虽然横纹肌溶解致 ATN 时也常见）
> - **Hyaline cast 永远无诊断意义** — 见到不要往病理想

### 2.5.2 HRS 肝肾综合征 ⭐⭐⭐（来源 [[mistakes/uworld-mistakes#^Q4752]]）

| 指纹 | **肝硬化 + AKI + bland UA + 尿 Na < 10 + albumin 扩容无反应** ★★★ |
|---|---|
| **机制核心** | Portal HTN → splanchnic vasodilation → **EABV ↓** → RAAS/SNS/ADH ↑ → renal vasoconstriction（**functional**，不是结构损伤）|
| **关键鉴别** | HRS = "假 prerenal"：UA 完全像 prerenal（bland + 低 FENa + 尿 Na 低），唯一区分 = **NS + albumin 1 g/kg × 2 天扩容是否有反应** ★（HRS 不应；prerenal 应）|
| **诊断 6 条** | 肝硬化 + 腹水 / AKI (Cr↑≥0.3 或翻倍) / 扩容无反应 / 排休克 / 排肾毒药 / 无肾实质病 |
| **分型** | **HRS-AKI (旧 Type 1)** 急性 Cr 翻倍 / **HRS-NAKI (旧 Type 2)** 慢性 + 顽固腹水 |
| **诱因（必查）** | **SBP（最常）** / GI 出血 / 大量放腹水不补 albumin / 过度利尿 |
| **治疗 1st** | **Terlipressin + Albumin** ★（V1 收 splanchnic → 血赶回循环 → 肾灌注恢复）|
| **治疗 alt** | Midodrine + Octreotide + Albumin / Norepinephrine + Albumin (ICU) |
| **桥接** | **TIPS**（HRS-NAKI 更适合 / HRS-AKI 二线 / HE 禁忌中招率高）/ **RRT**（不治 HRS，是 holding pattern）|
| **根治** ⭐ | **Liver Transplant** ★★★ |
| **SBP→HRS 预防** | Cefotaxime + **IV Albumin** (Day1 1.5 g/kg + Day3 1 g/kg)，触发指标 **Cr>1 / BUN>30 / Bili>4** |
| **反直觉三连** | ① 全身水超载但 EABV↓（水都在腹腔池里）② 肾本身好（移植给别人能用）③ **利尿剂禁用**（与心衰相反！）|
| **详见** | [[完整笔记/专题笔记/GI/GI_HRS肝肾综合征]] / [[完整笔记/专题笔记/GI/GI_门脉高压治疗]] / [[mistakes/uworld-mistakes#^Q4752]] |

> [!danger] HRS vs Prerenal vs ATN 三连鉴别（看完这一表就够）
>
> | | **HRS** | **Prerenal** | **ATN** |
> |---|---|---|---|
> | 尿 Na | < 10 | < 20 | > 40 |
> | FENa | < 1% | < 1% | > 2% |
> | UA | bland | bland | **muddy brown casts** |
> | Albumin 扩容反应 | **无 ✗** | **有 ✓** | 无 |
>
> "扩容反应"就是 HRS 和 prerenal 唯一区分键 — 题干读漏 = 必错

### 2.6 Hyperkalemia ECG 五阶段指纹（Q19929 触发）

![[{0D71531D-614E-4DF9-BDA6-06BFD8580D72}.png]]
*↑ UWorld 官方 4 阶段进展示意图*

![[{F5468E78-C4BB-4C45-9B25-78E2C5F9684A}.png]]
*↑ 真实 12 导联 sine wave 例图（来自 [[mistakes/uworld-mistakes#^Q19929]]）*

| 指纹 | **ECG 进展 5 阶段**：Peaked T → P 变平/消失 → QRS 增宽 → **Sine wave** → Asystole/VF |
|---|---|
| **K⁺ 阈值** | 5.5-6.5 peaked T / 6.5-7.5 P 平 / 7.5-8.5 QRS 宽 / **>8.5 sine wave** / >9-10 心搏停 |
| **机制核心** | K⁺↑ → 心肌细胞静息电位上移 → 钠通道失活 → 复极加速（T 尖）→ 传导减慢（P 消失、QRS 宽）→ 整个 PQRST 融合 = sine wave |
| **加分指纹** | "Brady + Hypotension + P 波消失" = 高钾（**vs** Brady + P 在 = AV block）；休克 **HR 不代偿性增快**（伪正常 HR）是隐藏红旗 |
| **诊断金标准** | ECG 已诊断 → **不等血钾结果**（等化验 = 死亡）|
| **治疗第一步** | **Calcium gluconate** 1-2 g IV 慢推（1-3 min 起效，稳膜不降钾）→ 然后 Insulin+Glucose / β2 agonist（shift）→ Furosemide / Patiromer / 透析（remove）|
| **易混陷阱** | Sine wave ≠ Torsades：sine 用 **Calcium**（高钾），TdP 用 **Magnesium**（长 QT 多形性 VT）|
| **C-BIG-K-D** | **C**alcium → **B**icarb/**I**nsulin+**G**lucose → **K**ayexalate（现 Patiromer/SZC）→ **D**ialysis |
| **详见** | [[草稿/2026-05-17_高钾血症完整体系_v1]] / [[mistakes/uworld-mistakes#^Q19929]] / [[mistakes/uworld-mistakes#^Q19-2026-05-18]]（trigger vs permissive 元凶定位）|

> [!warning] 休克 + Brady + ECG 异常的鉴别铁律
> 看到 "HR 60 + BP 80/30 + ECG 异常" — 不要反射性给 Atropine。先扫 P 波：
> - P 波**在** + P-QRS 解离 → **AV block** → Atropine / pacing
> - P 波**消失** + 宽 QRS / sine wave → **Hyperkalemia** → **Calcium gluconate**

### 2.7 Struvite Stone（鸟粪石 / 鹿角石）⭐（来源 [[mistakes/uworld-mistakes#^Q15682]] — 做对题 2026-05-28）

| 指纹（**5 联**）| **反复 UTI 史** + **尿 pH > 8** + **Coffin lid 棺材盖晶体** + **大 staghorn 鹿角石** + **女性多发** |
|---|---|
| **机制核心** | **Urease+ 菌**水解 urea → 2 NH₃ + CO₂；NH₃ + H₂O → NH₄⁺ + OH⁻ → 尿碱化 → Mg²⁺/NH₄⁺/PO₄³⁻ 过饱和 → **MgNH₄PO₄·6H₂O 沉淀**（长得快，可数周成 staghorn）|
| **Urease+ 菌口诀** | **"PPP-S-K-U"** = **P**roteus / **P**rovidencia / **P**seudomonas（部分）/ **S**taph **sapro**phyticus / **K**lebsiella / **U**reaplasma ⚠️ **E. coli 不产 urease**（UTI #1 元凶却**不**致 struvite，经典反 pattern）|
| **加分指纹** | UA：LE⁺ + nitrite⁺ + WBC↑ + RBC↑ / 影像 = radiopaque + 充满肾盂肾盏的鹿角形 / 临床多 **反复发热-脓尿 > 典型肾绞痛**（大石不进 ureter）|
| **诊断金标准** | 影像（CT / US）+ UA pH > 8 + 尿培养 urease+ 菌 |
| **治疗第一步** | **完整取石（PCNL 经皮肾镜碎石）** + 长程抗生素 + 选择性 urease inhibitor (**acetohydroxamic acid**) 防复发；⚠️ **抗生素单用不够** — 菌藏 **stone matrix** 内 |
| **详见** | [[完整笔记/专题笔记/肾脏/肾脏_肾结石5型对照]]（5 型完整对照 + 钙石防复发 + 急性处理）/ [[mistakes/uworld-mistakes#^Q15682]] / [[mistakes/uworld-mistakes#^Q3895]] 钙草酸防复发 / [[完整笔记/Peixuan分科笔记/肾脏]] |

> [!tip] 5 大肾结石尿 pH 方向反差速记 ⭐
> | 结石 | 尿 pH | 治疗方向（pH 干预） |
> |---|---|---|
> | **Struvite / Ca phosphate** | **碱性（pH > 7）** | 酸化（治原发病为主；struvite 取石）|
> | **Uric acid / Cystine** | **酸性（pH < 5.5-6）** | **碱化尿**（potassium citrate）|
> | **Ca oxalate** | pH 不敏感 | thiazide + citrate |
>
> **反 pattern**：低钙饮食 / 脂肪吸收不良（IBD / 减重手术）→ Ca 被脂肪绑 → 游离 oxalate↑ → **Ca oxalate 反而↑**（与"补钙防结石"直觉相反）

> [!warning] "大 staghorn 反复 UTI > 肾绞痛" 反 pattern
> Struvite stone 长得大（充满肾盂肾盏）但**不进 ureter** → 表现为**反复 UTI 发热-脓尿**，**不**像 Ca oxalate 小石那样典型阵发肾绞痛向腹股沟放射。看到"反复 UTI + 不规则肾盂大石" → 锁 struvite 不锁 oxalate。

| **治疗** | **放血**（首选）/ deferoxamine（不能放血时）|

### 4.3 α1-抗胰蛋白酶缺乏

| 指纹 | **α1-AT 水平↓** + 全小叶肺气肿（下叶为主，年轻 / 不吸烟）+ 肝硬化 / 肝癌 |
|---|---|
| **机制** | Z 突变 α1-AT 在肝内聚积 → 肝损 + 全身缺 α1-AT → 弹性蛋白酶失抑 → 肺气肿 |

### 4.4 急性 / 慢性病毒性肝炎血清学（HBV）

|---|---|---|---|---|---|
| 急性感染早期 | + | - | IgM | + | 高传染性 |
| 慢性感染 | + | - | IgG | ± | 持续 > 6 月 |

## Part 5. 中毒指纹

### 5.1 醋酸氨基酚（acetaminophen）中毒

| 指纹 | 早期：N/V → 24-72h：**AST/ALT↑↑↑**（>1000）+ **PT/INR↑** + **lactate↑** + 急性肝衰 |
|---|---|
| **机制** | NAPQI 自由基耗尽 glutathione → 肝细胞坏死 |
| **诊断** | Rumack-Matthew nomogram（4-24h 血药浓度）|
| **治疗** | **N-acetylcysteine（NAC）** — 8 小时内最佳，但 24+ 小时仍给 |

### 5.2 渗透压间隙 + AG 中毒

| | 渗透压间隙 | AG | 特征 |
|---|---|---|---|
| **甲醇** | **↑** | **↑** | 视力丧失 / 视网膜水肿 |
| **乙二醇** | **↑** | **↑** | 草酸钙结晶尿（envelope-shaped）|
| **乙醇** | ↑ | 正常 | 醉酒史 |
| **异丙醇** | ↑ | 正常 | 酮症但无酸中毒 |

### 5.3 阿司匹林中毒

| 指纹 | **早期呼碱**（直接刺激呼吸中枢）→ **后期混合呼碱 + AG 代酸**（uncoupling 氧化磷酸化）+ **耳鸣** + 高热 + 谵妄 |
|---|---|
| **治疗** | 碱化尿液（NaHCO3，目标尿 pH 7.5-8）+ K 维持 + 严重透析 |

### 5.4 CO 中毒 ⭐⭐

| 指纹（**反直觉三联**） | **SpO₂ 假性 > 95%**（脉氧仪分不清 CO-Hb 与 Oxy-Hb）+ **樱桃红皮肤 / 唇**（NOT 紫绀！）+ **共室人同病** + 头痛 + 神志改变 + 乳酸↑ |
|---|---|
| **机制核心** | CO 与 Hb 亲和力 ≈ O₂ 的 **240×**；CO-Hb 占位 + **氧解离曲线左移** ⭐（剩下的 Hb 也不释放 O₂）→ 组织缺氧但血液鲜红 |
| **加分指纹** | **Saturation gap**（SpO₂ 高 vs co-oximetry 实测 SaO₂ ↓↓）；ABG PaO₂ 正常（溶解氧 OK）|
| **诊断金标准** | **Co-oximetry 测 CO-Hb%**（普通 SpO₂ 假高，会漏诊）|
| **病因 2 大场景** | ① **冬季燃气取暖 / 故障 furnace** + 全家头痛恶心 ② **火灾烟雾吸入** + 面部煤灰 + 焦鼻毛 + 神志改变（→ **CO + Cyanide 双中毒**，见 5.7 + 高频陷阱）|
| **治疗第一步** | **100% O₂ non-rebreather 面罩**（CO-Hb 半衰期：空气 4-6h → 100% O₂ 60-90 min → HBO 20-30 min）|
| **HBO 适应证** | CO-Hb > 25% / 意识改变 / 神经体征 / **妊娠**（任何 CO-Hb 水平）/ 心肌缺血 |
| **CO vs MetHb 反直觉点** ⭐⭐ | CO 患者**不紫绀**反樱桃红（外表骗人，"看起来还行"实际缺氧死亡）；MetHb 患者紫绀 + SpO₂ 卡 85%（看着糟实际还活着）|
| **详见** | **5.7 异常 Hb 双毒终极对照**；火灾双中毒陷阱 → [[完整笔记/专题笔记/USMLE/USMLE_高频陷阱]] "Carbon monoxide vs MetHb vs Cyanide" |

> [!danger] CO 中毒最大陷阱：NOT cyanotic
> CO 中毒患者通常**不紫绀**，反而**樱桃红**（CO-Hb 比 Oxy-Hb 还红）→ 第一印象"血色好得很" → 实际组织在缺氧死亡。**不要被外表骗了！**

> [!success] Stem 触发模式
> ① **冬季取暖**：gas heater / faulty furnace + 全家头痛恶心 + cherry red + SpO₂ 98%（假高）
> ② **火灾后**：smoke inhalation + 面部煤灰 + 焦鼻毛 + 神志改变 → **同时考虑 CO + Cyanide**

### 5.5 慢性酒精性多系统指纹 ⭐（来源 [[mistakes/uworld-mistakes#^Q2692]] AIC + [[mistakes/uworld-mistakes#^Q12432]] AUD vs AD 鉴别 — 做对题 2026-05-19）

| 指纹 | **MCV > 100**（无 B12/叶酸缺乏的大细胞贫血）+ **AST/ALT > 2:1**（"AST is higher"，"toAST = 致敬"）+ **三系轻度受累**（贫血 + Plt↓ ± WBC↓）|
|---|---|
| **机制** | 乙醛骨髓毒 → macrocytosis + 三系；直接肝毒 → AST 偏倚（线粒体 AST 受损）；直接心肌毒 → eccentric hypertrophy → DCM；硫胺素缺乏 → Wernicke / 心衰加剧 |
| **多系统连锁（一族指纹）** | 心：**AIC**（DCM，EF↓）/ 肝：脂肪肝 → AH → 肝硬化 / 血液：MCV↑ + 三系 / 神经：**Wernicke**（眼肌麻痹 + 共济失调 + 意识改变，见 6.6）/ 营养：**Thiamine 缺乏** / 胰：急/慢性胰腺炎 / GI：**Mallory-Weiss tear** / 内分泌：男性乳房发育 + 性腺萎缩 / **精神：失眠 / 抗焦虑 / 抑郁样症状**（见下）|
| **诊断 stem 触发** | "long-standing heavy alcohol use" + 任一系统失代偿（HFrEF / 肝硬化 / 大细胞贫血 / 急性神经症状）→ 一律扫 MCV + AST/ALT + 三系作交叉验证 |
| **治疗反直觉** ⭐ | AIC：**完全戒酒 > 任何 HFrEF 标准药**（病因可去除 → 优先病因治疗，**不是**条件反射 ACEi/ARNI/BB/MRA）；激素**无效**（≠ 酒精性肝炎用激素）|
| **可逆性** | 戒酒后 6-12 月 LV 功能可显著恢复（部分 EF 回到正常）；继续饮酒 → 不可逆纤维化 |

> [!success] 一句话锁定（USMLE）
> 看到 **HFrEF + DCM + 冠脉干净 + MCV↑ + AST/ALT > 2:1** → 立刻想 **AIC**，治疗答案是**戒酒**而非 ACEi。

> [!warning] 红鲱鱼陷阱
> - 长 pack-year 吸烟史是干扰项，冠脉造影干净就排除 ischemic CMP
> - "EF 25%" 不要条件反射 HFrEF 标准药 — **先问"病因能不能拿掉"**
> - 问题措辞 "**reverse** LV dysfunction" = 暗示选病因治疗

#### 5.5.x AUD vs Adjustment Disorder 鉴别（Q12432 触发，2026-05-19）⭐

> [!info] 临床场景
> 中年女性 + 应激事件（如离婚）后 1 年 + 失眠 + 焦虑 + 睡前饮酒助眠 + 工作迟到 + tremor + 实验室异常
>
> **关键鉴别**：是 Adjustment Disorder（[Q3382](mistakes/uworld-mistakes.md#^Q3382)）还是 Alcohol Use Disorder？

**Alcohol → 失眠的机制**（高频考点）：
```
睡前饮酒（自我治疗）
        ↓
快速入睡（GABA 增强）
        ↓
血酒精浓度下降（数小时）
        ↓
CNS hyperarousal 反弹（GABA 突然撤离）
        ↓
3 AM 觉醒 + 不能再入睡 ⭐ 经典 pattern
```

**NIAAA 不健康饮酒阈值**（必背）：

| 群体 | 单日不健康 | 单周不健康 |
|---|---|---|
| **女性 + 所有 ≥65 岁** | **> 3 drinks/day** | **> 7 drinks/week** |
| **男性 < 65 岁** | > 4 drinks/day | > 14 drinks/week |

> 1 drink = 12 oz 啤酒 / 5 oz 红酒 / 1.5 oz 烈酒

**AUD vs AD 鉴别铁律**：

| 维度 | **AUD** | **AD** |
|---|---|---|
| **时间窗** | 长期模式（months-years）| **应激源 ≤ 3 月起病**；缓解 ≤ 6 月 |
| **应激源 > 6 月** | 仍可诊断 | ❌ **排除 AD**（超时间窗）|
| **关键实验室** | **MCV ↑ + AST/ALT ≥ 2:1**（酒精指纹）| 无特异异常 |
| **躯体表现** | Tremor / sleep disturbance（3 AM）/ macrocytosis | 无 |
| **酒精量** | ≥ NIAAA 阈值 | 不达标 |
| **治疗** | 戒酒咨询 + 可能 naltrexone / acamprosate / disulfiram | **Psychotherapy** |

> [!success] Q12432 解题路径
> 离婚 1 年前（>6 月超 AD 时间窗）+ MCV 106 + AST/ALT = 85/42 = 2:1 + tremor + 红酒 2-3 杯/晚（女性 14-21 drinks/week，远超 NIAAA 7/wk）
> → **AUD**，不是 AD。

> [!warning] 离婚 stressor 双面性（USMLE 经典陷阱）
> 同样的"离婚后失眠 + 抑郁样症状"stem，**时间窗 + 实验室指纹决定不同诊断**：
> - 离婚 ≤ 3 月 + SIGECAPS < 5 + 实验室正常 → **AD**（psychotherapy 一线，见 [Q3382](mistakes/uworld-mistakes.md#^Q3382)）
> - 离婚 ≤ 3 月 + SIGECAPS ≥ 5 + 实验室正常 → **MDD**（SSRI/SNRI + psychotherapy）
> - 离婚 > 6 月 + MCV↑ + AST/ALT≥2:1 + tremor → **AUD**（戒酒咨询 ± naltrexone）
> - 离婚后 1-12 月内丧亲反应 → **Uncomplicated grief**（支持性心理治疗）
> - 离婚后 ≥ 2 年慢性抑郁不达 MDD → **Dysthymia / PDD**

### 5.6 Methemoglobinemia (MetHb) ⭐⭐

| 指纹（**三联**） | **紫绀** + **SpO₂ 顽固卡 ~85%（吸 100% O₂ 不改善）** + **巧克力棕血**（Chocolate brown） |
|---|---|
| **机制核心** | Hb 中 **Fe²⁺ → Fe³⁺**（失去携氧能力）；MetHb 让脉氧仪"两眼失明" — 660 nm 与 940 nm 吸收均被干扰 → 比值收敛到 1:1 → **SpO₂ 数学上卡在 ~85%** |
| **加分指纹** | **Saturation gap** = SpO₂（脉氧）− SaO₂（co-oximetry 实测）> 5%；ABG **PaO₂ 正常**（溶解氧 OK，肺没问题）但临床极差 |
| **诊断金标准** | **Co-oximetry** 实测 MetHb%（普通 ABG 的 SaO₂ 是从 PaO₂ "算"出来的，会假高 → 不能用） |
| **病因 4 大类** | ① **婴儿井水**（高 NO₃⁻ → 肠菌还原 NO₂⁻ → 氧化 Hb；月龄小 + 胃酸低更易中毒）② **Benzocaine**（TEE / 内镜 / 支气管镜咽喉局麻喷雾）③ **Dapsone**（HIV PCP 预防 / 麻风 / 疱疹样皮炎）④ Nitrates / Nitrites / 苯胺染料 |
| **症状 vs SpO₂** | < 15% 无症状 \| 15-30% 紫绀 + 头痛 + 心动过速 \| 30-50% 呼吸困难 + 晕厥 \| > 50% 抽搐 + 昏迷 + 酸中毒 \| > 70% 死亡 \| **但 SpO₂ 全程卡 ~85%（数字骗人）** |
| **治疗第一步** | **Methylene Blue IV**（1-2 mg/kg）→ 作 NADPH-MetHb reductase 辅因子 → Fe³⁺ → Fe²⁺ |
| **G6PD 双禁忌** ⭐⭐ | ① **没效**：G6PD ↓ → NADPH 不足 → MB 不工作 ② **有害**：MB 本身氧化剂 → 诱发溶血 → 改 **Vitamin C**（Ascorbic acid）/ exchange transfusion / HBO |
| **详见** | 紫绀鉴别 → [[完整笔记/专题笔记/USMLE/USMLE_鉴别诊断速查]] "Cyanosis 紫绀"；Methylene Blue 陷阱 → [[完整笔记/专题笔记/USMLE/USMLE_高频陷阱]] "Methylene Blue 在 MetHb：G6PD 双禁忌" |

> [!danger] 三种紫绀的"特征数字"对比（与 5.4 CO 中毒形成"异常 Hb 双陷阱"）
>
> | 类型 | SpO₂ 表现 | 吸 O₂ 反应 | 血液颜色 | 鉴别金标准 |
> |---|---|---|---|---|
> | **心源性紫绀**（R→L shunt） | **真性↓**（如 75%） | 不改善 | 暗红 | Echo 见分流 |
> | **肺源性紫绀** | **真性↓** | **改善** | 暗红 | 肺部影像 |
> | **MetHb** | **卡 ~85%**（不动） | **不改善** | **巧克力棕** ⭐ | **Co-oximetry MetHb%** |
> | **CO 中毒**（CarboxyHb） | **假性 > 95%**（致命假象） | 无意义 | **樱桃红** | **Co-oximetry CO-Hb%** |
>
> MetHb SpO₂ 看似还行（85%）实际很糟；CO SpO₂ 完美（>95%）可能致死。两个都靠 **co-oximetry** 识破。

> [!success] Stem 三种触发模式
> ① **婴儿井水冲奶**："4-month-old, well water formula" + perioral cyanosis + 巧克力棕血
> ② **Benzocaine 医源性**：TEE / bronchoscopy / endoscopy 后突发紫绀 + SpO₂ 85%
> ③ **Dapsone 慢性**：HIV PCP 预防 / 麻风 / 疱疹样皮炎患者出现紫绀

> [!tip] Memory Hook
> "**85% 卡死 + 巧克力棕 + 蓝色解药**"：SpO₂ 卡 85% 不动 → MetHb → 推 Methylene Blue → 但先问 G6PD（缺者改 Vit C）。

### 5.7 异常 Hb 双毒/三毒终极对照（MetHb vs CO vs Cyanide）⭐⭐⭐

> 用户标识为"总记不清"——建议手动加 `#难记`，考前默写本节终极对照表 + Memory Hook 三句话。

#### 终极对照表

| 项目 | **MetHb** | **CO（Carboxy-Hb）** | **Cyanide** |
|---|---|---|---|
| **Hb 改变** | Fe²⁺ → **Fe³⁺**（化学变性） | Fe²⁺ 不变，**位点被占** | Hb 不变 |
| **能否结合 O₂** | **完全不能** | 能（但位点被 CO 占） | 能 |
| **对其他 Hb 影响** | 无 | **氧解离曲线左移** ⭐（剩下的 Hb 也不释放） | 无 |
| **细胞层面** | 血液运不上 O₂ | 血液运不上 + 不释放 | **ETC complex IV 阻断**（有氧用不上） |
| **血液颜色** | **巧克力棕** | **樱桃红** ⭐ | 静脉血鲜红（动静脉色差消失） |
| **皮肤颜色** | **紫绀**（蓝灰） | **樱桃红 / 粉红**（**NOT 紫绀！**） ⭐⭐ | 樱桃红 / 紫绀（晚期） |
| **SpO₂ 表现** | **卡 ~85%** | **假性 > 95%** ⭐ | 正常 |
| **PaO₂** | 正常 | 正常（溶解氧 OK） | 正常 |
| **真实 SaO₂** | ↓ | ↓↓ | 正常（但组织用不上） |
| **诊断金标准** | Co-oximetry MetHb% | Co-oximetry CO-Hb% | 乳酸↑↑ + 临床（杏仁味呼气） |
| **一线治疗** | **Methylene Blue IV** | **100% O₂ non-rebreather** | **Hydroxocobalamin** |
| **重度升级** | Exchange transfusion | **HBO**（高压氧） ⭐ | Sodium thiosulfate + nitrites |
| **关键禁忌** | **G6PD → 用 Vit C** | — | **火灾患者慎用 nitrites**（再造 MetHb 雪上加霜） |

#### 临床第一印象陷阱（MetHb vs CO）

| 特征 | MetHb 患者 | **CO 中毒患者** |
|---|---|---|
| 外表 | 紫绀（蓝灰） | **樱桃红 / 粉红** |
| SpO₂ | 卡 85% | **> 95%（假高）** |
| 第一印象 | "这人缺氧了" | **"这人看起来还行"** ⚠️ |
| 真实状态 | 缺氧 | **缺氧（更隐蔽）** |

#### 组织缺氧四大类型（USMLE 总览）

```
组织缺氧四大类型
├─ ① Hypoxemic hypoxia（低氧血症性）
│   └─ PaO₂ ↓（肺病 / 高原 / 心脏分流）
│
├─ ② Anemic hypoxia（贫血性）
│   ├─ Hb 总量 ↓（贫血）
│   ├─ MetHb（功能性失活）⭐
│   └─ CO 中毒（位点被占 + 曲线左移）⭐
│
├─ ③ Circulatory hypoxia（循环性）
│   └─ 心衰 / 休克 / 局部缺血
│
└─ ④ Histotoxic hypoxia（组织毒性）
    └─ Cyanide / H₂S → 抑制 ETC complex IV
        细胞"有氧不能用"
```

> [!info] MetHb 和 CO 都属于 anemic hypoxia
> 不是肺的问题（PaO₂ 正常），不是循环的问题，是 **Hb 本身没法正常运 O₂**。Cyanide 不一样 — 它属于 **histotoxic**（细胞用不上）。

> [!warning] 火灾患者治疗陷阱（必背）
> 火灾 = **CO + Cyanide 双中毒**
> - Cyanide 经典疗法的 **Nitrites**（亚硝酸盐）通过**诱导 MetHb 来螯合 CN⁻** 起效
> - 但 CO 中毒患者本身就缺乏有效 Hb，再造 MetHb = **雪上加霜**（三重缺氧：CO + MetHb + 残余 Hb）
> - **火灾患者首选 Hydroxocobalamin**（不诱导 MetHb）+ 100% O₂；**避免单用 Sodium nitrite**

> [!success] Memory Hook 三句话（考前默写）
> - **MetHb = "装不上"**（Fe³⁺ 装不上 O₂）→ 巧克力棕 + 紫绀 + SpO₂ 卡 85% + Methylene Blue
> - **CO = "装上了不放 + 邻居一起摆烂"**（亲和力 240× + 曲线左移）→ 樱桃红 + 不紫绀 + SpO₂ 假高 + 100% O₂ / HBO
> - **Cyanide = "有氧用不上"**（ETC complex IV 阻断）→ 杏仁味呼气 + 静脉血鲜红 + Hydroxocobalamin

> [!tip] 画面感（机制记忆挂钩）
> - **MetHb**：生锈的螺丝（铁被氧化了，拧不上 O₂ 螺母）
> - **CO**：赖着不走的房客 + 让邻居一起罢工
> - **Cyanide**：断电的工厂（complex IV 罢工）

### 5.8 "心动过缓 + 低血压"三联中毒（β-blocker / CCB / Digoxin）⭐（来源 [[mistakes/uworld-mistakes#^Q2663]] β-blocker toxicity → Glucagon）

| 共有指纹 | **HR↓↓ + BP↓↓ + IVF/atropine 无效**（"自杀 + 心动过缓 + 低血压"三联触发）|
|---|---|
| **β-blocker 特异指纹** | + **Wheezing / bronchospasm**（non-selective）+ **低血糖** + CNS 抑制（脂溶性 propranolol → confusion / seizures）→ 药源：CAD / HTN / AFib / HF / 偏头痛 / 焦虑 |
| **CCB（verapamil/diltiazem）特异指纹** | + **高血糖**（抑制 β 细胞胰岛素释放）+ 反射性 → 药源：HTN / AFib / SVT |
| **Digoxin 特异指纹** | + **GI（恶心呕吐）+ 视觉异常（黄/绿视）+ 高钾** + "Scooped" ST + 双向性 VT → 药源：AFib / HF |
| **机制核心** | β-blocker / CCB 切断心肌 cAMP 通路 → 收缩力↓ + 心率↓；**Glucagon 绕过被阻断的 β-receptor**，从自己的 G-protein 受体直接激活 Gs/AC → ↑cAMP（殊途同归）|
| **诊断金标准** | 病史 + 用药史 → **药源推断**（CAD/HTN 老人 = BB/CCB 嫌疑；AFib/HF = dig 嫌疑）+ 特异线索（wheezing / 血糖方向 / GI+视觉+K）|
| **治疗第一步** | **β-blocker → Glucagon 3-10 mg IV bolus + 1-5 mg/h 输注**；**CCB → Calcium + insulin/dextrose（HIE）**；**Digoxin → Digoxin-Fab** |
| **关键陷阱** | "**Atropine fail → think Glucagon**"（atropine 走 vagal 通路绕不过 β-receptor 阻断）；Dobutamine 在 BB 中毒无效（受体级别被堵）；Pacing 因 cAMP↓ 心肌反应差（capture failure）|
| **详见** | [[mistakes/uworld-mistakes#^Q2663]] 完整三联鉴别表 + 高频中毒-解毒剂 17 项速查 + β-blocker 选择性差异（cardioselective / non-selective / lipid-soluble）|

> [!success] Stem 触发模式
> 自杀企图（suicide note）+ CAD/HTN 病史 + bradycardia + hypotension + **wheezing** + IVF/atropine 无效 → **β-blocker → Glucagon**
> 把"wheezing"或"血糖方向"当作三联中毒的**特异指纹**，不要只看共同症状（brady + hypotension）就乱选 digoxin。

> [!tip] Memory Hook（"3 个 B"）
> - β-**B**locker → **B**ronchospasm + low **b**lood sugar → Gluca**g**on
> - CC**B** → **B**ig blood sugar（高血糖）→ Cal**c**ium
> - Digoxin → **B**owel（GI）+ **B**lurry vision + high K → Fa**b**

## Part 6. 急诊 / 综合征指纹

### 6.1 急性胰腺炎

| 指纹 | **Lipase↑↑**（更特异，> 3× ULN）+ Amylase↑ + 上腹痛放射背 + N/V |
|---|---|
| **加分指纹** | 严重时：**Ca↓**（钙皂化）+ Grey Turner / Cullen（出血型）+ ARDS |
| **诊断** | 3 选 2：典型痛 + Lipase > 3× ULN + 影像 |
| **病因 (I GET SMASHED)** | 胆石 / 酒精（前两位）/ ERCP / 创伤 / 高 TG / 高 Ca / 病毒（mumps）|

### 6.2 Carcinoid 综合征

| 指纹 | **24h 尿 5-HIAA↑**（金标准）+ flushing + 腹泻 + wheezing + 右心瓣膜病（**TR/PR**）|
|---|---|
| **机制** | 类癌肿瘤分泌 serotonin → 经肝代谢，到肺前 → 仅当有**肝转移**时才进体循环 |
| **治疗** | **Octreotide**（缓解症状）+ 切除原发 / 肝转移 |
| **升降对照** | **尿 5-HIAA↑（外周·类癌）vs CSF 5-HIAA↓（中枢·自杀，见 10.1）** —— 方向相反，最易搞反 |

### 6.3 Zollinger-Ellison（ZES, gastrinoma）

| 指纹 | **Gastrin↑↑** (>1000) + 顽固消化性溃疡（多发 / 远端 / 巨大）+ 慢性腹泻 + 钙↑（MEN1）|
|---|---|
| **诊断** | secretin 刺激试验（正常时 gastrin 下降，ZES 时**反常上升**）|
| **MEN1 关联（3P）** | **P**ituitary 腺瘤（PRL / GH / ACTH）+ **P**arathyroid 增生（>90%，高钙 / 肾结石 / 骨密度↓）+ **P**ancreatic 神经内分泌瘤（**Gastrinoma → ZES** / **Insulinoma** → 低血糖 / VIPoma → 分泌性腹泻 + 低钾 / Glucagonoma → 体重↓ + 坏死性游走红斑 + 高糖）|
| **MEN1 stem 触发** | "复发性消化性溃疡 + 肾结石 + 头痛 / 视野缺损" → 三 P 全占 → MEN1 |

### 6.4 Rhabdomyolysis

| 指纹 | **CK↑↑↑** (>5× ULN) + **尿肌红蛋白**（dipstick 阳性 + 镜下无 RBC）+ K↑ + Phos↑ + Ca↓ + AKI |
|---|---|
| **病因** | 挤压伤 / statin / 长时间地上躺 / 强训 / 抽搐 / 中暑 |
| **治疗** | **大量补液**（首选，2-3 L/h IV NS）+ 监测 K + 必要时透析 |

### 6.5 Refeeding 综合征

| 指纹 | 长期饥饿 → 补食 → **Phos↓↓ + K↓ + Mg↓** + thiamine 缺乏 |
|---|---|
| **机制核心** | 长期饥饿 → 胰岛素低 + 营养库枯竭。再喂食（尤其碳水）→ **胰岛素↑↑** → 驱动 **Phos / K / Mg 进入细胞内合成 ATP / 蛋白** → 血浆三者暴跌 |
| **致命点** | **低磷血症**最危险 — Phos 是 ATP / 2,3-DPG 关键组分；低磷 → 心肌细胞 ATP 不足 → **心衰 + 心律失常 + 死亡**；同时呼吸肌无力（脱机失败） |
| **高危人群** | AN / 大量减重手术后 / 长期酒精 / TPN 起始 / 化疗后恢复 / 战俘 / 慢性消耗 |
| **预防** | 缓慢复食（前 24-48 h 限热量到 < 50% 目标）+ **先给 thiamine**（避免 Wernicke）+ 同步补 Phos / K / Mg + 监测每日电解质 × 1 周 |

### 6.6 Wernicke / Korsakoff

| 指纹（Wernicke 三联）| **共济失调 + 眼肌麻痹 + 意识 / 记忆障碍**（不一定全有，"AOC"）|
|---|---|
| **机制** | Thiamine（B1）缺乏 — 慢性酗酒 / 营养不良 / 透析 / 减肥手术 / 妊娠剧吐 |
| **关键陷阱** | 给葡萄糖**前**必须先给 thiamine（葡萄糖会耗尽残余 B1，加重 Wernicke）|

### 6.7 Hypovolemia / 容量丢失（脱水四联）⭐

| 指纹（**脱水四联**）| **Hct↑** + **BUN/Cr > 20**（prerenal）+ **干燥黏膜** + **低 K** |
|---|---|
| **机制核心** | 血浆容量↓ → 浓缩（Hct↑）+ 肾灌注↓（prerenal AKI，BUN/Cr 比> 20）；醛固酮代偿排 K |
| **加分指纹** | 心动过速 + **Orthostatic Hypotension HR↑≥30**（容量型）+ 站立时 SBP↓≥20 |
| **典型患者** | **运动员 weight-cutting**（如本题）/ 呕吐 / 腹泻 / 出汗 / 利尿剂 / 失血 / 烧伤 |
| **vs 神经源 OH** | 容量型：HR ↑≥30 + 实验室异常；神经源型：HR 几乎不动 + 实验室正常 + DM/Parkinson 背景 |
| **治疗** | **补液 + 补 K**（容量型立即改善，与神经源型 midodrine 不同）|
| **详见** | [[mistakes/uworld-mistakes#^Q21638]] Volume-mediated Orthostatic Syncope（摔跤运动员）|

> [!tip] "脱水四联"与"PAI 五联"对比
> - **PAI 五联**：Na↓ / **K↑** / HCO3↓ / Glu↓ / Eos↑（cortisol + aldo 双缺）
> - **脱水四联**：Hct↑ / BUN/Cr↑ / 干燥 / **K↓**（容量丢失 + 醛固酮代偿）
> - 关键反差：**PAI 高 K（aldo↓ 不能排）vs 脱水低 K（aldo↑ 代偿排 K）**

## Part 8. 比值 / 间隙 / 计算指纹

| 计算 | 用途 |
|---|---|
| **AG** = Na − (Cl + HCO3) | 代酸鉴别（高 AG: MUDPILES / 正 AG: HARDASS）|
| **渗透压间隙** = 测 − 计算 | 中毒鉴别（甲醇 / 乙二醇）|
| **FENa** = (尿 Na × 血 Cr) / (尿 Cr × 血 Na) × 100 | Pre-renal vs ATN |
| **TTKG** = (尿 K / 血 K) / (尿 Osm / 血 Osm) | 高钾鉴别（>7 提示肾正常排 K）|
| **A-a gradient** = PAO2 − PaO2 | 低氧机制（>15 提示分流 / V-Q 失配）|
| **SAAG** = 血 albumin − 腹水 albumin | 腹水鉴别（>1.1 = 门高压；<1.1 = 渗出 / 恶性 / TB）|
| **Reticulocyte index** = Retic% × (Hct / 45) × (1/maturation) | 区分溶血（>2）vs 生成不足（<2）|
| **Light's criteria** | 渗出 vs 漏出胸水 |

## Part 10. 神经 / 精神科标志物指纹（CSF · EEG · PSG）

> [!info] 本 Part 定位
> CSF / EEG / PSG / 颅脑影像类"标志物指纹" —— 症状层面常很像（痴呆 / 嗜睡 / 抑郁），靠**单个标志物**直接锁病。源 Q15897（CSF 5-HIAA）+ Q15924（MDD 睡眠 PSG），均为知识点补充、未入错题本。

### 10.1 CSF 5-HIAA ↓ → 自杀 / 冲动 / 暴力 ⭐⭐⭐

| 指纹 | **CSF 5-HIAA↓** + 反复自杀未遂 / 冲动 / 攻击行为 + episodic 抑郁 |
|---|---|
| **机制** | 5-HIAA = serotonin 主要代谢产物；CSF 5-HIAA↓ = 中枢 serotonin 功能低下。Serotonin = "刹车系统"（抑制冲动）→ 刹车油不够 → 冲动失控 → 自杀 / 暴力 |
| **考法** | 题问"最可能存在哪种异常"（考生化 correlate，不是问诊断）|
| **升降对照** | vs **尿 5-HIAA↑（外周·carcinoid，见 6.2）** —— CSF 低想自杀、尿高是类癌，方向相反最易搞反 |

> [!warning] 取样部位决定你测的是哪个"池子"（BBB 把中枢 / 外周 serotonin 隔开）
> - **评估中枢神经递质 / 中枢病变 → 取 CSF**：5-HIAA（自杀）、HVA（多巴胺周转）、orexin（narcolepsy）、14-3-3（CJD）。
> - **评估外周分泌 / 肿瘤 → 取 血 / 尿**：尿 5-HIAA（carcinoid）、尿 VMA / metanephrines（pheo）。
> - 三条逻辑：① 5-HIAA 几乎不过 BBB；② 全身 ~90% serotonin 在肠道，外周值被肠道主导；③ 外周升高反指向类癌。
> - 推广：中枢多巴胺周转测 **CSF HVA**，外周儿茶酚胺测 **尿 VMA / metanephrines** —— 同一个"中枢测 CSF、外周测血尿"逻辑。

### 10.2 CSF / EEG / 影像标志物联立速查 ⭐⭐

> [!tip] 用法
> 这 5 个标志物常被放进同一道题的 5 个选项 —— 每个对应一个**独立疾病**，认 stem 关键词对号入座。

| 标志物 / 表现 | 疾病 | Stem 关键词 |
|---|---|---|
| **CSF 5-HIAA ↓** | 自杀 / 冲动 / 暴力 | 反复自杀未遂 + 抑郁 |
| **CSF orexin (hypocretin) ↓** | Narcolepsy type 1 | 日间嗜睡 + 猝倒（cataplexy）|
| **CSF 14-3-3 ↑ + periodic sharp wave complexes** | CJD（克雅病）| 快速进展性痴呆 + 肌阵挛 |
| **3-Hz spike-and-wave EEG** | Childhood absence epilepsy（儿童失神癫痫）| 儿童凝视、突然中断、过度换气诱发 |
| **Hydrocephalus ex vacuo（CT）** | 脑萎缩（Alzheimer 等）| 老年痴呆、脑室代偿性扩大 |

> [!danger] 高频陷阱
> - **14-3-3 + periodic sharp wave complexes 常一起作两个选项** = 出题人在拼 CJD；没有 CJD 表现（快速痴呆 + 肌阵挛）就把这两个一起排除。
> - **3-Hz spike-wave = absence**，别和其他 EEG buzzword 张冠李戴。
> - Hydrocephalus **ex vacuo**（脑萎缩被动扩张）≠ **NPH**（正常压力脑积水：步态障碍 / 尿失禁 / 痴呆三联）。

> [!success] Memory Hook
> **"脑脊液低想自杀，尿里高是类癌瘤。"** —— CSF 5-HIAA↓ → suicide；尿 5-HIAA↑ → carcinoid。
> 取样口诀：**想看"脑子里"测 CSF，外周血 / 尿测的是"肠子里"。**

### 10.3 睡眠障碍 PSG 标志物鉴别 ⭐⭐（MDD / OSA / RBD / PLMD）

| 指纹 | **MDD 的 PSG「三降一升」**：↓ REM latency（REM 潜伏期缩短，标志性 biomarker）+ ↓ slow-wave sleep（深睡少）+ ↓ 睡眠连续性（易醒 / 早醒）+ ↑ REM 总量 & 密度 |
|---|---|
| **一句话** | 抑郁的睡眠 = "REM 来得早、来得多；深睡少、易早醒"；睡眠紊乱是抑郁**核心症状**，治疗后多恢复 |
| **Stem 关键词** | 3 周低能量 + **早醒（凌晨 4 点醒）** + 兴趣丧失 + psychomotor slowing → 问 PSG 发现 = **↓ REM latency** |

**睡眠障碍 PSG 联立鉴别（5 选项各对应一个病）：**

| 疾病 | PSG / 特征 | Stem 关键词 |
|---|---|---|
| **MDD** | ↓ REM latency | 早醒 + 全面情绪 / 兴趣下降 |
| **OSA**（阻塞性睡眠呼吸暂停）| sleep-related hypoventilation / apnea | 打鼾 / 憋醒 + **白天嗜睡** + 肥胖 |
| **RBD**（REM 睡眠行为障碍）| **REM without atonia**（REM 期肌张力不消失）| dream-enactment（梦境扮演、打人踢腿）；关联 α-突触核蛋白病 / 帕金森 |
| **PLMD**（周期性肢体运动障碍）| 睡眠中下肢周期性不自主抽动 → 夜间觉醒 | 下肢抽动致觉醒 → 白天嗜睡 |

> [!danger] ↓REM latency 不是 MDD 专属
> **Narcolepsy 也有 ↓REM latency**（SOREMP）。靠语境分：**抑郁症状群 → MDD**；**日间嗜睡 + 猝倒 + CSF orexin↓ → narcolepsy**（见 §10.2）。

> [!warning] 核心区分（疲劳是共性，靠"怎么个累法"分）
> - **早醒 + 白天没精神 + 抑郁症状群 → MDD**
> - **打鼾 / 憋气 / 白天嗜睡 + 肥胖 → OSA**（"睡了仍困"）
> - **梦境扮演 / 肢体动作 → RBD**（REM without atonia）
> - **下肢抽动致觉醒 → PLMD**

> [!success] Memory Hook
> 抑郁的大脑像"抢跑的运动员" —— REM 迫不及待提前冲出来（latency↓ + 量↑），真正修复身体的深睡（slow-wave）却缩水、凌晨还惊醒。
> buzzword 对照：**REM without atonia → RBD**；**hypoventilation / apnea → OSA**。

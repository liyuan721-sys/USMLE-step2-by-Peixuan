# 多系统重点病 总索引 (Multi-System Disease MOC)

> [!info] 这是"地图"不是"仓库"
> 本卡是 **MOC (Map of Content, 内容地图)** —— **不搬运知识**，只把那些**累及多个器官系统、必须当成整体掌握**的重点病聚合到一个入口。每病知识仍住在它的**学科主卡**里（点链接跳过去），这里只给一句话核心钩子 + 标记**覆盖缺口（🌱 待建）**，方便一眼看出"哪些大病还没建卡"。
> 设计原则：多系统病天然跨学科，硬塞进单一文件夹会脱离学科语境 + 断链，所以用索引而非物理文件夹。

## 一、自身免疫 / 炎症性多系统

- **SLE 系统性红斑狼疮** → [[完整笔记/专题笔记/风湿/风湿_SLE系统性红斑狼疮|风湿_SLE系统性红斑狼疮]]
  - 育龄女 + 非裔；**ANA 筛查 → anti-dsDNA(肾+活动度) / anti-Sm(最特异)**；狼疮肾炎(IV 型最重)；药物性狼疮(anti-histone, HIP 药)
- **血管炎谱 Vasculitis（大/中/小血管）** → [[完整笔记/专题笔记/免疫/免疫_血管炎大小血管谱系|免疫_血管炎大小血管谱系]]
  - 按受累血管口径分层；ANCA 相关 vs 免疫复合物型
- **川崎病 Kawasaki（幼儿中血管炎）** → [[完整笔记/专题笔记/儿科/儿科_川崎病Kawasaki|儿科_川崎病Kawasaki]]
  - 发热 ≥5 天 + CRASH ≥4；**冠脉瘤** + 亚急性期血小板↑；IVIG + 阿司匹林
- **结节病 Sarcoidosis** → [[完整笔记/专题笔记/pulmonary/pulmonary_结节病Sarcoidosis|pulmonary_结节病Sarcoidosis]]
  - 非裔女；非干酪样肉芽肿 / 双侧肺门淋巴结 BHL / 高钙(1α-羟化酶) / ACE↑ / 葡萄膜炎；Löfgren 预后好
- 🌱 **系统性硬化 Systemic Sclerosis / 干燥综合征 Sjögren** — 待建（结缔组织病抗体鉴别：Scl-70 / 着丝点 / anti-Ro·La）

## 二、遗传 / 代谢性多系统

- **SCD 镰状细胞病** → [[完整笔记/专题笔记/血液/血液_镰刀细胞病SCD|血液_镰刀细胞病SCD]]
  - 血管闭塞危象 / **功能性无脾→荚膜菌败血症** / 幼儿 dactylitis 手足综合征 / 急性胸 / 卒中
- **CF 囊性纤维化** → [[完整笔记/专题笔记/儿科/儿科_囊性纤维化CF完整多系统|儿科_囊性纤维化CF完整多系统]]（气道病原纵深见 [[完整笔记/专题笔记/儿科/儿科_囊性纤维化CF气道病原速查|气道病原速查]]）
  - CFTR 基因；胰腺外分泌不全 / 反复铜绿+金葡肺感染 / 男性不育 CBAVD / 脂溶维生素缺乏 / CFRD
- **Turner 综合征 (45,X)** → [[完整笔记/专题笔记/内分泌/内分泌_Turner综合征|内分泌_Turner综合征]]
  - 矮小 / 蹼颈 / 主动脉缩窄 + 二叶瓣 / 卵巢发育不全 / 淋巴水肿
- **DiGeorge 综合征 (22q11 缺失)** → [[完整笔记/专题笔记/心内/心内_CHD_DiGeorge综合征|心内_CHD_DiGeorge综合征]]
  - CATCH-22：心圆锥动脉干畸形 / 甲状旁腺↓低钙 / 胸腺↓T 细胞缺陷 / 腭裂
- **血色病 Hemochromatosis** → [[完整笔记/专题笔记/血液/血液_血色病Hemochromatosis|血液_血色病Hemochromatosis]]
  - HFE C282Y；铁过载"古铜糖尿病" + 肝硬化→HCC + 心肌病 + 关节病 + 性腺↓；放血一线；易感创伤弧菌
- **淀粉样变 Amyloidosis** → [[完整笔记/专题笔记/血液/血液_淀粉样变Amyloidosis|血液_淀粉样变Amyloidosis]]
  - AL(浆细胞)/AA(慢炎)/ATTR(老年心)；刚果红苹果绿双折射；限制型心肌病"二高一低" + 肾综 + 巨舌
- **Marfan 综合征** → [[完整笔记/专题笔记/心内/心内_Marfan综合征|心内_Marfan综合征]]
  - FBN1(AD)；主动脉根扩张→夹层(主死因) + 晶状体向上脱位 + 高瘦蜘蛛指；vs 同型半胱氨酸尿症

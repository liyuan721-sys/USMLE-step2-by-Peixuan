# 中毒昏迷 / 意识改变 床旁五轴鉴别（Comatose / Altered poisoned patient — 5-axis bedside DDx）
#考前必看

> 派生自 [[完整笔记/专题笔记/USMLE/USMLE_毒物综合征toxidrome速查]]（全景百科）的**操作聚焦视图**：把"昏迷/嗜睡的中毒病人如何床旁分流"单拎出来。
> 触发：2026-06-13 自测连续栽两个坑 —— **针尖瞳"干→阿片"反射没建立（Q22）** + **"皮肤干燥"误把 BZD 当 TCA（Q4）**。这两个都是"干性昏迷"内部的细分,本笔记专治。
> 用法：见"昏迷/意识改变 + 疑中毒",**按五轴逐项填** → 落到唯一 toxidrome。**别靠单一症状反射**（针尖瞳≠一定阿片；皮肤干≠一定抗胆碱）。

---

## 〇、五轴框架 ⭐（床旁就按这五栏问）

> [!info] 为什么是这五轴
> 单看一个征象必栽坑（你两次都是）。**五轴一起填**才能锁死:任何两个 toxidrome 至少在一轴上分得开。

```
轴1 瞳孔 pupil       ：针尖 miosis / 散大 mydriasis / 正常有反应
轴2 皮肤干湿 skin     ：干 dry / 大汗+腺体湿 wet / 中性
轴3 呼吸 respiration  ：抑制↓↓ / 正常 / 增快
轴4 生命体征 vitals   ：稳 / 心动过速↑ / 心动过缓↓ / 高热
轴5 特异征 + 解毒     ：肌束颤 / 局灶脑干征 / 宽 QRS / 暴露史 → 对应解毒剂
```

> [!danger] 两条铁律（你的两个错点直接对应）
> 1. **针尖瞳先分"干湿瘫"**:湿=有机磷、干=阿片、瘫+热+突发=脑桥出血。（治 Q22）
> 2. **"皮肤干"只排除了胆碱能,没确诊任何病**:还要看瞳孔（散瞳=抗胆碱/TCA;正常=BZD;针尖=阿片）。（治 Q4）

## 一、主对照大表（7 个易混 toxidrome × 五轴）⭐⭐

| Toxidrome | 瞳孔 | 皮肤 | 呼吸 | 生命体征 | 特异征 + 暴露 | 解毒 / 处理 |
|---|---|---|---|---|---|---|
| **阿片 opioid** | **针尖** | 中性/干 | **↓↓ 抑制** | 缓脉、低压、低体温 | 昏睡;注射痕;空药瓶 | **纳洛酮 naloxone** |
| **有机磷 organophosphate** | **针尖** | **大汗 + 流涎(全湿)** | 浅弱(分泌淹肺) | 缓脉(不可靠) | **肌束颤 fasciculations**;农药/储物棚 | **阿托品 atropine + 解磷定 pralidoxime** |
| **脑桥出血 pontine hemorrhage** | **针尖(对光仍有反应)** | 干、无汗 | 不规则 | **高热 + 高血压史** | **四肢瘫 + 突发 + 局灶脑干征** | **头 CT**(纳洛酮无效) |
| **抗胆碱能 anticholinergic** | **散大** | **干、红、热(无汗)** | 多正常 | **心动过速** | 谵妄狂躁、尿潴留、肠鸣↓ | 支持;重症 **毒扁豆碱 physostigmine** |
| **三环类抗抑郁药 TCA** | **散大** | 干、红、热 | 多正常→重症↓ | 心动过速、**低血压** | **宽 QRS >100ms**、抽搐、心律失常 | **碳酸氢钠 sodium bicarbonate** |
| **拟交感 sympathomimetic** | **散大** | **湿(出汗)** | 增快 | **心动过速 + 高血压 + 高热** | 欣快躁动;可卡因 cocaine/苯丙胺 amphetamine | 苯二氮䓬 benzodiazepine 镇静、降温 |
| **苯二氮䓬 BZD(镇静催眠)** | **正常、有反应** | 中性/干 | 多正常(单用),重度↓ | **常平稳** | 嗜睡可唤醒、共济失调、构音含糊 | 支持;氟马西尼 flumazenil(慎) |

> [!tip] 一眼定位法
> - **针尖瞳** → 只可能是 阿片 / 有机磷 / 脑桥出血(见 §二)。
> - **散瞳** → 抗胆碱 / TCA / 拟交感(见 §三),靠"干湿 + 宽 QRS"切。
> - **瞳孔正常 + 生命体征稳 + 可唤醒** → BZD(见 §四)。

## 二、针尖瞳(miosis)分支:干 / 湿 / 瘫热突发 ⭐（治 Q22）

> 昏迷 + 针尖瞳是高频 stem,三个病因都针尖,**别一看针尖就反射阿片**。

| 病因 | 决定性钥匙 | 皮肤干湿 | 处理 |
|---|---|---|---|
| **阿片 opioid** | **呼吸抑制 + 无湿征**（皮肤干燥温暖、无流涎、无肌束颤）| **干** | **纳洛酮 naloxone** |
| **有机磷 organophosphate** | **湿性胆碱能 DUMBELS**:大汗/流涎/缓脉/腹泻 + **肌束颤** | **全湿** | **阿托品 atropine + 解磷定 pralidoxime** |
| **脑桥出血 pontine hemorrhage** | **突发 + 四肢瘫 + 高热 + 对光仍有反应** + 高血压史 | 干 | **头 CT**;纳洛酮无效 |

> [!danger] 三秒分流 + 双向焊死
> ① **先给纳洛酮**:醒 = 阿片;不醒往下。② 找**湿征**(汗/涎/缓脉/肌束颤) = 有机磷。③ **瘫 + 热 + 突发** = 脑桥出血 → CT。
> 双向钥匙(单向反射就会栽 Q22):
> - **有湿征 → 有机磷 → 阿托品**
> - **无湿征 + 呼吸抑制 → 阿片 → 纳洛酮**
> - 农具棚/农药是**指向有机磷的陷阱暴露史**,但只要"皮肤干 + 无肌束颤" → 仍是阿片,别被场景带走。

---

## 三、散瞳(mydriasis)分支:干(抗胆碱/TCA) vs 湿(拟交感)⭐

> 散瞳 + 躁动 + 高热 + 心动过速,**抗胆碱能 / 拟交感最易混**(表现几乎一样),靠**汗 + 肠鸣**切。

| 钥匙 | 抗胆碱能 / TCA | 拟交感 sympathomimetic |
|---|---|---|
| **皮肤** ⭐ | **干燥无汗**(连汗腺都关) | **大汗淋漓** |
| **肠鸣音** | **减弱/消失** + 尿潴留 | **存在/活跃** |
| 黏膜 | 口干 | 可干 |
| 代表 | 阿托品、抗组胺(苯海拉明 diphenhydramine)、TCA | 可卡因 cocaine、苯丙胺 amphetamine、MDMA(亚甲二氧基甲基苯丙胺 / 摇头丸 ecstasy) |

> [!warning] 抗胆碱能 vs TCA —— 都"干红热散瞳",靠"宽 QRS"再切一刀
> TCA 本质就是强抗胆碱能,但**多了钠通道阻滞**:
> - **纯抗胆碱(苯海拉明/阿托品)**:干红热散瞳 + 心动过速,**QRS 正常** → 支持/(重症)毒扁豆碱 physostigmine。
> - **TCA**:上述 + **宽 QRS >100ms + 低血压 + 抽搐 + 心律失常**(致命) → **碳酸氢钠 sodium bicarbonate**(不是毒扁豆碱!)。
> 钥匙:**心电图宽 QRS = TCA(给碳酸氢钠);QRS 正常 = 单纯抗胆碱**。

---

## 四、"干性"昏迷三毒互鉴:阿片 vs 抗胆碱/TCA vs BZD ⭐⭐（治 Q4）

> [!danger] 这是 2026-06-13 Q4 的核心坑:"皮肤干燥"把 BZD 误判成 TCA
> **"皮肤干"只说明没有胆碱能湿征,不指向任何一个具体病。** 三个"不湿"的昏迷毒,**瞳孔是分水岭**:

| | 瞳孔 ⭐分水岭 | 呼吸 | 生命体征 | 神志 | 致命性 | 处理 |
|---|---|---|---|---|---|---|
| **阿片 opioid** | **针尖** | **↓↓ 抑制** | 缓脉、低压 | 昏睡难唤醒 | 高(呼吸停) | 纳洛酮 naloxone |
| **抗胆碱/TCA** | **散大** | 正常→(TCA)↓ | **心动过速**、(TCA)低压宽QRS | 谵妄狂躁 | (TCA)高 | (TCA)碳酸氢钠 / (抗胆碱)毒扁豆碱 |
| **BZD 镇静催眠** | **正常、对光有反应** | 多正常 | **常平稳** | **嗜睡可唤醒**、共济失调 | 低(单用) | 支持;氟马西尼(慎) |

> [!success] 三步锁死(背这个)
> 见"昏迷/嗜睡 + 皮肤不湿":
> 1. **看瞳孔**:针尖 → 阿片;散大 → 抗胆碱/TCA;**正常有反应 → BZD**。
> 2. **看生命体征**:稳 → BZD;心动过速/狂躁 → 抗胆碱/TCA;呼吸抑制 + 缓脉 → 阿片。
> 3. **TCA 再查心电图**:宽 QRS → 碳酸氢钠。
> 一句话:**"皮肤干"别反射 TCA —— 瞳孔正常 + 生命体征稳 + 能叫醒 = BZD,最良性那个。**

## 五、床旁分流决策树 ⭐

```
昏迷/意识改变 + 疑中毒
│
├─ 瞳孔针尖 miosis ──┬─ 有湿征(汗/涎/肌束颤) → 有机磷 → 阿托品+解磷定
│                    ├─ 无湿征 + 呼吸↓↓        → 阿片   → 纳洛酮
│                    └─ 瘫+热+突发+对光有反应   → 脑桥出血 → 头CT
│
├─ 瞳孔散大 mydriasis ┬─ 干 + 肠鸣↓ ──┬─ QRS正常 → 抗胆碱 → (重症)毒扁豆碱
│                     │                └─ QRS宽   → TCA   → 碳酸氢钠
│                     └─ 湿(大汗) + ↑BP↑HR → 拟交感 → 苯二氮䓬镇静
│
└─ 瞳孔正常有反应 + 生命体征稳 + 可唤醒 → BZD/镇静催眠 → 支持(氟马西尼慎)
```

## 六、Memory Hook

> [!success] 一句话
> **昏迷中毒按五轴(瞳孔/干湿/呼吸/生命体征/特异征)填,别单症状反射。**
> - **针尖瞳**:湿=有机磷、干=阿片、瘫热突发=脑桥出血。
> - **散瞳**:干=抗胆碱/TCA(宽QRS=TCA给碳酸氢钠)、湿=拟交感。
> - **瞳孔正常+生命体征稳+能叫醒=BZD**(最良性)。
> 两条防栽:**"农药场景+皮肤干+无肌束颤"仍是阿片;"皮肤干"≠TCA,先看瞳孔。**

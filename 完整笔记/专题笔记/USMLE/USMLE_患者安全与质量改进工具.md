## 九、Hospice & End-of-life Care — Q21141 错题考点

> [!info] 引用 Ethics_Master §八
> 详细 advance directive / DNR / brain death / double effect / PAS 见 [[完整笔记/Peixuan分科笔记/Ethics_Master]] 第八节。本节只列**Hospice eligibility 高频考点**。

### 9.1 Hospice 适用标准

| 项目 | 要求 |
|---|---|
| **病情** | Terminal illness + life expectancy **≤ 6 months** |
| **治疗意愿** | 同意**放弃 curative / life-prolonging treatment** ⭐ |
| **可继续** | **Palliative treatment**（缓解症状）— 包括 palliative radiotherapy / palliative surgery / 止痛 / 抗生素 / **针对急性可逆病** 的治疗（如 MI / pneumonia 不相关于主诊断）|

### 9.2 易混陷阱（Q21141 核心考点）

> [!danger] Q21141 4 大陷阱
> 1. "病人必须停 ALL cancer treatment 才能 hospice" ❌ — 只需停 **curative**；**palliative radiotherapy 可继续**
> 2. "继续 immunotherapy 也可以 hospice" ❌ — Immunotherapy (nivolumab + ipilimumab) 是 **curative intent**，必须停
> 3. "必须签 DNR 才能 hospice" ❌ — Hospice **不强制** DNR；多数病人自愿签但**不是 eligibility 要求**
> 4. "Hospice 病人不能接受任何手术" ❌ — Palliative surgery（如 bowel obstruction 缓解症状）可以

### 9.3 Hospice vs Palliative Care 区别

| 概念 | Hospice | Palliative care |
|---|---|---|
| **生存期** | ≤ 6 月 | 任何阶段（含早期） |
| **必须放弃 curative** | ✅ 是 | ❌ 否（可同时 curative） |
| **保险** | Medicare hospice benefit | 普通保险 |
| **场景** | Home / hospice facility / nursing home | 任何 |

## 十、9 大陷阱反 Pattern（应试反射）

| # | 错误反射 | 正确思维 |
|---|---|---|
| 1 | "Most effective = 培训员工" | ❌ — Education 是最弱层；选系统改造 |
| 2 | "新单位还没开 → 用 RCA 找潜在错误" | ❌ — RCA 是事后；事前选 FMEA |
| 3 | "Counting interrupted → 培训权威沟通" | ❌ — 真问题是 awareness/interruption；选 time-out |
| 4 | "多层 imperfect 防御 = hard stop" | ❌ — Hard stop 是单一物理；多层叠加 = Swiss Cheese |
| 5 | "Hospice 要必须签 DNR" | ❌ — Hospice ≠ DNR；只需放弃 curative |
| 6 | "Error 没 harm → 不必披露" | ❌ — Disclosure 与 harm 无关 |
| 7 | "Near-miss 不用上报" | ❌ — Near-miss **必须**上报（学习机会，无害 + 高信号） |
| 8 | "上报系统会用来惩戒员工" | ❌ — Non-punitive + system-level，才有人报 |
| 9 | "床号 + 房间号当 identifier" | ❌ — 必须用 **不会变** 的 2 个（姓名 + DOB） |

## 十一、Memory Hooks（一句话锁定）

> **"事前 FMEA，事后 RCA，监测 control chart，迭代 PDSA"** — QI 工具 0.5 秒锁定
>
> **"防御金字塔顶端 = 物理 hard stop；底端 = 培训"** — Most effective 题秒杀
>
> **"Swiss Cheese = 多层不完美叠加；Hard stop = 单层物理阻断"** — 区分 Q21477 陷阱
>
> **"Time-out = 暂停 + 同步 awareness + 减 interruption"** — 不是治权威 / intimidation
>
> **"Hospice 三要素 = 6 月 + 放弃 curative + 可保 palliative；不强制 DNR"** — Q21141 锁定
>
> **"Near-miss 必报；sentinel event 必 RCA"** — Error 名词记忆
>
> **"Just culture：human error 不罚 / at-risk 教育 / reckless 罚"** — 三层文化
>
> **"STEEEP = Safety + Timeliness + Effectiveness + Efficiency + Equity + Patient-centered"** — Healthcare quality 6 维（IOM 框架）
>
> **"Donabedian 5 类：Structural（人/床）→ Process（执行率）→ Outcome（结局）→ Balancing（系统间）→ Composite（综合）"** — measure 配对题 0.3 秒
>
> **"HRO 5 原则：Preoccupation with failure + Reluctance to simplify + Sensitivity to operations + Resilience + Deference to expertise"** — 防 zero-error utopia 陷阱
>
> **"ASP 多学科 → 减 C. diff / MDR / cost"** — Antibiotic stewardship 关键词
>
> **"BCA：ratio > 1 或 net benefit > 0 才实施"** — Benefit-cost analysis 决策铁律
>
> **"Common cause = inherent + 不可 trace → standardization；Special cause = sporadic + 可 trace → RCA"** — Variation 区分 ⭐⭐⭐
>
> **"海浪日日有 → 防波堤（standardize）；海啸罕见但致命 → 查震源（RCA）"** ⭐ — Common vs Special cause 视觉记忆 hook（v1.3.9 新增）
>
> **"SMART = Specific + Measurable + Assignable + Realistic + Timely"** — PDSA Plan stage 必背
>
> **"PDCA = check compliance（business）；PDSA = study + continuous learning（health care 首选）"** — 区分
>
> **"Pareto 找最大问题（80/20）；Control chart 判 variation（有 limits）；Run chart 看趋势（无 limits）"** — 3 大 measurement tool
>
> **"4 大 QI Methodology：PDSA（迭代）+ Lean（减 waste）+ Six Sigma（减 defect 到 6σ）+ Change management（人因 + champion / early adopter）"** ⭐⭐⭐（v1.3.15 AMBOSS 4 联表锚定）
>
> **"Medical errors = 3rd leading cause of death US (~250,000/yr)"** — Epidemiology 背景
>
> **"Adverse event 4 类：Preventable / Ameliorable（不可防但严重度可降）/ Near-miss / Never-sentinel"** — 注意 Ameliorable 新概念
>
> **"Medical error 4 维分类：Omission/Commission + Active/Latent + Individual/Systems + Execution/Planning"** ⭐⭐⭐
>
> **"Lab error 75% 在 preanalytical phase"** — hemolysis / 量不足 / 标错样本
>
> **"Hazard = 源头；Risk = probability；Risk factor = 增加 probability 的变量"** — 三层定义
>
> **"Teamwork 5 domains：Communication + Teamwork + Leadership + Situation monitoring + Conflict resolution"**
>
> **"Hierarchies = #1 teamwork barrier；Psychological safety 是反对策（不 blame junior nurse）"** ⭐
>
> **"NVC 4 components: Observation + Feelings + Needs + Requests（Marshall Rosenberg）"**
>
> **"Crisis Resource Management 借自航空业 — 角色 + 闭环 + 接受任何 team member input"**
>
> **"Procedural iatrogenic harm 配对：Para→bowel perf；Central line→thrombosis/bleed；Thoracentesis→pneumothorax；LP→paralysis/bleed"** ⭐⭐⭐
>
> **"Risk factors 3 大类：Workplace（understaffing/workload）+ HCP（inexperience/fatigue/Alert fatigue/Burnout）+ Patient（SDOH: health literacy/race/language/Medicaid）"**
>
> **"Alert fatigue 解决：减少 irrelevant alert + 分级 severity + Forcing function 留给 high-harm"** ⭐
>
> **"Implicit bias 经典：Hispanic/Black 给少 analgesia（即使疼痛评分相同）"** ⭐⭐
>
> **"Anchoring = 锚定第一信息；Availability = 锚定最易回忆案例；Premature closure = 第一个 dx 就停 = search satisficing"** ⭐⭐⭐
>
> **"Commission = 倾向做；Omission = 倾向不做"** — bias 区分
>
> **"Diagnostic error prevention：允许病人 uninterrupted speak ≥ 60 秒 + 检查 'differential 只有一个 dx?'"**
>
> **"HFE 5 methods：Forcing function + Standardization + Universal protocol + Simplification + Usability testing"**
>
> **"Universal protocol 3 步：preprocedural verification + site marking（病人清醒）+ time-out"** — Joint Commission
>
> **"Tall man lettering: DOBUTamine vs DOPamine；Do Not Use: U/IU/QD/QOD/MS/MSO4/trailing zero/leading zero 缺失"** ⭐⭐⭐
>
> **"Trailing zero / lack of leading zero → 10 倍剂量错"** — transcription error 高频
>
> **"Teach-back 黄金问法：'What questions do you have for me?'（不是 'Do you have any questions?'）"**
>
> **"Prescribing cascade 3 例：NSAIDs→HTN→antiHTN / Thiazide→hyperuricemia→gout / Antipsychotics→parkinsonism→antipark"** ⭐⭐
>
> **"C. diff 手卫生用 soap+water（alcohol gel 无效）；HAC 2008 起 CMS 不报销"**
>
> **"Disclosure 6 要素 + Apologize without qualification — 不说 'feel that way' / 'took it that way' / 'sorry BUT'"** ⭐⭐⭐
>
> **"IRS 只 7% errors 被报；Voluntary 自报偏倚；不能衡量 overall safety"**
>
> **"5 Whys root cause 在系统层，不停在 'nurse 粗心'"** ⭐⭐
>
> **"M&M vs RCA vs FMEA：M&M = retrospective peer review 多 case 法律保护；RCA = retrospective 单 event；FMEA = prospective"** ⭐⭐
>
> **"4 工具时间方向 + 范围：RCA = retro/单事件；FMEA = pro/未来；Common Cause = retro/多事件 themes；M&M = retro/周期 peer review"** ⭐⭐⭐（v1.3.4 Q22581 锚定）
>
> **"4 大经典 cognitive bias：Anchoring（第一印象）/ Availability（recency）/ Confirmation（只看支持证据）/ Framing（外部信息影响）；Anchoring = premature closure 互换"** ⭐⭐⭐（v1.3.5 Q18797 + Q18796）
>
> **"CDSS = HFE Automation 档；擅长 detect mathematical/dose conversion error；不能 detect patient ID/mislabeling（这些需 dual ID + barcode）"** ⭐⭐（v1.3.5 Q18754）
>
> **"Hand hygiene 时序：Step 1 systems（sinks/training/monitoring）→ Step 2 individual accountability（only after sustained systems reform）"** ⭐⭐（v1.3.6 Q106042）
>
> **"Systems Approach 4 Components：Culture of safety + Resilience + Team-based vigilance + Comprehensive analysis；Goal = Zero adverse events 不是 Zero human error"** ⭐⭐⭐（v1.3.6 Q22079）
>
> **"Value = Quality / Total Cost；单 metric = partial info；Total cost 含 long-term + short-term"** ⭐⭐（v1.3.6 Q22780）
>
> **"Anchoring = 自己的锚（medical fixates own initial impression）；Framing = 别人给的画框（external chart/context/wording frames you）；区分 bias 源头是自己还是外部"** ⭐⭐⭐（v1.3.7 用户深度提问）
>
> **"Raw fact（diagnosed/history/presents with）→ Anchoring；Interpretation（likely/probably/looks/clearly/consistent with）→ Framing；去掉 interpretation 测试 bias 是否消失"** ⭐⭐⭐（v1.3.8 二次深化）
>
> **"AMA 5 要素：Capacity + Severity + Reasons + Alternatives + Documentation"** ⭐⭐
>
> **"5 conflict styles: Avoiding / Accommodating / Competing / Compromising / Collaborating（多数情境推荐 Collaborating）"**
>
> **"Workplace harassment 3 forms: Verbal / Physical / Sexual；Address: set boundary + nurse 陪同 + ombuds office + 文档化"** ⭐
>
> **"Angry patient: Safety first（door/distance/security）+ calm + acknowledge + identify cause（fatigue/delay/error）"**
>
> **"Second victim: peer support + EAP + 不孤单 + 找 system-level 改进机会（不只 self-blame）"**
>
> **"Clinical uncertainty 4 策略：basic principles + review workup + consult + watchful waiting；沟通用 shared decision-making；持续症状考虑 somatic disorder"**
>
> **"Effective teams 6 元素：Safety culture / Collaboration（distinct roles avoid steep hierarchy）/ Psychologic safety（avoid blame）/ Responsiveness / Situational awareness / Transparency"** ⭐⭐⭐
>
> **"HFE 3 档 reliability：Highest = Forcing + Automation + Environment & physical layout；High = Standardization + Human-machine redundancy；Medium = Reminders + Double-checks；Less = Training/Policy/Education"** ⭐⭐⭐
>
> **"High variation = ↑ cost + ↓ quality + ↓ value；Reduce variation 首选 = Data-driven individual feedback vs benchmarks（不是 workshop / patient-centered / malpractice insurance）"** ⭐⭐
>
> **"Lean = 流畅 🌊（减 waste / wait time / no-show）；Six Sigma = 精准 🎯（减 defect / variation 到 6σ）"** ⭐⭐⭐（v1.3.15 视觉 Hook — Lean vs Six Sigma 5 维区分核心）
>
> **"Change management 三件套：Champion（推动者）+ Early adopter（先用者）+ Culture change（不只是流程改）"** ⭐⭐（v1.3.15 AMBOSS 新增第 4 methodology — 人因驱动）
>
> **"Swiss Cheese / RCA / Fishbone / 5 Whys = 分析框架（理解 error）；PDSA / Lean / Six Sigma / Change mgmt = 改进 methodology（防止 error）；FMEA / Control chart / Pareto = QI 工具（具体操作）"** ⭐⭐⭐（v1.3.15 — 3 层抽象边界）

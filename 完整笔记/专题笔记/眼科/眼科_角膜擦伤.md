# 角膜擦伤 Corneal Abrasion 处理（外伤性 vs 隐形眼镜 + 何时紧急转诊）

> [!info] 用途与定位
> 本篇是**角膜擦伤（角膜上皮缺损）处理流程的下沉专题**：先排"红旗"决定要不要**急诊眼科会诊**，再按**外伤/异物 vs 隐形眼镜**两分支给不同抗生素。核心高频点 = **隐形眼镜相关擦伤要覆盖铜绿假单胞 (Pseudomonas) + 绝不贴眼罩 + 24h 复查**。感染性溃疡（浸润/树枝状/羽毛状）见 [[完整笔记/专题笔记/眼科/眼科_感染性角膜炎]]；眼外伤秒选（open globe / hyphema / blowout）见 [[完整笔记/专题笔记/眼科/眼科_HighYield速查总览]] "六、眼外伤秒选表"。知识源 [[完整笔记/Peixuan分科笔记/眼科]]（只读）。

> [!note]- 缩写对照（全称 中英）
> - **FB** = foreign body 异物
> - **CL** = contact lens 隐形眼镜
> - **Pseudomonas** = *Pseudomonas aeruginosa* 铜绿假单胞菌
> - **Seidel test** = 荧光素下房水外漏试验（查角膜全层穿孔）

---

## 一、角膜擦伤是什么 + 怎么确诊

> [!tip] 一句话
> **角膜擦伤 = 角膜上皮的机械性缺损**（未穿透全层）。典型 = **眼痛 + 异物感 (foreign body sensation) + 畏光 + 流泪**，多有外伤/揉眼/戴镜史。

- **诊断**：**荧光素染色 (fluorescein) + 裂隙灯** → 缺损区染绿。
- **必做两件事**：
  - **翻上睑 (evert eyelid)** 找残留异物（尤其线状/多发划痕提示睑下异物）。
  - 怀疑穿孔 → **Seidel test**（荧光素被外漏房水冲开呈"小溪"= 阳性 = **open globe 开放性眼球损伤**，禁压眼球）。

---

## 二、处理流程（先排红旗，再分两支）

```
眼痛 + 异物感
     │
     ▼
① 先问：有没有"红旗"？（任一个 Yes → 急诊眼科会诊）
     ├─ 眼球穿孔 / Seidel+（open globe）
     ├─ 角膜浸润 (infiltrate)：白点 / 混浊 → 提示感染性溃疡
     ├─ 角膜溃疡 (corneal ulcer)
     ├─ 瞳孔不规则（提示眼球破裂）
     └─ 异物嵌顿、不易取出
     │
     └─ 全部 No → 按病因分两支处理
              │
      ┌───────┴────────────────────────┐
      ▼                                 ▼
【外伤 / 异物性擦伤】                【隐形眼镜相关擦伤】
 · 冲洗 (irrigation) 去异物            · topical 抗铜绿假单胞抗生素
 · topical 抗生素（erythromycin        （ofloxacin / ciprofloxacin
   红霉素 / polymyxin-trimethoprim       / tobramycin 妥布霉素）
   多黏菌素-甲氧苄啶）                  · ⚠️ 不贴眼罩（感染风险）
 · 小缺损(<3mm)+症状好转+视力正常        · 24h 复查，确认无浸润/溃疡
   +无异物 → 无需常规随访
```

> [!danger] 红旗 = 任一出现就急诊眼科会诊（别自己按普通擦伤处理）
> 眼球穿孔 / Seidel+、**角膜浸润（白点、混浊 → 已经是感染性溃疡不是单纯擦伤）**、角膜溃疡、**瞳孔不规则**（提示 open globe）、**异物嵌顿取不出**。这些说明"不只是擦伤"，延误 = 视力/眼球损失。

---

## 三、两分支对比（★最高频考点：CL 擦伤 ≠ 普通擦伤★）

| 维度 | 外伤 / 异物性擦伤 | **隐形眼镜 (CL) 相关擦伤** |
|---|---|---|
| 病因 | 指甲、纸张、树枝、睑下异物 | 戴镜不当 / 过夜 / 卫生差 |
| 病原顾虑 | 一般皮肤菌群 | **铜绿假单胞 (Pseudomonas)**——进展快、可数小时穿孔 |
| 抗生素 | **erythromycin 红霉素 / polymyxin-trimethoprim 多黏菌素-甲氧苄啶** | **抗铜绿：ofloxacin 氧氟沙星 / ciprofloxacin 环丙沙星 / tobramycin 妥布霉素**（氟喹诺酮或氨基糖苷） |
| 眼罩 | 一般也不推荐 | **绝对不贴眼罩**（封闭潮湿助细菌繁殖 → 感染） |
| 随访 | 小缺损<3mm + 好转 + 视力正常 + 无异物 → 可不随访 | **必须 24h 复查**，排早期浸润/溃疡 |

> [!warning] 为什么隐形眼镜擦伤要特殊对待
> 戴镜者角膜擦伤 = **Pseudomonas 角膜炎/溃疡的高危背景**。所以①抗生素必须**覆盖铜绿假单胞**（普通红霉素不够）；②**贴眼罩会制造厌氧潮湿环境助长细菌**，故禁忌；③**24h 内复查**抓早期溃疡。这三条正是 UWorld 反复考的分叉点。

---

## 四、几个高频"别做"（治疗禁区）

> [!danger] 处理角膜擦伤的三不要
> - **不要开"外用麻醉剂 (topical anesthetic，如 proparacaine 丙美卡因) 让病人回家点"**——短期查体可用，长期用**延迟上皮愈合 + 角膜毒性 + 掩盖恶化**。
> - **不要外用糖皮质激素**——延迟愈合 + 若合并 HSV（herpes simplex virus 单纯疱疹病毒）/真菌会加重感染。
> - **隐形眼镜擦伤不要贴眼罩**（见上）。普通擦伤贴眼罩也不改善愈合、已不推荐。

---

## 五、鉴别：擦伤 vs 该转诊的"更重角膜病"

| 诊断 | 荧光素 | 关键区分点 |
|---|---|---|
| **角膜擦伤 corneal abrasion** | 阳性（局限缺损） | 外伤/异物/戴镜史；**无白色浸润** |
| **感染性角膜炎/溃疡 keratitis/ulcer** | 阳性 + **基质浸润 (infiltrate)** | 白点/混浊 = 感染 → 急诊转诊；溃疡形态定病原（见 [[完整笔记/专题笔记/眼科/眼科_感染性角膜炎]]） |
| **复发性角膜糜烂 recurrent erosion** | 阳性 | 常**晨起睁眼时复发痛**，既往擦伤史 |
| **紫外线/光性角膜炎 UV/photokeratitis** | **弥漫点状 (punctate)** | 电焊/雪盲/日光浴、**双眼**、6–12h 后剧痛 |

### Photokeratitis 紫外线角膜炎

![[{9FC83E2A-3A5F-4B4C-AE98-CB072F303D99}.png]]

*Photokeratitis 紫外线角膜炎：UV 暴露（高原/雪地/电焊）6–12h 后双眼剧痛 + 畏光 + 异物感；荧光素见点状角膜染色；支持治疗 1–3 天自愈；预防 = UV 护目镜*

## 六、经典 stem → 处理反射

- **戴隐形眼镜**者眼痛异物感 + 荧光素染色缺损、无白色浸润 → **CL 擦伤** → **抗铜绿假单胞滴眼（ofloxacin/cipro/tobramycin）+ 不贴眼罩 + 24h 复查**。
- 指甲划伤/纸割后异物感 + 小缺损、视力正常 → **单纯外伤性擦伤** → erythromycin 红霉素，小缺损可不随访。
- 荧光素见白色**浸润灶/溃疡** → 已不是单纯擦伤 → **急诊眼科会诊**（感染性溃疡）。
- 擦伤后给外用麻醉剂回家点、数日不愈 + 角膜混浊 → **麻醉剂滥用致上皮毒性/不愈**。
- 高能钝伤后**瞳孔不规则 / Seidel+** → 疑 **open globe 开放性眼球损伤**，放 rigid shield 硬质眼盾、别压眼球、急诊。

## 七、自测（答完再判分）

> [!question]- Q1. 戴隐形眼镜者晨起眼痛、异物感、畏光，荧光素见角膜局限染色，无白色浸润。首选抗生素类别 + 为什么不贴眼罩 + 随访安排？
> **抗铜绿假单胞 (Pseudomonas) 抗生素**：氟喹诺酮 (ofloxacin/ciprofloxacin) 或 tobramycin 妥布霉素。**不贴眼罩**——封闭潮湿环境助细菌繁殖、增感染风险。**24h 复查**排早期角膜浸润/溃疡。

> [!question]- Q2. 角膜擦伤处理中，哪些"红旗"提示应立即眼科会诊而非按普通擦伤处理？
> 眼球穿孔/Seidel+、**角膜浸润（白点/混浊）**、角膜溃疡、**瞳孔不规则**、异物嵌顿取不出。

> [!question]- Q3. 为什么不能给角膜擦伤患者开外用麻醉剂 (proparacaine) 回家长期点？
> **延迟上皮愈合 + 角膜毒性 + 掩盖病情恶化**；外用麻醉剂只在查体时短期用。

> [!question]- Q4. 单纯外伤性小擦伤（<3mm、视力正常、无异物、症状好转）需要常规随访吗？隐形眼镜擦伤呢？
> 单纯小擦伤**可不随访**；**隐形眼镜擦伤必须 24h 复查**（Pseudomonas 溃疡高危）。

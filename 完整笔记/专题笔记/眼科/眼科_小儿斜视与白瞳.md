# 小儿斜视与白瞳鉴别（strabismus · leukocoria · retinoblastoma）

> [!info] 用途与定位
> 本篇收**小儿"眼位不正 / 红反射异常"的评估**：斜视 (strabismus) 的体征与"下一步"，白瞳 (leukocoria) 的鉴别，以及必须排的致命病 **视网膜母细胞瘤 (retinoblastoma)**。核心反射：**儿童斜视或白瞳 → 扩瞳眼底检查 (dilated fundoscopy) 排 retinoblastoma**。先天性青光眼（流泪畏光 + 角膜增大）见 [[完整笔记/专题笔记/眼科/眼科_青光眼]] "七、小儿青光眼"。知识源 [[完整笔记/Peixuan分科笔记/眼科]]（只读）。

> [!note]- 缩写对照（全称 中英）
> - **VA** = visual acuity 视力
> - **RB** = retinoblastoma 视网膜母细胞瘤
> - **ROP** = retinopathy of prematurity 早产儿视网膜病变
> - **EOM** = extraocular movement 眼外肌运动
> - **CN** = cranial nerve 颅神经
> - **ICP** = intracranial pressure 颅内压
> - **VEP** = visual evoked potential 视觉诱发电位
> - **MRI** = magnetic resonance imaging 磁共振成像

## 一、Strabismus 斜视：体征 + 下一步

> [!info] 定义
> **Strabismus（斜视）= 双眼眼位不对齐 (ocular misalignment)**。按方向：**esotropia 内斜（向鼻侧偏）/ exotropia 外斜（向颞侧偏）**。
> - **未治后果**：偏斜眼长期被抑制 → **弱视 (amblyopia)**（该眼视力发育停滞、永久 VA↓）——所以要早查早治。

> [!tip] 三个体征（都靠"两眼对不对称"发现）
> - **角膜映光不对称 (asymmetric corneal light reflex / Hirschberg 试验)**：正常两眼角膜反光点都在瞳孔中央对称；斜视时偏斜眼的反光点**偏离中央**。
> - **遮盖试验异常 (abnormal cover test)**：盯一个目标，遮住一眼；若**未遮眼要移动去重新对准目标** = 该眼原来是斜的（正常眼不动）。
> - **红反射不对称 (asymmetric red reflex)**：偏斜眼的红反射**更亮/更强**（Brückner 试验）。

> [!warning] 大多数斜视是特发/集合不足，但必须先排"继发性 (secondary) 病因"
> 斜视多为特发或**集合不足 (convergence insufficiency)**，但可能是**危及生命/视力**的病的信号：
> - **Retinoblastoma 视网膜母细胞瘤**（中央视力丧失 → 眼失去固视 → 斜视）——★必排★
> - **先天性白内障 cataract**、**CN III/VI 麻痹**（尤其 **CN VI 展神经麻痹 → 不能外展**，警惕 ↑ICP）、**颅内占位/↑ICP**
> - **➡ 下一步 = 扩瞳眼底检查 (dilated fundoscopy)** 找继发原因（尤其 retinoblastoma 的视网膜肿块）。

> [!danger] 别把"家族性青光眼"当本病线索
> 斜视 + 不对称角膜映光/红反射 ≠ 青光眼表现。**先天性青光眼**是**流泪 + 畏光 + 睑痉挛 + 角膜增大 + 视盘 cupping**（见 [[完整笔记/专题笔记/眼科/眼科_青光眼]] 七）；测眼压 (tonometry) 只在眼底见 cupping 提示青光眼时才做。斜视本身的"下一步"是扩瞳眼底，不是测眼压。

## 二、Leukocoria 白瞳鉴别（红反射异常）

> [!info] Leukocoria = 白瞳
> **Leukocoria（白瞳）= 瞳孔区呈白色反光 / 红反射消失或异常**——是儿童严重眼病的红旗信号。**白瞳或斜视 → 扩瞳眼底检查**。

| 病因 | 关键钩子 |
|---|---|
| **Retinoblastoma 视网膜母细胞瘤** | **最重要（致命恶性）**；白瞳（最常见）或斜视；乳白色视网膜肿块 |
| **先天性白内障 congenital cataract** | 晶状体混浊；TORCH 感染、代谢病、遗传 |
| **ROP 早产儿视网膜病变** | 早产 + 低出生体重 + 吸氧史 |
| **Coats 病 / 弓蛔虫病 (toxocariasis) / 永存胎儿血管** | 少见白瞳原因 |

> [!tip] 儿保筛查
> 常规儿保**红反射检查 (red reflex test)**：**不对称 / 白色 → 立即扩瞳眼底 + 转眼科**（排 retinoblastoma / cataract）。

## 三、Retinoblastoma 视网膜母细胞瘤

> [!danger] 最常见儿童眼内恶性肿瘤，致命但可治
> **Retinoblastoma（RB，视网膜母细胞瘤）= RB1 抑癌基因失活**导致的视网膜恶性肿瘤，多在 **<5 岁**。
> - **遗传型 (hereditary)**：常**双侧/多灶**、家族史、可伴其他肿瘤（骨肉瘤等）风险。
> - **散发型 (sporadic)**：多**单侧单灶**。

| 维度 | 表现 |
|---|---|
| 最常见表现 | **Leukocoria 白瞳**（相片红眼变白点） |
| 次常见表现 | **Strabismus 斜视**（中央视力丧失致固视丢失） |
| 诊断 | **扩瞳眼底检查**（乳白色结节状视网膜肿块）+ **眶/脑 MRI**（评估侵犯/视神经/松果体） |
| 处理 | 眼科肿瘤专科（激光/冷冻/化疗/放疗/眼球摘除，按分期）；遗传咨询 |

> [!danger] Item 8 (Q17184) 做对复盘：2 岁斜视 + 不对称角膜映光/红反射 → 下一步扩瞳眼底（排 retinoblastoma）
> 2 岁男孩体检，母诉**间断"斗鸡眼" (cross-eyed)**；**右眼轻度向鼻侧偏 (esotropia)** + **瞳孔红反射不对称** + **无突眼、EOM 正常无痛** + 父亲有青光眼史 → **strabismus（斜视）** → **下一步 = dilated fundoscopic examination（扩瞳眼底检查）**，排继发病尤其 **retinoblastoma**。
> - ❌ **Fluorescein examination（选 B）**：荧光素查**角膜擦伤**（眼痛/流泪/畏光）——斜视不引起，且擦伤无眼位偏 → 不符。
> - ❌ **Intraocular pressure measurement（选 C）**：测眼压用于**眼底见 cupping 提示青光眼**时；本例是斜视 + 不对称反射，非青光眼表现；"父亲青光眼史"是**烟雾弹** → 不是首步。
> - ❌ **MRI of brain（选 D）**：MRI 用于**眼底已见视网膜肿块**后确诊/分期 retinoblastoma，或展神经麻痹查颅内占位；**扩瞳眼底应在前** → 顺序不对。
> - ❌ **Visual evoked potential (VEP)（选 E）**：用于**视神经炎**（视力↓ + 眼球运动痛）；与斜视无关 → 不符。
> - 🔑 钥匙：**"儿童斜视 / 白瞳 → 先扩瞳眼底排 retinoblastoma"**；不对称角膜映光 + 不对称红反射 = 斜视的两把尺；家族性青光眼是干扰。

## 四、经典 stem → 处理反射

- 幼儿**间断斗鸡眼 + 角膜映光不对称 + 红反射不对称** → **斜视** → **扩瞳眼底检查**（排 retinoblastoma）。
- 相片里一眼**红眼、另一眼白点 (leukocoria)** / 儿保**红反射不对称** → 扩瞳眼底 → 疑 **retinoblastoma**（乳白肿块）→ 眶/脑 MRI。
- **双侧** retinoblastoma + 家族史 → **遗传型（RB1 胚系）**，遗传咨询 + 查其他肿瘤风险。
- 斜视 + **不能外展一眼** → **CN VI 展神经麻痹** → 查 ↑ICP / 颅内占位（MRI）。
- 婴儿**流泪 + 畏光 + 睑痉挛 + 角膜增大** → **先天性青光眼**（不是单纯斜视）→ 见 [[完整笔记/专题笔记/眼科/眼科_青光眼]] 七。

## 五、自测（答完再判分）

> [!question]- Q1. 2 岁男孩间断斗鸡眼，右眼轻度内斜 + 红反射不对称，无突眼、EOM 正常，父亲有青光眼史。下一步最佳处理？为什么不是测眼压或 MRI？
> **Dilated fundoscopic examination（扩瞳眼底检查）**，排继发病尤其 **retinoblastoma**。不测眼压：斜视 + 不对称反射非青光眼表现，家族青光眼史是干扰。不先 MRI：MRI 在眼底见肿块后确诊/分期，扩瞳眼底应在前。

> [!question]- Q2. 儿童斜视的两个体征 + 未治疗的主要后果是什么？
> **角膜映光不对称 (Hirschberg) + 遮盖试验异常**（红反射不对称也是）；未治 → **弱视 (amblyopia)**（偏斜眼视力发育停滞）。

> [!question]- Q3. 白瞳 (leukocoria) 的鉴别里最不能漏的诊断是什么？最常见与次常见表现各是？
> 最不能漏 = **retinoblastoma 视网膜母细胞瘤**（致命恶性）。最常见表现 = **白瞳**；次常见 = **斜视**（中央视力丧失致固视丢失）。

> [!question]- Q4. 单侧单灶 vs 双侧多灶的 retinoblastoma 分别提示什么遗传背景？
> **单侧单灶 → 多为散发型 (sporadic)**；**双侧/多灶 → 遗传型 (hereditary, RB1 胚系突变)**，需遗传咨询 + 关注其他肿瘤风险。

# 眶周与眼内急症（眶隔前 vs 眶蜂窝织炎 · 眶间隔综合征 OCS · 眼内炎）

> [!info] 用途与定位
> 本篇收**"眼球周围红肿突眼 / 眼内视力威胁"的急症**：①感染性——**眶隔前 (preseptal) vs 眶 (orbital) 蜂窝织炎**；②外伤性——**眶间隔综合征 (orbital compartment syndrome, OCS)**；③眼内感染——**眼内炎 (endophthalmitis)**。共同线索 = **突眼 / 视力↓ / 眼球运动痛**，靠"有无外伤、有无鼻窦炎、有无飞蚊"分家。眼睑局部红肿结节（麦粒肿/泪囊炎）见 [[完整笔记/专题笔记/眼科/眼科_眼睑与泪器炎症]]；OCS vs 急性闭角青光眼的"眼球硬"鉴别见 [[完整笔记/专题笔记/眼科/眼科_青光眼]] §8.3。知识源 [[完整笔记/Peixuan分科笔记/眼科]]（只读）。

> [!note]- 缩写对照（全称 中英）
> - **OCS** = orbital compartment syndrome 眶间隔综合征
> - **EOM** = extraocular movement 眼外肌运动
> - **RAPD** = relative afferent pupillary defect 相对性传入性瞳孔障碍
> - **VA** = visual acuity 视力
> - **CT** = computed tomography 计算机断层扫描
> - **TPN** = total parenteral nutrition 全肠外营养
> - **MRSA** = methicillin-resistant *S. aureus* 耐甲氧西林金黄色葡萄球菌
> - **IV** = intravenous 静脉

## 一、一图定位：突眼 / 视力威胁往哪想

```
眼球周围红肿 / 突眼 / 视力威胁
  │
  ├─ 眼睑红肿 + 发热
  │     ├─ 无突眼、无 EOM 痛、视力正常 ── Preseptal 眶隔前蜂窝织炎（口服抗生素）
  │     └─ 突眼 + EOM 痛/受限 + 视力↓ ── Orbital 眶蜂窝织炎（IV 抗生素，急症；多继发筛窦炎）
  │
  ├─ 外伤后突眼 + 眼睑硬如石 + 视力↓ + RAPD ── OCS 眶间隔综合征（立即 canthotomy，别等 CT）
  │
  └─ 住院/术后/静脉导管 + 飞蚊 + 进行性视力↓（痛可无）── Endophthalmitis 眼内炎
```

> [!tip] 一句话钥匙
> **突眼 + 鼻窦炎 = 眶蜂窝织炎（感染）；突眼 + 外伤 + 眼睑硬如石 + RAPD = OCS（外伤，别等影像）；飞蚊 + 术后/静脉导管 = 眼内炎。**

## 二、眶隔前 vs 眶蜂窝织炎（感染性突眼）

> [!danger] 分水岭 = 眶隔 (orbital septum) 前还是后
> **眶隔**是一层从眶缘连到睑板的纤维膜，把眼睑（前）与眶内容（后）隔开。感染在**隔前**（眼睑软组织）= preseptal；突破到**隔后**（眶内脂肪/肌肉/视神经）= orbital。**突眼 + 眼球运动痛 + 视力改变 = 隔后 = orbital**（隔前绝无这三样）。

| 维度 | **Preseptal 眶隔前蜂窝织炎** | **Orbital 眶蜂窝织炎（急症）** |
|---|---|---|
| 解剖 | 眶隔**之前**（眼睑软组织） | 眶隔**之后**（眶内） |
| 突眼 proptosis | ❌ 无 | ✅ **有** |
| 眼球运动 | 正常、无痛 | **疼痛 + 受限 (ophthalmoplegia 眼肌麻痹)** |
| 视力 | 正常 | **可↓ / 色觉↓ / RAPD**（视神经受累） |
| 常见诱因 | 皮肤破损、虫咬、睑腺炎、**泪囊炎**、局部外伤 | **★细菌性鼻窦炎（尤其筛窦 ethmoid）邻近蔓延★**（最常见）；也可眶外伤、牙源 |
| 病原 | 皮肤菌群（*S. aureus* 金葡菌、链球菌） | 链球菌 *Streptococcus*、*S. aureus*（含 **MRSA**）、*H. influenzae* 流感嗜血杆菌 |
| 影像 | 一般不需 | **眶 + 鼻窦 CT**（找可引流的脓肿 orbital abscess） |
| 治疗 | **口服抗生素** | **IV 抗生素（vancomycin 万古霉素 + ceftriaxone 头孢曲松）** + 脓肿引流 + 眼科/耳鼻喉会诊 |

> [!warning] 眶蜂窝织炎的并发症（为什么算急症）
> 眶内感染可向后扩散 → **海绵窦血栓 (cavernous sinus thrombosis)、颅内脓肿/脑膜炎、视神经受压致失明**。所以突眼 + 眼肌麻痹 + 视力改变要**立即 IV 抗生素 + 影像**，不能当普通眼睑感染口服打发。

> [!danger] Item 19 (Q8784) 做对复盘：儿童突眼 + 眼肌麻痹 + 鼻窦炎史 → orbital cellulitis，诱因 = 细菌性鼻窦炎
> 5 岁男孩，右眼痛 + **视物模糊** + **结膜充血 + 突眼 (proptosis) + 眶周红肿** + **眼球外展受限 + 眼球运动痛 (ophthalmoplegia)** + 发热 39.4℃ + **反复鼻窦炎史 + 右颊压痛** → **orbital cellulitis（眶蜂窝织炎）**，最可能诱因 = **bacterial sinusitis（细菌性鼻窦炎，筛/上颌窦邻近蔓延）**。
> - ❌ **Atopic dermatitis / Insect bite（选 A/D）**：皮肤破损/虫咬 → **preseptal（眶隔前）**，不引起突眼/眼肌麻痹/视力改变 → 不符本例。
> - ❌ **Dental abscess（选 C）**：**上颌**牙（近鼻窦/眶）可诱发眶蜂窝织炎，但本例龋齿在**对侧下颌**（离眶远）；且鼻窦炎是更常见诱因 → 排。
> - ❌ **Otitis media（选 E）**：中耳与眶**不相邻**，中耳炎并发是乳突炎/脑膜炎，不是眶蜂窝织炎；且本例无活动性中耳炎。
> - 🔑 钥匙：**"儿童突眼 + 眼肌麻痹/运动痛 + 视力改变 = orbital cellulitis；头号诱因 = 邻近的细菌性鼻窦炎（筛窦）"**；preseptal 无这三联。

![[{0576767D-67B8-4C55-9811-F9FAED765D69}.png]]

*Preseptal cellulitis 眶隔前蜂窝织炎：眼睑红肿 (lid swelling & erythema)，无突眼、无眼球运动痛（对照 orbital 有突眼 + 眼肌麻痹 + 视力改变）*

## 三、眶间隔综合征 OCS（外伤性突眼，别等影像）

> [!danger] 机制：眶内压骤升 → 视神经/眼球缺血
> **Orbital compartment syndrome（OCS，眶间隔综合征）**：眶是**封闭骨腔**，外伤后**眶内出血/水肿** → 眶内压急升 → 压迫视神经 + 眼球灌注↓ → **缺血 → 永久失明（数小时内）**。**眼外伤是最常见诱因**；也见于凝血障碍、眶感染、眶手术后。

| 维度 | 表现 |
|---|---|
| 症状 | **急性眼痛 + 视力↓ + 复视 (diplopia)** |
| 体征 | **眶张力高"紧眶 (tight orbit)"**：眼睑硬如石 (rock-hard)、**抵抗眼球回推 (resistance to retropulsion)**；**突眼 (proptosis)**；**RAPD**；弥漫**结膜下出血 (subconjunctival hemorrhage)**；EOM 受限 |
| 诊断 | **临床诊断**（不靠影像确诊） |
| 处理 | **急诊！立即外眦切开 + 松解 (lateral canthotomy + cantholysis) 减压**——**在任何影像 (CT) / 测眼压 / 扩瞳检查之前** |

![[{F2E18EC5-C717-4FCA-A2B5-B02E3104BFEF}.png]]

*OCS 眶减压：lateral canthotomy（外眦切开）+ cantholysis（外眦韧带松解，切断外眦韧带下支）——眶张力高时的急诊减压操作*

> [!danger] Item 10 (Q19040) 做对复盘：paintball 外伤 + 突眼 + 眼睑硬如石 + RAPD → OCS → 立即眶减压
> 16 岁男孩，**paintball 打中面部**约 1 小时，眼痛 + 视物困难 + **左眼睑硬如石 (rock-hard induration) + 突眼 + 眶周瘀斑水肿** + **左眼 VA 20/100 + 左眼 RAPD** → **orbital compartment syndrome (OCS)** → **紧急眶减压（lateral canthotomy/cantholysis），先于影像**。
> - ❌ **Cool compress + ibuprofen（选 A）**：仅用于**孤立眶周血肿 (isolated periorbital hematoma)**（无视力↓/RAPD/突眼）；本例有视神经受累征象 → 不够。
> - ❌ **CT scan of face（选 B）**：**减压不能等影像**——延迟 = 增加不可逆视力损失；CT 是减压之后的事。
> - ❌ **Dilated fundoscopy（选 C）/ Serial tonometry（选 E）**：详细眼科评估/连续测眼压**不应先于**急诊减压（延迟致残率↑；OCS 减压后眼压自会回落）。
> - 🔑 钥匙：**"外伤 + 紧眶（硬如石/抵抗回推）+ 突眼 + RAPD/视力↓ = OCS = 立即 canthotomy/cantholysis，别等 CT"**。

## 四、眶蜂窝织炎 vs OCS（都突眼，感染 vs 外伤）

> [!tip] 两者都可突眼 + 视力威胁，靠"背景 + 眼睑质地 + 全身征"分家
> 一个是**感染**（慢起、发热、鼻窦炎史），一个是**外伤**（急起、眼睑硬如石、结膜下出血）。处理完全不同：抗生素 vs 立即手术减压。

| 维度 | **Orbital cellulitis 眶蜂窝织炎** | **OCS 眶间隔综合征** |
|---|---|---|
| 背景 | **感染**（鼻窦炎、牙源、局部） | **外伤**（钝伤/paintball；或凝血障碍/术后） |
| 起病 | 亚急（数小时~天）+ **发热** | **急性（外伤后数十分~小时）**，无发热 |
| 眼睑 | 红肿热 | **硬如石 + 抵抗眼球回推（紧眶）** |
| 特征体征 | 眼球运动痛、鼻窦炎征 | **结膜下出血、瘀斑** |
| 影像 | **需要**（眶/鼻窦 CT 找脓肿） | **不等影像**，先减压 |
| 处理 | **IV 抗生素**（vancomycin + ceftriaxone）+ 引流 | **立即 lateral canthotomy/cantholysis** |

## 五、眼内炎 endophthalmitis（眼内感染）

> [!info] 定义
> **Endophthalmitis（眼内炎）= 眼球内腔（玻璃体/房水）的感染**，视力威胁急症。两条主线：**血源性真菌（Candida）** vs **术后/注射后细菌**。

### 5.1 Candida endophthalmitis 念珠菌眼内炎（血源播散）

> [!warning] 机制 + 高危背景
> **一过性/持续念珠菌血症 (candidemia) → 血行播散到脉络膜 → 侵入视网膜/玻璃体腔。** 高危：**中心静脉导管 (indwelling central catheter)、TPN（全肠外营养）、广谱抗生素、腹部/胃肠手术、免疫抑制**；多见于**住院患者或近期眼手术/外伤**。

| 维度 | 表现 |
|---|---|
| 症状 | **单眼飞蚊 (floaters) + 进行性视力↓**；**痛少见（直到晚期）**；**可无发热、患者看似无恙** |
| 诊断 | **血培养（常阴性）** + **玻璃体取样培养 (vitreous sampling)** |
| 治疗 | **系统性抗真菌** + **玻璃体切除 (vitrectomy) + 玻璃体内注药**（累及玻璃体腔时） |

### 5.2 术后/注射后细菌性眼内炎（对照）

> [!danger] 急性剧烈，与念珠菌的"隐匿"相反
> **白内障术后数天 / 玻璃体内注射后**突发**眼痛 + 红 + 前房积脓 (hypopyon) + 视力快速↓** → **急性细菌性眼内炎** → **玻璃体内抗生素 (intravitreal antibiotics)** ± 玻切。**与 Candida（隐匿飞蚊、痛少、血源）对比**：细菌性是**急性、痛、术后/注射后**。

## 六、经典 stem → 处理反射

- 儿童**鼻窦炎史** + 突眼 + **眼球运动痛/受限 + 视力改变** + 发热 → **orbital cellulitis** → 眶/鼻窦 CT + **IV vancomycin + ceftriaxone** + 引流；诱因问 **细菌性鼻窦炎（筛窦）**。
- 仅眼睑红肿 + 发热，**无突眼/无 EOM 痛/视力正常** → **preseptal cellulitis** → 口服抗生素。
- **面部钝伤/paintball** 后突眼 + **眼睑硬如石 + RAPD + 视力↓** → **OCS** → **立即 lateral canthotomy/cantholysis，别等 CT**。
- **住院 + 中心静脉导管/TPN** + 单眼飞蚊 + 视力↓（不痛、无热）→ **Candida endophthalmitis** → 系统抗真菌 + 玻切/玻璃体内注药。
- **白内障术后 1 周**突发痛/红/脓/视力↓ → **术后细菌性眼内炎** → 玻璃体内抗生素。

## 七、自测（答完再判分）

> [!question]- Q1. 5 岁男孩右眼痛 + 视物模糊 + 突眼 + 眼球运动痛/受限，发热，既往反复鼻窦炎。诊断、最可能诱因、处理？如何与眶隔前区分？
> **Orbital cellulitis（眶蜂窝织炎）**；诱因 = **细菌性鼻窦炎（筛/上颌窦邻近蔓延）**；处理 = 眶/鼻窦 CT + **IV vancomycin + ceftriaxone** + 引流脓肿。与 **preseptal** 区分：眶隔前**无突眼、无眼球运动痛、视力正常**。

> [!question]- Q2. 16 岁 paintball 击中面部后眼痛、视力 20/100、眼睑硬如石、突眼、RAPD。诊断与**下一步**？为什么不先做 CT？
> **Orbital compartment syndrome (OCS)** → **立即 lateral canthotomy + cantholysis 眶减压**。不先 CT：减压不能延迟（缺血数小时致永久失明），影像/测眼压/扩瞳都排在减压之后。

> [!question]- Q3. 眶蜂窝织炎与 OCS 都可突眼 + 视力威胁，如何一眼区分？
> **背景 + 眼睑 + 全身征**：眶蜂窝织炎 = **感染（鼻窦炎史、发热、眼球运动痛）**；OCS = **外伤 + 眼睑硬如石/抵抗回推 + 结膜下出血、无热**。处理：抗生素 vs 立即手术减压。

> [!question]- Q4. 住院患者有中心静脉导管 + TPN，出现单眼飞蚊 + 进行性视力↓，无明显疼痛、无发热。最可能诊断 + 关键检查 + 治疗？
> **Candida endophthalmitis（念珠菌眼内炎，血源播散）**；查**血培养（常阴）+ 玻璃体取样培养**；治 **系统性抗真菌 + 玻璃体切除/玻璃体内注药**。

# 结膜炎 & 新生儿眼分泌物鉴别（机制 + 时间窗 + 干扰项深挖）

> [!info] 用途与定位
> 本篇是**"红眼流泪 / 眼分泌物"前眼段感染的鉴别下沉专题**。考前秒查用 [[完整笔记/专题笔记/眼科/眼科_HighYield速查总览]] 的"四、感染/角膜/眼睑"+"五、新生儿眼病时间窗"；此处放**为什么**——成人结膜炎三分的钩子、新生儿结膜炎按日龄的病原-治疗对应、以及最容易被混的**"婴儿眼分泌物"四选一（结膜炎 vs 鼻泪管阻塞 vs 泪囊炎 vs 角膜擦伤）**。知识源 [[完整笔记/Peixuan分科笔记/眼科]]（只读）。眼睑/泪囊化脓性炎症见姊妹篇 [[完整笔记/专题笔记/眼科/眼科_眼睑与泪器炎症]]；角膜溃疡见 [[完整笔记/专题笔记/眼科/眼科_感染性角膜炎]]。

> [!note]- 缩写对照（全称 中英）
> - **NLD** = nasolacrimal duct 鼻泪管
> - **GC** = *Neisseria gonorrhoeae* 淋球菌
> - **HSV** = herpes simplex virus 单纯疱疹病毒
> - **URI** = upper respiratory infection 上呼吸道感染
> - **IHPS** = infantile hypertrophic pyloric stenosis 婴儿肥厚性幽门狭窄
> - **IM/IV** = intramuscular/intravenous 肌注/静脉
> - **SAFE** = Surgery-Antibiotics-Facial cleanliness-Environment（WHO 沙眼防治策略）

## 一、成人急性结膜炎：病毒 vs 细菌 vs 过敏

> [!tip] 一句话钥匙
> **分泌物性状 + 单双眼 + 痒不痒 + 前驱史** 四件套定三分：**水样 + 前驱 URI + 接触史 = 病毒；脓厚 + 晨起粘睫 = 细菌；双眼 + 痒 + 季节 = 过敏。**

| 类型 | 常见病因 | 分泌物 | 特征钩子 | 治疗 |
|---|---|---|---|---|
| **病毒性** | **腺病毒 adenovirus**（最常见） | **水样 watery** | 单眼起→常波及双眼、**滤泡性睑结膜炎 (follicular)**、异物感/gritty、**前驱 URI**（rhinorrhea/咽痛/发热）、幼儿园/学校暴发、极具传染性 | **支持** = 冷敷 (cool compress) + 人工泪液；自限约 2 周；勤洗手防传播 |
| **细菌性** | S. aureus、S. pneumoniae；隐形眼镜 → **Pseudomonas** | **脓性 purulent、黏厚** | 晨起**睫毛粘连 matting**、单侧多、乳头状 (papillary) 反应 | topical **erythromycin / polymyxin-trimethoprim**；隐形眼镜佩戴者 → **fluoroquinolone** |
| **过敏性** | 空气过敏原（花粉、尘螨、宠物皮屑） | **水样 + 拉丝黏液** | **双侧 + 显著瘙痒 (pruritus) + 结膜水肿**、季节性/过敏原暴露 | topical **抗组胺 + 肥大细胞稳定剂**（olopatadine 奥洛他定 / ketotifen 酮替芬） |

> [!danger] Item 7 (Q8925) 陷阱详解：为什么是"支持治疗"，不是抗病毒/抗过敏/激素
> 28 岁幼儿园老师，**前驱 rhinorrhea + 咽痛 + 低热**，右眼 gritty + 水样分泌物 + **滤泡性睑结膜炎** + 轻度充血 → 典型**病毒性结膜炎** → **cool moist compress（支持）**。
> - ❌ **Oral valacyclovir**（选 B）：治 **HSV 角膜炎**（树枝状溃疡 + 疼痛 + 视力↓ + 畏光），本例无这些 → 见 [[完整笔记/专题笔记/眼科/眼科_感染性角膜炎]]。
> - ❌ **Topical olopatadine**（选 C，抗组胺）：治**过敏性**——但过敏是**双侧 + 明显痒**，本例单眼 + 无瘙痒 + 有全身前驱 → 不符。
> - ❌ **Topical polymyxin-trimethoprim**（选 D，抗菌）：细菌性是**厚脓 + 睫毛粘**，且细菌一般**不呈滤泡性**（乳头状），本例水样 + 滤泡 → 不符。
> - ❌ **Topical prednisolone**（选 E，激素）：病毒性结膜炎**不常规上激素**（角膜瘢痕风险、可能加重疱疹）；激素是前葡萄膜炎等的用药。
> - 🔑 **前驱 URI + 学校接触 + 水样 + 滤泡性** = 病毒的四联签名；本题"history of herpes labialis（唇疱疹）"是**干扰**，唇疱疹 ≠ 眼 HSV，别被带去选 valacyclovir。

> [!info] 病毒性结膜炎的红旗（何时不是"支持"就够）
> 若出现**剧痛 / 视力下降 / 畏光 / 角膜受累（keratoconjunctivitis）** → 警惕**腺病毒角膜结膜炎 (EKC)** 或误诊 HSV，需眼科评估。单纯结膜炎无这些红旗才走支持治疗。

## 二、新生儿结膜炎 ophthalmia neonatorum（按日龄时间窗）

> [!warning] 核心逻辑：日龄 = 病原 = 治疗（★必背时间轴★）
> 新生儿结膜炎按**症状出现的日龄**分层，日龄直接锁病原、病原直接锁治疗。分泌物"脓 vs 水/黏"+ 眼睑水肿程度是辅助钩子。

```
出生 ──┬── <24h ───────── Chemical（红霉素预防的化学刺激）  → 人工泪液，24h 自限
       │
       ├── 第 2–5 天 ───── Gonococcal 淋球菌（GC）           → IM/IV ceftriaxone（或 cefotaxime）
       │     大量脓 profuse purulent + 显著充血 + 眼睑水肿      单剂；角膜穿孔急症
       │
       ├── 第 5–14 天 ──── Chlamydial 衣原体                  → ★oral erythromycin★（监测 IHPS）
       │     水样/浆血性/黏脓 + 轻度眼睑肿                       topical 单用无效
       │
       └── 出生–2 周 ───── HSV（伴皮肤/头皮疱疹、角膜炎）       → IV acyclovir
```

| 日龄 | 病原 | 分泌物 / 特征 | 治疗 | 关键陷阱 |
|---|---|---|---|---|
| **< 24 h** | **Chemical** 化学性 | 轻度结膜刺激 + 流泪 | **人工泪液**，24h 内自限 | 是**局部红霉素预防**本身的一过性刺激 |
| **2–5 天** | **Gonococcal 淋球菌** | **大量脓 hyperacute purulent** + 显著结膜充血 + **眼睑水肿** | **IM/IV ceftriaxone**（高胆红素婴儿改 **cefotaxime**）单剂 | 可致**角膜穿孔/失明**；播散风险 |
| **5–14 天** | **Chlamydial 衣原体** | **水样 / 浆液血性 / 黏脓** + 轻度眼睑肿 | **oral erythromycin**（口服） | ⚠️ 见下方 danger |
| **出生–2 周** | **HSV** | 结膜炎 + **皮肤/头皮疱疹**、可累及角膜 | **IV acyclovir** | 找皮肤疱疹线索；防播散性 HSV |

> [!danger] Item 3 (Q3758) 陷阱详解：13 天龄衣原体 → 为什么是 oral erythromycin，不是 IM cefotaxime / topical
> 13 天龄 + 双眼**轻度眼睑肿 + 充血 + 少量水样/黏脓** + 已做常规 topical 红霉素预防 → 落在 **5–14 天窗 → Chlamydial** → **oral erythromycin**。
> - ❌ **IM cefotaxime**（选 B，你选的）：这是 **gonococcal** 的治疗——但淋球菌是 **2–5 天 + 大量脓 profuse purulent + 显著充血/眼睑水肿**；本例分泌物**少 (scant)**、日龄**偏晚** → 不是 GC。**"分泌物量少 + 日龄 5–14 天"就把 GC 排掉了。**
> - ❌ **Artificial tears**（选 A）：那是 **chemical**（<24h、给药后 24h 内自限），13 天龄 + 感染征象 → 不符。
> - ❌ **Topical erythromycin**（选 E）：**局部单用高失败率**——衣原体会**定植鼻咽 (nasopharynx)**，topical 治不到鼻咽 → 再感染 + 漏防**衣原体肺炎**；必须**系统性口服**根除。
> - ❌ **Nasolacrimal massage**（选 C）：那是**鼻泪管阻塞**（结膜清亮、无充血）→ 见 §三。
> - 🔑 记忆：**"衣原体 = 口服红霉素 + 盯幽门狭窄 (IHPS)"**。口服大环内酯是婴儿 **IHPS** 的已知副作用，UW 爱在这里追问监测项。
> - 📌 **预防**：孕期 maternal screening（<25 岁或有风险）+ 产前治疗；**常规 topical 红霉素预防只防淋球菌，不防衣原体**（所以本例做了预防仍得衣原体结膜炎）。

> [!warning] 治疗药名澄清：oral erythromycin vs azithromycin（避免记混）
> - **新生儿衣原体结膜炎首选 = oral erythromycin ×14 天**（UW/CDC 经典答案，本题正确项）；azithromycin 是**可接受替代**。
> - **沙眼 (trachoma) 首选 = 单剂 oral azithromycin**（见 §四）——同是衣原体，但**成人/儿童沙眼**用 azithromycin，**新生儿结膜炎**默认 erythromycin。
> - 速查总览"五、新生儿眼病"表把第 2 周衣原体简写为 azithromycin 是**浓缩**；做**新生儿这道题**时以 **oral erythromycin** 为标准答案。

## 三、婴儿"眼分泌物"四选一大鉴别（★最爱考★）

> [!danger] 同是"婴儿眼睛有分泌物/结痂"，四个诊断靠"结膜红不红 + 脓不脓 + 泪囊红肿否 + 荧光素染色"分开
> 这一族是新生儿/婴儿眼科的高频陷阱区，Item 3、Item 11、以及泪囊炎 Item 30 全在这张表里。

| 诊断 | 起病/日龄 | 结膜 | 分泌物 | 关键鉴别体征 | 处理 |
|---|---|---|---|---|---|
| **NLD 阻塞**（congenital dacryostenosis 鼻泪管狭窄） | 生后数周、慢性/间歇 | **清亮 clear、不充血** | 溢泪 + 黏液 + **睫毛结痂 matting** | 无红肿；荧光素**均匀分布**（无角膜染色缺损） | **泪囊按摩**（向下压）+ 安慰；6–12 月自愈，持续 → **probing 探通** |
| **结膜炎**（GC/chlamydial/viral） | 按日龄时间窗（§二） | **充血 injected** | 脓（GC）/ 水样黏脓（衣原体/病毒） | 结膜红 + 分泌物性状定病原 | 按病原（§二） |
| **Dacryocystitis 泪囊炎** | NLD 阻塞继发感染 | 多正常 | punctum 可挤出脓 | **内眦下方红肿热痛**（泪囊区）± 发热 | **系统性抗生素**（抗 G+）→ 见 [[完整笔记/专题笔记/眼科/眼科_眼睑与泪器炎症]] |
| **Corneal abrasion 角膜擦伤** | 急性 | 可红 + 流泪 | 反射性流泪 | **荧光素局限染色缺损** + 畏光/激惹 | 抗生素眼膏 ± 随访 |

> [!danger] Item 11 (Q17225) 陷阱详解：2 周龄"左眼晨起粘住" → 为什么是 NLD 阻塞，不是泪囊炎
> 2 周龄 + 数日来晨起左眼睑**结痂粘住** + 每 2 小时清理易擦掉 + **结膜双侧清亮 (clear conjunctivae)** + 左脸/下睑几条**线状抓痕 (self-inflicted abrasions)** + **荧光素在角膜和结膜均匀分布** → **congenital NLD obstruction**。
> - ❌ **Dacryocystitis**（选 B，你选的）：是 NLD 阻塞的**感染并发症**，表现为**泪囊区（内眦下）红肿热痛 + punctum 脓**；本例**无红肿、无脓、结膜清亮** → 还没到感染那步。**"清亮结膜 + 无泪囊红肿"就排除 dacryocystitis。**
> - ❌ **Corneal abrasion**（选 A）：荧光素会呈**局限性染色缺损**；本例**均匀分布**（无缺损）+ 那几条抓痕是脸上皮肤划痕不是角膜 → 排除。
> - ❌ **Gonococcal conjunctivitis**（选 C）：2–5 天 + **大量脓** + 结膜充血；本例清亮 + 做过 newborn 预防 → 不符。
> - ❌ **Viral conjunctivitis**（选 E）：结膜会**红**、常伴病毒前驱；本例结膜清亮（"mild nasal congestion"是新生儿鼻道窄的正常现象，不是眼病依据）→ 不符。
> - 🔑 记忆：**"结膜清亮 + 溢泪结痂 + 荧光素均匀 = NLD 阻塞（按摩 + 等自愈）；泪囊区一旦红肿热痛 = 升级为 dacryocystitis（全身抗生素）"**。
> - 🌊 **NLD 阻塞并发症链**：dacryocystocele（泪囊膨出）→ dacryocystitis（泪囊炎）→ 少数 periorbital/orbital cellulitis。

![[{9E56036F-30F1-46B7-9C35-1CABEA5D5EDD}.png]]

## 四、Trachoma 沙眼（慢性衣原体，致盲首位）

> [!info] 病原与流行
> **Chlamydia trachomatis 血清型 A/B/C**（≠ 生殖道感染的 D–K 型）。**全球感染性致盲第一位**，流行于低资源、拥挤地区（东南亚、非洲、中东）；经**眼/鼻分泌物人–人接触、污染物 (fomites)、蝇媒**传播。反复感染是致盲关键。

```
活动期（active，儿童多）
  上睑结膜【苍白滤泡 pale follicles】+ 炎症性增厚、轻度红/痒/分泌物
  ↓ 反复感染 × 多年
瘢痕期（cicatricial）
  睑结膜瘢痕 → 眼睑内翻 + 倒睫 trichiasis（睫毛内倒）
  ↓ 睫毛长期摩擦角膜
角膜溃疡 → 角膜混浊 opacity → 失明
```

> [!success] 诊断与治疗
> - **诊断 = 临床**（翻上睑看睑结膜滤泡 + 炎症）。
> - **治疗 = 单剂 oral azithromycin**，且**患者 + 所有密切接触者一起治**（限制流行区传播）；WHO **SAFE 策略**（手术矫正倒睫 - 抗生素 - 洗脸 - 改善环境）。

> [!danger] Item 27 (Q2857) 陷阱详解：东南亚移民幼儿双眼滤泡 → 为什么是 trachoma，不是腺病毒
> 4 岁 + 家庭**东南亚移民 2 天前** + 2 周前起**双侧轻度红 + 少量黏脓** + **无咳嗽/鼻涕** + **1 年前有类似自愈史** + 睑结膜见**苍白滤泡 + 炎症** + 视力 20/20 → **trachoma**。
> - ❌ **Adenoviral keratoconjunctivitis**（选 A，你选的）：腺病毒通常**有前驱 URI**（本例明确无 cough/rhinorrhea）、**无反复发作史**、**无流行区暴露**；且 keratoconjunctivitis 累及角膜 → **疼痛 + 畏光**（本例无）。**"无 URI 前驱 + 反复史 + 流行区来源"三点指向沙眼而非腺病毒。**
> - ❌ **Gonococcal conjunctivitis**（选 B）：主要是**新生儿期 (2–5 天)** + 大量脓；出生后期罕见。
> - ❌ **HSV keratitis**（选 C）：角膜树枝状溃疡 + 疼痛畏光，无（本例角膜/视力正常）。
> - ❌ **Orbital cellulitis**（选 D）：单侧 + 发热 + 突眼 + 眼球运动痛/受限，无（本例双侧、无全身毒性）。
> - 🔑 记忆：**"流行区移民 + 反复滤泡性睑结膜炎 + 无 URI = 沙眼 → 全员 azithromycin"**。滤泡在**上睑结膜**、反复、地理线索是三把钥匙。

## 五、缝合钩子：滤泡性结膜炎 (follicular) 谁会有

> [!tip] 为什么 Item 7（病毒）和 Item 27（沙眼）都出现"follicular / pale follicles"
> **滤泡 (follicle) = 睑结膜下淋巴细胞聚集**，是**病毒**与**衣原体**的共同特征；**细菌**多为**乳头状 (papillary) + 脓**，**过敏**为乳头 + 痒。所以"follicular appearance"把答案框在**病毒 vs 衣原体**，再靠**前驱史/流行区/日龄**分手。

| 反应形态 | 谁会有 | 提示 |
|---|---|---|
| **滤泡性 follicular** | **病毒（腺病毒）、衣原体（衣原体结膜炎 / 沙眼）** | 淋巴聚集；配前驱 URI→病毒、流行区反复→沙眼 |
| **乳头状 papillary** | 细菌、过敏、隐形眼镜相关 | 细菌配脓、过敏配痒 |

> [!warning] 三个"衣原体眼病"别混
> 同是 *C. trachomatis*，靠**血清型 + 人群 + 治疗**分三种：
> - **新生儿衣原体结膜炎**（D–K 型，母产道获得，5–14 天）→ **oral erythromycin** + 监测 IHPS。
> - **成人包涵体结膜炎 (inclusion conjunctivitis)**（D–K 型，性传播眼病，慢性滤泡）→ oral azithromycin/doxycycline。
> - **沙眼 trachoma**（A–C 型，流行区反复，致盲）→ 单剂 oral azithromycin + 治接触者。

## 六、自测（答完再判分）

> [!question]- Q1. 13 天龄男婴，双眼轻度眼睑肿 + 结膜充血 + 少量水样/黏脓分泌物，已做常规新生儿红霉素预防。最恰当治疗？为什么不是 IM cefotaxime？
> **Chlamydial 结膜炎（5–14 天窗）→ oral erythromycin**（监测婴儿肥厚性幽门狭窄 IHPS）。不选 IM cefotaxime 因为那是**淋球菌**（2–5 天 + **大量脓** + 显著充血/眼睑水肿）；本例分泌物少 + 日龄偏晚 → 非 GC。topical 单用无效（鼻咽定植再感染）。

> [!question]- Q2. 2 周龄婴儿晨起单眼睫毛结痂粘住，结膜清亮无充血，荧光素在角膜结膜均匀分布。诊断与处理？如何与泪囊炎区分？
> **先天性鼻泪管阻塞 (NLD obstruction / dacryostenosis)** → **泪囊按摩 + 安慰**，6–12 月多自愈。与 **dacryocystitis** 区分：泪囊炎有**内眦下泪囊区红肿热痛 + punctum 脓**，本例无红肿、结膜清亮 → 还没感染。

> [!question]- Q3. 28 岁幼儿园老师，前驱鼻涕咽痛低热，单眼水样分泌物 + 滤泡性睑结膜炎 + gritty。最恰当处理？为什么不上激素/抗病毒？
> **病毒性结膜炎（腺病毒）→ 冷敷 + 人工泪液（支持）**，自限约 2 周 + 勤洗手。不上激素（角膜瘢痕/加重疱疹风险）；不用 valacyclovir（那是 HSV 角膜炎：树枝状 + 痛 + 视力↓）；不用抗组胺（过敏是双侧 + 痒）。

> [!question]- Q4. 4 岁东南亚移民幼儿，双眼慢性反复轻度红 + 少量黏脓 + 上睑结膜苍白滤泡，无 URI 前驱。诊断与治疗？为什么不是腺病毒？
> **沙眼 trachoma（C. trachomatis A–C 型）→ 单剂 oral azithromycin + 治所有密切接触者**。不是腺病毒：无 URI 前驱、有反复史、有流行区暴露，且腺病毒 keratoconjunctivitis 会疼痛畏光（本例无）。

> [!question]- Q5. 同为 *C. trachomatis*，新生儿结膜炎与沙眼的首选口服药各是什么？
> **新生儿衣原体结膜炎 = oral erythromycin**（azithromycin 为替代）；**沙眼 = 单剂 oral azithromycin**。别记反。

# 眼科 High-Yield 速查

> [!info] 用途与来源
> 考前快速过眼科核心反射。知识源自 [[完整笔记/Peixuan分科笔记/眼科]]（只读分科笔记），本篇为高频提炼速查版。配合 [[完整笔记/_衍生_鉴别诊断速查]]、[[完整笔记/_衍生_高频陷阱]] 使用。

> [!note]- 缩写对照（全称 中英）
> - **CRAO** = central retinal artery occlusion 视网膜中央动脉阻塞
> - **CRVO** = central retinal vein occlusion 视网膜中央静脉阻塞
> - **AMD** = age-related macular degeneration 年龄相关性黄斑变性
> - **RAPD** = relative afferent pupillary defect 相对性传入性瞳孔障碍
> - **INO** = internuclear ophthalmoplegia 核间性眼肌麻痹
> - **MLF** = medial longitudinal fasciculus 内侧纵束
> - **IOP** = intraocular pressure 眼压
> - **EOM** = extraocular movement 眼外肌运动
> - **VEGF** = vascular endothelial growth factor 血管内皮生长因子
> - **AREDS** = Age-Related Eye Disease Study（黄斑变性补充剂方案）
> - **ICP** = intracranial pressure 颅内压
> - **IOFB** = intraocular foreign body 眼内异物
> - **CN III** = cranial nerve III 第三对颅神经（动眼神经 oculomotor）
> - **HSV** = herpes simplex virus 单纯疱疹病毒
> - **CMV** = cytomegalovirus 巨细胞病毒
> - **TPN** = total parenteral nutrition 全肠外营养
> - **TB** = tuberculosis 结核
> - **MS** = multiple sclerosis 多发性硬化
> - **PCA** = posterior cerebral artery 大脑后动脉
> - **DM** = diabetes mellitus 糖尿病

## 〇、一图定位：眼科思维流程

```
单眼/双眼？     疼痛？        定位
  │              │
  ├ 单眼无痛失明 ─┬─ Cherry-red spot ── CRAO（急症）
  │              ├─ 番茄酱眼底 ──────── CRVO
  │              ├─ 幕布+闪光飞蚊 ───── Retinal detachment（急症）
  │              └─ 直线变波浪 ───────── Wet AMD
  │
  ├ 单眼有痛红眼 ─┬─ 眼球硬如石+固定中度散大瞳 ── 急性闭角青光眼（急症）
  │              ├─ 树枝状溃疡（荧光素）─────── HSV keratitis
  │              ├─ 眼球运动痛+proptosis ───── Orbital cellulitis
  │              └─ ciliary flush+hypopyon ── Anterior uveitis
  │
  └ 双眼/视野 ───┬─ 双颞侧偏盲 ── 垂体瘤（视交叉）
                 └─ 同侧偏盲 ──── 交叉后病变（越后越对称+黄斑回避→枕叶）
```

## 一、青光眼 Glaucoma（最高频）

| | Open-angle 开角型 | **Closed-angle 闭角型（急症）** |
|---|---|---|
| 起病 | 渐进、**无痛** | 突发、**剧痛 + 眼球硬如石** |
| 视野 | **Tunnel vision 周边丢失** | 视物模糊、灯周光晕 (halos) |
| 瞳孔 | 正常 | **固定中度散大 (fixed mid-dilated)** |
| 机制 | 房水**流出↓**（小梁网阻力↑） | 瞳孔阻滞；诱因=暗光、decongestant、抗胆碱药、磺胺 |
| 诊断 | tonometry（IOP↑）、视盘 cupping、cup/disc↑ | **gonioscopy > tonometry** |
| 治疗 | **1st: topical prostaglandin (latanoprost/bimatoprost)** + timolol（哮喘慎用）+ laser trabeculoplasty | **先降压：IV acetazolamide + timolol/pilocarpine/apraclonidine ± mannitol → laser iridotomy 根治** |

![[眼科_急性青光眼外眼.jpeg]]
*正常眼 vs 急性闭角青光眼 (acute angle-closure glaucoma)：角膜水肿混浊 + 固定中度散大瞳 + 结膜充血。*

![[眼科_青光眼杯盘比.png]]
*视盘对比：正常 (cup/disc <0.5) vs 开角青光眼（杯盘比 >0.6、盘缘变薄、视盘苍白=视神经萎缩）。*

> [!danger] 两个"固定瞳孔"别混
> - **Fixed mid-dilated pupil** = 急性闭角青光眼
> - **Fixed dilated pupil + ptosis + down-and-out** = 压迫性 CN III 麻痹（→ 排动脉瘤/钩回疝，until proven otherwise）

> [!warning] 小儿青光眼
> 流泪 / 畏光 / 睑痉挛 + 角膜或眼球增大 + 视盘 cupping + IOP↑。病因含 Sturge-Weber、激素诱导。Tx：手术 ± 降压眼药。

## 二、视网膜三大急症 + AMD

| 疾病 | Buzzword | 表现 | 处理 |
|---|---|---|---|
| **CRAO** 视网膜中央动脉阻塞 | **Cherry-red spot 樱桃红点** | 突发无痛单眼全盲、amaurosis fugax 前驱 | **立即眼科急症** |
| **CRVO** 视网膜中央静脉阻塞 | **"Blood & thunder" 番茄酱眼底** | 无痛视力下降（比动脉慢）、tortuous veins、出血 | Fundoscopy；抗 VEGF |
| **Retinal detachment** 视网膜脱离 | **Curtain 幕布 + flashes & floaters** | 无痛视力丧失、眼底灰白起伏网膜 | **手术（急症）** |
| **Dry AMD** 干性黄斑变性 | **Drusen 玻璃膜疣** | 慢、模糊、中央暗点 | Amsler 网格自测 + **AREDS 维生素/抗氧化剂** |
| **Wet AMD** 湿性黄斑变性 | **Metamorphopsia 直线变波浪** | 快、出血、neovascularization | 荧光造影 + **玻璃体内抗 VEGF (ranibizumab/aflibercept)** |

> [!tip] AMD 最早信号 = 直线扭曲 → 用 **Amsler grid** 自测；干性每日自测监测"干转湿"，一变波浪立即抗 VEGF。

![[眼科_CRVO番茄酱眼底.png]]
*CRVO 眼底："blood and thunder / 番茄酱"——静脉迂曲扩张 + 弥漫火焰状出血 + 棉绒斑 + 视盘水肿。*

**Diabetic retinopathy 糖网**：Non-prolif = microaneurysm + cotton-wool + flame hemorrhage + **hard exudates**；Prolif = neovascularization。
**Hypertensive retinopathy 高血压网**（鉴别）：**AV nicking + copper/silver wiring** + flame hemorrhage + cotton-wool。

![[眼科_非增殖性糖网NPDR.png]]
*非增殖性糖网 (NPDR)：点状出血 (hemorrhage) + 微动脉瘤 (microaneurysm) + 硬性渗出 (hard exudate)。*

![[眼科_增殖性糖网PDR.png]]
*增殖性糖网 (PDR)：视网膜新生血管 (neovascularization)——是 anti-VEGF/激光的指征。*

![[眼科_高血压视网膜病变.png]]
*高血压视网膜病变：动静脉压迹 (AV nicking) + 小动脉变窄 + 棉绒斑 + 硬性渗出。*

## 三、视野缺损定位（神经眼科）

> [!info] 两条铁律
> ① **交叉前 → 单眼**；② **交叉后 → 双眼同侧 (homonymous)，越往后越对称 (congruous) + 黄斑回避**。

```
视神经(交叉前) ── 单眼全盲/暗点
视交叉(chiasm) ── 双颞侧偏盲 ★垂体瘤★
视束(optic tract) ── 对侧同侧偏盲
Meyer's loop/颞叶 ── 对侧上象限盲 "Pie in the sky"
顶叶背侧放射 ── 对侧下象限盲 "Pie on the floor"
枕叶皮层/PCA卒中 ── 同侧偏盲 + 黄斑回避（macular sparing）
```

![[眼科_视野缺损定位.jpeg]]
*视野缺损定位：从黄斑/视神经/视交叉/视束/Meyer 环/背侧放射到视皮层 7 个层级，对应左右眼视野缺损图样。*

![[眼科_视盘水肿对比.png]]
*视盘对比：正常 vs 视盘水肿 (papilledema)——盘缘模糊、生理凹陷消失，提示颅内压增高 (↑ICP)。*

| 瞳孔异常 | 特征 | 关联 |
|---|---|---|
| **Marcus Gunn / RAPD** | swinging flashlight：照患眼双眼反而散大 | optic neuritis(MS)、严重视网膜病、视神经受压 |
| **Argyll-Robertson** | 瞳小不规则、**对光无反应、近反射保留** | **神经梅毒**、DM（伴 tabes dorsalis） |
| **INO** 核间性眼肌麻痹 | 患侧眼不能内收 + 对侧眼外展震颤 | MLF 病变（MS）；"左坏左不收，右看右震颤" |

## 四、感染 / 角膜 / 眼睑

**Keratitis 角膜炎（按溃疡形态秒选）**

| 形态 | 病因 | 治疗 |
|---|---|---|
| 中央圆形溃疡 + 黏脓 | 细菌 | topical 抗生素 |
| **树枝状 dendritic 溃疡** | **HSV** | oral valacyclovir/acyclovir |
| 羽毛状边缘 + 卫星灶 | 真菌 | topical 抗真菌 |
| **隐形眼镜佩戴者** | **Pseudomonas** | fluoroquinolone (moxi/cipro)；24h 随访、**不贴眼罩** |

![[眼科_HSV树枝状角膜炎.png]]
*HSV 角膜炎 (herpes simplex keratitis)：荧光素染色见经典**树枝状溃疡 (dendritic ulcer)** → oral valacyclovir。*

**结膜炎 Conjunctivitis**

| 类型 | 钩子 | 治疗 |
|---|---|---|
| 细菌性 | 单侧、**脓性** | topical erythromycin / polymyxin-trimethoprim |
| 病毒性 | 单侧、**水样、腺病毒**、前驱咽炎 | 支持 |
| 过敏性 | **双侧、水样、痒** | topical antihistamine (olopatadine) |

> [!warning] Orbital vs Preseptal cellulitis（高频鉴别）
> - **Orbital 眶蜂窝织炎**：眼球运动痛 + **proptosis** + 视力改变 + EOM 受限 + 发热（多继发筛窦炎）→ **IV vancomycin + ceftriaxone**
> - **Preseptal 眶隔前**：仅眼睑红肿，无 proptosis / 无 EOM 痛 / 无视力改变 → **口服抗生素**

**麦粒肿 vs 霰粒肿**：Hordeolum = **疼痛**（S. aureus）；Chalazion = **无痛**（肉芽肿，反复发作排 sebaceous carcinoma）。两者皆热敷 + baby shampoo。

## 五、新生儿眼病（按时间窗 ★必背★）

| 时间窗 | 病因 | 治疗 |
|---|---|---|
| **0–24h** | Chemical conjunctivitis | 冲洗 |
| **第 1 周** | **Gonococcal 淋球菌** | **Ceftriaxone** |
| **第 2 周** | **Chlamydial 衣原体** | **Azithromycin** |
| **第 3 周** | **HSV**（伴头皮病灶线索） | **Acyclovir** |

> [!tip] Pearls
> - 最常见**新生儿失明 = Chlamydia**；最常见**先天性失明 = CMV**。
> - 2 周婴儿单侧水样分泌物 + 结膜清亮 → **congenital dacryostenosis 鼻泪管狭窄**，6–12 月自愈，持续 → probing。

## 六、眼外伤秒选表（USMLE 经典）

| Stem 题眼 | 诊断 | 关键处理 |
|---|---|---|
| 视力骤降 + 眼球形态异常 / 瞳孔不规则 / Seidel+ | **Open globe 开放眼球损伤** | **绝不压眼球、不测眼压**、放 rigid shield、急会诊、CT 查 IOFB |
| 外伤后突眼 + 眼压高 + 视力掉 + 眼睑张力高 + RAPD | **Orbital compartment syndrome** | **别等影像，立即 lateral canthotomy/cantholysis** |
| 酸/碱入眼、剧痛畏光 | **Chemical burn** | **先大量冲洗到 pH 正常**，检查排后 |
| 向上看复视 + 上转受限 + 同侧颊/上唇麻 | **Blowout fx + 下直肌嵌顿** | CT 见 orbital floor fracture |
| 钝伤后前房积血 | **Traumatic hyphema** | 排 open globe；eye shield + 抬高床头 + 监测再出血/眼压 |
| 鲜红结膜下片状出血、不痛、视力正常 | **Subconjunctival hemorrhage** | benign，1–2 周自愈 |

> [!danger] 总口诀
> 视力掉想 open globe；眼球突想 compartment；化学伤先冲洗；上视双影想 blowout；前房见血是 hyphema。**"别等影像"三急症：orbital compartment / chemical burn / CRAO。**

![[眼科_blowout下直肌嵌顿CT.png]]
*眶底爆裂骨折 (blowout fracture) 冠状 CT：下直肌 (inferior rectus) + 眶脂肪疝入上颌窦 → 上转受限 + 复视。*

## 七、经典 stem → 诊断反射

- 年轻女 + 单眼视力下降 + **转眼痛** + RAPD + "washed-out"色觉 + 中央暗点 → **Optic neuritis → MS**；MRI orbits/brain；IV 激素
- 抗 TB 治疗中视力↓ + 色觉↓ + 中央暗点 → **Ethambutol 中毒性视神经病**
- 17 岁 + **夜盲** + 中周边视野缺损 + 血管变细/视盘苍白 + bone-spicule 色素 → **Retinitis pigmentosa**
- HIV + 亚急性进行性视力下降 + 眼底黄白颗粒灶沿血管 → **CMV retinitis**

![[眼科_CMV视网膜炎.png]]
*CMV 视网膜炎 (cytomegalovirus retinitis)：黄白色絮状视网膜坏死灶 + 视网膜出血（"番茄奶酪/pizza pie"样），沿血管分布；见于 HIV CD4<50。*

- 疼痛红眼 + ciliary flush 睫状充血 + hypopyon 前房积脓 → **Anterior uveitis 前葡萄膜炎**（关联 HLA-B27 / 结节病 sarcoidosis / 炎症性肠病 IBD）→ 散瞳 (cyclopentolate) + topical 糖皮质激素

![[眼科_前葡萄膜炎.png]]
*前葡萄膜炎 (anterior uveitis)：裂隙灯见睫状充血 (ciliary flush) + 角膜后沉着，伴疼痛畏光。*

- 住院 + 中心静脉导管/TPN + 飞蚊 + 眼底白棉球浸润 → **Candida endophthalmitis**
- 前额单侧痛 + 皮疹 + **Hutchinson sign（鼻尖）** → **Herpes zoster ophthalmicus (V1)** → oral valacyclovir
- 白内障术后 1 周突发痛/红/脓/视力快降 → **术后细菌性 endophthalmitis** → intravitreal antibiotics
- 2 岁斗鸡眼 + **red reflex 不对称** → 下一步 **dilated funduscopy（排 retinoblastoma）**
- 婴幼儿 opsoclonus-myoclonus + 腹部包块 + **periorbital ecchymosis（眶转移）** → **Neuroblastoma**

## 八、检查工具速记

| 检查 | 异常 → 最该想到 |
|---|---|
| RAPD | 视神经病变 / 严重视网膜病 |
| **Red reflex** 白瞳/不对称 | **retinoblastoma、cataract、vitreous hemorrhage** |
| Fluorescein 染色+ | corneal abrasion/ulcer |
| Fundoscopy papilledema | ↑ICP |
| Tonometry IOP↑ | glaucoma |
| **Pinhole test** 改善 / 不改善 | 屈光不正 / 角膜·晶状体·视网膜·视神经结构病变 |

- **Slit lamp = 前段**（角膜/前房/虹膜/晶状体）；**Fundoscopy = 后段**（视网膜/视盘/黄斑/血管）
- 视力筛查：常规 **age 4** 起；4 岁 >20/40、≥5 岁 >20/30 → 转眼科；斜视筛查至 age 5
- **高度近视 ≥6 diopters** → 视网膜变薄 → 脱离风险

## 九、自测（答完再判分）

> [!question]- Q1. 癫痫不直接相关——老年男性突发无痛单眼全盲，眼底见樱桃红点。最可能诊断与首要处理？
> **CRAO**；立即眼科急症处理（眼球按摩/降眼压/查栓子来源），属"别等影像"急症。

> [!question]- Q2. 60 岁女性诉看门框变成波浪状、中央模糊数日。Amsler 网格扭曲。下一步最佳治疗？
> **Wet AMD** → 玻璃体内**抗 VEGF**（ranibizumab/aflibercept）；先荧光造影确认 neovascularization。

> [!question]- Q3. 突发剧烈眼痛、头痛、恶心，看灯有光晕，眼球触诊硬如石，瞳孔中度散大固定。诊断最准确的检查？
> 急性**闭角青光眼**；**gonioscopy**（> tonometry）。先 IV acetazolamide + 降压眼药，再 laser iridotomy。

> [!question]- Q4. 出生后第 6 天双眼大量脓性分泌物。最可能病原与治疗？
> **淋球菌 (Gonococcal)** → **Ceftriaxone**（第 2 周才轮到 Chlamydia → azithromycin）。

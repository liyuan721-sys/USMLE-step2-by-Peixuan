# 眼底病机制总览（AMD × DR × 两轴框架）

> [!info] 用途与来源
> 把眼底/视网膜病按**两条分类轴**缝成一张图，重点讲 **AMD** 与 **Diabetic retinopathy (DR, 糖尿病视网膜病变)** 的机制，以及它们在 **VEGF/新生血管**这个终点的汇合。
> 速查/buzzword 见 [[完整笔记/专题笔记/眼科/眼科_HighYield速查总览]]（本篇为其机制下沉专题）；只读分科笔记 [[完整笔记/Peixuan分科笔记/眼科]]。
> 跨科病因：DM 见 [[完整笔记/Peixuan分科笔记/endocrine]] · [[完整笔记/专题笔记/内分泌/内分泌_HighYield速查总览]]；HTN 见 [[完整笔记/Peixuan分科笔记/心内]] · [[完整笔记/专题笔记/心内/心内_继发性高血压鉴别框架]]。配合 [[完整笔记/_衍生_高频陷阱]]。

> [!note]- 缩写对照（全称 中英）
> - **AMD** = age-related macular degeneration 年龄相关性黄斑变性
> - **DR** = diabetic retinopathy 糖尿病视网膜病变
> - **NPDR** = non-proliferative DR 非增殖性糖网
> - **PDR** = proliferative DR 增殖性糖网
> - **RPE** = retinal pigment epithelium 视网膜色素上皮
> - **CNV** = choroidal neovascularization 脉络膜新生血管
> - **NVD/NVE** = neovascularization of disc/elsewhere 视盘/他处新生血管
> - **VEGF** = vascular endothelial growth factor 血管内皮生长因子
> - **PRP** = panretinal photocoagulation 全视网膜激光光凝
> - **AGEs** = advanced glycation end-products 晚期糖基化终产物
> - **PKC** = protein kinase C 蛋白激酶 C
> - **CRVO** = central retinal vein occlusion 视网膜中央静脉阻塞
> - **CRAO** = central retinal artery occlusion 视网膜中央动脉阻塞
> - **DME** = diabetic macular edema 糖尿病黄斑水肿
> - **RD** = retinal detachment 视网膜脱离
> - **OCT** = optical coherence tomography 光学相干断层扫描
> - **FA** = fluorescein angiography 荧光素血管造影
> - **ESR** = erythrocyte sedimentation rate 红细胞沉降率；**CRP** = C-reactive protein C 反应蛋白
> - **IOP** = intraocular pressure 眼压
> - **TIA** = transient ischemic attack 短暂性脑缺血发作
> - **AREDS** = Age-Related Eye Disease Study（黄斑变性补充剂方案）
> - **CFH** = complement factor H 补体因子 H（AMD 易感基因）
> - **GCA** = giant cell arteritis 巨细胞动脉炎
> - **SLE** = systemic lupus erythematosus 系统性红斑狼疮
> - **HIV** = human immunodeficiency virus 人类免疫缺陷病毒
> - **DM** = diabetes mellitus 糖尿病；**HTN** = hypertension 高血压

## 一、两轴框架：这些眼底病什么关系

> [!tip] 一句话
> **速查总览那张表 = 「今天突然看不见」或「老年黄斑退变」（轴 A）；DR / Hypertensive retinopathy = 「多年慢性全身病泡出来的眼底改变」（轴 B）。** 两者不是同一层并列，是两条正交的分类轴，靠 §六 的 4 个交叉点缝在一起。

```
视网膜 / 眼底疾病
│
├─ 轴 A：原发【眼/视网膜本身】的病（单眼视力主诉；起病分钟→年不等）
│   ├─ 血管急性阻塞 → CRAO / CRVO        （分钟~天，突发/亚急无痛视力↓）
│   ├─ 机械性        → Retinal detachment  （突发，幕布 + 闪光飞蚊）
│   └─ 黄斑退变       → Dry / Wet AMD       （★慢性异类★ 干性年级 / 湿性亚急，中央视力·老年）
│
└─ 轴 B：全身病【DM/HTN】继发的慢性视网膜病变（双眼·筛查发现）  ← DR + HTN retinopathy 在这
    ├─ Diabetic retinopathy（DM 慢性高血糖微血管病）
    │   ├─ NPDR：microaneurysm / dot-blot 出血 / hard exudates / cotton-wool
    │   └─ PDR ：neovascularization 新生血管
    └─ Hypertensive retinopathy（慢性高血压）
        └─ AV nicking / copper-silver wiring / flame hemorrhage / cotton-wool / papilledema
```

> [!warning] 轴 A ≠ 全是急性（AMD 是慢性异类，别被"轴 A"误导）
> 轴 A 的共性**不是"急性"，而是"眼睛自己的病 + 单眼视力主诉"**——起病速度从**分钟级（CRAO 急诊）到年级（干性 AMD 随访）**都有。轴 B 才是清一色慢性：**全身病（DM/HTN）连累眼底**，双眼、随访筛查发现。
> → 真正的分界是 **原发眼病（轴 A）vs 全身病继发（轴 B）** + **单眼主诉 vs 双眼筛查**，不是快慢。

**轴 A 起病速度光谱（同属"原发眼病"，快慢却横跨分钟→年）：**

| 病 | 起病速度 | 性质 |
|---|---|---|
| **CRAO** 视网膜中央动脉阻塞 | **秒~分**，突发无痛全盲 | 真急症（"别等影像"） |
| **Retinal detachment** 视网膜脱离 | **突发**（闪光飞蚊 → 幕布） | 手术急症 |
| **CRVO** 视网膜中央静脉阻塞 | **亚急性**（数小时~数天） | 亚急 |
| **Wet AMD** 湿性黄斑变性 | **亚急性**（数天~数周，直线变波浪） | 慢病急性加重（acute-on-chronic） |
| **Dry AMD** 干性黄斑变性 | **慢**（数月~数年） | 纯慢性 |

> [!tip] 用"起病速度 + 有无痛 + 中央还是全部"三件套定位
> - **突发无痛全盲 / 数秒** → **CRAO**（不是 AMD）
> - **数天~数周单眼中央变形、直线变波浪** → **Wet AMD**
> - **数月~数年单眼中央渐糊、中央暗点** → **Dry AMD**
> AMD 永远是**无痛、中央（黄斑）、慢或亚急**；CRAO/RD 是**无痛、突发、全部或大片**。

## 二、解剖地基：黄斑 ⊂ 视网膜 ‖ 脉络膜（AMD/DR/CRAO 共用底图）

> [!tip] 三者是两种不同关系
> **黄斑 (macula) 和视网膜 (retina) 是「部分 vs 整体」；脉络膜 (choroid) 和视网膜是「上下两层」。** 搞清这张底图，AMD、DR、CRAO 的血供逻辑全通。

```
视网膜 Retina  = 眼球后壁那一整张「感光的神经膜」
    │
    ├─ 黄斑 Macula  = 视网膜里【中央一小块】特化区（部分 ⊂ 整体）
    │       └─ 中心凹 Fovea = 黄斑正中，视锥最密，中心视力最尖
    │
脉络膜 Choroid  = 视网膜【外面/下面】那一层「血管膜」（另一层，供血用）
```

### 黄斑是干嘛的（功能定义了「黄斑病」的症状）

```
黄斑 Macula（视网膜中央特化区）
    │
    ├─ ① 中心视力 Central vision —— 看正前方、看细节（中心凹 Fovea 分辨率最高）
    ├─ ② 高分辨率 High acuity —— 阅读、认人脸、看数字
    ├─ ③ 色觉 Color vision —— 视锥 cones 集中于此
    └─ ④ 明视觉 Photopic —— 视锥在亮光下工作
```

靠什么实现——**视锥 (cones) vs 视杆 (rods) 分工**：

| | **黄斑/中心凹（视锥 cones 为主）** | **周边视网膜（视杆 rods 为主）** |
|---|---|---|
| 管什么 | **中央视力、细节、颜色、白天** | **周边视野、夜视、运动觉** |
| 分辨率 | 极高（近一对一接神经节细胞） | 低（多视杆共用一个神经节细胞） |
| 光照 | 明视觉 photopic | 暗视觉 scotopic |

> [!tip] 一句话
> **黄斑 = 看得「清」（中央 + 细节 + 颜色）；周边视网膜 = 看得「广、暗、动」。** 黄斑坏 → 中央看不清但周边还在（能走路不撞，却读不了书、认不出脸）。

> [!danger] 黄斑受损 = 中央视力丧失、周边保留（对比青光眼相反）
> - **AMD (age-related macular degeneration, 年龄相关性黄斑变性)**：中央暗点 (central scotoma) + 直线变波浪 (metamorphopsia)，**周边视野正常**。
> - **Diabetic macular edema (糖尿病黄斑水肿)**：DR 里**视力下降的主因**（水肿波及黄斑）。
> - **CRAO cherry-red spot**：全视网膜苍白，唯独**中心凹**透出脉络膜的红（中心凹薄、无内层）。
> - 查黄斑功能 = **Amsler grid（Amsler 网格）**，网格线扭曲即提示黄斑病变。
> - 记忆钩：**"黄斑管中央，周边管余光"** —— 黄斑病中央先糊/扭/黑、周边照常；**青光眼 (glaucoma) 反过来**——周边先丢（管状视野 tunnel vision）、中央最后丢。「中央 vs 周边谁先丢」是 UW 区分黄斑病与青光眼的经典钩子。

### 从里到外的层次栈（光从玻璃体侧进）

```
玻璃体 Vitreous（眼内腔）  ← 光从这边进来
┌────────── 视网膜 Retina（神经膜）──────────┐
│ Ganglion cells 神经节细胞   ┐               │
│ Bipolar cells 双极细胞       ├ 内层 inner    │
│ ───────────────────────────  ┘               │
│ Photoreceptors 光感受器      ┐ 外层 outer    │
│ RPE 视网膜色素上皮 ← 视网膜最外层 ┘           │
└─────────────────────────────────────────────┘
  Bruch's membrane Bruch 膜  ← 分界「滤网」
┌─────────────────────────────────────────────┐
│ Choroid 脉络膜（choriocapillaris 脉络膜毛细血管）│
└─────────────────────────────────────────────┘
  Sclera 巩膜（最外壳）
```

![[眼科_眼球矢状解剖三层结构.png]]
*整眼矢状切面：后段三层同心结构 **Retina 视网膜（inner，最内）→ Choroid 脉络膜（middle，中）→ Sclera 巩膜（outer，最外）**；Fovea 中心凹 = 黄斑正中；Central retinal artery/vein 视网膜中央动/静脉从 optic disc 视盘进出（供内层视网膜）。对照上面 ASCII 分层图看「深浅方向」。*

> [!info] RPE 是视网膜的一部分，不是脉络膜的
> RPE (retinal pigment epithelium, 视网膜色素上皮) 属于视网膜**最外层**，坐在 Bruch 膜上、下靠脉络膜——是两层间的中转站。**AMD 就坏在这个 RPE–Bruch–脉络膜界面上。**

### 供血分家（考点核心）

视网膜**内外两层血供来源完全不同**：

| 视网膜层 | 供血来源 | 相关病 |
|---|---|---|
| **内层**（神经节 / 双极细胞） | **Central retinal artery 视网膜中央动脉**（眼动脉分支） | **CRAO / CRVO / DR**（视网膜血管出问题） |
| **外层**（光感受器 + RPE） | **脉络膜 Choroid**（弥散供氧，外层本身无血管） | **AMD**（脉络膜–RPE 界面） |

> [!danger] 一个解剖点串起三个诊断
> - **CRAO (central retinal artery occlusion, 视网膜中央动脉阻塞) → Cherry-red spot 樱桃红点**：中央动脉供血的**内层视网膜**缺血水肿变苍白，而**中心凹本身薄、无内层、直接透出底下红色的脉络膜** → 苍白背景中一点红。**"樱桃红"红的正是脉络膜的颜色。**
> - **Wet AMD → 脉络膜新生血管 (choroidal NV)**：病变来自**脉络膜**，穿 Bruch 膜到视网膜下。
> - **Proliferative DR → 视网膜新生血管 (retinal NV)**：病变来自**视网膜血管**。
> → 「AMD 新生血管来自脉络膜、DR 来自视网膜」，本质就是**外层脉络膜血供 vs 内层视网膜血供**两套系统的区别（见 §五）。

> [!question]- 黄斑为什么特别依赖脉络膜？
> **中心凹 (fovea) 是视网膜最薄处**：它把内层神经细胞往旁边挪开、**没有视网膜血管（foveal avascular zone 黄斑无血管区）**，好让光直达视锥。代价是——中心凹几乎**完全靠脉络膜供养**。所以：① **AMD**（脉络膜/RPE 病）直接砸中央视力；② **CRAO** 时中心凹反而"幸存显红"（不靠中央动脉），才有樱桃红点。

> [!question]- 樱桃红点「透出脉络膜」是什么方向？为什么只有中心凹透得出
> **"底下 / 透出 / 外层"都指「沿检查者视线更深的一层」，不是解剖上下、也不是颞侧**——脉络膜垫在**整张视网膜的后方（靠巩膜侧）**，眼底镜从瞳孔往里看：玻璃体 → 视网膜（浅）→ 脉络膜（深）→ 巩膜。
> 
> 同一层脉络膜哪里都在，可只有中心凹变红，差别在**它前面挡的视网膜透不透明**：
> 
> ```
> 横断面（← 检查者从左看进去）
> 
> 周边黄斑区                    中心凹 Fovea
> 玻璃体 │                      玻璃体 │
>        │ 内层厚·缺血水肿             │ 内层被推开→薄·透明 ✓
>        │ → 变白·不透明 ✗            │ 视锥 + RPE
> ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
> 脉络膜（红，睫状后动脉供血未断）← 同一层一直都在
> ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
> 巩膜
> ```
> 
> - **周边**：内层厚、缺血水肿 → 变白不透明（毛玻璃）→ 挡住脉络膜红 → 苍白。
> - **中心凹**：只剩薄·透明的视锥 + RPE（透明玻璃窗）→ 脉络膜红透得出来 → 红点。
> - 脉络膜由 **posterior ciliary arteries 睫状后动脉**单独供血、CRAO 时**没断** → 照样发红，才有红可透。
> - ⚠️ "凹 (foveal pit)"是**内表面（玻璃体侧）**的凹陷，开口朝前；脉络膜不顶进坑里，只是这点视网膜变薄透明。

## 三、AMD 机制：黄斑「垃圾清运系统」崩溃

> [!info] 承接 §二 解剖
> AMD 战场 = **黄斑区的 RPE–Bruch 膜–脉络膜界面**。RPE 是核心细胞：一边吞噬光感受器每天脱落的外节（phagocytosis），一边在光感受器和脉络膜间运营养、运废物。**AMD 的本质 = RPE 随年龄 + 氧化应激功能衰竭。**

### Dry AMD（干性 / 萎缩性，约占 85%）

```
衰老 + 氧化应激 + 补体失调（CFH 基因）
    ↓
RPE 消化光感受器碎片不彻底 → lipofuscin（脂褐质）堆积
    ↓
RPE 与 Bruch 膜之间堆出 Drusen（玻璃膜疣，脂质/蛋白沉积）  ← buzzword
    ↓
Bruch 膜增厚 → 营养/废物交换受阻
    ↓
RPE 萎缩 → 上面的光感受器「断粮」死亡
    ↓
Geographic atrophy（地图状萎缩）→ 慢、中央暗点 scotoma
```

> [!info] 干性 = 慢性退变 + 萎缩，**无**新生血管、**无**出血
> 处理 = **Amsler 网格自测**监测「干转湿」 + **AREDS 抗氧化维生素**延缓进展；无特效逆转药。

### Wet AMD（湿性 / 渗出性 / 新生血管性，约占 15% 但致盲更快）

```
RPE / 脉络膜层缺血 + 炎症 → VEGF 释放↑
    ↓
Choroidal neovascularization（CNV，脉络膜新生血管）
    ↓  新生血管从【脉络膜】穿过 Bruch 膜破口，长到视网膜下
血管壁不完整、脆 → 渗液 + 出血
    ↓
视网膜下积液/出血 → 视网膜被顶起变形
    ↓
Metamorphopsia（直线变波浪）、快速中央视力↓  ← buzzword
    ↓  终末
Disciform scar（盘状瘢痕）
```

> [!danger] 记忆钩：湿性 AMD 的新生血管来自**脉络膜（choroidal）**，方向是「从下往上穿 Bruch 膜」
> 处理 = **荧光造影确诊 + 玻璃体内抗 VEGF（ranibizumab 雷珠单抗 / aflibercept 阿柏西普）**。

> [!warning] AMD 危险因素
> 年龄（最大）· **吸烟（最大可控因素，必考）** · 白人 · 家族史 / CFH 基因多态。

## 四、DR 机制：慢性高血糖「腐蚀微血管」

DR 不从退变起，而是**全身慢性高血糖直接腐蚀视网膜微血管**。先记高血糖伤血管的通路，再落到视网膜。

### 高血糖伤血管的 4 条通路（机制题根源）

| 通路 | 干了什么 |
|---|---|
| **Sorbitol / polyol 通路** | 醛糖还原酶把葡萄糖→山梨醇，渗透 + 氧化应激，**伤 pericyte（周细胞）** |
| **AGEs（晚期糖基化终产物）** | 糖基化血管基底膜蛋白 → 增厚、炎症 |
| **PKC 激活** | → VEGF↑、血流异常、通透性↑ |
| **Hexosamine 通路** | 促炎、促纤维化 |

### 落到视网膜上的连锁反应

```
慢性高血糖
    ↓
Pericyte（周细胞）丢失 + 基底膜增厚      ← DR 的起点
    ↓
① 毛细血管壁变弱 → Microaneurysm（微动脉瘤，最早体征）
② 内皮紧密连接破坏 → 通透性↑ → 渗漏
        → Hard exudates（硬性渗出，脂质）+ Macular edema（黄斑水肿）  ← DR 视力↓ 主因！
③ 毛细血管闭塞 → 视网膜缺血 → Cotton-wool spots（棉絮斑，神经纤维层微梗死）
```

### NPDR → PDR 的机制分界（就一个词：neovascularization 有没有）

**NPDR（非增殖期）** = 上面那些改变，缺血还没重到逼出新生血管：
- microaneurysm、dot-blot 出血、hard exudates、cotton-wool spots、venous beading（静脉串珠）
- ⚠️ **Macular edema（黄斑水肿）可在任何期发生** → 是 NPDR 也致盲的原因 → 抗 VEGF

**PDR（增殖期）** = 缺血累积到临界点：
```
广泛毛细血管闭塞 → 大面积视网膜缺血
    ↓
VEGF 大量释放
    ↓
Retinal neovascularization（视网膜新生血管，NVD/NVE）
    ↓  新生血管脆
├─ 破 → Vitreous hemorrhage（玻璃体积血，突发飞蚊/视力↓）
├─ 纤维增殖收缩 → Tractional retinal detachment（牵拉性视网膜脱离）
└─ 长到虹膜 → Rubeosis iridis → Neovascular glaucoma（新生血管性青光眼）
```

> [!danger] PDR 处理 = Panretinal photocoagulation（PRP，全视网膜激光光凝）± 抗 VEGF
> PRP 激光「烧掉」缺血的周边视网膜 → 从**源头减少 VEGF 的驱动**（不是直接烧血管，而是消灭产 VEGF 的缺血组织）。这个机制常被考。

### 对照 Hypertensive retinopathy（同轴 B，鉴别）

慢性高血压 → 小动脉硬化：**AV nicking（动静脉压迹）+ copper/silver wiring（铜丝/银丝征）** + flame hemorrhage + cotton-wool；恶性高血压加 **papilledema（视盘水肿）**。与 DR 同属「全身病泡出的眼底改变」，靠**血糖 vs 血压病史 + 特异体征**区分。

| | Diabetic retinopathy | Hypertensive retinopathy |
|---|---|---|
| 病因 | 慢性高血糖 | 慢性高血压 |
| 特异体征 | microaneurysm、hard exudates、neovascularization（PDR） | **AV nicking、copper/silver wiring** |
| 共享体征 | cotton-wool spots、flame/dot 出血 | cotton-wool spots、flame 出血 |
| 新生血管 | PDR 有（retinal NV） | 一般无 |

![[眼科_非增殖性糖网NPDR.png]]
*非增殖性糖网 (NPDR)：点状出血 (dot-blot hemorrhage) + 微动脉瘤 (microaneurysm) + 硬性渗出 (hard exudate)。*

![[眼科_增殖性糖网PDR.png]]
*增殖性糖网 (PDR)：视网膜新生血管 (retinal neovascularization)——是 anti-VEGF / PRP 激光的指征。*

![[眼科_高血压视网膜病变.png]]
*高血压视网膜病变：动静脉压迹 (AV nicking) + 小动脉变窄 + 棉绒斑 + 硬性渗出。*

## 五、汇合点：两条路都通向 VEGF

> [!tip] 最容易混、也最爱考的一句话
> AMD 与 DR 起点机制完全不同，但**晚期都被 VEGF 驱动出新生血管** → 共用抗 VEGF。区别在**新生血管来源不同**：一个来自脉络膜，一个来自视网膜。

| | **Wet AMD** | **Proliferative DR** |
|---|---|---|
| 起点机制 | RPE 衰老退变 + 补体炎症 | 慢性高血糖微血管闭塞缺血 |
| 新生血管**来源** | **脉络膜**（choroidal NV，穿 Bruch 膜） | **视网膜**（retinal NV） |
| 共同末端 | VEGF↑ → 新生血管 → 渗漏/出血 | 同左 |
| 共用治疗 | 抗 VEGF | 抗 VEGF（**+ PRP 激光**） |
| 特异处理 | — | PRP（消灭缺血组织降 VEGF） |

> [!success] 收尾一句
> **AMD = 「黄斑清运系统老化」**（退变→drusen→萎缩，或穿 Bruch 膜的脉络膜新生血管）；
> **DR = 「高血糖腐蚀微血管」**（周细胞死→渗漏/缺血→视网膜新生血管）。
> 两者晚期殊途同归于 **VEGF**，故共用抗 VEGF——但**一个来自脉络膜、一个来自视网膜，别混**。

## 六、4 个交叉点（缝合两轴的考点）

> [!warning] 这些 crossover 把轴 A 和轴 B 连起来，是 stem 里最容易设陷阱的地方

**1. DM 和 HTN 是 CRVO 的头号危险因素**
CRVO（"番茄酱眼底"）的 stem 几乎必给中老年 + **高血压/糖尿病**史 → 轴 B 的病因诱发轴 A 的血管阻塞。（CRAO 则更多挂 **carotid atherosclerosis / cardiac embolism / GCA**。）

![[眼科_CRVO番茄酱眼底.png]]
*CRVO 眼底："blood and thunder / 番茄酱"——静脉迂曲扩张 + 弥漫火焰状出血 + 棉绒斑 + 视盘水肿；危险因素含 HTN/DM。*

**2. Proliferative DR → 牵拉性视网膜脱离 + 玻璃体积血**
PDR 的新生血管很脆 → **vitreous hemorrhage** 和 **tractional retinal detachment** → 把 DR 直接接到轴 A 的 **Retinal detachment**。区别：轴 A 表里那种是 **rhegmatogenous（裂孔性，幕布 + 闪光）**，PDR 是 **tractional（牵拉性）**，机制不同但都叫脱离。

**3. Neovascularization → 治疗共用 anti-VEGF**
**Wet AMD** 与 **Proliferative DR** 共享 VEGF 终末机制 → 都用玻璃体内抗 VEGF；PDR 还加 **PRP**。看到 "neovascularization" 别只想 wet AMD，PDR 也是。

**4. Cotton-wool spots（棉絮斑）是共享体征，不是诊断**
NPDR、hypertensive retinopathy、HIV retinopathy、SLE 都能有 cotton-wool spots（神经纤维层微梗死）。它是「缺血」的**非特异**信号，靠**全身病史 + 其他体征**区分，不能单凭它下诊断。

## 七、临床表现 + 下一步（各病速查）

> [!info] 用法
> 机制在前面各节，这里是**"stem 长什么样 + 该做什么"**的临床落点汇总。轴 A 血管急症（CRAO/CRVO/RD）的完整急诊 workup 见 [[完整笔记/专题笔记/眼科/眼科_HighYield速查总览]]，此处只给下一步要点。

### AMD 年龄相关性黄斑变性

| 病 | 临床表现（症状 + 眼底） | 下一步 |
|---|---|---|
| **Dry AMD 干性** | 老年、数月~数年**渐进性中央视力↓** + 中央暗点 (central scotoma)、阅读/认脸难；眼底 **drusen 玻璃膜疣** + RPE 色素改变，晚期 geographic atrophy 地图状萎缩 | **Amsler grid 自测**监测干转湿 + **AREDS2 补充剂**（抗氧化维生素 + 锌）延缓 + **戒烟**；无逆转药 |
| **Wet AMD 湿性** | 数天~数周**快速中央视力↓ + metamorphopsia 直线变波浪**（Amsler 扭曲）；眼底视网膜下积液/出血、灰绿 CNV 灶 | **OCT + 荧光血管造影 (FA) 确诊 CNV** → **玻璃体内抗 VEGF**（ranibizumab 雷珠单抗 / aflibercept 阿柏西普） |

### DR 糖尿病视网膜病变

| 病 | 临床表现（症状 + 眼底） | 下一步 |
|---|---|---|
| **NPDR 非增殖** | 常**无症状**（筛查发现）；microaneurysm、dot-blot 出血、hard exudates、cotton-wool、venous beading | **控糖 / 控血压 / 控血脂** + 定期扩瞳眼底 + OCT 随访 |
| **PDR 增殖** | 可无症状直到并发症；**neovascularization (NVD/NVE)**；并发 vitreous hemorrhage（突发飞蚊/视力↓）、tractional RD、neovascular glaucoma | **PRP 全视网膜激光光凝** ± 抗 VEGF；玻璃体积血/牵拉脱离 → **vitrectomy 玻璃体切除术** |
| **DME 糖尿病黄斑水肿** | **中央视力↓/模糊**（DR 视力↓ 的主因），任何分期均可发生；OCT 见黄斑增厚积液 | **抗 VEGF（一线）** ± 局灶激光 / 玻璃体内激素 |

> [!tip] DR 筛查时机（常考）
> **1 型糖尿病 (T1DM)：诊断后 5 年**起筛；**2 型糖尿病 (T2DM)：确诊即刻**起筛（发病时间不明、可能已有病变）；**孕期**加密随访。之后每年扩瞳眼底一次。

### Hypertensive retinopathy 高血压视网膜病变

| 病 | 临床表现 | 下一步 |
|---|---|---|
| **慢性高血压网** | 多**无症状**（筛查）；AV nicking、copper/silver wiring、flame hemorrhage、cotton-wool | **控血压** + 查其他靶器官损害 |
| **Malignant HTN 恶性高血压** | 头痛 / 视力↓ + **papilledema 视盘水肿** + flame 出血 | 急症，**可控地降压** + 查靶器官（肾 / 脑 / 心） |

### 轴 A 血管急症（要点；完整急诊 workup 见速查总览）

| 病 | 临床表现 | 下一步 |
|---|---|---|
| **CRAO 视网膜中央动脉阻塞** | 突发**无痛单眼全盲** + cherry-red spot 樱桃红点，可有 amaurosis fugax（一过性黑矇）前驱 | **眼科急症**：眼球按摩 / 降眼压等即时措施 + 查栓子来源（颈动脉 / 心源）；>50 岁查 **GCA (giant cell arteritis, 巨细胞动脉炎) 的 ESR/CRP** |
| **CRVO 视网膜中央静脉阻塞** | 亚急性无痛视力↓ + "blood & thunder" 番茄酱眼底 | 查危险因素（HTN / DM / 高凝）；黄斑水肿 / 新生血管 → **抗 VEGF** ± 激光 |
| **Retinal detachment 视网膜脱离** | **闪光 (flashes) + 飞蚊 (floaters) + 幕布样视野缺损**；眼底灰白起伏网膜 | **急诊眼科手术**（laser / cryopexy 冷冻固定 / scleral buckle 巩膜扣带 / vitrectomy） |

![[L15844.jpg]]

*Retinal detachment 视网膜脱离：神经视网膜与色素上皮 / 脉络膜分离（矢状剖面）*

![[{C856D514-0293-4856-9326-082FA58E7F20}.png]]

*视网膜脱离多从周边起病 → 周边视野缺损（幕帘/黑影自周边向中央推进）+ 飞蚊 / 闪光感 (floaters / photopsia)；黄斑未累及前中心视力可保留*

> [!danger] CRAO 视网膜中央动脉阻塞 (central retinal artery occlusion) 急救 bundle
> 视网膜只耐受缺血 **~90–100 分钟**，一切措施都为**尽快恢复灌注**。眼球按摩 (digital ocular massage) 靠两条路：
> 
> ```
> 眼球按摩（压 ~10 秒 → 松，反复）
>     ├─ ① 机械震动 → 把栓子从主干【挤到更远分支】→ 梗死面积↓ → 保住更多中央视力
>     └─ ② 压→松间 IOP 骤降 → 灌注压 = 动脉压 − IOP，IOP↓ 则灌注压↑
>            → 血流更有力，把栓子后方的血【推过去/绕过去】
> ```
> 
> **核心 = 灌注压 (perfusion pressure) = 动脉压 − IOP（眼压）**：降 IOP 就是抬高这个压差，让血冲过阻塞点；按摩的压-松节律**既降压又机械松栓**。
> 
> 同一 bundle 的其他即时措施都在做同一件事（降 IOP 或扩血管）：
> - **前房穿刺 (anterior chamber paracentesis)** —— 放房水，快速降 IOP
> - **IV acetazolamide 乙酰唑胺 / topical timolol 噻吗洛尔** —— 减少房水生成，降 IOP
> - **carbogen（95%O₂+5%CO₂）/ 纸袋呼吸** —— 高碳酸 → 视网膜血管扩张 + 提高氧供
> 
> ⚠️ 这些措施**证据等级不高、成功率有限**（缺血窗太短），CK 记"**方向对**"即可：降 IOP + 挤栓 + **>50 岁查 GCA（动脉炎型上激素保对侧眼）** + 查栓源（颈动脉/心源，等同 TIA 防脑卒中）。

> [!tip] 高度近视 (high myopia) 是裂孔性视网膜脱离的重要危险因素
> **进展性近视 (myopia, 近视)** = 眼球**轴长 (axial length) 增加** → 图像焦点落在视网膜前 → 远视力模糊、近视力正常。**高度近视 (high myopia, ≥6 diopters 屈光度需要 ≥6D 矫正)** → 巩膜/脉络膜/视网膜被拉伸变薄 → **视网膜裂孔 → rhegmatogenous（裂孔性）视网膜脱离**风险↑ + 黄斑变性风险↑。所以青少年高度近视随访要问闪光/飞蚊/幕布（RD 前驱）。减缓进展：低浓度阿托品 (atropine) 滴眼 / 角膜塑形 (orthokeratology)。

## 八、自测（答完再判分）

> [!question]- Q1. 62 岁男性，2 型糖尿病 15 年，眼底见广泛视网膜新生血管 + 一处玻璃体积血。分期与首选处理机制上如何解释？
> **PDR（增殖性糖网）**。广泛毛细血管闭塞→视网膜缺血→VEGF↑→retinal neovascularization；血管脆破裂致玻璃体积血。首选 **PRP（全视网膜激光光凝）**：烧掉缺血周边视网膜以**从源头降低 VEGF 驱动** ± 抗 VEGF。

> [!question]- Q2. 78 岁女性，数日内看门框变波浪、中央模糊，Amsler 网格扭曲，眼底见视网膜下出血。机制来源与治疗？
> **Wet AMD**。**脉络膜**新生血管（CNV）穿 Bruch 膜到视网膜下，渗漏/出血顶起视网膜致 metamorphopsia。荧光造影确诊 + 玻璃体内**抗 VEGF**。

> [!question]- Q3. 为什么 wet AMD 和 proliferative DR 都用抗 VEGF，却说它们「机制不同」？
> 两者末端都是 **VEGF 驱动的新生血管**（故共用抗 VEGF）；但**起点与来源不同**——AMD 是 RPE 退变→**脉络膜**新生血管，DR 是高血糖微血管缺血→**视网膜**新生血管。

> [!question]- Q4. NPDR 与 PDR 的分界点是哪一个词？NPDR 会不会致盲？
> 分界 = **neovascularization（新生血管）有没有**。NPDR **会**致盲——**黄斑水肿（macular edema）可在任何期发生**，是 DR 视力下降的主因，需抗 VEGF。

> [!question]- Q5. 干性 AMD 患者的核心自我监测手段是什么？监测的是什么转变？
> **Amsler 网格**每日自测；监测「**干转湿**」——一旦直线变波浪（metamorphopsia）提示出现脉络膜新生血管，立即就诊抗 VEGF。

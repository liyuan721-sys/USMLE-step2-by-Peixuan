# 青光眼 Glaucoma 专题（机制 + 治疗 + 鉴别深挖）

> [!info] 用途与定位
> 本篇是**青光眼机制/药物/深度鉴别下沉专题**。考前秒查用 [[完整笔记/专题笔记/眼科/眼科_HighYield速查总览]] 的"一、青光眼"表；此处放**为什么**——房水循环解剖、开角 vs 闭角病理、每个药的作用机制、两个"固定瞳孔"与红眼的横向鉴别。知识源 [[完整笔记/Peixuan分科笔记/眼科]]（只读）。

> [!note]- 缩写对照（全称 中英）
> - **IOP** = intraocular pressure 眼压
> - **POAG** = primary open-angle glaucoma 原发性开角型青光眼
> - **AACG** = acute angle-closure glaucoma 急性闭角型青光眼
> - **TM** = trabecular meshwork 小梁网
> - **PG** = prostaglandin 前列腺素
> - **CAI** = carbonic anhydrase inhibitor 碳酸酐酶抑制剂
> - **RAPD** = relative afferent pupillary defect 相对性传入性瞳孔障碍
> - **CN III** = cranial nerve III 动眼神经
> - **OCS** = orbital compartment syndrome 眼眶间隔综合征

## 一、一图定位：青光眼两分叉

```
青光眼（视神经病变 + 常伴 IOP↑）
  │
  ├─ 起病慢 + 无痛 + 周边视野丢失(tunnel vision) + 视盘 cupping
  │        │
  │        └──► 开角型 POAG（最常见，筛查病）
  │             机制 = 房水【流出↓】(小梁网阻力↑)，房角是开的
  │
  └─ 起病急 + 剧痛 + 眼球硬如石 + 固定中度散大瞳 + 红眼 + 恶心呕吐 + 灯周光晕
           │
           └──► 急性闭角型 AACG（眼科急症）
                机制 = 瞳孔阻滞 → 房角【物理关闭】→ 房水出不去
```

> [!tip] 一句话钥匙
> **"慢+无痛+周边丢 = 开角(筛子堵)；急+痛+硬如石+散瞳 = 闭角(门关死)"**。开角是"排水口滤网变堵"，闭角是"排水口被虹膜盖住"。

## 二、房水循环解剖（理解一切的地基）

> [!info] 房水通路
> 房水 (aqueous humor) 由**睫状体 (ciliary body)** 分泌 → 经**后房** → 穿**瞳孔** → 入**前房** → 从**房角 (anterior chamber angle)** 的**小梁网 (trabecular meshwork)** → **Schlemm 管 (canal of Schlemm)** → 静脉回流。另有 **uveoscleral 葡萄膜巩膜通路**（前列腺素类药作用点）。

![[眼科_眼球矢状解剖三层结构.png]]
*眼球矢状解剖：睫状体 (ciliary body) 分泌房水 → 后房 (posterior chamber) → 瞳孔 → 前房 (anterior chamber) → 房角（虹膜 iris 与角膜 cornea 交界处）。房角这个夹角开或闭，就是开角 vs 闭角青光眼的分水岭。*

```
睫状体分泌房水 ──► 后房 ──► 瞳孔 ──► 前房 ──► 房角
                                            │
                        ┌───────────────────┴──────────────┐
                        ▼                                   ▼
                小梁网→Schlemm管(主通路,~80%)         葡萄膜巩膜通路(~20%)
                     ↑ POAG 在这里阻力↑                  ↑ 前列腺素药在这里增流出
```

- **生成 ↑ 或 流出 ↓ → IOP ↑ → 压迫视神经乳头 → cupping（杯盘比↑）→ 视野缺损**。
- 视野丢失顺序：**周边先 → 中央后**（黄斑对应的中央视野最后受累，所以 tunnel vision 是晚期，早期常无症状）。
- 治疗两条思路：**① 减少房水生成**（β 阻滞剂、α2 激动剂、CAI）；**② 增加房水流出**（前列腺素走葡萄膜巩膜、拟胆碱药/激光开小梁网）。

## 三、开角型青光眼 POAG（渐进无痛的"筛子堵"）

> [!warning] 病理机制
> 房角是**开放**的，但**小梁网对房水流出阻力↑** (↑ resistance through trabecular meshwork / drainage channel) → 慢性 IOP↑。
> ⚠️ 旧教学曾说"房水生成过多 (overproduction)"，**Step 2 已更新为"流出↓ (decreased outflow)"** —— 选项里若同时出现，选**流出减少**。

**临床特征**

| 维度 | 表现 |
|---|---|
| 起病 | 隐匿、**无痛**、双眼（可不对称，一眼重） |
| 症状 | 早期**无症状**；进展 → **周边视野渐丢 (peripheral/tunnel vision)** |
| 眼底 | 视神经乳头萎缩：**cup 增大、盘缘变薄 (rim thinning)、杯盘比↑ (cupping)**、视盘苍白 |
| 危险因素 | 年龄、非裔、家族史、高眼压、糖尿病、高度近视 |
| 诊断 | tonometry（IOP↑）+ 眼底 cupping + 视野计 |

![[眼科_青光眼杯盘比.png]]
*视盘对比：正常 (cup/disc <0.5) vs 开角青光眼（杯盘比 >0.6、盘缘变薄、视盘苍白 = 视神经萎缩）。*

> [!success] 治疗阶梯（记忆：一线前列腺素，加 β 阻滞剂，激光/手术兜底）
> 1. **1st line：局部前列腺素类 topical prostaglandin**（latanoprost 拉坦前列素 / bimatoprost 比马前列素）→ **↑ 葡萄膜巩膜通路流出**。副作用：虹膜/睫毛变深、结膜充血。
> 2. **加 β 阻滞剂 topical beta-blocker**（timolol 噻吗洛尔）→ **↓ 房水生成**。⚠️ **哮喘/COPD (chronic obstructive pulmonary disease 慢性阻塞性肺病)/心动过缓慎用**（全身吸收致支气管痉挛、心动过缓）。
> 3. **laser trabeculoplasty 激光小梁成形术** / 手术 → 增加流出。

## 四、急性闭角型青光眼 AACG（眼科急症）

> [!danger] 病理机制：瞳孔阻滞 (pupillary block)
> 有**易感解剖**（前房浅、房角窄、远视眼、亚裔/老年）的人，当**瞳孔中度散大**时，虹膜与晶状体贴合 → 房水从后房到前房被堵 (pupillary block) → 后房压力把周边虹膜顶向房角 → **房角物理关闭 → 房水急性无法流出 → IOP 骤升**。

**诱因（题干常给的"扳机"）**

```
瞳孔散大 = 扳机
  ├─ 暗环境 (low ambient light，如看电影/夜间)
  ├─ 减充血剂 decongestants (拟交感)
  ├─ 抗胆碱药 anticholinergics (trihexyphenidyl 苯海索 / tolterodine 托特罗定 / 抗组胺 / TCA 三环类抗抑郁药)
  └─ 磺胺类 sulfonamide (特异质反应致睫状体水肿前移)
```

**临床特征（一眼、剧烈）**

| 维度 | 表现 |
|---|---|
| 症状 | **急性剧烈眼痛 + 头痛 + 恶心呕吐**（可误诊急腹症/偏头痛）、视物模糊、**灯周光晕 halos** |
| 体征 | **固定中度散大瞳 (fixed, mid-dilated pupil)**、结膜充血 (conjunctival redness)、**角膜混浊水肿 (corneal opacification)** |
| 触诊 | **眼球硬如石 (stony-hard / rock-hard globe)** |
| 诊断 | **gonioscopy 房角镜 > tonometry** —— 房角镜直接看到房角关闭是确诊；眼压计测 IOP↑ 支持 |

![[眼科_急性青光眼外眼.jpeg]]
*正常眼 vs 急性闭角青光眼：角膜水肿混浊 + 固定中度散大瞳 + 结膜充血。*

![[{54C6FC8D-871E-4D84-9D03-820FA06DB255}.png]]

> [!danger] 治疗：先内科快速降压，再激光根治（顺序不能反）
> **第一步——快速降 IOP（多药联合）**
> - **IV acetazolamide 乙酰唑胺**（CAI）→ ↓ 房水生成（全身快速）
> - 局部多药：**timolol 噻吗洛尔**（β 阻滞，↓生成）+ **apraclonidine 阿可乐定**（α2 激动，↓生成）+ **pilocarpine 匹罗卡品**（拟胆碱缩瞳，拉开房角）
> - ± **mannitol 甘露醇**（高渗，进一步快速降压）
>
> **第二步——根治**
> - **laser iridotomy 激光虹膜切开术** → 在虹膜打孔，解除瞳孔阻滞、平衡前后房压力（根治 + 预防对侧眼发作）

> [!warning] pilocarpine 的时机陷阱
> 拟胆碱药 pilocarpine 缩瞳能拉开房角，但**在急性高眼压期，虹膜括约肌缺血可能对 pilocarpine 无反应**，故先用 acetazolamide/timolol/mannitol 把压降下来，再用 pilocarpine 更有效。考试记"先降压药起效，再缩瞳"。

## 五、抗青光眼药物机制大全（按"降生成 vs 增流出"两分）

> [!info] 一张表锁死每个药的机制与陷阱

| 药物 | 类别 | 机制 | 净效应 | 关键陷阱 / 副作用 |
|---|---|---|---|---|
| **Latanoprost / Bimatoprost** 拉坦/比马前列素 | 前列腺素类 | ↑ 葡萄膜巩膜流出 | 流出↑ | POAG **一线**；虹膜/睫毛变深、结膜充血 |
| **Timolol** 噻吗洛尔 | β 阻滞剂 | ↓ 睫状体房水生成 | 生成↓ | **哮喘/COPD 慢阻肺/心动过缓慎用**（全身吸收） |
| **Acetazolamide / Dorzolamide** 乙酰唑胺/多佐胺 | CAI 碳酸酐酶抑制剂 | ↓ 房水生成 | 生成↓ | 全身 acetazolamide 用于**急性闭角快速降压**；磺胺过敏注意；代谢性酸中毒 |
| **Apraclonidine / Brimonidine** 阿可乐定/溴莫尼定 | α2 激动剂 | ↓ 房水生成（+ 稍↑流出） | 生成↓ | 用于急性发作辅助 |
| **Pilocarpine** 匹罗卡品 | 拟胆碱（M 激动） | 缩瞳 → 牵拉虹膜、**开放房角** + 牵拉小梁网 | 流出↑ | **闭角**首选缩瞳；急性高压期可能失效（先降压） |
| **Mannitol** 甘露醇 | 高渗利尿 | 渗透性脱水、↓玻璃体容积 | IOP↓ | 急性发作**兜底快速降压** |

> [!tip] 记忆钩子
> **开角一线 = 前列腺素（增流出）**；**闭角急救 = 乙酰唑胺 + 噻吗洛尔 + 缩瞳 ±甘露醇 → 激光虹膜切开**。
> 口诀："**开角靠前列腺素增流出，闭角先降压再打孔**"。

> [!danger] 别把 Pilocarpine 和 Apraclonidine 的机制记反（高频混淆点）
> 这两个都用在急性闭角，但**受体系统完全不同**，考试爱互换选项挖坑：
> - **Pilocarpine 匹罗卡品 = 拟胆碱 (M/muscarinic 激动)** → **缩瞳 → 拉开房角、增流出**（不是 α2、不是降生成）。
> - **Apraclonidine 阿可乐定 = α2 肾上腺素能激动** → **降房水生成**（不缩瞳、不是 M 激动）。
> 一句话："**Pilo 走胆碱管缩瞳开门（增流出），Apra 走 α2 管关水龙头（降生成）**"。

## 六、开角 vs 闭角治疗对比（总表 — 一处看全）

> [!info] 核心差异：慢病长期控压 vs 急症快速砸压
> **开角 POAG = 门诊长期用药、一线单药起步**（目标平稳降 IOP 保视野）；**闭角 AACG = 急诊多药齐上快速降压 → 激光根治**（目标数小时救视力）。

| 维度 | 开角型 POAG（慢性） | 急性闭角型 AACG（急症） |
|---|---|---|
| 治疗目标 | 长期平稳降 IOP、保视野 | **数小时内快速降 IOP** → 再根治 |
| 一线 | **前列腺素 latanoprost 拉坦前列素 / bimatoprost 比马前列素**（↑葡萄膜巩膜流出） | **多药联合**（非单药） |
| 加药顺序 | 加 **timolol 噻吗洛尔**（↓生成）；不够再加 CAI / α2 | — |
| 急性降压组合 | 不适用 | IV **acetazolamide 乙酰唑胺** + **timolol 噻吗洛尔** + **apraclonidine 阿可乐定**（↓生成）+ **pilocarpine 匹罗卡品**（缩瞳开房角）± **mannitol 甘露醇**（高渗） |
| 根治/手术 | **laser trabeculoplasty 激光小梁成形术**（拓宽滤网） | **laser iridotomy 激光虹膜切开术**（虹膜打孔解除瞳孔阻滞 + 预防对侧眼） |
| 缩瞳药 pilocarpine | 非一线，一般不用 | **要用**——缩瞳把虹膜从房角拉开 |

> [!info] 机制逻辑：为什么组合不同

```
开角 = 排水滤网(小梁网)堵 → 只要"多排水 or 少产水"就够,慢慢调
        ├─ 增流出：前列腺素(一线) / 拟胆碱 / 激光小梁成形
        └─ 降生成：β阻滞 / CAI / α2

闭角 = 虹膜盖住排水口 → 既要立刻降总量,又要把盖子(虹膜)掀开
        ├─ 立刻降总量：乙酰唑胺 + 噻吗洛尔 + α2 ± 甘露醇（多路并进）
        └─ 掀盖子：pilocarpine 缩瞳拉开房角 → 最终 iridotomy 打孔根治
```

> [!danger] 4 个治疗对比陷阱
> 1. **两个"根治手术"别记反**：开角 = **小梁成形 trabeculoplasty**（修滤网）；闭角 = **虹膜切开 iridotomy**（掀盖子/打孔）。
> 2. **pilocarpine 的时机**：急性高压期虹膜括约肌缺血可能**对缩瞳药无反应** → **先用乙酰唑胺/噻吗洛尔/甘露醇把压降下来，再缩瞳**。
> 3. **闭角易感者要避开散瞳类**（**抗胆碱药、减充血剂、抗组胺、TCA 三环类抗抑郁药**）——它们在闭角是诱因/禁忌，但对开角无此顾虑。
> 4. **顺序不能反**：闭角**先内科降压、后激光**；压没降就直接上激光是错的。

## 七、小儿青光眼 Glaucoma in children

> [!warning] 与成人不同——看"三联征 + 眼球增大"
> **病理**：视神经病变 ± IOP↑，因**房水引流受损**（房角发育异常 angle dysgenesis / 原发解剖异常 / 继发）。
>
> **病因**：原发性房角发育异常、**Sturge-Weber 综合征**、**皮质激素诱导 (corticosteroid-induced)**、肿瘤/外伤/感染累及房角。
>
> **关键特征**：
> - **流泪 + 畏光 + 睑痉挛 (tearing, photophobia, blepharospasm)** ——经典三联征
> - **角膜/眼球增大 (enlarged cornea/globe，"牛眼" buphthalmos)** ——小儿眼球壁弹性大，高压把眼球撑大
> - 视盘 cupping、tonometry IOP↑
>
> **处理**：**手术 (surgery)** 为主 ± 降压眼药。

## 八、高频横向鉴别

### 8.1 两个"固定瞳孔"别混（★题眼★）

> [!danger] 一个是眼病急症，一个是致命神经急症
> - **Fixed MID-dilated pupil（固定中度散大）** = **急性闭角青光眼 AACG** —— 伴红眼、眼痛、硬如石。
> - **Fixed FULLY-dilated pupil + 上睑下垂 (ptosis) + 眼球向下外偏 (down-and-out)** = **压迫性 CN III 麻痹** —— 排**后交通动脉瘤 / 钩回疝 (uncal herniation)**，until proven otherwise。
>
> 差别：闭角青光眼瞳孔**中度**散大 + 眼**痛红硬**；CN III 麻痹瞳孔**完全**散大 + **眼位异常 + 眼睑下垂**、眼不红不硬。

### 8.2 急性红眼鉴别（AACG 混在其中）

| 诊断 | 疼痛 | 瞳孔 | 关键鉴别点 |
|---|---|---|---|
| **急性闭角青光眼** | **剧痛** + 头痛恶心 | **固定中度散大** | 眼球硬如石、角膜混浊、灯周光晕、gonioscopy |
| **前葡萄膜炎 anterior uveitis** | 中度、畏光 | **缩小/不规则** | ciliary flush 睫状充血、hypopyon 前房积脓、HLA-B27 |
| **HSV 角膜炎 keratitis** | 中度、异物感 | 正常 | 荧光素**树枝状溃疡 dendritic** |
| **细菌性结膜炎** | 轻、刺激感 | 正常 | 脓性分泌物、无视力改变 |

> [!tip] 秒选：**红眼 + 剧痛 + 恶心呕吐 + 看灯有光圈 + 瞳孔固定中度散大 → 急性闭角青光眼**（口诀："红眼头痛又想吐，看灯有圈瞳孔住；固定散大视力差，闭角青光眼先答"）。

### 8.3 眼球硬：AACG vs 眼眶间隔综合征 OCS

| 特点 | 急性闭角青光眼 | **OCS 眼眶间隔综合征** |
|---|---|---|
| 硬的部位 | **眼球**硬如石 | **眼睑 + 眼眶张力**↑ |
| 机制 | 眼**内**压↑ | 眼**眶**压↑（多因外伤后眶内出血） |
| Proptosis 突眼 | ❌ 无 | **✅ 有** |
| 处理 | 药物降压 → 激光虹膜切开 | **别等影像，立即 lateral canthotomy/cantholysis** |

## 九、经典 stem → 诊断反射

- 老年人**看电影/暗环境后**突发单眼剧痛 + 头痛恶心呕吐 + 看灯光晕 + 眼球硬如石 + 瞳孔固定中度散大 → **急性闭角青光眼** → **gonioscopy** 确诊 → IV acetazolamide + 局部降压 → laser iridotomy。
- 刚服**抗胆碱药/减充血剂/抗组胺**后突发红眼剧痛 → 想到**药物诱发闭角青光眼**。
- 无症状老年人**常规查眼底见杯盘比↑ (cupping)** + 视野周边缺损 → **POAG** → 一线 latanoprost。
- 哮喘患者 POAG 用 timolol 后**喘息加重** → β 阻滞剂全身吸收致支气管痉挛 → 换非 β 阻滞类。
- 婴儿**流泪 + 畏光 + 睑痉挛 + 角膜增大** → **先天性/小儿青光眼**（想 Sturge-Weber、激素诱导）→ 手术。

## 十、自测（答完再判分）

> [!question]- Q1. 68 岁女性看电影后突发右眼剧痛、头痛、恶心呕吐、看灯有彩色光圈；右眼结膜充血、角膜雾状、瞳孔中度散大且固定、触诊眼球硬如石。最能确诊的检查是？
> **急性闭角型青光眼 (AACG)**；确诊检查 = **gonioscopy 房角镜**（> tonometry）。处理：先 IV acetazolamide + timolol/apraclonidine/pilocarpine ± mannitol 降压，再 **laser iridotomy** 根治。

> [!question]- Q2. POAG 的房水动力学核心异常是房水生成过多还是流出减少？一线药物及其机制？
> **流出减少 (decreased outflow through trabecular meshwork)**（旧说法"生成过多"已被更新，别选）。一线 = **topical prostaglandin (latanoprost/bimatoprost)**，机制 = **↑ 葡萄膜巩膜通路流出**。

> [!question]- Q3. 急性闭角青光眼与压迫性 CN III 麻痹都可有"散大瞳孔"，如何一眼区分？
> **AACG = 固定中度散大 + 眼痛红硬 + 头痛恶心**；**CN III 麻痹 = 完全散大 + ptosis + down-and-out 眼位**，眼不红不硬（提示后交通动脉瘤/钩回疝，急查）。

> [!question]- Q4. 3 月龄婴儿流泪、畏光、频繁眯眼，双眼角膜较同龄偏大。最可能诊断与首选处理？
> **先天性/小儿青光眼**（三联征 tearing/photophobia/blepharospasm + 角膜增大 buphthalmos）；首选 **手术** ± 降压眼药；病因想 Sturge-Weber、皮质激素诱导。

> [!question]- Q5. POAG 患者加用 timolol 后出现喘息加重，机制是什么？该换成哪类降压方向相同的药？
> timolol 是**非选择性 β 阻滞剂**，全身吸收 → **支气管痉挛**（哮喘/COPD 慎用）。同为"↓房水生成"方向可换 **CAI (dorzolamide)** 或 **α2 激动剂 (brimonidine)**。

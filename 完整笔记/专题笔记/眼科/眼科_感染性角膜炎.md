# 感染性角膜炎 Infectious Keratitis（三病原 + 溃疡形态定诊断）

> [!info] 用途与定位
> 本篇是**"角膜溃疡 / 角膜炎"病原鉴别的下沉专题**。考前秒查用 [[完整笔记/专题笔记/眼科/眼科_HighYield速查总览]] 的"四、感染/角膜/眼睑"（按溃疡形态秒选表）；此处放**为什么**——三大病原（细菌 / HSV / 真菌）的危险因素-溃疡形态-治疗对应，重点是最高频的 **HSV 树枝状角膜炎**，以及**免疫抑制/HIV 患者眼病谱**里 HSV 角膜炎 vs CMV 视网膜炎 vs 口腔念珠菌病的区分。红眼但无角膜溃疡（结膜炎/葡萄膜炎）见 [[完整笔记/专题笔记/眼科/眼科_结膜炎与新生儿眼分泌物鉴别]]。知识源 [[完整笔记/Peixuan分科笔记/眼科]]（只读）。

> [!note]- 缩写对照（全称 中英）
> - **HSV** = herpes simplex virus 单纯疱疹病毒
> - **VZV** = varicella-zoster virus 水痘-带状疱疹病毒
> - **CMV** = cytomegalovirus 巨细胞病毒
> - **HIV** = human immunodeficiency virus 人类免疫缺陷病毒
> - **CD4** = CD4+ T lymphocyte CD4 阳性 T 淋巴细胞
> - **Keratitis** 角膜炎（cornea 角膜的炎症）

## 一、总览：感染性角膜炎三病原

> [!tip] 一句话钥匙
> **所有角膜炎都有眼痛 + 红 + 视物模糊 + 畏光**（共性，不能靠这些分病原）；**靠溃疡形态 + 危险因素**分三病原：**圆形中央溃疡 + 隐形眼镜/外伤 = 细菌；树枝状分支 + 免疫抑制/复发 = HSV；羽毛状边缘 + 卫星灶 + 植物/土壤外伤 = 真菌。**

| 病原 | 危险因素 | 溃疡形态 / 特征 | 治疗 |
|---|---|---|---|
| **细菌** (*S. aureus* 金葡菌、*Pseudomonas* 铜绿假单胞) | **隐形眼镜佩戴不当**、角膜外伤/异物 | **中央圆形溃疡 + 基质脓肿 (stromal abscess) + 黏脓**、急性 | topical 抗生素（隐形眼镜者 **fluoroquinolone** 氟喹诺酮） |
| **HSV 单纯疱疹病毒** | **免疫抑制、HIV**、复发激活 | **树枝状分支溃疡 (dendritic ulcer) + ↓角膜知觉 + 水样分泌 + 复发** | topical（trifluridine 曲氟尿苷 / ganciclovir 更昔洛韦）或 oral（acyclovir 阿昔洛韦 / valacyclovir 伐昔洛韦） |
| **真菌** (*Candida* 念珠菌、*Fusarium* 镰刀菌) | **免疫抑制 + 角膜外伤（受污染土壤/植物，如园艺、荆棘）** | **羽毛状边缘 (feathery margins) + 卫星灶 (satellite lesions)**、进展**缓慢 (indolent)** | topical 抗真菌 |

## 二、HSV 角膜炎（树枝状溃疡，最高频）

> [!info] 特征三件套
> - **树枝状溃疡 (dendritic ulcer)**：荧光素染色见**分支线状 (branched/linear)** 溃疡 —— HSV 角膜炎的**病理签名**。
> - **角膜知觉下降 (↓ corneal sensation)**：HSV 损伤三叉神经支配，是特征性体征。
> - **复发 (recurrent)**：潜伏于三叉神经节，可反复激活。
> - **免疫抑制/HIV → 双侧受累 + 更重**（免疫正常者多单侧）。

![[{A64B4195-06A2-47D0-9250-D904E5572EC6}.png]]
*病毒性角膜炎 (viral keratitis)：单纯疱疹病毒 (herpes simplex virus, HSV) / 水痘带状疱疹病毒 (varicella zoster virus, VZV) 感染角膜 → 角膜水疱、混浊、**树枝状溃疡 (dendritic ulcer)**；治 topical ganciclovir 更昔洛韦 / trifluridine 曲氟尿苷（双嵌自卡 [[mistakes/uworld-mistakes_2026-07#^Q2374]] 的干扰项）。*

> [!danger] 治疗禁区：别单用糖皮质激素
> HSV 角膜炎用**抗病毒**（topical trifluridine/ganciclovir 或 oral acyclovir/valacyclovir）；**单用局部糖皮质激素会加重感染、诱发地图状 (geographic) 溃疡**——见到树枝状溃疡先想抗病毒，不是激素。

> [!danger] Item 22 (Q4099) 陷阱详解：HIV + 树枝状角膜炎 → 为什么是 HSV，不是 Candida
> 35 岁男，**HIV 未服抗病毒药** + 双眼进行性视力恶化（先眼痛红 → 视物模糊畏光） + **profuse clear drainage（大量清亮分泌）** + 裂隙灯见 **marked linear branching keratitis（显著树枝状角膜炎）** + 眼底/视神经正常 + **口腔颊黏膜/腭/舌白斑 (white plaques)** → **HSV 角膜炎**。
> - ❌ ***Candida albicans*（白念珠菌）**（选 A，你选的）：真菌角膜炎是**羽毛状边缘 + 卫星灶**、通常**角膜外伤 + 污染土壤**、进展慢；**口腔白斑（鹅口疮 thrush）是黏膜念珠菌病，但与眼部感染无因果**——这是本题最大陷阱：**别把"口腔 thrush"和"眼病"因果连起来**。thrush 只是提示他免疫抑制重，不代表眼病也是念珠菌。
> - ❌ **Cytomegalovirus (CMV)**（选 B）：HIV 最常见严重眼病是 **CMV retinitis（视网膜炎）**，但那是**视网膜病**（无痛 + 眼底黄白颗粒灶 + 出血 "pizza pie"，CD4<50）；本例**眼底正常 + 树枝状角膜溃疡**（角膜病，不是视网膜）→ 不符。
> - ❌ ***Pseudomonas aeruginosa*（铜绿假单胞）/ *S. aureus*（金葡菌）**（选 D/E）：细菌是**中央圆形浸润 + 黏脓**、隐形眼镜/外伤相关，**非树枝状分支** → 不符。
> - 🔑 记忆：**"linear branching（树枝状）= HSV，天塌了也是 HSV"**；HIV/免疫抑制 → 双侧 + 复发。**口腔 thrush 是免疫抑制的旁证，不是眼病病原**（同一病人可以有两种不相干的机会感染）。

## 三、免疫抑制/HIV 患者的眼病谱（★干扰项区★）

> [!warning] 一个 HIV 病人可以同时有多个机会感染——别把"口腔病灶"当"眼病病因"
> UW 爱在 HIV 病人身上堆多个机会感染线索（口腔 thrush、树枝状角膜炎、眼底病灶），考你**分清哪个线索指向眼病诊断**。

| 部位 | 病 | 关键特征 | 与眼诊断的关系 |
|---|---|---|---|
| **角膜** | **HSV keratitis** | 树枝状溃疡 + 痛 + ↓知觉 | Item 22 的**眼病诊断** |
| **视网膜** | **CMV retinitis** | 无痛 + 眼底黄白灶 + 出血 "pizza pie"、CD4<50 | 眼底正常则**排除** |
| **口腔黏膜** | **Candida（鹅口疮 thrush）** | 颊/腭/舌白斑（可刮除） | **免疫抑制旁证，非眼病病因**（干扰） |
| 视网膜（外周） | Candida endophthalmitis | 静脉导管/TPN + 飞蚊 + 白棉球灶 | 与角膜树枝状溃疡无关 |

## 四、红眼何时想"角膜炎"：横向鉴别

> [!info] 用"荧光素染色 + 有无视力↓/畏光"把角膜炎从红眼堆里拎出来
> 详细急性红眼鉴别见 [[完整笔记/专题笔记/眼科/眼科_青光眼]] §8.2；此处只落角膜层面。

| 诊断 | 荧光素染色 | 疼痛/畏光 | 视力 | 钩子 |
|---|---|---|---|---|
| **感染性角膜炎** | **阳性（溃疡）** | **痛 + 畏光** | 可↓ | 溃疡形态定病原 |
| **角膜擦伤 corneal abrasion** | **阳性（局限缺损）** | 异物感/痛 | 多正常 | 外伤/异物史 |
| **结膜炎 conjunctivitis** | 阴性 | 轻/gritty | 正常 | 分泌物性状（→ 笔记1） |
| **前葡萄膜炎 anterior uveitis** | 阴性 | 痛 + 畏光 | 可↓ | ciliary flush 睫状充血 + hypopyon 前房积脓 |
| **急性闭角青光眼** | 阴性 | 剧痛 + 头痛恶心 | ↓ | 硬如石 + 固定中度散大瞳 |

## 五、隐形眼镜相关角膜炎（补充）

> [!warning] 隐形眼镜佩戴者角膜炎的两个必想病原
> - ***Pseudomonas aeruginosa*（铜绿假单胞）**：进展快、绿色黏脓 → **fluoroquinolone**（moxifloxacin 莫西沙星 / ciprofloxacin 环丙沙星）；**24h 随访、不贴眼罩**（眼罩助厌氧/病原繁殖）。
> - ***Acanthamoeba*（棘阿米巴）**：用**自来水冲洗镜片 / 戴镜游泳**史；**剧痛与体征不成比例 + 环形浸润 (ring infiltrate)**。

> [!tip] 隐形眼镜相关**擦伤**（非感染但同 Pseudomonas 高危）的处理见 [[完整笔记/专题笔记/眼科/眼科_角膜擦伤]]
> 戴镜者角膜擦伤要用**抗铜绿假单胞抗生素（ofloxacin 氧氟沙星 / ciprofloxacin 环丙沙星 / tobramycin 妥布霉素）+ 绝不贴眼罩 + 24h 复查**——荧光素只染局限缺损、无白色浸润时是擦伤；一旦见浸润/溃疡就归本篇的感染性角膜炎急诊转诊。

## 六、自测（答完再判分）

> [!question]- Q1. HIV 未治疗患者，双眼进行性视力↓ + 眼痛畏光 + 裂隙灯见树枝状角膜溃疡，眼底正常，口腔见白斑。最可能病原？为什么不是 Candida 或 CMV？
> **HSV（单纯疱疹病毒）角膜炎**。树枝状（linear branching）= HSV 签名。不是 **Candida**（真菌是羽毛状 + 卫星灶 + 外伤史；口腔 thrush 只是免疫抑制旁证，与眼病无因果）；不是 **CMV**（那是视网膜炎，眼底会有黄白灶，本例眼底正常）。

> [!question]- Q2. 感染性角膜炎三病原的溃疡形态各是什么？
> **细菌 = 中央圆形溃疡 + 基质脓肿**；**HSV = 树枝状分支溃疡 + ↓角膜知觉**；**真菌 = 羽毛状边缘 + 卫星灶（indolent）**。

> [!question]- Q3. 见到 HSV 树枝状角膜炎，为什么不能单用局部糖皮质激素？
> 单用糖皮质激素会**加重 HSV 感染、诱发地图状 (geographic) 溃疡**；应用**抗病毒**（topical trifluridine/ganciclovir 或 oral acyclovir/valacyclovir）。

> [!question]- Q4. 隐形眼镜佩戴者急性角膜溃疡，最需覆盖的病原与用药？还要想哪个"剧痛不成比例"的病原？
> **Pseudomonas（铜绿假单胞）→ fluoroquinolone**，24h 随访、不贴眼罩。剧痛与体征不成比例 + 环形浸润 + 自来水/游泳史 → **Acanthamoeba（棘阿米巴）**。

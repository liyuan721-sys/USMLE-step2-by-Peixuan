# 医患沟通对话术 SOP（Motivational Interviewing 核心）

> [!info] 这篇笔记是什么
> 整合 **Q11936 + Q21169** 两道连错的 Motivational Interviewing (MI, 动机性访谈) 题，建立"对话术题"的统一解题框架。当前覆盖 MI；后续坏消息告知 (SPIKES)、知情同意、患者拒绝治疗等对话术错题积累后继续补入。

---

## Part 0. 对话术题的统一入口

> [!tip] 拿到对话术题先做两件事
> 1. **判断患者所处的 Stages of Change 阶段**（不在乎？矛盾？想改？正在改？）
> 2. **按阶段选话术**——阶段不同，"最该说的话"完全不同。
>
> 错题反射纠正：**绝大多数对话术题，"讲后果/威胁/说教"选项都是错的。** MI 的本质是"让患者自己说出改变的理由"，不是"医生把利害讲清楚"。

---

## Part 1. Stages of Change（变化阶段模型 / Transtheoretical Model）

```
变化阶段模型 → 医生该做什么
│
├─ Precontemplation（无意图期）
│   特征：不认为自己有问题，"unconcerned" / 否认
│   医生目标：提高 AWARENESS，制造矛盾（行为 ↔ 价值观）
│   👉 Q11936 在这里
│
├─ Contemplation（意图期）
│   特征：开始觉得"也许是个问题"，矛盾心理（ambivalence）
│   医生目标：探讨利弊、强化改变动机、evoke change talk
│   👉 Q21169 患者打 2/5，处于这一带
│
├─ Preparation（准备期）
│   特征：打算行动、主动问"我该怎么办"
│   医生目标：帮助制定计划
│
├─ Action（行动期）
│   特征：正在改变
│   医生目标：提供具体方案（减量 / 戒断 / AA / 转介）
│
└─ Maintenance（维持期）
    特征：巩固成果、防复发
    医生目标：支持、预防复发
```

> [!warning] 为什么这样
> 阶段决定能做什么。对**还没承认有问题**的患者（Precontemplation）谈具体戒断方案（AA、减量）= **跳阶段**，是经典陷阱。必须先建立觉察，患者才接得住方案。

## Part 2. Motivational Interviewing（动机性访谈）核心

### MI 四组件（UWorld 配图考点）

```
Engaging  建立关系：非评判、开放式、合作设定议程、发掘患者优势
Focusing  聚焦：请患者锁定 1-2 个行为目标
Evoking ⭐ 引出 change talk：挖出患者自己的"承诺 + 改变理由"
Planning  计划：识别具体步骤、预判障碍、定义成功
```

### MI 四大精神原则（OARS 精神）

```
表达共情 Express empathy ──── 不评判
制造矛盾 Develop discrepancy ─ 让患者自己看到"行为 ↔ 目标"冲突
避免对抗 Roll with resistance ─ 不争辩、不贴标签、不站队
支持自我效能 Support self-efficacy ─ 患者主导、相信自己能改
```

> [!success] 一句话区分本质
> **MI = 心理咨询师，不是说教的老师。** 医生的角色是"提问 / 引导 / 镜映"，让患者**自己说出**改变的理由——患者自己说出来的动机，远强于医生灌输的。

## Part 3. Readiness Ruler（准备度标尺）— 向下问铁律

```
0 ─────────── 2 ─────────────────── 5
低              ↑患者在这                高
                │
        ┌───────┴────────┐
   ❌ 向上问                ✅ 向下问
   "怎么到 5？"            "为什么是 2 不是 0？"
   = 聚焦缺陷/障碍          = 挖出已有的改变理由
   = 暗含"该打5"的评判      = 强化动机 → change talk
```

> [!danger] Readiness Ruler 铁律
> 患者打了**非零分**，说明心里**已有改变的理由**。永远**向下问**（为什么不是 0），请患者自己说出这些理由 = change talk。**绝不向上问**（怎么到 5）——那会让患者去想障碍和缺陷，反而削弱动机，且暗含评判。

## Part 4. 五大对话术陷阱（口诀：讲 / 贴 / 站 / 跳 / 上）

> [!warning] 看到这五类选项 → 直接排除
> | 陷阱 | 错误动作 | 例 |
> |---|---|---|
> | **讲**（后果） | 反复陈述医学危害 / 威胁后果 | "继续喝会肝损伤" / "不改风险很高" |
> | **贴**（标签） | 给患者下定性判断 | "你在否认" |
> | **站**（队） | 在患者人际冲突中选边 | "你妻子是不是因此不高兴？" |
> | **跳**（阶段） | 还没建立觉察就开方案 | 建议减量 / 参加 AA |
> | **上**（向上问） | readiness ruler 向上问 | "怎样能到 5？" |
>
> **唯一正道**：非评判 + 开放式提问 + 把行为连到患者价值观 + readiness ruler 向下问。

### 「讲」陷阱升级 — 疑问句式威胁（最隐蔽变体）

> [!danger] 疑问句形式的威胁，依然是威胁
> "讲后果"坑不止有陈述句，还会**伪装成疑问句**，更难识别：
> - 陈述式：「继续喝会肝损伤」/「不改风险很高」← 容易认出
> - **疑问式**：「**你不担心**会得肺癌/心脏病**吗？**」← 隐蔽，常被误选
>
> 两者本质相同：**用恐惧 / 让患者去想"坏结果"来推动改变**——这违背 MI（MI 让患者想**自己要的好结果**，不是怕坏结果）。

#### 反射规则（更新版）

> [!warning] MI 题排除标准
> 凡是让患者**去想"坏结果"**的选项，**一律排除**，无论句式：
>
> | 形式 | 例 | 判定 |
> |---|---|---|
> | 陈述式威胁 | "不改会失明、肾衰" | ❌ 讲后果 |
> | 命令式威胁 | "你必须现在就戒" | ❌ 讲后果 + 命令 |
> | **疑问式威胁** ⭐ | "你不担心早早得癌症吗？" | ❌ 讲后果（隐蔽变体） |
>
> **正道反向**：MI 永远让患者想**自己想要的好结果**——
> "如果有一天你决定戒，会是出于什么原因？" / "如果戒掉了，生活会有什么不同？"

#### 来源

- 2026-05-20 MI 巩固练习第 1 题：52 岁吸烟男（precontemplation），误选 E「你不担心会得肺癌/心脏病吗？」= 疑问句式威胁。正确 C = 非评判认可 + 开放式提问引向患者自己的改变理由。
- 关联：[[mistakes/uworld-mistakes#^Q11936]] / [[mistakes/uworld-mistakes#^Q21169]]（同一"讲后果"反射，连错根源）

## Part 5. 解题 SOP（5 步）

> [!tip] 对话术题 5 步流程
> 1. **判阶段**：患者 unconcerned / 否认？矛盾？想改？正在改？
> 2. **定目标**：当下这一步该提觉察、强化动机、还是给方案？
> 3. **排陷阱**：用"讲 / 贴 / 站 / 跳 / 上"五字诀秒删错误选项。
> 4. **选正道**：剩下的选项里挑"非评判 + 提问 + 患者主导 + 连到价值观"的那个。
> 5. **验方向**：MI 永远"向下挖理由"，不"向上指差距"、不"讲后果说服"。

## Part 7. Memory Hook

> [!success] 一句话锁定
> **"判阶段 → 排五坑（讲贴站跳上）→ 选提问。MI 永远向下挖理由，绝不讲后果。"**

> [!tip] 类比挂钩
> - **MI = 心理咨询师，不是说教老师**：说教(讲)=失败老师，下判断(贴)=法官，选边(站)=拉偏架，没诊断就开药(跳)，盯着空杯子(上)。只有"温和地问一个让患者自己醒悟的问题"才对。
> - **Readiness Ruler = 半杯水**：永远问"为什么已经有 2 分"（看已有的水），不问"为什么还差 3 分"（盯空的）。

> [!info] 口诀 + 反向记忆
> - **正向口诀**：判阶段 → 排五坑 → 选提问。
> - **五坑口诀**：讲 / 贴 / 站 / 跳 / 上。
> - **反向记忆**（Peixuan 专属反射纠正）：我两道 MI 题都选了 **B = 讲后果/威胁**。考场看到 MI + "如果不改后果严重"类选项 → **闭眼排除**。

# 自杀风险评估与降险 — USMLE 高频集成

> [!info] 用途
> Psych + Ethics 双考点重叠区。覆盖：① 风险因子识别 ② 急诊评估三要素 ③ 药物降险（4 类有证据）④ 治疗 / 程序（ECT/CBT/DBT）⑤ 系统层（safety planning / means restriction / 非自愿住院）⑥ USMLE 题干 → 答案对照。

---

## 一、最强单一风险因子 ⭐（Q2517 核心）

**既往自杀未遂 = 单一最强预测因子**（past behavior predicts future）。
任何题干提"prior suicide attempt" → 风险等级立即升高。

| 高风险因子（多个叠加 = 风险倍增）| 备注 |
|---|---|
| **既往自杀未遂** ⭐ | 单一最强 |
| 男性（已遂率高，未遂率女性高）| |
| 老年（>65）/ 青少年 | 两端 |
| 精神病史（MDD、Bipolar、Schizophrenia、BPD、SUD）| |
| **物质 / 酒精滥用** ⭐ | 巨大可改变风险因子 |
| **firearm access** ⭐ | 美国 / USMLE 必问 |
| 慢性疼痛 / 慢性病 | 可改变 |
| 社会孤立、近期丧失（亲人 / 工作 / 关系）| |
| 家族自杀史 | |
| 严重 anhedonia / hopelessness | |

> [!warning] NSSI ≠ Suicide attempt（Q2517 易混）
> Non-Suicidal Self-Injury（割伤等）= **无死亡意图**，调节负性情绪，常见于 BPD / 进食障碍。
> Suicide attempt = **有死亡意图**（深伤 / 可致死手段 / 计划性）。
> 二者可共存，处置按急性自杀风险驱动。

## 二、急诊评估三要素 → 住院门槛（Q2360 / Q4877 核心）

```
SI 评估三要素：
   ① Ideation 意念    ─ 有想法吗？
   ② Plan 计划        ─ 具体方法？
   ③ Intent 意图      ─ 真的想做吗？
            ↓
住院门槛 = 三要素齐全 + 急性危险
```

| 情境 | 处置 |
|---|---|
| 仅 ideation（无 plan / intent）| 风险评估 + 加强随访 + safety planning + 治原发病 |
| ideation + plan + intent（齐全）| **非自愿住院**（safety > autonomy）|
| 未成年 + 急性危险 + 家长拒绝住院 | **可越过家长同意**非自愿住院（Q4877 核心）|
| 免责书 release of liability | **不解除** duty to protect → 仍住院 |
| 中毒暂无法沟通 | 留观至清醒再评估（仅此情形适用"过夜"）|

> [!danger] Q4877 教训
> 急性自害风险的未成年人 → 父母签免责书 + 承诺看护 = **错答案**。正解：**非自愿住院**（Safety > Autonomy；免责书不解除医生 duty to protect）。

## 三、药物干预（4 类有证据） ⭐⭐⭐

| 药 | 适应人群 | 证据强度 | 关键 |
|---|---|---|---|
| **Lithium** ⭐⭐⭐ | **双相 + 复发性单极抑郁** | **最强**（~5 倍降率）| **独立于稳心境** —— 这是 Lithium 在抑郁谱系里的"独门附加值"。题干强调"自杀风险"+"双相倾向" → Lithium 被点名 |
| **Clozapine** ⭐⭐⭐ | **精分 / 分裂情感障碍 + 复发性自杀** | FDA 特批此适应症 | **唯一**有"reducing recurrent suicidal behavior" FDA 适应症的抗精神病药 |
| **Esketamine** ⭐⭐ | **TRD + MDD with acute SI** | 快速起效（小时 - 72h）| 详见 [[完整笔记/专题笔记/psych/psych_快速抗抑郁药_Esketamine]] |
| **SSRI / 抗抑郁药** ⭐ | 任何 MDD（成人）| **间接** —— 治好 MDD 即降险 | ⚠️ <25 黑框警告 = 念头 / 行为短期↑（非 completed suicide）；长期降险（Q7289 核心）|

> [!tip] Lithium 在抑郁题里 = "降自杀 / 双相暗示"信号
> 抑郁题选项里出现 Lithium 而你诊断的不是双相 → 出题人可能在暗示：① 强调抗自杀（独立机制）② 隐藏的双相倾向。

## 四、治疗 / 程序

| 干预 | 适应 | 关键 |
|---|---|---|
| **ECT** ⭐⭐⭐ | 重度 MDD + 急性 SI / 拒食 / 妊娠 / 难治 / 精神病性 | 起效**天级**；无绝对禁忌；详见 [[mistakes/uworld-mistakes_2026-05#^Q15811]] |
| **CBT（含 CBT-SP）** ⭐⭐ | MDD / 任何 SI 患者 | CBT for Suicide Prevention 降未遂 ~50% |
| **DBT** ⭐⭐ | **BPD + 自伤 / 自杀** ⭐ | BPD 专属一线（详见 [[mistakes/uworld-mistakes_2026-05#^Q16064]] 心理治疗五选）|

## 五、系统层 / 急诊处置 ⭐

| 干预 | 关键 |
|---|---|
| **Safety planning intervention** ⭐ | 急诊离院前**必做**；六步模板 |
| **Means restriction** ⭐⭐ | 尤其**枪支**（美国 USMLE 必问）—— 单一最有效的人群层措施 |
| **非自愿住院** | 急性 SI + plan + intent 三联齐全（Q4877）|
| **Caring contacts** | 出院后电话 / 邮件随访降复发未遂 |
| **治原发病 + 戒物质** | naltrexone（AUD）/ buprenorphine（OUD）间接降险 |

### 5.1 Safety Planning 六步模板（记 USMLE 高频选项）

```
1. Warning signs        识别即将复发的警告信号
2. Internal coping       自助技巧（深呼吸 / 转移注意 / 物理活动）
3. Social contacts      社交联系人（不一定谈危机，只为分心）
4. Professional help     危机时联系人（治疗师 / 危机热线 988）
5. Means restriction     限制致命手段（枪支 / 药物锁起来）
6. Reasons to live       理由清单（家人 / 宠物 / 信念）
```

### 5.2 Means Restriction 重点

- **枪支** = 美国自杀的主要手段（>50%）；任何 SI 患者**必问** "do you have access to firearms?"
- **答案**：让家人 / 朋友保管；储物锁；上交警局；卖掉
- 药物 stockpile 也要清（尤其 TCA / opioid / acetaminophen）
- **限制致命手段是单一最有效的群体干预**（比任何药都强）

## 六、USMLE 题干钩子 → 答案对照表 ⭐

| 题干描述 | 选什么 |
|---|---|
| 双相 + 反复自杀 | **Lithium** |
| 精分 / 分裂情感 + 反复自杀 | **Clozapine** |
| 重度 MDD + 即将自杀 / 拒食 / 妊娠 / 精神病性 | **ECT** |
| MDD + acute SI **需快速** + 不愿 / 不能 ECT | **Esketamine** |
| BPD + 反复自伤 + 不稳定人际 | **DBT** |
| 任何 MDD + SI | **CBT + SSRI**（治原发病）|
| ED 离院前 / 门诊随访 | **Safety planning + means restriction**（尤其问枪支）|
| 急性 SI + plan + intent 齐全 | **非自愿住院** |
| 抑郁少女母亲担心"抗抑郁药致自杀" | "**小幅增加 <25 自杀念头 / 行为；继续治疗 + 密切监测**"（Q7289）|
| 未成年 + 急性自害 + 父母拒绝住院 | **非自愿住院**（越过家长同意）（Q4877）|

## 七、易混陷阱

| 陷阱 | 真相 |
|---|---|
| "Antidepressants cause suicide" | **不全错**：<25 短期念头 / 行为↑（黑框）；长期治疗整体**降**完成自杀率。沟通题正确话术 = 承认小风险 + 继续治疗 + 密切监测（Q7289）|
| "签了 release of liability 可以让她回家" | ❌ 免责书不解除 duty to protect。急性自害风险 → 仍住院（Q4877）|
| "NSSI = suicide attempt" | ❌ NSSI 无死亡意图。但 SI 评估仍照做 |
| "Bipolar 抑郁用 SSRI 单药" | ❌ 诱发转躁。Lithium / quetiapine / lurasidone（你 Q15849 核心）|
| "口服 Lithium / Valproate 救急性激越" | ❌ 急性激越 = IM antipsychotic ± BZD（Q15856 / Q2355）|
| "Antipsychotic 任何抗精神病药都降自杀" | ❌ **只有 clozapine** 有此 FDA 适应症 |

## 八、Memory Hook

- **"5 大降自杀工具"**：Li（双相）/ Clozapine（精分）/ ECT（重度急迫）/ Esketamine（快）/ DBT（BPD）
- **3 大系统层措施**：safety planning + **means restriction（特别枪支）** + 非自愿住院（急性）
- **既往自杀未遂 = 单一最强预测因子**（其他都是叠加）
- **沟通题黑框警告答案**：「念头 / 行为 + <25 + 小幅 + 治疗 + 监测」五要素（Q7289）
- **NSSI vs Suicide attempt**：意图，不是手段；NSSI 在 BPD 高频
- **未成年急性自害 + 家长拒绝**：**非自愿住院**越过家长同意；免责书不豁免医生（Q4877）
- **K 字头 = Quick**：esketamine 几小时起效，TRD 长期 / MDD-SI 桥接（详见对应 esketamine 衍生）

## 一、核心鉴别原则（做题反射）

> [!tip] 一句话锁定
> **先问 stem 给的是「动作」还是「借口」：**
> - **动作** = 病人在**做什么**（研究 / 分析 / 帮助 / 转移情绪 / 退行）→ 多为 **不成熟 / 中性 / 成熟机制**
> - **借口** = 病人**为何这么做**（解释 / 找理由 / 合理化失败）→ **Rationalization**

**做题 3 步**：

```
Step 1：识别情绪源（病人在回避什么感受？— 死亡/失败/虚荣/嫉妒/敌意）
Step 2：识别动作方向（用什么方式回避？— 知识/借口/转移/相反/退行/分裂）
Step 3：对号入座（查 §四 6 组高频对照，排掉同动作邻居）
```

## 二、10 大不成熟防御机制速查

| 机制 | 一句话定义 | 经典 stem |
|---|---|---|
| **Acting out** | 用**冲动行为**表达不可接受的情绪 | 青少年丧亲后打架 / 翘课 / 飙车 |
| **Denial** | 行为像"现实不存在" | 拒承认癌症诊断 / 不去看医生 / "I'm not sick" |
| **Displacement** | **情绪转移**到更安全对象 | 被老板骂 → 回家踢狗；对配偶生气 → 对孩子吼 |
| **Intellectualization** ⭐ | 用**事实/分析**回避情绪 | **医生研究自己的癌症**；焦虑病人列疾病清单 |
| **Passive aggression** | 用拖延 / 沉默 / 不合作**隐性表达敌意** | 答应做但故意拖到 deadline 后；冷战 |
| **Projection** | 把**自己的感受/动机**投射到别人 | 自己想出轨 → 指控配偶不忠；自己嫉妒 → 说别人嫉妒 |
| **Rationalization** ⭐ | 用**借口/理由**合理化行为或失败 | 没考上医学院 → "反正我不想做医生"；外遇 → "妻子不理解我" |
| **Reaction formation** | 表现成**相反方向**（冲动本身被否认）| 暗恋的人 → 故意冷漠刻薄；讨厌孩子 → 过度溺爱 |
| **Regression** | **退回早期发展**阶段 | 住院成人开始尿床 / 吸吮拇指；压力下变婴儿语 |
| **Splitting** | **非黑即白**（BPD 经典；可伴 **transient paranoia / dissociation**）| "你是天使" → "你是恶魔" 来回切换；BPD 高压下短暂偏执 |
| **Compensation** ⚠ | **用一个领域的成就补偿另一领域的失败**（Neurotic 中性机制）| DUI 后过度做义工；学业差 → 拼命健身 / 玩乐器 |

> [!warning] Compensation 分类备注
> Compensation 严格属 **Neurotic（中性）** 机制（Vaillant 分类），介于不成熟与成熟之间 — 不像 sublimation 那样把冲动转化为社会贡献，而是**侧重于"代偿失败"**。USMLE 常与 sublimation / reaction formation 混淆：
> - **Compensation**：失败的 A → 拼命做 B（领域不同；为掩盖 A 的失败感）
> - **Sublimation**：不可接受的冲动 X → 社会可接受出口 X'（**同一冲动**的变形释放）
> - **Reaction formation**：冲动 X → 做相反的 -X（否认 X 本身）

## 三、4 大成熟防御机制速查

| 机制 | 一句话定义 | 经典 stem |
|---|---|---|
| **Sublimation** | 把**不可接受的冲动**转化为**社会可接受**的行为 | 攻击性 → 拳击手；性冲动 → 创作艺术；愤怒 → 跑马拉松 |
| **Suppression** | **有意识**地把情绪暂时搁置以应对现实 | "我现在不想，等手术做完再哭"；考试前先不想分手 |
| **Humor** | 用**幽默**化解痛苦 | 临终病人开自己的玩笑；丧亲后写笑话 |
| **Altruism** | 通过**服务他人**化解自身痛苦 | 失去孩子的母亲创办公益基金；癌症幸存者做支持组织 |

> [!info] 成熟 vs 不成熟
> 成熟 = 既释放情绪又**符合社会规范**，**不扭曲现实**；不成熟 = 通过**扭曲现实**或**伤害他人**来缓解焦虑。

## 四、6 组高频混淆对照（核心交付，背它）⭐

### 4.1 Intellectualization vs Rationalization（最高频）

| 维度 | **Intellectualization** 智化 | **Rationalization** 合理化 |
|---|---|---|
| **核心动作** | 用**事实 / 数据 / 分析**包裹现状 | 用**借口 / 理由**解释行为或失败 |
| **要避开什么** | **感受情绪**（恐惧/悲伤/无助）| **承认真实动机**（虚荣/嫉妒/失败/不当行为）|
| **时间方向** | 进行中：用知识分心 | 事后：把已发生的事合理化 |
| **承认事实吗** | ✅ 承认（"I have cancer"）只是不感受 | ❌ 扭曲（"我不是失败，是我不在乎"）|
| **首字母钩** | **I**nformation 信息 | **R**easons 理由 |
| **典型 stem** | 病人**研究 chemo / 分析检查 / 讨论 5 年生存率** | "I can't rest **because** patients need me" / "rest **wouldn't help anyway**" |

**反射钩**：「智化 = 承认事实，回避感觉；合理化 = 扭曲解释，回避真相。」

### 4.2 Suppression vs Repression

| 维度 | **Suppression** 抑制（成熟）| **Repression** 压抑（不成熟）|
|---|---|---|
| **意识层面** | **有意识**地搁置 | **无意识**地遗忘 |
| **能否调出** | ✅ 想想就能调出 | ❌ 完全想不起来 |
| **典型 stem** | "I'll deal with this **after** the surgery" | 童年虐待 → 成人**完全遗忘** / 创伤性记忆缺失 |
| **首字母钩** | **S**uppression = **S**aving for later 暂存 | **R**epression = **R**emoval 移除 |

### 4.3 Sublimation vs Reaction formation

| 维度 | **Sublimation** 升华 | **Reaction formation** 反向形成 |
|---|---|---|
| **真实冲动** | **保留**冲动，导向社会可接受出口 | **否认**冲动，做相反行为 |
| **行为来源** | 真冲动的**变形释放** | 真冲动的**对立面表演** |
| **典型 stem** | 攻击性 → 当**拳击手**（释放攻击欲）| 暗恋同事 → 故意**冷漠刻薄**（否认爱意）|
| **如何区分** | 行为**符合**冲动本质（暴力 → 暴力运动）| 行为**反对**冲动本质（爱 → 恨）|

### 4.4 Displacement vs Projection

| 维度 | **Displacement** 转移 | **Projection** 投射 |
|---|---|---|
| **转移的是** | **情绪**（生气 / 怨恨）| **自己的感受 / 动机**（认为是别人的）|
| **方向** | 自己 → 更安全对象 | 自己内在 → 贴到别人身上 |
| **典型 stem** | 被老板骂 → **踢狗** / 吼孩子（情绪转去更弱者）| 自己想出轨 → **指控配偶**不忠（动机贴别人）|
| **首字母钩** | **D**isplacement = **D**irection change 方向换 | **P**rojection = **P**aste on others 贴别人 |

### 4.5 Denial vs Repression

| 维度 | **Denial** 否认 | **Repression** 压抑 |
|---|---|---|
| **知道这件事吗** | ✅ 知道，但**拒绝接受**（"I'm not sick"）| ❌ **完全遗忘**（被埋的记忆）|
| **意识层面** | 部分意识（边界模糊）| 完全无意识 |
| **典型 stem** | 癌症诊断 → "Tests are wrong / 我没病" | 童年性虐待 → 成人完全不记得 |

### 4.6 Altruism vs Reaction formation（"善行"鉴别）

| 维度 | **Altruism** 利他 | **Reaction formation** 反向（伪善）|
|---|---|---|
| **动机真伪** | 真正**化解自身痛苦**的服务 | 用善行**掩饰相反冲动** |
| **行为感受** | 满足、平静、有意义 | 强迫性、过度、紧绷 |
| **典型 stem** | 失去孩子的母亲 → 创办**儿童公益基金** | 讨厌孩子的母亲 → **过度溺爱**到窒息 |
| **如何区分** | 痛苦经历 → 帮**同病相怜**他人 | 真冲动是**敌意**，行为是过度的**好意** |

### 4.7 Transference vs Countertransference（医患关系专属）

> [!info] 严格说不算"防御机制"
> Transference / Countertransference 属于**心理动力学概念**，不是经典 defense mechanism，但 USMLE 常与防御机制同框出题（医患沟通题、Psychiatry 行为科学题），归在此节统一鉴别。

| 维度 | **Transference** 移情 | **Countertransference** 反移情 |
|---|---|---|
| **主体** | **病人** 把过去关系的情感转移到**医生**身上 | **医生** 把自己的情感投射到**病人**身上 |
| **方向** | 病人 → 医生 | 医生 → 病人 |
| **典型 stem** | 病人对女医生说"You remind me of my mother" / 突然依赖 / 突然敌意 | 医生对某病人**特别想保护 / 特别愤怒 / 特别想避开** |
| **临床应对** | 识别 + **不强化**（不顺着病人投射演下去）；用于理解病人核心关系模式 | 医生**自我觉察 + 寻求督导**；不让自己情感干扰临床判断 |
| **首字母钩** | **T**ransference = T 在前（**患者主动**）| **C**ountertransference = **C**linician 反过来 |

**与其他防御机制的关系**：
- Transference 类似 病人的 **Projection**（投射动机），但特定指向**医生**这个对象
- 强烈 countertransference 提示医生自己可能有**未解决的情感问题**（如对某类病人反复有强烈反应 → 督导）

**经典 USMLE stem 三件套**：
1. 病人说 "You remind me of someone in my past" / "I trust you like family" / "I hate you like I hate my father" → **Transference**
2. 医生发现自己**特别想给某病人多花时间** / **反常地讨厌某病人** / **想约病人吃饭** → **Countertransference**
3. 医生发现自己对某病人产生**性 / 浪漫感受** → **Erotic countertransference**（**绝对禁忌跨越界限**，必须立刻寻求督导 + 不可双重关系）

## 五、USMLE 反射钩（看到这些 stem → 立刻锁机制）

| Stem 关键词 | 立刻锁定 |
|---|---|
| **医生 / 科学家 / 工程师**病人 + 用专业知识看自己的病 + "research / analyze / discuss survival rates" | → **Intellectualization** ⭐（[[mistakes/uworld-mistakes_2026-05#^Q3536]] 教科书 stem）|
| **学生考砸 / 被甩 / 没升职 / 没拿到 offer** + 给体面解释（"反正我也不想 / 那个机构 sucks / 她也没那么好"）| → **Rationalization**（酸葡萄）|
| **父母虐待 / 体罚孩子** + 解释「**为了孩子好 / 我小时候也这样**」 | → **Rationalization**（不当行为合理化）|
| 焦虑病人 + 上网查 WebMD / 列疾病清单 / 给医生看 spreadsheet | → **Intellectualization**（知识当焦虑盾牌）|
| **暴力倾向**青少年 → 当**拳击手 / 橄榄球**；**性冲动**青少年 → **创作艺术** | → **Sublimation** |
| 临终病人 + **开自己玩笑** / "至少我不用再交税" | → **Humor** |
| 化疗病人："**等疗程做完**我再考虑离婚" | → **Suppression** |
| 痛失亲人 → **创办公益**帮同病相怜者 | → **Altruism** |
| 配偶外遇被发现 → **指控对方先出轨** / 自己嫉妒 → **说别人嫉妒** | → **Projection** |
| **被老板骂** → 回家**吼孩子 / 踢狗** | → **Displacement** |
| 被讨厌的人在场 → **过度热情 / 恭维** | → **Reaction formation** |
| 住院压力下 + 成人**尿床 / 吸吮拇指 / 婴儿语** | → **Regression** |
| 童年创伤 → 成人**完全不记得** | → **Repression**（无意识）|
| 癌症诊断 → "**Tests are wrong / 我没病**" + 拒做后续检查 | → **Denial** |
| BPD 病人 → "**你是天使**" → 几小时后 "**你是恶魔**" | → **Splitting** |
| 答应做某事 → **故意拖延 / 装忘记 / 阳奉阴违** | → **Passive aggression** |
| 青少年丧亲 → **打架 / 翘课 / 飙车** | → **Acting out** |
| **DUI 后 / 学业失败后** → 拼命做义工 / 拼命健身 / 钻研另一爱好补偿失败感 | → **Compensation**（Neurotic 中性）|
| 病人对医生说 "**You remind me of my mother / father**" / 突然依赖或敌意（投射过去关系）| → **Transference**（病人 → 医生）|
| 医生发现自己**特别想保护 / 特别讨厌 / 特别想多花时间**某病人 | → **Countertransference**（医生 → 病人）|
| 医生对病人产生**性 / 浪漫感受** | → **Erotic countertransference** ⚠ 立刻寻求督导 + 禁双重关系 |

# MDD 治疗抵抗（TRD）阶梯 + 三大特殊亚型（atypical / melancholic / psychotic）治疗偏好

> [!info] 一句话定位
> 抑郁治疗反复错的核心 = **没区分 MDD 亚型 → 走错阶梯**。本笔记按"标准 TRD 阶梯 + 3 大特殊亚型偏好"组织：① 标准 MDD → SSRI/SNRI → bupropion/mirtazapine → augment → MAOI → ECT ② **atypical** → 早期偏 bupropion/mirtazapine，TRD 阶梯 **MAOI 经典专属** ③ **melancholic / severe** → TCA / ECT 选项更突出 ④ **psychotic** → SSRI + SGA 双药 → ECT。**用户 5-23 浩斯宾 Q15 + 5-24 浩斯宾 Q24 连续两次错 atypical → MAOI 阶梯**，触发本衍生。

> [!warning] 用户反复错家族
>
> | 次数 | 日期 | 题号 | 错点 | 选 | 正解 |
> |---|---|---|---|---|---|
> | 1 | 2026-05-23 浩斯宾 Q15 | 生成题 | atypical MDD 4 药失败 → ？ | lithium augmentation | **phenelzine (MAOI)** |
> | 2 | 2026-05-24 浩斯宾 Q24 | 生成题 | 同题型再问 | lithium augmentation | **phenelzine (MAOI)** |
>
> + [[mistakes/uworld-mistakes_2026-05#^Q15925]] atypical 诊断角度（5-21）

## 一、标准 MDD 治疗阶梯（适用于大多数 MDD）

```
═══════════════════════════════════════════════════════════════
                  标准 MDD 治疗阶梯（无特殊亚型）
═══════════════════════════════════════════════════════════════
Step 1：SSRI（一线 — sertraline / fluoxetine / escitalopram 等）
         或 SNRI（venlafaxine / duloxetine — 同等一线）
         足量足程 ≥ 4-6 周
         │ 部分应答 / 无应答
         ▼
Step 2：优化（增加剂量到 max tolerated）
         │ 仍不够
         ▼
Step 3：换药
         ├─ 换另一 SSRI / SNRI（同类）
         ├─ 换 bupropion（NDRI — atypical / 性功能保留偏好）
         └─ 换 mirtazapine（NaSSA — 失眠 / 食欲↓ 偏好）
         │ 仍不够
         ▼
Step 4：Augmentation（增效）
         ├─ Lithium（经典 TRD 增效）
         ├─ T3 / T4（thyroid augmentation）
         ├─ Aripiprazole / quetiapine（SGA augmentation — FDA 批 MDD 增效）
         └─ Bupropion（如未用 — 加在 SSRI 上）
         │ 仍不够
         ▼
Step 5：换 MAOI（phenelzine / tranylcypromine）⭐
         需要 washout 期：
         • SSRI 停 2 周 → MAOI
         • Fluoxetine（长半衰期）停 5 周 → MAOI
         需要忌口（tyramine 食物）
         │ 失败
         ▼
Step 6：神经调控 / 难治性
         ├─ ECT（金标准，起效最快，但需麻醉）
         ├─ TMS（重复经颅磁刺激 — 门诊可做，证据较弱）
         ├─ Esketamine（鼻喷 — FDA 批 TRD，需诊室监测）
         ├─ Ketamine IV（off-label，起效数小时）
         └─ DBS / VNS（实验性 / 严选病例）
═══════════════════════════════════════════════════════════════
```

### TRD（Treatment-Resistant Depression）定义
**至少 2 种不同类别抗抑郁药足量足程治疗失败** = TRD。

## 二、三大特殊亚型治疗偏好（USMLE 高频考点）⭐

### 2.1 Atypical MDD（非典型抑郁）⭐ 用户反复错

**诊断要件**（[[mistakes/uworld-mistakes_2026-05#^Q15925]]）：
- **Mood reactivity（心境反应性）** — 能对正性事件高兴 ⭐ 必有
- ≥ 2 项：① 食欲 / 体重↑ ② Hypersomnia 嗜睡 ③ Leaden paralysis 铅样麻痹 ④ Interpersonal rejection sensitivity 拒绝敏感

**治疗阶梯**（atypical 偏好）：

```
Step 1：SSRI / SNRI（仍是一线 — 大多数 MDD 起手）
         │ 失败
         ▼
Step 2：Bupropion 或 Mirtazapine（atypical 偏好早登场）⭐
         理由：atypical 三联（嗜睡 / 增重 / leaden paralysis）→ bupropion 抗嗜睡 + 不增重；
              mirtazapine 改善失眠但增重（适合**已失眠**型 atypical）
         │ 失败
         ▼
Step 3：Augment（Li / T3 / aripiprazole）— 通用 TRD 选项
         │ 失败
         ▼
Step 4：MAOI（phenelzine / tranylcypromine）⭐ ATYPICAL 经典专属
         历史上 MAOI 在 atypical MDD 的有效率显著高于 TCA
         "愿尝试经典药" + atypical → MAOI 钩 ⭐ USMLE 高频
         │ 失败
         ▼
Step 5：ECT / TMS / Esketamine
```

**MAOI 关键点**（背熟）：

| 维度 | 内容 |
|---|---|
| 代表药 | **Phenelzine**（最常考）/ Tranylcypromine / Isocarboxazid / Selegiline 透皮（高剂量）|
| 机制 | 不可逆 MAO-A + MAO-B 抑制 → 5-HT / NE / DA 降解↓ → 突触间神经递质↑ |
| Atypical 适应症 | **经典专属**（vs TCA 历史对照证据强）|
| **Washout 时间** | SSRI 停 **2 周** → MAOI；**Fluoxetine 停 5 周**（半衰期长）→ MAOI；停 MAOI ≥ 2 周才能用 SSRI |
| **忌口（tyramine HTN crisis）** | 陈年奶酪 / 红酒 / 腌肉 / 发酵豆 / 熟透水果 / 啤酒 / 酱料 — 见 [[草稿/2026-05-23_NMS_MH_Serotonin三联鉴别_专题草稿_v1]] §二 / [[mistakes/uworld-mistakes_2026-05#^Q4879]]|
| **5-HT 综合征禁联** | SSRI / SNRI / TCA / tramadol / meperidine / linezolid（弱 MAOI）/ DXM / 曲坦 / methylene blue |
| 副作用 | 直立性低血压 / 失眠 / 体重↑ / 性功能障碍 |
| HTN crisis 解药 | **Phentolamine（α 阻断）/ Nitroprusside / Labetalol（α+β）**；**禁单用 β 阻断**（α 失对抗→血压更高）|

**USMLE Atypical → MAOI 三大暗号**：
1. **"willing to try a more classical / older medication"** ⭐
2. **"informed about dietary restrictions"** ⭐
3. **"history of failed multiple antidepressants" + atypical features** ⭐

---

### 2.2 Melancholic MDD（忧郁型）

**诊断要件**：
- **Pervasive anhedonia**（彻底失乐 — 无 mood reactivity）⭐ 与 atypical 相反
- ≥ 3 项：① 晨重暮轻 ② 早醒（凌晨）③ 显著精神运动 retardation/agitation ④ 显著食欲 / 体重↓ ⑤ 过度内疚

**治疗阶梯**（melancholic 偏好）：

```
Step 1：SSRI / SNRI（仍一线）
         │ 严重 / 失败
         ▼
Step 2：TCA（historic preference — 比 SSRI 在严重 melancholic 中证据稍强）
         或 Venlafaxine（高剂量 ≥ 225 mg → NE 显著激动）
         │ 失败
         ▼
Step 3：Augment（lithium / T3 / SGA）
         │ 失败
         ▼
Step 4：MAOI（备选，不如 atypical 显著）
         │ 失败
         ▼
Step 5：ECT ⭐（severe melancholic 早登场 — 严重消瘦 / 拒食 / 紧张症 → 跳级到 ECT）
```

> [!tip] Melancholic vs Atypical 反向对比
>
> | 维度 | Melancholic | Atypical |
> |---|---|---|
> | Mood reactivity | ❌ 没有（pervasive anhedonia） | ✅ 必有 |
> | 食欲 / 体重 | ↓ | ↑ |
> | 睡眠 | 早醒 / insomnia | hypersomnia |
> | Lead paralysis | — | ✅ |
> | TRD 偏好 | TCA / **ECT 早跳级** | bupropion/mirtazapine / **MAOI 经典** |

---

### 2.3 Psychotic MDD（伴精神病性特征）⭐ 5-23 复盘 Q7

**诊断要件**：MDD + 妄想（多与抑郁主题一致 — guilt / nihilism / persecution）和/或幻觉

**治疗**（必须双药 — 单 SSRI 不够）：

```
Step 1：SSRI + SGA 双药 ⭐
         例：sertraline + olanzapine
              fluoxetine + quetiapine
              对照试验显示双药 > 单 SSRI 或单 SGA
         │ 失败 / 严重（拒食 / 即将自杀 / 紧张症）
         ▼
Step 2：ECT ⭐ 跳级
         适应症：① 起效快（即刻降自杀率）② 不依赖摄入（拒食/脱水病人）
                  ③ 紧张症
                  ④ 多药失败
                  ⑤ 妊娠期 / 老年（避免药物副作用）
```

> [!warning] 单药都不够
> Psychotic MDD 用 **单 SSRI / 单 SGA / 单 lithium / BZD / MAOI 都不够** — 必须 **SSRI + SGA** 或 **ECT**。[[mistakes/uworld-mistakes_2026-05#^Q15088]] 🔴 反复错。

## 三、亚型治疗偏好统一表 ⭐ 核心

> [!tip] 速记对照 ① —— 标准 MDD vs Atypical MDD：Divergence 在 Step 3 + Step 5 ⭐
>
> | 阶梯位 | 标准 MDD | Atypical MDD | 关键区别 |
> |---|---|---|---|
> | **Step 1 一线** | SSRI / SNRI | SSRI / SNRI（相同）| ✅ 一致 |
> | **Step 2 优化** | 加剂量 | 加剂量 | ✅ 一致 |
> | **Step 3 换药** | 换 SSRI / SNRI / bupropion / mirtazapine（平等）| **直接换 bupropion / mirtazapine**（早登场 — 反向利用副作用谱抗嗜睡 + 不增重 + 抗 leaden paralysis）⭐ | **第一处 divergence** |
> | **Step 4 Augment** | Li / T3 / SGA / bupropion | Li / T3 / aripiprazole | ✅ 大致一致 |
> | **Step 5** | MAOI（第 5 阶梯通用 TRD 选项）| **MAOI（phenelzine）专属**（第 4 阶梯就上 — atypical 历史 RCT 证据显著优于 TCA）⭐⭐ | **第二处 divergence — CK 高频错点** |
> | **终极** | ECT / TMS / Esketamine | ECT / TMS / Esketamine | ✅ 一致 |
>
> **3 大 atypical → MAOI 暗号（CK 高频）**：
> ① "willing to try a more **classical / older medication**" ⭐
> ② "informed about **dietary restrictions**" ⭐（tyramine 忌口）
> ③ "history of **failed multiple antidepressants**" + atypical features ⭐
>
> **生理逻辑**：Atypical 症状（嗜睡 / 增重 / leaden paralysis / 拒绝敏感）涉及**奖赏环路 DA 通路失调** → DA 能药（bupropion 直接覆盖 DA + MAOI 覆盖 5-HT/NE/DA 三联）更有效 → 这就是为什么 atypical TRD ≠ lithium augmentation（用户 5-23 / 5-24 反复错点）。
>
> **一句话钉死**：**"标准 MDD = SSRI 阶梯线性走；Atypical MDD = bupropion/mirtazapine 早登场 + MAOI 第 4 步专属（不要再回 lithium！）"**

> [!tip] 速记对照 ② —— 标准 MDD vs Melancholic MDD：2 个早跳级 + 1 个降权 ⭐
>
> Melancholic = **重症轴**（不是"轻型 MDD"），策略**更激进 + 更快上重武器**。
>
> | 阶梯位 | 标准 MDD | Melancholic MDD | 关键区别 |
> |---|---|---|---|
> | **诊断要件** | Pervasive 低落 + ≥ 5/9 SIGECAPS | **Pervasive anhedonia**（**完全不能高兴**，无 mood reactivity）⭐ + ≥ 3 项（晨重暮轻 / 早醒 / 显著精神运动迟滞或激越 / 体重↓ / 过度内疚）| **诊断更严** — 重症体征 |
> | **Step 1 一线** | SSRI / SNRI | SSRI / SNRI **或 TCA 早登场**（严重时）| **TCA 早登场** |
> | **Step 2 优化** | 加剂量 | 加剂量 | ✅ 一致 |
> | **Step 3 换药** | 换 SSRI / SNRI / bupropion / mirtazapine | **TCA / 高剂量 venlafaxine ≥ 225 mg**（NE 激动主导）⭐ | **偏 NE 能药** |
> | **Step 4 Augment** | Li / T3 / SGA / bupropion | Li / T3 / SGA | ✅ 大致一致 |
> | **Step 5 MAOI** | MAOI — 第 5 阶梯通用 TRD 选项 | MAOI（**备选**，**不如 atypical 显著**）| **降权**（用得更少）|
> | **终极** | ECT 第 6 阶梯（按部就班走完才考虑）| **ECT 严重时早跳级**（不必走完阶梯）⭐ | **ECT 早跳级** |
>
> **ECT 早跳级 5 大触发**（severe melancholic 任一就跳）：
> ① **严重消瘦 / 拒食**（不能口服，药物 4-6 周等不及）
> ② **紧迫自杀风险**（active SI + plan）
> ③ **紧张症 (catatonia)**（ECT 一线）
> ④ **MDD with psychotic features 重症**
> ⑤ 多药失败 / 妊娠 / 老年避药
>
> **生理逻辑**：Melancholic 症状（精神运动迟滞 / 早醒 / 食欲废）涉及 **NE 通路失调** → TCA + 高剂量 venlafaxine（5-HT + NE 双通道）比 SSRI 单通道更有效；ECT 起效最快（数天 vs 药物 4-6 周）→ 重症等不了。
>
> **一句话钉死**：**"标准 MDD = 阶梯线性走；Melancholic MDD = TCA 早登场 + ECT 严重时早跳级 + MAOI 不是首选（降权）。"**

> [!success] 速记对照 ③ —— 4 亚型治疗反射一表钉死 ⭐⭐ 最核心
>
> | 亚型 | 诊断关键 | 治疗反射 |
> |---|---|---|
> | **标准 MDD** | Pervasive 低落 + SIGECAPS（无特殊亚型特征）| **SSRI 阶梯线性走**（SSRI → 换药 → augment → MAOI → ECT）|
> | **Atypical**（能高兴）| **Mood reactivity** + 嗜睡 / 增重 / leaden paralysis / 拒绝敏感 | **Bupropion / Mirtazapine 早登场 → MAOI（phenelzine）第 4 步专属** ⭐ |
> | **Melancholic**（不能高兴）| **Pervasive anhedonia** + 晨重暮轻 / 早醒 / 显著消瘦 / 精神运动异常 | **TCA / 高剂量 venlafaxine 早登场 → ECT 严重时早跳级** ⭐ |
> | **Psychotic**（伴幻妄）| MDD + delusions / hallucinations（常 mood-congruent — guilt / nihilism / persecution）| **SSRI + SGA 双药 / ECT**（**单药都不够**）⭐ |
>
> **CK 高频 stem 暗号识别**：
>
> | Stem 关键词 | 锁亚型 | 反射答案 |
> |---|---|---|
> | "能对正性事件高兴 + 嗜睡 + 增重 + 灌铅腿 + 拒绝敏感" + failed 多药 + "willing to try classical / older medication" | **Atypical TRD** | **Phenelzine (MAOI)** ⭐ |
> | "彻底失乐 + 早醒 + 晨重暮轻 + 显著消瘦 + 过度内疚" + **严重 / 拒食 / 紧张症 / 紧迫 SI** | **Severe Melancholic** | **ECT 跳级** ⭐ |
> | "彻底失乐 + 早醒 + 体重↓" + 第 1 阶梯换药 | **Melancholic** | **TCA / 高剂量 venlafaxine**（不是再换 SSRI）|
> | "Guilt-themed delusions + auditory hallucinations + SIGECAPS" | **Psychotic MDD** | **SSRI + SGA 双药 / ECT** ⭐ |
> | 一般 pervasive 低落 + SIGECAPS + 无特殊亚型特征 | **标准 MDD** | 阶梯线性走（SSRI 起 + 4-6 周）|
>
> **反射钩 — 最高频错点防御**：
> - ❌ "Melancholic + atypical 治疗一样" → 两个**对立**：atypical 是 MAOI 专属，melancholic 是 TCA / ECT 偏好
> - ❌ "Severe melancholic 必须走完阶梯到 ECT" → **错** → 拒食 / 紧张症 / 紧迫 SI 就**跳级 ECT**
> - ❌ "Atypical TRD = lithium augmentation" → **错** → atypical 第 4 步 = **MAOI（phenelzine）**（用户 5-23 / 5-24 反复错点）
> - ❌ "Melancholic 首选 SSRI 就够" → 严重时 SSRI 单 5-HT 通道不够 → **早 TCA / 高剂量 venlafaxine**
> - ❌ "Atypical 早醒 + 食欲废" → 方向反了 → 早醒 + 食欲废是 **melancholic**；atypical 是**嗜睡 + 增重**
> - ❌ "Psychotic MDD 单 SSRI / 单 SGA 就够" → **错** → 必须**双药**（SSRI + SGA）或 **ECT**

### 完整统一表（4 亚型 × 阶梯位 — 速查用）

| 阶梯位 | 标准 MDD | Atypical | Melancholic | Psychotic |
|---|---|---|---|---|
| **一线** | SSRI / SNRI | SSRI / SNRI | SSRI / SNRI / **TCA** | **SSRI + SGA**（双药）⭐ |
| **二线（换药）** | 换另 SSRI / SNRI / bupropion / mirtazapine | **Bupropion / Mirtazapine** ⭐ | **TCA / 高剂量 venlafaxine** ⭐ | 换 SSRI + 换 SGA |

| **四线** | **MAOI** | **MAOI（经典专属）** ⭐⭐ | MAOI（备选） | ECT |
| **五线** | ECT / TMS / esketamine | ECT / TMS / esketamine | **ECT（早跳级）** ⭐ | （已上 ECT）|

## 四、特殊场景偏好（USMLE 高频钩）

### 4.1 孕期 MDD
- **轻-中度 → CBT 一线**（不上药）
- **中-重度 → SSRI**（sertraline 偏好 — 母乳 / 胎儿曝露最少证据）
- **禁 paroxetine**（致畸 — 心脏 / 神经管缺陷）
- **禁 valproate**（神经管缺陷）+ **禁 lithium 早孕**（Ebstein anomaly）
- **严重 / 拒食 / 紧张症 / 自杀 → ECT**（孕期安全）

### 4.2 老年 MDD
- **避免 TCA**（抗胆碱副作用 / 直立性低血压 / 跌倒）
- **避免 paroxetine**（强抗胆碱 + 高停药综合征）
- **偏好 SSRI**（sertraline / escitalopram — 副作用谱友好）
- **Mirtazapine** 优势：改善失眠 + 食欲（老年消瘦 / 失眠常见）

### 4.3 性功能保留 MDD
- **Bupropion**（NDRI — 无性功能副作用）
- **Mirtazapine**（次选）

### 4.4 共病焦虑 MDD
- SSRI / SNRI 仍一线
- 避免 bupropion 单药（NDRI 可激动焦虑）
- **Mirtazapine** 镇静辅助

### 4.5 共病慢性疼痛 MDD
- **Duloxetine（SNRI）** ⭐ 一线 — FDA 批 DPN / 纤维肌痛 / 慢性骨骼肌痛
- TCA（amitriptyline / nortriptyline）—— 但老年慎用

### 4.6 季节性 MDD（SAD）
- **Light therapy**（光疗）⭐ 一线
- + Bupropion XL 维持（FDA 批 SAD 预防）

## 五、Antidepressant 综合反应时间表

| 时间 | 改善什么 |
|---|---|
| 1-2 周 | 睡眠 + 食欲（最早 — 镇静 / 食欲改善）|
| 2-4 周 | 精力 / 注意力 / 精神运动 |
| 4-6 周 | **核心抑郁情绪 / SI 缓解** ⭐ |
| 6-12 周 | 完全缓解（remission）|

> [!warning] 不到 4-6 周不算"治疗失败"
> USMLE stem 给 "patient on sertraline for 2 weeks, no improvement" → **不是治疗失败**，应**继续到 4-6 周再评估**。给 "sertraline 8 weeks max dose, no improvement" → 真失败 → 换药 / augment。

### Antidepressant 自杀风险黑框警告
- **< 25 岁起始 SSRI** → 早期 SI 风险**轻度↑**（非 completed suicide）
- **抑郁本身的 SI 风险远 > SSRI 风险**
- 处理 = **治疗 + 密切随访**（不是不用药）

## 六、Antidepressant 转躁（药源性躁狂）⭐ 5-21 复盘

**Stem 模式**：MDD 起 SSRI 后数天 - 数周内出现 mania（精力↑ / 睡眠需求↓ / 多语 / 冲动 / 自大）

**机制**：揭示潜在 **bipolar spectrum**（之前未诊断的 bipolar）

**第一步**：
1. **立即停 SSRI**
2. **起 mood stabilizer**（lithium / valproate / lamotrigine / 二代抗精神病药）
3. 后续按 bipolar 维持治疗（4 选 1 — 见 [[mistakes/uworld-mistakes_2026-05#^Q2357]]）

## 七、易混陷阱（普适）

> [!warning] 高频陷阱
> - **"atypical = 不严重"反射 ❌**：mood reactivity（能高兴）是 atypical 的**诊断要件**，不是排除 MDD 的理由。看到"能对好事高兴 + 嗜睡 + 增重 + 拒绝敏感"→ atypical MDD 而不是"轻型 MDD"。
> - **"TRD = lithium augmentation"反射 ❌**：lithium augment 是**通用 TRD 选项**，不是 **atypical 特异**。atypical TRD 第 4 阶梯 = **MAOI 经典专属**（用户 5-23 + 5-24 反复错点）。
> - **"愿尝试经典药"暗号**：USMLE 把"愿试更老 / 经典 / classical medication" + atypical → 几乎等同 MAOI 钩。看到立刻锁 MAOI。
> - **"Psychotic MDD = 单 SSRI 或单 SGA"反射 ❌**：必须**双药（SSRI + SGA）**或**ECT**。单药都不够。
> - **"换 SSRI 立刻 = 治疗失败"反射 ❌**：≤ 4-6 周 + max tolerated dose 才算治疗失败。早换药 = USMLE 钓饵。
> - **"SSRI 转躁 = 加大 SSRI 剂量"反射 ❌**：转躁 = 揭示 bipolar → **停 SSRI + 上 mood stabilizer**，不是加 SSRI。
> - **"Bupropion 在 atypical = 永远首选"反射部分错**：bupropion 是 atypical 早期偏好（vs 标准 MDD），但**第 4 阶梯 TRD atypical = MAOI**，不是再回到 bupropion augment。阶梯位置看 stem 给的失败药数量。
> - **"MAOI + SSRI 联用"反射 ❌ 致命**：5-HT 综合征 / HTN crisis 风险，禁联。换药需 washout 期。
> - **"老年用 TCA 治 MDD"反射 ❌**：老年抗胆碱副作用 + 跌倒风险高，避免 TCA（除非有强适应症如疼痛共病且无替代）。
> - **"孕期用 paroxetine"反射 ❌**：paroxetine 致畸（心脏 / 神经管），孕期 SSRI 首选 sertraline。

## 八、Memory Hook

> [!success] 记忆挂钩
> - **一句话锁定 atypical 阶梯**：**"Atypical 三联（嗜睡 / 增重 / 灌铅腿）→ 早期偏 bupropion/mirtazapine → TRD 第 4 档专属 MAOI（phenelzine）→ 终极 ECT。"**
> - **MAOI atypical 钩**：**"愿试经典药 + atypical → MAOI"**（USMLE 高频暗号）
> - **三亚型治疗偏好口诀**：
>   - **Atypical**（能高兴）→ **MAOI** ⭐
>   - **Melancholic**（不能高兴）→ **TCA / ECT 早跳级** ⭐
>   - **Psychotic**（有幻妄）→ **SSRI + SGA 双药 / ECT** ⭐
> - **MAOI 三联鉴别记忆**：**"MAOI = 经典药 + 忌口（tyramine）+ 禁联 5-HT 能药"**。三点缺一不可。
> - **ECT 适应症 5 字**：**"快不食症孕老"** — 急需起效（自杀 / 紧张症）/ 拒食 / psychotic / 紧张症 / 孕期 / 老年避药。
> - **Antidepressant 反应时间口诀**：**"睡食 1-2 周，精力 2-4 周，情绪 4-6 周，完全 6-12 周。"** 没到 4-6 周别说失败。
> - **类比**：MDD 治疗阶梯像爬楼 — atypical 走 MAOI 楼梯（专门通道），melancholic 走 TCA/ECT 直升梯（重症优先），psychotic 必须双手扶栏（SSRI+SGA 双药）。

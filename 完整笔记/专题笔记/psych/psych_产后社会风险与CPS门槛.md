# 产后社会风险 → 支持优先 vs CPS 门槛

> [!info] 一句话锁定
> 面对**脆弱家庭背景**（父亲入狱、家族物质滥用/犯罪史）但**母婴当前安全、母亲无物质使用**时，最合适的下一步是**支持性随访（visiting nurse follow-up，访视护士随访）**，而不是启动 **CPS (child protective services, 儿童保护服务)** 等惩罚性/剥夺性处置。**社会风险 ≠ 当前虐待证据；家族行为问题 ≠ 遗传宿命。**

---

## 一、核心原则：当前危险证据 > 背景风险

NBME/USMLE 反复考的决策逻辑：

```text
脆弱家庭背景（社会风险）
  → 提示需要关注/支持
  → 但不等于"当前虐待或忽视"
  → 母亲现无物质使用 + 母婴当前安全
  → 提供支持 / 教育 / 连接资源（visiting nurse 最低侵袭、最有帮助）
  → 不做基于"未来可能性"的过度干预
```

**两个独立误区**：
1. **遗传决定论**：把"家族男性有攻击/物质滥用/被捕史"误读成"孩子必然遗传行为障碍"。常见行为问题（品行障碍、反社会人格）**以环境/社会因素为主，无一致遗传决定性**，不能据此预设孩子未来有精神障碍。
2. **风险=证据**：把"父亲涉毒入狱 + 家族史"直接等同于 child abuse risk（虐童风险）→ 误报 CPS。**缺的恰恰是"当前危险证据"。**

---

## 二、处置阶梯（社会风险 → 动作）

| 情境 | 最合适动作 |
|---|---|
| 正常产后 + 社会风险背景 + **无当前虐待/忽视证据** | **支持性随访（visiting nurse follow-up）** ✅ |
| 明确母亲物质使用 / 不安全家庭 / 忽视征象 / 婴儿有伤 | 考虑 **CPS（儿童保护服务）介入** |
| 只是担心未来行为"会遗传" | 咨询 (counseling) + 支持，不做基于预测的干预 |
| 新生儿当前查体无异常 | 不需预防性精神评估，更不需"4 岁时做精神评估"这类预设 |

---

## 三、CPS 触发门槛 = "当前证据"清单 ⭐

启动 CPS / 强制报告需要**当前**（不是"背景"或"未来担忧"）证据，任一即可：

- 母亲**正在**物质使用 / 酗酒影响照护能力
- 家庭暴力 / 不安全居住环境
- 忽视征象（喂养不足、就医延误、卫生极差）
- 婴儿身上有**不可解释的损伤**（可疑虐待）

> [!warning] 反模式
> 仅有"父亲入狱""家族犯罪/物质史""母亲贫困/低教育"等**背景脆弱性**——**不足以**触发 CPS。先支持、教育、连接资源。

---

## 四、Memory Hook

**"有风险先支持，没证据别升级"**
- 社会脆弱性 ≠ child endangerment（当前危害）
- 家族行为问题 ≠ 必然遗传给孩子
- 产后默认落点 = **visiting nurse 支持性随访**

## 一、诊断时长阈值（Psych timelines — 一图定生死）

> [!danger] 时长是精神病性 + 创伤 + 抑郁谱系最高频考点
> 题干给"持续多久" = 直接锁诊断；改一个数字就换答案。

```
精神病性谱系（症状相同，只比时长）
├─ Brief psychotic disorder 短暂精神病性障碍   < 1 个月（常由巨大压力诱发，完全恢复）
├─ Schizophreniform 精神分裂样障碍            1 – 6 个月
├─ Schizophrenia 精神分裂症                  > 6 个月（含 ≥6 个月功能下降）
└─ Schizoaffective 分裂情感性障碍             > 6 个月 + 必须有 ≥2 周「单纯幻觉/妄想、无情绪症状」

创伤谱系（生命威胁事件后）
├─ Acute stress disorder 急性应激障碍 (ASD)    < 1 个月
└─ PTSD 创伤后应激障碍                        > 1 个月

应激源相关
└─ Adjustment disorder 适应障碍              应激源 3 个月内起病，6 个月内缓解；> 6 个月 → 转 MDD / GAD

抑郁 / 心境慢病
├─ MDD 重性抑郁障碍                          ≥5 个 SIGECAPS（抑郁症状助记口诀）+ ≥2 周（<2 周 = acute reactive depression 急性反应性抑郁）
├─ Persistent depressive disorder 持续性抑郁（dysthymia 心境恶劣）   2 年
├─ Cyclothymia 环性心境障碍                   > 2 年
└─ Grief 正常哀伤                            可正常持续约 1 – 1.5 年

焦虑
├─ Panic disorder 惊恐障碍                    反复非预期发作 + 发作后 worry > 1 个月
├─ GAD 广泛性焦虑障碍                         担忧多领域 > 6 个月
└─ Delusional disorder 妄想障碍               妄想 > 1 个月（单一、貌似合理但不太可能）
```

> [!tip] Schizoaffective vs Mood disorder with psychotic features（高频鉴别）
> - 幻觉**只在**抑郁/躁狂发作时出现 → **Mood disorder with psychotic features**（心境障碍伴精神病性）
> - 有 **≥2 周只有幻觉/妄想、没有情绪症状** → **Schizoaffective**（分裂情感性）

🧩 精神病性 + 精神症状选药 5 步阶梯 → [[完整笔记/专题笔记/psych/psych_PD_psychosis_5步阶梯]]
🧩 创伤 / 应激源时间线鉴别 + 选对治疗 → [[完整笔记/专题笔记/psych/psych_精神障碍选对治疗SOP]]

## 二、情感障碍（Mood disorders）

### 2.1 MDD next-step 骨架

```
抑郁症状 → ① Best initial step = Rule out suicide（先评自杀，不反射开药）
                   ↓
       ② 一线治疗 = Therapy（心理治疗）+ SSRI
                   ↓
       ③ 有精神病性（妄想/幻觉）→ 抗抑郁药 + 抗精神病药（缺一不可）
```

> [!danger] MDD with psychotic features = 反复错重灾区 🔴
> 妄想**必须加抗精神病药 (antipsychotic)**；锂 (lithium) 不治妄想 = 经典诱饵。
> 钥匙：「抑郁 + 妄想 → SSRI/抗抑郁 **＋** 抗精神病药」。

> [!warning] 儿童 / 青少年抑郁 + 自杀意念 (suicidal ideation, SI)
> - Passive SI（"消失就好了"）vs Active SI（有计划/手段）必须区分
> - 精神科 5 步 SOP：**Safety → Stabilization → Diagnosis → Treatment → Follow-up**（不能跳）
> - 任何 SI + 高危（支持差/无随诊/应激源）→ 急诊精神评估，倾向住院
> - SSRI 有 **black box warning**（<25 岁开药后 1–2 周自杀风险最高）→ 评估安全前禁反射开药

### 2.2 双相障碍（Bipolar）

| 类型 | 核心 | 关键鉴别 |
|---|---|---|
| **Bipolar I 双相 I 型** | Mania 躁狂（± 抑郁）| 损害生活质量、racing thoughts |
| **Bipolar II 双相 II 型** | Hypomania 轻躁狂（± 抑郁）| **不**损害生活质量、pressured speech 但无 racing thoughts |

- **一线 = 锂 (lithium)**。
- 🔴 **Stable ≠ Stop**：双相稳定**不能**停药（维持治疗，反复错）。
- **Borderline (hours) vs bipolar (days–weeks)**：边缘人格情绪波动以「小时」计、双相以「天–周」计。

🧩 双相选药（含抑郁相、维持、孕期）决策树 → [[完整笔记/专题笔记/psych/psych_双相障碍用药决策树]]

### 2.3 其他抑郁谱系速记

| 诊断 | 钥匙 |
|---|---|
| **Atypical depression 非典型抑郁** | 心境随好事改善 + 嗜睡 + 食欲↑ + 灌铅样麻痹；一线仍 SSRI（**MAOI 是多步失败后**，非一线）|
| **Seasonal affective disorder 季节性** | 冬季抑郁 → 光照治疗 + SSRI |
| **Pseudodementia 假性痴呆** | 老年抑郁伪装认知下降 → 治抑郁可逆 |
| **Postpartum depression 产后抑郁 (PPD)** | 产后 4 周内常见、持续 > 2 周 → SSRI / 心理咨询 |
| **Postpartum blues 产后忧郁** | 产后 2–3 天起、< 2 周 → 观察支持 |
| **Postpartum psychosis 产后精神病** | 幻觉/妄想 + 婴儿危险 → 住院 + 药物/ECT |

## 三、抗抑郁药（Antidepressants — 选药钥匙 + 副作用）

> [!tip] 选药一句话钥匙
> - 想**戒烟** / 怕**性功能障碍** / 怕胖 → **安非他酮 (bupropion)**；**禁**癫痫 + 进食障碍（降癫痫阈值）。
> - 老人**消瘦 + 失眠** → **米氮平 (mirtazapine)**（增食欲 + 助眠）。
> - 神经病理性疼痛共病抑郁 → **SNRI**（如度洛西汀 duloxetine）。

| 药 | 标志性副作用 / 钥匙 |
|---|---|
| **SSRI** | 胃肠道反应 (GI) / 性功能障碍 / 抗利尿激素分泌异常 (syndrome of inappropriate ADH, SIADH) / 转躁；孕期安全除**帕罗西汀 (paroxetine)** |
| **西酞普兰/艾司西酞普兰 citalopram / escitalopram** | **QT 延长** |
| **安非他酮 bupropion** | **癫痫**（进食障碍/癫痫禁用）|
| **米氮平 mirtazapine** | **体重增加** |
| **曲唑酮 trazodone** | **阴茎异常勃起 (priapism)** |
| **三环类 TCA** | 过量致死 — **QRS 时限**作预后指标；过量抗胆碱 + 宽 QRS → **碳酸氢钠 (sodium bicarbonate)** |
| **MAOI（如苯乙肼 phenelzine）** | **酪胺 (tyramine) 高血压危象**（肾上腺素能爆发，≠ 5-HT 综合征）|

> [!info] 抗抑郁药随访时间窗
> - **起效**：症状改善需 **4–6 周**（别因 1–2 周无效就换药）。
> - **疗程**：首次发作缓解后维持 ≥ 6 个月。
> - **监测窗**：开药后 **1–2 周**自杀风险窗（尤 <25 岁）。
> - **抗抑郁诱发转躁**：first step = 停抗抑郁药（提示潜在双相）。

> [!warning] 圣约翰草 (St. John's Wort) 是 buzzword
> 草药 + 抗抑郁药 → **5-羟色胺综合征 (serotonin syndrome)** 风险；诱导 CYP → 降很多药浓度（口服避孕药失效）。

## 四、精神病性 + 抗精神病药（Antipsychotics）

### 4.1 典型 vs 非典型副作用钥匙

| 类别 | 钥匙 |
|---|---|
| **高效价一代（氟哌啶醇 haloperidol / 氟奋乃静 fluphenazine）** | **更多 EPS**（锥体外系）|
| **低效价一代（氯丙嗪 chlorpromazine / 硫利达嗪 thioridazine）** | 更多自主神经 + 抗胆碱、EPS 少 |
| **不依从精分** | → **IM haloperidol（肌注长效）** |

二代关键单药联想：

| 药 | 标志副作用 |
|---|---|
| **阿立哌唑 aripiprazole** | 副作用最少 |
| **喹硫平 quetiapine** | 最镇静 |
| **齐拉西酮 ziprasidone** | **QT 延长** |
| **奥氮平 olanzapine** | 最大**体重增加** + 糖耐量异常 |
| **氯氮平 clozapine** | **粒细胞缺乏 (agranulocytosis)** + 降癫痫阈值 |
| **利培酮 risperidone** | **泌乳素↑** → 换阿立哌唑/喹硫平 |

### 4.2 EPS 四联（按出现顺序）

```
Acute dystonia 急性肌张力障碍    数小时–数天    斜颈/眼上翻  → 苯托品 benztropine / 苯海拉明 diphenhydramine
Akathisia 静坐不能              数天–数周      坐立不安    → β 阻滞剂 propranolol（首选）± BZD
Parkinsonism 帕金森综合征        数周          僵慢震颤    → 苯托品 / 金刚烷胺 amantadine（精分者禁左旋多巴，会加重幻觉）
Tardive dyskinesia 迟发运动障碍  数月–数年      口面不自主  → VMAT2（囊泡单胺转运体 2）抑制剂 缬苯那嗪 valbenazine / 氘代丁苯那嗪 deutetrabenazine（唯一难逆）
```

## 五、副作用急症四联鉴别（NMS / SS / MH / EPS）⭐⭐⭐

> [!danger] 高热 + 肌强直找触发药史 = 一秒分流
> NMS 找「抗精神病药」史；MH 找「麻醉药（琥珀胆碱/氟烷）」史；SS 找「5-HT 能药」史。

| 综合征 | 触发 | 起病 | 神经体征钥匙 | 处理 |
|---|---|---|---|---|
| **抗精神病恶性综合征 neuroleptic malignant syndrome (NMS)** | 抗精神病药 | 数天（慢）| **铅管样强直 (lead-pipe rigidity)** + bradykinesia 运动迟缓 + 意识改变 (altered mental status, AMS) + 自主神经不稳；白细胞 (WBC)↑ 陷阱 | 停药 + 补液 + **溴隐亭 bromocriptine** + **丹曲林 dantrolene** |
| **5-羟色胺综合征 serotonin syndrome (SS)** | 5-HT 能药（SSRI/MAOI/MDMA）| 数小时（快）| **反射亢进 (hyperreflexia) + 阵挛 (clonus)** + 激越 + 黏膜湿 | 停药 + 支持 + **赛庚啶 cyproheptadine** |
| **恶性高热 malignant hyperthermia (MH)** | 麻醉（琥珀胆碱/氟烷）| 麻醉中/后 | 强直 + 高热 | **丹曲林 dantrolene** + 降温 |
| **EPS（急性肌张力障碍）** | 抗精神病药 | 数小时–数天 | 局部肌痉挛/斜颈，无高热 | 苯托品 / 苯海拉明 |

> [!tip] 一句话锁定
> 「**湿 + 阵挛 + 反射亢进 = SS**（赛庚啶）；**干 + 铅管 + 慢起病 = NMS**（溴隐亭 + 丹曲林）；**麻醉后强直高热 = MH**（丹曲林）。」

## 六、Catatonia（紧张症）+ ECT

```
普通 catatonia → 一线 Lorazepam（劳拉西泮 challenge 诊断兼治疗，数十分钟戏剧起效）
              → 二线 ECT（lorazepam 无效/部分有效/危及生命）
              → 抗精神病药诱发者 = 停抗精神病药（"加/换抗精神病药"是陷阱，可诱发 NMS）
恶性 catatonia（高热 + 自主神经不稳 + 严重强直 → 内科急症）→ ECT 紧急上升为一线
```

> [!info] ECT 指征 + 优势
> - 指征：**急性自杀倾向 / 躁狂 / 难治性抑郁 / catatonia / 拒食脱水 / 药物无效的精神病性抑郁 / 孕期不宜用药**。
> - 副作用：**记忆丧失 (memory loss)**；机制：诱发癫痫。
> - **无绝对禁忌**（最大优势）；孕妇、老人均安全。相对高风险：严重 CVD/近期 MI、占位性脑病变、近期卒中、不稳定动脉瘤。

## 七、焦虑谱系（Anxiety — 先数"担忧靶点"）

> [!warning] 焦虑题先数担忧靶点
> 多个不相关领域 → **GAD**（探照灯）；单一聚焦 → 看靶点类型。**一条社交担忧 ≠ Social Anxiety**（经典误选）。

| 诊断 | 钥匙 | 治疗 |
|---|---|---|
| **Panic attack 惊恐发作** | 急性自主神经亢进（心悸/出汗/濒死感）| **ER → 苯二氮䓬 (benzodiazepine)** |
| **Panic disorder 惊恐障碍** | 反复非预期发作 + worry > 1 个月；先排除甲亢 (hyperthyroidism, 查 TSH 促甲状腺激素) / 嗜铬细胞瘤 (pheochromocytoma, 查尿儿茶酚胺) / 心律失常 (查心电图 ECG) | 急性苯二氮䓬 (BZD) + **认知行为治疗 (CBT) + SSRI**（长期）|
| **Specific phobia 特定恐惧** | 对特定刺激（如血）稳定强烈恐惧 | BZD/β 阻滞剂 + 脱敏 |
| **GAD 广泛性焦虑** | 多领域担忧 > 6 个月 | **CBT + SSRI** |
| **OCD 强迫障碍** | 列表/计数/重复动作 | **高剂量 SSRI + CBT-ERP（暴露与反应预防 exposure and response prevention）**（一线）；二线**氯米帕明 clomipramine** |

> [!tip] OCD 心理治疗只认 CBT-ERP
> 精神动力治疗 (psychodynamic) / 辩证行为治疗 (dialectical behavior therapy, DBT) / 丁螺环酮 (buspirone) 都是诱饵。

> [!info] 儿童焦虑时长
> **GAD in children**：广泛焦虑 ≥6 个月，再加**另一**症状（成人需 3 个）；一线 = SSRI（如舍曲林 sertraline）。

## 八、物质使用 / 中毒 / 戒断（Substance use）

### 8.1 中毒 vs 戒断（弹簧模型）

| 物质 | 中毒（压弹簧）| 戒断（弹簧反弹）|
|---|---|---|
| **抑制剂（酒精/苯二氮䓬 BZD）** | 极静、呼吸慢、言语含糊、共济失调 | 极兴奋、震颤、**癫痫**（致死）|
| **兴奋剂（可卡因/安非他命）** | 极兴奋、瞳孔大、心动过速 | 极抑郁、嗜睡、食欲↑（Crash，自杀风险）|
| **阿片类 (opioids)** | **针尖瞳 (miosis)** + 呼吸抑制 < 10/min + 便秘 | 流泪/呕吐/排尿/肌痉挛（**不致死**）|
| **大麻 (marijuana)** | 结膜充血(红眼) + 食欲↑ + 偏执 + 反应慢 | 易怒、失眠、没胃口 |

### 8.2 4 大戒断 vital signs（口诀「GABA 飙 / 阿片平 / 兴奋剂沉」）

| 戒断 | HR/BP | 体温 | Seizure | 致死 |
|---|---|---|---|---|
| **Alcohol 酒精** | ↑↑↑ | ↑（震颤谵妄 DT 高热）| 有（12–48h）| **致死（震颤谵妄 delirium tremens, DT）** |
| **BZD** | ↑↑↑ | ↑ | 有 | **致死** |
| **Opioid 阿片** | 正常/略↑ | 正常 | 无 | 不致死（成人）|
| **Stimulant 兴奋剂** | 正常/↓ | 正常 | 无 | 心理风险（自杀）|

> 钥匙：**vital signs 正常 = NOT alcohol/BZD withdrawal**。

### 8.3 关键解毒 / 处理

- **阿片过量 (opioid overdose)** → **纳洛酮 (naloxone)**（半衰期短，可能需重复/持续滴注防再昏迷）；术后 AMS + 呼吸 < 10/min → 先查**动脉血气 (arterial blood gas, ABG)**。
- **酒精/BZD 戒断** → **长效 BZD**（氯氮䓬 chlordiazepoxide / 地西泮 diazepam）以毒攻毒。
- **氟马西尼 (flumazenil)**：BZD 竞争性拮抗剂，**慢性使用者/混合中毒禁用**（诱发不可控癫痫）；仅短期暴露/术后苏醒慢可考虑。
- **可卡因胸痛** → 苯二氮䓬 + 阿司匹林 + 硝酸甘油/钙通道阻滞剂；**禁 β 阻滞剂**（无对抗 α，冠脉痉挛）+ **禁溶栓**（ICH 风险）。
- **酒精中毒低血糖**：乙醇代谢 NADH↑ → 抑制糖异生；补糖前/同时给**维生素 B1 (thiamine)** 防 Wernicke。
- **PCP 中毒** → BZD + 抗精神病药；**垂直 + 水平眼震 + 暴力 + 解离 + 镇痛**。
- **MDMA / 合成卡西酮(bath salts)**：常规安非他命尿检**测不出**；SSRI 用户 → 5-HT 综合征 + 高热 + 低钠（SIADH + 过度饮水）。

## 九、心境稳定剂监测（Mood stabilizers）

| 药 | 标志毒性 / 监测 |
|---|---|
| **锂 (lithium)** | 致畸（**Ebstein 畸形**，孕早期）/ 甲减 / 肾性尿崩 / 甲旁亢；治疗前查 BUN/Cr·尿常规·TSH·钙·ECG(老年)·妊娠；每 6–12 月复查肾甲功 |
| **丙戊酸 (valproate)** | 肝毒性 / 胰腺炎 / 神经管缺陷 / 震颤 / 血小板↓ / 脱发 |
| **卡马西平 (carbamazepine)** | 粒细胞缺乏 |
| **拉莫三嗪 (lamotrigine)** | **Stevens-Johnson 综合征 (SJS)** |

> [!warning] 锂中毒阈值
> 血药 **> 1.5 mEq/L** 中毒：急性 = GI（恶心呕吐腹泻）+ 心律失常；慢性 = 粗大震颤/共济失调/神志不清/癫痫（脱水/低钠饮食/NSAID·ACEI·噻嗪诱发）。
> **> 4.0 或严重神经症状 → 血液透析 (hemodialysis)**。肾性尿崩仍须用锂 → **阿米洛利 (amiloride)**（阻上皮钠通道 ENaC）。

## 十、人格障碍 ABC 三簇（Personality disorders）

```
A 簇「古怪 (Weird)」
├─ Paranoid 偏执     多疑、不信任
├─ Schizoid 分裂样   主动独处、享受孤独
└─ Schizotypal 分裂型 奇异/魔幻思维

B 簇「戏剧 (Wild)」
├─ Borderline 边缘   非黑即白、怕被抛弃、情绪以「小时」波动、一过性偏执/精神病性 → 一线 DBT
├─ Histrionic 表演   寻求关注
├─ Antisocial 反社会 ≥18 岁（<18 = conduct disorder 品行障碍）
└─ Narcissistic 自恋

C 簇「焦虑 (Worried)」
├─ Avoidant 回避     想要朋友、怕被拒、没朋友
├─ Dependent 依赖    需别人替自己做决定
└─ OCPD 强迫型人格   高度有序、完美主义（≠ OCD：OCPD 自我和谐无强迫思维）
```

> [!tip] 高频陷阱
> **Borderline (hours) vs Bipolar (days–weeks)**；**OCPD（人格、无 obsession）vs OCD（有 obsession + compulsion）**。

## 十一、神经发育 + 儿童青少年（Neurodevelopmental）

| 诊断 | 钥匙 |
|---|---|
| **ASD 自闭症谱系** | ① 社交沟通缺陷（无目光接触/不分享快乐）② 受限重复行为（刻板动作/坚持流程/痴迷物体）|
| **ADHD 注意缺陷多动** | 成人 + 物质滥用史/拒成瘾药 → **托莫西汀 (atomoxetine)**（非兴奋剂）|
| **ODD 对立违抗障碍** | 易怒 + 争辩违抗 + 记仇报复（"跟你对着干"，不严重伤害他人）|
| **Conduct disorder 品行障碍** | 侵犯他人权利/社会规范（更重、更外化、伤害他人/破坏规则）；<18 岁 |
| **DMDD 破坏性心境失调障碍** | >6 岁才诊断，onset before 10 岁 |
| **Tourette 抽动秽语综合征** | 运动 + **发声 tic（抽动）**（发声 tic ≠ OCD）；习惯逆转训练 (habit reversal training, HRT) + α2 激动剂失败 → 抗多巴胺药 |

> [!info] 儿童 GAD vs Social anxiety
> 多个不相关领域担忧 = GAD；单一社交聚焦才是 Social Anxiety（一条社交担忧不够）。

## 十二、进食障碍三兄弟（Eating disorders）

| 诊断 | 钥匙 | 一线药 |
|---|---|---|
| **Anorexia nervosa 神经性厌食 (AN)** | 极低 BMI（< 16）；清除型 → 低钾低氯**代谢性碱中毒**；ECG 见 **U 波/T 波平/ST 压低 = 低钾** | **无药一线（CBT + 营养）**；二线奥氮平 (olanzapine) |
| **Bulimia nervosa 神经性贪食 (BN)** | **正常 BMI** | **氟西汀 (fluoxetine)** |
| **Binge eating disorder 暴食障碍 (BED)** | 暴食无清除 | SSRI（舍曲林）/ 备选**赖氨酸苯丙胺 (lisdexamfetamine)** |

> [!danger] 进食障碍两条铁律
> - **安非他酮 (bupropion) 进食障碍永远禁**（降癫痫阈值）。
> - 极低 BMI + ECG 异常 → **永远先看 K⁺**（U 波 = 低钾；高尖 T = 高钾）。
> - **再喂养综合征 (refeeding syndrome)**：治疗后胰岛素↑ → **磷↓↓**（核心，致心衰）+ 钾↓ + 镁↓ + 维 B1↓。

## 十三、躯体症状障碍谱（Somatic symptom spectrum — 4 个关键鉴别）

| 诊断 | 钥匙 |
|---|---|
| **Somatic symptom disorder 躯体症状障碍** | 躯体有症状 + **非常焦虑** |
| **Illness anxiety disorder 疾病焦虑障碍** | **无症状** + 非常焦虑（怕某一特定病）|
| **Conversion disorder 转换障碍** | 神经症状（动不了手脚）+ **病人不在意 (la belle indifférence)** |
| **Factitious disorder 做作性障碍** | 故意装病、**享受当病人**（无外部获益）|
| **Malingering 诈病** | 故意装病、**有目的**（外部获益）|

> [!tip] 管理躯体症状障碍
> 建立医患关系 + 短而规律的随访 + 最少检查 + 慢慢建立信任（避免反复昂贵检查）。

## 十五、医患沟通 + Ethics 触发点（精神科常见）

- **Ethics 做题准则**：永远选最尊重**病人自主权 (autonomy)**、最专业沟通、不推卸责任的选项。
- **Capacity vs Competence**：医生判 **capacity（临床能力）**；法院判 **competence（法律资格）**。
- **保密例外（可破保密）**：对他人有危险 / 虐待 / 法院命令 / 公共卫生上报。
- **家属要求隐瞒病情**：先问家属为什么，但最终须告知病人（除非病人自己说不想知道）。
- **失能同事**：医学生/住院医 → 报告上级 (attending)；合伙人/主治 → 报告医学委员会 (medical board)。
- **未成年部分解放 (partial emancipation)**：怀孕 / 避孕 / 性病 (sexually transmitted disease, STD) / 物质滥用 / 心理治疗**不需父母同意**；危及生命/肢体可越过父母拒绝。

## 十六、复习题（折叠自测）

> [!question]- 抑郁 + 妄想，下一步治疗选什么？为什么锂是诱饵？
> SSRI/抗抑郁药 **＋ 抗精神病药 (antipsychotic)**，两者缺一不可。锂 (lithium) 不治妄想 → 经典诱饵。MDD with psychotic features 反复错重灾区 🔴。

> [!question]- 高热 + 铅管样强直 + 慢起病 + WBC↑ vs 高热 + 反射亢进 + 阵挛 + 黏膜湿，分别是什么？怎么治？
> 前者 = **NMS**（抗精神病药史）→ 停药 + 补液 + **溴隐亭 + 丹曲林**；后者 = **5-羟色胺综合征**（5-HT 能药/MDMA）→ 停药 + **赛庚啶**。湿 + 阵挛 = SS；干 + 铅管 = NMS。

> [!question]- 精神病性谱系四个诊断的时长阈值分别是？
> Brief psychotic < 1 个月；Schizophreniform 1–6 个月；Schizophrenia > 6 个月；Schizoaffective > 6 个月 + ≥2 周「单纯幻觉/妄想无情绪」。

> [!question]- 哪种戒断 vital signs 正常？这条钥匙怎么用？
> **阿片戒断 (opioid withdrawal)**：HR/BP 正常或略↑、体温正常、无 seizure、不致死。钥匙：**vital signs 正常 = NOT alcohol/BZD withdrawal**（口诀「GABA 飙 / 阿片平 / 兴奋剂沉」）。

> [!question]- 极低 BMI + ECG 见 U 波，第一步看什么？再喂养综合征核心指标是？
> 永远**先看 K⁺**（U 波 = 低钾）。再喂养综合征核心 = **磷↓↓**（致心衰），伴钾↓、镁↓、维 B1↓。

> [!question]- Borderline 与 Bipolar 情绪波动如何区分？OCPD 与 OCD 呢？
> Borderline 情绪以**小时**波动、Bipolar 以**天–周**波动。OCPD = 人格、自我和谐、无 obsession；OCD = 有 obsession + compulsion、自我不和谐。

> [!question]- 抑郁伴自杀意念，next step 是什么？最强单一风险因子？
> next step = **先做自杀风险评估**（不反射开药）。最强单一风险因子 = **既往自杀未遂**。

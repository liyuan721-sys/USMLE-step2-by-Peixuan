# 快速抗抑郁药 — Esketamine / Ketamine

> [!info] 一句话本质
> **Esketamine（Spravato，鼻喷剂）= ketamine 的 S-对映体**，FDA 2019 批准。**NMDA 拮抗剂**，绕过 SSRI 慢调 5-HT 的传统路径，**几小时-72 小时**抗抑郁，专攻"等不及"的急性 SI 和难治抑郁（TRD）。
> 机制基础详见 [[完整笔记/专题笔记/神经/神经_NMDA受体临床速查]]。

---

## 一、两套用法 ⭐（分清楚就赢一半）

| 适应症 | 性质 | 疗程 |
|---|---|---|
| **MDD with acute suicidal ideation / behavior** | **短期桥接** | 急性危机几次 - 数周；等口服抗抑郁药起效 |
| **Treatment-Resistant Depression (TRD)** | **可长期维持** | 数月 - 数年（按反应调整）|

> [!warning] 别只记成"桥接"
> Esketamine 在 TRD 适应症下是**长期维持药**（SUSTAIN-1 试验支持 ≥16 周维持显著降复发）。"只用桥接"是常见误解。
> - 急性 SI 用法 = 短期（桥接到 SSRI 起效）
> - TRD 用法 = 长期（维持治疗）

---

## 二、TRD 标准给药节奏

```
Induction 诱导期    → 2 次/周 × 4 周（共 8 次）
                      ↓
Maintenance 维持期  → 1 次/周 × 4 周
                      ↓
Long-term 长期      → 1 次/周 或 1 次/2 周（按反应调）
```

**永远配口服抗抑郁药**（FDA 标签要求）—— 没有单药指征。

---

## 三、机制（NMDA 拮抗 → 谷氨酸反爆发）

```
Esketamine 拮抗 NMDA
         ↓ 抑制 GABA 中间神经元
谷氨酸"反爆发"（disinhibition burst）
         ↓ 作用于 AMPA 受体
       BDNF 释放
         ↓
快速突触发生（synaptogenesis）
         ↓
几小时-72 小时显效（vs SSRI 4-6 周）
```

> [!tip] 为什么绕过 SSRI 路径就快
> SSRI 慢慢调单胺（5-HT / NE）；esketamine 直接动突触可塑性（BDNF + 突触发生）。这是为什么 TRD（SSRI 失败的）也能起效。

---

## 四、给药 / REMS / 限制 ⭐

| 维度 | 内容 |
|---|---|
| **剂型** | 鼻喷剂（Spravato）|
| **场所** | **REMS 项目** —— 只能在认证诊所给 |
| **观察** | 给药后**院内观察 2 小时**（监测 BP / 解离 / 镇静）|
| **驾车** | 当日不能自驾 |
| **联用要求** | **必须 + 口服抗抑郁药**（SSRI / SNRI 等）|
| **DEA** | **Schedule III**（同 ketamine）|
| **禁忌** | 主动脉瘤 / 颅内出血史 / 严重 CVD / 不能配合监测 |

---

## 五、副作用速查

| 类别 | 表现 | 处理 |
|---|---|---|
| **解离 dissociation** ⭐ | 峰值 ~40 分钟，给药当天消退 | 观察、安抚；不一定停药 |
| **镇静** | 给药后 1-2 小时 | 观察期内 |
| **血压一过性↑** | 给药后 40-60 分钟峰值 | 观察 BP；高血压 / 心血管病慎用 |
| **恶心 / 眩晕** | 常见 | 对症 |
| **滥用 / 依赖风险** | 长期 / 高剂量风险 | REMS 限制、不带回家 |
| **膀胱毒性**（cystitis）| 长期娱乐 ketamine 报道；治疗剂量风险小 | 监测 |
| **认知** | 反复给药累积效应数据仍在收集 | 监测 |
| **黑框警告** | 镇静 + 解离 + 滥用 + 自杀念头（同所有抗抑郁药 <25 岁）| 标签声明 |

## 六、Esketamine vs SSRI vs ECT — 三方对比

| 维度 | SSRI | ECT | **Esketamine** |
|---|---|---|---|
| **起效** | 4-6 周 | 几天 | **几小时 - 72 小时** ⭐ |
| **给药方式** | 口服，居家 | 全麻 + 电击，住院 / 门诊 | 鼻喷，REMS 诊所 |
| **侵入性** | 无 | 高（全麻 + 电）| 低 |
| **适用阶段** | 一线 MDD（所有患者）| 危急 + 难治 + 妊娠 + 拒食 | 急性 SI 桥接 + TRD 维持 |
| **优势** | 普及、便宜、家用 | 最强效、起效快 | **快 + 不需麻醉 + 可重复** |
| **缺点** | 慢；<25 黑框 | 全麻风险 / 记忆副作用 / 频率 / 污名 | 贵、REMS 限制、滥用风险 |
| **价格** | $5-50/月 | 每次 $1500-2500 | **每次 $600-900+**（仍贵）|
| **配药要求** | 单药 OK | 单独可以 | **必配 oral antidepressant** |

## 七、USMLE 题干钩子 → 选 Esketamine 的逻辑

| 题干 | 选 Esketamine 的逻辑 |
|---|---|
| "Patient with TRD after **≥2 adequate antidepressant trials** failed" | TRD 适应症 |
| "MDD + **acute suicidal ideation**, **needs rapid effect**, refuses ECT" | 急性 SI 桥接 |
| "Patient on long-term SSRI + **wants alternative to ECT** for severe depression" | TRD 替代 ECT 选项 |
| 题干提"in-office administration" + "monitored for 2 hours" | 暗示 esketamine（REMS）|

**vs 选 ECT 的题干钩子**：
- 拒食拒饮 + 即将自杀（**真急 + 来不及吃药**）→ ECT
- 妊娠 + 严重 MDD + 精神病性 → ECT
- 已经在 esketamine + 仍不响应 / catatonia → ECT

---

## 八、Memory Hook

- **"K 字头 = Quick"** —— **K**etamine / es**K**etamine 都是 K 开头，记 **Quick**（几小时起效）
- **两套用法**：「**急性 SI 桥得短、TRD 用得长**」
- **三铁律**：**不单飞**（必配口服 AD）/ **不带回家**（REMS）/ **不开车**（当日）
- **机制类比**：SSRI = 慢慢调音量；Esketamine = 直接重接电路（突触发生）
- **反向钩**：看到 "rapid effect for MDD + SI / TRD" 就想 esketamine（除非题干让你选 ECT）

## 0. 核心记忆框架 "3+3+3"（先记这个 ⭐）

> [!success] "3 + 3 + 3" 法则（拿走即用）
>
> **🎯 3 个轴（定位物质）**
> 1. **兴奋 / 抑制** —— 决定生命体征方向（§2）
> 2. **抗胆碱 / 胆碱能** —— 决定"干"还是"湿"（§8）
> 3. **中毒 / 戒断** —— 互为镜像（§9）
>
> **👁 3 个一眼诊断（30 秒分诊，§1）**
> 1. **瞳孔**：针尖 → Opioid / organophosphate；散大 → 兴奋剂 / 致幻剂 / 抗胆碱；多方向眼震 → **PCP** ⭐
> 2. **呼吸**：抑制 → Opioid / BZD+酒 / Alcohol / Barbiturate；正常或↑ → 其他
> 3. **干 vs 湿**："干红热瞎疯" → 抗胆碱；"出汗流涎腹泻" → 胆碱能
>
> **💊 3 个治疗原则（§10）**
> 1. **激越 → BZD**（覆盖 90%：PCP / cocaine / meth / 酒精戒断 / BZD 戒断）
> 2. **抑制 → 对应解毒剂**（Opioid→Naloxone；APAP→NAC；OP→Atropine+Pralidoxime；抗胆碱重症→Physostigmine；Methanol/乙二醇→Fomepizole）
> 3. **戒断 = 中毒的镜像**（致命戒断只有 **Alcohol + BZD**）

> [!tip] 一句话总结
> **"先看瞳孔 + 呼吸 + 干湿 → 定位物质 → 激越给 BZD，抑制给解毒剂"**

## 1. 30 秒分诊 — 一眼诊断（瞳孔 · 呼吸 · 干湿）

### 1.1 瞳孔

```
针尖瞳 (Pinpoint / Miosis)
├─ Opioids ⭐⭐⭐ 最高频
├─ Organophosphate（胆碱能）
├─ Clonidine 中毒
└─ 桥脑出血（非药物）

正常瞳孔
├─ Alcohol
├─ Benzodiazepines
├─ Marijuana
└─ Inhalants

散瞳 (Mydriasis)
├─ 兴奋剂全家：Cocaine / Meth / MDMA / Amphetamine
├─ 致幻剂：LSD / Psilocybin / Ketamine
└─ 抗胆碱：TCA / 抗组胺 / Datura / 阿托品

特殊
└─ PCP：瞳孔大小不定，但有【多方向眼震】⭐（水平 + 垂直 + 旋转）
```

> [!tip] 瞳孔规律
> **"瞳孔大 = 兴奋 or 抗胆碱 or 致幻；瞳孔小 = 阿片 or 胆碱能"**

#### 眼征细分 — 一眼锁定

| 眼部表现 | 指向 |
|---|---|
| **多方向眼震**（水平 + 垂直 + 旋转）| **PCP** ⭐ |
| 单纯水平眼震 | Alcohol / BZD / Phenytoin / Lithium |
| 双侧散瞳 mydriasis | Cocaine / Meth / Amphetamine / 抗胆碱 |
| 针尖瞳 miosis | **Opioid** ⭐ / Organophosphate / Clonidine |
| **眼阵挛 ocular clonus** | **Serotonin Syndrome** ⭐ |
| 眼震 + 共济失调 + 眼肌麻痹 | **Wernicke**（立即给 thiamine）|

### 1.2 呼吸

```
呼吸抑制 (RR < 12)
├─ Opioids ⭐
├─ Alcohol 中毒（重度）
├─ BZD + Alcohol 合用
└─ Barbiturates

呼吸正常或 ↑
└─ 几乎所有其他物质
```

> [!danger] 致命组合
> **BZD 单用很少致死，但 BZD + Alcohol 合用 → 呼吸抑制致死**（高频考点）。

### 1.3 干 vs 湿

一眼判"干 / 湿"是 Axis 2，详见 §8。速记：**干红热瞎疯 = 抗胆碱；出汗流涎腹泻 = 胆碱能**。

## 2. Axis 1 — 兴奋 vs 抑制轴 + Toxidrome 速查

```
═══════════════════════════════════════════
            兴奋类             抑制类
═══════════════════════════════════════════
血压          ↑                 ↓
心率          ↑                 ↓
体温          ↑                 ↓
瞳孔          ↑（散大）         ↓ 或 正常
呼吸          ↑                 ↓ ⭐ 致死点
意识          激越 / 幻觉       嗜睡 / 昏迷
═══════════════════════════════════════════
```

> [!tip] 一句话锁定
> **"兴奋全部 ↑，抑制全部 ↓，呼吸抑制最致命"**

**兴奋类（Stimulants + Hallucinogens + Dissociatives）**
- 拟交感：Cocaine / Methamphetamine / MDMA / Amphetamine
- 致幻：LSD / Psilocybin
- 解离：PCP / Ketamine

**抑制类（Depressants）**
- **Opioids** ⭐（针尖瞳 + 呼吸抑制 = 最高频死亡）
- Alcohol（中毒）
- Benzodiazepines
- Barbiturates / GHB

### 2.1 Toxidrome 速查表

| Toxidrome | 心率 | 血压 | 体温 | 瞳孔 | 皮肤 | 神志 |
|---|---|---|---|---|---|---|
| Sympathomimetic（兴奋剂）| ↑ | ↑ | ↑ | ↑ | **湿（出汗）** | 激越 |
| Anticholinergic | ↑ | ↑ | ↑ | ↑ | **干、红、热** | 谵妄 |
| Cholinergic（OP）| ↓ | ↓ | 正 | ↓ | **湿（流涎、出汗）** | 抑制 |
| Opioid | ↓ | ↓ | ↓ | ↓↓ | 正 | 抑制 |
| Sedative-hypnotic（BZD / 酒）| 正 / ↓ | ↓ | ↓ | 正 | 正 | 抑制 |

> [!warning] 高频陷阱
> **兴奋剂（cocaine / meth）也出汗，但因高热，不是胆碱能。** 鉴别点：瞳孔大（兴奋剂）vs 瞳孔小（胆碱能）。

## 3. 物质中毒鉴别速查表 ⭐⭐⭐

| 物质 | 瞳孔 | 体温 | 出汗 / 黏膜 | 肠鸣音 | 神经体征 | 独特线索 |
|---|---|---|---|---|---|---|
| **[[完整笔记/专题笔记/物质单药/psych_物质_MDMA\|MDMA]] / [[完整笔记/专题笔记/物质单药/psych_物质_Methamphetamine\|Amphetamine]]** | 散大 | **↑↑ Hyperthermia** | 出汗 | ↑ | **Myoclonus + Hyperreflexia + Seizure** | **+ SSRI = SS / 低钠 / 派对场景** |
| **[[完整笔记/专题笔记/物质单药/psych_物质_Cathinones\|Synthetic cathinone（bath salts）]]** | 散大 | 正常或 ↑ | 出汗 | — | **极端 combative 攻击性 + 精神病/谵妄 + 肌阵挛 + seizure** | **病程数天–1 周（最长）⭐ + 常规尿筛测不出 + “crazy rage”** |
| **[[完整笔记/专题笔记/物质单药/psych_物质_Cocaine\|Cocaine]]** | 散大 | ↑ | 出汗 | 正常 | 激越 + seizure | 鼻中隔穿孔 / 急性 MI |
| **Anticholinergic** (DPH / atropine / TCA) | **散大** | **↑ Hyperthermia** | **皮肤干 + 黏膜干 + 肠鸣音 ↓** ⭐ | **↓** | 谵妄 / bizarre / 尿潴留 | "干红怒热" — Hot/Dry/Red/Crazy |
| **[[完整笔记/专题笔记/物质单药/psych_物质_PCP\|PCP]]** | 垂直 + 水平 **nystagmus** | ↑ | — | — | 暴力 / 解离 / 镇痛 / 不响应疼痛 | NMDA 拮抗，**强烈攻击性**不是 somnolence |
| **[[完整笔记/专题笔记/物质单药/psych_物质_Cannabis\|Cannabis]]** | 红眼（结膜充血）| 正常 | — | — | 反应慢 / 偏执 / 食欲 ↑ | **不致 hyperthermia / myoclonus** |
| **[[完整笔记/专题笔记/物质单药/psych_物质_Opioids\|Opioid]]** | **针尖 miosis** | 正常 / ↓ | — | ↓ 便秘 | **CNS + 呼吸抑制** | Naloxone 反应阳性 |
| **[[完整笔记/专题笔记/物质单药/psych_物质_Benzodiazepines\|BZD]]** | — | 正常 | — | — | CNS depression + ataxia + 言语含糊 | 生命体征**正常**（vs 阿片呼吸抑制）|
| **Nitrous oxide**（笑气）| — | 正常 | — | — | LOC / seizure（大量）| 效果仅 **15-45 分钟**；慢性用 → B12 缺乏（亚急性联合变性）|
| **Cholinergic**（OP / 蘑菇）| **缩小 miosis** | — | **大量出汗 / 流涎 / 流泪** | ↑↑ 腹泻 | **SLUDGE + DUMBELS** | bradycardia；Atropine + Pralidoxime |

> [!tip] 物质中毒速记口诀
> - **"干红怒热"** = Anticholinergic（DPH / atropine / TCA）— Hot / Dry / Red / Crazy
> - **"湿乱抽"** = MDMA / SS — 出汗 + 肌阵挛 + 高热
> - **"流出 SLUDGE"** = Cholinergic — 流涎/流泪/排尿/腹泻/胃肠激惹/呕吐
> - **"针尖呼抑"** = Opioid — miosis + 呼吸抑制
> - **"散瞳火热"** = Stimulant（cocaine / amphet / MDMA）

> [!warning] 隐形陷阱：urine tox 阴性 ≠ 排除中毒
> 常规免疫筛查（针对 d-amphetamine）catch 不到 **MDMA / bath salts / 合成 cathinone / 合成大麻**。临床表现**永远优先**于 tox 阴性。**尿药筛查完整判读（假阴性 / 假阳性）见 §13。**

## 4. Serotonin Syndrome (SS)

### 4.1 三联征 "M.A.N"

| 三联 | 临床表现 |
|---|---|
| **M**ental change | 激越 / 谵妄 / 昏迷 / hallucination |
| **A**utonomic instability | HR ↑ / BP ↑ / 出汗 / **hyperthermia** / 散瞳 / 流涎 |
| **N**euromuscular hyperactivity ⭐ | **Hyperreflexia + Myoclonus + Clonus + Tremor + 肌张力 ↑** |

**N 是 SS 的指纹** —— 与 NMS（铅管样僵硬 + bradyreflexia）正好相反。

### 4.2 触发药（≥ 2 个 5-HT 类合用最危险）

- **SSRI / SNRI / TCA**（sertraline / fluoxetine / venlafaxine）
- **MAOI**（最危险 — 14 天 washout）
- **Triptan**（偏头痛药）
- **MDMA / Ecstasy / amphetamine 衍生物** ⭐
- **Tramadol**（弱 SNRI）
- **Linezolid**（弱 MAOI）
- **Dextromethorphan**（咳嗽糖浆）
- **St John's Wort**（草药）
- **Lithium**

### 4.3 治疗

| 优先级 | 措施 |
|---|---|
| 1 | **停药**（核心）|
| 2 | 支持（fluids / 降温 / BZD 减激越）|
| 3 | **Cyproheptadine**（5-HT2A 拮抗剂；中-重度 SS）|

> [!info] Cyproheptadine 作用
> 抗 5-HT（核心）+ 抗组胺 + 抗胆碱（副作用）

## 5. Neuroleptic Malignant Syndrome (NMS)

| 维度 | NMS |
|---|---|
| **触发** | 抗精神病药（D2 阻断 — typical / atypical 都可）/ 突然停 levodopa（PD 患者）|
| **起病** | **数天**（缓慢）|
| **四联征** | **高热 + lead-pipe rigidity（铅管样僵硬）+ AMS + autonomic instability** |
| **实验室** | **CK ↑↑↑**（横纹肌溶解）+ AKI（myoglobinuria）|
| **反射** | **Bradyreflexia / 反射 ↓** ⚠️（与 SS hyperreflexia 完全相反）|
| **治疗** | 停药 + 支持 + **Dantrolene** + **Bromocriptine**（D2 激动恢复）|

## 6. Malignant Hyperthermia (MH)

| 维度 | MH |
|---|---|
| **触发** | **吸入麻醉药（halothane / sevoflurane）+ Succinylcholine** —— 术中场景 |
| **起病** | **分钟-数小时**（最快）|
| **机制** | RyR1 基因突变 → Ca²⁺ 失控释放 → 持续肌收缩 |
| **临床** | 高热 + **肌僵硬**（masseter rigidity 早期）+ tachycardia + **高 ETCO₂** + 酸中毒 |
| **实验室** | CK ↑↑↑ + myoglobinuria + 高钾 |
| **治疗** | **停麻醉药 + Dantrolene + 降温 + 支持** |

## 7. 高热 + 肌强直 — 五联鉴别（SS / NMS / MH / PCP / 抗胆碱）⭐⭐⭐

### 7.1 三大致命高热综合征详细对照

| 维度 | **Serotonin Syndrome** | **NMS** | **Malignant Hyperthermia** |
|---|---|---|---|
| **触发药** | 5-HT 类（SSRI / MDMA / Triptan 等 ≥ 2 合用）| 抗精神病药（D2 阻断）/ 停 levodopa | 吸入麻醉药 + Succinylcholine |
| **起病** | **小时**（最快）| **数天**（最慢）| **分钟-数小时**（术中）|
| **场景** | 加药 / 换药 / 派对吸 MDMA | 长期用抗精神病药 | **麻醉术中 / 术后** |
| **肌张力** | **Hyperreflexia + Myoclonus + Clonus** ⭐ | **Lead-pipe rigidity + Bradyreflexia** ⭐ | **肌僵硬**（masseter rigidity 早期）|
| **AMS** | 有（激越 → 昏迷）| 有 | 有（高热相关）|
| **自主神经** | 亢进（出汗 / HR↑ / BP↑ / T↑）| 亢进 | 亢进 + **高 ETCO₂** ⭐ |
| **CK** | 可 ↑（较轻）| **↑↑↑** | **↑↑↑** |
| **首选药** | **Cyproheptadine**（5-HT 拮抗）| **Dantrolene + Bromocriptine**（D2 激动）| **Dantrolene**（核心）|
| **关键鉴别词** | "**Hyper**reflexia + Myoclonus" → SS | "**Lead-pipe** + 缓慢" → NMS | "**麻醉术中** + 高 ETCO₂" → MH |

### 7.2 扩展 — 加入 PCP 和抗胆碱（高热 + 肌强直 / 激越的另两个面孔）

| 综合征 | 触发 | 关键鉴别点 | 治疗 |
|---|---|---|---|
| **PCP intoxication** | 滥用 PCP | **多方向眼震**、暴力、反复滥用史、不响应疼痛 | **BZD** |
| **Serotonin syndrome** | SSRI + 其他血清素药 | **反射亢进、阵挛**、ocular clonus、腹泻 | 停药 + BZD + cyproheptadine |
| **NMS** | 抗精神病药 | **"铅管"强直**、起病慢（数日）、CK 极高 | 停药 + dantrolene / bromocriptine |
| **Malignant hyperthermia** | 吸入麻醉 / 琥珀酰胆碱 | 麻醉中急性发作、家族史、高 ETCO₂ | **Dantrolene** + 停麻醉 |
| **Anticholinergic** | TCA / 抗组胺 / Datura | **干、红、热、瞎、疯**、肠鸣音消失（vs 以上全是"湿"）| 支持；重者 physostigmine |

> [!danger] 4 大高热综合征解药矩阵 ⭐⭐⭐（v1.x 升级 2026-05-26 — 加 4 字母首字母口诀）
>
> **首字母口诀**：**"SS=Cyp / NMS=Dan+Brom / 抗胆碱=Phys / MH=Dan"** ⭐
>
> | 综合征 | 解药 | 一句话机制 |
> |---|---|---|
> | **Serotonin Syndrome (SS)** ⭐ | **Cyp**roheptadine + BZD + cooling | 5-HT2A/2C antagonist → 关 5-HT 信号 |
> | **NMS** | **Dan**trolene + **Brom**ocriptine | Ca²⁺ 阻断（肌浆网）+ 重启 D2 |
> | **抗胆碱毒性** | **Phys**ostigmine（重者）+ 支持 | 重启 ACh（cholinesterase 抑制）|
> | **Malignant Hyperthermia (MH)** | **Dan**trolene（核心）+ 停麻醉 | Ca²⁺ 阻断（肌浆网）|
>
> 共用基础：停药 + 支持 + 降温

> [!warning] 反射钩矩阵 — 哪个解药用在哪？⭐
>
> | 解药 | 适用 | 误用后果 |
> |---|---|---|
> | **Cyproheptadine** ⭐ | **SS only**（MDMA / SSRI / MAOI / Tramadol / Linezolid / DXM 引发）| — |
> | **Dantrolene** | NMS + MH（**不是 SS！**）| 用在 SS = 无效（5-HT 介导 hyperthermia 不是肌肉收缩）|
> | **Bromocriptine** | NMS only（重启 D2）| 用在 PD ψ = 加重 psychosis |
> | **Physostigmine** | 抗胆碱 only（重启 ACh）| 用在 SS / NMS = 加重 cholinergic crisis |

> [!info] 为什么 Dantrolene 不治 SS（关键边界）
> - Dantrolene = ryanodine receptor 拮抗剂 → 阻止肌浆网 Ca²⁺ 释放 → 适用于**肌肉过度收缩 + 高热**（NMS 铅管样 / MH）
> - SS 的 hyperthermia 来源**不是肌肉过度收缩**，是 **5-HT 介导**的下丘脑温调 + 自主神经亢进
> - Dantrolene 不针对 5-HT → SS 中无效
> - SS 必须 **Cyproheptadine**（H1 antihistamine + 5-HT2A/2C antagonist）从信号源头关

> [!success] 一句话锁定
> **"湿热抖 (SS) → Cyp / 湿热僵 (NMS) → Dan+Brom / 干热不抖 (抗胆碱) → Phys / 麻醉术中 (MH) → Dan"** ⭐

> [!tip] 速记决策树
> - **起病速度**：术中（MH）/ 小时（SS）/ 天（NMS）
> - **反射**：↑ 亢进（SS）/ 铅管 ↓（NMS）/ 僵硬（MH）
> - **触发药**：麻醉 + Succinylcholine（MH）/ 抗精神病药（NMS）/ 5-HT 类（SS）
> - **眼震 + 暴力 + 滥用史** → 跳出三联，想 **PCP**
> - **干 + 肠鸣音消失** → 跳出三联，想 **抗胆碱**（其余四个都"湿"）

> [!warning] Q3191 教训（PCP 误判为 SS）
> muscle rigidity + 眼动异常 + hyperthermia **不等于** Serotonin Syndrome。关键鉴别：**用药史**（滥用 PCP vs SSRI 联用）+ **反射**（PCP 不必反射亢进；SS 必有 hyperreflexia / clonus）。PCP 首选 **BZD**，不是 cyproheptadine。

## 8. Axis 2 — Anticholinergic vs Cholinergic（"干 vs 湿"）

```
═══════════════════════════════════════════════════
   抗胆碱（"干"）           胆碱能（"湿"）
═══════════════════════════════════════════════════
皮肤    干、红、热             出汗、苍白
口腔    口干                   流涎
眼      瞳孔大、视物模糊       瞳孔小（针尖）
肠      肠鸣音消失、便秘       腹泻、肠鸣音活跃
泌尿    尿潴留                 尿失禁
心率    tachycardia            bradycardia
═══════════════════════════════════════════════════
代表   TCA / 抗组胺 / Datura  Organophosphate / 毒蘑菇
治疗   Physostigmine（重症）  Atropine + Pralidoxime
═══════════════════════════════════════════════════
```

### 8.1 Anticholinergic Toxidrome — "Hot / Dry / Red / Crazy"

| 维度 | Anticholinergic |
|---|---|
| **触发药** | DPH / TCA / 阿托品 / 颠茄碱 / 老人误用助眠药 / Jimson weed |
| **皮肤** | **干 + 红**（皮肤干 + 发红）|
| **黏膜** | **干**（口干）|
| **瞳孔** | **散大** mydriasis |
| **温度** | **↑ Hyperthermia** |
| **CNS** | 谵妄 / bizarre / 幻觉 |
| **肠** | **肠鸣音 ↓** + 便秘 + 尿潴留 |
| **HR** | tachycardia |
| **治疗** | 支持 + **Physostigmine**（重度 — AChE 抑制剂；**禁忌：TCA 过量 + QRS 增宽** → arrhythmia）|

> [!warning] 经典口诀
> **抗胆碱**："**Dry as a bone, Red as a beet, Hot as Hades, Blind as a bat, Mad as a hatter**"（干红热瞎疯）。

> [!info] 为什么抗胆碱能 → 散瞳？瞳孔双重神经支配机制
> ```
> 瞳孔大小 = 交感 vs 副交感 拔河
> ─────────────────────────────────────────
> 副交感（CN III → M3 受体）→ 瞳孔括约肌 sphincter pupillae → 缩瞳 miosis
> 交感  （颈上节  → α1 受体）→ 瞳孔开大肌 dilator pupillae  → 散瞳 mydriasis
> ─────────────────────────────────────────
> 平时副交感占主导 → 维持 baseline 瞳孔大小
> ```
> **抗胆碱药阻 M3 → 副交感失能 → 交感"无敌手" → 散瞳 + 调节麻痹 (cycloplegia)**
> - **"Blind as a bat"** 实际含两重：① 瞳孔散大如 bat ② 睫状肌麻痹 → 近视力丧失（看近模糊）
> - **TCA 特殊**：除阻 M3 外还**阻断 NE 再摄取** → 突触间 NE↑ → α1 激动开大肌 → **双重散瞳**（副交感关 + 交感强）
> - **拟交感（cocaine / amphetamine）**：直接激动 α1 → 也散瞳，但**皮肤湿**（汗腺 M 未被阻）→ 鉴别 = "干 vs 湿"（见 §8.4）
>
> **散瞳 4 大类毒物速记**：
> | 毒物 | 机制 | 配套指纹 |
> |---|---|---|
> | 抗胆碱（TCA / atropine / DPH / Jimson weed）| M3 阻断 | **干红怒热** + 肠鸣音 ↓ |
> | 拟交感（cocaine / amphetamine / MDMA）| α1 激动 | **湿汗** + 高血压 |
> | Opioid **戒断**（不是中毒，中毒是 miosis）| 自主神经亢进 | 流泪 / 流涕 / 鸡皮 |
> | 脑干疝（CN III 受压）| 副交感纤维断 | **单侧固定散大** + 意识障碍 |

> [!danger] ⚠️ TCA 过量：抗胆碱能症状显眼，但真正的杀手是心脏
> TCA 同时阻滞 4 个靶点：M 受体（抗胆碱能毒性 — 显眼但不致命）/ α1（低血压）/ H1（镇静）/ **Na 通道**（中枢 → seizures；**心肌 → QRS 增宽 → VT/VF ⭐ 致命**）。
> - **致死靠心律失常**，不靠抽搐、不靠低血压。
> - **QRS 时限 = TCA 过量并发症的最佳预后指标**：> 100 ms → 室性心律失常风险↑（**NaHCO₃ 指征**）；> 160 ms → 抽搐 + 室速风险显著↑。
> - 解毒：**NaHCO₃**（补 Na⁺ 克服通道阻滞 + 碱化↑蛋白结合）。**血药浓度与心脏毒性相关性差，从不靠血药决策**。
> - 上表 physostigmine **禁用于 TCA 过量**（QRS 增宽时会诱发心律失常）。

### 8.2 Cholinergic Toxidrome — "SLUDGE / DUMBELS"

| 维度 | Cholinergic |
|---|---|
| **触发** | 有机磷（OP / 杀虫剂 / 神经毒剂）/ Carbamate / 某些毒蘑菇（muscarine）|
| **SLUDGE 助记** | **S**alivation / **L**acrimation / **U**rination / **D**efecation / **G**I 激惹 / **E**mesis |
| **DUMBELS 助记** | **D**iarrhea / **U**rination / **M**iosis / **B**radycardia-Bronchospasm-Bronchorrhea / **E**mesis / **L**acrimation / **S**alivation/Sweating |
| **瞳孔** | **缩小** miosis |
| **HR** | **Bradycardia** |
| **治疗** | **Atropine**（高剂量直到分泌物干）+ **Pralidoxime / 2-PAM**（OP 专用，再激活 AChE 必须在老化前）|

### 8.3 关键对比

> [!success] "干 vs 湿" 一秒鉴别
> - **干** = Anticholinergic（皮肤干 + 黏膜干 + 肠鸣音 ↓ + tachycardia + 散瞳）
> - **湿** = Cholinergic（流涎流泪 + 出汗腹泻 + 肠鸣音 ↑ + bradycardia + miosis）
> - **湿 + 高热 + 肌阵挛 + 反射亢进** = Serotonin Syndrome / MDMA

### 8.4 抗胆碱能 vs 拟交感（Dry vs Wet）⭐ 高频鉴别

抗胆碱能（TCA / atropine / 抗组胺）与拟交感（cocaine / amphetamine）**都有心动过速 + 高热 + 散瞳 + 躁动**，极易混。鉴别靠"干 vs 湿"：

| 鉴别点 | 抗胆碱能（TCA / atropine / 抗组胺）| 拟交感（cocaine / amphetamine）|
|---|---|---|
| **皮肤** | **干燥**（无汗）⭐ | **湿润**（大汗）⭐ |
| **肠鸣音** | **减弱 / 消失** | 正常或增强 |
| 尿潴留 | 有 | 无 |
| 黏膜 | 干燥 | 正常 |
| 反射 | 正常 | 亢进 |

> [!tip] 一句话：Dry vs Wet
> 皮肤干 + 肠不动 = **抗胆碱能**；大汗 + 肠蠕动 = **拟交感**。
> 与 §8.3 区分：§8.3 是抗胆碱能 vs **胆碱能**；本节是抗胆碱能 vs **拟交感** —— 两个"湿"是不同的药。

## 9. Axis 3 — 戒断综合征（镜像规律 + AWS 4 阶段时间轴 + 对症急救药）⭐

> [!info] 本节定位 — 戒断的"速查 + 时间轴 + 急救药"自包含章节
> 本节给戒断的完整速查框架：**镜像规律 + 致命戒断 + AWS 4 阶段精细时间轴 + 各物质对症急救药速表**。
> 戒断后的**维持期长期用药**见 §15；**停药综合征横向鉴别 + 抗抑郁药 discontinuation** 见 §16。

### 9.1 戒断 = 中毒的镜像

| 物质            | 中毒时                   | 戒断时（镜像）                                                                                      |
| ------------- | --------------------- | -------------------------------------------------------------------------------------------- |
| **Opioid**    | 缩瞳、便秘、嗜睡、**呼吸抑制（致死）** | 散瞳、腹泻、激越、竖毛、流涕、打哈欠（**难受但不致死**）                                                               |
| **Alcohol**   | 镇静、共济失调、呼吸抑制          | 兴奋、震颤、癫痫、**DT/delirium tremens（致命）⭐**；6-24h 震颤 → 12-24h hallucinosis → 24-48h 癫痫 → 48-96h DT |
| **BZD**       | 镇静、共济失调               | 焦虑、震颤、**癫痫（致命）⭐** —— 必须缓慢减量                                                                  |
| **Stimulant** | 亢奋、不眠、食欲↓、欣快          | "**crash**"：嗜睡、食欲↑、抑郁、自杀念头（**情绪致命、生理不致命**）                                                   |
| **Nicotine**  | 兴奋（短）                 | 焦虑、易怒、食欲↑、注意力↓                                                                               |

> [!success] 镜像规律
> **"中毒抑制 → 戒断兴奋；中毒兴奋 → 戒断抑制"**

> [!danger] 致命戒断只有 2 个
> **Alcohol + Benzodiazepine**（同 GABA 通路）→ 必须用 **BZD 替代后缓慢减量**。其他戒断（opioid / stimulant）难受但不致死。
> （巴比妥 Barbiturate 同属 GABA 类，戒断也可致死 —— "致命戒断 = GABA 三兄弟"，详见 §16。）

### 9.2 AWS 4 阶段精细时间轴表 ⭐⭐⭐（Alcohol Withdrawal Syndrome）

```
时间轴：
最后一杯酒 ─► 6-24h 轻度戒断 ─► 12-48h seizure/hallucinosis ─► 48-96h DT ─► 维持期（终身）
                  ↓                      ↓                         ↓               ↓
            BZD 起始              BZD（防进展 + 控制 seizure）   ICU + 大剂量 BZD   Naltrexone/
            （急性期药）              （急性期药）                  （急性期药）       Acamprosate/Disulfiram
                                                                                     （维持期药，见 §15）
```

| 阶段 | 时间窗 | 表现 | 定向力 / 生命体征 |
|---|---|---|---|
| **Mild withdrawal** | **6-24h** | 焦虑、失眠、震颤、出汗、心慌、胃肠不适 | 定向力**正常** |
| **Withdrawal seizure** | **12-48h** | 单次 / 多次 generalized **tonic-clonic** | 发作后 postictal |
| **Alcoholic hallucinosis** | **12-48h** | 视 / 听 / 触幻觉 | 定向力**正常** + 生命体征**稳定** ⭐ |
| **Delirium tremens (DT)** | **48-96h** | 混乱、躁动、发热、心动过速、高血压、出汗、幻觉 | 定向力**丧失** + 自主神经**风暴** + 可致死 |

> [!warning] Alcoholic hallucinosis vs DT — 别混淆
> 都出现在 12-96h 窗口且都有幻觉，但严重度天壤之别：
> - **Alcoholic hallucinosis** = 幻觉 + 定向力**正常** + 生命体征**稳定** → BZD + 支持即可
> - **DT** = 幻觉 + **定向力丧失** + 自主神经**风暴**（HR↑/BP↑/T↑）→ ICU + 大剂量 BZD，**可致死**

> [!tip] AWS 时间窗口口诀 "6-12-12-48"
> **6**h 颤 / **12**h 抽 / **12**h 幻 / **48**h 谵 — 越晚出现越严重。

### 9.3 各物质戒断对症急救药速表

| 物质戒断 | First-line | 关键机制 | 危险信号 |
|---|---|---|---|
| **Alcohol** | **Benzodiazepines** | 替代 GABA-A 兴奋抑制 | DT（震颤谵妄）、戒断性癫痫 — 可致死 |
| **Benzodiazepines** | **长效 BZD 缓慢递减**（Diazepam / Chlordiazepoxide）| 同上 | 戒断性癫痫 — 可致死 |
| **Opioids** | **Methadone / Buprenorphine**（最佳）或 **Clonidine** 对症 | 替代 μ 激动 / 抑制蓝斑交感 | 痛苦但**通常不致死**（成人）|
| **Nicotine** | **NRT** | 替代尼古丁 | 不致死 |
| **Stimulants**（cocaine, amphetamine）| **支持治疗**（无 FDA 药）| — | 抑郁 / 自杀念头 — 心理监测 |
| **Cannabis** | **支持治疗** | — | 不致死 |

> [!danger] 戒断会致死的只有 3 类
> **Alcohol / Benzodiazepines / Barbiturates**（共同点：GABA-A 系统依赖）
> 其它戒断（阿片、可卡因、大麻、尼古丁）"痛苦但不致死"。

## 10. 治疗的 "3 法则" + 必记解毒剂

### 10.1 法则 ① — 默认规则（覆盖 ~70% 题）

> [!success] "激越 → BZD；抑制 → 解毒剂"

```
═══════════════════════════════════════════
激越类（兴奋剂 / 致幻剂 / PCP / 酒精戒断 / BZD 戒断）
        ↓
    第一线 BZD（lorazepam IV/IM）
    第二线 加抗精神病药（haloperidol）
═══════════════════════════════════════════

        ↓
    Opioid           → Naloxone

    Methanol / 乙二醇 → Fomepizole + 透析
═══════════════════════════════════════════
```

### 10.2 法则 ② — 常见药物过量速查（致死机制 · 监测 · 解毒剂）

| 药物 | 致死机制 | 关键监测 / 预后指标 | 解毒剂 |
|---|---|---|---|
| **TCA** | 心肌 Na 通道阻滞 → QRS 增宽 → 室速/室颤 | **QRS > 100 ms** ⭐（> 160 ms 抽搐+室速风险↑）| **NaHCO₃** |
| **Acetaminophen** | NAPQI → 肝坏死 | 血清水平 + Rumack-Matthew 列线图 | **N-acetylcysteine (NAC)**（24h 内最有效）|
| **Aspirin（水杨酸）** | 氧化磷酸化解偶联 + 中枢刺激 | 动脉血气（混合性酸碱）+ 血清水杨酸 | **NaHCO₃** 碱化尿液 + 透析 |
| **Opioid** | 呼吸抑制 | 呼吸频率 + SpO₂ | **Naloxone** |
| **Benzodiazepine** | 呼吸抑制（弱，单用少致死）| 呼吸频率 | Flumazenil（少用 — 慢性用者诱发癫痫）|
| **β-blocker** | 心动过缓 + 低血压 | HR + BP + ECG | **Glucagon** |
| **Calcium channel blocker** | 心动过缓 + 低血压 | HR + BP | **Calcium + 高剂量胰岛素** |
| **Digoxin** | 心律失常 + 高钾 | 血钾 + ECG | **Digoxin 特异性 Fab** |
| **Methanol / 乙二醇** | 代谢性酸中毒（毒性代谢物）| 阴离子间隙 + 渗透压间隙 | **Fomepizole + 透析** |

> [!tip] "5 个核心解毒剂"先背牢
> Opioid → **Naloxone** · APAP → **NAC** · 有机磷 → **Atropine + Pralidoxime** · Methanol/乙二醇 → **Fomepizole** · 抗胆碱重症 → **Physostigmine**（⚠️ **TCA 过量禁用** —— QRS 增宽时诱发心律失常）。

### 10.3 法则 ③ — 不要做的事（高频陷阱）

> [!danger] USMLE 陷阱清单
> - **β-blocker 用于 cocaine 高血压** ❌ → unopposed α，反而恶化（用 phentolamine 或 BZD）
> - **Flumazenil 用于慢性 BZD 使用者** ❌ → 诱发癫痫（仅医源性可用）
> - **Urinary acidification 治 PCP** ❌ → 已废弃（清除率只增 1%，且加重酸中毒）
> - **肾上腺素用于 inhalant 中毒** ❌ → 心律失常猝死（"sudden sniffing death"）
> - **Naloxone 过量** ⚠️ → 突发戒断（流涕、躁动，但不致死）

### 10.4 Flumazenil 详解 ⚠️ —— 为什么慢性 BZD 用者禁用

**Flumazenil** = BZD 受体的**竞争性拮抗剂**。

> [!danger] 核心机制：长期耐受 → 人为极速撤药 → 不可控 seizure
> 如果一个人长期服用 BZDs，他们的大脑已经适应了药物带来的抑制状态。如果你给予氟马西尼，会**瞬间剥夺所有的 BZD 效应，相当于人为制造了"极速撤药"** → 诱发**急性癫痫发作 (Precipitate seizures)**，且由于 **BZD 受体被占据**，传统的苯二氮卓类止惊药（如地西泮）将**无法起效** → **不可控 seizure**。

> [!info] Flumazenil 使用原则
> - **Avoid in chronic benzodiazepine users**（慢性 BZD 用者禁用）
> - **Consider only in selective short-term exposures**（仅在选择性的短期暴露考虑）
> - **Otherwise supportive care**（否则支持治疗）

| 情况 | 推荐处理方案 | 理由 |
|----|----|----|
| 术后苏醒慢（短期使用 Midazolam） | 可以考虑使用 Flumazenil | 患者无耐受，安全性高。 |
| 混合药物中毒（BZD + 来源不明药物） | 禁用 Flumazenil | 极大可能诱发**不可控癫痫**。 |
| 长期失眠患者误服大剂量 BZD | 禁用 Flumazenil | 存在慢性耐受，易导致**撤药反应**。 |
| 单纯 BZD 中毒且呼吸平稳 | 严密观察 + 支持治疗 | 药物中毒通常不致命，支持治疗最稳妥。 |

> [!tip] 一句话锁定
> Flumazenil **只用于"医源性、单纯、短期"的 BZD 镇静**（如术后 Midazolam 苏醒慢）；**慢性用者 / 混合中毒 / 误服 → 一律禁用**（极速撤药 → 受体被占 → 经典 BZD 止惊药失效 → 不可控 seizure）。

## 13. 尿药筛查（UDS）判读 — 假阴性 / 假阳性 ⭐

> [!info] 为什么筛查会"漏"和"误报"
> 常规尿药筛查 = **免疫法（immunoassay）**，靠抗体识别**特定分子结构**。它只对每类药的"**原型化合物**"灵敏 —— 结构改得越多（designer drug）、或不经标准代谢物代谢的药，越容易测不到。**确证试验 = GC-MS / LC-MS**（结构特异；法医 / 就业 / 监护权场景必须做）。

### 13.1 假阴性 — 免疫筛查"漏检"的物质 ⭐

| 筛查类别 | 测得到（原型）| 漏检 / 不可靠 |
|---|---|---|
| **Amphetamine** | amphetamine / methamphetamine | **bath salts / 合成 cathinone**（基本测不出）；**MDMA**（交叉反应不稳定，可阴可阳，不能靠它排除）|
| **Opiate** | morphine / codeine（天然阿片；heroin 经 morphine 代谢物）| **fentanyl / methadone / tramadol / buprenorphine**；oxycodone（半合成，常漏）|
| **Benzodiazepine** | 经 nordiazepam / oxazepam 代谢物（diazepam、chlordiazepoxide 测得好）| **lorazepam / clonazepam / alprazolam**（容易漏）|
| **Cannabinoid (THC)** | THC-COOH | **合成大麻（K2 / Spice）** |

> [!tip] 漏检规律一句话
> 免疫筛查只认"**原型**"。**designer drug（bath salts / 合成大麻）、合成阿片（fentanyl / methadone）、不经标准代谢物的药（lorazepam / clonazepam）** → 都可能筛查阴性。
> → 呼应 §3 callout：**阴性 UDS ≠ 排除中毒，临床表现永远优先。**

### 13.2 假阳性 — 免疫筛查的“误报” ⭐

筛查**阳性 ≠ 一定滥用** —— 免疫法靠抗体识别结构，**结构相似的常用药 / 食物会交叉反应致假阳性**。意外阳性（尤其与病史不符）→ **GC-MS 确证后**才下结论。

| 筛查“假阳性”结果 | 常见诱因（实际在用的药 / 食物）|
|---|---|
| **Amphetamine** | Pseudoephedrine / ephedrine（麻黄碱）、Bupropion、Selegiline、Trazodone、Labetalol、Ranitidine、Atomoxetine、Phentermine |
| **PCP** ⭐ | **NSAIDs（ibuprofen / naproxen）**、Dextromethorphan（DXM 止咳）、Diphenhydramine（DPH）、Venlafaxine |
| **THC（大麻）** | NSAIDs（ibuprofen / naproxen）|
| **Opioids** | Dextromethorphan、**罂粟籽（poppy seed）** |
| **Methadone** | Diphenhydramine |
| **Benzodiazepine / LSD** | Sertraline / Fluoxetine（SSRI）|

> [!warning] UW stem 套路
> ① ADHD 患儿 / 服 bupropion 的人 amphetamine 阳性、② 服 NSAID / DXM 的人 PCP 阳性、③ 吃罂粟籽贝果后 opiate 阳性 → **意外阳性先想假阳性**：不要直接判滥用 / 改治疗 / 上报，先 **GC-MS 确证**（呼应 [[mistakes/uworld-mistakes_2026-05#^Q13573]]）。

### 13.3 两条 USMLE 心法

- **阴性筛查不排除中毒** —— designer drug / 合成衍生物漏检率高 → 看临床表现（呼应 §3、[[mistakes/uworld-mistakes_2026-05#^Q11853]]）。
- **阳性筛查需确证** —— 免疫法假阳性多；涉及法医 / 就业 / 监护权后果时，**GC-MS 确证后**才下结论。

## 14. Opioid 处方规范与监测（PDMP / UDS / 受控物质协议）⭐

> [!info] 定位
> 长期 opioid（慢性疼痛）处方的"安全处方"考点 —— 与 §13 衔接：§13 讲 UDS 结果怎么判读，本节讲 opioid 处方全程怎么监测、misuse 怎么**早发现**。
> 来源 UW Q13547；对标错题 [[mistakes/uworld-mistakes_2026-05#^Q13573]]（长期阿片治疗 + 处方监控 + UDS 假阳性）。

### 14.1 Opioid 处方最佳实践（UW 原表）

| 阶段 | 措施 |
|---|---|
| **启动前 Prior to initiation** | 筛查 misuse 危险因素 · 做基线 UDS · **查 PDMP 数据库** · 签受控物质协议（**1 个处方医生 + 1 家药房**）|
| **治疗中 During therapy** | 每 **1-3 个月**随访 · **每次续方前查 PDMP** · 做**随机** UDS |

### 14.2 Opioid misuse 定义

misuse = 用药偏离处方的「**方式**」（过量、转卖 diversion）或「**目的**」（如拿来抗焦虑）。也包括：从**多个处方医生**拿药、用非处方 opioid、同时用娱乐性药物 / 其他非处方受控物质。

### 14.3 PDMP — 早发现 misuse 的核心工具 ⭐

- **PDMP（prescription drug monitoring program）** = 州级受控物质处方记录（数量 / 开药日期 / 处方医生）。
- 能抓出「**从多个处方医生拿 opioid**」（doctor shopping）→ 提示 overuse / diversion。
- **每次续方前查 PDMP** → 早期发现 misuse 的最佳手段。

### 14.4 易混陷阱 — 为什么早发现 misuse 选 PDMP

> [!warning] misuse 监测 ≠ 疗效监测 ≠ 中毒识别
> - **关节活动度 / 疼痛评分** → 评估**疗效**、指导调量，**不反映 misuse**
> - **生命体征** → 只显示**急性 opioid 中毒**（呼吸↓ / 心率↓ / 低血压），无急性中毒时不能提示 misuse
> - **UDS 必须随机** ⭐ → **scheduled（定期）UDS** 会让患者提前准备 → 结果失真；**random UDS** 才能抓 diversion（尿检 opioid 阴性）/ 娱乐药 / 隐瞒的受控物质
> - **PDMP** → 唯一能抓「多处方医生 doctor shopping」的工具 → **早发现 misuse 最佳**

> [!success] 一句话锁定
> **"早发现 misuse → 查 PDMP；UDS 要随机、不要定期。"**

## 15. 成瘾维持期药物治疗（戒酒 / 戒烟 / 戒阿片）⭐⭐⭐

> [!info] 本节定位 — 维持期"边用 / 已戒"决定选药
> §9 / §16 处理"急性戒断 + 停药综合征"；本节处理戒断之后的**维持期长期用药**。
> 核心难点：**"患者当前是否还在用该物质"决定选哪个药** —— 这是 USMLE 最常考的临床决策点。
> 拿到一道成瘾维持题，按 3 步走：① 是哪种物质 → ② 患者现在什么状态（急性戒断 / 仍在用想戒 / 已戒断防复发）→ ③ 有无禁忌（肝 / 肾 / 心 / 共用其它成瘾物 / 怀孕 / 精神病史）。

### 15.1 AWS seizure 用药对比 — 干扰项陷阱 ⭐（急性戒断 6-96h，BZD 救命药）

急性 AWS 期 **Benzodiazepines = 救命药**。BZD 选药看肝功能：

| 药物 | 特点 | 适用 |
|---|---|---|
| **Chlordiazepoxide** | 长效（半衰期长，自我递减）| 肝功能正常患者首选 |
| **Lorazepam** | 短效，**仅经肾排泄**（无活性代谢产物）| **肝病/老年/危重患者**首选（LOT 法则：Lorazepam, Oxazepam, Temazepam）|
| **Diazepam** | 长效，起效快 | 严重戒断、DT |

> [!tip] LOT 法则记忆
> **L**orazepam / **O**xazepam / **T**emazepam — 这三个 **不经肝代谢**（无活性产物），**老年/肝病** 首选。

| 药物 | 用于 AWS seizure | 机制 | 备注 |
|---|---|---|---|
| **Benzodiazepines** ⭐ | ✅ **First-line** | 直接激动 **GABA-A** → 恢复抑制张力 | Lorazepam / Diazepam IV 起效快 |
| **Phenobarbital** | ✅ 备选（status epilepticus 时加于 BZD 之上）| 也作用 **GABA-A** | 大剂量 BZD 抑制不够时补位 |
| **Phenytoin** | ❌ **无效** ⚠️ | **Na 通道阻断** — 不针对 GABA 失衡 | **USMLE 高频干扰项** |

> [!danger] 陷阱：Phenytoin 治 AWS seizure ❌
> AWS seizure 是 **GABA-A 受体脱敏 + rebound overexcitation** 引起 → 必须**补 GABA 抑制**（BZD 一线，Phenobarbital 补位）。
> Phenytoin 是 **Na 通道阻断**（针对过度兴奋的电传导），**作用点错** → 对 AWS seizure 完全无效。
> 口诀："**抗惊厥三候选**（BZD / Phenobarbital / Phenytoin）都是 seizure 题常见选项，但机制分两派 —— **GABA 派**（BZD + Phenobarbital，沾 GABA-A）vs **Na 通道派**（Phenytoin）；AWS seizure = GABA 失衡 → **只有 GABA 派有效**"。

> [!info] 名字别混：Phenobarbital vs Pentobarbital
> 都是 barbiturate，名字相似但临床角色完全不同：
> - **Pheno**barbital = **长效 AED**（抗癫痫药）—— AWS / status epilepticus **二/三线 + 维持**用药
> - **Pento**barbital = **短效麻醉级 barbiturate** —— **难治性 SE** ICU 持续输注（插管+昏迷诱导）/ 颅高压
>
> **"Pheno → 抗癫痫；Pento → 麻醉重症"**。neuro 笔记 status epilepticus 难治期写的 pentobarbital 是后者。

> [!danger] AUD 戒断会致死
> DT（震颤谵妄）+ 戒断性癫痫 → BZD 是**必给**药，不是可选。
> 注意：alcohol / BZD / barbiturate 三类戒断都依赖 GABA-A 系统，**只有这三类戒断会致死**（见 §9.3）。

> [!warning] 高频场景：术后 / 住院突发 seizure
> 长期酗酒患者术后 24-72h 内 seizure + 自主神经亢进 + 震颤 + 定向力差 → **首想 AWS**（不是 sepsis / meningitis / 头外伤）。
> 住院 / 术后患者常 **underreport alcohol use** —— USMLE 经典"住院后突发症状"鉴别陷阱。

→ 同源酒精指纹见 [[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全#5.5 慢性酒精性多系统指纹]]（MCV↑ + AST/ALT≥2:1）
→ 错题反链：[[mistakes/uworld-mistakes_2026-05#^Q3187]] AWS withdrawal seizure / Phenytoin 陷阱（2026-05-19 入库，触发本节扩展）

### 15.2 AUD 维持期 3 药对比 ⭐⭐⭐（USMLE 高频考点）

| 维度 | **Naltrexone** | **Acamprosate** | **Disulfiram** |
|---|---|---|---|
| **一线/二线** | First-line | First-line | Second-line |
| **机制** | μ-opioid 拮抗 → 喝了不爽 → 减渴 | NMDA/GABA 调节 → 防戒断后稳态失衡 | ALDH 抑制 → 乙醛蓄积 → 喝了想吐（厌恶疗法）|
| **患者是否需先戒酒？** | ❌ **不需要**（边喝边启动）| ✅ **建议先戒酒**（用于已戒断的维持）| ✅ **必须先戒酒**（否则触发反应）|
| **核心适应症** | 渴望强烈、heavy drinking | 已戒断、防复饮 | 高动机、监督下用药 |
| **关键禁忌** | 急性肝炎/肝衰、用阿片（包括术后镇痛）| 严重肾损（CrCl<30）| 心衰、近期使用甲硝唑/含酒精药物 |
| **剂型亮点** | 有月剂型 IM（提高依从性）| 一日 3 次（依从性差）| 需家属/诊所监督 |

> [!danger] 一句话锁定（USMLE 必考）
> - **"Cravings too strong" + still drinking** → **Naltrexone**
> - **"Already abstinent" + 防复饮 + 肝病** → **Acamprosate**（肝衰也能用！）
> - **"Highly motivated" + abstinent + 监督环境** → **Disulfiram**

> [!warning] Disulfiram = second-line，"三必备"缺一不选 ⭐（Q15104 触发）
> 上表 Disulfiram "高动机、监督下用药" 还不够 —— 完整条件是 **3 个必须同时满足**：
> 1. **First-line 失败**（Naltrexone / Acamprosate 试过无效）
> 2. **Highly motivated**（戒酒决心铁 —— 不是"想喝酒"！USMLE 反向陷阱）
> 3. **Supervised**（有人监督服药防作弊，如配偶给药）
>
> 缺任何一条 → 回 first-line。
>
> **核心机制差异**：Disulfiram **不降 craving**（只靠"喝了难受"威慑）→ 有 craving 的患者恰恰需要 **anti-craving 药**（Acamprosate / Naltrexone 直接降冲动），不该用依赖意志力的 Disulfiram。

> [!tip] AUD 药物决策树（Q15104）
> ```
> moderate-severe AUD 需要药物
>   ├─ 有 withdrawal 症状 → Benzodiazepine（短期脱毒，≠ 长期治疗）
>   ├─ 已戒酒 + craving / 复发风险（first-line）
>   │    ├─ 肝病 / 用阿片药 → Acamprosate（肾排泄，肝安全）
>   │    ├─ 肾病 → Naltrexone
>   │    └─ 无禁忌 → Naltrexone 或 Acamprosate 均可
>   └─ first-line 失败 + motivated + supervised → Disulfiram
> ```

### 15.3 Naltrexone 双重身份警告 ⚠️（AUD vs OUD 启动时机）

| 用于 | 启动时机 |
|---|---|
| AUD | **边喝边启动 OK**（喝酒不会触发问题）|
| OUD | **必须先脱毒 7-10 天**（否则诱发严重戒断！）|

> [!warning] 反直觉点
> 同一个药，AUD 时"宽松"，OUD 时"严苛"——因为 AUD 患者体内没阿片，拮抗剂对他无所谓；OUD 患者体内有阿片，拮抗剂会立刻把阿片踢下受体 → 急性戒断爆发。

### 15.4 Nicotine 戒烟 — 一线 / 二线 + 4 阶段行为干预

#### 15.4.1 是否还在抽烟 — 影响 NRT 和 Varenicline 启动时机

| 药物 | 机制 | 启动时机 | 关键禁忌 |
|---|---|---|---|
| **NRT**（贴片/口香糖/含片/吸入剂/喷雾）| 尼古丁替代 | **Quit Day 当天**起始（贴片）；或减量阶段开始 | 急性 MI / 严重心律失常（相对禁忌）|
| **Bupropion**（NDRI 抗抑郁药）| ↑ DA + NE 释放 | **戒烟前 1-2 周**启动，达到稳态后定 Quit Day | ❌ **癫痫** / **饮食障碍**（bulimia/anorexia）/ 近期 MAOI |
| **Varenicline**（α4β2 nAChR 部分激动剂）| 部分激动 → 缓解戒断，又阻断尼古丁奖赏 | **戒烟前 1 周**启动 | 严重肾损调整剂量；既往精神病史需评估 |

> [!warning] Bupropion 双重身份
> - 戒烟（Zyban 商品名）
> - 抗抑郁（Wellbutrin 商品名）
> - **不引起性功能障碍**（对比 SSRI 的优势）
> - **不引起体重增加**（饮食障碍患者吸引但 ❌ 禁用，因癫痫风险↑）

#### 15.4.2 一线 vs 二线

| 一线 | 二线 |
|---|---|
| NRT / Bupropion / Varenicline（任选 1 或组合）| Nortriptyline / Clonidine（副作用大，少用）|

> [!tip] 组合方案
> NRT 长效贴片 + 短效口香糖 / 含片：基础+按需，效果优于单药

#### 15.4.3 分阶段行为干预（Stage of Change）

![[{B919C993-90B0-454E-BA67-18F43DC7801C}.png]]
*↑ UW 原图：Smoking cessation interventions by stage of change（戒烟 4 阶段干预对照）*

> [!info] 烟草成瘾 = 三个层面，药物只覆盖前两个
> - **生理依赖**（physical）—— 戒断、craving → NRT / varenicline / bupropion 处理
> - **心理依赖**（psychological）—— 靠抽烟缓解压力
> - **习惯行为**（habitual）—— 饭后、开车、晨起自动点烟 → **只能靠行为咨询**处理
>
> 所以光给药不够，多数患者还需要 **behavioral counseling**；而行为干预要**按患者所处的"改变阶段"匹配**，不能一刀切。

| 阶段 | 干预重点 |
|---|---|
| **未准备戒**（not ready）| 动机访谈（**4R**：**R**elevance 相关性 / **R**isks 风险 / **R**ewards 益处 / **R**oadblocks 障碍）；每次就诊重复；可给 varenicline 走"边减边戒"策略（→ **持久戒断率 ×2**）|
| **准备戒**（ready）| 定**明确 quit date** + 丢掉所有烟具；行为咨询 + 调动支持系统；开始药物（varenicline / bupropion / NRT）|
| **戒得吃力 / 反复早期复吸**（struggling）| **肯定部分成就**（不否定）；**识别触发 → 链接替代活动**；生物反馈（呼气 CO 监测 / app gamification）|
| **刚戒断**（recently quit）| 祝贺 + 持续支持；药物**再续约 12 周**；引导主动反思"生活有什么改变" |

> [!tip] 核心行为策略：触发 → 替代活动链接（trigger-linking）⭐
> 用于 **struggling 阶段**（已行动、但守不住、反复早期复吸）的患者：
> - 大多数烟瘾是**一过性的，只持续 5-10 分钟** → 熬过去就消退
> - **识别习惯性触发**（晨起咖啡、排便、开车）→ 把它**链接到一个分散注意力的替代活动**（剔牙、嚼口香糖）
> - 替代活动**最好由患者自己提出**（依从性更高）
> - 对**当天第一支烟**做这个链接尤其有效；**能延长"醒来 → 第一支烟"的时间 = 戒烟成功的强预测因子**

> [!warning] 阶段匹配陷阱（USMLE 高频）
> - **已进入 action 阶段**（已定药、已参加支持小组）的患者 → 不要再让他"列戒烟的益处和坏处" —— 那是 **not-ready 阶段**动机访谈的内容；已行动的人需要**具体行为策略**，不是再被说服。
> - **struggling 患者** → 不要简单"重定一个 quit date"：再错过会强化挫败、加重复吸循环。应做**触发识别 + 替代活动链接**。
> - **不要要求"压力降到最低再戒"** —— 生活压力是常态，戒烟咨询本身就包含教患者用更健康的方式应对压力。
> - 恐吓式（烟盒贴病变图）靠 guilt，可能促成一时决心，但**改变不了触发-习惯回路** → 不是持久策略。

### 15.5 OUD 维持期 3 药对比 ⭐⭐⭐

| 维度 | **Methadone** | **Buprenorphine** | **Naltrexone** |
|---|---|---|---|
| **机制** | μ **完全激动剂** | μ **部分激动剂** + κ 拮抗 | μ **拮抗剂** |
| **患者状态** | 在用阿片 / 戒断中均可 | 在用阿片 / 戒断中均可（但要等轻度戒断出现再给）| **必须已脱毒** |
| **启动前提** | 可立即启动 | 等 COWS 评分提示轻度戒断（避免 precipitated withdrawal）| 末次短效阿片 ≥ 7 天；末次长效（美沙酮）≥ 10-14 天 |
| **滥用风险** | 高（黑市流通）| 低（部分激动 + 含 naloxone 复方）| 无（拮抗剂）|
| **处方限制** | 仅限注册诊所（OTP）| 门诊医生可开（DATA 2000 / X-waiver 已取消）| 任何医生可开 |
| **过量风险** | 可致死（呼吸抑制）| 低（部分激动有"天花板效应"）| 无 |
| **孕妇** | ✅ 推荐 | ✅ 推荐 | ❌ 不推荐 |

### 15.6 Precipitated Withdrawal（医源性戒断）⚠️

> [!danger] 患者体内有阿片时，给 Buprenorphine 或 Naltrexone → 把强效阿片踢下 μ 受体 → 急性剧烈戒断
> - **Buprenorphine**：等 COWS ≥ 12 再给
> - **Naltrexone**：等完全脱毒 ≥ 7 天（短效阿片）/ ≥ 10-14 天（美沙酮等长效）

### 15.7 三阶段绑死 — Naltrexone / Naloxone / 戒断管理药 三角关系 ⭐

> [!warning] 高频误区纠正
> 学生常说"Naltrexone 用于酒精和阿片**戒断**，Naloxone 只用于阿片**戒断**" —— **这句话两个错误**。
> 
> **"过量（overdose）" ≠ "戒断（withdrawal）" ≠ "维持（maintenance）"** —— 三个完全不同的临床阶段，对应三类完全不同的药。

| 阶段 | 临床场景 | Alcohol | Opioid |
|---|---|---|---|
| **急性过量**（overdose）| 救命 | 支持治疗（无特效解毒剂）| **Naloxone** |
| **急性戒断**（withdrawal）| 停药后症状管理（震颤/癫痫/痛苦）| **BZD** | **Methadone / Buprenorphine / Clonidine** |
| **维持/防复发**（maintenance）| 长期不让复用 | **Naltrexone / Acamprosate / Disulfiram** | **Naltrexone**（脱毒后）/ Methadone / Buprenorphine |

> [!danger] 两条否定句必背
> - **Naltrexone 不治戒断症状** —— 它治"防复发"。AUD 急性戒断仍要用 BZD；阿片急性戒断仍用 Methadone/Buprenorphine/Clonidine。
> - **Naloxone 不治阿片戒断** —— 它逆转"急性过量"。给戒断中的患者用 Naloxone = 雪上加霜（加重戒断），临床绝不这么做。
> 
> **反向陷阱**：Naltrexone 给正在用阿片的患者 → 诱发严重 precipitated withdrawal（见 §15.6）；Naloxone 给阿片过量患者 → 救命但**会突然把患者推入急性戒断**（短暂、可控，但要预期）。

> [!tip] 一句话锁定三角关系
> - 救命 = **Nalox**one（**N**ow，鼻喷/IV，30 分钟代谢完）
> - 戒断管理 = **BZD（酒）/ Methadone-Buprenorphine（阿片）**
> - 防复发 = **Naltrex**one（**L**ong-term，口服/IM 月剂）

## 16. 停药综合征鉴别（GABA 类 / 阿片 / 抗抑郁药 discontinuation）⭐

> [!info] 本节定位 — 停药 / 漏药 / 减量后会出什么事
> 本节只回答一个问题：**药停了 / 漏药 / 减量后会出什么事** —— 哪些会 seizure、哪些会致命、各自是什么综合征。
> 与 §9（戒断时间轴 + 急救药）、§15（维持期长期用药）互补：本节做**横向鉴别** + 覆盖 §9/§15 不涉及的**抗抑郁药停药（SSRI/SNRI/TCA discontinuation）**。
> **触发错题**：[[mistakes/uworld-mistakes_2026-05#^Q3762]]（短效 BZD 漏药 2 天 → 新发 generalized tonic-clonic seizure）

> [!tip] 术语别混：Withdrawal ≠ Discontinuation
> - **Withdrawal（戒断）** —— 用于会产生**生理依赖**的物质：Alcohol / BZD / Barbiturate / Opioid。
> - **Discontinuation syndrome（停药综合征）** —— 专指 **SSRI/SNRI 等抗抑郁药**。刻意不叫 "withdrawal"，是为了强调抗抑郁药**不是成瘾物质**（无 craving、无 drug-seeking），只是突然撤药后的神经适应反跳。

### 16.1 核心三问决策框架

> [!tip] 拿到"停药 / 漏药 / 减量后出现新症状"的题，按 3 问走
> 1. **会不会 seizure？** —— 只有 **GABA 类**（Alcohol / BZD / Barbiturate）停药会 seizure。
> 2. **会不会致命？** —— 同样只有 **GABA 类**会（seizure + delirium / DT）。
> 3. **什么综合征性质？** —— GABA 类 = withdrawal（兴奋反跳）；阿片 = withdrawal（痛苦但不致命）；SSRI/SNRI = discontinuation（FINISH）。

```
新发症状 ← 最近停药 / 漏药 / 减量？
        │
        ▼
   是哪一类药？
        │
 ┌──────┼───────────────────┐
 ▼      ▼                   ▼
GABA 类   Opioid          抗抑郁药（SSRI / SNRI）
（酒/BZD/  → withdrawal     → discontinuation
 巴比妥）   吐/泻/痛/失眠/      FINISH（见 §16.3）
 → with-   竖毛/散瞳         不致命、无 seizure
   drawal  痛苦但不致命
   焦虑→震颤  无 seizure
   →自主神经
   →SEIZURE
   →delirium
   ⚠️ 可致命
```

> [!danger] 一句话锁定
> **能 seizure + 能致命的停药综合征只有 GABA 三兄弟（Alcohol / BZD / Barbiturate）。** 其它一律"难受但不死、不抽"。

### 16.2 停药综合征大一统对照表

| 类别 | 机制 | 代表药 | Seizure? | 致命? | 特征症状 | 起病窗口 |
|---|---|---|---|---|---|---|
| **BZD withdrawal** | GABA-A 受体下调 → 停药失抑制 | Alprazolam（短效最危险）| ✅ | ⚠️ **可致命** | 焦虑、震颤、失眠 → 自主神经亢进 → **seizure** → delirium | 短效数小时-1 天；长效数天 |
| **Alcohol withdrawal** | 同上（GABA-A）| 乙醇 | ✅ | ⚠️ **可致命** | 6h 震颤 → 12-48h seizure / 幻觉 → 48-96h DT | 6-96h |
| **Barbiturate withdrawal** | 同上（GABA-A）| Phenobarbital | ✅ | ⚠️ **可致命** | 同 BZD（兴奋反跳）| 数天 |
| **Opioid withdrawal** | μ 受体撤药 | Heroin、Oxycodone | ❌ | ❌（痛苦但不致命）| 流泪流涕、散瞳、腹泻、肌痛、竖毛、打哈欠 | 短效 4-12h；长效 24-48h |
| **SSRI/SNRI discontinuation** | 5-HT 突然撤药、神经适应反跳 | Paroxetine、Venlafaxine（短半衰期最重）| ❌ | ❌ | **FINISH**（见 §16.3）| 停药后 1-3 天 |
| **TCA 停药** | 胆碱能反跳（rebound）| Amitriptyline、Clomipramine | ❌ | ❌ | 轻度 **cholinergic rebound**：恶心、腹泻、流泪、出汗、失眠 | 停药后 1-3 天 |

> [!warning] 读表要点
> - **Seizure + 致命 = 只有上 3 行（GABA 类）。** 这是本表最高频的考点。
> - **Opioid withdrawal 很痛苦但不致命**（成人）——"vital signs 大致正常"是它和 GABA 类戒断的决定性区别（详见 §9.3 + UW 6 物质戒断对照表）。
> - **SSRI/SNRI discontinuation 综合征可以很难受，但绝不 seizure** —— "综合征严重 ≠ 会 seizure"是经典陷阱（见 §17）。
> - 酒精 / 阿片 / BZD 戒断的**具体处理用药**见 §9.3 + §15；本表只做"会不会出事"的鉴别。

### 16.3 BZD 专题 — 半衰期决定 withdrawal 风险

#### 16.3.1 BZD 半衰期排序 = withdrawal / seizure 风险

| BZD | 半衰期 | Withdrawal / seizure 风险 | 临床用途 |
|---|---|---|---|
| **Triazolam** | 超短（2-5h）| ⚠️⚠️⚠️ 极高 | 失眠（短效催眠）|
| **Alprazolam (Xanax)** | 短（~12h）| ⚠️⚠️⚠️ **最高** | 焦虑、panic |
| **Lorazepam / Oxazepam / Temazepam** | 中（10-20h）| ⚠️⚠️ 中 | **"LOT" → 肝病 / 老年安全**（无活性代谢物，仅经肾排）|
| **Clonazepam** | 长（30-40h）| ⚠️ 低-中 | 癫痫、panic |
| **Chlordiazepoxide** | 长 | ⚠️ 低 | **酒精 withdrawal 首选** |
| **Diazepam** | 长（>24h，活性代谢物 >100h）| ⚠️ 低（**自我 taper**）| 抗焦虑、肌松、status epilepticus |

> [!danger] 一句话锁定
> 半衰期**越短 → 停药后血药浓度断崖式下跌 → withdrawal 越爆烈 → seizure 风险越高**。
> **Alprazolam = "the Xanax bar"**：短、强、快 → 最高滥用 + 最高 withdrawal。

#### 16.3.2 BZD withdrawal 时间轴

```
长期 BZD 使用 ──► 突然停药 / 漏药
            │
            ▼
GABA-A 受体下调（chronic use 的代偿性适应）
            │
            ▼
突然失去 GABA 抑制 ──► 中枢过度兴奋
            │
   ┌────────┼────────────────────────────────┐
   ▼        ▼              ▼                  ▼
24h 内    1-3 天        严重               最严重
anxiety   autonomic      SEIZURES          psychosis
tremor    hyperactivity  （generalized      delirium
insomnia  （HTN、心动      tonic-clonic）    （可致命）
知觉异常    过速、出汗）
```

> [!info] 与酒精戒断同机制
> BZD 和酒精都作用于 **GABA-A**，所以 BZD withdrawal 和 alcohol withdrawal 的机制、时间轴、致命性**完全平行**。两者都能 seizure、都能进展到 delirium。

#### 16.3.3 长效 BZD 的临床应用（半衰期逻辑的两个落地）

> [!tip] 长半衰期 = 自带"缓释 taper"（降落伞）
> - **戒酒 / 戒 BZD 首选长效**（Chlordiazepoxide / Diazepam）：长效药代谢慢，血药浓度自己平滑下降 = 内置 taper，避免短效药"停了又反跳"的循环。
> - **短效 BZD（Alprazolam）绝不能突然停** —— 必须换长效或缓慢递减，否则 withdrawal seizure。

> [!tip] LOT 法则 —— 肝病 / 老年首选
> **L**orazepam / **O**xazepam / **T**emazepam **不经肝代谢**（无活性代谢物，仅经肾结合排泄）→ 肝病、老年、危重患者首选。
> （急性 AWS 期 BZD 选药同此逻辑，详见 §15.1）

### 16.4 SSRI/SNRI Discontinuation Syndrome（FINISH 助记）

| 项 | 内容 |
|---|---|
| **高危药物** | **短半衰期**：Paroxetine（SSRI 最严重）、Venlafaxine（SNRI 最严重）|
| **低危药物** | **长半衰期**：Fluoxetine（半衰期长 + 活性代谢物 = 自动 taper，几乎不发生）|
| **起病** | 停药后 **1-3 天** |
| **持续** | **1-2 周** |
| **症状（FINISH）** | **F**lu-like（流感样）· **I**nsomnia · **N**ausea · **I**mbalance（头晕 / 失衡）· **S**ensory disturbance（"electric shocks" / brain zaps）· **H**yperarousal（焦虑 / 易激惹）|
| **关键** | **无 seizure**（这是它和 GABA 类 withdrawal 的根本区别）|
| **处理** | 恢复原药 → 之后**缓慢 taper**；或换成长半衰期的 Fluoxetine 过渡 |

> [!warning] 为什么 Fluoxetine 几乎不发生 discontinuation
> Fluoxetine 半衰期长（活性代谢物 norfluoxetine 更长达 1-2 周）→ 停药后血药浓度**自己缓降** = 自带 taper。短半衰期的 Paroxetine / Venlafaxine 停药后断崖式下跌 → 综合征最重。
> **半衰期决定停药反应严重度 —— 这条规律对 BZD（§16.3）和抗抑郁药完全一致。**

### 16.5 用药本身 vs 停药 — seizure 来源鉴别

> [!danger] 本节是 USMLE 最爱的双向陷阱
> "某药 + seizure"出现时，必须分清 seizure 是**用药本身的副作用**还是**停药综合征**。同一组精神科药，有的怕"吃"、有的怕"停"、有的怕"起始 titrate"。

| 药物 / 类别 | **停药** → seizure? | **用药本身** → seizure? | 备注 |
|---|---|---|---|
| **Alprazolam**（短效 BZD）| ✅ **会**（withdrawal seizure，可致命）| ❌ | 用药本身风险是镇静、**依赖**；危险在"停" |
| **Bupropion**（NDRI）| ❌ | ✅ **会**（**剂量相关**）| **eating disorder（bulimia / anorexia）、癫痫史禁用**；电解质紊乱进一步↑风险 |
| **TCA**（Clomipramine / Amitriptyline）| ❌（仅轻度 cholinergic rebound）| ✅ **会**（**过量时**）| 过量三联征：**seizure + arrhythmia + anticholinergic** |
| **Lamotrigine** | ❌（仅在**已有癫痫**患者中诱发）| ❌（药本身的风险是 **SJS/TEN**）| 必须 **slow titration** 防皮疹；是双相维持首选 |
| **SSRI / SNRI** | ❌（discontinuation = FINISH，无 seizure）| ❌ | 综合征再重也不 seizure |
| **Mirtazapine** | ❌ | ❌ | 无明显停药综合征 |

> [!tip] 记忆抓手 —— 每个药都有"特征性风险时机"
> - **Bupropion → "吃的时候"怕 seizure**（瘦人 / eating disorder 禁用）
> - **TCA → "过量时"**三联征（seizure + 心律失常 + 抗胆碱）
> - **Lamotrigine → "起始 titrate 时"怕 SJS/TEN**
> - **Alprazolam → "停的时候"怕 seizure**

> [!warning] Lamotrigine 停药陷阱
> "Lamotrigine 停药 → seizure" 只在患者**本身已有癫痫**时成立（停了抗癫痫药当然会复发）。**无癫痫史的患者停 Lamotrigine 不会 seizure**。Lamotrigine 真正的标志性风险是**起始期 SJS/TEN**。

### 16.6 记忆挂钩

> [!success] 一句话锁定
> "**Short-acting BZD + missed doses + seizure**" = Alprazolam 的经典 stem。

> [!tip] 四条口诀
> - **GABA 药物（BZD + 酒 + 巴比妥）withdrawal → 会 seizure、可致命**
> - **5-HT 药物（SSRI/SNRI）discontinuation → FINISH，不 seizure**
> - **Opioid withdrawal → 痛但不死**
> - **Bupropion / TCA 的 seizure → 用药副作用，不是停药**

> [!tip] 半衰期是贯穿全篇的主线
> - 短半衰期 BZD（Alprazolam / Triazolam）→ withdrawal 最爆烈
> - 短半衰期抗抑郁药（Paroxetine / Venlafaxine）→ discontinuation 最重
> - 长半衰期 = 自带"缓释 taper / 降落伞"：Diazepam 用来戒酒戒 BZD；Fluoxetine 几乎不发生停药综合征

## 17. USMLE 高频陷阱合集

> [!info] 本节定位
> 汇总成瘾维持期 + 停药综合征 + 急性中毒急诊的高频陷阱（已去重，急诊侧陷阱另见 §10.3）。

### 17.1 成瘾维持期陷阱

| 陷阱 | 正解 |
|---|---|
| Stem 提 smoking 12 年 + 问 AUD 用药 → 选 Bupropion ❌ | Bupropion 是**戒烟**药，不治 AUD（Q12541 触发）|
| Stem 患者 still drinking → 选 Disulfiram ❌ | Disulfiram 要**先戒酒** |
| AUD 有 craving / 怕复发 → 选 Disulfiram ❌（Q15104）| Disulfiram **不降 craving**；有 craving 用 **anti-craving 药**（Acamprosate / Naltrexone）|
| 戒酒一段时间 / 患者 motivated → 直接上 Disulfiram ❌（Q15104）| Disulfiram 是 **second-line**，"三必备" = first-line 失败 + motivated + supervised，缺一不选 |
| AUD + 肝衰 → Naltrexone ❌ | 肝衰用 **Acamprosate**（肾排，肝病安全）|
| OUD 患者还在用海洛因 → Naltrexone ❌ | 必须**脱毒 7 天**才能用 Naltrexone |
| 阿片过量 → Naltrexone ❌ | 救命用 **Naloxone**（短效 IV/IM/IN）|
| 急性阿片戒断症状管理 → Naloxone / Naltrexone ❌ | 戒断用 **Methadone / Buprenorphine / Clonidine**；Naloxone 会**加重**戒断（见 §15.5）|
| "Naltrexone 是戒断药" ❌ | Naltrexone 是**防复发**药（维持期），不是戒断症状管理药；AUD 戒断仍用 BZD |
| 老年/肝病 alcohol withdrawal → Chlordiazepoxide ❌ | 用 **Lorazepam**（LOT 法则）|
| AWS seizure → Phenytoin ❌（Q3187 高频干扰项）| AWS seizure 用 **BZD**（GABA-A）；status epilepticus 加 **Phenobarbital**；Phenytoin 是 Na 通道，作用点错 |
| 术后 24-72h 突发 seizure → sepsis / 头外伤 ❌ | 长期酗酒者首想 **AWS**（住院常 underreport alcohol use）|
| 戒烟选 Bupropion + 患者有癫痫 / 神经性贪食 ❌ | Bupropion **禁忌**：癫痫、饮食障碍 |
| BZD 戒断突然停药 ❌ | 必须**缓慢递减**，否则癫痫致死 |

### 17.2 停药综合征 seizure 来源陷阱

| 陷阱 | 真相 |
|---|---|
| "焦虑患者新发 seizure = 原发癫痫" | ❌ 先想**药物**（BZD withdrawal）/ **代谢**（电解质）/ **物质**（酒精 withdrawal）。无家族史 + 急性起病 + 漏药 → 药物因素。|
| "所有停药综合征都会 seizure" | ❌ 只有 **BZD + Alcohol + Barbiturate**（GABA 机制）。Opioid、SSRI/SNRI、TCA 都不会。|
| "Bupropion 突然停 → seizure" | ❌ Bupropion 的 seizure 是**用药本身、剂量相关的副作用**（eating disorder / 电解质紊乱时尤甚），**不是停药**。|
| "TCA 突然停 → seizure" | ❌ TCA seizure 是**过量副作用**（伴 arrhythmia + anticholinergic 三联征）；停药只引起轻度 cholinergic rebound，不 seizure。|
| "Lamotrigine 停药 → seizure" | ❌ 仅在**已有癫痫**的患者中诱发。无癫痫史 → 停药不 seizure。Lamotrigine 真正风险是**起始时 SJS/TEN**。|
| "Paroxetine / Venlafaxine 停药综合征严重 = 会 seizure" | ❌ **综合征严重 ≠ seizure**。SSRI/SNRI discontinuation = FINISH，**始终无 seizure**。|
| "Alprazolam 用药本身会 seizure" | ❌ BZD 用药本身是抗惊厥的；BZD 的 seizure 风险在**停药**。|
| 戒断 + vital signs 正常 → 选 alcohol/BZD 戒断 ❌（Q3189 触发）| Opioid 戒断 vital signs **正常**；vitals 飙 = **alcohol/BZD 戒断**（GABA 类）— "GABA 飙 / 阿片平 / 兴奋剂沉" |
| 看不到 dilated pupils + yawning + piloerection → 不诊断 opioid 戒断 ❌（Q3189 反思）| 这些是 **"may also be evident"**（体检发现，可有可无）；**症状 4-5 联**（吐/泻/痛/失眠/酸痛）才是 **"primarily include"**（指纹必现）。体检缺失**不能否定**诊断，必须用**排除法**（详见 §9.3 + 错题 [[mistakes/uworld-mistakes_2026-05#^Q3189]]）|
| 派对后 seizure + 高热 + myoclonus + SSRI 用户 → 选 cannabis ❌（Q11853 触发）| **MDMA + Serotonin Syndrome**：高热 + 肌阵挛 + 反射亢进 + 低钠 + SSRI 用户 = 指纹（见 §3 / §4 + [[mistakes/uworld-mistakes_2026-05#^Q11853]]）|

### 17.3 急性中毒急救陷阱（重点：Cocaine 胸痛 β-blocker 禁用）

> [!danger] Cocaine 胸痛 — β-blocker 禁用陷阱（unopposed α）⭐
> Cocaine 引起的胸痛 = **冠脉血管收缩**（coronary vasoconstriction）。处理胸痛时：
> - **Benzodiazepines** → 控制 **blood pressure & anxiety**
> - **Aspirin** → 抑制 **cocaine 诱导的血小板聚集**
> - **Nitroglycerin & calcium channel blockers** → 解 **血管收缩性疼痛**
> - 必要时立即心导管（如 STEMI）
>
> **⚠️ 慎用 / 禁用 β-blocker**：β 受体被阻断后，cocaine 对 α 受体的兴奋**失去 β 介导的血管舒张拮抗** → **unopposed α stimulation** → 冠脉血管收缩进一步加重（理论风险）。混合型 α/β-blocker（如 labetalol）有时被考虑，但纯 β-blocker 应避免。
> 同理：**β-blocker 用于 cocaine 高血压**也错（§10.3），首选 phentolamine 或 BZD。

| 陷阱 | 正解 |
|---|---|
| Cocaine 胸痛 / 高血压 → 用 β-blocker ❌ | unopposed α → 冠脉收缩加重；用 **BZD + 硝酸甘油 / CCB**，HTN 用 **phentolamine** |
| Flumazenil 用于慢性 BZD 使用者 / 混合中毒 ❌ | 人为极速撤药 + 受体被占 → **不可控 seizure**；仅医源性短期暴露可用（详见 §10.4）|
| Urinary acidification 治 PCP ❌ | 已废弃（清除率只增 1%，加重酸中毒）|
| 肾上腺素用于 inhalant 中毒 ❌ | 心律失常猝死（"sudden sniffing death"）|

## 18. 自测题

> [!info] 自测题来源
> §18.1 = 成瘾维持期自测（原成瘾药物对照表 §8，10 题）；§18.2 = 停药综合征自测（原停药综合征鉴别速查 §7，8 题）。建议合并后 3 天盲答。

### 18.1 成瘾维持期自测

> [!question]- Q1：45 yo 男性，alcohol use disorder，每日饮 8 罐啤酒，想戒但 cravings 强。无肝病。最佳药物？
> **Naltrexone**（still drinking 也能启动；cravings 强是 Naltrexone 的靶适应症）

> [!question]- Q2：同上患者，但 AST 800 / ALT 600 + bilirubin 5。最佳药物？
> **Acamprosate**（Naltrexone 急性肝炎禁忌；Acamprosate 经肾排，肝病安全）

> [!question]- Q3：38 yo 阿片成瘾者，正在使用海洛因，要求戒毒。可否立即给 Naltrexone？
> ❌ 不可。会诱发 precipitated withdrawal。必须**先脱毒 7-10 天**。可改用 Buprenorphine（等 COWS ≥ 12 后）或 Methadone。

> [!question]- Q4：72 yo 老年男性，alcohol withdrawal 第 1 天，肝硬化 Child B。BZD 怎么选？
> **Lorazepam**（LOT 法则：经肾排，无活性代谢产物，肝病/老年首选）

> [!question]- Q5：30 yo 女性，想戒烟（1 PPD × 10 年），有 bulimia 病史。能选 Bupropion 吗？
> ❌ 不能。Bupropion **禁忌**：饮食障碍 + 癫痫。改用 NRT 或 Varenicline。

> [!question]- Q6：阿片过量呼吸 4 次/分，瞳孔针尖。给哪个药？
> **Naloxone**（不是 Naltrexone！短效救命药）

> [!question]- Q7：35yo 男性术后 30h 出现 tonic-clonic seizure + 震颤 + 出汗 + 反射亢进，长期酗酒史。下一步？给 Phenytoin 行吗？
> **IV BZD**（lorazepam / diazepam）。**Phenytoin ❌ 无效**：AWS seizure 是 GABA-A 脱敏 + rebound overexcitation，Phenytoin 是 Na 通道阻断 → 作用点错。Status epilepticus 时可加 **Phenobarbital**（也是 GABA-A）。

> [!question]- Q8：17yo 男孩 24h 呕吐 / 水样腹泻 / 失眠 / 全身酸痛 + 出汗 + 干黏膜 + **vital signs 全正常**（T 36.6 / HR 88 / BP 110/74）。诊断？为什么不是 BZD 戒断？
> **Heroin (Opioid) Withdrawal**。BZD / Alcohol 戒断**必有 autonomic storm**（HR/BP/T 全飙 + AMS + 可 seizure），本题 vital signs 全正常 → 直接排除 GABA 类戒断。Opioid 戒断 = "痛苦但不死"，vital signs 通常正常。口诀 "GABA 飙 / 阿片平 / 兴奋剂沉"。

> [!question]- Q9：21yo 大学男生派对后 seizure，PMH MDD on sertraline。T 40°C / HR 120 / BP 170/96 / diaphoretic + 肠鸣音↑ + DTR 3+ + myoclonus + Na 122。诊断？为什么不是 cannabis？
> **MDMA (Ecstasy) intoxication + Serotonin Syndrome**。Cannabis 没有 hyperthermia / myoclonus / hyperreflexia / 低钠。MDMA = synthetic amphetamine（推 NE/DA/5-HT），与 SSRI 合用 → 严重 SS。低钠机制：SIADH + 过度饮水降温。"干 vs 湿" 鉴别：anticholinergic（DPH）= 干（皮肤干 + 肠鸣音↓）；MDMA/SS = 湿（diaphoresis + 肠鸣音↑）。

> [!question]- Q10：中年女性 AUD 戒酒 2 周，occasional cravings + 怕复发，无 withdrawal 症状、无肝病、无阿片药。最佳药物？为什么不是 Disulfiram？
> **Acamprosate 或 Naltrexone**（first-line anti-craving 药）。**不选 Disulfiram**：① 是 second-line，需 first-line 失败才考虑 ② Disulfiram 不降 craving，只靠"喝了难受"威慑 ③ "三必备"= failed first-line + highly motivated + supervised，本题一条都不满足。有 craving = 需 anti-craving 药主动降冲动，不靠意志力。

### 18.2 停药综合征自测

> [!question]- Q11：28 岁焦虑女性，PCP 处方某药 6 个月，无癫痫史、无酗酒、无家族史，漏药 2 天后新发 generalized tonic-clonic seizure + confusion。最可能服用的是哪一类药？为什么？
> **短效 BZD（Alprazolam 最典型）**。长期 BZD 使用 → GABA-A 受体下调；突然停药 → GABA 抑制丧失 → 中枢过度兴奋 → withdrawal seizure。短半衰期 = 停药后血药浓度断崖下跌 = withdrawal 最爆烈。无癫痫史 / 家族史 + 急性起病 + 漏药 → 指向药物因素而非原发癫痫。

> [!question]- Q12：把以下 BZD 按 withdrawal / seizure 风险从高到低排：Diazepam、Alprazolam、Lorazepam、Triazolam、Chlordiazepoxide。
> **Triazolam（超短）> Alprazolam（短）> Lorazepam（中）> Chlordiazepoxide ≈ Diazepam（长）**。规律：半衰期越短，风险越高。

> [!question]- Q13：患者停用 Paroxetine 2 天后出现流感样症状、失眠、恶心、头晕、"电击感（brain zaps）"。这是什么综合征？会 seizure 吗？怎么处理？
> **SSRI discontinuation syndrome**（FINISH：Flu-like / Insomnia / Nausea / Imbalance / Sensory disturbance / Hyperarousal）。**不会 seizure**。处理：恢复原药 → 之后缓慢 taper。Paroxetine 半衰期短，是 SSRI 中最易发生的。

> [!question]- Q14：为什么戒酒和戒 BZD 首选长效 BZD（Chlordiazepoxide / Diazepam），而不是 Alprazolam？
> 长半衰期药代谢慢、血药浓度自己平滑下降 = **内置"缓释 taper"**，避免短效药"停了又反跳 → 再给 → 再反跳"的循环。短效 Alprazolam 反而是 withdrawal 风险最高的药。

> [!question]- Q15：下列哪些药的"停药"会导致 seizure：BZD / Alcohol / Opioid / SSRI / TCA？
> **只有 BZD 和 Alcohol**（再加 Barbiturate）—— 都是 GABA-A 机制。Opioid、SSRI、TCA 停药都不会 seizure。

> [!question]- Q16：Bupropion 的 seizure 风险是"用药本身"还是"停药"引起的？哪些患者禁用？
> **用药本身**（剂量相关副作用），不是停药。禁用于 **eating disorder（bulimia / anorexia）和癫痫史**患者；电解质紊乱会进一步增加风险。

> [!question]- Q17：TCA 过量的三联征是什么？TCA 停药会怎样？
> 过量三联征：**seizure + arrhythmia（心律失常）+ anticholinergic（抗胆碱）**。停药则只引起轻度 **cholinergic rebound**（恶心、腹泻、流泪、出汗、失眠），**不 seizure**。

> [!question]- Q18：Lamotrigine 的两个标志性风险时机是什么？一个无癫痫史的双相患者突然停 Lamotrigine 会 seizure 吗？
> 两个时机：① **起始 titration 时 → SJS/TEN**（所以必须慢加量）；② **已有癫痫的患者停药 → 诱发 seizure**。无癫痫史的患者停 Lamotrigine **不会 seizure**。

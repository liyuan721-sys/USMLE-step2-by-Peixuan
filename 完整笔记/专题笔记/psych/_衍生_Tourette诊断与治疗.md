# Tourette 综合征 — 诊断与治疗全套

> [!info] 一句话总览
> **诊断**：>1 yr 多种 motor tic + ≥1 vocal tic，起病 <18 岁。**关键拷问**：echolalia / coprolalia / 重复动作背后**有没有 obsession 驱动**？没有 = tic（Tourette）；有 = OCD。
> **治疗**：三级阶梯 **HRT → α2 激动剂 → 抗多巴胺药**。"用了一段时间仍恶化"= 升级题，找下一阶梯。
> **方向三角**：抗 DA 治 tic / 抗胆碱治 EPS（不治 tic）/ 兴奋剂加重 tic。

## 一、DSM-5 诊断标准

> [!tip] Tourette Syndrome (TS) — DSM-5

| 项 | 内容 |
|---|---|
| **运动 tic** | **≥2 种** motor tic |
| **发声 tic** | **≥1 种** vocal tic（可以不同时出现）|
| **病程** | tic 持续 **>1 年**（频率可消长 wax-and-wane）|
| **起病** | **<18 岁** |
| **排除** | 非药物 / 物质 / 其他疾病引起（如 Huntington / 脑炎后）|

**相关诊断**（病程不够 1 年或单类 tic）：

| 诊断 | 与 TS 区别 |
|---|---|
| **Persistent (chronic) motor or vocal tic disorder** | 只有 motor tic 或只有 vocal tic（不同时），病程 >1 年 |
| **Provisional tic disorder** | tic 病程 **<1 年** |

## 二、神经生物学基础

```
基底节 (basal ganglia) 多巴胺 (DA) 过度活跃
        ↓
   纹状体 D2 通路失衡
        ↓
   抑制不充分的运动 / 发声冲动 → tic
        ↓
   治疗方向：↓DA
        ├─ VMAT2 抑制剂（突触前耗竭 DA）
        └─ D2 拮抗剂（突触后阻断）
```

> [!info] 与 OCD 神经回路对比
> - **Tourette tic** = **基底节 DA 过度活跃**（运动通路）
> - **OCD** = **皮层-纹状体-丘脑-皮层 (CSTC) 环路**异常（认知-情感通路）
> - **治疗方向对应**：tic 抗 DA / OCD SSRI + ERP — **两条独立线**，共病时两条都要走，不能互相替代

## 三、诊断鉴别 ⭐

### 3.1 tic vs OCD 金标准鉴别（高频陷阱）

```
关键拷问："这个重复行为背后有没有侵入性思维 (obsession) 在驱动？"
        │
   ┌────┴────┐
   ▼         ▼
  有        没有
  OCD       tic (Tourette)
(怕 XX →    (纯粹忍不住，
 必须做 YY   有 premonitory urge 先兆冲动，
 缓解焦虑；   做完短暂缓解；
 仪式有主题)  动作无认知"主题")
```

| 维度 | **Tic (Tourette)** | **OCD compulsion** |
|---|---|---|
| **驱动力** | premonitory urge（先兆冲动，**身体**感觉）| obsession（侵入性思维，**心理**焦虑）|
| **认知"主题"** | ❌ 无（纯粹忍不住）| ✅ 有（怕脏 → 洗手；怕忘 → 反复检查）|
| **可短暂抑制** | ✅ 可短暂抑制（**像憋打喷嚏**）| 抑制困难（焦虑暴涨）|
| **wax-and-wane** | ✅ 显著（数周到数月波动）| 相对稳定（应激加重）|
| **典型形式** | 眨眼 / 耸肩 / 转头 / 清嗓子 / **echolalia / coprolalia** | 反复洗手 / 检查 / 排列 / 计数 |
| **治疗方向** | 抗 DA（VMAT2 / D2 拮抗）| SSRI + CBT-ERP |

> [!danger] 高频陷阱（Q19552 触发）
> - **echolalia / coprolalia = 发声性 tic**，**最易误判成 OCD 重复**。USMLE 故意把 echolalia 放在儿童 + 父母 OCD 史的 stem 里测你是否记得"发声 tic"概念。
> - **家族 OCD 史 + 患者"anxious / remorseful"= 烟雾弹**：TD 常共病 OCD / ADHD / 焦虑抑郁，但**不能用家族史或共病情绪定主诊断** — 主诊断回到患者本人**核心症状形式**（重复动作的驱动力是 urge 还是 obsession）。
> - **ERP + fluoxetine 同时出现 = OCD 一线"双钓饵"**：诊断对了 Tourette → 两个都倒（都不是 tic 治疗）；诊断错了 OCD → 两个看起来都对。**B 选项错不是治疗错是诊断错。**

### 3.2 tic 与其他不自主运动鉴别

| 动作 | 关键特征 | 病因 | 治疗 |
|---|---|---|---|
| **Tic** | **可短暂抑制** + premonitory urge | Tourette | 三级阶梯（HRT → α2 → 抗 DA） |
| **Myoclonus** | 突发短促闪电样，**不可抑制** | CJD / AD 晚期 / 缺氧 / 癫痫 | 治原发病 |
| **Chorea** | 不规则**舞蹈样流动**动作 | Huntington / Sydenham（链球菌后）| VMAT2 抑制剂（HD） |
| **Dystonia** | 持续性肌张力异常**扭转** | 抗精神病药急性 EPS / 遗传 | 抗胆碱药（急性 EPS）|
| **Tardive Dyskinesia (TD)** | lip smacking / 舌动 / 舞蹈样肢体 | 抗精神病药**慢性** | 停药 / VMAT2 |
| **Stereotypy** | **节律性、重复**（拍手 / 摇晃），无 urge | 自闭谱系 / 智障 / 正常幼儿 | 多无需药物 |

> [!warning] 关键反射
> 看到 "不自主运动" → 强制问 **"是震颤 / 舞蹈 / myoclonus / tic / dystonia 哪一种？"** —— 抓**关键特征**（可否抑制 / 有无 urge / 动作形态）锁定，不要凭"运动相关"模糊联想。

### 3.3 tic 分类

| 分类 | Simple | Complex |
|---|---|---|
| **Motor** | 眨眼 / 耸肩 / 摇头 / 面部抽搐 | 跳跃 / 摸物 / **copropraxia（淫秽手势）** / echopraxia（模仿动作）|
| **Vocal** | 清嗓子 / 哼哼 / 吸鼻 | **echolalia（重复别人话）** / **coprolalia（淫秽语，<10% TS 患者，非诊断必需）** / palilalia（重复自己话）|

> [!info] coprolalia 不是诊断必需
> 大众文化把 Tourette 等同于 coprolalia，但实际**<10%** 患者有 coprolalia，**不是诊断标准**。USMLE 用 echolalia 测发声 tic 概念更常见。

## 四、治疗三级阶梯 ⭐

### 整体阶梯

```
Tourette 综合征（确诊 — >1 yr motor + vocal tic，起病 <18 岁）
    │
    ├─ 第 1 阶梯：HRT（Habit Reversal Therapy，行为治疗，一线）
    │   ※ 培养 competing response 替代 tic；适合轻中度
    │   │ 无效 / 不可用 / 重度功能受损
    │   ▼
    ├─ 第 2 阶梯：α2 激动剂（证据较弱，共病 ADHD 尤佳）
    │   ※ Guanfacine（首选）/ Clonidine
    │   ※ 镇静 = 副作用 → 助眠 + 减焦虑 → 间接减 tic
    │   │ 无效（如 1 年仍恶化）
    │   ▼
    └─ 第 3 阶梯：抗多巴胺药（最强、证据最足）
        ├─ DA 耗竭：Tetrabenazine / Deutetrabenazine / Valbenazine（VMAT2 抑制剂）
        └─ D2 阻断：Risperidone（SGA 较常用）/ Haloperidol（FGA 强效）/ Pimozide
            ※ Pimozide 注意 QT 延长 → 用前查 ECG
            ※ Risperidone 平衡疗效与 EPS 风险，临床较常用
```

### 4.1 第 1 阶梯：HRT (Habit Reversal Therapy)

| 项 | 内容 |
|---|---|
| **核心 3 步** | ① **awareness training**（识别 tic 的 premonitory urge）<br>② **competing response training**（培养与 tic 不兼容的替代动作，如手抖 → 握拳压腿）<br>③ **social support** |
| **适用** | 轻中度 tic / 患者能配合 / 一线 |
| **与 CBT / CBT-ERP 区别** | ❌ **不是标准 CBT**（CBT 治抑郁/焦虑/惊恐）<br>❌ **不是 CBT-ERP**（治 OCD）<br>✅ HRT 是 **tic 专属行为治疗** |
| **题干识别** | 题给"behavioral therapy"对 Tourette **默认指 HRT** |

### 4.2 第 2 阶梯：α2 激动剂

| 药 | 半衰期 / 用法 | 副作用 | 共病加分 |
|---|---|---|---|
| **Guanfacine**（首选）| 长效（每日 1-2 次）| **镇静轻** + 低血压 | 共病 ADHD + 失眠时尤佳 |
| **Clonidine** | 短效（每日 2-4 次或贴片）| **镇静重** + 低血压 + 突停反跳 HTN | 共病失眠 / tics 重 |

机制：α2 突触前自身受体激动 → ↓NE 释放 → 间接 ↓ 边缘系统兴奋 → 减 tic + 减焦虑 + 助眠。

### 4.3 第 3 阶梯：抗多巴胺药

| 类别 | 代表药 | 机制 | 关键警戒 |
|---|---|---|---|
| **VMAT2 抑制剂**（DA 耗竭）| Tetrabenazine / Deutetrabenazine / Valbenazine | 突触前 VMAT2 抑制 → DA 不能装进囊泡释放 | **无 EPS 风险**，但可诱发**抑郁 / 自杀念头**（HD BBW）|
| **D2 拮抗剂 — SGA**（较常用）| **Risperidone** / Aripiprazole | 突触后 D2 阻断 | 平衡疗效与 EPS / 代谢副作用 |
| **D2 拮抗剂 — FGA**（强效）| Haloperidol / Pimozide / Fluphenazine | 强 D2 阻断 | EPS 风险高 |
| **Pimozide 特别警戒** | — | — | **QT 延长** → 用前 + 滴定中查 ECG；已 QT 长 / 低 K / 低 Mg 时避开 |

> [!success] USMLE 选药策略
> - **没明说禁忌** → 默认 **risperidone**（SGA 平衡）或 **tetrabenazine**（无 EPS 风险）
> - **QT 已长 / 低钾镁** → 避开 pimozide，选 risperidone / tetrabenazine
> - **抑郁史 / 自杀风险** → 避开 tetrabenazine，选 D2 拮抗
> - **代谢综合征 / 体重↑担忧** → 避开 risperidone / olanzapine 类 SGA，选 tetrabenazine 或 aripiprazole

## 五、Tic 治疗方向三角 ⭐

> [!danger] 反射钩（Q3375 错点 — 最易混陷阱）

| 药 | 作用方向 | 治什么？ | 在 tic 题里角色 |
|---|---|---|---|
| **抗多巴胺药**（VMAT2 / D2 拮抗） | **↓ DA** | **治 tic** ✅ | Tourette 第 3 阶梯（HRT + α2 失败后）|
| **抗胆碱药**（benztropine / trihexyphenidyl）| ↓ ACh | **治抗精神病药 EPS**（急性 dystonia / drug-induced parkinsonism） | ❌ **不治 tic**；最常见陷阱选项 |
| **兴奋剂**（methylphenidate / amphetamine）| ↑ DA | 治 ADHD | ❌ **可能加重 tic**；共病 ADHD 时改 α2 或 atomoxetine |
| **SSRI / TCA** | ↑ 5-HT | 治抑郁 / OCD | ❌ 不直接治 tic（共病 OCD 可联用，但不替代）|
| **MAOI** | ↑ NE/5-HT | 难治性抑郁 | ❌ 不治 tic |

### 方向三角口诀

```
DA 太多 → 治 tic     （抗 DA：VMAT2 / D2 拮抗）
ACh 太多 → 治 EPS    （抗胆碱：benztropine）
DA 太少 → 治 PD     （增 DA：levodopa / DA 激动剂）
```

> [!warning] "踩油门 vs 踩刹车"类比
> - 脑子 DA 油门踩太死 (tic) → 要**踩刹车** (anti-DA)
> - 抗胆碱药 = 修"刹车踩过头后的副作用"（治 EPS），**不是踩刹车本身**
> - 兴奋剂 = **再踩一脚油门** = 越踩越糟（加重 tic）

## 六、共病管理

> [!info] Tourette 三大共病

| 共病 | 患病率 | 用药策略 | 关键注意 |
|---|---|---|---|
| **ADHD** | ~50% | **α2 激动剂（首选）** 或 **atomoxetine（NRI 非兴奋剂）** | ❌ 避免 methylphenidate / amphetamine（兴奋剂 ↑DA 可能加重 tic）|
| **OCD** | ~40% | **SSRI + CBT-ERP**（OCD 独立线）| ❌ SSRI **不替代** tic 治疗；如 tic 仍需控制，加抗 DA |
| **焦虑 / 抑郁** | 常见 | **SSRI** 独立线 | 不影响 tic 治疗主线 |

> [!success] 共病联用记忆
> - **Tourette + ADHD** → **α2 一石二鸟**（α2 既治 ADHD 又减 tic）→ 不够再加抗 DA
> - **Tourette + OCD** → **抗 DA + SSRI 双线并行**（两条独立回路，两套药）
> - **Tourette + 焦虑** → **抗 DA + SSRI**（不要把焦虑当 OCD 治）

### 关联错题（共病用药对照）

- Q14348 ADHD 成人首选 atomoxetine（物质滥用史 / 拒成瘾药）
- Q107408 PMT + ADHD 跨场所标准（儿科 ADHD 药物三档 = 兴奋剂 → atomoxetine → α2）

## 七、USMLE 高频陷阱

> [!danger] 容易"自信选错"的反射

| # | 陷阱 | 解药 |
|---|---|---|
| 1 | **echolalia / 重复动作 → 选 OCD 治疗 (ERP / SSRI)** | 关键拷问"有无 obsession 驱动"；发声 tic 是嘴巴打嗝，不是 OCD |
| 2 | **家族 OCD 史 + 患者焦虑 → 诊断 OCD** | 家族史 + 共病情绪 = 烟雾弹；主诊断回患者本人**症状形式** |
| 3 | **tic = 不自主运动 → 选抗胆碱药** | 方向相反：抗胆碱治 EPS（ACh ↑）；tic 治抗 DA（DA ↑）|
| 4 | **共病 ADHD → 加兴奋剂** | 兴奋剂 ↑DA 可能加重 tic；改 α2 / atomoxetine |
| 5 | **"X 药 1 年仍恶化" → 加量同一药** | 升级题，找下一阶梯；α2 失败 → 抗 DA，不是换另一 α2 |
| 6 | **看到 tetrabenazine = 钓饵划掉** | **诊断是 Tourette / HD / TD 时 tetrabenazine 是真答案**；只有焦虑 / 抑郁等无 DA 过度活跃疾病里才是钓饵 |
| 7 | **Pimozide 没看 QT 直接选** | Pimozide 用前必查 ECG；已 QT 长 / 低 K/Mg 时避开，选 risperidone / tetrabenazine |
| 8 | **B 选项治疗对就锁定** | 治疗看起来对 → 先回头确认诊断对没对（OCD 治疗在 Tourette 题里看起来对，但诊断错了 → 治疗也错）|

## 八、Memory Hooks 汇总

> [!success] 一句话锁

- **诊断锁**：「**echolalia / coprolalia = 嘴巴的 tic，不是 OCD**」
- **鉴别拷问**：「**有没有 obsession 在驱动**？有 → OCD；没有 → tic」
- **烟雾弹钩**：「**家族 OCD 史 + 焦虑 = 撒胡椒面，别因为辣就以为是辣椒**」
- **治疗三级钩**：「**HRT 行 → α2 镇静减 tic → 抗 DA 摁油门**」
- **方向三角钩**：「**DA 太多治 tic，ACh 太多治 EPS，DA 太少治 PD**」
- **升级 SOP 钩**：「**用了一段仍恶化 = 升级题，找下一阶梯，不在同阶梯横移**」
- **tetrabenazine 双面钩**：「**焦虑题里出现 = 钓饵；Tourette / HD / TD 题里出现 = 真答案**」
- **共病联用钩**：「**Tourette + ADHD = α2 一石二鸟；Tourette + OCD = 双线并行不替代**」
- **B 选项陷阱钩**：「**治疗看起来对 → 先确认诊断对没对**」

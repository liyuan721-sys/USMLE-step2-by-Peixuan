# 抗精神病药四大 EPS 诊断治疗对照

> [!info] 一句话总览
> 抗精神病药（含 D2 阻断的止吐药）→ 锥体外系反应 (EPS)。**按起病时间轴四分**：dystonia → akathisia → parkinsonism → TD。**每类治疗药不同，混用会加重**（最高频死法 = 抗胆碱能用在 TD 上）。读题先扫「用了什么药 + 多久」+「动作类型」，时间窗 + 动作类型双锁定。

---

## 一、核心机制 — DA / ACh 跷跷板

```
   抗精神病药 / 止吐药
        ↓ 阻断 D2 (穿 BBB)
   纹状体 / 基底节  DA↓ ⇄ ACh 相对↑
        ↓
   不自主运动 / 张力异常 (EPS)
        ↓
   按时间窗演化：
   小时-天   → Acute dystonia
   天-周     → Akathisia
   周-月     → Drug-induced parkinsonism
   月-年     → Tardive dyskinesia (TD)
```

> [!tip] 治疗逻辑
> - Dystonia / parkinsonism = **DA↓·ACh↑** → 压 ACh（**抗胆碱能**：benztropine / diphenhydramine / amantadine）
> - Akathisia = β-能驱动 → β 阻断（**propranolol**）
> - TD = **慢性 D2 阻断 → D2 受体超敏（supersensitivity）**，与急性 DA↓·ACh↑ **机制不同** → 抗胆碱**无效甚至加重**；用 **VMAT2 抑制剂**减少突触前 DA 释放，或**减量 / 换低 D2 亲和药**

## 二、四大 EPS 速查金表（背熟它）

| EPS | 起病时间 | 经典表现 | 一线治疗 | Stem buzzwords |
|---|---|---|---|---|
| **Acute dystonia** 急性肌张力障碍 | **小时–天** | 头颈持续收缩、torticollis 斜颈、**oculogyric crisis 动眼危象**、舌肌痉挛、jaw clenching；神志清醒 | **Benztropine** 或 **diphenhydramine**（IM 数分钟起效）| "stiff neck"、"eyes rolled up"、"tongue protruding"、术后/止吐药打完 1 小时 |
| **Akathisia** 静坐不能 | **天–周** | **主观**内在不安、坐立不能、踱步、fidget；加量更糟 | **减量 + propranolol（首选）**；备选 benztropine / 苯二氮卓 | "can't sit still"、"pacing"、"restless"、加量后更糟 |
| **Drug-induced parkinsonism** 药源性帕金森 | **周–月** | 静止性震颤、**铅管样**僵直、运动迟缓、慌张步态、面具脸 | **减量** / **amantadine** / benztropine | "tremor at rest"、"rigidity"、"mask-like face"、"shuffling gait" |
| **Tardive dyskinesia (TD)** 迟发性运动障碍 | **月–年（>6 月）** | **口面部**舞蹈样：lip smacking、吐舌、grimacing、咀嚼样；肢体/躯干 choreoathetoid 运动；神志清醒 | **VMAT2 抑制剂**（valbenazine / deutetrabenazine）；**减量**（首选但常不可行）；**换 quetiapine / clozapine**；★抗胆碱无效甚至加重★ | "long-term use"、"lip smacking"、"tongue protrusion"、"chewing"、>6 月用药 |

> [!warning] 时间窗 + 治疗药速记
> **"时-天 dystonia → 抗胆碱；天-周 akathisia → β阻滞；周-月 parkinsonism → 减量·amantadine；月-年 TD → VMAT2·换药"**

## 三、治疗药物 → 治什么 / 不治什么 / 加重什么

| 药 | 机制 | 治 | 加重 / 无效 |
|---|---|---|---|
| **Benztropine** 苯托品 | 抗胆碱 | acute dystonia / parkinsonism | **TD（加重）**|
| **Diphenhydramine** 苯海拉明 | 抗组胺 + 强抗胆碱 | acute dystonia | TD |
| **Amantadine** 金刚烷胺 | 促 DA 释放 + 弱 NMDA 拮抗 | parkinsonism | — |
| **Propranolol** 普萘洛尔 | β 阻滞 | **akathisia 首选** | — |
| **Benzodiazepine** | GABA-A 激动 | akathisia 备选 | — |
| **Valbenazine / Deutetrabenazine** | **VMAT2 抑制**（突触前↓ DA 释放）| **TD 一线** | — |
| **Quetiapine / Clozapine** | 低 D2 亲和 SGA | **TD 换药**；clozapine 兼治难治性精分 | — |
| **Haloperidol** | 高效价 D2 阻断 (FGA) | **EPS 治疗题里永远是错答案**（自己就是元凶）| 全部 EPS |

> [!danger] ★最高频死法★
> **"用对药、认错病"** —— Benztropine / diphenhydramine 治 dystonia / parkinsonism 一流，**用到 TD 上无效甚至加重**。看到「lip smacking / 吐舌 / 长期用药数月-数年」→ TD → 选 VMAT2 / 换药，**绝不选 benztropine**。

## 四、各亚型快卡

### 4.1 Acute Dystonia（急性肌张力障碍）

- **谁致病**：D2 拮抗药（高效价 FGA 如 haloperidol；止吐药如 prochlorperazine / metoclopramide）
- **何时**：用药后**小时-天**
- **动作**：**头颈为主**的持续性肌肉收缩 —— torticollis、oculogyric crisis（眼上翻定住）、舌外伸、jaw clenching
- **关键**：神志清醒、生命体征正常
- **一线**：**Benztropine IV/IM** 或 **diphenhydramine IV/IM**（数分钟起效）；可与 D2 止吐药同给作预防
- **陷阱**：误当低钙抽搐（"muscle spasm" 通用词诱导）→ 答案藏在**用药史**

### 4.2 Akathisia（静坐不能）

- **谁致病**：任何抗精神病药（剂量依赖，**最常见 EPS**）；SSRI 也可诱发
- **何时**：换药 / 加量后**天-周**
- **动作**：**主观**内在不安 + 客观 fidget / pacing；不像 dystonia 是被动收缩
- **关键陷阱**：**加量更糟**（误当激越加抗精神病药 = 火上浇油）；易被误读为"焦虑 / 激越"
- **一线**：**减量 + propranolol**；备选 benztropine / 苯二氮卓
- **方向性原则**：剂量是"减"不是"加"；换药换"更低 EPS 风险"不是更高（绝不换 haloperidol）

### 4.3 Drug-Induced Parkinsonism（药源性帕金森）

- **谁致病**：高效价 FGA 最高（haloperidol / fluphenazine）；任何 D2 阻断药（risperidone / metoclopramide 等）
- **何时**：用药 / 加量后**周-月**
- **动作**：**静止性震颤 + 铅管样僵直 + 运动迟缓 + 慌张步态 + 面具脸 + 拖步** —— 与特发性帕金森 (IPD) 形态相似
- **vs Catatonia 的 negativism**：parkinsonism 僵是**被动恒定的铅管 / 齿轮样**阻力；catatonia negativism 是**主动对抗、越推越顶（resistance ∝ force）**
- **DIP 治疗阶梯**：① **减量**（首选，常不可行）→ ② **换低 D2 SGA**（quetiapine / clozapine）→ ③ **amantadine** 或 **benztropine** 对症

> [!danger] ★Levodopa 在抗精神病药致 DIP 中禁用★
> ① 增加 DA → **对抗抗精神病作用、加重精神病**；② DIP 根因是 D2 受体**阻断**（不是 DA 缺失），加 DA 突破不了阻断。**Levodopa 完全不在 DIP 治疗阶梯里**。详见 [[mistakes/uworld-mistakes#^Park1-2026-05-22]]。

**IPD vs DIP 三轴鉴别（「称·程·药」口诀）**

| 维度 | **Idiopathic PD** | **Drug-Induced Parkinsonism** |
|---|---|---|
| **对称性** | **不对称**（一侧主导）⭐ | **对称** |
| **病程** | 数月-数年缓慢进展 | 数周-数月（与用药 / 加量时序相关）|
| **致病药** | 无 | 抗精神病药（FGA > SGA）/ metoclopramide / 罕见 valproate·lithium |
| 嗅觉减退 hyposmia | 早期常见、有特异性 | 一般无 |
| 治疗 | **Levodopa-carbidopa 一线** | **减量 / 换低 D2 SGA**；**禁** levodopa |
| 可逆性 | 不可逆、进展性 | **撤药可逆**（数周-数月）|

完整对照见 [[mistakes/uworld-mistakes#^Park2-2026-05-22]]。

**TRAP 四联**（IPD 临床诊断）：**T**remor at rest / **R**igidity / **A**kinesia-bradykinesia / **P**ostural instability

### 4.4 Tardive Dyskinesia（迟发性运动障碍）

- **谁致病**：长期 D2 阻断 → D2 受体**超敏（supersensitivity）**；FGA > SGA；**risperidone 在 SGA 中较高**；**clozapine / quetiapine 低**；metoclopramide 长期用也可致 TD
- **何时**：用药**月-年（>6 月）**；"tardive" = 迟到
- **动作**：**口面部**为典型 —— lip smacking、吐舌、grimacing、咀嚼样；肢体/躯干 choreoathetoid 运动
- **三阶梯处理**：
  1. **减量**（首选但常不可行——减量会恶化精神病）
  2. **VMAT2 抑制剂** valbenazine / deutetrabenazine（FDA 批准用于 TD）
  3. **换低 TD 风险抗精神病药** quetiapine / **clozapine**；**多药反应差史 → clozapine 优先**（同时是难治性精分的金标准）
- **★死穴★**：**抗胆碱能无效甚至加重 TD**；加量抗精神病药短期"掩盖"长期恶化 → 都是错答案
- **包装陷阱**：出题人常用"情绪 / 行为变化 / 同事觉得她生气 / 自卑 / 独自吃饭"等心理叙事包装客观的不自主运动 → 看到长期用药 + 口面动作直接锁 TD，别被情绪线带跑

## 五、抗精神病药 TD 风险分层

| 类别 | 代表 | TD 风险 |
|---|---|---|
| FGA（高效价）| haloperidol、fluphenazine | **最高** |
| FGA（低效价）| chlorpromazine | 高（另有胆汁淤积性黄疸、QT、抗胆碱副作用）|
| SGA（较高）| **risperidone**、paliperidone | 中-高 |
| SGA（中等）| olanzapine、aripiprazole | 中 |
| SGA（低）| **clozapine、quetiapine** | **最低** |
| 止吐药（D2 拮抗）| metoclopramide、prochlorperazine | 长期用可致 TD |

> [!tip] 选药口诀
> **"已得 TD 想换药 → quetiapine 或 clozapine；多药失败 + 难治精分叠加 TD → clozapine 一药两担。"**

> [!info] 完整 SGA 三维副作用（代谢 / EPS / QT）UW 官方表
> 本节只展示 TD 风险纵切；**全 SGA 三维副作用对比表**（含代谢综合征、EPS、QT 延长 7 药对照）见 [[完整笔记/专题笔记/psych/psych_精神科药物副作用速查#2.2 SGA 各药副作用表]]（图 UW 来源 → [[mistakes/uworld-mistakes#^Park1-2026-05-22]]）。该表与本节 TD 风险分层对齐：**Risperidone EPS 高** ↔ TD 风险中-高；**Clozapine / Quetiapine EPS 极低** ↔ TD 风险最低。

## 六、读题 SOP（防止误判）

```
看到「不自主运动 / 张力异常」病人：
  ① 先扫用药史：用过什么 D2 阻断药？多久了？
  ② 看动作类型：头颈持续？踱步？震颤+僵？口面舞蹈？
  ③ 时间窗 + 动作 双锁定 → 选治疗药
  ④ 反包装：情绪 / 心理叙事可能是不自主运动的伪装
```

- ❌ 别条件反射"加量抗精神病药" —— akathisia 会更糟、TD 会长期恶化
- ❌ 别给 TD 抗胆碱能 —— 无效 / 加重
- ❌ 别给 EPS 治疗题选 haloperidol —— 自己是元凶
- ❌ 别误当低钙抽搐 / 焦虑 / 心理问题 —— 钥匙都在用药史

## 七、易混陷阱（普适）

| 陷阱 | 真相 |
|---|---|
| 抗胆碱能（benztropine）治所有 EPS | ❌ 只治 dystonia / parkinsonism；**对 TD 无效甚至加重** |
| Akathisia = 焦虑 / 激越 → 加量 | ❌ 加量更糟；**减量 + propranolol** |
| TD = 长期 EPS → 给 benztropine 试试 | ❌ TD 机制是 D2 超敏不是 ACh↑；抗胆碱无效 |
| Haloperidol 既然 D2 强、能"治" EPS | ❌ 永远是错答案，自己就是元凶 |
| Lithium 致震颤 = TD | ❌ Lithium tremor 是细震颤，**不是** TD 那种舞蹈样 |
| 加量"掩盖" TD = 处理对 | ❌ 短期掩盖、长期恶化，是陷阱选项 |
| 病人"心情不好、不愿出门" → 心理问题 | ❌ 可能是 TD 的"包装"；先看用药史 + 口面动作 |

## 八、与"高热 + 肌强直急症"的衔接

EPS 是"不发热 / 神志清"族；以下急症族 = "**发热 + 自主神经不稳 + 神志改变**"：

| 急症 | 致病 | 起病 | 关键 | 治疗 |
|---|---|---|---|---|
| **NMS** | D2 阻断（任何抗精神病药）| 数天 | 高热 + 铅管样僵直 + AMS + 自主神经不稳 + CK↑ + 白细胞↑ + 肌红蛋白尿 | 停药 + dantrolene / bromocriptine；难治 / 重症可 ECT |
| **5-HT 综合征** | 5-HT 药物（SSRI/SNRI/MAOI/林奈+其他）| 数小时 | 高热 + **下肢为主 clonus / 反射亢进** + AMS | 停药 + cyproheptadine |
| **Malignant catatonia** | catatonia 叠加恶性化 | 急 | 高热 + 严重肌强直 + 自主神经不稳 + 横纹肌溶解 | Lorazepam 试，危重 **ECT 急迫上**；与 NMS 高度重叠 |
| **Malignant hyperthermia** | 麻醉触发（succinylcholine / 卤代醚）| 麻醉中 | ETCO₂↑↑ + 高热 + 肌强直 | Dantrolene |

> [!info] 见 [[mistakes/uworld-mistakes#^Q12203]] NMS 实验室表现（白细胞↑、肌红蛋白尿）；详细高热肌强直急症鉴别待生成 `_衍生_高热肌强直急症鉴别`。

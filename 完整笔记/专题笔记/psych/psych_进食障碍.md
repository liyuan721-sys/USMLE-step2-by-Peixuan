# 进食障碍与医学并发症 (Eating Disorders & Medical Complications)

> [!info] 用途与来源
> 填补 vault 缺口（psych 专题原无进食障碍）。触发错题：[[mistakes/uworld-mistakes_2026-06#^Q16620]]（男性神经性厌食 → 应力性骨折 → 取膳食史）。

---

## §一 神经性厌食 (Anorexia Nervosa, AN)

> [!note] 诊断三要素
> ① 限制能量摄入致**显著低体重 (BMI <18.5)**；② **强烈怕胖 / 持续妨碍增重的行为**；③ **体象障碍 (distorted body image)**。
> 亚型：限制型 (restricting) / 暴食-清除型 (binge-purge)。

> [!warning] 男性 AN 易漏诊（本题钥匙）
> 男性 AN 常表现为**追求肌肉感/体型 (drive for muscularity)** 而非单纯怕胖 → 膳食史可见**为"增肌"而限制饮食 + 滥用蛋白补剂**。低体重 + **心动过缓** + 对"被迫停止运动"过度痛苦 → 即使男性也要想 AN。

---

## §二 AN 医学并发症（系统）

| 系统 | 并发症 |
|---|---|
| **骨** | **骨密度↓ (decreased bone mineral density, BMD) / 骨质疏松** → **应力性骨折 (stress fracture)** 风险↑ |
| **内分泌** | 闭经 (amenorrhea)、低 T3 综合征、皮质醇↑、生长激素抵抗 |
| **心血管** | **心动过缓 (bradycardia)**、低血压、心肌萎缩、QT 延长/心律失常 |
| **体温/代谢** | 低体温、电解质紊乱 |
| **皮肤/毛发** | 胎毛 (lanugo)、脱发、皮肤干 |
| **胃肠** | 胃轻瘫、便秘 |

> [!danger] 再喂养综合征 (refeeding syndrome)
> 重度营养不良者过快补充热量 → **低磷 (hypophosphatemia)**、低钾、低镁 → 心衰/心律失常/横纹肌溶解。再喂养须**缓慢 + 补磷 + 监测电解质**。

---

## §三 运动相关：相对能量缺乏 / 女性运动员三联

> [!info] 概念
> **相对能量缺乏 (relative energy deficiency in sport, RED-S)** / **女性运动员三联 (female athlete triad)**：
> **能量可用性低（进食失调）+ 月经紊乱 + 骨密度↓** → 应力性骨折。
> → 年轻运动者**反复应力性骨折 + 低体重** = 筛查进食障碍/能量缺乏。

---

## §四 考点：下一步 = 取膳食史

> [!success] 决策（[[mistakes/uworld-mistakes_2026-06#^Q16620]]）
> 低体重 + 应力性骨折 + 对"限制运动"过度 distress → **下一步 = 取全面膳食史 (comprehensive dietary history)**（热量摄入、进食模式、暴食/清除、代偿行为），而非换运动项目或先做手术。

**易混下一步（干扰项）**：

| 选项 | 为何错 |
|---|---|
| 建议换项运动 | 不处理潜在进食障碍 |
| 查外源性类固醇 | 类固醇→增肌量 + 高血压/痤疮/男乳/睾丸萎缩；本例低体重无 |
| 甲状腺功能 | 甲亢致骨密度↓+消瘦，但伴食欲↑/心动过速/震颤（本例反向：心动过缓） |
| 手术 | 小应力性骨折初始 = 休息/镇痛/支具 |

---

## §五 神经性贪食 (Bulimia Nervosa, BN) 对照

| | 神经性厌食 AN | 神经性贪食 BN |
|---|---|---|
| 体重 | **显著低** (BMI<18.5) | 多**正常/略高** |
| 核心 | 限制 + 怕胖 + 体象障碍 | 反复暴食 + 代偿（催吐/泻药/运动） |
| 体征 | 心动过缓/胎毛/闭经 | **腮腺肿大、牙釉质蚀、Russell 征（手背茧）、低钾性代谢性碱中毒** |
| 一线治疗 | 营养复健 + 心理治疗（药物次要） | **认知行为治疗 (cognitive behavioral therapy, CBT) + 氟西汀 (fluoxetine)**（禁安非他酮 bupropion——降癫痫阈） |

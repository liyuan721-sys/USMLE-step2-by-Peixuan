# 功能性神经症状障碍 与躯体症状谱系鉴别

> Functional Neurological Symptom Disorder (FND) / Conversion Disorder 转换障碍
> + Somatic Symptom & Related Disorders（躯体症状及相关障碍五大鉴别）

> [!info] 这张笔记解决什么
> NBME/UWorld 把 **conversion disorder（转换障碍）、somatic symptom disorder（躯体症状障碍）、illness anxiety disorder（疾病焦虑障碍）、factitious disorder（做作性障碍）、malingering（诈病）** 五个挤在一组互为干扰项。
> 定诊靠两条轴：① **症状是否真实/是否有意识制造** ② **动机是什么**。
> FND（功能性神经障碍）尤其要会**正向识别**——抓 positive inconsistency signs（阳性不一致征），不是"查不出病所以归精神科"的排除诊断。
> 派生自 [[NBME/nbme-records/NBME 11/NBME11_错题本#^Q112]]（44F 功能性震颤误选 malingering）。

---

## Part 1 · 五大鉴别（一图锁定）⭐⭐⭐

> [!danger] 核心两轴
> 第一轴：**症状真实吗 / 是不是故意制造的（有无意识）**
> 第二轴：**动机是什么（外部利益 / 病人角色 / 焦虑）**

| 诊断 | 症状真实？ | 有意识制造？ | 动机 / 钩子 | 一句话 |
|---|---|---|---|---|
| **Conversion disorder / FND**（转换障碍/功能性神经症状障碍） | **真实**（患者真感受到） | **否**（非自主） | 无清晰动机；潜在心理冲突 | 有**神经**症状但**不在意**（indifferent） |
| **Somatic symptom disorder**（躯体症状障碍） | **真实** | 否 | 对症状**过度焦虑**、反复就医 | 躯体**有症状** + **非常焦虑** |
| **Illness anxiety disorder**（疾病焦虑障碍，旧称 hypochondriasis 疑病症） | — | 否 | 怕"得了大病" | **没有（或极轻）症状** + 非常焦虑 |
| **Factitious disorder**（做作性障碍，Munchausen 综合征） | 真实或**制造** | **是** | **想扮演"病人角色"**（无外部利益） | 很**享受一直生病**、不想好 |
| **Malingering**（诈病，非精神障碍） | 制造 | **是** | **明确外部利益**（钱/逃避义务/法律/脱罪） | **有目的**地故意装病 |

> [!tip] 记忆钩子（来自分科笔记原话）
> - Malingering → **有目的性故意装病**
> - Factitious → **很享受一直生病不想好**
> - Somatic symptom disorder → **躯体有症状且非常焦虑**
> - Conversion / FND → **有神经症状但不在意**
> - Illness anxiety → **没有症状但非常焦虑**

### 1.1 USMLE 决策树

```
神经/躯体症状 → 5 诊断决策：
   │
   ▼  器质性病因排除？(MRI/EEG/labs 正常)
   ┌── No ──→ 器质性疾病
   │
   ▼ Yes
   症状是真实的吗（患者真感受到）？
   ┌── No（故意制造）──→ 有外部利益？
   │                        ┌── Yes ──→ Malingering（诈病）
   │                        └── No  ──→ 想扮病人角色？──→ Factitious（Munchausen）
   │
   ▼ Yes（真实、非自主）
   是"神经"症状（瘫/麻/震颤/假性发作/感官）？
   ┌── Yes ──→ Conversion Disorder / FND ★（且常 la belle indifférence）
   │
   ▼ No（一般躯体不适）
   有客观躯体症状 + 过度焦虑反复就医？
   ┌── Yes ──→ Somatic Symptom Disorder
   │
   ▼ 几乎无症状只是怕得大病？
   └── Yes ──→ Illness Anxiety Disorder
```

> [!warning] 最常见的两个坑
> 1. **症状"不像真病"≠ malingering / factitious**。诈病/做作必须是**有意识故意制造**；功能性是**非自主**。**功能性 ≠ 故意。**
> 2. **Malingering 必须有"外部获益"硬条件**（赔偿/逃避/法律）。无外部利益却选 malingering = 错（这正是 Q112 的错点）。
> 没有外部利益、是为了"当病人被照顾" → 是 **factitious**，不是 malingering。

## Part 2 · Conversion Disorder / FND 深入

### 2.1 定义

> **Conversion Disorder（转换障碍）= 现称 Functional Neurological Symptom Disorder (FND, 功能性神经症状障碍)**
> 患者出现**真实**的神经系统症状（运动/感觉/意识/感官），但**与已知神经解剖、生理相矛盾**，且**非故意、非伪装**。

```
核心 5 特征：
1. 神经系统症状（运动/感觉/意识/感官）
2. 检查显示"症状与解剖/生理不符"（positive inconsistency）
3. 排除器质性病因（影像/电生理正常 = 支持但非充分；MRI 磁共振 / EEG 脑电图 / labs 实验室检查）
4. 不是装病 ★ 关键（非自主）
5. 通常但不强求有心理应激背景（acute，常紧跟应激）
```

### 2.2 四大症状群

| 类别 | 表现 | 占比 |
|---|---|---|
| **运动症状** | 假性瘫痪 / 假性步态异常 (astasia-abasia) / **假性震颤 functional tremor** | ~50% |
| **感觉症状** | 不符神经分布的感觉缺失/麻木 / 假性失明（管状视野） | ~30% |
| **假性发作/假性晕厥** | PNES (Psychogenic Non-Epileptic Seizure, 心因性非癫痫发作) / psychogenic pseudosyncope (心因性假性晕厥) | ~20% |
| **特殊感官** | 假性失语/失声/失聪/吞咽困难 | 较少 |

### 2.3 阳性不一致征（Positive Signs）⭐ 正向诊断的关键

> [!success] 核心理念
> FND 不是"查不出病就算"，而是要抓 **positive inconsistency signs（阳性不一致征）**——症状在不同情境下**前后不自洽**，影像/电生理正常**支持但非充分**。

**运动症状：**

```
- Hoover 征 ★ 最可靠：functional 下肢无力，患者主动伸髋无力，
  但让其用对侧腿对抗阻力屈髋时，患侧伸髋力量"自动回来"
  （= 力量随注意力转移而恢复 → 非器质性）
- Drift without pronation（抬手下落但不旋前 = 假瘫）
- Give-way weakness（抗阻时"给-放"式忽强忽弱）
```

**功能性震颤特征（4 大正向征）：**

```
1. Distractibility（可分散性）★ 分散注意时震颤减轻或消失 ← Q112 核心钩子
2. Entrainment（频率牵引）让对侧手按固定节律敲击，
   功能性震颤频率会"跟"上那个节律（被牵引/同步）→ 器质性震颤不会
3. Variability（多变）频率/幅度/部位/出现时机变化大
4. Abrupt onset / abrupt remission（突起突停）+ 与器质模式 inconsistent
```

**感觉症状：**

```
- Midline splitting（感觉异常正好卡在中线，器质性会跨过中线少许）
- 非神经分布的 "stocking / glove" 袜套-手套样分布
```

**假性发作（PNES）：**

```
- Forceful eye closure（用力闭眼抗阻睁眼）★ 强特征（真癫痫多睁眼/凝视一侧）
- side-to-side 头/身体左右摇摆、骨盆推动、不对称肢动
- 持续 > 5-10 min；快速清醒、无 postictal confusion（发作后意识混乱）；保留发作记忆
- 常在有目击者时发作；正常乳酸/CK (creatine kinase, 肌酸激酶)；发作期 video-EEG（视频脑电图）无痫样放电
```

### 2.4 功能性震颤 vs 其他震颤（鉴别表）

| 类型 | 何时最明显 | 典型特点 |
|---|---|---|
| **Functional tremor**（功能性） | 可静可动、不固定 | 突然起病；**分散注意后减轻/消失**；entrainment；表现不一致 |
| **Essential tremor**（特发性） | 姿势/动作时 | 双手；逐渐起病；可家族性；**饮酒可缓解** |
| **Parkinson tremor**（帕金森） | **静止时** | 搓丸样；伴运动迟缓、肌强直 |
| **Cerebellar tremor**（小脑） | 接近目标时 | intention tremor；伴共济失调 |
| **Physiologic tremor**（生理性） | 轻微动作/紧张 | 高频低振幅；咖啡因/焦虑/甲亢加重 |

> [!example]- Q112 stem 模式（44F 功能性震颤）
> 44F 2 周右手进行性震颤+慢步态，怕像父亲得帕金森；**静止有震颤、指鼻(动作)时反而加重、左手玩硬币(分心)时减轻**；步慢但平衡正常；MRI 正常；工作压力大。
> → **静止有震颤 + 动作放大 + 分心减轻 = 模式自相矛盾 + distractibility = 功能性震颤**，不是 Parkinson，更不是 malingering（无外部获益）。

### 2.5 La Belle Indifférence（美丽的冷漠）⭐

> **La belle indifférence**：患者对自己严重的神经症状表现出**异常平静、漠不关心**（"有神经症状但不在意"）。

```
现代证据：敏感性 + 特异性都差（conversion 仅 21% 有，stroke 也 29% 有），
DSM-5 已从必要标准移除。
→ 但 USMLE 仍把它作为经典提示（教科书答题 ≠ 临床决策）。
```

> [!warning] 见到这些措辞 → 几乎确定 conversion
> - "She appears **unconcerned** about her sudden inability to walk"
> - "Despite complete paralysis, she **calmly** describes symptoms while **smiling**"

## Part 3 · 治疗与医患沟通 ⭐

> [!danger] 第一原则：不要直接说"是心理的/是假的"
> 会让患者感觉被否定、撤回信任。应说：**"症状是真实的，但检查显示不是结构性病变 / 大脑发出了错误信号，但结构是好的。"**

```
FND 治疗阶梯：
Step 1: 教育 + 共情验证症状真实性（非对抗式告知诊断）
Step 2: Physical therapy（运动症状首选起步）
Step 3: CBT (Cognitive Behavioral Therapy, 认知行为治疗) ★ 首选心理治疗
Step 4: SSRIs (Selective Serotonin Reuptake Inhibitors, 选择性5-羟色胺再摄取抑制剂)（若合并焦虑/抑郁）
Step 5: 多学科团队（神经+精神+物理+心理）
不做不必要的检查（防强化"病人角色"）

预后好：急性起病 / 短病程 / 已识别应激源 / 早治疗
预后差：慢性(>6mo) / 多共病 / 继发获益 / 依从差
```

**躯体症状障碍家族通用管理（somatoform management）：**

```
- 建立稳定医患关系（single provider）
- 定期、简短的随访（scheduled short visits）
- 最小化检查与转诊（minimal testing）
- 逐步建立信任，关注功能而非"找病"
- factitious imposed on self → psychotherapy（心理治疗）
```

## Part 4 · USMLE Stem 模板速认

| 诊断 | 典型 stem 钩子 |
|---|---|
| **Conversion / FND** | 应激事件**次日**突发瘫/震颤/失明；Hoover+ / distractible tremor；**unconcerned, smiling**；影像正常 |
| **Factitious (Munchausen)** | 多次住院（如 17 次）；**医护职业**；拒绝家属联系；对反复检查"满意"；如**偷用胰岛素 insulin / 磺脲类 sulfonylurea**自伤（insulin 胰岛素↑ + C-peptide C肽↑ → 磺脲类）|
| **Malingering** | 工伤后腰痛 + **申请工伤赔偿/逃避**；仅被观察时夸大；体检不一致 + **明确外部利益** |
| **Somatic symptom d/o** | 慢性腹痛 6 月，看了 12 个医生、8 次正常内镜，**坚信有胃癌**、极度焦虑 |
| **Illness anxiety d/o** | **几乎无客观症状**，但反复担心/检查自己得了重病 |

> [!tip] 病因/风险线索
> **childhood abuse / 成人 sexual trauma → 终生 somatic symptom disorder 风险↑**；FND/PNES 也常有创伤史 → stem 出现创伤史是加分锚点，但**定诊仍靠阳性体征不一致**。

# 精神障碍选对治疗 SOP（v1.2 — 2026-05-30 加 §3.7 急性激越即刻处理正向 SOP）

> [!info] 这是什么
> 把"诊断对了 → 治疗错了"这一类错题统一收口的 SOP。核心抓手：**先定准诊断 + 亚型 → 再按亚型对号入座**，不要凭"大方向感觉"选治疗。
>
> 触发来源：2026-05-23 单日 psych 治疗 4 连击（[[mistakes/uworld-mistakes_2026-05#^Q14348|Q14348]] ADHD / [[mistakes/uworld-mistakes_2026-05#^Q16153|Q16153]] GAD / [[mistakes/uworld-mistakes_2026-05#^Q16490|Q16490]] BED / [[mistakes/uworld-mistakes_2026-05#^Q8923|Q8923]] OCD / [[mistakes/uworld-mistakes_2026-05#^Q15088-redo|Q15088 重做仍错]]）+ 历史同框架题（[[mistakes/uworld-mistakes_2026-05#^Q15811|Q15811]] / [[mistakes/uworld-mistakes_2026-05#^Q2496|Q2496]] / [[mistakes/uworld-mistakes_2026-05#^Q2353|Q2353]] / [[mistakes/uworld-mistakes_2026-05#^Q12541|Q12541]] 等）。

## 一、5 大「选对治疗」陷阱（先看这个，再看下面的表）

> [!danger] 反复出现的核心陷阱
>
> 1. **大方向对，亚型错**：知道是焦虑谱系/抑郁/进食障碍，漏掉"哪个亚型" → 选了同谱系但邻居亚型的一线药（Fluoxetine 给了 AN、Clonidine 给了成人 ADHD）。
> 2. **主诉 ≠ 治疗目标**：病人主诉是烟幕弹，治真病不治表诉（BED 主诉「想减肥」→ 别给减肥药）。
> 3. **关键词反射 — 没数齐线索**：一条显眼 buzz 词就锁答案（GAD 题里一条社交担忧 → 锁 SAD；MDD 题里一条「不急」→ 觉得妄想可不治）。
> 4. **熟悉的药 ≠ 适应症的药**：知道是 nonstimulant / 知道要联合，但具体哪个填空错（Clonidine vs Atomoxetine / Lithium vs Risperidone）。
> 5. **不急 ≠ 不治**（[[mistakes/uworld-mistakes_2026-05#^Q15088-redo|Q15088 反复错根因]]）：治疗的"紧迫性"≠ "必要性"。妄想不治会一直拖；急不急只决定**要不要 ECT/急救**，不决定**要不要对因治疗**。

## 二、SOP 三步法

```
Step 1：盘点 stem 全部线索（写下所有担忧/症状/病程/既往用药/共病）
        │ ⚠️ 不能被一个 buzz 词锁定 → 一定要"清点完才下结论"
        ▼
Step 2：定准诊断 + 亚型（不只是"焦虑谱系"，要到"GAD" vs "SAD"）
        │ 亚型决定治疗，不是谱系决定治疗
        ▼
Step 3：按亚型对号入座（查 §三表，排掉同谱系邻居的一线药）
        │ 同时排"听起来贴合主诉但适应症错"的诱饵
        ▼
        正确治疗
```

## 三、对号入座表（核心交付，背它）

### 3.1 ADHD 用药决策

| 亚型 | 一线 | 备选 | 排除诱饵 |
|---|---|---|---|
| **成人 + stimulant 可用** | Methylphenidate / Amphetamine | — | — |
| **成人 + 物质滥用史/拒成瘾药** ⭐ | **Atomoxetine（NRI，无成瘾）** | — | Clonidine（儿童款）/ Modafinil（仍 scheduled）|
| **儿童一般** | Methylphenidate / Amphetamine | — | — |
| **儿童 + tics/失眠** | **Clonidine / Guanfacine（α2-agonist）** | — | — |
| **< 6 岁 / 仅家中行为问题** | **PMT 行为干预** | — | 直接上药 |

> [!warning] 同谱系邻居陷阱
> Atomoxetine（成人）vs Clonidine/Guanfacine（儿童）— 都是 nonstimulant，但**人群不同**。成人题里出现 Clonidine = 干扰项。
> - 错题：[[mistakes/uworld-mistakes_2026-05#^Q14348|Q14348]] 成人 + 物质滥用 → 误选 Clonidine
> - 反例：[[mistakes/uworld-mistakes_2026-05#^Q107408|Q107408]] 儿童仅家中行为问题 → PMT 而非药物

### 3.2 焦虑谱系（先数靶点 → 再分配药）

| 亚型 | 担忧靶点 | 一线 | 备选 | 排除 |
|---|---|---|---|---|
| **GAD** | 多个、不相关领域 | SSRI/SNRI + CBT | **Buspirone**（无成瘾、专属 GAD）| BZD 长期 |
| **Social Anxiety** | 单一：社交被负评 | SSRI + CBT | β-blocker（performance only）| — |
| **Separation Anxiety**（儿童）| 单一：与照护者分离 | CBT；SSRI 备选 | — | — |
| **OCD** | 侵入思维 + 仪式行为 | **SSRI 高剂量 + CBT-ERP** | **Clomipramine（TCA 二线）** | **Psychodynamic / DBT / Buspirone** |
| **Panic Disorder** 急性发作 | 反复不可预期 panic | **BZD 救急 + SSRI 维持** | — | SSRI 单药救急（4-6 周起效太慢）|
| **Specific Phobia** | 单一：特定物体/情境 | Exposure therapy | — | SSRI（无效）|
| **PTSD** | 创伤再体验 + 回避 | SSRI/SNRI；Prazosin（噩梦） | Trauma-focused CBT | BZD（恶化）|

> [!warning] 焦虑题反陷阱
> - **一条社交担忧 ≠ Social Anxiety**：先数 stem 里担忧靶点数量。多领域 → GAD（[[mistakes/uworld-mistakes_2026-05#^Q16153|Q16153]]）。
> - **OCD 心理治疗只认 CBT-ERP**：Psychodynamic 在 OCD 题里 = 诱饵（[[mistakes/uworld-mistakes_2026-05#^Q8923|Q8923]]）。
> - **Buspirone 只 buy GAD 一种保险**，不治 OCD / Panic / Social。
> - **生活事件外壳 ≠ 单一议题**：怀孕/学业/搬家是"外壳"，看担忧是否辐射多领域（[[mistakes/uworld-mistakes_2026-05#^Q21090|Q21090]] 围产期 GAD）。
> - **冷僻药 ≠ 答案**：焦虑题里看到 **Tetrabenazine / Valbenazine / Deutetrabenazine** = VMAT2 抑制剂（Tourette / HD / TD），与焦虑零关系 → 直接划掉（[[mistakes/uworld-mistakes_2026-05#^Q20930|Q20930]]）。

#### 3.2.1 焦虑用药一句话区分（5 个常见 + 干扰项）

| 药 | 适应症 | 起效 | 成瘾性 | 关键定位 |
|---|---|---|---|---|
| **SSRI**（sertraline / escitalopram / fluoxetine） | **慢性 GAD / Panic / SAD / OCD / PTSD 一线**（儿童同）| 4-6 周 | 无 | 焦虑谱全场 MVP；儿童 BBW 低剂量 + 监测 |
| **SNRI**（venlafaxine / duloxetine）| GAD / Panic / SAD（duloxetine FDA 批 pediatric GAD ≥7岁）| 4-6 周 | 无 | SSRI 同位；GAD + 慢痛共病优 |
| **Buspirone** | **GAD 二线专属**（augment SSRI）| 2-4 周 | **无** | 无镇静、无成瘾；不治 OCD/Panic/SAD |
| **Benzodiazepine**（lorazepam / alprazolam / clonazepam）| **Panic 救急 + GAD 桥接 SSRI 起效期** | 分钟 | **有** + 呼吸抑制 | 短期；PTSD 禁用（恶化）|
| **Hydroxyzine**（H1 抗组胺） | **短期 / 情境性**（preop 焦虑 / 急性应激 / 瘙痒）| 30-60 min | **无** | **不治慢性 GAD**；老年人 Beers（抗胆碱）|
| **Propranolol**（β-blocker） | **Performance anxiety 仅**（演讲 / 表演 / 公开讲话）| 30-60 min | 无 | 控躯体（震颤 / 心悸）；不治弥漫性 GAD |
| **Tetrabenazine / Valbenazine / Deutetrabenazine**（VMAT2 抑制剂） | **Tourette / Huntington 舞蹈病 / Tardive dyskinesia** | — | — | **❌ 不治焦虑**；焦虑题里 = 钓饵 |

> [!tip] 短期 vs 慢性 vs 表演型「三把锁」
> - **短期 / preop**：Hydroxyzine（无成瘾）或 BZD（更快但有成瘾）
> - **慢性弥漫**：SSRI/SNRI 一线 → Buspirone 二线（绝不上 BZD 长期、绝不靠 Hydroxyzine 维持）
> - **表演型**：Propranolol（β 阻 → 控躯体）
> - **冷僻钓饵**：Tetrabenazine 类 → 划

### 3.3 进食障碍三兄弟

| 亚型 | 关键鉴别 | 一线 | 备选 | **禁用** |
|---|---|---|---|---|
| **Anorexia (AN)** | BMI<18.5 + 怕胖 + 体象扭曲 | **无药一线（CBT + 营养）** | Olanzapine（一线无效时） | — |
| **Bulimia** | 暴食 + **有代偿** + 体重常正常 | **Fluoxetine** | 其他 SSRI | **Bupropion（癫痫阈值↓）** |
| **BED** | 暴食 + **无代偿** + 常超重 | **SSRI（sertraline）** | **Lisdexamfetamine**、Topiramate | Bupropion |

> [!warning] 减肥药 ≠ BED 药
>
> | 药 | 适应症 | 治 BED？|
> |---|---|---|
> | Orlistat（脂肪酶抑制剂）| 肥胖 | ❌ |
> | Phentermine（拟交感）| 肥胖（短期）| ❌ |
> | Bupropion-naltrexone | 肥胖 | ❌（进食障碍慎用） |
> | **Lisdexamfetamine** | ADHD + **BED** ⭐ | ✅（BED 唯一 FDA 批准兴奋剂）|
> | **SSRI** | 抑郁/焦虑/进食障碍 | ✅ |
>
> - 错题：[[mistakes/uworld-mistakes_2026-05#^Q16490|Q16490]] BED → 误选 Orlistat
> - 错题：[[mistakes/uworld-mistakes_2026-05#^Q2353|Q2353]] AN + euthyroid sick → Fluoxetine 是 bulimia 一线 ≠ AN

### 3.4 MDD 治疗按「有无精神病性」分流 ⭐⭐⭐

```
        重度抑郁 + 妄想/幻觉？
                 │
       ┌─────────┴──────────┐
      YES                   NO
       │                     │
 精神病性抑郁           普通/难治非精神病 MDD
       │                     │
       ▼                     ▼
 抗抑郁药 + 抗精神病药      单药 → 无效则:
 (或 ECT)                  ① SWITCH 换药（完全无反应）
       │                   ② AUGMENT 增效（部分反应）→ Lithium/T3/atypical AP
       │                   ③ COMBINE 加第二 AD → +bupropion/mirtazapine
       ▼
 急不急？
   不急 → 口服 AD+AP（sertraline+risperidone）
   急（拒食/急自杀/妊娠/catatonia）→ ECT
        │
        └─ ⚠️ ≠ IM 抗精神病药（那是治"急性激越"的，不治精神病性抑郁本体；
                                catatonia 用 BZD/ECT，禁用 antipsychotic）
```

| 情境 | 一线 | 药物示例 |
|---|---|---|
| 抑郁 + 妄想/幻觉（不急） | **AD + AP**（必须有 AP）| sertraline + risperidone |
| 抑郁 + 妄想/幻觉（急）| **ECT** | — |
| 抑郁，单药有效 | 抗抑郁单药 | SSRI/SNRI/mirtazapine |
| 难治**非**精神病 | 换药 / 双 AD / AD + 增效剂 | escitalopram + bupropion；SSRI + lithium/T3 |
| 双相抑郁 | Quetiapine / Lurasidone / Lamotrigine / Lithium / **OFC** | — |

> [!danger] 「急」要拆成两件不同的事 — IM antipsychotic 治的是「行为急症」，不是「精神病性抑郁本体」
>
> | 「急」的内容 | 真正含义 | 一线 | IM 抗精神病药角色 |
> |---|---|---|---|
> | **拒食/拒水** | 需快速起效（药 4-6 周太慢；无法口服）| **ECT** | ❌ 不解决问题（只压症状，治不了抑郁） |
> | **急性自杀** | 需数天显效防自杀 | **ECT** | ❌ 同上 |
> | **妊娠** | 避免胎儿致畸药物 | **ECT**（无绝对禁忌、对胎儿相对安全）| ⚠️ 可用但有 EPS/代谢风险 |
> | **Catatonia** | 紧张症本身 | **Lorazepam IV challenge**（2-4 mg）→ 无效 → **ECT** | ❌❌ **禁用**（可加重 + 诱发 NMS）|
> | **急性激越/攻击性/失控** | 行为安全急症（**不是 MDD 本体**）| **IM haloperidol ± lorazepam** 或 **IM olanzapine/ziprasidone** | ✅ **这才是 IM 的真适应症** |
>
> **核心逻辑链断点**：「急 → 需要快 → IM 起效快 → 选 IM 抗精神病药」 断在第 3 步：
> - **IM 抗精神病药**起效分钟级，但只压**激越/敌意/精神病性烟雾**，**对抑郁本体零作用**
> - **ECT**起效"慢一点"（数次治疗 = 数天），但**同时治抑郁 + 精神病性症状**
>
> 精神病性抑郁的"急"病人**通常不是激越的** — 是**精神运动迟滞、退缩、拒食**（[[mistakes/uworld-mistakes_2026-05#^Q15088-redo|Q15088]] "whispers / slow"）。这种病人给 IM haloperidol 没用 — 他不在打你，他在腐烂。
>
> **Catatonia 雷区（必背）**：单用抗精神病药（IM 或口服）严禁，可能**加重 catatonia + 诱发恶性 NMS**（NMS 有学派视为 catatonia 极端亚型）。一线 = lorazepam IV，二线 = ECT。
>
> **IM 抗精神病药真正适应症**：见 [[mistakes/uworld-mistakes_2026-05#^Q2355|Q2355]]（急性躁狂激越拒口服 → IM olanzapine/haloperidol）/ [[mistakes/uworld-mistakes_2026-05#^Q15856|Q15856]]（急性躁动 lorazepam ± haloperidol IM 三件套）— stem 关键词 = 砸门/扔东西/攻击性/拒绝口服。

> [!danger] 联合治疗对号入座（防 C/B/F 混淆 — Q15088 反复错根因）
>
> | 联合方案 | 治什么 | 第二个药作用 |
> |---|---|---|
> | AD + **抗精神病** | **精神病性抑郁** | 打掉妄想/幻觉 |
> | AD + **Lithium / T3** | **非精神病**难治抑郁 | 增效 augmentation |
> | AD + 另一 AD | **非精神病**难治抑郁 | 双机制增强 |
>
> **Lithium 在抑郁里的特殊价值**（不是"温和版 AP"）：
> - 经典增效剂
> - **独立降自杀风险**证据
> - 双相核心心境稳定剂
> - **对妄想/幻觉零作用** ← 这是 Q15088 反复错的根
>
> 错题：[[mistakes/uworld-mistakes_2026-05#^Q15088|Q15088 首次错]] / [[mistakes/uworld-mistakes_2026-05#^Q15088-redo|Q15088 重做仍错]] / [[mistakes/uworld-mistakes_2026-05#^Q15811|Q15811]] → ECT

### 3.4.bis Atypical MDD 6 阶梯 — first-line = SSRI（不是 MAOI！）⭐⭐⭐（v1.1 新增 2026-05-28）

> [!danger] 5-28 浩斯宾 Q17 暴露 — UWorld 反射"Atypical = phenelzine"外推到 first-line 错
> Atypical MDD 治疗阶梯**和普通 MDD 完全一样从 SSRI 起步**；"Atypical 经典专属 MAOI（phenelzine）"是 70-90 年代教学遗留，**modern Step 2 / UWorld 2024 已修正为 SSRI 一线**。MAOI 仅 multi-step 失败后才用。

#### Atypical MDD 诊断三要素

- ⭐ **Mood reactivity**（心境反应性 — 能对正性事件高兴）
- ≥ 2 项：① 食欲/体重↑ ② Hypersomnia 嗜睡 ③ **Leaden paralysis（铅样麻痹）** ④ Interpersonal rejection sensitivity

#### 6 阶梯治疗 ⭐

```
First-line:    SSRI (Sertraline / Fluoxetine) 或 SNRI (Venlafaxine)
                ↓ 6-8 weeks no response / intolerance
Second-line:   Bupropion 或 Mirtazapine
                ↓ partial response
Augment:       Lithium / T3（甲状腺素）/ Aripiprazole
                ↓ multi-step 失败 = 难治性
MAOI:          Phenelzine ⭐ atypical 经典专属（modern = "难治性首选 MAOI"）
                ↓ 极难治
ECT / TMS / Esketamine
```

#### 反射钩修正 ⭐

> [!success] 一句话钉死
> **"Atypical 一线 — 现代 SSRI；MAOI 留给走投无路"** ⭐
>
> - 看到 atypical buzzword（leaden paralysis / mood reactivity / hypersomnia / 增重）→ **先想 SSRI/SNRI first-line**
> - 看到 "SSRI tried 6 weeks 无 response + Bupropion tried 4 weeks 无 response + augmentation 失败" → 才反射 phenelzine
> - 仅有 "history of MDE" ≠ "SSRI failed" — 必须 explicit 写明前药 trial 失败

#### MAOI Phenelzine 启用前 3 禁忌（必背）

- **饮食 tyramine 限制** — chesese / aged meat / wine / fermented foods（hypertensive crisis 风险）
- **Serotonergic 药 14 day washout** — SSRI / SNRI / triptans / linezolid / tramadol / TCA（SS 风险）
- **Sympathomimetic 禁用** — pseudoephedrine / amphetamine / cocaine / MDMA（hypertensive crisis）

#### 易混陷阱 — "Atypical 经典专属 MAOI" 误外推

| 时代 | 教学 | 是否 first-line |
|---|---|---|
| **70-90 年代** | "Atypical depression → MAOI is the drug of choice" | 历史 first-line（已淘汰）|
| **现代 2024 / UWorld Step 2** | "SSRI/SNRI first-line + MAOI 留给 augment 失败" | ❌ first-line 是 SSRI；MAOI 多步失败后才用 |

> [!warning] 反 pattern 警示
> **"经典专属药物 ≠ first-line"** — 同类陷阱：
> - PD ψ → quetiapine 是 first-line atypical（不是 pimavanserin — pimavanserin 是 FDA 专批但 cost 限制）
> - Atypical MDD → SSRI 是 first-line（不是 phenelzine — phenelzine 是经典专属但 modern 留多步失败后）
> - Tourette α2 失败 → risperidone / aripiprazole（不是 haloperidol — haloperidol 强 D2 副作用大）

### 3.5 心理治疗对号入座

| 心理治疗 | 主治 | 关键词 | 不治什么 |
|---|---|---|---|
| **CBT-ERP** | **OCD** | Exposure + Response Prevention | 不要选 Psychodynamic / DBT |
| **DBT** | **BPD（边缘型）** | 情绪调节、自伤、人际不稳 | 不治 OCD |
| **Psychodynamic** | 部分抑郁 / 人格 | 潜意识、童年、领悟 | **OCD 无效** |
| **标准 CBT** | MDD / GAD / PTSD / Panic | 认知重构 + 行为激活 | — |
| **Exposure therapy** | 恐怖症 / PTSD | 暴露于恐惧刺激 | OCD（需 ERP，不只 exposure）|
| **PMT** | 儿童行为问题（仅家中）| Antecedents-Behaviors-Consequences | 跨场所行为问题（→ ADHD 评估）|

### 3.6 BPD 治疗层级（adjunct vs 一线，**不可跨诊断套用**）

> [!danger] BPD 一线 = DBT；所有药物只是 adjunct
> 反复在 BPD 题里"情绪不稳 → 给 mood stabilizer"是 **Bipolar 反射**误用 — BPD 的治疗范式是**心理治疗**，药物不能取代。

| 层级 | 治疗 | 适应症 | 不能干嘛 |
|---|---|---|---|
| **一线 ⭐** | **DBT**（Dialectical Behavior Therapy）| BPD 核心病理（人际不稳 / 自我形象碎 / 情绪反应性 / 自伤）| — |
| 一线备选 | MBT / TFP / Schema-focused CBT | DBT 不可及时的替代 psychotherapy | — |
| **二线 adjunct** | **Atypical AP**（quetiapine / olanzapine）| Mood instability + transient psychotic symptoms | **不能单独用**，必须配 DBT |
| **二线 adjunct** | **Mood stabilizer**（lithium / valproate / topiramate）| Impulsivity / 情绪不稳 | **不能单独用**；不是 BPD 一线（是 Bipolar 一线）|
| 仅共病 | **Antidepressant**（SSRI / SNRI）| 共病 **MDD** 或 **焦虑障碍** | 单独 BPD 慢性抑郁感/空虚感**无效**（本题 SSRI 失败的根）|

**BPD vs Bipolar 情绪不稳鉴别（防"mood stabilizer 反射"）：**

| 维度 | BPD | Bipolar |
|---|---|---|
| 漂移时长 | **minutes → hours** | **days → weeks**（episode）|
| 触发 | **situational stressor** | 自发，无需触发 |
| 治疗范式 | **DBT 一线** | Mood stabilizer / Atypical AP 一线 |

**反射钩**：「**B**P**D** 漂 **几分钟**；**B**ipola**r** 漂 **几周**。」

错题：[[mistakes/uworld-mistakes_2026-05#^Q15071|Q15071 BPD → DBT，选了 valproate 错]]（mood stabilizer 反射误用 + 漏选 DBT 一线）

### 3.7 急性激越（Acute Agitation）即刻处理 SOP ⭐⭐⭐（v1.2 新增 2026-05-30）

> [!danger] 反复错维度："急性即刻 (acute/immediate) ≠ 长期维持 (maintenance)"
> 这是 [[mistakes/uworld-mistakes_2026-05#^Q2355|Q2355]] / [[mistakes/uworld-mistakes_2026-05#^Q15856|Q15856]] 🔴 反复错的核心。stem 给 **acute psychotic agitation**（ED / immediate control），干扰项全是**维持药**（lithium / valproate / lamotrigine / SSRI）。维持药起效要**天到周**，急性行为急症**用不上** — 只能选**分钟级起效**的 IM 药。
>
> §3.4 讲的是反向（精神病性抑郁的"急" ≠ IM）；本节是**正向**（真·行为激越 = IM 是答案）。两节互为镜像。

#### 一线 = IM Antipsychotic ± IM Benzodiazepine ⭐

| 方案 | 组成 | 适用 |
|---|---|---|
| **Olanzapine IM ± lorazepam** ⭐ | 非典型 AP + BZD | 一般急性精神病性激越（首选之一）|
| **Haloperidol + lorazepam (± diphenhydramine)** | 典型 AP + BZD（± 抗胆碱防 EPS）| 经典三件套（"B52" 变体）|
| **Ziprasidone / Aripiprazole IM** | 非典型 AP | 替代选择 |
| **Lorazepam IM 单药** | BZD | 物质 / 戒断 / 病因不明时更安全（避免 AP）|

> [!warning] Olanzapine IM + 肠外 BZD 的雷点
> Olanzapine IM 与肠外 benzodiazepine **同时给**有 **呼吸抑制 + 低血压**报道 → 教科书建议**错开给 / 监测**。但 CK 层面"olanzapine + lorazepam"作为急性激越方案仍是正解。

#### 反 Pattern：维持药在 "acute/immediate" 题里全是干扰 ⭐⭐⭐

| 干扰药 | 真实适应症 | 起效 | 为什么不能急用 |
|---|---|---|---|
| **Lithium** | 双相维持 / 抗躁狂 | **5-7 天** | 急性激越来不及；需监测血药浓度 |
| **Valproate** | 双相维持 / 抗躁狂 | **数天** | 同上 |
| **Lamotrigine** | 双相抑郁维持 | **数周**（需滴定防 SJS）| 起效最慢 |
| **SSRI / Fluoxetine** | 抑郁 / 焦虑维持 | **4-6 周** | 与激越无关 |
| **口服 antipsychotic** | 维持 / 配合 | 较慢 + 需配合度 | 激越拒口服时无法给 |

#### 反射钩 ⭐

```
Stem 出现 → 0.3 秒锁 IM 路径：
─────────────────────────────────────────
"ED / acute / immediate control"
"agitation / aggression / 砸门 / 扔东西 / 攻击性"
"refusing oral medication / 拒绝口服"
      ↓
一线 = IM antipsychotic (olanzapine / haloperidol) ± lorazepam
      ↓
所有"心境稳定剂 / 抗抑郁"选项 = 维持药 = 干扰，闭眼划掉
```

#### 三个相邻"急"陷阱一次分清（交叉校验 §3.4）⭐

| "急"的类型 | stem 画面 | 一线 | IM antipsychotic 角色 |
|---|---|---|---|
| **急性激越 / 攻击性**（本节）| 砸门 / 攻击 / 拒口服 | **IM AP ± lorazepam** | ✅ 真适应症 |
| **精神病性抑郁的"急"**（§3.4）| 拒食 / 迟滞 / 退缩 / 急自杀 | **ECT** | ❌ 治不了抑郁本体 |
| **Catatonia**（§3.4）| 蜡样屈曲 / 缄默 / 拒动 | **Lorazepam IV → ECT** | ❌❌ 禁用（加重 + 诱发 NMS）|

> [!success] Memory Hook
> **"Acute 激越 = 分钟级 IM（olanzapine/haloperidol + lorazepam）；所有'稳定剂/抗抑郁'都是慢药 = 干扰。"** ⭐
> **"急性激越打人 → IM 压；精神病性抑郁烂在床上 → ECT 救；Catatonia → 先 lorazepam 别碰 AP。"** ⭐⭐⭐

## 五、反向钩（看到这些线索 → 立刻警铃）

| 线索 | 触发的快速判断 |
|---|---|
| 「成人 ADHD + 物质滥用/不想成瘾」 | → Atomoxetine；划掉 stimulant / Modafinil / Clonidine |
| 「孩子在家闹、学校乖」 | → PMT 而非 ADHD 药物 |
| 「想减肥的暴食病人」 | → 治暴食（SSRI），不是治体重；划掉 Orlistat / Phentermine |
| 「OCD + Psychodynamic 选项」 | → 立刻划掉；OCD 心理治疗 = CBT-ERP |
| 「Body decaying / infected / rotting / I deserve to die」 | → psychotic features 标签；必须有 AP 或 ECT |
| 「refusing to eat/drink + 抑郁 + 妄想」 | → ECT |
| 「Bupropion + 任何进食障碍」 | → 禁忌（癫痫阈值↓）|
| 「焦虑/强迫 → fidgety + 注意力差」 | → 不要诊断 ADHD / 不要上 stimulant（加重焦虑）|
| 「正在吃 ADHD 药 + 多系统症状」 | → 排除药物归因（毒检 + 病程）；不是 amphetamine psychosis（[[mistakes/uworld-mistakes_2026-05#^Q11967]]）|
| 「砸门 / 扔水杯 / 攻击性 / 拒绝口服」 | → IM 抗精神病药 ± BZD（行为急症路径，**不是** MDD-psychotic 路径）|
| 「Catatonia / 蜡样屈曲 / 缄默 / 紫绀但拒动」 | → **Lorazepam IV challenge** → 无效 → ECT；**禁用 antipsychotic**（可加重 + 诱发 NMS）|
| 「青少年起 + 自伤 + 关系崩 + 情绪 minutes 漂 + SSRI 无效」 | → **BPD → DBT 一线**；划掉 mood stabilizer（valproate/lithium = adjunct，不是 next step）；划掉 antidepressant（仅共病 MDD）（[[mistakes/uworld-mistakes_2026-05#^Q15071]]）|
| 「焦虑/GAD 题里出现 Tetrabenazine / Valbenazine / Deutetrabenazine」 | → **直接划掉**（VMAT2 抑制剂 = Tourette / Huntington's chorea / Tardive dyskinesia，**与焦虑零关系**）；儿童 GAD 一线 = **SSRI（sertraline）+ CBT**（[[mistakes/uworld-mistakes_2026-05#^Q20930]]）|
| 「儿童 fidget + 注意力差 + catastrophic 想法 + since little + 多领域 worry」 | → **儿童 GAD**（不是 ADHD — ADHD 要 < 12 岁前 + 多场所；不是 Tourette — 没 vocal tic）；一线 SSRI + CBT（[[mistakes/uworld-mistakes_2026-05#^Q20930]]）|

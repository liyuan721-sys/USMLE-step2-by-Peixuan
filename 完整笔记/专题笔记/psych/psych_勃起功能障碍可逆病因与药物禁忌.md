# 勃起功能障碍 (ED) 可逆病因与药物禁忌

> [!info] 一句话定位
> 新发 **勃起功能障碍 (ED, erectile dysfunction, 勃起功能障碍)** 题：**先除可逆病因（戒酒 / 停药）→ 再查用药禁忌（硝酸酯 vs PDE-5）→ 仍持续才评器质 / 血管性**。两个最爱考的"必死陷阱"：① 把所有 ED 机械化为"开西地那非 (sildenafil)"；② 在用 **硝酸酯 (nitrate, 硝酸盐类)** 的人身上加 **PDE-5 (phosphodiesterase-5, 磷酸二酯酶 5 型) 抑制剂** → 致命性低血压。

---

## 一、ED 处理三步法（核心决策序）

> [!tip] 出题反射
> NBME / UWorld 的 ED 题问"最合适的**建议 / 下一步**"，几乎永远不是直接开药，而是问你**有没有先做最安全、最基础的那步** = 找可逆病因。

```
新发 ED
  │
  ├─ ① 除可逆病因 ← 永远先做这一步
  │     酒精 / 吸烟 / 药物 / 睾酮低下 / 心理性 / 控制不佳的慢病
  │     → 戒酒、停相关药、改善血糖血压
  │
  ├─ ② 查用药禁忌（决定能否用 PDE-5）
  │     正用硝酸酯？ → 绝对禁用 PDE-5（致命低血压）
  │     用 α-阻滞剂？ → 相对禁忌（叠加低血压，需错峰）
  │
  └─ ③ 仍持续 → 评器质 / 血管性
        夜间勃起 (NPT) 消失？ 血管病 / 糖尿病 / 神经源性
        → PDE-5（无禁忌时一线）、查睾酮、血管评估
```

| 步骤 | 该考虑什么 | NBME11 Q100 对应 |
|---|---|---|
| ① 可逆病因 | reversible causes | **每日 2 杯酒 → 戒酒**（正确答案）|
| ② 用药禁忌 | medication contraindication | 正用硝酸甘油 → **不能上 sildenafil** |
| ③ 器质评估 | persistent organic eval | 戒酒后仍持续才进一步评血管 / 睾酮 |

## 二、可逆病因清单（步骤①展开）

> [!warning] 高频可逆病因
> 这些病因**去除后 ED 可能逆转**，所以题目里出现就是在提示你"先处理它，别急着开药"。

| 可逆病因 | 机制 | 处理 |
|---|---|---|
| **慢性酒精 (alcohol)** | ↓ 下丘脑-垂体-性腺轴 (HPG axis, hypothalamic-pituitary-gonadal axis) + ↓ 一氧化氮 (NO, nitric oxide) 介导海绵体舒张 + ↓ 性欲 (libido) | **戒酒**观察（NBME11 Q100 答案）|
| **吸烟 (smoking)** | 内皮功能障碍 + 血管收缩 | 戒烟 |
| **致 ED 药物** | 见 §四 | 停药 / 换药 |
| **睾酮低下 (hypogonadism)** | ↓ libido + ↓ 勃起 | 查晨间睾酮，确诊后替代 |
| **心理性 (psychogenic)** | 焦虑 / 抑郁 / 关系问题 | 夜间勃起保留 → 心理干预 |
| **控制不佳的慢病** | 糖尿病 / 高血压 / 甲状腺病 | 优化原发病管理 |

> [!info] 慢性酒精如何致 ED（NBME11 Q100 机制）
> ```
> chronic alcohol use
>   ↓
> ↓ HPG 轴信号 + ↓ NO 效应
>   ↓
> ↓ libido + 海绵体舒张受损
>   ↓
> 难以维持勃起（可逆）
> ```

## 三、心理源性 vs 器质源性鉴别

> [!tip] 一个问题分两类
> **夜间 / 晨间勃起 (nocturnal penile tumescence, NPT) 还在吗？**

| 特征 | 心理源性 (psychogenic) | 器质源性 (organic) |
|---|---|---|
| **夜间 / 晨勃** | **保留** | **消失** |
| 起病 | **突然、情境性** | 渐进、持续 |
| 诱因 | 焦虑 / 抑郁 / 关系 | 血管 / 神经 / 内分泌 |
| 伴随 | 无血管病线索 | 糖尿病 / 外周动脉病 / 高血压 |

> [!info] 血管源性 ED 的经典考法 — Leriche 综合征
> **Leriche 综合征 (aortoiliac occlusion, 主髂动脉闭塞) 三联征** = 臀部 / 大腿跛行 + **血管源性 ED**（夜间勃起消失）+ 股动脉及远端搏动减弱。中年男性吸烟者出现此三联即首先考虑。详见 [[NBME12A_breakdown_S1_Q001-Q050]]（Leriche 题）。

> [!danger] 神经源性 ED 的经典考法 — 糖尿病自主神经病变
> **糖尿病自主神经病变 (diabetic autonomic neuropathy)** 是器质性 ED 的高频病因，常与**同源**的泌尿/射精症状一起出现（同为自主神经损伤）：
> - **三联 pattern**：勃起功能障碍 + **神经源性膀胱 (neurogenic bladder)**（大残余尿 PVR, post-void residual, 排尿后残余尿、尿潴留）+ **逆行射精 (retrograde ejaculation)**（膀胱颈关闭不全 → 射精入膀胱、射精量少 / 尿液浑浊）。
> - 常伴其他自主神经表现：**胃轻瘫 (gastroparesis)**、体位性低血压、静息性心动过速。
> - Stem 钩子：**"糖尿病多年 + ED + 排尿异常 / 射精量少"** → 直指自主神经病变，而非心理性。
> - 详见 [[完整笔记/Peixuan分科笔记/endocrine]]（糖尿病自主神经病变）。

> [!info] 内分泌病因 workup — 睾酮 + 泌乳素（步骤③器质评估的核心两项）
> 器质性 ED 排查内分泌时查 **晨间睾酮 (morning testosterone)** + **泌乳素 (prolactin)**。
>
> **① 性腺功能减退 (hypogonadism) 用 LH (luteinizing hormone, 黄体生成素) / FSH (follicle-stimulating hormone, 卵泡刺激素) 定位原发 vs 继发**：
>
> | 类型 | LH / FSH | 常见病因 |
> |---|---|---|
> | **原发性（睾丸）** | **↑ 高** | 克兰费尔特综合征 (Klinefelter syndrome)、化疗（烷化剂 alkylating agents）、酮康唑 (ketoconazole)、腮腺炎性睾丸炎 (mumps orchitis)、慢性肾病 (CKD, chronic kidney disease) |
> | **继发性（垂体 / 下丘脑）** | **↓ 低或不适当正常** | **高泌乳素血症**、糖尿病、外源性雄激素、病态肥胖、垂体瘤 |
>
> **② 高泌乳素血症 (hyperprolactinemia) 致 ED**（继发性性腺减退的高频可逆 / 器质病因）：
> - 男性泌乳素瘤 (prolactinoma) 三联：**ED + ↓ 性欲 (libido) + 头痛 / 视野缺损 (双颞侧偏盲)**；机制 = 泌乳素抑制 GnRH 脉冲 → ↓ LH/FSH → ↓ 睾酮。
> - **数值分界定机制**：泌乳素 **> 200 ng/mL** = 泌乳素瘤直接分泌；**< 200 ng/mL** 多为**垂体柄效应 (stalk effect)**（无功能大腺瘤压迫垂体柄 → 切断多巴胺"刹车"→ 泌乳素轻升，且同时压迫致 LH/FSH/TSH (thyroid-stimulating hormone, 促甲状腺激素) ↓）。
> - 治疗：泌乳素瘤首选**多巴胺激动剂 (dopamine agonist，如卡麦角林 cabergoline / 溴隐亭 bromocriptine)**。
> - **可逆陷阱**：原发性甲减 (primary hypothyroidism) → **TRH (thyrotropin-releasing hormone, 促甲状腺激素释放激素) ↑ → 泌乳素 ↑**（继发性高泌乳素）→ 补甲状腺素即可逆，别误当泌乳素瘤。

## 四、致 ED / 性功能障碍的常见药物（步骤②筛查）

> [!warning] "药物是可逆病因" — 停 / 换即可能恢复
> Stem 里给了一长串用药史时，先扫有没有这些"老牌罪犯"。

| 药物类 | 代表 | 性功能影响 |
|---|---|---|
| **抗高血压** | **噻嗪类 (thiazide)**、**β-阻滞剂 (β-blocker)**、螺内酯 (spironolactone) | ED；螺内酯还致男性乳腺发育 |
| **抗抑郁 SSRI (selective serotonin reuptake inhibitor, 选择性 5-羟色胺再摄取抑制剂)** | 氟西汀 (fluoxetine) / 舍曲林 (sertraline) 等 | 性欲 / 射精 / 勃起障碍 |
| **抗精神病** | 经典抗精神病药（↑ 泌乳素 prolactin）| ↓ 性欲 (libido) + ED |
| **5α-还原酶抑制剂 (5α-reductase inhibitor)** | 非那雄胺 (finasteride) | ED + ↓ 性欲 |
| **抗雄 / 抗雌** | 螺内酯 (spironolactone)、酮康唑 (ketoconazole)、GnRH (gonadotropin-releasing hormone, 促性腺激素释放激素) 类似物 | ED |

> [!tip] 高频考点 — 高血压伴 ED 该选 / 换哪个降压药
> "患者抱怨 ED + 查用药史" 时，降压药对性功能影响分三档：
>
> | 降压药 | 对勃起功能 |
> |---|---|
> | **噻嗪类 (thiazide)**、**β-阻滞剂 (β-blocker)** | **恶化 ED**（最常见罪犯）|
> | **α-阻滞剂 (α-blocker)** | **中性 / 可改善**（还兼治 BPH 良性前列腺增生 benign prostatic hyperplasia）|
> | **血管紧张素转换酶抑制剂 (ACEi, angiotensin-converting enzyme inhibitor)** / **血管紧张素受体阻滞剂 (ARB, angiotensin receptor blocker)** / **钙通道阻滞剂 (CCB, calcium channel blocker)** | **中性** |
>
> → 高血压患者新发 ED 且在用 β-阻滞剂 / 噻嗪 → 考虑**换成 ACEi/ARB/CCB（或 α-阻滞剂）**。

> [!tip] 反向高频小考点 — 不致 ED 反"过度勃起"
> **曲唑酮 (trazodone) → 阴茎异常勃起 (priapism, 病理性持续勃起)**，与 ED 方向相反，是另一个独立考点。详见 [[完整笔记/专题笔记/psych/psych_精神科药物副作用速查]]。

> [!success] 想兼顾"不影响性功能"的选择
> 需抗抑郁 / 戒烟又怕性功能障碍 → **安非他酮 (bupropion)**（性功能副作用低）；禁癫痫 + 进食障碍。

## 五、药物禁忌核心 ⭐（最爱考的"必死陷阱"）

> [!danger] 硝酸酯 + PDE-5 抑制剂 = 绝对禁忌
> **正在用硝酸酯 (nitrate，如硝酸甘油 nitroglycerin) 的人，绝对不能加 PDE-5 抑制剂（西地那非 sildenafil / 他达拉非 tadalafil / 伐地那非 vardenafil）。**
> 机制：两者都升高 **NO (一氧化氮) / cGMP (cyclic GMP, 环鸟苷酸)** → 叠加 → 平滑肌过度舒张 → **顽固性、致命性低血压**。
> - 即使日后想用 PDE-5，也**不能在硝酸酯使用者身上直接上**。
> - 时间窗：用过 sildenafil/vardenafil 后 **24h 内**、tadalafil 后 **48h 内** 禁用硝酸酯（反之亦然）。

| 联用 | 后果 | 处理 |
|---|---|---|
| **硝酸酯 + PDE-5** | **致命低血压**（绝对禁忌）| 永不联用；ED 改先处理可逆病因 |
| **α-阻滞剂 (α-blocker, 如多沙唑嗪 doxazosin) + PDE-5** | 叠加低血压（相对禁忌）| 错峰、低剂量起始 |
| 近期 MI (myocardial infarction, 心肌梗死) / 不稳定心绞痛 / 严重心衰 | 性活动 / PDE-5 心血管风险 | 先评估心脏储备 |

> [!info] 为什么 NBME11 Q100 不能选"开 sildenafil"
> 患者按需含服硝酸甘油（心绞痛）→ 与 PDE-5 抑制剂绝对禁配。这就是为什么即便他确实有 ED，正确答案也是**戒酒**（处理可逆病因），而不是开药。

> [!tip] 防混淆 — PDE 抑制剂家族对比（同族不同用途 / 不同禁忌）
> 磷酸二酯酶 (PDE, phosphodiesterase) 抑制剂按亚型分，用途和禁忌完全不同，别混：
>
> | 亚型 | 代表药 | 主要用途 | 关键禁忌 |
> |---|---|---|---|
> | **PDE-3** | 西洛他唑 (cilostazol) | **间歇性跛行 (intermittent claudication)** 改善症状 | **心衰 (heart failure)**（PDE-3 抑制剂↑死亡率）|
> | **PDE-5** | 西地那非 (sildenafil) / 他达拉非 (tadalafil) | **ED** + **肺动脉高压 (PAH, pulmonary arterial hypertension)** | **硝酸酯 (nitrate)**（致命低血压）|
>
> 记忆钩：PDE-**3** 治腿（跛行）怕心衰；PDE-**5** 治"性"与肺（PAH）怕硝酸酯。

> [!info] 性活动的心脏风险 — METs 阈值（心脏病人能否恢复 / 用 PDE-5）
> 性活动能量消耗 ≈ **3–5 METs (metabolic equivalents, 代谢当量)**（约等于快走 / 爬两层楼）。
> - **能无症状爬两层楼 / 快走（≥4–5 METs）** = 功能储备足够 → 可安全恢复性活动、可考虑 PDE-5（无硝酸酯等禁忌时）。
> - **近期 MI (myocardial infarction, 心肌梗死) 无并发症** → 一般 **~1 周后**评估、稳定即可逐步恢复。
> - 不稳定心绞痛 / 失代偿心衰 / 严重瓣膜病（如重度主动脉瓣狭窄 severe aortic stenosis）→ 先稳定再议。

## 六、易混陷阱（普适）

> [!warning] 三大反 pattern
> 1. **"ED 就开蓝药丸"** — 机械反射；先除可逆病因（酒精 / 吸烟 / 药物 / 睾酮）。
> 2. **硝酸酯 + PDE-5 联用** — 致命低血压绝对禁忌；stem 给了硝酸甘油 = 在明示禁忌。
> 3. **新发器质症状甩锅"正常老化"** — 新发"不能维持勃起" ≠ 单纯老化，需处理。

## 七、复习自测

> [!question]- Q1. 72 岁男性，新发 1 个月难以维持勃起，既往性生活无碍；心绞痛、按需含服硝酸甘油；每日 2 杯酒；查体正常。最合适的建议是？
> **戒酒 (abstain from alcohol)**。新发 ED 先除可逆病因，慢性酒精是高频可逆病因；且他在用硝酸酯，**绝不能直接上 sildenafil**（致命低血压）。

> [!question]- Q2. ED 处理的三步顺序是什么？
> ① 除可逆病因（酒精 / 吸烟 / 药物 / 睾酮低下 / 心理）→ ② 查用药禁忌（硝酸酯禁 PDE-5）→ ③ 仍持续才评器质 / 血管性。

> [!question]- Q3. 如何快速区分心理源性与器质源性 ED？
> 问**夜间 / 晨间勃起 (NPT)**：**保留** → 心理源性（突然、情境性）；**消失** → 器质源性（渐进、伴血管 / 糖尿病线索）。

> [!question]- Q4. 为什么硝酸酯与 PDE-5 抑制剂绝对禁配？
> 两者都升高 **NO/cGMP** → 平滑肌过度舒张叠加 → **顽固性致命低血压**。sildenafil/vardenafil 后 24h、tadalafil 后 48h 内禁用硝酸酯。

> [!question]- Q5. 需抗抑郁但担心性功能障碍，选哪个药？禁忌是什么？
> **安非他酮 (bupropion)**（性功能副作用低）；禁**癫痫**和**进食障碍**（降癫痫阈值）。

> [!question]- Q6. 糖尿病多年男性，ED + 排尿费力（大残余尿）+ 射精量明显减少 / 尿液浑浊。最可能机制？
> **糖尿病自主神经病变 (diabetic autonomic neuropathy)**。同一自主神经损伤同时致：器质性 ED + 神经源性膀胱 (neurogenic bladder) + 逆行射精 (retrograde ejaculation)；常伴胃轻瘫 / 体位性低血压。这是器质性而非心理性。

> [!question]- Q7. 男性 ED + ↓ 性欲 + 头痛 + 双颞侧偏盲，泌乳素 350 ng/mL。诊断 + 一线治疗？高血压患者若同时新发 ED 且在用 β-阻滞剂该怎么办？
> 泌乳素 >200 → **泌乳素瘤 (prolactinoma)**（泌乳素抑制 GnRH→继发性性腺减退）；一线 **多巴胺激动剂 (dopamine agonist，如 cabergoline)**。降压方面：β-阻滞剂 / 噻嗪恶化 ED → 换 **ACEi/ARB/CCB 或 α-阻滞剂**（中性 / 可改善）。

> [!question]- Q8. 西洛他唑 (cilostazol) 与西地那非 (sildenafil) 同属 PDE 抑制剂，用途和禁忌各是什么？心脏病人恢复性活动 / 用 PDE-5 的功能储备门槛？
> **PDE-3 (cilostazol)** = 治间歇性跛行，**禁心衰**；**PDE-5 (sildenafil)** = 治 ED / 肺动脉高压，**禁硝酸酯**。性活动 ≈ **3–5 METs**；能无症状**爬两层楼 / 快走（≥4–5 METs）**即功能储备够，可恢复 / 考虑 PDE-5（无禁忌时）。

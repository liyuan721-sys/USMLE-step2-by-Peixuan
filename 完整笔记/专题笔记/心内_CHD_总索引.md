# _衍生_先心病总索引（心内主笔记 × 4 衍生 × 综合征 三轴导航）

> [!info] 用法
> 心内.md 里有 ~1200 行先心病手写笔记（口诀 + 中文注解），4 个衍生提供系统鉴别 / 病理生理 / 综合征联动。
> **本文件不重复内容，只指路** — 用哪个看哪个：
> - **查临床/病理**：跳心内.md
> - **做鉴别题**：跳 `_衍生_发绀型先心病鉴别`
> - **做新生儿急救**：跳 `_衍生_PDA依赖型先心病`
> - **查综合征**：跳 `_衍生_DiGeorge综合征` 或心内.md 综合征关联表

## 7. 双轴鉴别 — 拿到 stem 30 秒分诊 ⭐

```
新生儿发绀 / 休克
        ↓
轴 1：紫绀类型？
   ├─ 全身灰白 → HLHS / Critical AS（→ §3 跳第一行）
   ├─ 差异性紫绀（上红下紫）→ IAA-B（→ IAA 专题）
   ├─ 上下肢 BP 悬殊（无紫绀）→ Coarctation
   └─ 全身紫绀均匀 → 5T 系列
        ↓
轴 2：PDA 给 PGE₁ 反应？
   ├─ 救命 → TGA / Tricuspid / HLHS / IAA / Critical Coarc / Pulm atresia
   ├─ 无效 → TAPVR（依赖 ASD）⭐ 经典陷阱
   └─ 不需 → Truncus（已混合）
        ↓
轴 3：综合征线索？
   ├─ 腭裂 + 低钙 + 反复感染 → DiGeorge → conotruncal 三巨头
   ├─ 矮小 + 蹼颈 → Turner → Coarc
   ├─ 巨大儿 + 糖妈 → TGA
   ├─ 拇指畸形 → Holt-Oram → ASD
   └─ 先天白内障 + 耳聋 → Rubella → PDA
```

详解：[[完整笔记/专题笔记/心内_CHD_发绀型鉴别]] §10（终极判断流）

# 营养_锌（Zinc, 锌）

> [!tip] 一句话锁定（Step 2）
> **伤口不愈 + 味觉/嗅觉减退 + 口周与肢端皮炎 + 脱发 + 腹泻** → 想锌 Zinc 缺乏。婴儿断母乳后出现"口/眼/肛周 + 手足端红斑 + 腹泻 + 脱发"三联 → 肠病性肢端皮炎 acrodermatitis enteropathica。长期全肠外营养 TPN 未加锌是经典获得性病因。

## 一、生理作用（Step 2 够用即可）

- 锌是 **100+ 种酶**的必需辅因子（含碱性磷酸酶、超氧化物歧化酶、碳酸酐酶等）。
- 构成 **锌指 zinc finger**（转录因子的结构基序）→ 参与基因表达调控。
- 关键功能：伤口愈合、免疫、味觉/嗅觉、性发育、上皮完整性。

## 二、缺乏 Deficiency

### 临床表现（vignette buzzwords）

- **伤口愈合延迟 delayed wound healing**（术后/慢性溃疡不愈的经典 vignette）。
- **味觉减退 dysgeusia + 嗅觉减退/丧失 anosmia**（"吃东西没味道"）。
- **免疫抑制**：反复感染。
- **性腺功能减退 male hypogonadism**、青春期性发育延迟；成人腋毛/面毛/阴毛稀疏（脱发）。
- **皮炎**：口周、眼周、肛周（perioral/periorificial）+ 手指/脚趾端（acral）的红斑/糜烂/脱屑。
- **腹泻 diarrhea**、易激惹、生长迟缓（儿童）。

> [!danger] 肠病性肢端皮炎 Acrodermatitis Enteropathica
> **先天性肠道锌吸收缺陷**（常染色体隐性，SLC39A4 突变）。经典**三联征**：
> - **脱发 alopecia（hair loss）**
> - **腹泻 diarrhea**
> - **身体开口周围（perioral/periorificial）+ 肢端（acral，指趾尖）的皮疹**
>
> 典型 vignette：婴儿**断母乳后**出现口周/肛周红斑糜烂 + 腹泻 + 脱发（母乳锌生物利用度高，断奶后暴露缺陷）。补锌迅速好转。

### 高危人群 / 病因

> [!warning] 获得性锌缺乏高频病因
> - **全肠外营养 TPN (total parenteral nutrition, 全胃肠外营养) 未添加锌**（Step 2 经典：住院长期 TPN 后出现皮炎 + 伤口不愈 + 味觉改变）。
> - **吸收不良**：克罗恩病 Crohn、乳糜泻、慢性腹泻、短肠综合征。
> - **减重手术 bariatric surgery** 后、酗酒、慢性肝病、严重烧伤（丢失↑）。
> - 素食/膳食植酸盐 phytate 高（抑制吸收）。

### 诊断 / 下一步

- 多为临床诊断（典型三联 + 高危背景）；可测**血清锌↓、血浆碱性磷酸酶 ALP (alkaline phosphatase, 碱性磷酸酶)↓**（锌依赖酶）。
- 诊断性治疗：**补锌后皮疹/腹泻快速改善**支持诊断。

### 治疗

- **口服/静脉补锌 zinc supplementation**；肠病性肢端皮炎需终身补充。

## 三、过量 / 毒性 Excess

> [!danger] 锌过量 → 铜缺乏（考点）
> 长期大剂量补锌（或用锌治疗时）→ 诱导肠上皮金属硫蛋白 → **抑制铜 copper 吸收 → 继发铜缺乏**：
> - **贫血**（似缺铁/巨幼贫，补铁无效）；
> - **脊髓病 myelopathy / 周围神经病**——表现**极似维生素 B12 缺乏的亚急性联合变性 SCD**（后索/侧索），但 B12 正常。
> - HDL (high-density lipoprotein, 高密度脂蛋白)↓。
> 见于过量锌补剂、含锌假牙粘合剂滥用。→ 不明原因脊髓病 + B12 正常 → 查铜/锌。

## 四、Step 2 高频关联 & 陷阱

- **TPN 微量元素**：长期 TPN 患者出现皮炎 + 脱发 + 伤口不愈 + 味觉障碍 → 锌缺乏（记得 TPN 要补微量元素）。
- **味觉/嗅觉减退**鉴别里，锌缺乏是可逆病因之一。
- **锌的治疗用途**：Wilson 病 Wilson disease（肝豆状核变性）维持治疗——锌阻断铜吸收；普通感冒锌含片缩短病程（证据有限）。
- **锌↔铜跷跷板**：补锌过量 → 铜缺乏 → 贫血 + 脊髓病（似 B12）。这是"B12 正常却像 SCD"的重要鉴别。

## 五、Memory Hook

> [!success] 记忆钩子
> - **锌 = 长、闻、尝、愈、御**：生长、嗅觉、味觉、伤口愈合、免疫——缺了全塌。
> - **肠病性肢端皮炎三联**："掉毛 + 拉肚 + 洞口周围起疹"（脱发 + 腹泻 + periorificial/acral 皮炎）。
> - **TPN 忘加锌 → 皮炎 + 不愈合**。
> - **锌太多挤走铜** → 铜缺乏贫血 + 似 B12 的脊髓病。

## 六、复习自测

> [!question]- Q1. 8 月龄婴儿断母乳后出现口周、肛周红斑糜烂，指趾端脱屑，伴慢性腹泻和头发稀疏脱落。最可能诊断及处理？
> **肠病性肢端皮炎 acrodermatitis enteropathica**（先天性肠锌吸收缺陷）。三联：脱发 + 腹泻 + periorificial/acral 皮炎，断母乳后显现。处理：**补锌**（需长期），皮疹与腹泻迅速改善。

> [!question]- Q2. 一名短肠综合征患者长期全肠外营养（TPN），近期出现伤口愈合不良、口周皮炎和味觉减退。最可能缺乏的微量元素？
> **锌 Zinc**。长期 TPN 未充分补锌是经典获得性锌缺乏；表现为伤口不愈、皮炎、味觉/嗅觉减退、免疫下降。补锌纠正。

> [!question]- Q3. 一名长期大量服用锌补剂的老人出现进行性步态不稳、本体觉减退和贫血，血清维生素 B12 正常。最可能机制？
> **锌过量诱发铜缺乏**。铜缺乏致**脊髓病（似 B12 的亚急性联合变性）+ 贫血**；B12 正常是关键区分点。查血清铜/锌，停锌补铜。

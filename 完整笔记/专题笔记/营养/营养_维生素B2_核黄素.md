# 营养_维生素B2（Riboflavin, 核黄素）

> [!tip] 一句话锁定（Step 2）
> **口角炎 cheilosis（口角皲裂）+ 角膜血管化 corneal vascularization + 品红舌 magenta tongue** → 核黄素缺乏（记"2 个 C"）。一般无严重系统性后果。

## 一、生理作用（Step 2 够用即可）

Riboflavin (核黄素) 是两个重要黄素辅酶的前体：

- FAD (flavin adenine dinucleotide, 黄素腺嘌呤二核苷酸)
- FMN (flavin mononucleotide, 黄素单核苷酸)

它们在**氧化还原反应 / 电子传递链**中充当电子载体（如琥珀酸脱氢酶、脂肪酸 β 氧化、谷胱甘肽还原酶）。

> [!info] Step 2 联动记忆
> 核黄素（B2）是**合成烟酸（B3）所需的辅助因子之一**（色氨酸 → 烟酸途径需 B2、B6）。故 B2 缺乏可间接加重 B3 相关表现——把 B2/B3 串起来记。

## 二、缺乏 Deficiency

### 2.1 临床表现（vignette buzzwords）

核黄素缺乏（ariboflavinosis）主要累及**黏膜与皮肤**，记忆核心"**2 个 C**"：

- **C**heilosis / angular stomatitis（口角炎 / 口角皲裂 — 嘴角干裂、皲裂、结痂）
- **C**orneal vascularization（角膜血管化 — 角膜出现新生血管、畏光）

其他表现：

- Magenta tongue / glossitis（品红色舌 / 舌炎 — 舌面呈品红/洋红色）
- Seborrheic dermatitis（脂溢性皮炎，尤其鼻唇沟）
- 咽炎、正常细胞正色素性贫血（较少见）

> [!warning] 鉴别小心
> "口角炎 angular cheilosis"不特异：也见于铁缺乏、B12/叶酸缺乏、义齿/念珠菌感染。B2 的特征组合是"口角炎 **+ 角膜血管化 + 品红舌**"。

### 2.2 高危人群 / 病因

- 整体营养不良 / 酗酒（常与其他 B 族缺乏并存）
- 摄入不足（乳制品、蛋、绿叶菜、强化谷物为主要来源）
- 吸收不良
- 长期光疗新生儿（核黄素光敏感、易被分解）

### 2.3 诊断

- 主要**临床诊断**（典型口腔/眼/舌表现）
- 可测红细胞谷胱甘肽还原酶活性系数 (erythrocyte glutathione reductase activity coefficient)，或尿核黄素排泄

### 2.4 治疗 / 下一步

- 口服核黄素补充
- 因常与其他 B 族维生素同时缺乏，通常给复合 B 族维生素纠正整体营养

## 三、过量 / 毒性 Excess

无明确毒性综合征。水溶性、过量经肾排出，高剂量摄入可致**尿液呈亮黄色**（无害，属良性现象，可用于提示服药依从性）。

## 四、Step 2 高频关联 & 陷阱

> [!warning] 高频考点与陷阱
> - 记忆锚："2 个 C" = Cheilosis + Corneal vascularization，再加 Magenta tongue（品红舌）。
> - B2 缺乏一般**不致命、无严重系统后果**——考题常作为"良性口腔黏膜表现"的对照选项。
> - 与 B3（烟酸）联记：B2 是色氨酸→烟酸通路的辅因子。
> - 尿变亮黄 = 良性，别误判为病理性血尿/胆红素尿。
> - 口角炎单独出现别急着选 B2，要看有没有角膜血管化 / 品红舌的组合。

## 五、Memory Hook

> [!success] 记忆钩子
> **"B2 = 2 个 C + 品红舌"**：**C**heilosis（口角裂）+ **C**orneal vascularization（角膜长血管）。
> 谐音记："核黄素" → **黄** → 过量时尿也变**亮黄**。
> 品红舌 magenta = 洋红色，像被核黄素"染"了色。

## 六、复习自测

> [!question]- Q1. 营养不良患者主诉嘴角反复干裂结痂，查体见舌呈洋红色、角膜边缘有新生血管。最可能缺乏哪种维生素？
> Riboflavin（核黄素 / B2）。经典组合：cheilosis（口角炎）+ magenta tongue（品红舌）+ corneal vascularization（角膜血管化）。

> [!question]- Q2. 患者开始服用某维生素补充剂后发现尿液呈鲜亮黄色，无其他不适，尿常规正常。最可能原因？是否需处理？
> 核黄素（B2）过量经肾排出使尿呈亮黄，属**良性无害现象**，无需处理；可作为服药依从性的线索。

> [!question]- Q3. 为什么核黄素（B2）缺乏可以间接影响烟酸（B3）水平？
> 色氨酸 tryptophan 合成烟酸的途径需要 B2（及 B6）作为辅因子，故 B2 缺乏可削弱内源性烟酸合成，加重 B3 相关表现。

## ⭐ B2 vs B3 缺乏分水岭（Q4302 锚定）

> [!tip] 一眼分 B2（核黄素 riboflavin）vs B3（烟酸 niacin）缺乏
> 两者都有口炎/舌炎/皮炎，靠这两点分开：
> - 有无贫血：B2 缺乏 → 正细胞正色素性贫血 (normocytic normochromic anemia)；B3 缺乏（糙皮病 pellagra）不引起贫血
> - 有无胃肠/神经症状：B3 = 3D（腹泻 diarrhea / 皮炎 dermatitis / 痴呆 dementia）+ 日晒部位对称红疹；B2 无腹泻、无神经症状
>
> 反射：口角炎 + 舌炎 + 脂溢性皮炎 + 有正细胞贫血 + 没腹泻没神经症状 → 选 B2；有 3D + 日晒疹 + 不贫血 → 选 B3。

# 营养_维生素B3（Niacin, 烟酸/尼克酸）

> [!tip] 一句话锁定（Step 2）
> **糙皮病 Pellagra "3D"：Dermatitis（光暴露区）+ Diarrhea + Dementia** → 烟酸缺乏。高频继发病因：Hartnup 病、类癌综合征 carcinoid、异烟肼 isoniazid。

## 一、生理作用（Step 2 够用即可）

Niacin (烟酸 / 尼克酸, B3) 是两个关键辅酶的前体：

- NAD⁺ (nicotinamide adenine dinucleotide, 烟酰胺腺嘌呤二核苷酸)
- NADP⁺ (nicotinamide adenine dinucleotide phosphate)

参与几乎所有能量代谢的氧化还原反应（糖酵解、TCA、氧化磷酸化、脂肪酸合成）。

> [!info] 合成来源（Step 2 必记）
> 烟酸可由氨基酸 **tryptophan（色氨酸）内源性合成**，此过程需要 **B2（核黄素）和 B6（吡哆醇）**。因此凡是"抢走色氨酸"或"耗竭 B6"的情况都会导致继发性烟酸缺乏（见下文继发病因）。

## 二、缺乏 Deficiency

### 2.1 糙皮病 Pellagra — 经典"3D"（4D）

```
烟酸缺乏 → Pellagra 糙皮病
      ├── Dermatitis 皮炎 ── 光暴露区对称性
      │        颈部 Casal necklace（Casal 项链样）
      ├── Diarrhea 腹泻 ── 黏膜受累
      ├── Dementia 痴呆 ── 神经精神（记忆↓/意识改变）
      └── Death 死亡 ── 重症未治疗（第 4 个 D）
```

vignette buzzwords：**光照后加重的对称性色素沉着/脱屑皮疹（尤其颈部 "Casal necklace"、手背、面部）+ 腹泻 + 精神状态改变/痴呆**。

### 2.2 高危人群 / 病因

原发（膳食）：

- 以玉米 maize 为主食且未经碱处理（玉米中烟酸结合态、色氨酸低）——历史经典人群
- 慢性酗酒 / 全面营养不良

⭐继发（Step 2 最爱考的"抢色氨酸 / 耗 B6"三大机制）：

| 病因 | 机制 |
|------|------|
| Hartnup 病（Hartnup disease） | 常染色体隐性；肾小管与肠道**中性氨基酸（含色氨酸）转运障碍**→色氨酸吸收↓→烟酸合成原料不足 |
| 类癌综合征 carcinoid syndrome | 肿瘤把大量**色氨酸转去合成 5-羟色胺 serotonin**→色氨酸被"截胡"→烟酸合成↓ |
| 异烟肼 isoniazid (INH) | 抗结核药**耗竭 B6（吡哆醇）**→色氨酸→烟酸通路受阻→继发糙皮病样表现 |

> [!warning] Isoniazid 双陷阱
> Isoniazid（异烟肼）耗竭 B6 → 既可引起**周围神经病**（B6 缺乏经典），也可导致**糙皮病样烟酸缺乏**。临床上抗结核治疗常规**共同补充 B6（吡哆醇）**预防。

> [!info] Hartnup 记忆
> Hartnup = "**H**"→ 想成 "**Homeless（无家可归的色氨酸转运体）**"，中性氨基酸没了"运输车"，也可有小脑共济失调、光敏皮疹（糙皮病样），治疗为高蛋白饮食 + 烟酸补充。

### 2.3 诊断

- 主要**临床诊断**（皮疹分布 + 消化 + 神经三联）
- 可测尿代谢产物 N-methylnicotinamide 降低（不常用）
- 结合病史（酗酒、玉米饮食、抗结核用药、类癌相关潮红/腹泻）

### 2.4 治疗 / 下一步

- 补充 **nicotinamide（烟酰胺）**（首选，较少潮红）或烟酸
- 纠正基础病因（治疗类癌、Hartnup 高蛋白饮食、抗结核者补 B6）

## 三、过量 / 毒性 Excess & 治疗性用途

烟酸也作为**降脂药**使用（Step 2 药理高频）：

- 作用：↓ VLDL / LDL（低密度脂蛋白），并**显著 ↑ HDL（高密度脂蛋白，升 HDL 最强的经典药）**，也降甘油三酯
- ⭐副作用：
  - **Flushing（潮红）**——前列腺素 prostaglandin 介导的皮肤潮红/瘙痒；**服药前给阿司匹林 aspirin（或 NSAID, non-steroidal anti-inflammatory drug, 非甾体抗炎药）可减轻**
  - Hyperglycemia（高血糖 — 糖尿病患者慎用）
  - Hyperuricemia（高尿酸血症 / 诱发痛风 gout）
  - 肝毒性（尤其缓释剂型）

> [!danger] 药理高频陷阱
> 服烟酸后**面部潮红 + 瘙痒** = 前列腺素介导，**不是过敏**，预服阿司匹林可缓解。别把它当过敏停药。烟酸三大代谢副作用记"糖、尿酸、肝"。

## 四、Step 2 高频关联 & 陷阱

> [!warning] 高频考点与陷阱
> - "3D"皮疹要强调**光暴露区对称性 + Casal necklace（项链样）**。
> - 类癌综合征患者（潮红 + 腹泻 + 右心瓣膜病）若再出现糙皮病样皮疹 → 色氨酸被夺去合成 serotonin。
> - 抗结核患者出现神经病 / 糙皮病样表现 → isoniazid 耗 B6，补吡哆醇。
> - Hartnup = 常染色体隐性、中性氨基酸转运障碍，糙皮病样 + 小脑共济失调。
> - 降脂用烟酸的潮红 = 前列腺素，非过敏，aspirin 预处理。
> - 烟酸升 HDL 最强，但因副作用现临床已少用一线。

## 五、Memory Hook

> [!success] 记忆钩子
> **Pellagra = "3D"**：**D**ermatitis + **D**iarrhea + **D**ementia（重症加 **D**eath）。
> 继发三凶手抢色氨酸："**Hartnup 抢吸收、Carcinoid 抢去做血清素、INH 抢走 B6**"。
> 降脂烟酸副作用 **"红糖酸肝"**：Flushing 潮**红** + 高血**糖** + 高尿**酸** + **肝**毒性；红脸先吃阿司匹林。

## 六、复习自测

> [!question]- Q1. 长期以玉米为主食的酗酒者出现颈部及手背对称性、光照后加重的鳞屑性色素沉着皮疹，伴慢性腹泻和记忆减退。诊断及缺乏的维生素？
> Pellagra 糙皮病，缺乏 **Niacin（烟酸 / B3）**。"3D" = Dermatitis（光暴露区 Casal necklace）+ Diarrhea + Dementia。治疗补烟酰胺 + 纠正病因。

> [!question]- Q2. 类癌综合征患者除潮红、腹泻外出现糙皮病样皮疹，机制是什么？
> 类癌肿瘤将大量**色氨酸 tryptophan 转去合成 5-羟色胺 serotonin**，色氨酸被截留，内源性烟酸合成原料不足 → 继发烟酸缺乏（糙皮病样表现）。

> [!question]- Q3. 患者因高胆固醇服用烟酸后出现面部潮红和皮肤瘙痒。机制是什么？如何预防？是否需停药？
> 前列腺素 prostaglandin 介导的皮肤潮红（**非过敏**），无需停药；**服药前给阿司匹林 aspirin / NSAID** 可显著减轻。此外注意烟酸的高血糖、高尿酸、肝毒性。

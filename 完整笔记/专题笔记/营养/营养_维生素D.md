# 营养_维生素D（Vitamin D / Calciferol, 钙化醇）

> [!tip] 一句话锁定（Step 2）
> 儿童 O 型腿 + 串珠 → 佝偻病 rickets；成人骨痛 + 肌无力 + ALP↑ → 骨软化 osteomalacia → 缺维生素 D（查 **25-OH-D**）。反过来 结节病 sarcoidosis / 结核 TB 出现 **高钙血症 hypercalcemia** → 想 **肉芽肿巨噬细胞异位 1α 羟化 → 维生素 D 内源过量**。

## 一、生理作用与代谢（Step 2 够用即可）

核心作用：**升高血钙、血磷** → 维持骨矿化。促进肠道钙、磷吸收；协同 PTH 动员骨钙。

代谢通路（vignette 常考"查哪一个/哪一步出问题"）：

```
来源：
  D3 胆钙化醇 cholecalciferol  ← 皮肤日晒(UVB) + 动物性食物
  D2 麦角钙化醇 ergocalciferol ← 植物 / 强化食品
        │
        ▼  肝 liver：25 位羟化
  25-OH-D（骨化二醇 calcidiol）= 储存/循环形式 ← 【查血就查这个，反映体内储量】
        │
        ▼  肾 kidney：1α 羟化酶（受 PTH↑、低磷 促进）
  1,25-(OH)2-D（骨化三醇 calcitriol）= 活性形式
```

> [!info] 缩写全称
> - PTH = parathyroid hormone，甲状旁腺激素。
> - CKD = chronic kidney disease，慢性肾病。
> - ALP = alkaline phosphatase，碱性磷酸酶。

- 【查缺乏 → 查 25-OH-D】：25-OH-D 半衰期长、量大，最能反映储备。
- 【活性 = 1,25-(OH)2-D】：由肾脏在 PTH（甲状旁腺激素 parathyroid hormone）驱动下生成；CKD（慢性肾病 chronic kidney disease）时此步受损。

## 二、缺乏 Deficiency

### 临床表现（vignette buzzwords）

- **儿童：佝偻病 rickets** —— 生长板矿化障碍：
  - **O 型腿 genu varum（膝内翻）**、下肢弯曲；
  - **肋骨串珠 rachitic rosary（佝偻病串珠）**、**手镯征 wrist widening（腕部膨大）**；
  - 前囟闭合延迟、方颅、出牙延迟。
- **成人：骨软化 osteomalacia** —— 已矿化骨的类骨质增多：
  - **弥漫性骨痛 + 近端肌无力**（上楼/起立费力，鸭步）；
  - X 线 **Looser 带（假骨折线 pseudofractures）**。
- **低钙表现**：手足搐搦 tetany、口周/肢端麻木、Chvostek 征、Trousseau 征、重则抽搐。

> [!danger] 高危人群 / 病因（Step 2 反复考）
> - **纯母乳喂养且未补维生素 D 的婴儿**（母乳维生素 D 含量低）→ ⭐ **所有纯/主母乳婴儿出生后即补维生素 D 400 IU/天**（AAP = American Academy of Pediatrics，美国儿科学会；考"下一步/预防"）。
> - **深肤色 dark skin**（黑色素挡 UVB）、**少日晒**（住院、长期室内、遮盖多、高纬度冬季、老年人）。
> - **吸收不良**（脂溶性）：乳糜泻 celiac、Crohn、囊性纤维化、胆汁淤积、**减重手术后**、脂肪泻。
> - **慢性肾病 CKD**：1α 羟化↓ → 活性 D↓（肾性骨营养不良）。
> - **慢性肝病**：25 位羟化↓。

### 诊断（关键 labs）

首选检查：**血清 25-OH-D↓**（确诊缺乏）。

营养性缺乏典型 lab 组合：

| 指标 | 变化 | 说明 |
| --- | --- | --- |
| 25-OH-D | ↓ | 首选、确诊储备不足 |
| PTH | ↑ | 继发性甲旁亢代偿 |
| 血钙 Ca | 低或正常 | 常被 PTH 代偿到正常低值 |
| 血磷 P | ↓ | PTH 促磷排 + 吸收↓ |
| 碱性磷酸酶 ALP | ↑ | 成骨活跃（rickets/osteomalacia 标志） |

> [!warning] 鉴别陷阱
> **CKD 的矿化骨病**：低钙、**高磷（磷排不出）**、PTH↑、1,25-(OH)2-D↓ —— 与营养性缺乏的"低磷"相反。看到"低钙 + 高磷 + 肾功能差"想 CKD-MBD 而非单纯营养缺乏。

### 治疗 / 下一步

- **补充维生素 D**（D2 或 D3）**± 钙剂**。
- 婴儿预防：纯母乳补 400 IU/天。
- CKD 者：因 1α 羟化受损，需给**活性维生素 D（骨化三醇 calcitriol）**，而非普通 D2/D3。

## 三、过量 / 毒性 Excess

> [!danger] ⭐ 核心毒性 = 高钙血症 Hypercalcemia
> 无论**外源过量**（过量补剂）还是**内源生成过多**，机制都是 1,25-(OH)2-D 过多 → 肠钙吸收↑ → **高钙血症、高尿钙 hypercalciuria、肾结石 nephrolithiasis / 肾钙质沉积**。
> 症状："stones, bones, groans, psychiatric overtones" —— 肾结石、骨痛、腹痛便秘/多尿多饮、乏力/意识改变。

**内源过量（Step 2 最爱考）——肉芽肿病 granulomatous disease：**

- **结节病 sarcoidosis（结节病）**、**结核 TB（tuberculosis）**、以及某些淋巴瘤。
- 机制：肉芽肿内**活化的巨噬细胞异位表达 1α 羟化酶**，不受 PTH/反馈调控地把 25-OH-D → 1,25-(OH)2-D → **高钙血症**。
- vignette：结节病患者出现高钙 + 肾结石；PTH **被抑制（↓）**、1,25-(OH)2-D↑；日晒后加重。
- 处理：**糖皮质激素 glucocorticoids**（抑制巨噬细胞 1α 羟化）、避免日晒/维生素 D/钙负荷、补液。

## 四、Step 2 高频关联 & 陷阱

- **佝偻病 vs 成人骨软化**：同一疾病、不同年龄——生长板未闭 = 佝偻病（O 型腿/串珠），生长板已闭 = 骨软化（骨痛/假骨折）。
- **高钙血症鉴别**：PTH↑ 想原发/三发甲旁亢；**PTH↓** 且 1,25-D↑ 想**维生素 D 介导（肉芽肿）**或维生素 D 中毒；PTH↓ 且 PTHrP↑（parathyroid hormone-related peptide，甲状旁腺激素相关肽）想恶性肿瘤。
- **纯母乳 + 未补 D + 深肤色婴儿冬季** = 佝偻病经典拼图 → 预防答案永远是"补维生素 D"。
- 别把"低磷佝偻病"（X 连锁低磷血症 XLH = X-linked hypophosphatemia，FGF23 相关）与营养性维生素 D 缺乏混——前者 25-OH-D 正常、补维生素 D 无效。

## 五、Memory Hook

> [!success] 记忆钩子
> - **"D for Dem bones"**：D 管骨——缺 D → 骨软（孩子 O 腿、大人骨痛）。
> - **查缺乏认准 25**：仓库（25-OH-D）先空；活性（1,25）由肾+PTH 现产，别拿它查储备。
> - **缺 D 的 lab 口诀"钙磷都低、PTH 高、ALP 高"**；CKD 反着来（磷高）。
> - **肉芽肿 = 巨噬细胞自己造活性 D**：Sarcoid/TB 高钙、PTH 低、晒太阳更糟 → 给激素。

## 六、复习自测

> [!question]- Q1. 一名 14 月龄纯母乳喂养、深肤色男婴，冬季就诊，家长诉下肢弯曲、出牙延迟。查体见双膝内翻（O 型腿）、肋软骨交界处串珠样隆起。最可能诊断、典型实验室改变及预防措施？
> **佝偻病 rickets（维生素 D 缺乏）**。典型 lab：**25-OH-D↓、PTH↑、磷↓、ALP↑、钙低或正常**。预防：所有纯/主母乳婴儿自出生起补**维生素 D 400 IU/天**（本患儿深肤色、少日晒、纯母乳三重高危）。

> [!question]- Q2. 一名 48 岁女性诊断结节病 sarcoidosis，近期因肾绞痛就诊，发现血钙升高、尿钙高、肾结石；PTH 降低，1,25-(OH)2-D 升高。高钙的机制与首选处理？
> 机制：肉芽肿内**活化巨噬细胞异位表达 1α 羟化酶**，不受调控地将 25-OH-D 转成活性 1,25-(OH)2-D → 肠钙吸收↑ → **高钙血症/高尿钙/肾结石**（故 PTH 被反馈抑制）。处理：**糖皮质激素（glucocorticoids）**，并避免日晒、限制钙/维生素 D 摄入、补液。TB 也可有同样机制。

> [!question]- Q3. 一名慢性肾病 CKD 5 期患者出现低钙、**高磷**、PTH 显著升高、骨痛。与营养性维生素 D 缺乏相比，本例活性维生素 D 通路哪一步受损？补哪种维生素 D？
> CKD 破坏**肾脏 1α 羟化酶** → 1,25-(OH)2-D（骨化三醇）生成↓，且磷排泄↓ 导致**高磷**（区别于营养性缺乏的低磷）。因近端羟化步骤缺失，应补**活性维生素 D：骨化三醇 calcitriol**，而非普通 D2/D3，并配合降磷（磷结合剂）。

# 营养_维生素A（Vitamin A / Retinol, 视黄醇）

> [!tip] 一句话锁定（Step 2）
> 夜盲 night blindness + 干眼/Bitot 斑 → 缺 Vitamin A → 补维生素 A（麻疹患儿尤其记得补）；反过来看到 慢性头痛 + 视物模糊 + 脱发/皮肤脱屑 → 想 **维生素 A 过量 → 假性脑瘤 pseudotumor cerebri**；育龄女性用 异维A酸 isotretinoin → 想 **致畸 + iPLEDGE**。

## 一、生理作用（Step 2 够用即可）

- **视觉 vision**：Retinal（视黄醛）是视杆细胞感光色素 rhodopsin（视紫红质）的组成部分 → 缺乏最早、最特异的表现就是 **夜盲**。
- **上皮分化 epithelial differentiation**：维持结膜、角膜、呼吸道、泌尿道等上皮正常分化 → 缺乏 → 上皮角化、干燥（干眼、毛囊角化）。
- **免疫 immunity**：维持黏膜屏障与免疫功能 → 缺乏易反复感染（麻疹病死率↑）。
- 类型：视黄醇 retinol（动物性，肝、蛋、奶）与 β-胡萝卜素 β-carotene（植物性前体，橙黄色蔬果）。

## 二、缺乏 Deficiency

### 临床表现（vignette buzzwords）

- **夜盲 night blindness**：最早、最典型（开车看不清、暗处看不清）。
- **干眼症 xerophthalmia**：结膜/角膜干燥。
- **Bitot 斑 Bitot spots**：结膜上泡沫状白色角化斑（buzzword，几乎= Vitamin A 缺乏）。
- **角膜软化 keratomalacia**：角膜溶解、软化 → 可致失明（发展中国家儿童可预防性失明首因）。
- **毛囊角化 follicular hyperkeratosis**：皮肤干燥、鸡皮样。
- **免疫力下降**：反复呼吸道/消化道感染，麻疹加重。

> [!danger] 高危人群 / 病因
> - **发展中国家营养不良儿童**（经典 vignette，尤其合并麻疹）。
> - **脂肪吸收不良**：Vitamin A 是脂溶性维生素 → 囊性纤维化 cystic fibrosis、慢性胰腺炎、胆汁淤积、乳糜泻 celiac、Crohn、**减重手术后（bariatric surgery）**、脂肪泻。
> - 极端限制饮食 / 长期酗酒。

### 诊断

- 临床诊断为主：典型眼部表现（夜盲 + Bitot 斑）。
- 可查 **血清 retinol（视黄醇）↓** 确认。
- vignette 常直接给"发展中国家 + 夜盲 + 结膜白斑" → 不必等 lab，直接想缺乏。

### 治疗 / 下一步

- **口服/肌注补充维生素 A**。
- ⭐ **麻疹 measles 患儿：无论是否缺乏，WHO 推荐补 Vitamin A**（降低病死率与并发症，尤其发展中国家/住院重症）→ Step 2 高频"下一步给什么"。

## 三、过量 / 毒性 Excess（维生素 A 有明确毒性综合征）

维生素 A 是脂溶性 → 会蓄积 → 有真实毒性（对比水溶性维生素多经尿排出）。

### 急性中毒（acute）

- 大剂量一次摄入：**头痛、恶心呕吐、眩晕、视物模糊**，皮肤脱屑。
- 经典冷知识 vignette：**北极探险者吃北极熊/海豹肝**（肝富含维生素 A）→ 急性中毒。

### 慢性中毒（chronic）

- **脱发 alopecia**、皮肤干燥脱屑、口唇干裂。
- **肝毒性 / 肝大 hepatomegaly**（可进展至肝硬化）。
- **关节痛、骨痛**（骨膜增生）。
- ⭐ **特发性颅内高压 IIH（idiopathic intracranial hypertension，特发性颅内高压）/ 假性脑瘤 pseudotumor cerebri（pseudotumor cerebri，假性脑瘤）**：慢性头痛 + 视乳头水肿 papilledema + 一过性视物模糊 / 复视，MRI/CT 正常、腰穿压力↑。

> [!warning] 药物相关维生素 A 类似物也会引起 IIH
> **异维A酸 isotretinoin**、**全反式维A酸 ATRA（all-trans retinoic acid, 全反式维甲酸）**、以及合用 **四环素类 tetracyclines（如 多西环素 doxycycline）** 均可引起假性脑瘤 → vignette：痤疮青少年用 isotretinoin（尤其再加 doxycycline）出现头痛 + 视物模糊 → 停药、查眼底。

## 四、Step 2 高频关联 & 陷阱

### 治疗性用途（考"给药"而非"缺乏"）

| 药物 | 适应证 | 记忆点 |
| --- | --- | --- |
| Vitamin A（视黄醇） | 麻疹患儿、缺乏症 | WHO 推荐麻疹补 A 降病死率 |
| 全反式维A酸 ATRA（all-trans retinoic acid, 全反式维甲酸） | 急性早幼粒细胞白血病 APL（acute promyelocytic leukemia, M3, t(15;17)/PML-RARA） | 诱导幼稚细胞分化；警惕分化综合征 differentiation syndrome |
| 异维A酸 isotretinoin | 重度结节囊肿性痤疮 severe nodulocystic acne | 强致畸 + iPLEDGE + 查血脂/肝功 |

> [!danger] ⭐ 致畸 Teratogenicity（Step 2 妇产/皮肤高频）
> **异维A酸 isotretinoin 是强致畸剂**（维A酸胚胎病 retinoid embryopathy）：颅面畸形、**唇腭裂 cleft lip/palate**、心脏与中枢神经缺陷、耳畸形。
> 处方前 **iPLEDGE 项目**要求：
> - 用药前 **2 次阴性妊娠试验（β-hCG）**；
> - 全程使用 **2 种有效避孕方法**；
> - 每月复查妊娠试验。
> vignette：育龄女性用 isotretinoin 治痤疮 → 问"处方前必须做什么" → **两次阴性妊娠试验 + 两种避孕**。

> [!warning] 分化综合征 Differentiation syndrome（原 ATRA 综合征）
> APL 用 ATRA（±砷剂）诱导时：发热、呼吸困难、体重增加、肺浸润、低血压 → 处理 **激素 dexamethasone（地塞米松）**。

### 其他易混点

- **夜盲**除维生素 A 缺乏外，也见于遗传性视网膜色素变性 RP（retinitis pigmentosa，视网膜色素变性） —— 但 vignette 给"营养不良/吸收不良/Bitot 斑"就锁 Vitamin A。
- 假性脑瘤的其他病因：肥胖育龄女性、四环素、维生素 A 过量 → 三者常一起考。

## 五、Memory Hook

> [!success] 记忆钩子
> - **"A for the eyes"**：A = 视觉，缺 A 先夜盲，重则角膜软化失明；**Bitot 斑 = A 缺乏的名片**。
> - **麻疹想到 A**：Meas**A**les → 补 Vitamin A。
> - **过量 = "头发掉、头很痛、肝很大"**：脱发 + 假性脑瘤（头痛/视物模糊）+ 肝大。
> - **isotretinoin = 两个 2**：**2 次阴性妊娠试验 + 2 种避孕**（iPLEDGE），因为它会造唇腭裂。

## 六、复习自测

> [!question]- Q1. 一名发展中国家 3 岁男童，反复呼吸道感染，家长诉其"傍晚后看不清路"。查体见双眼结膜干燥、颞侧可见泡沫状白色斑块。最可能的诊断及最恰当的下一步？
> **诊断：维生素 A 缺乏（Vitamin A deficiency）**。夜盲 + 干眼 + **Bitot 斑**三联即可锁定。**下一步：补充维生素 A**。若该患儿同时诊断麻疹，无论缺乏与否，WHO 均推荐补维生素 A 以降低病死率。

> [!question]- Q2. 一名 22 岁女性因重度结节囊肿性痤疮拟用异维A酸 isotretinoin。开药前医生必须确保完成的两项关键措施是什么？为什么？
> 因 isotretinoin **强致畸**（可致唇腭裂、心脏/CNS 畸形），按 **iPLEDGE**：需 **两次阴性妊娠试验（β-hCG）** 并全程使用 **两种有效避孕方法**（每月复查妊娠试验）。此外还应监测血脂与肝功能。

> [!question]- Q3. 一名 30 岁女性长期自服大剂量维生素 A 补充剂，出现数周慢性头痛、间歇性视物模糊，眼底见视乳头水肿，头颅 MRI 正常，腰穿开放压升高。诊断？机制类比？
> **假性脑瘤 / 特发性颅内高压（pseudotumor cerebri / IIH）**，由**慢性维生素 A（及其类似物 isotretinoin、ATRA，或合用四环素）过量**诱发。处理：停用维生素 A/相关药物，必要时乙酰唑胺 acetazolamide 降颅压、随访视力/视野。

# 营养_蛋白质能量营养不良（Protein-Energy Malnutrition, PEM）

> [!tip] 一句话锁定（Step 2）
> **有水肿、大肚子、皮肤脱屑、脂肪肝 → Kwashiorkor（蛋白缺、热量尚可）**；**极度消瘦、肌肉萎缩、无水肿、老人样面容 → Marasmus（总热量都缺）**。二者常一起考"哪个有 edema"。⚠️ 重度营养不良患者**再喂养要防再喂养综合征 refeeding syndrome**。

## 一、两型对比（核心表）

| 维度 | Kwashiorkor（恶性营养不良/夸希奥科病） | Marasmus（消瘦症/干瘦型） |
| --- | --- | --- |
| 主要缺什么 | **蛋白质缺乏**，热量可能尚够 | **总热量（含蛋白）全面缺乏** |
| 水肿 edema | **有**（低白蛋白 → 血浆胶体渗透压↓） | **无** |
| 体型/外观 | 四肢瘦但**腹部膨隆**、"大肚子小孩" | **极度消瘦**、皮包骨、"老人样/皱缩面容" |
| 皮下脂肪 | 相对保留 | **显著消耗殆尽** |
| 肌肉 | 消耗，但被水肿掩盖 | **严重肌肉萎缩 muscle wasting** |
| 肝脏 | **脂肪肝 fatty liver**（载脂蛋白合成↓、脂肪堆积） | 无脂肪肝 |
| 皮肤/毛发 | 皮炎（角化过度、脱屑、"剥漆样"）、毛发褪色 | 干燥、松弛，改变较轻 |
| 其他 | 贫血、易激惹、免疫↓ | 明显发育停滞、警觉可保留 |

> [!info] 缩写/名词
> PEM = protein-energy malnutrition，蛋白质-能量营养不良。两者可重叠（marasmic kwashiorkor）。经典诱因：断母乳后改喂**低蛋白高淀粉饮食**（Kwashiorkor）、或长期总量摄入极度不足（Marasmus）。

## 二、Kwashiorkor（蛋白缺乏为主）

> [!warning] 记忆口诀 MEALS（蛋白缺乏 → "缺蛋白的一餐"）
> - **M**alnutrition 营养不良（蛋白）
> - **E**dema 水肿（低白蛋白 hypoalbuminemia → 胶体渗透压↓）
> - **A**nemia 贫血
> - **L**iver（fatty liver）脂肪肝
> - **S**kin lesions 皮肤病变（角化过度 hyperkeratosis、脱屑 desquamation、"剥漆样皮炎 flaky-paint dermatosis"）

- 病理生理核心：**蛋白（白蛋白 albumin）不足 → 血浆胶体渗透压下降 → 全身性水肿 + 腹水 → 腹部膨隆**；同时肝内载脂蛋白 apolipoprotein 合成↓ → 脂肪运不出去 → **脂肪肝**。
- 典型 vignette：非洲/发展中地区**断奶幼儿**、四肢消瘦却**腹大**、下肢凹陷性水肿、毛发变浅、易怒。

## 三、Marasmus（总热量缺乏）

- 病理生理：能量全面不足 → 机体分解**脂肪 + 肌肉**供能 → 皮下脂肪消失、**肌肉萎缩**、体重严重低下。
- **不出现水肿**（与 Kwashiorkor 的关键区别）。
- 外观："老人脸"、皮包骨、皮肤松弛。
- 生长：**急性营养不良时线性（身高）生长可暂时维持**；慢性营养不良才出现矮小 stunting（身高别落后）。

## 四、诊断 & 处理（Step 2 视角）

### 评估

- 人体测量：体重别/身高别 Z 评分、中上臂围 MUAC (mid-upper arm circumference, 中上臂围)。
- 实验室：Kwashiorkor **白蛋白 albumin↓**；两型均可低血糖、电解质紊乱、贫血、免疫低下。

### ⭐再喂养综合征 Refeeding Syndrome（最高频陷阱）

> [!danger] 再喂养综合征 Refeeding syndrome
> 严重营养不良者**过快、过量再喂养（尤其大量碳水）** → 胰岛素骤升 → 细胞摄取电解质 →
> - **低磷血症 hypophosphatemia（最标志性）**
> - **低钾血症 hypokalemia**、**低镁血症 hypomagnesemia**
> - 维生素 B1（硫胺素 thiamine）骤耗 → Wernicke 风险
> 后果：心律失常、心衰、呼吸衰竭、抽搐，可致死。
> **防治**：**缓慢起始热量、逐步加量**；**先补/监测磷钾镁 + 先补硫胺素**再喂；密切监测电解质与容量。

### 治疗原则

- 稳定期先纠正脱水（谨慎，避免容量过负荷诱发心衰）、低血糖、低体温、感染、电解质。
- 逐步、渐进补充能量与蛋白（防再喂养综合征）。
- 补充微量营养素（锌、维生素 A、铁——铁在急性期常延后补）。

## 五、Memory Hook

> [!success] 记忆钩子
> - **Kwashiorkor = "Kwashi = 大肚水肿孩"**：蛋白缺 → 水肿 + 脂肪肝 + 大肚子（记 MEALS）。
> - **Marasmus = "muscle wasting，木乃伊样干瘦"**：全热量缺 → 消瘦 + 肌肉萎缩 + **无水肿**。
> - **区分钥匙 = 有没有水肿**：有水肿 → Kwashiorkor；纯消瘦无水肿 → Marasmus。
> - **喂回来最怕再喂养综合征**：低磷/钾/镁 + 缺 B1 → 慢慢喂、先补电解质和硫胺素。

## 六、复习自测

> [!question]- Q1. 非洲某地 2 岁幼儿，近期断母乳改喂稀粥（几乎无蛋白）。查体：四肢消瘦、腹部膨隆伴凹陷性水肿、毛发变浅、皮肤脱屑。血白蛋白显著降低。诊断及机制？
> **Kwashiorkor（蛋白缺乏型 PEM）**。低蛋白 → 血浆胶体渗透压↓ → 水肿 + 腹水（大肚子）；肝载脂蛋白合成↓ → 脂肪肝。记 MEALS（营养不良/水肿/贫血/脂肪肝/皮肤病变）。

> [!question]- Q2. 一名婴儿极度消瘦、皮包骨、明显肌肉萎缩和皮下脂肪消失，但**无水肿**。最可能诊断？与 Q1 的关键区别是什么？
> **Marasmus（总热量缺乏型 PEM）**。区别关键：Marasmus **无水肿**、以肌肉/脂肪消耗为主；Kwashiorkor 有水肿 + 脂肪肝、蛋白缺为主而热量可能尚够。

> [!question]- Q3. 一名严重营养不良患者住院后开始积极营养支持，第 3 天出现肌无力、心律失常，血磷、血钾、血镁均低。诊断及预防要点？
> **再喂养综合征 refeeding syndrome**。快速再喂（尤其碳水）→ 胰岛素↑ → 磷/钾/镁向细胞内转移 + 硫胺素骤耗。预防：**低热量缓慢起始、逐步加量，先纠正并监测磷/钾/镁、先补硫胺素（维生素 B1）**。

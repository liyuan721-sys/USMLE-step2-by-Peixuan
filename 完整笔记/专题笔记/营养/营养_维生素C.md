# 营养_维生素C（Vitamin C, Ascorbic acid 抗坏血酸）

> [!tip] 一句话锁定（Step 2）
> 老年"tea and toast"/酗酒/极少果蔬 → **牙龈肿胀出血 + 毛囊周围出血 perifollicular hemorrhage + corkscrew hair（螺旋状毛发）+ 伤口不愈 + 瘀斑** → 坏血病 scurvy → 补维生素 C 数天内改善。

## 一、生理作用（Step 2 够用即可）

维生素 C（抗坏血酸）是水溶性还原剂/辅因子：

- **胶原合成 collagen synthesis**：**脯氨酸/赖氨酸羟化酶（prolyl / lysyl hydroxylase）辅因子** → 稳定三螺旋胶原。缺乏 → 血管/结缔组织脆弱 → 出血、伤口不愈。
- **促非血红素铁吸收（non-heme iron absorption）**：把 **Fe³⁺（铁离子）还原为 Fe²⁺（亚铁）** 利于小肠吸收。
- **多巴胺 β-羟化酶（dopamine β-hydroxylase）辅因子** → 参与去甲肾上腺素 norepinephrine 合成。
- 抗氧化。

## 二、缺乏 Deficiency

### 临床表现（vignette buzzwords = 坏血病 Scurvy）

- **牙龈肿胀、出血、松动/脱牙 swollen bleeding gums**。
- **毛囊周围出血 perifollicular hemorrhage**（毛囊周点状出血）。
- **corkscrew hair（螺旋状/开瓶器样毛发）** — 高度特异 buzzword。
- 皮肤瘀斑 ecchymoses、瘀点 petechiae，易碰伤。
- **伤口愈合不良 poor wound healing**，旧瘢痕裂开。
- **关节积血 hemarthrosis / 骨膜下出血**（尤儿童，致假性瘫痪、肢痛拒动）。
- 贫血 anemia（出血 + 铁吸收↓）、乏力、免疫力下降、易感染。

### 高危人群 / 病因

- **老年独居"tea and toast"饮食**（茶+吐司，几乎无果蔬）。
- **酗酒者 alcohol use disorder（酒精使用障碍）**、营养不良。
- 极少摄入水果蔬菜者、精神疾病/饮食失调、极端限制性饮食婴儿（长期只喂加热牛奶）。

### 诊断 / 下一步

- 主要**临床诊断**（典型皮肤/牙龈表现 + 饮食史）。
- 可测**血浆/白细胞维生素 C 水平↓**。
- 补充维生素 C 后症状迅速改善也支持诊断。

### 治疗

- **口服维生素 C 补充**，症状（牙龈、出血、乏力）数天内明显改善。
- 纠正饮食、处理基础病因（酗酒/营养不良）。

## 三、过量 / 毒性 Excess

- 大剂量（常来自补充剂）：**恶心、腹泻、腹部绞痛**（渗透性）。
- **草酸钙肾结石 calcium oxalate stones**：维 C 代谢产草酸盐 oxalate 增多。
- **增强铁吸收 → 铁过载风险**：对 **血色病 hemochromatosis（遗传性铁过载）** 或**输血依赖者（如地中海贫血 thalassemia）**尤其危险。

## 四、Step 2 高频关联 & 陷阱

> [!warning] 陷阱：坏血病"出血"但凝血正常
> scurvy 是**血管/结缔组织缺陷**（胶原不稳）导致出血，**PT、PTT、血小板、INR 均正常**。别误判为维 K 缺乏或血友病 hemophilia。鉴别钥匙 = 牙龈 + 毛囊周围出血 + corkscrew hair + 伤口不愈 + 饮食史。

- **corkscrew hair + perifollicular hemorrhage** = scurvy 招牌，见到直接选维生素 C。
- 儿童**下肢痛、拒绝行走（假性瘫痪）+ 骨膜下出血** + 偏食 → 想坏血病（易误诊为骨髓炎/虐待/白血病）。
- 缺铁性贫血患者补铁时**同服维生素 C 促铁吸收** — 正向应用考点。
- 反向陷阱：血色病 / 输血依赖者**不宜大量维生素 C**（加重铁过载）。

## 五、Memory Hook

> [!success] 记忆钩子
> - **"C for Collagen + Corkscrew hair"** → 缺 C = 胶原崩 + 螺旋发 + 牙龈出血 + 伤口不愈。
> - **"Scurvy = 血管出血，凝血指标干净"** → PT/PTT/血小板正常。
> - **"C 帮铁进门（Fe³⁺→Fe²⁺）"** → 促铁吸收：既治缺铁，也是血色病/输血者的雷。
> - **"过量 C → Ca oxalate 结石"**。

## 六、复习自测

> [!question]- Q1. 78 岁独居老人，长期只吃茶和吐司。牙龈肿胀出血、下肢毛囊周围点状出血、螺旋状毛发、旧伤口裂开。凝血正常。诊断与治疗？
> **坏血病 scurvy（维生素 C / 抗坏血酸缺乏）**。"tea and toast"饮食缺果蔬 → 胶原合成障碍 → 血管脆弱出血、伤口不愈、corkscrew hair。凝血正常（非维 K/血友病）。治疗：**口服维生素 C**，数天内改善。

> [!question]- Q2. 缺铁性贫血患者口服硫酸亚铁 ferrous sulfate，医生建议同服维生素 C，机制是什么？
> 维生素 C 把 **Fe³⁺ 还原为 Fe²⁺**，促进十二指肠**非血红素铁吸收**，提高补铁疗效。

> [!question]- Q3. 长期大剂量服维生素 C 补充剂的患者，最可能出现哪种肾脏并发症？为何血色病患者要避免？
> **草酸钙肾结石 calcium oxalate stones**（维 C 代谢生成草酸增多）。血色病 hemochromatosis / 输血依赖者应避免大量维 C，因为它**增强铁吸收**、加重铁过载。

# 营养_维生素B12（Cobalamin, 钴胺素）

> [!tip] 一句话锁定（Step 2）
> **巨幼细胞性贫血 megaloblastic anemia + 神经症状（振动/本体觉↓、共济失调、Babinski+）+ MMA↑ 且 homocysteine↑** → B12 缺乏。病因先想 **恶性贫血 pernicious anemia / 回肠病变 / 二甲双胍 metformin / PPI / 纯素食 / 胃切除**。治疗 **补 B12（恶性贫血终身）**。
> ⭐ 神经症状可**先于/独立于贫血**出现——别等贫血才想 B12。

> [!info]- 缩写全称（首次出现速查）
> - MCV = mean corpuscular volume，平均红细胞体积
> - MMA = methylmalonic acid，甲基丙二酸
> - homocysteine = 同型半胱氨酸
> - IF = intrinsic factor，内因子（胃壁细胞分泌）
> - SCD = subacute combined degeneration，亚急性联合变性
> - PPI = proton pump inhibitor，质子泵抑制剂
> - PA = pernicious anemia，恶性贫血
> - N₂O = nitrous oxide，一氧化二氮（笑气）
> - SAM = S-adenosylmethionine，S-腺苷甲硫氨酸（甲基供体）
> - CBC = complete blood count，全血细胞计数

---

## 一、生理作用（Step 2 够用即可）

B12 是**两个酶**的辅因子：

- **甲硫氨酸合成酶 methionine synthase**：homocysteine → methionine（生成 SAM 甲基供体）。缺乏 → homocysteine↑ + 髓鞘甲基化不足 → **脱髓鞘（神经症状）**，同时 folate 被困在 methyl-THF（"folate trap"）→ 巨幼贫。
- **甲基丙二酰-CoA 变位酶 methylmalonyl-CoA mutase**：methylmalonyl-CoA → succinyl-CoA。缺乏 → **MMA↑** + 异常脂肪酸掺入髓鞘（也促成神经损伤）。

> [!info] 吸收 & 储备（考点密集）
> - **来源**：**仅动物性食物**（肉、蛋、奶、鱼）；植物几乎不含活性 B12。
> - **吸收路径**：胃酸/胃蛋白酶释放 B12 → 唾液 R-binder 携带 → 胰酶在十二指肠转交 → **胃壁细胞的内因子 IF** 结合 → **回肠末端 terminal ileum** 吸收。这条链任一环断 → 缺乏。
> - **储备大**：肝储备可用 **数年**（3-5 年）→ 起病隐匿、慢，比叶酸慢得多。

## 二、缺乏 Deficiency

### 2.1 临床表现（vignette buzzwords）—— 两条独立通路

```text
B12 缺乏
  ├─ 血液通路 → 巨幼细胞性贫血
  │     MCV↑ + hypersegmented neutrophils（核分叶过多）
  │     舌炎 glossitis、乏力、苍白
  │
  └─ 神经通路 → 亚急性联合变性 SCD + 周围神经病 + 神经精神改变
        后索 dorsal column → 振动觉/本体觉↓、Romberg+、感觉性共济失调
        侧索 lateral CST → Babinski+、痉挛、腱反射亢进（UMN 征）
        周围神经 → 手套袜套样麻木、踝反射消失（LMN 征）
        认知/情绪 → 记忆↓、抑郁、"megaloblastic madness"
        ⭐ 神经症状可先于贫血、甚至无贫血就出现
```

> [!warning] "UMN + LMN 并存"陷阱
> SCD 可**同时**有 UMN 征（Babinski+、腱反射亢进）和 LMN/周围神经征（踝反射消失、感觉丧失）。看到"混合体征" + 巨幼贫 → 想 B12。SCD 的脊髓横断面定位机制详见 [[完整笔记/专题笔记/神经/神经_脊髓病变]]（Posterior Cord / B12 SCD 节），此处不重复解剖细节。

### 2.2 高危人群 / 病因（Step 2 高频，按机制分）

| 机制 | 具体病因 | 关键点 |
|---|---|---|
| **自身免疫**（IF 缺）| **恶性贫血 pernicious anemia** ⭐⭐ | 抗内因子抗体 + 抗胃壁细胞抗体；萎缩性胃炎；与其他自免病、胃癌风险相关 |
| **胃源性**（IF/酸缺）| 萎缩性胃炎、**胃切除/胃旁路减重术**、长期 **PPI 质子泵抑制剂** | 无 IF 或无胃酸释放 B12 |
| **回肠吸收障碍** | **克罗恩病 Crohn / 回肠末端切除**、乳糜泻、热带口炎性腹泻 | 回肠末端是唯一吸收位点 |
| **药物** | **二甲双胍 metformin** ⭐（长期）、PPI、N₂O 笑气滥用 | metformin 干扰回肠 B12-IF 吸收；N₂O 氧化钴使 B12 失活 |
| **竞争/感染** | **阔节裂头绦虫 Diphyllobothrium latum**（生淡水鱼）| 绦虫夺取 B12 |
| **摄入不足** | **纯素食 vegan**（数年）、纯素母亲哺乳的婴儿 | 详见 [[完整笔记/专题笔记/营养/营养_素食营养缺乏]] |

> [!info] 恶性贫血 pernicious anemia 是"考点之王"
> - 机制：自身免疫破坏胃壁细胞 → IF↓ + 胃酸↓（萎缩性胃炎）。
> - 伴随：可合并其他自免病（甲状腺病、白癜风、1 型糖尿病）；**长期增加胃腺癌风险** → 需胃镜监测。
> - 诊断：查 **抗内因子抗体（anti-IF）**（特异度高）± 抗壁细胞抗体；血清胃泌素↑、胃酸↓。

### 2.3 诊断（关键 labs / 首选检查）

| 指标 | B12 缺乏 | 意义 |
|---|---|---|
| 血清 B12 | **↓**（可"正常低值"仍缺）| 首选筛查，但敏感度有限 |
| **MMA** | **↑** ⭐ | 最敏感/特异；区别叶酸缺乏（叶酸 MMA 正常）|
| **homocysteine** | **↑** | 叶酸/B12 都会↑ |
| MCV | ↑ | 巨幼贫（神经型可无贫血）|
| 涂片 | 大卵圆红细胞 + hypersegmented neutrophils | 巨幼贫标志 |
| 抗内因子抗体 | 阳性→恶性贫血 | 病因诊断 |

> [!tip] 诊断流程要点
> 1. 巨幼贫 + homocysteine↑ → 查 **MMA** 区分：**MMA↑ = B12**；MMA 正常 = 叶酸。
> 2. 确认 B12 缺乏后 → 查病因，尤其 **抗内因子抗体**（恶性贫血），问减重手术史/回肠病史/metformin/PPI/素食史。
> 3. 有神经症状但 B12 处于"临界" → MMA/homocysteine 升高即支持缺乏，别被正常 B12 值误导。

### 2.4 治疗 / 下一步

- **补充 B12**：肌注 IM（尤其吸收障碍/恶性贫血/严重神经症状）或高剂量口服。
- **恶性贫血 / 全胃切除 / 回肠切除 → 需终身补充**（IF 或吸收位点永久缺失）。
- ⭐ 有神经症状**尽早治疗**——SCD 早期可逆，晚期可遗留永久损害。
- ⭐ 若同时缺叶酸，**先补/先确认 B12**，避免单补叶酸掩盖神经损伤（见陷阱）。

## 三、过量 / 毒性 Excess

> [!info] B12 基本无毒性综合征
> B12 是水溶性、储量受吸收限速，**无明确中毒综合征**，大剂量口服/注射安全。临床上血清 B12 显著升高反而更多提示**潜在病理**（肝病、骨髓增殖性肿瘤等），不是"补过量中毒"。

## 四、Step 2 高频关联 & 陷阱

> [!danger] ⭐⭐ 致命陷阱：单补叶酸掩盖 B12 神经损伤
> 给 B12 缺乏者单独补叶酸 → 巨幼贫血象好转，但**神经损伤（SCD）继续甚至加速**且被"血象正常"掩盖。**任何巨幼贫先排/先纠正 B12**；有神经线索一定是 B12。详见 [[完整笔记/专题笔记/营养/营养_维生素B9_叶酸]]。

> [!warning] B12 vs 叶酸速查
> | 维度 | B12 | 叶酸 |
> |---|---|---|
> | 神经症状 / SCD | **有** | 无 |
> | MMA | **↑** | 正常 |
> | 储备耗尽 | 数年 | 数月 |
> | 来源 | 仅动物性 | 绿叶菜/豆类 |

> [!warning] 其他高频钩子
> - **年轻人不明原因 SCD** → 问 **N₂O（笑气）滥用**史（氧化钴使 B12 失活，labs 可正常 B12 但 MMA↑）。
> - **减重胃旁路/胃切除术后数年**贫血 + 神经症状 → B12（也常合并铁、叶酸缺乏）。
> - **长期 metformin 的糖尿病人**出现周围神经病 → 别只归咎糖尿病神经病变，查 B12（metformin 致 B12 吸收↓）。
> - **纯素母亲纯母乳婴儿**发育迟缓/肌张力低 → 婴儿 B12 缺乏，见 [[完整笔记/专题笔记/营养/营养_素食营养缺乏]]。
> - **生淡水鱼史（北欧/湖区）+ 巨幼贫** → 阔节裂头绦虫 Diphyllobothrium latum。

## 五、Memory Hook

> [!success] 记忆锚
> - **B12 = 动物 + 神经**：只来自动物性食物，且是**唯一管神经**的巨幼贫维生素。
> - **两条路**：巨幼贫（MCV↑、核分叶过多）+ 神经（后索/侧索/周围神经 + 认知）。
> - **MMA↑ = B12**（叶酸缺乏 MMA 正常）——分两者的钥匙。
> - **吸收链**：胃酸 → IF（壁细胞）→ 回肠末端；任一环断即缺（PA / 胃切 / Crohn / 回肠切）。
> - **储备数年**：起病慢，隐匿；但神经症状可先于贫血。
> - **药物记 3 个**：metformin、PPI、N₂O。
> - **恶性贫血 = 抗 IF 抗体 + 终身补 + 查胃镜**。

## 六、复习自测

> [!question]- Q1. 62 岁女性，疲乏 + 双足麻木、步态不稳，Romberg 阳性。CBC：MCV 116，涂片见核分叶过多中性粒细胞。homocysteine 与 MMA 均升高。最可能诊断 + 下一步确诊病因的检查？
> **B12（cobalamin）缺乏**，伴亚急性联合变性 SCD。**MMA↑ 且 homocysteine↑** 指向 B12（叶酸缺乏 MMA 正常）。神经症状 + 巨幼贫 = 经典 B12。下一步查 **抗内因子抗体（anti-IF）** 明确是否 **恶性贫血 pernicious anemia**（老年女性 + 萎缩性胃炎最常见病因）。治疗：IM B12，恶性贫血需终身补充；尽早治神经症状以求可逆。

> [!question]- Q2. 58 岁男性，2 型糖尿病服二甲双胍 metformin 8 年，近来手足对称性麻木、平衡差。除糖尿病神经病变外，还应排查什么？为什么？
> 应查 **血清 B12（+ MMA/homocysteine）**。**长期 metformin 干扰回肠末端 B12-IF 吸收**，可致 B12 缺乏，其周围神经病/SCD 症状易被误当成单纯糖尿病神经病变。漏诊会延误可逆性神经损伤的治疗。

> [!question]- Q3. B12 缺乏与叶酸缺乏都出现大细胞贫血。仅凭一项实验室指标区分二者，选哪个？结果如何解读？
> 选 **MMA（甲基丙二酸）**。**MMA↑ = B12 缺乏**（因 MMA 通路依赖甲基丙二酰-CoA 变位酶，需 B12）；**MMA 正常 = 叶酸缺乏**。homocysteine 两者都升高，不能区分。临床意义：有神经症状 + MMA↑ → 锁 B12，避免单补叶酸掩盖神经损伤。

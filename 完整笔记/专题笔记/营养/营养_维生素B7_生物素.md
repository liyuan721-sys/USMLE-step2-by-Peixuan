# 营养_维生素B7（Biotin, 生物素）

> [!tip] 一句话锁定（Step 2）
> 皮炎 + 肠炎 + 脱发，且病史里有"大量生食蛋清"或"长期广谱抗生素" → 想 Biotin (B7, 生物素) 缺乏 → 记忆句 "avidin in egg whites avidly binds biotin"（生蛋清里的亲和素牢牢结合生物素）。

## 一、生理作用（Step 2 够用即可，简短）

- 维生素 B7 = Biotin (生物素)，是多种 Carboxylase (羧化酶) 的辅因子，负责羧化反应（把 CO₂ 装到底物上）。
- Step 2 记住三个关键依赖生物素的羧化酶：
  - Pyruvate carboxylase (丙酮酸羧化酶)：丙酮酸 → 草酰乙酸，糖异生的第一步。
  - Acetyl-CoA carboxylase (乙酰辅酶 A 羧化酶)：脂肪酸合成的限速步。
  - Propionyl-CoA carboxylase (丙酰辅酶 A 羧化酶)：奇数链脂肪酸 / 某些氨基酸代谢。
- 一句话：生物素 = 羧化酶的搬 CO₂ 助手，管糖异生与脂肪酸合成。

## 二、缺乏 Deficiency

> [!info] 缺乏罕见
> 生物素由肠道菌群合成并广泛存在于食物，孤立缺乏少见；出现时几乎都有明确诱因（生蛋清 / 抗生素）。

### 临床表现（vignette buzzwords）

- Dermatitis (皮炎)：常为口周 / 面部鳞屑性红斑。
- Enteritis (肠炎) / 胃肠道症状。
- Alopecia (脱发)。
- 也可有神经系统症状（意识改变、感觉异常）、结膜炎、乳酸酸中毒（羧化酶功能受损）。

> [!tip] B7 与 B5 皮肤三联重叠
> Biotin (B7) 与 Pantothenic acid (B5, 泛酸) 缺乏共享【皮炎 + 肠炎 + 脱发】三联。鉴别靠病史：B7 = 生蛋清 / 抗生素；B5 = 全面营养不良 + 烧灼足 (burning feet)。

### 高危人群 / 病因

> [!warning] 经典病因——记这两条就够应付 Step 2
> - ⭐大量长期生食蛋清 (raw egg whites)：蛋清含 Avidin (亲和素)，与生物素以极高亲和力牢固结合，使其无法吸收。健美 / 生鸡蛋饮食者是经典 vignette。加热可使 avidin 变性失活，所以熟蛋不致病。
> - 长期广谱抗生素 (broad-spectrum antibiotics)：杀灭合成生物素的肠道菌群。
> - 其他：长期全肠外营养 (TPN, total parenteral nutrition) 未补生物素；遗传性生物素酶缺乏 (biotinidase deficiency) 或 holocarboxylase 缺乏（婴儿，新生儿筛查覆盖）；长期抗癫痫药 (如 phenytoin 苯妥英、carbamazepine 卡马西平) 可加速生物素代谢。

### 诊断（关键 labs / 首选检查）

- 多为临床诊断（典型三联 + 病史）。
- 可测血清生物素水平及尿有机酸（羧化酶缺陷时异常）。
- 婴儿遗传性生物素酶缺乏由 newborn screening (新生儿筛查) 发现。

### 治疗 / 下一步

- 补充 Biotin (生物素)；停止大量生蛋清摄入 / 停用诱因抗生素。
- 遗传性生物素酶缺乏：终身口服生物素替代。
- 症状可逆。

## 三、过量 / 毒性 Excess

> [!info] 无明确毒性综合征
> 生物素水溶性、无已知毒性上限。
>
> ⚠️ 但有一个考试 / 临床陷阱：大剂量生物素补剂可干扰某些基于 biotin-streptavidin 的免疫检测，造成实验室假象——典型是假性 TSH (thyroid-stimulating hormone, 促甲状腺激素) 降低 + 假性 T4 升高（类似甲亢的假象），以及某些检测中 troponin (肌钙蛋白) 假性降低。遇到"化验像甲亢但临床不符 + 在服生物素补剂"时，停补剂复查。

## 四、Step 2 高频关联 & 陷阱

- ⭐经典 vignette：健美 / 生蛋饮食者出现脱发 + 皮炎 → 生物素缺乏（avidin 结合）。
- 陷阱：生物素补剂干扰甲状腺功能检测 → 假甲亢化验，别误诊 Graves；处理是停生物素后复查，而非抗甲状腺治疗。
- 婴儿难治性皮疹 + 脱发 + 代谢性 / 乳酸酸中毒 + 惊厥 → 想遗传性 biotinidase deficiency（生物素酶缺乏），补生物素。
- 与 B5 皮肤表现鉴别靠病史（见上）。

## 五、Memory Hook

> [!success] 记忆钩子
> - "avidin in egg whites AVIDLY binds biotin" —— avidin（亲和性 avid）像强力胶粘住生物素。
> - "B7 → 三个羧化酶（丙酮酸 / 乙酰 / 丙酰）搬 CO₂" → 缺 → 糖异生 + 脂肪合成瘫痪。
> - 缺乏三联 = 皮炎 + 肠炎 + 脱发（与 B5 共享，B7 有生蛋清病因）。
> - 补剂陷阱："吃 biotin 让化验假装甲亢" → 停药复查。

## 六、复习自测

> [!question]- Q1. 一名健身者为增肌每天生吞多枚生鸡蛋，数月后出现脱发、鳞屑性皮炎和乏力。最可能缺乏的维生素及机制？
> Biotin (B7, 生物素) 缺乏。生蛋清中的 Avidin (亲和素) 以极高亲和力牢固结合生物素、阻断吸收（"avidin avidly binds biotin"）。加热可使 avidin 失活，故熟蛋无此风险。

> [!question]- Q2. 一名无甲状腺症状的女性体检示 TSH 极低、T4 偏高，临床与影像均不支持甲亢；追问其每天服用高剂量生物素美容补剂。下一步？
> 停用生物素补剂后复查甲状腺功能。大剂量生物素干扰 biotin-streptavidin 免疫检测，造成假性 TSH 降低 + T4 升高的假甲亢化验，而非真正甲状腺疾病。

> [!question]- Q3. 生物素缺乏与泛酸 (B5) 缺乏的皮肤表现高度重叠，如何用病史区分？
> 两者都可有皮炎 + 肠炎 + 脱发。B7 的特异病因是大量生蛋清或长期广谱抗生素；B5 缺乏见于全面营养不良并有特征性 "burning feet syndrome"（烧灼足）与肾上腺功能不全。

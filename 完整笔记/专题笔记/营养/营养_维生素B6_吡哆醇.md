# 营养_维生素B6（Pyridoxine, 吡哆醇）

> [!tip] 一句话锁定（Step 2）
> Isoniazid (异烟肼, 抗结核药) 或 OCP (oral contraceptive pill, 口服避孕药) 患者出现周围神经病 / 惊厥 / 铁粒幼细胞贫血 → 想 Pyridoxine (B6, 吡哆醇) 缺乏。异烟肼中毒难治性癫痫 → 解药是大剂量 B6。

## 一、生理作用（Step 2 够用即可，简短）

- 维生素 B6 = Pyridoxine (吡哆醇)，活性型是 PLP (pyridoxal phosphate, 磷酸吡哆醛)。
- PLP 是大量代谢反应的辅酶，Step 2 需要记住的几条主线：
  - Transamination (转氨基作用) → ALT / AST 等转氨酶都需要 PLP。
  - Decarboxylation (脱羧作用)。
  - Heme synthesis (血红素合成)：ALA synthase (δ-aminolevulinic acid synthase, δ-氨基酮戊酸合成酶) 需要 PLP → 缺乏 → 血红素合成障碍。
  - Neurotransmitter (神经递质) 合成：serotonin (5-羟色胺 / 血清素)、dopamine (多巴胺)、GABA (γ-aminobutyric acid, γ-氨基丁酸)、norepinephrine (去甲肾上腺素)。
- 记住一句：PLP 管"转氨 + 脱羧 + 血红素 + 神经递质"——这四条解释了它缺乏时的贫血 + 神经系统全套表现。

## 二、缺乏 Deficiency

### 临床表现（vignette buzzwords）

- Convulsions / seizures (惊厥 / 抽搐)：GABA（抑制性神经递质）合成需 PLP，缺乏 → 兴奋性失衡 → 抽搐。见于新生儿及异烟肼过量，可为难治性癫痫 (refractory seizures)，用 B6 解救。
- Peripheral neuropathy (周围神经病)：对称性感觉运动性。
- Sideroblastic anemia (铁粒幼细胞贫血)：因血红素合成障碍，骨髓可见 ring sideroblasts (环形铁粒幼细胞)——小细胞性、铁沉积于线粒体环绕核周。
- Angular cheilitis / stomatitis (口角炎 / 口炎) 与 glossitis (舌炎)。
- Dermatitis (皮炎)、易激惹、意识混乱。

> [!danger] 异烟肼中毒经典三联
> Isoniazid (异烟肼) 过量 → 【难治性癫痫 refractory seizures + 阴离子间隙代谢性酸中毒 anion-gap metabolic acidosis + 昏迷 coma】。机制：异烟肼耗竭 PLP → GABA 合成骤降。解药：大剂量 Pyridoxine (维生素 B6)，剂量常按摄入异烟肼克数 1:1 给（不知量时经验性大剂量）。标准抗癫痫药对此常无效，必须补 B6。

### 高危人群 / 病因

> [!warning] 高频病因——药物诱导
> - Isoniazid (异烟肼)：抗结核一线药，与 PLP 结合并促其排泄 → B6 缺乏 → 周围神经病。⭐临床上用异烟肼时常规预防性补充 B6，尤其糖尿病、酗酒、营养不良、妊娠、HIV 感染者、老年等高危人群。
> - OCP (oral contraceptive pill, 口服避孕药) / 雌激素。
> - 其他：Hydralazine (肼屈嗪, 降压药)、Penicillamine (青霉胺) 也可致 B6 缺乏。
> - 酗酒、慢性营养不良、透析。

### 诊断（关键 labs / 首选检查）

- 多为临床诊断（药物史 + 典型表现）。
- 可测血浆 PLP 水平。
- 铁粒幼细胞贫血时：小细胞或正细胞贫血、血清铁 / 铁蛋白升高、骨髓普鲁士蓝染色见环形铁粒幼细胞。

### 治疗 / 下一步

- 补充 Pyridoxine (吡哆醇)。
- 用异烟肼者：预防性 B6（见上）。
- 异烟肼中毒 / 难治性癫痫：大剂量静脉 B6 解救。
- B6 反应性铁粒幼细胞贫血：补 B6 可显著改善。

## 三、过量 / 毒性 Excess

> [!danger] 过量 B6 → 感觉性周围神经病
> 长期大剂量补充 Pyridoxine (吡哆醇) → Sensory peripheral neuropathy (感觉性周围神经病)：肢体麻木、本体感觉 / 振动觉受损、共济失调步态。这是维生素中"缺也神经病、过也神经病"的经典双向陷阱——见到大量服用 B6 补剂者出现感觉性神经病，别再往缺乏想。

## 四、Step 2 高频关联 & 陷阱

- ⭐最高频：结核患者 (TB, tuberculosis) 用 isoniazid → 数月后手足麻木 / 感觉异常 → 答案是补 B6（既是治疗也是本该预防的缺乏）。
- ⭐异烟肼过量急诊：难治性癫痫 + 代谢性酸中毒 + 昏迷 → 立即给大剂量 B6，标准抗癫痫药无效。
- 铁粒幼细胞贫血鉴别：B6 缺乏、铅中毒 (lead poisoning, 铅也抑制血红素合成酶)、酒精、遗传性 (X 连锁 ALAS2 突变) 都可致。vignette 里"抗结核药 / 长期营养不良 + 环形铁粒幼细胞" → 指向 B6。
- 双向陷阱：缺乏 → 周围神经病；过量补剂 → 也是感觉性周围神经病。靠病史（服补剂 vs 缺乏背景）区分。
- 新生儿难以控制的惊厥、常规抗癫痫药无效 → 试 pyridoxine-dependent seizures（吡哆醇依赖性癫痫），给 B6。

## 五、Memory Hook

> [!success] 记忆钩子
> - "B6 = Pyridox-in" → 想到 iso-niazid，两者绑定：用 INH 必补 B6。
> - PLP 管四件事："转氨 + 脱羧 + 血红素 + 神经递质" → 缺 → 贫血（铁粒幼）+ 神经全套（抽搐 / 周围神经病）。
> - GABA 靠 B6 合成 → B6 掉 → 刹车失灵 → 抽搐（异烟肼中毒难治性癫痫的核心）。
> - "太少也麻，太多也麻"：B6 缺乏与过量都能造成周围神经病。

## 六、复习自测

> [!question]- Q1. 一名新诊断肺结核患者开始四联抗结核治疗，3 个月后诉双手足对称麻木、刺痛。最可能病因及处理？
> Isoniazid (异烟肼) 诱导的 Pyridoxine (B6, 吡哆醇) 缺乏性周围神经病。处理：补充维生素 B6（且本应在用异烟肼时预防性给予，尤其高危人群）。

> [!question]- Q2. 急诊送来一名误服大量异烟肼的患者，反复全身强直阵挛发作，苯二氮䓬类与常规抗癫痫药无效，血气示阴离子间隙代谢性酸中毒。最有效的解救药物是？
> 大剂量 Pyridoxine (维生素 B6)。异烟肼耗竭 PLP → GABA 合成骤降 → 难治性癫痫；标准抗癫痫药无效，必须补 B6（常按摄入异烟肼克数 1:1）。

> [!question]- Q3. 一名健康女性长期自行大量服用维生素 B6 补剂，出现下肢麻木、振动觉减退和步态不稳。诊断？
> 维生素 B6 (Pyridoxine) 过量 → Sensory peripheral neuropathy (感觉性周围神经病)。提示 B6 缺乏与过量均可致周围神经病，靠补剂史区分。

# 营养_素食营养缺乏

> [!tip] 一句话锁定
> 题干出现 **strict vegan / 纯素多年**（不吃肉、蛋、奶、鱼）→ 几乎都在考 **可预测的营养素缺乏**。第一反射 **维生素 B12（cobalamin，钴胺素）缺乏**（巨幼贫 + 亚急性联合变性 Subacute Combined Degeneration, SCD），其次 **铁**、**维生素 D / 钙**。

> [!info] 素食分型（先分清题干问的是哪种）
> - **蛋奶素 Lacto-ovo vegetarian**：吃蛋 + 奶 → B12 通常**不缺**（蛋奶含 B12）。
> - **严格素食 / 纯素 Vegan**：完全无动物性食物 → **B12 高危**，是考点主体。
> - 越"纯"、时间越长、覆盖到孕妇/婴儿 → 越危险。

> [!info]- 缩写全称（首次出现速查）
> - **MCV** = mean corpuscular volume，平均红细胞体积
> - **MMA** = methylmalonic acid，甲基丙二酸
> - **homocysteine** = 同型半胱氨酸
> - **SCD** = Subacute Combined Degeneration，亚急性联合变性
> - **SAM** = S-adenosylmethionine，S-腺苷甲硫氨酸（甲基供体）
> - **CST** = corticospinal tract，皮质脊髓束
> - **UMN / LMN** = upper / lower motor neuron，上/下运动神经元
> - **ferritin** = 铁蛋白；**TIBC** = total iron binding capacity，总铁结合力
> - **PTH** = parathyroid hormone，甲状旁腺激素；**ALP** = alkaline phosphatase，碱性磷酸酶
> - **TSH** = thyroid stimulating hormone，促甲状腺激素；**T4** = thyroxine，甲状腺素
> - **EPA / DHA** = 二十碳五烯酸 / 二十二碳六烯酸（Omega-3 脂肪酸）；**ALA** = alpha-linolenic acid，α-亚麻酸
> - **N₂O** = nitrous oxide，一氧化二氮（笑气）

## 一、缺乏素总览（按 USMLE 频率排序）

| 缺乏营养素 | 疾病 / 后果 | 关键实验室 | 高频陷阱 / 鉴别 |
|---|---|---|---|
| **维生素 B12 cobalamin** ⭐ | 巨幼细胞性贫血 megaloblastic anemia + **亚急性联合变性 SCD**（神经） | MCV↑、hypersegmented neutrophils、**MMA↑ + homocysteine↑** | 与叶酸缺乏鉴别；神经症状可先于贫血；单补叶酸掩盖并加重神经损伤 |
| **铁 iron** | 缺铁性贫血 iron deficiency anemia | 小细胞低色素、ferritin↓、TIBC↑ | 植物铁是非血红素铁 non-heme，吸收差 + 植酸盐 phytate 抑制吸收 |
| **维生素 D + 钙 calcium** | 儿童佝偻病 rickets / 成人骨软化 osteomalacia / 骨质疏松 osteoporosis | 25-OH-D↓、Ca↓/正常、PTH↑、ALP↑ | 无日晒 + 深肤色 + 母乳纯喂养未补更危险 |
| **锌 zinc** | 味觉减退、伤口愈合差、脱发、肢端/口周皮炎 | 血锌↓ | 表现类似 acrodermatitis enteropathica |
| **碘 iodine** | 甲状腺肿 goiter / 甲减 hypothyroidism | TSH↑、T4↓ | 不用碘盐 iodized salt 时风险↑ |
| **Omega-3（EPA/DHA）** | 主要来自鱼类，纯素摄入低 | — | 一般靠亚麻籽/核桃 ALA 部分代偿 |
| **蛋白质-能量 protein-energy** | 计划良好的成人罕见；喂养不当**婴幼儿**可致 kwashiorkor | 白蛋白↓ | 成人精心搭配豆类 + 谷物即可满足 |

## 二、维生素 B12 缺乏 —— 素食题的"标准答案"⭐

> [!danger] 为什么素食者必缺 B12
> B12 天然来源**几乎只存在于动物性食物**（肉、蛋、奶、鱼）。植物性食物基本不含活性 B12 → 纯素者若不额外补充或吃强化食品，**数年后**（肝储备耗尽）必发病。母乳纯喂养的纯素母亲可致**婴儿 B12 缺乏**（发育迟缓、肌张力低）。

### 2.1 两条独立后果通路（都要记）

```text
B12 缺乏
  ├─ 血液通路 → 巨幼细胞性贫血
  │     MCV↑ + hypersegmented neutrophils（中性粒核分叶过多）
  │
  └─ 神经通路 → 亚急性联合变性 SCD
        后索 + 皮质脊髓侧束 + 脊髓小脑束（对称受累）
        ⭐ 神经症状可先于贫血出现，甚至无贫血
```

### 2.2 亚急性联合变性 SCD 机制（脊髓专题精讲）

- B12 是甲硫氨酸合成酶 methionine synthase 辅酶 → 缺乏 → **SAM（甲基供体）↓ → 髓鞘甲基化不足 → 脱髓鞘**。
- 同时甲基丙二酰-CoA 变位酶 mutase 失活 → **异常奇数碳/支链脂肪酸掺入髓鞘** + **MMA↑**。
- 病灶（对称、最长纤维束最先受累）：

```text
后索 Dorsal columns   → 振动觉/本体觉丧失、感觉性共济失调、Romberg (+)
皮质脊髓侧束 CST      → 上运动神经元 UMN 体征：痉挛性瘫、腱反射亢进、Babinski (+)
脊髓小脑束            → 共济失调加重
（常合并周围神经病）  → 手套袜套样感觉异常、踝反射消失
```

> [!warning] 经典"混合体征"陷阱
> SCD 可同时出现 **UMN 体征（Babinski +、腱反射亢进）** 和 **LMN/周围神经体征（踝反射消失、感觉丧失）**——看到"UMN + LMN 并存"要想到 B12 缺乏。完整机制见 [[完整笔记/专题笔记/神经/神经_脊髓病变]]。

### 2.3 B12 vs 叶酸（最高频干扰点）

| 维度 | 维生素 B12 cobalamin | 叶酸 folate |
|---|---|---|
| 主要来源 | **动物性食物** | 绿叶菜、豆类（**素食反而不缺**）|
| 素食者风险 | **高** | 低 |
| 巨幼贫 | 有 | 有 |
| **神经损伤 / SCD** | **有** | **无** |
| MMA | **↑** | 正常 |
| homocysteine | ↑ | ↑ |

> [!danger] 致命陷阱（务必记）
> 给 B12 缺乏者**单独补叶酸**：能纠正贫血血象，却**不能阻止、甚至加速神经损伤**（叶酸不参与 MMA 通路、补不上 SAM）。任何巨幼贫**先查/先纠正 B12**。有神经线索的巨幼贫 → 一定是 B12 不是叶酸。

## 三、其他缺乏要点

> [!info] 铁 Iron
> 植物铁为**非血红素铁 non-heme iron**，吸收率远低于血红素铁；植酸盐 phytate、多酚进一步抑制吸收。→ 小细胞低色素性贫血，ferritin↓。搭配维生素 C 可促铁吸收。

> [!info] 维生素 D + 钙 Calcium
> 儿童 → 佝偻病 rickets；成人 → 骨软化 osteomalacia；长期 → 骨质疏松 osteoporosis。深肤色、少日晒、纯母乳未补者叠加风险。

> [!info] 锌 Zinc / 碘 Iodine
> - 锌：味觉减退、伤口难愈、脱发、口周与肢端皮炎（似 acrodermatitis enteropathica）。
> - 碘：不用碘盐时可致甲状腺肿 / 甲减。

## 四、Memory Hook

> [!success] 记忆锚
> - **不吃动物性食物 → 补 B12**（叶酸在菜里、B12 不在菜里；且只有 B12 管神经）。
> - **B12 缺乏两条路**：巨幼贫（MCV↑、核分叶过多）+ SCD（后索/侧索、Romberg + Babinski）。
> - **MMA↑ = B12**（叶酸缺乏 MMA 正常）——区分两者的钥匙。
> - **纯素还要盯**：铁（非血红素）、维生素 D/钙、锌、碘。
> - **N₂O（笑气）滥用**产生完全相同的 SCD（钴被氧化使 B12 失活）——年轻人不明 SCD 问笑气史。

## 五、复习自测

> [!question]- Q1. 健康年轻女性，严格素食 5 年（不吃肉、蛋、奶），已每日补铁。最应额外补充哪种营养素？
> **维生素 B12（cobalamin）**。B12 天然来源几乎只在动物性食物；缺乏致巨幼贫 + SCD 神经损伤。叶酸是最强干扰项，但素食者叶酸通常不缺。

> [!question]- Q2. 巨幼细胞性贫血 + 本体觉减退/共济失调 + Babinski 阳性 → 缺哪种？为什么不是叶酸？
> **维生素 B12**。只有 B12 缺乏累及后索/皮质脊髓束产生神经损伤（SCD），叶酸缺乏**无**神经症状。可查 MMA：B12 缺乏 MMA↑，叶酸缺乏正常。

> [!question]- Q3. 纯素妈妈纯母乳喂养 8 月龄婴儿，出现发育迟缓、肌张力低下、贫血。最可能的营养缺乏？
> **维生素 B12**。母乳 B12 依赖母体状态，纯素母亲乳汁 B12 极低 → 婴儿 B12 缺乏。

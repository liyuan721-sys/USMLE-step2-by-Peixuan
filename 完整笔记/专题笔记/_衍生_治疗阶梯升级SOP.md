# 治疗阶梯升级通用 SOP（USMLE Step 2 高频框架）

> [!tip] 一句话总览
> 看到 stem 出现 "**X 药 / X 治疗用了一段时间仍恶化 / 控制不充分 / 复发 / breakthrough**" → 这是**升级题**（step-up question）→ 找**下一阶梯**，不是同一阶梯横向换药，也不是简单加量。每个高频慢性病都有自己固定的阶梯次序，**记住次序 = 选对答案**。

---

## §一 升级题识别 — 6 大触发短语

> [!warning] 看到这 6 类短语 = 99% 是升级题
> 不要在同一阶梯里横移（α2 失败别换另一个 α2），要纵向升级到下一档治疗。

| 触发短语 | 中文 | 隐含信息 |
|---|---|---|
| **"despite X therapy / despite adequate trial of X"** | 尽管已用 X 治疗 | 该药已给够时间、剂量，仍未控制 → 升级 |
| **"inadequate control / inadequately controlled"** | 控制不充分 | 当前阶梯疗效不够 → 升级 |
| **"breakthrough symptoms"** | 突破性症状 | 维持药下症状仍冒头 → 升级（不只是加量）|
| **"progressed / worsening / progressively worse"** | 进行性恶化 | 疾病在进展，不是稳定 → 必须改方案 |
| **"failed adequate trial of X"** | X 充分试药失败 | 给够剂量 + 时间仍无效 → 换类别（不是换同类别另一个药）|
| **"X 用了 N 个月 / N 年仍恶化"** | 时间短语具体化 | USMLE 经典升级钩；如 Q3375 "guanfacine 用 1 年仍恶化" |

> [!info] 一个反例对照
> "**X 出现新副作用**" → ❌ 不是升级题，是**换同阶梯不同分子**（避免副作用，保持疗效层级）。
> 例：lithium 出现震颤 → 不是失效 → 换 valproate（同为一线维持），不是升到下阶梯。

---

## §二 升级 vs 同阶梯换药 vs 加量 — 三种"调整"陷阱

> [!danger] USMLE 最常混淆的 3 种"调药"动作
> 三者方向不同，看错 stem 关键词就选错。

| 调药动作 | 触发 | 例子 | 关键词 |
|---|---|---|---|
| **① 升级（step-up）** | 当前阶梯**疗效不足** | Tourette α2 用 1 年仍恶化 → 升到抗 DA（Q3375）| despite / inadequate / failed / progressed |
| **② 同阶梯换药** | 当前药**出现禁忌或副作用，疗效尚可** | Lithium 出现震颤 → 换 valproate（Q2357 镜像 — 那题是肾否决一线，不是疗效不足） | adverse effect / contraindication / new lab abnormality |
| **③ 加量（dose escalation）** | 当前药**未到上限 + 部分有效** | SSRI 用 6 周部分应答 → 先加量再考虑换 | partial response / not yet at maximum dose |

> [!tip] 决策口诀
> **"没效升一级，有事换一家，没满加一档"** — 疗效不足升阶梯 / 有禁忌换同阶梯 / 部分应答先加量。

## §三 各疾病升级阶梯模板（核心）⭐

> [!info] 7 大高频疾病阶梯
> 每个一张表 + 一个 ASCII 树，按"哪类病人卡在哪阶梯"组织。表格里所有 Q 号都是真实错题锚点，可点跳。

### 1. Tourette 三级阶梯 — from [[mistakes/uworld-mistakes#^Q3375]]

```
Tourette 综合征（>1 yr motor + vocal tic，起病 < 18 岁）
    │
    ├─ 第 1 阶梯：HRT（Habit Reversal Therapy，行为治疗，一线）
    │     ※ 培养 competing response 替代 tic；适合轻中度
    │     │ 无效 / 不可用 / 重度功能受损
    │     ▼
    ├─ 第 2 阶梯：α2 激动剂（共病 ADHD 尤佳）
    │     ※ Guanfacine（首选）/ Clonidine
    │     ※ 镇静 = 副作用 → 助眠 + 减焦虑 → 间接减 tic
    │     │ 无效（如 Q3375 "guanfacine 用 1 年仍恶化"）
    │     ▼
    └─ 第 3 阶梯：抗多巴胺药（最强、证据最足）⭐
          ├─ DA 耗竭：Tetrabenazine / Deutetrabenazine / Valbenazine（VMAT2 抑制剂）
          └─ D2 阻断：Risperidone（SGA 较常用）/ Haloperidol / Pimozide（用前查 ECG）
```

| 阶梯 | 治疗 | 卡点 / 升级触发 |
|---|---|---|
| 1 | HRT 行为治疗 | 轻中度首选；重度 / 行为治疗无效 → 升 2 |
| 2 | α2 激动剂（guanfacine 首选） | 共病 ADHD 尤佳；α2 失败别换另一 α2，要升 3 |
| 3 | 抗多巴胺药（risperidone / tetrabenazine） | tic 本质 = DA 过度活跃 → 降 DA 是终极 |

### 2. Bipolar I 维持治疗 — from [[mistakes/uworld-mistakes#^Q2357]]

```
Bipolar I 急性期稳定后 → 维持治疗 4 大一线（疗效相当）
    │
    └─ 选哪个 = 完全取决于患者个体禁忌（4 选 1 决策树）
         │
         ├─ Lithium      → 100% 肾排泄；肾损伤 / PKD 家族史 / 妊娠 → ❌
         ├─ Valproate    → 肝代谢；妊娠（NTD）/ 肝病 → ❌
         ├─ Quetiapine   → 肝代谢；代谢综合征 / QT 长 → ⚠️
         └─ Lamotrigine  → 肝代谢；防抑郁相最强；滴定快 → SJS/TEN
    │
    │（一线 4 个轮过仍控制不充分）
    ▼
联合治疗（lithium + valproate / lithium + quetiapine 等）
    │
    │（难治性双相）
    ▼
ECT（电休克）/ Clozapine（难治性 + 自杀风险高时）
```

| 阶梯 | 治疗 | 升级触发 |
|---|---|---|
| 1 | 一线单药（4 选 1：Li / VPA / Quet / LTG）| 按器官 / 妊娠 / 共病否决项选；Q2357 = 肾否决 Li → 换 VPA |
| 2 | 一线 4 个轮过仍不够 / 部分应答 | 联合（2 药）|
| 3 | 难治性 + 自杀风险 / 极重躁狂 | ECT / Clozapine |

> [!warning] Q2357 是**"两种升级触发机制"**的经典示范
> 这题不是"疗效不足升下阶梯"，而是"**器官功能否决一线药** → 换同阶梯另一个分子"（lithium → valproate，都属一线）。**这是 §二 第 ② 类"同阶梯换药"，不是真正的升级**。同一题里可能两种机制并存（如：肾差不能用 Li + 已用 VPA 也不够 → 升级到联合），要逐个识别。

### 3. PD Psychosis 5 步流程 — from [[mistakes/uworld-mistakes#^Q12152]]

```
PD 患者出现幻觉 / 妄想
    │
    ├─ 第 1 步：排除 infection / delirium / dementia（有 → 先治原发）
    │     │
    │     ▼ 排除阴性
    ├─ 第 2 步：症状轻微？→ 仅监测（monitor），不动药
    │     │
    │     ▼ 症状显著
    ├─ 第 3 步：减 / 停【非 PD 精神活性药】（镇静 / 抗焦虑 / TCA）
    │     │
    │     ▼ 题中无 / 减完无效
    ├─ 第 4 步：减【PD 药】，从效力最低的开始
    │     ※ 顺序：anticholinergic → amantadine → DA 激动剂 → MAO-B → COMT-I → levodopa
    │     │
    │     ▼ 减药无效 / 运动症状反弹无法承受（Q12152 卡这里）
    └─ 第 5 步：上【低 / 零 D2 抗精神病药】⭐ Q12152 答案
          ├─ Quetiapine（极弱 D2，临床首选 — availability + safety）
          ├─ Pimavanserin（零 D2，FDA 唯一专批 PD psychosis，但贵）
          └─ Clozapine（弱 D2，需 ANC 监测 agranulocytosis）
```

| 阶梯 | 治疗 | 关键 |
|---|---|---|
| 1-2 | 排除 / 监测 | 别一来就上药 |
| 3 | 减非 PD 精神活性药 | 看 stem 有没有 BZD / TCA / 镇静药 |
| 4 | 减 PD 药（从效力最低开始）| Q16164 在此 — 减药就够 |
| 5 | 上低 D2 抗精神病药（Quet 首选） | Q12152 在此 — 减药失败必上 quetiapine |

> [!danger] PD psychosis 特殊点 — "减药失败也算升级"
> 与一般慢性病不同，PD psychosis 的"升级"前几步是**减药**（因为元凶是 DA 过多），第 4 步减药失败 / 运动症状反弹 → 才升到第 5 步**加药**（低 D2 抗精神病药）。**"减"和"加"在同一流程里串联**，是 PD psychosis 独有的反直觉点。

### 4. 哮喘（Asthma）Step-up 阶梯

```
哮喘控制不充分 → 升一阶
    │
    ├─ Step 1：SABA PRN（轻间歇）
    │     ※ ≥2 次 / 周症状 → 升 2
    │     ▼
    ├─ Step 2：低剂量 ICS（首选）/ LTRA
    │     ▼
    ├─ Step 3：中剂量 ICS / 低剂量 ICS + LABA
    │     ▼
    ├─ Step 4：中-高剂量 ICS + LABA
    │     ▼
    ├─ Step 5：高剂量 ICS + LABA + LAMA / 生物制剂
    │     ※ Omalizumab（IgE 高 / 过敏哮喘）
    │     ※ Mepolizumab / Benralizumab（嗜酸性 哮喘）
    │     ※ Dupilumab（type 2 高）
    │     ▼
    └─ Step 6：高剂量 ICS + LABA + 系统类固醇（最后手段）
```

| Step | 治疗 | 升级触发（"control 不够"信号）|
|---|---|---|
| 1 | SABA PRN | ≥2 d/wk 症状 / ≥2 次 / 月夜醒 → 升 2 |
| 2 | 低剂量 ICS（首选）/ LTRA | 仍突破性发作 → 升 3 |
| 3 | 中 ICS / 低 ICS+LABA | 急诊就医 / 系统类固醇 ≥1/yr → 升 4 |
| 4 | 中-高 ICS+LABA | 仍未控制 → 升 5 |
| 5 | 高 ICS+LABA+LAMA / 生物制剂 | 按 IgE / 嗜酸 / type 2 表型选 |
| 6 | 系统类固醇 | 最后 + 副作用大 |

> [!tip] USMLE 反陷阱
> 看到"哮喘 + 现有方案控制不够" → **升一档不要跳级**；看到"频繁急性发作 + 系统类固醇 ≥2/yr" → 直接考虑生物制剂（Step 5），不是无限加 ICS 剂量。

### 5. HTN（高血压）加药阶梯

```
HTN（确诊 + 已生活方式干预 3-6 月）
    │
    ├─ 一线单药（按合并症选）
    │     ├─ 无合并症 / 黑人 / 老年 → CCB / Thiazide
    │     ├─ DM / CKD / 蛋白尿 → ACEI / ARB
    │     ├─ 心衰 / 心梗后 → ACEI + β-blocker + 醛固酮拮抗
    │     ▼ 单药未达标
    ├─ 二联（不同类联合）
    │     ※ ACEI/ARB + CCB / Thiazide 经典组合
    │     ▼
    ├─ 三联（ACEI/ARB + CCB + Thiazide）
    │     ▼ 三联仍未控（= Resistant HTN 定义）
    └─ Resistant HTN workup
          ├─ 确认依从性 + 排除白大衣 + 排除 OSA / NSAIDs / 酒
          ├─ 排除继发性 HTN（PA / RAS / 嗜铬 / Cushing / 主动脉缩窄）
          ├─ 加 Spironolactone（4th 药首选 — 即使 PA 阴性也常有效）
          └─ 介入：肾动脉去神经 / RAS 支架（按病因）
```

| 阶梯 | 治疗 | 升级触发 |
|---|---|---|
| 1 | 一线单药 | BP 未达标（>130/80 with DM/CKD；>140/90 general）→ 升 2 |
| 2 | 二联 | 仍未达标 → 升 3 |
| 3 | 三联（含一个利尿剂）| 仍未达标 = **Resistant HTN** 定义 |
| 4 | 加 spironolactone + 排继发病因 | 继发病因 → 针对治疗（PA 切肾上腺；RAS 支架）|

### 6. MDD（重度抑郁）部分应答阶梯

```
MDD 患者 SSRI 用 6-8 周 → 评估应答
    │
    ├─ 完全应答（remission）→ 维持 4-9 月
    ├─ 部分应答（response partial）→ 升级路径 A
    └─ 无应答（non-response）→ 升级路径 B
    │
═══════════════════════════════════════════
升级阶梯（路径 A + B 通用，按情况插入）
═══════════════════════════════════════════
    │
    ├─ 第 1 阶梯：加 SSRI 剂量到最大耐受
    │     ▼ 仍部分 / 无应答
    ├─ 第 2 阶梯：换 SSRI 类（fluoxetine → sertraline 等）
    │     ▼ 仍无效
    ├─ 第 3 阶梯：换不同类（SNRI / Bupropion / Mirtazapine）
    │     ※ Bupropion = 性功能保留 + 减体重 + 戒烟（共病优选）
    │     ※ Mirtazapine = 改善睡眠 + 增食欲（失眠 / 厌食优选）
    │     ※ SNRI = 共病慢痛 / 神经痛优选
    │     ▼ 部分应答（≥50% 改善但未 remission）
    ├─ 第 4 阶梯：Augmentation 增强治疗
    │     ├─ + Lithium（经典）
    │     ├─ + Atypical antipsychotic（aripiprazole / quetiapine / olanzapine）
    │     ├─ + T3（甲状腺激素）
    │     └─ + Buspirone
    │     ▼ 仍难治（Treatment-Resistant Depression, TRD）
    └─ 第 5 阶梯：神经调控 / 快速起效
          ├─ ECT（电休克 — 最强 + 起效最快，孕妇 / 老年精神病性抑郁 / 自杀危急首选）
          ├─ rTMS（经颅磁刺激 — 副作用最小）
          └─ Esketamine（鼻喷雾，需诊室监测 2 h）
```

| 阶梯 | 治疗 | 升级触发 |
|---|---|---|
| 1 | SSRI 加量到最大耐受 | partial response 先加量再换 |
| 2 | 换同类 SSRI | 一种 SSRI 无效不代表所有 SSRI 无效 |
| 3 | 换不同类（SNRI / Bupropion / Mirtazapine） | 按共病选 |
| 4 | Augment（Li / atypical / T3） | 部分应答时加第二药保留第一药 |
| 5 | ECT / rTMS / Esketamine | TRD = ≥2 类充分试药失败 |

### 7. T2DM 药物阶梯

```
T2DM 新诊断 + A1c 目标未达标
    │
    ├─ 基线：生活方式（饮食 + 运动 + 减重）+ Metformin
    │     ※ Metformin 一线（除非 eGFR <30 / 严重肝病）
    │     ▼ A1c 仍未达标（一般 >7%，按个体调）
    ├─ 第 2 阶梯：按合并症加第二药
    │     ├─ ASCVD / 高危 ASCVD → 加 GLP-1 RA 或 SGLT2i
    │     ├─ 心衰（HFrEF）→ 加 SGLT2i ⭐
    │     ├─ CKD（eGFR 25-60 / 蛋白尿）→ 加 SGLT2i ⭐
    │     ├─ 肥胖 → GLP-1 RA（减重最强）
    │     └─ 无以上 → DPP-4i / SU / TZD（成本 / 副作用考量）
    │     ▼ 仍未达标
    ├─ 第 3 阶梯：三药联合（覆盖不同机制）
    │     ▼
    ├─ 第 4 阶梯：加 Basal Insulin（一般睡前 NPH / glargine）
    │     ▼ Basal 控制 FBG 但餐后高
    └─ 第 5 阶梯：Basal-Bolus 强化方案 / 复合方案（Basal + GLP-1 RA 复合）
```

| 阶梯 | 治疗 | 升级触发 |
|---|---|---|
| 1 | 生活方式 + Metformin | A1c >7%（个体化）→ 升 2 |
| 2 | + GLP-1 RA / SGLT2i（按合并症） | 仍未达 → 升 3 |
| 3 | 三药口服 ± 注射 | 仍未达 → 升 4 |
| 4 | + Basal insulin | FBG 控制但 PPG 高 → 升 5 |
| 5 | Basal-Bolus / 复合方案 | 个体化强化 |

> [!warning] T2DM 升级**不是简单加 metformin 剂量**
> Metformin 已到最大耐受（2000-2550 mg/d）仍 A1c 不达标 → 加第二药（不是无限加 metformin）。USMLE 经典陷阱：stem 已说 metformin 2000 mg + A1c 8.5% → 选项里"加 metformin 至 3000 mg" = 错（已到上限）；正确答案是按合并症加 SGLT2i / GLP-1 RA。

## §四 升级 vs 同阶梯 vs 加量 — 决策树

```
看到 stem "X 治疗已用一段时间 → 控制不够 / 副作用 / 新症状"
        │
        ▼
   先问三连问：
        │
        ├─ Q1：当前药给够剂量了吗？
        │     ├─ 没到上限 + partial response → ② 加量
        │     └─ 已到上限 → 继续 Q2
        │
        ├─ Q2：当前药出问题是因为"无效"还是"有禁忌 / 副作用"？
        │     ├─ 出现新禁忌 / 副作用，疗效尚可 → ① 同阶梯换药（找同档不同分子）
        │     └─ 疗效不足（despite / inadequate / failed） → 继续 Q3
        │
        └─ Q3：当前在哪一阶梯？下一阶梯是什么？
              └─ 查 §三 对应疾病阶梯表 → 选下一档 ⭐
                    ※ 注意：同阶梯有多个分子时（如 Bipolar 一线 4 个）
                       要先轮过同阶梯再升下阶梯
```

> [!tip] 终极口诀
> **"先看到位没（加量？）→ 再看是事还是没效（换 vs 升）→ 升要查阶梯次序，不要跳级"**

## §五 通用陷阱（USMLE 高频钩）

> [!danger] 6 大反复错点
> 这些都是真实错题暴露出来的陷阱，每条挂对应 Q 号。

1. **"已用 1 年仍恶化" = 升级题 ≠ 加量**（Q3375）
   - α2 已用 1 年仍恶化 → 不是"加 guanfacine 剂量"，是升到抗 DA。
   - 反射钩：**时间短语 + 进行性恶化** = 升级，不是加量。

2. **升级 ≠ 换同阶梯药**（Q3375）
   - α2 失败别换 clonidine ↔ guanfacine（同档横移无意义），要**升到下档**抗 DA。
   - 同档分子互换只在"有禁忌 / 副作用"时才合理（§二 第 ② 类）。

3. **"两种升级触发机制"并存**（Q2357 教训）
   - **机制 A**：器官功能否决一线药（肾差 → 不能用 Li → 换 VPA）= 同阶梯换药
   - **机制 B**：疗效不足升下阶梯（单药轮过都不够 → 联合 → ECT）
   - 同一题 stem 可能两机制并存 — 逐个识别，不要混为一谈。

4. **"减药失败" = 升级**（Q12152 教训）
   - PD psychosis 第 4 步减 PD 药失败 / 运动症状反弹 → 第 5 步上 quetiapine。
   - 反直觉：升级不一定是"加药档次"，可能是从"减"转向"加另一类药"。

5. **"反射性选最经典药"**（Q2357 反复错根因）
   - 双相 → 反射性选 lithium 不看 Cr / family hx → 错。
   - 反射钩：**药物选择题先扫 lab + family hx + 既往史的否决项，再看选项**。

6. **"加量 vs 换 vs 升"3 种调药动作混淆**
   - SSRI 6 周部分应答 → 先**加量**到最大耐受（不是直接换）。
   - SSRI 加量到最大仍无效 → **换同类 SSRI**（不是直接升 ECT）。
   - 两种 SSRI 都无效 → **换不同类**（SNRI / Bupropion / Mirtazapine）。
   - 三个抗抑郁药都无效 + augment 也无效 → **升 ECT / rTMS / Esketamine**。

## §六 Memory Hook 汇总

> [!success] 一句话钩子集
> - **"用了一段仍恶化 → 找下一档，不是同档换"** — 升级通用钩
> - **"先到位没？再问没效还是没事？再查阶梯次序"** — 三连问决策（§四）
> - **"没效升一级，有事换一家，没满加一档"** — 三种调药动作口诀
> - **"Tourette HRT → α2 → 抗 DA，三档到顶"**（Q3375）
> - **"Bipolar 一线 4 兄弟，按 lab + 妊娠 + 代谢 否决，剩谁选谁"**（Q2357）
> - **"PD 看见鬼 → 先减药；减救不回运动 → 上 Quetiapine"**（Q12152）
> - **"哮喘升一档，生物制剂留 5 档；系统类固醇压底"**
> - **"HTN 三联仍不够 = Resistant HTN → 4th 药 spironolactone + 排继发"**
> - **"MDD 加 / 换同 / 换异 / augment / ECT — 5 档全套别跳级"**
> - **"T2DM 看合并症选第 2 药：心衰 CKD → SGLT2i；ASCVD 肥胖 → GLP-1 RA"**

## §八 复习节奏建议

> [!info] 这是元方法论 — 复习方式比内容更重要
> 阶梯类题型反复错率高，因为光记"答案"不够，要把**"识别升级题 → 查阶梯次序"**这个动作链固化成反射。

1. **首次复习**（建后 1 天）：默写 §三 7 大疾病阶梯 + §一 6 大触发短语
2. **第 3 天**：默写 §四 决策树（先加量？再换 vs 升？再查次序？）+ §二 三种调药动作对比
3. **第 7 天**：默写 §五 6 大陷阱 + 重做 Q3375 / Q2357 / Q12152 验证反射
4. **第 14 天**：把任何新增的"升级类"错题入库时，**对照本 SOP 找对应疾病阶梯**；如果是新疾病（如 GERD / RA）→ 把新阶梯加入 §三
5. **每周扫一次** §三 阶梯表（共 7 个，每个 ≤30 秒，全部 < 5 分钟）— 保持反射
6. **考前 1 周**：复习 §六 Memory Hook 一句话集 + 把"升级题反射"加入考试做题 SOP（看到时间短语 / despite / inadequate → 自动激活）

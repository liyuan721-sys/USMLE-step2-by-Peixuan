## 1. 病理生理 — 4 个解剖缺陷

**胚胎学根源**：infundibular septum 偏离正常位置 → 同时造成 4 个缺陷

```
        Infundibular septum 偏移
                  ↓
   ┌──────────────┼──────────────┐
   ↓              ↓              ↓
1. RVOT 阻塞   2. VSD (大)   3. Overriding aorta
(肺动脉狭窄)   (室间隔缺损)   (主动脉骑跨)
   ↓
4. RVH (concentric)
   (右室向心性肥厚 — 继发于 RVOT 阻塞)
```

**PROVe** 助记：
- **P**ulmonary stenosis (RVOT obstruction)
- **R**VH
- **O**verriding aorta
- **V**SD

> [!tip] 关键认知
> RVH 是**继发**的（不是先天解剖异常），是 RVOT 阻塞后右室代偿肥厚。
> ToF 严重度 = **RVOT 阻塞严重度**（决定一切临床表现）

## 2. 血流动力学 — 为什么发绀？

```
        RVOT 阻塞 (高阻力)
             vs
        SVR (系统阻力)
             ↓
        谁阻力小血就往谁那走
             ↓
   通过 VSD：右室血直接 → 主动脉（去氧血进入体循环）
             ↓
        中央性紫绀
```

**核心规律**：
> R→L 分流量 = f(RVOT 阻力 / SVR 比值)
> - **SVR↓** → R→L↑ → 紫绀加重
> - **SVR↑** → R→L↓ → 紫绀缓解

## 3. 临床表现 — 按严重度分层

| 严重度 | 出现时间 | 临床表现 | 干预 |
|---|---|---|---|
| **严重** | 新生儿期（PDA 关闭后）| Profound cyanosis | **PGE1** 维持 PDA → 早期手术 |
| **中度** ⭐ | 婴儿期 3-6 月 | **Tet spells**（哭/喂奶时发绀）| 择期手术 |
| 轻度 | 儿童期 / 成年 | 心衰 / "pink Tet"（无紫绀但有杂音）| 择期手术 |

## 4. ⭐ 体征 ↔ 机制对照（卡点纠正核心）

### 4.1 完整对照表

| 体征 | 机制 | 反直觉 / 鉴别要点 |
|---|---|---|
| 3-month-old 才发绀 | 中度 RVOT 阻塞（PDA 关闭后逐渐显现）| 严重 ToF 新生儿就紫绀，轻度可成年才发现 |
| Bluish during **crying/feeding** | **SVR↓ → R→L 分流↑** → Tet spell | ⭐ 最高频考点：活动让发绀**加重** |
| SpO2 80% | 动脉混合去氧血 | 中央紫绀客观指标 |
| **Lips/tongue cyanosis** | **中央紫绀**（动脉血脱氧）| ≠ 周围紫绀（手脚冷、Raynaud、休克）|
| No digital clubbing | 病程 <6 月，慢性低氧未形成 | 杵状指需数月-数年 |
| **Crescendo-decrescendo SEM @ LUSB** | **PS 杂音**（RVOT 狭窄的湍流）| ❗ **不是 VSD 杂音** |
| **Single S2** | **P2 soft/absent**（肺血流↓ + 肺动脉发育不良 → 关闭力弱）| ≠ ASD 的 fixed split |
| 四肢 BP 均匀 | 主动脉骑跨但无缩窄 | 排除 coarctation（上肢>下肢）|
| HR 155, RR 56 | 低氧代偿性 tachycardia + tachypnea | 非特异 |
| 胸片 **boot-shaped heart** | RVH + 肺动脉段凹陷 | 经典影像 buzzword |

### 4.2 🤔 卡点（做对题但没真懂）

#### 卡点 1：杂音到底是 VSD 还是 RVOT 的？
- **答案：杂音 = RVOT 阻塞（PS 样），不是 VSD**
- ToF 的 VSD 是**大缺损 → 左右压力均衡 → 无压力梯度 → 无湍流 → 无杂音**
- 反推规律：**Loud VSD murmur = small VSD**（"响 = 小洞 = 好消息"）

#### 卡点 2：Single S2 — P2 去哪了？
- **答案：P2 soft/absent**，因为肺血流↓ → 关闭肺动脉瓣的力量↓
- 严重时 P2 完全消失，听起来只有一个 S2

#### 卡点 3：哭/喂奶为什么会让发绀更严重？
- **答案：SVR↓ → R→L 分流↑** → Tet spell
- 反直觉：活动让肺血更**少**（血走阻力小的主动脉）
- Squatting 是反向：↑SVR → 血逼回肺循环 → 发绀缓解
- 大孩子会自己学会蹲下（ToF 经典 buzzword）

#### 卡点 4：紫绀部位为什么是唇舌不是手脚？
- **答案：中央紫绀**（动脉血脱氧）vs 周围紫绀（局部血流差）
- 唇/舌血供丰富，直接反映动脉血氧饱和度

## 5. 易混鉴别表

### 5.1 ToF 的杂音 vs 单纯 VSD 杂音 ⭐

| | ToF 杂音 | 单纯小 VSD 杂音 |
|---|---|---|
| **来源** | RVOT 狭窄（PS 样）| VSD 本身的湍流 |
| **位置** | **LUSB**（左上胸骨缘）| **LLSB**（左下胸骨缘）|
| **形态** | **Crescendo-decrescendo (ejection)** | **Holosystolic** |
| **响度规律** | 杂音**越小** = RVOT 越严重（流不动了）⚠ | 杂音**越大** = 洞越小（压力梯度大）|

> [!warning] 反向逻辑
> - **ToF**：杂音消失 = 病情恶化（Tet spell 发作，几乎无肺血流）
> - **VSD**：杂音消失 = 病情恶化（Eisenmenger 化，压力均衡）
>
> 两种"没声音"都是坏消息，但机制完全不同。

### 5.2 S2 分裂模式鉴别 ⭐

| S2 模式 | 疾病 | 机制 |
|---|---|---|
| **Fixed split S2** | **ASD** | 右心容量超负荷 → P2 始终延后 |
| **Single S2 (soft P2)** | **ToF**, severe PS | 肺血流↓ → P2 弱到听不见 |
| **Single S2 (loud)** | **TGA**, Eisenmenger, severe pulmonary HTN | A2 前位响亮 / P2 高压重叠 |
| **Wide split (随呼吸变)** | RBBB, 轻度 PS | 右室排空延迟 |
| **Paradoxical split** | LBBB, AS, HCM | 左室排空延迟 → P2 反在 A2 前 |

### 5.3 紫绀型先天性心脏病鉴别（5T 助记）

| 疾病 | 出现时间 | 杂音 | S2 | 关键鉴别 |
|---|---|---|---|---|
| **Truncus arteriosus** | 新生儿 | 多变 | Single | 一根大动脉发出 |
| **TGA** (Transposition) | **生后 24h** | 通常无 | **Single, loud** | 出生立刻严重紫绀；egg-on-string |
| **Tricuspid atresia** | 新生儿 | VSD 杂音 | Single | 左轴偏离 + LVH |
| **ToF** ⭐ | **3-6 月**（中度）| **SEM @ LUSB** | **Single, soft P2** | Tet spell；boot-shaped |
| **TAPVR** | 新生儿 | Fixed split | Wide fixed | snowman heart |

### 5.4 中央 vs 周围紫绀

| | 中央紫绀 | 周围紫绀 |
|---|---|---|
| 部位 | **唇、舌、口腔黏膜** | 手指、脚趾、四肢末端 |
| SpO2 | ↓ | 正常 |
| 机制 | 动脉血脱氧（R→L 分流 / 肺病）| 局部血流↓（低温、Raynaud、休克）|
| ToF | ✅ | — |

## 6. Tet Spell 急救 ⭐⭐⭐

```
        Tet spell（哭闹后发绀加重，杂音变小）
                ↓
        目标：↑SVR + ↓RVOT 痉挛 + ↑前负荷 = 物理撑开痉挛的 RVOT 漏斗 + 把血"逼"过肺循环
                ↓
   1. Knee-to-chest position ← 一线（↑SVR）
   2. O2
   3. Morphine ← 镇静 + 降低儿茶酚胺驱动的 RVOT 痉挛
   4. IV fluids ← ↑前负荷
   5. Phenylephrine (α 激动) ← ↑SVR
   6. β-blocker (propranolol) ← 缓解 RVOT 漏斗痉挛
```

> [!danger] 禁忌
> - ❌ **β-agonist** (isoproterenol)：↑心肌收缩 → ↑RVOT 痉挛
> - ❌ **扩血管药** (nitrates, ACEi)：↓SVR → R→L 分流↑
> - ❌ **利尿剂**：↓前负荷 → 右心更难推血

#### 三药方向对照 — Tet spell 药理铁律
#难记

| 药物 | 类别 | 心血管效应 | Tet spell 中的地位 |
|---|---|---|---|
| **Isoproterenol** | β1 + β2 agonist | **加油门**（↑ 心肌收缩 → ↑ RVOT 漏斗痉挛）| ❌ **禁忌** |
| **Propranolol** | β-blocker（non-selective）| **刹车**（缓解 RVOT 漏斗痉挛）| ✅ **急救用** ⭐ |
| **Phenylephrine** | α1 agonist | **↑ SVR**（外周血管收缩 → 逼血回肺循环）| ✅ **急救用** ⭐ |

> [!warning] 方向反向陷阱（Q17 错点）
> 题目最爱反着考"哪个不能用"。**β-agonist 与 β-blocker 在 Tet spell 中方向完全相反**：
> - β-agonist（iso）= 加油门 → 让 RVOT 漏斗更紧 → 紫绀加重
> - β-blocker（propranolol）= 刹车 → 让 RVOT 漏斗放松 → 紫绀缓解
>
> 同样道理：**α1-agonist（phenylephrine）↑ SVR**（救命）vs **扩血管药**（nitrates / ACEi）↓ SVR（禁忌）— 也是方向反向。

> [!warning] 高频默写错点（2026-05-20 浩斯宾 self-test 暴露）
> - **α1 激动 vs 拮抗方向再次答反**：Tet spell 用 **Phenylephrine = α1 激动剂**（↑SVR → 逼血回肺循环）；误答“α1 拮抗剂” → ↓SVR → R→L 分流↑ → spell 加重。
> - 同时漏答 **膝胸位（knee-to-chest）+ 吸氧** 这 2 个一线措施。
> - 链回：[[mistakes/uworld-mistakes#^Q2429]]（已升 🔴）· [[复盘/2026-05-20_浩斯宾判分_v1]]

## 8. 治疗

| 阶段 | 治疗 |
|---|---|
| 急性 Tet spell | 见 §6 |
| 严重新生儿 ToF | **PGE1** 维持 PDA → 早期手术 |
| 中度 ToF | 择期完全修复手术（3-6 月）|
| 完全修复 | 关 VSD + 解除 RVOT 阻塞 |
| 长期随访 | 注意残余肺动脉瓣反流（最常见远期并发症）|

## 9. Memory Hooks

> [!success] 核心口诀
> 1. **PROVe** = 4 缺陷（PS, RVH, Overriding aorta, VSD）
> 2. **"杂音 = PS，不是 VSD"** — ToF 的 VSD 太大没杂音
> 3. **"响 = 小洞 = 好消息"**（VSD 杂音响度反比）
> 4. **"哭闹 = 紫绀加重"** — SVR↓ → R→L↑（反直觉）
> 5. **"蹲下救命"** — squatting ↑SVR 缓解 spell
> 6. **"Boot-shaped + Single S2 + LUSB SEM" = ToF 三联**

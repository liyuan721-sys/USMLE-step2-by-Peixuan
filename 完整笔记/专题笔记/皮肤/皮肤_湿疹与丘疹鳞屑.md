# 湿疹与丘疹鳞屑病 Eczema & Papulosquamous 详解

> [!info] 一句话定位
> 红斑鳞屑/丘疹一大类，靠**部位 + 鳞屑性状 + 病史**切分：屈侧痒（特应性）/ 接触区（接触性）/ 硬币斑（钱币状）/ 油腻黄痂鼻唇沟（脂溢性）/ 紫色多角形 + 蕾丝纹（扁平苔藓）/ 母斑圣诞树（玫瑰糠疹）/ 鱼鳞样干燥（鱼鳞病）。机制 + 治疗放本篇；银屑病单列 [[完整笔记/专题笔记/皮肤/皮肤_银屑病]]。

> [!note]- 缩写对照（全称 中英）
> - **AD** = atopic dermatitis 特应性皮炎
> - **ACD / ICD** = allergic / irritant contact dermatitis 变应性 / 刺激性接触性皮炎
> - **NE** = nummular eczema 钱币状湿疹
> - **DE** = dyshidrotic eczema 汗疱疹 / 出汗不良性湿疹（pompholyx）
> - **SD** = seborrheic dermatitis 脂溢性皮炎
> - **LSC** = lichen simplex chronicus 慢性单纯性苔藓 / 神经性皮炎
> - **CVI** = chronic venous insufficiency 慢性静脉功能不全
> - **LP** = lichen planus 扁平苔藓
> - **PR** = pityriasis rosea 玫瑰糠疹
> - **FLG** = filaggrin 丝聚蛋白
> - **HCV** = hepatitis C virus 丙型肝炎病毒
> - **HHV-6/7** = human herpesvirus 6/7 人类疱疹病毒 6/7 型

## 一、湿疹谱 Eczematous

### 特应性皮炎 Atopic dermatitis（AD）

- **机制**：**FLG 丝聚蛋白基因缺陷 → 皮肤屏障受损** + Th2 炎症；个人/家族特应史（哮喘、过敏性鼻炎）。
- **临床**：**剧痒**；**婴儿面颊/伸侧 → 儿童及成人转屈侧 (flexural)**；慢性苔藓化；伴掌纹增多 (hyperlinear palms)。
- **治疗**：润肤剂 (emollient) 为基础 + 外用糖皮质激素；面部/皱褶用钙调磷酸酶抑制剂 (tacrolimus 他克莫司)；重度难治 → 系统（dupilumab 度普利尤单抗 等）/ 短期口服激素。

![[{135F9D5D-D54F-40BA-BECF-D4FDEFC639BD}.png|360]]
*特应性皮炎 (atopic dermatitis, AD)：**屈侧 (flexural) 瘙痒性红斑/苔藓化**，多**儿童起病**、常伴其他特应病（过敏性鼻炎、哮喘）。*

![[{7E10091F-2472-426B-9FA9-929704FFA6C9}.png|360]]
*特应性皮炎·慢性期（另例）：**苔藓化 (lichenified)、鳞屑性斑块**，成人多见于**屈侧 (flexural surfaces)**——慢性反复搔抓的结局；急性期则为剧痒红斑性丘疹/水疱。与鱼鳞病的弥漫干燥鱼鳞样鳞屑不同。*

![[{F943470C-4EAA-4B3B-89FD-5C91FB9C2790}.png|360]]
*特应性皮炎·慢性苔藓化（另例）：屈侧增厚、皮纹加深的**苔藓样斑块 (lichenified plaques)** + 抓痕——重度难治者方短期口服激素，并非鱼鳞病的处理。*

### 接触性皮炎 Contact dermatitis

| | **ACD 变应性** | **ICD 刺激性** |
|---|---|---|
| 机制 | **IV 型迟发超敏（致敏后 48–72h）** | 直接化学损伤，**无需致敏** |
| 形态 | 可**线状**（毒葛 poison ivy）、可超出接触区 | 局限接触区、皲裂脱屑 |
| 处理 | 避过敏原 + 外用激素（广泛口服激素）| 避刺激 + 润肤 + 外用激素 |

![[{478D84FD-FC37-44E2-B821-D50DD782091A}.png|360]]
*变应性接触性皮炎 (allergic contact dermatitis, ACD)：接触区**红斑 + 水疱 (vesicle) / 大疱 (bulla) + 显著瘙痒**——IV 型迟发超敏（毒葛 poison ivy、镍 nickel）。*

![[{071A2102-C9DA-4FD5-B6CE-203190BD8D47}.png|360]]
*刺激性接触性皮炎 (irritant contact dermatitis, ICD)：暴露部位（多为**手**）红斑 + 慢性**苔藓化 (lichenification)**，灼/刺痛感——非免疫性化学损伤（溶剂、清洁剂）。*

### 钱币状湿疹 Nummular eczema（NE）

- **机制**：病因未明 (idiopathic)，属**海绵水肿性 (spongiotic) 湿疹谱**——核心是**皮肤屏障障碍**：**干燥 (xerosis) → 经表皮失水 ↑ + 皮脂/皮肤脂质 ↓** 破坏屏障 → 刺激物 / **金黄色葡萄球菌 (Staphylococcus aureus, S. aureus) 定植**穿透 → 激活 **Th2 炎症 (IL-4/IL-13/IL-31)**，IL-31 介导剧痒 → **痒-抓循环 (itch–scratch cycle)** 自我放大。组织学 = **表皮海绵水肿 (spongiosis)**（与所有湿疹共有）。**非** I 型超敏 / 非单一过敏原驱动。
- **临床**：四肢**硬币状 (coin-shaped) 瘙痒、鳞屑、皲裂**斑块，可渗液，常伴皮肤干燥；中老年多。
- **治疗**：**外用糖皮质激素**（betamethasone 倍他米松等）+ 润肤 + 少用刺激性肥皂。
- **vs 体癣**：NE **实心斑、无中央清亮、KOH 阴性**；体癣环形 + 中央清 + KOH 阳性。

This patient has a pruritic, scaly, fissured plaque with intermittent exudate consistent with **nummular eczema (NE)**.  NE is an idiopathic inflammatory disorder that most commonly affects the skin of the extremities.  The incidence is highest in middle-aged and older individuals.  The cause is uncertain, but frequent occurrence in the setting of chronically dry skin suggests decreased production of skin lipids.  The name refers to the characteristic round lesions that would fit under a large coin (_numm-_ is a Latin prefix referring to "coin").

### 汗疱疹 / 出汗不良性湿疹 Dyshidrotic eczema（DE，pompholyx / 急性掌跖湿疹）

- **临床**：**反复、急性发作**；**手足（掌、跖、手指侧缘）深在性 (deep-seated) 瘙痒小水疱/大疱**——典型"**西米样 (tapioca-like)** 小水疱"；并发症：脱屑 (desquamation)、慢性皮炎、继发感染。
- **病因**：多因素、机制未明——常与**刺激性/变应性接触性皮炎 (ICD/ACD)、紫外线、对远处变应原或真菌的免疫反应**相关（**非**单纯"出汗"所致，旧名误导）。
- **诊断**：**临床即可**；必要时活检 = **表皮内海绵水肿 (intraepidermal spongiosis) + 淋巴细胞浸润 (lymphocytic infiltrate)**。
- **治疗**：**外用润肤剂 (emollient)** + 避刺激/防冷湿；需要时**高/超高效价外用糖皮质激素**（betamethasone dipropionate 倍他米松双丙酸酯）。

> [!tip] 一句话定位
> **手指侧缘/掌跖**深在性**西米样小水疱 + 剧痒 + 反复发作** = 汗疱疹。与水疱型接触性皮炎重叠（常互为诱因），但汗疱疹**对称、好发指侧缘、无明确单一接触物**。

![[{6F2E592D-FE5B-4682-8525-C34D92B4EF33}.png|320]]
*汗疱疹 (dyshidrotic eczema)：手指/掌部**深在性 (deep-seated) 小水疱**密集——典型"西米样"外观，剧痒、反复发作。*

### 脂溢性皮炎 Seborrheic dermatitis（SD）

- **机制**：皮脂腺丰富区 + Malassezia 马拉色菌相关。
- **临床**：**头皮屑、眉、鼻唇沟 (nasolabial fold)、耳、胸**的**油腻黄色鳞屑**红斑；婴儿=摇篮帽 (cradle cap)；**泛发/顽固 → 查 HIV / 帕金森病 (Parkinson disease)**。
- **治疗**：**外用抗真菌**（ketoconazole 酮康唑、selenium sulfide 二硫化硒）± 低效价激素。
- **关键钥匙**：**累及鼻唇沟（不避开）→ SD 或酒渣鼻 (rosacea) 皆可**；**避开鼻唇沟 → SLE 颊部红斑**。⚠️ **SD ≠ 酒渣鼻**（两者只是都不避开鼻唇沟、借此与 SLE 鉴别）：**有油腻黄鳞屑 = SD**；**红斑 + 潮红 + 毛细血管扩张 + 丘疹脓疱但无鳞屑/无粉刺 = 酒渣鼻**。

![[{8E0AE226-26C3-41D9-90A9-EA20F1F47A21}.png|360]]
*脂溢性皮炎 (seborrheic dermatitis, SD)：**中面部（鼻翼/鼻唇沟）油腻黄色鳞屑**红斑——累及鼻唇沟是与 SLE 颊部红斑鉴别的关键（SLE 避开鼻唇沟）。*

> [!note]- 婴儿型 = 摇篮帽 Cradle cap（SD 的婴儿亚型）
> - **机制**：**母体雄激素 (maternal androgen)** 残留刺激皮脂腺 + **马拉色菌 (Malassezia)**；新生儿~3 月龄常见。
> - **临床**：**头顶油腻黄痂/鳞屑**，**不痒、婴儿状态好**（进食生长正常）——自限，多 6–12 月自愈。
> - **治疗（阶梯）**：① 一线 **保守软化清除**——婴儿油/矿物油软化 + 软刷 + 温和洗发 + reassurance（安抚家长）；② 顽固/泛发 → **酮康唑 ketoconazole 2%** 外用，或短程**氢化可的松 hydrocortisone 1%**。
> - **vs 特应性皮炎 (AD)**：cradle cap **不痒、局限头皮、婴儿不烦躁**；AD **剧痒、累及面颊/伸侧、患儿烦躁**。
> - **陷阱**：**泛发顽固 + 生长迟缓/腹泻/反复感染** → 警惕 **朗格汉斯细胞组织细胞增生症 (Langerhans cell histiocytosis, LCH)** / **锌缺乏 (zinc deficiency)** / 免疫缺陷，非普通摇篮帽。

> [!question]- SD vs 酒渣鼻 Rosacea 鉴别（都不避开鼻唇沟，但是两个病）
>
> | | **酒渣鼻 Rosacea** | **脂溢性皮炎 SD** |
> |---|---|---|
> | 本质 | **血管运动/炎症性**疾病 | **皮脂腺 + 马拉色菌 (Malassezia)** 相关湿疹谱 |
> | 形态 | 持续红斑 + **毛细血管扩张 (telangiectasia)** + 丘疹脓疱，**无鳞屑/无粉刺 (comedone)** | **油腻黄色鳞屑**红斑（有屑是关键）|
> | 诱因/特征 | **阵发性潮红 (flushing)**，遇热/酒/辛辣/日晒；可有 **眼型 (ocular rosacea)** | 头皮屑样油腻鳞屑；泛发顽固查 HIV/帕金森 |
> | 部位 | 面**中部凸起**（鼻、颊、额、下巴）| 头皮、眉、鼻唇沟、耳后、胸 |
> | 治疗 | 避诱因 + **甲硝唑 metronidazole 外用**；丘疹脓疱多 → 口服 **多西环素 doxycycline**；潮红型 **溴莫尼定 brimonidine**；鼻赘 (rhinophyma) 激光 | **外用抗真菌**（酮康唑 ketoconazole、二硫化硒 selenium sulfide）± 低效价激素 |
>
> **一句话**：有油腻黄鳞屑 = SD；红斑 + 潮红 + 毛细血管扩张 + 丘疹脓疱但**无鳞屑无粉刺** = 酒渣鼻。两者**都不避开鼻唇沟** → 借此与 **SLE 颊部红斑（避开鼻唇沟）** 鉴别。

### 淤积性皮炎 Stasis dermatitis（瘀积性湿疹）⭐

- **机制**：**慢性静脉功能不全 (chronic venous insufficiency, CVI)** → 静脉高压 + 含铁血黄素 (hemosiderin) 沉积 → 慢性炎症。
- **临床**：**双侧小腿/内踝**红斑、鳞屑、瘙痒 + **棕褐色含铁血黄素色素沉着** + 凹陷性水肿 ± 静脉曲张 (varicose veins) / 内踝静脉性溃疡。
- **治疗**：**加压治疗 (compression therapy)** 治本（针对 CVI）+ 润肤 + 急性渗出期短程外用糖皮质激素。

> [!danger] 最大陷阱：双侧腿红 = 淤积性皮炎，不是蜂窝织炎
> **Cellulitis 蜂窝织炎几乎总是单侧 + 急性发热 + 触痛**；**双侧对称小腿红斑 + 慢性 + 色素沉着 = stasis dermatitis 淤积性皮炎**。把双侧淤积性皮炎误判蜂窝织炎滥用抗生素是经典考点。机制/外周血管详见 [[完整笔记/专题笔记/心内/心内_外周血管病]]（CVI 段）。

### 乏脂性湿疹 Asteatotic eczema（裂纹性湿疹 / eczema craquelé）

- **临床**：**老年人 + 冬季 + 小腿**为主——**干燥 + "碎瓷样/干涸河床样 (cracked porcelain / dried-riverbed)" 浅红裂纹鳞屑** + 痒；过度热水/肥皂清洗、低湿度诱发，由皮肤干燥症 (xerosis) 进展而来。
- **治疗**：**润肤剂 (emollient) 为主** + 减少热水烫洗/肥皂 + 外用低-中效糖皮质激素。

### 慢性单纯性苔藓 Lichen simplex chronicus（LSC，神经性皮炎）

- **机制**：**瘙痒-搔抓循环 (itch-scratch cycle)** → 反复摩擦 → 局部**苔藓化 (lichenification)**；常继发于特应性皮炎 AD 或慢性局部刺激。
- **临床**：**境界清的增厚斑块 + 皮纹加深**，好发**够得着的部位**（颈后/枕、踝、外阴/阴囊、小腿伸侧）。
- **治疗**：**打断搔抓**——高效价外用糖皮质激素 ± 封包、夜间抗组胺/止痒、行为干预。
- **vs**：银屑病 = 银白厚鳞屑 + Auspitz 征；扁平苔藓 = 紫色多角形 + Wickham 蕾丝纹。

> [!info] 两个相关概念（不是独立亚型，知道即可）
> - **疱疹性湿疹 Eczema herpeticum**：AD 基础上 HSV 播散 → **凿孔样糜烂 (punched-out erosions) + 出血性结痂 + 发热**，**立即系统 acyclovir 阿昔洛韦**（详见 [[完整笔记/专题笔记/皮肤/皮肤_水疱大疱病鉴别]] §四 / [[完整笔记/专题笔记/皮肤/皮肤_浅表感染]] 病毒段）。
> - **自身敏感性湿疹 / Id 反应 (autoeczematization)**：一处剧烈皮炎（如足癣、淤积性皮炎）后，**远隔部位**（常对称分布于手/躯干）爆发**瘙痒性丘疱疹**——是对原发灶抗原的免疫反应；**治原发灶**即可消退。

## 二、丘疹鳞屑 Papulosquamous

### 扁平苔藓 Lichen planus（LP）

- **机制**：T 细胞介导；与 **HCV 丙型肝炎**、药物（ACEi、噻嗪类利尿剂）关联。
- **临床**：**5P**（Pruritic 痒 / Purple 紫 / Polygonal 多角形 / Papules 丘疹 / Plaques 斑块）+ **Wickham striae 白色蕾丝纹**；屈侧腕踝、口腔颊黏膜、生殖器、甲；**Köbner 现象**。
- **处理**：自限（约 2 年）；高效价外用激素缓解；**有吸毒/肝酶异常 → 筛 HCV**。

![[{32F76B5C-6897-436F-9917-98B139711380}.png|360]]
*扁平苔藓 (lichen planus, LP)：腕屈侧**瘙痒性紫色 (violaceous)、多角形扁平丘疹 (polygonal papules)**——5P 经典形态与好发部位。*

![[{39C40A56-A89F-4259-A977-0FD7CB002F9A}.png|360]]
*扁平苔藓·Wickham 纹：紫色斑块表面**网状白色蕾丝样细纹 (lacy white markings, Wickham striae)**——LP 特征性表面征。*

### 玫瑰糠疹 Pityriasis rosea（PR）

- **机制**：HHV-6/7 相关，自限（6–8 周）。
- **临床**：先**母斑 (herald patch)** → 1–2 周后躯干沿皮纹**圣诞树样**小卵圆鳞屑斑，**避开手掌脚掌**。
- **关键鉴别**：**手掌脚掌受累 → 立刻改查二期梅毒 (secondary syphilis)**（RPR/VDRL）。

![[Pasted image 20260629220730.png|360]]
*玫瑰糠疹 (pityriasis rosea, PR)：最上方**最大、最圆、边界清的母斑 (herald patch)**（早 1–2 周出现，中央略消退呈环形）+ 随后散在较小的卵圆形鳞屑斑（子斑，沿皮纹分布 → 躯干"圣诞树样"）。*

![[{45225BDE-6AD8-491B-B8DA-541EFF0DB5CD}.png|360]]
*玫瑰糠疹示意图（另例）：左=初发**单个圆形母斑 (herald patch)**；右=1 周后躯干**卵圆斑沿皮纹斜行排列 = "圣诞树样 (Christmas tree)"**。*

![[{4D272B75-C497-4880-8829-76E5E6F1BDE6}.png|320]]
*玫瑰糠疹临床照（另例）：躯干多发小卵圆形鳞屑斑，对称散布——**手掌脚掌干净**（受累须改查二期梅毒）。*

### 寻常型鱼鳞病 Ichthyosis vulgaris

- **机制**：**FLG 丝聚蛋白基因功能缺失** → 屏障受损 + 脱屑障碍（与 AD 共背景）。
- **临床**：婴儿/儿童起；**干燥、鱼鳞样鳞屑 (fish-like scales)**、伸侧为主、**避开屈侧/皱褶**；冬季加重；常伴掌纹增多 + 特应性；家族史。
- **治疗**：**外用角质溶解剂 (keratolytics)** —— **α-羟基酸 (alpha hydroxy acid / 乳酸)、尿素 (urea)、水杨酸 (salicylic acid)** + 频繁润肤 + 长时间泡浴去鳞。
- **陷阱**：**不**用口服维 A 酸（治结节囊肿型痤疮）、calcipotriene 卡泊三醇（治银屑病）、口服激素（治重度 AD）。

![[{24E3A68E-E68A-4FDC-8445-CD2FC8633665}.png|360]]
*寻常型鱼鳞病 (ichthyosis vulgaris)：躯干/伸侧**弥漫干燥、鱼鳞样多角形鳞屑 (fish-like scales)**——遗传性 FLG 丝聚蛋白基因缺陷致脱屑障碍，避开屈侧与面部、冬季加重。（Q22174 题图）*

![[{A0BADE49-0973-4459-88E4-440791063688}.png|360]]
*寻常型鱼鳞病·掌纹增多 (palmar hyperlinearity)：手掌**掌纹加深/增多**——与鱼鳞病、特应性皮炎共享的 FLG 缺陷体征，是支持诊断的伴随线索。（Q22174 题图）*

## 四、易混陷阱（普适）

- **累及鼻唇沟 → SD/酒渣鼻；避开 → SLE**。
- **双侧小腿红 + 慢性 + 色素沉着 = 淤积性皮炎**；单侧急性发热触痛才是蜂窝织炎（别给双侧淤积性皮炎滥用抗生素）。
- **碎瓷样裂纹干燥（老年/冬季小腿）= 乏脂性湿疹**；**境界清增厚 + 皮纹加深苔藓化 = 慢性单纯性苔藓**（搔抓所致）。
- **硬币状无中央清 + KOH− = 钱币状湿疹**；环形中央清 + KOH+ = 体癣。
- **紫色多角形 + 蕾丝纹 = 扁平苔藓**（顺手查 HCV）；不是疱疹样皮炎。
- **鱼鳞样 + 家族 + 冬重 = 鱼鳞病 → 外用角质溶解 + 润肤**，不口服维 A 酸。
- **玫瑰糠疹手脚干净**；手脚受累改查二期梅毒。

## 五、自测（答完再判分）

> [!question]- Q1. 48 岁男性鼻唇沟 + 眉 + 头皮屑、油腻细黄鳞屑红斑。诊断 + 与花斑癣/SLE 怎么分？
> **脂溢性皮炎**；花斑癣在**躯干**、晒不黑；**累及鼻唇沟**排除 SLE（SLE 避开鼻唇沟）。

> [!question]- Q2. 30 岁男性多年弥漫鱼鳞样干燥、冬重、母与妹同病。最佳处理？
> **外用角质溶解剂（尿素/乳酸/水杨酸）+ 润肤**（寻常型鱼鳞病，FLG 缺陷）。

> [!question]- Q3. 49 岁男性 1 年硬币状瘙痒斑、偶清亮渗液、换皂润肤无效。诊断 + 与体癣怎么分？
> **钱币状湿疹**（外用激素）；体癣有**环形中央清亮 + KOH+**，NE 没有。

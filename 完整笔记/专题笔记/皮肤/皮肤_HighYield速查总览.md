# 皮肤科 Dermatology High-Yield 速查

> [!info] 用途与来源
> 考前快速过皮肤科核心反射与 buzzword。知识源自 [[完整笔记/Peixuan分科笔记/derm]]（只读分科笔记），本篇为高频提炼速查版。配合 [[完整笔记/_衍生_鉴别诊断速查]]、[[完整笔记/_衍生_高频陷阱]] 使用。

> [!note]- 缩写对照（全称 中英）
> - **SJS** = Stevens-Johnson syndrome 史蒂文斯-约翰逊综合征
> - **TEN** = toxic epidermal necrolysis 中毒性表皮坏死松解症
> - **SSSS** = staphylococcal scalded skin syndrome 葡萄球菌性烫伤样皮肤综合征
> - **DRESS** = drug reaction with eosinophilia and systemic symptoms 伴嗜酸粒细胞增多和系统症状的药物反应
> - **EM** = erythema multiforme 多形红斑
> - **BCC** = basal cell carcinoma 基底细胞癌
> - **SCC** = squamous cell carcinoma 鳞状细胞癌
> - **AK** = actinic keratosis 光化性角化病
> - **BSA** = body surface area 体表面积
> - **ACD / ICD** = allergic / irritant contact dermatitis 变应性 / 刺激性接触性皮炎
> - **PCT** = porphyria cutanea tarda 迟发性皮肤卟啉病
> - **AIP** = acute intermittent porphyria 急性间歇性卟啉病
> - **KOH** = potassium hydroxide 氢氧化钾（真菌镜检）
> - **AN** = acanthosis nigricans 黑棘皮病
> - **DH** = dermatitis herpetiformis 疱疹样皮炎
> - **HHV** = human herpesvirus 人类疱疹病毒
> - **HSV / VZV** = herpes simplex / varicella-zoster virus 单纯疱疹 / 水痘-带状疱疹病毒
> - **HIV** = human immunodeficiency virus 人类免疫缺陷病毒
> - **HepC** = hepatitis C 丙型肝炎
> - **I&D** = incision and drainage 切开引流
> - **GAS** = group A Streptococcus A 组链球菌
> - **PSGN** = post-streptococcal glomerulonephritis 链球菌感染后肾小球肾炎
> - **DIP** = distal interphalangeal joint 远端指间关节
> - **TSH** = thyroid-stimulating hormone 促甲状腺激素
> - **5-FU** = 5-fluorouracil 5-氟尿嘧啶

## 〇、核心规则（先记这几条）

> [!tip] Core Rules
> - **真菌检测**：best initial = **KOH prep 氢氧化钾镜检**；most accurate = **fungal culture 真菌培养**。
> - **真菌通则**：浅表 → topical azole 外用唑类；**毛发 (capitis) 和指甲 (unguium) 必须口服**（外用渗不进毛囊/甲）。
> - **感染严重度阶梯**：局限→topical / 广泛→oral / 不稳定→IV / 组织坏死→**手术**。
> - **活检指征**：变化的痣 / 可疑色素灶 / 慢性不愈 / 长在瘢痕烧伤创面 / 突发爆发性。
> - **反复念珠菌病 (candidiasis) → 查 HIV / DM**。

## 一、Buzzword → 诊断反射（最快速查）

| Buzzword | 诊断 |
|---|---|
| Spaghetti and meatballs（镜下）/ 晒不黑减色斑 | **Tinea versicolor 花斑癣** (Malassezia 马拉色菌) |
| Honey-crusted 蜜黄色结痂 | **Impetigo 脓疱疮** |
| Pearly rolled border + telangiectasia 珍珠样卷边+毛细血管扩张 | **BCC 基底细胞癌** |
| Keratin pearls 角珠（组织学） | **SCC 鳞状细胞癌** |
| Herald patch + Christmas tree 母斑+圣诞树（避开手脚心） | **Pityriasis rosea 玫瑰糠疹** |
| 皮疹**累及手掌脚掌** | **Secondary syphilis 二期梅毒**（首要鉴别） |
| Sandpaper rash + strawberry tongue 砂纸疹+草莓舌 | **Scarlet fever 猩红热** |
| Slapped cheek 掌掴颊 | **Erythema infectiosum 传染性红斑** (Parvovirus B19 细小病毒) |
| Target/iris lesion 靶形+近期 HSV | **Erythema multiforme 多形红斑** |
| Bull's eye 扩大 + 蜱叮 | **Erythema migrans 游走性红斑** (Lyme 莱姆病) |
| Wickham striae 白色蕾丝纹 + 5P | **Lichen planus 扁平苔藓** |
| 茶/酒色尿 + 日晒区无痛水疱 + HepC | **PCT 迟发性皮肤卟啉病** |
| Umbilicated 脐凹丘疹 | **Molluscum contagiosum 传染性软疣** |
| **Painless** black eschar 无痛黑焦痂 | **Anthrax 炭疽** (Bacillus anthracis) → ciprofloxacin 环丙沙星 |
| **Painful** eschar 疼痛焦痂 | **Brown recluse 棕色隐士蜘蛛咬伤** |
| Wood-grain 木纹样同心带 | **Erythema gyratum repens**（副肿瘤，先于癌诊断） |
| 老年突发天鹅绒色素斑 + 体重↓ | **Malignant acanthosis nigricans 恶性黑棘皮病** → 查 GI/GU 癌 |

![[皮肤_多形红斑靶形.png]]
*多形红斑 (erythema multiforme)：经典**三色靶形 (target/iris) 损害**——中央暗紫/水疱 + 苍白环 + 红边；多 HSV 后 1–2 周发病。*

![[皮肤_游走性红斑莱姆.png]]
*游走性红斑 (erythema migrans)：蜱叮后从中心向外扩大的**牛眼/靶心红斑** → 莱姆病 (Lyme disease, Borrelia)。*

![[皮肤_边缘性红斑风湿热.png]]
*边缘性红斑 (erythema marginatum)：一过性、粉红、**环形/匐行边界 + 中央清除**——急性风湿热 (acute rheumatic fever) 表现。*

## 二、皮肤急症 & 红旗（最高优先）

| 急症 | 关键特征 | 处理 |
|---|---|---|
| **SJS / TEN** | 发热+痛性红斑→水疱剥脱、**Nikolsky 征+**、**口/眼/生殖器黏膜受累**、面+躯干起始（spare 头皮/掌跖）；机制=角质细胞凋亡 | **立即停诱发药**（卡马西平 carbamazepine、磺胺、别嘌醇 allopurinol、抗癫痫药）+ **送烧伤科** + 创面护理 + 补液 ± IVIG |
| **DRESS** | 药疹 + **嗜酸粒细胞增多** + 器官损害，潜伏期长（2–8 周） | 停药 + 系统糖皮质激素 |
| **SSSS** | 婴幼儿；S. aureus 金黄色葡萄球菌剥脱毒素；Nikolsky+；**不累及黏膜**；颗粒层浅剥脱、愈合快不留瘢 | nafcillin 萘夫西林 + 支持 |
| **Necrotizing fasciitis 坏死性筋膜炎** | 剧痛（与体征不符）+ 捻发音 crepitus + 紫色大疱 + 低血压 | 影像见气体、**别触碰** → 急诊清创 + penicillin 青霉素 + clindamycin 克林霉素 |
| **Eczema herpeticum 疱疹性湿疹** | 特应性皮炎 atopic dermatitis 患儿 + **punched-out 凿孔样糜烂** + 出血性结痂 + 发热 | **立即系统 acyclovir 阿昔洛韦** |
| **Ecthyma gangrenosum 坏疽性臁疮** | Pseudomonas 铜绿假单胞菌菌血症 + 中性粒减少 → 出血性脓疱/坏死黑痂 + 脓毒 | 抗假单胞菌治疗 |

> [!danger] SJS/TEN vs SSSS 一句话
> 都 Nikolsky+，但 **SJS/TEN 累及黏膜 + 表皮全层坏死（深）**；**SSSS 不累及黏膜 + 仅颗粒层剥脱（浅）**。BSA 阈值：SJS <10% / overlap 10–30% / TEN >30%。

![[皮肤_EM黏膜受累.png]]
*多形红斑 (erythema multiforme) 黏膜重型：口唇/口腔糜烂 + 靶形红斑——与 SJS 谱系黏膜受累相邻，常 HSV 诱发。*

![[皮肤_疱疹性湿疹.png]]
*疱疹性湿疹 (eczema herpeticum)：特应性皮炎基础上**凿孔样糜烂 (punched-out erosions) + 出血性结痂** → 立即系统 acyclovir 阿昔洛韦。*

![[皮肤_坏疽性臁疮.png]]
*坏疽性臁疮 (ecthyma gangrenosum)：出血性脓疱迅速演变为**坏死性溃疡 + 周边红斑**——铜绿假单胞菌 (Pseudomonas) 菌血症 + 中性粒减少。*

## 三、重点鉴别对比表

### 1. 严重药物/水疱反应

| 疾病 | 核心 | 黏膜 | 处理 |
|---|---|---|---|
| **SJS/TEN** | Nikolsky+、表皮剥脱 | **常受累** | 烧伤病房、停药 |
| **SSSS** | 婴儿、剥脱毒素 | **无/罕见** | nafcillin + 支持 |
| **EM 多形红斑** | **靶形损害** | 轻/可有 | 多 HSV 诱发，支持 |
| **DRESS** | **嗜酸粒↑** + 器官损害 | 较少 | 停药 + 系统激素 |

### 2. 玫瑰糠疹 vs 二期梅毒 vs 体癣 vs 药疹

| | 母斑 | 圣诞树小斑 | 手掌脚掌 | 检查 |
|---|---|---|---|---|
| **玫瑰糠疹** | 有 | 有 | **避开** | 临床 |
| **二期梅毒** | 无 | 有 | **累及** ★ | RPR/VDRL |
| **体癣 tinea corporis** | 仅"大斑" | 极少 | 任何处 | KOH |
| **药疹** | 无 | 有 | 对称 | 药物史 |

> [!warning] 安全清单：有母斑+圣诞树+手脚干净→玫瑰糠疹；**手脚受累立刻改梅毒**；高热/草莓舌/淋巴结→猩红热或 Kawasaki 川崎病。

![[皮肤_川崎病体征.png]]
*Kawasaki 川崎病 (mucocutaneous lymph node syndrome) 黏膜皮肤体征：发热>5 天 + 双侧非渗出性结膜炎 + 草莓舌 + 多形皮疹 + 手足肿胀脱皮 + 颈淋巴结肿大（CRASH and burn）。*

### 3. 大疱性类天疱疮 vs 寻常型天疱疮

| | **Bullous pemphigoid 大疱性类天疱疮** | **Pemphigus vulgaris 寻常型天疱疮** |
|---|---|---|
| 抗原 | 抗 hemidesmosome 半桥粒（基底膜） | 抗 desmoglein 桥粒芯蛋白（细胞间） |
| 水疱 | **紧张性 tense、厚、不易破** | **松弛、易破** |
| 黏膜 | 罕见 | **口腔受累** |
| Nikolsky | **−** | **+**（acantholysis 棘层松解） |
| 痛 | 无痛 | 疼痛 |
| Tx | 糖皮质激素 | 激素，难治→cyclophosphamide 环磷酰胺/IVIG/血浆置换 |

### 4. 皮肤癌切缘 & 高危处理

| 癌 | 关键 | 处理 |
|---|---|---|
| **BCC 基底细胞癌** | 珍珠样卷边 + 毛细血管扩张，上面部 | low-risk 躯干<2cm→切除 4mm 切缘 / topical；high-risk（面颈、≥2cm、边界不清）→ **Mohs 手术** |
| **SCC 鳞状细胞癌** | 角珠、下面部/唇、溃疡结痂 | 切除 4–6mm 切缘；**Marjolin ulcer**=慢性烧伤瘢痕恶变→SCC |
| **Melanoma 黑色素瘤** | **ABCDE**；marker **S100**；**深度 (Breslow) = 首要预后** | **全层活检（full-thickness）**；切缘按厚度 1–3cm；深者+干扰素 interferon |
| **Gorlin 综合征** | AD 遗传、**PTCH 突变**、多发 BCC + 掌凹点 + 颌囊肿 + 叉状肋 | — |

> [!tip] Melanoma 排查陷阱：指甲褐色纵纹（尤其外伤史）、眼内褐色灶（"晒太阳不戴墨镜→眼黑色素瘤"）。

### 5. 良性 vs 恶性黑棘皮病 (Acanthosis nigricans)

- **良性**：年轻 + 胰岛素抵抗（DM/肥胖/PCOS 多囊卵巢综合征）+ skin tags 软纤维瘤
- **恶性**：老年、突发、体重下降、不寻常部位（黏膜/掌跖）→ 查 **GI/GU 癌**

### 6. 淋巴管炎 vs 丹毒 vs 蜂窝织炎 vs 浅静脉炎（NBME 秒杀）

| | **丹毒 Erysipelas** | **蜂窝织炎 Cellulitis** | **淋巴管炎 Lymphangitis** | **浅静脉炎 Phlebitis** |
|---|---|---|---|---|
| 深度 | 真皮浅层+浅淋巴管 | 深真皮+皮下 | 淋巴主干 | 浅静脉 |
| 边界 | **极清楚、略隆起** | **模糊云雾状** | 线状红条纹 | 沿浅静脉 |
| 特征 | **耳受累** + 寒战高热（极重） | 伤口/足癣诱因 | **红条纹+淋巴结大** | **可触及硬索 cord** |
| 主菌 | GAS A 组链球菌 | GAS / S. aureus | GAS / S. aureus | — |

### 7. 脂溢性皮炎 vs SLE 颊部红斑

- 累及**鼻唇沟 (nasolabial fold)** → 脂溢性皮炎 seborrheic dermatitis / 酒渣鼻 rosacea
- **不累及鼻唇沟** → SLE 系统性红斑狼疮 malar rash

### 8. 头癣 vs 斑秃

- **Tinea capitis 头癣**：鳞屑/红斑/脓疱(Kerion) + **黑点（断发）** + 枕部淋巴结大 + 瘙痒；Wood 灯（Microsporum 小孢子菌）绿荧光
- **Alopecia areata 斑秃**：皮肤光滑 + **惊叹号发 (exclamation point hairs)** + 无淋巴结 + 无症状

## 四、分类速记（按机制成组）

### 真菌 Tinea
- **corporis 体癣**：环形+中央清亮 → topical azole
- **versicolor 花斑癣**：减色斑、spaghetti & meatballs → selenium sulfide 二硫化硒 / ketoconazole 酮康唑
- **capitis 头癣 / unguium 甲癣**：**必须口服** griseofulvin 灰黄霉素 / terbinafine 特比萘芬
- **Kerion 脓癣**：强超敏 → boggy 海绵感炎性肿块

![[皮肤_Kerion脓癣.png]]
*脓癣 (Kerion)：头皮**海绵感 (boggy) 隆起炎性肿块 + 脓性渗出 + 断发**——头癣 (tinea capitis) 重型，强细胞免疫超敏反应。*

### 链球菌皮肤病（LINES 助记）
Lymphangitis 淋巴管炎 / Impetigo 脓疱疮 / **N**ecrotizing fasciitis 坏死性筋膜炎 / Erysipelas 丹毒 / Scarlet fever 猩红热
- **治疗可防风湿热 rheumatic fever，但不能防 PSGN 链球菌感染后肾炎**

### 皮炎/丘疹鳞屑
- **Contact dermatitis 接触性皮炎**：ACD = IV 型迟发超敏（48–72h）；ICD = 直接化学损伤无需致敏
- **Psoriasis 银屑病**：伸侧银白鳞屑、**Auspitz 征**（刮之出血）、**Köbner 现象**、累及 DIP；Tx 局部高效价激素+维 D (calcipotriene 卡泊三醇)，中重度→光疗/MTX 甲氨蝶呤/生物制剂；**禁系统性糖皮质激素**（停药诱发脓疱型）；急性快速进展→查 HIV / 链球菌 / 药物
- **Lichen planus 扁平苔藓**：**5P**（Pruritic, Purple, Polygonal, Papules, Plaques）+ Wickham 纹
- **Dermatitis herpetiformis 疱疹样皮炎**：**乳糜泻 celiac 相关** + 抗组织转谷氨酰胺酶 IgA → **dapsone 氨苯砜 + 无麸质饮食**
- **Pityriasis rosea 玫瑰糠疹**：HHV-6/7、herald patch、圣诞树、自限 6–8 周

![[皮肤_玫瑰糠疹母斑.png]]
*玫瑰糠疹 (pityriasis rosea)：先出现单个较大**母斑 (herald patch)**，1–2 周后躯干沿皮纹散布小卵圆斑（圣诞树样），避开手掌脚掌（手脚受累→改查二期梅毒）。*

### 痤疮阶梯 Acne（Cutibacterium acnes 痤疮丙酸杆菌）
粉刺型 → topical retinoid 维 A 酸；炎症型 → +benzoyl peroxide 过氧化苯甲酰 ± topical/oral 抗生素；结节囊肿型难治 → **oral isotretinoin 异维 A 酸**（需妊娠试验 + **两种避孕**）
> **Acne + 光敏 = doxycycline 多西环素**；药物性痤疮（激素）=同一阶段、形态均一、累及上背/肩、任何年龄。

## 五、自测（答完再判分）

> [!question]- Q1. 7 岁男孩头皮局限性脱发斑，伴鳞屑、黑点、枕部淋巴结肿大、瘙痒。最佳治疗？
> **Tinea capitis 头癣** → **口服** griseofulvin 灰黄霉素或 terbinafine 特比萘芬（外用无效）。鉴别斑秃（皮肤光滑、惊叹号发、无淋巴结）。

> [!question]- Q2. 中年男性服用别嘌醇 2 周后发热、痛性红斑迅速进展为水疱剥脱，口腔与结膜糜烂，Nikolsky+，累及 25% 体表。诊断与第一步处理？
> **SJS-TEN overlap**（10–30% BSA）→ **立即停别嘌醇 allopurinol + 转烧伤科** + 补液创面护理。

> [!question]- Q3. 30 岁女性背部先出现一处椭圆鳞屑斑（1 周前），后躯干沿皮纹出现多发小斑呈圣诞树样，手掌脚掌无皮疹。最该排除什么？下一步？
> 临床像 **玫瑰糠疹 pityriasis rosea**，但**必须排二期梅毒** → 查 RPR/VDRL（梅毒会累及手掌脚掌）。

> [!question]- Q4. 老年男性面部出现珍珠样隆起边缘、表面毛细血管扩张的丘疹。最可能诊断与面部高危病灶首选术式？
> **BCC 基底细胞癌** → 面部/高危 → **Mohs 显微外科手术**。

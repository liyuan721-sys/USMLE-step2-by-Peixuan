> [!note]- 缩写对照（全称 中英）
> - **SJS** = Stevens-Johnson syndrome 史蒂文斯-约翰逊综合征
> - **TEN** = toxic epidermal necrolysis 中毒性表皮坏死松解症
> - **SSSS** = staphylococcal scalded skin syndrome 葡萄球菌性烫伤样皮肤综合征
> - **DRESS** = drug reaction with eosinophilia and systemic symptoms 伴嗜酸粒细胞增多和系统症状的药物反应
> - **EM** = erythema multiforme 多形红斑
> - **BCC** = basal cell carcinoma 基底细胞癌
> - **SCC** = squamous cell carcinoma 鳞状细胞癌
> - **AK** = actinic keratosis 光化性角化病
> - **BSA** = body surface area 体表面积
> - **ACD / ICD** = allergic / irritant contact dermatitis 变应性 / 刺激性接触性皮炎
> - **PCT** = porphyria cutanea tarda 迟发性皮肤卟啉病
> - **AIP** = acute intermittent porphyria 急性间歇性卟啉病
> - **KOH** = potassium hydroxide 氢氧化钾（真菌镜检）
> - **AN** = acanthosis nigricans 黑棘皮病
> - **DH** = dermatitis herpetiformis 疱疹样皮炎
> - **HHV** = human herpesvirus 人类疱疹病毒
> - **HSV / VZV** = herpes simplex / varicella-zoster virus 单纯疱疹 / 水痘-带状疱疹病毒
> - **HIV** = human immunodeficiency virus 人类免疫缺陷病毒
> - **HepC** = hepatitis C 丙型肝炎
> - **I&D** = incision and drainage 切开引流
> - **GAS** = group A Streptococcus A 组链球菌
> - **PSGN** = post-streptococcal glomerulonephritis 链球菌感染后肾小球肾炎
> - **DIP** = distal interphalangeal joint 远端指间关节
> - **TSH** = thyroid-stimulating hormone 促甲状腺激素
> - **5-FU** = 5-fluorouracil 5-氟尿嘧啶

## 〇、核心规则（先记这几条）

> [!tip] Core Rules
> - **真菌检测**：best initial = **KOH prep 氢氧化钾镜检**；most accurate = **fungal culture 真菌培养**。
> - **真菌通则**：浅表 → topical azole 外用唑类；**毛发 (capitis) 和指甲 (unguium) 必须口服**（外用渗不进毛囊/甲）。
> - **感染严重度阶梯**：局限→topical / 广泛→oral / 不稳定→IV / 组织坏死→**手术**。
> - **活检指征**：变化的痣 / 可疑色素灶 / 慢性不愈 / 长在瘢痕烧伤创面 / 突发爆发性。
> - **反复念珠菌病 (candidiasis) → 查 HIV / DM**。

## 一、Buzzword → 诊断反射（最快速查）

| Buzzword | 诊断 |
|---|---|
| Spaghetti and meatballs（镜下）/ 晒不黑减色斑 | **Tinea versicolor 花斑癣** (Malassezia 马拉色菌) |
| Honey-crusted 蜜黄色结痂 | **Impetigo 脓疱疮** |
| Pearly rolled border + telangiectasia 珍珠样卷边+毛细血管扩张 | **BCC 基底细胞癌** |
| Keratin pearls 角珠（组织学） | **SCC 鳞状细胞癌** |
| Herald patch + Christmas tree 母斑+圣诞树（避开手脚心） | **Pityriasis rosea 玫瑰糠疹** |
| 皮疹**累及手掌脚掌** | **Secondary syphilis 二期梅毒**（首要鉴别） |
| Sandpaper rash + strawberry tongue 砂纸疹+草莓舌 | **Scarlet fever 猩红热** |
| Slapped cheek 掌掴颊 | **Erythema infectiosum 传染性红斑** (Parvovirus B19 细小病毒) |
| Target/iris lesion 靶形+近期 HSV | **Erythema multiforme 多形红斑** |
| Bull's eye 扩大 + 蜱叮 | **Erythema migrans 游走性红斑** (Lyme 莱姆病) |
| Wickham striae 白色蕾丝纹 + 5P | **Lichen planus 扁平苔藓** |
| 茶/酒色尿 + 日晒区无痛水疱 + HepC | **PCT 迟发性皮肤卟啉病** |
| Umbilicated 脐凹丘疹 | **Molluscum contagiosum 传染性软疣** |
| **Painless** black eschar 无痛黑焦痂 | **Anthrax 炭疽** (Bacillus anthracis) → ciprofloxacin 环丙沙星 |
| **Painful** eschar 疼痛焦痂 | **Brown recluse 棕色隐士蜘蛛咬伤** |
| Wood-grain 木纹样同心带 | **Erythema gyratum repens**（副肿瘤，先于癌诊断） |
| 老年突发天鹅绒色素斑 + 体重↓ | **Malignant acanthosis nigricans 恶性黑棘皮病** → 查 GI/GU 癌 |

![[皮肤_多形红斑靶形.png]]
*多形红斑 (erythema multiforme)：经典**三色靶形 (target/iris) 损害**——中央暗紫/水疱 + 苍白环 + 红边；多 HSV 后 1–2 周发病。*

![[皮肤_游走性红斑莱姆.png]]
*游走性红斑 (erythema migrans)：蜱叮后从中心向外扩大的**牛眼/靶心红斑** → 莱姆病 (Lyme disease, Borrelia)。*

![[皮肤_边缘性红斑风湿热.png]]
*边缘性红斑 (erythema marginatum)：一过性、粉红、**环形/匐行边界 + 中央清除**——急性风湿热 (acute rheumatic fever) 表现。*

## 二、皮肤急症 & 红旗（最高优先）

| 急症 | 关键特征 | 处理 |
|---|---|---|
| **SJS / TEN** | 发热+痛性红斑→水疱剥脱、**Nikolsky 征+**、**口/眼/生殖器黏膜受累**、面+躯干起始（spare 头皮/掌跖）；机制=角质细胞凋亡 | **立即停诱发药**（卡马西平 carbamazepine、磺胺、别嘌醇 allopurinol、抗癫痫药）+ **送烧伤科** + 创面护理 + 补液 ± IVIG |
| **DRESS** | 药疹 + **嗜酸粒细胞增多** + 器官损害，潜伏期长（2–8 周） | 停药 + 系统糖皮质激素 |
| **SSSS** | 婴幼儿；S. aureus 金黄色葡萄球菌剥脱毒素；Nikolsky+；**不累及黏膜**；颗粒层浅剥脱、愈合快不留瘢 | nafcillin 萘夫西林 + 支持 |
| **Necrotizing fasciitis 坏死性筋膜炎** | 剧痛（与体征不符）+ 捻发音 crepitus + 紫色大疱 + 低血压 | 影像见气体、**别触碰** → 急诊清创 + penicillin 青霉素 + clindamycin 克林霉素 |
| **Eczema herpeticum 疱疹性湿疹** | 特应性皮炎 atopic dermatitis 患儿 + **punched-out 凿孔样糜烂** + 出血性结痂 + 发热 | **立即系统 acyclovir 阿昔洛韦** |
| **Ecthyma gangrenosum 坏疽性臁疮** | Pseudomonas 铜绿假单胞菌菌血症 + 中性粒减少 → 出血性脓疱/坏死黑痂 + 脓毒（详见 [[完整笔记/专题笔记/皮肤/皮肤_浅表感染]] §一，含与"深部脓疱疮 ecthyma"对比） | 抗假单胞菌治疗 |

> [!danger] SJS/TEN vs SSSS 一句话
> 都 Nikolsky+，但 **SJS/TEN 累及黏膜 + 表皮全层坏死（深）**；**SSSS 不累及黏膜 + 仅颗粒层剥脱（浅）**。BSA 阈值：SJS <10% / overlap 10–30% / TEN >30%。

![[皮肤_EM黏膜受累.png]]
*多形红斑 (erythema multiforme) 黏膜重型：口唇/口腔糜烂 + 靶形红斑——与 SJS 谱系黏膜受累相邻，常 HSV 诱发。*

![[皮肤_疱疹性湿疹.png]]
*疱疹性湿疹 (eczema herpeticum)：特应性皮炎基础上**凿孔样糜烂 (punched-out erosions) + 出血性结痂** → 立即系统 acyclovir 阿昔洛韦。*

![[皮肤_坏疽性臁疮.png]]
*坏疽性臁疮 (ecthyma gangrenosum)：出血性脓疱迅速演变为**坏死性溃疡 + 周边红斑**——铜绿假单胞菌 (Pseudomonas) 菌血症 + 中性粒减少。*

![[Pasted image 20260629220216.png|360]]
*坏疽性臁疮 (ecthyma gangrenosum)（另例）：肢体出血性脓疱 → 中央**坏死性黑痂溃疡 + 周边红斑**——铜绿假单胞菌 (Pseudomonas aeruginosa) 菌血症 + 中性粒减少。*

## 三、重点鉴别对比表

### 1. 严重药物/水疱反应

| 疾病 | 核心 | 黏膜 | 处理 |
|---|---|---|---|
| **SJS/TEN** | Nikolsky+、表皮剥脱 | **常受累** | 烧伤病房、停药 |
| **SSSS** | 婴儿、剥脱毒素 | **无/罕见** | nafcillin + 支持 |
| **EM 多形红斑** | **靶形损害** | 轻/可有 | 多 HSV 诱发，支持 |
| **DRESS** | **嗜酸粒↑** + 器官损害 | 较少 | 停药 + 系统激素 |

### 2. 玫瑰糠疹 vs 二期梅毒 vs 体癣 vs 药疹

| | 母斑 | 圣诞树小斑 | 手掌脚掌 | 检查 |
|---|---|---|---|---|
| **玫瑰糠疹** | 有 | 有 | **避开** | 临床 |
| **二期梅毒** | 无 | 有 | **累及** ★ | RPR/VDRL |
| **体癣 tinea corporis** | 仅"大斑" | 极少 | 任何处 | KOH |
| **药疹** | 无 | 有 | 对称 | 药物史 |

> [!warning] 安全清单：有母斑+圣诞树+手脚干净→玫瑰糠疹；**手脚受累立刻改梅毒**；高热/草莓舌/淋巴结→猩红热或 Kawasaki 川崎病。

![[皮肤_川崎病体征.png]]
*Kawasaki 川崎病 (mucocutaneous lymph node syndrome) 黏膜皮肤体征：发热>5 天 + 双侧非渗出性结膜炎 + 草莓舌 + 多形皮疹 + 手足肿胀脱皮 + 颈淋巴结肿大（CRASH and burn）。*

### 3. 大疱性类天疱疮 vs 寻常型天疱疮

| | **Bullous pemphigoid 大疱性类天疱疮** | **Pemphigus vulgaris 寻常型天疱疮** |
|---|---|---|
| 抗原 | 抗 hemidesmosome 半桥粒（基底膜） | 抗 desmoglein 桥粒芯蛋白（细胞间） |
| 水疱 | **紧张性 tense、厚、不易破** | **松弛、易破** |
| 黏膜 | 罕见 | **口腔受累** |
| Nikolsky | **−** | **+**（acantholysis 棘层松解） |
| 痛 | 无痛 | 疼痛 |
| Tx | 糖皮质激素 | 激素，难治→cyclophosphamide 环磷酰胺/IVIG/血浆置换 |

### 4. 皮肤癌切缘 & 高危处理

| 癌 | 关键 | 处理 |
|---|---|---|
| **BCC 基底细胞癌** | 珍珠样卷边 + 毛细血管扩张，上面部 | low-risk 躯干<2cm→切除 4mm 切缘 / topical；high-risk（面颈、≥2cm、边界不清）→ **Mohs 手术** |
| **SCC 鳞状细胞癌** | 角珠、下面部/唇、溃疡结痂 | 切除 4–6mm 切缘；**Marjolin ulcer**=慢性烧伤瘢痕恶变→SCC |
| **Melanoma 黑色素瘤** | **ABCDE**；marker **S100**；**深度 (Breslow) = 首要预后** | **全层活检（full-thickness）**；切缘按厚度 1–3cm；深者+干扰素 interferon |
| **Gorlin 综合征** | AD 遗传、**PTCH 突变**、多发 BCC + 掌凹点 + 颌囊肿 + 叉状肋 | — |

> [!tip] Melanoma 排查陷阱：指甲褐色纵纹（尤其外伤史）、眼内褐色灶（"晒太阳不戴墨镜→眼黑色素瘤"）。

### 5. 良性 vs 恶性黑棘皮病 (Acanthosis nigricans)

- **良性**：年轻 + 胰岛素抵抗（DM/肥胖/PCOS 多囊卵巢综合征）+ skin tags 软纤维瘤
- **恶性**：老年、突发、体重下降、不寻常部位（黏膜/掌跖）→ 查 **GI/GU 癌**

### 6. 淋巴管炎 vs 丹毒 vs 蜂窝织炎 vs 浅静脉炎（NBME 秒杀）

| | **丹毒 Erysipelas** | **蜂窝织炎 Cellulitis** | **淋巴管炎 Lymphangitis** | **浅静脉炎 Phlebitis** |
|---|---|---|---|---|
| 深度 | 真皮浅层+浅淋巴管 | 深真皮+皮下 | 淋巴主干 | 浅静脉 |
| 边界 | **极清楚、略隆起** | **模糊云雾状** | 线状红条纹 | 沿浅静脉 |
| 特征 | **耳受累** + 寒战高热（极重） | 伤口/足癣诱因 | **红条纹+淋巴结大** | **可触及硬索 cord** |
| 主菌 | GAS A 组链球菌 | GAS / S. aureus | GAS / S. aureus | — |

### 7. 脂溢性皮炎 vs SLE 颊部红斑

- 累及**鼻唇沟 (nasolabial fold)** → 脂溢性皮炎 seborrheic dermatitis / 酒渣鼻 rosacea
- **不累及鼻唇沟** → SLE 系统性红斑狼疮 malar rash

### 8. 头癣 vs 斑秃（含瘢痕性 vs 非瘢痕性）

- **Tinea capitis 头癣**：鳞屑/红斑/脓疱(Kerion) + **黑点（断发）** + 枕部淋巴结大 + 瘙痒；Wood 灯（Microsporum 小孢子菌）绿荧光
- **Alopecia areata 斑秃**：皮肤光滑 + **惊叹号发 (exclamation point hairs)** + 无淋巴结 + 无症状

> [!tip] 瘢痕性 (scarring) vs 非瘢痕性脱发 —— 钥匙 = 毛囊口还在吗
> - **非瘢痕性（可逆，毛囊口保留）**：斑秃 / 头癣 / 休止期脱发 telogen effluvium / 雄激素性脱发。
> - **瘢痕性（永久，毛囊口消失）**：**盘状红斑狼疮 (DLE)** —— 红斑鳞屑斑块 + **毛囊角栓** + 中央萎缩色脱；扁平苔藓样 (lichen planopilaris)。
> - 见到**头皮瘢痕性脱发 + 面部界清瘢痕斑块 + 毛囊角栓** → 想 **DLE**，详见 [[完整笔记/专题笔记/风湿/风湿_SLE系统性红斑狼疮]] §3.1（皮肤型狼疮三型）。

## 四、分类速记（按机制成组）

### 真菌 Tinea
- **corporis 体癣**：环形+中央清亮 → topical azole
- **versicolor 花斑癣**：减色斑、spaghetti & meatballs → selenium sulfide 二硫化硒 / ketoconazole 酮康唑
- **capitis 头癣 / unguium 甲癣**：**必须口服** griseofulvin 灰黄霉素 / terbinafine 特比萘芬
- **Kerion 脓癣**：强超敏 → boggy 海绵感炎性肿块

![[皮肤_Kerion脓癣.png]]
*脓癣 (Kerion)：头皮**海绵感 (boggy) 隆起炎性肿块 + 脓性渗出 + 断发**——头癣 (tinea capitis) 重型，强细胞免疫超敏反应。*

### 链球菌皮肤病（LINES 助记）
Lymphangitis 淋巴管炎 / Impetigo 脓疱疮 / **N**ecrotizing fasciitis 坏死性筋膜炎 / Erysipelas 丹毒 / Scarlet fever 猩红热
- **治疗可防风湿热 rheumatic fever，但不能防 PSGN 链球菌感染后肾炎**

### 皮炎/丘疹鳞屑
- **Contact dermatitis 接触性皮炎**：ACD = IV 型迟发超敏（48–72h）；ICD = 直接化学损伤无需致敏
- **Psoriasis 银屑病**：伸侧银白鳞屑、**Auspitz 征**（刮之出血）、**Köbner 现象**、累及 DIP；Tx 局部高效价激素+维 D (calcipotriene 卡泊三醇)，中重度→光疗/MTX 甲氨蝶呤/生物制剂；**禁系统性糖皮质激素**（停药诱发脓疱型）；急性快速进展→查 HIV / 链球菌 / 药物 → 详见 [[完整笔记/专题笔记/皮肤/皮肤_银屑病]]
- **Lichen planus 扁平苔藓**：**5P**（Pruritic, Purple, Polygonal, Papules, Plaques）+ Wickham 纹
- **Dermatitis herpetiformis 疱疹样皮炎**：**乳糜泻 celiac 相关** + 抗组织转谷氨酰胺酶 IgA → **dapsone 氨苯砜 + 无麸质饮食**
- **Pityriasis rosea 玫瑰糠疹**：HHV-6/7、herald patch、圣诞树、自限 6–8 周

![[皮肤_玫瑰糠疹母斑.png]]
*玫瑰糠疹 (pityriasis rosea)：先出现单个较大**母斑 (herald patch)**，1–2 周后躯干沿皮纹散布小卵圆斑（圣诞树样），避开手掌脚掌（手脚受累→改查二期梅毒）。*

### 痤疮阶梯 Acne（Cutibacterium acnes 痤疮丙酸杆菌）
粉刺型 → topical retinoid 维 A 酸；炎症型 → +benzoyl peroxide 过氧化苯甲酰 ± topical/oral 抗生素；结节囊肿型难治 → **oral isotretinoin 异维 A 酸**（需妊娠试验 + **两种避孕**）
> **Acne + 光敏 = doxycycline 多西环素**；药物性痤疮（激素）=同一阶段、形态均一、累及上背/肩、任何年龄。
> *光毒性 vs 光变态反应 + 致光敏药谱 → [[完整笔记/专题笔记/皮肤/皮肤_光敏性药物反应]]*

## 五、自测（答完再判分）

> [!question]- Q1. 7 岁男孩头皮局限性脱发斑，伴鳞屑、黑点、枕部淋巴结肿大、瘙痒。最佳治疗？
> **Tinea capitis 头癣** → **口服** griseofulvin 灰黄霉素或 terbinafine 特比萘芬（外用无效）。鉴别斑秃（皮肤光滑、惊叹号发、无淋巴结）。

> [!question]- Q2. 中年男性服用别嘌醇 2 周后发热、痛性红斑迅速进展为水疱剥脱，口腔与结膜糜烂，Nikolsky+，累及 25% 体表。诊断与第一步处理？
> **SJS-TEN overlap**（10–30% BSA）→ **立即停别嘌醇 allopurinol + 转烧伤科** + 补液创面护理。

> [!question]- Q3. 30 岁女性背部先出现一处椭圆鳞屑斑（1 周前），后躯干沿皮纹出现多发小斑呈圣诞树样，手掌脚掌无皮疹。最该排除什么？下一步？
> 临床像 **玫瑰糠疹 pityriasis rosea**，但**必须排二期梅毒** → 查 RPR/VDRL（梅毒会累及手掌脚掌）。

> [!question]- Q4. 老年男性面部出现珍珠样隆起边缘、表面毛细血管扩张的丘疹。最可能诊断与面部高危病灶首选术式？
> **BCC 基底细胞癌** → 面部/高危 → **Mohs 显微外科手术**。

### 2. 细菌 & 病毒 & 寄生虫

![[{F1D26F73-3583-42D5-90D3-6C905747B17A}.png|360]]
*脓疱疮 (impetigo 脓疱疮)：**蜜黄色结痂 (honey-crusted)**；GAS/金葡菌；治疗防风湿热但**不防 PSGN**。*

![[{08C8DFBC-4E11-4F47-8D39-FF97B095C49C}.png|360]]
*臁疮 (ecthyma 深脓疱疮)：穿凿样溃疡 + 厚痂 = 累及真皮的"深部脓疱疮"（GAS 化脓链球菌；详见 [[完整笔记/专题笔记/皮肤/皮肤_浅表感染]] §一）。*

![[{5BEEC2B8-99FB-4210-A361-1B01BD471E26}.png|360]]
*带状疱疹 (herpes zoster 带状疱疹)：**单侧皮节分布 (dermatomal)** 簇集水疱，不过中线；valacyclovir 伐昔洛韦。*

![[{792B5872-E32F-4AF1-94EC-2D6EFA749EEF}.png|360]]
*疱疹性瘭疽 (herpetic whitlow 疱疹性瘭疽)：指端 HSV 成簇水疱；**勿切开** (与细菌性脓肿区别)。*

![[{EA764CEA-09AD-4FF4-98BC-39ED53F89F79}.png|360]]
*疥疮 (scabies 疥疮)：**指缝/腕屈侧**线状隧道 + 剧痒（夜间加重）；permethrin 氯菊酯，接触者同治。*

![[{DA18AE0B-CE2A-4B8B-B6D2-19E6AFAED042}.png|360]]
*结痂性（挪威）疥疮 (crusted/Norwegian scabies)：免疫抑制者厚痂 + 极高传染性；ivermectin 伊维菌素 + 外用。*

![[{4C7CD5BF-AA97-464E-9905-CC51C915528C}.png|360]]
*红斑癣 (erythrasma 红斑癣)：间擦区红棕斑、边界平、细鳞屑；**Wood 灯珊瑚红荧光**；Corynebacterium minutissimum 微细棒状杆菌。*

![[{378EE415-C677-492E-A100-97AB4876AD9E}.png|360]]
*猫抓病 (cat-scratch disease)：接种处红色丘疹/结节 + 区域淋巴结肿大；Bartonella henselae 汉赛巴尔通体 → azithromycin 阿奇霉素。*

![[{C5913A51-F9A7-40DB-A163-F777AD5EE4D7}.png|360]]
*猩红热 (scarlet fever)：咽炎后出现**细小、粉红、砂纸样疹 (fine, pink, sandpaper-like rash)**，皱褶区加重，后期**脱屑 (desquamation)**——产红斑毒素的 A 组链球菌 (group A Streptococcus)；**不形成水疱、Nikolsky 阴性**（与 SSSS 区分）。*

![[{5025E5DC-315D-4452-9FE9-126CCFD8DBD5}.png|360]]
*脓疱疮（另例）：口周蜜黄色结痂糜烂。*

![[{DDD05762-F30A-4B7C-A319-220C1BE3BDE4}.png|360]]
*疥疮（婴儿）：足部弥漫红色丘疹脓疱（婴儿掌跖受累为特点）。*

![[{A81172F2-808D-4195-89C7-2FE830CA3EA9}.png|360]]
*疥疮（另例）：散在红色丘疹 + 抓痕（剧痒夜重）。*

![[{9FA8445C-76EB-4660-9E45-FD3D996753A3}.png|360]]
*结痂性（挪威）疥疮（另例）：手部厚角化痂壳 + 甲下碎屑——免疫抑制者，极强传染性。*

### 3. 湿疹/皮炎 & 丘疹鳞屑

![[{C04A4CFB-27DE-492C-A0A5-BC16220CA2AB}.png|360]]
*特应性皮炎 (atopic dermatitis 特应性皮炎) 慢性期：屈侧苔藓样变 + 抓痕；个人/家族特应史（哮喘/过敏性鼻炎）。*

![[{633310E7-4993-4BA0-A083-0296DE70EDCA}.png|360]]
*特应性皮炎年龄分布：婴儿**面/伸侧** → 儿童及成人转**屈侧 (flexural)**。*

![[{A89EA365-2288-4ECC-A775-37DABF1BCF16}.png|360]]
*慢性刺激性接触性皮炎 (chronic ICD)：反复接触弱刺激物 → 干燥鳞屑、皲裂、苔藓样变；处理 = 避刺激 + 润肤剂 (emollient) + 外用激素。*

![[{2FBE523F-FF9A-4C56-96CC-22D8E731B0F5}.png|360]]
*变应性接触性皮炎 (allergic contact dermatitis, ACD)：**线状排列**红斑水疱（毒葛 poison ivy 经典）= IV 型迟发超敏。*

![[{B150B7C0-A576-43FD-8E32-7C7AB1FA6B1E}.png|360]]
*ACD vs ICD 对照：变应性 = 致敏后迟发、可超出接触区；刺激性 = 直接化学损伤、局限接触区。*

![[{9FE32335-2107-4945-B9F7-E12263E435A0}.png|360]]
*钱币状湿疹 (nummular eczema 钱币状湿疹)：四肢**硬币状 (coin-shaped)** 瘙痒斑块。*

![[{FE327EAF-CFA4-49AE-8996-6E602B1E99BC}.png|360]]
*脂溢性皮炎 (seborrheic dermatitis 脂溢性皮炎)：头皮/眉/鼻唇沟**油腻黄痂**；泛发查 HIV / 帕金森病。*

![[{62C81A5E-7C2A-4B1A-A503-AADBD48CC211}.png|360]]
*婴儿脂溢性皮炎 / 摇篮帽 (cradle cap)：头顶油腻黄色厚鳞屑，自限。*

![[{43AF2191-BA20-45D8-903F-AD0DF8C5854C}.png|360]]
*玫瑰糠疹 (pityriasis rosea) 圣诞树样分布：躯干沿皮纹散布卵圆斑（母斑后 1–2 周）；避开手掌脚掌（受累 → 查二期梅毒）。*

![[{35B1570B-52F0-4613-99C0-4B65A0AD2087}.png|360]]
*扁平苔藓 (lichen planus 扁平苔藓)：**5P**（瘙痒 Pruritic / 紫红 Purple / 多角形 Polygonal / 丘疹 Papules / 斑块 Plaques）+ Wickham 白色蕾丝纹。*

![[{F1C70928-E5DD-440A-BCBE-10ADA6C65CD0}.png|360]]
*特应性皮炎（慢性，另例）：苔藓化 + 抓痕丘疹。*

![[{CAFDE090-4072-4B6B-8765-4D9C861C3767}.png|360]]
*特应性皮炎（慢性，另例）：屈侧苔藓样变。*

![[{9337F626-0D14-47AB-9490-31BD970048FD}.png|360]]
*特应性皮炎（丘疹型）：屈侧红斑丘疹。*

![[{4A9F4C30-800F-4AA0-8F88-1CC21A331ACE}.png|360]]
*特应性皮炎（丘疹型，另例）：屈侧红斑丘疹 + 抓痕。*

![[{B85EE1D3-3CB9-4DAE-8646-37CCC10D6088}.png|360]]
*特应性皮炎：红斑鳞屑性丘斑（婴幼儿面颊常见）。*

![[{BEB91DB0-F3E8-4B77-9AEC-910B6494381B}.png|360]]
*⚠️存疑（图非特异）：考虑特应性皮炎丘疹型，需结合病史排接触性皮炎 / 疥疮。*

![[{2FF5FD9B-81F1-4231-9DFB-131C7D973967}.png|360]]
*特应性皮炎相关体征：掌纹增多 (hyperlinear palms)——特应性体质标志（丝聚蛋白 filaggrin 缺陷）。*

![[{F2132D59-74C7-470A-AC3E-9F34A25D00A0}.png|360]]
*特应性皮炎婴儿分布（示意图）：面颊 / 伸侧 / 躯干。*

![[{BB4B33C7-65DA-4902-AC5C-EFD3E2768025}.png|360]]
*特应性皮炎年龄分布（示意图，另版）：婴伸侧 → 成人屈侧。*

![[{A50E9415-3693-41C9-AFB0-78D5191F8A62}.png|360]]
*急性刺激性接触性皮炎 (acute ICD)：颈部融合性红斑丘疹（强刺激物单次暴露）。*

![[{8F23EEC2-1188-41F9-8C4A-86D090A7098A}.png|360]]
*慢性刺激性接触性皮炎 (chronic ICD)（另例）：指端皲裂脱屑。*

![[{BC56F09C-54FB-4F33-B6E6-2FC33598F393}.png|360]]
*刺激性接触性皮炎：指端皲裂脱屑（反复湿 / 洗）。*

![[{B58DEE69-A681-4558-A5B3-68D151D7EFF4}.png|360]]

![[{036397AB-64C3-4EC1-8753-3F968FC30C09}.png|360]]
*急性变应性接触性皮炎：红斑丘疹 + 渗出水疱。*

![[{B53F95DA-5512-4843-B8C4-2637E659B56B}.png|360]]
*慢性变应性接触性皮炎：腕部红斑 + 苔藓样变。*

![[{A19EB848-8369-4E7E-A659-177AAC33FFA3}.png|360]]
*钱币状湿疹（另例）：硬币状鳞屑性瘙痒斑（与体癣鉴别——癣有中央清亮 + KOH+）。*

![[{B01ACA59-AE27-4E57-8B66-8EF97EE4E1CB}.png|360]]
*⚠️存疑（难与体癣 / 银屑病区分）：考虑钱币状湿疹，鳞屑皲裂斑块。*

![[{2E021120-D3EE-4BF6-87A9-450768B54DDE}.png|360]]
*脂溢性皮炎（面，另例）：鼻周油腻黄痂红斑。*

### 4. 水疱 & 自身免疫

![[{138747AC-AF00-4792-A7A2-0F597F3A0B1C}.png|360]]
*寻常型天疱疮 (pemphigus vulgaris)：松弛**易破水疱 + 口腔糜烂**，Nikolsky+；抗 desmoglein 桥粒芯蛋白（细胞间）。*

![[{E6D353DE-6689-47D7-85DE-AA645BEBA139}.png|360]]
*疱疹样皮炎 (dermatitis herpetiformis, DH) 组织学：真皮乳头顶**中性粒微脓肿** + IgA 沉积；乳糜泻 (celiac) 相关 → dapsone 氨苯砜 + 无麸质饮食。*

![[{099DF996-2305-4BF3-9704-F32482DCFA50}.png|360]]
*亚急性皮肤型红斑狼疮 (SCLE)：日晒区环形/银屑病样鳞屑斑，anti-Ro/SSA，不留瘢痕；瘙痒少见。*

![[{B136CA91-917E-450E-971C-25DA9CA70F95}.png|360]]
*疱疹样皮炎 (DH) 组织学（另例）：真皮乳头顶中性粒微脓肿（与大疱性类天疱疮 bullous pemphigoid 鉴别）。*

### 5. 肿瘤 / 癌前 / 副肿瘤

![[{D3DE0262-B2BB-4661-9CFF-222AFC1C6218}.png|360]]
*光化性角化病 (actinic keratosis, AK)：日晒区**砂纸样粗糙鳞屑**斑，SCC 癌前；5-FU / 冷冻。*

![[{68D56ADA-F96E-45C1-8534-F51040A267BF}.png|360]]
*浅表型基底细胞癌 (superficial BCC)：躯干红斑样薄斑块、卷边；低危可外用 / 切除。*

![[{A8E5CCF2-97FB-48DB-8645-8EE7A5813E6A}.png|360]]
*Bowen 病 / 原位鳞癌 (SCC in situ)：边界清红斑 + 厚鳞屑，长期不愈。*

![[{F0087363-7497-43C5-8DAD-444B8CDA2FE5}.png|360]]
*脂溢性角化病 (seborrheic keratosis)：**"贴附样" (stuck-on)** 蜡样棕色丘疹，良性。*

![[{1165C477-2A99-45CA-9B57-8AC0842990DC}.png|360]]
*Leser-Trélat 征：脂溢性角化**突然爆发性增多** = 副肿瘤（多胃肠腺癌），查潜在恶性。*

![[{7C33E9BB-D5D9-431C-87B4-7AD7108151D5}.png|360]]
*肥厚型光化性角化病 (hypertrophic AK)（另例）：红斑基底 + 厚角质鳞屑（与皮角 / SCC 鉴别）。*

![[{B40A6BE0-FC54-4957-8A68-9BD28C88755B}.png|360]]
*肛周原位鳞癌 / Bowen 病 (perianal SCC in situ)：边界清红色斑块（HPV 相关；与肛周湿疹 / 乳房外 Paget 病鉴别）。*

![[{00C7F439-F80C-4E6A-9B7E-9AFFB882BDB8}.png|360]]
*Sweet 综合征（急性发热性嗜中性皮病）：**疼痛性水肿性红斑块/丘疹**（非银白鳞屑），发热 + 中性粒↑；查血液恶性肿瘤。*

![[Pasted image 20260629220402.png|360]]
*Sweet 综合征 (acute febrile neutrophilic dermatosis)（另例）：面部疼痛性红色水肿性丘疹/斑块 + 口腔/舌溃疡——发热 + 外周中性粒↑，常伴血液系统恶性肿瘤（如 AML 急性髓系白血病 / acute myeloid leukemia）。*

![[Pasted image 20260629220227.png|360]]
*坏疽性脓皮病 (pyoderma gangrenosum, PG)：**迅速扩大的痛性溃疡 + 紫红色潜行性边界 (violaceous undermined border)**，好发下肢；属中性粒细胞性皮病，强关联**炎症性肠病 (inflammatory bowel disease, IBD)**、关节炎、血液病；**勿清创（pathergy 同形反应会加重）**，治以系统激素/免疫抑制。*

![[{34B7DB7B-11FD-4134-8AFF-2442EB899171}.png|360]]
*坏疽性脓皮病 (pyoderma gangrenosum, PG)（另例）：单个**扩张性溃疡 + 脓性/纤维素性基底 + 不规则紫红色边界 (purulent base, irregular violaceous border)**——常由局部创伤诱发 (pathergy 同形反应)，~30% 伴系统性炎症病（IBD 炎症性肠病、类风湿关节炎、急性髓系白血病 acute myeloid leukemia, AML）；溃疡缘活检 + 排除感染/肿瘤后临床诊断。（Q160067 题图）*

> [!note] 下肢溃疡——PG 的高频鉴别（同组干扰项）
>
> | 病 | 关键指纹 |
> |---|---|
> | **坏疽性脓皮病 (PG)** | 痛性、紫红潜行边界、pathergy、伴 IBD / 关节炎 / 血液病；勿清创 |
> | **皮肤小血管炎 (cutaneous small vessel vasculitis)** | **可触性紫癜 (palpable purpura)**、瘀点，偶有小水疱/小溃疡；大而孤立溃疡 + pathergy 不典型 |
> | **静脉功能不全性溃疡 (venous insufficiency ulcer)** | **内踝 (malleoli) 处、疼痛轻**，背景慢性淤积改变（褐色色沉、真皮萎缩 brawny discoloration） |
> | **坏疽性臁疮 (ecthyma gangrenosum)** | 出血性脓疱 → 黑痂坏死溃疡，铜绿假单胞菌 (Pseudomonas aeruginosa) 菌血症 + 中性粒减少（局部痛轻） |

![[{8802A999-0906-4FFE-9859-2FCAF5DBC0FD}.png|320]]
*皮肤小血管炎 (cutaneous small vessel vasculitis)：**不褪色、隆起的瘀点/紫癜 (nonblanching raised petechiae or purpura，即可触性紫癜)**——可有散在小水疱/小溃疡，但**大而孤立溃疡 + pathergy** 更提示坏疽性脓皮病 (PG)。*

![[{A274D52C-6F39-4596-9B63-7C35598932D7}.png|320]]
*静脉功能不全性溃疡 (venous insufficiency ulcer)：好发**内踝 (malleoli)**、**疼痛轻**，背景为慢性淤积改变（**褐色色素沉着 brawny discoloration、真皮萎缩**）——与 PG 的剧痛性紫红潜行边界溃疡区分。*

### 6. 其他常见

![[{354D6219-FC36-4E4D-8055-25C1DEE9159F}.png|360]]
*白癜风 (vitiligo 白癜风)：边界清楚**完全脱色斑**；自身免疫（查甲状腺 / T1DM）。*

![[{1960DB6C-1B3F-4C0B-B7F3-2C3845D6E105}.png|360]]
*荨麻疹 (urticaria 荨麻疹)：**一过性 (<24h) 风团**、剧痒；I 型超敏 / 肥大细胞脱颗粒。*

![[{1CF730B2-98F3-4E8B-8600-34661FFEEE81}.png|360]]
*血管性水肿 (angioedema 血管性水肿)：深层不对称肿胀（唇 / 眼睑）；ACEI 相关或 C1 酯酶抑制物缺乏。*

![[{F4FDB716-43DC-4065-B793-8F82534DA286}.png|360]]
*结节性红斑 (erythema nodosum)：小腿伸侧**痛性红色结节**（隔膜性脂膜炎）；查链球菌 / 结节病 / IBD / 药物。*

![[Pasted image 20260629215824.png|360]]
*老年性紫癜 (senile / actinic purpura, 又称日光性紫癜)：**老年人手背/前臂日晒受损皮肤**上轻微外伤即出现的**瘀斑 (ecchymoses)**，愈后留**棕色含铁血黄素 (hemosiderin) 沉着**；机制为血管周围弹力纤维丢失 → 血管脆性增加（抗凝药/激素/NSAIDs 加重）；**良性、无需进一步检查**（与凝血障碍、虐待鉴别）。*

![[{6FB73EC0-4242-4CB9-96D8-9A0B9BFE7F64}.png|360]]
*红痱 / 热疹 (miliaria rubra, heat rash)：**细小、密集的红色丘疹**（汗腺导管阻塞），见于闷热/多汗；细小红色丘疹为主、**无脓疱**——重症可短期低效价外用激素。*

![[{581941F6-60C7-4606-90E3-CF84F2D51164}.png|360]]
*结节囊肿型痤疮 (nodulocystic acne)：背 / 面深部结节囊肿 + 瘢痕 → **口服异维 A 酸 (isotretinoin)**（需妊娠试验 + 两种避孕）。*

![[{8F5A7EF7-FDFF-409D-A2FD-9E9EF483586F}.png|360]]
*痤疮分级治疗 (acne treatment)：粉刺型 → 外用维 A 酸；炎症型 → +过氧化苯甲酰 ± 抗生素；结节囊肿 → 口服异维 A 酸。*

![[{B3BBFDFA-0285-440B-805D-1959E72AAB69}.png|360]]
*黑棘皮病 (acanthosis nigricans, AN)：颈 / 腋天鹅绒样褐色增厚 ± skin tags 软纤维瘤；年轻 = 胰岛素抵抗（DM / 肥胖 / PCOS），老年突发 = 查 GI/GU 癌。*

![[{BDC5CCA2-92B3-4C23-87AC-D9606011B562}.png|360]]
*病毒疹 (viral exanthem)：发热 + 非瘙痒性红斑斑丘疹，伴病毒前驱症状，数日自退。*

![[{541A038C-59BD-428E-BEB2-881ED11608D5}.png|360]]
*结节病 (sarcoidosis) 胸片：双侧对称肺门淋巴结肿大——**结节性红斑 (erythema nodosum)** 常见关联病因之一（Löfgren 综合征）。*

#### 环形 / 靶形红斑（补图，呼应 §一 / §二 buzzword）

![[{773EC2B2-D337-46BD-A4FD-0E96AB8F5C2D}.png|360]]
*边缘性红斑 (erythema marginatum)：一过性、环形 / 匐行边界 + 中央清除——急性风湿热 (acute rheumatic fever)。*

![[Pasted image 20260629220233.png|360]]
*边缘性红斑 (erythema marginatum)（另例）：躯干一过性粉红环形斑、边界锐利/匐行 + 中央清除——急性风湿热 (acute rheumatic fever) 主要表现之一。*

![[{E0C7234E-3685-4565-A9FE-4887586904E2}.png|320]]
*边缘性红斑 (erythema marginatum)（另例）：多发**环形/匐行 (serpiginous) 红斑 + 中央清除**——一过性、不痒、儿童多见；急性风湿热 (acute rheumatic fever) 皮肤征。*

![[{8FCE152F-2D54-4ABB-932F-9E931C8971DC}.png|360]]
*游走性红斑 (erythema migrans)：蜱叮后牛眼 / 靶心扩大红斑——莱姆病 (Lyme disease)。*

![[{3A9ED0D3-9D32-4A1B-B461-B30F8D04D94C}.png|360]]
*游走性红斑（另例）：环形红斑 + 中央清除、**无隆起边界**，常伴全身症状 → doxycycline 多西环素。*

![[Pasted image 20260629220240.png|360]]
*游走性红斑 (erythema migrans)（另例）：蜱叮处向外扩大的**靶心/牛眼红斑 (targetoid bull's-eye)**、中央可见叮咬点——莱姆病 (Lyme disease, 伯氏疏螺旋体 Borrelia burgdorferi)。*

![[{CD7FB0E4-9ACE-4B42-864E-787BBFE66871}.png|360]]
*多形红斑 (erythema multiforme, EM)：三色**靶形 (target / iris)** 损害——多 HSV 单纯疱疹病毒后诱发。*

### 7. 甲病变 Nail Disorders

![[{63401044-5105-4798-974D-B42567D47549}.png|360]]
*银屑病甲 (psoriatic nails)：**点状凹陷 (pitting) + 油滴征 (oil spot) + 甲剥离 (onycholysis)**。*

![[{FF821A23-CD8C-4D82-97CC-2755FC6132CA}.png|360]]
*扁平苔藓甲 (lichen planus nails)：纵嵴、甲板变薄、翼状胬肉 (pterygium)。*

![[{846F475E-318C-4B7B-BF23-3733A79F4480}.png|360]]
*反甲 / 匙状甲 (koilonychia)：甲反向凹陷呈勺状 → **缺铁性贫血 (iron deficiency)**。*

![[{A2FC2387-BB8D-4868-9164-9D8958A24497}.png|360]]
*甲真菌病 (onychomycosis) 处理：甲增厚 / 变色 / 碎裂；确诊 KOH/PAS/培养，治疗**口服** terbinafine 特比萘芬。*

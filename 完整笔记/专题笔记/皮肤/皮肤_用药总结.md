## 〇、核心用药原则（先背这几条）

> [!danger] CK 高频决策红线
> 1. **可疑恶性病灶 → 先活检 (biopsy first)，不要先经验性用药**（详见 [[完整笔记/专题笔记/皮肤/皮肤_皮肤肿瘤与癌前]]）。
> 2. **银屑病 (Psoriasis, 银屑病) 禁用系统性糖皮质激素 (systemic corticosteroids)** → 停药反跳诱发**脓疱型银屑病 (pustular psoriasis)**。
> 3. **Isotretinoin (异维A酸)** = 强致畸 (teratogen)，用前 **iPLEDGE + 两种避孕 + 两次 β-hCG**；查 **肝功 + 空腹血脂**。
> 4. 外用糖皮质激素**部位决定效价**：面部/皱褶/婴儿 → 只用**低效价 (low-potency, 如 hydrocortisone 氢化可的松)**；掌跖/苔藓化斑块 → 高效价。
> 5. **Nystatin (制霉菌素) 只覆盖念珠菌 (Candida)，对皮肤癣菌 (dermatophyte/tinea) 无效** → 体股足癣要用 azole 或 allylamine。
> 6. 外用药选择跟着**病变形态**走：湿→干（渗出用湿敷/收敛），干→润（苔藓化用软膏 ointment 封包）。

## 一、外用糖皮质激素 Topical Corticosteroids（外用激素）

> [!tip] 用药原理
> 结合胞内**糖皮质激素受体**→ 抑制 **NF-κB / AP-1** → 下调 IL、TNF-α、磷脂酶 A2（↓花生四烯酸→↓前列腺素/白三烯）→ **抗炎 + 免疫抑制 + 抗增殖 + 缩血管**。是湿疹、接触性皮炎、银屑病斑块的一线外用。

**效价阶梯（记两头即可）：**

```
超强效 Class I ── clobetasol propionate (氯倍他索) / betamethasone dipropionate
   高效       ── fluocinonide (氟轻松) / triamcinolone 0.5%
   中效       ── triamcinolone acetonide 0.1% (曲安奈德)
   低效 Class VII ── hydrocortisone 1-2.5% (氢化可的松) ← 面部/皱褶/婴儿唯一安全区
```

> [!warning] 部位 × 剂型原则
> - **面部、腋窝、腹股沟、婴儿** → 皮肤薄、吸收强 → **只用低效价**，疗程短（否则萎缩 atrophy、毛细血管扩张 telangiectasia、口周皮炎 perioral dermatitis、类固醇痤疮）。
> - **剂型**：软膏 (ointment) 封闭性强、最适**干燥/苔藓化**；乳膏 (cream) 通用；洗剂/凝胶 (lotion/gel) 用于**毛发区/渗出**。
> - **长期大面积** → 经皮吸收可致 **HPA 轴抑制（下丘脑-垂体-肾上腺轴 hypothalamic-pituitary-adrenal axis suppression）**、库欣样表现、生长抑制（儿童）。

> [!danger] 局部副作用速记
> 皮肤**萎缩 / 萎缩纹 (striae) / 毛细血管扩张 / 痤疮样疹 / 口周皮炎 / tinea incognito（激素误用于真菌感染→掩盖体征、皮损扩大）**。

## 二、维生素 D₃ 类似物 & 外用维 A 酸 Vitamin D analogs & Topical Retinoids

| 药 | 中文 | 用药原理 (机制) | 主要用途 |
|---|---|---|---|
| **Calcipotriene / Calcitriol** | 卡泊三醇 / 骨化三醇 | 激活**维生素 D 受体** → 抑制角质细胞**过度增殖** + 促分化 + 免疫调节 | 银屑病斑块（常与外用激素**联用/交替**，减少各自副作用） |
| **Tretinoin / Adapalene / Tazarotene** | 维A酸 / 阿达帕林 / 他扎罗汀 | 激活**视黄酸受体 (RAR)** → 正常化滤泡角化、**溶解粉刺 (comedolytic)**、抗炎、促胶原 | 痤疮（**粉刺 comedone 一线**）、银屑病 (tazarotene)、光老化 |

> [!tip] 用药要点
> - 外用维 A 酸初期**刺激/脱屑/光敏** → 夜间用、少量、加保湿；**孕妇避免** tazarotene（X 类）。
> - Calcipotriene **不要超量**（>100 g/周有高钙血症风险），**别涂面部**（刺激）。

## 三、痤疮阶梯用药 Acne（按机制拆解四大靶点）

> [!info] Acne 四大发病机制 → 四类药各打一环
> ① 滤泡角化过度 ② 皮脂↑（雄激素驱动）③ **Cutibacterium acnes (痤疮丙酸杆菌)** 定植 ④ 炎症。
> 详见 [[完整笔记/专题笔记/皮肤/皮肤_毛囊皮脂腺与附属器]]。

```
轻度粉刺型      → 外用维A酸 (①) ± benzoyl peroxide (③④)
轻中度炎性丘疹脓疱 → + 外用抗生素 clindamycin(②③) 【必配 BPO 防耐药】
中重度          → + 口服抗生素 doxycycline / minocycline (③④, 抗炎)
女性激素型/顽固   → + 螺内酯 spironolactone (②抗雄) 或 口服避孕药 (COC)
重度结节囊肿/瘢痕  → 口服 isotretinoin (①②③④ 全打)
```

| 药 | 中文 | 用药原理 |
|---|---|---|
| **Benzoyl peroxide** | 过氧化苯甲酰 | 释放自由基**杀 C. acnes** + 轻度角质剥脱；**无耐药** → 与外用/口服抗生素同用防耐药 |
| **Clindamycin (topical)** | 克林霉素 | 抑菌 + 抗炎；**单用易耐药**，必须配 BPO |
| **Doxycycline / Minocycline** | 多西环素 / 米诺环素 | 抑菌 + **抗炎**（抑中性粒趋化）；doxycycline **光敏**（见 [[完整笔记/专题笔记/皮肤/皮肤_光敏性药物反应]]）；孕妇/<8 岁禁（牙着色） |
| **Spironolactone** | 螺内酯 | **抗雄激素**（阻断雄激素受体 + ↓睾酮合成）→ ↓皮脂；查钾，孕妇避免 |
| **Isotretinoin** | 异维A酸 | 口服维 A 酸：**缩小皮脂腺 + 正常化角化 + 抗炎 + 间接↓C. acnes** = 唯一能改变病程 |

> [!danger] Isotretinoin 监测（CK 常考）
> - **强致畸** → iPLEDGE，用前后各需避孕 1 月 + 用前 2 次 β-hCG（−）。
> - 查 **肝功 (LFT) + 空腹血脂 (甘油三酯↑)**；干燥/唇炎/夜盲；情绪变化需问诊。
> - **禁与四环素类 (tetracycline) 同服** → 叠加**假性脑瘤 / 特发性颅内高压 (pseudotumor cerebri / idiopathic intracranial hypertension, IIH)** 风险。

## 四、银屑病系统治疗 & 生物制剂 Psoriasis systemics（详见 [[完整笔记/专题笔记/皮肤/皮肤_银屑病]]）

```
局限   → 外用高效激素 + calcipotriene（一线）
广泛   → 窄谱 UVB 光疗 (NB-UVB)
中重度系统 → MTX / acitretin / cyclosporine / apremilast
难治/银屑病关节炎 (PsA, psoriatic arthritis) → 生物制剂
```

| 药 | 中文 | 用药原理 | 关键监测/陷阱 |
|---|---|---|---|
| **Methotrexate** | 甲氨蝶呤 | 抑二氢叶酸还原酶 → 抗增殖 + 抗炎 | **叶酸补充**；肝纤维化、骨髓抑制、致畸；查全血细胞计数 (CBC, complete blood count) / 肝功 (LFT) |
| **Acitretin** | 阿维A酸 | 系统维 A 酸，调节角化 | **强致畸，停药后 3 年内避孕**（乙醇会延长半衰期）；唇炎、血脂 |
| **Cyclosporine** | 环孢素 | 抑钙调磷酸酶 → ↓IL-2、抑 T 细胞 | 肾毒、高血压；**快速起效**，短期用 |
| **Apremilast** | 阿普斯特 | **磷酸二酯酶4抑制剂 (PDE4, phosphodiesterase-4 inhibitor)** → ↑cAMP → ↓TNF/IL-23 | 口服、较安全；腹泻、体重↓、抑郁 |
| **TNF-α 抑制剂** (adalimumab, etanercept, infliximab) | — | 中和 TNF-α | 用前查 **结核 (PPD/IGRA) + 乙肝**；避免活疫苗 |
| **IL-17 (secukinumab) / IL-23 (ustekinumab, guselkumab)** | — | 阻断 Th17 轴关键细胞因子 | IL-17 者禁用于**炎症性肠病 (IBD, inflammatory bowel disease)**（可诱发/加重） |

> [!danger] 反复考点：银屑病急性暴发/进展性 → 查 **HIV、链球菌感染（点滴型）、诱因药物**（β 阻滞剂、锂、抗疟药、系统激素撤药）。

## 五、抗真菌药 Antifungals（外用 vs 系统 + 谱）

> [!info] 用药原理（两大类）
> - **Azoles (唑类：clotrimazole 克霉唑 / ketoconazole 酮康唑 / fluconazole 氟康唑 / itraconazole 伊曲康唑)**：抑 **14-α-去甲基酶 (CYP)** → 阻麦角固醇 (ergosterol) 合成 → 膜缺陷。**广谱（癣菌 + 念珠菌）**。
> - **Allylamines (丙烯胺类：terbinafine 特比萘芬)**：抑 **角鲨烯环氧化酶 (squalene epoxidase)** → 麦角固醇↓ + 毒性角鲨烯堆积 → 杀菌。**对皮肤癣菌最强**。
> - **Griseofulvin (灰黄霉素)**：破坏微管纺锤体，沉积于角蛋白。
> - **Nystatin (制霉菌素) / Amphotericin**：多烯类，结合麦角固醇打孔。**只覆盖念珠菌，无抗癣菌活性**。

| 感染 | 首选 | 原理/陷阱 |
|---|---|---|
| **体癣/股癣/足癣 tinea** (局限) | 外用 terbinafine 或 azole | **Nystatin 无效**（癣菌） |
| **头癣 tinea capitis** | **口服** griseofulvin 或 terbinafine | 外用穿不透毛干 → **必须口服** |
| **甲真菌病 onychomycosis** | 口服 terbinafine（一线） | 疗程长（指 6 周 / 趾 12 周）；先**KOH/培养确诊**再治；查 LFT |
| **花斑癣 tinea versicolor** | 外用 selenium sulfide / 酮康唑洗剂 | 马拉色菌；**减色斑晒不黑** |
| **皮肤念珠菌/间擦疹** | 外用 nystatin 或 azole | 潮湿皱褶；控危险因素（糖尿病、潮湿） |

> [!warning] 系统唑类 = **CYP450 抑制剂** → 与他汀、华法林、多种药相互作用；itraconazole 负性肌力→**心衰禁用**；口服酮康唑肝毒**已基本弃用系统**。

## 六、皮肤抗菌 / 抗病毒 Antibacterials & Antivirals

| 药 | 中文 | 用药原理 | 用途 |
|---|---|---|---|
| **Mupirocin (topical)** | 莫匹罗星 | 抑异亮氨酸 tRNA 合成酶 | **局限性脓疱疮 (impetigo, 脓疱疮)** 一线；MRSA 鼻腔去定植 |
| **Cephalexin / Dicloxacillin** | 头孢氨苄 / 双氯西林 | β-内酰胺，抗甲氧西林敏感金葡菌 (MSSA) + A 组链球菌 (GAS, group A Streptococcus) | 蜂窝织炎/广泛脓疱疮（口服） |
| **TMP-SMX / Doxycycline / Clindamycin** | 复方磺胺 / 多西环素 / 克林霉素 | 覆盖**社区获得性耐甲氧西林金葡菌 (CA-MRSA, community-associated methicillin-resistant _S. aureus_)** | 有脓（脓肿/化脓性蜂窝织炎） |
| **Acyclovir / Valacyclovir** | 阿昔洛韦 / 伐昔洛韦 | 病毒**胸苷激酶**磷酸化激活 → 抑 DNA 聚合酶（选择性作用于感染细胞） | 单纯疱疹病毒 (HSV, herpes simplex virus)、水痘-带状疱疹病毒 (VZV, varicella-zoster virus)；带状疱疹 **72h 内**起效最好；保证水化防肾结晶 |

## 十二、自测（答完再判分）

### Q1.
一名 17 岁女性因中重度炎性痤疮就诊，外用维 A 酸 + 过氧化苯甲酰 + 口服多西环素治疗 4 个月效果不佳，面颊仍有炎性丘疹、下颌区加重、月经前明显。查体无粉刺为主的病灶。下一步最合适的口服药物是？

A. 增加多西环素剂量
B. 螺内酯 (spironolactone)
C. 口服泼尼松
D. 口服酮康唑
E. 立即开始异维A酸而不做任何检查

### Q2.
一名 6 岁男孩头皮出现边界不清的脱发斑，伴断发黑点、局部鳞屑及枕后淋巴结肿大。KOH 见发内孢子。最合适的治疗是？

A. 外用 terbinafine 乳膏
B. 外用 nystatin
C. 口服 griseofulvin
D. 局部糖皮质激素
E. 外用 ketoconazole 洗剂单用

### Q3.
一名 28 岁女性银屑病斑块广泛，拟开始甲氨蝶呤。用药前**最重要**的常规补充/监测是？

A. 补钾
B. 补叶酸 + 查 CBC/肝功
C. 查 G6PD
D. 查空腹血脂
E. 每周查 β-hCG

---

> [!success]- 判分
> **Q1 = B**。已足量外用+口服抗生素无效、经前加重、下颌分布、非粉刺为主 → 激素型痤疮 → **螺内酯抗雄**。异维A酸也是合理升级，但选项 E "不做检查即开始"错（须避孕+β-hCG+血脂肝功）；泼尼松/酮康唑无适应证。
> **Q2 = C**。头癣穿不透毛干 → **必须口服**（griseofulvin 或 terbinafine）；外用一律无效；nystatin 更是只打念珠菌。
> **Q3 = B**。MTX 抑二氢叶酸还原酶 → **补叶酸**减毒 + 监测骨髓/肝毒（CBC/LFT）。G6PD 是 dapsone；空腹血脂/β-hCG 是维 A 酸类。

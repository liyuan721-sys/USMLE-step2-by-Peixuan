# 色素异常 Pigmentary Disorders 详解

> [!info] 一句话定位
> 减色/脱色斑先分**完全脱色 vs 部分减色**：**界清完全脱色 = 白癜风 (vitiligo) → 自身免疫 → 查甲状腺 (TSH)**；**部分减色 + 细鳞屑 + 晒不黑 = 花斑癣 (tinea versicolor)**。增色看黄褐斑/炎症后色沉。机制 + 关联放本篇；速查见 [[完整笔记/专题笔记/皮肤/皮肤_HighYield速查总览]]。

> [!note]- 缩写对照（全称 中英）
> - **TSH** = thyroid-stimulating hormone 促甲状腺激素
> - **T1DM** = type 1 diabetes mellitus 1 型糖尿病
> - **HCV / HIV** = hepatitis C virus / human immunodeficiency virus 丙型肝炎病毒 / 人类免疫缺陷病毒
> - **PIH** = post-inflammatory hyperpigmentation 炎症后色素沉着

---

## 一、白癜风 Vitiligo ⭐

- **机制**：**自身免疫性黑素细胞 (melanocyte) 破坏** → **界清、完全脱色 (depigmented) 斑**；遗传 + 环境也参与。
- **分布**：肢端/面部 (acral/facial)、黏膜、节段型、泛发型；缓慢进展，10–20% 可自发复色。
- **自身免疫关联（高频考点）**：**自身免疫甲状腺病（Hashimoto 桥本甲状腺炎 / Graves 病）最确立 → 推荐查血清 TSH**；也可伴恶性贫血、T1DM、Addison 病、斑秃、Sjögren 综合征。
- **治疗**：局限 → 外用糖皮质激素 / 钙调磷酸酶抑制剂 (tacrolimus 他克莫司)；广泛/难治 → 口服激素、光疗 (phototherapy)。

> [!danger] 白癜风首查 TSH，不是 HCV
> **白癜风 = 自身免疫**，最该筛**甲状腺 (TSH)**。
> - **HCV 丙肝抗体**对应的皮病是 **PCT 迟发性皮肤卟啉病 / 扁平苔藓 / 冷球蛋白血症性血管炎**，**不是白癜风** → 选 HCV 是关联错位。
> - **HIV** 增加的是**花斑癣（部分减色）**，白癜风是**完全脱色**。
> - HbA1c（T1DM）、PTH（甲旁减）发生率低于甲状腺病，仅在有症状时查。

---

## 二、减色 vs 脱色鉴别

| 病 | 颜色改变 | 部位 | 钥匙 | 处理 |
|---|---|---|---|---|
| **白癜风 Vitiligo** | **完全脱色**（界清）| 肢端/面/黏膜 | 自身免疫 → **查 TSH** | 外用激素 / tacrolimus / 光疗 |
| **花斑癣 Tinea versicolor** | **部分减色/着色**（晒不黑）| 上躯干 | 细鳞屑、KOH "意大利面+肉丸" | selenium sulfide 二硫化硒 / ketoconazole 酮康唑 |
| **炎症后色素减退** | 减色 | 原炎症处 | 既往皮炎/损伤史 | 治原病、时间恢复 |
| **白色糠疹 Pityriasis alba** | 轻度减色 + 细鳞 | 儿童面颊 | 特应性背景 | 润肤 ± 弱效激素 |

## 三、增色（简表）

- **黄褐斑 Melasma**：面部对称褐斑，**妊娠/口服避孕/日晒**诱发 → 防晒 + 外用氢醌 (hydroquinone) / 维 A 酸。
- **炎症后色素沉着 PIH**：原炎症/损伤处褐斑，深肤色多见 → 防晒 + 时间，外用美白。
- **黑棘皮病 Acanthosis nigricans**：颈/腋天鹅绒褐色增厚 → 年轻=胰岛素抵抗，老年突发=查 GI/GU 癌（详见 HighYield 总览）。

---

## 四、易混陷阱（普适）

- **完全脱色 = 白癜风（查 TSH）**；**部分减色晒不黑 = 花斑癣（KOH）**。
- **白癜风查甲状腺，不查 HCV/HIV**；HCV→PCT/LP，HIV→花斑癣。
- 黄褐斑认**妊娠/避孕/日晒**三诱因。

---

## 五、自测（答完再判分）

> [!question]- Q1. 22 岁女性 2 年手/足/面界清完全脱色斑、缓慢进展、无痛痒。推荐检查？
> **血清 TSH**（白癜风 = 自身免疫，最常关联自身免疫甲状腺病）。**不**查 HCV（那是 PCT/LP）。

> [!question]- Q2. 上躯干色素不均斑、晒不黑、细鳞屑、KOH 见"意大利面+肉丸"。诊断 + 处理？
> **花斑癣 tinea versicolor** → selenium sulfide 二硫化硒 / 酮康唑。

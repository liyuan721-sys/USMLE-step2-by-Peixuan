# 光敏性药物反应 Drug-Induced Photosensitivity

> [!info] 一句话定位
> **服某药后短暂日晒 → 暴露区（面/颈/胸 V 区/手背前臂）皮损**。分两型：**光毒性 (phototoxic)**（像夸张晒伤、即刻、无需致敏、最常见）与**光变态反应 (photoallergic)**（湿疹样、迟发、需预先致敏）。最高频考点药 = **四环素类 tetracyclines（如多西环素 doxycycline）**。机制 + 药谱放本篇；速查见 [[完整笔记/专题笔记/皮肤/皮肤_HighYield速查总览]]。

---

## 一、光毒性 vs 光变态反应 对照 ⭐

| 特征 | **光毒性反应 Phototoxic** | **光变态反应 Photoallergic** |
|---|---|---|
| 机制 | 药物代谢物 + 紫外线 (UV) → 产生**活性氧 (reactive oxygen species, ROS)** → 直接损伤细胞膜与 DNA | UV 改变药物结构 → 诱发**迟发型 (IV 型) 超敏反应** |
| 需否预先致敏 | **不需要**（首次暴露即可发病） | **需要**（既往致敏后再发） |
| 起病速度 | **即刻～数小时**（快） | **迟发 24–72 小时** |
| 外观 | **夸张的晒伤**：红斑、疼痛、水肿、**水疱/大疱** | **湿疹样 (eczematous)**：红斑、丘疹、脱屑、瘙痒 |
| 分布 | 严格限**日晒暴露区** | 多在暴露区，**可扩散至非暴露区** |
| 剂量相关 | 是（剂量依赖） | 否（免疫介导） |
| 频率 | **更常见** | 较少见 |

> [!tip] 一句话辨别
> **像晒伤、晒后立刻起、痛 + 水疱 → 光毒性**；**像湿疹、晒后 1–3 天起、痒 + 脱屑、会蔓延 → 光变态反应**。

---

## 二、常见致光敏药物谱 ⭐

> [!warning] 高频药表（UWorld 同款）
> 考试最爱：**抗生素（四环素类）+ 噻嗪类利尿剂 + 胺碘酮**。

| 类别 | 代表药 |
|---|---|
| **抗生素 Antibiotics** | **四环素类 tetracyclines（多西环素 doxycycline）**；磺胺类 sulfonamides；氟喹诺酮 fluoroquinolones |
| **抗精神病药 Antipsychotics** | 氯丙嗪 chlorpromazine、丙氯拉嗪 prochlorperazine、异丙嗪 promethazine |
| **利尿剂 Diuretics** | 呋塞米 furosemide、氢氯噻嗪 hydrochlorothiazide (HCTZ) |
| **抗心律失常 Antiarrhythmic** | 胺碘酮 amiodarone |
| **非甾体抗炎药 NSAID** | 吡罗昔康 piroxicam |
| 其他 | 维 A 酸类 retinoids、伏立康唑 voriconazole、补骨脂素 psoralen |

## 三、临床要点 + 处理

> [!note] 钥匙
> - **痤疮患者用多西环素 doxycycline → 日晒 → 暴露区夸张晒伤样皮损 = 光毒性药疹**（四环素类是痤疮常用药、也是光敏最常见原因）。
> - **诱发所需日晒量可远小于普通晒伤** —— 短时暴露（如海滩 1 小时）即可发病。
> - **处理**：① **减少日晒** + 防晒霜 (sunscreen) + 防护衣物；② 识别并**停用/更换致敏药**；③ 对症（重者可短期外用激素）。

> [!example]- Q2771 题干线索 → 临床意义
> - "**痤疮新开始新药 (doxycycline 多西环素)**" → 提示四环素类，最常见光敏药。
> - "**海滩 1 小时即起灼烧感**" → 即刻起病、小剂量日晒即诱发 = **光毒性**（非需致敏的光变态）。
> - "面/颈/上胸/手背前臂（**日晒暴露区**）红斑水肿水疱" → 严格限暴露区 = 光毒性分布。
> - "**无黏膜病变**" → 排除 SJS。

> [!danger] 易混陷阱
> - **Stevens-Johnson 综合征 (SJS) 史蒂文斯-约翰逊综合征**：高热 + **黏膜糜烂（口/眼）** + 靶形/大疱 + Nikolsky+；诱因磺胺/抗癫痫药。**有黏膜 = 不是单纯光敏**。
> - **Allergic contact dermatitis 变应性接触性皮炎**：皮损限**药物涂抹处**、瘙痒、需接触；不会累及未涂药的颈/胸。
> - **Benzoyl peroxide 过氧化苯甲酰**：也可致光敏，但远少于四环素，且不累及未涂药区。

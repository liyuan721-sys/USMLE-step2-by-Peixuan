# 脱发鉴别 Alopecia / Hair Loss 速查

> [!info] 一句话定位
> 脱发题三步走：① **瘢痕性 vs 非瘢痕性**（钥匙 = 毛囊口还在吗）→ ② **弥漫 vs 局限** → ③ **拔发试验 (hair pull test) + 病史诱因**。USMLE 高频考"产后弥漫脱发 = 休止期脱发"和"女性型脱发首选外用米诺地尔"两个反射。

> [!note]- 缩写对照（全称 中英）
> - **TE** = telogen effluvium 休止期脱发
> - **AA** = alopecia areata 斑秃
> - **AGA** = androgenetic alopecia 雄激素性脱发
> - **FPHL / MPHL** = female / male pattern hair loss 女性 / 男性型脱发
> - **DHT** = dihydrotestosterone 双氢睾酮
> - **DLE** = discoid lupus erythematosus 盘状红斑狼疮
> - **PCOS** = polycystic ovary syndrome 多囊卵巢综合征
> - **TSH** = thyroid-stimulating hormone 促甲状腺激素

---

## 一、第一刀：瘢痕性 vs 非瘢痕性（钥匙 = 毛囊口 follicular ostia）

> [!tip] 决策核心
> - **非瘢痕性 (non-scarring)**：毛囊口**保留** → **可逆**。绝大多数考题（TE / AA / AGA / 头癣 / 拔毛癖）。
> - **瘢痕性 (scarring / cicatricial)**：毛囊口**消失** + 萎缩 → **永久**。见 **DLE 盘状红斑狼疮**（红斑鳞屑斑块 + 毛囊角栓 + 中央萎缩色脱）、扁平苔藓样 (lichen planopilaris)。

```
脱发
├── 非瘢痕性（毛囊口在，可逆）
│   ├── 弥漫 ── Telogen effluvium（产后/手术/重病/精神，pull test+）
│   │         └ 雄激素性 AGA（顶部/中线渐进，pull test−）
│   └── 局限 ── Alopecia areata（光滑、惊叹号发、无鳞屑）
│             ├ Tinea capitis（儿童、鳞屑/黑点/枕淋巴结/Kerion）
│             ├ Trichotillomania（不规则、断发长短不一、心理）
│             └ Trichorrhexis nodosa（发干脆断、理化损伤）
└── 瘢痕性（毛囊口消失，永久）
    └── DLE 盘状狼疮 / 扁平苔藓样 lichen planopilaris
```

## 二、各病鉴别总表 ⭐

| 病 | 分布 | 头皮外观 | 关键钩子 | 拔发试验 | 处理 |
|---|---|---|---|---|---|
| **Telogen effluvium 休止期脱发** | **弥漫**、非瘢痕 | 正常（无红斑/鳞屑）| **应激事件后 2-4 月**：产后 / 手术 / 重病高热 / 精神创伤 / 骤减体重；发干正常 | **阳性 (>10-15% 拔出)** | **找并去除诱因 + 安抚 (reassurance)**；自限，可达 1 年恢复 |
| **Androgenetic alopecia (AGA) / 女性型 FPHL** | **顶部 + 中线** 渐进性变细（女性保留发际线；男性发际线后退 + 顶部）| 正常 | 慢性进行性、**毛囊微小化 (follicular miniaturization)**；女性雄激素多正常（少数 PCOS 等高雄）| 阴性 | **女性首选 topical minoxidil 外用米诺地尔**；男性 finasteride 非那雄胺 + minoxidil；**finasteride 在女性证据不足/不常规** |
| **Alopecia areata 斑秃** | **局限**圆斑、非瘢痕 | **光滑如瓷**，无红斑鳞屑 | 自身免疫；**惊叹号发 (exclamation point hairs)**、甲凹点；可全秃/普秃 | 活动缘阳性 | **病灶内糖皮质激素 (intralesional steroids)** |
| **Tinea capitis 头癣** | 局限、鳞屑性 | **炎性**：鳞屑/红斑/脓疱(Kerion) | **儿童**；**黑点（断发 black dots）** + **枕部淋巴结肿大** + 瘙痒；Wood 灯(小孢子菌)绿荧光 | — | **必须口服**：griseofulvin 灰黄霉素 / terbinafine 特比萘芬（外用渗不进毛囊）|
| **Trichotillomania 拔毛癖** | 不规则斑、非瘢痕 | 正常 | **断发长短不一**、边界怪异；强迫性拔发、心理疾病；可致毛石 (trichobezoar) | — | 心理治疗 (CBT) |
| **Trichorrhexis nodosa 结节性脆发** | 弥漫/局限 | 正常 | 发干**脆性增加、结节状断裂**；过度烫染/热/盐水等理化损伤 | — | 减少理化损伤 |
| **DLE 盘状狼疮（瘢痕性）** | 局限、**瘢痕性** | **红斑鳞屑斑块 + 毛囊角栓 + 中央萎缩色脱** | 毛囊口消失 = **永久**；面部界清瘢痕斑块 | — | 防晒 + 外用/病灶内激素 + 抗疟药（见风湿_SLE）|

## 三、两个高频反射（本组错题核心）

> [!danger] 反射 1：产后弥漫脱发 = Telogen effluvium，不是 Alopecia areata
> 产后 / 大手术 / 重病 / 精神创伤后数月**弥漫性、非炎性**脱发，**头皮和发干外观正常**，拔发试验**一拔一大把 (>10-15%)** → **休止期脱发 (telogen effluvium)**。
> - **AA 斑秃** = **局限**圆斑 + 惊叹号发，不是弥漫；
> - **AGA** = 顶部/中线**渐进**变细，不是急性弥漫脱落、拔发试验阴性。
> - 处理 = **去除诱因 + 安抚**（自限，可达 1 年），不需要激素。

> [!danger] 反射 2：女性型脱发 (FPHL) 首选 topical minoxidil，不是 oral finasteride
> 绝经后女性顶部/中线渐进性变细、发际线保留、无炎症 → **女性型脱发 (FPHL)**。
> - **一线 = topical minoxidil 外用米诺地尔**。机制：本是**ATP 敏感性钾通道开放剂（K_ATP channel opener）**的降压药，经磺基转移酶转为活性 minoxidil sulfate → 开放毛囊 K_ATP 通道 → **延长毛囊生长期 (anagen)、增大毛囊**（旧说法"扩血管增头皮血流"只是其一）；口服时的**多毛症 (hypertrichosis)** 副作用即外用治脱发的原理。全身药理见 [[完整笔记/专题笔记/心内/心内_降压药副作用速查]] §2.5。
> - **finasteride 非那雄胺（抗雄）在女性证据不足**，仅用于米诺地尔失败 + 有高雄表现者（且育龄女性致畸禁忌）。
> - **reassurance/观察**错：FPHL 不治会进展（≠ 自限的 TE）。
> - terbinafine 治头癣、topical 激素治斑秃/炎症 —— 都不对 FPHL。

## 四、易混陷阱（普适）

- **拔发试验阳性 → 想 TE**（休止期毛大量进入脱落期）；AGA / AA 缘外拔发多阴性。
- **"自限可观察" 只对 TE**；FPHL / AGA 不治会进展，AA 多需治疗。
- **儿童局限脱发 + 鳞屑/黑点/枕淋巴结 → 头癣**（口服抗真菌），别当斑秃（光滑无鳞屑）。
- **光滑无瘢痕 + 惊叹号发 = AA**；**红斑鳞屑 + 毛囊角栓 + 中央萎缩 = 瘢痕性（DLE）**。
- **断发长短不一 + 心理背景 = 拔毛癖**；**发干脆断 + 理化史 = 结节性脆发**。

---

## 五、自测（答完再判分）

> [!question]- Q1. 23 岁女性产后 4 个月洗头时大量脱发，头皮无红斑鳞屑，拔发试验 >20% 拔出。诊断 + 处理？
> **Telogen effluvium 休止期脱发**（产后为经典诱因）→ **去除/解释诱因 + 安抚**，自限（可达 1 年恢复），无需激素。

> [!question]- Q2. 53 岁绝经后女性顶部与中线头发渐进变细、发际线保留、无炎症、无多毛。下一步最佳处理？
> **Topical minoxidil 外用米诺地尔**（FPHL 一线）。**不**首选口服 finasteride（女性证据不足），也不能只"观察"（会进展）。

> [!question]- Q3. 局限圆形脱发斑、头皮光滑如瓷、边缘见近端细远端粗的惊叹号发。诊断 + 一线治疗？
> **Alopecia areata 斑秃** → **病灶内糖皮质激素**。

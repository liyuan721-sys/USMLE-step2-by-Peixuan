# 银屑病 Psoriasis 总览

> [!info] 一句话定位
> Psoriasis (银屑病) = T 细胞介导的慢性炎症，表皮过度增殖 → **伸侧 (extensor)、边界清楚、银白色鳞屑斑块 (silvery scale plaques)**，慢性复发病程。考点三连：**形态识别 + 治疗阶梯 + 快速进展查继发因素**。

---

## 〇、亚型总览速查表 Subtype Overview ⭐

> [!info] 全局速查（机制 / 临床 / 治疗一张表）
> 下表是全文浓缩；机制详见 § 三、危险因素详见 § 四、治疗与预后详见 § 五–六。

| 亚型 | 发病机制 Mechanism | 临床表现 Presentation | 治疗 Treatment |
|---|---|---|---|
| **斑块型 Chronic plaque** | IL-23/Th17 主轴（最经典）| 伸侧 (extensor) 界清红斑 + **银白鳞屑**、Auspitz 征、Köbner 现象、甲凹点；慢性 | **分级阶梯**：局限→外用强效激素 + 维生素 D 衍生物 (calcipotriene 卡泊三醇)；中重度→光疗 / MTX (methotrexate 甲氨蝶呤) / apremilast (阿普斯特) / 生物制剂 |
| **点滴型 Guttate** | IL-23/Th17 + **分子模拟 (molecular mimicry)**（链球菌 M 蛋白 ↔ 角蛋白）| 儿童/青年，**链球菌咽炎后 1-3 周**，躯干密集 **<1cm 小水滴样**鳞屑斑 | **常自限**；广泛者首选**窄谱中波紫外线光疗 (narrowband UVB, nbUVB)**；查治链球菌 |
| **脓疱型 GPP (generalized pustular psoriasis, 泛发性脓疱型)** | **IL-36 / 固有免疫 (innate immunity)**（IL36RN → DITRA）—— 机制异类 | **全身红斑上无菌脓疱 (sterile pustules) + 发热**；**停系统激素 / 妊娠 / 低钙**诱发；**急症** | **急症**：支持治疗（补液/电解质/防感染）+ **acitretin (阿维 A) 一线**、cyclosporine (环孢素) / infliximab (英夫利昔单抗) 快控；急性发作用 **spesolimab (佩索利单抗，抗 IL-36R)** |
| **红皮病型 Erythrodermic** | IL-23/Th17（全身泛发）| **全身 (>90% BSA) 弥漫红斑脱屑**，体温/体液失调；停激素、过度光疗诱发；**急症** | **急症**：支持治疗 + **快起效药 cyclosporine (环孢素) / infliximab (英夫利昔单抗)**，其次 acitretin / MTX |
| **反向型 Inverse** | IL-23/Th17（皱褶部位）| 腋 / 腹股沟等**皱褶区光滑红斑、鳞屑极少**；肥胖、摩擦、念珠菌 (Candida) 加重 | 皱褶薄皮区**避强效激素（防萎缩 atrophy）**；用低效价激素、维生素 D 衍生物、**钙调磷酸酶抑制剂 (tacrolimus 他克莫司)** |

![[Pasted image 20260629220518.png|360]]
*银屑病五大亚型 (psoriasis subtypes) 一图对照：**大斑块型 (large plaque)** 厚鳞屑、伸侧；**小斑块/点滴型 (small plaque / guttate)** <1cm、链球菌咽炎后；**脓疱型 (pustular)** 痛性斑块 + 脓疱 ± 系统症状、掌跖；**红皮病型 (erythrodermic)** >75% BSA 重度泛发红斑、可由药物/感染诱发（如系统激素、HIV）。*

> [!tip] 一行抓全表
> **机制**：除脓疱型 (IL-36) 外全是 IL-23/Th17 → 点滴型多个"分子模拟"触发。**临床**：看鳞屑（斑块银白 / 点滴小水滴 / 反向几乎无）、看脓疱 (GPP)、看泛发（红皮病）。**治疗**：斑块阶梯化、点滴自限+光疗、两急症型快起效药（环孢素/英夫利昔）+支持、反向避强效激素。**全程忌系统性糖皮质激素。**

## 一、临床识别 Presentation

> [!tip] 经典指纹（一眼锁定）
> **Extensor surfaces (伸侧) + silvery scale (银白鳞屑) + nail pitting (甲凹点) + chronic recurrent (慢性复发)**

- **皮损形态**：边界清楚的红斑 + **银白色 / 灰色鳞屑斑块 (silvery/gray scaly plaques)**，对称分布
- **好发部位**：
  - **Large plaque (大斑块)**：背部、**膝/肘伸侧 (extensor knees & elbows)**
  - **Small plaque (<1 cm)**：头皮 (scalp)、**手/足背 (dorsal hands/feet)**、躯干、**臀沟 (gluteal cleft)**
- **指甲**：**nail pitting (甲凹点)**、油滴征、甲剥离

![[{F5DF6C75-F240-416C-A4A4-500D7CC35E5D}.png|360]]
*寻常型银屑病 (psoriasis vulgaris)：膝/肘**伸侧**界清红斑斑块 + **银白色鳞屑 (silvery scale)**——慢性、瘙痒，最经典表现。*

![[{CB79B272-502F-4CB3-9229-31F9806E4EC6}.png|320]]
*斑块型银屑病（另例·深肤色）：膝部**增厚斑块 + 白/银色厚鳞屑**，好发摩擦/受压伸侧（膝、肘）。*

![[{084D21E8-CE9C-4542-8FF7-24B014597DF2}.png|320]]
*斑块型银屑病（另例·手背）：**边界锐利的红斑斑块**——与湿疹"边界模糊"对比，银屑病界清是识别钥匙。*

![[{7D14C5B0-9B15-4137-9D76-53572A322F1B}.png]]
*斑块型银屑病（另例·标注）：界清红斑斑块 + **银白鳞屑 (silvery scale)** + 刮鳞后**点状出血 (Auspitz sign 奥斯皮茨征)**——三大识别钥匙一图全（双嵌自卡 [[mistakes/uworld-mistakes_2026-06#^Q12332]]）。*

### 两大体征 Signs

| 体征 | 英文 | 机制 / 表现 |
|---|---|---|
| **奥斯皮茨征** | **Auspitz sign** | 刮除鳞屑后**点状出血**（真皮乳头血管暴露）|
| **同形反应** | **Köbner phenomenon** | **创伤部位 (trauma site) 出现新皮损** → 好发于反复轻微摩擦区 |

## 二、亚型 Subtypes

```
银屑病亚型
├── Chronic plaque (慢性斑块型) ── 最常见，伸侧银白鳞屑
├── Guttate (点滴型) ────────── post-strep (链球菌咽炎后)，躯干小点滴样鳞屑斑，儿童/青年
├── Erythrodermic (红皮病型) ── 全身弥漫红斑脱屑，可由系统激素停药诱发 (急症)
├── Pustular (脓疱型) ───────── 无菌脓疱，激素停药/妊娠诱发 (急症)
└── Inverse / 反向型 ───────── 皱褶部位 (腋/腹股沟)，鳞屑少
```

> [!warning] 快速进展 = 查继发因素
> Psoriasis 本应**惰性、缓慢进展 (indolent, slowly progressive)**。若题干强调 **rapid onset / 急性加重**，找继发诱因：
> - **HIV 感染**（刺激表皮 + 内皮增殖）→ 新发或暴发性银屑病要查 HIV
> - **Streptococcal pharyngitis (链球菌咽炎)** → 典型诱发 **guttate psoriasis (点滴型)**
> - **药物**：**系统性糖皮质激素停药 (systemic glucocorticoid withdrawal)**、抗疟药 (antimalarials)、锂 (lithium)、β 受体阻滞剂 (beta blocker)

## 三、发病机制 Pathogenesis（按亚型对比）⭐

> [!info] 核心结论
> **大多数亚型共享同一条免疫主轴 (IL-23/Th17)，但脓疱型 (pustular) 机制"另起炉灶"，由 IL-36 / 固有免疫驱动 —— 这是亚型间最重要的机制分水岭，也决定了靶向药不同。**

### 共享主轴：IL-23 / Th17（plaque / guttate / erythrodermic / inverse）

```
树突状细胞 (dendritic cell) → IL-23
        ↓
   Th17 细胞活化
        ↓
 IL-17A / IL-22 + TNF-α
        ↓
角质细胞过度增殖 (keratinocyte hyperproliferation) + 中性粒细胞趋化
```

这条轴解释了为什么这些亚型用同一类生物制剂有效：抗 TNF-α (etanercept 依那西普 / adalimumab 阿达木单抗)、抗 IL-17 (secukinumab 司库奇尤单抗 / ixekizumab 依奇珠单抗)、抗 IL-23/12-23 (guselkumab 古塞奇尤单抗 / ustekinumab 乌司奴单抗)。

### 各亚型机制钩子

| 亚型 | 机制特点（USMLE 钩子）|
|---|---|
| **斑块型 Chronic plaque** | 最经典 IL-23/Th17；银白鳞屑、Auspitz 征、Köbner 现象 |
| **点滴型 Guttate** | **链球菌咽炎后**经 **分子模拟 (molecular mimicry)**：CD8⁺ T 细胞识别链球菌 **M 蛋白**，交叉反应于角蛋白 (keratin) → 与斑块型最大的机制差异在"触发与诱因"，下游仍偏 T 细胞 |
| **红皮病型 Erythrodermic** | 同主轴但全身泛发，伴体温调节 / 体液丢失 → **急症** |
| **反向型 Inverse** | 同主轴，皱褶部位、无鳞屑 |

> [!danger] 真正的机制异类：脓疱型 Pustular (GPP)
> **泛发性脓疱型银屑病 (generalized pustular psoriasis, GPP)** 不是 Th17 驱动，而是 **IL-36 / 固有免疫 (innate immunity)** 驱动：
> - 关键基因：**IL36RN 突变 → DITRA** (Deficiency of IL-36 Receptor Antagonist)；另有 CARD14、AP1S3
> - 失去 IL-36 受体拮抗剂 → IL-36 信号失控 → 大量中性粒细胞聚集成**无菌脓疱 (sterile pustules)**
> - 靶向药也不同：**spesolimab (佩索利单抗，抗 IL-36 受体)** 专用于 GPP，而非抗 IL-17/23
> - von Zumbusch 型：突发发热 + 全身红斑上无菌脓疱，可危及生命

## 四、危险因素 / 发病背景 Risk Factors & Triggers（按亚型对比）⭐

### 通用危险因素（所有亚型共享的"土壤"）

> [!info] 遗传 + 免疫 + 环境三重背景
> - **遗传**：最强易感基因 **HLA-Cw6 (PSORS1)**（尤与早发型、点滴型相关）；阳性家族史
> - **代谢综合征关联**：肥胖 (obesity)、糖尿病、高血压、血脂异常 —— 银屑病是**系统性炎症病**，不只是皮肤病
> - **生活方式**：吸烟 (smoking，尤其脓疱型/掌跖型)、酗酒 (alcohol)、肥胖
> - **Köbner 现象 (同形反应)**：搔抓、外伤、手术、晒伤、纹身处出新皮损 —— 通用诱发/加重机制

### 亚型特异性诱因 / 发病背景

> [!warning] 点滴型 Guttate
> - **背景人群**：儿童 / 青少年 / 青年
> - **核心诱因**：**A 组 β-溶血性链球菌咽炎/扁桃体炎 (group A strep pharyngitis)**（发作前 1-3 周）
> - 也可由 **HIV** 诱发或加重（HIV 患者银屑病常突发或难治）

> [!warning] 脓疱型 Pustular (GPP)
> - **最典型诱因 = 突然停用全身性糖皮质激素 (systemic glucocorticoid withdrawal)**（UW 最爱考的医源性诱因）
> - 其他：感染、低钙血症 (hypocalcemia)、妊娠 → **疱疹样脓疱病 (impetigo herpetiformis)** = 妊娠期 GPP；某些药物
> - 遗传背景：IL36RN (DITRA)、CARD14、AP1S3

> [!warning] 红皮病型 Erythrodermic
> - 常由**已有银屑病的不当处理诱发**：突然停激素、过度光疗、严重晒伤、感染、系统性疾病
> - 也可由 GPP 进展而来 → **急症**

> [!tip] 斑块型 Plaque & 反向型 Inverse
> - 主要靠通用因素（遗传 + 代谢 + Köbner + 应激）
> - 反向型在皱褶/潮湿摩擦部位，**肥胖、摩擦、念珠菌 (Candida) 定植**易加重

### 诱发 / 加重药物（跨亚型通用，UW 高频）

> [!danger] 口诀 "BLAANI"
> - **B**eta-blocker (β 受体阻滞剂)
> - **L**ithium (锂，最经典)
> - **A**ntimalarials (抗疟药：hydroxychloroquine 羟氯喹 / chloroquine 氯喹)
> - **A**CEi (ACE 抑制剂)
> - **N**SAIDs
> - **I**nterferon / imiquimod (咪喹莫特) / TNF-α 抑制剂（矛盾性银屑病 paradoxical psoriasis）
> - ⚠️ **系统性糖皮质激素**：用时压制、**突然停 → 反跳为脓疱型/红皮病型**（故银屑病一般忌用系统激素，见 § 五治疗）

### 速查表

| 亚型 | 典型人群 | 标志性诱因 / 背景 |
|---|---|---|
| **斑块型 Plaque** | 成人，慢性 | 遗传 + 代谢综合征 + Köbner + 应激 |
| **点滴型 Guttate** | 儿童/青年 | **链球菌咽炎**（前 1-3 周）；HIV |
| **脓疱型 GPP** | 各年龄 | **突然停全身激素**；妊娠；低钙；感染 |
| **红皮病型 Erythrodermic** | 已有银屑病者 | 停激素、过度光疗、晒伤、感染（急症）|
| **反向型 Inverse** | 肥胖者 | 皱褶摩擦、潮湿、念珠菌 |

> [!question]- 自测：双相情感障碍患者用药后银屑病皮损明显加重，最可能的药物诱因？
> **锂 (lithium)** —— 最经典的诱发/加重银屑病药物（口诀 BLAANI 的 L）。

> [!question]- 自测：长期口服泼尼松控制其他疾病的患者，骤停激素后全身红斑上出现密集无菌脓疱伴高热。诊断 + 机制？
> **泛发性脓疱型银屑病 (GPP)**，系统激素停药反跳诱发；机制为 **IL-36 / 固有免疫**通路（IL36RN/DITRA），非 Th17 主轴。

## 五、治疗阶梯 Treatment

> [!success] 按受累范围分级（核心考点）

| 分级 | 标准 | 首选 |
|---|---|---|
| **局限性 limited** | 局部 / <5% BSA (body surface area, 体表面积) | **局部高效价糖皮质激素 (topical high-potency corticosteroids)**，如 betamethasone (倍他米松)、fluocinonide (氟轻松) |
| **中重度 moderate-severe** | >5% BSA | **光疗 (UV phototherapy)** 或 **系统治疗**：MTX (methotrexate, 甲氨蝶呤)、apremilast (阿普斯特，PDE4i, 磷酸二酯酶-4 抑制剂)、生物制剂如 etanercept (依那西普, TNF-α 抑制剂) |
| **合并 PsA** | psoriatic arthritis | 直接上 **系统治疗 (systemic)** |

### 局部用药补充

- **Vitamin D derivatives (维生素 D 衍生物)**：calcipotriene (卡泊三醇)
- **Topical retinoids (外用维 A 酸)**
- **Calcineurin inhibitors (钙调磷酸酶抑制剂)**：tacrolimus (他克莫司) — **用于面部 / 皮肤薄处**（避免激素致皮肤萎缩）

> [!danger] 禁用系统性糖皮质激素
> **Systemic glucocorticoids are NOT used.** 停药 (withdrawal) 可诱发**严重红皮病型或脓疱型银屑病 (erythrodermic / pustular psoriasis)** —— 经典送分陷阱。

## 六、亚型特异性治疗 + 预后对比 Subtype-specific Tx & Prognosis ⭐

> [!info] 与 § 五的关系
> § 五是**按 BSA (body surface area, 体表面积) 通用分级**的治疗阶梯；本节补**亚型间的差异** —— 点滴型多自限、两个急症型 (GPP / 红皮病) 要快起效药 + 支持治疗。**全程忌系统性糖皮质激素**（见 § 五 `禁用系统性糖皮质激素` callout）。

### 亚型特异性治疗

| 亚型 | 治疗重点（与通用阶梯的差异）|
|---|---|
| **斑块型 Plaque** | 标准**分级阶梯**：局限→外用强效激素 + 维生素 D 衍生物 (calcipotriene 卡泊三醇)；中重度→光疗或系统（MTX 甲氨蝶呤 / apremilast 阿普斯特 / 生物制剂）|
| **点滴型 Guttate** | **常自限**（首发 / 儿童 post-strep 多在数周-数月消退）；广泛者首选 **窄谱中波紫外线光疗 (narrowband UVB, nbUVB)**；查并治链球菌（抗生素改善皮损证据不强）；反复发作 + 链球菌相关偶考虑扁桃体切除（有争议）|
| **脓疱型 GPP (generalized pustular psoriasis)** | **急症**！支持治疗（补液 / 电解质 / 监测低钙 / 防感染）+ 快起效系统药：**acitretin (阿维 A，口服维 A 酸) 一线**、cyclosporine (环孢素) / infliximab (英夫利昔单抗) 快速控制；急性发作有 **spesolimab (佩索利单抗，抗 IL-36R)** 靶向药 |
| **红皮病型 Erythrodermic** | **急症**！支持治疗（体温调节 / 补液 / 防蛋白丢失 / 防感染）+ 快起效药：cyclosporine (环孢素) 或 infliximab (英夫利昔单抗) 起效最快，其次 acitretin / MTX (methotrexate, 甲氨蝶呤) |
| **反向型 Inverse** | 皱褶 / 薄皮区 → **避免长期强效激素（防皮肤萎缩 atrophy）**；首选低效价激素、维生素 D 衍生物、**钙调磷酸酶抑制剂 (tacrolimus 他克莫司)** |

> [!tip] 起效速度记忆点（急症必备）
> 急症型 (GPP / 红皮病) 要选**快起效**药 → **cyclosporine 环孢素、infliximab 英夫利昔单抗**（数天起效）；**MTX、acitretin 较慢**，适合维持。MTX 不用于急性危及生命的快速控制。

### 预后对比

| 亚型 | 预后 |
|---|---|
| **点滴型 Guttate** | **最好** —— 单次 post-strep 发作常自限完全消退；但约 **1/3** 日后进展为慢性斑块型 |
| **斑块型 Plaque** | 慢性、终身、**复发-缓解 (relapsing-remitting)** 病程；可控不可治愈 |
| **反向型 Inverse** | 慢性但通常可控 |
| **脓疱型 GPP** | **最凶险** —— 可致命：脓毒症 (sepsis)、高输出量心衰 (high-output heart failure)、低钙血症 (hypocalcemia)、大量体液丢失 |
| **红皮病型 Erythrodermic** | **凶险** —— 体温失调、体液 / 蛋白丢失、高输出量心衰、继发感染，可致命 |

> [!warning] 别忽略的系统性预后
> 中重度银屑病是**系统性炎症病** → 与**心血管疾病 (cardiovascular disease, CVD)、代谢综合征 (metabolic syndrome)** 相关，重度患者**全因死亡率升高**。约 **30%** 发展为**银屑病关节炎 (psoriatic arthritis, PsA)**，未治疗致**不可逆关节破坏** → 早识别早系统治疗改善关节预后（详见 § 七）。

> [!question]- 自测：停用泼尼松后全身红斑上出现密集无菌脓疱伴发热、低钙，需快速控制病情。首选哪类系统药？为何不选 MTX？
> **cyclosporine (环孢素) 或 infliximab (英夫利昔单抗)** —— 急症型 (GPP) 需**数天内起效**；**MTX 起效慢**，不适合急性危及生命的快速控制（仅用于后续维持）。

## 七、关节受累 Psoriatic Arthritis (PsA)

> [!tip] PsA 指纹
> **Dactylitis (腊肠指) + nail pitting (甲凹点) + DIP (distal interphalangeal, 远端指间关节) 受累 + 晨僵**；属脊柱关节病 (SpA, spondyloarthritis)，**RF (rheumatoid factor, 类风湿因子) 阴性**。

- **关节炎可先于皮损数年** → 怀疑 PsA 但无明显皮损时，**下一步 = 全面皮肤检查 (comprehensive skin exam)**，找头皮/脐/腋/腹股沟隐匿斑（详见 [[mistakes/uworld-mistakes_2026-06#^Q4581]]）
- 治疗：合并 PsA → 系统治疗（MTX / 生物制剂）

## 八、易混鉴别 DDx

| 疾病 | 鉴别要点 |
|---|---|
| **Sweet syndrome (急性发热性嗜中性皮病)** | **疼痛性水肿性丘疹/斑块**（非银白鳞屑），头颈上肢，伴血液系统恶性肿瘤 |
| **SCLE (亚急性皮肤型红斑狼疮)** | 银屑病样鳞屑斑但**环形 + 光敏 + 日晒区 + anti-Ro/SSA**，不留瘢痕 |
| **Lichen planus (扁平苔藓)** | 屈侧 (flexor) 紫色多角形扁平丘疹，Wickham 纹 |
| **Seborrheic dermatitis (脂溢性皮炎)** | 油腻黄色鳞屑，鼻唇沟/头皮，非银白干燥 |

---

> [!question]- 自测：60 岁男性肘膝伸侧银白鳞屑斑块，刮之点状出血，指甲见点状凹陷，受累 <3% 体表。下一步最佳处理？
> **局部高效价糖皮质激素 (topical high-potency corticosteroid)**（局限性 → 局部治疗起步）。注意：**不**用系统激素（停药诱发脓疱/红皮病型）。

> [!question]- 自测：青年咽痛 2 周后躯干出现密集小点滴样鳞屑斑。诊断 + 关键检查？
> **Guttate psoriasis (点滴型银屑病)**，post-strep 诱发；查 **ASO (antistreptolysin O, 抗链球菌溶血素 O) / 咽拭子链球菌**。

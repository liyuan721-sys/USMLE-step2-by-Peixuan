# 浅表皮肤软组织感染 Superficial Skin Infections 详解

> [!info] 一句话定位
> 按病原四分：**细菌（蜜黄痂/大疱/红肿）/ 真菌（环形中央清+KOH）/ 病毒（成簇水疱/疣状/脐凹）/ 寄生虫（指缝夜痒）**。机制 + 治疗细节放本篇；buzzword 一眼速查见 [[完整笔记/专题笔记/皮肤/皮肤_HighYield速查总览]]。大疱性/剥脱性（SSSS）的"水疱"视角见 [[完整笔记/专题笔记/皮肤/皮肤_水疱大疱病鉴别]]。

> [!note]- 缩写对照（全称 中英）
> - **S. aureus** = Staphylococcus aureus 金黄色葡萄球菌
> - **GAS** = group A Streptococcus（S. pyogenes 化脓链球菌）A 组链球菌
> - **MSSA / MRSA** = methicillin-sensitive / -resistant S. aureus 甲氧西林敏感 / 耐药金葡菌
> - **SSSS** = staphylococcal scalded skin syndrome 葡萄球菌性烫伤样皮肤综合征
> - **KOH** = potassium hydroxide 氢氧化钾（真菌镜检）
> - **HSV / VZV** = herpes simplex / varicella-zoster virus 单纯疱疹 / 水痘-带状疱疹病毒
> - **HPV** = human papillomavirus 人乳头瘤病毒
> - **HFMD** = hand-foot-and-mouth disease 手足口病
> - **I&D** = incision and drainage 切开引流
> - **PSGN** = post-streptococcal glomerulonephritis 链球菌感染后肾小球肾炎

## 一、细菌感染 Bacterial

> [!info] 先看形态 → 定深度 / 病原（USMLE 出题主线）
> **五类形态一眼分诊**：
> - **蜜黄色痂 (honey-colored crust)** → 脓疱疮 impetigo（非大疱性：S. aureus 金葡菌 或 GAS 化脓链球菌）
> - **松弛大疱 + 领圈状脱屑 (collarette)** → 大疱性脓疱疮 bullous impetigo（**仅金葡菌**剥脱毒素）；若**泛发全身 + Nikolsky+ + 幼儿、无黏膜** → **SSSS 葡萄球菌性烫伤样皮肤综合征**（同毒素、血源播散）
> - **红肿热痛片状** → 蜂窝织炎谱，按**边界/深度**分：边界极清 + 耳受累 = 丹毒 erysipelas；边界模糊 = 蜂窝织炎 cellulitis；**剧痛>体征 + 捻发音 + 紫色大疱 + 低血压 = 坏死性筋膜炎 necrotizing fasciitis（急诊）**
> - **穿凿溃疡 / 坏死黑痂** → 穿凿样溃疡 + 厚痂 + **留瘢** = 臁疮 ecthyma（深部, GAS）；出血性脓疱 → 黑痂坏死 + **中性粒减少** = 坏疽性臁疮 ecthyma gangrenosum（**Pseudomonas aeruginosa 铜绿假单胞菌菌血症**）
> - **毛囊为中心脓疱 / 结节** → 毛囊炎 folliculitis → 疖 furuncle → 痈 carbuncle

```
     浅 ◄──────────────── 感染深度（侵袭性谱）────────────────► 深
  ┌────────┬───────────┬───────────┬────────────┬───────────────┐
  表皮       真皮浅层     真皮         深真皮+皮下    筋膜
 脓疱疮      丹毒         臁疮         蜂窝织炎       坏死性筋膜炎
 impetigo   erysipelas   ecthyma     cellulitis    nec. fasciitis
 蜜黄痂      边界极清+耳   穿凿溃疡+留瘢  边界模糊       剧痛>体征+捻发音
 金葡/GAS    GAS          GAS          GAS/金葡菌     紫大疱+低血压→急诊清创

  ── 旁支 ──────────────────────────────────────────────────────
  毒素介导（金葡菌剥脱毒素）：局限=大疱性脓疱疮  /  泛发+Nikolsky+=SSSS
  特殊病原：坏疽性臁疮=铜绿假单胞菌菌血症（免疫低下）；热水浴毛囊炎=铜绿
  毛囊轴：毛囊炎 folliculitis → 疖 furuncle → 痈 carbuncle（→ I&D 切开引流）
```

### 脓疱疮 Impetigo（病原是高频考点）

| | **非大疱性 Nonbullous** | **大疱性 Bullous** |
|---|---|---|
| 病原 | **S. aureus 金葡菌 或 GAS 化脓链球菌** | **仅 S. aureus 金葡菌**（剥脱毒素 A）|
| 皮损 | **蜜黄色结痂 (honey-crusted)** 丘疹脓疱 | 松弛大疱 + 黄疱液 → 破后**领圈状脱屑** |
| 治疗 | 局限：外用 mupirocin 莫匹罗星；广泛：口服（cephalexin 头孢氨苄 / dicloxacillin 双氯西林）| **口服抗生素**（cephalexin）|

> [!danger] 大疱性脓疱疮 = 只金葡菌
> 见到**松弛大疱 + 黄疱液 + 领圈状脱屑** → 病原选 **S. aureus 金葡菌**（毒素）；"金葡菌**或**链球菌"只适用于**非大疱性**。治疗脓疱疮可防风湿热 (rheumatic fever)，**不防 PSGN 链球菌感染后肾炎**。

![[Pasted image 20260629220133.png|300]]
*脓疱疮 (impetigo)（识别供参考）：浅表糜烂上覆**蜜黄色结痂 (honey-colored crust)**——非大疱性多为金葡菌或化脓链球菌 (GAS)。*

![[{9F653F2D-D9E8-4D5B-B9D4-2DA5DA753AAD}.png|360]]
*大疱性脓疱疮 (bullous impetigo)：局部**松弛大疱 + 黄色疱液 (flaccid bullae with yellow fluid)** → 破溃后周边留**领圈状脱屑 (collarette of scale)**——仅由金黄色葡萄球菌 (Staphylococcus aureus) 剥脱毒素引起，口服头孢氨苄 (cephalexin)。（Q16581 题图）*

![[{E6C808D3-AF23-4D32-B84E-BC952F54447E}.png|320]]
*非大疱性脓疱疮 (nonbullous impetigo)（大疱性对照）：丘疹脓疱融合破溃，覆**厚而黏着的蜜黄色结痂 (thick, adherent honey-colored crust)**——可由金黄色葡萄球菌（最常见）**或**化脓链球菌 (Streptococcus pyogenes) 引起；而大疱性**仅**金黄色葡萄球菌。*

### 臁疮 Ecthyma（深部脓疱疮）+ 坏疽性臁疮 Ecthyma gangrenosum

> [!danger] 名字像，是两个完全不同的病——别混
> | | **Ecthyma 臁疮（深部脓疱疮）** | **Ecthyma gangrenosum 坏疽性臁疮** |
> |---|---|---|
> | 病原 | **GAS 化脓链球菌 (group A Streptococcus)** 为主（± 金葡菌继发） | **Pseudomonas aeruginosa 铜绿假单胞菌**菌血症 (bacteremia) |
> | 宿主/背景 | 卫生差 / 糖尿病 / 抓挠、下肢多见 | **中性粒细胞减少 (neutropenia) / 免疫抑制 + 全身脓毒 (sepsis) 重症** |
> | 皮损 | **穿凿样溃疡 (punched-out ulcer) + 厚黏着痂**，**愈合留瘢痕** | 无痛红斑 → 血疱 → **出血性脓疱 → 中央坏死性黑痂溃疡 (necrotic ulcer + black eschar) + 周边红斑** |
> | 治疗 | 清创 + 同 impetigo（口服 cephalexin 头孢氨苄 / dicloxacillin 双氯西林） | 血培养 + **抗假单胞菌**（piperacillin-tazobactam 哌拉西林-他唑巴坦 / cefepime 头孢吡肟 ± 氨基糖苷） |

- **Ecthyma 臁疮**本质 = **累及真皮的"深部 impetigo 脓疱疮"**（impetigo 只在表皮、不留瘢；ecthyma 入真皮、留瘢）。
- **Ecthyma gangrenosum 坏疽性臁疮**虽共用 "ecthyma" 一词，但**机制完全无关**：是 Pseudomonas 菌血症在皮肤的播散灶——见到**中性粒减少/免疫抑制患者 + 出血性脓疱迅速坏死黑痂**，立刻想"假单胞菌脓毒症"，查血培养、上抗假单胞菌经验治疗。

![[{08C8DFBC-4E11-4F47-8D39-FF97B095C49C}.png|300]]
*臁疮 (ecthyma 深脓疱疮)：**穿凿样溃疡 + 厚痂** = 累及真皮的"深部脓疱疮"，GAS 化脓链球菌为主，愈合留瘢。*
![[{58FB1B70-2F4F-4F8F-BE96-2E13371B6946}.png|300]]
*臁疮 (ecthyma，另例)：链球菌性皮肤感染，圆形红斑斑块 + 脓疱/小溃疡，与脓疱疮 (impetigo) 同谱但更深；注意与坏疽性臁疮 (ecthyma gangrenosum，铜绿假单胞菌 Pseudomonas 菌血症)区分。*

![[{7570AFF8-D793-4F7F-8D08-E4A689FD9D39}.png|320]]
*臁疮 (ecthyma)（另例）：肢体多发**红斑斑块 + 溃疡 (erythematous plaques with ulceration)**——链球菌性"深部脓疱疮"，与 impetigo 相关但更深、易留瘢。*

![[皮肤_坏疽性臁疮.png|300]]
*坏疽性臁疮 (ecthyma gangrenosum)：出血性脓疱 → **坏死性黑痂溃疡 + 周边红斑**——铜绿假单胞菌 (Pseudomonas aeruginosa) 菌血症 + 中性粒减少。*

![[{4F5DC0C5-ECFC-474A-850D-862C3A771CA0}.png|320]]
*坏疽性臁疮 (ecthyma gangrenosum)（另例）：**出血性脓疱 (hemorrhagic pustules) + 周围红斑**迅速进展为**坏死性溃疡 (necrotic ulcer)**——铜绿假单胞菌 (Pseudomonas aeruginosa) 菌血症皮肤播散灶，常伴发热等全身症状；本病为坏疽性脓皮病 (pyoderma gangrenosum) 的重要鉴别（但其灶痛更轻、有脓毒背景）。*

### SSSS（毒素血源播散，详见水疱专题）

- 金葡菌剥脱毒素**血源播散** → 全身红斑 + **Nikolsky+** + 浅表（颗粒层）剥脱 + 口周放射状结痂、**无黏膜受累**；幼儿多见 → **静脉抗葡萄球菌（nafcillin 萘夫西林 / vancomycin 万古霉素）+ 支持**。与局限的大疱性脓疱疮"同毒素不同范围"。

### 蜂窝织炎谱（深度/边界鉴别）

| 病 | 深度/边界 | 钩子 | 治疗 |
|---|---|---|---|
| **Erysipelas 丹毒** | 真皮浅层、**边界极清略隆起** | **耳受累** + 寒战高热、GAS | 稳定 dicloxacillin / 不稳 IV nafcillin |
| **Cellulitis 蜂窝织炎** | 深真皮+皮下、**边界模糊** | 伤口/足癣诱因，GAS/金葡菌 | 同上 |
| **Necrotizing fasciitis 坏死性筋膜炎** | 筋膜 | **剧痛(与体征不符)+捻发音+紫色大疱+低血压** | 影像见气体、**别触碰** → 急诊清创 + penicillin 青霉素 + clindamycin 克林霉素 |

### 毛囊炎谱

- **Folliculitis 毛囊炎**：金葡菌 → 温敷 + mupirocin；**热水浴 (hot tub) → Pseudomonas 铜绿假单胞菌**。
- **Furuncle 疖（深毛囊炎）/ Carbuncle 痈（疖群）** → 温敷 + **I&D 切开引流** ± 口服抗生素。

## 二、真菌感染 Fungal

> [!tip] 真菌通则
> 最佳初筛 **KOH 镜检**，最准 **真菌培养**；浅表 → 外用唑类 (azole)；**毛发 (capitis) / 指甲 (unguium) 必须口服**（外用渗不进）。反复念珠菌病 → 查 **HIV / 糖尿病**。

> [!info] 先看病原三分 → 再看部位定"外用 vs 口服"（USMLE 出题主线）
> - **皮肤癣菌 Dermatophyte（tinea 癣）** — **环形 + 中央清 + 隆起活动鳞屑边**，KOH 见**分枝菌丝**；按部位命名；**关键决策：毛发 (capitis) / 甲 (unguium) 必须口服，其余外用即可**。
> - **酵母 Yeast** — **Candida 念珠菌**（皱褶牛肉红 + **卫星灶脓疱**）/ **Malassezia 马拉色菌**（花斑癣，KOH "意面 + 肉丸"）。
> - **皮下真菌 Subcutaneous** — **Sporothrix 申克孢子丝菌**（玫瑰刺接种 → 沿淋巴管串珠结节溃疡），是结节/溃疡而非水疱。

```
                          浅表真菌
                             │
      ┌──────────────────────┼──────────────────────┐
  皮肤癣菌 Dermatophyte     酵母 Yeast          皮下真菌 Subcutaneous
 (tinea：环形+中央清+KOH菌丝)                    (Sporothrix 孢子丝菌)
      │                       │                 玫瑰刺·淋巴管串珠结节
  按部位命名          ┌───────┴───────┐
      │            Candida          Malassezia
  ┌───┴────┐      皱褶红+卫星脓疱      花斑癣·意面肉丸
外用即可     必须口服（外用渗不进）
corporis    capitis（毛发）
pedis       unguium（甲）
manuum
```

> [!tip] 真菌感染 buzzword 速查

| 病 | 病原 | 形态 buzzword | 治疗 |
|---|---|---|---|
| **体癣 tinea corporis** | 皮肤癣菌 dermatophyte | 环形 + 中央清 + 隆起鳞屑边 | **外用** terbinafine 特比萘芬 / 唑类 |
| **足癣 / 手癣 pedis / manuum** | 皮肤癣菌 | **两足一手 (two feet–one hand)** 分布、趾/掌鳞屑 | **外用** |
| **头癣 tinea capitis** | 皮肤癣菌 | 儿童、断发黑点、枕淋巴结、Kerion 脓癣 | **口服** griseofulvin 灰黄霉素 / terbinafine |
| **甲癣 tinea unguium** | 皮肤癣菌 | 甲增厚、变色、碎裂 | **口服** terbinafine / itraconazole 伊曲康唑 |
| **花斑癣 versicolor** | Malassezia 马拉色菌 | 上躯干**晒不黑**色素不均斑、KOH 意面肉丸 | selenium sulfide 二硫化硒 / ketoconazole 酮康唑 |
| **念珠菌间擦疹 intertrigo** | Candida 念珠菌 | 皱褶**牛肉红 + 卫星灶脓疱** | **外用** nystatin 制霉菌素 / 唑类 |
| **孢子丝菌病 sporotrichosis** | Sporothrix 申克孢子丝菌 | 玫瑰刺 → **淋巴管串珠结节溃疡** | **口服** itraconazole 伊曲康唑 |

> [!danger] 一个决策问题定生死：外用还是口服？
> 见到癣先问"**在毛发还是甲上？**"——**头癣 (capitis) / 甲癣 (unguium) 必须口服**（外用药渗不进毛囊/甲板，选外用即错）；体癣/足癣/手癣**外用即可**（广泛或难治才口服）。这是真菌题最高频的送分/送命点。

- **Tinea corporis 体癣**：环形 + **中央清亮 + 隆起活动性鳞屑边缘**，接触动物/人/污染物 → **外用 terbinafine 特比萘芬 / 唑类**；广泛/难治才口服。

  ![[{DE9727B7-ABDF-43A6-B965-3D316C217405}.png|320]]
  *体癣 (tinea corporis)：**圆形/环形斑块 + 隆起鳞屑边缘 + 中央趋清 (central clearing)**——经典皮肤癣菌环形损害。*

  ![[{9523E48D-2CD3-4365-BBC0-607EB415F278}.png|320]]
  *体癣 (tinea corporis)（另例·标注版）：**红色环形斑 (red annular patch) + 中央清亮 (central clearing) + 边缘鳞屑 (scales)**——三要素一图标全，对比钱币状湿疹（无中央清、KOH−）。*

  ![[{0BD2261E-9481-47B5-9B6A-7FC7C7EA1933}.png|320]]
  *面癣 (tinea faciei，体癣发生在面部)：**环形红斑、鳞屑、隆起活动边缘 + 中央趋清**——KOH 阳性；勿误当湿疹/SLE 用激素（"tinea incognito 隐匿癣"会加重）。*

- **Tinea pedis 足癣 / manuum 手癣**：趾/掌鳞屑（"两足一手 (two feet–one hand)"分布）。

  ![[{2E401986-EBC9-4513-8AA8-04779C84A96F}.png|320]]
  *手癣 (tinea manuum)：手掌/腕**环形红斑斑块 + 轻度鳞屑 (annular plaque with mild scaling)**——常单侧、伴足癣；勿误当湿疹用激素。*
- **Tinea capitis 头癣**：儿童、鳞屑/黑点（断发）/枕淋巴结/Kerion 脓癣 → **口服 griseofulvin 灰黄霉素 / terbinafine**。

![[Pasted image 20260629220810.png|360]]
*头癣脓癣 (tinea capitis with kerion)：头皮上**隆起、化脓、渗出性的炎性包块**（kerion 脓癣是严重炎症型）伴断发/脱发——**必须口服**抗真菌（外用渗不进毛囊）。*

- **Tinea unguium 甲癣**：甲增厚变色 → **口服 terbinafine / itraconazole 伊曲康唑**。
- **Tinea/Pityriasis versicolor 花斑癣**：上躯干**色素不均斑、晒不黑**，Malassezia 马拉色菌，KOH "意大利面+肉丸" → selenium sulfide 二硫化硒 / ketoconazole 酮康唑。
- **Candidal intertrigo 念珠菌间擦疹**：皱褶红斑 + **卫星灶脓疱** → 外用 nystatin 制霉菌素 / 唑类。

  ![[{6FC91BD5-DD96-47B6-92AE-F71F1DA03E57}.png|360]]
  *念珠菌间擦疹 (candidal intertrigo)：皱褶区**牛肉红斑块 + 周边卫星灶 (satellite lesions) 红色丘疹/脓疱**——白色念珠菌 (Candida albicans)，潮湿闭塞环境。*

- **Sporotrichosis 孢子丝菌病（深部/皮下真菌，非浅表）**：申克孢子丝菌 (Sporothrix schenckii)，**园丁/玫瑰刺/苔藓接种 ("rose gardener")**；接种处结节 → 沿**淋巴管链 (lymphatic chain)** 串珠样**结节性溃疡 (nodular ulcerations)**（淋巴管型 lymphocutaneous）——是手部**结节/溃疡**性病变的鉴别（**非水疱大疱**，借此与大疱性脓疱疮区分）→ 口服伊曲康唑 (itraconazole)。

  ![[{1BAFF7A4-B66C-49AB-B456-17AC01E24410}.png|320]]
  *孢子丝菌病 (sporotrichosis)：沿淋巴管走行的**串珠样结节性溃疡 (draining nodular ulcerations along the lymphatic chain)**——申克孢子丝菌 (Sporothrix schenckii) 经皮接种，表现为结节/溃疡而非大疱（与大疱性脓疱疮鉴别）。*


> [!danger] nystatin 制霉菌素 = 念珠菌专用，对皮肤癣菌无效
> **体癣是皮肤癣菌 (dermatophyte)**，用 terbinafine / 唑类；**nystatin 只抗念珠菌**，治体癣无效（经典送分陷阱）。clindamycin 凝胶治痤疮、fluocinonide 氟轻松是激素（不治真菌）也是干扰项。

## 三、病毒感染 Viral

> [!info] 先看形态 → 定病毒（USMLE 出题主线）
> 病毒性皮损**按形态四分**，认对形态基本就锁定病毒：
> - **成簇水疱 (grouped vesicles) + 痛** → 疱疹病毒 **HSV / VZV**（HSV 局灶、易复发；VZV **单侧皮节、不过中线**）
> - **疣状 / 过度角化丘疹** → **HPV 疣 (warts)**
> - **肤色 + 中央脐凹 (umbilicated) 丘疹** → **传染性软疣**（痘病毒 poxvirus）
> - **双侧手足 + 口腔疹** → **HFMD 手足口**（柯萨奇病毒 coxsackievirus）

```
                        病毒性皮损：先看形态
                                │
      ┌───────────────┬────────┴────────┬────────────────┐
      │               │                 │                │
  成簇水疱+痛      疣状/角化          肤色脐凹丘疹      双侧手足+口腔
 (grouped vesicles) (verrucous)      (umbilicated)      (H-F-M)
      │               │                 │                │
   疱疹病毒          HPV 疣          传染性软疣          HFMD
 Herpesviridae     (warts)         (poxvirus)      (coxsackie A)
      │
  ┌───┴────┐
局灶·易复发   单侧·皮节分布·不过中线
  HSV          VZV（带状疱疹 zoster）
```

> [!tip] 病毒感染 buzzword 速查（按病毒分组，含"同毒不同部位"病名中英）

| 病毒 | 临床病名（同毒不同部位，中英） | 形态 buzzword | 分布 / 钩子 | 治疗 |
|---|---|---|---|---|
| **HSV** 单纯疱疹 | 唇疱疹 herpes labialis · 疱疹性瘭疽 herpetic whitlow · 生殖器疱疹 genital herpes · **疱疹性湿疹 eczema herpeticum**（全部见上方 HSV 汇总表） | 红斑基础**成簇水疱** + 痛 | 唇缘 / 指端 / 生殖器；**易复发** | 自限；重/免疫低下 acyclovir 阿昔洛韦 |
| **VZV** 水痘-带状疱疹 | 水痘 varicella（原发·泛发）· 带状疱疹 herpes zoster / shingles（再激活·皮节） | 成簇水疱 + 痛 | zoster **单侧、皮节、不过中线** | valacyclovir 伐昔洛韦 |
| **HPV** 人乳头瘤 | 跖疣 plantar wart (verruca plantaris) · 寻常疣 common wart (verruca vulgaris) · 扁平疣 flat wart (verruca plana) · 尖锐湿疣 condyloma acuminatum (生殖器疣) | **疣状 / 过度角化**，削面**点状出血** | 跖疣（vs 胼胝 callus 无出血） | 自限；顽固冷冻 / 水杨酸 salicylic acid |
| **痘病毒** poxvirus | 传染性软疣 molluscum contagiosum | 肤色圆顶**中央脐凹** | 儿童四肢躯干 / 成人生殖区 | 自限；**巨大/数百/面部广泛 → 查 HIV** |
| **柯萨奇** coxsackie A | 手足口 HFMD (hand-foot-and-mouth) · 疱疹性咽峡炎 herpangina（仅口咽） | 手足**双侧**斑丘疹/水疱 + 口腔疹 | **双侧对称**是关键 | 自限对症 |

> [!note]- HSV「一毒多面」—— 所有 herpes 相关病名汇总
> **同一个 HSV (herpes simplex virus, 单纯疱疹病毒) 按部位 / 宿主不同 → 一堆病名。** 其中 **Eczema herpeticum 疱疹性湿疹 = HSV 感染**（多为 HSV-1，播散在特应性皮炎皮肤上，旧称 **Kaposi 水痘样疹 Kaposi varicelliform eruption**），是这堆里最凶的皮肤科急症。
>
> | 病名 | 部位 / 人群 | 关键钩子 / 陷阱 | 处理 |
> |---|---|---|---|
> | **唇疱疹 herpes labialis** | 唇红缘，复发 | 拖 >1 月不愈 → 排下唇鳞状细胞癌 (squamous cell carcinoma, SCC) | 自限 ± 抗病毒 |
> | **疱疹性龈口炎 gingivostomatitis** | 幼儿**原发**感染、口腔广泛糜烂 + 发热 | 首次感染最重、最广泛 | 支持 ± acyclovir 阿昔洛韦 |
> | **生殖器疱疹 herpes genitalis** | 生殖器，性传播 | 复发性**痛性**溃疡（vs 梅毒硬下疳 syphilitic chancre 无痛） | valacyclovir 伐昔洛韦 |
> | **疱疹性瘭疽 herpetic whitlow** | 指端，接触史 / 医护 | **勿切开引流** | 自限 / 抗病毒 |
> | **角斗士疱疹 herpes gladiatorum** | 摔跤运动员暴露皮肤 | 皮肤-皮肤接触传播 | 抗病毒 |
> | **疱疹性湿疹 eczema herpeticum** | 特应性皮炎基础、**播散** | 凿孔样糜烂 + 出血痂 + 发热 = **皮肤科急症** | **立即系统 acyclovir 阿昔洛韦 / valacyclovir** |
> | **疱疹性角膜炎 herpetic keratitis** | 眼，**树枝状 (dendritic) 溃疡** | **禁用激素**（会恶化 / 穿孔） | 外用抗病毒 + 急转眼科 |
> | **疱疹性脑炎 HSV encephalitis** | **颞叶**，发热 + 意识 / 人格改变 | 别等结果 | **立即 IV acyclovir 静脉阿昔洛韦** |
> | **新生儿疱疹 neonatal HSV** | 出生经产道 | 皮肤-眼-口 → 可播散 / 脑炎 | IV acyclovir |
>
> **VZV (varicella-zoster virus, 水痘-带状疱疹病毒) 也是疱疹病毒科** → 水痘 (varicella) / 带状疱疹 (zoster)，见下条。HSV 与 VZV 皮损都是"红斑基础成簇水疱"，靠**分布**分：**HSV 局灶、易复发**；**VZV 单侧皮节、不过中线**。

- **HSV 单纯疱疹病毒**：前驱发热/不适 → **红斑基础上成簇水疱 (grouped vesicles)** + 刺痛 ± 区域淋巴结，2–3 周自愈；免疫低下 → acyclovir 阿昔洛韦。

  ![[{8629CE13-1EC0-464B-875C-5363212E8738}.png|360]]
  *HSV 单纯疱疹病毒感染：**红斑基础上成簇水疱 (groups of vesicles on erythematous base)**——经典形态（破溃后结痂，2–3 周自愈）。*

  ![[Pasted image 20260629215856.png|300]]
  *疱疹性瘭疽 (herpetic whitlow)：手部 HSV 感染，红斑基础上成簇水疱/糜烂、痛——成人多为接触活动性生殖器疱疹 (HSV-2)，医务人员可为 HSV-1；**勿切开**。（Q4011 题图，详见 [[mistakes/uworld-mistakes_2026-06#^Q4011]]）*

  ![[{BCDC7492-D080-4AD4-B957-FBC6A4B82F04}.png|320]]
  *疱疹性瘭疽 (herpetic whitlow)（另例·标注版）：单个手指**红斑基础上的水疱 (vesicles on an erythematous base)**——痛而非痒、不会双手对称（与汗疱疹鉴别）；勿切开引流。*

  ![[{AD8F0808-8180-44F2-A08E-21094074F396}.png|320]]
  *单纯疱疹病毒感染 (herpes simplex virus, HSV)（另例）：**成簇痛性水疱 (clusters of painful vesicles)**，经典见于唇红缘 (vermilion border)，亦可累及手指（疱疹性瘭疽 herpetic whitlow）或手部——与大疱性脓疱疮的松弛大疱/领圈状脱屑区分。*

  - **Herpetic whitlow 疱疹性瘭疽**：指端成簇水疱、痛、性接触/口腔接触史；**勿切开**。
  - **Herpes labialis 唇疱疹**：红唇缘痛性水疱→结痂，**约 2 周自愈**（与下唇 SCC 慢性溃疡区分）。

    ![[{C432A06D-4603-4817-AF77-BCA497DA028F}.png|360]]
    *唇疱疹 (herpes labialis)：唇红缘 (vermilion border)**成簇痛性水疱**→破溃结痂，**约 2 周自愈**——关键鉴别：拖到一个月不愈的下唇病变要排鳞癌 (SCC)。*

  - **Eczema herpeticum 疱疹性湿疹**（急症）：特应性皮炎基础 + **凿孔样糜烂 (punched-out) + 出血痂 + 发热** → **立即系统 acyclovir / valacyclovir 伐昔洛韦**，可播散内脏。

    ![[Pasted image 20260629220001.png|360]]
    *疱疹性湿疹 (eczema herpeticum)：特应性皮炎区域成片**凿孔样糜烂 (punched-out erosions)** + 出血性结痂 (hemorrhagic crusts) + 棕黄色结痂——HSV (herpes simplex virus, 单纯疱疹病毒) 在湿疹皮肤上播散，属皮肤科急症。*

    ![[{3A428815-AE27-4790-B0E5-F17974C848D4}.png|320]]
    *疱疹性湿疹 (eczema herpeticum)（另例）：湿疹斑片上成簇水疱进展为**凿孔样糜烂 + 出血性结痂 (punched-out erosions with hemorrhagic crust)**，疼痛——与大疱性脓疱疮（松弛大疱 + 领圈状脱屑）鉴别；处理为立即系统阿昔洛韦 (acyclovir)。*

- **VZV 带状疱疹 Herpes zoster**：**单侧皮节分布**簇集水疱、痛；valacyclovir。

  ![[{693013D1-9A11-4784-8506-3431873F3573}.png|320]]
  *带状疱疹 (herpes zoster / shingles)：**单侧、皮节分布 (dermatomal) 的痛性水疱与大疱**，不过中线——水痘-带状疱疹病毒 (varicella-zoster virus, VZV) 再激活，多见于成人/免疫低下；婴幼儿少见、疼痛轻者不支持此诊断（与大疱性脓疱疮鉴别）。*
- **HPV 疣 Warts**：
  - **Plantar wart 跖疣 (verruca plantaris)**：足底疣状菜花样过度角化丘疹、成群；削面见**血栓毛细血管点状出血**（与**胼胝 callus** 区分：胼胝无点状出血、骨突摩擦处）。
  - 寻常疣 (verruca vulgaris)、扁平疣 (verruca plana)；可自限，顽固 → 冷冻 / 水杨酸 (salicylic acid)。

    ![[{00159B83-BC86-488B-BF73-B6E9CD8ED73B}.png|320]]
    *寻常疣 (verruca vulgaris)：质硬、**过度角化 (hyperkeratotic)、表面裂隙/疣状赘生**的丘疹——**不溃疡**（与溃疡性肿瘤鉴别）。*
    ![[{E25FA0D9-5EB0-4962-A8EF-5EAB98204B0D}.png]]
    ![[{8C5F6963-DA93-4DBE-AF8D-A4475817FD1A}.png]]


- **Molluscum contagiosum 传染性软疣（痘病毒 poxvirus）**：
  - **病原/传播**：痘病毒科 (poxvirus)；**皮肤-皮肤接触 (skin-to-skin)** 或污染物 (fomites) + 自体接种 (autoinoculation) 播散；儿童最常见，成人可经性接触。
  - **临床**：小的、**肤色、中央脐凹 (umbilicated) 的丘疹**，可微痒；儿童多见于四肢/面/躯干，成人性传播者见于下腹与肛门生殖区。
  - **病程/处理**：**自限 (6–12 月自愈)**；需要时刮除 (curettage)、冷冻 (cryotherapy) 或外用药（podophyllotoxin 鬼臼毒素）以防播散/改善外观。
  - **陷阱**：**巨大 (>10 mm)、数目众多 (数百)、面部受累或广泛分布** → 提示**细胞免疫受损**，应**查 HIV (human immunodeficiency virus, 人类免疫缺陷病毒)**。

  ![[{956C1B6C-3404-4356-834F-37A37EDB25C8}.png|320]]
  *传染性软疣 (molluscum contagiosum, MC)：成群**肤色、圆顶、中央脐凹 (umbilicated) 的丘疹**——痘病毒 (poxvirus) 感染，自限；若巨大/数百/面部广泛 → 查 HIV。*

> [!warning] 手部水疱/糜烂三鉴别
> **成簇水疱 + 痛 + 性接触史 = HSV 瘭疽（勿切开）**；**扩大松弛大疱 + 黄疱液 + 领圈脱屑 = 大疱性脓疱疮（金葡菌）**；**双侧手足 + 口腔疹 = HFMD 手足口（柯萨奇病毒 coxsackievirus）**。

![[{97DEFC03-8B2B-41E3-9595-AD8A130FA120}.png|320]]
*手足口病 (hand-foot-and-mouth disease, HFMD)：**手掌/足底双侧、非触痛、非瘙痒的斑丘疹/水疱 (maculopapular and/or vesicular rash)** + 口腔病变——柯萨奇病毒 (coxsackievirus)；**双侧对称**是关键，若为孤立、单侧手部病变则不支持（如大疱性脓疱疮）。*

## 四、寄生虫 Parasitic

> [!info] 一句话三鉴别（钩子对照 —— 臭虫是经典干扰项）

| | **疥疮 Scabies** | **虱病 Pediculosis** | **臭虫 Bed bug（干扰项）** |
|---|---|---|---|
| 病原 | Sarcoptes scabiei 疥螨 | 虱 (lice) | Cimex 臭虫（叮咬，非皮肤定植）|
| 钩子 | **指缝/屈腕剧痒 + 夜间加重 + 家人同患**、隧道 (burrow) | 发干紧贴**白色虫卵 (nits)**、局部瘙痒 | 暴露部位**线状"早-中-晚 (breakfast-lunch-dinner)"排列**、不进指缝、家人不会同时起 |
| 治疗 | permethrin 氯菊酯 / ivermectin 伊维菌素，**全家/密切接触者同治** + 烫洗衣物 | permethrin / ivermectin / spinosad 多杀菌素 + 烫洗床品 | 环境灭虫（非皮肤用药）|

- **Scabies 疥疮**：**Sarcoptes scabiei 疥螨**，**指缝/屈腕/肘/腋/脐/生殖器剧痒、夜间加重**，隧道 (burrow) 特征性，**高度接触传染（家人常同患）** → **外用 5% permethrin 氯菊酯 或 口服 ivermectin 伊维菌素**，**家人/密切接触者同治** + 烫洗衣物。结痂性（挪威）疥疮见于免疫抑制，极强传染性。
  - 鉴别 **臭虫 bed bug**：常**线状"早中晚"排列**、不进指缝、不会让家人同时起。

  ![[{D432DC21-F898-4D55-998B-5DF8A7447E4D}.png|320]]
  *疥疮 (scabies)：**指缝 (finger webs) 瘙痒性抓破丘疹 (excoriated papules)**——好发指缝/屈腕，剧痒夜重；与汗疱疹（深在性水疱、对称掌跖）鉴别。*
- **Pediculosis 虱病**：发干紧贴的白色虫卵 (nits) → permethrin / ivermectin / spinosad；烫洗衣物床品。

## 五、新生儿头部脓疱病（与"新生儿痤疮"）

- **Neonatal cephalic pustulosis 新生儿头部脓疱病**：约 **3 周龄、仅面/头皮**红斑丘疹脓疱、**无粉刺**、患儿状态好，Malassezia 马拉色菌相关 → **温和清洁 (gentle soap & water)**，自限；重者外用酮康唑/低效价激素。**避免**油性润肤剂、mupirocin、permethrin、水杨酸。其余新生儿良性疹（毒性红斑/粟丘疹/痱子/脓疱性黑变病）速查见 [[完整笔记/专题笔记/皮肤/皮肤_HighYield速查总览]] §八。

## 六、易混陷阱（普适）

- **大疱性脓疱疮只金葡菌**；非大疱性才金葡菌或链球菌。
- **nystatin 只治念珠菌**，体癣无效。
- **成簇水疱 = HSV**；**双侧手足口 = 柯萨奇**；**单侧皮节 = 带状疱疹**。
- **跖疣削面点状出血**；胼胝没有。
- **指缝 + 夜痒 + 家人同患 = 疥疮**（全家同治）；臭虫是线状排列。
- **凿孔样糜烂（特应性皮炎背景）= 疱疹性湿疹 → 系统抗病毒**，不是细菌。
- **新生儿仅面部脓疱、3 周龄、娃好 → 温和清洁**，别上药。

## 七、自测（答完再判分）

> [!question]- Q1. 4 岁湿疹史女孩手部扩大水疱、黄疱液 + 领圈状脱屑、轻痒。病原？
> **S. aureus 金葡菌**（大疱性脓疱疮，剥脱毒素 A）→ 口服 cephalexin。

> [!question]- Q2. 42 岁男性上臂环形、隆起边缘 + 中央清亮、瘙痒斑，新养猫。最佳治疗？
> **外用 terbinafine / 唑类**（体癣）。**nystatin 无效**（只抗念珠菌）。

> [!question]- Q3. 空乘 10 天指缝/腕屈侧剧痒（夜重），女儿 2 天前同症状。诊断 + 处理？
> **Scabies 疥疮** → permethrin / ivermectin + **家人同治** + 烫洗衣物。

> [!question]- Q4. 2 岁特应性皮炎患儿颈/上背水疱→凿孔样糜烂 + 出血痂 + 发热，抗生素无效。处理？
> **疱疹性湿疹 → 立即系统 acyclovir**。

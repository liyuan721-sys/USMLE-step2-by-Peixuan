# 水疱 / 大疱病鉴别 Vesiculobullous Disorders 速查

> [!info] 一句话定位
> 见到"水疱 (vesicle) / 大疱 (bulla) / 糜烂 (erosion)"，按 **5 个轴**切：① **年龄**（婴幼儿 vs 成人 vs 老年）② **黏膜受累 (mucosal involvement)** ③ **Nikolsky 征** ④ **诱因**（摩擦/日晒/药物/感染/遗传）⑤ **疱壁松弛 flaccid vs 紧张 tense**。本组错题集中暴露：**PV vs BP**、**SSSS vs TEN**、**EB vs 后天大疱病**、**PCT 日晒区瘢痕性水疱**、**bullous impetigo 局限 vs SSSS 泛发**。

> [!note]- 缩写对照（全称 中英）
> - **PV** = pemphigus vulgaris 寻常型天疱疮
> - **BP** = bullous pemphigoid 大疱性类天疱疮
> - **DH** = dermatitis herpetiformis 疱疹样皮炎
> - **EB** = epidermolysis bullosa 大疱性表皮松解症
> - **PCT** = porphyria cutanea tarda 迟发性皮肤卟啉病
> - **UROD** = uroporphyrinogen decarboxylase 尿卟啉原脱羧酶
> - **SSSS** = staphylococcal scalded skin syndrome 葡萄球菌性烫伤样皮肤综合征
> - **SJS / TEN** = Stevens-Johnson syndrome / toxic epidermal necrolysis 史蒂文斯-约翰逊综合征 / 中毒性表皮坏死松解症
> - **EM** = erythema multiforme 多形红斑
> - **BSA** = body surface area 体表面积
> - **IF** = immunofluorescence 免疫荧光
> - **anti-tTG IgA** = anti-tissue transglutaminase IgA 抗组织转谷氨酰胺酶 IgA
> - **HCV** = hepatitis C virus 丙型肝炎病毒
> - **S. aureus** = Staphylococcus aureus 金黄色葡萄球菌

## 一、决策框架（5 轴切分）

```
水疱 / 大疱 / 糜烂
├── 婴幼儿 / 儿童
│   ├── 泛发红斑→浅表剥脱 + Nikolsky+ + 口周痂、无黏膜 ─ SSSS（金葡菌毒素血源播散）
│   ├── 局限松弛大疱→领圈状脱屑、患儿一般状态好 ──── Bullous impetigo（金葡菌局部毒素）
│   └── 摩擦部位(掌跖)起疱、奶瓶致口腔疱、婴儿期起 ── Epidermolysis bullosa（遗传，角蛋白）
├── 成人（中年 40-60）
│   ├── 松弛易破 + 口腔黏膜先受累 + Nikolsky+ + 痛 ── Pemphigus vulgaris（抗桥粒芯蛋白）
│   ├── 肘膝臀剧痒小疱 + 乳糜泻 ──────────────── Dermatitis herpetiformis（抗 tTG IgA）
│   ├── 日晒区(手背)无痛疱 + 愈合瘢痕 + 皮肤脆 + HCV ─ PCT（UROD 缺陷）
│   └── 靶形损害 + 近期 HSV ──────────────────── Erythema multiforme（多 HSV 诱发）
├── 老年（>60）
│   └── 紧张厚疱 + 瘙痒 + 黏膜罕见 + Nikolsky− ──── Bullous pemphigoid（抗半桥粒）
└── 任何年龄 + 用药后 + 黏膜重 + 大面积剥脱 + Nikolsky+ ─ SJS/TEN（药物，见 HighYield总览 §二）
```

> [!tip] 三把快刀
> - **Nikolsky 征阳性谱**：PV、SSSS、SJS/TEN（**BP 阴性**）。
> - **黏膜受累**：PV / SJS-TEN **常累及**；BP / SSSS **罕见**。
> - **疱壁**：PV/SSSS **松弛易破**；BP **紧张不易破**。

## 二、自身免疫性大疱病：PV vs BP vs DH ⭐

| | **Pemphigus vulgaris 寻常型天疱疮** | **Bullous pemphigoid 大疱性类天疱疮** | **Dermatitis herpetiformis 疱疹样皮炎** |
|---|---|---|---|
| 好发年龄 | **40-60** | **>60** | 成人 |
| 抗原 / 层次 | 抗 **desmoglein 桥粒芯蛋白**（细胞间，**表皮内 intraepidermal**）| 抗 **hemidesmosome 半桥粒**（基底膜，**表皮下 subepidermal**）| 真皮乳头 **anti-tTG IgA**（表皮下）|
| 疱壁 | **松弛、易破 → 糜烂** | **紧张、厚、不易破** | 小疱、剧痒、抓破 |
| 黏膜 | **几乎必累及（口腔常为首发）** | **罕见** | 罕见 |
| Nikolsky | **+**（acantholysis 棘层松解）| **−** | − |
| 痛 / 痒 | **疼痛** | **瘙痒** | **剧痒** |
| 免疫荧光 IF | **网状细胞间 IgG**（鱼网样）| **基底膜线状 IgG + C3** | 真皮乳头**颗粒状 IgA** |
| 部位 | 口腔 + 躯干（正常皮肤背景）| 躯干 + 屈侧（**不累及掌跖**）| **肘、膝、臀、后颈、头皮**对称 |
| 关联 | — | — | **乳糜泻 celiac disease**（麸质敏感肠病）|
| 诊断 | **皮肤活检 + IF**（必须，治疗终身且毒性大）| 活检 + IF | 活检 + IF；查乳糜泻血清 |
| 治疗 | **系统糖皮质激素 ± rituximab 利妥昔单抗**；难治→环磷酰胺/IVIG/血浆置换 | **系统/外用糖皮质激素** | **dapsone 氨苯砜 + 无麸质饮食 (gluten-free diet)** |

> [!danger] PV vs BP 一句话锁定
> **痛 + 松弛 + 口腔 + Nikolsky+ + 中年 + 细胞间鱼网 IgG = PV**；**痒 + 紧张 + 无黏膜 + Nikolsky− + 老年 + 基底膜线状 IgG = BP**。**两者都不累及掌跖**（掌跖摩擦疱想 EB）。
![[{36ABECC1-584D-4D36-8F38-CB9C34A793EB}.png]]
![[{6E9BC3DE-697B-48D6-8864-16AD5F511C63}.png|360]]
*疱疹样皮炎 (dermatitis herpetiformis, DH)：**肘/膝/臀对称分布的剧痒成簇丘疹、小疱及抓痕**——抗 tTG IgA、强关联乳糜泻 (celiac disease)；治疗 dapsone 氨苯砜 + 无麸质饮食。*

## 三、遗传性 & 代谢性大疱：EB vs PCT ⭐

### Epidermolysis bullosa 大疱性表皮松解症（EB，遗传）

- **本质**：遗传性上皮脆性病，**轻微摩擦/创伤 → 水疱、糜烂、溃疡**；缺陷在表皮内/真皮表皮连接的黏附蛋白。
- **四型**：simplex（最常见，**角蛋白 keratin 基因突变**，表皮内裂隙）、junctional、dystrophic、Kindler。
- **临床钩子（EB simplex）**：**婴幼儿/儿童期**起病；**摩擦诱发，好发掌跖 (palms & soles)** 及暴露部位；**婴儿吸奶瓶 → 口腔疱**；可有**足底皮肤增厚**；多**愈合不留瘢**。
- **诊断**：新鲜疱**活检 + 免疫荧光镜检**定位裂隙层；基因检测确诊。
- **治疗**：以**创面护理 + 支持**为主（无根治）。

> [!warning] EB 与后天大疱病的区分（本组错题陷阱）
> **掌跖摩擦起疱 + 婴儿期发病 + 奶瓶致口腔疱 = 遗传性 EB**。
> - **BP / PV** 是**后天自身免疫**、主要**中老年**、**不累及掌跖** → 排除。
> - **PCT** 主要**日晒区（手背）**、愈合**留瘢**、与 HCV/酒精/雌激素相关 → 部位与诱因不同。

### Porphyria cutanea tarda 迟发性皮肤卟啉病（PCT，最常见卟啉病）

- **机制**：**UROD（尿卟啉原脱羧酶）缺陷** → 卟啉蓄积 → **光敏 (photosensitivity)**，紫外激发产生活性氧损伤皮肤。
- **临床指纹**：**日晒区（手背、前臂、面）无痛性水疱/大疱** + **愈合留瘢 + 皮肤脆性增加 (skin fragility)** + 多毛 (hypertrichosis) + 色素沉着 + 茶/酒色尿。
- **关联（高频）**：**丙型肝炎 (hepatitis C)**、HIV、**酗酒**、**雌激素 (estrogen) 使用**、铁过载。
- **诊断**：**血浆/尿卟啉升高**（尿 uroporphyrin↑）；轻度肝酶↑、铁过载。
- **治疗**：**放血 (phlebotomy) / 羟氯喹 (hydroxychloroquine)** + 避光 + **治疗 HCV / 戒酒 / 停雌激素**。

![[Pasted image 20260629220443.png|360]]
*PCT 手背指背：**日晒区无痛性水疱/糜烂 + 愈合留瘢 + 皮肤脆性增加 (skin fragility)**——轻微外伤即破，为最经典部位与表现。*

![[Pasted image 20260629220512.png|620]]
*慢性丙型肝炎 (chronic hepatitis C, HCV) 系统/肝外关联速查——皮肤表现栏列出 **PCT 迟发性皮肤卟啉病**与**扁平苔藓 (lichen planus, LP)**；故见到 PCT/LP 要顺手筛 HCV（抗-HCV IgG 筛查、HCV RNA PCR 确诊）。*

> [!tip] 一句话指纹
> **"日晒区无痛疱 + 愈合瘢痕 + 皮肤脆 + HCV/酗酒/雌激素" = PCT**；**"摩擦/掌跖疱 + 婴儿期 + 不留瘢" = EB simplex**。

## 四、感染 / 毒素介导的大疱：Bullous impetigo vs SSSS ⭐

| | **Bullous impetigo 大疱性脓疱疮** | **SSSS 葡萄球菌性烫伤样皮肤综合征** |
|---|---|---|
| 病原 / 机制 | **S. aureus 金黄色葡萄球菌**产**剥脱毒素 A**，**局部**作用 | 同毒素但**血源播散 (disseminated)**、全身作用 |
| 年龄 | 儿童（也可成人）| **<5 岁幼儿**；成人见于肾衰/免疫抑制 |
| 范围 | **局限**于原发感染灶 | **泛发全身** |
| 一般状态 | 多**良好** | **发热、易激惹、皮肤触痛**（前驱）|
| 皮损 | 松弛大疱 → 破溃留**领圈状脱屑 (collarette of scale)**、黄色渗液结痂 | 弥漫红斑 + **Nikolsky+** + 浅表剥脱（颗粒层）+ **口周放射状结痂**、**无黏膜受累** |
| 诊断 | 临床；疱液培养金葡菌 | 临床；血/感染灶培养（**疱液无菌** = 毒素源在别处）|
| 治疗 | **口服抗生素**（cephalexin 头孢氨苄）；局限可外用 mupirocin 莫匹罗星 | **静脉抗葡萄球菌**（nafcillin 萘夫西林 / vancomycin 万古霉素）+ 支持 |

> [!danger] Bullous impetigo 病原 = 金葡菌（不是链球菌）
> 大疱性脓疱疮**只由 S. aureus 金黄色葡萄球菌**（剥脱毒素 A）引起；而**非大疱性 (nonbullous) impetigo** 才可由金葡菌**或** Streptococcus pyogenes 化脓链球菌引起。见到**松弛大疱 + 黄色疱液 + 领圈状脱屑** → 选金葡菌。

> [!warning] 大疱/水疱手部病变的鉴别（呼应本组手部题）
> - **Eczema herpeticum 疱疹性湿疹**（HSV）：特应性皮炎基础上**凿孔样糜烂 (punched-out erosions) + 出血性结痂 + 痛** → 系统 acyclovir 阿昔洛韦（见 HighYield总览 §二）。
> - **Bullous impetigo**：扩大的松弛大疱 + 领圈状脱屑、轻度触痛/痒。
> - **Herpetic whitlow 疱疹性瘭疽**（HSV）：指端**成簇水疱**、痛 → **勿切开**。

![[{0EE0B083-849A-4D79-AFE7-D3465BD85AF4}.png|360]]
*葡萄球菌性烫伤样皮肤综合征 (staphylococcal scalded skin syndrome, SSSS)：婴幼儿**泛发红斑基础上浅表松弛大疱/剥脱 + 口周放射状结痂 (perioral crusting)**，Nikolsky 征阳性、不累及黏膜——金黄色葡萄球菌 (Staphylococcus aureus) 剥脱毒素血源播散；静脉抗葡萄球菌（nafcillin 萘夫西林 / vancomycin 万古霉素）。（Q2778 题图）*

![[{03AFA18A-C149-4C46-A7DC-5097BCD670B0}.png|320]]
*大疱性脓疱疮 (bullous impetigo)（SSSS 对照）：**局限**于原发灶的松弛大疱、患儿一般状态好、**Nikolsky 征阴性**——同为金黄色葡萄球菌剥脱毒素，但局部作用（vs SSSS 血源播散泛发），口服头孢氨苄 (cephalexin) 即可。*

## 五、Nikolsky 征阳性谱 & 一图流陷阱

> [!info] Nikolsky+ = 侧向压力使表皮剥脱
> **阳性**：Pemphigus vulgaris、SSSS、SJS/TEN。**阴性**：Bullous pemphigoid、EB（机械脆性≠Nikolsky 概念，临床别混）。

- **SSSS vs TEN**（本组错题）：都泛发剥脱 + Nikolsky+，但 **SSSS 不累及黏膜 + 仅颗粒层浅剥脱 + 幼儿 + 毒素**；**TEN 累及黏膜 + 表皮全层坏死 + 成人 + 药物**。BSA：SJS <10% / overlap 10-30% / TEN >30%。

![[{428F9D8A-C930-4C4E-B873-6F6B039ACC1E}.png|320]]
*SJS/TEN 谱（Stevens-Johnson syndrome / toxic epidermal necrolysis, 史蒂文斯-约翰逊综合征 / 中毒性表皮坏死松解症）：肢端水疱/靶样损害——重症黏膜皮肤药物反应（磺胺、抗惊厥药等），伴高热 + 痛性出血性口腔糜烂。*

![[{250CCDF2-E072-42F2-B33C-8EC7082BB689}.png|360]]
*中毒性表皮坏死松解症 (toxic epidermal necrolysis, TEN)（另例·SSSS 对照）：**广泛 (>30% 体表) 表皮全层水疱剥脱 + 黏膜受累 (mucous membrane involvement)**，Nikolsky 征阳性、多由药物触发——与 SSSS 区别在于**累及黏膜 + 表皮全层坏死 + 成人 + 药物**（SSSS 不累黏膜、仅颗粒层浅剥脱、幼儿、毒素）。*
- **EM 多形红斑 (erythema multiforme)**（本组错题）：**靶形 (target) 损害** + 近期 HSV；**临床诊断、无需额外检查/活检**（典型者）；与 SJS/TEN 谱系区别在面积小、药物史不突出、预后好。
  - **机制/诱因**：**细胞介导免疫 (cell-mediated immunity)**，**非药物性**；最常 **HSV 单纯疱疹病毒 (herpes simplex virus)** 诱发，次为 **肺炎支原体 (Mycoplasma pneumoniae)**。
  - **临床**：急性对称、肢端为主的**三色靶形 / 虹膜样损害 (target / iris lesion)**——中央暗紫±水疱 + 中环苍白/水肿 + 外环红晕；可伴**黏膜痛性糜烂**（黏膜型与 SJS 谱相邻）。
  - **诊断/治疗**：典型者**临床诊断**，**不查血培养 / 补体 / IgE / 活检**；急性期**支持 ± 外用糖皮质激素**，**复发型**用抑制性抗病毒（acyclovir 阿昔洛韦）。
  - **与 SJS/TEN 分界**：EM = **面积小 + Nikolsky 阴性 + 药物史不突出 + 自限预后好**；SJS/TEN = **大面积剥脱 + Nikolsky 阳性 + 药物触发 + 重症累黏膜**。

> [!tip] EM Memory Hook
> **"三色靶形 (target) + 近期唇疱疹 (HSV) + 别乱查 = EM"**；靶形像"打靶红心"、HSV 是最常见的"扳机"，典型者一眼临床诊断——想开血培养/补体/IgE 就是掉进干扰项。

![[Pasted image 20260629220604.png|360]]
*多形红斑 (erythema multiforme, EM) 手部：**红斑性靶形损害 (targetoid lesions)** + 部分中央**水疱/大疱 (central vesicles/bullae)**——典型肢端分布。*

![[Pasted image 20260629220544.png|460]]
*EM 多形红斑要点表：**细胞介导免疫**，常由感染诱发（**HSV 单纯疱疹病毒**、肺炎支原体 Mycoplasma pneumoniae）；皮肤急性靶形损害、黏膜痛性大疱糜烂；处理——急性期支持 ± 外用激素，复发型用抑制性抗病毒（如 acyclovir 阿昔洛韦）。*

![[Pasted image 20260629220616.png|360]]
*EM 多形红斑靶形特写：手背三色**靶形 / 虹膜样损害 (target / iris lesion)**——中央暗紫/水疱 + 苍白中环 + 红色外环。*

![[Pasted image 20260629220531.png|360]]
*EM 多形红斑黏膜受累：**口唇 / 口腔痛性糜烂** + 靶形红斑——黏膜型与 SJS (Stevens-Johnson syndrome, 史-约综合征) 谱系相邻，多 HSV 诱发。*

![[{AEC3F0B4-5F58-4CF3-B631-872BAD55C862}.png|360]]
*EM 多形红斑经典靶形（兼 SSSS 对照锚点）：肢体**三色靶形损害**——中央暗紫 (dusky center) + 中环苍白/水肿 + 外环红晕 (erythematous halo)；散在分布、可有中央大疱 (central bullae)，但 **Nikolsky 征阴性 + 自限性 + 多由感染（单纯疱疹病毒 herpes simplex virus, HSV）诱发**——与 SSSS 泛发剥脱 / Nikolsky 阳性截然不同。*

## 六、易混陷阱（普适）

- **掌跖摩擦疱 + 婴儿期 = EB**，别选 BP/PV（后天、中老年、不累及掌跖）。
- **日晒区无痛疱 + 留瘢 + HCV = PCT**，别只想"接触性皮炎/虫咬"。
- **大疱 + 黄疱液 + 领圈脱屑 = bullous impetigo（金葡菌）**，别选链球菌。
- **泛发剥脱**：幼儿无黏膜 → SSSS；成人有黏膜 + 用药 → TEN。
- **靶形 + HSV = EM → 不查血培养/补体/IgE，临床诊断即可**。
- **PV 必须活检 + IF 确诊**（治疗终身毒性大），不能仅凭外观开药。

## 七、自测（答完再判分）

> [!question]- Q1. 45 岁女性先口腔痛性糜烂，2 周后躯干出现松弛、易破大疱，Nikolsky+，背景皮肤正常。诊断 + 确诊方法？
> **Pemphigus vulgaris 寻常型天疱疮** → **皮肤活检 + 免疫荧光（细胞间鱼网状 IgG）**；血清抗 desmoglein 支持。

> [!question]- Q2. 21 岁女性打高尔夫后双掌起痛性大疱，既往徒步常足底起疱、婴儿期奶瓶喂养时口腔起疱，足底皮肤增厚。最可能诊断？
> **Epidermolysis bullosa（EB simplex）** —— 遗传性、摩擦诱发、掌跖、婴儿期奶瓶口腔疱、足底增厚。**非** BP/PV（后天、不累及掌跖）。

> [!question]- Q3. 4 岁患儿局部出现扩大的松弛大疱、黄色疱液、破后领圈状脱屑，一般状态好。最可能病原？
> **Staphylococcus aureus 金黄色葡萄球菌**（剥脱毒素 A）→ 大疱性脓疱疮；口服 cephalexin 头孢氨苄。

> [!question]- Q4. 33 岁女性手背反复无痛水疱、轻微外伤即破、愈合留瘢，8 年前诊断丙肝未治。最可能机制？
> **卟啉蓄积致光敏（UROD 尿卟啉原脱羧酶缺陷）= PCT**；查尿/血卟啉，放血/羟氯喹 + 治丙肝。

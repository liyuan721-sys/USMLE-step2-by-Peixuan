# 皮肤肿瘤与癌前 Skin Cancer & Precancer 详解

> [!info] 一句话定位
> 皮肤肿瘤题三关：① **形态识别**（珍珠卷边=BCC / 角珠=SCC / 砂纸样=AK / 贴附蜡样=SK / ABCDE=黑色素瘤）② **诊断步骤——可疑恶性先活检 (biopsy first)，别先宽切除或安抚** ③ **切缘 / 术式**（BCC 4mm 或面部 Mohs；SCC 4–6mm；黑色素瘤按 Breslow 深度）。本篇放机制与决策细节；考前一眼速查见 [[完整笔记/专题笔记/皮肤/皮肤_HighYield速查总览]]。

> [!note]- 缩写对照（全称 中英）
> - **BCC** = basal cell carcinoma 基底细胞癌
> - **SCC** = squamous cell carcinoma 鳞状细胞癌
> - **AK** = actinic keratosis 光化性角化病
> - **SK** = seborrheic keratosis 脂溢性角化病
> - **SCC in situ** = squamous cell carcinoma in situ（Bowen 病）原位鳞癌
> - **UV** = ultraviolet 紫外线
> - **HHV-8** = human herpesvirus 8 人类疱疹病毒 8 型
> - **HAART** = highly active antiretroviral therapy 高效抗反转录病毒治疗
> - **Mohs** = Mohs micrographic surgery 莫氏显微外科手术
> - **5-FU** = 5-fluorouracil 5-氟尿嘧啶
> - **SLNB** = sentinel lymph node biopsy 前哨淋巴结活检

---

## 〇、核心决策：可疑恶性病变先活检 ⭐

> [!danger] 诊断步骤铁律（本组最常错的点）
> 怀疑皮肤恶性 → **先做确诊性活检 (biopsy first)** → 据组织学（类型 + 高危特征）再定切除范围。**不要**一上来"宽切除"或"安抚随访"。
> - **活检方式**：shave 削切 / punch 环钻 / **excisional 切除活检（小病灶首选——既确诊又近治疗）** / incisional 切取活检（大病灶取一部分）。
> - **错误选项面孔**：① "wide excision 1cm 切缘" = 黑色素瘤原位方案，用错癌种；② "sentinel lymph node biopsy" = 黑色素瘤分期用，BCC/SCC 极少转移不做；③ "reassurance/随访" = 延误诊断。

```
可疑皮肤新生物
   │
   ├─ 先活检确诊（shave/punch/excisional）
   │
   ├─ BCC ───────→ 4mm 切缘切除；面部/高危 → Mohs
   ├─ SCC（侵袭）─→ 4–6mm 切缘；面部/高危 → Mohs
   ├─ AK（癌前）──→ 孤立冷冻 / 弥漫 field therapy（5-FU/imiquimod/tirbanibulin）
   └─ Melanoma ──→ 全层活检定 Breslow → 按深度 1–2cm 切缘 ± SLNB
```

## 一、基底细胞癌 BCC（最常见皮肤癌）

- **机制**：慢性 UV 紫外线损伤 → **PTCH1（patched）/ Hedgehog 通路**异常；公平肤色、电离辐射也增风险。
- **形态**：**珍珠样卷边 (pearly rolled border) + 毛细血管扩张 (telangiectasia)** 的肤色/半透明结节，**上面部**多见；可中央溃疡（"rodent ulcer 鼠咬状溃疡"），**局部侵袭、极少转移**。

  ![[{B6750895-B7D3-4935-8EDD-24673970C65D}.png|360]]
  *基底细胞癌 (basal cell carcinoma, BCC)：**肤色/半透明珍珠样结节 (pearly nodule) + 表面毛细血管扩张 (telangiectasias)**，面部日晒区；通常**不鳞屑**（与 SCC 区分）。*

  ![[{BE49471B-7ED9-4C59-9333-18D5AD5B4A77}.png|360]]
  *基底细胞癌·结节型 (nodular BCC)：**珍珠样肤色结节 (pearly skin-colored nodule)** + 表面**毛细血管扩张 (telangiectasia)**（标注版指纹图）。*

  ![[{781C6174-76F7-4E91-AA2D-0F93D440929D}.png|360]]
  *基底细胞癌·三联征标注 (BCC, classic triad)：**卷边 (rolled border) + 毛细血管扩张 (telangiectasia) + 中央溃疡 (central ulceration / "rodent ulcer 鼠咬状溃疡")**，鼻旁日晒区——一图记全 BCC 三大识别点。*

- **诊断**：shave / punch / excisional **活检**。
- **治疗**：低危（躯干 <2cm）→ 切除 4mm 切缘 / 外用（imiquimod 咪喹莫特、5-FU）；**高危（面颈、≥2cm、边界不清、复发）→ Mohs 显微外科手术**。

![[{4B71919B-3B6E-4D27-ABAB-1F73108ADCCA}.png|420]]
*Mohs 显微外科手术 (Mohs micrographic surgery)：逐层切除 + 即时全切缘镜检，分次"薄层→镜检→再切残留"直至切缘阴性——**最大限度保留正常组织**，用于面部/高危/复发 BCC、SCC。*

- **Gorlin 综合征（基底细胞痣综合征）**：常染色体显性、**PTCH 突变** → 多发 BCC + 掌跖凹点 + 颌骨牙源性囊肿 + 叉状肋 + 大脑镰钙化 + 额部隆起。

## 二、鳞状细胞癌 SCC

- **机制**：UV / 电离辐射、**慢性瘢痕/创面/烧伤（Marjolin ulcer 马乔林溃疡）**、免疫抑制、**高危 HPV（生殖器/肛周）**。
- **形态**：鳞屑斑块/结节 ± 溃疡结痂；高分化=角化结节（组织学**角珠 keratin pearls**），低分化=牛肉红易出血。**唇 SCC 95% 在下唇红唇**（日晒），**慢性不愈溃疡**是恶性信号（疱疹 2 周自愈，拖一个月别当疱疹）。

  ![[{DB73BF25-13AD-49F1-BB30-8ECB5DBFB471}.png|360]]
  *鳞状细胞癌 (squamous cell carcinoma, SCC)：日晒区**角化性鳞屑结节 (scaly nodule)** ± **中央溃疡结痂**（箭头）——慢性日光损伤背景上持续不愈的肿物，慢性不愈即恶性信号。*

  ![[{EFF82E8D-A415-4D7A-934C-E0EB89B23E33}.png|360]]
  *皮肤鳞癌·低分化 (poorly differentiated cutaneous SCC)：**牛肉红 (beefy red)、易出血的隆起肿瘤**——分化越差越缺角化、越易出血。*

  ![[{000097CD-ACA8-4680-A159-9FC1798816D2}.png|360]]
  *鳞状细胞癌·高分化 (well-differentiated SCC)：日晒区**角化性丘疹结节 (hyperkeratotic papulonodule)**（标注版）——表面厚角质 / 角化是高分化 SCC 的标志（对比上图低分化的牛肉红无角化）。*

- **SCC in situ（Bowen 病）**：边界清红斑 + 厚鳞屑、长期不愈。
- **诊断**：punch/shave/excisional **活检**。治疗：侵袭性切 4–6mm 切缘 / Mohs；原位可切除 / 冷冻 / 外用 5-FU、imiquimod。**区域淋巴结肿大 → 疑转移**。

> [!question]- 既然 Bowen 要活检证明"没浸润"，那病灶不就被切光了？外用药还有什么意义？
> **关键：活检 (biopsy) ≠ 全片切除。活检只取"一小块"代表性组织做病理，病灶绝大部分仍留在皮肤上——那才是外用药 / 冷冻要清除的对象。**
>
> - **"看全层" ≠ "全部取材"**：全层 (full-thickness) 指**表皮的纵向深度**（基底→表面），不是病灶的**横向整片面积**。
> - 一个 **punch（环钻）** 直径才 3–4 mm，但它**纵向穿透整层表皮 + 部分真皮**——单凭这一小柱，病理就能判断"这一点上异型细胞有没有突破基底膜进真皮"。
> - 而 Bowen 斑片常 **1–几 cm**：取走 3 mm 那一块，**剩下的一大片病灶还在** → 这才是 5-FU / imiquimod / 冷冻 清除的目标。
>
> ```
> Bowen 斑片 (横向 2 cm)
> ┌───────────────────────────┐
> │ ▓▓▓▓▓▓▓ [punch 3mm] ▓▓▓▓▓▓│ ← 取一个 3mm 全层小柱送病理
> │ ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│
> └───────────────────────────┘
>         ↓ 病理：full-thickness atypia，基底膜完整，无浸润
>         ↓
>    剩下整片病灶 → 外用药 / 冷冻清除
> ```

> [!tip] 部分活检 vs 切除性活检——正好对应"用药 vs 不用药"
> | | incisional / partial（部分取材：shave/punch） | excisional（切除性活检） |
> |---|---|---|
> | 取多少 | 取**一块** | 把**整个**病灶切下来 |
> | 诊断后病灶 | **还在** → 之后涂药 / 冷冻 | **已没了** → 切除本身即治疗 |
> | 适合 | **大片** Bowen、面部等不宜大范围切 | **小而局限** |
>
> 用户那句"全部切除确诊后才用药"= **excisional 这一路**：一开始就端掉整片，既诊断又治愈，**不必再用药**。但病灶大 / 部位不宜切 → 先 **punch 取一块**确诊 in situ → 剩下整片用 **5-FU / imiquimod / 冷冻**。**这就是外用药存在的意义。**

> [!warning] 取样误差 (sampling error)——部分活检的真实局限
> 只取一块，万一病灶**别处**已浸润可能没采到。故：① 取材挑**最可疑处**（最厚 / 硬结 / 溃疡 / 出血）；② 临床高度疑浸润、或**外用治疗后不退** → **重新活检 / 改切除**；③ 这也是 **invasive SCC 一律手术切除**而不靠外用药的原因——外用药清不掉真皮浸润灶，也无法评估切缘。
>
> **钩子**：病变还"关在表皮里"(in situ/AK) → 取一小块取证后，剩下整片涂药/冷冻/切三选一；一旦"越狱"进真皮(invasive) → 只能切。

> [!note] SCC 的良性模拟者：角化棘皮瘤 Keratoacanthoma (KA)
> **角化棘皮瘤 (keratoacanthoma, KA)** = 中老年日晒/毛发区**快速生长 (数周)** 的**火山口样结节 + 中央角化栓 (volcano-like nodule with central keratotic plug)**；表面**干燥、不湿润/不易碎**。多数可自行消退，但临床/组织学与**高分化 SCC 难分** → 实务上**按 SCC 处理（切除活检）**。与化脓性肉芽肿 (pyogenic granuloma) 的湿润、易出血表面正好相反。

![[{288E6199-38B8-4F11-9AF5-78C0BCF14D6F}.png|320]]
*角化棘皮瘤 (keratoacanthoma, KA)：迅速增大的**火山口样结节 + 中央角化栓 (central keratotic plug)**，表面干燥——好发中老年毛发区；难与高分化鳞癌 (well-differentiated SCC) 区分，故按 SCC 切除处理。*

## 三、光化性角化病 AK（SCC 癌前）

- **机制**：累积 UV 损伤 → 基底角质细胞异型增生（癌前）。
- **形态**：日晒区（头皮/面/手背）**砂纸样粗糙鳞屑斑/斑块**，背景常有日光弹性变性（毛细血管扩张、深皱纹）。
- **诊断**：**临床**；高危特征（>1cm / 硬结溃疡 / 快速生长 / 触痛 / 治疗无效）才活检排 SCC。
- **治疗**：**孤立灶 → 冷冻 (cryotherapy)**；**弥漫/大面积 → field therapy（外用 5-FU 5-氟尿嘧啶 / imiquimod 咪喹莫特 / tirbanibulin 替拉那布林）**。

![[{2E85D935-9397-47F7-A9EC-662970EC39BB}.png|360]]
*光化性角化病 (actinic keratosis, AK)：面颊日晒区**砂纸样粗糙鳞屑斑 (scaly, sandpaper-like / papulonodular texture)**（图右箭头）+ 背景**深皱纹 (deep wrinkles, photodamage)**、毛细血管扩张等日光弹性变性（图左箭头）——SCC 鳞状细胞癌癌前病变。*

## 四、黑色素瘤 Melanoma（关键：深度定预后）

- **标志物**：S100。**ABCDE**：Asymmetry 不对称 / Border 边界不规则 / Color 颜色不均 / Diameter 直径>6mm / Evolving 演变。
- **预后**：**Breslow 厚度（深度）= 首要预后因素**。
- **诊断**：**全层（excisional / full-thickness）活检**——别削切（影响测深度）。治疗：按 Breslow 深度切 1–2cm 切缘 ± SLNB ± 全身治疗。
- **排查陷阱**：指甲褐色纵纹（尤其外伤史）→ 甲下黑色素瘤；眼内褐色灶 → 眼黑色素瘤；**无色素性黑色素瘤 (amelanotic) 可呈粉红结节**（故可疑病灶先活检）。

## 五、良性易混：脂溢性角化病 SK & Leser-Trélat

- **SK 脂溢性角化病**：**"贴附样 (stuck-on)" 蜡样棕色丘疹**，与日晒无关、良性；削切活检；冷冻/刮除。

  ![[{29F6DE92-4D39-424F-9A98-15AE2D335496} 1.png|320]]
  *脂溢性角化病 (seborrheic keratosis, SK)：表面粗糙、似**蜡样"贴附 (stuck-on)"**的棕色丘疹/斑块——良性，与日晒无关（对比 AK 砂纸样癌前）。*

- **Leser-Trélat 征**：SK **突然爆发性增多** = 副肿瘤（多胃肠腺癌）→ 查潜在恶性（上消化道内镜等）。**注意与"暴发型银屑病→查 HIV"区分**（见 [[完整笔记/专题笔记/皮肤/皮肤_银屑病]]）。

## 六、卡波西肉瘤 Kaposi Sarcoma（KS）

- **机制**：**HHV-8 人类疱疹病毒 8 型**驱动的高度血管化内皮肿瘤。
- **四型**：

```
经典型 ── 地中海/东欧老年；惰性，内脏罕见
地方型 ── 赤道非洲
医源型 ── 移植后免疫抑制（常第1年内）；减免疫抑制常自行消退
AIDS型 ── MSM、CD4<200；HAART 后改善
```

- **皮损**：**短期内多发**的**紫红/褐色斑→丘疹/结节**，**不痛、不痒、不坏死**，可伴淋巴水肿；移植后/AIDS 型易**内脏受累（肺、胃肠道）**。
- **处理**：**减少免疫抑制**（医源型）/ **HAART**（AIDS 型）；必要时局部或系统化疗（paclitaxel 紫杉醇 / 脂质体多柔比星 liposomal doxorubicin）。
- **鉴别**：黑色素瘤（病程长得多、不会短期多发）、血管瘤（婴儿）、BCC/SCC（短期多发不符）。

> [!warning] KS vs 恶黑（Malignant melanoma）——三轴判别（高频干扰项）
> 免疫抑制/移植患者**多种恶性肿瘤风险都升高**（皮肤癌排序其实 SCC > BCC > 黑色素瘤 > KS），所以"恶黑也可能"是对的——但选答案靠**临床指纹匹配**，不是"理论上有没有可能"。三条轴全部把恶黑排掉：
>
> | 判别轴 | Kaposi sarcoma (KS) | Malignant melanoma 恶黑 |
> |---|---|---|
> | **颜色** | **红 / 紫红 (violaceous)**，非色素性 | **棕黑色素性**，ABCDE、颜色不均 |
> | **数目** | **多发**、同时出现 | 通常**单发**原发灶 |
> | **病程 tempo** | **数周**爆发 | **数月～数年**缓慢演变 |
>
> KS 的特异性在于 **HHV-8 再激活** + 移植后**第 1 年内** + **多发红紫、不痛不痒、快速**——这套组合拳别的肿瘤复制不出来。只有看到**色素性、单发、慢**才往恶黑想。
> （注：amelanotic melanoma 无色素型可呈粉红结节——但仍是**单发、慢**，与 KS 的短期多发区分。）

![[{C0D2D9FE-5998-4896-9C8B-899577BFCB3B}.png|320]]
*卡波西肉瘤 (Kaposi sarcoma, KS)：移植后免疫抑制患者背部出现的**红/紫色斑→丘疹 (red-violaceous papule)**（箭头），早期常被误认为"擦伤/不退的瘀斑"；典型为**短期内多发、不痛不痒不坏死**。*

![[{DB3B565D-83E6-4BB2-9991-6FAED053994D}.png|300]]
*鉴别——婴儿血管瘤 (infantile hemangioma)：**鲜红色结节 (bright red nodule)**，婴儿期出现并自然增生-消退。与 KS 区别：发生于**婴儿**、单发、非"成人短期多发"——故成人短期多发紫红结节选 KS 而非血管瘤。*

## 六·补、良性 / 反应性血管增生：化脓性肉芽肿 & 杆菌性血管瘤

> [!info] 为什么放这里
> KS、化脓性肉芽肿、杆菌性血管瘤、婴儿血管瘤都是**红/紫色血管性丘疹结节**，靠**人群背景 + 病程 + 全身症状**区分。这组是 KS 与"友善 (friable) 易出血结节"的高频鉴别。

### 1. 化脓性肉芽肿 Pyogenic granuloma（= 分叶状毛细血管瘤 lobular capillary hemangioma）

- **本质**：良性**血管性肿瘤 (vascular tumor)**，由异常毛细血管 + 肉芽组织构成（**与感染/化脓无关**，名字是误称）。
- **人群/诱因**：青年多见，尤其**孕妇 (pregnant women)**（孕期血管内皮生长因子 VEGF 升高促其生长）；亦见于局部小创伤后。
- **临床**：小 (<1 cm)、**友善易碎、圆顶状的红色结节 (friable, dome-shaped red nodule)**，好发**手指/手、躯干、口腔黏膜/牙龈**（孕期牙龈更常见）；数周至数月内较快长成，**轻微外伤即明显出血 (bleeds with minor trauma)**。
- **诊断/处理**：临床诊断；治疗有手术切除、激光、外用硝酸银 (silver nitrate)；**孕期发生者产后多自行消退 (regress postpartum)**，可不处理。

  ![[{A67DDCEC-2DF8-4B9F-B357-5AC47484699A}.png|320]]
  *化脓性肉芽肿 (pyogenic granuloma / 分叶状毛细血管瘤 lobular capillary hemangioma)：**友善易碎、圆顶状红色结节 (friable dome-shaped nodule)**，表面湿润、轻触即出血——青年/孕妇好发于手指、躯干、口腔黏膜，孕期者产后多消退。*

### 2. 杆菌性血管瘤 Bacillary angiomatosis（鉴别）

- **病原/人群**：**巴尔通体 (Bartonella) 感染**，主要见于**晚期 HIV (CD4 显著低下)** 等免疫严重抑制者。
- **临床**：多发的小**红/紫色丘疹 → 友善结节 (friable nodular lesions)**，常伴**全身症状（发热、乏力、盗汗）**；可累及内脏（肝脾紫癜样病变 peliosis）。
- **鉴别钥匙**：单发友善结节、青年/孕妇、无全身症状 → 化脓性肉芽肿；**多发 + 免疫抑制 + 全身症状** → 杆菌性血管瘤（治以红霉素 erythromycin / 多西环素 doxycycline）。

  ![[{044AC21E-CCB4-460D-95CC-2EA03DD15A78}.png|320]]
  *杆菌性血管瘤 (bacillary angiomatosis)：多发小的**红/紫色丘疹演变为友善结节 (reddish-purple papules evolving into friable nodules)** + 全身症状——巴尔通体 (Bartonella) 感染，多见于晚期 HIV (human immunodeficiency virus, 人类免疫缺陷病毒)；与单发的化脓性肉芽肿区分。*

### 3. 婴儿血管瘤 Infantile hemangioma（增殖-消退 / 治疗指征）⭐

> 派生自 [[mistakes/uworld-mistakes_2026-07#^Q4404]]（误选 cherry angioma）。核心是**年龄 + 自然史**，别与成人血管性病灶混。

- **本质**：良性**毛细血管肿瘤 (benign capillary tumor)**，内皮细胞异常增殖（strawberry hemangioma 草莓状血管瘤）。
- **自然史（考点钥匙）**：**生后数天-数周出现**（出生时常无）→ **1 岁内快速增殖**（鲜红、软、隆起斑块/结节）→ **1 岁后自行消退 (spontaneous involution)**（数年内变淡/变平）。
- **处理**：**多数仅观察**（会自消）。**高危 → beta blocker β 受体阻滞剂 propranolol 普萘洛尔**：大/面部/节段/快速增长/**溃疡**、**眶周（视力损害）**、**肝（高输出心衰）**、**声门下（气道梗阻）**。
- **特殊评估**：**≥5 个皮肤病灶 → 肝超声**（内脏血管瘤）；**面部节段 → PHACE 综合征**（Posterior fossa 后颅窝、Hemangioma、Arterial 动脉、Cardiac 心脏、Eye 眼异常；查超声心动 + 头 MRI）；颈面"胡须"分布 → 喉镜；**腰骶部 → 脊髓超声**。
- **鉴别钥匙**：
  - **Cherry angioma 樱桃状血管瘤** = **成人**躯干小红点、随龄增多（非婴儿快速增大）。
  - **Nevus flammeus 鲜红斑痣 (port-wine stain)** = **出生即有、扁平、永不消退**（可伴 Sturge-Weber）。
  - **Nevus simplex（鲑鱼斑/鹳吻痕 salmon patch/stork bite）** = 眼睑/眉间/颈项，褪色、多 1-2 岁自消。
  - **Pyogenic granuloma 化脓性肉芽肿** = 小、易出血、圆顶丘疹，**不消退**（见本节 1）。

## 七、易混陷阱（普适）

- **诊断步骤**：可疑恶性 = **先活检**，别先宽切/安抚；1cm 切缘是黑色素瘤的尺，SLNB 是黑色素瘤分期。
- **砂纸样日晒鳞屑 = AK（会变 SCC）**；**贴附蜡样棕丘疹 = SK（良性）**——别混。
- **下唇慢性不愈溃疡 = SCC**，疱疹 2 周就好。
- **短期多发紫红结节不痛 + 免疫抑制/HIV = KS**；黑色素瘤是慢慢长的。
- **爆发性 SK（Leser-Trélat）→ 查内脏癌**；**暴发性银屑病 → 查 HIV**（别张冠李戴）。

## 八、自测（答完再判分）

> [!question]- Q1. 老年园丁颈后缓慢增大的珍珠样卷边 + 毛细血管扩张 7mm 结节。最佳下一步？
> 怀疑 BCC → **excisional biopsy 切除活检先确诊**（小病灶最佳）；确诊后 4mm 切缘。**不**直接 1cm 宽切除（那是黑色素瘤方案）。

> [!question]- Q2. 农民下唇红唇区持续 4 周的鳞屑丘疹伴中央溃疡。最可能诊断？
> **SCC 鳞状细胞癌**（下唇 95%、慢性不愈溃疡）；疱疹 2 周自愈，拖月不符。

> [!question]- Q3. 肾移植 3 月、用大剂量激素，躯干下肢短期多发红色无痛非痒结节。诊断 + 一线处理？
> **医源型 Kaposi sarcoma（HHV-8）** → **减少免疫抑制**（常自行消退）。

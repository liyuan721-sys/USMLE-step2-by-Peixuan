# 休克分类与 Next-Step 决策

> [!info] 用途
> USMLE 高频"休克病人下一步"题的统一框架：① 4 大类（+1 内分泌） ② 血流动力学指纹（CVP / PCWP / CO / SVR） ③ 每类一线 next-step ④ 易混镜像题。
> 锚点错题：[[mistakes/uworld-mistakes_2026-05#^Q4212]] AAA / [[mistakes/uworld-mistakes_2026-05#^Q4725]] PEA / [[mistakes/uworld-mistakes_2026-05#^Q12118]] PPCM / [[mistakes/uworld-mistakes_2026-05#^Q20254]] 心包填塞 / [[mistakes/uworld-mistakes_2026-05#^Q17169]] Addison 危象 / [[mistakes/uworld-mistakes_2026-05#^Q19929]] 高钾伪 brady。

## §零 速查 — 血流动力学指纹（核心金表）

| 休克类型 | **CVP / RA** | **PCWP** | **CO** | **SVR** | **皮温** |
|---|---|---|---|---|---|
| **Hypovolemic 低容量** | ↓ | ↓ | ↓ | ↑ | 冷 |
| **Cardiogenic 心源性** | ↑ | **↑** | ↓ | ↑ | 冷 |
| **Obstructive — Tamponade 心包填塞** | **↑（= PCWP）** | **↑（= CVP，相等）** | ↓ | ↑ | 冷 |
| **Obstructive — Massive PE** | ↑ | 正常 / ↓ | ↓ | ↑ | 冷 |
| **Obstructive — Tension PTX** | ↑ | ↓ | ↓ | ↑ | 冷 |
| **Distributive — Septic（早 / warm）** | ↓ | ↓ | **↑** | **↓↓** | **暖** |
| **Distributive — Septic（晚 / cold）** | ↓ | ↓ | ↓ | ↓ | 冷 |
| **Distributive — Neurogenic** | ↓ | ↓ | ↓ | ↓ | **暖 + 不代偿性 brady** |
| **Distributive — Anaphylactic** | ↓ | ↓ | 变 | ↓ | 暖 + 红疹 / 喘鸣 |
| **+ Adrenal Crisis（内分泌"休克"）** | ↓ | ↓ | ↓ | ↓ | 冷 |

> [!tip] 5 分钟分型 SOP
> ① **CVP / 颈静脉怒张？**
>   - 高 → cardiogenic / obstructive
>   - 低 → hypovolemic / distributive / adrenal
> ② **皮温？**
>   - 冷 → hypovolemic / cardiogenic / obstructive
>   - 暖 → distributive（septic warm / neurogenic / anaphylactic）
> ③ **CVP = PCWP？** → 心包填塞
> ④ **不代偿性 brady？** → 神经源性 / 高钾 / BB-CCB-Digoxin / 脊髓损伤

## §一 初始评估（Initial Approach）

```
休克病人 (SBP <90 / MAP <65 / 末梢灌注差)
   │
   ▼
1. ABC：气道、呼吸、循环；2 条大孔静脉、ECG、O2、监护
2. 测：BP / HR / RR / SpO2 / mental status / 末梢灌注 / 尿量
3. 抽血：CBC / chem / 乳酸 / 凝血 / 血气 / 血型配血
4. 床旁检查：FAST、心脏超声、ECG
   │
   ▼
分型（§零血流动力学 + §一速判）
   │
   ▼
特异性 next-step（§二-§六）
   │
   ▼
所有休克通用：纠正酸中毒 / 电解质、监测尿量、收 ICU
```

> [!warning] 通用 next-step 例外（铁律）
> - **心源性 + 严重 MR / 主动脉夹层** → **避免大量补液**（加重肺水肿 / 撕扩）
> - **心包填塞** → **不靠液体，靠 pericardiocentesis**（液体只是 bridge）
> - **Anaphylaxis** → **IM epi 第一步**，不是先升压药
> - **Tension PTX** → **针刺减压**（不靠液体不靠药）

## §二 Hypovolemic（低容量休克）

### 2.1 类型
- **出血性 (hemorrhagic)** — 创伤、GI 出血、产科、**ruptured AAA** ⭐（[[mistakes/uworld-mistakes_2026-05#^Q4212]]）
- **非出血性** — 严重脱水 / GI 丢失 / 烧伤 / 第三间隙（胰腺炎）

### 2.2 Next-Step

| 步骤 | 内容 |
|---|---|
| ① 控源 | 压迫止血 / 内镜止血 / OR / TXA / 烧伤创面处理 |
| ② 液体复苏 | **晶体液**（NS / LR）20-30 mL/kg 快速；评估反应 |
| ③ 输血 | 出血性 + Hb <7（或血流动力学不稳）→ 输 RBC；大出血 → **1:1:1 RBC:FFP:plt** |
| ④ 检查反应 | 反应不足 → 找漏诊（继续出血 / 心源性合并）|

> [!danger] Ruptured AAA 钥匙
> 男性老年 + 长期吸烟 / HTN + 突发腹/背痛 + 低血压 + 搏动性腹部包块 → **OR 直接送**（不等 CT 确诊；不稳定患者不做 CT）。详见 [[mistakes/uworld-mistakes_2026-05#^Q4212]]。

## §三 Cardiogenic（心源性休克）

### 3.1 病因（"泵坏了"）
- **大面积 MI（最常见）**，含 RV 梗死
- **机械并发症**：papillary muscle rupture → 急性 MR / VSR（[[mistakes/uworld-mistakes_2026-05#^Q12340]]）/ 游离壁破裂
- **PPCM / 暴发性心肌炎 / 心动过速性心肌病**（[[mistakes/uworld-mistakes_2026-05#^Q12118]] / [[mistakes/uworld-mistakes_2026-05#^Q4238]]）
- 重度心律失常 / 重度瓣膜病
- 心包填塞算 obstructive（见 §四）

### 3.2 Next-Step

| 步骤 | 内容 |
|---|---|
| ① 灌注 vs 充盈 | 经典 4 象限（湿冷 / 干冷 / 湿暖 / 干暖）—— 湿冷 = 典型 cardiogenic |
| ② 通气支持 | NIPPV / 必要时插管（湿冷常呼吸窘迫）|
| ③ Inotrope | **Dobutamine**（β1 激动）或 milrinone；血压允许时优先 inotrope |
| ④ Pressor | **Norepinephrine** 维持 MAP（不靠 phenylephrine — α 单独↑后负荷压垮心脏）|
| ⑤ 减后负荷 | 血压允许 → 硝酸甘油 / 硝普 |
| ⑥ 病因治疗 | MI → **紧急 PCI**；机械并发症 → **紧急外科**；心律失常 → 同步电复律 |
| ⑦ 机械支持 | IABP / Impella / VA-ECMO（refractory）|

> [!warning] Cardiogenic 不能大量补液
> 心源性休克给 1L NS 可能致死。**先做床旁心超**（评估 EF / 机械并发症 / 心包 / RV），再决定液体。
> 例外：**RV 梗死**（下壁 MI + RA 怒张 + 清肺 + 低 BP）→ **需要适量补液 + 避免硝酸甘油**（依赖前负荷）。

### 3.3 PEA / 心搏停（[[mistakes/uworld-mistakes_2026-05#^Q4725]]）

```
PEA（电活动有 / 无脉搏）
  │
  ▼ ACLS：高质量 CPR + Epi 1mg q3-5min
  │
  ▼ 找可逆因 5H + 5T
  5H: Hypovolemia / Hypoxia / H+ acidosis / Hyper-Hypokalemia / Hypothermia
  5T: Tension PTX / Tamponade / Thrombosis (PE/MI) / Toxins / Trauma
```

## §四 Obstructive（梗阻性休克）

### 4.1 心包填塞（Cardiac Tamponade）— [[mistakes/uworld-mistakes_2026-05#^Q20254]]

| 维度 | 内容 |
|---|---|
| **病因** | 恶性肿瘤（最常见非创伤）/ 主动脉夹层 / 急性 IE / 心脏穿透伤 / 心脏术后 / 尿毒症 / 急性 MI 游离壁破裂 |
| **三联（Beck）** | **低血压 + 颈静脉怒张 + 心音遥远** |
| **奇脉 (pulsus paradoxus)** | 吸气期 SBP↓>10 mmHg |
| **ECG** | **Electrical alternans**（QRS 电压交替高低）+ low voltage |
| **心超** | 心包积液 + **RV 舒张期塌陷** + IVC plethora |
| **血流动力学** | **CVP = PCWP**（等压）|
| **治疗** | **Pericardiocentesis**（紧急；血流动力学不稳时不等影像）+ 容量 + inotrope bridge |

### 4.2 大面积肺栓塞（Massive PE）

- 突发胸痛 / 呼吸困难 / 低氧 + 颈静脉怒张 + **不明原因休克** + RV 衰竭
- **ECG**：S1Q3T3、新发 RBBB、sinus tachy
- **影像**：CTA-PA 或不稳定时床旁超声（RV 扩张 / McConnell 征）
- **治疗**：**tPA 溶栓**（不稳定 / hemodynamic compromise）；不能溶 → **embolectomy**（外科 / 介入）
- 稳定 PE → 抗凝（LMWH / DOAC）

### 4.3 张力性气胸（Tension PTX）

- 单侧呼吸音消失 + 气管偏向健侧 + 颈静脉怒张 + 低血压
- **治疗**：**立即针刺减压**（2nd ICS 锁骨中线 — 成人；或 5th ICS 腋前线），再放胸管
- 不靠胸片确诊（临床诊断）

## §五 Distributive（分布性休克）— 关键鉴别群

### 5.1 Septic Shock（脓毒症休克）

| 步骤 | 内容（Sepsis-1 Hour Bundle）|
|---|---|
| ① 测乳酸 | 起点；治疗中复查 |
| ② 血培养 ×2 | 抗生素前抽 |
| ③ **广谱抗生素 ≤1 小时** ⭐ | 经验性覆盖（基于 source）|
| ④ **晶体液 30 mL/kg** | 灌注差 / 乳酸 ≥4 时 |
| ⑤ Pressor | MAP <65 不达标 → **Norepinephrine 一线**；refractory → 加 vasopressin / epi；难治性 → hydrocortisone |
| ⑥ Source control | 引流脓肿 / 拔感染导管 / OR |

> [!tip] 暖性 vs 冷性脓毒症
> - **早期 / 暖**：心排↑↑（高动力）+ SVR↓↓ → 暖、洪脉
> - **晚期 / 冷**：心排↓（心肌抑制） → 冷
> 都用 norepinephrine 升 MAP；冷期加 inotrope 支持。

### 5.2 Neurogenic Shock（神经源性休克）

| 维度 | 内容 |
|---|---|
| 病因 | **脊髓损伤 ≥T6**（高位）切断交感传出 |
| **关键指纹** | **低血压 + 不代偿性 brady**（伴 warm peripheries）⭐ |
| **vs septic** | septic 早期是 tachy；neurogenic 是 brady（交感切断 → 心脏失去 β1 驱动）|
| 治疗 | ① **IV fluids**（先复容）→ ② **Vasopressor with α**（norepi / phenylephrine 恢复 SVR）③ Atropine 治 brady ④ Spinal precautions / OR |

> [!warning] 不代偿性 brady + 低血压 鉴别
> 见到 "BP 80/40 + HR 60"，三个第一反应：
> 1. **高钾**（Sine wave、P 波消失 — Q19929）
> 2. **Neurogenic shock**（脊髓损伤史、warm）
> 3. **BB / CCB / Digoxin 中毒**（药物史）

### 5.3 Anaphylactic Shock — vs Adrenal Crisis（[[mistakes/uworld-mistakes_2026-05#^Q17169]]）

| 维度 | **Anaphylaxis** | **Adrenal Crisis** |
|---|---|---|
| 起病 | 暴露后秒-分 | 漏药 / 应激 / 长期糖皮质激素停用后 |
| 皮肤 / 黏膜 | **荨麻疹 / 喉头水肿 / 喘鸣** ⭐ | 无皮疹；继发性无色素沉着；原发性有 |
| 肺 | 喘鸣 / 支气管痉挛 | 清 |
| 电解质 | 一般正常 | **低 Na + 高 K + 低糖**（原发型；继发型仅低糖低 Na）|
| **第一步治疗** | **IM Epinephrine 0.3-0.5 mg 大腿外侧** ⭐ | **IV Hydrocortisone 100 mg** ⭐ + IV NS |
| 反应液体 / 升压 | 可能反应；但 epi 先 | **对液体 / 升压药反应极差**（除非给皮质醇）|

> [!danger] 低血压 + 对液体/升压药反应差 → Adrenal crisis
> 这是 Q17169 的核心。**继发性 / 医源性**（长期外源糖皮质激素停用）→ **无色素沉着**（ACTH 低）。原发性（Addison）→ **有色素沉着**。
> 第一步：先给 hydrocortisone（不等确诊），再走 workup。

## §七 易混 / 高频陷阱

| 陷阱 | 真相 |
|---|---|
| 所有休克给 1L NS 试试 | ❌ 心源性 / 心包填塞 / tension PTX 都不靠液体；anaphylaxis 先 IM epi |
| Brady + 低 BP = sinus brady 给 atropine | ❌ 先排高钾（P 波消失）+ BB/CCB/digoxin 中毒 + 神经源性 |
| 脓毒症先用 phenylephrine | ❌ 一线 **norepi**；phenylephrine 不提高 CO |
| 心源性休克给 phenylephrine 升压 | ❌ 单纯 α 激动 → 后负荷↑ → 心脏更崩；用 norepi（α+β1）|
| Anaphylaxis 给 IV epi push | ❌ 一线 **IM 0.3-0.5 mg 大腿外侧**（IV 用于 refractory，剂量更低）|
| 不稳定 AAA 先 CT | ❌ 不稳定 → 直接送 OR；稳定 → CT 确诊 |
| 心包填塞先 OR | ❌ **Pericardiocentesis 第一步**（除非外科原因如夹层 / 创伤）|
| 张力性气胸先拍 CXR | ❌ **临床诊断 → 立即针刺减压**，再拍 CXR / 放胸管 |
| RV 梗死给硝酸甘油 / morphine | ❌ RV 依赖前负荷；给液体，避血管扩张 |
| Cushing reflex（HTN + brady + 不规则呼吸）= 休克 | ❌ ICP↑警告，不是休克；查神经 + 头 CT |
| 大量补液后还低血压 = 加更多液体 | ❌ 评估反应；考虑 hemorrhage 持续、cardiogenic 合并、adrenal crisis、anaphylaxis、neurogenic、sepsis 需 pressor |

## §八 易混镜像题对照

| 镜像对 | 鉴别钥匙 |
|---|---|
| Anaphylaxis vs Adrenal crisis | 皮肤 / 喉水肿 vs 应激史 + 对液体反应差 |
| Septic warm vs Neurogenic | tachy vs **brady**（脊髓损伤）|
| 心包填塞 vs 张力性气胸 | 心音 vs 呼吸音 / 气管偏移 |
| RV 梗死 vs 心包填塞 | 都 RA↑ / 清肺；ECG 下壁 STEMI 选 RV vs 电交替选填塞 |
| 高钾伪 brady vs 真 sinus brady | P 波**消失** vs P 波在 / P-QRS 解离 |
| Cardiogenic vs PE | 心电图 / 心超：LV vs RV 主导 |
| Hypovolemic 出血 vs 第三间隙 | 显性出血 vs 胰腺炎 / 烧伤 |

## §九 Memory Hooks

- **血流动力学一句话**：
  - **Cold + JVD up** → 心源性 / 梗阻性
  - **Cold + JVD down** → 低容量
  - **Warm + JVD down** → 分布性（septic warm / neurogenic / anaphylactic）
- **Septic 1-hour bundle**：**乳酸 → 培养 → 抗生素 → 30 mL/kg → norepi**
- **Anaphylaxis = "皮 + 喘 + 休 = 大腿打针"**（IM 大腿外侧 epi）
- **Adrenal crisis = "对液体不响应的低血压 + 应激史 = 先给皮质醇"**
- **心包填塞 = Beck 三联 + 电交替 + 等压 → 立刻穿刺**
- **张力 PTX = 单侧呼吸消失 + 气管偏 + JVD → 针刺先于 CXR**
- **不代偿 brady = 高钾 + 神经源 + BB/CCB/Dig 中毒"三连查"**

## §零 速查 — ECG 表现 → 第一反应病因

| ECG buzzword | 第一反应 | 验证线索 |
|---|---|---|
| **Sine wave** | 重度高钾 | K >8、肾衰、ACEi+spiro+KCl 三联 |
| **Peaked T**（窄尖对称）| 高钾（早期）| K 6-6.5；vs 超急期 MI（宽不对称 + ST↑）|
| **U 波** | **低钾 / 低镁** | 利尿剂、呕吐、CYP 干扰 |
| **长 QT (>500ms)** | **低 K / 低 Mg / 低 Ca / 药物 / 先天 LQTS** | TdP 风险 |
| **短 QT** | **高钙 / 高钾 / Digoxin 治疗效应** | 评估病因（高钙 phase2 / 高钾 phase3）|
| **Osborn (J) 波** | **低温**（<32°C）⭐ / SAH / 高钙 / ICH | 测体温！ |
| **Scooped ST ("Dali")** | Digoxin **治疗剂量效应**（≠ 中毒）| 看用药 |
| **Bidirectional VT** | **Digoxin 中毒** ⭐ | 老人 + 黄视 + 低 K |
| **Wide QRS + R 波 aVR** | **TCA 中毒** | 抗胆碱综合征 |
| **Delta 波 + 短 PR** | **WPW** | 心悸 / preexcited AF |
| **Deep symmetric T 倒置 + 长 QT** | **ICH / SAH**（神经源性）| 头痛、瞳孔异常 |
| **Brady + 低血压 + P 波正常** | β-blocker / CCB / Digoxin 中毒 | 药物史 |
| **Brady + 低血压 + P 波消失** | **高钾**（伪 brady）| 立即查 K + 给 Ca |

> [!danger] 一句话原则
> "看似 brady"/"看似 TdP"/"看似 MI" — **必查电解质 + 用药史 + 体温**。代谢/中毒/神经源性常伪装成原发心脏病。

## §一 钾 — 最致命的 ECG 元凶

### 1.1 高钾 ECG 5 阶段（顺序必背）

| K+ | ECG | 紧迫性 |
|---|---|---|
| 5.5-6.0 | 可能正常 | 低 |
| 6.0-6.5 | **Peaked T**（窄、尖、对称，胸前导联）| 中 |
| 6.5-7.5 | **P 变平 + PR 延长** | 中-高 |
| 7.5-8.0 | **P 消失 + QRS 增宽** | 高 |
| >8.0 | **Sine wave**（QRS 与 T 融合呈正弦波）| **濒死** |
| 极高 | VFib / Asystole / PEA | 心搏停 |

**口诀**：**T**ent → **P**ass → **W**ide → **S**ine → **F**latline。
**急救**：ECG 一异常 → **Calcium gluconate（葡萄糖酸钙）不等血钾**（详见 [[完整笔记/专题笔记/肾脏/肾脏_高钾血症完整体系]]）。

> [!warning] Peaked T：高钾 vs 超急期 MI
> - **高钾 peaked T**：**窄、尖、对称**，胸前导联最明显，**全导联**广泛存在
> - **超急期 MI ("hyperacute T")**：**宽、不对称、底基增宽**，**局限于缺血区导联**，伴 ST↑
> - 鉴别钥匙 = 形态 + 分布 + 伴随改变（ST/Q 波）

### 1.2 低钾 ECG

```
低 K 进展：
正常 → T 变平 / 倒 → ST 压低 → U 波出现（T 后小波）→ QT/QU↑ → TdP / VFib
```

- **U 波** = 病理性出现在 T 波后；T-U 融合时易误认为长 QT
- 常伴 PR / QT 延长、室早、TdP
- **低 K + 低 Mg = 致命组合**（同时缺则补不上，TdP 高发）
- Digoxin 治疗中的低 K → digoxin 中毒（Q4454）

## §二 钙 — QT 的尺子

### 2.1 高钙 → **短 QT**

- 校正 QTc 缩短（<350ms）
- 重度：J 波 / 类 Osborn 波；可室颤
- 鉴别：恶性肿瘤（PTHrP / 溶骨）/ 原发甲旁亢 / 肉芽肿 / 噻嗪 thiazide / VitD（维生素 D）中毒

### 2.2 低钙 → **长 QT**

- 校正 QTc 延长 → **TdP 风险**
- 常伴 Chvostek / Trousseau / 抽搐 / 喉痉挛
- 病因：甲旁减、CKD、急性胰腺炎、TLS、大量输血、低镁
- 治疗：症状性 / 抽搐 → **IV calcium gluconate（葡萄糖酸钙）**（不用 chloride 氯化钙）

> [!tip] Ca 与 QT 的关系
> 高钙 = QT 短；低钙 = QT 长（"高短低长"）。和钾相反方向：**高 K → T 高尖**；**低 K → T 平 + U 波**。

## §三 镁

- **低镁**：长 QT → **TdP**；常伴低 K（低 Mg 致低 K 难补）
- **高镁**：PR / QRS / QT 全延长 → 严重时心搏停；DTR↓ 首发；产科 eclampsia（子痫）MgSO4（硫酸镁）治疗时监测

> [!danger] TdP 急救铁律
> **TdP = IV Magnesium 一线**（无论 Mg 水平）。同时纠正基础长 QT 原因（低 K / 低 Ca / 药物）。

---

## §四 体温 — Osborn 波

### 4.1 低温（Hypothermia, T <35°C）

- **Osborn (J) 波** ⭐：QRS 终点与 ST 起点之间的圆顶状正向偏移（J 点抬高）
- 越冷越明显（T <32°C 显著）
- 伴 sinus brady、PR/QRS/QT 延长、房颤
- 处理：**Slow rewarming**（核心温度 <30°C 时心律失常对去颤反应差，先复温）

### 4.2 Osborn-like 波的其他病因

- **高钙**（重度）
- **SAH / ICH** 神经源性
- **Brugada 综合征**（V1-V2 的 J 波 + ST 抬高）

## §五 药物中毒 ECG

### 5.1 Digoxin（地高辛）

| 状态 | ECG |
|---|---|
| **治疗剂量效应**（不是中毒）| **Scooped ST**（"Salvador Dali mustache" — ST 下斜呈胡须状）+ **短 QT** + T 倒置 |
| **中毒** ⭐ | **任何心律失常都可能**：**bidirectional VT**（指纹）、PAT with block、各种 AV block、室早、严重 brady、VF |

**中毒诱因**（Q4454）：**低 K / 低 Mg / 高 Ca** + 利尿剂 diuretics / 胺碘酮 amiodarone · 维拉帕米 verapamil · 奎尼丁 quinidine + 肾衰。
**急救**：**Digoxin 特异性 Fab 抗体（地高辛免疫 Fab）**；纠正电解质（**注意 Ca 慎用** — 经典"stone heart"传统教学）。

> [!warning]- 诱因双机制 + 钾的方向陷阱（Q2 延伸，易混必看）
> 上面"诱因"一行**混了两套独立机制**，钾的方向也分急慢性，别搞混：
>
> **① 低 K / 低 Mg → 药效学 (pharmacodynamic)**
> 地高辛和 K⁺ **竞争钠钾泵 (Na⁺/K⁺-ATP 酶) 同一结合位点**。血钾↓ → 竞争对手少 → 地高辛结合↑ → 泵抑制↑ → 毒性↑。**利尿剂泛指排钾的袢（呋塞米 furosemide）+ 噻嗪（thiazide）**，不是专指噻嗪。
> （故慢性中毒若血钾不高 → 补钾把地高辛挤下泵。）
>
> **② 肾衰 → 药代动力学 (pharmacokinetic)**
> 地高辛**经肾排泄**，肾衰 → 清除↓ → **药物蓄积、血药浓度↑** 致毒。**与钾无关** —— 肾衰的高钾是"肾排钾↓"另一回事，不是它致地高辛毒的途径。
>
> **③ 高 Ca → 药效学**：协同增加细胞内钙 → 心律失常↑。
>
> > [!danger] 钾方向反转（慢性 vs 急性）
> > - **慢性中毒**：低钾是**因**（利尿剂致低钾 → 诱发）→ 可补钾
> > - **急性大剂量中毒**：高钾是**果**（全身钠钾泵被阻 → K⁺ 泵不进细胞 → 血钾↑）；**K >5.5 = 预后差 = Fab 指征**；此时**禁补钾**（已高钾，再补致命）

### 5.2 TCA（三环类抗抑郁药）中毒

- **宽 QRS**（>100ms = 心律失常风险；>160ms = 室速 / 室颤风险）
- **R 波 aVR >3mm** ⭐（钠通道阻断的指纹）
- 长 QT、sinus tachy、抗胆碱综合征（散瞳、口干、尿潴留）
- **治疗**：**NaHCO3（碳酸氢钠）IV**（碱化 + 钠负荷反转钠通道阻断）

### 5.3 β-Blocker / CCB 中毒

| 维度 | β-Blocker（β受体阻滞剂）| CCB（非二氢吡啶 — 维拉帕米 verapamil / 地尔硫卓 diltiazem）|
|---|---|---|
| **BP** | 低 | 低 |
| **HR** | brady | brady |
| **血糖** | **低**（β2 抑制糖原分解）| **高**（胰岛素分泌依赖 Ca）|
| **ECG** | PR↑、AV block | 同 |
| **解药** | **Glucagon（胰高血糖素）** ⭐（绕开 β 受体激活 cAMP）| **Calcium 钙**（高剂量）/ insulin-glucose（胰岛素-葡萄糖）/ lipid emulsion（脂肪乳）|

### 5.4 可卡因 cocaine / 安非他命 amphetamine

- Sinus tachy、HTN、宽 QRS（钠通道阻断）、长 QT
- **胸痛 + 可卡因 = 不给 β-blocker（β受体阻滞剂）**（无对抗 α → 冠脉痉挛↑）
- 用 BZD（苯二氮卓）+ 硝酸甘油 nitroglycerin + Ca 阻滞剂（钙通道阻滞剂）

## §六 神经源性 ECG — ICH / SAH

- **深、对称、广泛的 T 倒置**（"cerebral T waves"）
- **长 QT**（→ TdP 风险）
- 偶可见 ST 抬高 / 压低、Q 波（假 MI）
- 机制：交感风暴 → 神经源性心肌损伤（**stress cardiomyopathy / Takotsubo** 类似）
- 钥匙：突发头痛 + 神经体征 → 头 CT 先于心脏 workup

## §七 ECG 形态学速查表

### 7.1 T 波形态

| T 波 | 病因 |
|---|---|
| 高尖窄对称 | **高钾**（早期）|
| 高宽不对称 + ST↑ | **超急期 MI** |
| T 平 / 倒 | 低 K / 缺血 / 全身病 / 神经源性 |
| 深对称广泛 T 倒置 | **ICH / SAH** / massive PE |
| Wellens 双相 / 深倒置 V2-V3 | LAD 临界狭窄 |

### 7.2 QT 长短

| QT | 病因 |
|---|---|
| 长 QT | **低 K / 低 Mg / 低 Ca**、药物（**列单**：抗心律失常 IA/III、azithromycin 阿奇霉素 / fluoroquinolones 氟喹诺酮 / TCA 三环类 / SSRI / 抗精神病药 ziprasidone 齐拉西酮 / methadone 美沙酮 / ondansetron 昂丹司琼）、ICH/SAH、先天 LQTS |
| 短 QT | **高钙**（缩 phase2 平台期）、**高钾**（加速 phase3 复极）、digoxin 治疗效应、先天短 QT 综合征 → 纵深见 [[完整笔记/专题笔记/心内/心内_QT间期长短与TdP]] |

### 7.3 PR / QRS 异常

| 异常 | 病因 |
|---|---|
| 短 PR + delta 波 | **WPW** |
| 长 PR（单一）| 1 度 AV block：迷走、训练有素、药物（BB/CCB/digoxin）、缺血、电解质 |
| Wenckebach（PR 渐长后脱落）| Mobitz I — 良性 |
| PR 固定突然脱落 | Mobitz II — 需起搏器 |
| 宽 QRS（无 BBB）| TCA / 高钾 / 钠通道阻断剂 → 鉴别+解毒见 [[完整笔记/专题笔记/心内/心内_宽QRS鉴别与解毒]] |
| Sine wave | 重度高钾 |
| J / Osborn 波 | 低温 / 重度高钙 / SAH / Brugada |

### 7.4 心律 / QRS 形态特殊

| 心律 | 病因 |
|---|---|
| **Bidirectional VT** | **Digoxin 中毒** ⭐ / CPVT |
| **Torsades de pointes** | 长 QT 任何病因 |
| **Brugada pattern** | 钠通道病；coved ST V1-V2 |
| 不规则宽 QRS（preexcited AF）| **WPW + AF** → 用 procainamide（普鲁卡因胺），禁 AV 结阻滞药 |

## §八 易混题型对照

| 看到... | 别只想第一反应 |
|---|---|
| Brady + 低血压 + 老人 | 排除高钾 sine wave（P 波消失）、digoxin / BB / CCB 中毒 |
| TdP | 不要"治心律失常" — 先 **IV Mg**，再纠低 K / 低 Ca / 撤药 |
| 长 QT 在 ICU 老人 | 扫药物清单（azithro 阿奇霉素 / FQ 氟喹诺酮 / haloperidol 氟哌啶醇 / methadone 美沙酮）+ 电解质 |
| 心律失常 + 阿尔茨海默 / 老人服多药 | 想 digoxin 中毒（黄视 / 恶心 / 心律失常三联）|
| 突发头痛 + ECG 异常 | 头 CT 先（SAH 致心电）|
| 复温前体温 30 → Osborn 波 | 不电除颤无效；先复温至 30+ |
| 抗抑郁药过量 + 宽 QRS | TCA → **NaHCO3** |
| 心悸 + 不规则宽 QRS | WPW + AF → procainamide（普鲁卡因胺）；**禁** adenosine 腺苷 / BB / CCB / digoxin |

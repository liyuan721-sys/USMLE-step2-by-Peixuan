# 暴发性肺水肿 (Flash Pulmonary Edema) 五类触发鉴别

> [!info] 一句话总览
> **Flash pulmonary edema（暴发性/急骤肺水肿）= 一个终末通路，不是一个病**：左房/肺静脉压**骤升**超过淋巴清除能力 → 肺泡瞬间灌水。背后几乎都是「**僵硬/不耐受的左室（舒张功能障碍）+ 一个急性触发**」。特点：来得快、治了就好、触发不除则**反复发作**。按触发分 5 类记最省力。
>
> **触发来源**：源自 RAS（renal artery stenosis 肾动脉狭窄）→ flash 机制讨论延伸（见 [[完整笔记/专题笔记/心内/心内_继发性高血压鉴别框架]] §5.4）。

> [!note]- 本篇缩写速查（首次全称中英）
> - **SCAPE** = sympathetic crashing acute pulmonary edema 交感激增型急性肺水肿
> - **ACS** = acute coronary syndrome 急性冠脉综合征
> - **MR / AR / MS** = mitral regurgitation 二尖瓣反流 / aortic regurgitation 主动脉瓣反流 / mitral stenosis 二尖瓣狭窄
> - **AF with RVR** = atrial fibrillation with rapid ventricular response 房颤伴快速心室率
> - **HCM** = hypertrophic cardiomyopathy 肥厚型心肌病；**LVH** = left ventricular hypertrophy 左室肥厚
> - **HFpEF** = heart failure with preserved ejection fraction 射血分数保留型心衰
> - **RAS** = renal artery stenosis 肾动脉狭窄；**SAH** = subarachnoid hemorrhage 蛛网膜下腔出血

---

## 一、统一机制

```
僵硬左室（舒张功能障碍 / 非顺应性心室）
          +
急性触发（后负荷骤升 / 容量骤增 / 心率骤快 / 瓣膜骤坏）
          ▼
左房压瞬间爆表 → 肺泡水肿 (flash) → 治疗后缓解、触发再犯
```

---

## 二、五类触发 ⭐

> [!info] ① 后负荷 / 交感骤升砸向僵硬左室（最常见 = SCAPE）
> - **高血压急症 (hypertensive emergency)** ← **最常见**；BP 骤升 → 后负荷爆 → 治疗 = **血管扩张剂（硝酸甘油 nitroglycerin）+ 无创通气 (NIV)**
> - **双侧肾动脉狭窄 (bilateral RAS) = Pickering 综合征**（见下方 [!danger]）
> - **嗜铬细胞瘤 (pheochromocytoma)** / 儿茶酚胺激增 → 后负荷↑ + 儿茶酚胺心肌病
> - **神经源性肺水肿 (neurogenic pulmonary edema)**：SAH、癫痫、颅脑外伤后交感风暴

> [!info] ② 急性心肌缺血
> - **急性冠脉综合征 / 心梗 (ACS / acute MI)** → 一过性左室功能崩 或 **缺血性二尖瓣反流 (ischemic MR)** → flash
> - 钩子：胸痛 + ECG 改变 + 肌钙蛋白 (troponin)↑

> [!danger] ③ 急性瓣膜灾难（容量骤砸进没扩张的腔室）— 别漏！
> - **急性二尖瓣反流 (acute MR)**：**乳头肌断裂 (papillary muscle rupture，心梗后)** / 腱索断裂（黏液样变性、感染性心内膜炎 infective endocarditis）→ 左房/左室来不及代偿 → 暴发肺水肿
> - **急性主动脉瓣反流 (acute AR)**：感染性心内膜炎、**主动脉夹层 (aortic dissection)** → 左室容量骤超载
> - 招牌：**突发肺水肿 + 新出现的杂音 + 近期 MI / 发热 / 夹层背景**

> [!info] ④ 舒张功能差 + 心动过速骤发
> - **新发房颤伴快速心室率 (AF with RVR)** / 室上速：心率快 → 舒张充盈时间骤短 + 丢心房收缩 → 僵硬左室（LVH / HCM / 老年 HFpEF）瞬间失代偿

> [!info] ⑤ 二尖瓣狭窄 (MS) + 诱因
> - 固定梗阻，平时代偿；**新发房颤 / 妊娠 / 发热 / 运动**一来 → 左房压骤升 → 肺水肿
> - 经典：**年轻孕妇突发气短**（妊娠血容量↑ 撞上风湿性 MS）

## 三、Pickering 综合征（双侧 RAS → 反复 flash）⭐⭐

> [!danger] 反复 flash = 双侧 RAS，"没有健肾泄压阀"
> RAS 同时拉高**前负荷 + 后负荷**砸向僵硬左室：① **血管紧张素 II (angiotensin II)↑** → 血管收缩 → 后负荷↑；② **醛固酮 (aldosterone)↑ + 压力性尿钠排泄受损 (impaired pressure natriuresis)** → 钠水潴留、容量↑；③ 慢性 HTN → 左室肥厚 / 舒张障碍。
> - **单侧 RAS**：对侧**正常肾**能"压力性排钠"当泄压阀 → 容量不爆 → 一般**不**发 flash。
> - **双侧 RAS（或孤立功能肾的单侧 RAS）**：两肾都在狭窄远端"看到"低灌注 → 都不排钠 → **无泄压阀 → 容量爆 → 反复 flash** = **Pickering 综合征**。
> - 临床意义：多数动脉粥样硬化性 RAS 药物为主（CORAL / ASTRAL 试验），但 **反复 flash pulmonary edema 是公认的血运重建（revascularization / 支架）指征**。

## 四、快速鉴别钥匙

| 线索 | 指向 |
|---|---|
| BP 极高、无新杂音 | 高血压急症 (SCAPE) |
| **反复** flash + 腹部连续杂音 + ACEi 后 SCr↑ | **双侧 RAS (Pickering)** |
| 胸痛 + ECG 改变 + 肌钙蛋白↑ | ACS / 缺血性 MR |
| **突发 + 新杂音** | 急性 MR/AR（瓣膜灾难） |
| 阵发头痛 + 心悸 + 出汗 + 阵发 HTN | 嗜铬细胞瘤 |
| ECG 房颤快室率 | AF 诱发（舒张障碍底子） |
| 年轻孕妇 / 风湿史 + 舒张期隆隆样杂音 | 二尖瓣狭窄 + 诱因 |

## 五、非心源性"快速肺水肿"尾巴（机制不同，鉴别时偶露面）

> [!warning] 这些不是真"心源性 flash"，但起病也快
> - **复张性肺水肿 (re-expansion pulmonary edema)**：大量胸水/气胸**快速抽吸**后
> - **负压性肺水肿 (negative pressure pulmonary edema)**：拔管后**喉痉挛**用力吸气
> - **高原肺水肿 (HAPE, high-altitude pulmonary edema)**
> - **输血相关 (TACO 输血相关循环超负荷 / TRALI 输血相关急性肺损伤)**

## 六、Memory Hook

> [!success] 一句话锁定
> - **Flash = 僵硬左室 + 急性触发**；最常见 = **高血压急症**。
> - **反复 flash → 想双侧 RAS (Pickering)**；**突发 + 新杂音 → 别漏急性瓣膜灾难（乳头肌/腱索断裂、心内膜炎、夹层）**。
> - 治标共性：**坐起 + 吸氧/无创通气 + 血管扩张剂（硝酸甘油）+ 利尿**；治本看触发。

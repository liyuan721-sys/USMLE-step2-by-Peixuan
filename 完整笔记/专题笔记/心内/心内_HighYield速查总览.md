## 一、胸痛鉴别 + 冠心病（Chest pain & CAD）

### 1.1 胸痛鉴别速认（钥匙 → 病）

| 钥匙 | 病 | 下一步 |
|---|---|---|
| 体位性（坐起减轻）+ 放射肩 + 摩擦音 + 弥漫 ST 抬高/PR 压低 | 心包炎 pericarditis | Echo（排压塞/破裂）|
| 撕裂样、放射到**背**、两臂血压不等、脉搏不对称 | 主动脉夹层 aortic dissection | 见 §七 |
| 吞咽相关、corkscrew 食道造影 | 食管痉挛 esophageal spasm | 最准 = 测压 manometry |
| 局部按压痛、可重复 | 肋软骨炎 costochondritis | ECG（best initial）排除心源 |
| 扩瞳 + 胸痛 + 拟交感 | 可卡因 cocaine | benzodiazepine + 硝酸酯，**禁 β-blocker** |
| 不可预测、濒死感、过度通气 | 惊恐发作 panic attack | 排器质后心理治疗 |

> [!tip] 稳定型心绞痛 (stable angina) 评估路径
> 门诊**压力测试 (stress test)** 是核心。糖尿病人"干活胃疼、休息缓解" = 不典型心绞痛，术前先做压力测试，别只当胃病。
> - 不能运动 / 用助行器 → **药物负荷**
> - 慢阻肺 COPD (chronic obstructive pulmonary disease) / 哮喘 → 禁腺苷/双嘧达莫，用多巴酚丁胺；起搏器 / 左束支阻滞 LBBB (left bundle branch block) → 影像选择有讲究。

### 1.2 ACS / MI 处理骨架

```
胸痛 → ① best initial = 心电图 ECG (electrocardiogram) + MONA(吗啡 morphine/氧 oxygen/硝酸甘油 nitroglycerin/阿司匹林 aspirin)
        ⚠️ 下壁 MI / RV 梗死 / 用过西地那非 → 禁硝酸甘油（前负荷依赖）
   │
   ├─ ST 抬高 (STEMI) → PCI 若 door-to-balloon <90 min
   │                    重大延迟 → 溶栓（无 PCI 条件）  ※MI 不做 CTA！CTA 是 PE 的
   │
   └─ 无 ST 抬高 → serial ECG(15–30min) + serial troponin
                  酶阳 = NSTEMI ; 酶阴 + 持续症状 → 床旁 echo ; ACS 后 → 压力测试
```

> [!warning] 围术期 MI (perioperative MI)
> 非心脏手术 + 术中出血需输血（心肌供氧↓）→ 常**无胸痛**（术后镇痛掩盖）；处理 = **补液**，禁溶栓。

🧩 心梗 MI (myocardial infarction) 并发症时间轴（再梗 / 乳头肌断裂 / 室间隔破裂 VSR, ventricular septal rupture / 游离壁破裂 / Dressler）→ [[完整笔记/专题笔记/心内/心内_MI并发症时间轴]]
🧩 抗凝抗板与急性动脉血栓 → [[完整笔记/专题笔记/心内/心内_抗凝抗板_急性血栓]]
🧩 降脂药与 ASCVD 一级/二级预防、他汀强度分层、statin 不耐受 → [[完整笔记/专题笔记/心内/心内_降脂药与ASCVD]]

> [!tip] 稳定性缺血性心脏病 SIHD (stable ischemic heart disease) 二级预防（↓MI 风险）
> 抗血小板（阿司匹林 aspirin）+ 高强度他汀（阿托伐他汀 atorvastatin / 瑞舒伐他汀 rosuvastatin）+ 降压控糖戒烟运动。
> **冠脉搭桥 CABG (coronary artery bypass grafting) 指征**：三支病变 / 糖尿病 / 左主干。

## 二、心律失常（Arrhythmias）

### 2.1 房颤 AF (atrial fibrillation) 三步走

```
① 稳定性？ 不稳（低血压/急性心衰/严重胸痛）→ 立即同步电复律（不看心律）
② 稳定 → rate control 先于 rhythm（"rate is atrium, rhythm is ventricle"）
   ├─ 无心脏病史 → CCB（地尔硫䓬/维拉帕米）most common
   ├─ 长期 HTN + S4 → β-blocker（美托洛尔 metoprolol）
   └─ 扩张心/S3/HFrEF + 低血压 → 地高辛 digoxin
③ 抗凝：CHA₂DS₂-VASc ≥2 → 长期抗凝（首选 DOAC 如 阿哌沙班 apixaban）
   ⚠️ 中重度二尖瓣狭窄 or 机械瓣 → 必须华法林 warfarin，禁 DOAC
```

> [!danger] 复律前抗凝铁律
> AF >48h 或时长不明 → 复律前必须标准抗凝（或 TEE 排血栓），否则栓塞风险。
> 特殊：年轻 + 新发 AF → 查 **TSH**（甲亢）；酗酒后新发 AF → **holiday heart syndrome**（无需抗凝）；CABG/瓣膜术后 AF 常见。

### 2.2 SVT / WPW / 室速 决策

| 心律 | 钥匙 | 处理 |
|---|---|---|
| 窄 QRS 规则（AVNRT/AVRT）| 阵发心悸、骤起骤停 | 迷走动作 → **腺苷 adenosine** |
| WPW + 预激 AF | 宽、不规则、delta 波 | **普鲁卡因胺 procainamide**；禁腺苷/β/CCB/地高辛（阻 AVN 逼电流走旁路 → 室颤）|
| 宽 QRS 室速 (VT) 稳定 | 有脉、宽规则 | 普鲁卡因胺 / 胺碘酮 amiodarone |
| 室速有脉**不稳** | 低血压 | 同步电复律（不 CPR）|
| 无脉 VT / VF | 无脉 | **除颤 + CPR + 肾上腺素 epinephrine** → 胺碘酮/利多卡因 |
| PEA / asystole | 有电无脉 / 平线 | CPR + 肾上腺素（找 H&T 病因：大块 PE / 张力气胸 / 低血容量）|

> [!tip] 多源性房速 (MAT, multifocal atrial tachycardia)
> 乱而有 P、≥3 种 P 波形态、率 >100；多继发肺病/低氧 → **先治病因**，节律常自愈。

🧩 宽 QRS 鉴别与药物解毒 → [[完整笔记/专题笔记/心内/心内_宽QRS鉴别与解毒]]
🧩 抗心律失常药 Vaughan-Williams 全表 → [[完整笔记/专题笔记/心内/心内_Vaughan_Williams抗心律失常药]]

### 2.3 缓慢心律 + AV 阻滞

| 阻滞 | ECG 钥匙 | 处理 |
|---|---|---|
| 一度 | PR 固定延长 >0.20s | 观察 |
| 二度 Mobitz I (文氏) | PR **渐长 → 脱落** | 多良性，有症状先**阿托品 atropine** |
| 二度 Mobitz II | PR 固定、**随机脱落** | 高危 → 起搏器（先药后起搏）|
| 三度（完全）| **房室分离 AV dissociation** + cannon a 波 | 起搏器 |

> [!tip] "发热但心率正常" (Faget sign / 相对缓脉) → 想伤寒、军团菌、药物等；完全阻滞也可见 cannon a 波。

🧩 AV Block 完整鉴别 → [[完整笔记/专题笔记/心内/心内_AV_Block完整鉴别]]
🧩 β受体阻滞剂分类大全 → [[完整笔记/专题笔记/心内/心内_BetaBlockers分类大全]]

### 2.4 长 QT / TdP + 电解质 ECG

> [!warning] 尖端扭转 (TdP, torsades de pointes)
> 纠正诱因：**补钾、补镁**；优化基础药首选 β-blocker；**禁 I 类抗心律失常药**。先天性长 QT（如 Jervell & Lange-Nielsen 伴耳聋）。

| 电解质 | ECG 钥匙 |
|---|---|
| 高钾 hyperkalemia | 高尖 T → QRS 增宽 → 正弦波 |
| 低钾 hypokalemia | U 波、ST 压低（钾困细胞内 → 复极延迟）|
| 低钙 hypocalcemia | QT 延长 |
| 高钙 hypercalcemia | QT 缩短 |

## 三、瓣膜病 + 杂音（Valvular disease & murmurs）

### 3.1 听诊手法（核心反射）

```
站立 / Valsalva → 静脉回流↓ preload↓ 室腔变小
   └─ 杂音变响的两个例外：HOCM（梗阻加重）、MVP（脱垂提前）
Handgrip → afterload↑ → 多数杂音变软；但 MR/VSD/AR 变响
Squatting → preload↑ + afterload↑ → 室腔变大 → HOCM/MVP 变软
吸气 → 右心杂音增强（TR）；呼气 → 左心杂音增强
```

### 3.2 四大瓣膜 next-step

| 瓣膜病 | 钥匙 | best initial / 最准 | 手术指征 |
|---|---|---|---|
| 主动脉狭窄 AS (aortic stenosis) | 喷射性收缩杂音放射颈动脉、pulsus parvus et tardus | TTE / TTE | **有症状**（晕厥/心绞痛/心衰）→ AVR；severe = 瓣口 <1.0 cm²，脉压 <25 mmHg |
| 主动脉反流 AR (aortic regurgitation) | 舒张早期递减杂音、宽脉压 | TTE | 无症状 ACEI 降后负荷；有症状 → AVR |
| 二尖瓣狭窄 MS (mitral stenosis) | 舒张中期隆隆音、风湿史 | TTE / TEE | — |
| 二尖瓣脱垂 MVP (mitral valve prolapse) | mid-systolic click + 晚收缩杂音 | TEE 确认 | — |

> [!tip] 病因 / 急性反流钥匙
> - 老年 AS = 钙化；年轻 AS = 二叶瓣；MS = 风湿（绝对第一）。
> - **急性 AR** = 夹层 / 感染性心内膜炎 IE / 外伤；**急性 MR** = MI 后乳头肌断裂 / IE（无代偿扩张时间 → 突发肺水肿）。
> - **Heyde 综合征**：老年 + AS + vWF 减少 + 血管发育不良消化道出血 → AVR 可纠正。
> - 新杂音/疑瓣膜病 → **TTE**；疑 IE / 人工瓣 / 看赘生物或瓣周漏 → **TEE**。

> [!tip] 人工瓣选择
> 生物瓣 (bioprosthetic)：寿命短、不需长期抗凝（老年/想怀孕）；机械瓣 (mechanical)：耐久、需终身华法林。

## 四、心衰 + 心肌病（HF & cardiomyopathy）

### 4.1 HFrEF vs HFpEF

射血分数降低性心衰 HFrEF (heart failure with reduced ejection fraction) vs 射血分数保留性心衰 HFpEF (heart failure with preserved ejection fraction)。

| | 收缩性 HFrEF | 舒张性 HFpEF |
|---|---|---|
| 射血分数 EF (ejection fraction) | ↓ | 保留 |
| 心音 | S3 | S4 |
| 治疗 | 指南导向药物治疗 GDMT 四联（见下）| 控压控率、利尿对症 |

> [!success] HFrEF GDMT (guideline-directed medical therapy)（改善生存）
> 一线 β受体阻滞剂 β-blocker；加 血管紧张素转换酶抑制剂/血管紧张素受体拮抗剂 ACEI/ARB；加 螺内酯 spironolactone；加 钠-葡萄糖共转运蛋白2抑制剂 SGLT-2 inhibitor；地高辛 digoxin = 二线（控症状不改善生存，AF+HFrEF+低血压时有用）。
> **起始顺序（高频陷阱）**：失代偿 / 明显淤血期**先利尿 + ACEI/ARB**；**β 阻滞剂待容量稳定（euvolemic）后再起始或加量**（淤血时起始 β 阻滞剂的负性肌力会加重低心输出）。〔源：NBME11A Q109 → [[NBME11_错题本#^Q109]]〕

> [!danger] 急性失代偿心衰 ADHF (acute decompensated heart failure) — LMNOP
> 急性肺水肿 = Lasix(呋塞米 furosemide) + Morphine + Nitrates + Oxygen + Position；目标 = 改善氧合 + 降肺毛细血管压 + 减前后负荷 + 治病因。

### 4.2 心肌病谱（钥匙锁定）

| 类型 | 钥匙 | 处理 |
|---|---|---|
| 扩张型 DCM (dilated cardiomyopathy) | 心大、EF↓、functional MR、心内血栓；化疗心毒看 MUGA (multigated acquisition) 核素扫描 | GDMT；酒精性 AIC (alcoholic cardiomyopathy) → 戒酒可逆 |
| 肥厚型 HOCM (hypertrophic obstructive cardiomyopathy) | 常染色体显性、左室流出道 LVOT (left ventricular outflow tract) 梗阻、站立/Valsalva 杂音变响、运动猝死 | **β-blocker**（增舒张充盈时间）；重症间隔切除/消融；EF≤35% 或持续 VT → 植入式心律转复除颤器 AICD (automatic implantable cardioverter-defibrillator) |
| 限制型 RCM (restrictive cardiomyopathy) | 淀粉样变 amyloidosis（"厚心壁 + 低电压"矛盾、巨舌/腕管/蛋白尿）| 治原发 |
| 应激性 Takotsubo | 老年女性受刺激、像心梗但冠脉通 | 支持治疗 |

> [!tip] Kussmaul 征 vs 奇脉 (pulsus paradoxus)
> Kussmaul = 吸气时 JVP 反升（右心"装不进"：缩窄性心包炎 / RCM / RV 梗死）；奇脉 = 吸气时收缩压掉 >10 mmHg（压塞典型）。

## 五、心包病 + 休克（Pericardium & shock）

### 5.1 心包炎分型 → 治疗

> [!danger] 新发胸痛第一步永远是 Echo（排压塞 / 游离壁破裂）

| 类型 | 时机 / 钥匙 | 首选治疗 |
|---|---|---|
| 急性心包炎 | 弥漫 ST 抬高 + PR 压低、摩擦音 | NSAID + 秋水仙碱 colchicine |
| 梗死后早期 (<4天) | MI 后 1–4 天 | **Acetaminophen**；**禁 NSAID/激素**（增破裂风险）|
| PCIS / Dressler | 损伤后数周、免疫性 | 大剂量阿司匹林 + 秋水仙碱；Dressler 慎抗凝 |
| 尿毒症性 | BUN >60、**ECG 常无 ST/PR 改变** | 血液透析 dialysis；禁抗凝 |
| SLE 相关 | SLE 活动期 | NSAID；肾功不全改激素 prednisone |

> [!tip] 缩窄性心包炎 (constrictive pericarditis)
> 钥匙：Kussmaul + 心包叩击音 pericardial knock + 陡 y 降 + 呼吸依赖；vs RCM（心肌问题、BNP 更高、不受呼吸影响）。

### 5.2 心脏压塞 + 休克分类

> [!danger] 心脏压塞 (cardiac tamponade)
> Beck 三联（低血压 + 颈静脉怒张 + 心音遥远）+ 奇脉 + 电交替 electrical alternans。最准 = echo。
> BP 正常 → 心包穿刺 pericardiocentesis；**BP 低 → 先补液 fluids**。

中心静脉压 CVP (central venous pressure) / 肺毛细血管楔压 PCWP (pulmonary capillary wedge pressure) / 心输出量 CO (cardiac output)。

| 休克 | CVP/PCWP | CO | 钥匙 |
|---|---|---|---|
| 心源性 cardiogenic | PCWP↑ | ↓ | 多为大面积 MI |
| 梗阻性（压塞/张力气胸/大 PE）| CVP↑ | ↓ | 肺前型（PE/气胸）PCWP↓；肺后型（左心/主动脉）PCWP↑ |
| 低血容量 hypovolemic | ↓ | ↓ | 左室变小、EF 反而高（"没血可打"）|

## 六、高血压（Hypertension）

### 6.1 分级 → 处理（阈值照源）

```
Elevated BP   120–129 / <80      → 生活方式
Stage 1 HTN   130–139 或 80–89   → 生活方式；若 ASCVD/CKD/DM 或 10 年 ASCVD 风险高 → 加药
Stage 2 HTN   ≥140 或 ≥90        → 生活方式 + 药物；若 ≥20/10 高于目标 → 2 药起始
```

> [!tip] 一线选药（无合并症）+ 合并症定制
> 无合并症：ACEI/ARB、噻嗪 thiazide、二氢吡啶 CCB（氨氯地平 amlodipine）任一。
> - CAD / 心绞痛 → β-blocker / CCB；MI 后 → ACEI/ARB + β-blocker
> - CKD / 糖尿病肾病 → ACEI/ARB
> - HFrEF → ACEI/ARB + β + 利尿 + SGLT-2 + 螺内酯
> - 痛风 → 氯沙坦 losartan（促尿酸排泄）/ CCB，避噻嗪；骨质疏松/钙肾结石 → 噻嗪；偏头痛 → β/CCB
> - 生活方式中 **DASH 饮食 (Dietary Approaches to Stop Hypertension)** 降收缩压最强（约 11 mmHg）；戒烟不显著降压但最防终点事件。

### 6.2 继发性高血压（何时查）

> [!warning] 触发线索（不是人人都查）
> 年龄 <30 + HTN / 顽固性（≥3 药仍不控）/ 特殊症状。
> - **低钾** → 原发性醛固酮增多症（醛固酮↑ → 保钠排钾）
> - **肾动脉狭窄 RAS (renal artery stenosis)**：老年男性 = 动脉粥样硬化；年轻女性 = 纤维肌性发育不良 FMD (fibromuscular dysplasia)
> - 年轻 + LVH + 鼻出血/头痛 + 上肢压高下肢压低、股动脉弱 → **主动脉缩窄 coarctation**

## 七、主动脉急症 + 外周血管（Aortic & vascular）

### 7.1 主动脉夹层 (aortic dissection)

```
撕裂样胸痛放射背 + 两臂血压不等 + 脉搏不对称 ± 神经缺损
最初 = CXR(纵隔增宽)；最准 = CT/TEE
治疗第一步：先降压（拉贝洛尔 labetalol + 硝普钠 nitroprusside）
   ├─ 升主动脉 (Stanford A) → 手术
   └─ 降主动脉 (Stanford B) → 药物降压（顺序：先 β 降 HR 再扩血管，防反射性心动过速）
```

### 7.2 腹主动脉瘤 (AAA, abdominal aortic aneurysm)

| 情境 | 下一步 |
|---|---|
| 疑 AAA、血流动力学稳定 | 先床旁/聚焦超声，确诊后 CT 测大小 |
| 不稳定 + 已知 AAA | 直接急诊修补 |
| 不稳定 + 疑 AAA 无既往史 | 床旁超声 |

> [!tip] AAA 手术阈值：男 **5.5 cm** / 女 **5.0 cm** / 半年增 **>0.5 cm**；戒烟最减缓扩张。股动脉瘤 ≥3 cm 或远端栓塞 → 手术；发现一处动脉瘤要查全身（常伴 AAA / 腘动脉瘤）。

### 7.3 外周动脉 / 静脉

| 病 | 钥匙 | 处理 |
|---|---|---|
| 外周动脉病 PAD (peripheral artery disease) | 间歇跛行；踝肱指数 ABI (ankle-brachial index) 诊断 | 戒烟（一线）+ 阿司匹林/他汀 + 监督下运动 → 西洛他唑 cilostazol（**禁心衰**）→ 血运重建 |
| 急性肢体缺血 ALI (acute limb ischemia) | 6P、栓子（AF）| 立即 **IV 肝素 heparin** 防血栓蔓延，再影像/手术 |
| 慢性静脉功能不全 | 内踝溃疡、不痛、棕色色素沉着 | 抬腿 + 加压 |
| Leriche 综合征 | 跛行 + 阳痿 + 股动脉搏动消失 | 主髂闭塞 |

## 八、感染性心内膜炎（Infective endocarditis）

| 类型 | 病原 / 瓣膜 | 经验治疗 |
|---|---|---|
| 急性 acute | 金黄色葡萄球菌 S. aureus，健康瓣（静脉吸毒者 IVDU, intravenous drug user → 三尖瓣）| 万古霉素 vancomycin + 庆大霉素 gentamicin |
| 亚急性 subacute | 草绿色链球菌 S. viridans，受损二尖瓣 | 同上 |

> [!danger] IE + 新发传导阻滞 / 新主动脉反流杂音 = 瓣周脓肿 (perivalvular abscess)
> → 立即 **TEE** + 手术咨询。主动脉瓣更易瓣周脓肿→新发 AV block；二尖瓣更易腱索断裂→急性心衰。
> 手术指征：急性心衰（新 AR/MR → 肺水肿）、瓣周脓肿、栓塞等。
> 血培养阳性查 **Strep gallolyticus（bovis）** → 查结肠癌。

## 九、晕厥 + 运动猝死（Syncope & exertional)

> [!tip] 劳力性晕厥分诊
> - 老年 → 排 **severe AS**（看颈动脉搏动）
> - 青年 → 排 **HOCM**（Valsalva 试验）
> - 有心脏病史 → 警惕 **VT**
> - 首选检查 = **Echo 看结构** → ECG、生化/电解质。

| 类型 | 钥匙 |
|---|---|
| 血管迷走性 vasovagal | 诱因明确、前驱症状 → 无需进一步检查 |
| 体位性 orthostatic | 站立 BP 明显下降（容量/自主神经）|
| 心律失常性 | 无前驱、有心脏病史 |
| 心因性假性晕厥 | 长时间意识丧失 LOC (loss of consciousness)、无客观体征、la belle indifférence |

## 十、先天性心脏病（Congenital heart disease）

### 10.1 发绀 vs 非发绀 + 导管依赖

```
发绀型（5 个 T 起手）：TGA / TOF / Tricuspid atresia / Truncus / TAPVR
   ├─ 平行循环 → TGA（出生 2h 重度发绀、给氧无效、egg-on-a-string、single loud S2）
   ├─ 右心梗阻（肺血少）→ TOF（boot-shaped、tet 发作、蹲位缓解）/ Tricuspid atresia（左轴+高尖P）/ critical PS
   ├─ 左心梗阻（导管依赖体循环）→ 左心发育不全综合征 HLHS (hypoplastic left heart syndrome) / critical coarctation / critical AS（导管关闭后休克样）
   └─ 混血/肺血多 → Truncus / 梗阻型 完全性肺静脉异位引流 TAPVR (total anomalous pulmonary venous return)（发绀 + 肺水肿）
导管依赖 → 前列腺素 E1 PGE1 (prostaglandin E1) 保持 动脉导管未闭 PDA (patent ductus arteriosus) 开放
```

### 10.2 综合征 → 缺损（必背 8 组）

| 综合征 | 缺损 |
|---|---|
| Down 唐氏 | 房室间隔缺损（完全型 CAVSD, complete atrioventricular septal defect：fixed split S2 + 6 周喂养出汗心衰）|
| DiGeorge | TOF / truncus / 主动脉弓中断 IAA (interrupted aortic arch)（+ 低钙抽搐 + 反复感染）|
| Turner | 主动脉缩窄 coarctation |
| Williams | 瓣上型主动脉狭窄（+ elfin 面容 + 高钙）|
| Noonan | 肺动脉瓣狭窄 |
| 先天性风疹 rubella | PDA（机器样连续杂音）|
| Marfan | 主动脉根部扩张 |
| Holt-Oram | ASD |

## 十一、复习题（自测）

> [!question]- AF + 中度二尖瓣狭窄患者，CHA₂DS₂-VASc = 3，抗凝该选什么？
> **华法林 warfarin**，不能用 DOAC。中重度 MS 或机械瓣是 DOAC 的禁忌。

> [!question]- 老年女性受情绪刺激后胸痛、ECG 像 STEMI，但冠脉造影通畅。诊断 + 处理？
> **Takotsubo（应激性心肌病）**；支持治疗，多可逆。"老妇受刺激，像心梗，冠脉却通"。

> [!question]- 下壁 MI 患者，给硝酸甘油后血压骤降。为什么？
> 下壁 MI 常累及右室，**前负荷依赖**；硝酸甘油降前负荷 → 严重低血压。下壁/RV 梗死、用过西地那非时禁硝酸甘油。

> [!question]- WPW 合并预激性房颤（宽、不规则、delta 波），首选药？为何不用腺苷？
> **普鲁卡因胺 procainamide**。阻断 AVN 的药（腺苷/β/CCB/地高辛）会逼电流全走旁路 → 心室率飙升诱发室颤。

> [!question]- 感染性心内膜炎患者出现新发 PR 间期延长 / 新主动脉反流杂音，提示什么？下一步？
> **瓣周脓肿**；立即 **TEE** + 手术咨询。

> [!question]- AAA 男性手术阈值？女性？快速增长定义？
> 男 5.5 cm / 女 5.0 cm / 半年增 >0.5 cm。

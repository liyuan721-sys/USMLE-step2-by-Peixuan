## Part 1 — 核心方法论（Why & How）

### 1.1 一句话原则

> [!danger] 核心反射
> **老年慢病 + 多药 + 新发症状 = 先怀疑药，再怀疑病**

**为什么**：
1. 老年人药物种类多 → 副作用 / 中毒的**先验概率**显著高于普通患者
2. 药物之间互相加重（如 furosemide → 低钾 → 加重 digoxin 毒性）
3. 病情急性加重通常表现"戏剧化"（急性肺水肿、晕厥、胸痛）
4. **中毒往往是"慢性渗透式"症状**：数日的恶心 / 乏力 / 食欲↓ / 意识改变

### 1.2 阅读 stem 的新顺序

```
Step 1: 基础疾病 → 推测标准用药套餐
        （即使 stem 没列全药，也要心里有个"应该在吃啥"的预期）
   │
   ▼
Step 2: 用药列表 → 逐个药"心里过一遍"副作用谱
        （把每个药当成"嫌疑人"，不是背景信息）
   │
   ▼
Step 3: 新发症状 → 问"哪个药能解释最多症状？"
        （Occam's Razor：一个原因解释所有症状 > 多个原因）
   │
   ▼
Step 4: 诱因 → 老年 / AKI / 脱水 / 互作 → 让谁的浓度飙升？
        （触发因素 = 中毒的"扳机"）
   │
   ▼
Step 5: 第一步诊断 → 测可疑药物血药浓度 / 相关电解质 / 肾功
        （**不是** 心超 / CT / 复杂检查）
```

### 1.3 触发"查用药"反射的临床信号

> [!warning] 触发条件
> 看到 stem 中出现以下**任意**组合，立刻**逐药排查副作用**：

| 基础疾病 | + 新发症状 |
|---|---|
| HFrEF + AFib | GI 症状（恶心、食欲↓） |
| HFrEF | 乏力 / 意识改变 |
| AFib（在用 rate control 药） | 心动过缓 / AV block |
| CKD（任何分期） | 非特异性多系统症状 |
| 老年慢病（任何） | 数日 - 数周亚急性发作 |
| 肝硬化 | 意识改变 / 嗜睡 |
| 移植后 | 任何新症状（直到证伪） |

### 1.4 "病情加重 vs 药物毒性"快速鉴别

| 维度 | 病情急性加重 | 药物毒性 |
|---|---|---|
| 起病速度 | 数小时 - 1 天 | 数日 - 数周 |
| 症状方向 | 单一系统戏剧化（如急性肺水肿）| 多系统、非特异（GI + 神经 + 心血管） |
| 触发 | 不依从、感染、心肌缺血 | 脱水、AKI、新加药、剂量改变 |
| 体征 | JVD ↑、水肿 ↑、湿啰音 ↑、SpO2 ↓ | 浓度依赖（如缓慢心率、震颤） |
| 第一步 | BNP、CXR、ECG、心超 | **血药浓度、电解质、肾功** |

## Part 2 — 5 大高频疾病组合的"标准用药套餐"

### 2.1 HFrEF + AFib

```
HFrEF + AFib
   │
   ├── 心率控制 (Rate control)
   │     ├── β-blocker (metoprolol succinate, carvedilol, bisoprolol)
   │     └── Digoxin (二线，β-blocker 不够或低血压时加)
   │
   ├── HFrEF GDMT (指南导向药物治疗)
   │     ├── ARNI (sacubitril-valsartan) 或 ACEi / ARB
   │     ├── β-blocker (与上面重叠)
   │     ├── MRA (spironolactone, eplerenone)
   │     └── SGLT2i (dapagliflozin, empagliflozin)
   │
   ├── 容量管理
   │     └── Loop diuretic (furosemide, torsemide, bumetanide)
   │
   └── 抗凝 (AFib 卒中预防, CHA₂DS₂-VASc ≥ 2)
         └── DOAC (apixaban, rivaroxaban) 或 warfarin
```

> [!tip] 经典 stem 模式
> "X-year-old with HFrEF (EF 25%) and AFib on furosemide, sacubitril-valsartan, metoprolol, digoxin, and apixaban..."
> → 看到这串药，立刻识别为 HFrEF + AFib 标准套餐

### 2.2 CKD（任何分期）

```
CKD
   │
   ├── 血压控制
   │     ├── ACEi / ARB（首选，肾保护）
   │     └── CCB / β-blocker / 利尿剂（必要时加）
   │
   ├── 血糖控制（若 DM）
   │     ├── SGLT2i（肾保护）
   │     ├── GLP-1 RA
   │     └── 避免 Metformin（GFR < 30）
   │
   ├── 容量 / 电解质
   │     ├── Loop diuretic
   │     ├── 不用 K-sparing（高钾风险）
   │     └── 磷结合剂、钙剂
   │
   ├── 贫血
   │     └── ESA (erythropoietin) ± 静脉铁
   │
   ├── CKD-MBD（矿物质骨病）
   │     ├── Vit D analogs (calcitriol)
   │     └── Cinacalcet（继发性甲旁亢）
   │
   └── 心血管预防
         └── Statin
```

> [!danger] CKD 患者的"剂量陷阱"
> **任何经肾排泄的药物都要按 GFR 调整剂量**，否则容易蓄积中毒：
> - Digoxin、Lithium、Gabapentin / Pregabalin
> - Metformin（GFR<30 禁用）、Allopurinol
> - 多数 abx（vancomycin, aminoglycosides, β-lactams）
> - LMWH（enoxaparin, GFR<30 减量）、DOAC

→ ACEi/ARB 双通路机制详见 [[完整笔记/专题笔记/心内/心内_降压药副作用速查#2.2 ACEi/ARB 副作用机制（AngII 双通路 — 高钾 + AKI + Type 4 RTA 三联）]]

### 2.3 Cirrhosis（肝硬化）

```
Cirrhosis
   │
   ├── 腹水 / 容量
   │     ├── Spironolactone（首选）
   │     ├── Furosemide（加用，比例 100:40）
   │     └── Albumin（大量穿刺后 / SBP / HRS）
   │
   ├── 静脉曲张预防 / 治疗
   │     ├── Non-selective β-blocker (propranolol, nadolol, carvedilol)
   │     └── EVL (内镜套扎)
   │
   ├── 肝性脑病 (HE)
   │     ├── Lactulose（首选）
   │     └── Rifaximin（二线 / 维持）
   │
   ├── SBP 预防
   │     └── Norfloxacin / Ciprofloxacin
   │
   └── 病因治疗
         ├── HCV: DAAs (sofosbuvir-velpatasvir)
         ├── HBV: Entecavir, Tenofovir
         └── ALD: 戒酒、营养支持
```

> [!danger] Cirrhosis 患者的药物陷阱
> - **肝代谢药物半衰期延长**：benzo、opioid → 蓄积 → 加重 HE
> - **NSAIDs 禁用**：诱发 AKI / HRS、消化道出血
> - **Aminoglycosides 禁用**：肾毒
> - **意识改变首先想**：HE（药物诱发的 HE！）→ 查 NH₃、用 lactulose

→ 门脉高压 Non-selective β-blocker 必选规律详见 [[完整笔记/专题笔记/心内/心内_BetaBlockers分类大全#7.1 门脉高压必须 Non-selective]]

### 2.4 老年 Polypharmacy（多药 ≥ 5 种）

> [!info] Beers Criteria 必背的"老年应避免药物"
> （非全面，挑高频考点）

| 药物类别 | 代表药 | 老年陷阱 |
|---|---|---|
| **抗胆碱能** | Diphenhydramine, Oxybutynin, TCA | 谵妄、便秘、尿潴留、跌倒 |
| **苯二氮卓** | Diazepam, Alprazolam | 跌倒、谵妄、依赖 |
| **Z-drugs** | Zolpidem | 跌倒、复杂睡眠行为 |
| **第一代抗精神病** | Haloperidol（长期） | EPS、QT 延长、跌倒 |
| **Muscle relaxants** | Cyclobenzaprine | 抗胆碱 + 镇静 |
| **NSAIDs（长期）** | Ibuprofen, Naproxen | AKI、GIB、HTN 加重 |
| **Sulfonylureas（长效）** | Glyburide | 低血糖（首选 glipizide）|
| **α-blockers（首选降压）** | Doxazosin | 直立性低血压 → 跌倒 |
| **Digoxin（高剂量）** | > 0.125 mg/day | 中毒（同样肾下降） |
| **PPI（> 8 周无指征）** | Omeprazole | C. diff、骨折、低镁、肺炎 |

> [!warning] 老年 Polypharmacy 的"多米诺骨牌"
> ```
> 多药 → 任一药副作用 → 加另一药对症 → 新副作用 → 又加药...
>      ↓
>   Prescribing cascade（处方瀑布）
>      ↓
>   经典例子：CCB → 水肿 → 加利尿剂 → 低钾 / 跌倒 / AKI
> ```

### 2.5 实体器官移植后（SOT）

```
移植后免疫抑制方案（典型三联）
   │
   ├── Calcineurin Inhibitor (CNI)
   │     ├── Tacrolimus（首选，绝大多数）
   │     └── Cyclosporine
   │
   ├── Antimetabolite
   │     ├── Mycophenolate (MMF)
   │     └── Azathioprine
   │
   └── Corticosteroid
         └── Prednisone（低剂量维持）

+ 预防性抗感染（视移植种类 / 时间）：
   ├── TMP-SMX（PJP 预防，移植后 6-12 月）
   ├── Valganciclovir（CMV 预防，3-6 月）
   └── Nystatin / Fluconazole（真菌预防）
```

> [!danger] 移植后新症状 = 灾难性鉴别
> **3 大方向，按时间窗鉴别**：

| 移植后时间 | 优先怀疑 |
|---|---|
| < 1 月 | 手术并发症、医院感染、术前感染再激活 |
| 1 - 6 月 | **机会感染**（CMV、PJP、Aspergillus、BK virus）+ 急性排异 |
| > 6 月 | 社区感染、慢性排异、**药物毒性**、PTLD（移植后淋巴增殖）|

> [!warning] 免疫抑制剂的经典毒性
>
> | 药物 | 经典毒性 |
> |---|---|
> | **Tacrolimus / Cyclosporine** | 肾毒、神经毒（震颤、PRES）、高钾、HTN、DM、TMA |
> | **MMF** | 骨髓抑制、GI 症状（恶心、腹泻）、致畸 |
> | **Azathioprine** | 骨髓抑制（**TPMT 缺陷者爆发性**）、肝毒、胰腺炎 |
> | **Prednisone（长期）** | 骨质疏松、感染、DM、Cushing、青光眼、白内障、AVN |
> | **Sirolimus / Everolimus (mTOR)** | 口腔溃疡、伤口愈合差、肺炎、蛋白尿、高脂 |

## Part 3 — 每个药的副作用陷阱速查

### 3.1 心血管药物

| 药物 | 经典副作用 / 毒性 | 触发场景 | 第一步检查 |
|---|---|---|---|
| **Digoxin** ⭐ | GI（恶心、食欲↓、呕吐）+ 神经（confusion、黄/绿视、乏力）+ 缓慢心律 / AV block / 室早 | 老年、AKI、furosemide → 低钾、新加 amiodarone / verapamil | **血 digoxin + K⁺ + Cr** |
| **β-blocker** (metoprolol, carvedilol) | 心动过缓、AV block、乏力、支气管痉挛、低血糖症状被掩盖、阳痿 | 哮喘、心率已慢、新加 verapamil/diltiazem | ECG、心率、肺听诊 |
| **CCB（DHP）** (amlodipine) | 外周水肿、头痛、潮红、反射性心动过速 | 老年 | 体格检查 |
| **CCB（non-DHP）** (verapamil, diltiazem) | 心动过缓、AV block、便秘、加重 HF | 与 β-blocker 联用！ | ECG |
| **ACEi** (lisinopril) | 干咳、高钾、AKI、**血管性水肿**、致畸 | 双侧肾动脉狭窄、CKD、容量不足 | K⁺、Cr |
| **ARB** (losartan) | 同 ACEi 但**无干咳、血管性水肿少** | 同 ACEi | K⁺、Cr |
| **ARNI** (sacubitril-valsartan) | 低血压、高钾、AKI、血管性水肿 | 不能与 ACEi 联用（需停 36h）| BP、K⁺、Cr |
| **MRA** (spironolactone) | 高钾、男性乳房发育、月经不调 | CKD、合用 ACEi/ARB/ARNI | K⁺ |
| **Eplerenone** | 高钾（无乳房发育） | 同上 | K⁺ |
| **SGLT2i** | UTI / 生殖器念珠菌、euglycemic DKA、容量↓、AKI | 老年、脱水、围手术期 | Cr、酮、尿检 |
| **Hydralazine** | 反射性心动过速、**药物诱导 SLE**、头痛 | 长期使用 | ANA、anti-histone |
| **Amiodarone** ⭐ | 肺纤维化、肝毒、**甲减/甲亢**、皮肤蓝灰、角膜沉积、QT 延长、肝酶升高 | 长期使用（数月-年）| TSH、LFT、CXR/PFT |

### 3.2 利尿剂

| 药物 | 经典副作用 | 关键监测 |
|---|---|---|
| **Furosemide（loop）** | 低钾、低镁、低钙、低钠、脱水、AKI、耳毒(高剂量 IV)、痛风、高血糖 | K⁺、Mg²⁺、Cr |
| **HCTZ（thiazide）** | **HyperGLUC**（高血糖、高脂、高尿酸、高钙）+ 低钾、低钠、低镁、阳痿 | 电解质、血糖 |
| **Spironolactone（K-sparing/MRA）** | 高钾、男性乳房发育 | K⁺ |
| **Triamterene/Amiloride** | 高钾 | K⁺ |
| **Acetazolamide** | 代谢性酸中毒、低钾、肾结石 | HCO₃⁻、K⁺ |

> [!tip] 利尿剂电解质口诀
> - **Loop**: "Loops Lose K Mg Ca"（loop 丢钾镁钙）
> - **Thiazide**: "Thiazides Keep Ca"（保钙！）+ HyperGLUC
> - **K-sparing**: 保钾 → 高钾风险

### 3.3 抗凝 / 抗血小板

| 药物 | 经典副作用 | 监测 / 解毒 |
|---|---|---|
| **Warfarin** | 出血、皮肤坏死（早期）、致畸、**warfarin-induced skin necrosis**（蛋白 C 缺乏） | INR / 维生素 K、FFP、PCC（4 因子凝血酶原复合物） |
| **Heparin (UFH)** | 出血、**HIT**（4T score, > 50% 血小板下降）、骨质疏松（长期）| aPTT、血小板、**Argatroban / Bivalirudin** 救 HIT |
| **LMWH** (enoxaparin) | 出血、HIT（罕见）、CKD 蓄积 | anti-Xa（CKD/孕妇/肥胖时）、Protamine（部分逆转）|
| **DOAC**（apixaban, rivaroxaban）| 出血（**比 warfarin 颅内出血少**）| **Andexanet alfa**（Xa 抑制剂逆转）、PCC |
| **Dabigatran** | 出血、GI 症状 | **Idarucizumab**（特异性单抗逆转）|
| **Aspirin** | GIB、消化性溃疡、AKI、过敏、Reye 综合征（儿童）| - |
| **Clopidogrel** | 出血、**TTP**（罕见）、CYP2C19 慢代谢者疗效差 | - |

### 3.4 抗生素

| 药物 | 经典副作用 |
|---|---|
| **Aminoglycosides** (gentamicin) | **肾毒 + 耳毒**（前庭/听力）+ NMJ 阻滞 |
| **Vancomycin** | 肾毒、耳毒、**Red man syndrome**（输注过快，非过敏）、DRESS |
| **Fluoroquinolones** (cipro, levo) | **肌腱断裂**（尤其 Achilles，老年 + 激素）、QT 延长、AAA 风险、神经病变、低血糖、谵妄 |
| **Macrolides** (azithro, clarithro) | QT 延长、肝毒、CYP3A4 抑制（互作多）|
| **TMP-SMX** | SJS/TEN、高钾、骨髓抑制、肾功能"假性升高"（抑制 Cr 分泌）、与 warfarin 互作 |
| **β-lactams** | 过敏（包括 SJS）、间质性肾炎、C. diff、神经毒（高剂量 cefepime → 脑病）|
| **Linezolid** | 骨髓抑制（> 14d）、**5-HT syndrome**（与 SSRI / MAOI）、乳酸酸中毒、视神经病变 |
| **Daptomycin** | 横纹肌溶解（停 statin！）、嗜酸性肺炎 |
| **Clindamycin** | **C. diff 经典凶手** |
| **Metronidazole** | Disulfiram-like reaction（戒酒！）、金属味、周围神经病变 |
| **Tetracyclines** (doxy) | **儿童牙齿黄染、骨发育影响**（< 8 岁禁）、光敏、致畸 |
| **Isoniazid** | **B6 缺乏 → 周围神经病变**、肝毒、药物性 SLE |
| **Rifampin** | **橙色体液**、肝毒、CYP 强诱导（OC 失效！）|
| **Ethambutol** | **视神经炎**（色盲 / 视力↓）|
| **Pyrazinamide** | 肝毒、高尿酸（痛风加重）|

### 3.5 精神科药物

| 药物 | 经典副作用 |
|---|---|
| **SSRI** (fluoxetine, sertraline) | GI、性功能障碍、**SIADH（老年）**、QT（citalopram）、停药综合征、出血风险、5-HT syndrome |
| **SNRI** (venlafaxine, duloxetine) | 同 SSRI + 高血压 |
| **TCA** (amitriptyline) | **抗胆碱 + α-block + QRS 增宽**、过量致死、镇静、体重↑ |
| **MAOI** | **高血压危象**（酪胺食物、拟交感药）、5-HT syndrome |
| **Bupropion** | 降低癫痫阈值（**进食障碍 / 癫痫禁用**）、失眠、激越 |
| **Mirtazapine** | 镇静、体重↑、口干 |
| **Lithium** ⭐ | 震颤、多尿（肾性 DI）、甲减、致畸（Ebstein anomaly）、中毒（GI + 神经 + 心律）| 监测：浓度、TSH、Cr |
| **Valproate** | 肝毒、胰腺炎、致畸（NTD）、血小板↓、PCOS |
| **Lamotrigine** | **SJS/TEN**（缓慢加量减少风险）|
| **Carbamazepine** | SIADH、骨髓抑制、SJS（HLA-B\*1502 亚洲人）、CYP 诱导 |
| **第一代抗精神病** (haloperidol) | **EPS**（dystonia、akathisia、parkinsonism、TD）、NMS、QT、催乳素↑ |
| **第二代抗精神病** (olanzapine, risperidone) | **代谢综合征**（体重、DM、血脂）、催乳素（risperidone）、QT |
| **Clozapine** | **粒细胞缺乏**（必查 ANC）、心肌炎、流涎、癫痫、便秘（致命性肠梗阻）|
| **Benzodiazepines** | 跌倒、谵妄（老年）、依赖、呼吸抑制(合用 opioid 致命）|

## Part 4 — 同类思维迁移（Stem 模式速查）

| Stem 组合 | 触发药物怀疑 | 第一步 |
|---|---|---|
| **HFrEF + AFib + 老年 + 新发 GI/乏力** | Digoxin toxicity | 血 dig level + K⁺ + Cr |
| **CKD + 新发 GI/神经/意识改变** | 经肾药物蓄积（digoxin, lithium, gabapentin, metformin, baclofen）| 药物浓度 + 肾功 |
| **Cirrhosis + 意识改变** | HE（可能药物诱发：benzo, opioid, 利尿剂过量、感染、GIB）| NH₃、感染筛查、停可疑药 |
| **老年 + 跌倒 + 多药** | Polypharmacy（benzo, anticholinergic, opioid, antihypertensive, hypoglycemic）| Beers 药物清单审查、直立位 BP |
| **Psych 病史 + 高热 + 肌强直 + 自主神经不稳** | NMS（抗精神病药）vs Serotonin syndrome（SSRI+MAOI/tramadol/linezolid）| CK、停药、支持治疗 |
| **移植后 + 任何新症状** | 免疫抑制剂副作用 / 机会感染（按时间窗）| CNI 浓度、CMV PCR、感染筛查 |
| **新加 amiodarone + 数月后乏力 / 呼吸困难 / 体重变化** | 甲状腺功能异常、肺纤维化 | TSH、CXR、PFT |
| **抗 TB 治疗中 + 视力下降** | Ethambutol 视神经炎 | 停药 + 视力评估 |
| **HIV + TMP-SMX + 高钾** | TMP 抑制 ENaC（类阿米洛利）| K⁺、停药 |
| **新加 statin + macrolide / fibrate + 肌痛** | 横纹肌溶解风险 | CK、UA（肌红蛋白）|
| **老年 + 抗胆碱药 + 谵妄 / 尿潴留** | 抗胆碱毒性 | 停药 |
| **SSRI + tramadol/linezolid/MAOI + 高热 + 肌阵挛 + 反射↑** | Serotonin syndrome | 停药 + cyproheptadine |

## Part 5 — 实战 SOP 卡片（考前贴墙版）

> [!success] 看到老年 + 多药 + 新发症状的反射动作
>
> 1. **不要先想 "病加重了"** —— 先想 "药出问题了"
> 2. **逐药扫描**：每个药能解释哪些症状？哪个药能解释**最多**？
> 3. **找扳机**：什么诱因让浓度飙升？（脱水、AKI、新加药、剂量改变、低钾、互作）
> 4. **第一步永远是**：可疑药物**血药浓度** + 相关**电解质** + **肾功**
> 5. **不是**：心超、CT、stress test、LP、TFT（除非有明确指向）

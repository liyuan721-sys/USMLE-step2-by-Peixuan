# 地高辛：药理 · 相互作用 · 中毒（Digoxin — pharmacology, interactions & toxicity）

> [!info] 用途
> 地高辛 (digoxin) 散落在多张笔记里（ECG 效应 / HFrEF (heart failure with reduced ejection fraction, 射血分数下降的心衰) 二线 / 高钾），这张是**单一真相源**：从**机制 → 药代 → P-糖蛋白相互作用 → 适应证 → 中毒全谱 → 禁忌**一条龙。
> 强调三个 USMLE 高频点：**① 治疗效应 ECG ≠ 中毒**；**② P-gp 抑制剂抬高血药浓度**；**③ 中毒"心 + 胃肠 + 神经"三系统，钾的方向陷阱**。
> 错题地基：[[mistakes/uworld-mistakes_2026-05#^Q4454]] Digoxin toxicity 三联 + 诱因。

---

## §零 一句话钥匙

> [!tip] 三把钥匙
> 1. **机制**：抑制 **Na⁺/K⁺-ATP 酶** → 细胞内 Na↑ → Na/Ca 交换↓ → **细胞内 Ca↑ → 正性肌力**；另**增迷走张力** → 减慢 **SA 结**（sinoatrial node, 窦房结）/ **AV 结**（atrioventricular node, 房室结）（**负性变时 + 变传导**，运动时被交感盖过 → 单药控房颤率差）。
> 1. **窄治疗窗 + 肾排泄**：老人（瘦体重↓、GFR (glomerular filtration rate, 肾小球滤过率)↓、多药）最易中毒；**P-gp（P-糖蛋白 P-glycoprotein）抑制剂**（胺碘酮等）抬血药浓度。
> 1. **中毒 = 心 + 胃肠 + 神经三联**：黄视 xanthopsia + 恶心 + 任何心律失常（**PAT with block**（paroxysmal atrial tachycardia with block, 房速伴阻滞）/ **双向 VT**（ventricular tachycardia, 室速）几乎是指纹）；急救 **Fab 抗体**（digoxin-specific antibody fragment, 地高辛特异性抗体片段）。

## §一 机制（Mechanism）

```
                  ┌─ 抑制心肌 Na⁺/K⁺-ATP 酶
                  │     ↓ 细胞内 Na⁺ 蓄积
   ① 正性肌力 ────┤     ↓ Na⁺/Ca²⁺ 交换体外排 Ca 减少
   (inotropy ↑)   │     ↓ 细胞内 Ca²⁺ ↑
                  └─→ 收缩力 ↑

   ② 迷走张力 ↑ ──┬─ SA 结：负性变时 (chronotropy ↓) → 心率慢
   (vagal tone)   └─ AV 结：负性变传导 (dromotropy ↓) → PR 延长、控室率
                     ※ 迷走效应"温和"，运动/交感可盖过 → 静息时最有效
```

- **正性肌力 (positive inotropy)**：抑 Na⁺/K⁺-ATP 酶 → 胞内 Na↑ → Na⁺/Ca²⁺ 交换体把 Ca 外排减少 → 胞内 Ca²⁺↑ → 收缩力↑。
- **迷走张力↑**：负性变时 (negative chronotropy) + 负性变传导 (negative dromotropy)。**因为靠迷走，所以运动（交感）时控率失效** → 单药控房颤率差，这是它只能当二线加用的根本原因。

### 治疗剂量 ECG"地高辛效应"（digoxin effect ≠ 中毒）⭐

> [!warning] 高频陷阱：治疗效应 ≠ 中毒
> 治疗血药浓度下出现下面三个 ECG 改变是**预期表现**，**不代表中毒**，别看到 scooped ST 就喊中毒：

| ECG 表现 | 机制 |
|---|---|
| **PR 间期延长** | AV 传导减慢（迷走）|
| **"Scooped" ST 压低**（Salvador Dali 胡须征）| 胞内 Ca↑ → 复极改变 |
| **QT 间期缩短** | 胞内 Ca↑ → 动作电位 / 不应期缩短 |

## §二 药代动力学（Pharmacokinetics）⭐ 增量

> [!info] 为什么 PK 是考点
> 地高辛的"窄治疗窗 + 肾排泄 + 大 Vd"直接解释**谁会中毒、什么药会抬血药浓度、为什么要负荷量**。

- **生物利用度**：口服 ~**70%**；有口服 + 静脉两种剂型。
- **分布容积大 (large Vd)**：需要快速达治疗浓度时 → **给负荷量 (loading dose)**。
- **窄治疗窗 (narrow therapeutic window)**：略超治疗量即可迅速中毒。
  - 目标血药浓度：**心衰 0.5–1 ng/mL**；**房颤 0.8–2 ng/mL**。
  - HFrEF 中 **>1.6 ng/mL 与死亡率↑相关**；中毒体征一般在 **>2 ng/mL** 出现。
- **清除：主要经肾 (renal clearance)** → 肾功能不全 / 老人 → 蓄积 → 中毒风险↑。

### P-糖蛋白相互作用（P-glycoprotein interactions）⭐⭐ 核心增量

> [!danger] 机制：P-gp 抑制剂 → 地高辛血药浓度↑
> 地高辛的**肠道吸收**和**肾排泄**都由 **P-糖蛋白 (P-glycoprotein) 外排转运体**介导：
> - **肠道**顶膜 P-gp 把地高辛泵回肠腔 → 抑制它 → **吸收 / 生物利用度↑**。
> - **肾小管**顶膜 P-gp 把地高辛泵入尿液排泄 → 抑制它 → **排泄↓**。
> - 两头都让血药浓度↑ → **中毒**。

**会抬高地高辛血药浓度的 P-gp 抑制剂**（记忆：抗心律失常 + 钙拮抗 + 大环内酯 + 杂项）：

| 类别 | 代表药 | 备注 |
|---|---|---|
| 抗心律失常 | **胺碘酮 amiodarone** | 经典！合用时**地高辛减量 50%** |
| 钙通道阻滞剂 | **维拉帕米 verapamil · 地尔硫䓬 diltiazem**（非 DHP）| DHP（dihydropyridine, 二氢吡啶，如氨氯地平 amlodipine）类不显著 |
| 大环内酯 | **红霉素 erythromycin · 克拉霉素 clarithromycin** | 阿奇较弱 |
| 免疫抑制 | 环孢素 cyclosporine | |
| 抗痛风 | 秋水仙碱 colchicine | 本身也窄窗，叠加毒性 |
| 质子泵抑制剂 | 部分 PPI（如奥美拉唑 omeprazole）| |
| SSRI | 部分（如舍曲林 sertraline）| |

> [!tip] 考试套路
> Stem：老人房颤吃着地高辛，**新加了胺碘酮 / 维拉帕米 / 克拉霉素** → 数天后恶心、黄视、心律失常 → 问机制 = **P-gp 抑制 → 地高辛蓄积**；问处置 = 测血药浓度 + 减量。

## §三 临床适应证（Indications）

```
   地高辛能用的两个场景（都是二线 / 替补）
   ├─ HFrEF（射血分数下降的心衰）
   │    指征：基石四联（ARNI/BB/MRA/SGLT2i）optim 后仍有症状
   │    获益：↓症状、↓住院  ——  ✗ 不降死亡率
   │    目标谷浓度 0.5–1.0；>1.6 → 死亡率↑
   │
   └─ 房颤 / 房扑 控室率
        指征：已用 β阻滞剂 和/或 非DHP CCB 仍需加强控率时"加用"
        单药差：迷走控率运动时失效（见 §一）
        甜点人群：AF + HFrEF —— 同时改善室率 + 收缩功能
```

> [!warning] 必背：地高辛**不降死亡率**
> HFrEF 里地高辛只 **↓症状 + ↓住院**，**不改善生存**。题目问"哪个改善**预后/生存**" → **别选地高辛**（也别选袢利尿剂），选还没上的基石药。详见 [[完整笔记/专题笔记/心内/心内_HFrEF四联药物方案]]。

## §四 中毒（Toxicity）⭐ 重点

> [!info] 谁中毒
> **老人**最常见：瘦体重↓、GFR↓、常合并抬血药浓度的药（见 §二）。体征一般在 **>2 ng/mL** 出现，累及**心 / 胃肠 / 神经**三系统。

### 4.1 心脏（cardiac）—— 双机制产生特征性心律失常

> [!danger] 两套机制叠加
> ① **迷走张力↑** → 窦缓 + 各级 AV 阻滞（**二度几乎都是 Mobitz I**，几乎不见 Mobitz II）。
> ② **胞内 Ca↑ → 自律性↑** → 房性 + 室性快速心律失常。
> **同时具备抑制传导 + 增自律性** → 出现下面这些"上慢下快"的怪组合，正是地高辛指纹。

| 心律失常 | 要点 | 特异度 |
|---|---|---|
| **房性心动过速 + AV 阻滞 (PAT with block)** | 房率~**200/min**（< 房扑的 ~300）；同时占两套机制 | **几乎诊断性** ⭐ |
| **完全 AV 阻滞 + 加速性交界性逸搏** | 交界律本应 40–60，被自律性↑加速到 **>60**；若底下是 AF → 表现为**"规律化的房颤" (regularized AF)** | 高度提示 ⭐ |
| **室性早搏 (PVC)** | 自律性↑，**最常见**的中毒心律 | 常见但不特异 |
| **双向性室速 (bidirectional VT)** | 两处室内触发灶交替 → QRS 形态交替（下壁导联最明显）| **较特异** ⭐ |
| 窦缓 / VT / VF（ventricular fibrillation, 室颤）| 重症可致命 | — |

> [!tip] 两个"反直觉"指纹一句话
> - **"规律化的房颤"**（原本 irregularly irregular 突然变规律）→ 想地高辛中毒（完全阻滞 + 加速交界律接管）。
> - **房速带阻滞 (PAT with block)** + 正在吃地高辛 → 基本就是中毒。

### 4.2 胃肠（GI）

迷走张力↑ → **厌食、恶心呕吐、腹痛**。常是最早出现的非特异前驱。

### 4.3 神经 + 视觉（neuro / visual）

- 中枢离子通道抑制 → **乏力、全身无力、意识模糊 (confusion)**。
- **黄视 (xanthopsia, 看东西发黄)** 及其他视觉异常 → 视网膜光感受器离子紊乱。⭐ buzzword

### 4.4 钾的方向陷阱（必看）⭐

> [!danger] 高钾 vs 低钾 —— 因果方向相反，处理相反
> **① 低钾 → 加重中毒（药效学，pharmacodynamic）**
> 地高辛与 K⁺ **竞争心肌 Na⁺/K⁺-ATP 酶同一结合位点**。血钾↓ → 竞争对手少 → 地高辛结合↑ → 即使血药浓度不高也中毒。**慢性中毒（利尿剂致低钾诱发）→ 可补钾把地高辛挤下泵。**
>
> **② 高钾 → 急性大剂量中毒的"果"（药效学）**
> 全身 Na⁺/K⁺-ATP 酶被阻 → K⁺ 泵不进细胞 → **血钾↑**。**K > 5.5 ≈ 预后差 = Fab 指征；此时禁补钾。**
>
> **③ 关键：高钾别给静脉钙！**
> 中毒时胞内 Ca 本已高 → 治高钾常规的**静脉钙剂一般应避免**（传统"stone heart"顾虑）。

### 4.5 处置（management）

```
   中毒处置
   ├─ 停药
   ├─ 纠正低钾（若存在）
   ├─ 进展性 AV 阻滞 → 阿托品 atropine ± 临时起搏
   └─ 重症 → 地高辛特异性抗体片段 (digoxin-specific Fab)
        指征（任一）：
         ① 心律失常致血流动力学不稳
         ② 中毒合并高钾
         ③ 血药浓度极高（如 >10 ng/mL）
```

## §五 妊娠（pregnancy）

- **未见致畸 / 直接胎儿损害**；但窄治疗窗 → 母 / 胎中毒风险 → **仅明确指征时用**。
- 少量入乳，但哺乳期已安全使用，未见婴儿损害。

## §六 禁忌证（contraindications）⭐

> [!warning] 一句话逻辑：会被"减慢传导 + 增强收缩"加重的病都禁
> 地高辛干两件事 → ① 减慢 AV/窦房传导；② 增收缩力 ↑ 耗氧。任何被这两件事恶化的情况 → 禁。

| 禁忌 | 为什么 |
|---|---|
| 心动过缓 / 进展性 AV 阻滞 | 本就慢/阻滞，再压传导 → 恶化 |
| 室速 / 室颤 | 增自律性 → 恶化室性心律 |
| **WPW（Wolff-Parkinson-White 综合征，预激综合征）** ⭐ | 阻断 AV 结 → 逼电流走旁路 → 预激↑ → 快室率/室颤 |
| **急性冠脉综合征 (ACS)** | 增收缩力 → 心肌耗氧↑ → 加重缺血 |
| **心肌炎 (myocarditis)** | 可加重心肌炎症 |
| **HFpEF（射血分数保留的心衰）** | 收缩功能本就正常，再增收缩力 → 血流动力学紊乱 |

> [!tip] WPW + 预激性房颤记忆链
> WPW 宽不规则 + delta 波 → **禁腺苷 / β / CCB / 地高辛**（凡阻 AV 结的都禁）→ 选**普鲁卡因胺 procainamide**。详见 [[完整笔记/专题笔记/心内/心内_HighYield速查总览]]。

## §七 速查总览（一表带走）

| 维度 | 要点 |
|---|---|
| 机制 | 抑 Na/K-ATP 酶 → 胞内 Ca↑（正性肌力）+ 迷走↑（减慢传导）|
| 治疗 ECG | PR↑、scooped ST、QT↓（≠ 中毒）|
| PK | 口服 70%、大 Vd（需负荷量）、肾排泄、窄窗 |
| 抬血药浓度 | P-gp 抑制剂：胺碘酮 / 维拉帕米 / 地尔硫䓬 / 大环内酯 / 环孢素 / 秋水仙碱 / 部分 PPI·SSRI |
| 适应证 | HFrEF（↓症状住院，不降死亡）、AF 控率二线（AF+HFrEF 最佳）|
| 中毒指纹 | PAT with block、规律化 AF + 加速交界律、双向 VT、黄视三联 |
| 钾 | 低钾加重（竞争泵，可补）；急性高钾是果（禁补钾、禁静脉钙）|
| 急救 | 停药 + 纠低钾 + 阿托品/起搏；重症 Fab（不稳/高钾/>10）|
| 禁忌 | 缓/阻滞、VT/VF、WPW、ACS、心肌炎、HFpEF |

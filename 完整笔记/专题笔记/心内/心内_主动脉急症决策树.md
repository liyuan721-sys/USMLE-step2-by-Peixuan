# _衍生_主动脉急症决策树（Aortic Emergency Decision Tree）

> [!info] 本笔记定位
> 跨学科整合：**主动脉破裂统一机制**（解剖 → 浆膜密封性 → 4 段破裂表现） → **AAA 全周期管理**（筛查 → 监测 → 择期 → 急诊） → **Grey Turner 多病因鉴别** → **出血性胰腺炎对照**。
> 
> **核心思维**：体征 ≠ 疾病 → 体征指向"病理过程"，疾病靠 demographics + 起病 + 伴随症状定位。
> 
> **来源**：错题 Q4212（2026-05-14）+ 后续 AAA / aortic dissection / 急诊血管题积累

---

## Part 1. 主动脉破裂的统一机制（核心理解）

> [!tip] 一句话贯通
> **主动脉破裂的所有临床表现 = 高压血流穿透周围薄浆膜 / 筋膜的结果**。不用死记每段表现，记一个机制 + 推每段解剖关系即可。

### 1.1 解剖位置回顾

```
主动脉分段（与体腔关系）：
┌─────────────────────────────────────────────┐
│ Ascending aorta（升主动脉）                 │
│   位置：心包腔内（intrapericardial）         │
├─────────────────────────────────────────────┤
│ Aortic arch（主动脉弓）                     │
│   位置：上纵隔                              │
├─────────────────────────────────────────────┤
│ Descending thoracic aorta（胸降主动脉）     │
│   位置：后纵隔（左偏，紧贴脊柱左侧）         │
├═════════════════════════════════════════════┤
│ 横膈（T12 主动脉裂孔）                       │
├═════════════════════════════════════════════┤
│ Abdominal aorta（腹主动脉，T12-L4）         │
│   位置：★ 腹膜后全段 ★                       │
│   紧贴脊柱左前方                            │
└─────────────────────────────────────────────┘
```

> [!warning] 关键事实
> **腹主动脉 100% 位于腹膜后**。但**腹膜后和腹腔只隔一层薄壁层腹膜** —— 所以前壁破裂时高压血流可穿透腹膜入腹腔。"在腹膜后" ≠ "血只能留在腹膜后"。

### 1.2 浆膜腔密封性对比（决定破裂后果）

| 腔 / 膜 | 密封性 | 容量 | 破裂后果 |
|---|---|---|---|
| **心包**（致密纤维 + 浆膜双层）| **高** | **小**（<50 mL 致填塞）| **心包填塞** — 少量血致命 |
| **纵隔**（疏松结缔组织，无致密膜）| 低 | 大 | 血肿扩散，纵隔增宽，可多向蔓延 |
| **胸膜**（薄浆膜，<0.1 mm）| 低 | **巨大**（每侧数升）| 大量血胸但不立即填塞 |
| **腹膜后**（疏松大空间）| 低 | 大 | 暂时填塞 → 有抢救窗口 |
| **腹腔**（壁层腹膜包裹）| 低 | 巨大 | 大出血迅速休克 |

> [!info] 推论
> - 升主动脉破裂死亡率最高 = 心包是"小密封袋"
> - 胸 / 腹主动脉破裂相对"宽容" = 纵隔 / 腹膜后给了抢救时间

### 1.3 四段主动脉破裂对照表

| 主动脉段 | 第一去向 | 第二去向（穿浆膜后） | USMLE 经典表现 |
|---|---|---|---|
| **升主动脉** | 心包腔 | 较少穿出 | **心包填塞**（Beck 三联）|
| **主动脉弓** | 上纵隔 | 左胸腔 / 气管 / 食管压迫 | 纵隔增宽 + 声嘶 + 咯血 / 呕血 |
| **胸降主动脉** | 后纵隔 | 穿左纵隔胸膜入左胸 | **纵隔增宽 + 左侧血胸**（同一过程两时点）|
| **腹主动脉** | 腹膜后（80%）| 穿壁层腹膜入腹腔（20%）| 80% Grey Turner + 短暂稳定；20% 急性休克 |

> [!example] 心肌破裂作为内部破裂的对照例子
> Free wall rupture（游离壁破裂）是"主动脉破裂统一机制"的镜像翻版：高压血流穿透致密的纤维心包困难，所以血液**留在小密封袋里 → 立即填塞 / PEA**。
>
> - **窗口期**：MI 后 **5 天–2 周**（再灌注延迟 + 软化巨噬期）
> - **责任血管**：多为 **LAD**（前壁大梗）
> - **临床表现**：突发胸痛 → **PEA**（电机械分离） + JVD + 心音低远 + **无肺水肿**
> - **机制对照**：升主动脉破裂 = 高压血流冲入心包；free wall rupture = 心室自身高压血液冲入心包 → **都是心包小密封袋 → 都是立即填塞**
>
> 与"3-5-7 月口诀"配合记忆：3-5 天乳头肌 / VSD；~1 周 free wall rupture；数月 true aneurysm。

## Part 1.4 Cardiac Tamponade 完整临床

> [!danger] 一句话
> **Beck 三联 + Pulsus paradoxus + low voltage / electrical alternans + clear lungs** = tamponade。**升主动脉破裂、MI 后 free wall rupture、尿毒症、恶性肿瘤、抗凝大量积液**都可以是病因。

### 急性 vs 亚急性对比

| 项目 | Acute Tamponade | Subacute Tamponade |
|---|---|---|
| **积液速度** | 分钟–小时 | 天–周 |
| **积液量** | 100-200 mL 就致命 | 1-2 L 才有症状 |
| **CXR 心影** | **正常**（心包还来不及撑大）| **enlarged, globular**（球形扩大）|
| **典型病因** | 主动脉破裂 / free wall rupture / 抗凝出血 / 外伤 | 恶性肿瘤 / 尿毒症 / 感染 / 自身免疫 |

### 急性血流动力学指纹

```
Acute Tamponade Hemodynamics (Obstructive Shock):
- CVP / RA pressure ↑ (右心先被压)
- PCWP ↑ (但 LV 充盈不足)
- 各腔室舒张压趋于相等 (equalization)
- Clear lungs (回心血进不去, 肺淤血来不及形成)
- CI ↓ → hypotension
→ Beck triad: hypotension + JVD + muffled heart sounds
```

### ECG / 影像 / JVP 三角

| 项目 | Tamponade 表现 |
|---|---|
| **ECG** | low-voltage QRS + **electrical alternans**（心脏在液体中摇摆 → 逐跳 QRS 振幅交替）|
| **CXR** | 急性：normal silhouette + clear lungs；亚急性：enlarged, globular silhouette |
| **Echo** | RA / RV diastolic collapse + IVC plethora |
| **JVP** | **x descent 保留 / 明显**；**y descent 钝化 / 消失**（与 constrictive pericarditis 的 sharp y descent 相反）|

### 治疗顺序

| 状态 | First step |
|---|---|
| BP 偏低 / 边缘 | **IV fluids 先**（提高 right-sided preload）|
| 不稳定 / Beck 三联 | **Pericardiocentesis**（床旁，超声引导） |
| 反复 / 恶性 / 外伤 | **Pericardial window**（开窗持续引流）|

### 高频 stem 触发词与陷阱

| 情境 | 关键提示 | 处理 |
|---|---|---|
| **尿毒症 / ESRD + 透析前** | "BUN/Cr 高 + 大量心包积液 + 无典型 ST↑" | **透析前必排 tamponade**（>50% 尿毒症心包炎伴积液）|
| **抗凝 + 大量积液** | "on warfarin/DOAC + new effusion" | 出血性心包积液风险 → **避免在 Dressler 中加抗凝** |
| **MI 后 ~1 周 + PEA + JVD** | "5-14 天 + 突然崩溃 + clear lungs" | Free wall rupture → 立即 echo + pericardiocentesis + 外科 |
| **Cath 后低压 + JVD + alternans** | 介入后即刻 | 心包穿孔 → echo → 不稳定则穿刺引流 |
| **主动脉夹层 + 低血压** | Stanford A | 升主动脉破入心包 → 立即手术 |

## Part 2. AAA 全周期管理

### 2.1 筛查

| 人群 | 推荐 |
|---|---|
| **65-75 岁男性吸烟者** | **一次性腹部超声筛查**（USPSTF B 级）|
| 65-75 岁男性非吸烟者 | 个体化决定 |
| 女性 | 不推荐常规筛查 |

### 2.2 监测（无症状 AAA）

| 直径 | 随访频率 |
|---|---|
| < 3.0 cm | 无需常规随访 |
| 3.0-3.9 cm | US **每 3 年** |
| 4.0-4.9 cm | US **每 12 个月** |
| 5.0-5.4 cm | US **每 6 个月** |

### 2.3 择期手术阈值

- 直径 **≥ 5.5 cm**
- **生长速率 > 0.5 cm / 6 个月**
- **有症状**（背痛 / 腹痛 / 触痛）→ 紧急修复（即使未破）

### 2.4 药物管理（所有 AAA 患者）

| 药物 / 干预 | 目的 | USMLE 权重 |
|---|---|---|
| **戒烟** | 减缓扩张 + 降低破裂 | ⭐⭐⭐ **最重要**，效果 > 任何药 |
| **Statin** | 抗炎 + 心血管二级预防 | ⭐⭐⭐（AAA 患者心血管死亡 > AAA 本身）|
| **β-blocker** | 减少血流剪切力 | ⭐⭐ |
| **ACEI / ARB** | 控制血压 | ⭐⭐ |
| **Aspirin** | 心血管二级预防 | ⭐⭐ |
| ❌ **避免氟喹诺酮** | 增加破裂风险（FDA 黑框）| ⭐⭐（老男 UTI 题陷阱）|

### 2.5 择期手术方式

| 方式 | 适应 | 优缺点 |
|---|---|---|
| **EVAR**（腔内修复）| **首选**（解剖合适）| ✓ 围术期死亡率低（~1%）✓ 恢复快 ✗ 终身影像随访 ✗ 内漏风险 |
| **Open repair**（开放）| EVAR 解剖不合适、年轻患者 | ✓ 一次性根治 ✓ 长期耐久 ✗ 围术期死亡率高（3-5%）|

### 2.6 AAA 关联动脉瘤筛查（"发现一个查全身"）

> [!tip] 核心规则
> 外周大动脉瘤之间高度伴发：**找到任意一处动脉瘤 → 全身动脉评估**。腘动脉瘤与 AAA 伴发率最高。

| 动脉瘤 | 手术阈值 | 主要风险 | AAA 关联率 |
|---|---|---|---|
| **AAA**（腹主动脉瘤）| **5.5 cm** | **破裂** → 大出血 | 基础病变 |
| **Femoral aneurysm**（股动脉瘤）| **3.0 cm** 或有症状 | 远端栓塞（蓝趾 / 趾坏死）| **常伴发 AAA** |
| **Popliteal aneurysm**（腘动脉瘤）| **2.0 cm** 或有症状 | 急性血栓 → 小腿完全缺血 | **~50%**（最高伴发率）|

> [!warning] USMLE 触发词
> - "Pulsatile groin mass + blue toe" → 股动脉瘤 → **同时筛 AAA + 对侧 / 腘**
> - "Popliteal pulsation prominent" → 腘动脉瘤 → **AAA 几率约一半**
> - 影像选择：确诊 / 随访 = **US**；术前规划 = **CTA**

### 2.7 Aortopathy 综合征关联

> [!info] 一句话
> 主动脉根扩张 / 夹层的年轻患者 → 必想 **结缔组织病 / 二叶瓣 / Turner**。

| 综合征 | 基因 / 染色体 | 主动脉特点 | 附带特征 |
|---|---|---|---|
| **Marfan syndrome (FBN1)** | FBN1（fibrillin-1）AD | **Aortic root dilation** + AR + **MVP** | 瘦高 / 蜘蛛指 / 晶状体上脱位 / 主动脉夹层 |
| **Loeys-Dietz syndrome** | TGFBR1/2 AD | **侵袭性主动脉根 / 全主动脉夹层**（发病更早 + 更小直径就破）| Hypertelorism / 悬雍垂裂 / 动脉迂曲 |
| **Ehlers-Danlos vascular type** | COL3A1 AD | **中大动脉自发破裂**（不只是主动脉）| 透明薄皮 / 关节松弛 / 子宫破裂 / 肠破裂 |
| **Turner syndrome (45,X)** | 45,X | **Coarctation + bicuspid aortic valve** → 后续主动脉根扩张 / 夹层 | 矮 / 蹼颈 / 上肢压 > 下肢压 |
| **二叶主动脉瓣 (BAV)** | NOTCH1 等 | **升主动脉扩张**（独立于狭窄程度） | 收缩 ejection click + AS / AR |

> [!warning] MVP 与 Marfan 的关系
> - **MVP**：mid-systolic click + 晚收缩杂音；**站立 / Valsalva 让 click 更早 + 杂音更长**（回流↓）
> - **瘦高 + Marfan + mid-systolic click + 站立更早** = MVP（高频组合）
> - Marfan 患者 MVP 常合并 aortic root dilation → 后续 AR / 夹层

## Part 3. AAA 急诊决策树 ⭐ 高频考点

> [!danger] 必背决策树
> 
> | 场景 | 下一步 |
> |---|---|
> | 不稳定 + **已知 AAA** | → **直接 OR**（emergent repair） |
> | 不稳定 + **无既往史** | → **床旁聚焦腹部超声**确诊 → OR |
> | 稳定 + 症状性 | → **CT 腹部** |

### 3.2 为什么这样分？

| 场景 | 关键考量 |
|---|---|
| 已知 AAA + 不稳 | 诊断已明确，转运 CT = 致命延误 → **直接开刀** |
| 无既往 + 不稳 | 万一不是 AAA 开错刀，但又不能送 CT → **床旁 US < 5 min 确诊** |
| 稳定患者 | 有时间做高分辨率影像 → CT 优于 US |

### 3.3 双相破裂概念

```
AAA 双相破裂时序：
T+0：腹膜后破裂 → 组织填塞 → 短暂稳定（"假性恢复期"）
       ↓
T+ 数分钟到数小时：填塞压力 > 壁层腹膜耐受 → 二次破入腹腔 → 失血休克
       ↓
       这就是为什么 AAA 破裂必须"立即"手术 —— 填塞效应是临时的
```

## Part 4. Ruptured AAA 鉴别诊断

### 4.1 Grey Turner sign 完整鉴别

| 病因 | Demographics + 关键线索 | 下一步 |
|---|---|---|
| **Ruptured AAA** | > 60 男性吸烟 + 突发崩溃 + 无 N/V | 不稳→床旁 US；稳→CT |
| **Hemorrhagic pancreatitis** | 酗酒 / 胆石 + 数小时渐进 + N/V / 发热 + 放射背 | 补液 + lipase + CT |
| **抗凝相关腹膜后血肿** | 老年 + Warfarin / DOAC + INR ↑ | 停抗凝 + 凝血纠正 + 影像 |
| **心导管术后股穿血肿** | 介入术后突发侧腹痛 + 血压降 | CT + 血管外科 |
| **骨盆骨折大出血** | 外伤史 | CT + 创伤外科 + 骨盆固定 |
| **腹膜后肿瘤出血** | 已知肿瘤史（RCC / 嗜铬细胞瘤 / 肉瘤）| CT + 肿瘤外科 |

### 4.2 类似体征家族（一并记）

| 体征 | 位置 | 解剖通路 | 经典关联（都非专属）|
|---|---|---|---|
| **Grey Turner** | 侧腹瘀斑 | 血沿腰方肌筋膜 → 侧腹皮下 | Ruptured AAA / 出血胰腺炎 |
| **Cullen** | 脐周瘀斑 | 血沿镰状韧带 / 圆韧带 → 脐周 | **Ruptured ectopic** / 出血胰腺炎 |
| **Fox** | 腹股沟下 / 大腿根瘀斑 | 血沿腹股沟管 → 大腿上部 | 严重腹膜后出血 |

> [!warning] 核心思维 — 体征 ≠ 疾病
> 看到经典体征：
> 1. 先问"指向什么**病理过程**？" → Grey Turner = 腹膜后出血
> 2. 再问"哪些**疾病**能造成这过程？" → 列鉴别
> 3. 最后用 demographics + 起病 + 伴随症状定病

## Part 5. 出血性胰腺炎处理（AAA 最常考鉴别）

> [!warning] 重要区别
> 出血性胰腺炎**绝大多数不需要紧急手术**，处理原则与普通胰腺炎相同。**与 AAA 处理完全不同**。

### 5.1 处理顺序

```
Step 1: ABC + 大量补液（LR 5-10 mL/kg/h，前 24h）
Step 2: 实验室（lipase 更特异）+ 影像（CT 48-72h 最准）
Step 3: 病因导向
   ① 胆石性 → ERCP（如胆管炎 / 梗阻黄疸）
   ② 酒精性 → 戒酒 + 营养
   ③ 高 TG → 胰岛素 / 血浆置换
Step 4: 支持（镇痛 + 早期肠内营养 + NPO 仅严重 N/V 时）
Step 5: 并发症（感染性坏死 → 抗生素 + 延迟清创；假性囊肿 → 观察）
```

### 5.2 AAA vs 出血性胰腺炎"第一步"对比

| 情景 | 第一步 | 为什么 |
|---|---|---|
| 不稳定急性胰腺炎 | **IV fluids（LR 大量补液）** | 第三间隙丢失 + SIRS = 低血容量，**不是失血** |
| 不稳定 AAA 破裂 | **床旁 US 确诊或直接 OR** + 输血准备 | **失血性休克**，单纯补液无效，需手术止血 |
| 不稳定 GI 出血 | **输血 + EGD** | 失血性休克，先补血再找源 |

### 5.3 USMLE 陷阱
**"Serum amylase and lipase"** 在急诊不稳定题里几乎从不是 next step：
1. 不影响急诊处理（治疗一样是补液 + 支持）
2. 等结果延误抢救
3. 同理 "查 BNP / 查肌钙蛋白" 在不稳定患者题里也常错

## Part 6. Aortic Dissection 完整版

### 6.1 起病 + 撕裂痛定位

> [!danger] 一句话
> **Severe, tearing pain, maximal at onset** + **撕裂痛位置区分升降** + **不对称体征**（脉搏 / 神经 / 杂音）= aortic dissection 直到证伪。

| 部位 | 撕裂痛位置 | 解剖关联 |
|---|---|---|
| **Ascending (升主动脉)** | 偏 **前胸**（"anterior chest"） | 心包腔内段 |
| **Descending (胸 / 腹降主动脉)** | 偏 **背部 / 肩胛间** | 后纵隔 / 腹膜后段 |

### 6.2 不对称体征家族（"分支灌注受损"）

| 体征 | 机制 | 鉴别提示 |
|---|---|---|
| **Asymmetric pulses / SBP arms > 20 mmHg** | 分支血管（无名 / 锁骨下）灌注受损 | **上肢之间差异** = dissection；**上下肢差异** = coarctation |
| **Stroke / neuro deficit** | 颈动脉 / 椎动脉 / 脊髓动脉受累 | 突发胸痛 + 偏瘫 |
| **Horner syndrome**（ptosis + miosis + anhidrosis）| 颈部交感链受压 | 单侧 |
| **新发 diastolic murmur** | 夹层撕到 aortic root → 急性 AR | 脉压宽 / 水冲脉可不典型（急性） |
| **MI / ECG 改变** | 撕到冠脉口（常右冠 → 下壁）| ⚠️ 误溶栓致命 |
| **Hemothorax**（左侧多）| 胸降主动脉撕破入左胸 | CXR 左侧大片密度 |
| **Pericardial effusion / tamponade** | 升主动脉撕破入心包 | Beck 三联 + 低血压 |
| **Mesenteric / Renal ischemia** | 肠系膜 / 肾动脉受累 | 腹痛 + ↑Cr |
| **Lower-extremity paralysis** | 脊髓动脉受累 | 截瘫 |

### 6.3 Stanford 分型 + 处理分流

```
Stanford 分型 (按是否累及升主动脉):
                   │
       ┌───────────┴───────────┐
       │                       │
   Stanford A                Stanford B
   累及升主动脉              仅累及降主动脉
   (Ascending)              (Descending, 锁骨下动脉远端)
       │                       │
       ▼                       ▼
   高死亡率                  相对稳定
   并发症:                   除非:
   - tamponade                - 进展性 (恶性 HTN 控不住)
   - acute AR                 - 灌注不良 (mesenteric/renal/limb)
   - MI (冠脉口)              - 破裂 / 即将破裂
   - stroke                   - 顽固疼痛
       │                       │
       ▼                       ▼
   Emergency Surgery       Medical first
   (开放置换升主)          (BP control)
                          + TEVAR if 复杂
```

### 6.4 影像选择

| 情境 | 首选 |
|---|---|
| **稳定 + 怀疑 dissection** | **CT angiography**（最常用，速度快，全段可见）|
| **不稳定 + 床旁** | **TEE**（不需要转运 + 实时；缺点：升主弓部段盲区 + 需镇静）|
| **肾功能差 / 造影剂禁忌** | MRI（少急诊用）/ TEE |
| **CXR**（初筛）| **Widened mediastinum**（敏感但不特异）+ 心影增大（filaments效应）|

> [!warning] 不稳定 dissection 题
> **如果 unstable + 怀疑升主动脉 dissection → echo (TEE) 而不是 CTA**（不能离开急诊室）。

### 6.5 治疗 — 药物顺序的核心逻辑

> [!danger] 必背顺序：β-blocker 先，nitroprusside 后
> **不能反过来**。原因如下：

| 药物 | 作用 | 单用风险 |
|---|---|---|
| **β-blocker (IV esmolol / labetalol)** | ↓ HR + ↓ dP/dt（**剪切力**） | — |
| **Sodium nitroprusside** | 强力血管扩张 → ↓ SBP | **若单用** → 反射性 HR↑ + **dP/dt↑** → **撕裂扩展加重** |

```
正确顺序:
Step 1: Pain control (morphine) — 同时降交感张力
Step 2: IV β-blocker (esmolol) — 降 HR + dP/dt
        目标 HR 60 bpm
Step 3: 若 SBP > 120 仍未达标 → 加 nitroprusside
        目标 SBP 100-120
Step 4 (Stanford A): 紧急外科 (升主置换)
Step 4 (Stanford B):
   - 单纯型: 继续 BP 控制 + 长期随访
   - 复杂型 (进展/灌注不良/破裂): TEVAR 或开放
```

### 6.6 危险因素

| 类别 | 例子 |
|---|---|
| **慢性 HTN** | **最强、最常见**风险 |
| **Aortopathy 综合征** | Marfan / Loeys-Dietz / Ehlers-Danlos vascular / Turner / BAV |
| **可卡因 / 安非他命** | 急性 HTN + 高交感 |
| **妊娠**（晚孕 / 产后） | 容量↑ + 雌孕激素重塑主动脉壁 |
| **创伤**（减速伤）| 主动脉峡部（韧带固定点）|
| **血管炎** | GCA / Takayasu |

### 6.7 与 coarctation 的鉴别

| 项目 | Aortic dissection | Coarctation |
|---|---|---|
| **BP 差异** | **左右上肢之间** | **上肢 vs 下肢** |
| **起病** | 急性撕裂痛 | 慢性 / 体检发现 |
| **CXR** | Widened mediastinum | "**3 sign**" + rib notching |
| **关联综合征** | Marfan / Ehlers-Danlos | **Turner**（45,X）+ BAV |

## Part 7. 高频陷阱总结

| 陷阱 | 正确认知 |
|---|---|
| ❌ Grey Turner sign = 胰腺炎 | ✓ Grey Turner = 腹膜后出血邮编，多病因 |
| ❌ "Sudden onset" 可以是胰腺炎 | ✓ 胰腺炎是"acute"渐进性，秒级崩溃 = 血管事件 |
| ❌ "No rigidity" 鉴别 AAA vs 胰腺炎 | ✓ 早期两者都无腹膜征，鉴别靠 demographics |
| ❌ 不稳定患者先查 lab | ✓ ABC > Diagnosis，先稳定再确诊 |
| ❌ 不稳定 + 已知 AAA → CT | ✓ 转运致命延误，**直接 OR** |
| ❌ 不稳定 + 无既往 AAA → OR | ✓ **先床旁 US**，避免开错刀 |
| ❌ 出血性胰腺炎需急诊手术 | ✓ 保守 + ICU + 介入栓塞（必要时），手术仅特殊指征 |
| ❌ 老年男 UTI 用氟喹诺酮 | ✓ AAA 患者**避免氟喹诺酮**（FDA 黑框，增破裂风险）|

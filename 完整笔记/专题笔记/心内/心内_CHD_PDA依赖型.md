# PDA 依赖型先天性心脏病大全（PGE₁ 救命药完整指南）

> [!info] 一句话定位
> **出生时正常 → PDA 关闭后突发恶化**的先心病 = "PDA 依赖型"
> 救命药统一是 **PGE₁ (Alprostadil / Prostaglandin E1) IV 输注**
> USMLE Step 2 高频跨学科考点

> [!success] ⭐⭐ 核心口诀
> **"PDA 是救命管道，PGE₁ 是救命药水"**
>
> 不是先心病有 PDA 都"依赖"它 —— 是病儿**生存必须靠 PDA**
> PDA 关闭 = 数小时-数天内死亡

---

## 1. PDA 解剖 + 生理（胎儿期 → 出生 → 各病种血流方向）⭐

### 解剖定位

> [!info] PDA 连接
> **主肺动脉 / 左肺动脉近端** ↔ **降主动脉**（左锁骨下动脉远端）
>
> 出生后正常关闭顺序：
> - **功能性关闭**：12-24h（平滑肌收缩）
> - **解剖性关闭**：2-3 周 → 残留 **ligamentum arteriosum**（动脉韧带）

```
       主动脉弓
        ↓
   ┌────┴────┐
   ↓         ↓
头臂干等3支  降主动脉
              ↑
              PDA（这一段）
              ↑
       主肺动脉
```

### 血流方向（取决于压力差）⭐⭐

| 时期 / 病种 | 方向 | 原因 |
|---|---|---|
| **胎儿期** | **PA → 主动脉**（右→左）| 胎盘 + 肺不通气 → PVR 高 / SVR 低 |
| **出生后正常 PDA 未闭** | **主动脉 → PA**（左→右）| 第一口气后 PVR↓ / SVR↑ → 反向 |
| **TGA**（PDA 依赖混合）| 双向混合 | PDA 是两套循环唯一混合通道 |
| **HLHS / IAA / Critical Coarc / Critical AS** | **PA → 主动脉**（右→左）| 体循环靠 RV → PA → PDA → 主动脉**逆灌** |
| **Tricuspid atresia / Pulm atresia / Severe TOF** | **主动脉 → PA**（左→右）| 肺血流靠主动脉 → PDA → PA |
| **Eisenmenger 演变型 PDA** | 反转为**右→左** | 长期未修复 → PVR↑↑ → 反向 |

> [!success] 速记
> **体循环梗阻型** → PDA 走右→左（PA → 主动脉 救体循环）
> **肺循环梗阻型** → PDA 走左→右（主动脉 → PA 救肺血流）
> **TGA** → 双向混合（救命点）
> **简单未闭 PDA** → 左→右（**机器样杂音**来源）

---

### 胎儿循环

```
胎儿期：
胎盘提供氧合
        ↓
肺没用（PVR 极高）
        ↓
血液避开肺循环：
①FO（卵圆孔）：RA → LA（避开右心-肺）
②PDA（动脉导管）：肺动脉 → 主动脉（避开肺）
        ↓
所有血液从右心绕过肺直接到主动脉
        ↓
高效用胎盘氧合
```

### 出生后 PDA 关闭机制

```
出生 → 第一口呼吸
        ↓
①肺扩张 → PVR ↓↓
②胎盘脱离 → SVR ↑↑
        ↓
PDA 内血流方向改变（变为左→右分流）
        ↓
③PaO₂ ↑（动脉血氧饱和度升高）
   ↓
   收缩 PDA 平滑肌
④胎盘来源的 PGE₂ ↓（PG 维持 PDA 开放，断脐后消失）
   ↓
   失去维持 PDA 开放的"刺激"
        ↓
功能性关闭：12-24 小时
解剖性关闭：2-3 周 → 变成 ligamentum arteriosum
```

> [!tip] 核心机制
> **维持 PDA 开放需要 PGE 信号**
> 出生后内源性 PG 消失 → PDA 关闭
> **给外源性 PGE₁ → 重新打开 PDA**

## 2. PDA 依赖型先心病的三大类型 ⭐⭐

> [!info] 按"PDA 为什么救命"分类
> 不同先心病用 PDA 解决不同问题

### Type A: 肺循环依赖型（Pulmonary Circulation Dependent）

> [!warning] 核心问题：肺血流不足
> 心脏结构使肺血流极度减少 / 完全阻塞
> → 必须靠 PDA 把主动脉血"逆流"到肺动脉 → 让血到肺氧合

| 病 | 机制 |
|---|---|
| **Tricuspid Atresia**（合并 PS）| 右心入肺通路严重受限 |
| **Pulmonary Atresia**（with intact septum）| 肺动脉瓣完全闭锁 |
| **Severe TOF / Pulm atresia with VSD** | 严重 PS → 肺血流不足 |
| **Severe Ebstein anomaly** | 右心功能差 → 肺血少 |

### Type B: 体循环依赖型（Systemic Circulation Dependent）

> [!warning] 核心问题：体循环血流不足
> 左心或主动脉病变 → 体循环灌注不足
> → 必须靠 PDA 把肺动脉血"顺流"到主动脉 → 保证体循环灌注

| 病 | 机制 |
|---|---|
| **HLHS**（左心发育不良）⭐ | LV 不能泵血 → 全身循环靠 RV→PDA→主动脉 |
| **IAA**（主动脉弓中断）⭐ | 弓断开 → 下半身靠 PDA |
| **Critical Coarctation**（重度缩窄）| 主动脉狭窄严重 → 下半身靠 PDA |
| **Critical AS**（重度主动脉瓣狭窄）| LV 流出受阻 → 体循环靠 RV→PDA |

### Type C: 混合依赖型（Mixing Dependent）

> [!warning] 核心问题：两个循环分离
> 体循环和肺循环不混合 → 全身缺氧
> → 必须靠 PDA 让两个循环血液混合

| 病 | 机制 |
|---|---|
| **D-TGA** ⭐ | 两个平行循环 → PDA 创造混合点 |

## 4. PDA 关闭后突发休克 — 4 大鉴别 ⭐⭐

> [!info] 经典 USMLE 题型
> "**新生儿出生时正常，2-3 日龄突发恶化**" → 看以下表格分诊

| 病 | 上肢 BP | 下肢 BP | 4 肢脉 | 紫绀 | 差异性紫绀 | 关键鉴别 |
|---|---|---|---|---|---|---|
| **HLHS** | 弱 | 弱 | 4 肢均弱 | 灰白 | ✗ | 全身灌注差 |
| **IAA-B** | **强** | **极低** | 下肢消失 | 中 | **✓** ⭐ | 上下半身分裂 + **DiGeorge** |
| **Critical Coarctation** | 强 | 弱 | 下肢弱 | 轻 | 轻（不明显）| 上下肢 BP 差 + 主动脉狭窄相连 |
| **Critical AS** | 弱 | 弱 | 4 肢均弱 | 灰白 | ✗ | LV 流出受阻 + 主动脉杂音 |

> [!tip] 鉴别神技 — 4 肢 BP + 紫绀模式
> - **全弱 BP** → HLHS / Critical AS
> - **上强下弱 + 差异性紫绀** → **IAA** ⭐
> - **上强下弱 + 无明显差异性紫绀** → **Coarctation**

## 5. ⭐⭐ 时间窗分类（5T+H+IAA × 1.1/1.2/1.3 映射）

> [!info] 双框架对位
> **5T+H+IAA**：按"是不是紫绀型 CHD"分（7 病，机制分类）
> **时间窗 (1.1/1.2/1.3)**：按"什么时候临床显现"分（13 病，时机分类）
> 桥梁是 **PDA 依赖性 + PVR 演变**

### 5.1 ⭐ 5T+H+IAA → 时间窗 映射

| 5T+H+IAA | 时间窗 | 原理 |
|---|---|---|
| **TGA** | 1.1 立即（0-24h）| 两套循环平行，PDA 是唯一混合点 → 出生即崩 |
| **HLHS** | 1.1 立即 | LV 不能泵，全身体循环靠 RV→PDA→主动脉 |
| **Tricuspid atresia** | 1.1 立即 | 右心无出路 → 肺血流靠 PDA |
| **TAPVR (obstructed)** | 1.1-1.2 | 肺静脉回流梗阻 → 急性肺水肿；**PDA 无关，PGE₁ 无效** ⭐ |
| **IAA** | 1.2（24-48h，DA 关后）| 出生 PDA 开 → 看似正常；DA 关 → 下半身突发休克 |
| **TOF**（严重型 / Pulm atresia 变种）| 1.2 | 重度 RVOT 梗阻 → PDA 依赖肺血流 |
| **TOF**（经典 / 中度）| 1.3 数周-月 | Tet spells 渐现，RVOT 渐重 |
| **Truncus arteriosus** | 1.3 数周-月 | **不**依赖 PDA（共干已混合）→ PVR↓ → 肺血流↑↑ → 心衰 |
| **TAPVR (non-obstructed)** | 1.3 数周-月 | PVR 自然下降才显症 |

### 5.2 时间窗 → 紫绀分类 反向映射 #难记

| 时间窗 | 紫绀型（5T+H+IAA 子集）| 非紫绀但 PDA 依赖（体循环梗阻）| 非 5T+H 但落在此窗 |
|---|---|---|---|
| **1.1 立即（0-24h）** | TGA / HLHS / Tricuspid atresia | — | Pulm atresia / Severe Ebstein |
| **1.2 DA 关后（24-48h）** | TOF（严重型）/ IAA | Critical Coarc / Critical AS / Critical PS | — |
| **1.3 数周-月** | Truncus / TAPVR（非梗阻）| — | 大 VSD / 大 PDA / IDM-HCM / AV canal defect |

### 5.3 三窗发病机制 ⭐

```
PDA 依赖性 = 1.1 / 1.2 出现的根本原因
PVR 自然下降 = 1.3 出现的根本原因

PDA 依赖 ─┬─ 混合型（TGA）───────────────────────── 1.1
          ├─ 肺循环型（Tricuspid / Pulm atresia / Sev TOF）── 1.1-1.2
          └─ 体循环型（HLHS / IAA / Critical Coarc/AS/PS）── 1.1-1.2

非 PDA 依赖 ─┬─ 已混合（Truncus）──────────────── 1.3（PVR↓ → 肺血流↑↑）
              ├─ 依赖 ASD（TAPVR）──────────────── 1.1-1.3（视梗阻程度）
              └─ 左→右分流（VSD/PDA/AV canal）── 1.3（PVR↓ → CHF）
```

### 5.4 USMLE Stem 时间窗速判 ⭐

| Stem 描述 | 先想 | 救命药 |
|---|---|---|
| "出生 30 分钟即紫绀 + egg on string" | TGA | **PGE₁** + 房隔造口 |
| "出生时正常 → 36h 突发休克 + 下肢脉消" | **IAA**（或 Critical Coarc）| **PGE₁** |
| "出生时正常 → 60h 全身灰白 + 4 肢弱脉" | HLHS / Critical AS | **PGE₁** |
| "出生时正常 → 6 周后喂养困难 + 出汗 + 体重不增" | 大 VSD / Truncus / AV canal | 利尿 + ACEi |
| "1 岁内自愈的非梗阻型 LVH（糖妈儿）" | **IDM-HCM** ⭐ | 支持，**禁 inotrope / 利尿剂** |
| "obstructed TAPVR + 急性肺水肿" | TAPVR 梗阻型 | 急诊手术，**PGE₁ 无效** ⭐ |

### 5.5 三窗反射记忆

> [!success] 看到时间窗秒反应
> - **1.1**（0-24h）→ "PDA 是唯一通路" → **PGE₁ 第一时间**
> - **1.2**（24-48h DA 关后）→ "DA 关了暴露病变" → **PGE₁ + 急诊手术**
> - **1.3**（数周-月）→ "PVR 下降 → L→R 分流加大" → **利尿 + ACEi + 修补**

## 6. ⭐⭐⭐ PGE₁ (Alprostadil) 完整指南

### 药理学

> [!info] PGE₁ = Alprostadil
> - **Prostaglandin E1** (PGE₁) 静脉滴注
> - 商品名：**Alprostadil**（注意：Alprostadil 也用于勃起功能障碍局部注射，但 USMLE 中"PGE₁ IV"几乎都指新生儿先心病）
> - **机制**：作用于 PDA 内的 PGE 受体 → 维持平滑肌松弛 → 维持开放

### 剂量与监测

| 项目 | 内容 |
|---|---|
| **起始剂量** | 0.05-0.1 μg/kg/min IV |
| **维持剂量** | 0.01-0.05 μg/kg/min |
| **起效时间** | 数分钟内（PDA 开始重新打开）|
| **监测** | 血压 + SpO₂ + 体温 + 呼吸 |

### 副作用（USMLE 高频考点）⭐

> [!warning] PGE₁ 副作用 — 三大经典
>
> **1. 呼吸抑制 / Apnea**（10-12%）⭐⭐ 最重要
> - 给 PGE₁ 时**必须准备插管**
> - 新生儿可能突发呼吸暂停
>
> **2. Hypotension**（低血压）
> - 全身血管扩张
> - 可能加重休克
>
> **3. Fever**（发热）
> - 直接体温调节中枢影响
> - 不是感染！避免误判 sepsis
>
> 其他：水肿、皮肤潮红、抽搐（罕见）

> [!tip] 记忆口诀
> **PGE₁ 三大副作用：呼吸暂停 + 低血压 + 发热**
> ("**A**pnea + **H**ypotension + **F**ever" = AHF)

### 给药前准备

> [!danger] PGE₁ 给药前必须准备：
> 1. **气道管理设备**（插管 + 球囊面罩）
> 2. **建立 2 条 IV 通路**（一条给 PGE₁，一条液体复苏）
> 3. **持续监护**（心电 + SpO₂ + BP）
> 4. **温暖环境**（保暖箱 / 辐射床）
> 5. **NICU 会诊**

## 7. ⭐ 与 NSAID / Indomethacin 的对比（USMLE 重点）

### PG 通路的双向调节

```
                    维持 PDA 开放
                          ↑
     ──── PGE / Prostaglandin ────
                          ↓ COX 抑制（NSAID）
                          ↓
                    PDA 关闭
```

| 目的 | 药物 | 机制 |
|---|---|---|
| **维持 PDA 开放** | **PGE₁** (Alprostadil) | 直接激活 PG 受体 |
| **关闭 PDA** | **Indomethacin** 或 **Ibuprofen** | COX 抑制 → 减少内源性 PG |

### 临床应用对比

| 临床情境 | 药物 | 原因 |
|---|---|---|
| **早产儿 PDA 患儿**（PDA 没关）| **Indomethacin / Ibuprofen** | 关闭 PDA → 减少左→右分流 |
| **先心病 PDA 依赖**（PDA 必须开）| **PGE₁** | 维持 PDA → 维持循环 |
| **足月儿 PDA 关闭**（先心病诊断前）| **PGE₁** | 重新打开 PDA |

> [!warning] 经典 USMLE 陷阱
> **同一个新生儿 + PDA**，治疗完全相反：
> - **早产 + PDA 杂音 + 心衰** → 关 PDA（Indomethacin）
> - **足月 + PDA + 紫绀 / 休克 + 先心病** → 开 PDA（PGE₁）
>
> 题干区分关键：**早产 vs 足月**、**简单 PDA vs 合并先心病**

### 孕期药物影响 PDA ⭐

> [!warning] 孕期 NSAID 警告
> 妊娠**晚期**（third trimester）使用 NSAID（如 Ibuprofen / Indomethacin / Aspirin）：
> - 抑制胎儿 PG 合成
> - **早期 PDA 关闭** → 胎儿肺动脉高压 + 心衰
> - **羊水过少**（NSAID 减少胎儿尿量）
> - → 妊娠晚期 **NSAID 禁忌**
> - 偶尔故意用于治疗胎儿多羊水（早产相关）

## 8. PDA 依赖型先心病的处理 SOP ⭐

```
新生儿急症（紫绀 / 休克 / 心衰）
        ↓
①ABC 稳定
   - 气道：插管准备
   - 呼吸：100% O₂（注意：可能加速 PDA 关闭！）
   - 循环：建立 IV + 液体复苏
        ↓
②急查
   - SpO₂（4 肢测量看 differential）
   - 4 肢 BP（上下肢对比）
   - Hyperoxia test（区分心/肺源性）
   - ABG + 血糖 + 电解质 + 血钙
   - CXR（看心影 + 肺血流）
   - ECG
        ↓
③Echo（确诊金标准）⭐
   - 确定具体先心病类型
   - 评估 PDA 大小 + 方向
        ↓
④PGE₁ 立即开始（在等 Echo 结果时就可经验性给）⭐⭐
   - 0.05-0.1 μg/kg/min IV
   - 准备插管（呼吸抑制风险）
   - 监测 BP / SpO₂ / 体温
        ↓
⑤合并畸形评估
   - 22q11.2（如 IAA / Truncus / TOF）
   - 染色体（如 Turner / Down）
   - 母亲糖尿病史（如 TGA）
        ↓
⑥手术规划
   - 一期根治（TGA → Switch；IAA → 弓重建；Truncus → Rastelli）
   - 分期姑息（HLHS → Norwood；Tricuspid → Glenn → Fontan）
   - 桥接（Balloon septostomy / BT shunt）
```

> [!success] 关键时间点
> - PGE₁ 起效：数分钟
> - 临床稳定：数小时
> - 转入有先心病手术能力的医院：24 小时内
> - 手术：取决于病种（数天到数周）

## 9. 100% O₂ 的"双刃剑"陷阱 ⭐⭐

> [!danger] USMLE 反直觉考点
> **100% O₂ 对 PDA 依赖型先心病可能有害！**
>
> 机制：
> - 高 PaO₂ → 收缩 PDA 平滑肌 → 加速 PDA 关闭 ⚠️
> - 高 PaO₂ → 肺血管扩张 → 进一步增加肺血流 → "肺充血盗血"
>
> 实际处理：
> - 维持 SpO₂ 在 **75-85%**（不要追求 95%+）
> - 给适度的氧（如鼻导管低流量）
> - 主要靠 PGE₁ 维持 PDA

> [!warning] 经典 Stem 陷阱
> 新生儿紫绀 → 医生给 100% O₂ → 突然恶化
> 原因：100% O₂ 加速 PDA 关闭 → PDA 依赖型先心病崩溃
> → 立即 PGE₁ + 降低 FiO₂

## 10. PDA 依赖型先心病的合并畸形与综合征关联 ⭐

### 综合征关联速查

| 先心病 | 高危关联 | 提示 stem |
|---|---|---|
| **HLHS** | Turner（极少）| 一般 sporadic |
| **IAA**（特别 Type B）| **22q11.2 / DiGeorge** (50-90%) ⭐⭐ | 腭裂 + 低钙 + 反复感染 |
| **Critical Coarctation** | **Turner** ⭐ | 矮小 + 蹼颈 + 性腺发育不良 |
| **TGA** | **糖尿病母亲** ⭐ | 巨大儿 |
| **Pulm atresia** | 部分 22q11.2 | — |
| **Tricuspid atresia** | — | — |

### 跨学科网络

```
PDA 依赖型先心病（专题枢纽）
        │
        ├─ DiGeorge / 22q11.2 → IAA / Truncus
        │
        ├─ Turner 综合征 → Coarctation
        │
        ├─ 母亲糖尿病 → TGA
        │
        ├─ 母亲 SLE → 完全性 AV block（非 PDA 依赖，但相关）
        │
        └─ 染色体三体 → 多种心脏畸形
```

## 11. ⭐ PDA 依赖型 vs 非 PDA 依赖型紫绀鉴别题 SOP

> [!info] 看到"新生儿紫绀"题先问 4 个问题：
>
> **Q1: PDA 还开着吗？**
> - 出生 < 24 小时 → 通常还开 → 可能症状轻
> - 出生 24-72 小时 → PDA 开始关 → 突发恶化
> - 出生 > 72 小时 → PDA 基本关闭
>
> **Q2: 给 PGE₁ 有效吗？**
> - 有效 → PDA 依赖型（HLHS / IAA / TGA / Tricuspid / Pulm atresia / Critical Coarc）
> - 无效 → **TAPVR**（ASD 依赖）⭐ 或 Truncus（不需要）或非心源性
>
> **Q3: 上下肢 BP / SpO₂ 差异？**
> - 全弱 → HLHS / Critical AS
> - 上强下弱 + 差异性紫绀 → IAA ⭐
> - 上强下弱无明显差异 → Coarctation
>
> **Q4: 看 CXR 形状**
> - Egg on string → TGA
> - Boot → TOF
> - Snowman → TAPVR
> - Globular → HLHS

## 12. Memory Hook ⭐⭐

> [!success] 终极记忆三连
>
> **1. 三大类型口诀**
> **"肺血少给 PDA 送血，体循环差给 PDA 送血，循环平行给 PDA 混合"**
> - Type A 肺循环依赖：Tricuspid / Pulm atresia
> - Type B 体循环依赖：HLHS / IAA / Coarc / Critical AS
> - Type C 混合依赖：TGA
>
> **2. PGE₁ 救命口诀**
> **"6 个 PDA 依赖型先心病，PGE₁ 是统一救命药"**
> TGA / Tricuspid / HLHS / IAA / Critical Coarc / Pulm atresia
>
> **3. 陷阱口诀**
> **"TAPVR 无效，Truncus 不需"**
> 两个看似紫绀但不是 PDA 依赖
>
> **4. 副作用口诀**
> **"PGE₁ 三大副作用：AHF"**
> **A**pnea + **H**ypotension + **F**ever

> [!tip] 画面感
> 一个新生儿出生时看起来很好 →
> 2-3 天后突然恶化（PDA 自然关闭）→
> 急救医生说"快推 PGE₁！同时准备插管！"（防呼吸暂停）→
> PDA 重新打开 → 患儿稳定 → 准备转 NICU 做手术

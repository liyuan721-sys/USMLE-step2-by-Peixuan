## Part 1. AF 复发 — 可逆病因筛查

> [!info] 两轴模型：基质 (substrate) vs 诱因 (trigger) ⭐
> AF 的可逆病因可拆成**两类机制**，记一张图就够：
> - **慢性基质 (substrate)**：HTN / HFpEF / MS（二尖瓣狭窄 mitral stenosis）/ 慢性 OSA（阻塞性睡眠呼吸暂停）→ 长期**左房重构 (atrial remodeling)** / 纤维化 / 牵张 → **维持 + 复发** AF。要**长期管**（控压 / 减重 / 治瓣膜 / CPAP）。
> - **急性诱因 (trigger)**：甲亢 (hyperthyroidism) / 酒精 binge / PE（肺栓塞 pulmonary embolism）/ Pheo（嗜铬细胞瘤 pheochromocytoma）/ 电解质 / 脓毒症 / 脱水 (dehydration) / 拟交感 → 急性 tip over → **新发**多能**纠诱因后自转 (treat the trigger)**。
> - **叠加**：真实病人常 = 基质 + 诱因（如 HTN 左房大【基质】+ 一次酗酒/脱水【诱因】→ **首发 AF**）。基质决定"为什么容易得"，诱因决定"这次为什么发"。
>
> 对应本笔记：**§1.2 = 基质为主**（复发筛查，长期管）；**Part 1B = 诱因为主**（新发，纠因 + 观察）。

### 1.1 临床场景

**Trigger 模式**：
- Paroxysmal AF 复发（cardioversion 后又发）
- 常规 workup（echo / 电解质 / TFT）阴性
- 没有明显急性诱因

→ 不要继续重复 cardioversion，要找**可逆的潜在病因**。

### 1.2 必查清单（USMLE 高频）— 按基质 / 诱因分组

> [!info] 读法
> 左侧加了**机制**列点明走哪条轴（基质 = 左房重构 / 诱因 = 急性 tip over），右侧"干预"对应"基质长期管 vs 诱因纠因"。

#### A. 慢性基质 (substrate) — 左房重构，长期管理

| 病因 | 机制（→ 左房重构）| 筛查 | Stem 提示 | 干预 |
|---|---|---|---|---|
| **OSA**（阻塞性睡眠呼吸暂停）⭐ | 夜间缺氧 + 慢性交感↑ | Sleep study (PSG) | **BMI ≥30**, snoring, daytime fatigue | CPAP + 减重 |
| HTN（高血压）| LVH → 左房扩大 | BP monitoring | LVH on echo | 降压（长期）|
| Valvular（MS, mitral stenosis 二尖瓣狭窄）| 左房压↑ → 左房扩大 | Echo | Diastolic murmur | 瓣膜修复 |
| HFpEF（射血分数保留心衰）| 舒张障碍 → 左房压↑ | Echo (diastolic dysfunction) | DOE, edema | 减重 / 利尿 |

#### B. 急性诱因 (trigger) — tip over，纠因多自转

| 病因 | 机制（→ 急性 tip over）| 筛查 | Stem 提示 | 干预 |
|---|---|---|---|---|
| Hyperthyroidism（甲亢）| 代谢 + 肾上腺素能↑ | TSH | Weight loss, tremor, heat intolerance | 抗甲状腺药 / RAI |
| Alcohol（酒精 binge）| Holiday heart syndrome | History | "Holiday heart" / binge | 戒酒 |
| PE（肺栓塞，急性诱发）| 缺氧 + 右心牵张 | CTPA, D-dimer | Chest pain, syncope, hypoxia | 抗凝 |
| Pheo（嗜铬细胞瘤，罕见）| 儿茶酚胺↑ | Plasma metanephrines | Episodic HTN + headache + sweating 三联征 | 切除 |
| 电解质 / 脱水（dehydration）| 低钾 / 低镁 + 交感↑ | BMP 基础代谢套餐（K⁺/Mg²⁺）+ 病史 | 呕吐腹泻 / 利尿剂后心悸 | 补液 + 补 K/Mg |

> [!tip] 一句话区分
> **基质组（A）** stem 多给"慢性病史 + echo 结构改变 (LVH/瓣膜/左房大)"，处理是**长期管**；**诱因组（B）** stem 多给"急性事件 + 实验室/影像异常"，处理是**纠因后常自转**。脱水/电解质从 B 组进来（详见 Part 1B）。

### 1.3 OSA 促 AF 的机制

```
        肥胖 (BMI 35) + 上气道软组织松弛
                       ↓
              夜间反复气道塌陷 = OSA
                       ↓
        ┌──────────────┴──────────────┐
        ↓                              ↓
  Nocturnal hypoxemia          Nocturnal hypoventilation
        ↓                              ↓
  ↑ Sympathetic tone           Hypoxic pulmonary
  (慢性交感↑)                   vasoconstriction
        ↓                              ↓
  ↑ Atrial automaticity        ↑ Pulmonary artery pressure
  (心房自律性↑)                 ↓
        ↓                       Right atrial dilation
        └──────────┬────────────────┘
                   ↓
        Atrial remodeling (结构 + 电重构)
                   ↓
        AF 复发 / 持续
```

> [!tip] 一句话锁定
> **AF 复发 + BMI ≥30 + 常规检查阴性 → Sleep study**

### 1.4 长期管理决策 — Rate vs Rhythm + 抗凝（三条独立线）

> [!info] 三件事**分开**决策，别搅在一起 ⭐
> AF 长期管理 = **① 控室率 (rate) 还是 转复律 (rhythm)** + **② 抗凝 (anticoagulation)** + **③ 治可逆病因（§1.2 / Part 1B）**。
> **关键陷阱：抗凝是独立轴 —— 不论你选 rate 还是 rhythm、不论是否成功转回窦律，抗凝都按 CHA₂DS₂-VASc 单独决定。**"转回窦律了所以不用抗凝"是高频错误。

#### Rate control（默认首选）vs Rhythm control

| | **Rate control（控室率，默认）** | **Rhythm control（转复律 / 维持窦律）** |
|---|---|---|
| 适用 | **多数患者**：年长、症状轻、持续性 AF | 年轻 / **症状重（控率后仍不适）** / 首发 / **AF 诱发或加重心衰** / 控率难达标 / tachycardia-mediated cardiomyopathy（心动过速性心肌病）|
| 循证 | **AFFIRM 试验**：多数患者 rate 与 rhythm **生存率相当**，rate 更简单、副作用少 → **默认 rate** | 不改善总生存,主要为**改善症状** / 特定人群获益 |
| 手段 | β-blocker（β阻滞剂）或 非二氢吡啶 CCB（地尔硫䓬 diltiazem / 维拉帕米 verapamil）；**地高辛 digoxin** 作为合并 HF / 久坐者的加用 | 抗心律失常药（**无结构性心脏病** → 氟卡尼 flecainide / 普罗帕酮 propafenone；**有结构病** → 胺碘酮 amiodarone / 多非利特 dofetilide / 索他洛尔 sotalol）；电复律；**导管消融（肺静脉隔离 PVI）** 用于难治/症状重 |
| 靶值 | 宽松 HR <110（静息，RACE II 证据）；症状性收紧到 <80 | 维持窦律 |

> [!danger] 选药雷区
> - **HFrEF（射血分数下降心衰）控率**：用 **β-blocker 或 digoxin**，**禁非二氢吡啶 CCB**（地尔硫䓬/维拉帕米负性肌力 → 加重心衰）。
> - **WPW（预激综合征 Wolff-Parkinson-White）合并 AF**：**禁 AV 结阻滞药**（β-blocker / CCB / digoxin / adenosine）→ 会经旁路加速下传诱发 VF → 选**普鲁卡因胺 procainamide** 或电复律。

> [!tip] 复律前抗凝时机（高频）
> AF **>48 h 或时间不明** + 拟复律 → **两选一**：① 复律前**治疗性抗凝 ≥3 周** + 复律后 ≥4 周；或 ② **TEE（经食道超声）排除左房血栓** → 即刻复律 + 抗凝。AF <48 h 血栓风险低。
> → 抗凝评分（CHA₂DS₂-VASc）与药物选择详见 [[完整笔记/专题笔记/心内/心内_抗凝抗板_急性血栓]] §E.2（**本节不复制，避免双处维护**）。

## Part 1B. 急性诱因性房颤 — 稳定 + 室率正常先观察 ⭐（NBME11 Q062）

**场景**：明确急性可逆诱因（**脱水 dehydration / 热暴露 / 急性病 / 电解质紊乱 / 酒精 binge / 感染**）+ 新发 AF + **血流动力学稳定 (hemodynamically stable) + 室率正常（HR 不快）** → **纠正诱因 + observation only（仅观察）**，不急于 rate / rhythm control，更不上器械。

### AF 可逆诱因全谱 — PIRATES ⭐

> [!info] 新发 AF 第一反应 = 先找可逆诱因（treat the trigger）
> 纠正诱因后窦律常能自转。记忆口诀 **PIRATES**：

```
P — Pulmonary 肺源       PE 肺栓塞(pulmonary embolism)/ 肺炎 / COPD 慢阻肺(chronic obstructive pulmonary disease)急性加重 / 缺氧
    Pericarditis 心包炎 / Pheochromocytoma 嗜铬细胞瘤(儿茶酚胺↑)
I — Ischemia 缺血 / MI(心肌梗死，尤其下壁) / Idiopathic 特发性(lone AF)
R — Rheumatic 风湿性瓣膜病  二尖瓣狭窄(MS, mitral stenosis)→ 左房扩大 ⭐
    Respiratory 呼吸源
A — Anemia 贫血 / Atrial dilation 心房扩大 / Atrial myxoma 心房黏液瘤
T — Thyrotoxicosis 甲亢 ⭐⭐(新发 AF 必查 TSH 促甲状腺激素 thyroid-stimulating hormone) / Theophylline 茶碱
E — Elevated BP 高血压(HTN) → LVH/左房扩大 [基质] ⭐
    Ethanol 酒精 → Holiday heart syndrome(假日心脏综合征) [诱因] ⭐
    Electrolytes 电解质(低钾 hypokalemia / 低镁 hypomagnesemia) [诱因]
S — Sepsis 脓毒症 / 全身急性病 / Sleep apnea 阻塞性睡眠呼吸暂停(OSA) ⭐
    Stimulants 拟交感(可卡因 cocaine/安非他明 amphetamine/咖啡因 caffeine) / Surgery 术后(尤其心胸外科)
    + 容量不足/脱水 + 热暴露(本节 Q062 诱因)
```

> [!tip] PIRATES 按两轴归位（呼应 §1.2 的基质/诱因分组）
> 同一口诀拆到两轴上记，和 §1.2 的 A/B 两组互相印证：
>
> | 轴 | PIRATES 字母（基质 vs 诱因）|
> |---|---|
> | **基质 substrate**（长期管）| R（MS 二尖瓣狭窄）· A（Atrial dilation 心房扩大）· S（Sleep apnea/OSA）· E（Elevated BP/HTN）|
> | **诱因 trigger**（纠因自转）| P（PE/肺炎/Pheo）· I（Ischemia 缺血）· T（Thyroid 甲亢/Theophylline 茶碱）· E（Ethanol 酒精/Electrolytes 电解质）· S（Sepsis 脓毒症/Stimulants 拟交感/Surgery 术后/脱水）|
>
> 注：**P / S / E 跨两轴**（P 含 PE[诱因] 也含慢性肺[基质倾向]；S 含 OSA[基质] + Sepsis/Surgery[诱因]；E 含 HTN[基质] + Ethanol/电解质[诱因]）——这正是"同病人基质+诱因叠加"的体现。

> [!danger] USMLE 必抓 4 个高频诱因
> - **甲亢 (hyperthyroidism)** → 新发 AF **必查 TSH**；老年**淡漠型甲亢 (apathetic hyperthyroidism)** 可只表现为 AF + 消瘦。
> - **酒精 binge → Holiday heart** → 周末/节假日酗酒后**心脏结构正常**者新发 AF，戒酒自转。
> - **阻塞性睡眠呼吸暂停 (OSA)** → **反复 / 难治性 AF** 要筛 OSA（见 [[mistakes/uworld-mistakes_2026-05#^Q21530]]）。
> - **急性诱因（脱水 / 感染 / 术后 / 电解质）** → 稳定 + 室率正常时**先纠诱因 + 观察**（见 [[NBME11_错题本#^Q062]]）。

> [!tip] 基础筛查三件套
> 新发 AF 至少做：**TSH（甲亢）+ 电解质（K/Mg）+ 找急性病/诱因**；稳定且室率不快时，纠诱因 + 观察往往就够。

### AF 急诊分层（稳定性 × 室率）

| 场景 | 关键特征 | 处理主轴 |
|---|---|---|
| 不稳定 AF | 休克 / 缺血 / 肺水肿 / 灌注差 | 紧急同步电复律 (synchronized cardioversion) |
| 稳定 AF + 室率快（RVR, rapid ventricular response）| HR >100~110 | rate control（β-blocker β阻滞剂 / CCB 钙通道阻滞剂）± 治诱因 |
| **稳定 AF + 室率正常 + 可逆诱因** | HR 正常、稳定（如 HR 74、脱水）| **纠正诱因 + 观察** |
| **慢 AF（室率慢，HR <60）** ⭐ | AV 结放行少 → 心动过缓 ± 晕厥 | **找原因**：AV 结阻滞药 / 病窦(SSS) / 完全性 AVB / 地高辛中毒；**症状性 → 起搏器**（先停诱因药）|

> [!info] AF 也能心动过缓 —— 心室率取决于 AV 结放行，不取决于心房
> 心房在 AF 时恒定 400–600 次/分；能下传的只是 **AV 结（房室结）筛选放行**的部分。AV 结被抑制 → **慢 AF**：
> - **AV 结阻滞药**：β-blocker / 非二氢吡啶 CCB（地尔硫䓬 diltiazem / 维拉帕米 verapamil）/ 地高辛 digoxin / 胺碘酮 amiodarone（控速药过量最常见）
> - **病窦综合征（SSS, sick sinus syndrome）= 快-慢综合征（tachy-brady）**：阵发 AF 与 窦停/心动过缓**交替**，AF 终止后**长间歇**→ 晕厥 → **起搏器**（治慢）+ 控速/抗凝（治快）
> - **甲减 (hypothyroidism) / 高钾 (hyperkalemia) / 高迷走张力**（运动员、睡眠）

> [!danger] 反直觉信号①：AF 由"不规则"变"规则" = 出事，不是好转
> AF 本应**不规则**；若心室率变得**慢而规则** → 多半叠加了**完全性房室传导阻滞（三度 AVB）**，心室靠交界区/室性逸搏驱动。
> **最经典病因 = 地高辛中毒 (digoxin toxicity)** → "**规律化的房颤 (regularized AF)**"，常伴恶心 / 黄视 (xanthopsia) / 低钾 → 查地高辛水平 + 停药。

> [!danger] 反直觉信号②：起搏器 (pacemaker) ≠ 房颤处理（Q062 错点）
> **起搏器治"太慢"**（高度 / 三度房室传导阻滞、病窦综合征、顽固症状性心动过缓），**不治"乱但不慢"的 AF**。Q062 误选起搏器 = 见异常心律就反射器械干预、漏看 **HR 74 = 室率正常**。同理 **adenosine（腺苷）** 只对规则性 SVT（AF 不规则无效）、**lidocaine（利多卡因）** 对室性心律——见"心律乱"别乱套工具。
> ※ 区分：**症状性慢 AF（病窦/AVB）→ 起搏器** 是对的；Q062 错在 **HR 74 不慢**，不是起搏器本身错。

> [!tip] 一句话锁定（Part 1B）
> **"房颤先看快不快、塌不塌"**：稳定 + 室率正常 + 有可逆诱因 → 补液 / 治诱因 + 观察。

## Part 2. 心律失常诊断/操作工具 — 适用边界

> [!info] 姊妹篇:本 Part 讲"工具能干嘛",处理流程见 → [[完整笔记/专题笔记/心内/心内_SVT急性处理]]
> 规则窄 QRS SVT 的**急诊处理阶梯**（迷走→腺苷→AV结阻滞药→复律）、**SVT 分型鉴别**、**预激(WPW)+AF 警示** 已拆为独立笔记 [[完整笔记/专题笔记/心内/心内_SVT急性处理]]。本 Part 保留"工具适用边界"视角。

### 2.1 核心对照表

| 工具 | 适用 | 机制 | ❌ 不适用 / 陷阱 |
|---|---|---|---|
| **Carotid Sinus Massage (CSM) / Vagal** | AVNRT, AVRT（规则窄 QRS SVT）；房扑揭示 F 波 | ↑迷走 → ↑AV 阻滞 | ❌ AF（已不规则）；❌ 不定位起搏点 |
| **Adenosine 6 mg IV** | AVNRT 一线终止；房扑揭示 F 波；宽 QRS 鉴别 SVT vs VT | 短暂完全 AV 阻滞 | ❌ AF；❌ 不定位起搏点；哮喘/COPD 慎用 |
| **Atropine 0.5 mg IV** | 症状性 bradycardia；症状性 AV block；**鉴别 nodal vs infranodal** | 抗迷走 → ↑SA 放电 + ↑AV 传导 | ❌ AF（反而加速室率）；❌ 不定位起搏点 |
| **EPS (electrophysiology study)** | **精确定位起搏点 / mapping 折返环 / 消融** | 心内电极标测 | 创伤性，难治 SVT/VT 才用 |

### 2.2 易混三连击（USMLE 高频陷阱）

#### 陷阱 1：CSM 用在 AF 上

> [!tip] 口诀锁定 — **"CSM 治 R 不治 I"**
> **R**egular SVT 有用，**I**rregular AF 没用。
>
> **适用（Regular 节律）**：
> - AVNRT / AVRT 终止（规则窄 QRS SVT 的一线非药物试验）
> - 揭示**房扑 F 波**（延缓 AV 传导让 F 波"裸露"出来）
> - 宽 QRS 鉴别 SVT with aberrancy vs VT（SVT 可中断、VT 不变）
>
> **不适用 / 陷阱（Irregular 节律）**：
> - ❌ **AF**（已是不规则乱搏，CSM 毫无诊断 / 治疗价值）— Q21530 的核心错点
> - ❌ **不定位起搏点**（CSM / Adenosine / Atropine 都不是 GPS；只有 **EPS** 能精确 mapping）
>
> **机制**：刺激颈动脉窦压力感受器 → ↑迷走张力 → ↑AV 阻滞（延缓房室传导）。
>
> "CSM 可以区分起搏点位置吗？" — **不能**。它的作用只是**延缓 AV 传导**，让规则节律的折返环或 F 波"暴露"出来。

#### 陷阱 2：Atropine 用来"定位起搏点"
> "Atropine 是不是有定位功能？" — **不是定位起搏点**。
> Atropine 唯一接近"定位"的用途 = **鉴别 AV block 的阻滞位置**（nodal vs infranodal），见 Part 3。

#### 陷阱 3：把 SVT 诊断流程套到 AF 上
> AF 一旦确诊，问题层级就变了：不再是"诊断什么心律失常"，而是"如何长期管理 / 找可逆病因"。
> 看到 AF + 复发 → 思路立刻切到 **Part 1 的可逆病因清单**。

### 2.3 操作选择决策树

```
                    心律失常?
                       ↓
        ┌──────────────┴──────────────┐
        ↓                              ↓
   规则节律                        不规则节律
        ↓                              ↓
   窄 QRS SVT?                      AF / Aflutter (变传导)
        ↓                              ↓
   是 → CSM / Adenosine            → 不用 CSM/Adenosine
        - 终止 AVNRT                  → Rate control (β/CCB)
        - 揭示房扑 F 波                → 抗凝 (CHA₂DS₂-VASc)
        - 鉴别 SVT vs VT (宽 QRS)      → 找可逆病因 (Part 1)

   慢心律? (Brady/AV block)
        ↓
   症状性 → Atropine 0.5 mg
        ↓
   有效? → Nodal (良性, 观察)
   无效? → Infranodal (危险, TCP → 永久起搏器)
```

## Part 3. AV Block 定位 — Atropine 反应判断

> [!info] 与 [[完整笔记/专题笔记/心内/心内_AV_Block完整鉴别]] 的分工
> - **那一份**：5 种 AV block 的 ECG 识别（PR 模式 / QRS 宽度 / 漏拍前后比较 / UWorld 原图）— "看图分类型"
> - **本节**：用 Atropine 反应 + MI 部位 + 非 MI 病因来**定位**（nodal vs infranodal）+ 决策起搏器 — "拿什么治"

### 3.1 解剖基础

```
        SA node
            ↓
        心房
            ↓
    ━━━━━━━━━━━━━━━━━━━ 受迷走支配
            ↓
        AV node ◄── Nodal (结上)
            ↓
    ━━━━━━━━━━━━━━━━━━━ 不受迷走支配
            ↓
        His bundle
        Bundle branches  ◄── Infranodal (结下)
        Purkinje fibers
            ↓
        心室
```

**关键事实**：
- AV 结 = **迷走密集分布**（副交感支配）
- His-Purkinje = **几乎无迷走支配**

### 3.2 Nodal vs Infranodal 对照
#难记

| 特征 | Nodal (AV 结) | Infranodal (His 以下) |
|---|---|---|
| **类型** | 1° AVB, Mobitz I (Wenckebach) | Mobitz II, 3° 高位完全 AV block |
| **QRS escape** | **窄** (junctional 40-60 bpm) | **宽** (ventricular 20-40 bpm) |
| **PR 模式** | 渐进延长后漏搏 | 突然漏搏（PR 固定）|
| **诱因** | **下壁 MI** (RCA → AV 结缺血), ↑迷走 (运动员) | **前壁 MI** (LAD → His 缺血), 退行性传导病 |
| **Atropine 反应** | ✅ 改善（迷走支配可解除）| ❌ 无效甚至恶化 |
| **预后** | 良性，多自限 | **危险，进展 → 起搏器** |
| **治疗** | 观察 / 治病因 | TCP → **永久起搏器**（即使后续恢复也装） |

> [!info]- "QRS escape 窄 / 宽"到底什么意思 ⭐
> AV 传导被阻断（尤其 3° 完全 AV block）时，心房信号传不下来，心室靠阻滞点**以下**的逃逸起搏点（escape pacemaker）自救。**这个逃逸 QRS 是窄是宽，直接反推逃逸点位置 → 反推阻滞在哪一层。** 核心律：**逃逸点越往下，QRS 越宽、频率越慢、越危险。**
>
> | | **窄 QRS escape** | **宽 QRS escape** |
> |---|---|---|
> | 逃逸起搏点 | **Junctional（房室交界区，His 束及以上）** | **Ventricular（心室肌内，His 分叉以下）** |
> | 下传路径 | 仍走正常 **His-Purkinje 系统** → 两室**同步**激动 → 窄（<120 ms）| His-Purkinje 已断 → 信号靠心肌细胞间**慢传导**铺开 → 两室**不同步** → 宽（≥120 ms，形似室性）|
> | 频率 | **40–60 bpm**（junctional 固有频率较快）| **20–40 bpm**（ventricular 固有频率慢、不可靠）|
> | 定位含义 | 阻滞高（**nodal**），逃逸点紧贴阻滞下缘 | 阻滞低（**infranodal**），逃逸点位置低、随时停搏 |
> | 预后 / 处理 | 相对良性 → 观察（atropine 阿托品可能有效）| 危险 → 永久起搏器（atropine 阿托品无效）|
>
> **一句话类比**：窄 = 信号还能借**正常高速公路（His-Purkinje）**→ 同步 → 窄；宽 = 高速公路断了，信号在心室里走**泥巴路**→ 不同步 → 宽。
>
> > [!warning] 3° AV block 的逃逸 QRS **窄宽都可能** —— 别反射"3° = 宽 = 起搏器"⭐
> > "3° block"只说明房室**完全断开**（AV dissociation，房率 > 室率各跳各的），**没说断在哪一层**。断点高低决定逃逸点高低，进而决定 QRS 窄宽：
> >
> > | | **窄 QRS 的 3° block** | **宽 QRS 的 3° block** |
> > |---|---|---|
> > | 断点 | AV 结水平（nodal）| His 以下（infranodal）|
> > | 逃逸点 | junctional（交界区）| ventricular（心室）|
> > | 室率 | 40–60 bpm | 20–40 bpm |
> > | 典型病因 | **下壁 MI**（RCA→AV 结）、↑迷走、地高辛 digoxin 中毒、**先天性 3° AVB**（母 SLE/抗 SSA）| **前壁 MI**（LAD→His）、退行性传导病（Lenegre/Lev）|
> > | 处理 | 多观察 / 治因（atropine 阿托品可能有效）| **永久起搏器（pacemaker）** |
> >
> > **配套记忆**：1° 与 Mobitz I 几乎总是窄（nodal）；Mobitz II 常宽（infranodal）；**唯独 3° block 是"窄宽都可能"的那一个** → 所以 3° 才特别强调"看逃逸 QRS 宽窄再定位"。
>
> **USMLE 钩子**：3° AV block + **宽 QRS、HR 30** → 别等、别给 atropine（阿托品），直接 **pacemaker（起搏器）**；但 3° AV block + **窄 QRS**（如下壁 STEMI / 先天性）→ 逃逸点高、相对稳定 → 可能只需观察 / atropine。

### 3.3 为什么 Atropine 对 Infranodal 会恶化

```
        给 Atropine (抗迷走)
              ↓
        SA 放电↑ → 心房率从 60 → 100
              ↓
        但 His-Purkinje 已损伤，传不动这么多冲动
              ↓
        房室传导比例 2:1 → 3:1 或 4:1
              ↓
        心室率反而下降 ⚠️
```

> [!danger] 临床教训
> 怀疑 infranodal 时**不要依赖 atropine**，直接经皮起搏 (TCP) + 备永久起搏器。

### 3.4 高频题型

#### 题型 A：下壁 MI + AV block
> 65 岁男性，急性下壁 STEMI，3° AV block，HR 38，**窄 QRS escape 45**。给 atropine → 心率↑80，传导恢复。
>
> **诊断**：Nodal（RCA 供应 AV 结 90%）
> **处理**：Atropine, 通常自限，**不**需永久起搏器

#### 题型 B：前壁 MI + AV block
> 70 岁男性，急性前壁 STEMI，Mobitz II，HR 35，**宽 QRS escape**，atropine **无反应**。
>
> **诊断**：Infranodal（LAD → His 缺血）
> **处理**：立刻 TCP → **永久起搏器**（即使后续恢复也装）

#### 题型 C：运动员迷走过强
> 25 岁运动员，HR 45，1° AVB，PR 220 ms，**窄 QRS**，无症状。
>
> **诊断**：迷走张力高 → Nodal
> **处理**：观察，不治疗

### 3.5 冠脉解剖 — 为什么下壁/前壁 MI 对应不同 block

| 传导系统部位 | 主要血供 |
|---|---|
| SA node | RCA (~60%) / LCx (~40%) |
| **AV node** | **RCA (~90%)** / LCx (~10%) ⭐ |
| **His 束 + 束支** | **LAD (septal branches)** + 少量 RCA |
| Purkinje | 心内膜下，双重供应 |

```
        RCA (下壁 MI)               LAD (前壁 MI)
            ↓                            ↓
        供应 AV 结                   供应 His + 束支
            ↓                            ↓
        AV 结缺血                    His-Purkinje 缺血
            ↓                            ↓
        Nodal block                  Infranodal block
        窄 QRS, atropine 有效        宽 QRS, atropine 无效
        多自限                       需起搏器
```

### 3.6 对应规律 — 强倾向但**不是 100% 固定** ⚠

> [!warning] USMLE 默认规律
> - 下壁 MI + AV block → **默认 nodal** → atropine + 观察
> - 前壁 MI + AV block → **默认 infranodal** → 起搏器
>
> 但 ECG (QRS 宽度) + Atropine 反应才是**金标准**，MI 部位只是先验概率。

**三类例外**：

**例外 A**：大面积下壁 MI 偶尔造成 infranodal（累及间隔下部 / RCA+LCx 双病变）→ 按 ECG + atropine 反应处理

**例外 B**：前壁 MI 的 AV block 罕见为 nodal 起源，但**临床原则**：前壁 MI 出现的任何 AV block 都倾向于装永久起搏器（His 受损隐患极高）

**例外 C**：非 MI 病因

| 病因 | 倾向位置 | Stem 关键词 |
|---|---|---|
| **退行性传导病** (Lenegre / Lev) | Infranodal | 老年人，无 MI，渐进出现 |
| **迷走过度** | Nodal | 运动员、夜间、Valsalva |
| **药物** (β-blocker, CCB, digoxin, amiodarone) | Nodal | 用药史 |
| **Lyme 病** ⭐ | Nodal | **年轻人 + tick exposure + erythema migrans + 高位 AV block** |
| **结节病、淀粉样变** | Infranodal | 浸润性疾病侵及 His-Purkinje |
| **先天性 3° AVB** | Nodal（位置高）| 母亲 SLE / **抗 SSA (Ro) 抗体** |
| **主动脉瓣置换术后** | Infranodal | 手术机械损伤 His |

#### 例外 C 鉴别 SOP — 先问"可逆吗"，再决定起搏器 ⭐

> [!info] 核心轴 = 可逆 vs 不可逆（决定临时起搏 vs 永久起搏器 PPM）
> 非 MI 病因的 AV block，**真正的分诊问题不是 nodal/infranodal，而是"病因可逆吗"** —— 可逆的纠因 + 必要时临时起搏即可，**多不装永久起搏器 (PPM, permanent pacemaker)**；不可逆的结构/退行/浸润/术后损伤 → **PPM**。

```
            非 MI 的 AV block
                   ↓
         ① 有无可逆病因? ⭐
        ┌──────────┴───────────┐
        ↓                      ↓
   可逆（治因后多恢复）       不可逆（结构性损伤）
        ↓                      ↓
   - 药物(β/CCB/dig/胺碘酮)    - 退行性 Lenegre/Lev
     → 停药                    - 浸润性 结节病/淀粉样变
   - Lyme 心炎 → 抗生素        - 主动脉瓣术后/TAVR His 损伤
   - 迷走过度(运动员) → 观察   - 先天性 3° AVB(母 SLE/抗SSA)
        ↓                      ↓
   纠因 ± 临时起搏           → 永久起搏器 (PPM)
   多数不装 PPM              (浸润性同时治原发病)
```

| 病因 | 年龄/钩子 | 可逆? | 处理 |
|---|---|---|---|
| **Lyme 心炎 (Lyme carditis)** ⭐ | 年轻 + 蜱叮咬 / 环形红斑 (erythema migrans) / 露营 | **可逆** | 抗生素（头孢曲松 ceftriaxone IV 重症 / 多西环素 doxycycline）→ 多自愈;重度可临时起搏,**通常不装 PPM** |
| **药物** (β-blocker / CCB / 地高辛 digoxin / 胺碘酮 amiodarone) | 用药史 / 中毒 | **可逆** | 停药 ± 临时起搏;地高辛中毒查地高辛水平 |
| **迷走过度** | 运动员 / 夜间 / Valsalva | **可逆/良性** | 观察 |
| **退行性 (Lenegre / Lev)** | 老年、渐进、无 MI | 不可逆 | **PPM** |
| **浸润性（结节病 sarcoidosis / 淀粉样变 amyloidosis）** | 多系统受累 / 影像浸润征 | 不可逆 | **PPM** + 治原发病 |
| **主动脉瓣置换 / TAVR（经导管主动脉瓣置换）术后** | 术后新发 | 不可逆 | **PPM**（His 机械损伤）|
| **先天性 3° AVB** | 新生儿 / 母 SLE（系统性红斑狼疮）+ **抗 SSA(Ro) 抗体** | 不可逆 | 症状性 / 婴儿宽 QRS 逸搏 → **PPM** |

> [!tip] 一句话锁定（例外 C）
> **"年轻 + 蜱/红斑 → Lyme → 抗生素(可逆,别急着 PPM)";"老年渐进 / 术后 / 浸润 → 不可逆 → PPM"。** 先分"可逆否",再谈起搏。

### 3.7 实战判断流程（不要只看 MI 部位）

```
              AV block 患者
                   ↓
        ECG: escape rhythm QRS 宽度 ⭐
                   ↓
        ┌──────────┴──────────┐
        ↓                      ↓
   窄 QRS (40-60)         宽 QRS (20-40)
   junctional escape      ventricular escape
        ↓                      ↓
   倾向 Nodal             倾向 Infranodal
        ↓                      ↓
   验证: atropine         验证: atropine
        ↓                      ↓
   有效 → 确认 Nodal      无效 → 确认 Infranodal
        ↓                      ↓
   良性，观察 / 治病因    危险，TCP → 永久起搏器
```

> [!tip] USMLE 命题哲学
> 题目通常把所有线索**对齐**（部位 + QRS + atropine 三个都指向同一答案）。
> 偶尔"反着出"考鉴别：**以 ECG 和 atropine 反应为准，不是 MI 部位**。

### 3.8 实战澄清 — "不用治 vs 治"过度简化纠正

> [!warning] 常见误解
> "下壁 MI 的 AV block 不用治，前壁 MI 的 AV block 要治"是**过度简化**。
> 精确说法：**"不装永久起搏器 vs 必装永久起搏器"** — 急性期监测和应急处理两边都要做。

#### 总结对比

| MI 部位 | AV block 类型 | 处理 | 关键差异 |
|---|---|---|---|
| **下壁 (RCA)** | Nodal | 症状性 → **atropine ± TCP 备用** → 观察自限（数小时-数天）| **不装永久起搏器** |
| **前壁 (LAD)** | Infranodal | TCP **立即上** → 永久起搏器 | 即使后续传导恢复**也装**（His 受损隐患极高）|

#### 下壁 MI 的 AV block — 分情景细化

- **无症状** → 观察
- **症状性**（HR <50 + 低血压 / 头晕 / 心衰 / syncope）→ **atropine 0.5 mg IV** + TCP 备用
- Atropine **有效** → 确认 nodal → 继续观察，**不装 PPM**
- 关键是**不放永久起搏器**，但**急性期监测和应急处理仍要做**（不等于"完全不治"）

#### 前壁 MI 的 AV block — 硬规则

- 哪怕 atropine 偶尔有效，前壁 MI 出现的任何 AV block 都**倾向于装 PPM**（临床原则 — 见 §3.6 例外 B）
- 因为 His 区域 LAD 供血一旦缺血，**复发率高 + 猝死风险大**
- 即使后续传导恢复也装 PPM — 防止数天-数周后复发性 infranodal 阻滞导致猝死

> [!success] 一句话锁定
> **"下壁 MI 的 AV block 多是临时罢工（nodal），观察就好；前壁 MI 的 AV block 是永久断电（infranodal），装机器"**

## Memory Hooks

> [!success] 核心口诀
> 1. **"BMI ≥30 + AF 复发 + 常规检查正常 → 闭眼选 Sleep Study"**
> 2. **"CSM 治 R 不治 I"**：Regular SVT 有用，Irregular AF 没用
> 3. **"Atropine 是油门，不是 GPS"**：能治 brady，但不定位起搏点；唯一"定位"用途 = AV block 的 nodal/infranodal
> 4. **"窄良宽危，下壁活前壁亡"**：窄 QRS escape = nodal = 良性；宽 QRS escape = infranodal = 危险
> 5. **"治 AF 不治 OSA = 救火不关煤气"**：cardioversion 治标，找可逆病因才治本
> 6. **"部位是概率，ECG + atropine 是金标准"**：下壁→nodal、前壁→infranodal 只是先验，QRS 宽度 + atropine 反应才能确诊

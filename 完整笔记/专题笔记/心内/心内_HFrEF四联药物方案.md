# HFrEF 药物治疗阶梯（Pharmacotherapy for Chronic HFrEF）

> [!info] 笔记定位
> 慢性**射血分数降低型心衰**（HFrEF, heart failure with reduced ejection fraction，**LVEF ≤40%**；LVEF = left ventricular ejection fraction, 左室射血分数）的指南导向药物治疗（**GDMT**, guideline-directed medical therapy, 指南导向药物治疗）总表 + 启动顺序 + 高频陷阱。
> 分级用 **NYHA**（New York Heart Association, 纽约心脏协会）I–IV 级。
> 这是 [[完整笔记/专题笔记/心内/心内_BetaBlockers分类大全]] §TODO 计划生成的衍生笔记（HFrEF 四联用药）。
> **核心记忆**：现代 "四联基石（Fantastic Four）" = **ARNI/ACEI/ARB + β-blocker + MRA + SGLT2i**，四类都**降死亡率**，应尽早全部上齐再滴定。

---

## 一、总表（按治疗阶梯）

> [!tip] 一句话
> 凡 LVEF ≤40% 都该上的两类基石 = **血管紧张素系统阻断剂（首选 ARNI）+ β-blocker**；有容量负荷再加**袢利尿剂**（仅改善症状，不降死亡）。基石稳定后**依次叠加** MRA → SGLT2i。

| 药物 | 适应证 | 关键获益 | 备注 |
|---|---|---|---|
| **ARNI**（angiotensin receptor–neprilysin inhibitor, 血管紧张素受体-脑啡肽酶抑制剂）\* | NYHA I–IV + **LVEF ≤40%** | **延缓病程 + 降死亡率** | 首选；ACEI（angiotensin-converting enzyme inhibitor, 血管紧张素转换酶抑制剂）/ ARB（angiotensin receptor blocker, 血管紧张素受体阻滞剂）为替代 |
| **β-blocker**（β 受体阻滞剂） | NYHA I–IV + LVEF ≤40% | **降住院 + 降死亡率** | **仅 3 种**：美托洛尔琥珀酸盐 metoprolol succinate / 卡维地洛 carvedilol / 比索洛尔 bisoprolol |
| **袢利尿剂**（loop diuretic, 袢利尿剂） | NYHA **II–IV + 容量负荷** | 改善症状 + 降住院（**无死亡获益**） | 呋塞米 furosemide ± 美托拉宗 metolazone |
| **MRA**（mineralocorticoid receptor antagonist, 盐皮质激素受体拮抗剂；即醛固酮拮抗剂 aldosterone antagonist） | NYHA II–IV | **降住院 + 降死亡率** | 第 2 步叠加；螺内酯 spironolactone / 依普利酮 eplerenone |
| **SGLT2i**（sodium-glucose cotransporter-2 inhibitor, 钠-葡萄糖共转运体 2 抑制剂） | （LVEF ≤40%，症状性） | **减症状 + 降死亡率** | 第 3 步叠加；达格列净 dapagliflozin / 恩格列净 empagliflozin |
| **硝酸异山梨酯 + 肼屈嗪**（isosorbide dinitrate + hydralazine） | **血管紧张素系统阻断剂不耐受** | 改善症状 + 可能降死亡率 | 替补；自认黑人患者 NYHA III–IV 额外加用 |
| **地高辛**（digoxin） | 已用其他治疗仍持续有症状 | **降住院，无死亡获益** | 替补；窄治疗窗，肾功能不全 / 低钾 → 中毒 |

\* 其他血管紧张素系统阻断剂（**ACEI 或 ARB**）为 ARNI 的替代选项。

## 二、启动顺序与逻辑（UWorld 高频考点）

> [!success] 四类基石（均降死亡率）
> ```
> 第一线（凡 LVEF ≤40% 立即启动）
> ├── ① 血管紧张素系统阻断剂  ← 首选 ARNI；不能用则 ACEI；ACEI 咳嗽/血管性水肿 → ARB
> └── ② β-blocker（3 选 1）   ← 必须在容量稳定（euvolemic）后启动，不在急性失代偿期加量
>
> 第二步  ③ MRA（醛固酮拮抗剂）   ← 基石稳定后叠加
> 第三步  ④ SGLT2i              ← 再叠加（HFrEF 用药不依赖是否糖尿病）
> ```
>
> **袢利尿剂**独立于上面四类：**只在有容量负荷（肺水肿/外周水肿/JVD, jugular venous distension, 颈静脉怒张）时用**，纯改善症状，**不进"四联降死亡率"行列**。

> [!warning] 启动 / 切换的高频陷阱
> - **ARNI ↔ ACEI 不能同用、需洗脱**：ACEI 换 ARNI 必须停 ACEI **≥36 h**，否则血管性水肿（angioedema）风险叠加。
> - **β-blocker 不在急性失代偿期启动 / 加量**：必须先利尿达到容量稳定（euvolemic）再上 BB；急性肺水肿期加 BB 会压低心输出量。
> - **BB 只认 3 个名字**：metoprolol **succinate**（琥珀酸盐=缓释，非酒石酸盐 tartrate）、carvedilol、bisoprolol；其余 BB 无 HFrEF 死亡率证据。
> - **MRA 用前查 K⁺ 和 eGFR**：K⁺ >5.0 或 eGFR <30 → 不启动（高钾风险）。
> - **CCB（calcium channel blocker, 钙通道阻滞剂）多数禁用**：非二氢吡啶类（地尔硫䓬 diltiazem / 维拉帕米 verapamil）负性肌力 → HFrEF 禁用；只有氨氯地平 amlodipine / 非洛地平 felodipine 在需要降压时相对安全（无获益但不害）。

## 三、按获益类型分组（速记）

| 获益 | 药物 | 记忆钩 |
|---|---|---|
| **降死亡率** ✅ | ARNI/ACEI/ARB、β-blocker、MRA、SGLT2i、（肼屈嗪+硝酸酯） | "四联基石" + 不耐受替补的肼屈嗪+硝酸酯 |
| **仅改善症状 / 降住院，无死亡获益** | 袢利尿剂、地高辛 | "**利尿剂 + 地高辛 = 让你舒服但不让你活得更久**" |

> [!danger] 最常考的反直觉点
> **袢利尿剂和地高辛都不降死亡率**。题目若问"以下哪个能改善该 HFrEF 患者的**生存率/预后**"，**别选利尿剂、别选地高辛** —— 选还没上的那个基石（ARNI / BB / MRA / SGLT2i）。

## 四、特殊人群补充

> [!info] 自认黑人（self-identified Black）NYHA III–IV
> 在已用 ARNI/ACEI + BB + MRA 基础上**额外加** 肼屈嗪 + 硝酸异山梨酯（A-HeFT 试验：该人群额外死亡率获益）。注意：这里是"**加用**"，不是"替代不耐受"那一栏。

> [!info] 器械治疗（药物之外，知道触发即可）
> - **ICD**（植入式心律转复除颤器）：GDMT ≥3 个月后 LVEF 仍 ≤35% + NYHA II–III → 一级预防猝死。
> - **CRT**（心脏再同步化）：LVEF ≤35% + **LBBB（left bundle branch block, 左束支传导阻滞）+ QRS ≥150 ms** + NYHA II–IV。
> - 详见 [[完整笔记/专题笔记/心内/心内_ACLS急救决策树]] / 主笔记。

## 五、自测

> [!question]- Q1. 一位 NYHA II 级、LVEF 30% 的 HFrEF 患者，已规律服用 ARNI、carvedilol、spironolactone、呋塞米，无容量负荷，血压心率可耐受。下列哪项最能进一步降低其死亡率？
> **SGLT2i（如 dapagliflozin / empagliflozin）**。患者已上三类基石，缺的是第四联 SGLT2i（降死亡率）。
> 干扰项：加大袢利尿剂剂量 = 只改善症状无死亡获益；加 digoxin = 降住院无死亡获益。

> [!question]- Q2. HFrEF 患者因 ACEI 干咳准备换 ARNI，需注意什么？
> 停 ACEI **≥36 小时**再启 ARNI（避免血管性水肿叠加）；二者**不可同服**。

> [!question]- Q3. 哪些药物对 HFrEF "只改善症状/降住院但不降死亡率"？
> **袢利尿剂 + 地高辛**。

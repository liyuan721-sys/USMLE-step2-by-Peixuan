# 新生儿期紫绀 / 休克的发绀型先天性心脏病鉴别（5T + H + IAA 完整版）

> [!info] 一句话定位
> 新生儿期严重紫绀 + 休克的先心病 = **5T + H + IAA**（共 7 个核心病）
> USMLE 几乎必考其中 1-2 个 → 必须能秒答 + 互相鉴别

> [!success] ⭐⭐ 核心口诀（v3 升级版）
> **"糖妈生鸡蛋，DiGeorge 出靴干弓"**
>
> - **糖妈**（糖尿病母亲）→ **鸡蛋**（Egg on string）= **TGA**
> - **DiGeorge**（22q11.2）→
>   - **靴**（Boot heart）= **TOF**（15-20%）
>   - **干**（共干 Truncus）= **Truncus**（50%）
>   - **弓**（弓中断 IAA）= **IAA-B**（50-90% ⭐ 最高）
>
> 一个口诀锁定 4 大发绀型先心病的高危关联

## 1. 7 大新生儿紫绀 / 休克先心病 ⭐⭐

### 完整列表

| 数字 / 编号 | 病 | 缩写 | 紫绀 / 休克类型 |
|---|---|---|---|
| **1** | **T**runcus arteriosus | T | 紫绀（轻）+ 心衰 |
| **2** | **T**ransposition (D-TGA) | T | 紫绀（重）|
| **3** | **T**ricuspid atresia | T | 紫绀 |
| **4** | **T**etralogy of Fallot | T | 发作性紫绀 |
| **5** | **T**APVR | T | 紫绀 + 呼吸窘迫 |
| **+H** | **H**LHS | H | 全身休克 + 灰白 |
| **+IAA** | **I**nterrupted **A**ortic **A**rch ⭐ v3 新增 | IAA | **差异性紫绀** + 下肢休克 |

> [!warning] 为什么把 IAA 也加入？
> - IAA 与 22q11.2 / DiGeorge 关联最强（Type B 高达 50-90%）
> - 临床表现极具特色：**差异性紫绀** + **PDA 关闭后突发休克** + **下肢脉搏消失**
> - USMLE Step 2 高频考点
> - 与 HLHS / Coarctation 容易混淆，必须能鉴别

| 数字 | 病 | 数字含义 |
|---|---|---|
| 1 | **T**runcus | 1 个共干 |

| 3 | **T**ricuspid atresia | "Tri" = 3 |

| 5 | **T**APVR | 5 个字母 |

## 3. ⭐⭐ Conotruncal 畸形三巨头详解（v3 重点）

> [!info] Conotruncal 是什么？
> **圆锥动脉干（Conotruncal）** = 心脏胚胎发育中的"流出道"区域
> 神经嵴细胞参与此处发育
> 22q11.2 缺失 → 神经嵴细胞迁移异常 → conotruncal 畸形

### Conotruncal 三大畸形 + 22q11.2 关联

| 病 | 22q11.2 关联 | 数字法 | USMLE 主要特征 |
|---|---|---|---|
| **Truncus arteriosus** | **~50%** | 1 | 共干 + VSD + **轻紫绀重心衰** + 洪脉 |
| **TOF** | ~15-20% | 4 | **Tet spells + 蹲下缓解** + Boot heart |
| **IAA (B 型)** ⭐ | **50-90%** ⭐⭐ 最高 | — | **PDA 关后突发休克** + **差异性紫绀** + 下肢脉消 |

> [!success] DiGeorge 三大心脏病
> **"TOF + Truncus + IAA"** = 22q11.2 缺失三巨头
> 都是 conotruncal 畸形
> **IAA-B 关联最强（50-90%）**，Truncus 中等（50%），TOF 最弱（15-20%）

### 反直觉重点

> [!warning] 绝对数量 vs 相对比例
> - **绝对数量上**：TOF 是 DiGeorge 患者最常见的心脏畸形（因为 TOF 本身高发）
> - **相对比例上**：IAA-B 与 22q11.2 关联最强（90%）
> - **两者都对**，看题目问的角度

## 4. ⭐ IAA 单独详解（v3 新增）
Interrupted Aortic Arch（主动脉弓中断）
22q11.2 / DiGeorge 三大 conotruncal 畸形之一
### 一句话锁定

> [!tip] 黄金特征 **新生儿期 PDA 关闭后突发**下半身灌注不良 + 上下肢血压悬殊 + DiGeorge 三联（腭裂 + 低钙 + 反复感染） = IAA（Type B 最常见，**22q11.2 关联高达 50-90%**）⭐⭐
> 
### 解剖三型

```
正常主动脉弓：
                 ┌── 头臂干
升主动脉 ── 弓 ──┤── 左颈总
                 └── 左锁骨下 ── 降主动脉

IAA Type B 最常见 (~70%)：
     头臂干 ─ 左颈总 ┊ 左锁骨下 ─ 降主动脉
                    ↑
                  这里断
```

### 病理生理

```
出生时 PDA 开放：临床看似正常 ✓
        ↓
PDA 关闭（24-72h）→ 下半身完全无血
        ↓
下肢脉消 + 灰白 + 休克 + 代酸 + 死亡
```

### Stem 关键词

| 关键词 | 意义 |
|---|---|
| "Sudden shock at day 2-3" | PDA 关闭后突发 |
| **"Absent femoral pulses"** ⭐ | 下肢脉搏消失 |
| **"Differential cyanosis"** ⭐⭐ | 差异性紫绀 |
| "Upper > lower extremity BP" | 上下肢 BP 悬殊 |
| **"Cleft palate + hypocalcemia + recurrent infections"** | DiGeorge 三联 |
| "FISH for 22q11.2" | DiGeorge 确诊 |

### 治疗

> [!danger] 三步走
> 1. **PGE₁ 救命**（维持 PDA → 恢复下半身灌注）
> 2. **稳定 + DiGeorge 评估**：补钙 + **输血照射血液制品**（防 GVHD）+ 避免活疫苗
> 3. **一期主动脉弓重建 + 修 VSD**（80-90% 合并 VSD）

### IAA vs Coarctation 鉴别 ⭐

| 项目 | Coarctation | IAA |
|---|---|---|
| 解剖 | 主动脉**狭窄但相连** | 完全**断开** |
| 严重程度 | 变化大 | 极重 |
| 22q11.2 | 弱 | **强**（Type B 50-90%）|
| CXR | "3" sign + 肋骨切迹（年长儿）| 心增大 + 肺血 ↑ |

> [!tip] 一句话鉴别
> **Coarctation = 狭窄**（血流减少但相连）
> **IAA = 断开**（血流完全靠 PDA 绕过）

## 5. 22q11.2 = DiGeorge 综合征完整解析

> [!danger] 核心概念
> **22q11.2 deletion syndrome ≡ DiGeorge syndrome**
> 同一个病的两个名字（在 USMLE 上完全等价）

### 病因学

```
22q11.2 缺失
        ↓
TBX1 基因缺失（关键转录因子）
        ↓
神经嵴细胞迁移异常
        ↓
影响第三、第四咽弓发育
        ↓
┌──────────┬──────────┬──────────┐
↓          ↓          ↓          ↓
心脏        胸腺       甲状旁腺   面部 + 腭
↓          ↓          ↓          ↓
Conotruncal T 细胞     低钙       特征面容
畸形        缺陷       抽搐       + 腭裂
↓
TOF + Truncus + IAA
```

### CATCH-22 助记符 ⭐⭐

> [!success] DiGeorge = CATCH-22
> - **C**ardiac defects（TOF / Truncus / **IAA**）⭐
> - **A**bnormal facies（小下颌、低位耳、内眦距宽）
> - **T**hymic aplasia（T 细胞缺陷 → 反复感染）⭐
> - **C**left palate（腭裂）
> - **H**ypocalcemia（甲状旁腺发育不良 → 低钙、抽搐）⭐
> - **22** = 22q11.2 染色体缺失

### DiGeorge 患儿的特殊管理

> [!warning] 不能忽视的两点
> 1. **输血必须照射血液制品**（防 GVHD，T 细胞缺陷）⭐
> 2. **避免活疫苗**（MMR / Varicella / 卡介苗 / 黄热病 / 鼻喷流感疫苗）⭐⭐

## 6. 双轴鉴别（最重要的快筛）⭐

### 轴 1：肺血流方向

| 肺血 ↑（5 个）| 肺血 ↓（2 个）|
|---|---|
| Truncus / TGA / TAPVR / HLHS / IAA | TOF / Tricuspid atresia |

### 轴 2：PDA 依赖性

| **PDA 依赖**（PGE₁ 救命）⭐ | **非 PDA 依赖** |
|---|---|
| D-TGA（混合）| Truncus（已混合）|
| Tricuspid atresia（肺循环）| TAPVR（依赖 ASD/PFO）|
| **HLHS（体循环）**⭐ | TOF（除非肺血极少）|
| **IAA（下半身）**⭐ | |
| Severe TOF / Pulm atresia | |
| Critical Coarctation | |

> [!danger] 三大经典考点
> 1. **TAPVR：PGE₁ 无效** → ASD/PFO 依赖
> 2. **HLHS：PGE₁ 救命** → 全部体循环
> 3. **IAA：PGE₁ 救命** → 下半身体循环
> 4. **TGA：PGE₁ 救命** → 混合通道

## 8. PDA 关闭后突发休克 — 四大鉴别（v3 新增专题）

> [!info] 概念
> 出生时看似正常 + **PDA 关闭后（24-72h）突发休克** = 体循环依赖型先心病

| 病 | 上肢 BP | 下肢 BP | 差异性紫绀 | DiGeorge | 鉴别要点 |
|---|---|---|---|---|---|
| **HLHS** | 弱 | 弱 | 无 | 无 | 全身灌注差 |
| **IAA** | **强** | **极低** | **有** ⭐ | **强** ⭐ | 上下半身分裂 |
| **Critical Coarctation** | 强 | 弱 | 有（轻）| 弱 | 主动脉狭窄但相连 |
| **Critical AS** | 弱 | 弱 | 无 | 无 | 主动脉瓣狭窄 → LV 流出受阻 |

> [!tip] 鉴别神技
> **看 4 肢 BP**：
> - 全弱 → HLHS / Critical AS
> - 上强下弱 + 有差异性紫绀 → **IAA** ⭐
> - 上强下弱 + 无明显差异性紫绀 → **Coarctation**

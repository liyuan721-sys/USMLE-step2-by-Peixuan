# IE 预防适应证与换药（Infective Endocarditis Prophylaxis）

> [!info] 笔记定位
> **IE = 感染性心内膜炎 infective endocarditis**。本笔记是从 Q127（人工瓣膜 + 牙科 + 青霉素过敏换药）派生的**聚焦专题**，只回答两道门槛问题：
> 1. **谁 + 哪些操作**需要预防（适应证）
> 2. **用什么药 / 过敏怎么换**（换药表）
>
> 完整 IE 诊断 / 治疗 / 并发症 / 手术指征体系见 [[完整笔记/专题笔记/感染/感染_感染性心内膜炎完整体系]] §十二（本笔记是其预防章节的"换药放大版"，不重复诊断与治疗）。
>
> 派生自 [[NBME/nbme-records/NBME 11/NBME11_错题本#^Q127]]（兑现 bd Q127 TODO）。

---

## 一、核心两道门槛

> [!danger] 决定 IE 预防只看两件事 ⭐
> ① **高危心脏病变？**（不是杂音、不是年龄、不是 AS 严重度在决定）
> ② **高危操作？**（穿破牙龈/根尖/口腔黏膜的牙科操作；某些呼吸道；感染的皮肤/肌骨）
>
> **两者都满足** → 给预防（首选 **阿莫西林 amoxicillin 2 g**，术前 30–60 min）。
> **缺任意一个** → 不给。
>
> 第三步才是：**青霉素过敏？** → 换非 β-内酰胺替代药（见 §四换药表）。

---

## 二、谁需要预防 — 高危心脏病变（AHA 现行标准）

> [!warning] 只有"最高危"这 6 类才预防
> 现行 AHA 标准**大幅收窄**了适应证：只有 IE 后果最严重的心脏病变才预防，普通瓣膜病/分流**不再**预防。

| 高危心脏病变（满足才考虑预防） | 备注 |
|---|---|
| **人工心脏瓣膜（prosthetic valve）** | 含生物瓣 / 机械瓣 / **TAVI 经导管瓣**；也含用于瓣膜修复的**人工材料/环（prosthetic material）** |
| **既往 IE 史（previous infective endocarditis）** | 复发风险高 |
| **未修复的发绀型先心病（unrepaired cyanotic CHD）** | 含姑息性分流/导管 |
| **先心修复后 6 个月内**（用人工材料/装置修复） | 内皮化需 ~6 个月；6 个月后无残余缺损则停 |
| **修复后有残余缺损（residual defect）** | 在补片旁或装置旁有残余分流 → 内皮化受阻 → 持续高危 |
| **心脏移植后出现瓣膜病变（cardiac transplant with valvulopathy）** | 移植心瓣膜反流/狭窄 |

> [!tip] 一句话锁定高危心脏病变
> **"人工的、得过的、发绀没修的、刚修不到半年的、修了没修干净的、移植心长了瓣膜病的"** —— 6 类之外（风心、MVP、单纯 MR、ASD/VSD 修复后无残余、二叶瓣）**都不预防**。

## 三、哪些操作需要 — 高危 vs 低危操作

| 类别 | 具体操作 | 是否预防 / 给什么 |
|---|---|---|
| **高危·牙科** ⭐ | 操作**穿破牙龈（gingival）/ 根尖周（periapical）/ 口腔黏膜（oral mucosa）** —— 拔牙、牙周手术、种植、根管 | **预防**：阿莫西林 amoxicillin 2 g PO |
| **高危·呼吸道** | **切开/活检呼吸道黏膜**（如扁桃体切除、支气管镜下活检） | 预防：阿莫西林 amoxicillin |
| **高危·感染皮肤/肌骨** | 在**已感染的皮肤、皮下或肌肉骨骼组织**上手术 | 预防需覆盖葡萄球菌 → 含**万古霉素 vancomycin / 头孢** |
| **GI / GU** | 单纯胃肠/泌尿内镜或操作（**无活动感染**） | **不预防** ⭐（2007 起踢出名单） |
| **低危·常规** | 常规洁牙（不穿黏膜）、局麻注射、补牙、X 线、矫正器调整、阴道分娩/剖宫产、机械瓣做**筛查性结肠镜** | **不预防** |

> [!danger] GI / GU 单纯操作 since 2007 = OUT ⭐
> **"机械瓣患者去做 routine screening colonoscopy 要不要预防？" → 不要。**
> 2007 年前 GI/GU 高风险操作曾用 **IV 氨苄西林 ampicillin + 庆大霉素 gentamicin** 预防，**现已完全废除**，但 USMLE 干扰项高频出现。
> - 例外：GI/GU 操作时**正存在活动性感染** → 给抗生素是为了**治疗那个感染**（按感染源选药，氨苄西林 ampicillin 覆盖肠球菌），**不是 IE 预防剂量**。

## 四、用药 + 青霉素过敏换药表 ⭐⭐（Q127 核心）

> [!danger] 换药铁律：青霉素过敏 → 跳出整个 β-内酰胺类
> **阿莫西林 amoxicillin = 氨基青霉素（aminopenicillin）= β-内酰胺**。青霉素过敏者对它**交叉过敏** → 整个 β-内酰胺类都要避开。
> **氨苄西林 ampicillin 也是青霉素类**（Q127 陷阱选项），照样禁 —— 不能拿它替代阿莫西林。

牙科/呼吸道操作的标准方案（术前 **30–60 min** 单剂 PO，下表均按成人剂量记"档"，CK 重逻辑不重精确 mg）：

| 情况 | 药物 | 备注 |
|---|---|---|
| **首选（无过敏）** | **阿莫西林 amoxicillin 2 g PO** | 一线；覆盖口腔 viridans 链球菌 |
| 不能口服 | **氨苄西林 ampicillin IV/IM** 或 **头孢唑林 cefazolin / 头孢曲松 ceftriaxone IV** | 仅"不能 PO"时用，非过敏替代 |
| **青霉素过敏（非速发/非重症，如单纯皮疹）** | **头孢氨苄 cephalexin（第一代头孢）** | 与青霉素交叉过敏率低；**速发过敏/过敏性休克史则禁用任何头孢** |
| **青霉素过敏（含速发/严重，如荨麻疹·血管性水肿·过敏性休克）** | **大环内酯 macrolide：阿奇霉素 azithromycin 或 克拉霉素 clarithromycin**（PO）| ⭐ Q127 答案路径（既往 hives = 荨麻疹 → 跳到大环内酯）|

> [!warning] 克林霉素 clindamycin 已被现行指南删除 ⭐（按最新核对）
> **旧版** AHA 把克林霉素 clindamycin 列为青霉素过敏的 IE 预防替代；**最新版（AHA 2021 更新）已不再推荐 clindamycin**（因其腹泻 / 艰难梭菌 *C. difficile* 结肠炎风险高于获益）。
> - 现行青霉素过敏替代 = **头孢氨苄 cephalexin（无速发过敏时）/ 阿奇霉素 azithromycin / 克拉霉素 clarithromycin**。
> - ⚠️ CK 过渡期：部分题库/旧资料仍把 clindamycin 当正确替代。**若选项里既有大环内酯又有克林霉素**，优先选**大环内酯**（更稳的现行答案）；若**只有克林霉素**而无大环内酯，按该题库版本可能仍接受 —— 但默认记忆按"已删除"。

> [!success] Memory Hook
> **人工瓣做牙科 → 要 IE 预防；首选阿莫西林 2 g，青霉素过敏 → 大环内酯（阿奇/克拉霉素）。氨苄西林也是青霉素类，别选；克林霉素现指南已删。**

## 五、易混陷阱（普适）

> [!danger] 五大反向陷阱
> - ❌ **"有杂音就预防"** → 错。决定预防的是**高危病变（人工瓣等 6 类）**，不是杂音 / AS 严重度。很多人有杂音，不据此预防。
> - ❌ **"青霉素过敏 → 换氨苄西林 ampicillin"** → 错。氨苄西林仍是**β-内酰胺/青霉素类**，照样禁；要换**大环内酯**（Q127 我做错点）。
> - ❌ **"机械瓣做筛查性结肠镜要预防"** → 错。单纯 GI/GU（无活动感染）**不预防**（since 2007）。
> - ❌ **"GI/GU 操作 → IV 氨苄西林 ampicillin + 庆大霉素 gentamicin"** → 错。2007 前老方案，已废除（高频干扰项）。
> - ❌ **"IV vancomycin 是常规牙科预防"** → 错。万古霉素仅用于**感染皮肤/肌骨手术**覆盖葡萄球菌，不是门诊牙科首选。

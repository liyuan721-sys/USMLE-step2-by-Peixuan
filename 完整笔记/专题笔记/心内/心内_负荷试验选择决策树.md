# 负荷试验选择决策树（Stress Test — Which One, When）

> [!info] 定位
> 稳定型疑似冠心病 (CAD, coronary artery disease, 冠心病) 的无创缺血评估，考点几乎永远是**"选哪一种负荷试验"**，而不是"要不要做"。核心两个轴：**① 能不能充分运动 ② 静息 ECG (electrocardiogram, 心电图) 能不能解释 ST 段**。
> 派生自 [[NBME11_错题本#^Q079]]（糖尿病女性劳力气短 + 双膝 OA (osteoarthritis, 骨关节炎) 不能运动 → 腺苷核素，误选静息心超）。

---

## 一、何时做负荷试验（前置）

- **稳定劳力性症状（胸痛 / 气短 / anginal equivalent 心绞痛等价症）+ CAD 危险因素**，且**无急性冠脉综合征 (ACS, acute coronary syndrome, 急性冠脉综合征) 表现** → 无创负荷分层。
- **急性胸痛 / 疑 ACS** → 查肌钙蛋白 (troponin) + ECG，**不是**负荷试验。
- **已知高危 / 药物难治性心绞痛** → 直接冠脉造影 (cardiac catheterization, 心导管)。

---

## 二、选择决策树 ⭐⭐⭐

```
稳定劳力症状 + CAD 危险因素，无 ACS
        ↓ 无创负荷
┌─ 能充分运动达标？(达 ~85% 最大预测心率)
│
├─ 能 + 静息 ECG 可解释 → 运动平板 ECG (exercise treadmill ECG)  ← 首选、最便宜
│
├─ 能运动，但静息 ECG 不可解释
│     (LBBB 左束支传导阻滞 / 起搏心律 / 静息 ST 压低 / 预激 / 地高辛效应)
│        → 运动 + 影像（运动核素 SPECT 或 运动负荷超声）
│
└─ 不能充分运动
      (膝/髋 OA 骨关节炎、外周动脉病、截肢、体弱、肺病、严重肥胖)
         → 药物负荷影像 (pharmacologic stress imaging)
              ├─ 血管扩张剂核素：腺苷 adenosine / 双嘧达莫 dipyridamole / regadenoson + SPECT
              └─ 多巴酚丁胺 dobutamine 负荷超声（不能用血管扩张剂时，如严重哮喘/COPD）
```

---

## 三、为什么"不能运动"要换药物负荷 ⭐

- 运动平板靠**真实运动**诱发心肌缺血 + 看 ST 改变；达不到目标心率 → **假阴性**，无诊断价值。
- 药物负荷绕过运动：**血管扩张剂（腺苷/双嘧达莫）** → 正常冠脉扩张、狭窄段不扩 → 灌注**不均**（核素显像差异）；**多巴酚丁胺** → 增心率心肌耗氧 → 诱发室壁运动异常（超声）。

| 不能运动的常见 stem 钩子 | 例子 |
|---|---|
| 关节/骨科 | **双膝/髋骨关节炎**、严重下背痛、截肢 |
| 血管 | 外周动脉病 (PAD, peripheral arterial disease) 跛行 |
| 全身 | 体弱老人、严重肥胖、慢性肺病、神经病变 |

---

## 四、药物负荷选哪种

| 方式 | 机制 | 避免/禁忌 |
|---|---|---|
| **腺苷 / 双嘧达莫 / regadenoson + 核素 SPECT** | 冠脉血管扩张 → 灌注差异 | **严重哮喘 / COPD 支气管痉挛**（腺苷可诱发）；高度房室传导阻滞；近期咖啡因/茶碱 |
| **多巴酚丁胺 dobutamine 负荷超声** | ↑心率心肌耗氧 → 室壁运动异常 | 严重心律失常、流出道梗阻；血管扩张剂禁忌时改用 |

---

## 五、各检查定位（干扰项速查）

| 检查 | 用途 | 不是首选当 |
|---|---|---|
| 心肌酶（肌钙蛋白）| 急性 ACS | 稳定亚急性缺血评估 |
| 运动平板 ECG | 能运动 + 静息 ECG 可解释 | 不能运动 / ECG 不可解释 |
| 药物负荷影像 | 不能运动 / ECG 不可解释 | — |
| 静息超声心动图 | 看结构 / EF (ejection fraction, 射血分数) / 瓣膜 | **不是缺血功能分层**（误区，见 Q079）|
| 冠脉造影 | 无创异常后、高危、急性、药物难治 | 作为首个筛查太侵入 |

---

## 🎣 Memory Hook

> [!success] 一句话锁定
> **查冠心先负荷；能跑 → 运动平板，跑不动（膝 OA / PAD / 虚弱）→ 药物负荷影像。静息 ECG 不可解释（LBBB / 起搏）→ 也得加影像。静息心超看结构不查缺血。**

# 血管影像选择决策 (Vascular Imaging — Which Test When)

> [!danger] 先破一个错误口诀
> ❌ **"CT 血管造影 (CT angiography, CTA) 看动脉、超声 (ultrasound, US) 看静脉"——错。**
> 选哪个检查**不取决于动脉/静脉**，而取决于 **① 查什么病 ② 哪个部位 ③ 急不急 ④ 患者能不能用辐射/造影剂**。
> - 超声照样是**多个动脉病的首选**：腹主动脉瘤筛查、颈动脉狭窄、外周动脉病初筛。
> - CT 照样查静脉：**CT 静脉造影 (CT venography, CTV)** 查盆腔深静脉、脑静脉窦血栓。
> - 真正的"静脉→超声"只有一个铁律场景：**下肢 DVT（deep vein thrombosis, 深静脉血栓）首选加压超声**。

---

## §一 五大手段的特长与短板

> [!info] 缩写
> CTA = CT angiography（CT 血管造影）、MRA = magnetic resonance angiography（磁共振血管造影）、US = ultrasound（超声）、DSA = digital subtraction angiography（数字减影血管造影，即导管法）、CIN = contrast-induced nephropathy（造影剂肾病）、NSF = nephrogenic systemic fibrosis（肾源性系统性纤维化）。
> 场景缩写：DVT = deep vein thrombosis（深静脉血栓）、PE = pulmonary embolism（肺栓塞）、CTPA = CT pulmonary angiography（CT 肺动脉造影）、V/Q = ventilation/perfusion scan（通气灌注扫描）、AAA = abdominal aortic aneurysm（腹主动脉瘤）、PAD = peripheral arterial disease（外周动脉病）、ABI = ankle-brachial index（踝肱指数）、RAS = renal artery stenosis（肾动脉狭窄）、TEE = transesophageal echocardiography（经食道超声）、CTV = CT venography（CT 静脉造影）、MRV = MR venography（磁共振静脉造影）、FAST = focused assessment with sonography for trauma（创伤超声重点评估）。

| 手段 | 最强场景 | 优势 | 短板/禁忌 |
|---|---|---|---|
| **多普勒超声 (Doppler US)** | DVT、颈动脉、AAA 筛查、PAD 初筛、肾动脉初筛 | **无辐射、无造影剂、床旁、便宜、可重复**；测血流方向/速度 | 操作者依赖、深部/含气部位差（肺、肠气遮挡）、视野局限 |
| **CT 血管造影 (CTA)** | 肺栓塞、主动脉夹层、急性肠系膜缺血、创伤、术前路线图 | **快、全景、空间分辨率高**、急诊可得 | **碘造影剂 → CIN（肾功能差慎用）+ 过敏**；辐射（孕妇慎） |
| **磁共振血管造影 (MRA)** | 肾动脉、颅内外血管、不能用碘造影剂者 | 无电离辐射；软组织对比好 | 慢、贵、不适合不稳定患者；**钆造影剂 → 重度肾病 (eGFR<30) NSF 风险** |
| **导管血管造影 (DSA, 金标准)** | 确诊 + **同期介入治疗**（支架/取栓/栓塞） | **金标准**、可同时治疗 | **侵入性**（出血/夹层/栓塞/CIN）；只在要干预时上 |
| **V/Q 通气灌注扫描** | 肺栓塞但**不能用碘造影剂**（肾衰/碘过敏）或孕妇优选 | 避开碘造影剂 | 有基础肺病时结果模糊；不能查别的诊断 |

## §二 按临床场景选（核心高频表）⭐

> [!warning] 这才是出题点：题干给"哪个病 + 患者背景"→ 选检查
> "next best step in diagnosis" 类题，答案常卡在 **首选(初筛) vs 确诊/术前** 这一步，别一上来就选金标准 DSA。

| 临床场景 | 首选/初筛 | 确诊 / 术前 / 不能用造影剂时 |
|---|---|---|
| **下肢 DVT** | **加压超声 (compression US)** | （US 阴性但高度怀疑 → 复查或 D-dimer 联合） |
| **肺栓塞 (PE)** | **CT 肺动脉造影 (CTPA)** | 不能用碘造影剂/孕妇 → **V/Q 扫描**；先查腿 DVT |
| **主动脉夹层 (aortic dissection)** | 稳定 → **CTA**（首选） | 不稳定/不能搬动/肾衰 → **经食道超声 (TEE)**；备选 MRA |
| **腹主动脉瘤 (AAA) 筛查/随访** | **超声**（一次性筛查 65–75 岁吸烟男） | 术前规划/稳定患者评估 → **CTA** |
| **AAA 破裂** | 不稳定 → **直接手术**（不等影像）；稳定 → CTA | — |
| **颈动脉狭窄 (carotid stenosis)** | **颈动脉双功超声 (carotid duplex)** | 术前确认 → **CTA / MRA** |
| **外周动脉病 (PAD)** | **ABI（踝肱指数）** ± 多普勒 US | 拟血运重建 → CTA/MRA → **DSA（同期介入）** |
| **肾动脉狭窄 (renal artery stenosis)** | **多普勒超声**（或 CTA/MRA） | 金标准 **DSA**（拟支架时） |
| **急性肠系膜缺血 (acute mesenteric ischemia)** | **CTA**（首选） | 金标准 DSA（可同期溶栓/取栓） |
| **脑静脉窦血栓 (cerebral venous sinus thrombosis)** | **MRV 或 CTV** | — |

## §三 患者因素——一票否决（高频陷阱区）

> [!danger] 题干埋的"造影剂禁忌"钩子
> 看到下面这些背景，**首选检查要换成"无造影剂/无辐射"路线**——这是 CK 最爱埋的转折。

```
患者背景 → 影像调整
├── 肾功能差 (CKD, ↑Cr / eGFR↓)
│     ├── 避碘造影剂 (CIN) → CTA 换成 超声 / MRA(无钆) / V/Q
│     └── 重度肾病 eGFR<30 → 连钆也避 (NSF) → 优先超声 / 平扫
├── 碘造影剂过敏史
│     └── CTPA 换 V/Q；CTA 换 MRA / US
├── 孕妇
│     ├── 首选无辐射：超声 / MRI(不打钆)
│     └── 疑 PE：先双下肢加压超声；必要时 V/Q 或低剂量 CTPA
└── 血流动力学不稳定 / 不能搬去 CT
      └── 床旁能做的：超声 / TEE（夹层）/ FAST（创伤）
```

## §四 决策骨架

```
怀疑血管病 → 问四件事
│
├─ ① 哪个病 / 哪个部位？ ── 决定"看哪段血管"
├─ ② 急不急？
│     急 + 要全景/路线图 → CTA（快、广）
│     不急 / 筛查 / 随访 → 超声（无辐射可重复）
├─ ③ 要不要同期治疗？
│     要介入（取栓/支架/栓塞）→ 导管造影 DSA（金标准 + 治疗）
└─ ④ 患者能用造影剂/辐射吗？
      肾差/碘过敏/孕妇 → 避碘 → 超声 / MRA / V/Q（见 §三）
```

## §五 高频陷阱

> [!warning] 易错点清单
> - **AAA 筛查选 CT** ❌ → 是**超声**（无辐射、便宜，适合一次性人群筛查）。
> - **颈动脉狭窄首选 CTA/血管造影** ❌ → 先 **颈动脉双功超声**，术前才上 CTA/MRA。
> - **PAD 一上来做血管造影** ❌ → 先 **ABI**，拟血运重建才逐级升到 DSA。
> - **肾衰患者疑 PE 仍做 CTPA** ❌ → 换 **V/Q 扫描**（避碘造影剂 CIN）。
> - **不稳定夹层推去 CT** ❌ → 不能搬动/肾衰 → **TEE 床旁**。
> - **把 DSA 当首选** ❌ → DSA 是金标准但侵入，**只在要同期介入治疗时上**。

> [!tip] 一句话锁定
> **影像选择 = "查什么病 + 急不急 + 要不要治 + 能不能用造影剂"四问，不是"动脉/静脉"二分。** 超声 = 无辐射无造影剂的初筛/筛查王（含动脉）；CTA = 急诊全景王（怕肾/孕妇/碘过敏要让位）；DSA = 要治疗才上的金标准。

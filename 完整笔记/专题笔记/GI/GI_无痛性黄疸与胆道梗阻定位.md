# GI_无痛性黄疸与胆道梗阻定位（painless jaundice / Courvoisier 征 / 梗阻部位分流）

> [!info] 这张的定位
> 无痛性黄疸 (painless jaundice) 在 [[完整笔记/专题笔记/GI/GI_HighYield速查总览]] §10.2 只占一行（"胰腺癌 = 无痛性黄疸 + Courvoisier 征"）。本篇把**梗阻性黄疸的定位思路**展开：先分"痛 vs 不痛"，再按梗阻平面（肝内 → 肝门 → 远端胆总管 → 壶腹）分流，避免只会背胰头癌一个答案。
> 源题：[[NBME/nbme-records/NBME 11/NBME11A_S2_breakdown#^Q056]]（老年 + 无痛黄疸 + 体重下降 + 可触及无压痛胆囊 → 胰头癌 pancreatic head cancer）。

## 一句话核心

**无痛性梗阻性黄疸 (painless obstructive jaundice) + 体重下降 + 可触及无压痛胆囊（Courvoisier 征）→ 首先想远端胆道恶性梗阻（胰头癌 pancreatic head adenocarcinoma / 壶腹癌 / 胆管癌）**；有痛（biliary colic）更偏结石。

## 一、第一步分流：痛 vs 不痛

> [!tip] 一眼分流
> **黄疸 + 绞痛 (biliary colic) / 发热 → 良性梗阻（胆总管结石 choledocholithiasis、胆管炎 cholangitis）** 概率高。
> **黄疸 + 无痛 + 体重下降 + 消耗 → 恶性梗阻**（胰头癌 / 胆管癌 cholangiocarcinoma / 壶腹癌 ampullary carcinoma）概率高。

```
梗阻性黄疸 (conjugated hyperbilirubinemia + 直接胆红素↑ + ALP/GGT↑)
        │
   ┌────┴─────────────────┐
  有痛/发热              无痛 + 体重下降
   │                        │
胆总管结石              远端恶性梗阻
胆管炎 (Charcot 三联)   胰头癌 / 胆管癌 / 壶腹癌
急性胆囊炎               └─ Courvoisier 征常阳性
```

> [!warning] Courvoisier 征的逻辑
> **可触及、无压痛的肿大胆囊 + 黄疸 = Courvoisier 征 → 提示恶性远端梗阻，而非结石。**
> 机制：结石性梗阻多有**长期慢性胆囊炎 → 胆囊壁纤维化 → 无法扩张**；恶性梗阻是**新发进行性远端梗阻 → 胆汁淤积于既往正常胆囊 → 无痛性膨大**。所以"胆囊能被撑大"本身反而指向恶性。

## 二、第二步：按梗阻平面定位

> [!info] 梗阻平面决定影像与哪些结构受累
> 关键推理：**看 conjugated 胆红素 + ALP/GGT 升高**先确认是梗阻（肝后性），再按平面找病因。

| 梗阻平面 | 代表病因 | 典型线索 |
|---|---|---|
| 肝内胆管 | [[完整笔记/专题笔记/GI/GI_原发性硬化性胆管炎PSC]]、转移瘤 | ALP↑ 为主、胆红素可不高 |
| 肝门（近端胆管）| 肝门胆管癌 Klatskin tumor | 双侧肝内胆管扩张、胆总管不扩张 |
| 远端胆总管 | 胰头癌、远端胆管癌、胆总管结石 | 胆总管 + 肝内胆管均扩张 |
| 壶腹 (ampulla, 十二指肠乳头) | 壶腹癌 ampullary carcinoma | 胆管 + 胰管"双管征 double duct sign"、可有黑便/贫血 |

## 三、恶性远端梗阻三巨头鉴别

| 诊断 | 更典型特征 | 关键区分点 |
|---|---|---|
| 胰头癌 (pancreatic head adenocarcinoma) | 无痛黄疸 + 体重下降 + Courvoisier；吸烟/慢性胰腺炎史 | 胰头肿块、双管征、可伴 Trousseau 游走性血栓性静脉炎 |
| 胆管癌 (cholangiocarcinoma) | PSC（原发性硬化性胆管炎）/ 肝吸虫 / 胆管囊肿背景 | 胆管壁增厚狭窄、CA19-9（糖类抗原 19-9）↑ |
| 壶腹癌 (ampullary carcinoma) | 黄疸可**波动**、隐性消化道出血 | 内镜可直视壶腹、双管征 |

> [!tip] 检查路径记忆
> **无痛梗阻性黄疸 → 初查腹部超声（US, ultrasound）看胆管扩张与结石 → CT（computed tomography, 胰腺方案）定肿块与分期 → 需要时 MRCP（磁共振胰胆管成像 magnetic resonance cholangiopancreatography）/ ERCP（内镜逆行胰胆管造影 endoscopic retrograde cholangiopancreatography, 同时可减黄置管）。** 结石可 ERCP 取石；恶性看可切除性（胰头癌 → Whipple 胰十二指肠切除术）。

## 四、易混陷阱（普适）

> [!warning] 别把"黄疸"一律当肝炎
> - **肝细胞性黄疸（肝炎 hepatitis）**：AST/ALT 显著↑、常有恶心呕吐/RUQ 压痛、conjugated 与 unconjugated 混合升高 → 不是本篇的"梗阻"画面。
> - **溶血 / Gilbert**：**unconjugated（间接）胆红素↑为主、ALP 正常、尿胆红素阴性** → 不是梗阻，见 [[完整笔记/专题笔记/GI/GI_遗传性高胆红素血症分流]]。
> - 药物性胆汁淤积需**强药物时间关联**，本题给 atorvastatin/aspirin 属干扰项，机械性恶性梗阻解释更顺。

## 五、复习题

> [!question]- Q1. 黄疸患者查体触及无压痛肿大胆囊，这一体征叫什么？更提示良性还是恶性梗阻？为什么结石反而摸不到大胆囊？
> **Courvoisier 征**；更提示**恶性远端胆道梗阻**（胰头癌/胆管癌/壶腹癌）。结石性梗阻多伴长期慢性胆囊炎 → 胆囊壁纤维化增厚 → 无法扩张膨大；恶性是新发进行性远端梗阻，胆汁淤积于既往正常胆囊 → 无痛性撑大。

> [!question]- Q2. "双管征 (double duct sign)"指什么？提示梗阻在哪个平面？
> 影像上**胆总管 (CBD) 与胰管 (pancreatic duct) 同时扩张**；提示**远端胆总管 / 壶腹平面梗阻**，常见于胰头癌、壶腹癌。

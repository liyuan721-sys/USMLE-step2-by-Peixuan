# GI_吞咽痛 odynophagia 与感染性食管炎 infectious esophagitis 鉴别

> [!tip] 一句话锁定
> **吞咽痛 (odynophagia, 吞咽时疼痛)** ≠ 单纯吞咽困难，它提示**食管黏膜病变**。若背景是**免疫抑制（HIV / 移植 / 化疗 / 长期激素）**，先把 **感染性食管炎 infectious esophagitis** 拉到最前面，最常见三病原是 **念珠菌 Candida、巨细胞病毒 CMV、单纯疱疹病毒 HSV**。

---

## 一、odynophagia 是定向词，别被"胸骨后胸痛"带偏

NBME 常用的坑：拿 **substernal chest pain（胸骨后痛）** 逼你先做心电图 ECG (electrocardiogram) / 心脏检查。但只要**疼痛与吞咽强相关（吞咽诱发 / 加重）**，优先级就已从心脏转向**食管**。

- **心脏检查正常 + poor dentition（口腔卫生差）+ 免疫抑制背景** → 更指向食管黏膜感染，而非心源性。
- **不明原因淋巴结肿大 + 静脉药瘾 IVDU (intravenous drug use) 史** → 提示**未诊断的 HIV（人类免疫缺陷病毒 human immunodeficiency virus）**，进一步支持机会性食管感染。

> [!tip] 首选下一步：EGD（上消化道内镜 esophagogastroduodenoscopy）
> 免疫抑制 + odynophagia，**EGD** 既能直视病灶形态，又能取样活检 / 培养——是最直接的确诊步骤。
> 例外：**HIV / 鹅口疮 (oral thrush) 背景 + 典型表现**可先**经验性氟康唑 fluconazole**（念珠菌最常见），无效再 EGD。

---

## 二、三大病原：内镜形态 = 钥匙

表格前空行。

| 病原 | 内镜 / 溃疡特征 | 伴随线索 | 治疗 |
|---|---|---|---|
| **念珠菌 Candida**（最常见）⭐ | 白色**豆渣样 / 假膜**斑块 | 口腔鹅口疮 oral thrush | 氟康唑 fluconazole |
| **巨细胞病毒 CMV (cytomegalovirus)** | **单个、大而线状 / 深**溃疡 | CD4 常 <50 | 更昔洛韦 ganciclovir |
| **单纯疱疹病毒 HSV (herpes simplex virus)** | **多发、小而浅**溃疡（火山口 / punched-out） | 可见口唇疱疹 | 阿昔洛韦 acyclovir |

表格后空行。

> [!info] 记忆钩
> - **Candida = 白色斑块**（豆渣）；
> - **CMV = 单个大溃疡**（CMV 字母少 = 溃疡少而大）；
> - **HSV = 多个小溃疡**（HSV 联想 herpes 成簇 = 多而小）。

## 三、鉴别：不是所有 odynophagia 都是感染

> [!warning] 药物性食管炎 pill esophagitis 别漏
> **四环素 tetracycline / 双膦酸盐 bisphosphonate / 铁剂 / 钾剂 / 非甾体抗炎药 NSAIDs (nonsteroidal anti-inflammatory drugs)** 干服或卧位服 → 局部黏膜灼伤 → odynophagia。
> 处理：**随大量水服 + 服后直立 30–60 分钟**。stem 给"刚开始某种药 + 睡前干吞"即为此。

- **反流性食管炎 reflux esophagitis / GERD（胃食管反流病）**：更多烧灼感、平卧加重，痛不一定与吞咽严格锁定。
- **心源性胸痛**：劳力诱发、无吞咽相关 → ECG ± 心脏检查（本篇陷阱选项）。

> [!question]- 静脉药瘾 + 不明淋巴结肿大 + 吞咽时胸骨后痛，最合适的下一步？为什么不是 ECG？
> **EGD**。odynophagia + 免疫抑制背景（疑未诊断 HIV）指向感染性食管炎，需直视 + 活检 / 培养确诊。心脏检查正常且疼痛与吞咽锁定，故 ECG 非优先。

> [!question]- 内镜见食管单个大而深的线状溃疡，最可能病原 + 用药？
> **巨细胞病毒 CMV** → 更昔洛韦 ganciclovir。（多发小溃疡想 HSV → 阿昔洛韦；白色豆渣斑块想 Candida → 氟康唑。）

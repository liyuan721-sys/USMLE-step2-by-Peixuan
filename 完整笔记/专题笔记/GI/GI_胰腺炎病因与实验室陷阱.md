# GI_胰腺炎病因与实验室陷阱（acute pancreatitis 病因分层 / lipase vs amylase / 重症判读）

> [!info] 这张的定位
> [[完整笔记/专题笔记/GI/GI_HighYield速查总览]] §10 已有急性胰腺炎的诊断标准、酶陷阱 callout、TG（甘油三酯 triglyceride）阶梯与处理。本篇把**两个最爱考的点单独钉牢**：①病因分层（谁最常见、谁靠药物/机制识别）②实验室陷阱（lipase 更特异；重症看 BUN/Cr/钙，不是靠淀粉酶高低）。避免把"酶越高病越重"这个错误反射带进考场。
> 源题：[[NBME/nbme-records/NBME 11/NBME11A_S1_breakdown#^Q008]]（钝性上腹外伤 → 外伤性胰腺炎 traumatic pancreatitis；淀粉酶↑但脂肪酶落在正常范围的内部不一致须保留）。

## 一句话核心

**急性胰腺炎 (acute pancreatitis) 诊断 = 3 取 2（典型上腹痛 / 脂肪酶 lipase >3 倍上限 / 影像证据）；lipase 比淀粉酶 amylase 更特异、升得更久 → 首选；重症与预后靠 BUN/肌酐 (creatinine)、钙 (calcium)、血细胞比容 (hematocrit) 等系统指标，不看酶的绝对值高低。**

## 一、病因分层（谁最常见 / 谁靠机制识别）

> [!tip] 记忆锚 "I GET SMASHED"
> 最常考前两位：**胆石 (gallstones, MCC 最常见病因) 与 酒精 (alcohol)**。其余靠 stem 特征识别。

| 病因 | 识别线索 |
|---|---|
| **胆石 gallstones**（最常见 MCC）| 女性/肥胖/结石史；**ALT >150 强烈提示胆源性** |
| **酒精 alcohol**（第二常见）| 长期饮酒史、反复发作 |
| 高甘油三酯 (hypertriglyceridemia) | **TG >1000 mg/dL**；淀粉酶可假性正常（脂血干扰）|
| 高钙血症 (hypercalcemia) | 甲旁亢 / 恶性肿瘤背景 |
| **外伤 trauma** | **钝性上腹外伤**（胰腺被挤向脊柱），儿童"自行车把手"、Q008 "被压腹部" |
| ERCP（内镜逆行胰胆管造影 endoscopic retrograde cholangiopancreatography）术后 | 操作后数小时腹痛 + 酶↑ |
| 药物 (drugs) | 硫唑嘌呤 azathioprine、去羟肌苷 didanosine、丙戊酸 valproate、GLP-1 类等 |
| 自身免疫 (autoimmune) | IgG4↑、类固醇反应 |
| 蝎毒 / 感染（腮腺炎 mumps 等）| 少见、按 stem 背景 |

## 二、实验室陷阱（考试核心）⭐

> [!warning] 陷阱一：别用淀粉酶高低下结论
> **淀粉酶 (amylase) 不特异**——很多非胰腺情况都能升高，正常也不能排除胰腺炎：
> - 唾液腺源：腮腺炎 (mumps / parotitis 流行性腮腺炎)、酒精、头颈放疗
> - 腹腔急症：肠穿孔、肠缺血、异位妊娠破裂
> - 清除↓：肾衰 (renal failure)、糖尿病酮症酸中毒 (DKA, diabetic ketoacidosis)
> - **巨淀粉酶血症 (macroamylasemia)**：淀粉酶与免疫球蛋白结合无法经肾清除 → 假性持续升高、**无症状**（良性，勿误当胰腺炎）
> ∴ **脂肪酶 (lipase) 对胰腺更特异、升得更久 → 首选**。

> [!warning] 陷阱二：酶正常/不高 ≠ 排除
> 脂肪酶也有假阴性：**早期就诊、外伤性胰腺炎、高 TG 干扰**时酶可不典型。Q008 即"淀粉酶↑但脂肪酶落在正常范围"——**别因脂肪酶不高就排除**，结合外伤机制 + 临床 + 影像判断。

> [!danger] 陷阱三：酶的高低 ≠ 病情严重度
> **酶升高的**倍数**与预后无关。** 判重症与死亡风险看**系统性指标**：
> - **BUN（血尿素氮 blood urea nitrogen）持续升高 / 入院 BUN 高 → 死亡率最相关**（"BUN 看命"）
> - **肌酐 (creatinine) ↑、血细胞比容 (hematocrit) ↑（血液浓缩）**
> - **低钙 (hypocalcemia)**（脂肪皂化）、乳酸酸中毒
> 评分工具：Ranson 标准、BISAP、APACHE II。**看到"淀粉酶 2000"别当重症依据。**

## 三、重症床旁提示 + 处理边界（速览）

> [!info] 更深处理见速查总览
> - 出血性体征：**Cullen（脐周瘀斑）/ Grey Turner（侧腹瘀斑）** → 提示坏死出血性胰腺炎。
> - 处理基石：**积极 IV 补液 + 镇痛 + 早期肠内营养 + NPO 视情**；胆源性 → **本次住院期内尽早腹腔镜胆囊切除 (cholecystectomy)**；坏死 >30% + 感染 → 抗生素 + 干预。
> - 完整阶梯（TG 性用胰岛素/血浆置换、胰周积液时间线）见 [[完整笔记/专题笔记/GI/GI_HighYield速查总览]] §10。

## 四、复习题

> [!question]- Q1. 急性胰腺炎诊断标准？为什么脂肪酶 (lipase) 优于淀粉酶 (amylase)？胆源性的 ALT 阈值？
> **3 取 2**：典型上腹痛 / 脂肪酶（或淀粉酶）>3 倍上限 / 影像证据。脂肪酶对胰腺**更特异、升得更久**（淀粉酶受唾液腺/肾衰/巨淀粉酶血症等干扰）。**ALT >150 强烈提示胆源性。**

> [!question]- Q2. 哪些指标判断急性胰腺炎的严重度与预后？酶的绝对值可以吗？
> **BUN（尤其持续升高）、肌酐、血细胞比容（血液浓缩）、低钙、乳酸酸中毒**，以及 Ranson/BISAP/APACHE II 评分。**酶的绝对值高低与严重度无关**，不能用来判重症。

> [!question]- Q3. 钝性上腹外伤后上腹痛 + 淀粉酶↑但脂肪酶正常，能否排除胰腺炎？
> **不能。** 外伤性胰腺炎 (traumatic pancreatitis) 与早期就诊时酶可不典型；应结合外伤机制（胰腺被挤向脊柱）+ 临床 + CT 影像判断，必要时复查酶/影像，而非因单一正常值排除。

## 五、慢性胰腺炎 (chronic pancreatitis)（Q4362 锚定）

> [!info] 与本篇急性部分互补
> 上面 §一~§三 讲急性胰腺炎；慢性胰腺炎是另一套考点，这里单独钉牢。

慢性胰腺炎 = 进行性炎症致不可逆外分泌 + 内分泌功能损害。最常见病因：慢性酒精 (chronic alcohol)。

| 维度 | 慢性胰腺炎要点 |
|---|---|
| 病因 | 酒精（最常见）；儿童想囊性纤维化 (cystic fibrosis, CF)；导管梗阻（结石/恶性）；自身免疫 |
| 疼痛 | 慢性上腹痛、可放射背、⭐前倾/坐起缓解 (leaning forward)；间歇可数月至一年 |
| 外分泌不全 | 脂肪泻 (steatorrhea)、体重下降、脂溶性维生素缺乏 |
| 内分泌不全 | 糖尿病 (diabetes mellitus) |
| ⭐实验室陷阱 | 淀粉酶/脂肪酶可正常（"烧尽的胰腺" burned-out）→ 酶正常不能排除 CP，不敏感 |
| 确诊 | 腹部 CT 或平片见胰腺钙化 (calcifications)、导管扩张、胰腺增大；CT 还可排胰腺癌/假性囊肿 |
| 治疗 | 止痛、戒酒戒烟、少食多餐、胰酶替代 (pancreatic enzyme supplements) |

> [!warning] 干扰项
> CA 19-9 灵敏/特异性有限，不用于诊断疑似胰腺癌，腹部 CT 才是第一步；血清脂肪酶不敏感（可正常）不能确诊 CP；上消化道内镜查溃疡/胃炎，CP 常无需内镜。

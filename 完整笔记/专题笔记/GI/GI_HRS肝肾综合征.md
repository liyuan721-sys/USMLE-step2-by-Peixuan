### 如何读这张流程图（HRS 是"排除性诊断"，3 道关逐个排）⭐

> [!info] 核心逻辑：HRS 没有"确诊检查"，只能把别的原因都排掉，剩下的才是它
> 肝硬化病人突然 AKI（Cr 升高），这张图就是"逐个排除"的顺序。**每一关都在问"是不是别的更明确的病因"，全排掉才落到 HRS**。

> [!tip]- 第 1 关：有尿路梗阻吗？(urinary obstruction) → 做 renal US
> - **有梗阻**（积水 / 结石 / 肿瘤压输尿管）→ 去处理梗阻，**不是 HRS**
> - **US 正常**（无梗阻，也顺带排除慢性肾病、肾缩小）→ 进第 2 关
> - 👉 这就是为什么 Q2219 给你"renal US 正常"——是在**替你过掉第 1 关**

> [!tip]- 第 2 关：尿常规 UA 长啥样？→ 收 UA
> - **UA 异常**（有蛋白 / 血 / 管型）= 肾"真的病了"，提示特异病因：
>   - muddy brown casts → **ATN**（肾小管坏死）
>   - WBC casts + 嗜酸细胞 → **AIN**（间质性肾炎）
>   - → 去治这些，**不是 HRS**
> - **Bland UA（干净尿：无蛋白、无血、无管型）** → 肾结构没坏 → 进第 3 关
> - 👉 "bland UA" 是 HRS 的入场券——因为 HRS 的肾**结构是好的**（见 [[#一、本质：一句话讲透]]），尿当然干净

> [!tip]- 第 3 关：高度疑 HRS → 同时做两件事
> ① **测尿钠 (urine sodium)** ② **试验性扩容**（IV albumin）
> - **扩容后改善** → 其实是**真的血容量不足**（prerenal，如利尿过度 / 出血）→ 继续补液，**不是 HRS**
> - **扩容后毫无改善** → **确认 HRS** → 内脏缩血管剂 + 求肝移植

> [!warning] 第 3 关两个"为什么"（Q2219 的真考点）
> **为什么测尿钠**：HRS 的肾极致保钠 → **尿钠 < 10**（比经典 prerenal < 20 还低；ATN 因小管坏掉保不住钠 > 40）。一个便宜尿检就定方向。完整 cutoff 见 [[#4.3 HRS vs prerenal vs ATN — 鉴别金表 ★★★]]。
> **为什么要扩容试验**：HRS 和"真缺水"早期长得一模一样（都肝硬化 + 少尿 + 尿钠低），**唯一区分 = 补液后 Cr 降不降**——HRS 补再多也不降（问题不是缺水，是血被内脏血管劫走了）。这也是为什么必须先停 Q2219 那病人的利尿剂（furosemide + spironolactone）排除"利尿过头"。

> [!danger] 一句话锁住整张图
> **肝硬化 AKI + 肾 US 正常（排梗阻）+ bland UA（排肾实质病）→ 下一步测尿钠 + 试扩容 → 扩容无效 = HRS**

## 一、本质：一句话讲透

> [!danger] HRS 的核心
> **肾脏本身是好的，是被"饿死"的。**
> 不是肾小球坏了、不是肾小管坏了、不是免疫攻击 — 是**血流被劫持**导致的**功能性肾衰**。
> 把肝移植了，肾就活了 → 证明肾本身没病。

### 水管比喻（直觉理解）

> [!tip] 客厅水龙头开太大了
> 想象家里的水管系统：
> - **水厂** = 心脏（出水量固定）
> - **主管道** = 大动脉
> - **客厅水龙头** = 肠道腹腔血管 (splanchnic)
> - **卧室水龙头** = 肾脏血管
>
> 肝硬化时发生了什么？
> → **客厅水龙头莫名其妙开到最大**（splanchnic vasodilation）
> → 水都跑去客厅了，**卧室水龙头出水变小**（renal hypoperfusion）
> → 卧室住户（肾）以为"全家缺水"，拼命**关小自己的水龙头省水**（renal vasoconstriction + Na 潴留）
> → 卧室越关越紧，自己先渴死（AKI）
>
> **悖论：明明全身水多到溢出来（腹水），肾却以为身体脱水。**

### ASCII 复述

```
第 1 步：肝硬化 + Portal HTN（门脉高压）
   │
   │  门脉压↑ → 肠血管内皮被"撑得"分泌大量 NO
   ▼
第 2 步：内脏血管扩张（Splanchnic vasodilation）
   │
   │  NO（一氧化氮）= 强力血管扩张剂
   │  腹腔内脏血管全部松弛 → 血液"积压"在腹腔池
   ▼
第 3 步：有效动脉血容量 ↓（↓ EABV）⚠️ 关键概念
   │
   │  EABV = Effective Arterial Blood Volume
   │  = 真正"灌注组织"的血量
   │
   │  全身总血量没少（甚至更多，因腹水）
   │  但血都"窝在"腹腔扩张血管里 → 真正循环的少
   │  身体感知 = "我缺血！"
   ▼
第 4 步：神经体液激活（代偿过头）
   │
   │  身体看到 EABV ↓，启动三大保水系统：
   │  • SNS（交感神经）↑ → 血管收缩、心率快
   │  • RAAS（肾素-血管紧张素-醛固酮）↑ → Na/水潴留
   │  • ADH（抗利尿激素）↑ → 自由水潴留
   ▼
第 5 步：肾血管收缩 + 钠水潴留
   │
   │  RAAS 不分敌我，肾血管首当其冲被收缩
   │  → 肾灌注 ↓↓ → GFR ↓ → Cr ↑
   │
   │  Na 拼命重吸收 → 尿 Na 极低（<10）
   │  水也潴留 → 加重腹水 + 稀释性低钠
   ▼
第 6 步：触发事件（precipitant）一推就崩
   │
   │  • SBP（自发性细菌性腹膜炎）★★★ 最经典
   │  • GI 出血（食管静脉曲张）
   │  • 大量放腹水不补 albumin
   │  • 过度利尿
   ▼
第 7 步：HRS-AKI（旧名 Type 1）
        急性少尿型肾衰，Cr 数天内翻倍
        不治疗中位生存 2 周
```

> [!warning] EABV ≠ 总血量
> **总血量正常或↑（腹水 + 水肿）** vs **EABV ↓**（血都窝在腹腔池）
> 这就是为什么 HRS 病人**看起来肿胖**却**肾以为脱水**的悖论根源

## 三、分型：HRS-AKI vs HRS-NAKI（2015 新分类，CK 必背）

| 旧名 | 新名 | 病程 | Cr 变化 | 预后（不治疗） |
|---|---|---|---|---|
| **Type 1** | **HRS-AKI** | 急性（<2 周） | 数天内 Cr 翻倍或 ↑ ≥0.3 mg/dL | 中位生存 2 周 |
| **Type 2** | **HRS-NAKI**（CKD-like）| 慢性、隐匿 | 缓慢 ↑ | 中位生存 6 个月，伴顽固性腹水 |

> [!danger] 出题模式对比
> - **Type 1 (HRS-AKI)**：几乎总有**触发事件**（最常 SBP），急性发病
> - **Type 2 (HRS-NAKI)**：无明显诱因，跟着难治性腹水一起出现

## 四、诊断 — 排除法（5 项 AKI 鉴别 SOP）

### 4.1 必须满足（简化版 ICA-AKI 标准）

> [!info] 诊断 6 条
> 1. **肝硬化 + 腹水**
> 2. **AKI**（Cr ↑ ≥0.3 mg/dL 或翻倍）
> 3. **停利尿剂 + albumin 扩容 48 h 无改善** ★ 金标准
> 4. **排除休克**
> 5. **排除肾毒药**（NSAIDs / 造影剂 / 氨基糖苷）
> 6. **无肾实质病** → 无蛋白尿（<500 mg/d）、无血尿、肾脏 B 超正常

### 4.2 肝硬化 + Cr 升高 强制 5 项 SOP ⭐

```
肝硬化 + Cr 上升
  ├─ ① Prerenal volume depletion（GI 出血 / 利尿过度 / 第三间隙）
  │     → 扩容 (NS + albumin) 改善 → 是这个
  ├─ ② ATN（缺血 / 造影剂 / aminoglycoside / cephalosporin 数天后）
  │     → UA: muddy brown casts + ↑ FENa
  ├─ ③ AIN（药物 — 抗生素 / NSAIDs / PPI）
  │     → UA: WBC casts + eosinophiluria + 皮疹 / 嗜酸 / 发热
  ├─ ④ Glomerular（HCV-MPGN / IgA / HBV-MN）
  │     → UA: RBC casts + proteinuria
  └─ ⑤ HRS ⭐（排除诊断）
        → 扩容 NS + albumin 无改善 + UA bland + 肾正常大小
        → Type 1: 急性 / Type 2: 慢性 + 难治腹水
```

### 4.3 HRS vs prerenal vs ATN — 鉴别金表 ★★★

| | **HRS** | **Prerenal AKI** | **ATN** |
|---|---|---|---|
| **尿 Na** | < 10 | < 20 | > 40 |
| **FENa** | < 1% | < 1% | > 2% |
| **尿沉渣** | 干净 (bland) | 干净 | **muddy brown casts** |
| **Albumin 扩容反应** | **无反应 ✗** | **有反应 ✓** | 无反应 |

> [!danger] 关键鉴别点
> **HRS 和 prerenal 都是尿 Na 低、FENa 低**（都是"肾灌注不足"）
> **唯一区分**：补液（albumin 1 g/kg × 2 天）后 Cr 是否下降
> - **HRS 不降** → 因为不是真的脱水，是血流"被劫持"了
> - **Prerenal 下降** → 真容量不足，补回去就好

> [!warning] HRS = "假 prerenal"
> 看起来像 prerenal（bland UA + 低 FENa），但扩容不应
> 不读"扩容是否有效"就分不清 — Q4752 元错误就在这

## 五、治疗阶梯（6 步完整版）

```
HRS 诊断成立
    │
    ▼
① 去除诱因（SBP / GI 出血 / 停利尿）+ 停 nephrotoxins
    │
    ▼
② Albumin 1 g/kg × 2 天扩容（同时是 prerenal 排除步骤）
    │
    ▼
③ 血管收缩药 + Albumin：
   • Terlipressin（首选 ★ 2022 FDA 批）
   • Midodrine + Octreotide + Albumin（terlipressin 不可及时）
   • Norepinephrine + Albumin（ICU 情境）
    │
    ▼ 药物无反应 + 无 TIPS 禁忌 + 等移植期
④ TIPS（HRS-AKI 二线 / HRS-NAKI 更优）
    │
    ▼ TIPS 失败 / 不能 TIPS
⑤ RRT 桥接（透析维持，等肝移植）
    │
    ▼
⑥ ★ 肝移植 = 唯一根治 ★
```

### 5.1 一线药物 — Terlipressin + Albumin

> [!success] 机制
> Terlipressin = V1 受体激动 → **splanchnic 动脉收缩**
> → 把腹腔池里"窝着"的血**赶回循环**
> → EABV ↑ → RAAS/SNS 退场 → 肾血管放松 → GFR 回升

| 药 | 机制 | 备注 |
|---|---|---|
| **Terlipressin** | V1 选择性，长效合成血管加压素 | **2022 FDA 批，HRS-AKI 首选** ★ |
| **Midodrine + Octreotide + Albumin (MOA)** | midodrine α1 升压 + octreotide 抑制扩血管激素 | terlipressin 不可及时 |
| **Norepinephrine + Albumin** | α1 升压 | ICU、需中心静脉通路 |

> [!warning] Terlipressin ≠ Desmopressin
> 都是 vasopressin 类似物，但完全不同：
> - **Terlipressin** = V**1** 选择 → **血管收缩** → HRS / 静脉曲张出血
> - **Desmopressin (DDAVP)** = V**2** 选择 → **保水 + 释放凝血因子** → DI / 遗尿 / vWD / 血友病 A / 尿毒症出血
> 详见 [[完整笔记/专题笔记/GI/GI_门脉高压治疗#一、核心公式]]

### 5.2 RRT（血液透析）在 HRS 的角色

> [!warning] 透析不治 HRS — 是 holding pattern
> 透析本身**不解决 splanchnic vasodilation 问题**，只作为**移植前的桥接维持**（防 K↑、酸中毒、容量过负荷）
> **不要把"透析"当成 HRS 的治疗答案**

### 5.3 肝移植 — 唯一根治

> [!success] 为什么移植能根治
> 换新肝 → 门脉高压消失 → splanchnic vasodilation 信号源消失
> → EABV 恢复 → RAAS/SNS 退场 → 肾血管放松 → **肾功能 1-3 个月内逐渐恢复**
> 这就是为什么"HRS 的肾移植给别人是好的" — **肾本身没病**

## 六、TIPS 在 HRS 中的角色

> [!info] 机制：从 R 端釜底抽薪
> HRS 根源 = 门脉高压 → splanchnic 扩张 → EABV ↓ → 肾血管收缩
> TIPS 把门脉血直接引到肝静脉 → **门脉压 ↓** → splanchnic 驱动消失 → 肾灌注恢复
>
> 证据：TIPS 后 HRS-AKI 约 40-60% Cr 改善；HRS-NAKI 效果更好

### 6.1 为什么 TIPS 不是一线（3 原因）

**① 一线药物 + albumin 已够强** — Terlipressin + albumin 对 60-80% HRS-AKI 有反应

**② HRS 病人多有 TIPS 禁忌** — 终末期肝病几乎都踩这些禁忌：

| TIPS 禁忌 | HRS 人群常见度 |
|---|---|
| **重度肝性脑病 (HE)** | 极常见（TIPS 加重 HE）★ |
| **Total Bilirubin > 5** | 常见 |
| **INR > 2** | 常见 |
| **MELD > 18** | 几乎都符合 |
| **重度心衰 / 肝硬化心肌病** | 常见 |
| **重度肺动脉高压** | — |
| **败血症未控制** | SBP 诱发 HRS 常见 |

**③ TIPS 后 HE 大爆发** — 氨绕过肝直接进体循环 → HE 风险 +20-40%，HRS 病人脑功能本已边缘

### 6.2 HRS-AKI vs HRS-NAKI 的 TIPS 角色

| 类型 | TIPS 角色 | 适应症 |
|---|---|---|
| **HRS-AKI（Type 1）** | **二线，证据较弱** | 药物无反应 + 不能马上移植 + 无禁忌 → 桥接 |
| **HRS-NAKI（Type 2）** | **更可靠**，作用持久 | 顽固腹水 + Type 2 + 无禁忌（等移植期） |

> [!info] Type 2 更适合 TIPS
> Type 2 慢性病程，时间够走 TIPS 评估流程；常以**顽固腹水**为主诉，**TIPS 同时解决腹水 + HRS** 两个问题

> [!danger] CK 出题概率
> "肝硬化 + HRS-AKI" next-step **优先**：
> 1. Terlipressin + albumin ✅
> 2. Midodrine + octreotide + albumin ✅
> 3. Liver transplant evaluation ✅
> 4. TIPS — 只有题干明确"药物失败 + 无 HE + 等移植"才选
>
> TIPS 在 HRS 题里**最常见的正确场景** = HRS-NAKI Type 2 + 顽固腹水

## 七、SBP → HRS 预防链（高频考点）

```
SBP（腹水 PMN ≥ 250）
    │
    ▼
经验抗生素 Cefotaxime / Ceftriaxone IV
    +
★ IV Albumin ★
    Day 1: 1.5 g/kg
    Day 3: 1.0 g/kg
    │
    ▼
预防 HRS 发生 + 降低死亡率（已成共识）
```

> [!danger] SBP 必"双打" — Albumin 不能漏 ★★
> CK 经典 stem：肝硬化 + 腹水 + 发热腹痛 → 腹穿 PMN 320 → next step?
> 错答："Cefotaxime alone" ✗
> **正确**：**Cefotaxime + Albumin** ✓

> [!warning] Albumin 触发指标（不是所有 SBP 都给）
> 三选一：**Cr > 1** / **BUN > 30** / **Bilirubin > 4** → 给 albumin
> 这三个指标 = "有发生 HRS 的高危特征" → albumin 预防 HRS
> 参考 [[完整笔记/Peixuan分科笔记/消化腺]] line 1258

## 八、大量放腹水（LVP）不补 albumin → HRS 诱因

> [!danger] Paracentesis-Induced Circulatory Dysfunction (PICD)
> 大量放腹水（>5 L）瞬间释放腹腔压力 → splanchnic vasodilation 进一步加剧
> → EABV 急性下降 → 触发 HRS

**预防 SOP**：放每 **1 L 腹水 → 补 6-8 g albumin** ★

> [!question]- 经典 stem
> 肝硬化大量腹水穿刺 8 L 后第二天 Cr 1.0 → 2.5。机制？
> A: **PICD → HRS-AKI**
> 预防 = 当时同步补 albumin（漏了 → 出事）

## 九、3 个"反直觉"必背点

> [!danger] 记忆锚
> 1. **全身水超载，肾却以为脱水**
>    → 因为水都在"错误的地方"（腹腔池子里），EABV ↓ 才是肾感知的真相
> 2. **HRS 的肾移植给别人是好的**
>    → 肾本身没病，是肝硬化"劫持血流"导致
> 3. **不能用利尿剂！**（与心衰相反 ★）
>    → 心衰 = 容量过负荷 → 利尿
>    → HRS = 看起来过负荷 + 实际 EABV ↓ → **利尿越多 EABV 越低 → HRS 越重**
>    → HRS 诊断流程第一步就是**停利尿剂**

## 十、CK 经典考法

> [!question]- Q1: 经典 HRS-AKI 诊断题
> 60 yo M 肝硬化 + 腹水。3 天前 SBP 入院已打 cefotaxime。今天 Cr 1.0 → 2.8，尿量 ↓。
> 尿检：蛋白阴性、无血尿、**尿 Na = 8**。
> Albumin 1 g/kg × 2 天后 Cr 仍 2.7。
> 机制 / 诊断 / 下一步 / 根治？
>
> A: 机制 = **Splanchnic vasodilation → EABV ↓ → renal vasoconstriction**
> 诊断 = **HRS-AKI（Type 1）**
> 下一步 = **Terlipressin + albumin** 或 Midodrine + Octreotide + Albumin
> 根治 = **Liver transplant**

> [!question]- Q2: HRS vs Prerenal 鉴别
> 肝硬化病人 Cr 1.0 → 2.8，尿 Na 8，FENa 0.5%。给 NS + albumin 1 g/kg × 2 天后 Cr 降到 1.2。
> 诊断？
>
> A: **Prerenal AKI**（不是 HRS — 因为扩容有反应）

> [!question]- Q3: HRS vs ATN 鉴别
> 肝硬化 + 入院第 5 天打 gentamicin 后 Cr 升高，UA 见 **muddy brown casts**，FENa 3%。
> 诊断？
>
> A: **ATN**（不是 HRS — 因为 UA 有 casts + FENa > 2）

> [!question]- Q4: SBP next step 经典陷阱
> 肝硬化 + 腹水 + 发热腹痛，腹穿 PMN 320，Cr 1.5，BUN 35。Next step?
>
> A: **Cefotaxime + IV Albumin（1.5 g/kg Day 1 + 1 g/kg Day 3）**
> Albumin 不能漏 ★ — Cr/BUN/Bili 任一阳性即触发

> [!question]- Q5: 大量放腹水后 Cr 飙升
> 肝硬化 + 8 L 腹水穿刺，**未补 albumin**，次日 Cr 1.0 → 2.5。机制 + 预防？
>
> A: 机制 = **PICD → HRS-AKI**
> 预防 = 每放 1 L 同步补 **6-8 g albumin**

> [!question]- Q6: TIPS 在 HRS 的角色
> 65 yo M HRS-AKI, terlipressin + albumin 5 天 Cr 未改善，无 HE，bilirubin 3.5，MELD 16，等肝移植。Next step?
>
> A: **TIPS 桥接**（药物失败 + 无 HE + bilirubin < 5 + MELD < 18 + 等移植 → 罕见齐全场景）
> 若 HE 或 bili > 5 → **RRT 桥接 + 加速移植**

> [!question]- Q7: HRS-NAKI Type 2 + 顽固腹水
> 70 yo F 肝硬化，6 个月 Cr 慢慢从 1.0 升到 2.2，顽固腹水反复抽液。Next step?
>
> A: **TIPS** — Type 2 比 Type 1 更适合 TIPS，且**同时解决腹水和 HRS**

> [!question]- Q8: 反直觉 — HRS 病人能用利尿剂吗？
> A: **不能 ★**。HRS 诊断流程第一步就是停利尿剂 — 利尿越多 EABV 越低 → HRS 越重（与心衰相反）

## 十一、记忆口诀

> [!tip] 三步链
> **门脉高压 → splanchnic vasodilation → EABV ↓ → renal vasoconstriction**

> [!tip] HRS 锁定指纹
> **肝硬化 + 扩容无效 AKI + bland UA = HRS** ★★★

> [!tip] HRS = 假 prerenal
> 看起来像 prerenal（bland UA + 低 FENa），但扩容不应 → 关键看"扩容反应"

> [!tip] 治疗梯次记忆
> "**T → M → N → TIPS → RRT → 移植**"
> Terlipressin → Midodrine+Oct → Norepinephrine → TIPS → RRT → Liver Transplant

> [!tip] SBP "双打"
> **Cefotaxime + Albumin** — Albumin 在 SBP 是必给的，不是可选

> [!tip] 反直觉三连
> 1. 水超载但肾觉得脱水（EABV vs 总血量）
> 2. 肾本身好的（移植根治）
> 3. 不能利尿（与心衰相反）

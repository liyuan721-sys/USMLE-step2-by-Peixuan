# GI_肛周疾病与肛瘘

> [!success] 一句话总纲
> 肛周病看两件事：**有没有波动性肿块（急性期）** + **有没有持续外口流脓（慢性期）**。
> 波动性触痛肿块 = 肛周脓肿 (perianal abscess) → **立即切开引流 (incision and drainage, I&D)**；
> 慢性外口 + 恶臭分泌物 = 肛瘘 (fistula in ano / anorectal fistula) → **手术（瘘管切开术 fistulotomy）**。
> 两者是**同一个病的两个时相**，只有时相变了、处理就完全不同。

---

## 一、钥匙：肛周脓肿 → 肛瘘的时相关系

> [!danger] 一眼分流（本主题最高频考点）
> **急性期**：数天内剧痛 + 肛周**波动性 (fluctuant)、触痛的肿块**，可伴发热 → **床边/手术室 I&D**，不是抗生素、不是等待。
> **慢性期**：既往一次剧痛后**自行缓解**（= 脓肿自发破溃泄压），此后**持续/间歇恶臭分泌物、瘙痒、排便时痛**，查体见**外口 (external opening) 呈炎性丘疹/脓疱样病灶 + 皮下可扪及索条状硬结**，**直肠指检无肿块** → **择期手术评估 (fistulotomy)**，不是抗生素、不是结肠镜。
> ⭐ 病史里"**痛过一次、自己好了、之后一直流脓**"就是脓肿破溃留下瘘管的因果钥匙。

```
                肛隐窝腺体阻塞 + 感染
                          |
                          v
            肛周脓肿 (perianal abscess)【急性】
              波动性触痛肿块 ± 发热
                          |
        +-----------------+------------------+
        |                                    |
   及时 I&D 引流                     未引流 / 自发破溃
        |                                    |
        v                                    v
   多数愈合                       残留上皮化窦道 = 肛瘘【慢性】
   (约 1/3~1/2 仍会成瘘)            外口流脓 + 索条 + 排便痛
                                             |
                                             v
                                   手术: fistulotomy
                                   (复杂瘘先挂线 seton)
```

> [!info] 为什么脓肿引流后还会成瘘
> 肛腺 (anal gland) 的感染灶在**内口 (internal opening)** 侧持续存在，脓腔虽引流、通道却上皮化成慢性窦道。因此 I&D 后仍有相当比例（教材常引约 1/3–1/2）最终发展为瘘 —— 这不是"引流做错了"，而是自然史。

## 二、肛瘘 (fistula in ano)

**定义**：肛管/直肠与肛周皮肤之间的**上皮化异常通道**（内口在肛管齿状线附近的肛隐窝，外口在肛周皮肤）。

### 2.1 病因（考点顺序）

| 病因 | 线索 |
|---|---|
| **肛周脓肿破溃（最常见）** | 既往一次剧痛后自愈；无 IBD、无全身病史 |
| **克罗恩病 (Crohn disease)**（炎症性肠病 inflammatory bowel disease, IBD 之一） | 慢性腹泻、腹痛、体重下降、口腔溃疡、多发/复杂瘘、直肠病变 |
| **恶性肿瘤**（直肠癌 / 肛管鳞癌 anal squamous cell carcinoma） | 老年、体重下降、硬结不规则肿块、便血 |
| **放射性直肠炎 (radiation proctitis)** | 盆腔放疗史（前列腺癌 / 宫颈癌） |
| **感染**：性病淋巴肉芽肿 (lymphogranuloma venereum, LGV, 沙眼衣原体 *Chlamydia trachomatis* L1-L3)、结核 (tuberculosis)、放线菌病 (actinomycosis) | 受累人群/性行为史、腹股沟淋巴结病、直肠炎 |
| **外伤 / 产伤 / 术后** | 器械操作、分娩撕裂、既往肛门手术史 |

### 2.2 临床表现

- **肛周痛**（尤其**排便时**加重 —— 粪便经过内口刺激瘘管）
- **慢性、间歇性分泌物**：脓性/粪臭，污染内裤；瘙痒 (pruritus ani)
- 外口为**炎性丘疹或脓疱样病灶**，可有肉芽组织
- 触诊沿外口指向肛管方向可扪及**索条状硬结（indurated tract）**
- 反复发作史：肿痛 → 破溃流脓 → 暂时好转 → 再肿（提示引流不畅）

### 2.3 诊断

> [!tip] 诊断以临床为主，别过度检查
> - **视诊 + 触诊**：外口 + 索条 → 基本确诊。
> - **内口定位**：**肛门镜 (anoscopy)** 或**轻柔探针 (gentle probing)**；也可用软式乙状结肠镜 (flexible sigmoidoscopy) 观察直肠黏膜。
> - **麻醉下探查 (examination under anesthesia, EUA)**：术中用软探针**划定瘘管全程**，是治疗的一部分。
> - **影像仅用于复杂病例**：直肠腔内超声 (endoanal ultrasonography / endosonography)、瘘管造影 (fistulogram)、**磁共振成像 (magnetic resonance imaging, MRI)**（复杂/多发/克罗恩相关瘘首选影像）。
> - **常规不做结肠镜** —— 仅在瘘复杂、复发、非典型或**怀疑 IBD** 时用于评估全结肠。

**Goodsall 规则（定位内口的经验法则）**：外口在肛门**前半**（横线以前）→ 瘘管**直行放射状**通向最近的肛隐窝；外口在**后半** → 瘘管**弯曲**走向**后正中线**。

### 2.4 治疗

> [!warning] 肛瘘的答案是"手术"，不是"抗生素观察"
> - **单纯低位瘘**：**瘘管切开术 (fistulotomy)** —— 切开并敞开整条通道使其二期愈合。
> - **复杂瘘**（高位经括约肌型 high transsphincteric、女性前壁瘘、多发瘘管、克罗恩相关、既往失禁风险）：**挂线引流 (seton placement)**、推移瓣 (advancement flap)、**括约肌间瘘管结扎术 (ligation of intersphincteric fistula tract, LIFT)** 等**保括约肌**术式 —— 直接大范围切开会伤及肛门括约肌 → **大便失禁 (fecal incontinence)**。
> - **必须处理干净整条瘘管**：残留瘘管 → 症状持续、反复脓肿、复发。
> - 抗生素**不是**脓肿源性肛瘘的治疗（仅在合并蜂窝织炎、免疫抑制、糖尿病、瓣膜病/心内膜炎高危、全身脓毒表现时辅助使用）。

## 三、克罗恩相关肛瘘：唯一走"药物路线"的例外

> [!danger] 例外路线（别把它套到普通瘘上，也别把普通瘘套进它）
> **克罗恩病肛周瘘 (perianal Crohn fistula)**：
> 1. **抗生素**：甲硝唑 (metronidazole) 和/或 环丙沙星 (ciprofloxacin)
> 2. **抗肿瘤坏死因子 (anti-tumor necrosis factor, anti-TNF)**：**英夫利昔单抗 (infliximab)** —— 促进瘘管闭合的核心药
> 3. ± 免疫调节剂（硫唑嘌呤 azathioprine）；**治疗基础 IBD 才是根本**
> 4. 合并脓肿仍需**先引流**；复杂瘘常放**挂线 (seton)** 维持引流，再上生物制剂
> 5. **避免大范围 fistulotomy**（克罗恩患者愈合差 + 失禁风险高）
>
> **而脓肿源性（隐窝腺源性）瘘保守治疗失败率高 → 必须手术。** 题干若明确"无腹痛、无腹泻、无血便、无既往病史"，就是在**主动排除克罗恩**，把你推向手术选项。

## 四、肛周四病（+ 藏毛窦）一眼分流对比表

| 疾病 | 典型主诉 | 查体 | 诊断 | 一线处理 |
|---|---|---|---|---|
| **肛瘘 (fistula in ano)** | 慢性/间歇**恶臭分泌物** + 瘙痒 + 排便痛；常有"数月前剧痛后自愈"史 | 肛周**外口**（炎性丘疹/脓疱）+ 皮下**索条状硬结**；**无波动性肿块** | 临床；内口用**肛门镜/探针**；复杂才 MRI / 腔内超声 | **手术**：低位 fistulotomy；复杂 → seton / 保括约肌术式 |
| **肛周脓肿 (perianal abscess)** | 数天内**持续剧痛**、坐卧不安，可发热 | 肛周**波动性、触痛、红肿肿块**（深部脓肿可仅直肠指检触及触痛隆起） | 临床；深部/复杂用计算机断层扫描 (computed tomography, CT) 或 MRI | **立即 I&D**；抗生素仅在蜂窝织炎/免疫抑制/糖尿病/瓣膜病/全身中毒时加用 |
| **肛裂 (anal fissure)** | 排便时**撕裂样剧痛**，便后持续数分钟至数小时 + **便纸上鲜红血** | **后正中线 (posterior midline) 纵行裂口**；慢性者有前哨痔 (sentinel pile) + 肥大肛乳头 | 临床（剧痛常无法完成指检/肛门镜，可推迟） | 保守：**坐浴 (sitz bath)** + 高纤维 + 大便软化剂 + 局部**硝酸甘油 (nitroglycerin)** 或 **地尔硫䓬 (diltiazem)** 软膏；顽固 → 肉毒毒素 (botulinum toxin) 注射或**侧位内括约肌切开术 (lateral internal sphincterotomy)** |
| **痔 (hemorrhoids)** | 内痔：**无痛性**鲜红便血 / 脱垂；外痔血栓：突发肛缘**剧痛肿块** | 内痔在**齿状线以上**（内脏神经，不痛）；外痔在齿状线以下（躯体神经，痛），血栓性为紫蓝色触痛结节 | 视诊 + **肛门镜**；便血伴报警征/年龄达标 → 加结肠镜排除肿瘤 | 内痔：纤维 + 坐浴 → **套扎 (rubber band ligation)**；**血栓性外痔 <72 h 剧痛 → 切除血栓/痔切除**，>72 h 保守 |
| **藏毛窦/藏毛病 (pilonidal disease)** | **骶尾部中线**（臀沟上方，**不在肛缘**）疼痛/流脓，青年多毛男性、久坐 | 中线**藏毛的凹窝 (pit)** ± 侧方脓腔 | 临床 | 急性脓肿 → **I&D**；慢性 → 切除窦道 + 脱毛/局部卫生 |

> [!tip] 位置就是答案
> 齿状线 (dentate line) 以上不痛（内脏神经）→ 内痔的"无痛性出血"；以下痛（躯体神经）→ 肛裂、血栓性外痔、脓肿都很痛。
> **病灶不在肛缘、在骶尾中线** → 直接想藏毛窦，不是肛周脓肿。
> **肛裂长在侧方（lateral）而非后正中线** → 提示不典型病因：克罗恩病、结核、HIV、梅毒 (syphilis)、白血病 → 要查病因，不是单纯保守。

## 五、干扰项为什么错（对应 Q16095）

| 干扰项 | 它真正的主场 | 本题为什么不选 |
|---|---|---|
| **肛门直肠测压 (anorectal manometry)** | **慢性便秘 / 大便失禁 (fecal incontinence)** 的神经肌肉与括约肌功能评估；先天性巨结肠 (Hirschsprung disease) 辅助诊断 | 瘘管患者可因瘘口渗漏而"内裤脏"，但**括约肌功能正常**；测压不诊断瘘、也不改变处理 |
| **结肠镜 (colonoscopy)** | 复杂/复发/非典型瘘、**怀疑 IBD**、便血需排除肿瘤 | 本例病史典型（脓肿后遗）、无 IBD 线索；**找内口用肛门镜或软式乙状结肠镜足够**，全结肠镜是过度检查、延误手术 |
| **经验性口服抗生素** | **克罗恩相关肛瘘**（甲硝唑/环丙沙星 + 抗 TNF）；或瘘/脓肿合并蜂窝织炎、免疫抑制、糖尿病、心内膜炎高危 | 隐窝腺源性瘘**保守治疗失败率高**，抗生素不能闭合上皮化通道 → 仍要手术 |
| **大便虫卵与寄生虫检查 (stool ova and parasites, O&P)** | **慢性腹泻**评估（贾第虫 *Giardia intestinalis* / 隐孢子虫 *Cryptosporidium* / 溶组织内阿米巴 *Entamoeba histolytica*） | 本例**无腹泻**；肛周分泌物来自瘘管而非肠道感染 |
| （常见第五类）**局部糖皮质激素/止痒药膏** | 单纯肛门瘙痒 (pruritus ani)、轻度皮炎 | 对已成形的瘘管无效，只掩盖症状 |

## 六、复习题

> [!question]- Q1. 40 岁男性，数月前有一次剧烈肛门痛持续 2 天后自行缓解；此后间歇有恶臭分泌物污染内裤、排便时痛。查体：肛缘可见炎性脓疱样病灶，周围皮下可扪及索条状硬结，直肠指检轻压痛但**无肿块**，生命体征正常。最合适的下一步是什么？为什么不是抗生素？
> **答案：转诊行手术评估（瘘管切开术 fistulotomy），术中麻醉下探查划定瘘管全程。**
> 病史 = 肛周脓肿自发破溃 → 残留上皮化窦道 = 肛瘘。**无波动性肿块** = 此刻没有需要急诊 I&D 的脓腔，走**择期手术**。
> 抗生素错在：隐窝腺源性瘘的通道已上皮化，**非手术治疗失败率高**；抗生素路线只属于**克罗恩相关肛瘘**（甲硝唑 metronidazole / 环丙沙星 ciprofloxacin + 英夫利昔单抗 infliximab），或合并蜂窝织炎/免疫抑制等特殊情况的辅助用药。

> [!question]- Q2. 同样是肛周疼痛，如何一句话区分"该今天切开"和"该排择期手术"？另外，若病灶其实在骶尾部中线而非肛缘，诊断怎么变？
> **答案：摸有没有波动性 (fluctuant) 触痛肿块。**
> 有波动性肿块 + 数天内持续剧痛 ± 发热 = **肛周脓肿 → 立即 I&D**（急诊/当天）。
> 无肿块、但有**外口 + 慢性流脓 + 索条** = **肛瘘 → 择期 fistulotomy**。
> 若病灶位于**骶尾部中线（臀沟上方）、可见藏毛的凹窝**，则是**藏毛病 (pilonidal disease)**：急性脓肿一样 I&D，慢性则切除窦道 + 脱毛与局部卫生 —— 它与肛管**不相通**，不是肛瘘。

> [!question]- Q3. 一位排便时撕裂样剧痛 + 便纸带血的患者，检查发现裂口在**左侧壁**而不是后正中线。除对症治疗外，你还要想什么？
> **答案：想"非典型肛裂 (atypical anal fissure) 的继发病因"。**
> 典型肛裂 90% 以上在**后正中线**（前正中线次之，多见于产后女性），一线是**坐浴 sitz bath + 高纤维 + 大便软化剂 + 局部硝酸甘油 nitroglycerin 或地尔硫䓬 diltiazem**，顽固者行**侧位内括约肌切开术 (lateral internal sphincterotomy)**。
> **偏离中线的裂口**提示：**克罗恩病 (Crohn disease)**、结核 (tuberculosis)、**人免疫缺陷病毒感染 (human immunodeficiency virus, HIV)**、梅毒 (syphilis)、白血病 (leukemia)/免疫抑制、肛管癌 (anal carcinoma) —— 需要针对病因检查（如肠镜、感染筛查、活检），而不是只给软膏。

## ⭐ 直肠脱垂 (rectal prolapse)（Q19033 锚定）

直肠脱垂 = 直肠经肛门脱出，多因盆底/直肠筋膜缺陷。最常见于女 >40；危险因素：多产、阴道分娩、慢性便秘/用力、卒中/痴呆、盆底功能障碍。

| 维度 | 要点 |
|---|---|
| 典型征 | ⭐红斑肿块 + 同心环 (concentric rings)、用力(Valsalva)时增大、放松回缩 = 全层脱垂 (full-thickness)；放射状皱襞 (radial invaginations) = 非全层 |
| 症状 | 排便梗阻感/便秘、大便失禁、肿块脱出 |
| 处理 | 全层或致残症状（失禁/便秘）→ 手术（直肠固定术 rectopexy 首选）；非全层/体弱 → 纤维 + 盆底肌锻炼 ± 生物反馈 |

> [!warning] 一眼分 直肠脱垂 vs 脱垂内痔
> 直肠脱垂 = 红斑肿块 + 同心环 → 手术；脱垂内痔 (prolapsed hemorrhoid) = 蓝紫色隆起 → 橡皮圈套扎 (rubber band ligation)。别把脱垂当痔套扎。

# 消化系统检查选择速查（原理 + 速查 + 推断流程）

> [!info] 这篇解决什么
> 病你都认得，但 stem 最后一句 "next step in diagnosis / most accurate test" 经常靠**选错检查**扣分。本篇不按疾病罗列，按**最容易混的逻辑轴**讲——先给原理（能现场推），再给速查表（能快速查），最后给 6 步推断流程（能落地）。

---

## 一、总纲：检查 = 选"能直接看见这个病物理本质"的那个模态

每种模态只擅长看**一类物理改变**。先问自己：这个病的本质是 **气体异常 / 管腔走向 / 软组织厚度·肿块 / 功能 / 血管** 哪一种，检查就自己跳出来。

```
病的物理本质            →  对应模态                  →  例子
──────────────────────────────────────────────────────────────
气体分布异常            →  腹部平片 X-ray            →  双泡(梗阻近端积气)/肠壁积气/游离气/气液平
管腔走向·通畅·瘘        →  造影 contrast study       →  中肠扭转(十二指肠去哪)/失弛缓鸟嘴/瘘管
管壁厚度·肿块·积液      →  超声 US / CT              →  幽门肌厚/套叠donut/胆囊壁/脓肿/憩室
功能(动力·分泌·异位)    →  生理·核素检查             →  测压/pH/HIDA胆流/Meckel异位胃黏膜
血管                    →  CT 血管成像 CTA           →  肠系膜缺血
```

> [!tip] 一句话总纲口诀
> **平片看"气"，造影看"路"，US/CT 看"肉"，核素/测压看"活儿"，CTA 看"血"。**

> [!warning] 易混名词：Upper GI series 是**造影**，不是 CT ⭐
> **Upper GI series (上消化道造影)** = 患者**吞下造影剂 (contrast，通常钡剂 barium)** + **X 线透视 (fluoroscopy，动态实时 X 光)** 追踪造影剂流过食管→胃→十二指肠的**走行**。本质是"**看路**"，不是 CT 的横断面切片。
>
> 它能确诊中肠扭转，正因为扭转的本质是"肠管绕系膜血管轴**走向异常**"——只有动态造影能直观显示：十二指肠**不过中线** + corkscrew (开瓶器/螺旋样) 下行。CT 静态切片反而不如它。
>
> ```
> 检查名 → 是什么 → 看什么
> ───────────────────────────────────────────
> Upper GI series  → 吞钡 + X 线透视造影 → 管腔走向(路)
> CT               → X 线横断面切片重建   → 软组织/肿块/积液(肉)
> 腹部平片 X-ray   → 单张静态 X 光        → 气体分布(气): 双泡/游离气
> 超声 US          → 声波                 → 管壁厚度/肿块(肉): 幽门肌/套叠
> ```
>
> **命名钥匙**：名字里有 **series / swallow / enema / 造影** 的 = **吞/灌造影剂 + X 线透视**这一类 (看路)，如 upper GI series、barium swallow、barium enema、Gastrografin esophagram **全是造影家族**；只有 **"CT"** 才是切片 (看肉)。

## 二、五条推论（"会混"的解药）

### 推论一：胆汁性 vs 非胆汁性 = 用胆汁当"解剖标尺"定位梗阻

胆汁从 **Vater 壶腹 (ampulla of Vater，十二指肠第二段)** 进入肠腔，所以呕吐物有没有胆汁 = 梗阻在壶腹上还是下。

- **吐胆汁 (bilious)** → 梗阻在**壶腹以下** → 累及十二指肠及远端 → 最怕**中肠扭转 (midgut volvulus，外科急症)** → 必须马上看肠子怎么转 → **上消化道造影 (upper GI series，看"路")**。
- **不吐胆汁 (non-bilious)** → 梗阻在**壶腹以上** (幽门/近端) → **肥厚性幽门狭窄 (HPS)** → 本质是肌肉肥厚 → 看"肉" → **超声 (ultrasound)**。

> [!danger] Q020 钩子
> "bilious" 这个词本身就把检查从 US 推向造影。看到新生儿胆汁性呕吐先想中肠扭转 + upper GI series。

### 推论二：造影剂选钡 vs 水溶性 = 赌"会不会漏出去"

- **钡剂 (barium)**：黏膜细节最清楚，但漏到腔外 (穿孔)→ 钡惰性、清不掉 → **化学性腹膜炎/纵隔炎 (chemical peritonitis/mediastinitis)**，高死亡率。
- **水溶性 (Gastrografin，泛影葡胺)**：漏了能吸收清除，但细节差、误吸 → 肺水肿。

> [!warning] 推法
> **怀疑穿孔 (Boerhaave、术后吻合口、急性穿孔) → 永远水溶性；不怀疑穿孔、要看精细黏膜 → 钡剂。**

### 推论三：US-first vs CT-first = 辐射管理 + "够不够用"

CT (computed tomography，计算机断层) 有辐射 (radiation)。

- **儿童 / 孕妇** → 能用超声回答就先超声 (无辐射)：阑尾炎 (appendicitis) 在孩子/孕妇是 US 先，成人才直接 CT。
- **成人急腹症** → CT 解剖全景更优，可直接上 (憩室炎定位、SBO (small bowel obstruction，小肠梗阻) 定因、肠系膜缺血)。

附带原则：**超声看不清才升级到金标准**——胆囊炎 (cholecystitis) US 不确定 → HIDA (hepatobiliary iminodiacetic acid scan，肝胆显像)；"先便宜无创、不够再上贵的"。

### 推论四：best initial vs gold standard = stem 问法决定答哪层

同一病常有**两个正确检查**，看 stem 用词给哪层（这是"会混"的最大来源）。

| stem 问法 | 给哪层 | 例 |
|---|---|---|
| "next step / best **initial**" | 便宜·快·无创 | 胆囊炎→US；失弛缓→钡餐；GERD (胃食管反流病)→PPI (质子泵抑制剂) 试验 |
| "most **accurate** / confirm / 金标准" | 确诊 | 胆囊炎→HIDA；失弛缓→测压；GERD→24h pH |

### 推论五：能"一举两得"的检查，问 next step 时优先

一个操作**既诊断又治疗**时常是答案：

- **肠套叠 (intussusception) → 空气/造影灌肠 (air/contrast enema)**：确诊同时复位。
- **胆总管结石 (choledocholithiasis) → ERCP (内镜逆行胰胆管造影)**：看见同时取石。
- **下消化道出血 (lower GI bleed，下消化道) → 结肠镜 (colonoscopy)**：找到同时止血/切除。

> [!danger] 反面——会害人的检查别选（stem 给它=陷阱）
> 急性憩室炎做结肠镜 (穿孔)；穿孔用钡剂；Zenker 憩室盲插 NG 管 (nasogastric tube，鼻胃管)/内镜 (穿孔)。

## 三、疾病速查表

### 3.1 新生儿/婴儿呕吐 + 梗阻（最常混）

| 病 | 关键 stem | 首选/确诊检查 | 影像钥匙 |
|---|---|---|---|
| 中肠扭转/旋转不良 Midgut volvulus | 新生儿**胆汁性**呕吐 | **Upper GI series 上消化道造影**(金标准)；不稳定→OR (operating room，手术室) | 十二指肠不过中线、corkscrew 螺旋征、Treitz 右移 |
| 十二指肠闭锁 Duodenal atresia | 出生即胆汁性呕吐、Down 综合征(唐氏) | **腹部平片**(无需造影) | Double bubble 双泡征、远端无气 |
| 肥厚性幽门狭窄 HPS | 3-6 周、**非胆汁性**喷射呕吐、橄榄包块 | **腹部超声** | 幽门肌厚 >3-4mm、长 >14-16mm、靶征 |
| 肠套叠 Intussusception | 6 月-3 岁、果酱样便、阵发腹痛 | **超声**初查 → **空气/造影灌肠**(诊断+治疗) | donut 甜甜圈征；灌肠同时复位 |
| 先天性巨结肠 Hirschsprung | 足月 >48h 未排胎便、腹胀 | **直肠抽吸活检**(金标准:无神经节细胞) | 造影 transition zone 移行段 |
| 坏死性小肠结肠炎 NEC | **早产儿**、喂养后腹胀血便 | **腹部平片** | Pneumatosis 肠壁积气、门静脉积气 |
| 食管闭锁/TEF (tracheoesophageal fistula，气管食管瘘) | 唾沫多、NG 管插不进 | **平片 + NG 管**(看盘曲) | NG 管上段 coiling |

> [!danger] 这一簇的分诊钥匙
> **胆汁性** → 梗阻在壶腹下 → 中肠扭转(急!)/十二指肠闭锁 → 上消化道造影 或 平片(双泡)。
> **非胆汁性** → 幽门狭窄 → 超声。

### 3.2 儿科 GI 出血/黄疸

| 病 | stem | 检查 |
|---|---|---|
| 梅克尔憩室 Meckel diverticulum | 幼儿**无痛性**便血 | **锝-99m 高锝酸盐扫描 (Meckel scan)** |
| 胆道闭锁 Biliary atresia | >2 周黄疸、**结合胆红素↑**、陶土样便 | **HIDA 扫描**(无排泄)+超声；金标准术中胆道造影/活检 |

### 3.3 成人食管（造影选择是考点）

| 病 | 首选 | 金标准 | 陷阱 |
|---|---|---|---|
| 贲门失弛缓 Achalasia | 钡餐(鸟嘴 bird's beak) | **食管测压 manometry** | 内镜排除假性失弛缓(癌) |
| GERD | 临床+PPI 试验 | **24h 食管 pH 监测** | 报警症状才内镜 |
| Boerhaave 食管破裂 | **水溶性造影 Gastrografin** + CT | — | 禁钡剂(入纵隔→纵隔炎) |
| Zenker 憩室 | 钡餐 barium swallow | — | 禁盲插 NG/盲目内镜(穿孔) |

### 3.4 成人急腹症/肝胆（US-first vs CT-first）

| 病 | 首选 | 备注 |
|---|---|---|
| 急性胆囊炎 Cholecystitis | **超声** | US 不确定→HIDA(金标准) |
| 胆总管结石 Choledocholithiasis | 超声→**MRCP (磁共振胰胆管成像)** | ERCP 是治疗，不当首诊 |
| 憩室炎 Diverticulitis | **CT 腹盆(增强)** | 急性期禁结肠镜(穿孔) |
| 急性肠系膜缺血 Mesenteric ischemia | **CT 血管成像 CTA (CT angiography)** | 腹痛剧烈与体征不符 |
| 小肠梗阻 SBO | 立位平片(气液平)→**CT**定因 | — |
| 阑尾炎 Appendicitis | 成人→**CT**；儿童/孕妇→**超声**先 | — |
| 急性胰腺炎 Pancreatitis | 诊断靠**临床+脂肪酶**，CT 看并发症 | 早期 CT 无意义 |

## 四、6 步现场推断流程

遇到任何"消化系统 next step 查什么"，按序走：

```
1. 这病的本质是哪类? → 气/路/肉/活儿/血 → 定模态            (总纲)
2. 新生儿呕吐? → 先分胆汁性? → 定梗阻层面                   (推论一)
3. 要造影? → 怀疑穿孔吗? → 钡 or 水溶性                     (推论二)
4. 孩子/孕妇? → 能 US 就 US,别先 CT                         (推论三)
5. stem 问 initial 还是 confirm? → 给对应那层              (推论四)
6. 有没有"一举两得"或"会害人"的选项? → 加分/排雷            (推论五)
```

> [!success] 一句话收尾
> 走完这 6 步是**推**出来的，不是回忆——这才是"检查会混"的真正解药。

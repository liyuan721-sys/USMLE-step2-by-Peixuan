## 一、核心反射：见黄疸先问"直接还是间接"

> [!danger] 一句话锁定
> **"青少年良性黄疸" ≠ 一律 Gilbert。** 先看 **bilirubin fraction（直接 conjugated vs 间接 unconjugated）**，再看 **酶谱（AST/ALT/ALP）+ 影像（超声）**，最后才猜病名。先猜病名 = NBME 设的陷阱（[[NBME11_错题本#^Q118]] 就是这么挖的）。

胆红素代谢三步 + 各步缺陷对应病：

```
RBC 破坏 → 非结合(间接)胆红素 unconjugated bilirubin
   │              （脂溶、入血与白蛋白结合、不溶于水、尿中查不到）
   │  ① 肝摄取 uptake (OATP1B1/1B3)  ── 缺陷 → Rotor（部分机制）
   ▼
肝细胞内 UGT1A1 葡糖醛酸化 conjugation
   │  ② 此酶 ↓/缺 ── 缺陷 → Gilbert(轻↓) / Crigler-Najjar(重缺)
   ▼
结合(直接)胆红素 conjugated bilirubin（水溶、可入尿）
   │  ③ 经 MRP2 排入胆小管 canalicular excretion ── 缺陷 → Dubin-Johnson
   ▼
胆汁 → 肠道
```

> [!tip] 两步分流口诀
> **第一刀 = 直接 vs 间接**（哪一半胆红素升高）；**第二刀 = 酶 + 影像正常吗**（正常 = 良性遗传病；异常 = 肝细胞损伤 / 梗阻）。

---

## 二、间接 vs 直接两大阵营（含非遗传干扰项）

> [!warning] 先把"假性遗传病"排掉
> 遗传性高胆红素血症都是 **"孤立性胆红素↑ + 其余肝功 + 影像正常"** 的良性病。一旦 **酶↑（AST/ALT 或 ALP）或影像异常 → 不是这 4 个病**，要想肝炎 / 梗阻 / 溶血。

| 病 | 胆红素模式 | 酶/影像 | 其他钥匙 |
|---|---|---|---|
| **Gilbert（吉尔伯特）** | **非结合(间接)↑** | 正常 | **应激/饥饿/感染/劳累诱发**，最常见(~5%)，良性 |
| **Crigler-Najjar（克里格勒-纳贾尔）** | **非结合(间接)↑↑↑** | 正常 | 新生儿；I 型重→核黄疸 kernicterus；II 型轻 |
| **Dubin-Johnson（杜宾-约翰逊）** | **结合(直接)↑** | 正常 | **肝黑** black liver（色素沉着），良性 |
| **Rotor（罗特）** | **结合(直接)↑** | 正常 | 似 DJ 但**肝不黑**，更罕见、更轻 |
| — 溶血 hemolysis | 非结合↑ | 正常但**贫血 + 网织红 reticulocyte↑ + LDH(乳酸脱氢酶 lactate dehydrogenase)↑ / 触珠蛋白 haptoglobin↓** | 有贫血表现（鉴别 Gilbert 关键） |
| — 病毒/自免肝炎 hepatitis | 混合 | **AST/ALT↑** | 炎症性肝损伤模式 |
| — 胆道梗阻 obstruction | 结合↑ | **ALP↑ + 影像异常** | 多有 RUQ 痛 / 胆管扩张 |

## 三、4 病逐个击破

### 3.1 间接(非结合)阵营 —— UGT1A1 酶活性问题

> [!info] Gilbert syndrome（吉尔伯特综合征）
> - **机制**：UGT1A1（葡糖醛酸转移酶 UDP-glucuronosyltransferase）活性**轻度↓**（~30%）→ 结合能力不足。
> - **诱因**：**应激、禁食/饥饿、感染、剧烈运动、月经** → 间接胆红素一过性↑ → 轻度黄疸。
> - **临床**：人群 ~3–7%，**完全良性、无需治疗**；常体检偶然发现或应激时巩膜黄染。
> - **实验室**：孤立性**非结合↑**，CBC（全血细胞计数 complete blood count）/ 网织红 reticulocyte / 酶 / 影像全正常（与溶血区分靠"无贫血、网织红不高"）。
> - **考点**：年轻人禁食/熬夜/考试季后轻度黄疸 + 其余全正常 = Gilbert。

> [!warning] Crigler-Najjar syndrome（克里格勒-纳贾尔综合征）
> - **机制**：UGT1A1 **严重缺乏/完全缺失**（比 Gilbert 重得多）。
> - **I 型**：酶**完全缺失** → 极重非结合高胆红素 → **核黄疸 kernicterus（胆红素脑病）**，**对苯巴比妥 phenobarbital 无反应**；需光疗 phototherapy + 肝移植 liver transplant。
> - **II 型（Arias 综合征）**：酶**部分残留** → 较轻，**对苯巴比妥 phenobarbital 有反应**（诱导残余酶）。
> - **考点**：新生儿持续严重非结合高胆红素 → CN；**"苯巴比妥有效=II 型 / 无效=I 型"** 是高频分流点。

### 3.2 直接(结合)阵营 —— 胆红素已结合、但出不去肝细胞

> [!info] Dubin-Johnson syndrome（杜宾-约翰逊综合征）
> - **机制**：**MRP2（多药耐药相关蛋白 2 multidrug resistance-associated protein 2，基因 ABCC2）胆小管转运体缺陷** → 结合胆红素**无法排入胆小管** → 反流入血 → **孤立性结合(直接)高胆红素**。
> - **特征**：**肝呈黑色**（black liver，溶酶体内 epinephrine 代谢色素沉着）；**完全良性、无需治疗**。
> - **实验室**：结合↑ + 酶/影像正常；尿中可见结合胆红素（水溶）。
> - **考点**（[[NBME11_错题本#^Q118]]）：青少年黄疸 + **conjugated 3 / total 4** + 余肝功正常 + 超声正常 = DJ（**不要反射 Gilbert——Gilbert 是间接**）。

> [!info] Rotor syndrome（罗特综合征）
> - **机制**：**OATP1B1/1B3（有机阴离子转运多肽 organic anion-transporting polypeptide）肝摄取/储存缺陷**（结合胆红素的肝再摄取障碍）→ 结合↑。
> - **特征**：**肝不黑**（vs DJ 关键鉴别）；更罕见、更轻；同样良性无需治疗。
> - **DJ vs Rotor 一刀切**：影像/活检"**肝黑 = DJ，肝不黑 = Rotor**"；考试很少深挖 Rotor，知道它"像 DJ 但肝不黑"即可。

## 四、Memory Hook

> [!tip] 锁定口诀
> - **"间接 vs 直接"两阵营**：间接(unconjugated)= **G**ilbert + **C**rigler-Najjar（UGT1A1 **酶**问题，"造不出"结合胆红素）；直接(conjugated)= **D**ubin-Johnson + **R**otor（**运输**问题，"结合了却运不走"）。
> - **"Direct 高、其他静悄悄 = Dubin-Johnson"**；Gilbert 记死是 **indirect + 应激诱发**。
> - **CN 分型**：**苯巴比妥有效 = II 型（轻）；无效 = I 型（重，核黄疸）**。
> - **DJ vs Rotor**：**黑肝 = DJ**，不黑 = Rotor。
> - **排干扰**：酶↑→肝炎；ALP↑+影像异常→梗阻；贫血+网织红↑→溶血。良性遗传病一定是"**只有胆红素一项不对**"。

## 五、复习题

> [!question]- Q1. 15 岁男孩巩膜黄染，total bilirubin 4 / conjugated 3，其余肝功正常，肝超声正常。最可能诊断？为何不是 Gilbert？
> **Dubin-Johnson syndrome**。关键 = **结合(直接)胆红素升高**（3/4）+ 余肝功 + 影像正常。Gilbert 是**非结合(间接)**升高，故排除。先分直/间接再猜病名。

> [!question]- Q2. 把 4 个遗传性高胆红素血症按"间接 vs 直接"分两组，并各写一句机制。
> **间接(非结合)**：Gilbert（UGT1A1 轻度↓，应激诱发）/ Crigler-Najjar（UGT1A1 重度缺乏）。**直接(结合)**：Dubin-Johnson（MRP2 胆小管排泄缺陷，肝黑）/ Rotor（OATP1B1/1B3 肝摄取缺陷，肝不黑）。

> [!question]- Q3. 新生儿持续重度非结合高胆红素，用苯巴比妥 phenobarbital 后胆红素明显下降。是 Crigler-Najjar 哪一型？反之无反应是哪型、最怕什么并发症？
> **有反应 = II 型（Arias，酶部分残留）**；无反应 = **I 型（酶完全缺失）**，最怕 **核黄疸 kernicterus（胆红素脑病）**，需光疗 + 肝移植。

> [!question]- Q4. 一个年轻人禁食/熬夜后轻度黄疸，间接胆红素↑，但 CBC、网织红、LDH、触珠蛋白都正常。Gilbert 还是溶血？靠什么区分？
> **Gilbert**。溶血也是间接↑，但会有**贫血 + 网织红↑ + LDH↑ + 触珠蛋白↓**；本例血象全正常 → 排除溶血 → Gilbert（应激/饥饿诱发）。

> [!question]- Q5. Dubin-Johnson 和 Rotor 都是孤立性结合胆红素升高，怎么一刀切开？
> **肝脏颜色**：Dubin-Johnson **肝黑**（black liver，色素沉着）；Rotor **肝不黑**。两者均良性、无需治疗。

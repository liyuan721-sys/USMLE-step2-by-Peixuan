# GI 大肠梗阻三鉴别（机械性 / 粪便嵌塞 / Ogilvie）

> [!info] 一句话定位
> 住院 / 老年 / 术后患者出现**腹胀 + 结肠扩张**，本质是回答一个问题：**这扩张是「真有东西堵」还是「肠子不动」？**
> 三大类：①机械性大肠梗阻 mechanical LBO（有移行带的真梗阻）②粪便嵌塞 fecal impaction（直肠壶腹堵硬便）③急性结肠假性梗阻 Ogilvie syndrome（无机械梗阻的动力性扩张）。
> 派生自 **Q104**（NBME11A，Fighter#4，结肠假性梗阻 Ogilvie）。

> [!abstract] 缩写表（首现全称）
> - **LBO** = large bowel obstruction，大肠梗阻
> - **SBO** = small bowel obstruction，小肠梗阻
> - **ACPO** = acute colonic pseudo-obstruction，急性结肠假性梗阻（= Ogilvie 综合征）
> - **CRC** = colorectal cancer，结直肠癌
> - **DRE** = digital rectal examination，直肠指检
> - **NPO** = nil per os，禁食；**NG** = nasogastric，鼻胃管
> - **transition point** = 移行带（影像上近端扩张 + 远端塌陷的交界，机械梗阻标志）

---

## 一、总览决策树（先看「有无移行带 + 直肠是否空」）

```
住院/老年/术后患者：腹胀 + 腹平片示结肠扩张
        │
        ▼
  ① 直肠指检 DRE：直肠壶腹是否触到硬便？
        │
   ┌────┴─────┐
  有硬便       直肠空
   │            │
   ▼            ▼
 粪便嵌塞    ② 影像看有无「移行带 transition point」
 fecal           （近端扩张 + 远端突然塌陷）
 impaction        │
            ┌─────┴──────┐
          有移行带       无移行带
        （局灶截断）    （盲肠→肛门连续弥漫扩张）
            │              │
            ▼              ▼
       机械性大肠梗阻    Ogilvie 综合征
       mechanical LBO   (ACPO 假性梗阻)
       └ 癌 / 扭转 volvulus    └ 动力性，无真堵
```

> [!tip] 三个分水岭问题
> 1. **直肠有没有硬便？** → 有 = 嵌塞；空 = 看影像。
> 2. **影像有没有移行带？** → 有 = 机械梗阻；无（连续扩张到肛门）= Ogilvie。
> 3. **有没有腹膜征 / 游离气 / 便血？** → 有 = 怀疑穿孔 / 缺血，升级处理（这是任何一类失代偿的红旗）。

## 二、① 机械性大肠梗阻 mechanical LBO（大肠梗阻 large bowel obstruction）

> [!warning] 核心特征
> **真有梗阻物**堵住肠腔 → 影像有**移行带**（近端扩张 + 远端塌陷）。最常见病因 **结直肠癌 CRC**，其次**扭转 volvulus（乙状结肠 / 盲肠）**。

### 病因排序

| 病因 | 钩子（stem 线索） | 关键点 |
|---|---|---|
| **结直肠癌 colorectal cancer (CRC)** ⭐最常见 | 老年 + **慢性排便习惯改变 / 便血 / 体重↓ / 缺铁性贫血** + 局灶移行带 | 病程**慢**、左侧癌易环周缩窄致梗阻；确诊后查结肠镜 + 活检 |
| **乙状结肠扭转 sigmoid volvulus** | **老年、卧床、长期便秘 / 精神科 / 养老院**患者 | 影像 **coffee-bean 咖啡豆征**（扩张肠袢指向右上）；造影 **bird-beak 鸟嘴征** |
| **盲肠扭转 cecal volvulus** | **较年轻**患者（解剖游离盲肠） | 扩张盲肠移位至**左上腹**；常需手术 |
| 憩室狭窄 / 粪石 / 吻合口狭窄 | 既往憩室炎 / 术后 | 少见 |

### 影像鉴别（buzzword）

```
乙状结肠扭转 sigmoid volvulus：
  腹平片 → coffee-bean / 咖啡豆 / "倒 U" 征
           （巨大扩张肠袢，闭袢，指向右上象限）
  造影/CT → bird-beak / 鸟嘴征（扭转点呈鸟嘴样渐窄）

盲肠扭转 cecal volvulus：
  扩张盲肠（含气）移位至 左上腹（"咖啡豆"开口朝右下）
```

### 处理阶梯

> [!success] 乙状结肠扭转处理（高频）
> 1. **无腹膜征 / 无坏死** → **内镜下乙状结肠镜减压 endoscopic decompression**（首选，可同时放减压管）。
> 2. 减压后**择期乙状结肠切除**（复发率高，不切会再扭）。
> 3. **有腹膜征 / 坏死 / 穿孔 / 减压失败** → **急诊手术（切除）**。
>
> 盲肠扭转：内镜减压成功率低 → **多数直接手术（右半结肠切除 / 盲肠固定）**。
> CRC 致梗阻：稳定者可放**支架 stent 桥接**或急诊切除。

## 三、② 粪便嵌塞 fecal impaction（粪便嵌塞）

> [!warning] 核心特征
> 大量**硬便堵在直肠壶腹 / 远端结肠**，常见长期便秘 / 卧床 / 阿片 opioid / 老年痴呆患者。**DRE 可触到硬便**——这是与 Ogilvie 的关键分水岭。

- **临床**：腹胀 + 排便费力；可有**溢出性失禁 overflow incontinence**（稀便绕过硬便块漏出，貌似「腹泻」是陷阱）。
- **诊断**：**直肠指检 DRE 首选**（直接触到嵌塞硬便）；腹平片见直肠大量粪团。
- **处理**：**手抠 disimpaction + 灌肠 enema**（先解远端），再口服渗透性泻药（聚乙二醇 PEG / 乳果糖 lactulose）维持；阿片相关加 PAMORA（甲基纳曲酮 methylnaltrexone）。

> [!danger] 陷阱：scant stool / 溢出性失禁
> stem 写「排便少 scant stool」或「漏稀便」易诱向嵌塞，但**必须 DRE 确认直肠是否真有硬便**；若 DRE 直肠空 + 全结肠弥漫扩张 → 反指向 Ogilvie（见 Q104）。

## 四、③ 急性结肠假性梗阻 Ogilvie 综合征（ACPO acute colonic pseudo-obstruction）

> [!warning] 核心特征
> **无机械性梗阻**，却因**自主神经失调**致结肠大面积扩张：**盲肠到肛门连续弥漫扩张、无移行带、直肠空**。

### 诱因（stem 钩子）

- **高龄 + 术后**（尤其骨科 / 髋术后，如 Q104 髋骨折修复术后 1 周）
- 重症 / 卧床 / 创伤 / 感染
- **电解质紊乱**（低钾 / 低镁 / 低钙）
- **阿片 opioid**（氢可酮 hydrocodone 等抑制结肠动力，加分钩子）
- 抗胆碱药

### 处理阶梯（高频考点）

> [!success] Ogilvie 处理阶梯
> 1. **排除机械梗阻**（腹平片 / CT 造影确认无移行带、无游离气）。
> 2. **保守支持（盲肠 <10–12 cm、无腹膜征）**：NPO 禁食 + NG 鼻胃管 + 肛管减压 + **停阿片 / 抗胆碱药** + **纠正电解质（补钾镁）**。
> 3. **保守 24–48 h 失败 或 盲肠 >10–12 cm** → **新斯的明 neostigmine（拟胆碱药 cholinesterase inhibitor）** 静推（需心电监护，警惕**心动过缓**，备阿托品 atropine）。
> 4. **新斯的明禁忌 / 无效** → **结肠镜减压 colonoscopic decompression**。
> 5. **盲肠 >12 cm + 缺血 / 穿孔征象** → 急诊手术。

> [!danger] 盲肠直径阈值（Laplace 定律）
> 盲肠管径最大 → 张力最高 → **盲肠 >10–12 cm 是穿孔 / 缺血红旗**。即便整体是「假性」梗阻，盲肠仍可真穿孔。

## 五、三者对照表（诱因 / 影像 / DRE / 处理）

| 维度 | ①机械性 LBO | ②粪便嵌塞 | ③Ogilvie (ACPO) |
|---|---|---|---|
| **本质** | 真有梗阻物 | 硬便堵远端 | 动力性，无真堵 |
| **常见诱因** | 老年 + CRC / 扭转 | 长期便秘 / 阿片 / 卧床 | 高龄 + 术后 / 重症 / 电解质 / 阿片 |
| **病程** | 较慢（癌）/ 急（扭转） | 慢性累积 | 急性（数天） |
| **DRE 直肠指检** | 多为空（远端肿块可触） | **可触硬便** ⭐ | **空** ⭐ |
| **影像移行带 transition point** | **有**（局灶截断） | 直肠大量粪团 | **无**（连续扩张到肛门）⭐ |
| **影像 buzzword** | coffee-bean / bird-beak（扭转）| 直肠粪石 | 全结肠弥漫积气，无截断 |
| **便血 / 绞痛** | 癌可便血；扭转可缺血 | 无 | 无（有则疑缺血穿孔） |
| **首选确诊步骤** | CT / 造影找移行带 | **DRE** | 腹平片 / CT 排机械梗阻 |
| **核心处理** | 扭转→内镜减压±手术；癌→支架/切除 | 手抠 + 灌肠 | 纠诱因→**新斯的明**→结肠镜减压 |
| **穿孔红旗** | 闭袢 / 缺血 | 少见 | **盲肠 >10–12 cm** |

> [!tip] 一图记忆口诀
> - **能触到硬便 → 嵌塞**（DRE 阳性）。
> - **影像有移行带 → 机械梗阻**（coffee-bean = 乙状扭转）。
> - **直肠空 + 连续扩张到肛门 → Ogilvie**（治疗词记 neostigmine、盲肠 12 cm）。

## 六、复习自测

> [!question]- Q1. 92 岁男髋术后 1 周，腹胀，腹平片示盲肠到肛门连续弥漫扩张、无膈下游离气，DRE 直肠空、无硬便，便潜血阴。最可能诊断？下一步关键处理？
> **诊断**：急性结肠假性梗阻 Ogilvie 综合征（ACPO）。
> 线索：高龄 + 术后 + **全结肠连续弥漫扩张无移行带** + **直肠空**（排除嵌塞）+ 无游离气（排除穿孔）+ 便潜血阴（排除缺血性结肠炎）。
> **处理**：先排机械梗阻 + 停阿片 + 纠正电解质（钾镁）+ NPO/NG 减压；盲肠 >10–12 cm 或保守失败 → **新斯的明 neostigmine**（备阿托品）→ 仍不行结肠镜减压。

> [!question]- Q2. 老年养老院长期便秘患者，腹平片见巨大「咖啡豆 coffee-bean」征扩张肠袢指向右上腹。诊断 + 首选处理？
> **诊断**：乙状结肠扭转 sigmoid volvulus（机械性大肠梗阻）。
> **处理**：无腹膜征 → 内镜下乙状结肠镜减压（首选），减压后择期乙状结肠切除（防复发）；有腹膜征 / 坏死 → 急诊手术。

> [!question]- Q3. 长期阿片患者诉「腹泻」漏稀便，腹胀。如何与 Ogilvie 区分？关键查体？
> 这是粪便嵌塞 fecal impaction 的**溢出性失禁 overflow incontinence**（稀便绕过硬便块）。
> 关键：**直肠指检 DRE** —— 嵌塞**触到硬便**（处理：手抠 + 灌肠）；Ogilvie **直肠空 + 全结肠弥漫扩张**。

> [!question]- Q4. 为什么 Ogilvie 即使「无真梗阻」仍要紧盯盲肠直径？阈值多少？
> Laplace 定律：管径最大处（盲肠）壁张力最高 → 最易缺血穿孔。**盲肠 >10–12 cm** 是穿孔红旗，需升级到新斯的明 / 结肠镜减压，>12 cm + 缺血征象直接手术。

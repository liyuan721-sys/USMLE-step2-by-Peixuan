# 门脉高压治疗逻辑（Portal Hypertension — 药与机制大全）

> [!info] 这一篇要解决什么
> 把所有"治门脉高压"的药 / 操作 / 时机用 **一个公式 (P = Q × R)** 串起来，
> 配套覆盖：**急性静脉曲张出血处理 / 一级 & 二级预防 / HRS / SBP / HE 交叉**。

---

## 一、核心公式：P = Q × R

> [!danger] 一句话本质
> **门静脉压力 = 流入血量 (Q) × 流出阻力 (R)**
> 肝硬化时 **Q ↑ 和 R ↑ 同时发生**，双重打击。
> 治疗只有三条路：**↓Q / ↓R / 绕过 R**。

```
门脉压 P = Q × R
            │   │
            │   └──→ R ↑ : 硬化肝实质阻塞肝窦
            │              干预: TIPS（绕过）/ 肝移植（换新）
            │
            └──→ Q ↑ : Splanchnic vasodilation（NO 介导）
                       让大量血涌入门脉系统
                       干预: Terlipressin / Octreotide / 非选 β-blocker
```

---

## 二、解剖前提：搞清楚门静脉的"上下游"

> [!tip] 门静脉是下游收集站
> 门静脉**不是动脉**，是 splanchnic 动脉的**汇合下游**。
> 关掉上游内脏动脉水龙头 → 下游门脉水位下降。

```
            主动脉
              │
       ┌──────┴──────┐
       │             │
   肠系膜上/下动脉   脾动脉      ← 这里收缩（药物作用点）
       │             │
       ▼             ▼
    小肠/大肠        脾
       │             │
       ▼             ▼
   肠系膜静脉      脾静脉
       │             │
       └──────┬──────┘
              ▼
         ★ 门静脉 ★        ← 这里测压力（HVPG）
              │
              ▼
            肝脏（硬化、R↑）  ← 这里被堵
              │
              ▼
           肝静脉 → IVC
```

> [!info] HVPG (Hepatic Venous Pressure Gradient)
> 临床用 **HVPG = 肝静脉楔压 − 游离压** 间接反映门脉压力。
> - HVPG **≥ 10 mmHg** → 临床显著门脉高压（CSPH），开始出现静脉曲张
> - HVPG **≥ 12 mmHg** → 静脉曲张出血风险显著
> - 治疗目标：HVPG **< 12** 或较基线 **↓ ≥ 20%**

## 二.5、门体侧支循环 — 为什么爆在食管 / 直肠 / 脐

> [!danger] 核心理解
> 食管下段静脉**正常情况下两边都通**：一边通门脉（经胃左静脉），一边通体循环（经奇静脉）。
> 平时血流以体循环为主，门脉压一高 → 血**倒灌**入食管下段黏膜下静脉丛 → 鼓起来 = **静脉曲张**。
> 全身**3 个门体吻合点同时被点亮** → 食管 / 直肠 / 脐周 三处出血点。

### 食管静脉的"双向出口"解剖

```
                      食管下段黏膜下静脉丛
                    (Submucosal venous plexus)
                              │
                  ┌───────────┴───────────┐
                  │                       │
              （向下走）              （向上走）
                  │                       │
                  ▼                       ▼
          ★ 胃左静脉 ★              食管静脉
       (Left gastric /              (Esophageal veins)
        coronary vein)                    │
                  │                       │
                  ▼                       ▼
              门静脉                  奇静脉 (Azygos)
                  │                       │
                  ▼                       ▼
               肝脏                   上腔静脉 (SVC)
                  │                       │
                  ▼                      心脏
              肝静脉
                  ▼
                IVC → 心
```

> [!info] 关键吻合点
> 食管下段黏膜下静脉丛 = **门体吻合 (portosystemic anastomosis)**
> 胃左静脉（门脉系统）+ 奇静脉（体循环系统）在此"握手"
> 正常时血走奇静脉（顺路上腔），门脉堵了反走胃左 → 食管丛鼓爆

### 门脉高压时的血流倒灌

```
正常状态：
食管丛 ──→ 胃左 ──→ 门脉 ──→ 肝 ──→ IVC ──→ 心
                                ✓ 压力正常

肝硬化 + 门脉高压：
食管丛 ──→ 胃左 ──╳ 门脉（堵！压力倒灌）
              │
              ▼ 血走不通，反向冲
食管丛压力↑↑ → 黏膜下细静脉鼓起 → 食管静脉曲张
              │
              ▼ 张力突破承受力
            破裂出血 = 呕血
```

> [!warning] 为什么这些静脉特别容易破？
> - 食管黏膜下静脉**壁薄、无支撑**（不像皮肤静脉有肌肉/筋膜保护）
> - 一旦扩张鼓起，随食物擦伤 / 反流酸刺激 / 咳嗽腹压增高 → **破裂出血**
> - 出血量大 + 不易自止（因为门脉压力持续推血出去）

### 三大门体侧支循环（CK 经典 triad）★★

| 位置 | 门脉侧 | 体循环侧 | 临床表现 |
|---|---|---|---|
| **① 食管下段** | 胃左静脉 (Left gastric / coronary) | 奇静脉 (Azygos) | **食管静脉曲张** → 呕血 ★★★ |
| **② 直肠上段** | 直肠上静脉 (Superior rectal) | 直肠中/下静脉 | **门脉性直肠静脉曲张**（不是普通痔疮！）|
| **③ 脐周** | 脐旁静脉 (Paraumbilical) | 浅腹壁静脉 (Superficial epigastric) | **Caput medusae 美杜莎头** |

> [!danger] CK 高频陷阱 — 痔疮 vs 门脉性直肠曲张
> "肝硬化病人有痔疮 = 门脉高压所致" — **不一定！**
> - **普通痔疮 (hemorrhoids)** = 直肠下静脉丛扩张，跟门脉无关
> - **门脉性直肠静脉曲张** = 长在更高位置（直肠上段），出血时是**无痛的鲜红血**
> - 真考点：见 caput medusae + 食管曲张 + 高位直肠曲张 = **门脉高压 triad**

> [!tip] 解剖死记
> **胃左静脉 + 奇静脉 在食管下段"握手" = 门体吻合**
> 上 / 中 / 下三个出口，门脉压力高了都开始喷：
> - **上**（食管） → 呕血（最危险）
> - **中**（脐周） → Caput medusae（最戏剧化）
> - **下**（直肠） → 便血（最不痛）

## 二.6、为什么自然侧支救不了 — 物理学硬限制

> [!danger] 核心理解
> **侧支循环 IS 在工作 — 静脉曲张本身就是侧支被迫上岗的物理表现。**
> 但这条"备用路"**容量太小 + 阻力太大**，永远无法把门脉压拉回正常。
> 曲张 ≠ 侧支失败，曲张 = 侧支"超载到撑破"的物理表现。

### 类比：高速路堵了，绕村道

```
正常：
肠道血 → 门脉 → 肝（8 车道高速，畅通）→ IVC → 心
              流量 1500 mL/min，毫无压力

肝硬化：
肠道血 → 门脉 → 肝（高速塞死）
              │
              ▼ 走"村道"绕行
        胃左 → 食管丛 → 奇静脉 → SVC
              │
              ▼ 但村道只有 2 车道宽，1500 mL/min 流量挤进去
        村道车堵 + 路面承受不住压力 → 路面崩裂（曲张破裂出血）
```

**核心 insight**：村道不是没用，是**容量不够**。流量再强行挤进去，就把路撑坏。

### 三个物理原因

#### ① 容量根本不够（流量 vs 通道）

| 通道 | 正常处理流量 | 设计承受压力 |
|---|---|---|
| **门脉 → 肝** | 1000-1500 mL/min（占心输出 25%）| 正常门脉压 3-5 mmHg |
| **食管黏膜下静脉丛** | 几 mL/min（黏膜微循环用）| 设计承受 ~5 mmHg |

让黏膜下小静脉**承担 1500 mL/min 门脉血** = 让排水沟当主下水道
流量根本走不掉 → 上游门脉压依然 >12 mmHg（爆点）

#### ② 阻力物理：泊肃叶定律 R ∝ 1/r⁴

> [!warning] Poiseuille's Law — 生理学硬限制
> **血管阻力 R ∝ 1 / r⁴**（与半径 4 次方成反比）

| 血管 | 直径 | 相对阻力 |
|---|---|---|
| 门脉 | ~13 mm | 1（基准） |
| 食管黏膜下静脉 | ~1-3 mm | **几百倍** |

哪怕侧支全开、再扩张 2-3 倍，**总阻力仍然远高于正常肝脏**
→ P = Q × R 公式里 R 拉不下来 → 门脉压拉不下来

#### ③ Splanchnic vasodilation 还在火上浇油

> [!danger] 流量持续被推大
> 肝硬化时 NO 介导的内脏血管扩张 → **流入门脉的血量 Q 反而 ↑**
> 不是流量正常被堵，而是流量本来就在涨，侧支再开都赶不上
> → 侧支永远在"追赶"，永远追不上

### 重新框定：曲张 IS 侧支工作的产物

> [!success] 重新理解 "varices"
> - 食管曲张 / 痔疮曲张 / Caput medusae **都是侧支循环工作的证据**
> - 这些静脉就是被门脉血"撑大"才变成"曲张"
> - 没有侧支 = 这些静脉看不到
> - **侧支 + 高压 = 曲张 = 必然结果**，不是缺陷
>
> 换句话说：**身体已经尽力打开所有能用的旁路，但物理学不允许这条路通畅**

### 为什么 TIPS / 手术分流能成功？

> [!tip] 关键区别：**人工大口径**

| | 自然侧支（食管丛） | TIPS |
|---|---|---|
| 直径 | 1-3 mm | **8-10 mm** |
| 阻力 R | 极高 | **极低** |
| 流量承载 | 几 mL/min | **接近门脉全流量** |
| 结果 | 压力不降 → 曲张破裂 | **门脉压立刻 ↓ 50%+** |

→ 治疗逻辑：自然侧支太细不够用，**直接造一个大口径分流** = 真正卸压

```
TIPS：
门脉 → ★ 大口径支架（绕过硬化肝）★ → 肝静脉 → IVC
       R 极低 → 流量畅通
     门脉压 ↓↓ → 自然侧支减压
         → 食管曲张萎缩、不再破裂
```

> [!info] 三个 take-home
> 1. **曲张 = 侧支在工作的证据**，不是侧支失败
> 2. **R ∝ 1/r⁴** — 小血管再多也救不了门脉压（生理硬限制）
> 3. **TIPS 之所以有效 = 直径够大** → 自然侧支永远不够、人工分流必要

## 三、三大干预方向

### ① 减少 Q（流入门脉的血）→ 收缩 splanchnic 动脉

| 药 | 作用机制 | 主要用途 | 备注 |
|---|---|---|---|
| **Terlipressin** | V1 受体 → splanchnic 动脉收缩 | **HRS-AKI** ★ / 急性静脉曲张出血 | 2022 FDA 批 |
| **Vasopressin** | V1 + V2 | 急性出血（已少用） | 心脏 / 肠系膜缺血副作用大 |
| **Octreotide** | 生长抑素类似物，抑制扩血管激素（胰高血糖素等）→ splanchnic 收缩 | **急性静脉曲张出血一线** ★（美国） | 副作用小，可持续输 2-5 天 |
| **Somatostatin** | 同上 | 同上（欧洲多用） | — |
| **非选择性 β-blocker** | β1 ↓CO + **β2 阻 → α 无对抗 → splanchnic 收缩** | **一级 + 二级预防** ★ | 见下表 |

#### 同样收 splanchnic，为什么药物分工不同 — 时间维度决定一切 ⭐

> [!danger] 一句话本质
> **急性出血（3-5 天）≠ 长期预防（月/年）** — 用药逻辑完全反着来
> Octreotide / Terlipressin = **强效但只能短期 IV** → 救火
> β-blocker = **温和但可口服多年** → 防火

**各药"班次表"**：

| 药 | 给药途径 | 用药时长 | 主要场景 |
|---|---|---|---|
| **Octreotide** | IV 持续输注 | 2-5 天 | 急性出血 ✓ / 长期 ✗ |
| **Terlipressin** | IV bolus q4-6h | 3-5 天 | 急性出血 / HRS ✓ / 长期 ✗ |
| **Vasopressin** | IV | 24-72 h | 急性出血（已少用，副作用大）|
| **非选 β-blocker** | **口服 PO** | **月 / 年** | **一级 + 二级预防 ✓** |
| **Nitrates (ISMN)** | 口服 | 月 / 年 | 历史上 + β-blocker 联用，**现已弃用**（无获益 + 副作用）|
| **EVL（内镜套扎）** | 操作 | 数次完成 | 急性 + 二级预防 |

##### 为什么 Octreotide / Terlipressin 不能长期用 — 5 个硬限制

| 限制 | 解释 |
|---|---|
| **① 给药途径** | 全部 IV / SC，**无口服剂型** — 病人不能在家天天输液 |
| **② 内脏缺血** | 长期强力收 splanchnic → MI / 肠系膜缺血 / 肢端缺血（坏死）— 急性 3-5 天可忍，长期累积必出事 |
| **③ Tachyphylaxis** | V1 / SST 受体长期激活会**下调** → 效果递减、剂量翻倍、副作用翻倍 |
| **④ 成本** | Terlipressin ~$2000-3000/天 vs nadolol ~$10-20/月，**相差几百倍** |
| **⑤ 无证据** | 长期预防从未通过 RCT；β-blocker 有 **40 年数据**（首次出血率 ↓30-50% + 降死亡率） |

##### 为什么 β-blocker 特别适合长期 — "双机制 + 温和持续"

| 受体 | 机制 | 效果 |
|---|---|---|
| **β1 阻断** | ↓ 心输出量 (CO) | 心脏泵血少 → 流向门脉的血少 |
| **β2 阻断** | 阻 β2 扩血管 → **α1 无对抗** → splanchnic 收缩 | 直接 ↓ 流入门脉血量 |
| **Carvedilol 额外** | **α1 阻断 → 肝内血管舒张** → ↓ R | 同时 ↓ Q 和 ↓ R = P=Q×R 公式**两端都打**，降 HVPG 比 propranolol 强 ~30% ★ |

##### 反向考点：β-blocker 在急性出血时要**停**

> [!warning] 时间维度反转
> 急性出血时 → BP 已低 + 需心率代偿（↑ CO 维持灌注）→ β-blocker **停或减**
> 急性靠 **Octreotide / Terlipressin**（IV 急救）+ EVL
> 出血控制后 BP 稳定 → **重新启动 β-blocker** 做二级预防

##### 类比：扑火 vs 防火

> [!tip] 直觉理解
> - **Octreotide / Terlipressin** = 消防车 + 高压水枪（强力、急用、短期；不能驻扎家里）
> - **β-blocker** = 烟雾报警器 + 自动喷淋（温和、长期、自动化）
> - 两者**不可互换**，是同一逻辑下的不同时间尺度工具

```
            用药逻辑
      ┌──────────┴──────────┐
      ▼                      ▼
   急性救火                 长期值班
  (3-5 天 IV)             (月/年 口服)
      │                      │
  Octreotide              非选 β-blocker
  Terlipressin            (Nadolol / Propranolol / Carvedilol)
      │                      │
  强 + 短期 OK             温和 + 长期安全
  长期: 缺血/耐受/贵       长期: 稳定 / 便宜 / 双机制
```

#### 非选择性 β-blocker — CK 高频陷阱

> [!danger] 为什么必须"非选择性"
> 选择性 β1（如 **metoprolol**）**不阻 β2** → 没有 splanchnic 收缩 → **对门脉高压无效**
> 题干见肝硬化 + 中等以上曲张 → 选 nadolol / propranolol / carvedilol，**绝对不选 metoprolol**

| 药 | 受体 | 备注 |
|---|---|---|
| **Propranolol** | 非选 β1+β2 | 经典 |
| **Nadolol** | 非选 β1+β2 | 半衰期长，一天一次 |
| **Carvedilol** | 非选 β + **α1** | **更强降 HVPG**（α1 阻断 → 肝内血管也舒张 → 同时 ↓Q ↓R）；现指南偏好 ★ |

> [!warning] β-blocker 禁忌（→ 改 EVL 内镜套扎）
> - **难治性腹水**（β-blocker 会 ↓BP，加重肾灌注差）
> - SBP 后、低 BP（SBP < 90）
> - 心率 < 55

→ 详见 [[完整笔记/专题笔记/心内/心内_BetaBlockers分类大全]]

### ② 减少 R（肝实质阻力）→ TIPS / 肝移植

| 操作 | 机制 | 适应症 | 代价 |
|---|---|---|---|
| **TIPS** (Transjugular Intrahepatic Portosystemic Shunt) | 颈静脉穿刺，在肝内造支架把门脉血直接引到肝静脉 → 绕过硬化肝实质 | ① 药物 + 内镜失败的难治性出血 ② 难治性腹水 ③ Budd-Chiari | **肝性脑病 (HE)** 风险大增 ★（血氨绕过肝过滤） |
| **肝移植** | 换新肝 → R 重置 | HRS / 终末期肝病 / MELD 高 | 等待时间、免疫抑制 |

> [!warning] TIPS 禁忌
> - **重度 HE**（会更重）
> - **重度心衰**（突然增加回心血量，急性心衰）
> - 严重肺动脉高压
> - 多发肝囊肿/肝癌广泛

## 四、食管静脉曲张出血 — 全流程 SOP

### 4.1 急性出血处理（最高频考点）

```
诊断疑诊（呕血/黑便 + 肝硬化）
   │
   ▼
① 复苏：两路大孔静脉 + 晶体 / 输血
   目标 Hb 7-8 g/dL（不要过度输血，过度输 → 门脉压反弹 ↑ → 再出血）★
   │
   ▼
② 立即给药（不等内镜）：
   • Octreotide IV bolus + 持续输 2-5 天 ★
   • 预防性抗生素 ★★（ceftriaxone IV × 7 天）
     —— 即使没 SBP 也要给，降低再出血 + 死亡率
   • PPI IV
   │
   ▼
③ 内镜（12 小时内）：
   • Endoscopic Variceal Ligation (EVL) ★ 首选
   • 硬化剂注射（次选）
   │
   ▼
④ 失败 → Balloon tamponade (Sengstaken-Blakemore) 临时桥接
   │
   ▼
⑤ TIPS（早期 TIPS = 高危出血 72h 内做，改善生存）
   │
   ▼
⑥ 出院后 → 二级预防（章节 4.3）
```

> [!danger] 三个"反直觉"必背
> 1. **预防性抗生素必须给**（即使没感染征象）— 降死亡率 ★★
> 2. **输血目标 Hb 7-8 不要更高** — 过度输血 → 门脉压反弹 → 再出血
> 3. **PPI 治静脉曲张出血没机制证据**，但常规给（因为分不清是溃疡还是曲张）

### 4.2 一级预防（有曲张但没出过血）

| 曲张分级 | 处理 |
|---|---|
| 小曲张 | 非选 β-blocker（高风险特征：red wale signs / Child B-C 时） |
| **中-大曲张** | **非选 β-blocker（nadolol/propranolol/carvedilol）** 或 **EVL**（任选） |
| β-blocker 禁忌 | EVL |

### 4.3 二级预防（已经出过一次血）

> [!success] 黄金组合
> **非选 β-blocker + EVL（每 1-4 周一次直到曲张消失）** ★
> 二选一不够，必须**联合**。

## 五、腹水 + SBP（自发性细菌性腹膜炎）

### 5.1 腹水管理阶梯

```
① 限钠 (<2 g/d) + 限水（仅当 Na <125）
② 利尿剂：Spironolactone + Furosemide (100:40 比例)
③ 难治性腹水：
   • 大量放腹水 (LVP) + Albumin 补充
     ★ 每放 1 L → 补 6-8 g albumin（防 paracentesis-induced circulatory dysfunction → HRS）
   • TIPS
   • 肝移植
```

> [!warning] 大量放腹水不补 Albumin → HRS 诱因 ★
> 这是 CK 经典出题点（"为什么这个肝硬化病人放完腹水第二天 Cr 飙升？"）

### 5.2 SBP（Spontaneous Bacterial Peritonitis）

| 项目 | 标准 |
|---|---|
| 诊断 | 腹水 **PMN ≥ 250/mm³**（不需要等培养） |
| 经验治疗 | **Cefotaxime IV** × 5 天 ★ |
| 关键加药 | **Albumin 1.5 g/kg day 1 + 1 g/kg day 3** — 降低 HRS / 死亡率 ★★ |
| 一级预防 | 既往 GI 出血 / 腹水蛋白 <1.5 + 进展期肝病 → **Norfloxacin / Bactrim** 长期口服 |
| 二级预防 | 既往 SBP → **Norfloxacin** 长期口服 |

> [!danger] CK 经典 stem
> 肝硬化 + 腹水 + 发热腹痛 → 立即诊断穿刺 → PMN 250+ → 经验 Cefotaxime + **Albumin**
> （Albumin 在 SBP 是必给的，不是可选，CK 题干常考 next-step 选哪个）

## 六、HRS（肝肾综合征）— 治疗深化

> [!info] HRS = 门脉高压晚期"内脏血管劫持血流"导致的功能性肾衰
> 治疗核心 = **Terlipressin（V1 → splanchnic 收缩 → 把血赶回循环 → 肾灌注恢复）+ Albumin**
> 唯一根治 = **肝移植**
>
> 详细机制 / 诊断 / 分型 / 鉴别 → 见 [[完整笔记/专题笔记/GI/GI_HRS肝肾综合征]]（独立衍生 ✓ 2026-05-28 建成）

### 6.1 UW 病理生理链图 + ASCII 复述

![[{3009AD02-E8A7-4E2E-8364-012ADE092F5D}.png]]

```
Advanced liver disease + Portal HTN
          │
          ▼
内脏 NO / vasodilators 释放↑ → Splanchnic vasodilation
          │
          ▼
血液池在腹腔血管 → 有效动脉血容量 (EABV) ↓
          │
          ▼
SNS + RAAS + ADH 激活 → Renal vasoconstriction + Na/水潴留
          │
          ▼ + 触发事件 (SBP / GI 出血 / 大量放腹水无 albumin)
Overwhelming renal vasoconstriction → 急性 oliguric AKI（HRS Type 1）
          │
          ▼
唯一根治 = Liver transplant
```

> 图同步嵌入自 [[mistakes/uworld-mistakes_2026-05#^Q4752]]（方案 A 双地点）

### 6.2 完整治疗阶梯（6 步）

```
HRS 诊断成立
    │
    ▼
① 去除诱因（SBP / GI 出血 / 停利尿）+ 停 nephrotoxins
    │
    ▼
② Albumin 1 g/kg × 2 天扩容（同时是 prerenal 排除步骤）
    │
    ▼
③ 血管收缩药 + Albumin：
   • Terlipressin（首选 ★ 2022 FDA 批）
   • Midodrine + Octreotide + Albumin（terlipressin 不可及时）
   • Norepinephrine + Albumin（ICU 情境）
    │
    ▼ 药物无反应 + 无 TIPS 禁忌 + 等移植期
④ TIPS（HRS-AKI 二线 / HRS-NAKI 更优）
    │
    ▼ TIPS 失败 / 不能 TIPS
⑤ RRT 桥接（透析维持，等肝移植）
    │
    ▼
⑥ ★ 肝移植 = 唯一根治 ★
```

> [!warning] RRT（血液透析）在 HRS 的角色
> 透析本身**不治 HRS**（不解决 splanchnic vasodilation 问题），但作为**移植前的桥接维持**（防 K↑、酸中毒、容量过负荷）。
> 不要把"透析"当成 HRS 的治疗答案 — 是 holding pattern。

### 6.3 TIPS 在 HRS 中的角色（重要）

> [!info] TIPS 机制：从 R 端釜底抽薪
> HRS 根源 = 门脉高压 → splanchnic 扩张 → EABV ↓ → 肾血管收缩
> TIPS 把门脉血直接引到肝静脉 → **R ↓↓ → 门脉压 ↓**
> → splanchnic 扩张驱动消失 → EABV 恢复 → RAAS/SNS 退场 → **肾血管放松 → GFR 回升**
>
> **证据**：TIPS 后 HRS-AKI 约 40-60% Cr 改善；HRS-NAKI（Type 2）效果更好

#### 为什么 TIPS 不是 HRS 一线（3 原因）

**① 一线药物 + albumin 已经够强** — Terlipressin + albumin 对 60-80% HRS-AKI 有反应，没轮到 TIPS 就在等移植了

**② HRS 病人多数有 TIPS 禁忌** — 终末期肝病几乎都踩这些禁忌：

| TIPS 禁忌 | HRS 人群常见度 |
|---|---|
| **重度肝性脑病 (HE)** | 极常见（TIPS 加重 HE）★ |
| **Total Bilirubin > 5** | 常见 |
| **INR > 2** | 常见 |
| **MELD > 18** | 几乎都符合 |
| **重度心衰 / 肝硬化心肌病** | 常见 |
| **重度肺动脉高压** | — |
| **败血症未控制** | SBP 诱发 HRS 常见 |

**③ TIPS 后 HE 大爆发是真问题** — 氨绕过肝直接进体循环 → HE 风险 +20-40%，HRS 病人脑功能本来已边缘

#### HRS-AKI（Type 1）vs HRS-NAKI（Type 2）的 TIPS 角色不同

| 类型 | TIPS 角色 | 适应症 |
|---|---|---|
| **HRS-AKI（Type 1，急性）** | **二线，证据较弱** | 药物无反应 + 不能马上移植 + 无禁忌 → 桥接 |
| **HRS-NAKI（Type 2，慢性 + 顽固腹水）** | **更可靠**，作用持久 | 顽固腹水 + Type 2 + 无禁忌（移植等待期） |

> [!info] 为什么 Type 2 更适合 TIPS
> Type 2 慢性病程，时间够走 TIPS 评估流程；Type 2 主诉常是顽固腹水，**TIPS 同时解决两个问题**（腹水 + HRS）。

> [!danger] CK 出题视角
> "肝硬化 + HRS-AKI" next-step **优先**：
> 1. Terlipressin + albumin ✅
> 2. Midodrine + octreotide + albumin ✅
> 3. Liver transplant evaluation ✅
> 4. TIPS — 只有题干明确"药物失败 + 无 HE + 等移植"才选
>
> TIPS 在 HRS 选项里**更常见的正确场景**：HRS-NAKI Type 2 + 顽固腹水

## 七、肝性脑病 HE — 速查

| 项目 | 要点 |
|---|---|
| 机制 | 氨经门脉绕过肝 → 中枢氨毒性 |
| 诱因（**必查**） | GI 出血 / 感染（SBP）/ 电解质（**低 K** / 碱中毒）/ 便秘 / **过量利尿** / **TIPS 后** / 镇静药 |
| 一线治疗 | **Lactulose**（结肠酸化 → NH3 → NH4+ 不吸收 + 通便） |
| 二线 / 联合 | **Rifaximin**（不吸收抗生素 → ↓产氨菌） |
| **不查** | **血氨水平 ≠ 严重度** ★ 临床诊断为主，不要被"血氨正常就排除 HE"骗 |

> [!danger] 利尿性 HE — 高频 stem
> 肝硬化 + 腹水 + 用了 spironolactone/furosemide → 突发意识模糊 → 查血 **低 K + 代谢性碱中毒** → ?
> **不是先给 Lactulose！是先补钾！** ★
> 机制：低 K → 肾小管 H⁺/K⁺ 交换增加 → 排 H⁺、留 NH4⁺ → 肾内 NH4⁺ 解离释放 **NH3 进循环** → 加重脑病
> **补 K → 直接逆转氨产生**，是利尿诱发 HE 的初始关键步骤
> 参考 [[完整笔记/Peixuan分科笔记/消化补充内容]] L13

## 八、把所有药放一张表（终极速查）

| 药 / 操作 | 机制 (P=QR) | 主要适应症 |
|---|---|---|
| **Nadolol / Propranolol / Carvedilol** | ↓Q (β2 阻 → splanchnic 收缩) | 一级 + 二级预防 |
| **Octreotide** | ↓Q (抑制扩血管激素) | 急性出血一线 |
| **Terlipressin** | ↓Q (V1 收缩 splanchnic) | HRS / 急性出血 |

| **肝移植** | ↓R (换新肝) | HRS / 终末期 |

| **Norfloxacin** | 抗菌 | SBP 预防 |

| **Spironolactone + Furosemide** | 利尿 | 腹水 |
| **Lactulose / Rifaximin** | ↓血氨 | HE |

## 九、CK 高频陷阱合集

> [!question]- Q1: 肝硬化 + 中等曲张，无出血史，预防用药？
> A: **Nadolol / Propranolol / Carvedilol**（非选 β）
> 陷阱选项：**Metoprolol**（错，β1 选择性无 splanchnic 效应）

> [!question]- Q2: 急性食管静脉曲张出血，内镜前给什么？
> A: **Octreotide IV** + **Ceftriaxone IV**（抗生素必给！）

> [!question]- Q3: 出血控制后 Hb 8.5，要不要输血到 10？
> A: **不要**。目标 Hb 7-8。过度输血 → 门脉压反弹 → 再出血

> [!question]- Q4: 肝硬化 + 腹水 + 发热腹痛，腹穿 PMN 320 →?
> A: **Cefotaxime + Albumin**（Albumin 别漏！）

> [!question]- Q5: 肝硬化大量放腹水第二天 Cr 1.0 → 2.5，机制？
> A: Paracentesis-induced circulatory dysfunction → **HRS-AKI**
> 预防 = 放每 1 L 补 6-8 g albumin

> [!question]- Q6: TIPS 后 1 周出现意识模糊、扑翼样震颤 → 诊断？
> A: **肝性脑病**（TIPS 绕过肝 → 氨直接进体循环）
> 治疗：Lactulose + Rifaximin；必要时关闭 TIPS

> [!question]- Q7: 难治性腹水 + 低 BP 90/55，能用 nadolol 预防曲张吗？
> A: **不能**。β-blocker 在难治性腹水 / 低 BP 是禁忌 → 改 EVL

> [!question]- Q8: 肝硬化病人腹壁见放射状曲张静脉从脐辐射出去 → 诊断？
> A: **Caput medusae** — 脐旁静脉（门脉系统残留）扩张；常伴食管曲张 + 直肠上段曲张 = 门脉高压 triad

> [!question]- Q9: 肝硬化病人查体见痔疮 → 是门脉高压所致吗？
> A: **不一定**。普通痔疮（hemorrhoids）= 直肠下静脉丛，与门脉无关
> 真正的门脉性直肠曲张长在直肠上段（直肠上静脉），表现 = **无痛鲜红便血**

> [!question]- Q10: 既然有奇静脉这条体循环出路，为什么食管曲张还会破裂？
> A: 自然侧支**容量太小 + 阻力太大**：
> - 食管黏膜下静脉丛只能承载几 mL/min，但门脉流量 1500 mL/min
> - **R ∝ 1/r⁴**（1-3 mm vs 13 mm 门脉 → 阻力几百倍）
> - 流量被强行挤入小血管 → 撑爆 = 曲张破裂
> 曲张本身就是侧支"被迫上岗"的物理表现，不是侧支失败

> [!question]- Q11: 为什么 TIPS 能降门脉压而自然侧支不能？
> A: TIPS 直径 **8-10 mm**（人工大口径），R 极低，可承载接近门脉全流量
> 自然侧支 1-3 mm，R 太高，永远卸不掉压

> [!question]- Q12: HRS-AKI 病人 terlipressin + albumin 5 天 Cr 未改善，无 HE，bilirubin 3.5，MELD 16，正在等肝移植。Next step?
> A: **TIPS** 桥接（药物失败 + 无 TIPS 禁忌 + 等移植中 — 三个条件齐全才轮到 TIPS）
> 若有 HE / bilirubin >5 / MELD >18 → 不能 TIPS → **RRT 桥接 + 加速移植**

> [!question]- Q13: HRS-NAKI（Type 2）+ 顽固腹水反复抽液，下一步？
> A: **TIPS** — Type 2 比 Type 1 更适合 TIPS，且**同时解决腹水和 HRS** 两个问题

> [!question]- Q14: 中等曲张未出血史，**长期预防**用什么？为什么不用 octreotide 长期 SC？
> A: **非选 β-blocker**（口服，月/年）
> Octreotide 不能长期用 5 原因：① 只能 IV/SC 无口服 ② 长期 splanchnic 收缩致内脏缺血 ③ 受体下调耐受 ④ 成本 $200-500/天 ⑤ 无 RCT 证据

> [!question]- Q15: 肝硬化病人平时口服 nadolol 预防曲张。今天突发呕大量鲜红血，BP 88/52。继续 nadolol 吗？
> A: **停或减 nadolol** ★
> 急性出血时 BP 已低 + 需心率代偿，β-blocker 反而有害；急性靠 **Octreotide IV + Ceftriaxone + EVL**；出血控制后 BP 稳定再**重启 β-blocker** 做二级预防

## 十、记忆口诀

> [!tip] 三个 "P"
> - **Propranolol/Nadolol/Carvedilol** = **P**revention（预防）
> - **Procedure (EVL)** = 急性 + 预防
> - **Pressure relief (TIPS)** = 难治时绕过

> [!tip] SBP 一定要"双打"
> **Cefotaxime + Albumin** —— 任何 SBP 题选 next-step，Albumin 不能漏

> [!tip] 输血"低目标"
> 静脉曲张出血 Hb 目标 **7-8**（不是 10）—— 反直觉 ★

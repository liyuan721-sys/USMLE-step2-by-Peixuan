# GI_原发性硬化性胆管炎 PSC（串珠样胆管 / UC 强关联 / 胆管癌风险）

> [!info] 这张的定位
> PSC 此前只在 [[完整笔记/专题笔记/GI/GI_HighYield速查总览]] §胆道表里占一行。本篇做深度展开（诊断路径 / 与 PBC 鉴别 / 并发症监测 / 治疗边界），图也归这里。
> **PSC** = primary sclerosing cholangitis 原发性硬化性胆管炎；**PBC** = primary biliary cholangitis 原发性胆汁性胆管炎。

## 一句话核心

**肝内外胆管的慢性纤维化炎症 → 多灶性狭窄 + 节段扩张（"串珠样 beaded"）**；典型 = **中年男性 + 溃疡性结肠炎 (ulcerative colitis, UC) + 碱性磷酸酶 (alkaline phosphatase, ALP) ↑**；进展性、无内科根治、**胆管癌 (cholangiocarcinoma) 风险高**。

## 一、识别钩子（UWorld stem 模式）

> [!tip] 一眼锁定
> **年轻/中年男性 + 已知 UC + 疲乏/瘙痒 + 胆汁淤积型肝功（ALP/GGT↑、胆红素可正常）→ MRCP 串珠样 = PSC。**
> 反向陷阱：**中年女性 + 瘙痒 + 抗线粒体抗体 (AMA) 阳性 + 小胆管 = PBC**（别混）。

- **人群**：男 > 女，30–50 岁；**70–80% 合并炎症性肠病 (inflammatory bowel disease, IBD)，绝大多数是 UC**（但 PSC 病程与肠病活动**不平行**，结肠切除也不阻止 PSC 进展）。
- **症状**：早期常**无症状**，仅 ALP 升高被发现；进展期 → 乏力、**瘙痒 (pruritus)**、黄疸、右上腹不适；反复**细菌性胆管炎**多见于显性狭窄或胆道器械操作后。

## 二、诊断路径

```
胆汁淤积型肝功（ALP↑↑ + GGT↑，± 胆红素↑）
        │
   影像（首选无创 MRCP）
        │
   多灶性狭窄 + 节段扩张 = 串珠样 (beading) ──► PSC
        │
   辅助：p-ANCA 常阳（非特异）；IgG4 必查（排 IgG4 相关胆管病）
        │
   ERCP 仅在需取活检 / 球囊扩张 / 支架时用（有创、有胰腺炎风险）
```

- **MRCP（磁共振胰胆管造影 magnetic resonance cholangiopancreatography）= 首选**：无创，见**肝内外胆管多灶狭窄 + 扩张交替**。
- **ERCP（内镜逆行胰胆管造影 endoscopic retrograde cholangiopancreatography）**：影像等效但有创 → **保留给治疗性操作或取细胞学**。
- **生化**：**ALP↑↑ + GGT↑**（确认胆源性，非骨源）；胆红素早期可正常。
- **抗体**：**p-ANCA（核周型抗中性粒细胞胞浆抗体）常阳但不特异**；**AMA 阴性**（阳性想 PBC）；查 **IgG4** 排除激素敏感的 IgG4 相关性胆管炎。
- **活检**：非必需；经典"洋葱皮样 (onion-skin) 胆管周围纤维化"，但分布不均、诊断主要靠影像。

*PSC 的 MRCP：肝内外胆管**多灶性狭窄与节段性扩张交替 = 串珠样 (beaded) 改变**（图中节段性狭窄 + 扩张）。*

> [!warning] PSC vs PBC vs 继发性硬化性胆管炎
> - **PSC**：男、UC、大+小胆管、串珠样 MRCP、p-ANCA、胆管癌风险。
> - **PBC（原发性胆汁性胆管炎）**：**中年女、AMA 阳、小胆管、顽固瘙痒**，治疗 ursodiol 有效。
> - **继发性硬化性胆管炎**：胆道结石/术后狭窄/缺血/复发性化脓性胆管炎等**有明确诱因**。

## 三、并发症（高频考点）⭐

| 并发症 | 关键点 |
|---|---|
| **胆管癌 cholangiocarcinoma** ⭐ | 终身约 10–15%；**CA19-9 + MRCP/ERCP 监测**；新发快速黄疸/显性狭窄要警惕 |
| **结直肠癌 colorectal cancer** | PSC + UC 患者风险**更高** → **确诊即起每年结肠镜监测**（早于单纯 UC 的间隔）|
| **胆汁淤积后果** | **脂溶性维生素 (ADEK) 缺乏**、骨质疏松/骨软化、顽固瘙痒 |
| **反复细菌性胆管炎** | 多见于显性狭窄或 ERCP 操作后 |
| **胆汁性肝硬化 → 门脉高压** | 终末期，进展为肝衰 |

## 四、治疗边界

- **无内科根治**。**ursodiol（熊去氧胆酸 ursodeoxycholic acid）**改善生化指标，但**不改善生存、不作常规推荐**；**高剂量 ursodiol 有害**（增不良结局），别选"大剂量熊去氧胆酸"。
- **显性狭窄 (dominant stricture)** → **ERCP 球囊扩张 ± 支架**（同时刷检排胆管癌）。
- **瘙痒** → **cholestyramine（考来烯胺）**一线。
- **终末期 / 早期胆管癌符合标准** → **肝移植 (liver transplant) = 唯一延长生存的手段**（移植后仍可复发）。
- **监测套餐**：胆管癌（CA19-9 + 影像）、结肠镜（合并 IBD）、骨密度、脂溶性维生素。

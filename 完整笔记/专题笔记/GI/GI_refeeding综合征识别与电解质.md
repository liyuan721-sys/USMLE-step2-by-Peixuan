# GI_再喂养综合征 refeeding syndrome 识别与电解质

> [!tip] 一句话锁定
> **长期营养不良 / 恶病质**的病人**刚开始进食或肠内营养（PEG / TPN）后 24–72 小时内**，最该警惕、最招牌的实验室异常是 **低磷血症 hypophosphatemia**（核心），常伴**低钾 hypokalemia + 低镁 hypomagnesemia**。这是 **再喂养综合征 refeeding syndrome** 的定义性表现。

---

## 一、为什么"营养终于开始了"反而危险

NBME 常把 stem 包装成"好消息"——食管癌病人放了 PEG 管、终于开始肠内营养——真正的考点却是：**你有没有意识到 refeeding can kill**。

机制链（胰岛素是开关）：

```text
慢性饥饿 (chronic starvation)
  → 胰岛素↓、全身磷/钾/镁总储备已被耗竭（血值可仍正常）
  → 重新喂养（尤其碳水）
  → 胰岛素↑↑（surge）
  → 葡萄糖 + 磷 / 钾 / 镁一起被驱动进入细胞
     （用于 ATP、2,3-DPG、糖原、蛋白合成）
  → 血浆磷 / 钾 / 镁骤降
  → 低磷血症最致命
```

> [!warning] 关键陷阱：入院时电解质"正常"不能排除
> 饥饿期总储备已空，但血值可正常。**是"喂养"这个动作触发了细胞内转移**，才让血值崩塌。因此危险窗口在**喂养开始之后**，不是入院当时。

## 二、三大电解质与后果

表格前空行。

| 电解质异常 | 频率 / 价值 | 后果 |
|---|---|---|
| **低磷 hypophosphatemia** | 最经典 ⭐ | ATP / 2,3-DPG 不足 → 心衰、呼吸肌无力（脱机失败）、横纹肌溶解、溶血、癫痫 |
| **低钾 hypokalemia** | 常见 | 心律失常、肌无力、肠麻痹 |
| **低镁 hypomagnesemia** | 常见 | 心律失常、神经肌肉激惹；低镁使低钾 / 低钙难纠 |
| 维生素 B1 (thiamine) ↓ | 伴随 | 补糖前不补 B1 → 诱发 Wernicke 脑病 |

## 三、高危人群 + 预防

- **高危**：神经性厌食 anorexia nervosa / 长期酒精滥用 / 减重手术后 / 慢性消耗（癌症恶病质、食管癌摄食受阻）/ 长期禁食或住院 / 化疗后恢复 / 战俘。
- **预防（考点常问"下一步"）**：
  - 缓慢起始热量（约 10–20 kcal/kg/d），逐步加量；
  - 喂养**前 + 喂养中**监测并补充**磷 / 钾 / 镁**；
  - 补糖**前先补 thiamine（B1）**；
  - 密切监测电解质（尤其头几天）。

> [!question]- 恶病质病人放 PEG 后 36 小时内，最该预期哪个实验室异常？机制？
> **低磷血症 hypophosphatemia**。饥饿后再喂养 → 胰岛素↑ → 磷（连同钾、镁）随葡萄糖被驱入细胞用于 ATP / 糖原 / 蛋白合成 → 血磷骤降 → 心衰 / 呼吸肌无力。

> [!question]- 为什么入院时电解质正常，仍可能发生 refeeding syndrome？
> 饥饿期全身储备已耗竭但血值靠转移代偿仍正常；**喂养触发胰岛素驱动的细胞内转移**才使血值崩塌。危险在喂养之后，不是入院当时。

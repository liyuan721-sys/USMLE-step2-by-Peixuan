## 一、食管（Esophagus）

### 1.1 吞咽困难 dysphagia 分流（首检三选一）

```
报警症状(消瘦/贫血/进行性 固体→液体) → EGD（上消化道内镜 esophagogastroduodenoscopy/胃镜，排恶性）
口咽性启动困难(卒中/帕金森/重症肌无力 MG、呛咳鼻反流) → VFSS（电视荧光吞咽造影 videofluoroscopic swallow study）
Zenker 征象(口臭/反流/颈包块) / 术后/烧伤狭窄史 → 钡剂造影（盲插胃镜有穿孔风险）
```

> [!tip] 动力障碍 vs 结构障碍（钡剂特征 = 钥匙）
> - **食管失弛缓 achalasia（贲门失弛缓）**：固体 + 液体**同时**困难（早期、非进行性）；钡剂 **bird's beak 鸟嘴**；金标准 **测压 manometry → LES 压力↑ + 蠕动↓**；成人病因 Chagas（**苄硝唑 benznidazole/硝呋替莫 nifurtimox**）；**首选治疗气囊扩张 pneumatic dilation（best）**，次选肉毒素 botulinum toxin。
>
> ![[{A955E256-1E92-431E-BB79-44319FFD9BCA}.png]]
> *贲门失弛缓 Achalasia 钡餐：鸟嘴征 bird-beak narrowing（LES 松弛障碍 + 食管体蠕动消失）*
> - **硬皮病 scleroderma 食管受累**：下食管括约肌 LES (lower esophageal sphincter) 压力**↓** + 严重 GERD（胃食管反流病，与 achalasia 镜像）；终身 PPI（质子泵抑制剂 proton pump inhibitor）；肾危象 → ACEI（血管紧张素转换酶抑制剂）。
> - **弥漫性食管痉挛 DES (diffuse esophageal spasm)**：胸痛 + 吞咽困难，**冷饮诱发**；钡剂 **corkscrew 螺旋/串珠**；治疗 CCB（钙通道阻滞剂 calcium channel blocker）。

| 结构病 | 典型位置 | 解剖层级 | 钥匙 |
|---|---|---|---|
| Zenker 憩室 | 咽食管交界(后壁) | 黏膜+黏膜下(假性) | 口臭 halitosis + 反流未消化食物 + 误吸；钡剂诊断、喉镜最准、手术 |
| 食管蹼 web (Plummer-Vinson) | 上段 | 仅黏膜 | 缺铁性贫血三联 → 补铁 + 维生素 C；鳞癌风险↑ |
| Schatzki 环 | 下段 | 黏膜+黏膜下 | 间歇性固体吞咽困难 + 裂孔疝；扩张 + PPI |

### 1.2 吞咽痛 odynophagia（感染性食管炎）

```
免疫正常 + 无用药史 → EGD（无背景的痛必动镜，排恶性/糜烂）
HIV/鹅口疮背景 → 经验性氟康唑 fluconazole（白色豆渣 = 念珠菌 Candida，最常见）
```

- **CMV（巨细胞病毒）**：单个**大而线状**溃疡 → 更昔洛韦 ganciclovir。
- **HSV（单纯疱疹病毒）**：**多个小**溃疡（火山口状）→ 阿昔洛韦 acyclovir。
- **药物性食管炎**（四环素 tetracycline / 双膦酸盐 bisphosphonate / 铁 / 钾 / NSAIDs 非甾体抗炎药）：随水服 + 服后直立 30–60 min。

### 1.3 食管撕裂 + 癌

> [!danger] Mallory-Weiss vs Boerhaave
> - **Mallory-Weiss（黏膜撕裂）**：剧烈呕吐后呕血 → IV 补液，多数自愈，持续出血 → 内镜治疗。
> - **Boerhaave（全层破裂）**：呕吐 + 剧烈胸痛 + 纵隔增宽 + **左侧胸腔积液**；初查 CXR，最准 = 水溶性造影（稳定）/ CT（不稳定快）；治疗 = IV 补液 + 抗生素 + **急诊手术**。

- **食管癌**：进行性吞咽困难(固体→液体) + 消瘦，50–60 岁；初查钡剂，最准 EGD + 活检，CT 分期。**鳞癌** = 吸烟/酒/Plummer-Vinson，近中段；**腺癌** = Barrett，远段。
- **Barrett 食管**监测：无异型 2–3 年；低度异型 6 个月；高度异型 → 切除。GERD 监测每 5 年镜检。

![[{37BE0E64-1A0D-4BD1-8D50-017E6DF30771}.png]]
*Barrett 食管 Barrett esophagus：鳞柱交界 squamocolumnar junction 上移、远端肠化生（GERD 慢性并发 → 腺癌前病变）*

## 二、胃 + PUD（Stomach & Peptic Ulcer Disease）

### 2.1 GERD 与 Nissen 术后

- **GERD（胃食管反流病 gastroesophageal reflux disease）**：烧灼胸痛 + 平卧加重 + 反酸；临床诊断（不清 → pH 监测）；生活方式 + PPI；难治 → Nissen 胃底折叠术。
- **婴儿 GERD / 反流食管炎 reflux esophagitis**：生理性吐奶（无痛、生长好、自限）vs **反流食管炎**（喂后**反弓 + 疼痛** = 食管黏膜受累）→ 改喂养（少量多次 / 喂后竖抱 / 增稠）± 试 PPI；鉴别牛奶蛋白不耐受（**血便 / 腹泻 / 生长差**）、幽门狭窄（**喷射性非胆汁呕吐**，3–6 周龄）。〔源：NBME11A Q175 → [[NBME11_错题本#^Q175]]〕
- **Nissen 术后三并发症**：吞咽困难（LES 过紧，12 周内自愈）/ 气胀综合征 gas-bloat（不能打嗝 → 西甲硅油 simethicone）/ **胃轻瘫**（迷走神经损伤 → 早饱呕吐 + **看到体重↓要警惕**）。

![[{A042BF0F-8999-4381-BBCA-E613C0A49F15}.png]]
*食管裂孔疝 Hiatal hernia：滑动型 sliding(GE 交界上移，最常见) vs 食管旁型 paraesophageal(胃底疝入、可嵌顿)*

### 2.2 胃炎 + H. pylori（幽门螺杆菌）

- **A 型（自身免疫）**：胃底/体；抗壁细胞抗体 → 内因子↓ → B12 缺乏 + 无酸 achlorhydria。
- **B 型（最常见）**：胃窦；H. pylori / NSAIDs。
- **检测**：粪抗原 / 尿素呼气试验 / IgG 血清。**4-4-1 复查铁律**：停抗生素 **4 周** + 停 PPI **2 周**（不停 → 假阴性）。
- **治疗**：三联 阿莫西林 amoxicillin + 克拉霉素 clarithromycin + PPI；青霉素过敏 → 甲硝唑 metronidazole + 克拉霉素 + PPI；四联 = PPI + 铋 bismuth + 甲硝唑 + 四环素。

> [!tip] 胃溃疡 GU vs 十二指肠溃疡 DU（进食影响 = 钥匙）
> - **胃溃疡**：进食后**加重** + 消瘦；NSAIDs > H. pylori；血型 A；**有恶变风险 → 必须活检**。
> - **十二指肠溃疡**：进食后**缓解**、夜间痛 + 体重增；H. pylori(90%+) > NSAIDs；血型 O；极少恶变。

### 2.3 PUD 并发症（IHOP）+ ZES

| 并发症 | 钥匙 | 下一步 |
|---|---|---|
| 穿孔 Perforation | 腹膜炎、突发 | **立位胸/腹 X 线**（膈下游离气体）→ 阴性高度怀疑则 CT |
| 出血 Bleeding | 呕血、黑便 | **EGD 内镜** |
| 梗阻 Obstruction | 振水音、呕吐 | NG 管 + 内镜 |

- **手术指征 IHOP**：Intractable 顽固痛 / Hemorrhage 出血 / Obstruction 梗阻 / Perforation 穿孔。
- **胃泌素瘤 ZES（卓-艾综合征 Zollinger-Ellison syndrome）**：顽固/多发溃疡 + 慢性腹泻**脂肪泻**（过量酸使胰脂肪酶酸性失活）。空腹胃泌素 >1000 pg/mL + 胃液 pH<2 确诊；中度升高(110–1000) → **促胰液素激发试验 secretin test → 悖论性升高**；定位 EUS（超声内镜 endoscopic ultrasound）/ 奥曲肽扫描 OctreoScan；切除 + 终身 PPI + 排除 **MEN1（多发性内分泌肿瘤 1 型 multiple endocrine neoplasia type 1）**。
- **胃癌**：东亚移民 + 上腹痛/消瘦 + **缺铁性贫血**；肝大 + ALP↑ → 想肝转移；Virchow 结（左锁骨上）= GI 恶性信号。

## 三、动力 / 功能性（Motility & Functional）

- **胃轻瘫 gastroparesis**：糖尿病（最常见，迷走神经病变）/ 迷走损伤 / GLP-1 激动剂；早饱 + 恶心呕吐；**胃排空显像 gastric scintigraphy**（4h 残留 >10% 确诊）；低脂 + 高碳水蛋白 + **甲氧氯普胺 metoclopramide / 红霉素 erythromycin**。
- **倾倒综合征 dumping**（减重术后）：**早期拉水**（饭后 10–30 min，高渗 → 液体入肠腔 → 腹泻头晕心悸）；**晚期低糖**（1–3h，糖吸收过快 → 胰岛素过多 → 反应性低血糖）；少食多餐 + 高蛋白脂 + 低碳水，重者奥曲肽 octreotide。
- **IBS（肠易激综合征 irritable bowel syndrome）**：Rome IV 标准 = 反复腹痛 ≥1 天/周 × 3 月 + ≥2（与排便相关/性状/频率改变）；**除外性诊断**，先 CBC（全血细胞计数），腹泻型加粪培养 + celiac（乳糜泻）+ CRP（C 反应蛋白）+ 钙卫蛋白 calprotectin。
- **类癌综合征 carcinoid**：**潮红 + 腹泻 + 哮鸣** + 烟酸 niacin 缺乏（糙皮病 pellagra 3D）；**右心**瓣膜病（5-HT 经肺 MAO 灭活，左心免）；查 **尿 5-HIAA**，定位 CT + OctreoScan；治疗奥曲肽。

## 四、小肠吸收不良（Malabsorption）

> [!tip] D-木糖试验 D-xylose（定位钥匙）
> 不需胰酶、只靠小肠黏膜吸收。**尿 D-xylose 正常 = 胰腺病**（吸收完好）；**低 = 小肠病**（celiac/sprue）；SIBO 会代谢掉 → 也低（假阳性）。

- **乳糜泻 celiac（口炎性腹泻 celiac disease）**：脂肪泻 + 疱疹样皮炎 dermatitis herpetiformis + 缺铁贫血 + 矮小/青春期延迟；**抗组织转谷氨酰胺酶 IgA 抗体 tTG-IgA**（+ 总 IgA）；**IgA 缺乏（celiac 常见）→ tTG-IgA 假阴性 → 改查 tTG-IgG / DGP（脱酰胺麦胶蛋白肽 deamidated gliadin peptide）**；最准小肠活检（绒毛萎缩）；无麸质饮食 + **氨苯砜 dapsone** 治疱疹样皮炎。
- **热带口炎性腹泻 tropical sprue**：旅行史 + celiac 样 → 四环素 tetracycline / TMP-SMX（复方磺胺甲噁唑 trimethoprim-sulfamethoxazole）长程。
- **Whipple 病 Whipple disease**：吸收不良 + 神经 + 关节痛；**PAS（过碘酸希夫染色 periodic acid-Schiff）阳性巨噬细胞** → TMP-SMX 长程。
- **SIBO（小肠细菌过度生长 small intestinal bacterial overgrowth）**：解剖异常(狭窄/Roux-en-Y)/动力(硬皮病)/PPI；**维生素 B12↓ + 叶酸↑/维生素 K↑**；碳水呼气试验 → **利福昔明 rifaximin**。

> [!info] 回肠末端病变（克罗恩/切除）→ 胆汁酸的微妙阈值
> - 切除 **<100 cm**：胆汁酸吸收↓入结肠 → **胆汁酸性腹泻/水样泻**。
> - 切除 **>100 cm**：胆盐池枯竭 → **脂肪泻 + 脂溶性维生素缺乏**。
> - 末端回肠损伤同时致 **草酸钙肾结石**（enteric hyperoxaluria）+ 胆固醇胆结石。

## 五、IBD（炎症性肠病 inflammatory bowel disease）：克罗恩 vs 溃结

| 维度 | 克罗恩 Crohn | 溃疡性结肠炎 UC |
|---|---|---|
| 分布 | 跳跃 skip、回肠±结肠、全层 | 连续(直肠→近端) |
| 便 | 腹泻 + 吸收不良 | 血性腹泻 |
| 抗体 | ASCA | p-ANCA |
| 钡剂 | string sign / 鹅卵石 / creeping fat | lead pipe 铅管 |
| 活检 | 非干酪样肉芽肿 | 隐窝脓肿 crypt abscess |
| 并发 | 肾结石、胆结石、瘘 | — |

> ASCA = 抗酿酒酵母抗体 anti-Saccharomyces cerevisiae antibodies（克罗恩）；p-ANCA = 核周抗中性粒细胞胞浆抗体 perinuclear anti-neutrophil cytoplasmic antibodies（溃结）。

- **治疗阶梯（按疾病严重程度升级）**：

| 严重程度 | 首选药物 | 关键点 / 注意 |
|---|---|---|
| **轻度（结肠型）** | 5-ASA（5-氨基水杨酸 5-aminosalicylic acid）：柳氮磺吡啶 sulfasalazine / 美沙拉嗪 mesalamine | 轻度结肠型一线；UC（溃疡性结肠炎）也用于维持缓解 |
| **中度 / 急性发作** | 糖皮质激素 corticosteroid：布地奈德 budesonide / 泼尼松 prednisone | budesonide 局限回肠/右结肠（首过代谢高、全身副作用少）；prednisone 病变范围广用；**仅诱导缓解、不用于维持** |
| **难治 / 中重度** | 生物制剂 biologic "-mab" | 抗 TNF（肿瘤坏死因子 tumor necrosis factor）：英夫利昔 infliximab / 阿达木 adalimumab；抗 IL-12/23（白介素 interleukin）：乌司奴 ustekinumab；抗整合素 integrin：维多珠 vedolizumab。**启动前先查 PPD（结核菌素皮试 tuberculin skin test）/IGRA（γ-干扰素释放试验 interferon-gamma release assay）排结核** |

- **生物制剂监测（前 vs 后）⭐**：**启动前**查 PPD/IGRA 排结核 + 乙肝/丙肝血清学（抗 TNF 可重激活）；**启动后**查 **CBC（complete blood count，全血细胞计数）**——抗 TNF（如 infliximab 英夫利昔）可致**贫血（anemia，约 10%）**，合用 **azathioprine（硫唑嘌呤）有全血细胞减少（pancytopenia）**风险；肝功（LFTs）可监测但优先级低于 CBC（NBME 高频：1 月随访首选 CBC，不是 LFT）。关联错题 [[NBME11_错题本#^F15]]。
- **IBD 癌症筛查**：确诊 **8 年后**起，此后**每年结肠镜**。

## 六、感染性腹泻（Infectious Diarrhea）

> [!tip] 血性腹泻病原（"一碗啊"助记）
> **E**HEC / **Sh**igella / **Sa**lmonella / **Y**ersinia / **C**ampylobacter / **E**ntamoeba。

- **Campylobacter（弯曲杆菌）**：未熟禽肉；并发 **吉兰-巴雷综合征 GBS (Guillain-Barré syndrome)**；**轻症仅支持（自限<7天），抗生素只给重症/高危（阿奇霉素 azithromycin，非氨苄/多西）**——药敏阳 ≠ 治疗指征。
- **EHEC（肠出血性大肠杆菌 enterohemorrhagic E. coli）**：未熟牛肉 → **HUS（溶血尿毒综合征 hemolytic uremic syndrome）**；忌经验抗生素（增 HUS）。
- **Giardia（贾第鞭毛虫 Giardia lamblia）**：野外淡水 → 油腻恶臭漂浮便 → **甲硝唑 metronidazole**。
- **病毒**：轮状病毒 rotavirus（小儿，疫苗 2/4/6 月）/ 诺如 norovirus（游轮暴发）。
- **快速食物中毒(<8h)**：金葡菌 / 蜡样芽孢杆菌 / 产气荚膜梭菌。
- **C. difficile（艰难梭菌）**：抗生素/PPI/老年；水样泻 + 白细胞↑；**粪 PCR/EIA（酶免疫法 enzyme immunoassay）查毒素**；非重症单药 **非达霉素 fidaxomicin / 口服万古霉素 vancomycin**；重症（暴发/中毒性巨结肠/休克）**口服万古 + IV（静脉）甲硝唑**。中毒性巨结肠 = 突然不泻但腹胀恶化 + 发热白细胞↑ → **腹部平片** + 禁食减压。

## 七、下消化道（结直肠 / 肛门 / 梗阻）

### 7.1 结直肠癌筛查 CRC (colorectal cancer)

```
一般人群 → 45 岁起，每 10 年结肠镜
一级亲属 → 40 岁或亲属确诊年龄前 10 年起，每 5–10 年
```

- **遗传综合征**：FAP（家族性腺瘤性息肉病 familial adenomatous polyposis；10 岁起筛、青春期镜检）；Gardner = 息肉 + 骨瘤；Turcot = 息肉 + 脑瘤；**Lynch/HNPCC（遗传性非息肉病性结直肠癌 hereditary nonpolyposis colorectal cancer）**（错配修复基因 MLH1/MSH2/MSH6/PMS2；3-2-1 规则 = 3 亲属/2 代/1 例<50 岁；21 岁起每 2 年）；Peutz-Jeghers（STK11、色素黏膜、胰/卵巢癌、肠套叠风险）。
- **钥匙**：**S. gallolyticus（牛链菌）/ C. septicum 菌血症 → 结肠镜找结肠癌**（S. bovis 还查感染性心内膜炎）。

### 7.2 憩室 / 阑尾 / 痔 / 梗阻

- **憩室病 diverticulosis**：**无痛便血** → 结肠镜 + 高纤维。
- **憩室炎 diverticulitis**：发热 + 左下腹痛 → **CT（口服+静脉造影）** + 肠道休息 + 抗生素（环丙 + 甲硝唑）；缓解 **6–8 周后**结肠镜。
- **阑尾炎**：脐周→右下腹痛；最特异 CT；手术（穿孔先保守）。
- **痔**：内痔(齿状线上、无痛、便血/脱垂)→ 套扎 banding；外痔(齿状线下、痛)→ 坐浴，**<72h 剧痛血栓 → 痔切除**。
- **肠梗阻**：小肠梗阻 SBO (small bowel obstruction) 最常见**粘连** → NPO（禁食）/NG 管（鼻胃管）/IV 补液/必要时手术；大肠 → X 线。
- **急性结肠假性梗阻 Ogilvie 综合征 (acute colonic pseudo-obstruction)**：高龄 / 术后 / 创伤 / 阿片 (opioid) / 重症 → 自主神经失调 → **全结肠连续弥漫扩张（盲肠到肛门、无移行带）但无机械性梗阻**；直肠指检空（vs 粪便嵌塞 fecal impaction 可触硬便）。处理：排机械梗阻 + 纠电解质 / 停阿片 + NG 减压 → 盲肠 >12 cm 或保守失败 → **新斯的明 neostigmine（拟胆碱）** → 结肠镜减压。鉴别钥匙：机械梗阻有**移行带**、Ogilvie 连续扩张；缺血性结肠炎 (ischemic colitis) 有便血绞痛。〔源：NBME11A Q104 → [[NBME11_错题本#^Q104]]〕

### 7.3 便秘 + 阿片性便秘 OIC (opioid-induced constipation)

- **OIC 一线**：刺激性泻药（番泻叶 senna / 比沙可啶 bisacodyl）+ 渗透性（聚乙二醇 PEG / 乳果糖 lactulose）；难治二三线 = **PAMORA（外周 μ 阿片受体拮抗剂 peripherally acting μ-opioid receptor antagonist；甲基纳曲酮 methylnaltrexone）**（不过血脑屏障，不减镇痛）。
- **溢出性失禁 overflow incontinence**：长期阿片 → 粪嵌塞 → 稀便绕流 → **首选直肠指检 DRE (digital rectal examination) 确诊** → 手抠 + 灌肠。

### 7.4 肠系膜缺血 mesenteric ischemia

> [!danger] 钥匙：pain out of proportion（痛与查体不成比例）
> 急性 = 房颤 atrial fibrillation 栓子 / 动脉粥样硬化，突发剧痛、迟发便血 + 乳酸↑代酸；慢性 = 餐后腹绞痛 abdominal angina（数月）。诊断 **CTA（CT 血管造影 CT angiography）**；静脉型 → 肝素 heparin，动脉型 → 罂粟碱 papaverine。

## 八、上 / 下消化道出血（GI Bleed）

```
呕血/黑便(上消化道) → 复苏 2 大孔径 IV(静脉通路) → EGD
血便(下消化道，多为憩室/血管发育不良) → 复苏 → 结肠镜
输血阈值：Hb(血红蛋白 hemoglobin) < 7 g/dL（静脉曲张出血同此门槛）
```

- **静脉曲张 esophageal varices（食管静脉曲张）**（门脉高压）：大量呕血 + 肝衰征。急性 = **IV 补液 + 奥曲肽 octreotide（降门脉压）+ 抗生素（头孢曲松 ceftriaxone，降 SBP/死亡率）** → 稳定后 12h 内 **EVL（内镜下静脉曲张套扎 endoscopic variceal ligation）**；难治 → **TIPS（经颈静脉肝内门体分流术 transjugular intrahepatic portosystemic shunt；肝性脑病时避免）**。一级预防：中/大曲张 → **非选择性 β 阻滞剂 NSBB (nonselective beta blocker)（纳多洛尔 nadolol/普萘洛尔 propranolol）** 或 EVL（代偿期可用卡维地洛 carvedilol）。二级预防 = NSBB + EVL 联合。预后标志 = INR/PT（凝血酶原时间）上升。
- **Dieulafoy 病变 Dieulafoy lesion**：胃近端恒径动脉破裂 → 间歇大量呕血，内镜见孤立搏动出血点。
- **血管发育不良 angiodysplasia**：与 ESRD（终末期肾病 end-stage renal disease）/ vWD（血管性血友病 von Willebrand disease）/ 主动脉狭窄 AS (aortic stenosis) 相关（下消化道隐匿出血）。

## 九、肝（Liver）

### 9.1 肝硬化腹水 + SAAG

> [!tip] SAAG（血清-腹水白蛋白梯度）= 门脉高压钥匙
> **SAAG ≥1.1** = 门脉高压（肝硬化 / 心衰 / Budd-Chiari）；**SAAG <1.1** = 非门脉高压（恶性 / 结核 / 肾病综合征）。
> 评估**新发腹水病因/排肿瘤 → 超声**；排**感染 SBP → 穿刺**。

- **腹水治疗三部曲**：戒酒 + 限钠(<2000 mg/d) + **螺内酯 spironolactone + 呋塞米 furosemide 联合**；避免 ACEI/ARB/NSAIDs；难治 → 大量放腹水 + TIPS。
- **SBP（自发性细菌性腹膜炎 spontaneous bacterial peritonitis）**：肝硬化 + 发热/腹痛/AMS（精神状态改变 altered mental status，**意识模糊可为唯一表现，必须穿刺**）；**穿刺 PMN（中性粒细胞 polymorphonuclear leukocytes）≥250/mm³**（E. coli/Klebsiella）；**培养回报前即开始 头孢噻肟 cefotaxime/头孢曲松 ceftriaxone**；Cr>1 或 BUN>30 或 Bili>4 → 加 **IV 白蛋白 albumin**（防肝肾综合征 HRS）；长期预防氟喹诺酮 fluoroquinolone。
- **肝性脑病 HE（hepatic encephalopathy）**：诱因 = 利尿剂致**低钾/碱中毒**、感染(SBP)、GI 出血、镇静剂；**见 HE 先找诱因，有低钾必先补钾**（减少肾产氨）+ 容量复苏；降氨 **乳果糖 lactulose（一线）+ 利福昔明 rifaximin**；难治新霉素 neomycin / 肝移植。

🧩 HRS（肝肾综合征 hepatorenal syndrome）机制 + 处理深挖 → [[完整笔记/专题笔记/GI/GI_HRS肝肾综合征]]

### 9.2 胆汁淤积性肝病 + 铜铁沉积

### 9.4 病毒性肝炎 + HCC + Budd-Chiari

- **HBV（乙型肝炎病毒 hepatitis B virus）血清学**：核心抗体 + 表面抗体 = 恢复；**仅核心抗体 = 窗口期**；仅表面抗体 = 已接种。

## 十、胰腺（Pancreas）

### 10.1 急性胰腺炎 acute pancreatitis

> [!tip] 诊断 = 3 取 2
> ① 急性上腹痛放射至背（前倾缓解）② **脂肪酶/淀粉酶 >3 倍上限**（脂肪酶更特异、持续更久）③ 影像符合。
> **ALT >150 → 胆源性**；**BUN 升高 = 容量复苏不足 + 坏死风险**（酶诊断、BUN 看命）。

- **病因**：胆石(~40%) / 酒精(~40%) / **甘油三酯 TG >1000** / 药物（硫唑嘌呤 azathioprine、丙戊酸 valproic acid、噻嗪 thiazide）/ post-ERCP / **钝性腹外伤 (blunt trauma，儿童自行车把手、成人挤压)** / 高钙。

> [!warning] 外伤性胰腺炎 + 酶判读陷阱（NBME11 Q008）
> 钝性上腹外伤(胰腺被挤向脊柱)是病因之一;**酶可不典型——可只有淀粉酶↑而脂肪酶仍正常**,别因"脂肪酶不高"排除,结合机制 + 临床判断;影像 CT 评估损伤分级。源:[[NBME11_错题本#^Q008]]

> [!info] 为什么脂肪酶比淀粉酶更可信 — 高淀粉酶血症 (hyperamylasemia) 来源很杂
> 淀粉酶**不特异**,很多非胰腺情况都能让它升高,所以单纯淀粉酶↑ **不能确诊**胰腺炎、正常也**不能排除**:
> - **唾液腺源**:腮腺炎 (mumps / parotitis 流行性腮腺炎)、酒精、头颈放疗
> - **肠源**:肠缺血/梗死 (mesenteric ischemia 肠系膜缺血)、肠梗阻 (bowel obstruction)、肠穿孔 (perforation)——钝性腹外伤的肠挫伤也可轻升
> - **妇科**:异位妊娠破裂 (ectopic pregnancy 宫外孕)、输卵管炎
> - **代谢/清除↓**:糖尿病酮症酸中毒 (DKA, diabetic ketoacidosis)、肾衰 (renal failure,清除减少)
> - **巨淀粉酶血症 (macroamylasemia)**:淀粉酶与免疫球蛋白结合、无法经肾清除 → 假性持续升高、**无症状**(良性、勿误当胰腺炎)
>
> ∴ **脂肪酶 (lipase) 对胰腺更特异、升得更久 → 首选**;但脂肪酶也有假阴性(早期 / 外伤性可正常,见上 Q008)。钝性上腹伤的横向鉴别(十二指肠血肿等)见 [[完整笔记/专题笔记/外科/外科_腹部外伤BAT速查]]。
- **处理**：<30% 坏死 → NPO + IV 补液 + 镇痛；>30% 坏死 → 抗生素 + 手术。**胆源性 → 本次住院期内尽早腹腔镜胆囊切除**（延迟 25% 复发）。
- **出血性体征**：Cullen（脐周）/ Grey Turner（侧腹）。
- **TG 性阶梯**：≥500 考虑 **胰岛素输注**（激活 LPL 脂蛋白脂肪酶 lipoprotein lipase）；>1000 或重症（乳酸酸中毒/低钙）→ **血浆置换 apheresis**。

### 10.2 慢性胰腺炎 + 胰腺癌

- **慢性胰腺炎**：反复发作 + 持续钝痛 + **胰腺钙化 + 脂肪泻 + 糖尿病**；酶可正常；初查 CT，**粪弹力蛋白酶 fecal elastase 低**（外分泌不足），金标准促胰液素激发；治疗酶替代。
- **胰腺癌**：**无痛性黄疸（>55 岁）**；吸烟/慢性胰腺炎/酒精；初查超声 → CT±ERCP；**Whipple 术**；**Courvoisier 征**（无痛黄疸 + 可触大胆囊 = 胰头/胆道恶性）；**Trousseau 综合征**（游走性血栓性静脉炎 = 副肿瘤高凝）。
- **胰周积液时间线**：脓肿 abscess ~1 周（发热白细胞↑ → 抗生素 + 引流）；假性囊肿 pseudocyst 数周后（**>6 cm 才引流**）。

## 十一、胆道（Biliary）四病决策

> [!tip] 胆道四病一眼分流（痛/热/黄三要素）
> ```
> 餐后间歇 RUQ(右上腹 right upper quadrant) 痛、痛间缓解(40 肥胖女) = 胆绞痛/胆石 → 超声 → 择期胆囊切除
> 持续 RUQ 痛放射右肩 + 发热 + Murphy(+) = 胆囊炎 → 超声(看不清→HIDA 肝胆显像 hepatobiliary iminodiacetic acid scan) → <72h 早切，>72h 延迟
> 痛 + 黄疸 + ALP↑ = 胆总管结石 choledocholithiasis → ERCP(诊断兼治疗)
> 痛 + 发热 + 黄疸(Charcot 三联) = 上行性胆管炎 ascending cholangitis → 抗生素 + ERCP(确定性)
> ```

- **胆石管理**：无症状观察；有症状择期腹腔镜胆囊切除（差手术者熊去氧胆酸 ursodeoxycholic acid）；**并发症（胆囊炎/胆总管结石/胆石性胰腺炎）→ 72h 内切**。
- **胆囊炎抗生素**：覆盖肠道菌（G- 肠杆菌 + G+ 肠球菌 + 厌氧）→ **哌拉西林-他唑巴坦 piperacillin-tazobactam**。
- **无结石性胆囊炎 acalculous cholecystitis**：危重/TPN（全肠外营养 total parenteral nutrition）/创伤后；治疗 = 抗生素 + **胆囊穿刺造瘘 cholecystostomy**（不耐手术者先引流），稳定后切除。
- **胆管炎 Reynolds 五联** = Charcot 三联 + 低血压 + AMS（更重）。

## 十二、年龄段脂肪泻 + 小儿便血（补充内容钥匙）

> [!info] 小儿脂肪泻/血便钥匙（源自 [[完整笔记/Peixuan分科笔记/消化补充内容]]）
> - **囊性纤维化 CF (cystic fibrosis)**：油腻便 + 反复肺感染 + 长得瘦 → 汗液氯试验 + 补胰酶/脂溶维生素。
> - **胆道闭锁 biliary atresia**：2–8 周黄疸 + **陶土样便** → 超声/HIDA → Kasai 术/肝移植。
> - **牛奶蛋白过敏 MPIAP（milk protein-induced allergic proctocolitis）**：2–8 周、血丝/黏液便、**看起来很健康**（病变在直肠结肠，无脂肪泻）。
> - **肠套叠 intussusception**：**果酱样便 + 阵发哭闹 + 嗜睡** → 空气灌肠。
> - **继发性乳糖不耐受**：急性胃肠炎后一喝奶就腹泻产气（刷状缘乳糖酶暂时丢失）。

> [!warning] 异物/腐蚀剂误服（高频急诊）
> - **尖锐物（鱼刺/针）/ 纽扣电池 / 多个磁铁 / 梗阻 → 紧急内镜取出**；禁催吐、禁钡剂。
> - **腐蚀剂 caustic**：口咽溃疡 + 流涎 + **意识清醒** → 24h 内内镜分级；**禁活性炭(挡镜)/催吐/中和(放热)/激素**；碱液化坏死(穿透深)、酸凝固；远期狭窄 + 鳞癌。

## 复习题

> [!question]- Q1. achalasia 与 scleroderma 食管受累，测压上如何一刀切分？各自首选治疗？
> achalasia = **LES 压力↑** + 蠕动↓（气囊扩张 best）；scleroderma = **LES 压力↓** + 严重 GERD（终身 PPI）。两者镜像。

> [!question]- Q2. 胆道四病：痛+黄疸（无发热）vs 痛+发热+黄疸，下一步分别是什么？
> 痛 + 黄疸 + ALP↑ = 胆总管结石 → ERCP；痛 + 发热 + 黄疸（Charcot）= 上行性胆管炎 → 抗生素 + ERCP（确定性）。

> [!question]- Q3. SAAG ≥1.1 提示什么？SBP 穿刺确诊阈值 + 何时加白蛋白？
> SAAG ≥1.1 = 门脉高压（肝硬化/心衰/Budd-Chiari）。SBP = **PMN ≥250/mm³**；Cr>1 或 BUN>30 或 Bili>4 → 加 IV 白蛋白防 HRS。

> [!question]- Q4. 急性胰腺炎诊断标准？哪个酶更特异？胆源性的 ALT 阈值？
> 3 取 2（痛 / 酶>3 倍 / 影像）；脂肪酶 lipase 更特异且持续更久；**ALT >150 提示胆源性**。

> [!question]- Q5. 静脉曲张急性出血三件套药物？输血阈值？一级预防（中/大曲张）选什么？
> IV 补液 + 奥曲肽 + 抗生素（头孢曲松）；Hb<7 g/dL 输血；一级预防 NSBB 或 EVL。

> [!question]- Q6. 克罗恩 vs 溃结：抗体、钡剂、活检各自钥匙？
> 克罗恩 = ASCA / string sign·鹅卵石 / 非干酪样肉芽肿；溃结 = p-ANCA / lead pipe / 隐窝脓肿。

> [!question]- Q7. celiac 患者 tTG-IgA 阴性但高度怀疑，下一步？
> 查总 IgA（celiac 常合并 IgA 缺乏致假阴性）→ 改用 tTG-IgG 或 DGP。

> [!question]- Q8. 肠系膜缺血的标志性症状？急性两大危险因素？诊断 + 静脉/动脉型治疗？
> Pain out of proportion；房颤栓子 + 动脉粥样硬化；CTA 诊断；静脉型肝素、动脉型罂粟碱。

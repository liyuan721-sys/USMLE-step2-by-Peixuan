# GI_吞咽困难 dysphagia：机械性 vs 动力性

> [!tip] 第一刀怎么切
> 拿到吞咽困难 (dysphagia) 先分两大类——**机械性梗阻 (mechanical obstruction)** vs **动力障碍 (motility disorder)**。分法看**"固体先还是固液同时"**：
> - **固体为主、进行性（先固体 → 后液体）** → 机械性梗阻（管腔越来越窄）。
> - **固体 + 液体从一开始就同时困难** → 动力障碍（蠕动 / 括约肌本身出问题，跟食物质地无关）。

---

## 一、机械性 vs 动力性：核心对比

表格前空行。

| 维度 | 机械性梗阻 mechanical | 动力障碍 motility |
|---|---|---|
| 吞咽困难模式 | **固体先**，进行性加重后累及液体 | **固液同时**，早期即如此 |
| 病程 | 进行性（癌）或间歇性（环 / 蹼） | 慢性、相对稳定或缓慢进展 |
| 病理 | 管腔狭窄（肿块 / 环 / 蹼 / 狭窄） | 蠕动 / 下食管括约肌 LES (lower esophageal sphincter) 功能异常 |
| 代表病 | 食管癌、Schatzki 环、食管蹼、消化性狭窄 | 贲门失弛缓 achalasia、硬皮病 scleroderma、弥漫性食管痉挛 DES (diffuse esophageal spasm) |

表格后空行。

```text
吞咽困难 dysphagia
│
├─ 固体先、进行性 ─────→ 机械性梗阻
│     ├─ + 消瘦/贫血/报警症状 → 食管癌（先钡剂，确诊 EGD 上消化道内镜 + 活检）
│     ├─ 间歇性、仅固体 + 裂孔疝 → Schatzki 环
│     └─ 缺铁性贫血三联、上段 → 食管蹼 web（Plummer-Vinson）
│
└─ 固体 + 液体同时 ────→ 动力障碍
      ├─ 钡剂鸟嘴 bird-beak + 消瘦 → 贲门失弛缓 achalasia
      ├─ LES 压力↓ + 严重反流 + 硬皮 → 硬皮病 scleroderma 食管
      └─ 胸痛 + 冷饮诱发 + 螺旋 corkscrew → 弥漫性食管痉挛 DES
```

## 二、动力障碍三兄弟（测压 manometry 是钥匙）

> [!info] 贲门失弛缓 achalasia（贲门失弛缓）
> - **机制**：食管肌间神经丛 (myenteric plexus) 神经元退变 → **LES 松弛障碍 (failure of LES relaxation) + 食管体蠕动消失 (aperistalsis)**。
> - **临床**：固液同时吞咽困难 + 体重下降；可反流未消化食物。
> - **影像**：钡餐 **鸟嘴征 bird-beak**（远端渐窄 + 近端扩张）。
> - **金标准**：食管测压 manometry → LES 压力↑ + 蠕动↓。
> - **必做**：内镜排除**假性失弛缓 pseudoachalasia**（远端食管 / 贲门癌模仿）。
> - **治疗**：气囊扩张 pneumatic dilation（首选 / best）或 Heller 肌切开；不能耐受手术 → 肉毒素 botulinum toxin 注射。成人 Chagas 病因 → 苄硝唑 benznidazole / 硝呋替莫 nifurtimox。

> [!info] 硬皮病 scleroderma 食管（与 achalasia 镜像）
> LES 压力**↓** → 严重胃食管反流病 GERD (gastroesophageal reflux disease)；治疗终身 PPI（质子泵抑制剂 proton pump inhibitor）。与 achalasia 的 "LES↑" 正好相反。

> [!info] 弥漫性食管痉挛 DES
> 胸痛 + 吞咽困难，**冷饮诱发**；钡剂 **螺旋 / 串珠 corkscrew**；治疗钙通道阻滞剂 CCB (calcium channel blocker) / 硝酸酯。

## 三、易混陷阱

> [!warning] 常见混淆
> - **口咽性 (oropharyngeal) vs 食管性 (esophageal) 吞咽困难**：口咽性 = 启动困难、呛咳、鼻反流（卒中 / 帕金森 / 重症肌无力 MG），查电视荧光吞咽造影 VFSS (videofluoroscopic swallow study)；本篇讨论的是食管性。
> - **别把 achalasia 当癌**：achalasia 固液同时 + 慢性；食管癌固体先 + 进行性 + 报警症状。但两者影像可近似——**测压 + 内镜活检**定夺，必须排 pseudoachalasia。
> - **裂孔疝 hiatal hernia** 更多是反流症状，不给 achalasia 那种 distal taper 影像。

> [!question]- 6 个月固液都吞咽困难 + 消瘦 + 钡餐远端渐窄近端扩张 = ?
> **贲门失弛缓 achalasia**。固液同时 = 动力障碍；鸟嘴征 bird-beak 基本钉住诊断。金标准测压 manometry（LES↑ + 蠕动↓），首选气囊扩张。

> [!question]- "固体先、进行性"提示哪一大类？为何？
> **机械性梗阻**。管腔逐渐狭窄时，大块固体先卡不下，随狭窄加重才累及液体；动力障碍与食物质地无关，故固液从一开始就同时困难。

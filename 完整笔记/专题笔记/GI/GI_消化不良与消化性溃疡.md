# GI 消化不良与消化性溃疡（Dyspepsia & Peptic Ulcer Disease）

> 核心命题：一个上腹症状患者走进来，Step 2 CK 只考一件事——**下一步是 test-and-treat（查/治 H. pylori）还是转内镜（EGD）**。这道分岔完全由"恶性风险"决定：看年龄（≥60）和报警征（alarm features）。

> [!info] 一句话锚点
> **消化不良 = 上腹痛/烧灼 + 餐后饱胀/早饱，持续数周。** 先做的不是开 PPI，而是先判断恶性风险 → 决定 test-and-treat 还是 EGD。

---

## 一、消化不良 Dyspepsia：定义与病因

**定义**：以**上腹（epigastric）为中心**的持续或反复不适，包含以下一项或多项，持续 ≥ 数周：

- 上腹痛（epigastric pain）或烧灼感（burning）
- 餐后饱胀（postprandial fullness）
- 早饱（early satiety，没吃几口就饱）

**病因分层（记住比例）**：

| 病因 | 占比/特点 | 关键线索 |
|---|---|---|
| 功能性/特发性消化不良（functional dyspepsia） | ~75%，最常见，排除性诊断 | 无器质病灶，检查阴性 |
| 消化性溃疡病（peptic ulcer disease, PUD） | H. pylori 或 NSAID 所致 | 夜间痛、空腹痛、可出血 |
| 恶性（malignancy） | 胃癌（gastric）/食管癌（esophageal） | 报警征、≥60 岁、体重下降 |
| 药物性（drug-induced） | NSAID、双膦酸盐（bisphosphonates，如 alendronate 阿仑膦酸钠） | 用药史 |
| 胃食管反流病（GERD, gastroesophageal reflux disease）重叠 | 烧心、反流为主时归 GERD | 症状偏胸骨后、反酸 |

> [!tip] 为什么先想 H. pylori 与 NSAID
> PUD 是"可治愈的器质病因"里最常见的，而且两大驱动因素——**幽门螺杆菌（Helicobacter pylori）** 与 **非甾体抗炎药（NSAID）**——都是可干预的。所以低风险患者的第一步就是 test-and-treat H. pylori。

---

## 二、⭐处理分层决策（Step 2 CK 高频）

整套决策只有一个分岔口：**恶性风险高不高？**

```
消化不良患者
   │
   ├─ 低恶性风险 = <60 岁 且 无报警征
   │     └→ Test-and-treat H. pylori
   │          （粪抗原 stool antigen / 尿素呼气试验 urea breath test）
   │          阳性 → 三联/四联根除 → 未缓解再考虑 EGD
   │          阴性 → 经验性质子泵抑制剂（PPI, proton pump inhibitor）试验
   │
   └─ 高恶性风险 = ≥60 岁 或 有任一报警征
         └→ 消化科转诊 + 上消化道内镜（EGD, esophagogastroduodenoscopy）
              直接看黏膜 + 活检
```

> [!warning] 报警征清单（Alarm Features）——见任一即走 EGD
> - 消化道出血（黑便、呕血、**粪隐血阳性（fecal occult blood test, FOBT+）**、缺铁性贫血）
> - 不明原因体重下降（unintentional weight loss）
> - 缺铁性贫血（iron-deficiency anemia）
> - 淋巴结肿大（lymphadenopathy）或可触及腹部包块（palpable mass）
> - 吞咽困难（dysphagia）/ 吞咽痛（odynophagia）
> - 持续呕吐（persistent vomiting）
> - 上消化道恶性肿瘤家族史

> [!danger] 检测方法陷阱：不要用血清学做初筛
> 查 H. pylori 首选 **粪抗原（stool antigen）** 或 **尿素呼气试验（urea breath test, UBT）**——两者都能反映**现症活动性感染**。
> **血清学（serology/IgG 抗体）不能区分现症感染 vs 既往已清除感染**，故不作为首选。
> ⚠️ UBT 与粪抗原前需停 PPI（约 2 周）、停抗生素/铋剂，否则假阴性。

## 三、幽门螺杆菌 Helicobacter pylori 与十二指肠溃疡

**机制链条（背下来）**：

```
H. pylori 定植胃窦（antrum）
   → 产脲酶（urease-producing），分解尿素产氨中和胃酸而存活
   → 慢性炎症刺激 G 细胞 → 促胃泌素（gastrin）↑
   → 壁细胞（parietal cell）泌酸增多
   → 过量酸冲入十二指肠 → 十二指肠溃疡（duodenal ulcer, DU）
```

**临床特征**：

- **夜间痛醒**（nocturnal pain）——胃酸分泌有昼夜节律，凌晨基础泌酸达峰
- **空腹痛加重、进食后缓解**（DU 的经典时相）
- 餐后腹胀、恶心
- 可出血 → 黑便、粪隐血阳性、缺铁性贫血
- **确诊**：EGD 于**胃窦（antrum）活检**（组织学 / 快速尿素酶试验 CLO test）；非侵入性初筛用粪抗原 / UBT

> [!info] 脲酶是"钥匙分子"
> H. pylori 的 urease 既是它的**生存机制**（产氨中和胃酸），也是我们**检测它的靶点**（UBT 让患者喝标记尿素，细菌脲酶把它分解成标记 CO₂ 呼出来测到）。一个酶串起机制与诊断。

## 四、十二指肠溃疡 vs 胃溃疡对比

| 特征 | 十二指肠溃疡 DU（duodenal） | 胃溃疡 GU（gastric） |
|---|---|---|
| 疼痛与进食时相 | 空腹/夜间痛，**进食后缓解**（食物中和酸） | **进食后加重**（约 30 min 内），进食恐惧→可体重下降 |
| H. pylori 关系 | 强相关（~90%+） | 相关（~70%），另 NSAID 占重要比例 |
| 泌酸 | 常泌酸↑ | 泌酸正常或↓（黏膜屏障问题为主） |
| 恶性风险 | 极低，几乎不恶变 | **有恶变可能** → 治疗后应复查 EGD 确认愈合 + 活检 |
| 好发年龄 | 偏年轻 | 偏年长 |

> [!tip] 一句话区分口诀
> **进食后"好一点"→ 十二指肠（Duodenal = Decreased pain with food）；进食后"更痛"→ 胃（Gastric = Greater pain with food）。** 且只有**胃溃疡**需要警惕恶性、复查内镜活检。

## 五、鉴别陷阱（为什么不是这些）

> [!danger] 高频干扰项拆解
> - **GERD / 下食管括约肌功能障碍（LES dysfunction）**：以烧心、反流、平卧/弯腰加重为主，**不会导致粪隐血阳性/消化道出血**。若 stem 出现出血/贫血证据，指向的是**黏膜溃疡（PUD）**而非单纯 LES 松弛。（← Q3588 的错点：把出血性上腹痛误归为 LES 功能障碍）
> - **胃泌素瘤 / Zollinger-Ellison 综合征（ZES）**：罕见；表现为**多发/难治性溃疡 + 慢性分泌性腹泻 + 体重下降**，查空腹促胃泌素显著升高。没有腹泻/难治线索时不首选。
> - **乳糜泻（celiac disease）**：以慢性腹泻、脂肪泻、缺铁性贫血、体重下降、腹胀为主，与麸质相关；上腹烧灼夜间痛不是它的典型表现。
> - **恶性（胃/食管癌）**：≥60 岁或报警征时必须先排除——这正是分层走 EGD 的理由。

## 六、复习题

> [!question]- Q1. 一位 42 岁女性，上腹烧灼痛、夜间常被痛醒、餐后腹胀，粪隐血阳性，无体重下降、无吞咽困难、无贫血。下一步最合适的处理是？
> 因为出现了**报警征（粪隐血阳性 = 消化道出血证据）**，即使 <60 岁也应**转诊行上消化道内镜 EGD**，而非单纯 test-and-treat。
> 若题目改成"完全无报警征的 <60 岁患者"，则答案变为 **test-and-treat H. pylori（粪抗原 / 尿素呼气试验）**。

> [!question]- Q2. 为什么初筛 H. pylori 不首选血清学（IgG 抗体）？
> 血清抗体在感染清除后仍可长期阳性，**不能区分现症感染与既往感染**，故不用于判断"当前是否需要根除"。首选**粪抗原**或**尿素呼气试验**，两者反映活动性感染。

> [!question]- Q3. 十二指肠溃疡与胃溃疡在"进食后疼痛"上的区别？哪个需警惕恶性？
> 十二指肠溃疡：进食后**缓解**（空腹/夜间痛）。胃溃疡：进食后**加重**。**只有胃溃疡有恶变风险**，治疗后需复查 EGD + 活检确认愈合。

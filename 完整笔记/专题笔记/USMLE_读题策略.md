## Part 1. 题型识别 + 三步读题法 ⭐⭐⭐（Q3504 / Q15054 / Q2355 / Q7289 母题）

### 1.1 核心问题

**"为什么治疗合理的选项是错的？"**
答：因为题型是**程序性安全题**而非临床决策题——你看到了"治疗"，题在问"安全检查"。

### 1.2 4 信号 → 是程序题不是决策题

> [!warning] 看到下面 4 个信号 → 默认程序性安全题

1. Stem **详细描述某个操作过程**（CVC / LP / 胸管 / 内镜 / 手术 / NG 置入）
2. 问 "best next step" 但**操作刚结束**
3. 选项里**"确认 / 影像"类**（CXR / US / Capnography）和**"治疗"类**（药 / 液）**并存**
4. 临床背景看似紧急（**让你想跳步骤**）

### 1.3 答题原则

> [!success] 程序题铁律
> **永远先选"确认 / 安全检查"，再选"治疗"**。位置未确认前不给药 / 不补液 / 不输血 / 不喂食。

### 1.4 USMLE 4 类临床题型识别

| 题型 | 触发词 | 答题方向 |
|---|---|---|
| **临床决策**（诊断 / 治疗）| 典型 stem + "most likely diagnosis / treatment" | 整合 demographics + 病程 + 体征 + 实验室 |
| **程序性安全** | "刚做完 X，best next step" | **位置确认优先**（CXR / Capno / 波形）|
| **并发症识别** | "操作后患者出现 [症状]，most likely cause" | 时间 + 位点 + 症状 → 推并发症 |
| **禁忌识别** | "which patient should NOT receive" | 识别绝对禁忌（基底骨折 + NG / 尿损 + Foley / 颅压↑ + LP） |

### 1.5 一句话锁定

> **看到 "best next step" + 刚操作完 → 程序题；看到 "most likely diagnosis" + 复杂 stem → 决策题。**

🔗 详见 [[完整笔记/专题笔记/USMLE_侵入操作安全流程]] Part 11

### 1.6 先读问句 → 判"它要哪类答案" ⭐⭐⭐（Q15054 / Q2355 / Q7289 母题）

> [!danger] 第一印象陷阱（凭感觉做题的人专属坑）
> 第一印象是好雷达（快速认病），但**不能直接当答案**。USMLE 常把"这个病的默认答案"塞进选项 —— 你认出病就反射甩默认答案 = 正中陷阱。
> **同一个病，问法不同、答案不同**：认出"病" ≠ 知道这题要选什么。

**USMLE 5 种"问法" —— 答案类型完全不同：**

| 问法 | 题干信号词 | 要选的是 |
|---|---|---|
| **诊断** | most likely diagnosis | 病名 |
| **即刻处理 / 第一步** | initial / first / next step、"此刻正在…" | 当下控场 / 救命那一步（常 ≠ 长期方案）|
| **长期治疗** | maintenance、long-term、definitive | 维持 / 根治方案 |
| **沟通** | best response、how should you respond、"对家属说什么" | 一句**话**（措辞），不是医疗操作 |
| **检查** | most appropriate test、confirm diagnosis | 检查手段 |

**三步读题法（每题都走，含"一眼会"的简单题 —— 练成肌肉记忆）：**

```
① 先读最后一句（问句）—— 还没读病例，先看它问什么
     圈信号词: diagnosis? first/next step? best response? test?
              │
② 带着问句读题干 —— 现在知道在找什么
     盯"场景信息": 时间点 / 谁在说话 / 病人此刻状态
     （正解藏在这里，不在病名里）
              │
③ 选答案前回头对一眼问句
     自问"我这答案，回答的是它问的那个问题吗?"
     第一印象 = 假设，不是判决
```

**3 道母题（知识全对，全错在没读"问哪一面"）：**

| 题 | 认出的"病" | 反射甩出 | 题目其实问 | 正解 |
|---|---|---|---|---|
| Q15054 | 抗抑郁药转躁 | 双相 → 锂 | **first step** | 停诱因药 |
| Q2355 | 躁狂 | 口诀 LV2 → 丙戊酸 | **此刻砸门拒药的人怎么办** | IM 抗精神病药 |
| Q7289 | 抗抑郁药 + 自杀顾虑 | 选个"稳的治疗方案" | **怎么回应母亲** | 如实告知 + 权衡的沟通 |

> [!success] 一句话锁定
> **先读问句、再读题干；选答案前回头确认"答的是它问的那一面"。** 第一印象当假设、不当交卷。

## Part 2. 完整 criteria 逐项核对法 ⭐⭐⭐（Q3354 母题）

### 2.1 核心问题

**"我看到 'no active bleeding' 就觉得排除 hard signs 了。"**
答：错——只排除了 hard signs 4 项中的 **1 项**（B-Blood spurting），还有 3 项要逐项核对。

### 2.2 反向阴性体征陷阱

> [!danger] 反向阴性陷阱
> **见 "no X" 时**：X 只是某个完整 criteria 清单中的**单项**。不能用 1 项的缺失推断**整个类别**不成立。

### 2.3 读题流程（v2 升级版）

```
见到 "no X" / "negative for X" / "without X" 时：
        │
        ▼
Step 1: 问"X 属于哪个完整 criteria 清单？"
        │
        ▼
Step 2: 把完整清单的所有项列出（4 项 / 5 项 / 6 项）
        │
        ▼
Step 3: 逐项核对：
        - "no X" → X 项不满足
        - 其他项呢？stem 写了什么？
        │
        ▼
Step 4: 任 1 项满足（或 ≥ 阈值）→ 类别诊断成立
       全部不满足 → 才能真正排除
```

### 2.4 USMLE 高频"质变 criteria"清单

> [!info] 看到下面任一 → 自动调用"逐项核对"流程

| Criteria | 阈值 | 出现场景 |
|---|---|---|
| **4B's of Hard signs** | 任 1 项 | 肢体血管创伤（Q3354）|
| **Light's criteria for exudate** | 任 1 项 | 胸水鉴别 |
| **SIRS criteria for sepsis** | ≥ 2 项（4 项中）| 脓毒症诊断 |
| **Jones criteria for rheumatic fever** | 2 大 / 1 大 + 2 小 | 风湿热 |
| **Wells score for PE / DVT** | DVT ≥ 2；PE > 4 | 静脉血栓评估 |
| **HEART score for ACS** | ≥ 4 | ACS 风险分层 |
| **APACHE II / Ranson for pancreatitis** | 阈值见表 | 胰腺炎重症度 |
| **Centor criteria for strep** | ≥ 3 | 链球菌咽炎 |
| **PERC criteria** | 全部满足 = 排除 PE | PE 排除 |
| **CURB-65** | ≥ 2 入院 | 肺炎严重度 |
| **CHA2DS2-VASc** | ≥ 2 抗凝 | AFib 卒中风险 |
| **HAS-BLED** | ≥ 3 出血高风险 | AFib 抗凝出血风险 |
| **Glasgow Coma Scale** | ≤ 8 插管 | 意识评估 |
| **MELD score** | > 15 移植 | 肝移植排序 |
| **Child-Pugh** | A/B/C | 肝硬化分级 |
| **PAI 五联指纹** | 3-5 项 | 肾上腺危象（Q17169）|

### 2.5 一句话锁定

> **见 "no X" / "X is normal" / "denies X" → 不下结论，先列完整清单逐项核对**。一项缺失 ≠ 整个类别排除。

## Part 3. 跳过影像直接 OR 决策 ⭐⭐⭐（Q4212 + Q3354 共建）

### 3.1 核心问题

**"严重患者 + 怀疑诊断 = 做影像确认？"**
答：要看是否处于"**时间敏感型急症**"。某些诊断是**临床诊断**或**hard signs 阳性** → **跳过影像直接处理**。影像 = 致命延误。

### 3.2 USMLE "跳过影像直接 OR / 治疗"清单

| 急症 | 触发条件 | 直接处理（不等影像）|
|---|---|---|
| **不稳定 + 已知 AAA** | 突发剧痛 + 低血压 | **直接 OR**（Q4212）|
| **不稳定 + 无既往 AAA** | 急腹痛 + 低血压 + 老烟枪 | **床旁 US**（不送 CT）|
| **肢体血管创伤 + Hard signs** | 4B's 任 1 项 | **直接 OR**（Q3354）|
| **Rutherford IIb ALI** | 感觉↓ + 运动↓ + Doppler 动脉无 | **Emergent OR**（不等 CTA）|
| **张力性气胸** | 单侧呼吸音消失 + 气管偏 + JVD + SBP↓ | **针刺减压**（不等 CXR）|
| **大块 PE 不稳定**（SBP < 90）| 突发呼吸困难 + 低血压 | **床旁 TTE → tPA**（不等 CTPA）|
| **心包填塞** | Beck 三联 + 失代偿 | **床旁 echo → 心包穿刺**（不等正式影像）|
| **Adrenal crisis** | 慢性 PAI + 应激 + 难治性低血压 + PAI 五联 | **Hydrocortisone IV**（不等 cortisol 结果）|
| **怀疑细菌性脑膜炎** | 发热 + 颈强 + 头痛 | **先经验性抗生素 → CT → LP**（不能等 CT 延误抗生素）|
| **不稳定 ALI（6 P's 全有）** | 6 P's + < 6h | **IV Heparin → 紧急 revasc**（不等影像）|

### 3.3 共同特征

> [!danger] 跳影像的 3 个共同条件
> 1. **诊断 = 临床诊断**（hard signs / Beck 三联 / 6 P's / PAI 五联）
> 2. **影像 = 致命延误**（搬到 CT 室 → 失血休克 / 肌肉坏死 / 脑疝）
> 3. **首步处理 = 救命行为**（OR / 减压 / 溶栓 / 激素）

### 3.4 反例 — 什么时候要做影像？

| 情境 | 做什么 | 为什么 |
|---|---|---|
| **稳定患者**（生命体征 OK）| CT / CTA / 正式 echo | 时间充裕 + 高分辨率指导手术 |
| **诊断不明**（多种可能）| CT 鉴别 | 减少开错刀概率 |
| **Soft signs 创伤** | IEI → CT angio | 避免阴性探查 |
| **PE 稳定 + 高可疑** | CTPA | CT 同时找位置指导取栓 |

### 3.5 完整决策树范例 — Blunt chest trauma（UWorld 官方图，Q19889）

![[{97EDC4FC-B930-442E-84D2-8A302E2BF024}.png]]

> [!info] 图示读法
> - **左路（不稳定）**：直接复苏 + 三件套（eFAST / CXR / ECG）+ stabilizing intervention → 稳定后做 additional tests；未稳定 → **OR thoracotomy**（跳过影像）
> - **右路（稳定）**：分两叉
>   - 有 high-risk mechanism / serious injury → 走"左路"的三件套评估
>   - 无 → 看 CXR / ECG 是否异常 → 异常做 additional tests；正常 → **可出院 / 观察**
> 
> **核心教学点**：稳定 + 三件套全阴性 + 无症状 = **可出院**（Q19889 答案 D）；"跳影像直接 OR" 仅左下路（不稳定且复苏未稳定）使用。
> 
> 🔗 详见 [[mistakes/uworld-mistakes#^Q19889]]

### 3.5 一句话锁定

> **不稳定 + 临床诊断已成立 → 跳影像直接 OR / 治疗；稳定 + 诊断不明 → 做影像**。

## Part 4. "质变升级"规则 ⭐⭐（Q3354 母题）

### 4.1 核心问题

**"Feeble pulses 是 Hard 还是 Soft sign？"**
答：**单独看是 Soft；+ 远端缺血任 1 项（cool / pale / paresthesia）= 整体升级为 Hard**。

### 4.2 质变升级规律

> [!warning] 1 + 1 > 2 — 组合可质变
> 单独看每个元素都是"轻微 / 间接证据"，但组合在一起可触发"严重 / 直接证据"。

### 4.3 USMLE 高频"质变升级"场景

| 单独看 | 组合后 | 整体升级 |
|---|---|---|
| Feeble pulses（Soft）| + cool / pale / paresthesia | **Hard**（distal ischemia）|
| Small hematoma（不算 hard）| + 进行性扩大 | **Hard**（expanding）|
| History of bleeding（Soft）| + 当前 bruit | **Hard**（bruit）|
| Neurologic abnormality（Soft）| + 缺血表现 | **Hard**（distal ischemia）|
| Eos↑（过敏？）| + Na↓ + K↑ + Glu↓ | **PAI 指纹**（Q17169）|
| Sudden onset 腹痛（多种可能）| + 循环崩溃 + 无 N/V | **AAA 破裂**（非胰腺炎）|
| 数月体重↓（不特异）| + 应激事件 + 难治性低血压 | **Adrenal crisis** |
| 单一 SIRS 项（无意义）| + 第 2 项 | **SIRS 诊断成立** |
| 单条 Light's 满足 | 即可 | **Exudate**（任 1 项）|

### 4.4 读题流程

```
见到"单一弱信号"时：
        │
        ▼
Step 1: 标记该信号属于哪个 criteria
        │
        ▼
Step 2: 扫 stem 找"其他可能升级因素"
        │
        ▼
Step 3: 组合 → 检查是否触发质变
        │
        ▼
Step 4: 升级后重新判断答案
```

### 4.5 一句话锁定

> **任何 soft / 弱 / 不特异信号 + 远端 / 缺血 / 严重伴随 → 检查是否"质变升级"**。不能孤立看每个项。

## Part 5. 症状不特异 → 指纹思维 ⭐⭐⭐（Q17169 母题）

### 5.1 核心问题

**"症状全是疲劳 / 体重↓ / GI 不适，没有诊断方向。"**
答：**跳过症状迷雾，直接看 Lab panel**。症状不特异时，**实验室指纹是锚点**。

### 5.2 指纹思维 6 步法

```
做内分泌 / 代谢 / 血液 / 中毒 / 电解质题：

Step 1: 跳过症状 → 直接看 Lab panel
Step 2: 圈出所有异常（任何"↑↓"标记）
Step 3: 数异常项 + 判断"组合形态"
        ├─ 3+ 项聚集 → 想"哪个指纹病？"
        ├─ 单一极端 → 想"金标准指标？"
        └─ 形态学 → 想"涂片 / 影像？"
Step 4: 回头看症状 + 慢性背景验证
Step 5: 阴性体征排除竞争诊断
Step 6: 锁定 → 选答案
```

### 5.3 何时切换为"指纹思维"

> [!success] 触发条件（任一）
> - Stem 给完整 Lab panel（≥ 5 项）
> - 症状非常不特异（疲劳 / 体重↓ / GI / 头晕）
> - 答案选项有"金标准 lab 检查"（5-HIAA / metanephrines / M-spike / ceruloplasmin）
> - 涉及形态学（schistocytes / Auer rods / target cells）

### 5.4 一句话锁定

> **症状不特异时，回到实验室。指纹组合 = 诊断锚点，症状 = 干扰项。**

## Part 6. 慢性背景 + 急性触发 ⭐⭐（Q17169 母题）

### 6.1 核心问题

**"数月体重↓ 是来就诊的原因吗？"**
答：不是。USMLE 不写废话。**数月慢性病程**通常是**诊断核心舞台**，不是"附加信息"。

### 6.2 双时间轴模型

```
Stem 双时间轴：

慢性背景（数月-数年）              急性触发（数小时-数天）
─────────────────────────         ───────────────────────
"6 months of fatigue + 10 kg     "Today during endoscopy
 weight loss"                     suddenly hypotensive"
        ▲                                 ▲
        │                                 │
    诊断核心舞台              失代偿按下了开关
（慢性病的真身）              （应激触发危象）

最终诊断 = 慢性背景 × 急性触发
        = Chronic PAI × propofol stress
        = Adrenal crisis on chronic PAI
```

### 6.3 USMLE 经典"慢性 + 急性"组合

| 慢性背景 | 急性触发 | 最终诊断 |
|---|---|---|
| 数月体重↓ + GI 症状 | 应激事件（手术 / 麻醉 / 感染）| **Adrenal crisis on PAI** |
| 数年 HTN + 吸烟 | 突发腹痛 + 低血压 | **Ruptured AAA** |
| 数年 AFib | 突发肢体冷痛 | **栓塞性 ALI** |
| 数年 SLE | 突发胸痛 | **APS / 心包炎 / 冠脉炎** |
| 数月酗酒 | 突发上腹痛 | **急性胰腺炎** |
| 数年糖尿病 | 突发昏迷 | **DKA / HHS** |
| 多年类风湿 | 突发颈痛 + 四肢瘫 | **C1-C2 半脱位**（环枢椎不稳）|

### 6.4 读题流程

```
见 stem 写"数月-数年 X" + "突发 Y"：
        │
        ▼
不要忽略慢性背景：它是诊断舞台
        │
        ▼
问："X 是什么慢性病？" + "Y 是什么应激？"
        │
        ▼
Y 触发 X 的失代偿 → 最终诊断 = X × Y
```

### 6.5 一句话锁定

> **数月-数年症状 + 突发事件 = 慢性病急性失代偿**。USMLE 不写废话，所有慢性背景都是诊断核心。

## Part 7. 体征 ≠ 疾病 ⭐⭐⭐（Q4212 母题）

### 7.1 核心问题

**"看到 Grey Turner sign 就想到急性胰腺炎。"**
答：错。Grey Turner = **腹膜后出血的邮编**（病理过程），不是**胰腺炎的姓名**（具体病）。

### 7.2 体征思维 3 步法

```
看到经典体征时：
        │
        ▼
Step 1: 这个体征指向什么【病理过程】？
        Grey Turner → 腹膜后出血
        Murphy → 胆囊炎
        Janeway → 心内膜炎栓塞
        ...
        │
        ▼
Step 2: 哪些【疾病】能造成这个病理过程？
        腹膜后出血 = AAA / 出血胰腺炎 / 抗凝血肿 / 介入 / 外伤 / 肿瘤
        │
        ▼
Step 3: 用 demographics + 起病 + 伴随症状定病
        老烟枪 + 突发 + 无 N/V → AAA
        酗酒 + 渐进 + N/V + 发热 → 胰腺炎
```

### 7.3 USMLE 经典"多病因体征"清单

| 体征 | 病理过程 | 病因（按 USMLE 考频）|
|---|---|---|
| **Grey Turner**（侧腹瘀斑）| 腹膜后出血 | AAA 破裂 / 出血胰腺炎 / 抗凝血肿 / 介入血肿 / 骨盆骨折 / 肿瘤出血 |
| **Cullen**（脐周瘀斑）| 腹膜后 + 镰韧带 | **Ruptured ectopic** / 出血胰腺炎 |
| **Murphy sign** | 胆囊炎症 | 急性胆囊炎 / 胆囊积脓 |
| **Janeway lesions** | 感染性栓塞 | IE（无痛性掌跖红斑）|
| **Osler nodes** | 免疫复合物沉积 | IE（**疼痛性**指端结节）|
| **Roth spots** | 视网膜出血 + 苍白中心 | IE / 严重贫血 / 白血病 |
| **Cushing 三联**（HTN + 心动过缓 + 呼吸不规则）| ICP↑ | 任何严重颅压增高（出血 / 占位 / 水肿）|
| **Beck 三联**（低血压 + JVD + 心音遥远）| 心包填塞 | 病毒性 / 尿毒症 / 创伤 / 主动脉夹层逆撕 / 心梗游离壁破 / 肿瘤 |
| **Virchow node** | 锁骨上淋巴结肿大 | 胃癌（最经典）/ 其他腹腔 / 胸腔恶性 |
| **Trousseau 征**（游走性血栓性静脉炎）| 高凝状态 | 胰腺癌 / 肺癌 / 其他实体瘤 |
| **Charcot 三联**（发热 + RUQ 痛 + 黄疸）| 胆管炎 | 胆总管结石 / 狭窄 / 肿瘤 |
| **Reynolds 五联** | 化脓性胆管炎 | Charcot + 低血压 + 意识改变 |
| **Kussmaul respiration** | 代谢性酸中毒 | DKA / 尿毒症 / 乳酸酸中毒 / 中毒 |
| **Kussmaul sign**（吸气 JVD↑）| 右心填塞 | 心包填塞 / 缩窄性心包炎 / 严重 RV 衰 |

### 7.4 读题流程

```
看到经典体征：
        │
        ▼
不要直接跳到"经典病"
        │
        ▼
问："这个体征指向什么病理过程？"
        │
        ▼
列出能造成该过程的所有疾病（鉴别表）
        │
        ▼
用 demographics + 起病 + 伴随症状定病
```

### 7.5 一句话锁定

> **经典体征指向【病理过程】，不指向【具体病】**。先想过程，再列鉴别，最后用 demographics 定病。

### 8.1 Q4212 — Ruptured AAA vs Hemorrhagic Pancreatitis

| 错位类型 | 体征 ≠ 疾病 |
|---|---|
| 错误路径 | Grey Turner → 急性胰腺炎 → 查 lipase |
| 正确路径 | Grey Turner = 腹膜后出血 → demographics（老烟枪）+ 起病（突发崩溃）→ AAA → 床旁 US |
| 修正习惯 | 看到经典体征 → **先问病理过程，再列鉴别** |

### 8.2 Q17169 — Adrenal Crisis vs Anaphylaxis

| 错位类型 | 症状不特异 + 慢性 / 急性双轴未整合 |
|---|---|
| 错误路径 | Eos 12% → 过敏 → Anaphylaxis |
| 正确路径 | PAI 五联指纹（Na↓ / K↑ / HCO3↓ / Glu↓ / Eos↑）+ 数月体重↓ + 应激事件 → Adrenal crisis |
| 修正习惯 | 症状不特异时 → **指纹思维**；慢性 + 急性 = 失代偿 |
| 链回 | [[mistakes/uworld-mistakes#^Q17169]] |

### 8.3 Q3504 — CVC 置入后必做 CXR

| 错位类型 | 题型识别错位 |
|---|---|
| 错误路径 | 脓毒症 → 抗生素 → 通过新置 CVC 给药 |
| 正确路径 | Stem 详细描述置管步骤 → **程序性安全题** → CXR 确认位置 → 外周 IV 给抗生素同步 |

### 8.4 Q3354 — Hard vs Soft Signs

| 错位类型 | 反向阴性 + 质变未识别 |
|---|---|
| 错误路径 | "no active bleeding" → 排除 hard signs → 走 soft 路径 → Doppler |
| 正确路径 | 列完整 4B → hematoma + cool + feeble pulses → **2 项 Hard signs** → 直接 OR |

## Part 9. 读题前 5 秒检查清单（贴在错题本封面）

> [!success] 拿到 stem 后 5 秒预判

```
□ 先读最后一句（问句）—— 它要哪类答案？
  └─ 诊断 / first step / 长期治疗 / 沟通(说什么) / 检查 —— 5 类答案不同

□ 这是什么题型？
  ├─ 操作刚结束 + best next step → 程序题 → 先选确认
  ├─ Lab panel ≥ 5 项 + 不特异症状 → 指纹题 → 跳症状
  ├─ 经典体征 + 多种鉴别 → 体征 ≠ 疾病题 → 列病理过程
  └─ 不稳定 + 临床诊断已立 → 跳影像题 → 直接处理

□ 见到 "no X" 了吗？
  └─ 列完整 criteria 清单，逐项核对

□ 看到 soft / 弱 / 不特异信号了吗？
  └─ 扫"质变升级"因素

□ Stem 写了"数月-数年"吗？
  └─ 这是慢性背景，不是废话；找急性触发
```

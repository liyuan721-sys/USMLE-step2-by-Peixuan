# _衍生_晕厥鉴别决策树（Syncope Differential & Decision Tree）

> [!info] 本笔记定位
> **晕厥 / 短暂意识丧失（TLOC）的完整鉴别框架**。USMLE Step 2 CK 高频考点，跨**心内 + 神经 + 急诊 + 精神科**。
> 
> **核心思维**：
> 1. **前驱症状 + 触发** = 晕厥分类的两大锚点
> 2. **Orthostatic 不是单一病**，是**容量型 vs 神经源型**两大机制
> 3. 关键鉴别 = **HR 反应**（站起来 HR↑≥30 = 容量型；HR 几乎不动 = 神经源型）
> 4. **金标准**：Postictal confusion = seizure 印章；其他特征都可能重叠
> 5. **典型 + 无红旗 → No further testing**（USMLE 反向应用框架）
> 
> **来源**：错题 Q21579（经典 Vasovagal）+ Q21638（容量型 OH）+ Q3910（LQTS）+ Q4725（PEA）+ Q4212（AAA 致晕厥）+ 多轮深度追问

## Part 0. 核心方法论

### 0.1 鉴别 7 大锚点（"触前心醒，背伴 ECG"）

> [!success] 看到晕厥题 → 先问这 7 个问题
> 1. **触发**：运动 / 卧位 / 站起 / 情绪 / 闪光 / 排尿？
> 2. **前驱**：有 prodrome（全身）？aura（局灶）？还是无前驱？
> 3. **HR 反应**：↓（迷走）/ ↑≥30（容量 OH）/ 不动（神经源 OH）/ 异常（心源）
> 4. **清醒**：立即 / 延迟 + postictal / >5min（Conversion）
> 5. **背景**：年龄 + 慢病 + 家族猝死史
> 6. **伴随**：胸痛 / 心悸 / 舌咬 / 失禁 / 神经局灶
> 7. **ECG**：长 QT / Brugada / 宽 QRS / AVB / 缺血?

## Part 1. 晕厥的定义与基础

### 1.1 定义

> **晕厥（Syncope）= 短暂意识丧失（LOC, Loss of Consciousness）+ 完全自发恢复**
> 
> - 持续 < 5 分钟（通常 < 1 分钟）
> - 由**脑灌注短暂不足**引起
> - 与癫痫 / conversion / 代谢性 LOC 区分

### 1.2 晕厥流行病学

- 一生患病率：35-40%
- 急诊 LOC 患者中约 50% 是晕厥
- **老年人晕厥首要病因：心源性**（> 50%）
- **年轻人晕厥首要病因：反射性**（> 50%）

> [!warning] 晕厥 ≠ 癫痫 ≠ 跌倒 ≠ 心脏骤停
> - **晕厥**：自行恢复 + < 1 min + 无 postictal
> - **癫痫**：有 aura + 抽搐 + postictal confusion
> - **跌倒**（无意识丧失）：意识清楚
> - **心脏骤停**：不自行恢复，需 CPR

## Part 3. 反射性晕厥 3 子型 ⭐

### 3.1 三子型对比表

| 维度 | **Vasovagal** | **Situational** | **Carotid Sinus Hypersensitivity** |
|---|---|---|---|
| 触发 | 情绪 / 看血 / 长站 / 疼痛 | **特定生理行为** | 颈部刺激（剃须 / 紧领 / 转头）|
| 典型患者 | 任何年龄（年轻多）| 中老年 + 慢病 | **> 60 岁** + 颈动脉粥样硬化 |
| 前驱 | **典型 5 联**（sweat / warm / 恶心 / 视暗 / 听弱）| 短暂或无 | 短暂或无 |
| HR 反应 | ↓（迷走）| ↓（迷走）| ↓（显著缓）|
| 诊断 | 病史 | 触发因素特殊 | **Carotid massage**（HR↓ > 3s 或 SBP↓ > 50）|
| 治疗 | 教育 + counterpressure | 治根本病 + 行为修改 | 起搏器（严重）|

### 3.2 Vasovagal Syncope 详解

#### 3.2.1 双相反射机制（Bezold-Jarisch reflex）⭐

```
T=0: 触发（情绪 / 疼痛 / 看血 / 长站）
   ↓
─── 第 1 相：交感激活 (T=0-30s) ───
- 心率↑ + 心肌强收缩
- 出汗↑（汗腺胆碱能交感）★ prodrome
- 温暖感（皮肤血流↑）★ prodrome
   ↓
─── 第 2 相：心室异常感受 (T=30-60s) ───
- 站立 → 下肢血液淤积 → 心室容量↓
- 但交感激活让心室强烈收缩 → "空射"
- 左室机械感受器（C 纤维）异常激活
- 误传"心脏过满"信号到脑干
   ↓
─── 第 3 相：反常迷走爆发 (T=60-90s) ───
- 副交感↑↑↑ → HR↓↓
- 交感↓↓↓ → 血管扩张
- BP 急剧↓
   ↓
─── 第 4 相：晕厥 ───
- 脑灌注↓ → 失意识
- 倒地后水平体位 → 血液回流 → 立即清醒
```

> [!warning] 关键反陷阱
> 晕厥时 HR ↓（迷走爆发），不是 ↑（很多人误以为心率代偿上升）。

#### 3.2.2 Vasovagal 5 联指纹（必背）

| # | 元素 | 描述 |
|---|---|---|
| 1 | **触发** | 情绪 / 疼痛 / 看血 / 长站 / 拥挤 |
| 2 | **Prodrome** | sweat + warm + 恶心 + 视暗 / 听弱 |
| 3 | **短暂** | < 1 分钟 |
| 4 | **自发恢复** | 立即清醒 |
| 5 | **无 postictal** | 无 confusion / 失忆 / 抽搐 |

**见这 5 个 → 直接诊断 vasovagal，不需要任何检查**

#### 3.2.3 Vasovagal 治疗阶梯

```
一线（教育 + 行为）：
- 识别 prodrome → 立即仰卧 / 坐下 + 抬腿
- Counterpressure maneuvers（防晕动作）：
  - 双腿交叉 + 收缩
  - 双手紧握 + 收缩前臂
  - 蹲下
- 避免诱因（脱水 / 长站 / 拥挤）
- 增加盐 + 水摄入

二线（顽固病例）：
- Midodrine（α₁ 激动）
- Fludrocortisone
- β-blocker（仅特定患者）
- 心理治疗（如焦虑诱发）

罕见：
- 起搏器（仅严重心抑制型 + 反复 + 不响应其他治疗）
```

### 3.3 Situational Syncope 8 种触发场景

| 触发 | 机制 | 经典 stem |
|---|---|---|
| **Micturition**（排尿）| Valsalva + 副交感 | 中老年男性 BPH + 夜间排尿 |
| **Defecation**（排便）| Valsalva + 反射 | 老年便秘用力排便 |
| **Cough**（咳嗽）| Valsalva 持续 | COPD + 长串咳嗽 |
| **Swallow**（吞咽）| 食管刺激迷走 | 吞冷饮 / 冰激凌 |
| **Weight-lifting**（举重）| Valsalva 屏气 | 健身房用力时 |
| **Sneeze / Laugh** | Valsalva 短爆发 | 大笑 / 喷嚏后 |
| **Trumpet / 吹奏 / 潜水** | 持续 Valsalva | 演奏 / 屏气潜水 |
| **Post-prandial**（餐后）| 内脏血流↑ | 老年人 30 min after large meal |

## Part 4. Orthostatic Hypotension（OH）两型 ⭐⭐⭐

### 4.1 OH 两型核心鉴别

| 维度 | **Volume-mediated**（容量型）| **Neurogenic / Autonomic**（神经源型）|
|---|---|---|
| **机制** | 血容量不足 | 反射弧受损 |
| **血容量** | 低 | 正常 |
| **反射弧** | 完好 | 受损 |
| **HR 反应** | **↑ ≥ 30 bpm** ⭐ | **不变 / ↑ < 10-15 bpm** ⭐ |
| **Hct** | **↑**（浓缩）| 正常 |
| **BUN/Cr** | **> 20**（prerenal）| 正常 |
| **K** | 可低 | 正常 |
| **黏膜** | **干燥** | 正常 |
| **皮肤** | 干 | 出汗异常（无汗 / 多汗）|
| **仰卧 BP** | 偏低 | **可正常 / 仰卧 HTN（反常）** |
| **附带症状** | 脱水征 | **胃轻瘫 / 便秘 / 阳痿 / 神经源膀胱** |
| **典型患者** | 年轻 / 急性 / 运动员 | **60yo DM / Parkinson / MSA / 高龄** |
| **病程** | 急性（小时-天）| 慢性进行性（月-年）|
| **治疗反应** | **补液后快速改善** | **顽固，需药物（midodrine）** |

### 4.2 类比理解

> [!success] OH 两型本质类比
> - **容量型 = "池子里水不够"**：年轻 / 急性 / HR 努力补救（但失败）
> - **神经源型 = "池子里水够但水泵开关坏了"**：老年 / 慢性 / HR 不动

### 4.3 脱水四联指纹（容量型 OH 必背）

```
见到这 4 个组合 → 锁定 hypovolemia：
1. Hct ↑（hemoconcentration，男 > 53%）
2. BUN/Cr 比 > 20（prerenal AKI）
3. 干燥黏膜
4. 低 K（利尿剂 / 出汗 / 醛固酮代偿）
```

🔗 详见 [[完整笔记/专题笔记/USMLE_实验室指纹诊断大全#6.7 Hypovolemia / 容量丢失（脱水四联）⭐]]

### 4.4 OH 病因 5 类速记（"湿干自主药老"）

| 类别 | 病因 |
|---|---|
| **湿**（容量丢失）| Hypovolemia：脱水 / 呕吐 / 腹泻 / 出汗 / 烧伤 / 失血 / **weight-cutting** |
| **干**（容量不补）| 摄入不足 / 长期卧床 / 痴呆患者 |
| **自主**（autonomic）| **DM / Parkinson / MSA / PAF / 高龄 / 淀粉样变 / 副肿瘤 / HIV / 酗酒 / 脊髓 T6 以上损伤** |
| **药**物 | α 阻滞（terazosin）/ TCA / 抗 HTN（β / CCB）/ 利尿 / 抗精神病 |
| **老**（内分泌 / 混合）| **PAI**（adrenal hypovolemia）/ 嗜铬细胞瘤 / HF / AS / 餐后 OH |

### 4.5 OH 诊断标准

```
床旁测试：
1. 仰卧 5 min → 测 BP + HR
2. 站立 1 / 3 min → 复测

诊断阳性：
- SBP ↓ ≥ 20 mmHg 或
- DBP ↓ ≥ 10 mmHg

分型（关键）：
- 容量型：HR ↑ ≥ 30 bpm（反射保留尝试代偿）
- 神经源型：HR 几乎不变 / ↑ < 10-15 bpm
```

### 4.6 OH 治疗阶梯

```
容量型治疗（多数补液后解决）：
1. 补液 + 补电解质
2. 停"作恶"药物（利尿剂 / α 阻滞）
3. 治根本病因（出血止血等）
4. 体位渐变

神经源型治疗（顽固，需长期管理）：
1. 非药物（首选）：
   - 体位渐变（30 秒规则）
   - 增 Na + 水（2-3 L/天）
   - 弹力袜（thigh-high）
   - 床头抬高 10-20°
   - 避热水浴 / 大餐 / 酒精
2. 药物（顽固病例）：
   - Fludrocortisone（保 Na 增容量）
   - Midodrine（α₁ 激动 → 血管收缩）⭐ 神经源 OH 关键药
   - Droxidopa（NE 前体）
   - Pyridostigmine
3. 治根本病（优化血糖等）
```

### 4.7 DM 自主神经病五联（神经源 OH 经典 stem）

```
长期 DM（> 10 年）+ HbA1c 高 + 以下任何组合：
1. 心血管：OH + 静息心动过速 + 心率变异性消失 + **仰卧 HTN**
2. 胃肠：胃轻瘫（早饱 / 餐后腹胀）+ 便秘 / 腹泻交替
3. 泌尿生殖：神经源性膀胱（残余尿 / 反复 UTI）+ **阳痿 (ED)**
4. 出汗：无汗症 / 异常出汗
5. 瞳孔：反应迟钝

→ 见 DM 患者 + OH + 任何 2 项 = 自主神经病高度怀疑
```

## Part 5. Cardiac Syncope（红旗类）⭐

### 5.1 心源性晕厥的两大病因

```
① 心律失常
   - 室速（VT）/ 室颤（VF）
   - 完全性 AV 阻滞 / Stokes-Adams 发作
   - 病态窦房结综合征（SSS）
   - 长 QT 综合征（LQTS）
   - Brugada 综合征
   - WPW

② 流出道梗阻 / 结构性心脏病
   - 肥厚梗阻型心肌病（HCM）
   - 主动脉狭窄（AS）
   - 大块肺栓塞（PE）
   - 心包填塞
```

### 5.2 心源性晕厥的临床特征

| 特征 | 心源性晕厥 |
|---|---|
| 前驱 | **无**（突发 "switch off"）|
| 触发 | **运动中** / 卧位 / 无诱因 |
| 持续 | 数秒 |
| 恢复 | 立即清醒（无 postictal）|
| HR | 异常（快 / 慢 / 不规则）|
| 心脏体检 | 可有杂音 |
| ECG | 多数异常 |
| 伴随 | 胸痛 / 心悸 / 呼吸困难 |
| 家族史 | **年轻猝死史**（< 50 岁）|

### 5.3 LQTS 三亚型 + Romano-Ward / J&L-N

| 维度 | **LQT1** | **LQT2** | **LQT3** |
|---|---|---|---|
| 基因 | KCNQ1 | KCNH2 (hERG) | SCN5A |
| 通道 | IKs↓ | IKr↓ | INa 持续 |
| **触发** | **运动 / 游泳** ★ | **听铃 / 惊吓 / 产后 / 低 K** | **睡眠 / 安静** |
| **ECG T 波** | 宽底 | 切迹 / 双相 | 长 ST + 晚出 T |
| 性别偏好 | 1:1 | **女性多** | 男性多 |
| 猝死率 | 中 | 中-高 | **最高** |
| β-blocker 反应 | **极好** | 中等 | **较差** |
| 特殊治疗 | 严格避游泳 | **补 K + Mg** | **Mexiletine (IB)** |

**触发口诀**："游泳 1，听铃 2，睡觉 3"
**ECG 口诀**："1 宽 2 双 3 晚"
**治疗口诀**："1 β 救命，2 补 K 防铃，3 美西律救场"

**Romano-Ward vs J&L-N**：
- Romano-Ward = AD 杂合，仅心脏
- J&L-N = AR 纯合（必是 KCNQ1）= 心脏 + **双侧感音神经性聋**

🔗 完整内容见 [[完整笔记/专题笔记/心内_Vaughan_Williams抗心律失常药]] Part 11

### 5.4 心源性晕厥的"红旗" 10 条 ⭐

口诀：**"运动卧 5 心家电杂老伤"**

1. **运动**中
2. **卧**位
3. **5** 分钟以上失意识
4. **心**悸 / 胸痛 / 呼吸困难
5. **家**族年轻猝死（< 50 岁）
6. 已知**结构性**心脏病
7. ECG / **电**生理异常
8. 心**杂**音
9. **老**年（> 60）
10. 严重外**伤**（提示突发无防御）

→ **任 1 项红旗 → 必进一步检查**（Echo / Holter / EP study）

### 5.4.x Cardiac Syncope 5 大病因 stem/ECG 速查表 ⭐（来源 Q4456）

| 病因 | 关键 stem 线索 | ECG 线索 |
|---|---|---|
| **Aortic stenosis / HCM** | **运动诱发**晕厥 + 收缩期杂音 | LVH（有杂音）|
| **VT** | 无前驱 + 心肌病 / 既往 MI | 持续 VT，**EF 降低** |
| **Sick sinus syndrome** | 前驱**疲劳 / 头晕** | **Sinus pauses** |
| **Advanced AV block / 间歇性 AV 阻滞** | 反复**无诱因**晕厥（老年 + 长期 HTN）| **PR 延长 / Bifascicular block / QRS 延长 / Dropped QRS** |
| **Torsades de pointes** | 无前驱 + **延长 QT 药物 / 低 K⁺ / 低 Mg²⁺** | **QTc 延长** |

### 5.4.y 间歇性 bradyarrhythmia 识别陷阱 ⭐⭐⭐（来源 Q4456）

> [!danger] 核心反陷阱：当下 ECG 正常 ≠ 排除间歇性心律失常
> - Bradyarrhythmia ≠ 当下心率慢 → **bradyarrhythmia = 心率有"间歇性变慢 / 停搏"的潜在风险**
> - 看 **传导间期（PR / QRS）+ 反复发作病史**，不是看当下心率数字
> - **PR 延长（AV 节点 / His 上段）+ QRS 延长（His-Purkinje）= 双系统弥漫病变** → bifascicular / trifascicular block → 高风险进展为完全性 AV 阻滞

```
间歇性 AV 阻滞机制：
正常时               间歇性发作
═══════════           ═════════════════
SA 64/min ✓          SA 64/min ✓
   │                    │
AV (PR 长)⚠️         AV (PR 长)⚠️
   │                    │
His ⚠️ 纤维化         His ⚠️ 突然完全失败
   │                    │
QRS 长⚠️             心室停搏几秒
   │                    │
心室 ✓               晕厥 → 自发恢复
                     → 下次 ECG"正常"
```

> [!success] 识别口诀
> - **"心率正常 ≠ 没有心律失常"** — 看传导间期 + 反复发作病史
> - **"PR 长 + QRS 长 + 老年晕厥 = pacemaker 候选人"**
> - 类比：**老化电线接触不良**。平时灯亮（心率正常），偶尔闪一下（晕厥）；ECG 拍到的是灯亮瞬间，但**电线本身（PR/QRS 延长）已告诉你它有问题** → 别只看灯亮不亮，看电线

### 5.4.z 传导系统病变 7 大病因（USMLE 高频）⭐（来源 Q4456）

| 病因 | 机制 | 典型场景 |
|---|---|---|
| **特发性纤维化**（Lenègre / Lev 病）⭐ | 老化 + HTN 累积损伤 | **老年 + 长期 HTN**（Q4456 经典）|
| **缺血性**（下壁 MI）| RCA → AV 节点供血 | 急性下壁 MI + 心动过缓 |
| **药物**：β / CCB / Digoxin / Amiodarone | AV 节点抑制 | 老年新用药后晕厥 |
| **Lyme 病** | 莱姆心肌炎 | 年轻 + 流行区旅行 + 红斑 |
| **风湿热** | 急性期 PR 延长 | 儿童 + GAS 感染后 |
| **结节病 / 淀粉样变** | 浸润性心肌病 | 心肌肥厚 + 传导异常 |
| **主动脉瓣手术 / TAVR 后** | 主动脉环邻近 His 束 | 术后新发 AV block |

🔗 详细 AV block 决策树 + atropine 起效部位 + 下/前壁 MI 对 AV block 的区别：见 [[完整笔记/专题笔记/心内_Vaughan_Williams抗心律失常药]] Part 7.6

### 5.5 心源性晕厥的检查

```
Step 1: 12 导联 ECG（必做）
   - QT 长 → LQTS
   - ST 抬高 V1-V3 → Brugada
   - δ 波 + 短 PR → WPW
   - 完全 AV 阻滞 → 传导病
   - 显著 LVH → HCM / AS
   - T 波倒置 V1-V3 → ARVC

Step 2: 超声心动图（Echo）
   - 结构性心脏病
   - 流出道梗阻
   - LV 功能
   - 心包积液

Step 3: 动态 ECG 监测
   - 24h Holter（如发作频繁）
   - 事件记录器 30 天（如发作不频繁）
   - 植入式监测器 ILR（罕见但严重）

Step 4: 电生理 EP 研究
   - 怀疑严重心律失常时
```

## Part 7. 非晕厥鉴别诊断（Conversion / 代谢性）⭐⭐⭐

### 7.1 Seizure（最重要鉴别）

详见 **Part 8 完整 Seizure vs Syncope 鉴别**

### 7.2 Conversion Disorder / Psychogenic Pseudosyncope ⭐⭐⭐

#### 7.2.1 定义

> **Conversion Disorder（转换障碍）= 现称 Functional Neurological Symptom Disorder (FND)**
> 
> 患者出现真实的神经系统症状（运动 / 感觉 / 意识 / 感官异常），但**无器质性病变**能解释，且**不是装病**。

```
核心特征：
1. 神经系统症状（运动 / 感觉 / 意识 / 感官）
2. 检查显示"症状与解剖 / 生理不符"
3. 排除器质性病因
4. 不是装病 ★ 关键
5. 通常（但不强求）有心理应激背景
```

#### 7.2.2 4 大症状群

| 类别 | 表现 | 占比 |
|---|---|---|
| **运动症状** | 假性瘫痪 / 假性步态异常（astasia-abasia）/ 假性震颤 | ~50% |
| **感觉症状** | 不符神经分布的感觉缺失 / 麻木 / 假性失明（管状视野）| ~30% |
| **假性发作 / 假性晕厥** ★ | PNES / Psychogenic pseudosyncope | ~20% |
| **特殊感官** | 假性失语 / 失声 / 失聪 / 吞咽困难 | 较少 |

#### 7.2.3 La Belle Indifférence ⭐（USMLE 经典考点）

> **La belle indifférence**（法语 / la bɛl ɛ̃difeʁɑ̃s /）
> 字面意思："the beautiful indifference"（美丽的冷漠）
> 临床含义：**患者对自己严重的神经症状表现出异常平静、漠不关心**

**经典例子**：
- 患者左侧肢体完全瘫痪
- 正常反应应是焦虑、急切要求检查
- La belle indifférence：平静叙述 + 像讨论天气 + 甚至微笑 + 不积极询问预后

**历史地位**：
- 1859 Briquet 首次描述
- Charcot（19 世纪法国神经学之父）强调
- Freud 沿用并理论化（"症状已解决心理冲突，所以患者不焦虑"）
- DSM-III 至 IV 列为经典特征

**现代证据**：
```
- conversion 患者中只 21% 有
- 器质性疾病患者中也 29% 有（如 stroke）
- 敏感性差 + 特异性差
- DSM-5 已从必要标准移除
```

> [!warning] USMLE 答题注意
> 虽然现代证据显示 la belle indifférence 不够特异，但 **USMLE 题里仍把它作为经典提示** — 这是教科书答题，不是临床决策。
> 
> 看到 stem 写：
> - "She appears unconcerned about her sudden inability to walk"
> - "He shows surprising lack of concern about his blindness"
> - "Despite complete paralysis, she calmly describes her symptoms while smiling"
> 
> → **几乎确定是 Conversion Disorder**

#### 7.2.4 现代更可靠的"positive signs"（USMLE 也考）

```
运动症状：
- Hoover 征（瘫痪侧无对侧抗阻反射）★ 最可靠
- Drift without pronation（抬手不旋前 = 假瘫）
- Give-way weakness（抗阻力时给-放）

感觉症状：
- Midline splitting（感觉异常正好在中线）
- "Stocking" / "Glove" 分布（不符神经分布）

假性发作：
- 闭眼抗阻 ★ 强特征
- 眼睑震颤 / 眼球向下
- 不对称肢体活动 / 骨盆推动
- 持续 > 5-10 min
- 无 postictal confusion
- 正常乳酸 / CK
- 发作期 EEG 正常
```

#### 7.2.5 假性晕厥（Psychogenic Pseudosyncope）10 个识别特征

```
高度提示 conversion / psychogenic pseudosyncope：
1. 持续时间 > 5-10 分钟 ★
2. 多次发作 / 反复
3. 闭眼 + 抗张眼检查
4. 倒下时有"保护性"动作
5. 倒下后 BP / HR 正常
6. 有目击者时更易发作
7. La belle indifférence
8. 心理创伤 / 应激背景
9. 无客观损伤（侧舌咬 / 大小便失禁罕见）
10. 多种神经主诉同时存在
```

#### 7.2.6 假性晕厥 vs 真癫痫（PNES vs Seizure）

| 特征 | **PNES**（假性）| **真癫痫** |
|---|---|---|
| **持续** | **> 5 min**（多 10-30 min）| < 2-5 min |
| **抽搐模式** | **非典型**（不对称、骨盆推动、左右交替）| **节律性对称**强直阵挛 |
| **眼** | **闭眼抗阻** ⭐ | 睁开（凝视一侧）|
| **舌咬** | 舌尖（少见）| **侧舌** ⭐ |
| **失禁** | 罕见 | 常有 |
| **意识** | 部分保留 | 完全丧失 |
| **乳酸**（10 min 后）| **正常** ⭐ | **> 2.5 mmol/L** ⭐ |
| **CK**（24h）| 正常 | **明显升高** |
| **EEG 发作期** | 正常 | 异常 |
| **抗癫痫药** | **无效** | 有效 |

> [!success] USMLE 客观指标金标准
> **发作后 10-15 min 乳酸 / CK** 是最可靠区分：
> - 真癫痫：乳酸↑ + CK↑（真正强直阵挛 + 缺氧 + 肌肉损伤）
> - PNES：完全正常

#### 7.2.7 4 个精神科诊断完整鉴别 ⭐⭐⭐

| 诊断 | 症状真实？| 是否有意识？| 动机 |
|---|---|---|---|
| **Conversion Disorder** | **真实**（患者感受到）| **无意识** | 无清晰动机；潜在心理冲突 |
| **Factitious Disorder** | 真实或制造 | **有意识** | **想扮演"病人角色"**（无外部利益）★ Munchausen syndrome |
| **Malingering**（诈病）| 制造 | **有意识** | **明确外部利益**（金钱 / 逃避义务 / 法律）|
| **Somatic Symptom Disorder** | 真实 | 无意识 | 过度担心 + 反复检查 |

#### 7.2.8 4 个诊断的 USMLE stem 模板

**Conversion**：
> "32yo woman 发现丈夫出轨**第二天**突发左下肢瘫痪，Hoover 征阳性，**She seems unconcerned about her inability to walk, smiling while describing symptoms**"

**Factitious (Munchausen)**：
> "45yo woman with **17 prior hospitalizations**, works as nurse aide, **refuses contact with family**, seems satisfied with workups"

**Malingering**：
> "28yo man with low back pain after work injury, **applying for workers' compensation**, symptoms inconsistent on exam, exaggerates pain only when observed"

**Somatic Symptom**：
> "35yo man with **6 months of chronic abdominal pain**, **12 doctors / 8 normal endoscopies**, **convinced he has stomach cancer**"

#### 7.2.9 USMLE 4 诊断决策树 ⭐

```
LOC / 神经症状 → 4 诊断决策：
   │
   ▼
器质性原因排除 (MRI / EEG / labs 正常)?
   ┌──No──→ 器质性疾病
   │
   ▼ Yes
   症状是真实的吗?
   ┌──No──→ Factitious / Malingering
   │        │
   │        有外部利益?
   │        ┌──Yes──→ Malingering（诈病）
   │        │
   │        ▼ No
   │        想做病人角色?
   │        └──Yes──→ Factitious (Munchausen)
   │
   ▼ Yes
   症状是神经症状（瘫 / 麻 / 感官异常）?
   ┌──Yes──→ Conversion Disorder ★
   │
   ▼ No
   过度担心 + 反复就医?
   └──Yes──→ Somatic Symptom Disorder
```

#### 7.2.10 Conversion 处理（USMLE 考点）

```
关键原则：
1. ★ 不要直接说"是心理的 / 是假的"
   → 会让患者感觉被否定 / 撤回信任
   → 应说"症状真实，但检查显示不是结构性病变"
2. 共情 + 验证症状真实性
3. 强调好的预后
4. 不做不必要的检查（防强化"病人角色"）

治疗阶梯：
Step 1: 教育 + 解释（"大脑发出错误信号，但结构是好的"）
Step 2: 物理治疗（运动症状）
Step 3: ★ CBT（首选心理治疗）
Step 4: SSRIs（如有焦虑 / 抑郁共病）
Step 5: 多学科团队（神经 + 精神 + 物理 + 心理）

预后好的因素：
- 急性起病 / 短病程 / 已识别应激源 / 早期治疗

预后差的因素：
- 慢性 (> 6 个月) / 多种共病 / 继发性获益 / 依从性差
```

#### 7.2.11 现代神经科学理解（背景）

```
不是"装病"，也不是"纯心理"——是真正的脑功能异常：
- 边缘系统（情绪）→ 运动皮层异常抑制
- "运动意图"信号被异常抑制
- fMRI 可见功能性异常（但结构正常）

这就是 DSM-5 改名 "Functional Neurological Symptom Disorder" 的原因。

La belle indifférence 现代解释：
- 不是"不在乎"
- 是边缘系统 → 运动皮层异常通路同时影响情绪反应
- 解离（dissociation）的体现
```

### 7.3 代谢性 LOC（不算晕厥但常被误为）

|---|---|
| **低血糖** | 糖尿病 + 胰岛素 / 磺脲药 + 出汗 + 心悸 + 渐进 confusion → **不立即清醒** |
| **低 Na**（严重）| Na < 125 + 头痛 + N/V + 进行性 confusion → 不立即清醒 |
| **CO 中毒** | 头痛 + 群发性 + 不典型晕厥 |

## Part 8. Seizure vs Syncope 完整鉴别 ⭐⭐⭐（核心）

### 8.1 Seizure 5 期完整时序

```
T= -hours/days  ── Prodrome（前驱期，模糊）
                   数小时-数天的非特异不适
                   烦躁 / 疲劳 / 头痛 / 失眠 / 情绪改变
                   ★ 不是所有患者都有
                   ★ 不属于 ictal phase
                          ↓
T= -seconds     ── Aura（先兆，局灶性）⭐
                   数秒-30 秒
                   局灶性症状（嗅觉 / 视觉 / 感觉 / déjà vu）
                   ★ 已经是发作的一部分（focal seizure）
                   ★ 反映癫痫起源脑区
                          ↓
T= 0            ── Ictal phase（发作期）
                   强直 / 阵挛 / 失张力 / 失神 / automatism
                   持续数秒-数分钟
                          ↓
T= +minutes     ── Postictal phase（发作后期）⭐⭐⭐ 金标准
                   意识混乱 / 嗜睡 / Todd 麻痹
                   持续 5-30 分钟以上
                          ↓
T= +hours       ── Recovery（恢复期）
                   完全恢复（数小时）
```

### 8.2 Seizure 内部"prodrome vs aura"区别 ⭐

| 维度 | **Seizure Prodrome（前驱期）** | **Seizure Aura（先兆）** |
|---|---|---|
| **时间** | 数小时-数天 | 数秒-30 秒 |
| **性质** | 非特异，模糊不适 | **局灶性，特异** |
| **是否癫痫电活动** | 否（推测是神经化学改变）| **是**（focal seizure 的开始）|
| **EEG** | 通常正常 | **局灶性放电** |
| **常见症状** | 烦躁 / 疲劳 / 头痛 / 失眠 / 情绪改变 | 嗅觉幻觉 / 视觉幻觉 / déjà vu / 上腹不适 / 麻木 |
| **临床意义** | 患者自我警报 | 提示**癫痫起源部位** |

> [!success] 核心区别
> **Prodrome 在发作"前"（不是发作本身），Aura 是发作"开始"（已是 focal seizure）**。
> USMLE stem 写 "felt anxious for several days before episode" = prodrome；
> 写 "smelled burning rubber 10 seconds before LOC" = aura。

### 8.3 Vasovagal vs Seizure 完整鉴别表 ⭐⭐⭐（最终金标准）

| 维度 | **Vasovagal** | **Seizure** |
|---|---|---|
| **触发** | 情绪 / 看血 / 长站 | 闪光 / 睡眠剥夺 / 停药 |
| **前驱性质** | **Prodrome**（反射反应）| **Aura**（局灶发作）|
| **前驱时长** | 30-60 秒 | 数秒-30 秒 |
| **前驱症状** | 全身性（sweat / warm / 恶心 / 视暗）| 局灶性（嗅觉 / 视觉 / 感觉 / déjà vu）|
| **意识** | 渐进失意识 | 突然失意识（局灶后扩散到全身）|
| **抽搐** | **罕见**（短暂缺氧 "convulsive syncope" 可有）| **常有**（强直 / 阵挛）|
| **舌咬伤** | 罕见 | **常有**（**侧舌咬** ⭐）|
| **大小便失禁** | 罕见 | 常有 |
| **持续** | < 1 min | 数分钟 |
| **恢复** | **立即清醒** | **Postictal confusion**（5-30 min+）⭐ 金标准 |
| **HR** | ↓（迷走介导）| 不定 |
| **诊断** | 病史 | **EEG / MRI** |

> [!success] 终极金标准
> **判断 seizure vs vasovagal = "postictal confusion"**
> - **立即清醒 → 晕厥**
> - **延迟清醒 + confusion → seizure**
> 
> 其他特征都可能"重叠"（如 convulsive syncope 也有短暂抽搐），唯独 postictal confusion 是 seizure 的"印章"。

### 8.4 Seizure Aura 各脑叶定位 ⭐

| Aura 类型 | 起源脑叶 | 典型 stem 描述 |
|---|---|---|
| **嗅觉幻觉**（焦糊味）| **颞叶** | "smelled burning rubber" |
| **Déjà vu**（"似曾相识"）| **颞叶** | "feeling I've been here before" |
| **Jamais vu**（"陌生感"）| **颞叶** | "familiar place suddenly felt unfamiliar" |
| **上腹 rising sensation** | **颞叶**（最常见 55%）| "something rising from stomach to chest" |
| **视觉幻觉**（闪光 / 几何图案）| **枕叶** | "flashing lights / geometric patterns" |
| **听觉幻觉**（嗡嗡 / 音乐）| **颞叶上回** | "buzzing sound / music" |
| **躯体感觉异常**（一侧麻木）| **顶叶** | "numbness / tingling in one limb" |
| **运动**（局部抽搐）| **额叶** | "twitching of one arm" |
| **恐惧 / 焦虑**（强烈情绪）| **杏仁核 / 颞叶内侧** | "sudden intense fear without reason" |
| **自主神经**（潮红 / 心悸 / 出汗）| **岛叶 / 颞叶** | ⚠️ 易误为 vasovagal |

### 8.5 Déjà vu 详解

```
Déjà vu = 法语"already seen"
体验："这个新场景我以前经历过"的强烈错觉

两种来源：
① 正常生理（60-70% 健康人有过，多 15-25 岁）
   - 偶发 + 数秒 + 无伴随 → 无病理意义
② 颞叶癫痫 aura ⭐ USMLE 考点
   - 反复频繁 + 伴其他症状（嗅觉 / 上腹 / 恐惧）
   - 后续 ictal phase（automatism / 失意识 / postictal）

机制：
颞叶内侧"熟悉感开关"被错误激活
（海马 + 杏仁核 + 内嗅皮层异常放电）
```

**相关词**：
- **Déjà vécu**（"已经经历过"）：更强版本，几乎专属癫痫
- **Jamais vu**（"从未见过"）：相反体验，也见于颞叶癫痫
- **Presque vu**（"几乎见过"）："话到嘴边却说不出"，正常人常有

**Déjà vu 判别口诀**：
- "偶发 + 无伴随" = 正常（无病理）
- "反复 + automatism / postictal" = 颞叶癫痫 aura

### 8.6 颞叶癫痫"三联"特征

```
① Aura（数秒）
   - 上腹 rising / déjà vu / 嗅觉 / 恐惧
② Ictal phase（1-2 min）
   - Automatism（自动症）★
     - Lip-smacking（咂嘴）⭐ 最经典
     - Chewing / picking at clothes / wandering
   - Impaired awareness（意识受损）
③ Postictal（15-30+ min）
   - Confusion / 头痛 / 嗜睡
   - 不记得发作过程
```

**治疗**：抗癫痫药（Carbamazepine / Lamotrigine / Levetiracetam）

### 8.7 Convulsive Syncope 陷阱 ⭐

```
晕厥时短暂抽搐 (myoclonic jerks)：
- 脑灌注↓ → 皮层抑制解除 → 短暂运动激活
- 持续 < 15 秒
- 无 postictal ⭐
- 病史符合晕厥（有触发 + 前驱）
→ 仍是 vasovagal / OH，不是 seizure

真正 seizure：
- 抽搐 > 1 min
- Postictal confusion
- 舌咬伤 / 失禁
- 无典型晕厥触发
```

> [!warning] USMLE 陷阱
> **"晕厥 + 短暂抽搐"不要立即诊断 seizure**。区别在 postictal + 持续时间。

## Part 9. 晕厥 7 维分类框架 ⭐⭐⭐（核心方法论）

口诀：**"触前心醒，背伴 ECG"**

| 维度 | 主作用 | 关键鉴别 |
|---|---|---|
| **1. 触发** | 区分各亚型 | 运动 = Cardiac；情绪 = Vasovagal；站起 = OH；排尿 = Situational；闪光 = Seizure |
| **2. 前驱** | 反射 vs 心源 vs 癫痫 | 有 prodrome = 反射 / OH；无前驱 = Cardiac；aura（局灶）= Seizure |
| **3. HR** | 区分反射 vs OH 亚型 | ↓ = Vasovagal / Situational；↑ ≥ 30 = 容量 OH；不动 = 神经源 OH |
| **4. 清醒** | 晕厥 vs 癫痫 vs Conversion | 立即 = 晕厥；延迟 + postictal = Seizure；> 5min = Conversion |
| **5. 背景** | 调整概率权重 | > 60 → 心源↑；DM → 自主 OH↑；BPH → 排尿性 Situational |
| **6. 伴随** | 排查严重病因 | 胸痛 → MI / PE；舌咬 → Seizure；腹痛 → AAA / 异位 |
| **7. ECG** | 心源筛查 | 异常 = 锁定 Cardiac 亚型 |

## Part 11. Tilt-Table Testing 精确适应症

### 11.1 4 大适应症

```
1. 反复不明原因晕厥 + 床旁评估阴性
2. 鉴别 vasovagal vs OH（典型场景困难）
3. 怀疑 POTS（年轻女性 + 站立心悸 + BP 不降）
4. 评估治疗效果 / 长期随访
```

### 11.2 不需要 tilt-table

```
❌ 典型 vasovagal
❌ 单次发作
❌ 床旁 OH 测试已阳性
❌ 已知心源性 → 应做 Echo / Holter
❌ 已知癫痫 → 应做 EEG / MRI
```

### 11.3 原理

**被动倾斜（60-70°）= 去掉"肌肉泵代偿"，纯粹考察自主神经反射 + 血容量**。比主动站立更能诱发隐匿性 vasovagal / OH。

### 11.4 操作流程

```
Step 1: 准备
- 平卧 10-20 min（基线稳定）
- 测基础 BP + HR
- 禁食 4 小时

Step 2: 倾斜（被动）
- 床头抬高到 60-70°
- 患者不主动用力站立（足踏板支撑）
- 持续 30-45 分钟
- 每 2-3 分钟测 BP + HR

Step 3: 药物激发（如果被动测试阴性）
- 给 isoproterenol 或 sublingual nitroglycerin
- 继续监测 15-30 分钟

Step 4: 复位
- 床回到水平
- 监测恢复

总测试时间：60-90 分钟
```

### 11.5 结果判读（VASIS 分类）

```
3 种阳性反应：
- Type 1: Mixed type（混合型）— HR↓ 但 > 40 + BP↓
- Type 2: Cardioinhibitory（心抑制型）— 严重 HR↓ 或停搏
- Type 3: Vasodepressor（血管抑制型）— HR 稳定 + BP↓↓

其他模式：
- 立即 BP 下降（< 3 min）+ HR 不动 = Classic OH（神经源型）
- 立即 BP 下降 + HR↑↑ ≥ 30 = POTS / 容量型 OH
- 渐进 BP 下降 = Delayed OH
- 无变化 = 阴性
```

## Part 12. POTS（姿势性心动过速综合征）

### 12.1 诊断标准

```
POTS = Postural Orthostatic Tachycardia Syndrome：
- 站立后 10 分钟内：
  - HR↑ ≥ 30 bpm（青少年 ≥ 40 bpm）
  - 但 SBP 不下降 ≥ 20 mmHg ⭐ 关键
- 伴慢性症状（头晕 / 心悸 / 疲劳 / 脑雾）
- 症状持续 ≥ 3 个月
```

### 12.2 临床特征

| 维度 | POTS |
|---|---|
| 好发人群 | **年轻女性（15-50 岁）** |
| 触发因素 | 病毒感染后 / 妊娠 / 创伤 / 慢性疾病 |
| 关键症状 | 站立时心悸 + 头晕 + 疲劳 + 脑雾 + 运动不耐 |
| 体位 BP 反应 | **不下降** ⭐（不同于 OH）|
| 体位 HR 反应 | **大幅升高** ⭐ |
| 诊断 | **Tilt-table testing**（金标准）|
| 治疗 | 增 Na / 水 + 弹力袜 + β-blocker / ivabradine / midodrine |
| 与长 COVID 关联 | 高度相关 |

> [!warning] POTS vs OH 关键区别
> - **OH**：BP **下降** + HR 不定
> - **POTS**：BP **不下降** + HR **大幅升高**
> 
> POTS 容易被误诊"焦虑" — 血压正常但心率快。

## Part 13. USMLE 答题策略 + Memory Hook

### 13.1 答题流程（5 步）

```
Step 1: 用 7 维框架扫读（30 秒）
   "触前心醒，背伴 ECG"

Step 2: 核对红旗 10 条（30 秒）
   "运动卧 5 心家电杂老伤"

Step 3: 形成诊断 + 确认指纹
   - Vasovagal 5 联
   - 脱水四联
   - 5 大类决策表

Step 4: 决定"查"还是"不查"
   - 典型 + 无红旗 → No further testing
   - 不典型 / 红旗 → 具体检查

Step 5: 选择答案
   - 是否有"No further testing"选项？
   - 是否有适当的具体检查？
```

### 13.2 "No further testing" 经典场景 ⭐⭐⭐

```
USMLE 越来越多考"何时 NOT to test"：

经典场景：
1. ★ 典型 vasovagal（Q21579）
2. ★ 典型 GERD（年轻 + 经典烧心 + 无报警征 → PPI 试验）
3. ★ 单纯性头痛（无红旗 → 不做 CT）
4. ★ 自限性病毒感染（流感 / 普通感冒 → 支持治疗）
5. ★ 典型偏头痛（无红旗 → 不做影像）
6. ★ 低风险 ACS 评分（HEART score 低 → 出院）
```

**答题方法论**：
1. 诊断已经清楚吗？
2. 进一步检查能改变处理吗？
3. 选项里有 "No further testing" 吗？很可能是答案

### 13.3 Memory Hook 集合

#### 13.3.1 晕厥鉴别 "前 5 后" 速记

> **前**驱症状 → 反射性或 orthostatic（**有前驱**）
> 突发**无**前驱 → 心源性
> **5** 分钟以上无意识 → Conversion disorder
> 苏醒**后** confusion → 癫痫（postictal）

#### 13.3.2 OH 病因 5 类口诀

> **"湿干自主药老"** = 容量丢失 / 摄入不足 / 自主神经病 / 药物 / 内分泌混合

#### 13.3.3 OH 两型类比

> **"水池水不够（容量）vs 水泵坏（神经源）"**

#### 13.3.4 神经源 OH 反常征

> **"DM + OH + 仰卧 HTN + 胃轻瘫 + ED" = 自主神经病五联**

#### 13.3.5 心源性晕厥红旗

> **"无前驱突发 + 运动 / 卧位 + ECG 异常 + 家族猝死"**

#### 13.3.6 LQT 三亚型 3 套口诀

> 触发："游泳 1，听铃 2，睡觉 3"
> ECG："1 宽 2 双 3 晚"
> 治疗："1 β 救命，2 补 K 防铃，3 美西律救场"

#### 13.3.7 颞叶癫痫 4 大经典 aura

> 1. **上腹 rising sensation**（最常见 55%）
> 2. **Déjà vu / Jamais vu**
> 3. **嗅觉幻觉**（焦糊味）
> 4. **恐惧 / 焦虑感**（无外因）
> → 后续常出现 **lip-smacking automatism**（咂嘴自动症）

#### 13.3.8 终极金标准

> **Postictal confusion = Seizure 印章**
> - 立即清醒 → 晕厥
> - 延迟清醒 + confusion → seizure

#### 13.3.9 选对题深挖机制方法论 ⭐⭐⭐

> [!success] NBME 230 → 250 关键
> 直觉对 ≠ 机制懂。**选对的题尤其要问**：
> 1. 为什么答案是这个？
> 2. 如果 stem 改一个细节（年龄 / 实验室 / 体征），答案变吗？
> 3. 这个诊断的"亚型 / 分型"是什么？
> 4. 每种亚型的鉴别要点 + 治疗差异？

#### 13.3.10 "过度评估倾向"修正 ⭐

> [!warning] Q19889 + Q21579 第 2 次掉同陷阱
> 见 "next step" 题 → 先问"诊断清楚吗？无红旗吗？" → 答案可能是"不查 / 观察 / 出院"

## Part 14. 运动员晕厥（USMLE 高频）⭐

### 14.1 摔跤 / 搏击运动员 3 类考点

| 考点 | 临床表现 |
|---|---|
| **Weight-cutting 致脱水** | Orthostatic + 脱水四联 + 低 K（**Q21638**）|
| **运动中猝死** | HCM / Brugada / LQTS / **Commotio cordis**（胸前撞击 → R-on-T → VF）|
| **AAS（合成代谢类固醇）滥用** | 男化 / 不孕 / HCM / 肝瘤 / 痤疮 / 心衰 |
| **过度训练综合征** | 慢性疲劳 + 表现下降 + 抑郁 |
| **Female Athlete Triad** | 月经异常 + 骨密度低 + 进食障碍 |

### 14.2 运动员心电图特征（生理性 vs 病理性）

| 生理性（可接受）| 病理性（需进一步评估）|
|---|---|
| 窦缓（≥ 30 bpm）| 严重窦缓（< 30）+ 症状 |
| 1° AVB / Mobitz I | Mobitz II / 完全 AVB |
| 早期复极 | **Brugada pattern** |
| 左室高电压 | **HCM**（深 Q + LV 肥厚）|
| 早搏（罕见）| 复杂室速 / 多形 VT |
| 窦性心律不齐 | **长 QT**（QTc > 480）|

### 14.3 PPE（赛前体检）红旗

> [!danger] 任 1 条 → 心脏科评估
> - 运动中或刚停止后晕厥
> - 不明原因运动相关胸痛 / 呼吸困难
> - 家族猝死史（< 50 岁）
> - 已知心律失常综合征 / HCM
> - 既往 Marfan 表现（高瘦 + 长指）

## Part 16. 高频陷阱总结

| 陷阱 | 正确认知 |
|---|---|
| ❌ 晕厥都要查 Holter / Echo / Tilt-table | ✓ 典型 vasovagal 病史诊断，不需检查 |
| ❌ Vasovagal = 单纯迷走过强 | ✓ 是**双相反射**（先交感激活，后迷走爆发）|
| ❌ 晕厥时 HR 应该上升代偿 | ✓ vasovagal 时 HR 反而↓（迷走介导）|
| ❌ Prodrome 有"冰冷苍白" | ✓ 典型 prodrome 是"sweat + warm"（第 1 相交感激活）|
| ❌ Tilt-table 是 vasovagal 诊断金标准 | ✓ 典型 vasovagal 靠病史；tilt-table 用于反复 / 不典型 |
| ❌ OH 就是脱水 | ✓ 神经源型容量正常；不是所有 OH 都补液能好 |
| ❌ HR↑ 排除 OH | ✓ HR↑ 反而是容量型特征 |
| ❌ HR 正常排除 OH | ✓ 神经源型 HR 几乎不变才是诊断特征 |
| ❌ 仰卧 BP 正常 / HTN 排除 OH | ✓ 仰卧 HTN 反而支持神经源型（反常征）|
| ❌ DM 患者晕厥 = 低血糖 | ✓ 也要考虑自主神经病致 OH |
| ❌ 看到"心梗家族史"就选心源性检查 | ✓ 父亲 MI（非年轻猝死）不构成红旗 |
| ❌ 反射性晕厥只有 vasovagal | ✓ 还有 Situational 和 Carotid sinus 两子型 |
| ❌ 运动中晕厥用 vasovagal 框架 | ✓ 运动中 = Cardiac 红旗（HCM / LQT1 / AS）|
| ❌ Seizure prodrome = Seizure aura | ✓ Prodrome 是数小时-天的模糊不适；Aura 是数秒的局灶性发作开始 |
| ❌ Déjà vu 一定是癫痫 | ✓ 正常人 60-70% 有过；**反复 + 伴随症状**才是颞叶癫痫 aura |
| ❌ 看到"晕厥 + 短暂抽搐"诊断 seizure | ✓ 可能是 convulsive syncope（无 postictal）|
| ❌ 颞叶癫痫的自主症状（心悸 / 出汗）= vasovagal | ✓ 岛叶 / 颞叶 aura 可包含这些；区别在 postictal |
| ❌ 心源性 stem 选 CT head | ✓ 应选 Echo / Holter（cerebrovascular 晕厥必有局灶征）|
| ❌ 单次典型 vasovagal 做 tilt-table | ✓ 不必要 |
| ❌ Conversion disorder = 装病 | ✓ 是真实症状但无器质性病变；与 Malingering / Factitious 不同 |
| ❌ La belle indifférence 一定是 Conversion | ✓ 敏感性差，但 USMLE 仍把它作经典提示 |

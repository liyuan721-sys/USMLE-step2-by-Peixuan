# 补液：晶体 / 胶体对照（IV Fluids 选液速查）

> [!info] 用途
> Step 2 "该给哪种液体"题的统一速查。核心：先按**渗透压**（等渗 / 低渗 / 高渗）分三档 → 对应 **ECF / ICF** 移动方向 → 锁定临床适应证。派生自 [[完整笔记/专题笔记/急诊/急诊_ER_DrJ急诊医学大纲]]（Dr. J ER）补液章；与 [[完整笔记/专题笔记/肾脏/肾脏_低钠血症诊疗SOP]]、[[完整笔记/专题笔记/外科/外科_烧伤管理速查]]（Parkland）配套。

## 一、五液对照金表 ★

| 溶液 Solution | 渗透 Osmolarity | ECF 细胞外 | ICF 细胞内 | 临床用途 Clinical Use |
|---|---|---|---|---|
| **0.9% NS** 生理盐水 (Normal Saline) | 等渗 Isotonic | ↑ | — | 出血、呕吐、腹泻（等渗性丢失）；浓缩红细胞同输液 |
| **D5W / 0.45% NS** | 低渗 Hypotonic | ↓ | ↑ | 精神性多饮 (psychogenic polydipsia)、维持补液（低渗性获得） |
| **3% saline / mannitol** 高渗盐 / 甘露醇 | 高渗 Hypertonic | ↑ | ↓ | 症状性低钠、脑水肿/↑ICP、DI 尿崩、酒精性脱水（低渗性丢失） |
| **Lactated Ringer's (LR)** 乳酸林格 | 等渗 + 电解质 | ↑ | — | 创伤 / 烧伤复苏（Parkland）；**避免与浓缩红细胞同输** |
| **Colloid (Albumin)** 白蛋白 | ↑ 胶体渗透压 | ↑ | — | 肝硬化 SBP / 大量放腹水、肝肾综合征、肾病综合征、肝衰竭 |

> [!tip] 三档反射
> - **等渗**（NS / LR）→ 只填 ECF，不进细胞 → 复苏、补容量首选
> - **低渗**（D5W / ½NS）→ 水最终进 ICF → 纠自由水缺失 / 维持
> - **高渗**（3% / 甘露醇）→ 把水从 ICF 拉回 ECF → 脑水肿、症状性低钠

## 二、D5W 的"伪等渗"陷阱 ★

> [!warning] D5W 标称等渗，体内表现为低渗
> **5% 葡萄糖水 (Dextrose 5% in Water, D5W)** 在袋子里和血浆等渗，但葡萄糖一旦被细胞摄取代谢 → 只剩自由水 → **等效于纯水（低渗）** → 水随之进入细胞 (ICF↑)。
> 所以 D5W **不能用于复苏**（不留在血管内、不升血压），只用于补自由水 / 配药载体。

### 高血糖的假性低钠校正

> [!info] 血糖每 +100 mg/dL（>200 部分）→ 实测血钠 ↓ 约 1.6–2.0 mEq/L
> 高血糖把水拉到血管内稀释钠 → 测得"低钠"是**假象**。校正钠 = 实测钠 + 2 ×〔(血糖−100)/100〕。糖尿病高血糖危象见 [[完整笔记/专题笔记/肾脏/肾脏_低钠血症诊疗SOP]]。

## 三、LR vs NS：与血制品同输的铁律 ★

> [!danger] Lactated Ringer's 不能与浓缩红细胞 (packed RBC) 同输
> LR 含 **Ca²⁺**；血制品里的 **枸橼酸 (citrate) 抗凝剂会结合钙** → 一方面诱发**凝血**（管路内血凝块），另一方面理论上加重**低钙血症**。
> - **浓缩红细胞 (PRBC) → 必须用 0.9% 生理盐水 (NS) 同输**（不用 LR、不用含钙 / 含糖液）。
> - 大量输血时枸橼酸结合钙 → 监测**离子钙**，必要时补钙。

| 维度 | 0.9% NS 生理盐水 | Lactated Ringer's 乳酸林格 |
|---|---|---|
| 成分 | Na⁺ Cl⁻（高氯） | Na⁺ K⁺ Ca²⁺ 乳酸（更接近血浆） |
| 大量输注副作用 | **高氯性代谢性酸中毒** | 乳酸代谢为 HCO₃⁻，较少酸中毒 |
| 与 PRBC 同输 | ✅ 可 | ❌ 禁（含钙） |
| 高钾 / 肾衰 | 优先（不含钾） | 含 K⁺，严重高钾慎用 |
| 创伤 / 烧伤大复苏 | 可，但大量→高氯酸中毒 | ✅ 首选（Parkland 用 LR） |

## 四、尿量复苏目标

> [!info] Urine output goal（复苏终点比公式更重要）
> - **成人**：0.5–1 mL/kg/hr
> - **婴儿 / 儿童**：≥ 1 mL/kg/hr
> - 烧伤复苏滴定尿量，不死守 Parkland 算出的数字（见 [[完整笔记/专题笔记/外科/外科_烧伤管理速查]]）。

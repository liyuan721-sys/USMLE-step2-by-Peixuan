# 肾脏 — 肾炎综合征 (Nephritic) vs 肾病综合征 (Nephrotic) 对照

> [!info] 笔记定位
> Nephritic vs Nephrotic 是肾小球病的总分水岭。本笔记目标：**① 5 秒二分（看 UA：管型 + 蛋白量）② 各自病因谱 ③ 肾病综合征并发症（高频考点）**。来源：[[mistakes/uworld-mistakes_2026-06#^Q2244]]（肾病综合征并发症 → 加速动脉粥样硬化；总钙↓但 PTH 正常陷阱）。

---

## Part 1. 二分总表 ⭐⭐⭐（先记这条）

| 维度 | **Nephritic（肾炎）** | **Nephrotic（肾病）** |
|---|---|---|
| 核心 | **炎症 → GBM 破损出血** | **足细胞损伤 → 大量漏蛋白** |
| 蛋白尿 | 轻-中（<3.5 g/d）| **重（>3.5 g/d）** ⭐ |
| 血尿 | **肉眼/镜下 + 变形 RBC + RBC 管型** ⭐ | 无/微量 |
| 水肿 | 轻（晨起眶周）+ **HTN** | **重（全身性 + 腹水 + 眶周）** |
| 低白蛋白 | 轻 | **重 → 高脂血症** |
| 尿沉渣 | **RBC 管型 + 红细胞**（"活动性"）| **脂肪管型 / oval fat body / "Maltese cross"** |
| 关键词 | HTN + 血尿 + 少尿 + RBC casts | 大量蛋白尿 + 低白蛋白 + 水肿 + 高脂 |

> [!tip] 一眼二分
> - **RBC 管型 + HTN + 血尿 = Nephritic**
> - **大量蛋白尿 + 低白蛋白 + 全身水肿 + 高脂 = Nephrotic**
> - 部分病（MPGN / 狼疮 / 糖尿病晚期）可**重叠**两征。

---

## Part 2. Nephritic 病因谱

| 病因 | 指纹 | 补体 |
|---|---|---|
| **PSGN**（链球菌感染后）| 感染后 **2–3 周**、儿童、可乐色尿、低 C3 | **C3↓** |
| **IgA nephropathy**（Berger）| URI **同期/1–2 天**（synpharyngitic）、亚裔 | 正常 |
| **RPGN（新月体）** | 急进性 AKI；anti-GBM(Goodpasture 肺-肾) / **ANCA**(GPA/MPA/EGPA) / 免疫复合物 | 看类型 |
| **Alport** | 遗传（X 连锁 IV 型胶原）+ **感音神经性耳聋 + 眼**异常 | 正常 |
| **Lupus nephritis / MPGN** | 系统性 SLE / 低补体；可重叠 nephrotic | **C3↓** |

> [!tip] 低 C3 nephritic 缩圈：**PSGN / 狼疮 / MPGN / 心内膜炎相关**。

---

## Part 3. Nephrotic 病因谱

| 病因 | 典型人群 / 钩子 |
|---|---|
| **Minimal change disease (MCD)** | **儿童 #1**；URI/疫苗后、Hodgkin 相关；**激素敏感** |
| **FSGS** | **成人 #1**（尤其非裔）；**HIV / 肥胖 / 海洛因** |
| **Membranous nephropathy** | 成人；**anti-PLA2R**；实体肿瘤 / **乙肝** / SLE / 药物 |
| **Diabetic nephropathy** | **总体最常见**；Kimmelstiel-Wilson 结节、病程长 |
| **Amyloidosis** | MM / 慢性炎症；Congo red 苹果绿双折射 |

---

## Part 3.5. 慢性炎症 → 肾综鉴别谱（≠ 都是 amyloid）⭐⭐

> [!warning] 别外推："慢性炎症 + 肾综"不一定是淀粉样变
> AA 淀粉样变是**最经典**的那一条，但慢性炎症/感染致肾综有个谱——**关键看哪种炎症 + 补体 + 活检**。

| 慢性炎症病因 | 倾向的肾病 | 不是 amyloid 时是什么 |
|---|---|---|
| **长期 RA / IBD / 慢性骨髓炎 / TB / 支扩 / FMF** | **AA 淀粉样变** ⭐ | （SAA 急性期蛋白↑ → AA 沉积）|
| **SLE** | 狼疮性肾炎（V 型膜性 / 增殖性）| 免疫复合物 + **低补体 + RBC 管型** |
| **HCV** | MPGN / 冷球蛋白血症性 | **低补体 + RF + 冷球蛋白**（见 [[mistakes/uworld-mistakes_2026-06#^Q2232]]）|
| **HBV** | 膜性肾病 / PAN | 免疫复合物（anti-PLA2R 可阴）|
| **慢性感染（心内膜炎 / 脓肿）** | MPGN | 免疫复合物 + 低补体 |

> [!key] 怎么区分（补体 + 活检是分水岭）
> - **AA 淀粉样变**：病程**很长**（多年 RA/IBD/化脓/FMF）+ 肾综**无 cells/casts** + **Congo red 阳性、苹果绿双折射**（活检定锤）+ **补体正常**。
> - **SLE / HCV / HBV / 心内膜炎**：走**免疫复合物**路 → **低补体 + 活动性尿沉渣（RBC 管型）+ 特异血清/抗体**——这些 amyloid **都没有**。

> [!tip] 反射
> - "**长期 RA/IBD/化脓 + 肾综 + 无管型 + Congo red**" → **AA 淀粉样变**
> - "**SLE/HCV + 肾综 + 低补体 / RBC 管型**" → **免疫复合物 GN（≠ amyloid）**
> 即 Q16296"长期 RA + 肾综 → AA"是对的反射，但别外推成"凡慢性炎症肾综都是 amyloid"。

## Part 4. 肾病综合征并发症谱 ⭐⭐⭐（Q2244 核心）

> [!danger] 按"尿丢什么蛋白"推并发症
> 肾小球漏蛋白 → 不只漏白蛋白，**抗凝蛋白 / 免疫球蛋白 / 维 D 结合蛋白 / 转铁蛋白都漏**。

| 丢失的蛋白 | 后果 |
|---|---|
| 白蛋白 | 水肿、低渗 |
| **脂代谢紊乱**（肝合成脂蛋白↑ + **PCSK9↑ → LDL 清除↓**）| **高脂血症 → 加速动脉粥样硬化** ⭐ |
| **抗凝血酶 III + 蛋白 C/S** | **高凝 → DVT / PE / 肾静脉血栓** ⭐ |
| **免疫球蛋白** | **荚膜菌感染**（肺炎链球菌等）|
| 维生素 D 结合蛋白 | 维 D 缺乏 → **总钙↓但离子钙正常 → PTH 正常** |
| 转铁蛋白 / EPO | 贫血 |

> [!warning] 经典陷阱：总钙↓ ≠ 甲状旁腺功能减退 ⭐（Q2244 我错点）
> 肾病丢**维 D 结合蛋白** + 低白蛋白结合钙 → **总钙↓**，但**活性的离子钙正常 → PTH 正常**，不是 hypoparathyroidism。看到"肾病 + 低钙"别反射选甲旁减。

> [!info] 延伸：那"真·甲状旁腺功能减退"是什么病因？
> 
> | 类别 | 代表 |
> |---|---|
> | **术后**（最常见）⭐ | 甲状腺 / 甲状旁腺 / 颈部手术误切或损伤血供 |
> | **自身免疫** | 孤立性 / **APS-1（APECED：甲旁减 + 肾上腺功能不全 + 皮肤黏膜念珠菌）** |
> | **先天** | **DiGeorge（22q11）**——胸腺 + 甲旁腺发育不全 |
> | **浸润性**（罕见）| Wilson（铜）/ 血色病（铁）/ 转移癌 / **淀粉样** |
> | **功能性** ⭐ | **低镁血症**（Mg↓ 损 PTH 分泌 + 外周抵抗）|
> | 其他 | 放疗 / 放射性碘 |
> 
> **淀粉样问题澄清**：淀粉样**可**沉积于甲旁腺、也被列为甲旁减的**罕见浸润病因**，但临床极少表现甲旁减、非高产。**淀粉样内分泌浸润真正要记的是肾上腺 → 原发性肾上腺功能不全（Addison）**；其次甲状腺（含 MTC 间质淀粉样）。
> 
> ⚠️ 注意：本题（Q2244）的甲旁减纯属"总钙↓陷阱"（离子钙/PTH 正常），**与淀粉样无关**——别把"淀粉样是肾病病因"错连成"淀粉样致甲旁减"。

> [!info] 肾静脉血栓 (RVT) 高频
> 肾病高凝 → **RVT** 最经典（尤其膜性肾病）；表现：腰痛 + 血尿 + 肾功能恶化 + LDH↑。

---

## Part 5. 易混陷阱

- **总钙↓ → 选甲旁减**：❌ 离子钙正常 PTH 正常（维 D 结合蛋白丢失 + 白蛋白低）。
- **肾病 + 突发水肿 → 肝硬化**：看 **PT**——肝硬化 PT 延长且数年缓起；肾病 PT 正常、数周急起。
- **MCD 成人少见**：成人 nephrotic 先想 **FSGS / 膜性 / 糖尿病**。
- **Nephritic 重叠 nephrotic**：MPGN / 狼疮 / 糖尿病晚期可两征并存，别非此即彼。

---

## Part 6. Memory Hook

> - **二分**：RBC 管型 + HTN = **肾炎**；大量蛋白尿 + 低白蛋白 + 水肿 + 高脂 = **肾病**。
> - **肾病四大丢失**：**脂↑(动脉粥样硬化) + 抗凝丢(血栓 RVT) + Ig 丢(荚膜菌) + 维 D 丢(总钙↓离子钙正常 PTH 正常)**。
> - **总钙低别选甲旁减**（离子钙正常）。
> - **低 C3 肾炎**：PSGN / 狼疮 / MPGN / 心内膜炎。

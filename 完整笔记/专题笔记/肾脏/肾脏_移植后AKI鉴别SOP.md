# 移植后 AKI 鉴别 SOP (Post-Transplant Acute Kidney Injury) 完整框架

## 一、核心原则

**移植肾 = 单肾**（除非保留 native kidneys），任何 AKI 都比双肾患者**更脆弱**，且需要按**移植专属**的 5 项鉴别 SOP 逐一排查。

## 二、移植肾 AKI 5 项鉴别 SOP ⭐⭐⭐（CK 核心）

```
Post-Transplant Cr ↑
   │
   ├─ ① Acute Rejection（急排）
   │     - 病程：post-tx 6 mo 内最多见
   │     - 临床：graft 区压痛 + 发热 + 突发 Cr ↑
   │     - 诊断：biopsy（金标准）
   │     - 治疗：methylprednisolone pulse → ATG → IVIG / rituximab（抗体介导）
   │
   ├─ ② CNI Toxicity（calcineurin inhibitor 毒性）⭐ Q16485
   │     - 病程：任何时期；常 + 加 CYP3A4 抑制剂后急升
   │     - 临床：HTN + bland UA + 血药 ↑
   │     - 诊断：tacrolimus / cyclosporine 血药浓度
   │     - 治疗：降剂量 + 调 CYP3A4 互动药
   │
   ├─ ③ Transplant Renal Artery Stenosis（RAS）⭐ Q16323
   │     - 病程：post-tx 1-2 yr 内最多见
   │     - 临床：resistant HTN + ACEI 后 Cr 急升 + ± graft bruit
   │     - 诊断：Doppler US（首选）→ CTA / MRA → angiography
   │     - 治疗：PTA ± stent；停 ACEI 改其他降压
   │
   ├─ ④ BK Virus / CMV Reactivation
   │     - 病程：post-tx 3-12 mo 内最多见（免疫抑制最强时）
   │     - 临床：渐进 Cr ↑ + 无 graft 压痛
   │     - 诊断：血 / 尿 BK PCR + decoy cells；CMV PCR + 组织学
   │     - 治疗：减免疫抑制 + Cidofovir (BK 重症) / Ganciclovir (CMV)
   │
   └─ ⑤ ATN / Mechanical Obstruction
         - 病程：ATN 多 post-tx **第 1 周**（围手术期缺血）；梗阻不一
         - 临床：ATN 急升 + muddy brown casts；梗阻 ↓ urine output ± hydronephrosis
         - 诊断：UA (muddy brown for ATN)；transplant US (hydronephrosis for 梗阻)
         - 治疗：ATN 支持等待恢复；梗阻 → 介入解除（输尿管支架 / PCN）
```

## 三、5 项鉴别"分秒锁"特征对照表 ⭐

| 鉴别 | 时间窗 | 关键临床 | UA / 影像 | 诊断金标 | 治疗 | 可逆性 |
|---|---|---|---|---|---|---|
| **① 急排** | post-tx **6 mo 内** | Graft 压痛 + 发热 + 突发 Cr↑ | UA 多 bland；US 可见 graft 肿胀 | **Biopsy** | 类固醇 pulse → ATG | 大多可逆 |
| **② CNI 毒** ⭐ | **任何**（常 + 新药）| HTN + bland UA + **血药↑** + **CYP3A4 抑制剂史** | UA bland；US 正常 | **血药浓度** | **降剂量** + 调互动药 | **可逆**（急性）/ 不可逆（慢性纤维化） |
| **③ Transplant RAS** ⭐ | post-tx **1-2 yr 内** | **Resistant HTN + ACEI 后 Cr 急升** + ± graft bruit | UA bland；US Doppler 见动脉狭窄 | **Doppler US** → CTA / MRA → angiography | **PTA ± stent** + 停 ACEI | 治疗后可逆 |
| **④ BK / CMV** | post-tx **3-12 mo** | 渐进 Cr↑ + 无 graft 压痛 | 尿 decoy cells；血尿 BK/CMV PCR | **PCR + 病毒载量 + 组织学** | 减免疫抑制 + Cidofovir/Ganciclovir | 早治可逆 |
| **⑤ ATN / 梗阻** | ATN **第 1 周** / 梗阻不一 | ATN 急升 + muddy brown casts；梗阻 ↓ urine + hydronephrosis | UA muddy brown (ATN) / US hydronephrosis (梗阻) | UA + US | ATN 支持；梗阻介入 | 多可逆 |

## 四、CNI (Calcineurin Inhibitor) 毒性深度 ⭐⭐（Q16485 触发）

### 4.1 CNI 双阶段毒性

| 阶段 | 机制 | 表现 | 可逆性 |
|---|---|---|---|
| **急性** ⭐ | 入球 + 出球小动脉 **vasoconstriction** → ↓RBF / ↓GFR | AKI + HTN + UA bland（prerenal pattern）| **降剂量可逆** |
| **慢性** | 血管缺血 + 直接小管毒 + 间质纤维化 + striped fibrosis | 进行性 ↓GFR + arteriolopathy on biopsy | **不可逆** |

### 4.2 CYP3A4 与 CNI 血药互动雷区 ⭐⭐⭐

| **CYP3A4 抑制剂**（↑ CNI 血药 → 中毒）| **CYP3A4 诱导剂**（↓ CNI 血药 → 排斥）|
|---|---|
| **Diltiazem / Verapamil**（非二氢吡啶 CCB）| Rifampin |
| Azole 抗真菌（fluconazole / ketoconazole / voriconazole）| Phenytoin / Carbamazepine / Phenobarbital |
| Macrolides（**erythromycin / clarithromycin**；**azithromycin 安全**）| St. John's wort |
| Protease inhibitors（HIV ritonavir 等）| |
| **葡萄柚汁** | |

> [!tip] CYP3A4 抑制剂记忆口诀 = **"D-A-M-P-G"**
> **D**iltiazem / **A**zole / **M**acrolide (-azithro) / **P**rotease inhibitor / **G**rapefruit

### 4.3 移植 AF 患者 CCB 选药原则

```
Transplant patient with AF needs rate control
   │
   ├─ ❌ 避 Diltiazem / Verapamil（非二氢吡啶 CCB → CYP3A4 抑制 → tacro 血药 ↑）
   ├─ ✅ 选 Amlodipine / Nifedipine（二氢吡啶 CCB → 不抑 CYP3A4）
   └─ ✅ β-blocker 也是好选择（metoprolol 等不经 CYP3A4）
```

## 五、Transplant RAS 深度 ⭐⭐（Q16323 触发）

### 5.1 Transplant RAS 三联指纹

| 要素 | 表现 |
|---|---|
| **时间窗** | post-tx **前 2 年内最多见**（吻合口狭窄 / 手术创伤 / 供肾动脉粥样硬化 / CMV / BK virus）|
| **Resistant HTN** | 多药控不住；可伴 flash pulmonary edema |
| **ACEI / ARB 后 Cr 急升** ⭐ | **单肾 = 等效双侧 RAS** → 加 ACEI 后 GFR 急剧↓ |
| 体征 | Graft 区 / 上腹 **bruit**；可有 lateralizing |
| 诊断 | **Doppler US**（首选无创）→ CTA / MRA → angiography（金标准）|
| 治疗 | **PTA ± stent**（angioplasty）+ 术后停 ACEI 改其他降压 |

### 5.2 ACEI 后 Cr 急性升 → 3 个鉴别（任何患者通用）

| 鉴别 | Cr 上升幅度 | 思路 |
|---|---|---|
| **双侧 RAS / 移植肾 RAS / 孤立肾 RAS** | **>30%** | ACEI 解除出球小动脉收缩 → GFR 崩塌 → **停 ACEI + 影像** |
| 单侧 RAS | <30% 或正常 | 对侧肾代偿 |
| 容量收缩（脱水 / 利尿过度）| 缓慢 | 补液后好转 |

## 六、CNI 毒 vs Transplant RAS 鉴别（最易混）⭐

| | **CNI 毒** | **Transplant RAS** |
|---|---|---|
| 时间窗 | 任何时期 | **post-tx 1-2 yr** |
| 触发 | **新加 CYP3A4 抑制剂** | **新加 ACEI / ARB** |
| 血药 | **tacro / cyclosporine ↑** | 范围内 |
| HTN | 有 | **Resistant**（多药控不住）|
| UA | bland | bland |
| US | 多正常 | **Doppler 见动脉狭窄** |
| 治疗 | **降 CNI 剂量** | **PTA ± stent + 停 ACEI** |

> [!warning] 反 pattern
> 看 Q16323 stem："tacrolimus level within therapeutic range" = **强排除 CNI 毒**信号；接下来扫"ACEI 后 Cr↑ + resistant HTN + 1-2 yr post-tx" 就锁定 Transplant RAS。

## 七、做题 SOP（CK 实战）

### 第一步：扫 stem 时间窗
- **第 1 周** → ATN（围手术缺血）
- **6 mo 内** → 急排（biopsy）
- **3-12 mo** → BK / CMV（PCR）
- **1-2 yr** → Transplant RAS（Doppler）
- **任何时期** → CNI 毒（血药）

### 第二步：扫 stem 新加药 / 触发事件
- **新加 ACEI / ARB → Cr ↑** → RAS
- **新加 CYP3A4 抑制剂（diltiazem / azole / macrolide / 葡萄柚）** → CNI 毒
- **新感染 / 疫苗 / 减免疫抑制** → 急排 OR 病毒激活
- **造影剂 / aminoglycoside / NSAID** → 药毒 ATN

### 第三步：扫 UA + 影像
- **UA bland + HTN + 血药 ↑** → CNI 毒
- **UA bland + ACEI 后 Cr↑ + resistant HTN** → Transplant RAS
- **muddy brown casts** → ATN
- **WBC casts + 嗜酸 + 皮疹** → AIN（少见，但药多 → 可能）
- **Hydronephrosis** → 梗阻

## 八、复习 Q&A

> [!question]- Q1. 移植肾 AKI 5 项鉴别？
> **答**：① 急排（biopsy）/ ② CNI 毒（血药）/ ③ Transplant RAS（Doppler）/ ④ BK / CMV（PCR）/ ⑤ ATN / 梗阻（UA + US）

> [!question]- Q2. CNI 急性 vs 慢性毒性的病机部位区别？
> **答**：**急** = 入/出球小动脉 vasoconstriction（功能性 prerenal-like，降剂量可逆）；**慢** = 血管缺血 + 直接小管毒 + 间质纤维化 + striped fibrosis（不可逆）。

> [!question]- Q3. CYP3A4 抑制剂口诀？为什么 transplant + AF 选 amlodipine 而非 diltiazem？
> **答**：**"D-A-M-P-G"** = Diltiazem / Azole / Macrolide (-azithro) / Protease inhibitor / Grapefruit。**Diltiazem 抑 CYP3A4 → tacro 血药 ↑ → 急毒**；**amlodipine（二氢吡啶 CCB）不抑 CYP3A4**，安全。

> [!question]- Q4. Transplant RAS 三联指纹？为什么 stem 强调 "tacrolimus within therapeutic range"？
> **答**：① post-tx **1-2 yr 内** + ② **resistant HTN** + ③ **ACEI 后 Cr 急升**。"Tacro 范围内" 是 stem 的**强排除 CNI 毒信号** — 排掉 CNI 毒后就锁定 RAS。

> [!question]- Q5. ACEI 后 Cr 急性升 >30% 提示什么？
> **答**：**双侧 RAS / 孤立肾 RAS / 移植肾 RAS**（移植肾 = 单肾 = 等效双侧）。停 ACEI + 影像（Doppler）。

> [!question]- Q6. 见 transplant + Cr 急升 + 新加 azithromycin，是不是 CYP3A4 互动？
> **答**：**不是**。**Azithromycin 不抑 CYP3A4**（其他 macrolide 如 erythromycin / clarithromycin 抑制），所以 transplant 患者抗感染优选 azithromycin。

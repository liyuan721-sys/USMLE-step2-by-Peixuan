# 间质性肾炎家族 (Interstitial Nephritis Family) — AIN vs CTIN 对照

## 一、核心定义

**Interstitial nephritis** = 炎症 / 损伤定位在**肾间质 + 小管**（区别于 glomerular = 肾小球，vascular = 血管）。临床两大类：

| 分类 | 全称 | 病程 | 典型病因 |
|---|---|---|---|
| **AIN** | Acute Interstitial Nephritis | **天 - 周**（快）| **药物超敏**（~75%）+ 感染 + 自免 |
| **CTIN** | Chronic Tubulointerstitial Nephritis | **月 - 年**（慢）| **慢性肾毒素累积**（analgesic / lithium / 重金属 / 慢性梗阻 / 慢性 reflux / 痛风）|

## 二、AIN vs CTIN 5 维对照 ⭐⭐⭐（USMLE 核心）

| 维度 | **AIN（急）** | **CTIN（慢）** |
|---|---|---|
| **病因** | **药物**（~75%）/ 感染（Legionella, CMV, TB, EBV）/ 自免（SLE / Sjögren / Sarcoidosis） | **Analgesic nephropathy** ⭐ / 慢性 lithium / 慢性 obstruction / 重金属 (Pb / Cd) / 慢性 reflux / 痛风 (chronic urate nephropathy) |
| **病程** | 暴露后**天 - 周**（β-lactam 快）或 **wks-mo**（NSAID / PPI / allopurinol 慢） | **月 - 年**累积 |
| **临床** | AKI + malaise + polyarthralgia + 低热 + **maculopapular rash**（**classic triad 发热+皮疹+嗜酸只 10% 全套**） | 慢性 CKD + 贫血 + flank pain（papillary necrosis 时） + ± hematuria |
| **UA** | **WBC + WBC casts + 轻蛋白 + ± RBC + eosinophiluria (Hansel 染色)** | **WBC + WBC casts + 轻蛋白 (<1-1.5 g/d) + 有 papillary necrosis 时 RBC** |
| **实验室** | **外周嗜酸 >5%** + 轻 anemia | 慢性 CKD pattern（↑ BUN/Cr + 贫血 + 钙磷代谢紊乱）|
| **影像** | **正常 / 轻度增大** | **双侧对称小肾 + papillary calcification + 不规则轮廓** |
| **诊断** | 临床诊断；持续 / 严重 → biopsy 见小管间质炎 + edema + 嗜酸 + 肉芽肿 | 影像 + 临床史 + UA + 排他；biopsy 见纤维化 + 萎缩 |
| **治疗** | **停 offending drug + 支持** → 持续 → **glucocorticoids** → 严重 → hemodialysis | **停 analgesics**（不可逆，停可减缓进展）+ 治继发肾衰 |
| **预后** | **可逆**（早停药 + 类固醇）| **不可逆**，但停诱因可阻止进展 |

## 三、AIN 高频致病药 6 类 ⭐⭐⭐（CK 必背）

| 类别 | 药物 | 潜伏期 |
|---|---|---|
| **抗生素** | β-lactam（penicillin / cephalosporin — 经典）/ FQ / 磺胺 / Rifampin / Vancomycin | **天 - 周**（β-lactam 最快）|
| **NSAIDs** | ibuprofen / naproxen / ketorolac 等 | **wks - mo**（慢，常被忽略）|
| **PPI** | omeprazole / esomeprazole / pantoprazole 等 | **wks - mo**（长期使用）|
| **Allopurinol** | 痛风降尿酸 | **wks - mo** ⭐ 本笔记触发题 Q4007 |
| **利尿剂** | thiazide / furosemide | wks |
| **其他** | 5-ASA / cimetidine / phenytoin | 不一 |

> [!warning] AIN 反 pattern：classic triad（fever + rash + 嗜酸）只 ~10% 全套
> 不能等 triad 齐全才考虑 AIN。**外周嗜酸 >5% + WBC casts + 新药史** 已足够锁定（敏感性远高于 triad）。

## 四、CTIN 详解 — Analgesic Nephropathy 病机链 ⭐

```
长期联合服用 acetaminophen (APAP) + aspirin / NSAID
                │
                ├─ APAP → 反应性代谢物（NAPQI 样）→ 直接小管毒
                │      └─ Aspirin / NSAID 耗 glutathione → APAP 毒性放大
                └─ NSAID → 直接氧化损伤 + ↓ PG → ↓ medullary blood flow
                                  │
                                  ▼
            Chronic tubulointerstitial nephritis ± papillary necrosis
                                  │
                                  ▼
            双侧对称小肾 + papillary calcification + 慢性肾衰 + 贫血
```

> [!tip] OTC + 慢性疼痛的高危人群
> Analgesic nephropathy 多见于**慢性疼痛**（背痛 / 头痛 / 关节痛 + 自己买 OTC 长期吃）人群。Stem 钩子：**"chronic pain × 数十年 + OTC analgesics + 双侧小肾"**。

## 五、Renal Papillary Necrosis 5 大病因 — 口诀 "DNA-Sickle" ⭐

| 字母 | 病因 | 备注 |
|---|---|---|
| **D** | **D**iabetes mellitus | 微血管病变 → 髓质灌注↓ |
| **N** | **N**SAIDs / **A**nalgesic abuse | ↓ PG → ↓ 髓质血流（本笔记重点）|
| **A** | **A**nalgesic（与 N 合并）| — |
| **S** | **S**ickle cell disease/trait | 髓质低氧 + 高渗 → HbS 聚合 → 髓质缺血 |
| **i** | **i**nfection（severe pyelonephritis）/ TB | — |
| **C** | **C**hronic obstruction | — |

> [!info] Papillary necrosis 经典 stem 钩子
> "Flank pain + gross hematuria + 自认 muscle pain 一过性发作" — 可能是**sloughed papilla**（脱落乳头堵尿路）。**Flank pain ≠ 一定是 stone**（USMLE 反 pattern，见 Q4004）。

## 六、Urinary Casts 指纹族系 ⭐⭐⭐（任何 AKI 题先扫）

| Casts 类型 | 锁定家族 | 典型病 |
|---|---|---|
| **RBC casts**（红细胞管型）| **Glomerular** | GN（IgA / postinfectious / RPGN / Lupus nephritis）|
| **WBC casts**（白细胞管型）| **间质性家族** ⭐ | **AIN / CTIN / pyelonephritis** |
| **Muddy brown casts**（颗粒褐色管型）| **ATN** | 缺血 / 药毒 / 横纹肌溶解 |
| **Waxy casts**（蜡样管型）| 慢性肾衰 | 终末期 CKD |
| **Fatty casts + oval fat bodies**（Maltese cross）| 肾病综合征 | MCD / FSGS / MN / DM 肾病 |
| **Granular casts**（颗粒管型）| 非特异 | 多种 |
| **Hyaline casts**（透明管型）| 非特异（脱水 / 运动后正常）| — |

> [!success] AKI 题做题 SOP
> 0.5 秒先扫 UA casts：**RBC → GN；WBC → 间质性；muddy brown → ATN；fatty → 肾病；waxy → 慢肾衰；hyaline → 非特异**。然后才回头看病因 / 治疗选项。

## 七、AIN vs DIL（Drug-Induced Lupus）鉴别 — Q4007 我错的关键

| | **AIN** | **Drug-Induced Lupus (DIL)** |
|---|---|---|
| **致病药** | Allopurinol / β-lactam / NSAID / PPI / 磺胺 / 利尿剂 | **Procainamide / Hydralazine / Isoniazid / Minocycline / Methyldopa / Quinidine** |
| **病程** | 新药后**天 - mo** | 长期用药后**月 - 年** |
| **临床** | AKI + 嗜酸 + maculopapular rash + 关节痛 + 低热 | Malaise + arthralgia + 浆膜炎 + rash；**AKI 罕见** |
| **UA** | **WBC casts + 嗜酸 + 轻蛋白** | **RBC casts + 中重度蛋白尿**（如有肾受累 = GN） |
| **抗体** | 无特异 | **抗组蛋白抗体 +**（>95% 阳性，区别于 SLE）|
| **治疗** | 停药 + 类固醇 | 停药 + NSAIDs / 类固醇 |

> [!warning] 鉴别钩子
> **WBC casts 排 DIL**（DIL 致 AKI 罕见且经 GN）；**RBC casts 排 AIN**。两者**临床表现高度重叠**（malaise / arthralgia / rash），但 UA + AKI 频率分得清。

## 八、CK 解题 SOP

```
AKI + 慢性 OR 急性？
  │
  ├─ 急（天-周）+ 新药 + 嗜酸 + 皮疹 + WBC casts → AIN
  │     → 停药 + 类固醇（持续不缓解时）
  │
  └─ 慢（月-年）+ 长期镇痛 + 双侧小肾 + WBC casts → CTIN (Analgesic nephropathy)
        → 停 analgesics + 治继发 CKD
```

### 快速排除三步
1. **UA casts 类型**（WBC = 间质性家族）
2. **病程急慢**（决定 AIN vs CTIN）
3. **影像肾大小**（正常 = AIN；双侧小肾 = CTIN）

## 九、复习 Q&A

> [!question]- Q1. 见 AKI + UA WBC casts + 外周嗜酸 7% + 新药史 2 mo，最可能诊断？为什么不是 DIL？
> **答**：AIN（多由 allopurinol / β-lactam / NSAID / PPI / 磺胺 致）。**WBC casts 排 DIL** — DIL 致 AKI 罕见且经 GN（RBC casts + 重蛋白）；DIL 多 procainamide / hydralazine / INH / minocycline 致，抗组蛋白抗体阳性。

> [!question]- Q2. AIN classic triad 是什么？敏感性？
> **答**：**Fever + maculopapular rash + 外周嗜酸**。但**只 ~10% 全套**，不能等 triad 齐才诊断。**外周嗜酸 >5% + WBC casts + 新药史**已足够。

> [!question]- Q3. Analgesic nephropathy 病机两大支柱？
> **答**：① **APAP 代谢物**（NAPQI 样直接小管毒，aspirin/NSAID 耗 glutathione 放大毒性）；② **NSAID ↓PG → ↓髓质血流**（直接氧化 + 缺血）。

> [!question]- Q4. Papillary necrosis 5 大病因口诀？
> **答**：**"DNA-Sickle"** = **D**M + **N**SAID / **A**nalgesic + **S**ickle cell + **i**nfection / TB + **C**hronic obstruction。

> [!question]- Q5. 见 severe flank pain + 慢性背痛 + OTC analgesic 史 + UA WBC casts + 双侧小肾，flank pain 最可能解释？
> **答**：**Sloughed papilla**（papillary necrosis 脱落乳头堵尿路），不是 nephrolithiasis（影像应见 stone / acoustic shadowing）。Analgesic nephropathy 经典钩子。

> [!question]- Q6. Urinary casts 5 大族系？
> **答**：**RBC**→GN / **WBC**→间质性（AIN/CTIN/pyelonephritis）/ **muddy brown**→ATN / **fatty**→肾病综合征 / **waxy**→慢肾衰。

# 💧 低钠血症诊疗 SOP（Hyponatremia Master）

> [!info] 笔记定位
> 把错题本里的低钠/SIADH 簇（Q2180 / Q2181 / Q117977 / Q114931 / Q106381 / Q114863）整合成一条**可执行决策链**：先定性（真假低钠）→ 定容量 → 定病因（尿渗/尿 Na）→ 按严重度治疗。

## 一、三步诊断框架（USMLE 最爱）⭐

```
低钠 Na <135
  │
① 血渗 (serum osm) —— 定"真假低钠"
  │   ├─ >295 高渗  → 转移性低钠（高血糖 / 甘露醇）—— 假性下降，水被拉出细胞
  │   ├─ 正常 (275–295) → 假性低钠 (pseudohyponatremia)（高脂血症 / 副蛋白血症）
  │   └─ <275 低渗  → ✓ 真性（低渗性）低钠 —— 继续 ②
  │
② 容量状态 (ECV) —— 定大方向
  │   ├─ 低容量 (hypovolemic)
  │   ├─ 等容量 (euvolemic)
  │   └─ 高容量 (hypervolemic)
  │
③ 尿 Na / 尿渗 —— 定具体病因（见下表）
```

> [!tip] 一句话主线
> **血渗定真假 → 容量定大方向 → 尿渗/尿 Na 定病因 → 严重度定治疗。**

## 二、低渗性低钠分类大表 ⭐⭐⭐

| 血渗 | 容量 | 尿指标 | 病因 |
|---|---|---|---|
| **低渗 <275** | 低容量 | **尿 Na <20** | 肾外丢盐：呕吐、腹泻、第三间隙 |
| 低渗 <275 | 低容量 | **尿 Na >20** | 肾丢盐：利尿剂、原发性肾上腺功能不全、脑性盐耗 (CSW) |
| 低渗 <275 | **等容量** | **尿渗 <100** | **原发性多饮 (primary polydipsia)** / malnutrition (beer potomania) |
| 低渗 <275 | **等容量** | **尿渗 >100 且尿 Na >40** | **SIADH**（先排甲减 + 继发性肾上腺功能不全）|
| 低渗 <275 | 高容量 | 尿 Na <20（多数）| 心衰、肝硬化、肾病综合征（EABV↓ → ADH↑）|
| 低渗 <275 | 高容量 | 尿 Na >20 | 急/慢性肾衰 |
| 等渗 | — | — | 假性低钠（副蛋白 / 高脂）|
| 高渗 >295 | — | — | 转移性（高血糖：Glu 每↑100，Na 实测↓~1.6–2.4）/ 甘露醇 |

> [!warning] 等容量低钠 = SIADH vs 原发多饮，分水岭就是**尿渗 100**
> - **尿渗 <100** = 稀释尿（ADH 被适当抑制）→ **原发多饮**（拼命喝水）
> - **尿渗 >100** = 不当浓缩（ADH 过多）→ **SIADH**

## 三、SIADH 专节

> [!info] 错题来源
> 诊断指纹见 [[mistakes/uworld-mistakes#^Q2180]]；病因 workup 见 [[mistakes/uworld-mistakes#^Q114931]]；治疗见 [[mistakes/uworld-mistakes#^Q2181]]；与渗透性利尿对照见 [[mistakes/uworld-mistakes#^Q117977]]。

### 3.1 SIADH 五联（诊断指纹）

1. 低渗性低钠（血渗 <275）
2. **等容量**（黏膜湿、无 JVD、无水肿、无体位性）
3. **尿渗 >100**（不当浓缩）
4. **尿 Na >40**
5. **血尿酸低（<4 mg/dL）** + K 正常 + 酸碱正常

### 3.2 为什么 SIADH 血尿酸低（机制）

ADH 致水潴留 → **轻度容量扩张**（虽临床 euvolemic）→ 近端小管 Na⁺ 重吸收↓ → 与 Na⁺ 偶联的**尿酸重吸收↓ + 排泄↑（FEurate↑）** → 血尿酸↓。
**尿酸方向 = 容量轴的镜子**：SIADH（容量足）尿酸**低**；低容量低钠（容量缺，近端拼命吸钠/尿酸）尿酸**高**。补钠纠正后尿酸回升。

### 3.3 SIADH 病因

| 类别 | 例子 |
|---|---|
| CNS | 卒中、出血、外伤、感染 |
| 肺 | 肺炎、**小细胞肺癌 (SCLC) 异位 ADH** |
| 药物 | **SSRI**、carbamazepine、cyclophosphamide、NSAID |
| 其他 | 疼痛、**恶心**（强 ADH 刺激）、术后 |

## 四、治疗 SOP（按严重度）⭐⭐⭐

| 严重度 | 表现 | Na | 治疗 |
|---|---|---|---|
| 无/轻症 | 嗜睡、健忘 | 120–130 | **限水 <800 mL/d** ± 盐片；治原发 |
| **重症** | seizure / coma / profound confusion | **<120 或有重度神经症状** | **3% 高张盐水**紧急升钠 |

> [!danger] 纠正速率铁律
> **≤8 mEq/L / 24 h**（慢性低钠更严）。过快 → **渗透性脱髓鞘综合征 (ODS / central pontine myelinolysis)**：dysarthria、quadriparesis、可致命。

### 4.1 SIADH 治疗的最大陷阱：别给 NS / half-NS ⭐

SIADH 里**输液必须比尿液更浓**才能升钠：

| 液体 | 张度 (mOsm/kg) | 在 SIADH（尿渗常 ~500）|
|---|---|---|
| 3% 高张盐水 | ~1026 | ✅ 有效升钠 |
| 0.9% NS | ~300 | ❌ < 尿渗 → 自由水净潴留 → **钠更低** |
| 0.45% half-NS | ~150 | ❌ 更糟 |

> 口诀：**尿渗 > 盐水渗 → 给盐水反而稀释**。SIADH 重症只认 3% 高张盐水。

## 五、按容量轴的其他低钠（鉴别要点）

- **低容量低钠**：尿 Na <20（肾外）/ >20（肾内）；黏膜干、体位性↓、**尿酸高**。治疗 = 等张盐水（NS）补容量（与 SIADH 相反！）。
- **高容量低钠**（心衰 / 肝硬化 / 肾综）：EABV↓ → RAAS+ADH↑。**限水 + 治原发**；利尿用袢，**避免噻嗪**。见 [[mistakes/uworld-mistakes#^Q106381]]。
- **噻嗪致低钠** ⭐：经典药物性低钠（强降压 + ADH↑ + 远端稀释能力↓）；可重到 seizure。见 [[mistakes/uworld-mistakes#^Q114863]]。
- **原发多饮**：尿渗 <100；GAD / 精神病 / 怕结石猛喝水。治疗 = 限水。

## 六、🧬 指纹速查（嵌入版）

> [!note] 为什么放这里
> 当前 [[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全]] 文件已损坏（单行无换行、无小节锚点），暂无法安全写入；低钠相关指纹先收在本 SOP，待大全修复后再迁移。

- **等容量低钠 + 尿渗 >100 + 尿 Na >40 + 尿酸低** → SIADH
- **等容量低钠 + 尿渗 <100** → 原发多饮
- **低钠 + 尿酸高 + 黏膜干 + 体位性↓** → 低容量低钠
- **多尿 + 尿渗 >600 + 高血糖** → 溶质性（渗透性）利尿（≠SIADH；SIADH 无多尿）
- **低钠 + seizure/coma 或 Na <120** → 重症 → 3% 高张盐水
- **SIADH 用 NS 后钠更低** → 因尿渗 > NS 渗

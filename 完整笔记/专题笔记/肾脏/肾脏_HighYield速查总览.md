## 一、急性肾损伤（acute kidney injury, AKI）三分法

> [!tip] 第一决策：肾前性 (prerenal) vs 肾性 ATN vs 肾后性 (postrenal)
> 三把钥匙：**BUN:Cr 比值 + FeNa (排钠分数 fractional excretion of Na) + 尿沉渣 (urine sediment)**。

| 类型 | BUN:Cr | FeNa | 尿钠 (UNa) | 尿渗 (Uosm) | 尿沉渣钥匙 |
|---|---|---|---|---|---|
| 肾前性 prerenal | >20:1 | <1% | <20 | ↑（浓缩） | bland（干净）/ 透明管型 |
| 肾性 ATN (急性肾小管坏死 acute tubular necrosis) | <20 | >2% | >40 | ↓（等渗） | **muddy brown casts 泥棕色颗粒管型** |
| 肾后性 postrenal | 不定 | ↑ | 不定 | — | 梗阻（影像找） |

```
AKI（Cr↑/少尿）
  ① 容量不足/有效循环↓（呕泻、出血、HF、肝硬化）
        → 肾前性：BUN:Cr>20、FeNa<1%、UNa<20、尿浓缩 → 补液
  ② 缺血/肾毒（造影剂、氨基糖苷、横纹肌溶解）
        → ATN：FeNa>2%、muddy brown casts → 支持治疗 + 补液
  ③ 排尿困难/独肾/盆腔肿块/BPH
        → 肾后性：先 Renal US 查肾积水 → 解除梗阻
```

> [!warning] 经典陷阱
> - **造影剂肾病 (contrast-associated AKI, CA-AKI)**：48–72h 内 Cr↑；预防 = **IV 0.9% NS 等渗补液**（停 metformin 二甲双胍是防乳酸酸中毒，**不是**防 AKI）。
> - **横纹肌溶解 (rhabdomyolysis)**：UA 血 (+) 但**镜下无 RBC**（= 肌红蛋白 myoglobin）；电解质四联 **K↑ / PO4↑ / Ca↓ / CPK↑**；Tx = 大量补液 ± 甘露醇。
> - **胆固醇结晶栓塞 (cholesterol embolism)**：导管术后数周、嗜酸粒细胞↑ + 低补体 + 蓝趾 / livedo。
> - **肝肾综合征 (hepatorenal syndrome, HRS)**：肝硬化 + AKI + UA bland + 肾 US 正常 → 直接**测尿钠（<10）**，不先扩容、不先影像。

## 二、间质性肾炎 AIN vs ATN vs 慢性间质性肾炎 CTIN

| 病变 | 病因钥匙 | UA 指纹 | 病程 |
|---|---|---|---|
| 急性间质性肾炎 AIN (acute interstitial nephritis) | 药物（非甾体抗炎药 NSAID / β-内酰胺 / 质子泵抑制剂 PPI / 利尿剂 / 别嘌醇 allopurinol / 磺胺）| **WBC casts 白细胞管型 + 嗜酸 eosinophiluria** | 数周 |
| ATN | 缺血 / 造影剂 / 肾毒 | muddy brown casts | 急性 |
| 慢性间质性肾炎 CTIN | 长期镇痛药（analgesic nephropathy）| 双侧小肾 + 乳头坏死 | 慢性 |

> [!danger] AIN vs 药物性狼疮 (drug-induced lupus, DIL) 分水岭（反复错点）
> **WBC casts + 嗜酸 = AIN**（经间质）；DIL 经肾小球 → **RBC casts + 重度蛋白尿**，且 DIL 致 AKI 罕见。
> AIN 处理：**停药** + 类固醇评估。

## 三、肾炎综合征 vs 肾病综合征（nephritic vs nephrotic）

> [!tip] 3 秒分型
> **肾炎 = 血尿为主 + 炎症**（RBC casts、变形 RBC、高血压 HTN (hypertension)、少尿）；**肾病 = 蛋白尿为主**（>3.5 g/d、低白蛋白、水肿、高脂、泡沫尿）。

| 维度 | 肾炎 Nephritic | 肾病 Nephrotic |
|---|---|---|
| 核心 | 血尿 + RBC casts | 蛋白尿 >3.5 g/d |
| 屏障 | 机械屏障损（炎症）| 足突 / GBM 蛋白选择性损 |
| 高频病 | 链球菌感染后肾炎 PSGN (poststreptococcal GN) / IgA 肾病 / 急进性肾炎 RPGN (rapidly progressive GN) / Alport / 狼疮 IV 型 | 微小病变 MCD (minimal change disease)（儿童）/ 局灶节段性肾小球硬化 FSGS / 膜性肾病 / 糖尿病肾病 / 淀粉样变 |
| 治疗主线 | 降容量（袢利尿）+ 治原发；活动性沉渣 → 肾活检 | ACEi/ARB 减蛋白尿 + 限盐利尿；多需肾活检定型 |

> [!info] 低补体 (low complement) 分水岭（爱考）
> **C3/C4 低**：PSGN / 狼疮 / 膜增生性肾炎 MPGN (membranoproliferative GN) / 心内膜炎相关 / 冷球蛋白（免疫复合物型）。
> **补体正常**：Goodpasture（抗肾小球基底膜 anti-GBM, glomerular basement membrane）/ 抗中性粒细胞胞浆抗体 ANCA (antineutrophil cytoplasmic antibody) 相关 GN（pauci-immune 寡免疫型）。

> [!warning] 肾病综合征并发症（buzzword）
> 高凝（丢抗凝血酶 III → 肾静脉血栓 / DVT-PE）/ 感染（丢 Ig → 荚膜菌）/ 加速动脉粥样硬化（高脂）/ 总钙↓但离子钙正常（丢白蛋白结合钙）。
> **长期类风湿关节炎 RA (rheumatoid arthritis) / 炎症性肠病 IBD (inflammatory bowel disease) + 肾综无管型 = AA 淀粉样变 (amyloidosis)**（刚果红 Congo red 阳性、苹果绿双折射）。

## 四、电解质急症（最危险器官 = 心脏 → 先 ECG）

> [!danger] 高钾血症 (hyperkalemia) 三级火箭
> **ECG 变了先给钙**（稳膜，1 分钟）→ **胰岛素 + 葡萄糖**（移 K 入细胞，"INSulin is IN"）+ β激动剂雾化 ± 碳酸氢钠 → **排 K**（袢利尿 / K 结合剂 / 透析）。
> ECG 进展口诀：**T 尖 → PR 长 → P 消失 → QRS 宽 (sine wave) → VF/停搏**。

| 紊乱 | 钥匙 / ECG | 处理要点 |
|---|---|---|
| 高钾 hyperkalemia | T 波高尖 peaked T；挤压伤 / AKI / 醛固酮↓药 | 钙 → 胰岛素+糖 → 排 K |
| 低钾 hypokalemia | **T 波平 + U 波** + ST 压低 | 补钾（先纠低镁否则补不进）|
| 高钠 hypernatremia | 水缺 / 尿崩症 DI (diabetes insipidus)；血浓尿稀 | 补自由水（5% 葡萄糖液 D5W）；降钠 <10–12 mEq/L/d |
| 低钠 hyponatremia | 先看血浆渗透压 Serum Osm → 有效循环容量 ECV → 尿渗 Uosm | 见 §五 SOP |
| 高钙 hypercalcemia | QT 短；门诊 = PHPT / 住院 = 恶性 | **先 NS 补液**再用药（噻嗪禁用，反升钙）|
| 低钙 hypocalcemia | QT 长；Chvostek/Trousseau | 静脉钙剂 |
| 低镁 hypomagnesemia | TdP（尖端扭转）；**酗酒 + PPI** | 补镁（钾/钙补不进的"隐藏 boss"）|

> [!info] 校正钙公式（源文件原文）
> 校正钙 = 测量钙 + 0.8 × (4.0 − 白蛋白)。

## 五、低钠血症 (hyponatremia) 诊疗 SOP

```
Step 0：先看 Serum Osm（排假性/高渗低钠）
  高渗 → 高血糖（纠正钠 = 测量 Na + 1.6×(Glu−100)/100）/ 甘露醇
  等渗 → 假性低钠（高脂/高蛋白）
  低渗（真低钠）↓
Step 1：看 ECV（容量状态）
  低容量 → UNa<20 外丢（呕泻）/ UNa>20 肾丢（利尿剂/矿皮缺）
  等容量 → 看 Uosm：>100 = 抗利尿激素分泌失调 SIADH / 甲减 / 肾上腺；<100 = 原发性多饮
  高容量 → 心衰 HF / 肝硬化 / 肾综（有效循环↓ → 抗利尿激素 ADH 开）
```

> [!danger] 纠钠红线（双向，方向不同数值不同）
> **纠正低钠（升 Na）：≤8 mEq/L/24h**（慢性；ΔNa ≤0.5/hr）——防 ODS。
> **纠正高钠（降 Na）：≤10–12 mEq/L/day**——防脑水肿。
> **Low to High, your pons will die**（低钠补太快 → 渗透性脱髓鞘 ODS / 桥脑）；**High to Low, your brain will blow**（高钠降太快 → 脑水肿）。
> 重症（Na<120 或 seizure / 重度意识障碍）→ **3% 高张盐水**（NS 反加重，HCO3 无效）。

## 六、酸碱（acid-base）代偿 SOP + AG + RTA

> [!info] 正常值（源文件原文）
> pH 7.35–7.45 ｜ HCO₃⁻ 22–28 ｜ CO₂ 35–45。
> **Anion Gap (AG, 阴离子间隙)** = Na − (Cl + HCO₃)，正常 8–12，口诀 **MUDPILES**。
> **Winter's Formula**：预期 CO₂ = (1.5 × HCO₃) + 8 ± 2（查混合型 mixed disorder）。

```
三步法
  ① pH → 酸血症 (<7.35) vs 碱血症 (>7.45)
  ② 原发 = 代谢 (HCO3) 还是 呼吸 (CO2)？
  ③ 代偿是否到位（Winters / 1-for-10 等）→ 不到位 = 混合型
```

> [!tip] 肾小管酸中毒 (renal tubular acidosis, RTA) 分型（源文件原文）
>
> | 型 | K⁺ | 钥匙 |
> |---|---|---|
> | Type 1（远端 distal）| ↓ | **尿 pH >5.5**（不能酸化尿）|
> | Type 2（近端 proximal, 近曲小管 PCT 缺陷）| ↓ | 近端 HCO3 重吸收障碍 |
> | Type 4 | ↑ | 醛固酮缺陷 (aldosterone defect) |

## 七、肾结石 5 型（nephrolithiasis）

| 类型 | 钥匙 | 尿 pH | X 线 | 处理 |
|---|---|---|---|---|
| 草酸钙 Ca oxalate（最常见）| Envelope 信封晶体 | — | 阳性 | 噻嗪 thiazide（减尿钙）|
| 鸟粪石 struvite | Urease+（脲酶）菌、coffin-lid、鹿角石、反复尿路感染 UTI (urinary tract infection) | **碱性 >8** | 阳性 | 酸化尿 + 取石（经皮肾镜取石 PCNL, percutaneous nephrolithotomy）+ 抗生素 |
| 尿酸石 uric acid | 痛风 gout、肿瘤溶解综合征 TLS (tumor lysis syndrome) | **酸性** | **radiolucent 透 X 线阴性** | **碱化尿** |
| 胱氨酸 cystine | Hexagonal 六角晶体 | 酸性 | — | 碱化尿 |

> [!info] 影像下一步（源文件原文）
> 首选 **无对比 CT KUB（肾输尿管膀胱 kidney-ureter-bladder）**（最敏感）；孕妇 / 儿童 **超声优先**（避辐射）。
> 急诊泌尿引流指征：发热/败血、单肾或梗阻 + AKI、顽固痛/呕吐、巨大结石。

> [!tip] 输尿管结石处理（按大小）
> ≤5 mm 保守 / 5–10 mm MET（药物排石治疗 medical expulsive therapy，**坦索罗辛 tamsulosin**）/ >10 mm 外科。

## 八、囊肿 / 多囊肾（cyst / PKD）

> [!tip] 单纯囊肿 vs 恶性（Bosniak）
> 单纯囊肿（薄壁 / 单腔 / 无 septa / 无 enhancement / 无症状）→ **reassurance 安慰**。
> 关键恶性指标 = **强化 enhancement**。Bosniak：I–II 无需处理 / IIF 随访 / III–IV 手术。

> [!info] 常染色体显性多囊肾 (autosomal dominant PKD, ADPKD) 6 联正向指纹
> 双侧腰痛 + 肝大/包块 + 早发 HTN（高血压）+ 长期夜尿 + 慢性肾脏病 CKD (chronic kidney disease) 伴贫血 + 家族史。
> Extrarenal "Brain-Liver-Heart-Colon-Hernia"：**Berry 浆果动脉瘤**（颅内出血）/ 肝囊肿 / 二尖瓣脱垂 / 结肠憩室 / 疝。

## 九、血尿 (hematuria) 与下尿路梗阻 / BPH

> [!tip] 血尿分期定位
> **Initial（初始）= 尿道**；**Terminal（终末）= 膀胱颈/三角区**；**Total（全程）= 整个尿路 / 肾**（如肾乳头坏死 RPN, renal papillary necrosis）。
> **Clots 血块 = NOT 肾小球**（肾小球血尿 3 标志：RBC casts / 变形 RBC / 蛋白尿）。

> [!warning] 膀胱癌 (bladder cancer) 钥匙
> 老年男 + 吸烟 + **无痛肉眼血尿 + 血块** → **膀胱镜 cystoscopy**。

> [!info] 良性前列腺增生 (benign prostatic hyperplasia, BPH) 药物
> **α1 阻滞剂坦索罗辛 tamsulosin**（即刻松弛，注意白内障术前 IFIS 术中虹膜松弛综合征 intraoperative floppy iris syndrome）；**5α 还原酶抑制剂非那雄胺 finasteride**（缩体积，需数月，缩小腺体 + 降 PSA）。

## 十、肾移植（transplant）排异与术后

> [!tip] 移植后 AKI（post-transplant AKI）5 鉴别
> 急性排异 / 钙调磷酸酶抑制剂 (calcineurin inhibitor, CNI：tacrolimus 他克莫司 / cyclosporine 环孢素) 毒性 / 移植肾动脉狭窄 (RAS) / BK-CMV 病毒 / ATN-梗阻。

> CNI 急性毒性 = 血管收缩（加 diltiazem/azole/macrolide 经 CYP3A4 升血药 → Cr↑）→ 测血药浓度。

## 十一、利尿剂 + 紧急透析指征

> [!info] 利尿剂分类钥匙
> 袢利尿剂（呋塞米 furosemide）= 强力 + **耳毒性**（肾衰/高剂量/合用其他耳毒药）；噻嗪（thiazide）= 升钙降钙尿、低钠/低钾；保钾（螺内酯 spironolactone / amiloride）= 高钾；乙酰唑胺 acetazolamide = 代谢性酸中毒。
> **磺胺过敏可用 ethacrynic acid 依他尼酸**（其余利尿剂多含 sulfa）。

> [!danger] 紧急透析指征 "AEIOU"（源文件原文）
> **A**cidosis（pH<7.1 难纠正）/ **E**lectrolytes（K>6.5 或 ECG 改变）/ **I**ntoxication（甲醇、水杨酸、锂）/ **O**verload（顽固肺水肿）/ **U**remia（尿毒症：心包炎、脑病）。

## 十二、泌尿生殖 GU (genitourinary) 肿瘤（与外科交叉，简表）

| 肿瘤 | 钥匙 buzzword | 下一步 |
|---|---|---|
| 肾细胞癌 RCC (renal cell carcinoma) | 血尿 + 侧腹痛 + 包块（± 促红细胞生成素 EPO↑ → 红细胞增多）| 增强 CT → 根治性肾切除 |
| 膀胱癌 bladder cancer | 无痛肉眼血尿（吸烟/血吸虫）| 膀胱镜 → 经尿道膀胱肿瘤切除 TURBT (transurethral resection of bladder tumor) |
| 前列腺癌 | 多无症状或似 BPH；硬结节 | 直肠指检 DRE (digital rectal exam) + 前列腺特异抗原 PSA (prostate-specific antigen) → 活检 |

> [!info] 跨学科提示
> RCC / 膀胱癌 / 前列腺癌的外科处理见外科卡；本卡只保留识别钥匙 + 首步。

# 肾结石 5 型对照（Nephrolithiasis）

> [!info] 一句话总览
> 5 大肾结石靠 **3 条线索**一眼分型：**晶体外形 + 尿 pH + X 线透/阻**。
> 题干给晶体形（envelope / coffin lid / hexagonal / rhomboid）或尿 pH 方向，基本就锁定类型 + 病机 + 治疗。

---

## 一、5 型对照总表 ⭐⭐（核心）

| 类型 | 占比 | 病机 / 触发 | 晶体外形 | 尿 pH | X 线 | 治疗 |
|---|---|---|---|---|---|---|
| **Ca oxalate**（草酸钙）| ~75% | 高 Ca / 高 oxalate / **低柠檬酸** / 脂肪泻→游离 oxalate↑ / **低钙饮食反致↑** | **Envelope 信封/八面体** 或 dumbbell | acidic–neutral（pH 不敏感）| **radiopaque** | 多饮水 + **thiazide**（↓尿 Ca）+ citrate |
| **Ca phosphate**（磷酸钙）| ~15% | **RTA type 1**（远端）/ hyperPTH | wedge / rosette | **alkaline 碱性** | **radiopaque** | thiazide + citrate；治原发病 |
| **Struvite**（鸟粪石 MgNH4PO4）| ~10–15% | **urease+ 菌**水解 urea→NH3→尿氨↑ + pH>8 | **Coffin lid 棺材盖** | **>8 强碱** | radiopaque（大 staghorn）| **完整取石 PCNL** + 抗生素 + urease inhibitor |
| **Uric acid**（尿酸）| ~5–10% | gout / **酸性尿** / 肿瘤溶解 / 慢性腹泻丢 HCO3 | rhomboid 菱形 / rosette | **<5.5 酸性** | **radiolucent 透 X 线** ⚠️ | **碱化尿**（potassium citrate）+ allopurinol |
| **Cystine**（胱氨酸）| <1% | **Cystinuria（AR）**转运缺陷，**儿童/青年反复石** | **Hexagonal 六角** | acidic | weakly radiopaque | 碱化尿 + 大量饮水 + chelator |

> [!tip] 3 秒分型口诀
> - **信封形 → Ca oxalate**（最常见，pH 不挑）
> - **棺材盖 + 反复 UTI + pH>8 + staghorn → Struvite**
> - **六角形 → Cystine**（年轻 + 家族 + AR）
> - **菱形 + 酸性 + radiolucent → Uric acid**（CT 看得见，普通 X 线看不见）
> - **碱性 + RTA1 / hyperPTH → Ca phosphate**

---

## 二、尿 pH 互反 pattern ⭐（高频考点）

```
碱性尿 (pH↑) 沉淀 ───── Struvite (pH>8)
                  └──── Ca phosphate
                  → 治疗：酸化思路 / 治原发；不要再碱化

酸性尿 (pH↓) 沉淀 ───── Uric acid (pH<5.5)
                  └──── Cystine
                  → 治疗：碱化尿 (potassium citrate) ⭐

pH 不敏感 ──────────── Ca oxalate（最常见，靠饮食 + thiazide）
```

> [!warning] 反向陷阱
> **碱化尿（potassium citrate）只用于酸性石（uric acid / cystine）**。
> 若是 struvite / Ca-P（本就碱性）再碱化只会**助长**结石 —— 题干给"碱性尿 + UTI"绝不能选碱化尿。

---

## 三、Struvite 专题（唯一"感染石"）⭐⭐

> [!danger] Struvite = 唯一与"细菌感染"直接因果的结石
> **urease+ 菌**水解 urea → **NH3（氨）** → 碱化尿（pH>8）→ Mg/NH4⁺/PO4 过饱和沉淀 → **长得快 + 大 staghorn 鹿角石**。

### Struvite 5 联指纹（题干见到即锁定）

1. **反复 UTI 史**（dysuria / fever / 轻度 flank pain）
2. **尿 pH > 8**（urea→NH3 碱化）
3. **Coffin lid 棺材盖晶体**
4. **大 staghorn 鹿角石**（充满 calyces）
5. **女性多发**（女性 UTI 多）

### Urease+ 菌口诀 "PPP S K U"

> [!tip] 产 urease 的菌
> **P**roteus / **P**rovidencia / **P**seudomonas（部分）/ **S**taph saprophyticus / **K**lebsiella / **U**reaplasma
>
> ⚠️ **E. coli 不产 urease** —— 最常见 UTI 元凶却**不**致 struvite（经典反 pattern）。

### 治疗三件套 + Stone matrix 概念

> [!danger] Struvite 治疗 = 取石 + 抗生素 + urease inhibitor
> 1. **完整取石（PCNL 经皮肾镜碎石）** —— 最关键
> 2. **抗生素**
> 3. **urease inhibitor**（acetohydroxamic acid）防复发
>
> **为什么抗生素单用不够**：菌**藏在 stone matrix（结石基质）内**，抗生素打不到 —— 不取干净石就反复复发。

> [!info] 反 pattern：大 staghorn 不绞痛
> 大 staghorn 结石**不进 ureter** → 表现为**反复 UTI > 典型肾绞痛**。
> 经典肾绞痛（colic）是**小石过 UJ（输尿管狭窄处）**的特征。

## 三点五、Uric acid 尿酸石专题（酸性石）⭐⭐

> [!danger] Uric acid 石 = 唯一"酸性尿驱动 + radiolucent"石
> 核心驱动不是尿酸总量，而是 **尿 pH<5.5**：酸性环境下可溶的 **urate** 转为不溶的 **uric acid** → 溶解度骤降 → 过饱和析晶。

### 三点五.1 尿酸石病机三驱动

| 驱动 | 触发 | 机制 |
|---|---|---|
| **酸性尿 (pH<5.5)** ⭐核心 | gout 体质 / **慢性腹泻丢 HCO3** / 代谢综合征-DM（胰岛素抵抗→肾产氨↓→尿酸化）/ 高动物蛋白 | 酸性环境 uric acid（不溶）占比↑ |
| **尿酸排泄↑** | gout / 骨髓增殖病 / **肿瘤溶解综合征（TLS）** / 高嘌呤饮食 | 成石原料↑ |
| **尿浓缩** | 脱水 / 炎热干旱气候 | 浓度↑ → 过饱和 |

> [!info] TLS（肿瘤溶解综合征）钩子 ⭐
> 化疗后大量细胞裂解 → **尿酸↑↑ + K↑ + PO4↑ + Ca↓** → 尿酸结晶阻塞肾小管致 **AKI**。
> 预防：化疗前 **水化 + allopurinol**（高危用 **rasburicase**）。这是尿酸"急性"成石 / 肾病最高产的考场场景。

> [!warning] radiolucent 复习
> 尿酸石普通 **KUB 透明** → 平片阴性 ≠ 无石；**non-contrast CT 才是金标准**（CT 上仍可见）。

> [!success] 治疗 —— 少数可药物"溶石"的类型
> **碱化尿（potassium citrate，目标 pH 6–7）** 让 uric acid→可溶 urate → **溶解既有石**（不一定需碎石）+ **allopurinol**（↓尿酸生成）+ 多饮水。

## 四、钙结石防复发（Ca oxalate / Ca phosphate）⭐⭐

> [!success] 防复发原则
> ↓ 尿中**成石物质**（Ca、oxalate）+ ↑ **抑制因子**（citrate）。

| 措施 | 机制 |
|---|---|
| ↑ 饮水（产 >2 L/日尿）| ↑ 尿流，↓ 溶质浓度 |
| **↓ 钠（<2300 mg/日）** ⭐ | ↑ 近端小管 Na-水重吸收，耦联 ↑ Ca 重吸收 → **↓ 尿钙** |
| ↑ 枸橼酸 citrate（果蔬）| 结合尿钙、抑制成石 |
| ↑ 钾 | ↑ 尿枸橼酸排泄 |
| ↓ 动物蛋白 | ↓ 尿钙排泄 + ↑ 枸橼酸 |
| **正常 / 充足膳食钙（1200 mg/日）** ⭐ | 肠道结合 oxalate → **↓ oxalate 吸收** |
| ↓ oxalate（菠菜 / 土豆）| ↓ 尿 oxalate 排泄 |
| 药物 thiazide | ↑ 肾 Ca 重吸收（↓ 尿钙）|
| 药物 potassium citrate | ↑ 尿枸橼酸浓度 |

> [!danger] 经典反直觉陷阱：钙石 ≠ 少吃钙 ⭐⭐
> "钙结石 → 少吃钙" 是直觉但**错**：
> **低膳食钙 → 肠道结合 oxalate 减少 → oxalate 吸收↑ → 尿 oxalate↑ → 结石更多。**
> 正确做法 = **维持正常膳食钙 1200 mg/日（膳食非补剂）+ 降钠**。
>
> 区别：**随餐的膳食钙**在肠腔绑 oxalate（好）；**钙补剂**另算（可能反致风险）。

### 钙-草酸"跷跷板"机制 ⭐（肠源性高草酸尿）

> [!warning] 一图解释两个反直觉点
> **吸收不良（Crohn / 小肠切除 / 胃旁路）**和**低钙饮食**殊途同归——都让肠腔可绑 oxalate 的钙减少 → oxalate 游离被吸收 → 草酸钙石↑。

```
正常：膳食钙 在肠腔绑 oxalate → 不溶草酸钙 → 随粪排出（oxalate 吸收少）

钙被"抢走"的两条路：
  ① 吸收不良（Crohn/小肠切除/胃旁路）
       脂肪吸收↓ → 肠腔游离脂肪酸↑ → 抢先皂化结合钙
  ② 低钙饮食
       本就没有足够膳食钙去绑 oxalate
            │
            ▼  可绑 oxalate 的钙↓
       游离 oxalate↑ → 肠吸收↑（enteric hyperabsorption）
            │
            ▼
       尿 oxalate↑（hyperoxaluria）→ 草酸钙结石
```

> [!tip] 高钠 stem → 锁降钠
> 题干见"预制食品 / 罐头 / 苏打饮料 / 外卖"= 高钠负荷 → **首选降钠**（不是降钙）。

## 五、X 线透 / 阻速记

```
radiopaque（X 线看得见，亮）：
  Ca oxalate / Ca phosphate / Struvite  ── 含钙或大石

radiolucent（X 线看不见，透）⚠️：
  Uric acid  ── 普通 KUB 漏诊，但 CT 仍可见

weakly radiopaque：
  Cystine
```

> [!warning] Uric acid 是唯一"普通 X 线透明"石
> 普通 KUB 看不到 → **诊断金标准仍是 non-contrast CT（CT 上所有石都可见）**。
> 题干"X 线阴性但临床高度疑结石" → 想 uric acid。

## 六、急性肾绞痛处理（任何类型通用）

> [!info] 诊断 + 处理
> - **题眼**：阵发**绞痛（坐立不安）**→ 向**腹股沟放射**；恶心呕吐；± 排尿刺激征
> - **UA**：血尿常见；通常**无 WBC casts**；± 晶体
> - **影像首选**：**non-contrast CT KUB**（最敏感）；**孕妇 / 儿童 → 超声优先**
> - **止痛 + 排石**：NSAID（如 ketorolac）+ 补液；± **tamsulosin** 助排（远端小结石 MET）

> [!danger] 急诊泌尿介入指征（需引流：支架 / 肾造瘘）
> - **发热 / 败血**（梗阻 + 感染 = 急症）
> - **单肾或梗阻 + AKI**
> - 顽固疼痛 / 呕吐
> - **巨大结石**（如 staghorn）

> [!tip] 10 秒超级对比
> - **阵发绞痛、坐立不安 → 结石**
> - **发热 + CVA 叩击痛 + WBC casts → 肾盂肾炎**

> [!warning] 儿童结石"非典型"——常无痛 ⭐（Q15685）
> 成人结石靠"过输尿管的急性绞痛"，但**幼儿结石多位于肾内（未下行）→ 无痛**，可仅表现**孤立无痛肉眼血尿**（无腰腹痛）。
> - **别因无痛就排除结石**；UA 可仅血尿、不一定见晶体。
> - **hypercalciuria = 儿童结石头号危险因素** → 查 **尿 Ca:Cr 比值**。
> - **影像首选肾+膀胱超声（避辐射）**，超声不确定再 non-contrast CT；**cystoscopy 不用于儿童**。

## 七、反 pattern / 易混陷阱汇总

> [!warning] 高频反直觉点
> - **Hypoparathyroidism（不是 hyper-）**→ 肾 Ca 重吸收↓，但因总 Ca 低 → Ca stone **罕见**（反直觉）
> - **低钙饮食 / 脂肪吸收不良** → 游离 oxalate↑ → **Ca oxalate stone 反而↑**
> - **大 staghorn 不绞痛** → 表现为反复 UTI（绞痛是小石过 UJ）
> - **E. coli 不产 urease** → 不致 struvite（最常见 UTI 菌的例外）
> - **碱性石不能再碱化**（struvite / Ca-P）；碱化尿只给酸性石（uric acid / cystine）
> - **Uric acid 石 X 线透明** → 别因 KUB 阴性就排除结石

## 八、Memory Hooks ⭐

> [!tip] 一句话锁定
> - **"信封 = oxalate；棺材盖 = struvite；六角 = cystine；菱形 = uric acid"**（晶体形 4 锁）
> - **"Coffin lid + pH>8 + staghorn + 反复 UTI = Struvite，机制 = 尿氨↑（urease 菌）"**
> - **"Struvite 治疗 = 取石（PCNL）+ 抗生素 + urease inhibitor 三件套"**（菌藏 stone matrix）
> - **Urease+ "PPP-S-K-U"；E. coli 不算**
> - **"酸性石碱化尿（uric acid / cystine）；碱性石别再碱化（struvite / Ca-P）"**
> - **"钙草酸石防复发 = 降钠（↓尿钙）+ 维持正常膳食钙（肠道绑 oxalate）"**
> - **"Uric acid 石 radiolucent —— KUB 透明，CT 才见"**

## 九、自测题

> [!question]- Q1. 女患反复 UTI，UA 见尿 pH 8.5 + "棺材盖" 晶体，US 左肾盂 2 cm 不规则结石。① 结石类型？② 形成机制？③ 一线治疗？
> ① **Struvite（MgNH4PO4 鸟粪石）**
> ② **urease+ 菌水解 urea → 尿氨↑ → 碱化尿（pH>8）→ Mg/NH4/PO4 沉淀**
> ③ **完整取石 PCNL + 抗生素 + urease inhibitor（acetohydroxamic acid）**；抗生素单用不够（菌藏 stone matrix）

> [!question]- Q2. 50yo M 反复钙草酸结石，常吃预制食品 + 苏打饮料。最能降低复发风险的饮食改变？为什么不是"少吃钙"？
> **降钠（<2300 mg/日）** → 近端小管 Na-水重吸收↑ 耦联 Ca 重吸收↑ → ↓尿钙。
> **不选少吃钙**：低膳食钙 → 肠道绑 oxalate↓ → oxalate 吸收↑ → 尿 oxalate↑ → 结石更多；应维持正常膳食钙 1200 mg/日。

> [!question]- Q3. 哪类肾结石普通 X 线（KUB）看不见？诊断该用什么？
> **Uric acid（尿酸石）= radiolucent**，普通 KUB 透明。
> 诊断金标准 **non-contrast CT**（CT 上所有结石都可见）。治疗：碱化尿（potassium citrate）+ allopurinol。

> [!question]- Q4. 哪些结石需要碱化尿？哪些绝不能碱化？
> **碱化尿（酸性石）**：Uric acid、Cystine。
> **绝不碱化（本就碱性石）**：Struvite、Ca phosphate —— 再碱化助长成石。

> [!question]- Q5. 产 urease 的菌有哪些？哪个最常见 UTI 菌却是例外？
> **"PPP-S-K-U"**：Proteus / Providencia / Pseudomonas（部分）/ Staph saprophyticus / Klebsiella / Ureaplasma。
> 例外：**E. coli 不产 urease**（最常见 UTI 菌，但不致 struvite）。

# 🧪 低钠血症：先按血清渗透压分类（Hyponatremia — Step 1: Classify by Serum Osmolality）

> [!info] 笔记定位（与姊妹笔记的分工）
> 这是低钠 workup 的**第一步专注卡**：见低钠 **先别套 SIADH / 多饮模板**，先用**血清渗透压 (serum osmolality, Posm)** 把低钠分成**高渗 / 等渗 / 低渗**三桶。本卡深挖容易被忽略的**高渗性（转位性）**和**等渗性（假性）**两桶（= NBME Q123 的陷阱）。
> - **低渗性（真低钠）那一桶的下游算法**（容量 → 尿渗 → 病因 → 治疗）见 [[完整笔记/专题笔记/肾脏/肾脏_低钠血症诊疗SOP]]（本卡只链接，不重复）。
> - **低钠在多电解质语境**（治疗双轴矩阵 / thiazide HypoNa / seizure 鉴别）见 [[完整笔记/专题笔记/肾脏/肾脏_电解质急症速查]] §2.1。
> - 派生自 [[NBME11_错题本#^Q123]]（HHS 高糖致高渗性低钠，误归因 ramipril）。

---

## 一、为什么"按渗透压"是第一步（核心顿悟）⭐

> [!success] 一句话
> **血钠浓度 = 钠 ÷ 水（一个比值），不是钠的绝对量。** 所以"低钠"可能是钠少，也可能是水多、或水被拉到血里稀释。**先量 Posm 验明正身**——是真"水多"（低渗），还是"假象/水位移"（等渗 / 高渗）。

- 见低钠就反射 **SIADH（抗利尿激素分泌不当综合征 syndrome of inappropriate ADH）** / 心因性多饮 = **跳步**：SIADH 和多饮都是**低渗性（真低钠）**桶里的病因，**必须先确认 Posm <275** 才能往那个方向走。
- 高渗、等渗两桶的处理与低渗**完全不同**（高渗治原发/降糖；等渗根本不用治低钠），错分桶 → 错治疗。

```
低钠 Na <135
   │
①【先量血清渗透压 Posm】 ← 本卡的第一步
   │
   ├─ Posm >295（高渗）→ 高渗性 / 转位性低钠（高血糖、甘露醇）→ §三
   ├─ Posm 275–295（等渗）→ 假性低钠（高脂、副蛋白）→ §四
   └─ Posm <275（低渗）→ 真低钠 → 转 [[肾脏_低钠血症诊疗SOP]] 查容量/尿渗 → §五
```

> [!tip] Posm 估算公式（USMLE 常算）
> $$\text{Posm} \approx 2\,[\text{Na}^+] + \frac{\text{Glucose}}{18} + \frac{\text{BUN}}{2.8}$$
> 正常范围 **275–295 mOsm/kg**。Glu 用 mg/dL，**BUN（血尿素氮 blood urea nitrogen）** 用 mg/dL。

## 二、三类渗透压分桶大表 ⭐⭐⭐

> [!danger] 这张表是低钠的"总入口"——先定桶再谈病因

| 血清渗透压 Posm | 分类 | 机制一句话 | 典型病因 | 下一步 |
|---|---|---|---|---|
| **>295（高渗）** | 高渗性 / 转位性低钠 (hypertonic / translocational) | 有效渗透物聚在**细胞外液 (ECF, extracellular fluid)** → 水从细胞内**外移**稀释血钠（体内总钠不少）| **严重高血糖**（糖尿病酮症酸中毒 DKA / 高渗性高血糖状态 HHS）、甘露醇 (mannitol)、高渗造影剂 | 算**纠正钠**；治原发（降糖）|
| **275–295（等渗）** | 等渗性 = 假性低钠 (pseudohyponatremia) | 血浆固相（脂/蛋白）↑ → 每升血浆"水相"被挤占 → 间接法**测得**钠假性↓（真实钠正常）| 严重高甘油三酯血症、**副蛋白血症**（多发性骨髓瘤 MM / Waldenström）| 用**直接离子选择电极 (direct ISE)** / 血气复测；**不治低钠** |
| **<275（低渗）** | 低渗性 = 真低钠 (hypotonic) | 自由水相对钠**真的多**（多为 **ADH（抗利尿激素 antidiuretic hormone）** 不当留水）| SIADH、多饮、心衰/肝硬化、利尿剂、肾上腺/甲减…… | 转 [[肾脏_低钠血症诊疗SOP]]：查**容量 → 尿渗 → 尿钠** |

> [!warning] 只有"低渗 <275"才是真低钠
> 高渗、等渗两桶的血钠"低"都是**测量/位移现象**，细胞内并没有进水、没有脑水肿风险（高糖反而是脑细胞脱水）。**别给这两桶限水或高张盐水。**

## 三、高渗性 / 转位性低钠（Hypertonic / Translocational）⭐⭐

> [!info] 这是 Q123 的考点
> 严重高血糖 → 葡萄糖是 ECF 有效渗透物 → 水从细胞内被拉到细胞外 → **稀释血钠**。体内总钠**没丢**，是"被水位移稀释" = translocational。

### 3.1 高血糖纠正钠公式（Corrected Na）

> [!tip] 纠正钠（把高糖的稀释效应"扣回去"）
> $$\text{Corrected Na} \approx \text{Measured Na} + 1.6 \times \frac{(\text{Glucose} - 100)}{100}$$
> - 经典系数 **1.6**（每 Glu↑100 mg/dL，血钠实测↓约 1.6 mEq/L）。
> - **极重高血糖（Glu >400）** 用 **2.4** 更贴合真实下降。
> - **纠正后钠正常 → 证明是单纯高渗性低钠**（病人其实不缺钠）；纠正后仍低 → 合并真低渗性低钠。

**串例（Q123）**：Glu ≈ 700、Na ≈ 125、HCO₃ 正常（非 DKA）、脱水 → **HHS**。纠正钠 ≈ 125 + 1.6 ×(700−100)/100 ≈ 125 + 9.6 ≈ **135（正常）** → 证明血钠"低"全是高糖稀释 = 高渗性低钠，**不是 ramipril、不是 SIADH**。

### 3.2 高渗性低钠病因

| 病因 | 要点 |
|---|---|
| **严重高血糖**（DKA / HHS, 高渗性高血糖状态 hyperosmolar hyperglycemic state）| 最常考；Glu 常 >400–1000；HHS 无明显酮症（HCO₃ 正常）|
| **甘露醇 (mannitol)** | 渗透性利尿剂；外源有效渗透物，机制同高糖 |
| **高渗造影剂 / 高渗输注** | 短暂转位性低钠 |

> [!warning] 区分"有效"vs"无效"渗透物
> **尿素 (BUN)、乙醇 (ethanol)** 能自由进出细胞，**不**造成水跨膜位移 → 升 Posm 但**不**引起转位性低钠。只有**留在 ECF 的有效渗透物（葡萄糖、甘露醇）**才把水拉出来稀释钠。

## 四、等渗性低钠 = 假性低钠（Pseudohyponatremia）⭐

> [!info] 机制：是"测量假象"，不是真低钠
> 血浆 ≈ 93% 水相 + 7% 固相（脂质/蛋白）。当**脂质或蛋白大幅↑**，固相占比变大 → 每升血浆里"水相"变少 → **按整升血浆稀释后测量（间接离子选择电极 indirect ISE / 火焰光度法）** 得到的钠偏低；但**水相里真实的钠浓度正常**。

| 特征 | 假性低钠 |
|---|---|
| **Posm** | **正常（275–295）** — 关键鉴别点 |
| 血浆外观 | 乳糜状 (lipemic) / 高度浑浊（高脂时）|
| 复测方法 | **直接离子选择电极 (direct ISE)** 或血气分析仪 → 钠**正常** |
| 治疗 | **不治低钠**，治原发（降脂 / 治骨髓瘤）|

**病因**：

- **严重高甘油三酯血症 (severe hypertriglyceridemia)**（常 TG >1000，如急性胰腺炎背景）。
- **副蛋白血症 (paraproteinemia)**：**多发性骨髓瘤 (multiple myeloma, MM)**、Waldenström 巨球蛋白血症 → 血浆蛋白极高。

> [!tip] 一眼锁定假性低钠
> **低钠 + Posm 正常 + 血浆乳糜/高球蛋白** → pseudohyponatremia，别瞎补钠或限水。

## 五、低渗性低钠（真低钠）→ 转入 SOP（不在本卡展开）

确认 **Posm <275** 后，才进入真低钠的下游算法：

```
真低钠（Posm <275）
   │
② 容量状态（ECV, 有效循环血容量 effective circulating volume）：低容量 / 等容量 / 高容量
   │
③ 尿渗 (Uosm) + 尿钠 (UNa)：定具体病因
   │
④ 严重度（有无 seizure/coma、Na <120）→ 定治疗
```

> [!info] 下游详见姊妹笔记（避免重复）
> - 完整决策链 + SIADH 五联 + 治疗 SOP + **ODS（渗透性脱髓鞘综合征 osmotic demyelination syndrome）** 纠正速率铁律 → [[完整笔记/专题笔记/肾脏/肾脏_低钠血症诊疗SOP]]
> - 治疗双轴矩阵（容量 × 严重度）/ thiazide 致低钠指纹 / 电解质致 seizure 横向鉴别 → [[完整笔记/专题笔记/肾脏/肾脏_电解质急症速查]] §2.1
> - 选哪种液体纠正（D5W / NS / 3% 高张）→ [[完整笔记/专题笔记/肾脏/肾脏_补液晶体胶体对照]]

## 六、特例：TURP 综合征 / 灌洗液致低钠 ⭐

> [!warning] 经尿道前列腺电切术 (TURP) 综合征
> 术中用**甘氨酸 (glycine)** / **山梨醇 (sorbitol)** / 甘露醇灌洗液，经开放静脉窦大量吸收 → **稀释性低钠**（可真低渗 + 神经/视觉症状）。
> - 甘氨酸吸收还有**直接神经毒性 + 一过性视力模糊/失明**。
> - 灌洗液吸收会**抬高渗透压间隙**（见 §七）。
> - 处理：停灌洗、限水、重症高张盐水（同真低渗低钠）。

## 七、渗透压间隙（Osmolar Gap）——抓"未测渗透物"⭐

> [!tip] 测得 Posm − 计算 Posm > 10 = 有"隐形"渗透物
> $$\text{Osmolar gap} = \text{实测 Posm} - \text{计算 Posm}\;(>10\ \text{为异常})$$
> 提示存在公式未纳入的渗透物：**甲醇 / 乙二醇 (methanol / ethylene glycol)**、**甘露醇**、**TURP 灌洗液（甘氨酸/山梨醇）**、丙二醇。
> - 高糖/甘露醇致**高渗性低钠**时，渗透压间隙也可帮忙识别"无效 vs 外源渗透物"。
> - 详见 [[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全]]（渗透压间隙 / **AG（阴离子间隙 anion gap）** 算法）。

## 八、Memory Hook + 速记决策树

> [!success] 一句话锁死
> **见低钠先问三件事：血糖高不高？血脂/球蛋白高不高？都不高才是真低钠。**
> 高糖 → 高渗性（算纠正钠）｜高脂/高蛋白 → 等渗假性（不治）｜都正常且 Posm<275 → 真低钠（进 SOP）。

```
低钠 → 量 Posm
  >295  → 看血糖：高 → 高渗性低钠（纠正钠 = MeasNa + 1.6×(Glu-100)/100）
  正常  → 看血脂/蛋白：高 → 假性低钠（direct ISE 复测，不治）
  <275  → 真低钠 → [[肾脏_低钠血症诊疗SOP]]（容量→尿渗→尿钠→治疗）
```

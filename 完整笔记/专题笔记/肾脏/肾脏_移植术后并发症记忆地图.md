# 移植术后并发症记忆地图 (Post-Transplant Memory Map)

## 一、两句话总骨架 ⭐（先背这个，细节自己长出来）

> [!tip] 整块知识只挂在一条主轴 + 一个核心问题上
> - **主轴**：「距离手术多久？」→ 定**感染**窗口 + 定**鉴别**优先级
> - **核心问题**：「移植肾 **Cr 为什么升高**？」→ 五选一（药毒 / 排异 / BK / RAS / ATN）
>
> 记住这两句，90% 的内容能自己回忆出来。其余"肾外表现"（腹泻 / 肺浸润 / 血细胞↓）靠 §四 反查表分流。

---

## 二、记忆钩子（只背这三组）

### 钩子① 时间轴定感染（背窗口，不背病原体清单）

```
<1 月   = 还在医院  → 手术 / 导管 / 伤口（细菌、C. diff）
1–6 月  = 免疫最低谷 → 机会感染（CMV / BK / PCP）  ★最高频★
>6 月   = 回归社会  → 社区感染（普通肺炎 / UTI）
```

> [!tip] 一句话
> **"一月医院，半年机会，半年后社区。"**

### 钩子② 所有 Cr↑ 都问同一句话，再五选一

```
Cr ↑ → "拿什么区分？" → trough / BK-PCR / Doppler / ACEI史 / 活检
   ① 药毒性(trough高) | ② 排异(活检) | ③ BK(减药!) | ④ RAS(ACEI后骤升) | ⑤ ATN(第1周)
```

深度 5 项鉴别表见 [[完整笔记/专题笔记/肾脏/肾脏_移植后AKI鉴别SOP]] §二。

### 钩子③ 药物靠"人设"记，不背副作用表

```
环孢素 Cyclosporine = 爱美的人   → 多毛 hirsutism + 牙龈增生 gingival hyperplasia
他克莫司 Tacrolimus = 手抖的糖友 → NODAT 新发糖尿病 + 震颤 tremor（神经毒）
MMF 霉酚酸酯        = 拉肚子的孕妇 → 腹泻 + 致畸 + 骨髓抑制 + CMV 感染风险↑
硫唑嘌呤 Azathioprine = 别碰别嘌醇 → +allopurinol = 致命骨髓抑制（XO 互动）
西罗莫司 Sirolimus  = 伤口不愈的胖子 → 伤口差 + 高脂 + 蛋白尿 + 间质肺炎；但【不肾毒】
激素 Steroids       = 升糖升压突停死 → NODAT + 骨质疏松 + AVN + 突停 adrenal crisis
```

深度通路 + 监测 + 疫苗见 [[完整笔记/专题笔记/免疫/免疫_免疫抑制剂全谱]]。

## 三、对立配对（USMLE 专坑，成对记最牢）⭐

> [!danger] 这三对"看着一样、处理相反"——搞反会害死移植肾

| 看着一样（同一表现） | 处理 / 归属相反 |
|---|---|
| **BK 肾病 vs 急性排异**（都 Cr↑、都无典型症状）| **BK → 减免疫抑制 / 排异 → 加免疫抑制** |
| **他克莫司 vs 环孢素**（同机制 CNI）| **他克 → 糖尿病 + 震颤 / 环孢 → 牙龈增生 + 多毛** |
| **MMF 腹泻 vs CMV 肠炎**（都腹泻）| **减药好转 = MMF / 发热 + 血便 + 血细胞↓ = CMV** |
| **CNI 毒 vs Transplant RAS**（都 Cr↑、UA bland）| **trough 高 + 新加 CYP3A4 抑制剂 = CNI / 新加 ACEI 后骤升 + 难治 HTN = RAS** |
| **Leucovorin 配 MTX vs 配 5-FU** | **MTX → 救正常细胞 / 5-FU → 增强抗肿瘤**（相反） |

## 四、全症状导向反查表 ⭐⭐（"混在一起"那部分的解法）

> [!warning] 核心思路
> 免疫抑制剂副作用之所以"混进"排异 / 感染 / RAS，是因为它们**共享同一临床表现**。
> 拿到 stem 给的一个异常 → 先定**时间窗** → 再按本表把【药物 vs 排异 vs 感染 vs 血管】四方并排排除。

| 临床表现 | 药物原因 | 排异 | 感染 | 血管 / 其他 | 怎么分（关键鉴别动作） |
|---|---|---|---|---|---|
| **Cr 升高** | CNI 毒（trough 高）| 急性排异 | BK / CMV | Transplant RAS | trough、BK-PCR、Doppler、ACEI 史、**活检** |
| **腹泻** | MMF | — | **CMV 肠炎** / C. diff | — | 减药好转 = MMF；发热 + 血便 + 血细胞↓ = CMV |
| **白细胞 / 血小板↓** | MMF、Aza、Sirolimus、ATG | — | CMV、败血症 | — | CMV-PCR；查近期药 / 剂量 / 新加 allopurinol |
| **肺部浸润 + 干咳** | **Sirolimus 间质肺炎**（非感染）| — | **PCP / CMV / 真菌** | — | 病原学（PCP 染色、CMV-PCR）+ 停药试验 |
| **蛋白尿** | **Sirolimus** | 慢性排异 | — | 原病复发（**FSGS** 最经典）| 活检；病程（早期复发 FSGS / 慢排 IF/TA）|
| **高血糖 / 新发糖尿病** | **他克莫司、激素** | — | — | — | 时间相关性、减量 / 换药；NODAT |
| **难治性高血压** | CNI（轻）、激素 | — | — | **Transplant RAS** | RAS：ACEI 后 Cr 骤升 + graft bruit → Doppler |
| **发热** | （ATG 输注反应）| 急性排异（+ graft 压痛）| 任何机会感染 | — | 先按时间窗定感染谱 + 查 graft 区压痛 |

> [!tip] 用法
> 这张表是"症状 → 多病因"反查；若想"药物 → 症状"正查（见牙龈增生想环孢素等），用 [[完整笔记/专题笔记/免疫/免疫_免疫抑制剂全谱]] §十五 Pattern 速查。

## 五、感染预防（背两条就够）

| 预防药 | 防什么 | 备注 |
|---|---|---|
| **TMP-SMX** | **PCP**（兼防 UTI / Toxo / Nocardia）| 标准术后预防 |
| **Valganciclovir** | **CMV**（尤其供体+/受体− 高危）| 高危者延长疗程 |

> [!info] 疫苗一句话
> 移植**后**终生**禁活疫苗**（MMR / 水痘 / 鼻喷流感 / BCG / 黄热）；移植前抓紧补（≥4 周缓冲）。详见全谱篇 §十。

## 六、一页速记（考前扫一眼）

```
主轴：  距手术多久？ → 定感染窗 + 定鉴别
核心：  移植肾 Cr 为何升？ → 药毒/排异/BK/RAS/ATN 五选一

感染：  <1月医院 | 1–6月机会(CMV/BK/PCP) | >6月社区
        CMV→更昔洛韦；BK→减药（别和排异搞反！）
        预防：TMP-SMX(PCP)、缬更昔洛韦(CMV)

排异：  超急性=预存抗体(术中蓝) | 急性细胞=T细胞+tubulitis
        AMR=DSA+C4d | 慢性=IF/TA+蛋白尿；金标准=活检

RAS：   难治HTN + ACEI后Cr骤升 + bruit → Doppler → PTA±stent

药物人设：环孢=牙龈+多毛 | 他克=糖尿+震颤 | MMF=腹泻+致畸+CMV
        Aza+别嘌醇=致命 | 西罗莫司=伤口差+蛋白尿+间质肺炎(不肾毒)
        激素=升糖升压+突停危象

对立坑：BK减药 vs 排异加药 | MMF腹泻 vs CMV肠炎 | CNI毒 vs RAS
```

# 电解质急症速查（5 种电解质 + 酸碱）

> [!info] 用途
> 一张表速查 5 种电解质的高/低状态：定义 / 病因 / 临床 / ECG / 治疗 / 关键陷阱。深度版见各专题：[[完整笔记/专题笔记/肾脏/肾脏_高钾血症完整体系]] 等。ECG 变化集中见 [[完整笔记/专题笔记/心内/_衍生_代谢性ECG改变]]。

## §零 总览 — 哪个最致命？哪个看 ECG？

| 紊乱 | 致命性 | ECG 改变 | 急诊"先 stabilize" |
|---|---|---|---|
| **高钾** ⭐⭐⭐ | 心搏停 | **必看**（sine wave 濒死）| **Calcium gluconate** |
| 低钾 ⭐⭐ | TdP / 心室颤动 | U 波 / 长 QT | 补 K + 必查 Mg |
| 高钠 ⭐ | 脑出血（神经损伤）| — | 慢补水（防脑水肿）|
| 低钠 ⭐⭐ | 脑水肿 / 中央桥脑髓鞘溶解 | — | **慢补 Na** 防 ODS |
| 高钙 ⭐⭐ | 心律失常 / 昏迷 | **短 QT** | IV NS + calcitonin + bisphosphonate |
| 低钙 ⭐ | 喉痉挛 / TdP | **长 QT** | IV calcium gluconate |
| 高镁 ⭐ | 呼吸抑制 / 心搏停 | PR / QRS↑ | IV Ca + 透析 |
| 低镁 ⭐⭐ | TdP / 心律失常 | 长 QT | IV Mg + 必查 K |

> [!danger] 一句话原则
> **任何"看似 brady / 看似 TdP / 看似 shock" 的病人 → 必查电解质**。电解质紊乱常是病因（不是结果）。

## §一 钾 (Potassium)

### 1.1 高钾（K+ >5.0）

> [!info] 完整体系见 → [[完整笔记/专题笔记/肾脏/肾脏_高钾血症完整体系]]
> 错题：[[mistakes/uworld-mistakes#^Q19929]] 急性 ECG / [[mistakes/uworld-mistakes#^Q19-2026-05-18]] 三联炸弹元凶定位

**速查**：
- **ECG 5 阶段**：peaked T → P 变平 + PR↑ → P 消失 + QRS↑ → sine wave → asystole
- **三步治疗 (3 S)**：**Stabilize**（Calcium gluconate）→ **Shift**（Insulin/Glucose、Albuterol、NaHCO3）→ **Sink**（Furosemide、Patiromer、Dialysis）
- **三联炸弹**：ACEi/ARB + spironolactone + KCl
- **隐藏升 K**：**N**SAIDs / **T**MP-SMX / **S**piro / **H**eparin（NTSH）

### 1.2 低钾（K+ <3.5）

| 维度 | 内容 |
|---|---|
| **病因** | 利尿剂（袢/噻嗪）、呕吐/腹泻 GI 丢失、原发醛固酮增多症（Conn）、Cushing、RTA 1/2 型、Mg↓（K 跟着丢）、insulin、β激动剂、refeeding、Liddle |
| **临床** | 乏力、肌无力、便秘、肠麻痹、心悸；重度可呼吸肌瘫痪 |
| **ECG** | **U 波**（T 后小波）、ST 压低、T 平/倒、**长 QT/QU** → **TdP** |
| **关键陷阱** | **低镁不补 → 低钾补不上**；TdP 第一想低 K/Mg；地高辛中毒被低 K 放大 |
| **治疗** | 经口优先（KCl 40-80 mEq/d 分次）；重度/有心律失常用 IV KCl ≤10-20 mEq/h 中心静脉；**同时补 Mg** |

> [!warning] 低镁不补钾补不进
> 低 Mg 激活 ROMK → 持续漏 K → 补再多 KCl 无效。**所有低 K 都查 Mg**（电解质题最高频陷阱之一）。

### 1.3 高低钾镜像

| 维度 | 高钾 | 低钾 |
|---|---|---|
| 心脏死法 | 传导阻滞 → asystole | TdP / VF |
| ECG buzzword | sine wave / peaked T | U 波 / 长 QT |
| 急救 | Calcium gluconate | IV K + Mg |
| 与地高辛 | 急性中毒**结果** | 中毒**诱因** ← Q4454 |

## §二 钠 (Sodium)

### 2.1 低钠（Na+ <135）

> [!tip] 算法核心 — "三步定型"
> ① 渗透压（Posm）：低渗 vs 等/高渗（假性、高血糖）
> ② 容量状态：低容量 / 等容量 / 高容量
> ③ 尿钠 / 尿渗透压

| 类型 | 病因 | UNa | 处理 |
|---|---|---|---|
| **低容量低钠** | 利尿剂、呕吐/腹泻、肾上腺功能不全、烧伤、CSW（脑耗盐）| 利尿剂 / 肾上腺/CSW: **UNa >20**；GI/烧伤: UNa <10 | **IV NS** 扩容 |
| **等容量低钠** | **SIADH** ⭐、甲减、glucocorticoid↓、psychogenic polydipsia、reset osmostat | SIADH: **UNa >40, Uosm >100** | **限水**；重症 hypertonic saline |
| **高容量低钠** | HF、肝硬化、肾病综合征、CKD | UNa <10（CKD 可↑）| 限水 + 利尿（HF）+ 治原发 |

> [!danger] 补 Na 速度铁律
> **Na 上升 ≤8-10 mEq/L/24h**（重症 ≤12 mEq/L/24h）。过快 → **Osmotic Demyelination Syndrome (ODS / 中央桥脑髓鞘溶解)**，"locked-in" 永久残疾。
> 慢性低钠（>48h）尤其要慢；急性低钠（<48h）可稍快。

**SIADH 钥匙**：低钠 + 高 Uosm + 高 UNa + 等容量 + 无肾上腺/甲状腺病 → 找 SCLC / CNS / 药物（SSRI、carbamazepine、cyclophosphamide）

> [!info] 血渗 + 尿渗 怎么读：两步法 ⭐（来源 [[mistakes/uworld-mistakes#^Q2180]]）
> 低钠题给的 **血渗(Posm)** 和 **尿渗(Uosm)** 各管一步，别混着看：
>
> **第一步 — 血渗：给低钠"验明正身"（是不是真低渗）**
>
> | 血渗 (mOsm/kg) | 类型 | 含义 |
> |---|---|---|
> | **<275（低渗）** | 真性低渗低钠 ✓ | 往下走容量轴 / SIADH |
> | 275–295（等渗） | 假性低钠 | 高血脂 / 高蛋白（pseudohyponatremia）|
> | >295（高渗） | 高渗性低钠 | 高血糖 / 甘露醇把水拽出稀释 Na |
>
> Q2180 血渗 260 < 275 → 真低渗低钠（SIADH 的**前提门槛**，SIADH 定义就含低血渗）。
>
> **第二步 — 尿渗：看 ADH 是否"不恰当地开闸"**
>
> 血都稀了（Posm↓），正常反应是**关掉 ADH、把尿稀释到 Uosm <100** 拼命排水。但若 **Uosm >100** = 肾还在浓缩留水 = **ADH 不恰当分泌** = SIADH 核心证据。
>
> | 尿渗 (mOsm/kg) | 含义 |
> |---|---|
> | **<100（稀释尿）** | ADH 正常关闭 → 原发性烦渴 / beer potomania（低溶质）|
> | **>100（浓缩不当）** | ADH 在活动 → SIADH / 低容量 / 高容量都可能，再看容量轴 |
>
> Q2180 尿渗 500 >100 → ADH 在留水（与低血渗"打架"）。
>
> > [!danger] 一句话锁定
> > **血渗低 = 确认水真多了；尿渗却高 = 肾却还在留水 → 两者方向打架 = ADH 乱来 = SIADH。**
> > 关键不是尿渗绝对值，是它**和血渗方向相反**（血稀尿浓）这个"不匹配"。
> > 但 Uosm>100 **只说明 ADH 在活动**，分不出 SIADH/低容量/高容量 —— 三者都有 ADH —— **真正切开三者靠下面的尿 Na + 尿酸 + 容量轴**。

> [!tip] SIADH vs 低容量低钠：低尿酸 + 尿钠 是分水岭 ⭐（来源 [[mistakes/uworld-mistakes#^Q2180]]）
> 呕吐/口干等会诱你选"低容量低钠"，但二者关键不同：
> - **SIADH（等容量）**：**尿 Na > 40 + 血尿酸低（uric acid ↓，稀释 + 尿排↑）** + 黏膜湿润 + 无体位性低血压。
> - **低容量低钠**：尿 Na < 20（肾保钠）+ 血尿酸**高**（血液浓缩）+ 黏膜干 + 体位性↓。
> - **诱因**：SSRI + 老年 = 经典 SIADH；**恶心本身是强 ADH 刺激**（会加重 SIADH）。

#### 2.1.1 HypoNa 治疗双轴矩阵 ⭐⭐⭐（来源 [[mistakes/uworld-mistakes#^Q114863]] Thiazide hypoNa seizure）

> [!danger] 反 pattern — "HypoNa 不要反射 NS"
> HypoNa 治疗看**两个轴**：① 容量状态 ② 严重度（有无 seizure / 重症状）。**不是所有 HypoNa 都给 NS！**

| 容量 | 无症状 / 轻 | **Seizure / 严重症状** ⭐ |
|---|---|---|
| **Hypovolemic**（GI 丢失 / Addison / 烧伤 / CSW） | **0.9% NS 扩容** ✅ | **3% 高张盐水** + 后续 NS |
| **Euvolemic**（SIADH / Thiazide / 低甲 / 精神多饮） | **限水**（不 NS！） | **3% 高张盐水**（不是 NS！） |
| **Hypervolemic**（HF / 肝硬化 / 肾综 / CKD） | **限水 + 袢利尿剂**（不 NS！） | **3% 高张盐水** + 利尿 |

> [!warning] 给 NS 会反向加重的 2 种 HypoNa ⚠️
> 1. **SIADH 给 NS → Na 更低**（机制：ADH 让 free water retained，NS 给的水留在体内，钠却排出 → 越补越低）
> 2. **HF / 肝硬化给 NS → 加重水肿 + 加重 HypoNa**（总水 > 总钠 增加）
>
> 这两种正确做法是**限水**（轻）或 **3% 高张盐水**（重症）。

> [!tip] 高容量低钠（肝硬化）"轻症 = 不需特殊治疗" + 选药 ⭐（来源 [[mistakes/uworld-mistakes#^Q106381]]）
> - **Na > 120 且无症状** → **低钠成分本身不需特殊 Rx**，重点是 **减 ADH 释放**（减低血压 / 停降压药）+ 治原发；治疗门槛 = **Na < 120 或症状性（seizure）** 才上 3% 高张盐水。
> - **避免噻嗪**（噻嗪是加重低钠的经典药：强降压 → ADH↑ + 远端稀释能力↓）；高容量利尿用 **螺内酯 + 袢利尿**。
> - **避免 NS / 等张盐水**（肝硬化除非容量不足如呕吐腹泻）；**盐片**用于 euvolemic（SIADH），高容量限钠。
> - 反 pattern：别"想利尿既治水肿又治低钠"就反射开噻嗪 —— 既加重低钠，且轻症本不需治。

#### 2.1.2 Thiazide-Induced HypoNa 指纹（来源 [[mistakes/uworld-mistakes#^Q114863]]）

| 要素 | 内容 |
|---|---|
| **高危人群** | **Age >75 + 女 + low BMI**（瘦小老太太经典） |
| **时间窗** | **3-4 周内**起病 / 急性事件（diarrhea）后 |
| **容量状态** | 多 **euvolemic**（moist mucosa） — 不是 hypovolemic |
| **双机制** | ① **ADH-dep**（mild volume depletion → ADH↑）+ ② **ADH-indep**（thiazide 直接↓远端稀释能力 + 老年 GFR↓ + dilution↓）|
| **治疗** | **永久停 thiazide** + IV NS（mild）/ **3% 高张盐水**（severe seizure） |

#### 2.1.3 电解质致 Seizure 横向 5 鉴别 ⭐⭐（来源 [[mistakes/uworld-mistakes#^Q114863]]）

| 电解质 | 致 seizure？ | 机制 / 钩子 |
|---|---|---|
| **HypoNa** ⭐⭐ | ✅✅ **最常见** | 脑细胞水肿；老女 + thiazide / SIADH pattern |
| **HypoCa**（重度） | ✅ | 神经兴奋性 **↑** → tetany / Chvostek-Trousseau；多有诱因（CKD / 甲旁切术后 / VitD 缺） |
| **HypoMg** | ✅ | 多伴 HypoCa / HypoK；TdP / 抽搐 |
| **HypoK** | ❌ 不典型 | 多 muscle weakness / paralysis / 心律失常 |
| **HyperCa** | ❌ **反向** | 神经兴奋性 **↓** → 迟钝 / 便秘 / weakness（"stones-bones-groans-psychic-moans"） |
| **HyperNa** | ⚠️ 罕 | critical illness + 渴感受损 + bedridden + tube feed；急性下降时致脑细胞水肿 |

> [!tip] 反射记忆
> **致 seizure 强弱排序**：HypoNa >> HypoCa（重度）> HypoMg > HypoGlc > HyperNa（罕）。**HyperCa 不致 seizure 是反 pattern**（与 HypoCa 相反）。

### 2.2 高钠（Na+ >145）⭐（来源 [[mistakes/uworld-mistakes#^Q4853]] — 婴儿 hypovolemic-symptomatic 先 isotonic 错点）

| 病因 | 钥匙 |
|---|---|
| 单纯丢水（无渴感/不能喝）| 老人、卧床、**婴儿无法主诉口渴** ⭐ |
| **GI 丢失** | gastroenteritis（婴儿头号）/ 渗透性腹泻 / 呕吐 |
| **不显性丢失↑** | 发热 / 高温 / 烧伤 / 早产儿（体表面积大）|
| **Diabetes insipidus** | 中枢（垂体后叶受损）vs 肾性（Li、高Ca、CKD）；多尿稀尿（Uosm 低）|
| 渗透性利尿 | 高血糖、甘露醇 |
| 输入高张液 | iatrogenic |

- **临床**：渴、躁动、嗜睡 → 昏迷、抽搐；**脑萎缩出血**（脑收缩拉血管）
- **儿童脱水 5 体征**（symptomatic 判定）：HR↑ / BP↓ / **干黏膜** / **cap refill > 3s** / 嗜睡 / DTR↓ / 张力↓ / 尿少（**单尿布/天**）/ 凹陷囟门 / 哭无泪

#### 2.2.1 复苏决策树 ⭐⭐⭐（Q4853 学到 — 反 "hypernatremia 必给 hypotonic" 反射）

![[{53105C27-3D2C-479C-9D32-9F475D3DE01B}.png]]
*↑ UWorld 官方 Hypernatremia 复苏决策树*

| 状态 | 复苏液 | 顺序 |
|---|---|---|
| **Euvolemic** | **Hypotonic**（5% dextrose 或 0.45% saline）| 直接缓慢纠 Na |
| **Hypovolemic + Asymptomatic** | Hypotonic | 直接缓慢纠 Na |
| **Hypovolemic + Symptomatic** ⭐ | **Step 1: 0.9% saline isotonic bolus** 20 mL/kg → **Step 2: hypotonic 缓慢纠 Na** | **先救命再纠 Na** |

#### 2.2.2 Na 校正速度（铁律）⭐
- **< 0.5 mEq/L/hr** 或 **24hr 内 < 10-12 mEq/L**
- 过快降 Na → 水进脑细胞 → **cerebral edema / 抽搐 / 永久神经损伤 / 死亡**

- **DI 治疗**：desmopressin（中枢）或低盐 + 噻嗪 + amiloride（肾性 lithium DI 专属神药，paradox 利尿）— 见 [[mistakes/uworld-mistakes#^Q4425]]
- **水缺量公式**：`Water deficit = TBW × (current Na/140 − 1)`

> [!warning] 反 pattern 警示（高钠 4 大）⭐
> 1. **"Hypernatremia 给 hypotonic" 反射在 symptomatic 时错** — symptomatic 必须先 isotonic 救命
> 2. **5% dextrose = free water**（代谢后），算 hypotonic — 不算"补容量"
> 3. **NaHCO3 是 hypertonic** — 高钠时禁用（即使有严重代酸 — hypovolemic 致 lactic acidosis 自愈不必上 bicarb）
> 4. **HCO3↓ 严重代酸 ≠ 必上 NaHCO3** — 灌注差致 lactate↑，恢复 perfusion 自愈

## §三 钙 (Calcium)

### 3.1 高钙（Ca²⁺ corrected >10.5 / ion Ca >1.3）⭐（来源 [[mistakes/uworld-mistakes#^Q2169]] — 急症先治不诊断错点）

> [!tip] 助记 "Stones, Bones, Groans, Thrones, Psychiatric overtones"
> 肾结石 / 骨痛 / 腹痛-便秘-胰腺炎 / 多尿 / 抑郁-精神错乱

#### 3.1.1 病因（CHIMPANZEES 12 因 + 简化 6 大）

| 病因 | 钥匙 |
|---|---|
| **原发性甲旁亢** ⭐（最常见门诊原因）| PTH↑ + 高 Ca + 低 P |
| **恶性肿瘤** ⭐⭐（最常见住院原因）| 3 机制：**PTHrP**（鳞癌 lung/头颈/食管/肾/乳腺）/ **溶骨**（**MM** ⭐ / 乳腺 / 前列腺）/ **1,25-VitD↑**（**淋巴瘤** / 肉芽肿）|
| 肉芽肿病 | 结节病、TB → 1,25-VitD↑ |
| 维生素 D / A 中毒 | 摄入史 |
| 噻嗪利尿剂 / Lithium | 减少 Ca 排泄 |
| Milk-alkali、固定不动、家族性低尿钙性高钙（FHH）| 各自钥匙 |

> [!warning] 老年 + 不明原因高钙 + pancytopenia / 骨痛 / Cr↑ → **MM "CRAB" 必想**
> **C**a↑ / **R**enal failure / **A**nemia / **B**one lesion + M-spike — 但 **MM 诊断（bone marrow biopsy）等急症治后再做**

- **ECG**：**短 QT**（Q19929 反向参照）

#### 3.1.2 3 级分层管理 ⭐（UWorld 高频图）

| 严重度 | Ca 阈值 | Short-term | Long-term |
|---|---|---|---|
| **Severe 或 Symptomatic** ⭐ | **Ca > 14 mg/dL** 或任何症状（confusion/呕吐/痛/严重多尿）| **NS hydration + Calcitonin**（避免 loop diuretic 除非 volume overload）| **Bisphosphonate** (zoledronic acid) |
| **Moderate** | Ca 12-14 mg/dL | Usually 无 immediate（除非 symptomatic）| 同 severe 框架 |
| **Asymptomatic / Mild** | Ca < 12 mg/dL | 无 immediate | **避免 thiazide / lithium / 容量耗竭 / 长卧床** |

#### 3.1.3 治疗 6 选对照 ⭐⭐⭐

| 治疗 | 机制 | 起效 | 何时用 |
|---|---|---|---|
| **Normal saline** ⭐ Step 1 | 容量复苏 → GFR↑ → Ca 尿排↑ | 数小时 | **所有 severe / symptomatic** — first step |
| **Calcitonin** ⭐ Step 2 | 抑 osteoclast 骨吸收 | **24-48 hr 快**（但 tachyphylaxis 耐药 → 仅短期）| Severe，与 NS 同时 |
| **Bisphosphonate**（zoledronic 4 mg / pamidronate 60-90 mg IV）⭐ long-term | 抑 osteoclast（持久）| **2-4 天慢**（所以放后）| 所有 severe，长期维持 |
| **Glucocorticoid** | 抑 1,25-VitD 形成 + 抑淋巴细胞 | 2-5 天（慢）| 仅 **Vit D 过量 / 肉芽肿 (sarcoid/TB) / 淋巴瘤** |
| **Hemodialysis** | 直接除 Ca | 即刻 | 仅 **肾衰 / HF 不能耐受 saline** |
| **Denosumab** | RANKL 抑制 | 数天 | 难治 / bisphosphonate 失败后备 |
| **Loop diuretic (furosemide)** | 减升支 Ca 重吸收 | 数小时 | **仅 volume overload (HF)**，不 routine ⚠️ |

> [!success] 急救顺序铁律 ⭐⭐⭐（Q2169 学到）
> **Severe / Symptomatic Hypercalcemia = 急症，先治不诊断**
>
> **Step 1: NS 200-300 mL/hr 立刻** → **Step 2: Calcitonin 4 IU/kg q12h** 同时上 → **Step 3: Bisphosphonate**（zoledronic）维持
>
> ❌ 不先 bone marrow biopsy 等诊断病因（即使强烈怀疑 MM）— 急症先稳定，**确诊延后**
> ❌ 不 routine furosemide（仅 volume overload）
> ❌ 不 routine glucocorticoid（仅 Vit D / 肉芽肿 / 淋巴瘤）
> ❌ 不 routine hemodialysis（仅肾衰 / HF 不能 saline）

> [!warning] 反 pattern 警示（高钙 6 大）
> 1. **急症先治不诊断** — bone marrow biopsy / 影像延后做
> 2. **Thiazide ❌** — 减 Ca 排泄加重高钙
> 3. **Furosemide ≠ routine** — 仅 volume overload (HF)；routine 用反致 AKI
> 4. **Bisphosphonate 起效慢 2-4 天** — 不是急救，是 long-term
> 5. **Insulin 是 hyperK 不是 hyperCa** — USMLE 干扰项
> 6. **Glucocorticoid 病因特异** — 只 Vit D / 肉芽肿 / 淋巴瘤，不是 first-line

#### 3.1.4 NS 在 HyperCa 是核心治疗机制（不是"扩容到等容再处理"）⭐

> [!info] 与 HypoNa 的对比澄清
> 用户常误解"严重 HyperCa 和严重 HypoNa 都是先 NS 扩容到等容再处理" — **错**。两者机制完全不同：

| 紊乱 | NS 是核心？ | 机制 |
|---|---|---|
| **Severe HyperCa** | ✅ **核心治疗** | HyperCa → 多尿 → **脱水**；NS 同时 ① 纠脱水 ② 扩容→GFR↑→**Ca 尿排↑**；扩容本身就在治高钙（不是"先扩容才能处理"） |
| **Severe HypoNa** | ❌ 不一定 | 看容量轴（hypovolemic ✅ / euvolemic ❌ / hypervolemic ❌）+ 严重度轴（seizure → 3% 高张盐水，不是 NS）|

#### 3.1.5 AKI + HyperCa → MM 直到证否（关键反 pattern）⭐⭐（来源 [[mistakes/uworld-mistakes#^Q16044]]）

> [!danger] 反 pattern — AKI 通常致 HypoCa
> 一般 AKI → ① Pi 排泄↓ → ↑ blood Pi → CaPi 复合物 → 游离 Ca↓；② 1,25-VitD 活化↓ → **HypoCa**。
> **若 AKI + HyperCa → 反向** → 必有 underlying malignancy driving Ca↑ → **MM 直到证否**。

**MM 双机制致 AKI + HyperCa**：
- ① **Osteoclast activating factors** → 溶骨 → **HyperCa**
- ② **Light chain cast nephropathy** → tubule 堵塞 → **intrinsic AKI**

**MM CRAB 诊断**：**C**alcium↑ / **R**enal failure / **A**nemia / **B**one lytic + M-spike

> [!warning] Bence-Jones 尿蛋白 dipstick 假阴性 ⚠️
> dipstick 只测 **albumin** 不测 light chain → 大量 Bence-Jones 蛋白尿可能 dipstick **阴性** → 必做 24h 尿蛋白 + **urine protein electrophoresis** 才能识别。

### 3.2 低钙（Ca²⁺ corrected <8.5 / ion <1.1）

| 病因 | 钥匙 |
|---|---|
| 甲旁减（术后、自身免疫、DiGeorge）| PTH↓ |
| 维生素 D 缺乏 / 吸收不良 | 低 25-OH-VitD |
| **CKD** | 高 P + 低 1,25-VitD |
| 急性胰腺炎 | 钙皂化 → 突发低钙 |
| 输大量枸橼酸血 / 大量输液 | 螯合 |
| 低镁（PTH 释放抑制）| 必查 Mg |
| Rhabdomyolysis | Ca 沉积坏死肌肉 |
| 假性低钙 | 低白蛋白；用 **corrected Ca = 测得 Ca + 0.8 × (4 − albumin)**，或测 ion Ca |

- **临床**：**Chvostek 征**（面神经叩击 → 嘴角抽）/ **Trousseau 征**（袖带充气 3min → 腕痉挛）/ paresthesia / 喉痉挛 / 抽搐
- **ECG**：**长 QT** → TdP
- **治疗**：症状性或抽搐 → **IV calcium gluconate**（**不**用 Ca chloride 外周，外渗坏死）；慢性 → 口服 Ca + Vit D；CKD → 控 P + activated D（calcitriol）

## §四 镁 (Magnesium)

### 4.1 低镁（Mg <1.5）

| 病因 | 钥匙 |
|---|---|
| 利尿剂、酒精滥用、慢性腹泻、PPI 长期、aminoglycosides、cisplatin、refeeding | 多药 / 慢性病 |

- **临床**：**伴随低 K / 低 Ca**；震颤、抽搐、心律失常
- **ECG**：长 QT → **TdP**
- **治疗**：**IV MgSO4 2-4 g**（TdP 一线）；口服补充慢性病例

> [!danger] TdP 第一反应
> **多形性 VT (TdP) = IV Mg 第一线**（无论 Mg 值高低）。

### 4.2 高镁（Mg >2.5）

| 病因 | 钥匙 |
|---|---|
| 肾衰 + 含 Mg 制剂（抗酸药、泻药）| 老人多见 |
| **产科 — eclampsia / preeclampsia 治疗** | 监测过量 |
| 肿瘤溶解 | 罕见 |

- **临床进阶**：DTR↓（首发） → 嗜睡 → 呼吸抑制 → **bradycardia / 心搏停**
- **ECG**：PR / QRS / QT 延长
- **治疗**：**IV calcium gluconate**（拮抗）+ 利尿 / 透析；停 Mg 来源

> [!info] 产科 Mg 中毒监测
> Eclampsia 用 MgSO4 → 必查 **DTR**（首先消失）+ **呼吸频率** + 尿量。DTR 消失 → 停 Mg + 给 Ca。

## §五 磷 (Phosphate)

### 5.1 高磷（P >4.5）

- **病因**：**CKD**（最常见）/ TLS / 横纹肌溶解 / VitD 中毒
- **后果**：**继发性甲旁亢**、转移性钙化、**低钙**
- **治疗**：限磷饮食 + **磷结合剂**（sevelamer、Ca acetate）+ 透析

#### 5.1.1 CKD-MBD 治疗 5 阶梯 ⭐⭐（来源 [[mistakes/uworld-mistakes#^Q106299]]）

> [!info] 机制链
> CKD ↓GFR → ① ↓ 1,25-VitD 活化 → ↓肠 Ca 吸收 → **↓血 Ca** + ② ↓磷排泄 → **↑血 Pi** → CaPi 复合物→进一步↓游离 Ca → **↑PTH（2° HPT）** → 持续 → **Osteitis fibrosa cystica**（高骨转换 + 脱矿 + 纤维化 + 骨折↑）。

| 阶梯 | 治疗 | 备注 |
|---|---|---|
| ① | **低 Pi 饮食** | 首选，最便宜 |
| ② | **Pi 结合剂** | **Ca-based**（CaCO3 / Ca-acetate）便宜但 hyperCa 风险 ↔ **Non-Ca**（Sevelamer / Lanthanum）贵但安全 |
| ③ | **Active Vit D** | calcitriol / paricalcitol；**Pi 正常后再给**（先给会 ↑ CaPi 沉积） |
| ④ | **Calcimimetics** | cinacalcet — 增 CaSR 敏感性 → ↓PTH |
| ⑤ | **Parathyroidectomy** | 难治 3° HPT（autonomy → hyperCa）|

#### 5.1.2 Renal Osteodystrophy 4 型（U 型治疗曲线）

| 类型 | PTH | 骨转换 | 病机 |
|---|---|---|---|
| **Osteitis fibrosa cystica** ⭐ | **↑↑** | **高** | 治疗不足 → 持续 2° HPT → **subperiosteal resorption + bone cysts + brown tumors** |
| **Adynamic bone disease** | **↓** | **低** | **过度治疗**（Pi binder + VitD + Calcimimetic 过度）→ PTH 被压太低 |
| Osteomalacia | 不定 | 低矿化 | VitD 缺乏 + Al 蓄积（旧透析液） |
| Mixed | 中 | 混合 | 上述混合 |

> [!warning] U 型治疗曲线
> 不足 → osteitis fibrosa cystica；过度 → adynamic bone disease；**两端都 ↑ 骨折风险**。

> [!tip] Subperiosteal Resorption = PTH ↑ pathognomonic
> 部位：桡骨指骨（最特异）/ 锁骨外端 / 颅骨 / 髂骨。**任何 1°/2°/3° HPT 都可见**。

#### 5.1.3 1°/2°/3° HPT 鉴别

| 类型 | Ca | Pi | PTH | 原因 |
|---|---|---|---|---|
| **1° HPT** | ↑ | ↓ | ↑ | 原发腺瘤 / 增生 |
| **2° HPT** | **↓**（HPT 在代偿） | ↑ | ↑ | CKD / VitD 缺乏 |
| **3° HPT** | ↑（autonomy） | 不定 | ↑↑ | 多见**移植后**或长期 2° 进展 → 腺自主分泌 |

### 5.2 低磷（P <2.5）

- **病因**：**Refeeding**（饥饿后再喂养）⭐ / DKA 治疗中 / 酒精戒断 / 过度通气（呼吸性碱中毒）/ 维生素 D 缺乏 / 原发甲旁亢
- **临床**：肌无力（含**呼吸肌**）、横纹肌溶解、心衰、溶血、感觉异常
- **治疗**：口服磷酸盐；IV 仅在 P <1.0 + 症状（IV 太快 → 低 Ca + 转移性钙化）

> [!warning] Refeeding 综合征
> 长期饥饿 → 再喂养 → insulin↑ → 细胞内摄 P / K / Mg → **三低**（低 P / 低 K / 低 Mg）+ 心衰、横纹肌溶解。
> **高危**：AN（神经性厌食）、酒精滥用、长期住院禁食。**预防**：缓慢起始热量（10-20 kcal/kg/d）+ 提前补 P/K/Mg + 监测。

## §六 酸碱失衡（速查框架）

### 6.1 四步法

```
Step 1: 看 pH      → <7.35 acidemia / >7.45 alkalemia
Step 2: 看 PCO2 + HCO3
   PCO2↑ + HCO3↑ → 呼吸性酸（HCO3 代偿）
   PCO2↓ + HCO3↓ → 呼吸性碱（HCO3 代偿）
   HCO3↓ + PCO2↓ → 代谢性酸（PCO2 代偿）
   HCO3↑ + PCO2↑ → 代谢性碱（PCO2 代偿）
Step 3: 代谢性酸 → 算 Anion Gap = Na − (Cl + HCO3) (正常 8-12)
   AG↑    → MUDPILES
   AG 正常 → HARDASS / GI 丢 / RTA
Step 4: AG↑酸 → 算 delta-delta = ΔAG / ΔHCO3
   ~1     → 单纯 AG↑酸
   <1     → 合并非 AG 酸（HCO3 丢更多）
   >2     → 合并代谢性碱（HCO3 高于预期）
```

### 6.2 病因助记

| 失衡 | 助记 |
|---|---|
| **AG↑代酸** | **MUDPILES**：Methanol / Uremia / DKA / Propylene glycol / Iron-INH / Lactic acidosis / Ethylene glycol / Salicylates |
| **AG 正常代酸（HARDASS）** | Hyperalimentation / Addison's / RTA / Diarrhea / Acetazolamide / Spironolactone / Saline infusion |
| **代谢性碱** | 呕吐、利尿剂、Conn、Cushing、Bartter/Gitelman、低 K |
| **呼吸性酸** | 肺通气不足（COPD 急加重、神经肌肉病、阿片中毒）|
| **呼吸性碱** | 过度通气（焦虑、肺栓塞、脓毒症早期、salicylate 早期、高 alt）|

> [!tip] 渗透压间隙 → 中毒性醇
> AG↑代酸 + **渗透压间隙↑（>10）** = **甲醇 / 乙二醇中毒**（参 [[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全]]）。

> [!tip] 代谢性碱中毒：saline-responsive vs resistant（用尿 Cl 分）⭐（来源 [[mistakes/uworld-mistakes#^Q4866]]）
> 代碱由"丢 H⁺/Cl"**启动**、由 **Cl 耗竭/容量收缩 perpetuate**（低 Cl 损害肾排 HCO3）。分两类用**尿 Cl**（比尿 Na 可靠，因 RAAS 会让尿 Na 失真）：
>
> | 类型 | **尿 Cl** | 病因 | 治疗 |
> |---|---|---|---|
> | **Saline-responsive**（容量↓）| **< 20**（如 9）| **呕吐 / NG 吸引 / 既往利尿剂 / 收缩性碱** | **0.9% NS**（补容量 + Cl → 肾恢复排 HCO3）+ 补 K |
> | **Saline-resistant**（容量↑/正常）| **> 20** | **Conn / Cushing / Bartter-Gitelman / 当前利尿剂** | 治原发（Conn → 螺内酯；容量过载利尿致碱 → acetazolamide）|
>
> - **指纹**：HCO3↑ + **低 K + 低 Cl** + 尿 Cl 低 = 呕吐型 saline-responsive 代碱（bulimia / 幽门梗阻）。
> - **干扰项**：acetazolamide / ammonium chloride 用于**容量过载**无法给 NS 者；容量**不足**者首选 **NS**。Conn / Bartter 会有**高血压或尿 Cl 高**，可区分。

### 6.3 代偿快速估算

| 原发 | 代偿公式 | 用法 |
|---|---|---|
| 代谢性酸 | **Winter's**：PCO2 ≈ 1.5 × HCO3 + 8 ± 2 | 测得 PCO2 偏离 → 合并呼吸障碍 |
| 代谢性碱 | PCO2 ≈ 0.7 × HCO3 + 20 ± 5 | |
| 急性呼吸性酸 | ΔHCO3 = 0.1 × ΔPCO2 | |
| 慢性呼吸性酸 | ΔHCO3 = 0.35 × ΔPCO2 | |
| 急性呼吸性碱 | ΔHCO3 = −0.2 × ΔPCO2 | |
| 慢性呼吸性碱 | ΔHCO3 = −0.4 × ΔPCO2 | |

### 6.4 NaHCO3 适应症详解（窄）⭐⭐（用户高频概念点 + Q2790 触发）

> [!danger] 反 pattern — "严重代酸 HCO3 9 不直接上 NaHCO3" 的真实原因
> **不是"高钠不能用低钠就能用"** — 是 NaHCO3 在 lactic acidosis 中**机制本身无 outcome benefit + 多副作用**。钠负荷是次要原因。

#### 6.4.1 NaHCO3 在 lactic acidosis 不 routine 用的 5 个机制原因

| 原因 | 机制 |
|---|---|
| **无 outcome benefit** | RCT 证据 NaHCO3 不改善 lactic acidosis 死亡率；**恢复 perfusion → lactate 自动清除**（lactate → HCO3 内源生成）|
| **CO2 生成** | HCO3 + H+ → H2O + **CO2**；CO2 自由扩散入细胞 → 加重**细胞内**酸中毒 |
| **氧解离曲线左移** | 碱化 → Hb 与 O2 亲和力↑ → **组织氧供↓** |
| **离子 Ca↓** | Alkalemia → albumin 结合 Ca↑ → **心肌收缩↓** |
| **反弹 alkalosis** | lactate 自动清除（→ HCO3）→ 多给的 NaHCO3 致碱中毒 |
| **钠 / 容量负荷**（次要）| 1 amp = 50 mEq Na = **hypertonic**；HF / hyperNa 时加重 |

#### 6.4.2 NaHCO3 真正适应症（窄）⭐

| 适应症 | 机制 |
|---|---|
| **Non-AG 代酸**（RTA / chronic CKD acidosis / 严重腹泻） | 无内源 HCO3 生成机制（不像 lactic / DKA 可自愈）|
| **Toxic alcohol**（methanol / ethylene glycol） | 碱化尿液 + 抗 cellular toxicity |
| **Salicylate poisoning** | **碱化尿液加排**（trap ionized salicylate 在尿中）|
| **TCA overdose** | 稳膜（拮抗 Na+ channel block） |
| **Hyperkalemia** | **Shift K 入细胞**（"Shift" 阶段一员）|
| **极端 pH <7.1-7.15 + 失代偿** + 无法纠正 underlying cause | 争议大；guideline 弱推荐 |

> [!question]- "低钠 + 严重代酸"能用 NaHCO3 吗？
> **取决于代酸原因**，不取决于钠：
> - 低钠 + **Lactic acidosis（sepsis）**→ 仍不 routine 用（机制问题）
> - 低钠 + **RTA 1 型 / 严重腹泻** → ✅ 可以用（适应症本身就匹配）
> - 低钠 + **DKA** → 不 routine 用（insulin 治酮体 → 自产 HCO3 自愈）
>
> **钠考量是次要权衡**，主导是适应症本身。

### 6.5 Mixed Acid-Base 5 大典型组合 ⭐⭐（来源 [[mistakes/uworld-mistakes#^Q2790]]）

> [!info] 识别钥匙
> ① pH **极端** (<7.2 / >7.6) 或**接近正常**但 HCO3/PaCO2 显著异常 → 怀疑 mixed。
> ② 代偿值用公式（Winters 等）算 expected，**偏离即第二原发**。

| 组合 | HCO3 | PaCO2 | pH | 经典 Stem |
|---|---|---|---|---|
| **Met acidosis + Resp acidosis** ⭐ | ↓ | ↑ | **Very low** | **Sepsis 末期 + lactic acidosis + 呼吸肌疲劳**（Q2790）/ cardiac arrest |
| Met alkalosis + Resp alkalosis | ↑ | ↓ | Very high | Hyperemesis gravidarum + hyperventilation of pregnancy |
| **Met acidosis + Resp alkalosis** | ↓ | ↓ | Near normal | **Salicylate toxicity**（经典）/ sepsis 早期 |
| Met alkalosis + Resp acidosis | ↑ | ↑ | Near normal | Overdiuresed HF + 基础 COPD |
| Met acidosis + Met alkalosis | ↔ | ↔ | Near normal | **DKA + 重度呕吐** |

### 6.6 Sepsis 三阶段酸碱演化 ⭐

| 阶段 | 主导紊乱 | 机制 |
|---|---|---|
| **早期** | **Resp alkalosis** | Tachypnea / hyperventilation 应激 → PaCO2↓ |
| **中期** | **Mixed met acid + resp alk**（Salicylate-like） | Tissue hypoperfusion → lactic acidosis；hyperventilation 仍在 |
| **晚期** ⭐ | **Mixed met + resp acidosis**（最危险，pH 崩塌） | Respiratory muscle fatigue → 不能 hyperventilate → PaCO2 反↑ |

> [!warning] 临床信号 — "Fast and shallow breathing" = respiratory muscle fatigue
> 是即将呼衰 / **intubation indication** 信号。Stem 看到要警惕 mixed met + resp acidosis。

### 6.7 代偿矛盾 = Mixed 钥匙

**单一紊乱代偿规则**：HCO3 与 PaCO2 应**同向移动**（met acid → 都↓ / met alk → 都↑ / resp acid → 都↑ / resp alk → 都↓）。**反向**或**不充分** = mixed。

| 实测组合 | 单一紊乱期望 | 偏离表现 |
|---|---|---|
| HCO3 ↓ + PaCO2 ↑ | met acid 期望 PaCO2 ↓代偿 | **反向** → mixed met+resp acid |
| HCO3 ↑ + PaCO2 ↓ | met alk 期望 PaCO2 ↑代偿 | **反向** → mixed met alk+resp alk |
| HCO3 ↑ + PaCO2 ↑（HCO3↑较代偿期望值低）| acute resp acid 期望 HCO3 ↑ 0.1×ΔPaCO2 | **不够** → 同时 met acid |

## §七 高频跨电解质陷阱

| 陷阱 | 真相 |
|---|---|
| 低 K 补不上 | **同时查 Mg** — 低 Mg 是低 K 不响应的最常见原因 |
| 低 Ca 用 Ca chloride 外周静脉 | ❌ 用 **Ca gluconate**；Ca chloride 仅中心静脉（外渗坏死）|
| TdP 第一线 | **IV Mg**（不论 Mg 水平）|
| 高钙先用 thiazide？| ❌ thiazide 减少 Ca 排泄、加重高钙；**袢利尿剂也不是 routine**（仅 volume overload/HF 时考虑，routine 用反致 AKI）；先 **NS hydration + Calcitonin** |
| 高钙急症 = 先做 bone marrow biopsy 查病因？| ❌ **急症先治不诊断** — Ca>14 或 symptomatic 立刻 NS+Calcitonin，MM 等诊断稳定后再做（[[mistakes/uworld-mistakes#^Q2169]]）|
| DKA "高钾" 不补 K | ❌ 总钾耗竭；胰岛素一上必须监测，**K <5.3 即开始补 K** |
| 慢补 Na 慢到什么程度 | ≤8-10 mEq/L/24h；急性可稍快 |
| Refeeding 综合征预防 | 慢起始热量 + 提前补 P/K/Mg + 监测 |
| 高钾 sine wave 当 brady 起搏 | ❌ 起搏 capture failure；先 Ca gluconate 稳膜 |
| **AKI + HyperCa → 反射 ATN / 脱水**？ | ❌ AKI 通常 **HypoCa**；**AKI + HyperCa = MM 直到证否**（cast nephropathy + 溶骨）|
| **HyperCa 和 HypoNa 都先 NS**？ | ❌ HyperCa NS 是核心治疗（脱水 + 增 Ca 尿排）；HypoNa 看容量 + 严重度（SIADH 给 NS 反加重；severe seizure 用 3% 高张盐水不是 NS）|
| **严重代酸 HCO3 9 必上 NaHCO3**？ | ❌ Lactic acidosis 不 routine 用（无 outcome benefit + CO2 反加重细胞内酸 + 氧解离左移 + Ca↓ + 反弹碱）；适应症窄（RTA / toxic alcohol / salicylate / TCA / hyperK / 极端 pH<7.1 失代偿）|
| **代偿同向 vs 反向** | 单一紊乱 HCO3 与 PaCO2 应**同向**（met acid 都↓ / resp acid 都↑）；**反向 = mixed** |
| **Bence-Jones dipstick 阴性 → 排除 MM**？ | ❌ dipstick 只测 albumin 不测 light chain；必做 **urine protein electrophoresis** |
| **Subperiosteal resorption 只见于 1° HPT**？ | ❌ **任何 1°/2°/3° HPT 都可见**（PTH ↑ pathognomonic，不是 1° 专属） |

## §八 易混表格 — 似而不同

### 8.1 容易混的 ECG 表现

| 表现 | 病因 |
|---|---|
| **Sine wave** | 重度高钾（详见 [[完整笔记/专题笔记/心内/_衍生_代谢性ECG改变]]）|
| **Torsades de pointes (TdP)** | 长 QT（低 K / 低 Mg / 低 Ca / 药物 / 先天 LQTS）|
| **Peaked T** | **高钾**（窄、尖、对称）vs **超急期 MI**（宽、不对称、伴有 ST↑）|
| **U 波** | 低钾 / 低镁 |
| **Osborn (J) 波** | **低温** > 高钙、SAH、ICH |
| **Short QT** | 高钙、digoxin（伴 scooped ST）|

### 8.2 容易混的临床综合征

| 综合征 | 电解质 | 钥匙 |
|---|---|---|
| **Addison's** | **低 Na + 高 K** + 低糖 + 低血压 | 色素沉着、皮质醇低、肾上腺源 |
| **SIADH** | **低 Na**（单一）| Uosm > Posm、UNa↑、等容量 |
| **TLS** | 高 K + 高 P + 高 UA + **低 Ca** + AKI | 化疗后 |
| **Refeeding** | **低 P + 低 K + 低 Mg** | 长期饥饿/AN 再喂养 |
| **Conn（醛固酮↑）** | **低 K + 代谢性碱** | 高血压不响应多药 |
| **Bartter / Gitelman** | 低 K + 代谢性碱 + 正/低 BP（似利尿剂）| 家族性 |

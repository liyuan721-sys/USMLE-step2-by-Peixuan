# 下尿路梗阻诊疗决策（Lower Urinary Tract Obstruction）

> [!info] 一句话框架
> 排尿困难 / 尿潴留 / LUTS 的题，先按 **"急性 vs 慢性 + 人群"** 定位病因（BPH / 尿道狭窄 / AUR / 神经源膀胱 / 药物），再决定 **检查（bladder US / RUG / 肾 US）→ 处理**。
> 核心陷阱：**慢升 Cr 别只盯肾内，要排肾后性梗阻**；**老年男术后躁动想 AUR，别冲感染**；**年轻男梗阻想尿道狭窄，别套 BPH**。

---

## 一、病因鉴别（一表定位）

| 病因 | 典型人群 / 线索 | 关键特征 | 首选处理方向 |
|---|---|---|---|
| **BPH** | 老年男 + 渐进性 LUTS | 尿流弱 / 夜尿 / 排尿不尽；前列腺增大 | α1 阻滞剂 ± 5-ARI（详见 [[完整笔记/专题笔记/肾脏/肾脏_BPH药物治疗]]）|
| **急性尿潴留（AUR）** | 老年男 + BPH + 术后 / 药物（opioid、抗胆碱）| 突发不能排尿 + **耻骨上压痛** + 躁动 | **bladder US ≥300 mL** → Foley + UA |
| **尿道狭窄（urethral stricture）** | **年轻男** + 器械史 / 外伤 / 淋病史 | 梗阻性排尿，前列腺不大 | 逆行尿道造影 → 尿道扩张 / 切开 |
| **神经源性膀胱** | 脊髓损伤 / MS / DM / 帕金森 | 残余尿多 ± 失禁；神经背景 | 尿动力学 + 间歇导尿 |
| **梗阻性肾病（BPH 等致）** | 慢性梗阻 | **慢升 Cr** + 双肾积水 | **先肾 US 排 hydronephrosis** |
| **药物性尿潴留** | opioid / 抗胆碱 / α 激动剂 | 用药时间相关 | 停诱因 + 必要时导尿 |
| **产后尿潴留** | 产后 + 区域麻醉 | 滴尿 = overflow（≠ 观察）| 导尿减压 |

---

## 二、决策树

```
排尿困难 / 尿潴留 / LUTS
   │
   ├─ 急性不能排尿 + 耻骨上压痛 ─→ 疑 AUR
   │        └─ bladder US ≥300 mL → Foley（诊断+治疗）+ UA 排 UTI
   │           （肥胖/腹水致 US 不准 → 直接 Foley）
   │
   ├─ 慢性渐进 LUTS
   │     ├─ 老年男 + 前列腺大 ────→ BPH → 药物（α1 ± 5-ARI）
   │     ├─ 年轻男 + 器械/外伤史 ─→ 尿道狭窄 → 逆行尿道造影
   │     └─ 神经背景（SCI/MS/DM）→ 神经源膀胱 → 尿动力 + 间歇导尿
   │
   ├─ 慢性进行性 Cr↑（疑肾后梗阻）─→ 先肾 US 排 hydronephrosis
   │
   └─ 用药相关（opioid / 抗胆碱）──→ 药物性 → 停诱因 ± 导尿
```

---

## 三、关键陷阱与速记

> [!danger] 三大高频陷阱
> 1. **慢升 Cr 的 BPH 老人** → 第一步**肾脏超声排积水**，不是直接上 BPH 药（[[mistakes/uworld-mistakes#^Q20536]]）。
> 2. **老年男术后躁动 + 耻骨上压痛** → 先想 **AUR**（最常见可逆病因），**WBC 正常 + 无热 + 术口清洁 = 反对感染**，别冲血培养（[[mistakes/uworld-mistakes_2026-06#^Q4615]]）。
> 3. **年轻男梗阻性排尿** + 器械 / 外伤史 → **尿道狭窄**，不是 BPH（[[mistakes/uworld-mistakes#^Q15906]]）。

> [!tip] AUR 诊疗速记
> - 危险因素：**男 + 高龄 + BPH + 神经病 + 手术（关节置换）+ 诱发药（opioid / 抗胆碱）**。
> - 确诊：**bladder US ≥300 mL**；肥胖 / 腹水 US 不准 → 直接 Foley。
> - 处理：Foley（既诊断又治疗）+ UA 排 UTI；拔管后排尿试验可加 **tamsulosin** 提高成功率。

> [!warning] 上尿路 vs 下尿路梗阻别混
> 本笔记是**下**尿路（前列腺 / 尿道 / 膀胱出口）。**上**尿路梗阻（输尿管结石）见 [[mistakes/uworld-mistakes_2026-06#^Q4749]]：腰痛 + 尿少↔多尿交替（梗阻后利尿）+ 独肾易肾衰。

## 四、复习题

> [!question]- Q1. 老年男 BPH，慢性进行性 Cr 升高。第一步检查？为什么？
> **肾脏超声**——排**肾后性梗阻（hydronephrosis）**。慢性下尿路梗阻可逆向损肾；先确认 / 解除梗阻再谈别的。

> [!question]- Q2. 82 岁男髋术后第 2 天躁动 + 耻骨上压痛，WBC 正常无发热。最可能病因 + 下一步？
> **急性尿潴留（AUR）**；下一步 **bladder US**（≥300 mL 确诊）→ Foley + UA。无热 + WBC 正常 + 术口清洁 → 反对感染，不先血培养。

> [!question]- Q3. 25 岁男渐进性尿流弱，有尿道淋病 / 导尿史，前列腺不大。最可能诊断 + 检查？
> **尿道狭窄**；**逆行尿道造影（RUG）**定位。≠ BPH（BPH 是老年 + 前列腺增大）。

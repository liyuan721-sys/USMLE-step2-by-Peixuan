# BPH 药物治疗 · 速查版（一页秒扫）

> 详细版 → [[完整笔记/专题笔记/肾脏/肾脏_BPH药物治疗]]

## 两类主力药对比（背这张就够）

| 维度 | **α1 阻滞剂** | **5α-还原酶抑制剂** |
|---|---|---|
| 代表 | 非选择性：doxazosin / terazosin / prazosin；选择性 α1A：tamsulosin / silodosin / alfuzosin | finasteride（II）/ dutasteride（I+II）|
| 起效 | **快（数天）** | **慢（数月-年）** |
| 作用 | 松平滑肌（功能）| 缩腺体（结构）|
| 体积 | 不变 | ↓ |
| PSA | 不变 | **↓ 约 50%** |
| 降压 | 有（非选择性强）| 无 |
| 招牌 SE | 体位性低血压 / **IFIS** / 逆行射精 | 性欲↓ / ED / 男乳 / **致畸** |

## 选药决策树

```
中重症 LUTS（已排癌 / 狭窄 / 神经源）
   ├─ 想快缓解 ─→ α1 阻滞剂
   │     ├─ 合并 HTN ─→ doxazosin / terazosin（兼降压）
   │     └─ 血压正常 ─→ tamsulosin（少降压）
   ├─ 大前列腺 / 防进展 ─→ 加 5-ARI（常 α + 5-ARI 联用）
   ├─ 合并 ED ─→ tadalafil（PDE5i）
   └─ 难治 / 并发症 ─→ TURP
```

## 三大副作用陷阱（一句话）

- **IFIS**：α1（尤 tamsulosin）松虹膜扩张肌 → 白内障术中虹膜脱垂。**术前披露 + 术者改良技术（Malyugin 环 / 前房 phenylephrine）；不停药、不取消手术**。
- **5-ARI ↓PSA 50%**：筛查前列腺癌时测得 PSA **×2 还原**；不降反升 → 警惕癌。
- **5-ARI 致畸（Cat X）**：育龄/孕妇禁触压碎片（完整包衣片安全）。

## 干扰药速排（都不是 BPH 缩体积药）

- **bethanechol**（胆碱能）→ 治尿潴留促排空
- **oxybutynin**（抗胆碱）→ 治 OAB；BPH 残余尿多时慎用（诱发 AUR）
- **phenazopyridine** → 仅尿路镇痛
- **finasteride** → 不用于急性症状缓解（起效要数月）

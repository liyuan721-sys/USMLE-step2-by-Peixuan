# 酸碱实战速判卡（拿到 ABG 照着走）

## ⭐ 核心铁律（先刻进脑子）

> [!danger] 一句话定生死
> **判原发先读数，看 HCO3 / PaCO2 哪个"朝 pH 方向"变 —— 数定原发，史定病因。**
> 病史词（插管 / 呕吐 / 腹泻 / 肺炎）**只能在定完原发之后**用来找病因，**绝不能抢在第 ② 步之前**带你选答案。你最近 3 道错题（Q2786 / Q2826 / Q13842）栽的全是这一点。

## 做题 4 步流程

```
① pH？           <7.35 = 酸血症        >7.45 = 碱血症
        │
② 谁朝 pH 方向动？（看 HCO3 和 PaCO2）
        HCO3 低 → 代酸      HCO3 高 → 代碱
        PaCO2 高 → 呼酸     PaCO2 低 → 呼碱
        ※朝 pH 方向变的那个 = 原发；另一个是代偿
        │
③ 若是代酸 → 算 AG = Na −(Cl+HCO3)  正常 8–12
        AG 高 → MUDPILES（DKA / 乳酸 / 甲醇乙二醇 / 尿毒症 / 水杨酸）
        AG 正常 → 走第 ④ 步（NAGMA）
        │
④ 收尾：代酸算 Winter 看代偿够不够；
        代碱先看病史 —— 病因明确(呕吐/利尿剂/NG)直接定，
        只有病因不明才查尿 Cl 分型(<20 NS / >20 治病因)
        Winter：预期 PaCO2 = 1.5×HCO3 + 8 (±2)，偏离 = 叠加呼吸紊乱
        ※题里给的"血 Cl"是血指纹，≠尿 Cl，别混（Q2826 给的是血 Cl 101→92）
```

## 速查表 1 — 原发判读（第 ② 步用）

| 看到 | 原发是 | 常见病因（定完原发再对） |
|---|---|---|
| pH↓ + **HCO3 低** | 代谢性酸中毒 | 见表 2 / 表 3 |
| pH↓ + **PaCO2 高** | 呼吸性酸中毒 | COPD / 镇静过度 / 神经肌肉 |
| pH↑ + **HCO3 高** | 代谢性碱中毒 | 见表 3 |
| pH↑ + **PaCO2 低** | 呼吸性碱中毒 | 焦虑 / 肺炎 / PE / 水杨酸早期 |

> [!warning] 最高频陷阱
> **呼碱 HCO3 低；代碱 HCO3 高** —— 别搞反。"插管 / 机械通气"≠ 呼吸性，**先看 HCO3 方向**（Q2826：HCO3=31 高 → 是代碱不是呼碱）。
> **急性呼吸紊乱 HCO3 几乎正常**（肾代偿要数天）；HCO3 明显异常 = 慢性或合并代谢紊乱。

## 速查表 2 — 高 AG 代酸（AG 大 → 背 MUDPILES）

> [!info] MUDPILES
> **M**ethanol / **U**remia / **D**KA（+酒精/饥饿酮症）/ **P**ropylene glycol / **I**ron·INH / **L**actate（脓毒症·缺血）/ **E**thylene glycol / **S**alicylate
> 实战：见 AG↑ → 查酮体 / 乳酸 / 渗透间隙 / 水杨酸 / 肾功能。**水杨酸 = 高 AG 代酸 + 呼碱**（混合）。

## 速查表 3 — NAGMA（正常 AG 代酸）+ 代碱分型 ⭐（实战最爱考）

**NAGMA = 低 HCO3 + 高 Cl + 正常 AG。先用血钾切：**

| 血钾 | 尿 pH | 尿 AG | 结论 |
|---|---|---|---|
| **高 K** | <5.5 | 正 | **Type 4 RTA**（老糖人 + 糖尿病肾病）|
| 低 K | **>5.5** | 正 | **Type 1 远端 RTA**（Sjögren / 肾结石 / 婴儿遗传性）|
| 低 K | <5.5 | 正 | **Type 2 近端 RTA**（Fanconi / MM）|
| 低 K | <5.5 | **负**（neGUTive）| **腹泻** |

**代碱 → 病因不明时才查尿 Cl 分型**（病史明给呕吐/利尿剂/NG → 直接定，跳过此表）：

| 尿 Cl | 类型 | 处理 | 病因 |
|---|---|---|---|
| **<20** | saline-**responsive** | 给 NS | 呕吐 / NG / **利尿剂(停药后)** |
| **>20** | saline-**resistant** | 治病因 | Conn / Cushing / Bartter-Gitelman / **在用利尿剂** |

> [!tip] 利尿剂代碱指纹
> 袢 / 噻嗪 → **HCO3↑ + 血 K↓ + 血 Cl↓（一高两低）= 收缩性代碱**（Q2826）。详见 [[完整笔记/专题笔记/肾脏/肾脏_利尿剂全总结]]。
> ⚠️ 这三个是**血指纹**（血 Cl，不是尿 Cl）。病史明给利尿剂时直接定，**不必再查尿 Cl**。

> [!tip] GI 丢失双子（钾都低、酸碱相反）⭐
> 上 vs 下消化道丢液，**钾都掉，但酸碱方向相反**——高频对子：
> - **腹泻（下消化道）= 低 K + 非 AG 代谢性酸中毒（NAGMA）**：丢富钾、富 HCO₃⁻ 的肠液；尿 AG **负**（neGUTive）。
> - **呕吐 / NG 引流（上消化道）= 低 K + 代谢性碱中毒**：丢 HCl（→ 碱中毒）+ **容量收缩性碱中毒**（继发醛固酮↑ → 远端进一步排 K）；saline-**responsive**（尿 Cl <20，给 NS）。
> 来源 [[mistakes/uworld-mistakes_2026-06#^Q3917]]（AIDS 腹泻 K 3.2）：**低钾是腹泻的非特异结果，不指向病原**——别拿它选答案。

## 60 秒自检（每天扫一遍 = 被动复习）

> [!question]- 不看答案，能脱口而出吗？
> 1. 判原发第一动作是什么？→ **读数，看谁朝 pH 方向变（不是看病史）**
> 2. pH 高 + HCO3 高 = ？→ **代碱**（不是呼碱！呼碱 HCO3 低）
> 3. 代酸下一步算什么？→ **AG = Na−(Cl+HCO3)**
> 4. NAGMA 第一刀用什么切？→ **血钾**（高 K → Type 4；低 K → 看尿 pH）
> 5. 尿 pH >5.5 的 NAGMA = ？→ **Type 1 远端 RTA**
> 6. 尿 AG 负值 = ？→ **腹泻**（neGUTive）
> 7. 代碱**病因不明**时查什么分型？→ **尿 Cl**（<20 给盐水 / >20 治病因）；病史明给则直接定，题里"血 Cl"≠尿 Cl
> 8. Winter 公式？→ **预期 PaCO2 = 1.5×HCO3 + 8 (±2)**
> 9. 腹泻 vs 呕吐的酸碱？→ **腹泻 = 低 K + 非 AG 代酸（NAGMA）；呕吐 = 低 K + 代碱**（钾都低、酸碱相反）

# 肾脏 / 肾小管酸中毒 (RTA) 四型完整体系

> [!info] 一句话定位
> **RTA（renal tubular acidosis 肾小管酸中毒）= 肾小管泌 H⁺ / 重吸收 HCO₃⁻ 障碍导致的正常 AG 代谢性酸中毒（NAGMA, normal anion gap metabolic acidosis）。**
> 本笔记 = RTA **主笔记**（四型机制 + 两步定位 + 先天遗传 + 儿科镜像 + PHA），原 [[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全]] §2.3 已精简为浓缩复习版并指回本篇。
> 配套实战：[[完整笔记/专题笔记/肾脏/肾脏_酸碱实战速判卡]]（秒判表）/ [[完整笔记/专题笔记/肾脏/肾脏_酸碱代偿完整SOP]]（ABG 错题串讲）。

## §一 四型速判（先建框架）

| 类型 | K | 尿 pH | AG | 机制 | 经典病因 |
|---|---|---|---|---|---|
| **Type 1（远端 distal）** | **↓** | **> 5.5** | 正常 | DCT（远曲小管 distal convoluted tubule）泌 H⁺ 障碍 | Sjögren 干燥综合征 / 自免 / 两性霉素 B amphotericin |
| **Type 2（近端 proximal）** | **↓** | < 5.5（早期）| 正常 | PCT（近曲小管 proximal convoluted tubule）重吸收 HCO₃⁻ 障碍 + Fanconi 综合征 | 多发性骨髓瘤 MM / Wilson 病 / 重金属 / 乙酰唑胺 acetazolamide |
| **Type 4（醛固酮抵抗）⭐** | **↑** | < 5.5 | 正常 | 醛固酮 aldosterone 缺乏 / 抵抗 → K 不能排 + H⁺ 排泌↓ | 老年糖尿病 + 糖尿病肾病（见 §四）|

> [!danger] 入口反射：「低钾 + 高氯」先想 NAGMA，再用血钾切 Type 4 ⭐
> **看到「低 K + 高 Cl + 低 HCO₃」第一反应 = NAGMA（正常 AG 代酸）**，别急着算 AG 也已八九不离十。
> **关键捷径——先用血钾把 Type 4 切出去**：Type 4 是四病因里唯一**高钾**的（醛固酮抵抗 → 排不出 K）。
> - **低钾 + 高氯 → 基本排除 Type 4**，只剩 **腹泻 / Type 1 / Type 2**（再走 §二两步）
> - **高钾 + 高氯 → 直奔 Type 4**（老糖人 + 糖尿病肾病）
> 一句话：**「钾低看远近端 RTA / 腹泻，钾高直接 Type 4」**。

## §二 NAGMA 两步定位法（尿 pH + 尿 AG）⭐

> [!tip] NAGMA 四病因两步定位
> NAGMA = 低 HCO₃ + **正常 AG** + 高 Cl（hyperchloremic 高氯性）。四病因 = **腹泻 / Type 1 / Type 2 / Type 4 RTA**。
>
> **第 1 步——看尿能否酸化（代酸时尿 pH）**：
> - 尿 pH **> 5.5（不恰当偏碱）= 远端 Type 1 RTA**（DCT 泌 H⁺ 障碍，尿酸化不了）⭐
> - 尿 pH **< 5.5（肾正常代偿）= 腹泻 / Type 2 / Type 4**
>
> **第 2 步——分腹泻 vs 肾性，算尿 AG（UAG, urine anion gap 尿阴离子间隙）= U_Na + U_K − U_Cl**：
> - **负值 = 腹泻**（"**neGUT-ive**"，肠丢 HCO₃，肾代偿排 NH₄Cl 把 Cl 拉高 → 公式变负）
> - **正值 = RTA**（肾排不出 NH₄⁺/H⁺，无足够未测阴离子 Cl）
> - 本质：**UAG 是"尿铵 NH₄⁺"的影子**（NH₄⁺ 是未测阳离子，陪 Cl 一起排）。负 = 铵多 = 肾排酸正常 → 病在肠；正 = 铵少 = 肾排酸障碍 → RTA。

> [!warning] UAG 什么时候"不准"——3 个失效条件 ⭐
> UAG **只在单纯 NAGMA 可信**，下列情况假性偏正（误判成 RTA），改用**尿渗透间隙 urine osmolal gap** 估铵：
> 1. **只用于正常 AG 代酸**；高 AG 代酸不算 UAG。
> 2. **尿里有大量其他未测阴离子** → 假性偏正：**酮症 / DKA（糖尿病酮症酸中毒 diabetic ketoacidosis，酮酸根）、HCO₃ 尿、马尿酸（甲苯吸入 toluene）、青霉素 penicillin / 大量药物阴离子**（这些阴离子陪 NH₄⁺ 排出，Cl 没涨 → UAG 不变负）。
> 3. **尿 Na < 25** → 容量不足肾保钠 → UAG 失真，不可信。

**NAGMA 四病因一表分（K + 尿 pH + 尿 AG + Stem 线索）⭐**

| 病因 | K | 尿 pH | 尿 AG | Stem 线索 |
|---|---|---|---|---|
| **腹泻**（GI 丢 HCO₃）| **↓** | 酸 < 5.5 | **负**（neGUTive）| 肠丢碱，肾正常酸化 |
| **Type 1 远端 RTA** | **↓** | **碱 > 5.5** ⭐ | 正 | Sjögren / 自免 / 肾结石 / 两性霉素 B |
| **Type 2 近端 RTA** | **↓** | 多变（早 < 5.5）| 正 | Fanconi / MM / Wilson / 乙酰唑胺 |
| **Type 4 RTA** | **↑** | < 5.5 | 正 | 老糖人 + 糖尿病肾病 + 低肾素低醛 |

> 读表顺序：**先血钾**（高钾 → Type 4；低钾 → 余三者）→ **尿 pH**（>5.5 = Type 1）→ **尿 AG**（负 = 腹泻，正 = RTA）。

## §三 机制原理（能现推，不用背）⭐

> [!info]- 三条机制原理（点开看）
> **① 为什么 NAGMA 一定"高氯"——电中性填空**
> 血浆电中性：Na ≈ Cl + HCO₃ + 未测阴离子（AG）。**直接丢 HCO₃⁻**（腹泻丢肠液 / RTA 肾丢碱）时，HCO₃ 腾出的位置被 **Cl⁻ 顶上来填**（肾代偿多重吸收 Cl）→ HCO₃↓ + Cl↑ 但 **AG 不变**。所以 **hyperchloremic = normal AG 是同义**，看到"低 HCO₃ + 高 Cl"不算 AG 也知道是 NAGMA。
> 对比：高 AG 代酸（DKA / 乳酸）加进来的是**新阴离子**（酮酸/乳酸根）占了空位 → **AG 撑大，Cl 不涨**。
>
> **② 为什么血钾能切 Type 4——醛固酮统一轴（灵魂）⭐**
> 醛固酮在集合管**一手排 K⁺（principal cell 主细胞）一手排 H⁺（α-intercalated cell 闰细胞）**，两件事绑定。
> - **腹泻 / Type 1 / Type 2**：多伴容量丢失 → RAAS（肾素-血管紧张素-醛固酮系统 renin-angiotensin-aldosterone system）触发 **继发性醛固酮↑（系统在加班）** → K 被排掉 → **低钾**
> - **Type 4**：本质就是**醛固酮缺乏或抵抗**（老糖人 JG 细胞 juxtaglomerular cell 球旁细胞坏 = 低肾素低醛；或 ACEI/ARB/螺内酯/肝素阻断）→ K 排不出 → **高钾**
> > 钉死：**"前三个醛固酮在干活(低钾)，Type 4 醛固酮罢工(高钾)；血钾就是醛固酮系统的体检表。"**
>
> **③ 为什么 Type 4 也酸、尿 pH 还 < 5.5——双重机制**
> - 醛固酮也驱动 H⁺ 排泌，它少了 → H⁺ 排不出 → 酸。
> - **高钾本身抑制近端氨生成（ammoniagenesis 产铵）** → 尿 NH₄⁺（排酸主力载体）产量↓ → 排酸能力↓ → 酸中毒。
> - **关键区别**：Type 4 的 H⁺ 泵本身是好的（尿能酸化 → 尿 pH < 5.5），坏在**缓冲不够**（没足够 NH₄⁺ 携带 H⁺）；Type 1 是 **H⁺ 泵真坏了**（DCT 泌 H⁺ 障碍）→ 尿酸化不了 → **尿 pH > 5.5**。这就是表里 Type 1 vs Type 4 尿 pH 分道扬镳的机制根。

## §四 Type 4 RTA 完整指纹 + 三镜像

#### Type 4 RTA 完整指纹（"低肾素低醛 + 老糖人 + 高钾酸中毒"）

| 项目 | Type 4 RTA |
|---|---|
| **指纹组合** | **高 K + 正常 AG 代酸 + 尿 pH < 5.5 + 低肾素 + 低醛固酮** |
| **典型患者** | **老年 + 长病程糖尿病 + 糖尿病肾病** |
| **机制** | 糖尿病 / 间质肾炎 → JG 细胞受损 → 肾素↓ → 醛固酮↓ → 远端 K 排泌↓ + H⁺ 排泌↓ |
| **诱因 / 加重药** | **ACEI（血管紧张素转换酶抑制剂）/ ARB（血管紧张素受体拮抗剂）/ 螺内酯 spironolactone / 肝素 heparin / NSAID（非甾体抗炎药）/ 钙调磷酸酶抑制剂**（都进一步降低醛固酮或拮抗其效应）|
| **治疗** | ① 停止上述药 ② **氟氢可的松 fludrocortisone**（补盐皮质）③ 袢利尿剂 loop diuretic（促 K 排）④ 限钾饮食 ⑤ 碳酸氢钠 NaHCO₃ |

#### Type 4 RTA vs DKA vs Addison 三镜像（"低钠高钾代酸"三剑客）

| 维度 | **DKA**（"甜的酸"）| **Addison/PAI 原发肾上腺功能不全**（"黑的酸"）| **Type 4 RTA**（"老的酸"）|
|---|---|---|---|
| **血糖 Glu** | **显著↑** (>250) | **↓** | 通常正常 |
| **AG** | **↑↑** | 正常 | **正常** |
| **全身总 K** | **显著↓**（假性高钾 pseudohyperK）| ↑（醛固酮缺乏）| ↑（醛固酮抵抗 / 缺乏）|
| **核心机制** | 胰岛素缺乏 → K 出细胞 + 酮酸 | 醛固酮缺乏 → 不排 K | 醛固酮抵抗 → 不排 K |
| **特征体征** | Kussmaul 呼吸 + 烂苹果味 + 脱水 | **皮肤色素沉着** + 低血压 | 老糖人 + 肾病史，无色素，无高 Glu |
| **第一步** | NS + 胰岛素 + K | 静脉氢化可的松 hydrocortisone + NS | 停损害药 + 氟氢可的松 |

> [!tip] 一句话记忆钩
> "Type 1/2 低 K，Type 4 高 K；Type 1 尿 pH 高，Type 2/4 尿 pH 低；**老糖人 + 高 K + 代酸 = Type 4**；甜 = DKA；黑 = Addison；老 = Type 4。"

## §五 先天 / 遗传病因（按年龄分两套）⭐

> [!warning] 病因按年龄分两套
> **成人新发 RTA = 后天获得**（自免 / 药 / 副蛋白）；**婴儿 / 新生儿期起病 = 原发遗传性**（基因突变直接坏掉小管转运体）。同一个泵坏，病因换一套。儿科 FTT（生长迟缓 failure to thrive）+ NAGMA 就是遗传性 RTA 的典型首发（来源 [[mistakes/uworld-mistakes_2026-05#^Q4828]]）。

| 类型 | 遗传病 / 基因 | 遗传方式 | 识别钩子 ⭐ |
|---|---|---|---|
| **Type 1 远端** | **ATP6V1B1**（H⁺-ATPase B1 亚基）| AR（常染色体隐性）| **伴感音神经性耳聋**（早发）⭐ |
| Type 1 远端 | ATP6V0A4（H⁺-ATPase a4 亚基）| AR | 耳聋较晚 / 可变 |
| Type 1 远端 | **SLC4A1**（AE1，Cl⁻/HCO₃⁻ 交换体）| AD（多）/ AR | 可伴**遗传性球形 / 椭圆形红细胞增多** |
| **Type 2 近端**（多为 **Fanconi** 一部分）| **胱氨酸贮积症 cystinosis（CTNS）** | AR | **儿童 Fanconi 最常见病因** ⭐ + 角膜胱氨酸结晶 + 生长迟缓 |
| Type 2 / Fanconi | **Wilson 病**（ATP7B）| AR | K-F 环 + 肝病 + 神经精神 |
| Type 2 / Fanconi | 半乳糖血症 / 遗传性果糖不耐受 | AR | 喂奶 / 进果糖后呕吐 + 低血糖 + 肝大 |
| Type 2 / Fanconi | 酪氨酸血症 I 型 | AR | 肝衰 + 煮卷心菜味 |
| Type 2 / Fanconi | **Lowe 综合征**（OCRL）| **X 连锁** | 眼脑肾三联（先天白内障 + 智力障碍 + Fanconi）|
| Type 2 / Fanconi | Dent 病（CLCN5）| X 连锁 | 低分子蛋白尿 + 高钙尿 + 肾结石 |
| **Type 4**（醛固酮抵抗）| **假性醛固酮减少症 1 型 PHA1**：ENaC（SCNN1，AR 重型）/ MR 受体（NR3C2，AD 轻型）| AR / AD | 新生儿**失盐危象**（低钠 + 高钾 + 代酸 + 醛固酮↑↑）；AD 型随龄改善 |
| Type 4（高钾高血压型）| **Gordon 综合征 / PHA2**（WNK1/4、KLHL3、CUL3）| AD | **高钾 + 高血压 + NAGMA + 低肾素**，**噻嗪特效** ⭐ |

> [!tip] 三个最高频钩
> - **远端 RTA 婴儿 + 耳聋 → ATP6V1B1（H⁺-ATPase）突变** ⭐
> - **儿童 Fanconi（Type 2）找代谢病 → 首位 cystinosis**；再想 Wilson / 半乳糖血症 / 果糖不耐受
> - **PHA1（婴儿失盐 + 高钾，醛固酮高 = 抵抗不是缺乏）** vs **Addison/CAH（醛固酮低 = 缺乏）**——醛固酮高低是分水岭

## §六 儿科「婴儿低钠 + 高钾 + 代酸」镜像 + PHA 专题

> [!info] 共同表现 = 新生儿失盐危象
> 1–4 周龄婴儿 **低钠 + 高钾 + 正常 AG 代酸 + 脱水 / 喂养差 / 体重不增**。先分两步：**① 皮质醇够不够（有无低血糖 / 低血压休克）② 醛固酮是低（缺乏）还是高（抵抗）**。

| 病因 | 醛固酮 | 皮质醇 | 雄激素 / 生殖器 | 一锤定音 |
|---|---|---|---|---|
| **21-羟化酶缺乏 CAH（先天性肾上腺皮质增生 congenital adrenal hyperplasia）失盐型** ⭐ | **↓** | **↓** | **↑**（女婴外生殖器男性化）| **17-OHP（17-羟孕酮）↑↑**；最常见；1–2 周龄危象 |
| **PHA1**（假性醛固酮减少症）| **↑↑（抵抗）** | 正常 | 正常 / 正常 | **醛固酮高是关键**；补盐，AD 型随龄改善 |
| **醛固酮合成酶缺乏** | **↓（孤立）** | 正常 | 正常 / 正常 | 单纯失盐，皮质醇 / 雄激素都正常 |
| **原发肾上腺功能不全**（新生儿 Addison）| **↓** | **↓** | 视病因 | 少见：肾上腺出血 / 发育不全 / ALD（X 连锁，男婴）|

> [!tip] 解题三步钩
> - **女婴外生殖器男性化 + 失盐 + 17-OHP↑↑ = 21-OH CAH** ⭐（治疗：氢化可的松 + 氟氢可的松 + 补盐 / NaCl）
> - **醛固酮 ↑↑（不是↓）→ PHA1**（受体抵抗，越补越高；治疗只能补盐 + 限钾，不是补盐皮质）
> - **皮质醇低（休克 / 低血糖）= 肾上腺源（CAH / Addison）；皮质醇正常 = 肾 / 受体源（PHA1 / 醛固酮合成酶缺乏 / Type 4）**
>
> 镜像对照：成人三剑客 = DKA（甜）/ Addison（黑）/ Type 4（老）；婴儿三剑客 = **CAH（生殖器）/ PHA1（醛固酮反高）/ 肾上腺危象**。逻辑同源：**醛固酮缺乏 vs 抵抗**定生死。

> [!info]- PHA（假性醛固酮减少症 pseudohypoaldosteronism）= 真性 vs 假性 + PHA1/PHA2 镜像（点开看）
> **"假性"= 表现像醛固酮不够（失盐 / 高钾 / 代酸），但醛固酮本身正常或↑↑，问题在肾小管抵抗。**
>
> | 维度 | **真性醛固酮减少**（Addison / CAH / 合成酶缺乏）| **PHA（假性）** |
> |---|---|---|
> | 醛固酮水平 | **↓（真低）** | **正常 / ↑↑** |
> | 病灶 | 肾上腺（产不出）| 肾小管（受体 / 通道抵抗）|
> | 氟氢可的松 | **有效** | **无效**（受体不听）→ 只能补盐 / 限钾 |
>
> **PHA 两型长得相反，唯一共性 = 高钾 + NAGMA：**
>
> | | **PHA1**（经典型，儿科）| **PHA2 = Gordon 综合征**（成人 / 青少年）|
> |---|---|---|
> | 基因 / 机制 | ENaC（SCNN1，AR 重）/ MR（NR3C2，AD 轻）失活 → 醛固酮抵抗 | WNK1/4、KLHL3、CUL3 → **NCC（噻嗪靶点）活性过强** |
> | 钠 | **保不住 → 失盐** | **吸太多** |
> | 血压 | **低 / 正常** | **高** ⭐ |
> | 醛固酮 / 肾素 | 醛固酮 **↑↑** | **低肾素** |
> | 典型 stem | 新生儿失盐 + 高钾 + 醛固酮↑↑ + 皮质醇正常 | 高血压 + 高钾 + 低肾素 + NAGMA |
> | 治疗 | 补盐 + 限钾（AD 型随龄改善）| **噻嗪 thiazide 特效** ⭐ |
>
> 一句话：**"PHA1 = 抵抗失盐 → 低钠高钾低血压（婴儿，补盐）；PHA2/Gordon = 钠吸太多 → 高钾高血压低肾素（噻嗪特效）。"** Gordon 正好是 **Gitelman 的镜像**（Gitelman = NCC 失活 → 低钾低血压代碱；Gordon = NCC 过强 → 高钾高血压代酸）。

## §七 一句话钩总集（考前扫这块）

- **入口**：低 K + 高 Cl + 低 HCO₃ = NAGMA → 先看血钾切 Type 4。
- **尿 pH > 5.5 的 NAGMA = Type 1 远端 RTA**（H⁺ 泵真坏）。
- **尿 AG 负 = 腹泻（neGUTive）；正 = RTA**（UAG 是尿铵的影子）。
- **老糖人 + 高 K + 代酸 + 尿 pH<5.5 = Type 4**（低肾素低醛）。
- **甜 = DKA / 黑 = Addison / 老 = Type 4**（低钠高钾代酸三剑客）。
- **婴儿远端 RTA + 耳聋 = ATP6V1B1**；**儿童 Fanconi 首位 = 胱氨酸贮积症**。
- **婴儿失盐 + 醛固酮↑↑ = PHA1（抵抗）**；**醛固酮↓ = CAH/Addison（缺乏）**。

# 肾脏 — ADPKD 多囊肾 正向锁定 6 联指纹

> [!info] 笔记定位
> ADPKD (Autosomal Dominant Polycystic Kidney Disease) 是 USMLE Step 2 肾科高频题型。本笔记目标：**从"排除法选对"升级到"5 秒正向锁定"**。来源：[[mistakes/uworld-mistakes_2026-05#^Q4474]]（5-28 做对题但**排除法 + 正向反射未建立** → 早建 v1）。

---

## Part 1. 6 联正向指纹（≥3 项即锁，不必家族史明示）⭐⭐⭐

| 指纹 | Stem 关键词 | 机制 / 高频考点 |
|---|---|---|
| **① Bilateral flank pain** ⭐ | "**Bilateral**" 是 buzzword | 单侧 = stone / RCC / pyelo；**双侧 = ADPKD**（高特异）|
| **② Palpable bilateral abdominal mass / 肝大** ⭐⭐ | "Liver enlarged" + "flank mass on deep palpation" | **巨大多囊肾**（不是肿瘤）+ **80% ADPKD 有 hepatic cysts** |
| **③ 早发 HTN** | 中年 + BMI 正常 + 无烟酒 + BP↑ | 囊肿压肾实质 → 局部缺血 → **RAAS↑ renin↑** → 继发性 HTN |
| **④ Long-standing nocturia / polyuria** | "Nocturia × 多年" | **远端肾小管损 → urinary concentrating defect**（nephrogenic DI-like）— 可早 Cr↑ **10-20 yr** |
| **⑤ CKD 渐进 + 贫血** | Cr↑ + Hb↓ + BUN↑ | EPO↓ 致 CKD anemia + ADPKD 自然病程：**30-40 起，55-60 ESRD** |
| **⑥ 家族史** | 60% 阳性 | 不明示**不排除** — 新发突变 10% / stem 省略 |

> [!success] 5 秒锁定 SOP ⭐
> 看到中年 + 以下 **≥3 项** 立刻锁 ADPKD（不必排除法）：
> 1. Bilateral 任何泌尿症状
> 2. Palpable bilateral mass / 肝大
> 3. 早发 HTN（无肥胖/无烟酒）
> 4. Long-standing nocturia / polyuria
> 5. CKD + 贫血
> 6. Hematuria（囊肿破裂）/ 反复 UTI（囊肿感染）

## Part 1.5 关键解剖卡点：为什么 stem 写"单侧 mass"但 ADPKD 是双侧？⭐⭐⭐

> [!warning] USMLE 高频陷阱
> Stem 描述的是**体检发现**，不是**病理范围**。**双侧多囊肾，但右侧触诊更容易摸到** —— 不是"只有右侧有 mass"。

### 解剖原因：左右肾触诊难度差

```
                    前面
                ┌─────────┐
                │  ━ 胃 ━  │
                │  ━ 脾 ━  │← 遮挡左肾（前方屏障）
        ┌───────┤          ├───────┐
        │ 肝 ↓ │           │      │
        │ 推下 ↓│          │      │
        │ 右肾 │   脊柱    │ 左肾 │
        │（低位）│          │（高位）│
        └───────┴──────────┴───────┘
                    后面
```

| 因素 | 右肾 | 左肾 |
|---|---|---|
| **位置** | **低 ~1-2 cm**（被肝推下）| 高 |
| **前方遮挡** | 肝（实质但稳定）| **胃 + 脾**（含气 + 移动）|
| **正常触诊** | 下极常可触及（瘦人）| 极难触及 |
| **ADPKD 触诊** | **大囊肾从肋缘下凸出** ⭐ | 被胃/脾遮 → 体检难摸到 |

### Stem 双侧线索的真正位置 ⭐

回看 Q4474 stem 逻辑链：

```
"Bilateral flank pain" ← 病理双侧性的 buzzword（题干自报答案）⭐
                ↓
体检 "liver enlarged + right flank mass" ← 临床发现（解剖学使然右侧明显）
                ↓
US 必然双侧多囊（stem 不必重复说，buzzword 已定）
```

### 反 pattern 高频陷阱 ⭐⭐⭐

| 看到 | ❌ 错反射 | ✅ 正确解读 |
|---|---|---|
| "Right flank mass" 单侧描述 | 反射 RCC | 先扫 stem 有无 "bilateral" buzzword |
| 单侧 mass + bilateral pain | 矛盾不解？ | **信病理 buzzword 不信体检发现** ⭐ |
| 真正单侧 mass 锁 RCC | "Right flank mass" 一句话 | RCC stem 必有 unilateral hematuria + paraneoplastic **polycythemia** + weight loss + fever，**绝不写 "bilateral" 任何线索** |

### 类比的临床场景
- **Cardiomegaly 全心扩大** → stem 常只写 "PMI displaced **laterally**"（非 "bilateral"）
- **Hepatosplenomegaly** → stem 可能只写 "hepatomegaly" 或 "splenomegaly" 其一
- **Bilateral lung 病变** → 体检常只描述一侧听到的体征

### 一句话锁定
**ADPKD 双肾都大，但右肾在体检中更容易"凸出来被摸到"**（左肾位置高 + 被胃脾遮）。Stem 写 "right flank mass" 是体检表象，**"bilateral flank pain" 才是病理范围 buzzword** ⭐⭐⭐

## Part 2. Extrarenal Features 8 项 ⭐

记忆口诀：**"Brain-Liver-Heart-Colon-Hernia"** 5 大系统 + 胰腺 / 精囊 / 肾内并发症

| 系统 | 表现 | 高频考点 |
|---|---|---|
| **脑** ⭐⭐⭐ | **Berry aneurysm**（Circle of Willis）| **SAH 突发剧烈头痛** = 致死风险 |
| **肝** ⭐ | Hepatic cysts（80%）| 肝大 / 偶发肝囊肿 |
| **心** | **MVP + AR**（瓣膜 prolapse / 反流）| 心脏杂音 |
| **结肠** | **Diverticulosis** | 反复憩室炎 |
| **腹壁** | **Ventral / inguinal hernias** | 反复疝 |
| 胰腺 | Pancreatic cysts | 多无症状 |
| 精囊 | Seminal vesicle cysts | 男性不育 |
| 肾内 | 肾结石↑ + UTI↑（囊肿感染）| — |

## Part 3. 病理生理与遗传

### 3.1 遗传
- **AD 显性**（PKD1 染色体 16，~85%；PKD2 染色体 4，~15%）
- **PKD1 比 PKD2 进展快**（PKD1 ESRD ~55 yo / PKD2 ~75 yo）
- 60% 有家族史，10% 新发突变

### 3.2 病机
```
PKD1/PKD2 突变 → polycystin 异常
              ↓
        肾小管上皮细胞 → 增殖 + 分泌液体
              ↓
        无数囊肿渐进性扩张（每年 ~5% 体积↑）
              ↓
        ┌─ 压肾实质 → 局部缺血 → RAAS↑ → HTN
        │
        ├─ 压远端肾小管 → concentrating defect → nocturia/polyuria
        │
        ├─ 压肾血管 → CKD 渐进 → ESRD 55-60 yo
        │
        ├─ 囊肿破裂 → flank pain / hematuria
        │
        └─ 囊肿感染 → 反复 UTI / 囊肿脓肿
```

### 3.3 cAMP 通路是治疗靶点
- ADH → V2 受体 → cAMP↑ → 囊肿生长↑
- **Tolvaptan (V2 receptor antagonist)** 阻 cAMP → 减缓囊肿生长

## Part 4. 诊断（US 阈值）⭐

| 年龄 | 家族史阳性的诊断阈值 |
|---|---|
| **15-39 yo** | **总 ≥ 3 囊肿** |
| **40-59 yo** | **每侧 ≥ 2 囊肿** |
| **60+ yo** | **每侧 ≥ 4 囊肿** |

- **首选**：肾脏 US（便宜 + 无辐射）
- **不确定时**：MRI / CT（精度更高）
- **基因检测**：仅家族史不明 / 计划生育 / 移植供者评估

---

## Part 5. 管理 4 阶梯 ⭐

| 阶段 | 干预 | 机制 |
|---|---|---|
| **早期（HTN 控制）** ⭐ | **ACEi / ARB 首选** | 抑 RAAS + 保肾 |
| **囊肿生长抑制** | **Tolvaptan**（V2 受体拮抗） | 阻 cAMP 通路 |
| **生活方式** | 大量饮水（抑 ADH）/ 限盐 / 戒烟 | 减 cAMP / 减 HTN |
| **ESRD** | 透析（HD / PD）/ **肾移植** | — |

### 并发症监测
- **Cerebral aneurysm 筛查（MRA）**：仅以下 3 种情况 →
  - 家族 SAH 史
  - 高危职业（飞行员 / 卡车司机）
  - 计划重大手术（防术中破裂）
- **盲筛不推荐**（成本效益差）
- 心脏杂音 → echo 评估 MVP/AR
- 反复腹痛 → 排囊肿出血 / 感染

## Part 6. 易混陷阱（5 选 1 鉴别）⭐⭐

| 干扰项 | Stem **缺**的指纹 | 用什么排 |
|---|---|---|
| **BPH (Bladder outlet obstruction)** | 严重前列腺增大 + 排尿困难 + 残尿 | 前列腺**只 mildly** + 无尿急尿频 |
| **Horseshoe kidney** | UPJ obstruction / 反复 UTI / 单侧 stone | 不解释**肝大 + bilateral pain + 早 HTN** |
| **Nephrolithiasis** | **单侧**阵发绞痛 + 向腹股沟放射 + UA 血/晶体 | **Bilateral** + 数年长史 + 不解释肝大 |
| **ADPKD** ⭐ | （正向锁 6 联）| ✅ |
| **RCC** ⭐ 关键反 pattern | **单侧** mass + hematuria + fever + weight loss + **paraneoplastic EPO↑→红细胞增多 (polycythemia)** | **Bilateral pain + 肝大 + 贫血**（贫血是 RCC 的反向指纹）⭐⭐⭐ |

### 反 pattern 警示 ⭐⭐
- **"Bilateral" 是 ADPKD buzzword** — 单侧鉴别题不写 bilateral
- **"Liver enlarged + flank mass" 双胞胎组合 = 双囊肾 + 肝囊肿**，不是 RCC 肿瘤
- **RCC paraneoplastic EPO↑ → polycythemia**（红细胞增多） vs **ADPKD CKD → 贫血** — 血常规方向相反 ⭐
- **早发 HTN 50 yo + BMI 24 + 无烟酒** = 必找继发性病因（不是 essential）
- **Nocturia 长史 + 前列腺只 mildly** = 不是 BPH（BPH 前列腺要严重增大 + 残尿）
- **家族史不明示≠排除** — 60% 阳，新发突变 10%
- **ADPKD 不致单侧 mass** — 单侧 mass 锁 RCC / hydronephrosis

## Part 7. ADPKD vs Simple Cyst vs RCC 三联鉴别 ⭐（同 CT 偶发囊肿但路径不同）

| 特征 | **Simple renal cyst (Bosniak I)** | **ADPKD** | **RCC / Bosniak III-IV** |
|---|---|---|---|
| 数量 | **单个**（多侧偶 1-2 个）| **双侧无数**（按年龄阈值）| 单个肿块 |
| 壁 | 薄光滑 | 薄光滑（早期）| 厚 / 不规则 |
| Enhancement | **无** | 无（囊液性）| **有** ⭐ |
| 临床 | 无症状 | **6 联指纹**（bilateral pain / 肝大 / 早 HTN / nocturia / CKD / 贫血）| **单侧** mass / hematuria / fever / weight loss / **polycythemia** |
| 家族史 | 无关 | **60% 阳** | 散发为主 |
| 管理 | **Reassurance + 无 follow-up** | ACEi + Tolvaptan + 透析/移植 | **Partial / radical nephrectomy** |

> 详见 [[完整笔记/专题笔记/肾脏/肾脏_肾囊肿Bosniak分类]]（Simple cyst + Bosniak 分级 + 偶发病变 4 原则）

## Part 8. Memory Hook

> **"Bilateral flank + 肝大 + 早 HTN + 长期 nocturia + CKD + 贫血 = ADPKD 6 联"** ⭐⭐⭐ 5 秒锁定
>
> **"Bilateral" 是 ADPKD buzzword** — 单侧 = stone / RCC
>
> **Extrarenal "Brain-Liver-Heart-Colon-Hernia"** 5 大系统口诀 ⭐
>
> **"贫血反 RCC（RCC = polycythemia 增多）"** 关键反 pattern ⭐
>
> **HTN 治疗 = ACEi/ARB 首选 + Tolvaptan 抑囊肿 + 大量饮水 + ESRD 透析/移植**
>
> **Berry aneurysm 致死 SAH** — 家族 SAH 史 / 高危职业才筛 MRA
>
> **"Liver enlarged + flank mass" 双胞胎组合 = 双囊肾 + 肝囊肿**（不是 RCC 肿瘤）

# 肾脏 — 血尿（Hematuria）分期定位与病因鉴别

> [!info] 笔记定位
> 血尿是 USMLE Step 2 肾科 / 外科高频题型。本笔记目标：**5 秒按"分期 + 风险因素 + 反 pattern"定位病因 → 下一步检查**。来源：[[mistakes/uworld-mistakes#^Q4751]]（5-28 错题 — 53yo M terminal hematuria + clots + 吸烟，错选 Nephrolithiasis 应选 Bladder cancer → 早建 v1）。

---

## Part 1. 血尿分期定位三联 ⭐⭐⭐（最高产框架）

按**血出现于排尿哪一段**定位来源 — USMLE 看到"red urine"先问"是 initial / terminal / total？"

| 分期 | Stem 描述 | 提示来源 | 经典病因 |
|---|---|---|---|
| **Initial（初始）** | 排尿**起始**带血，后段清 | **尿道（urethra）** | Urethritis / Urethral injury（Foley catheter trauma）|
| **Terminal（末段）** ⭐ | "**Turns red by the end of voiding**" | **膀胱颈 / 三角区 / 后尿道 / 前列腺** | **Urothelial cancer**（bladder neck/trigone）/ Cystitis（infectious/postradiation）/ Urolithiasis（膀胱结石）/ BPH / Prostate cancer |
| **Total（全程）** | 整个排尿过程都有血 | **整个泌尿道任何位置**（上至肾 下至膀胱） | Renal mass / Glomerulonephritis / Urolithiasis / **PCKD** / Pyelonephritis / Urothelial cancer（典型 painless total）/ Trauma / Renal papillary necrosis |

> [!tip] 反射记忆
> - **Initial 红 = 尿道**（最远端出血只染初始尿）
> - **Terminal 红 = 膀胱底 / 前列腺**（排尿末段挤压膀胱底带血）
> - **Total 红 = 上游任何位置**（血充分混入整段尿液）

---

## Part 2. 血尿病因 6 大类全谱

| 大类 | 具体病因 | 经典指纹 |
|---|---|---|
| **Renal mass** | Renal cell carcinoma / Wilms tumor（儿） / PCKD | 全身症状 + 腹部 mass + 副瘤综合征 |
| **Glomerular** | IgA / PSGN / RPGN / Lupus nephritis / Alport / Thin BM | **RBC casts** + dysmorphic RBC + 蛋白尿；**无 clots** |
| **Urolithiasis** | Ca-oxalate / Ca-Pi / Uric / Cystine / Struvite | **Acute flank-groin colic**；hematuria 多 microscopic |
| **Polycystic** | ADPKD / ARPKD | Bilateral pain + bulky kidneys + 早 HTN |
| **Pyelonephritis** | UTI 上行性感染 | 发热 + flank pain + CVA tenderness + WBC casts |
| **Urothelial cancer** ⭐ | TCC（bladder / ureter / renal pelvis）/ SCC（schistosoma / chronic cystitis） | **Painless gross hematuria + clots + 老男 + 吸烟** |

---

## Part 3. Urothelial (Bladder) Cancer 经典指纹 ⭐⭐

> [!warning] 5 秒锁定 SOP
> **"老男 + 烟 + painless gross hematuria + clots"** 四要素 → **bladder cancer 直到证否，下一步 Cystoscopy**。

### 3.1 风险因素 7 大

| 因素 | 内容 |
|---|---|
| **Age >40** | 高峰 60-70 |
| **Male** | M:F ≈ 3:1 |
| **Smoking** ⭐ | **最重要可改变因素**，OR 3-4×；约占 50% 病例 |
| **Aromatic amines / Aniline dye** | 染料 / 橡胶 / 印刷工业 — benzidine / 2-naphthylamine |
| **Cyclophosphamide** | → acrolein → hemorrhagic cystitis + 长期 bladder ca；预防用 **mesna** |
| **Schistosoma haematobium** | 中东 / 北非 — 致 **SCC**（不是 TCC） |
| **Chronic indwelling catheter / chronic cystitis** | → SCC |
| **Pelvic radiation** | 既往 prostate ca / cervix ca 放疗史 |

### 3.2 Bladder cancer 表现谱

- **Painless total hematuria**（最经典 — 起源于膀胱体）
- **Terminal hematuria + clots**（起源于膀胱颈 / 三角区 — 本笔记触发题 Q4751）
- Irritative voiding symptoms（晚期）
- Cystoscopy 看到 **papillary mass** 或 **flat plaque**（in situ）

---

## Part 4. Cystoscopy 适应症 ⭐

| 类别 | 触发条件 |
|---|---|
| **Hematuria** | **Gross hematuria**（任何，需排除可逆原因）/ **Microscopic hematuria + 尿路恶性风险因素** |
| **Infection** | 反复 UTI / 抗生素无效 UTI |
| **下尿路症状** | Obstructive 症状疑狭窄 / 结石；Irritative 症状但无感染 |
| **其他疑恶性** | Abnormal bladder imaging（CT）/ Abnormal urine cytology |

> [!warning] ⚠️ 反 pattern — 不需要 cystoscopy
> **有 glomerular bleeding 证据**（RBC casts / dysmorphic RBC）→ 不做 cystoscopy；这是肾性血尿，做肾活检或肾科评估。

---

## Part 5. Clots（血块）反 pattern ⭐⭐

> [!danger] 黄金规则
> **有 clots → NOT 肾小球**（下尿路 / 上尿路非肾小球来源）。
> **无 clots + RBC casts + dysmorphic RBC + 蛋白尿** → 肾小球。

**机制**：肾小球疾病出血量小，且流经远端肾单元被尿激酶溶解 → 难形成块。

### 肾小球血尿 3 大标志

1. **RBC casts**（肾性 pathognomonic）
2. **Dysmorphic RBC**（变形红细胞）
3. **Proteinuria**（不同程度，常 <3.5 g）

---

## Part 6. 6 选 1 鉴别（Q4751 触发题）

| 病因 | 指纹 | 反 pattern / 钓饵识别 |
|---|---|---|
| **Bladder (Urothelial) Cancer** ⭐ | Terminal/Total hematuria + clots + 男 >40 + 吸烟 | 答案 |
| **Glomerular disease** | **RBC casts** + dysmorphic RBC + 蛋白尿；可 total / microscopic | **血块极罕见**（有血块即排除） |
| **Nephrolithiasis（结石）** | **Acute flank-groin colic（绞痛）** | **慢性背痛 ≠ renal colic**（chronic back pain 是干扰项） |
| **PCKD（ADPKD）** | Bilateral pain + bulky mass + total hematuria + 家族史 | 无 mass / 无家族史 ≠ ADPKD |
| **Urethral injury / urethritis** | **Initial hematuria** + irritative voiding（dysuria） | 不是 terminal |
| **UTI（pyelo / cystitis / urethritis）** | Pyelo: 发热 + flank + 系统症状 / Cystitis: dysuria + frequency / Urethritis: dysuria + 性病史 | 无 LUTS / 无发热即排除 |

---

## Part 7. Smoking 致癌三联（高频反射）⭐

吸烟同时是 **lung ca / bladder ca / RCC** 的最强可改变因素。

| 癌 | 吸烟 OR | 备注 |
|---|---|---|
| **Lung cancer** | 10-30× | 小细胞 / 鳞癌强相关 |
| **Bladder (urothelial) cancer** | 3-4× | 约 50% 病例归因 |
| **Renal cell carcinoma** | 1.5-2× | 较弱但显著 |

**反射**：任意"老男 + 烟 + 无解释 painless 症状" → 先想癌（lung / bladder / RCC 三选一）。

## Part 8. 工作流（下一步检查 SOP）⭐

```
Hematuria
  ├─ 确认是真性血尿（dipstick + UA 镜检红细胞）
  │     ├─ Hb 阳性 + 无红细胞 → myoglobin（rhabdo） / hemoglobin（hemolysis）
  │     └─ 红细胞阳性 → 真性血尿
  │
  ├─ 看分期 → 定位
  │     ├─ Initial → 尿道
  │     ├─ Terminal → 膀胱颈/三角区/前列腺
  │     └─ Total → 整路
  │
  ├─ 看 UA 沉渣
  │     ├─ RBC casts / dysmorphic RBC → 肾小球 → 肾科 + 活检
  │     ├─ WBC casts + 发热 → pyelo → 抗生素 + 培养
  │     └─ Bland + clots → 非肾小球 → 影像 + cystoscopy
  │
  └─ 有 cystoscopy 适应症？
        ├─ Gross hematuria 任何 → ✅ cystoscopy
        ├─ Microscopic + 风险因素 → ✅ cystoscopy
        └─ 仅 microscopic + 无风险 → 重复 UA + 监测
```

---

## Part 9. 反 pattern 黑名单 ⚠️

| 反 pattern | 内容 | 警示 |
|---|---|---|
| **Chronic back pain = 结石** | ❌ 结石痛是 **acute colic 急性绞痛** 非 chronic | Q4751 我选 C 的原因 |
| **任何血尿都做 cystoscopy** | ❌ **RBC casts 阳性**（肾小球证据）不做 cystoscopy | 改做肾活检 |
| **Smoking = 只想 lung ca** | ❌ Smoking 同时是 **bladder / RCC** 强风险 | 三联癌反射 |
| **Schistosoma → TCC** | ❌ Schistosoma **致 SCC**（不是 TCC） | 中东 / 北非旅行史 → SCC |
| **AKI + hyperCa → 急救** | ❌ AKI + hyperCa **MM until proven otherwise**（cast nephropathy） | 跨主题反 pattern |

## Part 10. 复习题（可折叠 Q&A）

> [!question]- Q1. 53yo M + 排尿末段红 + 血块 + 慢性背痛 + 吸烟 — 最可能病因？
> **Bladder (urothelial) cancer**（不是结石）。Terminal hematuria + clots + 男 >40 + 烟 = 经典指纹。下一步 cystoscopy。
> Chronic back pain ≠ renal colic（结石痛必急性绞痛）。

> [!question]- Q2. 血尿分期 3 联各对应哪个解剖部位？
> - **Initial → 尿道**（urethra）
> - **Terminal → 膀胱颈 / 三角区 / 后尿道 / 前列腺**
> - **Total → 整个泌尿道**（任何位置）

> [!question]- Q3. 看到血块是支持还是反对肾小球疾病？为什么？
> **反对**。肾小球出血量小 + 流经远端肾单元被尿激酶溶解 → 极少形成血块。肾小球血尿的 3 大标志是 RBC casts / dysmorphic RBC / 蛋白尿。

> [!question]- Q4. Bladder cancer 7 大风险因子？
> ① Age >40 ② Male ③ **Smoking**（最强可改变，~50% 病例）④ Aromatic amines（染料 / 橡胶 / 印刷）⑤ **Cyclophosphamide**（→ acrolein → mesna 保护）⑥ **Schistosoma haematobium**（致 SCC）⑦ Chronic indwelling catheter / chronic cystitis（致 SCC）⑧ Pelvic radiation。

> [!question]- Q5. Cystoscopy 不做的反 pattern 是什么？
> **有 glomerular bleeding 证据**（RBC casts / dysmorphic RBC）→ 改做肾科评估 / 肾活检，不做 cystoscopy。

> [!question]- Q6. Smoking 致癌三联是哪三个？
> **Lung / Bladder / RCC**（kidney）。任意"老男 + 烟 + 无解释 painless 症状"先在这三个里想。

> [!question]- Q7. Schistosoma haematobium 致哪一型膀胱癌？
> **Squamous cell carcinoma (SCC)** — 不是经典的 transitional cell carcinoma (TCC)。中东 / 北非 / 撒哈拉以南非洲旅行 / 居住史 = stem 钩子。

# 肾脏 — Renal Cyst Bosniak 分类 + 偶发病变管理

> [!info] 笔记定位
> Renal cyst Bosniak 分类是 USMLE Step 2 偶发肾病变管理的核心决策树。**Simple renal cyst（Bosniak I）**在 > 50 岁人群普及率 ~50%，几乎全部良性，无需 follow-up。"Reassurance only" 是高频正确答案，反过度医疗。本笔记同时覆盖 incidentaloma 元 pattern（跨 4 大：肾 / 肾上腺 / 甲状腺 / 肺 nodule）。来源：[[mistakes/uworld-mistakes#^Q4181]]（错题 2026-05-28，体系早建 v1）。

---

## Part 1. Simple Cyst vs Malignant Cystic Mass — 6 项鉴别 ⭐⭐⭐

> [!success] USMLE 高频对照表
> 任何"CT 偶发肾囊肿"题，6 秒扫这 6 项 → 良性 (reassurance) vs 可疑 (urology)。

| 特征 | **Simple cyst（良性）** | **Malignant cystic mass** |
|---|---|---|
| **壁 (Wall)** | 薄 / 光滑 / 规则 | **厚 / 不规则** |
| **腔数 (Locularity)** | **单腔** (unilocular) | **多腔** (multilocular) |
| **Septa** | 无 | **多 / 厚 / 可钙化** |
| **内容 (Content)** | 均质（液体）| **异质**（solid + cystic）|
| **Contrast enhancement** | **无** | **有** ⭐⭐⭐ 最关键 |
| **临床** | 无症状 | 痛 / 血尿 / HTN |
| **管理** | **Reassurance + 无需 follow-up** | 影像 + urology 评估 + 可能手术 |

### 关键单标志 ⭐
**Contrast enhancement = 恶性最关键单标志** —— 任何 enhancement 都让 Bosniak 升级 → 至少 IIF 起跳 / III+ 必转 urology。

![[{59569AB0-1C81-4842-B468-E7B1D50EA3A4}.png]]
*↑ Bosniak I 简单囊肿典型 CT 冠状位（来源 [[mistakes/uworld-mistakes#^Q4181]]）：圆形低密度灶 + 薄壁 + 无内分隔*

![[{D6D5097C-96A8-4A3D-A27B-6DF5F1C2870F}.png]]
*↑ Hydronephrosis 对照（**非囊肿** — 是肾盂扩张 / ureter 梗阻征）：仅此情况才考虑 ureteral stent placement；不要与 simple cyst 混淆*

## Part 2. Bosniak 5 级分类（管理决策树）⭐⭐⭐

```
Bosniak Class I
    薄壁 / 无 septa / 无 enhancement / 均质 / 单腔
    → 良性 → 无需任何 follow-up ✅
    [本题 Q4181 所处]
                ↓
Bosniak Class II
    细薄 septa（≤ 1 mm）+ 微小钙化 + 无 enhancement
    → 良性 → 无需 follow-up ✅
                ↓
Bosniak Class IIF（"F" = Follow）
    Borderline（多 thin septa / 较厚钙化）+ 无 enhancement
    → 不确定 → 6-12 月 follow-up 影像观察变化
                ↓
Bosniak Class III
    厚壁 / 不规则 septa / 厚 enhancement
    → 可疑恶性 ~50% → Partial nephrectomy 手术
                ↓
Bosniak Class IV
    明显 enhancement + solid 成分
    → 恶性高度可能（>90% RCC）→ Partial / Radical nephrectomy
```

### 速查记忆 ⭐
- **I-II 无需 → IIF 短期随访 → III-IV 手术**
- "F" = **Follow**（IIF 名字本身就是随访的提示）
- Enhancement 0 → Class I-II；轻度 → IIF；明显 → III-IV

## Part 3. 偶发病变 (Incidentaloma) 4 大处理原则 ⭐

| 影像特征 | 临床 | 处理 |
|---|---|---|
| **良性** + 无症状 | 无 | **Reassurance + 无 follow-up** ⭐（USMLE 反过度医疗陷阱）|
| **Borderline** | 无 / 模糊 | 短期 follow-up 影像（6-12 月）|
| **可疑恶性** | 任意 | Urology / oncology + 进一步 imaging / biopsy / 手术 |
| **症状性大囊肿 / 感染** | 痛 / 发热 / 感染 | Aspiration / marsupialization / 抗生素 |

> [!tip] 偶发病变元 pattern 跨 4 大 ⭐⭐
> 同样的"良性影像 + 无症状 = reassurance"框架适用：
> - **肾**：Simple renal cyst（本笔记）
> - **肾上腺**：Adrenal incidentaloma < 4 cm + 无激素活性 → reassurance + 6 月随访
> - **甲状腺**：Thyroid nodule < 1 cm + 良性 US → 无需 FNA
> - **肺**：Solitary pulmonary nodule < 6 mm + 低危 → 无需 follow-up

## Part 4. 易混陷阱（普适）⭐

| 选项 | 适用场景 | 不适用 simple cyst |
|---|---|---|
| **Antibiotics** | UTI / 脓肿 | 无 fever / UA 正常 ❌ |
| **Percutaneous aspiration** | **大而有症状** / 感染脓肿 / 难鉴别诊断 | 无症状 + 良性影像 ❌ |
| **Reassurance only** ⭐ | **简单囊肿 + 无症状** | ✅ |
| **Surgical excision / nephrectomy** | Bosniak III-IV / RCC | 良性 ❌ |
| **Ureteral stent** | 尿路梗阻（hydronephrosis / hydroureter / stone）| 无梗阻 ❌ |

### 反 pattern 警示
- **"看到肾囊肿 → 反射 aspiration / 切除"是错的** — Simple cyst 无需任何介入
- **过度介入有风险**：aspiration → 感染 / 出血 / 复发；surgery → 不必要风险
- **AAA 家族史 + 非吸烟** = 干扰背景，不影响囊肿管理
- **Ureteral stent 仅梗阻时用** — 任何"无 hydronephrosis"的肾病变 stent 都是错的
- **Bosniak IIF "F" = Follow 不是 Forget** — 真正 Follow up 6-12 月

## Part 5. Bosniak III/IV → RCC 鉴别考点（延伸）

### Renal Cell Carcinoma (RCC) 经典 stem 触发
- **年龄 > 50 + 吸烟 / 肥胖 / HTN / 慢性血透**
- **三联征**（< 10%）：flank pain + 血尿 + 腰部肿块
- **副肿瘤综合征**：
  - **EPO ↑** → 红细胞增多症（与贫血相反）
  - **PTHrP ↑** → 高钙血症
  - **ACTH-like / renin-like** → 罕见
- **Stauffer 综合征**：肝功异常 + 转氨酶↑ 无肝转移
- **左睾静脉曲张突现** = 左肾静脉血栓（RCC 侵 IVC）
- **诊断**：CT with contrast → biopsy 或直接 partial / radical nephrectomy

## Part 6. Memory Hook

> **"简单肾囊肿 = 良性偶发 + Reassurance only + 无需 follow-up"** ⭐⭐⭐
>
> **Simple cyst 5 联**：薄壁 / 单腔 / 无 septa / 均质 / **无 enhancement**
>
> **Bosniak 5 级**：I-II 无需 → **IIF "F"=Follow** → III-IV 手术
>
> **Enhancement = 恶性最关键单标志** ⭐⭐⭐
>
> 反 pattern：**偶发 + 良性 + 无症状 = 不做任何事**（反过度医疗）
>
> 元 pattern：**Incidentaloma 4 大跨学科**（肾 / 肾上腺 / 甲状腺 / 肺 nodule）

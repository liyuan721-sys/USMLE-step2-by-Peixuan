> [!info] 笔记定位
> 利尿剂（Diuretics）横跨**肾 / 心内 / 电解质 / 酸碱**的高频药理核心。本笔记按**作用位点**串五大类，配电解质指纹 + 适应证/禁忌/副作用 + **利尿剂↔遗传病镜像** + 致酸碱紊乱机制。
> 纳入错题：[[mistakes/uworld-mistakes_2026-05#^Q2826]] 肝硬化袢利尿 → 收缩性代碱（低 K + 低 Cl）。

# 利尿剂全总结（Diuretics）

## 一、五大类全景表 ⭐（位点 → 靶点 → 电解质指纹）

| 类 | 代表药 | 位点 | 靶点 | K | 其他电解质 / 酸碱 | 一句话标签 |
|---|---|---|---|---|---|---|
| **碳酸酐酶抑制剂 CAI** | acetazolamide | 近端 PCT | 碳酸酐酶 | ↓ | **NAGMA（丢 HCO3，Type 2 样）** + 碱化尿 | "丢碱利尿" |
| **袢利尿** | furosemide / bumetanide / torsemide / ethacrynic acid | 髓袢升支粗段 | **NKCC2**（Na-K-2Cl）| **↓** | **↓Ca ↓Mg ↑尿酸**、**代碱** | "最强 + 丢钙" |
| **噻嗪** | HCTZ / chlorthalidone / metolazone | 远曲小管 DCT | **NCC**（Na-Cl）| **↓** | **↑Ca**（保钙）、**↓Na ↓Mg ↑尿酸 ↑糖 ↑脂**、**代碱** | "保钙 + 低钠" |
| **保钾（集合管）** | spironolactone / eplerenone（醛固酮拮抗）；amiloride / triamterene（ENaC 阻断）| 集合管 CD | 醛固酮受体 / ENaC | **↑** | **NAGMA（Type 4 样）**、↓尿 Ca | "升钾 + 抗醛固酮" |
| **渗透性** | mannitol | 近端 + 髓袢（全程）| 渗透梯度 | — | 早期**稀释性低钠**、晚期可**高渗** | "拉水" |

> [!tip] 一图记位点（从近到远）
> ```
> 近端 PCT ──── 髓袢升支 ──── 远曲小管 DCT ──── 集合管 CD
>   │              │              │                │
> acetazolamide        袢(NKCC2)      噻嗪(NCC)       保钾(醛固酮/ENaC)
> 丢HCO3         最强,丢Ca       保Ca,低钠        升钾,Type4样
> NAGMA          代碱            代碱             NAGMA
> ```

## 二、电解质 / 酸碱方向速记（最易混的三组对比）⭐

### 2.1 钙的方向：袢 vs 噻嗪相反 ⭐⭐

> [!warning] UW 最爱的对比
> - **袢 → 丢钙（↓Ca）**：抑制 NKCC2 → 髓袢 K 回漏减少 → lumen 正电位消失 → **Ca/Mg 顺电位的重吸收消失** → 尿钙↑ → 用于**治高钙血症**
> - **噻嗪 → 保钙（↑Ca）**：DCT 阻 NCC → 细胞内 Na↓ → 增强基底侧 Na/Ca 交换 → **钙重吸收↑** → 尿钙↓ → 用于**钙性肾结石、骨质疏松**，但会**暴露/加重高钙血症**（如甲旁亢）
>
> 一句话：**"袢丢钙治高钙；噻嗪保钙防结石（但查出高钙别用）。"**

### 2.2 酸碱方向：谁致代碱、谁致 NAGMA

| 利尿剂 | 酸碱后果 | 机制 |
|---|---|---|
| **袢 / 噻嗪** | **代谢性碱中毒** ⭐ | 排 Na/Cl/水 → **收缩性碱（contraction alkalosis）** + 容量收缩激活 RAAS → 醛固酮↑ → 远端排 H⁺/K⁺↑ → 低 K + 低 Cl 代碱 |
| **acetazolamide** | **NAGMA（Type 2 RTA 样）** | 抑碳酸酐酶 → 近端 HCO3 重吸收障碍 → 丢 HCO3 → 高氯性正常 AG 代酸 |
| **保钾（spironolactone/amiloride）** | **NAGMA（Type 4 RTA 样）** | 拮抗醛固酮 → 远端排 H⁺/K⁺↓ → 高钾 + 正常 AG 代酸 |

> [!example] Q2826 实战（[[mistakes/uworld-mistakes_2026-05#^Q2826]]）
> 肝硬化腹水患者用**袢利尿**：HCO3 19→**31↑** + **K 4.6→3.1↓** + **血 Cl 101→92↓** = **一高两低（HCO3↑ / K↓ / Cl↓）= 利尿剂收缩性代碱**。
> ⚠️ 这里的 **Cl 是血氯**（血指纹的一部分），**不是尿氯**。本题病因（腹水利尿）已由病史直接给出，**无需查尿 Cl 分型**。
> 坑点：患者"插管机械通气" → 易反射选"过度通气（呼碱）"；但**呼碱 HCO3 应低**（肾代偿排 HCO3），本题 HCO3=31 高 → 铁证代碱。**判原发先看 HCO3 方向，别被"插管"字眼带走。**

### 2.3 钾的方向：四类排钾、一类保钾

- **排钾（低钾）**：CAI、袢、噻嗪（远端 Na 增多 → 钠重吸收驱动 K 排泄↑ + 醛固酮↑）
- **保钾（高钾）**：spironolactone / eplerenone / amiloride / triamterene
- **联用逻辑**：噻嗪/袢（丢钾）+ 保钾药 → **抵消低钾**（临床常配伍）

## 三、利尿剂 ↔ 遗传病镜像 ⭐⭐（转运体方向题高频）

每个利尿剂靶点都有一个"基因把它敲掉/敲强"的遗传病，表现 = 长期吃该药 / 反向：

| 利尿剂（靶点）| 失活 = 像吃药（遗传病）| 表现 | 反向（靶点过强）|
|---|---|---|---|
| **袢（NKCC2）** | **Bartter 综合征** | 低 K + **低 Ca（尿钙↑）** + 代碱 + 正常/低血压 | — |
| **噻嗪（NCC）** | **Gitelman 综合征** | 低 K + **低 Mg** + **低尿钙** + 代碱 + 正常血压 | **Gordon / PHA2**（NCC 过强）→ 高 K + 高血压 + NAGMA + 低肾素 → **噻嗪特效** |
| **保钾（ENaC）** | — | — | **Liddle 综合征**（ENaC 过强）→ 低 K + 代碱 + **高血压** + 低肾素低醛 → **amiloride特效**（spironolactone无效！）|

> [!tip] 镜像三钩
> - **Bartter = 吃袢一辈子**（低钙）；**Gitelman = 吃噻嗪一辈子**（低镁 + 低尿钙）
> - **Gordon = 噻嗪特效**（NCC 过强）；**Liddle = amiloride特效**（ENaC 过强，**不是spironolactone** —— Liddle 醛固酮低，受体没问题，得直接堵 ENaC）
> - 记钙鉴别：**Bartter 尿钙高（似袢），Gitelman 尿钙低（似噻嗪）** —— 和母体药方向一致

## 四、每类适应证 / 禁忌 / 标志性副作用

| 类 | 主要适应证 | 标志性副作用 / 禁忌 |
|---|---|---|
| **acetazolamide** | 青光眼、**高山病预防**、碱化尿（某些中毒/尿酸）、代碱矫正 | NAGMA、低钾、肾结石（磷酸钙）、磺胺过敏；**肝硬化慎用**（碱化致氨↑→脑病）|
| **袢** | **急性肺水肿 / 心衰**、**高钙血症**、容量过载、高钾辅助 | **耳毒性**（剂量相关，ethacrynic acid最重）、低钾低钙低镁、代碱、高尿酸（痛风）、磺胺过敏 |
| **噻嗪** | **一线降压**、钙性肾结石、骨质疏松、**肾性尿崩（矛盾性抗利尿）** | **低钠（经典！老年女性）**、低钾低镁、高钙、高尿酸、高血糖高血脂、磺胺过敏 |
| **spironolactone** | **心衰（降死亡率）**、Conn、肝硬化腹水、抗雄（多毛/痤疮）| **高钾**、**男性乳房发育 / 性功能障碍**（eplerenone无此副作用）、Type 4 样酸中毒 |
| **amiloride / triamterene** | 配伍防低钾、Liddle（amiloride）、锂致肾性 DI | 高钾、Type 4 样酸中毒 |
| **mannitol** | **脑水肿（↓ICP）**、急性青光眼 | **心衰 / 肺水肿禁用**（早期扩容）、稀释性低钠、高渗 |

> [!warning] 磺胺交叉过敏 ⭐
> **acetazolamide / 袢（furosemide）/ 噻嗪**都是**磺胺类衍生物** → 磺胺过敏者慎用。**例外：ethacrynic acid 不含磺胺** → 磺胺过敏患者需要袢利尿时的替代。

### 4.1 袢利尿剂耳毒性（深入）⭐

> [!danger] 袢利尿剂耳毒性：什么时候考、长什么样
> **表现**：**双侧对称性高频感音神经性聋 SNHL**（典型从高频起）± 耳鸣 / 平衡障碍；多**可逆**（停药后），但可不可逆。
> **高危三情境**（命题钩，满足越多越像）：
> - **肾衰 / CKD**（药物清除↓，内耳暴露↑）
> - **高剂量**（如反复心衰失代偿住院的大剂量呋塞米 furosemide）
> - **合用其他耳毒药**（叠加效应）
> **药内排序**：**ethacrynic acid 依他尼酸最重**（也是磺胺过敏替代袢）。
> 处理：听力图 + 尽量停/换药。

> [!warning] 耳毒"四药" + 抗生素陷阱
> 耳毒四药：**氨基糖苷 aminoglycoside / 顺铂 cisplatin / 大剂量水杨酸 salicylate / 袢利尿剂 loop diuretic**。
> ⚠️ **抗生素里只有氨基糖苷耳毒；氟喹诺酮（环丙沙星 ciprofloxacin）不耳毒**。命题常用"近期治感染（前列腺炎等）"把你诱向环丙沙星 → 真凶其实是反复心衰的大剂量袢利尿（[[mistakes/uworld-mistakes_2026-06#^Q2838]]）。

## 五、几个高频"反直觉"点

> [!tip] 反直觉清单
> - **噻嗪治肾性尿崩（nephrogenic DI）**：矛盾性——利尿剂反而减尿量。机制：轻度容量收缩 → 近端 Na/水重吸收↑ → 到达远端的水↓ → 尿量↓。
> - **袢"天花板"效应低**：袢是高效能（high-ceiling），肾功能差时仍有效；噻嗪在 **eGFR < 30** 时基本无效（除 metolazone）。
> - **心衰利尿抵抗**：袢 + 噻嗪（如 metolazone）**序贯阻断** → 协同（sequential nephron blockade）。
> - **spironolactone起效慢**（基因转录层面，数天）；急性利尿靠袢。
> - **eplerenone vs spironolactone**：eplerenone选择性醛固酮拮抗，**无抗雄副作用**（男性乳房发育时换它）。

## 六、考前一句话锁定 ⭐

> [!success] Memory Hooks
> 1. **位点助记**：acetazolamide(近端丢碱)→ 袢(髓袢最强丢钙)→ 噻嗪(DCT 保钙低钠)→ 保钾(集合管升钾)
> 2. **钙方向**："袢丢钙治高钙，噻嗪保钙防结石"
> 3. **酸碱**："袢/噻嗪→收缩性代碱(一高两低 HCO3↑K↓Cl↓)；acetazolamide→NAGMA;保钾→Type4样高钾酸"
> 4. **镜像**："Bartter=吃袢(低钙)，Gitelman=吃噻嗪(低镁低尿钙)；Gordon 噻嗪特效，Liddle amiloride特效"
> 5. **磺胺**："acetazolamide/袢/噻嗪含磺胺，ethacrynic acid不含(替代)"
> 6. **eplerenone无男性乳房发育**

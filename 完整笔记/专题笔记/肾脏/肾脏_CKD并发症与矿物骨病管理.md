# CKD 并发症与矿物骨病管理（含饮食）

> [!info] 起源
> 派生自 NBME11 Q138（12 岁慢性肾衰 → 防骨病/继发甲旁亢 → **限磷**）。兑现 [[完整笔记/专题笔记/肾脏/肾脏_HighYield速查总览]] 版本记录里"CKD 分期与并发症待积累错题后单建"的 TODO。本笔记把**慢性肾脏病 (chronic kidney disease, CKD)** 的代谢并发症链 + 饮食管理作为复习单一真相源。

---

## §一 一句话锁定

> [!tip]
> CKD 并发症问"如何减少 / 长期预防"→ 高收益答案是**限磷 (restrict phosphorus)**，因为 CKD **排磷受损 → 高磷血症 → 继发性甲状旁腺功能亢进 → 肾性骨病**。

---

## §二 CKD 矿物骨病链（CKD-MBD）⭐

> [!info] CKD-MBD = chronic kidney disease–mineral and bone disorder（CKD 矿物质与骨代谢紊乱）

```text
慢性肾脏病 (CKD)
   │  ① 排磷↓
   ▼
高磷血症 (hyperphosphatemia)
   │  磷与钙结合 → 血钙↓；且 1,25-(OH)₂-维生素 D 合成↓ → 肠吸收钙↓
   ▼
低钙血症 (hypocalcemia)
   │  刺激甲状旁腺
   ▼
继发性甲状旁腺功能亢进 (secondary hyperparathyroidism, 继发甲旁亢)
   │  PTH↑ 持续动员骨钙
   ▼
肾性骨病 (renal osteodystrophy) + 血管钙化 (vascular calcification)
```

> [!success] 阻断点 = 限磷
> 饮食**限磷** ±**磷结合剂 (phosphate binders)** 是打断这条链的高收益第一步；同时监测 Ca / P / PTH，必要时活性维生素 D / 拟钙剂。

## §三 CKD 饮食管理对照表 ⭐

| CKD 并发症 | 机制 | 饮食方向 |
|---|---|---|
| 高磷血症 | 排磷↓ | **限磷** ✅（NBME11 Q138 考点）|
| 高钾血症 (hyperkalemia) | 排钾↓ | 限钾 |
| 容量超负荷 (volume overload) | 排液↓ | 限液（**别"多喝水洗肾"**）|
| 儿童生长障碍 | 慢病 / 营养 | **不盲目限热量**（保生长）|
| 贫血 (anemia) | 促红细胞生成素 (erythropoietin, EPO)↓ | 补铁 / 红细胞生成刺激剂 (ESA)，**非维生素 B12 (cobalamin)** |
| 代谢性酸中毒 | 排酸/产 HCO₃⁻↓ | 碳酸氢钠 (sodium bicarbonate) 补充 |

> [!danger] 四大钓饵（NBME11 Q138 干扰项）
> ① **多喝水**：CKD 易容量超负荷，有水肿时反而限液。
> ② **补钾**：CKD 怕高钾，应限钾。
> ③ **限热量**：生长期儿童会伤发育，不限。
> ④ **补维生素 B12 (cobalamin)**：CKD 贫血主因是 **EPO 不足**（正细胞正色素），不是 B12 缺乏的巨幼细胞性贫血。

### §3.1 限磷饮食 (phosphorus restriction) 具体怎么做 ⭐

> [!info] 定义
> **限磷饮食 = 限制每日磷摄入**（CKD 患者目标常 ~800–1000 mg/日），打断"高磷 → 低钙 → 继发甲旁亢 → 肾性骨病 + 血管钙化"链的饮食基础。

| 高磷、要限 | 低磷、相对安全 |
|---|---|
| **乳制品**（牛奶 / 奶酪 / 酸奶）| 多数新鲜蔬菜水果 |
| **动物内脏、蛋黄** | 蛋清 |
| **坚果、豆类、全谷物** | 精制米面 |
| **深色碳酸饮料**（可乐含磷酸）| 白色 / 透明饮料 |
| **加工食品 / 含磷酸盐添加剂** ⭐ | 新鲜未加工食物 |

> [!tip] 高频要点：无机磷 vs 有机磷
> 食品添加剂里的**无机磷（磷酸盐防腐剂）吸收率近 100%**，远高于天然食物的**有机磷（吸收 40–60%）**。所以限磷重点是**少吃加工食品**，不只是少吃天然高磷食物——这是营养考点。

> [!warning] 限磷 ≠ 单靠饮食 —— 三管齐下
> ① **限磷饮食**（基础）；② **磷结合剂 (phosphate binder)** 随餐服，在肠道结合磷阻止吸收——**首选非钙类**（司维拉姆 sevelamer / 碳酸镧 lanthanum），避免含钙结合剂加重血管钙化；③ **充分透析**清除磷。

## §四 CKD 并发症速查（一图全）

```text
CKD 主要并发症（"水电酸 + 贫骨"）
   ├─ 水：容量超负荷 / 高血压 → 限液 + 控压（如 ACEI/ARB）
   ├─ 电：高钾 → 限钾
   ├─ 酸：代谢性酸中毒 → 补碳酸氢钠
   ├─ 贫：贫血（EPO↓）→ 补铁 + ESA
   └─ 骨：高磷 → 继发甲旁亢 → 肾性骨病 → 限磷 + 磷结合剂 + 活性维 D
```

# 水杨酸中毒 salicylate toxicity（急性 vs 慢性）

> [!info] 用途
> 水杨酸中毒高频且易误判（像 NMS/脓毒症/DKA）。一张图锁**识别三联 + 急慢性差别 + 血透指征 + "为什么不用利尿/乙酰唑胺"**。
> 错题地基：[[mistakes/uworld-mistakes_2026-06#^Q18925]]（急性识别，误选丹曲林）+ [[mistakes/uworld-mistakes_2026-06#^Q19677]]（慢性 + 肾衰肺水肿 → 血透，误选甘露醇）。

---

## §零 一句话锁死 ⭐

> [!danger] 识别 + 处理
> **高热 + 过度通气 + 耳鸣 + 高阴离子间隙代谢性酸中毒 (high anion-gap metabolic acidosis) = 水杨酸**。诊断 = **血清水杨酸水平 serum salicylate level**；治疗 = 碱化（碳酸氢钠 sodium bicarbonate）±活性炭，**重症/肾衰/肺水肿 → 血液透析 hemodialysis**。

---

## §一 识别：经典三联 + 指纹

```
水杨酸 salicylate（阿司匹林 aspirin / 水杨酸甲酯 methyl salicylate 冬青油）
   │
   ├─ 刺激延髓呼吸中枢 → 过度通气 → 呼吸性碱中毒 respiratory alkalosis
   ├─ 解偶联氧化磷酸化 → 产热 → 高热 + 干扰 Krebs → 乳酸↑ → 高 AG 代谢性酸中毒
   └─ 经典伴随：耳鸣 tinnitus、恶心呕吐、AMS（神经糖↓/脑水肿）
```

> [!tip] 指纹 = 混合性酸碱
> **呼吸性碱中毒 + 高 AG 代谢性酸中毒并存**（成人最典型）。再加**高热 + 过度通气 + 耳鸣** → 几乎锁定水杨酸。

---

## §二 急性 vs 慢性对比 ⭐

| 维度 | 急性 acute | 慢性 chronic |
|---|---|---|
| 人群 | 自杀/误服，单次大量 | **老人 / 幼儿**；外用水杨酸甲酯、长期用阿司匹林 |
| 病史 | 明确近期大量摄入 | **常无明确史**，隐匿 2 周渐进 |
| 血药水平 | 高 | **可仅轻升甚至正常**（诊断更难）|
| 表现 | 呕吐/耳鸣/过度通气/高热 | 不典型：**意识混乱 + 脱水**，易误诊 |
| 并发 | — | **非心源性肺水肿**、肾衰更常见 |

> [!warning] 慢性最坑
> 老人"意识混乱 + 耳鸣 + 外用冬青油/关节痛长期用药" → 想慢性水杨酸。**血药水平正常不能排除**。

---

## §三 治疗 + 血透指征 ⭐

```
轻-中：① 活性炭 activated charcoal（2h 内、清醒者）
       ② IV 碳酸氢钠 sodium bicarbonate → 碱化血+尿，促水杨酸排泄（需大量补液）
       ③ 补钾（碱化需正常血钾）
重症 → 血液透析 hemodialysis
```

> [!danger] 血透 hemodialysis 指征（高频）
> - **肾衰 / ESRD**（清除差 + 不耐受大量碳酸氢钠补液）
> - **肺水肿 pulmonary edema**（无法耐受补液）
> - 严重摄入：休克、癫痫、显著 CNS 障碍、脑水肿
> - 顽固酸中毒 / 补碳酸氢钠后仍恶化
> - 血药水平极高

### 3.1 三个"别"（错题暴露点）

> [!warning] 别用这些
> - ❌ **甘露醇 mannitol / 呋塞米 furosemide**（Q19677 我选甘露醇）：水杨酸肺水肿是**非心源性（毛细血管渗漏）**，非容量超负荷；患者反而**容量不足**（呕吐/高热）→ 利尿无益甚至有害
> - ❌ **乙酰唑胺 acetazolamide**：虽碱化尿，但丢碳酸氢根 → **降血 pH** → 更多水杨酸以非解离形式入脑（**加重 CNS 毒性，禁用**）
> - ❌ 意识障碍者用活性炭：误吸风险

## §四 易混鉴别

| 误判为 | 区分钥匙 |
|---|---|
| **NMS（神经阻滞剂恶性综合征 neuroleptic malignant syndrome）/ 恶性高热 MH** | 水杨酸**无肌强直**、有高 AG 代酸 + 过度通气；NMS 铅管样强直、MH 麻醉场景（Q18925 我误选丹曲林 dantrolene）|
| **脓毒症 sepsis** | 水杨酸有耳鸣 + 混合酸碱；查血清水杨酸 |
| **DKA（糖尿病酮症酸中毒 diabetic ketoacidosis）/ 其他高 AG 代酸** | MUDPILES 记忆；水杨酸有呼吸性碱中毒并存 + 耳鸣 |
| **一氧化碳 CO (carbon monoxide) 中毒** | CO 无高热，查碳氧血红蛋白 carboxyhemoglobin |

---

## §五 Memory Hook

> [!success] 锁定句
> **"烧（高热）+ 喘（过度通气）+ 鸣（耳鸣）+ 酸（高 AG 代酸）= 水杨酸 → 查血清水杨酸。"**
> 治疗："**碱化促排，肾衰/肺水肿就上透析**；别利尿/甘露醇（非容量超负荷）、别乙酰唑胺（加重入脑）。"
> 慢性版："**老人 + 冬青油/长期阿司匹林 + 意识混乱 + 耳鸣**，水平正常也别放过。"

## 一、AKI 分类伞 + 5 兄弟定位

```
                          AKI (急性肾损伤 = Cr ↑ ≥ 0.3)
                                    │
            ┌───────────────────────┼────────────────────────┐
            ▼                       ▼                        ▼
       Prerenal              Intrinsic (内源)            Postrenal
       灌注不足                  ↓                        梗阻
   (脱水 / 出血 / HRS              4 个解剖层               (BPH/结石/肿瘤/血块)
   / 药物功能性)                    │
                       ┌───────────┼───────────┬──────────────┐
                       ▼           ▼           ▼              ▼
                    Tubular   Interstitial   Glomerular    Vascular
                       │           │             │             │
                       ▼           ▼             ▼             ▼
                     ★ATN★     ★AIN★          GN          血管炎/RAS/胆固醇栓塞
                   小管坏死   间质过敏        肾小球              + 髓质/乳头层
                                                                   │
                                                                   ▼
                                                            ★乳头坏死★
                                                       慢性版 → Analgesic neph (CTIN+乳头坏死)
```

> [!tip] 5 兄弟核心 1 句话
> - **ATN** = 肾小管细胞被毒 / 缺血**直接杀死**（4 子型见 §二.1）
> - **AIN** = 间质对**药物过敏**（IV 型超敏）
> - **乳头坏死** = 肾髓质尖端**血供差**被饿死（NSAID 助记 §二.5）
> - **Analgesic nephropathy** = 慢性 NSAID/APAP/aspirin → CTIN + 乳头坏死 双打
> - **CTIN** = AIN 的慢性版

## 二、Intrinsic AKI 详细病因谱（按机制 + 时间窗）

### 2.1 ATN 4 大病因细分

| 亚型 | 经典原因 | 时间窗 | UA / Lab |
|---|---|---|---|
| **缺血性 ATN** | 休克 / 大出血 / 大手术 / 脱水未纠正 / 主动脉夹层 | 数小时-1 天 | Muddy brown casts + ↑ FENa (>2%) |
| **毒物性 ATN（外源）** | **造影剂 / 氨基糖苷 / Amphotericin B / 顺铂 / 万古霉素 / Cidofovir / Foscarnet** | 见 §三 各药 | Muddy brown casts |
| **色素性 ATN（内源）** | **Rhabdomyolysis (myoglobin) / 大溶血 (hemoglobin)** | 数小时 | **尿色深红/棕** + UA **dipstick blood ✓ 但镜下无 RBC** ★ |
| **结晶 / 蛋白性** | **MM light chain / TLS 尿酸 / 乙二醇 草酸钙 / MTX / 阿昔洛韦 / 磺胺 / 茚地那韦** | 数小时-数天 | 镜下见结晶（菱形草酸钙 / 砖红色尿酸 / 针形 MTX）|

### 2.2 AIN 药物谱（≥ 70% 病因是药物）— "**SAMPN**" 助记

> [!info] 5 大类药
> - **S**ulfonamide / 磺胺
> - **A**ntibiotics: **β-lactam（penicillin / cephalosporin）** / Rifampin / Vancomycin / Ciprofloxacin
> - **M**TX / **Allopurinol** ★（Q4007 触发）/ 西咪替丁
> - **P**PIs（omeprazole, pantoprazole）★ — CK 高频
> - **N**SAIDs（慢性）

| 项 | 内容 |
|---|---|
| 时间窗 | 用药后 **5 天-3 周**（IV 型超敏需时间）|
| 典型三联 | **皮疹 + 发热 + 关节痛** ★（但只有 10-30% 病人全具备）|
| UA | **WBC casts + 嗜酸尿 + 蛋白少**（与 GN 区别）|
| 血 | **外周嗜酸 ↑**（不一定全有）|
| 治疗 | **停药**（首要！）± 类固醇（重症 / 不缓解时）|
| 详读 | [[完整笔记/专题笔记/肾脏/肾脏_间质性肾炎AIN_CTIN对照]] |

### 2.3 血管层病因

| 层级 | 病因 | 关键特征 |
|---|---|---|
| **大血管** | RAS（动脉粥样硬化 / FMD）/ **Transplant RAS** ★ Q16323 / 主动脉夹层 / RVT（肾静脉血栓）| RVT: 肾病综合征 + 腰痛 + 血尿；RAS + ACE-I → Cr ↑ |
| **小血管** | **胆固醇栓塞** ★ / TTP-HUS / 血管炎 (GPA/MPA/PAN) / 恶性高血压 / 妊高 | 胆固醇栓塞 = post-procedure 1-8 wks + livedo + 嗜酸 |
| **灌注劫持** | **HRS** ★ Q4752 / **CNI 急性毒** ★ Q16485 | HRS = 肝硬化 + 扩容无应答；CNI = afferent 收缩 + 血药↑ |

### 2.4 肾小球层 GN（简表，详读 [[完整笔记/Peixuan分科笔记/肾脏]]）

| 类型 | 经典 stem |
|---|---|
| **RPGN** | ANCA 血管炎 / Goodpasture / 狼疮 / IgA 重症 |
| **PSGN** | 链球菌感染后 1-3 wk |
| **IgA** | URI 同日血尿（"synpharyngitic"）|
| **MPGN** | HCV + cryoglobulin |

### 2.5 髓质 / 乳头层 — 乳头坏死 "**NSAID**" 5 大病因

> [!success] 神助记 — "NSAID" 本身就是 5 因首字母
>
> | 字母 | 病因 |
> |---|---|
> | **N** | **NSAIDs** 慢性使用（PG 抑制 → 髓质血供 ↓）|
> | **S** | **Sickle cell** disease（HbS 在低氧高渗髓质沉淀阻血流）★ |
> | **A** | **Acute pyelonephritis**（重症，感染性血管炎症）|
> | **I** | **obstrucIon / Infection** |
> | **D** | **Diabetes mellitus**（微血管病变累及髓质血供）|

```
正常肾髓质：本就低氧 + 高渗（浓缩尿用）
   → 髓质血供本就脆弱（vasa recta 长袢稀疏）
       │
       ▼ 任何"减少髓质血流 + 增加缺血负担"事件
   N + S + A + I + D 任一
       │
       ▼
   肾乳头（最远端、最缺血）→ 缺血性坏死
       │
       ▼
   坏死的乳头脱落 → 进入尿液（"sloughed papilla"）
       │
       ▼
   症状：腰痛 + 肉眼血尿 + 可能尿出"组织块"
```

**Analgesic Nephropathy = 慢性版双打**（CTIN + 乳头坏死） — Q4004 触发：长期 OTC APAP + NSAID/aspirin → 双侧小肾 + WBC casts + 慢性肾衰 + 贫血

![[{37FC27AE-0625-494A-B16D-CF569F58DC2E}.png]]

> 图同步自 [[mistakes/uworld-mistakes_2026-05#^Q4004]]（机制 = APAP 耗竭 glutathione + NSAID/Aspirin 氧化损伤 → CTIN ± papillary necrosis；UA = WBC casts ± 蛋白尿 + 有 papillary necrosis 时 RBC；影像 = 双侧小肾 + papillary calcification + 不规则轮廓）

> [!warning] 镰刀贫 + NSAID 双高危
> 镰刀贫病人**禁用 NSAID** 治疼痛危象 — 否则乳头坏死加重
> 治疼痛用 acetaminophen / opioid / hydroxyurea 预防

## 三、★★★ 特殊病因深读（9 个）

### 3.1 造影剂肾病 (Contrast-Induced AKI / CIN) — Q4491 触发

| 项 | 内容 |
|---|---|
| **机制** | ① 直接小管毒（高渗造影剂）② 髓质血管收缩（缺血）③ 自由基 |
| **时间窗** | **24-48 h 内 Cr 升** ★，3-5 天达峰，1-2 周恢复 |
| **UA** | Muddy brown casts（典型 ATN）|
| **高危** | **CKD (eGFR <60)** / DM / 老年 / 心衰 / 多骨髓瘤 / 容量不足 / 大剂量造影 |
| **预防（最重要）** ★ | **术前/术中 IV 0.9% NS 水化**（围手术扩容，最有效证据）/ 用 iso-osmolar 或 low-osmolar 造影剂 / 停 NSAID / 限造影量 |
| **N-acetylcysteine** | **证据弱**，不再常规推荐 |
| **Hold metformin** ★ | 造影**前后 48 h 停** — 防 **乳酸酸中毒**（不是防 AKI！）★ Q4491 陷阱点 |

> [!danger] Q4491 元错误
> "Hold metformin" 选项 = **防乳酸酸中毒**，不是防 CI-AKI 本身。
> 真预防 = **IV NS 水化**

### 3.2 氨基糖苷肾毒 (Aminoglycoside Nephrotoxicity)

| 项 | 内容 |
|---|---|
| **机制** | 在近曲小管细胞内累积 → 毒性 ATN |
| **时间窗** ★ | 用药 **5-10 天后** Cr 升（不是数小时，是亚急性）|
| **UA** | Muddy brown casts |
| **典型药** | Gentamicin / Tobramycin / Amikacin / Streptomycin / Neomycin |
| **危险因素** | 长疗程 / 高剂量 / 老年 / 与利尿剂或 CNI 联用 / 已 CKD |
| **预防** | **Once-daily 给药**（vs 多次小剂量）+ 监测 trough levels + 短疗程 |

### 3.3 横纹肌溶解 (Rhabdomyolysis → 色素 ATN)

| 项 | 内容 |
|---|---|
| **机制** | 肌细胞死 → myoglobin 释放 → 小管毒 + 阻塞 + 血管收缩 → ATN |
| **buzz** | 严重外伤 / 长时间不动（地震废墟、毒品昏睡、跌倒躺地几小时）/ 剧烈运动 / **他汀 + fibrate 联用** / 神经阻滞剂综合征 (NMS) |
| **Lab 指纹** ★ | **CK ↑↑↑**（≥ 5x normal，常 >10000）/ **K↑** / **PO4↑** / **Ca↓** / 尿酸↑ |
| **尿** ★ | **尿色深红/棕** + **dipstick blood ✓ 但镜下无 RBC**（myoglobin 造假阳性）|
| **治疗** | **大量 IV NS** 冲刷（维持 UOP > 200-300 mL/h）/ 监测电解质 / 透析 if 顽固高 K / volume overload |

> [!warning] 鉴别"血尿"
> 尿 dipstick blood 阳性 + 镜下无 RBC = **myoglobin (rhabdo) 或 hemoglobin (溶血)**
> 真血尿 = dipstick + 镜下 RBC

### 3.4 肿瘤溶解综合征 (Tumor Lysis Syndrome, TLS)

| 项 | 内容 |
|---|---|
| **机制** | 化疗 → 肿瘤细胞大量死 → 释放 K + PO4 + 核酸（→ 尿酸）→ **尿酸结晶 ATN** |
| **典型病** | **高 burden 肿瘤** → **ALL / Burkitt 淋巴瘤 / 高白细胞 AML**（化疗 24-72 h）|
| **Lab 4 联** ★ | **K ↑ / PO4 ↑ / 尿酸 ↑ / Ca ↓**（PO4 高 → Ca-PO4 沉积 → Ca ↓）|
| **预防** | **化疗前 IV 水化** + **Allopurinol**（低风险）或 **Rasburicase**（高风险）|
| **治疗** | 同上 + 透析 if 顽固高 K / AKI |

> [!danger] G6PD 缺乏 + Rasburicase = 禁忌
> Rasburicase 把尿酸氧化成 allantoin，产物是 H2O2 → G6PD 缺乏 → 溶血危象
> 用前查 G6PD

### 3.5 乙二醇中毒 (Ethylene Glycol → 草酸钙结晶肾病)

| 项 | 内容 |
|---|---|
| **机制** | 乙二醇 → 甘醇酸 → 草酸 → 与 Ca 结合形成 **草酸钙结晶**（菱形/信封形）→ 小管阻塞 + 直接毒 |
| **buzz** | 自杀 / 误饮防冻液 / "AKI + 高 AG 代酸 + 渗透压间隙 ↑" + UA 见菱形结晶 |
| **解毒** ★ | **Fomepizole**（首选 — 抑 alcohol dehydrogenase）或 **乙醇 IV** + 透析（去除毒物 + 纠酸）|
| **甲醇中毒** | 同机制，毒产物 = 甲酸 → 视神经损伤；解毒同上 |
| **异丙醇** | 不致酸中毒，只致渗透压间隙 ↑ + 酮血（无 AG 升）|

### 3.6 多发性骨髓瘤 cast nephropathy (Myeloma kidney)

| 项 | 内容 |
|---|---|
| **机制** | **Light chain (Bence Jones protein)** 在小管内与 Tamm-Horsfall 蛋白形成蜡样管型 → 阻塞 + 小管毒 → ATN |
| **buzz** | **老年 + 骨痛 + 高钙 + 贫血 + AKI + UA 蛋白尿少（dipstick）但 UPEP 见 M-spike** ★ |
| **诊断陷阱** ★ | **Dipstick 测 albumin 但不测 light chain** → 蛋白阴性也不能排除 MM 肾病 → 要做 **UPEP / SPEP / serum free light chain** |
| **治疗** | 治 MM + IV 水化 + 避免肾毒（NSAID / 造影剂） |

### 3.7 胆固醇栓塞 (Atheroembolic Disease)

| 项 | 内容 |
|---|---|
| **机制** | 主动脉斑块碎屑栓塞肾小血管 → 缺血 + 异物反应 + 嗜酸炎症 |
| **触发** | **导管术 / 主动脉手术 / 抗凝 / 溶栓** 后 |
| **时间窗** ★ | **术后 1-8 周**（不是马上 — 与 CIN 24-48 h 完全不同）|
| **三联指纹** ★ | **AKI + Livedo reticularis（网状青斑）/ 蓝趾 + 嗜酸（血 + 尿）** |
| **UA** | 嗜酸尿 + 偶见嗜酸管型 |
| **vs CIN** | CIN = 48 h 内 + UA muddy brown；胆固醇栓塞 = 1-8 周 + livedo + 嗜酸 |
| **治疗** | 支持 + 他汀 + 戒抗凝（如果可能）|

### 3.8 ACE-I / ARB 触发的 AKI（Bilateral RAS 揭幕） — Q16323 关联

| 项 | 内容 |
|---|---|
| **机制** | RAS 时**肾依赖 Ang II 收缩 efferent** 维持 GFR；ACE-I 阻 Ang II → efferent 扩张 → GFR ↓↓ |
| **buzz** | 老人 + 顽固 HTN + 给 ACE-I 后 **Cr 急升 >30%** 或腹部听诊有 bruit / 移植肾 1-2 yr + 同样反应 (Transplant RAS) |
| **诊断** | **停 ACE-I → Cr 恢复** + **Doppler / MRA / CTA** 确诊 RAS |
| **治疗** | 停 ACE-I + 治 RAS（动脉粥样：药物 ± 支架；FMD：PTA；Transplant RAS：PTA + stent + 停 ACE-I）|

> [!warning] 不要把"ACE-I 致 Cr 升"反射理解为"肾毒"
> ACE-I 本身**不肾毒**，是**揭幕**：① Bilateral RAS ② 单肾 ③ Transplant RAS ④ 心衰严重 ⑤ 极度脱水
> 详读 [[完整笔记/专题笔记/肾脏/肾脏_移植后AKI鉴别SOP]]

### 3.9 HRS — Q4752 触发

> [!info] HRS = 肝硬化 + 扩容无应答 + bland UA
> 详读 [[完整笔记/专题笔记/GI/GI_HRS肝肾综合征]]

### 3.10 Amphotericin B 肾毒 + Type 1 RTA + 电解质三联 ★

> [!warning] 抗真菌"金标准"的代价 — 几乎人人肾毒
> Amphotericin B（常规剂型）肾毒发生率高达 **30-80%**，是住院真菌感染治疗最大的副作用

| 项 | 内容 |
|---|---|
| **机制** | 与肾细胞胆固醇结合（特别是远端小管 + 集合管）→ 膜通透性 ↑ → 小管功能紊乱 + 直接细胞毒 |
| **3 大表现** | ① **ATN**（GFR ↓ + Cr ↑）② **Type 1 (Distal) RTA**（H+ 排泄障碍 → 非 AG 代酸 + 高 Cl）③ **电解质丢失**：**低 K + 低 Mg** ★ |
| **时间窗** | 用药数天-2 周开始；累积剂量相关 |
| **UA** | 多种 — 可见 muddy brown 或单纯小管功能损伤 (低 K + 高 Cl 代酸) |
| **危险因素** | 高剂量 / 长疗程 / 与利尿剂或 CNI 联用 / 已 CKD / 容量不足 |
| **预防** | **IV NS 水化**（前后）+ **补 K + 补 Mg** 预防性 |
| **更安全选项** | **Liposomal Amphotericin B**（脂质体）肾毒大幅 ↓ — 现已是高风险病人首选 ★ |

> [!danger] CK 经典 stem
> 真菌感染病人用 conventional amphotericin B 2 周 → Cr 升 + **K 2.8 + Mg 1.2 + 高氯非 AG 代酸**（HCO3 ↓）→ 诊断？
> A: **Amphotericin B 致 ATN + Type 1 RTA + 低 K/Mg**
> 治：换 liposomal 制剂 + 补 K + 补 Mg + IV NS 水化

> [!info] Type 1 RTA 常见病因记忆
> "**SLE-PAS**" — **S**jögren / **L**upus / **E**A multiforme / **P**lasma cell dyscrasia (MM) / **A**mphotericin B ★ / **S**ickle cell

详细 RTA 4 型对比见 [[完整笔记/专题笔记/肾脏/肾脏_RTA四型完整体系]]（RTA 主笔记）

### 3.11 药物诱发免疫性肾病（DIL-GN / ANCA-AAV / Membranous）★

> [!info] 这是 §2.2 AIN 之外的"另一种药物-免疫肾病通路"
> AIN = T 细胞介导，肾间质 + WBC casts
> 本节 = 抗体 / 免疫复合物介导，**肾小球** + **RBC casts + 蛋白尿**
> 同一些药（hydralazine / penicillamine）可走不同通路 → 别混

#### 通用主线：DIL **典型 90% 不累肾** → 极少数累肾时走 GN 通路

| 4 情境 | 致病药 | 肾累及类型 | UA / Lab |
|---|---|---|---|
| **① 典型 DIL，无肾累及**（主线 ★） | **Procainamide** ★★★ / Quinidine / Methyldopa | 无 | 正常 |
| **② DIL + Lupus-like Nephritis** | **Anti-TNF (infliximab/etanercept)** ★ | 免疫复合物 GN | **RBC casts + 蛋白** + ANA+ + 抗组蛋白+ |
| **③ Drug-induced Membranous Nephropathy** | **Penicillamine** ★ / Gold / NSAIDs（慢性）| 膜性肾病 | **大量蛋白尿（>3.5 g/d）→ 肾病综合征** + 少 RBC |
| **④ Drug-induced ANCA-Associated Vasculitis (AAV)** | **Hydralazine / Propylthiouracil (PTU) / Minocycline / Levamisole** ★ | **RPGN / 新月体 GN** | **RBC casts + 急 Cr↑ + MPO-ANCA (p-ANCA)** ★ + 可肺出血 + 紫癜 |

#### 机制速记

```
DIL 主线（抗组蛋白）→ 关节 + 浆膜 + 皮肤 → 停药即愈
     │
     ├─ Anti-TNF 亚组 → 免疫复合物沉积肾小球 → Lupus-like nephritis
     │
     ├─ Penicillamine 亚组 → 抗体直接结合 PLA2R / 改变足细胞 → Membranous Nephropathy
     │
     └─ Hydralazine/PTU/Minocycline 亚组
         → 诱导 MPO-ANCA → 中性粒细胞激活 → 小血管炎
         → RPGN + 肺出血 + 紫癜（类似 GPA/MPA）★
```

#### 与 SLE / AIN 的最终鉴别（CK 决战表）

| | **典型 DIL** | **SLE** | **AIN** | **DIL-肾累及变体**（少数）|
|---|---|---|---|---|
| **肾累及频率** | 罕见（<10%）| 30-50% | 100% | 看亚组 |
| **UA**（有累及时）| RBC casts + 蛋白 | RBC casts + 蛋白 | **WBC casts + 嗜酸** | RBC casts + 蛋白 |
| **机制** | 免疫复合物（少）| 免疫复合物 | **T 细胞超敏** | 免疫复合物 / 抗体 / ANCA |
| **anti-dsDNA** | 阴 | **阳** ★ | — | 一般阴 |
| **抗组蛋白** | **阳** ★ | 50-70% | — | 阳 |
| **补体** | 正常 | **低** | 正常 | 视亚组（ANCA 通常正常）|
| **CNS 累及** | 无 | 有 | 无 | 无 |
| **治疗** | 停药 | 长期免疫抑制 | 停药 ± 类固醇 | 停药 + 类固醇 ± 强化免疫抑制 |

> [!danger] 看药锁定亚组
> - 题干 **anti-TNF** + 肾累及 → **DIL + Lupus-like nephritis** → 停 + 类固醇
> - 题干 **penicillamine** + 大蛋白尿 → **Membranous Nephropathy** → 停 + 换药（Wilson 换 trientine）
> - 题干 **hydralazine / PTU / minocycline** + RPGN + MPO-ANCA → **药物诱发 ANCA 血管炎** → 停 + 类固醇 + cyclophosphamide/rituximab
> - 题干 **procainamide** + 关节痛 → **典型 DIL（肾正常）** → 停即可

## 七、CK 高频陷阱合集

> [!question]- Q1: 65 yo CKD + 心导管造影后 48 h Cr 1.0 → 2.6。最有效的预防？
> A: **术前 IV 0.9% NS 水化** ★（Q4491）
> 陷阱选项 "Hold metformin" = 防**乳酸酸中毒**，不是防 AKI

> [!question]- Q2: 70 yo M 主动脉斑块明显，3 周前做了腹主造影。今天 Cr ↑ + 脚趾发紫 + 网状青斑 + 血嗜酸 12%。诊断？
> A: **胆固醇栓塞** (atheroembolic AKI)
> 关键：3 周后才出 ≠ CIN（CIN 在 48 h 内）

> [!question]- Q3: 急诊外伤压挤腿，CK 25000 + 尿色棕红 + dipstick blood ✓ 但镜下无 RBC。第一步？
> A: **Rhabdomyolysis → 色素 ATN**
> 第一步：**大量 IV NS** 冲刷（目标 UOP 200-300 mL/h）

> [!question]- Q4: 25 yo Burkitt 淋巴瘤化疗 36 h 后 K 6.5, PO4 8, 尿酸 18, Ca 7。诊断 + 高风险预防药？
> A: **TLS**
> 高风险预防：**Rasburicase**（G6PD 阳性 → 改 allopurinol）+ 大量 IV 水化

> [!question]- Q5: 30 yo M 误饮防冻液，意识模糊 + 高 AG 代酸 + 渗透压间隙 ↑ + UA 菱形结晶。解毒？
> A: **Fomepizole**（首选）+ 透析

> [!question]- Q6: 70 yo F 几个月骨痛 + 反复肺感染 + 贫血 + Cr 升 + UA dipstick 蛋白阴性。下一步？
> A: **UPEP / SPEP + 血清自由 light chain**（不能信 dipstick — MM 的 light chain dipstick 不测）
> 诊断 **MM cast nephropathy**

> [!question]- Q7: 痛风病人开始 **allopurinol 3 周** 后 Cr 升 + 皮疹 + 发热 + 关节痛 + 外周嗜酸 + WBC casts。诊断？
> A: **AIN by Allopurinol** ★（Q4007）— 不只抗生素 / PPI / NSAID 致 AIN，**allopurinol 也是经典 AIN 原因**

> [!question]- Q8: 75 yo M HTN 顽固，开始 lisinopril 第 5 天 Cr 1.0 → 1.8 + 腹部听诊 bruit。诊断 + 下一步？
> A: **ACE-I 揭幕 Bilateral RAS**
> 下一步：**停 ACE-I + Doppler / MRA** 确诊

> [!question]- Q9: 60 yo M 肺炎打 gentamicin 第 7 天 Cr 升 + UA muddy brown。机制 + 预防？
> A: **氨基糖苷 ATN**
> 预防：**Once-daily 给药** + 监测 trough levels + 短疗程

> [!question]- Q10: 70 yo F 几十年用 OTC APAP + aspirin 复方镇痛药，渐进 Cr ↑ + 贫血 + 双侧小肾 + UA WBC casts。诊断？
> A: **Analgesic Nephropathy**（Q4004）= **CTIN + 乳头坏死**双打

> [!question]- Q11: 肝硬化 + 腹水 + Cr 1.0 → 2.8 + 尿 Na 8 + Albumin 1 g/kg × 2 天后 Cr 仍 2.7。诊断 + 治疗？
> A: **HRS-AKI**（Q4752）→ **Terlipressin + Albumin**；根治 = **肝移植**

> [!question]- Q12: 移植肾病人 tacrolimus 维持，新加 diltiazem 治 AF → 2 周后 Cr 升。机制？
> A: **CNI 急性毒**（Q16485）— diltiazem 抑 CYP3A4 → tacrolimus 血药 ↑ → afferent 收缩 → AKI
> 治疗：降 CNI 剂量 + 换不互动的钙拮抗（amlodipine）

> [!question]- Q13: 一个 stem 同时给"老年 + 慢性 NSAID + AKI + 血尿 + 漂浮组织"，最可能机制？
> A: **乳头坏死**（NSAID 慢性 + 髓质血供 ↓）+ 可能合并 Analgesic Nephropathy 慢性背景

> [!question]- Q14: 真菌感染病人 conventional amphotericin B 用 2 周后，Cr 升 + K 2.8 + Mg 1.2 + 非 AG 代酸（HCO3 ↓ + Cl ↑）。诊断 + 下一步？
> A: **Amphotericin B 致 ATN + Type 1 (Distal) RTA + 低 K/Mg**
> 下一步：**换 Liposomal Amphotericin B + 补 K + 补 Mg + IV NS 水化**
> 经典三联：肾毒 + 远端 RTA + 电解质丢失

> [!question]- Q15: 顽固 HTN 老年用 hydralazine 8 个月 → 急性肾衰 + RBC casts + 肺出血 + **MPO-ANCA (p-ANCA) +**。诊断 + 治疗？
> A: **药物诱发 ANCA 血管炎（drug-induced AAV → RPGN）** — **不是单纯 DIL**
> 治：**停 hydralazine + 大剂量类固醇 + cyclophosphamide / rituximab**
> 同样亚组：PTU（抗甲亢）/ Minocycline / Levamisole（污染可卡因）

> [!question]- Q16: RA 病人 infliximab 6 个月后 + 关节痛加重 + 浆膜炎 + ANA+ + 抗组蛋白+ + **Cr 升 + RBC casts + 蛋白尿 2 g/d**。诊断？
> A: **Anti-TNF 诱发 DIL + Lupus-like Nephritis**（少数 DIL 真累肾的代表）
> 治：**停 anti-TNF + 类固醇** ± 转换其他生物制剂
> 反 pattern：典型 DIL 不累肾（procainamide / hydralazine 单纯 DIL）—— **anti-TNF 是例外**

> [!question]- Q17: Wilson 病用 penicillamine 1 年后下肢水肿 + 24h 蛋白 6 g + Alb ↓ + 高脂血。诊断 + 治疗？
> A: **Penicillamine 诱发 Membranous Nephropathy** → 肾病综合征
> 治：**停 penicillamine + 换 trientine**（Wilson 替代螯合剂）
> 同 NSAIDs 慢性也可致 membranous

> [!question]- Q18: Procainamide 1 年后关节痛 + 胸膜炎 + ANA+ + 抗组蛋白+ + dsDNA- + 补体正常。Cr 0.9 + UA 正常。诊断 + 治疗？
> A: **典型 DIL（procainamide 是最高风险药）— 无肾累及（90%+ DIL 都这样）**
> 治：**停 procainamide**（换 amiodarone 等）→ 数周-数月自愈
> 与 SLE 鉴别：dsDNA-、补体正常、无肾/CNS、停药即愈

## 八、记忆锚

> [!tip] 时间窗速记
> - **24-48 h** = **造影剂**
> - **5-10 天** = **氨基糖苷**
> - **5 天-3 周** = **AIN（药物过敏）**
> - **7-14 天** = **β-lactam AIN**（subtype）
> - **1-8 周** = **胆固醇栓塞**
> - **数小时** = **缺血 / Rhabdo / TLS / 乙二醇 / 急性 prerenal / 结晶**
> - **几十年** = **Analgesic Nephropathy / 长期 NSAID 乳头坏死**

> [!tip] UA 速记
> - **Muddy brown casts** = ATN（缺血 + 毒物 + 色素）
> - **WBC casts + 嗜酸** = AIN（药物过敏）/ Analgesic Neph 慢性 / 胆固醇栓塞
> - **RBC casts + 蛋白** = GN
> - **Bland** = Prerenal / HRS / CNI 急性毒 / ACE-I 揭幕 / 胆固醇栓塞 早期
> - **结晶** = 乙二醇（菱形）/ 尿酸（砖红）/ MTX-Acyclovir（针形）
> - **Dipstick blood ✓ 镜下无 RBC** = **myoglobin / hemoglobin** 色素 ATN

> [!tip] 5 兄弟一句话
> - **ATN** = Brown Muddy（**M**echanical death）
> - **AIN** = **R**ash + **E**osinophils
> - **乳头坏死** = "**NSAID**" 5 因（N/S/A/I/D）+ 漂浮组织
> - **Analgesic Nephropathy** = 几十年镇痛药 + CTIN + 乳头钙化
> - **CTIN** = AIN 的慢性版

> [!tip] "看到 ACE-I + Cr 升" 不要反射"肾毒"
> ACE-I **不肾毒**，是**揭幕** RAS / 心衰严重 / 单肾的诊断信号 → 停 + 找病因

> [!tip] CIN 防错记
> **Hold metformin = 防乳酸酸中毒**（不是防 AKI！）
> **IV NS 水化 = 防 CI-AKI 的真正招**

> [!tip] DIL 肾累及 4 情境（反直觉补丁）
> **DIL 主线不累肾**，但少数走 GN 通路：
> - **Anti-TNF** → DIL + lupus-like nephritis
> - **Penicillamine** → Membranous Nephropathy
> - **Hydralazine / PTU / Minocycline** → 药物诱发 ANCA 血管炎 (RPGN)
> - **Procainamide** → 典型 DIL（**不**累肾）
> 看药锁定亚组，看 UA 锁定通路

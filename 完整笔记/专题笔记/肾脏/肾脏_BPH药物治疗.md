# BPH 药物治疗（Benign Prostatic Hyperplasia, 良性前列腺增生）

> [!info] 一句话框架
> BPH = **DHT（双氢睾酮）驱动前列腺移行带增生** → 机械性梗阻 + 平滑肌张力↑ → **LUTS（下尿路症状）**。
> 药物治疗两条线：**α1 阻滞剂（快，松平滑肌，不缩体积）** + **5α-还原酶抑制剂（慢，缩体积，↓PSA）**；大前列腺联用。
> 病情进展 / 难治 / 并发症（反复 UTI、结石、AUR、肾后性梗阻）→ 手术（TURP）。

---

## 一、两大主力药 + 辅助药

### 1. α1-肾上腺素能阻滞剂（α1-blockers）

机制：阻断前列腺 / 膀胱颈 / 前列腺包膜的 **α1 受体 → 平滑肌松弛 → 降低出口阻力**。**数天起效**（快），**不改变前列腺体积**。

| 亚类 | 代表药 | 受体选择性 | 临床要点 |
|---|---|---|---|
| **非选择性 α1** | **Doxazosin / Terazosin / Prazosin** | α1A + α1B（血管）| 同时降血压 → **合并 HTN 的 BPH 首选**；但**首剂体位性低血压**明显 |
| **选择性 α1A** | **Tamsulosin / Silodosin / Alfuzosin** | α1A（前列腺为主）| 降压副作用小 → **血压正常者首选**；但 **IFIS + 逆行射精**更突出 |

### 2. 5α-还原酶抑制剂（5-ARI, 5α-Reductase Inhibitors）

机制：阻断 **睾酮 → DHT** 转化 → 前列腺**缩小** → 减机械梗阻。**起效慢（数月-数年）**，**缩体积** + **↓血清 PSA 约 50%**。

| 药 | 抑制亚型 | 备注 |
|---|---|---|
| **Finasteride** | Type II | 经典；也治男性型脱发（低剂量）|
| **Dutasteride** | Type I + II | 抑制更全面 |

### 3. 辅助 / 特定情境药

| 药 | 机制 | 用途 |
|---|---|---|
| **Tadalafil（PDE5 抑制剂）** | 松弛前列腺 / 膀胱平滑肌 | BPH **合并 ED** 时一举两得 |
| **抗胆碱药（oxybutynin 等）** | 抑制逼尿肌 | BPH **储尿期症状（尿急/尿频/OAB）**为主时加用；⚠️ 残余尿多 / 高 AUR 风险者慎用（可诱发尿潴留）|

## 二、α-blocker vs 5-ARI 核心对比（必背）

| 维度 | α1 阻滞剂 | 5α-还原酶抑制剂 |
|---|---|---|
| **起效速度** | 快（数天）| 慢（数月-数年）|
| **作用层面** | 功能性（松平滑肌）| 结构性（缩腺体）|
| **前列腺体积** | 不变 | ↓ 缩小 |
| **血清 PSA** | 不变 | **↓ 约 50%** ⭐ |
| **最适人群** | 急性症状缓解 / 任意体积 | **大前列腺**、想长期减进展 |
| **降血压** | 有（非选择性更强）| 无 |
| **招牌副作用** | 体位性低血压、**IFIS**、逆行射精 | 性欲↓、ED、男性乳腺发育、**致畸** |

> [!tip] 联合治疗（large prostate）
> 大前列腺 / 中重症 → **α-blocker（快缓解症状）+ 5-ARI（长期缩体积、防进展）联用**（MTOPS 研究支持：联合显著降低 BPH 临床进展、AUR 与手术需求）。

## 三、高频副作用深挖（USMLE 陷阱集中地）

> [!warning] IFIS（术中虹膜松弛综合征，Intraoperative Floppy Iris Syndrome）⭐⭐
> **所有 α1 阻滞剂**（尤其 **tamsulosin**）松弛**虹膜扩张肌** → 白内障手术时虹膜松脱 / 脱垂 + 瞳孔骤缩 → 后囊破裂风险↑。
> - **停药基本无用**：扩张肌受累可持续数月甚至永久。
> - **不是手术禁忌**：正确处理 = **术前披露用药史 → 术者改良技术**（前房注射 phenylephrine/epinephrine、虹膜拉钩、**Malyugin 瞳孔扩张环**、高黏弹剂）。
> - **考点核心**：披露目的是"让术者准备"，**不是取消手术、也不是单纯停药**。
> - 反向：先白内障、后启 tamsulosin 通常无碍；风险窗 = "已/曾用药 + 尚未手术"。

> [!warning] 5-ARI 的两大考点
> - **↓PSA 约 50%**：用 finasteride/dutasteride 的患者，前列腺癌筛查时**测得 PSA 要 ×2 还原**，否则漏诊。任何"用药后 PSA 不降反升 / 不降"都应警惕前列腺癌。
> - **致畸（Pregnancy Category X）**：DHT 是男胎外生殖器发育所需 → **育龄 / 妊娠女性禁止接触压碎或破损药片**（完整包衣片接触一般安全）。

> [!tip] α-blocker 其他 SE
> - **首剂体位性低血压（first-dose effect）**：睡前服、从小剂量起。非选择性（doxazosin/terazosin/prazosin）更明显。
> - **逆行射精 / 异常射精**：选择性 α1A（tamsulosin/silodosin）更突出。

## 四、选药决策树

```
BPH + LUTS（已排除前列腺癌 / 尿道狭窄 / 神经源膀胱）
   │
   ├─ 轻症、症状轻微 ───────────────→ watchful waiting + 生活方式
   │
   ├─ 中重症，想快速缓解 ───────────→ α1 阻滞剂
   │       ├─ 合并 HTN ─────────────→ 非选择性（doxazosin / terazosin）一举两得
   │       └─ 血压正常 ─────────────→ 选择性 α1A（tamsulosin）少降压
   │
   ├─ 前列腺明显增大 / 想防长期进展 ─→ 加 5-ARI（finasteride）→ 大腺体常 α + 5-ARI 联用
   │
   ├─ 合并 ED ──────────────────────→ 考虑 tadalafil（PDE5i）
   │
   ├─ 储尿期症状（尿急尿频）为主 ────→ 谨慎加抗胆碱（先评估残余尿，防 AUR）
   │
   └─ 难治 / 并发症（反复 UTI、膀胱结石、肾后梗阻 Cr↑、复发性 AUR）
                                       └─→ 手术（TURP 等）
```

## 五、USMLE 高频陷阱

> [!danger] 易错点清单
> - **finasteride 不用于急性症状缓解**——起效要数月；想"这周就好"选 α-blocker。
> - **慢性进行性 Cr↑ 的 BPH** → 先做**肾脏超声排 hydronephrosis（肾后性梗阻）**，不是直接上药（见 [[mistakes/uworld-mistakes_2026-05#^Q20536]]）。
> - **年轻男性梗阻性排尿** + 器械史 / 外伤史 → 想**尿道狭窄（urethral stricture）**，不是 BPH（见 [[mistakes/uworld-mistakes_2026-05#^Q15906]]）。
> - **bethanechol（胆碱能）治尿潴留促排空、oxybutynin（抗胆碱）治 OAB、phenazopyridine 仅尿路镇痛**——都不是 BPH 缩体积药，别混。
> - **白内障术前**问出 tamsulosin → 告知术者，不是停药/取消手术。

## 六、复习题

> [!question]- Q1. 一名 BPH 患者血压正常，想尽快缓解尿流弱，最合适的初始药物类别？若同时有高血压呢？
> 血压正常：**选择性 α1A（tamsulosin）**——降压副作用小。
> 合并 HTN：**非选择性 α1（doxazosin / terazosin）**——同时控压 + 缓解 LUTS，一举两得。

> [!question]- Q2. 患者服 finasteride 1 年，PSA 由 4.0 降至 2.0。如何解读这个 PSA？
> 5-ARI **降 PSA 约 50%**，筛查时**应 ×2 还原**（即等效 ~4.0）。若用药期间 PSA **不降反升 / 不降**，警惕**前列腺癌**，需进一步评估。

> [!question]- Q3. BPH 患者准备做白内障手术，术前用药史发现 tamsulosin。正确处理？
> **告知手术团队（IFIS 风险）→ 术者改良技术**（前房 phenylephrine、Malyugin 环、虹膜拉钩）。**不停药**（无效）、**不取消手术**。

> [!question]- Q4. 为什么 5-ARI 对育龄女性有接触禁忌？哪种剂型相对安全？
> DHT 是**男胎外生殖器发育**所需 → 5-ARI **致畸（Category X）**。**压碎 / 破损药片**禁止育龄女性接触；**完整包衣片**接触一般安全。

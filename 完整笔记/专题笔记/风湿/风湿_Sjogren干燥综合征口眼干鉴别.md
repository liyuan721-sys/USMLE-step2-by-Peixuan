# 干燥综合征 Sjögren Syndrome 口眼干鉴别

> [!tip] 一句话锁定
> **口干 (xerostomia) + 眼干 (干燥性角结膜炎) + 腮腺肿大 + 龋齿骤增** = 干燥综合征 (Sjögren syndrome) 标准画像。像了之后**下一步不是先吃药或做 CT，而是做血清学确认**：ANA、anti-Ro/SSA、anti-La/SSB。

> [!note]- 缩写对照（全称 中英）
> - **ANA** = antinuclear antibody 抗核抗体
> - **SSA / anti-Ro** = 抗 Ro 抗体
> - **SSB / anti-La** = 抗 La 抗体
> - **KCS** = keratoconjunctivitis sicca 干燥性角结膜炎
> - **RA** = rheumatoid arthritis 类风湿关节炎
> - **SLE** = systemic lupus erythematosus 系统性红斑狼疮

---

## 一、核心：sicca 三联 + 血清学优先

> [!info] 机制
> 自身免疫性**淋巴细胞浸润**破坏**泪腺 + 唾液腺 (lacrimal / salivary glands)** → 泪液↓ + 唾液↓ → 干眼红眼 + 口干 → **龋齿增多 + 腮腺肿大**。

```text
淋巴细胞攻击泪腺/唾液腺
  → 泪液减少 + 唾液减少
  → 干燥性角结膜炎（眼干红）+ 口干（吞咽困难）
  → 龋齿骤增 + 双侧腮腺肿大
  → 血清学确认：ANA / anti-Ro(SSA) / anti-La(SSB)
```

> [!tip] 分型
> - **原发性 (primary)**：单纯 Sjögren。
> - **继发性 (secondary)**：合并 RA 类风湿关节炎 / SLE 系统性红斑狼疮等其他结缔组织病。

---

## 二、诊断路径（考试反射）

> [!success] 典型 sicca → 先血清学，非影像/经验用药
> - **首选**：ANA + anti-Ro/SSA + anti-La/SSB（低侵入、直接）。
> - 辅助/进阶：Schirmer 试验（泪液分泌）、唇腺小唾液腺活检（金标准，见局灶性淋巴细胞浸润）。
> - **错误下一步**：先给抗生素（非感染）、先给激素（未确诊）、先做腮腺 CT/影像（不解决自身免疫确认）。

## 三、鉴别：口干/眼干/腮腺肿的其他原因

表格前空行。

| 疾病 | 关键区别点 |
|---|---|
| **Sjögren 干燥综合征** | sicca 三联 + 龋齿↑ + SSA/SSB 阳性；可原发或继发 |
| **腮腺炎/流行性腮腺炎 (mumps)** | 急性、发热、单/双侧痛性肿大、无慢性口眼干 |
| **结节病 (sarcoidosis)** | 腮腺肿 + 葡萄膜炎 + 面瘫（Heerfordt）+ 肺门淋巴结、高钙、ACE↑ |
| **IgG4 相关病** | 泪腺唾液腺对称肿大 (Mikulicz)，IgG4↑ |
| **药物性口干** | 抗胆碱能药/抗组胺/利尿剂史；无自身抗体、无眼干红 |
| **HIV 弥漫性浸润性淋巴细胞增多综合征 (diffuse infiltrative lymphocytosis syndrome, DILS)** | CD8+ 淋巴细胞浸润腮腺，查 HIV |

## 四、并发症与随访（高频延伸）

> [!warning] Sjögren 两大高危延伸
> - **B 细胞淋巴瘤（黏膜相关淋巴组织淋巴瘤 mucosa-associated lymphoid tissue lymphoma, MALT lymphoma）风险↑**：持续/进行性腮腺肿大要警惕恶变。
> - **孕妇 anti-Ro/SSA 阳性 → 胎儿先天性心脏传导阻滞 (congenital heart block)**：SSA 抗体经胎盘攻击胎儿传导系统。这是产科交叉高频点。
> - 对症：人工泪液、毛果芸香碱 pilocarpine（促唾液/泪液）、口腔护理防龋。

## 五、自测（答完再判分）

> [!question]- Q1. 52 岁女性口干致吞咽困难、双颊肿、龋齿骤增、双眼干涩发红，无用药史。下一步最合适的检查？
> **血清 ANA + anti-Ro/SSA + anti-La/SSB**。典型 sicca 综合征先做血清学确认，不先经验性用药或影像。

> [!question]- Q2. 一名 anti-Ro/SSA 阳性的孕妇，其胎儿最需监测的并发症是什么？
> **先天性心脏传导阻滞 (congenital heart block)**——母体 SSA 抗体经胎盘攻击胎儿心脏传导系统。

# 系统性红斑狼疮 SLE (Systemic Lupus Erythematosus)

> [!info] 一句话定位
> **SLE (systemic lupus erythematosus, 系统性红斑狼疮) = 育龄女性多系统自身抗体病**，免疫复合物 (immune complex, III 型超敏) 沉积多器官。好发**育龄女性 + 非裔/西裔**。考点核心 = **抗体谱判读 + 多系统受累 + 狼疮肾炎 + 药物性狼疮**。

> [!tip] 做题反射钩子：多系统拼图 → Occam 剃刀（来源 [[mistakes/uworld-mistakes_2026-05#^Q11967]]）
> 一道题若堆了 **≥3 个看似无关的系统异常**（如 脑/精神症状 + 关节炎 + 血三系↓ + 肾/蛋白尿），别逐个找独立病因——**强制找"一个能统一解释全部"的诊断 (Occam's razor) → 育龄女性首先想 SLE → 第一步筛 ANA**。尤其 **青少年 / 年轻人急性精神病 (acute psychosis)** 伴其他系统线索时，先排器质性，把 **SLE / NPSLE (neuropsychiatric SLE, 神经精神性狼疮)** 放进鉴别，而非直接归精神科。

---

## 一、诊断思路（筛查→确诊→活动度）

> [!tip] 三步抗体逻辑（最高频）
> ```
> ANA (antinuclear antibody, 抗核抗体)
>   = 敏感筛查（>95% 阳性，阴性基本排除 SLE）但不特异
>        │ 阳性 ↓
>   anti-dsDNA + anti-Smith = 特异确诊
>        ├── anti-dsDNA：特异 + 与狼疮肾炎/疾病活动度相关（随病情波动，可监测）
>        └── anti-Smith (anti-Sm)：最特异，但不随活动度变（确诊用，不监测）
> ```

> [!warning] 活动度监测三件套
> 疾病活动 → **anti-dsDNA 升高 + 补体 C3/C4 降低（消耗）+ 临床发作**。补体低 = 免疫复合物正在消耗补体，是活动标志。

经典 11 条标准助记 **SOAP BRAIN MD**（≥4 条支持，了解即可，CK 更爱考抗体 + 单系统表现）：

```
S - Serositis 浆膜炎（胸膜炎/心包炎）
O - Oral ulcers 口腔溃疡（无痛）
A - Arthritis 关节炎（非侵蚀性、对称小关节）
P - Photosensitivity 光敏感
B - Blood 血液三系↓（溶血性贫血/白细胞↓/血小板↓）
R - Renal 肾（蛋白尿/管型）
A - ANA 阳性
I - Immunologic 免疫（anti-dsDNA / anti-Sm / 抗磷脂）
N - Neurologic 神经（癫痫/精神病）
M - Malar rash 颊部蝶形红斑（鼻唇沟豁免）
D - Discoid rash 盘状红斑
```

## 二、抗体谱（必须背熟的判读表）

> [!danger] 抗体 = SLE 最高频考点
> 缩写：dsDNA = double-stranded DNA（双链 DNA）、Sm = Smith、aPL = antiphospholipid antibody（抗磷脂抗体）。

| 抗体 | 临床意义 | 钩子 |
|---|---|---|
| **ANA 抗核抗体** | 敏感筛查（阴性≈排除）；不特异 | 首选筛查，阳性再查特异抗体 |
| **anti-dsDNA 抗双链DNA** | **特异** + **狼疮肾炎** + **活动度**（随病情波动）| 肾受累 / 监测复发 |
| **anti-Smith (anti-Sm)** | **最特异**（但敏感性低、不随活动变）| 确诊"一锤定音"，不用于监测 |
| **anti-histone 抗组蛋白** | **药物性狼疮**（drug-induced lupus）| 见 §五 HIP 药 |
| **anti-Ro/SSA + anti-La/SSB** | **新生儿狼疮 + 先天性心脏传导阻滞**；亦见于 Sjögren 干燥综合征 | 孕妇查；亦致亚急性皮肤狼疮 + 光敏 |
| **抗磷脂抗体 aPL**（狼疮抗凝物/抗心磷脂/抗β2GP1）| **抗磷脂综合征 APS**：动静脉血栓 + 反复流产；矛盾性 **aPTT 延长但高凝** | 见 §六 |

## 三、多系统受累（按器官扫，这是"多系统重点病"的核心）

```
SLE 多系统地图
├── 皮肤：颊部蝶形红斑（鼻唇沟豁免）/ 盘状红斑 / 光敏 / 口腔无痛溃疡
├── 关节：非侵蚀性对称多关节炎（vs RA 侵蚀）；Jaccoud 关节病可复位
├── 肾  ：狼疮肾炎（见 §四）—— 预后 + 治疗强度决定因素
├── 血液：自身免疫溶血性贫血 / 白细胞↓ / 血小板↓ / 抗磷脂血栓
├── 浆膜：胸膜炎 + 心包炎（浆膜炎性胸痛）
├── 心  ：Libman-Sacks endocarditis 疣状心内膜炎（无菌性、瓣膜两面赘生物）
├── 肺  ：胸膜炎 / 间质病 / 弥漫肺泡出血
└── 神经：癫痫 / 精神病 / 脑卒中（NPSLE = neuropsychiatric SLE 神经精神狼疮）
```

> [!warning] 高频鉴别钩子
> - **关节炎非侵蚀**（与 RA 侵蚀性区分；SLE 的畸形是可复位的 Jaccoud 关节病）。
> - **Libman-Sacks 疣状心内膜炎**：无菌性赘生物长在瓣膜**两面**，与抗磷脂抗体相关，可栓塞；血培养阴性（区别于感染性心内膜炎 [[完整笔记/专题笔记/感染/感染_感染性心内膜炎完整体系]]）。
> - **发热 SLE 患者两难**：疾病活动（狼疮发作）vs 免疫抑制下感染 —— 看补体（活动↓）+ anti-dsDNA（活动↑）+ 感染指标帮区分。
> - **小儿 SLE 血液受累更突出**：常见**全血细胞减少 pancytopenia**（溶血性贫血 + 白细胞↓ + 血小板↓），本身即活动度提示。（来源：分科笔记 [[完整笔记/Peixuan分科笔记/肾脏]] SLE 段）
> - **肺-肾共病分流（10 秒做题）**：SLE 累肾时先分 **肾炎型 nephritic（血尿 + 管型 + 高血压）vs 肾病型 nephrotic（大量蛋白尿 + 低白蛋白）** → 再看补体 → 决定是否肾活检。

### 3.1 皮肤型狼疮三型（重点：盘状 DLE）

> [!info] 缩写：CLE = cutaneous lupus erythematosus（皮肤型红斑狼疮）；ACLE/SCLE/CCLE = 急性/亚急性/慢性皮肤型。

| 型 | 招牌 | 抗体/关联 | 瘢痕? | 与系统 SLE 关系 |
|---|---|---|---|---|
| **ACLE 急性皮肤型** | **颊部蝶形红斑**（鼻唇沟豁免）+ 光敏 | ANA / anti-dsDNA，随系统活动平行 | **不留瘢痕** | **强关联**（常是 SLE 一部分）|
| **SCLE 亚急性皮肤型** | 环形/丘疹鳞屑、躯干上部、光敏 | **anti-Ro/SSA**；**药物可诱发**（如氢氯噻嗪 hydrochlorothiazide / 特比萘芬 terbinafine） | 通常不瘢痕 | 中等 |
| **CCLE / DLE 盘状** | 见下方 danger | ANA 常阴/低 | **留瘢痕** | **弱**（多皮肤局限）|

> [!danger] 盘状红斑狼疮 (DLE, discoid lupus erythematosus, 盘状红斑狼疮)
> CCLE (chronic cutaneous lupus erythematosus, 慢性皮肤型红斑狼疮) 最常见型。
> - **皮损**：边界清的**红斑鳞屑斑块 + 毛囊角栓 (follicular plugging)**；好发**日晒部位（面/头皮/耳廓）**；光敏。
> - **愈合留瘢痕**：中央**萎缩 + 色素脱失**、周边色素沉着；**头皮受累 → 瘢痕性脱发 (scarring alopecia)（永久不可逆）**。
> - **多为皮肤局限**：单纯 DLE 患者**仅约 5–10% 进展为系统性 SLE**；反过来 SLE 患者也可出现盘状皮损。ANA 常阴性或低滴度。
> - **罕见远期风险**：慢性瘢痕内可恶变 **SCC (squamous cell carcinoma, 鳞状细胞癌)**。
> - **治疗**：**严格防晒** + **外用 / 皮损内糖皮质激素**；广泛者加 **羟氯喹 hydroxychloroquine**；**戒烟**（吸烟降低抗疟药疗效）。

> [!tip] 记忆锁定
> **蝶形红斑（ACLE）= 不留瘢痕、跟系统走；盘状（DLE）= 留瘢痕 + 毛囊角栓 + 头皮永久脱发、多皮肤局限、少数才变 SLE。** 看到"**面部瘢痕性斑块 + 毛囊角栓 + 瘢痕性脱发**"想 DLE，不是单纯蝶形红斑。

> 🏥 皮肤科交叉：瘢痕性 vs 非瘢痕性脱发鉴别（DLE / 扁平苔藓样 vs 斑秃 / 头癣）见 [[完整笔记/专题笔记/皮肤/皮肤_HighYield速查总览]] §8；SCLE vs DLE 皮损对照见 [[完整笔记/Peixuan分科笔记/derm]]。

## 四、狼疮肾炎 Lupus Nephritis（决定预后 + 治疗强度）

> [!danger] 关键：任何 SLE 出现蛋白尿/血尿/管型 → 肾活检定型
> 分型 (ISN/RPS, International Society of Nephrology / Renal Pathology Society 分类)：

| 类 | 病理 | 临床/处理 |
|---|---|---|
| I / II | 系膜微小 / 系膜增生 | 轻，常无需强免疫抑制 |
| **III / IV 增殖性**（最常见 + 最重）| 局灶 / **弥漫增殖** | **肾活检确诊 → 激素 + 免疫抑制剂（mycophenolate 霉酚酸酯 或 cyclophosphamide 环磷酰胺）** |
| V 膜性 | 膜性 → 肾病综合征 | 蛋白尿为主 |
| VI 硬化 | 终末硬化 | 不可逆 |

> [!tip] 锁定：anti-dsDNA↑ + C3/C4↓ + 蛋白尿 → 狼疮肾炎活动 → **活检**（IV 型弥漫增殖最常见，需强免疫抑制）。详见 [[完整笔记/专题笔记/肾脏/肾脏_肾炎肾病综合征对照]]；分科详表见 [[完整笔记/Peixuan分科笔记/肾脏]]（LN I–VI 一页速记）。

> [!info] 病理对标（高频病理题，来源：分科笔记 肾脏 LN 详表）
> - **IV 型弥漫增殖**：电镜见**内皮下沉积 (subendothelial deposits)** → 光镜"**wire-loop 铁丝圈**"病变。
> - **V 型膜性**：**上皮下沉积 (subepithelial deposits)** → "**spike & dome 钉突-圆顶**"形态（同膜性肾病机制）。
> - 记忆：增殖型(III/IV) = 内皮下 + wire-loop + 肾炎型 + 需强免疫抑制；膜性(V) = 上皮下 + spike&dome + 肾病综合征。

> [!warning] 补体 C3/C4 模式速判（鉴别低补体肾炎，来源：分科笔记 肾脏"低补体 GN"段）
> 确认肾炎 + **补体低** → 看模式选病：
> - **C3 单独↓、C4 正常**（旁路途径）→ 偏 **PSGN (post-streptococcal GN, 链球菌感染后肾炎)**（且 C3 多在 6–8 周自发回升）。
> - **C3 + C4 都↓**（经典途径消耗）→ 偏 **SLE 狼疮肾炎 / 冷球蛋白血症 / 感染性心内膜炎**。
> - SLE 活动时 **anti-dsDNA↑ 与 C3/C4↓ 同步**，是监测复发的动态指标。

## 五、药物性狼疮 Drug-Induced Lupus（独立高频考点）

> [!warning] HIP 药 + anti-histone + 肾/CNS 豁免
> | 要点 | 内容 |
> |---|---|
> | **诱因药（HIP）** | **H**ydralazine 肼屈嗪 / **I**soniazid 异烟肼 / **P**rocainamide 普鲁卡因胺（也常考 minocycline 米诺环素、TNF-α 抑制剂）|
> | **抗体** | **anti-histone 抗组蛋白抗体阳性**（anti-dsDNA/anti-Sm 多阴性）|
> | **临床** | 关节痛 + 浆膜炎 + 发热为主；**典型豁免肾脏和中枢神经** |
> | **处理** | **停药即缓解**（数周内），一般无需长期免疫抑制 |

> [!tip] 记忆锁定：**"HIP 药致狼疮、anti-histone 阳、停药就好、不碰肾和脑"** —— 与原发 SLE（累肾累脑、anti-dsDNA/Sm 阳）正好相反。

## 六、抗磷脂综合征 APS + 治疗 + 妊娠

> [!info] 抗磷脂综合征 (antiphospholipid syndrome, APS)
> aPL 抗体（狼疮抗凝物 / 抗心磷脂 / 抗 β2 糖蛋白1）→ **动静脉血栓 + 反复自然流产**；实验室**矛盾**：体外 **aPTT 延长**但体内**高凝**。处理：血栓事件后**长期抗凝（warfarin 华法林）**；孕期用 **LMWH (low-molecular-weight heparin, 低分子肝素) + 低剂量阿司匹林**。

> [!success] 治疗基石
> - **Hydroxychloroquine 羟氯喹 = 所有 SLE 患者基石**（降发作、改善长期预后；注意**视网膜毒性**需眼科年检）。
> - 急性发作/重症：**糖皮质激素**；器官威胁（肾/CNS）加**免疫抑制剂**（mycophenolate / cyclophosphamide）。
> - **防晒**（光敏感）+ 监测心血管（SLE 加速动脉粥样硬化，是远期主要死因之一）。

> [!danger] 妊娠 / 新生儿狼疮（anti-Ro/SSA）
> 母体 **anti-Ro/SSA + anti-La/SSB** 经胎盘 → **新生儿狼疮**：暂时性皮疹/血细胞↓（可逆）+ **先天性心脏传导阻滞 (congenital heart block)（不可逆，需起搏器）**。孕前查抗体，孕期胎心监测。

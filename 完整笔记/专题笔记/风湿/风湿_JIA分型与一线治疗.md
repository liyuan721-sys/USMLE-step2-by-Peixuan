# 幼年特发性关节炎 JIA 分型与一线治疗 (Juvenile Idiopathic Arthritis)

> [!info] 用途与来源
> 把散在 [[完整笔记/专题笔记/风湿/风湿_关节炎鉴别诊断决策树]]（上游单/寡/多关节分诊）、[[完整笔记/专题笔记/风湿/风湿_HighYield速查总览]]（全身型 Still 病）、[[完整笔记/专题笔记/儿科/儿科_HighYield速查总览]] 里的 JIA 碎片整合成**单一真相源**：三大分型怎么分、各型招牌与并发症、一线治疗阶梯、眼科筛查。
> 触发错题：[[mistakes/uworld-mistakes_2026-07#^Q16830]]（多关节型 → NSAID，被素食/缺乏干扰）。
> **JIA (juvenile idiopathic arthritis, 幼年特发性关节炎)** 定义 = **16 岁前**起病、单/多关节炎 **持续 ≥6 周**、**排除其他病因**（感染/反应性/白血病/SLE 等）后的诊断，是儿童**最常见的慢性关节炎**。

---

## §一 三大分型一表分清 ⭐⭐⭐

> [!danger] 先记这张表——分型决定 ANA/RF、眼病风险、预后、治疗
> 三型按"**累及关节数 + 全身症状**"切开：**全身型**（发热皮疹主导）/ **寡关节型**（≤4 关节，最常见）/ **多关节型**（≥5 关节）。

| 维度           | 全身型 Systemic (Still)                               | 寡关节型 Oligoarticular                                                              | 多关节型 Polyarticular                          |
| ------------ | -------------------------------------------------- | -------------------------------------------------------------------------------- | ------------------------------------------- |
| **占比**       | ~10%                                               | **~50%（最常见）**                                                                    | ~40%                                        |
| **起病年龄**     | 任何年龄（<16）                                          | **幼儿 2–4 岁**                                                                     | **双峰：2–5 岁 + 10–14 岁**                      |
| **性别**       | 男 = 女                                              | **女 > 男**                                                                        | **女 > 男**                                   |
| **关节数**      | ≥1（可变）                                             | <font color="#ff0000">**≤4，不对称</font>、下肢大关节（<font color="#ff0000">膝</font>最常见）** | <font color="#ff0000">**≥5，对称</font>，含小关节** |
| **全身症状**     | **弛张高热 (quotidian fever) + 鲑鱼色一过性皮疹 + 肝脾大 + 淋巴结大** | 无/轻                                                                              | 无/轻（少数低热乏力）                                 |
| **RF 类风湿因子** | **阴**                                              | **阴**                                                                            | 多数阴；**少数 RF 阳（青春期女，预后差、似成人 RA）**            |
| **ANA 抗核抗体** | 阴                                                  | **<font color="#ff0000">ANA</font>常阳 → <font color="#ff0000">眼病高危**</font>       | 可阳                                          |
| **葡萄膜炎风险**   | 低                                                  | **最高（慢性前葡萄膜炎，无症状）**                                                              | 中                                           |
| **招牌/关联**    | ferritin 铁蛋白显著↑；并发 MAS 巨噬细胞活化综合征                   | ANA 阳 + 无症状慢性虹膜睫状体炎                                                              | RF 阳亚型 → 类似成人 RA 侵蚀                         |
| **一线治疗**     | **NSAID → 全身激素 / IL-1、IL-6 抑制剂**                   | **NSAID → 关节腔激素 → 甲氨蝶呤**                                                         | **NSAID → 甲氨蝶呤 → 生物制剂**                     |

> [!tip] 一句话锁定各型
> - **发热 + 鲑鱼色皮疹 + 关节炎 + ferritin 高 = 全身型 (Still)**。
> - **幼女 + 单膝肿 + ANA 阳 + 眼睛没症状但要查裂隙灯 = 寡关节型**。
> - **≥5 关节对称 + <font color="#ff0000">晨僵日间改善</font> = 多关节型**（RF 阳亚型最凶）。

## §二 全身型 Systemic JIA / Still 病 — 发热皮疹主导

> [!info] 核心三联 + 系统征
> 全身型 JIA (systemic JIA, 又称 Still 病) 关节炎可**晚于全身症状数周至数月**出现，早期像"感染/恶性肿瘤查不出源"：
> - **弛张热 (quotidian fever, 弛张热)**：每日 1–2 次高峰 ≥39℃、峰间回落正常，**≥2 周**。
> - **鲑鱼色一过性皮疹 (evanescent salmon-colored rash)**：发热时出现、退热消失（随体温来去）。
> - **关节炎** + **肝脾大 (hepatosplenomegaly) + 淋巴结大 + 浆膜炎（心包积液）**。
> - 实验室：**RF / ANA 正常**、**铁蛋白 ferritin 显著↑**、WBC/血小板/ESR/CRP↑（炎症风暴）。

> [!danger] 必须警惕的致命并发症：MAS 巨噬细胞活化综合征
> **MAS (macrophage activation syndrome, 巨噬细胞活化综合征)** = 全身型 JIA 的急症，本质是**继发性噬血细胞性淋巴组织细胞增多症 (secondary HLH, hemophagocytic lymphohistiocytosis)**：
> - 持续高热 + **血细胞三系下降** + **肝功能恶化 + 凝血障碍 (DIC, disseminated intravascular coagulation 弥散性血管内凝血)** + 神经症状。
> - 招牌反转：**ferritin 极度↑、而 ESR 反而↓（纤维蛋白原被消耗）**、甘油三酯↑。
> - 处理：大剂量糖皮质激素 ± 环孢素 (cyclosporine) / IL-1 抑制剂，急症。

> [!tip] 治疗：全身型走"抗 IL-1 / IL-6"路线（与其他型不同）
> 全身型 JIA 炎症由 **IL-1 / IL-6（白细胞介素 1 / 6）** 驱动 → **NSAID (nonsteroidal anti-inflammatory drug, 非甾体抗炎药) → 全身糖皮质激素 → IL-1 (anakinra 阿那白滞素) / IL-6 (tocilizumab 托珠单抗) 抑制剂**；甲氨蝶呤对全身型效果不如对多关节型。

## §三 寡关节型 Oligoarticular — 最常见 + 眼病最高危 ⭐⭐

> [!info] 招牌：幼女、单膝、ANA 阳、眼睛"哑"病
> - **最常见亚型**，典型 = **2–4 岁幼女**、**≤4 个关节、不对称、下肢大关节（膝最常见）**，关节肿但全身状态好。
> - **ANA 常阳 → 慢性前葡萄膜炎 (chronic anterior uveitis, 慢性前葡萄膜炎) 风险最高**，且**无症状（不红不痛）** → 不查会致盲。

> [!danger] 眼科筛查是本型的命门（高频跨科考点）
> **无症状慢性前葡萄膜炎**是 JIA 头号致残并发症，**寡关节型 + ANA 阳 + 幼龄**风险最高。
> - **确诊 JIA 即转眼科做裂隙灯 (slit-lamp) 检查**，之后**定期复查**（ANA 阳/低龄者约每 3 个月）。
> - 不能等眼红眼痛——JIA 葡萄膜炎**通常无症状**，等有症状常已并发（带状角膜病变、白内障、青光眼）。
> - 记忆钩子：**"寡关节 + ANA 阳 = 眼睛定期裂隙灯"**。

> [!tip] 治疗：局部为主
> **NSAID → 关节腔糖皮质激素注射 (intraarticular corticosteroid)（关节少，局部即可）→ 甲氨蝶呤 methotrexate（顽固/进展）**。葡萄膜炎另需局部激素 ± 甲氨蝶呤/TNF 抑制剂。

## §四 多关节型 Polyarticular — ≥5 关节对称（Q16830 落点）⭐

> [!info] 招牌：≥5 关节、对称、晨僵日间改善
> - **6 个月内累及 ≥5 个关节**，**对称**，含**小关节（手/腕）+ 大关节（膝/踝/肘/腕）**。
> - **双峰发病**（幼儿 + 青春期女孩），女 > 男。
> - 炎症性关节炎招牌：**晨僵 (morning stiffness) + 活动后日间改善 + 关节肿胀压痛**（与机械性/缺乏性痛"越用越痛、不肿"相反）。
> - 两个亚型按 **RF**：**RF 阳**（青春期女，侵蚀性、似成人 RA、预后差）vs **RF 阴**（较轻）。

> [!warning] Q16830 的陷阱：别被"素食 + 乏力 + 体重降"带去营养缺乏
> 14 岁女、双踝起病蔓延至膝/肘/腕对称肿 + **晨僵日间改善** → **多关节型 JIA → 一线 NSAID（萘普生 naproxen）**。
> - **晨僵 + 日间改善 + 对称关节肿 = 炎症性关节炎**的硬指标；**维 D/钙缺乏**是**酸痛不肿、日间不改善**，选它是掉进饮食史干扰。
> - 本题各选项归属见 §六 干扰项表。源：[[mistakes/uworld-mistakes_2026-07#^Q16830]]

> [!tip] 治疗：全身用药阶梯
> **NSAID（一线，如萘普生 naproxen）→ 甲氨蝶呤 methotrexate（首选 DMARD, disease-modifying antirheumatic drug 改善病情抗风湿药）→ 生物制剂（TNF (tumor necrosis factor, 肿瘤坏死因子) 抑制剂 etanercept 依那西普 等）**。关节多，不靠关节腔局部注射。

## §五 一线治疗阶梯总图（按分型分流）

```
JIA 确诊（<16 岁 + 关节炎 ≥6 周 + 排除他病）
   │
   ├─ 全身型 Systemic (Still) ── NSAID → 全身激素 → IL-1(anakinra)/IL-6(tocilizumab)
   │        └─ 警惕 MAS：三系↓ + ferritin↑↑ + ESR↓ + 肝损/DIC → 急救激素±环孢素
   │
   ├─ 寡关节型 Oligo ────────── NSAID → 关节腔激素注射 → 甲氨蝶呤(顽固)
   │        └─ ANA 阳 → 眼科裂隙灯定期筛查（无症状慢性前葡萄膜炎）
   │
   └─ 多关节型 Poly ─────────── NSAID → 甲氨蝶呤(首选DMARD) → 生物制剂(TNF抑制剂)
            └─ RF 阳亚型 = 侵蚀重、预后差，早上 DMARD

通用一线 = NSAID；DMARD 首选 = 甲氨蝶呤 methotrexate；
生物制剂按型选（全身型 IL-1/IL-6；多关节 TNF）。
```

> [!tip] 三个"别弄反"
> - **全身型走 IL-1/IL-6**，多关节型走 **甲氨蝶呤 + TNF 抑制剂**——生物靶点不同。
> - **寡关节型靠关节腔局部注射**（关节少）；多关节型靠**全身 DMARD**（关节多）。
> - **一线永远先 NSAID**；不是一上来就激素/生物制剂（除非全身型重症或 MAS）。

## §六 易混陷阱与干扰项（普适）⭐

> [!warning] Q16830 五选项各归各病（"儿童关节痛"鉴别反射）
> 
> | 选项 | 对应病 | 一句话钩子 |
> |---|---|---|
> | **萘普生 naproxen** ✅ | 多关节型 JIA | 对称多关节肿 + 晨僵日间改善 |
> | 头孢曲松 ceftriaxone | 淋球菌关节炎 (gonococcal) | 脓疱疹 + 腱鞘炎 + 性活跃 |
> | 多西环素 doxycycline | 莱姆关节炎 (Lyme) | 疫区 + 游走性红斑 → 单膝大积液 |
> | 左甲状腺素 levothyroxine | 甲减 | 关节痛 (arthralgia) 非真关节炎、生长速度↓ |
> | 维 D + 钙 | 维 D/钙缺乏 | 全身骨肌痛、日间不改善、无关节肿 |

> [!danger] JIA 之外必须排除的"红旗"——白血病伪装关节痛
> 儿童关节/骨痛别只想 JIA，**急性淋巴细胞白血病 (ALL, acute lymphoblastic leukemia, 急性淋巴细胞白血病) 可伪装关节炎**：
> - 红旗：**夜间痛/痛醒、非关节的骨痛、与关节炎不成比例的重、血细胞三系异常（贫血/血小板↓/异常 WBC）、肝脾淋巴结大 + 体重下降**。
> - 遇发热 + 骨痛 + 血象异常 → 查外周血涂片 ± 骨髓，别急着按 JIA 上激素（激素会掩盖白血病）。

> [!tip] 炎症性关节炎 vs 其他儿童腿痛的分水岭
> - **JIA（炎症）** = 晨僵 + 日间改善 + 关节肿胀压痛、病程 ≥6 周。
> - **生长痛 (growth pains)** = 傍晚/夜间双腿痛、白天正常、**无肿无压痛、查体正常**、不影响活动。
> - **化脓性关节炎 (septic arthritis)** = 单关节急性红肿热 + 发热 + 拒动、滑液 WBC >50,000（急诊穿刺）。
> - **反应性关节炎 (reactive arthritis)** = 前驱 GU/GI 感染 1–4 周后、不对称寡关节 + 附着点炎。
> - **一过性滑膜炎 (transient synovitis)** = 病毒后髋痛跛行、状态好、自限（Kocher 标准鉴别化脓）。

## §七 自测（答完再判分）

> [!question]- Q1. 3 岁女孩右膝无痛性肿胀 6 周，行走略跛但玩耍如常，无发热皮疹，ANA 阳性、RF 阴性。除关节治疗外，最重要的下一步筛查是什么？为什么？
> **寡关节型 JIA** → **转眼科做裂隙灯 (slit-lamp) 检查排慢性前葡萄膜炎**，并定期复查。该型 + ANA 阳眼病风险最高且**无症状**，不查会致盲。

> [!question]- Q2. 5 岁男孩弛张高热 2 周（每日峰 ≥39℃）、发热时躯干鲑鱼色一过性皮疹、肝脾大、多关节痛；RF/ANA 阴、ferritin 显著升高。诊断？若随后出现三系下降 + ESR 反降 + 肝功恶化，警惕什么？
> **全身型 JIA (Still 病)**；随后 = **MAS 巨噬细胞活化综合征（继发 HLH）**，急症（大剂量激素 ± 环孢素/IL-1 抑制剂）。

> [!question]- Q3. 13 岁女，双踝起病 3 个月内蔓延至膝/肘/腕对称肿胀压痛，晨僵、活动后日间改善；素食、乏力、体重百分位略降。最合适的药物？为何不是维 D + 钙？
> **多关节型 JIA → 萘普生 naproxen（一线 NSAID）**。晨僵 + 日间改善 + 对称关节肿 = 炎症性关节炎；维 D/钙缺乏是酸痛不肿、日间不改善。源 Q16830。

> [!question]- Q4. 全身型 JIA 与多关节型 JIA 在生物制剂选择上有何本质区别？
> **全身型走 IL-1 (anakinra)/IL-6 (tocilizumab) 抑制剂**（IL-1/IL-6 驱动）；**多关节型走甲氨蝶呤 + TNF 抑制剂 (etanercept)**。靶点不同。

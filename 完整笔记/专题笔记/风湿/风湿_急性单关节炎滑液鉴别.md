# 急性单/少关节炎 滑液鉴别 (Acute Monoarthritis — Synovial Fluid Analysis)

> [!info] 用途与来源
> 把分散在 [[完整笔记/专题笔记/风湿/风湿_HighYield速查总览]]、[[完整笔记/专题笔记/感染/感染_蜱传疾病鉴别]]、[[完整笔记/专题笔记/感染/感染_STI分泌物综合征鉴别]]、[[完整笔记/专题笔记/骨科/骨科_HighYield速查总览]] 中的**滑液 WBC 分层 + 各病原特征**整合成单一真相源。
> 触发错题：[[mistakes/uworld-mistakes_2026-06#^Q3836]]（Lyme）/ [[mistakes/uworld-mistakes_2026-06#^Q15847]]（DGI）/ [[mistakes/uworld-mistakes_2026-06#^Q15845]]（婴儿化脓）。
> 关节穿刺 (arthrocentesis) 滑液三件套：**WBC + 革兰染色/培养 + 偏光镜晶体 (crystals)**。

## §一 滑液 WBC 四分层（核心钥匙）

> [!danger] 一表定病因层级
> 滑液**白细胞计数 (synovial fluid WBC)** 是区分"非炎症 vs 炎症 vs 化脓"的第一刀。

| 类别 | WBC/mm³ | 中性% | 代表病 |
|---|---|---|---|
| **正常** | <200 | <25% | — |
| **非炎症** | 200–2,000 | <25% | 骨关节炎 (OA)、创伤性积液、神经性关节病 |
| **炎症** | 2,000–50,000 | 常 >50% | **痛风 (gout) / 假性痛风 (calcium pyrophosphate deposition, CPPD)**、**类风湿关节炎 (rheumatoid arthritis, RA)**、**反应性关节炎**、**莱姆病 (Lyme) 关节炎**、**播散性淋球菌感染 (disseminated gonococcal infection, DGI)** |
| **化脓 (septic)** | **>50,000**（常 >100k） | >75% | 非淋球菌细菌性化脓关节炎（金葡 S. aureus 等） |

> [!warning] 重叠地带（易错点）
> 炎症与化脓**有重叠**：WBC 在 20,000–50,000 不能仅凭数字定论，必须结合**革兰染色/培养 + 晶体**。**WBC 越高、中性越占优 → 化脓概率越大**；但**晶体阳性不能排除合并感染**。

> [!info] 炎症层各病的完整体系（本卡只做滑液鉴别角度）
> 本卡聚焦"滑液 WBC + 培养 + 晶体"这一刀；各病的诱因 / 关节外表现 / 治疗阶梯见对应专题：
> - **反应性关节炎 (reactive arthritis)**：感染后无菌性炎症滑液（WBC ~2,000–4,000，培养阴）→ 三联征 + 衣原体 (Chlamydia)/GI 诱因 + HLA-B27 (人类白细胞抗原 B27) + 一线 NSAIDs 见 [[完整笔记/专题笔记/风湿/风湿_脊柱关节病SpA对照]]
> - **莱姆病 (Lyme) 关节炎** → [[完整笔记/专题笔记/感染/感染_蜱传疾病鉴别]]
> - **播散性淋球菌 (DGI)** → [[完整笔记/专题笔记/感染/感染_STI分泌物综合征鉴别]]

## §二 急性单关节炎 五大病因横向对照

| 病因 | 典型 stem 钥匙 | 滑液 WBC | 革兰/培养 | 晶体 | 处理 |
|---|---|---|---|---|---|
| **细菌性化脓 (septic, 金葡)** | 发热、剧痛、不能负重 | **>50,000** | **革兰/培养常阳** | 阴 | 引流 + 抗生素 |
| **播散性淋球菌 (DGI)** | 青少年/性活跃、脓疱+腱鞘炎+游走 | **<50,000** | **培养常阴**（挑剔菌） | 阴 | 静脉头孢曲松 ceftriaxone + 泌尿核酸扩增 (NAAT) |
| **Lyme 关节炎** | 东北部林区、单膝大积液**微痛** | **10,000–25,000** | **革兰/培养阴** | 阴 | 血清抗体；口服多西环素 doxycycline |
| **痛风 gout** | 第一跖趾、夜间剧痛、酒/海鲜 | 2,000–50,000 | 阴 | **负双折光 针状 尿酸钠** | 非甾体抗炎药 (NSAID)/秋水仙碱 (colchicine)/激素 |
| **假性痛风 CPPD** | 老年、膝/腕、软骨钙化 | 2,000–50,000 | 阴 | **正双折光 菱形 焦磷酸钙** | NSAID/激素 |
| **创伤/OA** | 外伤史/慢性负重痛 | <2,000 | 阴 | 阴 | 对症 |

> [!tip] 三个高频分水岭
> - **化脓 (>50k 培养阳) vs DGI (<50k 培养阴)**：淋球菌挑剔难培养 + WBC 偏低 → 别因"培养阴/WBC 不到 5 万"就排除感染性关节炎。
> - **Lyme (10–25k, 大膝, 微痛) vs 化脓 (>50k, 剧痛, 不能负重)**：疼痛程度 + WBC 高低。
> - **痛风 vs 假性痛风**：偏光镜——尿酸钠**负**双折光针状；焦磷酸钙**正**双折光菱形。

## §三 婴儿/儿童特殊（化脓性关节炎）

> [!danger] 为什么婴儿尤其高发（机制）
> 婴儿（<18 月龄）化脓性关节炎 (septic arthritis) 高发 = **解剖直通 + 滑膜易定植 + 免疫不成熟** 三重弱点：
>
> - **跨骺板血管 (transphyseal vessels)** ★婴儿独有：干骺端 (metaphysis) → 穿未骨化骺板 (physis) → 关节腔 ⇒ **骨髓炎 (osteomyelitis) 直通关节，二者常并存**。
> - **干骺端位于关节囊内 (intracapsular metaphysis)**（髋/肩）：干骺端脓液直接破入关节 ⇒ **髋 (hip) 最高发**。
> - **滑膜 (synovium) 血供丰富、无基底膜屏障**：菌血症时细菌极易血行定植 (hematogenous seeding)。
> - **免疫不成熟**：中性粒/补体/调理素 (opsonin) 不足、母传 IgG 下降 ⇒ 菌血症难清除 + 全身体征不典型（可不发热）。
>
> - **血源播散为主**（脐部/置管/产道/皮肤呼吸道感染），非外伤直接接种。
> - **后果凶险**：细菌酶 + 中性粒蛋白酶数天内破坏无血管关节软骨 (avascular cartilage)；紧邻骺板 → 生长板损伤致肢体不等长/畸形、股骨头缺血坏死 (avascular necrosis, AVN)。⇒ **不等培养，先穿刺 + 引流 + 经验抗生素**。
> - 病原谱"<3 月龄加 **B 族链球菌 (Group B Streptococcus, GBS)** + 革兰阴性杆菌"= 与**新生儿败血症 (neonatal sepsis)** 同一套产道获得逻辑。

> [!warning] 婴儿化脓性关节炎隐匿（[[mistakes/uworld-mistakes_2026-06#^Q15845]]）
> - 体征隐匿：**假性瘫痪 (pseudoparalysis, 不动患肢)**、换尿布/搬动时异常烦躁、患**髋屈曲-外展-外旋**（扩大关节腔减压）。
> - 病原**按月龄**：**金黄色葡萄球菌 (S. aureus)** 各龄最常见；**<3 月龄**加 **B 族链球菌 (Group B Streptococcus, GBS)** + **革兰阴性杆菌**。
> - 
> - **化脓性关节炎病原 × 人群**
> - |人群|首位病原|额外覆盖|
|---|---|---|
|**新生儿 <3 月**|**S. aureus**|GBS、G− 杆菌(大肠/克雷伯)|
|1–5 岁|S. aureus|Kingella 金氏菌(G−)|
|年轻性活跃|N. gonorrhoeae 淋球菌(DGI)|—|
>**下一步**:穿刺送 Gram/培养 → **不等培养即经验抗生素**(覆金葡 + 按月龄加 GBS/G−)+ **关节引流/冲洗**(急症,防软骨/骺板破坏)。
> - 实验室：白细胞↑、红细胞沉降率 (ESR)/C 反应蛋白 (CRP)↑。
> - 处理：**关节穿刺 + 抗生素**（不等待即经验性覆盖）。
> - 对照：一过性滑膜炎 (transient synovitis) = 学龄、病毒后、状态好、无白细胞↑；发育性髋关节发育不良 (developmental dysplasia of the hip, DDH) = 髋不稳无炎症。
>

## §四 决策骨架

```
急性单关节肿痛
   │
   ├─ 关节穿刺 arthrocentesis（必做）
   │     ├─ WBC >50k + 中性高 + 革兰/培养阳 → 细菌化脓 → 引流+抗生素
   │     ├─ WBC <50k + 培养阴 + 脓疱/腱鞘炎/游走 + 性活跃 → DGI → 头孢曲松+NAAT
   │     ├─ WBC 10–25k + 单膝大积液微痛 + 林区/蜱 → Lyme → 抗体+多西环素
   │     ├─ 晶体阳（负针/正菱） → 痛风 / 假性痛风
   │     └─ WBC <2k 无晶体无菌 → OA/创伤
   │
   └─ 婴幼儿：假性瘫痪+发热+白细胞↑ → 化脓性关节炎（穿刺+抗生素）
            └─ 跨骺板血管 → 同时排合并骨髓炎 (osteomyelitis)（查 X 线/MRI）
```

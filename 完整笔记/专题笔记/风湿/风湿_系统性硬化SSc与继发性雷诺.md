# 系统性硬化 (SSc) 与继发性雷诺现象

> [!info] 一句话定位
> **雷诺现象 (Raynaud phenomenon, RP)** 是「入口症状」，**系统性硬化 (systemic sclerosis, SSc / 硬皮病 scleroderma)** 是它最经典的「继发病因」。两个 UWorld 高频错点都在这条线上：
> 1. 怀疑**继发** RP → **下一步先无创血清学 (ANA)，有创动脉造影靠后**（错题 [[mistakes/uworld-mistakes_2026-06#^Q9939]]）
> 2. 皮肤增厚 + RP + 肺底啰音 = 弥漫型 SSc → **anti-Scl-70**（错题 [[mistakes/uworld-mistakes_2026-06#^Q3322]]）

---

## 一、雷诺现象 Raynaud Phenomenon (RP)

寒冷 / 情绪应激诱发指（趾）端血管痉挛，典型**三色变化**：

> [!tip] 三色 = 白 → 蓝 → 红
> - **白 (white)** = 血管痉挛、缺血（pallor）
> - **蓝 (blue)** = 缺氧、发绀（cyanosis）
> - **红 (red)** = 复温后反应性充血（reperfusion hyperemia）

### 1.1 原发 vs 继发（★核心鉴别★）

| | 原发 RP (Primary) | 继发 RP (Secondary) |
|---|---|---|
| 人群 | 年轻、**女**、15–30 岁 | **>40 岁**、可男可女 |
| 发作 | **对称**、双手 | **不对称** |
| 组织损伤 | **无**（不留溃疡/坏疽） | **有：digital ulcer 指端溃疡 / 坏疽** |
| 甲皱毛细血管 (nailfold) | 正常 | **异常**（扩张/缺失） |
| 自身抗体 | 阴性 | **ANA 阳性**等 |
| 背后 | 无系统病 | **SSc / SLE (systemic lupus erythematosus 系统性红斑狼疮) / 混合结缔组织病 / 药物（amphetamine 苯丙胺）/ 血管炎** |

> [!danger] 继发 RP 的「危险信号」三连
> **>40 岁新发 + 不对称 + 组织损伤（溃疡/坏疽）** → 别当原发，必须查继发病因。

### 1.2 workup 顺序（★Q9939 错点★）

> [!warning] 先无创血清学，有创造影永远靠后
> 怀疑**继发** RP → 下一步 = **无创血清学 / 炎症筛查**，不是动脉造影。

```
怀疑继发性 RP
   │
   ├─ 第一步（无创）：CBC (complete blood count 全血细胞计数)
   │                  代谢谱、尿检
   │                  ★ ANA (antinuclear antibody 抗核抗体) + RF (rheumatoid factor 类风湿因子)
   │                  ESR (erythrocyte sedimentation rate 红细胞沉降率) + 补体 C3/C4
   │
   ├─ ANA 阳性 → 查特异抗体（anti-Scl-70 / anti-centromere → SSc；dsDNA → SLE）
   │
   └─ 动脉造影 (arteriogram)【有创，靠后】
            └─ 用于 Buerger 病，且【排除系统性病之后】才做
```

> [!danger] Buerger 病 = 造影诱饵，别上钩
> **Buerger 病（血栓闭塞性脉管炎 thromboangiitis obliterans）**：男、40–45 岁、**重度吸烟**、远端肢体缺血 / 溃疡 / 坏疽。它确实会被「吸烟 + 指端溃疡」勾起来 → 诱你选**动脉造影**。但 Buerger 是**排他性诊断**，要先用无创血清学（ANA 等）排除自身免疫/高凝病因。**所以 ANA 排在造影前。**

### 1.3 治疗

- **原发 RP**：保暖 / 戒烟 / 避免冷与应激；需药 → **CCB (calcium channel blocker 钙通道阻滞剂)：nifedipine 硝苯地平 / amlodipine 氨氯地平**（一线，二氢吡啶类）。
- 不耐受 CCB → **losartan 氯沙坦 / PDE-5 抑制剂**（二线）。
- **继发 RP**：治疗原发病 + 同上扩血管；指端溃疡风险高者考虑 **aspirin 阿司匹林**。

## 二、系统性硬化 Systemic Sclerosis (SSc) / 硬皮病

> [!info] 本质
> 成纤维细胞功能障碍 → **结缔组织过度纤维化** + **血管病变**。皮肤增厚从肢端（手/足）起，伴皮肤附属器消失（毛囊/汗腺）。

### 2.1 弥漫型 vs 局限型 临床鉴别（★核心★）

| 鉴别点 | 弥漫型 Diffuse cutaneous | 局限型 Limited / CREST |
|---|---|---|
| 皮肤范围 | **躯干 + 四肢近端（肘/膝以上）+ 面** | **仅肘/膝以下远端 + 面**（不累及躯干） |
| 起病 / 进展 | **快**，皮肤短期内快速增厚 | **慢**，数年隐匿 |
| Raynaud 时序 | 与皮肤改变**几乎同时**（短期内即出现内脏病） | **Raynaud 早于皮肤改变数年** |
| 内脏受累 | **早、重**：ILD (interstitial lung disease 间质性肺病) / **硬皮病肾危象** | **晚、局限**：**PAH (pulmonary arterial hypertension 肺动脉高压)**（迟发）/ PBC (primary biliary cholangitis 原发性胆汁性胆管炎) |
| 钙质沉着 / 毛细血管扩张 | 较少 | **突出**（即 CREST 的 C 和 T） |
| 特异抗体 | **anti-Scl-70（anti-topoisomerase I 抗拓扑异构酶 I）**；anti-RNA polymerase III（抗 RNA 聚合酶 III，肾危象↑） | **anti-centromere 抗着丝点抗体** |
| 预后 | **差**（内脏纤维化早） | 相对好 |

> [!tip] CREST = 局限型硬皮病五联
> **C**alcinosis 钙质沉着 / **R**aynaud 雷诺 / **E**sophageal dysmotility 食管动力障碍 / **S**clerodactyly 指端硬化 / **T**elangiectasia 毛细血管扩张。

> [!info] 两型「共有」特征 → 不能用于分型
> - **Raynaud 现象**：几乎所有 SSc 都有（两型的入口症状）。
> - **食管动力障碍 + GERD**：~80–90%，**弥漫 / 局限都有**——CREST 把「E」列进去，但**食管病变不是局限型专属**，不能拿来分型。
> - **sclerodactyly 指端硬化**：两型都有（定义性皮肤征）。
> - 分型 = 看**皮肤范围 + 特异抗体 + 哪种内脏并发为主（ILD/肾危象 vs PAH）**，不是看"有没有雷诺/食管病"。

> [!danger] 抗体一句话锁定（★Q3322 错点★）
> - **ANA** 几乎所有 SSc 都阳性 → 敏感但**不特异**（不能定型）。
> - **anti-Scl-70** → **弥漫型 + 广泛病变（含 ILD）**，特异。
> - **anti-centromere** → **局限型 / CREST**。
> 题干给「皮肤增厚 + Raynaud + 肺底细湿啰音 (ILD)」= 弥漫型 → **选 anti-Scl-70**，不是抗着丝点、更不是抗 CCP（那是 RA）。

### 2.2 临床特征（按系统）

| 系统 | 表现 |
|---|---|
| 皮肤 | 指端硬化 sclerodactyly、puffy hands 手肿胀、telangiectasia、calcinosis、digital ulcer |
| 血管 | **Raynaud（几乎人人有）** |
| 消化 | **食管动力障碍 → dysphagia 吞咽困难 / GERD (gastroesophageal reflux disease 胃食管反流病)**（LES, lower esophageal sphincter 下食管括约肌 张力↓） |
| 肺 | **ILD 间质性肺病**（弥漫型，肺底 fine crackles 细湿啰音 + 气短）、PAH（局限型） |
| 肾 | **硬皮病肾危象 scleroderma renal crisis** |
| 心 | 心肌纤维化、心包炎/积液 |

### 2.3 硬皮病肾危象（★高频急症★）

> [!danger] Scleroderma Renal Crisis
> **恶性高血压 (malignant HTN, hypertension 高血压) + AKI (acute kidney injury 急性肾损伤) + MAHA (microangiopathic hemolytic anemia 微血管病性溶血性贫血)**（血涂片见破碎红细胞 schistocyte）。
> **首选 ACE-I（angiotensin-converting enzyme inhibitor 血管紧张素转换酶抑制剂）：captopril 卡托普利**——即使肌酐升高也要用，是唯一改善预后的药。
> ★硬皮病患者突发明显高血压 → 先想肾危象，立刻 ACE-I★

### 2.4 分系统处理 SOP

| 问题 | 处理 |
|---|---|
| Raynaud | **CCB (nifedipine 硝苯地平)** |
| 皮肤增厚 | penicillamine 青霉胺、免疫抑制 |
| 食管/GERD | **PPI (proton pump inhibitor 质子泵抑制剂) 终身** |
| 高血压 / 肾危象 | **ACE-I（首选）** |
| PAH 肺动脉高压 | bosentan 波生坦 + prostacyclin 前列环素 |
| ILD | 免疫抑制（mycophenolate 霉酚酸酯等） |

## 三、SSc vs RA 鉴别（Q3322 干扰项）

> [!warning] 「手肿胀僵硬」≠ 类风湿
> 患者手肿+晨僵很容易反射 **类风湿关节炎 (rheumatoid arthritis, RA) → anti-CCP (anti-cyclic citrullinated peptide 抗环瓜氨酸肽抗体)**。分水岭：

| | SSc | RA |
|---|---|---|
| 皮肤 | **皮肤增厚 / sclerodactyly** | 无 |
| Raynaud | **几乎都有** | 少见 |
| 关节 | 僵、挛缩为主 | **对称侵蚀性多关节炎**（晨僵 >1h、骨侵蚀） |
| 抗体 | ANA / Scl-70 / 着丝点 | **RF / anti-CCP** |
| 肺 | **ILD（弥漫型）** | 可有 ILD，但关节症状更突出 |

## 四、整体决策树

```
指端遇冷变白→蓝→红 = Raynaud
   │
   ├─ 年轻女 / 对称 / 无溃疡 / 化验正常 ──► 原发 RP → 保暖+戒烟(+CCB)
   │
   └─ >40 / 不对称 / 溃疡坏疽 ──► 继发 RP
            │ 第一步：ANA + 炎症标志物（无创！造影靠后）
            │
            ├─ 皮肤增厚 + 广泛 + 肺底啰音(ILD) → 弥漫型 SSc → anti-Scl-70
            ├─ CREST 五联 + PAH → 局限型 SSc → anti-centromere
            ├─ malar rash + dsDNA → SLE
            └─ 男+重度吸烟+排除自免后 → Buerger → 此时才动脉造影
```

## 五、高频陷阱（个人错点提炼）

> [!danger] 两个焊死点
> 1. **继发 RP 的下一步是 ANA，不是动脉造影**——有创检查永远排在无创血清学之后；造影是排除系统病后查 Buerger 用的。
> 2. **弥漫型 SSc（含 ILD）= anti-Scl-70**；局限型/CREST = anti-centromere；「手肿胀」别反射 RA/anti-CCP（SSc 有皮肤增厚 + Raynaud + 肺底啰音三联）。

## 六、复习题

> [!question]- Q1. 47 岁女，指端遇冷变白发绀 6 月、右示指尖小溃疡，不对称。最佳下一步？
> **ANA + 炎症标志物**（无创血清学）。这是继发 RP（>40 / 不对称 / 溃疡），先排自免病；动脉造影有创、靠后。

> [!question]- Q2. 50 岁女，皮肤增厚 + Raynaud + 肺底细湿啰音 + 气短。最可能阳性抗体？
> **anti-Scl-70 (anti-topoisomerase I)**——弥漫型 SSc 伴 ILD 的特异抗体。

> [!question]- Q3. 45 岁硬皮病女突发 BP 210/120、肌酐快速升高、血涂片见破碎红细胞。首选药？
> **ACE-I（captopril 卡托普利）**——硬皮病肾危象，即使肌酐升高也要用。

> [!question]- Q4. CREST 综合征对应哪个抗体？哪型 SSc？
> **anti-centromere 抗着丝点**；**局限型 SSc**（标志并发 = PAH 肺动脉高压）。

## 〇、关节炎诊断框架（先定性）

```
非创伤关节肿胀
  ├ 感染性 Septic ── 急性/不能负重/单关节/WBC↑炎症标志物↑ → 紧急穿刺
  ├ 炎症-风湿 ───── 亚急慢/晨僵重/可负重/多关节/贫血RBC↓ → 自身抗体
  └ 肿瘤 Neoplastic ── 夜间-傍晚痛/WBC·血小板↓

分布 pattern：
  单关节 Mono → Gout / OA / Septic
  寡关节 Oligo → 脊柱关节病 Spondyloarthropathy
  多关节 Poly → SLE / RA / 病毒
  游走 Migratory → "Michael Jordan" = Gonococcal 淋球菌 / Lyme 莱姆 / Rheumatic fever 风湿热
```

**关节液 WBC 分级（高频）**

| | 正常 | 非炎症 (OA) | 炎症 (晶体/RA) | 化脓 Septic |
|---|---|---|---|---|
| WBC/mm³ | <200 | 200–2,000 | 2,000–100,000 | **>50,000**（PMN>80–90%） |

> [!tip] Reactive arthritis 滑液 WBC 仅 2,000–4,000、Gram/培养阴性；Pseudogout 急性 15,000–30,000。

## 一、退行性 & 炎症性关节炎

### OA vs RA（最高频对比）

| | **OA 骨关节炎** | **RA 类风湿关节炎** |
|---|---|---|
| 年龄/性别 | >40–60，随龄增 | 20–40 女性多 |
| 关节 | 膝/髋、**DIP (Heberden)**、PIP (Bouchard)、第一腕掌 | **MCP / PIP / 腕、spares DIP**、对称 |
| 晨僵 | 无/短(10–15min) | **>1 小时**、活动后改善 |
| 全身症状 | 无 | 发热/乏力/体重↓ |
| 查体 | 硬骨性增大、crepitus | 软/温热、海绵样 |
| X 线 | 间隙窄+骨赘+软骨下硬化 | **边缘性骨侵蚀 marginal erosions**+关节周围骨质疏松 |
| 抗体 | — | **RF（敏感初筛）、anti-CCP（更特异+预后差）** |
| 治疗 | 减重/肌力/PT → **acetaminophen 对乙酰氨基酚** → NSAIDs(+PPI 或 celecoxib 塞来昔布) → 置换 | flare 短期激素 → **DMARDs** |

![[风湿_手OA结节.png]]
*手骨关节炎 (osteoarthritis)：DIP 远端指间关节 Heberden 结节 + PIP 近端指间关节 Bouchard 结节。*

> [!warning] OA 陷阱：ACL（前交叉韧带）损伤 10 年内发继发性 OA（不论是否手术）；OA 非炎症 → 重机械支持+肌力，不是抗炎主导。

> [!tip] RA 畸形：Swan-neck 天鹅颈（PIP 过伸+DIP 屈）/ Boutonniere 纽扣花（PIP 屈+DIP 过伸）。

**DMARD 用药速记**

| 药 | 机制 | 关键副作用/用前筛查 |
|---|---|---|
| **Methotrexate 甲氨蝶呤** | 抑二氢叶酸还原酶 | 肝毒/口腔溃疡/骨髓抑制(大细胞贫血)/肺纤维化；**补叶酸 folic acid** |
| **Leflunomide 来氟米特** | 抑嘧啶合成 | 肝毒、血细胞减少 |
| **Hydroxychloroquine 羟氯喹** | — | **视网膜病变**；用前排黄斑变性+G6PD 缺乏 |
| **Sulfasalazine 柳氮磺吡啶** | — | 肝毒/口腔溃疡/溶血 |
| **TNF 抑制剂**（adalimumab 阿达木 / etanercept 依那西普 / infliximab 英夫利昔） | 抗 TNF-α | **用前 PPD/IGRA 排结核**；感染/脱髓鞘/CHF |

> [!danger] RA 关节外表现
> - **Felty 综合征** = RA + 脾大 + 中性粒减少
> - **Caplan 综合征** = RA + 尘肺 + 肺结节
> - **寰枢椎半脱位 atlantoaxial subluxation** = 插管/全麻前必查 → 张口位颈椎 X 线（亦见 Down 综合征）
> - 肺：间质性肺病；**渗出性胸腔积液（葡萄糖极低、LDH 高、pH 低）**
> - **Baker cyst 腘窝囊肿**（似 DVT 深静脉血栓）

### 脊柱关节病（血清阴性，HLA-B27）

> [!info] PEAR 字母表
> **P**soriatic 银屑病 / **E**(IBD) 炎症性肠病相关 / **A**nkylosing 强直性脊柱炎 / **R**eactive 反应性。共性：HLA-B27、附着点炎 enthesitis、香肠指 dactylitis、葡萄膜炎 uveitis、RF 阴性。

**Ankylosing Spondylitis 强直性脊柱炎**
- 年轻男 <40，炎症性背痛 >3 月，**晨僵重、运动改善、夜间加重**
- 体征：**Schober test** 前屈↓、骶髂关节炎、胸廓扩张↓、Achilles enthesitis、uveitis
- 并发：**主动脉瓣反流 AR**、椎体骨折、cauda equina 马尾
- 检查：**骶髂关节 X 线**（晚期 **bamboo spine 竹节脊柱**）；**MRI 最敏感**；HLA-B27 仅易感；基线 PFT
- 治疗：PT+NSAIDs → 失败 **TNF-α 抑制剂** 或 **抗 IL-17 (secukinumab 司库奇尤)**

**Psoriatic Arthritis 银屑病关节炎**
- **DIP 受累 + nail pitting 甲凹点 + 银屑病鳞屑斑 + dactylitis**；X 线 **"pencil-in-cup 笔入帽"**
- 陷阱：DIP+指甲改变 → PsA 不是 RA（RA spares DIP）

**Reactive Arthritis 反应性关节炎**
- 感染后 **1–4 周**、下肢大关节、**不对称寡关节炎**
- 前驱：GU = Chlamydia trachomatis 沙眼衣原体；GI = Salmonella/Shigella/Yersinia/Campylobacter
- 三联：**"不能看（结膜炎/葡萄膜炎）、不能尿（尿道炎）、不能爬（关节炎）"** + circinate balanitis 环状龟头炎
- 治疗：治原发感染 + NSAIDs

![[风湿_环状龟头炎反应性关节炎.png]]
*环状龟头炎 (circinate balanitis)：龟头**地图样斑 + 白色隆起边界**——反应性关节炎 (reactive arthritis) 的经典皮肤黏膜征。*

**Still / JIA 成人 Still 病 / 幼年特发性关节炎**
- **每日高峰热 + 鲑鱼色一过性皮疹 + 关节痛**；**RF/ANA 正常、ferritin 铁蛋白显著↑** → Steroids

## 二、结缔组织病

### SLE 系统性红斑狼疮
- 钩子：**Malar rash 颊部红斑（不累及鼻唇沟）**、对称多关节痛（**非侵蚀性**）、浆膜炎、肾（**膜性/膜增生性**）、Raynaud
- 抗体：**ANA（最敏感初筛）→ anti-dsDNA（特异+关联狼疮肾炎/活动度）→ anti-Smith（最特异）**；**活动期 C3/C4↓**
- 治疗：器官受累→系统激素±免疫抑制；malar rash→防晒+外用激素
- 生育：受孕正常，问题在**维持妊娠**（胎盘灌注差→流产，尤其合并 APS）

> [!warning] Drug-Induced Lupus 药物性狼疮
> 触发药 **"HIPP-EQ"**：Hydralazine 肼苯哒嗪 / Isoniazid 异烟肼 / Phenytoin 苯妥英 / Penicillamine 青霉胺 / Procainamide 普鲁卡因胺 / Ethosuximide 乙琥胺 / Quinidine 奎尼丁。
> 抗体 = **anti-histone 抗组蛋白**（dsDNA 正常、补体正常）；肾/CNS 罕受累；**停药即改善**。

### APS 抗磷脂综合征
- 动/静脉血栓 + 反复流产 + 胎盘灌注差
- 抗体：**lupus anticoagulant（体外 aPTT 延长但体内高凝）**、anticardiolipin（梅毒试验假阳性）、anti-β2 糖蛋白 I
- 妊娠：**LMWH 低分子肝素 + 低剂量 aspirin 阿司匹林**；产后转 **warfarin 华法林**（孕期禁用，过胎盘）

### Systemic Sclerosis 系统性硬化 / 硬皮病

| | **Diffuse 弥漫型** | **Limited / CREST** |
|---|---|---|
| 范围 | 广泛皮肤+内脏 | 末梢为主、进展慢 |
| 抗体 | **Scl-70 (anti-topoisomerase I)** | **anti-centromere 抗着丝点** |

- CREST = Calcinosis 钙质沉着 / Raynaud 雷诺 / Esophageal dysmotility 食管动力障碍 / Sclerodactyly 指端硬化 / Telangiectasia 毛细血管扩张
- 分系统处理：
  - Raynaud → **CCB (nifedipine 硝苯地平)**
  - 皮肤增厚 → penicillamine 青霉胺
  - 食管/GERD → **PPI 终身**
  - 高血压 → **ACE-I**
  - **Scleroderma renal crisis 硬皮病肾危象** = 恶性高血压+AKI+MAHA → **ACE-I（首选）** ★硬皮病出现明显高血压先想肾危象★
  - **PAH 肺动脉高压** → bosentan 波生坦 + prostacyclin 前列环素；体征右侧 S4

### 炎性肌病 PM / DM
- 共性：**对称性近端肌无力**（上楼/起立/梳头难）；CK/aldolase↑；最准 = **肌活检**；治疗 Steroids
- **PM 多发性肌炎**：细胞介导、无皮肤征
- **DM 皮肌炎**：补体介导 + **Heliotrope 紫红眼睑 + Gottron 丘疹（关节伸侧）**；抗体 **anti-Jo-1**（提示 ILD）
- **DM/PM ↑ 恶性肿瘤风险（尤其 DM）→ 诊断后做年龄相应肿瘤筛查**
- **【源笔记仅列名·补充】Inclusion body myositis 包涵体肌炎**：老年男性、**远端+近端均累及**、对激素反应差

**肌无力 mimic 鉴别（高频）**

| 病因 | 痛/压痛 | CK | ESR | 关键 |
|---|---|---|---|---|
| 糖皮质激素肌病 | 无 | 正常 | 正常 | 下肢萎缩 |
| PM/DM | 少 | ↑ | 正常/↑ | 多系统、↑恶性肿瘤 |
| Statin 他汀肌病 | 有 | 正常/↑ | 正常 | 停药仍重→免疫介导坏死性肌病 |
| 甲减肌病 | 有 | ↑ | 正常 | **DTR 延迟**、怕冷、TSH↑ |
| 甲亢肌病 | 少 | 正常/↑ | — | **DTR 亢进**、T4↑ |
| Trichinella 旋毛虫 | 有 | ↑ | — | 生/半熟野味、嗜酸↑、眼周水肿 |

## 三、血管炎 Vasculitis

### 大血管
**GCA / Temporal arteritis 巨细胞（颞）动脉炎**
- >50 岁，**jaw claudication 下颌跛行（最特异）+ 颞部头痛 + 视觉异常**；颞动脉压痛/索状；全身症状
- 视觉：amaurosis fugax 一过性黑矇 → **AION 前部缺血性视神经病（永久失明）**
- 实验室：**ESR/CRP↑**；确诊 = **颞动脉活检**

> [!danger] GCA 治疗时间窗（防失明）
> **高度怀疑立即上高剂量激素，不等活检**。
> - PMR only → 低剂量 prednisone 泼尼松 10–20 mg/d
> - GCA → 高剂量 prednisone 40–60 mg/d
> - **GCA + 视力丧失 → IV methylprednisolone 甲泼尼龙 1000 mg/d × 3 天** 后转口服

**PMR 风湿性多肌痛**
- >50 岁，**双侧肩/髋带痛+晨僵**，梳头穿衣费劲；**疼痛非肌肉破坏**
- 实验室：**ESR↑、CK 正常、肌活检正常**（与 PM 关键区别）→ **低剂量激素戏剧性缓解**；必排 GCA

**Takayasu 大动脉炎**
- **亚洲年轻女性**、**"无脉病" pulseless disease**、双上肢血压不对称、动脉杂音
- 影像 **CT/MR 血管造影**（壁增厚/管腔狭窄）→ **系统性糖皮质激素**

### 中血管 【源笔记未覆盖·补充】
**PAN 结节性多动脉炎**：**HBV 关联**；肾/肠系膜/皮肤/神经（mononeuritis multiplex），**肺不受累**；血管造影**串珠样动脉瘤**；**ANCA 阴性**；治疗 cyclophosphamide 环磷酰胺 + steroids

### 小血管 【源笔记未覆盖·补充】
| 病 | 抗体 | 特征 |
|---|---|---|
| **GPA 肉芽肿性多血管炎** | **c-ANCA / PR3** | 上(鼻窦/鞍鼻)+下呼吸道+肾小球肾炎 |
| **MPA 显微镜下多血管炎** | **p-ANCA / MPO** | 肺+肾，无肉芽肿、无上呼吸道 |
| **EGPA (Churg-Strauss)** | **p-ANCA** | **哮喘 + 嗜酸粒细胞增多** + 血管炎 |
| **Cryoglobulinemia 冷球蛋白血症** | — | **HCV 关联**、可触性紫癜+关节痛+肾炎、**低补体** |
| **IgA vasculitis (Henoch-Schönlein 过敏性紫癜)** | — | 儿童、**紫癜（下肢）+ 腹痛 + 关节炎 + IgA 肾病** |

![[风湿_HSP紫癜四联.png]]
*IgA 血管炎 / 过敏性紫癜 (Henoch-Schönlein purpura) 四联：肾病 + 腹痛/肠套叠 + 关节炎 + 踝/臀/大腿可触性紫癜。*

### 可变血管
**Behçet 白塞病**：**痛性口腔溃疡 + 痛性生殖器溃疡 + 葡萄膜炎** → Steroids + **colchicine 秋水仙碱**

## 四、晶体性关节炎

| | **Gout 痛风** | **Pseudogout / CPPD 假性痛风** |
|---|---|---|
| 晶体 | **针状 needle、负性双折射** | **菱形 rhomboid、正性双折射** |
| 成分 | monosodium urate 尿酸单钠 | calcium pyrophosphate 焦磷酸钙 |
| 部位 | 第一跖趾 (1st MTP) / 膝 | 膝（最常见） |
| X 线 | 痛风石 erosion + overhanging edge | **chondrocalcinosis 软骨钙化** |
| 关联 | 酒/红肉/利尿剂/TLS 肿瘤溶解/CKD | **甲旁亢 / 甲减 / 血色病 hemochromatosis** |

![[风湿_痛风石手.png]]
*慢性痛风石性痛风 (chronic tophaceous gout)：指间多发白垩样**痛风石 (tophi)**，可红肿溃破。*

![[风湿_痛风骨侵蚀X线.png]]
*痛风足 X 线：关节旁**穿凿样骨侵蚀 (punched-out erosion) + 悬挑边缘 (overhanging edge)**，关节间隙相对保留。*

> [!warning] Gout 急性 vs 慢性
> - 诊断：arthrocentesis 关节穿刺 → 滑液晶体分析（金标准）
> - **急性发作（抗炎，非降尿酸）**：一般 → **NSAIDs (indomethacin 吲哚美辛)**；酗酒者 → **colchicine 秋水仙碱**；PUD/肾衰 → **关节内/全身激素**
> - **慢性降尿酸（≥2 次/年）**：生活方式 + **allopurinol 别嘌醇 / probenecid 丙磺舒**；**losartan 氯沙坦**辅助促排
> - **加重痛风的药 = 噻嗪类 thiazide**（急性期勿启动/调整降尿酸药）

![[风湿_软骨钙化CPPD.png]]
*软骨钙化症 (chondrocalcinosis)：膝关节软骨内钙化线，焦磷酸钙沉积 (CPPD)/假性痛风的影像标志。*

![[风湿_CPPD偏振光晶体.png]]
*CPPD 偏振光镜检：**菱形 (rhomboid) + 正性双折射 (positively birefringent)** 焦磷酸钙晶体（对比痛风=针状负性双折射）。*

> [!tip] CPPD 必查继发病因：**甲旁亢（高 PTH→高钙）、甲减、血色病（铁沉积；2nd/3rd MCP、hook 状骨赘、bronze diabetes → phlebotomy 放血）**。

> [!danger] 发热 + 急性单关节炎 → **紧急 arthrocentesis 先排 septic arthritis**（Gram/培养、WBC>50k）。

## 五、其他

**Raynaud 雷诺现象**：寒冷/紧张诱发指端白→蓝→红；继发线索（大龄新发、溃疡/坏疽）查硬皮病/SLE → **CCB (nifedipine)**

**Fibromyalgia 纤维肌痛**：20–40 女性，**广泛 trigger point 压痛 + 睡眠障碍 + 乏力/抑郁**；**化验全正常**；须排 Cushing/甲减 → **运动/PT + amitriptyline 阿米替林 / gabapentin 加巴喷丁 / pregabalin 普瑞巴林**

**Paget Disease of Bone 骨 Paget 病**：50–60 岁，**听力↓ / 帽子变紧 / 鞋码变大 / 背痛**；破骨异常+骨转换↑；**ALP↑（可能唯一异常）、钙磷 PTH 正常**；**bone scan 最敏感**；并发 osteosarcoma 骨肉瘤 → **bisphosphonates 双膦酸盐**

### Fibromyalgia vs PM vs PMR（三联鉴别 ★高频★）

| | 人群 | 痛 vs 无力 | CK/酶 | ESR | 关键 |
|---|---|---|---|---|---|
| **Fibromyalgia** | 年轻-中年女 | 广泛痛、**无真无力** | 正常 | **正常** | trigger points、化验全正常 |
| **Polymyositis** | 成人 | **近端无力**、痛轻 | **↑** | 正常/↑ | anti-Jo-1、活检 |
| **PMR** | >50 岁 | 肩髋带**僵>痛**、无真无力 | 正常 | **↑** | 伴 GCA、激素戏剧性缓解 |

## 六、自测（答完再判分）

> [!question]- Q1. 35 岁女性双手 MCP、PIP 对称肿痛，晨僵 2 小时，活动后改善 3 个月。最特异且预后相关的抗体？一线改变病程治疗？
> **RA 类风湿关节炎**；最特异+预后差 = **anti-CCP**（RF 更敏感）；改变病程 = **DMARD（首选 methotrexate 甲氨蝶呤）**。

> [!question]- Q2. 72 岁男性新发左颞部头痛、咀嚼时下颌酸痛、一过性单眼黑矇，ESR 90。下一步处理顺序？
> **GCA 巨细胞动脉炎**；**先立即高剂量糖皮质激素（不等活检）防失明**，再做颞动脉活检确诊。有视力丧失则 IV methylprednisolone 冲击。

> [!question]- Q3. 45 岁硬皮病女性突发血压 210/120、肌酐快速升高、血涂片见破碎红细胞。首选药物？
> **Scleroderma renal crisis 硬皮病肾危象** → **ACE-I（captopril 卡托普利等，首选）**，即使肌酐升高也要用。

> [!question]- Q4. 服用肼苯哒嗪数月后关节痛+浆膜炎，无肾/CNS 受累。最可能抗体？处理？
> **Drug-induced lupus 药物性狼疮**（HIPP-EQ）→ **anti-histone 抗组蛋白抗体**（dsDNA/补体正常）→ **停药**。

> [!question]- Q5. 急性第一跖趾关节红肿剧痛，滑液见针状负性双折射晶体。慢性酗酒+消化性溃疡史者，急性期首选药？
> **Gout 痛风**；酗酒+PUD 不宜 NSAID → 首选**关节内/全身糖皮质激素**（colchicine 在酗酒者可考虑但 GI 不耐受需谨慎）；急性期不启动降尿酸药。

# 反应性关节炎 ReA — 纵向深挖（3 次反复错专攻）

> [!danger] 为什么单独建这一页
> 反应性关节炎 (reactive arthritis, ReA) 是你**反复错的重灾区**：治疗题 (Q4577) **连续 3 次**都选抗生素、表现题 (Q115763) 把环状龟头炎误当软下疳。横向 4 病对照见 [[完整笔记/专题笔记/风湿/风湿_脊柱关节病SpA对照]]；**本页只做纵向深挖 + 焊死两个反复丢分点**：① 一线永远 NSAIDs（感染已愈抗生素无用）② 环状龟头炎 = 无痛，别当性病溃疡。

---

## 一、一句话定位

> [!info] 定位
> ReA = 一种**血清阴性 (seronegative, RF 阴性) + HLA-B27 相关**的**脊柱关节病 (spondyloarthritis, SpA)**，由**特定泌尿生殖道 (GU) 或胃肠道 (GI) 感染后 1–4 周**触发的**免疫介导无菌性炎症**。关节腔内**无活菌**——这是"为什么抗生素不治关节炎"的根。

---

## 二、触发感染（记全两条通路）

```
ReA 触发感染 (1–4 周前)
├── 泌尿生殖道 GU ── Chlamydia trachomatis (沙眼衣原体) ★最常考
│
└── 胃肠道 GI ──┬── Salmonella   沙门菌
                ├── Shigella     志贺菌
                ├── Yersinia     耶尔森菌
                ├── Campylobacter 弯曲杆菌
                └── Clostridioides difficile 艰难梭菌
```

> [!tip] 记忆钩子
> GI 五菌 = "**沙-志-耶-弯-艰**"。GU 只记一个 = **衣原体 Chlamydia**（淋球菌 gonococcus 引起的是 DGI，不是 ReA，见 §六）。

---

## 三、临床三联征 + 关节 / 关节外表现

> [!tip] 经典三联征助记："can't see / can't pee / can't climb（the tree）"
> - **can't see** = 结膜炎 / 前葡萄膜炎 (conjunctivitis / anterior uveitis)
> - **can't pee** = 尿道炎 / 宫颈炎 (urethritis / cervicitis)
> - **can't climb** = 关节炎 (arthritis)
> 三联全齐是少数（且经典三联本身少见）；**任一 + 前驱感染史**就要高度怀疑。

### 3.1 肌肉骨骼

| 表现 | 特点 |
|---|---|
| **不对称寡关节炎 (asymmetric oligoarthritis)** | 好发**下肢大关节**（膝 / 踝），非对称、少数几个 |
| **附着点炎 (enthesitis)** | 肌腱 / 韧带止点炎，**跟腱 (Achilles) + 足底筋膜 + 胫骨结节**最典型 |
| **指 / 趾炎 (dactylitis, 香肠指)** | 整根手指 / 脚趾弥漫肿胀（SpA 家族通用） |

### 3.2 关节外（高频 buzzword）

| 系统 | 表现 |
|---|---|
| **眼** | 结膜炎 (conjunctivitis)、前葡萄膜炎 (anterior uveitis) |
| **泌尿生殖** | 尿道炎 (urethritis)、宫颈炎 (cervicitis)、前列腺炎 (prostatitis) |
| **皮肤 / 黏膜** | **环状龟头炎 (circinate balanitis)**、脓溢性皮肤角化病 (keratoderma blennorrhagicum, 掌跖)、**口腔溃疡 (oral ulcers, 无痛)** |

## 四、环状龟头炎 circinate balanitis — 反复错点 ②

> [!danger] 一句话焊死
> **环状龟头炎 = 龟头上无痛、环状（地图样）浅表糜烂，边缘略隆起发白 = ReA 的关节外表现，不是独立性病溃疡。** 关键鉴别词是 **"无痛 (painless / nontender)"**。

### 生殖器溃疡「痛 vs 无痛」快速鉴别表

| 溃疡 | 痛? | 关键特征 | 病原 / 归属 |
|---|---|---|---|
| **环状龟头炎 (circinate balanitis)** | **无痛** | 环状 / 地图样浅表糜烂、边缘白 + 关节炎 / 附着点炎 | ReA（免疫，非活菌）|
| **硬下疳 (chancre, 一期梅毒)** | **无痛** | 单个、硬、边界清、基底洁净 | 梅毒螺旋体 Treponema pallidum |
| **软下疳 (chancroid)** | **剧痛** | 深凿样溃疡脓性 + 触痛腹股沟淋巴结 | 杜克雷嗜血杆菌 Haemophilus ducreyi |
| **生殖器疱疹 (genital herpes)** | **痛** | 红斑基底上成簇水疱 → 溃疡 | 单纯疱疹病毒 HSV |
| **腹股沟肉芽肿 (granuloma inguinale)** | 无痛 | 牛肉红易出血肉芽 | 克雷伯肉芽肿杆菌 Klebsiella granulomatis |

> [!warning] Q115763 就栽在这
> 溃疡"无痛 + 环状 + 配关节炎 / 跟腱附着点炎 + 前驱衣原体" = 环状龟头炎 → ReA；被"生殖器溃疡"钩去选**软下疳（应剧痛）**是典型错法。**先问溃疡痛不痛**：痛 → 软下疳 / HSV；无痛 → 梅毒硬下疳 / 环状龟头炎 → 再看有没有关节炎定 ReA。

## 五、诊断 + 治疗

### 5.1 诊断

> [!info] 诊断要点
> - **主要靠临床**：前驱 GU / GI 感染史 + 不对称寡关节炎 + 关节外表现。
> - **HLA-B27** 常阳（增加易感 + 提示更重 / 中轴受累），非确诊必需。
> - **关节滑液 = 炎症性但无菌**：WBC 约 **2,000–4,000/mm³**（炎症级）、**革兰染色 + 培养阴性**（无活菌）——这条既支持 ReA 又排除化脓 / 淋球菌性关节炎。
> - 该查衣原体 (NAAT)、HIV、其他 STI（合并感染）。

### 5.2 治疗 — 反复错点 ①（Q4577 连错 3 次）

> [!danger] 焊死：一线永远 NSAIDs，感染已愈抗生素无用
> - **一线 = 非甾体抗炎药 (NSAIDs, nonsteroidal anti-inflammatory drugs，如布洛芬 ibuprofen / 萘普生 naproxen)**。
> - **抗生素只在"现症"感染时用**（如 stem 说还有活动性衣原体尿道炎，则治那个感染）；**泌尿 / 肠道症状已缓解 = 感染已清除 → 抗生素既不治关节、也不改变关节炎病程 = 错误答案**。
> - ReA 多**自限（约 6 个月内缓解）**。
> - 顽固 / NSAID 无效：局部糖皮质激素注射、柳氮磺吡啶 (sulfasalazine) 或甲氨蝶呤 (methotrexate, MTX)。
> - **环状龟头炎**若需处理 = **低效外用糖皮质激素**（如 1% 氢化可的松 hydrocortisone）。

```
ReA 治疗决策（把"感染是否现症"当开关）
│
├── 关节炎（感染已愈）── 一线 NSAIDs ──┬─ 有效 → 观察（多自限 ~6 月）
│                                      └─ 无效/顽固 → 局部激素注射 / 柳氮磺吡啶 / MTX
│
├── 仍有现症感染（活动性衣原体等）── 抗生素治【那个感染】（不是治关节）
│
└── 环状龟头炎 ── 低效外用糖皮质激素
```

> [!warning] 为什么"感染后病 = 用抗生素"是坑
> ReA 的关节损伤是**感染触发后的免疫反应**，关节腔里**没有活菌**可杀。这与**化脓性 / 淋球菌性关节炎（关节内有活菌，必须抗生素 + 引流）**本质不同——别把"感染后"与"感染中"混为一谈。

## 六、易混边界（高频陷阱）

> [!danger] ReA vs 播散性淋球菌感染 DGI —— 都是"性活跃 + 关节炎"，最常被拿来对撞

| 维度 | 反应性关节炎 ReA | 播散性淋球菌感染 DGI (disseminated gonococcal infection) |
|---|---|---|
| **病原 / 机制** | 衣原体 / 肠道菌**感染后免疫反应**，关节内无菌 | **淋球菌 (Neisseria gonorrhoeae) 血行播散**，关节 / 皮损处有菌 |
| **时序** | 感染后 **1–4 周** | 现症淋球菌感染期 |
| **关节** | 不对称**寡关节**（下肢）+ 附着点炎 | 化脓性单关节炎 **或** 腱鞘炎 + 游走性多关节痛 |
| **皮肤** | 环状龟头炎 / 脓溢性角化病（**无阴茎脓疱**）| **肢端脓疱 / 水疱脓疱疹**（无阴茎溃疡）|
| **滑液** | WBC 2,000–4,000，革兰 / 培养**阴** | 可培养出淋球菌、革兰可阳 |
| **治疗** | **NSAIDs** | **头孢曲松 (ceftriaxone) 静脉** |

> [!warning] 其他边界
> - **vs 痛风 / 假性痛风**：ReA 滑液无尿酸 / 焦磷酸钙结晶；别嘌醇 (allopurinol) / 秋水仙碱 (colchicine) 不用于 ReA。
> - **vs 银屑病关节炎 PsA / 强直性脊柱炎 AS / 肠病性关节炎**：同属 SpA，靠"前驱感染史 + 不对称下肢寡关节 + 关节外三联"把 ReA 拎出来——横向对照见 [[完整笔记/专题笔记/风湿/风湿_脊柱关节病SpA对照]]。
> - **vs 急性风湿热 ARF**：ARF 是**链球菌咽炎后**的**游走性**多关节炎 + 边缘性红斑 + 心炎（Jones）；ReA 是**下肢寡关节非游走** + 前驱 GU / GI。

---

## 七、Memory Hooks

> [!tip] 三句锁定
> 1. **触发**：衣原体 / 肠道五菌（沙-志-耶-弯-艰）后 1–4 周。
> 2. **治疗**：一线永远 **NSAIDs**；感染已愈 → 抗生素无用（现症感染才治那个感染）。
> 3. **环状龟头炎**：龟头**无痛**环状糜烂 = ReA 关节外表现，不是软下疳（软下疳剧痛）。

## 八、自测（答完再判分）

> [!question]- Q1. 28 岁男，3 周前尿道炎已愈，现右膝 + 左踝不对称肿痛 + 双眼红 + 左跟腱止点痛，滑液 WBC 3,500 革兰 / 培养阴。诊断 + 一线治疗？
> **反应性关节炎 (reactive arthritis, ReA)**；一线 **NSAIDs**。感染已愈 → 抗生素不治关节炎。

> [!question]- Q2. 上题患者龟头见多个无痛、环状排列的浅表糜烂。这是什么？要不要换抗生素？
> **环状龟头炎 (circinate balanitis)** = ReA 关节外表现（无痛）。**不换抗生素**；需处理时用低效外用糖皮质激素。别误当软下疳（软下疳剧痛 + 触痛淋巴结）。

> [!question]- Q3. 24 岁女，性活跃，发热 + 腕/踝游走性痛 + 腱鞘炎 + 肢端脓疱疹，无阴茎/生殖器溃疡。诊断与治疗？跟 ReA 怎么分？
> **播散性淋球菌感染 (DGI)** → **头孢曲松 (ceftriaxone) 静脉** + 黏膜 NAAT。与 ReA 分：DGI 是现症淋球菌血播散（肢端脓疱、可培养出菌），ReA 是感染后免疫（下肢寡关节 + 环状龟头炎、滑液无菌、一线 NSAIDs）。

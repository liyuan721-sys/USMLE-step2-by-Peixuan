# 脊柱关节病 SpA 四病对照

> [!info] 一句话定位
> SpA (spondyloarthropathy / spondyloarthritis, 脊柱关节病) = 一组**血清阴性 (seronegative, RF 阴性)**、**HLA-B27 (human leukocyte antigen B27, 人类白细胞抗原 B27) 相关**的炎症性关节病。本卡做**横向对照 + 鉴别决策树**；各病纵向细节见 [[完整笔记/专题笔记/风湿/风湿_HighYield速查总览]]（脊柱关节病段）。

---

## 一、PEAR 家族 + 五大共性

> [!tip] PEAR 助记
> **P**soriatic arthritis (PsA, 银屑病关节炎) / **E** = IBD (inflammatory bowel disease, 炎症性肠病) 相关关节炎 / **A**nkylosing spondylitis (AS, 强直性脊柱炎) / **R**eactive arthritis (ReA, 反应性关节炎)。

五大共性指纹（**所有 SpA 共享，单凭这些无法互相鉴别**）：

```
SpA 共性五件套
├── RF (rheumatoid factor, 类风湿因子) 阴性 ── "血清阴性"
├── HLA-B27 阳性倾向 ──────────────────── AS 最高 (~90%)，其余较低
├── Enthesitis (附着点炎) ───────────────── 跟腱/足底筋膜，SpA 特征性
├── Dactylitis (腊肠指/趾) ──────────────── 整指肿胀
└── 关节外：Uveitis (前葡萄膜炎) + 黏膜/皮肤受累
```

---

## 二、四病横向对照（核心表）

> [!warning] 鉴别主线：看「前驱史 + 关节分布 + 关节外特异征」三件套
> 缩写：GU = genitourinary (泌尿生殖道)、GI = gastrointestinal (胃肠道)、DIP = distal interphalangeal (远端指间关节)、NSAIDs = nonsteroidal anti-inflammatory drugs (非甾体抗炎药)、MTX = methotrexate (甲氨蝶呤)、AR = aortic regurgitation (主动脉瓣反流)。

| 维度 | AS 强直性脊柱炎 | PsA 银屑病关节炎 | ReA 反应性关节炎 | IBD 肠病性关节炎 |
|---|---|---|---|---|
| **好发人群** | 年轻男 <40 | 银屑病史患者 | 感染后年轻成人 | 克罗恩/溃结患者 |
| **前驱史** | 无（隐匿）| 银屑病皮损（可后到）| **GU/GI 感染后 1–4 周** | 肠道活动度相关 |
| **关节模式** | **中轴为主**（骶髂/脊柱）| **不对称寡关节 + DIP** | **不对称寡关节**（下肢大关节）| 外周（与肠病平行）+ 中轴 |
| **中轴受累** | **核心**（炎症性背痛）| 部分 | 可有骶髂炎 | 可有（独立于肠病活动）|
| **皮肤/黏膜** | — | **银屑病鳞屑斑 + 甲凹点 (nail pitting)** | **环状龟头炎 + 溢脓性皮肤角化病** | 结节红斑 / 坏疽性脓皮病 |
| **眼** | 前葡萄膜炎 | 葡萄膜炎 | **结膜炎 / 葡萄膜炎** | 葡萄膜炎 / 巩膜炎 |
| **特异 buzzword** | Schober↓、竹节脊柱 bamboo spine、**AR 主动脉瓣反流** | **"pencil-in-cup 笔入帽"** X 线、DIP | **"不能看/不能尿/不能爬"三联** | 关节炎随肠病波动（外周型）|
| **首选检查** | **骶髂关节 X 线**（早期阴性 → **MRI 最敏感**）| 临床 + 皮肤检查 | 临床（前驱史）| 肠镜 + 临床 |
| **治疗一线** | PT + NSAIDs → TNF-α 抑制剂 / 抗 IL-17 | 局部/系统（MTX、生物制剂）| **NSAIDs**（抗生素不治关节炎）| **治疗基础 IBD** |

## 三、鉴别决策树

```
炎症性关节炎 + RF 阴性 + HLA-B27 / 附着点炎 / 腊肠指 → 怀疑 SpA
│
├── 中轴为主（炎症性背痛、晨僵>30min、运动改善、夜间痛）
│     └── 骶髂关节炎 + 无前驱 → 【AS 强直性脊柱炎】
│           （X线骶髂 → 阴性查 MRI；伴 AR 杂音 / 葡萄膜炎）
│
├── 银屑病皮损 / 甲凹点 / DIP 受累 → 【PsA 银屑病关节炎】
│     └── 关节炎在前、无皮损 → next step = 全身皮肤检查找隐匿斑
│
├── 1–4 周前 GU(衣原体) 或 GI(沙门/志贺/耶尔森/弯曲) 感染
│     └── 不对称下肢寡关节 + 结膜炎 + 尿道炎 → 【ReA 反应性关节炎】
│           （一线 NSAIDs；感染已愈 → 抗生素无用）
│
└── 克罗恩 / 溃疡性结肠炎病史 → 【IBD 肠病性关节炎】
      （外周型随肠病活动波动；中轴型独立）
```

## 四、与易混病的边界（高频陷阱）

> [!danger] SpA vs RA — DIP 是分水岭

| 鉴别 | SpA (尤其 PsA) | RA (rheumatoid arthritis, 类风湿关节炎) |
|---|---|---|
| **DIP 远端指间关节** | **常受累**（PsA 标志）| **不受累 (RA spares DIP)** |
| **对称性** | 不对称、寡关节 | **对称、多关节** |
| **RF / 抗 CCP** | 阴性 | 多阳性 |
| **指甲** | nail pitting 甲凹点 | 无特异改变 |

完整 "DIP 受累俱乐部"（OA 骨关节炎 / PsA / 痛风 gout vs RA 豁免）汇总见 [[完整笔记/专题笔记/风湿/风湿_HighYield速查总览]] §一 · DIP 受累俱乐部。

> [!warning] 其他边界
> - **vs 痛风/假性痛风**：ReA 关节液**炎症性但无菌**（WBC ~2,000–4,000，Gram/培养阴性），无尿酸结晶；别嘌醇/秋水仙碱不用于 SpA。
> - **vs 淋球菌性关节炎 (gonococcal/DGI, disseminated gonococcal infection, 播散性淋球菌感染)**：DGI 也可有腱鞘炎 + 皮疹，但**关节液可培养出淋球菌、对抗生素 (ceftriaxone 头孢曲松) 有效**；ReA 是感染后免疫反应，关节内无活菌。
> - **vs 银屑病本身快速进展**：见 [[完整笔记/专题笔记/皮肤/皮肤_银屑病]]（快速进展查 HIV/链球菌/药物）。

## 五、体征特异性：dactylitis 与 pencil-in-cup 到底特不特异

> [!danger] 一句话结论
> **两个都不是银屑病特异（not pathognomonic）。** Dactylitis 是整个 **SpA 家族通用**、且还会被**非风湿病（镰状细胞）**模仿；pencil-in-cup 指向性强很多（PsA 招牌）但**仍非 100% 特异**。看到它们要靠 stem 其他线索定位，不能反射"= 银屑病"。

### 5.1 Dactylitis（指/趾炎，"香肠指" sausage digit）

整根手指/脚趾**弥漫肿胀**（关节 + 腱鞘 + 软组织一起肿，非单关节）。

> [!warning] 鉴别谱（按是否炎性分两类）
> 缩写：SpA = spondyloarthropathy（脊柱关节病）、SCD = sickle cell disease（镰状细胞病）、TB = tuberculosis（结核）。

| 病 | dactylitis | 关键定位线索（stem 给什么才选它）|
|---|---|---|
| **PsA** 银屑病关节炎 | ✅ 经典 | 银屑病皮损 / nail pitting 甲凹点 / 油滴征 oil drop sign / DIP 受累 |
| **ReA** 反应性关节炎 | ✅ 经典 | 1–4 周前 GU/GI 感染 + 结膜炎 + 尿道炎（"can't see/pee/climb"）|
| **AS** 强直性脊柱炎 / **IBD** 肠病性 | 偶见 | 中轴炎症性背痛 / IBD 病史 |
| **SCD** 镰状细胞病（**非炎性！**）| ✅ 但机制不同 | **1–2 岁黑人幼儿、双手双脚对称肿痛、贫血**；是血管闭塞性骨梗死（hand-foot syndrome 手足综合征），不是风湿 |
| TB / Sarcoid 结节病 | 罕见 | — |

> [!danger] 高频陷阱：dactylitis ≠ 自动想风湿
> 题干给"香肠指 + 幼儿 + 贫血 + 黑人"→ 反射 **SCD 的 dactylitis**（往往是镰状细胞婴儿**最早**的临床表现之一），不要往 SpA 走。SpA 的 dactylitis 一定配**成人 + 炎性钩子**（皮损/感染史/背痛）。若该幼儿再加发热 ≥5 天，也**别误诊川崎 Kawasaki**（CRASH 凑不够 4 条 + 发热镰状幼儿先当败血症/骨髓炎）→ 见 [[完整笔记/专题笔记/儿科/儿科_川崎病Kawasaki]] §二。

### 5.2 Pencil-in-cup deformity（笔帽样畸形）

影像学：近端指骨末端被侵蚀变尖（"铅笔"）+ 远端指骨基底膨大凹陷（"杯"），是 **arthritis mutilans（残毁性关节炎）** 的标志。

- **高度提示 PsA** —— CK 上看到这个 buzzword 基本就选 PsA。
- 但**非绝对特异**：重度 RA、其他重度侵蚀性关节病偶尔也能造出类似改变。
- 与 RA 的影像分水岭：PsA = **DIP 为主 + 同一骨既有 erosion 侵蚀又有 bone proliferation 新骨形成**；RA = **MCP/PIP 为主、纯侵蚀无新骨、对称、RF/抗 CCP 阳性**（详见 §四 DIP 表）。MCP = metacarpophalangeal（掌指关节）、PIP = proximal interphalangeal（近端指间关节）。

> [!tip] 记忆锁定
> **Dactylitis = "SpA 家族通用 + 镰状细胞捣乱"，要靠 stem 其他线索定位；Pencil-in-cup = PsA 招牌影像（指向性强但不是 100% 特异）。** 两个都不是"看到 = 银屑病"。

> [!question]- 自测：14 个月黑人男婴，双手双足对称性肿胀疼痛 3 天，Hb 8.2，外周血涂片见镰状红细胞。手 X 线见软组织肿胀。这个 "dactylitis" 的机制是？最可能诊断？
> **机制 = 血管闭塞性骨梗死（vaso-occlusive，非炎性）；诊断 = 镰状细胞病的 dactylitis / hand-foot syndrome 手足综合征**。是 SCD 婴儿最早表现之一，**不是** SpA。

---

> [!question]- 自测：28 岁男，3 周前尿道炎，现右膝 + 左踝不对称肿痛 + 双眼红，关节液 WBC 3,500 培养阴性。诊断 + 一线治疗？
> **Reactive arthritis 反应性关节炎**；一线 **NSAIDs**。抗生素**不治**已愈感染后的关节炎。

> [!question]- 自测：35 岁男慢性炎症性背痛 + 晨僵 1 小时运动后缓解，闻及舒张期杂音。最敏感的早期影像？
> **骶髂关节 MRI**（X 线早期可阴性）；杂音提示 **AS 并发主动脉瓣反流 (AR)**。

# 关节炎鉴别诊断决策树（Pattern 分诊枢纽）

> [!info] 这张卡是什么
> 关节痛 (arthralgia) / 关节炎 (arthritis) 题的**总入口**：考试反射顺序是 **先定性 → 再按分布 pattern 分诊 → 再叠加时间轴(急/慢)与对称性**，最后才落到具体病。本卡只做**分诊主干 + 决策树**，各病纵向细节链到下游专题：
> - 脊柱关节病 (SpA) 四病横向对照 → [[完整笔记/专题笔记/风湿/风湿_脊柱关节病SpA对照]]
> - 风湿全谱纵向速查（OA/RA/晶体/血管炎/肌病细节）→ [[完整笔记/专题笔记/风湿/风湿_HighYield速查总览]]
> - 按主诉分的单/多关节痛大杂烩 → [[完整笔记/专题笔记/USMLE/USMLE_鉴别诊断速查]]（§ Monoarticular / Polyarticular）
> - 学科主笔记 → [[完整笔记/Peixuan分科笔记/骨科]]（风湿段）

> [!note]- 缩写对照（全称 中英）
> - **OA** = osteoarthritis 骨关节炎
> - **RA** = rheumatoid arthritis 类风湿关节炎
> - **SLE** = systemic lupus erythematosus 系统性红斑狼疮
> - **SpA** = spondyloarthritis / spondyloarthropathy 脊柱关节病
> - **AS** = ankylosing spondylitis 强直性脊柱炎
> - **PsA** = psoriatic arthritis 银屑病关节炎
> - **ReA** = reactive arthritis 反应性关节炎
> - **IBD** = inflammatory bowel disease 炎症性肠病
> - **CPPD** = calcium pyrophosphate deposition 焦磷酸钙沉积（pseudogout 假性痛风）
> - **DGI** = disseminated gonococcal infection 播散性淋球菌感染
> - **MCP / PIP / DIP / MTP** = metacarpophalangeal / proximal / distal interphalangeal / metatarsophalangeal 掌指 / 近端指间 / 远端指间 / 跖趾关节
> - **RF / anti-CCP** = rheumatoid factor / anti-cyclic citrullinated peptide 类风湿因子 / 抗环瓜氨酸肽抗体
> - **ANA / dsDNA** = antinuclear antibody / anti-double-stranded DNA 抗核抗体 / 抗双链 DNA 抗体
> - **ESR / CRP** = erythrocyte sedimentation rate / C-reactive protein 血沉 / C 反应蛋白
> - **HBV / HCV** = hepatitis B / C virus 乙 / 丙型肝炎病毒
> - **IE** = infective endocarditis 感染性心内膜炎
> - **PAS** = periodic acid–Schiff 过碘酸-雪夫（染色）
> - **WBC / PMN** = white blood cell / polymorphonuclear 白细胞 / 中性粒细胞
> - **NSAIDs** = nonsteroidal anti-inflammatory drugs 非甾体抗炎药
> - **DMARD** = disease-modifying antirheumatic drug 改善病情抗风湿药
> - **HLA-B27** = human leukocyte antigen B27 人类白细胞抗原 B27

## 一、三步分诊主干（背下顺序）

```
关节痛 / 关节炎
│
├─ 第 1 步：定性 —— 是不是炎症？是不是化脓？  ← 决定"急不急"
│     用 关节液 (synovial fluid) WBC + 全身炎症征
│
├─ 第 2 步：数关节 + 看分布 pattern        ← 决定"往哪个病族走"
│     单关节 mono / 寡关节 oligo / 多关节 poly / 游走 migratory
│
└─ 第 3 步：叠加时间轴 + 对称性 + 关节外征   ← 落到具体病
      急性<6 周 vs 慢性≥6 周；对称 vs 不对称；皮/眼/黏膜/肾/前驱史
```

> [!danger] 任何"急性单关节红热肿 + 发热"先做一件事
> **关节穿刺 (arthrocentesis)** 排 **化脓性关节炎 (septic arthritis)**——24–48h 即毁关节，是关节炎里唯一的真急诊。晶体 (crystals) 与化脓**可并存**，查到晶体不能排化脓。

## 二、第 1 步：定性（关节液 WBC 分级）★最高频★

> [!warning] 表格记法：看 WBC 量级直接归类，化脓的硬线是 **>50,000**

| 类别 | WBC/mm³ | PMN | 代表病 | next step |
|---|---|---|---|---|
| 正常 | <200 | <25% | — | — |
| 非炎症 non-inflammatory | 200–2,000 | <25% | OA、创伤、出血性（外伤/凝血病） | 对症 + 影像 |
| 炎症 inflammatory | 2,000–50,000 | 50% | 晶体（痛风/CPPD）、RA、SLE、SpA、病毒、ReA | 晶体镜检 + 自身抗体 |
| 化脓 septic | **>50,000** | **>75–90%** | 细菌性化脓性关节炎 | **穿刺培养 + 经验 IV 抗生素 + 引流** |
| 出血 hemorrhagic | RBC++ | — | 创伤、血友病、抗凝过量 | 影像 + 纠凝血 |

> [!tip] 量级灰区（别被中间值骗）
> - **ReA / 病毒性**：滑液常仅 2,000–10,000、Gram 染色与培养阴性。
> - **急性 CPPD / 痛风**：可冲到 15,000–50,000，逼近化脓区 → **必须靠晶体镜检 + Gram/培养二次区分**，不能只看数字。
> - **晚期 Lyme 关节炎**：滑液 WBC 可达炎症高段甚至接近化脓段，但**临床上膝大量积液却相对微痛** → 别误当化脓。源：[[mistakes/uworld-mistakes_2026-06#^Q3836]]

晶体形状谱（6 种）细节见 [[完整笔记/专题笔记/风湿/风湿_HighYield速查总览]] §四；**急性单/少关节滑液（WBC + 培养 + 晶体）深入鉴别 + 婴儿化脓机制**见单一真相源 [[完整笔记/专题笔记/风湿/风湿_急性单关节炎滑液鉴别]]。

## 三、第 2 步：按分布 pattern 分诊（核心决策树）

```
数受累关节 + 分布
│
├─ 单关节 Mono（1 个）
│   急性 → 化脓性关节炎 / 痛风 / 假性痛风(CPPD) / 创伤·出血 / 急性发作的 OA
│   慢性 → OA / 结核·真菌性(慢性单关节) / 肿瘤 / 假体关节感染 PJI(有人工关节史)
│   ※ 发热 + 急性单关节 = 先穿刺排化脓
│
├─ 寡关节 Oligo（2–4 个，常不对称、下肢大关节）
│   → 脊柱关节病 SpA 家族（PEAR）：AS / PsA / ReA / IBD 相关
│   → 痛风（可多关节化）/ Lyme / 播散性淋球菌早期
│
├─ 多关节 Poly（≥5 个）  ← 见 §五 展开
│   对称 + 炎症 → RA / SLE / 病毒(parvovirus B19 等)
│   不对称        → PsA / 多关节痛风 / SpA
│   慢性 + 非炎症 → 多关节 OA（手 DIP/PIP、第一腕掌、膝髋）
│
└─ 游走 Migratory（此起彼伏、换关节）  ← 见 §六
    先抓 3 个感染源「淋·莱·风湿热」= Gonococcal 淋球菌(DGI) / Lyme 莱姆 / Rheumatic fever 风湿热（+ SLE 早期 / Whipple）
```

> [!tip] 一句话钥匙
> **单关节先想"红肿热急诊"（化脓/晶体）；寡关节先想 SpA；多关节对称想 RA/SLE/病毒；游走想感染源(淋球菌/Lyme/风湿热)。**

## 四、第 3 步：时间轴（急性 <6 周 vs 慢性 ≥6 周）

| | 急性 (<6 周) | 慢性 (≥6 周) |
|---|---|---|
| 单关节 | 化脓、痛风、CPPD、创伤/出血 | OA、慢性感染（TB/真菌）、肿瘤 |
| 寡/多关节 | 病毒（parvovirus/HBV/HCV）、ReA、急性风湿热、淋球菌、早期 RA/SLE | RA、SLE、PsA、AS、慢性多关节 OA |

> [!warning] "6 周线"是分诊钩子
> stem 给**对称小关节痛 ≥6 周 + 晨僵 >1h** → 强烈指向 **RA**（病毒性关节炎多在 6 周内自限）。病程长短常是区分**病毒性一过性多关节炎 vs 早期 RA** 的关键。

## 五、多关节炎 (Polyarthritis) 展开 ★本卡重点支★

> [!info] 三分法：先分"对称 vs 不对称"，再分"炎症 vs 非炎症"
> 多关节炎 = ≥5 个关节。第一刀切**对称性**，第二刀切**炎症性**。

```
多关节炎 Polyarthritis
│
├─ 对称 + 炎症性（晨僵重、可负重、贫血、ESR/CRP↑）
│   ├─ RA：MCP/PIP/腕、spares DIP、对称、晨僵>1h、anti-CCP 最特异、边缘性骨侵蚀
│   ├─ SLE：非侵蚀性、女性、malar rash/浆膜炎/肾、ANA→dsDNA/Smith、补体↓
│   └─ 病毒性：parvovirus B19 细小病毒（接触幼儿/教师/妈妈）、HBV、HCV、rubella(风疹) —— 多自限<6 周
│
├─ 不对称（或叠加 DIP / 中轴）
│   ├─ PsA：DIP + 甲凹点 + 香肠指 + 银屑病皮损、"笔入帽"
│   ├─ 多关节痛风（晚期 tophaceous）
│   └─ 其余 SpA（中轴 + 外周）
│
└─ 慢性 + 非炎症性（机械痛、晨僵<30min、无全身征）
    └─ 多关节 OA：DIP(Heberden)/PIP(Bouchard)、第一腕掌、膝/髋
```

### 对称炎症多关节炎三巨头对比

| | RA | SLE | 病毒性 (parvovirus B19) |
|---|---|---|---|
| 关节 | MCP/PIP/腕，**spares DIP**，对称 | 小关节对称、**非侵蚀** | 手小关节对称、一过性 |
| 病程 | **慢性 ≥6 周、进行性** | 慢性、随活动度波动 | **急性、多 <6 周自限** |
| 全身 | 乏力/低热/体重↓ | malar rash 颊部红斑、浆膜炎、肾、血液三系↓ | 接触史（幼儿/学校）、可有网状皮疹 |
| 抗体 | RF（敏感）、**anti-CCP（最特异+预后差）** | **ANA（筛）→ dsDNA/Smith（特异）**、补体↓ | parvovirus B19 IgM；**RF/anti-CCP 常阴性** |
| X 线 | **边缘性骨侵蚀** | 无侵蚀 | 无 |
| 钩子 | 晨僵>1h、对称小关节 | 育龄女性 + 多系统 | 暴露幼儿 + 一过性 |

> [!danger] 对称多关节炎"不能漏诊"
> - **SLE 伴肾炎 / CNS / 心肺** —— 多系统拼图（脑+关节+血+肾）→ 查 ANA→dsDNA/Smith→补体。源：[[mistakes/uworld-mistakes_2026-05#^Q11967]]
> - **病毒性 ≠ RA**：别把 parvovirus 一过性对称多关节痛当早期 RA 上 DMARD —— 看病程 <6 周 + 暴露史。源：[[mistakes/uworld-mistakes_2026-06#^Q4579]]
>
> （注：**急性风湿热、感染性心内膜炎** 的关节痛是**游走性**，不在对称支 → 见 §六游走性表）

## 六、游走性 (Migratory) 关节炎 —— 感染源优先

> [!tip] "换着关节痛"= migratory，几乎都指向**可治的感染/感染后**病因，别往 RA/OA 想

| 病 | 定位钩子 | next step / 治疗 |
|---|---|---|
| **播散性淋球菌 DGI** | 年轻性活跃、**腱鞘炎 tenosynovitis + 脓疱疹 + 游走多关节痛**三联；滑液常 <50k | 血/滑液/黏膜培养 + 经验 **ceftriaxone 头孢曲松** |
| **Lyme（莱姆病）** | 早期游走性肌肉关节痛（红斑后）；晚期 → 单膝大量积液微痛 | 早期 doxycycline 多西环素（孕期换 amoxicillin） |
| **急性风湿热** | 儿童、链球菌咽炎后、**游走性大关节炎 + Jones**（心炎/舞蹈病/环形红斑） | 抗链球菌 + 抗炎 |
| **感染性心内膜炎 IE** | 发热 + 新杂音 + **可游走性关节痛/反应性关节炎 + 免疫复合物征**（Osler 结节/Roth 斑） | 血培养 ×3 + 超声心动 |
| **Whipple 病** | 中年男、**游走性非畸形关节炎**（常先于 GI 数年）+ 体重↓/腹泻 + PAS+ 巨噬 | 小肠活检 PAS 染色 → **诱导 IV ceftriaxone 头孢曲松 2 周 → 口服 TMP-SMX 复方磺胺甲噁唑 ×1 年** |
| **SLE / 血清病** | 早期可游走；血清病 = 用药后发热+荨麻疹+关节痛+蛋白尿 | 对因 |

源：[[mistakes/uworld-mistakes_2026-06#^Q15847]]（DGI 三联）、[[mistakes/uworld-mistakes_2026-06#^Q3311]]（Whipple 游走性关节炎）、[[mistakes/uworld-mistakes_2026-06#^Q3836]]（晚期 Lyme 单膝）

> [!warning] DGI vs 普通化脓性关节炎（高频分水岭）
> 两个都是细菌、都急，但分诊路径不同：
> - **DGI 播散性淋球菌**：年轻性活跃、**不对称寡/多关节 + 腱鞘炎 + 脓疱疹**、滑液 **WBC <50,000**、**滑液 Gram/培养常阴性**（确诊靠血/黏膜/咽/直肠培养或核酸）→ ceftriaxone 头孢曲松 + 同治衣原体。
> - **普通化脓性关节炎**：多为**单关节**、滑液 **WBC >50,000**、**滑液培养易阳**。
> - 钥匙：**"游走 + 腱鞘炎 + 脓疱 + 滑液不到 5 万 + 滑液培养阴" = 想淋球菌，别因培养阴排掉感染。**

## 七、红旗 / 必排（reflex 题）

> [!danger] 见到这些立刻反射
> - **发热 + 急性单关节** → 关节穿刺排化脓（不论有无晶体）
> - **儿童/婴儿拒动肢体 + 假性瘫痪 + 髋屈曲外展外旋** → 婴儿化脓性关节炎，急诊穿刺。源：[[mistakes/uworld-mistakes_2026-06#^Q15845]]
> - **拳击咬伤 (fight bite) MCP 破口** → 人咬伤致 MCP 化脓性关节炎，手术清创 + ampicillin-sulbactam。源：[[mistakes/uworld-mistakes_2026-06#^Q18459]]
> - **儿童化脓性关节炎万古无效** → 加 ceftriaxone 覆盖 Kingella 金氏菌（G⁻）。源：[[mistakes/uworld-mistakes_2026-06#^Q15848]]
> - **ReA 一线 = NSAIDs**，感染已愈抗生素无用（高频陷阱，别反射抗生素）。源：[[mistakes/uworld-mistakes_2026-06#^Q4577]]
> - **PsA 疑诊但无明显皮损** → next step = 全面皮肤检查（头皮/脐/腹股沟隐匿斑）。源：[[mistakes/uworld-mistakes_2026-06#^Q4581]]
> - **术后/急性病后单关节 + 菱形正性双折光晶体** → CPPD，查继发病因（甲旁亢/血色病/甲减）。源：[[mistakes/uworld-mistakes_2026-06#^Q3151]]
> - **发热 + 多关节痛 + 新发心杂音** → 先排**感染性心内膜炎 IE**（血培养 ×3 + 超声心动），别只当风湿病处理——IE 是伪装成关节病的系统性感染。
> - **儿童髋/膝痛 + 跛行/拒负重，分不清化脓 vs 一过性滑膜炎** → 用 **Kocher 标准** 4 条评分（见下）。

### Kocher 标准（一过性滑膜炎 vs 化脓性关节炎）

> [!info] 4 条各 1 分 — 评分越高越像化脓，需穿刺
> 用于儿童急性髋痛区分 **transient synovitis 一过性滑膜炎（良性自限）** vs **septic arthritis 化脓性关节炎（急诊）**。

| Kocher 4 条（各 1 分） | 阈值 |
|---|---|
| 发热 | **≥38.5 ℃** |
| 不能负重 non-weight-bearing | 是 |
| ESR 血沉 | **≥40 mm/h**（CRP C 反应蛋白 ≥2 也算） |
| 外周血 WBC 白细胞 | **≥12,000/mm³** |

> [!tip] 评分解读
> **≥3 分 → 化脓性关节炎概率高（约 ≥93%）→ 关节穿刺**；0–1 分多为一过性滑膜炎（NSAIDs + 休息 + 随访）。2 分为灰区，结合超声/复查。

### 化脓性关节炎经验抗生素（按人群-病原谱）

> [!warning] 穿刺送检后**别等培养**，按人群经验启动

| 人群 | 主要病原 | 经验方案 |
|---|---|---|
| 成人（一般） | S. aureus 金黄色葡萄球菌、Streptococcus 链球菌 | **vancomycin 万古霉素 + ceftriaxone 头孢曲松**（覆盖 G⁻） |
| 年轻性活跃 | N. gonorrhoeae 淋球菌（DGI） | **ceftriaxone** + 同治衣原体 |
| 新生儿 | S. aureus、GBS（B 族链球菌 group B Streptococcus）、G⁻ 杆菌 | vancomycin + 三代头孢 |
| 1–5 岁 | S. aureus、**Kingella 金氏菌** | ceftriaxone（万古无效时加覆盖 Kingella） |
| IVDU 静脉吸毒者 | Pseudomonas 铜绿假单胞菌、S. aureus | 抗假单胞 β-内酰胺 + vancomycin |
| 假体关节 PJI | 急性<3 月 S. aureus/链球菌；慢性>3 月 **凝固酶阴性葡萄球菌 (coagulase-negative staph) / Cutibacterium 痤疮丙酸杆菌** | 取材培养 → 针对性 + 多需取出假体。源：[[mistakes/uworld-mistakes_2026-06#^Q9111]] |

## 八、自测（答完再判分）

> [!question]- Q1. 24 岁女性 1 周内右膝、左踝、右腕先后肿痛此起彼伏，伴手背脓疱样皮疹、握拳时腕背肌腱处疼痛。滑液 WBC 18,000、培养待回。最可能诊断 + 经验治疗？
> **播散性淋球菌感染 (DGI)**：游走多关节痛 + 腱鞘炎 + 脓疱疹三联、滑液常 <50k。经验 **ceftriaxone 头孢曲松**，同治可能合并的衣原体。

> [!question]- Q2. 38 岁女性双手 MCP、PIP 对称肿痛 + 晨僵 2 小时持续 8 周，幼儿园教师，近期班上"拍脸病"流行。区分 RA 与病毒性关节炎最关键的一点是什么？
> **病程时间轴**：parvovirus B19 病毒性关节炎多 **<6 周自限**；本例 ≥6 周 + 晨僵>1h 更像 **早期 RA** → 查 anti-CCP/RF。暴露史提示曾感染但不等于现在是病毒性关节炎。

> [!question]- Q3. 70 岁男性右膝突发红肿热痛 + 低热，3 天前刚做完髋置换术。滑液 WBC 32,000、镜下见菱形正性双折光晶体、Gram 染色阴性。下一步最重要的处理是什么？为什么晶体阳性还不能放心？
> 急性 **CPPD（假性痛风）**，术后/急性病常诱发。但 WBC 逼近化脓区 + 发热 → **仍需 Gram 染色 + 培养排化脓性关节炎**（晶体与化脓可并存）。同时查 CPPD 继发病因（甲旁亢/血色病/甲减）。

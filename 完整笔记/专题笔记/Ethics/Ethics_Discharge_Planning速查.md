## §一 6 档 Disposition 对照表 ⭐⭐⭐

| 档 | 设置 | 触发条件 | 服务 | Benefits / 注意 |
|---|---|---|---|---|
| 1 | **Outpatient follow-up** | Stable + 能出门 + 无 skilled needs | PCP / specialty 复诊 | 最 light — 适常规慢病管理 |
| 2 | **Home Health Care** ⭐ | **Medically stable + Homebound + Skilled needs**（三联）| Nursing / PT / OT / SW / wound / med management | 保 independence + 减 readmission + < SNF 贵 |
| 3 | **Assisted living** | Medically stable + ADL 部分需助 + 无 24/7 medical supervision | 协助洗澡 / 穿衣 / 吃药 + 社交 + 餐 | 介于 home 与 SNF |
| 4 | **Skilled Nursing Facility (SNF)** | Medically stable + **Severe CI / 不能 ADL / 持续 monitoring** | 24/7 nursing + rehab + 长期 medical care | 失 independence + 高 cost + 适 post-stroke rehab / 严重 dementia |
| 5 | **Hospice** | Terminal illness（预期 ≤ 6 mo）+ 放弃 curative | Symptom-focused + palliative + 家 / inpatient hospice / nursing home | 不强制 DNR；可保留 palliative tx |
| 6 | **Hospital admission** | **不 medically stable** / acute illness / 需 IV / 监护 / 手术 | 急性 medical / surgical | medically stable 时 admit = **net harm** ⚠️ |

## §三 Home Health Care 三联触发词 ⭐⭐⭐（CK 高频）

| 触发条件 | 含义 | Stem 暗号词 |
|---|---|---|
| **① Medically stable** | 不需 acute hospital level care | "medically stable" / "after recent discharge" / "vital signs normal" |
| **② Homebound + 需 intermittent monitoring** | 出门困难 / 限制 + 需定期专业上门 | "limited mobility" / "post-fall" / "递增 functional decline" / "递减社交" |
| **③ Skilled needs** | Nursing / PT / OT / SW / wound / med management 之一 | "多 medication" / "insulin" / "新 antidepressant" / "wound healing" / "post-stroke PT" |

**三联都满足 → home health care 首选**

### 3.1 Home Health Care 服务范围

| 服务 | 内容 |
|---|---|
| **Skilled nursing** | Med management / wound care / IV / 血糖监测 / 教育 |
| **PT (Physical Therapy)** | Fall prevention + gait training + 力量训练 |
| **OT (Occupational Therapy)** | ADL retraining + 辅助器具评估 + home safety assessment |
| **SW (Social Worker)** | Resources / 财务 / community connections / 心理支持 |
| **Speech therapy** | Post-stroke / dysphagia |

### 3.2 Home Health Care Benefits ⭐

| Benefit | 机制 |
|---|---|
| **Mental health + autonomy + independence 保持** | Aging in place — 不剥夺尊严 |
| **减 hospital readmission + LOS** | 早识别 symptoms + 防 functional decline |
| **减 healthcare spending** | < SNF (Medicare home health 比 SNF 便宜 50%+) |
| **减 nosocomial infection** | 避 hospital / nursing home environment |

## §四 6 档触发条件 + 反 pattern

### 4.1 ① Outpatient

**触发**：stable + 能出门 + 仅需常规复诊

**反 pattern**：
- 患者 medically stable + 多 chronic conditions + complex medication → **outpatient 不够 — 需 home health care**

### 4.2 ② Home Health Care ⭐

**触发**：见 §三 三联

**反 pattern**：
- "Pill organizer" / "family administer" — 不解决 inter-prescriber coordination + 不替代 skilled care
- "Office follow-up 4 weeks" — 高 risk 患者间隔太长

### 4.3 ③ Assisted Living

**触发**：medically stable + ADL 部分需助 + 无 medical supervision needs

**反 pattern**：
- Severe CI 不达 assisted living — 应 SNF
- Independent ADL 不需 assisted living — home + home health 即可

### 4.4 ④ SNF

**触发**：
- **Severe CI** + 不能独立 ADL
- **Post-stroke rehab**（需 daily PT + 24/7 nursing）
- **复杂 wound** 需 daily nursing
- **末期不能 home + 仍要 curative-leaning care**

**反 pattern** ⭐：
- ❌ "Mild CI + minor injury" 升级 SNF — 夺走 independence + 高 cost
- ❌ "Lives alone + 担心 fall" 升级 SNF — home health care 即可
- ❌ "Daughter 工作忙不能照顾" 升级 SNF — home health care 可 cover

### 4.5 ⑤ Hospice

**触发**：terminal illness 预期 ≤ 6 mo + 放弃 curative

**Hospice ≠ DNR**（不强制签 DNR）；**可保留 palliative tx**（symptom control / palliative radiotherapy）。

**反 pattern**：
- 同时 immunotherapy curative + hospice = ❌ 矛盾（hospice 必须放弃 curative）
- "All cancer treatment 停" 过严 — palliative tx 可继续
- Hospice + acute illness（MI / 感染）→ 可治不取消 hospice 资格

详见 [[完整笔记/专题笔记/Ethics/Ethics_EOL_Hospice与决策]]。

### 4.6 ⑥ Hospital Admission ⭐ 反 pattern 最常见

**触发**：**不 medically stable** / acute illness / 需 IV / 监护 / 手术

**反 pattern**：
- ❌ "Medically stable + admit for cognitive workup" = **net harm**
  - Cognitive assessment 可在家 / outpatient 完成
  - Admit medically stable 患者 = nosocomial infection + delirium + functional decline + iatrogenic + cost
- ❌ "Elderly + fall + 担忧" 反射 admit — medically stable 时应 home health care
- ❌ "多药 + 担心 ADE" 反射 admit — 应 home health med management + limit prescribers

## §五 老年 Care 升级红线 SOP ⭐

| 红线 | 应对 |
|---|---|
| **Medically stable** | 优先保 home + outpatient → 不 admit |
| **Mild CI + 能 ADL** | 优先保 independence → 不 SNF |
| **Acute illness / 不能 ADL / 持续 monitoring** | 才升级 SNF / hospital |
| **Terminal + 放弃 curative** | Hospice（不取代 acute care）|
| **Polypharmacy + ADE risk** | Limit prescribers（PCP 主管）+ home health med management（**不是** family administer / pill organizer）|

## §六 Memory Hook + 类比 ⭐

### 6.1 6 档强度类比

```
强度 0 ─ 完全独立 ─ ① Outpatient（自己开车去复诊）
       │
       ▼
强度 1 ─ 上门 skilled care ─ ② Home Health Care ⭐（专业护工每周来家）
       │
       ▼
强度 2 ─ 协助 ADL ─ ③ Assisted Living（搬进 ADL 协助型公寓）
       │
       ▼
强度 3 ─ 24/7 medical ─ ④ SNF（搬进养老康复院 24/7 监护）
       │
       ▼
强度 4 ─ Acute care ─ ⑥ Hospital（住院）
       │
       ▼
End-of-life ─ ⑤ Hospice（任何场所 — 家 / SNF / inpatient hospice — 关键是 goal-of-care 转向 symptom）
```

### 6.2 一句话锁 ⭐⭐⭐

- **"Medically stable + homebound + skilled needs → Home health care 首选"**（保 independence + 减 readmission + 省 cost）
- **"SNF 触发 = severe CI + 不能 ADL + 持续 monitoring；mild CI + minor injury ≠ SNF"**
- **"Medically stable + admit = net harm"**（nosocomial / delirium / functional decline）
- **"Hospice ≠ DNR；可保留 palliative tx"**

### 6.3 反 anchor 训练

不要反射"elderly + 担忧 → institution 升级"：
- 第一步永远是 **stable?** — stable 才有"保 home"的窗口
- mild CI ≠ SNF 触发；mild CI + medically stable + home health 是合理 spectrum

## §七 真题反射钩库

### 7.1 Q20431 / Q9-2026-05-27 — Home Health Care 双反复错卡

**Stem 模式**：elderly + medically stable + mild CI + 多 meds + fall + lives alone → 最佳干预？

**反 pattern 排除**：
- A admit → ❌ medically stable + admit = net harm
- B SNF → ❌ mild CI 不达 SNF 触发
- D family administer → ❌ daughter 工作忙 + 不解决 inter-prescriber
- E office 4 weeks → ❌ 高 risk 患者间隔太长

**正解**：**Home health care**（保 independence + skilled care + 减 readmission）

### 7.2 Q21141 — Hospice eligibility

**Stem 模式**：晚期 cancer + immunotherapy curative + palliative radiotherapy → 能 hospice 吗？

**核心**：Hospice 必须**停 curative**（immunotherapy）+ **可继续 palliative**（radiotherapy）+ **不强制 DNR**

### 7.3 Q20229 — Hospital Readmission Reduction

**Stem 模式**：CHF + 高 Medicare readmission penalty → 减 30-day readmission 策略？

**正解**：Telephone-based outreach by care coordinator within 2 weeks
- 同 patient-side intervention 哲学（vs provider-side communication）
- Discharge summary to PCP 不解决 patient 端 loss to follow-up

### 7.4 Q107185 — Geriatric Polypharmacy

**Stem 模式**：83yo + 多 specialists + 10+ meds → reduce ADE risk

**正解**：**Limit number of prescribers**（PCP 主管 + 与 specialists 协调）
- Pill organizer / family administer ❌ 不解决 inter-prescriber coordination

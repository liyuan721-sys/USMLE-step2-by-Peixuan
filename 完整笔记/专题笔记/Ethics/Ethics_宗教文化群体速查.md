# Ethics_宗教文化群体速查（USMLE Step 2 CK 高频 8 群体）

> [!info] 一句话定位
> CK Ethics 高频考"宗教 / 文化群体的医疗决策冲突"。**3 大跨群体铁律**：① 成年 + capacity + informed → **尊重 autonomy**（即使会死）② Minor + 父母拒救命标准治疗 → **Court order** ③ 不评判 + **探索替代方案**（绝大多数信仰允许 workaround）+ **多次随访**。**核心考点：哪些治疗被该群体绝对禁忌、哪些是 individual choice、哪些 workaround 可接受**。

> [!warning] CK 真正高频只有 3 个群体
> **Jehovah's Witness（最高频）/ Christian Science（高频，主考 minor）/ Catholic（中频，主考避孕 / IVF / end-of-life）**。Muslim / Orthodox Jewish 偶尔考（猪源 insulin / Ramadan / Sabbath）。Amish / Hindu / Sikh 罕考但有特征模式 → 看到要会识别**不要选"违反信仰"的答案**。

## 一、8 大群体速查表（**优先记前 3 个**）

| 群体（中 / 英） | 核心禁忌 | CK 高频 stem 模式 | 处理铁律 |
|---|---|---|---|
| **Jehovah's Witness 耶和华见证人** ⭐⭐⭐ | **所有血制品**（whole blood / RBC / platelets / plasma / WBC）；albumin / clotting factors / cell saver = individual choice | Adult 大出血拒输血 / **JW 父母拒给孩子输血** / pregnant JW 备产 | Adult 尊重 / **Minor → court order** / 替代：cell saver / EPO / IV iron / TXA / volume expander |
| **Christian Science 基督科学派** ⭐⭐ | **几乎所有医学治疗**（祈祷 = "spiritual healing"）— 拒药、拒手术、拒疫苗 | **DKA / 阑尾炎 / 白血病儿童**父母只祈祷不送医 | Adult 尊重 / **Minor → court order**（儿童保护服务报告 CPS）|
| **Catholic 天主教** ⭐⭐ | **人工避孕** / **堕胎** / **直接安乐死** / IVF（部分情况）；**不**反对常规医学 | 避孕咨询 / IVF / end-of-life / Catholic 医院转诊 / contraception in Catholic-affiliated hospital | 患者自决；医生若 Catholic 个人良心拒绝 → **转诊（refer to another provider）不能 abandon** |
| **Muslim 穆斯林** ⭐ | **猪源药品**（pork-derived insulin / heparin / gelatin capsules）/ **Ramadan 斋月**禁食 / female modesty | 1 型 DM 穆斯林斋月用 insulin / 猪源胰岛素 / 女患者拒男医生 | 提供 non-porcine 替代（合成 insulin / synthetic heparin）/ Ramadan 调整 dosing / 性别匹配 chaperone |
| **Orthodox Jewish 正统犹太** ⭐ | **Sabbath**（周五日落 - 周六日落不能开电 / 开车 / 写字）/ kosher / **brain death 定义争议** | Sabbath 期间不能离院；终末期 brain death 家属拒撤呼吸机 | 紧急医疗 = **pikuach nefesh**（救命大于 Sabbath，宗教允许）；brain death 争议 → ethics committee / rabbinic 顾问 |
| **Amish 阿米什人** | 拒**常规疫苗** / 部分现代医学（电 / 保险）；**不**拒紧急医学 | 儿童拒疫苗 / 难产送医 / 创伤救治 | Adult 尊重 vaccine refusal；**紧急救命 = 接受**（与 Christian Science 不同）|
| **Hindu 印度教** | 素食（含 gelatin / 部分疫苗辅料）/ 火化 / **不**拒输血 | 素食 + 给药（含 gelatin）/ end-of-life 接受度高 | 提供非动物源剂型 / 火化文化尊重 |
| **Sikh 锡克教** | **5K 戒律**：不剪头发 / 不取 kara（铁手镯）/ kirpan（短剑）随身 | 手术时 kara 不能取 / 头发 / 胡须修剪 | 手术中 kara 用胶带固定（不取）/ 头发尽量保留 / 沟通后 patient 自决 |

> [!tip] 一句话记 3 大铁律
> **"成人尊重 + 孩子救命 + 替代先上"** —— 不管什么宗教都套这条 SOP。

## 二、Jehovah's Witness（JW，耶和华见证人）⭐⭐⭐ 必背

> [!info] 教派背景（CK 不直接考，但帮理解 stem）
> 基督教分支，全球约 870 万信徒（数据 CK 不考）。教义基于《圣经》**Acts 15:29 "abstain from blood"** 的字面解释 — 输血 = "consume blood" → 违反神命令 → 死后失去"复活权"。**对 JW 而言，宗教救赎 > 生命延续**。

### 2.1 拒绝层级（三档 — 必背）

```
═══════════════════════════════════════════════════════════════
                JW 对血液 / 血制品的态度三档
═══════════════════════════════════════════════════════════════
档 1：教义绝对禁止（所有 JW 拒）⛔
  ├─ Whole blood 全血
  ├─ Packed RBCs 浓缩红细胞
  ├─ Platelets 血小板
  ├─ Plasma / FFP 血浆 / 新鲜冷冻血浆
  └─ White blood cells 白细胞
        ↓
档 2：Individual conscience（每个 JW 个人选择）⚠️ 必问
  ├─ Albumin 白蛋白
  ├─ Immunoglobulins 免疫球蛋白
  ├─ Clotting factors 凝血因子
  ├─ Cell saver 术中自体血回输（血未离开"循环连续性"）
  ├─ Hemodilution 术前血液稀释
  └─ Hemoglobin-based oxygen carriers（实验性）
        ↓
档 3：通常接受（绝大多数 JW 不拒）✅
  ├─ Non-blood volume expanders（NS / LR / colloid）
  ├─ Erythropoietin (EPO) 刺激自体造血
  ├─ IV iron 刺激自体造血
  ├─ Tranexamic acid (TXA) 止血药
  ├─ Vitamin K / desmopressin
  └─ 几乎所有手术 / 化疗 / 器官移植 / 药物
═══════════════════════════════════════════════════════════════
```

### 2.2 CK 4 大场景（按 capacity + 年龄分）⭐ 必背

| 场景 | Stem 关键 | 正解 | 错选 |
|---|---|---|---|
| **① Adult JW + capacity + 拒输血** | "Patient is alert, oriented, refuses transfusion" | **尊重 autonomy** + 提供替代（EPO / IV iron / TXA / cell saver）+ 详细记录 | Court order / 偷输 / 拒治疗他 |
| **② JW parents + minor + 拒救命输血** ⭐ 最高频 | "8-year-old bleeding child, parents refuse transfusion" | **紧急 → 立即输血 + 同时申请 court order**；非紧急 → **court order 先行** | 尊重父母 / 等家庭讨论 / 转院 |
| **③ Pregnant JW + 拒输血** | "32-year-old G2P1 with placenta previa, refuses blood" | **产前充分讨论 + 详细记录** + 优化 Hb（IV iron + EPO）+ **备 cell saver + TXA**；分娩时**仍尊重 wishes**（成人 + capacity）| 紧急偷输 / 法院介入（成人不适用）|
| **④ Unconscious JW + advance directive** | "Patient carries JW card refusing blood" | **遵循 advance directive 不输血** | Emergency presumption of consent（有明确文件 = 不能推定）|
| **⑤ Unconscious + 无任何文件 + 紧急** | "Found unconscious, no ID, no family" | **输血**（emergency presumption of consent — 没文件 = 不能推定他是 JW）| 因可能是 JW 而不输 |

### 2.3 经典陷阱（CK 反复考）

> [!danger] JW 高频陷阱
> - **陷阱 1**：JW 父母拒救命输血 → 尊重 autonomy ❌ → **Court order**（parental autonomy ≠ child's life）
> - **陷阱 2**：成年 JW 拒输血 → 找 ethics committee ❌ → **不是 intractable disagreement，他自己拒 = autonomy → 尊重**
> - **陷阱 3**：JW = 拒所有血制品 ❌ → **Albumin / IG / clotting factors / cell saver = individual choice**，**必须问** patient
> - **陷阱 4**：JW 孕妇大出血 → 紧急偷输救命 ❌ → 成年 + capacity + 产前明确拒 → **尊重即使会死**
> - **陷阱 5**：JW unconscious + 无文件 → 因为可能是 JW 不输血 ❌ → **没明确文件 = emergency consent 推定 = 输血**
> - **陷阱 6**："非理性 = no capacity" ❌ → JW 拒输血看似非理性，但只要 4 capacity 要素（understand / appreciate / reason / communicate）满足 → **有 capacity → 尊重**

### 2.4 Memory Hook

> [!success] JW 三句口诀
> 1. **成年自己拒 → 尊重 autonomy**（即使会死）
> 2. **父母替小孩拒 → Court order**（孩子的命 > 父母信仰）
> 3. **替代方案先上 → Cell saver / EPO / IV iron / TXA**（不是非血即死）
>
> **反射钩**："Parents refuse + life-saving + standard of care" → **Court order**（同时启动治疗）

## 三、Christian Science 基督科学派 ⭐⭐ 第 2 高频

> [!info] 教派背景
> 19 世纪美国 Mary Baker Eddy 创立。核心教义：**"疾病是 illusion 不是物质实在，唯一 healing 方式是 prayer + spiritual understanding"** → **拒绝几乎所有医学治疗**（药物 / 手术 / 疫苗 / 检测 / 诊断）。区别 JW 仅拒输血 — **Christian Science 拒整个西医体系**。

### 3.1 CK 高频场景

| 场景 | Stem 关键 | 正解 |
|---|---|---|
| **DKA 儿童父母只祈祷** ⭐ 经典 | "10-year-old child with new T1DM in DKA, parents are Christian Science, refuse insulin" | **Court order 强制 insulin + 报告 CPS**（儿童保护服务 Child Protective Services）|
| **阑尾炎儿童父母拒手术** | "12-year-old with appendicitis, parents refuse surgery, want prayer healing" | **紧急 → 立即手术 + court order**；CPS report |
| **白血病儿童父母拒化疗** | "7-year-old ALL, parents refuse chemo, opting for spiritual healing" | **Court order 强制 chemo**（可治愈白血病 = 救命标准）|
| **产妇 Christian Science 拒胎心监护** | "Christian Science woman in labor refuses fetal monitoring" | 成年 + 单纯拒**胎儿监护**（非救命）→ **尊重**；但若胎儿窘迫 + 拒紧急 CS → 复杂，考虑 court order（多重权益）|
| **Adult Christian Science + 心脏病拒治** | "Adult refuses heart catheterization, opts for prayer" | **尊重 autonomy**（同 JW 框架，成人 + capacity）|

### 3.2 核心陷阱

> [!danger] Christian Science 陷阱
> - **陷阱 1**：Adult 拒西医 = 强制治疗 ❌ → **尊重**（同 JW，capacity + informed 即尊重）
> - **陷阱 2**：Minor + 父母拒 = 尊重信仰 ❌ → **Court order + CPS report**（这是 child neglect / endangerment）
> - **陷阱 3**：CSP（Christian Science practitioner，他们的"宗教 healer"）= 医生 ❌ → **CSP 不是 licensed medical provider，他们的"医嘱"无法律效力**
> - **陷阱 4**："Christian Science 等于 JW" ❌ → **JW 只拒输血**；Christian Science **拒所有医学**

### 3.3 Christian Science vs JW 速记对照

| 维度 | **Jehovah's Witness** | **Christian Science** |
|---|---|---|
| 拒绝范围 | **仅血制品**（手术 / 药物 / 化疗多接受） | **几乎所有医学治疗** |
| 教义来源 | 《圣经》Acts 15:29 字面解释 | Mary Baker Eddy "Science and Health" |
| 替代方案 | Cell saver / EPO / IV iron 多接受 | 仅祈祷 / spiritual healing |
| Adult 处理 | 尊重 + 提供血液替代 | 尊重 + 提供医学信息（但不强求）|
| **Minor + 父母拒** | **Court order**（输血） | **Court order + CPS**（任何救命治疗）|
| **CK 经典 stem** | 出血儿童 / pregnant + previa | **DKA 儿童 / 阑尾炎儿童 / 白血病儿童** |

## 四、Catholic 天主教 ⭐⭐ 第 3 高频

> [!info] 教派背景
> 全球最大基督教派（约 13 亿）。**不**反对绝大多数医学治疗 — **不**像 JW / Christian Science 那样禁忌广泛。**只反对几个特定生殖 / 终末期问题**。

### 4.1 Catholic 反对的医疗（CK 必背 5 项）

```
═══════════════════════════════════════════════════════════════
   Catholic 教义反对的 5 项医疗（CK 反复考）
═══════════════════════════════════════════════════════════════
① 人工避孕 (Artificial contraception) ⭐
   ├─ OCP / IUD / condom / sterilization / emergency contraception
   └─ 仅接受：natural family planning (NFP / 排卵期回避)

② 堕胎 (Abortion) ⭐
   ├─ Elective 几乎全反对
   └─ 例外争议：母体 life threat（"double effect" 原则）

③ 直接安乐死 / Physician-Assisted Suicide ⭐
   ├─ Direct euthanasia 反对
   └─ 但接受：撤除 disproportionate means（不延长 dying process）+ 充分疼痛控制

④ IVF / 体外受精（部分情况）
   ├─ 反对：选择性胚胎销毁 / 第三方供精供卵 / surrogate
   └─ 个体决定 IVF 程度差异大

⑤ Sterilization 绝育（vasectomy / tubal ligation）
   ├─ 教义反对作为避孕目的
   └─ 接受：medical indication（如子宫切除治疗肿瘤）
═══════════════════════════════════════════════════════════════
```

### 4.2 关键概念：Ordinary vs Extraordinary Means（CK 中频）

| 概念 | 定义 | Catholic 立场 | CK 应用 |
|---|---|---|---|
| **Ordinary / Proportionate means** | 益处 > 负担的治疗 | **必须提供**（道德义务）| 抗生素 / IV 液 / 基础喂养 |
| **Extraordinary / Disproportionate means** | 负担 > 益处 + 不能逆转 underlying disease | **可拒绝 / 撤除**（不是安乐死）| ICU 终末期 / Aggressive chemo with no benefit |

> [!warning] Catholic end-of-life 关键区别
> - **撤除 disproportionate / extraordinary means**（如撤呼吸机 in dying patient）= **可接受**（不是杀人，是"接受死亡的自然过程"）
> - **直接安乐死 / PAS**（如给致死剂量 morphine 故意致死）= **反对**
> - **充分 pain control**（即使可能加速死亡 - "double effect"）= **可接受**

### 4.3 Catholic 医院政策（CK 高频陷阱）⭐

> [!danger] Catholic-affiliated hospital 政策
> 在美国约 1/6 医院是 Catholic-affiliated（属于 Ethical and Religious Directives, ERDs 管辖）：
>
> | 服务 | Catholic 医院政策 |
> |---|---|
> | **避孕咨询 / contraception** | **不提供** routine（除非治疗 medical condition） |
> | **Tubal ligation 输卵管结扎** | **不做**（除非合并其他 indication）|
> | **Vasectomy 输精管结扎** | **不做** |
> | **Emergency contraception 紧急避孕**（性侵后） | **多数不提供 plan B**（但 federal law 部分场景要求） |
> | **IVF** | **不做** |
> | **Elective abortion** | **不做** |
> | **End-of-life withdrawal of disproportionate means** | **接受** |
>
> **CK 经典 stem**：患者在 Catholic hospital + 要 OCP / tubal ligation / emergency contraception → 正解 = **referred to another provider / facility**（医生 / 医院**不能强迫提供违背机构政策的服务**，但**必须告知 + 转诊**，不能 abandon）

### 4.4 医生个人良心（Conscientious Objection）

> [!info] 医生个人 Catholic 信仰 + 患者要求避孕 / 堕胎
> - 医生**有权拒绝**亲自提供（conscientious objection）
> - **但必须**：
>   1. **告知**患者该服务**存在且合法**（不能隐瞒）
>   2. **转诊**给愿提供的医生（refer to another provider）
>   3. **不能 abandon / 谴责 / 延误**患者
>   4. **紧急情况** (life-threatening) → 必须救（个人良心**不能**凌驾紧急生命权）
>
> **CK 经典陷阱**：医生 Catholic 拒开 OCP → 选项
> - ✅ "Inform the patient and refer to another provider"
> - ❌ "Refuse and dismiss the patient"（abandon ❌）
> - ❌ "Provide OCP against personal beliefs"（强迫医生 ❌）
> - ❌ "Tell patient OCP is morally wrong"（不当 judgmental ❌）

### 4.5 Catholic 经典陷阱

> [!danger] Catholic 陷阱速查
> - **陷阱 1**：Catholic 反对所有医学 ❌ → **只反对 contraception / abortion / direct euthanasia / IVF（部分）/ sterilization**；绝大多数医学接受
> - **陷阱 2**：Catholic hospital 必须提供任何服务 ❌ → 可拒绝违 ERDs 的服务（避孕 / 绝育），**但必须转诊**
> - **陷阱 3**：医生 Catholic 个人良心 = 可以 dismiss patient ❌ → **必须转诊** + 告知服务存在
> - **陷阱 4**：撤呼吸机 = 安乐死 = Catholic 反对 ❌ → **撤 disproportionate means** ≠ **euthanasia**，Catholic 接受
> - **陷阱 5**：临终高剂量 morphine = 安乐死 ❌ → **"double effect"** — 主要意图缓解疼痛 + 可能副作用加速死亡 → **道德可接受**

## 五、Muslim 穆斯林 ⭐ 中频

> [!info] CK 主要考点
> 不是"拒绝医学"问题（穆斯林一般接受现代医学），而是**特定细节冲突**：猪源药品 / Ramadan 斋月 / female modesty / 死后处理。

### 5.1 CK 高频场景

| 场景 | Stem 关键 | 正解 |
|---|---|---|
| **猪源 insulin / heparin / gelatin capsules** | "Muslim patient with T1DM, family asks if insulin contains pork" | **提供合成 / 重组替代**（recombinant human insulin / synthetic heparin / non-gelatin capsule）；多数 Islamic scholars 在**无替代时**允许猪源（pikuach nefesh 类似 — 救命大于禁忌）|
| **Ramadan 斋月 + insulin** | "Muslim with T1DM during Ramadan refuses food / water sunrise to sunset" | **与患者讨论 dosing 调整**（pre-dawn meal Suhoor + sunset meal Iftar 之间用 long-acting insulin；short-acting 调整）；**鼓励但不强迫**患者**取得宗教 dispensation**（伊斯兰允许病人不 fast，但患者可能仍想 fast）|
| **Female patient + male physician** | "Muslim woman requests female physician for pelvic exam" | **尽量提供 female physician / female chaperone**；不能时**充分解释 + 同性陪护** |
| **Death + body handling** | "Muslim patient just died, family asks about washing / burial" | **24 小时内土葬**（ghusl 仪式清洗 + 白布裹尸 kafan + 朝麦加方向）；**避免 autopsy**（除非法律强制）；尊重家属 ritual |
| **Halal diet** | "Muslim patient in hospital, food preferences" | 提供 **Halal food**（无猪 / 无酒精 / 特定屠宰）|

### 5.2 关键概念

> [!info] Pikuach Nefesh 类似原则（Saving Life Trumps Restrictions）
> Islamic 教义类似 Judaism 的 **pikuach nefesh** — **救命大于宗教禁忌**：
> - 无猪源替代 + 救命 → **可接受猪源 insulin / heparin**
> - 救命急诊 + Ramadan → **可中断 fast**
> - 救命急诊 + 异性医生 → **可接受**

### 5.3 Muslim 陷阱

> [!danger] Muslim 陷阱
> - **陷阱 1**："Muslim 拒一切猪源" ❌ → **提供替代是首选**；**无替代 + 救命 → 多数允许**
> - **陷阱 2**："Ramadan 期间必须停 insulin" ❌ → **调整 dosing**（Suhoor / Iftar 窗口），不要停 insulin → DKA
> - **陷阱 3**：女患者拒男医生 → 转诊 / 拒诊 ❌ → **尽量满足 female provider**；不能时 explain + chaperone
> - **陷阱 4**：Muslim 全部拒 autopsy ❌ → **多数避免但非绝对**；**法律强制 autopsy**（凶杀 / 不明死因）必须执行

## 六、Orthodox Jewish 正统犹太 ⭐ 中频

> [!info] CK 主要考点
> Sabbath restrictions / Kosher diet / Brain death 定义争议 / Halakhic decision-making（rabbi 顾问）。

### 6.1 CK 高频场景

| 场景 | Stem 关键 | 正解 |
|---|---|---|
| **Sabbath 期间住院 / 出院** | "Orthodox Jewish patient cannot drive / use electricity from Friday sunset to Saturday sunset" | 紧急医疗 = **pikuach nefesh**（救命大于 Sabbath，全 rabbinic 共识）→ 患者**可破 Sabbath 接受治疗**；非紧急 → **推迟到 Sabbath 后** |
| **Brain death + 撤呼吸机** | "Patient declared brain dead by neurologic criteria, Orthodox Jewish family refuses withdrawal" | 部分正统派**只承认 cardiac death**（不承认 brain death）→ **rabbinic consultation + ethics committee**（这是真正 ethics committee 适用场景） |
| **Circumcision 8 天 (Brit milah)** | "Orthodox Jewish male newborn, parents request circumcision on day 8" | 文化 + 宗教仪式 → **配合** scheduling（hospital 多有 mohel 来做或 hospital 周期）|
| **Kosher food + 用药** | "Patient strictly kosher, asks about medication ingredients" | 提供 kosher 替代 / gelatin-free dosage forms；急救药例外 |
| **Male / female modesty** | 类似 Muslim | 性别匹配 provider 时尽量满足 |

### 6.2 关键概念

> [!info] Pikuach Nefesh（救命大于一切）
> 犹太法律核心：**几乎所有宗教戒律在救命前让步**（除三条不让：偶像崇拜、严重性 immorality、谋杀）。**CK 经典 stem**：Orthodox Jewish + Sabbath + 心梗 → **立即送医 + 治疗**，宗教允许 + 鼓励。

### 6.3 Brain Death 争议（高级考点）

> [!warning] Brain death vs cardiac death — Orthodox Jewish 内部分裂
> - **多数正统派 + 大部分美国 Orthodox 组织**：**接受 brain death = death**（与主流医学一致）
> - **少数严格派**（Chabad / 部分 ultra-Orthodox）：**只承认 cardiac death**
> - **CK stem**："Family from strict Orthodox community refuses withdrawal after brain death" → **ethics committee + rabbinic consultation + 多次 family meeting**（这是真正符合 ethics committee 触发条件的场景 — intractable disagreement）
>
> 这与 [[完整笔记/Peixuan分科笔记/Ethics_Master]] 的 ethics committee 触发条件 §15 同源。

### 6.4 Orthodox Jewish 陷阱

> [!danger] Orthodox Jewish 陷阱
> - **陷阱 1**：Sabbath 不能治疗 ❌ → **救命必须治疗**（pikuach nefesh）
> - **陷阱 2**：所有 Orthodox 都拒 brain death ❌ → **多数接受**；只少数严格派争议
> - **陷阱 3**：Kosher = 绝对禁忌 ❌ → 多数情况尊重；**救命药物例外**

## 七、Amish 阿米什人（简提）

> [!info] CK 罕考但要识别
> - **不拒紧急医学**（与 Christian Science 区别！）— 创伤 / 难产 / 心梗 → 送医
> - **拒部分现代医学**：常规疫苗（部分豁免）/ 健康保险 / 选择性手术
> - **教义重点**：community-centered / 自给自足 / 不依赖政府
> - **CK 经典 stem**：Amish 父母拒儿童 vaccine → **教育 + 不评判 + 尊重 vaccine refusal**（多数州法律允许 religious exemption）；**不是** court order（routine vaccine 不达 court order threshold — 见 [[完整笔记/Peixuan分科笔记/ethics]] §minor consent）

## 九、跨群体通用 SOP（4 步法）⭐

> [!success] 任何宗教 / 文化冲突的 4 步法
> ```
> Step 1：识别（Identify）
>    └─ 询问患者宗教 / 文化 belief 是否影响 medical decision
>       "Are there any beliefs or values that are important to your medical care?"
>
> Step 2：不评判（Non-judgmental）
>    └─ 不批评 / 不嘲讽 / 不强迫
>       "I want to understand and respect what's important to you"
>
> Step 3：探索（Explore + Educate）
>    └─ 探索 underlying reason（恐惧 / 误解 / 教义具体内容）
>    └─ 教育（提供准确医学信息）
>    └─ 与 patient / family / community elder / spiritual leader 沟通
>
> Step 4：替代方案（Alternatives + Workarounds）
>    └─ 多数信仰允许 workaround（cell saver / synthetic insulin / 调 dosing / 撤 extraordinary means）
>    └─ 与 ethics committee / rabbi / imam / priest 协作（必要时）
> ```

> [!warning] 4 步法**不**适用的情形
> - **Minor + 父母拒救命标准治疗** → 跳过 4 步直接 **Court order**（child 的最佳利益凌驾父母信仰）
> - **Emergency life-threatening + 无 advance directive** → 跳过 4 步直接**紧急救治**

## 十一、易混陷阱（普适跨群体）

> [!danger] 跨群体高频陷阱速查
> 1. **"非理性 = no capacity" ❌** → 宗教拒治疗看似非理性，但 4 capacity 要素满足 → **有 capacity → 尊重**（JW / Christian Science / Catholic 全适用）
> 2. **"成年 + 宗教拒治疗 = ethics committee" ❌** → 自己拒 = autonomy 行使 = **尊重**，不是 intractable disagreement
> 3. **"Minor + 父母拒救命 = 尊重信仰" ❌** → **Court order ± CPS**（参见 [[完整笔记/Peixuan分科笔记/ethics]] §minor consent / §court order）
> 4. **"医生个人信仰 = 可以 abandon patient" ❌** → 必须**告知服务 + 转诊**（refer not refuse）；紧急例外（必须救）
> 5. **"Catholic 医院 = 不能做任何避孕 / 绝育" ❌** → **病理 indication**（如子宫切除治疗肿瘤）可做；**单纯避孕**不做但**必须转诊**
> 6. **"JW = 拒所有血制品" ❌** → **Albumin / IG / cell saver = individual choice**（必须问）
> 7. **"Christian Science = JW" ❌** → JW **只拒血**；Christian Science **拒所有医学**
> 8. **"Amish = Christian Science" ❌** → **Amish 接受紧急救命**；Christian Science 拒
> 9. **"宗教信仰不能 override 法律" ❌** → 部分**可**（adult vaccine refusal / DNR / advance directive 都是 protected）；只**不能 override 儿童生命权**
> 10. **"看到宗教就找 ethics committee" ❌** → 多数情况按 capacity / surrogate 标准 SOP；ethics committee **只在真正 intractable disagreement**（如 Orthodox Jewish brain death 争议）

## 十二、Memory Hooks（跨群体记忆挂钩）

> [!success] 一句话钉死
> **"成人尊重，孩子救命，替代先上，转诊不弃"**
>
> - **成人尊重** = Adult + capacity + informed → respect autonomy（不管什么信仰）
> - **孩子救命** = Minor + 父母拒救命标准治疗 → Court order（± CPS for Christian Science）
> - **替代先上** = 多数信仰允许 workaround（cell saver / synthetic / non-pork / 调 dosing）→ 先探索替代
> - **转诊不弃** = 医生 conscientious objection → refer not abandon
>
> **CK 反射钩**（看到 stem 立刻问 3 问）：
> 1. **谁拒**？— Adult (尊重) vs Parent for minor (court order)
> 2. **拒什么**？— 救命 (强力 push) vs 非救命 (容易尊重) vs 仅是机构政策 (转诊)
> 3. **有替代吗**？— Cell saver / synthetic / 调 dosing / 改方案 → 先试替代

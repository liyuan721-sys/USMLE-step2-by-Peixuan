## 〇、终极心法（6 条搞定 90% 的题）

> [!success] 拿不准时按这 6 条选答案
> 1. **病人有 capacity（决策能力）→ 听病人的**（autonomy 自主权几乎永远胜出）。
> 2. **病人没 capacity → 先找 advance directive（预先指示），再找 surrogate（代理人）**。
> 3. **保密（confidentiality）默认守口**，除非：虐待 / 传染病 / 威胁特定他人（Tarasoff）/ 自伤 / impaired physician（受损同事）。
> 4. **沟通题 → 先承认情绪（acknowledge emotion），再给信息**；永远不选空洞安慰 / 太早转介 / 命令式。
> 5. **医疗差错 → 诚实披露（honest disclosure）**，与是否造成伤害无关。
> 6. **真拿不准 → 选「最尊重病人 + 最像开放式对话 + 不空洞 reassure」的那个选项。**

为什么是这 6 条：CK 的 Ethics 几乎不考"绝对禁止"，而是考**优先级排序**——谁说了算（capacity → directive → surrogate）、什么时候保密让位于安全、沟通时先情绪后信息。剩下 10% 是研究伦理 / 文化冲突 / minor 边界 / impaired physician，见下方对应板块。

## 一、4 大伦理原则（Four Principles）

| 原则 | 含义 | CK 倾向 |
|---|---|---|
| **Autonomy（自主）** | 病人对自己身体有完全控制权 | **几乎永远胜出** |
| **Beneficence（行善）** | 为病人最大利益行事 | 次要 |
| **Non-maleficence（不伤害，Do no harm）** | 不造成伤害 | 次要 |
| **Justice（公正）** | 公平分配资源（器官 / 资源有限时） | 罕见考 |

> [!tip] CK 黄金法则
> 选项里出现"**尊重病人意愿 / respect patient's wishes**" —— **大概率是答案**。

> [!info] 法律根基 —— 判例为什么重要
> **Schloendorff v. Society of New York Hospital（1914）**：确立"**未经病人同意进行治疗 = 殴打罪（assault and battery）**"。这一句话是现代知情同意（informed consent）整个制度的法律地基——CK 偶尔直接问"哪个判例确立病人自主权"，记住人名 + "未同意即殴打"即可。

## 二、谁说了算？决策能力与代理（Capacity → Directive → Surrogate）

### 2.1 Capacity（决策能力）vs Competence（行为能力）—— 别混

| 概念 | 谁判断 | 性质 |
|---|---|---|
| **Capacity（决策能力）** | **主管医生**（任何医生都可，不一定要精神科） | **临床判断**，可随时间 / 情境变化 |
| **Competence（行为能力）** | **法官** | **法律概念**，法庭裁定 |

**Capacity 4 要素**（Hook：「**理解—表达—推理—预见后果**」）：

```
① Understand（理解信息）
② Express a choice（表达选择）
③ Reason（理性推理）
④ Appreciate consequences（理解后果）← 最容易被忽略的一条
```

> [!warning] CK 高频陷阱（Capacity）
> - 病人做了你觉得"非理性 / 不对"的决定（如 Jehovah's Witness 耶和华见证人拒输血、老人拒化疗）→ **不要立刻判 incapacity**，先评估 4 要素；符合就**尊重**。
> - **自杀念头（suicidal）的患者视为 lack capacity** → 不可让其离开，必要时非自愿留观（involuntary hold）。
> - Capacity 是**动态**的：暂时谵妄 / 醉酒 / 镇静恢复后可重评 → surrogate 决定权**不是永久标签**（详见 🧩 + 错题 Q19048）。

### 2.2 Surrogate（代理决策）顺序 + 决策标准

**代理人顺序**（病人无 capacity 且无 advance directive 时，由高到低）：

```
配偶 Spouse → 成年子女 Adult children → 父母 Parents
→ 成年兄弟姐妹 Adult siblings → 其他亲属 Other relatives
```

**决策标准**（由高到低，这是 CK 最爱考的优先级）：

| 标准 | 优先级 | 含义 |
|---|---|---|
| **Substituted judgment（替代判断）** | **优先** | "病人若有 capacity **会怎么选**？"——基于病人本人价值观，**不是代理人自己想要什么** |
| **Best interest（最佳利益）** | 次之 | 完全无依据时，选"对病人客观最好的" |

> [!tip] 为什么 substituted judgment 优先
> 代理人的职责是"替病人说出病人的意愿"，不是替病人做"代理人认为对的决定"。所以题目里 surrogate 说"我希望全力抢救"但病人生前明确说过不要插管 → 答案要 refocus 回**病人本人意愿**（错题 Q23113 即此型）。

### 2.3 Advance Directives（预先指示）

| 类型 | 含义 |
|---|---|
| **Living will（生前遗嘱）** | 提前写明特定情境下的医疗意愿 |
| **Health-care proxy / DPOA（Durable Power of Attorney，医疗代理 / 持久授权书）** | 指定一个人替自己做决定；proxy 表达的是**病人的意愿，不是 proxy 自己的** |
| **DNR（Do Not Resuscitate，拒绝复苏）** | **只**意味着心肺骤停时不做 CPR（Cardiopulmonary Resuscitation，心肺复苏）；**其他所有治疗照常**（高频陷阱，见 §五） |

**决策冲突标准答案速查**：

| 场景 | 答案 |
|---|---|
| 病人有 capacity，家属反对 | **病人意愿优先** |
| 病人无 capacity，有 advance directive | **按 directive 执行** |
| 病人无 capacity，无 directive，有家属 | **按 surrogate 顺序 + substituted judgment** |
| 病人无 capacity，无 directive，无家属 | 医生 + 伦理委员会按 **best interest** |
| 病人无 capacity，无 surrogate，紧急且延误危及生命/器官 | **直接 emergency treatment**（implied consent 推定同意） |

## 三、知情同意（Informed Consent）

**4 要素**（缺一不成立）：

```
① Capacity（有决策能力）
② Disclosure（医生告知：风险 risks + 获益 benefits + 替代方案 alternatives）
③ Understanding（病人用大白话理解）
④ Voluntariness（自愿，无胁迫 coercion）
```

**不需要 informed consent 的例外**：

| 例外 | 说明 |
|---|---|
| **Emergency（紧急）** | 病人无意识 / 生命危急 → **implied consent（推定同意）**，直接救命 |
| **Therapeutic privilege（治疗特权）** | 极罕见，告知会严重伤害病人 —— **CK 几乎不选这个** |
| **Patient waiver（病人放弃）** | 病人主动放弃知情权，可指定家属代收信息 |
| **Public health emergency（公共卫生紧急）** | 强制隔离 / 治疗某些传染病 |

> [!warning] CK 高频陷阱（Informed Consent）
> - 家属说"**别告诉病人诊断**" → **拒绝**，病人有知情权（除非病人本人主动放弃）。处理 = 先探查家属担忧，再合作制定逐步告知计划，不立刻硬告知也不一味隐瞒。
> - 病人主动说"**我不想知道**" → **尊重**，可指定家属代收信息（错题 Q3614 衍生场景）。
> - 急诊昏迷大出血、家属联系不到 → **直接救命**（implied consent），不等任何审批 / 法院 / IRB。
> - 病人主动问"你做过几台这种手术" → 医生**必须如实披露**自己的经验水平。

## 四、未成年人同意（Minors）

**一般规则**：minor 普通医疗需**父母 / 监护人同意**；**Emancipated minor（已解放未成年人）**——已婚 / 参军 / 经济独立 / 法院裁定 / 自己已有孩子——**全权自己决定**。

**Partial emancipation（部分解放）—— 不需家长同意的领域**（Hook：「**性、孕、瘾、心**」）：

```
STDs/STI 检查与治疗   ← 所有州
避孕 Contraception     ← 所有州
怀孕相关护理 Pregnancy ← 所有州
物质成瘾 Substance abuse ← 多数州
精神健康 Mental health   ← 部分州
```

| 场景 | 标准答案 |
|---|---|
| Minor 紧急情况 | **立即治疗**，事后通知家长 |
| **家长替孩子拒绝救命治疗**（如 JW 拒给婴儿输血） | **法院介入，强制治疗** |
| 家长拒绝非紧急但有益治疗（如疫苗） | **教育沟通**，通常尊重家长 |
| 14 岁少女要避孕药、母亲反对 | **给避孕药 + 保密** |
| Minor 堕胎 | 因州而异，CK 选"按州法律" |

> [!warning] 成人拒输血 vs 父母替孩子拒输血 —— 两套处理（必考对比）
> - **成年 JW 自己有 capacity 拒输血** → **尊重**（autonomy）。
> - **父母替孩子（minor）拒救命输血** → **法院介入强制输血**——孩子的生命权高于父母的宗教自主权，父母不能替孩子拒明确必要的救命治疗。
> - 注意：**联合监护（joint custody）下，非紧急医疗通常一方家长同意即可**；只有"双方都拒绝救命治疗"才需要 court order（错题 Q3624）。

## 五、保密与例外（Confidentiality & Mandatory Reporting）

**一般原则**：未经病人同意不得披露 —— 但有例外。

**强制上报（必报，不需病人同意）**：

```
儿童虐待 Child abuse
老人虐待 Elder abuse（多数州）
法定传染病 Reportable diseases（TB 结核 tuberculosis、某些 STI 性传播感染 sexually transmitted infection）
枪伤 Gunshot wounds
法院命令 Court order
```

**不强制（需谨慎、尊重病人选择）**：

| 场景 | 标准答案 |
|---|---|
| 怀疑**成年伴侣家暴（IPV，Intimate Partner Violence 亲密伴侣暴力）** | **不强制上报**；提供资源、尊重病人选择（因州而异） |
| **HIV+ 不愿告诉伴侣** | **两步走**：① 先劝病人自己告知 → ② 不行则通过 public health（公共卫生部门）通知，不自己直接打电话 |

**危险行为类（必须介入）**：

| 场景 | 标准答案 |
|---|---|
| 病人威胁伤害**特定可识别他人**（Tarasoff） | **警告潜在受害者 + 通知警方** |
| 病人有自杀风险 | **不可让其离开**，精神科评估，必要时 involuntary hold |
| 司机有癫痫 / 视力问题不能开车 | 劝其停驾 + **报告 DMV**（Department of Motor Vehicles，机动车管理局）（多数州） |
| 同事酗酒上班 / 偷药（impaired physician） | **报告主管 / 医院**（见 §九） |

> [!info] Tarasoff 案 —— 为什么保密要让位
> 加州判例，确立治疗师对"**可识别的潜在受害者**"负有**警告义务（duty to warn）**。核心逻辑：**保密原则不能凌驾于第三方的生命安全之上**。CK 考点 = 病人具体说"我要杀 X" → 不是单纯保密，而是警告 X + 报警。

> [!warning] CK 高频陷阱（Confidentiality）
> - 成年女病人被丈夫家暴，你立刻报警 → **错**！成年 IPV **非强制上报**，提供资源、尊重选择。
> - HIV+ 不告诉伴侣，你立刻打电话给伴侣 → **错**！必须先劝病人自己告知。
> - 家属想私下告诉你病人的信息（collateral information）→ 可以**被动倾听**，但**不能反向透露病人的隐私**给家属，也不强制把病人拉进来（错题 Q3742）。

## 六、医疗差错披露（Disclosure of Medical Errors）

| 场景 | 标准答案 |
|---|---|
| **自己犯了错** | **主动诚实告知病人**（发生了什么 / 影响 / 下一步）+ 上报系统 |
| **同事犯错** | **先与同事本人沟通** → 不行再上报 |
| 病人问"是不是出错了" | **诚实告知** |
| 错误被掩盖、你想披露 | **披露**，无论后果 |

> [!warning] CK 高频陷阱（Disclosure）
> - 错误**没造成伤害**就觉得"没必要说" → **错**！诚实披露是绝对原则，**与是否造成 harm 无关**。
> - 同事犯错你立刻去找院长 → **错**！**先与同事本人沟通**（除非涉及病人安全需立即升级）。
> - 禁止：隐瞒 / 推给别人 / 改病历。

## 七、临终关怀（End-of-Life Care）

### 7.1 撤治疗 vs 不上治疗 —— 伦理等同

> [!info] 核心一句话
> **撤呼吸机（withdraw）= 不上呼吸机（withhold）= 撤人工营养/补液**，在伦理和法律上**完全等同**。人工营养/补液（artificial nutrition/hydration）算医疗治疗，不是必须永远维持。

### 7.2 几个必背区分

| 概念 | 关键点 |
|---|---|
| **DNR ≠ Do Not Treat** | DNR 只管心肺骤停不做 CPR；肺炎照样用抗生素、能溶栓照样溶栓 |
| **Doctrine of Double Effect（双重效应）** | 给吗啡缓痛即使可能加速死亡，只要**意图是缓痛**就允许；意图致死才是安乐死 |
| **PAS（医生协助自杀）** | 医生**提供手段**（开处方）、病人**自己用**；仅个别州合法，CK 默认拒绝 |
| **Euthanasia（安乐死）** | 医生**亲自执行**（如注射）；大部分州非法，即使病人要求也拒绝 |
| **Medical futility（医疗无效）** | 治疗几乎不可能达成病人目标 → 先沟通目标、争取一致；医生不被要求提供明确无益的治疗 |
| **Brain death（脑死亡）= death** | 全脑功能不可逆停止 = 法律死亡 → 可撤维持治疗（无需家属同意撤）、可讨论器官捐献 |
| **Persistent vegetative state（持续植物状态）** | 有睡眠觉醒周期、无意识、**不是死亡** → 撤治疗按 directive/surrogate 决定 |

> [!warning] CK 高频陷阱（End-of-Life）
> - "撤呼吸机算杀人" → **错**！撤 = 不上，伦理等同。
> - DNR 病人得了肺炎就不治了 → **错**！DNR ≠ 不治疗，该用抗生素就用，能 thrombolysis 就 thrombolysis（错题 Q107705；转移癌不是溶栓禁忌）。
> - 给吗啡怕"加速死亡" → 只要意图是缓痛 = 双重效应，**可以做**。
> - 病人要求安乐死你帮忙 → **错**！默认拒绝。

### 7.3 Hospice vs Palliative care

- **Palliative care（姑息治疗）**：任何严重疾病阶段都适用，**可与治愈性治疗并行**，目标缓解症状 + 提高生活质量。
- **Hospice（临终关怀）**：预期寿命较短（一般 < 6 个月）、**以舒适照护为主**——进 hospice = **放弃 curative（治愈性）治疗，但保留 palliative**（错题 Q21141）。

## 八、器官捐献与脑死亡（Organ Donation）

| 原则 | 说明 |
|---|---|
| **Voluntary（自愿）** | 必须基于自愿同意 |
| **Cannot be sold（不可买卖）** | 商业化器官交易违法 |
| **Family has final say** | 即使病人生前同意，家属仍有最终决定权 |
| **团队分离** | 治疗团队 ≠ 器官获取团队（OPO，Organ Procurement Organization）；**不能因器官需求提前终止治疗** |
| **死亡判定在先** | 必须先确诊死亡（含脑死亡），才谈捐献 |

## 九、利益冲突 / 收礼 / 边界 / 受损同事（COI, Gifts, Boundaries）

### 9.1 收礼与边界

| 场景 | 标准答案 |
|---|---|
| 病人送**小礼物**（贺卡、烘焙） | **可接受** |
| 病人送**贵重 / 现金 / 求优待** | **拒绝**（礼貌感谢 + 说明专业边界） |
| **判断力受损病人**（躁狂等）送礼 | **拒绝**（即使不贵重） |
| 病人邀约会（治疗中或结束后） | **拒绝**（精神科病人永久禁止） |
| 给家人 / 朋友看病 | **避免**，除非紧急；若看则 **standards of care 不能降**，不被要求提供 inappropriate intervention |
| Pharma rep（药代）请吃饭 / 送礼 / 给商业机会 | **一律拒绝** |

> [!tip] 拒礼物的"正确选项"语气
> "Thank you for thinking of me, but I'm not able to accept this gift." —— 感谢 + 说明不能收，不羞辱病人。注意陷阱选项"Doctors are never allowed to accept gifts"是**绝对化错误**（小礼物可以收）。

### 9.2 Impaired Physician（受损同事）—— 先拦人，再报告

> [!danger] 时序原则：病人安全第一
> 选项里同时出现"**立刻肉身拦住同事不让接触病人**"和"**去办公室打报告**"时，**选拦人那个**——一旦发生事故后果不可逆。

```
第一步 即刻 Immediate  → 阻止其继续诊治病人（保护病人安全）
第二步 短期 Short-term → 报告医学总监 Medical Director（内部调查）
第三步 长期 Legal      → 报告州医学委员会 Medical Board（执照层面）
第四步 关怀 Support    → 转介 Physician Health Program (PHP，医生健康计划) 帮助康复
```

- **场景 A**：看到同事术前喝酒 → **立即阻止手术**（干预优先）。
- **场景 B**：非工作时间听说同事长期酗酒、目前没接触病人 → 收集证据 + **报告主管**（报告优先）。
- **场景 C**：同事因能力不足出现差错 → 先向质量委员会 / 主管报告（system-based improvement）。

## 十、沟通技巧（Communication）

> [!tip] 核心原则：先 acknowledge 情绪，再给信息

**SPIKES 法 —— 告知坏消息 6 步**：

| 字母 | 步骤 | 做什么 |
|---|---|---|
| **S** — Setting | 设置环境 | 私密、坐下、关门 |
| **P** — Perception | 了解病人认知 | "您对自己情况了解多少？" |
| **I** — Invitation | 询问意愿 | "您想知道详情吗？" |
| **K** — Knowledge | 告知信息 | 大白话 + 先 **warning shot**（"我有不太好的消息"） |
| **E** — Emotion | 回应情绪 | acknowledge，不打断 |
| **S** — Strategy/Summary | 制定计划 | 总结 + 下一步 |

**"正确选项"语言模式**（CK 沟通题答案常含）：

```
"I can see this is very difficult for you..."
"Tell me more about your concerns."
"What worries you most?"
"How are you coping with this?"
```

**"错误选项"语言模式**：

```
"Don't worry, everything will be fine."  ← 空洞 reassurance（永远错）
"You shouldn't feel that way."           ← 否定情绪
"Let me refer you to..."                 ← 太早转介
"You need to..."                         ← 命令式
立刻甩一堆医学信息（没先回应情绪）       ← 错
```

**高频沟通场景**：

| 场景 | 标准答案 |
|---|---|
| 病人哭 / 愤怒 / 恐惧 | **先 acknowledge 情绪**，不立刻给信息（愤怒 = nondefensive + 开放式问题，不转给上级、不立刻解释原因） |
| 病人不会英语 | **用专业翻译**，**禁用家属翻译（尤其禁用孩子）** |
| 病人不依从治疗 | **探究原因**，不直接换医生 / 责怪 |
| 病人否认明显问题（如酗酒） | **共情 + 开放式问题**，不对抗 |
| 病人想换医生 / 想要不必要检查 | 前者尊重并帮助；后者**教育解释**，不直接拒绝或满足 |
| 跨文化照护 | 先**理解病人的认知 / 信念**，再 counsel；宣传册是辅助不是第一步 |

## 十一、医疗系统：质量、安全、医保（Healthcare Systems）

> [!info] 为什么这块单独拎出来
> CK 现在大量考"**系统层面**"题：差错根因分析、质量工具、出院安置、医保支付模式。这些**不是传统床边伦理**，是医院管理 + 公共卫生视角，钥匙词很容易混。

### 11.1 质量与安全工具（最易混，必背区分）

| 工具 | 用途 | 时间方向 | 钥匙词 |
|---|---|---|---|
| **RCA（Root Cause Analysis）** | 找**已发生**事件的深层原因 | **回顾性**（past events） | 已出事、找 underlying causes、external committee |
| **FMEA** | 找**未来**流程的潜在失效点 | **前瞻性**（future vulnerabilities） | 新流程上线**前**、prospective |
| **Swiss Cheese Model（瑞士奶酪模型）** | 解释事故 = **多层防御漏洞同时对齐** | 概念模型 | 多个 barrier 同时失效 |
| **Fishbone / Ishikawa（鱼骨图）** | 组织"原因分类" | 分析工具 | categorize causes |
| **Lean** | 减 waste（去掉多余步骤） | QI 方法 | 减浪费、no-show↓ |
| **Six Sigma** | 减 variation（减变异） | QI 方法 | 减偏差 |
| **PDSA** | 小规模迭代试验改进 | QI 循环 | Plan-Do-Study-Act |

> [!danger] 反复错红线：RCA vs FMEA 字眼陷阱（🔴 #考前必看 #难记）
> 题干出现 "**identify future vulnerabilities**" = **FMEA**（前瞻）；出现 "**underlying causes of a past event**" = **RCA**（回顾）。Peixuan 在 Q22581 此型**错 3 次**——RCA 是**事后**、由**外部委员会**做、找**过去**事件根因；别被 "future" 字眼骗去选 RCA。

### 11.2 Just Culture（公正文化）—— 三类行为三种处理

```
Human error（无心之失）   → Console 安慰 + 改系统
At-risk behavior（冒险）  → Coach 辅导
Reckless behavior（鲁莽） → Sanction 惩戒
```

- 当系统改进已做、个体仍持续 suboptimal（如手卫生屡教不改）→ 转向 **individual accountability（个体问责）**。

### 11.3 出院安置（Disposition）

| 选项 | 适应症 |
|---|---|
| **Home Health Care（家庭健康护理）** | medically stable（病情稳定）+ homebound（卧床/出门困难）+ skilled needs（需专业护理）→ **不是住院、不是 SNF** |
| **SNF（Skilled Nursing Facility，专业护理机构）** | 需更高强度 skilled care、家中无法满足 |

> [!warning] 反复错（🔴）：Home Health Care vs admit/SNF
> 病情稳定 + 居家受限 + 需技能护理 = **Home Health Care**，别反射去 admit 或 SNF（错题 Q20431，错 2 次）。

### 11.4 医保支付与公平（Payment & Equity）

| 概念 | 要点 |
|---|---|
| **P4P（Pay-for-Performance，按绩效付费）+ ACO（Accountable Care Organization，责任医疗组织）** | 双向优化**成本 + 质量**；P4P 必须按 patient demographics 调整，否则只奖励富裕地区 → **加剧 health disparities（健康不公平）** |
| **HIPAA（Health Insurance Portability and Accountability Act，健康保险流通与责任法案）** | 保护 PHI（Protected Health Information，受保护健康信息）；披露遵循 **minimum necessary（最小必要）** 原则；口头同意在特定场景可（Q3614） |
| **PCC（Patient-Centered Care，以患者为中心的照护）** | 综合 patient values（含成本 / 可负担性）+ shared decision-making（共享决策） |

## 十二、研究伦理（Research Ethics）

| 要素 | 要点 |
|---|---|
| **IRB（伦理审查委员会）** | **任何研究开始前**（含第一个受试者）方案必须先获 IRB 批准；没 IRB，知情同意**法律上无效** |
| **知情同意** | 研究招募需 IRB 批准 + 受试者知情同意 + 可随时退出 |
| **弱势群体**（囚犯 / 儿童 / 孕妇） | **额外保护**（extra protection） |
| **Placebo 控制** | **仅当无现有有效治疗时**才允许用安慰剂 |
| **COI（利益冲突）** | industry-sponsored 人员做 data analysis = 核心 COI；双盲设计盲**不能**消除分析阶段 bias（Q106885） |
| **Therapeutic misconception（治疗误解）** | 受试者误以为参加 trial = 获得 superior care → 解释 trial 性质 + 可能不获益 |
| **EFIC（急诊研究知情同意豁免）** | FDA 21 CFR 50.24；急诊危重无法即时同意 → **由 IRB 决定**是否可 deferred consent，**不是医生个人决定** |

> [!danger] 判题口诀：研究 vs 救命（最高频研究伦理陷阱）
> - 题干提到"**研究 / 评估效果 / 新药试验**" → 第一步永远是 **"Apply to the IRB"**（或 Consult Ethics Committee），研究尚未启动时**优先级最高**。
> - 题干是"**昏迷大出血找不到家属，下一步**" → **"Proceed with emergency treatment"**（直接救命，不等 IRB 不等法院）。
> - 陷阱：急诊研究去找家属同意 → **错**！家属同意只解决"代理授权"，解决不了"研究合法性"，**只有 IRB 批准**才是法定第一步。

## 十三、Malpractice（医疗事故）—— 4 要素

**必须全部满足**（Hook：「**4D：Duty—Dereliction/Breach—Damage—Direct causation**」）：

```
① Duty（义务）       医患关系成立，医生有照护义务
② Breach（违反）     未达 standard of care
③ Damage（损害）     病人受到实际伤害
④ Causation（因果）  损害直接由违反义务造成
```

> [!info] No harm = No malpractice
> 即使医生明显违规，只要**没造成实际损害**，就**不构成 malpractice**（缺第③要素）。CK 常给"违规但病人没事"→ 答案：不构成医疗事故。

## 复习自测

> [!question]- Q1. 病人无 capacity、无 advance directive，配偶要求"全力抢救"，但病人生前多次明确说不想插管。代理人决策应基于什么？
> **Substituted judgment（替代判断）** —— 代理人的职责是说出**病人本人会怎么选**，不是代理人自己想要什么。答案 refocus 回病人生前意愿（不插管）。Best interest 只在完全无病人意愿依据时才用。

> [!question]- Q2. 题干说"上线新流程前，团队想识别可能的失效点（future vulnerabilities）"——用哪个工具？换成"事故已发生，找深层原因"呢？
> 前者 = **FMEA**（前瞻性，future vulnerabilities）；后者 = **RCA**（回顾性，past event underlying causes，外部委员会做）。两个字眼互为陷阱，是 Peixuan 反复错点。

> [!question]- Q3. 成年耶和华见证人（capacity 完好）拒绝输血 vs 父母替 6 个月婴儿拒绝救命输血——分别怎么处理？
> 成人自己拒 → **尊重（autonomy）**；父母替孩子拒救命治疗 → **法院介入强制输血**。孩子生命权 > 父母宗教自主权，父母不能替孩子拒明确必要的救命治疗。

> [!question]- Q4. DNR 病人发生急性缺血性卒中、在溶栓时间窗内，转移癌病史。能溶栓吗？
> **能**。DNR 只管心肺骤停不做 CPR，不阻止 thrombolysis 等改善生活质量的干预；转移癌**不是**溶栓禁忌。DNR ≠ Do Not Treat。

> [!question]- Q5. 看到同事手术前在喝酒。选"立即阻止手术"还是"去向 Medical Director 报告"？
> **立即阻止手术**（肉身拦人，干预优先）。病人安全第一，事故后果不可逆；报告是第二步。若是非工作时间听说同事长期酗酒、当下没接触病人，则反过来——报告优先。

> [!question]- Q6. 病人威胁"我要去杀我前妻 Jane"。保密原则要求守口吗？
> **否**。Tarasoff 义务：对可识别的潜在受害者负有 **duty to warn** → 警告 Jane + 通知警方。保密不能凌驾于第三方生命安全之上。

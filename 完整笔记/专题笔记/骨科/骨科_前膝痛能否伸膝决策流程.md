# 骨科 前膝痛 (Anterior knee pain) — "能否伸膝"决策流程

> [!info] 用途与来源
> 从 [[完整笔记/专题笔记/骨科/骨科_膝及周围疾病鉴别]] §二拆出的**单页流程图**。前膝痛是膝错题最高发区（PFPS 反复错），核心分叉只有一个字：**能不能主动伸膝 (active knee extension)**。先用这一刀切成"手术急症 vs 慢性过度使用"，再细分。
>
> 缩写：**PFPS** = patellofemoral pain syndrome 髌股疼痛综合征；**OSD** = Osgood-Schlatter disease 胫骨结节骨软骨病；**NSAIDs** = nonsteroidal anti-inflammatory drugs 非甾体抗炎药；**ROM** = range of motion 关节活动度。

---

## 一、第一刀：能否主动伸膝？

> [!danger] 伸膝装置 (extensor mechanism) = 一条链，断在哪都伸不了膝
> 股四头肌 (quadriceps) → 股四头肌腱 (quadriceps tendon) → 髌骨 (patella) → 髌腱 (patellar tendon) → 胫骨结节 (tibial tuberosity)。
> **这条链任何一处完全断裂 → 不能主动伸膝 → 手术急症。**

```
前膝痛 Anterior knee pain
│
├─【能主动伸膝吗？】
│
├─ 否（伸膝迟滞 extension lag / 重创史）──► 伸膝装置断裂（手术！）
│        │
│        ├─ >40 岁 + 髌骨【上方】间隙 + 髌骨低位(baja) ─► 股四头肌腱断裂
│        │                                              Quadriceps tendon rupture
│        └─ <40 岁 + 髌骨【下方】间隙 + 髌骨高位(alta) ─► 髌腱断裂
│                                                       Patellar tendon rupture
│        口诀：「老上低 / 轻下高」
│
└─ 是（能完全伸膝，慢性/过度使用）──► 非手术，按"痛点定位 + 年龄"分
         │
         ├─ 痛【模糊】、跑/上下楼/久坐加重、压髌+绷股四头肌痛 ─► PFPS
         │     人群：年轻女性 │ 治疗：股四头肌+髋外展强化，忌制动 ⭐
         │
         ├─ 痛【局灶】在髌骨下极/髌腱、跳跃运动员 ─────────► 髌腱炎 (jumper's knee)
         │     治疗：相对休息 + 离心训练
         │
         ├─ 痛在【胫骨结节】、青春期活动多、结节肿痛 ───────► Osgood-Schlatter (OSD)
         │     治疗：限制活动 + 拉伸（自限）
         │
         └─ 髌骨【正前方表浅】肿包、长期跪地 ──────────────► 髌前滑囊炎
               Prepatellar bursitis (housemaid's knee)
               被动 ROM 基本正常；红热重 → 排 septic bursitis 化脓性滑囊炎
```

## 二、伸膝装置断裂：股四头肌腱 vs 髌腱（手术组）

| | **股四头肌腱断裂 Quadriceps tendon rupture** | **髌腱断裂 Patellar tendon rupture** |
|---|---|---|
| **年龄** | **>40 岁** | **<40 岁** |
| **可触间隙** | 髌骨**上方** | 髌骨**下方** |
| **髌骨位置(X 线)** | **低位 patella baja**（被往下拽）| **高位 patella alta**（被往上弹）|
| **共同点** | 都"**不能主动伸膝 / 直腿抬高**"、都需**手术修复** | 同左 |

> [!tip] "老上低 / 轻下高"
> - **老**(>40) → **上**(髌上间隙) → **低**(髌骨低位) = 股四头肌腱
> - **轻**(<40) → **下**(髌下间隙) → **高**(髌骨高位) = 髌腱

## 三、能伸膝的慢性前膝痛（非手术组）

| 疾病 | 人群 / 钩子 | 痛点定位 | 招牌体征 | 治疗 |
|---|---|---|---|---|
| **PFPS 髌股疼痛综合征** | 年轻女、跑/上下楼/久坐(theater sign) | **模糊**、定位不清 | 髌股压迫痛 + **股四头肌等长收缩痛** | **股四头肌 + 髋外展强化；忌制动** ⭐ |
| **髌腱炎 Patellar tendinitis** | 跳跃运动员 | 髌骨**下极/髌腱**、点清楚 | 髌腱局灶压痛 | 相对休息 + 离心训练 |
| **Osgood-Schlatter (OSD)** | 青春期 + 活动多 | **胫骨结节** | 结节压痛 + 肿 + 髌腱牵拉痛 | 限制活动 + 拉伸（自限）|
| **髌前滑囊炎 Prepatellar bursitis** | 长期跪地（园丁/清洁工）| 髌骨**正前方表浅** | 髌前局限肿，被动 ROM 基本正常 | 避免跪 + 对症；红热重排 septic |

> [!warning] PFPS 三大错点（你反复错的地方，默写）
> 1. **体征 ≠ 伸膝障碍**：PFPS 是"**绷股四头肌就痛**"，**能伸膝**；股四头肌**萎缩是废用性**，不是断裂。
> 2. **治疗 = 动，不是制动**：股四头肌强化为主；**Knee immobilizer 膝固定器是反向**（越制动越软）。
> 3. **"打软 giving way" ≠ 韧带不稳**：多种膝病都可有，不是手术/支具指征。

---

## 四、一句话钥匙

> [!success] 三步锁定前膝痛
> 1. **能伸膝吗？** 不能 → 伸膝装置断裂（手术）；能 → 往下。
> 2. **断裂分上下年龄**：老上低（股四头肌腱）/ 轻下高（髌腱）。
> 3. **慢性看痛点**：模糊+压髌=PFPS（练股四头肌）；髌腱局灶=髌腱炎；胫骨结节=OSD；髌前肿包=髌前滑囊炎。

## 五、自测（答完再判分）

> [!question]- Q1. 55 岁男搬重物时突感左膝"撕裂"，此后无法主动伸膝；查体髌骨上方可触及凹陷间隙，X 线示髌骨低位。诊断与处理？
> **股四头肌腱断裂 (quadriceps tendon rupture)**（>40 岁 + 髌上间隙 + 髌骨低位）→ **手术修复**。

> [!question]- Q2. 28 岁男篮球落地后膝前剧痛，不能伸膝；髌骨明显高位、髌骨下方可触及间隙。诊断？
> **髌腱断裂 (patellar tendon rupture)**（<40 岁 + 髌下间隙 + 髌骨高位 patella alta）→ 手术。

> [!question]- Q3. 16 岁女跑步运动员，3 月模糊前膝痛，上下楼+久坐加重，能完全伸膝，髌股压迫试验阳性，有废用性股四头肌轻度萎缩。最佳处理？
> **PFPS** → **股四头肌 + 髋外展肌强化锻炼；忌制动**。萎缩是废用性、非断裂；膝固定器是反向选择。

> [!question]- Q4. 13 岁足球少年膝前下方痛、胫骨结节处压痛肿胀，活动后加重，能伸膝。诊断？
> **Osgood-Schlatter (OSD)** 胫骨结节骨软骨病 → 限制活动 + 拉伸（自限性）。

## 一、第一刀：红旗筛查 → 要不要 MRI ⭐⭐⭐

> [!danger] 腰背痛分流的核心顺序：先排"红旗急症"，再谈机械性
> **无红旗 → 保守 4–6 周（不影像、不卧床）；有红旗 → MRI（部分立即手术）。** 别一上来就拍片。

```
腰背痛 Back pain
│
├─【先扫红旗 Red flags?】──── 有 ──► 急症（见下表）→ MRI
│     ├─ 尿潴留 + 鞍区麻 + 双侧无力 ──► 马尾综合征（立即 MRI+手术）
│     ├─ 癌症史 + 夜间/卧位加重 + 对称无力 ► 脊髓压迫（急诊 MRI+地塞米松）
│     ├─ 发热 + 局部脊柱痛 + IVDU/免疫抑制 ► 硬膜外脓肿
│     └─ 创伤 / 骨质疏松 + 急性痛 ────────► 椎体压缩骨折
│
└─ 无红旗 ──► 按部位 + 神经体征分
      │
      ├─ 腰椎机械病（strain / facet OA / 神经根病 / 椎管狭窄）► §二
      ├─ 颈椎机械病（strain / facet OA / 神经根病 / 脊髓型）─► §三
      └─ 结构性（青少年 / 老年特定）────────────────────► §四
```

### 红旗急症详表（最高优先 ★★★）

| 急症 | 红旗信号 | 处理 |
|---|---|---|
| **马尾综合征 Cauda equina** | **急性尿潴留 + 鞍区 (saddle) 麻木** + 双侧下肢无力 + 直肠张力↓（累及 S2–S4）| **立即 MRI + 紧急手术减压** |
| **脊髓压迫 Cord compression（肿瘤急症）** | 癌症史(肺/乳/前列腺) + 进行性**夜间/卧位加重**背痛 + 对称无力 + Babinski+ | **急诊 MRI + IV 地塞米松 dexamethasone** + 放疗/手术会诊 |
| **硬膜外脓肿 Spinal epidural abscess** | 三联：**发热 + 局部脊柱痛 + 神经缺损**；IVDU / 免疫抑制 / 脊柱操作后 | 急诊 MRI + IV 抗生素 ± 引流 |
| **椎体压缩骨折 Vertebral compression fx** | 骨质疏松老年女 / 创伤、急性背痛、身高下降 | X 线 / MRI；镇痛 + 治骨质疏松 |

> [!danger] 腰背痛红旗（3M 记忆）
> - **Malignancy 恶性**：癌症史 / 体重↓ / 年龄 >50 / **夜间痛、卧位不缓解**
> - **Microbes 感染**：发热 / IVDU / 局部脊柱压痛 → 硬膜外脓肿
> - **Mechanical-Neuro 神经急症**：马尾（尿潴留 + 鞍区麻）/ 急性创伤 / 严重双侧无力
> - 陷阱：单纯"年龄 >50"或"轻微夜间酸痛"**不够**，必须"**痛醒 / 无法平卧**"才算恶性红旗。

## 二、腰椎机械性病（无红旗）⭐⭐⭐

> [!warning] 腰背痛错题核心。四把钥匙：**放射过不过膝？SLR 阴阳？前屈痛还是后伸痛？走路诱发腿症状吗？**

### 2.1 四病鉴别：strain / facet OA / 神经根病 / 椎管狭窄

| 特征 | **机械性腰痛 strain** | **Facet OA 小关节骨关节炎** | **神经根病 radiculopathy** | **椎管狭窄 spinal stenosis** |
|---|---|---|---|---|
| **机制 / 人群** | 搬重物后急性、青壮 | **老年退变** | **椎间盘突出 (disc herniation)** | 退变、**老年** |
| **放射范围** | **不过膝**（臀/大腿后）| 轴性局部、**不过膝** | **过膝至小腿/足** ⭐ | 双下肢、站立行走诱发 |
| **加重 / 缓解** | 活动加重、休息缓解 | **后伸/站立加重，前屈/坐缓解** | **前屈/坐/Valsalva 加重** | **后伸/站立加重，前屈/坐缓解**（跛行）|
| **SLR 直腿抬高** | 阴性 | 阴性 | **阳性**（30–70°放射至膝下）| 多阴性 |
| **神经体征** | 正常 | 正常 | **单皮节**固定缺损 + 肌力↓ + DTR↓ | **双侧诱发性**麻木/无力，**静息常正常**、行走后才显现 ⭐ |
| **影像 / 处理** | **不影像、不卧床**；NSAIDs + 维持活动 | NSAIDs / PT，顽固可关节注射 | 1–2 周 NSAIDs → PT/口服激素 → **4–6 周无效或红旗 → MRI** | MRI；保守为主，重则减压 |

> [!tip] 四把钥匙快记
> - **过不过膝**：不过膝 = strain / facet；过膝至足 = 神经根。
> - **SLR**：阴 = strain / facet；阳（30–70°诱发膝下放射电击痛）= 神经根。
> - **前屈痛 vs 后伸痛**：**前屈/坐痛 → 椎间盘**（disc 前屈时压力最大）；**后伸/站立痛、前屈缓解 → 小关节 / 椎管狭窄**（facet 后伸负重、伸腰缩窄椎管）。
> - **走路诱发腿症状**：**腰 + 双腿、走一段才麻软、坐下/前倾缓解 = 椎管狭窄**（神经源性跛行）。

> [!warning] 椎管狭窄两大易混点（错题高发）
> **① 静息查体正常 ≠ 没有感觉改变。** 椎管狭窄的感觉症状是**诱发性**的：站立/行走（伸腰）时椎管才被挤窄、神经缺血 → **双侧、多节段、弥漫**麻木/无力（pseudoclaudication 假性跛行），前屈/坐下即缓解。stem 给"老人走一段路双腿又麻又软、坐下就好、**查体却正常**"——**正常查体反而是 stenosis 特征**，不是排除依据。对比 radiculopathy 是**单皮节、固定、静息即存在**（如 L5 足背持续麻）。
> **② 椎管狭窄 vs 腰椎 Facet OA：都"前倾缓解"，靠"腿参不参与"分。** 两者都后伸加重/前屈缓解（机制亲戚，且 facet 肥大本身是椎管狭窄三件套之一 + 黄韧带肥厚 + 椎间盘膨出，老年常并存）。分水岭：**只有腰痛、后伸动作就痛 = Facet OA**（轴性、无腿症状）；**腰 + 双腿、走出来的 = 椎管狭窄**。记法：**"光腰 = 小关节；带腿 + 走出来的 = 椎管狭窄"**。
> **③ 狭窄部位**：**中央椎管狭窄 (central stenosis)** → 双侧神经源性跛行（典型）；**侧隐窝/椎间孔狭窄 (lateral recess / foraminal stenosis)** → 压**单根**，长得像 radiculopathy（单皮节，重叠地带）。

![[L60502.jpg|400]]
*神经根病机制 (radiculopathy)：椎间盘突出 (disc herniation) 或骨赘 (osteophyte) 压迫神经根。*

![[L28541.jpg|400]]
*腰椎管狭窄 (lumbar spinal stenosis)：黄韧带肥厚 (ligamentum flavum) + 小关节骨赘 (facet osteophyte) + 椎间盘膨出 → 中央管/椎间孔狭窄（对比正常）。*

### 2.2 腰神经根定位（下肢，高频）

| 神经根 | 关键动作（无力）| 感觉区 | 反射 | 椎间盘 |
|---|---|---|---|---|
| **L4** | 伸膝（股四头肌）、足背伸 | 小腿**内侧**、足内侧 | **膝反射 (patellar)↓** | L3–L4 |
| **L5** | **踇趾背伸 / 足背伸**、足下垂 | 小腿外侧、足背、**踇趾蹼** | 无特异反射 | L4–L5（压 L5）|
| **S1** | **跖屈 / 踮脚**、屈膝 | 足底、小腿后、足外侧 | **踝反射 (Achilles)↓** | L5–S1（压 S1）|

> [!info] 这里是**按神经根**定位；同一足下垂/无力**按神经干**（股/坐骨/腓总/胫）的定位 + L5 根 vs 腓总神经鉴别见 [[完整笔记/专题笔记/神经/神经_下肢神经定位速查]]。

> [!info] 腰椎间盘突出"压下位根"规律（含 far-lateral 例外）
> **旁中央型 (posterolateral, 最常见)** 突出压**下一节段的根（traversing root）**：L4–L5 压 **L5**；L5–S1 压 **S1**（= 下位椎同号根，与颈椎数字规律一致，详见 §3.2）。
> **例外**：**极外侧 / 椎间孔型 (far-lateral / foraminal)** 突出改压 **exiting 上位根**（L4–L5 极外侧 → 压 **L4**）。
> 口诀：**"L4 膝反射 / S1 踝反射"**；**"L5 踇趾背伸（dorsiflexion of big toe）"**。

### 2.3 神经源性 vs 血管源性跛行（站立 vs 行走的陷阱）

| | **神经源性跛行（椎管狭窄）** | **血管源性跛行（PAD）** |
|---|---|---|
| **诱发** | 站立 + 行走（伸腰）| 行走（固定距离）|
| **缓解** | **前倾 / 坐下缓解**，单纯站立不缓解 ⭐ | **休息（站着也行）即缓解** ⭐ |
| **脉搏 / 皮肤** | 正常 | 脉搏减弱、皮肤萎缩 / 无毛 / 溃疡 |
| **上坡 vs 下坡** | 上坡(前倾)反而舒服 | 上坡更痛（耗氧）|

> [!tip] 一刀分跛行
> **"坐下/前倾才舒服 = 神经源（椎管狭窄）"**；**"站着歇会儿就好 + 脉弱 = 血管源（PAD）"**。

## 三、颈椎机械性病 ⭐⭐⭐

> [!info] 与腰椎同框架，但多一个致命分叉
> 颈椎机械病同样按"软组织 → 关节 → 神经根 → 脊髓"分层（与 §二 腰椎平行）。但颈椎最高频、最致命的一刀是：**颈神经根病 (cervical radiculopathy) 还是颈脊髓型 (cervical myelopathy)？** —— 前者压**神经根 = LMN**，后者压**脊髓 = UMN**，处理天差地别。
>
> **为什么腰椎没有"脊髓型"这一叉（解剖底牌）**：**脊髓 (spinal cord) 终止于 L1–L2 椎体水平**（下端 = 脊髓圆锥 conus medullaris），L2 以下椎管里**只剩马尾 (cauda equina，一束 LMN 性质的神经根)**。所以颈/胸椎管里有脊髓 → 压迫可致 **UMN 脊髓型 myelopathy**（反射亢进 / Hoffmann / Babinski / 痉挛步态）；腰椎管里没脊髓 → 压迫**全是 LMN**，单根 = 神经根病、多根大压迫 = **马尾综合征 (cauda equina syndrome)**（尿潴留 + 鞍区麻 + 反射↓，见 §一红旗）。⚠️ 陷阱：腰痛 + 下肢 **Babinski 阳性 / 反射亢进** → 病灶在**脊髓（颈/胸段）或更高**，不是腰椎（如 §三 Q4 前列腺癌胸段转移压脊髓）。

### 3.1 病变分层：strain / facet OA / 神经根病(LMN) / 脊髓型(UMN)

| 病 | 痛 / 症状 | 关键钥匙 | 神经查体 |
|---|---|---|---|
| **肌肉拉伤 strain / 挥鞭伤 (whiplash, 颈挥鞭样损伤)** | 局部、不放射 | 近期颈外伤（**追尾**）/ 姿势不良，活动时痛 / 僵 | 正常 |
| **Facet 骨关节炎 (facet osteoarthritis, 小关节骨关节炎)** | 局部、轴性颈痛 | **老年**，活动加重、休息缓解，晨僵 <30 min | 正常 |
| **颈神经根病 cervical radiculopathy** ⭐ | **放射至肩 / 臂**，沿皮节 (dermatome) | 椎间盘突出 / 骨赘压**单一神经根**；**Spurling 征 +** | **单根 LMN**：感觉↓ / 肌力↓ / **反射↓** |
| **颈脊髓型 cervical myelopathy (CSM)** ⭐ | **可不痛**；手笨拙、步态宽、二便障碍 | 椎管狭窄压**脊髓**；**Lhermitte 征 +** | **UMN**：**反射亢进 ↑**、**Hoffmann +**、Babinski +、痉挛步态 |

> [!danger] 颈神经根病 vs 颈脊髓型 —— Step 2 最高频分叉（决定保守还是手术）
> 看**运动神经元征**这一条就能分：
> - **下运动神经元征（反射↓、单一皮节感觉 / 肌力↓）= 神经根病 radiculopathy** → **保守治疗**（NSAIDs + PT + 颈托），多数 4–6 周自限；**进行性无力 / 6 周无效 → 颈椎 MRI**。
> - **上运动神经元征（反射↑、Hoffmann +、Babinski +、痉挛步态、二便障碍）= 脊髓型 myelopathy** → 这是 **red flag**：**颈椎 MRI + 神经外科会诊**，多需**减压手术**（拖延会不可逆瘫痪）。
> - 陷阱："手越来越笨 + 走路像踩棉花 + 颈前屈电击感"= 脊髓型，**别当颈椎病保守观察**。

### 3.2 颈神经根定位（上肢，与 §2.2 腰椎平行，高频）

| 神经根 | 关键动作（无力）| 感觉区 | 反射 | 责任椎间盘 |
|---|---|---|---|---|
| **C5** | 肩外展（三角肌 deltoid）、屈肘 | 肩 / 上臂**外侧** | **肱二头肌反射 (biceps)↓** | C4–C5 |
| **C6** | 屈肘、伸腕 | 前臂外侧 + **拇指 / 示指** | **肱桡肌 / 二头肌反射↓** | C5–C6（高频）|
| **C7** | **伸肘（三头肌 triceps）**、屈腕 | **中指** | **肱三头肌反射 (triceps)↓** | C6–C7（**最高频**）|
| **C8** | 屈指、手内在肌（握力 / 分指）| **环指 / 小指** | 无特异反射 | C7–T1 |

> [!info] 颈 vs 腰"压哪根"——数字相同，差别在解剖（这才是关键）
> **数字规律两者一致**：旁中央型（posterolateral，最常见）椎间盘突出都压**"下位椎同号的根"（= 上位椎 + 1）**：
> - 颈：**C5–C6 → C6**（拇指/示指麻、肱桡肌反射↓）；**C6–C7 → C7**（中指麻、三头肌反射↓、伸肘无力，最常见）
> - 腰：**L4–L5 → L5**；**L5–S1 → S1**
>
> 所以"颈腰不同"**不是指数字**——真正的解剖差别：
> - **颈椎**：8 对根 / 7 椎，根从同号椎**上方**穿出；旁中央突出压**该平面的 exiting root（正穿出根）**= 下位椎号。
> - **腰椎**：根从同号椎**下方**穿出；旁中央突出**避开**已穿出的上位根，压**traversing root（穿行根）**= 下位椎号。
> - **唯一数字会变的场景**：**腰椎"极外侧 / 椎间孔型 (far-lateral / foraminal)"突出**改压 **exiting 上位根**（L4–L5 极外侧 → 压 **L4**）；颈椎无此翻转。stem 强调 "far-lateral/foraminal" 时腰椎选**上位根**。

### 3.3 颈椎专属体征 + Memory Hook

> [!tip] 三个颈椎专属体征（看动作记意义）
> - **Spurling 征 (Spurling sign)**：头**后伸 + 向患侧侧屈 + 轴向下压** → 诱发**同侧上肢沿皮节放射痛** = 颈神经根受压（缩小椎间孔）。
> - **Lhermitte 征 (Lhermitte sign)**：颈**前屈** → 沿脊柱向下放射的**电击感** = 颈脊髓后柱病变（颈脊髓型 / 多发性硬化 MS / 维生素 B12 缺乏 亚急性联合变性都可阳性）。
> - **Hoffmann 征 (Hoffmann sign)**：弹中指末节 → 拇指 / 示指反射性屈曲 = 颈脊髓 UMN 征（上肢版 Babinski）。

> [!tip] Memory Hook — 颈痛一眼分层
> **"局部 = strain / facet；放射到手 + 反射↓ = 神经根（保守）；手笨 + 步态 + 反射↑ + Lhermitte = 脊髓（MRI + 手术）"**。
> 反射方向就是分水岭：**↓ 在根、↑ 在髓**。

### 3.4 别漏：寰枢椎不稳 (atlantoaxial subluxation, AAS) —— 术前 / 插管红旗 ⭐

> [!danger] RA / 唐氏 + 颈痛手麻 + 要插管 → 先拍张口位颈椎片（源自 [[完整笔记/Peixuan分科笔记/骨科]]）
> 不是退变压迫，而是 **C1（寰椎 atlas）相对 C2（枢椎 axis）半脱位** → **颈后伸时（尤其插管 / 全麻摆位）压迫脊髓**，可致急性四肢瘫。
> - **高危人群**：**长期 RA（rheumatoid arthritis, 类风湿关节炎）**（齿突周围滑膜炎 / 韧带破坏）、**唐氏综合征 (Down syndrome)**（韧带松弛）。
> - **经典 stem**：多年活动性 RA 患者**术前**诉**长期颈痛 + 手麻** → 最该警惕的**不是手指关节，而是颈椎不稳**。
> - **评估**：**张口位侧位颈椎 X 线 (lateral cervical spine X-ray, open-mouth / odontoid view)**，量寰齿间距；阳性 → 麻醉团队改用纤支镜清醒插管、避免颈过伸。

## 四、结构性脊柱病（青少年 / 老年特定）

| 疾病 | 人群 / 钩子 | 体征 | 影像 / 处理 |
|---|---|---|---|
| **Scheuermann 后凸 kyphosis** | **少年**、**僵硬性**胸椎后凸 + 痛、仰卧不能矫正 | 固定后凸、前屈更明显 | 胸椎侧位 X 线（≥3 个相邻椎体楔变 >5°）；PT / 支具 |
| **青少年特发性脊柱侧弯 (adolescent idiopathic scoliosis, AIS)** | **年龄 ≥10、生长期**、**多无症状**、学校筛查/体检发现 | **前屈试验 (Adam test) → 不对称肋/腰隆起 (rib hump)**、肩/髂嵴不平 | **首步脊柱正侧位 X 线**：量 **Cobb 角 >10°** + 评骨龄；按度数 观察/支具/手术（见下方 callout）|
| **脊椎滑脱 Spondylolisthesis** | **青少年 + 反复后伸**（体操/橄榄球）、后伸痛 | 腰部 **step-off 台阶感**、腘绳肌紧 | X 线侧位（峡部裂 spondylolysis → 滑脱）；保守，重则融合 |
| **椎体压缩骨折** | 老年骨质疏松 / 创伤（见 §一）| 急性背痛、身高下降、后凸 | X 线 / MRI + DEXA；镇痛 + 抗骨质疏松 |
| **强直性脊柱炎 AS（跨风湿）** | **年轻男、炎性背痛**：**晨僵 >30 min、活动改善休息加重、夜间痛** | 骶髂压痛、脊柱活动度↓、扩胸度↓ | 骶髂 MRI/X 线；详见 [[完整笔记/Peixuan分科笔记/风湿]] |

> [!tip] 炎性 vs 机械性背痛（AS 高频对照）
> **炎性（AS）= 年轻 + 晨僵 >30 min + 活动后改善 + 夜间痛 + 休息不缓解**；机械性正相反（活动加重、休息缓解）。看到"年轻男 + 越动越松 + 夜里痛醒"想强直性脊柱炎，别当普通腰肌劳损。

> [!warning] AIS 脊柱侧弯 workup 决策（[[mistakes/uworld-mistakes_2026-06#^Q16389]] 无症状首步 / [[mistakes/uworld-mistakes_2026-06#^Q16593]] 红旗）
> - **临床发现侧弯（前屈隆起）→ 第一步永远是脊柱正侧位 X 线**：确诊 + 量 **Cobb 角** + 评骨骼成熟度 (Risser/初潮)。
> - **别跳过 X 线直接"观察 6 月"**（Q16389 陷阱）——先量化严重度才能定 观察 vs 支具 vs 手术。
> - 按 Cobb 角：**10–25° 观察（~6 月随访）/ ~25–40°+ 有生长潜力 → 胸腰骶支具 / ≥40–50° → 手术固定**。
> - **MRI 仅在红旗**（Q16593：背痛/神经症状/快进展 ≥10°·年⁻¹/椎体异常）→ 排查脊髓肿瘤等病理性病因。
>

## 五、体征 / 检查 → 诊断速查（脊柱全项）

| 体征 / 试验 | 阳性 = |
|---|---|
| **SLR 直腿抬高**（30–70°膝下放射）| L4–S1 神经根 / 椎间盘突出 |
| **Crossed SLR（对侧抬高诱发患侧）** | 神经根受压**特异性更高** |
| **FNST 股神经牵拉** | L2–L4 上腰段神经根 |
| **膝反射↓ / 踝反射↓** | L4 / S1 神经根 |
| **前倾/坐位缓解的下肢痛** | 椎管狭窄（神经源性跛行）|
| **腰部 step-off 台阶感** | 脊椎滑脱 |
| **Spurling 征**（头后伸+侧屈+轴压诱发同侧臂痛）| 颈神经根受压 (cervical radiculopathy) |
| **Lhermitte 征**（颈前屈→沿脊柱向下电击感）| 颈脊髓后柱（颈脊髓型 myelopathy / MS / B12 缺乏）|
| **Hoffmann 征 / 上肢反射亢进 / Babinski+** | 颈脊髓型 myelopathy（UMN，需 MRI+神经外科）|
| **二头肌反射↓ / 三头肌反射↓** | C5–C6 / C7 神经根 |
| **鞍区麻 + 尿潴留 + 直肠张力↓** | 马尾综合征（立即 MRI+手术）|
| **夜间/卧位加重 + 癌症史 + Babinski** | 脊髓压迫（肿瘤急症）|
| **发热 + 局部脊柱痛 + IVDU** | 硬膜外脓肿 |

> [!tip] 体征手法图解（看动作记体征）
> ```
> SLR（直腿抬高）—— 神经根 / 椎间盘
>   仰卧，被动直腿抬高
>        ╱ 抬到 30–70°
>   阳性 = 诱发【膝以下】电击样放射痛 → L4-S1 根受压
>   注：只在 30–70° 阳性才特异；过高是腘绳肌紧
>
> Crossed SLR —— 特异性更高
>   抬【健侧】腿却诱发【患侧】放射痛 = 强提示椎间盘突出
>
> FNST（股神经牵拉）—— 上腰段 L2-L4
>   俯卧，屈膝并后伸髋
>   阳性 = 大腿【前】放射痛 → 高位腰椎根
>
> Spurling（颈）—— 颈神经根
>   头后伸 + 向患侧侧屈 + 轴向下压
>   阳性 = 诱发【同侧上肢沿皮节】放射痛 → 颈根受压
>
> 神经源 vs 血管源跛行 —— 姿势试验
>   让患者前倾推购物车 / 坐下：
>      症状缓解 = 神经源(椎管狭窄)
>      需完全停下站着歇 + 脉弱 = 血管源(PAD)
>
> 红旗筛查（每个腰背痛必问）
>   尿潴留/鞍区麻? 发热/IVDU? 癌症史+夜间痛醒?
>      任一阳性 → MRI（马尾立即手术）
> ```

## 六、自测（答完再判分）

> [!question]- Q1. 45 岁男搬重物后急性腰痛，向臀部及大腿后放射但不过膝，SLR 阴性，神经查体正常，无红旗。最佳处理？
> **机械性腰痛 (lumbosacral strain)** → **NSAIDs + 维持活动，不影像、不卧床**；多 4–6 周自限。

> [!question]- Q2. 38 岁男腰痛向右下肢放射至足背及踇趾，SLR 阳性，右踇趾背伸无力。最可能受累神经根与责任椎间盘？
> **L5 神经根**（踇趾背伸 + 足背感觉）→ **L4–L5 椎间盘突出**（压下位根）。

> [!question]- Q3. 70 岁男双下肢行走后酸痛无力，**前倾或坐下缓解**，下肢脉搏正常。诊断？如何与 PAD 区分？
> **椎管狭窄（神经源性跛行 pseudoclaudication）** → 前倾/坐位缓解 + 脉搏正常；PAD 是"休息即缓解 + 脉弱"。

> [!question]- Q4. 前列腺癌病史男性进行性背痛、夜间加重无法平卧、双下肢对称无力、Babinski 阳性。下一步？
> **脊髓压迫（肿瘤急症）** → **急诊脊柱 MRI + IV 地塞米松 dexamethasone** + 放疗/手术会诊。

> [!question]- Q5. 32 岁静脉吸毒男，发热 + 局部胸腰段脊柱剧痛 + 进行性下肢无力。最可能诊断与下一步？
> **硬膜外脓肿 (spinal epidural abscess)**（发热+脊柱痛+神经缺损三联）→ **急诊 MRI + IV 抗生素 ± 引流**。

> [!question]- Q6. 42 岁女颈痛向右上肢放射至中指，伸肘无力，右肱三头肌反射减弱，Spurling 征阳性。最可能受累神经根、责任椎间盘及处理？
> **C7 神经根**（中指 + 伸肘 + 三头肌反射↓）→ **C6–C7 椎间盘突出**（颈椎压同号下位椎的根）；下运动神经元征 = **颈神经根病 → 保守**（NSAIDs + PT），4–6 周无效或进行性无力再 MRI。

> [!question]- Q7. 58 岁男进行性双手笨拙（扣纽扣困难）+ 步态不稳，颈前屈时脊柱向下电击感，查体下肢反射亢进、Hoffmann 阳性、Babinski 阳性。诊断与下一步？
> **颈脊髓型 (cervical spondylotic myelopathy, CSM)** → UMN 征（反射↑/Hoffmann/Babinski/痉挛步态）+ Lhermitte = 脊髓受压；**颈椎 MRI + 神经外科会诊**（多需减压手术）。对比 Q6 颈神经根病是单根 LMN（反射↓）保守治疗。

# 儿童骨折分型 Pediatric Fracture Patterns

> [!info] 这篇笔记解决什么
> 儿童骨骼**未成熟**（含生长板 physis、骨膜厚而坚韧、骨质多孔可塑），骨折形态与成人完全不同：成人会断裂的地方，儿童常常是**弯曲、压扁、不全裂**。本笔记覆盖 ① 累及生长板的 **Salter-Harris (SH) 分型**；② 不全骨折家族（greenstick 青枝 / buckle-torus 隆突扣环 / plastic 塑性变形）；③ 牵拉肘 nursemaid elbow（不是骨折，是半脱位，**复位优先**）；④ 何时拍片 vs 直接处理；⑤ 何时该想"非意外创伤 NAI"。
>
> 派生自 NBME11A Q039（牵拉肘复位优先 vs 过度影像）。

---

## 一、为什么儿童骨折不一样（机制基础）

> [!tip] 三大儿童骨特性 → 三大独特骨折形态
> | 儿童骨特性 | 后果 | 对应骨折类型 |
> |---|---|---|
> | **生长板 physis（骺板）存在且力学薄弱** | 暴力优先经过软骨性骺板 | **Salter-Harris 骨骺骨折** |
> | **骨膜 periosteum 厚、坚韧、血供好** | 一侧裂另一侧靠骨膜连着不断 | **青枝骨折 greenstick** |
> | **骨皮质多孔、有弹性** | 受压只压扁不裂开 / 只弯不断 | **隆突骨折 buckle/torus、塑性变形 plastic** |

生长板 (physis / growth plate, 骺板) 是软骨，强度低于周围韧带与骨。所以在成人会**扭伤韧带**的暴力，在儿童常常是**骺板骨折**——这也是为什么"儿童扭伤"要警惕隐匿 SH 骨折。

## 二、Salter-Harris (SH, 骨骺骨折) 分型 I–V

> [!info] 解剖坐标先记牢
> 一根长骨从骨端往骨干数：**Epiphysis 骨骺**（关节侧）→ **Physis 生长板/骺板**（软骨盘）→ **Metaphysis 干骺端** → **Diaphysis 骨干**。SH 分型说的就是骨折线**相对 physis 走哪里**。

### 助记 SALTR（I→V）

```
SH 型   助记字母   骨折线相对 physis 走向            英文              中文
─────────────────────────────────────────────────────────────────────────
 I       S = Slipped/Separated   只穿 physis（骺板分离），不带骨        Straight through    单纯骺板滑脱
 II      A = Above               穿 physis + 斜向上带一块 metaphysis     Above physis        骺板 + 干骺端（最常见）
 III     L = Lower               穿 physis + 斜向下带一块 epiphysis      Lower / beLow       骺板 + 骨骺（入关节面）
 IV      T = Through/Together     纵向贯穿 epiphysis + physis + metaphysis  Through all three   三层全穿（跨关节面）
 V       R = Rammed/cRush        physis 被轴向压缩挤碎                   cRush / eRasure     骺板压缩伤
─────────────────────────────────────────────────────────────────────────
口诀：S-A-L-T-R = Slip / Above / Lower / Through / Rammed
另一套：SALTeR → I 同 physis, II Above, III beLow, IV Through, V cRush
```

> [!tip] 一句话定位每一型
> - **I 型**：骨折线 **=physis**（就在骺板里），常 X 线**看不到移位**（要靠压痛点 + 临床）。
> - **II 型**：physis **+ 上方干骺端三角块**（Thurston-Holland 征）——**最常见（~75%）**，预后好。
> - **III 型**：physis **+ 下方骨骺**，**进关节面** → 需精确复位。
> - **IV 型**：**骨骺 + 骺板 + 干骺端纵向一刀切**，跨关节面 → 常需手术内固定。
> - **V 型**：**压缩碾压**骺板，初诊 X 线常**正常**（隐匿），靠机制 + 后续随访发现。

### 各型机制 / 影像 / 预后对照表

| SH 型 | 累及结构 | X 线表现 | 生长障碍风险 | 典型处理 |
|---|---|---|---|---|
| **I** | 仅 physis | 常正常或仅轻度增宽/移位 | 低 | 多保守，闭合复位 + 制动 |
| **II** | physis + 干骺端 | 干骺端三角骨块 (Thurston-Holland) | 低 | 闭合复位 + 石膏（最常见、预后最好） |
| **III** | physis + 骨骺（入关节） | 骨骺骨折线达关节面 | 中 | 常需解剖复位（± ORIF 切开内固定） |
| **IV** | 骨骺 + physis + 干骺端 | 纵向贯穿三层 | **高** | 多需手术 ORIF（关节面 + 骺板对位） |
| **V** | physis 压缩 | 初诊常**正常**（隐匿） | **最高** | 制动 + 密切随访（易漏诊→生长停滞） |

> [!warning] 高 SH 型 = 高生长障碍风险
> 累及 physis 都可能 **growth arrest 生长停滞 → limb-length discrepancy（肢体长度不等）或 angular deformity（成角畸形）**。**III / IV 型还累及关节面**，后遗症更重；**V 型最阴险**——X 线初诊正常却预后最差。型号越高（III–V），越往"需要手术 + 长期随访骨科门诊"走。
> （分科笔记原文："Salter-Harris 一旦累及 physis，要先想到 growth arrest → limb-length discrepancy / angular deformity；尤其 type III 和 IV 累及关节面，后遗症更值得担心。"）

## 三、儿童不全骨折家族（成人不会有的形态）

这三种都是**骨皮质/骨膜未完全断裂**，是儿童因骨弹性大、骨膜韧产生的"半截骨折"。

### 3.1 青枝骨折 Greenstick fracture（青枝/嫩枝骨折）

- **机制**：弯曲暴力。像折一根**未干的青树枝**——一侧皮质 (cortex) 断裂张开，对侧皮质只弯不断（骨膜保持完整）。
- **影像**：**凸侧皮质中断 + 凹侧皮质连续/弯曲**。
- **部位**：多见于**前臂骨干**（桡尺骨）。
- **处理**：复位 + 石膏；成角明显时需手法矫正（有时要"折断"对侧皮质以稳定复位）。

### 3.2 隆突骨折 / 扣环骨折 Buckle (Torus) fracture（torus = 隆突/环形隆起）

- **机制**：**轴向压缩**（如 FOOSH 跌倒手撑地）。骨皮质被压得向外**鼓起/起皱**（buckle）而非裂开。
- **影像**：干骺端皮质局部**膨出/隆起一圈**（torus 拉丁语意"隆起"），**无明确骨折线**。
- **部位**：典型 **远端桡骨干骺端**（distal radius metaphysis）。
- **处理**：**最稳定的儿童骨折之一**——多数仅需短期夹板/腕托制动 3–4 周，预后极好。
- **记忆**：Buckle = "腰带扣 buckle"被压凸起；**压缩 → 鼓包 → 最良性**。

### 3.3 塑性变形 Plastic (bowing) deformation（塑性弯曲变形）

- **机制**：缓慢持续的弯曲力，骨被**永久弯曲**但**无肉眼骨折线**（微观多发微裂未贯通）。
- **影像**：长骨**整体弓形弯曲**，无皮质断裂。
- **部位**：常见尺骨、腓骨（常伴邻骨真骨折，如尺骨塑性变形 + 桡骨头脱位 = Monteggia 变异）。
- **处理**：明显畸形时需手法矫正复位；轻者塑形改建 (remodeling)。

> [!tip] 三种不全骨折一句话区分
> - **弯一侧断一侧 = greenstick 青枝**（bending）
> - **压扁鼓包不裂 = buckle/torus 隆突**（compression，最良性）
> - **整根弯但无断线 = plastic 塑性变形**（bowing）

## 四、牵拉肘 Nursemaid's Elbow（桡骨头半脱位——不是骨折！）⭐

> [!danger] 核心：牵拉肘是**临床诊断**，**直接复位、典型不拍片**
> **牵拉肘 = 桡骨头半脱位 (radial head subluxation)**：**环状韧带 (annular ligament, 环状韧带)** 滑过桡骨头并嵌顿于肱桡关节。它**不是骨折**，是软组织半脱位。

| 维度 | 要点 |
|---|---|
| **年龄** | **1–4 岁**幼儿（环状韧带松弛） |
| **机制** | 前臂在**伸肘位被轴向牵拉**（被拽手、被拎起、荡手臂）；**牵拉史常缺失**，家长常说"没受伤" |
| **典型姿势** | **肘屈/伸位 + 前臂旋前 (pronated)**，拒动患臂 = **假性瘫痪 pseudoparalysis**（题眼） |
| **查体** | **无肿胀、无瘀斑、无畸形**；脉搏 + 感觉正常；± 桡骨头轻压痛 |
| **处理** | **手法复位**：① **过度旋前 hyperpronation** 或 ② **旋后 + 屈肘 supination + flexion** → **数分钟内**恢复使用患臂；**复位后无需影像、无需固定** |

> [!warning] 何时才该拍片（牵拉肘的例外）
> 典型牵拉肘**不拍片**。出现下列任一 → 先 **X 线 (X-ray)** 排骨折：
> - **复位失败**（手法后仍不用臂）
> - **表现不典型**：有肿胀 / 瘀斑 / 畸形 / 明确外伤史 / 局部明显压痛
> - 怀疑**髁上骨折 (supracondylar fracture)** 等其他损伤

### 牵拉肘 "复位优先 vs 拍片" 决策表（Q039 核心）

| 表现 | 下一步 |
|---|---|
| 典型：幼儿 + 旋前拒动 + 无肿胀/畸形 + 神经血管完整 | **直接复位**（不拍片） |
| 复位**失败** / 表现不典型 | **X 线**找隐匿骨折 |
| 明显外伤 / 肿胀 / 压痛 / 畸形 | 先 **X 线**评估骨折 |
| 发热 + 红肿热 + 拒负重 | 排**化脓性关节炎 (septic arthritis, 化脓性关节炎)** → 关节穿刺 |

## 五、何时想"非意外创伤 NAI / 虐待"红旗 ★

> [!danger] NAI = Non-Accidental Injury（非意外创伤 / 儿童虐待 child abuse；又称 NAT non-accidental trauma）
> "家长不记得外伤"在**牵拉肘是常态**，**不等于虐待**。真正的虐待红旗是另一组信号。**Skeletal survey（全身骨骼摄片）只在怀疑 NAI 时做**——典型牵拉肘加 skeletal survey = **过度检查**（Q039 误选项）。

NAI 红旗清单（出现 → 升级评估，考虑 skeletal survey + 上报）：

```
病史层面
 ├─ 病史与伤情不符（机制解释不了损伤）
 ├─ 病史前后矛盾 / 反复变化
 ├─ 就诊延迟（伤后很久才来）
 └─ 不符合发育阶段的损伤（如"还不会走的婴儿"自己摔出骨折）

损伤层面
 ├─ 多处不同愈合阶段的骨折（multiple fractures, different ages）★最经典
 ├─ 高特异性骨折：
 │    后肋骨折 posterior rib fx、干骺端角/桶柄样骨折
 │    (metaphyseal corner / bucket-handle fx)、肩胛骨/胸骨/棘突骨折
 ├─ 婴儿（尤其 <1 岁、不能走动）的长骨螺旋骨折
 └─ 同时有不明瘀斑/烫伤/口腔损伤等

→ 处理：skeletal survey（<2 岁）+ 眼底（视网膜出血→虐待性头外伤）
        + 强制上报 CPS（child protective services, 儿童保护机构）
        （医生为强制报告人 mandated reporter）
```

> [!tip] 对比：牵拉肘的"无外伤史" vs 虐待的红旗
> - 牵拉肘：**单一**、机制经典（拽手）、查体干净、复位即愈 → **正常临床现象，别跳虐待**。
> - 虐待：**多发 / 不同愈合期 / 高特异部位 / 病史不符 / 不符发育阶段** → 才是红旗。

## 六、何时影像 vs 何时直接处理（整合决策树）

```
儿童上肢/下肢痛、拒动患肢
│
├─ 典型牵拉肘？（1–4 岁 + 旋前拒动 + 无肿胀/畸形 + 神经血管完整 + 拽手机制）
│     └─ 是 → 直接手法复位（不拍片）── 复位失败/不典型 → 拍 X 线
│
├─ 有肿胀 / 瘀斑 / 畸形 / 明确外伤 / 局部压痛？
│     └─ 是 → X 线（疑髁上→看 fat pad sign；疑骺板→看 physis 增宽；
│              疑 SH I/V 隐匿→压痛点定位 + 随访复查）
│
├─ 发热 + 关节红肿热 + 拒负重？
│     └─ 是 → 关节穿刺排化脓性关节炎（septic arthritis）
│
└─ 多发骨折 / 不同愈合期 / 高特异部位 / 病史不符 / 不符发育阶段？
      └─ 是 → 怀疑 NAI → skeletal survey + 眼底 + 上报 CPS
```

> [!warning] 三个"X 线看着正常别放心"的隐匿骨折
> - **SH I 型**：X 线可完全正常（骺板里）→ 靠骺板上压痛点临床诊断。
> - **SH V 型**：压缩伤 X 线初诊正常 → 靠机制 + 随访。
> - **髁上隐匿骨折**：靠 **anterior + posterior fat pad sign 脂肪垫征**（关节积血间接征）。

## 七、复习自测

> [!question]- Q1. 3 岁女孩被家长拽起手臂后不用左臂，肘伸前臂旋前拒动，无肿胀畸形，桡动脉搏动 + 感觉正常。最佳下一步？
> **手法复位**（hyperpronation 或 supination+flexion），**不先拍片**。这是典型牵拉肘（桡骨头半脱位）= 临床诊断。仅当复位失败或表现不典型才 X 线。加 skeletal survey = 过度检查（无虐待红旗）。

> [!question]- Q2. 学步儿前臂 X 线示干骺端骨折线斜穿生长板并带一块上方三角形干骺端骨块。SH 几型？预后如何？
> **SH II 型**（physis + 上方干骺端，Thurston-Holland 三角块）。**最常见（~75%）**，闭合复位 + 石膏，**预后好、生长障碍风险低**。助记 A = Above。

> [!question]- Q3. 哪一型 Salter-Harris 初诊 X 线常正常却预后最差？为什么？
> **SH V 型**（physis 轴向压缩碾压伤）。X 线初诊常无移位 → 易漏诊，但骺板被压坏 → **生长停滞风险最高**（limb-length discrepancy / angular deformity）。

> [!question]- Q4. FOOSH 跌倒后远端桡骨干骺端皮质局部向外鼓起、无明确骨折线。诊断 + 处理 + 这是最稳定还是最不稳定的儿童骨折？
> **Buckle / torus fracture 隆突（扣环）骨折**——轴向压缩致皮质鼓包。**最稳定的儿童骨折之一**，短期夹板/腕托 3–4 周，预后极好。

> [!question]- Q5. 1 岁、还不会走路的婴儿被发现股骨螺旋骨折 + 另有一处已部分愈合的肋骨后段骨折。最关键的下一步思路是什么？
> 警惕 **非意外创伤 NAI（虐待）**：多发骨折 + 不同愈合阶段 + 高特异部位（后肋骨折）+ **损伤不符发育阶段**（不会走却长骨骨折）。下一步 **skeletal survey + 眼底检查 + 强制上报 CPS**。

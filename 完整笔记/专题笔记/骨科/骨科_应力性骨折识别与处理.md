# 应力性骨折识别与处理（Stress Fracture）

> [!info] 为什么建这张
> 派生自 NBME11A Q120（久坐新兵突增训练 + 局灶跖骨压痛 + X 线阴性 → 仍按 stress fracture 制动；误选足弓垫）。CK 高频陷阱：**X 线阴性不能排除应力性骨折**，且要区分**低危 vs 高危部位**决定处理。

## 一、识别：典型图景

> [!tip] 应力性骨折 (stress fracture, 应力性骨折) = 重复性微损伤累积
> **不是一次急性外伤**，而是骨在反复负荷下的**微骨折 (microfracture) 累积**。

| 线索 | 含义 |
|---|---|
| 近期**突增活动**（新兵训练、突然开始跑步） | 典型 overuse（过度使用）触发点 |
| **训练不足 / 久坐基线** | 骨适应差，风险更高 |
| 随运动强度**渐进性局灶骨痛** | 不是突然扭伤的关节痛 |
| **局灶性骨压痛 (focal bony tenderness)** | 压痛点在骨上（vs 韧带/关节线）|
| **早期 X 线常阴性** | 微骨折 2–3 周后才显骨痂/透亮线 |

> [!danger] X 线阴性 ≠ 排除
> 早期应力性骨折 X 线可完全正常。临床图景典型 → **先按应力性骨折处理**；需影像确诊时最敏感的是 **MRI (magnetic resonance imaging, 磁共振)**（骨髓水肿）；骨扫描 (bone scan) 也敏感但特异性低。

```text
未训练个体突然重复负荷
  → 骨小梁微骨折累积
  → 局灶骨痛 + 骨压痛，随负荷加重
  → 早期 X 线可正常
  → 临床典型即按 stress fracture 处理（继续负重会进展为完全骨折）
```

## 二、Q120 主线：处理 = 低危休息制动 vs 高危手术

> [!danger] 低危 vs 高危部位分流（决定保守还是手术）
> - **低危 (low-risk)**：第 2–4 跖骨干、跟骨 (calcaneus)、胫骨后内侧、腓骨等 → **相对休息 + 制动/减负重 (rest + immobilization / protected weight-bearing)**，避免诱发活动，多可愈合。**Q120 第 5 跖骨局灶压痛按此处理 → cast immobilization（石膏制动）。**
> - **高危 (high-risk)**：**股骨颈张力侧 (femoral neck, tension side)**、髌骨、胫骨前皮质、第 5 跖骨基底 (Jones)、距骨、舟骨 (navicular)、大踇趾籽骨 → 易不愈合/移位/坏死 → **骨科转诊，常需手术内固定 (internal fixation)**。

> [!warning] 别选足弓垫/对症
> 局灶骨压痛 + 典型应力史 = 骨的问题，不是**力线/足弓 (arch support 足弓垫)** 或痛风等软组织/代谢问题。核心处理是**减负荷让骨愈合**，继续训练会让微骨折进展成完全骨折。

## 三、易混陷阱（普适）

> [!warning]
> - **X 线正常不能放病人回去继续跑** —— 这是最大陷阱。
> - **足弓垫 / 矫形鞋垫**治的是力线相关软组织痛，不是应力性骨折。
> - **股骨颈应力骨折**（腹股沟/大腿痛、负重痛）绝不能漏 —— 张力侧需**手术**，漏诊可致股骨头坏死 (avascular necrosis, AVN, 缺血性坏死)。
> - 反复应力性骨折 + 女性 → 想**女运动员三联征 (female athlete triad)**：能量不足 / 月经紊乱 / 骨密度低。

> [!question]- Q1. 新兵训练 10 天后第 5 跖骨背侧局灶压痛，X 线阴性。最合适的处理？
> 按**应力性骨折 (stress fracture)** 处理 → **制动 + 休息（如石膏 cast immobilization / 减负重）**。X 线阴性不排除；继续训练会进展为完全骨折。非足弓垫。

> [!question]- Q2. 哪些是"高危"应力性骨折部位，需骨科转诊/手术？
> **股骨颈张力侧、髌骨、胫骨前皮质、第 5 跖骨基底 (Jones)、距骨、足舟骨、大踇趾籽骨**。这些易不愈合/移位/坏死 → 内固定 (internal fixation)；低危部位（多数跖骨/跟骨/胫骨后内侧）保守制动即可。

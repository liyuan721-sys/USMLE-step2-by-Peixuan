# FOOSH 损伤分流 (Fall On OutStretched Hand)

> [!info] 这张笔记干什么
> **FOOSH (fall on outstretched hand, 伸手撑地跌倒)** 是 UWorld 上肢损伤题最常见的机制开头。同一个"摔了手撑地"，**按年龄 + 痛在哪**会分流到完全不同的骨折/脱位。这张是**分流导航卡**——快速锁定诊断；每条的"骨折→神经/并发症"细节链到 [[完整笔记/专题笔记/骨科/骨科_上肢骨折神经损伤定位]]，不在此重复。

---

## 一、分流树（年龄 + 部位）

```
FOOSH 伸手撑地跌倒
│
├─【儿童 child】
│   ├─ 肘痛/畸形 ───────────► 肱骨髁上骨折 Supracondylar
│   │                          → 正中/AIN + ⚠肱动脉；查脂肪垫征
│   ├─ 肘外侧痛(罕见) ───────► 肱骨外侧髁骨折 Lateral condyle
│   └─ 腕痛、无明显畸形 ─────► 桡骨远端 buckle(隆突)/greenstick(青枝)骨折
│
├─【年轻成人 young adult】
│   ├─ 鼻烟窝(snuffbox)压痛 ─► 舟骨骨折 Scaphoid ⭐（X线常阴→石膏，防 AVN）
│   ├─ 肘外侧痛 + 不能完全伸肘► 桡骨头骨折 Radial head（成人；后脂肪垫/sail 征）
│   └─ 肩痛 + 手臂外展外旋位伤► 前肩脱位 Ant. dislocation（→腋神经）
│
└─【老年 elderly / 骨质疏松】
    ├─ 腕痛 + 餐叉样畸形 ─────► Colles 骨折（桡骨远端背移→可急性正中神经）
    ├─ 腕痛 + 铲样畸形 ───────► Smith 骨折（桡骨远端掌移；屈腕位摔）
    ├─ 肩痛、外展无力 ────────► 肱骨近端/外科颈骨折（→腋神经）
    └─ 锁骨痛 ────────────────► 锁骨骨折（中段最常见）
```

## 二、分流速查表（一眼定位）

| 年龄 | 关键定位钥匙 | 诊断 | 必做 / 陷阱 |
|---|---|---|---|
| **儿童** | 肘痛畸形 + FOOSH | **肱骨髁上骨折** (supracondylar) | ⚠️ 查**桡动脉搏动**（肱动脉/Volkmann）+ 不能做 OK 手势（AIN, anterior interosseous nerve 骨间前神经）；脂肪垫征提示隐匿骨折 |
| **儿童** | 腕痛、X 线"皮质皱褶" | **buckle (torus) / greenstick 青枝骨折** | 儿童骨软 → 不完全骨折；多石膏保守 |
| **年轻成人** | **鼻烟窝压痛** | **舟骨骨折** (scaphoid) ⭐ | **早期 X 线常阴** → 拇人字石膏 + 1–2 周复查/MRI；逆行血供 → 防 **AVN (avascular necrosis 缺血性骨坏死)** |
| **成人** | 肘外侧痛 + 伸肘受限 | **桡骨头骨折** (radial head) | 成人肘 FOOSH 最常见；查后**脂肪垫征 / sail sign** |
| **老人** | **餐叉样畸形 dinner fork** | **Colles 骨折**（桡骨远端**背移**）| 可急性**正中神经**（腕管）；X 线即见 |
| **老人** | **铲样畸形 garden spade** | **Smith 骨折**（桡骨远端**掌移**）| Colles 反向；屈腕/手背着地 |
| **任意** | 肩外展无力 + 肩外侧麻 | **前肩脱位 / 外科颈骨折** | → **腋神经**（vs 肩袖：看感觉是否完整）|
| **任意** | 锁骨区痛 | **锁骨中段骨折** | 多保守(sling)；移位明显查锁骨下血管 |

## 三、三条最高频分流钥匙

> [!tip] 1. 看年龄定"层高"
> 同样摔手撑地，**力从远往近传，断在哪取决于年龄**：
> - **儿童** → 骨骺/干骺最弱 → 肘（髁上）或腕（青枝/buckle）。
> - **年轻成人** → 骨强、韧带强 → 应力集中**舟骨**。
> - **老人**（骨质疏松）→ **桡骨远端（Colles）** 或肱骨近端。

> [!danger] 2. 鼻烟窝压痛 = 舟骨，X 线阴也治
> 年轻人 FOOSH + **解剖鼻烟窝 (anatomic snuffbox) 压痛** → **按舟骨骨折处理（拇人字石膏 thumb spica）即使首片正常**。舟骨血供**逆行（远→近）**，漏诊 → 近端 **AVN 缺血坏死 + 不愈合**。"片子正常 ≠ 没骨折"。

> [!warning] 3. 儿童髁上骨折先查血管神经
> 儿童 FOOSH + 肘 → 警惕**肱骨髁上骨折**：同时威胁**肱动脉 (brachial artery)**（→ Volkmann 缺血挛缩）和**正中神经/AIN**（不能做 OK 手势）。X 线见**后脂肪垫征 (posterior fat pad sign)** → 即使骨折线不清也按骨折处理。

## 四、易混陷阱

> [!question]- Colles vs Smith：怎么记方向？
> 都是桡骨远端、都压正中神经，区别在**移位方向 + 畸形**：
> - **Colles** = 背侧 (dorsal) 移位 = **餐叉样 dinner fork**（最常见，伸腕摔）。
> - **Smith** = 掌侧 (volar) 移位 = **铲样 garden spade**（屈腕/手背着地）。

> [!question]- 成人 vs 儿童肘部 FOOSH，骨折点不同？
> - **儿童** → **肱骨髁上骨折**（远端最弱）。
> - **成人** → **桡骨头骨折**（成人肘 FOOSH 最常见）。
> 两者都可见**脂肪垫征**提示关节积血。

> [!question]- "鼻烟窝痛" vs "餐叉畸形" 各指向谁？
> - 鼻烟窝压痛 → **舟骨**（年轻人，X 线阴也治）。
> - 餐叉样畸形 → **Colles**（老人，X 线即见，可急性腕管）。

## 五、复习自测（折叠，先别看答案）

> [!question]- Q1. 22 岁男性篮球时摔倒手撑地，腕桡侧痛，鼻烟窝压痛，腕 X 线未见骨折。最佳处理？
> 按**舟骨骨折**处理：**拇人字石膏固定 + 1–2 周复查 X 线/MRI**（早期 X 线可阴性；漏诊 → AVN / 不愈合）。

> [!question]- Q2. 6 岁儿童 FOOSH 后肘部肿胀畸形，桡动脉搏动减弱，不能做 OK 手势。最优先评估？
> 评估**血管（桡动脉/灌注）**——肱骨髁上骨折伴肱动脉损伤 → Volkmann 缺血挛缩；OK 手势不能做提示 **AIN（正中）**受累。

> [!question]- Q3. 70 岁女性 FOOSH 后腕痛，查体见餐叉样畸形，复位后渐出现拇/食/中指麻 + 拇外展无力。骨折类型 + 受累神经？
> **Colles 骨折**（桡骨远端背移）→ 急性腕管 → **正中神经**。

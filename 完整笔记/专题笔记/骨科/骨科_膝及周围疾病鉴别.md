> [!note]- 缩写对照（全称 中英）
> - **ACL / PCL** = anterior / posterior cruciate ligament 前 / 后交叉韧带
> - **MCL / LCL** = medial / lateral collateral ligament 内 / 外侧副韧带
> - **PFPS** = patellofemoral pain syndrome 髌股疼痛综合征
> - **ITBS** = iliotibial band syndrome 髂胫束综合征
> - **OSD** = Osgood-Schlatter disease 胫骨结节骨软骨病
> - **CPPD** = calcium pyrophosphate deposition disease 焦磷酸钙沉积病（假性痛风 pseudogout）
> - **OA / RA** = osteoarthritis 骨关节炎 / rheumatoid arthritis 类风湿关节炎
> - **SCFE** = slipped capital femoral epiphysis 股骨头骺滑脱
> - **LCP** = Legg-Calvé-Perthes disease 儿童股骨头缺血性坏死
> - **DVT** = deep vein thrombosis 深静脉血栓
> - **PE** = pulmonary embolism 肺栓塞
> - **MRI** = magnetic resonance imaging 磁共振成像
> - **ROM** = range of motion 关节活动度
> - **ReA** = reactive arthritis 反应性关节炎
> - **WBC** = white blood cell 白细胞
> - **NSAIDs** = nonsteroidal anti-inflammatory drugs 非甾体抗炎药
> - **SCD** = sickle cell disease 镰刀细胞病
> - **THA** = total hip arthroplasty 全髋关节置换
> - **I&D** = incision and drainage 切开引流
> - **GU / GI** = genitourinary 泌尿生殖 / gastrointestinal 胃肠
> - **MPFL** = medial patellofemoral ligament 内侧髌股韧带
> - **ABI** = ankle-brachial index 踝肱指数
> - **CTA** = CT angiography CT 血管造影

## 背诵总览 ⭐（记不清时先默这一屏）

> [!success] 三层骨架 —— 记不清整章时，先默出这一屏，再回正文补细节
> 这章难不是因为某个病难，而是 20+ 个病堆在一起、每个都有"招牌钩子"，一多就串。破解 = 只背下面三层骨架 + 几组"冤家对"的唯一区分点，其余细节全挂上去。别一上来背病名。

### 第一层：先切 4 堆（永远先做）

> [!tip] 口诀：外伤 → 定位 → 红肿 → 查髋
> 任何膝痛先问 4 个问题，病立刻掉进 4 个桶。

```
① 有急性外伤/扭转/撞击?      ──► 韧带 / 半月板 / 髌骨脱位      (机械, 见 §一)
② 没外伤, 慢性/过度使用?     ──► 按痛点定位 前/内/外/后        (见 §二~§五)
③ 红肿热痛 ± 发热?           ──► 先关节穿刺! 化脓/痛风/假痛风/ReA (见 §六)
④ 膝查体正常, 痛是放射来的?   ──► 查髋! 别漏 SCFE               (见 §七)
```

### 桶② 补充：overuse（过度使用）组细分 —— 各挑什么运动

> [!tip] 5 个 overuse 病都跟活动有关；PFPS 易被误记成"不运动导致"
> 共同点：无急性外伤、都能主动伸膝、处理靠减量 + 康复不手术。区别在**各挑什么运动 + 痛点位置**。
>
> ```
> 5 个 overuse 病都跟活动有关，其中——
> · PFPS       → 不挑运动项目 + 久坐(theater sign 剧院征)也痛   ← 易记反的"特殊"点
> · 髌腱炎      → 挑「跳跃」(篮球/排球)
> · ITBS       → 挑「长跑/骑行/下坡」
> · OSD        → 青春期跑跳
> · 鹅足滑囊炎  → 跑步 (+肥胖/膝 OA)
>
> 真正和运动无关的 = 髌前滑囊炎（长期跪地，职业性 housemaid's knee）
> ```
>
> ⚠️ PFPS 属 overuse 组（跑步/上下楼/下蹲 + 久坐都能诱发），**不是"不运动导致"**；它的特殊只是「久坐也痛 + 不必是运动员」。真正非运动的是**髌前滑囊炎（跪地职业性）**。

### 第二层：6 对「冤家」的唯一区分钥匙

> [!warning] 你不是每个病都不会，是成对的两个病互相串。背死这 6 对的唯一区分点，这章就通了。

| 冤家对 | 唯一区分钥匙 |
|---|---|
| **ACL** (前交叉韧带 anterior cruciate ligament) vs 半月板 (meniscus) | 肿得**快**（几小时积血 hemarthrosis）= ACL；肿得**慢** + **交锁 locking** = 半月板 |
| **ACL** vs 髌骨外侧脱位 (patellar dislocation) | 都有 pop + 积血。**髌骨在位** = ACL；**髌骨外移块 + 滑车沟空凹 divot + 内侧痛** = 脱位 |
| **MCL**（内侧副韧带 medial collateral ligament）vs **LCL**（外侧副韧带 lateral collateral ligament）| 撞哪边、痛**对**边。外翻应力（外侧撞）→ 内侧痛 = MCL；内翻应力（内侧撞）→ 外侧痛 = LCL |
| 内侧半月板 vs 鹅足滑囊炎 (pes anserine bursitis) | 痛在**关节线本身** + 交锁 = 半月板；痛在**关节线下约 5 cm（两横指）** + 无机械症状 = 鹅足 |
| **ITBS**（髂胫束综合征 iliotibial band syndrome）vs 外侧半月板 | 痛在**股骨外上髁体表** + 跑者 = ITBS；痛在**关节线** + 交锁 = 半月板 |
| 痛风 (gout) vs 假痛风 **CPPD**（焦磷酸钙沉积病 calcium pyrophosphate deposition）| **针状** needle 负折光 + 第一跖趾 + 痛风石 tophi = 痛风；**菱形** rhomboid 正折光 + 膝腕 + **软骨钙化 chondrocalcinosis** + 术后第 1 天 = 假痛风 |

### 第三层：2 条救命分水岭（决定 next step）

> [!danger] 分水岭①：能不能主动伸膝 (active knee extension)？
> 前膝痛：**能伸膝** = 过度使用（PFPS 髌股疼痛综合征 / 髌腱炎 / OSD 胫骨结节骨软骨病），**绝不手术**；**不能伸膝** = 伸膝装置断裂（股四头肌腱 quadriceps tendon / 髌腱 patellar tendon），**手术急症**。
> 区分断哪根 —— 口诀「老上低 / 轻下高」：股四头肌腱 = 老（>40 岁）+ 上（髌骨上间隙）+ 髌骨低位 patella baja；髌腱 = 轻（<40 岁）+ 下 + 髌骨高位 patella alta。

> [!danger] 分水岭②：膝畸形两个极端别混
> **髌骨外侧脱位** = 只是髌骨移位，大血管安全 → 复位 + 康复。
> **膝（胫股）关节脱位 (knee/tibiofemoral dislocation)** = 高能创伤，整个胫股关节脱位 → 腘动脉损伤肢危急症：**复位后即使足背脉可触也必须测 ABI（踝肱指数 ankle-brachial index）**（≤0.9 → CTA CT 血管造影 + 血管外科）。

### 我的 3 个复盘缺口（自测暴露，重点盯）

> [!danger] 反复卡在"每对冤家的第二个鉴别点"，这 3 个最该盯
> 1. **髌骨脱位 vs ACL 的分水岭** —— 两者都 pop + 积血，靠「**髌骨在不在滑车内**」切开（在位 = ACL / 外移 + 内侧痛 + divot = 脱位）。
> 2. **假痛风 CPPD 的伴随征** —— X 线「**软骨钙化 chondrocalcinosis（半月板钙化）**」，别错选痛风石 tophi。
> 3. **鹅足滑囊炎 (pes anserine bursitis)** —— 反复完全想不起：痛点在**内侧关节线下约 5 cm（两横指）+ 无机械症状**，最爱被误当内侧半月板。

## 〇、第一刀：先按"主诉性质"分四大类（最重要的分流）

> [!tip] 别一上来就背病名 — 先问三个问题，膝病立刻被切成 4 堆
> 1. **有没有急性外伤 / 扭转 / 撞击？** → 走「A. 创伤-机械损伤」（韧带 / 半月板）
> 2. **没外伤，是慢性 / 过度使用的痛？痛在哪个面？** → 走「B. 部位定位」（前 / 内 / 外 / 后）
> 3. **是红肿热痛 + 发热 / 全身？** → 走「C. 炎症-感染」（化脓 / 痛风 / 假性痛风 / ReA）
> 4. **查体膝是正常的、痛却放射来的？** → 走「D. 不在膝」（髋 / SCFE / 骨髓炎）

```
膝痛 Knee pain
│
├─ 急性外伤/扭转/撞击 ──────► A. 创伤机械（§一）
│     ├─ pop + 快速积血 hemarthrosis ──► ACL（髌骨在位）
│     ├─ pop + 髌骨外移块/滑车 divot ──► 髌骨外侧脱位（撕 MPFL）
│     ├─ dashboard 撞胫骨上端 ─────────► PCL
│     ├─ 扭转 + 交锁 locking ──────────► 半月板 meniscus
│     ├─ 外翻应力痛（内侧）────────────► MCL
│     ├─ 内翻应力痛（外侧）────────────► LCL
│     └─ 高能创伤 + 严重畸形 ──────────► 膝(胫股)关节脱位 → 复位后测 ABI 查腘动脉
│
├─ 慢性/过度使用，无大外伤 ──► B. 按部位（§二~§五）
│     ├─ 前膝 anterior ──► PFPS / 髌腱炎 / OSD / 伸膝装置断裂 / 髌前滑囊炎
│     ├─ 内侧 medial ───► MCL / 内侧半月板 / 鹅足滑囊炎
│     ├─ 外侧 lateral ──► LCL / 外侧半月板 / 髂胫束 ITBS
│     └─ 后侧 posterior ► Baker cyst（伪装 DVT）
│
├─ 红肿热痛 ± 发热/全身 ────► C. 炎症感染（§六）
│     ├─ WBC>5万 + 拒动 ──► 化脓性关节炎（先穿刺！）
│     ├─ 第一跖趾/针状负折光 ► 痛风 gout
│     ├─ 膝/腕 + 软骨钙化 ──► 假性痛风 CPPD
│     └─ 前驱 GU/GI 1-4 周 ► 反应性关节炎 ReA
│
└─ 查体膝正常，痛是放射来的 ► D. 不在膝（§七）
      ├─ 肥胖青少年 + 内旋受限 ► SCFE（查髋！）
      ├─ 老人腹股沟痛放射至膝 ► 髋 OA
      └─ 镰刀细胞/儿童发热 + 骨压痛 ► 骨髓炎
```

## 一、A. 急性创伤：韧带 vs 半月板（运动损伤核心）

> [!warning] 一句话钥匙（每个都有独家钩子，别混）
> - **ACL** = "pop" + **几小时内快速肿（积血 hemarthrosis）** + 急停变向
> - **PCL** = **dashboard 仪表盘伤**（屈膝时胫骨上端被向后撞）
> - **半月板** = 扭转 + **catching / locking 交锁**（机械症状），肿得**慢**
> - **MCL** = 膝**外侧**被撞（外翻应力）→ **内侧**痛
> - **LCL** = 膝**内侧**被撞（内翻应力）→ **外侧**痛

| 疾病 | 机制 / Buzzword | 招牌体征 | 肿胀特点 | 处理 |
|---|---|---|---|---|
| **ACL tear 前交叉韧带** | 急停 / 变向 / 落地、**"pop"** | **Lachman**（最敏感）/ anterior drawer | **快速积血**（数小时）⭐ | 关节镜重建 |
| **PCL tear 后交叉韧带** | **Dashboard** 仪表盘伤 / 屈膝跌倒 | Posterior drawer / posterior sag | 中等 | 多保守，重则手术 |
| **Meniscus tear 半月板** | 足固定 + **扭转** | **McMurray** / 关节线压痛 / 交锁 | **慢**（数小时-天）⭐ | 修补 / 部分切除 |
| **MCL tear 内侧副韧带** | **外翻应力**（外侧撞击）| **Valgus stress test+** | 局限内侧 | 多保守 + 支具 |
| **LCL tear 外侧副韧带** | **内翻应力**（内侧撞击）| **Varus stress test+** | 局限外侧 | 多保守 |
| **Unhappy triad 不幸三联** | 膝**外侧**撞 + 外翻 + 扭转 | 三者体征叠加 | 快速积血 | ACL + MCL + 半月板，重建 |

> [!danger] 高频陷阱：ACL vs 半月板"肿得快慢"
> - **ACL = 肿得快（hemarthrosis 关节内出血）** — 韧带富血供，撕裂即出血。
> - **ACL 伤过的膝，10 年内长 OA（修不修都一样）**
> - **半月板 = 肿得慢 + 机械交锁** — 半月板大部分无血供，靠的是"卡住 / 锁住"提示，不是"肿最快"。
> - 题目给"pop + 1 小时内膝就肿胀绷紧" → **ACL until proven otherwise**；给"扭一下，过两天肿 + 偶尔卡住伸不直" → 半月板。

> [!tip] ACL 三大徒手试验：按"敏感 vs 特异"分工记（高频）
> | 试验 | 膝屈曲位 | 定位 |
> |---|---|---|
> | **Lachman test** | 20–30° | **敏感度最高 → 筛查/床旁首选** ⭐ |
> | **Anterior drawer 前抽屉** | 90° | 敏感度较低，急性期疼痛/肌痉挛 → 易假阴性 |
> | **Pivot shift 轴移** | 近伸直→缓屈 | **特异度最高**，测旋转不稳；需放松/麻醉，清醒急性期常做不出 |
>
> - 记忆：**Lachman 抓"有没有"（敏感），Pivot shift 定"是不是真的"（特异）**。
> - 三者均为**临床试验**；ACL 软组织损伤**确诊仍靠 MRI 磁共振 (magnetic resonance imaging)**。

> [!tip] 半月板两大诱发试验：都靠"旋转 + 弹响/关节线痛"（与上面 ACL 试验配对记）
> | 试验 | 体位 | 手法 → 阳性 |
> |---|---|---|
> | **McMurray test** | **卧位**，屈膝 | 小腿外旋+外翻测**内侧**半月板 / 内旋+内翻测**外侧**半月板，缓慢伸膝 → **弹响 (click)/关节线痛** = 阳性 |
> | **Thessaly test** | **站立单腿负重**，患膝屈约 20° 足踩地 | 身体左右**旋转扭膝** → 关节线痛 / **交锁 (locking)** = 阳性 |
>
> - 二者机制一致：**足固定 + 旋转**让撕裂的半月板被卡压。**Thessaly 是负重位**（更贴近真实症状）、McMurray 是卧位。
> - 与 ACL 试验的分工对照：**ACL = 查"松不松"（Lachman/前抽屉/轴移，测稳定性）；半月板 = 查"卡不卡"（McMurray/Thessaly，测旋转诱发的机械痛/弹响）**。
> - 半月板确诊同样靠 **MRI / 关节镜**；X 线正常不排除。

> [!tip] X 线小撕脱提示韧带（高频影像点）
> - **Segond fracture（胫骨外侧平台小撕脱）= ACL 撕裂的间接征象**（"看见骨头被拽掉一小块，也要想 ACL"）。
> - 韧带 / 半月板软组织损伤 **确诊都靠 MRI**；X 线主要排骨折。

> [!tip] 不幸三联的"内侧半月板"细节
> 经典 unhappy triad = ACL + MCL + **内侧半月板**（外侧撞→外翻→内侧结构受牵拉）。有些版本写外侧半月板，按 stem 描述哪侧关节线压痛走。

### 一-补：髌骨外侧脱位 (Patellar dislocation) — "pop" 但髌骨不在位 ⭐

> [!danger] 与 ACL 的分水岭：撞车点是"pop + 弥漫关节积血"，不是"内侧肿"
> 两者都因屈膝扭转/落地 → **"pop" + 撕裂处出血灌入关节腔 → 急性弥漫性关节积血 (hemarthrosis，整个膝肿胀绷紧，非内侧局限)** —— 这才是要鉴别的重叠点；经典错点：听到 **"pop"** 就跳 ACL。
> - **髌骨脱位**：髌骨甩到**外侧**成块状畸形 + 前方**滑车沟空凹 (divot)** + **内侧（MPFL 处）压痛** → 髌骨**已离开滑车**。⚠️ 内侧压痛是**指向髌骨脱位的区分线索，不是和 ACL 的共有点**。
> - **ACL 撕裂**：髌骨**仍在滑车内**、Lachman+，内侧一般不痛（除非合并 MCL / 不幸三联）。
> 源：[[mistakes/uworld-mistakes_2026-06#^Q13974]]

> [!warning] 追问：ACL 和髌骨脱位「两个都迅速肿」吗？—— 是，这一对**肿胀速度不能当钥匙** ⭐
> **两个都快**（数小时内关节积血 hemarthrosis，整膝肿胀绷紧），因为各自都有活动性出血源：
> - **ACL** = 富血供韧带（膝中动脉 middle genicular artery 供血），一撕即出血 → ACL 是**急性创伤性关节积血最常见原因**（约 70%）。
> - **髌骨外侧脱位** = 撕裂 MPFL（内侧髌股韧带）+ 髌骨刮过滑车常伴**骨软骨骨折 (osteochondral fracture)** → **两个出血源**，同样数小时肿满。
>
> 所以「pop + 快速积血」只能**排掉半月板**（半月板内 2/3 无血供 → 肿得**慢** + 交锁 locking），**排不开** ACL 和髌骨脱位 —— 这两个必须再靠**髌骨在不在滑车 + 内侧 MPFL 压痛**切第二刀。⚠️「肿得快慢」是 ACL↔半月板 的钥匙，**不是** ACL↔髌骨脱位 的钥匙，别拿错。
>
> ```
> pop + 快速关节积血 hemarthrosis
>   ├─ 半月板 meniscus   ✗ 排除（它肿得慢 + 交锁）
>   ├─ ACL              ← 髌骨在位 + Lachman+，内侧一般不痛
>   └─ 髌骨外侧脱位       ← 髌骨外移块 + 滑车沟空凹 divot + 内侧(MPFL)压痛
> ```

| 维度 | 内容 |
|---|---|
| **机制** | 屈膝快速**扭转/落地**（体操、舞蹈、变向急停）→ 股四头肌外向牵拉把髌骨拉向**外侧 (lateral)** → 撕裂提供内向反力的 **MPFL（内侧髌股韧带 medial patellofemoral ligament）** |
| **方向** | 几乎都是**外侧脱位**（股四头肌合力 + Q 角天然外偏）|
| **MPFL 损伤** | 急性外侧脱位**几乎都伴 MPFL 损伤**（约 90–100%），但是**一个谱**：拉伸 / 部分撕裂 / 完全断裂 / 起止点撕脱（多在股骨内上髁侧）——不全是"断成两截"。⚠️ **复发性 / 关节过度松弛者**可为**慢性 MPFL 松弛**而非每次急性断裂 |
| **危险因素** | 年龄 **<20**、女性、全身**关节松弛**、下肢力线不良（Q 角大/膝外翻）、髌骨/滑车发育不良、既往**髌骨半脱位 (subluxation)**、竞技运动/舞蹈/军训 |
| **查体** | 膝**屈曲位 + ROM↓**、髌骨外移块状畸形、前方**滑车沟空凹 (divot)**、**内侧髌骨压痛**（MPFL 撕裂）、关节积血 (hemarthrosis) |
| **影像** | 多为临床诊断；X 线（含**髌骨轴位/skyline view**）排**骨软骨骨折 (osteochondral fracture)** + 看髌骨轨迹；就诊时**常已自行复位** |
| **处理** | **首发多保守**：自行/闭合复位 → 伸直位夹板/支具 + **股四头肌（尤其股内侧斜肌 VMO, vastus medialis obliquus）强化康复**；**复发性 / 骨软骨游离体 → 手术（如 MPFL 重建）** |

> [!tip] 一句话锁定
> **屈膝落地 pop + 髌骨外移成块 + 滑车沟空凹 (divot) + 内侧痛 = 髌骨外侧脱位（撕 MPFL）；ACL 则是"髌骨在位 + 关节积血"。**

### 一-补2：急性膝外伤"pop / 畸形 / 积血"一表分流 ⭐⭐

> [!tip] 四个"急性 pop / 畸形"长得像，靠"对位 + next step + 肢危风险"切开

| 病 | 机制 / buzzword | 招牌（对位 + 体征） | 积血 | 关键 next step | 不能漏 |
|---|---|---|---|---|---|
| **ACL 撕裂** | 急停 / 变向 / 落地 + **pop** | 髌骨在位、**Lachman+** | 快速 hemarthrosis | **MRI**（软组织确诊） | Segond 骨折 = ACL 间接征 |
| **PCL 撕裂** | **dashboard 仪表盘**撞胫骨上端 / 屈膝跌倒 | **posterior drawer / 胫骨后塌 (sag)**、髌骨在位 | 中等 | 多保守；重则手术 | 高能时常合并多韧带 → 查血管 |
| **髌骨外侧脱位** | 屈膝**扭转** + pop | **髌骨外移块 + 滑车 divot + 内侧 MPFL 压痛**；常自行复位 | 弥漫 hemarthrosis | 复位 + 伸直夹板 + VMO 康复 | skyline 位排**骨软骨骨折** |
| **膝（胫股）关节脱位** ⭐ | **高能创伤**（车祸 / 高坠 / 运动重创） | 胫骨相对股骨明显移位（**后脱位最常见**），**可自行复位而"看似正常"** | 明显 | **复位后必测 ABI 踝肱指数 (ankle-brachial index) 查腘动脉**（脉可触也要测！） | **腘动脉损伤 = 肢危急症**；ABI ≤0.9 / 软征 → **CTA (CT 血管造影) + 血管外科** |

> [!danger] "膝畸形"两个极端别混（髌骨脱位 vs 膝关节脱位）
> 都可表现"膝部畸形 + pop"，但紧急程度天差地别：
> - **髌骨外侧脱位**：移位的只是**髌骨**，大血管安全 → 复位 + 康复。
> - **膝（胫股）关节脱位**：整个**胫股关节**脱位 → **腘动脉损伤肢危急症**；**复位后即使足背动脉可触，也必须测 ABI**（脉搏查体对腘动脉损伤不可靠，正常脉 + ABI >0.9 才相对安心，否则 CTA）。源：[[mistakes/uworld-mistakes_2026-06#^Q17060]]

## 二、B-前侧：前膝痛 (Anterior knee pain) — 最容易串的一组 ⭐⭐⭐

> [!warning] 这一组是错题高发区（PFPS 反复错）。核心：**能不能伸膝？痛点定位清不清？年龄？**
> ➡️ 单页决策流程图见衍生页 [[完整笔记/专题笔记/骨科/骨科_前膝痛能否伸膝决策流程]]

| 疾病 | 人群 / 钩子 | 痛点定位 | 能否伸膝 | 招牌 | 治疗 |
|---|---|---|---|---|---|
| **PFPS 髌股疼痛综合征** | **年轻女性**、跑步/上下楼/久坐(theater sign)加重 | **模糊**前膝痛、定位不清 | **能** | **髌股压迫痛 / 股四头肌等长收缩痛** | **股四头肌 + 髋外展强化；忌制动** ⭐ |
| **Patellar tendinitis 髌腱炎 (jumper's knee)** | 跳跃运动员 | **髌骨下极/髌腱局灶**、点清楚 | 能 | 髌腱压痛 | 相对休息 + 离心训练 |
| **Osgood-Schlatter (OSD)** | **青春期 + 活动多** | **胫骨结节 (tibial tuberosity)** 痛+肿 | 能 | 胫骨结节压痛 + **隆起**、**抗阻伸膝 / 下蹲诱发痛** ⭐ | 限制活动 + 拉伸（自限）|
| **股四头肌腱断裂** | **>40 岁** | 髌骨**上方**间隙 | **不能伸膝** ⭐ | 高位间隙 + 髌骨低位 | **手术** |
| **髌腱断裂 (patellar tendon)** | **<40 岁** | 髌骨**下方**间隙 | **不能伸膝** ⭐ | 低位间隙 + **髌骨高位 (patella alta)** | **手术** |
| **髌前滑囊炎 prepatellar bursitis (housemaid's knee)** | 长期**跪地**（园丁/清洁工）| 髌骨**正前方表浅**肿包 | 能（被动 ROM 基本正常）| 髌前局限肿 / 红 | 避免跪、对症；红热重排 septic bursitis |

> [!tip] 髌腱 (patellar tendon) 纵向地图：髌腱炎 vs OSD 别当成同一处 ⭐
> 髌腱是**竖**的一条，从髌骨下极连到胫骨结节；前膝几个过度使用病沿这条腱**按高度排开**，痛点一上一下差几 cm：
> 
> ```
>         髌骨 patella
>           │
>           ●  ← 髌骨下极 inferior pole    ★ 髌腱炎 jumper's knee（上端）
>           │
>           │  ← 髌腱 patellar tendon       ★ 髌腱断裂在此（突发不能伸膝）
>           │
>         ▇▇▇ ← 胫骨结节 tibial tubercle    ★ Osgood-Schlatter OSD（下端）
>         胫骨 tibia
> ```
> 
> | | **髌腱炎 (Patellar tendinitis / jumper's knee)** | **Osgood-Schlatter (OSD)** |
> |---|---|---|
> | **精确痛点** | **髌骨下极**（髌腱近端起点）| **胫骨结节**（髌腱远端止点的骨性突起）|
> | **病理** | 髌腱本身肌腱病 (tendinopathy) | **胫骨结节骨骺牵拉性骨软骨炎** (apophysitis)，生长板未闭 |
> | **人群** | 跳跃运动员（任何年龄，含成人）| **青春期 + 快速生长 + 活动多**（10–15 岁，男多）|
> | **招牌体征** | 髌骨下极压痛 | **胫骨结节压痛 + 局部骨性隆起/肿大**（摸得到鼓包）⭐ |
> | **转归** | 慢性，相对休息 + 离心训练 | **自限**（骨骺闭合后好），限制活动 + 拉伸 |
> 
> - 一刀切：**痛在"髌骨正下方一点" = 髌腱炎（上端）**；**痛在"小腿上端那个鼓包" + 青少年 + 结节肿大 = OSD（下端）**。
> - 共同点（所以易混）：都在膝**前方**、都在**同一条髌腱**上、都是**过度使用**、都**能主动伸膝**（区别于伸膝装置断裂）。

> [!danger] 决策分水岭：「能不能主动伸膝 (active knee extension)」
> 伸膝装置 = 股四头肌 → 股四头肌腱 → 髌骨 → 髌腱 → 胫骨结节。**这条链上任何一处断裂 → 不能主动伸膝 (extensor mechanism rupture)**，是**手术急症**。
> - **不能伸膝 + 重创病史** → 想股四头肌腱 / 髌腱断裂（按年龄 + 髌骨高低分）。
> - **能伸膝 + 慢性模糊痛** → PFPS / 髌腱炎 / OSD 这类过度使用，**绝不手术**。
> - ❌ 把 PFPS 的**废用性股四头肌萎缩**误读成"伸膝装置断裂"是经典错点（萎缩≠断裂，PFPS 伸膝完整）。

> [!tip] 股四头肌腱 vs 髌腱断裂的"上下 + 年龄"记忆
> - **股四头肌腱 = 老 (>40) + 上 (髌骨上方间隙) + 髌骨被往下拽（低位 patella baja）**
> - **髌腱 = 轻 (<40) + 下 (髌骨下方间隙) + 髌骨被往上弹（高位 patella alta）**
> - 口诀：**"老上低 / 轻下高"**。

> [!tip] PFPS 三连（错题反复区，默写）
> 1. **诊断**：年轻女 + 模糊前膝痛 + 上下楼/久坐加重 + 髌股压迫试验+。
> 2. **体征**：**绷股四头肌（isometric 等长收缩）就痛** + 压髌痛；**能伸膝**。
> 3. **治疗**：**股四头肌 + 髋外展肌强化；忌制动**（制动越练越软 → 越糟）。Knee immobilizer 是反向。

> [!info] 髌骨外侧脱位（Patellar dislocation）虽痛在前膝，但本质是**急性创伤**
> 屈膝落地 pop + 髌骨外移畸形 + 滑车 divot —— 完整小节（与 ACL 的"both pop"分水岭 + MPFL + 处理）已归到急性创伤段 [[#一-补：髌骨外侧脱位 (Patellar dislocation) — "pop" 但髌骨不在位 ⭐|§一-补 髌骨外侧脱位]]。

## 三、B-内侧：内侧膝痛 (Medial knee pain)

| 疾病 | 钩子 | 痛点 | 鉴别招牌 |
|---|---|---|---|
| **MCL 损伤** | 外翻应力 / 外侧撞击后 | 内侧副韧带处 | **Valgus stress test 阳性** |
| **内侧半月板撕裂** | 扭转 + 交锁 | **内侧关节线**压痛 | McMurray+ / catching·locking |
| **Pes anserine bursitis 鹅足滑囊炎** | **跑步 / 肥胖 / 膝 OA**、亚急性 | **内侧关节线下方约 5 cm（两横指）**（胫骨近端内侧）⭐ | 痛点**不在真关节线**、无机械症状 | 

> [!tip] 鹅足滑囊炎最易被当成"内侧半月板"
> - **鹅足滑囊炎痛点在关节线 *下方* 约 5 cm（两横指）**（缝匠肌/股薄肌/半腱肌共同止点处），**不在关节线本身**，且**没有交锁 / 卡顿**机械症状。
> - 处理 = **股四头肌 + 腘绳肌强化拉伸**（和 PFPS 类似的康复逻辑），不是手术。
> - 半月板才有"关节线压痛 + locking"。**"内侧痛点偏低 + 无机械症状 = 想鹅足"**。

## 四、B-外侧：外侧膝痛 (Lateral knee pain)

| 疾病              | 钩子               | 鉴别招牌                                                      |
| --------------- | ---------------- | --------------------------------------------------------- |
| **LCL 损伤**      | 内翻应力 / 内侧撞击      | **Varus stress test 阳性**                                  |
| **外侧半月板撕裂**     | 扭转 + 交锁          | 外侧关节线压痛 / 弹响 / locking                                    |
| **髂胫束综合征 ITBS** | **长跑 / 骑行**、过度使用 | 外侧**股骨外上髁**处摩擦痛、屈膝 ~30° 时痛、**Noble / Ober test**；无关节内机械症状 |

> [!tip] ITBS 是"跑者外侧膝痛"，不是关节内病
> 髂胫束综合征痛在**股骨外上髁体表**（髂胫束反复摩擦），随跑量加重；**没有积液、没有交锁**。处理 = 减量 + 拉伸 + 髋外展肌强化。和外侧半月板（关节线 + 机械症状）区分。

> [!warning] ITBS vs GTPS：都叫"外侧痛 + 跑者 + 髋外展强化"，但一个在膝、一个在髋 ⭐
> 髂胫束 (iliotibial band) 解剖上从髂嵴一路连到胫骨外侧 (Gerdy 结节)，所以同一条带子在**两端**各出一个过度使用综合征——跑者常同时被怀疑，靠**痛点在膝还是在髋**一刀切开。
> 
> | | **ITBS 髂胫束综合征** | **GTPS 大转子疼痛综合征 (greater trochanteric pain syndrome)** |
> |---|---|---|
> | **痛点位置** | **外侧膝**——股骨外上髁 (lateral femoral epicondyle) | **外侧髋**——大转子 (greater trochanter) |
> | **本质** | 髂胫束远端在外上髁反复摩擦 | 臀中/小肌肌腱病 ± 大转子滑囊炎 (旧称 trochanteric bursitis) |
> | **诱因人群** | **长跑 / 骑行**、增量过快、下坡 | **中年女**、跑者；髋外展肌弱 |
> | **招牌体征** | 屈膝 ~30° 摩擦痛、**Noble / Ober test** 阳性 | **侧卧患侧压痛**、大转子点压痛、外展抗阻痛 |
> | **关节内？** | 否（无积液/交锁）| 否（关节外）|
> | **治疗** | 减量 + 拉伸 + **髋外展肌强化** | 相对休息 + NSAIDs + **髋外展肌强化** |
> 
> - 一刀切：**膝外侧上髁痛 = ITBS；髋外侧大转子痛 + 侧卧痛 = GTPS**。两者**治疗高度重叠**（都练髋外展肌），但**定位完全不同**。
> - GTPS 详见 [[完整笔记/专题笔记/骨科/骨科_髋及周围疾病鉴别]] §二。

## 五、B-后侧：后膝 / 腘窝 (Posterior knee)

> [!danger] Baker cyst (腘窝囊肿) 破裂 = 伪装 DVT 的头号选手
> - **腘窝囊肿 (Baker / popliteal cyst)**：常继发于膝**OA / RA / 半月板病**（关节液经单向阀进腘窝）。
> - **破裂时** → 滑液漏入小腿 → **小腿肿胀 + 疼 + 有时内踝下方新月形瘀斑** → **酷似 DVT 深静脉血栓**。
> - **下一步永远先排 DVT（加压超声）**，超声同时能看到囊肿 → 确诊后对症 + 治原发关节病。
> - 钩子：**已知膝关节炎/RA 背景 + 突发小腿肿痛**，别只想血栓。

> [!warning] next-step 陷阱：是"加压超声"不是"关节穿刺"（自测错点）
> - **下一步 = 下肢静脉加压超声 (compression ultrasound)**，一举两得：①排除 DVT（漏诊致命）②直接看到囊肿。
> - ❌ **别选关节穿刺 (arthrocentesis)** —— 那是给"红、热、肿的**关节本身**"（疑化脓/晶体）用的；Baker 破裂问题在**小腿**，穿刺既排不掉 DVT 也不解决。
> - **确诊后处理（保守为主）**：休息 + 抬高 + NSAIDs；治原发关节炎（OA/RA 滑膜炎是源头）；大/反复 → 超声引导抽吸 + 囊内激素。
> - 分流口诀：**关节红肿 → 穿刺；小腿肿痛像 DVT → 超声**。源：[[mistakes/uworld-mistakes_2026-06#^SQ-Baker-0616]]

## 六、C. 非创伤性"红肿热痛"膝 — 炎症 vs 感染 vs 晶体 ⭐

> [!danger] 铁律：单关节急性红肿热痛 → **先关节穿刺 (arthrocentesis)**，在排除化脓之前不要假设是痛风/假性痛风
> 关节液 WBC 阶梯：正常 <200 → 非炎症(OA) <2000 → 炎症(晶体/RA) 2000–50,000 → **化脓 >50,000**

| | **化脓性关节炎 Septic** | **痛风 Gout** | **假性痛风 CPPD/pseudogout** | **反应性关节炎 ReA** |
|---|---|---|---|---|
| **关键钩子** | 高热 + **拒绝负重/ROM 剧降** | 高嘌呤/利尿剂诱发 | **术后/急性病第 1 天**急性单膝 ⭐ | 前驱 **GU(衣原体)/GI 感染 1–4 周后** |
| **常累及** | 膝（大关节）| 第一跖趾 (podagra) | **膝、腕** | 下肢大关节(膝/踝)**不对称寡关节** |
| **关节液 WBC** | **>50,000** | 2 万级 | 2 万级 | 炎症级 |
| **晶体 / 影像** | Gram 染色（敏感仅 30–50%）| **针状 needle、负性双折光** | **菱形 rhomboid、正性双折光** + **软骨钙化 chondrocalcinosis = 半月板钙化** ⭐ | 无晶体；附着点炎 |
| **伴随标志** | — | **Tophi 痛风石**（关节外尿酸盐）| **半月板/软骨钙化** + 关联血色病/甲旁亢 | 结膜炎 + 尿道炎（"不能看、不能尿、不能爬"）|
| **处理** | **穿刺 → IV 抗生素 + 引流 I&D** | NSAIDs / 秋水仙碱 colchicine / 激素 corticosteroid | NSAIDs / 激素 | NSAIDs；治原发 |

> [!danger] "滑液大量中性粒 (PMN) = 感染"是坑（ReA 治疗题反复错）
> 上表**每一种**（化脓 / 痛风 / CPPD / ReA）滑液都是**中性粒 (PMN) 为主**——PMN 占优是炎症共性，**不是化脓专属**。判化脓看 **WBC 总数 >50,000 + 革兰/培养阳**；ReA 是 **WBC 炎症级（常 ~10,000）+ 革兰阴无菌** → 一线 **NSAIDs（非甾体抗炎药）不是抗生素**。详见 [[完整笔记/专题笔记/风湿/风湿_反应性关节炎ReA]]（§5.2 治疗）。

> [!warning] 痛风 vs 假性痛风的"伴随征"串科陷阱（错题点）
> - **Tophi 痛风石 = 痛风的标志**（关节外尿酸盐沉积）。
> - **Chondrocalcinosis 软骨钙化（X 线见半月板/关节软骨线状钙化）= 假性痛风 CPPD 的标志**。
> - 别把两者对调！考"假性痛风最可能伴随哪项？" → 选**半月板钙化**，不是痛风石。
> - **术后第 1 天突发急性单膝红肿** → 别只想 septic，**CPPD 假性痛风**经典被手术/急性病诱发。

> [!info] OA 也会"膝痛"，但属慢性非炎症
> 膝 **OA**：慢性、**活动后加重 / 休息缓解**、晨僵 **<30 min**、骨擦感、X 线"关节间隙变窄 + 骨赘 + 软骨下硬化"。一线管理 = **运动 + 减重 + NSAIDs**（必要时关节腔激素）。和上面"急性红肿热痛"那组分清。
> ➡️ OA 完整体系（OA vs RA、危险因素、手/髋分布、治疗阶梯）见 [[完整笔记/专题笔记/骨科/骨科_OA骨关节炎完整体系]]；下面只留膝专项的继发诱因 + 管理 + 间室。

> [!warning] 膝专项钩子：年轻人单膝 OA 必问"既往关节伤"（派生自 Q15631）⭐
> **35 岁、BMI 正常、单膝 OA + 高中 ACL 重建史** → "最显著诱因"选**既往 ACL 损伤**（前交叉韧带伤后 10 年内多发膝 OA，不论是否手术）；旁边堆的**维 D 缺乏（管骨软化，非单膝 OA）/ 糖尿病**是干扰项。
> ➡️ 继发性 OA 完整危险因素表 + 其他病因 + 干扰项辨析见 [[完整笔记/专题笔记/骨科/骨科_OA骨关节炎完整体系]] §三。

> [!tip] 膝 OA 管理阶梯：保守走完 → 全膝置换 TKA（派生自 Q15632）
> 
> ```
> ① 非药物：减重 + 规律活动(步行) + 股四头肌强化   ← 一线，所有人
>      │ 症状持续
> ② 局部/口服 NSAIDs (非甾体抗炎药)；考虑 duloxetine (度洛西汀)、topical capsaicin (辣椒素)
>      │ 症状持续
> ③ 关节腔激素注射（± 透明质酸 hyaluronic acid）   ← 短期缓解
>      │ 保守充分失败 + 显著功能受限
> ④ TKA 全膝置换 (total knee arthroplasty)（可手术者） / 慢性疼痛管理（不可手术者）
> ```
> 
> - **走完②③仍进行性受限（如只能走 2 个街区）= TKA 候选**，别再回头加系统激素或阿片。
> - **铰链式 unloader (valgus) 膝支具**只对**单间室内侧 OA + 内翻畸形**有效（转移负荷至外侧间室）；**双间室广泛 OA 无效**。
> - **系统性糖皮质激素**：升血糖、副作用大，**糖尿病相对禁忌**；OA 不用系统激素。
> - **关节镜半月板切除**对已确立 OA 无益（反增 OA 风险）；**关节穿刺**仅用于炎症/急性大积液（化脓/血关节），OA 小积液无益。

> [!tip] 膝"间室 (compartment)"解剖 + 为什么 unloader 只对单间室（原理，派生自 Q15632）⭐
> **膝 = 3 个独立承重间室**：**内侧 (medial) / 外侧 (lateral)** 股骨-胫骨间室 (tibiofemoral) + **髌股 (patellofemoral)** 间室。
> - **单间室 OA (unicompartmental)** = 只坏 1 个（最常见 = **内侧**）；**双 / 三间室 OA (bi- / tricompartmental)** = 坏 2–3 个。
> - **力线 ↔ 间室**：**内翻 (varus, O 型腿) → 内侧间室**；**外翻 (valgus, X 型腿) → 外侧间室**。
> 
> **unloader 支具原理**：靠反向力矩把负荷从坏间室"卸"到**健康间室** → **前提：必须存在一个健康间室**。
> - 单间室 OA（内侧坏 / 外侧好）→ 支具把负荷推向健康外侧 → **有效** ✅
> - **双间室 OA（内外都坏）→ 无健康间室可卸 → 无效** ❌（本题 Q15632 双间室 → 直接排 unloader）
> 
> **间室数 → 术式（分水岭）**：
> - **单间室** + 年轻活跃 + 内翻 → **HTO 胫骨高位截骨 (high tibial osteotomy)** / **UKA 单髁置换 (unicompartmental knee arthroplasty)**；保守期可上 unloader 支具。
> - **双 / 三间室** + 终末期 + 保守失败 → **TKA 全膝置换 (total knee arthroplasty)**。

## 七、D. "膝痛但根源不在膝"——referred pain 必查髋 ⭐

> [!danger] 青少年/儿童膝痛、查体膝却正常 → 一定查髋（漏 SCFE 是大错）
> 髋关节病变（闭孔/股神经）常把痛**放射到膝**。膝查体阴性时务必查髋 ROM。

| 来源 | 人群 | 钩子 | 关键 / 处理 |
|---|---|---|---|
| **SCFE 股骨头骺滑脱** | **肥胖青少年 10–15 岁** | **膝/髋痛 + 跛行 + 髋内旋受限** | 双髋正位 + **frog-leg X 线**；**非负重 + 手术 pinning** |
| **LCP（儿童股骨头缺血坏死）** | **瘦小男孩 4–10 岁** | 无创伤渐进跛行 + 膝/大腿痛 | frog-leg X 线；休息 + 会诊 |
| **髋 OA** | 老年 | 腹股沟痛**放射至大腿/膝**、内旋受限、休息缓解 | X 线（间隙窄+骨赘）；保守→THA |
| **骨髓炎 osteomyelitis** | 儿童 / **镰刀细胞病 SCD** | 发热 + **干骺端局灶压痛肿胀** + 炎症指标↑ | MRI；血培养 + IV 抗生素（SCD 加覆沙门菌）|

> [!tip] "膝痛掩护髋病"金句
> 13 岁胖男孩**膝痛**跛行、膝查体正常但**髋内旋受限** → 不是膝病，是 **SCFE**（referred pain）→ 查髋 + frog-leg X 线。这是儿科 + 骨科双高频。

## 八、体征 / 检查 → 诊断速查（膝专项）

| 体征 / 试验 | 阳性 = |
|---|---|
| **Lachman / Anterior drawer** | ACL 前交叉韧带（Lachman 最敏感）|
| **Pivot shift test 轴移试验** | ACL 前交叉韧带（**特异性最高**，评估旋转不稳；需放松/麻醉）|
| **Posterior drawer / Posterior sag** | PCL 后交叉韧带 |
| **McMurray / 关节线压痛 + locking** | Meniscus 半月板 |
| **Valgus stress test（外翻）** | MCL 内侧副韧带 |
| **Varus stress test（内翻）** | LCL 外侧副韧带 |
| **髌股压迫痛 / 股四头肌等长收缩痛** | PFPS（能伸膝）|
| **不能主动伸膝 (active extension lag)** | 伸膝装置断裂（股四头肌腱 / 髌腱）— 手术 |
| **胫骨结节压痛（青少年）** | Osgood-Schlatter |
| **关节线下方约 5 cm（两横指）痛、无机械症状** | Pes anserine bursitis 鹅足滑囊炎 |
| **股骨外上髁摩擦痛（跑者）/ Noble·Ober** | 髂胫束综合征 ITBS |
| **小腿肿痛 mimics DVT** | Baker cyst 破裂（先超声排 DVT）|
| **X 线半月板/软骨线状钙化** | CPPD 假性痛风（chondrocalcinosis）|
| **Segond fracture（胫骨外侧平台小撕脱）** | ACL 间接征象 |
| **髌骨外移块状畸形 + 滑车沟空凹 (divot) + 内侧痛** | 髌骨外侧脱位（撕 MPFL）|
| **高能伤后膝明显畸形 / 复位后脉可触仍须测 ABI** | 膝（胫股）关节脱位 → 排腘动脉损伤（ABI≤0.9 → CTA）|

> [!tip] 关节线 (joint line) 在哪 / 怎么摸 + vs 髌股关节
> **胫股关节线 (tibiofemoral joint line)** = 股骨髁 (femoral condyle) 与胫骨平台 (tibial plateau) 相接的水平缝，**在髌骨下尖以下、髌腱 (patellar tendon) 两侧的软凹里**——**不是髌骨左右两侧**（髌骨偏高偏前）。
> 
> ```
>         髌骨 patella（偏高、偏前）
>          ___
>         /   \
>    ━━━━┙     ┕━━━━   股骨髁 femoral condyle
>         │ ● │         ← 髌骨尖 apex
>    ◣----│   │----◢    ← 关节线 joint line（横线，髌腱两侧凹陷 ★ 摸这里）
>    内侧  │   外侧
>          │            ← 髌腱 patellar tendon（竖绳）
>    ━━━━━━━━━━━━━━━    胫骨平台 tibial plateau
>
>    记：横线=关节线（内→外水平缝）；竖绳=髌腱（髌骨尖→胫骨结节）
> ```
> 
> **手法**：屈膝 90° 足踩床（间隙张开 + 肌肉松）→ 找髌骨下尖 → 手指放髌腱**内/外侧软凹**向后按 → 横行沟 = 内/外侧关节线，半月板边缘正在此线上。
> - 压痛在**关节线本身** → 半月板 / 胫股 OA。
> - 痛在**髌骨正后方**（按髌骨 / 绷股四头肌诱发）→ **PFPS / 髌股间室 OA**（关节线摸不到髌股，靠髌股压迫试验）。

> [!tip] 沿关节线"纵向"定位速查：上下差几 cm，诊断全变 ⭐
> 一条垂线从关节线往上下扫，痛点高度决定诊断：
> 
> | 痛点高度 | 诊断 | 钩子 |
> |---|---|---|
> | **关节线本身**（髌腱两侧凹陷）| 半月板撕裂 / 胫股 OA | 半月板=扭转+locking+McMurray+；OA=慢性+骨擦感 |
> | **内侧关节线下方约 5 cm（两横指）**（胫骨近端内侧）| 鹅足滑囊炎 (pes anserine bursitis) | 无机械症状、不在真关节线 |
> | **关节线外上、股骨外上髁体表** | 髂胫束综合征 (ITBS) | 跑者、屈膝 ~30° 摩擦痛、关节外 |
> | **关节线下、胫骨结节 (tibial tuberosity)** | Osgood-Schlatter (OSD) | 青春期、髌腱止点牵拉痛 |
> | **髌骨正后方**（不在关节线）| PFPS / 髌股 OA | 髌股压迫 + 股四头肌等长收缩痛 |
> 
> 出题钥匙：**读 stem 务必抓"压痛精确位置"**——"内侧关节线"vs"内侧关节线下 5 cm"差一句话，半月板 ↔ 鹅足就反转。

> [!tip] 体征手法图解（看动作记体征，别死背名字）
> ```
> Lachman test —— ACL 最敏感 ★
>   膝屈 20–30°，一手稳股骨、一手向前提胫骨
>        股骨 ━━━━━┓
>                  ┃→ 胫骨被向前拉
>        胫骨 ━━━━━┛
>   阳性 = 胫骨过度前移 + 终点"软"(soft endpoint) → ACL 撕裂
>
> Anterior / Posterior drawer —— ACL / PCL
>   膝屈 90°、足固定，前后推胫骨
>   向前松 = ACL ；向后松 / 胫骨平台后塌(sag) = PCL
>   注：anterior drawer 在屈 90° 位，急性期因疼痛/肌痉挛易假阴性 → 敏感度不如 Lachman
>
> Pivot shift test —— ACL 特异性最高 ★
>   膝近伸直位，施外翻应力 + 胫骨内旋，缓慢屈膝
>   阳性 = 屈约 20–30° 时胫骨外侧平台从前向后"咔"地复位(clunk) → ACL 撕裂(旋转不稳)
>   需患者完全放松/麻醉下做，清醒急性期常做不出 → 敏感度低、特异度高
>
> McMurray —— 半月板 (Meniscus)
>   屈膝→外旋+外翻=内侧板；内旋+内翻=外侧板；缓慢伸直
>   阳性 = 弹响 click / 关节线痛 → 半月板撕裂
>
> Valgus / Varus stress —— MCL / LCL
>   膝微屈 30°
>   推膝外侧(外翻应力)开内侧缝 = MCL
>   推膝内侧(内翻应力)开外侧缝 = LCL
>
> Patellar compression / Quad isometric —— PFPS
>   压住髌骨 + 让患者绷股四头肌(等长收缩)
>   阳性 = 髌股区痛，但仍能完全伸膝
>
> Active extension lag —— 伸膝装置断裂 (手术急症)
>   让患者直腿抬高 / 主动伸膝
>   抬不起 / 伸不到底 = 股四头肌腱或髌腱断裂
>
> Thompson (calf squeeze) —— 跟腱 (Achilles) 对照记
>   俯卧捏小腿肚 → 正常足跖屈；不动 = 跟腱完全断
> ```

> [!tip] 膝相关反射 / 节段
> 膝跳反射 (patellar) = **L3–L4**（"Kick the door 3-4"）；伸膝（股四头肌）= L2–L4。膝痛放射 + 反射改变要想腰椎神经根。

## 九、自测（答完再判分）

> [!question]- Q1. 22 岁篮球运动员变向落地时听到"pop"，1 小时内右膝明显肿胀绷紧。最敏感的体格检查与确诊影像？
> **ACL 前交叉韧带撕裂**（快速积血 hemarthrosis 是关键）→ **Lachman test** 最敏感 → **MRI** 确诊。Segond fracture 是其 X 线间接征象。

> [!question]- Q2. 17 岁女，3 个月模糊定位的前膝痛，跑步/久坐/上下楼加重，有"打软 giving way"感，髌股压迫试验阳性，能完全伸膝，无肿胀。最佳处理？
> **PFPS 髌股疼痛综合征** → **股四头肌 + 髋外展肌强化锻炼；忌制动**。Knee immobilizer（制动）是反向选择 → 越制动股四头肌越弱。

> [!question]- Q3. 老人术后第 1 天突发左膝红肿热痛、低热，关节液 WBC 3 万、中性 90%、革兰染色阴性、镜下少量菱形晶体。最可能伴随哪项影像？
> **CPPD 假性痛风** → **chondrocalcinosis 软骨钙化 = 半月板钙化**（不是 tophi 痛风石！）。术后/急性病第 1 天是经典诱因。

> [!question]- Q4. 60 岁膝 OA 患者突发右小腿肿胀疼痛，内踝下方见新月形瘀斑。下一步最合适？
> 先想 **Baker cyst 破裂 mimics DVT** → **下肢加压超声**（同时排 DVT + 看囊肿）→ 对症 + 治原发关节病。

> [!question]- Q5. 跑步爱好者主诉内侧膝痛，痛点在内侧关节线**下方约 5 cm（两横指）**，无交锁无弹响，McMurray 阴性。最可能诊断？若改为"关节线本身压痛 + 偶尔卡住伸不直"呢？
> 关节线**下方约 5 cm（两横指）+ 无机械症状** = **Pes anserine bursitis 鹅足滑囊炎**（处理：股四头肌+腘绳肌强化拉伸）；关节线**本身压痛 + locking** = **内侧半月板撕裂**。

> [!question]- Q6. 13 岁肥胖男孩诉左膝痛跛行数周，膝查体正常，但左髋内旋受限。最佳初步影像与处理？
> **SCFE 股骨头骺滑脱**（膝痛=referred）→ **双髋正位 + frog-leg lateral X 线** → **非负重 + 手术 pinning**。膝痛查体阴性务必查髋。

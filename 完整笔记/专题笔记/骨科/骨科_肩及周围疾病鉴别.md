# 骨科 肩 (Shoulder) 及周围疾病鉴别 — "总分不清"专攻

> [!info] 用途与来源
> 膝笔记的姊妹篇，同一"**部位 + 机制**"框架。知识源自 [[完整笔记/Peixuan分科笔记/骨科]]（只读分科笔记）+ [[完整笔记/专题笔记/骨科/骨科_HighYield速查总览]] §一。配合 [[完整笔记/专题笔记/骨科/骨科_膝及周围疾病鉴别]] 一起复习。

> [!note]- 缩写对照（全称 中英）
> - **RC** = rotator cuff 肩袖
> - **PMR** = polymyalgia rheumatica 风湿性多肌痛
> - **OA** = osteoarthritis 骨关节炎
> - **ROM** = range of motion 关节活动度
> - **MRI** = magnetic resonance imaging 磁共振成像
> - **NSAIDs** = nonsteroidal anti-inflammatory drugs 非甾体抗炎药
> - **PT** = physical therapy 物理治疗
> - **DM** = diabetes mellitus 糖尿病
> - **CN XI** = cranial nerve XI 第 11 对脑神经（副神经 accessory nerve）

## 〇、第一刀：急性外伤 vs 慢性肩痛 vs 放射/全身

```
肩痛 Shoulder pain
│
├─ 急性外伤/脱位 ─────► A. 脱位（§一）
│     ├─ 外展+外旋受力、最常见 ──► 前脱位 Anterior（伤腋神经）
│     └─ 癫痫/电击/重创、内旋贴身 ► 后脱位 Posterior（灯泡征）
│
├─ 慢性/活动相关肩痛 ─► B. 慢性肩痛谱（§二）
│     ├─ 外展/过顶痛、夜间痛 ──► 肩袖病 / 肩峰下撞击
│     ├─ 主动+被动 ROM 全面受限 ► 冻结肩（粘连性关节囊炎）
│     ├─ 肩前/二头肌沟痛 ──────► 二头肌腱病
│     └─ 老年、被动活动也痛+骨擦 ► 盂肱 OA
│
├─ 肘外/内侧痛（牵涉肩带）─► C. 肘部肌腱病（§三）
│
├─ 神经损伤定位 ──────► D. 神经（§四，腋/臂丛/翼状肩）
│
└─ 双肩僵 + 全身/老年 ─► E. 想 PMR（§五，非骨科，别漏）
```

## 一、A. 肩脱位：前 vs 后（高频对比）

| | **前脱位 Anterior dislocation**（最常见 >95%）| **后脱位 Posterior dislocation**（易漏）|
|---|---|---|
| **机制** | 手臂**外展 + 外旋**受力 | **SEE** = Seizure 癫痫 / Electrocution 电击 / Extreme trauma 重创 |
| **体位** | 手臂轻度外展外旋，不能内旋 | 手臂**内旋 + 内收**贴身，不能外旋 |
| **影像** | 前后位 X 线即见 | 需**腋位/Y 位**；正位见**灯泡征 lightbulb sign** |
| **神经陷阱** | 伤 **axillary nerve 腋神经**（三角肌区/肩外侧麻 + 外展无力）| — |
| **处理** | Closed reduction 闭合复位 + 制动 sling | Closed reduction + sling |

> [!danger] "癫痫后肩痛 = 后脱位"——最经典漏诊
> 癫痫发作 / 电击后肩痛、手臂内旋贴身、不能外旋 → 后脱位。正位 X 线易"看着正常"（灯泡征），务必加腋位片。

> [!warning] 脱位后外展无力：肩袖 vs 腋神经怎么分？
> - **>40 岁脱位复位后外展无力 + 肩外侧感觉完整** = **肩袖撕裂 (rotator cuff tear)**。
> - **外展无力 + 肩外侧 (regimental badge area) 感觉丧失** = **腋神经损伤**。
> - **关键就看感觉**：感觉在 = 肌腱问题；感觉丢 = 神经问题。

## 二、B. 慢性肩痛谱（最容易混的一组）⭐

| 疾病 | 钩子 / 人群 | 主动 vs 被动 ROM | 招牌体征 | 治疗 |
|---|---|---|---|---|
| **肩峰下撞击 Subacromial impingement** | 过顶活动、外展中段(60–120°)痛弧 | 主动痛、被动可 | **Neer / Hawkins+** | PT + NSAIDs ± 激素注射 |
| **肩袖肌腱病/部分撕裂 RC tendinopathy** | 中老年慢性、夜间痛、外展无力 | 主动受限、被动相对保留 | **Drop arm**（冈上肌全层撕裂）/ empty can | PT；全层撕裂或年轻外伤性 → 手术 |
| **冻结肩 Adhesive capsulitis (frozen shoulder)** | **DM / 甲减 / 制动后**、僵硬为主 | **主动 + 被动 ROM 都全面受限** ⭐ | 各向被动活动均受限 | PT 拉伸 ± 激素；多自限(数月-年) |
| **二头肌腱病 Biceps tendinopathy** | 肩前痛、屈肘/旋后痛 | 肩前/二头肌沟压痛 | Speed / Yergason | 休息 + PT |
| **钙化性肌腱炎 Calcific tendinitis** | 急性剧烈肩痛、X 线肩袖区钙化灶 | 痛性受限 | X 线钙化 | NSAIDs ± 抽吸/激素 |
| **盂肱 OA Glenohumeral OA** | 老年、**被动活动也痛 + 骨擦感** | 主+被动均受限、骨擦 | X 线间隙窄+骨赘 | NSAIDs/PT；重则置换 |

> [!tip] 一刀切开"冻结肩 vs 肩袖"
> - **冻结肩 = 主动和被动 ROM 都受限**（关节囊整个挛缩，别人帮你抬也抬不起来）。
> - **肩袖撕裂 = 主动抬不起，但被动能抬**（肌腱断了发不出力，关节本身不僵）。
> - 这条"被动能不能动"是两者最快的分水岭。

> [!warning] 肩袖最忌长期不动 → 继发冻结肩
> 肩袖病要"保持活动 + PT"，长期制动反而诱发 frozen shoulder。

![[L1451.jpg|400]]
*肩关节解剖：肩峰下/三角肌下滑囊 (subacromial/subdeltoid bursa)、冈上肌腱 (supraspinatus tendon)、盂唇 (glenoid labrum)——撞击征 (impingement) 与肩袖病的关键结构。*

## 三、C. 肘部肌腱病：网球肘 vs 高尔夫肘（顺带高频）

| | **外上髁炎 Lateral epicondylitis (网球肘)** | **内上髁炎 Medial epicondylitis (高尔夫肘)** |
|---|---|---|
| **动作** | **抗阻伸腕 / 被动屈腕**痛 | **抗阻屈腕 / 旋前**痛 |
| **压痛点** | 外上髁本身 | 内上髁 |
| **鉴别** | vs 桡管综合征：压痛在外上髁**远端**数 cm（旋后肌处）| — |
| **治疗** | 休息/冰/NSAIDs/counterforce 护具 | 同左 |

![[{6EA44513-D949-4884-84A6-E0348ADA1C37}.png|400]]
*外上髁炎 (lateral epicondylitis, 网球肘)：伸肌总腱起点 (common extensor tendon) 退变。*

![[{141D2621-F902-45B3-8C98-8E9C11B2DA7E}.png|400]]
*反力肘带 (counterforce brace)：置于伸肌腱起点远端分散张力——外上髁炎保守治疗。*

![[{0F654623-2DCA-43A5-8D82-C4C13521D6B8}.png|400]]
*鹰嘴滑囊炎 (olecranon bursitis)：肘后局限性囊性肿胀。*

![[{47DFAB7E-5236-4210-BD0A-0BDE5DDA483B}.png|400]]
*肘窝 (cubital fossa) 内容：肱二头肌腱、肱动脉、正中神经、桡神经及其深/浅支。*

## 四、D. 神经损伤定位（肩带专项）

```
前肩脱位 / 肱骨外科颈 ──── Axillary nerve 腋神经 ── 外展难 + 肩外侧(regimental badge)麻
肱骨干中段(radial groove) ─ Radial nerve 桡神经 ── Wrist drop 腕下垂
```

![[{5E3309D3-4595-4CFB-BBA1-41C1961C6F08}.png|400]]
*肱骨髁上骨折 (supracondylar humeral fracture)：近端碎片前外/前内移位可伤肱动脉 + 正中/桡神经。*

**臂丛 Brachial plexus（产伤/牵拉高频）**

| | **Erb-Duchenne（C5–C6 上干）** | **Klumpke（C8–T1 下干）** |
|---|---|---|
| 招牌 | **Waiter's tip 侍者讨小费手**（肩内收内旋、肘伸、前臂旋前）| **Claw hand 爪形手** |
| 机制 | 新生儿**肩难产 shoulder dystocia** | 上肢猛力上拉 / Pancoast 瘤（+ Horner 综合征）|

**翼状肩 Scapular winging**

| | **内侧翘 medial** | **外侧翘 lateral** |
|---|---|---|
| 神经 → 肌 | long thoracic nerve 胸长神经 → serratus anterior 前锯肌 | accessory nerve (CN XI) → trapezius 斜方肌 |
| 钩子 | 推墙试验阳性、乳腺/腋窝术后 | 耸肩无力、颈后三角术后 |

## 五、E. 别漏：双肩僵 + 全身 = PMR（跨风湿）

> [!warning] PMR 风湿性多肌痛 ≠ 肩关节本身病
> **>50 岁 + 双侧肩 / 髋带 晨僵 (>45 min) + 全身不适 + ESR/CRP 显著↑** → **PMR (polymyalgia rheumatica 风湿性多肌痛)**，对**小剂量激素 (低剂量 prednisone 泼尼松) 戏剧性反应**。
> - 警惕合并 **GCA (giant cell arteritis 巨细胞动脉炎)**：头痛 / 视力变化 / 颌跛行 → 立即大剂量激素 + 颞动脉活检。
> 肩 ROM 主动僵但被动尚可、无关节破坏，和肩袖/冻结肩区分。详见 [[完整笔记/Peixuan分科笔记/风湿]]。

## 六、体征 / 检查 → 诊断速查（肩专项）

| 体征 / 试验 | 阳性 = |
|---|---|
| **Neer / Hawkins** | 肩峰下撞击 Subacromial impingement |
| **Drop arm test** | 肩袖（冈上肌）全层撕裂 |
| **Empty can (Jobe)** | 冈上肌 supraspinatus |
| **Apprehension / Relocation** | 前向不稳 / 既往前脱位 |
| **主动+被动 ROM 全面受限** | 冻结肩（粘连性关节囊炎）|
| **Speed / Yergason** | 二头肌腱病 |
| **肩外侧感觉丧失 + 外展无力** | 腋神经损伤（vs 肩袖=感觉完整）|
| **推墙肩胛内侧翘** | 胸长神经 → 前锯肌（内侧翼状肩）|
| **正位灯泡征 lightbulb sign** | 后肩脱位 |

> [!tip] 体征手法图解（看动作记体征）
> ```
> Neer impingement —— 肩峰下撞击
>   检查者稳住肩胛 + 把患臂被动前屈过顶
>        ╲ 手臂被动抬高
>         ╲___→ 肩峰下空间夹挤
>   阳性 = 过顶弧出现疼痛 → 撞击
>
> Hawkins —— 肩峰下撞击（更敏感）
>   肩 90° 前屈 + 肘 90°，然后【内旋】前臂下压
>   阳性 = 诱发肩前外侧痛 → 撞击
>
> Drop arm —— 冈上肌全层撕裂
>   被动外展到 90° 后让患者自己慢慢放下
>        ──── 90°
>          ↓ 手臂"啪"地掉下、无法控制
>   阳性 = 不能平稳放下 → 肩袖大撕裂
>
> Empty can (Jobe) —— 冈上肌 supraspinatus
>   肩外展 90° + 肩胛面前屈 30° + 拇指朝下(倒空罐)，向下抗阻
>   阳性 = 无力/痛 → 冈上肌病变
>
> Apprehension / Relocation —— 前向不稳（既往前脱位）
>   仰卧，肩外展 90° + 【外旋】
>   阳性 = 患者"要脱位"的恐惧感；按住肱骨头(relocation)恐惧消失
>
> 脱位后"外展无力"分流：
>   摸肩外侧(regimental badge)感觉 →
>      感觉在 = 肩袖撕裂 ；感觉丢 = 腋神经损伤
> ```

## 七、自测（答完再判分）

> [!question]- Q1. 25 岁男癫痫发作后右肩持续疼痛，手臂内旋紧贴身体、不能外旋；前后位 X 线"大致正常"。下一步与诊断？
> **后肩脱位 (posterior shoulder dislocation)**（SEE 之一）→ 加拍**腋位/Y 位**确认（正位灯泡征）→ 闭合复位 + 制动。

> [!question]- Q2. 68 岁男慢性右肩痛，外展无力、夜间痛，肩外侧感觉完整，Drop arm 阳性。最可能诊断与确诊检查？
> **肩袖撕裂 (rotator cuff tear)**（感觉完整排除腋神经）→ **MRI** 确诊。

> [!question]- Q3. 58 岁糖尿病女，右肩进行性僵痛 3 月，自己抬不起、别人帮忙被动也抬不起，各向 ROM 均受限。诊断？
> **冻结肩 / 粘连性关节囊炎 (adhesive capsulitis)**（主动+被动均受限，DM 高危）→ PT 拉伸 ± 激素。

> [!question]- Q4. 新生儿肩难产后右上肢下垂、肩内收内旋、肘伸、前臂旋前（"侍者讨小费"姿势）。损伤定位？
> **Erb-Duchenne 麻痹（C5–C6 上干臂丛损伤）**。

> [!question]- Q5. 72 岁女双肩双髋带晨僵 1 月 >45 min + 乏力，ESR 90。最可能诊断与治疗？需警惕什么合并？
> **PMR 风湿性多肌痛** → 低剂量泼尼松 prednisone 戏剧性缓解；警惕合并 **GCA 巨细胞动脉炎**（头痛/视力/颌跛行 → 大剂量激素 + 活检）。

> [!note]- 缩写对照（全称 中英）
> - **AVN** = avascular necrosis 缺血性坏死（= osteonecrosis 骨坏死）
> - **SCFE** = slipped capital femoral epiphysis 股骨头骺滑脱
> - **LCP** = Legg-Calvé-Perthes disease 儿童股骨头缺血性坏死
> - **DDH** = developmental dysplasia of the hip 发育性髋关节发育不良
> - **OA** = osteoarthritis 骨关节炎
> - **SCD** = sickle cell disease 镰刀细胞病
> - **SLE** = systemic lupus erythematosus 系统性红斑狼疮
> - **MVC** = motor vehicle collision 车祸
> - **ORIF** = open reduction internal fixation 切开复位内固定
> - **THA** = total hip arthroplasty 全髋关节置换
> - **MRI** = magnetic resonance imaging 磁共振成像
> - **ROM** = range of motion 关节活动度
> - **WBC** = white blood cell 白细胞
> - **GTPS** = greater trochanteric pain syndrome 大转子疼痛综合征（含旧称大转子滑囊炎 trochanteric bursitis）

## 〇、第一刀：急性外伤 vs 成人非创伤 vs 儿童跛行 vs 牵涉痛

```
髋/腹股沟痛 Hip pain
│
├─ 急性高能外伤 ─────► A. 创伤（§一）
│     ├─ 车祸屈髋撞击、腿短缩+内旋 ─► 后髋脱位（伤坐骨神经）
│     ├─ 高能外展外旋、腿外展+外旋 ─► 前髋脱位（伤股神经/股动脉）
│     └─ 老人跌倒、腿短缩+外旋 ────► 髋骨折（ORIF）
│
├─ 成人慢性、非创伤 ─► B. 成人髋痛谱（§二）
│     ├─ 腹股沟深痛 + 危险因素(酒/激素) ► AVN 股骨头坏死
│     ├─ 腹股沟痛 + 活动加重休息缓解 ──► 髋 OA
│     ├─ 大转子【外侧】点压痛、侧卧痛 ─► 大转子滑囊炎
│     └─ 跑者腹股沟痛 + 应力史 ───────► 股骨颈应力骨折
│
├─ 儿童/青少年跛行 ─► C. 按年龄分（§三）⭐
│     └─ transient synovitis / LCP / SCFE / 化脓 / DDH
│
└─ 痛是放射来的/神经 ► D. 牵涉 + 神经（§四）
      ├─ 大腿外侧麻 ─► 感觉异常性股痛(meralgia paresthetica)
      └─ SCFE 把痛放射到膝（见膝笔记）
```

## 一、A. 急性创伤：后脱位 vs 髋骨折（"内旋 vs 外旋"高频）

| | **后髋脱位 Posterior hip dislocation** | **髋骨折 Hip fracture** |
|---|---|---|
| **人群/机制** | **MVC 车祸**（屈髋屈膝撞仪表盘）| **老人跌倒**（骨质疏松）|
| **肢位** | 腿**短缩 + 内旋 + 内收** ⭐ | 腿**短缩 + 外旋** ⭐ |
| **神经陷阱** | 伤 **sciatic nerve 坐骨神经**（足下垂/小腿后外侧麻）| — |
| **影像** | X 线；复位后必查神经血管 | X 线；阴性高度疑 → **MRI**（隐匿骨折）|
| **处理** | **紧急闭合复位**（防股骨头 AVN）| **ORIF / 髓内钉 / 置换**，早手术早活动 |

> [!warning] 一字口诀：内旋后脱、外旋骨折
> - 腿**内旋** = 后脱位（"内"→车祸年轻人）
> - 腿**外旋** = 骨折（"外"→老人跌倒）

> [!info] 补充：前髋脱位 (Anterior hip dislocation) + 方向→神经血管（源自 Anki 卡 KP::神经::髋脱位神经）
> 髋脱位按方向分两型，**方向决定伤哪条神经/血管**：
> - **后脱位 (posterior，占 ~90% 最常见)**：屈髋位撞击（车祸撞仪表盘 dashboard）→ 腿**短缩 + 内收 + 内旋**；伤 **坐骨神经 (sciatic nerve)** → 足下垂 (foot drop) / 小腿后外侧麻。
> - **前脱位 (anterior，少见)**：髋**外展外旋**位受力 → 腿**外展 + 外旋 +（略）屈**；伤 **股神经 (femoral nerve) + 股动脉 (femoral artery)** → 伸膝弱 / 股四头肌 (quadriceps) 无力、大腿前 + 小腿内侧麻、远端脉搏↓ / 缺血。
>
> > [!warning] 陷阱：外旋 ≠ 一定是骨折
> > "腿外旋"既见于 **前脱位（年轻高能 + 外展）**，也见于 **髋骨折（老人跌倒 + 短缩）**——靠**人群 + 机制 + 短缩方向**分。§一表里"外旋 = 骨折"口诀针对**老人跌倒**场景，别把年轻车祸的前脱位漏掉。

> [!success] 复位处理原则（前后脱位通用）⭐
> - **脱位无合并骨折 (without associated fracture)** → **紧急闭合复位 (emergent closed / nonoperative reduction)**（急诊室内即可），越快越好 → **降低股骨头缺血坏死 (AVN, avascular necrosis) 风险**。
> - **脱位合并骨折 (with fracture)** → **切开复位 (open / operative reduction)**。
> - 复位后**必查神经血管**（坐骨 / 股神经 + 远端脉搏）+ 复查影像。

> [!danger] Dashboard injury 三联 (仪表盘伤) —— 后脱位的经典连锁伤 ⭐高频
> 车祸时**屈髋屈膝**、膝盖撞仪表盘 (dashboard)，力沿**股骨长轴向后**传导 → 一次外力可同时造成三处伤，stem 给"屈膝撞仪表盘"要三个都想到：
> - **后髋脱位 (posterior hip dislocation)** —— 腿短缩 + 内收 + 内旋，伤 **坐骨神经 (sciatic nerve)**。
> - **后壁髋臼骨折 (posterior wall acetabular fracture)** —— 股骨头向后顶碎髋臼后缘；合并骨折 → 改**切开复位内固定 (ORIF, open reduction and internal fixation)**（见上条原则）。
> - **PCL 后交叉韧带损伤 (posterior cruciate ligament tear)** —— 同一向后的力在膝关节撞出，查体 **posterior drawer / sag sign 阳性**。
>
> > [!tip] 记忆钩子：一条向后的力，三处向后的伤
> > 髋向后脱、髋臼后壁碎、膝 PCL(后交叉)断 —— 全是"**posterior**"。看到"车祸 + 屈膝 + 撞仪表盘"别只答髋脱位，膝也要查 PCL。

## 二、B. 成人非创伤性髋痛谱 ⭐

> [!danger] 反复错警示（2026-07-19 连环踩·本节两题各错 2 次）
> - **髋 OA（Q4564）连续两次都误选"转子滑囊 E"** → 死记：**被动内旋 (passive internal rotation) 诱痛 = 关节内（OA/AVN）**；转子滑囊/GTPS 是**外侧点压痛 + 侧卧痛**，被动内旋**不痛**。
> - **GTPS（Q4546）两次都错**（选过髂胫束、感觉异常性股痛）→ 死记三定位：**GTPS = 大转子外侧点压痛** / **感觉异常性股痛 (meralgia paresthetica) = 大腿前外侧纯感觉麻** / **髂胫束综合征 (ITBS) = 外侧膝痛**。
> - 追踪卡：[[mistakes/uworld-mistakes_2026-07#^Q4564-R0719]]、[[mistakes/uworld-mistakes_2026-07#^Q4546-R0719]]

| 疾病 | 钩子 / 危险因素 | 痛 / 体征定位 | 影像 | 处理 |
|---|---|---|---|---|
| **股骨头坏死 AVN/Osteonecrosis** | **慢性酗酒 / 慢性高剂量激素 / SCD / SLE / 股骨颈骨折后** | 腹股沟深痛 + 髋 ROM↓（外展/内旋）| **X 线低敏**（晚期新月征/股骨头变扁）→ **MRI**（T1 蛇形线 + 骨髓水肿）| 早期保护负重 + 会诊；晚期 **THA** |
| **髋 OA** | **>40、肥胖、糖尿病**、退变；继发于既往关节伤/感染/畸形 | **缓慢进展**；腹股沟痛**放射至大腿/臀/膝**、**活动/负重加重、休息缓解**、**被动内旋 (passive internal rotation / Log roll) 诱痛 = 关节内钥匙**、**无滑膜炎（无红热）** | X 线：间隙窄 + 骨赘 + 软骨下硬化 | 运动 + 减重 + NSAIDs；重则 THA |
| **大转子滑囊炎 Greater trochanteric bursitis** | 中年女、跑者 | **大转子外侧点压痛**、**侧卧压痛**、外展抗阻痛 | 临床 | 相对休息 + NSAIDs + 髋外展强化 |
| **股骨颈应力骨折 Femoral neck stress fx** | 跑者/军训/突增运动、女运动员三联 | 腹股沟痛、负重痛、跳跃痛 | X 线常阴 → **MRI** | **限制负重**（移位风险高，及早处理）|

> [!tip] 三个最易混："深 vs 外 vs 应力"
> - **AVN / 髋 OA = 腹股沟"深"痛**（关节内）；OA 有骨擦 + X 线改变，AVN 早期 X 线常正常靠 MRI。
> - **大转子滑囊炎 = 大转子"外侧"浅压痛 + 侧卧痛**（关节外，不在腹股沟）。
> - **股骨颈应力骨折 = 有应力史 + X 线阴 + MRI 阳**，别当软组织拖延。
> - ➡️ 大转子滑囊炎现多称 **GTPS 大转子疼痛综合征**；与膝外侧的 **ITBS 髂胫束综合征**（同一条髂胫束两端、都练髋外展肌）对照表见 [[完整笔记/专题笔记/骨科/骨科_膝及周围疾病鉴别]] §四。
>

> *大转子区解剖：臀中/小肌 (gluteus medius/minimus) 腱 + 深/浅转子滑囊跨大转子 → GTPS 疼痛来源（UW，双嵌自卡 [[mistakes/uworld-mistakes_2026-06#^Q4546]]）*
>
> ![[{7A164392-9876-4F9B-BEAE-6ECA10CC54AE}.png]]
> *骨关节炎临床特征表 (osteoarthritis)：关节受累 / 晨僵 <30 min / 无全身症状（UW，双嵌自卡 [[mistakes/uworld-mistakes_2026-06#^Q4564]]）*

> [!warning] AVN 的"激素 vs 酒精"剂量陷阱（来源 [[mistakes/uworld-mistakes_2026-06#^Q21345]]）
> **短程激素 ≠ AVN 风险**（需慢性、超生理剂量）；**慢性大量饮酒**才达阈值，风险与"量 + 时长"成正比。**"1 年前用过 5 天泼尼松"是经典干扰项**，而"每日 5–6 杯持续多年"才是真凶。
> - 机制：酒精/激素 → **高甘油三酯 (hypertriglyceridemia) → 骨内脂肪栓塞 + 脂肪细胞肥大 → 髓内压↑ → 股骨头终末血供受阻 → 缺血坏死**。
> - **儿童版股骨头 AVN = LCP**。

## 三、C. 儿童 / 青少年跛行 — 按年龄一刀分 ⭐⭐⭐

| 疾病 | 年龄 | 人群 | 关键钩子 | 影像 / 处理 |
|---|---|---|---|---|
| **DDH 发育性髋发育不良** | 新生儿–婴儿 | 女婴 / 臀位 / 家族史 | **Barlow / Ortolani 弹响** | 超声(<6 月)；Pavlik 吊带 |
| **一过性滑膜炎 Transient synovitis** | 3–8 | **病毒感冒后** | 急性跛行、自限、**不太发热、可负重** | 临床/超声；NSAIDs（自限）|
| **LCP（股骨头缺血坏死）** | 4–10 | **瘦小男孩** | 无创伤渐进跛行、髋/膝痛、内旋外展受限 | **Frog-leg X 线**；休息 + 会诊 |
| **SCFE 股骨头骺滑脱** | 10–15 | **肥胖青少年** | 跛行 + **膝/髋痛(referred)** + **内旋受限** | **双髋正位 + Frog-leg**；**非负重 + 手术 pinning** |
| **化脓性关节炎 Septic arthritis** | 任何 | 任何 | **高热 + WBC 高 + 剧痛拒负重** | 关节穿刺；引流 + IV 抗生素 |

*Barlow / Ortolani 手法：DDH 新生儿筛查（脱位 vs 复位 "clunk"）（UW，双嵌自卡 [[mistakes/uworld-mistakes_2026-06#^Q15845]]）*

![[{CFAD9866-4AEF-4B25-9C4F-DDEDF53B3721}.png]]
*DDH 体征：大腿皮纹不对称 + 患侧多一道（UW，双嵌自卡 [[mistakes/uworld-mistakes_2026-06#^Q15845]]）*

> [!danger] 化脓性 vs 一过性滑膜炎：Kocher 四标准
> 发热 >38.5°C / 不能负重 / ESR >40 / WBC >12,000——符合越多越倾向**化脓性关节炎**（急症，穿刺引流）。一过性滑膜炎"能走、不太烧、自限"。

> [!warning] 特征"重叠"时的下一步 = 双侧髋超声（不是直接穿刺，也不是直接保守）⭐（[[mistakes/uworld-mistakes_2026-07#^Q16295]]）
> 真实病例常**两边特征都沾**（如低热 + 轻度 ESR 像一过性滑膜炎，但拒走 + 白细胞增多伴中性为主像化脓性）——Kocher 项数处于中间、诊断不清时：
> - **下一步 = 双侧髋超声 (bilateral hip ultrasonography)**，查关节积液 (joint effusion)。
>   - **化脓性积液 = 单侧**；**约 25% 一过性滑膜炎是双侧积液** → 做"双侧"才能看出这个分布差异。
>   - **超声优于 CT**：能敏感检出并定性积液，且**无辐射**（儿童首选）。
> - **超声见积液 → 关节穿刺 (arthrocentesis)** 取滑液（WBC >50,000 / 培养 → 化脓性）确诊。
> - 反面：**不能因"像一过性滑膜炎"就直接布洛芬 (ibuprofen) 随访**（拒负重需先排化脓）；腰大肌脓肿 (psoas abscess) 才考虑 CT 腹盆，本例腹软不支持。

> [!tip] SCFE 是膝笔记的镜像点
> 肥胖青少年**膝痛**查体膝正常 + **髋内旋受限** = SCFE（referred pain）。膝笔记 §七也收了这条——双向记忆。详见 [[完整笔记/专题笔记/骨科/骨科_膝及周围疾病鉴别]]。

![[{BC9271B0-FCA0-405A-AC37-B60BB8D8194B}.png|400]]
*Legg-Calvé-Perthes 病 (LCP)：骨盆 X 线——患侧股骨头骨骺**缺血坏死**（碎裂、塌陷变扁），对比正常侧；4–10 岁瘦小男孩。*

![[{D636EAA2-AF3E-4444-92D4-35C4065E685F}.png|360]]
*SCFE 股骨头骺滑脱：股骨颈相对骨骺**向前上滑移**（似冰淇淋球从蛋筒滑落）；10–15 岁肥胖青少年。*

## 四、D. 牵涉痛 + 神经

| 来源 | 钩子 | 鉴别 |
|---|---|---|
| **感觉异常性股痛 Meralgia paresthetica** | **大腿前外侧麻木/烧灼**、肥胖/紧腰带/孕 | **lateral femoral cutaneous nerve 股外侧皮神经**受压；纯感觉、无运动无力、无髋痛 |
| **L2–L4 神经根 / 腰椎** | 大腿前痛 + 反射改变 | 查腰椎；FNST 股神经牵拉+ |
| **SCFE referred 至膝** | 见 §三 | 膝查体阴性必查髋 |

> [!info] 感觉异常性股痛 Meralgia paresthetica — 诱因 + 治疗补全（源自 Nick 卡）
> - **诱因**（除肥胖 / 紧衣腰带 / 孕期外，再加）：**髋关节置换等手术损伤 (surgical injury, eg hip arthroplasty)**、**腰椎前凸增大 (increased lumbar lordosis)**、**糖尿病 (DM)**。
> - **机制**：股外侧皮神经 (LFCN) 在腹股沟韧带处受压的**单纯感觉性单神经病**——**无运动无力**（该神经不含运动纤维）。
> - **治疗**：保守为主 = **避免紧身衣物 + 减重**通常足够；**有氧运动助减重、缓解症状**，但**不需要专业理疗 (skilled PT)**；几乎不手术。

> [!info] 跛行类型也能定位
> - **Antalgic gait 疼痛步态**：患肢站立相缩短（哪痛都可能）。
> - **Trendelenburg gait**：**臀中肌 (gluteus medius) / 臀上神经 (superior gluteal nerve)** 无力 → 站患侧时对侧骨盆下垂。

## 五、体征 / 检查 → 诊断速查（髋专项）

| 体征 / 试验 | 阳性 = |
|---|---|
| **腿短缩 + 内旋 + 内收** | 后髋脱位（伤坐骨神经）|
| **腿外展 + 外旋（年轻高能）** | 前髋脱位（伤股神经 / 股动脉）|
| **腿短缩 + 外旋（老人跌倒）** | 髋骨折 |
| **髋内旋受限（早期）** | 髋 OA / SCFE |
| **大转子外侧点压痛 + 侧卧痛** | 大转子滑囊炎 |
| **Barlow / Ortolani** | DDH 婴儿髋不稳 |
| **Frog-leg lateral X 线** | SCFE / LCP |
| **FABER (Patrick)** | 髋 / 骶髂关节病 |
| **Trendelenburg sign** | 臀中肌 / 臀上神经无力 |
| **大腿前外侧纯感觉麻** | 感觉异常性股痛（股外侧皮神经）|
| **X 线阴性 + MRI 阳（应力史）** | 股骨颈应力骨折 / 早期 AVN |

> [!tip] 体征手法图解（看动作记体征）
> ```
> 急性肢位速判 —— 后脱位 vs 骨折
>   仰卧看患腿：
>      短缩 + 内旋内收 ──► 后髋脱位（车祸，伤坐骨神经）
>      短缩 + 外旋    ──► 髋骨折（老人跌倒）
>
> Log roll（滚腿试验）—— 髋关节内病变最敏感
>   仰卧腿伸直，检查者来回滚动整条腿
>      ◯ 大腿 ←→ 被动内外旋
>   阳性 = 髋内旋诱发腹股沟痛 → 髋关节本身病（OA/AVN/滑膜炎）
>
> FABER / Patrick —— 髋 or 骶髂关节
>   Flexion + ABduction + External Rotation（"4 字"架腿）
>        ┌─ 患腿踝搭对侧膝 ─┐
>   下压膝 + 对侧髂前 →
>   腹股沟痛 = 髋；臀/骶髂痛 = 骶髂关节
>
> Trendelenburg —— 臀中肌 / 臀上神经
>   让患者单腿站立（站患侧）
>      站患侧 → 对侧骨盆【下垂】= 阳性
>   提示患侧 gluteus medius / superior gluteal nerve 无力
>
> Barlow / Ortolani —— 婴儿 DDH
>   屈髋屈膝：
>      Barlow  = 内收+后推 → 股骨头滑【出】(脱位)
>      Ortolani= 外展+上抬 → 股骨头【复入】弹响(clunk)
>
> Frog-leg lateral X 线 —— SCFE / LCP
>   双髋外展外旋呈蛙腿位拍片，显示骺滑脱/股骨头变扁
> ```

## 六、自测（答完再判分）

> [!question]- Q1. 车祸后年轻男右腿短缩、内旋内收，足背伸无力。诊断、神经损伤、首要处理？
> **后髋脱位 (posterior hip dislocation)** → 伤 **坐骨神经 sciatic**（足下垂）→ **紧急闭合复位**（防股骨头 AVN）。

> [!question]- Q2. 55 岁男慢性右腹股沟深痛、髋内旋外展受限；长期大量饮酒；X 线大致正常。下一步最佳检查？
> 疑 **股骨头坏死 AVN**（慢性酗酒高危）→ X 线低敏 → **MRI**（T1 蛇形线 + 骨髓水肿）。

> [!question]- Q3. 4 岁瘦小男孩无外伤渐进性跛行 + 大腿/膝痛，髋内旋外展受限。诊断与首选影像？
> **LCP（Legg-Calvé-Perthes，儿童股骨头缺血坏死）** → **frog-leg lateral X 线**；休息 + 会诊。

> [!question]- Q4. 中年女跑者右髋外侧痛，**侧卧压患侧时痛**、大转子点压痛，髋 ROM 正常、X 线正常。诊断？
> **大转子滑囊炎 (greater trochanteric bursitis)**（关节外、外侧浅压痛）→ 休息 + NSAIDs + 髋外展强化。

> [!question]- Q5. 6 岁男孩拒绝负重、T 39、左髋剧痛、WBC 18,000、ESR 55。最可能诊断与处理？
> **化脓性关节炎 (septic arthritis)**（Kocher 多项阳性）→ **紧急关节穿刺 + 引流 + IV 抗生素**。

> [!question]- Q6. 高能外伤后患腿呈外展外旋位，查体伸膝无力、大腿前及小腿内侧感觉减退、足背动脉搏动减弱。脱位方向？伤及哪些结构？无骨折时如何处理？
> **前髋脱位 (anterior hip dislocation)** → 伤 **股神经 (femoral nerve) + 股动脉 (femoral artery)** → **紧急闭合复位 (emergent closed reduction)**（无合并骨折）；复位后必查神经血管。（对照 Q1 后脱位 = 内旋 + 坐骨神经）

> [!note]- 缩写对照（全称 中英）
> - **FOOSH** = fall on outstretched hand 手掌撑地跌倒
> - **NCS** = nerve conduction study 神经传导检查
> - **AVN** = avascular necrosis 缺血性坏死
> - **CMC** = carpometacarpal joint 腕掌关节
> - **MCP** = metacarpophalangeal joint 掌指关节
> - **DIP** = distal interphalangeal joint 远端指间关节
> - **UCL** = ulnar collateral ligament 尺侧副韧带
> - **OA** = osteoarthritis 骨关节炎
> - **DM** = diabetes mellitus 糖尿病
> - **NSAIDs** = nonsteroidal anti-inflammatory drugs 非甾体抗炎药
> - **ROM** = range of motion 关节活动度

## 〇、第一刀：外伤骨折 vs 神经卡压 vs 腱/腱鞘 vs 挛缩/包块

```
腕/手痛或麻 Wrist/Hand
│
├─ 外伤后(FOOSH/打拳) ─────► A. 骨折（§一）
│     ├─ 年轻 + 鼻烟窝(snuffbox)压痛 ──► 舟骨骨折（X线常阴！防 AVN）
│     ├─ 老人 + 餐叉样畸形 ───────────► Colles 桡骨远端骨折
│     └─ 握拳打击 + 第5掌骨颈 ────────► 拳击者骨折 Boxer's
│
├─ 麻木/刺痛(神经源) ──────► B. 神经卡压（§二）
│     ├─ 拇/食/中指夜间麻、甩手缓解 ──► 腕管综合征（正中神经）
│     └─ 环/小指麻 + 爪形手 ──────────► 尺神经卡压（肘管/Guyon 管）
│
├─ 活动相关痛(腱/腱鞘) ────► C. 腱鞘病（§三）
│     ├─ 新妈妈、拇根桡侧痛、Finkelstein+ ► de Quervain 腱鞘炎
│     ├─ 手指弹响/卡住再"咔"弹开 ───────► 扳机指 trigger finger
│     └─ 老年拇指根部关节痛 ────────────► 第一 CMC OA（拇指腕掌关节炎）
│
└─ 挛缩/包块/伸肌损伤 ─────► D. 结构（§四）
      ├─ 掌腱膜结节索条、4-5指屈曲挛缩 ► Dupuytren 掌腱膜挛缩
      ├─ 腕背圆滑囊性包块、随腕动 ──────► 腱鞘囊肿 ganglion cyst
      ├─ 指尖被顶、远节指不能伸直 ──────► 锤状指 mallet finger
      └─ 拇指 MCP 桡侧被外展掰、UCL 伤 ─► 滑雪者/猎场看守拇指
```

> [!danger] 本图未含的"手部感染急症"——拳击咬伤 Fight bite（指针，详见感染专题）
> **握拳打人牙齿致 MCP 关节处皮裂 (fight bite / clenched-fist injury)** → 极易 **MCP 化脓性关节炎 / 屈肌腱鞘炎**，常见菌 **啮蚀艾肯菌 (Eikenella corrodens)** + 口腔厌氧 → **需外科探查清创 + amoxicillin-clavulanate (阿莫西林-克拉维酸)**，不可一缝了之。完整处理见 [[完整笔记/Peixuan分科笔记/感染]] + 错题 [[mistakes/uworld-mistakes_2026-06#^Q18459]]。

## 一、A. 外伤骨折：舟骨 vs Colles vs 拳击者 ⭐

| 疾病 | 人群 / 机制 | 痛点 / 畸形 | 影像 | 处理 / 陷阱 |
|---|---|---|---|---|
| **舟骨骨折 Scaphoid fracture** | **年轻人 FOOSH** | **鼻烟窝 (anatomic snuffbox) 压痛** | **早期 X 线常阴性** ⭐ | 即使 X 线阴 → **拇人字石膏 (thumb spica) 固定 + 1–2 周复查/MRI**；漏诊 → 近端 **AVN 缺血坏死/不愈合** |
| **Colles 骨折（桡骨远端，背侧移位）** | **老人 FOOSH**（骨质疏松）| **餐叉样畸形 (dinner-fork)** | X 线即见 | 复位 + 石膏；可急性**正中神经**症状 |
| **拳击者骨折 Boxer's fracture** | **握拳打击**（打墙/打架）| **第 5 掌骨颈**肿痛、掌骨头塌陷 | X 线 | 多保守；成角大需复位 |

> [!danger] 舟骨骨折是"X 线阴性也要治"的经典
> 年轻人 FOOSH + 鼻烟窝压痛 → **按舟骨骨折处理（拇人字石膏）即使首片正常**。舟骨血供逆行，漏诊 → 近端缺血坏死 + 不愈合。"片子正常≠没骨折"。

## 二、B. 神经卡压：正中 vs 尺（最容易串的麻木分布）⭐

| | **腕管综合征 Carpal tunnel（正中神经 median）** | **尺神经卡压（肘管 cubital / Guyon 管）** |
|---|---|---|
| **麻木分布** | **拇 + 食 + 中 + 无名桡侧半**（掌侧）⭐ | **环指尺侧半 + 小指** ⭐ |
| **诱因/人群** | 重复腕劳损、**孕 / DM / 甲减 / 肢端肥大** | 肘部长期受压（肘管）/ 腕尺侧受压（Guyon，骑行）|
| **运动/萎缩** | 拇指外展对掌弱、**大鱼际 (thenar) 萎缩** | 骨间肌萎缩、**爪形手 (claw hand)**、Froment 征 |
| **检查** | **NCS 最敏感特异**；Phalen / Tinel / Durkan 敏感性低 | NCS；Tinel 于肘管 / Guyon |
| **治疗** | NSAIDs + **夜间中立位夹板**；顽固 → 松解术 | 改姿势 / 护肘；顽固 → 松解 |

> [!tip] 一句话锁定麻木分布
> - **正中神经 = 桡侧 3 个半指（拇食中 + 无名桡半）+ 大鱼际萎缩** → 腕管。
> - **尺神经 = 尺侧 1 个半指（小指 + 无名尺半）+ 爪形手** → 肘管/Guyon。
> - 腕管"夜间麻醒、甩手 (flick sign) 缓解"是经典钩子；确诊靠 **NCS**，不是 Phalen/Tinel。

![[{D25F4D75-DAB0-47A6-85AE-57494A516F6D}.png|400]]
*正中神经 (median nerve) 支配：运动——拇指屈曲/对掌、第 2–3 指屈曲、腕屈/外展、前臂旋前；皮支——桡侧三个半指掌面 (palmar digital branches) + 大鱼际区 (palmar cutaneous nerve)。腕管卡压即此分布麻木 + 大鱼际萎缩。*

## 三、C. 腱 / 腱鞘病

| 疾病 | 钩子 / 人群 | 痛点 / 体征 | 招牌 | 治疗 |
|---|---|---|---|---|
| **de Quervain 腱鞘炎** | **新妈妈**抱娃 / 拧毛巾、反复拇外展 | 拇指根部**桡侧腕痛**（第一伸肌腱室）| **Finkelstein test 阳性** ⭐ | 改抱姿 + 夹板 + 激素注射 |
| **扳机指 Trigger finger** | 重复抓握、DM | 手指屈曲后**卡住、再"咔"地弹开** | 掌侧 A1 滑车处结节、弹响 | 夹板 / 激素注射；顽固松解 |
| **第一 CMC OA（拇指腕掌关节炎）** | **老年女**、退变 | **拇指根部关节痛**、捏握痛 | grind test+、关节基底膨大 | 夹板 + NSAIDs；重则手术 |

> [!warning] 拇指根痛三选一：de Quervain vs CMC OA vs 舟骨
> - **de Quervain** = 拇外展/Finkelstein 诱发、腱鞘处痛、**新妈妈/反复用拇**。
> - **第一 CMC OA** = 拇指**关节**痛、捏握痛、**老年**、grind test+。
> - **舟骨骨折** = **外伤后**鼻烟窝压痛。
> 三者都在"拇指根附近"，靠"诱因（外伤/反复/退变）+ 痛在腱还是关节"分。

![[{44D5EB5D-526A-45BD-9721-CA6B5EEBA929}.png|400]]
*de Quervain 腱鞘炎 (de Quervain tendinopathy)：第一伸肌腱室的拇长展肌 (abductor pollicis longus, APL) + 拇短伸肌 (extensor pollicis brevis, EPB) 腱鞘增厚，桡骨茎突处痛；Finkelstein 试验阳性。*

## 四、D. 挛缩 / 包块 / 伸肌损伤

| 疾病 | 钩子 | 体征 | 鉴别要点 |
|---|---|---|---|
| **Dupuytren 掌腱膜挛缩** | 北欧裔 / 酒精 / DM / 男性 | **掌腱膜结节 + 索条**，**第 4–5 指渐进屈曲挛缩**、伸直受限 | 是**掌腱膜**纤维化，**非神经**（vs 尺神经爪形手是神经源）、**非弹响**（vs 扳机指）|
| **腱鞘囊肿 Ganglion cyst** | 常见、腕背 | 圆滑、囊性、随腕活动、透光 | 良性、可观察/抽吸；非实性肿瘤 |
| **锤状指 Mallet finger** | 指尖被球/物顶撞（伸肌腱止点撕脱）| **远节指 (DIP) 不能主动伸直**、下垂 | 伸肌腱终末断裂；夹板伸直位固定 |
| **滑雪者/猎场看守拇指 Skier's / Gamekeeper's thumb** | 拇指被强力**外展**（滑雪杖）| 拇指 **MCP 尺侧副韧带 (UCL) 损伤**、捏物不稳 | 不稳明显 → 手术（Stener 病变）|

> [!tip] "屈不开 vs 伸不直 vs 弹响"三连
> - **Dupuytren = 屈曲挛缩、伸不开**（掌腱膜索条，无痛进展）。
> - **Mallet finger = 远节伸不直**（伸肌腱止点断，外伤）。
> - **Trigger finger = 卡住后弹响**（屈肌腱滑车狭窄）。
> - **尺神经爪形手 = 神经源**（环小指掌指过伸、指间屈曲）。

![[{D5A0DA78-0E6B-4EB7-934D-5E23BD678D1A}.png|400]]
*Dupuytren 挛缩 (Dupuytren contracture)：掌腱膜纤维索带致 MCP 关节屈曲挛缩 + 皮肤皱缩。*

## 五、体征 / 检查 → 诊断速查（腕手专项）

| 体征 / 试验 | 阳性 = |
|---|---|
| **鼻烟窝 (snuffbox) 压痛 + FOOSH** | 舟骨骨折（X 线阴也按骨折处理）|
| **餐叉样畸形** | Colles 桡骨远端骨折 |
| **Phalen / Tinel / Durkan**（确诊用 NCS）| 腕管综合征（正中神经）|
| **大鱼际萎缩 + 桡侧 3½ 指麻** | 正中神经 / 腕管 |
| **爪形手 + 环小指麻 / Froment 征** | 尺神经卡压 |
| **Finkelstein test** | de Quervain 腱鞘炎 |
| **手指卡顿弹响** | 扳机指 |
| **拇指根 grind test+** | 第一 CMC OA |
| **远节指不能主动伸直** | 锤状指 mallet finger |
| **拇指 MCP 尺侧不稳** | 滑雪者拇指（UCL 损伤）|

> [!tip] 体征手法图解（看动作记体征）
> ```
> Finkelstein —— de Quervain 腱鞘炎
>   拇指握入掌心成拳 → 腕【尺偏】下压
>        ✊→ 向小指侧掰
>   阳性 = 拇根桡侧第一伸肌腱室剧痛
>
> Phalen / 反 Phalen —— 腕管综合征
>   双腕【掌屈】互抵 60 秒（Phalen）
>   阳性 = 正中神经分布(拇食中)麻木刺痛加重
>   Tinel = 叩腕掌侧正中神经 → 放电样麻
>   Durkan = 直接压腕管 30 秒（最敏感之一）
>
> Froment sign —— 尺神经麻痹
>   拇食指夹纸，检查者抽纸
>   尺神经弱 → 拇内收肌无力 → 改用拇长屈肌【屈拇末节】代偿
>
> 鼻烟窝 (anatomic snuffbox) —— 舟骨骨折
>   拇指伸展时桡骨茎突远端三角凹陷处压痛
>   FOOSH + 此处痛 = 按舟骨骨折治(即使 X 线阴)
>
> Finkelstein vs grind —— 拇根痛分腱/关节
>   Finkelstein(尺偏)痛 = de Quervain(腱鞘)
>   Grind(轴向旋转挤压 CMC)痛 = 第一 CMC OA(关节)
> ```

## 六、自测（答完再判分）

> [!question]- Q1. 24 岁男打球时手掌撑地跌倒，腕桡侧痛，鼻烟窝压痛明显，腕部 X 线未见骨折。下一步最合适？
> 按**舟骨骨折 (scaphoid fracture)** 处理 → **拇人字石膏固定 + 1–2 周复查 X 线 / MRI**（早期 X 线常阴；漏诊致近端 AVN）。

> [!question]- Q2. 50 岁女夜间右手拇、食、中指麻木刺痛，甩手可缓解，大鱼际轻度萎缩。最敏感特异的确诊检查？
> **腕管综合征（正中神经）** → **NCS 神经传导检查**最敏感特异（Phalen/Tinel 敏感性低）；治疗夜间中立位夹板。

> [!question]- Q3. 新手妈妈抱孩子 6 周后右腕桡侧、拇指根部痛，拇指握入掌心后腕尺偏诱发剧痛。诊断与处理？
> **de Quervain 腱鞘炎**（Finkelstein 阳性）→ 改抱姿 + 夹板 + 必要时激素注射。

> [!question]- Q4. 60 岁男手掌出现无痛性结节与索条，第 4、5 指逐渐屈曲挛缩、不能伸直，无麻木无弹响。诊断？与扳机指、尺神经爪形手如何区分？
> **Dupuytren 掌腱膜挛缩**（掌腱膜纤维化）。扳机指=弹响卡顿；尺神经爪形手=神经源 + 环小指麻。

> [!question]- Q5. 篮球击中指尖后，远节手指下垂、不能主动伸直，被动可伸。诊断与处理？
> **锤状指 (mallet finger)**（伸肌腱止点撕脱）→ DIP 伸直位夹板固定。

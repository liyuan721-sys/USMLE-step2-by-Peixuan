## 一、A. 足跟痛 (Heel pain) 鉴别 ⭐⭐⭐

> [!tip] 记忆钥匙：别背 6 列大表 → 记「感觉 → 位置 → 测试 → 年龄」+ 2 个共享陷阱 ⭐
> **第 0 问：有麻 / 烧灼 / 感觉异常吗？（先把神经性一票拨出去）**
> ```
> 足跟痛
> ├─ 有麻/烧灼/感觉异常 → 神经性 = 跗管综合征(Tinel 内踝后+)  ←一票拨出
> └─ 无感觉异常 → 机械性，再按【位置】砍两半：
>       ├─ 后上方(跟腱) → 只问 Thompson
>       │     ├─ 阴性 / 不能踮脚 / 突发"被踢" → 跟腱断裂 ⚠️急症
>       │     └─ 完整 / 止点上2-6cm慢性 → 跟腱病 → 离心训练
>       └─ 正下方(足底/跟骨) → 看招牌
>             ├─ 晨起第一步最痛 ───────► 足底筋膜炎
>             ├─ 儿童8-12 + 双侧 + 挤跟骨痛 ► Sever 病
>             └─ 成人 + 跑量突增 + 挤跟骨痛 + X线阴 ► 应力骨折(→MRI)
> ```
> **2 个共享测试陷阱（这才是"很像"的根源）：**
> - **calcaneal squeeze（挤跟骨两侧）阳性 = Sever + 应力骨折都有** → 靠**年龄**分：儿童 = Sever；成人跑量突增 = 应力骨折。
> - **Thompson（捏小腿肚）= 跟腱断裂 vs 跟腱病的分水岭** → 足**不动**(阴性)=断裂；能跖屈(完整)=跟腱病。
>
> **一句话总纲**：后上方看 Thompson（不动=断、完整=病）；正下方看招牌（晨起第一步=筋膜炎；挤跟骨痛分年龄=儿童 Sever / 成人应力骨折）。

> [!note]- 这张表的排序逻辑（为什么这样排 → 看懂排序=记忆钥匙）
> **纵轴 = 解剖位置「由下往上」**：① 足底筋膜 → ②③ 跟骨（骨突 / 骨体）→ ④⑤ 跟腱（断裂 / 病）。
> **横轴 = 把"共享同一检查"的易混对子排相邻**：
> - ②③ 共享 **calcaneal squeeze 跟骨挤压**（阳性都有 → 靠**年龄**分：儿童 = Sever / 成人跑量突增 = 应力骨折）。
> - ④⑤ 共享 **Thompson 试验**（→ 靠**足动不动**分：不动/阴性 = 断裂 / 完整 = 跟腱病）。
> 
> ① 排最前 = 足跟痛**最常见**病因（教学表惯例"最常见打头"）。
> → 记法：**纵向背位置（下→上），横向成对背"那一个分水岭"**，比背 5 行独立条目省一半力。

| ------------------------------ | ------------------------------------------- | ----------------------------------------------------- | -------------------------------------------------------------------- | ----------------- | ------------------------ |

> [!tip] 跟痛"晨起第一步 vs 挤压 vs 踮不起"三把钥匙
> - **晨起第一步最痛、内侧足底** = **足底筋膜炎**（最常见）。
> - **跟骨两侧一挤就痛** = 应力骨折（成人）/ Sever 病（儿童双侧）。
> - **突发 + 不能踮脚 + Thompson 阴性（捏小腿肚足不动）** = **跟腱完全断裂**（急症，别当扭伤）。

> [!info] 补充①：heel pain 还有「第 5 类」= 神经性 → 跗管综合征（UWorld 列进 DDx, differential diagnosis 鉴别诊断）
> 上表 4 类（足底筋膜炎 / 跟腱病 / 应力骨折 / Sever）都是**机械-骨性**痛，**无感觉异常**。UWorld 的 heel pain 鉴别里还放了 **跗管综合征 (tarsal tunnel syndrome)** —— 它是**神经性**的，钥匙完全不同：
> - **足底烧灼 / 麻木 / paresthesia 感觉异常** + **Tinel 叩击内踝后方（胫后神经 posterior tibial nerve）阳性**（="足底的腕管综合征"）→ 详见本笔记 §四。
> - **分诊加一刀**：先问"**有没有麻 / 烧灼 / 感觉异常？**" → **有 = 神经性（跗管）**；**无 = 机械性（上表 4 类）**。机械痛压痛在骨/筋膜，神经痛带"放电感 + 麻"。

> [!info] 补充②：Windlass 机制（解释足底筋膜炎两个招牌）
> 足底筋膜炎"**足趾（尤其大趾）背伸 → 跟痛加重**"= **Windlass 绞盘机制**：背伸足趾像绞盘一样把足底筋膜**绷紧**，牵拉发炎的跟骨内侧结节止点。
> - 这同一机制也解释"**晨起第一步最痛**"：睡眠中足处于跖屈、筋膜缩短，下床第一步骤然背伸牵张 → 剧痛；走几步"热身"后缓解。
> - 考试钩子：stem 给"**toe dorsiflexion 加重 + 晨起第一步**" → 直接锁 plantar fasciitis。

> [!warning] 跟腱断裂 vs 踝扭伤——最危险的混淆
> 跟腱完全断裂常被当"严重崴脚"耽误：关键是**主动跖屈仍可能部分保留（靠其他屈肌）但 Thompson 试验阴性 + 触及缺损**。任何"突发后跟啪一声 + 踮脚无力"先排跟腱断裂。

## 二、B. 踝扭伤 + 何时拍片（Ottawa rules）
![[{0C32697E-8FF1-449B-8429-689A52B38C77}.png]]

| 要点 | 内容 |
|---|---|
| **最常见** | **外侧踝扭伤**，内翻损伤伤 **ATFL 距腓前韧带**（最常累及）|
| **钩子** | 崴脚后突痛 + 快速肿、外踝前下压痛 |
| **何时拍 X 线** | **Ottawa ankle rules**：外踝/内踝后缘骨性压痛、或第 5 跖骨基底/舟骨压痛、或**不能负重 4 步** → 拍片排骨折 |
| **治疗** | **RICE + air-stirrup brace 充气夹板**（稳定外侧）+ 早期负重/功能锻炼 |

> [!tip] 高位踝扭伤 (high ankle sprain) 别漏
> 下胫腓联合 (syndesmosis) 损伤 = **squeeze test / 外旋应力痛**，恢复更慢，机制多为外旋。和普通外侧扭伤区分。

> [!danger] 处理别过度升级（高频陷阱）
> Ottawa 阴性的低级别外侧扭伤：**软支具 (soft brace) + 早期活动 (early mobilization) + 物理治疗 (PT, physical therapy)**，**不是石膏固定 (cast immobilization)、更不是手术重建**。即使**前抽屉 (anterior drawer) 2+ / 距骨倾斜 (talar tilt) 1+** 提示韧带松弛，初次普通扭伤也远不到手术指征；taping 是**复发预防**工具，不是急性期防慢性 morbidity 的首选。题眼出现 "to prevent morbidity（防慢性不稳）" → 指向**早期康复**。
>
> 关联错题：NBME11A Q153（误选 surgical reconstruction 手术重建）→ [[NBME11_错题本#^Q153]]

## 三、C. 前足 / 跖骨痛

| 疾病 | 钩子 | 痛点 / 体征 | 处理 |
|---|---|---|---|
| **Morton 神经瘤 Morton neuroma** | 中年女、窄高跟鞋 | **第 3–4 跖骨头间烧灼痛 + 趾麻**、**Mulder click** | 宽鞋 + 跖垫；激素注射 / 手术 |
| **跖骨应力骨折（行军骨折 march fracture）** | 跑者/新兵、跑量突增 | **第 2/3 跖骨干**局灶痛、负重痛 | X 线早期阴 → 复查/MRI；保护负重 |
| **痛风 Gout（podagra）** | 高嘌呤/利尿剂 | **第一跖趾 (1st MTP) 急性红肿热剧痛** | NSAIDs / 秋水仙碱 colchicine / 激素 corticosteroid |
| **拇外翻 Hallux valgus / 跖痛症 metatarsalgia** | 慢性、鞋 | 第一跖趾偏斜 / 跖骨头下压痛 | 鞋具调整；重则手术 |

> [!tip] 前足"间隙烧灼 vs 骨干局灶 vs 第一跖趾红肿"
> - **跖骨头之间（3–4）烧灼 + 趾麻 + Mulder click** = **Morton 神经瘤**（神经）。
> - **跖骨干局灶痛 + 应力史 + X 线可阴** = **应力骨折**（骨）。
> - **第一跖趾 (podagra) 急性红肿热痛** = **痛风**（晶体）——详见 [[完整笔记/专题笔记/骨科/骨科_膝及周围疾病鉴别]] §六晶体性关节炎。

## 四、D. 足底 / 趾麻 — 神经卡压

| 疾病 | 钩子 | 鉴别 |
|---|---|---|
| **跗管综合征 Tarsal tunnel syndrome** | 足底麻木/烧灼、夜间重 | **胫后神经 (tibial nerve)** 在内踝后受压；**Tinel 叩击内踝后阳性**；"足底的腕管综合征" |
| **Morton 神经瘤** | 见 §三 | 痛在**跖骨头间**、非整足底 |
| **L5 / S1 神经根** | 腰痛放射至足 | 查腰椎；伴反射/肌力改变 |

## 五、体征 / 检查 → 诊断速查（踝足专项）

| 体征 / 试验 | 阳性 = |
|---|---|
| **晨起第一步跟痛 + 内侧跟结节压痛** | 足底筋膜炎 |
| **Calcaneal squeeze（跟骨挤压）** | 跟骨应力骨折 / Sever 病（儿童双侧）|
| **Thompson (calf squeeze) 无被动跖屈** | 跟腱完全断裂 |
| **Ottawa ankle rules 阳性** | 需拍 X 线排踝/足骨折 |
| **Syndesmosis squeeze / 外旋痛** | 高位踝扭伤 |
| **Mulder click（跖骨头间）** | Morton 神经瘤 |
| **Tinel 内踝后阳性 + 足底麻** | 跗管综合征（胫后神经）|
| **第一跖趾急性红肿（podagra）** | 痛风 |

> [!tip] 体征手法图解（看动作记体征）
> ```
> Thompson (calf squeeze) —— 跟腱断裂 ★
>   俯卧、足悬于床缘，捏小腿肚(腓肠肌)
>      正常：足被动【跖屈】(脚尖下点)
>      断裂：足【不动】= Thompson 阳性 → 跟腱完全断
>
> Calcaneal squeeze（跟骨挤压）—— 应力骨折 / Sever
>   两手从内外侧挤压跟骨
>      ►◄ 挤压
>   阳性 = 跟骨痛 → 跟骨应力骨折（成人）/ Sever 病（儿童双侧）
>
> Ottawa ankle rules —— 何时拍 X 线
>   拍片若：外踝/内踝后缘骨性压痛 │ 第5跖骨基底 or 舟骨压痛
>           │ 受伤即刻 + 急诊均【不能走 4 步】
>   → 任一阳性就拍片排骨折
>
> Anterior drawer / Talar tilt（踝）—— ATFL 距腓前韧带
>   一手固定小腿、一手把足跟向前拉 / 内翻
>   阳性 = 距骨过度前移/内翻松 → 外侧韧带损伤
>
> Syndesmosis squeeze —— 高位踝扭伤
>   小腿中段挤压胫腓骨
>   阳性 = 远端下胫腓联合痛 → high ankle sprain
>
> Mulder click —— Morton 神经瘤
>   横向挤压跖骨头 + 拇指压趾蹼间隙
>   可触/可闻【弹响 click】+ 烧灼趾麻 → 神经瘤
>
> Tinel（内踝后方）—— 跗管综合征
>   叩击内踝后下方(胫后神经)
>   阳性 = 足底放电样麻 → tarsal tunnel
> ```

## 六、自测（答完再判分）

> [!question]- Q1. 45 岁跑者诉右足跟痛，**晨起下床第一步最痛**，走动后减轻，跟骨内侧结节压痛。诊断与一线治疗？
> **足底筋膜炎 (plantar fasciitis)** → 拉伸 + 鞋垫 + NSAIDs；顽固时激素注射。

> [!question]- Q2. 40 岁男打篮球急停时感右后跟"被人踢了一脚"，此后踮脚无力；俯卧捏小腿肚患足不动。诊断与机制？
> **跟腱断裂 (Achilles rupture)** → **Thompson 试验阴性**（捏小腿无被动跖屈）+ 可触缺损 → 手术或制动 + PT。

> [!question]- Q3. 22 岁新兵跑步训练 2 周后右足背中部局灶痛、负重加重，初始 X 线阴性。最可能诊断与下一步？
> **跖骨应力骨折（行军骨折 march fracture，第 2/3 跖骨）** → X 线常阴 → 复查/**MRI**；保护负重。

> [!question]- Q4. 中年女穿窄头高跟鞋，前足第 3–4 跖骨头间烧灼痛伴趾麻，挤压前足有可触及弹响 (click)。诊断？
> **Morton 神经瘤 (Morton neuroma)**（Mulder click 阳性）→ 宽鞋 + 跖垫；必要时激素/手术。

> [!question]- Q5. 崴脚后外踝肿痛，外踝后缘骨性压痛 + 不能连走 4 步。下一步？
> **Ottawa ankle rules 阳性** → 拍**踝 X 线**排骨折；若无骨折按外侧扭伤 RICE + air-stirrup brace。

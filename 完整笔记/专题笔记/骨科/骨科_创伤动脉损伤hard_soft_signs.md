# 创伤性动脉损伤 Hard/Soft Signs 与处理 (Traumatic Arterial Injury)

> [!info] 用途与来源
> 整合 [[完整笔记/专题笔记/骨科/骨科_上肢骨折神经损伤定位]]、[[完整笔记/专题笔记/骨科/骨科_HighYield速查总览]] 中分散的"脉弱→CTA / 无脉→手术"为单一真相源。
> 触发错题：[[mistakes/uworld-mistakes_2026-06#^Q3557]]（锁骨骨折血管软征→CTA）/ [[mistakes/uworld-mistakes_2026-06#^Q17060]]（膝后脱位→测 ABI）。

---

## §一 Hard signs vs Soft signs（核心表）⭐

> [!danger] 硬征 → 手术；软征 → 影像
> **硬征 (hard signs) = 明确动脉损伤 → 立即手术探查**；**软征 (soft signs) = 可能损伤 → 影像（CT 血管造影 CTA）确认/定位**。

| 类别 | 表现 | 处理 |
|---|---|---|
| **硬征 Hard signs** | • 远端缺血**6P**（疼痛 pain / 苍白 pallor / 无脉 pulselessness / 麻痹 paralysis / 感觉异常 paresthesia / 发凉 poikilothermia）<br>• **搏动消失 (absent pulse)**<br>• **活动性出血 / 迅速扩大血肿**<br>• 损伤处**杂音 (bruit) / 震颤 (thrill)** | **立即手术探查** |
| **软征 Soft signs** | • **搏动减弱 (diminished pulse)**<br>• 不明原因低血压<br>• **稳定血肿 (stable hematoma)**<br>• 伤时有记录的出血<br>• 伴**神经缺损**<br>• 伤口邻近大血管走行 | **CT 血管造影 (CTA)** 确认 |

---

## §二 关键骨折/脱位 → 关联血管（解剖反射）

| 损伤 | 关联血管 | 评估 |
|---|---|---|
| **锁骨骨折 / 上位肋 (1–3 肋)** | **锁骨下动脉 (subclavian artery)** + 臂丛 | 脉弱(软征)→CTA；无脉(硬征)→手术（[[mistakes/uworld-mistakes_2026-06#^Q3557]]） |
| **肱骨髁上骨折（小儿）** | 肱动脉 (brachial artery) | 脉/灌注评估 |
| **膝关节脱位 (knee dislocation)** | **腘动脉 (popliteal artery)** | **复位后测 ABI**（[[mistakes/uworld-mistakes_2026-06#^Q17060]]） |
| **膝周/胫骨平台** | 腘动脉/胫前动脉 | 脉 + ABI |
| **骨盆骨折** | 髂内血管丛 | 出血评估 |

## §三 踝肱指数 (Ankle-Brachial Index, ABI) ⭐

> [!warning] 脉搏可触 ≠ 排除血管损伤
> **脉搏查体对腘动脉损伤不敏感** → **膝脱位即使脉可触也要测 ABI**（[[mistakes/uworld-mistakes_2026-06#^Q17060]]）。

| ABI | 解读 |
|---|---|
| **>0.9** + 脉正常 | 基本**排除**临床意义血管损伤 |
| **≤0.9** 或任何软征 | → **CT 血管造影 (CTA) + 血管科会诊** |

```
骨折/脱位旁血管评估
   ├─ 硬征(无脉/喷血/扩大血肿/杂音震颤) → 立即手术
   ├─ 软征(脉弱/稳定血肿/不明低血压/神经缺损) → CTA
   └─ 膝脱位(即使脉可触) → 测 ABI
          ├─ >0.9 → 观察
          └─ ≤0.9 → CTA + 血管会诊
```

---

## §四 易混陷阱

> [!tip] 动脉 vs 静脉损伤
> - **动脉**软征（搏动减弱）→ **CTA**。
> - **静脉**损伤（上肢/下肢**水肿、发绀**）→ 静脉**双扫 (duplex)**；双扫查动脉较 CTA 慢且不敏感。
> - 别因"脉还在/血肿稳定"就放心——这些正是软征，需影像。

# 骨科 手术时机分诊：Emergency vs Urgent → Next Step

> [!info] 用途与来源
> 派生自 Q3564（股骨颈骨折 + 内科不稳 → 先查心肺再择期）+ Q16331（不稳定骨盆骨折出血 → 损伤控制）。
> **核心技能**：遇到骨科外科问题，先判这台手术是 **emergency（急诊）还是 urgent（紧急）** —— 这一步直接决定"**能不能先稳内科 / 复苏，再限时择期手术**"。
> ⚠️ 骨科**急症清单本身**（骨筋膜室 / 开放骨折 / 化脓关节 / 马尾…）已在 [[完整笔记/专题笔记/骨科/骨科_HighYield速查总览]] §〇 列全；**本篇不重复清单，专攻"分诊 → next step"的决策逻辑**。

> [!note]- 缩写对照（全称 中英）
> - **ECG** = electrocardiography 心电图
> - **CXR** = chest X-ray 胸片
> - **troponin** = 肌钙蛋白（心肌损伤标志物）
> - **echo** = echocardiography 超声心动图
> - **CTA** = CT angiography CT 血管造影
> - **I&D** = incision and drainage 切开引流
> - **ex-fix** = external fixation 外固定
> - **DCO** = damage control orthopedics 损伤控制骨科
> - **AVN** = avascular necrosis 缺血性坏死
> - **HTN / T2DM / COPD** = 高血压 / 2 型糖尿病 / 慢性阻塞性肺病

## 一、第 0 步：先分 Emergency vs Urgent（决定"能不能等"）

> [!danger] 这一步先于一切。分错 → next step 全错
> - **Emergency（急诊）** = 等待本身就是伤害（时间窗以**小时**计），**立刻手术**，内科问题**同步/边复苏边处理**，不为"优化内科"而拖。
> - **Urgent（紧急，非急诊）** = 可以等（时间窗以**小时~天**计），**先稳血流动力学 + 优化内科**，再在**限时窗内**做定型手术。

```
遇到需手术的骨科问题
│
├─ 这台是 emergency 吗?（等 = 丢肢体/丢命/永久残疾）
│     │
│     ├─ 是 ──► 立刻手术, 内科同步处理（别为优化而拖）   → §二清单
│     │
│     └─ 否（urgent）──► 看血流动力学 + 内科是否优化      → §三
│              ├─ 血流动力学不稳（休克）► 先复苏 + 查休克原因  → §五
│              ├─ 内科未优化/不稳       ► 先查因优化, 限时窗手术 → §三/§四
│              └─ 稳定且已优化           ► 早手术（限时窗内）
```

---

## 二、Emergency 清单（不能等，边复苏 / 边处理边开）

> [!danger] 这些不走"先稳内科再择期"—— 等待即伤害（明细见速查总览 §〇）
> 详细红旗信号 + 处理阈值见 [[完整笔记/专题笔记/骨科/骨科_HighYield速查总览]] §〇 骨科急症；此处只做"为什么不能等"的分诊记忆。

| 骨科急诊 | 为什么等不了 | 立刻做 |
|---|---|---|
| **骨筋膜室综合征 (compartment syndrome)** | ~6 h 缺血窗，过时肌肉坏死 | 急诊 fasciotomy 筋膜切开 |
| **开放性骨折 (open fracture)** | 感染窗，越拖越脏 | 急诊 I&D 清创 + 立即抗生素 + 破伤风 |
| **化脓性关节炎 (septic arthritis)** | 软骨数小时被酶解破坏 | 急诊关节引流冲洗 + IV 抗生素 |
| **神经血管损伤 / 肢危 (neurovascular / limb-threatening)** | 如膝脱位伤腘动脉，缺血 = 截肢 | 无脉/硬征 → 急诊血运重建；有脉弱 → CTA |
| **马尾综合征 (cauda equina)** | 拖 = 永久大小便 / 性功能障碍 | 急诊 MRI + 手术减压 |
| **不稳定骨盆骨折大出血 (unstable pelvis + hemorrhage)** | 静脉丛持续失血 → 失血性休克 | 损伤控制止血（见 §五 DCO）|

> [!tip] 一个"半急诊"细节：年轻人移位型股骨颈骨折
> 年轻患者**移位型股骨颈骨折**为降低 **AVN（股骨头缺血坏死）** 风险，倾向**尽早（数小时内）**复位固定，比一般髋骨折更急 —— 但仍以血流动力学稳定为前提。老年髋骨折则走 §三 的 urgent 时序。

## 三、Urgent 层：先稳内科 → 限时窗手术（以髋骨折为模板）

> [!warning] "早手术"不等于"立即手术"—— 内科没优化时要先查因
> 髋骨折修复是 **urgent 紧急、但不是 emergency 急诊**。带着未诊断的活动性内科病上麻醉风险极高。

| 血流动力学 | 内科状态 | Next step | 时间窗 |
|---|---|---|---|
| 稳定 | **已优化** | 早手术 | 理想 **<24–48 h** |
| 稳定 | **未优化 / 不稳**（Q3564：疑晕厥 / 心律失常 / 心衰） | 先 **ECG + troponin + CXR（± echo）查因优化** | 优化后手术，**不超 72 h** |
| 不稳（休克） | —— | 先复苏 + 查休克原因（见 §五） | 稳定后再择期 |

> [!danger] 两条铁律
> 1. **绝不因"老 / 虚弱"本身就无限拖**，也**绝不跳过 workup 直接开刀**。
> 2. **>72 h 延迟增加并发症 / 死亡率** —— 优化再快也要在 72 h 内落地手术。

---

## 四、两种"不稳"别混：medically unstable ≠ hemodynamically unstable（Q3564 核心）

> [!danger] 这是本主题最大陷阱：血压正常 ≠ 内科稳定
> Q3564 里病人 **BP 118/76 正常、脉 116 偏快** —— 不算"血流动力学不稳"，但**内科不稳**（一堆未诊断的活动性心肺问题）。

| 概念 | 含义 | 处理导向 |
|---|---|---|
| **血流动力学不稳 (hemodynamically unstable)** | 低血压 / 休克 / 灌注不足 | **立即复苏**（见 §五）|
| **内科不稳 / 未优化 (medically unstable)** | 有**活动性、未诊断或未控制**的内科病 → 抬高围术期风险 | 先**查因 + 优化**，再限时手术 |

> [!tip] 别被"血压正常"骗 —— 把异常都拎出来看（Q3564）
> 不明原因跌倒 + 不记得经过（疑**晕厥 syncope**）；HR 116 + 心悸（疑**心律失常 arrhythmia**）；RR 24 + 需 2L 吸氧 + 肺底啰音 + 一侧呼吸音↓（疑**心衰 / 肺炎 / 胸腔积液**）。
> → 合起来 = **正在进行、还没诊断的心肺问题** = medically unstable。所以选 **A（ECG + 心肌标志物 + 胸片查因优化）**，不是 B（立即手术）。

## 五、血流动力学不稳（休克）时的逻辑

> [!danger] 手术不是复苏手段 —— 稳住血流动力学后再修骨折
> 髋骨折修复是 urgent 不是 emergency，**不会"边休克边开刀"**。顺序：复苏（ABC 气道-呼吸-循环 + 大孔径 IV + 输液 / 输血）→ 找并处理休克原因 → 稳定后再择期修骨折。

> [!warning] 关键：股骨颈骨折本身不会造成休克 —— 休克要去别处找真凶
> 会不会因骨折失血休克，取决于**骨折类型**：

```
出血少, 不会休克 ──► 股骨颈 femoral neck（关节囊内 intracapsular）
出血多, 可致休克 ──► 骨盆 pelvic / 转子下 subtrochanteric / 股骨干 femoral shaft / 多发伤
```

> [!tip] 所以：股骨颈骨折病人若真休克 → 元凶几乎在别处
> 隐匿出血 / GI（胃肠 gastrointestinal）出血 / AAA（腹主动脉瘤 abdominal aortic aneurysm）破裂 / 脓毒症 sepsis / 心源性等 —— 光输液 + 修骨折治不到点子上，必须查真正休克源。

> [!danger] 唯一"不稳 → 立刻手术"成立的情形：手术本身 = 止血（DCO 损伤控制骨科）
> 典型 = **不稳定骨盆骨折持续大出血** → **先上骨盆带 (pelvic binder)** 缩小骨盆容积压迫止血（先于任何需转运的检查），继以 ex-fix 外固定 / pelvic packing 填塞 / 血管栓塞。**止血手术本身就是复苏的一部分。** 股骨颈骨折**不属于**此类。
> ➡️ 完整出血控制阶梯（骨盆带 → 血管栓塞 → 外固定 + 各步指征 + 与 GU 造影的先后）见 [[完整笔记/专题笔记/外科/外科_骨盆骨折出血处理]]。
> 源：[[mistakes/uworld-mistakes_2026-07#^Q16331-R0718]]（反复错 🔴）

## 六、一句话决策链 + 自测

> [!success] 决策链（记不清时默这一段）
> ```
> 第0步：emergency 还是 urgent?（能不能等）
>    ├─ emergency ──► 立刻手术, 内科边复苏边处理
>    └─ urgent ──► 看血流动力学 + 内科优化
>          ├─ 休克 ──► 先复苏 + 查休克原因（股骨颈≠原因, 找别处;
>          │            除非手术=止血 → 骨盆带 / DCO）
>          ├─ 内科未优化 ──► 查因优化 → 限时窗手术（髋骨折 <72 h）
>          └─ 稳定已优化 ──► 早手术（<24–48 h）
> ```
> **一句话**：先分 emergency / urgent（决定"能不能等"），再在 urgent 里按血流动力学 + 内科优化程度决定"等多久"。

> [!question]- Q1. 73 岁男，不明原因跌倒后左髋痛，股骨颈骨折；BP 118/76、HR 116、RR 24、SpO2 95%(2L)、肺底啰音、心悸、不记得是否晕厥。最佳下一步？
> **A：ECG + 心肌标志物 + 胸片（± echo）查跌倒病因并优化术前风险**。他是 **medically unstable（未诊断心肺问题）而非血流动力学不稳**（BP 正常）；髋骨折是 urgent 非 emergency → 先查因优化，再 <72 h 内手术。不是"立即手术"。

> [!question]- Q2. 高能创伤，开书样骨盆骨折，BP 84/52、HR 130（休克）。最佳下一步？
> **先上骨盆带 (pelvic binder) 止血复苏**（DCO 损伤控制）。此时手术 / 止血本身 = 复苏，先于任何需转运的检查（如逆行尿道造影）。与髋骨折"先择期"截然不同 —— 因为**骨盆骨折会大出血、且止血=救命**。

> [!question]- Q3. 长骨骨折后进行性剧痛、被动牵拉痛、张力性肿胀、感觉异常，脉搏尚可。为什么不能"先稳内科再择期"？
> 这是 **骨筋膜室综合征 (compartment syndrome)** = **emergency**，~6 h 缺血窗，过时肌肉坏死。必须**急诊 fasciotomy**，不走 urgent 的"先优化"路径（脉搏消失是晚期，别等）。

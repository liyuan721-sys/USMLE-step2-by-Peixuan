# 骨关节炎 (Osteoarthritis, OA) 完整体系 — 定义 / 鉴别 / 危险因素 / 分布 / 治疗

> [!info] 用途与来源
> 把 **OA (osteoarthritis, 骨关节炎)** 这个跨关节大主题（膝/髋/手/脊柱）从各处汇成单一真相源。素材来自 [[完整笔记/专题笔记/骨科/骨科_膝及周围疾病鉴别]]（§六 膝 OA 管理 + 间室）、[[完整笔记/专题笔记/骨科/骨科_髋及周围疾病鉴别]]（髋 OA）、[[完整笔记/专题笔记/风湿/风湿_HighYield速查总览]]（OA vs RA + 手 OA）+ 错题 Q15631 / Q15632 / Q4564。
>
> ⚠️ Step 2 考 OA 的三大角度：**① OA vs RA / 炎症性关节炎怎么分**、**② 年轻人单关节 OA 问"最显著诱因"（继发性 OA）**、**③ 治疗阶梯走到哪一步（保守 → 置换）**。本篇围绕这三点组织。

> [!note]- 缩写对照（全称 中英）
> - **OA** = osteoarthritis 骨关节炎
> - **RA** = rheumatoid arthritis 类风湿关节炎
> - **PsA** = psoriatic arthritis 银屑病关节炎
> - **ACL** = anterior cruciate ligament 前交叉韧带
> - **MMP** = matrix metalloproteinase 基质金属蛋白酶
> - **DIP / PIP / MCP** = distal / proximal interphalangeal 远端 / 近端指间关节、metacarpophalangeal 掌指关节
> - **NSAIDs** = nonsteroidal anti-inflammatory drugs 非甾体抗炎药
> - **PPI** = proton pump inhibitor 质子泵抑制剂
> - **TKA** = total knee arthroplasty 全膝关节置换
> - **THA** = total hip arthroplasty 全髋关节置换
> - **UKA** = unicompartmental knee arthroplasty 单髁置换
> - **HTO** = high tibial osteotomy 胫骨高位截骨
> - **RF / anti-CCP** = rheumatoid factor 类风湿因子 / anti-cyclic citrullinated peptide 抗环瓜氨酸肽抗体
> - **PT** = physical therapy 物理治疗
> - **ROM** = range of motion 关节活动度
> - **BMI** = body mass index 体重指数
> - **WBC** = white blood cell 白细胞

## 一、OA 是什么：慢性、非炎症、机械性"磨损"关节病

> [!info] 一句话本质
> OA = 关节软骨 (articular cartilage) 退变 + 软骨下骨重塑 + 骨赘 (osteophyte) 形成的**慢性非炎症性**关节病。核心是**机械负荷 > 修复能力**，不是自身免疫炎症。

**招牌临床特征（高频考点）**：

| 特征 | OA 表现 |
|---|---|
| 疼痛节律 | **活动后加重 / 休息缓解**（晚期可静息痛/夜间痛）|
| **晨僵 (morning stiffness)** | **< 30 min** ⭐（RA 是 > 1 小时）|
| 全身症状 | **无**（无发热/乏力/体重↓）|
| 查体 | **硬骨性增大 (bony enlargement) + 骨擦感 (crepitus)**；无温热/海绵样滑膜 |
| 关节液 | 非炎症性，WBC < 2000 |
| X 线四联 | **关节间隙变窄 + 骨赘 + 软骨下硬化 (subchondral sclerosis) + 软骨下囊肿** ⭐ |

> [!tip] X 线"LOSS"四字口诀
> **L** oss of joint space 间隙窄 / **O** steophytes 骨赘 / **S** ubchondral sclerosis 软骨下硬化 / **S** ubchondral cysts 软骨下囊肿。

> [!tip] 膝 OA 查体定位钥匙：痛点多在**内侧关节线 (medial joint line)**
> OA 最常累及**内侧间室 (medial compartment)**（多数人下肢轻度内翻 varus 力线 → 内侧负荷大 → 内侧软骨先磨损），所以膝 OA 的关节线压痛**典型偏内侧**，X 线也是内侧间隙先变窄。
> ⚠️ 但"关节线压痛"本身**不特异**——**半月板撕裂 (meniscus tear)** 也压痛；靠 **OA = 慢性 + 骨擦感 + 无交锁**、**半月板 = 急性扭转史 + locking + McMurray 阳性**区分（详见 [[完整笔记/专题笔记/骨科/骨科_膝及周围疾病鉴别]] §一/§三）。

## 二、OA vs RA —— 最高频的"分清炎症 vs 非炎症"⭐⭐⭐

| | **OA 骨关节炎** | **RA 类风湿关节炎** |
|---|---|---|
| **本质** | 退变 / 机械磨损（非炎症）| 自身免疫滑膜炎（炎症）|
| **晨僵** | 无 / 短（< 30 min）| **> 1 小时**、活动后改善 |
| **疼痛节律** | **活动后加重、休息缓解** | 休息 / 晨起重，活动后改善 |
| **全身症状** | 无 | 发热 / 乏力 / 体重↓ |
| **手部受累** | **DIP (Heberden 结节) + PIP (Bouchard 结节)** ⭐ | **MCP + PIP + 腕**；**不累及 DIP** ⭐ |
| **查体** | 硬骨性增大、crepitus 骨擦感 | 软 / 温热、海绵样滑膜 |
| **X 线** | 间隙窄 + 骨赘 + 软骨下硬化 | **边缘性骨侵蚀 (marginal erosions)** + 关节周围骨质疏松 |
| **抗体** | — | **RF（敏感初筛）、anti-CCP（更特异 + 预后差）** |
| **治疗** | 减重 / 肌力 / PT → 对乙酰氨基酚 (acetaminophen) → NSAIDs → 置换 | flare 短期激素 → **DMARDs 改善病情抗风湿药** |

> [!danger] 反向钥匙：RA **不**累及 DIP
> 题干一旦强调 **DIP (远端指间关节) 受累** → 基本排掉 RA（RA 打 MCP/PIP/腕，spares DIP）→ 往 **OA / PsA 银屑病关节炎 / 痛风 gout** 三选一。
> - **OA** = Heberden (DIP) + Bouchard (PIP)，硬骨性，无炎症。
> - **PsA** = DIP + 指甲点蚀 / 银屑病皮损 / 腊肠指 (dactylitis)。
> - **痛风** = 急性红肿剧痛 + 负性双折光针状晶体。

> [!tip] 手 OA 两结节记忆
> **He**berden = **Hi**gh（**远端** DIP，靠指尖"高处"）；**B**ouchard = **B**elow（**近端** PIP，靠手掌"低处"）。
> ![[风湿_手OA结节.png|400]]
> *手骨关节炎：DIP Heberden 结节 + PIP Bouchard 结节。*

## 三、原发 vs 继发性 OA + 危险因素 —— 年轻人单关节 OA 必问"既往关节伤"⭐

> [!danger] 继发性 OA (secondary osteoarthritis) 头号诱因 = 既往关节损伤/手术（派生自 Q15631）
> **前交叉韧带 (ACL) 损伤后多数患者 10 年内发膝 OA，无论是否手术修复** —— 机制：关节力学改变 + 急慢性炎症 → **基质金属蛋白酶 (MMP) ↑** → 软骨细胞死亡 + 软骨降解。
> **钩子**：**35 岁、BMI 正常、单膝 OA + 高中 ACL 重建史** → "最显著诱因"选**既往 ACL 损伤**。

**OA 危险因素表**：

| 可改变 (modifiable) | 不可改变 (non-modifiable) |
|---|---|
| 久坐、肥胖 (obesity)、职业性关节负荷、糖尿病 (diabetes) | 高龄、女性、家族史、关节力线异常、**既往关节外伤 (prior joint injury)** |

**继发性 OA 其他病因**：关节感染 (septic joint)、炎症性关节病（RA）、先天/获得性骨畸形、神经肌肉无力、代谢病（血色病 hemochromatosis、褐黄病 ochronosis）。

> [!warning] 继发膝 OA 的经典干扰项（错题点，Q15631）
> 题目爱在"年轻人单膝 OA + 既往 ACL 伤"旁边堆假诱因：
> - **维生素 D (vitamin D) 缺乏** → 致**骨软化 (osteomalacia)**（弥漫骨痛 + 肌无力 + 步态异常 + 骨折），**不引起单膝 OA**。❌ 最常被误选。
> - **血糖控制差（A1c 8%）** → 可轻度加速 OA，但仅轻升、影响小，**非"最显著"**。
> - **缺乏运动** → 与原发 OA 相关，但 **< 40 岁无外伤者原发 OA 风险低**。
> - **饮酒** → 仅**重度（> 3 杯/天）**显著增 OA，中度影响轻微。
> - 一句话：**维 D 缺乏管的是骨软化（弥漫骨痛），不是单膝负重痛 OA；ACL 伤史才是钥匙。**

> [!tip] 诊断推理：年轻人单膝 OA 怎么"认出来"——靠阴性体征排除链（派生自 Q15631）⭐
> 题目常**不给 X 线**，OA 全靠线索 + **阴性体征**拼出来。给"既往 ACL 史"是故意勾你想别的病，再用体征逐个堵死：
> 
> | Stem 线索 | 推理 |
> |---|---|
> | 跪 / 上下楼 / 活动后加重 | 机械性负重痛 = OA 节律 |
> | 内侧关节线压痛 | OA 最常累及内侧间室，定位符合 |
> | **Lachman 阴性** | 排除 **ACL 再撕裂 (re-tear)** |
> | **无交锁 (no locking)** | 排除**半月板撕裂 (meniscus tear)** 机械症状 |
> | **无红肿热 + 无全身症状** | 排除**化脓性 (septic) / 晶体性 / RA** 炎症组 |
> | 35 岁 + ACL 重建史 | 解释"为何这么年轻"= 继发性 OA |
> 
> - **钥匙：阴性发现和阳性发现一样重要**——阴性体征逐个排除（再撕裂 / 半月板 / 感染 / 晶体）→ 剩下 OA。
> - 反过来：若 **Lachman 阳性** → 再撕裂；若 **有交锁 + McMurray+** → 半月板；若 **红肿热 + 关节液 WBC↑** → 化脓 / 晶体。

## 四、按关节看 OA（膝 / 髋 / 手 / 脊柱）

```
OA 常见受累关节
│
├─ 膝 (knee) ──── 内侧间室最常坏（内翻 varus / O 型腿）；负重痛、上下楼/跪痛、骨擦感
├─ 髋 (hip) ──── 腹股沟深痛放射至大腿/臀/膝；早期"内旋受限"；活动加重休息缓解
├─ 手 (hand) ─── DIP Heberden + PIP Bouchard + 第一腕掌关节（拇指根 squaring）
└─ 脊柱 (spine) ─ 颈/腰椎骨赘 → 可致椎管狭窄、神经根压迫
```

> [!tip] 髋 OA 的"早期内旋受限"是定位钥匙
> 髋关节本身病变（OA / AVN 股骨头坏死 / 滑膜炎）→ **被动内旋 (passive internal rotation) 诱发腹股沟痛**（关节内征象）；与**大转子滑囊炎**的"外侧浅压痛 + 侧卧痛"（关节外）分清。详见 [[完整笔记/专题笔记/骨科/骨科_髋及周围疾病鉴别]]。源：[[mistakes/uworld-mistakes_2026-06#^Q4564]]（腹股沟痛活动加重 + 被动内旋诱痛 = 关节内 OA；维 D/糖尿病是干扰项）

> [!info] 膝 OA 间室 (compartment) 解剖
> 膝 = 3 个独立承重间室：**内侧 / 外侧** 股骨-胫骨间室 (tibiofemoral) + **髌股 (patellofemoral)**。力线 ↔ 间室：**内翻 (varus, O 型腿) → 内侧间室**；**外翻 (valgus, X 型腿) → 外侧间室**。这决定了后面的术式选择（见 §五）。完整间室+unloader 原理见 [[完整笔记/专题笔记/骨科/骨科_膝及周围疾病鉴别]] §六。

## 五、OA 治疗阶梯：保守走完 → 关节置换（高频 next-step）⭐

> [!tip] 通用治疗阶梯（膝/髋通用，派生自 Q15632）
> 
> ```
> ① 非药物（一线，所有人）：减重 + 规律低冲击活动(步行/游泳) + 肌力强化(膝→股四头肌) + PT
>      │ 症状持续
> ② 药物：对乙酰氨基酚 (acetaminophen) / 局部或口服 NSAIDs (+PPI 或选 celecoxib 塞来昔布)
>           考虑 duloxetine (度洛西汀)、topical capsaicin (辣椒素)
>      │ 症状持续
> ③ 关节腔激素注射（± 透明质酸 hyaluronic acid）  ← 短期缓解
>      │ 保守充分失败 + 显著功能受限
> ④ 关节置换：膝 → TKA 全膝置换 / 髋 → THA 全髋置换（可手术者）
>            慢性疼痛管理（不可手术者）
> ```

> [!danger] 治疗的高频陷阱（错题点）
> - **走完②③仍进行性受限（如只能走 2 个街区）= 置换候选**，别再回头加**系统性糖皮质激素**或**阿片 (opioid)**。
> - **系统性糖皮质激素**：升血糖、副作用大，**糖尿病相对禁忌**；**OA 不用系统激素**（OA 非炎症主导）。
> - **OA = 机械病 → 重机械支持 + 肌力，不是抗炎主导**；这是 OA vs RA 在"治疗哲学"上的根本差异。

> [!tip] 膝 OA 的术式按"间室数"分（分水岭，Q15632）
> - **单间室 OA (unicompartmental)** + 年轻活跃 + 内翻 → **HTO 胫骨高位截骨 / UKA 单髁置换**；保守期可上 **unloader (valgus) 支具**（把负荷卸到健康间室，**前提是有健康间室**）。
> - **双 / 三间室 OA (bi-/tricompartmental)** + 终末期 + 保守失败 → **TKA 全膝置换**；**unloader 支具无效**（无健康间室可卸）。
> - **关节镜半月板切除**对已确立 OA **无益**（反增 OA 风险）；**关节穿刺**仅用于炎症/急性大积液（化脓/血关节），OA 小积液无益。

## 六、自测（答完再判分）

> [!question]- Q1. 35 岁男，左膝负重痛、跪/上下楼加重数月；高中 ACL 重建史；BMI 24、久坐；内侧关节线轻压痛、Lachman 阴性、无交锁；化验示维生素 D 偏低、A1c 8%。最显著的发病诱因？
> **既往 ACL 损伤** → 继发性 OA（ACL 伤后 10 年内多发膝 OA，不论是否手术）。诊断推理：**内侧关节线压痛 = 内侧间室受累，定位符合 OA**；**Lachman 阴性排再撕裂、无交锁排半月板、无炎症征排化脓/晶体** → 锁定 OA。维 D 缺乏→骨软化（弥漫骨痛，非单膝 OA）、A1c 8% 仅轻度加速，都是干扰项。

> [!question]- Q2. 62 岁女，双手 DIP 硬性结节 (Heberden) + 第一腕掌关节痛，无晨僵延长、无全身症状，RF 阴性。最可能诊断？与 RA 如何一眼区分？
> **手 OA**：DIP 受累 + 硬骨性结节 + 无炎症 + RF 阴性。**RA 不累及 DIP**（打 MCP/PIP/腕）+ 晨僵 > 1h + RF/anti-CCP 阳性。

> [!question]- Q3. 68 岁男膝 OA，保守治疗（减重 + PT + NSAIDs + 关节腔激素）充分后仍只能走 2 个街区，X 线示内外侧 + 髌股三间室狭窄。下一步最合适？为什么不是 unloader 支具？
> **TKA 全膝置换**（保守失败 + 三间室终末期）。**unloader 支具无效** —— 三间室都坏，没有健康间室可"卸载"；支具只对单间室 OA + 内翻有效。

# 骨肿瘤影像鉴别速查（Bone Tumor Imaging）

> [!info] 为什么建这张
> 派生自 NBME11A Q084（12 岁女孩膝踝周围无痛骨性肿块 + X 线示近骺板、皮质连续的骨性突起 → osteochondroma）。CK 高频给一组"年龄 + 部位 + 影像形态 + 疼痛/全身症状"让你分**良性 vs 恶性**。

## 一、良性 vs 恶性：一刀分流

> [!tip] 关键钥匙：**皮质/髓腔连续性 + 骨膜反应形态 + 疼痛/全身症状**
> **良性** = 与母骨**皮质及髓腔连续 (cortical & medullary continuity)**、边界清、无侵袭性骨膜反应、多无痛。
> **恶性** = **侵袭性骨膜反应**（葱皮样 onion-skin / Codman 三角 / 日光放射 sunburst）+ 疼痛/夜痛 + 软组织肿块 ± 全身症状。

| 肿瘤 | 良/恶 | 典型年龄/部位 | X 线形态 | 钩子 |
|---|---|---|---|---|
| **骨软骨瘤 (osteochondroma, 骨软骨瘤 / 外生骨疣 exostosis)** | 良性（最常见良性骨肿瘤）| 儿童青少年，长骨**干骺端 (metaphysis)**、近骺板、背离关节 | **皮质/髓腔与母骨连续**、带软骨帽 (cartilage cap) 的骨性突起 | **无痛硬块**；多发 = 遗传性多发外生骨疣 (hereditary multiple exostoses)；观察 |
| **骨样骨瘤 (osteoid osteoma, 骨样骨瘤)** | 良性 | 青少年，长骨骨干 | 小透亮巢 (nidus) + 周围硬化 | **夜间痛，NSAID（非甾体抗炎药）显著缓解** |
| **巨细胞瘤 (giant cell tumor, 骨巨细胞瘤)** | 良性（局部侵袭）| 20–40 岁，长骨**骨骺端 (epiphysis)** | 偏心溶骨、"肥皂泡"样 | 骺板已闭合的成人 |
| **Ewing 肉瘤 (Ewing sarcoma, 尤因肉瘤)** | 恶性 | 儿童/青少年，**骨干 (diaphysis)** | 溶骨 + **葱皮样 (onion-skin) 骨膜反应** | 疼痛 ± 发热、软组织肿块；t(11;22) |
| **骨肉瘤 (osteosarcoma, 骨肉瘤)** | 恶性 | 青少年，**膝周（股骨远端/胫骨近端）干骺端** | **Codman 三角 / 日光放射 (sunburst)** | 疼痛、夜痛、肿块；可继发 Paget 病 (Paget disease) / 放疗 |

```text
X 线看骨性突起：
  皮质 + 髓腔与母骨连续 → 良性 osteochondroma（同源外生）
  侵袭性骨膜反应（onion-skin / Codman / sunburst）+ 疼痛软组织肿块 → 恶性
```

## 二、Q084 主线：无痛 + 骺板附近 + 皮质连续 = osteochondroma

> [!danger] 三联线索直接点名骨软骨瘤
> **青少年 + 膝/踝周围无痛可触肿块 + X 线近骺板、与皮质连续的骨性突起（well corticated，无溶骨/硬化侵袭征）** = **骨软骨瘤 (osteochondroma)**。
> - 本质：**软骨帽覆盖的骨性外生 (cartilage-capped bony exostosis)**，与母骨同源、背离关节生长；骺板闭合后停长。
> - 无全身炎症（无发热寒战）→ 排除感染/恶性。
> - 处理：**无症状观察**；若压迫神经血管、致畸、持续疼痛 → 手术切除。**恶变（→ 软骨肉瘤 chondrosarcoma）罕见**。

## 三、易混陷阱（普适）

> [!warning]
> - **无痛 + 皮质连续 = 良性**，别看到"骨性突起"就慌成恶性。
> - **轻微外伤"引起注意" ≠ 外伤致病**：骨软骨瘤常因小创伤偶然发现，本身无痛（除非压迫）。
> - **膝周干骺端 + 侵袭性骨膜反应 + 夜痛** → 想**骨肉瘤 (osteosarcoma)**，不是骨软骨瘤。
> - **骨干 + 葱皮样骨膜反应 + 发热** → **Ewing 肉瘤**（可拟感染/骨髓炎，别漏）。
> - **夜痛 + NSAID 神效** → **骨样骨瘤 (osteoid osteoma)**。

> [!question]- Q1. 12 岁女孩膝踝周围数月无痛肿块，X 线示近骺板、与皮质连续的骨性突起，无溶骨/硬化。诊断？
> **骨软骨瘤 (osteochondroma)**。无痛 + 骺板附近 + 皮质/髓腔连续 = 最常见良性骨肿瘤；无症状观察即可。

> [!question]- Q2. 三种恶性/侵袭性骨膜反应各对应哪种肿瘤？
> **葱皮样 (onion-skin) → Ewing 肉瘤**；**Codman 三角 / 日光放射 (sunburst) → 骨肉瘤 (osteosarcoma)**。良性骨软骨瘤则是皮质/髓腔与母骨连续、无侵袭性骨膜反应。

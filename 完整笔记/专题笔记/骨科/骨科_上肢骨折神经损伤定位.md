# 上肢骨折 → 神经损伤定位 (Upper-Extremity Fracture & Nerve Injury)

> [!info] 这张笔记解决什么
> UWorld 在"骨折/脱位 + 神经损伤"这块的考法极固定：**给你一个骨折位置（或畸形/X 线），问伴随哪条神经损伤、出现什么缺损**；或反过来**给神经缺损，反推骨折位置**。把"位置 ⇄ 神经 ⇄ 缺损"这条三角链打通就稳了。
>
> 核心规律一句话：**越近端的骨折压越近端的神经** —— 肩→腋，臂→桡，肘→正中/尺，腕→正中（腕管）。

## 二、总览对照表（从近端到远端）

| # | 骨折位置 | 受伤神经 | 标志缺损（stem 钥匙）| 必查/陷阱 |
|---|---|---|---|---|
| 1 | **肱骨外科颈** Surgical neck / **前肩脱位** Ant. dislocation | **腋神经** Axillary | 外展无力 + **肩外侧 (regimental badge) 感觉丧失** | 脱位后外展无力但**感觉完整** = 肩袖损伤，别选腋神经 |

| 5 | **桡骨远端 Colles**（**背侧**移位）/ **Smith**（**掌侧**移位）| **正中神经** Median | **急性腕管**：拇外展弱 + **桡侧 3½ 指**麻 | 餐叉 dinner fork = Colles；铲样 garden spade = Smith |

> [!tip] 加分项：钩骨 Hamate 骨折
> 自行车把/高尔夫球杆撞击钩骨钩 → **尺神经深支（Guyon 管，Guyon canal）** → 骨间肌无力（手指内收/外展），但**前臂尺侧感觉常保留**（背支在更近端分出）。与内上髁的尺神经损伤区分点：钩骨损伤位置更远、感觉相对保留。

## 三、逐条详解

### ① 肱骨外科颈 / 前肩脱位 → 腋神经 Axillary nerve

> [!note] 机制与定位
> - **谁**：老年人跌倒（外科颈骨折）；年轻人手臂**外展 + 外旋**受力（前肩脱位，最常见方向）。
> - **腋神经 (axillary nerve, C5–C6)** 绕肱骨外科颈走 → 此处骨折/脱位最易伤。
> - **运动**：三角肌 (deltoid) → **外展无力**（尤其 15°–90°）。
> - **感觉**：肩外侧"军徽样 (regimental badge)"小片区麻木 —— 这是 stem 最爱给的钥匙。

> [!warning] 经典干扰：腋神经 vs 肩袖损伤
> **脱位/外伤后外展无力**两种都可能：
> - **感觉丧失（肩外侧）** → **腋神经损伤**
> - **感觉完整** + 中老年 → **肩袖损伤 (rotator cuff)**（影像选 MRI）
>
> 一句话：**外展无力先问"肩外侧麻不麻" —— 麻=腋神经，不麻=肩袖。**

![[骨科_前肩脱位X线.png]]
*前盂肱关节脱位 (anterior glenohumeral dislocation)：肱骨头前下移位，是腋神经损伤的高发情景。*

---

### ② 肱骨干中段 → 桡神经 Radial nerve

> [!note] 机制与定位
> - **桡神经 (radial nerve)** 贴 **radial groove（桡神经沟）** 绕肱骨干中段走。
> - 干中段骨折 / 长时间压迫（醉酒压手臂 = **Saturday night palsy 周六夜麻痹**；止血带）→ 桡神经麻痹。
> - **运动**：伸腕/伸指肌瘫 → **Wrist drop 腕下垂**（抬手时腕和指下垂）。
> - **感觉**：**第 1–2 指间虎口背侧 (dorsal first webspace)** 麻。

> [!tip] 桡神经损伤"高 vs 低"
> - **高位（腋窝/螺旋沟以上）**：连**肱三头肌 (triceps)** 都受累 → 伸肘也弱。
> - **低位（前臂，桡管综合征 radial tunnel / 后骨间神经 PIN, posterior interosseous nerve）**：**只伸指弱、伸腕保留**，且**多无感觉缺损**（PIN 是纯运动支）→ 与网球肘 (lateral epicondylitis) 鉴别靠压痛点（见 [[mistakes/uworld-mistakes_2026-06#^Q3575]]）。

---

### ③ 肱骨髁上骨折（儿童）→ 正中神经 / AIN + 肱动脉

> [!danger] 这条最不能漏：先查血管！
> **肱骨髁上骨折 (supracondylar humerus fracture)** 是**儿童最常见肘部骨折**（FOOSH, fall on outstretched hand 跌倒手撑地；伸直型）。骨折远端向后移位 → 同时威胁**正中神经/骨间前神经 (AIN, anterior interosseous nerve)** 与 **肱动脉 (brachial artery)**。
> - **必做**：查**桡动脉搏动 + 毛细血管充盈** → 漏诊血管损伤 → **Volkmann 缺血性挛缩 (Volkmann ischemic contracture)**，永久残疾。
> - **AIN 损伤特征**：纯运动（无感觉缺损）→ **不能做 "OK" 手势**（拇长屈肌 FPL, flexor pollicis longus + 食指深屈肌 FDP, flexor digitorum profundus 瘫，拇食指捏不成圆圈 → 变成"扁 OK"/pinch sign）。

> [!info] 隐匿骨折看脂肪垫征
> 儿童肘外伤 X 线若见 **posterior fat pad sign（后脂肪垫征）** 或异常增大的 anterior fat pad（sail sign 帆征）→ 提示关节积血/**隐匿髁上骨折**，即使骨折线不清也要按骨折处理。

![[骨科_肱骨髁上脂肪垫征.png]]
*隐匿性髁上骨折：anterior + posterior fat pad sign 提示关节积血。*

### ④ 内上髁骨折 → 尺神经 Ulnar nerve

> [!note] 机制与定位
> - **尺神经 (ulnar nerve)** 经肘后**内上髁 (medial epicondyle)** 后方的尺神经沟（"麻筋 funny bone"处）。
> - 内上髁骨折 / 慢性肘管综合征 (cubital tunnel syndrome) → 尺神经损伤。
> - **感觉**：第 5 指 + 第 4 指尺侧半麻。
> - **运动**：手内在肌（骨间肌 interossei、第 3/4 蚓状肌）→ **爪形手 (claw hand)** 倾向（掌指关节过伸、指间关节屈曲，尤以第 4/5 指明显）。

> [!tip] "尺神经悖论 Ulnar paradox"
> 损伤位置**越远（腕部）**，爪形手**越明显**（远端蚓状肌失神经更彻底）；损伤越近（肘部）爪形反而轻 —— 因为指深屈肌也瘫了，屈指力下降。考点上知道"远端损伤爪手更重"即可。

---

### ⑤ 桡骨远端 Colles / Smith → 正中神经（急性腕管）

> [!note] 两种畸形方向相反，神经相同
> 桡骨远端骨折 (distal radius fracture) 都走腕管附近 → 压**正中神经 (median nerve)** → **急性腕管综合征 (acute carpal tunnel syndrome)**。
>
> ```
> Colles 骨折 ── 桡骨远端【背侧 dorsal】移位 ── 餐叉样畸形 (dinner fork)  ── FOOSH 最常见
> Smith  骨折 ── 桡骨远端【掌侧 volar】移位  ── 铲样畸形 (garden spade) ── 屈腕位跌倒/手背着地
> ```
>
> - **正中神经缺损**：拇短展肌 (abductor pollicis brevis) → **拇指对抗阻力外展无力** + **桡侧 3½ 指（拇/食/中/环桡侧）**感觉异常。
> - **时间线钥匙**：初无症状、复位/肿胀后**逐渐出现**手部麻木 = 急性腕管（[[mistakes/uworld-mistakes_2026-06#^Q16328]] 就考这个时间线）。

> [!warning] 别反射成尺神经
> 看到"腕部骨折 + 手麻"很多人反射尺神经 —— 错。**桡骨远端走腕管，压的是正中神经**。尺神经走 Guyon 管（钩骨那侧）。**Colles → 正中，记死。**

---

### ⑥ 舟骨骨折 → 无神经损伤，但防 AVN

> [!danger] 真正的陷阱不是神经，是漏诊 + 坏死
> - **舟骨骨折 (scaphoid fracture)**：FOOSH，**解剖鼻烟窝 (anatomical snuffbox) 压痛**。
> - **早期 X 线常阴性** → 不能因为"片子正常"就放走：**拇人字石膏 (thumb spica) 固定 + 1–2 周复查 X 线 / MRI**。
> - 舟骨血供**逆行（远→近）** → 近端骨折易 **AVN 缺血坏死 (avascular necrosis)** + 不愈合 → 长期腕痛 / 创伤性关节炎。
> - 神经？通常无 —— 这条放进来是为了和上面"骨折就配神经"的反射对冲：**FOOSH + 鼻烟窝痛 = 想血供，不是想神经。**

## 四、神经缺损 → 反查骨折（考试反向题）

```
肩外侧(regimental badge)麻 + 外展难 ──► 腋神经   ──► 外科颈 / 前肩脱位
腕下垂 wrist drop + 虎口背麻      ──► 桡神经   ──► 肱骨干中段
捏不成 OK 圈（+ 必查桡动脉脉搏！）──► 正中/AIN ──► 髁上骨折(儿童)
第 4-5 指麻 + 爪形手              ──► 尺神经   ──► 内上髁 / 肘管
拇外展弱 + 桡侧 3½ 指麻(急性腕管) ──► 正中     ──► Colles / Smith
鼻烟窝压痛 + X 线阴性             ──► (无神经) ──► 舟骨 → 防 AVN
```

## 五、两组最容易串台（你 Q16328 卡的就是这）

> [!question]- ① 正中神经 vs 尺神经，怎么分？
> **看部位 + 看哪几根手指**：
> - **桡侧 3½ 指（拇/食/中/环桡侧）麻 + 拇外展弱** = **正中神经** → 来源：Colles/Smith、急性腕管、髁上(AIN)。
> - **尺侧 1½ 指（小指 + 环指尺侧）麻 + 爪形手/骨间肌弱** = **尺神经** → 来源：内上髁、肘管、钩骨(Guyon)。
> - 反射纠正：**腕部骨折压正中（走腕管），不是尺神经。**

> [!question]- ② 腋神经 vs 桡神经，怎么分？
> **都在"肱骨上段"，但高度差一截**：
> - **外科颈（最上）→ 腋神经**：肩外侧麻 + 外展无力。
> - **干中段 → 桡神经**：腕下垂 + 虎口背麻。
> - 没有"肩外侧麻"、却有"腕下垂"= 桡神经，往下找干中段骨折。

## 六、复习自测（折叠，先别看答案）

> [!question]- Q1. 7 岁儿童 FOOSH 后肘痛畸形，X 线示髁上骨折。查体桡动脉搏动减弱、不能做 OK 手势。下一步最关键的是？
> 评估**血管（桡动脉/灌注）**最优先 —— 髁上骨折伴肱动脉损伤可致 Volkmann 缺血挛缩；同时提示 **AIN（正中）损伤**（不能做 OK 圈）。

> [!question]- Q2. 醉酒后手臂搭椅背睡一夜，醒来抬手时腕下垂、虎口背侧麻，但伸肘正常。受累神经 + 定位？
> **桡神经**（Saturday night palsy）；伸肘正常 = 病变在螺旋沟以下（三头肌支已发出），定位**肱骨干中段 radial groove**。

> [!question]- Q3. 老年女性跌倒后腕痛、餐叉样畸形；复位后逐渐出现拇/食/中指麻 + 拇外展无力。神经？
> **正中神经**（Colles 骨折 → 急性腕管综合征）。

> [!question]- Q4. 肩关节前脱位复位后，三角肌区一小片麻木 + 外展费力。神经？若改为"感觉完整、仅外展无力"，最可能诊断？
> 麻木 → **腋神经损伤**；若**感觉完整**仅外展无力 → **肩袖损伤（rotator cuff）**。

> [!question]- Q5. FOOSH 后腕桡侧痛，鼻烟窝压痛，X 线未见骨折。最佳处理？
> 按**舟骨骨折**处理：**拇人字石膏固定 + 1–2 周复查 X 线/MRI**（早期 X 线可阴性；漏诊 → AVN/不愈合）。

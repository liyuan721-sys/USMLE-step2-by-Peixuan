# 上肢骨折 → 神经损伤定位 (Upper-Extremity Fracture & Nerve Injury)

> [!info] 这张笔记解决什么
> UWorld 在"骨折/脱位 + 神经损伤"这块的考法极固定：**给你一个骨折位置（或畸形/X 线），问伴随哪条神经损伤、出现什么缺损**；或反过来**给神经缺损，反推骨折位置**。把"位置 ⇄ 神经 ⇄ 缺损"这条三角链打通就稳了。
>
> 核心规律一句话：**越近端的骨折压越近端的神经** —— 肩→腋，臂→桡，肘→正中/尺，腕→正中（腕管）。

## 三、逐条详解

### ① 肱骨外科颈 / 前肩脱位 → 腋神经 Axillary nerve

> [!note] 机制与定位
> - **谁**：老年人跌倒（外科颈骨折）；年轻人手臂**外展 + 外旋**受力（前肩脱位，最常见方向）。
> - **腋神经 (axillary nerve, C5–C6)** 绕肱骨外科颈走 → 此处骨折/脱位最易伤。
> - **运动**：三角肌 (deltoid) → **外展无力**（尤其 15°–90°）。
> - **感觉**：肩外侧"军徽样 (regimental badge)"小片区麻木 —— 这是 stem 最爱给的钥匙。

> [!warning] 经典干扰：腋神经 vs 肩袖损伤
> **脱位/外伤后外展无力**两种都可能：
> - **感觉丧失（肩外侧）** → **腋神经损伤**
> - **感觉完整** + 中老年 → **肩袖损伤 (rotator cuff)**（影像选 MRI）
>
> 一句话：**外展无力先问"肩外侧麻不麻" —— 麻=腋神经，不麻=肩袖。**

![[骨科_前肩脱位X线.png|400]]
*前盂肱关节脱位 (anterior glenohumeral dislocation)：肱骨头前下移位，是腋神经损伤的高发情景。*

> [!info] 外展角度 → 主动肌 → 神经（定位"外展无力"在哪一段）
>
> | 外展角度 | 主动肌 | 神经 |
> |---|---|---|
> | **0–15°** | 冈上肌 supraspinatus | 肩胛上神经 suprascapular |
> | **15–90°** | 三角肌 deltoid | **腋神经 axillary** |
> | **>90°（耸肩/上旋肩胛）** | 斜方肌 trapezius | 副神经 accessory (CN XI) |
> | **>90°（固定肩胛防翼状）** | 前锯肌 serratus anterior | 胸长神经 long thoracic |
>
> 助记 **SALT** = **S**erratus **A**nterior → **L**ong **T**horacic。

> [!danger] 外科颈骨折还会断血供 → 肱骨头 AVN
> 旋肱前/后动脉 (anterior/posterior circumflex humeral aa.) 在外科颈周围**吻合环**，发出逆行营养支供肱骨头；**外科颈骨折 → 断此逆行血流 → 肱骨头缺血坏死 (AVN, avascular necrosis / osteonecrosis)**——与**舟骨、股骨头**同机制。X 线：肱骨头**塌陷扁平** + 肩痛 + 活动度↓（隐匿起病）。

### ② 肱骨干中段 → 桡神经 Radial nerve

> [!note] 机制与定位
> - **桡神经 (radial nerve)** 贴 **radial groove（桡神经沟）** 绕肱骨干中段走。
> - 干中段骨折 / 长时间压迫（醉酒压手臂 = **Saturday night palsy 周六夜麻痹**；止血带）→ 桡神经麻痹。
> - **运动**：伸腕/伸指肌瘫 → **Wrist drop 腕下垂**（抬手时腕和指下垂）。
> - **感觉**：**第 1–2 指间虎口背侧 (dorsal first webspace)** 麻。
> - **握力↓ (↓grip strength)**：伸腕瘫 → 屈指肌失去最佳张力位（**伸腕是屈指发力的前提**）→ 抓握无力（非屈肌本身坏）。

> [!tip] 桡神经损伤"高 vs 低"
> - **高位（腋窝/螺旋沟以上）**：连**肱三头肌 (triceps)** 都受累 → 伸肘也弱。
> - **低位（前臂，桡管综合征 radial tunnel / 后骨间神经 PIN, posterior interosseous nerve）**：**只伸指弱、伸腕保留**，且**多无感觉缺损**（PIN 是纯运动支）→ 与网球肘 (lateral epicondylitis) 鉴别靠压痛点（见 [[mistakes/uworld-mistakes_2026-06#^Q3575]]）。

![[L25355.jpg|400]]
*桡神经 (radial nerve) 后面观：绕肱骨干中段 radial groove，沿途支配肱三头肌，肘前分浅桡(感觉)/骨间后 PIN(运动)支。*

#### 肘外侧痛三定位：网球肘 vs 桡管综合征 vs PIN 综合征

> [!warning] 同一条神经穿旋后肌 → 三病靠"压痛点 + 有无无力"分开
> 桡神经过肘分两支：**浅支 superficial radial（纯感觉，管手背桡侧/虎口背）** + **深支 = PIN 骨间后神经（纯运动）**；PIN 穿**旋后肌 (supinator)**，入口为 **Arcade of Frohse（弗罗斯弓，旋后肌腱性入口）**——卡压好发点，正在网球肘病灶**正下方**，故压痛点差几厘米。

![[骨科_肘外侧痛桡管vs网球肘.svg]]
*原创示意图：压痛点 ① 外上髁正上=网球肘；② 远端 3–4 cm（旋后肌）=桡管/PIN。*

| | 网球肘 Lateral epicondylitis | 桡管综合征 Radial tunnel | PIN 综合征 PIN syndrome |
|---|---|---|---|
| **本质** | 伸肌总腱 (ECRB, extensor carpi radialis brevis) 肌腱病 | PIN 受压·以**痛**为主 | PIN 受压·以**运动障碍**为主 |
| **主诉** | 肘外侧痛，握/提物加重 | 前臂背侧酸痛（似顽固网球肘）| **伸指无力 finger drop** |
| **压痛点 ★** | **外上髁正上**（骨突上）| **外上髁远端 3–4 cm**（旋后肌处）| 同桡管区 |
| **诱发动作** | **抗阻伸腕**痛 | **抗阻旋后 / 抗阻中指伸**痛 | 抗阻伸指无力 |
| **运动** | 正常 | 多正常（痛限制）| **伸指弱·伸腕保留但桡偏** |
| **感觉** | 正常 | 正常 | **正常**（PIN 纯运动）|

> [!tip] 三者一句话锁定
> - **网球肘 Lateral epicondylitis**：不累及神经 → **肌腱病**；压痛**外上髁正上**，**抗阻伸腕**痛。
> - **桡管综合征 Radial tunnel**：神经被**刺激** → **痛**为主、不瘫、EMG 常正常；压痛**外上髁远端 3–4 cm**。
> - **PIN 综合征**：神经被**阻断** → **finger drop** 为主、痛不显、EMG 异常；常有**占位病因**（脂肪瘤/腱鞘囊肿/Monteggia）。

> [!tip] Finger drop ≠ Wrist drop（定位高低）
> - **桡神经高位（肱骨干）** → **wrist drop 腕下垂**（连伸腕都瘫）+ 虎口背麻。
> - **PIN 低位（前臂）** → **finger drop 掉指**：掌指关节 (MCP) 伸不起，但**伸腕保留**——因**桡侧腕长伸肌 ECRL (extensor carpi radialis longus)** 在 PIN 分出**之前**由桡神经主干支配、未瘫；而尺侧腕伸肌 ECU (extensor carpi ulnaris) 归 PIN → 瘫 → 伸腕时向**桡侧偏 (radial deviation)**。
> - **手不麻**：浅桡感觉支在更近端已分出、不走 PIN。
>
> 记忆钩：**PIN = Posterior、纯运动、finger drop；伸腕桡偏 + 手不麻；压痛比网球肘往远 3 cm。**

> [!question]- 为什么 PIN 瘫了"伸腕还保留、却桡偏"？（机制深挖）
> 关键：**伸腕主力 ECRL (extensor carpi radialis longus, 桡侧腕长伸肌) 在桡神经分叉【之前】就由主干发出 → PIN 够不着它。**
> ```
> 桡神经主干 ─ 先发出 → 三头肌 / 肱桡肌 / ★ECRL(伸腕主力)
>      │  到肘才分叉
>      ├─ 浅支(纯感觉)
>      └─ 深支 = PIN ─ 穿旋后肌 → ECU(尺侧腕伸) / 伸指肌 / 伸拇肌 …
> ```
> 伸腕有 **3 条肌**，归属不同 → 净效果"伸腕保留但桡偏"：
> - **ECRL**（桡侧腕长伸，主干分叉前）+ **ECRB**（桡侧腕短伸 ext. carpi radialis brevis，多为主干/分叉处）→ **保留**，止于第 2/3 掌骨基底 → 把腕拉向**桡侧**。
> - **ECU 尺侧腕伸肌 (extensor carpi ulnaris)**（归 PIN）→ **瘫** → 失去尺侧反向平衡。
> - 合力 = 能伸腕，但**偏桡侧 (radial deviation)**。
> - ⚠️ **PIN 不是"只管伸指"**：它还管 ECU(伸腕) + 伸拇 + 旋后；只是桡侧两条伸腕肌(ECRL/ECRB)不归它，伸腕才没全废。
> - 对照：**桡神经高位（肱骨干/螺旋沟）** 病变在 ECRL 发出**之前** → 连 ECRL 都瘫 → **真 wrist drop 腕下垂**（伸腕全失）+ 虎口背麻。
>
> 一句话：**ECRL 在损伤"上游" = 伸腕保留 (PIN/finger drop)；ECRL 也瘫 = 腕下垂 (高位)。**

> [!tip] finger drop 累及范围：全指 2–5 + 拇指，掉在 MCP
> - PIN 包办**所有伸指肌（伸指总肌 + 示指/小指固有伸肌）+ 伸拇 / 外展拇肌群** → **食、中、环、小 4 指 + 拇指的掌指关节 (MCP) 一起伸不起** = finger drop（不是某几根，是全部）。
> - **掉的是 MCP、不是 IP**：指间关节 (PIP/DIP) 仍能伸 —— IP 伸靠**手内在肌（蚓状肌 / 骨间肌，正中 / 尺神经支配）**，不归 PIN。查体让病人"只伸末两节"仍能做。
> - vs 方向相反的屈肌病：**爪形手 claw hand（尺神经，4–5 指）、祝福手 benediction（正中高位，2–3 指）** 是屈/伸失衡；**PIN 是纯"伸不起"（全指 + 拇 MCP）**。

> [!note] 同神经同位置，为什么桡管"只痛"、PIN"只瘫"？
> 它俩**不是两个病，是同一卡压谱系 (spectrum) 的两端**——区别在压迫**程度/性质**，不在位置：
> - **桡管综合征 = 神经被"刺激"(excitation → 痛)**：轻/动态/间歇卡压 → 痛觉纤维被惹毛 → **痛**，但运动传导未阻断 → **不无力**，EMG 常正常。
> - **PIN 综合征 = 神经被"阻断"(block → 瘫)**：重/固定/占位卡压 → 运动轴突传导阻滞/变性 → **finger drop**，而这种麻木性压迫**痛反而不显**，EMG 异常。
>
> 一句话：**桡管 = 刺激态 (Pain)，PIN = 阻断态 (Palsy)。**

##### 临床情景：痛剧本 vs 瘫剧本

> [!example]- 桡管综合征 stem = "治不好的网球肘"（痛剧本）
> 反复**前臂旋前旋后/抓握**职业（机械工/流水线/球拍运动/乐手），**肘外侧深部酸痛数月、按网球肘治疗（含激素注射）无效**；压痛在**外上髁远端 3–4 cm**，**抗阻旋后 / 抗阻中指伸**诱发痛；**无力、麻木缺如**，EMG 正常。
> → 钥匙：**痛来诊 + 激素注射无效 + 压痛偏远**。

> [!example]- PIN 综合征 stem = "无痛性掉指"（瘫剧本）
> **进行性伸指无力**（手指抬不起 finger drop）、**几乎不痛不麻**；查体 **掌指关节 (MCP) 伸不起但伸腕保留 + 桡偏**、感觉正常；背后常有**肘部肿物（脂肪瘤 lipoma / 腱鞘囊肿 ganglion）、类风湿关节炎 (RA, rheumatoid arthritis) 肘滑膜炎**，或外伤 **Monteggia 骨折**（尺骨近端骨折 + 桡骨头脱位）。
> → 钥匙：**瘫来诊 + finger drop + 找占位/Monteggia**（影像查 MRI / X 线）。

> [!warning] 3 个常考干扰
> - **桡管 ↔ 网球肘**：都肘外侧痛 → 看**压痛点（外上髁正上 vs 远端 3–4 cm）+ 激素注射有无效**。
> - **PIN ↔ 高位桡神经麻痹**：都"伸不起" → 看**伸腕保留 (finger drop) vs 腕下垂 (wrist drop) + 虎口背麻有无**。
> - **PIN ↔ 尺神经爪形手**：PIN = **全指 MCP 伸不起**（伸肌瘫）；尺神经 = **4-5 指爪形**（内在肌瘫 → 长伸肌在 MCP、长屈肌在 IP 失拮抗），方向相反。

### ③ 肱骨髁上骨折（儿童）→ 正中神经 / AIN + 肱动脉

> [!danger] 这条最不能漏：先查血管！
> **肱骨髁上骨折 (supracondylar humerus fracture)** 是**儿童最常见肘部骨折**（FOOSH, fall on outstretched hand 跌倒手撑地；伸直型）。骨折远端向后移位 → 同时威胁**正中神经/骨间前神经 (AIN, anterior interosseous nerve)** 与 **肱动脉 (brachial artery)**。
> - **必做**：查**桡动脉搏动 + 毛细血管充盈** → 漏诊血管损伤 → **Volkmann 缺血性挛缩 (Volkmann ischemic contracture)**，永久残疾。
> - **AIN 损伤特征**：纯运动（无感觉缺损）→ **不能做 "OK" 手势**（拇长屈肌 FPL, flexor pollicis longus + 食指深屈肌 FDP, flexor digitorum profundus 瘫，拇食指捏不成圆圈 → 变成"扁 OK"/pinch sign）。

> [!info] 隐匿骨折看脂肪垫征
> 儿童肘外伤 X 线若见 **posterior fat pad sign（后脂肪垫征）** 或异常增大的 anterior fat pad（sail sign 帆征）→ 提示关节积血/**隐匿髁上骨折**，即使骨折线不清也要按骨折处理。

![[骨科_肱骨髁上脂肪垫征.png|400]]
*隐匿性髁上骨折：anterior + posterior fat pad sign 提示关节积血。*

![[{E0ED3B61-1694-4677-9271-E733939B7944}.png|400]]
*髁上骨折 (supracondylar fracture) 移位方向定损伤：近折端**前外侧移位 (anterolateral) → 桡神经 (radial nerve)**、**前内侧移位 (anteromedial) → 正中神经 (median nerve)**，两种都可拉伤/压迫**肱动脉 (brachial artery)**——复位前后必查桡动脉搏动。*

![[{E03500A3-3DFE-4B0E-A5D1-50C9C74D9953}.png|400]]
*肘窝 (cubital fossa) 解剖：上界两髁连线、外界肱桡肌、内界旋前圆肌；内容物含**肱动脉 brachial artery + 正中神经 median nerve**——髁上骨折正威胁此二者。*

### ④ 内上髁骨折 → 尺神经 Ulnar nerve

> [!note] 机制与定位
> - **尺神经 (ulnar nerve)** 经肘后**内上髁 (medial epicondyle)** 后方的尺神经沟（"麻筋 funny bone"处）。
> - 内上髁骨折 / 慢性肘管综合征 (cubital tunnel syndrome) → 尺神经损伤。
> - **感觉**：第 5 指 + 第 4 指尺侧半麻。
> - **运动**：手内在肌（骨间肌 interossei、第 3/4 蚓状肌）→ **爪形手 (claw hand)** 倾向（掌指关节过伸、指间关节屈曲，尤以第 4/5 指明显）。
> - **Froment 征 (Froment sign)**：拇内收肌 (adductor pollicis) 瘫 → 用力捏纸时改用拇长屈肌 (FPL) 代偿 → **拇指 IP 关节屈曲** = 尺神经损伤阳性。

> [!tip] "尺神经悖论 Ulnar paradox"
> 损伤位置**越远（腕部）**，爪形手**越明显**（远端蚓状肌失神经更彻底）；损伤越近（肘部）爪形反而轻 —— 因为指深屈肌也瘫了，屈指力下降。考点上知道"远端损伤爪手更重"即可。

> [!tip] 屈腕桡偏 vs 尺偏（高位损伤的定位线索，别和 PIN 混）
> 看"瘫掉哪条腕屈肌 → 腕被健侧拉向对侧"：
> - **尺神经高位**：尺侧腕屈肌 FCU (flexor carpi ulnaris) 瘫 → **屈腕时桡偏 (radial deviation)**。
> - **正中神经高位**：桡侧腕屈肌 FCR (flexor carpi radialis) 瘫 → **屈腕时尺偏 (ulnar deviation)**。
> - 对照 **PIN**：ECU 瘫 → **伸腕时桡偏**（注意是伸腕，不是屈腕）。
> 一句话：**屈腕看屈肌、伸腕看伸肌；哪侧肌瘫，腕偏对侧。**

![[L23776.jpg|400]]
*尺神经 (ulnar nerve) 支配区：运动=手内在肌(骨间肌/3-4 蚓状肌/拇内收肌)、4-5 指屈；感觉=尺侧 1½ 指(掌背侧)。*

![[L25348.jpg|400]]
*尺神经走行：经肘后内上髁→前臂(尺侧腕屈肌 FCU / 指深屈肌 FDP 尺侧半)→Guyon 管入手，深支支配骨间肌。*

### ⑤ 桡骨远端 Colles / Smith → 正中神经（急性腕管）

> [!note] 两种畸形方向相反，神经相同
> 桡骨远端骨折 (distal radius fracture) 都走腕管附近 → 压**正中神经 (median nerve)** → **急性腕管综合征 (acute carpal tunnel syndrome)**。
>
> ```
> Colles 骨折 ── 桡骨远端【背侧 dorsal】移位 ── 餐叉样畸形 (dinner fork)  ── FOOSH 最常见
> Smith  骨折 ── 桡骨远端【掌侧 volar】移位  ── 铲样畸形 (garden spade) ── 屈腕位跌倒/手背着地
> ```
>
> - **正中神经缺损**：拇短展肌 (abductor pollicis brevis) → **拇指对抗阻力外展无力** + **桡侧 3½ 指（拇/食/中/环桡侧）**感觉异常。
> - **时间线钥匙**：初无症状、复位/肿胀后**逐渐出现**手部麻木 = 急性腕管（[[mistakes/uworld-mistakes_2026-06#^Q16328]] 就考这个时间线）。

![[骨科_腕管横断面.svg]]
*原创横断面：正中神经在管内、最贴屈肌支持带 → CTS / Colles 最先压它（拇外展弱 + 桡侧 3½ 指麻）；尺神经在管外 Guyon 管 → 不累及。*

> [!warning] 别反射成尺神经
> 看到"腕部骨折 + 手麻"很多人反射尺神经 —— 错。**桡骨远端走腕管，压的是正中神经**。尺神经走 Guyon 管（钩骨那侧）。**Colles → 正中，记死。**

> [!tip] 正中神经手内在肌 = LOAF + 返支陷阱
> 正中神经在手部支配 **LOAF**（鱼际肌群，主要经**返支 recurrent branch**）：
> - **L** = Lateral 2 lumbricals 桡侧 2 蚓状肌
> - **O** = Opponens pollicis 拇对掌肌
> - **A** = Abductor pollicis brevis 拇短展肌
> - **F** = Flexor pollicis brevis 拇短屈肌
>
> **返支损伤（掌部浅表割伤）** → **猿手 ape hand**（鱼际萎缩、拇不能对掌/外展）+ **无感觉缺损**（返支纯运动）。
> 对比腕管综合征 (CTS)：压正中**主干** → 桡侧 3½ 指麻 + 鱼际无力；**返支割伤** → 只鱼际无力、**不麻**。

![[L23659.jpg|400]]
*Colles 骨折：桡骨远端**背侧移位** → 餐叉样畸形 (dinner fork deformity)。*

![[L1884.jpg|400]]
*正中神经 (median nerve) 支配区：运动=拇对掌/屈、2-3 指屈、屈腕、前臂旋前；感觉=桡侧 3½ 指掌侧。*

### ⑥ 舟骨骨折 → 无神经损伤，但防 AVN

> [!danger] 真正的陷阱不是神经，是漏诊 + 坏死
> - **舟骨骨折 (scaphoid fracture)**：FOOSH，**解剖鼻烟窝 (anatomical snuffbox) 压痛**。
> - **早期 X 线常阴性** → 不能因为"片子正常"就放走：**拇人字石膏 (thumb spica) 固定 + 1–2 周复查 X 线 / MRI**。
> - 舟骨血供**逆行（远→近）** → 近端骨折易 **AVN 缺血坏死 (avascular necrosis)** + 不愈合 → 长期腕痛 / 创伤性关节炎。
> - 神经？通常无 —— 这条放进来是为了和上面"骨折就配神经"的反射对冲：**FOOSH + 鼻烟窝痛 = 想血供，不是想神经。**

## 五、两组最容易串台（你 Q16328 卡的就是这）

> [!question]- ① 正中神经 vs 尺神经，怎么分？
> **看部位 + 看哪几根手指**：
> - **桡侧 3½ 指（拇/食/中/环桡侧）麻 + 拇外展弱** = **正中神经** → 来源：Colles/Smith、急性腕管、髁上(AIN)。
> - **尺侧 1½ 指（小指 + 环指尺侧）麻 + 爪形手/骨间肌弱** = **尺神经** → 来源：内上髁、肘管、钩骨(Guyon)。
> - 反射纠正：**腕部骨折压正中（走腕管），不是尺神经。**

> [!question]- ② 腋神经 vs 桡神经，怎么分？
> **都在"肱骨上段"，但高度差一截**：
> - **外科颈（最上）→ 腋神经**：肩外侧麻 + 外展无力。
> - **干中段 → 桡神经**：腕下垂 + 虎口背麻。
> - 没有"肩外侧麻"、却有"腕下垂"= 桡神经，往下找干中段骨折。

## 六、骨折并发症（Compartment / CRPS / 脂肪栓塞）

> [!danger] 骨筋膜室综合征 Acute compartment syndrome
> - **机制**：骨筋膜室内压↑（长骨骨折、挤压伤 crush injury、再灌注后大量补液）→ 灌注↓。
> - **早期 5P**：**P**ain（进行性、**被动牵拉痛 pain with passive stretch** 最早出现）→ **P**aresthesia 麻木 → **P**allor 苍白 → **P**ulselessness 无脉（晚期、少见）→ **P**aralysis 瘫痪（晚期）。⚠️ **等脉搏消失就晚了**。
> - **确诊**：测室内压 → **舒张压 − 室内压 ≤ 30 mmHg** 阳性。
> - **处理**：**紧急筋膜切开 fasciotomy**（小时级急症）。与髁上骨折的 **Volkmann 缺血性挛缩**同源（Volkmann = 未处理的前臂室综合征终末结局）。

> [!warning] CRPS 复杂区域疼痛综合征 (complex regional pain syndrome / RSD, reflex sympathetic dystrophy 反射性交感营养不良)
> - 外伤/手术**数周后**：疼痛**与损伤不成比例 (pain disproportionate)** + 灼痛 + **异常性疼痛 allodynia**（轻触即剧痛）+ 局部红肿/出汗/皮温异常 → 晚期**斑片状骨脱矿 (patchy osteopenia)**。
> - 机制：外周 + 中枢敏化 + 自主神经紊乱 + 炎症（非单纯神经离断）。

> [!info] 脂肪栓塞综合征 Fat embolism syndrome
> - **长骨/骨盆骨折**后 12–72 h：突发**低氧 hypoxia + 神经症状 + 瘀点 petechiae** 三联。多见股骨骨折；以支持治疗为主。
> - 完整三联 + 诊治 + vs 血栓 PE / DAI 鉴别（单一真相源）见 [[完整笔记/专题笔记/pulmonary/pulmonary_肺栓塞PE]] §四。

## 七、手姿三联 + 肩胛带神经（定位加分）

> [!tip] 手姿三联：一眼定位神经（⚠️ 祝福手是正中，不是尺！）
> ```
> Ape hand 猿手 ──────── 正中神经(高位) ── 鱼际萎缩、拇不能对掌，手掌平摊似猿(静息)
> Benediction 祝福手 ──── 正中神经(高位) ── 试图握拳时 2-3 指(食/中)伸直不能屈
> Claw hand 爪形手 ────── 尺神经 ── 静息即 4-5 指 MCP 过伸 / IP 屈
> ```
> - **猿手 + 祝福手 = 正中神经**：猿手=静息态鱼际萎缩；祝福手=握拳态食/中指伸直。
> - **爪形手 = 尺神经**（4-5 指）。
> - **Froment 征**见 §三④（尺神经拇内收代偿屈拇）。

> [!warning] 爪形手 vs 祝福手：别背反（机制 + 时机）
> 两者看着像（都"4-5 屈、2-3 直"），但**神经相反、出现时机相反**：
>
> | | 爪形手 Claw hand | 祝福手 Benediction |
> |---|---|---|
> | **神经** | 尺神经 ulnar | 正中神经 median（高位）|
> | **时机** | **静息**就有 | **试图握拳**才显 |
> | **哪几指** | 4-5 指（MCP 过伸/IP 屈）| 2-3 指（食/中伸直）|
> | **机制** | **内在肌（蚓状肌/骨间肌）瘫** → 长伸肌在 MCP、长屈肌在 IP 失拮抗 | 正中**屈肌（指浅屈肌 + 指深屈肌桡侧半 + 拇长屈肌）瘫** → 食/中屈不动 |
>
> - 都是"**少了谁 → 失衡**"，不是"屈肌过强"；爪形手在 MCP 那里其实是**伸肌**赢。
> - **尺神经悖论 ulnar paradox**：尺神经**低位(腕)** → 指深屈肌完好 → 爪形**更明显**；**高位(肘)** → 指深屈肌也瘫 → 爪形反而轻。

> [!note] 肩胛带神经（非骨折，但同区高频）
> - **翼状肩胛 winged scapula 两型**：
>   - **内侧翼**（推墙时肩胛内缘翘起）→ **胸长神经 (long thoracic n.)** / 前锯肌 → 典型**乳房切除 + 腋窝淋巴结清扫**后。
>   - **外侧翼**（耸肩无力、举臂过头困难）→ **副神经 (CN XI, spinal accessory n.)** / 斜方肌 → 典型**颈后三角淋巴结活检/手术**后。
> - **腋神经**还可因**四边孔综合征 (quadrilateral space syndrome) / 背包麻痹 (backpack palsy)** 受损（非仅骨折/脱位）。
> - 臂丛速记（详见 [[完整笔记/专题笔记/骨科/骨科_HighYield速查总览]] §六）：**Erb-Duchenne (C5–C6)** = waiter's tip 侍者讨小费手（肩难产）；**Klumpke (C8–T1)** = claw hand + Horner 综合征（上肢猛拉 / Pancoast 瘤）。

![[L69609.jpg|400]]
*背部及上肢浅层肌群：斜方肌 trapezius(副神经)、三角肌 deltoid(腋神经)、冈下肌 infraspinatus、三头肌 triceps、背阔肌 latissimus dorsi——肩胛带定位参照。*

## 八、复习自测（折叠，先别看答案）

> [!question]- Q1. 7 岁儿童 FOOSH 后肘痛畸形，X 线示髁上骨折。查体桡动脉搏动减弱、不能做 OK 手势。下一步最关键的是？
> 评估**血管（桡动脉/灌注）**最优先 —— 髁上骨折伴肱动脉损伤可致 Volkmann 缺血挛缩；同时提示 **AIN（正中）损伤**（不能做 OK 圈）。

> [!question]- Q2. 醉酒后手臂搭椅背睡一夜，醒来抬手时腕下垂、虎口背侧麻，但伸肘正常。受累神经 + 定位？
> **桡神经**（Saturday night palsy）；伸肘正常 = 病变在螺旋沟以下（三头肌支已发出），定位**肱骨干中段 radial groove**。

> [!question]- Q3. 老年女性跌倒后腕痛、餐叉样畸形；复位后逐渐出现拇/食/中指麻 + 拇外展无力。神经？
> **正中神经**（Colles 骨折 → 急性腕管综合征）。

> [!question]- Q4. 肩关节前脱位复位后，三角肌区一小片麻木 + 外展费力。神经？若改为"感觉完整、仅外展无力"，最可能诊断？
> 麻木 → **腋神经损伤**；若**感觉完整**仅外展无力 → **肩袖损伤（rotator cuff）**。

> [!question]- Q5. FOOSH 后腕桡侧痛，鼻烟窝压痛，X 线未见骨折。最佳处理？
> 按**舟骨骨折**处理：**拇人字石膏固定 + 1–2 周复查 X 线/MRI**（早期 X 线可阴性；漏诊 → AVN/不愈合）。

### 9.1 肌皮神经 Musculocutaneous nerve (C5–C7)

> [!note] 屈肘弱 + 前臂外侧麻 = 肌皮，别和腋神经混
> - **损伤**：上臂前外侧脱位 / 肱骨近端骨折 / 罕见过度用力屈肘。
> - **运动**：肱二头肌 (biceps) + 肱肌 (brachialis) + 喙肱肌 (coracobrachialis) → **屈肘 + 前臂旋后无力**。
> - **感觉**：前臂外侧皮神经 (lateral antebrachial cutaneous n.) → **前臂外侧**麻。
> - **反射**：肱二头肌反射 (biceps reflex, C5–C6) 减弱。
> - **vs 腋神经**：腋神经 = 三角肌**外展**弱 + **上臂**外侧（regimental badge）麻；肌皮 = **屈肘**弱 + **前臂**外侧麻。位置（上臂 vs 前臂）+ 动作（外展 vs 屈肘）双重区分。

![[L23778.jpg|400]]
*肌皮神经 (musculocutaneous n.)：运动=屈肘(肱二头肌/肱肌)+前臂旋后；皮支=前臂外侧皮神经(管前臂外侧感觉)。*

### 9.2 旋前圆肌综合征 Pronator teres syndrome

> [!warning] 与腕管综合征 (CTS) 的决定性鉴别 = 大鱼际掌侧皮肤
> - **机制**：正中神经在肘部被**旋前圆肌 (pronator teres)** 卡压（重复旋前/抓握，如拧螺丝、网球）。
> - **症状**：正中神经分布麻（桡侧 3½ 指）+ **前臂近端痛**，旋前加重；可伴 AIN 样运动弱。
> - **★ 鉴别 CTS**：正中神经的**掌皮支 (palmar cutaneous branch)** 在腕管**近端**分出、不穿腕管 →
>   - **CTS**：大鱼际**掌侧皮肤感觉保留**（掌皮支没被压）+ 夜间麻 + 甩手缓解 + Tinel 在腕。
>   - **旋前圆肌综合征**：卡压点在肘，**掌皮支也受累 → 大鱼际掌侧也麻** + 前臂痛 + Tinel 在前臂近端 + 夜间症状不明显。
>   - 一句话：**"大鱼际掌侧麻不麻" 分肘 vs 腕**。

![[L25344.jpg|400]]
*正中神经前臂段：穿**旋前圆肌 (pronator teres)** 两头之间，发出 AIN 支配 FPL/FDP 桡侧半/旋前方肌——旋前圆肌综合征与 AIN 综合征的解剖基础。*

### 9.3 神经损伤分级 Seddon classification（预后排序）

> [!note] Neurapraxia → Axonotmesis → Neurotmesis（轻→重）
> ```
> Neurapraxia 神经失用 ── 髓鞘局部损伤,轴突完整 ── 无 Wallerian 变性 ── 数周-数月全恢复(预后最好)
> Axonotmesis 轴突断裂 ── 轴突断,神经内膜框架完整 ── 有 Wallerian 变性 ── ~1mm/天(≈1吋/月)再生,可恢复
> Neurotmesis 神经断裂 ── 整条神经离断 ── 需手术吻合 ── 预后差,易成神经瘤(neuroma)
> ```
> - **EMG (肌电图, electromyography) / NCS (神经传导检查, nerve conduction study) 时机陷阱**：失神经电位（fibrillations 纤颤）需损伤后 **2–3 周**才出现 → **急性期做 EMG 意义有限**，太早查会假阴性。
> - 骨折后神经麻痹**多为 neurapraxia**（如桡神经/肱骨干）→ 先观察，多自行恢复。

### 9.4 胸廓出口综合征 TOS (Thoracic outlet syndrome)

> [!info] ① 一句话先懂 TOS
> **胸廓出口 (thoracic outlet)** = 神经血管从脖子根钻去手臂的**窄通道**（锁骨与第一肋之间、斜角肌缝）。**TOS = 这通道里的臂丛或锁骨下血管被挤压**（像一束电线 + 水管挤在窄门洞）。诱因：**颈肋 (cervical rib)**、异常第一肋、反复过头动作。

> [!note] ② 两型：挤到神经 vs 挤到血管
> - **神经型（约 95%，最常见）**：挤到臂丛**下干 (C8–T1)** → 手尺侧 + 前臂内侧麻 + **手部肌肉萎缩**。
> - **血管型**：挤到**锁骨下静脉** → 血栓 = **Paget-Schroetter（"用力性血栓" effort thrombosis）** → 手臂**肿胀、发紫**（常见于反复过头运动后）。

> [!warning] ③ 反直觉点：C8–T1 却"大鱼际"萎缩最重
> 疑问：下干是 C8–T1，而 C8-T1 主要走**尺神经**（管小指侧）。为什么萎缩最重的是**大鱼际**（拇指根，正中神经管的）？
> - 答：拇短展肌 (abductor pollicis brevis, 大鱼际主肌) 虽走正中神经，但其纤维**主要来自 T1**；TOS 压的就是下干里的 **T1**（且 T1 比 C8 压得重）→ **大鱼际最先、最重萎缩**。
> - **一句话**：TOS 压的是"**节段 (C8-T1)**"、不是"某条具体神经" → 萎缩**按 T1 走 → 大鱼际最惨**。这种手叫 **Gilliatt-Sumner hand**。

> [!tip] ④ 怎么和"普通尺神经病"分开（最实用）
> **TOS 压的是整条下干 (C8-T1)，所以是三块拼起来 —— 比尺神经病多两块"非尺"成员：**
> ```
> TOS (下干 C8-T1) =
> ├─【尺那份】小指/环指尺侧麻 + 骨间肌/小鱼际弱   ← 和尺神经病一样
> ├─【非尺·T1①】大鱼际萎缩 (拇短展肌, 经正中神经) ← 额外
> └─【非尺·T1②】前臂内侧麻 (前臂内侧皮神经)        ← 额外
> ```
> 单纯尺神经病**只有第 1 块**；TOS 多出**大鱼际 + 前臂内侧**两块。最实用的区分钥匙：
> - **前臂内侧麻** → 只可能是 T1 根/下干 (TOS)；**尺神经不管前臂皮肤** (medial antebrachial cutaneous n. = T1)。
> - **大鱼际萎缩** → 正中/T1（尺神经管的是小鱼际）。
> - 为什么正中的大鱼际也中招？压的是**上游下干** → 凡 **T1** 出来的纤维，不管后走正中还是尺，都在此被掐。
>
> 检查：Adson test / Roos (EAST) test。

> [!example]- ⑤ 极简版（背这个就够）
> ```
> 胸廓出口窄 → 挤神经血管
> ├─ 神经型(95%) ── 下干 C8-T1 ── 大鱼际萎缩为主(Gilliatt-Sumner) + 前臂内侧麻
> ├─ 血管型 ────── 锁骨下静脉栓 ── 手臂肿胀发紫(Paget-Schroetter)
> └─ 诱因：颈肋 cervical rib / 反复过头动作
> 分尺神经病：TOS 连"前臂内侧"也麻；尺神经病只麻到手。
> ```
> 记忆钩：**TOS = 脖子出口挤一整束 (C8-T1)；大鱼际先瘪、前臂内侧也麻 —— 比尺神经病"管得更宽"。**

### 9.5 AIN vs PIN：都在前臂、都纯运动（防混淆）

> [!warning] 常见误会：AIN/PIN 不是"一个前臂一个上臂"
> 两条**都在前臂**，"前/后 (anterior/posterior)"指前臂的**掌侧 vs 背侧间室**，不是前臂 vs 上臂；都在**肘部附近**从主干分出后走在前臂里。

| | **AIN** 骨间前神经 | **PIN** 骨间后神经 |
|---|---|---|
| 全称 | Anterior Interosseous Nerve | Posterior Interosseous Nerve |
| 来自 | **正中神经** median 分支 | **桡神经** radial 分支 |
| 间室 | 前臂**掌侧（前）** | 前臂**背侧（后）** |
| 性质 | **纯运动·无感觉** | **纯运动·无感觉** |
| 支配 | FPL + 食/中指 FDP + 旋前方肌 | 伸指/伸腕(背侧伸肌群) |
| 标志缺损 | **不能做 OK 圈**（拇/食指屈不起）| **伸指无力**，伸腕保留/桡偏 |
| 卡压综合征 | 骨间前神经综合征 / 髁上骨折 | 旋后肌综合征 (supinator syndrome)·桡管综合征（vs 网球肘）|

> [!tip] 纯运动陷阱
> 两条**都无感觉缺损** → stem 给"运动障碍但手完全不麻" → 想**前臂 AIN/PIN**：看是**屈**(AIN/正中系) 还是**伸**(PIN/桡系)。记忆钩：**前(AIN)管屈、后(PIN)管伸，都不管感觉。**

# 神经 / 肌阵挛 (Myoclonus) 鉴别诊断

> [!info] 一句话定位
> **肌阵挛 (myoclonus 肌阵挛) = 突发、短暂、电击样不自主肌肉抽动。**
> USMLE 极少单考"是不是肌阵挛"，而是把它当 **stem 钩子** —— 给你一个抽动 + 一段全身背景，逼你反推病因。**解题第一步永远是：先判正/负，再看全身背景。**
> 派生自昨日错题 [[mistakes/uworld-mistakes_2026-06#^Q20437]]（尼古丁中毒 nicotine poisoning）+ [[mistakes/uworld-mistakes_2026-06#^Q14453]]（肾衰阿片代谢物蓄积）。

---

## §一 第一步：正肌阵挛 vs 负肌阵挛（决定方向）

| 类型 | 定义 | 床边表现 | 高频指向 |
|---|---|---|---|
| **阳性肌阵挛 positive myoclonus** | 肌肉**主动收缩**致抽动 | 电击样跳动 | 癫痫 / 退行性 / 中毒 / 缺氧后 |
| **阴性肌阵挛 negative myoclonus = 扑翼样震颤 asterixis** | 肌张力**短暂丧失** | 伸手平举 → 手腕周期性"扑动"下垂 | **代谢性脑病**（肝/肾/CO₂/血糖） |

> [!tip] 第一反射
> - 题干"**伸手平举出现扑翼样动作**" = **asterixis = 代谢性脑病** → 先查肝功 / 肾功 / 血气 / 血糖。
> - 题干"**电击样抽跳、扔掉牙刷、被巨响惊吓诱发**" = **阳性肌阵挛** → 往下走癫痫 / CJD / 中毒。

### 负肌阵挛 (asterixis) 的代谢病因

- **肝性脑病 hepatic encephalopathy 肝性脑病**（肝硬化 + 高血氨）
- **尿毒症 uremia 尿毒症**（肾衰）
- **高碳酸血症 hypercapnia 高碳酸血症 / CO₂ 潴留**（COPD 慢性阻塞性肺病 chronic obstructive pulmonary disease 失代偿）
- **低血糖 hypoglycemia 低血糖**

> [!note]- asterixis 机制
> 代谢毒素（高血氨 / 尿毒症毒素 / CO₂ / 低糖）**干扰丘脑-皮层运动通路** → 维持姿势的**运动皮层间断性抑制** → 肌张力**瞬时同步丧失** → 手腕"扑动"。本质 = **持续收缩中断**，故归为"阴性"肌阵挛。

## §二 阳性肌阵挛病因总览（ASCII 决策树）

```
肌阵挛 myoclonus
├─ 青少年 + 晨起抽跳/甩早餐 + 熬夜诱发 ────── 青少年肌阵挛癫痫 (JME)
├─ 老人 + 数周快速痴呆 + 惊吓性肌阵挛 ──────── 克雅病 (CJD)
├─ 用 5-HT 能药 + 阵挛/反射亢进/高热 ──────── 5-羟色胺综合征
├─ 心脏骤停复苏后 ────────────────────────── 缺氧后肌阵挛 (Lance-Adams)
├─ 肾衰患者用阿片/某些药物 ─────────────────── 药物/毒物性
├─ 幼儿 + "跳舞眼·跳舞脚" + 共济失调 ───────── 眼阵挛-肌阵挛综合征 (OMS) → 神经母细胞瘤
├─ 未接种麻疹史 + 数年后认知衰退 ───────────── SSPE 亚急性硬化性全脑炎
└─ 正常人入睡时一跳 / 呃逆 ─────────────────── 生理性（勿过度诊断）
```

## §三 各病 stem 模式（UWorld 经典）

### 1. 青少年肌阵挛癫痫（JME, juvenile myoclonic epilepsy 青少年肌阵挛癫痫）

> [!example]- JME — 晨起甩 + 熬夜诱发
> - **Stem**：13–18 岁，**晨起**手抖把牙刷/咖啡甩飞，偶有全面强直阵挛发作 (GTC, generalized tonic-clonic 全面强直阵挛)，**熬夜 / 饮酒 / 闪光诱发**，认知正常。
> - **EEG（脑电图 electroencephalogram）**：**4–6 Hz 多棘慢波 (polyspike-wave)**。
> - **治疗**：**丙戊酸 valproate**（首选；备选 左乙拉西坦 levetiracetam / 拉莫三嗪 lamotrigine）。**避免卡马西平 carbamazepine / 苯妥英 phenytoin**（加重肌阵挛）。
> - **机制**：**全面性遗传性癫痫**（GABRA1 / EFHC1 等 GABA·钙通道基因）→ **丘脑-皮层环路 thalamocortical circuit 过度兴奋 + 异常同步化**，肌阵挛 = 皮层放电瞬时下传。钠通道阻滞剂（卡马西平/苯妥英）对**全面性**网络有**矛盾性加重**作用（故禁用）；丙戊酸广谱抑制全面性放电。

### 2. 克雅病（CJD, Creutzfeldt-Jakob disease 克雅病）

> [!example]- CJD — 快速痴呆 + 惊吓性肌阵挛
> - **Stem**：60+ 岁，**数周–数月内快速进展性痴呆** + **惊吓性肌阵挛 (startle myoclonus，巨响诱发)** + 共济失调。
> - **辅查**：EEG **周期性尖波/三相波**；CSF（脑脊液 cerebrospinal fluid）**14-3-3 蛋白 / RT-QuIC 实时震动诱导转化** 阳；MRI 皮层飘带征 + 基底节高信号。
> - **钥匙**：**数周内进展性痴呆 + 肌阵挛 = CJD（直到证明不是）**；致死、无特效治疗。
> - **机制**：**朊蛋白病 prion disease**——正常朊蛋白 PrP^C 错误折叠成致病型 PrP^Sc，**模板式诱导**周围正常朊蛋白连锁错折叠 → 聚集沉积 → 神经元**海绵状空泡变性 (spongiform change)** + 神经元丢失 + 星形胶质增生；皮层广泛兴奋/抑制失衡 → 惊吓性肌阵挛。
> - 详见痴呆维度对照 → [[完整笔记/专题笔记/神经/神经_老年痴呆鉴别]]（§7.5 CJD）。

### 3. 5-羟色胺综合征（serotonin syndrome 5-羟色胺综合征）

> [!example]- 5-HT 综合征 — 阵挛 + 反射亢进 + 高热
> - **Stem**：用 SSRI（选择性 5-羟色胺再摄取抑制剂 selective serotonin reuptake inhibitor）+ 另一种 5-HT 能药（曲马多 tramadol / 利奈唑胺 linezolid / MAOI 单胺氧化酶抑制剂 / 曲普坦 triptan），急性起病：**激越 + 反射亢进 + 下肢阵挛 clonus + 肌阵挛 + 高热 + 大汗 + 散瞳**。
> - **治疗**：停药 + **赛庚啶 cyproheptadine** + 支持降温。
> - **机制**：突触间隙 **5-羟色胺 serotonin 过量** → 过度激动 **5-HT2A 和 5-HT1A 受体** → 神经肌肉兴奋性亢进（阵挛/肌阵挛/反射亢进，下肢著）+ 自主神经亢进（高热/大汗）+ 精神状态改变。**赛庚啶 = 5-HT2A 受体拮抗剂**，对因解毒。
> - **对比**：抗胆碱"**干**"、有机磷"**湿 + 缩瞳**"、5-HT"**湿 + 阵挛/反射亢进**"。详见 [[完整笔记/专题笔记/USMLE/USMLE_毒物综合征toxidrome速查]] + [[完整笔记/专题笔记/USMLE/USMLE_高热肌强直急症鉴别]]。

### 4. 缺氧后肌阵挛（Lance-Adams syndrome 兰斯-亚当斯综合征）

> [!example]- 缺氧后 — 骤停复苏后
> - **Stem**：**心脏骤停复苏后**出现动作性肌阵挛 (action myoclonus)。
> - **预后**：复苏后**持续性肌阵挛状态 (status myoclonus) = 预后不良信号**（缺氧脑损伤）。
> - **机制**：全脑缺氧缺血损伤后，对缺氧敏感的**浦肯野细胞 Purkinje cell / 皮层神经元**死亡 → **5-羟色胺能传递失调 + 皮层-皮层下网络去抑制** → 动作性肌阵挛（故部分对丙戊酸 / 氯硝西泮 clonazepam / 5-HT 前体有反应）。

### 5. 药物/毒物性（高频，直连昨日两张卡）

> [!example]- 药物/毒物 — 肾衰蓄积 + 误服
> - **肾衰患者用阿片**：**吗啡 morphine** 活性代谢物（吗啡-6-葡糖苷酸 morphine-6-glucuronide）蓄积、**哌替啶 meperidine**（normeperidine 去甲哌替啶蓄积）→ 肌阵挛 / 抽搐 / AMS（急性意识改变 altered mental status）。→ [[mistakes/uworld-mistakes_2026-06#^Q14453]]
> - **尼古丁中毒 nicotine poisoning**（幼儿误服烟头 / 电子烟液 e-cigarette liquid）→ 刺激相含肌阵挛。→ [[mistakes/uworld-mistakes_2026-06#^Q20437]]
> - **肾衰时蓄积致神经毒**：锂 lithium / 加巴喷丁 gabapentin / 普瑞巴林 pregabalin / β-内酰胺类（亚胺培南 imipenem、头孢吡肟 cefepime、青霉素 penicillin）。
> - 铋 bismuth、三环类抗抑郁药 TCA。
> - **机制**：① **吗啡**→活性代谢物 **M6G（吗啡-6-葡糖苷酸）经肾排，肾衰蓄积**→μ 受体过度激动 + 兴奋性代谢物致肌阵挛；**哌替啶**→**去甲哌替啶 normeperidine（促惊厥代谢物）**蓄积。② **β-内酰胺类 = GABA-A 受体拮抗**（去抑制→兴奋毒性），肾衰蓄积加重。③ **尼古丁 = 烟碱受体 nAChR 激动**致中枢兴奋。共同主线 = **兴奋性↑ 或抑制性 (GABA) ↓**。

### 6. 眼阵挛-肌阵挛-共济失调综合征（OMS, opsoclonus-myoclonus-ataxia syndrome 眼阵挛-肌阵挛-共济失调综合征）

> [!example]- OMS — "跳舞眼·跳舞脚"
> - **Stem**：**幼儿**，"**跳舞的眼睛、跳舞的脚 (dancing eyes, dancing feet)**" + 共济失调 + 易激惹。
> - **病因**：副肿瘤 → **神经母细胞瘤 neuroblastoma 神经母细胞瘤**（查尿 VMA/HVA 香草扁桃酸/高香草酸 + 腹部/胸部影像）；亦可感染后。
> - **机制**：**副肿瘤性自身免疫**——神经母细胞瘤表达神经元抗原，激发**抗神经元自身抗体**交叉反应 → 免疫介导**小脑/脑干（omnipause 神经元失抑制致眼阵挛）**功能紊乱。治疗 = 切肿瘤 + 免疫治疗（激素 / IVIG 静脉免疫球蛋白 / 利妥昔单抗 rituximab）。

### 7. SSPE（subacute sclerosing panencephalitis 亚急性硬化性全脑炎）

> [!example]- SSPE — 麻疹数年后
> - **Stem**：**未接种麻疹 measles 麻疹** 的儿童/青少年，感染数年后**渐进性认知行为衰退 + 周期性肌阵挛**。
> - **辅查**：EEG 周期性复合波 + CSF 麻疹抗体滴度↑。
> - **机制**：**麻疹病毒持续/慢性感染中枢神经系统**——缺陷型病毒（**M 基质蛋白缺失**无法出芽）在神经元内**持续复制** → 慢性全脑炎症 + 神经元变性 + 脱髓鞘；周期性肌阵挛对应全脑同步放电。

## §四 别和这些"动作"混了（运动障碍鉴别）

| 长得像 | 关键区别 | 高频背景 |
|---|---|---|
| 震颤 tremor | **节律性**来回摆动 | 帕金森 / 甲亢 / 原发性 |
| 舞蹈症 chorea | **流动、舞蹈样**、不规则游走 | Huntington 亨廷顿 / 风湿热 Sydenham 舞蹈病 |
| 抽动 tics | **可短暂抑制** + 有冲动感 + 刻板 | Tourette（→ [[完整笔记/专题笔记/psych/psych_Tourette诊断与治疗]]） |
| 肌束颤 fasciculation | 肌肉**蠕动但不动关节** | LMN 下运动神经元 / ALS 肌萎缩侧索硬化 / **有机磷 organophosphate** |
| 阵挛 clonus | **牵张诱发**的节律抽动 | UMN 上运动神经元 / **5-HT 综合征** |
| 扑翼样震颤 asterixis | = **阴性**肌阵挛 | 代谢性脑病 |

## §五 钥匙速记

> [!danger] 一句话钥匙
> - **青少年晨起甩早餐 = JME** → 丙戊酸（避卡马西平/苯妥英）
> - **老人数周快速痴呆 + 惊吓抽 = CJD**（14-3-3 / 周期性 EEG）
> - **用 5-HT 药 + 阵挛 + 反射亢进 + 高热 = 5-HT 综合征** → 赛庚啶
> - **伸手扑翼 (asterixis) = 代谢性脑病** → 查肝/肾/CO₂/血糖
> - **幼儿跳舞眼·跳舞脚 = OMS** → 找神经母细胞瘤
> - **骤停复苏后抽 = 缺氧后肌阵挛 (Lance-Adams)**
> - **肾衰患者用吗啡/哌替啶 → 代谢物蓄积抽** → 换芬太尼/美沙酮
> - **未种麻疹 + 数年后衰退 + 周期抽 = SSPE**

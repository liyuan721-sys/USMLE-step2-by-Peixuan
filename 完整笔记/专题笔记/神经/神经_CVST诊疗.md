# 脑静脉窦血栓 CVST 诊疗（识别"颅压增高型 vs 局灶型" → MRV/CTV → 抗凝，即使出血也抗凝）

> [!info] 用途 + 缩写约定
> 从 [[完整笔记/专题笔记/神经/神经_颅压增高诊疗顺序]] §3.3 拆出的独立专题（原为 🌱TODO）。CVST 是 USMLE 高频"陷阱型卒中"——**年轻女性（产后 / 口服避孕药）+ 亚急性头痛 ± 癫痫 seizure ± 局灶缺损 focal deficit**，CT 常正常，**治疗是抗凝而非溶栓**，和动脉卒中完全两条路。本卡解决三个反复考点：**① 什么线索该想到 CVST（而不是动脉卒中 / IIH / 子痫 eclampsia）② 查什么（MRV/CTV，不是等 CT）③ 出血性静脉梗死 hemorrhagic venous infarct 了还抗不抗凝（抗）**。
> 缩写：**CVT** = cerebral venous thrombosis（脑静脉血栓，**总称**）；**CVST** = cerebral venous **sinus** thrombosis（脑静脉**窦**血栓，CVT 的主干亚型）；**CT** = computed tomography（计算机断层）；**MRV** = magnetic resonance venography（磁共振静脉造影）；**CTV** = CT venography（CT 静脉造影）；**LMWH** = low molecular weight heparin（低分子肝素）；**DOAC** = direct oral anticoagulant（直接口服抗凝药）；**tPA** = tissue plasminogen activator（组织型纤溶酶原激活剂，溶栓药）；**ICP** = intracranial pressure（颅内压）；**IIH** = idiopathic intracranial hypertension（特发性颅内高压）；**OCP** = oral contraceptive pill（口服避孕药）；**SSS** = superior sagittal sinus（上矢状窦）；**PDPH** = post-dural puncture headache（硬脊膜穿刺后头痛）；**IBD** = inflammatory bowel disease（炎症性肠病）；**AMS** = altered mental status（意识改变）；**CSF** = cerebrospinal fluid（脑脊液）。

> [!danger] ⭐ 置顶速记：CVST 三反射
> 1. **年轻女 + 产后/OCP/高凝 + 亚急性进行性头痛（数天，晨重、Valsalva 重）→ 想 CVST → 查 MRV / CTV**（不是非增强 CT 就完事，CT 常正常）。
> 2. **静脉性脑梗/出血的三个"不像动脉"信号**：① 病灶**跨动脉供血区**、位置怪（近凸面/双侧旁矢状）② 常**出血性**（venous infarct 天生易出血）③ 患者**年轻 + 癫痫多 + 头痛先行**。
> 3. **治疗 = 抗凝（LMWH → 华法林 warfarin），即使已有出血性静脉梗死照样抗凝**——抗凝降窦内压、防血栓扩展，获益 > 出血风险。**❌ 不溶栓、❌ 不因"有出血"停抗凝。**

## 一、概念分层：CVT ⊃ CVST（窦 vs 皮质静脉，临床表现不同）

> [!info] 一个总称，两种"人格"
> **CVT（脑静脉血栓）= 总称**，指血栓发生在脑静脉系统任何部位；按堵的位置分两类，表现差很多：

```
              CVT（脑静脉血栓，总称）
                     │
      ┌──────────────┴───────────────┐
      ▼                              ▼
  硬脑膜静脉窦血栓                皮质静脉 / 深静脉血栓
  CVST（如上矢状窦 SSS、          （cortical / deep veins）
  横窦 transverse、乙状窦）
      │                              │
      ▼                              ▼
 【颅压增高型】                  【局灶皮层刺激型】
 窦与 CSF 吸收相关 → 堵了 CSF     皮层附近静脉回流受阻 →
 回流受阻 → ICP↑                  局部水肿 / 出血性静脉梗死
  · 头痛（晨重、渐进）             · 局灶神经缺损（偏瘫/失语）
  · 视乳头水肿 papilledema        · 癫痫（很常见）
  · 呕吐                          · 可有出血
  · 癫痫可有可无
```

> [!tip] 一句话锁定
> **窦为主 → 更偏"颅压增高"（头痛+papilledema，像 IIH）；皮质静脉为主 → 更偏"局灶缺损/癫痫"（像卒中但会出血、会跨区）。** 实际常混合，取决于血栓位置与范围。

## 二、什么时候该想到 CVST（危险因素 = 高凝状态）⭐

> [!warning] CVST = "高凝状态在脑里的表现"，问病史找诱因
> 见到**不典型的年轻人卒中/头痛/癫痫**，反射去问这些：

| 类别 | 具体诱因 |
|---|---|
| **妊娠相关**（最经典）| **妊娠 / 产褥期 puerperium（产后数周内）**、先兆子痫 preeclampsia |
| **激素** | **口服避孕药 OCP**、激素替代 |
| **遗传性易栓症**（thrombophilia）| Factor V Leiden（凝血因子 V Leiden 突变）、凝血酶原基因突变 prothrombin mutation、蛋白 C/S 缺乏 protein C/S deficiency、抗凝血酶缺乏 antithrombin deficiency |
| **获得性易栓/自身免疫** | 抗磷脂抗体综合征 antiphospholipid syndrome、肾病综合征 nephrotic syndrome（丢抗凝蛋白）、白塞病 Behçet disease、炎症性肠病 IBD、恶性肿瘤 malignancy |
| **局部感染蔓延**（"化脓性"）| **乳突炎 mastoiditis / 中耳炎 otitis media → 横窦-乙状窦**；面部/鼻窦感染 sinusitis、蝶窦炎 sphenoid sinusitis → **海绵窦血栓 cavernous sinus thrombosis**（面部危险三角"death triangle"）|
| **其他** | 严重脱水、头部外伤、腰穿/神经外科术后 |

> [!tip] 海绵窦血栓（cavernous sinus thrombosis）单拎一句
> 面部危险三角 / 鼻窦 / 牙源性感染 → 海绵窦：**眼球突出 proptosis + 眼睑水肿 + 眼肌麻痹（CN III/IV/VI）+ 三叉神经 V1/V2 感觉异常 + 视力受累**，常双侧扩散。→ 急诊：**广谱抗生素 + 抗凝**。

## 三、诊断：CT 常正常 → MRV / CTV 是确诊 ⭐（最高频考点）

> [!danger] 别停在非增强 CT —— 阴性不能排 CVST
> 非增强 CT 在 CVST **约 1/3 正常**，正常 CT 绝不能排除。想到 CVST → 直接上 **MRV（磁共振静脉造影）或 CTV（CT 静脉造影）** 看静脉窦有无充盈缺损。

```
       亚急性头痛 ± 癫痫 ± 局灶缺损 + 高凝诱因（产后/OCP…）
                          │
                          ▼
          非增强 CT（先排出血/占位，但常正常或仅间接征）
                          │
        ┌─────────────────┴──────────────────┐
        ▼                                    ▼
  见间接征（提示但不确诊）              CT 正常 / 仍高度怀疑
  · 空 delta 征（增强）                        │
  · 索条征 cord sign（皮质静脉高密度）          │
  · 跨区 / 出血性静脉梗死                       │
        └─────────────────┬──────────────────┘
                          ▼
              ⭐ MRV 或 CTV = 确诊（静脉窦充盈缺损）
                          ▼
                     启动抗凝治疗
```

> [!info] CT 上的两个"影像 buzzword"（认得即可）
> - **空 delta 征 (empty delta sign)**：增强 CT 上，上矢状窦后部**强化的窦壁包绕不强化的血栓**，呈空心三角 → SSS 血栓典型征。
> - **索条征 (cord sign)**：非增强 CT 上血栓的皮质静脉呈**高密度条索**。
> - 两者敏感性都不高，**阴性不排除 → 仍靠 MRV/CTV**。

## 四、治疗：抗凝为核心，出血也抗凝 ⭐

> [!danger] 铁律：CVST 即使合并出血性静脉梗死，一线仍是抗凝
> **急性期：LMWH（低分子肝素）或普通肝素 unfractionated heparin → 序贯华法林 warfarin**（或按情况用 DOAC 直接口服抗凝药）。
> **"已经脑出血了还抗凝？"是的**——CVST 的出血是**静脉回流受阻、窦内高压**造成的；抗凝**打通回流、降窦内压、防血栓扩展**，改善症状，获益大于出血扩大风险。这与动脉性出血性卒中的处理**相反**，是经典考点。

| 环节 | 处理 |
|---|---|
| **抗凝**（核心）| 急性 LMWH/肝素 → 华法林；疗程 3-6 个月（诱因可逆）～ 长期（易栓症/复发）|
| **难治 / 恶化** | 血栓较大、抗凝下仍进展 → **血管内取栓 / 局部溶栓**（专科选择，非首选）|
| **颅高压 / 视力威胁** | 处理 ICP（抬头、渗透治疗）；顽固者分流；视力受累 → 参考 IIH 处理思路 |
| **癫痫** | 有癫痫发作 → 抗癫痫药；无发作不预防用药 |
| **病因** | 停 OCP、纠正脱水、治感染源（乳突炎/鼻窦炎）、查易栓症 |

> [!warning] ❌ 反 pattern（别踩）
> - ❌ **静脉溶栓 tPA / 机械取栓当一线**——那是**动脉**卒中路；CVST 一线是抗凝。
> - ❌ 看到"脑出血"就停抗凝 / 上止血——CVST 出血反而是抗凝指征。
> - ❌ CT 正常就排除 CVST 放回家——必须 MRV/CTV。
> - ❌ 把产后头痛一律当 PDPH / 子痫——见 §五 鉴别。

## 五、鉴别诊断：CVST vs 动脉卒中 vs IIH vs 产后头痛 ⭐

### 5.1 CVST vs 动脉性缺血卒中（venous vs arterial）

| 维度 | 动脉性缺血卒中 | CVST（静脉性）|
|---|---|---|
| 起病 | **突发**、秒-分钟达峰 | **亚急性、渐进**（数小时-数天），头痛常先行 |
| 人群 | 老年 + 血管危险因素 | **年轻**、女性、高凝诱因（产后/OCP/易栓症）|
| 病灶分布 | **符合单一动脉供血区** | **跨动脉供血区**、位置不典型（旁矢状、双侧、近凸面）|
| 出血 | 多为缺血（部分再灌注出血）| **常出血性**（静脉梗死天生易出血）|
| 癫痫 | 早期少见 | **常见** |
| 治疗 | 溶栓 / 取栓 / 抗血小板 | **抗凝**（含出血时）|

### 5.2 CVST vs IIH（都"颅压增高型"，靠影像分）

> [!info] 两者都能给"头痛 + papilledema + 搏动性耳鸣 + 视物模糊"
> 区别在**影像 + 诱因**：**IIH = 影像阴性（含 MRV 阴性）的排除性诊断；CVST = MRV/CTV 见窦充盈缺损**。所以在 IIH 诊断流程里，**CT 正常后疑高凝/产后/OCP → 先加 MRV 排 CVST，再走 IIH**（见 [[完整笔记/专题笔记/神经/神经_颅压增高诊疗顺序]] §二、§3.3）。治疗也相反：IIH 减重 + 乙酰唑胺；CVST 抗凝。

### 5.3 产后头痛四鉴别（CVST 是其中之一，OB 高频）

| 诊断 | 招牌线索 | 下一步 |
|---|---|---|
| **CVST** | 亚急性进行性头痛 ± 癫痫 ± 局灶缺损，高凝背景 | **MRV/CTV** → 抗凝 |
| **子痫前期 / 子痫 preeclampsia / eclampsia** | 高血压 + 蛋白尿；非体位性头痛；抽搐 | 硫酸镁 magnesium sulfate + 降压（见 [[完整笔记/专题笔记/OB_GYN/OB_GYN_先兆子痫子痫谱系]]）|
| **硬脊膜穿刺后头痛 PDPH** | **体位性**（坐/站加重、平卧缓解），椎管麻醉后 | 保守 → 血补丁 blood patch |
| **脑膜炎 meningitis** | 发热 + 颈强 nuchal rigidity + AMS | LP（腰椎穿刺）→ CSF 培养 |

## 六、复习题

> [!question]- Q1. 产后 10 天女性，进行性头痛 3 天、今日局灶性癫痫发作，头非增强 CT 见右侧旁矢状区出血。下一步 + 治疗方向？
> 高度疑 **CVST（脑静脉窦血栓）**：产后高凝 + 亚急性头痛 + 癫痫 + **跨区/不典型部位的出血性病灶**。下一步 **MRV 或 CTV** 确诊（窦充盈缺损）。治疗 = **抗凝（LMWH → 华法林）**——**即使已有出血也抗凝**（降窦内压、防扩展），不溶栓、不因出血止血。

> [!question]- Q2. 为什么 CVST 合并脑出血还要抗凝，而动脉性出血性卒中不能？
> CVST 的出血源于**静脉回流受阻、窦内高压**导致静脉性梗死渗血；**抗凝打通回流、降窦压、阻止血栓扩展**，从根上减轻出血机制，获益 > 风险。动脉性出血是血管破裂出血，抗凝会加重，机制相反。

> [!question]- Q3. 28 岁肥胖女，头痛 + papilledema + 搏动性耳鸣，头 CT 正常。她既往用口服避孕药。为何不能直接按 IIH 给乙酰唑胺？
> 因为 **CVST 会拟态 IIH（同为颅压增高型）**，且患者有 **OCP 高凝诱因**。IIH 是**排除性诊断**，CT 正常后应**先加 MRV 排 CVST**；若 MRV 见窦血栓 → 诊断/治疗完全不同（抗凝，而非减重 + 乙酰唑胺）。MRV 阴性才落 IIH。

> [!question]- Q4. 中耳炎/乳突炎患儿出现头痛、发热、papilledema，最可能血栓部位？海绵窦血栓又指向什么感染源？
> 乳突炎/中耳炎 → **横窦-乙状窦血栓**（感染邻近蔓延）。**海绵窦血栓**指向**面部危险三角 / 鼻窦 / 牙源性感染**，表现 proptosis + 眼肌麻痹（CN III/IV/VI）+ V1/V2 感觉异常，常双侧；处理 = 广谱抗生素 + 抗凝。

> [!question]- Q5. 一句话说清 CVST 的影像路径，以及"CT 正常"意味着什么？
> 想到 CVST → **MRV 或 CTV** 看静脉窦充盈缺损确诊。**非增强 CT 约 1/3 正常，正常不能排除**；CT 上的空 delta 征（增强）/ 索条征只是提示、敏感性低。所以 CVST 里"CT 正常"≠ 排除（与 IIH 里"CT 正常 = 已排占位可 LP"含义不同）。

# 神经_优势与非优势半球定位（Dominant vs Non-dominant Hemisphere）

> [!info] 一句话定位
> **优势半球（dominant hemisphere）≈ 95% 右利手 + ~70% 左利手都是左脑**（总体约 90%+ 人群左脑优势）。判失语、半球综合征的默认前提就是"**优势 = 左**"。
> - **优势（左）= 管"说什么"**：语言 / 读写算 / 逻辑 → 坏了 = **失语 Aphasia** + **Gerstmann**
> - **非优势（右）= 管"怎么说 + 在哪"**：空间-注意 / 情感语调 / 面孔 → 坏了 = **忽视 Neglect** + **失韵律 Aprosodia**

## 一、优势 vs 非优势 对照表 ⭐

| 维度 | 优势半球（dominant，多为左）| 非优势半球（non-dominant，多为右）|
|---|---|---|
| **核心功能** | 语言（language）、计算、读写、逻辑 | 空间-注意（spatial attention）、情感语调、面孔、音乐 |
| **卒中标志缺陷** | **失语 Aphasia** | **忽视 Neglect** + **失韵律 Aprosodia** |
| **顶叶综合征** | **Gerstmann 综合征**（优势角回 angular gyrus）| **半侧空间忽视 hemineglect**（非优势顶叶）|
| **对缺陷的自觉** | 通常知道自己病了 | 常**病感缺失 anosognosia**（否认瘫痪/忽视左侧）|
| **失用 apraxia** | 意念运动性失用（ideomotor apraxia，不会按指令做动作）| 构造性/穿衣失用（constructional / dressing apraxia）|

> [!tip] 为什么下面单拎"顶叶"这两个综合征当门面
> **顶叶是全脑最能说明"半球分工"的地方** —— 同一块解剖位置，换个半球，病完全不同：优势（左）顶叶 = **Gerstmann**（算写指左右）；非优势（右）顶叶 = **忽视 + 失韵律 + 病感缺失**（空间与情感）。三个理由：
> 1. **对比最戏剧**：额叶两侧差异没这么大、颞叶那侧被 Wernicke 失语占了，唯独顶叶左右各成一套独立综合征，最适合演示"优势 vs 非优势"。
> 2. **最爱被误当失语**：Gerstmann 有"失写"、右顶叶 aprosodia 说话"不对劲" → stem 引诱你选语言区，实则**语言完好**（错题钉死点）。
> 3. **承上启下到血管**：两者都是**皮质征**（血管定位第一问）→ Gerstmann≈优势 **MCA 下分支**、右顶叶忽视≈非优势 **MCA** 供血区，认出即同时圈定半球 + 血管。是 [[完整笔记/专题笔记/神经/神经_卒中血管综合征定位]] 的接口。
>
> （面孔失认 / 失读 / 失用等其他半球化功能重要性次一档，放 §五"其他"带过。）

## 二、Gerstmann 综合征（优势顶叶 / 角回 angular gyrus）—— 4 联征 ⭐

> [!warning] "算-写-指-左右" 4 个一起 = 优势角回
> 全是"读写算 + 手指左右"的**优势侧**高级功能：
> - **失算 Acalculia**（不会算数）
> - **失写 Agraphia**（不会写字）
> - **手指失认 Finger agnosia**（叫不出 / 认不出自己手指）
> - **左右失认 Left-right disorientation**（分不清左右）
>
> **陷阱**：这**不是失语**（错题钉死点）—— 别因 stem 写"读写困难"就反射到语言区。Gerstmann 病人**语言表达/理解可正常**，坏的是"算-写-指-左右"这一组。定位在**优势顶叶下部（角回）**，多为优势 MCA（middle cerebral artery，大脑中动脉）下分支供血区。

## 三、非优势（右）顶叶综合征 —— 空间与情感 ⭐

> [!warning] 右顶叶 = 空间-注意 + 情感 + 病感缺失
> - **半侧空间忽视 Hemispatial neglect**：忽略**对侧（左）**半边空间 —— 只吃盘子右半、只画钟面右半的数字、只刮右半边脸、撞左边门框。**几乎总是右顶叶损伤致左侧忽视**（左顶叶损伤右侧忽视少见，因右脑对双侧空间都有注意冗余）。
> - **病感缺失 Anosognosia**：**否认**自己有病 —— 明明左侧偏瘫却坚称肢体正常（右顶叶特征）。
> - **失韵律 Aprosodia**：见 §四。
> - **构造性失用 constructional apraxia**（画不出立方体/五边形）、**穿衣失用 dressing apraxia**（不会穿衣服）。

## 四、失韵律 Aprosodia（"情感版失语"）

> [!tip] Prosody = 语言的"音乐"（语调、节奏、重音、情感色彩）
> 优势半球管**语言内容**（说什么），非优势半球管**语言的情感语调**（怎么说）。右半球损伤 → 内容正常但**情感表达/理解垮掉**，恰好与左半球失语**平行对称**：
>
> | 类型 | 病灶（非优势侧）| 表现 |
> |---|---|---|
> | **Motor aprosodia**（运动性失韵律）| 右额（对应 Broca 的镜像位）| 说话**平板单调、无情感起伏**，但能听出别人语气 → "情感版 Broca" |
> | **Sensory aprosodia**（感受性失韵律）| 右颞（对应 Wernicke 的镜像位）| **听不出**别人话里的情感/讽刺语气，自己语调可正常 → "情感版 Wernicke" |
>
> **CK 钩子**：卒中后"说话对但**像机器人一样毫无感情**"或"**听不懂玩笑/讽刺**" → 想**右半球 aprosodia**，不是抑郁、不是经典失语。

## 五、其他半球化功能（补充）

- **面孔失认 Prosopagnosia**：认不出熟人面孔（靠声音/衣着认人）→ 病灶在**梭状回 fusiform gyrus**（多为**非优势/右侧或双侧**枕颞交界）。
- **失读不伴失写 Alexia without agraphia**：能写却读不了自己写的字 → **优势枕叶 + 胼胝体压部（splenium）** 断连（经典 PCA（posterior cerebral artery，大脑后动脉）卒中）。
- **意念运动性失用 Ideomotor apraxia**：能自发做、却不能**按口头指令**做动作（"假装刷牙"做不出）→ 多**优势半球**。

## 六、易混陷阱 + 一句话对仗

> [!danger] 三个最容易混的钉死点
> 1. **Gerstmann ≠ 失语** —— "读写算 + 手指左右" 是优势**顶叶**，不是语言区；语言可正常。
> 2. **忽视 neglect ≈ 永远是右（非优势）顶叶 → 忽略左侧**；stem 写"只画钟面右半/无视左侧"直接锁右顶叶，别去想失语。
> 3. **Aprosodia ≠ 抑郁/失语** —— 说话平板无感情但内容/理解正常 = 右半球，别误诊情感障碍。

> [!tip] 一句话对仗（背这句）
> **左脑管"说什么"（内容/语言）→ 坏了 = 失语（+ Gerstmann 读写算）；右脑管"怎么说 + 在哪"（语调/空间）→ 坏了 = 失韵律 + 忽视（+ 病感缺失）。**

## Memory Hook

> [!success]
> - **左 = Language + Logic**（语言逻辑）→ 失语 / Gerstmann（**算写指左右**）
> - **右 = Right there, in space + emotion**（空间 + 情感）→ **左侧**忽视 / aprosodia / 否认病（anosognosia）
> - **忽视永远忽略"对侧"，且几乎总是右脑病→忽略左侧**
> - **Aprosodia = 情感版失语**（motor 右额 / sensory 右颞，与 Broca/Wernicke 镜像）

## 复习题

> [!question]- Q1. 卒中患者能流利对话、理解正常，但说话毫无情感起伏、且听不出他人语气中的讽刺。病灶最可能在？
> **右（非优势）半球**（额 + 颞的 prosody 区）= **Aprosodia 失韵律**。语言内容功能完好 → 排除左半球经典失语；情感语调垮 → 右半球。别误诊抑郁。

> [!question]- Q2. 患者不会算简单加法、写不出字、分不清左右手，语言表达和理解均正常。综合征 + 定位？
> **Gerstmann 综合征** = 失算 + 失写 + 手指失认 + 左右失认；定位**优势（左）顶叶角回**。**不是失语**（语言正常是关键鉴别点）。

> [!question]- Q3. 右侧大脑半球大面积梗死后，患者只吃盘子右半食物、刮胡子漏掉左脸，并坚称自己没有任何肢体无力（实际左侧偏瘫）。两个体征各是什么？
> ① **左侧半空间忽视 hemineglect**（右顶叶）② **病感缺失 anosognosia**（否认自身缺陷，右顶叶特征）。

# Dementia (痴呆) 鉴别决策树 — USMLE Step 2 CK

> [!info] 笔记用途
> 本笔记是**鉴别 / 决策导向**的衍生工具：不是 AD 单病大百科，而是教你**怎么用 30 秒在 8 个亚型里锁到答案**。
> 来源：Neuro HY200 听课草稿 模块 3（DSM-5 → AD → VaD → LBD → FTD → NPH → CJD → HIV-D → Pseudodementia → B12 SCD → 可治疗 8 大 → HD → Wilson → 5 大病理蛋白 → Tauopathy 大家族）+ breakdown 错题 1（Pseudodementia 误选 Donepezil）+ breakdown 错题 2（CJD 误选 FTD）的实战 stem。

---

## 一、Step 1: 排除"非真痴呆"（先看金标准）

```
              【认知主诉】
                  │
                  ▼
        ┌─────────────────────┐
        │   ADL / IADL 受损？   │ ← 金标准
        └─────────────────────┘
                  │
       ┌──────────┴──────────┐
       ▼                      ▼
    ❌ 否                    ✅ 是
       │                      │
       ▼                      ▼
   MCI / 正常老化         真痴呆 (Major NCD)
   (28-30 = normal           → 进 Step 2
    24-27 = MCI)
       │
       ▼
   每年 5-15% 进展 → 痴呆
```

> [!tip] MMSE 数字三条线（必背）
> - **28–30** = Normal aging（"benign senility"）
> - **24–27** = MCI（Mild NCD）
> - **< 24** = Dementia threshold（金标准）
> - **18** = Donepezil ↔ Memantine 分水岭 ⭐
> - **< 10** = Severe AD

> [!warning] 三个易混"非痴呆"
> 1. **Delirium（谵妄）** = 急性 + 波动 + 注意力差 + 可逆 → 必须先排除（找 UTI / 药 / 电解质 / 缺氧）
> 2. **Depression-Pseudodementia** = 抑郁伪装成痴呆 → 实际是 MDD（见 Step 3 招牌）
> 3. **MCI** = IADL 保留 → 不是 Dementia，每年 5–15% 进展（监测、不立刻治）

### ADL vs IADL（受损顺序：IADL 先，ADL 后）

| 类型 | 缩写 | 6/8 项 |
|---|---|---|
| **ADL** 基础生活 | **DEATH-B** | **D**ressing / **E**ating / **A**mbulating / **T**oileting / **H**ygiene / **B**owel/Bladder continence |
| **IADL** 工具性 | **SHAFT-MT** | **S**hopping / **H**ousekeeping / **A**ccounting / **F**ood prep / **T**ransportation / **M**edication / **T**elephone |

⭐ **临床观察顺序**：理财 (Accounting) → 用药 (Medication) → 做饭 (Food prep) — IADL 先丢；后期才丢 ADL（洗澡 → 穿衣 → 吃饭）

## 二、Step 2: 病程速度决定方向（最强分类法）

```
                  【真痴呆 confirmed】
                          │
                          ▼
                  病程速度多快？
                          │
        ┌─────────────────┼─────────────────┐
        ▼                 ▼                 ▼
   数周-数月            数月-1 年          数年慢性
   (RPD)               (Subacute)         (Chronic)
        │                 │                 │
        ▼                 ▼                 ▼
   ★ CJD             ★ VaD（阶梯式）      ★ AD（最常见）
   ★ B12 SCD        ★ Drug-induced       ★ LBD
   ★ NPH            ★ Subdural hematoma   ★ FTD
   ★ Hypothyroid    ★ Neurosyphilis      ★ HD
   ★ Pseudodementia ★ HIV-D             ★ Wilson
   ★ Autoimmune     ★ 重金属/酒精        ★ PD-D
   ★ Wernicke-K
```

> [!success] 一句话锁速度
> - **数周内进展 + myoclonus → CJD（直到证明不是）** ⭐
> - **数月进展 + 抑郁征 + 自知力 → Pseudodementia**
> - **数月-年阶梯式 + 局灶征 → VaD**
> - **数年缓慢 + 顺行性遗忘首发 → AD**

## 三、Step 3: 招牌特征锁定诊断

```
                      【特征锁定】
                            │
   ┌────────────────────────┼────────────────────────┐
   ▼                        ▼                        ▼
首发"记忆"              首发"行为"               首发"运动"
   │                        │                        │
   ▼                        ▼                        ▼
AD                       FTD                      ★ Parkinson 谱
(顺行性遗忘)          (去抑制 / 偏甜             ├ PD-D (运动 > 1y 后才认知)
                       Hyperorality              └ LBD (运动 < 1y 内 + 幻觉)
                       MMSE 正常)                
                                                 ★ HD = chorea (舞蹈) + 精神 + 痴呆
                                                 ★ CJD = startle myoclonus
                                                 ★ Wilson = K-F + 拍翅 + < 35 岁

   ┌────────────────────────┼────────────────────────┐
   ▼                        ▼                        ▼
首发"步态"              首发"波动"               首发"抑郁"
   │                        │                        │
   ▼                        ▼                        ▼
NPH (Wet/Wobbly/Wacky)  LBD                      Pseudodementia
VaD (Binswanger)        (good days/bad days       (concerned + 
                         + 视幻觉 + RBD)            lost interest)
```

### 招牌钩子速查表

| 招牌钩子 | 锁定 | 治疗 |
|---|---|---|
| 顺行性遗忘 + 数年慢病 | **AD** | Donepezil (MMSE >18) → Memantine (≤18) |
| **阶梯式** + 局灶神经征 + MRI 多发腔隙 | **VaD** | **Aspirin + Statin + 控 CV**（不是 Donepezil！）|
| **视幻觉 + 自发帕金森 + RBD + 波动** | **LBD** | Rivastigmine ✅；**Haloperidol/Risperidone 致命** ⛔ |
| **去抑制 + Hyperorality + MMSE 正常 + 50–60 岁** | **FTD** | SSRI / Trazodone；**AChE-i 加重** ⛔ |
| **Wet / Wobbly / Wacky + sulci 正常** | **NPH** | **VP shunt** ⭐ 可逆 |
| **快速进展 + Startle myoclonus + EEG triphasic** | **CJD** | 无（致死）|
| **CD4 < 200 + 皮质下 + 抑郁** | **HIV-D** | ART |
| **数周 + concerned + lost interest + labs normal** | **Pseudodementia** | **SSRI**（Citalopram 老人 ≤ 20 mg/d）|
| **后索征 + macrocytic anemia + vegan/PPI/metformin** | **B12 SCD** | B12 IM 注射 |
| **Chorea + 家族史 + 40 岁 + 尾状核萎缩** | **HD** | VMAT2 抑制剂 (Tetrabenazine) |
| **< 35 岁 + K-F 环 + 肝病 + 拍翅震颤** | **Wilson** | Penicillamine / Zinc |

## 四、可治疗痴呆 8 大 ⭐ USMLE 必背（"漏一个就是医疗事故"）

> [!danger] 任何痴呆患者必筛 workup
> **TSH + Vit B12 + RPR/VDRL + CBC + CMP + Folate + 抑郁筛 + MRI**
> 漏掉 → 错过可治；考试漏掉 → 错过题。

| # | 病因 | Stem 钩子 | 治疗 |
|---|---|---|---|
| 1 | **Pseudodementia (MDD)** ⭐ | concerned about memory + lost interest + 数周 + labs 正常 | **SSRI** |
| 2 | **B12 SCD** | 后索征 (Romberg+/振动↓) + macrocytic anemia + vegan/PPI/metformin/N₂O | B12 IM |
| 3 | **Hypothyroidism** | 怕冷 + 便秘 + bradycardia + 体重↑ | Levothyroxine |
| 4 | **Neurosyphilis** | **Argyll-Robertson pupil + tabes dorsalis + (+) VDRL** | IV Penicillin |
| 5 | **Neuroborreliosis (Lyme)** | 流行区 + 蜱叮 + EM + 颜面瘫 history | Doxy / Ceftriaxone |
| 6 | **NPH** | **Wet / Wobbly / Wacky** + sulci 正常 | **VP shunt** |
| 7 | **慢性硬膜下血肿 (cSDH)** | 老人 + 跌倒 + 抗凝史 + 波动认知 | 手术引流 |
| 8 | **药物毒性** | 抗胆碱 (Benztropine/Diphenhydramine) / BZD / 阿片 | **停药** |

> [!tip] 早些时候做对 "labs 都正常 + 抑郁征 + 速度快"，**Pseudodementia 优先怀疑**（breakdown 错题 1 钉死的教训）

| 维度 | **AD** | **VaD** | **LBD** | **FTD** | **NPH** | **CJD** | **HIV-D** |
|---|---|---|---|---|---|---|---|
| **首发** | 顺行性遗忘 | 执行↓ + 局灶征 | 视幻觉 + 帕金森 | 行为 / 性格改变 | Wet / Wobbly / Wacky | 快速进展 + myoclonus | 皮质下 + 抑郁 |
| **病程** | 数年 | **阶梯式** | 数年波动 | 数年 | 数月-年 | **数周-月**（< 12 月致死）| 数月-年 |
| **MMSE** | 进行性↓ | 多次大跌 | 波动 | **正常或近正常** ⭐ | 可低 | 急剧↓ | 中等↓ |
| **病理** | Aβ + tau | (梗死) | α-synuclein | tau / TDP-43 / FUS | (脑积水) | Prion (PrP^Sc) | (HIV) |

## 六、Alzheimer Disease (AD) 深度模块 ⭐ 最高频

### 6.1 风险因素：早发 vs 晚发（必考对照）

| 类型 | 年龄 | 最强 risk | 遗传模式 |
|---|---|---|---|
| **早发家族性 AD (EOAD)** | < 65 岁 | **Family history** ⭐ | 常染色体显性（APP / PSEN1 / PSEN2）|
| **晚发散发 AD (LOAD)** | ≥ 65 岁 | **Age** ⭐ | 多基因 + APOE ε4 修饰 |

### 6.2 早发家族性 AD 三大基因

| 基因 | 染色体 | 功能 | 关联 |
|---|---|---|---|
| **APP** (Amyloid Precursor Protein) | **chr 21** | Aβ 前体 | **Down 综合征**（3 拷贝 → APP↑ 50% → 早发 AD）|
| **PSEN1** (Presenilin 1) | chr 14 | γ-secretase 催化亚基 | **最常见**早发家族性 AD |
| **PSEN2** (Presenilin 2) | chr 1 | γ-secretase 亚基 | 罕见 |

### 6.3 APOE 等位基因风险修饰

- **ε4 / ε4** → 风险↑↑（"**4 for AD**"）
- **ε2** → **保护**（"**pro-2-ctive**"）
- ε3 = neutral（最常见）

### 6.4 APP 切割通路 ⭐

```
                APP (Amyloid Precursor Protein)
                          │
              ┌───────────┴───────────┐
              ▼                       ▼
       α + γ-secretase           β + γ-secretase
        (NON-amyloidogenic)        (amyloidogenic)
              │                       │
              ▼                       ▼
        soluble fragments        Aβ peptide (Aβ40 / Aβ42)
        (safe, normal)            │
                                  ▼
                            细胞外聚集 → Senile plaques
                                  │
                                  ▼
                            突触功能↓ + 神经毒性
                            
   ⭐ PSEN1/PSEN2 突变 = γ-secretase 异常 → Aβ42/Aβ40 比例失衡 → 早发 AD
   ⭐ AD 唯一可"针对病理"的药 = anti-Aβ monoclonals (Lecanemab / Aducanumab)
                                  其余都是对症 (Donepezil / Memantine)
```

### 6.5 AD 病理 5 特征

1. **Senile plaques**（Aβ，**细胞外**）
2. **Neurofibrillary tangles (NFT)**（**hyperphosphorylated tau**，**细胞内** — 数量与痴呆程度相关）⭐
3. **Hirano bodies**（嗜酸性 actin 包涵体，海马）
4. **Granulovacuolar degeneration**（海马）
5. **海马 + 颞顶 cortex 萎缩 + sulci 增宽 (ex vacuo)**
- 并发：**CAA (Cerebral Amyloid Angiopathy)** = Aβ 血管壁沉积 → **脑叶 (lobar) 出血** ⭐

### 6.6 临床 5 大域受累顺序

| 域 | 表现 | 时期 |
|---|---|---|
| **记忆 (Memory)** | **顺行性遗忘** ⭐ | **首发** |
| 视空间 (Visuospatial) | 迷路（own neighborhood）| 早-中 |
| 执行 (Executive) | 不会理财 / 复杂任务 | 早-中 |
| 语言 (Language) | 找词难 / poverty of speech | 中 |
| 运动 (Motor) | gait apraxia / 跌倒 | **晚期** |

### 6.7 治疗分层（MMSE 18 cutoff）⭐⭐

```
   MMSE > 18 (Mild AD)         MMSE ≤ 18 (Mod-Severe)
        │                              │
        ▼                              ▼
   Donepezil (AChE-i)            Memantine (NMDA antagonist)
   首选                            ★ 加用，不是替换 ★
                                  (可联合 Donepezil)
        │                              │
        └──────────┬───────────────────┘
                   ▼
            早期 MCI / 早 AD（影像确认 Aβ+）
                   │
                   ▼
            Lecanemab (anti-Aβ monoclonal)
            ⚠️ ARIA 风险（脑水肿 / 出血），定期 MRI
```

> [!warning] 机制易混
> - **AChE-i ≠ ACEi**（别混！）
> - **Memantine ≠ anti-Aβ** — 是 NMDA 拮抗剂（截断兴奋毒性下游）
> - **所有 AD 药物本质都是对症** — 只有 anti-Aβ 单抗"针对病理"

### 6.8 AD "三层楼"用药模型 ⭐

```
   病因层：Aβ 沉积  ←  Anti-Aβ 单抗 "剪源头"
              │
              ▼
   毒性层：谷氨酸兴奋毒性  ←  Memantine "踩刹车"
              │
              ▼
   症状层：ACh ↓  ←  AChE-i (Donepezil / Rivastigmine / Galantamine) "打补丁"
```

### 6.9 Sundowning（日落综合征）处理

| 选择 | 对错 |
|---|---|
| **降噪 + 减刺激 + 规律作息** | ✅ 首选 |
| **熟悉的人陪伴 + 白天充足光线** | ✅ |
| **Bright illumination 24/7** | ❌ NBME 经典错误答案（破坏昼夜节律）|
| **BZD** | ❌ 加重 + 跌倒风险 |

### 6.10 BPSD（行为精神症状）金原则

**非药物 first**，按症状选药：

| 症状 | 一线 |
|---|---|
| 抑郁 | **SSRI** |
| 严重激越 / 幻觉 | **Quetiapine 短期低剂量**（黑框警告：↑死亡）|
| **失眠** | **Trazodone 首选** ⭐（不是 BZD）|

❌ **禁用**：BZD / 抗胆碱（Benztropine / Diphenhydramine / Hydroxyzine）/ Haloperidol 高剂量

### 6.11 Beers Criteria — 老年人禁忌药 5 类 ⭐

| 类别 | 例子 | 原因 |
|---|---|---|
| **抗胆碱** | 一代抗组胺、TCA、Oxybutynin、Diphenhydramine | 谵妄 / 跌倒 / 尿潴留 |
| **心血管** | α-1 blocker、中枢 α-2、多种抗心律失常药 | 体位性低血压 / 跌倒 |
| **CNS** | TCA、抗精神病、Barbiturate、**BZD** | 谵妄 / 跌倒 / 摔倒髋骨折 |
| **内分泌** | 长效磺脲（Glyburide）、Sliding scale insulin | 低血糖 |
| **疼痛** | 长期非选择性 NSAID、骨骼肌松弛药 | GI 出血 / 肾损 |

### 6.12 Down 综合征早发 AD

- APP 在 chr 21 → 3 拷贝 → APP ↑ 50% → Aβ ↑↑
- **40 岁脑内 AD 病理几乎 100%**
- **50–60 岁** 约 50–70% 临床痴呆
- 进展更快（3–7 年 vs 散发 8–10 年）

## 七、各亚型核心钩子细节

### 7.1 Vascular Dementia (VaD)

> [!example]- VaD 4 大金钩 (CASF)
> - **C**V 风险因素（HTN / DM / CAD / AFib）
> - **A**brupt / Stepwise decline ⭐
> - **S**ystem-localized focal neuro signs ⭐⭐
> - **F**indings on MRI（多发腔隙 / 白质病变）

**3 亚型**：
- Multi-infarct（多次大血管卒中累积）
- Strategic infarct（单个关键位置：丘脑 / 内囊膝 / 海马 / 扣带 → 急性）
- **Subcortical vascular encephalopathy / Binswanger** = 小血管病 + 弥漫白质 + 尿失禁 + 步态异常

**治疗 = 二级预防（不是 Donepezil！）**：Aspirin + Statin + 控 BP/DM + 戒烟 + 抗凝（AFib → DOAC）

⭐ **Pronator drift（旋前肌漂移）** = UMN 征灵敏（比 hyperreflexia / Babinski 更早出）

### 7.2 Lewy Body Dementia (LBD)

> [!example]- LBD 四联
> 1. **Visual hallucinations**（看到小孩 / 动物）
> 2. **Fluctuating cognition**（good days vs bad days）
> 3. **Spontaneous parkinsonism**（bradykinesia / rigidity / tremor）
> 4. **REM Sleep Behavior Disorder (RBD)**（梦中拳打脚踢）

⭐ **LBD vs PD-D "1 年规则"**：认知 vs 运动症状间隔 **< 1 年 = LBD**；运动数年后认知 (> 1 年) = PD-D

> [!danger] LBD 致命药物陷阱 ⭐⭐⭐
> LBD 患者**极度敏感于抗精神病药** → Haloperidol / Risperidone **致命 NMS 样反应**
> - 视幻觉首选：**Rivastigmine**（AChE-i）
> - 必要时：**低剂量 Quetiapine**（D2 亲和力低）

### 7.3 Frontotemporal Dementia (FTD)

> [!example]- bvFTD 6 大核心
> 1. **Disinhibition（去抑制）** ⭐ — 不当评论 / 性行为 / 偷窃
> 2. **Apathy / 惯性**
> 3. **同情心丧失**
> 4. **强迫 / 仪式化行为**
> 5. **Hyperorality / 饮食改变** ⭐ — 偏甜偏碳水 + 增重 + pica
> 6. **执行功能↓**

⭐ USMLE 招牌：**MMSE 正常或近正常**（记忆相对保留）+ **50–60 岁起病** + **额颞叶 MRI 萎缩** + **家族性 20–40%**（C9orf72 / MAPT / GRN）

**病理 3 类**：
- **FTLD-tau**（Pick bodies，MAPT 突变）
- **FTLD-TDP**（C9orf72, GRN）⭐ 与 **ALS 谱系**重叠
- **FTLD-FUS**

**治疗**：SSRI / Trazodone 对症；**AChE-i 可能加重** ❌

### 7.4 Normal Pressure Hydrocephalus (NPH) ⭐ 可治愈

> [!success] 金三联 **Wet / Wobbly / Wacky**
> - **Wet** = 尿失禁（**最先出现**）
> - **Wobbly** = 磁性步态（magnetic gait）
> - **Wacky** = 痴呆（**最后才出现**）

**影像 vs AD（金标准鉴别）** ⭐
- AD = 脑室扩大 + **sulci 增宽**（ex vacuo）
- NPH = 脑室扩大 + **sulci 正常**（true hydrocephalus）

**治疗**：
- **LP tap test**（诊断 + 部分治疗：抽 30–50 mL CSF 看症状改善）
- **VP shunt**（首选）⭐
- 早期 + 三联齐全 → 症状可逆 ⭐

### 7.5 Creutzfeldt-Jakob Disease (CJD)

> [!example]- CJD 4 大金标准（2+ 几乎不会错）
> 1. **快速进展性痴呆**（数周-月，< 12 月致死）
> 2. **Startle myoclonus**（**响声诱发**）⭐ 招牌
> 3. **EEG triphasic periodic sharp-wave complexes** ⭐
> 4. **CSF 14-3-3 protein / RT-QuIC**

**病理**：PrP^C → PrP^Sc β-折叠错构 → 抗蛋白酶 → 海绵状改变 (spongiform)

**类型**：散发 85% / 家族性 (PRNP 突变) / 医源性 (角膜移植、神经外科器械、生长激素提取) / 变异型 vCJD (疯牛病食用)

**治疗 = 无（致死）**

> [!tip] breakdown 错题 2 教训
> "Diffuse involuntary muscle jerking" + "**provoked by loud noises**" + "EEG triphasic" = CJD 三个金标准齐全
> 易错：选 FTD（FTD 数年进展，无 myoclonus，无 EEG 周期性尖波）

> [!warning] "痴呆 + 肌阵挛 + 视觉/小脑征"特异吗？——高度特征，但**不特异** ⭐
> 这组合是 CJD 招牌（USMLE 看到基本选它），但**真正锁定靠"快速 tempo + 确诊检查"，且必须排可治模仿者**：
> - **定义性 = 快速进展（周–月）+ CNS（中枢神经系统）功能障碍**；视觉异常 = **Heidenhain 变异型**、小脑征 = **Oppenheimer-Brownell 变异型**（变异型钩子，非特异判据）。
> - **必排的可治模仿者**（快速痴呆 + 肌阵挛的 DDx，漏诊可惜）：
>   - **桥本脑病 (Hashimoto encephalopathy)** = 抗甲状腺抗体↑ + 波动脑病 + 肌阵挛 → **激素可逆**
>   - **自身免疫/副肿瘤脑炎 (autoimmune/paraneoplastic encephalitis)**（anti-NMDA / anti-LGI1）→ **免疫治疗可逆**
>   - **中毒/代谢性脑病**（锂 lithium / 铋 bismuth / 尿毒症）→ 纠因可逆
> - **提供特异性的检查**：MRI **DWI 皮层飘带征 (cortical ribboning) + 基底节高信号**（高特异）/ EEG 周期性尖波 / CSF **RT-QuIC（现最特异）**、14-3-3（敏感不特异）；**确诊金标准 = 脑组织海绵状变性 (spongiform，活检/尸检)**。
> - 钥匙：**招牌组合 → 想 CJD，但先排桥本脑病/自免脑炎（可治），特异性靠 DWI + RT-QuIC。**

### 7.6 HIV-Associated Dementia (HIV-D)

- **CD4 < 200** 触发
- 皮质下功能障碍 + 共济失调 + 抑郁 + 锥体束征
- MRI: 弥漫灰质 + 皮层下萎缩 + ex vacuo
- 治疗：**ART**

**HIV+ 痴呆鉴别**：HIV-D / **PML (JC virus)** / CMV ventriculitis / **CNS lymphoma** / Cryptococcal meningitis

### 7.7 Pseudodementia（抑郁假性痴呆）⭐

> [!example]- Pseudodementia 5 大钩子
> 1. **起病数周-月**（快）
> 2. **病人自己关心**（自知力存在）⭐
> 3. **回答"我不知道"**（放弃式 — vs AD 编造 confabulation）
> 4. **抑郁征核心**（≥ 5/9 **SIG E CAPS + 抑郁情绪**：Sleep / Interest / Guilt / Energy / Concentration / Appetite / Psychomotor / Suicidal）
> 5. **抑郁先于认知**

**治疗 = SSRI ⭐**：Sertraline / Citalopram（**≤ 20 mg/d 老人**，QT 延长警告）4–6 周认知应改善 → 不改善 = 重查 dementia

> [!tip] 黄金鉴别口诀
> **"Pseudodementia patients complain; AD patients are brought in"**
> （假性痴呆病人**自己**抱怨；真痴呆病人**家属带来**）

> [!warning] breakdown 错题 1 教训
> 看到 78 岁 + 5 周健忘 → 反射性想 AD → 选 Donepezil ❌
> 真相：**5 周** (太快) + **concerned about memory** (自知力) + **5 项 MDD** (PMR/blunt/失眠/疲乏/兴趣丧失) + **labs 正常** → **Pseudodementia → Citalopram**

### 7.8 Subacute Combined Degeneration (B12 SCD)

**3 个束累 + 1 个不累（反向记忆法）** ⭐
- 累及：**Dorsal columns**（振动 / 本体）+ **Lateral CST**（UMN）+ **Spinocerebellar**（共济失调）
- **Spinothalamic 不累及**（**痛温觉保留**！）⭐ 反向记忆

**高危人群**：Vegan / "tea and toast diet" / **Pernicious anemia** / Crohn / **Metformin 长期** / PPI / **N₂O 滥用** (whippets / 拔牙麻醉滥用)

**诊断**：B12 ↓（但可正常）/ **MMA + Homocysteine 更敏感** / 神经症状**可早于贫血**

**治疗 = B12 IM 注射**

### 7.9 Huntington Disease (HD)

- **AD inheritance** / **HTT 基因** chr 4 / **CAG repeat > 40** + **anticipation**（**父系**更早 + 更重）
- 发病 ~ 40 岁；几乎必有家族史 (< 3% 散发)

**三联**：
1. **精神症状**（早期 + **自杀风险↑**）
2. **Chorea**（舞蹈样）⭐
3. 痴呆（皮质下，执行功能↓）

**病理**：**尾状核 (caudate) 萎缩** + 脑室代偿性扩大（"box-like" 侧脑室）+ mutant huntingtin

**治疗**：
- Chorea → **VMAT2 inhibitors**（Tetrabenazine / Deutetrabenazine / Valbenazine）
- 抑郁 → SSRI
- 精神病性 → 非典型抗精神病
- **无疾病修饰治疗**

### 7.10 Wilson Disease

- **AR inheritance** / **ATP7B 基因** chr 13 / **< 35 岁**发病 ⭐

**三系统**：
1. **肝** = 肝炎 → 肝硬化（**年轻人肝硬化想 Wilson**）
2. **脑** = 基底节铜沉积 → **拍翅样震颤 (asterixis-like wing-beating)** + 精神症状 + 痴呆
3. **眼** = **Kayser-Fleischer ring**（Descemet 膜铜沉积）

**诊断**：裂隙灯（K-F 环）+ **Ceruloplasmin ↓** + 24h 尿铜↑ + 肝活检（金标准）

**治疗**：低铜饮食 + **Penicillamine**（首选铜螯合，副作用 SLE 样反应）/ Trientine（二线）/ Zinc 盐（维持）/ 肝移植（暴发 / 失代偿）

## 十、USMLE 高频陷阱（NBME 反复考的圈套）

> [!danger] 11 个必记陷阱
> 1. **78 岁 + 数周健忘 → 反射性选 Donepezil = 错**（先想 Pseudodementia 8 大可治）
> 2. **Memantine ≠ anti-Aβ** — 是 NMDA 拮抗
> 3. **Donepezil 不是 ACEi！** — 是 AChE-i
> 4. **LBD + Haloperidol = 致命 NMS** ⛔（Rivastigmine 首选）
> 5. **FTD + AChE-i = 可能加重** ⛔
> 6. **VaD 不要给 Donepezil** — 给 Aspirin + Statin + 控 CV
> 7. **NPH 不要漏！** — Wet/Wobbly/Wacky → VP shunt → **可逆**
> 8. **AD sundowning 不要 "bright light 24/7"** — 破坏昼夜节律
> 9. **CJD 看到 startle myoclonus + EEG triphasic → 不要选 FTD/AD**
> 10. **Pseudodementia 答 SSRI，不是 Donepezil**
> 11. **B12 SCD 神经症状可早于贫血** — 别等 macrocytic anemia 才查 B12

## Memory Hook

> [!success] 痴呆决策三步走
> 1. **先排除"非痴呆"** — ADL/IADL 受损 + MMSE < 24 才是真痴呆
> 2. **看速度** — 数周（CJD/Pseudo/B12/NPH 8 大可治）/ 数月（VaD 阶梯）/ 数年（AD/LBD/FTD/HD/Wilson）
> 3. **看招牌** — 记忆 → AD / 行为 → FTD / 视幻觉 → LBD / 步态 → NPH / 阶梯 + 局灶 → VaD / 闪电 myoclonus → CJD / concerned → Pseudo

> [!tip] 三句话保命
> 1. "**Pseudodementia patients complain; AD patients are brought in**"
> 2. "**Haloperidol in LBD = NMS death**"
> 3. "**Wet/Wobbly/Wacky + sulci 正常 = NPH 漏诊医疗事故，VP shunt 可逆**"

> [!tip] 治疗速记口诀
> - AD = **D**onepezil → **M**emantine（MMSE 18 cutoff）→ Lecanemab
> - VaD = **Aspirin + Statin**（不是 D！）
> - LBD = **R**ivastigmine + 必要时 Quetiapine（永远不 Haloperidol）
> - FTD = **SSRI / Trazodone**（不要 AChE-i）
> - NPH = **VP shunt**
> - Pseudo = **SSRI**
> - CJD = 无
> - HIV-D = **ART**
> - B12 = **B12 IM**
> - HD chorea = **Tetrabenazine** (VMAT2)
> - Wilson = **Penicillamine / Zinc**

# 面瘫 vs 球麻痹 — UMN/LMN 不可跨神经类比

> [!info] 起源
> 派生自一次概念讨论：把 **Bell palsy（贝尔面瘫）** 误当成 **bulbar palsy（球麻痹）**，并进一步把"中枢性面瘫"误以为包含"LMN（球麻痹）+ UMN"。核心误区是把 **UMN/LMN 这个层次标签跨神经互相套用**。本笔记建立两套独立体系的鉴别框架，作为复习单一真相源。

---

## §一 一句话锁定

> [!tip]
> - **面瘫 (facial palsy)** 永远只讨论 **CN VII (cranial nerve VII, 第7对颅神经 / 面神经)**，分**周围性（LMN, lower motor neuron, 下运动神经元 = Bell）** vs **中枢性（UMN, upper motor neuron, 上运动神经元 = 卒中等）**。
> - **球麻痹 (bulbar palsy)** 讨论的是 **CN IX / X / XI / XII（后组颅神经）**，跟脸无关；它本身又分**真性球麻痹（LMN）** vs **假性球麻痹（pseudobulbar, UMN）**。
> - **UMN/LMN 只是"损伤层次"标签，不能跨神经类比**：面神经的 UMN ≠ 球部神经的 UMN，它们是不同的解剖通路。

---

## §二 关键认知：先问"哪根神经"

> [!danger] 最常见串台
> "中枢性面瘫包不包括球麻痹？" —— **不包括**。球麻痹根本不属于"面瘫"这个分类体系。两者是两套独立的神经，不能放进同一个 UMN/LMN 框里互相套。

```
面瘫 (Facial palsy) ────► 只管 CN VII
   ├─ 周围性 (peripheral) = LMN = Bell palsy → 整侧脸瘫(含额)
   └─ 中枢性 (central)    = UMN = 卒中等     → 额头保留, 只下半脸

球麻痹 (Bulbar palsy) ──► 只管 CN IX / X / XI / XII (跟脸无关)
   ├─ 真性球麻痹  = LMN (延髓核/根)
   └─ 假性球麻痹 (pseudobulbar) = UMN (双侧皮质延髓束)
```

- **中枢性面瘫 = 纯 UMN**（皮质延髓束在面神经核**以上**受损，如卒中），**只有 UMN 一种**，不含 LMN，更与球麻痹无关。
- **球麻痹是另一条赛道**：CN IX–XII 的问题，本身才分真性（LMN）/ 假性（UMN）。

## §三 面瘫：周围性 (Bell) vs 中枢性 — 机制对账

> [!info] 为什么"中枢性面瘫额头保留、周围性面瘫额头也瘫"
> 面神经核分**上半部**（管额肌 frontalis / 闭眼 orbicularis oculi）和**下半部**（管下半脸）。
> - 上半部接受**双侧**大脑皮质支配；下半部只接受**对侧**皮质支配。
> - **中枢性（UMN，核以上，如 stroke）**：对侧皮质通路断了，但上半部还有**同侧皮质兜底** → **额头保留 (forehead sparing)**，只下半脸瘫。
> - **周围性（LMN，Bell，核或核以下）**：整个面神经核/干坏死，没有兜底 → **整侧脸瘫，含额头**。

| | **周围性面瘫 (Bell palsy)** | **中枢性面瘫 (central, 如 stroke)** |
|---|---|---|
| 层次 | **LMN（下运动神经元）** | **UMN（上运动神经元）** |
| 定位 | 面神经核或核以下（周围段） | 皮质延髓束，面神经核**以上** |
| 额头/抬眉/闭眼 | **受累**（不能抬眉、闭眼不全） | **保留 (forehead sparing)** |
| 范围 | **整个半边脸** | 仅对侧**下半脸** |
| 伴随 | 味觉↓(前2/3舌)、听觉过敏、流泪异常 | 偏瘫、构音障碍等卒中其他定位征 |
| 病因 | 特发性（HSV-1, herpes simplex virus type 1, 1型单纯疱疹病毒 再激活相关）| 卒中、肿瘤等 CNS (central nervous system, 中枢神经系统) 病变 |
| 处理 | 72h 内口服**激素 (prednisone, 泼尼松)** ± **valacyclovir 伐昔洛韦**；眼保护 | 按卒中流程（影像 + 病因治疗）|

## §四 球麻痹：真性 (bulbar) vs 假性 (pseudobulbar)

| | **真性球麻痹 (bulbar palsy)** | **假性球麻痹 (pseudobulbar palsy)** |
|---|---|---|
| 层次 | **LMN**（延髓 IX–XII 核或神经根）| **UMN**（**双侧**皮质延髓束）|
| 舌 | **萎缩 + 纤颤 (fasciculations)** | **痉挛、无萎缩** |
| 咽反射 gag | **消失** | **亢进** |
| 下颌反射 jaw jerk | 正常/减弱 | **亢进** |
| 情绪 | 正常 | **情绪失禁 (emotional lability)**，无故哭笑 |
| 共同表现 | 均有 **吞咽困难 dysphagia + 构音障碍 dysarthria/dysphonia** | 同左 |
| 常见病因 | ALS (amyotrophic lateral sclerosis, 肌萎缩侧索硬化)（LMN 部分）、延髓型 MG (myasthenia gravis, 重症肌无力)、GBS (Guillain-Barré syndrome, 吉兰-巴雷综合征)、延髓卒中、polio (poliomyelitis, 脊髓灰质炎) | 双侧卒中、MS (multiple sclerosis, 多发性硬化)、ALS（UMN 部分）|

> [!warning] ALS 为什么经典
> **肌萎缩侧索硬化 (ALS, amyotrophic lateral sclerosis)** 可**同时**有真性球麻痹（LMN：舌萎缩纤颤）和假性球麻痹（UMN：情绪失禁、反射亢进）特征。**LMN + UMN 共存且无感觉受累**正是 ALS 的诊断核心。

| | 周围性面瘫(Bell) | 中枢性面瘫(卒中) | 真性球麻痹 | 假性球麻痹 |
|---|---|---|---|---|
| 神经 | CN VII | CN VII | CN IX–XII | CN IX–XII |
| 层次 | LMN | **UMN** | LMN | UMN |
| 额头 | 瘫 | 保留 | 与脸无关 | 与脸无关 |
| 招牌 | 整侧脸瘫 | 下半脸瘫 | 舌萎缩+纤颤、咽反射消失 | 情绪失禁、反射亢进 |

## §六 复习自测

> [!question]- 单侧面瘫，抬眉时患侧额纹**消失**、闭眼不全——周围性还是中枢性？为什么？
> **周围性（LMN，Bell palsy）**。额头受累说明面神经核或核以下损伤，没有同侧皮质兜底。若是中枢性（UMN）则额头保留。

> [!question]- "中枢性面瘫是不是既有 UMN 又有 LMN（球麻痹）？"
> **不是。** 中枢性面瘫 = 纯 UMN（CN VII 通路核以上受损），不含 LMN，也与球麻痹无关。球麻痹是 CN IX–XII 的另一套体系，它本身才分真性(LMN)/假性(UMN)。UMN/LMN 不能跨神经类比。

> [!question]- 真性 vs 假性球麻痹，舌头和情绪各有什么相反表现？
> 真性(LMN)：舌**萎缩+纤颤**、咽反射**消失**、情绪正常。假性(UMN)：舌**痉挛无萎缩**、反射**亢进**、**情绪失禁**。同时具备两者 + 无感觉受累 → 想 ALS。

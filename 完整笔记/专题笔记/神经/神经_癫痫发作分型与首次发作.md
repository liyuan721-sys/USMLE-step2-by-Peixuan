# 神经_癫痫发作分型与首次发作（Seizure Classification & First Unprovoked Seizure）

> [!info] 一句话定位
> 三条主线：**①发作分型（局灶 focal vs 全面 generalized）**；**②Todd 麻痹（postictal 一过性缺损）机制 + vs 卒中**；**③成人首次无诱因发作（first unprovoked seizure）的 workup 与影像选择**。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q2280]]（局灶发作误选 JME）+ [[mistakes/uworld-mistakes_2026-06#^Q110558]]（首次发作 + 肿瘤史 → MRI 误选补糖）。

## 一、发作分型（ILAE）

```
按起源
 ├─ Focal（局灶，起于单侧半球）
 │    ├─ 意识保留（aware，旧名 simple partial 简单部分）
 │    └─ 意识障碍（impaired awareness，旧名 complex partial 复杂部分）→ 自动症 automatisms
 │         最常起源 = 颞叶 temporal lobe ⭐（故 postictal confusion 尤其突出）
 │         可进展为 → focal to bilateral tonic-clonic（继发全面化）
 └─ Generalized（全面，起即累及双侧）
      ├─ Absence（失神）
      ├─ Tonic-clonic（强直阵挛）
      ├─ Myoclonic（肌阵挛）
      └─ Atonic / Tonic
```

## 二、局灶发作 vs 失神 ⭐（最高频鉴别）

| 特征 | **Focal（伴意识障碍）** | **Absence（失神）** |
|---|---|---|
| 起源 | **单侧半球（最常颞叶 temporal lobe）** | **全面（双侧）** |
| 时长 | 数分钟 | **10-20 秒** |
| 自动症 automatisms | **常有**（咀嚼、咂嘴、抠摸）| 偶有轻微 |
| 发作后 postictal | **有**（混乱/疲倦 ± Todd 麻痹）⭐ | **无**（立即恢复）⭐ |
| 过度通气诱发 | **不**诱发 | **诱发** ⭐ |
| EEG | 局灶痫样；**间期可正常** | **3 Hz 棘慢波** |
| 年龄/背景 | 任意；可有结构病灶 | 学龄儿童 |

> [!tip] postictal 与诱发是钥匙
> 有 postictal 混乱 + 自动症 + 单侧起源 = **focal**；无 postictal + 过度通气诱发 + 3Hz = **absence**。

> [!info]- Focal impaired awareness = complex partial ⭐ 最常定位颞叶（temporal lobe）
> **回答"局灶发作也会 postictal confusion 吗？"** —— 会，就是这一型。
> - **旧名对应**：focal aware seizure（局灶知觉保留）= simple partial（简单部分）；**focal impaired awareness seizure（局灶知觉受损）= complex partial（复杂部分）**。
> - **postictal confusion 看的不是放电范围大小，而是发作有没有打断意识网络**。complex partial 最常起于**颞叶内侧（mesial temporal，海马 hippocampus + 杏仁核 amygdala）**，扩散到双侧边缘系统（limbic）+ 丘脑-皮层意识环路 → 发作后这些网络需时间恢复 → **postictal confusion 尤其突出、可持续数分钟，并伴对发作过程的遗忘（amnesia）**。
> - **经典 stem**：愣神 / 行为终止（behavioral arrest, staring）+ **automatisms（自动症：咂嘴 lip-smacking、咀嚼、抠摸衣角）** + 事后茫然遗忘 → focal impaired awareness seizure，定位颞叶。
> - **可有 aura（先兆）**：déjà vu（似曾相识）/ jamais vu（陌生感）、上腹 rising sensation、嗅幻觉（olfactory hallucination，焦糊味）、强烈恐惧 —— 均为颞叶 aura；详见 [[完整笔记/专题笔记/心内/心内_晕厥鉴别决策树]]（颞叶癫痫 4 大经典 aura）。
> - **别和 absence 混**：absence（失神）也"愣神"，但**无 aura、无 automatisms（或极轻）、无 postictal、10-20 秒瞬间恢复、过度通气诱发、EEG 3Hz 棘慢波**（见上表）。**absence 无 postictal confusion 是反直觉考点**。

![[{C049E8DC-B236-427B-9345-23B617CB96EC}.png|550]]
*儿童急性偏瘫病因鉴别 (UWorld)：缺血性卒中 (ischemic stroke) / 颅内出血 (intracranial hemorrhage) / **Todd 麻痹 (Todd paralysis，发作后一过性局灶无力)** / 偏瘫型偏头痛 (hemiplegic migraine)。Todd 麻痹 = 发作后局灶无力、影像正常、通常 <36h 自行恢复（发作前肢体抽动/意识丧失 + 发作后意识模糊是钥匙）。*

## 二·补、自动症（automatisms）鉴别谱 ⭐

> [!warning] "有自动症 ≠ 一定是 focal"
> 自动症（automatisms）= **意识受损**状态下的**协调、无目的、刻板重复**动作，事后常遗忘。它见于多种 seizure，也有**清醒的非癫痫模仿者**——别一见咂嘴/摸索就反射 complex partial。

**按动作类型分（记类型比记病名更有用）：**

| 类型 | 表现 | 最常提示 |
|---|---|---|
| **口部/进食型 oroalimentary** | 咂嘴 lip-smacking、咀嚼、吞咽、舔唇 | **颞叶** focal impaired awareness ⭐ |
| **手部/姿势型 manual/gestural** | 抠摸衣角、搓手、摆弄 fumbling | 颞叶 / 额叶 |
| **过度运动型 hypermotor** | 蹬车样、骨盆冲撞、剧烈乱动 | **额叶 frontal lobe**（常误当心因性）|
| **走动/言语型 ambulatory/verbal** | 无目的走动、重复词句、哼唱 | 颞叶 / 额叶 |

**会出现自动症的 seizure（不止 focal）：**

- **Focal impaired awareness（complex partial 复杂部分）** —— 最经典，尤其颞叶；口部+手部自动症是招牌，伴 postictal confusion + 遗忘。
- **Absence（失神发作，全面性）** ⭐ **反直觉**：失神**也可有自动症**（轻微咂嘴、眼睑扑动、小幅手动），尤其发作 >7-10 秒者 → 故"有自动症"**不能**直接判 focal，靠**时长 + 无 aura + 无 postictal + 过度通气诱发 + 3Hz** 区分。
- **额叶发作 frontal lobe seizure** —— 怪异 hypermotor 自动症，夜间成串、时间短、恢复快，常误诊为 **PNES（psychogenic nonepileptic seizure，心因性非癫痫发作）**。
- **继发全面化 / postictal** —— GTC（generalized tonic-clonic，全面强直阵挛）发作后恢复期可有摸索、走动样自动症。

**清醒的"非癫痫模仿者"（USMLE 干扰项）：**

| 模仿者 | 像什么 | 拆穿钥匙 |
|---|---|---|
| **迟发性运动障碍 tardive dyskinesia (TD)** | 口-颊-舌不自主运动 ≈ 口部自动症 | **病人清醒、意识不受损、非阵发、长期抗精神病药史** ⭐ |
| **抽动 tics / 刻板行为 stereotypies** | 短促刻板动作 | **意识清楚、可短暂主动抑制**（自动症不能）|
| **谵妄 delirium 的摸空 carphologia/floccillation** | 抓被单、摸空 | 背景是**波动性意识混乱**、危重/终末 |
| **睡眠障碍 parasomnia**（梦游 / RBD（REM sleep behavior disorder，REM 睡眠行为障碍）/ confusional arousal 意识模糊性觉醒）| 夜间自动样行为 | 与**额叶夜间发作**需鉴别（睡眠期、脑电）|

> [!tip] 见咂嘴/摸索先问三件事
> ① **意识受损吗**（清醒 → 想 TD / tic）；② **多长时间 + 有没有 postictal**（区分 focal vs absence）；③ **能否主动抑制**（能抑制 → tic / stereotypy，非自动症）。

## 三、Todd 麻痹（Todd paralysis / Todd 瘫痪）⭐

> [!info] 定义
> **局灶性发作（focal seizure）之后**出现的**一过性局灶性神经缺损** —— 最常见是发作累及侧对应肢体的**偏瘫/无力（hemiparesis）**，也可为失语、凝视麻痹、感觉缺失、偏盲。典型持续**数分钟~数小时（通常 <48h，多在 24h 内）后完全恢复**，影像无急性病灶。属**发作后现象（postictal phenomenon）**。

### 3.1 机制（一过性"神经元力竭 + 抑制"状态）

> [!info]- 机制推导：发作后皮层「力竭」（神经元耗竭 + 主动抑制 + 灌注低谷）
> 原理尚无定论，主流机制是发作过后受累皮层进入暂时性"力竭"：
>
> ```
> 异常同步过度放电（发作）
>         │ 发作停止后
>         ▼
> ① 神经元耗竭 + 主动抑制
>    递质储备耗尽 + GABA（γ-氨基丁酸，主要抑制性递质）能回路代偿性过度激活
>    → 把该区皮层「按住」，对应功能暂时罢工
> ② 神经元代谢障碍
>    ATP 耗竭 / 乳酸堆积 / Na⁺-K⁺-ATPase 跟不上
>    → 膜电位无法复位，处于不可兴奋期
> ③ 脑血流失配
>    受累皮层局部灌注下降（hypoperfusion）+ 神经血管耦联失调
>         │ 代谢与抑制恢复正常后
>         ▼
>      功能完全恢复（<48h）
> ```

> [!tip] 一句话
> 发作把这块皮层「用到力竭」→ 发作后进入暂时性抑制 + 代谢/灌注低谷 → 对应功能暂时瘫掉 → 代谢和抑制回正就好了。

### 3.2 vs 缺血性卒中（真正的考点）⭐

> [!danger] 刚抽搐完的病人出现偏瘫 = 首要鉴别 Todd 麻痹 vs stroke

| 特征 | **Todd 麻痹** | **缺血性卒中 (ischemic stroke)** |
|---|---|---|
| 前驱 | **明确发作史在先** + postictal 意识模糊 ⭐ | 无发作，突发 |
| 病程 | 一过性、**逐渐自行恢复**（分钟~小时，<48h） | 持续不缓解 |
| 定位 | 缺损侧对应**放电起源皮层** | 对应**血管分布区** |
| 影像 | 无急性梗死 | 弥散受限 / 梗死灶 |
| 处理 | 临床判断即可**观察等待**；不确定仍按急性缺损排查 | 急诊评估溶栓 |

### 3.3 定位与临床价值

- **定位反推**：缺损侧 → 对侧运动皮层为致痫灶（如**右侧**肢体 Todd 无力 → **左侧**运动皮层放电）。
- Todd 麻痹**不等于结构病灶**，但**反复固定同一侧**提示该侧可能有致痫灶 → 值得 MRI 评估（见 §五 red flags）。
- 它是**发作后现象，不是发作本身**，也不提示卒中。

## 四、儿童常见癫痫综合征（鉴别陷阱）

| 综合征 | 钩子 |
|---|---|
| **JME（juvenile myoclonic epilepsy，青少年肌阵挛癫痫）** | 青少年、**晨起肌阵挛**（甩东西）、可伴 GTC；瓦普罗酸/左乙拉西坦 |
| **Lennox-Gastaut** | **≤5 岁**起、智力障碍、多种发作型、**慢棘慢波（<2.5Hz）** |
| **Childhood absence** | 学龄、凝视、过度通气诱发、3Hz；乙琥胺 ethosuximide |
| **BECTS（benign epilepsy with centrotemporal spikes，伴中央颞区棘波的自限性癫痫；旧名良性 Rolandic 癫痫）** ⭐ | **发育正常学龄儿**、**夜间、面部/口咽局灶抽动**（流涎、言语中断），可继发全面化；EEG **中央颞区棘波（睡眠激活）**；**MRI 正常、青春期自愈、常不需治疗** → "focal 但不是结构灶"的代表 |
| **Tic disorder（抽动）** | 短促刻板动作、**意识不受损**、可短暂抑制 |

## 五、成人首次无诱因发作 workup ⭐

> [!info] 定义
> **Unprovoked**（无诱因）= 无急性可逆诱因（如低血糖、低钠、酒精戒断、急性脑损伤）。

### 4.1 基础评估（先排急性诱因）

- 血糖（**看具体数值**：低糖致痫常 <45 且有交感症状；血糖 70 = 正常，不补糖）
- 电解质（**低钠致痫多 <120**）、钙、镁
- 毒物/酒精、肝肾功能、妊娠
- ECG（鉴别晕厥）

> [!tip] 选择性（非全套）检查的指征
> - **LP（腰穿）**：仅**疑脑膜炎/脑炎**时（发热、头痛、颈强、免疫抑制）。
> - **毒物筛查 tox screen**：仅**疑物质**时（聚会史、拟交感表现）。
> - **EEG**：**非常规**——若已明确代谢/中毒诱因（如尿毒筛阳性）则不必；**持续 24h EEG** 留给疑非惊厥性 SE（见 §六）。
> - **影像分工**：急诊首查**非增强 CT**（快速排出血/占位）；**结构灶定性靠 MRI（更敏感）**（见 §4.2）。

### 4.2 神经影像 —— MRI > CT ⭐

- 成人首次发作应做脑影像；**MRI（magnetic resonance imaging，磁共振）比 CT 敏感**（查肿瘤/海绵状血管畸形/皮质发育不良/陈旧卒中/海马硬化）。
- 急诊若需快速排出血/急性病变可先 CT，但**结构性病因定性靠 MRI**。

### 4.3 结构性病灶 red flags（提示需 MRI 找病灶）

- **局灶起源**（focal onset）或继发全面化
- **持续局灶神经缺损 / Todd 麻痹**
- **恶性肿瘤史** → 脑转移（brain metastasis）：**RCC（renal cell carcinoma，肾细胞癌）、肺癌、黑色素瘤、乳腺癌**是脑转移大户 ⭐
- 年龄较大、新发头痛、免疫抑制

### 4.4 是否启动抗癫痫药（AED）

- 单次无诱因 + 影像/EEG 正常 → 复发风险较低 → 通常**观察**。
- **有结构病灶 / EEG 痫样放电 / 异常神经查体** → 复发风险高 → **起始 AED**。

### 4.5 局灶发作 → 何时除外结构灶（structural lesion）⭐

> [!warning] "凡是 focal 都要找肿瘤"是过头，"focal 提高结构灶怀疑"才对
> **focal onset 本身 = 该处皮层有异常灶**，病灶前概率明显高于原发全面性发作 → **成人新发局灶发作默认 MRI 除外结构灶（肿瘤/转移、卒中、cavernoma 海绵状血管畸形、AVM 动静脉畸形、cortical dysplasia 皮质发育不良、mesial temporal sclerosis 海马硬化、脓肿/HSV（herpes simplex virus，单纯疱疹病毒）脑炎）**。但"凡是"不成立，分三档：

- **① 成人新发 focal → 一律 MRI 除外结构灶**（= §4.2/4.3 逻辑；focal onset / 持续局灶缺损 / Todd 麻痹 / 肿瘤史都是 red flag）。
- **② 儿童良性局灶综合征 = focal 但无结构灶** ⭐：典型 **BECTS（良性 Rolandic，见 §四）**——发育正常学龄儿 + 中央颞区棘波 + 查体正常 → **不追肿瘤，answer 是 reassure ± 不用药**；MRI 正常。（⚠️ 不典型：发育落后/局灶缺损/白天频发 → 仍要 MRI。）
- **③ provoked（诱发性）focal → 先纠诱因，但要再分两类**：
  - **代谢性（不是结构灶）**：**非酮症高血糖（nonketotic hyperglycemia）** 是引起局灶运动发作 / epilepsia partialis continua（部分性癫痫持续状态）的经典代谢原因（注意是"高"血糖）；严重电解质、药物/酒精、子痫 → 处理是**纠正诱因**。
  - **急性结构性**：**卒中、HSV 脑炎、外伤、脓肿**本身就是结构灶 → **仍要影像**。
- **④ 已确诊癫痫 + 既往已影像评估** → breakthrough focal 不必每次重复 MRI，去找诱因（漏药、睡眠剥夺、感染、药物相互作用）。

> [!tip] 一句话
> 核心思维不是"focal 就有肿瘤"，而是"**focal 就要主动排除结构灶**——排除后可能是**儿童良性综合征（BECTS）**或**代谢性（高血糖）**"。

## 六、易混陷阱（普适）

- 见癫痫**反射补葡萄糖** → 错：先看血糖数值（70 正常）。
- **持续 24h EEG** = 用于疑**非惊厥性 SE（nonconvulsive status epilepticus）**；首次发作后能唤醒/在恢复者不是 SE（见 [[完整笔记/专题笔记/神经/神经_癫痫持续状态SE]]）。
- **间期 EEG 正常不排除局灶癫痫**。

## Memory Hook

> [!success]
> - **focal = 单侧起源 + 自动症 + postictal**；**absence = 无 postictal + 过度通气诱发 + 3Hz**
> - **focal impaired awareness = 旧名 complex partial（复杂部分）= 最常起于颞叶（temporal lobe）** → 局灶发作照样有 postictal confusion，看的是"意识网络被没被打断"，不是"放电范围大不大"；颞叶起源者 postictal confusion 尤其突出 + 事后遗忘 + 可有颞叶 aura（déjà vu / 嗅幻觉 / 上腹上涌感）
> - **自动症 ≠ 只有 focal**：失神也可有（轻微、无 postictal）、额叶 hypermotor；清醒模仿者 = TD（长期抗精神病药、意识不受损）/ tic（可主动抑制）。见咂嘴问三件事：意识？时长+postictal？能否抑制？
> - **Todd 麻痹 = 发作后皮层「力竭」（神经元耗竭+抑制+灌注低谷）→ 一过性局灶无力 <48h 自愈**；刚抽完偏瘫先想 Todd vs 卒中（有发作史 + 逐渐恢复 = Todd）；Todd 需局灶皮层受累 → 只见于 focal / 局灶起源继发全面化，"貌似全面"后见 Todd = 局灶起源的定位线索
> - 成人**首次无诱因发作 + 局灶/肿瘤史/持续缺损 → MRI 脑找病灶**（比 CT 敏感）
> - **focal 不是"一定有肿瘤"而是"要主动除外结构灶"**：成人新发一律 MRI；例外 = 儿童 **BECTS（良性 Rolandic、中央颞区棘波、别追肿瘤）** + provoked 代谢性（**高血糖**致局灶发作，纠代谢）
> - 血糖 70 别补糖；低钠致痫多 <120

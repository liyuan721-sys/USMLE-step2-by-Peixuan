# 神经_癫痫发作分型与首次发作（Seizure Classification & First Unprovoked Seizure）

> [!info] 一句话定位
> 三条主线：**①发作分型（局灶 focal vs 全面 generalized）**；**②Todd 麻痹（postictal 一过性缺损）机制 + vs 卒中**；**③成人首次无诱因发作（first unprovoked seizure）的 workup 与影像选择**。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q2280]]（局灶发作误选 JME）+ [[mistakes/uworld-mistakes_2026-06#^Q110558]]（首次发作 + 肿瘤史 → MRI 误选补糖）。

## 一、发作分型（ILAE）

```
按起源
 ├─ Focal（局灶，起于单侧半球）
 │    ├─ 意识保留（aware）
 │    └─ 意识障碍（impaired awareness）→ 自动症 automatisms
 │         可进展为 → focal to bilateral tonic-clonic（继发全面化）
 └─ Generalized（全面，起即累及双侧）
      ├─ Absence（失神）
      ├─ Tonic-clonic（强直阵挛）
      ├─ Myoclonic（肌阵挛）
      └─ Atonic / Tonic
```

## 二、局灶发作 vs 失神 ⭐（最高频鉴别）

| 特征 | **Focal（伴意识障碍）** | **Absence（失神）** |
|---|---|---|
| 起源 | **单侧半球** | **全面（双侧）** |
| 时长 | 数分钟 | **10-20 秒** |
| 自动症 automatisms | **常有**（咀嚼、咂嘴、抠摸）| 偶有轻微 |
| 发作后 postictal | **有**（混乱/疲倦 ± Todd 麻痹）⭐ | **无**（立即恢复）⭐ |
| 过度通气诱发 | **不**诱发 | **诱发** ⭐ |
| EEG | 局灶痫样；**间期可正常** | **3 Hz 棘慢波** |
| 年龄/背景 | 任意；可有结构病灶 | 学龄儿童 |

> [!tip] postictal 与诱发是钥匙
> 有 postictal 混乱 + 自动症 + 单侧起源 = **focal**；无 postictal + 过度通气诱发 + 3Hz = **absence**。

![[{C049E8DC-B236-427B-9345-23B617CB96EC}.png|550]]
*儿童急性偏瘫病因鉴别 (UWorld)：缺血性卒中 (ischemic stroke) / 颅内出血 (intracranial hemorrhage) / **Todd 麻痹 (Todd paralysis，发作后一过性局灶无力)** / 偏瘫型偏头痛 (hemiplegic migraine)。Todd 麻痹 = 发作后局灶无力、影像正常、通常 <36h 自行恢复（发作前肢体抽动/意识丧失 + 发作后意识模糊是钥匙）。*

## 三、Todd 麻痹（Todd paralysis / Todd 瘫痪）⭐

> [!info] 定义
> **局灶性发作（focal seizure）之后**出现的**一过性局灶性神经缺损** —— 最常见是发作累及侧对应肢体的**偏瘫/无力（hemiparesis）**，也可为失语、凝视麻痹、感觉缺失、偏盲。典型持续**数分钟~数小时（通常 <48h，多在 24h 内）后完全恢复**，影像无急性病灶。属**发作后现象（postictal phenomenon）**。

### 3.1 机制（一过性"神经元力竭 + 抑制"状态）

原理尚无定论，主流机制是发作过后受累皮层进入暂时性"力竭"：

```
异常同步过度放电（发作）
        │ 发作停止后
        ▼
① 神经元耗竭 + 主动抑制
   递质储备耗尽 + GABA（γ-氨基丁酸，主要抑制性递质）能回路代偿性过度激活
   → 把该区皮层「按住」，对应功能暂时罢工
② 神经元代谢障碍
   ATP 耗竭 / 乳酸堆积 / Na⁺-K⁺-ATPase 跟不上
   → 膜电位无法复位，处于不可兴奋期
③ 脑血流失配
   受累皮层局部灌注下降（hypoperfusion）+ 神经血管耦联失调
        │ 代谢与抑制恢复正常后
        ▼
     功能完全恢复（<48h）
```

> [!tip] 一句话
> 发作把这块皮层「用到力竭」→ 发作后进入暂时性抑制 + 代谢/灌注低谷 → 对应功能暂时瘫掉 → 代谢和抑制回正就好了。

### 3.2 vs 缺血性卒中（真正的考点）⭐

> [!danger] 刚抽搐完的病人出现偏瘫 = 首要鉴别 Todd 麻痹 vs stroke

| 特征 | **Todd 麻痹** | **缺血性卒中 (ischemic stroke)** |
|---|---|---|
| 前驱 | **明确发作史在先** + postictal 意识模糊 ⭐ | 无发作，突发 |
| 病程 | 一过性、**逐渐自行恢复**（分钟~小时，<48h） | 持续不缓解 |
| 定位 | 缺损侧对应**放电起源皮层** | 对应**血管分布区** |
| 影像 | 无急性梗死 | 弥散受限 / 梗死灶 |
| 处理 | 临床判断即可**观察等待**；不确定仍按急性缺损排查 | 急诊评估溶栓 |

### 3.3 定位与临床价值

- **定位反推**：缺损侧 → 对侧运动皮层为致痫灶（如**右侧**肢体 Todd 无力 → **左侧**运动皮层放电）。
- Todd 麻痹**不等于结构病灶**，但**反复固定同一侧**提示该侧可能有致痫灶 → 值得 MRI 评估（见 §五 red flags）。
- 它是**发作后现象，不是发作本身**，也不提示卒中。

## 四、儿童常见癫痫综合征（鉴别陷阱）

| 综合征 | 钩子 |
|---|---|
| **JME（juvenile myoclonic epilepsy，青少年肌阵挛癫痫）** | 青少年、**晨起肌阵挛**（甩东西）、可伴 GTC；瓦普罗酸/左乙拉西坦 |
| **Lennox-Gastaut** | **≤5 岁**起、智力障碍、多种发作型、**慢棘慢波（<2.5Hz）** |
| **Childhood absence** | 学龄、凝视、过度通气诱发、3Hz；乙琥胺 ethosuximide |
| **Tic disorder（抽动）** | 短促刻板动作、**意识不受损**、可短暂抑制 |

## 五、成人首次无诱因发作 workup ⭐

> [!info] 定义
> **Unprovoked**（无诱因）= 无急性可逆诱因（如低血糖、低钠、酒精戒断、急性脑损伤）。

### 4.1 基础评估（先排急性诱因）

- 血糖（**看具体数值**：低糖致痫常 <45 且有交感症状；血糖 70 = 正常，不补糖）
- 电解质（**低钠致痫多 <120**）、钙、镁
- 毒物/酒精、肝肾功能、妊娠
- ECG（鉴别晕厥）

### 4.2 神经影像 —— MRI > CT ⭐

- 成人首次发作应做脑影像；**MRI（magnetic resonance imaging，磁共振）比 CT 敏感**（查肿瘤/海绵状血管畸形/皮质发育不良/陈旧卒中/海马硬化）。
- 急诊若需快速排出血/急性病变可先 CT，但**结构性病因定性靠 MRI**。

### 4.3 结构性病灶 red flags（提示需 MRI 找病灶）

- **局灶起源**（focal onset）或继发全面化
- **持续局灶神经缺损 / Todd 麻痹**
- **恶性肿瘤史** → 脑转移（brain metastasis）：**RCC（renal cell carcinoma，肾细胞癌）、肺癌、黑色素瘤、乳腺癌**是脑转移大户 ⭐
- 年龄较大、新发头痛、免疫抑制

### 4.4 是否启动抗癫痫药（AED）

- 单次无诱因 + 影像/EEG 正常 → 复发风险较低 → 通常**观察**。
- **有结构病灶 / EEG 痫样放电 / 异常神经查体** → 复发风险高 → **起始 AED**。

## 六、易混陷阱（普适）

- 见癫痫**反射补葡萄糖** → 错：先看血糖数值（70 正常）。
- **持续 24h EEG** = 用于疑**非惊厥性 SE（nonconvulsive status epilepticus）**；首次发作后能唤醒/在恢复者不是 SE（见 [[完整笔记/专题笔记/神经/神经_癫痫持续状态SE]]）。
- **间期 EEG 正常不排除局灶癫痫**。

## Memory Hook

> [!success]
> - **focal = 单侧起源 + 自动症 + postictal**；**absence = 无 postictal + 过度通气诱发 + 3Hz**
> - **Todd 麻痹 = 发作后皮层「力竭」（神经元耗竭+抑制+灌注低谷）→ 一过性局灶无力 <48h 自愈**；刚抽完偏瘫先想 Todd vs 卒中（有发作史 + 逐渐恢复 = Todd）
> - 成人**首次无诱因发作 + 局灶/肿瘤史/持续缺损 → MRI 脑找病灶**（比 CT 敏感）
> - 血糖 70 别补糖；低钠致痫多 <120

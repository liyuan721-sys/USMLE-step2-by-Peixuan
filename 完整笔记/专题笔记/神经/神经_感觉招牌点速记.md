# 神经感觉招牌点速记 (Sensory Autonomous Zones — One-Spot Recall)

> [!tip] 为什么背不住 + 换记法
> 感觉记不住 = 在背"一整片面积"（模糊、左右对称、上下肢互相干扰）。但 USMLE **只考一个点**（"针刺觉在 X 处消失 → 哪条神经"）。所以**只记每条神经的「招牌一点」(autonomous zone 自主支配区)**——纯区、不重叠、题目专挑它考。全身十个点，比背十片轻太多。

---

## §一 上肢：手上只记「三个点」+ 肩/前臂两点 ⭐

```
虎口背侧（拇指与食指之间的手背）      = 桡 Radial
食指指腹（掌侧指尖）                  = 正中 Median
小指（整根，掌背都算）               = 尺 Ulnar
肩外侧"军徽 regimental badge"        = 腋 Axillary
前臂外侧                             = 肌皮 Musculocutaneous
```

> [!danger] 上肢三点锁死
> - **桡神经 = 只给"手背 + 虎口"，绝不给指腹** → 记"**桡管背不管尖**"；wrist drop 的人**指尖照样有感觉**（干扰项爱在这挖坑）。
> - **正中 = 桡侧指腹**（拿东西那面）= 食指/中指掌侧尖。
> - **尺 = 小指**（一根小指定生死）。

---

## §二 下肢：脚上记「五个点」+ 大腿三片 ⭐

```
足底                        = 胫 Tibial（踩地那面）
足背（大部）                = 浅腓 superficial peroneal
第一趾蹼背（拇趾与2趾之间） = 深腓 deep peroneal
足外侧缘                    = 腓肠 sural
足内侧缘 + 小腿内侧         = 隐神经 saphenous（= 股 Femoral 的下延）

大腿前外侧                  = 股外侧皮神经 LFCN (lateral femoral cutaneous)（meralgia paresthetica 感觉异常性股痛）
大腿内侧                    = 闭孔 Obturator
大腿前面                    = 股 Femoral
```

> [!danger] 下肢易错点
> - **膝以下内侧 = 隐神经 (saphenous)**——它是**股神经**的分支，是"股神经能到膝以下"的唯一途径。所以**坐骨神经损伤时小腿内侧感觉保留**（那块归股/隐，不归坐骨）。
> - **第一趾蹼背 = 深腓**（唯一"背屈肌神经"给的一小片感觉），和上肢**虎口背**镜像。

---

## §三 秘诀：上下肢「镜像」，一起记省一半 ⭐⭐

> [!success] 伸肌神经 → 管"背面 + 蹼"，还都会"drop"
> | | 上肢 | 下肢 |
> |---|---|---|
> | **伸肌/背面神经** | **桡 Radial** → 手背 + 虎口背；**wrist drop** | **腓总 peroneal** → 足背 + 第一趾蹼背；**foot drop** |
> | **屈肌/接触面神经** | **正中 Median** → 手掌桡侧（抓握面）| **胫 Tibial** → 足底（踩地面）|
>
> 一句话：**"背面 + 蹼 + drop" = 桡 / 腓总（一对）；"抓握面 / 踩地面" = 正中 / 胫（一对）。**
> 虎口背 ↔ 第一趾蹼背 完全镜像（都是那条伸肌神经管的一小片背侧蹼）。

## §五 自测复习题

> [!question]- Q1. 患者手背拇指与食指之间一小片针刺觉消失，但所有指尖感觉正常。哪条神经？为什么指尖正常反而支持它？
> **桡神经 (radial)**。桡神经"管背不管尖"——只给手背 + 虎口背侧、**不给指腹**，所以指尖有感觉正好符合桡神经、排除正中。

> [!question]- Q2. 坐骨神经完全损伤，为什么**小腿内侧**感觉却保留？
> 小腿内侧 = **隐神经 (saphenous)** 支配，它是**股神经 (femoral)** 的分支，不属坐骨神经。坐骨管"膝以下除内侧"。

> [!question]- Q3. 足下垂患者，第一趾蹼背（拇趾与2趾之间）针刺觉消失。定位到腓总的哪一支？它和上肢哪条神经镜像？
> **深腓神经 (deep peroneal)**（第一趾蹼背是它的招牌点）。镜像 = 上肢**桡神经**的虎口背侧——两条都是"伸肌/背面神经"给的背侧小蹼。

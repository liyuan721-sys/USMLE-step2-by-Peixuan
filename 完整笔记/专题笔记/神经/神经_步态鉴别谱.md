# 神经_步态鉴别谱（Gait Disorders）

> [!info] 一句话定位
> 步态题靠 **步态描述 + 伴随体征 + Romberg** 定位。
> **Romberg 是分水岭**：阳性（闭眼摔）= 本体觉/后索（感觉性共济）；阴性 = 运动/小脑/锥体外系。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q4427]]（足下垂 steppage 误选小脑）。

## 零、步态背后的逻辑：控制回路 + 代偿 ⭐

> [!tip] 核心心法
> 步态 = 身体对**某个坏掉环节的"代偿动作"**，不是缺陷本身。看到步态题别背"像哪个"，问 **"身体在代偿什么？"**——画圈是腿锁死了、高抬腿是怕绊到、跺地是想找回感觉、小碎步是放大器没油。补救动作一想，诊断就出来。

走一步，大脑依次完成 4 件事，坏哪一环 → 出哪种步态：

```
①启动程序   →   ②输出力量   →   ③协调时序   →   ④感觉反馈（腿在哪?）
 额叶/基底节      UMN/LMN/肌肉      小脑            后索/前庭/视觉
```

| 步态 | 坏的环节 | 补救动作（= 看到的样子） | 一句话逻辑 |
|---|---|---|---|
| 痉挛 spastic | UMN（upper motor neuron，上运动神经元）失抑制 → 腿锁死 | 单侧画圈 / 双侧剪刀 | 抬不动 → 绕或夹 |
| 足下垂 steppage | LMN（lower motor neuron，下运动神经元）→ 勾脚（背屈）无力 | 高抬腿 + 被动拍地 | 脚耷拉 → 怕绊 |
| 小脑共济 cerebellar | 协调器坏 | 宽基 + 醉酒摇晃 | = 喝醉的样子 |
| 感觉性共济 sensory | 本体觉反馈断 | 盯脚 + 主动跺地 + 宽基 | 不知脚在哪 → 找感觉 |
| 帕金森 parkinsonian | 多巴胺"放大器"没油 | 小碎步 + 慌张追重心 | 步子放不大、发不出 |
| 额叶/NPH（normal pressure hydrocephalus，正常压力脑积水）frontal | 走路"程序"断 | 磁性脚黏地 + 起步难 | 腿没坏，是念头断 |
| 前庭 vestibular | 单侧平衡仪偏 | 向患侧倾倒 | 一侧误报"在倾斜" |

> [!info]- 逐个推：机制 → 代偿（展开看"为什么长这样"）
> - **痉挛 spastic**：UMN 正常工作是**抑制**下方张力；断了 → 失抑制 → 抗重力肌（伸肌/内收肌）张力爆表、腿僵直弯不动。腿抬不起 → 单侧**画半圈 circumduction** 绕过去；双侧内收肌紧 → **剪刀 scissoring** 交叉。
> - **足下垂 steppage**：背屈肌无力 → 脚自然下垂，且**落地控制不住缓放** → 只能高抬髋膝让脚尖越过地面、再**被动"啪"拍下去（foot slap）**。
> - **小脑共济**：小脑 = 实时协调/微调；坏了 → 站不稳 → 本能**张大双脚（宽基）** + 左右摇晃像醉汉。**酒精正是毒小脑**，所以"醉酒步态"就是它的原型（phenytoin 中毒、髓母细胞瘤同理）。
> - **感觉性共济**：后索/本体觉断 → 大脑收不到"腿在哪" → ①**死盯脚看**（视觉代偿）②**主动跺地**靠冲击挤出感觉 ③宽基防摔；闭眼/黑暗拿走视觉拐杖 → 立刻摔（**Romberg+**）。
> - **帕金森**：基底节多巴胺 = 运动程序的"音量放大器"；不足 → 动作幅度太小 + 启动难 → 小碎步、起步冻结；身体前倾重心前移而碎步追不上 → **慌张步态 festination**；自动摆臂被抑制 → **摆臂减少**。
> - **额叶/NPH**：不是力量/协调/感觉问题，是额叶**步态程序**断 → 躺床上腿动得好，一站脚却**像吸在地上（magnetic）**、起步转身卡住。NPH 三联（步态+认知+尿失禁）都是脑室周围/额叶受压。
> - **前庭**：单侧前庭一直误报"你在倾斜" → **向患侧倾倒** + 眩晕眼震。

## 一、六大步态对照表

| 步态 | 描述 | 伴随体征 | Romberg | 病因 |
|---|---|---|---|---|
| 额叶（失用）frontal/apraxic | **磁性步态**（脚像粘地）、起步/转身迟疑、宽基小步 | 痴呆、尿失禁 | — | **NPH（normal pressure hydrocephalus，正常压力脑积水）**、额叶变性 |
| 帕金森 parkinsonian | 小步、慌张（festinating）、窄基、前倾 | 运动迟缓、静止性震颤、↓摆臂、姿势不稳 | − | PD、DLB（Lewy 体痴呆）|
| 前庭 vestibular | 不稳、向**一侧**倾倒 | 眩晕、恶心、眼震 | − | 急性迷路炎、Ménière 病 |
| 小脑共济 cerebellar ataxia | **宽基、蹒跚、醉酒样**、左右摇摆 | 辨距不良 dysmetria、眼震、滴定震颤、轮替障碍 | − | 卒中、酒精、**Wernicke 脑病（Wernicke encephalopathy，韦尼克脑病）**、**phenytoin（苯妥英）中毒**、髓母细胞瘤 medulloblastoma（儿童）、遗传性 |
| **痉挛性 spastic ⭐** | **僵硬、足尖拖地、窄基**；偏瘫=单侧**划圈 circumduction**、截瘫=双下肢**剪刀步 scissoring** | **UMN 征**：肌张力↑（折刀样 clasp-knife）、反射↑、Babinski/Hoffmann+、无力 | − | 卒中（偏瘫划圈）、脊髓压迫/横断（截瘫剪刀）、**MS**、**ALS**（+LMN）、CP |
| **足下垂 / 跨阈 steppage ⭐** | **高抬髋膝 + 落地拍地 foot slap（常单侧）** | 远端**运动**无力（背屈↓）、可伴远端感觉↓ | **−** | **L5 神经根病 / common fibular（peroneal）腓总神经病变**；**急性双侧弛缓拖足 → GBS** |
| 感觉性共济 sensory ataxia | 高抬步、**宽基**、跺脚、**黑暗中加重** | ↓本体觉/振动觉（**双侧**）| **＋** | **B12 缺乏（后索）**、压迫性脊髓病、tabes dorsalis |

## 二、关键分水岭：Romberg & 单/双侧 ⭐

```
步态异常
  ├─ Romberg 阳性 + 宽基 + 双侧 + 黑暗加重 ──► 感觉性共济(后索/本体觉, B12)
  ├─ Romberg 阴性 + 单侧拍地 + 远端无力 ────► 足下垂 steppage (L5/腓总) ← Q4427
  ├─ Romberg 阴性 + 宽基醉酒 + 辨距不良 ───► 小脑共济
  ├─ Romberg 阴性 + 僵硬剪刀/划圈 + 反射↑Babinski+ ► 痉挛性(UMN: 卒中/脊髓/MS/ALS)
  └─ 磁性/慌张/前庭 ─────────────────────► 额叶/帕金森/前庭
```

> [!tip] 痉挛性步态（spastic gait）= 第二大"Romberg 阴性"运动型，别漏
> 步态僵硬 + **反射↑ + Babinski(+) + 肌张力↑** = UMN 通路损害（皮质脊髓束）。按"哪段 UMN + 单/双侧"再分：
> - **单侧划圈 circumduction（偏瘫步态）** → 对侧大脑/内囊 **卒中（stroke, 中风）**（伴上肢屈曲内收）。
> - **双下肢剪刀步 scissoring（截瘫步态）** → **脊髓压迫/横断**（平面以下 UMN + 二便障碍）、**脑瘫 CP（cerebral palsy）**（儿童双侧痉挛）。
> - **进行性痉挛 + 体温升高加重（Uhthoff 现象）/ 屈颈电击（Lhermitte 征）** → **MS（multiple sclerosis，多发性硬化）**脊髓脱髓鞘斑块。
> - **痉挛 + 同时肌萎缩/肌束颤 fasciculation + 但感觉完全正常** → **ALS（amyotrophic lateral sclerosis，肌萎缩侧索硬化）**（UMN+LMN 混合、**无感觉缺失**是与 CSM / B12 的分水岭）。

> [!warning] 痉挛性 vs 痉挛-共济混合（CSM）vs 纯后索（B12）—— 看"反射 + 感觉"
>
> | 病 | 反射 | Babinski | 感觉/Romberg | 关键鉴别 |
> |---|---|---|---|---|
> | 纯痉挛（卒中/脊髓压迫/MS/ALS）| ↑ | + | 多正常（ALS **一定**无感觉缺失）| UMN 征为主 |
> | CSM 痉挛-共济混合 | ↑ | + | 后索受累、Romberg± | UMN + **双手笨拙** + 后索 |
> | B12 / 后索（感觉性共济）| **↓**（合并周围神经）| ±（SCD 亚急性联合变性 subacute combined degeneration 可+）| Romberg**＋**、振动觉↓双侧 | 反射↓ + 黑暗加重 |

> [!warning] Q4427 陷阱
> "反复跌倒 + 步态异常"易反射小脑；但 **Romberg 阴性 + 单侧足拍地 + 屈髋抬腿** = 下运动单元**足下垂**（背屈无力），不是小脑（小脑应宽基醉酒 + 辨距不良 + **双侧**眼震）。感觉性共济虽也高抬步，但 **Romberg 阳性 + 双侧 + 黑暗加重**。

### 2.1 易混专题：steppage 足下垂 vs 感觉性共济（都"高抬腿重落地"）⭐

> [!warning] 外观都"抬得高、落得重"，但抬的原因和落的性质正好相反
> **落地是关键**：steppage = **被动拍地 foot slap**（背屈肌连缓放都控制不住，脚"啪"自己掉下去）；感觉性共济 = **主动跺地 stomp**（故意用力砸，靠冲击挤出感觉反馈）。一个"控制不住掉下去"，一个"故意砸下去找感觉"。

| 鉴别点 | steppage 足下垂 | 感觉性共济 sensory ataxia |
|---|---|---|
| 为什么抬高 | 脚尖**下垂**，不抬高就绊到自己 | **不知脚在哪**，甩腿找地、怕踩空 |
| 落地性质 | **被动拍地**（foot slap，松软） | **主动跺地**（stomp，用力） |
| 单/双侧 | 常**单侧**（L5 / common fibular 腓总神经） | **双侧**对称 |
| Romberg | **阴性**（闭眼不更糟） | **阳性** ⭐（闭眼/黑暗立刻垮） |
| 盯脚看 | 不用 | **死盯脚**（视觉代偿） |
| 力量 | 远端**运动无力**（勾不起脚、heel-walk 跟走做不出） | **正常** |
| 振动/位置觉 | 多正常 | **双侧↓** |
| 黑暗中 | 无差别 | **明显加重** |

> [!tip] 最快分水岭 = Romberg + 单双侧
> 单侧 + 闭眼不更糟 = **足下垂（运动问题）**；双侧 + 闭眼就摔 = **感觉性共济（反馈问题）**。

## 三、Stem 钩子

- "屈髋屈膝抬腿 + 落地啪一声(foot slap)" → steppage → 足下垂（L5/腓总）。
- "黑暗中/闭眼站不稳(Romberg+) + 振动觉↓" → 感觉性共济（B12 / 后索）。
- "脚像吸在地上、起步难 + 尿失禁 + 痴呆" → NPH 磁性步态。
- "老年 + 进行性步态不稳 + **双手笨拙** + 下肢反射↑/Babinski+/Hoffmann+" → **颈脊髓病 CSM（颈椎病脊髓型, Cervical Spondylotic Myelopathy）** → MRI 颈椎。
- "单侧上肢屈曲内收 + 下肢**划圈拖步** + 既往卒中" → 偏瘫痉挛步态（对侧 UMN）。
- "进行性僵硬剪刀步 + 反射↑Babinski+ + **肌萎缩/肌束颤** + **感觉完全正常**" → **ALS（amyotrophic lateral sclerosis，肌萎缩侧索硬化）**（无感觉缺失是钥匙）。
- "**数日内上行性、对称、弛缓**无力 + **反射消失 areflexia** + 拖足 + 前驱腹泻/上感" → **GBS（Guillain-Barré syndrome，吉兰-巴雷综合征）**（急性弛缓 vs 慢性 steppage）。
- "**共济失调步态 + 眼肌麻痹 ophthalmoplegia + 反射消失** 三联（抗 GQ1b 抗体）" → **Miller-Fisher 综合征**（GBS 变体；与纯小脑共济鉴别——小脑无眼肌麻痹+无反射消失）。
- "高血压/房颤老人 **突发**持续眩晕 + 宽基共济步态 + 不能独走 + CNS（central nervous system，中枢神经系统）体征 / HINTS 提示中枢" → **小脑卒中（cerebellar stroke）**（急性小脑共济亚型，别当外周眩晕）。
- "酗酒/营养不良/妊娠剧吐 + **宽基共济步态 + 眼肌麻痹 ophthalmoplegia/眼震 + 意识模糊** 三联" → **Wernicke 脑病（Wernicke encephalopathy，韦尼克脑病）** → **立即 IV 硫胺素 thiamine（维生素 B1），先于葡萄糖**。
- "儿童 + **晨起呕吐 + 头痛 + 宽基共济步态**（小脑蚓部 vermis）+ 脑积水" → **髓母细胞瘤（medulloblastoma）**（后颅窝肿瘤）→ 头颅 MRI。
- "服 **phenytoin（苯妥英）**（或新加 **TMP-SMX 复方磺胺甲噁唑** 等致血药浓度升高）+ 眼震 + 指鼻不准 + 宽基步态" → **phenytoin 中毒**（小脑功能障碍）→ 查血药浓度。
- "皮质下白质病变（Binswanger）+ **步态失用 + 尿失禁**" → **血管性痴呆（vascular dementia, VaD）**步态——与 NPH 磁性步态/正常压力脑积水重叠，靠血管危险因素 + MRI 白质病变鉴别。

> [!warning] 压迫性脊髓病 = 痉挛-共济"混合"步态（别只记成感觉性共济）
> 上表把"压迫性脊髓病"列在感觉性共济一行，但 CSM 实为**两条长束同时受压**：皮质脊髓束（→下肢 UMN 痉挛步态）+ 后索（→感觉性共济 + Romberg±）= **spastic-ataxic gait（痉挛-共济混合）**。所以 stem 既可见 Romberg±，又有**下肢反射亢进 + Babinski/Hoffmann+**（与纯 B12 后索病变"反射↓"不同）。机制详见 [[完整笔记/专题笔记/神经/神经_脊髓病变]] I-bis.2。

> [!info]- CSM 速览：颈椎病脊髓型（CSM, Cervical Spondylotic Myelopathy，颈脊髓病）是什么
> 颈椎**退变**（骨赘 osteophyte / 椎间盘 / 后纵韧带肥厚）从外压迫**颈段脊髓本身**（≠ 压神经根 radiculopathy）→ 成人**脊髓病 + 步态障碍最常见病因**。
> - **病灶在颈、症状在腿**：颈髓里走的是去腿的长束（皮质脊髓束 + 后索），压在脖子 = 掐了"路过去腿的客"。
> - **步态 = 痉挛-共济混合**：锥体束 → 下肢 UMN 痉挛（反射↑ / Babinski+）；后索 → 感觉性共济（Romberg±）。
> - **招牌**：老年 + 进行性步态不稳 + **双手笨拙**（系扣难/掉东西）+ Hoffmann+ → **颈椎 MRI** → 显著/进展则手术减压。
> - **三鉴别（靠反射 + 感觉）**：vs **B12 缺乏**（反射**↓** + 巨幼贫）；vs **ALS（amyotrophic lateral sclerosis，肌萎缩侧索硬化）**（**无任何感觉缺失**）；vs **神经根病 radiculopathy**（单根/皮节、无长束征、无步态）。
> - 完整机制 + 三鉴别表见 [[完整笔记/专题笔记/神经/神经_脊髓病变]] I-bis.2。

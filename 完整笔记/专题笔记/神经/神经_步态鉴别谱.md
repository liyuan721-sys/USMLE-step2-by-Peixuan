# 神经_步态鉴别谱（Gait Disorders）

> [!info] 一句话定位
> 步态题靠 **步态描述 + 伴随体征 + Romberg** 定位。
> **Romberg 是分水岭**：阳性（闭眼摔）= 本体觉/后索（感觉性共济）；阴性 = 运动/小脑/锥体外系。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q4427]]（足下垂 steppage 误选小脑）。

## 一、六大步态对照表

| 步态 | 描述 | 伴随体征 | Romberg | 病因 |
|---|---|---|---|---|
| 额叶（失用）frontal/apraxic | **磁性步态**（脚像粘地）、起步/转身迟疑、宽基小步 | 痴呆、尿失禁 | — | **NPH（normal pressure hydrocephalus，正常压力脑积水）**、额叶变性 |
| 帕金森 parkinsonian | 小步、慌张（festinating）、窄基、前倾 | 运动迟缓、静止性震颤、↓摆臂、姿势不稳 | − | PD、DLB（Lewy 体痴呆）|
| 前庭 vestibular | 不稳、向**一侧**倾倒 | 眩晕、恶心、眼震 | − | 急性迷路炎、Ménière 病 |
| 小脑共济 cerebellar ataxia | **宽基、蹒跚、醉酒样**、左右摇摆 | 辨距不良 dysmetria、眼震、滴定震颤、轮替障碍 | − | 卒中、酒精、遗传性 |
| **足下垂 / 跨阈 steppage ⭐** | **高抬髋膝 + 落地拍地 foot slap（常单侧）** | 远端**运动**无力（背屈↓）、可伴远端感觉↓ | **−** | **L5 神经根病 / common fibular（peroneal）腓总神经病变** |
| 感觉性共济 sensory ataxia | 高抬步、**宽基**、跺脚、**黑暗中加重** | ↓本体觉/振动觉（**双侧**）| **＋** | **B12 缺乏（后索）**、压迫性脊髓病、tabes dorsalis |

## 二、关键分水岭：Romberg & 单/双侧 ⭐

```
步态异常
  ├─ Romberg 阳性 + 宽基 + 双侧 + 黑暗加重 ──► 感觉性共济(后索/本体觉, B12)
  ├─ Romberg 阴性 + 单侧拍地 + 远端无力 ────► 足下垂 steppage (L5/腓总) ← Q4427
  ├─ Romberg 阴性 + 宽基醉酒 + 辨距不良 ───► 小脑共济
  └─ 磁性/慌张/前庭 ─────────────────────► 额叶/帕金森/前庭
```

> [!warning] Q4427 陷阱
> "反复跌倒 + 步态异常"易反射小脑；但 **Romberg 阴性 + 单侧足拍地 + 屈髋抬腿** = 下运动单元**足下垂**（背屈无力），不是小脑（小脑应宽基醉酒 + 辨距不良 + **双侧**眼震）。感觉性共济虽也高抬步，但 **Romberg 阳性 + 双侧 + 黑暗加重**。

## 三、Stem 钩子

- "屈髋屈膝抬腿 + 落地啪一声(foot slap)" → steppage → 足下垂（L5/腓总）。
- "黑暗中/闭眼站不稳(Romberg+) + 振动觉↓" → 感觉性共济（B12 / 后索）。
- "脚像吸在地上、起步难 + 尿失禁 + 痴呆" → NPH 磁性步态。
- "老年 + 进行性步态不稳 + **双手笨拙** + 下肢反射↑/Babinski+/Hoffmann+" → **颈脊髓病 CSM（颈椎病脊髓型, Cervical Spondylotic Myelopathy）** → MRI 颈椎。

> [!warning] 压迫性脊髓病 = 痉挛-共济"混合"步态（别只记成感觉性共济）
> 上表把"压迫性脊髓病"列在感觉性共济一行，但 CSM 实为**两条长束同时受压**：皮质脊髓束（→下肢 UMN 痉挛步态）+ 后索（→感觉性共济 + Romberg±）= **spastic-ataxic gait（痉挛-共济混合）**。所以 stem 既可见 Romberg±，又有**下肢反射亢进 + Babinski/Hoffmann+**（与纯 B12 后索病变"反射↓"不同）。机制详见 [[完整笔记/专题笔记/神经/神经_脊髓病变]] I-bis.2。

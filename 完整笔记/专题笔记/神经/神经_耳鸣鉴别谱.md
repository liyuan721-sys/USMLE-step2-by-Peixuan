# 神经_耳鸣鉴别谱（Tinnitus：搏动 vs 非搏动 + 耳毒药谱）

> [!info] 为什么单独建这张表（与眩晕谱分工）
> [[完整笔记/专题笔记/神经/神经_眩晕鉴别谱]] 已把 **Ménière 梅尼埃病 / 听神经瘤 vestibular schwannoma / Weber-Rinne / 感音聋频率定位** 讲透了——那些病以**眩晕**为核心。
> 但耳鸣 (tinnitus) 有一类**眩晕谱完全没覆盖**的病因：**搏动性耳鸣 (pulsatile tinnitus) 的血管谱**（IIH / glomus 球瘤 / 颈动脉狭窄）和**系统化的耳毒性药物谱**。
> 这张表只补这两块 + 给耳鸣的**第一分流框架**；具体"伴眩晕的病"一律链回眩晕谱，不重复造孤岛。

---

## 一、耳鸣第一分流：搏动 vs 非搏动 ⭐⭐

> [!tip] 看到 tinnitus 第一刀
> 先问一句：**"声音跟心跳同步吗 (synchronous with heartbeat)？"** 这一刀决定查什么、怕什么。

```
耳鸣 Tinnitus
├── Non-pulsatile（非搏动性，嗡嗡/嘶嘶/铃声，绝大多数）
│     → 听觉通路 / 感音神经问题 → 多为良性，查听力图
│     （Ménière / presbycusis / NIHL / 听神经瘤 / 耳毒药 / 耵聍）
│
└── Pulsatile（搏动性，与心跳同步 = whooshing 冲刷声）⚠️
      → 血管性 / 颅压问题 → 需影像（MRA/MRV），可能危险
      （IIH / glomus 球瘤 / 颈动脉狭窄 / dural AVF / 高输出状态）
```

> [!danger] 三个一眼红旗（任一出现都升级处理）
> - **单侧 (unilateral) 非搏动性耳鸣 + 单侧 SNHL (sensorineural hearing loss, 感音神经性聋)** → **MRI 排听神经瘤**（详见眩晕谱）
> - **搏动性 (pulsatile) 耳鸣** → 影像查血管 + 想 IIH / glomus
> - **耳镜见鼓膜后红色搏动性肿块** → glomus 球瘤，**禁活检**（会大出血）

## 二、非搏动性耳鸣总表（多链回眩晕谱）

> [!info] 这一节是"地图"不是"详解"
> 非搏动性耳鸣的病大多在眩晕谱里讲透了，这里只给一句话钥匙 + 指路；展开看 [[完整笔记/专题笔记/神经/神经_眩晕鉴别谱]]。

| 病因 | 一句话钥匙 | 详解去处 |
|---|---|---|
| **Presbycusis 老年性聋** | 老年双侧对称、高频先掉、"听得见听不懂" | 眩晕谱 §五高频聋 |
| **NIHL 噪声性聋 (noise-induced)** | 工人/乐手/枪声，**4000 Hz notch 切迹** | 眩晕谱 §五 |
| **Ménière 梅尼埃病** | 数小时眩晕 + 耳鸣 + 耳胀 + **低频**波动性聋；限盐 | 眩晕谱 §三② / §五低频聋 |
| **听神经瘤 Vestibular schwannoma** | **单侧**进行性 SNHL + 耳鸣 → MRI；双侧 = NF2 (Neurofibromatosis type 2, 神经纤维瘤病 2 型) | 眩晕谱 §三⑦ |
| **耵聍栓塞 Cerumen impaction** | 传导聋 + 耳闷，**清理后可逆**（最便宜别忘） | 本表 §六 |
| **耳硬化 Otosclerosis** | 青年女性家族史 + 传导聋 + 吵处反听得清 | 眩晕谱 §四 |
| **耳毒性药物 Ototoxic drugs** | 近期用药 + 双侧对称高频 SNHL ± 耳鸣 | 本表 §四 ⭐ |
| **TMJ 颞下颌关节紊乱 (temporomandibular joint dysfunction)** | 咀嚼/张口加重 + 关节弹响 | — |
| 抑郁/焦虑/失眠 | 无器质病变，功能性放大感知 | — |

## 三、搏动性耳鸣血管谱（眩晕谱缺口，重点）⭐⭐

> [!warning] 搏动性 = 听到自己的血流
> 与心跳同步的 "whooshing 冲刷声"。机制无非两类：**①血流异常增快/湍流**（动脉、瘘、高输出）或 **②对血流的感知增强**（颅压高、肿瘤贴着血管）。客观性 (objective) = 医生**听诊也能闻及杂音 bruit**。

| 病因 | 中英双语 | 招牌 stem | 关键 |
|---|---|---|---|
| **特发性颅内高压 IIH** | Idiopathic Intracranial Hypertension | **育龄肥胖女性** + 头痛 + 搏动性耳鸣 + 一过性视物模糊 + **视乳头水肿 papilledema** | LP (lumbar puncture, 腰椎穿刺) 测压↑、CSF (cerebrospinal fluid, 脑脊液) 成分正常；查 MRI/MRV (magnetic resonance venography, 磁共振静脉成像) 排静脉窦血栓 |
| **球瘤 Glomus tumor** | 副神经节瘤 paraganglioma (glomus tympanicum/jugulare 鼓室/颈静脉球) | 单侧搏动性耳鸣 + 传导聋；**耳镜见鼓膜后红色搏动肿块** | ⚠️ **禁活检**（大出血）；影像诊断 |
| **颈动脉粥样硬化/狭窄** | Carotid atherosclerosis/stenosis | 老年 + 血管危险因素 + 颈部 **bruit 杂音** | 颈动脉双功超声；可伴 TIA (transient ischemic attack, 短暂性脑缺血发作) |
| **硬脑膜动静脉瘘** | Dural AV fistula (dAVF, 硬脑膜动静脉瘘) | 单侧搏动性耳鸣，可有外伤/血栓史 | MRA (magnetic resonance angiography, 磁共振血管成像) / 脑血管造影 |
| **颈静脉球高位/憩室** | Jugular bulb anomaly | 鼓膜后**蓝色**肿块（vs glomus 红色） | 影像 |

> [!tip] 高输出 / 全身状态 → 双侧搏动性耳鸣（功能性、纠正即好）
> 不是局部血管病，而是全身血流加速 → 双侧、对称、随原发病好转消失：
> - **贫血 (anemia)**：心输出代偿↑ → 高动力循环
> - **甲亢 (hyperthyroidism, 甲状腺功能亢进)**：心率/输出↑
> - **妊娠 (pregnancy)**：血容量 + 心输出生理性↑
> - **重度高血压 (severe HTN)**
> 钥匙：**双侧 + 有全身病线索 → 先纠原发病**，不急着做脑血管造影。

> [!example] IIH 经典 stem 模式
> 28 岁肥胖女性，慢性头痛 + **搏动性耳鸣** + 阵发性视物模糊（弯腰时加重），眼底见**双侧视乳头水肿**，神经查体余正常，头 MRI 无占位。
> → **特发性颅内高压 IIH**。下一步：MRV 排静脉窦血栓 → LP 测开放压（↑）兼治疗性放液。治疗：减重 + **乙酰唑胺 acetazolamide**；视力威胁 → 视神经鞘开窗 / 分流。
> 诱发关联：**四环素 tetracycline / 维 A 酸 isotretinoin / 大量维生素 A / 口服避孕药**。

## 四、耳毒性药物谱（系统化）⭐⭐

> [!warning] 耳毒性药物表现高度一致
> 招牌 = **双侧对称高频 SNHL (sensorineural hearing loss, 感音神经性聋) ± 耳鸣 ± 平衡障碍**（耳蜗底部 base 管高频、最脆弱，先受损——原理见 [[完整笔记/专题笔记/神经/神经_眩晕鉴别谱]] §五）。stem 钩子 = **近期新用药 / 加量 / 肾功能差**。

| 药物 | 中英双语 | 可逆性 | stem 钩子 |
|---|---|---|---|
| **氨基糖苷类** | Aminoglycosides（庆大霉素 gentamicin / 妥布霉素 tobramycin / 阿米卡星 amikacin） | **不可逆 (irreversible)** | 重症感染长疗程、肾功能差、谷浓度高；耳蜗 + 前庭双毒 |
| **顺铂** | Cisplatin（铂类化疗 platinum chemo） | **不可逆** | 肿瘤化疗后高频聋 + 耳鸣 |
| **袢利尿剂** | Loop diuretics（呋塞米 furosemide / 布美他尼 bumetanide） | **多可逆 (reversible)** | **肾衰 + 大剂量 + 合用其他耳毒药**时风险倍增 |
| **大剂量水杨酸** | High-dose salicylate（阿司匹林 aspirin） | **可逆** | 超治疗量（~6-8 g/d）才致耳鸣/聋；**水杨酸中毒经典 = 耳鸣** |
| 奎宁 / 奎尼丁 | Quinine / quinidine | 可逆 | 抗疟 / 抗心律失常 |
| 万古霉素 / 红霉素 | Vancomycin / erythromycin | 多可逆 | 高浓度 |

> [!tip] 三个高频考点对照
> - **氨基糖苷 vs 袢利尿剂**：两者都耳毒，**合用风险叠加**；氨基糖苷**不可逆**、袢利尿剂**多可逆**。题目给"反复心衰大剂量呋塞米"→ 想呋塞米耳毒性（[[mistakes/uworld-mistakes_2026-06#^Q2838]]），别误选抗生素。
> - **阿司匹林耳鸣 = 可逆信号**：水杨酸中毒时耳鸣是早期标志，停药/降量可逆；与不可逆的氨基糖苷/顺铂区分。详见 [[完整笔记/专题笔记/肾脏/肾脏_水杨酸中毒急慢性]]。
> - 处理通用：**听力图 audiogram + 尽量停药 / 换药**，肾功能差者监测血药浓度。

## 五、单侧非搏动性耳鸣 = 听神经瘤红旗（不可漏）

> [!danger] 一条铁律
> **单侧 (unilateral) 耳鸣 + 单侧进行性 SNHL** → 必须 **MRI 内听道/桥小脑角** 排 **听神经瘤 (vestibular schwannoma / acoustic neuroma)**，CN VIII (cranial nerve VIII, 第八对颅神经/前庭蜗神经) 施万细胞瘤。
> - 长得慢、前庭代偿好 → **真性眩晕反而少**，主诉常就是"一只耳朵嗡嗡 + 越来越听不清 + 打电话要换耳朵"。
> - **双侧听神经瘤 = NF2 (Neurofibromatosis type 2, 神经纤维瘤病 2 型)**。
> - 详细展开见 [[完整笔记/专题笔记/神经/神经_眩晕鉴别谱]] §三⑦。

## 六、可逆又便宜的别忘：耵聍栓塞

> [!success] 最省钱的诊断
> **耵聍栓塞 (cerumen impaction)** = 传导聋 (conductive) + 耳闷 + 耳鸣，**耳道冲洗/清理后立即好转**。
> 考试里遇到"老人单侧听力下降 + 耳鸣，耳镜见耵聍堵塞" → 先清耵聍再说，别一上来 MRI。同组可逆传导性病因：中耳积液、鼓膜问题。

## 七、一眼分流速记 ⭐

> [!tip] 考场 5 秒决策
> ```
> 搏动性(跟心跳)? ── 是 → 血管谱
>   ├─ 肥胖育龄女 + 头痛 + 视乳头水肿 → IIH（→MRV/LP，acetazolamide）
>   ├─ 鼓膜后红色搏动肿块       → glomus 球瘤（禁活检）
>   ├─ 老年 + 颈部 bruit         → 颈动脉狭窄
>   └─ 双侧 + 贫血/甲亢/孕       → 高输出，纠原发病
>             │
>             否（非搏动）
>             ↓
> 单侧 + 进行性 SNHL?  ── 是 → MRI 排听神经瘤
>   │ 否
>   ├─ 数小时眩晕 + 耳胀 + 低频波动聋 → Ménière
>   ├─ 近期用药 + 双侧高频聋         → 耳毒药（氨基糖苷不可逆/呋塞米·阿司匹林可逆）
>   ├─ 老年双侧高频 / 4kHz notch     → presbycusis / NIHL
>   └─ 耳镜见耵聍                    → 耵聍栓塞（清理可逆）
> ```

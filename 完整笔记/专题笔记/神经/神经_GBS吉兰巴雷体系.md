# 神经_GBS 吉兰-巴雷综合征体系（Guillain-Barré Syndrome）

> [!info] 一句话定位
> **GBS（Guillain-Barré syndrome，吉兰-巴雷综合征）= 免疫介导的急性炎性脱髓鞘性多发性神经根神经病（AIDP，acute inflammatory demyelinating polyradiculoneuropathy）**。前驱感染后**分子模拟（molecular mimicry）**攻击周围神经髓鞘。
> 招牌 = **急性上行性对称无力 + 腱反射消失（areflexia）+ CSF 蛋白细胞分离**。
> 派生自做对题 [[mistakes/uworld-mistakes_2026-06#^Q3630]]（前驱腹泻 Campylobacter）。

> [!warning] 命名撞车（务必分清）
> 库里有**两个 GBS**：
> - **本笔记 GBS = Guillain-Barré syndrome（吉兰-巴雷综合征）** → 神经，周围神经病。
> - **GBS = Group B Streptococcus（B 族链球菌 / 无乳链球菌 S. agalactiae）** → 新生儿败血/脑膜炎首位（[[mistakes/uworld-mistakes_2026-06#^Q3671]]）、产科 IAP。两者毫无关系。

## 一、前驱与病因（molecular mimicry）

| 前驱 | 要点 |
|---|---|
| **空肠弯曲菌（Campylobacter jejuni）腹泻** ⭐ | **最常见**前驱（发病前 ~1-3 周）；相关 GBS 偏轴索型、**预后稍差、恢复慢** |
| 呼吸道感染 | 肺炎支原体（Mycoplasma pneumoniae）、流感等 |
| 病毒 | **CMV（cytomegalovirus，巨细胞病毒）**、EBV（Epstein-Barr virus，EB 病毒）、HIV、**Zika（寨卡病毒）** |
| 其他 | 近期**手术**、偶有疫苗（极罕见，如旧版流感疫苗）|

> [!tip] 机制一句话
> 前驱病原表面糖脂与周围神经**神经节苷脂（ganglioside）**结构相似 → 抗体交叉攻击髓鞘/轴索（分子模拟）。

## 二、临床表现 ⭐

| 维度 | 要点 |
|---|---|
| 无力 | **上行性、对称**（足 → 腿 → 躯干 → 上肢/呼吸肌）；**弛缓性（LMN，下运动神经元）** |
| 反射 | **腱反射↓/消失（hyporeflexia/areflexia）** ← 与 UMN 病变相反，核心鉴别点 |
| 感觉 | 多**正常或轻度异常**（肢端刺痛/麻）；运动受累 > 感觉 |
| 疼痛 | **神经病理性背痛 / 肢痛**（儿童尤其突出，易被误当骨科/肌病）|
| 自主神经 | **自主功能障碍**：心律失常（**心动过缓/过速**）、血压波动、肠梗阻、尿潴留 → 可致命 |
| 颅神经 | 双侧**面神经麻痹（facial diplegia）**、球部症状（吞咽/构音）|
| 呼吸 | 膈肌受累 → **呼吸衰竭**（最危险）|

```
GBS 进展方向（上行性）
  足/小腿无力 + 腱反射先消失
        ↓ 数小时~数天
  大腿/躯干 → 上肢
        ↓
  呼吸肌（膈肌）/ 球部 / 自主神经 → 致命风险
峰值多在 2-4 周内；之后平台→恢复
```

## 三、变体（variants）

| 变体 | 招牌 | 抗体 |
|---|---|---|
| **AIDP**（经典脱髓鞘型）| 上述典型表现；欧美最常见 | — |
| **Miller-Fisher 综合征（MFS）** ⭐ | **眼肌麻痹（ophthalmoplegia）+ 共济失调（ataxia）+ 腱反射消失** 三联 | **抗 GQ1b** |
| **AMAN**（acute motor axonal neuropathy，急性运动轴索型）| 纯运动、轴索受累；与 Campylobacter、亚洲/儿童相关 | 抗 GM1/GD1a |

> [!tip] Miller-Fisher 记忆钩
> **"眼 + 步 + 反射"三联（ophthalmoplegia + ataxia + areflexia）+ 抗 GQ1b = MFS**。看到"复视 + 走不稳 + 反射消失"别只想小脑/脑干，想 MFS。

## 四、诊断 ⭐

- **临床为主**：急性上行对称无力 + 反射消失 + 前驱感染史。
- **CSF（cerebrospinal fluid，脑脊液）= 蛋白细胞分离（albuminocytologic dissociation）**：**蛋白↑、白细胞正常（0-5）**。常发病 1 周后才出现（早期可正常）。
- **NCS/EMG（nerve conduction study / electromyography，神经传导/肌电图）**：脱髓鞘改变（传导速度↓、传导阻滞）；轴索型见波幅↓。
- **脊髓 MRI（磁共振）= 常正常**（或仅**前根 / 马尾神经根强化**）——GBS 是周围神经病，做影像主要是**排除脊髓压迫/脊髓炎**，不是确诊 GBS。

> [!warning] ⭐ 根性痛 ≠ 压迫灶（Q19805 错点）
> GBS 可有**根性电击样痛（放射至足）+ 背痛**（神经根炎症所致），易被误当**神经根病/脊髓压迫**去找结构病灶。但 **GBS 脊髓 MRI 常正常**。"背痛 + 腿软"四鉴别：**GBS（MRI 正常 + 前驱感染 + 上行对称 + 反射消失）/ 硬膜外脓肿（发热 + 剧痛叩击痛 + 环形强化）/ 转移瘤（恶性史 + 骨性压迫）/ 横贯性脊髓炎（感觉平面 + 尿便障碍 + 节段 T2 高信号）**。源 [[mistakes/uworld-mistakes_2026-07#^Q19805]]（误选硬膜外骨性压迫）。

> [!info] CSF 五行表里的 GBS（对照记忆）
> 见 [[mistakes/uworld-mistakes_2026-06#^Q15232]]（Q15232 病毒性脑膜炎卡内 CSF 五行鉴别表）：
>
> | 诊断 | WBC | 葡萄糖 | 蛋白 |
> |---|---|---|---|
> | 细菌 | >1000（中性）| ↓<40 | ↑↑ |
> | 病毒 | 10-500（淋巴）| 正常 | 轻度↑ |
> | **GBS** | **0-5（正常）** | **正常** | **↑↑（分离）** |
>
> GBS 关键 = **细胞不高、蛋白高** → "分离"。别和病毒性脑膜炎（细胞升）混。

## 五、处理 ⭐

> [!danger] 最紧急步骤 = 肺功能测定（PFT，pulmonary function test）
> GBS 上行性瘫痪可累及膈肌 → **先评估/监测呼吸（FVC 用力肺活量 forced vital capacity、NIF 最大吸气负压 negative inspiratory force）+ 自主神经**；呼吸衰竭 → **先上呼吸机，再做免疫治疗**。

| 环节 | 要点 |
|---|---|
| 监测 | **呼吸（FVC 用力肺活量）+ 自主神经（心律/血压）**——死因主要是呼衰 + 自主危象 |
| 免疫治疗 | **IVIG（intravenous immunoglobulin，静脉免疫球蛋白）或血浆置换（plasmapheresis）**——二选一，等效 |
| **禁用** | **糖皮质激素（glucocorticoid）无效**——GBS 不用激素（与 MS 急性发作相反！）|
| 支持 | 防 VTE（静脉血栓栓塞）、疼痛管理、康复 |

> [!warning] 高频陷阱：激素
> **MS 急性发作 → 大剂量激素**；**GBS → 不用激素（IVIG/血浆置换）**。两个脱髓鞘病的治疗在激素上**正好相反**，爱考。

## 六、鉴别诊断（防错选）⭐

| 疾病 | 鉴别钥匙 |
|---|---|
| **MS（multiple sclerosis，多发性硬化）** | 中枢 CNS（central nervous system，中枢神经系统）/**UMN（上运动神经元）**、时空多发、CSF **寡克隆带**（细胞正常）；GBS 是周围 PNS（peripheral nervous system，周围神经系统）/**LMN/反射↓**、CSF **蛋白细胞分离** → 见 [[完整笔记/专题笔记/神经/神经_多发性硬化MS体系]] |
| **脊髓灰质炎 polio** | **不对称**弛缓瘫 + 前驱发热/无菌性脑膜炎 + 不全疫苗史；GBS 是**对称上行** |
| **肉毒中毒 botulism** | **下行性**（颅神经→四肢）+ 瞳孔散大固定 + 婴儿便秘蜂蜜史；GBS 是上行 + 瞳孔正常 |
| **蜱瘫 tick paralysis** | 上行弛缓瘫**酷似 GBS**，但**找到蜱、去蜱即愈**、CSF 正常（无蛋白细胞分离）|
| **重症肌无力 MG** | 波动性、**眼睑下垂/复视**、越用越差、反射**正常**；GBS 反射消失 |
| **横贯性脊髓炎 transverse myelitis** | **感觉平面** + 括约肌障碍 + UMN 征（晚期）；GBS 无感觉平面 |
| **低钾性周期性麻痹** | 发作性、查 **K↓**、反射↓但无前驱感染、补钾即恢复 |

> [!tip] 一句话定位口诀
> **上行 + 对称 + 反射消失 + 蛋白细胞分离 = GBS**；下行想肉毒，不对称想 polio，找蜱想蜱瘫，眼+步+反射想 Miller-Fisher。

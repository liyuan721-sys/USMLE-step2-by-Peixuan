# ALS 肌萎缩侧索硬化 — 运动神经元病 (Motor Neuron Disease)

> [!info] 起源
> 从"面瘫 vs 球麻痹"讨论延伸建立。**ALS (amyotrophic lateral sclerosis, 肌萎缩侧索硬化)** 是 USMLE 神经科高频"指纹病"：它是少数**同一患者同时出现 UMN + LMN 体征、却无感觉受累**的病。本笔记作为 ALS 复习单一真相源，与 [[完整笔记/专题笔记/神经/神经_面瘫与球麻痹鉴别]] 的真/假性球麻痹框架互补。

---

## §一 一句话锁定

> [!tip] Step 2 Pearl
> **同时有 UMN（上运动神经元）+ LMN（下运动神经元）体征，且无感觉受累 = ALS**（Mixed UMN and LMN signs **without sensory loss** = ALS）。
> - **UMN (upper motor neuron, 上运动神经元)** 征：反射亢进 hyperreflexia、痉挛 spasticity、Babinski +、假性球麻痹（情绪失禁）。
> - **LMN (lower motor neuron, 下运动神经元)** 征：肌萎缩 atrophy、肌束颤 fasciculations、无力 weakness、真性球麻痹（**舌肌纤颤** tongue fasciculations）。
> - **保留**：感觉、眼外肌、括约肌、认知（多数）—— 这些"不受累"是排除其他病的关键。

---

## §二 临床表现 (Presentation)

> [!info] 流行病学
> 多见于 **40 多岁起病**（usually around the 40s），进行性、致死性；散发为主，部分家族性（与 **C9orf72 / SOD1** 等基因相关）。

- **进行性无力**：**上肢 + 下肢同时**受累，可一处起病后扩散，UMN 与 LMN 体征**并存**。
- **无感觉缺失 (no sensory loss)** —— 与脊髓病、周围神经病的关键区别。
- **舌肌纤颤 (tongue fasciculations)** —— 因 **CN XII (hypoglossal nerve, 舌下神经)** 失神经支配（真性球麻痹成分）。
- **球部症状 (bulbar)**：构音障碍 dysarthria、面肌无力、吞咽困难 dysphagia。
- **假性球麻痹/情绪失禁 (pseudobulbar affect)**：不合时宜且持续的**哭 / 笑 / 打哈欠**（双侧皮质延髓束受损去抑制）。
- **分泌物管理障碍 (hypersecretion)**：咽部黏液堆积（口臭 halitosis）+ 肺部分泌物 → 无法清理分泌物 + 咽反射消失 → **吸入性肺炎 (aspiration pneumonia)** = **ALS 最常见死因 (MCC of death)**。

```
ALS = 运动系统全程受累, 感觉全程幸免
        皮质 (UMN)  ─┐
                     ├─► 同一患者并存 (反射↑ + 萎缩纤颤)
   脊髓前角/脑干 (LMN)─┘
   感觉通路 ── 完好 (no sensory loss) ← 排他钥匙
   眼外肌 / 括约肌 / 认知 ── 多数保留
```

## §三 诊断与处理流程 (Workup & Management)

> [!warning] 顺序是考点：先评呼吸，再确诊，再治疗
> ALS 题最常考的不是"诊断"而是"**下一步**"——而下一步永远先盯**呼吸**。

| 步骤 | 措施 | 说明 |
|---|---|---|
| **最紧急 (Most Urgent)** | **肺功能检查 (PFT, pulmonary function tests)** | 呼吸肌受累是致命点，任何 ALS 评估先查呼吸储备 |
| **最佳初始诊断 (Best Initial Dx)** | **肌电图 (EMG, electromyography)** | 显示广泛**失神经 (denervation) + 纤颤电位 (fibrillations)**，多节段 LMN 受累证据 |
| **呼吸功能尚未受累** | **利鲁唑 (riluzole, 谷氨酸拮抗剂 glutamate antagonist)** | 唯一确证延长生存的疾病修饰药；另有 **依达拉奉 (edaravone, 自由基清除剂)** 作辅助 |
| **呼吸功能已受累** | **插管 / 通气支持 (intubation / ventilatory support)** | 出现呼吸衰竭 → 通气支持（NIV, noninvasive ventilation, 无创通气 / 气管切开），晚期 PEG (percutaneous endoscopic gastrostomy, 经皮内镜胃造瘘) 管营养 |

> [!tip] 记忆钩
> **PFT → EMG → riluzole（呼吸好）/ 通气支持（呼吸差）**。"先量呼吸"是 ALS 所有 next-step 题的统一答案模板。

## §四 鉴别诊断 (别误诊为 ALS)

> [!danger] ALS 是排他性临床诊断 —— 凡有"不该出现"的体征，先排除可治病因
> 出现**感觉缺失 / 眼肌受累 / 括约肌障碍 / 单纯 UMN 或单纯 LMN**，都要先想别的病。

| 疾病 | 与 ALS 的关键区别 |
|---|---|
| **颈椎病性脊髓病 (cervical spondylotic myelopathy)** | 有**感觉平面 / 颈痛**、括约肌障碍；UMN 在下肢、LMN 在上肢但**有感觉** → 颈椎 MRI 必做（可治、漏诊代价大）。见 [[完整笔记/专题笔记/骨科/骨科_脊柱及腰背痛鉴别]] |
| **重症肌无力 (MG, myasthenia gravis)** | **波动性、易疲劳、眼睑下垂/复视**，**纯 NMJ (neuromuscular junction, 神经肌接头) 无 UMN**、无萎缩纤颤；重复刺激波幅↓ |
| **多灶性运动神经病 (MMN, multifocal motor neuropathy)** | **纯 LMN**、传导阻滞、抗 GM1 抗体 → **可治（IVIG, intravenous immunoglobulin, 静脉免疫球蛋白）**，必须与 ALS 分开 |
| **脊髓性肌萎缩 (SMA, spinal muscular atrophy)** | 儿童/婴儿、**纯 LMN（前角）**、有舌纤颤但**无 UMN**；SMN1 基因。见 [[完整笔记/专题笔记/儿科/儿科_婴儿松软floppy_infant鉴别]] |
| **B12 缺乏 / 亚急性联合变性** | 有**感觉受累（后索）** + UMN；ALS 无感觉 |
| **吉兰-巴雷 (GBS, Guillain-Barré syndrome)** | **急性**上行、对称、**反射消失（纯 LMN）**、CSF 蛋白细胞分离。见 [[完整笔记/专题笔记/神经/神经_GBS吉兰巴雷体系]] |

> [!info] ALS 与额颞叶痴呆谱系重叠
> 部分 ALS 与 **FTD (frontotemporal dementia, 额颞叶痴呆) — TDP-43 / C9orf72** 谱系重叠，可有行为/认知改变。见 [[完整笔记/专题笔记/神经/神经_老年痴呆鉴别]]。

## §五 与"真/假性球麻痹"框架的衔接

> [!tip] ALS 为何能同时占两栏
> ALS 的延髓受累可**同时**表现为：
> - **真性球麻痹 (bulbar, LMN)**：舌萎缩 + 纤颤、咽反射消失。
> - **假性球麻痹 (pseudobulbar, UMN)**：情绪失禁、下颌反射亢进。
>
> 这正是 [[完整笔记/专题笔记/神经/神经_面瘫与球麻痹鉴别]] §四的核心——同一病人两栏并存 + 无感觉受累 → 锁定 ALS。

## §六 复习自测

> [!question]- 50 岁男性，进行性四肢无力，查体上肢肌萎缩+肌束颤、下肢反射亢进 Babinski+、舌肌纤颤，感觉完全正常。最可能诊断？最佳初始诊断检查？
> **ALS**（UMN + LMN 并存 + 无感觉受累）。最佳初始诊断检查 = **EMG**；但若问"最紧急的下一步" = **肺功能检查 (PFT)**。

> [!question]- 确诊 ALS、肺功能尚正常的患者，应给什么药？
> **利鲁唑 (riluzole, 谷氨酸拮抗剂)**（± 依达拉奉）。一旦呼吸功能受累 → 转向**通气支持**。

> [!question]- 哪一条体征出现就基本不是 ALS？
> **感觉缺失**（也包括明显眼外肌麻痹 / 括约肌障碍）。ALS 是纯运动 + 无感觉；有感觉受累先想颈椎脊髓病、B12 缺乏等可治病因。

> [!question]- ALS 最常见的死因？机制？
> **吸入性肺炎 (aspiration pneumonia)**。分泌物管理障碍（hypersecretion + 咽反射消失 + 吞咽困难）+ 呼吸肌无力 → 误吸 + 清除障碍。

# 颅压增高诊疗顺序（Increased ICP — 识别 → 先影像后腰穿 → 病因分流 → 危象阶梯）

> [!info] 用途 + 缩写约定
> 派生自 Q106（NBME 11，IIH 先影像后腰穿）。本卡把**颅内压增高 increased intracranial pressure (ICP, 颅内压) 的诊疗顺序铁律**串成一条决策链，重点解决 NBME 反复考的 **"腰穿 LP 前为什么必须先影像"** + **占位 vs IIH 分流**。
> 缩写：**ICP** = intracranial pressure（颅内压）；**IIH** = idiopathic intracranial hypertension（特发性颅内高压，旧称假性脑瘤 pseudotumor cerebri）；**LP** = lumbar puncture（腰椎穿刺）；**CSF** = cerebrospinal fluid（脑脊液）；**CT** = computed tomography（计算机断层）；**MRI** = magnetic resonance imaging（磁共振成像）；**MRV** = magnetic resonance venography（磁共振静脉造影）；**CVST** = cerebral venous sinus thrombosis（脑静脉窦血栓）；**NPH** = normal pressure hydrocephalus（正常压力脑积水）。

> [!danger] ⭐ 置顶速查：神经急症"下一步影像/处理"决策链（CT vs MRI vs LP）
> UWorld/NBME 反复考、也是我反复错的点（**Q12214 恶性半球梗死连错 3 次**）。**背这张表到自动化。**
>
> | 急症场景 | 关键线索 | 下一步 | 陷阱（别选什么）|
> |---|---|---|---|
> | 急性缺血性卒中初评 | 突发局灶缺损 <4.5h | **非增强 CT** | 先看"有没有血"才谈溶栓；不是 MRI |
> | **大面积卒中后急性恶化**（恶性半球梗死）| 48h 内嗜睡/偏瘫加重/眼偏斜 | **非增强 CT** | ❌ MRI（慢、查急性出血不如 CT）— Q12214 |
> | 疑 SAH | "一生最痛"、脑膜刺激征 | **非增强 CT**→阴性仍疑则 **LP** 找黄变 | 别一上来 LP / CTA |
> | **疑 IIH**（颅高压）| 肥胖女 + 视乳头水肿 + CN VI + **CT 正常** | **LP** 测开放压 | ❌ 抢先给乙酰唑胺 / ❌ 加 MRI（除非疑 CVST 才 MRV）— Q3162 |
> | **重型 TBI + 颅高压** | Cushing 三联 + 脑水肿 | 非增强 CT 定范围 + **立即高渗盐/甘露醇** | ❌ 糖皮质激素（TBI 增死亡）/ ❌ 尼莫地平 / ❌ LP — Q18943 |
> | **疑 DAI** | 车祸/坠落 + **GCS 低但 CT 干净** | **MRI**（白质点状出血 + 灰白交界模糊）| 别当"脑萎缩"放过 — Q3299 |
> | 疑脑膜炎要 LP + 有危险指征 | 意识改变/视乳头水肿/局灶征/免疫抑制/新发癫痫 | **先 CT 再 LP** | 未排占位直接 LP = 脑疝 |
> | 疑脊髓压迫（SEA）| 脊柱痛 + 发热 + 进行性神经缺损 | **脊柱增强 MRI** | 别等三联齐全 — Q19891 |
>
> **三条背死的反射**：
> 1. **急诊 + 神经突然变差 / 要查出血 → 不增强 CT**（快 + 查血敏感）——别想 MRI"更敏感"。（Q12214 连错 3 次的死穴）
> 2. **颅高压 + 视乳头水肿 + CT 正常 → LP**（先影像后腰穿，既诊断又治疗）——别抢着给药、别加 MRI。
> 3. **外伤 GCS 低但 CT 干净 → MRI**（DAI 白质点状出血）——别错配脑萎缩。

## 一、识别 ICP↑（先认出来才谈处理）

> [!tip] ICP 升高的临床钥匙
> - **头痛**（常晨起加重 / 平躺加重 / Valsalva 加重）+ **恶心呕吐**（呕吐中枢受刺激，可喷射状）
> - **视乳头水肿 papilledema**（眼底镜 = 客观 ICP↑征，把"普通头痛"拉向颅内压问题）
> - **视野缺损 / 一过性视物模糊**（transient visual obscurations，IIH 尤其）
> - **展神经麻痹 (CN VI palsy)**：复视、患眼外展受限 —— CN VI 颅内行程长，是"非定位性"假定位征 (false localizing sign)，单纯 ICP↑即可压迫
> - **意识改变 / 局灶体征 / 癫痫** → 提示占位或疝，危险信号
> - **Cushing 三联**（颅压↑/即将脑疝 impending herniation）：**高血压（脉压增大）+ 心动过缓 + 呼吸不规则** —— 出现即急症

## 二、关键决策：腰穿 LP 前必须先影像 ⭐（最高频考点）

> [!danger] 铁律：疑 ICP↑ → 先 CT/MRI 排占位 → 再决定能否 LP
> **为什么不能直接 LP？** 若存在**占位性病变 (mass lesion) 或梗阻性脑积水 (obstructive hydrocephalus)**，颅内各腔室间存在**压力梯度**。此时从腰部放出 CSF → 下方压力骤降 → 脑组织被上方高压"挤"向枕骨大孔 → **脑疝 (brain herniation)**（尤其小脑扁桃体疝 / 钩回疝）→ 脑干受压、呼吸骤停、死亡。
> **影像要看什么**：有无 **mass effect（占位效应）/ 中线移位 (midline shift) / 脑室受压或不对称 / 后颅窝 (posterior fossa) 病变 / 基底池消失**。影像**干净**（无占位、无梗阻）→ LP 才安全。

> [!warning] LP 前先 CT 的具体指征（neuro / 感染共用）
> 不是每个头痛都先 CT，但**出现以下任一即先影像**：
> - 意识改变 (altered mental status)
> - **视乳头水肿 papilledema**
> - 新发癫痫
> - 局灶神经体征（偏瘫 / 失语 / CN 麻痹等）
> - 免疫抑制状态（HIV / 移植后等）
> 　※ 本指征同样适用于**疑脑膜炎要 LP** 时（先排占位再放液），见 [[完整笔记/专题笔记/感染/感染_脑膜炎CSF鉴别]]。

```
                  疑 ICP 增高（头痛 + 呕吐 + papilledema …）
                              │
                              ▼
            ① 神经影像 first：非增强 CT（急诊快）/ MRI（更敏感）
                              │
        ┌─────────────────────┼─────────────────────┐
        ▼                     ▼                     ▼
   有占位 / 中线移位       有脑室扩大 +           影像干净
   / 后颅窝病变           梗阻征象               （无 mass、无梗阻）
        │                     │                     │
        ▼                     ▼                     ▼
  ❌ 禁 LP（脑疝风险）   梗阻性脑积水           ✅ 此时才可 LP
   → 神经外科           → 解除梗阻 / 引流        → 测开放压 + 查 CSF 成分
   （切瘤 / 减压 / 引流）  （非 LP 放液）          （IIH 既诊断又治疗）
```

## 三、病因分流（影像 → 三大方向）

> [!info] 三条路：占位 / 脑积水 / IIH
> 影像是分流的总闸门：**影像有阳性发现 → 占位或脑积水**；**影像阴性 + 仍 ICP↑ → 走 IIH 路**（LP 测压确诊）。

### 3.1 占位性病变 mass lesion（影像阳性）

- **类型**：肿瘤 (tumor) / 出血 (hemorrhage) / 脓肿 (abscess)。
- **钥匙**：进行性头痛 + 局灶体征 / 癫痫 / 人格改变；儿童幕下瘤 = 晨起呕吐 + 共济失调。
- **下一步**：影像定性（增强 MRI）→ 神经外科评估（切除 / 活检 / 减压）+ 降颅压。

### 3.2 脑积水 hydrocephalus（影像见脑室扩大）

| 类型 | 机制 | ICP | 钥匙 / 处理 |
|---|---|---|---|
| 梗阻性 / 非交通性 obstructive | 脑室系统内梗阻（瘤 / 导水管狭窄）| **↑** | 急性 ICP↑；解除梗阻 / 脑室引流，**不靠 LP 放液** |
| 交通性 communicating | CSF 吸收障碍（SAH / 脑膜炎后）| ↑ | 处理原发 + 分流 |
| 正常压力脑积水 NPH | 脑室大但**压力正常**（脑实质萎缩）| **正常** | 步态 + 认知 + 尿失禁三联；**LP 放液后症状改善 → 分流术**（见 [[完整笔记/专题笔记/神经/神经_老年痴呆鉴别]]）|

### 3.3 特发性颅内高压 IIH（影像阴性后的主角）

> [!success] IIH = 排除性诊断（改良 Dandy 标准）
> **颅压增高症状/体征 + 视乳头水肿 + 影像无占位 + LP 开放压↑（>25 cmH₂O，部分教材 >250 mmH₂O）且 CSF 成分正常 + 无其他可解释病因。**

- **典型人群**：**育龄肥胖女性 (obese woman of childbearing age)**；诱因 — 维生素 A 过量 (vitamin A) / 异维 A 酸 isotretinoin、四环素类 tetracyclines。
- **诊断顺序**（NBME 反复考）：**先影像排占位 → 影像阴性 → LP**。**LP 在 IIH 里"既诊断又治疗"**：测开放压确诊 + 放液即刻缓解头痛。
- **治疗**：
  - 一线：**减重 (weight loss)**（含减重手术）+ **乙酰唑胺 acetazolamide（碳酸酐酶抑制剂，减少 CSF 生成）**；可换 topiramate 托吡酯。
  - 停诱因药（vitamin A / 四环素）。
  - **威胁视力时**（进行性视野缺损 / 视力下降）→ 外科：**视神经鞘开窗 (optic nerve sheath fenestration)** 保视力 / **脑脊液分流术 (CSF shunt，如 VP 分流)**。

> [!warning] IIH 易混陷阱（普适）
> ① 见"IIH"反射开 **acetazolamide** = 把治疗抢在确诊 LP 之前（NBME 考的是**诊断顺序**，CT 正常后下一步是 LP 不是给药）。
> ② CT 已正常时**常规 MRI 不是下一步** —— 除非专门**怀疑脑静脉窦血栓 CVST**（产后 / 高凝 / 口服避孕药）才加 **MRV**：CVST 也表现"颅压增高型"（头痛 + papilledema + 呕吐 ± 癫痫），CT 可正常。
> ③ **未排占位前别贸然 LP**（脑疝）；反过来，**CT 已正常 = 占位已排 → 此时 LP 安全且必要**。
> ④ 🔴 **IIH ≠ NPH（正常压力脑积水，normal pressure hydrocephalus）——别找错招牌**（Q107461 栽点）：IIH = **育龄胖女 + 颅压高 + 头痛 + 视觉症状（一过性黑矇/视乳头水肿）+ 无尿失禁**；NPH = **老年 + 压力正常 + Wet-Wobbly-Wacky（尿失禁 + 磁性步态 + 痴呆）**。**尿失禁是 NPH 的征、不是 IIH 的征**——别因"没尿失禁"排除 IIH，也别因见尿失禁把老年脑积水当 IIH。NPH 对比见 [[完整笔记/专题笔记/神经/神经_步态鉴别谱]] / [[完整笔记/专题笔记/神经/神经_老年痴呆鉴别]]。
> ⑤ 🔴 **横窦狭窄 ≠ 血栓**（Q107461 栽点）：IIH 本身可致横窦狭窄；**MRV 无血栓就不是 CVST → 不抗凝（不上肝素 heparin）**。只有 MRV/影像见血栓才是 CVST，见 [[完整笔记/专题笔记/神经/神经_CVST诊疗]]。

### 3.3 IIH 三大影像征机制 + 附图

> [!info]- IIH 三大影像间接征都是"颅高压传导"的结果（Q107461 深挖）
> 都不是器官本身有病，而是**慢性升高的 ICP 沿不同路径传导**造成的：
>
> **① 空蝶鞍 (empty sella)**：颅高压 → CSF 经**鞍膈 (diaphragma sellae)** 薄弱处持续下压 → CSF 疝入蝶鞍 → **垂体被压扁贴鞍底** → MRI 鞍区被 CSF 填充、垂体几乎不见。= 慢性 ICP↑ 间接征。
> > ⚠️ **别反射"空蝶鞍→垂体功能低下"**：残留垂体组织多**保留功能**，**原发性空蝶鞍 / IIH 相关者绝大多数内分泌正常、无症状**（偶见轻度高泌乳素血症 mild hyperprolactinemia，垂体柄效应 stalk effect；真正垂体功能低下 hypopituitarism 少见）。**只有继发性空蝶鞍**（垂体先被手术/放疗/Sheehan 综合征/垂体卒中/腺瘤消退破坏）**才可能有垂体功能低下**——病根是原损伤，非 CSF 填充。见到空蝶鞍先想"慢性 ICP↑ 标志"，有垂体受损史才查垂体功能。
>
> **② 后巩膜变平 (scleral flattening)**：视神经鞘蛛网膜下腔与颅内蛛网膜下腔**相通** → 颅高压顺视神经鞘传到眼球后极 → 顶压后巩膜使之变平。**同一机制**也解释 papilledema（压视盘）+ 视神经鞘扩张迂曲。
>
> **③ 横窦狭窄 (transverse sinus stenosis) = 外源性压迫，非血栓**：横窦是**低压、管壁可塌陷**的静脉窦 → 升高的 ICP 从外把它压窄。并形成**恶性循环**：
> ```
> 颅高压 → 压扁横窦 → 静脉回流阻 → 静脉窦压力↑
>    ↑                                     │
>    └── CSF 吸收↓(蛛网膜颗粒靠"CSF−窦压差"吸收) ←┘
> ```
> 静脉窦压力↑ → 蛛网膜颗粒吸收 CSF 的压力差变小 → CSF 吸收↓ → ICP 更高 → 又更压横窦…… = IIH **"静脉高压假说"**。
> ⚠️ **是狭窄 (stenosis) 不是血栓 (thrombosis)** —— 管子被压窄、管内无血栓 → **MRV 无血栓 = 不是 CVST = 不抗凝**（Q107461 错选肝素的根）。

## 四、急性 ICP 危象处理阶梯（Cushing 三联 / 疝征 → 立即降颅压）

> [!danger] 出现疝征（Cushing 三联 / 瞳孔变化 / 去皮质→去大脑姿势 / 意识骤降）= 急症
> 脑疝可迅速致呼吸衰竭 → 第一步永远是 **ABC**。

```
急性 ICP↑ / 即将脑疝
   │
   ▼ ① ABC：保气道 —— 意识↓/呼衰 → 气管插管 + 机械通气
   ▼ ② 体位：抬高床头 30°（head elevation，促进静脉回流）
   ▼ ③ 渗透治疗：高渗盐水 (hypertonic saline) 或 甘露醇 mannitol（渗透性脱水降脑容积）
   ▼ ④ 过度通气 (hyperventilation)：降 PaCO₂ → 脑血管收缩 → 快速但短效（仅作桥接）
   ▼ ⑤ 外科：血肿清除 / 去骨瓣减压 / 脑室外引流（针对病因）
```

> [!tip] 钥匙
> - **过度通气**降 PaCO₂ → 脑血管收缩 → 脑血容量↓ → 快速降 ICP，但只是**短效桥接**手段（等外科）。
> - 颞部外伤 + **清醒间期 (lucid interval)** → 硬膜外血肿 (epidural hematoma) → 钩回疝 (uncal herniation) → 压迫 CN III → **同侧瞳孔散大 + 对侧偏瘫** → 急诊手术（见 [[完整笔记/专题笔记/神经/神经_HighYield速查总览]] §三 颅内出血）。
> - 缺血性卒中后**大面积脑水肿** → mass effect + ICP↑ → 也走此阶梯（去骨瓣减压）。

> [!danger] 陷阱：糖皮质激素 (glucocorticoid) 在创伤性脑损伤 (TBI) 中禁用
> - **重型 TBI 的 ICP↑ / 弥漫脑水肿 → 首选渗透治疗（高渗盐水 / 甘露醇），不是激素。**
> - 糖皮质激素只对**血管源性水肿（肿瘤 tumor / 脓肿 abscess 周围）**有效；用于 **TBI 反而增加死亡率（CRASH 试验）→ 不推荐**。
> - 同禁忌：**尼莫地平 nimodipine**（脑血管扩张↑脑血容积↑ICP，只用于动脉瘤性 SAH；机制/陷阱详见 [[完整笔记/专题笔记/神经/神经_SAH蛛网膜下腔出血与尼莫地平]]）；**腰穿 LP**（颅高压诱发脑疝）。

## 五、各型脑疝分类（Brain Herniation — 靠瞳孔/偏瘫定位反推疝型）⭐

> [!danger] 为什么单列一节

| **钩回疝 uncal**（颞叶钩回，最经典考点）              | 颞叶内侧钩回 uncus 被挤过**小脑幕切迹**           | **同侧动眼神经 CN III** → 后续中脑/**大脑后动脉 PCA** | <font color="#ff0000">**同侧瞳孔散大固定</font> "blown pupil"（散大固定瞳）+ 上睑下垂 + 眼球向下外偏**（CN III）＋<font color="#ff0000">**对侧偏瘫**</font>；PCA (posterior cerebral artery) 受压→对侧同向偏盲 | 颞部外伤/**硬膜外血肿 epidural hematoma (EDH)**、外侧幕上占位 |
| **中央（经小脑幕）疝 central transtentorial**   | 间脑/脑干被**垂直向下**推过小脑幕                 | 自上而下脑干（间脑→中脑→脑桥）                       | <font color="#ff0000">**意识进行性↓** + **双侧针尖样小瞳孔</font>（尚有反应）→ 后期固定居中**；去皮质→**去大脑**姿势；Cheyne-Stokes → 中枢过度通气                                                             | 双侧/中线幕上占位、弥漫脑水肿                               |
| **小脑扁桃体疝 tonsillar**（枕骨大孔疝，最致命）        | 小脑扁桃体被挤过**枕骨大孔**                    | **延髓 medulla**（心肺中枢）                   | <font color="#ff0000">**Cushing 三联**</font>（高血压/脉压增大 + 心动过缓 + 呼吸不规则）→ **呼吸骤停**；颈强/头位强迫                                                                                | 后颅窝占位；**未排占位直接 LP** 的经典诱因                     |

> [!info]- 瞳孔机制脚注：为什么各疝瞳孔不一样（交感散 vs 副交感缩）
> 瞳孔大小 = **交感 (sympathetic, 散瞳：下丘脑/间脑 → 脑干被盖下行)** vs **副交感 (parasympathetic, 缩瞳：中脑 Edinger-Westphal 核 → CN III)** 的拔河。**谁被掐断，另一方就无对抗地占上风：**
> - **钩回疝 → 压 CN III（副交感）→ 缩瞳力没了 → 单侧散大固定 blown pupil**（交感无对抗）。
> - **中央疝早期（间脑期 diencephalic）→ 压间脑的下行交感束 → 散瞳力没了 → 双侧小/针尖 + 仍对光有反应**（副交感 EW 核尚存、反射传出臂完好，所以"缩小但仍收缩"）。
> - **中央疝进展（中脑期 midbrain）→ 连 EW 核也压了 → 交感 + 副交感全断 → 中位固定 (mid-position fixed, 4-5mm) 无反应**（比"小有反应"更糟 = 病情自上而下滑的实时标尺）。
> - **记忆钩：副交感断 → 散大（钩回·单侧）；交感断 → 缩小（中央·双侧）** —— 一散一缩、一单一双，正好相反。

> [!warning] 高频定位陷阱（UWorld 爱设的坑）
> - **"blown pupil"（单眼散大固定）= 同侧钩回疝压 CN III**，不是视神经问题 → 提示同侧幕上占位、需**急诊减压**，不是先做 MRI。
> - **⚠️ 此"减压"= 手术减压（清血肿 hematoma evacuation / 去骨瓣 decompressive craniectomy），卸掉梯度【上端】的高压源 → 脑归位、治疝；与 LP 从梯度【下端】腰部放 CSF 方向相反（LP 反而加大梯度 → 致疝，见本节末条）。** 同是"放压力"，救命还是致命只看放在梯度哪一端。去骨瓣的逻辑 = 给肿胀脑一个**向颅外(可控)膨出**的出口，替代**向枕骨大孔(致命)压脑干**。
> - **偏瘫侧会骗人——别用偏瘫侧定位，用瞳孔。** 以下两种情况**都发生在钩回疝里**（不是"别的疝对侧、钩回疝同侧"）：
>   - **常态（多数钩回疝）：对侧偏瘫** —— 占位压**同侧**大脑脚 (cerebral peduncle)，皮质脊髓束 (corticospinal tract) 未交叉 → 管**对侧**肢体 → 对侧无力（= 占位对侧，"定位对"）。跟"任何半球病灶→对侧无力"同理，非钩回疝特有。
>   - **例外（Kernohan notch 现象）：同侧偏瘫 = 假定位征 false localizing sign** —— 占位太大把脑干整体推向对侧，压到**对侧**大脑脚（顶在对侧小脑幕缘 Kernohan 切迹）→ 该束交叉后管**占位同侧**肢体 → 同侧无力，骗你把病灶定到对侧。
>   - **可靠钥匙 = 散大瞳 blown pupil（CN III），几乎总在占位同侧**：当瞳孔与偏瘫"对不上"（同侧散大瞳 + 同侧偏瘫）→ 想 Kernohan，**信瞳孔别信偏瘫**。
> - **Cushing 三联 = 即将/已发生脑干受压（多为扁桃体疝或中央疝晚期）**，是**急症**不是"血压高要降压"——先 **ABC + 降颅压**（见 §四），降血压反而减脑灌注。
> - **LP 的致命并发症 = 扁桃体疝**：占位/梗阻性脑积水时抽 CSF → 幕上-椎管压力梯度骤增 → 扁桃体挤进枕骨大孔（呼应 §二"LP 前先影像"）。

以右侧幕上占位（钩回疝）为例，偏瘫落哪侧只取决于压到哪个大脑脚——瞳孔才是稳定钥匙：

```
                        右侧幕上占位（钩回疝）
          ┌──────────────────────┴──────────────────────┐
          ▼ 常态：压【同侧 = 右】大脑脚          ▼ Kernohan：占位把脑干推向左，
            （多数）                              压【对侧 = 左】大脑脚（少数）
          │ 右束交叉后 → 管左侧肢体               │ 左束交叉后 → 管右侧肢体
          ▼                                      ▼
     左侧偏瘫 = 占位【对侧】（定位对）      右侧偏瘫 = 占位【同侧】（假定位征！）
          └──────────────────┬───────────────────┘
                             ▼
     无论哪种：散大瞳（CN III）几乎总在占位同侧 = 右
     → 用【瞳孔】定位占位侧，别用偏瘫侧（偏瘫会翻）
```

> [!tip] 一句话锁定
> **散大瞳在哪侧、疝就在哪侧（钩回疝）；对侧偏瘫是常态、同侧偏瘫想 Kernohan；Cushing 三联出现 = 脑干已到枕骨大孔，先救命再降颅压。**

## 六、速查对照表：占位/梗阻 vs IIH（怎么一眼分诊疗路径）

> [!info] 决策核心 = 影像阴阳 + 局灶征

| 维度 | 占位 / 梗阻性脑积水 | IIH（特发性颅内高压）|
|---|---|---|
| 典型人群 | 任何（瘤/出血/脓肿/儿童幕下瘤）| 育龄肥胖女；vit A / isotretinoin / 四环素 |
| 局灶体征 | 常有（偏瘫/失语/CN/癫痫/意识↓）| **无**（除 CN VI 假定位性 + papilledema）|
| 影像 | **阳性**（mass / 中线移位 / 脑室扩大 / 后颅窝）| **阴性**（无占位、无脑积水）|
| LP 安全性 | **未排占位前禁 LP**（脑疝）| 影像阴性后 **LP 安全且确诊**（开放压↑、CSF 正常）|
| 关键下一步 | 神经外科（切/减压/引流）+ 降颅压 | 影像阴 → **LP** → 减重 + acetazolamide |
| 急症处理 | 抬头 + 高渗盐/甘露醇 + 过度通气 + 手术 | 威胁视力 → 视神经鞘开窗 / 分流 |

## 七、复习题

> [!question]- Q1. 27 岁肥胖女，复发头痛 + 呕吐 + 视乳头水肿，头 CT 正常。下一步最佳处理？
> **腰椎穿刺 (LP)** 测开放压。CT 已排占位 → IIH 诊断顺序"先影像后 LP"的影像步已完成，下一步是 LP 确诊（开放压↑ + CSF 成分正常），**不是**再加 MRI，也**不是**直接给 acetazolamide（治疗抢在确诊前）。（派生自 Q106）

> [!question]- Q2. 为什么疑 ICP 增高、见 papilledema 时不能直接 LP，要先 CT/MRI？
> 若有**占位或梗阻性脑积水**，颅内存在压力梯度，从腰部放 CSF → 下方压力骤降 → 脑组织被挤向枕骨大孔 → **脑疝**、脑干受压、呼吸骤停。先影像排 mass effect / 中线移位 / 后颅窝病变，影像干净才安全 LP。

> [!question]- Q3. 育龄肥胖女 + papilledema + 影像阴性 + LP 开放压↑、CSF 成分正常，诊断与一线治疗？
> 诊断 = **IIH（特发性颅内高压）**。一线治疗 = **减重 + 乙酰唑胺 acetazolamide**（碳酸酐酶抑制剂，减少 CSF 生成）；停诱因药（vit A / 四环素）。威胁视力 → 视神经鞘开窗 / CSF 分流。

> [!question]- Q4. CT 正常的 ICP↑患者，何时该加 MRV 而非直接 LP？
> 临床怀疑**脑静脉窦血栓 CVST**时（产后 / 高凝状态 / 口服避孕药 / 单侧征）。CVST 表现"颅压增高型"（头痛 + papilledema + 呕吐 ± 癫痫）但 CT 可正常 → 加 **MRV** 找窦血栓。否则 CT 正常即可 LP。

> [!question]- Q5. 昏迷患者：高血压（脉压增大）+ 心动过缓 + 呼吸不规则，说明什么？第一步？
> **Cushing 三联** = ICP 严重升高 / 即将脑疝（脑干受压）。第一步 = **ABC（气道）** → 意识↓/呼衰先气管插管机械通气；随后抬高床头 + 高渗盐/甘露醇 + 过度通气（短效桥接）+ 针对病因外科减压。

> [!question]- Q6. 颞部外伤后昏迷，右侧瞳孔散大固定、左侧偏瘫，是哪型脑疝？压迫了什么？
> **右侧钩回疝 (uncal herniation)**（右侧幕上占位，多为硬膜外血肿 EDH）。钩回压迫**同侧动眼神经 CN III** → 右瞳散大固定 "blown pupil" + 上睑下垂 + 眼球向下外偏；对侧皮质脊髓束受累 → **左侧偏瘫**。散大瞳的一侧 = 疝的一侧 → 需**急诊减压**，不是先做 MRI。（若反而出现同侧偏瘫 → 想 **Kernohan notch** 假定位征）

> [!question]- Q7. 疑颅高压患者未做影像直接腰穿后呼吸骤停，机制是哪型脑疝？
> **小脑扁桃体疝 (tonsillar herniation, 枕骨大孔疝)**。占位/梗阻性脑积水存在幕上-椎管压力梯度时抽 CSF → 下方压力骤降 → 小脑扁桃体被挤过**枕骨大孔**压迫**延髓 medulla（心肺中枢）** → Cushing 三联 → 呼吸骤停。这就是"LP 前必须先影像排占位"的铁律来源（§二）。

# 神经_周围神经病：轴突性 vs 脱髓鞘（Peripheral Neuropathy — Axonal vs Demyelinating）

> [!info] 一句话定位
> 周围神经 = **轴突（axon，"电线"）+ 髓鞘（myelin sheath，Schwann 细胞做的"绝缘层"）**。
> 病变打在哪一层，决定了**神经传导（NCS）表现、病因归类、恢复速度**——这是 USMLE 反复考的二分法。
> **糖尿病（diabetes）+ 酒精（alcohol）= 轴突性；GBS（Guillain-Barré syndrome，吉兰-巴雷综合征）/ CIDP（chronic inflammatory demyelinating polyneuropathy，慢性炎症性脱髓鞘性多神经病）= 脱髓鞘。**

---

## 一、轴突性 vs 脱髓鞘 总纲 ⭐

> [!tip] 一句话核心
> **轴突坏 = 波幅掉（amplitude↓）；髓鞘坏 = 速度掉（velocity↓）。** 这是肌电图（EMG，electromyography）/ 神经传导研究（NCS，nerve conduction study）判读的钥匙。

| 维度 | 轴突性 Axonal | 脱髓鞘 Demyelinating |
|---|---|---|
| 坏的部位 | 轴突本体（"电线断了"） | 髓鞘（"绝缘层被剥，电线还在"）|
| 传导速度 CV (conduction velocity) | 基本正常或轻降 | **显著变慢**（跳跃传导 saltatory conduction 崩） |
| 波幅 Amplitude（CMAP 复合肌肉动作电位 / SNAP 感觉神经动作电位） | **明显降低**（有功能的轴突少了） | 早期相对保留；可见**传导阻滞 conduction block** |
| 恢复 | 慢、常不完全（轴突再生 ~1 mm/天） | 快、可较完全（髓鞘重造快） |
| 典型分布 | length-dependent，**远端先**（最长轴突最先饿死）| 可近远端同时、甚至**近端为主** |
| 常见病因大类 | **代谢 / 中毒 / 营养缺乏** | **免疫 / 遗传 / 感染** |

> [!tip] Memory Hook
> **A**xonal → **A**mplitude 掉（都以 A 开头）；De**myelin**ating → **velocity** 掉。
> 轴突性"从脚起、爬得慢、修得慢"；脱髓鞘"可从近端起、跑得快（急性）、修得也快"。

## 二、病因归类（考点：看病因反推分型）

```
周围多神经病（polyneuropathy）
│
├─ 轴突性 Axonal（代谢/中毒/营养——最常见一大类）
│    ⭐ Diabetes 糖尿病        ← 发达国家周围神经病（PN, peripheral neuropathy）第一位
│    ⭐ Alcohol 酒精 + B1 缺乏
│    ├─ 尿毒症 Uremia
│    ├─ B12 缺乏（注意兼后索/侧索 → 亚急性联合变性）
│    ├─ 化疗药（cisplatin 顺铂 / vincristine 长春新碱 / paclitaxel 紫杉醇）
│    └─ 重金属（arsenic 砷 / lead 铅）、甲状腺功能减退
│
└─ 脱髓鞘 Demyelinating（免疫/遗传/感染）
     ├─ GBS / AIDP（acute inflammatory demyelinating polyradiculoneuropathy，急性炎性脱髓鞘性多发性神经根神经病）吉兰-巴雷（急性、前驱感染、上行性、蛋白细胞分离）
     ├─ CIDP（慢性炎症性脱髓鞘，>8 周，激素/IVIG 有效）
     ├─ Charcot-Marie-Tooth type 1（遗传，弓形足 pes cavus + 高位）
     └─ 白喉 diphtheria
```

> [!warning] GBS 的例外
> GBS 主型 AIDP 是脱髓鞘，但**轴索亚型 AMAN**（acute motor axonal neuropathy，与空肠弯曲菌 Campylobacter、亚洲/儿童相关，抗 GM1/GD1a）是轴突性——预后稍差、恢复慢。详见 [[完整笔记/专题笔记/神经/神经_GBS吉兰巴雷体系]]。

## 三、Distal Symmetric Polyneuropathy（远端对称性多神经病）临床骨架

代谢/中毒性（糖尿病、酒精为代表）几乎都长成这一个样子：

- **起病**：慢性、隐匿，**从脚趾/足底往上**（length-dependent）
- **分布**：对称 **stocking-glove（手套袜套样）**，先下肢后上肢
- **感觉**：麻木、**burning pain（灼痛）**、振动觉/本体觉减退、**踝反射（ankle reflex）先消失**
- **性质**：轴突性为主 → NCS 波幅↓、速度基本正常

> [!question]- 为什么糖尿病/酒精是"远端先、袜套样"，而 GBS 不是？
> 糖尿病/酒精是**轴突性 + length-dependent**：最长的神经（脚）离胞体最远、代谢最吃紧，最先撑不住 → 从脚起。
> GBS 是**脱髓鞘 + 免疫攻击神经根**：不遵守长度规律，可**近端和远端一起中招、甚至先累及呼吸肌/面肌**，典型是**上行性**而非单纯远端起。

## 四、糖尿病 vs 酒精性神经病（两个轴突性代表的鉴别）⭐

> [!info] 大方向
> 两者在床边**症状几乎一样**（远端对称、袜套样、振动觉↓、踝反射消失）——这一层是 stem 的"陷阱铺垫"。USMLE 想让你靠**合并线索 + 处理侧重**区分。

| | 糖尿病性 Diabetic | 酒精性 Alcoholic |
|---|---|---|
| 核心机制 | 高血糖直接毒性 + 微血管缺血 | **营养不良为主**：thiamine（B1，硫胺素）等 B 族缺乏 + 乙醇直接轴突毒性 |
| 疼痛 | 可有可无 | **常更突出、更早、burning 更重** |
| 伴随线索 | autonomic neuropathy（自主神经）、**CN III 单神经病（瞳孔回避 pupil-sparing）**、diabetic amyotrophy（腰骶神经根丛病）、retinopathy / nephropathy 同框 | Wernicke-Korsakoff、**巨幼细胞性贫血（macrocytic anemia）**、肝病体征、全身营养不良 |
| 关键处理 | **控糖** + 对症（加巴喷丁 gabapentin / 普瑞巴林 pregabalin / 度洛西汀 duloxetine / 三环类抗抑郁药 TCA, tricyclic antidepressant） | **戒酒 + 补 thiamine/B 族** + 对症 |

> [!warning] 易错点
> - 酒精性神经病**即使不缺 B1 也会发生**（乙醇本身有轴突毒性）；但 USMLE 里酒精题几乎必配 thiamine/B 族缺乏这条线——看到酗酒顺手想 **Wernicke + 补 B1**。
> - 补 thiamine **要在给葡萄糖之前/同时**：先给糖会耗竭残余 B1 → 诱发 Wernicke 脑病。

> [!tip] Memory Hook
> Stem 给"脚麻 + 振动觉↓ + 踝反射没了"——**看同框线索定病因**：
> 配 retinopathy / 血糖高 → 糖尿病；配酗酒史 / 巨幼贫 / Wernicke → 酒精 + B1 缺乏。
> **症状一样是幌子，处理侧重（控糖 vs 补 B1+戒酒）才是考点。**

## 五、糖尿病神经病变的其他面孔（别只记袜套样）

糖尿病神经病变**不止**远端对称性多神经病一种，stem 会换面孔考：

- **Autonomic neuropathy 自主神经病**：体位性低血压、胃轻瘫（gastroparesis）、无痛性心梗、勃起功能障碍、静息心动过速
- **CN III mononeuropathy 动眼神经单神经病**：复视 + 上睑下垂，**瞳孔回避（pupil-sparing）**（缺血打中央运动纤维，边缘副交感纤维靠软膜血供得以幸免）→ 与压迫性（动脉瘤/钩回疝，瞳孔散大）相反
- **Diabetic amyotrophy（腰骶神经根丛病）**：单侧大腿剧痛 + 近端无力萎缩
- **Mononeuritis multiplex**：多根孤立神经先后受累

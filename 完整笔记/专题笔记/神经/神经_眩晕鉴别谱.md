# 神经_眩晕鉴别谱（Vertigo 五病 + Weber/Rinne）

> [!danger] 为什么建这张表
> 2026-06-10 一场 ENT 块里**眩晕题错了 3 道**，且**两次反射性误选 BPPV**（[[mistakes/uworld-mistakes_2026-06#^Q3449]] 实为 Ménière、[[mistakes/uworld-mistakes_2026-06#^Q16433]] 实为外淋巴瘘）。
> **病根**：见"眩晕"先想耳石，不先核 **①时长 ②听力 ③诱因**。这张表把"先核三要素"固化成反射。
> 配合主笔记 [[完整笔记/Peixuan分科笔记/neuro]] 的 Vertigo 章节（men wear **VEST** 口诀）。

---

## 一、先核三要素（看到 vertigo 第一反应）⭐

```
眩晕题 → 别急着选 → 先问三句：
  ① 时长多久？   <1min / 数分-数小时 / 数天持续
  ② 有没有听力下降 / 耳鸣？   有 = 累及耳蜗
  ③ 什么诱发？   头位 / 压力(Valsalva) / 无 / 伴头痛
```

> [!tip] 一句话锁定
> - **<1 分钟 + 头位诱发 + 听力正常** = **BPPV**（别一见眩晕就选它！）
> - **数小时 + 耳鸣 + 感音聋 + 无诱因** = **Ménière**
> - **压力诱发(Valsalva/电梯/喷嚏) + 外伤 + 进行性聋** = **外淋巴瘘**
> - **数天持续 + 病毒后 + 听力正常** = **前庭神经炎**（"一场大晕好几天，听力没掉是前庭炎"）
> - **数天持续 + 病毒后 + 听力下降** = **迷路炎**（前庭+耳蜗都中招）

## 二、五（+）病大对照表 ⭐⭐

| 病 | 时长 | 诱因 | 听力/耳鸣 | 招牌钥匙 | 错题 |
|---|---|---|---|---|---|
| **BPPV 良性阵发性位置性眩晕** | **<1 min** | **头位变动** | **正常** | Dix-Hallpike 诱发眼震；耳石 | — |
| **Ménière 梅尼埃病** | **20min–24h** | **无明确诱因** | **感音聋 + 耳鸣/耳胀** | 内淋巴积水；men wear VEST | [[mistakes/uworld-mistakes_2026-06#^Q3449]] [[mistakes/uworld-mistakes_2026-06#^Q108098]] |
| **外淋巴瘘 Perilymphatic fistula** | 短暂（秒-分） | **压力变化** Valsalva/电梯/喷嚏/巨响 | **进行性 SNHL** | 外伤/气压伤 + 压力诱发眼震（Tullio 现象）| [[mistakes/uworld-mistakes_2026-06#^Q16433]] |
| **前庭神经炎 Vestibular neuritis** | **数天持续** | 病毒感染后 | **正常** | head thrust 异常；无耳蜗症状 | — |
| **迷路炎 Labyrinthitis** | 数天持续 | 病毒感染后 | **下降**（前庭+耳蜗）| 同前庭炎但**带聋** | — |
| **前庭性偏头痛** | 数分-数天 | 伴头痛/先兆 | 多正常 | 发作间完全正常；偏头痛史 | — |
| **听神经瘤 Vestibular schwannoma** | **持续进行性** | 无 | 单侧进行性 SNHL + 耳鸣 | 缓慢生长少真性眩晕（CN VIII）| — |
| **中枢（小脑/脑干卒中）** | 突发持续 | 无 | 多无 | **其他 CNS 体征** + 垂直/变向眼震 | — |

> [!warning] BPPV vs Ménière vs 外淋巴瘘（最常混的三角）
> 三者都可"短暂眩晕"，分水岭是**诱因 + 听力**：
> - BPPV：**头位**诱发，**听力正常**。
> - Ménière：**无诱因**，**感音聋 + 耳鸣**。
> - 外淋巴瘘：**压力**诱发（非头位），**进行性聋 + 外伤史**。

## 三、Weber / Rinne 判读（伴聋型眩晕必备）⭐

> [!info] 两个音叉试验定位聋的性质
> 用于 Ménière / 迷路炎 / 外淋巴瘘等"伴聋"题，判断是**传导聋**还是**感音神经性聋 SNHL**。

```
Weber（音叉置前额/颅顶，问哪侧响）
  传导聋 (CHL) → 偏向【患耳】（患耳屏蔽环境噪音，骨导更显）
  感音聋 (SNHL)→ 偏向【健耳】

Rinne（音叉先骨导后气导，问哪个长/响）
  正常 / SNHL → 气导 > 骨导（AC > BC，"Rinne 阳性"）
  传导聋      → 骨导 > 气导（BC > AC，"Rinne 阴性"）
```

> [!example]- 实战（[[mistakes/uworld-mistakes_2026-06#^Q3449]]）
> Weber **偏左响** + Rinne **气导>骨导** → 左为健耳 → **右耳 SNHL** → 配合发作性眩晕 + 耳鸣 = **Ménière**。
> （BPPV 的听力是**正常**的，这是和 Ménière 的硬分水岭。）

| | Weber | Rinne |
|---|---|---|
| **正常** | 居中 | 气导>骨导（双侧）|
| **传导聋 CHL** | 偏**患**耳 | 骨导>气导（患侧）|
| **感音聋 SNHL** | 偏**健**耳 | 气导>骨导 |

> [!tip] 记不住？背「一个动作」推全部（堵耳哼歌锚点）
> 亲自做一次记死：手指堵住自己一只耳朵 + 哼歌 → **被堵那只耳更响**。被堵 = 人为造的**传导聋** → **传导聋 Weber 偏患耳**。剩下全推：
>
> ```
> 传导聋 Weber→患耳（堵耳哼歌亲测✓）
>     ↓ 反过来
> 感音聋 Weber→健耳（坏神经那侧听不清）
>
> Rinne 正常人 AC>BC（气导本就强 = 阳性）
>     ↓ 传导聋堵了患耳的气导路
> 传导聋患耳 BC>AC = 阴性（唯一翻转的情况）
>     ↓ 感音聋两条路按比例同降
> 感音聋 仍 AC>BC = 阳性（只是整体变小声）
> ```

> [!warning] 两个最坑点：命名反直觉 + Weber 单独定不了患侧
> - **Rinne 阳性 = 正常**（positive = proper），只有**传导聋**才把 Rinne 翻成**阴性**。
> - **Weber 单看分不清患侧**：偏右 = 右传导聋 **或** 左感音聋（两种相反情况给出同样的偏右）→ 必须配 Rinne 才能拆开。

> [!tip] 解题顺序：先 Rinne 后 Weber
> ```
> ① 先 Rinne 分型（每耳单测，确定性强）：
>    某耳 BC>AC（阴性）→ 该耳 = 传导聋，患侧已定
>    两耳都 AC>BC（阳性）→ 无传导聋（正常 or 感音聋）
> ② 再 Weber 定位 / 验证：
>    已抓到传导聋耳 → Weber 应偏【该耳】= 自洽确认
>    两耳 Rinne 都阳性 → Weber 偏健耳 → 偏离的【对侧】= 感音聋患耳
> ```
> 一句话：**先 Rinne 分型抓传导耳 → 再 Weber 验证（传导同向）或反向定位（感音偏健耳）**。

## 四、外周 vs 中枢眩晕（排卒中）

| 特征 | Peripheral（外周）| Central（中枢）|
|---|---|---|
| 病因 | BPPV / 前庭神经炎 / Ménière / 迷路炎 | 卒中 / MS / 偏头痛 |
| 眼震 | 水平/旋转、**单向**、凝视可抑制、疲劳性 | 可**垂直**、**变向/双向**、凝视不能抑制、不疲劳 |
| 听力 | **可有**（Ménière/迷路炎）| 通常无 |
| 其他神经体征 | 通常无 | **常有 CNS 症状** |

> [!danger] 必须查中枢病因（怀疑小脑卒中/出血）
> **垂直/变向眼震、凝视不能抑制、伴其他 CNS 体征、HINTS 提示中枢** → 立即影像排卒中（尤其有血管危险因素的"孤立性持续眩晕"）。

## 五、治疗速览

| 病 | 一线 | 备注 |
|---|---|---|
| **BPPV** | **Epley 手法**（复位）| 急性可美克洛嗪 meclizine |
| **Ménière** | **限盐 + 限咖啡因 + 限酒**；急性美克洛嗪 | 二线利尿剂（氢氯噻嗪 hydrochlorothiazide）/ 倍他司汀 betahistine；**绝不增盐** |
| **外淋巴瘘** | 避免增内耳压活动 + **ENT 转诊** | 必要时手术修补 |
| **前庭神经炎/迷路炎** | 对症（美克洛嗪）+ treat the bug | 迷路炎带聋 |
| **前庭性偏头痛** | 偏头痛预防/急性方案 | — |

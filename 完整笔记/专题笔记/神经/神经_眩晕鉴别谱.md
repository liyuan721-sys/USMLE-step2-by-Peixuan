# 神经_眩晕鉴别谱（Vertigo 五病 + Weber/Rinne）

> [!danger] 为什么建这张表
> 2026-06-10 一场 ENT 块里**眩晕题错了 3 道**，且**两次反射性误选 BPPV**（[[mistakes/uworld-mistakes_2026-06#^Q3449]] 实为 Ménière、[[mistakes/uworld-mistakes_2026-06#^Q16433]] 实为外淋巴瘘）。
> **病根**：见"眩晕"先想耳石，不先核 **①时长 ②听力 ③诱因**。这张表把"先核三要素"固化成反射。
> 配合主笔记 [[完整笔记/Peixuan分科笔记/neuro]] 的 Vertigo 章节（men wear **VEST** 口诀）。

---

## 一、先核三要素（看到 vertigo 第一反应）⭐

```
眩晕题 → 别急着选 → 先问三句：
  ① 时长多久？   <1min / 数分-数小时 / 数天持续
  ② 有没有听力下降 / 耳鸣？   有 = 累及耳蜗
  ③ 什么诱发？   头位 / 压力(Valsalva) / 无 / 伴头痛
```

> [!tip] 一句话锁定
> - **<1 分钟 + 头位诱发 + 听力正常** = **BPPV**（别一见眩晕就选它！）
> - **数小时 + 耳鸣 + 感音聋 + 无诱因** = **Ménière**
> - **压力诱发(Valsalva/电梯/喷嚏) + 外伤 + 进行性聋** = **外淋巴瘘**
> - **数天持续 + 病毒后 + 听力正常** = **前庭神经炎**（"一场大晕好几天，听力没掉是前庭炎"）
> - **数天持续 + 病毒后 + 听力下降** = **迷路炎**（前庭+耳蜗都中招）

## 二、五（+）病大对照表 ⭐⭐

| 病 | 时长 | 诱因 | 听力/耳鸣 | 招牌钥匙 | 错题 |
|---|---|---|---|---|---|
| **BPPV 良性阵发性位置性眩晕** | **<1 min** | **头位变动** | **正常** | Dix-Hallpike 诱发眼震；耳石 | — |
| **Ménière 梅尼埃病** | **20min–24h** | **无明确诱因** | **感音聋 + 耳鸣/耳胀** | 内淋巴积水；men wear VEST | [[mistakes/uworld-mistakes_2026-06#^Q3449]] [[mistakes/uworld-mistakes_2026-06#^Q108098]] |
| **外淋巴瘘 Perilymphatic fistula** | 短暂（秒-分） | **压力变化** Valsalva/电梯/喷嚏/巨响 | **进行性 SNHL** | 外伤/气压伤 + 压力诱发眼震（Tullio 现象）| [[mistakes/uworld-mistakes_2026-06#^Q16433]] |
| **前庭神经炎 Vestibular neuritis** | **数天持续** | 病毒感染后 | **正常** | head thrust 异常；无耳蜗症状 | — |
| **迷路炎 Labyrinthitis** | 数天持续 | 病毒感染后 | **下降**（前庭+耳蜗）| 同前庭炎但**带聋** | — |
| **前庭性偏头痛** | 数分-数天 | 伴头痛/先兆 | 多正常 | 发作间完全正常；偏头痛史 | — |
| **听神经瘤 Vestibular schwannoma** | **持续进行性** | 无 | 单侧进行性 SNHL + 耳鸣 | 缓慢生长少真性眩晕（CN VIII）| — |
| **中枢（小脑/脑干卒中）** | 突发持续 | 无 | 多无 | **其他 CNS 体征** + 垂直/变向眼震 | — |

> [!warning] BPPV vs Ménière vs 外淋巴瘘（最常混的三角）
> 三者都可"短暂眩晕"，分水岭是**诱因 + 听力**：
> - BPPV：**头位**诱发，**听力正常**。
> - Ménière：**无诱因**，**感音聋 + 耳鸣**。
> - 外淋巴瘘：**压力**诱发（非头位），**进行性聋 + 外伤史**。

## 三、单病速记卡（逐病 stem + 治疗）⭐⭐

> [!info] 对照表给"横向鉴别"，这一节给"每个病纵向 stem + 处理"
> 表里分得清，考场还要会"看到这个病下哪一步"。每病：典型 stem → 招牌钥匙 → 诊断 → 治疗 → 陷阱。

**① BPPV 良性阵发性位置性眩晕（benign paroxysmal positional vertigo）**
- 本质：耳石（otoconia）脱落入**后半规管**，头位变动 → 短暂眩晕（最常见外周眩晕）
- Stem：翻身/抬头/躺下突发天旋地转，**<1 分钟**自停、反复，**听力正常、无耳鸣**
- 钥匙：**<1min + 头位诱发 + 听力正常**（三条齐）
- 诊断：**Dix-Hallpike** 诱发带潜伏期、疲劳性旋转眼震
- 治疗：**Epley 复位法**（canalith repositioning，一线根治）；美克洛嗪 meclizine 仅对症
- 陷阱：⚠️ 别一见眩晕就反射 BPPV——缺"听力正常"要想 Ménière / 迷路炎

**② Ménière 梅尼埃病（Ménière disease）**
- 本质：**内淋巴积水** endolymphatic hydrops，压迫前庭 + 耳蜗
- Stem：反复 **20min–24h** 眩晕、**无明确诱因**，单侧**感音聋 + 耳鸣 + 耳胀满**，听力阶梯式下降
- 钥匙：口诀 men wear **VEST**（Vertigo + SNHL + Tinnitus）；**数小时 + 无诱因 + 感音聋**
- 诊断：临床；听力图示低频感音聋
- 治疗：**低盐 lower the salt**（+ 利尿剂）为基础；急性发作 **美克洛嗪 meclizine**；难治 → 鼓室注射庆大霉素 gentamicin / 手术
- 陷阱：与 BPPV 分水岭 = **时长（数小时）+ 有聋耳鸣**

**③ 外淋巴瘘 Perilymphatic fistula**
- 本质：卵圆窗/圆窗破裂，**外淋巴漏入中耳**；外伤、气压伤、用力 Valsalva、巨响
- Stem：**压力诱发**短暂眩晕（擤鼻/喷嚏/电梯/潜水/举重/听到巨响 = **Tullio 现象**），**外伤/气压伤史 + 进行性 SNHL**
- 钥匙：**压力 / Valsalva 诱发（非头位）+ 外伤 + 进行性聋**
- 诊断：病史 + 瘘管试验（外耳道加压诱发眼震）
- 治疗：先**保守**（卧床、避 Valsalva、抬高床头）；不愈 → 手术修补
- 陷阱：Q16433 错点——别和 BPPV 混（BPPV 头位诱发、听力正常）

**④ 前庭神经炎 Vestibular neuritis**
- 本质：**病毒感染后** CN VIII **前庭支**炎症，**不累及耳蜗**
- Stem：**单场剧烈眩晕持续数天**（非发作性），**前驱病毒 / URI**，恶心呕吐、走路偏向患侧，**听力正常**
- 钥匙：口诀"一场大晕好几天，听力没掉是前庭炎"；**head thrust（甩头）异常**
- 诊断：临床；**HINTS 提示外周**（甩头纠正性扫视阳性、单向眼震、无 skew）→ 三步逐项判读见 **§六 HINTS 三步床旁检查**
- 治疗：治病毒 / 对症（treat the bug）+ **美克洛嗪 meclizine** 短期 + 前庭康复
- 陷阱：与迷路炎只差**听力**

**⑤ 迷路炎 Labyrinthitis**
- 本质：**病毒感染后**炎症**同时累及前庭 + 耳蜗**（labyrinth = 前庭 + 耳蜗）
- Stem：同前庭神经炎（数天持续 + 病毒后），但**多了听力下降 / 耳鸣**
- 钥匙：前庭炎"带聋" = 迷路炎
- 治疗：治病毒 + **美克洛嗪 meclizine**
- 陷阱：**聋了 = 迷路炎，没聋 = 前庭神经炎**

**⑥ 前庭性偏头痛 Vestibular migraine**
- Stem：数分–数天眩晕 + **头痛 / 先兆 / 畏光** + **偏头痛史**；发作间期完全正常
- 钥匙：反复眩晕 + 偏头痛史（年轻女性多）

**⑦ 听神经瘤 Vestibular schwannoma**
- Stem：**缓慢进行性单侧 SNHL + 耳鸣**为主，**真性眩晕少**（长得慢、前庭代偿）
- 钥匙：单侧进行性聋 → **MRI 桥小脑角**；双侧 = NF2

**⑧ 中枢眩晕（小脑 / 脑干卒中）⚠️**
- Stem：**突发持续**眩晕 + **CNS 体征**（构音障碍 / 复视 / 面肢无力麻木 / 站不住）+ 垂直或变向眼震、凝视不能抑制
- 钥匙：卒中危险因素 / 新发头痛 / CNS 体征 → **立即影像排小脑卒中出血**；HINTS 提示中枢

## 四、Weber / Rinne 判读（伴聋型眩晕必备）⭐

> [!info] 两个音叉试验定位聋的性质
> 用于 Ménière / 迷路炎 / 外淋巴瘘等"伴聋"题，判断是**传导聋**还是**感音神经性聋 SNHL**。

```
Weber（音叉置前额/颅顶，问哪侧响）
  传导聋 (CHL) → 偏向【患耳】（患耳屏蔽环境噪音，骨导更显）
  感音聋 (SNHL)→ 偏向【健耳】

Rinne（音叉先骨导后气导，问哪个长/响）
  正常 / SNHL → 气导 > 骨导（AC > BC，"Rinne 阳性"）
  传导聋      → 骨导 > 气导（BC > AC，"Rinne 阴性"）
```

> [!example]- 实战（[[mistakes/uworld-mistakes_2026-06#^Q3449]]）
> Weber **偏左响** + Rinne **气导>骨导** → 左为健耳 → **右耳 SNHL** → 配合发作性眩晕 + 耳鸣 = **Ménière**。
> （BPPV 的听力是**正常**的，这是和 Ménière 的硬分水岭。）

| | Weber | Rinne |
|---|---|---|
| **正常** | 居中 | 气导>骨导（双侧）|
| **传导聋 CHL** | 偏**患**耳 | 骨导>气导（患侧）|
| **感音聋 SNHL** | 偏**健**耳 | 气导>骨导 |

> [!tip] 记不住？背「一个动作」推全部（堵耳哼歌锚点）
> 亲自做一次记死：手指堵住自己一只耳朵 + 哼歌 → **被堵那只耳更响**。被堵 = 人为造的**传导聋** → **传导聋 Weber 偏患耳**。剩下全推：
>
> ```
> 传导聋 Weber→患耳（堵耳哼歌亲测✓）
>     ↓ 反过来
> 感音聋 Weber→健耳（坏神经那侧听不清）
>
> Rinne 正常人 AC>BC（气导本就强 = 阳性）
>     ↓ 传导聋堵了患耳的气导路
> 传导聋患耳 BC>AC = 阴性（唯一翻转的情况）
>     ↓ 感音聋两条路按比例同降
> 感音聋 仍 AC>BC = 阳性（只是整体变小声）
> ```

> [!warning] 两个最坑点：命名反直觉 + Weber 单独定不了患侧
> - **Rinne 阳性 = 正常**（positive = proper），只有**传导聋**才把 Rinne 翻成**阴性**。
> - **Weber 单看分不清患侧**：偏右 = 右传导聋 **或** 左感音聋（两种相反情况给出同样的偏右）→ 必须配 Rinne 才能拆开。

> [!tip] 解题顺序：先 Rinne 后 Weber
> ```
> ① 先 Rinne 分型（每耳单测，确定性强）：
>    某耳 BC>AC（阴性）→ 该耳 = 传导聋，患侧已定
>    两耳都 AC>BC（阳性）→ 无传导聋（正常 or 感音聋）
> ② 再 Weber 定位 / 验证：
>    已抓到传导聋耳 → Weber 应偏【该耳】= 自洽确认
>    两耳 Rinne 都阳性 → Weber 偏健耳 → 偏离的【对侧】= 感音聋患耳
> ```
> 一句话：**先 Rinne 分型抓传导耳 → 再 Weber 验证（传导同向）或反向定位（感音偏健耳）**。

> [!tip] 传导聋 CHL 经典病因：耳硬化 otosclerosis（青年传导聋首位）
> Rinne 阴性（BC>AC）= 传导聋；**后天青年传导聋最经典 = 耳硬化**：
> - **人群**：早–中 30 岁、白人、**女性多**；**常染色体显性、不全外显**（家族史，如母亲 40 岁做过耳手术）。
> - **机制**：骨吸收/沉积失衡 → **镫骨固定 stapes fixation** → 声能传导受阻 = 传导聋。
> - **招牌**：进行性传导聋 + **吵闹环境反而听得更清**（Willis 错听 paracusis willisii，CHL 盖掉背景噪声）+ 鼓膜后**淡红色**（Schwartze 征）。
> - **听力图**：**低频传导聋**起病；可见 **2 kHz Carhart 切迹**（骨导假性下陷）；鼓膜正常、无积液（与中耳炎/胆脂瘤鉴别）。
> - **处理**：助听器 或 **镫骨手术 stapedectomy**。（UW Q16535 做对补充）

> [!tip] 耳闷 + 传导聋另一常见因：咽鼓管功能障碍 ETD
> Rinne 阴性传导聋若**鼓膜内陷 / 积液**（非镫骨固定）→ 多为**咽鼓管功能障碍 Eustachian tube dysfunction**：
> - **诱因**：近期 URI / 过敏性鼻炎 / 鼻窦炎（黏膜水肿堵咽鼓管口）、**气压变化**（飞机 / 潜水 / 电梯）。
> - **表现**：耳闷胀堵塞感、**自声增强 autophony**；鼓膜内陷、活动度↓；可继发**浆液性中耳炎**。
> - **检查**：鼓室图 tympanometry——**type C（负压）= ETD；type B（平坦）= 积液 / 穿孔；type A = 正常**。
> - **处理**：治诱因（鼻用激素 / 抗组胺 / 短期减充血剂）+ 自充气，多自限。
> - ⚠️ **成人单侧持续浆液性中耳炎 → 先查鼻咽排鼻咽癌 NPC**（肿瘤堵咽鼓管口；见 [[完整笔记/专题笔记/外科/外科_头颈鳞癌HNSCC总结]] §五）。

> [!info] "耳闷 / 耳痛但耳道鼓膜正常"鉴别同组（连读）
> - **传导聋 + 鼓膜内陷 / 积液 + URI/气压 → ETD**（本节）
> - **传导聋 + 鼓膜正常 + 青年女性家族史 + 吵处听得清 → 耳硬化**（本节）
> - **耳痛 + 耳道鼓膜全正常 + 咀嚼加重 / 烟酒颈淋巴结 → 牵涉性耳痛 TMD / HNSCC**（见 [[完整笔记/专题笔记/感染/感染_外耳道炎谱鉴别]] §四）

## 五、感音聋频率定位：低频 vs 高频（听力图判读）⭐

> [!info] 一个解剖原理串起所有感音聋
> 耳蜗音频拓扑 tonotopy：**底部 base 管高频、顶部 apex 管低频**；底部最脆弱（行波先冲击、代谢最高）→ 后天聋几乎都"高频先掉"。

```
耳蜗 cochlea 展开：
  底部 base（近卵圆窗）─ 基底膜窄硬 → 【高频】← 最脆弱（缺血/毒素/噪声/衰老先伤）
  顶部 apex（蜗孔）──── 基底膜宽软 → 【低频】
```

记忆钩：**底部 base 管高频且最脆 → 后天聋"高频先走"；唯独 Ménière 反着来（低频先掉 + 波动）。**

**高频感音聋（base 受损，最常见）**

| 病因 | 招牌 |
|---|---|
| 老年性聋 presbycusis | 双侧对称、缓慢、高频先掉；"听得见听不懂"，吵处尤甚（辅音 = 高频先丢） |
| 噪声性聋 NIHL | 听力图 **4000 Hz 切迹（noise notch）**、8kHz 回升；噪声/枪声/演唱会史 |
| 耳毒性药物 | 氨基糖苷类 aminoglycosides、顺铂 cisplatin（不可逆）；袢利尿剂 loop diuretics（呋塞米 furosemide）、大剂量阿司匹林 aspirin（多可逆） |

**低频感音聋（apex 受损，少见但特异）**

- **首位 = 早期 Ménière 梅尼埃病**：内淋巴积水 endolymphatic hydrops 撑胀蜗管，顶部低频区先受累 → **波动性低频感音聋**；进展后变全频平坦
- "低频感音聋 + 眩晕 / 耳鸣 / 耳胀" ≈ Ménière 指纹

> [!tip] 听力图形状 → 秒指病因
> - 低频下陷 + 波动 + 眩晕 → **Ménière**
> - 高频下坡 + 老年双侧 → **presbycusis**
> - 4kHz 切迹 + 噪声史 → **NIHL**
> - 高频 + 近期氨基糖苷 / 顺铂 / 袢利尿剂 → **药物耳毒性**（查能否停药逆转）
> - 读图：横轴频率 250→8000 Hz、纵轴 dB 越下越聋；高频聋曲线右下倾，低频聋右上扬

## 六、外周 vs 中枢眩晕（排卒中）

| 特征 | Peripheral（外周）| Central（中枢）|
|---|---|---|
| 眼震 nystagmus | 从不纯垂直、**单向**、注视可抑制、**疲劳性(<1min)**、**有潜伏期** | **任意方向(可纯垂直)**、变向/双向、**注视不抑制**、**不疲劳(>1min)**、无潜伏期 |
| 随头位 | 随体位变化、**Dix-Hallpike 可诱发**（BPPV）| **不随头位改变** |
| 伴随 | **听力/耳鸣**（Ménière/迷路炎）、行走多保留、无 CNS 征 | **严重姿势不稳**（不能独走）、**有 CNS 征**（复视/构音障碍/面肢麻木）|
| 病因 | BPPV / 前庭神经炎 vestibular neuritis / Ménière / 迷路炎 labyrinthitis | **卒中(小脑)** / MS / 偏头痛 |

> [!danger] 必须查中枢病因（怀疑小脑卒中/出血）
> **垂直/变向眼震、凝视不能抑制、伴其他 CNS 体征、HINTS 提示中枢** → 立即影像排卒中（尤其有血管危险因素的"孤立性持续眩晕"）。

### HINTS 三步床旁检查（Head Impulse / Nystagmus / Test of Skew）⭐⭐

> [!warning] 最大反直觉：三步"看起来正常/温和"反而 = 中枢危险
> **HINTS** 用于**急性持续性眩晕 + 自发眼震**（acute vestibular syndrome, AVS，即"孤立性持续眩晕"）判**外周 vs 中枢**；在有经验者手中**比发病早期 MRI-DWI（弥散加权成像 diffusion-weighted imaging）更敏感**（后循环小脑/脑干卒中发病 24–48h 内 DWI 可**假阴性**，别因"MRI 阴性"就放人）。
> - ⚠️ **反直觉核心**：**head impulse 甩头试验"异常（阳性）反而提示外周（良性，前庭神经炎）"**；三步全部"正常/温和"（HI 正常 + 眼震变向 + 有 skew）才指向**中枢卒中**。"检查异常 = 坏事"在这里不成立。
> - ⚠️ **适用范围**：只用于**持续数天的眩晕且当下有眼震**者；**BPPV / 发作性眩晕（发作间期正常）不适用**——那类用 Dix-Hallpike。

| 步骤 | 外周（良性，如前庭神经炎） | 中枢（危险，小脑/脑干卒中） |
|---|---|---|
| **HI: Head Impulse 甩头试验** | **阳性 = 异常**：快速把头甩向患侧，眼球跟不上目标 → 出现**纠正性扫视 corrective saccade**（外周前庭反射坏了） | **正常**：眼球稳稳跟住、无纠正扫视 = **危险信号** |
| **N: Nystagmus 眼震** | **单向水平**、快相方向固定、注视可抑制 | **方向变换 direction-changing** / 纯垂直 / 扭转 |
| **TS: Test of Skew 交替遮盖** | **无 skew**：交替遮盖去除时眼球无垂直纠正 | **有 skew deviation 垂直错位**（提示脑干） |

```
口诀 INFARCT —— 出现任一"中枢型"即高危，查影像走卒中流程：
  Impulse Normal          → HI 正常（甩头不出纠正扫视）
  Fast-phase Alternating  → 眼震变向（direction-changing）
  Refixation on Cover Test→ 遮盖去除见垂直纠正（skew 阳性）

判读：
  三项全"外周型"（HI 阳性 + 单向眼震 + 无 skew） → 外周（前庭神经炎），可安心对症
  任一"中枢型"（HI 正常 / 眼震变向 / 有 skew）   → 中枢卒中，立即影像 + 卒中流程
       （MRI 磁共振成像 / MRA 磁共振血管成像 magnetic resonance angiography）
```

> [!tip] 一句话锁定
> 持续大晕 + 有眼震：**甩头有纠正扫视 = 外周（好）；甩头正常 + 眼震变向 + 眼位垂直错开 = 中枢（坏）**。别被"孤立性眩晕、神经查体看着还行"骗过去。

## 七、治疗速览

| 病 | 一线 | 备注 |
|---|---|---|
| **BPPV** | **Epley 手法**（复位）| 急性可美克洛嗪 meclizine |
| **Ménière** | **限盐 + 限咖啡因 + 限酒**；急性美克洛嗪 | 二线利尿剂（氢氯噻嗪 hydrochlorothiazide）/ 倍他司汀 betahistine；**绝不增盐** |
| **外淋巴瘘** | 避免增内耳压活动 + **ENT 转诊** | 必要时手术修补 |
| **前庭神经炎/迷路炎** | 对症（美克洛嗪）+ treat the bug | 迷路炎带聋 |
| **前庭性偏头痛** | 偏头痛预防/急性方案 | — |
| **中枢（卒中）** | **影像 MRI/MRA + 卒中流程** | **不用前庭抑制剂**；危险因素 / CNS 体征 / 垂直眼震时 |

> [!warning] 治疗最易串的两点（Q8 错点）
> - **BPPV = 耳石病 → Epley 复位**，几乎从不手术；美克洛嗪 meclizine **只短期对症**（长期反而抑制中枢代偿）。
> - **"手术修补"是外淋巴瘘的兜底，不是 BPPV 的** —— 两个"机械修复"别串：一个复位石头、一个补窗膜。

## 八、Oscillopsia（振动幻视 / 视振荡）鉴别 ⭐

> [!info] 什么是 oscillopsia
> **Oscillopsia = 静止的外界看起来在"晃动 / 跳动 / 来回摆"的错觉**。本质是**眼睛没能把外界像稳定钉在视网膜上**（image not stabilized on retina），几乎都指向两条路之一：
> - **① VOR（vestibulo-ocular reflex，前庭眼反射）失灵** → 头一动、眼睛跟不住（双侧前庭病最典型）
> - **② 异常眼球运动（acquired nystagmus，获得性眼震）** → 眼睛自己在动

> [!danger] 一句话金律：先天不晃、后天才晃
> - **Congenital nystagmus（先天性眼震）→ 几乎无 oscillopsia**（大脑从小已适应、有中枢抑制）。
> - **Acquired（获得性）异常 → 才有 oscillopsia**。
> - ⚠️ 所以病人**主诉"看东西在晃" = 这是新发 / 获得性病变**——见到 oscillopsia 别往先天走（先天眼震=常见干扰项）。

### 四大来源（USMLE 高频）

| 来源 | 招牌 stem | 招牌钥匙 | 下一步 |
|---|---|---|---|
| **① 双侧前庭病 = 氨基糖苷耳毒性** | 脓毒症住院用 **庆大霉素 gentamicin** 数天后，**走路 / 转头时视野跳、看不清路牌、读字困难；坐着不动正常**，黑暗更糟 | **动态视力 dynamic visual acuity 下降** + **双侧甩头试验 head impulse 阳性**；别名 Dandy syndrome | 停氨基糖苷；前庭康复（不可逆多见）|
| **② Downbeat nystagmus 下跳性眼震** | 持续 oscillopsia + 下跳眼震（向下注视加重）| 颅颈交界病变 → **Arnold-Chiari I 畸形** / 小脑变性（也见于**锂 lithium / 苯妥英 phenytoin 中毒**）| **MRI 颅颈交界** 排 Chiari |
| **③ INO 核间性眼肌麻痹** | 年轻女性，患侧内收不能 + **对侧外展眼眼震** → 眼震致 oscillopsia | 想 **MS（多发性硬化）**；双侧 INO 更提示 MS | MRI 脑 + 脊髓 |
| **④ Superior oblique myokymia 上斜肌微颤** | **单眼**、短暂**垂直 / 旋转**性视振荡、数秒发作 | 单眼、阵发、旋转 | 卡马西平 carbamazepine / 手术 |

> [!tip] 做题反射
> - **"走路时视野跳 + 近期用过氨基糖苷" → 双侧前庭病**（查甩头试验、动态视力），停药。
> - **"下跳性眼震（downbeat）" → 立刻 MRI 颅颈交界排 Chiari**（别当良性）。
> - **"先天性眼震却问 oscillopsia" → 干扰项**，先天几乎不晃。
> - Oscillopsia 与本笔记 §六 的**甩头试验（head impulse）**同源：氨基糖苷双侧前庭病 = 甩头**双侧**阳性（与前庭神经炎单侧阳性、中枢卒中阴性对照）。

> [!info] 缩写说明
> 本笔记中 **ICD = Impulse Control Disorder（冲动控制障碍）** — DA 激动剂诱发的病理性赌博 / 购物 / 性欲亢进 / 暴食。**不是**心内 ICD（除颤器）也不是 ICD-10 疾病分类。

## 一、核心逻辑：多巴胺双刃剑

PD 本质 = 黑质多巴胺能神经元退化 → DA 不足 → 运动症状。所有 PD 用药围绕"补 DA 信号"展开，但补多了就出精神 / 冲动问题。

```
多巴胺双刃剑
═══════════════════════════════════════════
多巴胺 ↑ (levodopa / DA 激动剂 / MAO-B / COMT 抑制剂)
    └─→ 救运动症状 ✓
        但 → psychosis（视幻觉）/ ICD / 异动症 ✗

多巴胺 ↓ (haloperidol / risperidone / metoclopramide)
    └─→ 抗幻觉 / 止吐 ✓
        但 → 恶化 parkinsonism（震颤、僵硬加重）✗

         ←────────── 走钢丝 ──────────→
   多巴胺太少 = 瘫        多巴胺太多 = 疯
```

> [!tip] 看到 PD 题先问
> **这次是 DA 不够（运动差）还是 DA 太多（幻觉 / 冲动）？** 方向定了选项就清晰。

## 二、用药词根速记

| 词根 | 类别 | 代表药 | 关键点 |
|---|---|---|---|
| **-pexole / -nirole / -gotine** | DA 激动剂 | Pramipexole / Ropinirole / Rotigotine | 直接激动 D2/D3；**最易 psychosis + ICD** |
| **-giline** | MAO-B 抑制 | Selegiline / Rasagiline | 抑 DA 降解；**联用 SSRI / 哌替啶 → 5-HT 综合征** |
| **-capone** | COMT 抑制 | Entacapone / Tolcapone | 延长 levodopa；**Tolcapone 肝毒** |
| **(carbidopa-)levodopa** | DA 前体 | Carbidopa-Levodopa | 金标准、最有效；carbidopa 防外周分解 |
| **Amantadine** | NMDA 拮抗 | Amantadine | **治 dyskinesia**；**livedo reticularis** ★ |
| **Benz- / Trihexy-** | 抗胆碱能 | Benztropine / Trihexyphenidyl | **主治震颤**；老人慎用（认知）|

> [!warning] 三个 "-one" 别混
> - **-cap·one**（Entacapone）= COMT 抑制 = PD 药
> - **-aps·one**（Dapsone）= 抗生素 / 麻风 / PCP 预防 = **不是** PD 药
> - 差一个字母，听结尾

## 三、各类药 — 适应人群 / 适应症 / 副作用

### 3.1 Levodopa（+ Carbidopa）— 金标准

| 维度 | 内容 |
|---|---|
| 适应人群 | **老年 / 症状严重者首选**（最有效）|
| 适应症 | PD 全部运动症状（尤其 bradykinesia / rigidity）|
| 副作用 | **运动并发症**（长期）：① dyskinesia ② **wearing-off** ③ **on-off 现象** + 直立性低血压 + 恶心；**较少诱发 psychosis**（vs DA 激动剂）|
| 关键点 | Carbidopa = 外周脱羧酶抑制剂 → 减外周副作用 + 增入脑量 |

> [!warning] Levodopa 长期困境
> 用得越久，运动并发症越多 → 年轻病人常先用 DA 激动剂"省着 levodopa"，把它留到后期

### 3.2 多巴胺激动剂（Pramipexole / Ropinirole / Rotigotine）

| 维度 | 内容 |
|---|---|
| 适应人群 | **年轻（<60-65）初始单药**，延缓 levodopa 启用；Rotigotine 透皮贴适合吞咽困难者 |
| 适应症 | PD 早期运动症状 / **不宁腿综合征 (RLS)** / Levodopa 辅助 |
| 副作用 ★★★ | ① **ICD（冲动控制障碍）** ② **psychosis / 视幻觉**（最易）③ **突发睡眠发作** ④ 直立性低血压 ⑤ 外周水肿 ⑥ 恶心 |
| 关键点 | 比 levodopa **更易诱发精神 / 冲动副作用**，但**更少运动并发症** |

### 3.3 MAO-B 抑制剂（Selegiline / Rasagiline）

| 维度 | 内容 |
|---|---|
| 适应人群 | 早期、症状轻患者单药；或 levodopa 辅助延长疗效 |
| 适应症 | 早期 PD 单药 / 辅助治疗 wearing-off |
| 副作用 ★ | **5-HT 综合征**（联用 SSRI/SNRI/TCA/哌替啶/右美沙芬）/ 失眠（selegiline 代谢产生苯丙胺类）|
| 关键点 | 最重要 = **药物相互作用** → 联用 5-HT 能药 → SS |

### 3.4 COMT 抑制剂（Entacapone / Tolcapone）

| 维度 | 内容 |
|---|---|
| 适应人群 | 已用 levodopa 但 **wearing-off**（**必须配 levodopa**，单用无效）|
| 适应症 | 延长 levodopa 作用，减 wearing-off |
| 副作用 | 加重 levodopa 多巴胺能副作用 / 尿汗变橙 / **Tolcapone → 肝毒**（需监测）|
| 关键点 | Entacapone 更常用（Tolcapone 肝毒）|

### 3.5 抗胆碱能（Benztropine / Trihexyphenidyl）

| 维度 | 内容 |
|---|---|
| 适应人群 | **年轻 + 震颤为主**（tremor-predominant）；**老人禁用** |
| 适应症 | 主治**震颤**（对运动迟缓 / 僵硬效果差）/ 抗精神病药致急性肌张力障碍 |
| 副作用 ★ | **抗胆碱能毒性**：口干 / 便秘 / 尿潴留 / 视模糊 / **认知损害 / 谵妄**（老人尤甚）/ 心动过速 |
| 关键点 | "**老人别用**"高频陷阱 — 抗胆碱能恶化老年认知 |

### 3.6 Amantadine（NMDA 拮抗）

| 维度 | 内容 |
|---|---|
| 适应人群 | **Levodopa 诱发的 dyskinesia 患者** |
| 适应症 | **抗异动症首选** / 轻度改善 PD 运动症状 |
| 副作用 ★ | **Livedo reticularis（网状青斑）** ⭐ 特征 / 下肢水肿 / 幻觉（肾排泄需减量）/ 失眠 |
| 关键点 | 见 PD + livedo reticularis → 立刻想 Amantadine |

## 四、选药决策树

```
PD 初始药物选择
═══════════════════════════════════════════════
                确诊 PD，需启动治疗
                        │
       ┌────────────────┴───────────────┐
       │                                │
  症状轻微？                         症状明显影响生活
       │                                │
       ▼                                ▼
 可观察 / MAO-B           ┌──────────年龄?──────────┐
 抑制剂                    │                         │
                     年轻(<60-65)              老年(>65)/严重
                          │                         │
                          ▼                         ▼
                  DA 激动剂为主             Levodopa 为主
                 (省着 levodopa,           (最有效，老人对
                  延缓远期并发症)            ICD/低血压更需避开 DA)
                          │
                ┌─────────┴─────────┐
            震颤为主?            运动迟缓/僵硬为主?
                │                     │
                ▼                     ▼
      年轻可加抗胆碱能        DA 激动剂 / Levodopa
      (benztropine)          (抗胆碱能对此类无效)
═══════════════════════════════════════════════

后期并发症处理：
  • wearing-off  → 加 COMT 抑制剂 (entacapone) 或 MAO-B
  • dyskinesia   → 加 Amantadine
  • psychosis    → 见 §五 (先减药 → 低 D2 抗精神病药)
```

> [!tip] 选药三句话
> 1. **年轻 → 先 DA 激动剂**（把 levodopa 留到后面）
> 2. **老年 / 重症 → 先 levodopa**（最有效；老人对 DA 的 ICD / 精神副作用更敏感）
> 3. **震颤为主 + 年轻 → 可加抗胆碱能**（老人禁，认知风险）

## 五、PD Psychosis 管理流程（详细 5 步见 psych 衍生）

PD psychosis（视幻觉为主、可有妄想 / 错觉）= 治疗副产物 / 疾病进展，影响达 40% PD 患者。

```
PD Psychosis 5 步管理
═══════════════════════════════════════════
1. 排除 infection / delirium / dementia
   (有 → 先治这些)
        │ 无
        ▼
2. 症状轻微？→ 仅监测
        │ 否
        ▼
3. 减/停【非 PD 精神活性药】
   (抗胆碱能 / 镇静 / TCA / BZD)
        │ 无效
        ▼
4. 减【PD 药】，从效力最低的开始
   (顺序: 抗胆碱能→DA激动剂→...→levodopa 最后减)
        │ 仍难治
        ▼
5. 上【低/零 D2 抗精神病药】
   (Pimavanserin / Quetiapine / Clozapine)
```

> [!tip] 减药顺序口诀
> **"先减杂药，再减帕药，帕药从弱减起。"**
> Levodopa 最有效、最不易诱发 psychosis → 最后减

详细 5 步阶梯（含 quetiapine vs pimavanserin vs clozapine 选择 + DLB vs PD ψ 反 pattern）→ [[完整笔记/专题笔记/psych/psych_PD_psychosis_5步阶梯]]

### 5.1 第 3 步深挖：4 类"非 PD 精神活性药"为什么诱发谵妄

> [!info] 两条主线
> - **主线 A：抗胆碱能负荷↑** → 中枢 M1 → ACh ↓ → 谵妄 / 幻觉
> - **主线 B：GABA 能镇静** → 意识 / 认知抑制

| 药类 | 主线 | 核心机制 | PD/老人风险点 |
|---|---|---|---|
| **抗胆碱能** (benztropine, trihexyphenidyl) | A | 阻 M1 → ACh 信号不足 | PD 本身 ACh 储备低（尤其 PDD/DLB）|
| **TCA** (amitriptyline) | A | **强抗胆碱能活性** | 与抗胆碱能同主线 vs SSRI 抗胆碱能低风险 |
| **抗焦虑 BZD** (lorazepam, alprazolam) | B | 增强 GABA-A → 意识混浊 | 用药中 / 突停 GABA 反跳戒断谵妄 |
| **镇静催眠** (zolpidem; diphenhydramine) | A 和/或 B | Z-drug 走 GABA；diphenhydramine = 抗组胺 + 强中枢抗胆碱能 | diphenhydramine 老人易谵妄 (Beers 慎用) |

> [!tip] Q16164 串起来
> 题干"减 sertraline (SSRI) 收益小、减 carbidopa-levodopa 才对" — **SSRI 抗胆碱能负荷低，不是元凶**；高负荷的是 TCA / 抗胆碱能药

> [!warning] PD 内源 psychosis vs 药源性谵妄
> - **PD 内源 ψ**（DA 药 + 病情进展）→ 视幻觉为主、**意识清醒**（早期保留洞察）
> - **药源性谵妄**（上 4 类）→ **意识波动、注意力障碍为核心**
> 两者常共存 → 先撤可逆外因（第 3 步），再动 PD 药本身（第 4 步）

## 六、PD 可用 / 禁用抗精神病药（D2 强弱谱）

核心原则：**D2 拮抗越弱 → 越安全用于 PD**

| 药物 | D2 拮抗 | PD 能否用 | 备注 |
|---|---|---|---|
| **Pimavanserin** | 零（纯 5-HT2A）| ✅ 首选 | 完全不碰运动系统 |
| **Quetiapine** | 极弱 | ✅ | 运动影响最小 |
| **Clozapine** | 弱 | ✅ | 有效，需查血象（粒缺）|
| **Risperidone**（-peridone）| 中高 | ❌ 避免 | 恶化运动 |
| **Haloperidol**（-peridol）| 强 | ❌ 禁用 | 严重恶化 parkinsonism |

```
安全谱（PD 最安全 → 最危险）
Pimavanserin → Quetiapine → Clozapine → ... → Risperidone → Haloperidol
   零 D2         极弱          弱                  中高          强 D2
   └──────── PD 安全区 ────────┘        └──── PD 禁区 ────┘
```

> [!danger] 3 类绝对不能在 PD 用的"运动毒药"
> - **Haloperidol**（-peridol）
> - **Risperidone**（-peridone）
> - **Metoclopramide**（甲氧氯普胺）— 止吐 / 促动力，**中枢 D2 拮抗，埋雷高频**；PD 止吐改用 **ondansetron**
> 记：**-peridol / -peridone = 强 D2 = PD 别碰**

## 七、High-Yield 高频陷阱

| 陷阱 | 错误惯性 | 正确思路 |
|---|---|---|
| PD 出现幻觉 | "幻觉就上抗精神病药" | 先减触发药；要上药只能上低/零 D2 |
| PD 出问题 | "加抗帕金森药对症" | 先分清是运动问题还是精神问题 |
| 减药对象 | "减任何精神活性药都行" | 减最可能元凶（DA / 抗胆碱能）；SSRI 抗胆碱能低非元凶 |
| PD 患者止吐 | "用 metoclopramide" | 它是 D2 拮抗会恶化 PD；改 **ondansetron** |
| -giline 联用 | 随意配抗抑郁药 | MAO-B + SSRI / 哌替啶 → 5-HT 综合征 |
| capone vs apsone | 看成同类 | -capone = COMT (PD)；-apsone = Dapsone (抗生素)|
| 老人用抗胆碱能 | "震颤就给 benztropine" | 老人禁（认知 / 谵妄）；只用于年轻 + 震颤为主 |
| PD 患者突然狂赌博 | "新发精神病" | DA 激动剂诱发的 **ICD** |
| PD + livedo reticularis | 找血管病因 | **Amantadine** 的特征性副作用 |

## 八、Memory Hook

> [!success] 一句话
> **"PD 看见鬼，先减药别加药。"**
> 类比：多巴胺能药 = **油门**。运动症状已够用 → 松一点油门即可；再猛踩 → 撞墙（psychosis）

> [!tip] 完整口诀
> - **用药选择**：PD 用药躲着 D2 走（抗精神病药选低/零 D2）
> - **减药顺序**：先减杂药，再减帕药，帕药从弱减起
> - **双刃剑**：多巴胺多了疯（psychosis/ICD），少了瘫（parkinsonism）
> - **年龄选药**：年轻先 DA，老年先 levodopa
> - **副作用反推**：赌博→DA；青斑→Amantadine；5-HT 综合征→MAO-B；肝毒→Tolcapone；老人谵妄→抗胆碱能
> - **谵妄机制**：抗胆碱能/TCA → ACh↓（主线 A）；BZD/镇静 → GABA↑（主线 B）；SSRI 负荷低 → 非元凶

> [!warning] 反向记忆
> 看到 **-peridol / -peridone / metoclopramide** 在 PD 题里出现 → 立刻警觉"强 D2，陷阱选项"

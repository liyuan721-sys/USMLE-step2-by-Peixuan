# A. 脊髓解剖基础

## A.1 脊髓的"功能分区"

```
脊髓横切面（颈髓 C7 水平为例）：

       后角（posterior horn）= 感觉接收
           ↓
     ┌─────────────────────────┐
     │                         │
     │   ╔═══════════════╗     │ ← Posterior column
     │   ║Dorsal column ║     │   （后柱）= 振动 / 本体 / 精细触觉
     │   ║   gracilis    ║     │   同侧上行
     │   ║   cuneatus    ║     │
     │   ╚═══════════════╝     │
     │   ┌─────────────┐       │ ← Central canal（中央管）
     │   │  中央管 +    │       │   + 周围灰质
     │   │  灰质 + 前白 │       │
     │   │  连合        │       │ ← Anterior white commissure
     │   └─────────────┘       │   （前白连合）= 痛温交叉处 ⭐
     │                         │
     │   ╔═══════════════╗     │ ← Lateral CST（外侧皮质脊髓束）
     │   ║ Lateral CST  ║     │   = 运动（已在 medulla 交叉）
     │   ║  + Spino-    ║     │
     │   ║  thalamic    ║     │ ← Spinothalamic（脊髓丘脑束）
     │   ║              ║     │   = 痛温（对侧，已在脊髓内交叉）
     │   ╚═══════════════╝     │
     │                         │
     │       前角（anterior     │ ← Anterior horn = LMN
     │       horn）= 运动输出   │
     │                         │
     └─────────────────────────┘
                ↓
            前根 → 肌肉
```

## A.2 三大束的关键解剖位置（USMLE 必背）

| 束 | 位置 | 功能 | 损伤后 |
|---|---|---|---|
| **Dorsal column** | **后柱**（中央偏内 = gracilis 腿，外 = cuneatus 手）| **振动 / 本体 / 精细触觉** | **同侧**↓ |
| **Lateral CST** | **外侧**（手部纤维**近中央**，脚部纤维**远中央**）| **运动**（已在 medulla 交叉）| **同侧**瘫 |
| **Spinothalamic** | **外侧**（前束 = 触觉，外侧 = 痛温）| **痛温觉**（脊髓内 1-2 节段已交叉）| **对侧**↓ |
| **Anterior horn** | **前角灰质** | LMN（运动输出）| **同节段**LMN 征 |
| **Lateral horn (T1-L2)** | **外侧角灰质** | 交感神经一级 | Horner（T1-T2）|

⭐ **核心解剖记忆**：
- 后面（后柱）= 同侧振动
- 外侧上下（CST + Spinothalamic）= 一个同侧瘫 + 一个对侧痛温
- 前面（前角）= 同节段 LMN
- 中央（前白连合）= 痛温交叉处 ⭐ Syringomyelia 第一站

## A.3 脊髓血供（必懂，解释 ACS）

```
脊髓血供 = 3 条动脉

┌─────────────────────────────────────────┐
│  Posterior Spinal Arteries (× 2)         │
│  ──────────────────────────              │
│  供血后 1/3：                        │
│  ✅ Dorsal column                       │
│  ✅ 部分后角                             │
│                                         │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─        │
│  ← 血供分界线（"watershed"）             │
│                                         │
│  Anterior Spinal Artery (× 1) ⭐         │
│  ──────────────────────────              │
│  供血前 2/3：                        │
│  - Spinothalamic                        │
│  - Lateral CST                          │
│  - 前角（LMN）                          │
│  - 外侧角（自主）                        │
└─────────────────────────────────────────┘
```

### Adamkiewicz 动脉（必考）⭐

- **大根动脉**，是脊髓下 2/3 主要供血来源
- 起源最常 **T9-T12 左侧**（~70%）
- AAA 术中**结扎 Adamkiewicz** → ACS

### Watershed Zone（必考）⭐

- 脊髓**血供最薄弱的区域** = **T4-T8**
- 是脊髓**两个供血系统的"接缝处"**：
  - 上方：颈段椎动脉血供衰减到此
  - 下方：Adamkiewicz 血供也到此减弱
- 类比脑 watershed（ACA/MCA 边界）

### Adamkiewicz vs Watershed 终极对比 ⭐⭐⭐

> [!warning] 高频混淆点 — 一条动脉 vs 一个区域

| 维度 | **Adamkiewicz Artery** | **Watershed Zone** |
|---|---|---|
| **本质** | **一条具体的动脉**（解剖名词）| **一个血供概念**（区域）|
| **中文** | Adamkiewicz 动脉 / 大根动脉 | 分水岭区 |
| **位置** | **起源 T9-T12 左侧** (~70%)| **T4-T8**（脊髓中段）|
| **角色** | 供血脊髓**下 2/3** | 脊髓**血供最薄弱区** |
| **梗死机制** | **直接结扎 / 损伤这条动脉** | **全身低灌注** → 边界区先死 |
| **典型病因** | **AAA 修复术结扎 Adamkiewicz** ⭐ | **严重低血压**（心脏骤停 / 休克 / sepsis）⭐ |
| **梗死部位** | T9-T12 以下脊髓 | **T4-T8 脊髓中段** |
| **临床** | 都表现为 **ACS**（前 2/3 梗死）| 都表现为 **ACS** |
| **类比记忆** | "**水管被剪断**" 🚰✂️ | "**远端花园浇不到水**" 🌱💦 |

### Adamkiewicz vs Watershed 完整临床鉴别表 ⭐⭐⭐

> [!success] 这张表把感觉平面和 USMLE stem 描述直接对应起来

| 特征 | **Adamkiewicz 损伤** (T9-T12) | **Watershed 梗死** (T4-T8) |
|---|---|---|
| **典型病因** | AAA 修复 / 主动脉手术 / 主动脉夹层 | **持续休克 / 心脏骤停后复苏** |
| **机制** | 直接动脉损伤 / 结扎 | 全身低灌注 |
| **损伤平面** | **Thoracolumbar**（约 T10 以下）| **中胸段**（T4-T8）|
| **感觉平面**（皮节定位）| **肚脐 (T10) / 腹股沟 (L1) 以下** ⭐ | **乳头 (T4) / 剑突 (T6) 以下** ⭐ |
| **上肢** | ✅ **正常** | ✅ **正常** |
| **下肢运动** | ❌ **瘫痪** | ❌ **瘫痪** |
| **痛温觉** | ❌ **丧失**（病变以下）| ❌ **丧失**（病变以下）|
| **本体感觉 / 振动** | ✅ **保留** ⭐ | ✅ **保留**（或轻度受累）⭐ |
| **大小便** | ❌ **失禁 / 潴留** | ❌ **失禁 / 潴留** |

⭐ **两者共同特征 = ASA 综合征（Anterior Spinal Artery Syndrome）**：
- 运动 + 痛温觉**受累**
- **本体感觉保留**
- 大小便障碍
- 上肢正常（病变在胸髓以下）

⭐ **唯一鉴别关键 = 看感觉平面 + 触发病因**：
- **病因 AAA 术后** + **感觉平面 T10/L1 以下** = **Adamkiewicz** ⭐
- **病因休克 / 心脏骤停** + **感觉平面 T4/T6 以下** = **Watershed** ⭐

### USMLE 皮节定位速查（鉴别钥匙）

| 皮节标志 | 节段 | 提示 |
|---|---|---|
| **乳头水平** | **T4** | Watershed 上界（最薄弱）|
| **剑突** | **T6** | Watershed 中段 |
| **脐** | **T10** | Adamkiewicz 起源附近 |
| **腹股沟** | **L1** | Adamkiewicz 供血远端 |

⭐ Stem 写"sensory loss below the nipples / xiphoid" = T4-T6 = **Watershed**
⭐ Stem 写"sensory loss below the umbilicus / inguinal" = T10-L1 = **Adamkiewicz**

### 两种 ACS 病因机制对比

#### 机制 1：Adamkiewicz 损伤型 ACS
```
AAA 修复手术 → 结扎 / 损伤 Adamkiewicz 动脉
   ↓
脊髓下 2/3 灌注突然失败
   ↓
ACS（T9-T12 以下双下肢瘫 + 痛温↓ + 振动留）
   ↓
感觉平面：脐 (T10) / 腹股沟 (L1) 以下
   ↓
关键：结构性血管损伤——不论血压多高都梗死
```

#### 机制 2：Watershed 型 ACS
```
严重低灌注事件：
- 心脏骤停 + CPR
- 大失血 / 创伤
- 严重 sepsis 休克
- 心源性休克
   ↓
全身灌注↓
   ↓
Watershed (T4-T8) 侧支血供最薄弱
   ↓
此区域先梗死
   ↓
ACS（T4-T8 以下双下肢瘫 + 痛温↓ + 振动留）
   ↓
感觉平面：乳头 (T4) / 剑突 (T6) 以下
   ↓
关键：血流动力学梗死——动脉树完整但血压不够维持薄弱区
```

### 关系：Adamkiewicz 决定了 Watershed 的位置

```
T1 ─────────────────────────────────────
T2                                    │
T3                                    │  上方颈血供衰减到此
T4 ─── ↓
T5  WATERSHED Zone（薄弱）
T6  T4-T8 = 两边都来不到
T7                                    │
T8 ─── ↑
T9                                    │  Adamkiewicz 起源
T10 → Adamkiewicz Artery (T9-T12 左)  │  供血下方
T11                                    │
T12 ─────────────────────────────────────
```

⭐ **Watershed 之所以在 T4-T8** = 因为 Adamkiewicz 从 T9-T12 起源，**上方供血就开始衰减**，最远端（T4-T8）= 最薄弱

### 临床意义

| 类型 | 预防 | 常见情境 | 梗死位置 |
|---|---|---|---|
| **Adamkiewicz 损伤** | 术中监测脊髓功能（SSEP / MEP）/ 维持灌注 | AAA 修复 / 主动脉夹层 / 主动脉缩窄 | T9-T12 以下 |
| **Watershed 低灌注** | 避免严重低血压 / **维持 MAP > 65** | 心脏骤停 / 大失血 / sepsis 休克 | **T4-T8** |

### USMLE 钩子（一句话）

> [!success] 
> - **"AAA 术后" → 想 Adamkiewicz**（动脉被结扎）
> - **"心脏骤停 / 休克后" → 想 Watershed**（低灌注）
> 
> 两者都引起 ACS，但**触发机制完全不同**

## A.4 灰质 vs 白质

| 部分 | 功能 | 损伤表现 |
|---|---|---|
| **灰质**（前角 + 后角 + 中央）| 神经元胞体 | **LMN 征 + 节段性感觉障碍** |
| **白质**（束 / 通路）| 神经纤维 | **UMN 征 + 平面以下感觉障碍** |

⭐ **临床推理**：节段性 vs 平面以下 = 灰质病变 vs 白质病变

# B. 三大束的交叉时机与临床意义 ⭐⭐⭐ 必懂

## B.1 三大束在哪里交叉

```
┌─────────────────────────────────────────────────────┐
│  Lateral CST（运动）                                 │
│      ↓                                              │
│   皮层运动区                                         │
│      ↓                                              │
│   ★ 在 延髓 (medulla) 交叉 ★                    │
│      ↓                                              │
│   进入脊髓后已在对侧                             │
│      ↓                                              │
│   脊髓损伤 = 同侧瘫（已经交叉过）⭐              │
├─────────────────────────────────────────────────────┤
│  Dorsal column（振动 / 本体 / 精细触觉）             │
│      ↓                                              │
│   感受器 → DRG（背根神经节）                         │
│      ↓                                              │
│   进入脊髓后同侧上行 ↑                          │
│      ↓                                              │
│   ★ 在 延髓 (medulla) 交叉 ★                    │
│   （在 nucleus gracilis / cuneatus 换神经元后交叉）  │
│      ↓                                              │
│   脊髓损伤 = 同侧↓（还没交叉）⭐                 │
├─────────────────────────────────────────────────────┤
│  Spinothalamic（痛温觉）                            │
│      ↓                                              │
│   感受器 → DRG                                      │
│      ↓                                              │
│   进入脊髓 + 1-2 节段内立刻交叉 ⭐               │
│      ↓                                              │
│   ★ 在 脊髓内（前白连合处）交叉 ★               │
│      ↓                                              │
│   交叉后 → 对侧 spinothalamic tract 上行             │
│      ↓                                              │
│   脊髓损伤 = 对侧↓（已经在脊髓内交叉了）⭐       │
└─────────────────────────────────────────────────────┘
```

## B.2 一句话口诀

> [!success] 
> **"两个在上面交叉，一个在脊髓里就交叉"**
> 
> - **CST**（运动）→ medulla 交叉 → 脊髓损伤**同侧**瘫
> - **Dorsal column**（振动）→ medulla 交叉 → 脊髓损伤**同侧**↓
> - **Spinothalamic**（痛温）→ 脊髓内 1-2 节段交叉 → 脊髓损伤**对侧**↓ ⭐

## B.3 交叉时机的临床意义

### 应用 1：解释 Brown-Séquard

```
单侧脊髓半切（如 T6 右侧）
        ↓
1. 同侧 CST 断 → 右侧 T6 以下同侧运动↓
   （CST 已在 medulla 交叉，进入脊髓后已是对侧的"过路客"）
   
2. 同侧 Dorsal column 断 → 右侧 T6 以下同侧振动 / 本体↓
   （Dorsal column 在脊髓内同侧上行，还没交叉）
   
3. 对侧 Spinothalamic 损伤 → 左侧 T6 以下对侧痛温↓
   （Spinothalamic 在脊髓内已交叉，损伤的是对侧来的"过路客"）
```

⭐ Brown-Séquard 的"同侧运动 + 同侧振动 + 对侧痛温" → **完全来自三大束的交叉时机差异**

### 应用 2：解释 Syringomyelia "双侧痛温丢失"

```
中央空洞扩张
        ↓
前白连合被切断
        ↓
两侧的 spinothalamic 交叉纤维都断了
（因为左右两侧的痛温纤维都要通过前白连合交叉）
        ↓
双侧痛温↓
        ↓
但 dorsal column 在后柱（没碰到）→ 振动 / 本体保留
但 CST 在外侧（没碰到）→ 早期运动保留
        ↓
= "分离性感觉障碍 + 节段性双侧分布"
```

⭐ Syringomyelia 的"双侧披肩样痛温↓" → **来自前白连合切断左右双侧的交叉纤维**

### 应用 3：解释 ACS "前 2/3 梗死，dorsal column 保留"

```
前脊髓动脉梗死
        ↓
前 2/3 全废（CST + spinothalamic + 前角）
        ↓
临床：双侧瘫 + 双侧痛温↓
        ↓
但 dorsal column 在后柱（后脊髓动脉供血）→ 保留
        ↓
= "运动 + 痛温丢，振动 + 本体保留"
```

## C.4 颈髓节段为什么和颈椎不对应

```
脊髓节段 ≠ 椎体水平
        ↓
胚胎期脊髓比椎骨长得慢
        ↓
脊髓在 L1-L2 椎体水平就结束（成人）
        ↓
所以：
- 颈椎 C7 椎体水平 = 颈髓 C7-C8 节段
- 胸椎 T6 椎体水平 = 胸髓 T7-T8 节段
- 腰椎 L1 椎体水平 = 脊髓 conus medullaris
- L2 椎体以下 = 马尾神经（脊髓已结束）
```

⭐ **临床意义**：
- **L1-L2 以下椎间盘突出 ≠ 脊髓损伤**，是**马尾损伤**（Cauda equina syndrome）
- **L1-L2 平面损伤 = Conus medullaris syndrome**

# D. 6 大脊髓综合征详解

## D.1 Central Cord Syndrome (CCS) ⭐ 老人创伤后

### 经典 stem
"**75 岁男** + 从楼梯上摔下来**撞下巴**（过伸伤）+ 立刻出现**双上肢瘫痪 > 双下肢瘫痪** + 颈痛 + 双手痛温觉↓"
![](https://www.uworld.com/media/L17579.jpg)
### 解剖损伤
```
颈椎过伸伤 + 颈椎管狭窄
        ↓
颈髓中央灰质 + 周围水肿 / 挫伤 / 出血
        ↓
累及：
- Lateral CST 中央部分（手部纤维）⭐
- 中央灰质 + 前白连合
- 部分前角细胞
        ↓
临床：
- 上肢瘫 > 下肢瘫（CST 手部纤维近中央先废）⭐ 招牌
- 披肩样痛温↓（前白连合切断）— 次要
- 上肢 LMN 萎缩（前角受累）— 慢性表现
```

### 为什么"Arms > Legs"

```
Lateral CST 内部躯体定位：

        ┌─────────────────┐
        │     外侧 CST     │
        │   ┌─────────┐   │
        │   │ 脚部纤维 │   │ ← 离中央远
        │   │   ↓     │   │
        │   │ 躯干纤维 │   │
        │   │   ↓     │   │
        │   │ 上肢纤维 │   │
        │   │   ↓     │   │
        │   │ 手部纤维 │   │ ← 离中央近 ⭐
        │   └─────────┘   │
        │     中央管       │
        └─────────────────┘

中央水肿 → 先压手部纤维 → 上肢瘫
水肿向外扩 → 影响躯干 / 脚部纤维 → 下肢瘫
但中央损伤先打上肢
```

### 临床三联

1. **运动**：**双上肢瘫 > 双下肢瘫** ⭐ 招牌
   - 急性期：弛缓性瘫（spinal shock）
   - 数天-周后：转为痉挛性
2. **感觉**：双侧披肩样痛温↓（可有可无）
3. **括约肌**：可有尿潴留

### 病因
- **老人 + 颈椎过伸伤** ⭐ 最常见
  - 跌倒撞下巴
  - 挥鞭样 (whiplash)
  - 颈椎管狭窄（osteophytes 后突）+ 过伸 = 中央夹压
- 年轻人创伤性（少见）
- 颈椎肿瘤 / 脊髓炎

### 影像首选 ⭐
**MRI cervical spine WITHOUT contrast** ⭐

- 骨折 → CT
- 脊髓本身（外伤）→ MRI **no** contrast
- 脊髓 + 怀疑肿瘤 / 感染 → MRI with contrast
- 血管 → CTA / MRA

⚠️ **CT with contrast 用于颈椎创伤 = 错** — 血液和造影剂 Hounsfield 接近，会**掩盖出血**

### 治疗
- **颈托制动**
- **静脉类固醇**（争议中，但 USMLE 倾向给）
- 颈椎管狭窄严重 → **手术减压**（laminectomy / 椎间盘切除）
- 早期康复（多数 6-12 月部分恢复）
- 预后：**上肢恢复差于下肢**（因为手部 CST 损伤更重）

### USMLE 钩子
> "Old man + fall + neck hyperextension + bilateral arm weakness > leg" → **CCS** → MRI cervical spine no contrast

## D.2 Anterior Cord Syndrome (ACS) ⭐ AAA 术后

### 经典 stem
"**68 岁男** + **腹主动脉瘤修复术后 12 小时** + 突发**双下肢瘫痪** + 双下肢痛温觉↓ + **振动 / 位置觉正常** + 尿潴留"
![](https://apps.uworld.com/media/L23796.jpg)
### 解剖损伤
```
前脊髓动脉梗死（最常 T9-T12 左侧 Adamkiewicz）
        ↓
脊髓前 2/3 梗死：
- ❌ Spinothalamic（痛温丢）
- ❌ Lateral CST（运动丢）
- ❌ Anterior horn（LMN 受累）
- ❌ 外侧角（自主功能受影响）
        ↓
Dorsal column 保留（后脊髓动脉供血）⭐
        ↓
临床：
- 病变以下双侧瘫
- 病变以下双侧痛温↓
- 振动 / 本体 / 精细触觉保留 ⭐
- 急性期：Spinal shock（弛缓性瘫 + 反射↓ + 自主神经异常）
- 数天-周后：转为痉挛性（spasticity + Babinski +）
- 尿便障碍（autonomic）
```

### 病因（USMLE 必记）— **两大机制**
| 病因 | 机制 | 梗死位置 |
|---|---|---|
| **AAA 修复术** ⭐⭐⭐ | **Adamkiewicz 动脉结扎** | T9-T12 以下 |
| **主动脉夹层** | 累及肋间动脉 → 脊髓供血失败 | 看夹层位置 |
| **脊髓动脉栓塞** | 心源性栓子 / 主动脉粥样硬化 | 看栓塞位置 |
| **严重低血压**（心脏骤停 / 大失血 / 休克）⭐ | **Watershed (T4-T8) 低灌注** | **T4-T8** |
| **椎间盘脱出 / 椎管狭窄** | 压迫前脊髓动脉 | 看压迫位置 |
| **NSAIDs / 减压病** | 罕见 | - |

⭐ **关键鉴别**：
- "**AAA 术后** + 突发双下肢瘫" → **Adamkiewicz 损伤型**（T9-T12 以下）
- "**心脏骤停 / 休克后** + 双下肢瘫" → **Watershed 型**（T4-T8）

详见 [[#A.3 脊髓血供]] Adamkiewicz vs Watershed 对比表

### Adamkiewicz 动脉详解
- **大根动脉**，是脊髓下 2/3 主要供血来源
- 起源最常 **T9-T12 左侧**（70%）
- AAA 术中**结扎 Adamkiewicz** → ACS

### 临床三联（USMLE 招牌）
1. **病变以下双瘫**
2. **病变以下双侧痛温↓**
3. **振动 / 本体 / 精细触觉保留** ⭐ 鉴别钥匙

### 治疗 / 预后
- **没有特异性治疗** ⚠️
- 维持灌注（升压、避免低血压）
- 早期康复
- **预后差**：多数遗留显著神经功能障碍

### USMLE 钩子
> "AAA repair + sudden bilateral leg weakness + pain/temp loss + **vibration/position preserved**" → **ACS** → 治疗：维持灌注

## D.3 Brown-Séquard Syndrome ⭐ 单侧半切

### 经典 stem
"**25 岁男** + **被刀刺中后背 T6 右侧** + 急诊查体：右下肢**运动↓ + 振动 / 本体↓** + 左下肢**痛温↓**"
![](https://apps.uworld.com/media/L17580.jpg)
### 解剖损伤
```
脊髓单侧半切（如 T6 右侧）
        ↓
右侧损伤：
1. CST 断 → 右下肢同侧运动↓（已在 medulla 交叉）
2. Dorsal column 断 → 右下肢同侧振动 / 本体↓（还没交叉）
3. Spinothalamic 断 → 但这条纤维是从左侧来的（已交叉）
   ↓
   所以损伤右侧 spinothalamic = 左下肢痛温↓（对侧）
4. 前角 / 后角 → 右侧 T6 节段 LMN 征 + 全感觉丢失（损伤节段）
```

### 经典三征（USMLE 招牌）
1. **同侧运动↓**（CST）
2. **同侧振动 / 本体 / 精细触觉↓**（Dorsal column）
3. **对侧痛温↓**（Spinothalamic）⭐

### 特殊征：感觉平面不同步

| 感觉类型 | 平面位置 |
|---|---|
| 同侧振动 / 本体 | 损伤平面**正好**位置 |
| 同侧节段感觉**全丢**（神经根受累）| 损伤平面 1-2 节段 |
| **对侧痛温**（spinothalamic）| 损伤平面**下 1-2 节段** ⭐ |

⭐ **为什么对侧痛温平面差 1-2 节段**：
- Spinothalamic 进入脊髓后**先在同侧上行 1-2 节段**才交叉
- 所以 T6 损伤实际切断的是来自 **T7-T8** 的痛温纤维
- 临床上对侧痛温↓ 表现为 **T7-T8 以下**（而不是 T6 以下）

### 病因
- **穿透伤**（刀刺 / 枪伤）⭐ 经典
- 脊柱手术
- 椎管内肿瘤（单侧压迫）
- MS（罕见）
- 脊髓栓系
- 颈椎间盘突出（侧方）

### 治疗 / 预后
- 病因治疗（外科减压 / 异物取出）
- 早期康复
- **预后较其他综合征好**（一半神经组织完整）
- 多数恢复行走能力

### USMLE 钩子
> "Stab wound to back + ipsilateral motor/vibration loss + contralateral pain/temp loss" → **Brown-Séquard**

## D.4 Posterior Cord Syndrome ⭐ B12 SCD / Tabes

### 经典 stem 1（B12 SCD）
"**65 岁女** + 严格素食 + **大细胞贫血** + 双下肢**振动 / 位置觉↓** + Romberg+ + 反射↓ + 痛温觉**正常** + 神经精神改变"

### 经典 stem 2（Tabes dorsalis 神经梅毒晚期）
"**55 岁男** + 多年前未治疗梅毒 + 双下肢**振动 / 位置觉↓** + **闪电样疼痛** + **Argyll-Robertson 瞳孔** + Charcot 关节"


![[{4AD76420-65A9-4C54-8A3E-ED37B1D917E4}.png]]
### 解剖损伤
```
Dorsal column 选择性损伤
        ↓
- ❌ Dorsal column（振动 / 本体 / 精细触觉）⭐
- ✅ Spinothalamic 保留（痛温觉正常）⭐
- ❌ Lateral CST 可受累（B12 SCD = "Subacute Combined Degeneration"）
        ↓
临床：
- 振动 / 位置觉↓（双侧）
- Romberg+（睁眼能站，闭眼倒）⭐
- Sensory ataxia（感觉性共济失调）
- 反射↓（早期，late stage UMN 上来）
- B12 SCD：UMN 征（Babinski+ / spasticity）
- 痛温觉保留 ← 反向分离 ⭐
```

### B12 SCD 详解（Subacute Combined Degeneration）

#### 累及的束（"Combined"）
1. **Dorsal column** → 振动 / 本体丢
2. **Lateral CST** → UMN 征
3. **Spinocerebellar** → 共济失调
4. **Spinothalamic 不累及** → 痛温保留 ⭐

#### 高危人群
- **严格素食**（vegan）⭐
- "Tea and toast diet"（老人偏食）
- **Pernicious anemia**（自身免疫性壁细胞 / IF）
- **Crohn / 回肠切除** → 影响 B12 吸收
- **Metformin 长期**（影响 B12 吸收）
- **PPI 长期**
- **N₂O（一氧化二氮）滥用** → 灭活 B12

#### 诊断
- B12 ↓（但可正常）
- **MMA + Homocysteine 升高**（更敏感）⭐
- CBC：大细胞贫血 / 高 MCV / 多分叶中性
- **神经症状可早于贫血** ⭐

#### 治疗
- **B12 IM 注射**（每日 → 周 → 月）
- 不要单独补叶酸 → 会**加重**神经症状（"folate trap"）⚠️

### Tabes Dorsalis（梅毒晚期）

![[{61F23A4E-263F-47B8-B10E-1C501EE265A0}.png]]

#### 病理
- **Treponema pallidum 慢性感染**
- 选择性破坏 **dorsal column + dorsal root**

#### 临床三联
1. **闪电样疼痛**（lancinating pain，腿部）
2. **Argyll-Robertson 瞳孔** ⭐（"prostitute's pupil"）
   - **调节反射保留**（看近物会缩）
   - **光反射消失**（光照不缩）
3. **Sensory ataxia + Romberg+**
- + Charcot 关节（无痛性关节破坏）
- + 反射↓

> [!tip] 深入机制见专题
> **Argyll-Robertson 瞳孔为什么"调节留、光反射丢"**（中脑顶盖前区 vs 腹侧近反射通路）+ **闪电样痛的异位放电机制** + ARP vs Adie tonic pupil 鉴别 → [[完整笔记/专题笔记/神经/神经_瞳孔异常鉴别]]

#### 治疗
- **IV Penicillin G**

### USMLE 钩子
- "Vegan / 大细胞贫血 + 振动丢 + Romberg+" → **B12 SCD** → IM B12
- "梅毒史 + 闪电样痛 + Argyll-Robertson + Charcot 关节" → **Tabes dorsalis** → IV PCN

## D.5 Syringomyelia ⭐ 中央向外烂

### 经典 stem
"**32 岁女** + **Chiari I malformation** + **6 个月**反复**双手无痛烫伤** + 双侧颈肩 + 双臂 + 上胸**披肩样痛温↓** + 触振正常"
![](https://apps.uworld.com/media/L13075.jpg)
### 解剖损伤
```
颈髓中央管周围空洞（syrinx）形成
        ↓
空洞从中央向外扩张（centrifugal expansion）
        ↓
按距离中央远近，损伤 5 步走：
```

### 5 幕剧（从内向外烂）⭐⭐⭐ 必背

```
幕 1：前白连合（最近）
       ↓
       双侧 spinothalamic 交叉纤维断
       → 双侧痛温丢（披肩样 C5-T2）⭐
       
幕 2：外侧角（T1-T2）
       ↓
       一级交感神经元断
       → 同侧 Horner 综合征（ptosis + miosis + anhidrosis）⭐
       
幕 3：前角细胞
       ↓
       LMN 损伤
       → 上肢 LMN 征（萎缩 + 肌力↓ + 反射↓）⭐
       
幕 4：外侧 Lateral CST
       ↓
       UMN 损伤
       → 下肢 UMN 征（hyperreflexia + Babinski +）
       
幕 5：Sacral CST 最外侧（最远）
       ↓
       Sacral sparing 最后破（屁股麻最晚）
```

⭐ **演员出场顺序 = 离中央距离的顺序**：中央 → 外侧

### 为什么"披肩样"？

```
颈髓中央空洞最常累及 C5-T2 节段
        ↓
对应皮节：
- C5 = 三角肌 + 上臂外侧
- C6 = 上臂外侧 + 拇指
- C7 = 中指
- C8 = 小指 + 前臂内侧
- T1 = 上臂内侧
- T2 = 腋下 + 胸上
        ↓
合起来：双侧颈肩 + 双臂 + 上胸
        ↓
看起来像披一件无袖背心 / 短披肩
        ↓
= "Cape-like distribution" ⭐
```

⭐ **关键解剖**：颈髓中央管在颈膨大处最宽（C5-T2）→ 空洞好发位置 → 临床上颈髓 Syringomyelia 最常见

### 病因
| 病因 | 机制 |
|---|---|
| **Chiari I malformation** ⭐ 最常见 | 小脑扁桃体下疝 → CSF 动力学异常 → 颈髓中央管扩张 |
| **MVC（motor vehicle collision, 机动车碰撞/车祸）/ Whiplash（挥鞭样损伤）数年后** | 创伤后空洞形成（Post-traumatic syringomyelia，创伤后脊髓空洞症）|
| **脊髓肿瘤**（Ependymoma 等）| 阻塞 CSF 流动 |
| **脊髓栓系** | 神经管发育异常 |
| **脊髓蛛网膜炎** | 感染后 / 出血后 |

### 影像
- **MRI cervical spine** ⭐
- 见**纵行空腔**（高 T2 信号管）
- + 评估 Chiari I（小脑扁桃体下疝 > 5 mm）

### 治疗
- **无症状小 syrinx** → 观察
- **症状 + Chiari I** → 神经外科 **后颅窝减压术**
- **进展 + 大 syrinx** → 直接引流（shunting）

### 创伤后 Syringomyelia ⭐ USMLE 高频陷阱

```
CCS（急性创伤）
     ↓
急性期：水肿 / 挫伤
     ↓
多数恢复（数周-月）
     ↓
少数遗留慢性空洞 → 数年后变成 Post-traumatic Syringomyelia
     ↓
临床表现：慢性进展性披肩样痛温↓ + 上肢 LMN
```

⭐ Stem 写 "**MVC 5 年前 + 现在 6 个月进行性上肢萎缩 + 披肩痛温↓**" → **Post-traumatic Syringomyelia**

### CCS vs Syringomyelia 完整对比 ⭐⭐⭐ 中央综合征终极鉴别

> [!warning] 两个"中央脊髓综合征"高频混淆
> 
> 都是**中央向外烂**的机制，都可能有**披肩样痛温↓**，但临床表现差异极大。

| 维度 | **Central Cord Syndrome (CCS)** | **Syringomyelia** |
|---|---|---|
| **起病** | **急性**（数小时内）⭐ | **慢性**（数月-年）⭐ |
| **年龄** | **老人为主**（> 65 岁）| 年轻人为主（20-50 岁）|
| **病因** | **颈椎过伸伤 + 椎管狭窄**（楼梯摔 / whiplash 即刻）| **Chiari I** / MVC 数年后 / 脊髓肿瘤 |
| **病理** | 中央灰质**水肿 / 出血 / 挫伤** | 中央**空洞**（cyst，CSF-like 液体）|
| **影像** | MRI 见**水肿 / 挫伤信号** | MRI 见**纵行空腔**（高 T2 信号管）|
| **主诉症状** | "**手没力气**"（运动为主）⭐ | "**Painless burns**"（感觉为主）⭐ |
| **运动症状** | **早期就有**（双上肢瘫 > 双下肢瘫）⭐ | **早期几乎无**（晚期上肢 LMN）|
| **披肩样痛温↓** | ⚠️ **可有但次要**（被运动症状掩盖）| ✅ **典型 / 招牌**（首发症状）⭐ |
| **肌力变化** | 急性弛缓 → 数周后转痉挛 | 晚期：上肢 LMN（前角受累，肌力↓ + 萎缩 + 反射↓）|
| **典型 stem** | "老人 + 楼梯摔 + 撞下巴 + 双臂无力" | "年轻人 + Chiari + 双手反复无痛烫伤" |
| **5 幕剧（病程演变）** | 无（急性 + 部分恢复）| ✅ 披肩 → Horner → 上肢 LMN → 下肢 UMN → Sacral sparing |
| **治疗** | 颈托 + 静脉类固醇 + 颈椎管减压（如严重）| 无症状观察 / 有症状 + Chiari → 后颅窝减压 |
| **预后** | 多数 6-12 月部分恢复（上肢恢复差于下肢）| 进行性加重 / 减压后稳定 |

⭐ **关键鉴别 3 个钥匙**：
1. **急/慢性**：急性 = CCS / 慢性 = Syringomyelia
2. **主诉**：运动主诉 = CCS / 感觉主诉 = Syringomyelia
3. **病因**：过伸伤即刻 = CCS / Chiari 或 MVC 数年后 = Syringomyelia

### "披肩样"在两个病的"优先权"

> [!tip] 当 USMLE stem 出现 "**Cape-like distribution**"
> 
> 1. **第一反应**：Syringomyelia（90% 的 USMLE 题）— 因为披肩样是 Syringomyelia 招牌
> 2. **次反应**：CCS（如果 stem 同时强调"急性 + 创伤 + 老人 + arms > legs"）
> 3. **罕见**：胸髓中央病变 / 多发性硬化中央病变

⭐ **披肩样优先权排序**：
- 慢性 + Chiari → **Syringomyelia** ⭐⭐⭐
- 急性 + 老人 + 创伤 → **CCS**（披肩样次要）
- 慢性进展 + MVC 数年前 → **Post-traumatic Syringomyelia**（桥接病例）

### CCS → Syringomyelia 的"自然演变"⭐ 罕见但 USMLE 高频陷阱

```
CCS（急性创伤）
   ↓
急性期：水肿 / 挫伤 / 出血
   ↓
多数恢复（数周-月）
   ↓
少数遗留慢性空洞
   ↓
数年后 → 临床进展为 Post-traumatic Syringomyelia
   ↓
临床进入 Syringomyelia 5 幕剧
   ↓
披肩样痛温↓ → Horner → 上肢 LMN → ...
```

⭐ **临床现实**：很多 Syringomyelia 病人有数年前 CCS / MVC 史 → 这是个**真实的疾病演变路径**

### "Arms > Legs" 三向对比（顺手延伸）

| 病 | 优势瘫痪 | 解剖机制 |
|---|---|---|
| **MCA stroke** | **上肢 > 下肢 + 面部** ⭐ | 皮层 homunculus：手 + 面在外侧（MCA 供血区）|
| **CCS** | **上肢 > 下肢** ⭐ | 髓内 CST 躯体定位：手部纤维近中央先废 |
| **ACA stroke** | **下肢 > 上肢** ⭐ | 皮层 homunculus：腿在内侧（ACA 供血区）|
| **Syringomyelia 晚期** | 上肢 LMN（不是瘫，是萎缩 + 反射↓）| 前角细胞受累（5 幕剧第 3 幕）|

⭐ **关键认知**：**"上肢 > 下肢"** 来自**两种不同解剖**：
- **MCA stroke**：皮层 homunculus（手脑大手区在外侧皮层）
- **CCS**：髓内 CST 躯体定位（手部纤维在内侧近中央）

### USMLE 钩子
- "**Painless burns on hands**" + Chiari I → Syringomyelia → MRI 颈髓
- "**Cape-like / Suspended sensory loss**" → Syringomyelia
- "MVC 数年前 + 慢性进展" → Post-traumatic Syringomyelia
- **"老人 + 过伸伤 + arms > legs 急性"** → CCS（不是 Syringomyelia）⭐

## D.6 Cauda Equina vs Conus Medullaris ⭐ USMLE 高频对比

⚠️ **关键解剖前提**：脊髓在**成人 L1-L2 椎体水平**结束
- L1 以上 = 脊髓本体
- **L1-L2 椎体水平** = **Conus medullaris**（脊髓圆锥，S1-S5 节段）
- **L2 椎体以下** = **Cauda equina**（马尾，神经根而非脊髓）

### Conus Medullaris Syndrome（脊髓圆锥）

#### 解剖
- S1-S5 脊髓节段（位于 L1-L2 椎体水平）

#### 临床
- **早期 + 严重的尿便障碍** ⭐
- **对称性下肢瘫**
- **Saddle anesthesia**（鞍区麻木 S2-S4）
- **UMN + LMN 混合**（既有脊髓部分，也有出髓神经根）

#### 病因
- L1 椎体骨折
- 脊髓圆锥肿瘤
- 脊髓终丝病
- 椎间盘脱出（L1 高位）

### Cauda Equina Syndrome（马尾）

#### 解剖
- L2 椎体以下的**神经根束**（已经不是脊髓）
- 所以是**纯 LMN 损伤** ⭐

#### 临床
- **不对称性下肢瘫**（神经根选择性受累）⭐
- **不对称感觉障碍**
- **Saddle anesthesia**（S2-S4 神经根）
- **晚期尿便障碍**（相对 Conus 晚）⭐
- **纯 LMN 征**（反射↓ + 萎缩）
- 严重**腰痛 + 神经根痛**

#### 病因（USMLE 必背）
| 病因 | 高频 |
|---|---|
| **中央椎间盘脱出**（L4-L5 / L5-S1）⭐ | 最常见 |
| **脊柱肿瘤** | 转移 / 原发 |
| **椎管狭窄** | 严重时 |
| **硬膜外脓肿 / 血肿** | 急症 |
| **强直性脊柱炎** | 慢性 |

#### 治疗 ⭐ 急症
- **急诊 MRI 腰椎** + 立刻减压
- **手术 < 48h 是黄金窗口** ⭐
- 延迟手术 → 永久神经功能障碍

### Cauda Equina vs Conus Medullaris 对比 ⭐

| 维度 | **Cauda Equina** | **Conus Medullaris** |
|---|---|---|
| **位置** | L2 椎体以下 | L1-L2 椎体水平 |
| **结构** | 神经根束 | 脊髓圆锥（S1-S5）|
| **起病** | 渐进**或**急性 | 急性 |
| **下肢瘫** | **不对称** ⭐ | **对称** |
| **反射** | **纯↓**（LMN）⭐ | UMN + LMN 混合 |
| **尿便障碍** | **晚期**（晚出现）⭐ | **早期**（早出现）⭐ |
| **疼痛** | **严重**（神经根痛）| 中-轻 |
| **Saddle 麻木** | ✅ | ✅ |
| **常见病因** | **L4-L5/L5-S1 椎间盘脱出** ⭐ | L1 骨折 / 圆锥肿瘤 |
| **治疗** | **< 48h 手术减压** ⭐ | **同为急症**：急诊 MRI + 减压（早期尿便受累更重，同样越早越好）|
| **是否急症** | ✅ 神经外科急症 | ✅ 神经外科急症（**别被鉴别表误导成"不急"**）|

### USMLE 钩子
- "**Bilateral leg weakness + asymmetric + saddle anesthesia + late urinary**" + L4-L5 disc herniation → **Cauda equina** → emergent MRI + surgery < 48h
- "**Symmetric leg weakness + early urinary + UMN + LMN mixed**" + L1 fracture → **Conus medullaris**

### 为什么是急症？（两者都急，处理相同）⭐

> [!danger] 定位 ≠ 紧急度：马尾与圆锥**都是神经外科急症**
> 上面的对比表只帮你**认部位/认模式**，不是分"谁急谁不急"。两者**处理完全相同**：急诊 MRI 腰骶段 → 紧急神经外科减压（±病因治疗：脓肿加抗生素、肿瘤性压迫可加激素）。
>
> **为什么急（time is function）：**
> - 受压的是**骶神经根 / 骶髓节段**（S2-S4 管膀胱、直肠、性功能）→ 持续受压 → 缺血 → 轴突损伤。
> - **膀胱功能一旦完全丧失往往不可逆**：从"尿潴留 + 充溢性尿失禁"进展到"括约肌完全失能"是预后分水岭；**等到完全性尿潴留才手术，恢复率显著下降** → 不能观察等待。
> - 常见病因本身是进行性/破坏性的（椎间盘脱出、硬膜外脓肿/血肿、肿瘤、外伤）→ 越拖压迫越重。

> [!warning] 圆锥"更急"的一面
> 圆锥内就是排尿/排便/性功能中枢（S2-S4）→ **尿便障碍出现得更早、更重、更对称**（马尾常晚期才累及）。所以看到"对称下肢 + 鞍区麻木 + **早期**尿潴留/大便失禁"别因为"圆锥治疗写病因治疗"就松懈，一样是抢时间。

> [!info] 最敏感的早期客观指标
> **尿潴留 / 排尿后残余尿（post-void residual, PVR, 排尿后残余尿量）增多**。急性腰背痛 + 排尿困难 → 先测 PVR，别漏；异常即按急症走。

> [!question]- Q. "急性腰痛 + 双侧坐骨神经痛 + 鞍区麻木 + 尿潴留"，你还没分清是马尾还是圆锥。下一步该做什么？为什么不能先观察？
> **下一步：立即急诊 MRI 腰骶段 + 紧急神经外科减压会诊**——不管最终判成 Cauda equina 还是 Conus medullaris，处理都一样。不能观察等待：受压的骶神经根/骶髓持续缺血，**膀胱括约肌功能一旦完全丧失可能永久不可逆**，减压越晚恢复率越低（time is function）。先测排尿后残余尿（PVR）作为最敏感的早期客观指标。

# F. "分离性感觉障碍" 概念详解

## F.1 什么是"分离性感觉障碍"

> [!info] 定义
> **某些感觉丢失 + 其他感觉保留** = 分离性感觉障碍（dissociated sensory loss）
> 
> 提示**脊髓内**有选择性束损伤（不是脊髓完全损伤，也不是周围神经病）

## F.2 4 种类型

| 类型 | 哪个束死 | 临床 | 代表病 |
|---|---|---|---|
| **前型**（痛温丢 + 触振留）⭐ | Spinothalamic + CST | 痛温↓ ± 运动↓ | **Syringomyelia / ACS** |
| **后型**（触振丢 + 痛温留）⭐ | Dorsal column | 振动↓ + Romberg+ | **B12 SCD / Tabes** |
| **半切型** | 单侧多束 | 同侧运动+振动 / 对侧痛温 | **Brown-Séquard** |
| **横断型**（全丢失）| 所有束 | 全感觉 + 运动丢 | 完全脊髓损伤 |

## F.3 "Suspended" vs "Sensory level"

### Suspended（悬垂式）⭐ Syringomyelia
```
                上方 ✅ 正常
                  ↓
              披肩样 ❌ 异常
                  ↓
                下方 ✅ 正常
```

### Sensory level（明确平面）⭐ 横断 / ACS / 横贯性脊髓炎
```
                上方 ✅ 正常
                  ↓
                T10 ─────── 明确分界线
                  ↓
                下方 ❌ 全部异常
```

⭐ **关键鉴别**：上下都正常 vs 以下全异常

# G. USMLE 经典题型

## G.1 直接锁定型（看 stem 就出答案）

### 题 1：CCS
> "75 岁男 + 从楼梯摔下撞下巴 + 双上肢瘫 > 双下肢瘫 + 双手痛温↓"
> → **Central Cord Syndrome** → MRI cervical spine no contrast

### 题 2：ACS
> "68 岁男 + AAA 修复术后 12h + 突发双下肢瘫 + 双下肢痛温↓ + 振动正常"
> → **Anterior Cord Syndrome** → 维持灌注 + 早期康复

### 题 3：Brown-Séquard
> "25 岁男 + 后背刀刺伤 + 右下肢运动↓振动↓ + 左下肢痛温↓"
> → **Brown-Séquard** → MRI 评估 + 减压

### 题 4：Syringomyelia
> "32 岁女 + Chiari I + 6 月双手无痛烫伤 + 披肩样痛温↓"
> → **Syringomyelia** → MRI 颈髓

### 题 5：B12 SCD
> "65 岁女 + 严格素食 + 大细胞贫血 + 振动↓ + Romberg+"
> → **B12 Subacute Combined Degeneration** → IM B12

### 题 6：Cauda Equina
> "45 岁男 + 突发腰痛放射 + 双下肢不对称瘫 + 鞍区麻木 + L4-L5 椎间盘脱出"
> → **Cauda Equina Syndrome** → emergent MRI + 手术 < 48h

## G.2 病程预测型（已诊断 → 下一步症状）

### 题 7：Syringomyelia 进展
> "Syringomyelia 病人，目前披肩样痛温↓。Further evaluation most likely to show?"
> → **Horner 综合征**（幕 2，外侧角扩张）⭐
> 
> 错误选项：
> - "Exaggerated biceps reflex"（反了！前角损伤 = LMN，不是 UMN）
> - "Decreased anal wink"（Sacral sparing！最后才破）
> - "Decreased MMSE"（颅神经无关）
> - "Jaw deviation"（CN V 在脑桥，不在脊髓）

## G.3 鉴别陷阱型（同类 vs 不同）

### 题 8：Syringomyelia vs ACS（"前型"分离性感觉障碍）
> "AAA 术后 12 小时双下肢瘫 + 痛温↓ + 振动保留"
> → **ACS**（不是 Syringomyelia，因为急性 + 双瘫 + 双下肢，不是披肩）

### 题 9：Syringomyelia vs CCS（中央综合征）
> "MVC 1 周后 + 双上肢瘫 > 双下肢瘫 + 披肩样痛温↓"
> → **CCS**（急性创伤后；不是 Syringomyelia，因为急性）

### 题 10：Cauda Equina vs Conus Medullaris
> "对称双下肢瘫 + 早期尿便障碍 + UMN+LMN 混合"
> → **Conus Medullaris**（不是 Cauda Equina，因为对称 + 早期尿便 + UMN）

### 题 11：B12 SCD vs Syringomyelia（反向分离）
> "65 岁素食 + 振动↓ + 痛温正常"
> → **B12 SCD**（反向，不是 Syringomyelia）

## G.4 解剖逻辑型（"为什么"）

### 题 12：Syringomyelia 招牌
> "Syringomyelia 病人为什么双侧痛温↓ + 触觉正常？"
> → **前白连合切断双侧 spinothalamic 交叉纤维** + dorsal column 完好

# I-bis. 神经根病 vs 脊髓病 + 颈痛鉴别 ⭐

> [!info] 定位钥匙：root（神经根）vs cord（脊髓）
> 颈椎退变（cervical spondylosis，颈椎病）既可压**神经根**（radiculopathy）也可压**脊髓**（myelopathy），考试靠体征区分。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q4205]]（颈神经根病影像 = facet 骨赘）。

| 维度 | 神经根病 radiculopathy | 脊髓病 myelopathy |
|---|---|---|
| 受累 | 单一**神经根（root, LMN）** | **脊髓（cord, 长束 UMN）** |
| 痛/感觉 | **皮节性（dermatomal）**放射至肩/臂 | 平面以下感觉障碍、Lhermitte 征 |
| 运动 | 该根支配肌无力（LMN：反射↓）| **下肢痉挛性轻瘫（UMN：反射↑、Babinski+）** |
| 二便/步态 | 多正常 | **步态不稳、二便障碍** |
| 体征 | **Spurling 征 +**（后伸侧屈诱发根痛）| 长束征、Hoffmann 征 |
| 影像 | X 线见 **facet/uncovertebral 骨赘 + 椎间孔狭窄**；MRI 看根 | MRI 看脊髓受压/T2 信号 |

> [!tip] 颈痛 7 病速记
> 肌肉拉伤（活动痛）/ facet 骨关节炎（老年、休息缓解）/ **神经根病（皮节放射 + Spurling+）** / 脊髓病（长束 + 步态二便）/ 脊柱关节病（年轻男 HLA-B27 晨僵）/ 转移瘤（夜痛不随体位）/ 椎体骨髓炎（发热盗汗、IVDU/免疫抑制）。
> 详见 [[mistakes/uworld-mistakes_2026-06#^Q4205]] 卡内颈痛鉴别表。

## I-bis.2 颈脊髓病（CSM）为什么先伤腿 / 步态 ⭐

> [!info] CSM = Cervical Spondylotic Myelopathy（颈椎病脊髓型 / 颈脊髓病）
> 退变性颈椎管狭窄（骨赘 + 椎间盘 + 后纵韧带肥厚/骨化）**压迫脊髓本身**（不是神经根）。是**老年人脊髓病 + 步态障碍最常见的病因**——所以"老人 + 进行性步态不稳"是 USMLE 招牌。

**核心逻辑：病灶在颈，症状在腿** —— 因为颈段脊髓里走的是**支配腿的长束（long tracts）**：往下的皮质脊髓束 + 往上的后索都"路过"颈段去管腿，压在脖子等于掐了去腿的"路过客"。

```
颈段脊髓受压（CSM）
   ├── 皮质脊髓束 (Corticospinal tract, 锥体束) 受压
   │      → 腿是 UMN 损伤（病灶在腰膨大以上）
   │      → 痉挛、无力、腱反射↑、Babinski(+)
   │      → 【痉挛步态 spastic gait】僵硬、剪刀步、拖步
   │
   └── 后索 (Dorsal column, 薄束/楔束) 受压
          → 本体感觉 (proprioception) + 振动觉↓
          → 不知道脚在哪 → 【感觉性共济失调 sensory ataxia】
          → 宽基底、不稳、Romberg(+)、黑暗加重
   ↓
 合并 = spastic-ataxic gait（痉挛-共济混合步态）
```

> [!tip] 为什么"腿先于手"——躯体定位（somatotopic lamination）
> 外侧皮质脊髓束里**管腿的纤维在最外层**，管手的在内层。退变骨赘/椎间盘**从外往里压** → 先碰外层的腿纤维 → **下肢、步态往往最先受累**。（注意：与 [[#D.1 Central Cord Syndrome (CCS) ⭐ 老人创伤后|CCS]] 相反——CCS 是中央水肿向外，手部纤维近中央先废 → arms > legs。同一条束、压迫方向相反，结果相反。）

> [!warning] 指纹：同一病人上下肢体征"相反"
> - 腿在病灶**以下** → **UMN**（反射↑、Babinski+、痉挛）
> - 手在病灶**节段** → 可有 **LMN**（前角/根受累：萎缩、反射↓）+ **Hoffmann(+)**（颈髓 UMN 征）
> - 主诉常是"**手笨拙**"（系扣子难、易掉东西）+ "走路发飘"

> [!success] USMLE 钩子
> 老年人 + **进行性步态不稳 + 双手笨拙** + 颈部不适；查体**下肢反射亢进 + Babinski(+) + Hoffmann(+)** ±Lhermitte 征（低头电击样）。
> → **下一步：MRI 颈椎**（首选确诊）→ 显著压迫/进展 → 神经外科减压（椎板切除 laminectomy / 颈前路椎间盘切除融合术 ACDF, Anterior Cervical Discectomy and Fusion）。
> 步态定位见 [[完整笔记/专题笔记/神经/神经_步态鉴别谱]]（CSM = 痉挛-共济混合，可同时 Romberg± + 下肢 UMN）。

> [!warning] CSM vs B12 缺乏 vs ALS —— 三个"下肢痉挛/步态不稳"的鉴别，全靠**反射 + 感觉** ⭐
>
> 三者都能出现下肢无力/步态障碍，但下肢腱反射方向 + 有无感觉缺失立刻分开：
>
> | 鉴别点 | **CSM 颈脊髓病** | **B12 缺乏（SCD 亚急性联合变性）** | **ALS（amyotrophic lateral sclerosis，肌萎缩侧索硬化）** |
> |---|---|---|---|
> | 下肢反射 | **↑** 亢进 | **↓**（合并周围神经病）⭐ | **↑**（UMN）|
> | 感觉 / 后索 | 后索受累、**Romberg±**、振动觉↓ | 后索受累、**Romberg+**、振动觉↓双侧 | **完全正常** ⭐ |
> | 上肢 | 节段 **LMN + Hoffmann(+)**、双手笨拙 | 周围感觉异常（手套袜套） | **LMN（萎缩 + 肌束颤 fasciculation）+ UMN** |
> | 其他线索 | 颈痛±、MRI 见颈髓受压 | 巨幼贫、**MMA（methylmalonic acid，甲基丙二酸）↑**、素食/PPI/二甲双胍 metformin | **EMG（electromyography，肌电图）广泛失神经**、无感觉症状 |
> | 钥匙 | **反射↑ + 颈髓压迫**（影像）| **反射↓ + 巨幼贫 + MMA↑** | **纯运动、无任何感觉缺失** |
>
> 一句话：**反射↓ → 先想 B12**（合并周围神经）；**反射↑ + 有后索感觉缺失 → CSM**（查颈椎 MRI）；**反射↑ 但感觉一点不缺 + 肌束颤 → ALS**。

# I-ter. Lhermitte 征病因鉴别 ⭐

> [!info] 定义 + 机制
> **Lhermitte 征（Lhermitte's sign，颈屈电击征）** = **屈颈（neck flexion）时**沿脊柱向下、可放射到四肢的**短暂电击样 / 触电样感觉（electric shock-like sensation）**。
> **本质 = 颈段脊髓后索（dorsal column，后柱）受激惹**——脱髓鞘或受压的后索轴突在屈颈牵拉时**异位放电（ectopic firing）**。
> ⚠️ 它是**定位征、不是病名**：意味着"颈髓后索出问题了"，但**不特异于 MS**。任何累及颈髓后索的病都可出现。

## I-ter.1 病因谱（MS 只是其一）

| 病因 | 机制 / stem 钩子 | 鉴别线索 |
|---|---|---|
| **Multiple sclerosis（MS，多发性硬化）** ⭐ | 颈髓脱髓鞘斑块 | 年轻女性、时间空间多发、视神经炎、**Uhthoff 现象**（热加重）|
| **Vitamin B12 deficiency → SCD（subacute combined degeneration，亚急性联合变性）** ⭐ | 后索 + 侧索脱髓鞘 | 巨幼贫、振动/本体觉↓ + UMN 征、**MMA（methylmalonic acid，甲基丙二酸）↑**、素食/恶性贫血 pernicious anemia |
| **Cervical spondylotic myelopathy（CSM，颈椎病脊髓型）** ⭐ | 退变致颈髓机械受压 | 老年、颈痛、双手笨拙、步态不稳、Hoffmann(+)（见 [[#I-bis.2 颈脊髓病（CSM）为什么先伤腿 / 步态 ⭐|I-bis.2]]）|
| **Radiation myelopathy（放射性脊髓病）** | 颈/胸/头颈**放疗后数月–数年**后索损伤 | Hodgkin 淋巴瘤 / 鼻咽癌 / 头颈癌放疗史 |
| **Cisplatin（顺铂）等神经毒性化疗** | 背根神经节 / 后索毒性 | 化疗史、手套袜套样感觉神经病 |
| **Nitrous oxide（N₂O，笑气）滥用** | 灭活 B12 → **功能性 B12 缺乏** → SCD 样 | 娱乐性吸食史（机制同 B12 缺乏）|
| **颈椎间盘突出 / 颈髓外伤（过伸伤）** | 后索机械激惹 | 外伤史 / 椎间盘影像 |
| **颈髓肿瘤（髓内/髓外压迫）** | 后索受压 | 进行性、夜痛、影像占位 |
| **Chiari 畸形 / Syringomyelia（脊髓空洞症）** | 颈髓结构异常 | 披肩样痛温↓（见 [[#D.5 Syringomyelia ⭐ 中央向外烂|D.5]]）|

## I-ter.2 USMLE 钩子 + 易混

> [!success] 看到 Lhermitte 先想 MS，再扫 stem 找"非 MS"替身
> - **素食 / 恶性贫血 / 笑气** → **B12（SCD）** → 查 MMA + 同型半胱氨酸，肌注 B12
> - **老年 + 颈痛 + 手笨 + 步态不稳** → **CSM** → 颈椎 MRI
> - **放疗史（Hodgkin/头颈癌）** → **放射性脊髓病**
> - **化疗史** → **cisplatin 神经毒性**
> 这四个是 UWorld 最爱当"非 MS 正解"的替身。

> [!warning] Lhermitte vs Uhthoff —— 两个都常见于 MS，别混
> - **Lhermitte 征** = **屈颈**诱发的电击样感（后索机械/脱髓鞘激惹）——是**动作诱发**。
> - **Uhthoff 现象（Uhthoff phenomenon）** = **体温升高**（运动、热水澡、发热）致症状**一过性加重**（脱髓鞘轴突传导阻滞）——是**温度诱发**。
> UWorld 爱把两者放同一 stem 里同时出现（都指向 MS），但诱因/机制完全不同。

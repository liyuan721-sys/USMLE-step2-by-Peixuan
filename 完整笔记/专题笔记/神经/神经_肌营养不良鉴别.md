# 神经_肌营养不良鉴别（Muscular Dystrophy）

> [!info] 一句话定位
> **肌营养不良 = 进行性肌无力 + CK（creatine kinase，肌酸激酶）↑**。USMLE 高频考点 = **遗传方式 + 区分钥匙**：Duchenne/Becker = X 连锁隐性近端假肥大；**Myotonic（强直性）= 常染色体显性 AD + grip myotonia（握力肌强直，松手延迟）+ 多系统**。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q3661]]（误把肌营养不良默认成隐性）。

## 一、三大类型对照

| 类型 | 遗传 / 基因 | 起病 | 临床表现 | 系统外表现 | 预后 |
|---|---|---|---|---|---|
| **Duchenne (DMD, 杜氏)** | X 连锁隐性 / **dystrophin（抗肌萎缩蛋白）缺失** | 2-3 岁 | 近端肌无力、**Gowers 征**（用手撑腿起立）、**腓肠肌假性肥大** | 心肌病、心律失常、脊柱侧弯 | 青春期前坐轮椅，20-30 岁死于呼吸/心衰 |
| **Becker (贝氏)** | X 连锁隐性 / **dystrophin 减少**（部分功能保留）| 5-15 岁 | 较 Duchenne 轻的近端无力、假肥大 | 心肌病、心律失常 | 40-50 岁死于心衰 |
| **Myotonic (DM, 强直性) ⭐** | **AD 常染色体显性 / *DMPK* 基因 CTG 三核苷酸重复扩增** | **12-30 岁**（最常见青少年/成人起病）| **面+远端无力、grip myotonia（握拳后松手延迟）、percussion myotonia（叩击大鱼际拇指外展持续）** | **心传导阻滞/心律失常、白内障、睾丸萎缩/性腺功能减退、额秃、胰岛素抵抗、吞咽困难、日间嗜睡** | 45-55 岁死于呼吸/心衰 |

## 二、区分钥匙（一眼定位）

```
进行性肌无力 + CK↑
        │
        ├─ 男婴/男童 + 近端 + 腓肠肌假肥大 + Gowers ──► Duchenne / Becker (X 连锁隐性, dystrophin)
        │
        ├─ 青少年/成人 + 面/远端 + 松手延迟(myotonia)
        │   + 白内障 + 睾丸萎缩 + 额秃 ──────────────► Myotonic (AD, DMPK CTG 重复)
        │
        └─ 肩盆带肌无力, 面部少累及 ──────────────────► Limb-girdle (多为 AR) / Congenital MD (AR)
```

> [!tip] 强直性肌营养不良三大记忆锚
> ① **grip myotonia 松手延迟**（握手后手张不开）= 招牌；② **多系统**（心-眼-性腺-代谢）；③ **anticipation（遗传早现）**——CTG 重复随代增多 → 代际起病更早、更重。

## 三、易混陷阱（遗传方式）

> [!warning] 别把"肌营养不良"默认成隐性
> - **AR（常染色体隐性）** = congenital muscular dystrophy（先天性肌营养不良）/ 多数 limb-girdle（肢带型）；肩盆带无力，**少累及面部**，无 grip myotonia。
> - **X 连锁隐性** = Duchenne/Becker；**近端**无力 + 假肥大，**无肌强直、无睾丸萎缩**。
> - **X 连锁显性（Fragile X 脆性 X 综合征）** = **macroorchidism 睾丸增大**（非萎缩）、智力障碍，无 myotonia——与 DM 的睾丸**萎缩**正好相反。
> - **线粒体病（mitochondrial myopathy）** = myalgia + 易疲劳 + 运动不耐受 + 母系遗传，无 myotonia。

## 四、Stem 钩子

- "握手后手松不开 / 叩大鱼际拇指外展持续" → myotonia → **DM**。
- "面颊凹陷 + 颞肌萎缩 + 额秃 + 白内障 + 睾丸小" → **DM 多系统**。
- "男童 + 腓肠肌粗大但无力 + Gowers" → **Duchenne**。
- 问遗传方式：DM → **AD**；Duchenne/Becker → **XR**。

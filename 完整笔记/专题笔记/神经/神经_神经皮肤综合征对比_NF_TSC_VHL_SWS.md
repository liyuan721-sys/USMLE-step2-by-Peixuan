# 神经皮肤综合征对比（NF1 / NF2 / TSC / VHL / Sturge-Weber）

> [!info] 一句话定位
> **神经皮肤综合征 (neurocutaneous syndromes / phakomatoses, 斑痣性错构瘤病)** = 一组同时累及**神经系统 + 皮肤 ± 内脏**的遗传性肿瘤易感综合征。USMLE 高频考"**从一个标志性病灶 → 认出综合征 → 反推该去哪儿筛第二个肿瘤**"。
> 触发：[[NBME11_错题本#^Q066]]（后颅窝血管母细胞瘤 → VHL → 查腹部肾/胰/嗜铬细胞瘤）。

---

## 一、五大对比表 ⭐

| 综合征 | 基因/遗传 | 神经标志 | 皮肤标志 | 内脏/其他 | 高频"第二肿瘤" |
|---|---|---|---|---|---|
| **NF1（1 型神经纤维瘤病 neurofibromatosis type 1）** | NF1（17 号；常显 AD）| 视神经胶质瘤 (optic glioma)、神经纤维瘤 | **咖啡牛奶斑 (café-au-lait)**、腋/腹股沟雀斑、虹膜 Lisch 结节 | 嗜铬细胞瘤 (pheochromocytoma)、骨发育不良 | 视路胶质瘤 / 嗜铬细胞瘤 |
| **NF2（2 型 neurofibromatosis type 2）** | NF2（22 号；AD）| **双侧前庭神经鞘瘤 (bilateral vestibular schwannoma)**、脑膜瘤、室管膜瘤 | 少（可有皮肤神经鞘瘤）| 青少年白内障 | 双侧听神经瘤 |
| **TSC（结节性硬化 tuberous sclerosis complex）** | TSC1/TSC2（AD）| 皮质结节、室管膜下结节/巨细胞星形细胞瘤 (SEGA, subependymal giant cell astrocytoma)、癫痫、智力障碍 | **灰叶斑 (ash-leaf)**、面部血管纤维瘤、鲨革斑 (shagreen) | 心脏横纹肌瘤 (rhabdomyoma)、肾血管平滑肌脂肪瘤 (angiomyolipoma)、肺淋巴管平滑肌瘤病 (LAM, lymphangioleiomyomatosis) | 心横纹肌瘤 / 肾 AML |
| **VHL（von Hippel-Lindau 病）** | VHL（3 号；AD，抑癌基因）| **小脑/视网膜血管母细胞瘤 (hemangioblastoma)** | 少 | **肾透明细胞癌 (clear cell RCC)、胰腺囊肿/肿瘤、嗜铬细胞瘤** | 肾癌 / 胰腺 / 嗜铬细胞瘤 |
| **Sturge-Weber（SWS，脑三叉血管瘤病）** | GNAQ 体细胞突变（散发，**非遗传**）| 软脑膜血管瘤、癫痫、偏瘫、智力障碍 | **三叉 V1 分布葡萄酒色斑 (port-wine stain)** | 青光眼 (glaucoma) | 同侧青光眼 / 软脑膜血管瘤 |

## 二、VHL 重点（Q066）

> [!tip] 识别链：病灶 → 综合征 → 筛查
> **后颅窝 (posterior fossa) 血管性肿瘤 + 颅压增高（晨起喷射性呕吐/视物模糊）+ 小脑征（dysmetria 辨距不良 / 轮替障碍 / 步态不稳）+ 家族史**
> → **cerebellar hemangioblastoma（小脑血管母细胞瘤）**
> → **VHL（常染色体显性抑癌基因病）**
> → 去**腹部 CT** 找 **肾透明细胞癌 / 胰腺肿瘤 / 嗜铬细胞瘤**（"第二个同基因肿瘤"）。

**机制**：VHL 抑癌基因失活 → HIF（缺氧诱导因子 hypoxia-inducible factor）降解失调 → 促血管生成 → 中枢/视网膜血管母细胞瘤 + 肾/胰/肾上腺肿瘤。

> [!warning] 干扰项逻辑（Q066）
> 看到"家族性癌症综合征"易反射 **colonoscopy（结肠镜，指向遗传性结直肠癌）**——但本题点名的是 **VHL 的血管性肿瘤谱**（肾/胰/嗜铬），不是肠癌。Bone marrow biopsy / mammography / CEA 均与 VHL 肿瘤谱不符。

## 三、Memory Hook

> [!success] 记忆挂钩
> - **NF1**：咖啡斑 + Lisch + 视路胶质瘤 + 嗜铬（17 号，"NF**1**七"）。
> - **NF2**：**双侧听神经瘤**（22 号，"**2**侧 2 听 2十2"）。
> - **TSC**：灰叶斑 + 癫痫 + 心横纹肌瘤 + 肾 AML。
> - **VHL**：**血管母细胞瘤（小脑/视网膜）+ 肾癌 + 胰 + 嗜铬** → "脑里长血管瘤，腹部再找一轮"。
> - **SWS**：V1 葡萄酒色斑 + 癫痫 + 青光眼（**散发**，非遗传）。
> - 共享"**嗜铬细胞瘤**"者：**NF1 + VHL**（+ MEN2）。

> [!tip] 染色体速记（FA 钩子，高频送分）
> - **NF1 → 17 号**："von Recklinghausen (von Recklinghausen 病)" 正好 **17 个字母**；NF1 又叫 von Recklinghausen disease。
> - **NF2 → 22 号**："NF**2** 影响 **2** 只耳（双侧听神经瘤）+ **2** 只眼（青少年白内障 juvenile cataract）"，**2-2-22**。
> - **VHL → 3 号**："**VHL** = **3** 个字母 = **3** 号染色体"；又 RCC 也在 3 号（3p 缺失）。
> - **TSC → TSC1=9 号（hamartin）/ TSC2=16 号（tuberin）**："**tu(two)berin → TSC2**"，tuberin 配 TSC2；hamartin 配 TSC1。

## 四、FA 助记词（mnemonics）⭐

> [!tip] STURGE — Sturge-Weber 七要素
> - **S** — **S**poradic 散发 + port-wine **S**tain（葡萄酒色斑，V1 三叉分布）
> - **T** — **T**ram-track calcifications（**轨道征/双轨样钙化**，沿脑回，头颅 CT/平片经典）
> - **U** — **U**nilateral 单侧（同侧软脑膜血管瘤 + 同侧病变）
> - **R** — **R**etardation（智力障碍 intellectual disability）
> - **G** — **G**laucoma 青光眼 + **G**NAQ 基因
> - **E** — **E**pilepsy 癫痫（± 对侧偏瘫 hemiparesis）

> [!tip] HAMARTOMAS(ES) — Tuberous sclerosis（"全是错构瘤 hamartoma"）
> - **H** — **H**amartomas（CNS + 皮肤错构瘤）
> - **A** — **A**ngiofibromas（面部血管纤维瘤，旧称 adenoma sebaceum）
> - **M** — **M**itral regurgitation（二尖瓣反流）
> - **A** — **A**sh-leaf spots（灰叶斑，**Wood 灯**下显形，婴儿最早体征）
> - **R** — cardiac **R**habdomyoma（心脏横纹肌瘤，常自发消退；可致胎儿/新生儿心律失常）
> - **T** — au**T**osomal dominant（常显，外显率可变）
> - **O** — intellectual disability（智力障碍）
> - **M** — renal angio**M**yolipoma（肾血管平滑肌脂肪瘤，可大出血 → Wunderlich syndrome 自发性肾周出血）
> - **A** — renal **A**ngiomyolipoma（同上，强调"肾 AML"）
> - **S** — **S**eizures（癫痫，婴儿痉挛 infantile spasms 经典）+ **S**hagreen patches（鲨革斑）
> - 额外：**SEGA**（室管膜下巨细胞星形细胞瘤，**孟氏孔 foramen of Monro** 旁，可阻塞致脑积水）+ **ungual/periungual fibromas**（甲周纤维瘤）+ **LAM**（肺淋巴管平滑肌瘤病）

> [!tip] CICLOPSS — NF1（"独眼巨人 Cyclops"）
> - **C** — **C**afé-au-lait spots（咖啡牛奶斑，≥6 个 ≥5mm 是诊断标准之一）
> - **I** — **I**ntellectual disability / 学习障碍
> - **C** — **C**utaneous neurofibromas（皮肤神经纤维瘤）
> - **L** — **L**isch nodules（虹膜 Lisch 结节 = **色素性虹膜错构瘤** pigmented iris hamartoma，裂隙灯下看）
> - **O** — **O**ptic glioma（视神经/视路胶质瘤）
> - **P** — **P**heochromocytoma（嗜铬细胞瘤）
> - **S** — **S**eizures / focal neurologic **S**igns（常源于伴发脑膜瘤 meningioma）
> - **S** — **S**keletal（骨病变，如 **蝶骨发育不良** sphenoid dysplasia、假关节）

> [!tip] HARP — VHL（"VHL 弹竖琴 harp"）
> - **H** — **H**emangioblastomas（小脑/脑干/视网膜/脊髓血管母细胞瘤，**核深染高血管性**）
> - **A** — **A**ngiomatosis（血管瘤病）
> - **R** — bilateral **R**enal cell carcinoma（双侧透明细胞肾癌）
> - **P** — **P**heochromocytomas（嗜铬细胞瘤）

## 五、对比记忆（cross-cutting）⭐

### 5.1 机制 / 通路对比（决定靶向治疗 + 临床彩蛋）

| 综合征 | 基因产物 | 通路 | 失活后果 | 高频彩蛋 |
|---|---|---|---|---|
| **NF1** | neurofibromin（神经纤维瘤蛋白）| **负向调控 RAS** | RAS↑ → 增殖↑ | 抑癌基因；唯一**100% 外显率** |
| **NF2** | merlin（schwannomin）| 膜-细胞骨架连接 | 雪旺细胞增殖 | 抑癌基因 |
| **TSC** | hamartin + tuberin | **抑制 mTOR** | **mTOR↑** | → **依维莫司 everolimus / 西罗莫司 sirolimus（即雷帕霉素 rapamycin；mTOR 抑制剂）** 治 SEGA / 肾 AML / LAM |
| **VHL** | pVHL（泛素连接酶）| **泛素化降解 HIF-1α** | **HIF-1α↑ → VEGF/EPO↑** | 血管母细胞瘤可**产 EPO → 继发性红细胞增多症 polycythemia** |
| **SWS** | GNAQ | G 蛋白信号 | 体细胞激活突变（**镶嵌**）| **唯一散发非遗传** |

> [!danger] 高频送分点
> - **VHL 血管母细胞瘤 → 分泌 EPO → 继发性红细胞增多症**（实验室见 Hct↑、EPO↑）—— UWorld 经典连环。
> - **TSC 用 mTOR 抑制剂**（everolimus / sirolimus）—— 机制题直接由 hamartin/tuberin→mTOR 推出。
> - **共享嗜铬细胞瘤**：**NF1 + VHL**（再加 MEN2A/2B）；其余三个不查嗜铬。

### 5.2 眼部表现对比（每病一个，最爱混）

| 综合征 | 眼部标志 | 一句话 |
|---|---|---|
| **NF1** | **Lisch 结节**（色素性虹膜错构瘤）| 裂隙灯下虹膜小结节，无症状 |
| **NF2** | **青少年白内障**（juvenile cataract）| "2 眼"；年轻人白内障想到 NF2 |
| **VHL** | **视网膜血管母细胞瘤**（retinal hemangioblastoma）| 眼底高血管病灶，可出血 |
| **SWS** | **青光眼**（glaucoma）+ 巩膜外层血管瘤 | V1 葡萄酒斑同侧，**新生儿必查眼压** |
| **TSC** | 视网膜错构瘤（astrocytic hamartoma）| 不如前四个高频 |

### 5.3 影像 / 体格 buzzword

```
SWS  →  "tram-track" gyriform 钙化（轨道征，CT/平片）+ V1 葡萄酒色斑
TSC  →  SEGA 长在孟氏孔(foramen of Monro)旁 → 阻塞 → 脑积水
        灰叶斑用 Wood 灯查；心脏横纹肌瘤产前超声可见
NF1  →  腋窝/腹股沟雀斑(Crowe sign) + 蝶骨发育不良
NF2  →  双侧 CN VIII 桥小脑角(CPA)肿块（MRI 增强）
VHL  →  后颅窝/视网膜高血管结节 + 双肾多发囊肿/肿瘤
```

### 5.4 一句话鉴别钥匙

> [!success] 见到 X → 想到 Y
> - **双侧听神经瘤** → NF2（单侧散发听神经瘤≠综合征）
> - **小脑血管母细胞瘤 / 红细胞增多** → VHL → 查腹部肾/胰/嗜铬
> - **婴儿痉挛 + 灰叶斑 / 心脏横纹肌瘤** → TSC
> - **V1 葡萄酒色斑 + 癫痫 / 青光眼** → SWS（散发）
> - **≥6 咖啡斑 + 腋窝雀斑 + Lisch** → NF1
> - **遗传方式**：四个 AD（NF1/NF2/TSC/VHL），**只有 SWS 散发**

## 六、诊断标准（NF1 / TSC）⭐

> [!info] 为什么单列
> CK 偶尔不给"诊断名"，而是堆体征让你**数够不够阈值**才能下诊断（典型 stem：娃有 5 个咖啡斑 + 1 个一级亲属 → 够不够 NF1？）。NF1 与 TSC 有成文阈值，**SWS / NF2 / VHL 多靠典型组合 + 基因/影像**，不背数。

### 6.1 NF1 诊断（NIH 标准：≥2 / 7 条）

> [!tip] NF1 七条（满足 **≥2** 条即诊断）
> 1. **≥6 个咖啡牛奶斑 café-au-lait macules**（青春期前 >5 mm / 青春期后 >15 mm）
> 2. **≥2 个任意型神经纤维瘤 neurofibroma**，或 **1 个丛状神经纤维瘤 plexiform neurofibroma**
> 3. **腋窝或腹股沟雀斑**（axillary/inguinal freckling = **Crowe 征**）
> 4. **视路胶质瘤 optic glioma**
> 5. **≥2 个 Lisch 结节**（虹膜色素性错构瘤 iris hamartoma）
> 6. **特征性骨病变**（蝶骨发育不良 sphenoid dysplasia，或长骨皮质变薄 ± 假关节 pseudarthrosis）
> 7. **一级亲属**（父母/同胞/子女）确诊 NF1
>
> 记忆：和助记词 **CICLOPSS** 同一批体征，只是加了"数量阈值 + 家族史"。2021 更新版另把 **脉络膜异常**（OCT 光学相干断层扫描可见）和 **致病性 NF1 基因变异** 也列入，但 CK 考经典 7 条。

### 6.2 TSC 诊断（2012/2021 标准：2 主 或 1 主 + ≥2 次）

> [!tip] 确诊规则
> **确诊 (definite)** = **2 条主要**，或 **1 条主要 + ≥2 条次要**；
> 另：**单独检出致病性 TSC1/TSC2 基因变异** 即可确诊（基因优先）。

| 主要特征 major（11 项）| 次要特征 minor（6 项）|
|---|---|
| ① 灰叶斑 hypomelanotic macules（≥3，≥5 mm）| ① 五彩纸屑样皮损 "confetti" |
| ② 面部血管纤维瘤 angiofibroma（≥3）或前额纤维斑块 | ② 牙釉质点状凹陷 dental enamel pits（≥3）|
| ③ 甲周纤维瘤 ungual fibroma（≥2）| ③ 口腔内纤维瘤 intraoral fibroma（≥2）|
| ④ 鲨革斑 shagreen patch | ④ 视网膜无色斑 retinal achromic patch |
| ⑤ 多发视网膜错构瘤 retinal hamartoma | ⑤ 多发肾囊肿 renal cysts |
| ⑥ 皮质结节/放射状迁移线 cortical tuber | ⑥ 非肾错构瘤 nonrenal hamartoma |
| ⑦ 室管膜下结节 subependymal nodule（≥2）| |
| ⑧ **SEGA**（室管膜下巨细胞星形细胞瘤）| |
| ⑨ 心脏横纹肌瘤 cardiac rhabdomyoma | |
| ⑩ 肺淋巴管平滑肌瘤病 LAM＊ | |
| ⑪ 肾血管平滑肌脂肪瘤 angiomyolipoma（≥2）＊ | |

> [!warning] 易错点
> ＊ **LAM + 肾 AML 同时存在时，只算 1 条主要**（避免重复计数）——CK 偶尔卡这个。其余主要特征各自独立计数。

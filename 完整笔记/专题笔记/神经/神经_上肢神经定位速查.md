# 上肢神经定位速查 (Upper Limb Nerve Localization)

> [!tip] 一句话总览
> 上肢考点两条主线：**① 骨折/脱位部位 → 哪条神经**（按部位，见 [[完整笔记/专题笔记/骨科/骨科_上肢骨折神经损伤定位]]）；**② 神经干本身 → 无力+麻+招牌手畸形**（本表，按神经干，与 [[完整笔记/专题笔记/神经/神经_下肢神经定位速查]] 对称）。卡压治疗阶梯见 [[完整笔记/专题笔记/神经/神经_周围神经卡压治疗阶梯]]。

> [!success] 晕了先看这张（考试骨架，只记两句）⭐⭐⭐
> **拇指侧 = 正中，小指侧 = 尺，伸不动 = 桡；爪 = 低位，握拳露馅 = 高位。**
> 下面所有表 / LOAF / 悖论都是这两句的展开，答不出来时再往下翻。

## §一 主干定位表（哪条神经 → 哪块肌无力 + 哪片麻 + 招牌 stem）⭐

| 神经 | 根 | 运动（无力表现）| 感觉（麻区）| 招牌损伤 / stem 钩子 |
|---|---|---|---|---|
| **肌皮神经 Musculocutaneous** | C5–C7 | **屈肘（肱二头肌 biceps + 肱肌）** + 旋后辅助 | 前臂**外侧**（前臂外侧皮神经 lateral antebrachial cutaneous）| 罕见；屈肘无力 + 前臂外侧麻；二头肌反射↓ |
| **腋神经 Axillary** | C5–C6 | **外展（三角肌 deltoid，15–90°）** + 小圆肌 | 肩**外侧**"军徽 regimental badge"小片 | **肱骨外科颈骨折 / 前肩脱位**；⚠️ 感觉丢=腋神经，感觉完整=肩袖 |
| **桡神经 Radial** | C5–T1 | **伸腕 + 伸指 + 伸肘（肱三头肌）+ 旋后** → **wrist drop 垂腕** | 手背桡侧 / **虎口背侧 (first webspace)** | **肱骨干中段 (radial groove) / Saturday night palsy / 止血带** |
| **正中神经 Median** | C6–T1 | 前臂旋前 + 屈腕 + **桡侧指屈** + **拇对掌/外展（大鱼际 APB 拇短展肌）** | 掌侧**桡侧 3½ 指** + 大鱼际（掌皮支）| **腕管 CTS / 肱骨髁上骨折 / Colles**；招牌手 = 猿手 + 誓言手 |
| **尺神经 Ulnar** | C8–T1 | **手内在肌（骨间肌 + 3/4 蚓状肌 + 小鱼际 + 拇内收）** + 尺侧腕屈 + 指深屈肌尺半 (flexor digitorum profundus, FDP 4/5) | **尺侧 1½ 指**（小指 + 无名尺半）+ 掌背皮支 | **内上髁骨折 / 肘管 / Guyon 管**；招牌手 = 爪形手 + Froment 征 |
| **胸长神经 Long thoracic** | C5–C7 | **前锯肌 (serratus anterior)** | 无 | **翼状肩胛 (winged scapula)**；腋窝手术（乳腺/淋巴结清扫）、背包压迫 |

*臂丛 (brachial plexus) 全图：根→干→股→束→终末支（肌皮/腋/桡/正中/尺 + 胸长神经）。上肢所有神经的"出厂总图"。*

> [!info] 臂丛五级 + 四大损伤（FA 补充）
> 五级顺序口诀：**Roots → Trunks → Divisions → Cords → Branches**（"Randy Travis Drinks Cold Beer"）。
>
> | 损伤 | 部位 | 机制 | 表现 |
> |---|---|---|---|
> | **Erb 麻痹**（waiter's tip 侍者手）| **上干 C5–C6** | 分娩侧向牵拉 / 成人肩颈暴力分离 | 臂内收内旋垂于体侧、"讨小费"位；三角肌/冈上下肌/肱二头肌弱 |
> | **Klumpke 麻痹** | **下干 C8–T1** | 分娩上举牵拉 / 抓物坠落 | **爪形手 (claw hand)**：手内在肌瘫；± Horner |
> | **胸廓出口综合征 TOS (thoracic outlet syndrome)** | 下干 + 锁骨下血管 | 颈肋 / Pancoast 瘤 / 前斜角肌 | 手内在肌萎缩 + 血管受压（缺血/水肿）|
> | **翼状肩胛 winged scapula** | **胸长神经 (long thoracic) C5–C7** | 腋窝淋巴清扫 / 刺伤 | 前锯肌瘫 → 推墙肩胛翘起、臂不能过头外展 |

*手部皮节 (dermatomes C6/C7/C8) vs 周围神经皮支 (median/ulnar/radial) 对照——区分**神经根病**（按皮节）vs **周围神经卡压**（按神经分布）。*

> [!success] 手内在肌口诀（FA 金牌，判正中 vs 尺）⭐
> - **大鱼际 = 正中（返支）**：**LOAF** = **L**umbricals（桡侧 1/2 蚓状肌）+ **O**pponens pollicis 拇对掌肌 + **A**bductor pollicis brevis 拇短展肌 + **F**lexor pollicis brevis 拇短屈肌浅头。功能都是 **OAF = Oppose / Abduct / Flex（对掌/外展/屈）**。
> - **小鱼际 = 尺神经，与大鱼际"同名镜像"，也是 OAF**：**O**pponens digiti minimi + **A**bductor digiti minimi + **F**lexor digiti minimi brevis。两组肌一一对应（pollicis 拇 ↔ digiti minimi 小指），功能同样是 **Oppose / Abduct / Flex**——**一套 OAF 背两个鱼际**，只是神经分家：**大鱼际归正中、小鱼际归尺**。
> - **尺神经手内在肌 = 除大鱼际外全部**：小鱼际三肌（OAF）+ 骨间肌 + 3/4 蚓状肌 + 拇内收 + FPB 深头。
> - **骨间肌口诀**：**DAB = Dorsals ABduct（背侧骨间外展）**；**PAD = Palmars ADduct（掌侧骨间内收）**——都归**尺神经**。
> - **蚓状肌**：1/2（桡侧）= 正中、3/4（尺侧）= 尺；功能 **屈 MCP + 伸 PIP/DIP**（瘫 → 伸不直 = 爪，见 §二"远亲伸不直"）。
> 一句话：**拇指的"精细外展/对掌"归正中（LOAF）；手指的"开合（DAB/PAD）"归尺。两个鱼际都是 OAF，只是分属正中/尺。**

## §二 手部三神经的招牌畸形 / 手势 ⭐⭐（最高频）

上肢的"foot drop 级"高频点 = **正中 / 尺 / 桡三条各有招牌手**，靠"**哪个动作做不出**"一眼定位：

| 神经 | 招牌畸形 / 手势 | 关键动作缺失 | 感觉丢失区 |
|---|---|---|---|
| **正中 Median**（高位）| **猿手 (ape hand)**：拇不能对掌，拇贴于掌平面；**誓言手 (hand of benediction)**：试握拳时 **2/3 指伸直** | 拇对掌/外展、2-3 指屈 | 桡侧 3½ 指掌面 + 大鱼际（低位腕管则**大鱼际皮肤保留**）|
| **尺 Ulnar**（低位/腕）| **爪形手 (claw hand)**：4/5 指掌指关节过伸 + 指间屈；**Froment 征**+ | 手指内收/外展、拇内收 | 尺侧 1½ 指 |
| **桡 Radial** | **wrist drop 垂腕** | 伸腕 + 伸指 | 手背桡侧 / 虎口背侧 |

> [!tip] 为什么表里正中挑"高位"、尺挑"低位"？（招牌表 = 最佳出片角度）⭐
> 这张"招牌表"给每条神经挑**画面最全 / 最典型**的那个损伤高度，不是漏了另一半：
> - **正中挑高位**：高位（髁上/前臂）**同时**废大鱼际 + 长屈肌 → **猿手 (ape hand) + 誓言手 (hand of benediction) 一次集齐**；低位（腕管 CTS）只废大鱼际 → **只有猿手、没有誓言手**（长屈肌保留），静息可见 median claw，且**大鱼际皮肤保留**（掌皮支绕过腕横韧带）。低位 = 减配版。
> - **尺挑低位**：低位（腕/Guyon）FDP 完好 → 爪**最丑最典型**；高位（肘）连 FDP 4/5 也废 → **爪反而变轻（悖论）**，改以"握不成拳 + 屈腕无力"为主。
>
> 补全另一半（与下方"FA 四手畸形表"对齐）：
>
> | | 正中低位（腕管 CTS）| 尺高位（肘 / cubital）|
> |---|---|---|
> | 招牌 | 猿手（**无**誓言手）+ 静息 median claw | 爪较轻（悖论）+ 握不成拳 |
> | 关键缺失 | 拇对掌；手指仍能屈 | 屈腕无力 (FCU)；4/5 末节屈不动 |
> | 感觉 | 桡侧 3½ 指掌面，**大鱼际保留** | 尺侧 1½ 指（同低位）|

> [!danger] 尺神经悖论 (ulnar paradox) — 反直觉高频
> **病变越高（肘），爪形手反而越轻**；**病变越低（腕/Guyon），爪形手越明显**。
> 原因：肘部损伤 = 支配 4/5 指的**指深屈肌 (FDP) 也瘫** → 末节屈不起来 → 爪不明显；腕部损伤 = FDP 完好 → 4/5 指末节仍强力屈 + 掌指过伸 → 爪最典型。
> 一句话：**"爪越丑，病越低"。**

> [!info] FA 四手畸形表：远端"静息爪" vs 近端"用力才现" ⭐
> 同一条神经，**远端损伤在静息就爪**（残留长屈肌还在拉），**近端损伤要做动作才暴露**（长屈肌也瘫、静息反而不爪——即上面的悖论）：
>
> | 手征 | 何时出现 | 定位 | 累及指 |
> |---|---|---|---|
> | **Ulnar claw 尺爪手** | **伸指/静息**时 | **远端尺神经**（腕/Guyon）| 4/5 指 |
> | **Trouble making a fist 握不成拳** | **试握拳**时 | **近端尺神经**（肘）| 4/5 指 |
> | **Median claw 正中爪手** | **伸指/静息**时 | **远端正中**（腕管/掌部裂伤）| 2/3 指 |
> | **Hand of benediction 誓言手** | **试握拳**时 | **近端正中**（髁上/前臂）| 2/3 指 |
>
> 记忆：**远端 = 静息就爪（claw at rest）；近端 = 做动作才现（benediction / 握不成拳）**。尺累及 4/5 指、正中累及 2/3 指。**大鱼际萎缩 → 正中；小鱼际萎缩 → 尺。**
>
> ⭐ **口诀：「近亲弯不动，远亲伸不直」**
> - **近亲（proximal 近端）弯不动** → 近端连 **FDS/FDP 屈肌**一起废 → **握拳时弯不下去** → *benediction 誓言手 / 握不成拳*
> - **远亲（distal 远端）伸不直** → 只废**蚓状肌**（管伸 PIP/DIP）→ 静息指间关节勾着、**伸不直** → *claw 爪*
> - 正好解释悖论：远端只丢蚓状肌 → 静息爪重；近端连屈肌带走 → 静息反不勾、握拳才露馅。

> [!warning] 外展无力：腋神经 vs 肩袖（别混）
> 脱位/外伤后**外展无力**两种都可能：**肩外侧感觉丢 = 腋神经损伤**；**感觉完整 + 中老年 = 肩袖损伤 (rotator cuff)**（选 MRI）。一句话：**外展无力先问"肩外侧麻不麻"。**

![[骨科_正中神经分支树_长屈肌分水岭.svg]]
*正中神经分支树：长屈肌群是"誓言手 vs 猿手"的分水岭——高位（其上游）→ 屈指屈拇全废 = 誓言手 + 不能 OK 圈；低位 CTS 在腕管（下游）→ 长屈肌保留、手指能屈，只丢大鱼际 = 猿手；掌皮支腕管近端分出 → CTS 大鱼际掌侧保留。*

## §三 主干 vs 纯运动分支（分不清就漏定位）

上肢有两条"纯运动"深支，损伤时**只有无力、没有感觉丢失**——这是区分近端 vs 分支的钥匙：

```
桡神经 (radial)
├─ 主干损伤（肱骨干中段）= wrist drop 垂腕 + 虎口背侧麻（有感觉丢）
└─ 后骨间神经 PIN (posterior interosseous, 纯运动)
     = finger drop 掌指关节伸不起、但腕仍能背伸(偏桡)、无感觉丢

正中神经 (median)
├─ 主干损伤（腕管/髁上）= 拇对掌弱 + 桡侧 3½ 指麻（有感觉丢）
└─ 前骨间神经 AIN (anterior interosseous, 纯运动)
     = 不能做 OK 手势(FPL 拇长屈 + FDP 2/3 + 旋前方肌)、无感觉丢
```

> [!tip] 记忆
> **"能屈腕/伸腕但精细动作丢 + 感觉正常" → 想纯运动深支（AIN / PIN）**。AIN = 不能 OK 手势；PIN = finger drop（伸指不能、伸腕保留）。

![[骨科_桡神经分支树_ECRL分水岭.svg]]
*桡神经分支树：ECRL（桡侧腕长伸肌 extensor carpi radialis longus）是"掉腕 vs 掉指"的分水岭——高位（肱骨干，ECRL 上游）→ 连伸腕都瘫 = wrist drop + 虎口背麻；低位 PIN（下游）→ ECRL 保留、只掉指 = finger drop + 伸腕桡偏 + 手不麻。与正中神经"长屈肌分水岭"完全对称。*

## §四 上肢卡压综合征（治疗见阶梯专题）

| 综合征 | 神经 / 部位 | 钩子 | 一线处理 |
|---|---|---|---|
| **腕管综合征 CTS** | 正中 ＠腕横韧带 | 桡侧 3½ 指夜间麻、甩手缓解；孕/甲减/肢端肥大 | **夜间腕夹板** ± NSAIDs |
| **肘管综合征 cubital tunnel** | 尺 ＠肘 | 4/5 指麻 + 手内在肌弱；**屈腕无力**；leaning on elbow | 避屈肘 / **伸肘夹板** |
| **Guyon 管综合征** | 尺 ＠腕 | 4/5 指麻 + 手内在肌弱；**屈腕正常**；handlebar palsy | 避压迫 / 护腕 |
| **桡神经麻痹** | 桡 ＠桡神经沟 | wrist drop + 虎口背麻；Saturday night palsy | **腕背伸夹板** + 观察（多自愈）|

> [!tip] 尺神经「肘 vs 腕」一句话区分
> **屈腕无力 = 肘部 (cubital)**；**屈腕正常 = 腕部 (Guyon)**——支配尺侧腕屈肌 (FCU) 的分支在肘以下发出，腕部卡压绕过它。完整阶梯 + CTS 详解见 [[完整笔记/专题笔记/神经/神经_周围神经卡压治疗阶梯]]。

> [!info] 神经–血管伴行"例外"（FA：同一部位神经伤 → 想到哪根动脉）⭐
> 多数神经与同名动脉伴行，下列**不同名**的组合是考点（骨折/损伤时神经 + 血管一起查）：
>
> | 部位 | 神经 | 伴行动脉 |
> |---|---|---|
> | 腋窝 / 外侧胸壁 | 胸长 Long thoracic | 胸外侧 Lateral thoracic |
> | 肱骨外科颈 | 腋 Axillary | 旋肱后 Posterior circumflex |
> | **肱骨干中段** | **桡 Radial** | 肱深 Deep brachial |
> | **肘窝 / 肱骨远端** | **正中 Median** | **肱 Brachial**（髁上骨折同时威胁）|
> | 腘窝 | 胫 Tibial | 腘 Popliteal |
> | 内踝后方 | 胫 Tibial | 胫后 Posterior tibial |

## §五 自测复习题

> [!question]- Q1. 醉酒压手臂一夜后垂腕 (wrist drop) + 虎口背侧麻。哪条神经、哪一段？为什么不是 PIN？
> **桡神经 ＠桡神经沟（肱骨干中段）**，Saturday night palsy。**有感觉丢**（虎口背侧）→ 是主干；PIN 是纯运动、无感觉丢、且 wrist drop 不明显（只 finger drop）。

> [!question]- Q2. 患者 Guyon 管卡压，爪形手很典型；另一患者肘部尺神经损伤，爪形手反而轻。为什么？
> **尺神经悖论**：肘部损伤连 FDP 4/5 也瘫 → 末节屈不起来 → 爪不明显；腕部 FDP 完好 → 4/5 指末节强屈 + 掌指过伸 → 爪最典型。"爪越丑，病越低"。

> [!question]- Q3. 儿童肱骨髁上骨折后不能做 OK 手势，但手部感觉正常。定位？还要立刻查什么？
> **前骨间神经 AIN（正中神经纯运动分支）**——不能屈拇末节 + 屈食指末节。⚠️ 髁上骨折必查 **肱动脉 (brachial artery) / 桡动脉搏动**（Volkmann 挛缩风险）。

> [!question]- Q4. 前肩脱位复位后外展无力 + 肩外侧一小片麻。神经？若感觉完整、患者 60 岁又该想什么？
> **腋神经 (axillary)**（军徽区感觉丢）。若**感觉完整** + 中老年 → **肩袖损伤 (rotator cuff)**，选 MRI。

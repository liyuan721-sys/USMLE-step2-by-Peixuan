# 神经_失语综合征定位（Aphasia Localization）

> [!info] 一句话定位
> **Aphasia（失语）= 优势半球（dominant hemisphere，多为左）语言网络受损**。定位三步：**①流利度 → ②理解 → ③复述**。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q3738]]（Broca 失语误选优势顶叶）。

## 一、三步定位法

```
第 1 步：流利吗？
   非流利（费力/电报式）── 前部病变（额叶 Broca 一带）
   流利（语量多）──────── 后部病变（颞顶 Wernicke 一带）

第 2 步：听得懂吗（理解）？
   理解好 ── Broca / 传导 / 经皮质运动
   理解差 ── Wernicke / 完全 / 经皮质感觉

第 3 步：能复述吗？
   复述差 ── Broca / Wernicke / 传导 / 完全
   复述保留 ── 经皮质类（transcortical，病变在分水岭、绕过核心语言区）
```

## 二、复述回路详解（为什么"复述"能定位）⭐

> [!info] 复述 = 一条完整回路
> 复述（repetition）= 医生说一句话（经典测试句 "No ifs, ands, or buts"），让病人**原样重复**。这个动作要求下面这条回路**全程完整**，任何一环断掉，复述就坏：

```
听到声音
   ↓
Wernicke 区（听懂 / 理解）          ← 优势颞上回后部
   ↓
弓状束 Arcuate fasciculus（把"听到的"传给"说出的"）  ← 白质连接束
   ↓
Broca 区（组织语言、说出来）        ← 优势额下回后部
   ↓
说出来
```

> [!tip] 关键洞察：复述为什么能一刀切分两大类
> Wernicke 区、弓状束、Broca 区**全都环绕在外侧裂 Sylvian fissure 周围**，供血来自 **MCA（middle cerebral artery，大脑中动脉）核心区**。
> - 梗死在 **MCA 核心** → 直接打到回路上的某个结构 → **复述坏**（Broca / Wernicke / 传导 / 完全）
> - 梗死在 **分水岭 watershed 区**（MCA 与 ACA（anterior cerebral artery，大脑前动脉）/ PCA（posterior cerebral artery，大脑后动脉）交界的最外围）→ 核心语言区**没被打到，只是被"孤立"** → **复述反而保留** = 经皮质类 transcortical

> [!question]- 为什么"失语了"复述却能保留？——两层同心圆模型
> 关键：**"语言"≠"复述"**，它们走**不同通路**。把语言系统想成两层同心圆：

```
        ┌─────────────────────────────────────┐
        │  外圈 = 联合皮质 association cortex   │
        │  （思想 / 概念 / 意义 / 记忆 / 想说什么）│
        │                                       │
        │      ┌───────────────────────┐        │
        │      │  内圈 = 核心语言环       │        │
        │      │  perisylvian core       │        │
        │      │  Wernicke→弓状束→Broca  │        │
        │      │  （听→懂→传→说 的回路）  │        │
        │      └───────────────────────┘        │
        │              ↑ 桥梁 bridge ↑           │
        └─────────────────────────────────────┘
```

> [!question]-
> - **内圈（核心环）**：只干一件事 —— 把听到的声音**原样转成说出的声音**。这就是复述的全部通路；由 **MCA 核心区**供血。
> - **外圈（联合皮质）**：负责**意义** —— 把词连到概念/想法/记忆（听懂含义、产生想说的内容）。
> - **两圈之间有桥**。
>
> **分水岭 = MCA 与 ACA/PCA 交界带，正好落在外圈这一圈**；血压骤降/缺氧时最先饿死的就是它。
> → 结果：**内圈核心环完好，但它通往外圈意义系统的桥断了** → 核心环被"孤立"（isolation of the speech area）。
> → 内圈还能自己转圈（**复述保留**），却拿不到外圈的意义（**理解或自发说话坏了** = 失语）。这就是"能鹦鹉学舌、语言却是坏的"看似矛盾组合的来源。
>
> **三亚型 = 桥在哪一边断**：
> - 经皮质**运动** = Broca↔额叶（启动/想法）桥断 → 不主动说，但听得懂
> - 经皮质**感觉** = Wernicke↔语义桥断 → 词进得来连不上意义，听不懂、说话空洞
> - **混合**经皮质 = 两桥全断只剩内圈孤岛 → 啥都不懂也说不出，**只会重复你的话（echolalia 模仿言语）**
>
> **镜像对照**：传导性 Conduction 病灶在**弓状束（内圈里的一段）** → 内圈断、桥好 → **复述坏、其余好**；经皮质 = 内圈好、桥断 → **复述好、其余坏**。

> [!success] 经皮质类（Transcortical）= "复述穿得过去"
> **名字由来**：病变在**皮质外周（分水岭）**，把中央语言核心区**绕过去了（spared）**，核心回路内部完整 → 复述做得到。
> **记忆钥匙**：带 **trans-** 的 = 复述 **trans（穿过）**得过去 = **复述保留**。三个亚型各是某个经典型的"翻版"，唯一区别就是复述好：
>
> | 经皮质亚型 | 像哪个经典型 | 理解 | 流利度 | 复述 |
> |---|---|---|---|---|
> | Transcortical **motor** 经皮质运动 | 像 Broca | 好 | 不流利 | **保留** |
> | Transcortical **sensory** 经皮质感觉 | 像 Wernicke | 差 | 流利 | **保留** |
> | **Mixed** transcortical 混合经皮质 | 像 Global | 差 | 不流利 | **保留** |
>
> 典型病史钩子：**心脏骤停 / 严重低血压 / CO（carbon monoxide，一氧化碳）中毒 / 缺氧**后醒来讲不清话 → 分水岭低灌注 → 想经皮质类（尤其 mixed）。

> [!danger] 别和传导性（Conduction）搞反 —— 方向正好相反
> - **传导性**：病灶在**弓状束（中间连接线断了）** → 理解好、说话流利，但**复述特别差（孤立性、disproportionate）**；病人**知道自己说错、拼命想纠正**（conduit d'approche）。= "线断了，传不过去"。
> - **经皮质类**：病灶在**外周分水岭**，回路完整 → **复述好**。= "核心区完好，只是被孤立"。
>
> stem 写 **"repetition disproportionately impaired"**（复述差得离谱、理解和流利都还行）→ 锁死 **Conduction**；stem 写 **"repetition intact / preserved"** → 锁死 **经皮质三兄弟之一**。

> [!question]- 第 3 步在整套定位里的意义
> 先用**复述**把"经皮质三兄弟 + 传导性"从一堆失语里**劈出来**，再回到第 1 步流利度 + 第 2 步理解，定具体是哪一个。复述是整张表**信息量最大的单一动作**。

## 三、五型失语横向对比 ⭐

| 失语类型 | 流利度 | 理解 | 复述 | 定位 | 伴随 |
|---|---|---|---|---|---|
| **Broca**（表达性 expressive）| **非流利**、费力 | **好** | 差 | 优势**额下回后部**（inferior frontal gyrus）| **右侧肢体无力**（邻初级运动皮质）|
| **Wernicke**（感受性 receptive）| 流利、**无意义** | **差** | 差 | 优势**颞上回后部**（superior temporal gyrus）| 上象限盲（视辐射）、无明显瘫 |
| **Conduction**（传导）| 流利 | 好 | **差（孤立性）** | **弓状束**（arcuate fasciculus）| phonemic 错语、频繁自我纠正 |
| **Global**（完全）| 非流利 | 差 | 差 | 大面积优势 MCA（middle cerebral artery，大脑中动脉）| 偏瘫 |
| **Transcortical**（经皮质）| 视亚型 | 视亚型 | **保留** | **分水岭区**（watershed） | 常见于低灌注/缺氧后 |

> [!tip] 复述是钥匙
> **复述保留** → 经皮质类（病灶绕开核心语言区）；**复述受损** → Broca/Wernicke/传导/完全。

## 四、血管定位（MCA 分支）⭐

- **Broca 区** = 优势额叶，由 **MCA 上分支（superior division）** 供血 → 上分支卒中 = Broca 失语 + 面臂为主无力。
- **Wernicke 区** = 优势颞叶，由 **MCA 下分支（inferior division）** 供血 → 下分支卒中 = Wernicke 失语，常无明显瘫。
- **完全性失语** = MCA 主干大面积梗死。

## 五、易混陷阱（普适）

- **优势顶叶 dominant parietal** = 对侧皮质感觉缺失 + **Gerstmann 综合征**（失算 acalculia / 失写 agraphia / 左右失认 / 手指失认），**不是流利度问题**（错题钉死点）。
- **非优势半球（nondominant）** = **失韵律 aprosodia**（说话/理解情感语调差）+ 忽视 neglect，非经典失语。
- Broca 失语患者**能理解并执行指令**，但**说与写**费力 —— 别因"speech difficulty"泛指任意皮质区。

## Memory Hook

> [!success]
> - **B**roca = **B**roken speech（说话费劲）但**听得懂** = 优势**额**叶（前=输出）+ 右侧无力
> - Wernicke = **流利胡说听不懂** = 优势**颞**叶（后=理解）
> - **复述保留 = 经皮质**（分水岭，绕过语言核心）
> - 孤立性复述差 + 流利 + 理解好 = **传导（弓状束）**

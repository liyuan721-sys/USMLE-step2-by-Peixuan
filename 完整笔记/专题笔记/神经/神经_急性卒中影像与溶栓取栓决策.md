# 急性卒中影像与溶栓取栓决策 (Stroke Imaging → tPA / Thrombectomy)

> [!info] 用途与定位
> 急性缺血性卒中 (acute ischemic stroke, 脑梗死) 的**影像选择 (CT vs MRI vs CTA vs CTP)** + 由影像驱动的**溶栓 / 取栓决策**。源自用户对话"CT、MRI 在脑梗中的作用"。
> 本卡 = **影像 + 再灌注通用决策**；DOAC/抗凝叠加溶栓的逆转分叉、抗凝逆转总表、恶性半球梗死深挖见姊妹篇 [[完整笔记/专题笔记/神经/神经_急性卒中_DOAC决策]]（不重复）。

> [!note]- 缩写对照（全称 中英）
> - **CT** = computed tomography 计算机断层扫描 ｜ **NCCT** = non-contrast CT 平扫 CT
> - **MRI** = magnetic resonance imaging 磁共振成像
> - **DWI** = diffusion-weighted imaging 弥散加权成像 ｜ **ADC** = apparent diffusion coefficient 表观弥散系数
> - **FLAIR** = fluid-attenuated inversion recovery 液体衰减反转恢复序列
> - **CTA** = CT angiography CT 血管成像 ｜ **CTP** = CT perfusion CT 灌注成像
> - **LVO** = large vessel occlusion 大血管闭塞
> - **tPA** = tissue plasminogen activator 组织型纤溶酶原激活物（阿替普酶 alteplase）
> - **sICH** = symptomatic intracranial hemorrhage 症状性颅内出血
> - **NIHSS** = NIH Stroke Scale 美国国立卫生院卒中量表 ｜ **ASPECTS** = Alberta Stroke Program Early CT Score
> - **MCA** = middle cerebral artery 大脑中动脉 ｜ **ICA** = internal carotid artery 颈内动脉

---

## 〇、一句话核心：两种影像，两个任务

> [!success] 不是"谁更好"，是分工不同
> - **平扫 CT (NCCT) = 急诊第一步**，任务 = **排除出血 (rule out hemorrhage)** → 决定能不能溶栓。**不是为了"看到"梗死**。
> - **MRI-DWI = 最敏感**，任务 = **早期确诊/定位缺血**（数分钟即阳性）。
> - **CTA = 找 LVO** → 决定能不能取栓 ｜ **CTP = 核心 vs 半暗带** → 决定延长窗能不能取栓。

---

## 一、平扫 CT (Non-contrast CT) —— 永远的第一步

> [!danger] 为什么 first（高频考点）
> 急性卒中第一步**永远是平扫 CT**，唯一急诊目的：**区分缺血 vs 出血**。
> - **出血**：立刻 **高密度 (hyperdense)** → 一眼可见 → 溶栓**禁忌**。
> - **缺血**：**早期 6–24 小时常阴性/正常**（最大坑：**CT 阴性 ≠ 没梗，更不能因此排除卒中**）。
> → 排除出血后才能在 **4.5 小时**窗口给 tPA。

| 早期缺血 CT 征象 | 含义 |
|---|---|
| 灰白质分界消失 (loss of gray-white differentiation) | 细胞毒性水肿早期 |
| **大脑中动脉高密度征 (hyperdense MCA sign)** | 血管内血栓 = 栓塞性闭塞 |
| 岛带征 (insular ribbon sign) | 岛叶皮质缺血 |
| 脑沟消失 (sulcal effacement) / 基底节模糊 | 局部水肿 |

- 完全形成的**低密度 (hypodense)** 梗死灶通常 **24–72 小时**才清晰。
- **ASPECTS 评分**：在 NCCT 上把 MCA 区分 10 分，早期缺血改变越多扣分越多；**ASPECTS <6（>1/3 MCA 区受累）→ 大核心梗死 → tPA/取栓获益↓、出血转化↑**。

---

## 二、MRI（重点 DWI）—— 最敏感、确诊与定位

> [!tip] DWI + ADC
> - **数分钟内（3–30 min）**即显示急性缺血 → **早期脑梗最敏感**。
> - 机制：细胞毒性水肿 → 水分子弥散受限 → **DWI 高信号 + ADC 低信号**（"DWI 亮、ADC 暗"= 急性）。
> - **ADC 假正常化**约 1–2 周后出现，可估梗死龄。

**MRI 不可替代的场景：**
- **后循环 / 脑干 / 小脑梗死**：NCCT 受颅后窝骨伪影干扰，MRI 远胜。
- **腔隙性 (lacunar) / 小灶梗死**、**短暂性脑缺血发作 (TIA, transient ischemic attack) 后**确认有无真实梗死。
- **醒后卒中 / 发病时间不明 (wake-up stroke)**：**DWI-FLAIR mismatch**（DWI 已亮、FLAIR 还未亮 ≈ 发病 <4.5h）→ 可据此给 tPA（WAKE-UP 试验）。

> [!warning] 为什么 MRI 不做急诊第一步
> 慢、不普及、有禁忌（起搏器/金属/不配合）→ **急诊定能不能溶栓靠 NCCT**，MRI 用于诊断不清 / 后循环 / 醒后卒中。

---

## 三、CTA 与 CT 灌注 —— 决定取栓

> [!info] 进阶影像（血管内治疗门槛）
> - **CTA**：找 **LVO**（ICA / MCA M1-M2 / 基底动脉 basilar / 椎动脉 vertebral）→ **机械取栓 (mechanical thrombectomy)** 候选。远端小分支（M3）进不去导管 = **不算 LVO**。
> - **CTP**：区分**梗死核心 (core, 已死)** vs **缺血半暗带 (penumbra, 可救)** → 核心小、半暗带大 = 可救组织多 → 把取栓窗延到 **6–24 小时**（DAWN / DEFUSE-3 试验）。

## 四、影像 → 再灌注决策（整合）

### 4.1 静脉溶栓 tPA (alteplase)

| 项 | 要点 |
|---|---|
| **时间窗** | 发病 **≤4.5 小时**（部分 ≤3h 标准窗）；醒后卒中靠 DWI-FLAIR mismatch |
| **前提** | NCCT **无出血** + 无禁忌 |
| **血压门槛** | 溶栓前必须 **<185/110**（拉贝洛尔 labetalol / 尼卡地平 nicardipine 降压达标才给）；溶栓后维持 **<180/105** ×24h |
| **关键禁忌** | 近期颅内出血/手术/大创伤、活动性出血、血小板 <100k、INR >1.7、血糖极端、**>1/3 MCA 区大梗死(ASPECTS<6)**、3 个月内卒中/头外伤 |
| **主要风险** | **sICH ~6%** |

> [!tip] DOAC/华法林叠加时的分叉
> 患者在抗凝药上（apixaban/dabigatran/warfarin）→ tPA 能不能给、要不要先逆转 → 见 [[完整笔记/专题笔记/神经/神经_急性卒中_DOAC决策]]（"-xaban 没药救、-gatran 有 Idaru 救" + 抗凝逆转总表）。

### 4.2 机械取栓 Mechanical Thrombectomy

- **不引入溶栓药 → 不依赖凝血 → 抗凝状态不是禁忌**（与 tPA 关键区别）。
- 门槛：**必须 LVO**（CTA 确认）。
- 时间窗：**≤6h 首选**；**6–24h** 需 **CTP / DWI-FLAIR mismatch** 证明有可救半暗带。
- tPA 与取栓**可序贯/并行**（符合两者标准时先 tPA 再取栓）。

### 4.3 不能溶栓 + 非 LVO

→ 无急性再灌注手段：**容许性高血压 (permissive HTN)**（不降至 <220/120 不动）、**抗血小板 (aspirin 阿司匹林)**、二级预防找病因、支持治疗（吞咽评估 / 深静脉血栓 (DVT, deep vein thrombosis) 预防 / 早康复）。详见 DOAC 卡 §六 + "1-3-6-12 rule"。

## 五、出血转化 / 恶性半球梗死（影像随访）

> [!danger] 大面积卒中后"突然变差" → 急查平扫 CT（不是 MRI）
> 大面积 MCA 梗死后快速神经恶化 = **恶性半球梗死 (malignant hemispheric infarction, MHI)**：脑水肿 ± **出血转化 (hemorrhagic transformation)** → 占位/颅高压/脑疝。
> - **下一步影像 = 急查平扫 CT**——快、可及、查急性出血敏感（MRI 对缺血更敏感但慢、不用于急诊定范围）。
> - 处理：渗透治疗（甘露醇/高渗盐水）+ 常需**去骨瓣减压术**。
> - 完整机制 + 错题见 [[完整笔记/专题笔记/神经/神经_急性卒中_DOAC决策]] §十一（派生自 [[mistakes/uworld-mistakes_2026-06#^Q12214]]）。

## 六、决策流（影像驱动）

```
怀疑急性卒中
   │
   ▼
① 平扫 CT (NCCT) ← 永远第一步：排除出血
   │
   ├─ 出血(高密度) → 降压/神外，禁 tPA（抗凝相关 → 逆转，见 DOAC 卡）
   │
   └─ 无出血(缺血/正常)
        │
        ├─ ≤4.5h + 无禁忌 + BP<185/110 → IV tPA(alteplase)
        │
        ├─ 疑 LVO → CTA
        │     └─ LVO(ICA/MCA M1-2/基底/椎) → 机械取栓
        │           ├─ ≤6h 首选
        │           └─ 6-24h → CTP/DWI-FLAIR mismatch 选患者
        │
        └─ 诊断不清 / 后循环 / 醒后卒中 → MRI-DWI(±FLAIR)
              └─ DWI-FLAIR mismatch → 仍可 tPA
```

## 七、Memory Hooks

> [!success] 一句话锁定
> - **CT 先排血、MRI 才看梗**（CT 早期阴性正常）。
> - **超急性 CT 正常别排卒中**——临床像 + 窗内无禁忌就给 tPA。
> - **hyperdense MCA sign = 血栓**；**>1/3 MCA / ASPECTS<6 = 大核心 → 别硬溶栓**。
> - **tPA 看时间(≤4.5h)，取栓看血管(LVO)+ 可看到 24h（靠灌注）**。
> - **溶栓前 <185/110、溶栓后 <180/105；不溶栓则容许性高血压**。
> - **后循环/醒后卒中 = MRI 的主场**（DWI-FLAIR mismatch）。

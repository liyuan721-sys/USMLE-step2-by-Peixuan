# 神经_谵妄评估 Delirium（CAM + 诱因导向 workup）

> [!info] 一句话定位
> **Delirium（谵妄）= 急性脑衰竭、内科急症**，常被漏诊。用 **CAM（Confusion Assessment Method，意识模糊评估法）**确认，再按"**最常见诱因优先 + 异常发现导向**"做检查。
> 派生自做对题 [[mistakes/uworld-mistakes_2026-06#^Q4644]]（先 UA）+ [[mistakes/uworld-mistakes_2026-06#^Q21331]]（肺炎征 → 胸片）。

## 一、CAM 诊断（4 特征）⭐

```
谵妄 = CAM 阳性
  必备:  ① 急性起病 / 病程波动
       + ② 注意力不集中 (inattention, 易分心)
  加其一: ③ 思维紊乱 (disorganized thinking)
       或 ④ 意识水平改变 (altered consciousness, 嗜睡/亢奋)
```

> [!tip] 谵妄 vs 痴呆 vs 抑郁
> 谵妄 = **急性 + 波动 + 注意力差 + 意识改变**（可有幻觉、睡眠倒错）；痴呆 = 慢性渐进、意识清；抑郁 = 情绪为主。**痴呆患者出现谵妄时，谵妄可能是急性疾病的唯一表现**（如老人无发热的感染）。

## 二、初始评估（找最常见诱因）

| 步骤 | 内容 |
|---|---|
| 病史 | 疼痛、外伤、物质、脱水、**尿潴留** |
| 查体 | 生命体征、心肺、神经、皮肤 |
| 用药回顾 | **阿片、苯二氮䓬、抗胆碱能药** |
| 化验 | CBC、代谢全套（含 LFT 肝功）、**UA（urinalysis，尿检）**、ECG |

> [!success] 老年谵妄 + 无定位 → 先 UA
> 老年最常见诱因 = **UTI（urinary tract infection，泌尿系感染）**；无局灶/外伤/脑膜刺激时，**UA 优先**（[[mistakes/uworld-mistakes_2026-06#^Q4644]]）。

## 三、异常发现 → 对应进阶检查 ⭐

| 谵妄线索 | 下一步 |
|---|---|
| **发热 + 肺底啰音 + 白细胞↑** | **胸片 chest x-ray（疑肺炎）** ← [[mistakes/uworld-mistakes_2026-06#^Q21331]] |
| 尿检异常 / 老年无定位 | UA → 治 UTI |
| 局灶体征 / 外伤 / 抗凝 | 头 CT（computed tomography）|
| 脑膜刺激征 | LP（lumbar puncture，腰穿）|
| 治疗无反应 / 疑非惊厥性癫痫 | EEG（electroencephalogram，脑电图）|
| 肝病征（黄疸/转氨酶↑）| 血氨 ammonia |

> [!warning] 别一上来就 CT/LP/EEG
> 这些是**进阶**检查：仅在**有局灶/外伤/抗凝（CT）、脑膜刺激（LP）、难治或疑癫痫（EEG）**，或**初始评估阴性/治疗无反应**时才做。先用 UA/血/代谢/药抓最常见诱因。

## 四、Stem 钩子（一对对照）

- 老年急性波动意识障碍 + **无局灶** → **UA**（抓 UTI）。
- 谵妄 + **发热 + 啰音 + WBC↑** → **胸片**（抓肺炎）。
- 关键：谵妄检查**永远跟着异常发现走**，不盲目影像。

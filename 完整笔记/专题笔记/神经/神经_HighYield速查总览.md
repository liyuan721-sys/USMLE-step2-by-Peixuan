## 一、卒中（Stroke）⭐ 最高频

> [!info] 缩写约定（首次给全称）
> TIA = transient ischemic attack（短暂性脑缺血发作）；CT = 计算机断层（computed tomography）；MRI = 磁共振成像；CTA = CT angiography（CT 血管造影）；DSA = digital subtraction angiography（数字减影血管造影）；ICH = intracranial hemorrhage（颅内出血）；SAH = subarachnoid hemorrhage（蛛网膜下腔出血）；CEA = carotid endarterectomy（颈动脉内膜剥脱术）；AFib = atrial fibrillation（房颤）；CCB = calcium channel blocker（钙通道阻滞剂）。

### 1.1 第一步永远是"有没有血"

```
怀疑卒中/TIA
   │
   ▼
① Non-contrast CT head（initial test，先排出血）
   │
   ├── 有血 → 出血性卒中管理（见 1.4）
   │
   └── 无血 → 缺血性卒中
         │
         ├── 发病 <4.5 hr + 致残性缺损 → tPA（IV alteplase）溶栓
         └── 发病 >4.5 hr → Aspirin 抗血小板
```

- **Initial test**：非增强 CT 头（non-contrast CT，先看有无出血）。
- **Most accurate imaging**：MRI（最敏感找缺血灶）+ 急诊 CTA head & neck（找大血管闭塞）。
- **最大可干预危险因素**：高血压 (hypertension)（归因风险 **4×** — 全场之冠）；吸烟 2.5×、糖尿病 (DM) 2×。

> [!danger] tPA（IV alteplase 阿替普酶）时间窗 + 红线禁忌
> **时间窗：症状发作 <3–4.5 hr**。绝对禁忌（照源文件）：
> - CT 见颅内出血 (ICH) / ICH 病史 / 疑蛛网膜下腔出血 (SAH)
> - **近 3 个月内缺血性卒中**
> - 血压 **>185/110 mmHg**
> - 抗凝中 **INR >1.7、PT >15 sec 或 aPTT 升高**
> - 血糖 **<60 mg/dL** 或血小板 **<100,000/mm³**

### 1.2 缺血性卒中二级预防 + 抗血小板阶梯

| 情境 | 下一步 |
|---|---|
| 发病 >4.5 hr（错过溶栓窗）| Aspirin 阿司匹林 |
| 已在用 Aspirin 仍卒中 | 加 dipyridamole 双嘧达莫，**或**停 Aspirin 换 clopidogrel 氯吡格雷 |
| 本框架内 | **不长期 Aspirin + clopidogrel 双联** |

### 1.3 找栓子来源（>50 岁三大方向）

```
缺血性卒中/TIA（年龄 >50）
   ├── 怀疑颈动脉狭窄 → Doppler 超声 → 狭窄 >70% 且 <100% → 颈动脉内膜剥脱术 (CEA)
   ├── 怀疑房颤 (AFib) → 先 EKG（无 P 波）→ EKG 正常仍疑 → Holter / 动态监测
   └── 怀疑瓣膜赘生物 (vegetation) → 超声心动图 (echocardiogram)
```

> [!tip] 反直觉钥匙
> - **若卒中归因于 AFib 或 vegetation → 即使确诊检查未出，也可早启抗凝。**
> - **TIA + 人工瓣膜 (prosthetic valve)** → 先查心源性栓塞来源 → echocardiography。
> - **CEA 手术阈值**：颈动脉狭窄 **>70% 且 <100%**（完全闭塞反而不做）。
> - **<50 岁卒中**找特殊病因：可卡因 (cocaine)、高凝状态、自身免疫 (抗心磷脂 cardiolipin)、VDRL 查梅毒。

### 1.4 出血性卒中 BP 管理（阈值照源文件）

| BP 情境 | 处理 |
|---|---|
| ≤220/120（未到极端范围）| 抬高床头 + 过度通气 + 甘露醇 (mannitol) 类降颅压 |
| >220/120 | Labetalol 拉贝洛尔 / IV nitroprusside 硝普钠；**降幅 ≤25%** |
| BP 严重升高总原则 | **缓慢降**（IV labetalol / IV nicardipine 尼卡地平），避免脑灌注压骤降 |

🧩 房颤/DOAC 抗凝启动时机深挖 → [[完整笔记/专题笔记/神经/神经_急性卒中_DOAC决策]]

### 1.5 血管综合征定位钥匙

| 血管 | 钥匙 |
|---|---|
| ACA | 对侧**下肢**为主无力 + **尿失禁**（双下肢→疑脊髓）|
| MCA | 对侧面/臂无力 + **失语**（优势半球 Broca/Wernicke）/ **忽略**（非优势半球）|
| PCA | 对侧偏盲 ± **黄斑回避** (macular sparing) |
| PICA（外侧延髓）| 眩晕 + 共济失调 + **同侧面部**症状（交叉体征）|
| Basilar | **闭锁综合征 (locked-in)**：只能眨眼/上下看，保留意识理解 |

> [!info] 脑干卒中铁律
> **脑干卒中 = 交叉体征 = 对侧身体偏瘫 + 同侧颅神经受损。**

## 二、头痛（Headaches）

### 2.1 原发性头痛三联鉴别 + 治疗钥匙

| 类型 | 钥匙 | 急性 / 流产 | 预防 |
|---|---|---|---|
| 偏头痛 migraine | 女性、单侧、搏动、恶心呕吐、畏光 ± aura（先兆）| Sumatriptan 舒马普坦（**冠心病 CAD / 心血管病 CVD 禁**，改 ergotamine 麦角胺）| ≥4 次/月或 >12 hr：β 阻滞剂 / topiramate 托吡酯 / amitriptyline 阿米替林 |
| 丛集 cluster | 男性、单侧眶周、**自主症状**（流泪流涕、Horner）、夜间、坐立不安 | **100% 氧气** ± triptan | verapamil 维拉帕米 |
| 紧张 tension | 双侧、束带样压迫、压力相关、无恶心 | NSAIDs | 频发用 amitriptyline |

> [!warning] 偏头痛 + 雌激素避孕的致命陷阱
> **migraine with aura（有先兆偏头痛）+ 含雌激素口服避孕药 (OCP, oral contraceptive pill)** → 卒中风险叠加 → 下一步是**换非雌激素避孕**（宫内节育器 IUD / 纯孕激素）。CDC（美国疾控中心）列为 **Category 4 禁忌**。

> [!tip] 药物过量性头痛 (MOH)
> 近**每日头痛 ≥3 个月** + 规律用急性止痛药（对乙酰氨基酚/阿片/布他比妥）；**晨起头痛**（夜间药效退）。处理 = **停止滥用止痛药 + 启动/加强预防药**。

### 2.2 必须先排器质病的头痛

| 情境 | 钥匙 | 下一步 |
|---|---|---|
| 霹雳样头痛 thunderclap | "一生最痛"、年轻人半夜急诊 | **非增强 CT → 阴性疑诊查 LP**（见三、SAH）|
| 假性脑瘤 pseudotumor cerebri（特发性颅内高压 IIH, idiopathic intracranial hypertension）| 育龄肥胖女、维生素 A 过量、视乳头水肿 | **CT 正常 + LP 开放压↑** → acetazolamide 乙酰唑胺 + 治疗性 LP + 停诱因药 |

## 三、颅内出血（Intracranial hemorrhages）

| 出血 | 钥匙 | 血管 | 下一步 |
|---|---|---|---|
| 硬膜外 epidural | 侧方外伤 + **清醒间期 lucid interval（分钟）** | 脑膜中**动脉**（急症）| 非增强 CT → **手术** |
| 硬膜下 subdural | 老人/酗酒、桥静脉、延迟数日–周 | 桥静脉 | CT |
| 蛛网膜下 SAH | "一生最痛"、脑膜刺激征 | 浆果样动脉瘤 | 见下 ▼ |

> [!danger] SAH 诊断 + 处理流程（钥匙密集）
> ```
> 疑 SAH
>    │
>    ▼ ① 非增强 CT（急性期极敏，但 early 可正常）
>    │
>    ├── CT 阳性 → 确诊
>    └── CT 阴性但高度怀疑 → ② LP 找 xanthochromia 黄变（仅第 1 管有血 = traumatic tap）
>                              │
>                              ▼ ③ CTA / DSA 脑血管造影定位动脉瘤
>                                 │
>                                 ▼ ④ 夹闭 clipping / 弹簧圈 coiling
> ```
> **预防并发症**：尼莫地平 (nimodipine，CCB) 防 **3–14 天血管痉挛**；硫酸镁 (MgSO₄) 防癫痫。
> **并发症时间线**：再出血 (rebleeding)（前 24h 最常见）→ 血管痉挛/迟发缺血（>3 天）→ 脑积水 / 癫痫 / 低钠（抗利尿激素分泌失调 SIADH, syndrome of inappropriate ADH secretion）。
> **动脉瘤定位钥匙**：最常见前交通动脉 (AComm, anterior communicating artery)；**多囊肾 (PKD, polycystic kidney disease) 相关 → 后交通动脉 (PComm, posterior communicating artery)**。

## 四、癫痫与首次发作（Seizures）

### 4.1 癫痫类型 → 首选药

| 类型 | 钥匙 | 首选药 |
|---|---|---|
| 部分/局灶 partial | 局灶定位症状 | carbamazepine 卡马西平 / phenytoin 苯妥英 |
| 失神 absence | 小孩上课"发呆"反复 | ethosuximide 乙琥胺 |
| 失张力/强直/肌阵挛 等全身 | 跌倒 / 强直尿失禁 / 电击样抽动（青少年肌阵挛癫痫 JME, juvenile myoclonic epilepsy）| valproic acid 丙戊酸 / lamotrigine 拉莫三嗪 |

### 4.2 首次发作要不要立刻治（看复发风险）

> [!tip] 首次发作后**立即启动**抗癫痫药的 3 个触发条件
> - 有**癫痫家族史**
> - **发作检查有任何异常**（影像 / 脑电图 EEG, electroencephalography）
> - **首次发作即为癫痫持续状态 (status epilepticus)**

### 4.3 癫痫持续状态 (Status epilepticus) — 急症阶梯

```
定义：≥5 分钟持续抽搐 OR ≥2 次发作间意识未恢复
   │
   ▼ ① ABC + 建 IV + 查指尖血糖（先保命排低糖）
   ▼ ② IV benzodiazepine（lorazepam / diazepam-可直肠 / midazolam-可肌注）— 止当前抽
   ▼ ③ IV 长效抗癫痫药（levetiracetam / fosphenytoin / valproic acid）— 防再抽
   ▼ ④ 仍抽 → 麻醉药持续输注（midazolam / propofol / pentobarbital）
   ▼ ⑤ 抽停但意识不恢复 → 持续脑电图 (continuous EEG) 查非惊厥性持续状态 (nonconvulsive SE, status epilepticus)
```

> [!info] 鉴别钥匙
> - **晕厥 vs 癫痫**：晕厥恢复快、无尿失禁；癫痫有定向障碍 / **咬舌（侧缘）** / 尿失禁。
> - 正确治疗后全身惊厥性持续状态 (GCSE, generalized convulsive status epilepticus) 通常 **10–20 分钟**恢复反应；持续不醒 = 苯二氮䓬残留镇静 or 非惊厥性 SE → 查 EEG。

## 五、痴呆鉴别（Dementia）

> [!info] 第一步：可逆病因 + 结构病因
> 疑认知下降先排可逆（维生素 B12、甲状腺、正常压力脑积水 NPH，见下表）+ **结构性病因用头颅 CT (head CT)**。

| 痴呆 | 招牌钥匙 | 处理 |
|---|---|---|
| 阿尔茨海默病 Alzheimer disease (AD) | 缓慢进行性遗忘、弥漫萎缩、乙酰胆碱↓ | donepezil 多奈哌齐（进展期 memantine 美金刚）|
| 血管性 vascular | **阶梯式 stepwise** 下降、执行功能差、白质高信号 | 控危险因素（高血压 HTN / 糖尿病 DM）± 抗血小板 |
| 额颞痴呆 FTD (frontotemporal dementia) / Pick 病 | 中年、**人格/行为先变**（没分寸）、原发进行性失语 | 同 AD（donepezil / memantine）|
| 路易体痴呆 DLB (dementia with Lewy bodies) | **视幻觉 + 帕金森样**、跌倒、快速眼动睡眠行为障碍 (REM sleep behavior disorder)；认知与帕金森样 (parkinsonism) **同年/1 年内** | donepezil + carbidopa-levodopa；**禁 haloperidol 氟哌啶醇** |
| 克雅病 CJD (Creutzfeldt-Jakob disease，朊病毒)| **快速**痴呆 + **肌阵挛 (myoclonus)** + 癫痫；EEG 周期尖波 | 支持治疗；腰穿 (LP, lumbar puncture) **14-3-3 蛋白** |
| 正常压力脑积水 NPH (normal pressure hydrocephalus) | 步态不稳 + 尿失禁 + 痴呆三联 | LP 后症状改善 → 分流术 |
| 进行性多灶性白质脑病 PML (progressive multifocal leukoencephalopathy，JC 病毒)| HIV（人类免疫缺陷病毒）/免疫抑制 + 白质脱髓鞘 + 局灶缺损 | — |

> [!tip] 一眼钥匙
> 阶梯下降 = 血管性；行为先变 = FTD；视幻觉 + 帕金森 = DLB；**快速痴呆 + 肌阵挛 = CJD**；步态 + 尿失禁 + 痴呆 = NPH。

🧩 痴呆全面鉴别（含 pseudodementia）→ [[完整笔记/专题笔记/神经/神经_老年痴呆鉴别]]

## 六、帕金森与帕金森叠加（PD & Parkinson-plus）

### 6.1 PD 用药（按年龄/严重度）

| 情境 | 首选 |
|---|---|
| 轻症、<50 岁（震颤为主）| 抗胆碱药 benztropine 苯扎托品 / trihexyphenidyl 苯海索 |
| 轻症、>60 岁 | amantadine 金刚烷胺 |
| 重症（不论年龄）| levodopa-carbidopa 左旋多巴-卡比多巴（并发 on-off 现象）|

> [!warning] 帕金森叠加（Parkinson-plus）钥匙
> - **PD + 共济失调 + 体位性低血压 = Shy-Drager 综合征 / 多系统萎缩 (MSA, multiple system atrophy)**。
> - **药物性帕金森 (drug-induced parkinsonism)**：抗精神病/止吐药多巴胺受体阻断 → 对称、急；vs 特发 PD 不对称、静止性震颤。

🧩 PD 全用药谱 + PD 精神症状阶梯（减药 → quetiapine）→ [[完整笔记/专题笔记/神经/神经_PD用药全解]]

## 七、神经肌接头 + 周围神经急症（神经肌接头 NMJ, neuromuscular junction & 吉兰-巴雷 GBS, Guillain-Barré syndrome）

### 7.1 MG vs Lambert-Eaton（核心对比）

| | 重症肌无力 myasthenia gravis (MG) | Lambert-Eaton 肌无力综合征 (LEMS, Lambert-Eaton myasthenic syndrome) |
|---|---|---|
| 抗体 | **突触后**乙酰胆碱受体 (AChR, acetylcholine receptor) 抗体 | **突触前**电压门控钙通道抗体 |
| 疲劳模式 | **越用越差** | **越用越好**（晨重暮轻反向）|
| 伴随 | 胸腺瘤 (thymoma) → 查胸部 CT | **小细胞肺癌 (SCLC, small cell lung cancer)** + 自主症状（口干 / 勃起功能障碍 ED）|
| 肌电图 EMG (electromyography) | 重复刺激**波幅↓** | 重复刺激**波幅↑** |
| 治疗 | pyridostigmine 吡斯的明 ± 免疫治疗 + 胸腺切除 | 治原发（SCLC）+ pyridostigmine |

> [!danger] 肌无力危象 (myasthenic crisis) — 处理顺序反直觉
> 累及吞咽/构音/呼吸肌 → **最紧急步骤 = 肺功能测定 (PFT, pulmonary function test)**。
> 危象处理：**静脉免疫球蛋白 (IVIG, intravenous immunoglobulin) / 血浆置换先降抗体负荷，再上呼吸机** —— "光推膈肌不降抗体会更糟"。
> **MG 禁用药**（产科尤其）：硫酸镁 MgSO₄（子痫预防改 valproic acid）、氨基糖苷/氟喹诺酮、β 阻滞剂、CCB、神经肌肉阻滞剂。

### 7.2 GBS（吉兰-巴雷）

> [!tip] GBS 三件套
> **上行性对称无力 + 近期感染（~2 周，弯曲杆菌 Campylobacter）+ 蛋白细胞分离 (albuminocytologic dissociation：CSF 蛋白↑、白细胞正常)**。
> - **最紧急步骤 = 肺功能测定**（瘫痪可达膈肌）；累及呼吸 → **先上呼吸机，再 IVIG/血浆置换**。
> - 治疗 = **IVIG 或血浆置换**（**不用激素**）。
> - **Miller-Fisher 变体**：眼肌麻痹 + 共济失调 + 腱反射消失；**抗 GQ1b** 抗体。
> - 完整体系（前驱/变体/诊断/处理/鉴别）→ [[完整笔记/专题笔记/神经/神经_GBS吉兰巴雷体系]]

## 八、脱髓鞘 — 多发性硬化（MS）

> [!info] MS 决策骨架
> 年轻女性、症状**时间 + 空间分离**、上运动神经元 (UMN, upper motor neuron) 体征。
> - **Best initial = MRI**；MRI 阴性仍疑 → **LP 找 oligoclonal bands 寡克隆带**（不特异）。
> - **急性发作 = 大剂量激素**（或血浆置换）。
> - **缓解-复发型 = interferon-β / glatiramer 格拉默**。
> - 招牌：视神经炎 (optic neuritis，眼痛)、核间性眼肌麻痹 (INO, internuclear ophthalmoplegia)、**Uhthoff 现象**（热加重）、Lhermitte 征（颈屈电击感）。
> - 症状控制：日间痉挛 baclofen 巴氯芬；夜间痉挛 tizanidine 替扎尼定；尿潴留 bethanechol 氯贝胆碱。

## 九、眩晕（中枢 vs 外周）

| 特征 | 外周 peripheral | 中枢 central |
|---|---|---|
| 病因 | 良性阵发性位置性眩晕 BPPV (benign paroxysmal positional vertigo)、前庭神经炎、梅尼埃病 (Ménière)、迷路炎 | 卒中（小脑）、MS、偏头痛 |
| 眼震 | 水平/旋转、**单向**、凝视固定可抑制、疲劳性 (<1 min) | 可**垂直**、变向/双向、**凝视不能抑制**、不疲劳 |
| 听力 | 可有（Ménière/迷路炎）| 通常无 |
| 步态 | 不稳但能走 | 可严重不能站 + 其他 CNS 征 |

> [!warning] 何时按中枢/小脑卒中查
> 眩晕 + 卒中危险因素 / 新发头痛 / 局灶神经体征（构音障碍、面肢麻木无力）→ 评估**小脑卒中或出血**。

> [!tip] 外周眩晕快速钥匙
> - **BPPV**：头位诱发、**<1 分钟**；Dix-Hallpike 诊断 → Epley 复位。
> - **Ménière**："men wear VEST" = 眩晕 + 感音聋 + 耳鸣（内淋巴积水）→ **限盐**、急性 meclizine。
> - **前庭神经炎 / 迷路炎**：病毒后；迷路炎**累及耳蜗有听力下降**，前庭神经炎无。

🧩 眩晕全鉴别谱 → [[完整笔记/专题笔记/神经/神经_眩晕鉴别谱]]　🧩 耳鸣鉴别 → [[完整笔记/专题笔记/神经/神经_耳鸣鉴别谱]]

## 十、颅神经病（Bell / Horner / 桥小脑角 CPA）

| 病 | 钥匙 | 下一步 |
|---|---|---|
| Bell 麻痹 (Bell palsy，特发性面神经麻痹)| **单侧面瘫累及上面部**（不能闭眼/抬眉）— vs 卒中上面部保留；单纯疱疹病毒 (HSV, herpes simplex virus) 再激活 | **72h 内**大剂量口服激素 ± valacyclovir 伐昔洛韦 + 角膜保护；3–4 月自愈 |
| Ramsay Hunt 综合征（带状疱疹耳炎）| 耳痛 + 面瘫 + **外耳道疱疹**（水痘带状疱疹病毒 VZV, varicella-zoster virus 再激活）| 抗病毒 + 激素 |

## 十一、脊髓病变 + 周围神经病（Cord & PNS）

> [!info] UMN vs LMN 一眼分
> **下运动神经元 (LMN, lower motor neuron)**：不对称、渐进、**反射减弱 hyporeflexia**。**UMN**（如脊髓病）：对称、突发、**反射亢进 hyperreflexia**。

- **横贯性脊髓炎 (transverse myelitis)**：感觉平面 + UMN 体征 + 膀胱功能障碍。
- **B12 缺乏 → 亚急性联合变性**（后索 + 皮质脊髓束）。
- **前索综合征 (anterior cord syndrome)**：见于降主动脉夹层 (type B) 脊髓前动脉缺血。
- **肌萎缩侧索硬化 (ALS, amyotrophic lateral sclerosis)**：UMN + LMN 体征并存，无感觉受累。
- **慢性砷中毒 arsenic**：手套袜套样周围神经病 + 皮肤改变 + 全血细胞↓。

## 十二、CNS 感染（脑膜炎 / 脑炎）速查

> [!tip] 脑膜炎处理顺序
> 先血培养 + 经验抗生素；**LP 前先 CT 的指征** = 意识改变 / 视乳头水肿 / 癫痫 / 局灶体征 / 免疫抑制。

| 识别钥匙 | 诊断 | 治疗 |
|---|---|---|
| 脑炎 + **嗅幻觉 / 颞叶** | 单纯疱疹脑炎 (HSV) | **IV acyclovir 阿昔洛韦** |
| HIV + **多发环形强化** | 弓形虫脑炎 (toxoplasmosis) | 抗弓形虫 |
| HIV + 视网膜炎/食管结肠炎 | 巨细胞病毒 CMV (cytomegalovirus) 脑炎 | ganciclovir 更昔洛韦 |
| 免疫抑制 + 亚急性 + **颅内压 ICP (intracranial pressure)↑** | 隐球菌 (Cryptococcus) | 抗真菌 |
| 旅行南美 + 环形/钙化囊 | 脑囊虫病 (neurocysticercosis，Taenia solium) | albendazole 阿苯达唑 |
| 蛋白↑糖↓淋巴为主、数周进展 | TB 脑膜炎 | 抗结核 |

![[{CF3B833D-F5A3-47DC-B543-CC3172ECC5F5}.png]]
*神经囊虫病 Neurocysticercosis MRI：囊内可见头节 scolex + 周边强化囊壁（全球获得性癫痫最常见病因）*

## 十三、脑瘤 + 意识/昏迷（速记）

- **脑瘤定位钥匙**：成人幕上（胶质母 = 蝴蝶状跨胼胝体 + 假栅栏坏死）；儿童幕下（髓母细胞瘤 = 小脑蚓部 + 晨起呕吐 + 共济失调）；儿童 + 双颞侧偏盲 = 颅咽管瘤；松果体瘤 = 上视不能 + Parinaud。
- **Cushing 三联**（颅压↑/疝）：高血压 + 心动过缓 + 呼吸不规则。颞部外伤 + 清醒间期 → 硬膜外血肿 → 钩回疝 → CN III 受压（瞳孔散大）。

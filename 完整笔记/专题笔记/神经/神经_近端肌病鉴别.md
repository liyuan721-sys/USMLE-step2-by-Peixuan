# 神经_近端肌病鉴别（Proximal Myopathy DDx）

> [!info] 一句话定位
> 近端肌无力（爬楼/起立/梳头难）鉴别用**三栏快速分型：疼痛/压痛 + ESR（erythrocyte sedimentation rate，红细胞沉降率）+ CK（creatine kinase，肌酸激酶）**。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q4443]]（糖皮质激素性肌病误选 PMR）。

## 一、五型横向对比 ⭐⭐⭐

| 病 | 无力 | 疼痛/压痛 | ESR | CK | 其他线索 |
|---|---|---|---|---|---|
| **糖皮质激素性肌病 glucocorticoid-induced myopathy** | **有**（下肢著）| **无** | **正常** | **正常** | 慢性激素史；近端萎缩；停/减药改善 |
| **风湿性多肌痛 PMR（polymyalgia rheumatica）** | **无**（客观肌力正常）| **有**（肩/髋僵痛）| **↑** | 正常 | 常合并 GCA；**激素数日内快速改善** |
| **多发性/皮肌炎 poly-/dermatomyositis** | 有 | 偶有 | 正常或↑ | **↑↑** | 多系统（关节/皮疹 Gottron/ILD）；恶性风险（皮肌炎）|
| **他汀肌病 statin-induced myopathy** | 可变 | 有 | 正常 | 正常或↑ | 与贝特类合用风险↑ |
| **甲减性肌病 hypothyroid myopathy** | 有 | 有 | 正常 | **↑** | 反射迟缓、黏液水肿、TSH↑ |

> [!tip] 三栏钥匙
> - **疼痛(−) + ESR 正常 + CK 正常 + 客观无力** → **激素性肌病**
> - **疼痛(+) + ESR↑ + 肌力客观正常** → **PMR**
> - **无力 + CK↑↑** → 炎性肌炎 / 甲减性肌病（看 TSH）

## 二、糖皮质激素性肌病（重点）

- 机制：慢性糖皮质激素 → 蛋白分解↑、合成↓ → **无痛性近端肌无力（下肢 > 上肢）**。
- 特征：**CK 正常 + ESR 正常 + 无炎症/压痛**；与剂量/疗程相关。
- 处理：**停药或减量**（tapering）后数周-月改善。
- 场景：GCA（giant cell arteritis，巨细胞动脉炎）/ 自身免疫病/移植等长期大剂量激素后新发近端无力。

## 三、PMR vs 激素性肌病（本题核心鉴别）⭐

| 维度 | **PMR** | **激素性肌病** |
|---|---|---|
| 客观肌力 | **正常**（疼痛限制活动）| **下降（4/5）** |
| 主诉 | 肩/髋**疼痛 + 晨僵** | **无痛**性无力 |
| ESR | **↑（常 >40）** | **正常** |
| 与激素关系 | **激素数日内快速缓解** | **激素是病因**（用药后出现，减量才好）|
| 合并 | 常合并 GCA | — |

> [!warning] GCA 题里的陷阱
> GCA 患者**正在用大剂量激素** + 新发近端无力 + **CK/ESR 正常** → 是**激素性肌病**，不是 PMR（PMR 会 ESR↑ 且肌力正常、激素快速改善）。

## 四、易混陷阱（普适）

- 见 GCA 就反射 PMR（二者确实共存 ≤50%），但要看**肌力 / ESR / 与激素的时间关系**。
- **炎性肌炎** = 有压痛 + **CK/ESR↑**（与激素肌病三项全阴相反）。
- **多发性单神经炎 mononeuritis multiplex** = 血管炎、不对称周围神经病。
- **MS（multiple sclerosis，多发性硬化）** = 中枢、发作性；**NMJ/MG（重症肌无力）** = 波动 + 眼/延髓征；**对称性多神经病** = 远端感觉异常为主。

## Memory Hook

> [!success]
> **长期激素 + 无痛近端无力（下肢著）+ CK/ESR 正常 = 激素性肌病（减量则好）**；
> **PMR = 痛 + 僵 + 肌力正常 + ESR↑ + 激素快速改善**；
> **炎性肌炎 = 痛 + CK↑↑**；**甲减肌病 = CK↑ + TSH↑**。

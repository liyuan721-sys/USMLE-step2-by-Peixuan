# 神经_电击样痛与根性痛鉴别谱（Lancinating / Radicular Pain）

> [!info] 一句话定位
> **电击样痛（electric shock-like / lancinating pain）** = 突发、短暂（秒级）、沿神经根或周围神经放射的**锐痛**——本质是受损/受激惹的感觉轴突**异位放电（ectopic firing）**。
> 鉴别靠 **三轴**：**部位/分布**（面 / 皮节 / 肢体放射）+ **诱发因素**（触碰扳机点 / 屈颈 / 吞咽 / 弯腰）+ **伴随征**（Argyll-Robertson / 水疱 / SLR＋ / 深感觉丢失）。
> ⚠️ 先分清是电击样**「痛」**（本篇：神经痛 / 根性痛）还是电击样**「抽动」**（= 肌阵挛 myoclonus，运动而非疼痛 → [[完整笔记/专题笔记/神经/神经_肌阵挛鉴别诊断]]）——见 §三。

| 诊断 | 部位 / 分布 | 诱发 / 招牌征 | buzzword | 处理 | 详解 |
|---|---|---|---|---|---|
| **三叉神经痛 Trigeminal neuralgia** | 单侧面部 **V2/V3** | 咀嚼 / 刷牙 / 触脸 / 冷风（**触发点**）；秒–2min | 年轻 / 双侧 → 查 **MS（multiple sclerosis 多发性硬化）** | **卡马西平 carbamazepine** | [[完整笔记/专题笔记/神经/神经_头痛鉴别谱]] |
| **舌咽神经痛 Glossopharyngeal neuralgia** | 咽喉 / 扁桃体 / 耳深部 | **吞咽 / 说话**诱发；秒级 | 可伴晕厥（迷走反射）| 卡马西平 | 本篇 §四 |
| **脊髓痨 Tabes dorsalis**（三期神经梅毒） | 下肢为重，「**闪电样痛** lightning」 | **Argyll-Robertson 瞳孔** + 感觉性共济 + Charcot 关节 | 未治梅毒史 | **IV 青霉素 G penicillin** | [[完整笔记/专题笔记/神经/神经_脊髓病变]] §D.4 |

## 二、关键分叉决策树

```
电击样「痛」（秒级、放射、锐痛）
│
├─ 面部 / 头颈区
│   ├─ 单侧 V2/V3 + 触碰扳机点 ────────► 三叉神经痛（卡马西平；年轻/双侧查 MS）
│   └─ 咽喉/耳深部 + 吞咽诱发 ──────────► 舌咽神经痛
│
├─ 沿脊柱 / 屈颈诱发 ───────────────────► Lhermitte 征 → 找病因（MS / B12 SCD / 颈椎病 / 顺铂 / 笑气 / 椎间盘）
│
├─ 肢体放射（沿皮节 / 神经根）
│   ├─ 弯腰搬重后 + SLR＋ + 下肢 ───────► 腰椎间盘突出（L5/S1 坐骨神经根）
│   ├─ 颈痛 + 放射至臂 + Spurling＋ ────► 颈神经根病（C6/C7）
│   ├─ 鞍区麻 + 尿潴留 + 双下肢 ────────► 马尾综合征（外科急症）
│   └─ 皮节水疱愈后 + 烧灼 allodynia ───► 疱疹后神经痛（PHN）
│
└─ 全身/背部 + 特殊征
    ├─ 梅毒史 + Argyll-Robertson + 共济 ─► 脊髓痨 tabes（闪电样痛；IV PCN）
    └─ 病毒/疫苗后 肩臂痛→无力 ──────────► 臂丛神经炎 Parsonage-Turner
```

## 三、易混陷阱：电击样「痛」vs 电击样「抽动」⭐

> [!warning] 一字之差，走向完全不同
> UW 题干常写「electric shock-like」，务必看清是**疼痛**还是**肌肉抽动**：
>
> | | 电击样**痛**（本篇） | 电击样**抽动** |
> |---|---|---|
> | 本质 | 感觉神经异位放电 → **疼痛** | 运动皮层/脊髓 → **肌肉不自主收缩** |
> | 代表 | 三叉神经痛 / tabes / PHN / 根性痛 | **肌阵挛 myoclonus**（JME 青少年肌阵挛癫痫 / CJD 克雅病 Creutzfeldt-Jakob / 缺氧后 / 中毒）|
> | 题干钩子 | 「触脸剧痛」「放射至腿」 | 「扔掉牙刷」「被巨响惊吓诱发」「晨起手抖跳」|
> | 去向 | 本篇鉴别 | [[完整笔记/专题笔记/神经/神经_肌阵挛鉴别诊断]] |

## 四、专题无落点的三个补充点

> [!tip] 舌咽神经痛（Glossopharyngeal neuralgia）
> 阵发电击样痛在**咽喉、扁桃体窝、耳深部（CN IX/X 分布）**，由**吞咽、咀嚼、说话、咳嗽**诱发。
> 少数因迷走反射伴**心动过缓 / 晕厥**（"vagoglossopharyngeal syncope"）。一线药同三叉神经痛（**卡马西平 carbamazepine**）。
> 与三叉神经痛的区别 = 部位（咽/耳 vs 面 V2/V3）+ 诱发（吞咽 vs 触脸）。

> [!tip] 疱疹后神经痛（Postherpetic neuralgia, PHN）
> **带状疱疹（herpes zoster）皮疹愈合后 ≥3 个月**仍持续的**皮节分布**神经病理痛：烧灼样 + 电击样 + **异常性疼痛 allodynia**（衣物轻触即痛）。
> 危险因素：**高龄、免疫抑制、急性期疼痛重**。
> 一线：**加巴喷丁 gabapentin / 普瑞巴林 pregabalin / 三环类 TCA（阿米替林 amitriptyline）/ 局部利多卡因**。
> ⚠️ 预防 PHN 最有效 = **重组带状疱疹疫苗 Shingrix（≥50 岁）**；急性期抗病毒（伐昔洛韦 valacyclovir）72h 内可减轻但不完全预防 PHN。

> [!tip] 臂丛神经炎（Parsonage-Turner / 神经痛性肌萎缩）
> 典型：**病毒感染 / 疫苗 / 手术后**数天突发**单侧肩臂剧痛**，数日后疼痛消退、代之以**斑片状无力 + 肌萎缩**（常累及冈上/冈下肌、前锯肌 → 翼状肩胛）。
> 自限性，多数数月–2 年恢复；治疗以**镇痛 + 物理康复**为主。与颈神经根病区别 = 无颈痛、非单一皮节、痛后才出现无力。

## 五、并非所有神经根病变都会电击样放射痛（三闸门）⭐

> [!warning] 「累及了根」≠「一定痛」
> 电击样放射痛的本质 = **背根（感觉纤维）被刺激 → 异位放电**。任何一个闸门不满足，就可以「碰了根却不痛」。

| 闸门 | 会痛（阳性症状） | 不痛 / 少痛 |
|---|---|---|
| **① 背根 vs 腹根** | **背根 (dorsal root，感觉)** 受累 → 放射痛、感觉异常 | **纯腹根/前角 (ventral root / anterior horn，运动)** → 只无力**不痛**：脊髓灰质炎 (poliomyelitis)、脊髓性肌萎缩 (SMA, spinal muscular atrophy) |
| **② 刺激性 vs 破坏性** | **刺激 / 炎症 / 压迫**（椎间盘突出、带状疱疹 herpes zoster、根炎 radiculitis、肿瘤压迫）→ 异位放电 | **纯缓慢破坏 / 脱髓鞘不伴激惹** → 麻木、无力、反射消失（阴性症状为主）|
| **③ 能否异位放电** | 受损轴突去抑制 + 异位放电（tabes 闪电痛、脱髓鞘）| 单纯轴突丢失、慢性适应 → 数值缺损而无剧痛 |

> [!tip] 反射级结论
> **「放射痛 = 背根被刺激」**。判断某根病变会不会痛，先问：碰没碰**感觉根** + 是不是**刺激/炎症性**。
> - 会痛：椎间盘突出、带状疱疹 / PHN、糖尿病神经根病、tabes、**GBS（吉兰-巴雷综合征 Guillain-Barré syndrome）早期根炎（radiculitis）**、癌性脑膜炎浸润根
> - **不痛**：**脊髓灰质炎 / SMA**（纯前角运动，只瘫不痛）、部分慢性压迫（麻木无力为主）
> - GBS 早期背痛正是「炎症刺激感觉根 → 放射痛」的活例 → 详见 [[完整笔记/专题笔记/神经/神经_GBS吉兰巴雷体系]] §四（背痛 = 根炎非压迫）

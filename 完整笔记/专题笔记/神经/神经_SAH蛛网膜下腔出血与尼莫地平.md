# 蛛网膜下腔出血 SAH 与尼莫地平（动脉瘤破裂 → 防血管痉挛/DCI → 口服尼莫地平，神经保护而非"撑开血管"）

> [!info] 用途 + 缩写约定
> 从概念讨论沉淀的独立专题，专攻 **aSAH → 血管痉挛 vasospasm → 迟发性脑缺血 DCI** 这条链，以及**尼莫地平 nimodipine 的机制与三大陷阱**。核心解决：**① 尼莫地平为什么扩脑血管、为什么理论上会 ↑ICP ② 为什么"只用于动脉瘤性 SAH" ③ 三个必考陷阱（机制不是减少造影痉挛 / 只能口服 / 主监测低血压）**。
> 缩写：**SAH** = subarachnoid hemorrhage（蛛网膜下腔出血）；**aSAH** = aneurysmal SAH（动脉瘤性 SAH）；**CCB** = calcium channel blocker（钙通道阻滞剂）；**DHP** = dihydropyridine（二氢吡啶类）；**BBB** = blood-brain barrier（血脑屏障）；**CBV** = cerebral blood volume（脑血容积）；**ICP** = intracranial pressure（颅内压）；**CPP** = cerebral perfusion pressure（脑灌注压）；**DCI** = delayed cerebral ischemia（迟发性脑缺血）；**CTA** = CT angiography（CT 血管造影）；**DSA** = digital subtraction angiography（数字减影血管造影）；**LP** = lumbar puncture（腰穿）；**BP** = blood pressure（血压）；**CSF** = cerebrospinal fluid（脑脊液）；**TBI** = traumatic brain injury（创伤性脑损伤）；**RCT** = randomized controlled trial（随机对照试验）。

> [!danger] ⭐ 置顶速记：尼莫地平三反射
> 1. **身份**：脑选择性 DHP 类 CCB（亲脂 → 穿 BBB → 作用在脑血管）。别的 DHP（硝苯地平 nifedipine 等）进不了脑、无此用途 —— **是 nimodipine 特异性，不能替换**。
> 2. **适应证**：**动脉瘤性 SAH（aSAH），口服 60 mg q4h × 21 天，尽早开始** —— 预防血管痉挛导致的 DCI、改善神经预后。
> 3. **三大陷阱**：① 机制是**神经保护**，**不显著减少造影上的血管痉挛**（选 "reduces angiographic vasospasm" = 错）② **只能口服/肠内，IV 有黑框警告**（致命）③ 主要副作用 = **系统性低血压**（不是 ICP！低血压反而降 CPP 加重缺血 → 需减量/停药）。

## 一、药理身份：脑选择性 DHP 类 CCB

> [!info] 先定性，这决定一切
> **尼莫地平 = 二氢吡啶类钙通道阻滞剂**（DHP CCB，和硝苯地平 nifedipine / 氨氯地平 amlodipine 同族），作用于血管平滑肌 **L 型钙通道**。
> 它与同族唯一的区别 —— 也是它存在的理由：**高度亲脂 lipophilic → 穿过血脑屏障 BBB → 对脑血管相对选择性**。别的 DHP 主要作用外周动脉，尼莫地平能进颅内。
> **"脑选择性 CCB" 这七个字 = 它的身份证。**

## 二、扩血管机制

```
DHP 阻断血管平滑肌 L 型 Ca²⁺ 通道
        ↓
胞内 Ca²⁺↓ → 肌球-肌动蛋白收缩减弱
        ↓
血管平滑肌松弛 → 血管扩张（尼莫地平重点落在脑血管）
```

## 三、"脑血管扩张 → ↑CBV → ↑ICP" 这条链（Monro-Kellie）

```
颅腔容积固定 = 脑组织 + CSF + 血液（三者之和恒定）
        ↓
脑血管扩张 → 颅内血液 CBV↑
        ↓
密闭颅腔内挤占空间 → ICP↑
```

> [!warning] ⭐ 必须校准的考试重点（别记反）
> 这条 "↑ICP" 是**所有脑血管扩张剂的通用原理**（硝普钠 nitroprusside、硝酸甘油 nitroglycerin、高碳酸血症 hypercapnia、吸入麻醉药都走这条）——用来解释**"为什么高 ICP 病人要避开血管扩张剂"**（含尼莫地平：见 [[完整笔记/专题笔记/神经/神经_颅压增高诊疗顺序]] §四，重型 TBI + 颅高压时 ❌ 尼莫地平）。
> **但尼莫地平自己在临床上真正被监测、真正会考的副作用不是 ICP，而是「系统性低血压」**（它照样扩外周血管）。
> - 用 "↑ICP" 当"它 niche 很窄、不是万能脑药"的记忆钩子 —— ✅ 逻辑通
> - stem 问 "nimodipine 最需监测什么" → 答 **血压 / 低血压**，**不是 ICP**。

## 四、为什么"只用于动脉瘤性 SAH"

> [!info] 真正原因不是 ICP，而是"只有 aSAH 有 RCT 证据改善预后"
> 它的战场是对付 aSAH 后一个致命并发症：**血管痉挛 → 迟发性脑缺血 DCI**。

```
动脉瘤破裂 → 血进蛛网膜下腔
        ↓
血液降解产物（氧合血红蛋白等）刺激脑动脉
        ↓
Day 3–14（峰值≈Day 7）血管痉挛 → 继发缺血性卒中（DCI）→ 致残/致死
        ↓
尼莫地平：口服 60 mg q4h × 21 天，SAH 后尽早开始
```

> [!danger] ⭐ 本药最大陷阱：机制是神经保护，不是"减少造影痉挛"
> **尼莫地平改善神经预后（reduces DCI / better outcomes），但并不显著减少造影上的血管痉挛（does NOT reduce angiographic vasospasm）。**
> 机制被认为是**神经保护 neuroprotection**（减少皮层扩散性缺血、微血栓，保护神经元），而非单纯"把痉挛的大血管撑开"。
> → 选项写 "works by preventing/reversing arterial vasospasm on angiography" = **经典错误选项**。正确表述 = "improves neurologic outcomes / reduces delayed cerebral ischemia"。

## 五、高频考点速查

| 点 | 内容 |
|---|---|
| **给药途径** | **只能口服 / 肠内**。**IV 有黑框警告 boxed warning** —— 静脉给药致命性心血管事件 / 死亡。stem 给 "IV nimodipine" 即错 |
| **主要副作用** | **低血压 hypotension**（外周扩血管）。低血压时减量 / 停药 —— 低血压降 CPP，反而加重脑缺血 |
| **不能替换** | nimodipine 特异性，**不是"随便一个 CCB"**。别的 DHP 进不了脑、无证据 |
| **不治动脉瘤本身** | 动脉瘤要**手术夹闭 clipping / 血管内弹簧圈 coiling**。尼莫地平只是防痉挛 / 护神经的辅助 |
| **DCI 时间窗** | Day 3–14，峰值 ≈ Day 7 |

## 六、经典 Stem 长相

```
雷击样头痛 thunderclap headache + 非增强 CT 见蛛网膜下腔高密度
        → 疑 SAH（CT 阴性仍疑 → LP 找黄变 xanthochromia）
        → CTA / DSA 定动脉瘤 → clipping / coiling
        → 【下一步预防 delayed neurologic deficit】= 口服尼莫地平 ⭐
```

- SAH 已 coiling，问"下一步预防迟发神经缺损" → **口服尼莫地平**
- SAH 后 **Day 7** 突发新发局灶缺损 / 意识下降 → 想 **血管痉挛 / DCI**，尼莫地平是预防它的
- 干扰项识别：把 "reduces angiographic vasospasm" 当机制 = 错；给 "IV nimodipine" = 错

## 七、复习题

> [!question]- Q1. SAH 病人动脉瘤已 coiling，为预防迟发神经缺损给尼莫地平。其改善预后的最佳机制描述是？
> **神经保护（neuroprotection，减少微血栓 / 皮层扩散性缺血 / 神经元损伤）** —— **不是**"减少造影上的动脉血管痉挛"。这是最高频陷阱：尼莫地平改善预后但不显著逆转 angiographic vasospasm。

> [!question]- Q2. 一位重型 TBI + 颅高压病人，为什么不给尼莫地平？
> 尼莫地平**扩脑血管 → ↑CBV → ↑ICP**（Monro-Kellie），在颅高压场景有害；且它只有 **aSAH** 有循证获益。参见 [[完整笔记/专题笔记/神经/神经_颅压增高诊疗顺序]] §四（同禁忌：糖皮质激素、LP）。

> [!question]- Q3. 给 aSAH 病人尼莫地平期间，最需要监测什么？为什么？
> **血压（低血压）**。尼莫地平系统性扩血管 → 低血压 → 降 CPP → 反而加重脑缺血。低血压时减量 / 停药。（注意：不是 ICP —— ICP 是"通用血管扩张剂原理"的记忆钩子，不是本药临床主监测项。）

# 神经_震颤鉴别谱（Tremor 按出现时机分诊）

> [!info] 一句话定位
> **震颤 (tremor 震颤) = 节律性、振荡性的不自主运动。** USMLE 极少直接问"是不是震颤"，而是给你一段 stem，逼你用**出现时机**（静止 / 维持姿势·动作 / 指向目标）三秒分型，再叠加全身背景反推病因。
> **解题第一步永远是：先问"什么时候抖最厉害" —— 静止时 → 帕金森；动作/端姿势时 → 原发性 / 生理性增强（甲亢·药物·戒断）；够物终点时 → 小脑。**
> 派生自 NBME11A [[NBME11A_S3_breakdown#^Q150|Q150]]（原发性震颤 essential tremor → propranolol 普萘洛尔）。整合自 [[完整笔记/Peixuan分科笔记/neuro]] + [[完整笔记/Peixuan分科笔记/psych]] + [[完整笔记/Peixuan分科笔记/endocrine]] 增量。

## 一、核心分诊树（按出现时机）

```
                    手在抖 —— 什么时候最明显？
                              │
        ┌─────────────────────┼─────────────────────┐
        │                     │                     │
   静止时明显               维持姿势 / 动作时        够物终点时加重
   动作时减轻               (postural / action)     + 辨距不良
   (resting)                     │                  (intention)
        │                        │                     │
   帕金森病 PD            ┌──────┴───────┐         小脑病变
   (不对称·搓丸·低频)      │              │         cerebellar
   + 齿轮样强直 + 运动迟缓  原发性震颤 ET   生理性增强         (卒中·脱髓鞘·肿瘤)
        │                (对称·家族史·    enhanced          + 共济失调
   药物性帕金森           饮酒缓解·高频)   physiologic        dysmetria
   (对称·起病快·有药史)         │           │
                          首选 propranolol  找诱因:
                          普萘洛尔          甲亢 / 低血糖 / 嗜铬
                                           药物(锂·丙戊酸·SSRI·β激动剂)
                                           咖啡因 / 焦虑 / 酒精戒断
```

> [!tip] TRAP 记忆法（帕金森四联，注意 T 是 Tremor 不是 Tardive）
> - **T**remor —— **静止性**搓丸样震颤 (resting pill-rolling tremor)
> - **R**igidity —— 齿轮样强直 (cogwheel rigidity)
> - **A**kinesia / bradykinesia —— 运动减少 / 迟缓
> - **P**ostural instability —— 姿势不稳

## 二、三型震颤主对照

| 维度 | 静止性 Resting | 姿势/动作性 Postural/Action | 意向性 Intention |
|---|---|---|---|
| **何时最明显** | 安静放松时；动作时减轻 | 端姿势 / 主动运动时 | 接近目标终点时进行性加重 |
| **代表病** | 帕金森病 (Parkinson disease, PD) | 原发性震颤 (essential tremor, ET) / 生理性增强 | 小脑病变 (cerebellar) |
| **典型外观** | 搓丸样 (pill-rolling)、低频 (3–5 Hz) | 双手对称、高频 (8–12 Hz) | 振幅随接近目标增大 |
| **对称性** | 多**不对称**起病 | 多**对称** | 同侧小脑病变侧 |
| **伴随体征** | 齿轮样强直 + 运动迟缓 + 步态 | 无其他神经体征 | 辨距不良 (dysmetria)、轮替不能、构音障碍、眼震 |
| **首选药** | 左旋多巴/卡比多巴 (Levodopa/Carbidopa)（重）/ 抗胆碱（轻·年轻·震颤为主） | **Propranolol 普萘洛尔**（首选）/ Primidone 扑米酮（二线） | 治原发病；Clonazepam 氯硝西泮（部分） |

> [!warning] ⭐ ET 治疗两个高频陷阱（Q19545 错点）
> - **哮喘 / COPD（chronic obstructive pulmonary disease，慢性阻塞性肺病）（反应性气道病）/ 心脏传导阻滞 = β 阻滞剂相对禁忌 → 首选改 primidone 扑米酮**（别硬上 propranolol）。
> - **"指鼻加重"≠ 一定是小脑意向性震颤**：ET 是**动作性震颤**，动作末（指鼻终点）也会加重。靠 **ET = 双侧 + 家族史 + 饮酒改善 + 步态/协调正常** 与小脑病变（辨距不良/构音障碍/眼震）区分。见 [[mistakes/uworld-mistakes_2026-07#^Q19545]]（误判小脑、错选苯海索）。
> - **层级别搞反**：**意向性震颤 (intention) ⊂ 运动性 (kinetic) ⊂ 动作性 (action)** —— 意向性本就是动作性的子集，所以"ET 是动作性"和"指鼻终点加重"不矛盾，冲突的是把"终点加重"直接等号成"意向性→小脑"。ET 主体是**姿势性 (postural) + 运动全程 (kinetic)**，约半数有轻度终末加重但只是**附加**，不带辨距不良。
> - **判别核心：看伴随征，不看"指鼻抖不抖"这一个动作**。终点加重 **+ 辨距不良 (dysmetria)/构音障碍/眼震/宽基步态 → 意向性 → 小脑通路**；终点加重 **但只是抖 + 双侧 + 家族史 + 饮酒缓解 + 协调正常 → ET**。「指鼻终点加重」单独出现是中性表现，不能定位。

> [!tip] 指鼻场景下只有两类，靠"终末 crescendo"分（别拆成三列）
> 「指鼻时手抖」这个笼统事实**不区分**——ET（动作性）和小脑（意向性）在指鼻全程都会抖。真正区分的是**时相/模式**：有没有在接近鼻尖终点时振幅进行性 crescendo 增大。
>
> ```
> 指鼻全程均匀抖              接近鼻尖时振幅进行性增大
> (无终末加重)                (terminal crescendo)
>      ↓                            ↓
> 运动性/动作性 kinetic          意向性 intention
> (ET)                          (小脑 cerebellar)
>                               + 辨距不良/构音障碍/眼震
> ```
>
> 完整层级（谁包含谁，别搞反）：
>
> ```
> 动作性 action(只要主动收缩肌肉就抖)
>    ├── 姿势性 postural(维持抗重力姿势,如双臂平举)← 端姿势查
>    └── 运动性 kinetic(运动过程中抖)
>           └── 意向性 intention(运动末接近目标时进行性加重)← 指鼻查
> ```
>
> - 为什么"动作性"和"运动性"像重复：**运动性 kinetic ⊂ 动作性 action**，指鼻测的就是 kinetic 这一支，所以在指鼻这个动作里两词指的是同一批病（ET），并列写才显得多余。
> - **postural 姿势性测不到**：它是 action 的**另一支**，要靠"双臂平举维持抗重力"这个动作查（见 §二主对照"何时最明显"列）。**指鼻不能用来区分姿势性 vs 意向性**——姿势性得另做平举维持才看得到。
> - 一句话反射：**指鼻只查 kinetic/intention 这条线；区分靠终末 crescendo + 伴随征，不靠"抖不抖"。**

> [!danger] ⭐ 意向性震颤病因谱 —— 不止小脑实质（指鼻越近越抖 + 辨距不良）
> **意向性震颤 = 小脑或其传入/传出通路（cerebellar circuit）任何一处损伤**，不限于小脑本身。先扫年龄 + 全身背景反推病因：
>
> | 病因 | stem 钩子 | 要点 |
> |---|---|---|
> | **多发性硬化 (multiple sclerosis, MS)** ⭐ | 年轻女性 + 意向性震颤 + 核间性眼肌麻痹 (internuclear ophthalmoplegia, INO) / 视神经炎 (optic neuritis) / 时间空间多发 | 脱髓鞘累及小脑传导束/齿状核通路——USMLE 最高频钩子 |
> | **卒中 / 肿瘤 (stroke / tumor)** | 急性起病 / 占位；小脑、脑桥、中脑病变 | 同侧肢体辨距不良 |
> | **Wilson 病 (肝豆状核变性)** | 年轻人 + 肝病 + 精神症状 + KF 环 (Kayser-Fleischer ring) | "wing-beating tremor 扑翼样"兼意向成分；查血浆铜蓝蛋白 (ceruloplasmin ↓) |
> | **Holmes / rubral tremor (红核·中脑震颤)** | 中脑/红核附近损伤 | 慢而粗大，**静止 + 姿势 + 意向三种成分并存**（区别于纯小脑） |
> | **中毒 (toxins)** | 慢性**酒精** (小脑蚓部变性 + 宽基步态) / **汞 mercury** / 锂 / 苯妥英 (phenytoin) | 去诱因 |
> | **遗传性小脑变性** | Friedreich 共济失调等 | 家族史 + 进行性共济失调 |
>
> **一句话反射**：年轻女性→MS；年轻人+肝损/KF 环→Wilson；静止+姿势+意向都有且粗大→Holmes/rubral；慢性酗酒+宽基步态→酒精性小脑变性。

## 三、ET vs PD —— 最高频鉴别（必背）

> [!warning] 这两个是 USMLE 最爱混的一对，记死 5 个区分点
> 核心一句：**ET 动作时抖、PD 静止时抖；ET 喝酒就好、PD 喝酒没用。**

| 鉴别点 | 原发性震颤 ET | 帕金森病 PD |
|---|---|---|
| **何时抖** | 端姿势 / 动作时（倒咖啡、写字加重） | 静止时；动作时反而减轻 |
| **频率/外观** | 高频、双手**对称** | 低频、搓丸样、**不对称**起病 |
| **饮酒反应** | **酒精暂时缓解**（经典钩子）⭐ | 无特殊 |
| **家族史** | 常有（常染色体显性 AD） | 多无 |
| **伴随征** | 无强直 / 无运动迟缓 | 齿轮样强直 + 运动迟缓 + 步态异常 |
| **首选药** | **Propranolol**（β 阻滞剂）；二线 Primidone | Levodopa/Carbidopa；年轻震颤为主可抗胆碱 |

> [!tip]- Primidone（扑米酮）是什么 —— 二线药机制
> **prodrug（前药）**，体内代谢成 **phenobarbital（苯巴比妥，真正的 barbiturate 巴比妥类）+ PEMA**——primidone 本身**不是**巴比妥，靠代谢产物起效（增强 GABA-A）。
> - 用途：①**抗癫痫 (antiepileptic)** ②**ET 二线**——当 propranolol 因**哮喘 / COPD (慢性阻塞性肺病) / 心脏传导阻滞**不能用时顶上（见上文 §二 ET 治疗陷阱）。
> - 过量 / 副作用走巴比妥那套：镇静、共济失调、呼吸抑制；起始易嗜睡/眩晕 → 低剂量慢加。

> [!example]- 经典 stem 钩子
> - "倒咖啡 / 端汤时手抖明显，静止放在腿上不抖" → **ET**
> - "喝一杯酒后震颤明显减轻" → **ET**
> - "走路时一只手静止搓丸样抖、面具脸、起步困难" → **PD**
> - "对称、起病快、近期加了抗精神病药 / 止吐药" → **药物性帕金森**（drug-induced parkinsonism），不是原发 PD

![[L41408.jpg|500]]
*帕金森病 (Parkinson disease, PD, 帕金森病) 生物学基础 (UWorld)：黑质 (substantia nigra) 多巴胺能神经元进行性退变 → 基底节输出失衡 → 静止性震颤 + 运动迟缓。*

## 四、姿势/动作性震颤的"找诱因" —— 生理性增强谱

姿势性震颤里，除了 ET，剩下全是**生理性震颤被放大 (enhanced physiologic tremor)**：高频、低振幅、细抖。看到细抖要主动扫一圈病因：

| 诱因 | 钩子 / 伴随 | 机制 | 处理 |
|---|---|---|---|
| **甲亢 (hyperthyroidism)** | 细震颤 + 心悸 + 怕热 + 消瘦 + 反射亢进；脉压宽 | β 肾上腺素能敏感性 ↑ | **Propranolol** 对症（快速压震颤/心悸）+ 治本（methimazole 甲巯咪唑 / 放射性碘 (radioactive iodine, RAI) / 手术）；危象 **静脉 (IV) β 阻滞剂**首线 |
| **低血糖 (hypoglycemia)** | 震颤 + 出汗 + 心悸 + 焦虑（neurogenic 症状） | Epinephrine 肾上腺素激增 | 立即补糖；长期 1 型糖尿病 (type 1 diabetes mellitus, T1DM) 可"无感知 hypoglycemia unawareness"——β 阻滞剂会掩盖警告症状，慎用 |
| **嗜铬细胞瘤 (pheochromocytoma)** | 阵发高血压 + 头痛 + 心悸 + 苍白 + 体位性低血压 | 儿茶酚胺大量释放 | **先 α 阻滞再 β 阻滞**（顺序反了→高血压危象） |
| **咖啡因 / 焦虑 / 疲劳** | 年轻人、大量咖啡后细抖，停用缓解 | 生理性震颤放大 | 去诱因 |
| **β2 激动剂 / 茶碱 theophylline** | 哮喘用药后细抖 | 直接交感样刺激 | 减量 / 评估 |

## 五、药物 / 戒断诱发的震颤（精神科高频）

| 药 / 状态 | 震颤类型 | 钩子 / 伴随 | 处理要点 |
|---|---|---|---|
| **锂 (lithium)** | 细震颤（治疗剂量常见）；**粗大震颤 = 中毒** | 粗抖 + 共济失调 + 构音障碍 + 意识模糊 → 查血锂（>1.5 mEq/L 中毒） | 细抖可加 **propranolol**；中毒则停药、补液、必要时血透；长期监测肾功/甲功 |
| **丙戊酸 (valproate)** | 姿势性震颤 | 伴体重增加、脱发、血小板↓ | 反复出血**先查血小板**；轻度可减量 |
| **选择性5-羟色胺再摄取抑制剂 (SSRI) / 安非他酮 (bupropion)** | 激活相关细震颤 | 启动期 + 失眠 + 焦虑 + 出汗 | 多耐受 / 减量；与血清素综合征 (serotonin syndrome) 区分（后者高反射 + 肌阵挛 + 高热） |
| **酒精戒断 (alcohol withdrawal)** | 粗大震颤（戒断三联：震颤 + 出汗 + 心动过速） | 末次饮酒后 **6–12 h** 起；可进展为癫痫(12–48h)→ 震颤谵妄 (delirium tremens, DT) | 长效苯二氮䓬 (long-acting benzodiazepine，如地西泮 diazepam / 氯氮䓬 chlordiazepoxide) |

> [!warning] 别把"抖"全算成震颤
> - **肌阵挛 (myoclonus)** = 电击样单次抽动（非节律振荡）→ 见 [[完整笔记/专题笔记/神经/神经_肌阵挛鉴别诊断]]
> - **静坐不能 (akathisia)** = 主观坐立不安 + 来回踱步（非真震颤），抗精神病药后；首选 propranolol
> - **迟发性运动障碍 (tardive dyskinesia)** = 口-舌-面重复不自主运动（非震颤）
> - **舞蹈样动作 (chorea) / 肌张力障碍 (dystonia)** 也常被患者描述成"抖"，靠节律性 + 出现时机区分

## 六、功能性（心因性）震颤 Functional tremor

| 特征 | 说明 |
|---|---|
| **分散性 (distractibility)** | 让患者用对侧手做复杂动作 / 数数 → 震颤**减轻或消失**（最关键） |
| **可变性 (variability)** | 频率、幅度、部位、时机多变 |
| **不一致 (inconsistency)** | 与任何已知器质性震颤模式都对不上 |
| **突发突止** | 起病与缓解都突然 |

> 与 [[完整笔记/专题笔记/psych/psych_功能性神经症状障碍]] 同源 —— "分散注意时减轻、与解剖不符"是功能性神经症状的统一指纹（参见 NBME11A Q112 转换障碍）。

## 七、自测

> [!question]- Q1. 68 岁男性，右手安静放在腿上时搓丸样抖动，伸手拿杯子时反而减轻；查体齿轮样强直 + 起步困难。最可能诊断？震颤是什么类型？
> **帕金森病 (Parkinson disease)**；**静止性震颤 (resting tremor)**。静止抖 + 动作减轻 + 不对称 + 搓丸 + 强直/运动迟缓 = 经典 PD。

> [!question]- Q2. 42 岁女性，双手端姿势时对称性高频细抖，写字与端汤时明显，喝一杯红酒后缓解，母亲有类似病史。诊断 + 首选药？
> **原发性震颤 (essential tremor)**；首选 **propranolol 普萘洛尔**（二线 primidone 扑米酮）。动作性 + 对称 + 家族史 + 饮酒缓解 = ET。

> [!question]- Q3. 患者伸手去够检查者手指，越接近终点抖得越厉害，伴辨距不良与构音障碍。震颤类型 + 定位？
> **意向性震颤 (intention tremor)**；定位**小脑 (cerebellum)**。终点加重 + dysmetria + 构音障碍 = 小脑通路病变。

> [!question]- Q4. 双相情感障碍患者服锂数年，近期出现粗大震颤 + 走路不稳 + 言语含糊 + 嗜睡。最该立即做什么？
> 查 **血锂浓度 (lithium level)**——细抖是治疗剂量常见，**粗大震颤 + 共济失调 + 构音障碍 = 锂中毒**（>1.5 mEq/L），停药 + 补液，重则血透。

> [!question]- Q5. 26 岁女性细震颤 + 心悸 + 怕热 + 体重下降 + 反射亢进。震颤机制 + 对症首选药？
> **甲亢 (hyperthyroidism)**，β 肾上腺素能敏感性 ↑ 致姿势性细震颤；对症首选 **propranolol**（同时压心悸/心动过速），治本用 methimazole / RAI / 手术。

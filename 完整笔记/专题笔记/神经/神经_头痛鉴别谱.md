# 神经_头痛鉴别谱（Primary & Secondary Headache）

> [!info] 一句话定位
> 头痛鉴别靠 **三轴**：**时长**（秒 / 分-时 / 时-天 / 持续）+ **部位/性质** + **伴随特征**（自主症状 / 触发点 / 畏光 / 视觉）。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q114669]]（原发性针刺样头痛误判 cluster）。

## 一、原发性头痛鉴别表

| 头痛                                        | 时长       | 部位 / 性质      | 钥匙特征                                   | 处理                                                      |
| ----------------------------------------- | -------- | ------------ | -------------------------------------- | ------------------------------------------------------- |
| **原发性针刺样 primary stabbing（ice pick 冰锥）⭐** | **数秒**   | 局灶、尖锐刺痛、可指出点 | **无触发、无自主症状**；多见偏头痛人群                  | **安慰**（良性自限）± 短程 **吲哚美辛 indomethacin**                  |
| 丛集性 cluster                               | 15min–3h | 单侧**眶周**剧痛   | **同侧自主症状**（流泪/鼻塞/缩瞳 Horner/流涕）、夜间发作、躁动 | 急性：**100% O₂ + 舒马普坦 sumatriptan**；预防：**维拉帕米 verapamil** |
| 偏头痛 migraine                              | 4–72h    | 单侧**搏动性**    | **畏光/畏声 + 恶心呕吐**、有/无先兆、活动加重            | 急性：NSAID/曲普坦 triptan；预防：普萘洛尔/托吡酯/阿米替林                   |
| 紧张型 tension                               | 30min–7d | 双侧**紧箍样**    | 无恶心、轻中度、无加重                            | NSAID / 对乙酰氨基酚                                          |
| 三叉神经痛 trigeminal neuralgia                | 数秒–2min  | **V2/V3** 分布 | **触发点**（咀嚼/刷牙/触脸/冷风）                   | **卡马西平 carbamazepine**                                  |

## 二、必须排除的继发性头痛（red flags）

| 提示 | 想到 |
|---|---|
| 突发"一生最剧"雷击样 | 蛛网膜下腔出血 SAH → 头 CT ± 腰穿 |
| 年轻肥胖女 + 视乳头水肿 + 搏动性 + 用力加重 + 视觉变化 | **IIH 特发性颅内高压**（idiopathic intracranial hypertension）→ 见 [[完整笔记/专题笔记/神经/神经_颅压增高诊疗顺序]] |
| 晨起加重 / 喷射性呕吐 / 局灶体征 | 占位、颅高压 |
| >50 岁 + 颞部压痛 + 颌跛行 + ESR↑ | 巨细胞动脉炎 GCA → 立即激素 + 活检 |
| 发热 + 颈强 + 畏光 | 脑膜炎 |

## 三、短刺痛三鉴别（高频混淆）⭐

```
极短(秒级)剧痛
   ├─ 无触发 + 无自主症状 + 能指出点 ──► 原发性针刺样(ice pick)
   ├─ 沿 V2/V3 + 有触发点(咀嚼/刷牙) ──► 三叉神经痛 (卡马西平)
   └─ 眶周 + 自主症状 + 持续数分钟-小时 ─► 丛集性 cluster (注意时长其实更长)
```

> [!warning] Q114669 陷阱
> 见"剧烈 + 年轻女"易反射 cluster，但 cluster = **眶周 + 自主症状 + 数分钟-小时**；原发性针刺样 = **数秒 + 无自主症状 + 局灶可指点**。核心区分 = **时长（秒 vs 分-时）+ 有无自主症状**。

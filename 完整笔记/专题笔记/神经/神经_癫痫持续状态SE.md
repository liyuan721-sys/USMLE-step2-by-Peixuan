# 神经_癫痫持续状态 SE（Status Epilepticus）

> [!info] 一句话定位
> **SE（status epilepticus，癫痫持续状态）= 单次发作 ≥5 分钟，或两次发作间未恢复至基线意识**。是神经急症——越早终止，继发脑损伤越少。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q4089]]（SE 最高风险并发症 = 皮质层状坏死）。

## 一、定义与机制

- **定义**：连续发作 **≥5 min**，或反复发作之间意识未恢复基线。
- **常见诱因**：抗癫痫药**停药/漏服**（最常）、酒精戒断、感染、代谢紊乱、卒中、外伤。
- **损伤机制**：持续放电 → **兴奋性毒性（excitotoxicity，谷氨酸介导钙内流）+ 代谢需求↑/缺氧** → 神经元死亡。

## 二、最高风险并发症 = 皮质层状坏死 ⭐

> [!danger] 为什么是皮质（cortex）
> 发作起源于**皮质**，故皮质神经元承受最强兴奋性毒性 → **cortical laminar necrosis（皮质层状坏死）**：MRI 见**皮质带状高信号** → 持续神经功能缺损 + 复发性癫痫。这是 SE 最相关的脑损伤后果。

| 干扰并发症 | 为何不是 SE 首要风险 |
|---|---|
| 小脑萎缩 cerebellar atrophy | 慢性**苯妥英 phenytoin**/酒精所致，非急性 SE |
| 梗阻性脑积水 | SE 时 ICP（intracranial pressure，颅内压）一过性升高，不足以致梗阻 |
| 脑桥出血 pontine hemorrhage | 高血压/血管畸形；脑桥**非**发作起源 |
| 继发性帕金森 | 基底节损伤需**严重全脑缺氧**（如心脏骤停）；SE 缺氧程度通常不够 |

## 三、阶梯化处理（管理记忆）

```
0-5 min   稳定 ABC + 监测 + 测血糖 (低糖给葡萄糖+硫胺)
5-20 min  一线: 苯二氮䓬 BZD —— 劳拉西泮 lorazepam IV (或咪达唑仑 midazolam IM)
20-40 min 二线: 负荷抗癫痫药 —— 左乙拉西坦 levetiracetam / 磷苯妥英 fosphenytoin / 丙戊酸 valproate
>40 min   难治性: 气管插管 + 麻醉剂量(咪达唑仑/丙泊酚 propofol) 持续输注 + EEG 监测
```

> [!tip] 考点反射
> 院前已用 BZD 仍发作 → 上**二线负荷药**，并查**继发损伤**（皮质层状坏死）；同时纠正缺氧/低糖/电解质。

## 四、GCSE 治疗后持续不应答 → EEG（查非惊厥性 SE）（Q19890）⭐

> [!danger] 惊厥停了人却不醒 = 上 EEG
> **GCSE（generalized convulsive status epilepticus，全面惊厥性癫痫持续状态）**经 BZD + AED 控制后，多 **10-20 min** 恢复应答。**持续不应答**有两种可能：
> - ① **BZD（benzodiazepine）镇静残留**；
> - ② **非惊厥性 SE（nonconvulsive SE，无外在抽动的持续电发作）**。
> 鉴别二者 → **EEG（electroencephalography，脑电图）**。若 EEG 仍有电发作 → 持续 BZD 输注 / 丙泊酚（propofol）/ 苯巴比妥（难治性 SE）。

| 持续不应答的处理 | 说明 |
|---|---|
| **EEG**（正解）| 鉴别镇静 vs 非惊厥 SE，**优先于影像** |
| flumazenil（氟马西尼）| ❌ 可诱发癫痫，"镇静 vs 电发作"不明确，不用 |
| MRI 脑 | 急症阶段 EEG 优先 |
| 活性炭 | 不吸酒精，意识↓有误吸风险 |

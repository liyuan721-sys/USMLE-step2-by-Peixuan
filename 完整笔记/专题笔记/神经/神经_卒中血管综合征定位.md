# 神经_卒中血管综合征定位（Stroke Vascular Syndromes：症状 → 血管）

> [!success] 🎯 背诵版（一分钟速刷 —— 细节机制看下方各节）
>
> **三步定位**：① 有皮质征（失语/忽视/偏盲/凝视/意识改变）吗？有 = 大血管皮质支；全无 + 纯运动或纯感觉 = **腔隙**。② 面臂 vs 腿谁重？面臂 = **MCA**；腿 + 尿失禁 = **ACA**。③ 有交叉体征（同侧脑神经 + 对侧肢体）吗？有 = **脑干**。
>
> **前循环**
>
> | 血管 | 一句话钉死 |
> |---|---|
> | ACA | 腿 > 臂无力 + 尿失禁 + abulia 意志缺失 |
> | MCA 主干 M1 | 面臂 > 腿 + 完整偏盲 + 凝视看向病灶 + 优势=完全失语/非优势=忽视 |
> | MCA 上分支 | 面臂无力 + Broca 失语，**【无偏盲】**（招牌）|
> | MCA 下分支 | 无瘫 + Wernicke 失语/忽视 + **上**象限盲（Meyer 袢）|
> | 分水岭 | 近端"桶中人" + 经皮质失语（骤停/低血压后）|
>
> **后循环**
>
> | 血管 | 一句话钉死 |
> |---|---|
> | PCA | 偏盲 + 黄斑回避；双侧 = 皮质盲，否认失明 = Anton |
> | 基底动脉 | locked-in：四肢瘫 + 意识清 + 只能垂直眼动/眨眼 |
> | Weber（中脑）| 同侧 CN III 麻痹 + 对侧偏瘫 |
> | Wallenberg/PICA（外侧延髓）| 眩晕 + 声嘶 + 交叉痛温（同侧面/对侧身）+ Horner，**无肢瘫** |
> | 内侧延髓 | 舌偏病灶侧 + 对侧偏瘫 + 对侧振动觉↓ |
> | 外侧脑桥/AICA | 面瘫 + 耳聋眩晕 + 对侧痛温↓ |
>
> **腔隙 lacunar**（"纯"，无皮质征，慢性 HTN/DM）：纯运动 / 纯感觉（丘脑 VPL）/ 共济失调轻偏瘫 / 构音障碍-笨手 / 混合感觉运动。
>
> **两个最易错**
> - **凝视**：皮质**看向病灶**、脑桥**背离病灶**（eyes toward cortical, away from pontine）；癫痫（刺激性）**背离痫灶**（与卒中反）。
> - **视路血供**：唯一"纯 MCA"= **视辐射**；视束 = **前脉络膜动脉**（非 MCA）；枕叶皮质 = **PCA**（黄斑回避）；**MCA 上分支通常无偏盲**（有偏盲反而否定它）。

> [!info] 一句话用法
> 拿到卒中题，先按**三问**把血管圈出来：
> 1. **有没有皮质征**（失语 / 忽视 / 偏盲 / 意识/凝视偏斜）？有 = 大血管皮质支（ACA/MCA/PCA）；**全无皮质征 + 纯运动或纯感觉** = **腔隙 lacunar**（小穿支）。
> 2. **面/臂 vs 腿**谁重？面臂重 = **MCA**；腿重 + 尿失禁 = **ACA**。
> 3. **有没有交叉体征**（同侧脑神经 + 对侧肢体）？有 = **脑干**（后循环）。

```
卒中定位大框架
├─ 前循环 anterior（颈内动脉系）
│   ├─ ACA ── 对侧腿>臂 + 尿失禁 + abulia
│   ├─ MCA主干 ── 对侧面臂>腿 + 偏盲 + 凝视偏向病灶 + 失语/忽视
│   ├─ MCA上分支 ── 面臂无力 + Broca失语（无偏盲）
│   ├─ MCA下分支 ── Wernicke失语/忽视 + 上象限盲（无瘫）
│   └─ 分水岭watershed ── 近端肢体"桶人" + 经皮质失语（全脑低灌注）
├─ 后循环 posterior（椎-基底系）
│   ├─ PCA ── 对侧偏盲 + 黄斑回避；双侧=皮质盲/Anton
│   ├─ 基底动脉 ── locked-in闭锁（腹侧脑桥）
│   └─ 脑干交叉综合征 ── 同侧脑神经 + 对侧肢体（Weber/Wallenberg/内侧延髓）
└─ 腔隙 lacunar（慢性HTN/DM，小穿支）
    └─ 5综合征：纯运动/纯感觉/共济失调轻偏瘫/构音障碍-笨手/混合——皆无皮质征
```

## 一、前循环（Anterior Circulation：颈内动脉 ICA 系统）⭐

| 血管                                      | 运动/感觉                                           | 特征性伴随                                                                                                   | 定位机制                              |
| --------------------------------------- | ----------------------------------------------- | ------------------------------------------------------------------------------------------------------- | --------------------------------- |
| **ACA**（anterior cerebral a.，大脑前动脉）     | 对侧**下肢 > 上肢**无力+感觉减退                            | **尿失禁**、**意志缺失 abulia**/淡漠、额叶释放征、对侧抓握反射                                                                 | 腿的运动皮质在**内侧半球面**（ACA 供）           |
| **MCA 主干**（middle cerebral a.，大脑中动脉 M1） | 对侧**面/臂 > 腿**无力+感觉减退                            | **同向偏盲**（视辐射）+ **双眼凝视偏向病灶侧**（away from 偏瘫）+ 优势=**完全性失语**/非优势=**忽视**                                     | 面臂皮质在外侧凸面（MCA 供）                  |
| **MCA 上分支**（superior division）          | <font color="#ff0000">面/臂无力+感觉</font>（腿相对保留）    | 优势=**<font color="#ff0000">Broca 失语</font>**；**通常无偏盲**                                                  | 供**额叶+Broca+运动皮质**                |
| **MCA 下分支**（inferior division）          | **无/轻微瘫**                                       | 优势=<font color="#ff0000">**Wernicke 失语**</font>/非优势=忽视；<font color="#ff0000">**对侧上象限盲**</font>（Meyer 袢） | 供**颞顶+Wernicke+视辐射下部**            |
| **分水岭 Watershed**（ACA-MCA / MCA-PCA 交界） | **近端肢体**无力（肩+髋，远端保留）= **"桶中人" man-in-a-barrel** | 经皮质失语（复述保留）、后分水岭视觉障碍                                                                                    | **全脑低灌注**（低血压/骤停/颈动脉重度狭窄）饿死最外围边界带 |

### 1.1 凝视偏斜 Gaze Deviation（高频 + 常考机制）⭐

> [!warning] 先分清：偏盲 ≠ 凝视（两件独立的事）
> - **偏盲/偏瘫 = 缺损**，永远在**病灶对侧**（左 MCA → 右偏盲 + 右偏瘫）。
> - **凝视 = 眼球指向**，是另一回事。那个"看向病灶"说的**只是凝视方向，不是偏盲**。
> - **脑桥"相反"只指凝视方向反**；脑桥一般**不产生偏盲**（视觉通路不过脑桥），别说"脑桥偏盲也反"。

**方向速查（以左侧病灶为例）：**

| | 左**皮质**（MCA/FEF）| 左**脑桥**（PPRF）|
|---|---|---|
| 偏瘫 | 右（对侧）| 右（对侧）|
| **凝视方向** | 偏**左** = **看向病灶** | 偏**右** = **背离病灶** |
| 相对偏瘫 | 背对偏瘫 | **看向偏瘫**（wrong-way eyes）|
| 偏盲 | 右（对侧）| 通常无 |

> 口诀：**eyes look toward a cortical lesion, away from a pontine lesion**（皮质看向病灶、脑桥背离病灶）。

#### 为什么会凝视（机制）= 两侧持续"拔河"，一侧断绳

> [!info]- 机制：水平凝视是 tonic 张力平衡，不是一次性动作
> 每侧**额叶眼动区 FEF (frontal eye field)** 持续把眼睛推向**对侧**；两边等大 → 抵消 → 眼球居中。**破坏性病灶剪断一侧的绳** → 健侧张力**无对抗** → 眼球被拽向健侧所推的方向。这就是"为什么会凝视"——不是主动看病灶，是**被动失衡**。
> 
> ```
> 正常：左 FEF ─推向右─►   ◄─推向左─ 右 FEF   → 抵消，眼球居中
> 左 FEF 梗死：左 FEF ✗ → 右 FEF「推向左」无对抗 → 眼球偏【左】= 病灶侧
> ```
> 
> 完整水平凝视通路（decussation 是皮质/脑桥方向相反的根源）：
> 
> ```
> FEF（额叶皮质）
>   │  ⭐ 通路在脑桥【交叉 decussate】
>   ▼
> 对侧 PPRF（脑桥旁正中网状结构，水平凝视中枢）
>   │
>   ▼
> 同侧 CN VI 核（外展核 abducens）
>   ├─►同侧 外直肌（该眼外展 abduct）
>   └─►经 MLF（内侧纵束）→ 对侧 CN III 内直肌 →对侧眼内收 adduct
>        ↓
>    双眼协同 conjugate 转向一侧
> ```
> 
> **为什么皮质 vs 脑桥方向相反 = 同一原理 + 一次交叉**：原理都是"**健侧张力无对抗**"，只因通路在脑桥前后**交叉**，驱动方向不同：
> - **FEF 推「对侧」** → 坏了眼球偏向**病灶同侧**（看向病灶）
> - **PPRF 推「同侧」** → 坏了眼球偏向**病灶对侧**（背离病灶）

> [!danger] 高频反转：破坏 vs 刺激
> 上面是**破坏性**病灶（卒中）。**癫痫发作（刺激性 irritative）** = 给痫灶那侧 FEF **加大马力** → 眼睛被推向**对侧**（背离痫灶、看向抽搐肢体）→ 和皮质卒中**正好相反**。UWorld 用这个区分"卒中 vs 发作后 Todd 麻痹"。

### 1.2 "桶中人" Man-in-a-barrel（分水岭梗死的招牌）⭐

> [!info] 一句话
> **双侧近端肢体无力（肩 + 髋）+ 远端手脚保留** = man-in-a-barrel，见于**全脑低灌注 → ACA-MCA 分水岭（watershed / border-zone）梗死**。

**症状画面：**

```
受累（无力）              保留（正常）
├─ 肩 shoulder            ├─ 手/手指 hand
├─ 上臂 upper arm         ├─ 前臂远端
├─ 髋 hip（有时）          └─ 足/脚趾 foot
└─ 大腿近端
```

- **抬不起胳膊**（够高处/梳头难）、**从椅子站起难**（髋近端），但**手精细动作、抓握、脚趾正常**
- 典型**双侧对称**（不是单侧偏瘫）；胳膊像被桶壁夹住只有露在外的手能动 → "桶中人"

> [!info]- 机制：为什么近端受累、远端保留
> ```
> 运动小人 homunculus 的血供地理
> 外侧凸面(MCA)————内侧面(ACA)
>     手/面 ←←← 肩臂髋(边界带) →→→ 腿
>                   ↑
>           ACA-MCA 分水岭：两血管供血末梢，全脑低灌注时最先饿死
> ```
> 
> - 控制**肩/髋近端**的皮质正落在 **ACA 与 MCA 交界的分水岭带**；控制**手**（MCA 核心）和**腿远端**（ACA 核心）在各自血管腹地 → 低灌注时边界带先缺血 → 近端先挂、远端保留。

> [!tip] stem 触发（全脑低灌注病史）
> 心脏骤停复苏后 / 术中大低血压 / 双侧颈动脉重度狭窄 + 血压骤降 → 常**同时伴经皮质型失语**（复述保留 repetition intact，语言分水岭区受累而 Broca/Wernicke 核心保住）。
> - **前分水岭（ACA-MCA）** → 肢体"桶中人"
> - **后分水岭（MCA-PCA）** → 偏**视觉**症状（视野/视觉失认），非桶人

> [!warning] 别混
> 也是近端无力，但 man-in-a-barrel 是**中枢/皮质**病因（有低灌注病史 + 常伴皮质征），**不是** neuromuscular（肌病/GBS 吉兰-巴雷）。

> [!note]- 顺带：abulia 意志缺失（ACA 招牌之一）
> **abulia = 发起动作/意志的严重减退**——不是瘫，是"懒得动、懒得说、反应慢（latency↑）、情感淡漠 apathy"，结构功能好、强刺激能应答。见于**额叶内侧面（扣带回前部/辅助运动区）损伤 = ACA 卒中**。加重 → **无动性缄默 akinetic mutism**（睁眼但不动不语）。⚠️ 别混抑郁（有情绪内容）/ 失语（语言结构在）。词源 a-（无）+ boulē（意志）。

## 二、后循环（Posterior Circulation：椎-基底 vertebrobasilar 系统）⭐

### 2.1 PCA（posterior cerebral a.，大脑后动脉）

- **对侧同向偏盲 + 黄斑回避 macular sparing**（枕叶，黄斑区有 MCA 侧支供血）。
- 优势侧枕叶 + 胼胝体压部 splenium 断连 → **失读不伴失写 alexia without agraphia**。
- **双侧 PCA** → **皮质盲 cortical blindness**；若否认失明 = **Anton 综合征**（anosognosia for blindness）。
- 近端 PCA 累及丘脑 → 对侧偏身感觉缺失 ± 丘脑痛。

### 2.2 基底动脉 Basilar Artery

> [!danger] 闭锁综合征 Locked-in Syndrome（腹侧脑桥梗死）
> **四肢瘫 + 不能说话/吞咽，但意识清醒、能垂直眼动 + 眨眼**（动眼纵向通路在中脑背侧幸免）。靠眨眼/上下看交流。基底动脉血栓致腹侧脑桥（皮质脊髓 + 皮质延髓束）双侧断。**别误判为昏迷/植物状态**。

### 2.3 脑干交叉综合征（Crossed Syndromes）= 同侧脑神经 + 对侧肢体 ⭐

| 综合征 | 血管 | 同侧（病灶侧）| 对侧（身体）|
|---|---|---|---|
| **Weber**（中脑腹侧）| PCA/基底穿支 | **CN III 麻痹**（眼向下外、上睑下垂、瞳孔散大）| 偏瘫（皮质脊髓束）|
| **Wallenberg / 外侧延髓**（lateral medullary）| **PICA**（或椎动脉）| 面部痛温觉缺失（CN V 脊束）、**Horner**、吞咽困难/声嘶（疑核 nucleus ambiguus）、共济失调、眩晕/眼震 | **躯体痛温觉缺失**（脊髓丘脑束）—— **面对侧交叉感觉** |
| **内侧延髓**（medial medullary）| 椎动脉/脊髓前动脉 ASA | **舌偏向病灶侧**（CN XII）| 偏瘫（锥体）+ 本体/振动觉缺失（内侧丘系）|
| **外侧脑桥**（lateral pontine）| **AICA**（anterior inferior cerebellar a.，小脑前下动脉）| 面瘫（CN VII）+ 耳聋/眩晕（CN VIII）+ 面部感觉 | 躯体痛温觉缺失 |

> [!warning] Wallenberg 记忆
> PICA（posterior inferior cerebellar a.，小脑后下动脉）外侧延髓 = "**吞咽 + 眩晕 + 交叉痛温 + Horner + 共济失调**"，**没有肢体无力**（腹侧锥体幸免）。经典 stem：突发眩晕 + 声嘶/呛咳 + 同侧面/对侧身麻木 + 同侧上睑轻下垂缩瞳。

## 二·补、视路血供分段与视野缺损定位（哪段是 MCA）⭐

### 视路血供分段表（前 → 后）

| 定位点 | 血供动脉 | 典型视野缺损 | 是否 MCA |
|---|---|---|---|
| 黄斑/视网膜 Macula/retina | 眼动脉 → 视网膜中央 a. | 中心暗点 central scotoma | ✗ |
| 视神经 Optic nerve | 眼动脉 Ophthalmic a.（ICA 分支）| 同侧单眼全盲 anopia | ✗ |
| 视交叉 Optic chiasm | ICA / 前交通 / 垂体上小支 | 双颞侧偏盲 bitemporal | ✗ |
| **视束 Optic tract** | **前脉络膜动脉 Anterior choroidal a.**（ICA 直接分支）| 对侧同向偏盲 | ✗（陷阱①）|
| **视辐射·Meyer 袢**（颞叶下束，携带上部视野）| 颞叶 = **MCA 下分支** | 对侧**上**象限盲 "pie in the sky" | ✓ |
| **视辐射·背侧束**（顶叶上束，携带下部视野）| 顶叶深部（深部 MCA）| 对侧**下**象限盲 "pie on the floor" | 深部 |
| 枕叶视皮质 Visual cortex | **PCA** | 对侧同向偏盲 + **黄斑回避** | ✗（陷阱②）|

> [!warning] ⚠️ 别把"视辐射上/下束"和"MCA 上/下分支"划等号（我曾在此栽跟头）
> 两个"上/下"是**不同维度**：**视辐射上/下束 = 按脑叶分**（顶叶上束 vs 颞叶下束，白质束解剖）；**MCA 上/下分支 = 按血管分**（superior/inferior division，血管临床）。别互相翻译。
> 
> 临床 MCA 分支综合征（可考版）实际是：
> - **MCA 上分支 → 通常无偏盲**：皮质 territory = 额叶 + 上部额顶，**位于深部视辐射之上、够不到视辐射** → 招牌 = Broca 失语 + 面臂无力**但视野正常**；出现偏盲/象限盲反而**否定**上分支。
> - **MCA 下分支 → 上象限盲**：Meyer 袢在**颞叶**（= 下分支 territory）→ Wernicke/忽视 + 上象限盲、无瘫。
> - **M1 主干 → 完整同向偏盲**：深部把整个视辐射一起端掉。
> - **下象限盲 "pie on the floor" = 顶叶病变**的定位征，**不是 MCA 上分支的临床招牌**（多见于顶叶深部/下分支向后延伸/深穿支）。

> [!success] 结论：象限盲按"哪个脑叶"记，别硬套 MCA 分支名
> - **Meyer 袢（颞叶，下束）→ 上象限盲** —— 颞叶 = **MCA 下分支** ✓
> - **背侧束（顶叶，上束）→ 下象限盲** —— 顶叶深部病变（≠ MCA 上分支招牌）
> - **M1 主干把整个视辐射端掉 → 完整同向偏盲**（= §一表 MCA 主干那行）
> - **MCA 上分支（额叶+上部）在视辐射之上 → 通常无偏盲**（这是区分上分支的关键）

### 视辐射上束 vs 下束（携带哪半视野）

![[{06B5CCED-072F-4C61-8A0B-CC8A0F28C1B2}.png]]

```
视辐射（optic radiation）分上下两束，视野"上下颠倒"
（⭐ 束的上/下 = 按脑叶；≠ MCA 上/下分支）
├─ 下束 = Meyer 袢（颞叶绕行）
│    携带 上部视野（来自下部视网膜）
│    损伤 → 对侧 上 象限盲 "pie in the sky"
│    （颞叶 = MCA 下分支 territory）
└─ 上束 = 背侧视辐射（顶叶直行）
     携带 下部视野（来自上部视网膜）
     损伤 → 对侧 下 象限盲 "pie on the floor"
     （顶叶深部病变；≠ MCA 上分支的招牌——上分支通常无偏盲）
```

> [!danger] 两大"不是 MCA"陷阱（高频）
> - **陷阱①·视束 = 前脉络膜动脉**：完整对侧同向偏盲、病灶在视束时想到 **anterior choroidal a.**，不是 MCA。AchA 梗死经典三联 = 对侧偏瘫（内囊后肢）+ 偏身感觉缺失（丘脑）+ 对侧同向偏盲（视束/外侧膝状体）。
> - **陷阱②·枕叶视皮质 = PCA**："同向偏盲 + **黄斑回避 macular sparing**"是 PCA 招牌。黄斑回避的原因正是**枕极黄斑代表区常有 MCA 侧支双重供血** → PCA 挂了这块仍存活，中央视力保留。双侧 PCA = 皮质盲；否认失明 = **Anton 综合征**。

## 三、腔隙性梗死（Lacunar Infarcts：小穿支，慢性 HTN/DM，脂质透明变性）⭐

> [!info] 铁律：腔隙 = 纯，无皮质征
> 小穿支供**皮质下**（内囊 internal capsule、丘脑、脑桥基底、基底节）→ **绝无失语/忽视/偏盲/意识改变/凝视偏斜**（那些都是皮质征）。危险因素 = 长期**高血压/糖尿病**。5 大经典综合征：

| 综合征 | 表现 | 常见定位 |
|---|---|---|
| **纯运动 Pure motor** | 对侧面+臂+腿等重无力，**无感觉/皮质征** | 内囊后肢 / 脑桥基底 |
| **纯感觉 Pure sensory** | 对侧偏身麻木，无运动 | **丘脑 VPL** |
| **共济失调轻偏瘫 Ataxic hemiparesis** | 无力 + 不成比例的同侧共济失调 | 内囊后肢 / 脑桥 |
| **构音障碍-笨手 Dysarthria-clumsy hand** | 说话含糊 + 一只手笨拙 | 脑桥基底 / 内囊膝 |
| **混合感觉运动** | 对侧偏身无力 + 麻木 | 丘脑 + 邻近内囊 |

## 四、症状钩子 → 血管（一句话反射表）⭐

| stem 钩子 | 锁定 |
|---|---|
| 腿无力 > 臂 + 尿失禁 | **ACA** |
| 面臂无力 + 失语/忽视 + 偏盲 | **MCA 主干** |
| 面臂无力 + Broca 失语（无偏盲）| **MCA 上分支** |
| Wernicke 失语/忽视 + 无瘫 | **MCA 下分支** |
| 骤停/低血压后近端"桶人"无力 + 复述保留 | **分水岭 watershed** |
| 偏盲 + 黄斑回避 / 皮质盲否认 | **PCA**（双侧 = Anton）|
| 四肢瘫但意识清、能眨眼上下看 | **基底动脉**（locked-in）|
| 同侧动眼麻痹 + 对侧偏瘫 | **Weber**（中脑）|
| 眩晕 + 声嘶 + 同侧面/对侧身痛温缺失 + Horner，无肢瘫 | **Wallenberg / PICA** |
| 舌偏一侧 + 对侧偏瘫 + 对侧振动觉缺失 | **内侧延髓** |
| 纯运动 / 纯感觉，无皮质征，长期 HTN | **腔隙 lacunar** |

## 五、易混陷阱

> [!danger]
> - **腔隙 vs 皮质**：题里出现**任何**失语/忽视/偏盲/凝视/意识改变 → **排除腔隙**；腔隙必须"纯"。
> - **ACA 的腿 vs MCA 的面臂**：搞反 = 送分题丢分。腿在内侧（ACA）、面臂在外侧凸面（MCA）。
> - **凝视偏斜**：皮质看向病灶、脑桥看向偏瘫侧，方向相反别记混。
> - **Wallenberg 无肢体无力**：若 stem 给了明显偏瘫，重新想内侧延髓或脑桥，而非外侧延髓。
> - **locked-in ≠ 昏迷**：意识清醒 + 垂直眼动保留是关键，别当植物状态处理。

## Memory Hook

> [!success]
> - **ACA = leg**（内侧腿区）+ 尿失禁 + abulia
> - **MCA = face/arm + 语言/忽视**（外侧凸面）；上分支=Broca、下分支=Wernicke
> - **PCA = 视觉**（偏盲+黄斑回避；双侧=Anton 否认失明）
> - **交叉体征 = 脑干**：同侧脑神经数字帮定层（III=中脑 Weber / 延髓=Wallenberg,XII 内侧延髓）
> - **腔隙 = 纯**（无皮质征 = 无失语无忽视无偏盲），慢性 HTN

## 复习题

> [!question]- Q1. 长期高血压男性突发左侧面、臂、腿等重无力，无感觉障碍、无失语、无偏盲，意识清醒。最可能的病灶 + 血管类型？
> **纯运动性腔隙梗死 Pure motor lacunar**，定位**内囊后肢或脑桥基底**，小穿支病变（慢性 HTN 脂质透明变性）。关键：**完全无皮质征** → 排除 MCA 皮质支。

> [!question]- Q2. 突发眩晕、声音嘶哑、吞咽困难；查体右侧面部痛温觉减退、左侧躯体痛温觉减退、右侧 Horner、右侧肢体共济失调，**四肢肌力正常**。综合征 + 血管？
> **Wallenberg（外侧延髓）综合征**，**PICA / 椎动脉**。交叉性痛温觉缺失（同侧面 / 对侧身）+ Horner + 疑核（声嘶吞咽）+ 无肢体无力是钉死点。

> [!question]- Q3. 心脏骤停复苏后患者双肩、双髋近端无力而远端手足力量相对保留（"桶中人"），并有经皮质型失语。缺血机制 + 部位？
> **全脑低灌注 → 分水岭（watershed）梗死**（ACA-MCA / MCA-PCA 边界带）。近端肢体皮质代表区正处边界带 → man-in-a-barrel；复述保留的经皮质失语同源。

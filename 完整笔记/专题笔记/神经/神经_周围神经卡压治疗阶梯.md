# 周围神经卡压综合征治疗阶梯 (Entrapment Neuropathy Treatment Ladder)

> [!tip] 一句话总览
> 几乎所有**周围神经卡压 (entrapment / compression neuropathy)** 都共用一条三级阶梯：**保守（活动调整 + 夹板/支具 ± NSAIDs）→ 局部糖皮质激素注射 → 手术减压/松解 (surgical decompression / release)**。选答案的钥匙只有一个：**症状严重度 + 有无运动受累（无力 / 肌萎缩 / EMG 失神经）**。轻症（仅感觉）→ 保守；有运动受累 / 保守失败 / 急性外伤致急性卡压 → 跳级到注射或手术。

---

## §一 通用三级治疗阶梯（适用于绝大多数卡压）

```
┌─────────────────────────────────────────────────────────┐
│  STEP 1  保守 (first-line, 轻–中症 / 仅感觉症状)            │
│    • 活动调整 / 避免诱发动作 (activity modification)        │
│    • 夹板 / 支具 (splinting / bracing) — 夜间或制动诱发位   │
│    • ± NSAIDs（非甾体抗炎药）（孕妇避免口服）              │
├─────────────────────────────────────────────────────────┤
│  STEP 2  局部糖皮质激素注射 (glucocorticoid injection)      │
│    • 指征：保守 ≥ 数周无效 / 中症顽固                       │
│    • 短期缓解为主，疗效可不持久                            │
├─────────────────────────────────────────────────────────┤
│  STEP 3  手术减压 / 松解 (decompression / release)         │
│    • 指征（任一即可，常需跳级）：                          │
│      ① 运动受累：无力 / 肌萎缩 (atrophy)                   │
│      ② EMG（肌电图）/ NCS（神经传导检查）失神经           │
│      ③ 保守 + 注射均失败                                  │
│      ④ 急性外伤 / 移位骨折致急性卡压 → 紧急               │
└─────────────────────────────────────────────────────────┘
```

> [!danger] 手术「即刻 / 跳级」红旗（不要再从 STEP 1 慢慢爬）
> - **进行性运动无力 / 肌萎缩** → 直接手术（拖延 = 不可逆肌损）
> - **EMG/NCS 提示失神经 (denervation / axonal loss)** → 手术
> - **急性外伤、移位骨折、骨筋膜室** 致**急性卡压** → 紧急减压 / 复位
> - 这些情况下「再试夹板」「先注射」都是错答案。

## §二 升级题识别（与 [[完整笔记/专题笔记/_衍生_治疗阶梯升级SOP]] 同框架）

> [!warning] severity → 阶梯映射（最常考的一步）
> stem 给的「严重度描述」直接决定阶梯，看清楚有没有**运动词**（weakness / atrophy / wasting）。

| stem 严重度线索 | 阶梯档位 | 答案方向 |
|---|---|---|
| 仅**感觉**症状（麻 / 刺痛 / 烧灼），夜间为主，**无萎缩无力** | 轻–中 | **保守：夹板 / 活动调整** |
| 保守 ≥ 数周仍**持续 / 反复** | 中（顽固）| **局部激素注射** |
| **大鱼际 / 手内在肌萎缩**、对掌或外展**无力**、EMG 失神经 | 重 | **手术减压** |
| **急性外伤 / 移位骨折**后**新发**感觉运动缺失 | 急性 | **紧急复位 / 手术** |

> [!info] 反例：别把「保守」无限延后
> 孕期 CTS（[[mistakes/uworld-mistakes_2026-06#^Q4636]]）中「**No treatment until delivery 等产后再治**」是错的——夹板既安全又**即刻**缓解，没有理由拖。「保守」不等于「不治」。

## §三 各卡压综合征速查表（神经 / 部位 / 钩子 / 治疗）

> [!info] 表内治疗均以**典型轻–中症一线**为准；出现运动受累 / 萎缩 / 失神经一律升级到手术减压。

| 综合征 | 卡压神经＠部位 | 经典 stem 钩子 | 一线（保守）| 手术升级钥匙 |
|---|---|---|---|---|
| **腕管综合征 CTS** (carpal tunnel) | 正中神经 median ＠腕横韧带 | 拇/食/中/桡半无名指麻、**夜间加重**、甩手缓解；Phalen/Tinel+；孕期/甲减/肢端肥大/RA/透析 | **夜间腕夹板** ± NSAIDs | 大鱼际萎缩 / 拇外展对掌无力 / 失神经 → 松解 |
| **肘管综合征** (cubital tunnel) | 尺神经 ulnar ＠肘（肘后沟）| 第 4–5 指麻 + 手内在肌弱；**leaning on elbow**、长期屈肘、透析体位 | 避免屈肘 / 夜间**伸肘夹板** + 护肘 | 骨间肌萎缩 / 爪形手 / 失神经 → 尺神经松解或前置 |
| **Guyon 管综合征** | 尺神经 ulnar ＠腕（Guyon 管）| 第 4–5 指麻 + 手内在肌弱、**不影响腕屈**；**handlebar palsy 自行车把**、钩骨钩骨折 | 避免压迫 / 护腕 | 运动受累 / 占位（钩骨骨折、神经节囊肿）→ 手术 |
| **桡神经麻痹** (radial / spiral groove) | 桡神经 radial ＠肱骨桡神经沟 | **wrist drop 垂腕** + 手背桡侧麻；**Saturday night palsy**（醉酒压臂）、肱骨干骨折 | **腕背伸夹板** + 观察（多自愈）| 多数神经失用 (neurapraxia)，预后好；**穿透伤 / 骨折断端嵌入**才手术 |
| **感觉异常性股痛** (meralgia paresthetica) | 股外侧皮神经 (LFCN, lateral femoral cutaneous nerve) ＠腹股沟韧带 | **大腿前外侧纯感觉**麻/灼痛、**无运动受累**；肥胖、紧身裤/腰带、孕期 | **减重 + 松开紧衣束带** ± NSAIDs/局部药 | 纯感觉，几乎**不手术**；顽固极少数松解 |
| **腓总神经麻痹** (common peroneal / fibular) | 腓总神经 ＠腓骨头 | **foot drop 足下垂** + 足背/小腿外侧麻；**跷二郎腿、长期下蹲、石膏/绑腿压迫、消瘦** | 去除压迫 + **踝足矫形器 (AFO, ankle-foot orthosis)** | 多神经失用自愈；占位 / 完全断裂 → 手术 |
| **跗管综合征** (tarsal tunnel) | 胫神经 tibial ＠内踝屈肌支持带 | **足底**麻痛、负重加重；扁平足、外伤、占位 | 矫形鞋垫 / 夹板 ± NSAIDs ± 注射 | 占位（腱鞘囊肿）/ 保守失败 → 松解 |

> [!tip] 尺神经「肘 vs 腕」一句话区分
> **腕屈无力 = 肘部 (cubital)；腕屈正常 = 腕部 (Guyon)**。因为支配尺侧腕屈肌 (FCU, flexor carpi ulnaris) 的分支在肘以下、腕以上发出——腕部卡压绕过它。详见 [[完整笔记/Peixuan分科笔记/neuro]] 上肢神经定位表。

## §四 腕管综合征 CTS 详解（最高频，单独展开）

![[{FF53B663-CCDE-4C06-9BE8-C57AA19AA288}.png|400]]
*腕管掌侧观 (carpal tunnel, palmar view)：屈肌支持带 (transverse carpal ligament) 深面正中神经 (median nerve) 受压解剖。*

![[{799A8936-C797-4B27-AACA-B68C53A49E5B}.png|400]]
*正中神经 (median nerve) 运动（拇指屈/对掌、2-3 指屈、旋前）+ 皮肤感觉（掌侧桡侧 3 指半 + 掌皮支）。*

![[{2F413AAD-21A8-48CC-8932-2CDAE07F3C04}.png|400]]
*正中神经手内分支：返支 (recurrent branch) 支配大鱼际 (thenar：APB/FPB/opponens)，掌皮支管掌侧桡侧感觉。*

![[{7680AB51-4008-4DB3-B390-C71BB79ECA7C}.png|400]]
*正中神经前臂走行（旋前圆肌、指浅/深屈肌、拇长屈肌）经腕管入手。*

> [!success] CTS 三级阶梯（默写级）
> 1. **轻–中症（仅感觉、无萎缩无力）** → **夜间腕部夹板 nocturnal wrist splint** ± 口服 NSAIDs
> 2. **顽固 / 中症** → **局部糖皮质激素注射 glucocorticoid injection**
> 3. **重症（大鱼际萎缩 thenar atrophy / 拇外展-对掌无力 / EMG 失神经）** → **腕管松解减压手术 carpal tunnel release**

> [!info] 诊断金标准
> 确诊 = **神经传导检查 NCS (nerve conduction study) / EMG**。**Phalen 征、Tinel 征敏感性低**，阴性不能排除；它们是床旁筛查，不是确诊。

![[{A34B6381-E0DC-41BC-9198-90C5567D531F}.png|400]]
*Tinel 征：轻叩腕管区诱发正中神经分布区刺痛/麻刺感。*

![[{D4AFE146-F0A5-4699-8FA7-28B0EC45FEE1}.png|400]]
*Phalen 试验：屈腕压迫腕管内正中神经，约 1 分钟内诱发麻木。*

### 特殊情境

> [!warning] 孕期 CTS（[[mistakes/uworld-mistakes_2026-06#^Q4636]]）
> 病因 = 激素 + 体液潴留压迫 → **多在产后自行缓解**。
> - 一线 = **夜间夹板**（安全 + 即刻缓解）
> - 口服 NSAIDs / 系统激素孕期尽量避免；局部激素注射虽相对安全但「夹板能控就不上」
> - ❌「等产后再治」「直接注射 / 手术」都是越级或拖延

> [!danger] 急性 CTS（外伤性，[[mistakes/uworld-mistakes_2026-06#^Q16328]]）
> **Colles 骨折（桡骨远端背侧移位）→ 压迫正中神经 → 急性腕管综合征** → 拇短展肌 (abductor pollicis brevis) 无力（拇对抗阻力外展弱）+ 桡侧 3½ 指感觉异常。
> - 这是**急性 + 外伤 + 进行性**的卡压 → **紧急复位 / 减压**，**不走「夹板观察」慢阶梯**。

> [!question]- CTS 的继发病因联想（stem 常埋伏笔）
> 看到 CTS + 这些线索，想到背后系统病：
> - **妊娠 pregnancy** → 体液潴留（产后缓解）
> - **甲减 hypothyroidism** → 黏液性水肿压迫
> - **肢端肥大 acromegaly** → 软组织增生
> - **类风湿关节炎 (RA, rheumatoid arthritis)** → 滑膜炎
> - **糖尿病 DM** → 易感
> - **长期透析 dialysis（DRA, β2-微球蛋白淀粉样变）** → 横韧带沉积（见 [[mistakes/uworld-mistakes_2026-05#^Q21041]]：透析 >5–10 yr + 双肩痛 + 腕管 + bone cysts 三联）

## §五 易混陷阱 / 反 pattern

> [!danger] 高频踩坑
> - **见 CTS / 任何卡压就跳注射或手术** = 越级。轻症（仅感觉、无萎缩）**永远先夹板 / 活动调整**。
> - **桡神经 Saturday night palsy（垂腕）** → 多为神经失用 (neurapraxia)，**夹板 + 观察即可自愈**，别急着手术（除非穿透伤 / 骨折嵌入）。
> - **感觉异常性股痛 meralgia paresthetica** → **纯感觉、无运动**，治疗是**减重 + 松开束带**，**几乎不手术**；选「手术减压」是陷阱。
> - **腓总神经足下垂** → 一线是去除压迫 + **AFO 支具**；提醒患者**别跷二郎腿 / 长蹲**。
>   - ⚠️ **vs MS（multiple sclerosis，多发性硬化）别混**（Q12243 错点）：腓总压迫缺损**数小时**自愈 + 严格落**单一腓总神经分布**（足/趾背屈 + 足背/外侧小腿）；**MS 缺损数天~数周才缓、跟中枢白质跨解剖走**。有 MS 家族史 + 反复短暂发作时尤其要靠"缓解速度 + 是否单一周围神经"分开。源 [[mistakes/uworld-mistakes_2026-07#^Q12243]]。
> - **急性外伤性卡压**（移位骨折、骨筋膜室）→ 与慢性卡压完全不同，**紧急复位 / 减压**，不要套用慢阶梯。
> - **运动受累（萎缩 / 无力）出现** = 阶梯立刻跳到手术档，别再「先保守」。

*手部皮节 (dermatomes C6/C7/C8) vs 周围神经皮支 (median/ulnar/radial) 对照——区分神经根病 vs 卡压。*

![[L44427_Q12243.jpg]]
*腓总神经病解剖（UWorld）：腓总神经绕**腓骨头**表浅走行，久蹲/盘腿压迫（右图）→ **踝背屈无力（足下垂）+ 趾伸展弱 + 足背/小腿外侧感觉↓**。浅支=足外翻+感觉，深支=足/趾背屈。（双嵌自卡 [[mistakes/uworld-mistakes_2026-07#^Q12243]]）*

## §六 自测复习题

> [!question]- Q1. 孕 30 周女，右手桡侧三指夜间烧灼麻痛、甩手缓解，无大鱼际萎缩、拇对掌正常，Tinel +。最佳下一步？
> **夜间腕部夹板 (nocturnal wrist splinting)**。轻症（仅感觉、无运动受累）一线 = 夹板；孕期 CTS 产后多自愈，避免越级到注射 / 手术，也不能「等产后再治」（夹板既安全又即刻）。

> [!question]- Q2. 何时 CTS 应直接考虑手术减压，而非夹板或注射？
> 出现**运动受累**：大鱼际萎缩 (thenar atrophy)、拇外展 / 对掌无力，或 EMG/NCS 提示失神经；以及保守 + 注射均失败、急性外伤致急性卡压（如 Colles 骨折）。

> [!question]- Q3. 长跑运动员习惯久蹲，出现足下垂 + 小腿外侧/足背麻。卡压的是哪条神经？一线处理？
> **腓总神经 (common peroneal / fibular nerve) ＠腓骨头**。多为神经失用、可自愈；处理 = 去除压迫（别久蹲 / 跷腿）+ **AFO 踝足矫形器**。

> [!question]- Q4. 肥胖男性大腿前外侧片状麻木灼痛、无任何肌力下降、无反射改变。诊断与治疗？
> **感觉异常性股痛 (meralgia paresthetica)** — 股外侧皮神经卡压，**纯感觉**。治疗 = **减重 + 松开紧身衣裤/腰带** ± NSAIDs；几乎不手术。

> [!question]- Q5. 透析 20 年患者，双肩痛 + 腕管症状 + X 线 radiolucent bone cysts 三联。腕管症状的机制？
> **透析相关淀粉样变 (DRA, β2-微球蛋白)**：β2-微球蛋白不能被透析膜清除，沉积于腕横韧带 → 正中神经卡压。属继发性 CTS。

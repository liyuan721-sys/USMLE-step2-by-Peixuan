# UMN + LMN 并存鉴别 — "同一个病人上下运动神经元都中招"

> [!info] 起源与定位
> 从"同时累及 UMN(upper motor neuron, 上运动神经元)和 LMN(lower motor neuron, 下运动神经元)的有哪些"这一问题建立。此前 vault 里相关内容散在 [[完整笔记/专题笔记/神经/神经_ALS运动神经元病]]（以 ALS 为核心）和 [[完整笔记/专题笔记/神经/神经_脊髓病变]]（B12 SCD / Syringomyelia / Conus 各自归在综合征框架下），**没有一个以"哪些病同时累及 UMN + LMN"为切入点的横向入口**。本笔记补这个缺口：一张鉴别表 + 一条决策树，把 ALS、B12 亚急性联合变性、Friedreich、颈椎病性脊髓病、铜缺乏、Taboparesis 拉平对比。

---

## §一 一句话主线 + 决策树

> [!tip] Step 2 核心分叉
> 看到**"萎缩 / 肌束颤 / 反射消失"这类 LMN 征** 和 **"反射亢进 / Babinski+ / 痉挛"这类 UMN 征** 出现在**同一个病人**身上 →
> **第一刀永远先切"有没有感觉 / 后索受累"**：
> - **纯运动、无感觉 → ALS**（amyotrophic lateral sclerosis, 肌萎缩侧索硬化）。
> - **带感觉受累(尤其振动 / 本体觉↓ = 后索) → 想可治病因**：B12 / 铜缺乏 / Friedreich / 颈椎病性脊髓病 / 神经梅毒。

```
同一患者: LMN 征(萎缩/纤颤/反射↓) + UMN 征(反射↑/Babinski+/痉挛)
              │
     ┌────────┴─────────┐
   无感觉受累          有感觉受累
   (纯运动)           (振动/本体觉↓ = 后索受累)
     │                     │
    ALS ⭐         ┌───────┼─────────┬────────────┐
  (排他诊断)       │       │         │            │
              大细胞贫血  青少年+   老年+颈痛+   梅毒史+
              /素食/PPI   共济失调+  感觉平面+    闪电痛+
              /N2O       心肌病+    Hoffmann征   Argyll-
                │        弓形足        │        Robertson瞳孔
             B12 SCD    Friedreich  颈椎病性     │
             (或铜缺乏,   共济失调    脊髓病    Taboparesis
              B12正常时想)              │       (神经梅毒)
                                    可治! MRI必做
```

> [!info] 决策树里的缩写全称
> **PPI** = proton pump inhibitor(质子泵抑制剂);**N2O** = nitrous oxide(一氧化二氮,笑气);两者长期使用/滥用均致 **B12(维生素 B12, cobalamin, 钴胺素)** 缺乏。**MRI** = magnetic resonance imaging(磁共振成像)。

> [!danger] ALS 是排他性临床诊断
> 只要出现"**不该有**"的体征——**感觉缺失 / 眼外肌麻痹 / 括约肌障碍 / 单纯 UMN 或单纯 LMN**——都要先排除可治病因,不能贸然扣 ALS 帽子。详见 [[完整笔记/专题笔记/神经/神经_ALS运动神经元病]] §四。

## §二 横向鉴别表 ⭐ 终极速查

> [!success] 这张表把"UMN+LMN 并存"的所有主角一次拉平

| 疾病 | LMN 表现 | UMN 表现 | 感觉 / 后索 | 招牌识别钩子 | 可治? |
|---|---|---|---|---|---|
| **ALS(肌萎缩侧索硬化)** ⭐ | 肌萎缩、肌束颤(含**舌肌纤颤**)、无力 | 反射亢进、Babinski+、痉挛、假性球麻痹 | **完全正常**(纯运动) | 纯运动 + 无感觉 + 无眼外肌/括约肌受累;不对称进展;40s 起病 | ❌(利鲁唑 riluzole 仅延缓) |
| **B12 亚急性联合变性(Subacute Combined Degeneration, SCD)** | 踝反射消失、周围神经病(袜套感觉↓) | 膝反射亢进、Babinski+、痉挛步态 | **后索↓(振动/本体觉)** + Romberg+ | 大细胞贫血、舌炎、素食/恶性贫血/N2O;神经症状可先于贫血 | ✅ 肌注维生素 B12(IM B12) |
| **铜缺乏性脊髓神经病(copper deficiency myeloneuropathy)** | 周围神经病、反射↓ | Babinski+、痉挛 | **后索↓**(几乎完全模拟 SCD) | **胃旁路术后 / 过量补锌 / B12 正常但像 SCD** | ✅ 补铜 + 停锌 |
| **Friedreich 共济失调(Friedreich ataxia)** | **深反射(DTR, deep tendon reflex)消失** | **跖反射伸性(Babinski+)** | **后索↓** + 小脑性共济失调 | 青少年 + 共济失调 + 脊柱侧弯 + 弓形足 + **肥厚型心肌病(hypertrophic cardiomyopathy)** + 糖尿病;GAA 重复(frataxin) | ❌(对症) |
| **颈椎病性脊髓病(cervical spondylotic myelopathy, CSM)** | 受压节段根性(**上肢**萎缩/反射↓) | 受压平面**以下**(**下肢**痉挛/反射↑/Babinski+) | **有感觉平面 + 颈痛** | 老年、Hoffmann征、Lhermitte征、括约肌障碍;MRI 脊髓受压 | ✅ 手术减压 |
| **Taboparesis(脊髓痨+麻痹性痴呆,神经梅毒晚期)** | 后索受累致 DTR 消失、感觉性共济失调 | 麻痹性痴呆的 UMN 征 | **后索↓** + 闪电样痛 | **Argyll-Robertson 瞳孔** + Charcot 关节 + 未治梅毒史 + 痴呆 | ✅ 静脉青霉素 G(IV Penicillin G) |

> [!info] 补充成员(机制上也 UMN+LMN,但常各自归综合征框架)
> - **Vitamin E 缺乏**:表现酷似 Friedreich(脊髓小脑 + 后索),见于脂肪吸收不良 / 无 β 脂蛋白血症。
> - **Syringomyelia(脊髓空洞症)**:空洞平面**前角 LMN**(手萎缩)+ 平面以下**外侧 CST UMN**(下肢),经典"披肩样"分离性感觉缺失。详见 [[完整笔记/专题笔记/神经/神经_脊髓病变]] D.5。
> - **Conus medullaris(脊髓圆锥综合征)**:原文即"UMN + LMN 混合",早期严重尿便障碍 + 鞍区麻木。详见 [[完整笔记/专题笔记/神经/神经_脊髓病变]] D.6。

## §三 各主角一句话锁定

> [!tip] 快速记忆
> - **ALS** = UMN + LMN + **零感觉**(纯运动排他)。
> - **B12 SCD** = 大细胞贫血 + 后索(振动/本体)↓ + Babinski+;查 **MMA(methylmalonic acid, 甲基丙二酸)↑ + 同型半胱氨酸(homocysteine)↑** 比血 B12 敏感;**别单独补叶酸**(folate trap 加重神经症状)。
> - **铜缺乏** = "B12 正常的 SCD",找 **胃旁路 / 补锌过量** 病史。
> - **Friedreich** = 招牌矛盾 **DTR 消失 + Babinski 阳性**;青少年 + 共济失调 + 心肌病(死因)。
> - **颈椎病性脊髓病** = **上肢 LMN + 下肢 UMN + 感觉平面 + 颈痛**;最重要的**可治**鉴别,MRI 必做。
> - **Taboparesis** = 后索(脊髓痨)+ 痴呆(麻痹性痴呆)双拼 + Argyll-Robertson 瞳孔。

## §三.5 "后索三兄弟"精细鉴别 ⭐ 最易混

> [!warning] B12 SCD / 铜缺乏 / Friedreich —— 三者极像,UWorld 爱互当干扰项
> 三者都同时打 **后索(dorsal column, 后柱：振动/本体觉↓ + Romberg+)** 和 **侧索(lateral CST, 外侧皮质脊髓束：Babinski+/痉挛)**,所以都表现为 **感觉性共济失调 + UMN+LMN 并存**。区分全靠**年龄 + 血液学 + 病史**三刀。

| 维度 | **B12 亚急性联合变性(SCD)** | **铜缺乏性脊髓神经病** | **Friedreich 共济失调** |
|---|---|---|---|
| 本质 | 后天营养/吸收 | 后天营养/吸收 | **遗传(AR 常染色体隐性, GAA 重复, frataxin↓)** |
| 起病年龄 | 成人/老年 | 成人(常有手术史) | **青少年(5-15 岁)** ⭐ |
| 累及束 | 后索+侧索(+脊髓小脑束) | 后索+侧索(几乎复刻 SCD) | 后索+侧索+**脊髓小脑束 spinocerebellar + 背根神经节 DRG** |
| 共济失调类型 | 感觉性(闭眼加重) | 感觉性 | **感觉性 + 真小脑性**(构音障碍 dysarthria、肢体辨距不良)⭐ |
| 血液学 | **巨幼细胞贫血**(纯) | **贫血 + 中性粒细胞减少**(可拟骨髓增生异常 MDS)⭐ | **无**(血象正常) |
| 招牌伴随 | 舌炎、精神/认知改变 | 减重手术/补锌病史 | **肥厚型心肌病(HCM, 死因)+ 脊柱侧弯 + 弓形足 pes cavus + 糖尿病**⭐ |
| 关键化验 | **MMA↑ + 同型半胱氨酸↑**,血 B12↓ | **血清铜↓ + 铜蓝蛋白 ceruloplasmin↓ + 锌↑**;**血 B12 正常** ⭐ | 基因检测(GAA);无营养异常 |
| 治疗 | 肌注维生素 B12(IM B12) | 补铜 + 停锌 | 对症(无根治) |

> [!tip] 三把鉴别钥匙
> 1. **青少年 + 心肌病/脊柱侧弯/弓形足 + 有真正小脑征** → **Friedreich**(DTR 消失 + Babinski+ 那个矛盾最招牌)。
> 2. **巨幼贫 + 素食/恶性贫血/笑气史** → **B12 SCD** → 查 MMA。
> 3. **化验像 SCD 但血 B12 正常 / 补 B12 不好转 / 减重手术或补锌史** → **铜缺乏**(B12 的"隐形双胞胎")。

> [!danger] 两个最阴的坑
> - **铜缺乏 vs B12**:脊髓 MRI + 神经查体分不开,**必须靠"血 B12 正常 + 病史(锌/胃旁路)"**;而且铜缺乏爱有**中性粒细胞减少**,B12 是纯巨幼贫 → 血常规能给线索。
> - **Friedreich 的共济不只是后索**:B12/铜是**纯感觉性**共济(闭眼才不稳),Friedreich 叠加**脊髓小脑束**,睁眼也有小脑性辨距不良/构音障碍——这是把它和另两个分开的解剖点。

> [!info] 一个记忆锚
> 三者共享"**后索 + 侧索**"这套解剖(SCD 的 "Combined" 就是指后索 + 侧索"联合"变性);**铜缺乏 = 照抄 SCD**;**Friedreich = 在此基础上多加一条脊髓小脑束 + 一颗肥厚的心**。

## §四 命题陷阱

> [!warning] USMLE 最爱的分叉点
> - **"有没有感觉受累"是 ALS 的生死线**:stem 只要提振动觉↓ / Romberg+ / 感觉平面 → **不是 ALS**,转去查 B12 / 颈椎 MRI。
> - **B12 缺乏神经症状可早于血象**:CBC(complete blood count, 全血细胞计数)正常 ≠ 排除;不确定时查 **MMA + 同型半胱氨酸**。
> - **铜缺乏是 SCD 的"隐形双胞胎"**:B12 补了不好转 / 有减重手术或补锌史 → 想到铜。
> - **Friedreich 的反射矛盾**(DTR 消失却 Babinski+)是送分识别点,别被"反射消失像 LMN 病"带偏。
> - **颈椎病性脊髓病漏诊代价最大**——它可治,凡"上肢萎缩 + 下肢痉挛 + 颈痛" 先做颈椎 MRI,别急着诊断 ALS。

## §五 复习自测

> [!question]- 60 岁男性,进行性四肢无力,上肢肌萎缩+肌束颤,下肢反射亢进 Babinski+,舌肌纤颤,**感觉完全正常**。最可能诊断?最佳初始诊断检查?最紧急的下一步?
> **ALS**(UMN+LMN 并存 + 无感觉受累)。最佳初始诊断 = **EMG(electromyography, 肌电图)**;最紧急下一步 = **肺功能(PFT, pulmonary function test)**(呼吸肌受累是致命点)。

> [!question]- 16 岁女孩,进行性步态不稳 + 脊柱侧弯 + 弓形足,查体 **DTR 消失但 Babinski 阳性**,超声见室间隔肥厚。诊断?死因?
> **Friedreich 共济失调**(GAA 三核苷酸重复 / frataxin)。最常见死因 = **肥厚型心肌病**导致的心衰/心律失常。

> [!question]- 58 岁女性,减重手术史,双下肢振动觉↓ + Babinski+ + 周围神经病,**血 B12 正常**,补 B12 无改善。下一步查什么?
> 查**血清铜 + 血浆锌**——**铜缺乏性脊髓神经病**(常因补锌过量 / 胃旁路吸收障碍,完全模拟 SCD);治疗 = 补铜 + 停锌。

> [!question]- 哪一条体征出现基本可排除 ALS?
> **感觉缺失**(也包括明显眼外肌麻痹 / 括约肌障碍)。ALS 是纯运动;有感觉受累先想颈椎脊髓病、B12/铜缺乏、Friedreich、神经梅毒等**可治或需鉴别**病因。

# 下肢神经定位速查 (Lower Limb Nerve Localization)

> [!tip] 一句话总览
> 下肢考点的钥匙：**先分「神经干 (peripheral nerve) 卡压」还是「神经根 (nerve root) 病」**——同一块无力/麻，两者定位不同。本表按**神经干**组织（对标上肢正中/尺/桡）；神经根 L4/L5/S1 定位见 [[完整笔记/专题笔记/骨科/骨科_脊柱及腰背痛鉴别]] §2.2。卡压治疗阶梯（保守→注射→松解）见 [[完整笔记/专题笔记/神经/神经_周围神经卡压治疗阶梯]]。

## §一 主干定位表（哪条神经 → 哪块肌无力 + 哪片麻 + 招牌 stem）⭐

| 神经 | 根 | 运动（无力表现）| 感觉（麻区）| 招牌损伤 / stem 钩子 |
|---|---|---|---|---|
| **髂腹下神经 Iliohypogastric** | T12–L1 | 腹横肌 + 腹内斜肌 | 耻骨上区 (suprapubic) | **腹部手术**后切口处烧灼/刺痛，放射至腹股沟/耻骨上 |
| **生殖股神经 Genitofemoral** | L1–L2 | 提睾肌 (cremaster) | 阴囊/大阴唇 + 大腿内侧 | **腹腔镜手术**；大腿前上感觉↓ + **提睾反射消失** |
| **股神经 Femoral** | L2–L4 | 屈髋（髂腰肌 iliopsoas）+ **伸膝（股四头肌 quadriceps + 缝匠肌/耻骨肌）** → 膝打软、上不了台阶 | 大腿前 + **小腿内侧（隐神经 saphenous）** | 盆腔/髋手术、**腹膜后血肿（抗凝）/ 腰大肌脓肿 (psoas abscess)**、**糖尿病性肌萎缩 (diabetic amyotrophy)**；**膝反射↓** + FNST（股神经牵拉试验）+ |
| **闭孔神经 Obturator** | L2–L4 | **大腿内收（内收肌群 adductors）** → 内收弱、步态外偏 | 大腿**内侧**一小片 | 盆腔手术、闭孔疝、难产/分娩 |
| **股外侧皮神经 LFCN** (lateral femoral cutaneous) | L2–L3 | **无运动**（纯感觉）| 大腿**前外侧** | **感觉异常性股痛 meralgia paresthetica**：肥胖 / 紧身裤腰带 / 孕期；减重 + 松束带 |
| **坐骨神经 Sciatic** | L4–S3 | 屈膝（腘绳肌 hamstrings）+ **膝以下全部**（经腓总 + 胫）| 膝以下**除内侧**（内侧=隐神经，属股神经）| **髋后脱位 (posterior hip dislocation)**、臀部注射不当、骨盆骨折 → 全足瘫 + 膝下大片麻 |
| **腓总神经 Common peroneal / fibular** | L4–S2 | **足背屈 + 趾伸 → foot drop 足下垂**；足外翻 | 小腿**外侧** + 足背 | **腓骨头 (fibular head) 表浅受压**：跷二郎腿、久蹲、石膏绑腿、消瘦；**跨阈步态 (steppage gait)** |
| **胫神经 Tibial** | L4–S3 | **足跖屈（腓肠肌/比目鱼）+ 足内翻 + 趾屈** → 踮不起脚 | **足底 (sole)** | 腘窝损伤；**跗管综合征 (tarsal tunnel)** ＠内踝；**踝反射 (Achilles)↓** |
| **臀上神经 Superior gluteal** | L4–S1 | **髋外展（臀中/小肌 + 阔筋膜张肌）** | 无 | **Trendelenburg 征**（站患侧 → **对侧**骨盆下垂）；臀肌注射不当 |
| **臀下神经 Inferior gluteal** | L5–S2 | **髋伸（臀大肌 gluteus maximus）** | 无 | 起立 / 爬楼 / 上坡困难 |
| **阴部神经 Pudendal** | S2–S4 | 尿道外括约肌 + 肛门外括约肌 | 会阴 (perineum) | **分娩牵拉 / 久骑车 / 骑马**；大/小便失禁；**阴部神经阻滞以坐骨棘 (ischial spine) 为标志** |

> [!tip] 臀上神经 Trendelenburg 一句话（FA 版）
> 病变**对侧 (contralateral)** = 骨盆下垂那一侧；**同侧 (ipsilateral)** = 单腿站立的那条腿。记："**站哪侧、哪侧坏；掉的是对侧。**"

*股神经 (femoral nerve)：屈髋 + **伸膝（股四头肌）**；皮支（隐神经 saphenous nerve）管小腿内侧/膝内侧感觉。*

![[L1898.png|320]]
*胫神经 (tibial nerve)：足**跖屈 + 内翻 + 屈趾**；皮支管**足底**（跗管综合征即此分布）。*

*Trendelenburg 征：单腿站立（站患侧）时患侧臀中肌/**臀上神经 (superior gluteal)** 无力 → **对侧**骨盆下垂（图右 abnormal）。*

## §二 Foot drop 足下垂：三个来源怎么分 ⭐⭐（最高频）

足下垂（踝背屈无力）可来自**腓总神经 / L5 神经根 / 坐骨神经**，靠"**足内翻**保不保 + 有没有背痛 + 感觉范围"三点分：

| | **腓总神经麻痹**（腓骨头）| **L5 神经根病** | **坐骨神经**（近端） |
|---|---|---|---|
| 足**背屈**（foot drop）| ❌ 弱 | ❌ 弱 | ❌ 弱 |
| 足**内翻**（tibialis posterior，L5-**胫神经**支配）| ✅ **保留** ⭐ | ❌ 弱 | ❌ 弱 |
| 足外翻 | ❌ 弱（浅支）| ❌ 弱 | ❌ 弱 |
| **背痛 / 放射痛** | 无 | **有**（放射过膝）+ SLR（直腿抬高）+ | 可有臀/腿痛 |
| 感觉 | 小腿外侧 + 足背（单一神经）| L5 皮节（含**踇趾蹼**）| 膝下大片（除内侧）|
| 病因钩子 | 跷腿/久蹲/腓骨颈骨折/石膏 | 椎间盘突出 L4-L5 | 髋后脱位/臀注射/骨盆骨折 |

> [!danger] 一击定位钥匙
> **足内翻 (foot inversion) 保留 = 腓总神经**（内翻由胫神经管，腓总只管背屈+外翻）；
> **足内翻也弱 + 背痛放射 = L5 神经根**；
> **足内翻弱 + 无背痛 + 膝下大片麻 + 屈膝也弱 = 坐骨神经近端**。

> [!success] FA 金牌口诀：PED vs TIP（背哪条神经管什么动作）⭐
> - **PED = Peroneal Everts and Dorsiflexes**（腓总神经 = 外翻 + 背屈）→ 损伤 = **foot dro-PED**（足下垂：静息呈**内翻 + 跖屈**、失外翻/背屈、steppage gait）。
> - **TIP = Tibial Inverts and Plantarflexes**（胫神经 = 内翻 + 跖屈）→ 损伤 = **站不了 TIP-toe（踮脚尖）**。
> 两条互补：腓总管"抬（背屈+外翻）"、胫管"踩（跖屈+内翻）"。这正是上面"足内翻保不保"钥匙的机制来源。

> [!warning] 腓总 vs MS 别混（Q12243 错点）
> 腓总压迫：缺损**数小时**自愈 + 严格落**单一腓总神经分布**；
> **多发性硬化 (MS, multiple sclerosis)**：缺损**数天~数周**才缓、跨解剖走中枢白质。
> 有 MS 家族史 + 反复短暂发作时，靠"**缓解速度 + 是否单一周围神经分布**"分开。源 [[mistakes/uworld-mistakes_2026-07#^Q12243]]。

## §三 腓总神经深支 vs 浅支（分不清就漏定位）

```
腓总神经 (common peroneal) 绕腓骨头后分两支：
├─ 深支 (deep peroneal)
│    运动 = 足背屈 (tibialis anterior) + 趾伸 → foot drop 主责
│    感觉 = 仅第 1–2 趾蹼一小片
└─ 浅支 (superficial peroneal)
     运动 = 足外翻 (peroneus longus/brevis)
     感觉 = 小腿外侧下段 + 足背大部
```

> [!tip] 记忆
> 深支 = "**背屈 + 一小片趾蹼**"（DEEP = Dorsiflexion）；浅支 = "**外翻 + 一大片足背感觉**"。单纯**足背屈无力但足背感觉正常** → 只累及**深支**。

![[L44427_Q12243.jpg]]
*腓总神经 (common peroneal / fibular nerve) 病解剖：绕**腓骨头 (fibular head)** 表浅走行，久蹲/盘腿压迫（右图）→ 踝背屈无力（足下垂）+ 趾伸弱 + 足背/小腿外侧感觉↓。浅支=足外翻+感觉，深支=足/趾背屈。（双嵌自卡 [[mistakes/uworld-mistakes_2026-07#^Q12243]]）*

## §四 下肢卡压综合征（治疗见阶梯专题）

| 综合征 | 神经 / 部位 | 钩子 | 一线处理 |
|---|---|---|---|
| **感觉异常性股痛 meralgia paresthetica** | 股外侧皮神经 ＠腹股沟韧带 | 大腿前外侧**纯感觉**、无运动；肥胖/紧衣/孕 | **减重 + 松束带** ± NSAIDs（几乎不手术）|
| **腓总神经麻痹** | 腓总 ＠腓骨头 | foot drop + 小腿外侧/足背麻；跷腿/久蹲 | 去除压迫 + **AFO 踝足矫形器 (ankle-foot orthosis)** |
| **跗管综合征 tarsal tunnel** | 胫神经 ＠内踝屈肌支持带 | **足底**麻痛、负重加重；扁平足/占位 | 矫形鞋垫/夹板 ± NSAIDs ± 注射 |

> 完整三级阶梯（保守 → 局部激素注射 → 手术松解）+ 上肢卡压对照见 [[完整笔记/专题笔记/神经/神经_周围神经卡压治疗阶梯]]。

## §五 自测复习题

> [!question]- Q1. 跑者久蹲后右足下垂 + 小腿外侧/足背麻，但**足内翻正常、无背痛**。定位？为什么不是 L5 根？
> **腓总神经 ＠腓骨头**。足内翻由**胫神经 (L5 纤维)** 支配、此处保留 → 排除 L5 神经根病（L5 根病内翻也弱 + 有背痛放射 + SLR+）。处理 = 去压迫 + AFO。

> [!question]- Q2. 髋关节后脱位复位后，患肢**全足瘫 + 膝以下大片麻（内侧除外）+ 屈膝无力**。哪条神经？
> **坐骨神经 (sciatic)**。膝下除内侧（内侧=隐神经属股神经）全麻 + 屈膝（腘绳肌）弱 = 坐骨主干；髋后脱位是经典机制。

> [!question]- Q3. 抗凝患者突发**伸膝无力 + 膝反射消失 + 大腿前/小腿内侧麻**。定位 + 想到什么病因？
> **股神经 (femoral)**。伸膝（股四头肌）+ 膝反射 + 隐神经感觉区 = 股神经；抗凝下想**腹膜后血肿 (retroperitoneal hematoma)** 压迫。

> [!question]- Q4. 站立单腿（患侧）时**对侧骨盆下垂**。哪条神经 / 哪块肌？
> **臀上神经 (superior gluteal)** → 臀中/小肌髋外展无力 → **Trendelenburg 征**。注意是**站患侧、对侧**骨盆掉。

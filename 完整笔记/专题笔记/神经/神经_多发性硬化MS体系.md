# 神经_多发性硬化 MS 体系（Multiple Sclerosis）

> [!info] 一句话定位
> **MS（multiple sclerosis，多发性硬化）= 中枢神经系统免疫介导脱髓鞘病**，特征 = 病灶**时空多发（disseminated in time & space）**。年轻女性高发。
> 派生自做对题 [[mistakes/uworld-mistakes_2026-06#^Q2285]]（INO 定位）+ [[mistakes/uworld-mistakes_2026-06#^Q15402]]（危险因素 vit D）。

## 一、危险因素

| 维度 | 要点 |
|---|---|
| 人口 | **女性**（<50 岁高发）|
| 遗传 | **HLA-DRB1** |
| 环境（可改）| **低维生素 D（vitamin D deficiency）**、吸烟 |
| 地理 | 高纬度、寒冷、日照少（美/欧/澳）|
| 感染 | **EBV（Epstein-Barr virus，EB 病毒）** |

> [!tip] 可改 vs 不可改
> 题目问"可干预降低风险"→ 选**低维生素 D**（补充）/ 戒烟；EBV、HLA、地理是触发/背景但难改。

## 二、常见临床表现

| 表现 | 要点 |
|---|---|
| 感觉/运动障碍 | 麻木、无力、膀胱/肠功能障碍 |
| **视神经炎 optic neuritis** | 单眼痛性视力下降、RAPD（相对性传入瞳孔障碍）|
| **INO（internuclear ophthalmoplegia，核间性眼肌麻痹）** | 见下方专节 |
| **Lhermitte 征** | 屈颈时沿脊背电击样感 |
| **Uhthoff 现象** | 体温升高（运动/热水浴）→ 症状一过性加重 |

## 三、INO 定位规则（Q2285）⭐

> [!danger] INO = MLF（medial longitudinal fasciculus，内侧纵束）受损
> **规则**：**不能内收（adduct）的那只眼 → 同侧 MLF 病灶**；对侧眼外展时伴**水平眼震**（代偿）。convergence（辐辏）与瞳孔光反射通常保留。
> - 例：左视时**右眼不能内收** → 病灶 = **right MLF**。
> - **双侧 INO** 经典见于 MS（MLF 近中线，斑块易双侧累及）。
> - 鉴别：CN VI（abducens，展神经）病变 = 外展麻痹（非内收障碍）；小脑病变 = **双眼**眼震 + 共济失调。

```
左视(向左看)
  正常: 左眼外展 + 右眼内收(协同)
  右 MLF 坏: 左眼外展+眼震, 右眼"过不了中线"内收不能 = 右侧 INO
```

## 四、诊断

- 临床**时空多发** + **MRI** 脑室周围 T2/FLAIR 高信号脱髓鞘斑块
- **CSF（cerebrospinal fluid，脑脊液）寡克隆带（oligoclonal IgG bands）**
- 急性发作：大剂量糖皮质激素 glucocorticoid；缓解期：疾病修饰治疗 DMT

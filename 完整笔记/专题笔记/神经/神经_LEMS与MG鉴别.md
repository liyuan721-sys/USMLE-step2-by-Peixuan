# LEMS vs MG — 神经肌接头病鉴别（突触前 vs 突触后）

> [!info] 起源
> 派生自 NBME11 Q136（肺癌 + 近端无力 + 口干便秘 + 无反射 → LEMS，确诊重复神经刺激）。本笔记把**神经肌接头 (neuromuscular junction, NMJ) 病**的两大主角 LEMS 与 MG 系统对比，作为复习单一真相源。

---

## §一 一句话锁定

> [!tip]
> - **LEMS (Lambert-Eaton myasthenic syndrome, 兰伯特-伊顿肌无力综合征)** = **突触前 (presynaptic)** 病 → 钙通道抗体 → ACh 释放↓ → **近端无力 + 自主神经症状 + 反射↓**，活动后**短暂改善**；常伴**小细胞肺癌 (small cell lung cancer, SCLC)**。
> - **MG (myasthenia gravis, 重症肌无力)** = **突触后 (postsynaptic)** 病 → 乙酰胆碱受体 (acetylcholine receptor, AChR) 抗体 → **眼肌/球部为主，疲劳后加重**，反射正常；常伴**胸腺瘤 (thymoma)**。

---

## §二 机制对比（突触前 vs 突触后）

```text
突触前神经末梢
   │  电压门控钙通道 (voltage-gated calcium channel, VGCC)
   │     └─ LEMS 抗体攻击此处 → Ca²⁺ 内流↓ → ACh 释放↓
   ▼
突触间隙（ACh）
   ▼
突触后膜
      乙酰胆碱受体 (AChR)
         └─ MG 抗体攻击此处 → 可结合受体↓
```

- **LEMS**：反复刺激 / 运动 → 突触前 Ca²⁺ 逐渐蓄积 → ACh 释放回升 → 力量**短暂改善**（facilitation，增幅现象）。
- **MG**：反复活动 → ACh 耗竭 → 力量**进行性下降**（疲劳，decrement）。

---

## §三 核心鉴别表 ⭐

| 维度 | **LEMS** | **Myasthenia gravis (MG)** |
|---|---|---|
| 病变部位 | 突触前 (VGCC 抗体) | 突触后 (AChR / MuSK 抗体) |
| 起病肌群 | **近端腿无力为主**（起身/爬楼难）| **眼肌/球部为主**（上睑下垂、复视、吞咽难）|
| 活动后 | **短暂改善**（增幅）| **疲劳加重**（傍晚重）|
| 深腱反射 | **减弱 / 消失** ⭐ | **正常** |
| 自主神经症状 | **有**（口干、便秘、阳痿、体位性低血压）⭐ | 无 |
| 伴随肿瘤 | **小细胞肺癌 (SCLC)**（副肿瘤）| **胸腺瘤 / 胸腺增生** |
| 确诊电生理 | 重复神经刺激 **高频递增 (incremental)** ⭐ | 重复神经刺激 **递减 (decremental)** + 单纤维肌电图 |
| 抗体 | 抗 VGCC（P/Q 型）| 抗 AChR（或 MuSK）|
| 治疗 | 治潜在 SCLC + **3,4-二氨基吡啶 (3,4-diaminopyridine, 3,4-DAP)**；IVIG | 胆碱酯酶抑制剂 **吡啶斯的明 (pyridostigmine)** + 免疫抑制 / 胸腺切除 |

> [!danger] 三大高价值分界词（题干一出现立刻偏向 LEMS）
> ① **深腱反射消失**（MG 反射正常）；② **自主神经症状（口干 / 便秘）**（MG 无）；③ **活动后改善**（MG 疲劳加重）。再叠加**肺癌 / 重度吸烟**背景 → 几乎钉死 LEMS。

---

## §四 诊断路径

```text
近端无力 + 怀疑 NMJ 病
   │
   ├─ 反射消失 + 口干便秘 + 肺癌/吸烟 + 活动后略好
   │      → LEMS → 重复神经刺激（高频递增）+ 抗 VGCC → 查/治 SCLC（胸部 CT）
   │
   └─ 眼肌/球部 + 疲劳加重 + 反射正常
          → MG → 重复神经刺激（递减）/ 单纤维肌电图 + 抗 AChR → 查胸腺（胸部 CT/MRI）
```

> [!warning] 别被"近端无力"带去炎性肌病
> 近端无力的另一大类是**多发性肌炎 (polymyositis) / 皮肌炎 (dermatomyositis)** → 但它们**无自主神经症状、无反射消失**，且 **CK (肌酸激酶) 显著升高**，确诊靠**肌活检 (muscle biopsy)**。
> NBME11 Q136 错点正是：看到"近端无力"反射选**股四头肌活检**，忽略了口干 + 便秘 + 无反射这组突触前 NMJ 病指纹 → 应选**重复神经刺激**。

---

## §五 横向鉴别（近端无力四鉴别）

| 疾病 | 反射 | 自主神经 | 活动后 | 关键检查 |
|---|---|---|---|---|
| **LEMS** | ↓/消失 | 有 | 改善 | 重复刺激递增 / 抗 VGCC |
| **MG** | 正常 | 无 | 加重 | 重复刺激递减 / 抗 AChR |
| **多发性肌炎** | 正常 | 无 | 无变化 | CK↑↑ / 肌活检 |
| **脊髓压迫** | 可亢进 + 病理征 | 可有膀胱直肠 | 无变化 | 脊髓 MRI |

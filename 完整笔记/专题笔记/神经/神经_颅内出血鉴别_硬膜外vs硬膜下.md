# 神经_颅内出血鉴别（硬膜外 vs 硬膜下 vs SAH vs 脑实质）

> [!info] 一句话定位
> 创伤性颅内出血靠 **CT 形态 + 是否跨颅缝 + 受累血管 + 病史** 区分。
> **硬膜外 epidural = 双凸不跨缝 + 脑膜中动脉 + 清醒间期 + 急诊手术**；**硬膜下 subdural = 新月跨缝 + 桥静脉 + 老人/酗酒/抗凝**。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q4923]]（进展性硬膜外血肿误选仅观察）。

## 一、四型对照表

| 类型 | CT 形态 | 跨颅缝? | 受累血管 | 典型病史 | 处理 |
|---|---|---|---|---|---|
| **硬膜外 EDH（epidural hematoma）⭐** | **双凸 / 透镜形 biconvex** | **否（受缝限制）** | **脑膜中动脉 middle meningeal artery**（蝶骨骨折）| 头部外伤 → **清醒间期 lucid interval** → 迅速恶化 | **有症状 → 急诊神经外科清除 evacuation** |
| 硬膜下 SDH（subdural hematoma）| **新月形 crescent** | **是（可跨缝）** | **桥静脉 bridging veins**（静脉）| 老人/酗酒/抗凝、轻微外伤、亚急性/慢性 | 视体积+症状：手术或观察 |
| 蛛网膜下腔出血 SAH（subarachnoid hemorrhage）| 脑沟/脑池高密度 | — | 动脉瘤破裂 / 外伤 | **雷击样"一生最剧"头痛** | CT → 阴性则腰穿；动脉瘤需介入/夹闭 |
| 脑实质出血 ICH（intracerebral hemorrhage）| 脑实质内高密度 | — | 穿支小动脉 | 高血压（基底节/丘脑）、淀粉样血管病（脑叶）| 控压 + 支持，部分手术 |

## 二、EDH vs SDH 记忆锚

```
双凸(lens) + 不跨缝 + 动脉(脑膜中) + 清醒间期 = 硬膜外 EDH → 快, 手术
新月(crescent) + 跨缝 + 静脉(桥静脉) + 老人/酗酒/抗凝 = 硬膜下 SDH → 慢
```

> [!tip] 为什么形态不同
> 硬膜在颅缝处与颅骨紧贴 → **硬膜外**血被缝"卡住"→ 双凸不跨缝；**硬膜下**血在硬膜与蛛网膜间，可沿表面铺开 → 新月跨缝。

## 三、处理陷阱（Q4923）

> [!danger] 有症状硬膜外血肿 = 手术，不是观察
> - **仅观察/随访 CT**：只适合**小而无神经功能障碍**的 EDH；出现 lethargy/呕吐/进行性恶化 → **急诊清除**，延误 → 脑疝死亡。
> - **甘露醇 mannitol / 高渗盐水**：可暂时降颅内压，但**清除血肿才是根治**；单纯渗透治疗 + 观察会增加死亡。
> - **PCC（prothrombin complex concentrate，凝血酶原复合物）**：仅用于**抗凝相关**颅内出血逆转；未抗凝患者不适用。
> - **硫胺 thiamine + 葡萄糖**：治 Wernicke 脑病，与双凸血肿无关。

## 四、Stem 钩子

- "斗殴/撞击 → 短暂清醒 → 又恶化 + CT 双凸" → **EDH** → 急诊手术。
- "老人跌倒数周后渐进认知↓ + 新月形" → **慢性 SDH**。
- "雷击样最剧头痛" → **SAH**。

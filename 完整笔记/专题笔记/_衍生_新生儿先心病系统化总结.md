# _衍生_新生儿先心病系统化总结（CHD by Timing × Cyanosis × Murmur）

> [!info] 本笔记定位
> **针对"症状出现时间 + 是否紫绀 + 杂音特点"三轴**的新生儿先心病速查框架。USMLE Step 2 CK 儿科 / 心内交叉高频考点，stem 往往只给 3 个线索（生后第几天 / SpO₂ / 杂音）就要你定病。
>
> **核心思维**：
> 1. **三轴定位**：时间（DA 关闭前后 / PVR 下降前后）+ 紫绀（SpO₂ 阈值）+ 杂音（时相 → 性质 → 位置 → 动作）
> 2. **PGE1 决策点**：导管依赖型 = 维持 PDA 才能活
> 3. **杂音 6 大模式**先按时相，再按性质
> 4. **动作试验**鉴别 HCM vs AS（Valsalva 反向）
>
> **来源**：错题 Q20421（IDM-HCM）+ 整合 2026-05-10 心血管讲课（妊娠 + 先心病）

## Part 0. 三轴诊断框架 ⭐⭐⭐

> [!success] 看到新生儿心脏题 → 先问这 3 个问题
> 1. **症状什么时候出现？**（生后 0-24h / 24-48h / 数周-月 / 数月-年）
> 2. **SpO₂ 多少？**（>92% 不发绀 / <92% 发绀 / 上下肢差）
> 3. **杂音什么模式？**（时相 → 性质 → 位置 → 动作反应）
>
> → 3 个答案交叉锁定病变

```
                    ┌─────────────────────────────────┐
                    │  新生儿心脏题 stem               │
                    └────────────┬────────────────────┘
                                 ▼
                    ┌─ 时间轴 ───┴─── 紫绀轴 ─── 杂音轴 ─┐
                    │                                      │
        ─────────────────────────────────────────────────
        生后立即(0-24h)    24-48h DA 关闭     数周-月 PVR 降     数月-年
        ─────────────────────────────────────────────────
        d-TGA (PGE1!)     CoA critical       VSD 大→CHF        ASD 漏诊
        HLHS (PGE1!)      AS critical        IDM-HCM 渐重→稳   AS 渐进
        Pulm atresia      PDA-dep 全部       Eisenmenger?       CoA 年长儿
        Tricuspid atresia                                       PDA 残存

        ─── 紫绀分诊（SpO₂）───
        >92% Acyanotic:  3D + Coarct + IDM-HCM + AS/PS
        <92% Cyanotic:   5T（TOF/TGA/TAPVR/Truncus/Tricuspid atresia）+ HLHS + Ebstein

        ─── 杂音定位（时相→性质→位置→动作）───
        Systolic ejection   AS / PS / HCM / IDM-HCM     （动作: Valsalva↑=HCM；↓=AS）
        Holosystolic        VSD / MR / TR                （Handgrip↑=都↑）
        Continuous          PDA                          （单一持续机器声）
        Diastolic           AR / PR / MS                 （罕见新生儿）
        Fixed split S2      ASD（永不变）⭐              （独门特征）
        Single S2           TOF / d-TGA / HLHS / Truncus（缺一个半月瓣）
```

## Part 1. 第一轴：症状出现时间 ⭐⭐⭐

> [!info] 时间决定病变类型
> 新生儿循环过渡：**胎儿循环（PDA + ASD + 高 PVR）→ 出生 → 肺呼吸 → PVR↓ + DA 关闭 + ASD 闭合**
>
> 不同 CHD 在过渡的**不同节点**暴露 → 时间是关键线索

### 1.1 生后立即（0-24h）— "PDA 不能救"

| 病变 | 关键 | 处理 |
|---|---|---|
| **d-TGA** | 生后立即紫绀（两套循环平行）+ "egg on string" | **PGE1**（维持 PDA 混合）+ 紧急房隔造口 + arterial switch |
| **HLHS** | 全身紫绀 + 弱搏动 + 休克样 | **PGE1**（PDA 灌全身）+ 分期 Norwood |
| **Pulmonary atresia** | 生后即紫绀（无 RV 出路）| **PGE1**（PDA 灌肺）|
| **Tricuspid atresia** | 紫绀 + **LAD in newborn**（罕见！）⭐ | **PGE1** + Glenn → Fontan |
| **Severe Ebstein** | 紫绀 + 心脏极大 + LAD | 支持 / 手术 |

**共同点**：**双循环依赖 PDA 混合或灌注** → 出生即崩 → **PGE1 第一时间**

### 1.2 24-48h（DA 关闭后）— "PDA 关了才崩"

| 病变 | 关键 | 处理 |
|---|---|---|
| **Critical Coarctation** | 前 1-2 天正常 → **休克 + 上下肢 BP 差 + 股动脉弱** + lactic acidosis | **PGE1 立即 + 紧急手术** ⭐ |
| **Critical AS** | DA 关闭 → LV 突然超负荷 → 休克 | PGE1 + 急诊球囊扩张 |
| **Critical PS** | DA 关闭 → RV 出不去 → 紫绀 | PGE1 + 急诊球囊扩张 |
| **Severe TOF / Tet spell** | 哭闹 / 吃奶诱发紫绀 + 蹲位缓解 | Knee-to-chest + O2 + morphine + IV fluid + phenylephrine（急性）→ 手术 |

**共同点**：**导管依赖型** — DA 关闭暴露病变 → 突发休克 / 紫绀

### 1.3 数周-数月（PVR 下降后）— "肺血流变化暴露"

| 病变 | 关键 | 处理 |
|---|---|---|
| **大 VSD** | 6-8 周后 PVR↓ → 左→右 shunt↑↑ → **CHF**（呼吸急促 / 喂养不耐受 / 出汗 / 体重不增）| 利尿 + ACEi → 手术修补 |
| **大 PDA** | 同上机制 | Indomethacin（早产）/ 经皮封堵（足月）|
| **IDM-HCM** | 生后渐重 → 数周-月达峰 → **1 岁前自愈** | **支持 + β-blocker**；禁 inotrope / 利尿剂 ⭐ |
| **AV canal defect (Down)** | 大 VSD + 大 ASD → 早期严重 CHF | 早期手术 |

**共同点**：**PVR 自然下降**（出生后 6-8 周完成）→ L→R 分流量加大 → CHF 显现

### 1.3.1 VSD 自然病程 4 阶段 ⭐⭐⭐（来源 Q16280）

> [!info] 一题打通：为什么"先天性"病偏偏 1 个月才发作？
> 所有**左→右分流 CHD** 的发病时机都**跟 PVR 下降曲线走**，不是"出生立刻发病"。

```
阶段 1: 胎儿期（in utero）
─────────────────────────
   PVR ≈ SVR（双侧压力相等）
   → 几乎无分流
   → 无症状 ✓

阶段 2: 出生 0-2 周（postnatal transition）
─────────────────────────────
   肺开始呼吸 → PVR 开始↓
   脐带剪断 → SVR↑
   PVR 还没降到底
   → 分流较小 → 症状轻
   → 杂音可能已出现（小 VSD 大声响）

阶段 3: 1-2 月龄 ⭐（Q16280 的患儿）
─────────────────────────────
   PVR 降至接近成人水平
   SVR/PVR 差最大 → 左→右分流 ↑↑↑ 达峰值
   ↓
   肺血流↑↑ → 肺淤血
   ↓
   LA + LV 容量超载（要泵 = 体循环 + 分流量）
   ↓
   高输出量心衰 ⭐（LV 收缩力正常但跟不上回流）
   → poor feeding + diaphoresis with feeds + 肝大

阶段 4: 多年后（未治疗 → Eisenmenger）
─────────────────────────
   持续肺血流↑ → 肺血管重塑（平滑肌增生 + 内膜增厚）
   → PVR ↑↑↑ 缓慢上升
   ↓
   PVR > SVR → 分流反转为右→左
   ↓
   Eisenmenger 综合征 → 紫绀 + 杵状指 + 不可逆
   → 杂音消失（双侧压力又相等）
```

> [!success] 一句话概括 4 阶段
> **"PVR 决定 VSD 的脸色 — 高时安静，低时心衰，再高就发紫"**

### 1.3.2 VSD 心衰本质 = high-output failure ⭐（反直觉考点）

> [!warning] VSD 心衰不是"泵衰竭"
> - **LV 收缩力正常**（甚至 hyperdynamic）
> - 真正问题：**容量超载** — LV 要泵的总量 = 体循环 + 分流量
> - 与 HFrEF（收缩力↓）**截然不同**
>
> **USMLE 答题陷阱**：题目问"根本血流动力学异常" → 答 **pulmonary overcirculation / 左→右分流**，**不是** decreased contractility

| 维度 | VSD high-output HF | 经典 HFrEF |
|---|---|---|
| LV 收缩力 | **正常 / hyperdynamic** | ↓ |
| EF | 正常 | ↓ |
| 病因 | 容量超载 | 泵衰竭 |
| 治疗 | **利尿 + ACEi → 修补缺损** | β-blocker + ACEi + 利尿 + ICD |
| Inotrope | **不需要** / 反而有害（增分流） | 急性失代偿可短期用 |

### 1.3.3 VSD afterload 反直觉 ⭐

> [!danger] VSD ↓ LV afterload，不是 ↑
> 反直觉点：VSD 是 LV 的**"减压阀"** — LV 血液经 VSD 走低阻肺循环（PVR ≪ SVR）→ **整体 afterload ↓ 或正常**
>
> 这就是为什么大 VSD 患儿 **EF 可以正常甚至 hyperdynamic** — afterload 没增加，反而被分流"卸压"了。

### 1.3.4 VSD 心衰症状出现的解剖时序 ⭐

```
1️⃣ 肺淤血（最先）
   → 呼吸急促 + retractions + 双肺啰音

2️⃣ 左心容量超载 + 交感激活
   → 喂养困难（婴儿版"运动不耐受"）
   → 出汗 + 苍白（交感激活 + 灌注不足）
   → 心动过速

3️⃣ 体循环淤血（婴儿敏感的肝脏先反映）
   → 肝大（轻度，3 cm 在婴儿属于早期信号）
   → 体重不增

4️⃣ 真正的双室泵衰竭（晚期，未治疗）
   → 严重肝大 + 腹水 + 颈静脉怒张
```

### 1.4 数月-数年（成长后）— "慢性逐渐显现"

### 1.4 数月-数年（成长后）— "慢性逐渐显现"

| 病变 | 关键 | 处理 |
|---|---|---|
| **ASD（小-中）** | 儿童多无症状 → 成人 dyspnea / AF / paradoxical embolism | 中-大 → 封堵 |
| **Mild Coarctation** | **上肢 HTN + 下肢 claudication + brachial-femoral delay + rib notching** | 手术 / 球囊 |
| **二瓣化 AS（成人）** | 进行性 ejection murmur → 运动晕厥 / 心绞痛 / HF | 手术（valve replacement）|
| **PDA 残存** | 持续 machinery + 慢性 LV 容量负荷 | 封堵 |
| **Eisenmenger** | 未治疗 ASD/VSD/PDA → 长期 PAH → **反转 R→L → cyanosis + clubbing + hemoptysis** | 不可逆，仅支持 |

### 1.5 婴儿心衰 vs 成人心衰对照 ⭐⭐⭐（来源 Q16280）

> [!info] 婴儿心衰看吃奶 — 不要套用成人症状
> 婴儿不会说"运动气短"，**吃奶 = 婴儿最大体力活动**（能量消耗 ≈ 成人慢走）。所以婴儿心衰的"运动不耐受"表现为 **poor feeding + diaphoresis with feeds**。

| 成人心衰 | 婴儿心衰 | 解剖 / 生理基础 |
|---|---|---|
| 爬楼气短 / 运动不耐受 | **Poor feeding**（吃奶费力 / 量↓ / 时间长）| 吃奶能量消耗 ≈ 成人慢走 |
| 端坐呼吸 | **Diaphoresis with feeds**（喂奶时出汗）| 交感激活 |
| 夜间阵发性呼吸困难 | 呼吸急促 + retractions | 同肺淤血机制 |
| 下肢水肿 | **肝大 + 体重不增** | 婴儿无重力依赖性水肿 |
| 颈静脉怒张 | 难评估 → **肝大替代** | 婴儿颈短 |
| BNP↑ | BNP↑ | 同 |
| "正常肝缘不可触及" | **正常肝缘可达肋下 1-2 cm** ⭐ | 婴儿肝解剖位置低 + 软 |

### 1.5.1 婴儿肝大的真实意义 ⭐（反陷阱）

> [!warning] 婴儿肝大 ≠ 严格的右心衰
> - **正常婴儿**肝缘可触及肋下 1-2 cm
> - **> 2 cm** → 早期体循环淤血（≠ 严重右心衰）
> - **> 4-5 cm + 腹水 + JVD** → 真正的双室泵衰
>
> **UWorld 语言陷阱**：UW 写 "signs of right-sided heart failure (eg, hepatomegaly)" 在临床偏松 — 更准确说"**体循环淤血征象**"。1 月龄 VSD 患儿肝大 3 cm = **轻度早期信号**，**不是**严重右心泵衰竭。

### 1.5.2 反射弧训练

> [!success] 高频识别口诀
> - **"1-2 月龄 + 喂养困难 + 出汗 + 不发绀 + LLSB holosystolic + thrill"** → **大 VSD + 高输出量心衰**
> - **"婴儿肝缘 ≤ 2 cm 正常；> 2 cm 早期淤血；> 4-5 cm 才考虑严重右心衰"**
> - **"任何左→右分流 CHD 发病时机 → 跟 PVR 下降曲线走"**（不是出生立刻发病）

## Part 2. 第二轴：紫绀 — Acyanotic vs Cyanotic ⭐⭐⭐

### 2.1 SpO₂ 第一分诊

> [!success] 第一分诊：SpO₂ 阈值 92%
> - **>92%**（不发绀）→ **L→R 分流** 或 **流出道梗阻**（左心病变 / 容量超负荷）
> - **<92%**（发绀）→ **R→L 分流** 或 **混合**（导管依赖多）
> - **上下肢差**（pre-ductal > post-ductal > 5%）→ **Coarctation**
> - **反向上下肢差**（pre < post）→ **d-TGA + Coarct/PAH**（罕见但经典）

### 2.2 Acyanotic CHD — "3D + Coarct + IDM-HCM + AS/PS"

```
左→右分流（容量超负荷）          左心阻塞（压力超负荷）
─────────────────────             ──────────────────────
- VSD（最常见）                   - AS / 二瓣化
- ASD                             - Coarctation
- PDA                             - HLHS（严重；归紫绀类，混合）
- AV canal（Down）                - HCM / IDM-HCM
                                  - PS
```

**记忆**：**"3D"** = VS**D** + AS**D** + P**D**A（都不发绀）

### 2.3 Cyanotic CHD — "5T + HLHS + Ebstein"

```
5T（混合 / R→L 分流）             其他紫绀
────────────────────              ──────────
1. Tetralogy of Fallot (TOF)      - HLHS（左心发育不良）
2. Transposition (d-TGA)          - Ebstein anomaly（三尖瓣下移；
3. Truncus arteriosus               母亲用 Lithium 经典）
4. Tricuspid atresia              - Critical pulmonary atresia
5. TAPVR
```

**记忆**：**"5 个 T"**

### 2.4 Hyperoxia Test ⭐（诊断辅助）

```
吸 100% O₂ 10 分钟 → 测 PaO₂
        │
        ├─ PaO₂ > 250 mmHg → 肺源性紫绀（TTN / RDS / 胎粪吸入 / 肺炎）
        ├─ PaO₂ 100-250 → 部分肺病 + 部分混合
        └─ PaO₂ < 100 → ★ Cyanotic CHD（混合 / R→L 分流）
                        → 立即 PGE1 + Echo + 转 NICU
```

### 2.5 紫绀机制分类

| 机制 | 病变 | 关键特征 |
|---|---|---|
| **R→L 分流**（旁路肺）| TOF / Pulm atresia / Tricuspid atresia / Ebstein | 静脉血直接进体循环 |
| **完全混合**（共腔）| Truncus / TAPVR / HLHS | 体 + 肺血在同一腔 |
| **平行循环**（双独立环）| **d-TGA** | 必须有 PDA + ASD 才能活 |
| **L→R 长期 + 反转**（Eisenmenger）| 未治疗 VSD/ASD/PDA | 晚期才发绀 |

## Part 3. 第三轴：杂音 ⭐⭐⭐（你说"都记不太清"，这是核心）

### 3.1 杂音 4 步分析法

```
Step 1: 时相（systolic / diastolic / continuous）
Step 2: 性质（ejection / holosystolic / rumble / decrescendo / machinery / split）
Step 3: 位置（RUSB / LUSB / LLSB / apex / 后背 / 左锁骨下）
Step 4: 动作反应（Valsalva / 下蹲 / 握拳 / 站立 / 吸气）
```

### 3.2 杂音 6 大模式 → 病因映射 ⭐

| 杂音模式 | 时相 / 形态 | 经典位置 | 病因 | USMLE buzzword |
|---|---|---|---|---|
| **Systolic ejection murmur** | 收缩中期 crescendo-decrescendo（"<>"形）| RUSB（AS）/ LUSB（PS）/ 左胸（HCM）| **流出道梗阻**：AS / PS / **HCM / IDM-HCM** | "harsh ejection" + ejection click |
| **Holosystolic** | 收缩全程 plateau（"━"形）| LLSB（VSD / TR）/ apex（MR）| **分流或反流**：VSD / MR / TR | "blowing" / 缺损越小越响（VSD）|
| **Continuous "machinery"** | 整个心动周期 | **左锁骨下** | **PDA** | 机器声 |
| **Diastolic decrescendo** | 舒张早期 | 左 LSB | **AR / PR** | 坐起前倾、呼气末更响（AR）|
| **Diastolic rumble** | 舒张中晚期 | apex | **MS**（罕见新生儿）/ severe ASD（tricuspid flow）| 开瓣音 + S1 响 |
| **Split S2 异常** | S2 | — | **Fixed split = ASD** ⭐ / Wide = RBBB-PS-PE / Paradoxical = LBBB-severe AS-HCM / Single = TOF/TGA/HLHS/Truncus | 分裂模式 |

> [!success] 杂音口诀
> **"喷射看梗阻（AS/PS/HCM），全收缩看分流（VSD/MR/TR），机器声看 PDA，分裂固定看 ASD"**

### 3.3 动作试验 — 鉴别 HCM vs AS 金标准 ⭐⭐⭐

> [!info] 核心逻辑
> - **HCM = 动态梗阻**（LV 越小 → 室间隔越靠近，梗阻越重）→ **减少前负荷 = 杂音↑**
> - **AS = 固定梗阻**（瓣口面积不变）→ **增加前负荷 = 流量↑ = 杂音↑**

| 动作 | 前负荷变化 | HCM | AS | MVP | 反流（MR/AR）|
|---|---|---|---|---|---|
| **Valsalva**（屏气用力）| ↓ 静脉回流 | **↑↑** ⭐ | ↓ | **↑↑** + 喀喇音提前 | ↓ |
| **站立**（突然）| ↓ 静脉回流 | ↑ | ↓ | ↑ | ↓ |
| **下蹲 / 抬腿** | ↑ 静脉回流 | ↓ | ↑ | ↓ | ↑ |
| **握拳**（↑ 后负荷）| ↑ SVR | ↓ | ↓ | ↑ | **↑↑**（MR / AR）|
| **吸气**（↑ 右心回流）| 选择性↑ 右心 | — | — | — | **TR ↑**（Carvallo 征）|

> [!success] 反向口诀
> **"HCM 怕空腔，AS 喜重负"** → Valsalva 让空腔更空 → HCM 杂音爆响；增负使 AS 流量↑ → AS 杂音响

### 3.4 新生儿 vs 儿童良性杂音（不需处理）⭐

| 杂音 | 特征 | 处理 |
|---|---|---|
| **Still's murmur**（最常见儿童良性杂音）| 低频 musical / vibratory，LLSB，**站立时↓** | Reassurance |
| **PPS**（外周肺动脉狭窄）| 软的 systolic 在上 LSB → 后背放射，新生儿 | 生后 6 月内自愈 |
| **Venous hum** | Continuous 锁骨上 / 下，**仰卧时消失，转头消失** | Reassurance |
| **Pulmonary flow murmur**（青少年）| 软的 systolic LUSB | Reassurance |

**通用判别**：**音柔 + 收缩期 + 局限 + 无 thrill + 无症状 + 体位变化** → 良性
**反之**（diastolic / continuous / holosystolic / thrill / 发绀 / 喂养不耐）→ 病理 → **Echo**

## Part 4. 各 CHD 详细速查表（11+ 病变）⭐

> [!info] 一表收尽 — stem 给 3 个线索就能定病

### 4.1 Acyanotic（SpO₂ 正常 >92%）

| 病变 | 时间出现 | 杂音 / 听诊 | 关键 stem 触发词 | ECG / Imaging | 治疗 |
|---|---|---|---|---|---|
| **VSD**（最常见 CHD 25%）⭐ | 出生杂音 → **1-2 月**（PVR 谷底）CHF | **Holosystolic LLSB** + thrill（≥4/6 必有 thrill）；**小 VSD 杂音反而响**（湍流大）；**大 VSD 杂音轻**（压差小）；**Eisenmenger 后杂音消失**（双侧压力相等）| **"1-2 月 + holosystolic LLSB + thrill + 喂养困难 + 出汗 + 宽脉压 + 肝大"** | LVH ± RVH（大 VSD 后期）| 小 → 自闭；大 + CHF → 利尿 + ACEi → 手术修补；**禁 inotrope**（VSD 是高输出量 HF，LV 收缩力正常）|
| **ASD** | 童年无症状 | **Fixed split S2** ⭐ + LUSB 收缩中期 + 严重时 LLSB rumble | "fixed split S2" | RBBB / Right axis | 中-大 → 封堵 |
| **PDA** | 24-48h 出现 → 持续 | **Continuous machinery** 左锁骨下 | "machinery murmur" + 早产儿 / maternal rubella | Echo | 早产 → **Indomethacin**；足月 → 封堵 |
| **CoA** | Critical 24-48h 休克；Mild 年长 HTN | 后背 / 左肩胛间 systolic | **上下肢 BP 差 + brachial-femoral delay + Turner / bicuspid AV** + rib notching | "3 sign" CXR + LVH | Critical → **PGE1 + 紧急手术**；Mild → 球囊 / 手术 |
| **IAA**（主动脉弓中断；type B 最常见，位于左颈总动脉与左锁骨下之间）⭐ | **24-48h DA 关闭**突发休克（同 critical CoA 机制）| 多无显著杂音；常合并 VSD → LLSB holosystolic + single S2 | **休克 + 上下肢 BP 差 + femoral pulse 弱/无 + 低钙抽搐 + 胸腺缺如 / 胎面 → DiGeorge 22q11 ~80%** ⭐ + **几乎 100% 合并 VSD** | Echo / CTA 见弓中断；ECG 早期正常 → LVH + RVH | **PGE1 立即**（导管依赖体循环）+ 一期手术修补弓 + 关 VSD + 评估 calcium / 22q11 FISH |
| **二瓣化 AS** | 童年起渐重 | **Systolic ejection RUSB → 颈部** + ejection click | "ejection click" + 运动晕厥 | LVH | β-blocker → 手术换瓣 |
| **PS** | 同上 | Systolic ejection LUSB | RVH / wide split S2 | RVH | 球囊 |
| **IDM-HCM** ⭐ | 生后即有 → 数月达峰 → 1 岁前自愈 | **Systolic ejection** + IDM 背景（巨大儿 / 锁骨折）| **室间隔肥厚 + 小 LV + 母亲糖尿病 + macrosomia** | LVH | **支持 + β-blocker**；禁 inotrope / 利尿剂 |
| **AV canal**（Down）| 数周 → 严重 CHF | 多种杂音（VSD + ASD + AVVR）| **Down syndrome + 早期 CHF** | LAD + 1° AVB | 早期手术 |
| **Supravalvular AS** ⭐（罕见但 William 招牌）| 童年渐进 | Systolic ejection **主动脉根部上方** + **右臂血压 > 左臂**（Coanda 效应）| **William 综合征**（弹性蛋白 ELN 基因缺失 / 鸡尾酒派对人格 / 低 + 高钙）| LVH + 升主动脉狭窄 | 手术 patch |

> - **VSD** — Ventricular Septal Defect（室间隔缺损）
> - **ASD** — Atrial Septal Defect（房间隔缺损）
> - **PDA** — Patent Ductus Arteriosus（动脉导管未闭）
> - **CoA** — Coarctation of the Aorta（主动脉缩窄）
> - **IAA** — Interrupted Aortic Arch（主动脉弓中断）
> - **AS** — Aortic Stenosis（主动脉瓣狭窄）；**二瓣化 AS** = Bicuspid Aortic Valve Stenosis
> - **PS** — Pulmonary Stenosis（肺动脉瓣狭窄）
> - **IDM-HCM** — Infant of Diabetic Mother — Hypertrophic Cardiomyopathy（糖尿病母亲婴儿肥厚型心肌病）
> - **AV canal** — Atrioventricular Canal Defect / Atrioventricular Septal Defect / AVSD（房室间隔缺损 / 房室管畸形）
> - **Supravalvular AS** — Supravalvular Aortic Stenosis（瓣上型主动脉狭窄；William 综合征）
> - **ALCAPA** — Anomalous Left Coronary Artery from the Pulmonary Artery（左冠状动脉异常起源于肺动脉）
> - **PFO** — Patent Foramen Ovale（卵圆孔未闭）

### 4.2 Cyanotic（SpO₂ 低 <92%）

| 病变 | 时间出现 | 杂音 / 听诊 | 关键 stem 触发词 | ECG / Imaging | 治疗 |
|---|---|---|---|---|---|
| **TOF** ⭐（最常见紫绀 CHD）| 出生紫绀 → tet spell 后渐重；**严重度按 PS 程度分级**（PS 越重紫绀越早 / 重 → 新生儿期发病；PS 轻则"pink TOF" 童年才发现）| **Systolic ejection LUSB**（PS 杂音）+ **single S2**；杂音越**轻**反而提示 PS 越**重**（流量更少）⭐ | **"Boot-shaped heart" + tet spell + 蹲位缓解 + DiGeorge** | RVH + RAD | 急性 spell：膝胸位 + O₂ + morphine + IVF + phenylephrine；根治：手术 |
| **d-TGA** | **生后立即紫绀** ⭐ | **常无杂音** + single S2 | **"Egg on a string" CXR + 母亲糖尿病** | RVH | **PGE1 + atrial septostomy + arterial switch** |
| **HLHS** | 24-48h 休克 + 全紫绀 | **Single S2** + 弱搏动 + 杂音不显 | 全身灌注差 + 单一 S2 + 弱 femoral pulse | 各种 | **PGE1 + Norwood / Glenn / Fontan** |
| **TAPVR** | 紫绀 + 重度肺水肿 | **Fixed split S2** + LUSB 收缩 | **"Snowman" / "Figure 8" CXR** | RVH | 手术急 |
| **Truncus** | 紫绀 + 早期 CHF | **Single S2** + systolic ejection | **Single S2 + DiGeorge (22q11)** | — | 手术 |
| **Tricuspid atresia** | 紫绀 + LAD | 单一 S1（无三尖瓣关闭音）| **LAD in newborn 罕见！** + 紫绀 | **LAD** ⭐ + RAE | PGE1 → Glenn → Fontan |
| **Ebstein anomaly** | 紫绀 + 极大心 | 多重三尖瓣 click | **母亲用 Lithium** + Wolff-Parkinson-White 关联 | RAE + RBBB + WPW | 支持 / 手术 |
| **Eisenmenger syndrome**（未治疗 L→R shunt 反转 R→L）⭐ | 数年-数十年潜伏 → 多 2-3 岁后渐显 / 30 岁达峰 | **杂音消失** ⭐（双侧压力相等）+ **响亮 P2** + RV heave + S4 | 未治疗 **VSD / ASD / PDA / AVSD**（VSD 风险最高）+ 紫绀 + **杵状指 + 红细胞增多 + 咯血 + 矛盾栓塞**；若由 PDA 反转 → **differential cyanosis（下蓝上粉）** ⭐ | RVH + RAE + 肺动脉扩张；心导管 PVR↑↑ + shunt 反向 | **不可逆**：支持 + O₂ + sildenafil / bosentan + 抗凝 + 静脉切血（红血症）→ 终末**肺-心移植**；❌ **禁孕**（30-50% 母死亡率）❌ **禁关原发 shunt**（已成体循环"出口"，关闭→急性右心衰）|

> [!info] Part 4.2 缩写对照
> - **TOF** — Tetralogy of Fallot（法洛四联症）
> - **d-TGA** — Dextro-Transposition of the Great Arteries（完全性大动脉转位 / 右型转位）
> - **HLHS** — Hypoplastic Left Heart Syndrome（左心发育不良综合征）
> - **TAPVR** — Total Anomalous Pulmonary Venous Return（完全性肺静脉异位引流）
> - **Truncus** — Truncus Arteriosus（永存动脉干 / 共同动脉干）
> - **Tricuspid atresia** — Tricuspid Atresia（三尖瓣闭锁）
> - **Ebstein anomaly** — Ebstein's Anomaly（三尖瓣下移畸形 / 埃布斯坦畸形）
> - **Eisenmenger syndrome** — Eisenmenger Syndrome（艾森曼格综合征）
>
> **常见心电/影像缩写**：**RVH** = Right Ventricular Hypertrophy（右室肥厚）/ **LVH** = Left Ventricular Hypertrophy（左室肥厚）/ **RAE** = Right Atrial Enlargement（右房扩大）/ **RAD** = Right Axis Deviation（右轴偏移）/ **LAD** = Left Axis Deviation（左轴偏移）/ **RBBB** = Right Bundle Branch Block（右束支阻滞）/ **WPW** = Wolff-Parkinson-White syndrome（预激综合征）

#### **反向差异性：上肢蓝，下肢粉** ⭐⭐ **USMLE 100% 特异**

- **病因**：**TGA + PDA** ⭐
- **机制**（必懂）：

```
  TGA：主动脉接 RV，肺动脉接 LV（两个平行循环）
        ↓
  右心去氧血 → 主动脉 → 升主动脉 → 上肢（蓝）⭐
        ↓
  左心含氧血 → 肺动脉 → 通过 PDA → 降主动脉 → 下肢（粉）⭐
        ↓
  "Reversed differential cyanosis" = 100% TGA
```

> [!tip] 差异性紫绀一句话 **"下蓝上粉 = PDA + PH"**（普通差异） **"上蓝下粉 = TGA + PDA"**（反向差异，100% 特异）⭐

## Part 5. PGE1 决策树 ⭐⭐⭐

### 5.1 PGE1 适应症 = 维持 PDA

```
新生儿紫绀 / 休克 + 怀疑 CHD：
       │
       ▼
SpO₂ < 85% 或休克 + Hyperoxia test (PaO₂ < 100 after 100% O₂)?
       │
       ├─ 是 → 怀疑 Duct-dependent CHD → **PGE1 立即开始**
       │            │
       │            ├─ 维持 PDA → 改善体循环或肺循环灌注
       │            └─ 副作用：**Apnea**（25% 发生率，需备插管）⭐
       │
       └─ 否 → 寻肺源 / 代谢源
```

### 5.2 用 PGE1 的"双类" 7 病变

| 类型 | 病变 | PGE1 作用 |
|---|---|---|
| **导管依赖肺血流**（紫绀型）| TOF severe / Pulm atresia / Tricuspid atresia / 严重 PS / Ebstein 严重 | 维持 PDA → 让血从主动脉经 DA 到肺 |
| **导管依赖体循环**（休克型）| **HLHS / Critical CoA / Critical AS / Interrupted aortic arch** | 维持 PDA → 让肺动脉血逆向到主动脉 |
| **平行循环混合**（紫绀型）| **d-TGA** | 维持 PDA + atrial septostomy 维持混合 |

### 5.3 不用 PGE1（避雷）

- **IDM-HCM** ⭐（不发绀，且 PGE1 无意义）
- **VSD / ASD / 简单 PDA**（自体血流足够）
- **TTN / RDS / 胎粪吸入**（肺源性紫绀，Hyperoxia test PaO₂ > 250）
- **Hyperoxia test PaO₂ > 250** = 肺源 → 不是 CHD → 不用 PGE1

### 5.5 反向机制 — 孕晚期 NSAIDs → DA 早期关闭 → PPHN ⭐

> [!warning] PGE1 vs NSAIDs 反向机制（USMLE 高频对照）
> | 药 | 对 PG 作用 | 对 DA 作用 | 临床场景 |
> |---|---|---|---|
> | **PGE1**（外源）| 给 PG | **维持 DA 开放** | 导管依赖型 CHD 新生儿急救 |
> | **NSAIDs / Indomethacin**（COX 抑制）| 抑制 PG 合成 | **促 DA 关闭** | ① 早产儿 PDA 关闭治疗；② **孕晚期 (>30 周) 误用 → 胎儿 DA 早期关闭 → 出生 PPHN** ⚠️ |
>
> **USMLE 题眼**：孕晚期母亲服用 NSAIDs / Indomethacin / Ibuprofen → 新生儿 **PPHN（持续性肺动脉高压）+ 三尖瓣反流 + 严重发绀** → 治疗 **iNO**（选择性肺血管扩张）+ 支持
>
> **记忆口诀**：**"PG 开 DA，NSAIDs 关 DA"** + **"孕晚期止痛只用 Tylenol（acetaminophen），不用 NSAIDs"**

### 5.4 PGE1 给药 + 副作用

| 项 | 内容 |
|---|---|
| 给药 | **0.05-0.1 μg/kg/min IV 持续输注** |
| 起效 | < 30 分钟 |
| **副作用最重要** | **Apnea**（呼吸暂停，25% 发生率）⭐ → **预备插管** |
| 其他副作用 | 低血压 / 发热 / 抽搐 / 周围血管扩张潮红 |

## Part 7. Memory Hook 集合 ⭐

### 7.1 三轴诊断口诀

> **"三轴定位：时间分诊、SpO₂ 分流、杂音分病"**

### 7.2 紫绀 5T

> **TOF / TGA / TAPVR / Truncus / Tricuspid atresia** = 5 个 T

### 7.3 Acyanotic 3D + 其他

> **VS-D + AS-D + P-D-A** = 3D（不发绀的三大左→右分流）

### 7.4 杂音 6 大模式

> **"喷射看梗阻（AS/PS/HCM），全收缩看分流（VSD/MR/TR），机器声看 PDA，分裂固定看 ASD，舒张减压差是 AR/PR，舒张隆隆是 MS"**

### 7.5 Valsalva 反向口诀

> **"HCM 怕空腔，AS 喜重负"**
> - HCM = 动态梗阻 → Valsalva 减容量 → 杂音↑↑
> - AS = 固定梗阻 → Valsalva 减流量 → 杂音↓

### 7.6 PGE1 适应症

> **"导管依赖才用 PGE1：紫绀（5T + Ebstein + Pulm atresia）+ 体循环导管依赖（HLHS / Critical CoA / Critical AS）"**
>
> **"PGE1 一开始就备插管"**（apnea 25%）

### 7.7 综合征-CHD 配对口诀

> - **Down → AV canal / 原发孔 ASD / VSD**
> - **Turner → 二瓣化 AV + CoA**
> - **DiGeorge → TOF + Truncus**
> - **William → 瓣上 AS**
> - **Marfan → AR + 主动脉夹层**
> - **母 DM → TGA + IDM-HCM**
> - **母 Lithium → Ebstein**
> - **母 Rubella → PDA + 白内障 + 聋**

### 7.8 IDM 三联征（Q20421 触发）

> **"IDM 三联：巨大儿 + 锁骨骨折 + 室间隔肥厚 → 全部 transient → 1 岁前自愈"**

### 7.9 类比理解

| 病变 | 类比 |
|---|---|
| **TGA** | 两台并联发动机但管子接错 — 必须开个洞（PDA/ASD）让油混一下才能跑 |
| **HLHS** | 左心是个废件 — 必须用右心 + PDA 给全身供血 |
| **CoA critical** | 主动脉中间夹了个钳子 — DA 关了下半身就断流 |
| **TOF** | 通往肺的门窄 + 主动脉骑墙 — 一哭就把蓝血推到主动脉（tet spell）|
| **VSD 大** | 左右心室之间有大洞 — 6 周后肺压降下来才"漏"出 CHF |
| **IDM-HCM** | 婴儿"被胰岛素喂胖"的心脏 — 断奶后自己瘦回来 |
| **Eisenmenger** | 长期单向漏水把右心压垮 — 最后反过来漏 |

## Part 8. 高频陷阱总结 ⭐

| 陷阱 | 正确认知 |
|---|---|
| ❌ 新生儿心脏异常 = 严重 = 必 PGE1 | ✓ 先看 SpO₂；>92% 多不需要 PGE1（IDM-HCM 是经典反例）|
| ❌ Systolic ejection = AS | ✓ 也可以是 PS / HCM / IDM-HCM；用动作试验鉴别 |
| ❌ HCM 用利尿剂减心负荷 | ✓ **HCM 禁利尿剂**（减容量加重动态梗阻）；β-blocker 是正路 |
| ❌ 母体糖尿病 → 一定是结构畸形 | ✓ **妊娠 DM = 一过性**（HCM/低血糖/RDS）；**孕前 DM = 真畸形**（caudal regression / TGA）|
| ❌ Fixed split S2 看右束支阻滞 | ✓ Fixed = **ASD**；Wide = RBBB/PS/PE |
| ❌ Continuous murmur 都是 PDA | ✓ 也可能 Venous hum（仰卧消失）/ AVM；位置 + 体位反应 |
| ❌ 紫绀 + boot heart = HLHS | ✓ Boot-shaped = **TOF**；HLHS 心脏多大 + 弱搏动 |
| ❌ TGA 必有响杂音 | ✓ **TGA 常无杂音**（两套循环独立）+ Single S2 + "egg on string" |
| ❌ 看到 Down 想 VSD | ✓ Down 经典是 **AV canal / Ostium primum ASD**（不只是 VSD）|
| ❌ Newborn LAD = lead misplacement | ✓ 真的 newborn LAD 罕见 → **Tricuspid atresia** ⭐ |
| ❌ IDM-HCM 需要手术 | ✓ **1 岁前自愈**；支持 + β-blocker 即可 |
| ❌ 上下肢 BP 差 = HTN | ✓ 经典 = **Coarctation**（critical 婴儿休克 / mild 年长 HTN）|
| ❌ Rib notching 任何年龄 | ✓ 需要侧支建立时间（年长儿 / 成人）|
| ❌ Tet spell 用 β-blocker 急救 | ✓ **首先 Knee-to-chest + O2**；药用 morphine + IVF + phenylephrine（↑SVR）|
| ❌ Eisenmenger 可以逆转 | ✓ **不可逆 PAH**；移植或仅支持 |
| ❌ 先天性 CHD = 出生立刻发病 | ✓ **左→右分流 CHD（VSD/ASD/PDA）都延迟发病**，依赖 PVR 下降曲线（1-2 月最重）；只有发绀型 / 导管依赖型才出生立即发病 |
| ❌ VSD 心衰 = LV 收缩力差 | ✓ **VSD = high-output failure（容量超载）**，LV 收缩力**正常 / hyperdynamic**；EF 正常；选项 "↓ contractility" 是干扰项 |
| ❌ VSD 增加 LV afterload | ✓ **VSD 反而 ↓ LV afterload**（"减压阀"经低阻肺循环）→ 这是反直觉但 USMLE 考点 |
| ❌ 婴儿肝大 = 严重右心衰 | ✓ **正常婴儿肝缘 1-2 cm 可触**；3 cm = 轻度淤血（早期左心容量超载传导）；> 4-5 cm + 腹水 + JVD 才是真双室衰 |
| ❌ 小 VSD 杂音轻 | ✓ **小 VSD 杂音越响**（小孔高湍流 + 大压差）；大 VSD 反而轻；Eisenmenger 后杂音消失 |
| ❌ 婴儿心衰看下肢水肿 | ✓ **婴儿心衰看吃奶**（poor feeding + diaphoresis with feeds = 婴儿版"运动不耐受"）+ 体重不增 + 肝大替代 JVD |

## Part 9. 跨学科陷阱

| 学科 | 陷阱 | 关键点 |
|---|---|---|
| **OB** | 妊娠 DM 控制不佳 → 胎儿心脏 | IDM-HCM + TGA + VSD 风险↑ |
| **OB** | 妊娠 Lithium | Ebstein anomaly |
| **OB** | 妊娠 Rubella 早期 | PDA + 白内障 + 聋 |
| **OB** | 妊娠 alcohol | FAS + VSD/ASD |
| **内分泌** | 母体 DM 影响胎儿 | 早期（孕前）→ 结构畸形；晚期（妊娠 DM）→ 一过性 |
| **遗传** | Down syndrome | AV canal / 原发孔 ASD |
| **遗传** | Turner syndrome | BAV + CoA 必查 |
| **遗传** | DiGeorge | TOF + Truncus + 低钙 |
| **风湿** | 风湿热 | 风湿性 MS（多年后才出现 diastolic rumble + opening snap）|
| **心内** | HCM Valsalva 增强 | 反直觉（其他杂音减弱）|
| **儿科** | 新生儿良性杂音 | Still's / PPS / Venous hum — reassurance |
| **心内 / IE** | **IE + 新发传导阻滞 + AR 杂音 → TEE 排除瓣周脓肿** ⭐ | "IE + 心跳慢 / 传导阻滞 = 脓肿" — TTE 漏诊率高，必 TEE；瓣周脓肿 = 紧急手术指征 |
| **心内 / Post-op CHD** | **修复后长期随访** ⭐ | ① **AS 术后**：升主动脉根部进行性扩张（即使换瓣后）→ 定期 Echo；② **PS 术后 / TOF 术后**：肺动脉瓣**反流**进行性 → RV 容量负荷 → 远期 RV 衰竭 / 室速 / SCD；③ **CoA 术后**：再缩窄 / HTN 不消失 / berry 动脉瘤未消除 |

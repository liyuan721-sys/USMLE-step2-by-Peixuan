# 硫酸镁临床应用与镁中毒 (Magnesium Sulfate: Clinical Use & Toxicity)

> [!tip] 一句话核心
> 硫酸镁 (magnesium sulfate, MgSO₄) 是**窄治疗窗**药物：治疗窗 4–7 mEq/L 紧贴中毒阈值。中毒是**剂量依赖**的可预期并发症 → **边用边监测（膝反射 + 呼吸 + 尿量）**；一旦中毒 → **停镁 + 静推葡萄糖酸钙 (calcium gluconate)**。

---

## 一、为什么产科要用镁（理解用途才理解中毒）

| 适应症 | 目标血镁 | 备注 |
|---|---|---|
| **子痫前期 / 子痫抽搐预防与治疗 (seizure prophylaxis/treatment)** | **4–7 mEq/L** | 抽搐一线，优于苯妥英 (phenytoin) / 地西泮 (diazepam) |
| **早产 <32 周胎儿神经保护 (fetal neuroprotection)** | 同上 | 降脑瘫 (cerebral palsy) 风险，兼弱 tocolysis（保胎） |

> [!info] 窄治疗窗
> 治疗窗 (4–7 mEq/L) 与中毒阈值挨得很近 → 中毒不是过敏，而是**可预期的剂量依赖并发症** → "边用边监测"是硬要求。

> [!danger] 禁忌症 (Contraindications) ⭐ 高频
> - **重症肌无力 (myasthenia gravis, MG) = 绝对禁用** ⭐：MG 本身 NMJ 处 ACh 传递已受损，镁再抑制突触前 ACh 释放 → 诱发 / 加重**肌无力危象 (myasthenic crisis)**（呼吸肌无力）。
>   - **替代抗抽搐**：左乙拉西坦 (levetiracetam) 或苯二氮䓬类 (benzodiazepine)；⚠️ 丙戊酸 (valproic acid) 孕期致畸，**不作替代**。
> - **相对禁忌 / 慎用**：肾功能不全 / 少尿（镁经肾排 → 蓄积中毒，见 §四）、心脏传导阻滞、近期心肌梗死。
> - **与硝苯地平 (nifedipine, CCB) 合用**须谨慎（叠加低血压 + 神经肌肉抑制）。

---

## 二、机制 —— 为什么表现是"从下往上被抑制"

> [!info] Mg²⁺ = 生理性钙拮抗剂 (physiologic calcium antagonist)
> - **神经肌肉接头 (neuromuscular junction, NMJ)** 突触前抑制 Ca²⁺ 内流 → **减少乙酰胆碱 (acetylcholine, ACh) 释放** → 神经肌肉传导阻滞 → 中毒表现是**肌肉/反射被抑制**（非兴奋）。
> - 同时：CNS 抑制（嗜睡）、血管扩张（潮红 / 低血压）、抑制心脏传导（PR/QRS 延长 → 缓脉 → 停搏）。
> - **抗抽搐机制**：脑血管扩张 + **NMDA 受体拮抗** + 膜稳定 → 提高抽搐阈值。

> [!success] 解药逻辑（一句话记死）
> 镁靠拮抗钙起效 → **补钙 (calcium gluconate) 在 NMJ 竞争性把镁顶回来** → 立即逆转。

---

## 三、剂量-症状阶梯 ⭐（考"顺序"，不是精确数值）

> [!danger] 血镁进展阶梯
> 
> | 血镁 (mEq/L) | 表现 |
> |---|---|
> | **4–7（治疗窗）** | 目标，无毒性 |
> | **~7–10（轻→中）** | **腱反射减弱 → 消失 (hyporeflexia → areflexia)** ⭐ 最早、最关键；+ 潮红、发热感、头痛、恶心、复视、言语含糊、嗜睡、低钙 |
> | **~10–13** | **呼吸抑制 → 呼吸麻痹 (respiratory depression → paralysis)** |
> | **>15（越高越危）** | **心脏传导阻滞、缓脉 (bradycardia)、心脏骤停 (cardiac arrest)** |
> 
> **ECG**：PR / QRS / QT 延长。

> [!warning] 为什么盯"膝反射"
> **深腱反射 (deep tendon reflex, DTR / 膝反射 patellar reflex) 消失出现在呼吸抑制之前** → "**反射还在 = 呼吸抑制基本不会发生**"。这是床旁监测优先于抽血的原因。

---

## 四、监测三件套 + 最高频危险因素

> [!danger] 床旁监测三件套（不是等抽血）
> 1. **DTR（首先消失，最敏感早期预警）**
> 2. **呼吸频率 (RR)**（<12/min 警惕）
> 3. **尿量 (urine output)**（<25–30 mL/h = 减量/停药信号）

> [!warning] 少尿 / 肾功能 = 中毒放大器 ⭐
> 镁 **100% 靠肾排泄**。子痫前期常伴**少尿 (oliguria) / 肾功能受损** → 镁蓄积 → **输注速度不变也会中毒**。
> Stem 常埋钩子：**"用镁背景 + 尿量骤减 + 随后反射消失"** = 蓄积性中毒。

---

## 五、处理（按顺序，别只记"给钙"）

```
怀疑镁中毒（反射消失 / RR↓ / 尿量↓）
  1. 立即停 MgSO₄ 输注 ⭐（第一步，最易漏）
  2. IV 葡萄糖酸钙 calcium gluconate 1 g（10% 10 mL）缓推 3–5 min ← 解药
  3. 支持：保气道 / 给氧，必要时辅助通气 / 插管
  4. 促排：补液 + 袢利尿剂（呋塞米 furosemide）加速肾排镁
  5. 肾衰 / 极重度 → 血液透析 (hemodialysis)
```

## 六、易混陷阱 & Stem 模式

> [!warning] 高频干扰 / 陷阱
> - **停药是第一步**：选项里"停镁"和"给钙"都对，但**先停输注**是最先动作。
> - **别当子痫恶化**：反射↓ + 嗜睡 + RR↓ 是**过量** → 停 + 拮抗，不能加镁。
> - **别选纳洛酮 (naloxone)**：阿片中毒 (opioid toxicity) **保留深腱反射**；镁中毒**反射消失** —— 这是区分点。
> - **反射方向对立**：镁中毒 = **肌无力 + 反射消失**；恶性高热 (malignant hyperthermia, MH) / 抗精神病药恶性综合征 (neuroleptic malignant syndrome, NMS) / 5-羟色胺综合征 (serotonin syndrome, SS) = **肌僵硬 + 反射亢进** → "高热肌强直急症"排镁中毒的第一刀。
> - **与硝苯地平 (nifedipine, CCB) 合用**：叠加低血压 + 神经肌肉抑制 → 中毒风险↑。
> - **新生儿受累**：临近分娩母体镁过量 → 新生儿**呼吸抑制、肌张力低下 (hypotonia)、嗜睡**（"floppy baby"）→ 新生儿也可能需钙 / 支持通气。
> - **单位坑**：美国实验室多报 **mg/dL**（正常 ~1.5–2.5 mEq/L ≈ 1.8–3.0 mg/dL）；题目给 mg/dL 时别按 mEq/L 判读。

> [!question]- 重度子痫前期产后泵镁防抽搐，现 RR 10、嗜睡、深腱反射消失、少尿 (50 mL/3h)。下一步？（干扰项：加镁 / 纳洛酮 / 拉贝洛尔 / 生理盐水冲）
> **停 MgSO₄ + 静推葡萄糖酸钙 (calcium gluconate)**。
> - 反射消失 + 呼吸抑制 + 用镁背景 = 镁中毒（少尿是放大器，肾功能受损而非低容量 → 冲盐水错）。
> - 反射消失 → 非阿片（阿片保留反射）→ 纳洛酮错；血压未达急性治疗标准 → 拉贝洛尔错。

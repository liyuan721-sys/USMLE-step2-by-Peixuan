# 盆腔器官脱垂 (Pelvic Organ Prolapse, POP) 定位与症状

> [!tip] 一句话主线
> 盆腔器官脱垂 (pelvic organ prolapse, POP, 盆腔器官脱垂) 题的解题关键是**用体检定位"哪一壁"**：**前壁 (anterior)→膀胱，后壁 (posterior)→直肠，宫颈 / 子宫下移→子宫脱垂**。风险背景恒为 **多产 (multiparity) + 绝经 (menopause) → 盆底支持减弱**。

---

## 一、定位决策树

```text
主诉：阴道 bulge（膨出）/ 坠胀 / 漏尿 / 排便困难
   │
   ├─ 阴道前壁 (anterior wall) 膨出 + 压力性尿失禁
   │        └──> 膀胱膨出 Cystocele（膀胱下垂）
   │
   ├─ 阴道后壁 (posterior wall) 膨出 + 便秘 / 排便困难
   │        └──> 直肠膨出 Rectocele（直肠下垂）
   │
   ├─ 阴道顶部 (superior / apical wall) 膨出 + 盆腔坠胀
   │        └──> 肠膨出 Enterocele（小肠下垂）
   │
   └─ 宫颈 / 子宫下移过 hymenal ring（处女膜环）
            └──> 子宫脱垂 Uterine prolapse
```

> [!info] 定位三句诀
> - **前壁看膀胱，后壁看直肠，看到宫颈下移再想子宫。**
> - 体检明确 `anterior vaginal segment descends` → cystocele。
> - Stem 特意写 `normal cervix / rectal exam normal` = 帮你排掉子宫脱垂与直肠膨出。

> [!info]- 附：子宫位置 (uterine version / flexion) 基础解剖（low-yield，从速查总览下沉）
> ![[{D10CDFEC-FA09-44FF-94CC-FDBAFCB1B405}.png]]
> *子宫位置 Uterine positioning：version = 宫颈相对阴道的角度（前倾 anteverted / 后倾 retroverted）；flexion = 宫体相对宫颈的角度（前屈 anteflexed / 后屈 retroflexed）*

---

## 二、核心鉴别表

表格如下：

| 疾病 | 解剖部位 | 高频表现 |
|---|---|---|
| 膀胱膨出 Cystocele | 阴道前壁 (anterior vaginal wall) | 压力性尿失禁、排尿不尽、前壁膨出 |
| 直肠膨出 Rectocele | 阴道后壁 (posterior vaginal wall) | 便秘、排便困难、后壁膨出（须用力协助排便） |
| 肠膨出 Enterocele | 阴道顶部 (superior / apical wall) | 小肠膨出，多为盆腔坠胀感 |
| 子宫脱垂 Uterine prolapse | 宫颈 / 子宫下移 | 宫颈或子宫明显下移过处女膜环 |
| 子宫内翻 Uterine inversion | 产后急症 | 产后剧痛、出血、休克（非慢性脱垂） |

---

## 三、膀胱膨出 Cystocele（本专题主角）

> [!success] Core Concept
> 膀胱膨出 (cystocele, 膀胱膨出) = **膀胱向阴道前壁膨出**，常见于多产、绝经后盆底支持减弱的女性。经典三联：**阴道前壁膨出 + 压力性尿失禁 (stress urinary incontinence, SUI) + 咳嗽 / 用力时加重**。

### 机制链

```text
multiparity + menopause
        ↓
pelvic floor / fascial support weakens
        ↓
bladder descends against anterior vaginal wall
        ↓
urethral support decreases
        ↓
cough or strain raises intra-abdominal pressure
        ↓
stress urinary incontinence + vaginal bulge
```

### 典型画像

> [!tip] Stem 招牌组合
> - 高龄多产妇（如 77 岁, para 4）
> - **剧烈咳嗽 / 用力后漏尿**（腹压上升 → 压力性尿失禁）
> - 主诉"有东西从阴道膨出 (something bulging)"
> - 体检：**阴道前壁下移过处女膜环**；**宫颈正常、子宫位置正常**（→ 排子宫脱垂）；**直肠指检正常**（→ 排直肠膨出）

### 处理要点

- **保守**：盆底肌训练 (Kegel / 凯格尔运动)、子宫托 (pessary, 阴道托)、局部雌激素。
- **手术**：前盆腔修补 (anterior colporrhaphy)；合并压力性尿失禁可加抗尿失禁手术。

---

## 四、鉴别击破（易混陷阱）

> [!warning] 膀胱膨出 vs 子宫脱垂——最强干扰
> "阴道 bulge"太泛，两者都可有坠胀 / 漏尿。真正定性线索：
> - **膀胱膨出 = 阴道前壁下移，宫颈 / 子宫位置正常**。
> - **子宫脱垂 = 宫颈 / 子宫本身明显脱出**。
> - Stem 写 `normal cervix` → 排子宫脱垂。

> [!info] 其他干扰项
> - **直肠膨出 Rectocele**：后壁膨出 + 便秘 / 排便困难，不以尿失禁为核心。
> - **肠膨出 Enterocele**：顶部膨出、盆腔坠胀，不定位在前壁。
> - **子宫内翻 Uterine inversion**：产后急性并发症（剧痛、出血、休克），不会在绝经后 20 余年慢性起病。

---

## 五、Memory Hook

> [!tip] 记忆钩
> **前壁膀胱、后壁直肠、宫颈才是子宫。**
> - anterior wall bulge + urine leak = 先想 cystocele
> - posterior wall bulge + constipation = 更像 rectocele
> - Stem 写 cervix 正常 = 主动帮你排 uterine prolapse

## 六、自测

> [!question]- 77 岁 para 4，绝经 25 年，剧烈咳嗽后漏尿并诉阴道有东西膨出；查体阴道前壁下移过处女膜环，宫颈及子宫位置正常，直肠指检正常。最可能诊断？
> **膀胱膨出 Cystocele**。
> 关键：前壁膨出 + 压力性尿失禁 = cystocele；`normal cervix` 排子宫脱垂，`rectal exam normal` 排直肠膨出。

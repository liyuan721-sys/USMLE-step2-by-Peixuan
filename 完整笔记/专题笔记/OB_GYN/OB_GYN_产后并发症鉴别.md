# 产后并发症鉴别（Postpartum Complications）

> [!info] 本篇聚焦
> 重点讲 **产后尿潴留 (Postpartum Urinary Retention)** 及其与其他"滴尿/漏尿/膀胱症状"病因的鉴别；末尾一句话导航其他产后并发症，各自另见专题。

---

## 一、产后尿潴留 (Postpartum Urinary Retention, PUR)

> [!danger] 定义
> 阴道分娩后 **≥6 小时** 无法自主排尿（或**拔除导尿管后 ≥6 小时**仍不能排尿）。是产后常见但易被忽视的并发症。

### 1. 临床表现

- **Overflow incontinence 充溢性尿失禁**：膀胱被动过度充盈 → 压力被动漏尿/滴尿 (dribbling)，而非主动排尿。
- **Small-volume voids 小量排尿** 或完全无法排尿；**incomplete bladder emptying 排空不全**。
- **下腹压迫痛** + 体检可**触及膨胀的膀胱** (palpable distended bladder)。

> [!warning] 核心影像
> "分娩后 6 小时排不出尿 / 只能滴出少量 + 下腹胀痛 + 摸到膨大膀胱" = 尿潴留导致的充溢性尿失禁。**膀胱膨胀是关键阳性体征**（用于和瘘鉴别，见下表）。

### 2. 机制 (Mechanism)

```
产后尿潴留两大机制
│
├─ 神经损伤通路
│   Perineal trauma 会阴创伤 / 第二产程延长 / 会阴裂伤
│        │
│        ▼
│   Pudendal nerve injury 阴部神经损伤
│        │
│        ▼
│   排尿感觉↓ + 尿道外括约肌功能障碍
│
└─ 麻醉 / 膀胱失张力通路
    Regional / neuraxial anesthesia 区域麻醉（如 epidural 硬膜外）
         │
         ▼
    抑制排尿反射 + 逼尿肌 (detrusor) 张力↓
         │
         ▼
    Bladder atony 膀胱失张力 → 膀胱无法有效收缩排空
```

> [!tip] 记忆钩
> 两条路殊途同归 → 膀胱排空不了 → 尿液潴留 → 被动充溢滴尿。
> **神经损伤**多与会阴创伤/产程延长相关；**膀胱失张力**多与区域麻醉相关。

### 3. 危险因素 (Risk Factors)

- Primiparity 初产
- Regional / neuraxial anesthesia 区域（椎管内）麻醉
- Operative vaginal delivery 器械助产（forceps 产钳 / vacuum 胎吸）
- Perineal injury 会阴损伤
- Cesarean delivery 剖宫产

### 4. 管理 (Management)

> [!success] 处理
> - **Intermittent catheterization 间歇导尿** 引流膀胱 + 安慰 (reassurance)。
> - 多为**自限性**，通常 **<1 周** 恢复自主排尿。

## 二、关键鉴别（Differential Diagnosis）

产后"滴尿 / 漏尿 / 膀胱症状"并非都是尿潴留，务必按下表区分：

| 病因 | 典型时机 | 膀胱膨胀? | 特征 | 与 PUR 的区分点 |
| --- | --- | --- | --- | --- |
| **Postpartum urinary retention 产后尿潴留** | 分娩后**即刻 (≥6h)** | **有**（膀胱膨大可触及） | 充溢性漏尿 + 下腹胀痛 + 排空不全 | 即刻发生 + 膀胱膨胀 |
| **Vesicovaginal fistula 膀胱阴道瘘** | 分娩后**数天** | **无**（尿经瘘持续外漏） | 持续性无痛性漏尿 | 时机晚 + 无膀胱膨胀 |
| **Detrusor overactivity 逼尿肌过度活动** | 不定 | 无 | 尿急 + **急迫性尿失禁**（膀胱痉挛） | 是急迫非充溢；无膨胀 |
| **Urethral injury 尿道损伤** | 产后 | 无 | 尿频、排尿痛、尿流慢 | 疼痛性排尿症状为主 |
| **Vaginal atrophy 阴道萎缩** | 需低雌激素状态才出现 | 无 | 干涩、萎缩性改变 | 哺乳期闭经低雌激素才有，**产后即刻不会** |

> [!warning] 高频陷阱：尿潴留 vs 膀胱阴道瘘
> 两者都可表现为"漏尿/滴尿"，但：
> - **尿潴留**：分娩后**即刻 (6h)** + **膀胱膨胀** + 下腹胀痛（尿排不出、被动溢出）。
> - **膀胱阴道瘘**：通常分娩后**数天**才出现，**无膀胱膨胀**（尿液经瘘口持续外漏，膀胱反而不充盈）。

## 三、其他产后并发症（导航）

各自另见对应专题，此处仅一句话定位：

- **Postpartum hemorrhage 产后出血 (PPH)**：产后失血过多（阴道分娩 ≥500 mL / 剖宫产 ≥1000 mL），子宫收缩乏力最常见。
- **Endometritis 子宫内膜炎**：产后**发热 + 子宫压痛**（另见 [[完整笔记/专题笔记/OB_GYN/OB_GYN_产后发热鉴别]]）。
- **Retained products of conception 胎盘残留**：可致晚发产后出血 + 子宫复旧不良。

## 四、自测题

> [!question]- Q1. 一名初产妇经硬膜外麻醉行产钳助产，分娩后约 7 小时诉下腹胀痛、仅能滴出少量尿液，查体下腹正中可触及膨大包块。最可能的诊断？
> **Postpartum urinary retention 产后尿潴留**（伴充溢性尿失禁）。
> 关键点：分娩后**即刻 (≥6h)** + **可触及膨胀的膀胱** + 危险因素齐（初产、区域麻醉、器械助产）。机制为区域麻醉致逼尿肌张力↓（bladder atony）± 会阴/器械创伤致阴部神经损伤。处理 = 间歇导尿 + 安慰，多 <1 周自限。

> [!question]- Q2. 一名产妇阴道分娩 5 天后出现持续性、无痛性漏尿，任何体位均有尿液流出，查体膀胱区**无膨胀**、无下腹压痛。最可能的诊断？如何与产后尿潴留区分？
> **Vesicovaginal fistula 膀胱阴道瘘**。
> 区分要点：瘘为**分娩后数天**出现的持续外漏，**膀胱不膨胀**（尿经瘘口漏走）；而产后尿潴留为**分娩后即刻 (≥6h)** 发生、伴**膀胱膨胀**与下腹胀痛。膨胀膀胱的有无是床旁快速鉴别的钥匙。

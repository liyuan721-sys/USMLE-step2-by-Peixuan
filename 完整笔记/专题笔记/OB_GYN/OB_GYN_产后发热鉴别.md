# 产后发热鉴别 (Postpartum Fever Workup)

> 派生自 [[NBME11_错题本#^Q126]]（剖宫产后 36h 发热先想子宫内膜炎，Fighter#17）。
> 关联主笔记 [[完整笔记/Peixuan分科笔记/OB]] / [[完整笔记/专题笔记/OB_GYN/OB_GYN_产科新生儿感染抗生素]] / [[完整笔记/专题笔记/OB_GYN/OB_GYN_HighYield速查总览]]

> [!info] 产褥期发热 (puerperal fever) 定义
> **Puerperal fever（产褥热）= 产后第 1 天后的任意 2 天（在产后 10 天内）口温 ≥ 38.0°C（100.4°F）。**
> 产后**头 24h 内**的一过性低热常为生理性（脱水 / 产程消耗 / 母乳上奶），**通常不算病理**；关键看「**术后第几天 + 哪个系统**」分流。
> USMLE 经典框架 = **经典 7 个 W（the 7 W's）**，按术后时间轴串起所有发热源。

---

## 一、经典 7 个 W 框架（按术后时间轴）

> [!tip] 7 W 助记 — 时间轴从早到晚
> 术后/产后发热按"哪天起热"先分流，再按系统定位。下面是**典型出现时间窗**（非绝对）。

```
术后/产后天数轴 ───────────────────────────────────────────►
  POD 0       POD 1-2        POD 2-3        POD 3-5      POD 4-7      POD 5-10+      任意时间
  ┌──────┐  ┌──────────┐  ┌───────────┐ ┌─────────┐ ┌────────┐ ┌──────────┐ ┌──────────┐
  │ Wind │  │ Wind     │  │ Womb      │ │ Water   │ │ Walk   │ │ Wound    │ │ Wonder   │
  │ 麻醉 │  │ 肺不张/  │  │ 子宫内膜炎│ │ 尿路感染│ │ DVT    │ │ 切口感染 │ │ drugs    │
  │ 即刻 │  │ 早期肺炎 │  │ (最高频!) │ │ (UTI)   │ │        │ │          │ │ 药物热   │
  └──────┘  └──────────┘  └───────────┘ └─────────┘ └────────┘ └──────────┘ └──────────┘
                                                                                   │
                                  ＋ Weaning / breast 乳腺炎（哺乳期任意时间，多 POD ≥7）
                                  ＋ 抗生素无效持续 spiking fever → 脓毒性盆腔血栓性静脉炎(SPT)
```

| W | 病因 (中英) | 典型时间窗 |
|---|---|---|
| **Wind** | 肺不张 atelectasis / 早期肺炎 pneumonia（呼吸系统） | POD 1–2 |
| **Womb** | 子宫内膜炎 endometritis（**剖宫产后最高频**） | POD 2–3（即 postoperative day 2–3，术后第 2–3 天；>24h） |
| **Water** | 尿路感染 UTI（导尿相关） | POD 3–5 |
| **Walk** | 深静脉血栓 DVT / 血栓栓塞 | POD 4–7 |
| **Wound** | 切口/伤口感染 wound infection | POD 4–7 |
| **Weaning / breast** | 乳腺炎 mastitis / 乳腺脓肿 abscess | 哺乳期任意，多 POD ≥7 |
| **Wonder drugs** | 药物热 drug fever（排除性诊断） | 任意时间 |
| **（W 之外）** | 脓毒性盆腔血栓性静脉炎 septic pelvic thrombophlebitis (SPT) | 抗生素治疗 ≥3–5 天仍 spiking fever |

## 二、逐病拆解（钩子 → 临床意义 → 查体 → 处理）

### 1. 子宫内膜炎 Endometritis（Womb，最高频，必考第一反应）

> [!danger] 剖宫产后发热的默认首选诊断
> **剖宫产 + 产后 >24h 发热 + 子宫底压痛 + 恶露异常 → 子宫内膜炎，先想它。**

- **典型出现时间**：POD 2–3（产后 >24h）。
- **题干钩子**：发热 + **子宫底压痛 (uterine fundal tenderness)** + **恶露脓臭 (purulent/foul lochia)** + 白细胞增多 (leukocytosis)。
  - → 临床意义：题干常**只写"弥漫下腹压痛"不直写 uterus**，本质是子宫压痛（Q126 考点）。
- **高危因素**：**剖宫产（最大）**、难产/产程延长 (prolonged labor)、胎膜破裂时间久（prolonged ROM, rupture of membranes，胎膜破裂）、绒毛膜羊膜炎 (chorioamnionitis)、反复阴道检查、B 族链球菌 (GBS, Group B Streptococcus) 定植、器械助产。
- **病原**：**多菌混合 (polymicrobial)** — 厌氧 + 需氧 G− + GBS/肠球菌。
- **处理**：**IV 克林霉素 clindamycin + 庆大霉素 gentamicin**（首选，覆盖厌氧 + G−）；疗效差 / 疑肠球菌 (Enterococcus) → **加氨苄西林 ampicillin**（头孢常覆盖不到肠球菌）。退热 48h 后可停。
  - 记忆：**clinda（厌氧）+ gent（G−）**；amp 补肠球菌。

> [!warning] 易混分流（Q126 烟雾弹）
> - **发热为主导 = 感染（内膜炎），不是出血** —— 腹腔内出血以**心动过速 / 低血压**为主，不以发热为主。
> - **导尿后尿 RBC 是干扰项** —— 单纯导尿后镜下血尿很常见，不解释发热 + 子宫压痛，不能据此跳到膀胱炎。

### 2. 切口 / 伤口感染 Wound Infection（Wound）

- **典型出现时间**：POD 4–7。
- **题干钩子**：**切口红肿热痛 (erythema, warmth, induration)**、**脓性/血清样渗液 (purulent drainage)**、切口裂开。
  - → 临床意义：发热**定位到切口本身**（剖宫产腹壁切口 / 会阴侧切 / 撕裂缝合处）。
- **病原**：皮肤菌群（金葡菌 S. aureus、链球菌）± 内膜炎同源混合菌。
- **处理**：**开放引流 (open & drain) + 清创**为核心；蜂窝织炎/全身症状加抗生素。
  - 注意与坏死性筋膜炎 (necrotizing fasciitis) 鉴别：剧痛失比例、捻发音、皮肤花斑/快速进展 → **急诊手术清创**。

### 3. 脓毒性盆腔血栓性静脉炎 Septic Pelvic Thrombophlebitis (SPT)

> [!danger] 「抗生素打不退的持续 spiking fever」= 想 SPT
> 排除诊断 (diagnosis of exclusion)：内膜炎规范抗生素 **3–5 天**仍**反复尖峰热 (spiking fever)**、**患者一般情况却不差 (well-appearing)**、查不到其他感染灶。

- **典型出现时间**：产后/术后，**抗生素治疗数天后**仍发热。
- **机制**：盆腔静脉（多卵巢静脉 ovarian vein）内**感染性血栓**持续播散菌血症 → 抗生素难穿透血栓。
- **题干钩子**：持续 spiking fever **+ 对抗生素无反应** + 影像（CT/MRI）可见盆腔/卵巢静脉血栓（但常为临床/排除诊断）。
- **诊断性试验 / 处理**：**经验性加用肝素 (heparin) 抗凝** —— **加肝素后退热 = 既是诊断又是治疗**（"therapeutic + diagnostic trial of heparin"）；同时**继续广谱抗生素**。
  - 记忆：**抗生素无效的产后持续热 → 试肝素，热退即确诊。**

### 4. 乳腺炎 Mastitis / 乳腺脓肿 Breast Abscess（Weaning / breast）

- **典型出现时间**：哺乳期，多 **POD ≥7**（甚至产后数周）。
- **题干钩子**：**单侧乳房楔形红、肿、热、痛 (focal wedge-shaped erythema)** + 发热 + 流感样肌痛；常有乳头皲裂 / 乳汁淤积。
  - → 临床意义：定位在**乳房而非子宫/切口**。
- **病原**：**金黄色葡萄球菌 S. aureus（最常见）**。
- **处理**：
  - **乳腺炎**：**继续哺乳 / 排空乳汁 (continue breastfeeding) + 双氯西林 dicloxacillin**（或头孢氨苄 cephalexin）；疑耐甲氧西林金葡菌 (MRSA, methicillin-resistant S. aureus) 用克林霉素 clindamycin / 复方磺胺甲噁唑 (TMP-SMX, trimethoprim-sulfamethoxazole)。
  - **乳腺脓肿 (abscess)**：触及**波动性肿块 (fluctuant mass)** / 超声示脓腔 → **切开引流 (I&D, incision and drainage) 或穿刺抽吸 (needle aspiration) + 抗生素**；脓肿侧可暂停亲喂、改吸奶器排空。

> [!tip] 乳腺炎 vs 乳腺脓肿一句话
> 红肿热痛无波动 → 乳腺炎（继续哺乳 + dicloxacillin）；**有波动性肿块 → 脓肿（引流）**。

### 5. 尿路感染 UTI（Water）

- **典型出现时间**：POD 3–5（**导尿 (Foley) 相关**）。
- **题干钩子**：尿频/尿急/尿痛 (dysuria)、耻骨上痛；**肾盂肾炎 (pyelonephritis)**则 +肋脊角叩痛（CVA tenderness, costovertebral angle，肋脊角）+ 高热寒战。
  - → 临床意义：必须有**尿路症状 + 尿检感染证据（脓尿 pyuria / 亚硝酸盐 nitrite / 细菌）**；单纯"导尿后镜下红细胞 (RBC, red blood cell)"**不算** UTI（尿路感染）。
- **病原**：E. coli 最常见。
- **处理**：尿培养 + 对症抗生素；肾盂肾炎需静脉 (IV, intravenous) 抗生素。

### 6. 肺不张 Atelectasis / 肺炎 Pneumonia（Wind）

- **典型出现时间**：POD 1–2（**全麻 / 剖宫产后最早期发热源**）。
- **题干钩子**：术后早期低热、呼吸音减弱/湿啰音、血氧饱和度 (SpO₂) 略降；肺炎则脓痰 + 实变。
  - → 临床意义：**最早期（POD 1–2）发热**优先想肺；鼓励**深呼吸 / 激励性肺量计 (incentive spirometry) + 早期下床**。
- **处理**：肺不张以肺复张为主；确诊肺炎给抗生素。

### 7. 深静脉血栓 DVT / 肺栓塞 PE（Walk）

- **典型出现时间**：POD 4–7（妊娠/产褥期本就是**高凝状态 hypercoagulable**）。
- **题干钩子**：单侧腿肿胀/压痛、Homan 征；肺栓塞 (PE, pulmonary embolism) 则突发呼吸困难 + 胸痛 + 低氧 + 窦速。
  - → 临床意义：产褥期是静脉血栓栓塞 (VTE, venous thromboembolism) 高危窗口，发热可较轻；别只盯感染。
- **处理**：多普勒超声（诊断 DVT）/ 肺动脉 CT 血管造影（CTPA, CT pulmonary angiography，诊断 PE）→ 抗凝（肝素 heparin / 低分子量肝素 LMWH, low-molecular-weight heparin）。

### 8. 药物热 Drug Fever（Wonder drugs，排除性）

- **典型出现时间**：任意时间，常与新药相关。
- **题干钩子**：发热但**一般情况好 (well-appearing)**、**相对缓脉 (relative bradycardia)**、嗜酸性粒细胞增多 (eosinophilia)、皮疹；查遍无感染灶。
- **处理**：**停可疑药 → 体温很快回落**即确诊。

## 五、高频考点 & 复习题

> [!success] 三句话锁定
> 1. **剖宫产 + 产后 >24h 发热 + 子宫压痛 → 子宫内膜炎；首选克林霉素 + 庆大霉素。**
> 2. **抗生素打不退的持续 spiking fever（患者却不差）→ 脓毒性盆腔血栓性静脉炎；加肝素，退热即确诊。**
> 3. **单侧乳房楔形红肿 → 乳腺炎（继续哺乳 + 双氯西林）；有波动性肿块 → 脓肿（引流）。**

> [!question]- Q1. 剖宫产后 48h，T 38.9，下腹弥漫压痛、恶露脓臭，WBC 16000。最可能诊断 + 首选处理？
> **子宫内膜炎 (postpartum endometritis)**。题干"弥漫下腹压痛"实为子宫底压痛。首选 **IV 克林霉素 clindamycin + 庆大霉素 gentamicin**；疑肠球菌再加氨苄西林。

> [!question]- Q2. 产后子宫内膜炎已用克林霉素 + 庆大霉素 5 天，仍每日尖峰热，但患者精神食欲均好，查不到新感染灶。下一步最佳处理？
> 考虑**脓毒性盆腔血栓性静脉炎 (SPT)**。**经验性加用肝素 heparin** —— 加肝素后退热既是诊断也是治疗，同时继续广谱抗生素。

> [!question]- Q3. 产后 10 天，左乳外上象限楔形红、肿、热、痛 + 发热寒战；触及一处波动性肿块。诊断 + 处理？
> **乳腺脓肿 (breast abscess)**（在乳腺炎基础上形成脓腔）。**切开引流 (I&D) 或穿刺抽吸 + 抗生素（覆盖金葡菌，如双氯西林）**；无波动性肿块的单纯乳腺炎才是"继续哺乳 + 双氯西林"。

> [!question]- Q4. 全麻剖宫产术后第 1 天 (POD 1) 低热，呼吸音双肺底减弱，SpO₂ 94%。最可能发热源 + 处理？
> **肺不张 (atelectasis)**（7 W 中最早的 Wind）。处理 = **激励性肺量计 (incentive spirometry) + 深呼吸 + 早期下床**。

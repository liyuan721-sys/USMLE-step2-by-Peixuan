# 妊娠期糖尿病 (Gestational Diabetes Mellitus, GDM)

> [!tip] 核心一句话
> GDM = 胎盘激素驱动的**外周胰岛素抵抗 (insulin resistance)**；**24-28 周普查**，管理从**饮食 → 胰岛素**阶梯升级，产后**必须复查**（因 T2DM 风险↑）。

> [!info] 缩写全称
> - **GDM = Gestational Diabetes Mellitus（妊娠期糖尿病）**
> - **hPL = human Placental Lactogen（人胎盘催乳素）**
> - **GCT = Glucose Challenge Test（葡萄糖负荷试验）**；**GTT = Glucose Tolerance Test（葡萄糖耐量试验）**
> - **OGTT = Oral GTT（口服葡萄糖耐量试验）**；**T2DM = Type 2 Diabetes Mellitus（2 型糖尿病）**

---

## 一、病理生理

**人胎盘催乳素 (hPL) 等胎盘激素分泌 → 母体外周胰岛素抵抗↑** → 血糖↑（把葡萄糖优先留给胎儿）。胰岛 β 细胞代偿不足者 → GDM。

---

## 二、筛查（universal，全员筛）

```
所有孕妇 24-28 周普查
  → 1h 50g GCT（口服 50g，1 小时测血糖，不需空腹）
      ├─ 阴性 → 结束
      └─ 阳性 → 3h 100g GTT（确诊试验）
高危因素（肥胖 / 既往 GDM / 巨大儿史 / 一级亲属 DM）→ 提早筛，阴性 24-28 周复筛
```

---

## 三、血糖目标（超任一即算控制不佳）

| 时点 | 目标 |
|---|---|
| 空腹 fasting | **≤95 mg/dL** |
| 餐后 1h | **≤140 mg/dL** |
| 餐后 2h | **≤120 mg/dL** |

---

## 四、管理阶梯

> [!success] 一线 → 二线
> - **一线：饮食 (diet) + 运动 (exercise)**（低碳、少食多餐）。
> - **二线（饮食下血糖仍超目标）：加药**——**胰岛素 (insulin) = 首选**（不过胎盘、剂量易调）；**glyburide（格列本脲）/ metformin（二甲双胍）** 用于拒绝/无法用胰岛素者。

> [!danger] 孕期禁用（高频陷阱）
> - **SGLT2i（钠-葡萄糖共转运蛋白2抑制剂 sodium-glucose cotransporter-2 inhibitor，如 canagliflozin 卡格列净）/ TZD（噻唑烷二酮 thiazolidinedione，如 pioglitazone 吡格列酮）孕期禁用**（胎儿不良）。
> - **减重 (weight loss) 孕期不推荐**（增 SGA (small for gestational age, 小于胎龄儿) / 早产风险）。

---

## 五、母胎风险（为什么要控糖）

| 母体 | 胎儿/新生儿 |
|---|---|
| 妊娠期高血压 / 子痫前期 (preeclampsia) | **巨大儿 (macrosomia)** → **肩难产 (shoulder dystocia)** |
| 剖宫产率↑ | 新生儿低血糖 (neonatal hypoglycemia) |
| 羊水过多 (polyhydramnios) | 新生儿低钙、红细胞增多、RDS (respiratory distress syndrome, 新生儿呼吸窘迫综合征) |

---

## 六、产后管理 ⭐（本次新增知识点）

> [!warning] 胎盘一娩出，血糖多恢复正常——但别掉以轻心
> - **机制**：胎盘娩出 → hPL 等消失 → 胰岛素抵抗↓ → 血糖多恢复正常。
> - **但所有 GDM 患者 T2DM 风险↑**（本身有胰岛素抵抗倾向；部分其实是孕前未诊断的 T2DM）。
> - **必查：产后 6-12 周做 2 小时 75g OGTT** 筛查糖尿病。
> - **阴性 → 每 3 年复查一次**（长期随访）。

```
GDM 产后
  → 6-12 周 2h 75g OGTT
      ├─ 正常 → 每 3 年复查
      └─ 异常 → 诊断糖尿病前期 / T2DM，转内分泌管理
```

---

## 七、易混点

> [!tip]
> - **筛查 = 24-28 周 1h 50g GCT → 3h 100g GTT**（孕期）；**产后 = 6-12 周 2h 75g OGTT**（注意剂量/时长/时机全不同，别混）。
> - 二线药记"**胰岛素首选**"；口服（格列本脲/二甲双胍）是替代，不是禁用，但**新型降糖药（SGLT2i/TZD）孕期禁**。

# 痛经伴月经过多的子宫病变鉴别

> [!tip] 一句话主线
> 育龄至围绝经期女性 **痛经 (dysmenorrhea) + 月经过多 (menorrhagia)** 组合，鉴别核心落在子宫本身：**子宫腺肌症 (adenomyosis, 子宫腺肌症)** / **子宫肌瘤 (leiomyoma / fibroid, 子宫平滑肌瘤)** / **子宫内膜息肉 (endometrial polyp, 子宫内膜息肉)** / **子宫内膜癌 (endometrial carcinoma, 子宫内膜癌)**。**决定性线索是子宫的"形态"与"宫腔"**——弥漫球形 vs 结节不规则 vs 宫腔局灶病变。

---

## 一、决策树：从子宫形态与宫腔切入

```text
痛经 + 月经过多
   │
   ├─ 规则月经 + 子宫弥漫增大、光滑、球形 (globular) + 宫腔正常
   │        └──> 子宫腺肌症 Adenomyosis
   │
   ├─ 子宫不规则、结节样 (nodular)、轮廓异常 + 可有压迫症状
   │        └──> 子宫肌瘤 Leiomyoma
   │
   ├─ 宫腔内局灶病变 (sonohysterography 见占位) + 常不规则出血
   │        └──> 子宫内膜息肉 Endometrial polyp
   │
   └─ 绝经后出血 / 长期无对抗雌激素暴露
            └──> 排 子宫内膜癌 Endometrial carcinoma（内膜活检）
```

> [!info] 关键分流点
> - **子宫形态** = 弥漫球形（腺肌症）↔ 结节不规则（肌瘤）
> - **宫腔** = 正常（腺肌症 / 肌瘤为主时）↔ 局灶病变（息肉 / 黏膜下肌瘤）
> - **年龄 + 出血模式** = 绝经后 / 异常出血须先排内膜癌

---

## 二、核心鉴别表

表格如下：

| 疾病 | 月经周期 | 子宫形态 | 宫腔 | 高频提示 |
|---|---|---|---|---|
| 子宫腺肌症 Adenomyosis | 常规则 | 弥漫增大、光滑、球形、可有压痛 (boggy) | 正常 | 痛经加重 + 经量增多 |
| 子宫肌瘤 Leiomyoma | 可增多 | 不规则、结节样、轮廓异常 | 可正常 / 黏膜下型突入宫腔 | 压迫症状、可扪及肿块 |
| 子宫内膜息肉 Endometrial polyp | 常不规则出血 | 子宫多正常大小 | 局灶宫腔病变 | 经间期 / 不规则出血 |
| 子宫内膜癌 Endometrial carcinoma | 常异常 / 绝经后出血 | 不靠球形子宫诊断 | 内膜增厚 / 异常 | 无对抗雌激素、绝经后出血 |

---

## 三、子宫腺肌症 Adenomyosis（本专题主角）

> [!success] Core Concept
> 腺肌症 = **子宫内膜腺体侵入子宫肌层 (endometrial glands within the myometrium)**。异位内膜仍随激素周期反应，反复在肌层内出血 → 子宫弥漫增大 + 炎症 → **又痛又多的月经**。

### 机制链

```text
endometrial glands invade myometrium
        ↓
cyclic hormonal response continues
        ↓
repeated bleeding within uterine muscle
        ↓
uterine enlargement + inflammation
        ↓
dysmenorrhea + menorrhagia + boggy globular uterus
```

### 典型画像

> [!tip] Stem 招牌组合
> - 40–50 岁经产妇 (multiparous)
> - 规则月经背景下 **痛经进行性加重 + 经量增多**
> - 查体 / 影像：**光滑、球形 (smooth globular)、弥漫增大、可有压痛的子宫**
> - 宫腔超声 (sonohysterography)：**宫腔正常，无局灶病变**

### 诊断与处理要点

- **首选影像**：经阴道超声 (TVUS)；MRI 更敏感（示 junctional zone 增厚）。
- **确诊**：既往靠子宫切除术后病理，现临床 + 影像即可拟诊。
- **处理**：对症（NSAIDs / 激素避孕 / 左炔诺孕酮宫内节育器 LNG-IUD）；根治为子宫切除 (hysterectomy)。

## 四、鉴别击破（易混陷阱）

> [!warning] 腺肌症 vs 子宫肌瘤——最强干扰
> 两者都可致 menorrhagia。分水岭在子宫轮廓：
> - **腺肌症 = 弥漫、光滑、球形 (globular)**；宫腔正常。
> - **肌瘤 = 局灶、结节样 (nodular)、轮廓不规则**。
> - Stem 反复强调 `smooth / globular / diffuse / no contour abnormalities` → 指向腺肌症而非肌瘤。

> [!info] 其他干扰项
> - **子宫内膜息肉**：更偏经间期 / 不规则出血，宫腔超声见局灶宫腔病变。
> - **子宫内膜癌**：多见于绝经后或长期无对抗雌激素暴露，靠内膜异常（而非球形子宫）提示，须内膜活检。
> - **PCOS (多囊卵巢综合征)**：月经稀发、无排卵、高雄激素，子宫多正常——**不符合本组规则月经 + 子宫体征**。

---

## 五、Memory Hook

> [!tip] 记忆钩
> **Adeno = diffuse inside the wall（腺肌症 = 内膜钻进肌层壁里）**
> - `globular boggy uterus` 比单纯"子宫大"更有辨识度
> - 经量多 + 痛经，先想肌层里的异位内膜
> - `smooth` 帮你排掉肌瘤的结节感

## 六、自测

> [!question]- 42 岁 G2P2，规则月经但近年痛经加重、经量增多；查体子宫弥漫增大、光滑、球形、轻压痛，宫腔超声示宫腔正常。最可能诊断？
> **子宫腺肌症 Adenomyosis**。
> 关键：规则月经 + 光滑球形弥漫增大子宫 + 宫腔正常 = 腺肌症的招牌组合；`smooth globular + normal cavity` 排除结节样的子宫肌瘤。

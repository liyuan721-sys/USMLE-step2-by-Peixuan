# GBS 产时抗生素预防 IAP (Intrapartum Antibiotic Prophylaxis)

> [!tip] 核心一句话
> **IAP = Intrapartum Antibiotic Prophylaxis（产时/分娩期抗生素预防）**，整套框架只有一个目的：**预防新生儿早发型 GBS 感染（early-onset neonatal GBS disease）**。所有指征、所有风险因素，本质都是在估计"分娩时垂直传播 GBS 的风险高不高"。

> [!info] 缩写全称
> - **IAP = Intrapartum Antibiotic Prophylaxis（产时抗生素预防）**
> - **GBS = Group B Streptococcus（B 族链球菌 / 无乳链球菌 *Streptococcus agalactiae*）**
> - **ROM = Rupture Of Membranes（胎膜破裂）**；**PROM = Premature ROM（胎膜早破）**
> - **NAAT = Nucleic Acid Amplification Test（核酸扩增检测）**
> - **EOS = Early-Onset Sepsis（早发型败血症，生后 <7 天，多在 24h 内）**

---

## 一、为什么要做 IAP：只防 early-onset，不防 late-onset

母亲阴道/直肠定植 GBS → 分娩时新生儿经产道被**垂直传播** → 新生儿败血症/肺炎/脑膜炎。

```
母体 GBS 定植 (rectovaginal colonization)
   → 分娩经产道暴露
   → early-onset GBS disease (<7 天, 多在 24h 内)
   → 败血症 / 肺炎 / 脑膜炎
```

> [!danger] 高频陷阱：IAP 只挡 early-onset
> - **Early-onset（产时经产道感染）** → IAP 有效 ✅
> - **Late-onset（>7 天, 多为院内/社区水平传播）** → IAP **无预防作用** ❌
> 考题若问"IAP 能预防哪种"，只选 early-onset。

---

## 二、谁需要打 IAP（指征）

| 情况 | 是否 IAP |
|---|---|
| 36 0/7–37 6/7 周直肠阴道培养 **阳性** | ✅ |
| 本次妊娠任何时候 **GBS 菌尿 (GBS bacteriuria)**，无论菌量 | ✅（且**不需要** 36 周再培养，自动打） |
| 既往生过 **侵袭性 GBS 病**的婴儿 | ✅（无论本次培养结果） |
| **GBS 状态未知** + 风险因素任一（见 §三） | ✅ |
| 本次妊娠 GBS 培养**阴性** | ❌ |
| **择期剖宫产 + 胎膜完整 + 未临产** | ❌（胎儿不经产道；即使 GBS 阳性也不打） |

> [!warning] 三个"单独就成立"的指征
> **GBS 菌尿**、**既往 GBS 患儿**、**本次培养阳性**——这三条哪怕足月、无发热、破膜时间短，也照打不误，不需要再看风险因素。

---

## 三、GBS 状态未知 → 走风险因素法（risk-based）

状态未知时，用 4 条风险因素估计传播风险。**足月本身只是消掉"早产"这一条，其余 3 条仍要逐条排查。**

| 风险因素 | 为什么它提示高风险 |
|---|---|
| 早产 <37 周 | 早产儿免疫弱、GBS 侵袭风险高 |
| 破膜 ROM ≥18 小时 | 上行感染时间窗长，GBS 更易侵入 |
| 产时发热 ≥38°C (100.4°F) | 提示可能已有感染/定植活跃 |
| 产时 NAAT 阳性（如有条件做） | 直接证据 |

- **4 条全不满足 → 不打** ✅
- **任一满足 → 打** ❌

> [!danger] "足月不是免死金牌"
> 常见错误：只记住"早产要打"，忘了**破膜 ≥18h** 和**产时发热**对足月产妇同样成立。
> 正确表述：**"未知状态 + 足月 + 破膜 <18h + 无产时发热 + NAAT 阴性/未做" → 才免打；任一不清 → 打。**

## 四、用什么药（青霉素过敏路径是重点）

- **首选：Penicillin G（青霉素 G）IV**；替代 **Ampicillin（氨苄西林）**。
- **青霉素过敏分层：**

```
青霉素过敏?
├─ 低风险 (非过敏反应型皮疹)
│     → Cefazolin (头孢唑林)
└─ 高风险 (曾过敏性休克 / 血管性水肿)
      → 查 clindamycin 敏感性
            ├─ 敏感  → Clindamycin (克林霉素)
            └─ 耐药  → Vancomycin (万古霉素)
```

- **时机：**理想上分娩前 **≥4 小时**给足，通常临产或破膜时开始。

## 五、关键分岔：防 GBS（IAP） vs 治宫内感染

> [!danger] 足月"产时发热"是分岔口
> 产时发热既是 IAP 的一条风险因素，也可能是 **chorioamnionitis / intraamniotic infection（绒毛膜羊膜炎/宫内感染）**的信号。两条路目的与用药完全不同，别混。

```
足月产妇, 产时发热
        │
        ├─ 孤立发热, 无感染证据
        │     → 归为 GBS 风险因素
        │     → IAP: 单药 Penicillin G (预防 GBS)
        │
        └─ 发热 + 宫底压痛 / 羊水臭 / 母胎心动过速 / WBC↑
              → intraamniotic infection (宫内感染)
              → 广谱: Ampicillin (氨苄西林) + Gentamicin (庆大霉素) (±厌氧覆盖)
                 + prompt delivery (尽快分娩)
              ※ 该方案已覆盖 GBS, 不再单独谈"GBS 预防"
```

| | IAP（GBS 预防） | 宫内感染治疗 |
|---|---|---|
| 目的 | **预防**新生儿 early-onset GBS | **治疗**已发生的母体感染 |
| 用药 | 单药 Penicillin G（青霉素 G） | **广谱**：Ampicillin（氨苄西林）+ Gentamicin（庆大霉素）（剖宫产加 clindamycin 克林霉素/metronidazole 甲硝唑 补厌氧） |
| 触发 | §二/§三 指征与风险因素 | 母体发热 + 宫底压痛/羊水臭/胎心快/母胎心动过速/WBC↑ |
| 附加 | — | prompt delivery（尽快分娩）|

> [!tip] 一句话锁定
> **表里的东西都是防 GBS；唯一会"升级"成另一件事的，是产时发热真的发展成宫内感染时——那时是"治感染"（广谱），不是"防 GBS"（单药青霉素）。**

### 5.1 宫内感染广谱方案的选药原理（为什么是 Ampicillin + Gentamicin，剖才加厌氧）

> [!info] 核心逻辑
> 宫内感染是**下生殖道菌群上行的混合感染 (polymicrobial)**，不是单一致病菌，所以要用能同时盖 G+、G−、（手术时）厌氧的组合拳，而不是单一窄谱。

**先看敌人是谁——一锅混合菌：**

| 菌类 | 代表 | 谁来盖 |
|---|---|---|
| 革兰阳性 (G+) | GBS（B 族链球菌）、Enterococcus（肠球菌）、Listeria（李斯特菌） | **Ampicillin（氨苄西林）** |
| 需氧革兰阴性杆菌 (aerobic G−) | E. coli（大肠杆菌）、Klebsiella（克雷伯菌） | **Gentamicin（庆大霉素）** |
| 厌氧菌 (anaerobes) | Bacteroides（拟杆菌）、Peptostreptococcus（消化链球菌） | **Clindamycin（克林霉素）/ Metronidazole（甲硝唑）** |

**逐个药为什么选它：**

> [!warning] ① Ampicillin（氨苄西林）—— 管 G+ 这一头
> 盖 **GBS + Enterococcus（肠球菌）+ Listeria（李斯特菌）** + 部分 G−。
> **为什么不用头孢替？** 关键在 **Enterococcus 和 Listeria——头孢普遍不覆盖**，而它们正是围产期混合感染里会出现的 G+。这是氨苄西林在产科感染里几乎不可替代的原因。

> [!warning] ② Gentamicin（庆大霉素）—— 管 G− 这一头
> 氨基糖苷类，主打**需氧革兰阴性杆菌**（E. coli 是最常见 G− 元凶，还有 Klebsiella）。
> 附加值：与 β-内酰胺（氨苄西林）合用对肠球菌/链球菌有**协同杀菌 (synergy)**。
> 代价：可过胎盘、理论有胎儿耳/肾毒性，但产科重症感染"获益>风险 + 短疗程"，照用，注意母体肾功能。

> [!warning] ③ Clindamycin / Metronidazole —— 只在剖宫产时加，补厌氧
> 专攻**厌氧菌（Bacteroides 等）**，因为氨苄+庆大这对组合**厌氧覆盖不足**。
> **为什么偏偏剖宫产才加？**（高频问点）
> - **阴道分娩**：子宫经产道自然引流，厌氧负荷低，氨苄+庆大 + 尽快分娩通常够。
> - **剖宫产**：手术切口 + 失活组织 (devitalized tissue) = 厌氧菌温床，术后**子宫肌内膜炎 (endomyometritis) 以厌氧菌为主**，必须补厌氧覆盖。

> [!danger] USMLE 高频对照：别和"产后子宫内膜炎"搞混
> 同为产科感染，起手药却不同，考试爱放一起：
>
> | 场景 | 首选方案 | 记忆逻辑 |
> |---|---|---|
> | **Chorioamnionitis（绒毛膜羊膜炎，产时）** | **Ampicillin + Gentamicin** + 立即分娩（剖才加 clinda/metro） | 混合菌，先盖 G+/G−，厌氧看是否手术 |
> | **Postpartum endometritis（产后子宫内膜炎）** | **Clindamycin + Gentamicin**（不够再加 ampicillin） | 产后=厌氧为主，厌氧(clinda)打头 + G−(gent)，疑肠球菌再补氨苄 |
>
> 差别关键：**绒毛膜羊膜炎从 β-内酰胺（氨苄）起手**（还没手术，厌氧非主力）；**产后内膜炎从 clindamycin（厌氧）起手**（产后厌氧当家）。

> [!tip] 选药原理一句话
> **"氨苄盖 G+（含头孢盖不了的肠球菌/李斯特），庆大盖 G−，剖了再加厌氧"**；和产后内膜炎的区别是**起手药换了头**（绒毛膜=氨苄起手，产后=克林起手）。
> 另记两点：① 治疗必须**绑定 prompt delivery（尽快分娩）**——感染源要娩出，剖宫产是另有产科指征而非因感染本身；② 疗程短，多数分娩后即停或再补一剂。

## 六、鉴别一眼过

> [!question]- 足月临产产妇，GBS 状态未知，破膜已 20 小时，无发热、宫底无压痛，下一步？
> **给 IAP（intrapartum penicillin）**。
> - 状态未知 → 走风险因素法；虽足月（排除早产这条），但 **ROM ≥18h** 满足 → 打。
> - 无发热 + 宫底无压痛 → 不是宫内感染，不需广谱抗生素，单药青霉素预防即可。
> - 记忆钩：足月不免死，破膜 ≥18h 照打。

> [!question]- 孕 39 周，GBS 阴性，择期剖宫产、胎膜完整、未临产，要不要 IAP？
> **不要**。胎儿不经产道，无产时垂直传播风险；且本次培养阴性。IAP 仅针对经产道分娩的 early-onset GBS 传播。

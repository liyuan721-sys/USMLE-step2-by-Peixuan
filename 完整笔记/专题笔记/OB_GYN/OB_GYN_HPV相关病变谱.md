# HPV 相关病变谱

> [!info] 笔记定位
> **一张图看懂 HPV（human papillomavirus 人乳头瘤病毒）在全身惹的事。**
> 把零散考点串成"**病毒型别 → 病变 → 部位 → 处理**"一条链：
> ① **病毒学**：低危(low-risk)管疣、高危(high-risk)管癌；E6/E7 致癌机制。
> ② **病变谱系**：尖锐湿疣 / CIN→宫颈癌 / 外阴阴道肛门癌 / 口咽癌 / 复发性呼吸道乳头状瘤(RRP)。
> ③ **疫苗 + 筛查**：Gardasil 9 + 链回 ASCCP 管理篇。
> **分工**：本篇是"谱系总图"；**Pap 异常怎么处理 → [[完整笔记/专题笔记/OB_GYN/OB_GYN_宫颈细胞学异常管理ASCCP]]**（姊妹篇，不在此重复）。

> [!note]- 缩写对照（全称 中英）
> - **HPV** = human papillomavirus 人乳头瘤病毒
> - **CIN** = cervical intraepithelial neoplasia 宫颈上皮内瘤变（CIN1/2/3）
> - **VIN / VaIN / AIN** = vulvar / vaginal / anal intraepithelial neoplasia 外阴/阴道/肛门上皮内瘤变
> - **SCC** = squamous cell carcinoma 鳞状细胞癌
> - **RRP** = recurrent respiratory papillomatosis 复发性呼吸道乳头状瘤
> - **Rb** = retinoblastoma protein 视网膜母细胞瘤蛋白（抑癌）
> - **Pap** = Papanicolaou smear 巴氏涂片
> - **ASCCP** = American Society for Colposcopy and Cervical Pathology 美国阴道镜与宫颈病理学会

## 一、病毒学：低危管疣、高危管癌 ⭐⭐⭐

| | **低危型 Low-risk** | **高危型 High-risk** |
|---|---|---|
| 代表型别 | **HPV-6、-11** | **HPV-16、-18**（+ 31/33/45/52/58）|
| 主要病变 | **尖锐湿疣（condyloma acuminatum 生殖器疣）**、RRP | **CIN → 癌**（宫颈/外阴/阴道/肛门/口咽/阴茎 SCC）|
| 致癌力 | 几乎不致癌 | **致癌**（整合入宿主基因组 → 持续表达 E6/E7）|

> [!danger] E6/E7 致癌机制（高频机制题）⭐
> 高危型整合后**持续表达两个癌蛋白**：
> - **E6 → 降解 p53**（失去"基因组卫士"/凋亡）
> - **E7 → 灭活 Rb**（视网膜母细胞瘤蛋白；释放 E2F → 细胞周期失控增殖）
> 记法：**"E6 吃 p53，E7 踢 Rb"** → 双抑癌基因同时失活 → 恶变。
> ⚠️ 只有**持续高危型感染**才致癌；多数 HPV 感染（尤年轻人）**1-2 年自行清除**——这正是 21-24 岁筛查/处理偏保守的根因（见 ASCCP 篇）。

## 二、病变谱系：HPV 在全身的"作案地图" ⭐⭐⭐

```
低危 6/11
  ├─ 尖锐湿疣 condyloma acuminatum（外阴/阴茎/肛周/阴道）—— koilocytes 挖空细胞
  └─ RRP 复发性呼吸道乳头状瘤（婴幼儿喉部 → 声嘶）—— 产道垂直传播

高危 16/18（+31/33/45…）→ 持续感染 → 上皮内瘤变 → 浸润癌
  ├─ 宫颈   CIN1/2/3 → 宫颈鳞癌(SCC)；18 偏腺癌(adenocarcinoma)
  ├─ 外阴   VIN → 外阴 SCC
  ├─ 阴道   VaIN → 阴道 SCC
  ├─ 肛门   AIN → 肛门 SCC（MSM 男男性行为者 / HIV / 免疫抑制高发）
  ├─ 口咽   口咽 SCC（扁桃体/舌根，16 型；HPV⊕ 预后较好）
  └─ 阴茎   阴茎 SCC
```

> [!tip] 组织学招牌：Koilocyte 挖空细胞 ⭐
> **Koilocyte（挖空细胞）= HPV 感染的细胞学/组织学标志**：核增大深染、**核周空晕(perinuclear halo)**、双核。Pap 或活检见挖空细胞 → 提示 HPV。**任何"生殖器疣 / 宫颈病变 + koilocyte"= HPV 反射**。

> [!warning] 复发性呼吸道乳头状瘤（RRP）= 低危型的"重病"
> **HPV-6/11** 经**产道（vaginal delivery）垂直传播** → 婴幼儿/儿童喉部乳头状瘤 → **进行性声嘶(hoarseness)、喘鸣、气道梗阻**；易复发、需反复清除。注意：剖宫产**不**常规用于预防（传播也可宫内/接触，证据不足）。母亲生殖器疣史 + 婴儿声嘶 = 想到 RRP。

## 三、尖锐湿疣 vs 两个高频"假疣"鉴别 ⭐

| | **尖锐湿疣 Condyloma acuminatum** | **扁平湿疣 Condyloma latum** | **传染性软疣 Molluscum contagiosum** |
|---|---|---|---|
| 病原 | **HPV-6/11** | **二期梅毒（secondary syphilis, *Treponema pallidum*）** | **痘病毒 Molluscum contagiosum virus（poxvirus）** |
| 形态 | **菜花样/乳头状、疣状突起**、可带蒂 | **扁平、湿润、灰白、宽基底**斑块 | **圆顶、蜡样、中央脐凹(umbilicated)** |
| 关键鉴别 | koilocytes 挖空细胞；醋酸白⊕ | **暗视野见螺旋体**；RPR/血清⊕；伴二期梅毒全身症状 | 中央脐凹；**Henderson-Paterson 包涵体（molluscum bodies 软疣小体）** |
| 处理 | 局部（**podophyllin 鬼臼毒素 / imiquimod 咪喹莫特 / TCA 三氯醋酸 trichloroacetic acid**）或冷冻/切除 | **治梅毒：苄星青霉素 benzathine penicillin G** | 自限；可刮除/冷冻 |

> [!danger] 最易混：尖锐湿疣 vs 扁平湿疣
> **同在生殖器、都叫 condyloma**，但：**acuminatum=尖（疣状突起）=HPV**；**latum=扁（扁平湿润斑块）=二期梅毒**。见"扁平湿润斑块"先查梅毒血清，别当 HPV 疣处理。详见 [[完整笔记/专题笔记/感染/感染_梅毒分期诊疗与血清学]]。

## 四、疫苗 + 筛查（处理细节链回姊妹篇）

> [!success] HPV 疫苗（Gardasil 9 九价）⭐
> - **覆盖**：**6/11（疣）+ 16/18（癌）+ 31/33/45/52/58**（额外高危型）。
> - **常规接种 9-26 岁**（理想 11-12 岁、**性暴露前**效果最好）；**27-45 岁**共同决策(shared decision-making)可补种。
> - **剂次**：**<15 岁起始 = 2 针**；**≥15 岁或免疫抑制 = 3 针**。
> - **是灭活/重组疫苗 → 孕期不推荐**（但非致畸，误种不需终止）；**哺乳期可接种**。
> - ⚠️ **接种不改变筛查时间表**：打过疫苗仍按年龄常规做 Pap/HPV 筛查。

> [!info] 筛查与异常处理 → 见姊妹篇
> "几岁开始筛/几岁停" → [[完整笔记/专题笔记/USMLE/USMLE_肿瘤筛查年龄停筛原则]]；
> "Pap/HPV 异常结果怎么办（ASC-US/LSIL/HSIL/AGC → 重复细胞学/阴道镜/LEEP）+ CIN 组织学处理" → [[完整笔记/专题笔记/OB_GYN/OB_GYN_宫颈细胞学异常管理ASCCP]]。本篇不重复。

## 五、Memory Hook 总集

```
病毒学
  低危 6/11  → 疣（condyloma acuminatum）+ RRP；不致癌
  高危 16/18 → 癌；E6 吃 p53、E7 踢 Rb（双抑癌失活）

病变地图（高危型）
  宫颈CIN→鳞癌(18偏腺癌) / 外阴VIN / 阴道VaIN / 肛门AIN(MSM·HIV) / 口咽SCC(16,预后好) / 阴茎SCC
  组织学招牌 = koilocyte 挖空细胞（核周空晕）

三疣鉴别
  尖(acuminatum 疣状) = HPV 6/11
  扁(latum 扁平湿润)  = 二期梅毒 → 苄星青霉素
  脐凹(umbilicated)   = 传染性软疣（痘病毒）

RRP = 6/11 产道传播 → 婴儿声嘶/喘鸣

疫苗 Gardasil 9（6/11/16/18+5型）
  9-26常规(性暴露前最佳)；27-45共决策
  <15岁=2针；≥15岁/免疫低=3针
  孕期不打（非致畸）；打了仍照常筛查
```

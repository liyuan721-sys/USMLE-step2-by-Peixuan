# 产后情绪障碍三分鉴别（Postpartum Mood Disorders）

产后情绪障碍是 USMLE Step 2 高频决策点。核心考法是把三种情况分清楚：**自限的产后忧郁（blues）**、**需要治疗的产后抑郁（depression）**、**属于精神科急症的产后精神病（psychosis）**。三者的关键分水岭是：症状**严重度**、**持续时间**、以及**是否有妄想/幻觉与伤婴风险**。

> [!info] 三个关键分辨轴
> 1. **时间轴**：产后 2-3 天起、14 天内自行消退 → blues；症状持续 >2 周 → depression；急性起病伴精神病性症状 → psychosis。
> 2. **精神病性症状**：出现妄想 delusions / 幻觉 hallucinations / 怪异行为 → 一定不是 blues，也不是单纯 depression，考虑 psychosis。
> 3. **风险评估**：任何产妇都要问自伤 / 伤婴念头（suicidal / infanticidal ideation）。

## 一、核心三分对照表

| 项目 | Postpartum blues 产后忧郁 | Postpartum depression 产后抑郁 | Postpartum psychosis 产后精神病 |
| --- | --- | --- | --- |
| 患病率 | 40-80% | 8-15% | 0.1-0.2% |
| 起病 | 产后 2-3 天，**14 天内消退** | 常产后 4-6 周（可至产后 1 年） | 数天至数周（常 <2 周，起病急） |
| 症状 | 轻度抑郁、流泪、易激惹、情绪波动 | ≥2 周中重度抑郁、睡眠/食欲紊乱、精力低、内疚、注意力差、**自杀意念** | 妄想 delusions、幻觉 hallucinations、思维紊乱、怪异行为、**杀婴 infanticide 风险** |
| 管理 | 安慰 reassurance + 观察（自限） | 心理治疗 psychotherapy、药物（SSRI），或两者联合 | 抗精神病药 + 抗抑郁药 + 心境稳定剂；**住院**（不可让母亲单独与婴儿相处） |

> [!tip] 一句话记忆
> Blues = 短、轻、自限（安慰即可）；Depression = 长（>2 周）、损功能（要治疗）；Psychosis = 妄想幻觉 + 伤婴急症（住院）。

## 二、产后抑郁（Postpartum Depression）诊断要点

诊断标准本质上就是 **major depressive episode（重性抑郁发作）** 发生在围产期（产后 4 周内起病，DSM「peripartum onset」标注；临床上产后 1 年内起病都常归入此类）。

> [!warning] Depression ≠ Blues 的关键
> - 症状**持续 >2 周**（blues 应在 14 天内自行消退）。
> - **中重度** + 伴**功能损害**（无法照料自己或婴儿）。
> - 一旦满足以上 → **需要治疗**，不能像 blues 那样只观察等待。

- 所有产妇产后随访都应**筛查抑郁**（常用 Edinburgh Postnatal Depression Scale, EPDS），并**主动询问自伤 / 伤婴念头**。
- 危险因素：既往抑郁 / 焦虑史、经前情绪障碍、缺乏社会支持、意外妊娠、婴儿健康问题。

## 三、一线治疗

> [!success] 首选 = SSRI
> - 初治首选 **SSRI**（选择性 5-羟色胺再摄取抑制剂）。轻度可单用心理治疗，中重度用 SSRI ± 心理治疗。
> - **哺乳期首选 sertraline 舍曲林**：进入乳汁量极低，婴儿血药浓度几乎测不到，安全性证据最充分。

```
产后抑郁治疗思路
│
├─ 轻度、无功能损害 ──► psychotherapy 心理治疗（CBT/IPT）可单用
│
├─ 中重度 / 功能受损 ──► SSRI（哺乳 → sertraline 舍曲林）± 心理治疗
│
└─ 有自伤/伤婴风险，或门诊无法依从 ──► 住院 inpatient
```

## 四、住院门槛（Inpatient Threshold）

> [!danger] 什么时候必须住院
> - 有**自伤 / 伤婴（suicidal / infanticidal）风险**。
> - **产后精神病**（妄想/幻觉/怪异行为）——一律按精神科急症处理，住院并**绝不可让母亲单独与婴儿相处**。
> - 门诊无法保证依从、无法安全随访。

> [!info] 辅助 ≠ 治疗
> 支持小组（support group）、育儿培训（parenting classes）是**有益辅助**，可改善社会支持，但**不足以替代抗抑郁治疗**。中重度产后抑郁仍需 SSRI，不能只给「多参加支持小组」就打发。

## 五、鉴别与躯体病因排查

产后情绪/精力异常不要一律当成原发精神障碍，先排除躯体病因：

| 检查 | 排除的病因 |
| --- | --- |
| Hb（血红蛋白） | 产后**贫血** anemia（乏力、心悸易被误当抑郁） |
| TSH | **甲减** hypothyroidism / **产后甲状腺炎** postpartum thyroiditis（可表现为疲乏、情绪低落） |

> [!tip] 考试钩子
> 产妇产后数周疲乏 + 情绪低落 + 心悸/怕冷/脱发 → 先想 TSH（产后甲状腺炎）；单纯乏力面色苍白 → 查 Hb。躯体病因纠正后情绪症状可缓解。

## 六、自测题

> [!question]- Q1. 产后 5 天产妇，间断流泪、易激惹、情绪波动，但能正常照料婴儿、无自杀或伤婴念头，睡眠尚可。最恰当的处理是？
> **产后忧郁 blues**（起病 2-3 天、轻度、可自照料、无危险念头）。处理 = **安慰（reassurance）+ 观察随访**，预期 14 天内自行消退。此时**不需要** SSRI，也不需要住院——若误判为抑郁而立即上药是典型陷阱。关键是随访确认症状按期消退；若 >2 周不退或加重，再重新评估是否为产后抑郁。

> [!question]- Q2. 产后 3 周产妇，突发言语混乱，坚信婴儿「被恶魔附身、必须让他解脱」，伴听幻觉。此为何种情况？住院门槛如何把握？
> **产后精神病 postpartum psychosis**（急性起病 + 妄想 + 幻觉 + 针对婴儿的怪异信念 = 杀婴 infanticide 高危）。属**精神科急症**，须**立即住院**，**绝不可让母亲单独与婴儿相处**；治疗为抗精神病药 + 心境稳定剂 ±抗抑郁药。住院门槛的通用原则：**只要存在自伤 / 伤婴风险，或门诊无法安全依从随访，即住院**；单纯中度产后抑郁而无危险念头者则可门诊 SSRI + 心理治疗，不必住院。

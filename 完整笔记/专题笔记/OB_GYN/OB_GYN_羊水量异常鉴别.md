# 羊水量异常鉴别 (Amniotic Fluid Volume Abnormalities)

羊水 (amniotic fluid) 中晚孕主要由**胎儿尿液 (fetal urine)** 产生，经**胎儿吞咽 (fetal swallowing)** 清除。因此羊水量异常本质是"产生 vs 清除"失衡，病因随孕期而变——这是 USMLE 分层考点。

> [!info] 羊水动力学（一句话记忆）
> **产尿多/吞咽少 → 羊水过多；产尿少/漏出多 → 羊水过少。**
> - 产生：胎儿尿 (fetal urine)、肺液
> - 清除：胎儿吞咽 (fetal swallowing)、经胎盘/胎膜吸收

---

## 一、定义与量化标准

两种超声定量方法：

- **羊水指数 (Amniotic Fluid Index, AFI)**：四象限最深池之和
- **单最深池 (Single Deepest Pocket, SDP)**：单个最大垂直深度

| 状态 | AFI | SDP |
| --- | --- | --- |
| 羊水过少 (Oligohydramnios) | ≤5 cm | <2 cm |
| 正常 (Normal) | 5–24 cm | 2–8 cm |
| 羊水过多 (Polyhydramnios) | ≥24 cm | ≥8 cm |

> [!tip] SDP 更适合多胎
> 多胎妊娠 (multiple gestation) 用 SDP 而非 AFI 评估单个胎儿。

---

## 二、羊水过少 (Oligohydramnios)：按孕期分层病因

这是本主题**最核心的考点**——同样是羊水过少，孕周不同病因完全不同。

```
羊水过少 (Oligohydramnios)
│
├─ 早孕 / 中孕早期 ──→ 胎儿源 (fetal cause)
│     ├─ 非整倍体 (aneuploidy)
│     ├─ 肾发育不全 (renal agenesis) —— 无肾 → 无尿 → 无羊水
│     └─ 后尿道瓣膜 (posterior urethral valves, PUV)
│            —— 男胎下尿路梗阻，尿排不出 → 羊水依赖胎儿尿产生受阻
│
├─ 中孕晚期 / 晚孕 ──→ 子宫胎盘功能不全 (uteroplacental insufficiency)
│     └─ 灌注↓ → 胎儿血流重分布保脑弃肾 → 尿↓
│            ★ 必伴 胎儿生长受限 (FGR)
│     └─ 母体因素：脱水 (maternal dehydration)
│
└─ 足月 / 新发 ──→ 最常见 = 自发性胎膜破裂 (spontaneous ROM)
      └─ 含亚临床缓慢渗漏 (subclinical slow leak)
             —— 表现为清亮阴道分泌物增多，患者常误以为白带/尿失禁
             —— 必须窥器检查排查 ROM
```

> [!danger] 足月新发羊水过少 = 先排 ROM
> 一个既往正常、足月出现羊水过少的孕妇，**最常见原因是自发性胎膜破裂 (spontaneous rupture of membranes)**，包括缓慢渗漏。哪怕她只主诉"清亮阴道分泌物增多"，也必须做**窥器检查 (sterile speculum exam)** 排查 ROM——查 pooling / ferning / nitrazine。不要一上来想胎盘功能不全。

> [!warning] 关键鉴别钩子：胎儿大小
> **胎儿大小正常（如 50th 百分位）→ 基本排除子宫胎盘功能不全。**
> 因为胎盘功能不全导致的羊水过少几乎必然伴随**胎儿生长受限 (FGR)**（同一个灌注不足的机制）。胎儿长得好但羊水少 → 把注意力转向 ROM 或胎儿泌尿系问题，而非胎盘。

> [!info] 早孕肾/尿路病因的辅助识别
> 早中孕羊水过少 + 男胎 + 膀胱扩张/"钥匙孔征 (keyhole sign)" → 后尿道瓣膜。
> 双肾未显影 + 羊水极少 → 双侧肾发育不全 (bilateral renal agenesis, Potter sequence 波特序列征)。

## 三、羊水过多 (Polyhydramnios)：病因

机制：胎儿**尿产生过多**（如母体高血糖致胎儿多尿）或**吞咽/吸收障碍**（消化道闭锁、神经管缺损）。

| 病因 | 机制 / 提示 |
| --- | --- |
| 特发性 (idiopathic) | **最常见**（约占多数），排除性诊断 |
| 母体糖尿病 (maternal DM) | 胎儿高血糖 → 渗透性多尿 → 尿↑ |
| 消化道闭锁 (GI atresia) | 十二指肠闭锁 (duodenal atresia，"double bubble 双泡征")、食管闭锁 → 吞咽受阻 |
| 神经管缺损 (neural tube defect) | 无脑儿/开放性缺损 → 吞咽反射障碍 |
| 多胎妊娠 (multiple gestation) | 尤其双胎输血综合征 (twin-twin transfusion syndrome, TTTS) 受血儿侧 |
| 胎儿水肿 (fetal hydrops) | 免疫性/非免疫性积液 |
| TORCH 感染 (TORCH infections) | 胎儿感染相关 |

> [!tip] 羊水过多的"double bubble"钩子
> 羊水过多 + 产前超声"双泡征" → **十二指肠闭锁 (duodenal atresia)**，且高度关联 **21-三体 (Down syndrome)**。

---

## 四、管理要点

> [!success] 羊水过少（足月）
> - **窥器检查排查 ROM**：pooling（后穹隆积液）/ ferning（羊齿状结晶）/ nitrazine 变蓝
> - 评估胎儿状况：BPP 生物物理评分、NST、脐动脉多普勒
> - 结合孕周 + 胎儿大小决定引产 vs 期待

> [!success] 羊水过多
> - **母体血糖筛查**（排/查糖尿病）
> - **详细胎儿解剖超声 (detailed fetal anatomy US)**：查 GI 闭锁、NTD、水肿
> - 必要时考虑感染 (TORCH) 及非整倍体评估

## 五、自测题

> [!question]- Q1. 一名 39 周初产妇常规产检发现 AFI = 4 cm。既往孕期正常，胎儿估重位于第 50 百分位，多普勒正常。她诉近两日"清亮阴道分泌物增多"。下一步最恰当的处理是？
> **窥器检查 (sterile speculum exam) 排查自发性胎膜破裂 (ROM)。**
> 分层逻辑：足月新发羊水过少最常见病因是 ROM（含缓慢渗漏）；**胎儿大小正常已基本排除子宫胎盘功能不全**（那会伴 FGR）。"清亮分泌物增多"正是缓慢渗漏的典型伪装。查 pooling / ferning / nitrazine。

> [!question]- Q2. 中孕早期超声发现羊水过少，男性胎儿，膀胱明显扩张伴"钥匙孔征"。最可能的病因是？
> **后尿道瓣膜 (posterior urethral valves)。**
> 男胎下尿路梗阻 → 尿排出受阻 → 羊水（依赖胎儿尿）减少 + 膀胱/后尿道扩张呈钥匙孔。对比：双肾未显影 + 羊水极少 → 肾发育不全。

> [!question]- Q3. 一名妊娠糖尿病控制欠佳的孕妇，超声示 AFI = 28 cm，胎儿超声见上腹"双泡征"。羊水过多最可能的两个相关因素是？
> **母体糖尿病 (maternal DM) 致胎儿渗透性多尿** + **十二指肠闭锁 (duodenal atresia) 致吞咽受阻**（双泡征）。
> 提示追查 21-三体 (Down syndrome) 关联。管理：查母体血糖 + 详细胎儿解剖超声。

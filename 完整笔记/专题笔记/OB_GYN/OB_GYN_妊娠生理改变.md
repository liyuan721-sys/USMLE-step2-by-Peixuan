# 妊娠期生理改变 Physiologic Changes of Pregnancy

> [!info] 一句话总览
> 妊娠几乎改变每个器官系统，CK 的考法永远是**"这个化验/症状是正常孕期生理，还是病理？"**。
> 母体把循环、血液、通气、肾滤过全部上调来供养胎儿；记住**哪些值"应该"变、变多少**，才能在题里把"生理"和"疾病"分开。
> 三个最常误判的"看着像病、其实正常"：**稀释性贫血、生理性白细胞增多、慢性代偿性呼吸性碱中毒**。

> [!note]- 本篇缩写速查（首次全称中英）
> - **CO** = cardiac output 心输出量；**SV** = stroke volume 每搏量；**HR** = heart rate 心率
> - **SVR** = systemic vascular resistance 体循环血管阻力；**BP** = blood pressure 血压
> - **Hb** = hemoglobin 血红蛋白；**WBC** = white blood cell 白细胞；**ESR** = erythrocyte sedimentation rate 红细胞沉降率
> - **VTE** = venous thromboembolism 静脉血栓栓塞症
> - **PFT** = pulmonary function test 肺功能；**VC** = vital capacity 肺活量；**FEV1** = forced expiratory volume in 1 s 第 1 秒用力呼气量
> - **FRC** = functional residual capacity 功能残气量；**RV** = residual volume 残气量；**TV** = tidal volume 潮气量
> - **GFR** = glomerular filtration rate 肾小球滤过率；**BUN** = blood urea nitrogen 尿素氮；**Cr** = creatinine 肌酐
> - **RAAS** = renin-angiotensin-aldosterone system 肾素-血管紧张素-醛固酮系统
> - **GERD** = gastroesophageal reflux disease 胃食管反流病
> - **hPL** = human placental lactogen 人胎盘催乳素；**hCG** = human chorionic gonadotropin 人绒毛膜促性腺激素
> - **TBG** = thyroxine-binding globulin 甲状腺素结合球蛋白；**TSH** = thyroid-stimulating hormone 促甲状腺激素
> - **CBG** = cortisol-binding globulin 皮质醇结合球蛋白；**T4/T3** = thyroxine/triiodothyronine 甲状腺素/三碘甲状腺原氨酸
> - **MCV** = mean corpuscular volume 平均红细胞体积
> - **GDM** = gestational diabetes mellitus 妊娠期糖尿病；**OGTT** = oral glucose tolerance test 口服葡萄糖耐量试验
> - **HELLP** = hemolysis, elevated liver enzymes, low platelets 溶血+肝酶升高+血小板减少综合征
> - **V/Q** = ventilation/perfusion 通气/血流（比值）；**PE** = pulmonary embolism 肺栓塞

## 一、心血管 Cardiovascular

| 指标 | 变化 | 机制 / CK 钥匙 |
|---|---|---|
| **血容量 (blood volume)** | ↑ 40–50% | 血浆 > 红细胞 → 稀释性贫血（见血液节） |
| **心输出量 (CO)** | ↑ 30–50% | 早期靠 **SV↑**、晚期靠 **HR↑**；峰值约孕 28–32 周 |
| **心率 (HR)** | ↑ 10–20 bpm | — |
| **体循环阻力 (SVR)** | **↓** | 孕酮 (progesterone) + 前列腺素 + 胎盘低阻分流 → 血管扩张 |
| **血压 (BP)** | **中孕期最低 (nadir)**，晚孕回升 | SVR↓ 主导；故"中孕血压略低"是正常 |

> [!tip] 正常 vs 异常的心脏体征
> - **正常**：生理性收缩期血流杂音 (systolic flow murmur)、**S3**、颈静脉轻充盈、下肢水肿
> - **异常（需查）**：**任何舒张期杂音 (diastolic murmur)**、响亮全收缩期杂音、S4 → 提示器质性心脏病
> - **仰卧位低血压综合征 (supine hypotension)**：增大子宫压迫下腔静脉 → 回心血量↓ → 头晕/低血压；处理 **左侧卧位 (left lateral decubitus)**。

## 二、血液 Hematologic（最爱考的"假性异常"）

| 指标 | 变化 | CK 钥匙 |
|---|---|---|
| **血红蛋白 (Hb)** | **↓（生理性/稀释性贫血）** | 血浆容量↑(~50%) > 红细胞量↑(~20–30%) → 稀释；正常孕期 Hb 下降是**生理**，非缺铁 |
| **白细胞 (WBC)** | **↑（生理性白细胞增多）** | 可达 ~15,000，**产程/产后更高（可达 20,000+）**；勿误判感染 |
| **凝血 (coagulation)** | **高凝状态 hypercoagulable** | 凝血因子 + 纤维蛋白原↑、蛋白 S↓、静脉淤滞 → **VTE 风险↑**（Virchow 三要素全凑齐）|
| **血小板 (platelet)** | 轻度↓（妊娠期血小板减少 gestational thrombocytopenia） | 轻度、无症状、产后恢复；重度↓要排子痫前期/HELLP |
| **ESR** | ↑ | 纤维蛋白原↑所致；孕期 ESR 失去诊断价值 |

> [!warning] 稀释性贫血 ≠ 缺铁性贫血
> 孕期 Hb 生理性下降，但**铁需求仍↑** → 仍建议补铁。真正缺铁看 **MCV↓ + 铁蛋白↓**；单纯稀释 MCV 正常。区分点：稀释是"水多了"，缺铁是"原料少了"。

## 三、呼吸 Respiratory（整合错题 Q19704）

| 指标 | 变化 | 机制 |
|---|---|---|
| **每分通气量 (minute ventilation)** | **↑**（主要靠 **潮气量 TV↑**，非呼吸频率） | **孕酮驱动过度通气** → 呼吸性碱中毒 |
| **PaCO₂** | **↓（~30 mmHg）** | 慢性代偿性呼吸性碱中毒 → 肾代偿 **HCO₃⁻↓**；利于胎儿卸 CO₂ |
| **功能残气量 (FRC) / 残气量 (RV)** | **↓** | 增大子宫顶高膈肌；轻度基底肺不张 |
| **肺活量 (VC) / FEV1** | **不变** | 膈肌活动度不减 → 通气功能保留 |
| **耗氧量** | ↑ | 母体 + 胎儿代谢需求 |

> [!danger] 妊娠生理性呼吸困难 vs 病理（Q19704 核心）
> "Dyspnea of pregnancy" = **分钟通气↑ + FRC↓ + VC/FEV1 正常** = 正常生理（排除性诊断）。
> 对账其他 PFT 模式：
> - **哮喘 (asthma)**：FEV1↓（阻塞）
> - **肥胖相关限制性通气 (obesity-related restriction)**：**VC↓ + 分钟通气↓**（与妊娠相反！别被高 BMI 带偏）
> - **过度通气综合征 (焦虑)**：分钟通气↑ 但 **FRC 正常**
> - **肺栓塞 (PE)**：急症、V/Q 失配，肺容积一般不变
> 详细 PFT 模式对照见 [[完整笔记/专题笔记/pulmonary/pulmonary_机械通气压力判读]] / [[完整笔记/Peixuan分科笔记/pulmonary]]。

## 四、肾 Renal

| 指标 | 变化 | CK 钥匙 |
|---|---|---|
| **GFR / 肾血流** | **↑（GFR ↑~50%）** | → **BUN 与肌酐 (Cr) 生理性下降**；孕妇"正常" Cr 比非孕低，**Cr 0.9 在孕期可能已是肾损** |
| **尿糖 (glucosuria)** | 生理性糖尿 | 滤过糖量超过近端小管重吸收上限；**孕期尿糖≠糖尿病**（诊断 GDM 靠 OGTT） |
| **集合系统** | 生理性肾积水 (hydronephrosis)，**右侧为主** | 孕酮致输尿管平滑肌松弛 + 子宫右旋压迫 → 尿淤滞 → 肾盂肾炎风险↑ |
| **酸碱** | 代偿性 HCO₃⁻↓ | 配合呼吸性碱中毒（见呼吸节） |

## 五、消化 GI

- **孕酮松弛食管下括约肌 + 胃排空延迟** → **GERD / 烧心**（孕期极常见）
- **胆囊排空↓ + 胆汁淤滞** → 胆固醇结石风险↑
- 肠蠕动↓ → **便秘**
- （注意区分：右上腹痛 + 瘙痒 + 胆汁酸↑ = **妊娠期肝内胆汁淤积症**，是病理不是普通生理）

## 六、内分泌 / 代谢 Endocrine-Metabolic

| 系统 | 变化 | CK 钥匙 |
|---|---|---|
| **甲状腺 (thyroid)** | **TBG↑ → 总 T4/T3↑，但游离 T4 正常** | hCG 弱 TSH 样作用 → **早孕 TSH 轻度↓**；甲状腺轻度增大；判甲功用 **游离 T4 + 孕期特异 TSH 区间** |
| **糖代谢** | **胰岛素抵抗↑** | **hPL（人胎盘催乳素）**等致抵抗 → 倾向妊娠期糖尿病 (GDM)；24–28 周 OGTT 筛查 |
| **肾上腺** | 皮质醇↑（CBG↑ + 游离皮质醇也↑） | 孕期可有类 Cushing 体征，需谨慎解读 |
| **催乳素 (prolactin)** | ↑ | 为泌乳准备；垂体增大 |

> [!tip] 孕期甲功判读
> 看到孕妇"总 T4 升高"别反射甲亢——那是 **TBG↑** 带来的结合型升高，**游离 T4 才是真相**。早孕 TSH 轻降也可由 hCG 生理性引起。

## 七、一句话"正常 vs 病理"对照（CK 高频陷阱）

```
看似异常，实为正常生理：
 ├─ Hb 下降 ──────── 稀释性贫血（MCV 正常）   ≠ 缺铁(MCV↓铁蛋白↓)
 ├─ WBC 升高(~15k) ── 生理性白细胞增多        ≠ 感染(看左移/发热/部位)
 ├─ PaCO₂ ~30 ────── 代偿性呼吸性碱中毒        ≠ 病理(看 VC/FEV1 正常)
 ├─ 尿糖阳性 ─────── 生理性糖尿(GFR↑)         ≠ GDM(靠 OGTT)
 ├─ 收缩期杂音/S3 ── 高动力血流杂音           ≠ 器质(舒张期杂音才异常)
 ├─ 右肾积水 ─────── 生理性(孕酮+右旋压迫)     ≠ 梗阻
 └─ 总 T4 升高 ───── TBG↑结合型              ≠ 甲亢(看游离 T4)
```

## 七-bis、孕期体重增长指南 + 营养不足风险 ⭐（Q18678）

> [!info] 目标增重按孕前 BMI (body mass index, 体质指数) 定
> 孕前 BMI 反映母体基线营养储备，储备越少要增得越多。

| 孕前 BMI (kg/m²) | 推荐总增重 |
|---|---|
| <18.5（低体重 underweight） | **12.7–18.1 kg（28–40 lb）** |
| 18.5–24.9（正常） | 11.3–15.9 kg（25–35 lb） |
| 25–29.9（超重 overweight） | 6.8–11.3 kg（15–25 lb） |
| ≥30（肥胖 obese） | 5–9 kg（11–20 lb） |

> [!danger] 两头都出问题（高频对照）
> - **增重不足 (inadequate weight gain)，尤其孕前低体重** → 营养供给不足 → **早产 (preterm delivery) + 低出生体重 (low birth weight) / SGA (small for gestational age, 小于胎龄儿)**。（Q18678：孕前 BMI 18 + 每周增 <0.5 kg + 宫高落后 24cm@26 周 + 胎儿 25 百分位）
> - **增重过多 / 孕前肥胖** → **妊娠期糖尿病 (GDM) + 巨大儿 (macrosomia) + 剖宫产 + 子痫前期 (preeclampsia)**。

> [!tip] 记忆
> **瘦妈增太少 → 早产/低体重；胖妈增太多 → GDM/巨大儿/剖。** 体重两头都要盯。

## 八、复习题

> [!question]- Q1. 孕 33 周呼吸困难，PFT：分钟通气↑、FRC↓、VC 正常、FEV1 正常。最可能解释？为何高 BMI 不选"肥胖限制性"？
> **妊娠正常生理改变（dyspnea of pregnancy）**。肥胖相关限制性通气应是 **VC↓ + 分钟通气↓**，与本题 VC 正常 + 分钟通气↑相反；妊娠 = 孕酮过度通气（TV 为主）+ 子宫顶膈致 FRC↓，而 VC/FEV1 保留。

> [!question]- Q2. 孕 30 周常规化验 Hb 10.5、WBC 14,000、血小板 110k，无症状。如何解读？
> 多为**正常孕期生理**：稀释性贫血（核对 MCV/铁蛋白排缺铁）、生理性白细胞增多、轻度妊娠期血小板减少。重度血小板↓或伴高血压/肝酶↑要排子痫前期/HELLP。

> [!question]- Q3. 孕妇血压在孕 24 周比孕前略低，是否异常？孕期血压如何变化？
> 正常。SVR↓ 使 **血压在中孕期降至最低 (nadir)**，晚孕回升。新发 ≥140/90 且 ≥20 周才考虑妊娠期高血压/子痫前期。

> [!question]- Q4. 孕妇总 T4 升高、TSH 轻度偏低、游离 T4 正常，无症状。诊断？
> **正常孕期生理**：TBG↑ 抬高总 T4（结合型），hCG 弱 TSH 样作用使早孕 TSH 轻降；**游离 T4 正常 = 甲功正常**，不是甲亢。

> [!question]- Q5. 为什么孕期是 VTE 高风险？列出 Virchow 三要素在孕期的对应。
> 高凝（凝血因子/纤维蛋白原↑、蛋白 S↓）+ 静脉淤滞（子宫压迫下腔静脉、孕酮致静脉扩张）+ 内皮损伤（分娩/剖宫产）。

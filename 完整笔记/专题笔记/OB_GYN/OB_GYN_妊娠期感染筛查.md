# 妊娠期感染筛查 (Prenatal Infectious Disease Screening)

> [!info] 本篇重点
> 核心是**孕期结核 (TB) 筛查阳性后的处理流程**（TST/IGRA 阳 → 胸片 → 潜伏 vs 活动），文末附常规产前感染筛查清单，方便 CK 快速对照。

---

## 一、孕期结核 (Tuberculosis, TB) 筛查与流程（核心）

### 1. 谁需要筛查（High-risk indications）

孕期不做普遍 TB 筛查，只针对高危人群 (targeted screening)：

- 与已知 TB 患者**密切接触** (close contact with a known active TB case)。
- 来自 TB **高发国家的移民** (immigrant from a high-prevalence country，如印度 India、部分亚非拉地区)。
- **高危环境暴露** (high-risk congregate settings，如收容所 homeless shelter、监狱 correctional facility、医疗机构 healthcare worker)。
- HIV 感染、免疫抑制等宿主高危因素。

### 2. 筛查方法（孕期均安全）

| 方法 | 全称 | 说明 |
| --- | --- | --- |
| TST | Tuberculin Skin Test 结核菌素皮试 (PPD) | 皮内注射，48–72h 读硬结；会被 BCG 卡介苗**假阳性**干扰 |
| IGRA | Interferon-Gamma Release Assay 干扰素γ释放试验 | 抽血；**不受 BCG 影响**，特异性更高 |

> [!tip] 关键
> **TST 和 IGRA 在孕期都安全**，可任选。孕期不是延迟筛查高危孕妇的理由。

### 3. 筛查阳性 → 下一步 = 胸片 (Chest X-ray, CXR)

> [!danger] 最高频考点
> TST 或 IGRA **阳性** → 下一步必须做 **胸片（带腹部铅屏蔽 abdominal lead shielding）**，用来区分**潜伏 (latent)** vs **活动 (active)** TB。
> 孕期胸片在会影响管理决策时是**安全且应当做**的（单张 CXR 胎儿剂量极低，加铅屏蔽后可忽略）。**不要因为怀孕而省略胸片。**

- **Latent TB (潜伏结核, LTBI)** = TST/IGRA 阳 **+ 胸片阴性 + 无症状**（无咳嗽、咯血、盗汗、体重下降）。潜伏者**无传染性**。
- **胸片异常**（如上叶浸润、空洞、纵隔淋巴结肿大）→ **怀疑活动性 TB** → 送**痰培养 + 痰涂片抗酸染色 (sputum culture / AFB smear)** 确诊。

### 4. 治疗

- **潜伏 TB (LTBI) 治疗 = 异烟肼 (isoniazid, INH)**（通常联合维生素 B6 吡哆醇预防神经病变）。
  - 必须**先经胸片确认为潜伏**后才启动。
  - 孕期某些情况下 INH 可**延至产后**再开始，取决于近期暴露/进展风险（近期转阳、HIV、密接高危者倾向孕期即治；低风险可产后启动）。
- **活动 TB 治疗 = 多药联合 (RIPE：rifampin 利福平 / isoniazid 异烟肼 / pyrazinamide 吡嗪酰胺 / ethambutol 乙胺丁醇)**。
  - 注：孕期 PZA 使用因地区指南而异；活动结核的治疗获益远大于风险，需专科管理。

### 5. IGRA / TST 与 BCG 卡介苗的坑

> [!warning] BCG 干扰
> - **TST 会被 BCG 卡介苗假阳性干扰**（打过卡介苗的人 TST 可阳性，难分真感染还是疫苗反应）。
> - **IGRA 不受 BCG 影响**——IGRA 阳性提示真实分枝杆菌暴露。
> - 因此对**有 BCG 接种史**（很多来自高发国家的移民都打过）的人，IGRA 更可靠。

> [!success] 已 IGRA 阳性者
> 若患者**已经 IGRA 阳性**，下一步就是**胸片**，**无需**再换用 TST 复测，也**无需产后重复筛查**——怀孕本身不会导致 IGRA 假阳性，结果依然可信。

### 6. 流程速览（ASCII tree）

```
高危孕妇 → TST 或 IGRA 筛查
                |
         +------+------+
       阴性           阳性
        |              |
     无需处理    胸片 CXR (加腹部铅屏蔽)
                       |
              +--------+---------+
          胸片阴性 + 无症状     胸片异常 (疑活动)
              |                    |
         潜伏 TB (LTBI)        痰培养 + 抗酸涂片
              |                    |
          异烟肼 INH          确诊活动 TB
       (±延至产后，看风险)          |
                              多药 RIPE 联合治疗
```

## 二、附：常规产前感染筛查清单 (Routine Prenatal Screening)

### 初诊 (First prenatal visit)

- **HIV**：所有孕妇 opt-out 常规筛查。
- **HBV**：查 **HBsAg (乙肝表面抗原)**，阳性者新生儿需 HBIG + 疫苗。
- **梅毒 Syphilis**：**RPR 或 VDRL** 非特异性试验筛查。
- **风疹 Rubella**：查免疫力 (immunity/IgG)，非免疫者产后补种（孕期不打活疫苗）。
- **淋球菌 / 衣原体 Gonorrhea / Chlamydia**：高危孕妇或 **年龄 <25 岁** 者查核酸 (NAAT)。
- **无症状菌尿 Asymptomatic bacteriuria**：**尿培养 (urine culture)** 筛查并治疗，防肾盂肾炎/早产。

### 孕早期或按指征

- **HCV (丙肝)**：现指南倾向每次妊娠常规筛查（各版本略有差异，高危必查）。

### 24–28 周

- **GDM 妊娠期糖尿病**：糖耐量试验 (glucose tolerance test)。（非感染，但属常规产检项，一并列出方便记忆。）

### 36–37 周

- **GBS (B 族链球菌, Group B Streptococcus)**：**直肠阴道拭子 (rectovaginal swab)** 培养，阳性者产时用青霉素预防新生儿早发型 GBS 感染。

## 三、自测题

> [!question]- Q1. 一名高危孕妇 IGRA 阳性、无症状，下一步最恰当的处理是什么？
> **胸片 (chest X-ray)，加腹部铅屏蔽**，用于区分潜伏与活动性 TB。
> - 胸片阴性 + 无症状 = 潜伏 TB → 异烟肼 (可视风险延至产后)。
> - 胸片异常 = 疑活动 → 痰培养确诊 → 多药治疗。
> - **不要**因为怀孕而省略胸片，也**不需要**改用 TST 复测或产后再筛。

> [!question]- Q2. 患者有 BCG 卡介苗接种史，为什么优选 IGRA 而非 TST？若 IGRA 已阳性还需要复查吗？
> **TST 会被 BCG 假阳性干扰**，难以区分真感染与疫苗反应；**IGRA 不受 BCG 影响**，阳性提示真实分枝杆菌暴露，故对有 BCG 史者更可靠。
> IGRA 已阳性则**无需换 TST 复测、无需产后重复**——怀孕不会造成 IGRA 假阳性，结果可信，直接进入胸片评估。

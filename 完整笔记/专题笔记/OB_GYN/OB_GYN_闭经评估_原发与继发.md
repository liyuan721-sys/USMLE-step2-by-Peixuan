# 闭经评估：原发与继发 (Amenorrhea — Primary & Secondary)

> [!tip] 一句话核心
> **原发闭经 (primary amenorrhea, 从未初潮)** 考的是**"何时该启动评估"**——「13 看发育，15 看月经」。**继发闭经 (secondary amenorrhea, 曾有月经、现停经 ≥3 个月)** 考的是**"分步算法"**——**β-hCG → TSH / 泌乳素 (prolactin) → 孕激素撤退试验 (progestin challenge)**，重点常落在**如何解读撤退出血**。

## 一、原发闭经 (Primary Amenorrhea) — 何时启动评估

> [!tip] 小节核心
> 正式评估阈值只有两条：**15 岁仍无月经但已有第二性征**，或 **13 岁仍无任何第二性征**。未达阈值（如 13 岁已有 Tanner 2 发育）→ 观察随访，不过度检查。

### 1. 两条评估触发线（必须背死）

> [!info] 何时启动 workup
> ```text
> 判断闭经时机
>   -> 13 岁「无」第二性征 (secondary sexual characteristics)          → 启动评估
>   -> 15 岁「有」第二性征但仍无初潮 (menarche)                        → 启动评估
>   -> 13 岁「已有」青春期发育（如 Tanner 2）、无其他异常              → 观察 + 1 年后复查
> ```

- 记忆口诀：**「13 看发育，15 看月经」**。
- 13 岁重点问：青春期**启动了没有**（有无乳房发育/阴毛）。
- 家属焦虑 ≠ 病理状态；已有第二性征说明轴在运转，别太早过度检查。

### 2. 场景 → 下一步速查

| 场景 | 下一步 |
|---|---|
| 13 岁「无」第二性征 | 启动原发闭经评估 |
| 15 岁「有」第二性征但无月经 | 启动原发闭经评估 |
| 13 岁已有 Tanner 2 发育、无其他异常 | 观察，1 年后复查 (reexamination in 1 year) |
| 已达评估时点 | 常从盆腔超声 (pelvic ultrasound) ± FSH/LH 起步 |

> [!warning] 高频陷阱
> - 最强干扰项常是 **经阴道超声 (transvaginal ultrasonography)** 或核型 (karyotype)——它们是「已到评估时点」后的检查，**先判断该不该启动 workup**，再谈查什么。
> - 只盯「没来月经」而忽略**年龄 + 青春期阶段**，是本类题最大误区。

### 3. 启动评估后的分流（衔接）

> [!info] 达阈值后：先看第二性征 + 子宫，再看 FSH/LH
> - 有子宫 + 无第二性征 + **FSH/LH 升高 (hypergonadotropic hypogonadism, 高促性腺激素性性腺功能减退)** → 性腺发育不全，查核型（如 Turner 综合征 (Turner syndrome)）。
> - 无子宫 → 查睾酮 + 核型（Müllerian agenesis / 雄激素不敏感 (androgen insensitivity)）。
> - 详见 [[完整笔记/专题笔记/内分泌/内分泌_Turner综合征]]。

## 二、继发闭经 (Secondary Amenorrhea) — 分步诊断

> [!tip] 小节核心
> 继发闭经 = 既往有月经、现停经 ≥ 3 个月（或原本规则者停经 ≥ 3 周期）。固定算法：**先排妊娠 (hCG) → 查 TSH / 泌乳素 → 孕激素撤退试验**。核心考点常落在**撤退出血怎么解读**。

### 1. 分步诊断树

```text
继发性闭经（既往有月经，现停经）
   │
   ① β-hCG（先排妊娠 Pregnancy）
   │     └─ 阳性 ──> 妊娠
   │     └─ 阴性 ↓
   │
   ② TSH + 泌乳素 Prolactin
   │     └─ TSH 异常 ──> 甲状腺功能异常相关闭经
   │     └─ 泌乳素 ↑ ──> 高泌乳素血症 / 泌乳素瘤 Prolactinoma（查垂体 MRI）
   │     └─ 均正常 ↓
   │
   ③ 孕激素撤退试验 Progestin challenge
   │     └─ 有撤退出血 ──> 体内有雌激素 + 流出道通畅 ──> 无排卵 Anovulation
   │     └─ 无撤退出血 ↓
   │
   ④ 雌孕激素联合试验 Estrogen-progestin challenge
         └─ 有出血 ──> 低雌激素状态（查 FSH / LH 定位卵巢 vs 中枢）
         └─ 无出血 ──> 流出道 / 子宫内膜问题（Asherman 综合征等）
```

### 2. 孕激素撤退试验 (Progestin Challenge) 解读（重点）

> [!success] Core Concept
> 撤退出血 (withdrawal bleeding) 一次性回答两件事：**① 子宫内膜已被内源性雌激素预先刺激（有雌激素）；② 流出道 (outflow tract) 通畅**。此前提下，闭经最可能源于**未能周期性排卵 = 无排卵 (anovulation, 无排卵)**。

生理逻辑：

```text
endogenous estrogen present
   -> endometrium proliferates（内膜增生）
   -> give progestin（给孕激素）
   -> withdraw progestin（停孕激素）
   -> withdrawal bleeding occurs（出现撤退出血）
   -> outflow tract is patent（流出道通畅）
   -> problem is failure to ovulate cyclically（问题在无周期性排卵）
```

结果对照表：

| 试验结果 | 意义 |
|---|---|
| 有撤退出血 (withdrawal bleeding present) | 有雌激素 + 流出道通畅 → 指向无排卵 |
| 无撤退出血 (no withdrawal bleeding) | 低雌激素状态，或流出道 / 内膜梗阻 |
| 泌乳素升高 (high prolactin) | 考虑泌乳素瘤 / 高泌乳素血症 |
| TSH 异常 (abnormal TSH) | 甲状腺功能异常相关闭经 |

### 3. 鉴别击破（易混陷阱）

> [!warning] 无排卵 Anovulation vs 高雄激素 Hyperandrogenism——最强干扰
> 多囊卵巢综合征 (PCOS, 多囊卵巢综合征) 确实以无排卵为机制，但**这类题考的是"机制层"而非"表型标签"**：
> - 若无多毛 (hirsutism)、痤疮 (acne)、肥胖等雄激素过多表型 → 答**无排卵 (anovulation)** 而非高雄激素。

> [!info] 其他干扰项
> - **高泌乳素血症 Hyperprolactinemia**：泌乳素正常即不支持。
> - **低促性腺激素性性腺功能减退 Hypogonadotropic hypogonadism（如 Kallmann 综合征 / 卡尔曼综合征）**：更偏**原发性**闭经、青春期发育问题，常伴嗅觉丧失 (anosmia)。
> - **流出道梗阻 Outflow tract obstruction**：若真梗阻，孕激素撤退后**不会**顺利出血。
> - **妊娠 Pregnancy**：任何闭经第一步都要想到，但若无性活跃背景且后续试验更支持无排卵则移后。

### 4. Memory Hook

> [!tip] 记忆钩
> **"撤退能出血，说明路通且有雌激素。"**
> - bleed = tract open（有出血 = 流出道通）
> - bleed = estrogenized endometrium（有出血 = 内膜被雌激素预热）
> - 此时最该想：排卵时序出了问题（无排卵）。

## 三、原发 vs 继发 速对照

| 维度 | 原发闭经 (Primary) | 继发闭经 (Secondary) |
|---|---|---|
| 定义 | 从未初潮（15 有性征无月经 / 13 无性征）| 曾有月经、现停经 ≥3 个月 |
| 第一问 | **该不该启动评估**（年龄 + 青春期阶段）| **排妊娠**（β-hCG）|
| 核心工具 | 盆腔超声 + FSH/LH + 核型 | TSH / 泌乳素 → 孕激素撤退试验 |
| 常见病因 | Turner、Müllerian agenesis、雄激素不敏感、体质性延迟 | 无排卵 (PCOS)、高泌乳素、甲状腺、下丘脑性、Asherman |
| 高频陷阱 | 未达阈值就过度检查（核型/阴超）| 见撤退出血直接贴 PCOS 标签，而非答"无排卵"机制层 |

## 四、复习自测

> [!question]- Q1. 13 岁女孩从未月经，乳房与阴毛 Tanner 2，身高体重正常百分位，母亲担忧。下一步？
> **1 年后复查 (reexamination in 1 year)**。已有第二性征说明青春期已启动，13 岁未初潮属正常变异，未达评估阈值；盆腔检查、核型、雌二醇测定都过早。

> [!question]- Q2. 何时才对无月经者启动原发闭经评估？
> 15 岁有第二性征仍无初潮，**或** 13 岁完全无第二性征。二者任一即启动。

> [!question]- Q3. 既往 28 天规则月经、现停经 10 个月；β-hCG 阴性，TSH、泌乳素正常，无多毛；孕激素撤退试验后出现撤退出血。最可能诊断？
> **无排卵 Anovulation**。关键：撤退出血 = 有雌激素 + 流出道通畅；已排妊娠 / 甲状腺 / 泌乳素，且无雄激素过多表型 → 机制层答无排卵（而非直接贴 PCOS 标签）。

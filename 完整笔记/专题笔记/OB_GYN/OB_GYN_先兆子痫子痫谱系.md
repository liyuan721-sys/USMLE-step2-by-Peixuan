# 先兆子痫 → 子痫谱系 (Preeclampsia–Eclampsia Spectrum)

> [!tip] 一句话核心
> 一条主线：**异常胎盘 → 母体血管内皮损伤**。由此展开——**诊断**（≥20 周新发高血压 + 蛋白尿）；**严重特征**（严重高血压 / 血小板↓ / 肝肾损 / 肺水肿 / 脑视觉症状）；**并发症**（内皮损伤下游：急性缺血性卒中、HELLP、胎盘早剥）；**子痫**（+ 新发抽搐 → 硫酸镁）；**根治**（分娩去除胎盘）。别把"孕妇脑事件"反射到蛛网膜下腔出血，也别把"孕妇抽搐"反射到查 EEG。

> [!info] 缩写全称
> - **SBP / DBP** = Systolic / Diastolic Blood Pressure（收缩压 / 舒张压）
> - **PRES** = Posterior Reversible Encephalopathy Syndrome（后部可逆性脑病综合征）
> - **SAH** = Subarachnoid Hemorrhage（蛛网膜下腔出血）
> - **HELLP** = Hemolysis, Elevated Liver enzymes, Low Platelets（溶血 + 肝酶升高 + 血小板低）
> - **MgSO₄** = Magnesium Sulfate（硫酸镁）

## 一、诊断与严重特征判据

> [!tip] 小节核心
> **先兆子痫 (preeclampsia)** = **≥20 周新发高血压 (SBP ≥140 或 DBP ≥90，间隔 4h 两次)** + **蛋白尿或其他终末器官损害征象**。在此基础上出现下表任一"严重特征"= **伴严重特征 (with severe features)**，处理升级。

### 1. 诊断门槛

| 项 | 阈值 |
|---|---|
| 起病孕周 | **≥20 周**（<20 周高血压多为慢性高血压 / 葡萄胎）|
| 血压 | SBP ≥140 或 DBP ≥90（间隔 ≥4h 两次）|
| 加上 | **蛋白尿** 或 终末器官损害征象（肝 / 肾 / 血小板 / 脑）|

### 2. 严重特征判据 ⭐（背熟——决定处理）

| 类别 | 阈值 |
|---|---|
| **严重高血压** | SBP ≥160 或 DBP ≥110 |
| **血小板减少** | <100,000/mm³ |
| **肾损伤** | 肌酐 >1.1 mg/dL 或 较基线翻倍 |
| **肝损伤** | 转氨酶 >2× 正常上限（常伴上腹 / 右上腹痛）|
| **肺水肿** | 新发 |
| **脑 / 视觉症状** | 严重头痛、视物模糊 / 暗点 (scotoma)、反射亢进 |

> [!danger] 风险因素（stem 背景钩子）
> 初产 (nulliparity)、肥胖、既往疾病（SLE、慢性高血压）、多胎妊娠、高龄产妇。

## 二、病理主线：内皮损伤 → 多器官

> [!info] 一条机制串起所有并发症
> **异常胎盘形成 + 螺旋动脉重塑不足 → 胎盘灌注不足 / 缺氧 / 缺血 → 释放抗血管生成因子 (antiangiogenic factors) → 母体全身血管内皮细胞损伤 (endothelial cell damage)。**
> 内皮损伤是把"高血压 / 蛋白尿 / 肝痛 / 卒中 / HELLP"全部连起来的钥匙：

```text
异常胎盘 → 抗血管生成因子 → 血管内皮损伤
   ├─ 肝：肝包膜牵张 → 上腹 / RUQ 痛、转氨酶↑
   ├─ 肾：滤过膜损伤 → 蛋白尿、肌酐↑
   ├─ 血液：血小板消耗 + 微血管病性溶血 → HELLP
   ├─ 脑：血流失调 + 血管痉挛 + 微血栓 → 卒中 / PRES / 抽搐
   └─ 全身：血管通透性↑ → 水肿、肺水肿
```

## 三、并发症：急性卒中（缺血性，非 SAH）⭐

> [!danger] 小节核心（Q18534 陷阱）
> 先兆子痫伴严重特征**增加急性卒中风险**，机制是内皮损伤下游：
> - **缺血性卒中 (ischemic stroke)**：凝血系统激活 + 血小板聚集 + **血管微血栓 (vascular microthrombi)** → 脑血管闭塞。
> - **出血性卒中 (hemorrhagic stroke)**：脑血流失调 + 血管痉挛 + 灌注压骤升 → 脑实质血管破裂。

> [!warning] 别反射到蛛网膜下腔出血 (SAH)
> **SAH = 破裂的浆果样 (berry) 颅内动脉瘤**，危险因素是**吸烟 + 慢性高血压 + 结缔组织病**——先兆子痫**不是** SAH 的危险因素。孕妇 + 脑事件 ≠ SAH；先兆子痫相关是**缺血性 + 脑实质出血性**卒中。
> → 降低卒中风险的关键：**严重高血压 (SBP ≥160 或 DBP ≥110) 及时降压**（拉贝洛尔 labetalol / 肼屈嗪 hydralazine / 口服硝苯地平 nifedipine）。

> [!question]- 37 周 G1P0，剧烈上腹痛，BP 168/94，尿蛋白 3+，RUQ 超声正常。最可能增加哪种并发症风险？（干扰：SAH / 上行胆管炎 / Boerhaave / 羊膜腔感染）
> **急性缺血性卒中 (acute ischemic stroke)**。
> - 严重高血压 + 蛋白尿 + 上腹痛（肝包膜牵张）= 先兆子痫伴严重特征 → 内皮损伤 + 微血栓 → 缺血性卒中。
> - SAH 是动脉瘤破裂（吸烟 / 慢性高血压），先兆子痫非其危险因素；上腹痛 + RUQ 超声正常排除胆管炎；无呕吐排除 Boerhaave；无流液排除羊膜腔感染。

## 四、子痫 (Eclampsia)：先兆子痫 + 新发抽搐 ⭐

> [!danger] 小节核心（Q20742 陷阱）
> **子痫 = 先兆子痫（常伴严重特征）+ 新发全身强直阵挛抽搐 (generalized tonic-clonic seizure)**，无其他病因。抽搐后**发作后状态 (postictal state)** 解释"倒地无反应 → 逐渐清醒"（10-20 分钟恢复），但仍有先兆子痫征象（头痛、反射亢进、肌酐↑）。

| 项 | 内容 |
|---|---|
| **临床** | 高血压 + 抽搐（常短 <1 min）+ 发作后状态 + 严重头痛 + 视觉症状 + 反射亢进 + 蛋白尿 |
| **诊断** | 主要**临床**；头 CT 常见**双侧额 / 枕叶水肿**（PRES 样，非出血 / 占位）|
| **处理** | **MgSO₄ 静滴**（控制 / 预防抽搐）+ 降压 + **分娩（根治）** |

> [!warning] 别做 EEG / 腰穿
> - **视频 EEG** 用于疑诊**心因性非癫痫发作 (PNES)**；对急性内科病因（子痫）的抽搐做 EEG 是危险的延误。
> - **腰穿**用于查 CNS 感染 / SAH——无发热 / 颈强 / 颅高压时不需要。
> - **曲普坦 (triptan)** 治偏头痛、孕期禁用；偏头痛为单侧搏动、无意识丧失 / 反射亢进 / 额叶水肿。
> - 抗抽搐首选 **MgSO₄**，优于苯妥英 / 地西泮；苯二氮䓬仅二线（→ [[完整笔记/专题笔记/OB_GYN/OB_GYN_硫酸镁临床应用与镁中毒]]）。

> [!question]- 34 周初产妇被发现倒地无反应，渐清醒 + 呕吐 + 严重额部头痛，BP 138/98，DTR 3+，尿蛋白 1+，CT 双侧额叶水肿无出血。下一步？（干扰：CT 静脉造影 / 腰穿 / 曲普坦 / 视频 EEG）
> **硫酸镁静滴 (magnesium sulfate infusion)**。
> - 孕期高血压 + 蛋白尿 + 反射亢进 + 抽搐后状态 + 双侧额叶水肿 = 子痫 → 硫酸镁（+ 降压 + 分娩）。
> - CT 静脉造影查脑静脉窦血栓 (cerebral venous thrombosis, CVT)（视乳头水肿 / 局灶体征，本例无）；腰穿查感染 / SAH（无发热 / 颈强）；EEG 用于 PNES；曲普坦孕期禁用。

## 五、处理总表（按孕周 + 严重程度）

> [!tip] 决策主线
> **去除胎盘 = 根治。** 是否立即分娩取决于孕周 + 有无严重特征。

| 情况 | 处理 |
|---|---|
| **<37 周 + 无严重特征** | 期待治疗 (expectant)，密切监测至 37 周 |
| **≥37 周**（或 ≥34 周伴严重特征）| **分娩 (delivery)** |
| **严重高血压** (SBP ≥160 / DBP ≥110) | 立即降压：静脉拉贝洛尔 (IV labetalol) / 静脉肼屈嗪 (IV hydralazine) / 口服硝苯地平 (PO nifedipine) |
| **抽搐预防 / 治疗（严重特征 / 子痫）** | **MgSO₄ 静滴** |

> [!success] 一句话锁定
> **先兆子痫谱系 = 内皮损伤病：严重特征看 6 条判据；卒中是缺血性非 SAH；抽搐 = 子痫 → 硫酸镁；根治 = 分娩。**

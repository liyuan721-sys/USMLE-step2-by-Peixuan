**【孕期致畸药】**

**一、最常考“致畸王”（看到就联想）**

**1) ACEI/ARB（如 captopril、enalapril、losartan）**

> **2–3孕期**：**胎儿肾发育不全/肾衰 → 少尿 → 羊水过少 → Potter 序列**（肺发育不全、四肢挛缩、面容改变）
>
> **颅骨骨化不良，前囟大且软（hypocalvaria/poor cranial ossification）**
>
> 题眼：新生儿少尿 + 羊水少史 + 大前囟/颅骨薄

**2) 异维A酸（Isotretinoin / 维A酸类）**

> **颅面畸形**（小下颌、唇腭裂）、**心脏畸形**、**CNS异常**
>
> 题眼：痤疮药、IPLEDGE、严重颅面+心脏

**3) 华法林（Warfarin）**

> **鼻软骨发育不良（nasal hypoplasia）**
>
> **点状软骨发育（stippling，chondrodysplasia punctata）**
>
> 还可有出血/中枢出血风险
>
> 题眼：机械瓣/血栓史但用 warfarin

**4) 丙戊酸（Valproate）**

> **神经管缺陷（spina bifida）**
>
> 题眼：癫痫/双相 + 叶酸缺乏线索

**二、抗癫痫药（高频考点）**

**Carbamazepine / Valproate**

> **神经管缺陷**（尤其丙戊酸更典型）
>
> 可能 IUGR、面部异常（与叶酸相关）
>
> 记忆：**“anti-epileptics → folate antagonism → NTD”**（给叶酸）
>
> **叶酸剂量** ⭐：服丙戊酸/卡马西平的育龄女性 → 孕前即需 **4 mg/day 高剂量叶酸**（普通孕妇仅 0.4 mg）。NTD 发生在受孕后 3-4 周（神经管闭合期），等确认怀孕再补已太晚
>
> **stem 关键词**：癫痫/双相孕妇 + **母血 AFP 升高** + 超声见脊柱裂/无脑儿 → Valproate

**Phenytoin（胎儿乙内酰脲综合征）**

> **生长受限、发育迟缓**
>
> **颅面异常**、**指（甲）发育不良**、唇腭裂
>
> 题眼：癫痫用药 + 指甲/面容异常

**三、抗感染药（看到就背）**

**四环素类（Tetracyclines, doxycycline）**

> **牙齿变色**、**骨生长抑制**（孕期整体避免）
>
> 题眼：牙黄/骨发育差

**氨基糖苷类（Aminoglycosides，如 gentamicin）**

> **耳毒性（听力损伤）**
>
> 题眼：孕期重症感染用 aminoglycoside

**磺胺类/甲氧苄啶（TMP-SMX）复方新诺明**

> **1孕期避免**：**TMP🡪叶酸拮抗 → NTD 风险↑**
>
> **临近分娩避免**：SMX🡪**核黄疸**风险（接近分娩/新生儿期更担心）
>
> 题眼：UTI用 TMP-SMX、未补叶酸

**氯霉素（Chloramphenicol）**

新生儿 **gray baby syndrome**（不是畸形，但常考新生儿毒性）

**氟喹诺酮（ciprofloxacin, levofloxacin）**

> 孕期一般避免（软骨/骨毒性担忧；考试常当“禁用”处理）

**Nitrofurantoin**治疗uncomplicated UTI

1,3孕周避免

**Oral fluconazole**治疗vulvovaginal candidiasis

1<sup>st</sup> trimester避免 (miscarriage, birth defects)，改用**intravaginal clotrimazole**

**四、镇痛/抗炎/阿片（更像“功能性并发症”）**

**NSAIDs（如 ibuprofen/indomethacin）**

> **晚孕期（尤其\>30周）**：**动脉导管提前关闭**→ 胎儿期肺血流↑、肺血管重构 → 出生后易**PPHN（Persistent Pulmonary Hypertension of the Newbor/新生儿持续性肺动脉高压）严重低氧，吸氧不能改善**
>
> 可导致**羊水少**（胎儿肾血流↓）

题眼：孕晚期止痛药 + 新生儿呼吸窘迫/PPHN **NSAIDs（ibuprofen, indomethacin）**

**阿片类（opioids）：**新生儿**戒断（NAS）**、呼吸抑制（更偏毒性，不是结构畸形）

**五、内分泌/皮肤/免疫（常被点名）**

**甲硫咪唑（Methimazole）**

> 经典：**胎儿头皮缺损（aplasia cutis）**
>
> 妊娠早期更偏用 **PTU**（因甲硫咪唑致畸风险）

**糖皮质激素（长期高剂量）：**可能唇腭裂风险略增（题目偶尔出现；不如上面高频）

**免疫抑制剂致畸"四杀"（备孕/孕妇换药考点）⭐**

> 致畸必停（停药 → 受孕）：**霉酚酸酯 mycophenolate (MMF)**（唇腭裂 + 小耳畸形 microtia + 流产，停 6 周）｜**甲氨蝶呤 methotrexate (MTX)**（抗叶酸 → NTD，停 ≥3 月，也用于宫外孕/药流）｜**环磷酰胺 cyclophosphamide**（孕早期致畸 + 不孕）｜**来氟米特 leflunomide**（致畸 + 半衰期超长，需消胆胺 cholestyramine 洗脱）。
>
> 孕期**首选替代**：**硫唑嘌呤 azathioprine (AZA)**（胎肝缺活化酶 → 胎儿暴露低，移植/IBD/SLE 孕妇首选）｜**CNI 钙调磷酸酶抑制剂**（他克莫司 tacrolimus / 环孢素 cyclosporine）｜**羟氯喹 hydroxychloroquine (HCQ)**（SLE 孕妇续用）｜泼尼松 prednisone。
>
> 题眼：备孕/孕妇的**移植 / IBD / SLE** 病人在用 MMF 或 MTX → 答案是**换成 AZA / CNI / HCQ**，不是停掉所有免疫抑制（停 = 排异 / flare）。
>
> → 完整机制 + 对照表见 [[完整笔记/专题笔记/免疫/免疫_免疫抑制剂全谱]]（§十一·补 孕期毒性专表）。

**六、情绪药/精神科（USMLE会考“新生儿并发症” + 双相孕期维持）**

**锂（Lithium）**

> **Ebstein anomaly**（三尖瓣下移 → RA扩大）
>
> 题眼：双相障碍 + 新生儿发绀/心杂音
>
> Lithium 维持期可用，但有 Ebstein 风险；**不应在有更低致畸替代（lamotrigine）时主动换成 lithium**

**抗癫痫类 mood stabilizer（双相也用）**

> **Valproate / Carbamazepine / Oxcarbazepine → 神经管缺陷**（anencephaly / myelomeningocele）+ 神经发育障碍
>
> first trimester = organogenesis 关键期，致畸风险最高 → 双相育龄女性应避开

**SSRI（尤其晚孕期）：**

> 新生儿适应不良/戒断样、PPHN 风险讨论（多为并发症，非结构畸形）
>
> **Paroxetine 特殊** ⚠️：早孕期暴露 → **胎儿心脏缺陷**（结构畸形，不只是新生儿并发症）→ 孕期抗抑郁药应首选**非 paroxetine** 的 SSRI

**Lamotrigine（孕期 mood stabilizer 安全选项）⭐**

> **favorable pregnancy safety profile** — 无明确致畸风险增加
>
> 双相孕期**急性抑郁**和**维持期**均可用，维持期可双向保护（防 manic + depressive episodes）

**Bipolar 孕期用药表（Q15103 触发）**

| 阶段 | 药物 |
|---|---|
| 急性躁狂 / 轻躁狂 | Typical（haloperidol）/ Atypical（risperidone, olanzapine）|
| 急性抑郁 | **Lamotrigine** / Quetiapine / Atypical antipsychotic + SSRI（除 paroxetine）|
| 维持期（防复发）| **Lamotrigine** / Atypical（quetiapine, risperidone）/ Lithium |
| 禁忌（致畸）| Valproate / Carbamazepine / Oxcarbazepine（神经管缺陷）/ Paroxetine（心脏缺陷）|

> **换药时序**：致畸 mood stabilizer（valproate 等）患者计划怀孕 → **先延迟怀孕 3-6 个月**做缓慢 cross-taper 换 lamotrigine → 稳定后再尝试怀孕（不是立刻怀孕）。

**精神科致畸药口诀 ⭐：“丙卡奥致管缺，锂帕致心”**

> **丙卡奥致管缺** → 神经管缺陷（Neural Tube Defects, NTD）
>
> **丙** = Valproate（丙戊酸）｜**卡** = Carbamazepine（卡马西平）｜**奥** = Oxcarbazepine（奥卡西平，卡马西平衍生物）
>
> 致畸表现：spina bifida（脊柱裂）、anencephaly（无脑儿）；机制 = 抗叶酸 → 干扰神经管闭合
>
> **锂帕致心** → 心脏畸形（cardiac defects）
>
> **锂** = Lithium → **Ebstein anomaly**（三尖瓣下移畸形）｜**帕** = Paroxetine（SSRI）→ 房/室间隔缺损（VSD/ASD）
>
> Lithium → Ebstein 是 USMLE 超高频：双相孕妇 + 胎儿超声“三尖瓣下移 + 右房扩大 + atrialized RV”（部分右室“心房化”）→ 立即锁定 Lithium

**其它**

**甲氨蝶呤（Methotrexate）**

> **抗叶酸 → 多发畸形/流产风险**（也用于宫外孕治疗）
>
> 题眼：RA/肿瘤/宫外孕药

**沙利度胺（Thalidomide）**

> **肢体畸形（phocomelia）**
>
> 题眼：多发性骨髓瘤/麻风反应

**孕期禁用抗病毒：Ribavirin，**强致畸（特别常在题库里作为禁忌项出现）

**己烯雌酚（Diethylstilbestrol, DES）**

> 宫内暴露 → 女胎青春期后 **阴道/宫颈透明细胞腺癌（clear cell adenocarcinoma）**、子宫畸形（T 形宫腔）、不孕；男胎泌尿生殖畸形
>
> 题眼：母亲 1940s-70s 孕期用药史 + 年轻女性阴道异常出血/透明细胞癌

**酒精（Alcohol）——最常见的致畸物**

> **胎儿酒精综合征（Fetal Alcohol Syndrome, FAS）**：面部异常（睑裂小 short palpebral fissures、人中平滑 smooth philtrum、上唇薄 thin vermilion border）、IUGR、CNS 异常（小头、智力障碍、行为问题）
>
> 无安全饮酒量——孕期酒精暴露是**最常见、且可完全预防**的致畸 + 智力障碍病因

**六·补、其它高频致畸药（2026-06-28 从 Demi RF/Screening/Vaccine 大全增量）**

**他汀类（Statins / HMG-CoA 还原酶抑制剂 HMG-CoA reductase inhibitor）**

> 抑制胎儿**胆固醇合成**（胆固醇是胎儿细胞膜 / 类固醇激素 / 髓鞘原料）→ 孕期**绝对禁用**
>
> 题眼：糖尿病/高脂血症孕妇用药清单里挑"最危险的那个"→ 选他汀；备孕即停

**麦角胺（Ergotamine / 麦角衍生物 ergot derivatives）**

> **强力血管收缩** → 胎盘动脉收缩 → IUGR / 胎死宫内；并可诱发**子宫收缩 → 早产**
>
> 题眼：偏头痛 (migraine) 孕妇用药禁忌
>
> 替代：急性发作 **舒马普坦 (sumatriptan)**；预防（每月≥8 次）**β 受体阻滞剂 / TCA 三环抗抑郁药 / 托吡酯 (topiramate)**（注：topiramate 本身有唇腭裂风险，权衡）

**达那唑（Danazol）**

> **促雄激素 (pro-androgenic)** → **女胎男性化 (virilization)**；男胎**性早熟 (precocious puberty)**
>
> 题眼：子宫内膜异位症 (endometriosis) 治疗中、计划怀孕 → 必须停的药

**苯二氮䓬类（Benzodiazepines）**

> 孕期**禁用**：唇腭裂风险 + 新生儿**松软儿 (floppy baby) / 戒断**
>
> 题眼：**子痫前期 / 子痫 (preeclampsia / eclampsia) 抽搐首选硫酸镁 (magnesium sulfate)**，苯二氮䓬仅二线

**可卡因（Cocaine，物质类）**

> 强拟交感 → 全身血管收缩 → 胎盘缺血 → **胎盘早剥 (placental abruption)** / 胎死宫内 / 不对称 IUGR
>
> 题眼：静脉吸毒孕妇 3 孕期突发锐痛 + 阴道鲜血 → 胎盘早剥（亦见 RF 钥匙：胎盘早剥 MCC = 创伤/可卡因）

**七、最常见“考试句型”总结**

> **“肾发育不全 + 羊水少 + 颅骨骨化差” → ACEI/ARB**
>
> **“鼻发育不良 + 点状软骨” → Warfarin**
>
> **“痤疮药 + 颅面/心/CNS” → Isotretinoin**
>
> **“癫痫药 + 脊柱裂” → Valproate（± carbamazepine）**
>
> **“牙齿变色/骨生长抑制” → Tetracycline**
>
> **“晚孕期NSAID + PPHN/动脉导管关闭” → NSAID**
>
> **“双相孕妇 + 胎儿三尖瓣下移/atrialized RV” → Lithium（Ebstein anomaly）**
>
> **“孕期 SSRI + 胎儿心脏间隔缺损” → Paroxetine**
>
> **“睑裂小 + 人中平滑 + 上唇薄 + IUGR + 小头” → 胎儿酒精综合征（FAS）**
>
> **“年轻女性阴道透明细胞腺癌 + 母亲孕期用药史” → DES**

A-A-W-M-T：ACEI/ARB、(retinoic) A、Warfarin、Methotrexate、Thalidomide”绝对别用；\
“晚孕 NSAID、早孕 TMP、全程 tetracycline/fluoroquinolone/aminoglycoside 尽量避。”

**产科/内分泌常考：哪些“可以用/替代”**

**（考试也常问“不能用A，那用什么”）**

> **高血压：首选 labetalol、nifedipine、methyldopa（不用 ACEI/ARB）**
>
> **甲亢：1孕期偏用 PTU；之后可转 methimazole（因 methimazole 早孕致畸：aplasia cutis 等）**
>
> **抗凝：需要抗凝时用 LMWH/UFH（不用 warfarin）**

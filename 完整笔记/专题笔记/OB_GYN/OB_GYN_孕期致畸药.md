# 孕期致畸药 (Teratogens in Pregnancy)

> [!tip] 一句话核心
> USMLE 的致畸题几乎都是"**特征表现 → 反推药物**"或"**某病孕妇不能用 A → 换什么**"。先锁 **A-A-W-M-T 绝对禁用**（ACEI/ARB、维A酸 retinoic Acid、Warfarin、Methotrexate、Thalidomide），再按器官特征对号入座；第二类高频是"不能用 A，那用什么"的**替代药**。

## 一、最常考"致畸王"（看到就联想）

### 1) ACEI / ARB（captopril、enalapril、losartan）

> [!danger] 致畸表现
> - **2–3 孕期**：胎儿**肾发育不全 / 肾衰 → 少尿 → 羊水过少 (oligohydramnios) → Potter 序列**（肺发育不全、四肢挛缩、面容改变）。
> - **颅骨骨化不良 (poor cranial ossification / hypocalvaria)**：前囟大且软。
> - **题眼**：新生儿少尿 + 羊水少史 + 大前囟 / 颅骨薄。

![[L69724.jpg|360]]

*ACEI/ARB 致畸机制流程：ACE 抑制剂 (ACE inhibitors) / 血管紧张素受体阻滞剂 (ARB, angiotensin II receptor blocker) 阻断 Angiotensin II → 胎儿肾发育不全/发育异常 (fetal renal hypoplasia/dysplasia) → 胎尿↓ → 羊水过少 (oligohydramnios) → 胎儿受压 (fetal compression = Potter 序列)*

### 2) 异维A酸（Isotretinoin / 口服维A酸，商品名 Accutane）

> [!danger] 致畸表现：维A酸胚胎病 (retinoic acid embryopathy)
> 用于**重度结节囊肿性痤疮 (severe nodulocystic acne)**（抗生素无效者）；强致畸剂：
> - **颅面**：小耳/无耳畸形 (microtia/anotia)、小下颌 (micrognathia)、唇腭裂 (cleft palate)
> - **心脏**：圆锥动脉干缺陷 (conotruncal defects，如 TOF 法洛四联症 / 大动脉转位 transposition of the great arteries)
> - **CNS**：脑积水 (hydrocephalus) / 小头畸形 (microcephaly)
> - **胸腺**发育异常；高流产率
> - **题眼**：痤疮药 + iPLEDGE + 严重颅面 + 心脏 + CNS。

> [!warning] iPLEDGE 是什么 ⭐（isotretinoin 强制"防怀孕"关卡）
> **iPLEDGE = FDA 对 isotretinoin 的风险管理程序 (REMS, Risk Evaluation and Mitigation Strategy)**，唯一目的是**防止用药期间怀孕**（因 isotretinoin 强致畸）。育龄女性开药前必须满足：
> - **两种避孕 (2 forms of contraception)**：用药前 **1 个月 + 全程 + 停药后 1 个月**，同时用两种有效避孕（或承诺完全禁欲 abstinence）。
> - **两次孕检阴性 (2 negative β-hCG)**：开药前**两次** β-hCG 阴性；之后**每月一次**验孕才续方，每次只开 30 天量。
> - 医生 / 药房 / 患者三方系统注册。
>
> 记忆 **"2 and 2"**：两种避孕 + 开药前两次验孕（此后每月验）。
>
> ⚠️ 只针对**口服 isotretinoin**；**局部维A酸 (topical retinoids，如 tretinoin)** 治轻中度痤疮，**不走 iPLEDGE**。
>
> **stem 模式**：育龄女性重度痤疮拟开 isotretinoin，问处方前必须做什么 → **两种避孕 + 两次 hCG 阴性**；干扰项 = 只查一次 hCG / 只用一种避孕 / 直接开药。

### 3) 华法林（Warfarin）

> [!danger] 致畸表现：胎儿华法林综合征 (fetal warfarin syndrome)
> - **鼻软骨发育不良 (nasal hypoplasia)**
> - **点状软骨发育 (stippling / chondrodysplasia punctata)**
> - 还可有出血 / 中枢出血风险
> - **题眼**：机械瓣 / 血栓史却用 warfarin（孕期应换 LMWH/UFH）。

### 4) 丙戊酸（Valproate）

> [!danger] 致畸表现
> - **神经管缺陷 (neural tube defect，如 spina bifida 脊柱裂)**
> - **题眼**：癫痫 / 双相 + 叶酸缺乏线索。

## 二、抗癫痫药（高频考点）

### Carbamazepine（卡马西平）/ Valproate（丙戊酸）

> [!danger] 致畸表现
> - **神经管缺陷 (NTD)**（尤其丙戊酸更典型）；可能 **IUGR（宫内生长受限，intrauterine growth restriction）**、面部异常（与叶酸相关）。
> - 记忆：**"anti-epileptics → folate antagonism → NTD"**（给叶酸）。
> - **stem 关键词**：癫痫 / 双相孕妇 + **母血 AFP 升高** + 超声见脊柱裂 / 无脑儿 → Valproate。

> [!tip] 叶酸剂量 ⭐
> 服丙戊酸 / 卡马西平的育龄女性 → **孕前即需 4 mg/day 高剂量叶酸**（普通孕妇仅 0.4 mg）。NTD 发生在受孕后 3–4 周（神经管闭合期），等确认怀孕再补已太晚。

### Phenytoin 苯妥英（胎儿乙内酰脲综合征 Fetal Hydantoin Syndrome）

> [!danger] 致畸表现
> - **生长受限、发育迟缓**
> - **颅面异常**：小头畸形 (microcephaly) + 宽前囟 (wide anterior fontanelle)
> - **远节指骨 / 指甲发育不良 (distal phalanx / nail hypoplasia)**、**唇腭裂 (cleft lip and palate)**、心脏畸形
> - **题眼**：癫痫用药 + 指甲 / 面容异常
> - 机制：抗癫痫药过胎盘 → 低叶酸 + 高氧化代谢产物 → 联合致畸（同类：carbamazepine 卡马西平 / valproate 丙戊酸）。

> [!danger] 胎儿乙内酰脲 vs 胎儿酒精综合征（FAS）——唇腭裂是分水岭 ⭐（Q2572 混淆点）
> 两者都有**小头畸形**，最易混；关键鉴别看**唇腭裂 + 指骨发育不良**：
>
> | 综合征 | 颅面关键特征 | 唇腭裂 | 肢端 |
> |---|---|---|---|
> | **胎儿乙内酰脲 (phenytoin)** | 小头 + **宽前囟** | **有** ✅ | **远节指骨 / 指甲发育不良** |
> | **胎儿酒精 (FAS)** | 小头 + **中面部发育不良**（睑裂短 + 人中平 + 上唇薄）| **无** ❌ | 无 |
>
> 一句话：**唇腭裂 + 远节指骨发育不良 = 乙内酰脲；FAS 是中面部发育不良、无唇腭裂。**
>
> → 对标错题：[[mistakes/uworld-mistakes_2026-07#^Q2572]]（唇腭裂 + 远节指骨发育不良 → phenytoin；做对但标记强化）；FAS 详见本笔记 §六「酒精」段。

![[L46798.jpg|360]]

*Fetal hydantoin syndrome 胎儿乙内酰脲综合征：小头畸形 (microcephaly) + 唇腭裂 (cleft lip & palate) + 心脏缺陷 (cardiac defects) + 指甲/远节指骨发育不良 (hypoplasia of nails & phalanges)。**无唇腭裂**的是 FAS——这是与酒精综合征的鉴别点。双嵌自卡 [[mistakes/uworld-mistakes_2026-07#^Q2572]]*

## 三、抗感染药（看到就背）

> [!danger] 结构性 / 器官毒性（孕期避免）
> - **四环素类 (Tetracyclines, doxycycline)**：牙齿变色、骨生长抑制（孕期整体避免）。题眼：牙黄 / 骨发育差。
> - **氨基糖苷类 (Aminoglycosides, gentamicin)**：耳毒性（听力损伤）。题眼：孕期重症感染用 aminoglycoside。
> - **氟喹诺酮 (fluoroquinolones，ciprofloxacin / levofloxacin)**：软骨 / 骨毒性担忧，孕期一般避免（考试常当"禁用"）。

> [!warning] 时相依赖 / 新生儿毒性
> - **磺胺甲噁唑-甲氧苄啶 (TMP-SMX，复方新诺明)**：**1 孕期**避免——TMP → 叶酸拮抗 → NTD 风险↑；**临近分娩**避免——SMX → 核黄疸 (kernicterus) 风险。题眼：UTI 用 TMP-SMX、未补叶酸。
> - **氯霉素 (Chloramphenicol)**：新生儿 **gray baby syndrome 灰婴综合征**（非畸形，常考新生儿毒性）。
> - **呋喃妥因 (Nitrofurantoin)**：治单纯性 UTI，**1、3 孕期**避免。
> - **口服氟康唑 (oral fluconazole)**：治外阴阴道念珠菌病 (vulvovaginal candidiasis)，**1 孕期**避免（流产 / 出生缺陷）→ 改用 **阴道克霉唑 (intravaginal clotrimazole)**。

## 四、镇痛 / 抗炎 / 阿片（更像"功能性并发症"）

> [!warning] NSAIDs（ibuprofen / indomethacin）
> - **晚孕期（尤其 >30 周）**：**动脉导管提前关闭 (premature closure of ductus arteriosus)** → 胎儿期肺血流↑、肺血管重构 → 出生后易 **PPHN (Persistent Pulmonary Hypertension of the Newborn，新生儿持续性肺动脉高压)**：严重低氧、吸氧不能改善。
> - 可致**羊水过少**（胎儿肾血流↓）。
> - 题眼：孕晚期止痛药 + 新生儿呼吸窘迫 / PPHN。

> [!warning] 阿片类（opioids）
> 新生儿**戒断 (NAS, neonatal abstinence syndrome)**、呼吸抑制（偏毒性，非结构畸形）。

## 五、内分泌 / 皮肤 / 免疫（常被点名）

> [!danger] 甲巯咪唑（Methimazole）
> - 经典：**胎儿头皮缺损 (aplasia cutis)**。
> - 妊娠早期更偏用 **丙硫氧嘧啶 (PTU, propylthiouracil)**（因 methimazole 早孕致畸）。

> [!info] 糖皮质激素（长期高剂量）
> 可能唇腭裂风险略增（偶尔出现；不如上面高频）。

> [!danger] 免疫抑制剂致畸"四杀"（备孕 / 孕妇换药考点）⭐
> **致畸必停（停药 → 受孕）**：
> - **霉酚酸酯 mycophenolate (MMF)**：唇腭裂 + 小耳畸形 (microtia) + 流产，停 6 周
> - **甲氨蝶呤 methotrexate (MTX)**：抗叶酸 → NTD，停 ≥3 月，也用于宫外孕 / 药流
> - **环磷酰胺 cyclophosphamide**：孕早期致畸 + 不孕
> - **来氟米特 leflunomide**：致畸 + 半衰期超长，需消胆胺 (cholestyramine) 洗脱
>
> **孕期首选替代**：
> - **硫唑嘌呤 azathioprine (AZA)**：胎肝缺活化酶 → 胎儿暴露低（移植 / IBD / SLE 孕妇首选）
> - **CNI 钙调磷酸酶抑制剂**：他克莫司 tacrolimus / 环孢素 cyclosporine
> - **羟氯喹 hydroxychloroquine (HCQ)**：SLE 孕妇续用；泼尼松 prednisone
>
> **题眼**：备孕 / 孕妇的**移植 / IBD / SLE** 病人在用 MMF 或 MTX → 答案是**换成 AZA / CNI / HCQ**，不是停掉所有免疫抑制（停 = 排异 / flare）。
>
> → 完整机制 + 对照表见 [[完整笔记/专题笔记/免疫/免疫_免疫抑制剂全谱]]（§十一·补 孕期毒性专表）。

## 六、情绪药 / 精神科（新生儿并发症 + 双相孕期维持）

### 结构畸形 / 高频致畸

> [!danger] 锂（Lithium）
> - **Ebstein anomaly（三尖瓣下移畸形）** → RA 扩大。
> - 题眼：双相障碍 + 新生儿发绀 / 心杂音。
> - Lithium 维持期可用，但有 Ebstein 风险；**不应在有更低致畸替代（lamotrigine）时主动换成 lithium**。

> [!danger] 抗癫痫类 mood stabilizer（双相也用）
> **Valproate / Carbamazepine / Oxcarbazepine → 神经管缺陷**（anencephaly 无脑儿 / myelomeningocele 脊髓脊膜膨出）+ 神经发育障碍。first trimester = organogenesis 关键期，致畸风险最高 → 双相育龄女性应避开。

> [!warning] SSRI（尤其晚孕期）
> - 新生儿适应不良 / 戒断样、PPHN 风险讨论（多为并发症，非结构畸形）。
> - **Paroxetine 特殊** ⚠️：早孕期暴露 → **胎儿心脏缺陷**（结构畸形，不只是新生儿并发症）→ 孕期抗抑郁药应首选**非 paroxetine** 的 SSRI。

### 孕期安全选项与用药策略

> [!success] Lamotrigine（孕期 mood stabilizer 安全选项）⭐
> - **favorable pregnancy safety profile** — 无明确致畸风险增加。
> - 双相孕期**急性抑郁**和**维持期**均可用，维持期可双向保护（防 manic + depressive episodes）。

**Bipolar 孕期用药表（Q15103 触发）**

| 阶段 | 药物 |
|---|---|
| 急性躁狂 / 轻躁狂 | Typical（haloperidol）/ Atypical（risperidone, olanzapine）|
| 急性抑郁 | **Lamotrigine** / Quetiapine / Atypical antipsychotic + SSRI（除 paroxetine）|
| 维持期（防复发）| **Lamotrigine** / Atypical（quetiapine, risperidone）/ Lithium |
| 禁忌（致畸）| Valproate / Carbamazepine / Oxcarbazepine（神经管缺陷）/ Paroxetine（心脏缺陷）|

*↑ UW 原图：Treatment of bipolar disorder during pregnancy（与上方表格同源）。来源 [[mistakes/uworld-mistakes_2026-05#^Q15103]]，图同时存于错题卡 + 本节，双地点 Obsidian 反向链接调取*

> [!tip] 换药时序
> 致畸 mood stabilizer（valproate 等）患者计划怀孕 → **先延迟怀孕 3–6 个月**做缓慢 cross-taper 换 lamotrigine → 稳定后再尝试怀孕（不是立刻怀孕）。
>
> → 对标错题：[[mistakes/uworld-mistakes_2026-05#^Q15103]]（孕期换 lamotrigine）/ [[mistakes/uworld-mistakes_2026-05#^Q15841]]（双相稳定不能停 lithium）。

> [!info] 精神科致畸药口诀 ⭐："丙卡奥致管缺，锂帕致心"
> **丙卡奥致管缺** → 神经管缺陷 (NTD)：**丙** = Valproate（丙戊酸）｜**卡** = Carbamazepine（卡马西平）｜**奥** = Oxcarbazepine（奥卡西平）。表现 = spina bifida / anencephaly；机制 = 抗叶酸 → 干扰神经管闭合。
>
> **锂帕致心** → 心脏畸形：**锂** = Lithium → **Ebstein anomaly**（三尖瓣下移）｜**帕** = Paroxetine → 房 / 室间隔缺损 (ASD/VSD)。
>
> Lithium → Ebstein 超高频：双相孕妇 + 胎儿超声"三尖瓣下移 + 右房扩大 + atrialized RV（部分右室心房化）"→ 立即锁定 Lithium。

### 其它高频致畸药

> [!danger] 结构致畸 / 强致畸
> - **甲氨蝶呤 (Methotrexate)**：抗叶酸 → 多发畸形 / 流产（也用于宫外孕治疗）。题眼：RA / 肿瘤 / 宫外孕药。
> - **沙利度胺 (Thalidomide)**：肢体畸形 (phocomelia 海豹肢)。题眼：多发性骨髓瘤 / 麻风反应。
> - **利巴韦林 (Ribavirin)**：强致畸，孕期禁用（题库常作禁忌项）。

> [!danger] 己烯雌酚（Diethylstilbestrol, DES）
> - 宫内暴露 → 女胎青春期后 **阴道 / 宫颈透明细胞腺癌 (clear cell adenocarcinoma)**、子宫畸形（T 形宫腔）、不孕；男胎泌尿生殖畸形。
> - 题眼：母亲 1940s–70s 孕期用药史 + 年轻女性阴道异常出血 / 透明细胞癌。

> [!danger] 酒精（Alcohol）——最常见的致畸物
> - **胎儿酒精综合征 (Fetal Alcohol Syndrome, FAS)**：面部异常（睑裂小 short palpebral fissures、人中平滑 smooth philtrum、上唇薄 thin vermilion border）、IUGR、CNS 异常（小头、智力障碍、行为问题）。
> - 无安全饮酒量——孕期酒精暴露是**最常见、且可完全预防**的致畸 + 智力障碍病因。

![[L14000.jpg|360]]

*Fetal alcohol syndrome 胎儿酒精综合征：小头畸形 (microcephaly) + 眼裂短 (small palpebral fissures) + 中面部发育不良 (midface hypoplasia) + 人中平滑 (smooth philtrum) + 上唇薄 (thin vermilion border)。**无唇腭裂**——与胎儿乙内酰脲综合征 (fetal hydantoin syndrome) 的鉴别钥匙*

## 六·补、其它高频致畸药

> [!info] 2026-06-28 从 Demi RF/Screening/Vaccine 大全增量

> [!danger] 他汀类（Statins / HMG-CoA 还原酶抑制剂 HMG-CoA reductase inhibitor）
> - 抑制胎儿**胆固醇合成**（胆固醇 = 胎儿细胞膜 / 类固醇激素 / 髓鞘原料）→ 孕期**绝对禁用**。
> - 题眼：糖尿病 / 高脂血症孕妇用药清单里挑"最危险的那个"→ 选他汀；备孕即停。

> [!danger] 麦角胺（Ergotamine / 麦角衍生物 ergot derivatives）
> - **强力血管收缩** → 胎盘动脉收缩 → IUGR / 胎死宫内；诱发**子宫收缩 → 早产**。
> - 题眼：偏头痛 (migraine) 孕妇用药禁忌。
> - 替代：急性发作 **舒马普坦 (sumatriptan)**；预防（每月 ≥8 次）**β 受体阻滞剂 / TCA 三环抗抑郁药 / 托吡酯 (topiramate)**（注：topiramate 本身有唇腭裂风险，权衡）。

> [!danger] 达那唑（Danazol）
> - **促雄激素 (pro-androgenic)** → **女胎男性化 (virilization)**；男胎**性早熟 (precocious puberty)**。
> - 题眼：子宫内膜异位症 (endometriosis) 治疗中、计划怀孕 → 必须停的药。

> [!warning] 苯二氮䓬类（Benzodiazepines）
> - 孕期**禁用**：唇腭裂风险 + 新生儿**松软儿 (floppy baby) / 戒断**。
> - 题眼：**子痫前期 / 子痫 (preeclampsia / eclampsia) 抽搐首选硫酸镁 (magnesium sulfate)**，苯二氮䓬仅二线。

> [!danger] 可卡因（Cocaine，物质类）
> - 强拟交感 → 全身血管收缩 → 胎盘缺血 → **胎盘早剥 (placental abruption)** / 胎死宫内 / 不对称 IUGR。
> - 题眼：吸毒孕妇 3 孕期突发锐痛 + 阴道鲜血 → 胎盘早剥（钥匙：胎盘早剥 MCC = 创伤 / 可卡因）。

## 七、最常考"考试句型"总结

> [!success] 特征 → 药物（反射级）
> - **"肾发育不全 + 羊水少 + 颅骨骨化差" → ACEI/ARB**
> - **"鼻发育不良 + 点状软骨" → Warfarin**
> - **"痤疮药 + 颅面 / 心 / CNS" → Isotretinoin**
> - **"癫痫药 + 脊柱裂" → Valproate（± carbamazepine）**
> - **"牙齿变色 / 骨生长抑制" → Tetracycline**
> - **"晚孕期 NSAID + PPHN / 动脉导管关闭" → NSAID**
> - **"双相孕妇 + 胎儿三尖瓣下移 / atrialized RV" → Lithium（Ebstein anomaly）**
> - **"孕期 SSRI + 胎儿心脏间隔缺损" → Paroxetine**
> - **"睑裂小 + 人中平滑 + 上唇薄 + IUGR + 小头" → 胎儿酒精综合征（FAS）**
> - **"年轻女性阴道透明细胞腺癌 + 母亲孕期用药史" → DES**

> [!danger] 绝对别用 / 尽量避（记忆锚）
> **A-A-W-M-T 绝对别用**：ACEI/ARB、(retinoic) A、Warfarin、Methotrexate、Thalidomide。
> **尽量避**：晚孕 NSAID、早孕 TMP、全程 tetracycline / fluoroquinolone / aminoglycoside。

> [!tip] "不能用 A，那用什么"——常考替代
> - **高血压**：首选 labetalol、nifedipine、methyldopa（不用 ACEI/ARB）。
> - **甲亢**：1 孕期偏用 PTU；之后可转 methimazole（因 methimazole 早孕致畸：aplasia cutis 等）。
> - **抗凝**：需抗凝时用 LMWH / UFH（不用 warfarin）。

# OB/GYN — 妇科手术并发症速查（Hysterectomy / 盆腔手术）

> [!info] 笔记定位
> 妇科盆腔手术（hysterectomy / oophorectomy / cesarean）术后并发症是 USMLE Step 2 GYN/外科高频。本笔记按"4 大常见并发症 × 临床/Cr/UA/治疗"组织，含 Ureter 3 大易伤位解剖图 + "Water under the bridge" 经典口诀 + 术后腹胀 5 鉴别决策树。来源：[[mistakes/uworld-mistakes#^Q16241]]（错题 2026-05-28，体系早建 v1）。

---

## Part 1. Ureter 3 大易伤位 ⭐⭐⭐（解剖学基础）

![[{3FE27EEC-055B-4038-9BE5-EBBE035299CA}.png]]
*↑ UWorld 官方 Ureter injury during pelvic surgery 解剖图（Posterior + Anterior 视图）— 红圈 = common site of injury*

| 位置 | 关联 ligament / 血管 | 手术情境 |
|---|---|---|
| **① 卵巢血管旁** | **Infundibulopelvic (suspensory) ligament**（含 ovarian artery + vein）| Salpingo-oophorectomy / ovarian cystectomy / hysterectomy 切 IPL 时 |
| **② 子宫动脉旁** ⭐ **最常见** | **Cardinal ligament**（含 uterine vessels）| **Hysterectomy 钳夹 uterine artery** 时 |
| **③ 入 bladder 前** | bladder trigone 区 | Anterior colporrhaphy / vaginal hysterectomy |

> [!success] "Water under the bridge" 口诀 ⭐⭐⭐
> **Ureter (water) 从 uterine artery (bridge) 下方 ~2 cm 穿过** —— 钳夹 uterine artery 时若钳子太深易夹 ureter。这是 **hysterectomy 最常 ureter 损伤位**。

---

## Part 2. 妇科术后并发症 4 大对照 ⭐⭐

| 并发症 | 时间点 | 临床 | Cr / 影像 | 处理 |
|---|---|---|---|---|
| **Uroperitoneum (Ureter laceration)** ⭐ | 术后 **days - 2 周** | 弥漫腹胀 + watery 阴道排液 + 低热 + 大量腹腔积液无回声 | **Cr / UA / 排尿全正常**（单侧对侧代偿）⭐ | **CT urography** → 外科修复 / stent |
| **Bladder injury** | 术中 / 即刻术后 | 血尿 + 低 UO + 腹胀 | UA 血 + Cr 可↑（双侧损） | 膀胱镜 + 修复 |
| **Vesicovaginal / Ureterovaginal fistula** | 术后 **2-4 周** | **持续 watery 阴道漏尿** + 无尿失禁感 | Tampon test + cystoscopy + IV indigo carmine | 外科修复（多需 3-6 月愈） |
| **Vaginal cuff dehiscence** | 术后 **6+ 周**（性交 / 重体力） | **突发**剧痛 + 肠外溢 + cuff 触诊裂开 | 阴道检查 cuff 开放 | **急诊手术修复** + 抗生素 |

---

## Part 3. Uroperitoneum 5 联指纹 ⭐（Q16241 核心）

1. **术后 days-2 周** 出现（不一定术中识破）
2. **弥漫腹胀腹痛 + 大量腹腔积液**（US/CT 见 no internal echoes 的 free fluid）
3. **Watery 阴道排液**（尿溢腹腔→未完愈合 vaginal cuff 渗出 — **cuff "closed" 也不矛盾**，cuff 缝合但未完全愈合）
4. **低热 + 恶心**（尿化学刺激腹膜非感染性）
5. **Cr / UA / 排尿全正常**（单侧损伤 + 对侧肾代偿）⭐⭐⭐ **USMLE 最大陷阱**

### 高危红旗组合 ⭐
- **既往盆腔手术 / 剖宫产 / endometriosis** → 解剖扭曲 + 粘连
- **大子宫 / 大附件肿块** → 解剖标志移位
- **急诊紧急手术** / 大出血操作匆忙

### 诊断 + 治疗
- **金标准影像**：**CT urography**（IV 造影 + 延迟相，见造影剂溢入腹腔）
- **替代**：retrograde pyelogram / IVP
- **治疗**：**外科修复**（ureteral reimplantation / end-to-end anastomosis / ureteral stent）

## Part 4. 术后腹胀 5 鉴别决策树 ⭐⭐

```
术后 + 腹胀
    │
    ├─ Hb / BP 异常（休克 + Hb↓）
    │      → Hemoperitoneum（出血）→ 紧急 CT + 复苏 + 手术
    │
    ├─ 高热 39+ / WBC↑↑ / 局限 tenderness / 包裹积液
    │      → Intraabdominal abscess → CT + drainage + 抗生素
    │
    ├─ 腹胀 + 肠鸣高调 + 呕吐 + 停止排气排便
    │      → Small bowel obstruction (SBO) → NG decompression + IVF + 看是否需手术
    │
    ├─ 弥漫积液 + watery 阴道排液 + 低热 + Cr 正常 ⭐
    │      → Uroperitoneum (Ureter laceration) → CT urography + 外科修复
    │
    └─ 术后 6+ 周突发剧痛 + 阴道 cuff 裂开 / 肠外溢
           → Vaginal cuff dehiscence → 急诊手术 + 抗生素
```

### 关键鉴别开关
- **Passing flatus 是否存在**：存在 → 排 SBO（肠通）/ 不存在 → SBO 高
- **阴道排液性质**：watery → 尿（uroperitoneum / fistula）/ purulent → 脓肿 / 血性 → hemoperitoneum-cuff dehiscence
- **Cr 是否升高**：单侧 ureter 损 Cr 正常（对侧代偿）/ 双侧损或 fistula 累及双侧 Cr↑
- **vaginal cuff 触诊**：closed 排 dehiscence / open 锁 dehiscence

## Part 5. 易混陷阱（普适）⭐

| 选项 | 适用场景 | 不适用 uroperitoneum |
|---|---|---|
| **Hemoperitoneum** | Hb↓ + BP↓ + 失血征 + 内部回声 | Hb 正常 + BP 正常 ❌ |
| **Intraabdominal abscess** | 高热 39+ + WBC↑↑ + 局限 tenderness + 包裹液 | 低热 38 + WBC 10K + 弥漫软腹 ❌ |
| **Small bowel obstruction (SBO)** | 腹胀 + 肠鸣高调 + 呕吐 + **停止排气排便** | **Passing flatus** + 腹软 + watery 排液 ❌ |
| **Uroperitoneum (Ureter laceration)** ⭐ | Hysterectomy + endometriosis 扭曲 + 弥漫积液 + watery 排液 + Cr 正常 | ✅ |
| **Vaginal cuff dehiscence** | 术后 6+ 周突发剧痛 + cuff 裂开 | **Cuff "closed"** ❌；时间早 ❌ |

### 反 pattern 警示
- **术后 GI 症状 ≠ 自动 SBO** — 先扫 "passing flatus / 阴道排液 / 腹软无 tender" → 转向 uroperitoneum
- **Cr / UA 正常不排 ureter 损伤** — 单侧损伤对侧肾代偿 ⭐⭐⭐ USMLE 最大陷阱
- **vaginal cuff "closed" ≠ 完全愈合** — 仍可让尿渗出
- **Endometriosis / 既往盆腔手术 / 剖宫产** = ureter 高危红旗组合
- **CT urography 是金标准**（不是普通 CT — 需 IV 造影 + 延迟相）

## Part 6. Memory Hook

> **"Hysterectomy 术后 watery 阴道排液 + 大量腹腔积液 + Cr 正常 = 单侧 ureter laceration (uroperitoneum)"** ⭐
>
> **"Water under the bridge"** — Ureter 在 uterine artery 下方 2 cm 穿过 ⭐⭐⭐
>
> **Ureter 3 易伤位**：IPL（卵巢血管）/ Cardinal（uterine artery）/ 入 bladder 前
>
> **单侧 → Cr/UA/排尿全正常**（USMLE 最大陷阱）⭐⭐
>
> **高危红旗组合**：Endometriosis + 既往盆腔手术 + 大子宫 / 大附件
>
> **术后腹胀 5 鉴别开关**：passing flatus（排 SBO）/ 阴道排液性质（watery=尿）/ Cr / cuff 触诊

# 早产管理：孕周分层与药物 (Preterm Labor Management)

> [!tip] 一句话核心
> Preterm labor (早产) 管理**先按孕周 (gestational age) 分层**。诊断先确认：**规律宫缩 + cervical change (宫颈变化：effacement/dilation)** 才算真早产，不是 Braxton Hicks。药物选择的考点不是"哪个药能让子宫收缩"，而是"**哪个药在这个孕周最该上、考试价值最高**"。

---

## 一、诊断早产

> [!info] 早产 = 宫缩 + 宫颈变化
> - **规律宫缩**（如每 5 分钟一次、持续数小时）——排除偶发 Braxton Hicks (假宫缩)
> - **cervical change**：partial effacement (部分消退) 或 dilation (扩张) —— 这一步把"有宫缩"升级为真正的 preterm labor
> - 既往早产史 (prior preterm delivery) = 强复发风险因素

---

## 二、孕周分层用药（核心考点）⭐

```text
确认 preterm labor（宫缩 + 宫颈变化）
  │
  ├─ < 37 周 ── 据情况 tocolysis (保胎) 延迟分娩
  │             + GBS 预防（B 组链球菌，据筛查/风险）
  │
  ├─ < 34 周 ── + betamethasone (倍他米松)
  │             促胎肺成熟 (fetal lung maturity)，降低 RDS (respiratory distress syndrome, 新生儿呼吸窘迫综合征) / NEC (necrotizing enterocolitis, 坏死性小肠结肠炎) / IVH (intraventricular hemorrhage, 脑室内出血)
  │
  └─ < 32 周 ── + magnesium sulfate (硫酸镁)
                fetal neuroprotection (胎儿神经保护，降低脑瘫)
                （同时有弱 tocolytic 作用）
```

> [!success] 三条硬阈值（务必记准）
> - **< 32 周 → 硫酸镁 magnesium sulfate 胎儿神经保护**
> - **< 34 周 → 倍他米松 betamethasone 促胎肺成熟**（antenatal corticosteroids, 产前糖皮质激素）
> - **< 37 周 → 据情况保胎 (tocolysis) + GBS 预防**

## 三、药物分工与"促宫缩药"黑名单

> [!warning] 保胎时所有"促宫缩药"都要本能排除
> 治疗目标是**延迟分娩**，任何促子宫收缩 / 终止妊娠的药方向相反，一律排除。

| 药物 | 作用 | 早产场景中的角色 |
|---|---|---|
| Magnesium sulfate (硫酸镁) | fetal neuroprotection + 弱 tocolysis | **< 32 周应选** |
| Betamethasone (倍他米松) | 促胎肺成熟 | **< 34 周应选** |
| Nifedipine / Terbutaline / Indomethacin | tocolytics (保胎，延迟分娩争取激素起效) | 保胎主力（据孕周/禁忌选） |
| Oxytocin (缩宫素) | 引产 / 加强产程 | ❌ 促分娩，方向相反 |
| Misoprostol (米索前列醇) | 前列腺素促宫缩 | ❌ 促分娩 |
| Ergonovine (麦角新碱) | 产后子宫收缩止血 | ❌ 会加速早产 |
| Mifepristone (米非司酮) | 终止妊娠 | ❌ 反向操作 |
| Desmopressin (去氨加压素) | DI (diabetes insipidus, 尿崩症) / vWD (von Willebrand disease, 血管性血友病) 等 | 与早产无关 |

> [!info] Tocolytics（保胎药）适应症 —— 据孕周 + 禁忌选
> **总适应症**：确诊 preterm labor（早产临产：规律宫缩 + 宫颈改变），孕 **24 0/7 – 33 6/7 周**，目的**只是延迟分娩 ~48 小时**，为**倍他米松 betamethasone 起效**（± 硫酸镁 magnesium sulfate 神经保护、± 转诊到有 NICU (neonatal intensive care unit, 新生儿重症监护室) 的医院）争取时间。**≥34 周一般不再保胎**（获益不抵风险，早产儿已相对成熟）。
>
> | 药物 | 首选孕周 | 关键禁忌/注意 |
> |---|---|---|
> | **Indomethacin（吲哚美辛，NSAID 非甾体抗炎药）** | **< 32 周首选** | **≥32 周禁用**：致**动脉导管早闭 (premature ductus arteriosus closure)** + **羊水过少 (oligohydramnios)**（胎儿肾灌注↓）；限用 ≤48h |
> | **Nifedipine（硝苯地平，CCB 钙通道阻滞剂）** | **32–34 周首选**（也可更早用） | 母体**低血压 (hypotension)**；与硫酸镁合用需谨慎（叠加低血压/神经肌肉抑制） |
> | **Terbutaline（特布他林，β2 激动剂）** | 仅**急性短期救援**（≤48–72h） | 母体**心脏病/心律失常、未控糖尿病 (hyperglycemia)**；FDA 警告禁长期/维持使用 |
>
> **口诀**：**"早用吲哚(<32)，中用硝苯(32–34)，特布只救急"**。

> [!danger] 什么时候"不该保胎"（tocolysis 禁忌 = 该让她生）
> 保胎的前提是"继续妊娠对母胎有利"。以下情况**继续妊娠反而有害 → 不保胎、应分娩**：
> - **宫内感染 IAI / chorioamnionitis（绒毛膜羊膜炎）** —— 感染源要娩出
> - **胎儿窘迫 (non-reassuring fetal status) / 胎死宫内**
> - **重度子痫前期/子痫 (severe preeclampsia/eclampsia)**
> - **显著胎盘早剥 (placental abruption)**、活动性大出血
> - 产程已进展过深（宫口开大、即将分娩）

## 四、复习自测

> [!question]- 孕 28 周，规律宫缩 4 小时 (每 5 分钟)，宫颈部分消退，既往 28 周早产史。最合适的 pharmacotherapy？
> **Magnesium sulfate (硫酸镁)**。
> - 宫缩 + 宫颈 effacement = 确诊 preterm labor。
> - **< 32 周** → 硫酸镁做 fetal neuroprotection（考试身份），兼弱 tocolysis。
> - **最强干扰项 = Oxytocin**：见"有宫缩"就反射缩宫素，但目标是延迟分娩，方向相反。

> [!question]- 早产阈值三连：< 37 / < 34 / < 32 周分别加什么？
> - **< 37 周** → 据情况保胎 (tocolysis) + GBS 预防
> - **< 34 周** → 倍他米松 betamethasone 促胎肺成熟
> - **< 32 周** → 硫酸镁 magnesium sulfate 神经保护

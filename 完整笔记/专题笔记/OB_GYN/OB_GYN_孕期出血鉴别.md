# 孕期出血鉴别（Bleeding in Pregnancy）

> [!info] 派生与定位
> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q047]]（NBME11 / 妊娠期宫颈癌出血 friable ulcer）。
> 本卡把"孕期任何阶段阴道出血"统一成**按孕周分流 + 痛/无痛分流**的鉴别框架，覆盖早孕期（流产 / 异位 / 葡萄胎）+ 中晚孕期（前置胎盘 / 胎盘早剥 / 前置血管 / 子宫破裂 / 宫颈病变含宫颈癌）。
> 相关：[[完整笔记/专题笔记/OB_GYN/OB_GYN_HighYield速查总览]]（A6 产前出血）/ [[完整笔记/Peixuan分科笔记/OB]] / [[完整笔记/Peixuan分科笔记/GYN]]。

---

## 一、第一分流：孕周（gestational age）

```
孕期阴道出血 (vaginal bleeding in pregnancy)
│
├── 早孕期 (< 20 周)  ────► 想"妊娠去向" + 宫颈/阴道局部
│     ├── 流产 abortion（看宫颈口 开/闭 + 妊娠物）
│     ├── 异位妊娠 ectopic（hCG 不达辨别带宫内仍空 + 附件包块/腹痛）
│     └── 葡萄胎 hydatidiform mole（hCG 爆表 + 子宫大于孕周 + snowstorm）
│
└── 中晚孕期 (≥ 20 周, antepartum hemorrhage) ──► 第二分流：痛 vs 无痛
      ├── 痛  ── 胎盘早剥 abruptio placentae / 子宫破裂 uterine rupture
      └── 无痛 ── 前置胎盘 placenta previa / 前置血管 vasa previa / 宫颈病变（含宫颈癌）
```

> [!danger] 铁律（贯穿全篇）
> **中晚孕期出血，在超声排除前置胎盘 (placenta previa) 之前，绝对禁止指检 (digital cervical examination) / 性交 / 阴道窥器盲探。** 指检可能戳破覆盖宫颈口的胎盘 → 致命大出血。出血孕妇先 **TAUS→TVUS** 看胎盘位置，再决定能否内诊。

## 二、早孕期出血（< 20 周）

### 2.1 流产 (Spontaneous abortion, 自然流产) — 看「宫颈口开/闭 + 妊娠物排出」

> [!info] 五型对照（钥匙 = 宫颈内口状态）

| 类型 | 阴道流血 | 宫颈内口 | 妊娠物排出 | 钩子 / 临床意义 |
|---|---|---|---|---|
| 先兆流产 Threatened | 有 | **闭** | 无 | 胎心通常存在；处理=观察、保胎 |
| 难免流产 Inevitable | 有 | **开** | 无 | 宫缩痛明显，流产不可避免 |
| 不全流产 Incomplete | 有（多） | **开** | 部分 | 宫内残留组织 → 需清宫/药物 |
| 完全流产 Complete | 有→止 | 闭 | 全部 | 子宫缩小、腹痛缓解；hCG 迅速下降 |
| 稽留流产 Missed | **无/少** | 闭 | 无 | **胎心消失、早孕反应减弱**，胚胎死亡未排 |

> [!tip] 反射钩子
> - **宫颈口"开" + 绞痛** → 难免/不全流产（与前置胎盘"无痛、宫颈口闭"对比 → 这正是 Q047 易错点）。
> - **完全流产**：hCG **迅速下降** + 宫内空虚 + 内膜薄。
> - **稽留流产**：出血少甚至无，靠"早孕反应消退 + US 无胎心"抓。

### 2.2 异位妊娠 (Ectopic pregnancy, 异位妊娠)

- **诱因**：盆腔炎 (PID, pelvic inflammatory disease) / 输卵管手术或既往异位史 / 宫内节育器 (IUD, intrauterine device) / 辅助生殖。
- **典型 stem 钩子 → 临床意义**：停经 + 单侧下腹痛 + 阴道出血；附件包块/压痛 → 输卵管妊娠；**晕厥/腹膜刺激征/休克 → 破裂出血急症**。
- **查体**：附件压痛 ± 包块，宫颈举痛。
- **首选检查**：**经阴超声 (TVUS, transvaginal ultrasound)** + 定量 **β-hCG（β 人绒毛膜促性腺激素 β-human chorionic gonadotropin）**。
- **处理**：稳定/未破裂 → **甲氨蝶呤 (MTX, methotrexate)**；破裂/不稳定 → 手术。

> [!warning] 辨别带 Discriminatory zone（高频陷阱）
> **β-hCG < 3,500 mIU/mL 时，即使 TVUS 没看到宫内孕，也不能直接诊断异位妊娠**（可能只是太早）。
> - β-hCG 上升缓慢（48h 增幅 < 35%）或平台 → 异位 / 早期流产。
> - β-hCG **> 3,500 且宫腔空虚** → 异位妊娠。
> - 策略：等 hCG 达辨别带后复查 US，仍不见宫内孕方诊断异位。

### 2.3 葡萄胎 (Hydatidiform mole, 葡萄胎)

- **典型 stem 钩子 → 临床意义**：
  - **子宫大于孕周** → 滋养细胞异常增生。
  - **β-hCG 爆表 (> 100,000)** → 妊娠剧吐 (hyperemesis)、甲亢样表现 (hCG 类 TSH 作用)。
  - **孕 < 20 周出现子痫前期 (preeclampsia)** → **= 葡萄胎，直到排除为止**（正常子痫前期 ≥ 20 周才出现）。
  - 阴道排出"水泡样/葡萄样"组织 (hydropic tissue)、theca lutein 卵巢囊肿。
- **首选检查**：US 见 **"暴风雪 / snowstorm" 征** + 定量 β-hCG + 清宫物病理。
- **处理**：**清宫 (suction D&C, dilation and curettage)** + 术后连续追踪 β-hCG（监测妊娠滋养细胞肿瘤 GTN）。

## 三、中晚孕期出血（≥ 20 周，Antepartum hemorrhage）

> [!danger] 一秒分流口诀
> **痛 = 早剥 / 子宫破裂；无痛 = 前置（胎盘 / 血管）。**
> **Hypertension + Painful bleeding = Abruption → DIC（弥散性血管内凝血 disseminated intravascular coagulation）。**

### 3.1 前置胎盘 (Placenta previa, 前置胎盘) — 无痛、母体出血

- **诱因**：**既往剖宫产史 (prior cesarean delivery)**（核心）、既往前置胎盘、多胎妊娠。
- **stem 钩子 → 临床意义**：**孕 > 20 周突发无痛性阴道出血**，量较大，**初期胎心常正常**（母体出血，胎儿尚未受累）。
- **查体**：**禁指检**；先超声定位。
- **首选检查**：**经腹超声 (TAUS, transabdominal ultrasound) → 经阴超声 (TVUS)**，见胎盘覆盖/接近宫颈内口。
- **处理**：禁性交、禁指检、出血发作住院；**持续型 → 36–37 周计划性剖宫产 (cesarean delivery)**。

> [!warning] 胎盘植入 (Placenta accreta spectrum, 胎盘植入)
> **前置胎盘 + 既往子宫手术（剖宫产 / 诊刮 D&C / 肌瘤剔除 myomectomy）= 最强预警**。产前 US 见胎盘陷窝 (placental lacunae) + 肌层变薄 (myometrial thinning)。
> 处理：**34–35 周计划性剖宫产 + 子宫切除 (cesarean hysterectomy)**，胎盘留置原位（强行剥离 → 致命出血）。

### 3.2 胎盘早剥 (Abruptio placentae, 胎盘早剥) — 痛、母体出血

- **诱因**：**高血压 / 子痫前期**、腹部外伤 (blunt trauma)、**可卡因 (cocaine) + 吸烟**、既往早剥史。
- **stem 钩子 → 临床意义**：
  - **突发痛性阴道出血（常暗红）+ 背/腹痛** → 胎盘剥离。
  - **板状/强直压痛子宫 (rigid, tender uterus) + 高频低强度宫缩** → 子宫激惹。
  - 胎心 (FHR, fetal heart rate, 胎心率) 见**复发性晚期减速 (late decelerations)** → 胎盘灌注受损。
- **首选检查**：**临床诊断**！**超声阴性不能排除**（仅 ± 见胎盘后血肿 retroplacental hematoma）。
- **并发症**：胎儿缺氧/早产/死亡；母体出血、**DIC**（剥离面释放组织因子 tissue factor → 激活外源凝血）；**Couvelaire 子宫**（血液渗入肌层 → 蓝紫色 → 产后宫缩乏力性出血）。

> [!danger] 妊娠期钝性腹部创伤 → 早剥 → 失血性休克：抢救优先级
> ① **母体稳定**（气道 + **血制品输血**）→ ② 子宫复苏（**左侧卧位 + 氧疗**）→ ③ 胎儿评估（连续 FHR 监测 4–24h）→ ④ Rh(−) 者查 **KB 试验 (Kleihauer-Betke)** 指导 anti-D 剂量。
> **母体复苏永远先于胎儿评估。**

### 3.3 前置血管 (Vasa previa, 前置血管) — 无痛、**胎儿出血**

- **诱因**：前置胎盘 / 低置胎盘、多胎妊娠、体外受精 (IVF, in vitro fertilization)、副胎盘叶 (succenturiate lobe)、帆状脐带附着。
- **stem 钩子 → 临床意义**：**破膜 (ROM, rupture of membranes, 胎膜破裂) 后无痛阴道出血** + **胎心迅速恶化（心动过缓 / 正弦波 sinusoidal）** → 胎儿血管破裂、**胎儿失血**（不是母体！量少也致命）。
- **首选检查**：产前多普勒超声（覆盖宫颈口的胎儿血管）。
- **处理**：**紧急剖宫产 (emergency cesarean delivery)**；产前已知者计划性早期剖宫产。

### 3.4 子宫破裂 (Uterine rupture, 子宫破裂) — 痛、急症

- **诱因**：**既往子宫切口（剖宫产 / 肌瘤剔除）行剖宫产后阴道试产 (TOLAC, trial of labor after cesarean / VBAC, vaginal birth after cesarean)**、催产素 (oxytocin) 过度刺激、产程梗阻。
- **stem 钩子 → 临床意义**：**突发剧烈腹痛 + 宫缩张力骤降 / 宫缩停止**、**胎先露回缩 (loss of fetal station)**、**胎心骤降 (突发胎儿心动过缓)**、母体休克。
- **首选检查/诊断**：**临床急诊**（不等影像）。
- **处理**：**立即剖腹产 + 修补或子宫切除** + 母体复苏。

### 3.5 宫颈/阴道局部病变（含宫颈癌）— 无痛、局部源

- **诱因**：无产检 → 未做 Pap（漏检宫颈病变）；HPV 感染；宫颈息肉 / 外翻 / 宫颈炎。
- **stem 钩子 → 临床意义**（Q047 核心）：
  - **窥器见宫颈可视脆性溃疡/外生病灶 (friable ulcer / exophytic lesion)** → **直接定位钩子 → 宫颈癌 (cervical cancer)**。
  - **无痛**出血（区别于早剥/流产的痛）。
  - 题里"**胎盘后底**"一句 = 让你**排除前置胎盘**，出血另有源。
  - **宫颈管长、内口闭** → 不像流产（流产宫颈口应开）。
- **首选检查**：**宫颈活检 (cervical biopsy)** —— 孕期可做，别因妊娠回避！
- **处理**：确诊后**多学科（肿瘤 + 产科）**定分娩 / 治疗时机。
- **鉴别**：宫颈息肉 (cervical polyp) = 光滑、无痛少量出血、非溃疡，视诊即可，良性。

## 四、鉴别速查表（一图流）

> [!info] 孕期出血总表（痛 / 无痛 + 来源 + 首选检查 + 处理）

| 病因 | 孕周 | 痛？ | 标志性钩子 | 出血来源 | 首选检查 | 处理 / 铁律 |
|---|---|---|---|---|---|---|
| 流产 abortion | < 20w | 绞痛 | 宫颈口**开** + 妊娠物 | 母体 | US + β-hCG | 按型（观察/清宫） |
| 异位 ectopic | < 20w | 单侧痛 | hCG 未达辨别带宫内仍空 + 附件包块 | 母体 | TVUS + β-hCG | MTX / 手术；破裂→急诊 |
| 葡萄胎 mole | < 20w | 多无 | hCG 爆表 + 子宫>孕周 + snowstorm | 母体 | US + β-hCG | suction D&C + 追踪 hCG |
| 前置胎盘 previa | ≥ 20w | **无痛** | 既往剖宫产；胎盘覆宫颈口 | 母体 | TAUS→TVUS；**禁指检** | 36–37w 剖宫产 |
| 胎盘早剥 abruption | ≥ 20w | **剧痛** | HTN/外伤/可卡因；板状压痛子宫 | 母体 | **临床**（US 阴性不排除） | 母体复苏优先；防 DIC |
| 前置血管 vasa previa | ≥ 20w | 无痛 | **破膜后出血 + 胎心骤恶（正弦波）** | **胎儿** | 多普勒 US | **紧急剖宫产** |
| 子宫破裂 rupture | ≥ 20w（产时） | 剧痛 | VBAC/TOLAC；先露回缩 + 宫缩停 | 母体 | **临床急诊** | 立即剖腹 + 修补/切除 |
| 宫颈癌 cervical cancer | 任意 | **无痛** | **宫颈可视脆性溃疡/病灶** | 局部 | **宫颈活检** | 多学科定时机 |

> [!tip] 三组易混对决（出题最爱）
> - **前置胎盘 vs 胎盘早剥**：无痛 vs 剧痛；US 阳性 vs **临床（US 阴性不排除）**。
> - **前置胎盘 vs 前置血管**：都无痛，但前置血管=**胎儿出血 + 破膜后胎心骤恶**，量少也致命。
> - **流产 vs 宫颈癌（Q047）**：都"孕期出血"，但宫颈癌=**无痛 + 宫颈可视溃疡 + 宫颈口闭**；流产=**绞痛 + 宫颈口开 + 妊娠物**。看到宫颈可视 friable lesion 直接跳宫颈癌，别困在产科病因里。

## 五、复习自测

> [!question]- Q1. 孕 27 周 G1，无产检，无痛阴道出血；16 周 US 示胎盘位于子宫后底（不覆宫颈口），窥器见宫颈 2 cm 脆性溃疡。最可能诊断 + 下一步？
> **宫颈癌 (cervical cancer)**；下一步 **宫颈活检 (cervical biopsy)**（孕期可做）。胎盘后底 → 已排前置胎盘；宫颈口闭 → 排流产；"可视脆性溃疡"是直接定位钩子。

> [!question]- Q2. 孕 32 周高血压孕妇突发剧烈腹痛 + 暗红色阴道出血 + 板状压痛子宫，胎心见复发性晚期减速。最可能诊断 + 首要病理并发症？
> **胎盘早剥 (abruptio placentae)**；剥离面释放组织因子 → 外源凝血激活 → **DIC**。记忆钩：Hypertension + Painful bleeding = Abruption → DIC。注意：超声阴性不能排除（临床诊断）。

> [!question]- Q3. 双胎 IVF 孕妇，自然破膜 (ROM) 后立即无痛少量阴道出血，随即胎心降至 80 并出现正弦波。诊断 + 处理？
> **前置血管 (vasa previa)**，出血来源是**胎儿**；**紧急剖宫产 (emergency cesarean delivery)**。

> [!question]- Q4. 停经 7 周女性，单侧下腹痛 + 少量出血，β-hCG 1,800 mIU/mL，TVUS 未见宫内孕。能否直接诊断异位妊娠？为什么？
> **不能**。β-hCG < 辨别带 3,500，宫内未见孕囊可能只是太早。需复查/追踪 hCG（48h 增幅 < 35% 提示异位），达辨别带后 US 仍宫内空虚方诊断异位。

> [!question]- Q5. 既往一次剖宫产史孕妇行 VBAC 试产，突发剧痛 + 宫缩张力骤降 + 胎先露回缩 + 胎儿心动过缓。诊断 + 处理？
> **子宫破裂 (uterine rupture)**；**立即剖腹产 + 修补或子宫切除** + 母体复苏。临床急诊，不等影像。

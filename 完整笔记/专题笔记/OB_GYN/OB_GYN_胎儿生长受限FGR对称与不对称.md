# 胎儿生长受限 (Fetal Growth Restriction, FGR) — 对称型 vs 不对称型

> [!info] 一句话核心
> FGR = 胎儿因病理原因未能达到其应有生长潜能。关键考点是靠**起病孕龄**和**病因谱**把它分成对称型 (symmetric) 和不对称型 (asymmetric)——两型病因几乎不重叠，是 CK 反复考的鉴别点。

## 一、定义 (Definition)

FGR (Fetal Growth Restriction，胎儿生长受限；旧称 IUGR, Intrauterine Growth Restriction) 定义为：

- 超声估计胎儿体重 (Estimated Fetal Weight, **EFW**) < 第 10 百分位 (10th percentile)；或
- 出生体重 (birth weight) < 第 3 百分位 (3rd percentile)。

> [!warning] SGA ≠ FGR
> **SGA (Small for Gestational Age，小于胎龄儿)** 只是统计学上体积小（可以是健康的体质性小个子）；**FGR** 强调存在**病理原因**导致未达生长潜能。二者常混用，但概念上 FGR 隐含病理性。

## 二、核心对照表 — 对称型 vs 不对称型

> [!danger] 必背 高频对照
> 这张表是本主题的复习核心。CK 给你一段病史，你要能立刻判断落在哪一列，进而反推病因和下一步。

|  | 对称型 Symmetric | 不对称型 Asymmetric |
| --- | --- | --- |
| 起病 Onset | 早孕 (1st trimester，孕早期) | 中晚孕 (2nd / 3rd trimester，孕中晚期) |
| 机制 Mechanism | 早期打击、全局细胞数目↓ (global ↓ cell number) | 血流再分配、保脑效应 (blood flow redistribution, brain-sparing) |
| 病因 Etiology | **胎儿染色体异常 (chromosomal anomalies，如 trisomy 21 / trisomy 18)、先天感染 (congenital infection，TORCH 如 CMV 巨细胞病毒 / toxoplasmosis 弓形虫)** | **uteroplacental insufficiency (子宫胎盘功能不全)：母体高血压 hypertension、preeclampsia 子痫前期、吸烟 smoking；母体营养不良 malnutrition** |
| 特征 Features | 头身同步、比例性落后 (global / proportionate growth lag) | 头部保留 (head-sparing)，腹围 (abdominal circumference, AC) 最先且最重受累 |

> [!tip] 记忆钩子
> - **对称型 = "从一开始就全身小"** → 想到**内因/胚胎本身的问题**（染色体、先天感染）。打击发生在细胞增殖期，细胞总数减少，头身等比例落后。
> - **不对称型 = "后期营养供给出问题"** → 想到**胎盘/母体供血问题**。胎儿把有限血流优先分给大脑（brain-sparing），所以**头相对保住、肚子（肝糖原/脂肪）先瘦**，腹围最先掉队。

```
FGR (EFW < 10th %ile)
├── 对称型 Symmetric  ── 早孕起病 ── 内因
│     ├── 染色体异常 (T21 / T18)
│     └── 先天感染 (TORCH: CMV / Toxo)
│           → 头身等比例小、Doppler 常正常
│
└── 不对称型 Asymmetric ── 中晚孕起病 ── 供血/营养
      ├── uteroplacental insufficiency
      │     (HTN / preeclampsia / smoking)
      └── 母体营养不良 malnutrition
            → head-sparing、AC 先掉、Doppler 常异常
```

## 三、关键考点钩子 (High-Yield Hooks)

> [!danger] 对称型 + 结构畸形 + 高龄 → 非整倍体
> 题干若给：**对称型 FGR + 合并先天结构畸形（如 ASD 房间隔缺损 atrial septal defect）+ 高龄产妇 (advanced maternal age)**，这是**非整倍体 (aneuploidy)**（尤其 **21-三体 trisomy 21**）的强烈信号。下一步方向是遗传学评估（如无创产前筛查 / 羊膜腔穿刺 amniocentesis 核型分析），而不是单纯归因于胎盘。

> [!warning] SSRI 不致 FGR、非致畸
> **SSRI (selective serotonin reuptake inhibitors，选择性 5-羟色胺再摄取抑制剂，抗抑郁药)** 一般**不导致 FGR，也不属于主要致畸剂**。题干若把母亲服用 SSRI 当作 FGR 或畸形的"元凶"干扰项，通常是错误方向。（注意：孕晚期用药可有新生儿适应综合征/短暂 PPHN 关联，但这与"致 FGR"是两回事，别混淆。）

> [!warning] 孕龄错误可造成假性 FGR
> **孕龄推算错误 (inaccurate dating)** 会让实际正常的胎儿被误判为 FGR（把胎儿"算得比实际大"，于是显得偏小）。
> - 排除方法：若有**准确的早孕期超声 (accurate first-trimester ultrasound)** 确定孕龄，即可排除 dating error，此时的小胎儿才是真正的生长问题。
> - 早孕 CRL (crown-rump length，头臀长) 测量孕龄最准，是"锚定"孕龄的金标准时机。

## 四、管理 (Management)

一旦诊断 FGR，重点是**产前监护 (antenatal surveillance)**，评估胎儿状况、决定分娩时机：

- **regular NST (Non-Stress Test，无应激试验)** — 规律进行。
- **weekly BPP (Biophysical Profile，生物物理评分)** — 每周。
- **serial umbilical artery Doppler (脐动脉多普勒)** — 系列监测血流；出现舒张末期血流消失/反向 (absent / reversed end-diastolic flow) 提示胎盘功能严重恶化，是加强监护/提前分娩的重要指标。
- **serial growth ultrasound (系列生长超声)** — 追踪生长趋势。

> [!tip] Doppler 在两型的差异
> - **不对称型**（子宫胎盘功能不全）→ 脐动脉 **Doppler 常异常**（阻力升高），因为病根就在胎盘血流。
> - **对称型**（染色体来源）→ 胎盘血流可正常，**Doppler 可正常**——因为问题在胎儿本身而非供血。这一点常被用来区分两型的病理定位。

## 四-bis、出生后新生儿并发症（FGR 娩出后）⭐

> [!info] 为什么补这节
> FGR（尤其不对称型 / 子宫胎盘功能不全）胎儿长期缺氧 + 营养耗竭，出生后即面临一组可预测的并发症。Q4889 考的正是这一组。

慢性缺氧 + 保脑（血流优先供脑、牺牲肝/脂肪储备）→ 出生后：

| 并发症 | 机制 |
|---|---|
| **红细胞增多症 (polycythemia)** | 慢性缺氧 → EPO（促红细胞生成素 erythropoietin）↑ → 红细胞代偿性↑ |
| **低血糖 (hypoglycemia)** | 慢性营养剥夺 → 肝/肌糖原储备低 |
| **低体温 (hypothermia)** | 皮下脂肪少 → 保温差、散热多 |
| **低钙 (hypocalcemia)** | 胎盘功能不全 → 钙转运↓ |
| 围产期窒息 / 胎粪吸入 (perinatal asphyxia / meconium aspiration) | 缺氧转型困难 + 宫内应激 |

> [!danger] 高频陷阱：母亲用硫酸镁的新生儿是"高镁"不是"低镁"
> **镁自由通过胎盘 → 胎儿血镁 = 母体血镁**。母亲因子痫前期 (preeclampsia) / 神经保护用**硫酸镁 (magnesium sulfate)** 者，新生儿是**高镁血症 (hypermagnesemia)** → 呼吸抑制 + 肌张力低 (poor tone)，**不是低镁 (hypomagnesemia)**。Q4889 把 "hypomagnesemia" 设为干扰项，方向反了。
> （另一干扰项：母亲子痫前期 → 新生儿可有**血小板减少 (thrombocytopenia)**，7-10 天自愈。）

> [!tip] 记忆钩
> **「FGR 娩出三低一多：低血糖、低体温、低钙 + 红细胞多（+缺氧窒息）；妈用镁 → 娃高镁。」**

## 五、自测题

> [!question]- Q1. 一名 39 岁高龄孕妇，孕 12 周超声即发现胎儿头身**比例性**偏小（对称型 FGR），同时可见房间隔缺损 (ASD)。最可能的病因类别是？
> **胎儿染色体异常 (aneuploidy)，尤其 21-三体 (trisomy 21)**。
> 依据：对称型 + 早孕起病 + 结构畸形 + 高龄 = 非整倍体的经典组合。下一步应做遗传学评估（核型/NIPT + 必要时羊穿），而非归因于胎盘功能。

> [!question]- Q2. 一名孕 32 周、患慢性高血压并进展为子痫前期的孕妇，超声示胎儿**腹围明显落后而头围相对保留** (head-sparing)。该 FGR 属哪型？主要机制是什么？
> **不对称型 (asymmetric)**。机制为 **uteroplacental insufficiency (子宫胎盘功能不全)** 导致胎儿血流再分配、保脑 (brain-sparing)，故头相对保住、腹围（肝/脂肪储备）最先受累。脐动脉 Doppler 常异常。

> [!question]- Q3. 下列哪项**不是**对称型 FGR 的典型病因：先天 CMV 感染 / trisomy 18 / 母体子痫前期 / 弓形虫感染？并说明理由。
> **母体子痫前期 (preeclampsia)** 不是——它属于 **uteroplacental insufficiency**，导致**不对称型** FGR。其余三项（CMV、trisomy 18、toxoplasmosis）都是早孕起病的内因，属对称型。
> 附加提醒：母亲服用 SSRI 也**不是** FGR 病因（非致畸、不致 FGR），常作干扰项。

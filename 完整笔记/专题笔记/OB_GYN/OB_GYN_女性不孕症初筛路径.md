# 女性不孕症初筛路径 (Female Infertility Initial Workup)

> [!info] 定义
> **不孕症 (infertility)** = 规律无保护性生活 **12 个月**（≥35 岁女性为 **6 个月**）仍未受孕。达标即启动系统评估。

---

## 一、第一步：先分"有无排卵"

评估思路的核心分叉，先看**月经模式**判断是否排卵：

```
不孕 (infertility)
   │
   ├─ 月经规律 (regular menses)
   │     → 提示有排卵 (ovulatory)
   │     → 查 male factor / tubal factor / uterine factor
   │
   └─ 月经稀发 / 闭经 (oligomenorrhea / amenorrhea)
         → 提示无排卵 (anovulation)
         → 病因学初筛（见第三节）
```

> [!tip] 无排卵的客观证据
> - **oligomenorrhea (月经稀发)** — 周期 >35 天 / 每年 <9 次
> - **monophasic basal body temperature (基础体温单相)** — 无排卵后孕酮升高带来的双相体温 → 提示未排卵
> - 排卵存在时：mid-luteal serum progesterone (黄体中期血清孕酮) 升高、BBT 双相

---

## 二、不孕四大病因框架

| 病因类别 | 初筛手段 |
|---|---|
| Ovulatory factor (排卵因素) | 病史 + BBT + 黄体中期孕酮 + 内分泌筛查 |
| Male factor (男方因素) | Semen analysis (精液分析) |
| Tubal / peritoneal factor (输卵管/腹膜因素) | Hysterosalpingography, HSG (子宫输卵管造影) |
| Uterine / cervical factor (子宫/宫颈因素) | 超声 / HSG / 宫腔镜 |

> [!warning] 男方因素易被忽略
> 男方因素约占不孕的 1/3，**semen analysis (精液分析)** 便宜、无创，应尽早做。但当女方已有明确无排卵证据时，先针对无排卵做病因筛查。

---

## 三、无排卵：病因学初筛（本题考点）

一旦锁定 anovulation，第一轮病因筛查按"常见 + 可逆"排序：

| 首查项目 | 针对病因 | 提示线索 |
|---|---|---|
| **TSH** | Hypothyroidism (甲状腺功能减退) | 体重增加、乏力、便秘、怕冷 |
| **Prolactin (泌乳素)** | Hyperprolactinemia (高泌乳素血症) | Galactorrhea (溢乳)、闭经 |
| FSH / LH / Estradiol | 卵巢功能 / 中枢性问题 | 卵巢早衰、下丘脑性闭经 |
| 游离雄激素 / 超声 | PCOS (多囊卵巢综合征) | Hirsutism (多毛)、痤疮、肥胖、月经稀发 |

> [!danger] 高频考点：甲状腺是可逆病因
> 无排卵 + **体重增加**、无 hirsutism、无 galactorrhea → **先查 TSH**。
> **hypothyroidism (甲减)** 是常见且**可逆**的无排卵原因，治疗甲减常能恢复排卵。

---

## 四、机制：甲状腺异常如何致无排卵

```
hypothyroidism (甲减)
   ↓
hypothalamic-pituitary-ovarian axis 紊乱
（TRH↑ 可致 prolactin↑）
   ↓
排卵不规律 / 无排卵
   ↓
oligomenorrhea + infertility
   ↓
monophasic basal body temperature (单相基础体温)
```

---

## 五、复习自测

> [!question]- 33 岁女性，13 个月未孕；月经每 4 个月一次，基础体温单相，妊娠试验阴性；近期体重增加 10 lb；无多毛、无溢乳。下一步最合适的检查？
> **Measurement of serum TSH (查血清 TSH)**。
> - monophasic BBT + oligomenorrhea → **anovulation (无排卵)**
> - 体重增加、无 hirsutism / galactorrhea → 指向 **hypothyroidism**（可逆病因，优先筛）
> - 无多毛降低 PCOS 优先级；无溢乳降低 hyperprolactinemia 优先级

> [!question]- 月经规律的不孕夫妇，女方评估未见异常，下一步最经济的检查？
> **Semen analysis (精液分析)**。月经规律提示有排卵，男方因素占约 1/3，精液分析无创便宜，应尽早排查。

## ⚠️ 反复错 & 高频陷阱速记（每日必扫）

> [!danger] 🔴 真反复错（错 ≥2 次，最优先）
> - **PPROM（未足月胎膜早破 preterm premature rupture of membranes）孕周分层**：<34 周未临产 → **保守延长孕龄 (latency)** + 潜伏期抗生素 + 激素，**不急着引产**；≥34 周 → 分娩。（详见 A9 / [[完整笔记/专题笔记/OB_GYN/OB_GYN_早产管理_孕周分层与药物]]）
> - **孕期早期 Lyme（莱姆病 Lyme disease）→ 阿莫西林 amoxicillin**（孕期**禁多西环素 doxycycline**）；非孕早期 Lyme 才用 doxy，播散神经/心脏才 **头孢曲松 ceftriaxone**。（反复错×3；跨科见 [[完整笔记/专题笔记/感染/感染_HighYield速查总览]]）
> - **尿失禁三型"方向相反"**（最爱错反）：急迫性 (urge) → **抗毒蕈碱 (antimuscarinic) 降逼尿肌**；充溢性/潴留（滴尿 + 残余尿 PVR↑ post-void residual，如产后尿潴留）→ **导尿减压**（**禁抗毒蕈碱**，会加重）；压力性 (stress) → 凯格尔 Kegel/悬吊，**产后压力性尿失禁多自限 → 先观察**。（详见 B13）

> [!warning] ⚠️ 高频易错陷阱（一眼扫，别再翻车）
> **产科出血/急救**
> - **痛 = 早剥/破裂，无痛 = 前置**；`高血压 + 痛性出血 = 早剥 → DIC（弥散性血管内凝血 disseminated intravascular coagulation）`。
> - **瘢痕子宫 + 胎先露回缩/消失 = 子宫破裂**（先露回缩是独有征）。
> - **妊娠钝性腹部创伤：母体复苏永远先于胎儿评估**（气道+输血 → 子宫复苏 → 胎心 FHR fetal heart rate）。
> - **异位妊娠 (ectopic pregnancy)**：血流动力学不稳/破裂 → **手术**；稳定小无胎心 → **甲氨蝶呤 MTX methotrexate**/期待。
>
> **子痫前期 / 硫酸镁**
> - 孕 ≥20 周新发抽搐 → **先想子痫 (eclampsia)、上硫酸镁 (magnesium sulfate)**，别急着 EEG（脑电图 electroencephalogram）/腰穿。
> - 先兆子痫相关卒中 = **缺血 + 脑实质出血**，不是 SAH（蛛网膜下腔出血 subarachnoid hemorrhage）。
> - 镁中毒（腱反射消失/嗜睡）→ **停镁 + 葡萄糖酸钙 calcium gluconate**；**重症肌无力 MG (myasthenia gravis) 禁镁**。
>
> **产程 / 产后**
> - **活跃期停滞 → 剖**；**第二产程停滞但胎头已下降 → 器械助产**（别混）。
> - **妊娠剧吐 (hyperemesis) 补液先加 B1（硫胺素 thiamine）** 防 Wernicke（韦尼克脑病 Wernicke encephalopathy）。
> - 肩难产 (shoulder dystocia) = 不可预测急症 → **靠模拟训练 (simulation)，不是风险分层**。
>
> **妇科 / 筛查**
> - 子宫切除术后 **watery 阴道排液 + 血 Cr（肌酐 creatinine）正常 = 输尿管损伤 (ureter laceration)**（Cr 正常是关键，别当肾功能问题）。
> - **绝经后新发胃肠症状**（便秘/腹胀/早饱）+ 结肠镜阴性 → 查 **CA-125 + 盆腔超声**（卵巢癌隐匿表现）。
> - 附件包块分流：**绝经前 = β-hCG + 超声；绝经后 = CA-125 + 超声**。
> - **ASC-US（意义不明非典型鳞状细胞 atypical squamous cells of undetermined significance）21–24 岁**：不论 HPV（人乳头瘤病毒 human papillomavirus），一律 **12 个月后复查细胞学**（先等一年）。
> - **乳腺癌筛查**：average-risk（平均风险）功能稳定者按功能状态而非年龄停筛，**停筛个体化在 75 岁后**（71 岁续筛）。
> - **无症状菌尿 (asymptomatic bacteriuria) 孕期也要治**；孕期避 **TMP-SMX（复方磺胺甲噁唑 trimethoprim-sulfamethoxazole）/ 多西环素 / 活疫苗（MMR 麻腮风 measles-mumps-rubella/水痘 varicella）**。

## A1. 产前保健与筛查（Prenatal care & screening）

> [!tip] 关键时间节点（Key Numbers，照源文件）
> - **早孕 (1st trimester)**：<14 周；**中孕 (2nd)**：14–28 周；**晚孕 (3rd)**：28–42 周。
> - **表面活性物质 (surfactant)**：24 周成熟。
> - **不在 28 周前剖宫产 (no C-section before 28 weeks)**。
> - **四联筛查 (quad screen)**：>15 周。
> - 产检频率：1st q4–6 周 → 2nd q4 周 → 3rd q2 周 → **36 周后每周**。

**遗传/染色体筛查（按时机选）**

| 检查 | 时机 | 风险 |
|---|---|---|
| 游离 DNA (cell-free DNA, cfDNA) | 首选第一步 | 无 |
| 绒毛膜取样 (CVS, chorionic villus sampling) | 10–12 周 | 肢体缺陷 (limb defects) |
| 羊膜腔穿刺 (amniocentesis) | 15 周 | 流产 (miscarriage) |
| 经皮脐血取样 (PUBS) | ~17 周 | 流产 |

- **21 三体 (Trisomy 21, 唐氏)**：↓ AFP、↓ estriol、↑ β-hCG、↑ inhibin。
- **18 三体 (Trisomy 18, Edwards)**：AFP↓、estriol↓、β-hCG↓，**inhibin A 正常**（← 与 T21 的区分钥匙：T21 的 hCG 与 inhibin 都 ↑，T18 仅前三项 ↓）。
- **孤立 MSAFP↑（母血甲胎蛋白升高）** = 神经管缺陷 (NTD, neural tube defect) / 腹壁缺损 (abdominal wall defect)；先复核孕龄 + 排多胎（假阳性）。

**孕期疫苗**：**Tdap 每次妊娠 27–36 周**接种、流感灭活苗任意孕周；**活疫苗 (MMR 麻腮风 / 水痘 varicella) 孕期禁用**，风疹 (Rubella) 疫苗须**孕前 8 周**接种；风疹非免疫者 → **产后立即补种 MMR**；水痘暴露 + IgG 阴性 → 给水痘免疫球蛋白 (VZIG)。

> [!tip] 孕期生理性改变（看似异常实为正常，别当病理）
> - **血液**：稀释性贫血 (dilutional anemia)（血浆↑>红细胞，Hb 可低至 ~11，MCV 正常）；生理性白细胞↑（可达 15k，产程更高）；高凝 → 静脉血栓 (VTE) 风险↑。
> - **呼吸**：慢性代偿性呼吸性碱中毒（PaCO₂~30，孕酮驱动分钟通气↑，VC/FEV1 正常）= 生理性呼吸困难 (dyspnea of pregnancy)。
> - **肾**：肾小球滤过率 (GFR)↑50% → 尿素氮/肌酐 (BUN/Cr) 生理性下降（Cr 0.9 在孕期可能已是肾损）。
> - **甲状腺**：总 T4↑（甲状腺结合球蛋白 TBG↑）≠ 甲亢，看游离 T4。
> - 孕期增重按孕前 BMI：太少→早产/低体重儿，太多→GDM/巨大儿 (macrosomia)。

## A3. 妊娠期高血压谱（Hypertensive disorders）

> [!warning] 分界线（照源文件）
> - **慢性高血压 (chronic HTN)**：孕前或 **<20 周** 已有 BP ≥140/90。
> - **妊娠期高血压 (gestational HTN)**：**≥20 周** 新发高血压，**无蛋白尿、无终末器官损害**。

```
子痫前期 (preeclampsia, 高血压 + 蛋白尿/终末器官损害)
   ├─ 轻度 (mild)：远离足月 → 控压观察；近足月 → 分娩
   └─ 重度 (severe)：头痛/右上腹痛/视觉变化
          → IV 降压 + 硫酸镁 (magnesium sulfate) 防抽搐 + 分娩
HELLP (溶血 Hemolysis / 肝酶↑ Elevated LFTs / 血小板↓ Low platelets)
   → 立即分娩 (deliver immediately)
妊娠急性脂肪肝 (acute fatty liver)：意识混乱 + 低血糖 → ICU + 分娩
```

> [!warning] 子痫前期"严重特征" (severe features) 6 条判据 — 任一即 severe → 硫酸镁 + 降压 + 计划分娩
> BP ≥160/110 · 血小板 <100k · 肌酐 Cr >1.1（或翻倍）· 转氨酶 >2× 上限 · 肺水肿 (pulmonary edema) · 新发脑/视觉症状（头痛/视物模糊）。
> ⚠️ **蛋白尿量不决定严重程度**；无蛋白尿也可凭上述任一诊断"子痫前期伴严重特征"。

> [!danger] 硫酸镁中毒 (magnesium toxicity) 阶梯 + 解药
> 轻度：恶心/潮红/头痛 + **腱反射减弱 (hyporeflexia)** → 中度：**腱反射消失 (areflexia)** + 低钙 + 嗜睡 → 重度：**呼吸麻痹 / 心脏骤停**。
> **处理：停镁 + IV 葡萄糖酸钙 (calcium gluconate) bolus**。
> 机制：镁竞争性拮抗钙 → 抑制突触前乙酰胆碱释放（∴ **重症肌无力 myasthenia gravis 绝对禁用**，避镁改用**左乙拉西坦 levetiracetam 或苯二氮䓬类 benzodiazepine**；⚠️ 丙戊酸 valproic acid 孕期致畸，非此处替代）。
> **降压安全药**：拉贝洛尔 labetalol / 肼屈嗪 hydralazine / 硝苯地平 nifedipine / 甲基多巴 methyldopa。

## A4. 妊娠剧吐 / GDM（孕期代谢）

- **妊娠剧吐 (hyperemesis gravidarum)**：顽固呕吐 + 体重↓>5% + 酮症/电解质紊乱 → **先 IV 补液（加维生素 B1 硫胺素 thiamine 防 Wernicke 韦尼克脑病 Wernicke encephalopathy）**，止吐阶梯 = 维生素 B6 ± 多西拉敏 doxylamine → 异丙嗪 promethazine/甲氧氯普胺 metoclopramide/昂丹司琼 ondansetron。⚠️ 早孕轻度 **TSH 略低 = hCG effect（生理性）**，无甲状腺毒症体征别当 Graves 病 (Graves disease)、别上 PTU。
- **妊娠期糖尿病 (GDM, gestational diabetes)**
  - **筛查**：**24–28 周**（高危更早）。激素元凶 = **人胎盘催乳素 (hPL, human placental lactogen)**。
  - 检测：**1 小时葡萄糖耐量 (1-hr GTT) 最敏感** → **3 小时 GTT 最特异**确诊。
  - 管理：轻 → 生活方式；重 → **胰岛素 (insulin)** + 生活方式（口服降糖一般不用；**SGLT2 抑制剂 (SGLT2i) / 噻唑烷二酮 (TZD) 孕期禁用**，孕期不建议减重）。血糖目标：**空腹 ≤95 / 餐后 1h ≤140 / 2h ≤120 mg/dL**。
  - **产后 4–12 周查 75g 2 小时糖耐量**（排查显性糖尿病）。
- **妊娠期甲状腺 (thyroid in pregnancy)**：诊断看**游离 T4↑ + TSH↓**（总 T4↑ 是 TBG 假象）。**孕早期 T1 用丙硫氧嘧啶 (PTU)**（避甲巯咪唑 methimazole 致头皮发育不全 aplasia cutis）；**T2/T3 换甲巯咪唑 methimazole**（避 PTU 肝毒）；**放射性碘 I-131 妊娠绝对禁忌**；两药均可致粒细胞缺乏 (agranulocytosis)（发热咽痛查血常规 CBC）。
- **妊娠期贫血分型**：给了 MCV/MCHC 就分型，别反射"稀释性"。**小细胞低色素 → 缺铁性贫血 (IDA, iron-deficiency anemia)（孕期最常见病理性贫血，铁蛋白 ferritin↓）**；MCV 正常 + ferritin 正常 = 稀释性（生理）；大细胞 → 叶酸/维生素 B12 缺乏。

## A5. 胎儿监护（Fetal monitoring：NST / BPP / 减速）

> [!tip] NST / BPP / CST
> - **无应激试验 (NST, nonstress test)**：**有反应 (reactive) = 2 次加速 (accelerations)**。无反应 → 声振刺激 → BPP。
> - **生物物理评分 (BPP, biophysical profile)** 4 项 + NST（满分 10）：胎动 / 张力 / 呼吸 / 羊水。**8–10 = 正常**；**6 = 可疑（24h 复查）**；**≤4 → 考虑分娩（或 CST 宫缩应激试验）**。
> - **宫缩应激试验 (CST, contraction stress test)**：用缩宫素；**阳性 (late decels) = 紧急分娩**。

**减速类型（VEAL CHOP 逻辑）**

| 减速类型 | 机制 | 首选干预 |
|---|---|---|
| 早期减速 (early) | 胎头受压 (head compression) | 观察（生理性） |
| 变异减速 (variable) | 脐带受压 (cord compression) | 改体位（左侧卧/膝胸位）；无效 → 羊膜腔输液 (amnioinfusion) |
| 晚期减速 (late) | 胎盘灌注不足 (uteroplacental insufficiency) | 侧卧 + 停缩宫素 + 补液 |
| 正弦波 (sinusoidal) | 自主调节丧失 = 严重胎儿贫血/急性失血 | 紧急分娩 |

> [!warning] 晚期减速的"找病因"反射
> - 晚期减速 + **宫缩过频 (tachysystole, >5 次/10 min)** → **立即停缩宫素 (oxytocin)**。
> - 晚期减速 + 硬膜外麻醉后低血压 → **补液 + 升压药 (去甲肾上腺素 norepinephrine)**。

**FHR 三分类**：Category I = 全正常（基线 110–160 + 中度变异 6–25 + 无晚/变异减速）；Category II = 中间型；**Category III**（变异消失 + 复发晚/变异减速或心动过缓，或正弦波）→ **剖宫产 (C-section)**。

**基线异常病因**：胎儿心动过速 (>160) = 母体发热/绒毛膜羊膜炎、β 受体激动剂、胎儿甲亢；胎儿心动过缓 (<110) = 母体低体温、β 阻滞剂、胎儿甲减、**胎儿心脏传导阻滞（anti-Ro/SSA、anti-La/SSB）**。

**胎儿生长受限 (FGR, fetal growth restriction)：对称 vs 不对称**

| 类型 | 起病 / 病因 | 特点 |
|---|---|---|
| 对称型 (symmetric) | 早孕起病，内因（染色体 T21/T18、TORCH 感染） | 头身同比例小；查核型 + TORCH |
| 不对称型 (asymmetric) | 中晚孕，子宫胎盘功能不全（高血压/子痫前期/吸烟） | 腹围先受累、头保留 (head-sparing) |

- 管理：NST + 每周 BPP + 系列脐动脉多普勒（舒张末血流消失/反向 AEDF/REDF 提示恶化）。娩出后"三低一多"（低血糖/低体温/低钙 + 红细胞增多症 polycythemia）；母用硫酸镁 → 新生儿**高**镁（非低镁）。

**双胎绒毛膜膜型 (chorionicity)**：单胎盘 = 单绒毛膜；无隔膜 = 单羊膜。**TTTS（双胎输血综合征 twin-twin transfusion syndrome）= 单绒双羊 (Mono-Di)**（共享胎盘、羊水过多+过少组合、超声 T 征）；**脐带缠绕 = 单绒单羊 (Mono-Mono)**。⚠️ 见单胎盘别直接反射 TTTS，先看有无隔膜。

## A6. 产前出血（Antepartum hemorrhage：痛 vs 无痛）

> [!danger] 一秒分流：痛 = 早剥/破裂；无痛 = 前置
> **Hypertension + Painful bleeding = Abruption → DIC**

| 特征 | 前置胎盘 placenta previa | 胎盘早剥 abruptio placentae | 前置血管 vasa previa |
|---|---|---|---|
| 疼痛 | 无痛 | **剧痛** | 无痛 |
| 出血来源 | 母体 | 母体 | **胎儿** |
| 经典钩子 | 既往剖宫产史 | 高血压/外伤/可卡因 cocaine + 烟 | 破膜 ROM 后出血 |
| 胎心 | 初期正常 | 常见晚期减速 | 迅速恶化/胎儿死亡（心动过缓/正弦波） |
| 首选检查 | **经阴超声 (TVUS)**，**禁内诊** | **临床诊断**（超声阴性不排除） | 多普勒超声 |
| 处理 | 持续型 → **36–37 周剖宫产** | 母体复苏优先（详见下） | **紧急剖宫产** |

> [!warning] 妊娠期钝性腹部创伤 → 早剥 → 失血性休克：抢救优先级
> ① 母体稳定（气道 + **血制品输血**）→ ② 子宫复苏（左侧卧 + 氧疗）→ ③ 胎儿评估（连续 FHR 监测 4–24h）→ ④ Rh(-) 者查 KB 指导 anti-D。**母体复苏永远先于胎儿评估。**

- **胎盘植入 (placenta accreta)**：前置胎盘 + 既往子宫手术（剖宫产/诊刮 D&C, dilation & curettage/肌瘤剔除）= 最强预警；产前超声见胎盘陷窝 (placental lacunae) + 肌层变薄 → **34–35 周计划性剖宫产 + 子宫切除 (cesarean hysterectomy)**，胎盘留置原位。

## A7. 产程异常与难产（Labor abnormalities & dystocia）

- **产程分期**：第一产程（0–6 cm 潜伏 → 6–10 cm 活跃）→ 第二（开全至娩出）→ 第三（胎盘）→ 第四（产后）。
- **停滞处理**：宫缩**过强 (hypertonic) → 吗啡 morphine**；宫缩**乏力 (hypotonic) → 缩宫素 oxytocin**。
- **臀位 (breech)**：36 周外倒转术 (external cephalic version, ECV)。
- **活跃期进展减慢 (active phase protraction)** 独立危险因素：肥胖 (BMI ≥30) + 增重过多，常合并头盆不称 (CPD)。
- **肩难产 (shoulder dystocia)** = 不可预测急症 → **靠模拟训练 (simulation-based training) 而非风险分层**（patient safety 考点）。

## A8. 产后出血 PPH 与产后感染（Postpartum）

> [!tip] PPH 四 T（4 T's）按发生率
> **Tone（宫缩乏力 uterine atony，MCC）> Trauma（裂伤 laceration）> Tissue（胎盘残留 retained placenta）> Thrombin（凝血障碍 coagulopathy）**

```
产后出血 PPH
   ├─ 宫缩乏力(最常见) → 子宫按摩 + 宫缩剂(缩宫素/麦角/前列腺素)
   ├─ 裂伤 → 手术修补
   └─ 胎盘残留(疑有碎片) → 清宫
```

- **产后子宫内膜炎 (endometritis)**：产后发热 + 子宫压痛 → **克林霉素 clindamycin + 庆大霉素 gentamicin**。
- **Couvelaire 子宫**：早剥血渗入肌层 → 表面紫蓝 → 产后宫缩乏力性出血。

> [!tip] PPH 宫缩剂禁忌配对（高频陷阱）
> 缩宫素 oxytocin 一线 → 甲基麦角新碱 methylergonovine（**高血压/先兆子痫禁**）→ 卡前列素 carboprost（**哮喘禁**）→ 米索前列醇 misoprostol（几乎都能用）。**前列腺素 + 子宫瘢痕 = 子宫破裂风险 → 禁前列腺素引产**。

- **脓毒性盆腔血栓性静脉炎 (SPT, septic pelvic thrombophlebitis)**：产后 **抗生素无效的持续 spiking 高热、但患者一般情况好** → 加**肝素 heparin**（退热即诊断即治疗）。
- **乳腺脓肿 (breast abscess)**：波动性肿块 → 切开引流 (I&D, incision & drainage)（区别乳腺炎 mastitis：继续哺乳 + 双氯西林 dicloxacillin）。

> [!tip] 产后情绪障碍三分（时间 × 症状 × 处理）
> - **产后忧郁 (baby blues)**：产后 2–3 天起、**≤2 周自限** → 安慰观察。
> - **产后抑郁 (PPD, postpartum depression)**：>2 周中重度 + 功能损害 → **舍曲林 sertraline（哺乳首选）** ± 心理治疗。
> - **产后精神病 (postpartum psychosis)**：妄想/幻觉 + **杀婴风险** → 精神科急症、住院、**绝不可单独与婴儿相处**。
> 所有产妇筛抑郁（EPDS 爱丁堡产后抑郁量表）+ 主动问自伤/伤婴念头；排 Hb（产后贫血）/ TSH（产后甲状腺炎）。

## A9. 早产 / PROM（Preterm & 胎膜早破）

- **早产孕周三阈值（背）**：**<32 周 → 硫酸镁 (magnesium sulfate) 神经保护**；**<34 周 → 倍他米松 betamethasone 促肺成熟**；**<37 周 → 保胎 (tocolysis) + GBS 预防**。保胎药按孕周：吲哚美辛 indomethacin <32 周 / 硝苯地平 nifedipine 32–34 周 / 特布他林 terbutaline 仅救急。**保胎禁忌（该让她生）**：绒毛膜羊膜炎、胎儿窘迫、重度子痫前期、显著早剥。**保胎黑名单药**（催产非保胎）：缩宫素/米索前列醇/麦角新碱/米非司酮。
- **GBS（B 族链球菌 group B strep）**
  - **筛查 36–37⁺⁶ 周**（现行 ACOG/CDC；旧版 35–37 已废）；阳性 → **产时青霉素 (intrapartum penicillin)**。青霉素过敏路径：头孢唑林 cefazolin（低危过敏）→ 克林霉素 clindamycin（须查敏感）→ 万古霉素 vancomycin（高危过敏/克林霉素耐药）。
  - **自动经验性治疗**（不需培养）：既往 GBS 患儿、**本次妊娠 GBS 菌尿（自动打）**、产时状态不明 + 风险、**破膜 ROM >18 h**、母体发热。
  - **不打**：择期剖宫产 + 胎膜完整 + 未临产（即使 GBS 阳性）。IAP 只防早发型 (early-onset)，不防晚发型。
- **感染分岔**：防 GBS（单药青霉素）≠ 治宫内感染。**绒毛膜羊膜炎 (chorioamnionitis) = 氨苄西林 ampicillin + 庆大霉素 gentamicin + 尽快分娩**（剖加克林霉素/甲硝唑 metronidazole）；PPROM latency = 氨苄西林 + 红霉素/阿奇霉素 azithromycin。「尽快分娩」≠「立刻剖宫产」。
- **羊水异常**：羊水过多 (polyhydramnios) ← 胎儿消化道梗阻，风险 = 早产；羊水过少 (oligohydramnios) ← 肾发育问题，风险 = **肺发育不良 (pulmonary hypoplasia)**。

## A10. 孕期感染 + TORCH

- **孕妇 UTI**：膀胱炎 (cystitis)/肾盂肾炎病原 = E. coli > Proteus > Klebsiella；**无症状菌尿孕期也要治**（防肾盂肾炎/早产）；安全药 = 阿莫西林 amoxicillin / 头孢氨苄 cephalexin / 呋喃妥因 nitrofurantoin（**TMP-SMX 孕期避免**）。
- **HIV**：筛 ELISA → Western blot 确认；**剖宫产优先**（除非病毒载量极低 + CD4 高）。
- **梅毒 (syphilis)**：筛 VDRL/RPR → FTA-ABS 确认 → 青霉素（过敏脱敏）。
- **生殖道疱疹 (genital HSV)**：核心风险 = 产时经产道传播（非孕早期致畸）。两个独立决策：**有生殖器 HSV 病史 → ≥36 周起阿昔洛韦 acyclovir/伐昔洛韦 valacyclovir 抑制**；**临产时有活动性病灶/前驱症状 → 剖宫产**。痛性溃疡 = HSV；无痛硬下疳 = 梅毒。
- **孕期结核 (TB) 筛查阳性（TST 结核菌素试验 或 IGRA γ-干扰素释放试验 阳性）→ 下一步胸片（腹部铅屏蔽）** 分潜伏 vs 活动：潜伏 = 异烟肼 INH（+ 维生素 B6）；活动 = RIPE 四联。IGRA 不受卡介苗 (BCG) 干扰（移民优选）。

**TORCH + 常考宫内感染鉴别（重要鉴别，对比表留总览）**

| 病原 | 暴露/母体题眼 | 胎儿/新生儿招牌 | 影像·超声钥匙 | 处理/预防 |
|---|---|---|---|---|
| 弓形虫 Toxoplasma | 生肉/猫粪/园艺土 | 脉络膜视网膜炎 + 脑积水 | **弥漫**颅内钙化 | 母体螺旋霉素 spiramycin |
| 风疹 Rubella | 未接种 + 孕早期出疹（脸→下） | 动脉导管未闭 (PDA)/肺动脉狭窄 + 白内障 + 耳聋 | FGR / 心脏结构线索 | 无治疗 → 孕前疫苗预防 |
| 巨细胞病毒 CMV | daycare 接触、母轻症 | **感音神经性耳聋（最常考）** | **室周**钙化 + 脑室扩大/小头 | 支持（重症更昔洛韦 ganciclovir） |
| 单纯疱疹 HSV | 分娩时活动病灶 | 新生儿播散/脑炎（皮眼口 SEM） | 无固定超声 | ≥36 周阿昔洛韦 acyclovir 抑制；临产病灶→剖 |
| 梅毒 Syphilis | 无产检 / 高危性史 | 先天梅毒；间质性角膜炎（Hutchinson） | 肝脾大/胎盘增厚/hydrops | 青霉素 penicillin |
| 细小病毒 Parvovirus B19 | 接触"第五病"患儿 | 重度胎儿贫血 → hydrops | 大脑中动脉 MCA 多普勒↑ | 宫内输血 |
| 李斯特菌 Listeria | 冷藏熟食 + 水样腹泻 | 宫内死胎 (IUFD)/早产/新生儿败血症 | 胎盘/胎儿弥漫感染 | 氨苄西林 ampicillin（±庆大 gentamicin） |

**眼病钥匙**：Toxo/CMV = 脉络膜视网膜炎 (chorioretinitis)；Rubella = 白内障 (cataract)；Syphilis = 间质性角膜炎 (interstitial keratitis)；HSV = 角膜炎 (keratitis)。

## B1. 异常子宫出血 AUB（PALM-COEIN）

> [!tip] PALM-COEIN 框架
> **结构性 PALM**：Polyp 息肉 / Adenomyosis 子宫腺肌病 / Leiomyoma 肌瘤 / Malignancy 恶性。

> **任何育龄女性月经异常第一步 = 尿/血 β-hCG。**

**急性 AUB（功能失调性出血 DUB / 无排卵）处理**

| 状态 | 一线 | 目的 |
|---|---|---|
| 血流动力学稳定 | **大剂量雌激素 (high-dose estrogen)** 口服或 IV | 稳定内膜 |
| 血流动力学不稳定 | **诊刮 (D&C, dilation & curettage)** | 机械止血 |
| 雌激素禁忌 | 大剂量孕激素 (high-dose progestin) | 抑制增生 |

- **长期管理**：止血后续用**口服避孕药 (OCPs)** 或周期孕激素调经。

## B2. 闭经（Amenorrhea：原发 vs 继发）

> [!warning] 第一步永远是 β-hCG（排妊娠）

> [!tip] 原发闭经何时启动评估
> **13 岁无第二性征（无乳房发育）或 15 岁有性征无初潮** → 开始 workup；未达阈值 → 观察。

**原发性闭经 (primary amenorrhea)** — 看乳房（雌激素）+ 子宫（苗勒管）

```
有乳房 + 有子宫 → 处女膜闭锁 (imperforate hymen) / 低体脂
有乳房 + 无子宫 → 苗勒管发育不全 (Müllerian agenesis/MRKH, 46,XX)
                  vs 雄激素不敏感 (AIS, 46,XY, 无阴毛)
无乳房        → Turner (45,X, ↑FSH) / Kallmann (↓FSH + 嗅觉缺失 anosmia)
```

**继发性闭经 (secondary amenorrhea)**：查 β-hCG → TSH → 泌乳素 (prolactin) → FSH/E2。

- **孕激素挑战试验 (progesterone challenge)**：撤退性出血 → 无排卵（有雌激素、内膜在）。
- **雌孕激素挑战**：出血 + **高 FSH = 卵巢早衰 (POI)**。

> [!tip] 三大"秒杀线索"（继发性）
> - 溢乳/视野缺损/头痛 → 高泌乳素血症 (hyperprolactinemia)。
> - 潮热/阴道萎缩 + **FSH↑ + E2↓** → 原发性卵巢功能不全 (POI)。
> - 减重/过度运动/进食障碍 + **FSH↓/正常低 + E2 低（非零）** → 功能性下丘脑性闭经 (FHA)。

## B3. PCOS（多囊卵巢综合征）

> [!info] 机制链
> **GnRH 脉冲频率↑ → LH↑（LH:FSH 比值升高）→ 卵巢雄激素↑ → 外周转 E1↑、无对抗雌激素** + 高胰岛素血症协同。

- **Rotterdam 标准 3 选 2**：① 高雄表现（临床/化验）② 月经稀发/不排卵 ③ 超声多囊卵巢。
- 风险：无对抗雌激素 → **子宫内膜增生/内膜癌**（→ 需孕激素保护）。
- 治疗按生育需求分层：不求育 → OCPs（调经 + 抗雄）；求育 → 来曲唑 letrozole（促排）；二甲双胍 metformin 改善胰岛素抵抗。
- 鉴别：非经典型 CAH（先天性肾上腺皮质增生症 congenital adrenal hyperplasia，病源在肾上腺，17-OHP↑）、卵巢泡膜增生症（绝经后，雄激素更高）。

🏥 PCOS 代谢/内分泌深挖 → 见跨学科区。

## B5. 盆腔包块与卵巢肿瘤（Pelvic mass & ovarian tumors）

> [!tip] 附件包块按绝经状态分流（第一步检查）
> - **绝经前 (premenopausal)**：**β-hCG + 超声**（首虑异位妊娠、卵巢扭转）。
> - **绝经后 (postmenopausal)**：**CA-125 + 超声**（首虑卵巢癌）。

- **异位妊娠 (ectopic pregnancy) 超声钥匙**：附件区孕囊 (adnexal gestational sac) + 周边血流 "ring of fire" 火环征（超声图见 [[完整笔记/专题笔记/OB_GYN/OB_GYN_孕期出血鉴别|孕期出血鉴别]] §2.2）。

- **卵巢扭转 (ovarian torsion)**：包块（常 >5 cm）+ **突发痛 + 恶心呕吐**（自主反射）；多普勒血流↓/消失（非 100% 敏感）→ **急诊手术解扭 (detorsion)**。
- **上皮性卵巢癌 (epithelial ovarian carcinoma)**：早期无症状 → 晚期腹水/腹胀/早饱（绝经后新发胃肠症状是隐匿表现）；标志物 **↑CA-125**；超声见实性 + 厚分隔 + 腹水；**禁穿刺活检** → 探查性剖腹 (exploratory laparotomy)。

**肿瘤标志物速记**

| 标志物 | 肿瘤 |
|---|---|
| 雌激素/抑制素 (estrogen/inhibin) | 颗粒细胞瘤 (granulosa) |
| β-hCG | 绒癌 (choriocarcinoma) |
| AFP | 卵黄囊瘤 (yolk sac) |
| 睾酮 (testosterone) | 支持-间质细胞瘤 (Sertoli-Leydig) |

## B7. 宫颈癌筛查与 HPV（Cervical screening）

> [!tip] 筛查年龄（照源文件）
> - **21 岁开始**；21–29 岁 **细胞学每 3 年**。
> - **≥30 岁**：细胞学 q3 年 **或** 细胞学 + HPV 共检 q5 年。
> - **65 岁停**（若既往充分阴性）。
> - HPV：**6/11 → 尖锐湿疣**；**16/18 → 癌**；疫苗 9–45 岁。

**异常结果分流（核心钥匙）**

```
ASC-US (意义不明非典型鳞状细胞)
   ├─ 21–24 岁 → 12 个月后复查细胞学（先等一年）
   └─ ≥25 岁 → 看 HPV：HPV(-) 常规；HPV(+) → 阴道镜 (colposcopy)
LSIL (低度鳞状上皮内病变)：≥25 岁 → 阴道镜；21–24 岁 → 同 ASC-US 等一年复查
ASC-H (不能排除高度病变) → 直接阴道镜 + 活检（不靠 HPV 分流）
HSIL → 立即阴道镜 + 活检（≥25 岁可 see-and-treat 直接 LEEP）
AGC (非典型腺细胞) → 阴道镜 + 宫颈管搔刮 (ECC) ± 内膜活检
CIN 1 → 观察；CIN 2/3 → LEEP（环形电切术 loop electrosurgical excision）/锥切 (cone biopsy)
CIN3(非孕) 切除后：6 个月 HPV 检测；切缘(+) → 再切；孕期 CIN3 → 监测 + 产后处理
怀疑浸润 → 诊断性锥切
```

- **宫颈癌**：无痛性**性交后出血 (postcoital bleeding)** 是经典钩子。

## B9. 外阴阴道炎 / PID（Vulvovaginitis / PID）

**阴道炎三联（按 buzzword）**

| 病原 | 钥匙 | 治疗 |
|---|---|---|
| 念珠菌 (Candida，最常见) | 白稠豆渣样 + 瘙痒，pH 正常 | 氟康唑 fluconazole / 唑类 |
| 细菌性阴道病 (BV, Gardnerella) | 鱼腥味 + 线索细胞 clue cells + pH↑ | **甲硝唑 metronidazole**（仅有症状才治） |
| 滴虫 (Trichomonas) | 活动滋养体 + 草莓样宫颈 | 甲硝唑（**伴侣同治**） |

## B10. 不孕（Infertility）

> [!tip] 评估时机 + 顺序
> 评估起点：**<35 岁未孕 ≥12 个月**；**≥35 岁 ≥6 个月**。
> 顺序"先男后女、先常见后侵入"：
> ① **男方精液分析 (semen analysis)** 最先（便宜高产出）。
> ② 女方三块：输卵管通畅 → **子宫输卵管造影 (HSG)**；排卵评估；卵巢储备（年龄大查 AMH/FSH）；宫腔 → 经阴超声 → 盐水灌注超声 / 宫腔镜。
> ⚠️ **甲减 (hypothyroidism) 是可逆的无排卵病因**：无排卵 + 体重增加 + 无多毛/无溢乳 → **先查 TSH**（别急着贴 PCOS）。

## B11. 更年期与 HRT / 骨质疏松（Menopause / Osteoporosis）

- **更年期 (menopause)**：均龄 ~51 岁；**FSH 先升**；血管舒缩症状（潮热）。
- **激素替代 (HRT)**：**已切子宫 → 仅雌激素 (estrogen-only)**；有子宫 → 雌 + 孕（孕激素防内膜癌）；**有 DVT/血栓风险 → 改 SSRI** 控潮热。
- **骨质疏松 (osteoporosis)**：**65 岁筛查（双能 X 线骨密度 DEXA/DXA, dual-energy X-ray absorptiometry）**；诊断 = **T 评分 ≤ -2.5** 或脆性骨折；一线**双膦酸盐 (bisphosphonates，如 alendronate)**（空腹 + 大量水 + 直立 1h；严重肾衰禁用；长期风险 = 下颌骨坏死/非典型骨折，**5 年后停**；2 年复查 DXA）。

## B12. 乳腺病变与乳腺癌评估（Breast）

- **良性**：纤维腺瘤 (fibroadenoma)、纤维囊性变 (fibrocystic)、导管内乳头状瘤 (intraductal papilloma)。
- **恶性钩子**：橘皮征 (peau d'orange)、**单侧血性溢液 (unilateral discharge)**、外上象限固定肿块。
- **可触及包块 workup（按年龄选影像）**：**<30 岁 → 超声**（年轻致密乳腺钼靶敏感度低）；**≥30 岁 → 钼靶 (mammography) ± 超声**。⚠️"摸着像良性"也须先影像分型、不能只观察；活检在影像之后（非一上来切除活检）；远亲弱家族史不改变年龄规则。
- **乳腺炎 (mastitis)**：金葡菌 → **继续哺乳** + 双氯西林 dicloxacillin。
- **男性乳腺癌**：Klinefelter / 肥胖 / 肝硬化（雌雄比异常）、BRCA → 乳腺钼靶 (mammography) → 多为浸润性导管癌（激素受体阳性）。

> [!tip] 乳腺癌筛查年龄框架（average-risk 平均风险）⭐
>
> | 人群 | 推荐 |
> |---|---|
> | average-risk 平均风险，**40–74 岁** | 定期乳腺钼靶 (periodic mammography)，每 1–2 年 |
> | **≥75 岁** + 预期寿命 (life expectancy) >10 年 | 个体化继续 |
> | 预期寿命短 / 严重衰弱 (frailty) | 筛查获益下降，可停 |
> | BRCA / 高危 | 加乳腺 MRI (magnetic resonance imaging) 策略 |
>
> **两个高频误读**：① "有慢病 (如 COPD 慢阻肺 / HTN 高血压 / osteoporosis 骨质疏松) = 寿命短 → 停筛"——错，要看**功能状态**（如能走几个街区、查体正常 = 预期寿命尚可）；② "for the rest of her life" 这类**绝对化**选项——75 岁后应个体化，不承诺终身。
> **年龄大 ≠ 立即停筛**；停筛个体化是 **75 岁后**，不是 71 岁。
>
> 关联错题：NBME11A Q155（71 岁 average-risk 功能稳定 → 续筛至 74；误选"按年龄停筛"）→ [[NBME11_错题本#^Q155]]

🧩 可触及乳房包块 workup（<30 岁超声 / ≥30 岁钼靶；"摸着良性也须先影像"）深挖 → [[完整笔记/专题笔记/OB_GYN/OB_GYN_乳房包块年龄分层检查路径|乳房包块年龄分层检查路径]]

## B13. 盆底脱垂 / 尿失禁（Prolapse / Incontinence）

> [!tip] 尿失禁三型决策（高频反差）
> - **压力性 (stress)**：咳/笑/运动漏尿 → Kegel/手术（尿道中段悬吊）。
> - **急迫性 (urge, 逼尿肌过度活动)**：尿急来不及 + 夜尿 → 膀胱训练 → **抗毒蕈碱 (antimuscarinic, 如 oxybutynin)**。
> - **充溢性 (overflow)**：持续滴尿 + 残余尿↑（如产后尿潴留）→ **导尿减压**（≠观察；抗毒蕈碱会加重）。

> [!tip] 盆腔器官脱垂 (POP, pelvic organ prolapse) 定位（一句诀：前壁膀胱、后壁直肠、宫颈才是子宫）
> - **前壁膨出 → 膀胱膨出 (cystocele)**（+ 常伴压力性尿失禁）。
> - **后壁膨出 → 直肠膨出 (rectocele)**（+ 便秘/排便困难）。
> - **顶部膨出 → 肠膨出 (enterocele)**。
> - **宫颈/子宫下移过处女膜环 → 子宫脱垂 (uterine prolapse)**。
> 背景 = 多产 + 绝经；治疗 Kegel/子宫托 (pessary)/局部雌激素 → 手术修补。⚠️ 子宫内翻 (uterine inversion) 是产后急症，别与慢性脱垂混。

> [!info]- 子宫位置 (version/flexion) 等基础妇科解剖为 low-yield，图已下沉专题 → [[完整笔记/专题笔记/OB_GYN/OB_GYN_盆腔器官脱垂定位与症状|盆腔器官脱垂定位与症状]]

## B14. 避孕（Contraception）

> [!tip] 有效性 + 首选逻辑
> **LARC（长效可逆避孕 long-acting reversible contraception：宫内节育器 IUD / 皮下埋植 implant）最有效**，多数人首选。避孕套 (condom) 是唯一显著降性传播感染 (STI) 的方法。

- **复方雌孕激素 (combined estrogen-progestin) 禁忌（背）**：静脉血栓 (VTE) 史/易栓症、**先兆偏头痛 (migraine with aura)**、**≥35 岁吸烟**、控制不良高血压、产后 <3 周、活动性乳腺癌、活动性肝病。
- **孕激素单方 (progestin-only) / 左炔诺孕酮 IUD (LNG-IUD) 优势**：无雌激素 → **VTE/易栓/哺乳期/产后早期可用**；LNG-IUD 减经量 → 适合异常子宫出血 (AUB)/贫血/腺肌病。
- **IUD 禁忌**：不明原因阴道出血、活动性盆腔感染（两类共有）；铜 IUD + 肝豆状核变性 (Wilson 病)；孕激素 IUD + 乳腺癌。
- **紧急避孕 (emergency contraception)**：**铜 IUD (copper IUD) ≤5 天最强**；乌利司他 ulipristal ≤5 天；左炔诺孕酮 levonorgestrel (Plan B) ≤72h（越早越好）。机制 = 延迟排卵。
- **醋酸甲羟孕酮针 (DMPA, depot medroxyprogesterone acetate)**：体重增加 + 可逆性骨密度↓。

## 复习自测

> [!question]- Q1. 孕 32 周高血压孕妇突发剧烈腹痛 + 暗红色阴道出血 + 子宫强直压痛，胎心见复发晚期减速。最可能诊断 + 首要病理并发症？
> 胎盘早剥 (abruptio placentae)；剥离处释放组织因子 → 启动外源凝血 → **DIC**。记忆钩：**Hypertension + Painful bleeding = Abruption → DIC**。

> [!question]- Q2. 子痫前期患者用硫酸镁防抽搐，出现腱反射消失 + 嗜睡。下一步？
> 提示镁中毒（areflexia = 中度）。**停镁 + IV 葡萄糖酸钙 (calcium gluconate)**。

> [!question]- Q3. 50 岁绝经后女性新发便秘、腹胀、早饱，结肠镜阴性。第一步评估盆腔？标志物？
> **盆腔超声**（首选、价廉）；标志物 **CA-125**（上皮性卵巢癌隐匿表现 = 胃肠症状）。绝经后附件包块分流 = CA-125 + 超声。

> [!question]- Q4. 23 岁女性 ASC-US，HPV 状态如何影响下一步？若为 22 岁呢？
> ≥25 岁：HPV(+) → 阴道镜，HPV(-) → 常规；**21–24 岁不论 HPV 一律 12 个月后复查细胞学（先等一年）**。

> [!question]- Q5. 中年女性间断漏尿，与咳笑无关 + 夜尿 + 来不及上厕所，膀胱训练无效，PVR 正常。最佳治疗方向？与充溢性失禁治疗有何相反点？
> 急迫性失禁 → **抗毒蕈碱 (antimuscarinic)**。充溢性（潴留/滴尿、PVR↑）反而**禁用抗毒蕈碱**（加重潴留），应导尿减压。方向相反是高频陷阱。

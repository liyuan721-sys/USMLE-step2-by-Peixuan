# 预防三级别与筛查 (Levels of Prevention & Screening)

> [!info] 一句话定位
> **看时间轴：病前 / 病早 / 病后 → 一级 / 二级 / 三级。**
> 一级预防 (primary prevention, 一级预防) = 疾病**发生前**减少风险暴露；二级 = 早期**筛查**发现；三级 = 已病减并发症 / 康复；四级 (quaternary, 四级预防) = 避免**过度医疗**。
> 公共卫生题（市政 / 学校 / 社区关键词）→ 优先**结构性建成环境干预**，不是个体治疗、也不是短期行为激励。

---

## 一、核心框架：预防四级别 (Levels of Prevention)

| 层级 | 定义（时间轴）| 目标人群 | 经典例子 |
|---|---|---|---|
| **一级 Primary** | 疾病发生**前**减少暴露 / 降低发病率 (incidence↓) | 健康全人群 | 疫苗 vaccination；控烟政策 tobacco policy；**建成环境改造 built environment**（步道 / 自行车道）；安全带 / 头盔法；加碘盐；安全性行为教育 |
| **二级 Secondary** | 疾病**早期**（无症状 / 临床前期）筛查发现 | 无症状高危人群 | 癌症筛查（乳腺 mammography 钼靶 / 宫颈 Pap (Papanicolaou) / 结直肠 colonoscopy 结肠镜）；血压 / 血脂 / 身体质量指数 (body mass index, BMI) 测量；新生儿筛查；结核菌素试验 (purified protein derivative, PPD) 筛查 |
| **三级 Tertiary** | 已确诊病，**减并发症 / 致残 / 复发**、康复 | 已患病人群 | 糖尿病足护理 / 教育；心梗后心脏康复 cardiac rehab；减重手术 bariatric surgery；卒中后理疗 |
| **四级 Quaternary** | 避免**过度诊断 / 过度治疗**（over-medicalization）| 被过度医疗风险人群 | 不给自限病开抗生素；停掉无获益的筛查（高龄停癌筛）；去处方 deprescribing |

> [!tip] 记忆钩子
> **Primary 防"得病"、Secondary 防"病进展"、Tertiary 防"残/死"、Quaternary 防"过度医疗"。**
> 时间轴一条线：**暴露 →（一级）→ 发病 →（二级早筛）→ 有症状 →（三级管理）→ 结局**。

## 二、关键判别维度（NBME 出题角度）

### 2.1 时间轴：病在不在？得没得？

```
风险暴露期        临床前期(无症状)      临床期(有症状)
  │                  │                    │
  ▼                  ▼                    ▼
[一级] 减暴露    [二级] 早期筛查      [三级] 减并发症/康复
疫苗/控烟/步道   癌筛/血压/BMI测     康复/教育/手术
```

- 题眼写 **"prevent future disease / before disease occurs / reduce incidence"** → **一级**
- 题眼写 **"detect early / asymptomatic / screening"** → **二级**
- 题眼写 **"already diagnosed / reduce complications / rehabilitation"** → **三级**

### 2.2 个体 vs 群体（公共卫生题专属）

> [!warning] 公共卫生主任 / 市政 / 学校 / 社区 关键词
> 一旦 stem 出现 **"public health official / city / community / population"** + "prevent future..."，
> 答案偏好 **全人群、结构性、可持续的建成环境 (built environment) 干预**，
> **不是**个体门诊治疗（手术 / 一对一教育），**也不是**短期行为激励（减肥比赛发奖）。

| 选项类型 | 层级 | 是否首选 | 理由 |
|---|---|---|---|
| 修步道 / 自行车道（建成环境）| 一级 | ✅ **首选** | 结构化、全人群、长期可持续 |
| 全校测 BMI + 干预 | 二级 | ✗ | 是"筛查"= 二级，非一级 |
| 减重手术中心 | 三级 | ✗ | 治已病 |
| 糖尿病教育员 | 三级 | ✗ | 已病管理 |
| 全市减肥比赛 + 发奖 | 沾一级但**短期不可持续** | ✗ **最强干扰** | 行为激励 < 建成环境（NBME 偏好结构性长期干预）|

## 三、二级预防深入：筛查 (Screening)

### 3.1 一个好筛查项目的条件（Wilson criteria 简化版）

1. 疾病**患病率够高** + 危害大（值得筛）
2. 有**临床前可识别期**（早治有效窗口）
3. 测试**敏感度 Sensitivity 高**（怕漏诊 → **SnNout**：Sn 高 → 阴性可排除）+ 可接受、便宜、安全
4. **早期治疗能改善结局**（否则只是早知道，不延寿）

> [!tip] 筛查 = 敏感度优先
> 筛查目的是"宁可错抓不可漏"→ 选 **高 Sensitivity**（阴性放心排除）；确诊 (confirmatory) 才转高 Specificity（**SpPin**：Sp 高 → 阳性可确诊）。

### 3.2 评估筛查项目的 4 大偏倚（高频陷阱）

| 偏倚 | 机制 | 一句话识别 |
|---|---|---|
| **Lead-time bias 领先时间偏倚** | 早诊断 → "存活期"延长，但实际死亡时间不变 | "筛查后平均生存 3→5 年" → **别说筛查有效**，可能只是知道得早 |
| **Length-time bias 长度时间偏倚** | 筛查更易抓到进展慢的惰性病灶 | 筛查组"预后好"是因为抓到的本就是惰性病 |
| **Overdiagnosis 过度诊断** | 发现永不进展的病变 → 过度治疗 | 筛出一堆不会致死的"癌" |
| **Selection bias 选择偏倚** | 愿来筛查者本身更健康 | 自愿筛查人群 ≠ 一般人群 |

> [!danger] 判定筛查"真获益"的唯一硬指标
> 只有 **疾病特异死亡率 (disease-specific mortality)↓** 才能证明筛查真有效；
> "生存期延长 / 5 年生存率↑" 都可能是 **lead-time / length-time / overdiagnosis** 的假象。

## 五、高频陷阱速记

> [!danger] 易混点
> - **筛查 = 二级，不是一级**（一级是病前减暴露，筛查是病早发现）。
> - **个体治疗手段（手术 / 教育）= 三级**，不要当成"预防"群体策略。
> - **行为激励（比赛发奖）≠ 建成环境**：NBME 公共卫生题偏好结构性、可持续、全人群干预。
> - **戒烟 (smoking cessation) 看人群**：对**没病的人**戒烟 = 一级；对**已患慢性阻塞性肺疾病 (chronic obstructive pulmonary disease, COPD) / 已心梗**的人戒烟 = 三级（防并发症）。同一动作、层级随人群变。
> - **筛查"延长生存期" ≠ 有效**：先排 lead-time / length-time bias，看疾病特异死亡率。

## 六、复习自测

> [!question]- Q1. 某市肥胖致残率偏高，公共卫生主任想预防未来肥胖并发症，对全人群最合适的一级预防策略是？
> **建成环境改造（修步道 / 自行车道）** = 结构性、全人群、可持续。
> 全校测 BMI = 二级（筛查）；减重手术中心 / 糖尿病教育员 = 三级；减肥比赛发奖 = 行为激励、短期不可持续（最强干扰）。

> [!question]- Q2. 戒烟咨询对一个健康吸烟者 vs 对一个已确诊 COPD 患者，分别属于几级预防？
> 健康吸烟者 = **一级**（防发病）；已 COPD 患者 = **三级**（防并发症 / 病情进展）。同一动作层级随人群（病前 / 病后）而变。

> [!question]- Q3. 一项筛查使乳腺癌"平均生存期从 3 年延长到 5 年"，能否说明筛查有效？应看什么指标？
> 不能 — 可能是 **lead-time bias**（早诊断使存活期"看起来"变长，死亡时间不变）。判定真获益的唯一硬指标是 **疾病特异死亡率 (disease-specific mortality)↓**。

> [!question]- Q4. 对 90 岁老人停止常规结直肠癌筛查，属于哪一级预防理念？
> **四级预防 (quaternary)** — 避免过度诊断 / 过度治疗（预期寿命短，筛查弊大于利）。

# _衍生_晕厥鉴别决策树（Syncope Differential & Decision Tree）

> [!info] 本笔记定位
> **晕厥 / 短暂意识丧失（TLOC）的完整鉴别框架**。USMLE Step 2 CK 高频考点，跨**心内 + 神经 + 急诊**。
> 
> **核心思维**：
> 1. **前驱症状 + 触发** = 晕厥分类的两大锚点
> 2. **Orthostatic 不是单一病**，是**容量型 vs 神经源型**两大机制
> 3. 关键鉴别 = **HR 反应**（站起来 HR↑≥30 = 容量型；HR 几乎不动 = 神经源型）
> 
> **来源**：错题 Q21638（Volume-mediated OH，摔跤运动员 weight-cutting）+ 后续 vasovagal / 心源性晕厥 / DM 自主神经病等错题积累

---

## Part 0. 核心方法论

### 0.1 晕厥 = 短暂意识丧失（TLOC）

定义：突发、短暂、自行恢复的意识丧失，由全脑灌注一过性下降所致。

> [!warning] 晕厥 ≠ 癫痫 ≠ 跌倒 ≠ 心脏骤停
> - **晕厥**：自行恢复 + < 1 min + 无 postictal
> - **癫痫**：有 aura + 抽搐 + postictal confusion
> - **跌倒**（无意识丧失）：意识清楚
> - **心脏骤停**：不自行恢复，需 CPR

### 0.2 鉴别 4 大类核心锚点

> [!success] 看到晕厥题 → 先问 4 个问题
> 1. **前驱症状有吗？** → 无 = 心源性；有头晕 = orthostatic / vasovagal
> 2. **触发因素？** → 站起来 = OH；情绪 / 疼痛 = vasovagal；运动 / 卧位 = 心源性
> 3. **恢复后清醒还是 confusion？** → confusion = 癫痫
> 4. **HR 怎么反应？**（OH 内部分型）→ HR↑≥30 = 容量型；HR 不动 = 神经源型

## Part 1. 晕厥 4 大类完整对照 ⭐⭐⭐

| 维度             | **心源性**               | **反射性（Vasovagal）**     | **Orthostatic**           | **癫痫**                    |
| -------------- | --------------------- | ---------------------- | ------------------------- | ------------------------- |
| **前驱症状**       | **无**（突发）             | 有（恶心 / 出汗 / 视暗）        | 有（**头晕** / 视暗）            | Aura                      |
| **触发**         | 运动 / 卧位 / 无诱因         | 情绪 / 疼痛 / 站久 / 排尿 / 咳嗽 | **站起来** ⭐                 | 闪光 / 睡眠剥夺                 |
| **持续**         | 数秒                    | < 1 min                | < 1 min    </fo</mark>nt> | 数分钟                       |
| **恢复**         | 立即清醒                  | 立即清醒                   | 立即清醒                      | **Postictal confusion** ★ |
| **舌咬 / 失禁**    | 罕见                    | 罕见                     | 罕见                        | **常有**                    |
| **HR**         | 异常                    | **缓慢**（迷走）             | 通常 ↑（容量型脱水代偿）/ 不动（神经源型）   | 不定                        |
| **心脏体检 / ECG** | 异常（杂音 / 长 QT / 宽 QRS） | 正常                     | 正常                        | 正常                        |

## Part 2. 心源性晕厥（最危险，必须排除）

### 2.1 病因分类

| 类型 | 病因 | 关键提示 |
|---|---|---|
| **心律失常** | LQTS / Brugada / SSS / Mobitz II / 完全 AVB / VT / VF / WPW | ECG 异常 / 家族猝死 / 突发 |
| **结构性** | **HCM** / AS / **HOCM** / PH / RV 异常 | 杂音 / 运动诱发 ⭐ / 心衰 |
| **缺血性** | MI / 严重冠心病 | 胸痛 + ECG 改变 |
| **机械梗阻** | 大块 PE / Tamponade / 主动脉夹层 | 急性 + 高危机制 |

### 2.2 心源性高危信号（San Francisco Syncope Rule "CHESS"）

> [!danger] 见任 1 条 → 入院评估
> - **C**HF history
> - **H**ematocrit < 30%
> - **E**CG abnormal
> - **S**hortness of breath
> - **S**ystolic BP < 90 mmHg

### 2.3 运动诱发晕厥红旗

> [!danger] 运动中晕厥 = 心源性 until proven otherwise
> - **HCM**（年轻运动员猝死最常见原因）
> - **LQT1**（"游泳 1"，运动诱发）
> - **Brugada**（发热 + V1-V3 ST↑）
> - **CPVT**（运动诱发多形 VT）
> - **AS**（老人）/ HOCM（年轻）
> - **Commotio cordis**（胸前钝击 R-on-T → VF）

详见 [[完整笔记/专题笔记/_衍生_Vaughan_Williams抗心律失常药对照]] Part 11.4 通道病鉴别 + [[完整笔记/专题笔记/_衍生_ACLS急救决策树]]

## Part 3. 反射性 / 神经介导性晕厥（Vasovagal）

### 3.1 三种反射性晕厥

| 类型 | 触发 | 机制 |
|---|---|---|
| **Vasovagal**（最常见）| 情绪 / 疼痛 / 见血 / 长时间站立 | 迷走亢进 → BP↓ + HR↓ |
| **Carotid sinus syncope** | 转头 / 紧领带 / 刮胡子 | 颈动脉窦压力 → 心动过缓 |
| **Situational** | 排尿 / 咳嗽 / 排便 / 吞咽 | 特定动作诱发迷走 |

### 3.2 Vasovagal 经典 stem

- 年轻女 + 见血 / 急诊抽血 / 长时间站立 → **前驱**（恶心 + 出汗 + 视暗 + 耳鸣）→ 短暂晕厥 → 立即清醒

### 3.3 处理

- **立即处理**：平卧 + 抬腿
- **预防**：避诱因 + 增液 + 增 Na + 体位训练（counter-pressure 动作如交腿）
- **药物**（顽固）：**β-blocker**（争议）/ Midodrine / Fludrocortisone

## Part 4. Orthostatic Hypotension（OH）两型 ⭐⭐⭐

### 4.1 诊断标准

```
床旁测试：
1. 仰卧 5 min → 测 BP + HR
2. 站立 1 / 3 min → 复测

诊断阳性：
- SBP ↓ ≥ 20 mmHg 或
- DBP ↓ ≥ 10 mmHg

分型（关键）：
- 容量型：HR ↑ ≥ 30 bpm（反射保留尝试代偿）
- 神经源型：HR 几乎不变 / ↑ < 10-15 bpm
```

### 4.2 两型核心鉴别 ⭐⭐⭐

| 维度 | **Volume-mediated**（容量型）| **Neurogenic / Autonomic**（神经源型）|
|---|---|---|
| **机制** | 血容量不足 | 反射弧受损 |
| **血容量** | 低 | 正常 |
| **反射弧** | 完好 | 受损 |
| **HR 反应** | **↑ ≥ 30 bpm** ⭐ | **不变 / ↑ < 10-15 bpm** ⭐ |
| **Hct** | **↑**（浓缩）| 正常 |
| **BUN/Cr** | **> 20**（prerenal）| 正常 |
| **K** | 可低 | 正常 |
| **黏膜** | **干燥** | 正常 |
| **皮肤** | 干 | 出汗异常（无汗 / 多汗）|
| **仰卧 BP** | 偏低 | **可正常 / 仰卧 HTN（反常）** |
| **附带症状** | 脱水征 | **胃轻瘫 / 便秘 / 阳痿 / 神经源膀胱** |
| **典型患者** | 年轻 / 急性 / 运动员 | **60yo DM / Parkinson / MSA / 高龄** |
| **病程** | 急性（小时-天）| **慢性进行性**（月-年）|
| **治疗反应** | **补液后快速改善** | **顽固，需药物** |

### 4.3 类比理解

> [!success] OH 两型本质类比
> - **容量型 = "池子里水不够"**：年轻 / 急性 / HR 努力补救（但失败）
> - **神经源型 = "池子里水够但水泵开关坏了"**：老年 / 慢性 / HR 不动

### 4.4 OH 病因 5 类速记 "湿干自主药老"

| 类别 | 病因 |
|---|---|
| **湿**（容量丢失）| 呕吐 / 腹泻 / 出汗 / 失血 / 烧伤 / **weight-cutting**（Q21638）|
| **干**（容量不补）| 摄入不足 / 老人 / 痴呆患者 |
| **自主**（autonomic）| **DM** / Parkinson / **MSA** / **PAF** / Lewy body / Amyloidosis / 副肿瘤 / HIV / 高龄 / 脊髓损伤（T6 以上）|
| **药**物 | **α 阻滞**（terazosin）/ TCA / 利尿剂 / 抗 HTN（β / CCB）/ 抗精神病 |
| **老**（内分泌 / 混合）| **PAI**（cortisol + aldo 双缺）/ 餐后 OH / HF / AS |

### 4.5 OH 治疗阶梯（按型不同）

| 型 | 治疗 |
|---|---|
| **容量型** | ① 补液 + 补电解质 ② 停"作恶"药物 ③ 治根本病因 ④ 体位渐变 |
| **神经源型（非药）** | ① 体位渐变（30 秒规则）② 增 Na + 水（2-3 L/天）③ **弹力袜**（thigh-high）④ 床头抬高 10-20° ⑤ 避热水浴 / 大餐 / 酒精 |
| **神经源型（药）** | **Fludrocortisone**（保 Na 增容量）+ **Midodrine**（α₁ 激动）⭐ + Droxidopa（NE 前体）+ Pyridostigmine |

> [!danger] 关键反陷阱
> - **OH 不全是脱水** — 神经源型容量正常，反射坏了
> - **HR ↑ 不排除 OH** — 反而是容量型特征
> - **HR 不变不排除 OH** — 是神经源型诊断特征
> - **仰卧 BP 正常 / HTN 不排除 OH** — 仰卧 HTN 反而支持神经源型（反常征）

### 4.6 DM 自主神经病五联（神经源 OH 经典 stem）

```
长期 DM（> 10 年）+ HbA1c 高 + 以下任何组合：
1. 心血管：OH + 静息心动过速 + 心率变异性消失 + **仰卧 HTN**
2. 胃肠：胃轻瘫（早饱 / 餐后腹胀）+ 便秘 / 腹泻交替
3. 泌尿生殖：神经源性膀胱（残余尿 / 反复 UTI）+ **阳痿 (ED)**
4. 出汗：无汗症 / 异常出汗
5. 瞳孔：反应迟钝

→ 见 DM 患者 + OH + 任何 2 项 = 自主神经病高度怀疑
```

## Part 5. 癫痫鉴别（vs 晕厥）

| 维度 | **晕厥** | **癫痫** |
|---|---|---|
| **前驱** | 头晕 / 恶心 / 视暗（自主神经）| **Aura**（嗅觉 / 视觉异常）|
| **持续** | < 1 min | 数分钟 |
| **抽搐** | **罕见**（短暂缺氧抽搐 ≤ 几秒）| **常有** + 持续 |
| **舌咬** | 罕见（即便有也是舌尖）| **常有**（舌侧）|
| **大小便失禁** | 罕见 | 常有 |
| **恢复** | **立即清醒** | **Postictal confusion** ★（数分钟-小时）|
| **诱因** | 站起 / 情绪 / 疼痛 | 闪光 / 睡眠剥夺 / 停药 |
| **EEG** | 正常 | **异常** |

### 5.1 Conversion disorder（鉴别）

- 持续意识丧失 > 5 min（晕厥很少超过 1 min）
- 心理诱发
- "La belle indifférence"（异样平静）
- 排除诊断

## Part 6. 危险信号 + 入院指征

### 6.1 San Francisco Syncope Rule（CHESS — 入院指征）

> [!danger] 任 1 条 → 入院观察
> - **C**HF history
> - **H**ematocrit < 30%
> - **E**CG abnormal
> - **S**hortness of breath
> - **S**ystolic BP < 90 mmHg

### 6.2 高危特征（红旗）

- **运动中晕厥**（心源性 until proven otherwise）
- **卧位晕厥**（心源性，无反射性可能）
- **家族猝死史**
- **结构性心脏病史**（HCM / AS / HF）
- **ECG 异常**（长 QT / Brugada / 宽 QRS / AVB）
- **心悸前驱**（心律失常）
- **无前驱突发**（心源性）

## Part 7. 诊断流程

```
晕厥患者评估：
        │
        ▼
Step 1: 详细病史 + 体检
   - 前驱症状（头晕？恶心？aura？）
   - 触发（站起？情绪？运动？卧位？）
   - 持续时间 + 恢复
   - 体位性 BP / HR（床旁 OH 测试）
        │
        ▼
Step 2: ECG（必做）
   - 长 QT / Brugada / 宽 QRS / AVB / 缺血?
        │
        ▼
Step 3: 实验室（按需）
   - CBC（贫血 → CHESS 之 H）
   - BMP（电解质）
   - Glu（低血糖）
   - hCG（年轻女 → 妊娠相关）
   - Troponin（怀疑 ACS）
        │
        ▼
Step 4: 进阶（按怀疑）
   - **Echo** — 怀疑结构性（HCM / AS / 先心病）
   - **Tilt table** — 顽固 vasovagal 或 OH
   - **Holter / loop recorder** — 间歇性心律失常
   - **EEG** — 怀疑癫痫
   - **CT 头** — 局灶神经体征 / 头外伤
        │
        ▼
Step 5: 风险分层 → 入院（CHESS）or 出院 + 随访
```

## Part 8. 运动员晕厥（USMLE 高频）⭐

### 8.1 摔跤 / 搏击运动员 3 类考点

| 考点 | 临床表现 |
|---|---|
| **Weight-cutting 致脱水** | Orthostatic + 脱水四联 + 低 K（**Q21638**）|
| **运动中猝死** | HCM / Brugada / LQTS / **Commotio cordis**（胸前撞击 → R-on-T → VF）|
| **AAS（合成代谢类固醇）滥用** | 男化 / 不孕 / HCM / 肝瘤 / 痤疮 / 心衰 |
| **过度训练综合征** | 慢性疲劳 + 表现下降 + 抑郁 |
| **Female Athlete Triad** | 月经异常 + 骨密度低 + 进食障碍 |

### 8.2 运动员心电图特征（生理性 vs 病理性）

| 生理性（可接受）| 病理性（需进一步评估）|
|---|---|
| 窦缓（≥ 30 bpm）| 严重窦缓（< 30）+ 症状 |
| 1° AVB / Mobitz I | Mobitz II / 完全 AVB |
| 早期复极 | **Brugada pattern** |
| 左室高电压 | **HCM**（深 Q + LV 肥厚）|
| 早搏（罕见）| 复杂室速 / 多形 VT |
| 窦性心律不齐 | **长 QT**（QTc > 480）|

### 8.3 PPE（赛前体检）红旗

> [!danger] 任 1 条 → 心脏科评估
> - 运动中或刚停止后晕厥
> - 不明原因运动相关胸痛 / 呼吸困难
> - 家族猝死史（< 50 岁）
> - 已知心律失常综合征 / HCM
> - 既往 Marfan 表现（高瘦 + 长指）

## Part 9. Memory Hook 集合

### 9.1 晕厥鉴别 "前 5 后" 速记

> [!success] "前 5 后"
> **前**驱症状 → 反射性或 orthostatic（**有前驱**）
> 突发**无**前驱 → 心源性
> **5** 分钟以上无意识 → conversion disorder
> 苏醒**后** confusion → 癫痫（postictal）

### 9.2 OH 病因 5 类口诀

> **"湿干自主药老"** = 容量丢失 / 摄入不足 / 自主神经病 / 药物 / 内分泌混合

### 9.3 OH 两型类比

> **"水池水不够（容量）vs 水泵坏（神经源）"**

### 9.4 神经源 OH 反常征

> **"DM + OH + 仰卧 HTN + 胃轻瘫 + ED" = 自主神经病五联**

### 9.5 心源性晕厥红旗

> **"无前驱突发 + 运动 / 卧位 + ECG 异常 + 家族猝死"**

### 9.6 选对题深挖机制方法论 ⭐⭐⭐

> [!success] NBME 230→250 关键
> 直觉对 ≠ 机制懂。**选对的题尤其要问**：
> 1. 为什么答案是这个？
> 2. 如果 stem 改一个细节（年龄 / 实验室 / 体征），答案变吗？
> 3. 这个诊断的"亚型 / 分型"是什么？
> 4. 每种亚型的鉴别要点 + 治疗差异？

## Part 10. 跨学科陷阱

| 学科 | 陷阱 | 关键点 |
|---|---|---|
| **心内** | 运动 / 卧位晕厥 → 默认心源性，不要先归 vasovagal | HCM / LQTS / Brugada / AS |
| **神经** | DM 患者晕厥 → 想低血糖 + 想自主神经病 | 慢性 DM + ED + OH = 自主神经病 |
| **内分泌** | PAI 致 OH | adrenal hypovolemia + cortisol/aldo 双缺 |
| **肾** | 利尿剂老人 + 晕厥 | 容量型 OH + 低 K |
| **psych** | conversion disorder vs 真晕厥 | 持续 > 5 min + 心理诱发 + La belle indifférence |
| **儿科** | 儿童运动晕厥 → HCM | 家族史 + 心音 + ECG |
| **OB** | 妊娠 + 晕厥 | 仰卧位低血压综合征（IVC 受压）+ 早期妊娠出血（异位 / 胎盘早剥）|
| **GI** | 排便 / 排尿后晕厥 | Situational vasovagal |
| **外科** | 摔跤 / 搏击运动员 | Weight-cutting 脱水 / Commotio cordis |

## Part 11. 高频陷阱总结

| 陷阱 | 正确认知 |
|---|---|
| ❌ OH 就是脱水 | ✓ 神经源型容量正常；不是所有 OH 都补液能好 |
| ❌ HR ↑ 排除 OH | ✓ HR ↑ 反而是容量型特征 |
| ❌ HR 正常排除 OH | ✓ 神经源型 HR 几乎不变才是诊断特征 |
| ❌ 仰卧 BP 正常 / HTN 排除 OH | ✓ 仰卧 HTN 反而支持神经源型（反常征）|
| ❌ DM 患者晕厥 = 低血糖 | ✓ 也要考虑自主神经病致 OH（尤其 DM + ED + 胃轻瘫）|
| ❌ Vasovagal = 心动过速 | ✓ Vasovagal 是迷走亢进 → **HR 慢** |
| ❌ 所有运动后晕厥 = vasovagal | ✓ 必须先排心源性（HCM / LQTS）|
| ❌ 卧位晕厥可能是 vasovagal | ✓ 卧位晕厥 = 心源性 until proven otherwise |
| ❌ ECG 正常排除心源性 | ✓ 间歇性心律失常需 Holter / loop recorder |
| ❌ OH 治疗都用 midodrine | ✓ 容量型补液就够；midodrine 是神经源型才需要 |
| ❌ 选对题不深挖机制 | ✓ 直觉对 ≠ 机制懂；变形 stem 会失误 |
| ❌ Postictal confusion 几秒就好 | ✓ Postictal 数分钟-小时；不是几秒 |

# 🧬 Turner 综合征专题（Turner Syndrome / 45,X）

> [!info] 为什么单独建专题
> Turner 综合征是 USMLE Step 2 的**跨科超高频考点**，涉及：
> - **妇科**：原发性闭经 / 性腺功能减退（最高频）
> - **儿科**：身材矮小 + 表型识别（高频）
> - **心内**：Bicuspid AV + Coarctation + Aortic dissection ⭐
> - **内分泌**：甲状腺、糖尿病、骨质疏松
> - **肾**：马蹄肾
> - **眼**：弱视、斜视
> - **听力**：传导性 + 感音神经性聋
>
> 出题率：约 5-8 题/UWorld

---

## 一、核心遗传机制

### 1.1 染色体异常

```
正常女性：46,XX
   ↓
Turner: 单条 X 染色体（缺失或异常）
   ↓
分类：
├─ 45,X (Monosomy X)         50-60%（最常见）
├─ 45,X / 46,XX (Mosaic)     15-25%（表型最轻）
├─ 46,X,i(Xq) (Isochromosome) 10-15%
├─ 46,X,r(X) (Ring X)        5%
└─ 46,XY mixed gonadal dysgenesis 罕见
```

### 1.2 关键点

> [!warning] 与其他染色体病的鉴别
> - **不是**因为**母**亲年龄大（不像 Down/Edwards/Patau）
> - 大部分是**散发**（sporadic）
> - **没有遗传性**（一般不传给后代，因不孕）

### 1.3 病理生理

```
单条 X → SHOX 基因剂量减半
   ↓
├─ 骨骼发育异常 → 矮小 + 多种骨骼畸形
└─ 性腺发育异常 → "Streak ovaries"（条索状卵巢）
       ↓
    Estrogen ↓↓ + FSH/LH ↑↑（高促性腺激素性性腺减退）
       ↓
    无月经 / 无第二性征 / 不孕
```

## 二、Stem 识别（USMLE 经典模式）

> [!warning] 看到这些组合立刻想到 Turner

### 🥇 经典 Stem 模式 1：**青春期女孩 + 矮小 + 无月经**

```
15-16 岁女孩
+ 身高 < 第 3 百分位（通常 < 150 cm）
+ 原发性闭经（primary amenorrhea）⭐
+ 未发育的乳房（Tanner 1-2）
+ 阴毛和腋毛正常（adrenarche 不依赖卵巢）
```

→ **Turner until proven otherwise**

### 🥈 经典 Stem 模式 2：**新生儿 + 水肿 + 心脏问题**

```
新生儿期：
+ **Lymphedema of hands/feet**（手脚淋巴水肿）⭐
+ **Nuchal cystic hygroma**（颈部水囊瘤）
+ Cardiac defect（coarctation / bicuspid AV）
+ Wide-spaced nipples
+ Short stature growth chart
```

→ Karyotype 立刻确诊

### 🥉 经典 Stem 模式 3：**急性主动脉夹层 in young female**

```
18-30 岁女性
+ 急性胸痛 / 背痛
+ Hx: short stature + amenorrhea
+ 体征：上下肢血压差异 / unequal pulses
+ 既往诊断 bicuspid AV / coarctation
```

→ **Aortic dissection in Turner** ⭐⭐⭐

### 🎯 经典 Stem 模式 4：**孕前评估**

```
24 岁 Turner 患者计划怀孕
+ Bicuspid AV
+ Aortic root 25 mm
+ 想用 ART
```

→ 考点：**aortic dissection 风险**，需 Echo + MRI

## 四、跨系统受累（USMLE 必考）

### 🫀 心血管系统（最重要！）

> [!danger] Turner = **心血管高风险**，约 50% 患者有 CV 异常

| 异常 | 频率 | 临床意义 |
|------|------|----------|
| **Bicuspid aortic valve** | **30-50%** ⭐⭐⭐ | 早发 AS/AR + IE 风险 |
| **Coarctation of aorta** | **15-30%** ⭐⭐⭐ | 上下肢 BP 差异 |
| **Aortic root dilation** | 30% ⭐⭐ | 夹层风险 |
| **Aortic dissection** | <2% but mortality 极高 | **妊娠期风险 ↑↑** |
| HTN | 30% | 多因素（CoA / 肾 / 内分泌）|
| Lymphatic anomalies | 常见 | 反映为新生儿淋巴水肿 |
| PAPVR (partial APVR) | 13% | Sinus venosus ASD 关联 |

#### Aortic Dissection 关键点 ⭐⭐⭐

```
Turner + Bicuspid AV / CoA + Aortic root dilation
    ↓
Aortic Size Index (ASI) > 25 mm/m² = 高风险
    ↓
妊娠 hypervolemia → 进一步压力
    ↓
急性夹层 = 高死亡率
```

**孕前评估必查**：
- ✅ Echocardiogram
- ✅ Cardiac MRI（金标准）
- ✅ ASI 计算

**详见** → [[心内]]

---

### 🤰 妇科 / 生殖系统

> [!warning] **原发性性腺功能减退**

#### 性腺
- **Streak ovaries**（条索状卵巢）
- 卵泡早期闭锁（fetal 时期 ↓↓）
- → **永久不孕**（除少数 mosaic）

#### 激素
- **Estrogen ↓↓**
- **FSH ↑↑↑**
- **LH ↑↑**
- 模式："Hypergonadotropic hypogonadism"（高促性腺激素性性腺减退）

#### 临床表现
- **Primary amenorrhea**（90%）⭐
- 无第二性征发育
- 阴毛 / 腋毛**正常**（adrenarche 不受影响）
- 无生育能力（除 5-10% mosaic）

#### 生育选择
- **Donor egg + IVF**（理论可行）
- ⚠️ **但**：心血管风险高，妊娠死亡率上升
- **必须**先做 Echo + cardiac MRI

**详见** → [[GYN]]

### 🦴 内分泌

| 异常 | 频率 | 备注 |
|------|------|------|
| **Hypothyroidism**（Hashimoto's）| **30-50%** ⭐ | 每年查 TSH |
| Type 2 DM | 25% | 成年后 |
| Type 1 DM | 增加 | |
| **Osteoporosis** | 50% | Estrogen 缺乏 + 骨结构异常 |
| GH deficiency | 部分 | SHOX 相关 |

→ **每年必查**：TSH, blood glucose, lipid panel, DEXA scan（成年）

### 🧠 肾脏

> [!info] 约 30-40% 有肾异常

| 异常 | 频率 |
|------|------|
| **Horseshoe kidney（马蹄肾）**⭐ | 15% |
| Duplicated collecting system | 10% |
| Renal agenesis | 5% |
| Renal ectopia | 5% |
| Vascular anomalies | 5% |

→ 诊断时**必查肾脏超声**

### 👂 ENT

#### 听力
- **Conductive hearing loss**（反复中耳炎）
- **Sensorineural hearing loss**（成年发展）
- → 每年听力筛查

#### 面部
- High-arched palate
- Micrognathia
- Crowded teeth

### 👁️ 眼

- **Amblyopia**（弱视）30%
- **Strabismus**（斜视）20%
- Ptosis
- Hyperopia / Myopia
- Red-green color blindness

### 🧠 神经认知

> [!tip] 智力**正常**（important！）
> 但有特定认知模式：

- **Visuospatial deficits**（视空间能力↓）
- 数学 / 几何困难
- 言语智商**正常或更高**
- 社交认知偏低
- 注意力问题（ADHD-like）

## 五、诊断 + 评估

### 5.1 确诊

> [!success] **Karyotype** = 金标准

```
临床怀疑 Turner
    ↓
Peripheral blood karyotype
    ↓
看 30+ 个细胞（找 mosaic）
    ↓
确诊 + 分型
```

### 5.2 初次诊断后**完整评估清单**

> [!warning] 每个新诊断的 Turner 都必须做

#### 心血管
- [ ] Echocardiogram（找 bicuspid AV / CoA / aortic dilation）
- [ ] Cardiac MRI（青春期 + 成年后）
- [ ] 4 肢血压 + ECG

#### 内分泌
- [ ] TSH, free T4
- [ ] Thyroid antibodies (anti-TPO)
- [ ] Fasting glucose, HbA1c
- [ ] Lipid panel
- [ ] Vitamin D, Calcium
- [ ] DEXA scan（青春期后）

#### 肾
- [ ] Renal ultrasound

#### 其他
- [ ] Audiology evaluation
- [ ] Ophthalmology eval
- [ ] Skeletal survey（手腕 X-ray for Madelung）

## 六、治疗（USMLE 必考决策）

### 6.1 童年期：Growth Hormone（GH）

> [!tip] **早期 GH 治疗 = 最终身高增加**

- **何时开始**：身高 < 第 5 百分位（通常 4-6 岁）
- **何时停止**：骨龄 14 岁 / 接受最终身高
- **效果**：增加最终身高 5-8 cm
- ⚠️ **不能完全恢复正常身高**

### 6.2 青春期：Estrogen Replacement

> [!success] 触发青春期发育 + 保护骨密度

```
何时开始：11-12 岁（个体化）
    ↓
低剂量 Estrogen（替代）
    ↓
逐渐增加 2-3 年（模拟正常青春期）
    ↓
2 年后加 Progesterone（防 endometrial hyperplasia）
    ↓
持续至成人（约 50 岁停）
```

⚠️ **不要太早开始** → 影响 GH 效果（提前关闭骨骺）

### 6.3 成人期管理

| 系统 | 监测频率 | 内容 |
|------|----------|------|
| 心血管 | 每年 | BP, ECG; 每 5-10 年 Cardiac MRI |
| 甲状腺 | 每年 | TSH |
| 糖尿病 | 每年 | HbA1c, fasting glucose |
| 骨质疏松 | 每 5 年 | DEXA |
| 听力 | 每 3-5 年 | Audiogram |
| 妇科 | 持续 | 激素 + 子宫超声 |

### 6.4 妊娠管理（高风险）

> [!danger] 关键陷阱

孕前：
- **必查** Cardiac MRI + Echo
- **ASI > 25 mm/m²** → 不建议怀孕
- 既往 aortic surgery → **绝对禁忌**

孕期：
- 严密 BP 监测
- 重复 Echo
- MFM 共管

## 八、关键鉴别诊断

### 🩺 Primary Amenorrhea 鉴别（高频）

| 病因 | 关键鉴别点 |
|------|-----------|
| **Turner (45,X)** ⭐ | 矮 + 表型特征 + FSH↑↑ |
| **Müllerian agenesis (MRKH)** | **正常身高** + 正常 secondary sex characteristic + **正常 FSH** + 46,XX + 无子宫 |
| **Androgen insensitivity (AIS)** | 高 + 乳房发育 + **无阴毛** + 46,XY + Testes 在腹股沟 |
| **Constitutional delay** | 矮 + 家族史 + 骨龄延迟 + FSH 正常 |
| **PCOS** | 通常正常身高 + hirsutism + obesity |
| **Imperforate hymen** | 周期性下腹痛 + bulging hymen |
| **Hypothalamic amenorrhea** | 厌食症 / 过度运动 + FSH↓ |

### 🧠 矮小 + 综合征鉴别

| 综合征 | 关键鉴别 |
|--------|----------|
| **Turner** ⭐ | 女 + 表型 + 性腺发育不全 |
| Noonan | "Male Turner" 但**正常染色体** + **Pulmonary stenosis** (vs CoA) |
| Williams | 友好性格 + 鸡尾酒派对人格 + Supravalvular AS + Hypercalcemia |
| Achondroplasia | 短肢矮 + 大头 + AD 遗传（FGFR3） |
| Down (21-trisomy) | 智力↓ + AV canal defect + 单掌纹 |

## 九、易混陷阱（USMLE 经典）

### ⚠️ 陷阱 1：**Turner 的 CoA vs 普通 CoA**

| | Turner CoA | Sporadic CoA |
|---|------------|--------------|
| 性别 | 女 | 男 = 女 |
| 关联 | Bicuspid AV (双重风险) | 单独可见 |
| 妊娠 | 极高风险 | 较高 |
| 监测 | Aortic dissection lifetime | 关注 HTN |

### ⚠️ 陷阱 2：**Turner 表型很轻怎么办**

- **Mosaic 45,X/46,XX** 患者可能**几乎正常**
- 仅表现为**不孕 + 轻度矮小**
- → 任何"原发不孕 + 矮"都查 karyotype

### ⚠️ 陷阱 3：**Noonan vs Turner**

| | Turner | Noonan |
|---|--------|--------|
| 性别 | 仅女 | 男女都可 |
| 染色体 | 45,X | **46,XX 或 46,XY**（正常）|
| 智力 | 正常 | 1/3 受影响 |
| 心脏 | **Coarctation + Bicuspid AV** | **Pulmonary stenosis + HCM** |
| 遗传 | 散发 | AD（PTPN11 基因）|

### ⚠️ 陷阱 4：**Turner 的特殊染色体型**

- **Mixed gonadal dysgenesis (45,X/46,XY)**：
  - 部分 Y → **Streak gonad 恶变风险**（gonadoblastoma）
  - **必须** prophylactic gonadectomy

## 十、Aortic Dissection in Turner（专题深入）

### 10.1 风险因素
- **Bicuspid AV**（最强 RR）
- **CoA**（强 RR）
- **Aortic root dilation** (ASI > 25 mm/m²)
- **HTN**
- **妊娠**（容量负荷）
- **激素替代治疗**（争议）

### 10.2 病理生理

```
胶原纤维异常（X-linked? 机制不完全清楚）
+ Bicuspid AV → turbulent flow
+ CoA → 上游高压
   ↓
Cystic medial degeneration（中层囊性退化）
   ↓
Aortic root + ascending aorta dilation
   ↓
急性 Stanford A 夹层
```

### 10.3 临床识别

```
Young female + chest pain + 既往:
   ├─ Short stature
   ├─ Primary amenorrhea
   └─ Bicuspid AV / CoA history
       ↓
立刻：
   1. CTA aorta（最快）
   2. TEE（如不稳定）
   3. Cardiac surgery 会诊
```

### 10.4 治疗

| Stanford 分型 | 治疗 |
|---------------|------|
| **Type A**（ascending aorta）| **紧急手术** ⭐ |
| **Type B**（descending）| Medical first（β-blocker + BP control）|

## 十三、考前 5 分钟极速复习

> [!danger] 闭眼能默写以下内容才算掌握

### 一句话识别
"**青春期女孩 + 矮小 + 原发性闭经 + 颈蹼/肘外翻**" = Turner

### 3 大心血管异常
1. **Bicuspid aortic valve**（30-50%）
2. **Coarctation of aorta**（15-30%）
3. **Aortic dilation / dissection 风险**

### 3 大非心血管异常
1. **Horseshoe kidney**（马蹄肾，15%）
2. **Hashimoto 甲状腺炎**（30-50%）
3. **听力丧失**（传导 + 感音）

### 诊断
**Peripheral blood karyotype**（30+ cells）

# 婴儿松软 Floppy Infant / NMJ 病变鉴别

> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q13219]]（婴儿肉毒中毒，我误选 ACh-R 抗体 = 重症肌无力）。
> 用法：见"婴儿 hypotonia / 喂养差 / 上睑下垂"，先分 **中枢 vs 周围（NMJ/肌肉）**，NMJ 内再用 **年龄 + 便秘 + 抗体 + 起病速度** 锁定。

---

## 一、第一刀：中枢 vs 周围

```
婴儿松软 hypotonia
├─ 中枢性（脑）── 意识/反射异常、惊厥、畸形、HIE 病史
│       ↑ 肌张力低但腱反射常存/亢进、有皮层体征
└─ 周围性（下运动单元）── 清醒警觉但全身软、腱反射↓/消失
        ├─ 前角细胞：SMA（脊髓性肌萎缩）
        ├─ NMJ：婴儿肉毒 / 新生儿一过性 MG / 先天性 MG
        └─ 肌肉：先天性肌病/肌营养不良、先天性肌强直性营养不良
```

### 1.1 中枢 vs 周围 —— 两把钥匙：腱反射 + 警觉度 ⭐

> [!info] 一句话分流
> **中枢（脑/UMN）= "软但反射在/亢进 + 神志/皮层异常"；周围（下运动单元）= "清醒警觉但软 + 反射消失"。**

| 维度 | **中枢性（脑/UMN）** | **周围性（前角/神经/NMJ/肌肉）** |
|---|---|---|
| 肌张力 | 低 | 低 |
| **腱反射 DTR** ⭐ | **保留或亢进** | **减弱 / 消失** |
| **意识/警觉** ⭐ | **异常**（嗜睡、互动差）| **清醒警觉**（"alert but floppy"）|
| 病理征 | **有**（见 1.2）| 无 |
| 代表病 | HIE、颅内出血、染色体/综合征、代谢病、Prader-Willi 早期 | SMA、婴儿肉毒、新生儿 MG、先天性肌病 |

### 1.2 皮层体征 / 中枢征清单（提示问题在"脑"）

> [!tip] 看到这些 = 往中枢（脑）想，不是下运动单元
> - **意识/警觉改变**：嗜睡、反应差、眼神互动少、喂养中枢协调差（≠ 单纯吸吮无力）
> - **腱反射保留/亢进 + 病理征**：**Babinski 巴宾斯基征、clonus 阵挛**，后期 **spasticity 痉挛**（上运动神经元释放）
> - **惊厥 seizures**（皮层放电）
> - **持续姿势异常**：握拳拇指内收（cortical thumb / fisting）、剪刀腿（scissoring）、角弓反张
> - **畸形 / 综合征面容 dysmorphism**、小头 / 巨头、囟门异常
> - **围产期病史**：**HIE（缺氧缺血性脑病）**、窒息、颅内出血、宫内感染（TORCH）
>
> **反之**——清醒警觉 + 反射消失 + 无上述皮层征 → 周围性 → 再用便秘 / 抗体 / 舌纤颤等钥匙分肉毒 / 新生儿 MG / SMA（见 §二、§三）。

## 二、NMJ 三病鉴别（核心）⭐

| 病 | 年龄/起病 | 标志线索 | 抗体 | 机制 | 处理 |
|---|---|---|---|---|---|
| **婴儿肉毒 infant botulism** | <12 月，**急性—亚急性** | **便秘（首发）+ 对称下行麻痹 + 上睑下垂 + 喂养差 + 口干（自主神经）** | 阴性 | 摄入孢子→肠道产毒→**阻突触前 ACh 释放** | **肉毒抗毒素 BIG-IV**（临床诊断，不等结果）；忌氨基糖苷 |
| **新生儿一过性 MG** | 生后**数小时—数天** | **母亲有重症肌无力** + 喂养差、哭声弱、全身软；**自限（数周）** | **母源 ACh-R 抗体经胎盘传入（阳性）** | 抗体一过性阻断 | 支持 ± 抗胆碱酯酶；自限 |
| **先天性肌无力综合征 CMS** | 出生起持续 | 上睑下垂/眼肌麻痹、易疲劳，**家族史**，**非抗体性** | 阴性 | NMJ 蛋白基因缺陷 | 按亚型（部分用 pyridostigmine）|

> [!danger] 我错的点（Q13219）
> "上睑下垂 + 松软"反射到测 **ACh-R 抗体** → 但抗体阳性的是**母亲有 MG 的新生儿**（生后数小时—数天）。本例 6 月龄、急性、**便秘 + 下行麻痹** = **婴儿肉毒**，诊断靠临床、治疗给抗毒素，不靠抗体。

## 三、周围其他（前角 / 肌肉）速记

| 病 | 钥匙 |
|---|---|
| **SMA（脊髓性肌萎缩 spinal muscular atrophy）I 型 = Werdnig-Hoffmann** | 进行性对称无力 + **舌肌纤颤 fasciculation** + 腱反射消失 + **钟形胸 bell-shaped chest**；**SMN1 基因缺失（5q，常隐）→ 前角运动神经元凋亡**；警觉但软（"alert but floppy"）|
| **先天性肌强直性营养不良** | **母亲有肌强直性营养不良**（握手不放松）+ 面瘫样/帐篷嘴；DMPK 三核苷酸扩增 |
| **Prader-Willi** | 新生儿严重 hypotonia + 喂养差 → 后期肥胖/贪食；15q11 印记 |

> [!info] SMA 全称 + 分型（按起病年龄 / 最大运动里程碑，型号越大越轻越晚）
> **SMA = Spinal Muscular Atrophy（脊髓性肌萎缩症）**：脊髓**前角细胞（下运动神经元）**变性 → 失神经性肌萎缩；**清醒警觉但软 + 反射消失 + 舌纤颤**（脑/皮层正常）。
> - **I 型 Werdnig-Hoffmann**：<6 月，**最重、不能坐**，呼吸衰竭（未治多 <2 岁）
> - **II 型**：能坐、不能独走
> - **III 型 Kugelberg-Welander**：能独走（较轻）
> - **IV 型**：成人起病
> - 靶向药（nusinersen / onasemnogene / risdiplam）已改写预后。

## 四、Memory Hook

> [!success] 一句话
> **婴儿便秘 + 从头软下去 + 眼皮耷拉 + 口干 = 肉毒（给抗毒素）。** ACh-R 抗体 = **妈妈有 MG 的新生儿**（生后几天）。舌纤颤 + 反射消失 = **SMA**。妈妈握手不放松 = **肌强直性营养不良**。

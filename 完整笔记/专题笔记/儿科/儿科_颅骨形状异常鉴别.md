# 儿科_颅骨形状异常鉴别（Abnormal Head Shape：Plagiocephaly vs Craniosynostosis）

> [!info] 一句话定位
> 婴儿"头形不对/头扁"两条主线：**Positional plagiocephaly（体位性斜头畸形，颅缝正常）** vs **Craniosynostosis（颅缝早闭，颅缝过早骨性融合）**。**核心判据：摸有没有骨嵴（palpable bony ridge）+ 颅缝活不活动**——早闭沿融合缝有**硬骨嵴**、耳位向后，体位性无骨嵴、耳位向前。
> **⚠️ 与「头皮肿胀」分家**：本篇讲**颅骨形状异常（无肿块）**；有边界清/弥散的**头皮肿块**看姊妹篇 [[完整笔记/专题笔记/儿科/儿科_新生儿头皮肿胀鉴别]]。**两者都不表现为 well-demarcated scalp swelling。**

## 一、先分两大类 ⭐

```
婴儿头形异常
├── 颅缝正常、可活动 → Positional（deformational）plagiocephaly 体位性斜头
│      枕部因外力受压变平；young infant（非 newborn）；良性
└── 颅缝过早融合 → Craniosynostosis 颅缝早闭
       沿闭合缝可摸到骨嵴；出生时/早期即有；可能需手术
```

- **关键区分动作**：沿颅缝**触诊有无骨嵴（bony ridge）** + 颅缝是否可活动/前囟是否开放。
- **两者都不是"头皮肿块"**——是**颅骨整体轮廓（contour）异常**，不是 well-demarcated scalp swelling（那属头颅血肿/产瘤，见姊妹篇）。

## 二、Positional Plagiocephaly（体位性斜头畸形）⭐

> [!info] 定义
> 外力（长期同一体位受压）致**枕部局部扁平**，颅缝完全正常、无融合。**最常见的婴儿头形异常**。

- **病因/诱因**：
  - **仰卧睡姻（supine positioning）** —— "Back to Sleep" 防 SIDS（婴儿猝死综合征）后发病率上升，是其副产物。
  - **颈部受限**：先天性斜颈（congenital muscular torticollis）→ 头总偏一侧；早产、宫内受限、发育迟缓少自主转头。
- **⚠️ 时间点考点**：见于 **young infants（约生后 2 月起、数月龄）**，**不是 newborn**——刚出生还没躺够时间压不平；题干"出生时即有头形异常"更偏 craniosynostosis。
- **形态**：**平行四边形头（parallelogram）**——枕部扁平侧的**同侧耳朵与前额向前移位（anterior displacement）**，从头顶俯视呈斜的平行四边形。
- **颅缝/前囟**：正常开放、无骨嵴。
- **诊断**：**临床**（视诊 + 触诊），典型者不需影像。
- **处理（阶梯）**：
  1. **保守为主**：加强**清醒时俯卧（tummy time）** + 交替头位/多变换抱姿/减少长时间平躺容器（车座、摇椅）。
  2. **合并斜颈** → 加**颈部理疗/牵伸（physical therapy）**。
  3. 保守 2-3 月无效或中重度（多 4-12 月龄）→ **头盔矫形（helmet/molding therapy）**。
- **预后**：良性，随生长与活动增多多自行改善，不影响脑发育。

## 三、Craniosynostosis（颅缝早闭）⭐

> [!info] 定义
> 一条或多条**颅缝过早骨性融合（premature suture fusion）**，颅骨沿融合缝方向停止生长、垂直方向代偿膨出 → 特征性头形 + 沿缝**骨嵴**。多为散发；也见于综合征型（Apert、Crouzon）。

- **病理生理（Virchow 定律）**：颅骨**沿闭合缝方向不能生长**，代偿性**向平行于该缝的方向扩张**——决定了每种缝早闭对应固定头形。
- **体征**：沿融合缝可触及**硬的骨嵴（palpable bony ridge）**；受累缝处前囟可早闭/紧张；严重多缝受累可致**颅内压增高（ICP）**。
- **按受累缝分型（高频对照）**：

| 融合缝 | 头形（英文） | 中文 | 要点 |
|---|---|---|---|
| **矢状缝 Sagittal** | Scaphocephaly / dolichocephaly | 舟状头（长头） | **最常见**；头**前后长、左右窄** |
| **单侧冠状缝 Unilateral coronal** | Anterior plagiocephaly | 前部斜头 | 同侧前额扁平 + 眶上缘抬高（harlequin eye 影像征）|
| **额缝 Metopic** | Trigonocephaly | 三角头 | 前额尖凸呈三角、眶距过窄（hypotelorism）|
| **单侧人字缝 Lambdoid** | Posterior plagiocephaly | 后部斜头 | **罕见**；需与体位性斜头鉴别（见§四）|

- **诊断**：疑诊靠**触诊骨嵴** → **头颅 CT（computed tomography，计算机断层扫描；含三维重建）** 确诊颅缝融合（金标准）；平片可初筛。
- **处理**：**神经外科 / 颅面外科手术**（多在 3-12 月龄；早期内镜条切除 + 头盔，或开放颅骨重塑），解除限制、防 ICP 增高与外观畸形。
- **警示**：多缝早闭/综合征型 → 监测 **ICP 增高**（呕吐、易激惹、视乳头水肿、发育倒退）。

## 四、⭐ 核心对决：体位性 vs 人字缝早闭（最爱考的一对）

后部斜头两大来源——**体位性斜头（常见、良性）** vs **单侧人字缝早闭（罕见、需手术）**——耳位方向相反是关键：

| 鉴别点 | Positional（体位性）| Lambdoid synostosis（人字缝早闭）|
|---|---|---|
| **顶视头形** | **平行四边形（parallelogram）** | **梯形（trapezoid）** |
| **同侧耳位** | **向前移位（anterior）** | **向后（+下）移位（posterior/inferior）** |
| **同侧前额** | **向前隆起**（饱满）| 正常/后缩 |
| **枕部** | 扁平但圆滑，无嵴 | 扁平 + **可触骨嵴** |
| **骨嵴** | **无** | **沿人字缝有** |
| **发生率 / 处理** | 常见；tummy time / 头位 / 头盔 | 罕见；**手术** |

> [!tip] 一句话记
> **耳朵往前 = 体位性（往前压出来的）；耳朵往后 = 人字缝早闭（骨头缝焊死了）。摸到硬骨嵴 → craniosynostosis。**

## 五、易混陷阱（普适）⭐

- **"头皮肿块" ≠ 颅骨形状异常**：本篇两病都**不表现为 well-demarcated scalp swelling**。
  - 边界清、不跨缝、硬 → **Cephalohematoma 头颅血肿**（骨膜下）。
  - 跨缝、软凹陷、出生即有 → **Caput succedaneum 产瘤**。
  - 跨缝、波动、进行性扩大、可致命 → **Subgaleal hemorrhage 帽状腱膜下出血**。
  - → 详见 [[完整笔记/专题笔记/儿科/儿科_新生儿头皮肿胀鉴别]]。
- **Newborn（出生时即有异常头形/骑跨颅缝）**：产道挤压可致一过性 **molding（颅骨塑形）**，数天自复位——别误当病理；而**出生即有 + 骨嵴** 更提示 craniosynostosis。
- **头围整体异常另走别的路**：
  - 头围**大** + 前囟膨隆 → 脑积水/ICP 增高（[[完整笔记/专题笔记/儿科/儿科_新生儿头皮肿胀鉴别]] §四有图）。
  - 头围**小**（craniosynostosis 多缝早闭也可小头）→ 与真性小头畸形鉴别，见 [[完整笔记/专题笔记/儿科/儿科_小头畸形microcephaly评估]]。
- **体位性斜头 = young infant（2 月+），不是 newborn**——这是 UWorld 常埋的时间点钩子。
- **看到"骨嵴（bony ridge）"或"出生时即有固定头形"** → 直接想 craniosynostosis + 转外科；**看到"仰睡/斜颈/耳向前/平行四边形/无骨嵴"** → 体位性 + tummy time。

## Memory Hook

> [!success]
> - **无骨嵴 + 平行四边形 + 耳向前 + 2 月龄 + 仰睡史 = Positional plagiocephaly** → tummy time / 头位（重症头盔）
> - **有骨嵴 + 特征头形 + 出生早现 = Craniosynostosis** → CT 确诊 + 外科
> - 分型口诀：**矢状→舟状（长头，最常见）/ 额缝→三角头 / 冠状→前斜 / 人字→后斜（罕见）**
> - 一句判据：**"摸到硬骨嵴 → 缝焊死了（早闭）；摸不到 + 耳向前 → 压扁的（体位性）"**
> - 与头皮肿胀分家：**形状异常无肿块；有肿块看血肿/产瘤篇**

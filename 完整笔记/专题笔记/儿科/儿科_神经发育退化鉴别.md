# 儿科_神经发育退化鉴别（Neurodevelopmental Regression）

> [!info] 一句话定位
> 婴幼儿"**先正常发育、后退化 (regression)**"或"发育停滞 + 特征性 buzzword"是 USMLE 一组高频一对一映射题。做题两步走：**① 先分有没有正常发育期**（退化型 vs 自始迟缓型）→ **② 退化型再按「倒退起始年龄 + 伴随器官/眼/皮肤线索」定位到具体病**。
> 派生自错题 [[mistakes/uworld-mistakes_2026-07#^Q15794]]（Rett 误选 PKU 霉味尿）。

## 一、第一步分岔：有没有"正常发育期" ⭐

```
                  婴幼儿发育异常
                        │
             之前有一段正常发育、之后退化？
                        │
          ┌─────────────┴─────────────┐
     有 (regression)              无 (自始迟缓 / 停滞)
     丢失已获得技能                从头就慢，不倒退
          │                            │
          ▼                            ▼
   见 §二 完整退化表              见 §三 完整迟缓表
   (Rett/Tay-Sachs/Niemann-      (Angelman/Prader-Willi/
    Pick/Krabbe/MLD/Krabbe/       Fragile X/Down/FAS/
    ALD/Hurler/Leigh/SSPE...)     Cerebral palsy/先天甲减...)
```

> [!warning] 分岔的核心动词
> - **退化型 (regression)** = **丢失**已经获得的技能（会走→不会走、会说→不说）。
> - **迟缓型 (delay)** = 从头就落后、**从未获得**里程碑，但**不倒退**（Angelman、Down 等）。
> - 问诊关键句："以前会不会做，现在还会不会做" —— 会→现在不会 = 退化；一直不会 = 迟缓。

## 二、分支一完整表：有正常发育期 → 退化（附倒退时间）⭐

> 按**倒退起始年龄由早到晚**排序，方便用"发病时间点"反查病。

| 病                                                 | 正常期 / 倒退起点                   | 招牌 buzzword / 关键线索                                                         | 基因·酶·机制                                       |
| ------------------------------------------------- | ---------------------------- | -------------------------------------------------------------------------- | --------------------------------------------- |
| **Krabbe**                                        | 正常至 3 月，**~3–6 月**起退化，2 岁前死  | 易激惹 + 发热 + **周围神经病** + 视神经萎缩；后期角弓反张                                        | Galactocerebrosidase↓；globoid cells（球形细胞）     |
| **Tay-Sachs**                                     | 正常至 3–6 月，**~6 月**起退化，2–4 岁死 | **cherry-red spot 樱桃红斑** + **夸张惊跳反射 (exaggerated startle)**；**无肝脾大**；进行性巨头 | Hexosaminidase A↓；GM2 神经节苷脂堆积                 |
| **Niemann-Pick A型**                               | 正常至 ~6 月，**~6 月**起退化，2–3 岁死  | **cherry-red spot + 肝脾大**（鉴别钥匙）；肌张力低下                                      | Sphingomyelinase↓；sphingomyelin 堆积；foam cells |
| **Rett**                                          | 正常至 **6–18 月**，之后停滞 → 退化     | **仅女孩**；语言退化 + **搓手/绞手 (hand-wringing)** + **后天性小头** + 呼吸异常                | MECP2 突变（X 连锁）                                |
| **Leigh（利氏病 / 亚急性坏死性脑脊髓病）**                       | 多在 **3–24 月**（常 <2 岁）起退化     | 退化 + **乳酸酸中毒** + 脑干体征（呼吸/眼动异常）、喂养差                                         | 线粒体病（母系遗传 / 核基因）                              |
| **Hurler（MPS I，黏多糖贮积症 I 型）**                      | 正常至 **6–12 月**，1–2 岁起退化      | 退化 + **粗糙面容 (coarse facies)** + 肝脾大 + **角膜混浊** + 关节僵硬                      | α-L-iduronidase↓；黏多糖堆积                        |
| **Metachromatic leukodystrophy（异染性脑白质营养不良, MLD）** | 晚婴型 **~12–24 月**起退化          | 退化 + **步态异常 / 脑白质病** + 周围神经病                                               | Arylsulfatase A↓                              |
| **Adrenoleukodystrophy（肾上腺脑白质营养不良, ALD）**         | 学龄 **~4–8 岁**起退化             | **男孩**；行为/学习退化 + **肾上腺功能不全**（皮肤色素沉着）                                       | X 连锁；VLCFA（极长链脂肪酸）↑                           |
| **Batten（juvenile NCL，神经元蜡样脂褐质沉积症）**              | **~5–10 岁**起退化               | 退化 + **视力丧失** + 癫痫                                                         | CLN3                                          |
| **SSPE（亚急性硬化性全脑炎）**                               | 麻疹感染后 **7–10 年**（学龄～青少年）起退化  | 认知/行为退化 + **肌阵挛 (myoclonus)**；**麻疹病史**（尤其 <2 岁患麻疹、未接种）                     | 麻疹病毒晚发持续感染                                    |

> [!tip] 用"倒退年龄"反查（做题第二步）
> - **婴儿期 ~6 月同一窗口**：Krabbe / Tay-Sachs / Niemann-Pick A → 靠**眼底 + 肝脾**再分：
>   - 樱桃红斑 **无**肝脾大 = **Tay-Sachs**（+ 夸张惊跳反射）
>   - 樱桃红斑 **有**肝脾大 = **Niemann-Pick A**
>   - 无樱桃红斑，易激惹 + 周围神经病 + 发热 = **Krabbe**
> - **6–18 月 + 女孩 + 搓手** → **Rett**
> - **1–2 岁 + 粗糙面容 / 角膜混浊 / 肝脾大** → **Hurler**
> - **学龄期起退化** → **ALD**（男孩 + 肾上腺）/ **Batten**（视力丧失 + 癫痫）/ **SSPE**（麻疹史 + 肌阵挛）

## 三、分支二完整表：无正常发育期 → 自始迟缓 / 停滞（非退化）

> 这一支特征是**从头就慢**、里程碑落后但**不倒退**；用"buzzword + 面容/行为特征"定位。

| 病 | 招牌 buzzword / 关键线索 | 机制 |
|---|---|---|
| **Angelman（天使综合征）** | **happy demeanor 快乐爱笑 + 阵发大笑** + 共济失调步态 (jerky ataxia) + 癫痫 + 几乎无语言（"Happy Puppet"） | **母源** 15q11-13 缺失；UBE3A |
| **Prader-Willi（普-威综合征）** | 婴儿期严重肌张力低 + 喂养困难/FTT (failure to thrive, 生长发育停滞) → 幼儿期后 **hyperphagia 暴食 → 肥胖**；杏仁眼、小手小脚、性腺发育不全 | **父源** 15q11-13 缺失 |
| **Fragile X（脆性 X 综合征）** | **最常见遗传性智力障碍（男孩）**；长脸大耳、**巨睾 (macroorchidism)**、自闭特征 | FMR1 CGG 三核苷酸重复扩增 |
| **Down（21 三体）** | 肌张力低 + 特征面容 + 房室隔缺损 + 通贯掌 | Trisomy 21 |
| **Fetal alcohol syndrome（胎儿酒精综合征, FAS）** | **人中平滑 + 上唇薄 + 眼裂小** + 宫内生长受限 | 孕期酒精暴露 |
| **Cerebral palsy（脑瘫, CP）** ⭐ | **静止性、非进行性**运动障碍；出生前后脑损伤；痉挛/张力异常但**不倒退** | 围产期脑损伤（缺氧/早产/感染）|
| **先天性甲减（cretinism, 克汀病）** | 未筛查未治疗：迟缓 + 粗糙面容 + 脐疝 + 巨舌（**可预防**） | 甲状腺发育/合成缺陷 |

## 四、核心 5 病招牌 buzzword 速对（考试最爱一对一）

| 病 | 招牌 buzzword | 一句话锁定 |
|---|---|---|
| **Rett** | 搓手 (hand-wringing) + 后天性小头 | 女孩 + 正常后语言退化 |
| **Angelman** | happy demeanor 爱笑 + 共济失调 | 迟缓非退化，UBE3A 母源缺失 |
| **PKU（phenylketonuria, 苯丙酮尿症）（未治疗）** | musty/mousy odor 霉味尿 | 未筛查未治，色素浅 |
| **Tay-Sachs** | cherry-red macula + 夸张惊跳 | 无肝脾大 |
| **DMD（Duchenne 肌营养不良）** | Gower 征 + 腓肠肌假性肥大 | 男孩 + 心肌病 + CK (creatine kinase, 肌酸激酶)↑，肌肉为主无中枢退化 |

## 五、Rett 综合征细化

- **诊断三联**：正常发育后 **① 语言退化 + ② 手目的性动作丧失（搓手 stereotypic hand-wringing）+ ③ 步态异常 / 刻板动作（摇晃身体）**。
- **常伴**：头围增长减速 → **后天性小头畸形 (acquired microcephaly)**、癫痫、**呼吸异常**（清醒时过度通气-屏气交替，情绪相关）、睡眠障碍、孤独样特征。
- **神经病理**：脑生长减速 (deceleration of brain growth)。
- **预后**：可活至中年，活动能力下降、癫痫、呼吸困难。

## 六、Tay-Sachs 时间线 + vs Niemann-Pick

### 6.1 Tay-Sachs 退化时间线

```
0–3 月       约 3–6 月               6–12 月及以后
出生正常  →  发育停滞 + 退化开始   →   进行性 hypotonia→spasticity
看似健康     首发常是               运动倒退(失去已获得技能)+视力丧失
             听觉过度惊跳            进行性 macrocephaly(巨头)
             (exaggerated startle)  → 2–5 岁前死亡
```

- **最早信号**：**对声音的过度惊跳反射 (exaggerated startle / hyperacusis)** —— 高频 buzzword。
- **约 3–6 月**：发育停滞后开始运动退化。
- **6–12 月及以后**：进行性 hypotonia → 后期 spasticity、**vision loss**、**进行性 macrocephaly**。
- **眼底**：**cherry-red spot** —— 黄斑中心因周围神经节细胞堆积 GM2 而对比出红点。
- 结局：2–5 岁前死亡。

### 6.2 Tay-Sachs vs Niemann-Pick（都有樱桃红斑）

> [!warning] 核心鉴别 = 有无肝脾大
> 两病都有 **cherry-red macula + 神经退化**，靠 **hepatosplenomegaly（肝脾大）** 一刀切开。

| 项目 | **Tay-Sachs (TSD)** | **Niemann-Pick A (NPD-A)** |
|---|---|---|
| **酶缺陷** | **Hexosaminidase A (Hex-A)** ↓ | **Sphingomyelinase（鞘磷脂酶）** ↓ |
| **堆积物** | GM2 ganglioside | Sphingomyelin（鞘磷脂）|
| **樱桃红斑** | ✅ 有 | ✅ 有 |
| **神经退化** | ✅ 有 | ✅ 有 |
| **肝脾大** ⭐ | **❌ 无**（鉴别钥匙）| **✅ 有**（鉴别钥匙）|
| **特征细胞** | — | **foam cells（泡沫细胞）** |
| **高发人群** | Ashkenazi Jewish（阿什肯纳兹犹太人）| Ashkenazi Jewish |
| **记忆钩** | Tay-**saCHS** = **No** hepatosplenomegaly | Niemann-**Pick** = **Pick 起肝脾**（有肝脾大）|

## 七、Niemann-Pick 时间线 + A/B/C 三型鉴别 ⭐

### 7.1 A 型（经典婴儿神经型）退化时间线

```
0 – ~6 月            约 6 月起            ~1 岁              2–3 岁
发育正常          → 神经退化开始      → 退化明显          → 死亡
(可有早期肝脾大    里程碑丢失、          cherry-red spot 出现
 / 喂养差,里程碑    进行性 hypotonia、    肝脾大显著
 仍在走)           精神运动倒退
```

- **出生～约 6 月**：发育正常（可有早期肝脾大、喂养差，但里程碑还在走）。
- **约 6 月起**：开始**神经退化** —— 已获得里程碑丢失、进行性肌张力低下、精神运动倒退。
- **1 岁左右**：退化明显 + **cherry-red spot** 出现、肝脾大显著。
- **2–3 岁前死亡**。

> [!tip] 记忆钩
> A 型倒退**起点 ~6 月，和 Tay-Sachs 几乎同一窗口** —— 这也是它俩总被放一起考的原因。**区分靠肝脾大（NP 有 / TSD 无），不是靠发病时间。**

### 7.2 三型时间线差异（别混）

| 型 | 倒退 / 发病时间 | 机制 | 特点 |
|---|---|---|---|
| **A 型** | **~6 月起退化**，2–3 岁死 | Sphingomyelinase↓ | 经典婴儿神经型；cherry-red + 肝脾大 |
| **B 型** | 儿童期起病，**基本无神经退化** | Sphingomyelinase↓（残留活性较高）| 内脏型；肝脾大 + 肺部受累，可活到成年 |
| **C 型** | **儿童～青少年**隐匿退化 | **胆固醇转运缺陷**（NPC1/NPC2，非酶缺陷）| **垂直凝视麻痹 (vertical gaze palsy)** + 共济失调 + 学习退化 |

> [!warning] C 型是独立陷阱位
> stem 给**学龄期/青少年缓慢退化 + 垂直凝视麻痹**，选 **C 型**，不是 ~6 月起病的 A 型 —— 机制（胆固醇转运 vs 鞘磷脂酶）和时间线完全错开。

## 八、Angelman 婴儿期表现 + vs Prader-Willi ⭐

### 8.1 Angelman 婴儿期（约 6–12 月）

> [!info] 本质是 delay 不是 regression
> Angelman 是**从头就慢的发育迟缓**，不是"先正常后退化" —— 这是它与 Tay-Sachs / Rett 等退化型的关键鉴别（退化型会**丢失**已获得技能）。

- **Developmental delay**：运动/坐立/爬行落后，喂养困难、吸吮差。
- **Hypotonia** + 躯干不稳。
- 之后逐渐显现（多在 1 岁后更明显）：
  - **Ataxia → 宽基步态 + 动作抖动**，"puppet-like"（提线木偶样步态）。
  - **Happy demeanor + 无诱因 paroxysms of laughter + 频繁微笑/拍手** —— buzzword "happy puppet"。
  - **Absent/minimal speech**。
  - **Seizures** 常在 2–3 岁出现。
  - microcephaly、金发白肤（色素较浅）。

### 8.2 Angelman vs Prader-Willi（同段 15q11-13，父母源相反）

> [!warning] 印记疾病经典配对
> **同一染色体区段 (15q11-13)，缺失来源相反 → 两个完全不同的病。**

| 项目 | **Angelman (AS)** | **Prader-Willi (PWS)** |
|---|---|---|
| **缺失来源** ⭐ | **母源 (maternal)** 15q11-13 缺失 / 父源 UPD (uniparental disomy, 单亲二体) | **父源 (paternal)** 15q11-13 缺失 / 母源 UPD |
| **责任基因** | **UBE3A**（母源表达）| SNRPN 等（父源表达）|
| **婴儿期** | 发育迟缓 + hypotonia + 喂养困难 | **严重 hypotonia + 喂养困难/吸吮差 + FTT** |
| **幼儿期后** | happy demeanor + 大笑 + 共济失调 + 癫痫 | **hyperphagia 暴食 → 肥胖** + 觅食执念 |
| **招牌** | "Happy puppet" 爱笑 + jerky ataxia | almond eyes 杏仁眼 + 小手小脚 + 性腺发育不全 |
| **智力** | 严重智障 + 几乎无语言 | 轻中度智障 |
| **确诊** | 甲基化分析 (methylation) 区分父/母源 | 甲基化分析 |
| **记忆钩** | **A**ngelman = m**A**ternal（母走子笑）| **P**WS = **P**aternal（父走子饿→暴食）|

## 九、易混陷阱（普适）+ 两大陷阱位

### 9.1 一对一 buzzword 混淆

- **"神经退化 + 癫痫" 别反射选 PKU** —— PKU 是**未治疗、无正常发育期**的智力障碍 + 霉味；Rett 有一段正常发育后退化 + 搓手 + 呼吸异常。
- **快乐爱笑 + 共济失调 = Angelman**（"Happy Puppet"），是**迟缓非退化**，无呼吸阵发。
- **樱桃红斑**：Tay-Sachs（无肝脾大）vs Niemann-Pick A（**有**肝脾大）—— 靠脏器肿大区分。
- **扩张型心肌病 + 男孩 + 步态异常 = DMD**（本组唯一以肌肉为主、无中枢退化）。
- **垂直凝视麻痹 + 青少年退化 = Niemann-Pick C**（≠ A 型婴儿）。

### 9.2 陷阱位一：PKU 放"自始异常"其实不严谨

> [!warning] PKU 本质更接近"born normal → 进行性"
> 未治疗的 PKU **出生时是正常的**，随苯丙氨酸堆积**进行性**智力障碍 —— 严格说更接近退化/进行性，而非"自始迟缓"。CK 真正考点是**新生儿筛查 + 低苯丙氨酸饮食可完全预防**。别被"它属于哪支"绕进去；抓 buzzword（霉味尿 + 色素浅）+ 可预防性即可。

### 9.3 陷阱位二：退化型自闭 (regressive autism)

> [!warning] "退化"≠一定是神经代谢病
> 约 1/3 自闭症儿童在 **18–24 月**出现语言/社交退化（曾有正常期），会落进**分支一**。与 Rett / 代谢病的区分靠：**无神经系统硬体征、无器官肿大 (HSM)、无脑白质改变、无樱桃红斑**。

### 9.4 别忘了非进行性对照：Cerebral Palsy

> [!info] CP = 所有神经退行病的反面
> **Cerebral palsy 是静止性 (static / non-progressive)**：损伤固定，症状可随生长"看起来变化"但**病灶不进展**。CK 爱用"运动症状但不倒退 / 病史稳定不恶化"来指向 CP，与本笔记所有退化型区分。

## Memory Hook

> [!success]
> - **两步法**：① 有没有正常发育期（退化 vs 迟缓）→ ② 退化型按「倒退年龄 + 眼/肝脾/皮肤线索」定位。
> - **~6 月退化三兄弟**：Tay-Sachs（樱桃红斑无肝脾+惊跳）/ Niemann-Pick A（樱桃红斑+肝脾）/ Krabbe（易激惹+周围神经病）。
> - **女孩 + 正常后语言退化 + 搓手 + 后天性小头 = Rett（MECP2）**。
> - **学龄期退化**：ALD（男孩+肾上腺）/ Batten（视力丧失）/ SSPE（麻疹史+肌阵挛）。
> - **迟缓非退化**：Angelman（爱笑，母源缺失）/ Prader-Willi（暴食，父源缺失）/ Fragile X（巨睾）/ CP（静止不进展）。
> - **霉味尿 + 可预防 = 未治疗 PKU**；**垂直凝视麻痹 + 青少年 = Niemann-Pick C**。

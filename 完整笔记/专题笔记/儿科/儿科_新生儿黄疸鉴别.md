# 新生儿黄疸鉴别 (Neonatal Jaundice)

> 聚焦**新生儿期**黄疸的分流与鉴别，重点是**非结合性高胆红素血症 (unconjugated hyperbilirubinemia)**。
> 遗传性慢性高胆红素血症（Gilbert / Crigler-Najjar / Dubin-Johnson / Rotor）见 [[完整笔记/专题笔记/GI/GI_遗传性高胆红素血症分流]]，本篇与之互补。

---

## 一、第一步永远是分流：结合 vs 非结合

> [!info] 直接胆红素占比是分水岭
> - **直接（结合）胆红素 > 总胆的 15-20%** → **结合性高胆红素血症 (conjugated hyperbilirubinemia)** = **永远是病理性**，必须查病因。
> - **直接胆红素 < 15%** → **非结合性高胆红素血症 (unconjugated hyperbilirubinemia)**，绝大多数良性/生理性，但要排溶血与核黄疸风险。

结合性（病理）的核心病因方向：

```
结合性高胆红素血症 (直接 >15-20%)
├── 胆道闭锁 (biliary atresia)        ← 最不能漏，需2月内手术
├── 新生儿肝炎 (neonatal hepatitis)
├── TORCH 感染                        ← 详见 OB_GYN_TORCH
├── 代谢病 (半乳糖血症/α1-抗胰蛋白酶缺乏 α1-antitrypsin deficiency/酪氨酸血症)
└── 胆总管囊肿 (choledochal cyst)
```

> [!warning] 结合性黄疸的红旗信号
> **陶土/白陶土便 (acholic/pale stool) + 深色尿 (dark urine) + 肝大** = 胆道梗阻方向，立即查直接胆红素、超声、转专科。这与非结合性黄疸的处理方向**完全相反**。

---

## 二、核心考点：喂养失败性黄疸 vs 母乳性黄疸

> [!danger] 这是最高频陷阱，名字像但机制/时间/处理都不同
> 两者都是**非结合性**、都发生在母乳喂养儿，但**时间轴**和**脱水有无**把它们分开。

| 鉴别点 | 喂养失败性黄疸 (Lactation Failure / Breastfeeding Failure Jaundice) | 母乳性黄疸 (Breast Milk Jaundice) |
|---|---|---|
| 别名 | breastfeeding jaundice、"母乳喂养不足性黄疸" | true breast milk jaundice |
| 机制 | 奶摄入不足 → 胆红素排出减少 + 肠肝循环 (enterohepatic circulation) 增加 | 母乳中 β-葡糖醛酸苷酶 (β-glucuronidase) ↑ → 肠内胆红素被解离重吸收 → 肠肝循环 ↑ |
| 出现时间 | **生后第 1 周内（<1 周，常 3-5 天）** | **>1 周出现，峰值约 2 周**，可持续数周 |
| 关键伴随 | **脱水**：体重降 >10%、湿尿布 < 日龄数、胎便/排便过渡延迟 | **喂养充分、不脱水**、体重正常增长、婴儿状态好 |
| 本质问题 | 奶**没喂够**（母乳量少/含接差/喂养间隔太长） | 奶**喂够了**，是母乳成分本身的作用 |
| 处理 | **加强喂养**（增加频率、纠正含接、必要时补充喂养/催乳），补足入量 | 多为**观察随访**，胆红素不达光疗线一般不停母乳；继续母乳喂养 |

> [!tip] 一句话记忆
> **喂不够 = 早（<1周）+ 脱水**；**母乳成分 = 晚（>1周）+ 不脱水**。
> 看到"体重掉太多 + 尿布少 + 才几天大" → 想**喂养失败**，不是母乳性黄疸。

## 三、生理性 vs 病理性黄疸

| 特征 | 生理性黄疸 (physiologic) | 病理性红旗 (pathologic) |
|---|---|---|
| 出现时间 | **>24 小时后**（常 2-4 天） | **生后 <24 小时出现** |
| 胆红素类型 | 非结合为主 | 结合升高，或非结合极高 |
| 上升速度 | 缓慢，自限 | 上升快（>5 mg/dL/天） |
| 总胆水平 | 不过高，不达换血线 | 过高 → **核黄疸 (kernicterus) 风险** |
| 病程 | 足月约 1 周内消退 | 持续 >2 周（迁延性）需查因 |

> [!danger] 核黄疸 (kernicterus)
> 极高**非结合**胆红素透过血脑屏障沉积基底节 → 不可逆神经损伤（肌张力异常、角弓反张、听力丧失、手足徐动型脑瘫）。这是积极监测/光疗/换血的根本理由。**结合胆红素不会引起核黄疸**（脂溶性差）。

## 四、其他非结合性病因（简表）

| 病因 | 识别线索 |
|---|---|
| 同种免疫溶血 — ABO (ABO incompatibility) | 母 **O 型**、婴 A 或 B 型；Coombs 可阳；生后早发黄疸 |
| 同种免疫溶血 — Rh (Rh isoimmunization) | 母 **Rh 阴性**、婴 Rh 阳性；常更重，可致胎儿水肿 |
| G6PD 缺乏（葡萄糖-6-磷酸脱氢酶缺乏 glucose-6-phosphate dehydrogenase deficiency） | 男婴、地中海/非洲/亚洲背景，氧化应激后溶血 |
| 头颅血肿 (cephalohematoma) / 产伤瘀血 | 血肿吸收 → 胆红素负荷增加，黄疸偏晚 |
| Gilbert 综合征 (Gilbert syndrome) | 轻度非结合升高，应激/禁食时明显；见分流笔记 |
| 红细胞增多症 (polycythemia) | 母胎/双胎输血、糖尿病母亲儿 |

## 五、实用口诀

> [!success] 床旁快速判断
> - **湿尿布数 ≈ 日龄天数**（第 3 天应有约 3-4 块湿尿布；明显少于此 = 入量不足）。
> - **体重较出生下降 >10% = 脱水红线**（生理性丢失通常 ≤7-10%）。
> - 才几天大 + 掉秤超线 + 尿布少 → **喂养失败**优先。
> - 直接胆红素 >15-20% → 跳出"良性"框架，查**胆道闭锁**。

## 六、方向相反的提醒：胆道闭锁 (Biliary Atresia)

> [!warning] 放这里是提醒鉴别方向相反
> - **结合性**高胆红素血症（与上面非结合病因方向相反）。
> - 出现时间 **2-8 周**（比生理性/喂养失败晚）。
> - **陶土便 (acholic stool) + 深色尿 + 肝大**。
> - 时间敏感：**Kasai 手术宜在生后 <2 个月**，漏诊代价大。
> - 见 [[完整笔记/专题笔记/GI/GI_无痛性黄疸与胆道梗阻定位]] 的胆道梗阻思路。

## 七、复习自测

> [!question]- Q1. 6 日龄纯母乳喂养男婴，每 4 小时喂一次，体重较出生下降 11%，24 小时仅 3 块湿尿布，非结合胆红素 14。最可能诊断？
> **喂养失败性黄疸 (lactation failure jaundice)**。
> 关键：生后 <1 周 + 明确脱水（掉秤 >10%、尿布少、喂养间隔过长入量不足）。虽也是母乳儿的非结合性黄疸，但**母乳性黄疸 (breast milk jaundice)** 通常 >1 周才出现且**不脱水**，故排除。处理 = **加强喂养/纠正喂养**补足入量。

> [!question]- Q2. 为什么"非结合胆红素 14、生后 <24 小时"要紧急处理，而结合胆红素升高不会引起核黄疸？
> 非结合（间接）胆红素脂溶性高，可穿过血脑屏障沉积基底节致**核黄疸 (kernicterus)**；结合胆红素水溶性、不入脑，但它升高意味着**病理性梗阻/肝病**，需查胆道闭锁等。生后 <24 小时出现的黄疸本身就是病理红旗（多为溶血）。

> [!question]- Q3. 母 O 型、婴 A 型，生后第 2 天出现黄疸，如何解释？
> **ABO 溶血 (ABO incompatibility)**：母体抗 A/抗 B IgG（免疫球蛋白 G immunoglobulin G）过胎盘致婴儿红细胞溶血 → 非结合胆红素升高、早发黄疸。查 Coombs、血型、血涂片。

# 儿科_婴儿喂养与营养补充时间轴

> [!tip] 一句话锁定
> **纯母乳喂养 (exclusively breastfed) 且生长正常的婴儿**，前 6 个月唯一需要常规补充的是 **维生素 D (vitamin D)**——热量/蛋白母乳基本够，唯独维生素 D 含量不足。

---

## 一、核心概念

> [!info] 为什么母乳婴儿仍要补维生素 D
> 母乳几乎能满足 0–6 月的热量与蛋白需求，但**维生素 D 含量偏低**，不足以预防佝偻病 (rickets)。因此健康、纯母乳、生长正常的婴儿仍需**常规补充维生素 D**。
> - 关键机制链：纯母乳 → 热量/蛋白足 → 维生素 D 不足 → 补充维生素 D。

---

## 二、婴儿喂养与营养时间轴

| 年龄 | 常规喂养 / 补充 | 记忆点 |
|---|---|---|
| 0–6 月 | 母乳或配方奶 (formula) 为主 | **纯母乳要补维生素 D** |
| ~4–6 月 | 视情况评估铁 (iron) / 氟 (fluoride) | 非本题主轴 |
| ~6 月 | 开始辅食 complementary foods（常从米粉/泥状开始） | 6 月前不推荐更早加辅食 |
| < 1 岁 | **不喝未改良全脂牛奶 (whole cow milk)** | 早给牛奶易致缺铁 (iron deficiency) |

> [!warning] 时间轴陷阱
> - 2 月龄谈"加辅食"= 过早（辅食约 6 月起）。
> - 生长百分位正常 → 不需要额外配方奶兜底，不改变"补维生素 D"这一常规建议。

---

## 三、补充剂速记

| 补充 | 对象 | 要点 |
|---|---|---|
| 维生素 D vitamin D | **所有纯母乳婴儿**（配方奶已强化则依摄入量） | 出生后即开始，预防佝偻病 |
| 铁 iron | 早产、纯母乳约 4–6 月起 | 足月纯母乳储铁约 6 月耗尽 |
| 氟 fluoride | 6 月龄后、水氟不足地区 | 防龋齿 |
| 维生素 B12 | 严格素食/纯素母亲的母乳婴儿 | 母乳 B12 依赖母亲状态 |

> [!success] Memory Hook
> **"Breast is best, but D is missing."** 母乳最好，唯独缺维生素 D。

---

## 四、母乳喂养禁忌 (Breastfeeding contraindications)

母乳是首选，但少数情况**禁止母乳喂养**——分**母体因素**（多为经乳汁/皮肤接触传播感染或药物暴露）与**婴儿因素**（代谢病无法处理乳糖）。

| 类别 | 禁忌 | 机制 / 说明 |
|---|---|---|
| **母体 (Maternal)** | 活动性未治疗结核 (active untreated TB, tuberculosis) | 呼吸道飞沫传播；规范治疗 ≥2 周、无传染性后可恢复 |
| | HIV 感染 (HIV infection)\* | 经乳汁垂直传播；\*仅在**发达国家配方奶易得**时为禁忌（资源匮乏地区母乳收益 > 风险，仍推荐） |
| | 乳房疱疹病灶 (herpetic breast lesions) | 单纯疱疹 (HSV) 直接接触传播；仅患侧乳房停喂、健侧可继续 |
| | 活动性水痘 (active varicella infection) | 水痘-带状疱疹病毒传播 |
| | 化疗或放疗 (chemotherapy or radiation therapy) | 细胞毒药物/放射性核素经乳汁暴露婴儿 |
| | 活动性物质使用障碍 (active substance use disorder) | 药物（酒精/可卡因/阿片等）经乳汁暴露 |
| **婴儿 (Infant)** | 半乳糖血症 (galactosemia) | 婴儿缺 GALT 酶无法代谢乳糖分解出的半乳糖 → 必须**无乳糖配方**，绝对禁母乳 |

> [!tip] 高频陷阱
> - **HIV 的星号**：USMLE 语境默认发达国家（配方奶易得）→ HIV 是禁忌；若题干强调"资源匮乏/无洁净水配奶"则母乳仍推荐。
> - **乳房疱疹是"患侧禁、健侧可"**，不是全面禁乳——别一刀切。
> - **半乳糖血症 = 唯一婴儿侧绝对禁忌**（新生儿筛查阳性 + 喂奶后呕吐/黄疸/白内障/大肠埃希菌败血症）→ 换无乳糖/无半乳糖配方。
> - **乳腺炎 (mastitis) 不是禁忌**：继续哺乳/排乳反而是治疗一部分（易被诱导错选停乳）。

---

> [!question]- 复习题：2 月龄、纯母乳、生长发育正常的婴儿，最合适的营养建议是？
> **开始补充维生素 D (vitamin D supplementation)**。
> 母乳热量蛋白够但维生素 D 不足；2 月龄加辅食过早；生长正常不需配方奶兜底。

> [!question]- 复习题：辅食 (complementary foods) 一般何时开始？1 岁前为何不喝全脂牛奶？
> 辅食约 **6 月**开始；1 岁前喝未改良全脂牛奶易致**缺铁**（低铁 + 影响吸收/隐性肠道失血）。

# 儿科_婴儿呕吐与喂养困难鉴别

> [!tip] 一句话锁定
> 婴儿**喂后吐奶 + 反弓 (back arching) + 像疼痛一样哭闹 + 打嗝频繁**，超过了"快乐吐奶者 (happy spitter)"，指向 **反流性食管炎 (reflux esophagitis) / 病理性胃食管反流病 (GERD, gastroesophageal reflux disease)**。

---

## 一、核心概念：生理性 GER vs 病理性 GERD

| | 生理性反流 GER | 病理性 GERD / 反流性食管炎 |
|---|---|---|
| 典型形象 | happy spitter（快乐吐奶者） | 喂后疼痛、反弓、拒食 |
| 生长 | 正常、体重增长好 | 增长欠佳 / 喂养厌恶 |
| 处理 | 安抚家长 + 体位/喂养调整，多自愈 | 加强抗反流措施，必要时 acid suppression |

> [!info] 升级线索（从 GER 变成 GERD）
> 出现下列任一即"不只是吐奶"：**喂后疼痛/反弓、喂养厌恶、体重增长不良、慢性呼吸道症状**。

---

## 二、机制

```
下食管括约肌 LES 不成熟
  → 胃内容物向上反流
  → 反复酸暴露刺激食管黏膜
  → 喂后疼痛 + 反弓（弓背是疼痛表现）
  → 喂养厌恶 / 体重增长不良
```

---

## 三、婴儿呕吐鉴别（高频对照）

| 诊断 | 高频线索 | 呕吐性质 |
|---|---|---|
| 生理性反流 GER | happy spitter，无痛 | 溢奶、无痛 |
| 反流性食管炎 / GERD | 反流 + 疼痛/反弓 + 喂养问题 | 喂后、非喷射 |
| 幽门狭窄 pyloric stenosis | 3–6 周龄、进行性、脱水、可触橄榄样包块 | **喷射性、非胆汁性** |
| 牛奶蛋白不耐受 CMPA (cow milk protein allergy) | 呕吐 + 腹泻 / 血便、湿疹 | 伴肠道症状 |
| 肠旋转不良 / 中肠扭转 malrotation/volvulus | 急性、病态 | **胆汁性**（急症） |

> [!danger] 别落入幽门狭窄陷阱
> 男婴 + 呕吐易让人反射想 **幽门狭窄 pyloric stenosis**，但幽门狭窄的关键是**进行性喷射性非胆汁性呕吐 + 脱水 + 低氯低钾代碱**；仅"喂后吐 + 反弓疼痛 + 生长偏低"而无喷射/脱水，更像 GERD。

> [!warning] 红旗（提示非单纯反流，需进一步查）
> 胆汁性呕吐、喷射性呕吐、血便、发热/嗜睡、腹胀、生长严重落后 → 需影像/进一步评估，不能当普通反流处理。

---

> [!question]- 复习题：4 月龄男婴喂后常吐奶、喂后弓背似疼痛、打嗝频繁、生长在 10th 百分位，最可能诊断？
> **反流性食管炎 / GERD**。
> "反弓 + 疼痛 + 喂养问题 + 增长偏低"= 从生理性反流升级为病理性；缺喷射性呕吐/脱水故不支持幽门狭窄。

> [!question]- 复习题：如何一句话区分 happy spitter 与 GERD？
> Happy spitter 吐奶但**无痛、生长好**；GERD 吐奶伴**疼痛/反弓/喂养厌恶/增长不良**。

## ⭐ 胃肠炎后继发性乳糖酶缺乏 (secondary lactase deficiency)（Q19783 锚定）

感染/炎症损伤小肠黏膜 → 暂时丢失刷状缘乳糖酶 (lactase) → 未吸收乳糖被结肠细菌发酵产氢 → 腹痛、腹胀鼓音、胀气 + 渗透性水样腹泻。多数周内随黏膜修复自愈。

| 维度 | 要点 |
|---|---|
| 钥匙 | 病毒胃肠炎后、重新引入乳制品(全脂牛奶)后腹泻加重；量多恶臭产气便；便无血无黏液；仅白天(渗透性) |
| 诊断 | 限乳糖饮食后症状缓解（多不需氢呼气试验） |
| 治疗 | 暂时限制乳糖 |

> [!warning] vs 牛奶蛋白诱导直肠结肠炎 (milk protein–induced proctocolitis)
> 牛奶蛋白过敏 = 非 IgE 介导、婴儿早期(<6月)、血丝黏液便；继发乳糖酶缺乏 = 紧接胃肠炎、恶臭产气便且无血。有血选过敏，产气恶臭选乳糖酶。

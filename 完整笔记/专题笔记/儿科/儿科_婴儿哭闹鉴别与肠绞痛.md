# 儿科_婴儿哭闹鉴别与肠绞痛

> [!tip] 一句话锁定
> **健康、生长良好、体格检查完全正常**的 <3 月龄婴儿，长时间无法安抚地哭闹，但**哭完之间是开心、警觉的 (happy and alert)** → **婴儿肠绞痛 (infantile colic)**；正确答案是**家长安慰 + 安抚技巧 (reassurance and soothing techniques)**，不是化验、不是影像、不是换配方、不是抗生素。

---

## 一、肠绞痛的诊断标准：3-3-3 + <3 月龄 ⭐

> [!tip] 记忆核心：3-3-3 规则（Wessel criteria 的通俗版）
> 一个**健康**婴儿，**无明显原因**的哭闹/烦躁发作：
> - **≥3 小时/天**
> - **≥3 天/周**
> - **持续 ≥3 周**
> - 年龄 **<3 月龄**（3 个 3 + 小于 3 个月）
>
> 自然病程：**6 周龄达高峰**，**3-4 月龄自行缓解 (self-limited)**。
> 好发时段：常在**每天同一时间段**发作，多为**傍晚/夜间 (evening)**。
> ⭐ **肠绞痛是排除性诊断 (diagnosis of exclusion)** —— 必须先完成**完整病史 + 全面体格检查**（含全身皮肤、四肢手指足趾、生殖器、眼、耳、囟门、腹部），全阴才能下这个诊断。

发作时的表现（不是必需，但常见）：面色涨红、握拳、**双腿蜷向腹部 (drawing legs up)**、腹部触之紧张、排气后似有缓解。这些"像肠痉挛"的表现**本身不诊断任何器质性病**，肠绞痛的病因至今不明（并非确证的"肠痉挛/胀气"）。

---

## 二、⭐ 关键判别线：发作**间期**是否正常

> [!danger] 这条线是分诊的命门
> **肠绞痛**：哭闹是**阵发**的，但**两次发作之间婴儿恢复正常 —— 开心、警觉 (happy and alert)、吃奶好、互动好、生长曲线正常**。
> **器质性急症**（尤以 **肠套叠 (intussusception)** 为代表）：阵发性哭闹之间婴儿**嗜睡、萎靡、无精打采 (lethargic / listless)**，越发作越差，中间不"回血"。
>
> 也就是说：判断不是看"哭得多凶"，而是看**不哭的时候是什么样**。

```
婴儿反复阵发性哭闹
        │
        ├─ 发作间期【开心、警觉、吃奶好、生长正常】
        │        └─ + 体检全阴 + 无发热
        │              └─ ⭐ 肠绞痛 → 家长安慰 + 安抚技巧
        │
        └─ 发作间期【嗜睡、萎靡、面色差】
                 └─ 器质性急症 ← 肠套叠 / 脓毒症 / 脑膜炎 / 扭转
                       └─ 立即查（影像 / 化验 / 全身体检）
```

> [!warning] 常见思维陷阱
> "哭了好几个小时、怎么哄都不停"这句话本身**不是**器质性疾病的证据 —— 肠绞痛的定义就是这样。真正提示器质性病的是**发热、发作间期萎靡、胆汁性呕吐、血便、腹部包块、生长不良、体检阳性发现**。

## 三、婴儿哭闹鉴别总表

下表按"最常见 → 不能漏"排列（基于 Nelson / AMBOSS / UpToDate 重述）。

| 诊断 | 关键特征 | 分诊要点 |
|---|---|---|
| 正常哭闹 (normal crying) | 间歇性、**可被安抚**、总量 **<3 小时/天**；生长正常 | 家长教育即可 |
| 婴儿肠绞痛 (infantile colic) | 3-3-3 + <3 月龄；傍晚为主；**发作间期开心警觉**；体检全阴 | 排除性诊断 → 安慰 + 安抚 |
| 胃食管反流病 (gastroesophageal reflux disease, GERD) | **频繁吐奶 + 喂后反弓 (back-arching)** + 似疼痛哭闹 | 重症（拒食 + 体重不增）可试 **H₂ 受体阻滞剂 (H₂ receptor blocker)** |
| 急性中耳炎 (acute otitis media, AOM) | 鼓膜**膨隆 (bulging)** ± 发热、揪耳 | 见 §四：单纯鼓膜发红不算 |
| 脑膜炎 (meningitis) | **发热 + 嗜睡 + 前囟膨隆 (bulging fontanelle)**；喂养差 | 急症，立即 workup |
| 化脓性关节炎 (septic arthritis) | **发热 + 患肢活动受限/假性瘫痪**，被动活动时哭 | 关节穿刺 |
| 尿路感染 (urinary tract infection, UTI) | **发热 + 呕吐 + 喂养差**，无定位体征 | 婴儿发热必查尿 |
| 肠套叠 (intussusception) | 阵发性哭闹 + **双腿蜷向腹部**、± **胆汁性呕吐 (bilious vomiting)**、**果酱样血便 (currant jelly stool)**、发作间期萎靡；好发 **6-36 月龄** | 超声（靶征）→ 空气/造影灌肠 |
| 睾丸扭转 (testicular torsion) / 卵巢扭转 (ovarian torsion) | 阴囊肿胀、触痛、提睾反射消失 / 腹部膨隆、可触包块 | 体检必须脱裤子看生殖器 |
| 头发止血带 (hair tourniquet) | 一根头发缠住**手指 / 足趾 / 阴茎**，远端红肿、环形压痕 | 脱袜脱衣逐指检查 → 剪除即愈 |
| 角膜擦伤 (corneal abrasion) | **流泪 + 畏光 (photophobia) + 眼睑痉挛**；**荧光素染色 (fluorescein staining)** 阳性 | 裂隙灯/荧光素 |
| 虐待性外伤 (child abuse) / 骨折 | **瘀斑、裂伤、肢体活动不对称**、病史与伤情不符、就诊延迟 | 骨骼系列 + 上报 |

> [!info] 年龄这条线很好用
> **肠绞痛 <3 月龄**，**肠套叠 6-36 月龄**（峰值 6-18 月）。一个 6 周龄婴儿阵发性哭闹、体检全阴、间期正常 —— 年龄本身就把肠套叠压得很低；反过来，10 月龄婴儿新发阵发性哭闹 + 间期萎靡，"肠绞痛"就是错误答案（年龄不符 + 早该缓解了）。

## 四、⚠️ 鼓膜发红 ≠ 急性中耳炎

> [!warning] 高频误诊陷阱
> **哭闹本身就会让鼓膜充血发红**（哭 → 血管扩张）。因此**单纯鼓膜红斑 (erythema alone) 不能诊断 AOM，也不能给抗生素**。
>
> 诊断 **急性中耳炎 (acute otitis media, AOM)** 需要：
> - **鼓膜膨隆 (bulging tympanic membrane)** —— 最特异的单一体征；和/或
> - **中耳积液 (middle ear effusion)** 的证据：**充气耳镜 (pneumatic otoscopy)** 下鼓膜**活动度/顺应性下降 (decreased mobility)**、气液平、鼓膜后积液
> - 加上急性起病的中耳炎症表现（耳痛/揪耳、发热）
>
> 一个哭到满脸通红的婴儿，两侧鼓膜都发红但**不膨隆、活动度正常** → 这是哭出来的，不是 AOM。

## 五、肠绞痛的管理：安慰 + 安抚技巧

> [!tip] 管理三件套
> 1. **家长安慰与教育 (parental reassurance)** —— 明确告诉家长：这是**良性、自限**的，**3-4 月龄会自行缓解**，不是家长的错，不是奶不好，也不代表孩子有病。（家长的焦虑与耗竭本身就是要处理的问题，也是**虐待性头外伤 / 摇晃婴儿综合征 (abusive head trauma / shaken baby syndrome)** 的风险因素，要主动交代"哄不住时可以把孩子放回婴儿床、离开几分钟冷静"。）
> 2. **复习安抚技巧 (soothing techniques)**：
>    - **安抚奶嘴 (pacifier)**
>    - **抱、轻摇 (rocking)、襁褓包裹 (swaddling)**
>    - **减少环境刺激**：暗房、安静、避免过度逗弄
>    - **秋千 (infant swing)、背带 (carrier)、推车/兜风**，同时也**让家长得以轮换休息**
> 3. **调整喂养技巧**：瓶喂时保持**较竖直体位**、选择合适奶嘴、喂中与喂后**拍嗝**，减少**吞气 (aerophagia)**；避免过度喂养。

> [!danger] 明确不需要做的事
> - ❌ **不需要影像学**（腹部超声/X 线）——体检全阴 + 间期正常时
> - ❌ **不需要抗生素**（尤其别为"鼓膜红"开药）
> - ❌ **不需要换配方**（不常规换无乳糖或深度水解配方）
> - ❌ **不用促动力药/解痉药当标准答案**：**西甲硅油 (simethicone)** 无效证据；**双环维林 (dicyclomine)** 在婴儿**禁用**（呼吸暂停、癫痫风险）
> - ❌ 母乳喂养不需要断母乳；母亲饮食回避不是一线

## 六、⭐ "Reassurance 才是答案"的反射训练

USMLE 在儿科最爱考的一个反射就是：**什么时候该忍住不查**。过度检查与过度用药 (overtesting / overtreatment) 是 Step 2 CK 的经典错误答案。

```
婴儿哭闹 → 先跑一遍【真的要查】清单
   ┌───────────────────────────────────────┐
   │ 发热                                   │
   │ 嗜睡 / 发作间期萎靡                     │
   │ 胆汁性呕吐                              │
   │ 血便（果酱样）                          │
   │ 腹部包块 / 腹胀 / 腹膜刺激征             │
   │ 生长不良（体重曲线掉线）                 │
   │ 体格检查任何阳性发现                     │
   │   （鼓膜膨隆 / 前囟膨隆 / 阴囊肿胀 /      │
   │     指趾环形压痕 / 瘀斑 / 肢体不动 /      │
   │     荧光素染色阳性）                     │
   └───────────────────────────────────────┘
          │                        │
     任一为【是】               全部为【否】
          │                        │
     针对性 workup            + 生长良好
     （超声 / 尿检 / 腰穿 /    + 发作间期正常
       X 线 / 荧光素…）              │
                                   ▼
                        ⭐ 安慰 + 安抚技巧（答案）
```

> [!info] 选项识别技巧
> stem 反复强调 "**afebrile**"、"**growing well / weight at 50th percentile**"、"**normal physical examination**"、"**feeds well and is alert between episodes**" —— 这些**阴性/正常描述不是废话，是答案的一部分**：出题人正在把所有器质性诊断逐个枪毙，剩下的唯一出口就是 reassurance。

## 七、婴儿"乳糖不耐"的真面目

> [!warning] 别把肠绞痛当乳糖不耐
> 婴儿真正需要换**无乳糖配方 (lactose-free formula)** 的**原发性乳糖不耐 (congenital / primary lactase deficiency)** 极为**罕见**。
> 典型表现是**泡沫状水样便 (frothy, watery diarrhea)** ± 腹胀、肛周红斑、体重不增。
> ⭐ 只要 stem 写了**排便正常 (normal stools)**，就可以直接排除乳糖不耐，也就没有换配方的理由。
> （婴儿更常见的是**继发性乳糖不耐**——急性肠胃炎后一过性；以及 **牛奶蛋白过敏 (cow's milk protein allergy, CMPA)**——表现为**血丝便 + 湿疹 + 呕吐/腹泻**，那是另一条赛道，见喂养困难篇。）

## 八、复习题

> [!question]- 6 周龄足月男婴，近 4 周每天傍晚哭闹 3-4 小时、难以安抚；不发热，吃奶好，体重在 60th 百分位且稳定上升，排便正常；发作之间婴儿开心、警觉；全面体检（含生殖器、指趾、眼、耳）完全正常。最恰当的下一步？
> **家长安慰 + 复习安抚技巧（reassurance and soothing techniques）**。
> 这是典型 **婴儿肠绞痛 (infantile colic)**：3-3-3 + <3 月龄 + 傍晚发作 + **发作间期开心警觉** + 体检全阴 + 生长良好。
> 关键点：明确告知**自限、3-4 月龄缓解**；教安抚奶嘴 / 抱摇 / 襁褓包裹 / 暗房安静 / 秋千背带；调整瓶喂体位减少吞气。
> ❌ 不做腹部超声、❌ 不给抗生素、❌ 不换无乳糖配方（排便正常已排除）。

> [!question]- 同一个哭闹婴儿，耳镜见双侧鼓膜发红。可以诊断急性中耳炎并开阿莫西林 (amoxicillin) 吗？
> **不可以**。**哭闹本身即可造成鼓膜充血发红**。
> 诊断 **急性中耳炎 (acute otitis media, AOM)** 需要 **鼓膜膨隆 (bulging)** 和/或 **中耳积液**证据（**充气耳镜 pneumatic otoscopy** 下活动度/顺应性下降）。
> 单纯红斑 + 鼓膜活动度正常 = 不是 AOM，不给抗生素。

> [!question]- 哪一条线索最能把肠绞痛与肠套叠 (intussusception) 分开？
> **发作间期的状态**。
> 肠绞痛：两次哭闹之间婴儿**开心、警觉 (happy and alert)**、吃奶好。
> 肠套叠：阵发哭闹之间**嗜睡、萎靡 (lethargic)**，并常有**胆汁性呕吐**、**果酱样血便**、腹部腊肠样包块；好发年龄 **6-36 月龄**，与肠绞痛的 **<3 月龄** 也不重叠。

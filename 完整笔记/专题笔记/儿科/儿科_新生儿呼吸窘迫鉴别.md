# 新生儿呼吸窘迫鉴别（Neonatal Respiratory Distress）

> [!info] 笔记定位
> **生后数分钟~数小时出现呼吸窘迫**（呻吟 grunting / 凹陷 retractions / 鼻翼扇动 / 发绀 / 呼吸增快）是新生儿急诊高频套路。考点固定三步：① **胎龄 + 分娩方式 + 羊水**先把肺源 4 病（RDS / TTN / MAS / 肺炎）分开 ② **一个独特体征**（舟状腹 / 单侧呼吸音消失 / 喂养发绀）锁外科/畸形病 ③ **第一步处理**最常考——尤其 CDH「禁 bag-mask」。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q2435]]（CDH 误选胸管）。
> 素材整合自 [[完整笔记/Peixuan分科笔记/儿科]] / [[完整笔记/Peixuan分科笔记/newborn care and screening]] / [[完整笔记/Peixuan分科笔记/pulmonary]]（均只读）。

## 一、肺源 4 病：胎龄 + 分娩方式定病因 ⭐⭐⭐

> [!tip] 三联鉴别钥匙（先背这一句）
> **早产 = RDS；剖宫产/急产足月 = TTN；足月 + 胎粪污染羊水 = MAS。** 加一句感染：**母 GBS / 胎膜早破 + 任何胎龄 = 新生儿肺炎/败血症**。

| 病 | 钥匙（病史钩子）| 机制 | CXR 影像 | 处理 |
|---|---|---|---|---|
| 新生儿呼吸窘迫综合征 NRDS (RDS, respiratory distress syndrome) | **早产 <34 周**、母糖尿病 | 表面活性物质 (surfactant) 缺乏 → 肺泡塌陷 + 肺内右向左分流 | **胸片 (CXR, chest X-ray) 白肺 white-out / 毛玻璃 + 支气管充气征**；卵磷脂/鞘磷脂比 (L/S, lecithin/sphingomyelin ratio) <2:1 | 产前倍他米松 (betamethasone) 预防；生后 surfactant + 持续气道正压 (CPAP, continuous positive airway pressure)/O₂ |
| 新生儿暂时性呼吸增快 TTN (transient tachypnea of newborn) | **剖宫产 / 急产**、足月、**生后即起、数小时~48h 自限** | 肺液吸收延迟 (retained fetal lung fluid) | **肺门纹理增多 + 叶间裂积液**（湿肺）| **支持治疗**（O₂、观察），自限 |
| 胎粪吸入综合征 MAS (meconium aspiration syndrome) | **足月/过期产 + 羊水胎粪污染 (meconium-stained)** | 胎粪堵气道 → 活瓣 air-trapping + 化学性肺炎（**胎粪本身无菌**）| **斑片影 + 过度充气**；可气胸 | O₂ + 呼气末正压 (PEEP, positive end-expiratory pressure)；重症 → 新生儿持续性肺动脉高压 (PPHN, persistent pulmonary hypertension of the newborn)，可需吸入性一氧化氮 (iNO, inhaled nitric oxide) / 体外膜肺氧合 (ECMO, extracorporeal membrane oxygenation) |
| 新生儿肺炎 / 早发败血症 | **母 B 族链球菌 (GBS, group B Streptococcus)(+) / 胎膜早破 (PROM, premature rupture of membranes) >18h / 母发热** | 宫内/产时感染（GBS、大肠杆菌 E. coli、李斯特菌 Listeria）| 双侧浸润、可似 RDS | **经验抗生素 氨苄西林 ampicillin + 庆大霉素 gentamicin** |

> [!warning] 母糖尿病的双重陷阱
> 母糖尿病婴儿（IDM, infant of diabetic mother）即使**足月**也可 RDS（高胰岛素延迟 surfactant 成熟）；同时是 **TGA（大动脉转位）+ 室间隔肥厚**的高危——发绀 + 单一响亮 S₂ 想 TGA（见下 §三）。

## 二、CDH：舟状腹 + 单侧呼吸音消失 + 心音对侧移位 ⭐⭐⭐

> [!danger] 一眼诊断 + 第一步（最高频考点）
> **先天性膈疝 (CDH, congenital diaphragmatic hernia)** 三联：
> **① 舟状腹 (scaphoid abdomen) ② 患侧（多左侧）呼吸音消失、圆隆胸 (barrel chest) ③ 心音移向对侧（右胸最响）。**
> 第一步 = **气管插管 (endotracheal intubation) + 胃管/鼻胃管 (NG/OG tube) 减压**。
> ❌ **禁 bag-mask 正压通气**——气进入疝入胸腔的肠管 → 肠胀 → 进一步压肺，越救越糟。
> ❌ **不是胸管 (chest tube)**——无气胸；胸管可刺破疝入的肠管。

```
左侧 CDH 病理链
膈肌（胸膜腹膜褶）融合不全
        │
   腹腔脏器疝入胸腔
        │
 ┌──────┴───────┐
患侧肺受压        纵隔右移
（肺发育不良       （心音右胸最响）
 pulmonary
 hypoplasia）
        │
 肺血管床↓ → 肺动脉高压 (PPHN)
        │
   生后顽固低氧 + 发绀
```

- **诊断**：产前超声常已发现；生后胸片（babygram）见胸腔内肠袢气体 + 纵隔移位 + 腹部气体少。
- **最终治疗**：稳定（插管通气 + 减压 + 处理 PPHN）后**择期手术修补**——先稳后修，不是急诊开刀。
- **为何易错**：「单侧呼吸音消失 + 呼吸窘迫」第一反射是张力气胸 → 想放胸管。**舟状腹 + 心音对侧移位 + 圆胸**是把 CDH 从气胸里拎出来的三个钩子。

## 三、外科/畸形 + 心源：别被「呼吸窘迫」带偏 ⭐⭐

| 病 | 独特钩子（一钩一病）| 第一步 / 关键 |
|---|---|---|
| **食管闭锁 + 气管食管瘘 (EA + TEF, esophageal atresia + tracheoesophageal fistula)** | **喂养即呛咳/发绀 + 口吐泡沫**；**NG 管在盲袋打卷**（X 线见管盘在上纵隔）；羊水过多史；查 **VACTERL** | 禁喂、抬高头位、盲袋吸引 → 手术 |
| **后鼻孔闭锁 (choanal atresia)** | **喂养时发绀、哭闹时反而转红**（婴儿强制经鼻呼吸）；**CHARGE association** | 经口气道/口咽通气道 → 手术 |
| **新生儿气胸 (pneumothorax)** | 真张力气胸：单侧呼吸音消失 + **气管/心音推向对侧** + **透光试验 (transillumination) 阳性**；MAS/正压通气后多见；**腹平坦不舟状** | 张力 → 紧急穿刺/胸管（与 CDH 的鉴别全靠舟状腹 + 透光试验）|
| **导管依赖型发绀先心 (ductal-dependent CHD, congenital heart disease)** | **发绀但呼吸窘迫轻、与缺氧不成比例**；**高氧试验 (hyperoxia test)：100% O₂ 后 PaO₂ 仍 <100**（肺源则 >250）；TGA = **单一响亮 S₂** + 母糖尿病 | **前列腺素 E1 (PGE1, alprostadil 前列地尔) 保持动脉导管开放** → 心超 → 手术 |

> [!tip] 高氧试验 (hyperoxia test) 一句话
> 给 100% O₂ 10–15 min 后测**右桡动脉** PaO₂：**>250 = 肺源**（RDS/TTN/MAS/肺炎，给氧能改善）；**<100 = 心源右向左分流**（导管依赖先心，给氧无效 → 上 PGE1）。

## 四、第一步处理速查（考点收口）

> [!danger] 「下一步最佳处理」高频映射
> | 线索组合 | 病 | 第一步 |
> |---|---|---|
> | 舟状腹 + 单侧呼吸音消失 + 心音对侧 | **CDH** | **插管 + 胃管减压；禁 bag-mask、非胸管** |
> | 早产 + 白肺 + 毛玻璃 | RDS | surfactant + CPAP（产前 betamethasone 预防）|
> | 剖宫产足月 + 数小时自限 | TTN | 支持 + O₂ |
> | 胎粪羊水 + 过度充气 | MAS | O₂ + PEEP（重症防 PPHN）|
> | 喂养呛咳 + NG 盲袋打卷 | EA+TEF | 禁喂 + 盲袋吸引 → 手术 |
> | 喂养发绀、哭时转红 + CHARGE | 后鼻孔闭锁 | 口咽通气道 → 手术 |
> | 发绀重 + 窘迫轻 + 高氧试验 PaO₂<100 | 导管依赖先心 | **PGE1** |
> | 单侧音消失 + 透光阳性 + 气管对侧 | 张力气胸 | 紧急穿刺/胸管 |

## 一、常染色体三体（Autosomal Trisomies）

> [!info]- Down syndrome（唐氏综合征，Trisomy 21 / T21，21 三体）
> - **机制/基因**：Trisomy 21（染色体数目异常，多一条 21 号）；与母亲高龄相关（>35 岁风险↑）；发病率 1/700（最常见活产染色体病）。
> - **面容/体征**：upslanting palpebral fissures（上斜眼裂）+ flat nasal bridge（扁平鼻梁）+ epicanthal folds（内眦赘皮）+ single palmar crease（单掌纹 / simian crease）+ 舌大 + 低位耳 + 短颈 + 第 5 指内弯 + sandal gap（拇趾分离）。
> - **多系统**：先心 **AVSD（房室间隔缺损）> VSD（室间隔缺损，ventricular septal defect）/ ASD（房间隔缺损，atrial septal defect）**；消化 **十二指肠闭锁（double-bubble sign）+ Hirschsprung 病**；血液 **ALL（急性淋巴细胞白血病，acute lymphoblastic leukemia）/ AML（急性髓系白血病，acute myeloid leukemia）风险↑（10–20 倍）+ 新生儿一过性骨髓增殖**；内分泌 **甲减风险↑**；神经 智力障碍 + 肌张力低 + **早发 Alzheimer（阿尔茨海默病，40 岁后）**；骨科 **寰枢椎不稳（atlantoaxial instability）→ 运动前查 X 线**；耳 传导性听力丧失（中耳炎 + 咽鼓管功能不良）。
> - **化验/诊断**：quad screen **↓AFP, ↑hCG, ↓uE3, ↑inhibin A**；确诊 **karyotype（核型分析）**。
> - **Buzzword/Hook**：单掌纹 + 上斜眼 + 舌大 + friendly demeanor + 先心 → **"Down's heart cushioned"（心内膜垫缺损 AVSD）**。

> [!info]- Edwards syndrome（爱德华兹综合征，Trisomy 18 / T18，18 三体）
> - **机制/基因**：Trisomy 18；发病率 1/6000；预后差（**>50% 1 月内死亡，<10% 活过 1 岁**）。
> - **面容/体征**：**小下颌（micrognathia）+ 突出枕骨（prominent occiput）** + 小眼 + **重叠指（overlapping fingers / clenched fist）** + **摇椅足（rocker-bottom feet）**。
> - **多系统**：先心 **VSD > ASD/PDA（动脉导管未闭，patent ductus arteriosus）**；消化 脐膨出 + 肠旋转不良；泌尿 马蹄肾；**严重喂养困难**（小下颌 + 张力差 + 中枢异常 + 先心疲劳）；多脏器畸形。
> - **化验/诊断**：quad screen **全部↓**（AFP/hCG/uE3/inhibin A 皆低）；确诊 karyotype。
> - **Buzzword/Hook**：重叠指 + 摇椅足 + 突出枕骨 → Edwards → **"Edwards' V"（18 倒过来像 V = VSD）**。

> [!info]- Patau syndrome（帕陶综合征，Trisomy 13 / T13，13 三体）
> - **机制/基因**：Trisomy 13；发病率 1/10000；预后最差（**80% 1 月内死亡**）。
> - **面容/体征**：**唇腭裂（cleft lip/palate）+ 小眼/无眼（microphthalmia）+ 前额倾斜** + **多指（polydactyly）+ 摇椅足** + **皮肤缺损（cutis aplasia）**。
> - **多系统**：先心 VSD/PDA/ASD（混合）；神经 **Holoprosencephaly（前脑无裂畸形）+ 严重智障**；消化 脐膨出；泌尿 多囊肾；严重喂养困难（唇腭裂 + 中枢异常）。
> - **化验/诊断**：quad 通常正常，需 **NIPT（无创产前检测）/ 超声**；确诊 karyotype。
> - **Buzzword/Hook**：唇腭裂 + 多指 + 皮肤缺损 + 小眼 → Patau → **"13 太严重，啥都坏"（混合心脏畸形 + 前脑无裂）**。

## 二、性染色体异常（Sex Chromosome Aneuploidies）

> [!info]- Turner syndrome（特纳综合征，45,XO）
> - **机制/基因**：单个 X（45,XO）；女性；发病率 1/2500 女婴；**SHOX 基因缺失 → 矮小**。
> - **面容/体征**：**颈蹼（webbed neck）+ 宽间距乳头 + 肘外翻 + 低发际 + 新生儿手足淋巴水肿（lymphedema）** + 盾状胸。
> - **多系统**：心脏 **Bicuspid aortic valve（二叶主动脉瓣）> Coarctation of aorta（主动脉缩窄）**；生殖 **卵巢发育不全（streak ovary，条索状卵巢）+ 原发闭经 + 不育**；泌尿 马蹄肾；肿瘤 若含 Y 物质 → 性腺母细胞瘤（需切条索状性腺）。
> - **化验/诊断**：**FSH/LH↑, estrogen↓**（高促性腺低性腺）；智力多正常但**空间/数学弱**；确诊 karyotype。
> - **Buzzword/Hook**：矮小 + 颈蹼 + 原发闭经 + 心脏缩窄 → **"Turner 上下不通"（上肢高压、下肢弱脉）**。

> [!info]- Klinefelter syndrome（克兰费尔特综合征，47,XXY）
> - **机制/基因**：多一条 X（47,XXY）；男性；发病率 1/500–1000 男婴。
> - **面容/体征**：**高瘦（长肢）+ 男性乳房发育（gynecomastia）+ 小睾（small firm testes）**。
> - **多系统**：生殖 **不育 + testosterone↓**；肿瘤 **乳腺癌风险↑（20–50 倍）+ 纵隔生殖细胞瘤**；认知 轻度学习障碍 + 语言弱（智力多正常）。
> - **化验/诊断**：**FSH/LH↑, testosterone↓**（高促性腺低性腺）；确诊 karyotype。
> - **Buzzword/Hook**：高瘦男 + 男乳 + 小睾 + 不育 → **"多 X 男乳大乳癌"（XXY → gynecomastia + 乳腺癌）**。

> [!info]- 47,XYY 与 Triple X（47,XXX）— 轻表型速记
> - **47,XYY**（男）：高大 + 轻度学习/行为问题；发病率 1/1000 男婴；多数正常生活。
> - **47,XXX / Triple X**（女）：高 + 轻度学习障碍；发病率 1/1000 女婴；**大多无症状、常漏诊**。
> - **Hook**：两者都"高 + 轻度学习障碍"，是 stem 里的低特异钓饵，别过度诊断。

## 三、微缺失 / 印记综合征（Microdeletion & Imprinting）

> [!info]- DiGeorge syndrome（迪乔治综合征，22q11.2 缺失）
> - **机制/基因**：22q11.2 微缺失（第三/四咽囊发育异常）。
> - **面容/体征**：abnormal facies（异常面容）+ cleft palate（腭裂）。
> - **多系统（CATCH-22 记忆）**：**C**ardiac defect（**Truncus arteriosus 永存动脉干 / ToF 法洛四联症 / 主动脉弓中断 IAA**）+ **A**bnormal facies + **T**hymic aplasia（胸腺发育不全 → T 细胞缺陷 → 反复病毒/真菌感染）+ **C**left palate + **H**ypocalcemia（甲状旁腺发育不全 → 低钙抽搐）。
> - **化验/诊断**：**血钙↓, PTH（甲状旁腺激素，parathyroid hormone）↓, T 细胞↓（CD3/CD4/CD8）**；确诊 **FISH（荧光原位杂交，fluorescence in situ hybridization）/ 染色体微阵列（CMA, chromosomal microarray）**。
> - **Buzzword/Hook**：新生儿低钙抽搐 + 反复病毒/真菌感染 + 主动脉弓异常 → **"22 个真没胸腺"（CATCH-22）**。

> [!info]- Williams syndrome（威廉姆斯综合征，7q11.23 缺失，含 elastin 基因）
> - **机制/基因**：7q11.23 微缺失（含 **elastin 弹性蛋白基因**）。
> - **面容/体征**：**精灵脸（elfin facies）+ 星状虹膜（stellate iris）**。
> - **多系统**：心脏 **Supravalvular aortic stenosis（SVAS，主动脉瓣上狭窄）+ 周围肺动脉狭窄**；代谢 **高钙血症（hypercalcemia）**；生长 FTT（生长发育停滞，failure to thrive）；行为 **过度友好外向（"cocktail party" personality）**。
> - **化验/诊断**：**血钙↑**；确诊 FISH / CMA。
> - **Buzzword/Hook**：精灵脸 + 高钙 + 瓣上主动脉狭窄 + 外向 → **三联 "W-AS-Ca-FTT"（瓣上窄 + 高钙 + 长不大）**。

> [!info]- Cri-du-chat syndrome（猫叫综合征，5p 缺失 / 5p-）
> - **机制/基因**：5 号染色体短臂缺失（5p-）。
> - **面容/体征**：圆脸 + 小头 + 眼距宽。
> - **多系统**：**高调猫叫样哭声（cat-like cry，喉部发育异常）** + 严重智障 + 喂养困难（喉异常 + 张力低 + 小下颌）+ 部分 VSD。
> - **化验/诊断**：确诊 FISH / CMA。
> - **Buzzword/Hook**：**婴儿猫叫样高调哭声** → Cri-du-chat。

> [!info]- Prader-Willi syndrome（普拉德-威利综合征 / 普-威综合征，15q11-13 父源缺失）
> - **机制/基因**：**父源（paternal）15q11-13 缺失** 或母源单亲二体（UPD, uniparental disomy）；印记疾病。
> - **面容/体征**：**almond eyes（杏仁眼）+ 小手小脚**。
> - **多系统（双相）**：**婴儿期 严重 hypotonia（肌张力低下）+ 喂养困难/吸吮差 + FTT** →（2–3 岁后）**hyperphagia（暴食）→ 肥胖 + 觅食执念**；性腺发育不全（hypogonadism）+ 轻中度智障。
> - **化验/诊断**：**甲基化分析（methylation analysis）** 区分父源缺失 vs 母源 UPD。
> - **Buzzword/Hook**：新生儿 floppy + 吃不下 → 后期暴食肥胖 → **"父走子饿"（P = Paternal）**。

> [!info]- Angelman syndrome（天使综合征，15q11-13 母源缺失）
> - **机制/基因**：**母源（maternal）15q11-13 缺失**（**UBE3A** 母源等位基因失活）或父源 UPD；印记疾病。
> - **面容/体征**：microcephaly（小头）+ 金发白肤（色素浅）+ 张口伸舌。
> - **多系统**：**发育迟缓（delay，非退化）+ 婴儿期 hypotonia + 喂养困难** →（后）**happy demeanor（异常快乐）+ 无诱因阵发大笑 + jerky ataxic gait（共济失调步态，"puppet-like"）+ 几乎无语言 + 癫痫（2–3 岁起）**。
> - **化验/诊断**：甲基化分析区分母源缺失 vs 父源 UPD。
> - **Buzzword/Hook**：快乐爱笑 + 共济失调 + 癫痫 → **"Happy Puppet"，"母走子笑"（A = mAternal）**。

## 四、三核苷酸重复病（Trinucleotide Repeat Disorders）

> [!info]- Fragile X syndrome（脆性 X 综合征，FMR1 基因）
> - **机制/基因**：**FMR1 基因 CGG 重复 >200**（X 连锁 + 表观甲基化沉默）；**第二常见智障病因（仅次于 Down）**；非染色体病。
> - **面容/体征**：**长脸 + 突出下颌与额（prominent chin & forehead）+ 大耳前突（large protruding ears）+ 巨头（macrocephaly）**。
> - **多系统**：**巨睾（macroorchidism，青春期后）** + tall stature（高大）+ 关节松弛 + 二尖瓣脱垂（MVP）；行为 **自伤行为（如咬手）+ 焦虑 + 自闭谱系（autistic behaviors）**。
> - **化验/诊断**：**FMR1 DNA 分析（CGG 重复 PCR）**。
> - **Buzzword/Hook**：男孩 + 高 + 大耳 + 长脸 + 自闭 + 大睾 → **"长脸大耳大睾丸"**；与 Down 身高方向相反。

> [!info]- Huntington disease（亨廷顿病，HTT 基因）
> - **机制/基因**：**HTT 基因 CAG 重复**；常染色体显性；**遗传早现（anticipation，父系遗传更明显）**。
> - **面容/体征**：中年起病（30–50 岁），非儿科 dysmorphism。
> - **多系统**：**舞蹈样动作（chorea）+ 进行性痴呆 + 精神症状（抑郁/易怒）**；尾状核萎缩（caudate atrophy → 侧脑室"box-car"扩大）。
> - **化验/诊断**：HTT CAG 重复基因检测；MRI 见尾状核萎缩。
> - **Buzzword/Hook**：中年 + 舞蹈 + 痴呆 + 家族史 + anticipation → Huntington。

> [!info]- Myotonic dystrophy（强直性肌营养不良，DMPK 基因）
> - **机制/基因**：**DMPK 基因 CTG 重复**；常染色体显性 + anticipation。
> - **面容/体征**：长面 + 颞肌/咬肌萎缩 + 额秃。
> - **多系统**：**肌强直（myotonia，握拳后松开困难）+ 面肌无力 + 白内障 + 秃顶 + 心律失常（传导阻滞）+ 胰岛素抵抗**。
> - **化验/诊断**：DMPK CTG 重复基因检测；EMG（肌电图，electromyography）见肌强直放电。
> - **Buzzword/Hook**：握手不松 + 白内障 + 额秃 + 心律失常 → Myotonic dystrophy。

> [!info]- Friedreich ataxia（弗里德赖希共济失调，FXN 基因）
> - **机制/基因**：**FXN 基因 GAA 重复**；常染色体**隐性**（唯一隐性的三核苷酸重复病）。
> - **面容/体征**：**足弓高（pes cavus）+ 脊柱侧弯**。
> - **多系统**：**进行性共济失调（步态 + 肢体）+ 构音障碍 + 深感觉丧失（后索）** + **肥厚型心肌病（HCM，死因）+ 糖尿病**。
> - **化验/诊断**：FXN GAA 重复基因检测；心超见 HCM。
> - **Buzzword/Hook**：儿童/青少年共济失调 + HCM + 糖尿病 + 高足弓 → Friedreich。

## 五、代谢病 / 溶酶体贮积病（Metabolic / Lysosomal Storage Diseases）

> [!info]- Tay-Sachs disease（泰-萨克斯病，TSD）
> - **机制/基因**：**己糖胺酶 A（Hexosaminidase A，Hex-A）缺陷 → GM2 ganglioside 堆积**；常染色体隐性；Ashkenazi Jewish（阿什肯纳兹犹太人）高发。
> - **面容/体征**：**进行性巨头（macrocephaly，神经元储积 + 胶质增生）**；**无肝脾大**（区别 Niemann-Pick 的关键）。
> - **多系统/时间线**：出生正常 → **约 3–6 月起发育停滞 + 退化**，首发常是**听觉过度惊跳反射（exaggerated startle / hyperacusis）** → 6–12 月进行性 hypotonia→spasticity + **视力丧失** → 2–5 岁前死亡；眼底 **cherry-red macula（樱桃红斑）**。
> - **化验/诊断**：血 **hexosaminidase A 活性↓**。
> - **Buzzword/Hook**：樱桃红斑 + **无肝脾大** + 惊跳亢进 → **"Tay-saCHS = No hepatosplenomegaly"**。

> [!info]- Niemann-Pick disease（尼曼-匹克病，NPD，A 型）
> - **机制/基因**：**鞘磷脂酶（sphingomyelinase）缺陷 → sphingomyelin（鞘磷脂）堆积**；常染色体隐性；Ashkenazi Jewish 高发。
> - **面容/体征**：**肝脾大（hepatosplenomegaly）**（与 Tay-Sachs 的鉴别钥匙）。
> - **多系统**：进行性神经退化 + hypotonia + FTT；眼底 **cherry-red macula（樱桃红斑）**（与 TSD 相同）。
> - **化验/诊断**：sphingomyelinase 活性↓；骨髓见泡沫细胞（foam cells）。
> - **Buzzword/Hook**：樱桃红斑 + **有肝脾大** → **"Niemann-Pick 起肝脾"**。

> [!info]- Phenylketonuria（苯丙酮尿症，PKU）
> - **机制/基因**：**苯丙氨酸羟化酶（phenylalanine hydroxylase）缺陷**（或 BH4 辅因子缺陷）→ 苯丙氨酸堆积；常染色体隐性；美国新生儿筛查覆盖。
> - **面容/体征**：**皮肤/毛发/虹膜色浅（黑色素↓，酪氨酸合成受阻）**。
> - **多系统**：**未筛查/未治疗**才发病 → **智力障碍 + 癫痫 + 自始发育迟缓（无正常发育期）**；**musty/mousy odor（霉味，尿/体味）**。
> - **化验/诊断**：血 **phenylalanine↑**；治疗 = 低苯丙氨酸饮食（限制天冬甜素 aspartame）。
> - **Buzzword/Hook**：霉味 + 色素浅 + 智障 + 无正常发育期 → PKU（**海外收养/未筛查**时尤其考虑）。

> [!info]- Hurler syndrome（MPS I，黏多糖贮积症 I 型，α-L-iduronidase 缺陷）
> - **机制/基因**：**α-L-iduronidase 缺陷 → 硫酸皮肤素 + 硫酸乙酰肝素堆积**；常染色体隐性；黏多糖病（mucopolysaccharidosis）中**较重型**。
> - **面容/体征**：**粗糙面容（coarse facial features）+ 巨舌 + 角膜混浊（corneal clouding）**。
> - **多系统**：**肝脾大 + 智力障碍 + 骨骼畸形（dysostosis multiplex）+ 关节僵硬 + 心脏瓣膜病**；发育退化。
> - **化验/诊断**：尿 GAGs（糖胺聚糖）↑；酶活性检测。
> - **Buzzword/Hook**：粗面容 + 角膜混浊 + 肝脾大 + 智障 → Hurler（**有角膜混浊**）。

> [!info]- Hunter syndrome（MPS II，黏多糖贮积症 II 型，iduronate sulfatase 缺陷）
> - **机制/基因**：**iduronate-2-sulfatase 缺陷**；**X 连锁隐性（只男孩）** —— 与 Hurler（常隐）关键区别。
> - **面容/体征**：粗糙面容 + 巨舌；**无角膜混浊**（与 Hurler 的鉴别钥匙）。
> - **多系统**：肝脾大 + 智力障碍 + 骨骼畸形 + **攻击行为**；较 Hurler 轻、进展慢。
> - **化验/诊断**：尿 GAGs↑；酶活性检测。
> - **Buzzword/Hook**：男孩 + 粗面容 + **无角膜混浊** + 攻击行为 → Hunter（**"Hunter 看得清，会打猎"= 无角膜混浊**）。

## 六、非遗传但常同框（鉴别用）

> [!info]- Fetal alcohol syndrome（胎儿酒精综合征，FAS）— 致畸非遗传
> - **机制**：**母亲孕期饮酒（teratogen，无安全剂量，first trimester 面容受累最重）**；非基因病、非染色体病。
> - **面容/体征（FAS 三联）**：**smooth philtrum（人中平坦）+ thin vermillion border（薄上唇红）+ small palpebral fissures（小眼裂）** + 小头 + 鼻短上翻。
> - **多系统**：**生长受限（IUGR，宫内生长受限，intrauterine growth restriction + 出生后矮小）+ 癫痫 + 先心（VSD/ASD）+ 学习障碍/行为问题**。
> - **化验/诊断**：**临床诊断（无基因检测）**；stem 问"诊断"选 maternal alcohol exposure history（母亲饮酒史）。
> - **Buzzword/Hook**：ID（智力障碍，intellectual disability）三大病因之一（Down = 染色体 / Fragile X = 单基因 / **FAS = 妈妈喝酒**）。

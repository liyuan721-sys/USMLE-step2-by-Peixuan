# 囊性纤维化（CF）气道病原速查

> 派生自 [[mistakes/uworld-mistakes_2026-06#^Q3830]]（CF 肺炎病原：幼儿 = 金葡 / 成人 = 铜绿）。
> 用法：CF vignette 问"最可能病原 / 经验抗生素"时，**先看年龄**再答。
> 版权：基于公开教科书概念整理，未复制 UWorld 题干。

---

## 一、一句话病理

**囊性纤维化（cystic fibrosis, CF）= CFTR 基因（Cl⁻ 通道）缺陷 → 气道黏液黏稠 → 黏液纤毛清除障碍 → 慢性反复气道感染**。最常见致死原因 = **慢性肺部感染 + 呼吸衰竭**。

---

## 二、气道病原随年龄演变（核心）⭐⭐⭐

```
CF 气道病原时间轴
婴幼儿 ─────────► 儿童/青少年 ─────────► 成人
金黄色葡萄球菌        铜绿假单胞菌            铜绿（黏液型 mucoid）为主
S. aureus  +        P. aeruginosa          ± 洋葱伯克霍尔德菌
流感嗜血杆菌          逐渐占主导              Burkholderia cepacia（晚期·预后差）
H. influenzae
```

| 年龄段 | 主要病原 | 备注 |
|---|---|---|
| **婴幼儿 / 儿童早期** | **金黄色葡萄球菌（Staphylococcus aureus）** + 流感嗜血杆菌（H. influenzae） | 最早定植 |
| **青少年 → 成人** | **铜绿假单胞菌（Pseudomonas aeruginosa）**，尤其**黏液型（mucoid）** | 逐渐占主导，慢性定植**难根除** |
| 晚期（部分） | **洋葱伯克霍尔德菌（Burkholderia cepacia）** | **病情快速恶化、预后差**，可成为肺移植禁忌 |

---

## 三、两个"最常见"别混 ⭐（高频陷阱）

> [!tip] 看年龄分别答
> - **CF 一生中总体最常见 = 铜绿假单胞菌（Pseudomonas）** —— 因成人期占多数、慢性定植。
> - **CF 幼儿期最常见 = 金黄色葡萄球菌（S. aureus）**。
> - 题干给**年龄**就按年龄：**小孩 → 金葡；青少年/成人 → 铜绿**。

> [!warning] 铜绿为何一旦定植就赖着不走
> 铜绿在 CF 气道**转成黏液型（mucoid）+ 形成生物膜（biofilm）** → 逃避免疫 + 抗生素 → **慢性感染 + 反复急性加重**。**黏液型铜绿出现 = 预后变差的标志**。

---

## 四、经验治疗一句话

- **急性加重**：覆盖**铜绿** → 抗假单胞 β-内酰胺（哌拉西林-他唑巴坦 piperacillin-tazobactam / 头孢吡肟 cefepime / 美罗培南 meropenem）**+ 妥布霉素（tobramycin，氨基糖苷协同）**；疑金葡（尤幼儿/MRSA 风险）加**万古霉素（vancomycin）**。
- **慢性铜绿定植**：**吸入妥布霉素（inhaled tobramycin）** ± 阿奇霉素（azithromycin，抗炎/抗生物膜）维持。
- **Burkholderia cepacia**：多重耐药，常需 TMP-SMX / 美罗培南 联合，预后差。

> [!tip] 为什么氨基糖苷偏偏选妥布霉素 tobramycin
> ① **氨基糖苷里抗铜绿最强**（活性 > 庆大霉素 gentamicin）；② **有 CF 专属吸入剂型（吸入妥布霉素 inhaled tobramycin / TOBI）**——气道局部浓度极高、全身吸收少（肾/耳毒性低），慢性铜绿定植维持就靠它；③ 急性期与抗铜绿 β-内酰胺 β-lactam **协同杀菌 + 防治疗中耐药**。（CF 药代改变：分布容积大、清除快 → 常需更高剂量）

---

## 五、Memory Hook

**「CF 小孩咳脓痰 = 金葡（S. aureus）；CF 青少年/成人反复加重 = 铜绿（Pseudomonas，黏液型）；快速恶化想 Burkholderia cepacia。总体最常见 = 铜绿，幼儿最常见 = 金葡。」**

> [!note] 别套错
> 这套"年龄换菌"**只属 CF**。普通社区获得性肺炎首位永远是**肺炎链球菌（S. pneumoniae）**；CF 才是金葡 → 铜绿的特殊谱。

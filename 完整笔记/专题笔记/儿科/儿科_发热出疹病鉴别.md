# 儿科发热出疹病鉴别

> [!info] 笔记定位
> **儿童发热出疹（exanthem）的"时间 + 招牌"两钥匙。**
> Step 2 CK 出疹题靠两点秒杀：① **出疹与发热的时间关系**（热中出疹 vs **退热后**出疹）② **皮疹招牌**（砂纸样 / 掌掴颊 / 手足口 / Koplik 等）。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q3969]]（猩红热误选 roseola）。

---

## 一、关键钥匙：出疹时间

```
发热"中"出疹（还在烧就出疹）
  → 猩红热 / 麻疹 / 风疹 / 传染性红斑 / HFMD / Kawasaki
发热"退后"才出疹（热骤退 → 出疹）⭐
  → 幼儿急疹 Roseola（HHV-6）
```

> [!danger] 最高频陷阱（你栽过）
> **Roseola = 高热 3-5 天骤退 → 退热后才出玫瑰疹**。看到"**发热中出疹**"就**排除 roseola**（[[mistakes/uworld-mistakes_2026-06#^Q3969]] 发热中砂纸疹却误选 roseola）。

## 二、发热出疹病总表 ⭐⭐⭐

| 病 | 病原 | 出疹时间 | 招牌 |
|---|---|---|---|
| **猩红热 Scarlet fever** | GAS（红疹毒素）| 热中 | **砂纸样疹 + 皮褶重(Pastia) + 草莓舌 + 口周苍白 + 渗出咽炎 + 脱皮** |
| **麻疹 Measles** | Measles 病毒 | 热中（高热）| **Koplik 斑** + 3C(cough/coryza/conjunctivitis) + 头→脚蔓延 |
| **风疹 Rubella** | Rubella 病毒 | 热中（低热）| **枕后/耳后淋巴结** + 头→脚淡疹 |
| **传染性红斑（第五病）** | **Parvovirus B19** | 热后/无热 | **掌掴颊（slapped cheek）** + 网状疹；可致再障危象/胎儿水肿 |
| **手足口 HFMD** | **Coxsackie A** | 热中 | **手/足疹 + 口腔溃疡** |
| **幼儿急疹 Roseola（第六病）** | **HHV-6** | **退热后** ⭐ | 高热骤退 → 玫瑰色斑疹；6 月-2 岁；可热性惊厥 |
| **Kawasaki** | 不明（血管炎）| 热中 | **≥5 天发热 + ≥4 标准**（结膜/口唇/肢端/疹/颈结）|
| **SSSS** | *S. aureus* 剥脱毒素 | — | 松弛大疱 + 大片剥脱 + Nikolsky⊕；婴幼儿<5 岁 |
| **水痘 Varicella** | VZV | 热中 | **各期并存**（斑/疱/痂同时）+ 向心分布 |

> [!success] 招牌一句话
> - **砂纸样 + 皮褶 = 猩红热**（GAS → penicillin）
> - **掌掴颊 + 网状 = 第五病**（parvovirus B19）
> - **退热后玫瑰疹 = 幼儿急疹**（HHV-6）
> - **Koplik 斑 + 3C = 麻疹**
> - **手足口腔 = HFMD**（Coxsackie）
> - **各期并存 = 水痘**
> - **≥5 天热 + 4 项 = Kawasaki**

> [!info] 猩红热两大褶皱/面部征 ⭐
> - **Pastia 线（Pastia lines）**：砂纸疹在**皮肤褶皱**（腋下/腹股沟/肘窝/颈）**最浓、汇成线状暗红**，**压之不褪色**（砂纸疹本身褪色，Pastia 线不褪），疹退后沿褶皱脱皮。题干"疹集中在腋下/腹股沟"= 暗示 Pastia → 猩红热。
> - **口周苍白（circumoral pallor）**：脸颊红，**嘴周一圈发白**。
> 两征常一起考，配砂纸疹 + 草莓舌 + 渗出咽炎锁死猩红热（GAS）。

## 二·五、疫苗可预防性 + 接种钥匙 ⭐⭐（高频鉴别线索）

> [!info] 疫苗状态是出疹题的暗钩——"已接种 vs 未接种"直接缩小鉴别
> | 病 | 疫苗 | 接种程序 | 临床钥匙 |
> |---|---|---|---|
> | **麻疹 Measles** | **MMR（麻腮风三联）** | **12-15 月 + 4-6 岁** | 多见**未接种 / 进口病例 / 未接种者旅行** → 见 Koplik+3C 想麻疹时问疫苗史 |
> | **风疹 Rubella** | **MMR** | 12-15 月 + 4-6 岁 | 未接种；**孕妇风疹 → 先天性风疹综合征（CRS）** |
> | **腮腺炎 Mumps**（非出疹但同 MMR） | **MMR** | 12-15 月 + 4-6 岁 | 腮腺肿；未接种聚集暴发 |
> | **水痘 Varicella** | **VZV 疫苗（水痘疫苗）** | **12-15 月 + 4-6 岁** | **2 剂；仅 1 剂可突破感染（breakthrough，更轻、疹少）**（[[mistakes/uworld-mistakes_2026-06#^Q2768]]）|
> 
> **不可疫苗预防（无疫苗）**：幼儿急疹 Roseola（HHV-6）、传染性红斑/第五病 Erythema infectiosum（Parvovirus B19）、猩红热 Scarlet fever（GAS）、手足口 HFMD（Coxsackie A）、川崎病 Kawasaki（非感染）、SSSS（*S. aureus*）。

> [!tip] 怎么用疫苗史做题
> - **"疫苗齐全（up to date）"** → **麻疹/风疹/水痘可能性↓**（但**水痘突破感染**仍可能，尤其 <4 岁只打了 1 剂 → 别因"疫苗齐全"排除水痘）。
> - **"未接种 / 移民 / 近期出国"** + Koplik+3C → **麻疹**（高度警惕，并发肺炎/脑炎/SSPE）。
> - **疫苗史无关的病**（roseola/第五病/猩红热/HFMD/Kawasaki）→ 接种状态**不帮你排除**，回到"时间 + 招牌"两钥匙。
> - **MMR + VZV 接种程序相同**（12-15 月 + 4-6 岁），记一套即可；都是**减毒活疫苗（live attenuated）→ 孕妇/免疫抑制禁用**。

## 三、易混对子 ⭐

> [!warning] 三组高频对照
> - **猩红热 vs 幼儿急疹**：热中砂纸疹（猩红热）↔ **退热后**玫瑰疹（roseola）。
> - **猩红热 vs 第五病**：砂纸样皮褶（GAS）↔ 掌掴颊网状（parvovirus）。
> - **麻疹 vs 风疹**：高热 + Koplik + 3C（麻疹，重）↔ 低热 + 枕后淋巴结（风疹，轻，孕期致畸）。

> [!tip] GAS 出疹与并发症
> 猩红热 = GAS 咽炎 + 毒素疹，**治疗同链球菌咽炎（penicillin/amoxicillin）**；并发 **ARF（风湿热）+ PSGN**。咽炎诊治详见 [[完整笔记/专题笔记/感染/感染_咽炎鉴别与GAS诊治]]。

## 四、Memory Hook

```
时间钥匙
  热中出疹 = 猩红热/麻疹/风疹/第五病/HFMD/Kawasaki
  退热后出疹 = 幼儿急疹 Roseola（HHV-6）⭐

招牌钥匙
  砂纸+皮褶+草莓舌 = 猩红热(GAS→penicillin)
  掌掴颊+网状 = 第五病(parvovirus B19)
  Koplik+3C = 麻疹
  手足+口溃疡 = HFMD(Coxsackie)
  各期并存 = 水痘
  ≥5天热+4项 = Kawasaki
```

# 儿童喘鸣 stridor / 急性气道梗阻鉴别

> [!info] 笔记定位
> **喘鸣（stridor）= 上气道梗阻的声音**——儿童急诊高频，套路固定：① **吸气/呼气/双相定位梗阻部位** ② **招牌（犬吠咳 / 流涎 / 突发呛咳）定病因** ③ **气道急症先稳气道、别乱查咽**。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q115197]]（croup = 副流感，误选 Hib）。
> 素材整合自分科主笔记 [[完整笔记/Peixuan分科笔记/pulmonary]] / [[完整笔记/Peixuan分科笔记/儿科]] / [[完整笔记/Peixuan分科笔记/感染]]。

## 一、第一步：喘鸣定位（吸气 vs 呼气 vs 双相）⭐⭐⭐

```
吸气性 stridor（inspiratory） → 声门上 / 声门（胸腔外上气道）
   croup / 会厌炎 / 喉软化 / 异物（喉）/ 过敏喉头水肿
呼气性 wheeze（expiratory）   → 胸腔内（下气道）
   毛细支气管炎 / 哮喘 / 气管软化
双相 biphasic                → 固定狭窄（声门下/气管）
   细菌性气管炎 / 声门下狭窄 / 血管环 / 气管异物
```

> [!tip] 一句话钥匙
> **stridor 在上（上气道）、wheeze 在下（下气道）、双相 = 固定狭窄**。先用相位把梗阻"定楼层"，再用招牌定病因。

## 二、4 大急性上气道梗阻（核心鉴别）⭐⭐⭐

| 病 | 年龄/起病 | 招牌三联 | 病原 | 影像 | 处理 |
|---|---|---|---|---|---|
| **Croup 喉气管炎** | 6 月-3 岁，URI 前驱 | **犬吠咳 + 声嘶 + 吸气 stridor**（低热，一般情况好）| **副流感病毒（parainfluenza）** | 颈正位 **steeple sign 尖塔征** | **地塞米松**；静息 stridor 加**雾化消旋肾上腺素** |
| **会厌炎 Epiglottitis** | 未接种 Hib，**突发** | **流涎 + 三脚架位 + 中毒貌高热**（无咳、不能平卧）| **Hib**（接种后 → 肺炎链球菌/GAS/金葡）| 颈侧位 **thumb sign 拇指征** | **先稳气道**（控制下插管/外科气道）+ **头孢曲松 + 万古**；**别压舌/别激惹咽部** |
| **细菌性气管炎 Bacterial tracheitis** | 任何，常 croup 后恶化 | **高热 + 大量脓痰 + 对肾上腺素无反应** | **金黄色葡萄球菌（S. aureus）** | 似 croup（声门下窄）+ **气管壁不规则** | 稳气道（常需插管）+ IV 抗生素（覆金葡）|
| **气道异物 Foreign body** | 6 月-3 岁，**突发呛咳** | **突发 + 单侧呼吸音减低/单侧喘鸣**（无前驱感染）| — | 吸/呼气片：**患侧过度充气 + 纵隔移位**（X 线 **30% 正常**）| **硬质支气管镜**取出（金标准）|

> [!danger] 三个救命陷阱
> - **会厌炎**：**别压舌板查咽、别激惹**——可诱发完全梗阻；**气道优先于一切检查/抗生素**。
> - **气道异物**：**X 线正常不排除异物**（30% 不显影）→ 病史突发呛咳 + 单侧体征即上**支气管镜**。
> - **肾上腺素只救 croup**（声门下黏膜水肿 α1 血管收缩消肿）——**对会厌炎/细菌性气管炎/异物无效**，反应差要想"不是 croup"。

> [!warning] Croup = 副流感（你的错题 Q115197）
> **犬吠咳 + 吸气 stridor + 一般情况好 = croup → 副流感病毒**；误选 **Hib = 会厌炎**（流涎 + 三脚架 + 中毒貌 + 无咳，完全不同）。**stridor ≠ 反射 Hib**——看伴随。

> [!info] 成人会厌炎（adult epiglottitis）—— 和儿童不一样
> - **病原**：成人以**肺炎链球菌 S. pneumoniae / 流感嗜血杆菌 H. influenzae** 为主（不只 Hib）；危险因素 **糖尿病、肥胖、URI 前驱**。
> - **更隐匿**：常无儿童经典"3 D"（流涎/吞咽困难/窘迫），早期只**咽痛 + 吞咽痛 + 声音闷**，但可**迅速进展**——**pooled 口腔分泌物 + stridor = 警报**。
> - **诊断**：气道**稳** → **颈侧位片 lateral neck radiograph**（thumbprint 拇指征 / 会厌肿大）或直接喉镜；气道**不稳 → 先建立气道再检查**。（UW Q15961）
> - **治疗**：必要时建立气道 + IV **头孢曲松 ceftriaxone + 万古霉素 vancomycin**。

## 三、下气道对照：毛细支气管炎（别和 croup 混）

> [!info] 毛细支气管炎 Bronchiolitis（下气道，wheeze 非 stridor）
> **<2 岁 + 秋冬 + RSV + 弥漫喘息/爆裂音 + 呼吸窘迫**；**呼气性 wheeze**（下气道），不是吸气性 stridor。**治疗仅支持**（吸鼻/补液/给氧）；**不常规用支扩/激素/肾上腺素**。
> **RSV 预防**：**Nirsevimab**（<8 月婴儿，一针管一季）/ **Palivizumab**（<24 月高危，每月一针）。

## 四、深颈间隙感染（颈源性气道梗阻）

> [!info] 咽后脓肿 RPA (Retropharyngeal abscess) vs 扁桃体周脓肿 PTA (Peritonsillar abscess)
> 
> | | **咽后脓肿 Retropharyngeal abscess（RPA）** | **扁桃体周脓肿 Peritonsillar abscess（PTA）** |
> |---|---|---|
> | 年龄 | **2-4 岁** | 青少年/年轻成人 |
> | 招牌 | **颈伸展痛/拒动颈 + 吞咽痛 + 后咽壁膨隆** | **单侧扁桃体肿 + 悬雍垂偏向对侧 + 张口困难（trismus）+ 含热土豆音** |
> | 病原 | 多菌（GAS + 金葡 + 口腔厌氧）| GAS + 厌氧 |
> | 处理 | 影像（侧位颈片/CT）+ IV 抗生素 ± 引流；警惕**纵隔扩散** | 穿刺/切开引流 + 抗生素 |

## 五、先天/慢性喘鸣（对照，非急症）

> [!info] 慢性/反复喘鸣 — 与急性气道急症区分
> - **喉软化 Laryngomalacia**：**最常见婴儿慢性吸气性 stridor**；**仰卧/喂养/哭闹加重，俯卧/竖位改善**；多自限。
> - **血管环 Vascular ring**：**双相 stridor + 喂养困难**，**颈后伸减轻**（外压气管）；CTA 确诊、手术。
> - **气管/声门下狭窄**：双相固定狭窄。
> 鉴别钥匙：**慢性/反复 + 体位相关 = 先天**（喉软化）；**急性 + 发热/突发 = 感染/异物急症**。

## 六、过敏性气道（别漏）

> [!warning] 过敏性喉头水肿 Anaphylaxis
> **stridor + 声嘶（上气道水肿）+ wheeze（下气道）+ 荨麻疹/血管水肿** + 暴露史（食物/药/蜂蜇）→ **立即 IM 肾上腺素**（大腿外侧）+ 气道管理 + 补液。NSAIDs/β 阻滞剂可加重。

## 七、决策树（三条线入手）⭐

```
急性喘鸣患儿
├─ 犬吠咳 + 吸气stridor + 一般情况好 + URI前驱 → Croup（副流感）→ 地塞米松±消旋肾上腺素
├─ 流涎 + 三脚架 + 中毒貌高热 + 无咳 + 未接种 → 会厌炎（Hib）→ 先稳气道! 别查咽
├─ croup样但高热+脓痰+肾上腺素无效 → 细菌性气管炎（金葡）→ 稳气道+IV抗生素
├─ 突发呛咳 + 单侧呼吸音↓（无感染）→ 气道异物 → 硬支气管镜（X线正常不排除）
├─ <2岁+秋冬+RSV+弥漫wheeze → 毛细支气管炎 → 支持（非stridor，下气道）
├─ 颈伸展痛/后咽膨隆（2-4岁）→ 咽后脓肿；单侧扁桃体+悬雍垂偏 → 扁桃体周脓肿
├─ 慢性+体位相关（仰卧重俯卧轻）→ 喉软化；双相+喂养难+颈后伸轻 → 血管环
└─ 荨麻疹+血管水肿+暴露史 → 过敏 → IM肾上腺素
```

## 八、Memory Hook

```
定位：stridor在上(上气道)/wheeze在下(下气道)/双相=固定狭窄
4急症招牌：
  犬吠咳=croup(副流感,地塞米松±消旋肾上腺素;steeple尖塔征)
  流涎三脚架毒性=会厌炎(Hib,先气道别查咽;thumb拇指征;ceftriaxone+vanc)
  高热脓痰肾上腺素无效=细菌性气管炎(金葡)
  突发呛咳单侧=异物(硬支气管镜;X线正常不排除30%)
铁律：肾上腺素只救croup;会厌炎别压舌;异物X线正常≠没有
对照：毛支=下气道wheeze(RSV支持);喉软化=慢性体位相关
```

# 儿科_小头畸形 microcephaly 评估（Microcephaly Evaluation）

> [!info] 一句话定位
> **Microcephaly（小头畸形）= 头围（HC，head circumference）< 同龄同性别第 3 百分位**。头围间接反映脑容量；**病理性 vs 良性家族性**分流决定要不要做**脑 MRI（magnetic resonance imaging，磁共振）**。
> 派生自错题 [[mistakes/uworld-mistakes_2026-06#^Q18791]]（病理性小头误选婴儿肉毒粪检）。

## 一、定义与意义

- **HC < 第 3 百分位**（或 <-2 SD）= 小头畸形。
- 头围随脑生长沿可预测轨迹增长；**头围增速下降（跨越多条主百分位线）= 脑生长不足**。
- 起病时机：**先天性（出生即小）** vs **出生后获得性（产后跨线下降）**。

## 二、病理性 vs 良性家族性分流 ⭐

```
HC < 第 3 百分位
        │
   有以下任一？
   ① 神经异常（肌张力↓ hypotonia / 发育迟缓 / 抽搐）
   ② 畸形特征（dysmorphic features）
   ③ 头围百分位快速下降（跨多条主线）
        │
   ┌────┴────┐
   否        是
   │         │
   ▼         ▼
良性家族性   病理性 → 脑 MRI
（测父母      （± 先天感染筛查 / 遗传/代谢检查）
  头围 +
  监测发育）
```

> [!tip] 良性家族性小头（benign familial microcephaly）
> **孤立性小头 + 生长正常 + 发育正常 + 父母头围也小** → 良性变异：测父母头围、监测发育，**不急做影像**。

## 三、病理性小头 → 脑 MRI 找什么

- **结构性脑畸形**：脑膨出 encephalocele、无脑回 lissencephaly（脑沟缺如）等。
- **围产期缺血缺氧损伤**（perinatal hypoxic-ischemic insult，白质损伤）。
- **先天感染**（如巨细胞病毒 CMV，cytomegalovirus → 钙化）或产后**外伤性脑损伤**（traumatic brain injury，如非意外创伤）。
- 必要时加遗传/代谢检查（畸形特征明显时）。

## 四、易混陷阱（普适）

- **肌电图/神经传导（EMG/NCS）** = 疑神经肌肉病（如先天代谢病、肌营养不良致肌张力↓）；但小头 = **脑生长异常**，**影像优先**，儿童多先做基因/影像。
- **高热卡强化配方（fortified formula）** = 用于热量不足致**生长迟滞（FTT，failure to thrive）**；FTT **先掉体重 → 再身长 → 最后头围**；本例体重/身长正常 → 不是喂养问题。
- **先天梅毒血清（serum syphilis）** = 先天梅毒多有肝脾大/皮疹/骨改变，且小头多**出生即**有。
- **婴儿肉毒（infant botulism）粪检**（本次错选）= 肉毒影响**神经肌肉接头（NMJ，neuromuscular junction）**致**急性进展**性松软 + 喂养差，**不致慢性小头 / 脑生长异常**。

## Memory Hook

> [!success]
> - **小头（HC<3rd）+ 神经异常 / 跨线下降 / 畸形 → 脑 MRI**（找结构/感染/缺氧病因）
> - **孤立小头 + 父母头小 + 发育正常 = 良性家族性**（只监测）
> - 喂养不足是 **FTT（先掉体重）**，不是小头主因；肉毒是**急性**松软不是慢性小头

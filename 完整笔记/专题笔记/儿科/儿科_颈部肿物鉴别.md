# 儿科_颈部肿物鉴别（中线 vs 侧方 vs 后三角）

> [!info] 为什么建这张
> 2026-06-10 UW Q16058（做对题补充）：12 岁男，URI 一周后**左颈侧囊性肿物 + SCM 前方小窝漏黏脓液** → **鳃裂囊肿 branchial cleft cyst**。小儿颈部肿物先按**位置（中线 / 侧方 / 后三角）**一秒分流。

---

## 一、按位置一秒分流 ⭐⭐⭐

```
小儿颈部肿物 → 先看位置：
  中线 midline ──┬─ 随吞咽 / 伸舌上下动 → 甲状舌管囊肿 TDC
                 └─ 不随伸舌动          → 皮样囊肿 dermoid
  侧方 lateral ──┬─ SCM 前、URI 后感染流脓 → 鳃裂囊肿 BCC
                 ├─ 硬、压痛、多发          → 反应性淋巴结
                 └─ 坏死结 + 皮肤紫红 + 瘘   → 非结核分枝杆菌 MAI
  后三角 posterior ── 出生即有 / 产前发现、透光 → 囊状水瘤 cystic hygroma
```

---

## 二、大对照表 ⭐⭐

| 肿物 | 位置 | 招牌钥匙 | 本质 |
|---|---|---|---|
| **甲状舌管囊肿 Thyroglossal duct cyst（TDC）** | **中线** | **随吞咽 / 伸舌上下移动**；常 URI 后出现；舌盲孔 → 甲状腺下降路径 | 甲状腺下降残留 |
| **皮样囊肿 Dermoid cyst** | 中线 | 含上皮碎屑；沿胚胎融合线；**不随伸舌动** | 滞留上皮 |
| **鳃裂囊肿 Branchial cleft cyst（BCC）** | **侧方（SCM 前）** | **URI 后继发感染 → 红肿 / 窦道流脓**；第 2 鳃弓最常见（瘘可达扁桃体窝） | 鳃器残留 |
| **反应性淋巴结 Reactive adenopathy** | 侧方 | **硬、常压痛、多发结节**；感染后 | 淋巴结增生 |
| **非结核分枝杆菌淋巴结炎 MAI** | 侧方 | **坏死淋巴结 + 表面皮肤紫红 + 常成瘘** | M. avium |
| **囊状水瘤 Cystic hygroma** | **后三角** | **扩张淋巴管、透光**；出生 / 产前发现；**Turner 关联** | 淋巴管畸形 |

---

## 三、最常考的两组鉴别

> [!tip] ① 中线"动不动" = TDC vs Dermoid
> **随伸舌 / 吞咽上下移动 = 甲状舌管囊肿**（连舌骨 / 甲状腺下降路径）；**不动 = 皮样囊肿**。TDC 治疗 = **Sistrunk 手术**（连舌骨中段一并切，否则易复发）；术前先确认有正常甲状腺（防切的是异位甲状腺）。

> [!tip] ② 中线 vs 侧方（都可 URI 后感染）
> **甲状舌管囊肿（中线）** 与 **鳃裂囊肿（侧方 SCM 前）** 都常在**上呼吸道感染后**被发现 / 继发感染流脓 → 关键分水岭就是**位置：中线 = TDC，侧方 = BCC**。（UW Q16058）

> [!warning] 别认错的干扰项
> - **囊状水瘤** = 后三角、透光、**出生即有** → 想 **Turner**（也见于胎儿水囊瘤）。
> - **喉气囊肿 laryngocele**（Q16058 干扰项）= 侧颈、**Valsalva 充气胀大**（吹奏者 / 吹玻璃工），非囊肿感染。
> - **放线菌 Actinomyces** 下颌肿块 + **硫磺颗粒**窦道，多在牙感染 / 免疫低下后。

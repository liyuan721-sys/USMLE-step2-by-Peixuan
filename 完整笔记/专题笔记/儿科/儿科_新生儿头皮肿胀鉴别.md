# 儿科_新生儿头皮肿胀鉴别（Neonatal Scalp Swelling）

> [!info] 一句话定位
> 新生儿产后头皮肿胀三大类：**Caput succedaneum（产瘤）/ Cephalohematoma（头颅血肿）/ Subgaleal hemorrhage（帽状腱膜下出血）**。**一条判据先分家：跨不跨骨缝（cross suture lines）** —— 不跨缝的只有 cephalohematoma（骨膜下受骨缝限制）。
> 派生自错题 [[mistakes/uworld-mistakes_2026-07#^Q2472]]（产钳后局部硬肿误选脑积水）。

## 一、三层解剖决定"跨不跨骨缝" ⭐

出血/水肿位于哪一层，决定它能否越过骨缝（suture）：

```
头皮由浅入深（记忆 SCALP）
  Skin 皮肤
  Connective tissue 皮下结缔组织  ← Caput（水肿在此，跨缝）
  Aponeurosis 帽状腱膜
  Loose areolar 疏松结缔组织      ← Subgaleal（血在腱膜与骨膜间，跨缝）
  Periosteum 骨膜                 ← Cephalohematoma（血在骨膜下，受骨缝限制→不跨缝）
  ────────────────
  颅骨 skull
```

- **骨膜紧贴单块颅骨、止于骨缝** → 骨膜**下**出血（cephalohematoma）被限制在一块骨上，**不跨骨缝**。
- 皮下（caput）与腱膜下（subgaleal）在骨膜浅层、不受骨缝约束 → **跨骨缝**。

![[L16572.jpg]]

*颅内外血肿的层次总览 (types of cranial hematomas)：由浅入深 —— Caput succedaneum（皮下）/ Subgaleal hemorrhage（帽状腱膜下）/ Cephalohematoma（骨膜下）为**头皮层**三鉴别；再深为硬膜外 epidural / 硬膜下 subdural（颅内，属产伤性颅内出血，非头皮肿胀）*

## 二、三鉴别速查表 ⭐

| 类型 | 层次 | 跨骨缝 | 出现时间 | 质地 | 关键伴随/预后 |
|---|---|---|---|---|---|
| **Caput succedaneum 产瘤** | 皮下（subcutaneous） | **跨缝** ✅ | 出生即有 | 软、凹陷性（boggy） | 界限不清；数天自消，良性 |
| **Cephalohematoma 头颅血肿** | 骨膜下（subperiosteal） | **不跨缝** ❌ | 生后数小时才现 | **硬、不波动** | 界限清、表皮正常；**高胆红素风险**（红细胞 RBC 分解）；~1 月吸收；罕见钙化/骨化 |
| **Subgaleal hemorrhage 帽状腱膜下出血** | 腱膜下疏松层 | **跨缝** ✅ | 数天内**进行性扩大** | 软、**波动感（fluctuant）** | 弥漫、±淤斑；**可致命大失血**（导静脉 emissary vein 剪切伤）→ 需监测心率 (HR) / 血细胞比容 (Hct) |

![[{880B25B3-5D4E-46EF-860E-DBC2EDA41F10}.png]]

*Cephalohematoma vs Caput succedaneum 冠状断面：**左** cephalohematoma = 骨膜下出血 (subperiosteal bleed)，**不跨骨缝 (does not cross suture lines)**；**右** caput succedaneum = 位于骨膜与帽状腱膜浅层的水肿/出血，**跨骨缝 (crosses suture lines)***

## 三、诊断与处理

- **诊断=临床**（部位 + 跨缝 + 质地 + 演变），一般不需影像。
- **Cephalohematoma**：观察 + 安慰；监测**高胆红素血症**（RBC 分解 → 黄疸）；不穿刺引流（增感染风险）。
- **Caput**：观察，数天自消。
- **Subgaleal**：**急症警戒** —— 可隐匿大出血致低血容量休克；密切监测生命征、血细胞比容 (Hct, hematocrit)、扩容/输血。
- **风险因素**：cephalohematoma 与 subgaleal 都与**产钳 (forceps) / 胎吸 (vacuum) 助产**相关。

## 四、易混陷阱（普适）

- **Hydrocephalus 脑积水** = **头围整体快速增大 + 前囟膨隆**，不是局部界清的头皮肿块 → 本题错选。局部肿 ≠ 脑积水。

![[L21009.jpg]]

*婴儿颅内压增高征 (signs of increased intracranial pressure)：头围增大 (increased head circumference) + 前囟宽大膨隆 (wide, bulging fontanel) + 头皮静脉显露 (prominent scalp veins) + **落日眼 (sunset eyes)** —— 这些是**弥漫全颅**表现，与局部界清的头皮肿块（cephalohematoma 等）截然不同*

- **Plagiocephaly 斜头畸形 / Craniosynostosis 颅缝早闭** = 颅骨**形状异常（扁平/特征头形）**，**无头皮肿胀（无肿块）** → 属另一条鉴别线，详见姊妹篇 [[完整笔记/专题笔记/儿科/儿科_颅骨形状异常鉴别]]（体位性 vs 颅缝早闭、耳位前/后、有无骨嵴）。
- **看到"跨骨缝"别急着选 caput** —— subgaleal 也跨缝，靠"波动 + 进行性扩大 + 可致命"与 caput（软凹陷、出生即有、数天消）区分。
- **看到"产钳/胎吸"想到 cephalohematoma 或 subgaleal**（都增风险），再用跨缝 + 质地 + 演变定位。

## Memory Hook

> [!success]
> - **不跨骨缝 + 硬 + 生后数小时现 = Cephalohematoma（骨膜下）** → 观察 + 防高胆红素
> - **跨缝 + 软凹陷 + 出生即有 + 数天消 = Caput（产瘤）** → 良性
> - **跨缝 + 波动 + 进行性扩大 + 可致命 = Subgaleal（腱膜下）** → 监测失血/休克
> - 一句判据：**"骨膜下被骨缝拴住 → 不跨缝"**

# 婴儿阵发性咳嗽鉴别（咳嗽模式 → 病原）

> [!info] 笔记定位
> **这张专题的轴心是「咳嗽模式 (cough pattern)」，不是气道定位**——同样是婴儿咳嗽，**阵发剧咳后发绀 = 百日咳**、**断续短促咳 (staccato) = 衣原体**、**犬吠咳 = croup**、**弥漫喘息 = RSV 毛支**。先听咳嗽的"声音和节奏"，再用伴随线索（间歇期肺清 / 结膜炎 / 吸气性喘鸣 / 弥漫啰音）锁病原。
> 与按"上/下气道梗阻定位"切入的 [[完整笔记/专题笔记/儿科/儿科_喘鸣stridor急性气道鉴别]] 是**互补双视角**（那张管 stridor 定楼层 + 4 大气道急症，本张管咳嗽模式定病原），croup/毛支的气道细节看那张，不重复。
> 错题驱动：[[NBME11_错题本#^Q113]]（婴儿百日咳，误选 RSV）。
> 素材整合自分科主笔记 [[完整笔记/Peixuan分科笔记/儿科]] / [[完整笔记/Peixuan分科笔记/pulmonary]] / [[完整笔记/Peixuan分科笔记/感染]] / [[完整笔记/Peixuan分科笔记/免疫，疫苗]]。

## 一、核心鉴别表：婴儿咳嗽病原（按咳嗽模式）⭐⭐⭐

表格前后空行；wikilink 不放表格内。

| 病原 | 咳嗽招牌 | 关键伴随线索 | 肺部听诊 | 年龄/季节 | 治疗 |
|---|---|---|---|---|---|
| **百日咳 Bordetella pertussis（百日咳鲍特菌）** ✅本题 | **阵发性剧咳 (paroxysmal)** + 咳后呕吐 (posttussive emesis) + **发绀 (cyanosis)/呼吸暂停 (apnea)**；婴儿可**无经典 whoop（鸡鸣样回声）** | **发作间歇期肺清、神志好**；**家中成人久咳**（传染源）| **间歇期几乎正常** | <1 岁高危，全年 | **大环内酯（阿奇霉素 azithromycin）** + 支持 |
| **衣原体 Chlamydia trachomatis（沙眼衣原体）** | **断续短促咳 (staccato cough)** | **新生儿结膜炎病史**（产道获得）+ **无热 afebrile** + 气促 | 散在啰音，间质浸润 | **1–3 月龄**（围产期感染）| **大环内酯（阿奇霉素 azithromycin）** |
| **RSV 呼吸道合胞病毒（毛细支气管炎 bronchiolitis）** | 咳嗽 + **弥漫喘息 wheeze** | **弥漫啰音/喘息 + 气促 + 呼吸窘迫**；<2 月龄可首发 **apnea** | **双肺弥漫喘息/爆裂音** | <2 岁，**秋冬** | **支持**（吸鼻/补液/给氧），不用支扩/激素/肾上腺素 |
| **副流感病毒 Parainfluenza（哮吼 croup）** | **犬吠样咳 (barking) + 声嘶** | **吸气性喘鸣 (inspiratory stridor)**，低热，一般情况好 | 上气道喘鸣（非肺实质）| 6 月-3 岁，URI 前驱 | **地塞米松 dexamethasone** ± 雾化消旋肾上腺素 racemic epinephrine |
| **肺炎支原体 Mycoplasma pneumoniae** | 持续干咳 | **学龄儿/青少年** atypical pneumonia，胸片>听诊；冷凝集 | 散在啰音 | 大龄（非婴儿）| **大环内酯** |
| **腺病毒 Adenovirus** | 咳嗽 + 发热 | 发热性上感 **± 结膜炎**（咽结膜热）| 不特异 | 各龄 | 支持 |

> [!tip] 一句话钥匙
> **"咳到要断气、肺却是清的 + 家里大人久咳" = 百日咳**（婴儿不一定有 whoop）；**"断续短促咳 + 有过结膜炎 + 不发热 + 1–3 月龄" = 衣原体**；**"弥漫喘息" = RSV**；**"犬吠咳 + 吸气性喘鸣" = croup**。

## 二、百日咳 Pertussis 深入（本题主角）⭐

### 2.1 三期（catarrhal → paroxysmal → convalescent）

```text
① 卡他期 Catarrhal（1-2 周）
   像普通感冒：流涕、轻咳、低热 —— 此期传染性最强、最难诊断
② 痉咳期 Paroxysmal（2-6 周）⭐ 招牌期
   阵发性连珠样剧咳 → 末尾吸气 whoop（鸡鸣样回声）
   → 咳后呕吐 posttussive emesis / 发绀 / 婴儿 apnea
   发作间歇期：神志清、肺听诊清（"咳得重、肺不重"）
③ 恢复期 Convalescent（数周-数月）
   咳嗽渐轻，"百日"之名由来
```

> [!danger] 婴儿特异表现：可无 whoop，首发可为 apnea ⭐（最易漏诊）
> 大龄儿/成人有经典 whoop（"百日"名源），但 **<6 月龄婴儿可完全没有 whoop，反而以阵发咳后发绀、posttussive gagging、甚至单纯呼吸暂停 (apnea) 起病**。
> **陷阱**：只问"有没有 whoop"会漏诊婴儿百日咳；婴儿期反而是**最凶险**的（apnea / 发绀 / 继发肺炎 / 死亡风险最高）。

> [!tip]- 辅助：淋巴细胞绝对增多 (lymphocytosis)
> 百日咳毒素抑制淋巴细胞回流淋巴结 → 外周血**淋巴细胞绝对增多**（婴儿尤其明显），是经典辅助线索（细菌感染却以淋巴细胞为主，反直觉）。确诊靠鼻咽 **PCR / 培养**。

### 2.2 治疗 + 预防

> [!info] 治疗时机 + 暴露后预防 (PEP)
> - **治疗**：**大环内酯（阿奇霉素 azithromycin / 克拉霉素 / 红霉素）**。**卡他期或痉咳早期（症状<3 周内）用药才能改变病程**；晚期用药主要为**减少传播**（缩短排菌），对已成形的痉咳症状改善有限。
> - **暴露后预防 (postexposure prophylaxis)**：密切接触者**无论疫苗状态都给同款大环内酯预防**（家庭/托幼接触者）。
> - 婴儿大环内酯注意：**<1 月龄用阿奇霉素**（红霉素 erythromycin 与**婴儿肥厚性幽门狭窄 IHPS, infantile hypertrophic pyloric stenosis** 相关，新生儿避免红霉素）。

> [!info] 疫苗（主动 DTaP / 被动 母亲 Tdap）⭐
> - **DTaP（大写 D，儿童高剂量）**：2、4、6、15–18 月 + 4–6 岁，**无细胞 acellular（aP）**疫苗（副反应小于全菌体 wP）。
> - **孕母 Tdap（每次妊娠都打，第三孕期最佳）**：母体 IgG 经胎盘传给胎儿 → **新生儿出生即有被动抗体，覆盖出生到首剂 DTaP 之前的空窗期**（婴儿最脆弱的<2 月龄）。这是**"包茧 cocoon"策略**外最直接的婴儿保护。
> - 跨链：疫苗程序细节见 [[完整笔记/Peixuan分科笔记/免疫，疫苗]]。

## 三、衣原体 Chlamydia trachomatis 肺炎（最强"非百日咳"鉴别）

> [!warning] staccato cough 断续短促咳 = 衣原体（围产期感染）
> **1–3 月龄婴儿** + **断续短促咳 (staccato cough)** + **气促** + **无热 (afebrile)** + **既往新生儿结膜炎病史** → **Chlamydia trachomatis 肺炎**（经母亲产道获得）。
> - **时间线**：生后 **5–14 天**衣原体结膜炎（脓性）→ 数周后 **1–3 月龄**起肺炎。
> - **影像**：间质浸润 (interstitial infiltrates)、过度充气。
> - **无热**是与多数病毒/细菌肺炎的关键分水岭。
> - **治疗**：**大环内酯（阿奇霉素 azithromycin）**——衣原体为细胞内菌，**β-内酰胺无效**。
> - 对照百日咳：百日咳是**阵发剧咳 + 咳后呕吐 + 发绀**（咳得很凶）；衣原体是**断续短促咳 + 无热 + 有结膜炎史**（咳得"碎"而不凶）。

## 四、咳嗽模式 → 病原 速查

```text
阵发剧咳 + 咳后呕吐/发绀/apnea + 间歇肺清 + 大人久咳  → 百日咳 Bordetella（大环内酯）
断续短促咳 staccato + 无热 + 结膜炎史 + 1-3月龄        → 衣原体 Chlamydia trachomatis（大环内酯）
弥漫喘息 wheeze + 秋冬 + <2岁 + 弥漫啰音               → RSV 毛细支气管炎（支持）
犬吠咳 barking + 声嘶 + 吸气性喘鸣 stridor             → croup 副流感（地塞米松±肾上腺素）
持续干咳 + 学龄/青少年 + 胸片>听诊 + 冷凝集            → 支原体 Mycoplasma（大环内酯）
```

> [!tip] 反射纠偏
> 婴儿"咳 + 掉氧"**别条件反射 RSV**——RSV 是**喘息 + 弥漫啰音**；百日咳是**阵发咳 + 间歇期肺清 + 大人久咳**。两条线索（间歇肺清、家庭久咳传染源）是百日咳与 RSV 最稳的分流点。

## 五、决策树

```text
婴儿阵发性/反复咳嗽
├─ 阵发连珠咳→咳后呕/发绀/apnea + 间歇肺清 + 家人久咳 → 百日咳（PCR；阿奇霉素 + 接触者PEP）
├─ 断续短促staccato + 无热 + 结膜炎史 + 1-3月龄        → 衣原体肺炎（阿奇霉素）
├─ 弥漫喘息 + 秋冬 + <2岁                              → RSV毛支（支持；<2月警惕apnea）
├─ 犬吠咳 + 吸气stridor + 一般情况好                   → croup副流感（地塞米松）
├─ 持续干咳 + 学龄/青少年 + 冷凝集                     → 支原体（阿奇霉素）
└─ 发热 + 咳 ± 结膜炎（咽结膜热）                      → 腺病毒（支持）
```

## 六、Memory Hook

```text
百日咳 = "咳到断气肺却清 + 家里大人久咳"（婴儿可无whoop，可只apnea）→ 阿奇霉素+接触者PEP
衣原体 = "断续碎咳staccato + 不发热 + 出生时红眼睛(结膜炎)" → 阿奇霉素，β-lactam无效
RSV   = "喘 + 弥漫啰音"（别看见婴儿咳掉氧就反射RSV）
croup = "犬吠咳 + 吸气性喘鸣"
三期口诀：感冒样(catarrhal,最传染) → 痉咳whoop(paroxysmal) → 拖百日(convalescent)
疫苗：孩子打DTaP；妈妈孕期打Tdap把抗体传给新生儿补空窗
```

# 矮小症与青春期延迟鉴别

> [!info] 笔记定位
> 青少年 / 儿童 **矮小（short stature）+ 青春期延迟（delayed puberty）** 的鉴别。核心考点：从 **宪法性生长延迟（constitutional growth delay, CGD, 「晚长」）** 中分辨出病理性病因（生长激素缺乏 growth hormone deficiency, GHD / 甲状腺功能减退 hypothyroidism / 慢性系统病 / 家族性矮小 familial short stature）。
> 钥匙不在「矮」，而在 **矮得稳定沿曲线 + 无系统症状 + 青春期与骨龄延迟 + 可追赶**。
> 错题驱动：[[NBME/nbme-records/NBME 11/NBME11A_S2_breakdown#^Q077]]（14 岁男孩自幼平行低于第 3 百分位 + Tanner 1 + 无系统症状 → CGD）。

---

## 一、一句话锁定 ⭐⭐⭐

> [!tip] 自幼偏矮但**平行沿自己曲线**生长 + **青春期延迟（Tanner 1）** + **无系统病征** + 体重正常
> = **宪法性生长延迟（CGD，「晚长」）**——常见正常变异，非 GHD 或甲减。
> 特征三连：**骨龄延迟 + 青春期延迟 + 最终身高可接近正常（catch-up）**。

---

## 二、机制：为什么「晚熟但能追上」

```
青春期启动时间延迟（delayed pubertal timing）
  ↓
生长突增（growth spurt）延后
  ↓
早青春期阶段孩子仍偏矮
  ↓
骨龄（bone age）< 实际年龄（生长板仍开放，剩余生长潜能大）
  ↓
青春期终于启动 → 追赶生长（catch-up）→ 成人身高常接近正常
```

> [!info] 骨龄是关键鉴别工具
> - **CGD**：骨龄**延迟**（< 实际年龄）→ 说明生长板还开着、还能长 → 预后好。
> - **家族性矮小**：骨龄**正常**（≈ 实际年龄）→ 就是遗传决定的矮，追不上父母以上。

---

## 三、核心鉴别表 ⭐⭐⭐

| 诊断 | 生长曲线 | 骨龄 | 青春期 | 关键线索 |
|---|---|---|---|---|
| **宪法性生长延迟 CGD** | 偏矮但**平行沿线** | **延迟** | **延迟** | 常有「父母也晚长」家族史；无系统症状；能追赶 ⭐本题 |
| 家族性矮小 familial short stature | 平行沿线（低位）| **正常** | 正常按时 | **父母矮**；终身矮但健康 |
| 生长激素缺乏 GHD | **跨百分位下滑**（生长速度↓）| 延迟 | 可延迟 | 生长速度异常低；可有中线畸形 / 微阴茎 |
| 甲减 hypothyroidism | 跨百分位下滑 | 延迟 | 可延迟 | **便秘 + 皮肤干 + 乏力 + 怕冷**；查 TSH |
| 慢性系统病 / 营养不良 | 下滑，**体重先受累** | 延迟 | 延迟 | 体重 / BMI 偏低在前；有基础病线索 |

> [!danger] 病理性 vs 正常变异的分水岭
> **跨百分位下滑（crossing percentiles）+ 系统症状 + 体重异常** → 想病理（GHD / 甲减 / 慢性病）。
> **稳定平行 + 无症状 + 青春期延迟 + 骨龄延迟** → 想 CGD（正常变异）。

---

## 四、题干线索 → 临床意义（以 Q077 为例）

| 线索 | 临床意义 |
|---|---|
| 自幼平行低于第 3 百分位 | 稳定低速、非骤降 → **不像获得性重病**（排 GHD/甲减跨线下滑）|
| BMI 第 50 百分位 | 体重保留 → **不支持慢性消耗 / 营养不良** |
| Tanner 1（青春期未启动）| 青春期延迟——**CGD 高价值线索** |
| 无便秘 / 皮肤干 / 尿频 | 主动否认甲减、糖尿病等内分泌 / 系统病 |
| 父母身高不矮 | **不支持家族性矮小** |

---

## 五、处理 / workup 思路

> [!info] CGD 的处理 = 安抚 + 观察
> - **确诊 CGD** 后多数只需 **确认 + 随访生长（reassurance + observation）**——它是正常变异，会追赶。
> - **筛查病理**：生长速度异常 / 跨线下滑时查 **骨龄片、TSH/游离 T4、IGF-1、CBC / 代谢 / 乳糜泻筛查** 等排除 GHD / 甲减 / 慢性病。
> - 严重青春期延迟且心理负担大者，可短程小剂量性激素「启动」（专科决定）。

## 六、Memory Hook

```
矮的钥匙不是「矮」，是「怎么矮」：
稳定平行 + 无症状 + 青春期延迟 + 骨龄延迟 + 会追赶 = CGD（晚长）
父母矮 + 骨龄正常 = 家族性矮小（终身矮但健康）
跨百分位下滑 = 报警 → GHD / 甲减 / 慢性病
甲减看「便秘+皮肤干+怕冷」；GHD 看「生长速度骤降」
```

## 七、复习自测

> [!question]- 14 岁男孩自 2 岁起身高平行低于第 3 百分位，BMI 第 50 百分位，Tanner 1，无便秘 / 皮肤干，父母身高正常。最可能诊断？
> **宪法性生长延迟（constitutional growth delay，晚长）**。稳定平行生长 + 无系统症状 + 青春期与骨龄延迟 + 体重正常，是正常变异；骨龄片会示骨龄延迟，成人身高多可接近正常。

> [!question]- 哪一项最能把「宪法性生长延迟」与「家族性矮小」区分开？
> **骨龄（bone age）**。CGD 骨龄**延迟**（生长板尚开放、能追赶）；家族性矮小骨龄**正常**（遗传决定、追不上）。另外家族性矮小常有**矮的父母**。

> [!question]- 哪些线索提示矮小是「病理性」而非宪法性延迟？
> **跨百分位下滑（生长速度骤降）+ 系统症状（便秘 / 皮肤干 / 乏力提示甲减；中线畸形 / 微阴茎提示 GHD）+ 体重 / BMI 异常（慢性病 / 营养不良）**。这些均不见于单纯 CGD。

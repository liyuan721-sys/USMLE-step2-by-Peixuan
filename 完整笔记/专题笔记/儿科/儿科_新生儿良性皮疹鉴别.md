# 新生儿良性皮疹鉴别 Benign Neonatal Rashes

> [!info] 一句话定位
> **生后第一周、状态良好 (well-appearing) 的婴儿 + 红斑性丘疹/脓疱 = 良性新生儿疹**，多为临床诊断、无需治疗、自限。考点核心：**与需治疗的危险皮疹（先天梅毒 / 新生儿 HSV / 葡萄球菌脓疱病）鉴别**——后者各有"破绽"（累及掌跖、水疱成簇、蜜黄痂、全身症状）。速查见 [[完整笔记/专题笔记/皮肤/皮肤_HighYield速查总览]]。

---

## 一、五种良性新生儿疹对照表 ⭐

> [!tip] 记忆主轴：**起病时间 + 分布 + 形态**
> 良性疹共性 = 婴儿吃奶好、无发热/嗜睡/肝脾大；处理几乎都是 **观察 (observation)**。

| 诊断 | 起病时间 | 分布 / 形态 | 处理 / 转归 |
|---|---|---|---|
| **Erythema toxicum neonatorum 新生儿毒性红斑 (ETN)** | 出生至 **3 天** | **躯干 + 近端肢体**红斑性斑疹/丘疹 → 进展为**脓疱**；**不累及掌跖**（无毛囊处） | 观察；**1 周内自行消退** |
| **Milia 粟丘疹** | 出生时 | **面部坚实白色丘疹** | 观察；1 个月内消退 |
| **Miliaria rubra 红痱（热疹）** | 任何年龄，**出生时无** | 闭塞区 / 间擦区**丘疹** | 避免过热（凉爽环境、薄棉衣）；重者外用糖皮质激素 |
| **Neonatal pustular melanosis 新生儿脓疱性黑变病** | 出生时 | **非红斑性脓疱** → 演变为**带领圈样鳞屑 (collarette of scale) 的色素沉着斑**；弥漫，**可累及掌跖** | 观察；脓疱数天消退，**色素沉着可持续数月** |
| **Neonatal cephalic pustulosis 新生儿头部脓疱病**（旧称"新生儿痤疮"） | **约 3 周龄** | **仅面部 + 头皮**红斑性丘疹脓疱；**无粉刺 (no comedones)**；与 **Malassezia 马拉色菌**相关 | **温和清洁 (gentle soap & water) 即可**、自限（数周-数月）；重者外用酮康唑 ketoconazole / 低效价激素 |

> [!success] 起病时间默写口诀（兑现 Q21580 TODO）
> **"生时 3 种、热疹随时、头疱三周"**
> - 出生即有：**Milia 粟丘疹**（白丘疹）、**Pustular melanosis 脓疱性黑变病**（脓疱→色沉）
> - 出生～3 天：**ETN 毒性红斑**
> - **任何时候、出生时无**：**Miliaria rubra 红痱**
> - **约 3 周**：**Cephalic pustulosis 头部脓疱病**

![[{F3A94CD0-B711-44B0-B0C7-D0B5A104BD20}.png|300]]
*红痱 / 热疹 (miliaria rubra, heat rash)：**细小 (2–4mm)、弥漫的红斑性丘疹**，无脓疱——过热/过度包裹的婴儿闭塞区/间擦区；处理 = 凉爽环境 + 薄棉衣（重者短程低效价外用激素）。与 ETN 区别：红痱**出生时无**、为均一细丘疹而非脓疱。*

## 二、ETN 深挖（最高频）

> [!note] ETN 病理 + 钥匙
> - **病因**：不明，可能为对皮肤细菌穿透毛囊的**先天免疫反应**（故不累及无毛囊的掌跖）。
> - **典型**：足月儿、生后 ≤3 天；小、**边界不清**的红斑性斑疹/丘疹 → 小而坚实的**脓疱**位于红斑基底；**除掌跖外任何部位**。
> - **婴儿状态良好、无症状**。
> - **临床诊断、无需治疗**；非典型者偶需查脓疱内容物 → **无菌 + 嗜酸性粒细胞浸润 (eosinophilic infiltrate)**（Wright 染色）。
> - 皮疹严重度可**波动 (wax and wane)**，但**1 周内自行消退** → **reassurance 安抚**即可。

> [!danger] Q3122 经典陷阱：母亲"唇疱疹/cold sore"病史是干扰
> 题干给"母亲偶发唇疱疹（HSV，单纯疱疹病毒 herpes simplex virus）"是为了**诱你选 acyclovir 阿昔洛韦**。但 ETN **缺水疱成簇、无掌跖受累、无全身症状**——不符合新生儿 HSV。生后第一周状态良好 + 躯干近端红斑丘疹脓疱 = ETN → **观察**。

![[{0C3DA29C-0FB4-4575-94E4-AB09B32F73EA}.png|300]]
*新生儿毒性红斑 (erythema toxicum neonatorum, ETN)：足月新生儿**躯干**散在**边界不清的红斑性斑疹/丘疹**（部分中央小脓疱）——典型分布，**不累及掌跖**。*

![[{F88ABC3E-ACB8-47DC-8639-ED0E97300D5C}.png|300]]
*ETN 另例：婴儿躯干 + 近端肢体红斑丘疹脓疱，婴儿状态良好 (well-appearing)；临床诊断、1 周内自行消退 → 安抚 (reassurance)。*

## 三、新生儿头部脓疱病深挖（易与新生儿痤疮混淆）

> [!note] Neonatal cephalic pustulosis (NCP, 新生儿头部脓疱病) 病理 + 钥匙
> - **旧称误导**："新生儿痤疮 (neonatal acne)"是旧名 → 现认为并非真正痤疮，**无粉刺 (no comedones / no open-closed comedones)**，而是对皮肤定植的 **Malassezia 马拉色菌**（一种酵母菌）的炎症反应。
> - **起病**：**约 3 周龄**（晚于 ETN 的生后 ≤3 天）；区别于真性婴儿痤疮 (infantile acne) ——后者更晚（约 3-6 个月）、**有粉刺**、可遗留瘢痕。
> - **典型**：**仅面颊、前额、头皮**红斑基底上的小丘疹脓疱；**无粉刺、无结节囊肿**；婴儿状态良好。
> - **处理**：**临床诊断**，**温和清洁 (gentle soap & water) 即可**，数周-数月**自限**；外观顽固/家长焦虑时可外用**酮康唑 ketoconazole** 乳膏（抗 Malassezia）或短程低效价**外用糖皮质激素 (topical corticosteroid)**。
> - **不需要**口服或外用抗生素、不需要异维 A 酸 (isotretinoin)。
> - **别上药陷阱**：避免**油性润肤剂 (oily emollients)**（堵塞、助长 Malassezia）、**mupirocin 莫匹罗星**（抗菌，非真菌）、**permethrin 氯菊酯**（抗疥螨）、**salicylic acid 水杨酸**（角质剥脱、刺激）——这些都是 stem 里诱选的错误处理，正解永远是温和清洁 + 安抚 (reassurance)。

> [!tip] NCP vs 真性婴儿痤疮 (infantile acne) 一句话
> **3 周 + 仅面/头皮 + 无粉刺 + Malassezia → NCP（温和清洁）**；**3-6 月 + 有粉刺/结节 + 可瘢痕 → infantile acne（需治疗、查雄激素源）**。

![[{9DB2ABCF-DA74-4110-87E6-41A71B67D92E}.png|360]]
*对照——粉刺性痤疮 (comedonal acne)：散在**开放性粉刺 (open comedones, "黑头" / 毛囊堵塞)**。NCP 的"破绽"正是 **无粉刺**；而真性婴儿痤疮 (infantile acne) 才**有粉刺**如图——见到粉刺即排除 NCP。粉刺性痤疮的干扰处理是外用水杨酸 (salicylic acid)，对 NCP 无效。*

## 四、必须排除的危险鉴别（各有"破绽"）⭐

> [!warning] 良性疹 vs 危险疹：看"红旗"
> 良性疹 = well-appearing + 无掌跖水疱/蜜黄痂 + 无全身症状。出现下列任一 → 警惕。

| 危险诊断 | 关键"破绽"（与良性疹不同点） | 处理 |
|---|---|---|
| **Congenital syphilis 先天梅毒** | 躯干/肢体斑丘疹，但**累及足底 (soles)** + **鼻炎 (snuffles)** + **肝脾大** + 淋巴结肿大；通常**无脓疱** | **静脉青霉素 penicillin** |
| **Neonatal HSV 新生儿单纯疱疹** | **红斑基底上成簇水疱 (clustered vesicles)**；可伴脓毒症样/脑炎 | **静脉阿昔洛韦 acyclovir** |
| **Staphylococcal pustulosis 葡萄球菌脓疱病 / 脓疱疮** | 局限红斑性脓疱/水疱/大疱 → 破溃成**糜烂 + 蜜黄色痂 (honey-colored crust)**；好发**尿布区或既往创面（如包皮环切处）** | **外用 mupirocin 莫匹罗星** |

> [!tip] 形态学速辨
> - **脓疱 + 躯干近端 + 掌跖不累及 + 状态好 → ETN（观察）**
> - **成簇水疱 → HSV（acyclovir）**
> - **蜜黄痂 / 尿布区 / 包皮创面 → 葡萄球菌脓疱病（mupirocin）**
> - **累及足底 + 鼻炎 + 肝脾大 → 先天梅毒（penicillin）**

![[{6EF34FF6-0664-4530-BE72-E36B77DCE05F}.png|360]]
*新生儿单纯疱疹 (neonatal herpes simplex virus, HSV) 的关键形态——**红斑基底上成簇水疱 (clustered vesicles on an erythematous base)**（标注版）。这正是 Q3122 母亲"唇疱疹史"想诱你套用的图；ETN 无此成簇水疱 → 不选阿昔洛韦 (acyclovir)。*

> [!note] 另一干扰项：婴儿疥疮 (scabies in an infant)
> 疥疮 (scabies, 疥螨感染) 也可呈红斑性丘疹/脓疱，外用氯菊酯 (permethrin) 治疗——但与良性新生儿疹不同：**弥漫受累、尤其累及头皮/手/足（掌跖）**，且**剧痒**（新生儿表现为烦躁、拒奶）。Q3122 患儿掌跖不累及、无烦躁 → 排除疥疮。
>
> ![[{E0216B53-E5A6-4FD0-9149-199C5A56C333}.png|320]]
> *婴儿疥疮 (scabies in an infant)：**累及足部 (palms/soles) 的弥漫红斑性丘疹及脓疱**——掌跖受累 + 剧痒是与 ETN 的分水岭。*

# 新生儿感染病原谱

> [!info] 笔记定位
> 新生儿（neonate，≤28 天）感染按 **① 时序（早发 <7 天 vs 晚发 >7 天）② 综合征（败血/脑膜/肺炎/结膜炎）③ 先天 TORCH（经胎盘）** 三轴定病原 + 预防 + 经验治疗。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q3671]]（晚发 GBS·IAP 不防）+ [[mistakes/uworld-mistakes_2026-06#^Q3676]]（先天弓形虫）。

---

## 一、一图总览 ⭐⭐⭐

```
新生儿感染（≤28 天）
├─ 围产期/出生后获得
│   ├─ 早发败血(<7 天)  产道垂直 → GBS⭐ / E.coli·G⁻ / Listeria        → IAP 可防
│   └─ 晚发败血(>7 天)  水平/环境(定植接触/导管) → GBS / G⁻ / 金葡·CoNS  → IAP 不防
├─ 按综合征
│   ├─ 脑膜炎 <1 月  GBS / E.coli / Listeria / HSV
│   ├─ 肺炎       早发 GBS·E.coli；4-12 周 衣原体(无热+staccato咳)
│   └─ 结膜炎     淋球菌(2-5天超急脓) / 衣原体(5-14天) / 化学(<24h)
└─ 先天(经胎盘 TORCH)  Toxo / 梅毒·HIV·parvo·VZV·Zika / Rubella / CMV⭐ / HSV
```

---

## 二、早发 vs 晚发败血/脑膜炎 ⭐⭐⭐

| | **早发（early-onset，<7 天）** | **晚发（late-onset，>7 天 ~28 天）** |
|---|---|---|
| **传播** | **产道垂直**（母生殖道/羊水上行）| **水平/环境**（定植接触者、未洗手、导管）|
| **首位病原** | **GBS（B 族链球菌 / S. agalactiae）** ⭐ | **GBS（晚发）+ E. coli/Klebsiella + 金葡/凝固酶阴性葡萄球菌（CoNS）** |
| **其他** | E. coli/G⁻ 杆菌、**Listeria（李斯特菌）**、HSV | Listeria |
| **IAP（产时抗生素预防）** | **可降低早发 GBS** | **不改变晚发 GBS**（[[mistakes/uworld-mistakes_2026-06#^Q3671]]）|

> [!danger] 新生儿首位仍 GBS + IAP 只防早发
> **≤28 天败血/脑膜首位 = GBS**；**晚发 GBS（>1 周）水平传播、IAP 不防** → "母亲没做 IAP/择期剖宫产未做"**不能排除 GBS**（[[mistakes/uworld-mistakes_2026-06#^Q3671]]）。
> **凶兆 = 低体温 + 白细胞减少 + 核左移**（不一定发热/白细胞升）。

## 三、按综合征定病原

> [!info] 脑膜炎（<1 月）
> **GBS / E. coli / Listeria / HSV**。经验 **氨苄西林（ampicillin）+ 头孢噻肟（cefotaxime）**（±acyclovir 若疑 HSV）；**不用头孢曲松（ceftriaxone）**——核黄疸 + 钙沉淀。详见 [[完整笔记/专题笔记/感染/感染_脑膜炎CSF鉴别]] §三·五。

> [!info] 肺炎
> **早发**：GBS、E. coli（同败血）。**4-12 周龄衣原体肺炎（Chlamydia trachomatis）**：经产道感染，**无热 + staccato（断续）咳 + 呼吸增快**，常有**包涵体结膜炎史**、外周嗜酸↑；治 **大环内酯（azithromycin/erythromycin）**。

> [!info] 新生儿结膜炎（ophthalmia neonatorum，按发病时间）⭐
> | 发病 | 病原 | 处理 |
> |---|---|---|
> | **<24h** | 化学性（硝酸银，已少用）| 观察 |
> | **2-5 天** | **淋球菌（N. gonorrhoeae）超急脓性** | **静脉/肌注头孢曲松（ceftriaxone）单剂** + 冲洗；查衣原体 |
> | **5-14 天** | **衣原体（C. trachomatis）包涵体** | **口服大环内酯（erythromycin/azithromycin）**（+ 防衣原体肺炎）|

## 四、先天感染（congenital，经胎盘）= TORCH ⭐

| 病 | 招牌 |
|---|---|
| **Toxoplasma（弓形虫）** | **弥漫颅内钙化 + 脑积水/巨头 + 脉络膜视网膜炎**；母猫粪/未熟肉（[[mistakes/uworld-mistakes_2026-06#^Q3676]]）|
| **梅毒 Syphilis** | 鼻炎(snuffles)、骨软骨炎、皮疹、肝脾大、晚期 Hutchinson 牙 |
| **风疹 Rubella** | **白内障 + PDA（动脉导管未闭）+ 感音神经性耳聋** |
| **巨细胞病毒 CMV** ⭐ | **室周钙化 + 小头 + 感音神经性耳聋**（最常见先天感染）|
| **HSV（围产）** | 生后 1-3 周：皮肤疱疹 / 脑炎 / 播散；治 **阿昔洛韦（acyclovir）** |
| **其他** | HIV / parvovirus B19（胎儿水肿）/ VZV / Zika（重度小头）|

> [!tip] 弓形虫 vs CMV 钙化（高频）
> **弓形虫 = 弥漫钙化 + 脑积水/巨头**；**CMV = 室周钙化 + 小头 + 耳聋**。详见 [[mistakes/uworld-mistakes_2026-06#^Q3676]] / [[完整笔记/Peixuan分科笔记/TORCH]]。

---

## 五、预防

- **GBS**：孕 **36-37 周**直肠阴道筛查 + **产时抗生素预防（IAP，青霉素）**；**仅防早发，不防晚发**。
- **HSV**：母有**活动性生殖器疱疹 → 剖宫产**。
- **新生儿结膜炎**：出生**红霉素眼膏（erythromycin ointment）**预防淋球菌。
- **乙肝 HBV**：母 HBsAg⊕ → 新生儿 **HBIG + 疫苗**。
- **梅毒/HIV**：孕期筛查 + 治疗。

---

## 六、经验治疗

| 情形 | 经验方案 |
|---|---|
| **新生儿败血/脑膜炎** | **氨苄西林（ampicillin）+ 庆大霉素（gentamicin）** 或 **氨苄西林 + 头孢噻肟（cefotaxime）** |
| **疑 HSV** | + **阿昔洛韦（acyclovir）** |
| **⚠️ 禁用** | **头孢曲松（ceftriaxone）**——置换胆红素→核黄疸 + 与含钙液沉淀致死 → 用 cefotaxime |
| **为何含氨苄西林** | **覆 Listeria + 肠球菌**（头孢盲区）|

## 七、Memory Hook

```
时序：早发(<7天)产道垂直 GBS/E.coli/Listeria(IAP可防) / 晚发(>7天)水平 GBS/金葡/CoNS(IAP不防)
首位永远 GBS；IAP 只防早发；凶兆=低体温+白细胞减少+核左移
脑膜<1月：GBS/E.coli/Listeria/HSV → amp+cefotaxime(禁ceftriaxone)+acyclovir
结膜炎：2-5天淋球菌(ceftriaxone) / 5-14天衣原体(口服大环内酯)
先天TORCH：弓形虫(弥漫钙化巨头) / CMV(室周钙化小头聋) / 风疹(白内障PDA聋) / HSV(疱疹脑炎)
```

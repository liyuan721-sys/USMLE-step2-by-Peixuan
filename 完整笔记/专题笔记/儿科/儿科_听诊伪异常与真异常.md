# 儿科_听诊伪异常与真异常

> [!info] 一句话主旨
> 小儿**上呼吸道感染 (upper respiratory tract infection, URI)** 时，上气道的拥塞噪音可**传导 (transmitted sounds)** 到胸前听诊区，听起来像额外杂音或异常呼吸音。若**病史是典型 URI + 肺部清 + 全身状况好 + 无真正心脏体征**，就是**伪异常听诊 (pseudoabnormal auscultation)**，回到最朴素的诊断：病毒性 URI。

---

## 一、核心概念：传导性上气道噪音

鼻咽部分泌物与拥塞使气流变"吵"，这种噪音沿气道**向下传导**，在胸前听诊时可被误听为胸腔内的病理性杂音/呼吸音。

```text
病毒性 URI
  ↓ 鼻/咽拥塞、分泌物
上气道气流噪音 (noisy upper-airway airflow)
  ↓ 声音向下传导
胸前听诊听到"额外声音" (transmitted sound)
  ↓
误判为病理性听诊发现（pseudoabnormal）
  ↓ 实际：无结构性心脏病，无下气道/肺实质病变
```

> [!tip] Memory Hook
> **"鼻子吵，胸口听到回声"** —— 病史像感冒 + 肺清 + 精神好 → 那声音是从上面传下来的，不是心脏或肺出了事。

---

## 二、解题反射：识别 NBME 的"音频干扰项"（NBME11 Q178）

> [!warning] 出题套路
> 题干给一个**典型 URI 故事**（发热 + 咳嗽 + 清亮鼻涕 3 天，既往健康），却附上一段"**cardiac examination 音频**"，诱导你往 murmur / VSD / 心包炎方向选。**别被音频带偏** —— 先看临床全貌。

**支持"伪异常 / 良性"的线索（本题全中）：**

- 3 天发热、咳嗽、**清亮鼻涕** → 经典 URI 故事
- 既往健康的 4 岁幼儿 → 慢性心脏病基线概率低
- **呼吸音正常** → 不支持肺炎、喘鸣、心衰肺水肿
- **其余查体无异常** → 与严重心肺病不匹配

> [!tip] 一句话锁定
> **典型 URI 病史 + 肺清 + 全身好 + 无固定杂音/结构性体征** → 诊断就是 **病毒性 URI**，那段"心脏音频"是传导性上气道噪音。

## 三、鉴别：什么时候"额外声音"是真病理

| 诊断 | 真正表现 | 为何本题不符 |
|---|---|---|
| 病毒性 URI（含传导性噪音）| 流涕 + 咳嗽 + 查体基本良性 | **最符合** |
| 充血性心衰 (CHF) | 呼吸困难、水肿/肝大、灌注差 | 呼吸循环稳定 |
| 室间隔缺损 (VSD) | **全收缩期杂音 (holosystolic murmur)**，常更早发现 | 无固定杂音/结构表现 |
| 心包炎 (pericarditis) | 体位相关胸痛、心包摩擦音 (friction rub) | 无胸痛，整体像 URI |
| 特发性肺动脉高压 (idiopathic pulmonary hypertension, HTN) | 劳力症状、P2 亢进、右心衰 | 年龄与表现均不符 |

> [!info] 良性 vs 病理的分界线
> - **传导性上气道噪音 (transmitted)**：随体位/清理鼻道/哭闹变化，**肺野清**，全身好。
> - **真心脏杂音**：**固定**、有明确定位/放射、可伴其他心脏体征（P2 异常、震颤、外周灌注/生长异常）。
> - **真下气道/肺实质音**（湿啰音 crackles、喘鸣 wheeze）：**双肺可闻及、不随上气道清理消失**。

## 四、复习自测

> [!question]- 4 岁既往健康儿童，发热咳嗽清亮鼻涕 3 天，呼吸音正常、余查体无异常，"心脏听诊音频"可闻及额外声音。最可能诊断？
> **上呼吸道感染 (upper respiratory tract infection, URI)**。那段额外声音是**传导性上气道噪音 (transmitted upper-airway sound)**，非病理性心脏杂音。

> [!question]- 如何把"传导性噪音"与真正的病理性心脏杂音区分开？
> 传导音**随体位/清理鼻道变化、肺野清、全身状况好**；真杂音**固定、有定位放射、常伴其他心脏体征**（P2 异常、震颤、生长/灌注异常）。

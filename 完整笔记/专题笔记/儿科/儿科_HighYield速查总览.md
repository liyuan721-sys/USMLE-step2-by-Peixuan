> [!tip] 儿科解题三轴（先抓这三个）

> 3. **问的是"最下一步 next step"**：最合适检查 / 最先处理 / 确诊方法 / 观察 vs 立即干预。

## 一、新生儿评估与产伤（Newborn assessment & birth trauma）

### 1.1 Apgar + 新生儿筛查 (newborn screening)

- **Apgar (阿氏评分)**：Appearance / Pulse / Grimace / Activity / Respirations；**1 min + 5 min** 评；正常 **≥7**。
- **新生儿筛查口诀**："Please check your baby before going home" = PKU / 囊性纤维化 (cystic fibrosis, CF) / 先天性肾上腺增生 (CAH) / β-地中海贫血 (β-thalassemia) / 生物素酶缺乏 (biotinidase deficiency) / 半乳糖血症 (galactosemia) / 甲减 (hypothyroidism) / 同型半胱氨酸尿症 (homocystinuria) / 严重联合免疫缺陷 (SCID)。
- ⚠️ **母乳喂养前必先排半乳糖血症**（乳糖→galactose 蓄积）。

### 1.2 产伤头皮肿块（跨缝 = 钥匙）

| 病变 | 钥匙 | 跨颅缝？ | 风险/处理 |
|---|---|---|---|
| 帽状腱膜下水肿 Caput succedaneum | 出生即有、头皮水肿 | **跨缝** | 自限 |
| 头颅血肿 Cephalohematoma | 骨膜下出血 | **不跨缝** | 可致高胆红素血症 |
| 帽状腱膜下出血 Subgaleal hemorrhage | 真正急症、vacuum 助产、弥漫波动性肿胀 | 跨缝弥漫 | **失血性休克**（苍白/心动过速/低血压）|

### 1.3 脐部异常（一钩一病）

```
脐带延迟脱落 (>3-4 周) + 反复感染无脓 + 中性粒↑ → 白细胞黏附缺陷 LAD
脐部持续出血（PT/PTT 正常）          → 因子 XIII 缺乏 (Factor XIII deficiency)
尿液从脐流出                          → 脐尿管未闭 (patent urachus)
粪便/胎便从脐流出                     → 卵黄管未闭 (patent omphalomesenteric duct)
```

🧩 新生儿良性皮疹鉴别（ETN 毒性红斑 / 粟丘疹 / 红痱 / 脓疱性黑变病 / 头部脓疱病 + 危险鉴别：先天梅毒·HSV·葡萄球菌脓疱病）→ [[完整笔记/专题笔记/儿科/儿科_新生儿良性皮疹鉴别]]

## 二、新生儿黄疸（Neonatal jaundice）

> [!danger] 一秒分流：出现时间
> **<24h 出现 = 病理性 (pathologic)**，永远要查（溶血/感染）；上升过快、持续 >2 周、结合胆红素↑、合并贫血/肝脾大也算病理。

| 类型 | 时间 | 机制 | 处理 |
|---|---|---|---|
| 生理性 physiologic | **>24h**，足月第 2-3 天峰 | UGT 活性低 + RBC turnover 高 | 观察 |
| 母乳喂养性 breastfeeding | **第 1 周** | 喂养不足/脱水/肠肝循环↑ | 增加喂养频率、纠正含接 latch |
| 母乳性 breast milk | **第 1 周后/第 2 周** | 婴儿健康、吃奶好 | 继续母乳 |
| ABO 溶血 | **<24h** + 母 O 型 | 抗体介导溶血 | 查胆红素 + 风险分层 |

- **核黄疸 (kernicterus) 风险**：嗜睡 + 喂养差 + 肌张力异常 + 尖叫 + **后弓反张 (opisthotonus)**。
- **治疗阶梯**：光疗 (phototherapy) → 换血 (exchange transfusion)（按 Bhutani 曲线分层）。

## 三、新生儿呼吸窘迫（RDS / TTN / MAS）

> [!tip] 三联鉴别钥匙
> **早产 = RDS；剖宫产/急产足月 = TTN；足月 + 胎粪污染 = MAS。**

| 病 | 钥匙 | 影像/机制 | 处理 |
|---|---|---|---|
| 新生儿呼吸窘迫综合征 NRDS | **早产**、surfactant <32 周 | CXR **white-out 白肺**；L/S <2:1 | 表面活性物质 (surfactant) + O2；产前倍他米松 (betamethasone) 预防 |
| 新生儿暂时性呼吸增快 TTN | **剖宫产/急产**、足月 | 滞留肺液 (retained fluid) | 支持治疗 |
| 胎粪吸入综合征 MAS | **足月** + 胎粪 | 胎粪本身无菌 | O2 + 呼气末正压 (PEEP, positive end-expiratory pressure) |
| 先天性膈疝 CDH | 舟状腹 + 患侧呼吸音消失 + 心音移位 | 肠疝入胸 | **气管插管 + 胃管减压**；❌ 禁 bag-mask（气进肠）|

> [!warning] 气道/食管畸形钥匙
> 喂养呛咳 + NG 管在盲袋打卷 → **食管闭锁 + 气管食管瘘 (EA + TEF)**；喂养时发绀、CHARGE association → **后鼻孔闭锁 (choanal atresia)**。

🫁 新生儿呼吸窘迫 6 病鉴别（肺源 4 病 + CDH + 导管依赖先心）+ 第一步处理速查 + 高氧试验 → [[完整笔记/专题笔记/儿科/儿科_新生儿呼吸窘迫鉴别]]

## 四、生长发育里程碑与矮小（Milestones & short stature）

### 4.1 高频里程碑 + 原始反射 (primitive reflexes)

```
陌生人焦虑 stranger anxiety  6 月
分离焦虑 separation anxiety  至 1 岁
发脾气 temper tantrums       15-24 月
画圆 copy circle             3 岁
集体游戏 group play          4 岁

Moro 拥抱反射  → 4-6 月消失（最早消失）
Rooting 觅食   → 约 4 月消失
成人保留原始反射 / Babinski → 病理性（额叶功能障碍）
```

### 4.2 矮小身材鉴别（先测骨龄 bone age）

| 诊断 | 骨龄 | 钥匙 | 处理 |
|---|---|---|---|
| 体质性生长延迟 constitutional growth delay | **落后** | 青春期延迟、最终身高接近正常、"晚长" | 安慰 reassurance |
| 家族性矮身材 familial short stature | **正常 (= 年龄)** | 父母矮、生长速度正常 | 安慰 reassurance |
| 病理性 / 生长激素缺乏 GH deficiency | 常落后 | **生长曲线下掉 (crossing percentiles)** | 进一步评估 |

> [!success] 一句话口诀
> **骨龄慢 = 体质晚长；骨龄齐 = 家族矮；曲线掉 = 去找病。**

## 五、疫苗与发热出疹病（Vaccines & febrile rash）

- **孕期 Tdap**：孕 **28 周**（防婴儿百日咳 pertussis）。
- **呼吸道合胞病毒 (RSV, respiratory syncytial virus) 预防**：帕利珠单抗 (palivizumab)（高危婴儿）。

| 出疹病 | 钥匙 | 病原/处理 |
|---|---|---|
| 传染性红斑 erythema infectiosum | **slapped cheek 掌拍颊** | Parvovirus B19 |
| 幼儿急疹 roseola | 高热数天 → **退热后出疹** | HHV-6 |
| 猩红热 scarlet fever | **砂纸样疹 + 草莓舌** | A 组链球菌 (GAS, group A Streptococcus) → 抗生素 |
| 川崎病 Kawasaki | **发热 ≥5 天** + 结膜炎 + 口腔/肢端改变 + 颈淋巴结 | **IVIG + 阿司匹林**；心脏并发 ① **冠脉瘤**(远期 MI) ② **淋巴细胞性心肌炎→心室功能↓→心衰**(婴儿急性:奔马律/肺水肿/肝大) |
| 过敏性紫癜 HSP (IgA 血管炎) | **可触性紫癜** + 腹痛 + 关节痛 + 血尿 | 支持治疗 |

## 六、发热婴儿按月龄 workup（Febrile infant）

> [!danger] 月龄阈值（next-step 必考）
> - **≤21 天新生儿发热**：腰穿 (LP) 是单独指征（免疫不成熟，脑膜炎风险）。
> - **婴儿无源发热**：最常见 = **occult UTI** → 经验抗生素前**先留尿** = **尿常规 (UA) + 尿培养**（导尿取样）。
> - UTI 高危：女婴 / 未割包皮男婴 / 泌尿畸形。

🧩 完整月龄分层决策树 → [[完整笔记/专题笔记/儿科/儿科_发热婴儿按月龄workup]]
🧩 分龄生命体征正常值（HR/BP 阈值）→ [[完整笔记/专题笔记/儿科/儿科_分龄生命体征正常值]]

> [!tip] 热性惊厥 (febrile seizure) — 单纯 vs 复杂 vs 脑膜炎相关
>
> | 类型 | 特征 | 处理 |
> |---|---|---|
> | **单纯性 simple** | 全身、<15 min、24h 内单次、发作后查体正常 | **只治发热病因（如急性中耳炎 AOM），不查惊厥**（不需 CT / 腰穿 LP / 抗癫痫药）|
> | 复杂性 complex | 局灶 / 反复 / >15 min | 酌情进一步评估 |
> | 脑膜炎相关 | 发热 + 颈强 / 意识改变 / 瘀点 | **腰穿 (lumbar puncture, LP) + CNS（中枢神经系统）感染检查** |
>
> 高发 6 月–5 岁；"短、全、醒得快"= 单纯性。〔源：NBME11A Q193 → [[NBME11_错题本#^Q193]]〕

## 七、上下呼吸道（Upper & lower airway）

### 7.1 急性气道 / 喘鸣 (stridor) 鉴别

| 病 | 钥匙 | next step |
|---|---|---|
| 喉软化 laryngomalacia | 慢性**吸气性喘鸣**、俯卧/直立改善、4-8 月峰 | 安慰（18 月缓解）|
| 血管环 vascular ring / 双主动脉弓 | **双相 biphasic stridor** + 喂养困难 + 颈后伸减轻 | **CTA chest** → 手术 |
| 哮吼 croup | **狗叫咳 + 声嘶 + steeple sign 尖塔征**、parainfluenza | 轻 = 地塞米松 (dexamethasone)；静息喘鸣 = + 雾化肾上腺素 (nebulized epinephrine) |
| 会厌炎 epiglottitis | **高热 + 流口水 (drooling) + 三脚架位 (tripod) + toxic**、Hib | **❌ 别压舌看咽 → 先保气道（插管）→ 再抗生素**；手术气道仅插管失败时 |
| 细菌性气管炎 bacterial tracheitis | toxic + 金葡 + 气管壁不规则 | 抗生素 + 气道管理 |

> [!warning] 会厌炎病原（疫苗状态分流）
> 未接种 → **Hib (b 型流感嗜血杆菌)**；已接种 → 链球菌/金葡 → ceftriaxone（金葡加万古霉素 vancomycin）。

🧩 喘鸣急性气道完整鉴别 → [[完整笔记/专题笔记/儿科/儿科_喘鸣stridor急性气道鉴别]]

### 7.2 毛细支气管炎 / 百日咳 / 肺炎

- **毛细支气管炎 bronchiolitis**：**RSV**、<2 岁、URI 前驱 + 喘息/湿啰音 → **支持治疗**（氧/吸痰/补液）。
- **百日咳 pertussis**：阵咳 + whoop + 咳后呕吐；**婴儿可仅 apnea** → 大环内酯 (azithromycin)。
- **肺炎年龄分型**：<5 岁 = 病毒；10-30 岁 = 支原体 (Mycoplasma)；0-2 月无热 staccato 咳 = 衣原体 (Chlamydia trachomatis)；**流感后恶化 = 金葡超级感染 (S. aureus superinfection)**。

### 7.3 中耳炎 / 外耳道炎

| 病 | 钥匙 | 处理 |
|---|---|---|
| 急性中耳炎 AOM | **鼓膜膨出 (bulging TM) + 光反射消失 + 气动耳镜活动度↓** | 所有 <6 月 / >6 月重症（高热>39、重度痛、双侧）→ **大剂量阿莫西林 (high-dose amoxicillin) 10 天**；近期用过/复发 → 阿莫西林-克拉维酸 (amoxicillin-clavulanate) |
| 中耳积液 OME | 鼓膜后积液、**无急性感染**、听力↓ | 先观察；持续 → 鼓膜置管 (tympanostomy tubes) |
| 外耳道炎 otitis externa | **拉耳廓/压耳屏 (tragus) 痛** | 鼓膜完整 = neomycin 滴耳；穿孔/置管 = **氟喹诺酮滴耳 (ciprofloxacin)** |

> [!info] 复发性 AOM → 传导性听力↓ → 影响言语发育
> 任何言语/语言障碍患儿的**第一步检查 = 听力测试 (hearing test)**（即使无频繁耳感染史）。可控危险因素 = **被动吸烟暴露 (secondhand smoke)**。

> [!warning] 鼓膜发红 ≠ 急性中耳炎
> **哭闹本身就能让鼓膜 (tympanic membrane, TM) 充血发红**。诊断 AOM 必须有**鼓膜膨隆**和/或**中耳积液**（充气耳镜 pneumatic otoscopy 下活动度/顺应性下降）。**红斑单独出现不诊断、不给抗生素**。〔源：Q4856 → [[mistakes/uworld-mistakes_2026-07#^Q4856]]〕

### 7.4 哮喘 & 囊性纤维化 CF

- **CF**：AR、7 号染色体、**CFTR**；诊断 = **汗氯试验 (sweat chloride，2 处 2 次)**；新生儿 = 胎粪性肠梗阻 (meconium ileus)，大孩子 = 远端肠梗阻；多系统 = 反复鼻窦肺感染 + 胰腺功能不全 (ADEK 缺乏) + 胆汁性肝硬化 + 输精管缺如 (男性不育) + 鼻息肉 + 杵状指。
- **CF 气道病原随龄变**：幼儿 = **金葡 (S. aureus)**；成人 = **铜绿假单胞菌 (Pseudomonas)**。

## 八、先天性心脏病（Congenital heart disease）

### 8.1 发绀型 5 T's（ductal-dependent → 先 PGE1）

```
Tetralogy of Fallot          法洛四联
Transposition (TGA)          大动脉转位（出生最致命）
Truncus arteriosus           永存动脉干
TAPVR                        完全肺静脉异位引流
Tricuspid atresia            三尖瓣闭锁

共同初始处理 → 前列腺素 E1 (PGE1, alprostadil)：keep PDA open 保持动脉导管开放
```

| 病 | 钥匙 | 处理 |
|---|---|---|
| TGA 大动脉转位 | 新生儿重度发绀 + **single loud S2** + **母亲糖尿病 (maternal diabetes)** | 先 PGE1 → 球囊房隔造口 → 手术 |
| TOF 法洛四联 | **PROVe**（PS / RVH / Overriding aorta / VSD）、**缺氧发作 tet spells**、蹲踞/膝胸位缓解 | 预后取决于 RV 流出道梗阻程度 |
| VSD 室间隔缺损 | **左下胸骨缘全收缩期粗糙杂音 (holosystolic)**、与 Down 相关 | 小缺损自闭；大缺损 → 心衰/生长不良/肺高压 |

### 8.2 良性杂音 (innocent murmur) vs 病理性

- **良性 (Still murmur)** 4 铁律：无症状 + 收缩期 + 短促柔和 + **站立时↓**（HCM 反向：站立↑）。家族史红旗 / 舒张期 / 持续性 → 病理性查超声。

## 九、消化（GI surgery & functional）

> [!danger] 胆汁性呕吐 (bilious emesis) = 外科急症直觉
> 新生儿胆汁吐 = 先想**梗阻/扭转**：NPO + NG 减压 + 补液 + 外科会诊 → 再影像（上消化道造影 upper GI series）。

| 病 | 钥匙 (buzzword) | 影像/确诊 | 处理 |
|---|---|---|---|
| 幽门狭窄 pyloric stenosis | **2-8 周、喷射性非胆汁性呕吐 (projectile nonbilious)、olive 包块、低氯低钾代碱** | **超声 ultrasound** | 先纠电解质 → 幽门肌切开 (pyloromyotomy) |
| 十二指肠闭锁 duodenal atresia | 首次喂养胆汁吐、羊水多、**Down**、**double bubble** | 双泡征 | NPO/NG/补液 → 手术 |
| 空肠闭锁 jejunal atresia | 早期胆汁吐、**triple bubble** | 三泡征 | 手术 |
| 肠旋转不良 + 中肠扭转 malrotation + volvulus | 新生儿**胆汁吐 + 病重 + 血便/休克** | upper GI series | **外科急症**，先稳再手术 |
| 坏死性小肠结肠炎 NEC | **早产 + 腹胀 + 喂养不耐受 + 血便** | **肠壁积气 pneumatosis intestinalis** | NPO + NG + 补液 + **广谱抗生素** ± 手术 |
| 先天性巨结肠 Hirschsprung | **48h 不排胎便 + 直肠指检 (DRE, digital rectal exam) 后喷射排便 (explosive stool)** | 确诊 = **直肠吸引活检 (rectal suction biopsy)**，无神经节细胞 | 切除无神经节段 |
| 肠套叠 intussusception | **阵发腹痛蜷腿 + 果酱样便 (currant jelly) + 香肠样包块** | **空气灌肠 (air enema) 诊断兼治疗** | 灌肠复位 |
| 梅克尔憩室 Meckel diverticulum | **无痛下消化道出血** | 锝-99m 扫描 (technetium-99m scan) | 手术切除 |

- **功能性便秘**：最常见 = **粪便潴留 (stool withholding)** → 解嵌 + 聚乙二醇 (polyethylene glycol, PEG) + 行为训练 + 增纤维。
- **半乳糖血症 galactosemia**：GALT 缺乏、喂养差 + 白内障 + 肝大 + **尿还原物质阳性** + **E. coli 败血症风险** → 去乳糖（**豆基配方 soy formula**）。
- **病毒性胃肠炎**：急性非血性水样泻 + 轻度脱水 → **口服补液 (oral rehydration solution, ORS) 居家**（最常见轮状病毒 rotavirus）。

> [!tip] 婴儿肠绞痛 (infantile colic)：3-3-3 + <3 月龄 → 安慰家长 + 安抚技巧
> **诊断标准**：健康婴儿无明显原因哭闹 **≥3 小时/天、≥3 天/周、持续 ≥3 周**，年龄 **<3 月龄**；**6 周龄达峰、3-4 月龄自行缓解**；常固定在每天同一时段（多为傍晚）。**排除性诊断 (diagnosis of exclusion)**，必须先做完整病史 + 体格检查。
> ⭐**判别线：发作间期是否正常**——colic 婴儿哭完恢复"开心、警觉"；器质性急症（肠套叠等）发作间期常**嗜睡萎靡**。
> **管理**：家长安慰 (reassurance) + 复习安抚技巧（安抚奶嘴、抱/摇/襁褓包裹 swaddling、暗房安静减少刺激、秋千/背带/推车、瓶喂时竖直体位减少吞气）。**不查影像、不给抗生素、不换配方**。
> ⚠️ 干扰项主场：腹部超声 = 肠套叠（**6-36 月**、阵发蜷腿 + 果酱样便）；H₂ 受体阻滞剂 = 重症胃食管反流病 (gastroesophageal reflux disease, GERD)（吐奶 + 喂后反弓 + 体重不增）；口服抗生素 = 需鼓膜膨隆/积液的 AOM；无乳糖配方 = 婴儿乳糖不耐极罕见且为**泡沫水样便**。〔源：Q4856 → [[mistakes/uworld-mistakes_2026-07#^Q4856]]〕

## 十二、遗传代谢与染色体病（Genetics）

| 病 | 钥匙 |
|---|---|
| Down syndrome (T21) | **AVSD/VSD + 十二指肠闭锁 + Hirschsprung 风险 + 肌张力低 + friendly demeanor** → 核型 karyotype |
| Edwards (T18) | clenched hands + 指重叠 + 摇椅足 (rocker-bottom feet) + VSD |
| Patau (T13) | 唇腭裂 + 多指 + 前脑无裂 (holoprosencephaly) |
| Fragile X | 大耳 + 长脸 + 学习问题 + **巨睾 (macroorchidism)**、FMR1 |
| PKU 苯丙酮尿症 | AR、**霉味 (musty odor)** + 湿疹 + 癫痫 + 智力障碍 → 限苯丙氨酸 + 补酪氨酸 |
| Prader-Willi | **paternal deletion**（chr15）、肌张力低 + 暴食 + 肥胖 + 性腺低 |

- **婴儿血管瘤 hemangioma**：出生后数周出现、先长后缩；高危特征（巨大/面部节段/眼周/声门下）→ **普萘洛尔 (propranolol)**；面部节段 → 查 PHACE 综合征（后颅窝畸形 Posterior fossa / 血管瘤 Hemangioma / 动脉异常 Arterial / 心脏异常 Cardiac / 眼异常 Eye）。

### 12.1 发育退化 vs 迟缓（两步定位）⭐

> [!tip] 第一步：分退化 vs 迟缓
> - **退化 (regression)** = **丢失**已获得技能（会走→不会走）→ 代谢/退行病。
> - **迟缓 (delay)** = 从头就慢、**从未获得**里程碑但**不倒退** → Angelman / Prader-Willi / Fragile X / Down / 脑瘫 (CP，静止非进行性)。

**第二步：退化型按倒退年龄 + 器官/眼/皮肤线索反查（结论）**

| 倒退年龄 | 病 | 一句话钥匙 |
|---|---|---|
| **~3–6 月** | Krabbe（克拉伯病）| 易激惹 + 发热 + 周围神经病，无樱桃红斑 |
| **~6 月** | Tay-Sachs（泰-萨克斯病）| 樱桃红斑 + 夸张惊跳，**无肝脾大** |
| **~6 月** | Niemann-Pick A（尼曼-匹克 A 型）| 樱桃红斑 **+ 肝脾大** |
| **6–18 月** | Rett（雷特综合征）| **女孩** + 语言退化 + 搓手 + 后天性小头 |
| **1–2 岁** | Hurler（MPS I，黏多糖贮积症 I 型）| 粗糙面容 + 角膜混浊 + 肝脾大 |
| **学龄 4–8 岁** | ALD（肾上腺脑白质营养不良）| **男孩** + 学习退化 + 肾上腺功能不全 |
| **5–10 岁** | Batten（juvenile NCL，神经元蜡样脂褐质沉积症）| 视力丧失 + 癫痫 |
| **麻疹后 7–10 年** | SSPE（亚急性硬化性全脑炎）| 认知退化 + 肌阵挛 + 麻疹史 |

> [!warning] 两大陷阱位
> - **PKU** 未治疗其实是"born normal → 进行性"（不是纯迟缓），考点在**筛查 + 低苯丙氨酸饮食可预防**。
> - **退化型自闭 (regressive autism)** 18–24 月语言退化会落进退化支，靠**无器官肿大/无脑白质病/无樱桃红斑**排除代谢病。
> - **Niemann-Pick C** = 青少年隐匿退化 + **垂直凝视麻痹**（胆固醇转运缺陷，≠ ~6 月的 A 型）。

## 十三、内分泌（CAH & 甲低）

> [!tip] CAH 血压口诀
> **21-羟化酶缺乏 (21-hydroxylase) = 低血压**（最常见，失盐：低钠高钾）；**11β-羟化酶缺乏 = 高血压**（脱氧皮质酮 DOC, 11-deoxycorticosterone ↑）。**"21 低压，11 高压"**。

- **先天性甲低 congenital hypothyroidism**：黄疸延长 + 肌张力低 + 大舌 + 脐疝 + 发育迟缓 → 新生儿筛查关键。
- **性早熟 (precocious puberty)**：先测**骨龄** → 测基础 **LH**（高 = 中枢性 CPP → 脑 MRI 排肿瘤；低 = 外周性 PPP → 腹盆影像）。

### 14.2 急性阴囊痛（提睾反射 cremasteric reflex 是钥匙）

| 病 | 钥匙 |
|---|---|
| 睾丸扭转 testicular torsion | 突发剧痛 + **提睾反射消失** + 高位横位 → **立即手术探查，别等检查** |
| 附睾炎 epididymitis | 渐进痛 + 尿路症状 + **提睾反射存在** + Prehn 阳性 |

| 鞘膜积液 hydrocele | 无痛 + **透光 (transillumination)** |
| 精索静脉曲张 varicocele | **bag of worms** + 左侧多 + 站立明显 |
| 睾丸肿瘤 | 无痛实性、不透光 |

### 14.3 虐待识别 + 青少年

- **虐待**：突发行为/学业改变 + 遗尿 + 不稳定家庭 + 照护者物质滥用 = 红旗；评估顺序 = **先单独问患儿（父母离场）> 问母亲 > 干预 > 心理治疗**；强制报告者**评估优先于报告**。
- **青少年筛查**：抑郁 **≥12 岁年度筛查**；性活跃女性 **≤25 岁**年度衣原体/淋病筛查；宫颈/HPV Pap **≥21 岁**。儿童抑郁可躯体化/易怒，SSRI 黑框警告（开药 1-2 周自杀意念最高，密切随诊）。

```

尼古丁中毒 nicotine     → 双相 + 毒蕈碱样 (muscarinic)
溺水 drowning           → 嗜睡不能护气道 → 气管插管
抗胆碱（环喷托酯）     → 散瞳/干黏膜/幻觉 → 毒扁豆碱 (physostigmine)
```

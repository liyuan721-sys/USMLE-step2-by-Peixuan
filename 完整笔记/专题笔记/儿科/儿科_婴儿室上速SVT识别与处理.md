# 婴儿室上速 SVT — 识别与处理

> [!info] 起源
> 派生自 NBME11 同主题两题：**Q004**（Block1，婴儿 SVT 首选处理 = 腺苷）+ **Q131**（Block3，婴儿 SVT 识别）。两题合看 = 婴儿 **室上性心动过速 (supraventricular tachycardia, SVT)** 的**识别 + 处理**闭环。

---

## §一 一句话锁定

> [!tip]
> **婴儿 (infant) + 心率 220–280/min + 规则窄 QRS 心动过速** = 首先诊断 **SVT**。婴儿不会诉"心悸"，常以**喂养差 (poor feeding) + 烦躁 (irritability) + 呼吸快 (tachypnea) + 苍白 (pallor)** 这种非特异表现出现。

---

## §二 识别要点

| 线索 | 意义 |
|---|---|
| **心率 220–280/min** | 婴儿 SVT 的量级钩子；最值钱的数字 |
| **规则 (regular) + 窄 QRS** | 上室起源；不规则 → 想房扑/房颤，宽 QRS → 想室速 |
| 喂养差 / 烦躁 / 呼吸快 / 苍白 | 低有效心输出的非特异表现（婴儿不诉心悸）|
| precordium 搏动强 (hyperdynamic) | 持续快速心律体征 |
| SpO₂ 正常、肺清 | 呼吸快非肺炎/缺氧 = 代偿性 |

> [!warning] 干扰项识别
> 婴儿 SVT 题常塞**轻度贫血 (anemia) + 轻度肝脾大**作钓饵 —— 这些解释不了**持续规则的 250/min 心律**。先看 ECG 的**心率量级 + 节律规则性 + QRS 宽窄**，别被贫血叙事带走。

---

## §三 处理（按血流动力学分层）⭐

```text
婴儿规则窄 QRS 心动过速 = SVT
   │
   ├─ 不稳定（休克 / 灌注差 / 意识改变）
   │      → 立即同步电复律 (synchronized cardioversion) 0.5–1 J/kg
   │
   └─ 稳定
          ├─ ① 迷走刺激 (vagal maneuvers)：婴儿用 冰水/冰袋敷面 (ice to face)
          │
          └─ ② 无效 → 腺苷 (adenosine) IV 快速推注（首选药物）
                     （Q004 考点：稳定婴儿 SVT 首选 adenosine，非维拉帕米）
```

> [!danger] 婴儿用药禁忌
> **维拉帕米 (verapamil, 钙通道阻滞剂) 禁用于 <1 岁婴儿**（可致低血压 / 心血管虚脱）→ 婴儿稳定 SVT 用**腺苷**而非维拉帕米。

---

## §四 横向鉴别（婴儿快心律 / 苍白）

| 诊断 | 最支持线索 | 为何不是 SVT |
|---|---|---|
| **SVT** ✅ | 220–280/min、规则窄 QRS | — |
| 房扑 (atrial flutter) | 锯齿样房波 / 变比传导 | 不如 SVT 常见 |
| 心肌炎 (myocarditis) | 发热 + 心衰体征 + S3 + 肺淤血 | 题干无感染/心衰 |
| 先天性溶血性贫血 (congenital hemolytic anemia) | 黄疸 + 脾大 + 贫血 | 解释不了 250 规则心律 |
| 缺铁性贫血 (iron deficiency anemia) | 母乳儿 >4–6 月渐起 | 不造成此节律 |
| 长 QT 综合征 (prolonged QT syndrome) | torsades（尖端扭转型室速）风险 | 非持续规则窄 QRS |

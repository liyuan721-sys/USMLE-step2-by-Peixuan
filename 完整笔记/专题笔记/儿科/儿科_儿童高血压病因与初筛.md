# 儿科_儿童高血压病因与初筛

> [!tip] 一句话锁定
> **年幼儿童（尤其学龄前）出现高血压 (hypertension, HTN)**，默认先按 **继发性高血压 (secondary hypertension)** 查；越小、越重，越不像原发。最常见的可查病因是 **肾源性 (renal / renovascular)**，初筛首选 **肾脏 + 多普勒超声 (renal ultrasonography with Doppler)**。

---

## 一、核心概念

| 年龄段 | 高血压最可能性质 | 首要评估方向 |
|---|---|---|
| 婴幼儿 / 学龄前 | 几乎都是 **继发性** | 肾实质病 / 肾血管性 |
| 学龄期 | 多为继发性 | 肾脏 + 主动脉缩窄 |
| 青春期 / 肥胖 | **原发性 (primary/essential)** 比例上升 | 生活方式 + 排除继发 |

> [!info] 关键判读
> 年龄越小 + 血压越高 → 越应积极找继发病因。原发性高血压在幼儿中罕见，属排除性诊断。

---

## 二、儿童继发性高血压病因（按系统）

```
儿童继发性 HTN 病因
├── 肾实质 renal parenchymal（最常见）
│     慢性肾病 / 反流性肾病 / 肾小球肾炎 / 多囊肾
├── 肾血管 renovascular
│     肾动脉狭窄（纤维肌发育不良 FMD 为儿童常见型）
├── 心血管
│     主动脉缩窄 coarctation（上肢高、下肢低 + 股动脉搏动弱/延迟）
├── 内分泌
│     嗜铬细胞瘤 / 甲亢 / CAH / 库欣 / 高醛固酮
└── 其他
      药物、OSA、颅内压升高
```

---

## 三、初筛下一步（本题考点）

> [!success] 初筛首选：肾脏 + 多普勒超声
> **Renal ultrasonography with Doppler** 能一次同时评估：
> - 肾脏**结构**（大小、瘢痕、囊肿、发育不良）
> - **肾动脉血流**（筛查肾血管性高血压）
>
> 因此是幼儿继发性高血压性价比最高的第一步影像。

### 为什么不是其他检查

| 选项 | 为什么不是最佳初筛 |
|---|---|
| 静脉肾盂造影 IV pyelography | 老式检查，非现代儿童 HTN 初筛首选 |
| 脑钠肽 BNP | 评估心衰，不是病因初筛主轴 |
| 多导睡眠图 polysomnography | 查阻塞性睡眠呼吸暂停 OSA，不符本年龄线索 |
| 甲状腺功能 thyroid tests | 可后续补充，非最佳第一步 |

> [!warning] 别被主动脉缩窄带偏
> 题干若特意给"四肢血压"且下肢**不低于**上肢 → 反而在**排除** coarctation（缩窄的典型是上肢高、下肢低 + 股动脉搏动减弱/延迟）。此时重点仍是"太小 + 太重 → 查肾"。

---

> [!question]- 复习题：3 岁儿童重度高血压，四肢血压相近（下肢未见明显偏低），肾功能与尿检正常，最合适的下一步？
> **肾脏 + 多普勒超声 (renal ultrasonography with Doppler)**。
> 幼儿高血压先按继发性查，肾源性最常见；四肢血压相近提示不像主动脉缩窄；肾功能正常**不能**排除肾血管性高血压。

> [!question]- 复习题：为什么正常的 Cr/BUN/尿检不能排除肾血管性高血压？
> 肾动脉狭窄早期可通过 RAAS 代偿维持肾小球滤过，血肌酐/尿检可正常；血流异常需靠多普勒或血管影像才能发现。

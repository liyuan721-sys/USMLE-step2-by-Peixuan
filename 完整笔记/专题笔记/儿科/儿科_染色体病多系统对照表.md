## 🧠 ID（智力障碍）三大常见病因速查 ⭐

> [!info] USMLE Step 2 高频
> Intellectual disability (ID) stem 最常考 3 大病因：**Down syndrome / Fragile X / Fetal alcohol syndrome (FAS)** —— 分别代表 **染色体病 / 单基因 X 连锁 / 产前致畸**三条路径。
> 触发：Q21629 (Down → karyotype, 2026-05-23) 解释表收编。

| 综合征 | 病因机制 | 面容 / 形态特征（dysmorphic） | 共病系统 | USMLE 反射钩 / 诊断 |
|---|---|---|---|---|
| **Down syndrome (T21, 21 三体)** | **染色体**（chromosomal）—— Trisomy 21（数目异常）| **upslanting palpebral fissures（上斜眼裂）** + **flat nasal bridge（扁平鼻梁）** + **epicanthal folds（内眦赘皮）** + **single palmar crease（单掌纹）** + 舌大 + 低位耳 + 短颈 | 先心 (**AVSD > VSD**) + **矮小** + **肥胖**倾向 + **传导性听力丧失**（OM + 咽鼓管功能不良）+ ALL/AML 风险↑ + 早发 Alzheimer (40+) + 甲减 + 颈椎不稳 | 多发畸形 + friendly demeanor + 先心 + 听力 → **karyotype** |
| **Fragile X syndrome** | **单基因 X 连锁**（X-linked）—— **FMR1** 基因 **CGG 重复 >200**（三核苷酸重复 + 表观甲基化沉默）| **long face with prominent chin & forehead（长脸 + 突出下颌额）** + **large, protruding ears（大耳前突）** + **macroorchidism（巨睾，青春期后）** + **macrocephaly（巨头）** | **self-injurious behavior（自伤行为，如 hand biting）** + 焦虑 / **autistic behaviors（自闭谱系）** + **tall stature（高大）** + 关节松弛 + MVP | 男孩 + 高 + 大耳 + 长脸 + 自闭 + 大睾 → **FMR1 DNA 分析**（CGG 重复 PCR） |
| **Fetal alcohol syndrome (FAS, 胎儿酒精综合征)** | **致畸**（teratogenic）—— **母亲孕期饮酒**（无安全剂量；first trimester 暴露面容受累最严重）| **smooth philtrum（人中平坦）** + **thin vermillion border（薄上唇红）** + **small palpebral fissures（小眼裂）**（**FAS 面容三联**）+ 小头 + 鼻短上翻 | **seizures（癫痫）** + **growth restriction（生长受限，IUGR + 出生后矮小）** + 先心 (VSD/ASD) + 行为问题 / 学习障碍 + 视听异常 | 母亲饮酒史 + 面容三联 + 生长受限 → **临床诊断**（无基因检测；问"诊断"选 maternal alcohol exposure history） |

> [!warning] 三大病因机制对比（一句话锁）
> - **Down = 染色体多了一条** → karyotype 看数目
> - **Fragile X = 基因重复变多** → FMR1 DNA 看 CGG 数
> - **FAS = 妈妈喝酒了** → 病史 + 面容三联（**无基因检测**）

### 与"染色体病多系统"全表的接口

| 综合征 | 是否染色体病 | 本衍生其他出现位置 |
|---|---|---|
| **Down** | ✅ T21 | §🔬 三大三体对比 / §🩺 多系统总览（心 / 喂养 / 智力 / 肿瘤 / 实验室） |
| **Fragile X** | ❌ 单基因 X 连锁（不是染色体病）| §🌱 三核苷酸重复 + 印记 + X 连锁 |
| **FAS** | ❌ 致畸（不是基因病）| **本节首次纳入** —— 未来积累 FAS 错题可扩展 [[完整笔记/专题笔记/_衍生_孕期致畸药物速查]] |

### USMLE 钓饵：ID 但不是这 3 个

> [!danger] 干扰项警戒
> ID stem 出现时，要警惕这些不是 Top 3 但常做钓饵的病：

| 综合征 | 关键区分（与 Down / Fragile X / FAS 的差异）|
|---|---|
| **Rett (MECP2)** | **几乎只女孩** + 正常发育后**退化** + 手部刻板动作 + acquired microcephaly（≠ 先天 dysmorphism） |
| **Prader-Willi (15q 父源)** | **新生儿严重 hypotonia + 喂养困难**史 → 2-3 岁后 hyperphagia 暴食 + almond eyes + 小手小脚（**面容 ≠ flat nasal bridge**） |
| **Williams (7q11.23 del)** | **精灵脸** + **高钙** + **SVAS** + "cocktail party" 过度友好（≠ DS 的 friendly demeanor — Williams 是病理性外向 + 高钙） |
| **Angelman (15q 母源)** | "Happy puppet" + 频繁大笑 + 共济失调 + 癫痫 + 严重智障 + 小头 |
| **Cri-du-chat (5p-)** | **猫叫样高调哭声** + 小头 + 严重智障 + 圆脸 |
| **Tay-Sachs** | 婴儿期 + cherry-red macula + 神经退化 + **死亡 < 5 岁**（活到学龄期可排除） |
| **PKU** | musty 味 + 抽搐 + 色素减低（现代美国新筛已发现；海外收养 / 出生史不详时考虑） |
| **DiGeorge (22q11.2 del)** | 低钙抽搐 + T 细胞缺陷 → 反复病毒/真菌感染 + 主动脉弓异常（**CATCH-22**） |

## 🔬 三大常染色体三体综合征对比

| 项目 | **Trisomy 21 (Down)** | **Trisomy 18 (Edwards)** | **Trisomy 13 (Patau)** |
|---|---|---|---|
| **发病率** | 1/700（最常见活产染色体病） | 1/6000 | 1/10000 |
| **母亲年龄** | 强相关（>35岁风险↑） | 相关 | 相关 |
| **预后** | 寿命 ~60 岁 | **>50% 1月内死亡，<10% 活过 1 岁** | **80% 1月内死亡** |
| **面容** | 上斜眼、内眦赘皮、扁脸、舌大、低耳 | **小下颌、突出枕骨**、小眼 | **唇腭裂、小眼/无眼、前额倾斜** |
| **手** | **单掌纹 (simian crease)**、第 5 指内弯 | **重叠指**（clenched fist）、掌纹缺失 | **多指 (polydactyly)** |
| **足** | 拇趾分离（sandal gap） | **摇椅足 (rocker-bottom)** | **摇椅足 + 多趾** |
| **心脏** ⭐ | **AVSD** > VSD/ASD | **VSD** > ASD/PDA | **VSD / PDA / ASD（混合）** |
| **神经** | 智力障碍、肌张力低、**早发 Alzheimer (40+)** | 严重发育迟缓、张力异常 | **Holoprosencephaly（前脑无裂）**、严重智障 |
| **消化** | **十二指肠闭锁 (double-bubble sign)**、Hirschsprung | 脐膨出、肠旋转不良 | 脐膨出 |
| **泌尿** | 一般正常 | 马蹄肾 | 多囊肾 |
| **血液/肿瘤** | **ALL / AML 风险↑**、新生儿一过性骨髓增殖 | — | — |
| **内分泌** | **甲减风险↑** | — | — |
| **喂养** | 喂养差（肌张力低 + 舌大 + 心脏疲劳） | **严重喂养困难**（小下颌 + 张力差） | 严重喂养困难（唇腭裂 + 中枢异常） |
| **其他** | 第一/二颈椎不稳（atlantoaxial instability）→ 运动前查 X 线 | 多脏器畸形 | **皮肤缺损 (cutis aplasia)** |
| **产前筛查** | quad screen: ↓AFP, ↑hCG, ↓uE3, ↑inhibin A | quad screen: **全部↓** | 通常 quad 正常，需 NIPT/超声 |

|---|---|---|---|---|

## 🧬 基因 / 染色体检测选择决策树（ID + dysmorphism stem 用）⭐
#难记

> [!tip] 用途
> ID（智力障碍）+ 多发畸形 stem 问"哪个检查最可能确诊？" — 选检测顺序的核心 SOP。
> 触发：Q21629 (Down → karyotype, 2026-05-23) + Q2468 (Edwards → karyotype, 2026-05-18) 反复考"选哪个基因/染色体检测"。

### 三步识别法（防单线索锚定）

```
Step 1：把 stem 里所有 dysmorphic features 列出（脸/头/手/足/皮肤）
Step 2：把所有 comorbid systems 列出（心/听力/眼/内分泌/血液/神经）
Step 3：找哪个综合征命中最多 → 反向验证不命中线索是否仍兼容
        ⚠️ 抓 1-2 个泛特征（如"矮+胖"）就锁定 = 陷阱
```

### 检测选择阶梯

```
ID + 多发畸形 stem
    │
Step 1：怀疑典型染色体综合征（DS / Edwards / Patau / Turner / Klinefelter）
    ▼
  KARYOTYPE 核型分析 ⭐ 一线
  ├─ 数目异常（aneuploidy）：T21 / T18 / T13 / 45,XO / 47,XXY / 47,XYY / 47,XXX
  └─ 大片段重排（易位 / 倒位 / 缺失大片段）
    │ 阴性 ↓
Step 2：怀疑微缺失 / 微重复（小到核型看不到）
    ▼
  FISH（特定探针）/ Chromosomal Microarray (CMA)
  ├─ 22q11.2 del → DiGeorge
  ├─ 7q11.23 del → Williams
  ├─ 5p- → Cri-du-chat
  ├─ 15q11-13 → 进 Step 3a 印记疾病
  └─ CMA = 现代一线（覆盖所有微缺失 + 拷贝数变异 CNV），FISH = 特定怀疑时定向查
    │
Step 3a：印记疾病（PW / Angelman — 15q11-13）
    ▼  Methylation analysis（DNA 甲基化分析）⭐ 印记金标准
       区分父源缺失（PW）vs 母源缺失（Angelman）vs 单亲二倍体（UPD）
    │
Step 3b：单基因病（特定基因 DNA / PCR）
    ▼  按怀疑疾病选基因：
    ├─ FMR1（CGG 重复 PCR）→ Fragile X
    ├─ MECP2 → Rett（女孩专属）
    ├─ FBN1 → Marfan
    ├─ HTT（CAG 重复）→ Huntington
    ├─ DMPK（CTG 重复）→ Myotonic dystrophy
    ├─ FXN（GAA 重复）→ Friedreich ataxia
    └─ NF1 / NF2 / TSC1/2 / VHL → 对应单基因综合征
    │
Step 3c：代谢病（IEM — 智障 + 特殊气味 / 退化 / 抽搐 / 肝大）
    ▼
    ├─ 血 phenylalanine → PKU（musty 味 + 抽搐 + 色素减低）
    ├─ 血 hexosaminidase A → Tay-Sachs（婴儿期 + cherry-red macula + 死亡<5 岁）
    ├─ 尿有机酸 + 血氨基酸 + 血氨 → 有机酸血症 / 氨基酸尿 / 尿素循环障碍
    ├─ 尿 GAGs → 黏多糖病（Hurler / Hunter — 粗面容 + 肝脾大 + 角膜混浊）
    └─ 串联质谱（新生儿筛查）→ 一站式 IEM 筛查
    │
Step 4：MRI head — ❌ 不是 ID + dysmorphism 一线
    ✅ 仅当：focal neuro deficit / 癫痫 / progressive degeneration
    ⚠️ 单纯 ID + 多发畸形 → MRI 多正常或非特异，不诊断
```

### 检测 → 代表疾病速查

| 检测 | 代表疾病 | Stem 关键线索 |
|---|---|---|
| **Karyotype** ⭐ | T21 (Down) / T18 (Edwards) / T13 (Patau) / 45,XO (Turner) / 47,XXY (Klinefelter) | 多发畸形 + 典型面容 + 经典系统组合 |
| **CMA / FISH** | DiGeorge / Williams / Cri-du-chat / Smith-Magenis | 微缺失特征性表型（CATCH-22 / 精灵脸 / 猫叫） |
| **Methylation** | Prader-Willi / Angelman | 新生儿 hypotonia → 暴食肥胖（PW）/ 快乐木偶 + 笑 + 共济失调（Angelman） |
| **FMR1 DNA** | Fragile X | 男 + 高 + 长脸 + 大耳 + 大睾 + 智障 + 自闭 |
| **MECP2** | Rett | **女孩** + 正常发育后退化 + 手部刻板动作 + acquired microcephaly |
| **Hexosaminidase A** | Tay-Sachs | 婴儿期 + cherry-red macula + 神经退化 + 死亡<5 岁 |
| **Serum phenylalanine** | PKU | musty 味 + 抽搐 + 色素减低 + 智障 |
| **MRI head** | Holoprosencephaly / 结构异常 / leukodystrophy | focal deficit / 癫痫 / progressive degeneration（不是 dysmorphism） |

### 易混陷阱

> [!danger] 高频陷阱

- **单线索锚定**：抓 1 个泛特征（"矮+胖" / "智障+学习困难"）就锁综合征 → 选项里没匹配就瞎选。USMLE 故意给 1-2 个泛特征 + 4-5 个高特异特征测**多线索整合**。
- **"矮 + 胖" ≠ Prader-Willi 专属**：DS / Williams / Cushing / 甲减都符合；PW 高特异线索是 almond eyes + 新生儿 hypotonia 史 + 食物执念 + 小手小脚。
- **MECP2 是 Rett（女孩）不是 Down 男孩**：男孩 stem 出现 MECP2 选项 → 警惕钓饵。
- **FMR1 Fragile X = 高 + 大耳 + 大睾**：与 Down 的"矮 + 上斜眼 + 单掌纹"**身高方向相反**。
- **MRI 钓饵**：智障 + 学习困难 → 想查脑结构。但 MRI 适应症 = focal neuro deficit / 癫痫 / progressive degeneration；ID + dysmorphism 单查不诊断。
- **海外收养 / 出生史不详**：USMLE 暗示**没做新生儿筛查 + 无家族史** — 提示要考虑各类基因/染色体/代谢病，但仍按 stem 当前线索综合诊断，不是无脑全套筛。
- **"我想的诊断没选项" SOP**：回 stem 重扫线索，**不是赌剩余选项** — 大概率是诊断错了。

### Memory Hook

> [!success] 三层口诀

- **基因检测三层**：「**核型 → 微阵列 → 单基因 / 甲基化 / 代谢**」
- **Down 五件套（看到 ≥3 即锁 karyotype）**：flat nasal bridge + LLSB holo murmur (VSD/AVSD) + 传导性听力丧失 + friendly demeanor + mild-moderate ID
- **印记金标准**：「**15q + 双相（饿→暴食 or 快乐木偶）→ 甲基化**」
- **女 Rett 男 Fragile**：MECP2 女孩退化 / FMR1 男孩 macroorchidism

## 🎯 USMLE 高频出题模式

> [!danger] 经典 stem 套路

### 套路 1：新生儿多发畸形 → 问诊断 / 心脏畸形
- 重叠指 + 摇椅足 → Edwards → **VSD**
- 唇腭裂 + 多指 → Patau → **混合心脏畸形**
- 单掌纹 + 舌大 + 上斜眼 → Down → **AVSD**

### 套路 2：婴儿期表现 → 问诊断
- **新生儿 floppy + 吃不下** + 后期暴食肥胖 → **Prader-Willi**
- **FTT + 高钙 + 心杂音** → **Williams**
- **猫叫样哭声** → **Cri-du-chat**
- **新生儿手足肿 + 颈蹼** → **Turner**

### 套路 3：青少年/成人表现
- 矮小女孩 + 原发闭经 → **Turner**（查 FSH↑、超声看流出道梗阻）
- 高瘦男 + 男乳 + 小睾 + 不育 → **Klinefelter**
- 中年发病 + 舞蹈 + 痴呆 + 家族史 → **Huntington**
- 男孩 + 长脸 + 大耳 + 智障 + 大睾 → **Fragile X**

### 套路 4：免疫/感染问题
- 新生儿低钙抽搐 + 反复**病毒/真菌**感染 → **DiGeorge**（T 细胞缺陷）

### 套路 5：印记疾病鉴别 ⭐
- **15q 父源缺失** = Prader-Willi（"父亲走了 → 孩子饿"）
- **15q 母源缺失** = Angelman（"妈妈走了 → 孩子笑"）

## 💡 Memory Hooks 汇总

> [!success] 一句话锁定

- **Down → AVSD**："Down's heart cushioned" (endocardial cushion)
- **Edwards → VSD**："Edwards' V" (18 倒过来像 V)
- **Patau → 混合**：13 三体太严重，啥都坏
- **Turner → Bicuspid AV / Coarctation**："Turner 上下不通"（上肢高压下肢弱脉）
- **DiGeorge → Truncus**："22 个真没胸腺"（CATCH-22）
- **Williams → Supravalvular AS**："精灵脸瓣上窄"
- **Williams 三联**："W-AS-Ca-FTT"（瓣上窄 + 高钙 + 长不大）
- **Prader-Willi**："父走子饿 → 后期暴食"（15q 父源缺失）
- **Angelman**："母走子笑"（15q 母源缺失，快乐木偶）
- **Klinefelter**："多 X 男乳大乳癌"（XXY → 男乳 + 乳腺癌风险）
- **Fragile X**："长脸大耳大睾丸"（青春期后 macroorchidism）

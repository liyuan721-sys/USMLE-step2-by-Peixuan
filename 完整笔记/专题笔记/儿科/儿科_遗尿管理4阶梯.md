# 儿科 — Primary Nocturnal Enuresis 管理 4 阶梯

> [!info] 笔记定位
> Primary nocturnal enuresis（PNE，单纯夜间遗尿）= 儿科泌尿高频。本笔记按 USMLE Step 2 高频考点组织：诊断阈值 / 病机 / 评估必查 / 4 阶梯治疗 / Alarm vs DDAVP 选择元规则 / 反 pattern。来源：[[mistakes/uworld-mistakes_2026-05#^Q2513]]（做对题 2026-05-28，体系早建 v1）。

---

## Part 1. 定义 + 诊断阈值

| 项 | 内容 |
|---|---|
| **年龄阈值** | **≥ 5 yo**（< 5 yo 还在 bladder control 成熟期，**不诊断**）|
| **频次阈值** | 夜间尿失禁 ≥ 2 次 / 周 × ≥ 3 月（DSM-5）|
| **类型** | **Primary**（从未持续 6 月以上夜间 dry）vs **Secondary**（先 dry 后又遗尿，提示新发心理 / DM / UTI / 神经 / 虐待）|
| **范围** | **Monosymptomatic**（仅夜间）vs **Non-monosymptomatic**（日 + 夜，提示 bladder dysfunction）|
| **流行病学** | 男 > 女；7yo ~10% / 10yo ~5% / 成人 ~1%；**自限**（每年 ~15% 自愈）|

---

## Part 2. 病机三因

```
Primary Nocturnal Enuresis 病机
        │
        ├─ ① Bladder control 延迟成熟
        │      (大脑觉醒阈值高 — 膀胱满了叫不醒)
        │
        ├─ ② 夜间尿生成↑
        │      (晚 ADH 分泌↓ → 夜尿浓缩不足 → 量↑)
        │
        └─ ③ 膀胱容量↓
               (功能性 — 不是结构性)
```

**家族史强阳**：
- 父母**任一**遗尿史 → 子 ~**45%**
- 父母**双方**都遗尿史 → 子 ~**75%**
- 本质 = **多基因遗传**（不是单基因 Mendelian）

---

## Part 3. 评估必查（先排基础病）

| 检查 | 排除什么 | 阳性意义 |
|---|---|---|
| **Urinalysis** ⭐ 必查 | DM (glucosuria) / DI (USG↓ Uosm↓) / UTI (LE/nitrite+) / 肾病（蛋白尿 / 血尿）| 异常 → 转继发病因评估 |
| **Voiding diary** ⭐ 必查 | 真实尿量 vs 频次 / 区分单纯夜间 vs 日夜兼有 | 日间也漏 → 加抗胆碱评估 bladder dysfunction |
| 详细病史 | Constipation（直肠压迫膀胱）/ OSA / 心理压力（学校 / 家庭）| 排加重因素 |
| **如有日间 incontinence** | bladder dysfunction → 加 oxybutynin | — |
| **如有反复 UTI** | Vesicoureteral reflux (VUR) → **VCUG** | — |

> [!warning] 反 pattern 警示
> **UA 已正常 → 不再查 serum glucose / urine culture**（UA 已涵盖 DM/UTI 筛查）。USMLE 经典陷阱：UA 阴性后再选 "serum glucose / urine culture / VCUG" 都是错的（早已排除）。

---

## Part 4. 治疗 4 阶梯 ⭐⭐⭐

```
Step 1: Behavioral modifications（行为干预）
        ├─ 限晚间饮水（睡前 2hr 内不喝）
        ├─ 睡前排尿
        ├─ 治疗加剧因素（constipation 便秘 / 暴饮咖啡因）
        └─ "Reward chart" 鼓励（gold star 法）
                 │ 行为效果不足
                 ▼
Step 2: Enuresis alarm（湿尿报警器）⭐ 长期最有效
        ├─ 治愈率 ~70%
        ├─ 起效慢（3-4 月）
        ├─ 复发率最低 ⭐
        └─ 缺点：全家睡眠中断 / 需家长高度配合
                 │ alarm 失败 / 想立即改善（夏令营 sleepover）
                 ▼
Step 3: Desmopressin (DDAVP, PO)⭐ 药物一线
        ├─ 起效快（数日）
        ├─ 适合短期需求（夏令营 / sleepover）
        ├─ 机制：↓ 夜尿生成（V2 受体激动）
        ├─ 注意：限晚饮水防低钠 / 头痛
        └─ 缺点：停药复发率高
                 │ 长期 DDAVP 失败 / 难治
                 ▼
Step 4: Imipramine (TCA) 后备
        ├─ 等效 DDAVP
        ├─ 但 cardiotoxic + 过量致命 + 自杀风险
        └─ 仅 DDAVP 失败 + ECG 安全后才用
```

### 4.1 治疗对照速查表

| 治疗 | 起效 | 治愈率 | 复发率 | 适用场景 | 副作用 |
|---|---|---|---|---|---|
| **Behavioral** | 数周 | 低 | — | 一线尝试 | 无 |
| **Enuresis alarm** ⭐ | **3-4 月** | **~70%** | **最低** | 长期治愈 + 家长耐心 | 全家睡眠中断 |
| **Desmopressin** ⭐ | 数日 | 中-高 | **高** | 短期需求 / alarm 失败 | 低钠 / 头痛 |
| **Imipramine** | 数日 | 中-高 | 高 | DDAVP 失败后备 | cardiotoxic / 自杀 / 致命过量 |
| **Oxybutynin** | 数周 | — | — | **仅日夜兼有 incontinence**（增 bladder capacity）| 抗胆碱（口干 / 便秘）|

### 4.2 "长期 vs 即刻" 元选择规则 ⭐

| 场景 | 首选 |
|---|---|
| 想**长期治愈** + 家长耐心 | **Enuresis alarm** |
| 想**立即改善 / 短期需求**（夏令营 sleepover）| **Desmopressin** |
| **Alarm 已失败** 3-4 月 | **Desmopressin** |
| **DDAVP 失败** | **Imipramine**（ECG 安全前提）|

> [!tip] 选择心算：长期 → Alarm；快 / 应急 → DDAVP；失败后备 → Imipramine

### 4.3 共病便秘优先治 + encopresis 识别 ⭐⭐（Q20248）

> [!danger] 遗尿合并便秘 → 先治便秘，常一并解决
> 处理遗尿的**第一步是治疗可纠正共病**，**便秘居首**。直肠粪潴留 → 压迫膀胱 → **膀胱功能容量↓ + 逼尿肌不稳** → 诱发/加重夜间遗尿。**先上泻药（PEG / polyethylene glycol）** 解决便秘后，遗尿常随之缓解 —— 此时**不先上 alarm / DDAVP，也不做影像**。

> [!warning] Encopresis = 便秘的标志（别当腹泻）
> **Encopresis（大便失禁）= 稀便绕过嵌塞粪块的溢出性失禁（overflow incontinence）**，本质是**便秘**不是腹泻。
> - 体征钩子：**左下腹可触及粪块 / 左侧腹较硬** + 内裤稀便漏。
> - 见"遗尿 + 稀便漏 + 左下腹粪块" → 锁便秘 → **泻药先行**。
> - **反陷阱**：此时选肾超声 / 腰骶 MRI 都错（无白天症状 / 无神经体征）；**白天限水更错**（加重便秘 + 脱水）。

## Part 5. 易混陷阱 / 反 pattern ⭐

| 选项 | 适用场景 | 不适用本题 |
|---|---|---|
| **Oxybutynin (抗胆碱)** | **日 + 夜 incontinence** — 增 bladder capacity | 单纯夜间遗尿 ❌ |
| **VCUG** | **反复 UTI** → 诊断 VUR | 无 UTI 史 ❌ |
| **Urine culture** | UA 异常（LE/nitrite+）/ 有 UTI 征 | UA 正常 ❌ |
| **Serum glucose** | UA 异常（glucosuria）/ 怀疑 DM | UA 正常 ❌ |
| **Imipramine** | DDAVP 失败后备 | 一线 ❌（毒性高）|

### 反 pattern 警示
- **"Alarm 4 月无效"≠ Alarm 失败** — Alarm 需 3-4 月起效，4 月是分界。**只有 stem 明确"frustrated / 想即刻改善 / 短期需求"才升级 DDAVP**
- **Imipramine ≠ first-line**（旧教材偶列"二选一"，现代答案 = DDAVP > Imipramine 因安全性差异）
- **Oxybutynin 不是 isolated nocturnal enuresis 选项**（仅日夜兼有 incontinence）
- **UA 已查清 = 不再查 glucose / culture / VCUG**（早已涵盖）
- **诊断年龄 ≥ 5 yo** — < 5 yo "遗尿"是发育正常，不诊断 / 不治疗（家长安慰即可）

---

## Part 6. Memory Hook

> **"7yo + 仅夜间 + alarm 4 月失败 + 即刻需求 = Desmopressin"** ⭐
>
> **治疗 4 阶梯**：Behavioral → Alarm → DDAVP → Imipramine ⭐⭐⭐
>
> **"Alarm 最有效但慢；Desmopressin 最快但复发；Imipramine 等效但毒"**
>
> **Alarm vs DDAVP 选**：长期治愈选 alarm；即刻改善选 DDAVP
>
> **诊断年龄阈值**：**≥ 5 yo**
>
> **UA 必查**：DM / DI / UTI / 肾病 4 排除
>
> **反 pattern**：UA 已查 → 不再 glucose / culture / VCUG

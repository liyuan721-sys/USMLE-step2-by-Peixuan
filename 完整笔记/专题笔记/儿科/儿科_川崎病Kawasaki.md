# 川崎病 Kawasaki Disease (KD)

> [!info] 一句话定位
> **Kawasaki disease (KD, 川崎病) = 幼儿急性中血管炎 (medium-vessel vasculitis)**，最怕**冠状动脉瘤 (coronary artery aneurysm, CAA) → 血栓 / 心梗**。诊断靠"**发热 ≥5 天 + CRASH 凑 ≥4 条**"，不是单凭某一条体征。好发 <5 岁，**亚裔高发**（但各种族都可）。
![[{BE953C28-EDFB-4088-81DF-691891DCC3CA}.png]]
---

## 一、诊断标准：发热 ≥5 天 + ≥4/5 主征（CRASH and burn）

> [!warning] KD 是"凑条件"诊断，不能凭单条反射
> 必须 **发热 ≥5 天** + 下列 5 条主征中 **≥4 条**。光"发热 + 手脚肿"只占 1 条，够不到门槛。

```
C - Conjunctivitis   双侧非渗出性结膜炎（角膜缘 limbus 豁免）
R - Rash             多形性皮疹（非水疱）
A - Adenopathy       颈淋巴结肿大（常单侧，≥1.5 cm）
S - Strawberry tongue 草莓舌 + 口唇皲裂红肿（口腔黏膜改变）
H - Hands/feet       急性期手足红斑 + 硬性水肿 → 亚急性期甲周脱皮 desquamation
   + burn = 持续高热 ≥5 天（核心门槛）
```

> [!warning] R - Rash 皮疹特点（与 H 手足改变是两条独立标准）
> - **多形性皮疹 (polymorphous exanthem)**：无固定形态——斑丘疹 / 麻疹样 morbilliform / 猩红热样 scarlatiniform 均可；弥漫红斑、通常不痒、发热头几天出现。
> - **无水疱、无大疱、无瘀点 (no vesicles / bullae / petechiae)** → 出现这些基本可**排除 KD**（想 SJS (Stevens-Johnson syndrome, 史蒂文斯-约翰逊综合征) / SSSS (staphylococcal scalded skin syndrome, 葡萄球菌烫伤样皮肤综合征) / 病毒疹 / 脑膜炎球菌血症 meningococcemia）。
> - **会阴/腹股沟区红斑 + 早期脱皮 (perineal accentuation + early desquamation)**：尿布区急性期就脱皮，是**特异早期线索**（早于亚急性期的甲周脱皮）。
> - **R（躯干四肢多形疹）≠ H（掌跖红斑+手足硬肿→甲周脱皮）**：CRASH 里分开计两条，别合并。

> [!tip] 与"发热出疹病"的关系
> KD 属热中出疹谱的一员，鉴别钥匙见 [[完整笔记/专题笔记/儿科/儿科_发热出疹病鉴别]]（"时间 + 招牌"两钥匙）。KD 是**非感染性血管炎**，无疫苗预防。

## 二、手足改变 vs SCD dactylitis（高频误判点）

> [!danger] "1–2 岁 + 双手双脚肿 + 发热"先别跳川崎
> 一个**镰状细胞病 (sickle cell disease, SCD) 幼儿**也会"双手双脚对称肿痛 + 发热"，且 SCD 患儿发热是**急症**（功能性无脾 → 败血症 / 骨髓炎）。两者手足改变本质不同：

| 维度 | KD 手足改变 | SCD dactylitis（手足综合征 hand-foot syndrome）|
|---|---|---|
| **机制** | 血管炎性**红斑 + 硬性水肿** induration | 血管闭塞性**骨梗死**（vaso-occlusive，疼）|
| **部位** | 掌跖红斑 + 手足背硬肿 → 后期甲周脱皮 | 手指/脚趾**弥漫肿胀剧痛**（dactylitis 香肠指）|
| **配套** | 必须凑 CRASH 其余条（结膜/疹/淋巴结/草莓舌）| 慢性溶血性贫血 + 镰状红细胞涂片 |
| **种族钩子** | 亚裔高发 | 非裔高发（**钩子相反**）|
| **首要处理** | IVIG + 阿司匹林 | 血培养 + 经验性抗生素 + 补液镇痛（先当败血症/骨髓炎）|

> [!tip] 记忆锁定
> **KD = "5 天烧 + CRASH 凑 4 条"，缺结膜/疹/淋巴结/草莓舌就不是；发热的镰状幼儿先当败血症/骨髓炎抢救，别被"发热 + 手脚肿"诱去川崎。** 两病种族钩子相反（亚裔→KD，非裔→SCD）。dactylitis 完整鉴别谱见 [[完整笔记/专题笔记/风湿/风湿_脊柱关节病SpA对照]] §5.1。

## 三、为什么血小板增多（thrombocytosis）

> [!info] 本质 = 炎症驱动的反应性血小板增多 (reactive thrombocytosis)
> 血小板在 KD 里和 ESR (erythrocyte sedimentation rate, 红细胞沉降率) / CRP (C-reactive protein, C 反应蛋白) 一样是**急性期反应物 (acute-phase reactant)**，不是克隆性增生。

```
中血管炎 → 炎症因子大量释放（IL-6 interleukin-6, 白介素-6 主导）
        │
        ├─→ IL-6 刺激肝脏合成 thrombopoietin (TPO, 血小板生成素)
        │
        └─→ TPO + IL-6 促进骨髓 megakaryopoiesis（巨核细胞生成）
                        ↓
                血小板产量↑ → reactive thrombocytosis
```

> [!warning] 时相陷阱：血小板增多是"亚急性期"现象
> - **急性期（第 1 周，高热期）**：血小板**正常甚至偏低**。
> - **亚急性期（第 2–3 周，约 day 10 起）**：血小板**飙升**（常 >450k，可 >1,000k），正好撞上**冠脉瘤 + 血栓风险最高窗**。
> 急性高热期拿血小板增多去支持诊断反而不典型，别记反。

> [!info] 完整实验室时间轴（支持诊断，非诊断标准）
> - **急性期（第 1 周）**：ESR↑、CRP↑、**正细胞性贫血**、**无菌性脓尿 (sterile pyuria)**（尿白细胞↑但培养阴性）、转氨酶↑、白蛋白↓。
> - **亚急性期（第 2–3 周）**：**血小板↑**（如上）、ESR 仍高。
> 这些都是炎症急性期反应物，**不进 CRASH 主征**，但帮你在"够不够 4 条"摇摆时支持川崎（尤其不完全川崎，见 §五）。

## 四、心脏受累全谱 + 治疗逻辑

> [!danger] KD 心脏并发有"两大桶"——别只记冠脉瘤
> 1. **急性期：淋巴细胞性心肌炎 (lymphocytic myocarditis) → 心室功能↓ → 充血性心衰 (congestive heart failure, CHF)**（见 4.1）
> 2. **远期：冠状动脉瘤 (coronary artery aneurysm, CAA) → 血栓 / 心梗 (MI, myocardial infarction)**（见 4.2）
>
> **考点钥匙：川崎病婴儿出现急性心衰 = 心肌炎所致，不是冠脉瘤。** 急性心衰看心肌炎，冠脉瘤是后患（远期 MI 风险）。

### 4.1 急性：淋巴细胞性心肌炎 → 心衰（来源 [[mistakes/uworld-mistakes_2026-06#^Q15468]]）

> [!warning] 心肌炎是 KD 急性期最常见的心脏受累
> KD 系统性血管炎/炎症**累及心肌 → 淋巴细胞性心肌炎 → 心室收缩功能下降 → CHF**，急性期即可出现（实际比冠脉瘤更早、更普遍）。**婴儿尤甚**，表现为：
> - **喂养差 / 喂奶出汗 (feeding difficulty / diaphoresis with feeds)**、易激惹、心动过速 (tachycardia) + 呼吸急促 (tachypnea)
> - **S3 奔马律 (S3 gallop)** + 双肺细湿啰音 (crackles) → 肺水肿 (pulmonary edema)
> - **肝大 (hepatomegaly) + 下肢水肿 (edema)** → 右心/全心衰体征
> - **IVIG 治疗后心功能多改善**（再次说明早治疗的重要性）

> [!info] 易混鉴别（婴儿心肌炎/心衰的病因）
> - **嗜酸性心肌炎 (eosinophilic myocarditis)** → 药物过敏，皮疹呈**湿疹/荨麻疹样、非脱屑**（KD 是甲周脱皮）
> - **急性风湿热 (acute rheumatic fever, ARF)** → **<3 岁罕见**，需二尖瓣反流杂音 + 其他 Jones 标准，**无结膜炎 / 无脱屑**
> - **细菌性心内膜炎 (bacterial endocarditis)** → 新发杂音，无黏膜皮肤炎（结膜/草莓舌/皮疹）

### 4.2 远期：冠状动脉瘤 → 血栓 / 心梗

> [!danger] 血小板↑ × 冠脉内皮损伤 = 血栓
> KD 损伤冠脉内皮 + 形成动脉瘤（血流淤滞）+ 叠加血小板增多 → **高凝 → 冠脉血栓 / 心梗**（KD 远期致死核心）。所有患儿需 **echocardiography 超声心动图**评估并随访冠脉瘤。

| 治疗 | 中英 | 作用 |
|---|---|---|
| **IVIG** | intravenous immunoglobulin 静脉免疫球蛋白 | 抗炎，**显著降低冠脉瘤发生率**（10 天内用最有效）|
| **Aspirin 阿司匹林** | 高剂量抗炎 → 退热后转低剂量 | 抗血小板，冲着"血小板增多 + 冠脉血栓"去 |

> [!tip] 这里是 aspirin 用于儿童的少数例外
> 儿童病毒感染期一般忌阿司匹林（Reye 综合征 Reye syndrome 风险），**KD 是公认例外**——因为抗血小板获益压过风险。

> [!danger] 阿司匹林剂量分期陷阱（高频干扰项，来源 [[完整笔记/专题笔记/USMLE/USMLE_高频陷阱]]）
> - **急性期 = 高剂量阿司匹林（high-dose，按体重 mg/kg）作 _抗炎_**，不是低剂量。
> - **退热后转低剂量（low-dose，3–5 mg/kg/d）作 _抗血小板_**，维持至冠脉评估正常。
> - **常见错答 = 直接选 81 mg 低剂量**（剂量不够，急性期无抗炎作用）。记：**先高（抗炎）后低（抗板）**。

> [!info] 冠脉瘤随访时机
> 所有患儿 **echocardiography 超声心动图** 基线评估 → 一般 **1–2 周、4–6 周**复查；**有冠脉瘤者**延长抗血小板/抗凝 + 更密集随访（必要时冠脉造影）。

## 五、不完全川崎 + IVIG 无应答（高危易漏）

> [!danger] 不完全川崎 (Incomplete Kawasaki Disease)
> **婴儿 <6 月** + 长时间发热但**主征不够 4 条** → 极易漏诊，而**冠脉瘤风险反而更高**。线索：发热 ≥5 天 + **CRP/ESR 明显↑** + 支持性实验室（贫血/血小板↑/无菌性脓尿/低白蛋白/转氨酶↑）+ 超声见冠脉改变 → **即使凑不够 4 条也按川崎治疗（IVIG + 阿司匹林）**。
> 记忆：**"小婴儿 + 烧不退 + 炎症高 + 冠脉变 = 不完全川崎，别等凑满条数。"**

> [!warning] IVIG 无应答 (IVIG-resistant KD)
> 约 10–20% 首剂 IVIG 后**持续/再发热**（炎症标志不退）→ 冠脉瘤风险升。二线：**重复一剂 IVIG**，或 **糖皮质激素（甲泼尼龙 methylprednisolone）**，或 **infliximab 英夫利昔单抗（TNF-α 抑制剂）**。

# IgA 血管炎 / 过敏性紫癜 完整体系 (Henoch-Schönlein Purpura, HSP)

> [!info] 用途与来源
> 整合分散在 [[完整笔记/专题笔记/风湿/风湿_HighYield速查总览]]（IgA vasculitis 行）与 [[完整笔记/专题笔记/儿科/儿科_HighYield速查总览]]、[[完整笔记/专题笔记/儿科/儿科_发热出疹病鉴别]] 中的 HSP 知识为单一真相源。
> 触发错题：[[mistakes/uworld-mistakes_2026-06#^Q3554]]（并发回肠套叠）/ [[mistakes/uworld-mistakes_2026-06#^Q20498]]（临床诊断、无需额外检查）。

---

## §一 核心概念

> [!note] 一句话
> **过敏性紫癜 / IgA 血管炎 (Henoch-Schönlein purpura, HSP)** = **IgA 介导的小血管炎 (small-vessel vasculitis)**，儿科最常见血管炎，好发**学龄儿 (3–10 岁)**，常**前驱上呼吸道感染 (upper respiratory infection, URI)**。
> 病机：白细胞碎裂性血管炎 (leukocytoclastic vasculitis) + **IgA、C3、纤维素沉积**于小血管/毛细血管后小静脉。

---

## §二 经典四联 (Classic Tetrad)

| 系统 | 表现 | 钥匙 |
|---|---|---|
| **皮肤** | **可触性紫癜/瘀点 (palpable purpura)**，对称分布**下肢 + 臀部** | 不褪色 (nonblanching)、**血小板正常**（非血小板减少性紫癜） |
| **关节** | 关节痛/关节炎，膝踝多见 | **可致拒走/拒动**，自限**不破坏关节** |
| **胃肠** | 绞痛性腹痛、恶心呕吐、消化道出血 | 警惕**肠套叠**（见 §四） |
| **肾** | 血尿 ± 蛋白尿（同 IgA 肾病机制） | 可**迟发数月**，需系列尿检随访 |

> [!tip] 其他：阴囊痛/肿（男孩），需与睾丸扭转鉴别。

## §三 诊断 = 临床（关键考点）⭐

> [!danger] 典型患儿无需额外检查（[[mistakes/uworld-mistakes_2026-06#^Q20498]]）
> **诊断标准**：**可触性紫癜/瘀点（无血小板减少、无凝血障碍）+ 以下 ≥2 项**：
> - 关节炎/关节痛
> - 绞痛性腹痛
> - 肾病（血尿 ± 蛋白尿）
> - （组织学：IgA 沉积）
>
> → **典型患儿 = 纯临床诊断，不做额外化验/影像**。血常规凝血正常即排除**免疫性血小板减少 (immune thrombocytopenia, ITP)** 等。

| 检查 | 何时用 |
|---|---|
| **皮肤活检** | 仅**不典型 / 成人**（示 IgA、C3、纤维素沉积于毛细血管后小静脉） |
| **肾活检** | 仅**分型 / 重症肾受累**，**非诊断必需** |
| **系列尿常规** | **所有患儿随访**，捕捉迟发肾病 |
| 肾超声 / 抗 DNA / 超声心动 | ❌ HSP 不常规（属其他病 workup） |

> [!danger] 反复错点：典型 HSP 别手痒加检查（[[mistakes/uworld-mistakes_2026-07#^Q20498-R0719]] 第 2 次错·又选肾超声）⭐
> 两次都错在"看到紫癜 + 腹痛 + 会伤肾就想加检查"。破解关键：**一串"正常值"不是背景板，而是逐一关掉需要干预的鉴别诊断——化验全正常本身就是"无需额外检查"的证据**。
>
> | stem 给的"正常" | 主动排除了 |
> |---|---|
> | CBC（全血细胞计数）正常 → 血小板正常 | 免疫性血小板减少 (ITP)、白血病 |
> | 凝血研究正常 | 凝血病性紫癜 / 弥散性血管内凝血 (DIC) |
> | ESR（红细胞沉降率）正常 | SLE（系统性红斑狼疮）、感染、风湿热 |
> | 尿常规正常 | 此刻无肾受累 → 更无理由查肾 |
> | 体温 / 血压 / 脉搏正常、心肺不显著 | 脓毒症、脓毒性关节炎、风湿热心炎 |
>
> **两个诱饵怎么拆**：
> - "HSP 伤肾" → 想查肾：肾监测靠**系列尿常规**不是肾超声；且尿检正常、此刻无肾受累。肾超声查**结构**（梗阻/囊肿），看不出肾小球炎症。
> - "腹痛 + 紫癜" → 想查腹：警惕肠套叠需**剧烈阵发绞痛 + 呕吐 + 果酱样血便 + 腊肠样包块**（本题仅"轻压痛"不满足，见 §四）；真要查也是**腹部超声**，不是肾超声。
>
> ✅ 正确动作：**什么都不加 → 临床诊断 + 系列尿检随访**。

## §四 最重要并发症：回-回肠套叠 ⭐

> [!warning] HSP 腹痛 ≠ 普通腹痛（[[mistakes/uworld-mistakes_2026-06#^Q3554]]）
> - 机制：**肠壁水肿 + 黏膜下出血 → 充当牵引点 (lead point) → 套叠 (intussusception)**。
> - 部位：**HSP 套叠是回-回肠 (ileoileal)、局限小肠**；与多数小儿特发性**回-结肠 (ileocolic)** 套叠不同。
> - 后果：**空气/造影灌肠常无效 → 多需手术复位**。
> - 典型：阵发剧烈腹痛 + **果酱样血便 (currant jelly stool)**。

```
HSP 腹痛
  ├─ 轻 → 支持治疗（补液 ± 激素缓解腹痛）
  └─ 剧烈阵发 + 血便 → 警惕回-回肠套叠 → 影像(超声) → 灌肠常无效 → 手术
```

---

## §五 治疗

| 程度 | 处理 |
|---|---|
| 多数（轻） | **支持治疗**：补液 + 非甾体抗炎药 (NSAID) 止关节痛（肾受累慎用） |
| 重症（剧烈腹痛 / 重肾 / 阴囊） | **系统性糖皮质激素 (systemic glucocorticoids)** 住院 |

> [!success] 预后
> 多数自限、预后好；**长期风险主要在肾**（少数进展慢性肾病）→ 故强调**系列尿检随访**。

## §六 易混鉴别

| 病 | 与 HSP 的分水岭 |
|---|---|
| **溶血尿毒综合征 (hemolytic uremic syndrome, HUS)** | **前驱腹泻**（产志贺毒素大肠杆菌）+ 溶血性贫血 + 血小板↓ + 急性肾损伤 (AKI)；**无可触性紫癜** |
| **免疫性血小板减少 (ITP)** | 紫癜但**血小板低**（HSP 血小板正常） |
| **脑膜炎球菌血症** | 紫癜 + 病重/休克/发热（HSP 患儿一般状态尚可） |
| **急性风湿热** | 游走性关节 + 心炎 + 环形红斑（非可触性紫癜） |
| **反应性关节炎** | 肠道/泌尿感染后，无可触性紫癜 |

## §七 "紫癜"大家族定位：HSP 属哪一支 ⭐

> [!info] 为什么需要这一节
> 「紫癜 (purpura)」只是"皮下出血、压不褪色 (nonblanching)"的统称，背后机制分三支。HSP 最常被和「**血小板减少性紫癜 (thrombocytopenic purpura)**」混为一谈，但两者**分属不同支**。面对任何紫癜，第一刀永远是：**① 数血小板 ② 摸紫癜可不可触**。

```
紫癜 Purpura（皮下出血、压不褪色 nonblanching）
├─ 血小板性（血小板↓ 或功能差）── 紫癜「平坦不可触」+ 瘀点 petechiae + 黏膜出血
│     ITP 免疫性血小板减少 / TTP 血栓性 / DIC 弥散性血管内凝血 / 白血病
├─ 血管性（小血管壁炎症/受损）──── 紫癜「可触隆起 palpable」+ 血小板正常
│     ★ IgA 血管炎 (HSP) ★ / 其他小血管炎 / 老年性·激素性紫癜
└─ 凝血性（凝血因子缺乏）────────── 大瘀斑 / 深部·关节腔出血（血友病 hemophilia 等）
```

> [!danger] HSP 的定位 + 与血小板减少性紫癜的分水岭
> HSP 属 **血管性紫癜**（小血管炎）→ 故 **血小板正常 + 紫癜可触**；而 **血小板减少性紫癜（ITP/TTP）= 血小板↓ + 紫癜平坦不可触**。这是两者的根本区别（同名"紫癜"≠同类病）。
> 钥匙：**「可触性紫癜 (palpable purpura) + 血小板正常 = 血管炎 (HSP)，不是血小板病。」**

| 维度 | **IgA 血管炎 (HSP)** | **血小板减少性紫癜 (ITP 免疫性 / TTP 血栓性)** |
|---|---|---|
| 机制 | 小血管 IgA 免疫复合物 → 白细胞碎裂性血管炎 | 血小板**数量↓**（ITP 免疫破坏 / TTP 微血栓消耗） |
| **血小板** | **正常** ⭐ | **↓↓** ⭐ |
| 紫癜性质 | **可触隆起**、下肢 + 臀对称 | **平坦不可触** + 瘀点 + 黏膜出血 |
| 伴随 | 关节痛 + 腹痛 + IgA 肾病（血尿） | ITP=孤立血小板↓；TTP=五联（MAHA 微血管病性溶血 + 血小板↓ + 肾 + 神经 + 发热） |
| 处理 | 支持 ± 糖皮质激素 | ITP 观察/激素/IVIG（intravenous immunoglobulin 静脉免疫球蛋白）；**TTP 血浆置换 (plasma exchange)**（急症，禁血小板输注） |

> [!tip] stem 反射
> 儿童**可触性紫癜 + 关节痛 + 腹痛 + 血尿、血小板正常** → **HSP**；若**紫癜平坦 + 血小板低** → 转去想 **ITP / TTP / HUS / 白血病 / 脑膜炎球菌血症 (DIC)**。

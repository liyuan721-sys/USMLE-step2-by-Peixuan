# 囊性纤维化 CF 完整多系统 (Cystic Fibrosis)

> [!info] 一句话定位
> **CF (cystic fibrosis, 囊性纤维化) = CFTR (cystic fibrosis transmembrane conductance regulator, 囊性纤维化跨膜传导调节蛋白) 基因缺陷的常染色体隐性病**，外分泌腺**分泌物黏稠** → **多器官受累**（肺/胰/肠/肝/生殖）。气道病原谱的纵深见 [[完整笔记/专题笔记/儿科/儿科_囊性纤维化CF气道病原速查]]，本卡管**诊断 + 全系统 + 治疗**。

---

## 一、病理生理 + 基因

```
CFTR 基因突变（最常见 ΔF508 缺失）
        │
   CFTR = 上皮细胞 cAMP 调控的 Cl⁻ 通道功能缺失
        │
   汗腺：Cl⁻ 不能重吸收 → 汗液 Cl⁻ / Na⁺ ↑（汗咸 → 诊断钥匙）
   气道/外分泌腺：分泌物脱水变黏稠 → 黏液纤毛清除障碍 + 导管堵塞
        │
        ↓
   肺反复感染 + 胰/肠/肝/输精管堵塞 → 多系统病
```

- **遗传**：常染色体隐性 (autosomal recessive)；**ΔF508 (Phe508del)** 是最常见突变。
- 携带者筛查、产前诊断属 CK 范畴（[[完整笔记/专题笔记/OB_GYN/OB_GYN_HighYield速查总览]] 可关联）。

---

## 二、诊断（新生儿筛查 → 汗氯试验金标准 → 基因）

| 步骤 | 方法 | 钩子 |
|---|---|---|
| **新生儿筛查** | **IRT (immunoreactive trypsinogen, 免疫反应性胰蛋白酶原) 升高** | 胰管堵塞 → 胰蛋白酶原反流入血 |
| **确诊金标准** | **汗氯试验 (sweat chloride test)：Cl⁻ ≥ 60 mEq/L** | 出汗多/咸；60 以上确诊 |
| **基因检测** | CFTR 突变（ΔF508 等） | 汗氯临界 (30–59) 时辅助 |

> [!tip] 诊断钥匙
> **"妈妈说孩子亲起来咸 (salty baby) + 反复肺感染 + 生长迟滞" → 汗氯试验。** 新生儿筛查 IRT 阳 → 汗氯确诊。

## 三、多系统受累地图

```
CF 多系统
├── 呼吸：支扩 bronchiectasis / 慢性铜绿定植 / 鼻窦炎 + 双侧鼻息肉 / 桶状胸 / 杵状指 clubbing
│        （病原年龄谱见气道病原速查卡）
├── 胰腺：外分泌不全 → 脂肪泻 + 生长迟滞 + ADEK 脂溶维生素缺乏
├── 肠道：新生儿胎粪性肠梗阻 meconium ileus / 年长 DIOS 远端肠梗阻综合征
├── 肝胆：胆管堵塞 → 胆汁淤积 → 局灶性胆汁性肝硬化
├── 内分泌：CF 相关糖尿病 CFRD（胰岛纤维化）
└── 生殖：男性 CBAVD 输精管缺如 → 不育（精子生成正常）
```

> [!warning] 各系统高频考点（来源：分科笔记 [[完整笔记/Peixuan分科笔记/儿科]] CF 多系统段）
> - **胰腺外分泌不全 (pancreatic exocrine insufficiency)**：脂肪/蛋白消化障碍 → **脂肪泻 + 生长迟滞 + 脂溶维生素 ADEK 缺乏**（维 K 缺 → 出血/凝血障碍；维 E 缺 → 神经/溶血；维 A → 夜盲；维 D → 骨）。
> - **胎粪性肠梗阻 (meconium ileus)**：新生儿不排胎便 + 腹胀呕吐 → **高度提示 CF**（是 CF 的新生儿首发表现之一）。年长版本 = **DIOS (distal intestinal obstruction syndrome, 远端肠梗阻综合征)**，黏稠粪便堵回盲部。
> - **CF 相关肝病**：胆管被黏稠分泌物堵 → 胆汁淤积 → **局灶性胆汁性肝硬化 (focal biliary cirrhosis)**。
> - **CFRD (cystic fibrosis-related diabetes, CF 相关糖尿病)**：胰岛因纤维化受损，**既非典型 1 型也非 2 型**（胰岛素分泌不足为主、通常无酮症、无典型自身免疫），青少年期起病，需胰岛素。
> - **CBAVD (congenital bilateral absence of vas deferens, 先天性双侧输精管缺如)**：**男性不育**主因；睾丸生精正常 → 可经睾丸取精辅助生育。

## 四、治疗（气道清理 + 抗感染 + 营养 + CFTR 调节剂）

> [!success] 四条腿
> 1. **气道清理 (airway clearance)**：胸部物理治疗 / 振动背心 + **dornase alfa 多酶α (重组 DNase, 降解中性粒 DNA 稀释痰)** + **吸入高渗盐水 (hypertonic saline)**。
> 2. **抗感染**：急性加重覆铜绿（抗假单胞 β-内酰胺 + 妥布霉素 tobramycin）；慢性定植吸入妥布霉素 ± 阿奇霉素 azithromycin —— 细节见气道病原速查卡。
> 3. **营养/消化**：**胰酶替代 (pancreatic enzyme replacement, pancrelipase 胰酶)** 随餐 + **补 ADEK 脂溶维生素** + 高热量饮食；CF 肝病可用 **熊去氧胆酸 ursodeoxycholic acid**。
> 4. **CFTR 调节剂 (CFTR modulators)**：按突变型选 —— **ivacaftor 伊伐卡托 (增强子 potentiator)**、**lumacaftor / tezacaftor / elexacaftor (校正子 correctors)**；三联 **elexacaftor-tezacaftor-ivacaftor** 大幅改善 ΔF508。是精准医学典范。

> [!danger] 最常见死因 = 慢性肺感染 + 呼吸衰竭
> 终末期可肺移植；**Burkholderia cepacia 洋葱伯克霍尔德菌定植 = 移植预后差/相对禁忌**（见气道病原速查卡）。

> [!question]- 自测：6 月龄婴儿反复肺炎 + 脂肪泻 + 生长曲线下滑，母亲诉亲吻时皮肤味咸。最佳确诊检查？该病男性成年后最可能的生殖问题及机制？
> **确诊 = 汗氯试验 (sweat chloride test, Cl⁻ ≥60 mEq/L)**（新生儿筛查 IRT 阳可先行提示）。生殖问题 = **CBAVD 先天性双侧输精管缺如 → 梗阻性不育**，但**生精功能正常**，可经睾丸取精行辅助生殖。

> [!question]- 自测：14 岁 CF 患者新出现多饮多尿 + 体重下降，随机血糖 220，无酮症，自身抗体阴性。最可能诊断 + 与 1 型/2 型糖尿病的关键区别 + 一线治疗？
> **CFRD (CF 相关糖尿病)**：胰岛因纤维化致**胰岛素分泌不足**，**不同于 1 型（无典型自身免疫/酮症）也不同于 2 型（非胰岛素抵抗为主）**；一线 **胰岛素**。

# A. 心音预测三段映射（推理总框架）

> [!tip] 见任何瓣膜病听诊题，先走这三步
>
> **Step 1 — 病变本质**：压力超负荷？容量超负荷？瓣叶钙化僵硬？
>
> **Step 2 — 心室适应**：向心性肥厚（壁厚腔小）or 离心性肥厚（腔大壁薄）？
>
> **Step 3 — 心音映射**：
> - 压力超负荷 / 向心性肥厚 → **S4**（AS / HTN / HCM）
> - 容量超负荷 / 离心性肥厚 → **S3**（MR / AR / DCM / HF / 妊娠 / 甲亢）
> - 瓣叶钙化动不了 → 关闭音**变软**（AS → soft A2；MS 晚期 → soft S1）
> - 瓣叶能动 + 关闭压差大 → 关闭音**增强**（MS 早期 → loud S1；HTN → loud A2；PAH → loud P2）

**为什么这样**：心音不是孤立现象 — 它是「瓣叶能不能动 × 关闭瞬间的压差 × 心室适应模式」三者的乘积。Step 1 决定 Step 2，Step 2 决定 Step 3。**跳过中间任何一步，就会得到错误的心音预测**（例如把"AS 压力超负荷"直接推到"S3"，就是 Q4277 第一次错的根本）。

# B. 三大底层规律

## B.1 杂音响度 ∝ √(跨瓣压差)，不是 ∝ LV 绝对压力

让血流加速产生湍流的是 **LV 与主动脉之间的压力差**，不是 LV 的绝对压力。

- Severe AS 时 LV 需要"攒"更长时间才能建立足够跨瓣压差 → peak 推迟到收缩中-晚期 = **late-peaking**
- Mild-moderate AS 跨瓣压差易达峰 → peak 在收缩早期 = **early-peaking**

> [!success] 推论
> **早 peak = 轻；晚 peak = 重**。这是从机制反推的硬规则，不需要死记。

## B.2 心音响度 = 瓣叶弹性 × 关闭瞬间的驱动压差

心音的响度由两个独立因素决定：

1. **瓣叶弹性**：瓣叶钙化僵硬 → 关闭无震动 → 心音变软（与压差无关也会变软）
2. **关闭瞬间的驱动压差**：心室顺应性差 → 舒张期心室压力↑ → 关闭瞬间跨瓣压差↓ → 心音进一步变软

> [!success] 推论
> - **Severe AS → soft A2**（瓣叶钙化 + LV 顺应性差）
> - **MS 晚期 → soft S1**（瓣叶钙化失去弹性）
> - **MS 早期 → loud S1**（瓣叶能动，被高房压顶得远，关得猛）

## B.3 病越重，听诊反而越"安静"（USMLE 反直觉考点 ⚠️）

这是 USMLE 高频陷阱：直觉以为"病重 = 体征显著"，但晚期/严重瓣膜病听诊反而**减弱**。

| 病种 | 早/中期表现 | 晚期/严重表现 |
|---|---|---|
| **MS** | Loud S1 + 清脆 OS + 短 rumble | Soft S1 + OS 消失 + 长 rumble |
| **Severe AS 失代偿** | Late-peaking 响 murmur | 杂音可变轻（每搏量↓ → 跨瓣血流↓）|
| **急性 severe MR** | 经典全收缩期吹风样 | 杂音可短暂或微弱（LA 顺应性差，压力快速均衡）|

> [!warning] 临床应用
> 题干给"老年 + 已知 MS + 现在听不到 OS" — **不是好转**，是**晚期瓣叶钙化**。同理"已知 AS + 杂音减轻 + 症状加重" — **不是缓解**，是**失代偿**。

# C. Severe AS 体征四件套 + S4

> [!danger] 必须全部识别（Q4277 两次错的核心）
>
> | 体征 | 机制 |
> |---|---|
> | **Pulsus parvus et tardus**（颈动脉小迟脉）| 每搏量↓ + 射血时间↑ |
> | **Narrow pulse pressure**（如 PP < 25 / 108-92=16）| 每搏量↓ |
> | **Late-peaking** crescendo-decrescendo systolic murmur | LV 需更长时间建立跨瓣压差 |
> | **Soft and single S2**（吸气时不分裂）| A2 延迟 + 减弱，与 P2 重合 |
> | (+) **S4** | 压力负荷 → 向心性 LVH → 心房用力对抗僵硬 LV |

**严重度判别的最强单一线索**：**Carotid pulse 描述**。看到 stem 说 "slow-rising and delayed carotid pulses" / "parvus et tardus" → 0.5 秒锁定 severe AS，不要等 echo 数据。

**为什么必须摸颈动脉，不摸桡动脉**：peripheral amplification（外周放大效应）会让 parvus et tardus 在桡动脉看起来正常 → 漏诊。USMLE stem 永远说 "carotid pulses are slow-rising and delayed"。

# D. 心音-疾病速查（S1/S2/S3/S4 + 分裂）

| 心音 | 提示疾病 |
|---|---|
| **Loud S1** | **MS 早期** / 短 PR / 高输出（贫血/甲亢/妊娠） |
| **Soft S1** | MS 晚期（瓣钙化）/ MR / 长 PR |
| **Loud A2** | 系统性 HTN |
| **Soft / single S2** | **Severe AS** ⭐ / 严重 PS |
| **Loud P2** | PAH |
| **Paradoxical S2 splitting**（呼气分裂、吸气合并）| **Severe AS** / LBBB / HCM |
| **Wide fixed S2 splitting**（不随呼吸变）| **ASD** |
| **Wide physiologic splitting** | RBBB / PS |
| **S3** | 容量超负荷：DCM / 严重 MR / AR / HF / 妊娠 / 甲亢 / 儿童正常 |
| **S4** | 压力超负荷 / 顺应性↓：**AS** / HCM / HTN / 缺血 |
| **Opening snap (OS)** | MS（A2-OS 间期 < 60 ms 提示 severe MS）|
| **Ejection click** | Bicuspid AV / Pulmonic stenosis |
| **Mid-systolic click** | MVP |

# E. 脉搏类型鉴别（高频）

| 脉搏类型 | 特征 | 病因 |
|---|---|---|
| **Parvus et tardus** | 小 + 迟（小而迟到）| **Severe AS** |
| **Water hammer / Corrigan** | 大振幅 + 迅速塌陷 | **Severe AR** / PDA / 高输出 |
| **Bisferiens**（双峰）| 收缩期 2 个峰 | **AS+AR** 或 **HCM** |
| **Paradoxus**（奇脉）| 吸气时 SBP↓>10 mmHg | **心包填塞** / 重度哮喘 / 张力性气胸 / 限制性心包炎 |
| **Alternans**（交替脉）| 强弱交替 | **HFrEF 失代偿** |
| **Bounding** | 大振幅 | 高心排（贫血 / 甲亢 / AR / PDA / 妊娠 / 发热）|

> [!tip] 拉丁词源记忆
> - **Parvus** ≈ "parvo"（细小病毒）→ **小**
> - **Tardus** ≈ "tardy"（迟到）→ **迟**
> - 合起来：**"又小又迟到"的脉搏** = severe AS

# F. S3 vs S4 对偶（容量 vs 压力）

| 维度 | **S3** | **S4** |
|---|---|---|
| 负荷类型 | **容量超负荷** | **压力超负荷** |
| 心室适应 | 离心性肥厚（腔大壁薄）| 向心性肥厚（壁厚腔小） |
| 病因 | Severe MR/AR / HF / DCM / 妊娠 / 甲亢 | **AS** / HTN / HCM / 缺血 |
| 时相 | 舒张早期（被动充盈快速期末）| 舒张晚期（心房收缩末）|
| 生理性 | 儿童 / 妊娠 / 青年运动员可有 | 任何年龄均**异常** |
| 机制 | 大量血涌入扩大的腔体 → 振动 | 心房用力推血进僵硬的腔体 |

**口诀**：
- **"4 个壁，3 个腔"** — S4 = 壁厚（向心）；S3 = 腔大（离心）
- **"压力 → 壁厚 → S4；容量 → 腔大 → S3"**

> [!warning] AS 不会有 S3 的解剖基础
> AS 是压力超负荷 → 向心性肥厚（壁厚腔**小**）→ **没有"扩大"** → 没有 S3。
> 题干 AS 患者出现 S3 → 想其他诊断（如合并 MR / 失代偿后 LV 扩张 / 心肌病）。

# G. MS 早期 vs 晚期（"病越重越安静"示例）

| 体征 | MS 早期 / 中期 | **MS 晚期 / 严重** |
|---|---|---|
| **S1** | 响（loud）| **软**（瓣叶钙化）|
| **Opening snap (OS)** | 清脆可闻 | **消失**（瓣叶失去弹性）|
| **Diastolic rumble** | 较短 | 较**长**（高跨瓣压差持续整个舒张期）|
| **A2-OS 间期** | > 80 ms | **< 60 ms**（严重度最佳指标）⭐ |
| **P2** | 正常 | **响**（继发 PAH）|

> [!tip] A2-OS 间期是 MS 严重度最佳单项指标
> LA 压力越高 → MV 越早被顶开 → A2-OS 间期越短 → 病越重。

# H. 高频陷阱（普适）

| 陷阱 | 正解 |
|---|---|
| "AS → LV 压力大 → LV 扩大 → S3" ❌ | AS 是**向心性肥厚（壁厚腔小）**，没有 S3 的解剖基础；正确是 **S4** |
| "杂音响度 ∝ LV 压力" → 推早 peak ❌ | 杂音响度 ∝ √跨瓣压差；severe AS → **late-peaking**（早 peak = 轻）|
| "病重体征更显著" 直觉 ❌ | MS 晚期 S1 软 + OS 消失；severe AS 失代偿杂音减轻 |
| 摸桡动脉判断 AS 严重度 ❌ | 外周放大效应会掩盖 parvus et tardus；**必须摸颈动脉** |
| Loud S1 / mid-diastolic rumble 当成 AS ❌ | 这是 **MS** 的体征；AS 不影响 S1 |
| AS + S3 题目误选 ❌ | AS 不产生 S3；如 stem 给 S3 → 考虑合并 MR / 失代偿 / 心肌病 |

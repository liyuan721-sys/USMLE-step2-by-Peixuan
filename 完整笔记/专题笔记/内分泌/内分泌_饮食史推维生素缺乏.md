# 内分泌_饮食史推维生素缺乏

> [!tip] 一句话锁定
> 题干给你一段 **饮食史 / 生活方式**（纯素、酗酒、吸收不良、极端偏食），就是要你 **反推最可预测的营养素缺乏**。核心动作：**diet history → predictable deficiency**，而不是背全部维生素功能表。

---

## 一、核心思路

> [!info] 解题三步
> ```text
> 1. 抓饮食/生活方式关键词（strict vegan / alcohol / malabsorption / 偏食）
> 2. 该模式切断了哪类食物来源？
> 3. 对应到该来源独有的营养素 -> 就是最可能缺乏项
> ```

最常被当干扰项：**folate (叶酸)** —— 因为它和 B12 都致 **巨幼细胞性贫血 (megaloblastic anemia)**。区分钥匙：**看食物来源 + 有无神经系统受累**。

---

## 二、饮食史 → 缺乏素 → 临床表现（主表）

| 饮食史 / 高危模式 | 最可能缺乏 | 为什么 | 典型临床表现 |
|---|---|---|---|
| 严格素食 / 纯素 strict vegetarian / vegan（不吃肉蛋奶） | 钴胺素 cobalamin, 维生素 B12 | B12 天然来源几乎只在 **动物性食物** | 巨幼细胞性贫血 + **神经损伤**（后索/皮质脊髓束：本体感觉↓、共济失调、亚急性联合变性 subacute combined degeneration, SCD） |
| 慢性酗酒 chronic alcohol use | 硫胺素 thiamine, 维生素 B1（+ 叶酸 folate） | 摄入差 + 吸收↓ + 消耗↑ | **Wernicke 脑病**（意识混乱 + 眼肌麻痹 + 共济失调）→ Korsakoff；给葡萄糖前先补 B1 |
| 脂肪吸收不良 fat malabsorption（乳糜泻 celiac、慢性胰腺炎、胆汁淤积、囊性纤维化 cystic fibrosis, CF） | 脂溶性维生素 fat-soluble A / D / E / K | 脂肪吸收障碍 → 脂溶性维生素随粪脂丢失 | A: 夜盲；D: 骨软化 osteomalacia / 低钙；E: 神经病变/共济失调；K: 出血、PT/INR↑ |
| 只喂纯山羊奶 / 极端偏食婴儿 | 叶酸 folate | 山羊奶叶酸含量极低 | 婴儿巨幼细胞性贫血（**无** B12 型神经损伤） |
| 极低果蔬摄入 / 水手坏血病模式 | 维生素 C ascorbic acid | 新鲜果蔬缺失 | **坏血病 scurvy**：牙龈出血、瘀点、伤口不愈、螺旋状毛发 |
| 玉米为主食、缺烟酸 | 烟酸 niacin, 维生素 B3 | 玉米中烟酸不可利用 | **糙皮病 pellagra**：皮炎 Dermatitis、腹泻 Diarrhea、痴呆 Dementia（3D） |
| 长期无日晒 / 深肤色 / 母乳纯喂养未补 | 维生素 D | 皮肤合成不足 + 饮食少 | 儿童佝偻病 rickets / 成人骨软化 osteomalacia、低钙、继发甲旁亢 |

---

## 三、B12 vs 叶酸（最高频干扰点）

| 维度 | 维生素 B12 (cobalamin) | 叶酸 folate |
|---|---|---|
| 主要来源 | **动物性食物**（肉/蛋/奶） | 绿叶菜、豆类 leafy greens / legumes |
| 素食者风险 | **高**（strict vegan 经典缺乏） | 低（素食反而不缺） |
| 巨幼贫 | 有 | 有 |
| **神经损伤** | **有**（后索/皮质脊髓束） | **无** |
| 补叶酸的陷阱 | 单补叶酸可纠正贫血但 **掩盖并加重 B12 神经损伤** | — |

> [!warning] 考试钥匙
> 纯素食预防题：**先补 B12**（不是叶酸）。有神经系统线索的巨幼贫 → 一定是 **B12** 不是叶酸。

---

## 四、Memory Hook

> [!success] 记忆锚
> - **不吃动物性食物 → 补 B12**（叶酸在菜里，B12 不在菜里；B12 还管神经）
> - **酒 → B1**（Wernicke：先 B1 后葡萄糖）
> - **脂肪吸收不良 → ADEK**（脂溶性一起丢）
> - **果蔬缺 → C（坏血病）**；**玉米/烟酸缺 → B3（糙皮病 3D）**

## 五、复习自测

> [!question]- Q1. 健康年轻女性，严格素食（不吃肉、蛋、奶），已每日补铁。最应额外补充哪种营养素？
> **维生素 B12 (cyanocobalamin)**。B12 天然来源几乎只在动物性食物；缺乏致巨幼贫 + 神经损伤。叶酸是最强干扰项，但素食者叶酸通常不缺。

> [!question]- Q2. 巨幼细胞性贫血 + 本体感觉减退/共济失调 → 缺哪种？为什么不是叶酸？
> **维生素 B12**。只有 B12 缺乏累及后索/皮质脊髓束产生神经损伤，叶酸缺乏 **无** 神经症状。

> [!question]- Q3. 慢性酗酒者出现意识混乱 + 眼肌麻痹 + 共济失调，最先该补什么？给糖注意什么？
> **硫胺素 thiamine (B1)** —— Wernicke 脑病。**先补 B1 再给葡萄糖**，否则输糖可诱发/加重 Wernicke。

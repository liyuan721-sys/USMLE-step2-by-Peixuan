# 尿崩症 DI 与多饮多尿 (Polyuria / Polydipsia) 鉴别

> [!info] 一句话总览
> **真性多尿 = 24h 尿量 >3 L**。看到"多饮 + 多尿 + 稀尿"先做**三件分流**：① 查尿糖排 **DM (diabetes mellitus 糖尿病)** ② 看血钠/血渗 分**高(脱水→ DI)** vs **低(水中毒→ 心因性多饮)** ③ 不确定 → **禁水试验 (water deprivation test)**。核心二分：**禁水后尿能不能浓缩** + **给 ADH 有没有用**。
>
> **触发错题**：[[NBME11_错题本#^Q045]]（12 岁男孩**发作性**多饮多尿 + 父母离婚应激 + 血糖正常 + 尿比重 1.015 → **心因性多饮**，误选 DI）。

> [!note]- 本篇缩写速查（首次全称中英）
> - **DI** = diabetes insipidus 尿崩症（≠ DM 糖尿病）
> - **ADH** = antidiuretic hormone 抗利尿激素 = **AVP** (arginine vasopressin 精氨酸加压素)
> - **DDAVP** = desmopressin 去氨加压素（ADH 类似物）
> - **USG** = urine specific gravity 尿比重
> - **Uosm / Posm** = urine / plasma osmolality 尿/血浆渗透压
> - **AQP2** = aquaporin-2 水通道蛋白 2
> - **ENaC** = epithelial sodium channel 上皮钠通道

## 一、第一步：多饮多尿四步问诊法 ⭐

```
真性多尿 (>3 L/24h)
│
├─① 尿糖有吗？ ── 有 → DM（溶质性利尿 solute diuresis）
│                    第一步：finger-stick glucose / UA
│
├─② 血 Na / 血渗 高还是低？
│      高钠高渗（dry outside, dry inside）→ 偏 DI（水丢失致脱水）
│      低钠低渗（wet inside）           → 偏 心因性多饮（水灌进来）
│
├─③ 禁水试验后尿会不会浓缩？
│      会浓缩 → 心因性多饮（ADH 轴正常）
│      不浓缩 → 真性 DI
│
└─④ 再给 desmopressin (ADH) 有没有用？
       有效（Uosm 明显↑）→ Central DI（中枢性，ADH 不够）
       无效（Uosm 不动）  → Nephrogenic DI（肾性，肾不响应 ADH）
```

> [!tip] 一句话方向标
> **DI = "dry inside"**（缺水 → 高钠高渗）；**心因性多饮 = "wet inside"**（水多 → 低钠低渗）。**做禁水试验前，血钠方向就已经给你大半个答案。**

## 二、多饮多尿四大鉴别表 ⭐

| | 血糖 | 尿比重/Uosm | 血钠/Posm | 节律 | 禁水后 | 给 ADH 后 |
|---|---|---|---|---|---|---|
| **DM 糖尿病** | **↑** | 高（含糖） | — | 持续 | — | — |
| **心因性多饮** (primary/psychogenic polydipsia) | 正常 | **低-正常** | **低-正常**（水中毒可低钠） | **发作 + 心理** | **能浓缩** | （已浓缩） |
| **Central DI 中枢性** | 正常 | **低（稀）** | **高（脱水）** | 持续 | **不浓缩** | **浓缩**（Uosm↑ >50%） |
| **Nephrogenic DI 肾性** | 正常 | **低（稀）** | **高（脱水）** | 持续 | **不浓缩** | **仍不浓缩**（<50%） |

> [!warning] 最易混的两对
> - **DI vs 心因性多饮**：靠**血钠方向**（DI 高 / 心因性低）+ **禁水能否浓缩**（DI 不能 / 心因性能）。
> - **Central vs Nephrogenic DI**：禁水都不浓缩，**靠给 desmopressin 后尿能不能浓缩**区分（中枢能 / 肾性不能）。

## 三、禁水试验 (Water Deprivation Test) 判读 ⭐⭐

> [!danger] 三步读法（USMLE 最爱考）
> 1. **禁水** → 测尿渗 Uosm：
>    - **Uosm 升上去（能浓缩，>600–800）** → **心因性多饮**（ADH 轴正常，停水就能保水）→ 治疗 = **限水 + 精神评估**。
>    - **Uosm 仍低（不浓缩）** → 真性 DI，进第 2 步。
> 2. **给 desmopressin (DDAVP)** → 再测 Uosm：
>    - **Uosm 明显↑（>50%）** → **Central DI**（外源 ADH 有效 = 自身 ADH 不够）→ MRI 查垂体/下丘脑。
>    - **Uosm 几乎不动（<50%）** → **Nephrogenic DI**（肾不认 ADH）。

> [!info] 一个 caveat：慢性多尿的"髓质冲刷 (medullary washout)"
> 长期大量稀尿（慢性心因性多饮 或 慢性 DI）会冲淡肾髓质浓度梯度 → 禁水时浓缩能力**部分钝化**，结果可能不够干净。现代可加测 **copeptin（AVP 替代标志物）** 辅助分型。

## 四、病因谱（分型记忆）

> [!info] Central DI（ADH 分泌不足）
> - **特发性 idiopathic**（常自身免疫）
> - **头外伤 (head trauma) / 垂体手术后 (post-neurosurgery)**
> - **鞍区/下丘脑肿瘤**（颅咽管瘤 craniopharyngioma、生殖细胞瘤）
> - **浸润性**：结节病 (sarcoidosis)、朗格汉斯细胞组织细胞增生症 (Langerhans cell histiocytosis, LCH)
> - **缺血**：Sheehan 综合征（产后垂体坏死）
> - 典型 stem：**脑外伤/垂体术后突发大量稀尿 + 极度口渴 + 血钠↑；禁水不浓缩，给 DDAVP 后浓缩。**

> [!info] Nephrogenic DI（肾不响应 ADH）
> - **药物**：**锂 (lithium)** ⭐ 最常见获得性病因 / **地美环素 (demeclocycline，原用于治 SIADH)** / **两性霉素 B (amphotericin B)**
> - **电解质**：**高钙血症 (hypercalcemia)** ⭐（维生素 D 过量/甲旁亢；查 electrolytes + calcium）/ **低钾血症 (hypokalemia)**
> - **机制（为何高钙/低钾致肾性 DI）**：削弱集合管对 ADH 的反应、**下调水通道 AQP2** → 浓缩障碍。高钙另经 **CaSR (calcium-sensing receptor 钙敏感受体)** 抑制髓袢升支粗段 (TAL, thick ascending limb) 的 **NKCC2 (Na-K-2Cl 共转运体)**（袢利尿样）。高钙致多尿的**完整谱（脱水 + 缩容性碱中毒 contraction alkalosis + 急性肾损伤 AKI 三联）见 [[完整笔记/Peixuan分科笔记/肾脏]]**。
> - **遗传**：V2 受体 / **AQP2** 基因突变（X 连锁，婴儿期）
> - **梗阻后**：post-obstructive uropathy
> - 典型 stem：**长期吃锂的双相患者多尿夜尿 + 喝冰水；禁水无效，DDAVP 也无效。**

> [!tip] 妊娠期 DI (gestational DI)
> 妊娠中后期胎盘产生 **血管紧张素酶 (vasopressinase)** → **降解 ADH** → 一过性尿崩，排大量稀尿；**分娩后酶↓、症状消失**。治疗用 **desmopressin**（DDAVP 不被 vasopressinase 降解）。

## 五、治疗（按型）

| 类型 | 一线 | 机制/要点 |
|---|---|---|
| **Central DI** | **Desmopressin (DDAVP)** | 补外源 ADH；MRI 查病因 |
| **Nephrogenic DI** | **去诱因**（停锂、纠正高钙/低钾）+ **噻嗪 (thiazide)** ± **阿米洛利 (amiloride)** + 低盐低蛋白饮食 | 见下方"矛盾利尿" |
| **心因性多饮** | **限水 (water restriction) + 精神评估/治疗** | 不要补盐/补水，治根在精神科 |

> [!warning] Nephrogenic DI 的"矛盾性抗利尿 (paradoxical antidiuresis)"
> **噻嗪 (thiazide)** 治肾性尿崩：轻度容量收缩 → 近端小管重吸收↑ → 到达远端的水/钠↓ → **尿量反而下降**。**阿米洛利 (amiloride)** 阻断集合管 **ENaC** → 减少锂经 ENaC 进入主细胞 → 对**锂致肾性尿崩**特别有效，常与噻嗪联用。（NSAID 如吲哚美辛 indomethacin 可作辅助。）

> [!success] Desmopressin (DDAVP) 三用途 + 一大坑
> - **止尿**：Central DI、儿童遗尿 (nocturnal enuresis)
> - **止血**：促 vWF 释放 → 治 **血管性血友病 (von Willebrand disease, vWD)** 的中重度月经出血、轻型血友病 A
> - ⚠️ **最怕水中毒 (water intoxication) → 低钠血症**（恶心、头痛、抽搐）→ 用药需限水 + 监测血钠。

## 六、心因性多饮 (Primary / Psychogenic Polydipsia) 深挖 — 源 Q045 ⭐

> [!danger] 三把钥匙把 DI 排掉（本题错点）
> 1. **发作性 (episodic)**：每 3–4 周、持续 2–3 天，发作间完全正常 → **DM/DI 都是持续性**，发作性强烈指向**行为/心理**。
> 2. **心理应激背景**：父母离婚、周末去父亲家 → 心因性多饮的经典钩子。
> 3. **化验"太正常"**：血糖正常（排 DM）、尿比重 1.015 偏低、尿糖/蛋白阴、**血钠不高**（排 DI 的高钠脱水）。
>
> ∴ 别一见"多饮多尿"就反射 DI/DM——**先看是不是发作性 + 有没有应激 + 血钠是高还是低**。

> [!info] 机制 + 处理
> 过量饮水 → 血浆稀释（**低钠低渗**）→ 抑制 ADH → 排大量稀尿。常见于**精神分裂/双相**患者（总抱水杯）。**禁水后尿能浓缩**（ADH 轴正常）= 与真性 DI 的分水岭。处理：**限水 + 精神科评估**，不是补液也不是 DDAVP。

## 七、USMLE 秒选 + 高频陷阱

> [!danger] 高频陷阱清单
> 1. **"多饮多尿"≠ 自动 DI/DM** —— 先分流：尿糖（DM）→ 血钠方向（DI 高 / 心因性低）→ 禁水试验。
> 2. **DI 是持续性 + 高钠脱水**；**发作性 + 应激 + 化验正常 = 心因性多饮**（Q045）。
> 3. **Central vs Nephrogenic** 只能靠**给 DDAVP**区分，不是靠禁水（两者禁水都不浓缩）。
> 4. **长期吃锂 + 多尿 = 锂致肾性尿崩** → amiloride；**高钙/低钾也致肾性 DI** → 查钙钾。
> 5. **孕中后期一过性尿崩 = vasopressinase（妊娠 DI）** → 用 desmopressin（不被酶降解）。
> 6. **DDAVP 用药后低钠抽搐 = 水中毒** → 限水。

> [!success] Memory Hook
> - **DI = dry inside**（高钠脱水、持续、稀尿、禁水不浓缩）；**心因性多饮 = wet inside**（低钠、发作+应激、禁水能浓缩）。
> - **禁水分"真假 DI"，DDAVP 分"中枢 vs 肾性"**（中枢有效 / 肾性无效）。
> - **锂 + 多尿 → 肾性 DI → amiloride**；**孕妇一过性 DI → vasopressinase → desmopressin**。

# 内分泌_库欣综合征与库欣病（Cushing Syndrome vs Cushing Disease）

> [!info] 用途
> 厘清三个同名易混概念（**库欣综合征 / 库欣病 / 库欣反射**），掌握**高皮质醇血症**的临床识别、鉴别诊断与**三步诊断流程**（确认 → 定位 ACTH → 大剂量 DST）。整合自分科笔记 [[完整笔记/Peixuan分科笔记/endocrine]]。

---

## 一、三个"Cushing"先分清 ⭐（最高频混淆）

> [!danger] 一句话辨析
> **库欣综合征 = 结果（任何原因的高皮质醇）｜库欣病 = 其中一个病因（垂体 ACTH 腺瘤）｜库欣反射 = 完全不相干（颅高压的神经体征）。**

| 名称 | 英文 | 本质 | 所属学科 |
|---|---|---|---|
| **库欣综合征** | Cushing **syndrome** | **伞概念**：**任何原因**导致的慢性高皮质醇血症（hypercortisolism）的临床综合征 | 内分泌 |
| **库欣病** | Cushing **disease** | 库欣综合征的**一个亚型**：**垂体 ACTH（促肾上腺皮质激素，adrenocorticotropic hormone）分泌腺瘤**（pituitary corticotroph adenoma）所致 | 内分泌 |
| **库欣反射 / 三联** | Cushing **reflex / triad** | **与皮质醇无关**！= 颅内压↑（ICP↑，intracranial pressure）的代偿：**高血压 + 心动过缓 + 不规则呼吸**。机制：ICP↑→脑灌注压↓→脑干缺血→**交感升压（HTN）**→压力感受器反射性**减慢心率**→脑干受压致**呼吸乱**（休克相反 = 低压+快心率）| 神经（见 [[完整笔记/Peixuan分科笔记/neuro]]）|

> [!tip] 包含关系
> **库欣综合征 ⊃ 库欣病**。所有库欣病都是库欣综合征，但库欣综合征不一定是库欣病（还可能是外源激素 / 肾上腺瘤 / 异位 ACTH）。

---

## 二、病因分类：ACTH 依赖 vs 非依赖 ⭐

> [!danger] 分类钥匙
> **先问"高皮质醇是不是 ACTH 驱动的"——这决定下一步查什么、影像查哪里。**

```
高皮质醇血症（Cushing 综合征）
│
├─ 外源性 / 医源性（最常见！）── 长期糖皮质激素治疗
│     → ACTH 低 + 双侧肾上腺萎缩（无肿块）
│
├─ ACTH 依赖（ACTH 正常 / ↑）
│     ├─ 库欣病 = 垂体 ACTH 腺瘤（内源性最常见，~70%）
│     └─ 异位 ACTH（ectopic）── 小细胞肺癌 SCLC / 类癌
│
└─ ACTH 非依赖（ACTH 低/受抑）
      └─ 肾上腺源 ── 腺瘤 / 腺癌 / 结节性增生
```

> [!warning] 两个"最常见"别搞反
> - **库欣综合征总体最常见病因 = 外源性糖皮质激素（医源性）** —— stem 给"长期用泼尼松 prednisone 治哮喘 / RA（类风湿关节炎，rheumatoid arthritis）/ IBD（炎症性肠病，inflammatory bowel disease）"。
> - **内源性最常见 = 库欣病（垂体腺瘤）** —— 占内源性约 70%。

## 三、临床症状（题干线索 → 临床意义）⭐

> [!info] 机制总纲
> 皮质醇 = **升糖 + 分解代谢 + 抗炎 + 弱盐皮质激素活性**；长期过量 → 下列各系统表现。

| 系统 | 招牌表现 | → 临床意义（为什么） |
|---|---|---|
| **体型** | **向心性肥胖 + 满月脸（moon face）+ 水牛背（buffalo hump）+ 锁骨上脂肪垫** | 皮质醇致**脂肪向心性重分布** |
| **皮肤** | **紫色宽大条纹（striae，>1cm）+ 皮肤变薄 + 易瘀斑（ecchymosis）** | 胶原**分解** → 皮肤脆、真皮变薄露出血管 |
| **肌肉** | **近端肌无力（proximal myopathy）** | 皮质醇**分解肌蛋白**（catabolic）→ 起立/爬楼难（**特征性，PCOS 没有**）|
| **代谢** | **高血糖 / 糖尿病 + 向心性肥胖** | 皮质醇**升糖**（促糖异生、拮抗胰岛素）|
| **心血管** | **高血压（HTN）** | 皮质醇弱**盐皮质激素**作用 + 增强血管对儿茶酚胺敏感性 |
| **电解质** | **低钾性代谢性碱中毒** | 皮质醇水平**极高**时溢出激活盐皮质激素受体 →（→ 临床意义：**强烈提示异位 ACTH**，如 SCLC）|
| **骨** | **骨质疏松（osteoporosis）** | 皮质醇**抗钙**（抑制成骨、减少肠钙吸收）|
| **生殖/雄激素** | **多毛 hirsutism / 痤疮 / 月经紊乱** | 肾上腺**雄激素**↑（尤肾上腺癌）|
| **精神** | **抑郁 / 情绪改变 / 失眠** | 皮质醇中枢作用 |
| **免疫** | 易感染、伤口愈合差 | 皮质醇**抗炎/免疫抑制** |

> [!tip] 时间线钩子
> **Cushing 进展较快**（数月内体重↑、长紫纹、肌无力）；若"数年慢性轻症"更像 PCOS / 代谢综合征。

## 四、鉴别诊断 ⭐

> [!warning] 最常考：库欣 vs PCOS（年轻女性多毛+肥胖+月经乱）
> 二者都有肥胖/多毛/月经紊乱，**靠"近端肌无力 + 宽紫纹 + 显著高血压"把库欣拎出来**。

| 特征 | 库欣综合征 Cushing | 多囊卵巢综合征 PCOS |
|---|---|---|
| **近端肌无力** | **有（特征性）** | 无 |
| **高血压** | **显著升高** | 正常或轻度 |
| **皮肤条纹** | **紫色、宽大 >1cm** | 无（或肥胖性白纹）|
| **骨质疏松** | 可伴有 | 无 |
| **多毛/痤疮** | 常见（肾上腺雄激素）| **核心症状（卵巢雄激素）** |
| **病程** | **较快（数月）** | 慢性（数年）|

> [!info] 其他需排除
> - **假性库欣（Pseudo-Cushing）**：慢性酒精 alcohol、重度抑郁、病态肥胖、未控糖尿病 → 可有轻度皮质醇↑ + DST 不全抑制；用**深夜唾液皮质醇 + 24h UFC**结合临床甄别。
> - **外源性糖皮质激素**：病史 + **ACTH 低 + 双侧肾上腺萎缩（无肿块）+ 血/尿皮质醇可低**（若用的是地塞米松等不被检测的合成激素）。
> - **代谢综合征**：肥胖+HTN+高血糖，但无紫纹/近端肌无力/易瘀。

## 五、诊断三步流程 ⭐⭐（必背）

> [!danger] 顺序铁律
> **① 先确认"真高皮质醇" → ② 测 ACTH 分依赖/非依赖 → ③（仅 ACTH 高者）大剂量 DST 分库欣病 vs 异位。** 顺序不能乱（先定性、再定位、最后定病灶）。

```
疑似 Cushing（向心肥胖 + 宽紫纹 + 近端肌无力 + HTN/高血糖）
│
第①步 确认高皮质醇血症（≥1–2 项异常，先排除外源激素）
│   ├─ 24 小时尿游离皮质醇 24-hr UFC ↑
│   ├─ 深夜唾液皮质醇 ↑（昼夜节律消失）
│   └─ 过夜小剂量地塞米松抑制试验（Low-dose DST，1mg）→ 不被抑制
│
└─ 确认后 → 第②步 测血浆 ACTH（定位）
     │
     ├─ ACTH 低（受抑）= ACTH 非依赖 → 肾上腺源
     │     └─ 影像：肾上腺 CT（腺瘤 / 腺癌 / 增生）
     │
     └─ ACTH 正常 / 高 = ACTH 依赖 → 第③步 大剂量 DST（High-dose，8mg）
           ├─ 被抑制 >50% → 库欣病（垂体腺瘤）→ 垂体 MRI
           │                  └ MRI 阴性/不确定 → IPSS（岩下窦静脉采血）
           └─ 不被抑制 → 异位 ACTH → 胸 / 腹 CT 找原发（SCLC、类癌）
```

> [!tip] 大剂量 DST 的原理
> **垂体腺瘤保留部分负反馈敏感性** → 大剂量地塞米松能压下去（>50%）；**异位 ACTH 肿瘤无反馈** → 压不动。这是区分**库欣病 vs 异位**的关键逻辑。

> [!warning] 异位 ACTH 的招牌组合
> **极高 ACTH + 严重低钾性代谢性碱中毒 + 肺部肿块 + 起病快/色素沉着** → 异位 ACTH，首位想 **小细胞肺癌（SCLC，small cell lung carcinoma）**；下一步**胸部影像**。见 [[完整笔记/Peixuan分科笔记/pulmonary]] 副肿瘤。

> [!info] 首选检查速记
> **筛查（定性）= 24-hr UFC 或 过夜 1mg DST 或 深夜唾液皮质醇（任一/两项阳性）｜定位 = ACTH → 大剂量 DST → MRI/CT。**

## 六、治疗（手术首选 + 药物细则）

### 6.1 各病因首选（手术为主）

| 病因 | 一线处理 |
|---|---|
| **库欣病（垂体腺瘤）** | **经蝶窦垂体腺瘤切除术（TSS，transsphenoidal surgery）**；术后复发可放疗 / 药物 |
| **肾上腺腺瘤 / 腺癌** | **肾上腺切除术** |
| **异位 ACTH** | 治原发肿瘤；不能切时**药物抑皮质醇** |
| **外源性激素** | **逐渐减量**（不可骤停，防肾上腺危象）|

### 6.2 药物治疗细则（按作用靶点三类）

**① 肾上腺甾体合成抑制剂**（不分病因都能用，直接掐断皮质醇合成）

| 药 | 机制 | 关键副作用 / 考点 |
|---|---|---|
| **酮康唑 ketoconazole** | 抑多个甾体合成酶（11β-羟化酶 / 侧链裂解等）| **肝毒性**（查肝功 LFT）；抑雄激素 → 男性乳腺发育 |
| **美替拉酮 metyrapone** | 抑 **11β-羟化酶**（11-脱氧皮质醇 → 皮质醇最后一步）| 上游 11-脱氧皮质酮↑ → **高血压 / 低钾**；雄激素↑ → 多毛 |
| **奥西卓司他 osilodrostat** | 强效 11β-羟化酶抑制（新药，专批库欣病）| 同类电解质 / 雄激素效应 |
| **依托咪酯 etomidate** | 静脉、起效快，抑 11β-羟化酶 | **重症 / 皮质醇危象** ICU 急用 |
| **米托坦 mitotane** | **肾上腺溶解**（毁皮质细胞）| 主用于**肾上腺皮质癌**；起效慢、毒性大 |

**② 针对垂体腺瘤**（仅**库欣病**，降 ACTH）

| 药 | 机制 | 考点 |
|---|---|---|
| **帕瑞肽 pasireotide** | 生长抑素类似物（结合 corticotroph 促皮质激素细胞上 SSTR5，生长抑素受体 5 型）→ ↓ACTH | **高血糖**（招牌副作用，加重糖尿病）|
| **卡麦角林 cabergoline** | 多巴胺激动剂（部分腺瘤表达 D2 受体）→ ↓ACTH | 口服、相对温和 |

**③ 糖皮质激素受体拮抗剂**

| 药 | 机制 | 考点 |
|---|---|---|
| **米非司酮 mifepristone（RU-486）** | 阻断**糖皮质激素受体**（外周不让皮质醇起效）| 用于**库欣伴高血糖 / 糖尿病**；⚠️ 皮质醇 / ACTH 反而↑ → **不能用皮质醇监测疗效**；低钾、肾上腺功能不全难察、堕胎作用 |

> [!info] 用药时机（什么时候选药而非手术）
> ① **术前过渡**（重症先压一压）② **术后未缓解 / 复发** ③ **异位 ACTH 原发灶切不掉** ④ **肾上腺皮质癌**（米托坦）⑤ **不能耐受手术**。手术（TSS / 肾上腺切除）仍是各类**首选根治**。

> [!tip] Step 2 三钩子
> **酮康唑 = 肝毒性（查肝酶）｜帕瑞肽 = 降 ACTH 但升血糖｜米非司酮 = 用于高血糖型，但皮质醇升高 → 改看临床 / 血糖追疗效（不能用皮质醇）。**

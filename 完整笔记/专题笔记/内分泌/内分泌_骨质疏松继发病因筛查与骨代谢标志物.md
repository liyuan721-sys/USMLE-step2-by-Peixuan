# 骨质疏松继发病因筛查与骨代谢标志物（继发性骨丢失 workup + ALP/PTH/钙磷解读）

> [!info] 这张的定位
> 绝经后女性骨质疏松 (osteoporosis) 多因雌激素↓（原发）；但出现"红旗"时须筛 **继发性骨丢失 (secondary bone loss)** 的病因。本篇做**"发现异常 → 找继发原因"的 workup 主场** + **骨代谢标志物（ALP / 钙 / 磷 / PTH / 维 D）解读**。
> 不重复的部分：高钙 PTH 分流细节 → [[完整笔记/专题笔记/内分泌/内分泌_高钙血症分层与PTH路径]]；多发性骨髓瘤全谱 → [[完整笔记/专题笔记/血液/血液_血液恶性肿瘤白血病淋巴瘤骨髓瘤]]；双膦酸盐不良反应 → [[完整笔记/专题笔记/内分泌/内分泌_双膦酸盐不良反应]]。
> 触发错题：[[mistakes/uworld-mistakes_2026-07#^Q17681]]（骨质疏松 → 多发性骨髓瘤 → SPEP）/ [[mistakes/uworld-mistakes_2026-07#^Q4572]]（类风湿关节炎 → 骨质疏松）。

---

## §一 何时怀疑"继发性"骨丢失（≠ 单纯绝经后）⭐

> [!warning] 继发性骨丢失红旗（出现任一 → 加做继发病因 workup）
> - **非局限性骨痛 / 骨压痛、无外伤骨折**（单纯骨质疏松在骨折前通常不痛）
> - **初查有异常值**：贫血 (anemia)、高钙血症 (hypercalcemia)、肌酐↑（肾功能不全）、碱性磷酸酶 (alkaline phosphatase, ALP) 异常
> - **不符流行病学**：男性、绝经前女性、无危险因素却重度骨质疏松
> - **生活方式性已排除**：不吸烟酗酒、BMI 正常、均衡饮食、规律负重运动 —— 反而更该找器质性继发原因

> [!info] 初始评估（所有骨质疏松都该做的基础 labs）
> 全血细胞计数 (CBC) + 代谢全套（含**钙、肌酐、25-羟维生素 D**）→ 目的是**排除继发原因**。有显著异常（如高钙）或非局限骨痛 → 再向下深挖。

## §二 继发性骨丢失病因筛查（"clues to secondary causes"）⭐

> [!info] 按"招牌症状 + 关键化验"逐类扫
> 表源思路同 UWorld "clues to secondary causes of bone loss"（客观事实重述，非原文）。

| 病因 | 招牌线索 | 关键化验 |
|---|---|---|
| **吸收不良 (malabsorption，如乳糜泻 celiac disease)** | 腹泻、体重↓ | **25-羟维生素 D↓ + 尿钙↓**；PTH 反应性↑；常伴缺铁性（小细胞）贫血 |
| **甲状腺功能亢进 (hyperthyroidism)** | 体重↓、怕热、震颤、甲状腺肿 | **TSH↓** |
| **皮质醇增多 (hypercortisolism / 库欣 Cushing)** | 向心性肥胖、满月脸、紫纹 | 高血糖；地塞米松抑制试验 / 24h 尿皮质醇 |
| **甲状旁腺功能亢进 (hyperparathyroidism)** | 高钙、高尿钙、肾结石 | **高钙 + 高/不当正常 PTH**（见 [[完整笔记/专题笔记/内分泌/内分泌_高钙血症分层与PTH路径]]）|
| **炎症性疾病（如类风湿关节炎 RA）** | 关节痛、晨僵 | **ESR / CRP↑**（见 [[完整笔记/专题笔记/风湿/风湿_RA术前风险与系统受累]] §三 骨质疏松行）|
| **性腺功能减退 (hypogonadism)** | ♀ 闭经 / ♂ 性欲↓、勃起障碍、体毛↓ | 性激素↓ |
| **多发性骨髓瘤 (multiple myeloma, MM)** ⭐ | 贫血 + 高钙 + 肌酐↑、非局限骨痛 | **蛋白间隙↑ + SPEP/UPEP**（见 §四）|

## §三 骨代谢标志物解读 ⭐核心

### 3.1 ALP（碱性磷酸酶）= 成骨细胞活性表

> [!danger] ALP（骨型）反映的是"成骨细胞 (osteoblast) 活性"，不是骨破坏 ⭐
> **碱性磷酸酶 (alkaline phosphatase, ALP)** 骨型由**成骨细胞分泌**，参与矿化。判断一个骨病 ALP 升不升，看的是"**成骨细胞有没有被拉起来干活**"，而非"骨在被吸收还是生成"。
> 关键反直觉点：很多**以破骨为主**的病 ALP 照样升高——因为骨代谢是**偶联的 (coupled)**，破骨活跃时成骨细胞被拉起代偿修复。

```
成骨细胞活跃 → ALP↑              成骨细胞被抑制 / 无参与 → ALP 正常
──────────────────              ──────────────────────────
Paget 病/变形性骨炎(高转换造乱骨)⭐ 多发性骨髓瘤(瘤细胞分泌 DKK1/IL-3
骨软化/佝偻(代偿矿化)               → 抑制成骨细胞)⭐ ── 几乎独此一家
成骨性转移(前列腺癌)
甲状旁腺功能亢进(高转换)
骨折愈合 / 儿童生长(生理性)
```

> [!warning] 多发性骨髓瘤 = "大量溶骨却 ALP 正常"的经典例外 ⭐⭐
> MM 瘤细胞分泌 **DKK1、IL-3 直接抑制成骨细胞**，同时经 **RANKL** 激活破骨 → **破骨狂拆、成骨被摁住**。成骨细胞不造骨、也就不分泌 ALP → 尽管广泛溶骨 + 骨痛 + 病理性骨折，**ALP 典型正常**。
> - 精确表述：不是"溶骨就不升 ALP"，而是"**MM 把成骨细胞抑制了所以不升**"；普通溶骨性转移（如乳腺癌）常伴混合成骨反应，ALP 仍会升。
> - 应试钥匙：**骨质疏松/溶骨 + 骨痛 + ALP 正常 + 贫血/高钙/肾衰 → 想 MM**（ALP 正常反而是支持点）。
> - 别忘 ALP 也有**肝/胆道来源**（胆汁淤积↑）：先看 **GGT（γ-谷氨酰转肽酶）**——GGT 升 = 肝胆源；GGT 正常 + ALP 升 = 骨源。

### 3.2 钙 / 磷 / PTH / 维 D 组合速记

| 化验组合 | 指向 |
|---|---|
| **高钙 + 高/不当正常 PTH** | 甲状旁腺功能亢进（PHPT / FHH，见高钙专篇）|
| **高钙 + PTH 低 + 贫血/肾衰/溶骨 + 蛋白间隙↑** | **多发性骨髓瘤（ALP 常正常）** ⭐ |
| **25-VitD↓ + PTH 反应性↑ + 低/正常钙** | 维生素 D 缺乏 / 骨软化 / 吸收不良（ALP 会↑）|
| **钙、磷、PTH、VitD 全正常 + T 值 ≤ −2.5** | 原发性（绝经后 / 老年）骨质疏松 |

## §四 多发性骨髓瘤：骨质疏松题里的"隐藏恶性"⭐

> [!danger] CRAB + 蛋白间隙 → SPEP
> 骨质疏松 / 骨痛 + 以下 → 别停在"原发骨质疏松"，查骨髓瘤：
> - **CRAB**：高钙 (**C**a↑) / 肾衰 (**R**enal) / 贫血 (**A**nemia，正细胞性) / 骨病 (**B**one，穿凿样溶骨)
> - **蛋白间隙 (protein gap) = 总蛋白 − 白蛋白 > 4 g/dL**：白蛋白不高反低、总蛋白却高 → 差值是异常单克隆球蛋白 (M 蛋白)
> - **下一步 = 血清蛋白电泳 (serum protein electrophoresis, SPEP)** 找 M 尖峰（后续 + 血清游离轻链 / 尿蛋白电泳 UPEP + 骨髓活检）
>
> 完整白血病/淋巴瘤/骨髓瘤谱 + MGUS/Waldenström 鉴别 → [[完整笔记/专题笔记/血液/血液_血液恶性肿瘤白血病淋巴瘤骨髓瘤]]。

## §五 高钙 → 永远先测 PTH（一句话 + 双链）

> [!tip] 高钙病因学第一步 = 测 PTH
> 高钙 + **PTH 被抑制**（如本题 PTH 8，正常 10–65）= 非甲状旁腺来源 → **枪毙甲旁亢**，推向恶性/维 D/骨髓瘤。完整决策树（PTH 依赖 vs 非依赖、PTHrP/维 D/溶骨三分支）→ [[完整笔记/专题笔记/内分泌/内分泌_高钙血症分层与PTH路径]]。

## §六 治疗注意（别急着上药）

> [!warning] 查明继发原因前不盲目开双膦酸盐
> - **未排除继发原因就直接口服双膦酸盐 (bisphosphonate) = 错**——本例真病因是骨髓瘤，需先确诊治疗。
> - **肾功能不全是双膦酸盐禁忌**（eGFR < 30–35 mL/min 一般不用）——本例肌酐 1.9 已是禁忌信号。
> - 双膦酸盐长期不良反应（下颌骨坏死 (osteonecrosis of the jaw, ONJ) / 非典型股骨骨折 (atypical femoral fracture, AFF) / 食管炎）→ [[完整笔记/专题笔记/内分泌/内分泌_双膦酸盐不良反应]]。

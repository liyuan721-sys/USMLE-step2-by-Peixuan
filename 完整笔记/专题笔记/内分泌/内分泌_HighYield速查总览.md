## 一、甲状腺（Thyroid）

### 1.1 甲状腺结节 (thyroid nodule) — 先 TSH 分流（高频）

**口诀「低碘高穿」**：发现结节后先查 TSH → **TSH 低 → 做碘扫描 (I-131 scintigraphy)**；**TSH 正常/高 → 超声 (US) + 细针穿刺 (FNA)**。

```
触诊甲状腺结节 → 第一步查 TSH
├─ TSH 低（功能亢进）→ 放射性碘扫描 → "热结节" 几乎不恶变 → 不做 FNA
└─ TSH 正常 / 高（非功能性，恶性风险高）→ 超声
       └─ 可疑特征 + >1 cm → FNA
              └─ FNA 良性 → 试 levothyroxine（左甲状腺素）压 TSH
```

> [!tip] 超声恶性"红旗"（→ 该穿刺）
> 微钙化 (microcalcifications) / 边缘不规则 (irregular margins) / 纵横比 >1 (taller-than-wide) / 极低回声 (hypoechoic)。最常见甲状腺癌 = 乳头状癌 (papillary thyroid carcinoma)。

### 1.2 甲亢/甲减 — TSH / T3T4 / TRH 三轴定位

| 情况 | T3/T4 | TSH | TRH | 钥匙 |
|:---|:---|:---|:---|:---|
| 原发性甲减 | 低 | 高 | 高 | 高 TRH → 高泌乳素倾向 |
| 原发性甲亢 | 高 | 低 | 低 | Graves / 毒性结节 |
| 垂体性甲亢（继发） | 高 | 高/不低 | 低 | 垂体自主分泌 TSH |
| 下丘脑问题 | 低 | 低/不高 | 低 | TRH 本身不足 |

- **Graves 病 (Graves disease)**：TSH receptor antibody（TSH 受体抗体）+ 眼病 (exophthalmos)；初治 **β 受体阻滞剂 (β-blocker) 缓症 → 甲巯咪唑 (methimazole) / 丙硫氧嘧啶 (PTU)**；根治 = 放射性碘消融 (radioactive iodine ablation) 或手术。

![[{2CE26B3F-3189-4D47-B977-77DDFC449C17}.png|360]]
*胫前黏液水肿 (pretibial myxedema)：双小腿前侧非凹陷性硬斑 / 结节、橘皮样 → **Graves 病**特异皮肤征（与突眼、杵状指共属甲状腺皮病）。*
- **外源性甲状腺激素 (exogenous thyroid hormone)**：高 T3/T4 + **低 TSH + 低甲状腺球蛋白 (Tg, thyroglobulin)**（甲状腺停工 → Tg 低，鉴别钥匙）。
- **毒性多结节甲状腺肿 (toxic multinodular goiter / Plummer disease)**：老年 + 甲亢 + 压迫症状。
- **甲状腺功能正常病态综合征 (euthyroid sick syndrome)**：人很病但甲状腺没真病；**反 T3 (rT3) 高**，治原病不治甲状腺。
- **亚急性甲状腺炎 (subacute / De Quervain thyroiditis)**：病毒感染后 **痛性** 颈部。

### 1.3 甲状腺危象 (thyroid storm) — 急症

> [!danger] 钥匙 + 处理顺序
> 诱因：甲状腺/非甲状腺**手术**、急性病（创伤/感染）、分娩、**急性碘负荷（含碘造影剂 iodine contrast）**。
> 表现：**高热 (>38.5°C) + 极度心动过速 (>140/min)** + 意识改变 + GI（呕吐/黄疸）+ 应激性高血糖。
> 处理顺序：① 降温 ② β 受体阻滞剂 ③ 抗甲状腺药 (PTU) ④ 碘 (iodine) ⑤ 糖皮质激素 (glucocorticoid)。

🧩 双膦酸盐 (bisphosphonate) 与骨相关不良反应（颌骨坏死 ONJ 等）→ [[完整笔记\专题笔记\内分泌\内分泌_双膦酸盐不良反应]]

## 二、肾上腺（Adrenal）

### 2.1 库欣综合征 (Cushing syndrome) — 三步走

**钥匙**：向心性肥胖 + 宽紫纹 + 近端肌无力 + 高血压 + 糖耐量异常；月经紊乱 + 多毛 + **近端肌无力 + 高血压/高血糖 = 必选库欣**（区别多囊卵巢综合征 PCOS, polycystic ovary syndrome）。

```
第一步 确认高皮质醇（任一异常即阳性）
   24h 尿游离皮质醇 (UFC)↑ / 深夜唾液皮质醇↑ / 小剂量地塞米松抑制试验 (Low-dose DST, 1mg) 不被抑制
第二步 测 ACTH 定位
   ├─ ACTH 低 → 病在肾上腺（腺瘤/增生）→ 肾上腺 CT
   └─ ACTH 高 → 垂体（库欣病）或异位 → 第三步
第三步（ACTH 高者）大剂量地塞米松抑制试验 (High-dose DST, 8mg)
   ├─ 能抑制（>50%）→ 库欣病（垂体腺瘤）→ MRI
   └─ 不能抑制 → 异位 ACTH（如肺小细胞癌）
```

> [!tip] 异位 ACTH 钥匙
> 库欣表现 + **极高 ACTH + 严重低钾碱中毒 + 肺部阴影** → 直接想异位分泌。

### 2.2 肾上腺皮质功能不全 (adrenal insufficiency) — 8AM 皮质醇 + ACTH

```
怀疑 AI → 8 AM 血皮质醇
   <3 提示 AI ; >18 可排除
→ 测基础 ACTH 定位
   ├─ ACTH 高 → 原发（Addison 病）→ 皮肤色素沉着 + 低 Na + 高 K
   └─ ACTH 低 → 继发/三发（垂体/下丘脑）→ 无色素沉着，K 正常
→ 确诊：促肾上腺皮质激素兴奋试验 (Cosyntropin stimulation test)
```

- **原发性肾上腺皮质功能不全 (primary adrenal insufficiency, Addison 病)** 病因：自身免疫性肾上腺炎（最常见）/ 结核 (TB) / 转移浸润。
- **实验室**：低钠 + 高钾（醛固酮↓）、嗜酸性粒细胞增多 (eosinophilia)（皮质醇↓）、晨皮质醇低 + ACTH 高。

🧩 原发 vs 继发 vs 三发鉴别 + 应激剂量 (stress dose) + 肾上腺危象 → [[完整笔记\专题笔记\内分泌\内分泌_肾上腺皮质功能不全]]

### 2.3 嗜铬细胞瘤 (pheochromocytoma) — 先生化后影像

**钥匙**：阵发性头痛 + 出汗 + 心动过速三联 + 高血压 + **直立性低血压**（血容量不足）+ 不明原因高血糖。

```
Step 1 生化（先）：血浆游离变肾上腺素 (plasma free metanephrines) 或 24h 尿变肾上腺素/VMA
Step 2 定位：腹部 CT/MRI（想腹膜后 retroperitoneal）
Step 3 功能显像：MIBG 扫描（仅 CT 阴性或疑恶性时）
生化可疑不确定 → 可乐定抑制试验 (clonidine suppression test)
```

> [!danger] 术前用药顺序铁律
> **先 α 受体阻滞剂 (α-blocker，如酚苄明 phenoxybenzamine) 10–14 天 → 控压扩容 → 后 β 受体阻滞剂 (β-blocker，如普萘洛尔 propranolol) 2–3 天 → 手术。**
> ❌ 先给 β → unopposed α（α 受体无对抗）→ 高血压危象。
> 注：嗜铬细胞瘤 10% 双侧 / 10% 肾上腺外 / 10% 恶性；可见于 MEN2（多发性内分泌瘤 2 型）/ NF1（1 型神经纤维瘤病 neurofibromatosis type 1）/ VHL（希佩尔-林道综合征 von Hippel-Lindau）。

### 2.4 原发性醛固酮增多症 (primary hyperaldosteronism / Conn 综合征) — ARR 三步

**钥匙**：高血压 + **低钾**（自发或利尿剂诱发的不成比例低钾）+ 代谢性碱中毒 + **肾素 (renin) 降低**。

```
Step 1 筛查：醛固酮/肾素比值 (ARR) >20 且 醛固酮 (PAC) >15 ng/dL = 阳性
Step 2 确诊：生理盐水负荷试验（2h 内静滴 2L NS）→ PAC >10 ng/dL 支持 PA（PAC <5 排除）
Step 3 分型：肾上腺 CT 或肾上腺静脉采样 (AVS)
   ├─ 单侧腺瘤 (Conn) → 手术切除
   └─ 双侧增生 (IHA) → 螺内酯 (spironolactone) / 依普利酮 (eplerenone)
```

> [!warning] ARR 药物干扰（高频）
> **螺内酯**升肾素 → ARR 假阴性，筛查前停 4–6 周；**β 受体阻滞剂**抑肾素 → 假阳性；ACEI/ARB、噻嗪类利尿剂 → 假阴性；低钾抑制醛固酮 → 测前先补钾。
> 原发性 = 肾素**降低**；继发性 (secondary，肾素-血管紧张素-醛固酮系统 RAAS 激活如肾动脉狭窄) = 肾素**升高**。

### 2.5 先天性肾上腺增生 (CAH, congenital adrenal hyperplasia)

| 缺陷酶 | 雄激素 | 血压 / 电解质 | 钥匙 |
|:---|:---|:---|:---|
| 21-羟化酶 (21-hydroxylase，最常见) | 高 | 低血压 + 失盐（低 Na 高 K） | 女婴外生殖器模糊；筛查 17-OHP (17-羟孕酮) |
| 11β-羟化酶 (11β-hydroxylase) | 高 | **高血压 + 低钾**（11-去氧皮质酮↑） | 与 21-羟化酶相反 |
| 17α-羟化酶 (17α-hydroxylase) | 低 | 高血压 + 低钾 | 性发育不足 |

CAH 首选筛查 = **17-OHP**；确诊 = 基因分析。

## 三、垂体（Pituitary）

### 3.1 泌乳素瘤 (prolactinoma)

**钥匙**：女性闭经/溢乳（常微腺瘤 microadenoma）；男性性欲减退/勃起功能障碍 (ED, erectile dysfunction)（常大腺瘤 macroadenoma）；大腺瘤压迫 → **双颞侧偏盲 (bitemporal hemianopsia)**。

```
查泌乳素 (prolactin) → MRI 前先排除：甲减（查 TSH）/ 怀孕 / 抗精神病药
泌乳素 >200 ng/mL → 泌乳素瘤可能大；<200 想垂体柄效应 / 药物
→ 治疗首选多巴胺激动剂 (dopamine agonist)：卡麦角林 (cabergoline) > 溴隐亭 (bromocriptine)
   （即便大腺瘤压迫视交叉，仍首选药物，观察缩小）
→ 难治/继续生长 → 经蝶窦手术 (transsphenoidal surgery)
```

> [!tip] 无功能腺瘤 vs 泌乳素瘤
> 无功能腺瘤泌乳素轻度升高 (<200，垂体柄效应) + 其他垂体激素↓（压迫）+ α-亚单位↑；**无功能大腺瘤致视力损害 → 首选经蝶窦手术**（与泌乳素瘤先药物不同）。20–30 岁泌乳素瘤 → 排除 MEN1。

### 3.2 肢端肥大症 (acromegaly)

```
Step 1 筛查：血清 IGF-1（半衰期长，最佳筛查）
Step 2 确诊：IGF-1 高 → 口服葡萄糖耐量试验 (OGTT) 抑制试验（生长激素 GH 不被抑制）
Step 3 定位：垂体 MRI
治疗：经蝶窦切除 (transsphenoidal resection) → 奥曲肽 (octreotide) → 培维索孟 (pegvisomant)
```

> [!warning] 伴随
> 最常见死因 = 心血管疾病（充血性心衰 CHF）；确诊后必查**结肠镜 (colonoscopy)**（结肠癌风险↑）。

![[{A3FAF8C9-8692-443E-AF42-EDFB0342F8C7}.png|400]]

*肢端肥大症 (acromegaly) 体征：手足增大、面容粗化、眉弓突出、下颌前突 (prognathism)、皮赘、腕管综合征。*

![[{54F10BC4-E4C1-4EAB-AB17-977E501F8E0D}.png|400]]

*生长激素轴：GHRH/生长抑素 (somatostatin) 调控 GH → 肝产 IGF-1 + 直接组织效应（胰岛素抵抗↑、脂解↑）。*

### 3.3 尿崩症 (DI, diabetes insipidus) vs 精神性烦渴 — 限水 + 去氨加压素

| 诊断 | 限水后尿渗透压 | 给去氨加压素 (desmopressin) 后 | 钥匙 |
|:---|:---|:---|:---|
| 正常 / 精神性烦渴 (primary polydipsia) | 显著升高 (>600) | 无明显变化 | 精神病史/抗精神病药口干 |
| 中枢性 DI (central DI) | 低（仍稀释） | 显著升高 (>50%) | 垂体手术/外伤后 → MRI + desmopressin |
| 肾性 DI (nephrogenic DI) | 低（仍稀释） | 无变化 | 长期锂盐 (lithium) / 高钙 → 去诱因 + 噻嗪 (thiazide)；**锂致者优选加阿米洛利 (amiloride) 阻上皮钠通道 (ENaC) 减锂入胞** |

> [!tip] DDAVP（去氨加压素）"两治一怕"
> 既能"止尿"（治尿崩）又能"止血"（治血管性血友病 vWD），但最怕"水中毒"（低钠血症 → 恶心/头痛/抽搐）。

> [!info] 深挖 → DI / 多饮多尿专题
> 四步问诊（尿糖→血钠→禁水→DDAVP）+ 四大鉴别表 + 禁水试验三步判读 + 病因/治疗全谱 + 心因性多饮 → [[完整笔记/专题笔记/内分泌/内分泌_DI尿崩与多饮多尿鉴别]]。

### 3.4 SIADH（抗利尿激素分泌不当综合征）

钥匙：**ADH 过量 = 尿少 + 低钠血症（euvolemic 等容性）**——与 DI（ADH 缺乏 = 多尿）方向相反。重度症状性低钠 → **3% 高张盐水 (hypertonic saline)**。

> [!warning] 妊娠期多尿鉴别（尿比重 USG, urine specific gravity 是钥匙）
> 妊娠期尿崩症 (gestational DI)：USG <1.006（胎盘血管紧张素酶 vasopressinase 降解抗利尿激素 ADH）；妊娠期糖尿病 (GDM, gestational diabetes mellitus)：USG >1.020（渗透性利尿）；生理性尿频：24h <3L，USG 正常。

## 四、糖尿病（Diabetes Mellitus）

### 4.1 诊断阈值（原文照搬）

> [!info] 糖尿病诊断标准（任一）
> - 随机血糖 >200 mg/dL **+ 症状**
> - 糖化血红蛋白 (HbA1c) >6.5%
> - 空腹血糖 >126 mg/dL（**2 次不同场合**）
> - 口服葡萄糖耐量试验 (OGTT) >200（75 g 葡萄糖）
>
> 筛查：所有高血压患者应筛查糖尿病。A1c 目标：**≤7%（年轻健康）/ ≤8%（年长、合并症、预期寿命有限）**；每 3–6 月查 A1c。

### 4.2 DKA（糖尿病酮症酸中毒）管理顺序（必背）

> [!danger] 顺序：先补液 → 先看钾 → 上胰岛素 → 血糖到 200 加 D5 → AG 关闭转皮下
> 1. **0.9% 生理盐水 (normal saline)** 扩容（前 1–2h）
> 2. **查钾**：K **<5.3 先补钾**，再上胰岛素
> 3. **静脉常规胰岛素 (IV regular insulin)**（regular 是唯一可静脉用的胰岛素）
> 4. 血糖降到 **200** → 液体换 **D5W**（继续打胰岛素直到酮症酸中毒真正解决）
> 5. 跟踪**阴离子间隙 (anion gap)**，目标 **8–12**
> 6. AG 关闭 → 改**皮下胰岛素 (subcutaneous insulin)** + 进食，重叠 1–2h 后停静脉胰岛素

> [!warning] DKA 假性高钾 (pseudohyperkalemia)
> 血钾可高，但**总体钾储备低**（细胞内钾外移）；上胰岛素后钾会进细胞 → 提前防低钾。

### 4.3 HHS（高渗性高血糖状态，hyperosmolar hyperglycemic state）

钥匙：极高血糖 + 高渗 + 脱水 + 意识改变，但**无显著酮症**（区别 DKA）；处理同序——**先补液 → 纠正电解质 → 上胰岛素**。

### 4.4 黎明现象 vs Somogyi 效应（晨高血糖）

| | 黎明现象 (dawn phenomenon) | Somogyi 效应 |
|:---|:---|:---|
| 机制 | 清晨升糖激素（GH）↑ → 肝糖输出↑ | 夜间低血糖 → 反调升糖激素反跳 |
| 鉴别钥匙 | **查 2–3 AM 血糖**：正常/高 = 黎明 | 2–3 AM **低** = Somogyi |
| 处理 | 补足凌晨基础胰岛素 (basal insulin) | 减少夜间胰岛素 / 睡前加餐 |

### 4.5 降糖药钥匙

| 药 | 钥匙 / 不良反应 |
|:---|:---|
| 二甲双胍 (metformin) | 2 型糖尿病 (T2DM) 一线；乳酸酸中毒；造影/肾损停用 |
| 磺脲类 (sulfonylureas) | 低血糖 + 体重增加 |
| 吡格列酮 (pioglitazone, TZD) | 水肿 + 体重增加 + **加重心衰** |
| α-葡糖苷酶抑制剂 (acarbose) | 延缓碳水吸收；胀气 |
| GLP-1 受体激动剂 (GLP-1 RA) | 减重 + 慢胃排空；警惕胰腺炎 |
| DPP-4 抑制剂 (DPP-4 inhibitor) | 让自身肠促胰素 (incretin) 活更久 |
| SGLT2 抑制剂 (SGLT2 inhibitor) | 利尿排糖，保护肾心 |

> [!tip] 大血管风险↓最有效
> 戒烟 + 血压控制 + 他汀 (statin)（不是只盯血糖）。

## 五、钙磷骨代谢（Calcium / PTH / Bone）

### 5.1 PTH 一句话

PTH → ↑骨吸收 + ↑肾钙重吸收 + **↓磷 (phosphate)**。

![[{C463931B-9BDA-4673-8B9A-2406B084B2CE}.png|400]]

*PTH 作用流程：骨吸收↑ + 肾钙重吸收↑、磷重吸收↓ → 血钙↑、尿磷↑。*

### 5.2 原发性甲状旁腺功能亢进 (primary hyperparathyroidism, PHPT)

**实验室**：高 PTH + 高钙 + **低磷**；最常见病因 = 甲状旁腺腺瘤 (parathyroid adenoma)，问题在腺体本身。

> [!info] 甲状旁腺切除 (parathyroidectomy) 手术指征（任一即手术，必考）
> - **年龄 <50 岁**
> - 症状性高钙血症
> - 钙 **>正常上限 1.0 mg/dL（即 >11.5）** 或 24h 尿钙 >400 mg/day
> - 骨：骨质疏松（T-score <−2.5）或脆性骨折
> - 肾：肾结石/肾钙质沉着；慢性肾病 (CKD, chronic kidney disease)（GFR <60 mL/min）
>
> 反之：老年（>50）+ 钙微涨（<11.5）+ 骨硬（T>−2.5）+ 肾不亏（GFR>60）→ **观察 + 水化 + 随访**，不开刀。

### 5.3 继发性甲旁亢 (secondary hyperparathyroidism)

慢性肾病 (CKD) → 磷排不出 → 结合钙 → 钙↓ → 刺激 PTH；**高磷 + 低钙 + 高 PTH**（与原发相反）。

### 5.4 重度高钙血症 (severe hypercalcemia) — 急症先治

> [!danger] Ca >14 或症状性 → 先救命再诊断
> **先 NS 补容量 + 降钙素 (calcitonin)**（救命），不是先做病因诊断（如骨髓活检）。
> 后续：双膦酸盐 (bisphosphonate)（起效慢但持久）/ 糖皮质激素（病因特异，如肉芽肿/淋巴瘤 1,25-VitD↑）/ 血液透析（肾衰且重度）。
> 分层：重度 >14 / 中度 12–14 / 轻度 <12 mg/dL。

### 5.5 低钙体征 (signs of hypocalcemia)

低钙血症 (hypocalcemia) → 神经肌肉兴奋性↑ → Trousseau 征 + Chvostek 征（床旁两大体征）。

![[{BAB14DE7-D467-4A7D-9298-1F80006E3C7A}.png|400]]

*低钙体征：Trousseau 征（血压袖带加压诱发腕足痉挛）+ Chvostek 征（叩面神经致同侧面肌抽动）。*

### 5.6 骨软化症 (osteomalacia)

维生素 D 缺乏 (vitamin D deficiency) → 骨基质矿化不足 → 骨软化症 (osteomalacia)；影像钥匙 = 假骨折 (pseudofracture / Looser 带)。

> [!tip] 实验室指纹 + 病因（NBME11 Q054）⭐
> **典型化验**：**25-羟维生素 D (25-OH vitamin D)↓↓ + PTH↑（继发性甲旁亢）+ 碱性磷酸酶 (ALP, alkaline phosphatase)↑ + 血钙低-正常 + 磷低-正常**；临床 = 弥漫骨痛 + 近端肌无力（hip flexor）。
> **常见病因**：**吸收不良（乳糜泻 celiac disease / 胃肠术后）**、日照/摄入不足、慢性肝肾病、抗癫痫药。
> **易混（高 PTH 陷阱）**：别见高 PTH 就反射原发性甲旁亢——**原发 = 高钙 + 低磷**；本型 = **低/正常钙 + 维 D 极低 = 维 D 缺乏驱动的继发性甲旁亢**。成人叫 osteomalacia，小孩叫 rickets（佝偻病）。
> 🔁 [[NBME11_错题本#^Q054]]（celiac → 骨软化，做对但标记）

![[{93A45B65-C133-4944-A995-DD9A09E5A596}.png|400]]

*骨软化症假骨折 (osteomalacia pseudofracture / Looser 带)：垂直于皮质的线状透亮带。*

## 六、MEN 综合征（Multiple Endocrine Neoplasia）

| 综合征 | 三联（"P"记忆） | 标志 |
|:---|:---|:---|
| MEN1 | 垂体 (pituitary) + 胰腺 (pancreatic) + 甲状旁腺 (parathyroid) | 泌乳素↑ / PTH↑→钙↑ / 胰岛素瘤 (insulinoma)·胃泌素瘤 (gastrinoma) |
| MEN2A (Sipple) | 甲状腺髓样癌 (MTC) + 嗜铬细胞瘤 + 甲状旁腺增生 | RET 原癌基因；降钙素↑ |
| MEN2B | 甲状腺髓样癌 + 嗜铬细胞瘤 + 黏膜神经瘤 + 马凡样体型 (Marfanoid) | RET（更侵袭）；无甲状旁腺受累 |

钥匙：MEN2A/2B 共有 **MTC + 嗜铬细胞瘤 + RET 突变**；2A 独有甲状旁腺增生，2B 独有黏膜神经瘤 + 马凡样体型。

## 七、低血糖（Hypoglycemia）

> [!tip] Whipple 三联征 = 真性低血糖准入门槛
> 症状 + 实测血糖低 + 补糖缓解，三者闭环才进一步测 insulin / C-peptide 找病因。
> （注意区分 Whipple 三联征 [低血糖] vs Whipple 病 [*T. whipplei* 感染]）

| 病因 | 胰岛素 (insulin) | C 肽 (C-peptide) | 钥匙 |
|:---|:---|:---|:---|
| 外源胰岛素（注射） | 高 | **低** | C 肽低 = 外源 |
| 胰岛素瘤 (insulinoma) | 高 | 高 | 自身分泌 |
| 偷用磺脲类 (sulfonylurea) | 高 | 高 | 筛磺脲类血药；factitious disorder |
| 酒精性低血糖 | 低 | 低 | NADH↑ 抑糖异生 |

# 内分泌_双膦酸盐不良反应 + 骨质疏松治疗谱（ONJ 机制 / 三大不良反应 / 抗吸收 vs 促成骨）

> [!danger] 为什么建这张
> 2026-06-10 [[mistakes/uworld-mistakes_2026-06#^Q18467]]：68 岁女拔松动磨牙后**创口不愈 + 下颌暴露死骨 + 唑来膦酸 zoledronic acid** → **双膦酸盐相关颌骨坏死 ONJ**（误选牙周脓肿）。核心要想通"**为什么偏偏拔牙后、偏偏颌骨**"——答案在骨愈合机制。

---

## 一、ONJ 机制：为什么"拔牙后 + 颌骨"才出事 ⭐⭐⭐

> [!info] 一句话因果链
> 拔牙创**必须靠骨重塑才能愈合**；双膦酸盐**把破骨细胞按死** → 骨修不好 → 暴露死骨 = ONJ。

**① 拔牙后需要骨愈合（关键前提）**

牙长在**牙槽骨 alveolar bone** 里；拔掉留下**牙槽窝（骨缺损 + 黏膜破口）**，愈合靠**骨重塑 bone remodeling**：

```
拔牙 → 牙槽窝（骨缺损 + 黏膜破口）
   ↓ 血凝块 → 肉芽组织
   ↓ 骨重塑 remodeling：
   破骨细胞 osteoclast 清掉受损骨 ┐
                                  ├→ 填成新骨 + 黏膜盖好 = 愈合
   成骨细胞 osteoblast 造新骨     ┘
```

**② 双膦酸盐卡在"破骨"这一步**

- 双膦酸盐（唑来膦酸 zoledronic acid / 阿仑膦酸钠 alendronate）**抑制破骨细胞**（治骨质疏松/骨转移的原理）；**地舒单抗 denosumab（抗 RANKL）同理也致 ONJ**。
- 拔牙后本需破骨细胞清创重塑 → 药把破骨按死 → **骨不能修复 → 暴露死骨**，黏膜盖不回去。
- 加重因素：双膦酸盐**抗血管生成**（血供差、愈合更慢）；破口骨**暴露在口腔细菌**里 → 继发感染坏死。

**③ 为什么偏偏颌骨（不是四肢骨）**

| 颌骨特点 | 后果 |
|---|---|
| **全身骨重塑最快**（咀嚼应力大） | 最依赖破骨细胞 → 被抑制后受累最重 |
| 覆盖黏膜**薄、易破**（拔牙即破口） | 骨直接暴露口腔 |
| 长期泡在**口腔细菌**里 | 不愈合的骨易感染坏死 |

> [!tip] 闭环一句话
> 拔牙创**恰恰需要骨愈合**，而双膦酸盐**正好不让骨愈合** → 所以"拔牙（有创牙科操作）= ONJ 经典触发"。**指南：启动双膦酸盐前先做完拔牙等牙科处理。**

## 二、双膦酸盐三大不良反应速记 ⭐⭐

| 不良反应 | 机制 / 关联 | 招牌 stem |
|---|---|---|
| **颌骨坏死 ONJ** | 抑破骨 → 拔牙创不愈、暴露死骨 | 双膦酸盐 + **拔牙后不愈 + 下颌暴露骨**；肿瘤大剂量肠外最高危 |
| **反流性食管炎 / 食管溃疡** | 口服片剂直接刺激食管黏膜 | 口服双膦酸盐 + 胸骨后痛 / 吞咽痛 |
| **非典型股骨骨折 atypical femoral fracture** | 长期抑制重塑 → 骨过度矿化变脆 | 长期用药 + **大腿 / 腹股沟前驱痛** + 轻微外力下**股骨干横行骨折** |

> [!warning] 非典型股骨骨折（atypical femoral fracture, AFF）—— 双膦酸盐的"悖论"
> - **机制**：**长期（>3–5 年）过度抑制重塑** → 微损伤累积、骨变脆 → 应力性骨折。本是防骨质疏松骨折，长期却致此 = 悖论。
> - **部位 / 形态**：**转子下或股骨干、横行骨折**（≠ 骨质疏松典型的股骨颈骨折）；**常双侧** → 查对侧股骨。X 线见**外侧皮质增厚 / 喙突样**。
> - **前驱**：骨折前数周–数月**大腿 / 腹股沟钝痛**。
> - **处理**：停双膦酸盐 + 髓内钉固定；可用**特立帕肽 teriparatide 促愈合**。
> - **预防理念**：低-中危者用药 **3–5 年后考虑"药物假期"（drug holiday）**，平衡防骨折 vs AFF/ONJ 风险。

## 三、临床要点

> [!warning] 处理与预防
> - **ONJ 处理以保守为主**：口腔卫生 + **抗菌漱口**；必要时抗生素 ± 局部清创；避免再做有创操作。
> - **预防**：启动双膦酸盐（尤肿瘤大剂量肠外）**前先完成牙科处理**。
> - **口服片剂服法**（防食管炎）：晨起空腹、整片吞、**大量水（约 240 mL）**、服后**直立 ≥30 分钟、勿躺**。

## 四、骨质疏松治疗谱：抗吸收 vs 促成骨 ⭐⭐

> [!info] 先分两大类
> - **抗骨吸收 antiresorptive（压破骨）**：双膦酸盐、地舒单抗、雷洛昔芬
> - **促骨形成 anabolic（催成骨）**：特立帕肽、罗莫佐单抗（兼抑吸收）

| 药 | 机制 | 定位 / 何时用 | 关键不良反应 / 注意 |
|---|---|---|---|
| **双膦酸盐 bisphosphonate**（阿仑膦酸钠 alendronate 口服 / 唑来膦酸 zoledronic acid IV） | 抑破骨（抗吸收） | **一线** | ONJ / 食管炎 / AFF；3–5 年后考虑药物假期 |
| **地舒单抗 denosumab**（抗 RANKL） | 抑破骨（抗吸收） | 高危、**肾功能不全**（不经肾代谢）、双膦酸盐不耐受 | ONJ/AFF 同样；**停药 → 反跳性骨丢失 + 多发椎体骨折**（不能随便停，停需序贯双膦酸盐）；低钙血症 |
| **特立帕肽 teriparatide**（PTH 1-34 类似物） | **促成骨**（间歇 PTH 刺激成骨） | **重度 / 极高危、其他药失败** | 疗程 **≤2 年**；禁用：骨骼放疗史 / Paget 病 / 不明原因 ALP↑（骨肉瘤风险）；勿与双膦酸盐同用（钝化合成代谢） |
| **雷洛昔芬 raloxifene**（SERM） | 抗吸收（弱）+ 降乳腺癌风险 | 绝经后兼防乳腺癌 | **VTE 风险↑**、潮热加重 |
| **罗莫佐单抗 romosozumab**（抗硬骨素 sclerostin） | **促成骨 + 抑吸收**（双效） | 极高危 | **心血管事件黑框警告**（近期 MI/卒中禁用） |

> [!tip] Step 2 决策钥匙
> - **一线 = 双膦酸盐**（口服 alendronate；不耐受/依从差 → IV zoledronic acid 年 1 次）。
> - **肾功能差 / 双膦酸盐不耐受 → 地舒单抗 denosumab**（但**绝不可擅自停药**）。
> - **极重度 / 多发椎体骨折 / 抗吸收失败 → 特立帕肽（促成骨）**，疗程 ≤2 年，之后序贯抗吸收维持。
> - **绝经后 + 乳腺癌风险 → 雷洛昔芬**（但 VTE 风险）。
> - 所有方案前提：**钙 + 维生素 D 充足**。

# 内分泌_低钙血症完整体系（按 PTH 分因 / 手足搐搦 / 长 QT / 先纠镁）

> [!info] 这张的定位
> 低钙体征此前只在 [[完整笔记/专题笔记/内分泌/内分泌_HighYield速查总览]] §5.5 一笔带过。本篇做深度展开（病因按 PTH 分流 + 临床 + 急救），与 [[完整笔记/专题笔记/内分泌/内分泌_高钙血症分层与PTH路径]] 正反对照。
> **PTH** = parathyroid hormone 甲状旁腺激素；**Ca** = calcium 钙；**Mg** = magnesium 镁。

## 一句话核心

**低钙 → 神经肌肉兴奋性↑（感觉异常 / 手足搐搦 / 长 QT）**；找病因**先看 PTH**：**PTH 低 = 甲状旁腺功能减退**（术后最常见）；**PTH 高 = 代偿性继发**（维生素 D 缺乏 / 慢性肾病 CKD）。

## 一、先排"假性低钙"

> [!warning] 低白蛋白 → 总钙假性偏低（离子钙正常）⭐
> 钙近一半与白蛋白结合。**低白蛋白血症**时**总钙 (total Ca) ↓ 但离子钙 (ionized Ca) 正常 = 无症状的假性低钙**。
> **校正钙 = 测得总钙 + 0.8 ×（4.0 − 白蛋白 g/dL）**；存疑直接测**离子钙**。别给无症状的低白蛋白患者补钙。

## 二、真性低钙的病因（按 PTH 分流）

```
真性低钙血症（离子钙↓）
        │
     看 PTH
   ┌────┴───────────────┐
PTH 低/不当正常          PTH 高（继发代偿）
= 甲状旁腺功能减退        = 腺体正常、在拼命代偿
        │                       │
 ├ 术后（甲状腺/甲状旁腺      ├ 维生素 D 缺乏（低钙 + 低磷 + 高 PTH）
 │   切除）= 最常见 ⭐        ├ 慢性肾病 CKD（高磷 + 低钙 + 高 PTH）
 ├ 自身免疫性甲旁减          ├ 假性甲旁减（PTH 抵抗，PTH 高但无效）
 ├ DiGeorge（22q11，胸腺+    ├ 急性胰腺炎（脂肪皂化耗钙）
 │   甲旁腺发育不良）         ├ 肿瘤溶解综合征 / 横纹肌溶解（高磷结合钙）
 └ 低镁 Mg（抑制 PTH 分泌     └ 大量输血（枸橼酸 citrate 螯合钙）
     + 末梢抵抗）⭐
```

> [!danger] 低镁 = 最易漏的元凶 ⭐⭐
> **低镁血症 (hypomagnesemia)** 既**抑制 PTH 分泌**又**致末梢 PTH 抵抗** → 低钙。**不纠正镁，钙怎么补都补不上来。** 顽固性低钙必查并先补 Mg（常见于酗酒、腹泻、PPI 长期、利尿剂）。

## 三、临床表现

| 系统 | 表现 |
|---|---|
| **神经肌肉** | 口周 / 肢端**感觉异常 (paresthesia)**、**手足搐搦 (tetany)**、喉痉挛 (laryngospasm)、癫痫发作 |
| **床旁体征** | **Chvostek 征**（叩耳前面神经 facial nerve → 同侧面肌抽动）；**Trousseau 征**（血压袖带加压 > 收缩压 3 分钟 → 腕足痉挛，**更特异**）|
| **心脏** | **长 QT (prolonged QT)** → 尖端扭转 (torsades) 风险（与**高钙的短 QT 正相反**）|
| **慢性** | 白内障 (cataract)、基底节钙化、牙釉质发育不良、皮肤干燥 |

![[{6B1295C5-460B-41E4-98BB-D09C006ED2F0}.png|400]]
*低钙血症 (hypocalcemia) 两大床旁体征：**Trousseau 征**（袖带加压诱发腕掌屈+掌指屈的腕足痉挛）+ **Chvostek 征**（轻叩耳前面神经引起同侧面肌收缩）——均反映神经肌肉兴奋性升高。*

## 四、处理

> [!tip] 急救分流
> - **有症状（手足搐搦 / 喉痉挛 / 抽搐 / 长 QT / 离子钙重度低）** → **静脉葡萄糖酸钙 (calcium gluconate IV)** ⭐（外周首选，外渗较氯化钙安全）。
> - **同时查并纠正低镁 (Mg)** ⭐⭐——否则补钙无效。
> - **慢性 / 无症状** → **口服钙 + 骨化三醇 (calcitriol，活性维生素 D)**；查并补普通维生素 D。
> - **针对病因**：术后甲旁减长期钙 + 活性 D；CKD 走 CKD-MBD 路径（控磷 + 活性 D）。

> [!warning] 经典 stem：甲状腺术后突发口周麻 + 手足搐搦
> **甲状腺/甲状旁腺切除术后数小时–数天**出现**口周/手指麻木 + Trousseau/Chvostek 阳性 = 术后甲状旁腺功能减退致低钙** → 查离子钙/镁 + 静脉葡萄糖酸钙。**"Hungry bone syndrome（骨饥饿综合征）"**：甲旁亢术后骨大量摄钙，亦致顽固低钙。

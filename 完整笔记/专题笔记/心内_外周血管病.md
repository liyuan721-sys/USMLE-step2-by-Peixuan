# A. 缩写速查 + 分类树

## A.1 血管病大分类

```
血管病大谱
│
├── 动脉病
│   ├── 慢性
│   │   ├── PAD (Peripheral Artery Disease) 外周动脉病 ⭐
│   │   ├── PVD (Peripheral Vascular Disease) - 旧术语（包含动+静脉，避免用）
│   │   ├── CAD (Coronary Artery Disease) 冠心病
│   │   ├── CVD (Cerebrovascular Disease) 脑血管病
│   │   ├── RAS (Renal Artery Stenosis) 肾动脉狭窄
│   │   ├── MAD (Mesenteric Artery Disease) 肠系膜动脉病
│   │   ├── Leriche syndrome（主髂闭塞）
│   │   └── CLI (Critical Limb Ischemia) 重度肢体缺血 ⭐
│   ├── 急性
│   │   ├── ALI (Acute Limb Ischemia) 急性肢体缺血 ⭐
│   │   ├── ACS (Acute Coronary Syndrome) 急性冠脉综合征
│   │   └── Stroke (Acute Cerebrovascular Event) 急性脑卒中
│   ├── 炎症性
│   │   ├── GCA / TA (Giant Cell Arteritis / Temporal Arteritis)
│   │   ├── Takayasu Arteritis 大动脉炎（< 40 岁亚洲女）
│   │   ├── PAN (Polyarteritis Nodosa) 结节性多动脉炎
│   │   ├── Buerger disease / TAO (Thromboangiitis Obliterans) ⭐
│   │   └── Kawasaki Disease 川崎病
│   ├── 痉挛性
│   │   └── Raynaud Phenomenon 雷诺现象
│   └── 结构异常
│       ├── AAA (Abdominal Aortic Aneurysm) 腹主动脉瘤
│       ├── TAA (Thoracic Aortic Aneurysm) 胸主动脉瘤
│       └── Aortic Dissection 主动脉夹层
│
└── 静脉病
    ├── 急性
    │   ├── DVT (Deep Vein Thrombosis) 深静脉血栓 ⭐
    │   ├── PE (Pulmonary Embolism) 肺栓塞 ⭐
    │   ├── VTE (Venous Thromboembolism) = DVT + PE 总称
    │   ├── SVT (Superficial Vein Thrombosis) 浅静脉血栓
    │   └── Phlegmasia Cerulea Dolens
    └── 慢性
        ├── CVI (Chronic Venous Insufficiency) 慢性静脉功能不全 ⭐
        ├── Varicose veins 静脉曲张
        ├── PTS (Post-Thrombotic Syndrome) 血栓后综合征
        └── Stasis dermatitis 淤积性皮炎
```

## A.2 5 组容易混淆名词

| 易混 | 区别 |
|---|---|
| **PAD vs PVD** | PAD = 仅动脉（现代术语）；**PVD = 旧术语**（动+静脉），避免用 |
| **CLI vs ALI** | CLI = **慢性**重症（rest pain/溃疡/坏疽）；ALI = **急性**突发（6 P's, 6h 窗口）|
| **DVT vs SVT** | DVT = **深静脉**（危险 → PE）；SVT = **浅静脉**（多自限）|
| **GCA vs Takayasu** | GCA = **> 50 岁** + 颞动脉 + 大血管；Takayasu = **< 40 岁亚洲女** + 主动脉弓 |
| **Buerger sign vs Buerger disease** | Sign = 体征（抬高变白）；Disease = TAO（年轻男 + 重度吸烟）|

## A.3 命名规律

| 前缀 | 含义 |
|---|---|
| **A**- | Acute → ALI, ACS, AAA |
| **C**- | Chronic / Critical → CLI, CVI |
| **D**- | Deep → DVT |
| **P**- | Peripheral → PAD, PE |
| **S**- | Superficial → SVT |
| **T**- | Thoracic → TAA |
| **V**- | Venous → VTE |

## A.4 VTE 总称概念

```
VTE (Venous Thromboembolism) = 静脉血栓栓塞症
   ↓
分两个表现：
   ├── DVT（血栓在静脉）
   └── PE（血栓栓塞到肺动脉）
   ↓
70-90% PE 来源于 DVT
```

⭐ **临床上 DVT 和 PE 是同一谱系的两个阶段**。

# B. 评分系统集合

> [!info] 设计原则
> 评分**单独整理**，不混在疾病讲解里。讲疾病时只**链接**到这里。

## B.1 全部评分速查表

| 评分 | 用于 | 关键 cut-off |
|---|---|---|
| **ABI** | 诊断 PAD | ≤ 0.9 = PAD；> 1.3 = 钙化 → 改 TBI |
| **TBI** | ABI 不可信时 | < 0.7 = PAD |
| **Rutherford** | PAD 严重度（北美）| 0-6（4-6 = CLI）|
| **Fontaine** | PAD 严重度（欧洲）| I-IV（III-IV = CLI）|
| **Wells DVT** | DVT 临床概率 | ≥ 2 = likely / < 2 = unlikely |
| **Wells PE** | PE 临床概率 | > 4 = likely / ≤ 4 = unlikely |
| **PERC** | 低风险 PE 排除 | 8 项全阴 → 排除 PE |
| **PESI** | PE 死亡率分层 | I-V，决定门诊 vs ICU |
| **CHA₂DS₂-VASc** | AFib 卒中风险（决定抗凝）| ≥ 2 (男) / ≥ 3 (女) → 抗凝 |
| **HAS-BLED** | AFib 抗凝出血风险 | ≥ 3 = 高出血风险 |

## B.2 ABI（Ankle-Brachial Index 踝肱指数）

```
ABI = 踝部 SBP / 上臂 SBP
        ↓
≥ 1.0  正常
0.9-1.0  临界
≤ 0.9   PAD（诊断阈值）⭐
0.4-0.9  Mild-moderate PAD
< 0.4   Severe PAD / CLI ⭐
> 1.3   钙化血管（DM/CKD）→ 不可信 → 改 TBI ⭐
```

## B.3 TBI（Toe-Brachial Index 趾肱指数）

⚠️ **注意缩写歧义**：TBI 在 Neuro 是 Traumatic Brain Injury，这里是 Toe-Brachial。

```
TBI = 大拇趾 SBP / 上臂 SBP
        ↓
TBI < 0.7 = PAD（趾动脉很少钙化，结果可靠）
```

适用：DM / CKD / 钙化血管 / ABI > 1.3 时

## B.4 Rutherford 分级（PAD 严重度）

| Category | 临床 | 治疗方向 |
|---|---|---|
| **0** | 无症状 | Risk factor 控制 |
| **1** | 轻度 claudication | Exercise + 药物 |
| **2** | 中度 claudication | Exercise + 药物 |
| **3** | 重度 claudication (< 1 block 痛) | Exercise + 药物 → 顽固→revasc |
| **4** | **Rest pain** ⭐ | **Revasc** |
| **5** | 小溃疡 / 组织丧失 | **Revasc + 伤口护理** |
| **6** | 大溃疡 / 坏疽 | **Revasc 或 截肢** |

⭐ **Rutherford 4-6 = Critical Limb Ischemia (CLI)**

## B.5 Fontaine 分级（PAD 严重度，欧洲）

| Stage | 临床 |
|---|---|
| **I** | 无症状 |
| **IIa** | 轻度 claudication（> 200m）|
| **IIb** | 重度 claudication（< 200m）|
| **III** | **Rest pain** ⭐ → CLI |
| **IV** | 溃疡 / 坏疽 → CLI |

⭐ **Fontaine III-IV = CLI**

## B.6 Wells DVT 评分 ⭐⭐ 9 项

```
活动性癌症（治疗中或 6 月内）          +1
肢体瘫痪 / 近期石膏制动               +1
卧床 > 3 天 或 大手术 < 12 周         +1
深静脉走向压痛                       +1
**整条腿肿胀**                       +1
小腿肿胀 > 对侧 3 cm                  +1
凹陷性水肿（症状侧重）                +1
非静脉曲张的侧支浅静脉                +1
**有其他更可能诊断 (-2)** ⭐ 减分项！
─────────────────────────────────
≥ 2 = DVT likely → Duplex US
< 2 = DVT unlikely → D-dimer
   阳性 → US；阴性 → 排除
```

### 记忆方法（结构记忆法 ⭐⭐⭐ 推荐）

按 **4 组分类** 记忆——理解后自然记住：

```
组 1：高凝状态（"Hypercoagulability"）— 2 项
├── 活动性癌症 (+1)
└── 卧床 > 3 天 / 大手术 < 12 周 (+1)
        ↓ Virchow 三联之"高凝 + 静脉淤滞"

组 2：静脉淤滞（"Stasis"）— 1 项
└── 肢体瘫痪 / 石膏制动 (+1)
        ↓ 不动 = 血流停滞

组 3：体征（"Signs"）— 5 项
├── 深静脉走向压痛 (+1)
├── 整条腿肿胀 (+1)
├── 小腿肿胀 > 对侧 3 cm (+1)
├── 凹陷性水肿（症状侧重）(+1)
└── 非静脉曲张的侧支浅静脉 (+1)
        ↓ 看到的物理表现

组 4：临床判断（"Clinical reasoning"）— 1 项
└── **有其他更可能诊断 (-2)** ⭐ 减分
        ↓ 排他诊断（Cellulitis / Baker's cyst / 肌肉拉伤）
```

⭐ **记忆框架**：**"高凝 + 淤滞 + 体征 + 排他"** = Virchow 三联 + 鉴别诊断

### 中文口诀记忆法

> "**癌、卧、瘫、痛、肿、肿、陷、浅、它**"
> 
> 1. **癌** = 活动性癌症
> 2. **卧** = 卧床 > 3 天 / 手术 < 12 周
> 3. **瘫** = 瘫痪 / 石膏制动
> 4. **痛** = 深静脉压痛
> 5. **肿**（整）= 整条腿肿
> 6. **肿**（差）= 小腿 > 对侧 3 cm
> 7. **陷** = 凹陷性水肿
> 8. **浅** = 侧支浅静脉
> 9. **它**（-2）= 有其他更可能诊断 ⭐

⭐ 一句话：**"癌卧瘫痛肿肿陷浅，其它减 2"**

### 场景画像记忆法

> [!success] 脑内画一个"典型 DVT 病人"
> 
> **"癌症卧床老人，左腿瘫了，整条腿肿胀，小腿比右腿粗 3 cm，按下去凹陷，沿着深静脉一按就痛，皮肤上能看见侧支浅静脉。"**
> 
> 这就是 9 项中的 8 项 ✅
> 第 9 项（减 2）：医生想 **"是不是 cellulitis 或 Baker's cyst？"**

## B.7 Wells PE 评分 ⭐⭐ 7 项

```
临床 DVT 征象（单侧腿肿+压痛）          +3 ⭐
**PE 是最可能诊断**                    +3 ⭐⭐ 主观项
HR > 100                              +1.5
近期 < 4 周制动 / 手术                  +1.5
既往 DVT/PE                            +1.5
咯血                                  +1
活动性癌症                            +1
─────────────────────────────────
> 4 = PE likely → CTPA
≤ 4 = PE unlikely → D-dimer
   阳性 → CTPA；阴性 → 排除
```

### 记忆方法（按分值分组 ⭐⭐⭐ 推荐）

按 **分值 3-3-1.5-1-1** 分 3 组：

```
组 1：+3 分 / 主要项（2 项）⭐⭐
├── DVT 临床征（单侧腿肿+压痛）
└── **PE 是最可能诊断**（主观判断）⭐ PE 评分独有！

组 2：+1.5 分 / 中度风险（3 项）
├── HR > 100
├── 近期 < 4 周制动 / 手术
└── 既往 DVT / PE

组 3：+1 分 / 次要项（2 项）
├── 咯血
└── 活动性癌症
```

⭐ **助记**：**"3-3-1.5-1.5-1.5-1-1 = 大三小三半半半"**

### 中文口诀记忆法

> "**PE 最可能 + DVT 征 (+3) | HR、制动、既往 (+1.5) | 咯血、癌 (+1)**"
> 
> 或者更简单的两个 +3 项：
> 
> ⭐ **核心 = "看腿 + 想 PE"**
> - "看腿"= DVT 征 +3（单侧腿肿 + 压痛）
> - "想 PE"= 临床判断 PE 是最可能诊断 +3

### 场景画像记忆法

> [!success] 脑内画一个"典型 PE 病人"
> 
> **"癌症卧床老人，刚做完手术 2 周，曾经有 DVT 史，突然胸痛 + 咯血，HR 110，左腿单侧肿胀压痛 (DVT 征)，医生一看：'这就是 PE 没跑了！'"**
> 
> 这就是 7 项全中 ✅

⚠️ **Wells DVT vs Wells PE 是两个评分**！同名（同一个 Wells 学者），但项目不同。
- 看**腿** → DVT 评分（9 项，≥ 2 likely）
- 看**胸** → PE 评分（7 项，> 4 likely）
- **PE 评分独有"PE 是最可能诊断"主观项 +3 分** ⭐

### Wells 分层记忆策略（最实用）⭐⭐⭐

> [!success] 不要试图一次记 16 个项目，按"重要程度"分 3 层
> 
> #### Layer 1（必背 ⭐⭐⭐）— 5 分钟内能记
> 
> 1. **DVT cut-off = ≥ 2 likely**
> 2. **PE cut-off = > 4 likely**
> 3. **PE 独有"PE 是最可能诊断" +3 分**
> 4. **DVT 独有"其他更可能诊断" -2 分**
> 5. 两个评分都有 "**癌症 / DVT 史 / 体征**"
> 
> #### Layer 2（应试加分 ⭐⭐）— 1 小时内能记
> 
> 1. DVT 9 项的"**4 组分类**"（高凝/淤滞/体征/排他）
> 2. PE 7 项的"**分值分布**"（3-3-1.5-1.5-1.5-1-1）
> 
> #### Layer 3（精确记忆 ⭐）— 多次复习记
> 
> 1. 每一项的具体定义（如"小腿差 > 3 cm"）
> 2. 各种衍生评分（PERC / Geneva / PESI 等）

⭐ **临床现实**：你**永远不需要在 30 秒内算精确分值**——USMLE 题给你 stem 描述，你判断 "likely vs unlikely"，然后选下一步检查。

### USMLE 答题速记（4 句话）

> [!tip] 应试场景
> 
> ```
> 1. 看到"单侧腿肿 + 风险因素" → 想 DVT → Wells DVT
>    - 评分高 (≥ 2) → Duplex US
>    - 评分低 (< 2) → D-dimer
>    
> 2. 看到"呼吸困难 / 胸痛 + 风险因素" → 想 PE → Wells PE
>    - 评分高 (> 4) → CTPA
>    - 评分低 (≤ 4) → D-dimer
>    
> 3. 评分时心里清单：
>    - 有"癌症 / DVT 史 / 制动 / 手术"加分
>    - 有"对应体征"加分
>    - PE 时问"PE 是最可能诊断吗？" 是 → 大加 3
>    - DVT 时问"有其他更可能诊断吗？" 是 → 减 2
>    
> 4. cut-off：DVT 2 / PE 4
> ```

## B.8 PERC（PE Rule-out Criteria）

**适用**：临床低风险 PE + 想避免 D-dimer 时

```
8 项必须**全部阴性**：
1. 年龄 < 50
2. HR < 100
3. SpO₂ > 94%
4. 无单侧腿肿
5. 无咯血
6. 近期无手术 / 创伤
7. 既往无 DVT/PE
8. 无激素使用（OCP / HRT）
        ↓
全阴 + 临床低概率 → **直接排除 PE**（无需 D-dimer）
任一阳性 → 走 D-dimer 路径
```

## B.9 PESI（PE Severity Index）⭐

**用于已确诊 PE 后的死亡率分层 + 治疗场所决策**

### PESI 完整版（11 项）

```
PESI 11 项评分：

人口学（3 项）：
- 年龄                          + age (岁数)
- 男性                          +10
- 癌症                          +30

合并症（2 项）：
- 心衰                          +10
- 慢性肺病（COPD）              +10

生命体征（5 项）：
- HR ≥ 110                      +20
- SBP < 100                     +30
- RR ≥ 30                       +20
- 体温 < 36°C                   +20
- 意识改变                      +60

氧合（1 项）：
- SaO₂ < 90%                    +20
─────────────────────────────
总分 = 年龄(岁数) + 上述加分
```

### 5 个分层（Class I-V）

| Class | 总分 | 30 天死亡率 | 治疗场所 |
|---|---|---|---|
| **Class I** | < 65 | **0-1.6%** | **门诊** ⭐ |
| **Class II** | 66-85 | 1.7-3.5% | 门诊（如条件允许）|
| **Class III** | 86-105 | 3.2-7.1% | **住院**（普通病房）|
| **Class IV** | 106-125 | 4-11.4% | 住院 + 监测 |
| **Class V** | > 125 | **10-24%** | **ICU** ⭐ 考虑溶栓 |

⭐ 简化记忆：
- **Class I-II = 低危**（可门诊 / DOAC）
- **Class III-IV = 中危**（住院 + 监测）
- **Class V = 高危**（ICU + 考虑 tPA）

### sPESI（simplified PESI）⭐⭐⭐ 临床更实用

USMLE 临床上**更常用 sPESI**（简化版 6 项，每项 +1 分）：

```
sPESI 6 项：
1. 年龄 > 80                    +1
2. 癌症史                       +1
3. 慢性心肺病（CHF 或 COPD）    +1
4. HR ≥ 110                     +1
5. SBP < 100                    +1
6. SaO₂ < 90%                   +1
─────────────────────────────
0 分 = **低危**（30 天死亡 1%）→ 可门诊 / DOAC
≥ 1 分 = **高危**（30 天死亡 10%）→ 住院 + 监测
```

⭐ **sPESI 比 PESI 简单 10 倍** —— USMLE Step 2 CK 一般考 sPESI 概念

### Memory Hook

> [!success] 
> **PESI 分层 = 看 5 个东西**
> 
> 1. **年龄 > 80**（老人危险）
> 2. **癌症 / 慢性心肺病**（合并症）
> 3. **HR > 110**（心率快）
> 4. **SBP < 100**（低血压）
> 5. **SaO₂ < 90%**（低氧）

⭐ **实际临床更简单**：**血压 + 右心负荷标志 → 决定 ICU vs 普通病房**
- BP < 90 → ICU + 考虑 tPA
- BP 正常 + 右心 strain（RV dilation / Troponin↑）→ 住院监测
- BP 正常 + 无右心 strain + sPESI 0 → 门诊 DOAC

# C. 动脉病

## C.1 PAD（Peripheral Artery Disease）⭐ 必考

### Stem 锁定钩子
- 老人/中年男 + smoker / DM / HTN + **走路时小腿/大腿/臀部疼，休息缓解**
- **ABI ≤ 0.9** → 诊断
- Rest pain / ulcer / gangrene → CLI

### 临床表现谱
- **Claudication**（运动诱发痛 / 休息缓解）⭐
- **Rest pain**（夜间足趾痛，**dangling 缓解**）⭐ → CLI
- 肢端冷 / 苍白 / 毛发↓ / 肌萎缩 / **Buerger sign**（抬高变白）
- 远端溃疡 / 坏疽

### 治疗按 Rutherford 决策树 ⭐⭐

```
所有 PAD 病人（无论分级）：
  ① 戒烟 > 一切（最重要）
  ② Antiplatelet (ASA/clopidogrel) + Statin
     → 降 MI/Stroke 死亡率（不是为腿）
  ③ HbA1c < 7%、BP < 130/80
        ↓
分症状决策：
        ↓
┌────────────────────────────────────────┐
│ 无症状 / Mild-Moderate Claudication    │
│ (Rutherford 0-3, Fontaine I-IIb)       │
│ → Supervised Exercise Therapy ⭐ 首选   │
│ → 二线 Cilostazol（HF 禁用！）          │
│ → 顽固 + 严重影响生活 → 考虑 revasc    │
├────────────────────────────────────────┤
│ Critical Limb Ischemia (CLI) ⭐⭐       │
│ Rutherford 4-6, Fontaine III-IV        │
│ 定义：                                  │
│   • Rest pain > 2 周 + ABI < 0.4       │
│   • Ischemic ulcer + ABI < 0.4         │
│   • Gangrene + ABI < 0.4               │
│ 招牌钩子：                              │
│   "Dangles leg over bed at night" ⭐    │
│   (重力辅助灌注缓解疼痛)                │
│ → Revascularization **必须**            │
│   流程：CTA/MRA 找可 revasc 血管        │
│   → PTA / Stent / Bypass / 截肢         │
└────────────────────────────────────────┘
```

### USMLE 高频陷阱
1. claudication 立刻 stent → 错。先 Exercise + 药物
2. Cilostazol HF 禁用（PDE3 → 心律失常）
3. Rest pain ≠ "claudication 加重版"，是 **CLI**（阶段质变）
4. DM 病人 ABI 看起来正常（钙化假性升高）→ 改 TBI
5. 所有 rest pain 都立刻 revasc？→ 不准确，需要符合 CLI 标准 + 可耐受手术 + 可 revasc 血管
6. **PAD 用 DAPT（双抗）？→ 错！稳定 PAD 单药就够** ⭐

### PAD 抗血小板：单药 vs DAPT ⭐ 高频混淆

> [!warning] PAD 不是 MI —— 抗血小板用法不同

| 情境 | 抗血小板方案 |
|---|---|
| **稳定 PAD**（无症状 / claudication）| **单药** ⭐（**ASA 81 mg 首选** / **Clopidogrel 75 mg 替代**）|
| PAD + 既往 MI / Stroke | 通常**仍单药**（除非冠脉 PCI 内 1 年）|
| PAD + 近期下肢 revasc / Stent | **短期 DAPT 1-6 月** → 转单药 |
| PAD + 高 CV 风险 + 低出血 | 考虑 ASA 81 + **Rivaroxaban 2.5mg BID**（COMPASS）|
| **CLI**（rest pain / 溃疡 / 坏疽）| ASA + Statin 长期；Revasc 后 1-6 月 DAPT → 单药 |
| PAD + AFib | **DOAC 单药**（不加抗板，除非近期 PCI）|

⭐ **为什么不是 DAPT**：
- DAPT 对**冠脉**（白血栓）很有效（MI / PCI 后）✅
- 对**稳定 PAD** → CHARISMA / CASPAR 试验证明：**出血风险 > 获益** ❌
- 反而增加大出血并发症

⭐ **USMLE 关键认知**：
- **MI / ACS** → **DAPT**（ASA + P2Y12）⭐
- **稳定 PAD** → **单药**（ASA 或 Clopidogrel 选一）⭐
- **PAD + 近期 revasc** → 短期 DAPT → 转单药

### Buerger sign vs Buerger disease（必混淆）
- **Buerger sign** = 体征（抬高变白，下垂变红）= severe PAD
- **Buerger disease** = 独立病（TAO）= 年轻男 + 重度吸烟（详见 [[#C.4 Buerger Disease (TAO)]]）

### Claudication 4 大类鉴别（含术语澄清）⭐⭐⭐

> [!warning] 术语澄清 — Claudication 不是 PAD 专属
> 
> **"Claudication" 是症状（运动时腿痛 → 休息缓解），不是诊断。**
> 
> 临床惯用法的"**Vascular Claudication**" = "**Arterial Claudication**" = PAD（不严谨，但约定俗成）
> 
> 严格按机制，Claudication 有 **4 大类**，"Vascular" 字面应该包括动脉 + 静脉：
> 
> ```
> Claudication（症状）
>   ├── I. Vascular（血管性）
>   │   ├── A. **Arterial**（动脉性）= PAD ⭐ 最常见
>   │   └── B. **Venous**（静脉性）= PTS / May-Thurner
>   └── II. Non-vascular
>       ├── A. **Neurogenic**（神经性）= 椎管狭窄
>       └── B. **Musculoskeletal**（肌骨性）= 关节炎
> ```
> 
> ⭐ **应试默认**：USMLE 写 "vascular claudication" → 锁 **arterial/PAD**；除非 stem 明示"venous"。

#### 4 大类完整对比表 ⭐⭐⭐

| 特征 | **Arterial**（PAD）⭐ | **Neurogenic**（椎管狭窄）| **Venous**（PTS）⭐ | **MSK** |
|---|---|---|---|---|
| **大类** | 血管性-动脉 | 神经性 | 血管性-静脉 | 机械性 |
| **机制** | **缺血** | **神经根压迫** | **静脉淤滞** | 机械损伤 |
| **疼痛性质** | 缺血性 / 抽筋样 | **麻 + 乏力 + 痛** | **爆胀 / 沉重感** ⭐ | 关节痛 |
| **位置** | 小腿 / 大腿 / 臀（看动脉）| 双下肢 + 腰 | **整条腿** | 局部关节 |
| **走 X 米就痛** | ✅ 可预测 | 不一定 | 不严格 | 不严格 |
| **站着不动缓解** | ✅ **是** ⭐ | ❌ | 缓解慢 | 缓解 |
| **坐下/弯腰缓解** | 无关 | ✅ **是** ⭐ | 缓解 | 缓解 |
| **抬高肢体** | **加重** ⭐ | 无关 | **缓解** ⭐ | 无关 |
| **骑自行车** | 加重 | **不痛** ⭐ | 加重 | 看部位 |
| **远端脉搏** | **↓** ⭐ | 正常 | 正常 | 正常 |
| **ABI** | **≤ 0.9** ⭐ | 正常 | 正常 | 正常 |
| **既往史** | CV 风险因素 | 老年 / 椎管退变 | **DVT 史** ⭐ | 创伤 / 关节炎 |
| **缓解速度** | < 5 分钟 | 数分钟（坐下后）| **10-20 分钟** ⭐ | 不固定 |

#### 3 个动作判断 Claudication ⭐ 临床问诊核心

> [!success] 看到病人 "走路腿痛 + 休息缓解" → 不要直接诊断 PAD，问 3 个动作：

```
Step 1：站着不动有用吗？
  YES → **Arterial**（PAD 缺血）✅
  NO  → 继续↓
  
Step 2：要坐下/弯腰才缓解吗？
  YES → **Neurogenic**（椎管狭窄）✅
  NO  → 继续↓
  
Step 3：抬高肢体有用吗？
  YES → **Venous**（PTS / 静脉性）✅
  NO  → **MSK**（关节/肌肉）
```

⭐ **反直觉**：
- **Arterial 病人 拒绝抬腿**（重力辅助灌注 → 抬腿加重缺血）
- **Venous 病人 喜欢抬腿**（重力帮助回流 → 抬腿缓解淤滞）
- 一个升 / 一个降 → **同一动作完全相反反应**

#### Venous Claudication 详解（USMLE 容易漏的考点）⭐

> [!tip] 典型 stem
> 
> "55 岁女 + **5 年前 iliofemoral DVT** + 现在走 10 分钟后**整条腿爆胀样疼痛** + **必须抬高 15 分钟才缓解** + 内踝棕色色素 + 凹陷性水肿"
> 
> ❌ A. PAD（ABI 正常，无 CV 风险因素）
> ❌ B. 椎管狭窄（不需弯腰缓解）
> ✅ **C. Venous claudication 继发于 PTS** ⭐
> ❌ D. 髋骨关节炎

#### Arterial vs Venous Claudication 5 大反差（关键鉴别）

| 维度 | Arterial（缺血）| **Venous**（淤血）|
|---|---|---|
| 起病 | 数年慢慢加重 | **DVT 后 1-2 年突然出现** ⭐ |
| 抬高效果 | **加重** | **缓解** ⭐ |
| 疼痛性质 | 抽筋样 / 缺血样 | **爆胀样 / 沉重感** ⭐ |
| 缓解速度 | 数分钟 | **10-20 分钟** ⭐ |
| 远端脉搏 | ↓ | 正常 |

⭐ **核心反差**：**抬高肢体反应** 是 Arterial 和 Venous 的"分水岭"

#### USMLE 答题策略 4 句话

```
1. "走路腿痛+休息缓解" → 不要直接 PAD，看上下文
2. 看 ABI：≤ 0.9 = arterial / 正常 = 其他 3 类
3. 看缓解动作：站立 / 坐下 / 抬高 — 锁定机制
4. 看既往史：DVT 史 → 警惕 venous
```

#### Memory Hook

> [!success] 
> **"Claudication 4 兄弟，3 动作判断"**
> 
> 4 类病因：
> - **Arterial** ⭐ 最常见
> - **Neurogenic**
> - **Venous** ⭐ 容易漏
> - **MSK**
> 
> 3 个动作：
> 1. **站着不动** → Arterial 缓解
> 2. **坐下弯腰** → Neurogenic 缓解
> 3. **抬高肢体** → Venous 缓解 / Arterial 加重

## C.3 急性肢体缺血 (Acute Limb Ischemia, ALI) ⭐ 急症

### 6 P's 必背
- **P**ain（剧烈）
- **P**allor（苍白）
- **P**ulselessness（无脉）
- **P**aresthesias（感觉异常）
- **P**oikilothermia（皮温变冷）
- **P**aralysis（瘫痪 - 晚期，**6 小时窗口**！）

### 病因（修正：不只是栓塞）
```
ALI 病因：
├── **栓塞** (~30%)
│   └── AFib → 左房血栓（最常见 ⭐）
│   └── 心室壁血栓（post-MI）
│   └── 主动脉夹层
└── **原位血栓** (~70%) ⭐ 实际更常见
    └── 既往 PAD 急性恶化
    └── 动脉粥样硬化斑块破裂
    └── Bypass 移植物血栓
    └── 创伤
```

### 治疗：黄金 6 小时窗口 ⭐⭐⭐

#### 时间窗 vs 保肢预后

| 时间窗 | 处理 | 保肢预后 |
|---|---|---|
| **< 6 h** | 立刻 IV Heparin + 紧急 revascularization | **保肢成功率高** |
| **6-24 h** | 仍尝试血运重建 | 保肢率显著下降 |
| **> 6 h** | 进入不可逆肌肉损伤区 | **截肢风险显著升高** + 再灌注损伤更重 |

> [!danger] 一句话锁定
> **6 h 内血运重建 → 保肢成功率高；> 6 h → 肌肉坏死 + 再灌注损伤 + 截肢风险显著升高**。Hard signs 阳性 = 时间敏感 = **跳过影像直接 OR**。

#### 处理流程

```
1. **立刻 IV Heparin** ⭐
   - 防止血栓**继续扩展**
   - 防止远端继发血栓
   - 维持侧支血管通畅
2. **紧急 revascularization**：
   - Surgical embolectomy / thrombectomy
   - Catheter-directed thrombolysis（tPA）
   - Bypass
3. 找病因：
   - 栓塞性 → 长期抗凝（如 AFib → DOAC 终身）
   - 原位血栓 → 抗板 + Statin + Revasc 维持
```

#### 再灌注损伤（Reperfusion Injury）⭐

血运重建恢复血流后，缺血肌肉的代谢产物 + 细胞内容物涌入循环 → 全身性后果：

| 并发症 | 机制 | 处理 |
|---|---|---|
| **Compartment syndrome** | 缺血肌肉肿胀 + 筋膜限制 → 室内压↑ → 二次缺血 | **Fasciotomy** |
| **横纹肌溶解** | 肌细胞坏死释放 myoglobin | 大量补液 + 监测 CK / 尿色 |
| **高钾血症** | 细胞内 K 释放入循环 | 警惕**心律失常**（VF / 心脏骤停）|
| **代谢性酸中毒** | 乳酸 + H+ 累积 | NaHCO3 + 监测 |
| **AKI** | Myoglobin 致急性肾小管坏死 | 大量补液 + 碱化尿液 |

### Rutherford ALI 分级 ⭐⭐⭐（决定紧急程度）

| 分级 | 感觉 | 运动 | Doppler 动脉 | Doppler 静脉 | 处理 |
|---|---|---|---|---|---|
| **I** Viable | 完整 | 完整 | 有 | 有 | 计划性血运重建（数小时-数天内）|
| **IIa** Marginally threatened | 轻↓（脚趾）| 完整 | 多无 | 有 | **紧急血运重建**（数小时内）|
| **IIb** Immediately threatened | ↓ + 痛 | ↓ | 无 | 有 | ⭐ **Emergent OR**（跳过影像）|
| **III** Irreversible | 感觉 / 运动皆无 + 大理石样 + 肌硬 | 麻痹 | 无 | 无 | **截肢**（不可救）|

### Hard Signs 决策（血管损伤 + ALI 共享原则）⭐⭐⭐

> [!danger] Hard Signs 阳性 → 跳过影像直接 OR

**Hard signs（任一阳性 = emergent OR，不做 CTA）**：
- Active / pulsatile hemorrhage（活动性 / 搏动性出血）
- **Pulsatile / expanding hematoma**（搏动性 / 扩张性血肿）
- Bruit / thrill 在伤口处（提示 AV fistula 或假性动脉瘤）
- Pulse deficit（远端脉搏缺失）
- **Distal ischemia / 6 P's**（远端缺血）

**Soft signs（做 CTA / ABI 进一步评估）**：
- 病史出血但已停
- 临近大动脉的伤口
- 神经损伤
- 不解释的低血压
- **ABI < 0.9**（可疑 occult 血管损伤）

> [!tip] 核心原则
> **Hard signs 阳性 = 时间敏感 = 跳过影像直接 OR**。CTA / 影像是为 soft signs + 病情未明且稳定的患者准备的。Rutherford IIb 同理 — 影像会延误窗口。

> [!example]- 同主题错题
> - [[mistakes/uworld-mistakes#^Q3354]] 肢体血管创伤 — 4B's of Hard vs HDBN of Soft + IEI 决策（反向阴性体征陷阱：feeble + cool 组合 = 整体升级为 Hard）

## C.4 Buerger Disease (TAO, Thromboangiitis Obliterans)

### 经典画像
- 年轻男（20-40 岁）
- **重度吸烟史** ⭐
- 远端肢体缺血（手指 / 足趾）
- 节段性血管炎症 + 血栓

### 治疗
- ⭐ **戒烟唯一有效**（不戒烟 → 截肢）
- 抗血小板 / 抗凝**无效**

## C.5 Raynaud 现象

### 三相色变
**白 → 蓝 → 红**（小动脉痉挛 → 缺血 → 反应性充血）

### 治疗
1. 避免诱因（冷 / 应激）+ 保暖手套
2. **CCB（Nifedipine / Amlodipine）⭐** 一线药物
3. α-blocker / SSRI / PDE5i / Prostacyclin（severe）
4. 严重溃疡 → IV Iloprost / 手术

### Primary vs Secondary Raynaud
- Primary = 单纯 Raynaud（年轻女，对称，无 ANA）
- Secondary = 合并 CTD（**Scleroderma / SLE / MCTD**，可不对称）

# D. 静脉病

## D.1 DVT（深静脉血栓）⭐⭐ 必考急症

### 诊断流程
```
怀疑 DVT（单侧腿肿胀/疼痛/红/Homans）
        ↓
Wells DVT 评分（详见 [[#B.6 Wells DVT 评分]]）
        ↓
┌─────────────┬─────────────────┐
< 2 (unlikely)  ≥ 2 (likely)
↓             ↓
D-dimer       **Duplex US** ⭐ 首选
↓             ↓
阴性→排除     找到 DVT → 确诊
阳性→US       未找到 → serial US 1 周
```

### Homans Sign（霍曼斯征）⭐ 不可靠，但 USMLE 会出现

#### 操作方法
```
病人仰卧位，膝关节伸直
        ↓
医生**快速背屈病人的足部**（dorsiflexion）
        ↓
观察反应：
- 阳性：腓肠肌 / 小腿后部**疼痛**
- 阴性：无疼痛
```

⭐ 物理原理：背屈足踝 → 牵拉小腿后侧深静脉 + 腓肠肌 → 如有血栓 → 触动血栓周围炎症 → 疼痛

#### 为什么不再常规使用 ⚠️

> [!warning] Homans sign 的临床问题
> 
> 1. **敏感性低**（30-50%）→ 大量**假阴性**（漏诊）
> 2. **特异性低**（~40%）→ 大量**假阳性**（误诊）
> 3. **可能挤压血栓** → 理论上栓子脱落 → PE 风险（实际罕见）
> 4. **不影响诊断决策** —— 不管阳性阴性都要做 Duplex US
> 
> ⭐ 现代指南**不推荐**常规使用 Homans sign

#### 现代 DVT 体征更重要

| 体征 | 临床意义 |
|---|---|
| **单侧腿肿胀** ⭐ | DVT 最常见 |
| **小腿差 > 3 cm** | Wells 评分项 |
| **凹陷性水肿**（症状侧重）| Wells 评分项 |
| **深静脉走向压痛** | Wells 评分项 |
| **皮温↑ + 红斑** | 炎症标志 |
| **非静脉曲张的侧支浅静脉** | Wells 评分项 |
| **Homans sign** | ⚠️ **不可靠，不常规用** |

#### USMLE 答题策略

> [!tip] 看到 Homans+ 怎么办
> 
> - 看到 stem 写 "**Homans sign positive**" → **不是诊断依据**
> - 仍需走标准流程：**Wells + D-dimer / Duplex US**
> - 不要直接基于 Homans+ 抗凝

经典题型：
> "45 岁女 + 单侧腿肿胀 + Homans sign positive"
> 
> "下一步最佳处理？"
> 
> ❌ A. 立刻抗凝（基于 Homans+）
> ✅ **B. Wells DVT 评分 + Duplex US** ⭐
> ❌ C. 直接 D-dimer

⭐ Memory Hook：**"Homans = 历史遗物，不靠谱"**——临床看到当个**次要支持证据**，**永远不作为金标准**

### 治疗按位置
```
近端 DVT（腘静脉以上） → **抗凝必须**
   ↓
首选：**DOAC**（Rivaroxaban / Apixaban）⭐ 现在一线
   - 无需 bridging / 无需 INR 监测
替代：LMWH (Enoxaparin) → Warfarin (INR 2-3)
   
远端 DVT（小腿） → 监测 (serial US 2 周) 或 抗凝（如症状重）

特殊情况：
- **妊娠** → **LMWH**（不能 DOAC/Warfarin）⭐
- **癌症相关** → DOAC > LMWH > Warfarin
- **严重肾衰** → UFH (IV)
- **抗凝禁忌** → **IVC filter**（不替代抗凝）
- **Phlegmasia cerulea dolens** → CDT/取栓（详见 [[#D.5 Phlegmasia Cerulea Dolens]]）
```

### 疗程
- **诱发因素**（术后/制动）→ **3 个月**
- **无诱发因素** → ≥ 3 个月，考虑长期
- **复发 / 易栓症 / 癌症** → **终身**

### 复发性 DVT (Recurrent DVT) 决策树 ⭐⭐⭐ 高频考点

```
Recurrent DVT（复发性深静脉血栓）
        ↓
是否正在抗凝？
        ↓
┌──────────────────────┬──────────────────────┐
NO（首次复发）           YES（抗凝中仍复发）⭐ 高频考点
↓                      ↓
开始 DOAC / LMWH         先评估为什么失败
（标准抗凝起步）          ↓
                       ┌──────────────┬──────────────┐
                       依从性差         **真治疗失败**
                       (病人没按时吃)    (病人严格依从)
                       ↓               ↓
                       **教育 + 工具**   **换成 LMWH** ⭐
                       - 闹钟           （不是加大原药剂量）
                       - Pill box       ↓
                       - 每日 1 次       **长期 / 终身抗凝**
                          DOAC          ↓
                       - 家属监督        同时找隐藏病因：
                       ↓               - 隐匿性癌症（最重要）⭐
                       仍差 → 考虑      - 易栓症（APS / Factor V Leiden）
                       LMWH + 护士     - 解剖原因（May-Thurner）
                       或 Warfarin
```

⭐ **关键区分**：
- **病人严格依从但仍复发** → 切换 LMWH（真治疗失败）⭐
- **病人没按时吃药** → 先教育 + 改善依从性（LMWH 不是首选）
- **依从性 OK + 失败的可能原因**：剂量 / 隐藏病因（癌症 / APS）

#### 为什么"抗凝中复发 → 切换到 LMWH"？

**抗凝中复发的可能原因**：
1. **当前药物失败 / 治疗剂量不足**
   - DOAC 剂量不够 / 病人没按时吃
   - Warfarin INR 不在治疗范围（2-3）
2. **隐藏的高凝病因**：
   - **癌症（Trousseau syndrome）⭐ 最常见**
   - 易栓症（APS / Factor V Leiden）
   - 妊娠 / OCP
3. **机械性原因**（如 May-Thurner 综合征）

**为什么 LMWH 是首选**：
- **LMWH 在癌症相关 VTE 中证据最强**（CLOT 试验 2003）
- **LMWH 比 Warfarin 更稳定**（不需 INR 监测）
- **LMWH 比 DOAC 在癌症 + 复发场景更有效**
- LMWH 剂量可精确调整

⚠️ **更新临床实践**（2018 年后）：
- 部分指南允许癌症相关 VTE 用 **DOAC**（Edoxaban / Apixaban）
- 但**复发场景**仍倾向 LMWH

#### 复发性 DVT 临床路径（4 步）

```
Step 1：确认是真复发
- Duplex US 看是不是"新血栓"
- 区分：真复发 vs PTS (Post-thrombotic syndrome 旧伤后遗症)

Step 2：检查抗凝依从性 + 治疗剂量
- DOAC：剂量对吗？（按 CrCl 调整？）
- Warfarin：INR 在 2-3 吗？
- **如果依从性差**：⚠️ **不是直接换 LMWH**！
  ↓
  **先尝试改善依从性**：
  - 闹钟 / pill organizer
  - 每日 1 次 DOAC（如 Rivaroxaban 优于 BID）
  - 家属监督
  - 教育病人重要性
  - 调整剂量（如有副作用）
  ↓
  **仍不能改善** + 有支持系统（家属 / 护士）：
  - **LMWH**（需要每日注射 + 监护人）
  - 或 **Warfarin** + INR 监测（监督感强）
  
  ⚠️ **LMWH 不是"治疗依从性差"的首选** —— 注射依从性反而比口服更差！

- **如果依从性 OK + 真治疗失败** → **直接切换 LMWH**（详见下方决策）

Step 3：找隐藏病因 ⭐ 关键
- **癌症筛查**（最重要）：
  - 年龄相关：colonoscopy / mammogram / PSA
  - CT 胸腹盆（如老年 / 不明原因 VTE）
  - 体重↓ / 食欲↓ → 警惕 Trousseau
- **易栓症筛查**：
  - **APS**（抗凝中复发的经典原因）⭐
  - Factor V Leiden / Prothrombin G20210A
  - Protein C/S / Antithrombin III 缺乏
- **解剖原因**：
  - May-Thurner 综合征（左髂静脉受右髂动脉压迫）
  - IVC 异常

Step 4：长期管理
- **终身抗凝** ⭐（特别如癌症 / APS）
- 首选 **LMWH**（如癌症相关）
- 次选 DOAC / Warfarin
- + 病因治疗（化疗 / APS 长期管理）
- + 监测出血风险（HAS-BLED）
```

#### Trousseau Syndrome（癌症相关 VTE）⭐

```
Trousseau syndrome：
- 癌症（特别是腺癌：**胰腺 / 肺 / 胃 / 卵巢 / 结肠**）⭐
- 高凝状态
- **反复发作的 superficial / deep / 内脏血栓**
        ↓
临床警示：
- **抗凝中仍反复血栓** ⭐
- **不明原因的 DVT / PE**
- **多部位血栓**（如 DVT + 上肢血栓同时）
        ↓
治疗：
- **LMWH 首选**（CLOT 试验循证）
- 治疗原发癌
```

⭐ **Armand Trousseau 自己**就是用自己的血栓性静脉炎诊断出胰腺癌——抗凝中复发或不明原因 VTE 是癌症筛查的**强烈信号**。

#### USMLE 经典题型

**题型 1：抗凝中复发 + 病人严格依从**
> "65 岁女 + 6 月前 DVT + **严格规律服用 Apixaban**（每日两次）+ 现对侧腿新发肿胀 + US 见新血栓"
> 
> "下一步治疗？"
> 
> ❌ A. 增加 Apixaban 剂量
> ✅ **B. 切换到 LMWH（Enoxaparin）** ⭐
> ❌ C. 加用 ASA
> ❌ D. 维持 Apixaban + 观察

⭐ **关键 stem 词**：**"严格规律服用"** = 真治疗失败 → 换 LMWH

**题型 1 变体：抗凝中复发 + 依从性差**
> "65 岁女 + 6 月前 DVT + Apixaban + **承认经常忘记服药 2-3 次/周** + 现对侧腿新发血栓"
> 
> "下一步最佳治疗？"
> 
> ❌ A. 切换到 LMWH（直接换药）
> ✅ **B. 教育 + 改用每日 1 次 Rivaroxaban + 家属监督** ⭐
> ❌ C. 增加 Apixaban 剂量
> ❌ D. IVC filter

⭐ **关键 stem 词**：**"经常忘记服药"** = 依从性问题 → 先改善依从性，**不是直接换 LMWH**

**题型 2：抗凝中复发 + 体重↓（找癌）**
> "70 岁男 + 不明原因 DVT 6 月前 + **严格服用** Apixaban + 新发 PE + **体重↓ 10 kg** + 食欲↓"
> 
> "下一步最关键检查？"
> 
> ✅ **C. 隐匿性癌症筛查（CT 胸腹盆 + 内镜）** ⭐

**题型 3：APS 抗凝失败**
> "32 岁女 + SLE + 2 年前 DVT + Warfarin INR 2.5 + 新发 PE + Lupus anticoagulant 阳性"
> 
> 诊断：**Antiphospholipid syndrome (APS)** → **切换 LMWH** 或 Warfarin（更高 INR）
> 
> ⚠️ **APS 患者 DOAC 失败率高** → 不推荐 DOAC

#### Recurrent DVT Memory Hook ⭐

> [!success] 
> **"抗凝中复发 = 换药 + 找癌"**
> 
> 3 个动作：
> 1. ✅ 确认真复发（Duplex US，不是 PTS）
> 2. ✅ **切换 LMWH**（不是加大原药剂量）⭐
> 3. ✅ **找癌症 + 易栓症**（特别 APS / Trousseau）
> 4. ✅ **终身抗凝**

> [!tip] 
> **"DOAC 失败的 3 大原因"**
> - 依从性差（最常见）
> - 隐匿性癌症 ⭐
> - APS（DOAC 在 APS 中失败率高）⭐

### Post-Thrombotic Syndrome (PTS, 血栓后综合征) ⭐⭐⭐ DVT 慢性后遗症

> [!warning] DVT 不仅是急性病——PTS 是它的"残疾后遗症"
> 
> **20-50% DVT 病人会发展为 PTS** —— 不是少数！

#### 定义
DVT 急性期过后，**深静脉永久性瓣膜损伤 + 血流动力学异常**，导致**慢性下肢静脉淤滞综合征**。

#### 病理生理

```
急性 DVT
   ↓
血栓机化 / 部分溶解（数周-月）
   ↓
即使抗凝成功 → 静脉**永久性结构损伤**
   ↓
┌──────────────────┬──────────────────┐
深静脉瓣膜破坏        深静脉狭窄
↓                  ↓
**Reflux**（反流）  **Obstruction**（梗阻）
   ↓
静脉高压（venous hypertension）
   ↓
组织液漏出 + 慢性炎症 + RBC 外渗
   ↓
PTS 临床表现
```

⭐ **核心认知**：PTS 病人的静脉瓣膜**永久受损**，不是"还有血栓残留"——抗凝**不能逆转**。

#### 临床表现（轻 → 重）

| 程度 | 表现 |
|---|---|
| **轻度** | 久站后小腿酸胀 + 轻度水肿（白天加重，抬高缓解）|
| **中度** | 持续凹陷性水肿 + **棕色色素沉着** (hemosiderin) + 静脉曲张 + 湿疹 |
| **重度** | **Lipodermatosclerosis**（皮下脂肪硬化）+ **Venous ulcer**（不愈合，gaiter area）⭐ |
| 终末 | "**Inverted champagne bottle leg**"（倒置香槟瓶腿）⭐ |

⭐ **Venous claudication** 也是 PTS 表现之一（见 [[#Claudication 4 大类鉴别（含术语澄清）⭐|Claudication 4 类鉴别]]）

#### 流行病学（USMLE 必记）

| 数据 | 数值 |
|---|---|
| DVT 后 PTS 发生率 | **20-50%** ⭐ |
| 严重 PTS（含 venous ulcer）| ~5-10% |
| 发病时间 | **DVT 后 1-2 年内**最常见 |
| **Iliofemoral DVT PTS 风险** | **60-70%** ⭐ |
| 远端 DVT PTS 风险 | 20-30% |

#### 治疗：预防 > 治疗 ⭐⭐⭐

```
1. 急性 DVT 处理决定 PTS 风险：
   - **充分抗凝**（至少 3 月）
   - **早期活动**（不要长期卧床）
   - **Iliofemoral DVT → 考虑 CDT**（见下方）⭐
   - **加压袜**（数据有争议但常用）
   
2. PTS 已发生 → 症状管理（不可逆 ⚠️）：
   - **加压袜 30-40 mmHg** ⭐ 核心
   - **抬高下肢**
   - 运动 + 减重（小腿肌肉泵）
   - Venous ulcer → Compression + 伤口护理
   - 严重 → 静脉手术 / 支架
```

⚠️ **PTS 不需要长期抗凝**（除非有持续血栓风险）——抗凝**不能逆转 PTS**。

#### PTS vs DVT 复发鉴别 ⭐ 高频陷阱

| 特征 | **PTS** | **DVT 复发** |
|---|---|---|
| 起病 | **慢性 + 进展** | **急性**（数小时-天）|
| 颜色 | **棕色色素** | 红 / 紫 |
| 水肿 | 慢性凹陷性 | **急性张力性** |
| 皮温 | 通常正常 | **↑** |
| **D-dimer** | 通常正常 | **升高** ⭐ |
| **Duplex US** | 旧血栓 / 反流 | **新血栓** ⭐ |
| 治疗 | Compression + 症状管理 | 抗凝升级 + 找原因 |

⭐ DVT 病人新发腿肿 → **必须 Duplex US** 区分 PTS 加重 vs 复发

### CDT (Catheter-Directed Thrombolysis) 与 PTS 预防 ⭐⭐⭐

> [!info] 为什么 iliofemoral DVT 要考虑 CDT？

#### 为什么"普通抗凝"不能预防 PTS？

```
普通抗凝（DOAC / LMWH / Warfarin）：
✅ 阻止血栓扩展
✅ 防新血栓
✅ 降 PE 风险
❌ **不溶现有血栓**（靠身体慢慢机化数月）
❌ **不保护瓣膜**（机化过程中瓣膜被破坏）
❌ **不防静脉狭窄**
   ↓
抗凝完美 → 血栓机化 → 瓣膜受损 → **PTS 必然形成**
```

⭐ **核心认知**：
- **抗凝** = "**等血栓自己消化**"
- **CDT** = "**主动把血栓清掉**" → 保护瓣膜

#### CDT 操作方法

```
1. 穿刺静脉（对侧股或腘静脉）
2. 导管穿过血栓
3. 导管末端开多个小孔（infusion catheter）
4. 持续输注 tPA（如 Alteplase）数小时-48h
5. 可同时机械碎栓 / 取栓（Pharmaco-mechanical thrombolysis）
6. 造影查残余狭窄
7. 如有狭窄 → 支架植入（特别 May-Thurner）
```

⭐ **技术优势**：
- tPA **局部高浓度** → 溶栓效率高
- **全身剂量低** → 出血风险低于全身溶栓
- **可同时处理解剖问题**（+ 支架）

#### CDT vs 全身溶栓对比

| 维度 | **Systemic tPA** | **CDT** |
|---|---|---|
| tPA 剂量 | 高（100mg）| 低（0.5-1 mg/h × 24h）|
| 出血风险 | **高** ⚠️ | 低 |
| 颅内出血 | 1-2% | < 1% |
| 疗效 | 中等 | **更好**（局部高浓度）|
| 支架配合 | ❌ | ✅ |
| DVT 适应 | 罕用 | ⭐ 选择性 iliofemoral |

⭐ DVT **几乎不用全身溶栓**（出血 > 获益）—— 如果用溶栓 = **CDT**

#### 为什么 iliofemoral DVT 特别适合 CDT？

```
原因 1：解剖学（"主干道"被堵）
- 髂股 = 下肢回流主干道
- 一旦堵了，没有备用通道
- 远端 DVT 仍有侧支

原因 2：血栓量大 + 瓣膜密集
- 近端静脉粗大 → 血栓量大
- 瓣膜多 → 损伤后果重
- PTS 风险：60-70%（远端仅 20-30%）

原因 3：常合并 May-Thurner（25-50%）
- 解剖压迫底层问题
- 光抗凝不解决 → 反复 DVT + PTS
- **CDT + 支架** = 一次解决两个问题
```

#### ATTRACT 试验（USMLE 必知）⭐

> 2017 NEJM 发表，评估 CDT 减少 PTS 效果

```
研究：692 个急性近端 DVT
随机：CDT + 抗凝 vs 仅抗凝
终点：2 年 PTS

结果：
- 整体人群：CDT 没显著减少 PTS（47% vs 48%）⚠️
- **iliofemoral 亚组**：
  - 中重度 PTS：CDT 18% vs 抗凝 28% ⭐
  - 显著获益
- 远端 DVT（股腘）：CDT 无明显获益
   ↓
**指南更新**：
- 远端 DVT → 仅抗凝
- **Iliofemoral + 年轻 + 症状重 → 考虑 CDT** ⭐
```

#### CDT 适应症（"4 个 yes"）

```
理想候选人：
1. **Iliofemoral 位置**（不是远端）✅
2. **症状重**（重度肿胀 / 剧痛）✅
3. **病程 < 14 天**（血栓未机化）✅
4. **预期寿命长**（年轻 / 健康）✅

+ 无禁忌：
- 近期手术 / 出血 / 颅内病变
- 妊娠 / 严重 HTN
- 凝血功能异常
- 高龄（个体化）
```

⭐ **决策原则**：**获益（防 PTS）> 风险（出血）** 才推荐

#### CDT 的两大目标

| 目标 | 时机 | 临床场景 |
|---|---|---|
| **急性保肢** | 数小时 | **PCD**（蓝色腿急症）|
| **长期防 PTS** | 14 天内最佳 | **Iliofemoral DVT**（未 PCD）|

⭐ 同一工具，两个不同时机和目标

#### USMLE 经典题型

**题型 1：CDT 适应症**
> "32 岁女 + 左侧髂股 DVT + OCP + 重度肿胀 + 病程 5 天 + 无出血风险"
> 
> ✅ **CDT + 评估 May-Thurner（可能支架）**

**题型 2：远端 DVT 不用 CDT**
> "腘静脉 DVT + 中度症状"
> 
> ✅ **仅抗凝**（ATTRACT 显示远端 DVT 无 CDT 获益）

**题型 3：CDT 禁忌**
> "78 岁 + 3 周前结肠手术 + 髂股 DVT"
> 
> ✅ **仅抗凝**（出血风险 > 获益）

### DVT 三大并发症（3 个 P）⭐ 终极汇总

> [!success] **"3 个 P"** 是 DVT 的 3 大并发症 — 一急两慢

| 并发症 | 时间 | 表现 | 治疗 | 死亡率 |
|---|---|---|---|---|
| **PE**（肺栓塞）| **急性**（数天）| 呼吸困难 / 胸痛 / 咯血 | 抗凝 ± tPA | Massive PE 30% |
| **PCD**（蓝色腿）| **急性**（数小时）| 整条腿青紫 + 脉弱 + 冰冷 | **CDT + 抗凝** | **20-40%** |
| **PTS**（血栓后综合征）| **慢性**（1-2 年）| 棕色色素 + Venous ulcer | **加压袜**（不可逆）| < 1%（残疾）|

⭐ **CDT 同时预防 PCD（急性）+ PTS（慢性）** —— 这就是为什么 iliofemoral DVT 越来越推荐 CDT。

#### Memory Hook ⭐

> [!tip] 
> **"DVT 3 P 并发症 + CDT 双重作用"**
> 
> - **PE** = 肺急性 → tPA / 抗凝
> - **PCD** = 肢体急性 → **CDT** + 抗凝
> - **PTS** = 肢体慢性 → 加压袜（不可逆）
> 
> ⭐ **"主干道（iliofemoral）堵了 → 尽快疏通（CDT）；小路（远端）堵了 → 让它自己机化"**

## D.2 PE（肺栓塞）⭐⭐ 必考急症

### 诊断流程（看血流动力学）

```
怀疑 PE
        ↓
血流动力学稳定？
        ↓
┌──────────────────────┬──────────────────────┐
**不稳定**             稳定
(BP < 90/休克)        ↓
↓                      Wells PE 评分
床旁回声立刻 ⭐         (详见 [[#B.7 Wells PE 评分]])
看右心：               ↓
- RV dilation           ┌──────────┬──────────┐
- RV hypokinesis        ≤ 4         > 4
- D-shaped LV           ↓           ↓
- McConnell sign        D-dimer     CTPA 直接
↓                      ↓
经验性 tPA              阳性→CTPA
+ 抗凝桥接              阴性→排除
```

### Massive PE 完整体征（USMLE 高频）⭐

```
床旁体征：
- SBP < 90 / 休克
- **JVD（颈静脉怒张）** ⭐
- Right-sided S3 / S4 奔马律
- Pulsus paradoxus（吸气 SBP 下降 > 10）
- Hepatojugular reflux
- 肢端冰冷 / 苍白 / 意识改变

影像 / 实验室：
- **床旁回声** ⭐⭐：
  • **RV dilation**（新发右心扩张）
  • **RV hypokinesis**（运动减弱）
  • **McConnell sign**（RV free wall 不动，apex 仍动）= 急性 PE 特异
  • **D-shaped LV**（室间隔向左移）
  • TR 加重 + PASP 升高
- **CTPA**：
  • **RV/LV 比 > 0.9** ⭐ 提示 RV strain
  • 主肺动脉鞍状血栓（saddle embolus）= massive
- **Troponin ↑**（RV 微梗死）
- **BNP / NT-proBNP ↑**（心室壁应力）
- **ECG**：
  • 窦速（最常见）
  • **S1Q3T3**（仅 20% 出现）⭐
  • **新发 RBBB** ⭐ ← RV 扩张牵拉右束支
  • 右心轴偏 / T 倒置 V1-V4
```

⭐ 看到 **新发 RBBB + 呼吸困难 + 低血压** → 0.5 秒锁 Massive PE → tPA 准备

### PE 风险分层 ⭐ 决定治疗

| 类型 | 定义 | 死亡率 | 治疗 |
|---|---|---|---|
| **Massive (高风险)** | SBP < 90 ≥ 15 min / 休克 / 心搏骤停 | > 30% | **tPA 必须** / 取栓 |
| **Submassive (中风险)** | BP 正常 + RV 障碍标志（≥ 1 项）| 3-15% | 抗凝为主 + 个体化考虑溶栓 |
| **Low-risk** | 稳定 + 无 RV 受累 | < 1% | 抗凝（DOAC），可门诊 |

**Submassive PE 的 RV 障碍标志**：
- 回声：RV dilation / hypokinesis
- CTPA：RV/LV > 0.9
- BNP / NT-proBNP ↑
- Troponin ↑（RV 微损伤）
- ECG：新发 RBBB / S1Q3T3

### 治疗按风险
```
Low-risk → DOAC，门诊或短住
Submassive → 住院 + DOAC/LMWH + 监测右心 / 心肌酶
              部分病例 → 评估溶栓（个体化）
Massive → ICU + **tPA**（溶栓禁忌 → 取栓）+ 抗凝桥接
```

### USMLE 经典 stem
> "65 岁女 + 大手术后 3 天 + 突发呼吸困难 + BP 78/40 + HR 130 + **JVD** + 床旁回声 **RV dilation + hypokinesis** + ECG **new RBBB**"
> 
> → **Massive PE** → **tPA 立刻**

## D.3 CVI（Chronic Venous Insufficiency）

### 机制
```
深静脉瓣膜失功 / 血栓后综合征
   ↓
血液淤滞 → 静脉高压
   ↓
铁质渗出 → **棕色色素沉着** (hemosiderin)
脂肪硬化 → **lipodermatosclerosis**
   ↓
**Venous ulcer**（内踝上方，gaiter area）
```

### 临床
- 久站加重 / **抬高缓解** ⭐
- 凹陷性水肿
- 棕色色素 / 静脉曲张
- 内踝上方溃疡（不规则、浅、渗液多、轻中度痛）

### 治疗
```
Step 1: 保守治疗
  ① **Compression therapy** ⭐⭐⭐ 核心（30-40 mmHg）
     ⚠️ **必须先查 ABI** 排除合并 PAD
     - ABI > 0.8 → 安全 compression
     - ABI 0.5-0.8 → 低压 compression
     - ABI < 0.5 → **禁用**（动脉缺血优先治）
  ② **抬高下肢**（休息时高于心脏）
  ③ 减重 / 运动（小腿肌肉泵）
  ④ 皮肤护理

Step 2: 溃疡处理
  - Compression（70% 4-6 月愈合）
  - Wound care（保湿敷料）
  - 感染 → 抗生素

Step 3: 顽固 / 严重
  - 静脉手术（valve repair / ligation）
  - 射频 / 激光消融（RFA / EVLA）
  - 硬化剂注射
```

⭐ **USMLE 黄金钩子**：**Compression therapy 前必查 ABI**

## D.4 Varicose Veins（静脉曲张）

### 治疗阶梯
1. 一线：弹力袜 + 抬高 + 减重 + 运动
2. 二线：硬化剂注射 / RFA / 激光消融
3. 三线：手术（stripping / ligation）

## D.5 Phlegmasia Cerulea Dolens ⭐ 罕见但必考

### 名词拆解
- Phlegmasia = 炎症
- Cerulea = 蓝色
- Dolens = 疼痛
- = **"蓝色 + 疼痛性的静脉炎"**

### 病理生理（核心）

> [!warning] PCD 不是"哪个具体血管的病"，是**临床综合征**
> 
> 不要被"髂股 DVT" 限定住 —— PCD 的本质是**广泛 + 完全的近端深静脉闭塞**，**髂股部位最常见（>90%）但不绝对**。

```
PCD 发生条件（3 个）：
1. 深静脉系统**完全闭塞**
2. **侧支静脉**也大部分闭塞
3. 血栓累及范围足够**广泛**
   ↓
静脉回流彻底中断
   ↓
组织间质压力↑↑↑
   ↓
动脉灌注被 venous hypertension 压迫
   ↓
**继发性动脉缺血** ⭐ ← 这才是危险所在
   ↓
肢体青紫 + 剧痛 + 巨大水肿
   ↓
Compartment syndrome / 肢体坏死 / 截肢
   ↓
**死亡率 20-40%**（多脏器衰竭）
```

⭐ **关键概念**：**静脉病引起的动脉缺血**——静脉淤血严重到压迫动脉。

### PCD 可发生部位

| 部位 | 频率 | 临床特点 |
|---|---|---|
| **髂股 DVT (iliofemoral)** | **>90%** ⭐⭐⭐ 最常见 | 整条腿肿+青紫+剧痛 |
| **股 + 腘静脉广泛闭塞** | 较少 | 大腿+小腿肿，需多条静脉一起累及 |
| **下腔静脉 (IVC) 闭塞** | 罕见 | **双侧**腿同时 PCD（罕见但致命）|
| **上肢广泛 DVT** | **极罕见 (<5%)** | 透析 / 起搏器 / Paget-Schroetter 极严重型 |
| 单独腘静脉以下 | **不引起 PCD** | 侧支足够代偿 |

⭐ **核心解剖逻辑**：**位置越近端、范围越大、侧支越少** → 越易发生 PCD

⭐ **为什么髂股是高发地**：
- 髂股层面侧支选择不多
- 整条腿主要回流通道
- 一堵就"瀑布式蔓延"

### 临床钩子

> [!warning] USMLE Stem 招牌写法
> 
> "65 岁女 + 卵巢癌 + 化疗中 + 突发左下肢**剧痛 + 巨大肿胀 + 紫色**，足部冰冷，**足背动脉脉搏微弱**，足趾发紫"

| Stem 特征 | 解析 |
|---|---|
| **极度肿胀 + 整条腿** | 完全静脉回流障碍 |
| **紫色 / 青紫** | venous congestion → 还原血红蛋白↑ |
| **剧痛**（远超普通 DVT）| 张力 + 缺血 |
| **足背动脉脉搏弱 / 冰冷** ⭐ | 继发动脉灌注差 |
| **可有水疱 / 起泡** | 严重组织损伤 |
| **诱因**：癌症 / 易栓症 / IVC filter 血栓 / May-Thurner 综合征 | 必有 |

### Phlegmasia Alba vs Cerulea
```
Phlegmasia ALBA dolens（白色）
- 早期 / 程度较轻
- 血栓闭塞深静脉但**侧支还工作**
- 反射性动脉痉挛 → 肢体**苍白**
- 仍然剧痛 + 水肿
   ↓
进展 →
   ↓
Phlegmasia CERULEA dolens（蓝色）⭐
- 严重晚期
- 侧支也闭塞 → 静脉淤血 → cyanosis
- **继发动脉缺血** ← 急症
```

### 治疗 ⭐ USMLE 必考
```
急症处理：
1. 抬高肢体（**有限**——平衡缺血风险）
2. **IV Heparin 立刻**
3. **取栓 + 溶栓**（catheter-directed thrombolysis, CDT）⭐
   或手术取栓（CDT 禁忌时）
4. **Fasciotomy**（如 compartment syndrome）
5. 找诱因：**癌症筛查必须**
6. 后续：IVC filter 评估 + 长期抗凝
```

⚠️ **关键**：普通 DVT → 抗凝足够；Phlegmasia → **抗凝 + 溶栓/取栓**（罕见的 DVT 溶栓适应症）

### 锁定句
> "DVT 史 / 癌症 + 突发整条腿剧痛 + 青紫 + 巨大水肿 + **足脉弱**" → **Phlegmasia Cerulea Dolens** → IV Heparin + CDT/取栓

### PCD vs 普通近端 DVT 鉴别表 ⭐ 高频混淆

| 特征 | 普通近端 DVT | **PCD** |
|---|---|---|
| 静脉系统 | 部分闭塞 | **完全 + 广泛闭塞** |
| 侧支 | 部分工作 | **大部分也闭塞** |
| 水肿 | 中度 | **巨大、张力性** |
| 颜色 | 正常 / 略红 | **青紫 (cyanosis)** ⭐ |
| 疼痛 | 中度 | **剧痛**（缺血样）|
| **动脉脉搏** | **正常** | **弱 / 消失** ⭐ |
| 皮温 | 正常 / 温暖 | **冰冷** ⭐ |
| 死亡率 | < 1% | **20-40%** |
| 治疗 | 抗凝 | **抗凝 + 溶栓 / 取栓** ⭐ |

⭐ **PCD 升级标志（看 stem 判断）**：**动脉脉搏弱 + 肢端冰冷 + 青紫** = 不只是严重 DVT，而是 PCD

### 顺手延伸：May-Thurner 综合征 ⭐ 髂股 DVT 经典病因

> [!info] 经典 USMLE 考点
> 
> **May-Thurner 综合征** = **右髂总动脉跨过左髂总静脉，长期压迫使其顶在 L5 腰椎上**

#### 精确解剖（必背）

```
解剖关系（俯视图）：

           ┌──────────────┐
           │  腹主动脉      │
           └──────┬───────┘
                  ↓ 分叉（在 L4 水平）
        ┌─────────┴─────────┐
        ↓                   ↓
   左髂总动脉            **右髂总动脉** ⭐（罪魁）
   (Left CIA)          (Right CIA)
        │                   │
        │                   ↓ 跨过
        │             ┌─────────────┐
        │             │ **左髂总静脉**│ ⭐（受害者）
        │             │ (Left CIV)  │
        │             └──────┬──────┘
        │                    ↓
        │                  顶在 L5 腰椎上 ⭐
        ↓
   下肢回流
```

⭐ **核心解剖事实**（为什么是左 vein 被压）：
- **下腔静脉（IVC）位于身体右侧** → 左髂静脉要"**绕到右侧**"才能汇入 IVC
- 这条**左髂静脉路径**正好穿过右髂动脉下方
- 右髂动脉**脉搏跳动** + 压迫 → 左髂静脉**被夹在动脉和 L5 腰椎之间**

⭐ **关键 3 步**：
```
右髂总动脉（脉搏跳动）⬇
   ↓
**左髂总静脉**（被压）
   ↓
**L5 腰椎**（坚硬底面）⬆
   ↓
内皮损伤 + 静脉淤滞 → "Spur"（纤维条索）→ DVT
```

#### 罕见变异（< 5%）⚠️ USMLE 可能考

| 类型 | 解剖 | 临床 |
|---|---|---|
| **典型 May-Thurner**（>90%）| 右髂动脉 → 压迫 → **左髂静脉** | **左下肢 DVT** ⭐ |
| **Right-sided May-Thurner**（罕见）| 左髂动脉 → 压迫 → 右髂静脉 | 右下肢 DVT |

#### 临床钩子
- **年轻女性 / 中年女性**（**20-30 岁第 2-3 个十年**最易）⭐
- **左下肢反复 DVT** ⭐（**不对称** — 左 >> 右）
- 通常无明显诱因（其他 DVT 风险因素不突出）
- 长跑 / 久坐 / 脱水 / OCP 加重

#### 流行病学
| 特征 | 数据 |
|---|---|
| 解剖变异（无症状）| **20% 普通人群**（autopsy） |
| 占下肢 DVT 的 | **2-5%** |
| 性别 | **女 > 男** |
| 年龄 | **20-30 岁** 高发 |
| 侧别 | **左侧 > 90%** |

⭐ **核心认知**：20% 的人都有这个变异，但**只有少数发病**——需要**额外诱因**（脱水 / 长期卧床 / OCP）才触发血栓。

#### 诊断
- **Venogram / CT venogram / MR venogram** 见左髂静脉狭窄
- US 有时可见血流模式异常（但敏感性低）
- IVUS（血管内超声）金标准

#### 治疗
- **抗凝**（标准 VTE 治疗）+ 一定时间
- + **支架植入左髂静脉**（机械解决压迫）⭐ 关键
- 严重 / PCD → CDT/取栓 + 后续支架

⭐ **USMLE 钩子**：**"年轻女性 + 反复左下肢 DVT + 无明显 risk factor"** → **May-Thurner** → 抗凝 + **左髂静脉支架**

### Iliofemoral DVT 自身考点 ⭐ 不止 PCD

| 特征 | 临床意义 |
|---|---|
| **位置** | 髂总 / 髂外 / 股总静脉 |
| **PE 风险** | **比远端 DVT 高 3-4 倍** ⭐ |
| **PTS 风险** | 60-70%（远端 DVT 仅 20-30%）|
| **可能引起 PCD** | 是（>90% PCD 来自此）|
| **常见病因** | **May-Thurner** / 癌症 / IVC 异常 |
| **治疗** | 抗凝必须；考虑 **CDT** 减少 PTS |

## D.6 SVT（Superficial Vein Thrombosis）

- 通常**自限**
- NSAIDs + 局部热敷 + compression
- **延伸到大隐静脉近端** → 抗凝（低剂量 LMWH）

# E. 动脉病 vs 静脉病治疗对比 ⭐⭐⭐ USMLE 必背

## E.1 核心原则
- **动脉病** = 缺血 → 治疗目标 **改善供血**
- **静脉病** = 淤血 → 治疗目标 **促进回流**
- ⚠️ **动脉病禁忌压迫**（加重缺血 = 截肢风险）

## E.2 5 大操作对比

| 操作 | 动脉病 | 静脉病 |
|---|---|---|
| **抬高肢体** | ❌ 加重疼痛（Buerger sign）| ✅ 缓解症状 |
| **压迫 (Compression)** | ❌ **绝对禁忌** ⭐⭐⭐ | ✅ **核心治疗** |
| **抗血小板** | ✅ 首选（ASA/Clopidogrel）| ❌ 几乎无效 |
| **抗凝** | 仅特殊情况（ALI / AFib + PAD）| ✅ 首选 |
| **运动** | ✅ Supervised Exercise（核心）| 辅助 |
| **溶栓** | ALI 偶用 | **不稳定 PE 救命** ⭐ |

## E.3 血栓性质（机制基础）

- **动脉血栓** = **白血栓**（血小板为主）→ 抗血小板
- **静脉血栓** = **红血栓**（纤维蛋白 + RBC）→ 抗凝

## E.4 USMLE 5 大经典陷阱

1. **Venous ulcer 直接 compression** → 错，先查 **ABI** 排除 PAD
2. **DVT 妊娠用 Warfarin / DOAC** → 错，**只能 LMWH**
3. **PAD claudication 直接 stent** → 错，先 **戒烟 + ASA + Statin + Exercise**
4. **DVT 抗凝禁忌就放弃** → 错，放 **IVC filter** 防 PE
5. **ALI 慢慢做影像确诊** → 错，**立刻 IV Heparin + 手术**（6h 窗口）

## E.5 Memory Hook
> "**动脉怕压，静脉爱压**"
> "**动脉抬了更痛，静脉抬了就好**"
> "**动脉缺血静脉淤，treatment 反着搞**"

# F. DVT vs PE 治疗区别原理

## F.1 谱系关系

```
DVT 和 PE 是同一谱系（VTE）的两个阶段
        ↓
DVT = 血栓在深静脉
        ↓
脱落 → 下腔静脉 → 右心 → 肺动脉
        ↓
PE = 血栓嵌顿肺动脉
        ↓
轻中度 = 缺氧 / 心率↑
        ↓
**Massive PE**：
  RV 急性扩张 → 室间隔向左移
  → LV 充盈↓ → 心输出量↓
  → 休克 / 死亡 ⭐
```

⭐ **核心区别**：
- **DVT** = 血栓在**外周静脉** → 主要风险 = **未来 PE**
- **PE** = 血栓在**肺动脉** → **当下心肺衰竭风险**
- 所以 PE 多了一个考量：**血流动力学**

## F.2 治疗对比表

| 维度 | DVT | PE |
|---|---|---|
| **核心治疗** | 抗凝 | 抗凝 + 评估稳定性 |
| **首选药** | DOAC | 同 DVT |
| **疗程** | 诱发 3 月 / 无诱发 ≥ 3 月 / 复发终身 | 同 DVT |
| **溶栓 (tPA)** | **不常规用** ⚠️ | **不稳定 PE 必用** ⭐ |
| **取栓** | 严重髂股 DVT (phlegmasia) | 溶栓禁忌的不稳定 PE |
| **IVC filter** | 抗凝禁忌时 | 抗凝禁忌时 |
| **额外评估** | 找诱因（癌症/易栓症）| **PESI 评分** → 决定门诊 vs 住院 |

## F.3 关键差异

### 差异 1：溶栓角色不同
- **DVT** 几乎不溶栓（血栓自然机化 + 出血风险大）
  - **例外**：Phlegmasia / 严重髂股 DVT 年轻患者（减少 PTS）
- **PE** 看血流动力学：
  - 稳定 → 抗凝
  - **Submassive**（RV strain）→ 个体化
  - **Massive**（休克）→ **tPA 必须** ⭐

### 差异 2：PE 有死亡风险分层（PESI）
PE 直接致死 → 需要 PESI 分级 → 决定门诊 vs ICU
DVT 不直接致死 → 没有这种分层

### 差异 3：监测重点
| 项 | DVT | PE |
|---|---|---|
| 监测 | Duplex US 复查 / 找诱因 | **回声**（评估 RV）/ Troponin / BNP |
| 急诊指标 | 一般无 | **HR / BP / SpO₂ / Shock index** |

## F.4 一句话总结

> [!success] 
> **DVT** = 慢性威胁（"防止它变 PE"）→ **稳态抗凝**
> **PE** = 即时威胁（"现在能死人"）→ **抗凝 + 看血流动力学决定要不要溶栓**

# H. 易混陷阱合集

## H.1 缩写歧义

| 缩写 | 含义 1 | 含义 2 |
|---|---|---|
| **TBI** | Toe-Brachial Index（PAD）| Traumatic Brain Injury（Neuro）|
| **AD** | Alzheimer Disease | Autosomal Dominant |
| **MS** | Multiple Sclerosis | Mitral Stenosis |
| **PE** | Pulmonary Embolism | Physical Exam / Preeclampsia |
| **HR** | Heart Rate | Hazard Ratio |
| **PVD** | 旧术语（含动+静脉），避免使用 | |

## H.2 同人不同评分

| 评分 | 区别 |
|---|---|
| **Wells DVT** | 9 项，≥ 2 likely → US |
| **Wells PE** | 7 项，> 4 likely → CTPA。**独有"PE 是最可能诊断" +3 主观项** |

⚠️ 看到 stem 写"Wells score = X" 不说清楚 → 看上下文：腿 → DVT；胸 → PE。

## H.3 治疗哲学陷阱

1. "动脉病用 compression" → ❌ 致命
2. "DVT 妊娠用 DOAC" → ❌ 只能 LMWH
3. "Ischemic Stroke 急性立刻抗凝" → ❌ 24h 后才用 ASA
4. "MI 立刻溶栓" → ❌ 首选 PCI 90 min，tPA 仅无 PCI 时
5. "Massive PE 等 CTPA 确诊" → ❌ 不稳定 → tPA 经验性给
6. "ALI 等影像" → ❌ 立刻 IV Heparin，6h 窗口
7. "Cilostazol 用于 HF 患者的 PAD" → ❌ 禁忌
8. "Phlegmasia 抗凝就够" → ❌ 必须 CDT/取栓
9. "Buerger disease 抗血小板" → ❌ 唯一有效 = 戒烟
10. "所有 rest pain 都立刻 revasc" → ❌ 看 CLI 标准 + 可耐受 + 可 revasc 血管

## H.4 时间窗

| 急症 | 时间窗 | 错过后果 |
|---|---|---|
| MI - PCI | **90 min**（door-to-balloon）| 心肌坏死扩大 |
| MI - tPA | < 12 h | 再灌注无益 |
| Stroke - tPA | **4.5 h** | 出血风险↑↑↑ |
| Stroke - 取栓 | **24 h**（LVO 选择性）| 神经损伤不可逆 |
| ALI | **6 h** | 肌肉坏死不可逆 |
| Phlegmasia | 数小时-天 | 截肢 / 死亡 |

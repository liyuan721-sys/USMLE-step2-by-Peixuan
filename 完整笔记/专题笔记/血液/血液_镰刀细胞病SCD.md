# 镰刀细胞病 SCD 全套（Sickle Cell Disease）

> [!info] 笔记定位
> SCD 是 Step 2 CK 高频"枢纽病"——同一个病不同问法考**完全不同**的病原/处理。本笔记把分散在 [[完整笔记/Peixuan分科笔记/hematology oncology]] 的 SCD 内容压成一页纸，**重点强化"无脾感染套餐"与"同病不同问法"对照**。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q3640]]（无脾脓毒症 → 肺炎链球菌）+ [[mistakes/uworld-mistakes_2026-06#^Q3571]]（骨髓炎 → 金葡 + Salmonella 双覆盖）。

---

## 零、一句话核心 ⭐⭐⭐

> [!danger] SCD 同一个病、三种问法、三组病原（最易混 → 你今天连考两道）
> | 问法 | 关键线索 | 病原 / 处理 |
> |---|---|---|
> | **脓毒症 / 发热** | 发热 + 低压 + 白细胞↑ + bandemia，**无骨痛** | **肺炎链球菌（荚膜菌第一）** → 经验 ceftriaxone（± vanc 覆耐药） |
> | **骨髓炎** | **有骨痛定位** + 局部红肿 + 血培养⊕ | **S. aureus + Salmonella 双覆盖** → 抗葡萄球菌(clinda/vanc) ＋ 三代头孢 |
> | **预防（无脾）** | 慢病管理题 | 肺炎球菌疫苗 + **青霉素至 ≥5 岁** + Hib/脑膜炎球菌疫苗 + 叶酸 + 羟基脲 |

---

## 一、本质与诊断

**SCD = 溶血（Hemolysis）+ 血管阻塞（Vaso-occlusion）两条路。**

- **基因**：常染色体隐性（AR），β 球蛋白点突变，**谷氨酸 → 缬氨酸（Glu → Val）**。
- **触发**：缺氧、脱水、感染、酸中毒、寒冷 → HbS 聚合 → 镰变 → 既堵又破。
- **起病**：**6 月龄**起症状（HbF 让位 HbS 后）；**~5–6 岁前**反复脾梗死 → **功能性无脾（autosplenectomy）**。

| 检查 | 内容 |
|---|---|
| **初筛** | CBC + 外周涂片（镰状 RBC、**Howell-Jolly 小体 = 无脾标志**）+ 网织红↑ |
| **确诊** | **血红蛋白电泳（hemoglobin electrophoresis）** |

> [!tip] 电泳"有没有 HbA"是关键
> 正常 = HbA 最多；**Sickle trait = HbA + HbS（仍有 A）**；**SCD/HbSS = 无 HbA**，HbS 主 + 少量 HbF；HbSC = HbS + HbC（无 A）；**Hydroxyurea → HbF↑**。

口诀：**「缺氧脱水感染冷，HbS 一聚就堵崩。」**

## 二、无脾与感染套餐 ⭐⭐⭐（本笔记重点）

> [!danger] 功能性无脾 = SCD 感染风险的总根源
> 反复脾梗死 → 无脾 → **荚膜菌（encapsulated）清除障碍** → 暴发性脓毒症。**脾切除/无脾后败血风险可持续 >30 年**。

### 2.1 感染谱（按部位记病原）

| 部位 | 最常见病原 |
|---|---|
| **脓毒症 / 菌血症** | **S. pneumoniae（第一）** > H. influenzae b > N. meningitidis（皆荚膜菌） |
| **肺炎** | S. pneumoniae |
| **脑膜炎** | S. pneumoniae |
| **骨髓炎 / 化脓性关节炎** | **S. aureus + Salmonella species（SCD 特异高频）** |

> [!warning] 脓毒症 vs 骨髓炎——病原分水岭看"骨痛"
> - **无骨痛 + 全身脓毒症** → 荚膜菌（**肺炎链球菌**第一）→ [[mistakes/uworld-mistakes_2026-06#^Q3640]]
> - **有骨痛定位** → **S. aureus + Salmonella 双覆盖**（不能因"SCD = Salmonella"漏掉金葡）→ [[mistakes/uworld-mistakes_2026-06#^Q3571]]

### 2.2 SCD 骨髓炎经验抗生素（双覆盖）

| 人群 | 病原 | 经验方案 |
|---|---|---|
| 健康儿童·低 MRSA | S. aureus | Nafcillin/oxacillin 或 cefazolin |
| 健康儿童·高 MRSA | MRSA | Clindamycin 或 vancomycin |
| **SCD 儿童** | **S. aureus + Salmonella** | 抗葡萄球菌(clinda/vanc) **＋ 三代头孢(ceftriaxone/cefotaxime)** |

> [!tip] "抗葡萄球菌"选药：**MSSA → nafcillin/oxacillin/cefazolin（首选）；MRSA/重过敏 → vanc/clindamycin**（nafcillin 只覆 MSSA 不覆 MRSA；万古能覆 MRSA 但治 MSSA 反不如 nafcillin）。详见 [[完整笔记/专题笔记/感染/感染_皮肤软组织感染分层鉴别]] §三。

> [!tip] 为什么单覆盖一边都错
> Salmonella 只是"相对健康儿童更突出"，**S. aureus 仍是 SCD 骨髓炎重要病原**；经验期不能赌单一病原 → **双覆盖**，出培养再降阶。

> [!danger] 缺口在金葡，不在 Salmonella（最易绕错的点）
> **Ceftriaxone（三代头孢）本身就同时覆盖肺炎链球菌 + Salmonella**（跨 G⁺/G⁻ 谱够宽），所以 Salmonella 这条线它一个药就管了。
> **但 ceftriaxone 不可靠覆盖 S. aureus（MSSA 非首选、MRSA 完全不覆盖）** → 这才是要补 **clindamycin/vancomycin** 的真正原因。
>
> | 药 | 肺炎链球菌 | Salmonella | 金葡(含 MRSA) |
> |---|---|---|---|
> | **Ceftriaxone(3 代)** | ✅ | ✅ | ❌(MRSA 不覆) |
> | **Cefazolin(1 代)** | ✅ | ❌ | ✅(MSSA) |
> | **Clindamycin/Vanc** | — | ❌ | ✅(含 MRSA) |
>
> 所以 SCD 骨髓炎不能指望单一头孢通吃：**ceftriaxone 补 Salmonella + clinda/vanc 补金葡**。反过来用 cefazolin(覆金葡却漏 Salmonella)也不行。详见 [[完整笔记/专题笔记/感染/感染_头孢菌素分代速查]] §2.3。

### 2.3 预防套餐（无脾患者长期管理）⭐

```
SCD 无脾感染预防 = 疫苗 + 青霉素 + 叶酸 + 羟基脲
├─ 疫苗：肺炎球菌(PCV13 + PPSV23) + Hib + 脑膜炎球菌（针对荚膜菌）+ 年度流感
├─ 青霉素预防：口服 PCN，到 ≥5 岁（penicillin prophylaxis until age 5）
├─ 叶酸补充（慢性溶血消耗）
└─ Hydroxyurea（反复 VOC / ACS → ↑HbF）
```

> [!warning] 高频考点：发热即按败血处理
> SCD 患儿发热来急诊 = **先抽血培养 + 立刻经验广谱抗生素（ceftriaxone）**，**别等结果**。本例若**未服青霉素预防**正是中招诱因（[[mistakes/uworld-mistakes_2026-06#^Q3640]]）。

## 三、急性危象谱（按系统）

### 3.1 两大"急性重度贫血"（看网织红 + 脾）

| 危象 | 诱因 | 网织红 | 体征 | 处理 |
|---|---|---|---|---|
| **Aplastic crisis（再障危象）** | **Parvovirus B19**（暂停红系） | **↓（关键）** | Hb 骤降、脾不一定大 | 输血支持 |
| **Splenic sequestration（脾扣押）** | 血被扣在脾 | **↑** | **脾迅速增大 + 休克 + 血小板↓** | 等渗补液 + RBC（**谨慎/可半量，防回流高黏滞**）；反复 → 脾切除 |

口诀：**「retic 低 = 工厂停（Parvo）；脾大休克 retic 高 = 血被扣（sequestration），输血别猛冲。」**

### 3.2 Vaso-occlusive 疼痛危象（VOC）

- 表现：急性剧痛常多部位（**dactylitis 6 月–4 岁**手足对称肿痛）、± 低热、常有脱水诱因。
- 处理：**补液 + 镇痛（急诊够量 IV 阿片 ± NSAID）+ 氧（低氧时）± 输血（看指征）**。
- **无发热**：水 + O₂ + 吗啡；**有发热**：**加 ceftriaxone + azithromycin**（无法区分肺炎/感染诱因）。

### 3.3 Acute Chest Syndrome（ACS）

- 诊断：**新肺部浸润（CXR）+ ≥1（发热/胸痛/咳喘/低氧/呼吸功↑）**。
- 触发：儿童多**感染**；成人多**脂肪栓塞**（骨髓坏死）；疼痛 → 肺不张 → 缺氧 → 更镰变（恶性循环）。
- 治疗：**Ceftriaxone + Azithromycin** + O₂ + 补液（别过量）+ 镇痛 + incentive spirometry；**重症 → 交换输血**。

### 3.4 卒中（Stroke）

- 缺血（血管病变 + 镰红黏附）/ 出血（血管壁弱 + 动脉瘤）。影像：**缺血 MRI / 出血 CT**。
- 治疗：**交换输血**（首选），不能则简单输血。
- **预防（儿童）**：**经颅多普勒 TCD 筛查** + 高危慢性输血 ± 羟基脲。

### 3.5 骨痛三鉴别（VOC vs 骨髓炎 vs AVN）

| | 疼痛 | 发热/炎症 | 其他 |
|---|---|---|---|
| **VOC** | 急性多部位 | 低热可有 | 脱水诱因，血培养多阴 |
| **Osteomyelitis** | 局灶持续渐进 | **发热明显且久** | 红肿热 + **血培养⊕**；Salmonella/Staph |
| **AVN** | **慢性加重**（股骨头） | **无发热** | 无红肿热 |

## 四、其他高频

- **肾**：Hyposthenuria（髓质 vasa recta 镰变 → 浓缩差、夜尿多尿，**血钠多正常**）。**Sickle trait 最常见并发症 = 无痛性血尿**。
- **Sickle cell trait**：杂合，**极端缺氧才镰变**（高空飞行/脱水酒精/消防员）→ 轻度血管内溶血、**脾梗死**（高原/酒精后 LUQ 痛）。
- **慢性呼吸困难**三鉴别：哮喘（可逆阻塞）/ 肺动脉高压（慢性溶血耗 NO → 血管重塑）/ 肺纤维化（限制性、蜂窝）。
- **妊娠 SCD**：基线 24h 尿蛋白 + 反复尿培养；**叶酸 + 低剂量阿司匹林 + 肺炎球菌疫苗**；系列胎儿生长超声（防 FGR）；并发子痫前期/胎盘早剥/早产。

## 五、急诊分流算法 ⭐

```
SCD 来急诊：先分层 → 有无"器官危象"
  有 → 救命路径（换血/输血/抗生素/ICU）
  无 → 疼痛危象路径（补液 + 镇痛）

四大红旗（必须先处理）
  A 发热/疑败血    → 抽血培养 + 立刻经验抗生素(ceftriaxone)，别等结果
  B ACS(胸痛/低氧/新浸润) → O₂ + ceftriaxone + azithromycin，重 → 交换输血
  C 卒中/神经缺损  → CT 排出血 + MRI 看缺血 → 交换输血
  D 急性重度贫血   → retic↓=Parvo再障(输血)；脾大休克=脾扣押(补液+RBC谨慎)
```

## 六、Memory Hook（一条串全图）

> **「SCD 两条路：溶血 + 堵。**
> **发热先当败血 → 肺炎球菌最常见（没吃青霉素就中招）；有骨痛才想骨髓炎 → 金葡 + Salmonella 双覆盖。**
> **急性重贫血：retic↓ = Parvo 再障；脾大休克 retic↑ = 脾扣押（输血别猛）。**
> **ACS 新浸润 + 呼吸/发热/低氧 → 头孢 + 阿奇，重就换血。卒中 → 换血。**
> **维持：疫苗 + 青霉素到 5 岁 + 叶酸 + 羟脲（HbF↑）。」**

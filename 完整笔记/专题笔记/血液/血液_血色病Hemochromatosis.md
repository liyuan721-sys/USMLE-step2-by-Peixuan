# 血色病 Hemochromatosis（铁过载）

> [!info] 一句话定位
> **Hereditary hemochromatosis (遗传性血色病) = HFE 基因缺陷 → 肠铁吸收过度 → 铁沉积多器官**。经典"**青铜色糖尿病 (bronze diabetes)**" = 皮肤色素沉着 + 糖尿病。男性多见、女性因月经失血发病晚。

---

## 一、遗传 + 机制

- **遗传**：**常染色体隐性 (autosomal recessive)**；最常见突变 **HFE C282Y**（H63D 次之）。〔C282Y 为标准补充〕
- **机制**（来源：分科 [[完整笔记/Peixuan分科笔记/hematology oncology]]）：
```
HFE 缺陷 → 肝 hepcidin 铁调素 不适当地↓
        ↓ ferroportin 铁转运蛋白 持续开放
肠道铁吸收↑ + 巨噬细胞释铁↑ → 血铁↑ → 实质器官铁沉积
```

---

## 二、经典多系统五联（铁沉积处即受损）

```
铁沉积地图
├── 肝  ：肝大 → 肝硬化 → 肝细胞癌 HCC（高危）
├── 胰  ：胰岛破坏 → 糖尿病 DM
├── 皮肤：古铜色 / 灰褐色色素沉着（黑色素 + 铁）
├── 心  ：限制型 / 扩张型心肌病 + 传导异常
├── 关节：关节病（第 2/3 掌指关节）+ 软骨钙化 chondrocalcinosis（继发假性痛风）
└── 性腺：继发性性腺功能减退（垂体铁沉积 → 促性腺激素↓）± 甲减
```

> [!tip] 记忆锁定
> **"青铜糖尿病 (bronze diabetes)" = 皮肤古铜 + DM；再加肝硬化 = 经典三联**。中年男性**关节痛 + 皮肤变深 + 血糖高 + 肝酶高** → 查铁。

> [!warning] 易感特定感染（铁是细菌养料，分科重点）
> 铁过载者对**嗜铁菌**易感：**创伤弧菌 Vibrio vulnificus（生海鲜/海水）、李斯特菌 Listeria、小肠结肠炎耶尔森菌 Yersinia enterocolitica**。（来源：分科 hematology）

---

## 三、诊断

| 步骤 | 指标 | 说明 |
|---|---|---|
| **首选筛查** | **转铁蛋白饱和度 (transferrin saturation)↑↑** | 最早异常、最敏感 |
| 储铁 | **血清铁蛋白 ferritin↑** + **TIBC (总铁结合力)↓** | ferritin 也是急性期蛋白，需结合饱和度 |
| 确诊基因 | **HFE 基因检测 (C282Y)** | 饱和度 + ferritin 高时 |
| 最准确（评纤维化）| **肝活检** 或 MRI 定量铁 | 评肝损/肝硬化 |

（来源：分科 hematology / [[完整笔记/Peixuan分科笔记/消化腺]]）

---

## 四、治疗

> [!success] 一线 = 放血
> - **治疗性放血 (therapeutic phlebotomy)** 一线，最有效（定期放血去铁）。
> - **不能放血者（如贫血/继发性铁过载）→ 铁螯合剂**：**去铁胺 deferoxamine**（静脉/皮下）或口服 **地拉罗司 deferasirox / 去铁酮 deferiprone**。〔后两者标准补充〕
> - 并发症对症：DM 用胰岛素、心肌病/性腺替代；**肝硬化者筛查 HCC**。
> - **避免维生素 C 大剂量 + 酒**（增加铁吸收/肝损）。〔标准补充〕

> [!info] 继发性血色病 (secondary)
> **反复输血**（β地中海贫血 thalassemia major、骨髓增生异常综合征 MDS 等输血依赖）→ 铁过载 → 用**螯合剂**（不能放血，因本身贫血）。（来源：分科 hematology）

> [!question]- 自测：52 岁男，乏力 + 双手第 2/3 掌指关节痛 + 皮肤变深 + 新发糖尿病 + 转氨酶轻升。最敏感的初筛指标？确诊基因？一线治疗？为何要避生海鲜？
> 初筛 = **转铁蛋白饱和度↑↑**（+ ferritin↑）；确诊 = **HFE C282Y 基因**；一线 = **治疗性放血 phlebotomy**；避生海鲜因铁过载易感**创伤弧菌 Vibrio vulnificus** 致命脓毒症。

> [!question]- 自测：β地中海贫血重型长期输血患儿出现心衰 + 内分泌障碍。铁过载该用放血还是螯合？为什么？
> **用铁螯合剂（去铁胺 deferoxamine 等），不能放血** —— 患者本身重度贫血，放血会加重。这是**继发性血色病**。

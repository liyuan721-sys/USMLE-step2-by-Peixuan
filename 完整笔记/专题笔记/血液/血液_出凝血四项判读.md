# 出凝血四项判读 — BT / PT / PTT / 血小板（4 维度 → 诊断）

> [!info] 用途
> Step 2 出血题的核心反射：拿到 **PT / PTT / BT（或血小板）** 四个轴的"正常 vs 延长"组合 → 锁定病因。先把**正常值**和**通路归属**背死，再练四维组合表。

---

## 一、正常值速查 ★

| 检验 | 全称 | 正常值 | 反映 |
|---|---|---|---|
| **BT** | Bleeding Time（出血时间） | **2–7 分钟** | **初级止血 primary hemostasis**（血小板功能 + 血管） |
| **PT** | Prothrombin Time（凝血酶原时间） | **11–15 秒** | **外源 extrinsic + 共同 common** 通路 |
| **INR** | International Normalized Ratio（国际标准化比值） | **0.8–1.2** | PT 的标准化值（抗凝监测 / 跨实验室比较） |
| **PTT / aPTT** | (Activated) Partial Thromboplastin Time（部分凝血活酶时间） | **25–40 秒** | **内源 intrinsic + 共同 common** 通路 |
| **血小板 Platelet** | Platelet count | **150–450 ×10⁹/L** | 初级止血（数量） |
| **纤维蛋白原 Fibrinogen** | — | **200–400 mg/dL** | 共同通路终末底物（DIC↓） |
| **D-二聚体 D-dimer** | — | 阴性 / <0.5 µg/mL | 纤溶产物（DIC / 静脉血栓栓塞 venous thromboembolism, VTE↑） |

> [!warning] 各实验室参考区间略有差异——**做题以题干给的参考范围为准**。BT 现临床多被 **PFA-100 血小板功能分析仪**替代，但 USMLE 仍考其意义。

## 二、通路归属（背这张就够做题）

```
内源 Intrinsic (PTT) 12,11,9,8         外源 Extrinsic (PT) 3,7
  XII → XI → IX → VIII              VII  ← 组织因子 TF / 三因子
        │                                        │
        └──────┬─────────────────────────────────┘
               ▼
        共同 Common (PT + PTT)
        X → V → II(凝血酶) → I(纤维蛋白原 → 纤维蛋白)
```

> [!tip] 监测口诀
> - **PT = 外（extrinsic）→ 监测华法林 warfarin**：口诀 **"PT-WET / 因子 1972"**（**W**arfarin / **E**xtrinsic / **T**issue factor；外源主角 **VII**，半衰期最短最先掉）
> - **PTT = 内（intrinsic）→ 监测普通肝素 heparin (UFH)**
> - **共同通路**因子（X / V / II / I）缺 → **PT + PTT 同时延长**

> [!warning] 华法林"VII 先掉→PT 先↑"的两个高频延伸 ⭐
> 因子按半衰期掉的顺序：**VII (~6h) → IX (~24h) → X (~36h) → II (~60h)**；天然抗凝物**蛋白 C protein C (~8h)** 也很短。由此两个陷阱：
> - **① INR 早达标 ≠ 真抗凝 → 肝素桥接 ≥5 天**：PT/INR 升高是 **VII 先掉**撑起的"假象"，真正抗栓要等**因子 II（凝血酶原 prothrombin, ~60h）**降下来（~3-5 天）。故急性血栓起始华法林**前 5 天必须重叠普通肝素 / 低分子肝素（low molecular weight heparin, LMWH）**，INR 达标 ≥24h 才停肝素。
> - **② 华法林皮肤坏死（warfarin-induced skin necrosis）**：**蛋白 C（天然抗凝物）半衰期也短、和 VII 一样早掉** → 起始头几天**短暂促凝**（抗凝物先没、促凝因子 II 还在）→ 微血栓 → 皮肤坏死，**蛋白 C 缺乏者尤甚** → 这是必须肝素桥接的第二个理由。
>
> 钥匙：**"查 PT/INR 是 VII 撑的；真抗栓靠 II 慢掉 → 桥接 5 天；蛋白 C 早掉 → 起始可皮肤坏死。"**

## 三、四维组合 → 诊断（核心表）⭐

| PT | PTT | BT / 血小板 | 最可能诊断 | 钥匙 |
|---|---|---|---|---|
| ↑ | 正常 | 正常 | **早期维生素 K 缺乏 / 华法林 warfarin / VII 因子缺乏** | 外源最先掉（VII 半衰期短） |
| 正常 | ↑ | 正常 | **血友病 A（hemophilia A，VIII 缺）/ B（IX 缺）**；**XI / XII 缺**；**肝素 heparin**；**抗磷脂抗体 antiphospholipid** | 男童关节腔/深部出血 = 血友病 |
| 正常 | ↑ | **BT↑** | **血管性血友病（von Willebrand disease, vWD）** | vWF 载 VIII（→PTT↑）+ 助血小板黏附（→BT↑）；**黏膜皮肤出血**，最常见遗传性出血病 |
| ↑ | ↑ | 正常 | **华法林过量 / 严重维生素 K 缺乏 / 肝病**（共同通路 / 多因子） | 肝病：所有因子↓（除 vWF/VIII） |
| ↑ | ↑ | **BT↑ + 板↓** | **弥散性血管内凝血（DIC）** | 板↓ + PT/PTT↑ + **纤维蛋白原↓** + **D-二聚体↑** |
| 正常 | 正常 | **BT↑（板正常）** | **血小板功能异常**：vWD（轻）/ **尿毒症 uremia** / **阿司匹林 aspirin·非甾体抗炎药 NSAID** / Glanzmann / Bernard-Soulier | 功能问题，数量不少 |
| 正常 | 正常 | **BT↑ + 板↓** | **血小板减少**：ITP / TTP-溶血尿毒综合征（hemolytic uremic syndrome, HUS）/ 骨髓衰竭 / 脾隔离 | 数量问题 |

> [!danger] 易混：vWD vs 血友病（hemophilia A/B）—— 都 PT 正常 + PTT↑ + 板数正常 ⭐⭐
> 这是最经典的"PTT↑ + PT 正常 + 血小板数正常"鉴别对，三项实验室一模一样，**分水岭 = 出血部位 + 出血时间（BT）+ 遗传**：
>
> | 特征 | **血管性血友病 vWD** | **血友病 A/B（hemophilia A/B）** |
> |---|---|---|
> | 遗传 | 多**常染色体显性**（男女均可、有家族史） | **X 连锁隐性**（多男性） |
> | **出血部位** ⭐ | **黏膜/皮肤**（鼻衄、牙龈、月经过多、GI） | **深部/关节**（关节积血 hemarthrosis、肌肉、内脏血肿） |
> | **出血时间 BT** ⭐ | **延长**（血小板黏附差） | **正常**（血小板功能不受累） |
> | 血小板计数 | 正常 | 正常 |
> | PT / PTT | PT 正常、**PTT↑** | PT 正常、**PTT↑** |
> | 缺陷 | **vWF**（± 继发因子 VIII↓） | **因子 VIII（A）/ IX（B）** |
> | 治疗 | **去氨加压素 desmopressin (DDAVP)** ± vWF 浓缩物 | 因子 VIII（A）/ IX（B）替代；轻型 A 也可 DDAVP |
>
> **为什么 vWD 也 PTT↑**：vWF 是因子 VIII 的**血浆载体**，vWF 缺 → 因子 VIII 半衰期缩短而**继发↓** → PTT 延长。所以 vWD"既像血小板病（BT↑、黏膜出血）又像凝血因子病（PTT↑）"，横跨两类。
>
> **一句话钥匙**：**"黏膜出血 + BT↑ = vWD；关节深部出血 + BT 正常 = 血友病。"** 两者 PTT 都↑、板数都正常，**只看 BT 和出血部位**。
>
> **下一步**：确诊靠 **vWF 抗原 + vWF 活性（瑞斯托霉素辅因子 ristocetin cofactor）+ 因子 VIII 水平**（vWD）；血友病查**因子 VIII / IX 活性**。

> [!danger] 易混：肝病（liver disease）PT↑+板↓ ≠ vWD PT正常+板正常 ⭐
> 两者都可能"PTT 偏高"，但**指纹方向相反**，是高频混淆点（PTT↑ 不要反射跳 vWD）：
>
> | | **肝病 liver disease** | **血管性血友病 vWD** |
> |---|---|---|
> | **PT** | **↑（最先、最敏感）** | **正常** |
> | PTT | ↑（进展期） | ↑ 或正常 |
> | **血小板** | **↓（真的少）** | **正常**（只是功能/黏附差） |
> | 出血时间 BT | 多正常 | **↑** |
> | 病因背景 | 酗酒 / 腹水 / 黄疸 / 肝硬化失代偿 | 遗传（常显）、黏膜出血、家族史 |
>
> **掰开记**：
> - **肝病 = "PT 最敏感 + 板被脾吃掉"**：肝合成所有因子（除 vWF/VIII），**因子 VII 半衰期最短 → PT/INR 最先↑**（合成功能最敏感指标）；门脉高压 **脾亢 hypersplenism** 扣押血小板 + 肝 **血小板生成素 thrombopoietin (TPO)↓** → **板数真的少**。
> - **vWD = "PT 正常 + 板数正常，只是黏不住"**：外源/共同通路不受影响（PT 正常），血小板**数量正常**只是黏附障碍（BT↑、PTT 因 vWF 载 VIII 而可↑）。
> - 一句话钥匙：**"板真少 + PT 长 = 肝；板数正常 + PT 正常 = vWD。"**

> [!danger] 易混：肝病（liver disease）vs 华法林（warfarin）/ 维生素 K 缺乏 —— 都 PT↑，怎么分 ⭐
> 三者都让 **PT↑**（共同/外源因子受累）。真正区分**肝病 vs 华法林**的只有两条：**因子 V** 和**维生素 K 试验**（血小板仅辅助）。**因子 VIII 在这对比里不区分——两者都正常**（它是用来分肝病 vs DIC 的，见下）：
>
> | 鉴别点 | **肝病 liver disease** | **华法林 / 维生素 K 缺乏** | 机制 |
> |---|---|---|---|
> | **因子 V (factor V)** ⭐ | **↓** | **正常** | V 由肝合成但**不依赖维 K** → 肝坏才掉，华法林动不了 |
> | **维生素 K 试验** ⭐ | 给维 K **不纠正** | 给维 K **可纠正** | 肝病=工厂坏；华法林/维 K 缺=缺底物，补上即好 |
> | **血小板** | **↓**（脾亢 + TPO↓） | 正常 | 仅辅助线索 |
> | 因子 VIII (factor VIII) | 正常 / ↑ | 正常 | **两边都正常 → 不区分本对比**；价值在肝病 vs DIC |
>
> **两因子组合判读**（肝病 vs 华法林/维 K）：
> - **V↓ + VII↓ = 肝病**（非维 K 依赖的 V 也掉 = 合成工厂问题）
> - **V 正常 + VII↓ = 维生素 K 缺乏 / 华法林**（只有维 K 依赖因子 II/VII/IX/X 掉，V 完好）
>
> **因子 VIII 的真正用途 = 分肝病 vs DIC**（不是分华法林）：
> - 肝病 VIII **正常 / ↑**（内皮来源、急性期）；**DIC VIII↓**（被消耗）。配合 DIC 还有纤维蛋白原↓ + D-二聚体↑ + 板↓。
>
> **临床第一刀（先看病史）**：抗凝药史 → 华法林；酗酒/腹水/黄疸/蜘蛛痣 spider angiomata → 肝病；吸收不良/新生儿/长期抗生素 → 维 K 缺乏。因子 V + 维 K 试验是实验室确认层。

> [!danger] 易混：肝病（liver disease）vs DIC（弥散性血管内凝血 disseminated intravascular coagulation）—— 最像的一对 ⭐
> 两者都 **PT↑ + PTT↑ + 板↓**，最难分。钥匙是**因子 VIII** 和 **D-二聚体**：
>
> | 指标 | **肝病 liver disease** | **DIC** | 区分价值 |
> |---|---|---|---|
> | **因子 VIII (factor VIII)** ⭐⭐ | **正常 / ↑** | **↓** | **最强单项**——VIII 由**血管内皮**产生（非肝），肝病不降；DIC 被消耗→↓ |
> | **D-二聚体 / FDP（纤维蛋白降解产物 fibrin degradation products）** ⭐ | 正常 / 轻↑ | **显著↑** | DIC 有活跃纤溶；肝病无 |
> | 纤维蛋白原 fibrinogen | 正常（晚期才↓） | **↓**（消耗） | DIC 降得早而明显（急性期蛋白反降 = 强提示）|
> | 裂红细胞 schistocytes | 无 | **有**（微血管病性溶血 MAHA）| DIC 涂片可见 |
> | 因子 V (factor V) | ↓（合成↓）| ↓（消耗）| **都↓，不区分**（V 能分肝病 vs 华法林，不能分 DIC）|
> | 血小板 / PT / PTT | 都异常 | 都异常 | **不区分** |
> | 病程/背景 | **慢性** + 腹水/黄疸/蜘蛛痣 | **急性** + 诱因（脓毒症/产科/恶性肿瘤/创伤）| 临床第一刀 |
>
> **一句话钥匙**：**"VIII 不降 + D-二聚体不高 = 肝病；VIII 降 + D-二聚体飙 = DIC。"**
>
> **底层逻辑（因子来源决定一切）**：
> - 肝合成的因子（II/V/VII/IX/X/XI/纤维蛋白原…）→ **肝病都↓**。
> - **因子 VIII、vWF 来自血管内皮，不靠肝 → 肝病不降（VIII 甚至↑）**。
> - DIC = 全身消耗 → 连内皮来源的 **VIII 也被消耗↓** + 启动纤溶（**D-二聚体↑**）。
> - 所以"肝病只伤肝产因子、放过内皮 VIII/vWF；DIC 谁都消耗 + 纤溶"——这就是 VIII 和 D-二聚体能分开两者的根本。

## 四、混合试验（mixing study）★ — PTT 延长时鉴别

> [!info] 原理：为什么"1:1 混合"能区分缺因子 vs 抑制物
> 正常血浆每个凝血因子按 100% 算；病人 + 正常 **1:1 混合**后，任一因子至少恢复到 **~50%**，而 PT/PTT 只要每个因子 **≥30–50% 活性**就跑正常。所以：
> - **缺因子** → 补进正常血浆的因子 → 浓度回 50%+ → **PTT 纠正**。
> - **有抑制物（抗体）** → 抗体把**刚加进来的正常因子也一起灭活** → **PTT 不纠正**。
> 一句话：**缺什么补什么就好（纠正）；有抗体连补进去的也被打掉（不纠正）。**

| 维度 | **因子缺乏 factor deficiency** | **抑制物 inhibitor** |
|---|---|---|
| 混合后 PTT | **纠正**（回正常） | **不纠正**（仍延长） |
| 机制 | 缺的因子被正常血浆补上 | 抗体把正常血浆的因子也灭活 |
| 代表病 | 血友病 A/B、vWD、肝病、维 K 缺乏 | 获得性因子 VIII 抑制物、狼疮抗凝物（→ APS） |
| 出血 or 血栓 | 出血 | 因子 VIII 抑制物 → **出血**；狼疮抗凝物 → **血栓**（陷阱）|

### ★ 即刻 vs 孵育混合（immediate vs incubated）——把"抑制物"再分两种

```
混合试验
│
├─ 即刻测(immediate) 就不纠正
│   └─► 狼疮抗凝物 lupus anticoagulant（作用快、干扰磷脂、不依赖时间温度）
│
└─ 即刻"部分纠正"，但 37℃ 孵育 1–2h 后又延长
    └─► 获得性因子 VIII 抑制物（抗体时间+温度依赖，慢慢灭活 VIII）
```

> [!tip] 钥匙
> **"立刻就不纠正 = 狼疮抗凝物；孵育后才不纠正 = 因子 VIII 抑制物。"** 因为抗因子 VIII 抗体是慢反应、温度依赖；狼疮抗凝物即刻干扰磷脂试剂、不用等。

> [!warning] 两个"不纠正"代表病——临床方向完全相反 ⚠️
>
> | | **获得性因子 VIII 抑制物**（acquired factor VIII inhibitor） | **狼疮抗凝物**（lupus anticoagulant → APS） |
> |---|---|---|
> | 抗体打谁 | 抗**因子 VIII** | 抗**磷脂**（干扰体外试剂） |
> | 临床表现 | **严重出血**（老年/产后/自身免疫，突发大片瘀斑血肿） | **血栓 + 反复流产**（动静脉栓塞、习惯性流产） |
> | 体内真相 | 真缺 VIII 活性 → 真出血 | 体外"假延长"，**体内反而高凝** |
> | 混合试验 | 孵育后不纠正 | 即刻不纠正 |
> | 确认试验 | 因子 VIII 活性↓ + Bethesda 滴度（Bethesda units 定量抑制物） | dRVVT（稀释罗素蝰蛇毒时间 dilute Russell viper venom time）+ 抗心磷脂抗体（anticardiolipin antibody）+ 抗 β2-糖蛋白1 抗体（anti-β2-glycoprotein 1, β2-GP1） |
>
> 钩子：**"老年/产后突发出血 + PTT↑不纠正" → 因子 VIII 抑制物**；**"反复血栓/流产 + PTT↑不纠正" → 狼疮抗凝物/APS**。

> [!info] 获得性因子 VIII 抑制物（acquired hemophilia A，获得性血友病 A）——"怎么获得的" ⭐
> 不是遗传，是**后天产生抗因子 VIII 的自身抗体（autoantibody，IgG）**，把自己原本正常的因子 VIII 中和掉。**5 大病因**：
>
> | 病因 | 钩子 | 占比 |
> |---|---|---|
> | **特发性 idiopathic** | **老年人**（60-80 岁），无诱因 | ~50%（最多）|
> | **妊娠相关 peripartum** | **年轻女性、产后数月内**（常首胎）| 经典考点 |
> | **自身免疫病** | 系统性红斑狼疮（SLE）、类风湿关节炎（RA） | 常见 |
> | **恶性肿瘤** | 淋巴增殖性疾病 / 实体瘤 | 常见 |
> | **药物** | 青霉素 penicillin、干扰素 interferon | 少见 |
>
> **vs 先天血友病 A**：获得性=后天突发、**无家族史**、**软组织/肌肉/黏膜大片瘀斑血肿**、混合试验**孵育后不纠正**；先天=自幼终身、X 连锁男性、**关节腔出血 hemarthrosis**、混合试验**纠正**。
>
> **机制对应混合试验**：抗体是 IgG、**时间+温度依赖** → 需 37℃ 孵育 1-2h 才充分灭活 VIII → **即刻部分纠正、孵育后又延长**（区别于狼疮抗凝物即刻不纠正）。
>
> **治疗**：① 止血用**旁路制剂 bypassing agent**——重组因子 VIIa（recombinant factor VIIa, rFVIIa）或活化凝血酶原复合物（activated prothrombin complex concentrate, aPCC）（有抗体在，直接补 VIII 会被中和）；② 清除抗体用**免疫抑制**——糖皮质激素 ± 利妥昔单抗 rituximab / 环磷酰胺 cyclophosphamide。

> [!warning] 抗磷脂综合征（antiphospholipid syndrome, APS）的陷阱
> **体外 PTT 延长（狼疮抗凝物干扰磷脂试剂）但体内反而高凝、易血栓** —— 实验室"像出血"，临床是"栓"。反复动静脉血栓 / 习惯性流产 + PTT↑不纠正 → 想 APS。

### PTT 延长完整决策流

```
PTT 延长（PT 正常）
│
├─ 做混合试验 (1:1 mixing)
│
├─ 纠正 → 因子缺乏
│   ├─ 出血时间↑ + 黏膜出血 → vWD
│   ├─ 关节/深部出血、男童、X连锁 → 血友病 A(VIII)/B(IX) → 查具体因子活性
│   └─ 肝病/营养不良背景 → 肝病 / 维 K 缺乏（注意这俩 PT 也↑）
│
└─ 不纠正 → 抑制物
    ├─ 即刻不纠正 → 狼疮抗凝物 → dRVVT → APS（血栓/流产）
    └─ 孵育后不纠正 → 因子 VIII 抑制物 → VIII 活性 + Bethesda（出血）
```

## 五、各病一行速记

| 病 | PT | PTT | BT/板 | 一句话 |
|---|---|---|---|---|
| **血友病 A/B（hemophilia A/B）** | 正常 | ↑ | 正常 | X-连锁隐性、男童；关节/深部血肿；A=VIII，B=IX |
| **血管性血友病（vWD）** | 正常 | ↑/正常 | **BT↑** | 最常见遗传性出血病；黏膜皮肤出血；治疗 **去氨加压素 desmopressin (DDAVP)** |
| **维生素 K 缺乏（vitamin K deficiency）** | **↑** | ↑（重）| 正常 | 因子 **II/VII/IX/X + 蛋白 C/S**；新生儿出血病、抗生素、吸收不良 |
| **肝病（liver disease）** | ↑ | ↑ | 板↓（脾亢）| 除 vWF/VIII 外所有因子↓；VII 最先 |
| **DIC（弥散性血管内凝血）** | **↑** | **↑** | **BT↑ 板↓** | 消耗性；纤维蛋白原↓ + D-二聚体↑；脓毒症/产科/恶性肿瘤 |
| **ITP（免疫性血小板减少性紫癜）** | 正常 | 正常 | **板↓** | 孤立性血小板↓，余正常；儿童病毒后自限 |
| **TTP（血栓性血小板减少性紫癜）** | 正常 | 正常 | **板↓** | 五联：微血管病性溶血性贫血（microangiopathic hemolytic anemia, MAHA）+ 板↓ + 神经 + 肾 + 发热；ADAMTS13↓ |
| **尿毒症 uremia** | 正常 | 正常 | **BT↑** | 血小板功能障碍；治疗 **去氨加压素 desmopressin** + 透析 |
| **抗磷脂综合征（APS）** | 正常 | **↑（不纠正）** | 正常 | 体外 PTT↑ 但体内**高凝栓塞** |

## 六、考前一眼扫：出血鉴别反射图 ⭐

```
PT↑ PTT↑ 板↓ 一堆异常 →
├─ 因子 VIII 正常/↑ + 维K不纠正 + 慢性肝病背景(腹水/黄疸) → 肝病
├─ 因子 VIII↓ + D-二聚体飙 + 纤维蛋白原↓ + 裂红细胞 + 急性诱因 → DIC
├─ 因子 V 正常 + 板正常 + 维K可纠正 → 维K缺乏 / 华法林(看有无药史)
└─ 板↓ 但 PT/PTT/纤维蛋白原正常 + 裂红细胞 → TTP/HUS(别和 DIC 混)

PTT↑ 单独(PT 正常) →
├─ 混合试验纠正 → 缺因子
│   ├─ 出血时间↑ + 黏膜出血 → vWD
│   └─ 关节/深部出血 + 男童 → 血友病 A(VIII)/B(IX)
└─ 混合试验不纠正 → 抑制物
    ├─ 即刻不纠正 → 狼疮抗凝物 → APS(血栓/流产)
    └─ 孵育后不纠正 → 获得性因子 VIII 抑制物(老年/产后突发出血)

BT↑ 单独(PT/PTT 正常) →
├─ 板数正常 → 血小板功能障碍(尿毒症 / 阿司匹林·NSAID / vWD轻 / Glanzmann)
└─ 板↓      → 血小板减少(ITP / TTP-HUS / 骨髓衰竭 / 脾隔离)
```

> [!success] 三把钥匙一句话收口
> - **因子 VIII** 分**肝病(正常↑) vs DIC(↓)**；**因子 V + 维 K 试验** 分**肝病 vs 华法林**；**PT + 血小板数** 分**肝病 vs vWD**。
> - **混合试验纠正 = 缺因子；不纠正 = 抑制物**（即刻→狼疮抗凝物·血栓；孵育后→因子 VIII 抑制物·出血）。
> - **板↓ 但 PT/PTT/纤维蛋白原正常 + 裂红细胞 = TTP/HUS**，别和 DIC 混。

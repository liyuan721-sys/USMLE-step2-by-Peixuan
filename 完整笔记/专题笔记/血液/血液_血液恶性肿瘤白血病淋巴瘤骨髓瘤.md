# 血液恶性肿瘤（Hematologic Malignancies — 白血病 / 淋巴瘤 / 浆细胞病）

> [!info] 这张笔记的定位
> 聚焦**白细胞 + 浆细胞恶性肿瘤**的**纵向深挖与横向鉴别**（四大白血病 / Hodgkin vs Non-Hodgkin / 浆细胞三病 / MDS-MPN）。
> 想要**全血液科决策导向速查**（含贫血 / 出血 / 输血 / 高凝）→ 看 [[完整笔记/专题笔记/血液/血液_HighYield速查总览]]。
> 本笔记是该总览 §5–6 的**展开版**，并整合错题 [[NBME11_错题本#^Q143]]（治疗相关 AML 病因）。

## 〇、诊断骨架（先记两个数字 + 两个步骤）

> [!tip] 通用反射
> - **骨髓原始细胞 (blasts) >20% = 急性白血病；<5% = 缓解 (remission)**。
> - **外周血涂片见 myeloblasts（原始粒细胞）/ lymphoblasts（原始淋巴细胞）** → 先把题**重新定性为白血病**，再回头找病因 / 分型。
> - **诊断金标准**：**骨髓活检 (bone marrow biopsy) + 流式细胞术 (flow cytometry) + 细胞遗传学 (cytogenetics)**。
> - **三系受累（贫血 + 血小板↓ + 白细胞异常）** = 骨髓被肿瘤细胞挤占 (marrow crowding / infiltration) 的标志。

```
全血细胞减少 (pancytopenia) / 外周原始细胞
        │
        ├─ 有 blasts / 异形细胞 → 白血病 / MDS / 骨髓浸润
        │       ├─ blasts >20% → 急性 (AML / ALL)
        │       └─ blasts <20% + 发育异常 → MDS（可进展为 AML）
        │
        └─ 无 blasts + 无肝脾大 + 无泪滴红细胞 → 再障 (aplastic anemia)
```

## 一、四大白血病鉴别（Acute vs Chronic × Myeloid vs Lymphoid）

> [!info] 一句话钥匙
> **年龄是第一钥匙**：小孩 → ALL；中年（30–50）无症状脾大 → CML；老人（>55）涂抹细胞 → CLL；双峰（15–30 / >55）+ Auer 小体 → AML。

| 白血病 | 年龄钥匙 | 标志性发现 | 遗传 / 治疗 |
|---|---|---|---|
| **ALL** (acute lymphoblastic leukemia, 急性淋巴细胞白血病) | **儿童（峰 2–5 岁）**，最常见儿童癌 | **TdT+ / CD10+**；骨痛、肝脾肿大、淋巴结肿大、**睾丸肿大 (testicular enlargement)**；**T 系 → 前纵隔肿块 (anterior mediastinal mass)** → 气道压迫 / 上腔静脉综合征；脑膜浸润 → 颅神经 (cranial nerve, CN) 麻痹 | **LP 查 CNS 浸润**；**t(12;21) 预后好，t(9;22) 费城染色体预后差** |
| **AML** (acute myeloid leukemia, 急性髓系白血病) | **双峰 15–30 与 >55** | **Auer 小体 (Auer rods)**、**苏丹黑 (Sudan black) / MPO+**、myeloblasts | **APL 亚型 t(15;17) → DIC 急症 → ATRA（全反式维 A 酸）**；**既往烷化剂化疗 / 放疗 → t-AML**（见下） |
| **CML** (chronic myeloid leukemia, 慢性髓系白血病) | **30–50** | 常**无症状**、脾大；中性粒↑各阶段成熟粒细胞、**嗜碱↑**、慢性期 blasts <10%；**LAP↓** | **t(9;22) BCR-ABL = 费城染色体 (Philadelphia)**；Tx = **伊马替尼 (imatinib，酪氨酸激酶抑制剂 TKI)** |
| **CLL** (chronic lymphocytic leukemia, 慢性淋巴细胞白血病) | **>55** | **涂抹细胞 (smudge cells)**、常无症状、可并发 **自身免疫性溶血性贫血 (autoimmune hemolytic anemia, AIHA)**；老年最常见白血病 | 进展期 **利妥昔单抗 (rituximab) 为基础**；**Richter 转化 → 弥漫大 B (DLBCL)** |

> [!warning] 几个钥匙别混
> - **LAP (leukocyte alkaline phosphatase, 白细胞碱性磷酸酶)**：**↓** 见 **CML / PNH**；**↑** 见**类白血病反应 (leukemoid reaction) / 真红 (PV)**。→ "高白细胞"题用 LAP 区分 CML（↓）vs 感染性类白血病反应（↑）。
> - **Auer 小体 → AML（髓系）**；**TdT+ → ALL（淋系，原始）**；**smudge cells → CLL**；**费城染色体 t(9;22)** 同时见于 **CML（标志）** 与 **ALL（预后差）**——靠年龄 + 急慢性区分。
> - **Down 综合征**：年幼（<5 岁）偏 **AML（尤其巨核细胞型）**，年长偏 **ALL**。

### 1.1 APL — 急性早幼粒细胞白血病（医疗急症）

> [!danger] APL (acute promyelocytic leukemia, 急性早幼粒细胞白血病) = 不能等的急症
> - 年轻 / 中年成人；外周血**异常早幼粒细胞 (atypical promyelocytes，胞浆紫色颗粒)**；遗传 **t(15;17) → PML-RARA 融合基因**。
> - **核心危险**：颗粒释放促凝 + 促纤溶 → **DIC (disseminated intravascular coagulation, 弥散性血管内凝血) / 凝血病** → 消耗血小板 / 纤维蛋白原 / 凝血因子 → **严重出血（颅内 / 肺 / 消化道）**。
> - **治疗 = 全反式维 A 酸 (ATRA, all-trans retinoic acid) ± 三氧化二砷 (arsenic trioxide)**，诱导异常早幼粒细胞分化成熟（**不是单纯杀伤**）；高度可治愈。

### 1.2 毛细胞白血病（Hairy cell leukemia）

> [!info]
> >55 岁、成熟 B 细胞肿瘤；**巨脾 (massive splenomegaly)** + 全血细胞减少 + 反复感染；骨髓**干抽 (dry tap)**、流式 **TRAP+**；Tx = 嘌呤类似物 **克拉屈滨 (cladribine) / 喷司他丁 (pentostatin)**。

### 1.3 治疗相关 AML（Therapy-related AML, t-AML）⭐ 错题 NBME11 Q143

> [!danger] t-AML / 继发性急性髓系白血病 — 经典"第二原发恶性肿瘤"
> **既往细胞毒化疗（烷化剂 alkylating agents）/ 放疗 (radiation)** 是 AML 的经典危险因素，常在原发肿瘤治疗后**数年至 ~10 年**作为**第二原发恶性肿瘤 (second primary malignancy)** 出现，而非原癌复发。
>
> **题干模板**：既往肿瘤（如卵巢癌 ovarian cancer）化疗史 + 现**三系异常**（贫血 + 血小板↓ → 瘀点 / 鼻出血）+ **外周血见 myeloblasts** → 锁定 **t-AML**。
>
> **反射**：**外周血 myeloblasts → 先把题重新定性为 AML，再回头找最强病因 = 化疗 / 放疗**。

横向鉴别（同一题各暴露 ↔ AML 匹配度，NBME 考"最经典 / 最强关联"）：

| 暴露 / 疾病 | 与 AML 匹配 | 关键理由 |
|---|---|---|
| **既往化疗（烷化剂）** ✅ | 最高 | 治疗相关 AML 经典危险因子 |
| Filgrastim (G-CSF, 非格司亭) | 次要 / 弱 | 仅轻度增髓系肿瘤风险，远弱于细胞毒化疗（NBME 钓饵） |
| 雌激素 (estrogen) | 低 | 偏激素依赖性实体瘤（如乳腺），非 AML |
| 原发癌（卵巢癌）复发 | 低 | 不以外周血 myeloblasts 为表现 |

> [!tip] Memory Hook
> **"化疗过几年，髓母细胞见 = 治疗相关 AML；先想 chemo / 放疗，不是 filgrastim。"**
> 烷化剂 → 典型 5–7 年潜伏、常伴 −5/−7 染色体异常；拓扑异构酶 II 抑制剂（依托泊苷 etoposide / 蒽环类）→ 潜伏更短（1–3 年）、常伴 11q23 (MLL) 易位。

> [!note] t-MDS → t-AML 连续谱
> 同样的烷化剂 / 放疗暴露也可先表现为**治疗相关骨髓增生异常综合征 (t-MDS)**（blasts <20% + 发育异常），再进展为 t-AML。见到既往化疗 + 血细胞减少 + 发育异常但 blasts 未达 20% → 想 t-MDS。

## 二、淋巴瘤（Lymphoma — Hodgkin vs Non-Hodgkin）

| 维度 | **Hodgkin 淋巴瘤 (HL)** | **Non-Hodgkin 淋巴瘤 (NHL)** |
|---|---|---|
| 年龄 | **双峰 15–35 与 >60** | 多 **>55**（中老年） |
| 标志细胞 | **Reed-Sternberg 细胞 (CD15+/CD30+)** | 无 RS；按亚型分子标志 |
| 扩散方式 | **连续 (contiguous)**、沿淋巴结链 | **跳跃 / 结外 (extranodal)**，常累及腹腔 / GI |
| 关联 | **免疫抑制下 EBV (Epstein-Barr virus, EB 病毒)**；最常见亚型 **结节硬化型 (nodular sclerosis)**（年轻女性） | **免疫缺陷 / 自免可继发**；HIV、移植后 |
| 特征线索 | **无痛性淋巴结肿大**、前纵隔肿块、**饮酒后疼痛**、瘙痒、嗜酸↑、LDH↑ | **B 症状**明显、体内深部淋巴结 / 腹部包块 |
| 经典遗传 | — | **t(8;14) → Burkitt (c-myc)**；**t(14;18) → 滤泡型 (bcl-2)** |
| 一线治疗 | **ABVD** | **CHOP（B 细胞型 +R = R-CHOP）** |

> [!info] B 症状 (B symptoms) + Ann Arbor 分期
> - **B 症状 = 发热 + 盗汗 + 体重下降**（提示更高负荷 / 更晚期 / 需化疗）。
> - **I**：单个淋巴结区；**II**：≥2 区**同侧膈肌**；**III**：**跨膈**；**IV**：广泛结外播散。
> - **HL**：I–II 期无 B 症状 → **放疗**；I–II 期有 B 症状或 III–IV 期 → **化疗 (ABVD)**。
> - **诊断金标准**：**淋巴结活检 (excisional biopsy)**（不是细针穿刺）。

> [!warning] Burkitt 淋巴瘤特别提示
> **t(8;14) c-myc**，倍增时间最短 → **肿瘤溶解综合征 (TLS) 最高风险**；"starry sky" 组织像；地方型（非洲）下颌肿块、散发型腹腔肿块。化疗前预防 TLS（别嘌呤醇 / 拉布立酶）——见 [[完整笔记/专题笔记/血液/血液_HighYield速查总览]] §7 + NBME12 Q140（ALL 化疗 TLS 预防）。

### 2.1 化疗方案毒性（高频"哪个药致哪个毒"）⭐

> [!danger] ABVD（Hodgkin）
>
> | 药 | 类别 | 标志毒性 / 解毒 |
> |---|---|---|
> | **Adriamycin = 多柔比星 (doxorubicin)** | 蒽环类 | **心肌毒性 / 扩张型心肌病**；解毒 **右雷佐生 (dexrazoxane)** |
> | **Bleomycin (博来霉素)** | 自由基致 DNA 断裂 | **肺毒性 / 肺纤维化** |
> | **Vinblastine (长春碱)** | 抑微管聚合，阻 M 期 | 骨髓抑制 |
> | **Dacarbazine (达卡巴嗪)** | 烷化样 | 骨髓抑制、恶心呕吐 |

> [!danger] CHOP（Non-Hodgkin）
>
> | 药 | 类别 | 标志毒性 / 解毒 |
> |---|---|---|
> | **C = 环磷酰胺 (cyclophosphamide)** | 烷化剂 | **出血性膀胱炎 (hemorrhagic cystitis)**；解毒 **美司钠 (MESNA)** |
> | **H = Hydroxydaunorubicin = 多柔比星 (doxorubicin)** | 蒽环类 | **心肌毒性 / 扩张型心肌病**；解毒 dexrazoxane |
> | **O = Oncovin = 长春新碱 (vincristine)** | 长春碱 | **周围神经病变 / 便秘（肠麻痹）** |
> | **P = 泼尼松 (prednisone)** | 糖皮质激素 | 高血糖 / 免疫抑制 |
>
> 记忆：环磷酰胺 → 膀胱（MESNA 救）；蒽环类 → 心（dexrazoxane 救）；长春新碱 → 神经（不是骨髓）；博来霉素 → 肺。

## 三、浆细胞病（Plasma Cell Dyscrasias — MM / MGUS / Waldenström）

| 病 | 关键表现 | 实验室 / 诊断 | M 蛋白 |
|---|---|---|---|
| **多发性骨髓瘤 MM** (multiple myeloma, 多发性骨髓瘤) | **CRAB**：高钙 (Calcium↑) / 肾衰 (Renal) / 贫血 (Anemia) / **穿凿样溶骨灶 + 骨痛 (Bone)**；缗钱状红细胞 (Rouleaux) | **血 M-spike >3 g/dL + 骨髓单克隆浆细胞 >10%**；尿 **Bence Jones 轻链** | **IgG（最常见）/ IgA / 轻链**（血清蛋白电泳 SPEP, serum protein electrophoresis 测） |
| **MGUS** (monoclonal gammopathy of undetermined significance) | **无症状**（无 CRAB）、X 线正常 | **M-spike <3 g/dL + 骨髓浆细胞 <10%**；MM 的前驱 | 同 MM 谱 |
| **Waldenström 巨球蛋白血症** (Waldenström macroglobulinemia) | **高黏滞综合征 (hyperviscosity)**：视物模糊 / **视网膜"香肠样"静脉 (sausage-link)** / AMS / 出血 / 神经病；**无溶骨灶**；肝脾 + 淋巴结肿大 | SPEP；骨髓 >10% 克隆性 B 细胞；γ gap↑、ESR↑ | **IgM** |

> [!tip] MGUS → MM 的"升档"线索
> M-spike <3 + 骨髓 <10% + **无 CRAB + X 线正常** = MGUS。出现 **高钙 (hypercalcemia)** / 溶骨灶 / 肾衰 / 贫血 → 升级为 MM。

> [!tip] MM 的碱性磷酸酶 (ALP) 为什么正常（高频陷阱）⭐
> MM 溶骨机制 = 瘤细胞分泌 **DKK1 / IL-3 抑制成骨细胞** + RANKL 激活破骨 → **成骨被摁住不造骨** → **碱性磷酸酶 (alkaline phosphatase, ALP) 典型正常**，尽管广泛溶骨 + 骨痛。别因"骨破坏"就期待 ALP 升——**ALP 正常反而支持 MM**。对比：成骨性转移（前列腺癌）/ Paget 病 / 骨软化 ALP↑。机制详见 [[完整笔记/专题笔记/内分泌/内分泌_骨质疏松继发病因筛查与骨代谢标志物]] §三。

> [!warning] 蛋白尿检测对比（骨髓瘤经典陷阱）
>
> | 检测 | 敏感目标 | 局限 |
> |---|---|---|
> | **尿试纸 (dipstick)** | **仅白蛋白** | **漏轻链** |
> | **SSA 试验 (磺基水杨酸)** | **所有总蛋白（白 + 球 + 轻链）** | 不分种类 |
> | **尿电泳 (UPEP)** | 分类各蛋白成分 | 操作复杂，非初筛 |
>
> **试纸阴性 + SSA 阳性"不一致" → 锁定骨髓瘤轻链 (Bence Jones)**。

> [!info] Waldenström 急症处理
> **高黏滞危象（视物模糊 / AMS / 出血）→ 血浆置换 (plasmapheresis)** 紧急降黏度，再化疗。MM 与 Waldenström 涂片都可见 **Rouleaux（缗钱状）**，靠 **IgG vs IgM + 有无溶骨灶**区分。
> γ gap = 总蛋白 − 白蛋白，>4 g/dL 提示免疫球蛋白↑：单克隆（MM / MGUS / Waldenström）vs 多克隆（慢性感染 TB / 自免 / 慢性肝病）——详见 [[完整笔记/专题笔记/USMLE/USMLE_高频评分_公式]]。

## 四、MDS / MPN（骨髓增生异常 vs 增殖性肿瘤）

> [!note] 与白血病的边界
> - **MDS (myelodysplastic syndrome, 骨髓增生异常综合征)**：**无效造血 → 血细胞减少 + 发育异常 (dysplasia)**，blasts **<20%**；环形铁粒幼细胞、假 Pelger-Huet；老年 / 化疗后；**可进展为 AML**。
> - **MPN (myeloproliferative neoplasm, 骨髓增殖性肿瘤)**：**细胞过度增殖**。四大：**CML (BCR-ABL)**、**真红 PV (JAK2，红细胞↑、水源性瘙痒、Budd-Chiari、LAP↑)**、**原发血小板增多 ET (血小板↑)**、**原发性骨髓纤维化 PMF（泪滴红细胞 teardrop cells、干抽、巨脾）**。
>
> 详细决策见 [[完整笔记/专题笔记/血液/血液_HighYield速查总览]] §6。

## 五、高频陷阱汇总

> [!warning] 易错点
> - **既往肿瘤 + 现 myeloblasts** → 优先想**第二原发血液恶性 (t-AML)**，**不是原癌复发**（复发不会出外周原始粒细胞）。
> - **"轻度关联" vs "经典主因"**：filgrastim 能轻度增髓系肿瘤风险，但 NBME 要的是**最强关联 = 细胞毒化疗 / 放疗**。
> - **APL 出血**不是单纯血小板低，是 **DIC**；治疗是 **ATRA 诱导分化**，不是单纯输血小板。
> - **CML 高白细胞** vs **类白血病反应**：用 **LAP**（CML↓ / 反应↑）+ **费城染色体**区分。
> - **骨髓瘤蛋白尿**：尿试纸可阴性（只认白蛋白），别因此排除——查 SSA / UPEP。
> - **HL 连续扩散 vs NHL 跳跃 / 结外**；**HL 双峰年龄**、**NHL 多老年**。

## 六、自测题（先别看答案）

> [!question]- Q1. 67 岁女性，8 年前因卵巢癌行手术 + 化疗。近 1 月夜间低热盗汗、瘀点、鼻出血；Hct 22%、WBC 30,000、Plt 30,000，外周血涂片见 myeloblasts。最可能病因？
> **既往细胞毒化疗（烷化剂）→ 治疗相关 AML (t-AML)**。外周 myeloblasts 先定性 AML，问病因选"最强关联"= 化疗 / 放疗，而非 filgrastim（弱）/ 雌激素 / 卵巢癌复发。

> [!question]- Q2. 年轻成人，外周血异常早幼粒细胞 + 严重出血（颅内 / 牙龈），PT/PTT 延长、纤维蛋白原低。诊断与遗传标志？最特异治疗？
> **APL（急性早幼粒细胞白血病）**，**t(15;17) PML-RARA**，合并 **DIC**；治疗 **ATRA ± 三氧化二砷**（诱导分化）。

> [!question]- Q3. 中年男性体检发现 WBC 显著升高、脾大，各阶段成熟中性粒细胞、LAP↓。最可能诊断与一线药？
> **CML**，**t(9;22) BCR-ABL 费城染色体**；一线 **伊马替尼 (imatinib, TKI)**。（LAP↓ 区别于类白血病反应 LAP↑。）

> [!question]- Q4. 老年男性，腰背痛 + 高钙 + 肾功能不全 + 贫血；尿试纸蛋白阴性但 SSA 阳性。下一步 / 诊断？
> **多发性骨髓瘤**：CRAB + 轻链（Bence Jones）→ 尿试纸漏轻链。查 **SPEP / UPEP + 骨髓活检（>10% 克隆浆细胞）+ 骨骼检查**。

> [!question]- Q5. Hodgkin 用 ABVD，哪个药致肺纤维化？哪个致心肌病、用什么解毒？CHOP 里环磷酰胺的特征毒性与解毒药？
> **Bleomycin → 肺纤维化**；**Doxorubicin（蒽环）→ 扩张型心肌病，dexrazoxane 解毒**；**环磷酰胺 → 出血性膀胱炎，MESNA 解毒**。

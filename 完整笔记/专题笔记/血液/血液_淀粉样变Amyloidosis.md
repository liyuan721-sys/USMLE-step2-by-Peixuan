# 淀粉样变 Amyloidosis

> [!info] 一句话定位
> **Amyloidosis (淀粉样变) = 错误折叠蛋白纤维沉积多器官**。诊断金标准 = **活检 + 刚果红染色 (Congo red) 偏振光下苹果绿双折射 (apple-green birefringence)**。分型决定病因与治疗。

---

## 一、分型（决定病因 + 治疗）

| 型 | 前体蛋白 | 病因 | 主累及 |
|---|---|---|---|
| **AL（原发）** | 免疫球蛋白**轻链 (light chain)** | **浆细胞病 / 多发性骨髓瘤 (multiple myeloma)** | 心 + 肾 + 神经 + 皮肤（最广）|
| **AA（继发）** | 血清淀粉样蛋白 A (serum amyloid A) | 慢性炎症：RA / IBD / TB / 骨髓炎 / FMF 家族性地中海热 | 肾（最常）|
| **ATTR** | 转甲状腺素蛋白 (transthyretin) | 老年野生型 / 家族遗传 | 心（限制型）+ 周围/自主神经 |
| **Aβ₂M** | β2-微球蛋白 (β2-microglobulin) | **长期血液透析** | 关节/腱鞘 → 腕管 |

（来源：分科 [[完整笔记/Peixuan分科笔记/肾脏]] 详表 + [[完整笔记/Peixuan分科笔记/心内]]）

---

## 二、多系统受累

```
淀粉样变地图
├── 心  ：限制型心肌病 → HFpEF (heart failure with preserved EF, 射血分数保留的心衰)；室壁增厚但 ECG 低电压（"二高一低"陷阱）
├── 肾  ：肾病综合征（大量蛋白尿）—— AA 最常累及肾
├── 神经：周围神经病 + 自主神经（直立性低血压）+ 腕管综合征 CTS
├── 皮肤：眶周紫癜 (raccoon eyes / periorbital purpura) + 蜡样皮肤 + 易瘀斑（AL 特征）
├── 舌  ：巨舌 macroglossia（AL 特征）
└── 肝脾：肝脾大 + 早饱 / 吸收不良
```

> [!danger] 心脏"二高一低"陷阱（分科重点）
> **超声室壁高度增厚 + 心衰症状重，但 ECG 低电压 (low voltage)** —— 因淀粉样蛋白不导电。这是与肥厚型心肌病 HCM（ECG 高电压）的关键区分。LVEF 常正常（HFpEF / 限制型）。（来源：分科 心内）

> [!tip] 题眼 → 分型（10 秒）
> - **眶周紫癜 / 巨舌 / M 蛋白 spike** → **AL**
> - **长期 RA / 慢性炎症 + 肾综** → **AA**
> - **老年 HFpEF + 腕管** → **ATTR**（先排 AL）
> - **透析多年 + 肩腕痛/腕管** → **Aβ₂M**（**DRA, dialysis-related amyloidosis 透析相关淀粉样变**；经典三联 = **双肩痛 + 腕管综合征 + 骨囊肿 bone cysts**；β₂-微球蛋白分子量大、不被普通透析膜清除 → 长期蓄积沉积于滑膜/骨）

> [!danger] AA 型肾受累的尿检"反 pattern"钩子（来源 [[mistakes/uworld-mistakes_2026-05#^Q16296]]）
> AA 淀粉样变是**慢性 RA (rheumatoid arthritis, 类风湿关节炎) 继发肾病综合征 (nephrotic syndrome) 的头号病因**。
> - 尿检是**肾病范围蛋白尿，但镜下"干净"——无红细胞、无细胞管型 (no cells / no casts)**（沉积在肾小球基底膜，是非炎症性损伤）。
> - 陷阱：别因"没有血尿 / 红细胞管型"就排除肾病、误判为肾炎综合征 (nephritic)。记：**nephrotic（淀粉样变属此）= 大量蛋白尿 + 镜下干净 + 显著水肿低白蛋白**；**nephritic = 血尿 + 变形红细胞 + RBC 管型**。

## 三、诊断

> [!warning] 金标准 = 组织活检 + 刚果红
> - **活检**：腹壁脂肪垫 (abdominal fat pad) / 直肠 / 受累器官 → **刚果红染色苹果绿双折射 (apple-green birefringence)**。〔活检部位为标准补充〕
> - **查 AL**：血/尿蛋白电泳 (SPEP/UPEP) + 免疫固定 + **血清游离轻链 (free light chains)**；阳性提示浆细胞病。
> - **分型 (typing)**：质谱 / 免疫组化区分 AL vs ATTR vs AA（治疗完全不同，必须分型）。〔标准补充〕

---

## 四、治疗（治本 = 针对病因）

> [!success] 按型治疗
> - **AL**：化疗清除异常浆细胞（如 **bortezomib 硼替佐米 / daratumumab 达雷妥尤单抗 / melphalan 美法仑** ± 自体干细胞移植）。〔标准补充〕
> - **AA**：**控制基础炎症**（RA 用 DMARD 改善病情抗风湿药/生物制剂、TB 抗痨、IBD 抗炎）。
> - **ATTR**：**TTR 稳定剂 tafamidis 氯苯唑酸**（遗传型）+ 心衰对症。〔标准补充〕
> - **Aβ₂M**：优化透析（高通量）/ 肾移植。
> - 通用支持：利尿剂缓解心衰（**谨慎，需维持充盈压**）；蛋白尿 ACEi/ARB。

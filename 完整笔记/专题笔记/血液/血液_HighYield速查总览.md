## 一、贫血总框架（Anemia — 先看网织红 + MCV）

> [!tip] 两个分流轴（最高频"下一步"）
> **第一步永远先看网织红 (reticulocyte, 网织红细胞)**：
> - **Retic ↓ / RPI ↓（reticulocyte production index, 网织红细胞生成指数）** → 生产问题（骨髓产量不足 / 骨髓衰竭）
> - **Retic ↑** → 外周丢失 / 破坏 / 潴留（溶血 / 出血 / 脾亢）
> 正常 retic 0.1–1.5%；急性溶血 retic 可 ~8%。
> 再叠加 **MCV (mean corpuscular volume, 平均红细胞体积)** 把小细胞 / 正细胞 / 大细胞分开。

```
贫血
├─ 先看 Retic
│   ├─ Retic↓ → 生产问题 → 再看 MCV
│   │     ├─ MCV↓ 小细胞 → IDA / 地中海 / 慢病(ACD) / 铁粒幼
│   │     ├─ MCV 正常 → 再障(AA) / 早期ACD / 骨髓浸润
│   │     └─ MCV↑ 大细胞 → B12 / 叶酸(folate)
│   └─ Retic↑ → 破坏/丢失 → 溶血(AIHA/G6PD/HS/PNH/MAHA) 或 急性失血
└─ Best test 起手：CBC + 外周涂片；铁代谢异常 → iron studies
```

> [!info] 婴儿生理性贫血 (physiologic anemia of infancy)
> 出生后**促红细胞生成素 (erythropoietin, EPO)↓** + 胎儿红细胞寿命短 → 血红蛋白 (Hgb) 生理性下降至**最低值 (nadir)**：**足月儿约 2 月龄低至 ~9 g/dL、早产儿约 1 月龄低至 ~7 g/dL**，之后自行回升，**无需处理**。常作"病理性贫血"的干扰（如 [[mistakes/uworld-mistakes_2026-06#^Q15845]] 2 月龄 Hgb 9.8 = 正常 nadir，非白血病/溶血）。
>
> ![[{1145EEB0-4CB5-4FF9-B611-F0112A91D9EB}.png]]
> *足月 vs 早产儿平均血红蛋白曲线：足月 ~2 月 nadir ≈9、早产 ~1 月 nadir ≈7 g/dL（UW，双嵌自卡 [[mistakes/uworld-mistakes_2026-06#^Q15845]]）*

### 1.1 小细胞性贫血（Microcytic, MCV↓）

| 病 | 钥匙(buzzword) | 铁代谢钥匙 | 下一步 / 治疗 |
|---|---|---|---|
| 缺铁性贫血 IDA (iron deficiency anemia, 缺铁性贫血) | 异食癖、匙状甲、最常见 | **铁蛋白 ferritin↓**、TIBC (total iron-binding capacity, 总铁结合力)↑、转铁饱和度↓ | 找出血源（成人 → 查 GI / 结肠镜）+ 补铁 |
| 地中海贫血 thalassemia (地中海贫血) | **小细胞 + RBC 数多 + target cells (靶形红细胞)**、RDW (red cell distribution width, 红细胞分布宽度) 正常 | 铁正常 | Hb 电泳；α 地（东南亚/亚裔背景）vs β 地（HbA2↑） |
| 慢性病性贫血 ACD (anemia of chronic disease, 慢性病性贫血) | 慢性炎症 / 感染 / 肿瘤 | **hepcidin↑** → ferritin↑/正常、TIBC↓ | 治原发病 |
| 铁粒幼细胞性贫血 sideroblastic anemia (铁粒幼细胞性贫血) | 环形铁粒幼细胞、铅中毒/酗酒/B6缺 | 铁↑ | 去诱因；铅中毒见下 |

> [!warning] hepcidin 一句话锁定
> 炎症/慢病/铁过载 → **hepcidin↑**（扣铁，ferritin↑）；血色病 hemochromatosis → **hepcidin 不当地低** → ferroportin 继续放铁 → 肠道吸铁↑、组织铁沉积。
> **铅中毒 (lead poisoning, 铅中毒)** = 获得性铁粒幼细胞性贫血：腹痛 + 神经症状；治疗 **EDTA / 二巯丁二酸 (succimer / DMSA)**，儿童铅脑病重症加 **二巯基丙醇 (dimercaprol, BAL)**。

> [!danger] 关节痛的老人贫血：先看关节是"炎"还是"磨"
> **骨关节炎 (osteoarthritis, OA) 不是炎症性关节病 → 不引起 ACD。** ACD 的机制是**炎症细胞因子**（白细胞介素-6 interleukin-6 → 铁调素 hepcidin↑ → 铁被扣在巨噬细胞内），主场是**类风湿关节炎 (rheumatoid arthritis, RA)、系统性红斑狼疮 (systemic lupus erythematosus, SLE)、慢性感染、恶性肿瘤**。
> - 有**滑膜炎 / 关节积液 / 红斑** → 想 ACD
> - **只有骨赘骨性肥大、无积液无红斑（OA）** → 别选 ACD，去看他**为止痛吃了什么**：**非甾体抗炎药 (nonsteroidal anti-inflammatory drug, NSAID) + 小剂量阿司匹林 → 消化道慢性隐性失血 → 缺铁性贫血**。⚠️ 小剂量阿司匹林常被患者当"保健品"不主动报告，老年用药史必须追问非处方药。
> - ⭐**成人查出缺铁 ≠ 终点诊断**，而是"**去找出血源**"的起点（低度慢性出血可预示日后灾难性大出血）→ 老年/男性/绝经后女性一律**双向内镜 (bidirectional endoscopy)**。
> - 老年还有**"老年特发性贫血 (idiopathic anemia of ageing)"** 这一基线，且常合并心衰/慢性肺病 → **对轻度贫血耐受很差**，不能因"只是轻度"就不查。〔源：Q3936 → [[mistakes/uworld-mistakes_2026-07#^Q3936]]〕

🧩 成人缺铁的病因追查与评估链深挖 → [[完整笔记/专题笔记/血液/血液_成人缺铁性贫血病因与评估]]

### 1.2 大细胞性贫血（Macrocytic, MCV>100）

| 病 | 钥匙 | 鉴别钩子 | 治疗 |
|---|---|---|---|
| B12 缺乏 (vitamin B12 / cobalamin deficiency, 维生素 B12 缺乏) | **神经症状（亚急性联合变性）** + 高分叶中性粒 + 甲基丙二酸↑ | 恶性贫血 (pernicious anemia) = 抗内因子抗体 | 补 B12 |
| 叶酸缺乏 (folate deficiency, 叶酸缺乏) | 高分叶中性粒、**无神经症状** | 酗酒/妊娠/甲氨蝶呤 (methotrexate) | 补叶酸（先排除 B12，否则恶化神经） |

> [!danger] B12 vs 叶酸一刀切
> 两者都高分叶中性粒 + 巨幼变；**有神经症状 → B12**。补叶酸前必须先查/排 B12，否则掩盖血象、神经损害继续进展。

### 1.3 正细胞溶血性贫血（Hemolytic — Retic↑）

> [!tip] 溶血 5 项一组
> **LDH↑ + 间接胆红素↑ + 网织红↑ + haptoglobin↓（抓游离 Hb）**；血红蛋白尿/haptoglobin 极低 → 偏**血管内溶血**。

```
溶血
├─ 自身免疫 AIHA → Coombs (DAT) 阳
│   ├─ 温抗体 IgG → 脾脏吞噬(血管外) → spherocytes → 类固醇/IVIG，难治用利妥昔
│   └─ 冷抗体 IgM → 补体介导 → agglutination + 手指/耳/鼻发蓝 → 避冷+治原发(支原体)+利妥昔
├─ G6PD → X-连锁隐性(男孩) → 诱因后急性溶血 → Heinz小体+咬细胞 → 停诱因+补液
├─ 遗传性球形红细胞增多症 HS → 溶血+黄疸+脾大 → spherocytes / EMA-binding → 脾切
├─ PNH → cola色晨尿+血栓(Budd-Chiari) → 流式 CD55/59↓ → eculizumab
└─ MAHA(微血管病性) → schistocytes(裂红) → TTP / HUS / DIC / 机械瓣
```

> [!warning] AIHA 温 vs 冷（钥匙锁定）
> - **温抗体 (warm AIHA)**：**IgG**，37°C，脾脏血管外溶血，**spherocytes (球形红细胞)**；病因 = 肿瘤 / 自免 / 淋巴瘤；最准确 = **直接 Coombs (DAT) 阳**；治 = 类固醇 + IVIG。
> - **冷抗体 (cold agglutinin disease, 冷凝集素病)**：**IgM**，补体介导，凝集 + 肢端发绀；5 个 bug "I AM HE"（流感 influenza / 腺病毒 adenovirus / 支原体 mycoplasma 最常见 / 乙肝丙肝 hep B&C / EBV）；治 = 避冷 + 治原发 + **利妥昔单抗 (rituximab)**。

> [!info] 几个易混溶血"标志"
> - **PNH (paroxysmal nocturnal hemoglobinuria, 阵发性睡眠性血红蛋白尿)**：PIGA 突变 → CD55/CD59↓ → 补体介导血管内溶血；晨起 cola 色尿、**血栓 → Budd-Chiari (布加综合征)**；**LAP↓**；最准确 = 流式 CD55/59；治 = **依库珠单抗 (eculizumab)** 抗 C5。
> - **LAP (leukocyte alkaline phosphatase, 白细胞碱性磷酸酶)**：↓ 见 PNH / CML；↑ 见类白血病反应 (leukemoid reaction) / PV。

### 1.4 再生障碍性贫血（Aplastic anemia, AA）

**全血细胞减少 (pancytopenia) + retic↓ + 骨髓 hypocellular(脂肪替代) + 无脾大 + 无泪滴细胞**。诱因：病毒 (EBV/HIV/肝炎/parvovirus)、药物（苯 benzene / 氯霉素 chloramphenicol / 烷化剂 / 抗代谢药）、放射、特发性。区别骨髓纤维化（脾大 + 泪滴细胞 + dry tap）。

> [!danger] 药物性骨髓抑制 / 粒细胞缺乏高频药
> **氯氮平 (clozapine)**（监测 ANC）、**甲巯咪唑/丙硫氧嘧啶 (methimazole / PTU)**（发热咽痛 → 立即查 CBC）、**复方磺胺甲噁唑 (TMP-SMX)**、卡马西平 (carbamazepine)、氯霉素、苯、化疗/放射。

## 二、出血性疾病（Bleeding — 先分血小板 vs 凝血）

> [!tip] 出血部位 = 一秒分流
> - **皮肤黏膜出血**（瘀点 petechiae、紫癜、鼻衄、牙龈、月经过多）→ **血小板/血管问题** → BT(出血时间)↑、PT/PTT 正常。
> - **关节/深部腔隙出血**（关节积血 hemarthrosis、深部血肿）→ **凝血因子问题** → PT 和/或 PTT↑。
> 正常值（按源文件）：BT 2–7 min；**PT 11–15 sec**；**PTT 25–40 sec**。

### 2.1 PT / PTT 判读骨架

```
PT  = 外源 + 共同通路 (因子 VII 最先掉) → 监测 warfarin、肝功能、维生素K缺、DIC
PTT = 内源 + 共同通路 (12,11,9,8)      → hemophilia A(VIII)/B(IX)、heparin、lupus anticoagulant
┌───────────────────────────────────────────────┐
│ PTT↑ PT 正常         → hemophilia / vWD / 肝素(UFH) │
│ PT↑ PTT 正常(早)     → warfarin / 维生素K缺(VII先掉)  │
│ PT↑ + PTT↑ + 血小板↓ + 纤维蛋白原↓ + schistocytes → DIC │
└───────────────────────────────────────────────┘
```

### 2.2 血小板减少（Thrombocytopenia）

| 病 | 钥匙 | 阈值 / 实验室 | 下一步 / 治疗 |
|---|---|---|---|
| 免疫性血小板减少 ITP (immune thrombocytopenia, 免疫性血小板减少症) | 病毒后 2 周、**孤立性**血小板↓、大血小板；排除性诊断 | 孤立 **<100,000**；查 **HIV/HCV** | 成人：皮肤症状 + Plt **≥30,000** → 观察；出血或 **<30,000** → 类固醇/IVIG/anti-D；难治 → 脾切 |
| 血栓性血小板减少性紫癜 TTP (thrombotic thrombocytopenic purpura, 血栓性血小板减少性紫癜) | **ADAMTS13↓** → 大 vWF 多聚体；五联(发热+神经+肾+MAHA+血小板↓) | schistocytes | **血浆置换 (plasmapheresis)** + 类固醇 ± 利妥昔；**禁输血小板** |
| 溶血尿毒综合征 HUS (hemolytic-uremic syndrome, 溶血尿毒综合征) | **血性腹泻后** STEC O157:H7、肾损为主 | schistocytes | 支持治疗 + 透析；**不用抗生素**（杀菌释毒加重） |
| 肝素诱导血小板减少 HIT (heparin-induced thrombocytopenia, 肝素诱导性血小板减少症) | 肝素后 **5–10 天**、**血小板↓ + 新血栓**（不是出血） | 抗 PF4/肝素抗体；血清素释放试验金标准 | **停肝素 → 阿加曲班 (argatroban)**；终生禁肝素 |

> [!danger] 血小板减少几条铁律
> - **HIT**：低血小板**反而血栓**（动+静脉）；Type I（<24h、可继续肝素、观察）vs **Type II（5–10 天、抗体介导、危险，停肝素换阿加曲班）**。
> - **TTP 与 HUS 都属 MAHA**，但 TTP 禁输血小板（火上浇油）；HUS 别用抗生素。
> - **AKI + 严重高血压 + 贫血 + 血小板↓ → 想 MAHA → 找 schistocytes**（硬皮病肾危象 / 机械瓣 / 恶性高血压都可触发）。
> - **假性血小板减少 (pseudothrombocytopenia)**：无症状 + 涂片见血小板大簇 → 复查（EDTA 引起聚集）。

### 2.3 凝血因子 / vWF 缺陷

| 病 | 遗传 | 钥匙 | 实验室 | 治疗 |
|---|---|---|---|---|
| 血管性血友病 vWD (von Willebrand disease, 血管性血友病) | 常染色体显性 AD | **最常见遗传性出血病**、产后出血、皮肤黏膜出血 | **BT↑ + PTT↑**（vWF 保护 VIII，轻症 PTT 可正常）；血小板形态正常 | **去氨加压素 (desmopressin, DDAVP)** 释放内皮 vWF |
| 血友病 A (hemophilia A, 血友病 A) | X-连锁隐性 | 关节腔出血、舅舅有病妈携带 | **PTT↑**、因子 VIII↓ | 轻症 DDAVP；重组 **因子 VIII**（频繁补 + 加重 → 抑制物形成） |
| 血友病 B (hemophilia B, 血友病 B) | X-连锁隐性 | 同上 | PTT↑、因子 IX↓ | 因子 IX 替代（因子检测区分 A/B） |
| 弥散性血管内凝血 DIC (disseminated intravascular coagulation, 弥散性血管内凝血) | 获得性 | 脓毒症/创伤/产科；广泛出血 | **PT↑PTT↑ + 纤维蛋白原↓ + 血小板↓ + schistocytes + D-dimer↑** | 治原发病 + 支持（FFP/冷沉淀/血小板） |

> [!info] 尿毒症性血小板功能障碍（易漏）
> **Cr/BUN↑ + 血小板数正常 + PT/PTT 正常 + 黏膜皮肤出血** → 尿毒症毒素 → 血小板黏不住聚不起。Tx：**透析**（根治），急用 **DDAVP**。

## 三、高凝状态 + 抗凝药（Hypercoagulable & anticoagulants）

- **Factor V Leiden (莱顿第五因子)**：**最常见遗传性易栓症 (thrombophilia)**。
- **抗磷脂综合征 (antiphospholipid syndrome, APS)** / **狼疮抗凝物 (lupus anticoagulant)**：体外 **PTT↑** 却体内**高凝/血栓**（反直觉）。
- **VTE (venous thromboembolism, 静脉血栓栓塞) 急性期**：首选抗凝 **LMWH (低分子肝素，如依诺肝素 enoxaparin)** 或 **Xa 抑制剂 (利伐沙班 rivaroxaban)**；**溶栓仅用于大面积/血流动力学不稳 PE**；**IVC 滤器仅用于抗凝禁忌**（近期大出血）。

> [!warning] 华法林"早期高凝"+ 桥接逻辑
> **华法林 (warfarin, 维生素 K 拮抗剂)** 先降 **蛋白 C/S（半衰期短）** → 一过性高凝 → 故起始需**肝素桥接**；同时因子 VII 半衰期最短 → **PT 最先升高**。

### 3.1 抗凝/溶栓拮抗剂（高频自测）

| 药 | 机制 | 拮抗剂 (antidote) |
|---|---|---|
| 普通肝素 UFH (unfractionated heparin) | 激活抗凝血酶 III → ↓IIa/Xa；监测 PTT | **鱼精蛋白 (protamine)** |
| 低分子肝素 LMWH | 主要 ↓Xa | 鱼精蛋白部分逆转 |
| 华法林 warfarin | ↓维生素 K 依赖因子 II/VII/IX/X、C/S | **维生素 K**；急用 **FFP / 凝血酶原复合物 PCC** |
| tPA / 阿替普酶 (alteplase) | 纤溶酶原 → 纤溶酶 溶栓 | **氨甲环酸/氨基己酸 (aminocaproic acid)** |

🌱 抗凝拮抗剂 4 对 + LMWH 逆转局限可独立拆 → 见 🔗 关联区错题。

## 四、输血反应（Transfusion reactions）

| 类型 | 机制钥匙 | 起病 | 处理 |
|---|---|---|---|
| 过敏性 anaphylactic | **IgA 缺乏者**抗供体 IgA | 秒-分钟 | 停输 + **肾上腺素 (epinephrine)** + 抗组胺/O2/液体/升压 |
| 荨麻疹性 urticarial | 受者 IgE 抗血浆可溶过敏原 | 数小时 | 停输 + **抗组胺 (苯海拉明 diphenhydramine)**；无其他症状可**恢复输注**，无需额外检查 |

> [!tip] 一句话区分
> 呼吸窘迫/喘鸣 + 血管性水肿 + 低血压 = **过敏（IgA 缺乏）→ 肾上腺素**；只痒/荨麻疹 = **荨麻疹型 → 抗组胺、可继续输**。

## 五、白细胞肿瘤（Leukemia / Lymphoma / Plasma cell）

> [!info] 起手原则
> **骨髓原始细胞 (blasts) >20% = 白血病；<5% = 缓解**。诊断金标准多为**骨髓活检 + 流式细胞术**。

### 5.1 四大白血病

| 白血病 | 年龄钥匙 | 标志 | 治疗 / 遗传 |
|---|---|---|---|
| ALL (acute lymphoblastic, 急性淋巴细胞白血病) | **儿童（峰 2–5 岁）**、最常见儿童癌 | **TdT+/CD10+**、骨痛、睾丸/纵隔肿块(T 系)、白血病侵脑 | LP 查 CNS；t(9;22) 预后差，t(12;21) 好 |
| AML (acute myeloid, 急性髓系白血病) | **双峰 15–30 与 >55** | **Auer 小体 (Auer rods)**、苏丹黑 | **APL 亚型 t(15;17) → DIC 急症 → 全反式维 A 酸 (ATRA, all-trans retinoic acid)** |
| CML (chronic myeloid, 慢性髓系白血病) | 30–50 | **t(9;22) BCR-ABL 费城染色体 (Philadelphia)**、**LAP↓**、>5% blasts | **伊马替尼 (imatinib, 酪氨酸激酶抑制剂)** |
| CLL (chronic lymphocytic, 慢性淋巴细胞白血病) | **>55** | **涂抹细胞 (smudge cells)**、常无症状 | 进展期 **利妥昔单抗 (rituximab)** |

> [!warning] 几个白血病急症/钥匙别混
> - **APL (acute promyelocytic leukemia, 急性早幼粒细胞白血病) = 医疗急症**：t(15;17)、异常早幼粒 → **DIC/凝血病 → 严重出血（颅内/肺/消化道）**；治 = **ATRA**（诱导分化）。
> - **毛细胞白血病 (hairy cell leukemia)**：>55、**巨脾**、**TRAP+** → 嘌呤类似物（克拉屈滨 cladribine / 喷司他丁 pentostatin）。
> - **LAP↓**：CML / PNH；**LAP↑**：类白血病反应 / PV。

> [!warning] 治疗相关 AML (therapy-related AML, t-AML / 继发性急性髓系白血病) — NBME11 Q143
> **既往细胞毒化疗（烷化剂 alkylating agents）/ 放疗 (radiation)** 是 AML 经典危险因素，常在原发肿瘤治疗后**多年（数年至 ~10 年）**作为**第二原发恶性肿瘤 (second primary malignancy)** 出现。
> - **题干模板**：既往肿瘤（如卵巢癌 ovarian cancer）化疗史 + 现 **全血细胞减少 / 三系异常**（贫血 + 血小板↓ 致瘀点鼻出血）+ **外周血见 myeloblasts（原始粒细胞）** → 不是原癌复发，是 **t-AML**。
> - **病因选"最强关联"**：化疗 / 放疗 ＞＞ **filgrastim (G-CSF, 粒细胞集落刺激因子)**（仅轻度增髓系肿瘤风险，是 NBME 钓饵；filgrastim 全药理对比见 [[完整笔记/专题笔记/血液/血液_造血生长因子对比]]）＞ 雌激素 (estrogen，偏激素依赖实体瘤) ＞ 原癌复发（不以外周 myeloblasts 表现）。
> - 反射：**外周血 myeloblasts → 先把题目重新定性为 AML，再回头找风险因子**。
> - 深挖（四大白血病 / APL / 淋巴瘤 / 浆细胞病展开 + ABVD/CHOP 毒性）→ [[完整笔记/专题笔记/血液/血液_血液恶性肿瘤白血病淋巴瘤骨髓瘤]]。

### 5.2 淋巴瘤（Hodgkin vs Non-Hodgkin）

| | Hodgkin (霍奇金) | Non-Hodgkin (非霍奇金) |
|---|---|---|
| 年龄 | **双峰 15–35 与 >60** | 多 >55 |
| 标志 | **Reed-Sternberg 细胞**、最多结节硬化型、**饮酒后痛**、连续扩散 | t(8;14) Burkitt、t(14;18) 滤泡型；跳跃扩散、可累及结外 |
| Dx | 淋巴结活检（best initial = biopsy） | 淋巴结活检 |
| 治疗 | I/II 无 B 症状 → 放疗；其余 → **ABVD** 化疗 | I/II 有 B 症状或 III/IV → **CHOP** |

> [!tip] B 症状 + 分期（通用）
> **B 症状 = 发热 + 盗汗 + 体重下降**。分期：I = 1 个淋巴结区；II = 同侧膈 >1 区；III = 跨膈两侧；IV = 广泛/结外。
> 化疗毒性钥匙：**博来霉素 (bleomycin) → 肺纤维化**；**多柔比星 (doxorubicin/阿霉素) → 心肌毒性（解毒 dexrazoxane）**；**环磷酰胺 (cyclophosphamide) → 出血性膀胱炎（解毒美司钠 mesna）**；**长春新碱 (vincristine) → 周围神经病/便秘**。

### 5.3 浆细胞病（MM / MGUS / Waldenström）

| | 钥匙 | 阈值 | 标志物 |
|---|---|---|---|
| 多发性骨髓瘤 MM (multiple myeloma, 多发性骨髓瘤) | **CRAB**（高钙 / 肾衰 / 贫血 / 溶骨）、缗钱状 (Rouleaux)、穿凿样溶骨灶 | **M-spike >3 + 骨髓单克隆浆细胞 >10%** | 血电泳 **IgG** M 峰；尿 **Bence Jones（轻链）**，尿试纸阴性 |
| 意义未明单克隆丙球病 MGUS (monoclonal gammopathy of undetermined significance) | 仅 IgG、**无 CRAB**、X 线正常 | **M-spike <3 + 骨髓 <10%** | 偏向 MM 的钩子 = **高钙血症** |
| 华氏巨球蛋白血症 Waldenström macroglobulinemia (华氏巨球蛋白血症) | **高黏滞综合征 (hyperviscosity)**、视物模糊+香肠样视网膜、神经病、无溶骨 | 骨髓 >10% 克隆 B 细胞 | **IgM** M 峰；治 = **血浆置换 (plasmapheresis)** |

> [!tip] 蛋白尿检测对比（骨髓瘤陷阱）
> **尿试纸只认白蛋白** → 漏轻链；**SSA 试验认全部总蛋白**；**UPEP 分类**。试纸阴性 + SSA 阳性不一致 → 锁定**骨髓瘤轻链**。
> **Gamma gap = 总蛋白 − 白蛋白**；>4 g/dL：单克隆（MM/MGUS/Waldenström）查 SPEP；多克隆（慢性感染 TB / 自免 / 慢性肝病）。

## 六、骨髓增殖性肿瘤（MPN）

| 病 | 钥匙 | EPO / 标志 | 治疗 |
|---|---|---|---|
| 真性红细胞增多症 PV (polycythemia vera, 真性红细胞增多症) | **JAK2**、**洗澡后瘙痒 (aquagenic pruritus)**、面部多血、红斑性肢痛 | **EPO 正常**、Hct↑、LAP↑ | **静脉放血 (phlebotomy)** ± 羟基脲；并发 Budd-Chiari/血栓 |
| 原发性血小板增多症 ET (essential thrombocytosis, 原发性血小板增多症) | 血小板显著↑、血栓/出血 | JAK2 部分 | （源文件未单列细节 → 🌱 TODO） |

## 七、肿瘤急症（Oncologic emergencies）

> [!danger] 五大急症一栏速记
> | 急症 | 钥匙 | 下一步 |
> |---|---|---|
> | 肿瘤溶解综合征 TLS | 高 K / 高磷 / 高尿酸 + **低钙** + AKI + 心律失常 | **IV 补液 + 拉布立海/拉布立酶 (rasburicase)** + 持续心电监护；预防加黄嘌呤氧化酶抑制剂（别嘌醇 allopurinol/非布司他 febuxostat） |
> | 高钙血症 hypercalcemia (恶性) | 嗜睡/便秘/多尿、MM/鳞癌/PTHrP (parathyroid hormone-related protein, 甲状旁腺激素相关蛋白) | **IV 生理盐水补液** → 双膦酸盐 (bisphosphonate) / 降钙素 |
> | 脊髓压迫 cord compression | 背痛 + 神经定位 + 尿潴留（前列腺/乳腺/肺转移） | **立即类固醇 (地塞米松 dexamethasone) + 急诊 MRI** |
> | 上腔静脉综合征 SVC syndrome | 面颈肿胀、上肢静脉怒张（肺癌/纵隔淋巴瘤、**ALL T 系**） | 影像定位 + 治原发肿瘤 |
> | 中性粒减少性发热 (neutropenic fever) | 化疗后 ANC (absolute neutrophil count, 中性粒细胞绝对计数)↓ + 发热 | **立即广谱抗生素**（抗假单胞 β-内酰胺，别等培养） |

> [!warning] TLS 电解质方向别记反
> TLS：**钾↑、磷↑、尿酸↑、钙↓**（钙磷结合成石）→ AKI/心律失常/癫痫。高危肿瘤 = 高负荷/高增殖（白血病、Burkitt 淋巴瘤）化疗后。

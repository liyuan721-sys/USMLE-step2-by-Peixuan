# 血小板减少 + 溶血四鉴别（HUS / TTP / DIC / ITP）

> [!info] 一句话定位
> 看到 **血小板减少 (thrombocytopenia)** 时，先问"**有没有 MAHA（微血管病性溶血性贫血）**"：
> - **有 MAHA** → 血栓性微血管病 **TMA (thrombotic microangiopathy, 血栓性微血管病)** 谱：**HUS / TTP / DIC**
> - **无 MAHA、单纯血小板↓** → **ITP（免疫性血小板减少性紫癜）**
>
> 再用 **凝血是否异常** 把 DIC 分出来，用 **前驱血便 vs 神经症状** 把 HUS 与 TTP 分开。
> 触发错题：[[NBME11_错题本#^Q067]]（19 岁女血便后 MAHA，误选 Salmonella typhi）。

---

## 一、第一步：认出 MAHA（微血管病性溶血三件套）⭐

**MAHA (microangiopathic hemolytic anemia, 微血管病性溶血性贫血)** = 红细胞被微血管内血栓"剪碎"的血管内溶血：

```
血管内微血栓
      ↓
红细胞经过狭窄血管被剪切
      ↓
┌─────────────┬──────────────┬─────────────┐
LDH↑↑        haptoglobin↓     间接胆红素↑/网织红↑
(乳酸脱氢酶     (结合珠蛋白       (溶血产物 + 骨髓代偿)
 lactate       haptoglobin       
 dehydrogenase) 被游离 Hb 耗尽)
      └──────── 外周血涂片：schistocytes（破碎红细胞）+ 血小板↓ ───────┘
```

> [!tip] 反射钩
> **"LDH↑ + haptoglobin↓ + schistocytes + 血小板↓" = MAHA → 想 TMA（HUS / TTP / DIC）**；若血小板↓但**无溶血** → ITP。

## 二、四鉴别对照表 ⭐

| | **HUS（溶血尿毒综合征）** | **TTP（血栓性血小板减少性紫癜）** | **DIC（弥散性血管内凝血）** | **ITP（免疫性血小板减少性紫癜）** |
|---|---|---|---|---|
| 全称 | Hemolytic Uremic Syndrome | Thrombotic Thrombocytopenic Purpura | Disseminated Intravascular Coagulation | Immune Thrombocytopenic Purpura |
| MAHA | ✅ | ✅ | ✅ | ❌ |
| 血小板↓ | ✅ | ✅ | ✅ | ✅（单纯）|
| **凝血 (PT/PTT)** | **正常** | **正常** | **异常↑** + 纤维蛋白原↓ + D-二聚体↑ | 正常 |
| 招牌器官 | **肾**（AKI 急性肾损伤）| **神经**（意识/局灶）+ 肾 | 全身出血 + 微血栓 | 皮肤黏膜出血（瘀点瘀斑）|
| 经典诱因/钩子 | **前驱血性腹泻**（STEC）| **ADAMTS13↓↓**（特发/获得）| 脓毒症 / 产科 / 恶性肿瘤 / 创伤 | 病毒感染后儿童 / 自身免疫成人 |
| 人群 | 多见儿童；亦见年轻成人 | 成人女性多 | 危重背景 | 儿童急性 / 成人慢性 |
| 治疗主轴 | **支持**（补液/透析），**避免抗生素** | **急症血浆置换 (PLEX)** ± 激素 | **治原发病** + 支持（输血制品按需）| 观察 / 激素 / IVIG（重症出血）|

> [!warning] 五个化验"分水岭"一眼定位
> 1. **凝血异常（PT/PTT↑、纤维蛋白原↓）→ DIC**（HUS/TTP 凝血正常！）
> 2. **前驱血便 → HUS（STEC）**
> 3. **神经症状 + ADAMTS13↓↓ → TTP**
> 4. **无 MAHA、单纯血小板↓ → ITP**
> 5. **肾损伤为主 + 无神经 → 偏 HUS；神经为主 → 偏 TTP**（两者有重叠，靠诱因 + ADAMTS13 切）

## 四、各病要点

### 4.1 HUS — STEC 型（典型 HUS）⭐
- **病原**：**STEC (Shiga toxin-producing *Escherichia coli*, 产志贺毒素大肠杆菌)**，经典血清型 **O157:H7**（未熟牛肉/生奶/农场暴露）。
- **机制**：志贺毒素 (Shiga toxin) → 肾血管内皮损伤 → 微血栓 → MAHA + 血小板消耗 + **AKI（acute kidney injury，急性肾损伤）**。
- **治疗陷阱**：**支持治疗为主（补液、必要时透析）；不常规用抗生素**——抗生素可能↑毒素释放、↑HUS 风险。
- **非典型 HUS (atypical HUS, aHUS)**：补体旁路失调，无血便前驱 → **eculizumab（依库珠单抗，抗 C5 单抗）**。

### 4.2 TTP — ADAMTS13 缺乏
- **机制**：**ADAMTS13（裂解 vWF 的金属蛋白酶）**严重缺乏（先天或获得性自身抗体）→ 超大 vWF 多聚体 → 血小板微血栓。
- **经典五联征**：MAHA + 血小板↓ + 神经症状 + 肾损伤 + 发热（**不必五项齐全**，前两项 + 无其他解释即需警惕）。
- **治疗**：**血浆置换 (plasma exchange, PLEX) 是急症一线** ± 糖皮质激素 ± 利妥昔单抗 (rituximab)。**禁忌输血小板**（火上浇油）。

### 4.3 DIC — 凝血被激活耗竭
- **背景**：脓毒症 (sepsis) / 产科灾难（羊水栓塞、胎盘早剥）/ 恶性肿瘤 / 大创伤、烧伤。
- **化验**：**PT/PTT↑、纤维蛋白原↓、D-二聚体↑、血小板↓** + MAHA（schistocytes）。
- **治疗**：**治原发病**；活动性出血按需输血小板 / 新鲜冰冻血浆 (FFP) / 冷沉淀。

### 4.4 ITP — 免疫性血小板减少
- **特征**：**单纯血小板↓、无 MAHA、凝血正常**；儿童常病毒感染后急性自限，成人多慢性。
- **治疗**：无/轻出血观察；重症出血 → 糖皮质激素 / IVIG（静脉免疫球蛋白）/ 重症加 TPO 受体激动剂。

## 五、Memory Hook

> [!success] 记忆挂钩
> - **第一刀**："血小板↓ → 有没有 MAHA（LDH↑/haptoglobin↓/schistocytes）"——有 = TMA，无 = ITP。
> - **第二刀**："凝血乱 = DIC；凝血正常再分 HUS/TTP"。
> - **第三刀**："**血便前驱 = HUS（大肠杆菌）；神经 + ADAMTS13↓ = TTP**"。
> - **治疗一句话**：**HUS 别给抗生素、TTP 立刻血浆置换、DIC 治原发、ITP 看出血**。
> - Q067 锚点："**血便之后掉血小板、LDH 爆高、haptoglobin 见底 = HUS = 先想 E. coli**"。

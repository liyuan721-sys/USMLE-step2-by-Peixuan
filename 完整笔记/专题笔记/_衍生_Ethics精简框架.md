# Ethics 精简框架 — USMLE Step 2 完整 9 大主题

> [!tip] 一句话总览
> **Autonomy 和 Privacy 都是 conditional 不是 unconditional** — 永远先扫例外 / 限制条件再决定操作。
> **好答案** = 先做基本步骤（assess baseline / reassess capacity / investigate reason）+ **collaborative**（与 family / patient 一起 plan）+ **不跳级**（不过度 escalate 到 court / ethics committee）。
> **坏答案** = ① **passive 误当 respect**（把 burden 推给 family / grandmother / adolescent 独自承担）② **过度 escalate**（一不一致就 court / 一困难就 ethics committee）③ **withholding 治疗作 leverage**（用医学合理性包装 coercion — "等 culture 阴性"/"必须签 AMA"）④ **unconditional 承诺**（"全部保密"/"把 hospital policy 当 HIPAA"）⑤ **跳过 baseline 步骤**（先 reassess capacity / 先 determine understanding，不要 jump 到 deep topic）⑥ **情绪驱动反射**（患者提 X 就直接问 X，没先 assess）。

## §一 Autonomy 4 大例外 — Override autonomy 触发条件

> [!info] 反射钩
> 看到"refusing treatment / refusing care"反射先扫这 4 大例外 — **任何一个命中 → 可 override**；都不命中 → respect autonomy。

### 4 大例外触发表

| 例外 | 触发场景 | 行动 |
|---|---|---|
| **① Public health threat** | Communicable disease（meningococcal / TB / smallpox / Ebola / plague / 反复传染高危）| **强制 hospitalize + isolate**（**不需 court order**）+ contacts prophylaxis + report 公共卫生 |
| **② Imminent harm to self / others** | Suicidality / homicidality / 严重 psychotic 危险行为 | Psychiatric hold（72-hr involuntary admission） |
| **③ Lack of decision-making capacity** | Intoxication / dementia / delirium / coma / 严重精神病 | Surrogate decision-maker（按 hierarchy）— 详 §二 |
| **④ Minor 未成年** | <18 岁（除特殊例外）| Parental consent；adolescent 可在特定主题独立 consent — 详 §六 |

### Meningococcal / 传染病例外细化（Q3625 锚定）

- **不需 court order** for infectious disease isolation — hospital + public health authority 直接 quarantine
- Meningococcal 公共卫生 SOP：**droplet precaution** + **close contact prophylaxis**（rifampin / cipro / ceftriaxone）+ report public health
- Patient 主动 refuse 让家属参与 → **不应 burden 家属做 coercion 工具**（家属角色 = support 不是 coercion）

### Autonomy override 经典陷阱

> [!warning] 4 大陷阱
> 1. **误以为需 court order**：公共健康例外（infectious disease quarantine）/ 父母不一致（joint custody 一方 consent 已足够）→ **都不需 court order**
> 2. **过度 respect autonomy 致命错**：Meningococcal home 治疗 = outbreak 风险（wife + 接触儿童）→ autonomy 不绝对
> 3. **Ethics committee 误当万能拖延工具**：明确教科书例外（公共健康 / single-parent consent）→ 不需 consult
> 4. **借家属做 leverage**：让 wife 来 convince / 让 grandmother 独自告知 → 都是把 burden 推给家属，违背 collaborative 原则

### 关联

- Q3625（meningococcal — 例外 ①）/ Q18702（IVDU AMA — autonomy 但 capacity 完好 → 不 override，努力说服）
- 例外 ③ capacity 细化 → §二
- 例外 ④ minor 细化 → §六

## §二 Decision-making Capacity 评估 4 要素

> [!info] 反射钩
> "Adult + 能说话 + refuse" ≠ 自动 autonomy；**患者拒治前必问 capacity 4 要素**。特别是患者**刚刚 incapacitated**（infection / metabolic / delirium / opioid）状态改善后 → 必须 **reassess capacity**。

### Capacity 评估 4 要素表

| 要素 | Patient task |
|---|---|
| **① Communicates a choice** | 能清楚表达 preferred treatment option |
| **② Understands information provided** | 理解 condition + treatment options |
| **③ Appreciates consequences** | 承认有此 condition + 理解各 treatment options（**含 no treatment**）的可能后果 |
| **④ Rationale given for decision** | 能 weigh risks/benefits + 给出 reasons |

> [!tip] 判定规则
> 4 要素**全部满足** → 有 capacity → 患者意愿 **supersede** surrogate / living will / family
> 任一不满足 → 无 capacity → 走 surrogate decision-maker

### Capacity 是动态的不是身份证（Q19048 核心）

```
Patient incapacitated
    │
    ▼
Surrogate 代为 decide（按 next of kin / advance directive / POA hierarchy）
    │
    │ ⏳ Patient 状态改善（infection 控制 / metabolic 恢复 / opioid 减少 / delirium 消退）
    ▼
★ 必须 reassess capacity ★（关键步骤）
    │
    ├─ Capacity 恢复 → Patient 意愿 supersede surrogate（即使与之前 surrogate decision 矛盾）
    └─ Capacity 仍无 → 继续按 surrogate
```

### Surrogate hierarchy

```
Spouse
    ↓
Adult children
    ↓
Parents
    ↓
Siblings
    ↓
其他 next of kin
```

### 4 类 escalation 工具对比

| 工具 | 触发场景 |
|---|---|
| **Surrogate decision-maker** | Patient 无 capacity + 有 next of kin / advance directive / POA |
| **Two-physician consent** | Patient 无 capacity + **无 surrogate + 无 living will**（some states） |
| **Court order** | Both parents refuse life-saving treatment for minor（详 §六）/ specific 强制治疗 non-emergent 情境（部分州 / 非传染病）|
| **Ethics committee** | **真正 intractable disagreement**（family vs care team active 冲突）— 明确教科书例外 / family 仅 reluctant 都不需 |

### 评估顺序（先 capacity 再其他）

> [!warning]
> "Adult + 能说话 + refuse = autonomy"反射陷阱 — 必须先 reassess capacity，特别患者刚刚 incapacitated。Depression 评估 / 走 surrogate / document refusal 都在 capacity 评估**之后**。

### 关联

- Q19048（capacity reassessment / surrogate ≠ permanent）
- Capacity 不满足 → §一 autonomy 例外 ③
- 慢性 dementia / delirium → §八 end-of-life decision

## §三 AMA/against medical advice 管理三步法 + Patient Communication

> [!info] 反射钩
> AMA discharge = **beneficence > 简单 respect autonomy** → 努力 prevent AMA。好答案永远 empathic + investigate + offer alternative；坏答案永远 conditional / coercion / punitive / withholding。

### AMA 管理三步法

| 步骤                                       | 内容                                      | 关键操作                                                                                                                                                                                                     |
| ---------------------------------------- | --------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **① Investigate / address 离院理由**         | 多 empathic + open-ended + nonjudgmental | Financial → social worker；**Substance withdrawal → MAT/medication-assisted treatment**（methadone / buprenorphine）⭐ IVDU 必问；Care delay 不满 → apologize + accelerate；Caregiver 责任 → home care / social work |
| **② Ensure + document informed refusal** | 确认 + 记录 capacity                        | Decision-making capacity ✓；understands risks/benefits ✓；reasoned basis ✓；sign AMA form（**documentation 不是 prerequisite 给药**）                                                                             |
| **③ Provide best possible alternative**  | 永远不 punitive                            | Offer appropriate Rx（即使非标 — empiric oral abx before culture 也行）；offer follow-up + 强调 "resume treatment anytime"；**不 withholding antibiotics 作 leverage**                                                 |

### IVDU + AMA 特殊点（Q18702 锚定）

> [!tip] IVDU AMA 必问 withdrawal/cravings → MAT
> IVDU 患者 AMA 高危 → 直接非 judgmental 问 **cravings / withdrawal**（不绕弯）→ 识别即可启动 MAT。
> 临床证据：**methadone 减 AMA 50% + 1-yr overdose 死亡 75%**。

### MAT 速查（Medication-Assisted Treatment）⭐

> [!info] 定义
> **MAT = FDA 批准药物 + 行为治疗 / counseling 联合**治疗物质使用障碍（SUD）。最常用 **OUD (Opioid Use Disorder)**，也覆盖 **AUD (Alcohol Use Disorder)**。
> 道德反射陷阱：「**MAT ≠ 把毒品合法化**」— evidence-based 一线，physician 不应 punitive 拒绝（同 Q18702 Choice E "won't prescribe"陷阱）。

#### OUD 三大药对比

| 药 | 类别 | 机制 | 关键特点 |
|---|---|---|---|
| **Methadone** | **Full** μ-opioid agonist | 完全激动 μ 受体 → 减 craving + 减 withdrawal | 长效 24 hr；**只能 federally licensed clinic 每日给药**；减 AMA / overdose 死亡证据最强 |
| **Buprenorphine**（± naloxone = **Suboxone**）| **Partial** μ-opioid agonist | 部分激动 + **ceiling effect**（过量较 methadone 安全）| **Office-based** physician 可开（2023 后取消 X-waiver）；Suboxone 加 naloxone 防 IV 滥用（po 无效，注射激活拮抗）|
| **Naltrexone**（IM = **Vivitrol** monthly）| **μ-opioid antagonist** | 阻断 μ 受体 → opioid 不 high | 不致成瘾；po 依从性弱 → monthly IM 解决；**也用于 AUD** |

#### 启动时机陷阱（precipitated withdrawal — USMLE 高频）

> [!danger] 三药启动时机不同 — 启动太早 = precipitated withdrawal（突发严重戒断）

| 药 | 启动前 patient 状态 | 误启风险 |
|---|---|---|
| **Methadone** | 等 **mild withdrawal sign**（COWS scale 评估） | 启动太早 + 体内已有 opioid → overdose |
| **Buprenorphine** | 等 **COWS ≥ 12** + 最后 short-acting opioid **≥ 12 hr** / methadone **≥ 36 hr** | 启动太早 → **precipitated withdrawal**（partial agonist 把 full agonist 踢下 receptor 反而暴发戒断）|
| **Naltrexone** | 等 **abstinent ≥ 7-10 天** | 启动太早 → precipitated withdrawal（直接 antagonist 暴力踢 opioid）|

#### AUD（Alcohol Use Disorder）三大药

| 药 | 机制 | 关键特点 |
|---|---|---|
| **Naltrexone**（一线）| μ-opioid antagonist | 减 alcohol craving + reward；同 OUD 同药 |
| **Acamprosate**（一线）| NMDA modulator | 减 protracted withdrawal symptoms；肾排泄（CKD 避）|
| **Disulfiram**（二线）| 抑 aldehyde dehydrogenase | 喝酒后头痛/呕吐/潮红 → aversion；依从性差（患者得自己配合不喝）|

#### MAT 反射钩

> [!success] 一句话锁
> - **OUD 三药**：「**Methadone full agonist clinic 给 / Buprenorphine partial office 给 / Naltrexone antagonist monthly IM**」
> - **启动时机**：「**Methadone 等 mild withdrawal / Bup 等 COWS≥12 / Naltrexone 等 abstinent 7-10 天**」— 都防 **precipitated withdrawal**
> - **AUD 三药**：「**Naltrexone craving / Acamprosate withdrawal / Disulfiram aversion**」
> - **道德反 pattern**：「**MAT 是治疗不是 enabling**」— 拒绝 prescribe MAT 给 IVDU 患者 = punitive + 违背 beneficence

### 好答案 vs 坏答案 features

```
✅ 好答案：
  • Empathic phrasing（"I want what's best for you"）
  • Open-ended question
  • Investigate 具体原因（IVDU 必问 cravings/withdrawal）
  • 不 coerce 不 punitive

❌ 坏答案 4 大模式：
  • Conditional："only if promise to return" / "only if sign AMA form"
  • Coercion 包装医学合理性："need cultures negative first"（withholding 作 leverage）
  • Punitive："won't prescribe any medication"
  • Withholding 治疗 = leverage（任何"X 条件满足才给治疗"= coercion）
```

### 关键反 pattern

> [!warning] "Withholding 治疗作 leverage" = coercion
> 不管包装多医学合理（如 cultures 还没回来）— 任何"必须 X 才给 Y"= coercion，违背 beneficence。
> AMA form = **documentation 工具**不是 prerequisite 给药；good-faith 失败仍要 offer best alternative（empiric oral abx），不 punitive。

### 关联

- Q18702（IVDU IE AMA empathic investigation）
- Capacity ✓ 是 AMA documentation 前提 → §二
- IVDU + MAT → 跨学科 [[完整笔记/Peixuan分科笔记/psych]]

## §四 Adolescent Conditional Confidentiality

> [!info] 反射钩
> 青少年问"你会告诉我父母吗？" → 反射答 **"private except very specific circumstances"** + 解释 circumstances。不要 unconditional promise（自掘坟墓）。

### 默认 private + 5 大 breach 例外

| 例外触发 | 内容 |
|---|---|
| **① Imminent risk** | Suicidality / homicidality / 严重自伤 |
| **② Safety threats** | Sexual abuse / 严重 physical abuse / intoxicated driving |
| **③ Reportable infections** | State-specific（某些 STI / TB 等）|
| **④ Legal mandate** | State-specific（如 child abuse reporting）|
| **⑤ Patient 自己同意 disclose** | — |

### Early transparent framework > unconditional promise

> [!tip] 长期 trust 的悖论
> 给 adolescent 明确知道哪些 private / 哪些必须 breach → 反而建立长期信任。
> Unconditional promise（"全部保密"）在 imminent risk 时**必破产** = unethical promise + 自掘坟墓。

### Breach 流程

```
识别需 breach 例外
    │
    ▼
先 discuss with adolescent
    │
    ▼
给 option 让 adolescent 主动告知 guardian（physician 陪同支持）
    │ 拒绝 / 不合作
    ▼
Physician 直接告知 guardian（必要时）
```

### HIPAA ≠ adolescent unconditional privacy（5 漏洞）

| 漏洞 | 原因 |
|---|---|
| **EMR / patient portal** | Guardian 可 legally access 青少年 medical record（除 sensitive 主题各州 vary）|
| **直接 medical record requests** | Guardian 法律权 access |
| **Insurance billing statements** | 可能露 diagnosis / 来诊原因 |
| **HIPAA 本身局限** | HIPAA 不 ensure adolescent unconditional privacy |
| **州法律** | 强制 disclose 特定情境 |

### 关键陷阱

> [!warning]
> - "Everything 保密 between you and me only" → unconditional 撒谎，imminent risk 时必破产
> - "Won't volunteer but respond truthfully if parents ask" → 给 parents 破解权 = unconditional 失败
> - "Obligated to keep parents informed because you're a minor" → physician 实际**不**默认 obligated（仅例外时）
> - "HIPAA ensures everything private" → HIPAA 不保 unconditional privacy（guardian 可 legally access）

### 关联

- Q105595（青少年 conditional confidentiality）
- HIPAA 完整 SOP → §五
- 青少年独立 consent 主题（STI/SUD/pregnancy/contraception）→ §六
- Adolescent depression + 物质试用 + suicidality 评估 → 跨学科 [[完整笔记/Peixuan分科笔记/psych]]

## §五 HIPAA Disclosure SOP

> [!info] 反射钩
> Disclosure 题 → 先问 authorization 在不在（**verbal OR written 都行**）→ 再问透露范围（**minimum necessary**）→ 排除 sensitive default 不透露。

### HIPAA disclosure 5 要素

| 要素 | 规则 |
|---|---|
| **① Authorization 形式** | **Verbal OR written 都 OK**（HIPAA 不强制 written；hospital policy 可 require written 但**非 HIPAA**）|
| **② Patient 在场** | **不必** — 已 authorize 即可 |
| **③ 身份验证** | HIPAA **不**要求 verify caller identity；hospital policy 可 require |
| **④ 披露范围** ⭐ | **Minimum necessary disclosure** — 只透露请求所需，不主动 expand |
| **⑤ Sensitive 信息特殊** | 即使 authorize 也 default **不**透露 sensitive，除非明确包含 |

### Sensitive info 5 类（default 不透露）

```
Mental health
Substance use disorder (SUD)
HIV
Abortion
Genetic information
```

### Minimum necessary 应用例（Q3614 employer disclosure）

| Employer 请求 | 可透露 | 不可透露 |
|---|---|---|
| "update" | confirm hospitalized | specific diagnosis（alcoholic hepatitis）|
| "likely return to work?" | 预期 return（一般 timeframe）| underlying condition / 治疗细节 |

### HIPAA ≠ hospital policy

> [!tip] 关键 reframe
> **HIPAA 是 floor 不是 ceiling** — hospital policy 可 stricter（require written / verify ID），但 **HIPAA 本身允许 verbal + 不 verify ID**。
> 把 hospital policy 误等于 HIPAA 强制 = USMLE 经典陷阱（过度保守）。

### 关键陷阱

> [!warning]
> - "Need written signed release" → 把 hospital policy 当 HIPAA 强制
> - "Patient must be present" → 已 verbal consent 即可，不必安排 meeting
> - "Cannot verify caller identity" → HIPAA 不要求（policy 可要）
> - "Disclose diagnosis 给 employer" → 超 minimum necessary + 透露 sensitive（SUD-related）

### 关联

- Q3614（HIPAA verbal consent + minimum necessary）
- Q105595（HIPAA 不保 adolescent privacy — 5 漏洞）→ §四
- Alcoholic hepatitis 跨学科 → [[完整笔记/Peixuan分科笔记/GI]]

## §六 Minor Consent + Court Order Escalation

> [!info] 反射钩
> 父母不一致 → 先看 ① 有没有 one parent consent？✓ 法律足够 ② best interest？✓ proceed ③ 都 refuse + life-saving？→ court。
> **不要看到"父母不一致"反射 court order**（过度 escalate）。

### Joint custody consent 决策树

```
父母 divorced + joint custody
    │
    ├─ 仅一方有 custody → 只该方 consent 有效
    ├─ Joint custody + both agree → consent ✓
    ├─ Joint custody + 一方 consent + 一方 refuse →
    │      → 该方 consent 法律足够 proceed（especially best interest）
    │      → Ideally 仍与双方 reach agreement 但不必
    │
    └─ Joint custody + both refuse:
           │
           ├─ Elective / routine（如常规疫苗）→ tolerate（herd immunity 风险低）
           └─ Life-saving / urgent → Court order ⭐
                 ※ 例：双方 refuse acute leukemia chemo
```

### 父母不一致 → 听谁的？⭐

> [!tip] 一句话答案
> **听想 consent 治疗的那方**（一方 consent 在法律上就够，另一方 refuse 不能单独 block treatment）。

#### 为什么是"听 consent 方"而不是"听 refuse 方"？

USMLE 隐含 default：**Indicated 治疗 = child's best interest**。

```
Joint custody + 双方分歧
    │
    ▼
默认假设：医学 indicated 治疗符合 best interest
    │
    ├─ 想 consent 那方 → 单独就能授权 treatment（✓ 满足 best interest + 法律 consent）
    └─ 想 refuse 那方 → 单独 block 不了（除非另一方也 refuse）
```

**因此 asymmetric 不对称**：
- 想 **consent** 的那方：单独就能授权 → 可 proceed
- 想 **refuse** 的那方：单独 block 不了 treatment

#### Default 反转的 7 种特殊场景

| 场景 | 听谁？ |
|---|---|
| **一方 consent + 一方 refuse + 治疗 indicated**（Q3624 模式） | 听 **consent 方** ✓ |
| 一方想做 elective / 非 indicated 高风险 + 一方 refuse | 听 **refuse 方**（保护 child from 非必要风险）|
| 双方都 refuse + 治疗 routine（如常规疫苗） | 通常 **tolerate**（herd immunity 风险低）|
| 双方都 refuse + life-saving（Jehovah's Witness 拒输血 / 拒化疗） | **Court order** ⭐ |
| Emergency（延误致 severe outcome） | **谁都不用听** — emergency exception 直接 proceed |
| Sole custody / divorce decree 指定 medical decision-maker | 按 decree 指定的那方 |
| Mature / emancipated minor / specific medical care (STI/SUD/pregnancy/contraception) | 听 **minor 自己** |

#### Q3624 题逻辑应用

```
Mother consent tetanus（urgent + high-risk puncture）
Father refuse（want herbal）
        │
        ▼
听 Mother（一方 consent = 法律足够）→ proceed
        │
        ├─ ※ Ideally 仍与 father reach agreement，但不必
        ├─ ※ 不需 court order（court 只在 both refuse life-saving 时）
        └─ ※ Bonus fail-safe：tetanus from deep puncture 也满足 emergency exception
```

#### 反射钩

> [!success] 一句话锁
> 「**Indicated 治疗 + joint custody + 一方 consent → 听 consent 方（一方就够）；不是"少数服从多数"也不是"必须 both agree"**」
> **想 consent ✓ 一方就够 / 想 refuse ✗ 一方 block 不了**（不对称）

### Minor 独立 consent 3 类例外

| 例外类别 | 触发场景 |
|---|---|
| **① Emergency care** | 延误致严重 impairment / death（如 tetanus from deep puncture — 紧急时不需 consent） |
| **② Emancipated minor** | Parent / married / military service / financially independent / HS graduate / homeless |
| **③ Specific medical care (adolescents)** | **STI / SUD / pregnancy care / contraception**（most states）|

### 4 类 escalation 工具适用场景对比

| 工具 | 用于... | 不用于... |
|---|---|---|
| **Court order** | Both parents refuse life-saving treatment for minor / 部分州 non-emergent 强制 | Single-parent refuses（一方 consent 已够）/ infectious disease quarantine（hospital + public health 直接）/ routine 疫苗双方拒 |
| **Ethics committee** | 真正 intractable disagreement（family vs care team active 冲突）/ ethical dilemma 不清楚 | 明确教科书例外 / family 仅 reluctant 不 disagree / 单纯拖延 |
| **Surrogate decision-maker** | Patient 无 capacity + 有 next of kin / POA / advance directive | Capacity 完好的 adult（即使 refuse）/ minor 有 parents（用 parental consent） |
| **Two-physician consent** | Patient 无 capacity + **无 surrogate + 无 living will**（some states）| 有 surrogate 时 |

### Unemancipated minor 法则

- **不能自己 consent**（可 assent 但 legal consent 必 parent / guardian）
- 但 emergency 例外允许 proceed without consent（fail-safe）

### 关键陷阱

> [!warning]
> - "父母不一致 → 反射 court order" → one parent consent 已足够（especially best interest）
> - "ask 9-year-old patient about his wishes" → unemancipated minor 没 legal authority（可 assent 不能 consent）
> - "Withhold treatment" → mother 已 consent = 法律足够；withholding life-saving = potential harm
> - Routine vaccine 双方 refuse → tolerate（herd immunity 风险低）；urgent treatment 双方 refuse → court

### 关联

- Q3624（joint custody one parent consent — tetanus）
- 公共健康例外不需 court order → §一
- Tetanus prophylaxis → 跨学科 [[完整笔记/Peixuan分科笔记/免疫，疫苗]]

## §七 Pediatric Chronic Illness Disclosure 全流程

> [!info] 反射钩
> 儿童 raise concerns → "Tell me what you understand about..." 开头（先 assess understanding）→ 再 empathic respond + 后续 collaborative plan with parents。
> 不能 ① defer 给 parents ② encourage 患儿独自 broach ③ evasive 撒谎 ④ abrupt 直接 raise difficult topic。

### Disclosure 4 阶段全流程

```
阶段 1：先 determine patient's current understanding（baseline）⭐
    │      ※ 关键 — 不能 defer / 不能 evasive / 不能直接 raise 死亡
    ▼
阶段 2：基于 understanding，empathic 提供 timely comfort + support
    │
    ▼
阶段 3：基于 patient understanding，与 parents collaboratively plan
    │      ※ Timing / approach / extent of disclosure
    │      ※ Joint participation（不让 family 独自承担 burden）
    ▼
阶段 4：joint disclosure（family + physician 一起 deliver）
           ※ Follow-up support + adherence monitoring + 性教育（如 HIV）
```

### Pediatric ethics 三方平衡

| 方 | 原则 | 操作 |
|---|---|---|
| **Patient（儿童）** | Primary duty + 知情权按发育水平 + emerging autonomy | 患者 raise 的 concerns 不能 defer 不能 dismiss |
| **Parents** | Authority to make decisions + respect concerns | 不 overtly 讨论 "negative"（暂时遵守），但同时 collaboratively 推进 plan |
| **Physician** | Balance + honesty + collaborative | 不撒谎 / 不 evasive / 不单方面 unilateral disclose |

### 4 大常见陷阱

> [!warning]
> 1. **Defer 给 parents**（"ask permission to share with parents"）→ 把 burden defer + 没回应 patient 当下 concerns
> 2. **Encourage 患儿独自 broach**（"encourage patient to talk to parents"）→ 让 11 岁独自承担 difficult discussion
> 3. **Evasive 撒谎安慰**（"will likely feel better after another round of chemo"）→ medical 不诚实 = 损 trust（induction 失败时不知道下轮会好）
> 4. **Abrupt 直接 raise difficult topic**（"inquire about death and dying perception"）→ 太早 jump 到 deep topic without 先 understand baseline

### Disclosure 之前 vs 之后好处对比（Perinatal HIV 特化 — Q19132）

| 因素 | 不 disclose 风险 | Disclose 后好处 |
|---|---|---|
| **Patient trust** | 知 truth 时 trust 受损 | Build patient-physician trust + 长期 relationship |
| **Medication adherence** | 不知 why 服药 → adherence ↓（adolescence 加重）| ↑Adherence（理解 why） |
| **Transmission prevention** | Sexual exploration + 不知 HIV → inadvertent transmission to partners | ↓Inadvertent transmission + 学会 safe sex / disclosure to partners |
| **Psychosocial** | Anxiety / 怀疑 / 偶然 self-discover 创伤 | Adjusted with support |

### Perinatal HIV 特殊（Q19132 锚定）

> [!tip] Adolescence 之前必须 disclose（3 大理由）
> 1. **Fosters trust** + autonomy
> 2. **↑Medication adherence**（理解 why 服药）
> 3. **↓Inadvertent transmission**（adolescence 是 sexual exploration 期）

### "Respect family" ≠ passive

> [!danger] 核心 reframe
> 真正 respect = 与 family **一起 plan**（give timeline + joint participation），**不是把 burden 全推给 family 独自承担**。
> 3 个 not-good 模式：① 只强调 risks of delay（instill distrust）② passive 推 burden 给 family（not collaborative）③ unilateral partial disclose without consent（不 respect family）。

### Advance Care Planning for Adolescent with Chronic Life-Limiting Illness（Q21658 锚定）⭐

> [!info] 反射钩
> Adolescent + **chronic life-limiting illness**（CF / HFrEF / 神经退行性 / cancer 等）+ **deteriorating** → **family-centered advance care planning**：patient 作 **participant alongside parents**。
> ❌ 不 defer 到 18 岁 ❌ 不 wait until stable ❌ 不只 parents 决定 ❌ 不 defer 到 PCP（急性恶化中不能拖）。

#### 核心原则：Patient 是 participant 不是 bystander

| 维度 | ❌ 错误 model | ✅ Family-centered model |
|---|---|---|
| **Adolescent 角色** | 被动 — wait until 18 才有发言权 | **Active participant** alongside parents |
| **Parents 角色** | 唯一 decision-maker | **Co-participant**（保留 legal authority + 与 adolescent 联合 discuss）|
| **Physician 角色** | 仅 transmit info to parents | **Facilitate joint discussion** + 教育 + 建立 trust |
| **时机** | Wait until stable / wait until 18 | **Early + 主动**（chronic life-limiting illness 一旦 deteriorating signal 出现）|

#### 启动信号（不能 defer — 必须立即启动 advance care planning）

```
Chronic life-limiting illness（CF / HFrEF / 神经退行性 / cancer 等）
    │
    │ 出现 deteriorating signal：
    ├─ Functional measure 显著下降（如 CF FEV1 <40% predicted）
    ├─ 反复住院 / exacerbations 频率增加
    ├─ 治疗 response 减弱
    ├─ 新并发症（如肺动脉高压 / 营养不良 / 心律失常）
    ▼
★ 立即启动 family-centered advance care planning ★
    （不 wait stable，不 wait 18，不只 parents 决定）
```

#### Family-centered 5 大好处（反向直觉 — USMLE 必考）

> [!success] 反向直觉点
> 直觉以为"讨论 advance care 会 demotivate adolescent / 加重 anxiety" → **反向真相**：

| 好处 | 机制 |
|---|---|
| **↓ Anxiety** | Adolescent 知道 wishes 被听见，不怕被忽视 |
| **↑ Maintain quality of life** | Preferences honored — 治疗与价值观一致 |
| **↑ Adherence** | Adolescent 理解 + 参与决定 → 更愿意 follow |
| **避免 crisis decision-making** | 提前 plan → 急性 decompensation 时不必 family panic 决定 |
| **Ensure preferences honored** | 若 status decline → 已知 patient preferences（防 lose opportunity）|

#### 关键陷阱（Q21658 触发）

> [!danger] "Minor = parents 全权" 反射陷阱
> Parents 确实保留 legal authority for minors，**但** chronic life-limiting illness 的 adolescent 有 **maturing capacity to appreciate medical implications** → physician 应 **encourage parents to include minors in treatment decisions**，不是 bypass adolescent。
> **不是 binary "成人/儿童"** — adolescent 是连续 spectrum，chronic life-limiting illness 触发 special participation status。

#### 与 §八 衔接（advance care planning → end-of-life decision）

```
adolescent 还有 capacity + chronic life-limiting illness deteriorating
    │
    ▼
§七 Family-centered advance care planning（Q21658）
    │  patient 是 participant，preferences 记录在案
    │
    │ 真到 incapacitated（severe decompensation）
    ▼
§八 End-of-life 3 philosophies（Q21627 NEOT）
    │  已有 advance directive → 按 patient wishes
    │  没有 / partial → surrogate 用 substituted judgment + shared values
```

## §八 End-of-Life Care 3 Philosophies

> [!info] 反射钩
> Family 卡在"全力救 vs 撤一切"两极 → 提 **NEOT 中间地带**；不要 dismiss "don't feel guilty"情绪 → reframe 选择。
> Surrogate 决定指南 = **substituted judgment + shared values**，不是 simply advising "don't feel guilty"。

### End-of-life 3 philosophies 对比表

| 维度 | No Escalation of Treatment (NEOT) | Time-Limited Trial | Withdrawal of Life Support |
|---|---|---|---|
| **核心** | 设 predetermined upper limits on treatment | Intensive care for predefined interval (2-3 days) | 完全 comfort measures only |
| **当前 care** | **不撤**已有 care | 暂时 escalate 看反应 | **主动撤**生命支持 |
| **新 escalation** | **不再**新增（no dialysis / no vent / no invasive）| 集中尝试后评估 | 不适用 |
| **避免** | Futile interventions with greatest morbidity | 草率 withdraw 而无尝试 | Futile prolongation |
| **Family 心理** | 减 "withdrawing" 的 guilt（不是 active 撤）+ 有时间 process loss | 有 closure：尝试过了 | 直接面对 loss + 可能 guilt |
| **结局** | 通常 active dying 随后 | 评估后或撤或继续 | Active dying soon follows |
| **适用场景** | Family reluctant withdraw + prognosis 明确 poor + further escalation futile | Diagnosis 不明 / treatment response 不明 / family 需 closure | Family 同意 + futile + 患者 wishes 明 |

### "Withdrawing life support ≠ withdrawing care" 关键 reframe

> [!success] NEOT 不是放弃
> "**Withdrawing life support ≠ withdrawing care**" — NEOT 是**继续提供 supportive care**（pain control + comfort + 已上药物 + IV fluids + 抗生素）+ 仅**限制新 escalation**。
> 这是 family 怕 guilt 时的关键 reframe — 不是放弃，是停在已尽 reasonable interventions 的 line 上。

### Surrogate decision 指南

> [!tip] Substituted judgment + shared values
> Surrogate 不应按"自己想让 patient 接受什么治疗"决定，应按 **"patient would have wanted"（substituted judgment）+ shared familial / cultural values** 决定。
> Simply advising "don't feel guilty" 无效 — 没解决 family 的核心 conflict；要给 surrogate **decision framework** 不是情绪 reassurance。

### 关键陷阱

> [!warning]
> - "Reassure family not to feel guilty"（A 类）→ 没给 surrogate decision framework，只是情绪 reassurance
> - "Ethics committee"（B 类）→ 用于 intractable disagreement（active 冲突），不是 family reluctant
> - "Quality of life argument"（D 类）→ Quality of life **主观 + 文化差异大**；physician 不单方面 judge
> - "Respect family wishes + continue escalation"（E 类）→ Patient deteriorating + family 实际 reluctant（不是 firm commitment）+ futile → frank discussion 才负责任

### Advanced directive vs Living will vs Surrogate

```
有 advanced directive / living will 明确指定 → 按 patient wishes
    │
    └─ 不指定 / partial → Surrogate decision-maker（hierarchy）
            │
            └─ 用 substituted judgment + shared values 决定
                    （不是 surrogate 自己 preference）
```

### 关联

- Q21627（NEOT — end-of-life 3 philosophies + surrogate substituted judgment）
- 前提是 capacity 无 → §二（Q19048 capacity reassessment + advanced directive 不指定时走 surrogate）
- 跨学科 [[完整笔记/Peixuan分科笔记/neuro]]（Advanced Alzheimer dementia 自然病程）/ [[完整笔记/Peixuan分科笔记/感染]]（重症 sepsis + multi-organ failure 预后）

## §九 Ethics 答题 6 大反射钩陷阱（元归纳）

> [!danger] 9 道错题元提炼 — 6 大反射钩陷阱
> 把所有 "看起来合理但实际是错答案" 的 pattern 拆解出 6 类。考前默念这 6 个，遇陷阱反射识别。

### 陷阱 1：情绪驱动反射

**Pattern**：Patient 提 X 就直接问 X，没先 assess baseline understanding。

**例**：
- Q7283：11 岁 girl 提"everyone dies" → 反射直接问 death/dying perception（错；应先 assess current understanding）
- Q21627：Family 说"don't feel guilty for withdrawing" → 反射 reassure "don't feel guilty"（错；应给 NEOT framework）

**对策**：永远先 **assess baseline / current understanding**，不要 jump 到 deep topic。

### 陷阱 2：Passive 误当 Respect

**Pattern**：把 burden 全推给 family / grandmother / adolescent / patient 独自承担，误以为是"respect autonomy"。

**例**：
- Q19132：grandmother 不让 disclose HIV → 选"won't discuss without permission but recommend tell ASAP"（错；passive 推 burden 给 grandmother，没 collaborative role）
- Q7283：选 "ask permission to share with parents" / "encourage patient to talk to parents"（错；defer 给患儿独自 broach）
- Q3625：选"让 wife 来 convince"（错；burden 家属做 coercion 工具）

**对策**："Respect family" = 与 family **一起 plan**（give timeline + joint participation），不是 hands-off。

### 陷阱 3：过度 Escalate

**Pattern**：一不一致就 court order / 一困难就 ethics committee。

**例**：
- Q3624：父母不一致 → 反射 court order（错；one parent consent 已足够）
- Q3625：传染病 refuse → 反射 court order（错；hospital + public health 直接 quarantine）
- Q3625 / Q21627：选 ethics committee 拖延（错；明确教科书例外 / family 仅 reluctant 不 active disagree 都不需）

**对策**：
- Court order 仅 both parents refuse life-saving 时
- Ethics committee 仅 active intractable disagreement 时
- 默认先用 standard procedure（one parent consent / surrogate / quarantine）

### 陷阱 4：跳过 Baseline 步骤

**Pattern**：跳过应先做的 baseline assessment，直接 jump 到结论 / 操作。

**例**：
- Q19048：患者 refuse surgery → 选 document refusal（错；应先 reassess capacity，patient 刚刚 incapacitated）
- Q7283：选 inquire about death perception（错；应先 determine current understanding）
- Q19048：选 assess depression（部分相关但先后顺序错；capacity 优先）

**对策**：
- 患者拒治 → 先 reassess capacity 4 要素
- 儿童 raise concerns → 先 determine current understanding
- AMA → 先 empathic investigate reason

### 陷阱 5：Withholding 作 Leverage

**Pattern**：用医学合理性包装 coercion — "等 X 条件满足才给 Y 治疗"。

**例**：
- Q18702：选"need cultures negative first"（错；withholding antibiotics 作 leverage 留 patient）
- Q18702：选"only if sign AMA form"（错；AMA form = documentation 不是 prerequisite 给药）
- Q18702：选"only if promise to return"（错；conditional coercion）
- Q18702：选"won't prescribe any medication"（错；punitive，good-faith 失败仍 offer alternative）

**对策**：任何 "X 条件满足才给治疗" = coercion，违背 beneficence；good-faith 失败仍 offer best alternative。

### 陷阱 6：Unconditional 承诺 / 误当 HIPAA 强制

**Pattern**：① 给 adolescent unconditional 保密承诺 ② 把 hospital policy 当 HIPAA 强制要求 ③ 引 HIPAA 作 unconditional 隐私承诺。

**例**：
- Q105595：选 "everything between you and me only"（错；imminent risk 时必破产 = unethical promise）
- Q105595：选 "HIPAA ensures everything private"（错；HIPAA 不保 adolescent unconditional privacy — 5 漏洞）
- Q3614：选 "need written signed release"（错；HIPAA verbal OK，hospital policy 可 stricter 但 ≠ HIPAA 强制）
- Q3614：选 "patient must be present"（错；HIPAA 不要求）

**对策**：
- Autonomy / Privacy / Confidentiality 全是 **conditional**，永远先扫例外
- HIPAA = floor 不是 ceiling（hospital policy 可 stricter，但 HIPAA 本身允许 verbal + 不 verify ID）
- 给 transparent framework（"private except specific circumstances"）= 长期 trust

# Horner Syndrome (霍纳综合征) — 三级定位与病因

> [!info] 笔记用途
> Horner 综合征是一个**跨多疾病的高价值 schema** — 解剖通路决定病因谱，**伴随征决定定位等级**。
> USMLE 几乎不会单独考"Horner 是什么"，而是考"Horner + X 伴随征 → 哪种病？"
> 来源：Q20435 病程错题（Syringomyelia → Horner 演变）+ Neuro HY200 听课素材整合。

---

## 一、核心三联征

> [!tip] Horner 三联（**单侧**！）
> 1. **Ptosis（上睑下垂）** — 轻度（1–2 mm，Müller 肌瘫）
> 2. **Miosis（瞳孔缩小）** — 同侧瞳孔小，对光反射保留
> 3. **Anhidrosis（无汗）** — 同侧面部，**仅一、二级有**；三级可缺失（partial Horner）
>
> 额外可见：**Enophthalmos**（眼球内陷，外观）+ 急性期同侧面部潮红（vasodilation）

**核心机制**：**同侧 oculosympathetic pathway（眼交感通路）** 受损 → 失去交感张力 → 提睑、扩瞳、出汗、血管收缩功能丧失。

---

## 二、解剖通路 — 三级神经元

```
                        【一级神经元】
            下丘脑后外侧 (Posterior Hypothalamus)
                          ↓
            经脑干被盖部 (brainstem tegmentum) 下行
                          ↓
            延髓外侧 (lateral medulla) — Wallenberg 位置
                          ↓
            脊髓侧索 (lateral funiculus)
                          ↓
            ──→ Ciliospinal center of Budge (T1–T2 lateral horn)
                ⭐ 一级末端、二级起点

                        【二级神经元 / 节前 Preganglionic】
            T1–T2 lateral horn (脊髓外侧角)
                          ↓
            经腹根 → white ramus → 交感链 (sympathetic chain)
                          ↓
            绕过 **lung apex（肺尖）** + 锁骨下动脉
                          ↓
            穿 stellate ganglion (星状神经节，无突触)
                          ↓
            ──→ Superior Cervical Ganglion (SCG, 颈上神经节)
                ⭐ 二级末端、三级起点

                        【三级神经元 / 节后 Postganglionic】
            SCG → 攀附 Internal Carotid Artery (ICA, 颈内动脉)
                          ↓
            穿 cavernous sinus (海绵窦) ── 与 CN VI 并行
                          ↓
            随 long ciliary nerves (V1 分支) 入眶
                          ↓
            ──→ 三个靶器官：
                ① Müller's muscle (上睑提肌辅助) → ptosis
                ② Iris dilator muscle (虹膜开大肌) → miosis
                ③ 面部汗腺、皮肤血管 → anhidrosis
                
            ⚠️ 关键：面部汗腺纤维**沿颈外动脉 (ECA) 分支走**
                → 三级（ICA 分支后）损伤可保留出汗 = "Partial Horner"
```

> [!warning] USMLE 三个解剖必背关键点
> 1. **一级二级交界 = T1–T2 lateral horn**（Syringomyelia 阶段 2 命中此处）
> 2. **二级绕肺尖 + 锁骨下动脉** = Pancoast 肿瘤、颈胸手术、第一肋骨折损伤口
> 3. **三级攀 ICA + 穿海绵窦** = 颈动脉夹层、海绵窦病变、Cluster headache 损伤口

## 三、3 级定位与病因对照表 ⭐⭐⭐

| 维度 | **一级（中枢/Central）** | **二级（节前/Preganglionic）** | **三级（节后/Postganglionic）** |
|---|---|---|---|
| **神经元起止** | 下丘脑 → T1–T2 lateral horn | T1–T2 → 锁骨下 → SCG | SCG → ICA → 眼眶 |
| **解剖经过** | 脑干 + 脊髓 | 肺尖 + 颈胸交界 | ICA + 海绵窦 |
| **Anhidrosis 范围** | 同侧半身（整个躯干 + 面） | 同侧面部 + 颈部 | **常缺失或仅额内侧**（汗腺沿 ECA 分流）|
| **典型病因** | **Syringomyelia** ⭐ / **Lateral medullary syndrome (Wallenberg)** ⭐ / 脑干/脊髓 stroke / MS / 脊髓肿瘤 / spinal cord injury | **Pancoast tumor** ⭐ / 颈胸创伤 / 第一肋骨折 / 锁骨下动脉瘤 / 颈部手术 / 甲状腺手术 | **Carotid artery dissection** ⭐⭐ / 海绵窦病变（thrombosis / tumor）/ **Cluster headache** / 颈动脉粥样硬化 / Raeder syndrome |
| **伴随征** | **脊髓征**（披肩痛温↓、四肢瘫）/ **脑干征**（dysphagia、ataxia、对侧痛温↓、同侧面部痛温↓、眩晕） | **C8–T1 神经根征**（手内在肌萎缩、Klumpke）/ **肺尖肿块 / 咯血 / 体重↓ / 慢性吸烟史** | **同侧颈痛 / 头痛 / 一过性卒中前兆** / **眼部痛（cluster）** |
| **USMLE Stem 钩子** | "Cape-like sensory loss" + Horner / "Vertigo + dysphagia + 对侧躯干痛温↓" | "Long-time smoker + apical lung mass + 手内在肌萎缩" / "shoulder pain + 第一肋骨折" | "Neck trauma / chiropractic manipulation / 颈痛后视力变化" / "男性 + 半夜痛醒踱步 + 单侧眶周痛" |
| **是否急症** | 视具体病因（Wallenberg = 急症 / Syringomyelia = 慢病）| 多为慢性 / 亚急性 | **常为急症**（夹层 → 卒中风险）|

## 四、各级标志性 stem 钩子（USMLE 速识）

### 一级（中枢）三大经典 stem

> [!example]- ① Syringomyelia（脊髓空洞）⭐ — Q20435 来源题
> **Stem 模式**：62 岁女性 + 数年前车祸 / Chiari I → 1 年进展性上肢感觉异常 + **手部无痛烫伤** + 上肢/上胸**披肩样温觉↓但触觉正常** → 问"病程下一步最可能见到？"
> 
> **锁定**：披肩痛温↓ + 触觉保留 + Chiari I/whiplash 史 → Syringomyelia
> 
> **Horner 出现时机**：**阶段 2**（空洞从前白连合扩到外侧角 T1–T2）→ 同侧 ptosis 是病程演变的下一步
> 
> **5 幕剧**（从内向外烂）：
> 1. 前白连合 → 双侧披肩痛温↓
> 2. **外侧角 → Horner** ⭐
> 3. 前角 → 上肢 LMN（biceps reflex ↓）
> 4. 外侧 CST → 下肢 UMN
> 5. Sacral sparing 最后破

> [!example]- ② Lateral Medullary / Wallenberg Syndrome（延髓背外侧综合征）
> **Stem 模式**：突发**眩晕 + 吞咽困难 + 声嘶 + 同侧面部痛温↓ + 对侧躯体痛温↓ + 同侧 Horner + 同侧小脑征**
> 
> **病因**：**PICA（小脑后下动脉）闭塞** 或 vertebral artery dissection
> 
> **锁定钩子**："5 个交叉征" — 同侧面部 + 对侧躯体痛温↓（spinothalamic 已交叉到对侧、trigeminal 同侧）
> 
> **关键**：CN IX / X 累及 → 吞咽困难 + 声嘶（Medial medullary 没有）

> [!example]- ③ MS / 脑干肿瘤 / 高位颈髓损伤
> 长束征 + 中枢病史 + 不典型 Horner — 一级的"杂项"。MS 患者出现 Horner 提示脑干斑块。

### 二级（节前）首选必背

> [!example]- ④ Pancoast Tumor（肺尖瘤）⭐⭐
> **Stem 模式**：60 岁慢性重度吸烟男 + **肩背痛**（最早）+ **手内在肌萎缩**（C8–T1，Klumpke）+ Horner + 慢性咳嗽 / 体重↓
> 
> **机制**：肺尖 NSCLC（多为 squamous）侵犯：
> - 下臂丛 (C8–T1) → 手内在肌萎缩、肢体痛
> - 二级交感链 → Horner
> - 第一肋 / 椎体 → 持续肩背痛
> 
> **影像**：CXR 肺尖肿块（易漏诊 → 一开始当肩周炎治）；CT 确认
> 
> **USMLE 钩子**："Shoulder pain + 手内在肌无力 + ptosis + 长期吸烟" = Pancoast 几乎不会错

> [!example]- ⑤ 颈胸创伤 / 第一肋骨折 / 锁骨下动脉瘤 / 甲状腺-颈淋巴清扫术后
> 解剖损伤 + 部位明确 + Horner — 直接看 stem 描述即可。

### 三级（节后）首选必背

> [!example]- ⑥ Carotid Artery Dissection（颈动脉夹层）⭐⭐ 急症
> **Stem 模式**：年轻-中年人 + **轻微颈部外伤史**（按摩 / chiropractic / 急转头）+ 突发**同侧颈痛或头痛** + Horner + **数小时-数天后偏侧无力 / 失语** (TIA / stroke)
> 
> **机制**：ICA 内膜撕裂 → 假腔形成 → 同侧 Horner（三级 ICA 攀附纤维受牵拉）+ 远端栓塞 / 闭塞 → 脑卒中
> 
> **诊断**：**CTA / MRA 颈部**（首选）/ Doppler 初筛
> 
> **治疗**：抗血小板 或 抗凝（防卒中）；血管内介入
> 
> **USMLE 钩子**：**"颈痛 + Horner = 颈动脉夹层，直到证明不是"** — 任何延误 = 卒中风险

> [!example]- ⑦ Cluster Headache（丛集性头痛）
> **Stem 模式**：男性 20–40 岁 + **半夜痛醒** + 单侧**眶周/眼后**钻刺痛 + **15–180 min** + **同侧 Horner + 流泪 + 流涕 + 鼻塞 + 结膜充血** + **踱步躁动**（不像 migraine 躲暗室）
> 
> **机制**：ICA 周围海绵窦区域血管炎症 → 临时性损伤三级交感纤维（发作期 Horner，间歇期可恢复）
> 
> **治疗**：
> - 急性：**100% O₂ 12–15 L/min** ⭐（非 Triptan）
> - 预防：**Verapamil**

> [!example]- ⑧ 海绵窦综合征 / Raeder paratrigeminal syndrome
> 海绵窦 thrombosis / 肿瘤 → Horner + 同侧 CN III/IV/V1/V2/VI 多支受累（complete ophthalmoplegia + V 区感觉↓）。

## 五、Horner ptosis vs CN III ptosis 鉴别 ⭐⭐

| 维度 | **Horner Ptosis** | **CN III (Oculomotor) Palsy Ptosis** |
|---|---|---|
| **损伤肌肉** | Müller 肌 (smooth muscle，交感支配) | **Levator palpebrae superioris** (striated，CN III 支配) |
| **下垂程度** | **轻度** (1–2 mm) | **完全下垂**（睑遮盖瞳孔，"complete ptosis"）|
| **瞳孔** | **Miosis（缩小）** | **Mydriasis（散大）**（CN III 的副交感纤维瘫）|
| **眼位** | 正常 | **"Down and out"**（外直肌 + 上斜肌仅存）|
| **对光反射** | 保留 | 散大瞳孔对光反射消失（如累及副交感）|
| **Anhidrosis** | 可有（一、二级）| 无 |
| **危急情况** | 颈动脉夹层（三级急症）/ Wallenberg（一级急症）| **PCom 动脉瘤压迫**（CN III + pupil involvement = 神经外科急症）⭐ |
| **典型病因谱** | 见三级分级表 | 糖尿病微血管病（pupil-sparing）/ PCom aneurysm（pupil-involving）/ uncal herniation |
| **机制口诀** | 交感失张力 → 上不起、缩、不出汗 | CN III 瘫 → 提上睑肌瘫 + 大部分眼外肌瘫 + 副交感瘫 |

> [!danger] 一句话生死鉴别
> **Ptosis + 散瞳 + "down and out" = CN III pupil-involving palsy → 立刻怀疑 PCom 动脉瘤，CTA 颈脑！**
> **Ptosis + 缩瞳 + 颈痛 = 颈动脉夹层 → CTA 颈！**
> 这两种都是急症，**永远不是糖尿病单纯神经病变**（DM neuropathy 是 pupil-sparing CN III palsy）。

### 药理验证（少数高难度题会考）

| 药物试验 | Horner（任意级别） | 正常 / 健侧 |
|---|---|---|
| **Cocaine 滴眼** 10%（一线诊断）| **不扩瞳**（NE 释放阻断已失效）| 扩瞳 |
| **Hydroxyamphetamine** 1%（定位三级）| **三级：不扩瞳**（节后神经元已死，无 NE 释放）；一、二级：扩瞳 | 扩瞳 |
| **Apraclonidine** 0.5%（替代 cocaine）| **扩瞳**（denervation supersensitivity）| 无变化 |

## 六、跨学科关联

### Neurology
- **Wallenberg syndrome** = PICA 闭塞 → 5 交叉征 + 一级 Horner
- **Syringomyelia** = Chiari I / 创伤后 → 5 幕剧第 2 幕 Horner
- **MS / 脑干肿瘤** = 偶发一级 Horner
- **Cluster headache** = 发作期三级 Horner（可恢复）

### Ophthalmology（眼科）
- **Ptosis 鉴别**：Horner（轻 + miosis）vs CN III palsy（重 + mydriasis "down and out"）vs MG（疲劳性、下午加重，无瞳孔异常）vs Levator dehiscence（老年退行性，无瞳孔/眼位异常）
- **Anisocoria（瞳孔不等大）workup**：先看**昏暗光线下哪个瞳孔小**（小的异常 = Horner）vs **明亮光线下哪个瞳孔大**（大的异常 = CN III / Adie / 药物性）

### Thoracic Surgery / Pulmonology（胸外 / 肺）
- **Pancoast tumor**（肺尖 NSCLC）= 二级 Horner 经典；任何"shoulder pain + 手内在肌萎缩 + 慢性吸烟" → CXR 看肺尖
- 第一肋骨折 / 锁骨下动脉瘤 / 甲状腺-颈淋巴清扫术后医源性损伤

### Cardiology / Vascular（心内 / 血管）
- **Carotid artery dissection** = 三级 Horner 急症 + 卒中前兆
- 颈动脉粥样硬化（罕见 — 慢性 Horner）
- 主动脉弓动脉瘤（罕见压迫交感链）

### Pediatrics（儿科）— 不在题目要求里但高频考点
- **儿童单侧 Horner + 颈/胸/腹包块** → **Neuroblastoma 直到证明不是** ⭐
  - 神经母细胞瘤来自交感链 → 压迫二级神经元
  - 24h 尿 VMA / HVA + CT / MIBG

## 七、Memory Hook

> [!success] **Horner 三级一图流**
> 
> ```
> 一级 = 中枢 = 脊髓/脑干征
>     ▲ 累及大范围（半身 anhidrosis）
>     ▲ 病：Syringomyelia / Wallenberg / MS
> 
> 二级 = 节前 = 肺尖征 + 手内在肌征
>     ▲ 同侧面颈 anhidrosis
>     ▲ 病：Pancoast / 颈胸创伤 / 第一肋骨折
> 
> 三级 = 节后 = 颈痛 / 头痛 / Cluster
>     ▲ 出汗常保留（Partial Horner）
>     ▲ 病：Carotid dissection ⭐ 急症 / Cluster / 海绵窦
> ```

> [!tip] 看伴随定级口诀
> - 伴**脊髓/脑干征** → **一级**（Syringomyelia, Wallenberg）
> - 伴**手内在肌萎缩 / 肺尖病变** → **二级**（Pancoast）
> - 伴**颈痛/头痛/卒中前兆** → **三级**（颈动脉夹层 / Cluster）

> [!tip] 三句话保命
> 1. **"颈痛 + Horner = 颈动脉夹层 until proven otherwise"** — CTA 颈
> 2. **"长期吸烟 + 肩痛 + 手内在肌萎缩 + Horner = Pancoast"** — CXR / CT 肺尖
> 3. **"披肩痛温↓ + Chiari/whiplash + Horner = Syringomyelia 阶段 2"** — MRI 颈髓

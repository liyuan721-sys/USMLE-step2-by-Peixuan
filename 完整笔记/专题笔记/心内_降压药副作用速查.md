# 降压药副作用速查（Antihypertensive Adverse Effects）

> [!info] 本笔记定位
> USMLE Step 2 CK 降压药题**不爱考经典副作用**（BB 心慢、ACEI 干咳），而绕道考 **metabolic side effects** 和**特殊人群禁忌**。
>
> 本笔记按"5+2 类药 × 副作用 × T2DM/HF/Pregnancy/CKD 优先级"建立矩阵。
>
> **触发错题**：[[mistakes/uworld-mistakes#^Q15973]]（weight gain + worsening glucose tolerance → Conventional BB）

## 一、5+2 类降压药副作用矩阵

| 药物类 | 代表药 | 体重 | 糖代谢 | 标志性副作用 | T2DM 患者 |
|---|---|---|---|---|---|
| **ACEI** | lisinopril, enalapril | 中性 | 改善/中性 | 干咳（bradykinin）、高钾、血管性水肿、AKI | ✅ **首选** |
| **ARB** | losartan, valsartan | 中性 | 改善/中性 | 高钾、AKI（**无干咳**）| ✅ **首选**（ACEI 干咳替代）|
| **α1 blocker** | prazosin, doxazosin | 中性 | **改善** | 体位性低血压、**first-dose syncope** | ✅ 可用，不是一线 |
| **Conventional BB** ⭐ | **metoprolol, atenolol, propranolol** | **↑** | **恶化** ⚠️ | bradycardia、bronchospasm、**metabolic syndrome** | ❌ **避免** |
| **Vasodilatory BB** | **carvedilol, labetalol** | 中性 | 中性 | 体位性低血压；无 metabolic 恶化 | ✅ **HF + DM 首选 BB** |
| **DHP CCB** | amlodipine, nifedipine | 液体潴留 | 中性 | **外周水肿**、反射性 HR↑、牙龈增生、面部潮红 | ✅ 可用 |
| **Non-DHP CCB** | verapamil, diltiazem | 中性 | 中性 | bradycardia、便秘、HF 恶化 | ⚠️ HF 禁用 |
| **Thiazide** | HCTZ, chlorthalidone | 中性 | **恶化**（剂量相关）⚠️ | 低钠、低钾、高钙、**高血糖、高尿酸、高脂** | ⚠️ 小剂量可用 |
| **Loop diuretic** | furosemide | ↓（容量）| 轻恶化 | 低钾、低镁、低钙、ototoxicity（大剂量）| ⚠️ HF 优先 |
| **K-sparing** | spironolactone, eplerenone | 中性 | 中性 | **高钾**、男性乳房（spiro）、抗雄激素效应 | ⚠️ 高钾监测 |

> [!warning] 两个"糖代谢恶化"的降压药
> **Conventional BB + Thiazide（高剂量）**。
> T2DM + HTN 患者题目里见到这两个 → **排除**；优先 ACEI / ARB（兼顾肾保护）。

## 二、副作用机制深度（BB + ACEi/ARB）

### 2.1 Conventional BB 恶化糖代谢（β2 封 + α1 单飞）

```
Conventional BB（metoprolol / atenolol / propranolol）
        ↓
    阻断 β1 + β2（非选择性，或 β1 选择性高剂量也漏 β2）
        ↓
    α1 失去 β2 对抗（β2 本来扩血管平衡 α1 缩血管）
        ↓
    骨骼肌血管收缩（不平衡的 α1）
        ↓
    骨骼肌血流 ↓
        ↓
    葡萄糖摄取 ↓ + 胰岛素抵抗 ↑
        ↓
    胰岛素抵抗 + 体重↑（脂肪倾向储存）+ 糖耐量↓
```

**为什么 Carvedilol / Labetalol 没事**：
- 这两药同时阻断 **α1** → 即使 β2 被封，α1 也被压住 → 骨骼肌血管不缩 → 葡萄糖摄取不受影响。

**为什么 α1 blocker 反而改善糖代谢**：
- 单独阻 α1 → 骨骼肌血管扩张 → 血流↑ → 葡萄糖摄取↑ → 胰岛素敏感性↑。
- 这就是为什么 prazosin / doxazosin 对 metabolic syndrome 患者**有利**（虽不是 HTN 一线，但兼顾 BPH 老年人很合适）。

### 2.2 ACEi/ARB 副作用机制（AngII 双通路 — 高钾 + AKI + Type 4 RTA 三联）

> [!danger] ACEi/ARB 的肾脏副作用 = 两条通路同时作用
>
> 不是单纯"抑制醛固酮"那么简单 — AngII 在肾脏有**两个独立作用点**，ACEi/ARB 同时把它们都关了。

#### 双通路完整链

```
ACEi/ARB 阻断 AngII
        │
        ├─────────────────────┬─────────────────────┐
        ↓                     ↓                     ↓
   血管效应                Aldosterone ↓        肾血流效应
   ↓ 全身 BP                  │                出球小动脉扩张
                              ↓                     ↓
                         远端肾单位            GFR ↓
                         ↓ K 排泄              Cr ↑
                         ↓ H 排泄
                              ↓
                         ⚠️ 高钾血症 ←──── 协同恶化 ────→ ⚠️ 急性肾损伤
                         ⚠️ Type 4 RTA      (GFR ↓ 也减少 K 滤过和排泄)
```

#### 为什么 ACEi/ARB 的高钾比单纯抗醛固酮（spironolactone）更严重？

1. **Aldosterone ↓** → 远端排 K ↓（直接机制）
2. **GFR ↓**（出球扩张）→ 滤过的 K 也 ↓（间接机制）
3. **两条通路叠加** → 高钾风险放大

→ 这就是为什么 ACEi + spironolactone + KCl **三联 K 炸弹**特别危险（见 [[mistakes/uworld-mistakes#^Q19-2026-05-18]]）：ACEi 自己已经走两条路降 K 排泄，再叠加 MRA + 外源 K，必爆。

#### High-Yield Summary Table（速查）

| 项目 | 机制 | 结果 |
|---|---|---|
| **降压主效应** | ↓ AngII → 全身血管扩张 | BP ↓ |
| **出球小动脉扩张** | ↓ AngII 对 efferent 的收缩 | **GFR ↓, Cr ↑ 20-30%（预期）**, 蛋白尿 ↓ |
| **↓ Aldosterone** | ↓ AngII → 肾上腺皮质球状带 ↓ 合成 | ↓ Na 重吸收, **↓ K 排泄, ↓ H 排泄** |
| **副作用三联**（上述综合）| AngII 双通路阻断 | **高钾 + 急性肾损伤 + Type 4 RTA** |
| **干咳 / 血管性水肿** | ACEi 特有：**↑ Bradykinin**（ARB 无此副作用）| 干咳 / angioedema |

#### Memory Hook — "AngII 是肾脏的守门员"

> [!tip] 类比锁定
> - **正常**：守门员（AngII）站岗在**出口**（efferent / 出球小动脉），关紧门 → 球内压力高 → GFR 维持
> - **阻断 AngII**：守门员下班 → 出口大开 → 球内压力泄掉 → **GFR ↓ → Cr ↑**
> - **同时**：守门员还兼职"指挥肾上腺分泌 Aldosterone" → 下班后 K 和 H 没人排 → **高钾 + 酸中毒（Type 4 RTA）**
>
> **一句话**：**ACEi/ARB = "拆 AngII 的台" = 出球泄压 + Aldo 罢工 = Cr↑ + K↑ + H↑**

#### 临床场景速记

| 场景 | 机制解释 |
|---|---|
| 糖尿病人用 ACEi 后 Cr 从 1.0 → 1.2 | **预期反应**（出球扩张 → GFR ↓ 20%），**继续用**（蛋白尿降低长期保肾）|
| **双侧肾动脉狭窄**患者用 ACEi → 急性肾衰 | 这类患者本就靠 AngII 收紧出球维持 GFR，一阻断 → GFR 崩溃（**ACEi/ARB 禁忌**） |
| ACEi + Spironolactone + 老年人 → 高钾住院 | **双重抗 Aldosterone** + GFR ↓，K 排泄通路被全面封锁 |
| ACEi 用了 1 周开始干咳 | **Bradykinin 蓄积**（ARB 无此问题，可替换 losartan）|

## 三、特殊人群首选 vs 避免

| 人群 | 首选 | 避免 / 慎用 |
|---|---|---|
| **T2DM + HTN** | ACEI / ARB（兼顾肾保护）| Conventional BB / Thiazide 高剂量 |
| **CKD（蛋白尿）** | **ACEI / ARB** ⭐ | K-sparing（高钾叠加）|
| **HFrEF** | ACEI/ARB + **Carvedilol/Metoprolol succinate/Bisoprolol** + MRA | Non-DHP CCB（负性肌力）；DHP CCB 中性 |
| **HF + DM** | ACEI/ARB + **Carvedilol** ⭐ | Conventional BB（metoprolol tartrate 形式也有 metabolic 风险）|
| **Pregnancy** | **Labetalol / Nifedipine / Methyldopa / Hydralazine** ⭐ | **ACEI/ARB 禁用**（致畸 + 肾畸形）；Thiazide 慎用 |
| **黑人（无 DM/CKD）** | Thiazide / **DHP CCB** 单药效果较好 | 单药 ACEI/ARB 效果略弱（联合可用）|
| **老年 + BPH** | **α1 blocker** + 其他 | 一线仍是 ACEI / 钙拮抗 / 利尿 |
| **冠心病稳定** | **β-blocker + ACEI/ARB** | — |
| **痛风** | ARB（**losartan 促尿酸排泄** ⭐）| **Thiazide / Loop**（升尿酸）|

## 四、易混陷阱（USMLE 高频）

> [!warning] 陷阱 1：BB 与 α1 blocker 糖代谢方向反向
> - **Conventional BB → 恶化**（β2 封 → α1 单飞 → 肌肉血流↓ → 糖摄取↓）
> - **α1 blocker → 改善**（α1 封 → 肌肉血流↑ → 糖摄取↑）
> 别搞反！

> [!warning] 陷阱 2：所有 BB 不等价 — 看 α1 阻断
> - **Conventional BB**（metoprolol / atenolol / propranolol）→ 没有 α1 阻断 → metabolic 恶化
> - **Vasodilatory BB**（**carvedilol / labetalol**）→ 兼有 α1 阻断 → metabolic 中性
> - HF + DM 患者 → 选 **carvedilol**（很多题考 Q stem 给"DM + HFrEF"问首选 BB，答案永远是 carvedilol，不是 metoprolol）

> [!warning] 陷阱 3：DHP CCB "体重增加" ≠ 真胖
> 是**外周水肿**（pretibial edema），液体潴留，不是脂肪增加。
> 鉴别：amlodipine 病人 + 双下肢凹陷性水肿 + BNP 正常 + 心功能保留 → DHP CCB 副作用，不是 HF。
> 处理：加 ACEI/ARB（不是利尿剂 — 静脉端不开扩）。

> [!warning] 陷阱 4：ARB 改善糖代谢（不是恶化）
> ARB / ACEI 对糖代谢**有益**（增加胰岛素敏感性、降低新发 DM 风险）。
> Stem 给"DM + HTN + 新发蛋白尿" → 一线 ACEI / ARB，**不是**避免。

> [!warning] 陷阱 5：Thiazide 的 metabolic 副作用是剂量相关
> 小剂量 HCTZ 12.5 mg 通常 metabolic 影响小；高剂量 50 mg 才显著恶化糖 / 脂 / 尿酸。
> Stem 给"HTN + 痛风" → 仍**避开** thiazide（尿酸更敏感）。

> [!warning] 陷阱 6：Losartan 是唯一**促尿酸排泄**的 ARB
> 痛风 + HTN 患者 stem → 答 **losartan**（其他 ARB 是中性，losartan 兼顾降尿酸）。

> [!warning] 陷阱 7：DHP-CCB 外周水肿 ≠ 心衰水肿
> Amlodipine / nifedipine 致的双下肢凹陷性水肿：
> - **机制**：**毛细血管前小动脉扩张**（小动脉先扩 → 毛细血管压↑）→ 液体外渗到组织（局部，不是容量超负荷）
> - **关键鉴别**：BNP **正常** + 心功能保留 + 颈静脉**不**怒张
> - **处理**：**利尿剂无效**（不是容量问题）→ **换药**（DHP 改 non-DHP / ACEi）**或加 ACEi/ARB**（同时扩静脉端，平衡毛细血管压）
> - 鉴别 stem：amlodipine 患者新发双下肢水肿 + BNP 正常 → DHP 水肿，**不**给利尿剂

## 五、Memory Hooks

> [!success] 核心口诀
>
> 1. **"BB 让肌肉饿肚子"** — β2 封死，骨骼肌血管关门，葡萄糖进不去 → 血糖高 + 人变胖
> 2. **"Carvedilol / Labetalol 是开窗的 BB"** — α1 也封了，肌肉照常吃饭
> 3. **"传统 BB 三宗罪"**：**心慢 + 喘 + 糖胖**（bradycardia / bronchospasm / metabolic syndrome）
> 4. **"两胖兄弟"**：Conventional BB + Thiazide 高剂量 = 糖代谢恶化双煞
> 5. **"α1 blocker 反向救糖"** — 别和 BB 搞反；α1 封 → 肌肉血流↑ → 糖摄取↑
> 6. **"DHP 水不是胖"** — amlodipine 脚肿是水肿不是脂肪
> 7. **"Losartan 痛风之友"** — 唯一降尿酸的 ARB
> 8. **"妊娠 4 兄弟"**：**L**abetalol / **N**ifedipine / **M**ethyldopa / **H**ydralazine（ACEI/ARB 禁用）
> 9. **"β-blocker 是代谢小偷"** — 偷走**肌肉血流**（β2 阻断 → α1 无对抗 → 血管收缩 → 葡萄糖进不去）+ 偷走**脂肪分解**（β1 阻断 → 脂肪锁住 → 体重↑）；但 **carvedilol / labetalol 不偷**（自带 α1 钥匙 → 血管开门）

> [!success] **血糖代谢三态**速记（Q15973 触发）
>
> | 三态 | 药物清单 | 一句话锁定 |
> |---|---|---|
> | ✅ **改善** | ACEi / ARB / **α1-blocker** (prazosin) | "扩张血管 → 肌肉吃饭" |
> | ⚪ **中性** | DHP-CCB / Non-DHP CCB / **Carvedilol / Labetalol / Nebivolol** | "不动糖代谢" |
> | ❌ **恶化** | **Conventional β-blocker** / **Thiazide**（剂量相关）| "肌肉饿肚子 / 利尿副作用" |
>
> **看到 stem "HTN + 体重↑ + 血糖↑"** → 闭眼选 **Conventional BB**（除非选项里有 thiazide 高剂量）。
> **看到 stem "HTN + 新诊 T2DM 选药"** → 闭眼选 **ACEi / ARB**（兼顾肾保护 + 改善胰岛素敏感性）。

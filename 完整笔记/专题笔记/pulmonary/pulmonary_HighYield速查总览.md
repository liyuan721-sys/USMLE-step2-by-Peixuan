## 一、PFT 判读（Pulmonary function test 肺功能）

> [!tip] 一眼分型（先看 FEV1/FVC）
> **FEV1/FVC ↓ (<0.7) = 阻塞性 (obstructive)**：慢阻肺 COPD / 哮喘 asthma / 支气管扩张 bronchiectasis。
> **FEV1/FVC 正常或↑ (≥0.7) = 限制性 (restrictive)**：肺纤维化 / 间质性肺病 ILD / 肥胖。
> **FEV1 越低病情越重**（阻塞限制通用）；**FEV1 正常 ≥80%**，决定 GOLD 分级。

```
第一步 FEV1/FVC ?
 ├─ <0.7  阻塞性 → 看 DLCO 区分
 │        ├─ DLCO↓ = 肺气肿 emphysema（肺泡破坏，膜没了）
 │        └─ DLCO 正常 = 慢性支气管炎 / 哮喘（大气道问题）
 └─ ≥0.7  限制性 → 看 DLCO 区分
          ├─ DLCO↓ = 肺实质病（IPF/ILD/尘肺/结节病）
          └─ DLCO 正常 = 胸壁/神经肌肉（肥胖/脊柱侧弯/神经肌无力）
```

| 概念 | 钥匙 |
|---|---|
| DLCO↓ | 「膜坏了 / 面积没了」= 肺泡壁或毛细血管面积出问题 |
| A–a gradient↑ | diffusion defect / V/Q mismatch / shunt（肺泡有氧血里没氧）|
| 哮喘 DLCO | 正常→**高**（气体陷闭 gas trapping）|
| 肺气肿 DLCO | **↓**（肺泡表面积丢失）|

> [!warning] 限制性早期气体模式（易混）
> 限制性肺病早期：**O2 低 + CO2 低 + pH 高（呼吸性碱中毒 respiratory alkalosis early）**；本质是 diffusion problem，进展可变 perfusion problem。
> 阻塞性：**O2 正常→低 + CO2 高 + pH 低（呼吸性酸中毒 respiratory acidosis）**，本质 ventilation problem。

## 二、阻塞性肺病（Obstructive）

### 2.1 哮喘 asthma 升阶治疗（stepwise）

**诊断**：支气管舒张试验 FEV1↑ 15–20%；或**乙酰甲胆碱激发 (methacholine challenge)** FEV1↓ >15–20%。嗜酸性粒细胞 eosinophilia。

```
① 低剂量 ICS-formoterol（吸入糖皮质激素 inhaled corticosteroid + 福莫特罗）
② 中剂量 ICS-formoterol
③ 加 LABA（沙美特罗 salmeterol）或 孟鲁司特 montelukast
④ 加 口服激素 oral steroids
```

> [!tip] 特殊型哮喘 → 特定药
> - **运动诱发 (exercise-induced)**：活动前用沙丁胺醇 albuterol。
> - **阿司匹林诱发 / AERD（阿司匹林加重性呼吸道疾病 aspirin-exacerbated respiratory disease，即 Samter 三联）**：哮喘 + 鼻息肉 nasal polyps + 阿司匹林/非甾体抗炎药 NSAID 不耐受 → **孟鲁司特 montelukast**（白三烯转移）。
> - **过敏性 (extrinsic)**：色甘酸钠 cromolyn（防肥大细胞脱颗粒）+ albuterol。

> [!danger] 哮喘急性发作 (asthma exacerbation)
> **听诊优先**：**无气流/无呼吸音 → 立即插管 intubate**（不是好转，是没气进出）。
> 有喘鸣 wheezing → **氧 + 动脉血气 ABG** → 阶梯：沙丁胺醇 albuterol → 异丙托溴铵 ipratropium → 静脉激素（甲泼尼龙 IV methylprednisolone）。出院给口服激素 3 周（长期用需减量 taper）。

### 2.2 COPD 分级与急性加重

> [!info] 慢支 vs 肺气肿（chronic bronchitis vs emphysema）
> **慢性支气管炎 (chronic bronchitis)** = blue bloater 蓝胖：定义为**连续≥2–3 年、每年咳痰≥2–3 月**；A–a 正常、DLCO 正常；确诊 = 活检 **Reid 指数↑**（正常 <40%，慢支 >50%）。
> **肺气肿 (emphysema)** = pink puffer 粉红瘦喘：肺泡破坏 → DLCO↓、A–a↑、RV↑↑、TLC↑。
> **年轻人 + 肺气肿 → 想 α1-抗胰蛋白酶缺乏 (A1AT deficiency)**。

> [!warning] A1AT 缺乏（α1-antitrypsin deficiency）该想到的线索
> 常染色体隐性 AR / **发病年龄 ≤45 岁** / 吸烟史轻或无 / **下叶为主 (basilar-predominant)** / 不明原因肝病史。下一步：**查血清 α1-抗胰蛋白酶水平**。（对比吸烟性中心型肺气肿累及**上叶**。）

**COPD 真正延长寿命的只有 3 项（mortality reduction）**：

```
① 戒烟 smoking cessation（第一位，减缓 FEV1 下降）
② 氧疗 oxygen therapy（仅符合指征的低氧血症）
③ 肺康复 pulmonary rehab
※ 吸入药（支扩剂/激素）只缓解症状、减少急性加重，不延长寿命
```

> [!danger] 家庭氧疗指征（home oxygen，记阈值）
> **PaO2 ≤55 或 SaO2 ≤88%**；或合并**肺心病 cor pulmonale（右心衰，红细胞压积 Hct >55%）时 PaO2 ≤59 或 SaO2 ≤89%**。
> ⚠️ 避坑：COPD 病人见 JVD + 肝大别条件反射选利尿剂 → **先看 SaO2，血氧低则氧气是最好的"强心药"**；稳定期先吸氧，利尿剂治不了肺血管收缩。

**急性加重 AECOPD 处理**：ABG + 低流量氧 → 吸入支扩剂 SABA（沙丁胺醇 albuterol）± SAMA（异丙托溴铵 ipratropium）→ 静脉激素 → 抗生素（头孢曲松 ceftriaxone 或呼吸喹诺酮 respiratory fluoroquinolone）。诱因 70% 感染（鼻病毒 rhinovirus / 不可分型流感嗜血杆菌）。

### 2.3 支气管扩张 bronchiectasis

> [!tip] 双钥匙
> **臭痰 (foul-smelling sputum) = bronchiectasis until proven otherwise**（厌氧菌）；**tram tracks = 扩张的支气管**。
> 影像：初查 CXR 见 tram tracking → 最准 CT 见**蜂窝样 honeycombing**。年轻人 + 童年重症肺感染史，也见于囊性纤维化 CF / 免疫缺陷 / 原发纤毛不动。
> 治疗：局限 → 手术切除 + 胸部理疗；弥漫 → 长期抗生素。

## 三、限制性与间质病（ILD / 肉芽肿 / 尘肺 / 药物毒物）

| 病 | 钥匙 (buzzword) | 处理 |
|---|---|---|
| 结节病 sarcoidosis | 肺门淋巴结肿大 + 结节性红斑 erythema nodosum；**ACE↑、钙↑**；活检**非干酪性肉芽肿** | 类固醇 steroids |
| 变应性支气管肺曲霉病 ABPA | 哮喘/CF 基础 + 黏液栓 + **血清 IgE 极高 (>1000 可疑)** + 游走性浸润 + 中心性支气管扩张 | 口服激素 + 伊曲康唑 itraconazole；随访看总 IgE |
| 矽肺 silicosis | **上叶钙化** → 继发 TB 风险；喷砂/铸造/采矿 | 支持 |
| 石棉肺 asbestosis | 胸膜斑 pleural plaques、间皮瘤 mesothelioma（特异）、支气管肺癌（更常见）、铁锈小体 ferruginous bodies；造船/屋顶/管道 | 支持 |
| 铍中毒 berylliosis | 非干酪性肉芽肿；核武/航天 | 类固醇 |
| 碳末沉着 anthracosis | 不致癌 → 肺纤维化 | — |
| 棉尘肺 byssinosis | 棉尘暴露 → 胸闷气短 | 脱离暴露 |
| 高铁血红蛋白血症 methemoglobinemia | 硝酸盐 nitrates；**正常 PaO2 + 巧克力色血 chocolate-colored blood** | **亚甲蓝 methylene blue** |

> [!question]- 结节病的标志性实验室双联是什么？
> **ACE↑ + 血钙↑**（钙升因肉芽肿巨噬细胞活化维生素 D）。活检见**非干酪性肉芽肿 noncaseating granulomas**；初查 CXR（双侧肺门淋巴结肿大）。一线类固醇。

## 四、肺血管（PE / 肺动脉高压）

### 4.1 肺栓塞 PE 诊断流程（pulmonary embolism）

**线索**：突发 SOB + 胸膜性胸痛（口服避孕药 OCP 风险）、小量胸腔积液、A–a↑、心动过速、低热、JVD；**大面积 PE → 右心衰 + 梗阻性休克 (obstructive shock)**。

```
血流动力学 稳定？
 ├─ 不稳定 → 急诊床旁超声心动图 TTE（transthoracic echo）看 RV（新发扩张/运动减弱 = massive PE）
 │            → tPA（组织型纤溶酶原激活剂 tissue plasminogen activator，溶栓）；tPA + 出血 → 取栓 embolectomy
 └─ 稳定 → Wells 评分
          ├─ Wells ≤4（PE unlikely）→ D-二聚体 D-dimer
          │     ├─ 阴性 → 排除
          │     └─ 阳性 → CT 肺动脉造影 CTA
          └─ Wells >4（PE likely）→ 直接 CTA
                  （造影禁忌 → V/Q 扫描，缺损区 high V/Q）
```

> [!tip] Wells 记忆「3 像不像 / 1.5 风险 / 1 伴随」
> - **3 分 = 像不像**：DVT 体征、其他诊断不像（"腿上有栓 + 别的不像"）。
> - **1.5 分 = 风险**：既往血栓、心动过速、术后卧床（"以前有、现在快、最近躺/刀"）。
> - **1 分 = 伴随**：咯血、癌症。

> [!danger] 治疗：稳的先抗凝，不稳才再通（先救循环）
> - **稳定**：肝素 heparin → 华法林 warfarin。**严重肾功能不全 → 普通肝素（UFH, unfractionated heparin）**（"小分子走肾、大分子走血"）。
> - **稳定 + 出血**：下腔静脉滤器（IVC filter, inferior vena cava 下腔静脉）。
> - **不稳定（RV 功能障碍 / SBP <90）**：tPA；**不稳定 + 出血**：取栓 embolectomy。
> 骨折后 → **脂肪栓塞综合征 (fat embolism)**：呼吸窘迫 + 神经功能障碍 + 瘀点皮疹 petechial rash。完整体系见 [[完整笔记/专题笔记/pulmonary/pulmonary_肺栓塞PE]] §四。

### 4.2 肺动脉高压 pulmonary hypertension (PH)

钥匙：劳力性呼吸困难 + **响亮 P2 (loud P2)** + 右室抬举 RV heave + 三尖瓣反流杂音 → 右心衰；CXR 肺动脉增宽、肺野清；ECG 右心劳损。PAH 时肺实质正常、FEV1 与 FEV1/FVC 正常（区别于 ILD 的限制型 PFT + DLCO↓）。

| 类别 | 药物 | 机制 |
|---|---|---|
| 内皮素拮抗剂 endothelin antagonist | 波生坦 bosentan | ↓血管收缩 |
| PDE-5 抑制剂 | 西地那非 sildenafil | ↑cGMP |
| 前列环素类似物 prostacyclin analog | 依前列醇 epoprostenol | ↑cAMP → 血管扩张 |

## 五、胸膜（胸腔积液 / 气胸 / 脓胸）

### 5.1 胸腔积液 Light 标准（pleural effusion）

**诊断流程**：初查 = **侧卧位 X 线 (decubitus x-ray)**（看有没有）→ 确诊 = **诊断性胸腔穿刺 (diagnostic thoracentesis)**（看是什么）。

> [!tip] Light 标准（渗出液 exudate 满足≥1 条即是）
> - 胸水蛋白/血清蛋白 **>0.5**
> - 胸水 LDH/血清 LDH **>0.6**
> - 胸水 LDH **> 正常血清 LDH 上限的 2/3**
> 三条全低 = 漏出液 transudate（心衰 CHF / 肝硬化 / 肾病综合征）；≥1 条高 = 渗出液（肺炎/恶性/TB/PE/胰腺炎/RA）。

**pH 阈值**（记数字）：正常胸液 ~7.60；漏出液 7.4–7.55；渗出液 7.30–7.45；**<7.30 = complicated effusion / 脓胸 empyema → 胸管引流 (chest tube)**。

| 胸水线索 | 指向 |
|---|---|
| 葡萄糖低（胸水/血清 <0.5）| TB / 恶性 / complicated parapneumonic / 类风湿 RA / 食管破裂 |
| 淀粉酶高 | 胰腺炎 / 食管破裂 |
| 单侧、浑浊 | 局部病变（渗出）|
| 双侧、清亮 | 全身性（漏出）|

### 5.2 肺炎旁积液阶梯

```
肺炎 → 胸腔渗出（parapneumonic effusion）
 ├─ 无菌 = uncomplicated → 抗生素，观察
 └─ 细菌入胸腔 = complicated / 脓胸 empyema → 抗生素 + 引流（胸管）
   （pH <7.30 = 复杂，必须引流）
```

## 六、肺炎（CAP / HAP / 非典型 / 吸入 / PCP）

### 6.1 CURB-65（是否住院）

```
C  Confusion 意识模糊
U  Urea/BUN 升高
R  RR ≥30
B  BP low 低血压
65 age ≥65 岁
```

> 细则（各项阈值 + PSI + ICU major/minor + 门诊/住院/ICU 三档选药决策树）见专题：[[完整笔记/专题笔记/pulmonary/pulmonary_CAP门诊与住院分层]]

### 6.2 CAP 经验治疗 + 特殊人群

> [!tip] CAP 经验用药（社区获得性肺炎 community-acquired pneumonia）——先按严重度分层再选 ⭐
> - **门诊·健康无共病**：**口服单药** = 阿莫西林 amoxicillin / 多西环素 doxycycline / 大环内酯（阿奇霉素 azithromycin）。⚠️ 生命体征稳定的门诊 CAP **别上 IV 广谱**（[[NBME11_错题本#^Q076]]：被镰状性状/哮喘标签带跑误选 IV 氨苄西林-舒巴坦）。
> - **门诊·有共病**（心/肺/肝/肾病、糖尿病、酗酒、免疫抑制）：呼吸氟喹诺酮单药，或 β-内酰胺 + 大环内酯。
> - **住院（含 ICU）**：β-内酰胺 + 大环内酯 = **头孢曲松 ceftriaxone + 阿奇霉素 azithromycin**；或 β-内酰胺 + 氟喹诺酮（左氧氟沙星 levofloxacin / 莫西沙星 moxifloxacin，老年避免 FQ：C. diff / 肌腱断裂 / 主动脉夹层）。
> - **青霉素过敏 penicillin allergy**：轻（皮疹）可用头孢；重（过敏反应 anaphylaxis）→ 多西环素 doxycycline 或呼吸 FQ。
> - **HAP（入院≥48h）**：覆盖铜绿假单胞菌 Pseudomonas + MRSA（高危）→ 万古霉素 vancomycin + 哌拉西林-他唑巴坦（Zosyn）或头孢吡肟 cefepime。
> - **CF + 肺炎**：必覆盖 MRSA → 静脉万古霉素（或利奈唑胺 linezolid）。

| 人群/年龄 | 病原 | 钥匙 |
|---|---|---|
| CF <20 岁 | 金黄色葡萄球菌 S. aureus | — |
| CF >20 岁 | 铜绿假单胞菌 Pseudomonas | — |
| 酗酒/糖尿病 | 克雷伯菌 Klebsiella | 醋栗果冻痰 |
| 0–2 月婴儿 | 衣原体 Chlamydia | 不发热 + staccato 咳 + 间质浸润 → 大环内酯 |
| 10–30 岁 | 支原体 Mycoplasma | 冷凝集 IgM + 肢端发绀（耳鼻指）→ 大环内酯 |
| >40 岁 | 军团菌 Legionella | 水源暴露 + GI 症状 + 低钠 + 意识模糊 → 氟喹诺酮 |

> [!warning] 吸入性肺炎 (aspiration pneumonia)
> 隐匿起病（数天–周）：醉酒/癫痫/吞咽障碍、**重力依赖部位浸润**、口臭/脓肿、臭痰（口腔厌氧菌）。

> [!info] PCP（耶氏肺孢子菌肺炎 Pneumocystis jirovecii pneumonia）
> HIV/免疫抑制 + 干咳 + 呼吸困难 + 低氧。一线 **TMP-SMX（复方磺胺甲噁唑）**；替代：阿托伐醌 atovaquone / 克林霉素 clindamycin + 伯氨喹 primaquine / 静脉喷他脒 pentamidine。

## 七、结核 TB（tuberculosis）

```
+PPD（结核菌素试验）→ 下一步 chest x-ray（不是直接治疗）
原发 primary：下/中叶 ；继发 secondary（再激活）：上叶
治疗 = RIPE
```

> [!tip] RIPE 四药副作用（高频）
> - **R** 利福平 rifampin：分泌物变红、P450 诱导剂
> - **I** 异烟肼 isoniazid：神经/肝损伤（周围神经病 + 肝毒）→ 给**维生素 B6**
> - **P** 吡嗪酰胺 pyrazinamide：关节痛 / 高尿酸 hyperuricemia
> - **E** 乙胺丁醇 ethambutol：视神经病 optic neuropathy

## 八、肺癌与孤立肺结节（lung cancer / SPN）

### 8.1 孤立肺结节 SPN 管理（solitary pulmonary nodule）

**定义**：圆形阴影、**≤3 cm**（>3 cm 是"肿块 mass"）、周围肺实质包绕、无淋巴结肿大。**结节大小最强相关恶性概率**。

```
SPN 大小分流（记阈值）
 ├─ <0.6 cm → 不太可能恶性，一般无需随访
 └─ >0.8 cm → 需进一步处理：活检/手术切除 或 监测（复查 CT）
 ※ >2 cm 独立相关 >50% 恶性概率
```

> [!warning] 提高恶性概率的因素
> 大小大（>2 cm）/ 年龄大 / 女性 / 吸烟（现/既往）/ 肺癌史 / **上叶位置** / **毛刺征 spiculated**。
> 诊断：**PET 扫描**高代谢 → 更可能恶性 → 活检或手术切除。
> 影像顺序：**CT 先于支气管镜 (CT precedes bronchoscopy)**；**中央型肿块用支气管镜 bronchoscopy**。

### 8.2 肺癌分型副肿瘤（paraneoplastic）

| 分型 | 副肿瘤 |
|---|---|
| 小细胞癌 small cell carcinoma | ADH (SIADH) / ACTH (Cushing) / TSH |
| 鳞癌 squamous cell carcinoma | PTHrP → 高钙血症 |

> [!tip] 肺叶切除术前预后指标（lobectomy）
> **FEV1 >1.5 L** 且 **DLCO >60%**。

## 十、ARDS / 呼衰 / 机械通气（ventilator）

> [!info] ARDS（急性呼吸窘迫综合征 acute respiratory distress syndrome）
> 机制：炎症 → 毛细血管渗漏 → 表面活性物质丢失。表现：O2 低、CO2 正常、CXR 白肺 white-out。
> **有明确肺损伤病因（如脓毒症）时临床诊断**（时间 1 周内 + 双侧肺泡浸润 + 排除心衰/容量过负荷）→ **无需额外检查**。治疗：呼吸机 + 激素。

> [!tip] 呼吸机旋钮（记哪个调什么）
> - **CO2** ← 分钟通气量：调**潮气量 VT + 呼吸频率 RR**。
> - **O2** ← 调 **FiO2 + PEEP**。
> - **ARDS 用低 VT (low VT)** = 肺保护，减少容积伤/气压伤。
> - **COPD 注意 auto-PEEP**。
> - 关键：**肺泡复张后不要降 PEEP**。

> [!tip] 气道压力判读（plateau vs peak）
> 「平台看顺应性，峰-平差值看阻力」：
> - **峰压↑ + 平台压↑ = 肺硬了（顺应性↓）**：肺水肿/ARDS/气胸。
> - **峰压↑ + 平台压正常 = 管道窄了（阻力↑）**：支气管痉挛/分泌物/管路扭折。

> [!success] 正压通气 PPV 用于急性心源性肺水肿
> PPV → RV 前负荷↓、LV 前负荷↓、LV 后负荷↓ → 促进 LV 充盈与射血 → 肺淤血↓、氧供↑。

## 十一、儿科气道速记（stridor 喘鸣）

```
吸气性 inspiratory stridor → 声门上/声门 supraglottic/glottic（喉软化 laryngomalacia / 声带功能障碍）
呼气性 expiratory stridor → 胸内气管 intrathoracic（气管软化 tracheomalacia）
双相 biphasic stridor → 固定梗阻（气管狭窄 tracheal stenosis）
```

> [!tip] 口诀
> 喉软最典型，会厌最危险，血管环要想吞咽难。

## 十二、慢性咳嗽与杂项

- **ACEI (-pril) 诱发慢性干咳 (chronic nonproductive cough)** → 停药即可。
- **URI 后亚急性咳嗽 / 上气道咳嗽综合征 UACS（鼻后滴漏）** → 一代抗组胺药。

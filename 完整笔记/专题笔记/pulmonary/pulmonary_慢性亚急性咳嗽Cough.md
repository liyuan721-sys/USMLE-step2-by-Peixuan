# 慢性 / 亚急性咳嗽 (Chronic / Subacute Cough)

> [!info] 一句话定位
> 咳嗽分诊第一步是**按病程分类**，再按"非吸烟者 + 查体正常 + 无报警征 → 经验治疗、暂不影像"的思路走。考试钩子：**URI（upper respiratory infection，上呼吸道感染）后 3–8 周咳 + postnasal drip（鼻后滴漏）+ 夜咳 = UACS（upper airway cough syndrome，上气道咳嗽综合征）→ 口服一代抗组胺**。

---

## 一、按病程分类 (Duration Classification) ⭐

| 类别 | 病程 | 最常见病因 |
|:---|:---|:---|
| **Acute（急性）** | **<3 周** | 病毒性 URI、**急性支气管炎 (acute bronchitis)**；先排肺炎 / PE（pulmonary embolism，肺栓塞）等急症 |
| **Subacute（亚急性）** | **3–8 周** | **感染后咳嗽 (post-infectious) / UACS**；考虑**百日咳 (pertussis)** |
| **Chronic（慢性）** | **>8 周** | 非吸烟者 >90% = **UACS / 哮喘 / GERD**（U→A→G） |

---

## 二、急性咳嗽：急性支气管炎 (Acute Bronchitis)

> [!note] 特征
> - 病因：**前驱呼吸道感染，90% 病毒 (viral)**
> - 咳嗽 **>5 天到 3 周**（± 脓痰 purulent sputum，来自上皮脱落 epithelial sloughing，**脓痰 ≠ 细菌感染**）
> - **无全身中毒征**（无高热/寒战）；可有 wheezing/rhonchi、胸壁压痛
> - 诊断 = **临床诊断**，**CXR（chest X-ray，胸片）仅在怀疑肺炎时做**

> [!tip] 治疗
> **对症 (symptomatic)**：NSAIDs（非甾体抗炎药）和/或支气管扩张剂；伴 wheeze/哮喘背景 → 可用 **albuterol（沙丁胺醇）**。
> **❌ 不推荐抗生素 (antibiotics not recommended)** —— 高频考点。

> [!warning] 鉴别百日咳 (Pertussis)
> 亚急性咳嗽 + **阵发性顿咳 (paroxysmal cough)、咳后吸气性"鸡鸣" (inspiratory whoop)、咳后呕吐 (post-tussive emesis)** → 想百日咳（成人疫苗减弱后常见）。诊断：**鼻咽 PCR**；治疗 **macrolide（大环内酯类，如 azithromycin 阿奇霉素）**（主要减少传播，对已起病的咳嗽改善有限）。

---

## 三、慢性咳嗽三大病因：U → A → G ⭐⭐

> [!info] 核心原则
> **非吸烟者、CXR 正常、不用 ACE-I 的慢性咳嗽，>90% 是这三个**。按 **U → A → G** 顺序经验性排查，每步"诊断性治疗"，有效即确诊。

```
非吸烟 + 查体正常 + 无报警征的慢性咳嗽
          │
   先停 ACE-I（若在用）
          │
          ▼
  U  UACS（上气道咳嗽综合征）──► 一代抗组胺 / 鼻用激素
          │（无效）
          ▼
  A  Asthma（哮喘）/ NAEB（非哮喘嗜酸性支气管炎）──► PFT + ICS 试验
          │（无效）
          ▼
  G  GERD（胃食管反流病）──► 经验 PPI 试验 + 生活方式
```

| 步骤 | 病因 | 钩子 | 经验治疗 |
|:---|:---|:---|:---|
| **U** | **UACS（上气道咳嗽综合征，旧称 postnasal drip）** | 鼻后滴漏感、清嗓、夜咳、过敏性/血管运动性鼻炎 | **一代抗组胺 (first-gen antihistamine，如 氯苯那敏 chlorpheniramine / 溴苯那敏 brompheniramine ± 伪麻黄碱 pseudoephedrine)**；或鼻用激素 (nasal steroids) |
| **A** | **Asthma（哮喘）/ 咳嗽变异性哮喘 (cough-variant asthma) / NAEB（non-asthmatic eosinophilic bronchitis，非哮喘嗜酸性支气管炎）** | 夜间/运动/冷空气诱发、可逆阻塞 | **PFT（肺功能）**，必要时 methacholine（甲胆碱）激发；**ICS（吸入糖皮质激素）试验** |
| **G** | **GERD（胃食管反流病）** | 烧心、晨起喉痛/声嘶、平卧加重；也可**无烧心 (silent)** | **经验 PPI（质子泵抑制剂）试验** + 抬高床头/减重 |

> [!danger] 第四大原因：ACE-I 诱导咳嗽
> **ACE inhibitor（血管紧张素转换酶抑制剂，"-pril"，如 lisinopril 赖诺普利）→ 慢性干咳 (chronic nonproductive cough)** —— 机制是缓激肽 (bradykinin) 蓄积。**处理 = 停药 (discontinue)**，换 ARB（血管紧张素受体拮抗剂，"-sartan"）。问诊任何慢性咳嗽**先问有没有吃 -pril**。

## 四、Q8815 重点：亚急性 URI 后咳嗽 ⭐

> [!example] 经典 stem（[[mistakes/uworld-mistakes_2026-06#^Q8815]]）
> 35 岁男，**3 周干性顿咳、夜间为主** + **鼻后滴漏感**；**4 周前 URI 已愈、仅遗留咳嗽**；不吸烟、无烧心/喘、查体正常、父肺癌（诱饵）。问最佳初始处理。
> **答案 = 口服一代抗组胺**（UACS）。

> [!warning] 三个干扰项辨析
> - **见"夜间咳"反射哮喘 → 选 ICS**：错。**postnasal drip + URI 后 = UACS**，先一代抗组胺，不是先 ICS。
> - **父肺癌 → 选胸片**：诱饵。**年轻 + 不吸烟 + 无报警征 → 不需先影像**。
> - **无烧心 → 不是 GERD**，别选 PPI；**无喘 → 不是哮喘**，别先 PFT。

> [!tip] 为什么 URI 后用"镇静的"一代抗组胺
> 二代抗组胺（如 loratadine 氯雷他定）对**感染后/非过敏性** UACS 效果差；**一代（有抗胆碱作用）**才有效，代价是嗜睡。AERD/纯过敏性鼻炎则鼻用激素也行。

## 五、何时做胸片 / 升级检查 (Red Flags) ⭐

> [!danger] 出现以下任一 → 先 CXR / 进一步查，不要只经验治疗
> - **咯血 (hemoptysis)**
> - **脓痰 + 全身中毒征**（疑肺炎）
> - **吸烟史 / 消瘦 (weight loss)**（疑恶性、COPD）
> - **免疫抑制 (immunosuppression)**
> - 经验治疗（U→A→G）**充分试验后仍无效**
>
> 反之：**非吸烟 + 查体正常 + 无上述报警 → 经验治疗优先，影像后置。**

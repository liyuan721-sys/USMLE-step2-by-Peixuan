# 呼吸衰竭插管指征 (When to Intubate — Cross-Discipline)

> [!tip] 一句话总览
> "什么时候插管"考三条独立线,别混成一条:**① 呼吸肌泵衰竭看 FVC(神经肌肉病)；② 昏迷保气道(GCS ≤8)；③ 气道要塌趁早插(水肿/烧伤/会厌炎)**。外加一条"**氧合/通气失败**"(ARDS/COPD)。核心陷阱:神经肌肉病**不看血氧、不等血气**,靠床旁肺功能预警。困难气道升级(CICV, Can't Intubate Can't Ventilate 不能插管不能通气 → 环甲膜切开)见 [[完整笔记/专题笔记/外科/外科_困难气道与紧急气道]]。

---

## §一 核心分诊：四类插管指征（整合框架）⭐⭐⭐

```
为什么要插管？先归类，指征完全不同：

① 通气泵衰竭（呼吸肌本身没力）  ← 神经肌肉病，本篇重点
     标准 = FVC / NIF（20/30/40），不是血氧！

② 气道保护（意识不清，防误吸）
     标准 = GCS ≤ 8 / 不能处理分泌物 / 呕吐反射消失

③ 气道即将梗阻（趁还开着赶紧插）
     标准 = 临床判断"会肿闭"，不等它闭 → 血管性水肿/烧伤/会厌炎

④ 氧合 or 通气失败（肺本身病 / 药物抑制）
     标准 = 顽固低氧(P/F↓)或高碳酸酸中毒；先试 NIV，可逆病因先解毒
```

> [!danger] 第一大陷阱：神经肌肉性呼吸衰竭不看"缺氧"
> **神经肌肉性呼吸衰竭 (neuromuscular respiratory failure)**——如 GBS、重症肌无力危象——**SpO₂ 和动脉血气 (ABG) 会一直正常到最后一刻,然后突然崩**(是呼吸肌泵衰竭,不是氧合障碍)。**等你看到血氧下降或 PaCO₂ 升高,已经太晚。** 所以靠床旁肺功能**预警性插管**,不靠症状/血气。

---

## §二 通气泵衰竭：神经肌肉病 → FVC/NIF 指导（本篇核心）⭐⭐⭐

> [!success] 20/30/40 法则（背这三个数）
> 满足任一 → 插管,别等血氧:
>
> | 指标 | 阈值 | 测什么 |
> |---|---|---|
> | **FVC**(用力肺活量 forced vital capacity)| **< 20 mL/kg**(或 < 1 L,或较基线↓ >30%)| 最重要,总吸气容量 |
> | **MIP / NIF**(最大吸气压 / negative inspiratory force)| **< 30 cmH₂O**(负不到 −30)| 吸气肌力 |
> | **MEP**(最大呼气压 maximal expiratory pressure)| **< 40 cmH₂O** | 咳嗽/排痰力 |
>
> 外加**球部无力 (bulbar weakness)**——不能吞咽/保护气道、分泌物堆积——即使 FVC 尚可也要插。

### 2.1 哪些病走这条线（按定位背：呼吸泵通路任何一段断都行）

| 定位 | 疾病 | 钩子 / 要点 |
|---|---|---|
| **神经肌肉接头 NMJ (neuromuscular junction)** | **重症肌无力危象 (myasthenic crisis, MG)** ⭐ | 与 GBS 同法 FVC/NIF 指导;诱因=感染/手术/药(氨基糖苷 aminoglycosides、β 阻滞剂)。区分**胆碱能危象 (cholinergic crisis)**(过量新斯的明 neostigmine,分泌物多+瞳孔缩) |
| | **肉毒中毒 (botulism)** | **下行性 (descending)** 麻痹 + 瞳孔散大 + 婴儿蜂蜜/家庭罐头;给**抗毒素 (antitoxin)** + 支持通气 |
| | **有机磷中毒 (organophosphate)** | 胆碱能危象:分泌物爆多 + 肌无力/纤颤;插管 + **阿托品 (atropine)** + **解磷定 (pralidoxime)** |
| | Lambert-Eaton (LEMS) | 少见累及呼吸 |
| **周围神经** | **GBS (吉兰-巴雷 Guillain-Barré syndrome)** ⭐ | **上行性 (ascending)** 弛缓瘫 + 反射消失 + 前驱感染;+自主神经不稳 (autonomic instability) |
| | **白喉 (diphtheria)** | 假膜阻气道 + 神经毒致呼吸肌瘫(气道+呼吸肌双威胁) |
| | 蜱麻痹 (tick paralysis) | 找到蜱拔掉即逆转 |
| **前角 / CNS** | **ALS (肌萎缩侧索硬化 amyotrophic lateral sclerosis)** | 慢性进展,详见下方 ⚠️;多先 NIV |
| | 脊髓灰质炎 (poliomyelitis) | 前角运动神经元 |
| **脊髓** | **高位颈髓损伤 (C3–C5)** ⭐ | **"C3, 4, 5 keep the diaphragm alive"**——膈神经 (phrenic nerve) 源自 C3–5;C5 以上损伤 → 膈肌瘫 → 立即插管 |
| **其他** | 破伤风 (tetanus) | 喉痉挛 + 呼吸肌强直 |

> [!warning] ALS 是例外：不等于无脑插管
> **ALS** 是慢性进展、不可逆病:急性失代偿前通常先上**无创通气 (NIV, 如 BiPAP)**,且涉及**目标性治疗讨论 (goals of care)**——很多患者预先选择不插管。真正气管插管更多见于急性可逆诱因(如肺炎)或患者明确选择全力抢救时。对比 GBS/MG 是**可逆**的,插管是"过桥"抢时间。详见 [[完整笔记/专题笔记/神经/神经_ALS运动神经元病]] / [[完整笔记/专题笔记/神经/神经_GBS吉兰巴雷体系]]。

## §三 气道保护 vs 气道梗阻（两条别混）

> [!info] ② 气道保护 (protect the airway)：意识不清防误吸
> **GCS ≤ 8 → 插管**(GCS = Glasgow 昏迷评分 Glasgow Coma Scale;"less than 8, intubate")。机制:意识障碍 → 舌后坠、呕吐反射消失、不能清理分泌物 → 误吸风险。适用:严重 TBI(创伤性脑损伤 traumatic brain injury)、卒中、深昏迷、过量镇静。
> ⚠️ **可逆药物抑制先解毒**:阿片过量呼吸抑制 → **先给纳洛酮 (naloxone)**,常无需插管;苯二氮䓬过量 → 支持为主(氟马西尼 flumazenil 慎用)。见 [[完整笔记/专题笔记/USMLE/USMLE_呼吸抑制低通气鉴别]]。

> [!danger] ③ 气道即将梗阻：趁气道还开着赶紧插（early / prophylactic）
> 机制与①完全不同——不是呼吸肌没力,是**气道解剖要肿闭**。看到"会肿闭"的信号就插,**不等它真的闭**(闭了插不进、只能环甲膜切开):
>
> | 病因 | 早插信号 |
> |---|---|
> | **血管性水肿 (angioedema)**(ACEI 诱发 / C1 酯酶抑制物缺乏)| 舌/唇/喉肿、声嘶、喘鸣 stridor |
> | **过敏性休克 (anaphylaxis)** | 喉头水肿(同时给肾上腺素 epinephrine)|
> | **吸入性烧伤 (inhalation injury)** | 面部烧伤、鼻毛焦、烟灰痰 (carbonaceous sputum)、声嘶 |
> | **会厌炎 (epiglottitis)** / **Ludwig 咽峡炎 (Ludwig angina)** | 三脚架体位、流涎、喘鸣 |
> | **扩张性颈部血肿 (expanding neck hematoma)** | 颈部术后/外伤,气管受压移位 |
>
> 插管失败 → CICV → 环甲膜切开,完整阶梯见 [[完整笔记/专题笔记/外科/外科_困难气道与紧急气道]]。儿童喘鸣气道见 [[完整笔记/专题笔记/儿科/儿科_喘鸣stridor急性气道鉴别]]。

## §四 氧合 / 通气失败（肺本身病）— 简版

| 类型 | 触发 | 先试什么 → 何时插管 |
|---|---|---|
| **低氧性 (I 型)** | ARDS(急性呼吸窘迫综合征 acute respiratory distress syndrome)、重症肺炎、心源性肺水肿 (cardiogenic pulmonary edema) | 心源性肺水肿/部分肺炎先试 **NIV**;顽固低氧(P/F 即 PaO₂/FiO₂ 持续↓)、意识差、NIV 失败 → 插管 + 机械通气 |
| **高碳酸性 (II 型)** | COPD(慢性阻塞性肺病 chronic obstructive pulmonary disease)急性加重、哮喘 (asthma) 濒死、药物抑制 | COPD 急性加重先 **BiPAP**;pH 持续↓/嗜睡/BiPAP 失败 → 插管 |

> [!tip] 与机械通气衔接
> 一旦决定插管 → 参数设置、气道压判读见 [[完整笔记/专题笔记/pulmonary/pulmonary_机械通气压力判读]] / [[完整笔记/专题笔记/pulmonary/pulmonary_低氧血症五机制]]。

## §五 陷阱速记 ⭐

> [!danger] 高频错点
> - **神经肌肉病看血氧 = 错**。SpO₂/ABG 正常≠安全,看 **FVC/NIF**。GBS 患者"血氧 98% 但 FVC 掉到 15 mL/kg" → 插管。
> - **"伸不直/喘不上"别都当肺病**:GBS/MG 是泵衰竭,肺是好的。
> - **阿片过量先纳洛酮**,不是先插管(可逆病因先逆转)。
> - **气道要肿闭时别等**:血管性水肿/烧伤"现在能插"就插,等 stridor 变重可能插不进。
> - **ALS 别无脑插**:先问 goals of care + NIV;GBS/MG 才是"过桥式"插管。
> - **胆碱能危象 vs 肌无力危象**(都在 MG 患者):分泌物爆多+瞳孔缩+腹泻 = 胆碱能(药过量,停药);无力为主 = 肌无力危象(加药/血浆置换)。

## §六 自测（答完再判分）

> [!question]- Q1. 28 岁女 GBS 上行性无力 3 天,现 SpO₂ 97%、ABG 正常,但诉说话要停顿换气、抬头费力。下一步监测/处理最关键是什么?为什么不能因血氧正常就观察?
> 床旁 **FVC + NIF (20/30/40)**;FVC < 20 mL/kg 或快速下降 → 插管。神经肌肉性呼吸衰竭血氧/血气**正常到最后一刻然后突然崩**,不能等缺氧。

> [!question]- Q2. 婴儿便秘 + 眼睑下垂 + 吸吮无力 + 呼吸浅,进行性下行性无力。诊断?通气策略?
> **婴儿肉毒中毒 (infant botulism)**(蜂蜜史)。下行性麻痹累及呼吸肌 → 支持性机械通气 + **肉毒抗毒素 (human botulism immune globulin, BIG-IV)**。

> [!question]- Q3. ACEI 服用者晨起舌唇肿胀、声嘶、轻度喘鸣。气道处理原则?为什么不能"先观察等消肿"?
> **血管性水肿 (angioedema)** → **趁气道还开着尽早插管**;水肿会进展至喉部完全梗阻,届时插不进只能环甲膜切开。停 ACEI。

> [!question]- Q4. 阿片过量,RR 6、针尖瞳、SpO₂ 82%、昏睡。先插管还是先做别的?
> **先给纳洛酮 (naloxone)** 逆转——可逆病因优先;多数可避免插管。若气道无法保护/纳洛酮无效再插。

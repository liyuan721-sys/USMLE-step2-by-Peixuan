# 哮喘 Asthma

> [!info] 一句话定位
> **Asthma (哮喘) = 气道慢性炎症 + 气道高反应性 (bronchial hyperresponsiveness)**，核心特征是**可逆性气流阻塞 (reversible obstruction)**。考试钩子：**年轻 + 发作性喘鸣 + 夜间/运动/冷空气诱发 + 过敏背景 + 嗜酸高**。

---

## 一、病理生理 (Pathophysiology)

**两条通路共同导致支气管收缩：**

```
Allergen (过敏原)
   │
   ▼
 IgE → Mast cell degranulation (肥大细胞脱颗粒)
   │
   ├─► Histamine (组胺) + Leukotrienes (白三烯)
   │        │
   │        ▼
   │   ① 炎症通路 (eosinophils 嗜酸 / Th2)
   │   ② 副交感兴奋 (M3 受体 → bronchoconstriction)
   │
   ▼
 Bronchoconstriction (支气管收缩) + Mucus (黏液) + Edema (水肿)
```

> [!tip] 记忆钩子
> 治疗思路 = **"一松一消"**
> - **松**：松弛支气管平滑肌（β₂ 激动剂、抗胆碱药、茶碱 theophylline）
> - **消**：控制炎症（吸入激素 ICS、抗白三烯 antileukotriene、生物制剂 biologics）

---

## 二、临床表现 (Presentation)

> [!note] 典型四联
> - **Episodic wheezing (发作性喘鸣)** — 呼气相为主，**worse at night（夜间加重）+ cold air（冷空气）**
> - Associated with **allergies (过敏)**
> - 运动诱发 (exercise-induced)
> - **Eosinophilia (嗜酸性粒细胞增多)**

**发作时血气 (ABG during attack)：**

- 早期/轻中度：**低 CO₂ + 呼碱 (respiratory alkalosis)**，因为过度通气
- **CO₂ 正常或升高 = 危险信号** → 患者快撑不住了（呼吸肌疲劳），考虑 intubation（插管）

**DLCO（弥散功能）：** Normal to **↑（高，因 gas trapping 气体陷闭）** — 这是与 COPD（DLCO ↓）的关键鉴别点。

---

## 三、诊断 (Diagnostics)

> [!info] 核心公式
> **Obstruction（阻塞性）：FEV₁/FVC < 70%**

| 项目 | 数值 / 意义 |
|:---|:---|
| **支气管扩张剂可逆性** (bronchodilator reversibility) | FEV₁ ↑ **≥12% 且 ≥200 mL**（SABA 后）→ 支持哮喘 |
| **Methacholine challenge** (激发试验) | FEV₁ ↓ >15–20% → 阳性 |
| **DLCO** | Normal / ↑ |

> [!tip] 诊断决策流
> Spirometry 显示阻塞 + 可逆 → Asthma。
> **若 spirometry 正常**（无症状期常见）→ 做 **Methacholine bronchoprovocation test（甲胆碱激发试验）**：
> - **Positive（阳性）→ Asthma possible**
> - **Negative（阴性）→ 排除 Asthma**（阴性预测值高）

## 四、长期治疗阶梯 Step 1–5（成人，GINA 思路）⭐

> [!warning] 当前 GINA 框架核心
> **首选 reliever（缓解药）= ICS–formoterol PRN**，不再单用 SABA。理由：单用 SABA 救急虽快但不抗炎，**反而增加重症发作与死亡风险**。

| Step | 症状频率 | 治疗 |
|:---|:---|:---|
| **Step 1** | Less than daily（间断发作） | **ICS–formoterol PRN**（或 ICS + SABA PRN） |
| **Step 2** | Less than daily 但略频繁 | **ICS–formoterol PRN** |
| **Step 3** | Most days（大多数天有症状） | **低剂量 ICS–formoterol 每日 + PRN**（maintenance + reliever 同一制剂 = MART）|
| **Step 4** | Daily 症状 + FEV₁ ↓ | **中/高剂量 ICS–formoterol 每日 + PRN** |
| **Step 5** | Step 4 后仍未控制 + 频繁需 OCS | **高剂量 ICS–LABA + LAMA**；加**生物制剂**（Anti-IgE / Anti-IL-5），必要时短程 OCS |

> [!info] MART 是什么（Step 3+ 的核心给药模式）
> **MART = Maintenance And Reliever Therapy（维持与缓解联合疗法）**：用**同一支 ICS–formoterol（如 budesonide 布地奈德–formoterol 福莫特罗）**，既当每日固定维持药、又当喘了按需吸的缓解药——**一支两用**。
> - **为什么只能用 formoterol（福莫特罗）**：它是**起效快的 LABA**（几分钟见效，像 SABA 一样快）又长效，所以能同时担任"救急"和"长期控制"。**salmeterol（沙美特罗）起效慢，不能当 reliever**，只能做维持。
> - **好处**：取代"ICS 维持 + 单独 SABA 救急"的两支方案；每次救急都顺带吸入抗炎 ICS → 减少重症发作。

> [!tip] 记忆钩子（阶梯）
> - **Step 1–2：Less than daily → ICS–formoterol PRN**
> - **Step 3：Most days → 每日低剂量 ICS–formoterol + PRN**
> - **Step 4：Daily + FEV₁↓ → 每日中/高剂量 ICS–formoterol**
> - **Step 5：还是不行 + 频繁 OCS → 加 LAMA + 生物制剂**

> [!danger] USMLE 必考陷阱
> **LABA 绝不能单用！** 必须与 ICS 合用，否则 → **↑死亡率**。看到题目给"单用 salmeterol/formoterol"——错。

## 五、急性发作 (Asthma Exacerbation) 处理

```
Auscultate (听诊)
   │
   ├── No airflow / silent chest (无气流/沉默肺) ──► INTUBATE (插管) ⚠️
   │
   └── Wheezing (有喘鸣)
          │
          ▼
       Oxygen + ABG
          │
          ▼
   Albuterol (SABA) → + Ipratropium (SAMA) → IV steroids (甲泼尼龙 methylprednisolone)
          │
          ▼
   Discharge: Oral steroids（短程 5–7 天 burst，NO taper）
```

> [!warning] "沉默肺"是最危险信号
> 喘鸣突然消失 + 气流消失（silent chest）≠ 好转，而是**气道几乎完全闭塞**，立即插管。不要被"听诊变安静"误导。

> [!tip] 激素选择
> **急性哮喘/COPD 加重 → IV methylprednisolone（甲泼尼龙）**，不是 prednisone（起效慢，需肝转化）、也不是 dexamethasone（太长效，抑轴+感染风险）。
> **Acute exacerbation 短程 systemic steroid burst（5–7 天）→ NO taper**（疗程太短不抑制 HPA 轴，研究证明 taper 无获益）。

## 六、特殊类型 (Special Forms)

| 类型 | 钩子 | 处理 |
|:---|:---|:---|
| **Exercise-induced（运动诱发）** | 运动/冷空气前喘 | 活动前吸 **albuterol** |
| **Aspirin-induced / AERD（阿司匹林诱发）** | 哮喘 + **nasal polyps（鼻息肉）** + NSAID/aspirin 不耐受 = **Samter 三联** | **Montelukast（孟鲁司特）** + 避免 NSAID |
| **Extrinsic / allergic（外源性/过敏性）** | 明确过敏原 | **Cromolyn（色甘酸钠）** 预防肥大细胞脱颗粒 + albuterol |

> [!question]- AERD（Samter 三联）为什么首选 Montelukast？
> AERD = **A**spirin-**E**xacerbated **R**espiratory **D**isease。机制：NSAID 抑制 COX → 花生四烯酸通路转向 **5-LOX → 白三烯过量** → 支气管收缩 + 鼻息肉。所以**抗白三烯药（montelukast）**直击病机。三联 = 哮喘 + 鼻息肉 + aspirin/NSAID 不耐受。

## 七、药物分类速查 (Drug Classes)

| 类别 | 代表药 | 机制 | 用途 |
|:---|:---|:---|:---|
| **SABA** | Albuterol（沙丁胺醇） | β₂ → ↑cAMP → 平滑肌松弛 | **急性发作救命药**（rescue） |
| **LABA** | Salmeterol / Formoterol | 长效 β₂ | **绝不单用，必须配 ICS** |
| **ICS** | Fluticasone / Budesonide | 抑 NF-κB → 抗炎 | **慢性维持一线**；副作用 oral thrush（鹅口疮）→ 用 spacer + 漱口 |
| **SAMA** | Ipratropium（异丙托溴铵） | 阻 M3 | 急性重症 add-on（雾化 albuterol + ipratropium = DuoNeb） |
| **Antileukotriene** | Montelukast / Zafirlukast（CysLT₁ 拮抗）；Zileuton（5-LOX 抑制，肝毒） | ↓白三烯 | 口服维持；**AERD、运动诱发、儿童夜咳、家长拒激素** |
| **Mast cell stabilizer** | Cromolyn（色甘酸钠） | 稳定肥大细胞 | 过敏性预防 |
| **Biologics（Step 5）** | Omalizumab（抗 IgE，allergic）；Mepolizumab/Reslizumab（抗 IL-5，eosinophilic） | 靶向 | 重症难治 |

> [!warning] 全身激素效力对比（Asthma/COPD 急救相关）
> **Prednisone（口服慢病）< Methylprednisolone（急救首选）；Dexamethasone（脑水肿/croup，不用于哮喘）。**
> 口诀：**Pred 用在口服慢病，Methyl 用在急救，Dex 用在水肿。**

> [!danger] SABA / β₂ 激动剂不良反应 —— 大剂量 → 低钾血症（Q3026）⭐
> **大剂量沙丁胺醇 (albuterol)（持续雾化）激活 Na⁺/K⁺-ATP 酶 → 把钾驱入细胞 → 低钾血症 (hypokalemia)**：
> - 表现：**肌无力 + 深腱反射减弱 (hyporeflexia) + 震颤 (tremor) + 心动过速/心律失常**（重者室颤）。
> - 急性哮喘"治疗后数小时新发肌无力 + 反射↓ + 手抖" → 先想**低钾**，下一步 **查血清电解质 + ECG → 补钾**。
> - **β₂ 常见副作用三连：震颤、心悸/心动过速、低钾**（剂量相关）。
> - 别误归到中毒：**高铁血红蛋白血症 (methemoglobinemia)**（硝酸盐/氨苯砜 dapsone/苯佐卡因 benzocaine → 吸氧不改善的发绀）与沙丁胺醇无关。

## 八、Asthma vs COPD 鉴别 ⭐

| 项目 | Asthma | COPD |
|:---|:---|:---|
| 年龄 | 年轻 | 年长（吸烟史） |
| FEV₁/FVC | <70% | <70% |
| **SABA 可逆性** | **可逆（≥12% + ≥200 mL）** | **不可逆 (nonreversible)** |
| **DLCO** | **Normal / ↑** | **↓** |
| 喘鸣 | **Episodic（发作性）** | Chronic（慢性） |
| 诱因 | 过敏 | 吸烟、感染 |
| 夜间症状 | **常见** | 少见 |

**急性加重鉴别（AECOPD vs acute asthma）：**

- 有 sputum（痰）+ 大量 pack-year（吸烟）+ 慢性进行性咳 → 偏 **AECOPD**
- PaCO₂：**asthma 通常低**，**COPD 通常高**
- 经验治疗：哮喘 = albuterol + IV 激素；**AECOPD = albuterol + IV 激素 + IV 抗生素**（3代头孢/氟喹诺酮）

> [!danger] Anaphylaxis（过敏性休克）vs Asthma exacerbation
> **突然呼吸困难 + 荨麻疹/血管性水肿 + 低血压 + 可疑药物/食物 → 一律先想 Anaphylaxis，立即 IM epinephrine（肾上腺素）**，不要纠结是不是哮喘。
> NSAID 既诱发 aspirin-sensitive asthma 又加重 anaphylaxis；看到 ibuprofen + 皮疹/喉紧 → 站 Anaphylaxis 这边。

# 纵隔肿块前中后定位（Mediastinal Mass 三分区鉴别）

> [!info] 为什么建这张
> 派生自 NBME11A Q144（全身乏力 + 复视 + 前上纵隔肿块 → thymoma）。CK 高频先用**肿块所在分区**收窄鉴别，再用伴随线索（如 MG 症状）锁定单一诊断。**先分区，后配症状**。

## 一、三分区解剖 + 高频肿块

> [!tip] 记忆锚：前纵隔 "4 Ts"，中纵隔 "血管淋巴气道"，后纵隔 "神经源性"
> 侧位胸片 (lateral CXR) 以心脏/大血管前缘、气管后缘划界。

| 分区 (compartment) | 常见肿块 | 关键钩子 |
|---|---|---|
| **前纵隔 (anterior)** | **胸腺瘤 (thymoma)**、畸胎瘤/生殖细胞瘤 (teratoma / germ cell tumor)、胸内甲状腺肿 (thyroid goiter)、（terrible）淋巴瘤 (lymphoma) | **"4 Ts"**：Thymoma / Teratoma / Thyroid / Terrible lymphoma |
| **中纵隔 (middle)** | 淋巴结/淋巴瘤 (lymphadenopathy)、支气管囊肿 (bronchogenic cyst)、心包囊肿 (pericardial cyst)、大血管病变 | 气道/血管/淋巴结结构 |
| **后纵隔 (posterior)** | **神经源性肿瘤 (neurogenic tumor)**：神经鞘瘤 / 神经母细胞瘤 (neuroblastoma, 儿童) | 沿脊柱旁沟；儿童后纵隔肿块先想 neuroblastoma |

```text
前纵隔 4 Ts:
  Thymoma  ── + MG / 复视乏力  → 直接锁 thymoma
  Teratoma ── 年轻 + 钙化/脂肪 + βhCG/AFP↑
  Thyroid  ── 颈部延伸 + 气管受压
  (Terrible) Lymphoma ── B 症状 + 双侧淋巴结
```

## 二、Q144 主线：Thymoma × Myasthenia Gravis

> [!danger] 前纵隔肿块 + fatigable weakness / 复视上睑下垂 = thymoma 关联 MG，几乎锁死
> **Myasthenia gravis (MG, 重症肌无力)** = 抗**乙酰胆碱受体 (acetylcholine receptor, AChR)** 抗体攻击突触后膜 → **易疲劳性无力 (fatigable weakness)**，眼肌最常受累（**复视 diplopia / 上睑下垂 ptosis**）。
> - **~15% MG 患者合并 thymoma**；**~30–50% thymoma 患者有 MG**（双向关联）。
> - 题目一旦同时给"全身乏力 + 复视"和"前纵隔肿块"，答案就是 **thymoma**，不必再纠结其他 T。

```text
异常胸腺免疫环境
  → 产生抗 AChR 自身抗体
  → 突触后神经肌接头传导受损
  → 眼肌 + 全身易疲劳性无力（复视/上睑下垂/晚间加重）
  → 胸部影像可见胸腺增生或 thymoma
```

> [!info] 诊断与处理速记
> - 确诊 MG：**血清抗 AChR 抗体**（首选）；辅以 ± 抗 MuSK 抗体、重复神经电刺激、单纤维肌电图。
> - **所有新诊断 MG → 查胸部 CT** 找 thymoma。
> - **有 thymoma → 胸腺切除 (thymectomy)**（可改善甚至部分缓解 MG）。

## 三、易混陷阱（普适）

> [!warning]
> - **别把"前纵隔肿块"直接当淋巴瘤**：无 B 症状、无广泛淋巴结、却有 MG 症状 → thymoma 优先。
> - **后纵隔肿块 ≠ 前纵隔 4Ts**：脊柱旁 → 神经源性（儿童 neuroblastoma）。
> - **胸内甲状腺肿**：常有颈部肿块延伸 + 气管移位/吞咽困难，与 thymoma 的神经肌症状不同。
> - **年轻男性前纵隔肿块 + AFP/βhCG 升高** → 生殖细胞瘤，非 thymoma。

> [!question]- Q1. 28 岁女性 2 周来全身乏力 + 间断复视，晨轻暮重，胸片示前上纵隔肿块。最可能诊断？
> **Thymoma（胸腺瘤）**。前纵隔肿块 + 易疲劳性眼肌/全身无力 = MG 关联 thymoma，经典绑定。下一步查血清抗 AChR 抗体 + 胸部 CT，确诊后 thymectomy。

> [!question]- Q2. 前纵隔 "4 Ts" 分别指什么？
> **Thymoma / Teratoma（畸胎瘤·生殖细胞瘤）/ Thyroid（胸内甲状腺肿）/ (Terrible) Lymphoma（淋巴瘤）**。后纵隔肿块则先想神经源性肿瘤。

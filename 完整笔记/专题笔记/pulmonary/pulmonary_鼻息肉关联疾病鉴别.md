# 鼻息肉 (Nasal Polyps) 关联疾病鉴别

> [!info] 这张笔记解决什么
> 鼻息肉 (nasal polyps, 鼻腔/鼻窦良性炎性赘生物) 在 USMLE 里几乎从不是"主角"——它是一个**指向背后疾病的钩子**。命题套路：给你鼻息肉，让你倒推**为什么这个人会长**。鉴别第一刀永远是 **年龄（儿童 vs 成人）**。

---

## 一、核心框架：按年龄分流

```
                    鼻息肉 (nasal polyps)
                          │
          ┌───────────────┴───────────────┐
        儿童                              成人
   （正常儿童几乎不长！                （炎性/嗜酸性最常见）
    出现 = 结构/外分泌/纤毛病）              │
          │                    ┌──────────┼──────────┐
   ┌──────┴──────┐          CRSwNP      AERD       EGPA
  CF           PCD        (最常见)    (Samter 三联) (血管炎+嗜酸爆高)
(囊性纤维化)  (纤毛不动)      │
          │            过敏/AFRS（变应性真菌性鼻窦炎）
   先查汗液氯化物试验
```

> [!danger] 第一考点（背下来）
> **儿童一旦出现鼻息肉 = 囊性纤维化 (CF) 直到排除！** 必做 **汗液氯化物试验 (sweat chloride test)**——哪怕没有典型肺/消化症状。第二位想 **原发性纤毛运动障碍 (PCD)**。

---

## 二、关联疾病总表

| 疾病 | 关键钩子 | 反射动作 |
|---|---|---|
| **囊性纤维化 (CF, cystic fibrosis)** | **儿童**息肉 + 反复鼻窦/肺感染 + 脂肪泻 + 发育迟缓 (FTT, failure to thrive) | **汗液氯化物试验**；早发支扩、金葡/铜绿气道定植 |
| **原发性纤毛运动障碍 (PCD, primary ciliary dyskinesia)** | 儿童息肉 + 慢性鼻窦炎 + 支气管扩张 + **内脏反位 = Kartagener** + 不育 | 鼻一氧化氮↓；纤毛活检/基因 |
| **阿司匹林加重性呼吸病 (AERD / Samter 三联)** | **成人**：哮喘 + 鼻息肉 + **阿司匹林/NSAID 后喘息**；嗅觉减退（食物变淡）| **孟鲁司特 (montelukast)** + 避 NSAID + 鼻用激素；需用 ASA → 脱敏 |
| **慢性鼻窦炎伴鼻息肉 (CRSwNP)** | 成人最常见，嗜酸性炎症，常合并哮喘 | 鼻用激素一线；难治 → 生物制剂（抗 IL-4/5）|
| **嗜酸性肉芽肿性多血管炎 (EGPA / Churg-Strauss)** | 哮喘 + 鼻息肉 + **外周嗜酸 ↑↑（>1500/>10%）** + 单神经炎 + 游走肺浸润 | p-ANCA/MPO；激素 ± 免疫抑制 |
| **变应性真菌性鼻窦炎 (AFRS, allergic fungal rhinosinusitis)** | 年轻特应质 + 黏稠分泌物 + 息肉 + CT 高密度真菌团 | 手术清创 + 鼻用激素 |

## 三、两条主线分论

### 3.1 儿童线：CF vs PCD

> [!tip] 为什么儿童息肉这么关键
> 正常儿童的鼻黏膜基本不形成息肉。儿童出现鼻息肉，等于身体在喊"我有底层的外分泌/纤毛/结构病"。

```
儿童鼻息肉
   ├─ + 脂肪泻/胰功能不全/FTT + 反复肺感染 → CF（查汗液氯化物）
   └─ + 内脏反位 + 支扩 + 不育/中耳炎 → PCD/Kartagener（鼻 NO↓）
```

- **CF**：CFTR 基因 → 黏液脱水变稠 → 鼻窦/气道反复感染 + 息肉；汗液氯化物 ≥60 mmol/L 确诊。
- **PCD (Kartagener)**：纤毛结构缺陷 → 黏液清除障碍。**三联 = 慢性鼻窦炎 + 支气管扩张 + 内脏反位**；男性不育（精子不动）。

### 3.2 成人线：AERD vs CRSwNP vs EGPA

最常见是 **CRSwNP**（单纯嗜酸性炎症）；但 USMLE 真正爱考的是 **AERD** 和 **EGPA** 这两个"哮喘 + 息肉"的兄弟。

> [!warning] AERD vs EGPA —— 都有"哮喘 + 鼻息肉"，靠什么分？
> 
> | | **AERD (Samter 三联)** | **EGPA (Churg-Strauss)** |
> |---|---|---|
> | 嗜酸 | 可轻度↑ | **极高（>1500 / >10%）** |
> | 血管炎 | **无** | **有**（单神经炎、可触性紫癜、游走肺浸润、肾）|
> | ANCA | 阴性 | **p-ANCA / MPO ~40% 阳** |
> | 触发 | **阿司匹林/NSAID 后发作** | 哮喘多年后进展为系统性 |
> | 治疗 | 孟鲁司特 + 避 NSAID + 脱敏 | 全身激素 ± 免疫抑制（如重症 + 器官受累）|
> 
> **钥匙**：哮喘 + 息肉只到 AERD；再叠加**嗜酸爆高 + 神经/皮肤/肾受累**，就从 AERD **升级到 EGPA**。

> [!question]- AERD 为什么首选孟鲁司特 (montelukast)？
> AERD = **A**spirin-**E**xacerbated **R**espiratory **D**isease。机制：NSAID 抑制 **COX-1** → 花生四烯酸 (arachidonic acid) 转向 **5-脂氧合酶 (5-LOX)** 通路 → **半胱氨酰白三烯 (cysteinyl leukotrienes) ↑↑** → 支气管痉挛 + 鼻息肉。所以**抗白三烯药（孟鲁司特 montelukast / 齐留通 zileuton）直击病机**。属**假性过敏（非 IgE）**。

## 四、易混陷阱

> [!danger] 单侧鼻息肉 ≠ 普通炎性息肉
> 普通炎性鼻息肉几乎都是**双侧灰白发亮**。**单侧**息肉（尤其儿童单侧、易出血、进行性鼻塞）要警惕**肿瘤**：
> - **青少年男性 + 单侧 + 反复鼻出血** → 青少年鼻咽血管纤维瘤 (juvenile nasopharyngeal angiofibroma)
> - 成人单侧 → 内翻性乳头状瘤 (inverted papilloma)、鼻窦肿瘤
> 
> → 需影像 + 活检，不能当炎性息肉处理。

> [!tip] 嗅觉减退（"食物变淡"）是鼻息肉的隐藏线索
> 双侧鼻息肉堵塞嗅裂 → **嗅觉减退/丧失 (hyposmia/anosmia)**。题里"食物吃起来没味/变淡"常是在暗示鼻息肉，而非真正味觉障碍。

## 五、复习题

> [!question]- Q1. 8 岁男孩反复鼻窦感染，查体双侧鼻息肉，最该做什么？
> **汗液氯化物试验 (sweat chloride test)**——儿童鼻息肉高度提示囊性纤维化 (CF)，即使无典型肺/消化症状也要排除（第二位想 PCD）。

> [!question]- Q2. 28 岁女性鼻塞 + 嗅觉减退，1 年前服萘普生后严重喘息，姐姐有哮喘。最可能原因？
> **鼻息肉（背景 = AERD / Samter 三联）**。哮喘 + 息肉 + NSAID 后喘息三联；治疗加**孟鲁司特** + 避 NSAID。

> [!question]- Q3. 成人哮喘多年，近来鼻息肉 + 外周嗜酸 18% + 足下垂（单神经炎）+ 游走性肺浸润。诊断？
> **EGPA (Churg-Strauss)**。哮喘 + 息肉 + **嗜酸爆高 + 血管炎（单神经炎）** → 超出 AERD，查 p-ANCA/MPO，全身激素治疗。

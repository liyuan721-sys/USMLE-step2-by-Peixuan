# 咯血鉴别与 HHT（Hemoptysis & Hereditary Hemorrhagic Telangiectasia）

> [!info] 一句话定位
> 咯血 (hemoptysis) stem 的套路：先按**来源（气道/肺实质/血管）**分类，再用**伴随线索**锁病因。本卡重点落到一个高陷阱病——**遗传性出血性毛细血管扩张症 (HHT)** 的肺动静脉畸形 (AVM)，它的"自幼鼻出血 + 连续性肺杂音 + 光滑无钙化结节 + 缺铁贫血"招牌极易被误判成血管炎。
> 错题驱动：[[mistakes/uworld-mistakes_2026-06#^Q19811]]（HHT 肺 AVM，误选 GPA）。
> 素材整合自 [[完整笔记/Peixuan分科笔记/pulmonary]] / [[完整笔记/Peixuan分科笔记/hematology oncology]]（只读）。

> [!note]- 本篇缩写速查（首次全称中英）
> - **HHT** = hereditary hemorrhagic telangiectasia 遗传性出血性毛细血管扩张症（= Osler-Weber-Rendu 综合征）
> - **AVM** = arteriovenous malformation 动静脉畸形
> - **GPA** = granulomatosis with polyangiitis 肉芽肿性多血管炎
> - **GN** = glomerulonephritis 肺小球肾炎
> - **TB** = tuberculosis 结核
> - **GI** = gastrointestinal 胃肠道
> - **CVA** = cerebrovascular accident 脑血管意外
> - **PAH** = pulmonary arterial hypertension 肺动脉高压
> - **IE** = infective endocarditis 感染性心内膜炎

## 一、咯血病因按来源分类 ⭐

| 来源 | 代表病 | 钥匙线索 |
|---|---|---|
| **气道 (airway)** | 支气管炎、支气管扩张、**支气管肺癌**、异物 | 慢咳脓痰（支扩）/ 吸烟+体重↓（癌）|
| **肺实质 (parenchyma)** | 肺炎、肺脓肿、**结核 TB**、真菌、Goodpasture/GPA（弥漫肺泡出血）| 发热盗汗（TB）/ 肺-肾综合征（血尿）|
| **血管 (vascular)** | **肺栓塞 PE**、二尖瓣狭窄、**肺 AVM（HHT）** | 胸膜痛+腿肿（PE）/ 自幼鼻出血+连续杂音（HHT）|

> [!tip] 大咯血的"先做什么"
> 大咯血（危及气道）→ **先保气道（患侧卧位 + 必要时插管）** → 定位（支气管镜/CT 血管造影）→ 止血（支气管动脉栓塞）。本卡聚焦病因识别，量级处理见分科笔记。

> [!warning] 体位逻辑：大咯血"坏肺朝下" vs 低氧"好肺朝下"（反直觉对比）⭐
> 同样是侧卧位，目标不同 → 朝向**正好相反**。关键看你要解决的是"血淹对侧"还是"氧合不足"。

| 情境 | 摆位 | 重力解决的问题 |
|---|---|---|
| **大咯血 (massive hemoptysis)** | **患侧（出血侧）朝下**（bad lung down） | 把血局限在患侧、**防止血液溢流淹没健侧肺 → 防窒息**（致死主因是窒息，不是失血） |
| **单侧肺病变致低氧**（单侧肺炎/肺挫伤等） | **健侧（好肺）朝下**（good lung down） | 重力增加健侧灌注 → 改善 **V/Q（通气血流比 ventilation/perfusion）匹配** → 升氧合 |

> [!tip] 记忆钩子
> **大咯血 = "bad lung down"（出血肺朝下，怕血淹好肺）；单侧病变低氧 = "good lung down"（好肺朝下，增灌注提氧合）。** 一字之差、方向相反——先判断"主要矛盾是出血污染还是氧合"，再决定朝向。前提都是**已知哪侧出血/病变**；未定位时先按气道保护通则（必要时插管、选择性隔离健侧）。

## 二、HHT / Osler-Weber-Rendu ⭐⭐⭐

> [!danger] HHT 一眼识别（三联）
> **① 反复鼻出血 (recurrent epistaxis，自幼) + ② 黏膜皮肤毛细血管扩张 (mucocutaneous telangiectasia) + ③ 内脏 AVM（含一级亲属史）** = HHT。常染色体显性，血管生成基因突变。

### 2.1 四个血管床的表现（按器官记）

| 血管床 | 后果 |
|---|---|
| **鼻/口/皮肤** | 反复鼻出血（最常见、最早）+ 口唇/指端毛细血管扩张 |
| **肺 (pulmonary AVM)** | **咯血 + 连续性肺杂音 (continuous bruit，湍流) + 胸片多发圆形光滑无钙化结节**；可矛盾栓塞 → **脑脓肿/卒中 (CVA)** |
| **GI（胃肠血管发育异常）** | 慢性消化道失血 → **缺铁性（微细胞）贫血** |
| **脑/肝** | 脑 AVM、肝 AVM（门脉高压/高输出心衰）|

### 2.2 诊断与处理

- **诊断**：临床 Curaçao 标准（鼻出血/毛细血管扩张/内脏 AVM/家族史，≥3 项确诊）；肺 AVM 用**胸部增强 CT / 肺血管造影**。
- **肺 AVM 处理**：症状性（咯血/低氧/矛盾栓塞）→ **肺血管造影 + 栓塞 (embolization)**。
- **筛查**：确诊者筛肺 AVM（防咯血/矛盾栓塞）+ 脑 AVM；一级亲属遗传咨询。

## 三、HHT vs 易混病（咯血 + 肺结节鉴别）⭐

| 病 | 鉴别钥匙 | 与 HHT 的分水岭 |
|---|---|---|
| **肉芽肿性多血管炎 (GPA)** | 鼻中隔坏死/破坏性鼻窦炎 + 弥漫肺泡出血（爆裂音+斑片）+ **镜下血尿（新月体 GN）** | **GPA 有血尿（肺-肾）；HHT 尿正常 + AVM 连续杂音** |
| **播散性结核 (disseminated TB)** | 全身症状（发热/恶病质/盗汗）+ **粟粒影** | TB 无自幼鼻出血；HHT 一般情况好 |
| **肺组织胞浆菌病 (histoplasmosis)** | **钙化**结节（愈合肉芽肿）+ 肺门淋巴结 | HHT 结节**无钙化**且伴杂音 |
| **脓毒栓子 (septic emboli)** | 空洞肺脓肿 + 脓痰 + 右心 IE（三尖瓣杂音）+ 急性病容发热 | HHT 非急性感染、结节实性光滑 |

> [!warning] 本题陷阱（Q19811）
> "咯血 + 肺结节"易反射肺-肾血管炎 (GPA)，但 **HHT 招牌 = 自幼鼻出血 + 连续性肺杂音 + 无钙化光滑结节 + 缺铁贫血 + 尿正常**。**尿正常基本否定 GPA**（GPA 该有血尿）。

## 四、Memory Hook

```
咯血先分来源：气道(支扩/癌) / 肺实质(TB/肺泡出血) / 血管(PE/肺AVM-HHT)
HHT三联 = 自幼鼻出血 + 毛细血管扩张 + 内脏AVM(+家族史)
肺AVM招牌 = 咯血 + 连续性肺杂音 + 光滑无钙化结节 + 缺铁贫血 → 栓塞
HHT vs GPA：尿正常+连续杂音=HHT ; 血尿(肺-肾)=GPA
"Osler"双胞胎别混：HHT=Osler-Weber-Rendu(血管畸形) ; IE=Osler结节(免疫,指尖痛)
```

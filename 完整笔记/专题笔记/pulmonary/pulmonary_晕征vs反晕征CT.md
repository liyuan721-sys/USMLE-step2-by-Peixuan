# 晕征 vs 反晕征 CT 影像征象对照

> [!info] 一句话总览
> 两个长得像"靶环"的肺 CT 征象，方向相反：
> **晕征 (Halo sign) = 中实外虚**（中央实性结节 + 周围磨玻璃环）→ 经典 **侵袭性肺曲霉病** (invasive pulmonary aspergillosis)，粒缺患者。
> **反晕征 (Reverse halo / Atoll sign, 环礁征) = 中虚外实**（中央磨玻璃 + 周围实变环）→ 经典 **隐源性机化性肺炎** (COP)；免疫抑制/DKA 患者则要警惕 **毛霉病** (mucormycosis)。

> [!note]- 本篇缩写速查（首次全称中英）
> - **CT** = computed tomography 计算机断层扫描
> - **GGO** = ground-glass opacity 磨玻璃影
> - **COP** = cryptogenic organizing pneumonia 隐源性机化性肺炎
> - **IPA** = invasive pulmonary aspergillosis 侵袭性肺曲霉病
> - **DKA** = diabetic ketoacidosis 糖尿病酮症酸中毒
> - **GPA** = granulomatosis with polyangiitis 肉芽肿性多血管炎
> - **BAL** = bronchoalveolar lavage 支气管肺泡灌洗
> - **GM** = galactomannan 半乳甘露聚糖（曲霉血清标志物）
> - **CMV** = cytomegalovirus 巨细胞病毒；**VZV** = varicella-zoster virus 水痘-带状疱疹病毒
> - **TB** = tuberculosis 结核；**NTM** = nontuberculous mycobacteria 非结核分枝杆菌
> - **ACE** = angiotensin-converting enzyme 血管紧张素转换酶
> - **c-ANCA** = cytoplasmic anti-neutrophil cytoplasmic antibody 胞浆型抗中性粒细胞胞浆抗体
> - **PE** = pulmonary embolism 肺栓塞；**PCP** = Pneumocystis jirovecii pneumonia 耶氏肺孢子菌肺炎

## 一、两征象方向对照

```
   晕征 Halo sign              反晕征 Reverse halo / Atoll
   （中实外虚）                  （中虚外实）

      ░░░░░░░░                    ●●●●●●●●   ← 外周实变环 (致密)
    ░░      ░░                  ●●        ●●
   ░░  ●●●●  ░░                ●●  ░░░░░░  ●●  ← 中央磨玻璃 GGO (淡)
    ░░  ●●  ░░                  ●●        ●●
      ░░░░░░░░                    ●●●●●●●●

   中央=实性结节 nodule          中央=磨玻璃 GGO
   周围=磨玻璃环 (出血)          周围=实变环 (机化/致密)
   → 侵袭性肺曲霉病              → COP / 毛霉
```

| 维度 | 晕征 Halo sign | 反晕征 Reverse halo sign |
|---|---|---|
| **中央** | 实性结节（致密） | 磨玻璃 GGO（淡） |
| **周围** | 磨玻璃环（出血/水肿） | 实变环（致密） |
| **病理** | 真菌血管侵袭灶 + 周围出血 | 中央肺泡炎 + 外周机化 |
| **最经典关联** | **侵袭性肺曲霉病 (IPA)** | **隐源性机化性肺炎 (COP)** |
| **典型人群** | 中性粒细胞减少 (neutropenic)、化疗后、移植 | 免疫正常、亚急性咳嗽气短 |

> [!tip] 记忆钩子
> - **Halo（晕）= 中实外虚** → "出血包着真菌结节" → **曲霉 Aspergillus**（粒缺 + 咯血）
> - **Reverse（反）= 中虚外实** → 反过来 → 默认 **COP 机化性肺炎**；但放在**免疫抑制 / DKA** 背景里要先想 **毛霉 Mucor**

## 二、晕征 (Halo sign) 鉴别

| 关联病 | 线索 | 备注 |
|---|---|---|
| **侵袭性肺曲霉病 (IPA)** ⭐ | 粒缺/移植 + **发热 + 胸膜痛 + 咯血**三联 | 最经典；血清 **GM + β-D-葡聚糖**、BAL，治 **伏立康唑 voriconazole ± 棘白菌素** |
| 其他血管侵袭性霉菌 | 同样免疫抑制背景 | 毛霉早期也可呈 halo |
| Nocardia 诺卡菌 | 结节 ± 脑脓肿、弱抗酸分支菌 | — |
| 脓毒栓 (septic emboli) | 右心心内膜炎、导管感染 | 多发结节 |
| 病毒性肺炎 (CMV、VZV) | 免疫抑制 | 较少见 |

> [!danger] 晕征核心钥匙
> **结节 + halo 征 + 咯血 + 粒缺 = 侵袭性肺曲霉病**。这是 CK/UW 在"化疗后中性粒细胞减少发热"题里最爱给的影像钩子，不要误选 CMV/PCP。

## 三、反晕征 (Reverse halo) 鉴别 —— 不止 COP

反晕征**不特异**，必须结合宿主免疫状态和病程：

| 关联病 | 线索 | 处理要点 |
|---|---|---|
| **隐源性机化性肺炎 (COP)** ⭐ | 免疫正常；亚急性干咳气短 + **抗生素无效的游走性浸润** | 活检见肺泡管内肉芽组织栓 (Masson bodies)；**糖皮质激素反应好** |
| **毛霉病 (mucormycosis)** ⚠️ | **DKA / 铁过载/去铁胺 / 粒缺**；血管侵袭、鼻-眶-脑 | 活检**宽大无隔菌丝直角分支**；**两性霉素 B amphotericin B + 手术清创** |
| 侵袭性曲霉病 | 免疫抑制 | 反晕也可见，但 halo 更典型 |
| 肺结核 (TB) / NTM | 慢性、午后低热盗汗 | — |
| 结节病 (sarcoidosis) | 双肺门淋巴结、ACE↑ | — |
| 肉芽肿性多血管炎 (GPA) | 上呼吸道 + 肾 + c-ANCA | — |
| 肺梗死 (pulmonary infarction) | PE 病史、胸膜性胸痛 | — |

> [!danger] 反晕征最致命分叉
> 同样是反晕征，宿主决定方向：
> - **免疫正常 + 亚急性 + 激素有效** → **COP**
> - **免疫抑制 / DKA / 粒缺 / 鼻窦受累** → **毛霉**，须活检 + 抗真菌（两性霉素 B）+ 手术
> ⚠️ **别把毛霉当 COP 上激素** —— 激素会加重真菌感染、致命。

## 五、复习题

> [!question]- Q1. 58 岁免疫正常女性，干咳、气短 6 周，无发热，按"社区获得性肺炎"用抗生素无效；CT 见双肺多发病灶，中央磨玻璃、周围实变环。最可能诊断 + 治疗？
> **隐源性机化性肺炎 (COP)**。特征：抗生素无效的游走性浸润 + 反晕征；活检见机化性肺炎 (Masson bodies)；**糖皮质激素反应良好**。

> [!question]- Q2. 急性白血病化疗后中性粒细胞减少，发热 + 胸膜性胸痛 + 咯血，CT 结节伴 halo 征。诊断 + 检查 + 治疗？
> **侵袭性肺曲霉病 (IPA)**。血清半乳甘露聚糖 (GM) + BAL；**伏立康唑 voriconazole ± 棘白菌素（卡泊芬净 caspofungin）**。

> [!question]- Q3. 糖尿病酮症酸中毒患者，肺部见反晕征 + 鼻窦/眼眶受累，活检宽大无隔直角分支菌丝。诊断 + 治疗？为何此处反晕征不能当 COP 处理？
> **毛霉病 (mucormycosis)**；**两性霉素 B amphotericin B + 手术清创**。免疫抑制/DKA 背景下反晕征提示血管侵袭性霉菌，误用糖皮质激素会加重真菌、致命。

> [!question]- Q4. 一句话区分 halo 与 reverse halo 的中央/周围密度，并各给一个最经典关联病。
> Halo = 中央实性结节 + 周围磨玻璃环 → 侵袭性肺曲霉病；Reverse halo = 中央磨玻璃 + 周围实变环 → 隐源性机化性肺炎 (COP)。

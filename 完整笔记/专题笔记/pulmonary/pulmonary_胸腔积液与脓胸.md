# 胸腔积液与脓胸 Pleural Effusion & Empyema 完整体系

> [!info] 一句话总览
> 胸腔积液 (pleural effusion) 的 CK 考点就两条主线：
> **① 定性 —— 渗出液 (exudate) vs 漏出液 (transudate)**（靠 **Light's criteria** 判，决定病因方向）；
> **② 肺炎旁积液 (parapneumonic effusion) 分阶段 —— uncomplicated → complicated → empyema（脓胸）**（靠 **pH / glucose / 培养** 判，决定要不要插胸管引流）。
> 所有题目都在问：**这是渗还是漏？要不要引流？**

> [!note]- 本篇缩写速查（首次全称中英）
> - **LDH** = lactate dehydrogenase 乳酸脱氢酶
> - **CHF** = congestive heart failure 充血性心力衰竭
> - **PE** = pulmonary embolism 肺栓塞
> - **TB** = tuberculosis 结核
> - **RA** = rheumatoid arthritis 类风湿关节炎
> - **SLE** = systemic lupus erythematosus 系统性红斑狼疮
> - **WBC** = white blood cell（count）白细胞（计数）
> - **CXR** = chest X-ray 胸片
> - **CTPA** = CT pulmonary angiography CT 肺动脉造影
> - **CABG** = coronary artery bypass grafting 冠状动脉旁路移植术（冠脉搭桥）
> - **MRSA** = methicillin-resistant *Staphylococcus aureus* 耐甲氧西林金黄色葡萄球菌
> - **VATS** = video-assisted thoracoscopic surgery 电视胸腔镜手术
> - **tPA** = tissue plasminogen activator 组织型纤溶酶原激活剂
> - **DNase** = deoxyribonuclease 脱氧核糖核酸酶
> - **ADA** = adenosine deaminase 腺苷脱氨酶

## 一、临床表现与诊断流程

### 1.1 临床表现 (Presentation)

> [!note] 典型 Stem
> - **胸膜性胸痛 (pleuritic chest pain)** —— 吸气时加重 (worse with inspiration)，**屏气则消失**
> - **患侧体征三联**：呼吸音↓ (decreased breath sounds) + 叩诊浊音 (dullness to percussion) + 触觉语颤↓ (decreased tactile fremitus)
> - 机制：胸腔内的**液体（积液）或气体（气胸）位于肺外**，会**隔绝**来自肺的声音与振动 → 呼吸音与语颤**双双减弱**（区别于实变/肺炎：实变时语颤**增强**）

> [!tip] 语颤方向是高频鉴别钥匙
> | 病变 | 叩诊 | 呼吸音 | 触觉语颤 |
> |---|---|---|---|
> | **胸腔积液 / 气胸** | 浊(积液)/鼓(气胸) | ↓ | **↓**（液/气隔音）|
> | **肺实变 / 肺炎 (consolidation)** | 浊 | 支气管呼吸音 | **↑**（实变传声好）|
> | **肺不张 (atelectasis, 阻塞性)** | 浊 | ↓ | ↓ |

### 1.2 诊断流程 (Diagnosis)

```
胸膜性胸痛 + 患侧呼吸音↓ + 叩浊
        │
        ▼
 ① 初筛影像 = decubitus CXR（侧卧位胸片）
    └─ 看积液是否「自由流动 free-flowing」（>1 cm 厚 → 可穿）
        │
        ▼
 ② 确诊 = diagnostic thoracentesis（诊断性胸腔穿刺）
    └─ 抽液送检 → 套 Light's criteria 定渗/漏
        │
        ▼
 ③ 定性后找病因 + 决定是否引流（见 §四）
```

> [!warning] 何时**不需要**先穿刺
> 明确**双侧、对称、典型 CHF（充血性心衰）** 背景的积液 → 可先**经验性利尿**，治疗有效即可，不必每例都穿。穿刺主要用于**单侧 / 不对称 / 病因不明 / 疑感染**的积液。

## 二、渗出 vs 漏出 —— Light's Criteria（核心）

### 2.1 机制与本质

| 项目 | 正常胸液（基线参考）| Transudate（漏出液） | Exudate（渗出液） |
|:---|:---|:---|:---|
| 本质机制 | 间皮滤过-吸收平衡 | **静水压↑ 或 胶体渗透压↓** | **炎症/感染/肿瘤 → 血管通透性↑** |
| 一句话 | 薄层润滑、几乎无积液 | 水被"挤"出来 | 蛋白和细胞"渗"出来 |
| 量 (volume) | 每侧约 **5–15 mL**（0.1–0.2 mL/kg）| 可大量 | 可大量 |
| 蛋白 protein | **1–2 g/dL** | **<3 g/dL**（胸/血蛋白比 **<0.5**）| **>3 g/dL**（蛋白比 **>0.5**）|
| LDH | **<½ 血清正常上限 (ULN, upper limit of normal)** | 胸/血 LDH 比 **<0.6** 且 <⅔ ULN | 比 **>0.6** 或 **>⅔ ULN** |
| 细胞 (WBC) | **<1000/μL**（单核/间皮为主）| 常 **<1000/μL** | 常 **>1000/μL**（中性↑=化脓；淋巴↑=TB/恶性）|
| 外观 | 清亮淡黄 | 清亮 | **较浑浊**（脓胸=脓性）|
| pH | **7.60–7.64** | **7.40–7.55** | **7.30–7.45**（复杂/脓胸 **<7.20**）|
| 葡萄糖 | **≈ 血浆（60–100 mg/dL）** | ≈ 血糖 | 可 **<60 mg/dL**（感染/肿瘤/TB/RA；RA 常 <30）|
| 累及侧 | 双侧生理性微量 | **全身性 → 多双侧** | **局部病变 → 多单侧** |
| 常见病因 | 生理状态（无病）| CHF心衰、肝硬化、肾病综合征 | 肺炎、恶性肿瘤、TB、PE、胰腺炎、RA |

### 2.2 Light's Criteria（满足任一条 = 渗出液）

> [!danger] 必背 —— Exudate 只要满足 ≥1 条
> 1. **胸液蛋白 / 血清蛋白 > 0.5**
> 2. **胸液 LDH / 血清 LDH > 0.6**
> 3. **胸液 LDH > 正常血清 LDH 上限的 2/3**
>
> **Transudate = 三条全部不满足（all 3 low）。**
> 记忆方向：**渗出 = 高蛋白、高 LDH（≥1 条超标）；漏出 = 全低。**

> [!tip] Memory Hook
> 「**渗(Exudate) = 炎症门户大开 → 蛋白 LDH 都往外漏 → ≥1 条高**；**漏(Transudate) = 单纯压力问题 → 三条全低**。
> 方向千万别记反：CHF 是**漏**不是渗。」

## 三、胸液指标深读（找病因的"指纹"）

### 3.1 pH —— 决定引流的关键数字

```
正常胸液 pH ≈ 7.60
   │
Transudate   7.4 – 7.55  （偏碱）
Exudate      7.30 – 7.45 （偏酸）
   │
pH < 7.20  ← complicated effusion / empyema（脓胸）
            → 必须 chest tube drainage（胸管引流）
```

> [!danger] pH < 7.2 = 引流红线
> 肺炎旁积液 **pH < 7.2** → 已是 **complicated / empyema** → **抗生素 + 胸管引流**，不能只观察。

### 3.2 葡萄糖 Glucose

| 胸液糖 | 比值 | 提示病因 |
|---|---|---|
| **低**（液/血糖比 < 0.5）| 被炎症/恶性/细菌**消耗** | **TB、恶性肿瘤、complicated parapneumonic、类风湿关节炎 (RA)、食管破裂** |
| **高 / 正常**（比值 ≈ 1）| 无明显消耗 | **漏出液、uncomplicated parapneumonic、病毒、PE、post-CABG** |

> [!tip] 极低糖（< 30）联想
> **类风湿关节炎 (RA)** 与 **脓胸 (empyema)** 的胸液糖可低到几乎测不出 —— 见"超低糖"先想这两个 + 恶性/TB。

### 3.3 淀粉酶 Amylase 升高

> [!note] 高淀粉酶胸液 = 两件事
> - **胰腺炎 (pancreatitis)** → 多为**左侧**积液
> - **食管破裂 (esophageal rupture / Boerhaave)** → 唾液淀粉酶 + **pH 极低 + 纵隔气肿**

### 3.4 其它"指纹"外观

| 外观 / 指标 | 指向 |
|---|---|
| **血性 (bloody) 渗出液** | **恶性肿瘤、PE（肺梗死）、外伤、TB** |
| **乳糜样 (milky, 高甘油三酯)** | **乳糜胸 chylothorax**（胸导管损伤 / 淋巴瘤）|
| **脓液 (frank pus)** | **empyema 脓胸** |
| **淋巴细胞为主 + ADA↑** | **结核 TB** |
| 嗜酸性粒细胞↑ | 气胸/血胸后、药物、寄生虫 |

## 四、肺炎旁积液三阶段 (Parapneumonic Effusion) —— 引流决策

> [!info] 主线逻辑
> **肺炎 → 胸膜毛细血管通透性↑ → 胸腔渗出液 (parapneumonic effusion)**
> - 只是无菌渗出 = **uncomplicated**（抗生素够）
> - 细菌跑进胸腔 = **complicated / empyema**（抗生素 **+ 引流**）
> - 一旦"复杂"，就要 **抗生素 + 胸管引流**。

### 4.1 三阶段对照表

| | ① Uncomplicated | ② Complicated | ③ Empyema（脓胸）|
|---|---|---|---|
| **本质** | 无菌性渗出 | 细菌已入胸腔，未成大量脓 | 明显脓液 (frank pus) / 涂片大量菌 |
| **影像** | 小～中量、**自由流动** | 中～大量、可 **loculated（分隔粘连）** | 常 loculated、大量 |
| **pH** | **≥ 7.2** | **< 7.2** | < 7.2（很低）|
| **Glucose** | ≥ 60 | **< 60** | < 60 |
| **LDH** | ≤ 1000 | **> 1000** | > 1000 |
| **WBC** | < 50,000 | > 50,000 | 极高 |
| **革兰染色/培养** | 阴性 | 可阳可阴 | 多阳性 / 肉眼脓 |
| **治疗** | **只需抗生素** | **抗生素 + 胸管引流** | **胸管引流 + 长疗程抗生素 2–4 周；分隔重 → VATS 手术清除** |

```
肺炎 + 积液
   │
   ├─ pH ≥ 7.2、糖 ≥ 60、培养阴、自由流动 → uncomplicated → 抗生素观察
   │
   └─ pH < 7.2 / 糖 < 60 / 培养阳 / loculated / 脓 → complicated/empyema
                                                    → 抗生素 + 引流（胸管 ± 胸腔内 tPA/DNase）
                                                    → 分隔严重/引流不畅 → VATS（胸腔镜清除）
```

> [!tip] 引流方式：胸管 vs VATS —— 看「分隔」不是看「复杂」（Q20107）⭐
> 一旦判为 **complicated**（含**革兰染色阳性单独成立**，不必等 pH<7.2），就要引流；选哪种引流看积液形态：
> - **自由流动 (free-flowing)** → **普通胸管 (chest tube)** 即可，哪怕革兰染色已阳性。
> - **loculated（分隔粘连）/ 纤维化 / 胸管引流不畅** → 才升级 **VATS（胸腔镜清除）** ± 胸腔内 tPA/DNase。
> 陷阱：见"complicated"就反射 VATS——错；**自由流动的 complicated 用胸管就够**，VATS 是给分隔/机化的晚期脓胸。

### 4.2 脓胸经验抗生素（含儿童）

> [!danger] 儿童脓胸经验治疗（高频，详见 ^Q20113）
> - 最常见病原：**肺炎链球菌 Streptococcus pneumoniae + 金黄色葡萄球菌 Staphylococcus aureus（含 MRSA）**
> - 住院/中毒貌经验方案 = **三代头孢（头孢曲松 ceftriaxone 或头孢噻肟 cefotaxime）+ 万古霉素 vancomycin（覆盖 MRSA）**
> - **换三代头孢 ≠ 够** —— 中毒貌患儿**必须补万古覆盖 MRSA**
> - 抗生素之外**必须引流**（胸管 ± 胸腔内纤溶 tPA；高度分隔 → VATS）
> - 厌氧菌（→ 克林霉素 clindamycin 优于甲硝唑 metronidazole）仅在**误吸倾向**（神经肌肉病）才加，儿童少见

### 4.3 儿童肺炎旁积液：按"大小 + 窘迫/缺氧"分流（Q12421）⭐

> [!info] 儿童管理流程 —— 不是每例都引流
> ```
> 儿童肺炎 + 胸片积液
>    │
>    ├─ 小量 且 无呼吸窘迫/缺氧 → 口服抗生素 (oral antibiotics) + 密切门诊随访
>    │
>    └─ 中~大量 或 呼吸窘迫 或 缺氧 → 超声 + IV 抗生素 + 引流（胸管 ± VATS）
> ```
> - **小量 + 一般情况好（呼吸舒适、SpO₂ 正常）= 口服抗生素门诊**，不必影像/引流——高频考点。
> - **CT** 仅用于评估罕见并发症（坏死性肺炎/脓肿），非常规。
> - **胸部理疗 (chest physiotherapy)** 用于**囊性纤维化 (CF) / 支气管扩张**排痰，对肺炎/积液**无效**，别误选。
> - 与成人逻辑一致：自由流动可口服/胸管，**革兰阳/复杂/大量/窘迫**才升级引流。

## 五、病因速查（按渗/漏归类）

> [!note] Exudate（渗出）常见病因 = 炎症/感染/肿瘤/自免
> - 感染：**肺炎 (TB、细菌)**
> - **恶性肿瘤 (malignancy)**
> - **风湿病 (RA、SLE)**
> - **肺栓塞 (PE)** —— 注意 PE 是渗出！（详见 ^Q4613）
> - 胰腺炎 (pancreatitis)
> - **Post-CABG**（冠脉搭桥术后，左侧）

> [!note] Transudate（漏出）常见病因 = 压力/容量
> - **心力衰竭 (heart failure)** —— 最常见，多双侧
> - **肝硬化 (cirrhosis → hepatic hydrothorax 肝性胸水)**
> - **肾病综合征 (nephrotic syndrome)**
> - 缩窄性心包炎 (constrictive pericarditis)

> [!warning] CK 高频"陷阱归类"
> - **PE → 渗出（exudate），常血性、小量、伴胸膜痛** —— 别因"无浸润影"误判肺炎；制动 + 突发 = PE（^Q4613）
> - **CHF → 漏出（transudate），pH 7.4–7.55、蛋白/LDH 低** —— 新发未利尿别选"渗出"（^Q4053）
> - **肝顶脓肿/膈下感染 → 右侧反应性（无菌）胸腔积液**，破入胸膜 → 脓胸（^Q17223）

## 六、假渗出 Pseudoexudate（易混陷阱）

> [!warning] 利尿后的"假渗出"
> **CHF 经大量利尿后**，约 **25%** 原本的漏出液会"浓缩"（蛋白/LDH 升高）→ **满足 Light 标准被误判为渗出**。
> - 判读 Light 时要结合"**是否已大量利尿**"
> - 鉴别工具：测**血清-胸液白蛋白梯度 (serum-effusion albumin gradient) > 1.2 g/dL** → 仍是漏出（提示 CHF）
> - **新发、未治疗的 CHF** → 是纯漏出液，不会假渗出

## 七、USMLE 秒选决策树 + 高频陷阱

```
胸腔积液题
   │
   ├─ 问「下一步诊断」？
   │     初筛 → decubitus CXR；确诊 → diagnostic thoracentesis
   │
   ├─ 问「渗还是漏」？ → 套 Light's（≥1 高 = 渗；全低 = 漏）
   │
   ├─ 问「肺炎 + 积液要不要插管引流」？
   │     pH < 7.2 / 糖 < 60 / 培养阳 / loculated / 脓 → 引流
   │     否则 → 只抗生素
   │
   └─ 问「积液病因」？ → 看指纹：
         血性→恶性/PE；乳糜→胸导管；高淀粉酶→胰腺炎/食管破裂；
         超低糖→RA/脓胸/TB/恶性；左侧→胰腺炎/post-CABG
```

> [!danger] 高频陷阱清单
> 1. **呼吸音↓ + 叩浊 ≠ 一定是肺炎** —— 看语颤（积液↓ vs 实变↑）+ 看有无浸润影
> 2. **PE 积液是渗出（exudate）不是漏出** —— 制动/突发/无浸润影优先想 PE，CTPA 确诊
> 3. **CHF 是漏出（transudate）** —— pH 偏碱、蛋白/LDH 低；Light 方向别记反
> 4. **pH < 7.2 = 引流红线** —— 别只给抗生素观察
> 5. **儿童脓胸换三代头孢还不够** —— 中毒貌必须加万古覆盖 MRSA + 引流
> 6. **利尿后假渗出** —— 别把利尿后 CHF 积液当成真渗出去查恶性/感染

## 八、复习自测题

> [!question]- Q1. 一肺炎患者胸腔积液诊断性穿刺：pH 7.10、糖 45 mg/dL、LDH 1200、革兰染色见菌、影像 loculated。下一步最佳处理？
> **抗生素 + 胸管引流 (chest tube drainage)**。这是 complicated parapneumonic effusion / 早期 empyema（pH < 7.2 + 糖 < 60 + LDH > 1000 + 培养阳 + 分隔），单纯抗生素不够，必须引流；分隔重/引流不畅再上 VATS。

> [!question]- Q2. 37 岁轮椅截瘫女性，突发左侧胸痛气短 2 天，小量左侧积液、**无浸润影**、WBC 正常。积液最可能病因？
> **肺栓塞 (PE)**。长期制动 + 突发 + 无浸润影 = PE 钥匙；PE 积液 = **小量、渗出 (exudate)、常血性、伴胸膜痛**，可无肺梗死。确诊 **CTPA**。别被"肺底叩浊"误导成吸入性肺炎（后者病程隐袭 + 有浸润影）。

> [!question]- Q3. 65 岁男，慢性心衰背景，进行性气短 2 月，右侧积液穿刺。下列哪项最符合？(A) 糖 < 60 (B) 高淀粉酶 (C) LDH 比值 > 0.6 (D) pH 7.45 (E) 蛋白比值 > 0.5
> **(D) pH 7.45**。CHF = **漏出液 (transudate)** → pH 偏碱 7.4–7.55、蛋白/LDH 比值低、糖正常。A/C/E 都指向渗出，B 指向胰腺炎/食管破裂，均不符。

> [!question]- Q4. 3 岁脓胸患儿，已用阿莫西林 amoxicillin 4 天无改善、中毒貌、loculated 积液，已换头孢曲松 ceftriaxone，还需加什么？
> **万古霉素 (vancomycin)** 覆盖 MRSA。儿童脓胸 = 肺炎链球菌 + 金葡（含 MRSA）；三代头孢不覆盖 MRSA，中毒貌必须加万古；且脓胸**必须引流**。

> [!question]- Q5. 胸腔积液患者，呼吸音↓ + 叩浊。如何用一个床旁体征区分"积液"与"肺实变/肺炎"？
> **触觉语颤 (tactile fremitus)**：**积液 → 语颤↓**（液体隔音）；**实变/肺炎 → 语颤↑**（实变传声增强）。叩诊两者都浊，靠语颤方向 + 有无浸润影区分。

# 低氧血症五机制（Five Mechanisms of Hypoxemia）

> [!info] 一句话定位
> Step 2 CK 给一个低氧 stem，本质在问"**哪一种机制**"。五大机制靠**两个判别器**一秒分流：① **A-a 梯度（肺泡-动脉氧分压差 alveolar-arterial gradient）正常 vs 增大** ② **吸 100% 氧能否纠正**。本卡把五机制压成一张表，并落到三道错题：肺炎=**分流**（[[mistakes/uworld-mistakes_2026-06#^Q4770]]）、右主支气管插管=**V/Q 失配**（[[mistakes/uworld-mistakes_2026-06#^Q4632]]）、溺水=**V/Q 失配/分流**（[[mistakes/uworld-mistakes_2026-06#^Q20184]]）。
> 素材整合自 [[完整笔记/Peixuan分科笔记/pulmonary]]（只读）。

> [!note]- 本篇缩写速查（首次全称中英）
> - **A-a 梯度** = alveolar-arterial oxygen gradient 肺泡-动脉氧分压差
> - **V/Q** = ventilation/perfusion ratio 通气/血流比
> - **PiO₂** = partial pressure of inspired oxygen 吸入氧分压
> - **PaO₂** = partial pressure of arterial oxygen 动脉血氧分压
> - **PAO₂** = partial pressure of alveolar oxygen 肺泡氧分压
> - **PaCO₂** = partial pressure of arterial carbon dioxide 动脉血二氧化碳分压
> - **HPV** = hypoxic pulmonary vasoconstriction 缺氧性肺血管收缩
> - **ETT** = endotracheal tube 气管导管
> - **PE** = pulmonary embolism 肺栓塞
> - **ILD** = interstitial lung disease 间质性肺病
> - **IPF** = idiopathic pulmonary fibrosis 特发性肺纤维化
> - **PAH** = pulmonary arterial hypertension 肺动脉高压
> - **ARDS** = acute respiratory distress syndrome 急性呼吸窘迫综合征
> - **COPD** = chronic obstructive pulmonary disease 慢性阻塞性肺病
> - **PEEP** = positive end-expiratory pressure 呼气末正压
> - **NPPV / CPAP** = noninvasive positive pressure ventilation 无创正压通气 / continuous positive airway pressure 持续气道正压

## 一、五大机制总表 ⭐⭐⭐

| 机制 | 本质 | A-a 梯度 | 吸 100% O₂ 能纠正? | 典型病 |
|---|---|---|---|---|
| **① 吸入氧分压低 (low PiO₂)** | 大气氧不够 | **正常** | 能 | **高海拔**、密闭缺氧环境 |
| **② 肺泡低通气 (hypoventilation)** | 通气泵衰（PaCO₂↑）| **正常** | 能 | 阿片/镇静过量、神经肌肉病、CNS 抑制 |
| **③ 弥散障碍 (diffusion limitation)** | 肺泡-毛细血管膜增厚 | **增大** | 能（多数）| **ILD/IPF**、肺气肿、PAH |
| **④ V/Q 失配 (V/Q mismatch)** | 通气与灌注不匹配（最常见）| **增大** | **能** | PE、肺炎早期、哮喘/COPD、**单侧肺病/插管位置异常** |
| **⑤ 分流 (shunt, 右→左)** | 血经过完全不通气的肺泡（V=0）| **增大** | **不能** ⭐ | **肺炎实变**、ARDS、肺水肿、肺不张、心内右→左分流 |

> [!tip] 两个判别器（一秒定机制）⭐
> 1. **A-a 梯度**：① 低 PiO₂ + ② 低通气 = **A-a 正常**（肺本身没病，是"进气少/吸气少"）；③④⑤ = **A-a 增大**（肺内气体交换出问题）。
> 2. **吸 100% O₂**：只有 **⑤ 分流不被纠正**（血根本不接触肺泡气）——其余四种都能被高浓度氧拉上来。**"给纯氧仍低氧 = 分流"**是最锋利的一刀。

---

## 二、分流 vs 死腔（最高频混淆）⭐⭐⭐

> [!danger] 一句话钥匙
> **分流 = 有血没气（V/Q=0）；死腔 = 有气没血（V/Q=∞）。**

| | **分流 shunt** | **死腔 dead space** |
|---|---|---|
| V/Q | **0**（灌注但不通气）| **∞**（通气但不灌注）|
| 一句话 | **有血流、无通气** | **有通气、无血流** |
| 经典病 | **肺炎实变 / 肺不张 / ARDS / 肺水肿 / 心内右→左** | **肺栓塞 (PE) / 过度 PEEP** |
| 主要后果 | **低氧**（吸纯氧不纠正）| 主要 **CO₂ 排出↓ / 通气浪费**（低氧相对轻）|
| 吸 100% O₂ | **不纠正** | （死腔本身不是低氧主因）|

> [!warning] 做题反射
> 题目问"肺炎为何低氧" → **肺内分流 (intrapulmonary shunting)**，实变肺泡不通气但仍灌注。**别选死腔通气**（死腔是 PE 的语言：有气没血）。这正是 [[mistakes/uworld-mistakes_2026-06#^Q4770]] 的陷阱。

## 三、体位与单侧肺病：good lung down ⭐

> [!info] 重力调节灌注 → 体位影响氧合
> 灌注随重力走（下垂区血多）。**单侧肺病时：**
> - **坏肺在下 (bad lung down)** → 重力把更多血灌向**不通气的病肺** → V/Q 失配/分流**加重** → **低氧加重**。
> - **坏肺在上 / good lung down（健肺在下）** → 血流重分布到**通气良好的健肺** → V/Q 改善 → **氧合改善**。

> [!tip] 临床推论
> 单侧肺实变/大量胸腔积液/单肺移植患者，**倾向健肺在下卧位**氧合更好。题干"左肺肺炎，左侧卧时 SpO₂ 下降" = 坏肺在下加重分流（[[mistakes/uworld-mistakes_2026-06#^Q4770]]）。

## 四、临床实例对照（三道错题落点）⭐

| 病例 | 机制 | 钥匙 | 处理 |
|---|---|---|---|
| **肺炎低氧**（[[mistakes/uworld-mistakes_2026-06#^Q4770]]）| **肺内分流** | 实变肺泡 V=0 仍灌注；坏肺在下加重 | 抗感染 + good lung down + 氧（重症需 PEEP）|
| **右主支气管插管**（[[mistakes/uworld-mistakes_2026-06#^Q4632]]）| **V/Q 失配** | ETT 过深入右主支气管 → 左肺不通气 + 单侧呼吸音↓ | **后退导管 (pull back ETT)** 至隆突上 2-5 cm |
| **溺水**（[[mistakes/uworld-mistakes_2026-06#^Q20184]]）| **V/Q 失配 / 分流** | 误吸 + 表面活性物质冲失 → 肺泡塌陷/水肿 | 纠正低氧（高流量氧 / CPAP / NPPV）|

> [!warning] 缺氧性肺血管收缩 (HPV) 是干扰项
> **HPV** = 肺泡缺氧时局部小动脉收缩，把血流**导离**通气差的区域 → 是身体**代偿** V/Q 失配的机制，**由肺泡氧张力决定、不受体位影响**。题目把它列出来时是干扰项（如 [[mistakes/uworld-mistakes_2026-06#^Q4770]] 选项 D）——它不是低氧的"原因"。

## 五、显著低氧 / 运动性降氧 → 疾病鉴别 ⭐⭐

> [!info] 为什么单列这一节
> PCP 的招牌不是"低氧"本身，而是**运动后血氧骤降（exertional desaturation）+ 低氧与影像/查体不成比例**。这两个短语各自有一串 DDx——拆开记，才不会一见"显著低氧"就只反射 PCP。机制根在前四节（弥散障碍 / 分流 / V-Q 失配）。

### A. 运动性去饱和（exertional desaturation：静息 SpO₂ 可正常，一动就掉）

> [!tip] 机制 = 弥散障碍 或 肺血管病
> 运动时心输出量↑、红细胞通过肺毛细血管加速；若**肺泡-毛细血管膜增厚（弥散障碍）**或**肺血管床固定/减少（肺血管病）**，红细胞来不及氧合 → 一动就去饱和。静息有弥散储备故可正常 → **静息血氧正常不能排这些病**。

| 病 | 机制 | 钥匙 |
|---|---|---|
| **ILD / IPF（特发性肺纤维化）** | 弥散障碍 | 干咳 + Velcro 底啰音 + ↓DLCO；最经典劳力去饱和 |
| **PCP（耶氏肺孢子菌肺炎）** | 弥散障碍 + 早期肺泡渗出 | HIV/CD4<200 + LDH↑ + 双肺弥漫磨玻璃 |
| **肺动脉高压 PAH / 慢性血栓栓塞性肺高压 CTEPH** | 肺血管床固定 | 孤立 DLCO↓、响 P2、右心负荷 |
| **肺气肿 emphysema（COPD）** | 肺泡膜面积丢失 | 吸烟、↑TLC、↓DLCO |
| **肝肺综合征 hepatopulmonary syndrome** | 肺内血管扩张性分流 | 肝硬化 + **直立位加重（platypnea-orthodeoxia 直立性低氧）** |

### B. 低氧与影像/查体"不成比例"（胸片相对干净，血氧却很低）

> [!tip] 机制 = 分流 或 严重 V/Q 失配
> 胸片看着没那么糟、低氧却很重 → 想**分流（吸纯氧不纠正）**或严重 V/Q 失配。

| 病 | 机制 | 钥匙 |
|---|---|---|
| **肺栓塞 PE** | 死腔 + 反射性 V/Q 失配 | 急性、胸片常正常、A-a↑、危险因素 |
| **PCP** | 弥散障碍 + 渗出 | 早期胸片轻、血氧已重 + HIV 风险 |
| **右→左分流** | 血绕过通气肺泡 | 心内分流（PFO/ASD→Eisenmenger）、**肺动静脉畸形（pulmonary AVM，见 HHT 遗传性出血性毛细血管扩张症）**、肝肺综合征；**吸纯氧不纠正** |
| **早期 ARDS / 弥漫肺泡出血 DAH** | 分流 | 急性呼衰；DAH = 血红蛋白降 + 咯血 ± 磨玻璃 |

### C. SpO₂ 读数"假性低/不可靠"——别误当肺病（pulse oximetry 脉氧仪陷阱）

> [!warning] 这些 PaO₂ 可正常，是血红蛋白/读数问题，不是气体交换障碍
> | 情况 | 表现 | 处理 |
> |---|---|---|
> | **高铁血红蛋白血症 methemoglobinemia** | SpO₂ 卡在 ~85% 不随给氧升、**saturation gap（脉氧与动脉血气氧饱和差）**、巧克力色血；达帕松 dapsone / 苯佐卡因 benzocaine / 硝酸盐诱因 | **亚甲蓝 methylene blue** |
> | **一氧化碳中毒 CO poisoning** | SpO₂ **假性正常**（测不出 COHb 碳氧血红蛋白）、头痛/同室多人 | co-oximetry 测 COHb + 100% O₂ |

> [!danger] 回到 PCP（Q128）
> PCP 的特异性 = **A（运动性去饱和）+ B（低氧不成比例）同时具备**，再加 **HIV 风险 + LDH↑**。但**只给"运动性降氧"** → 必须并列 ILD/PAH；**只给"低氧不成比例"** → 必须并列 PE / 右→左分流。别让单一短语锚定 PCP。

## 六、Memory Hook

```
五机制 = 低PiO₂ / 低通气 / 弥散障碍 / V-Q失配 / 分流
两把刀：A-a梯度(低PiO₂+低通气=正常；其余增大) + 吸100%O₂(只有分流不纠正)
分流=有血没气(肺炎实变,纯氧不纠正) ; 死腔=有气没血(PE)
单侧肺病：good lung down 氧合好(坏肺在下加重分流)
肺炎低氧=分流 / 右主插管=V-Q失配→后退 / 溺水=V-Q/分流
HPV=代偿(导血离开坏区,不受体位影响)=干扰项
```

# 结节病 Sarcoidosis

> [!info] 一句话定位
> **Sarcoidosis (结节病) = 病因不明的多系统肉芽肿病**，特征 **非干酪样肉芽肿 (noncaseating granuloma)**。好发**非裔年轻女性 (20–40 岁)**。考点核心 = **双侧肺门淋巴结肿大 + 高钙血症机制 + 排除性诊断 + 激素**。

---

## 一、流行病学 + 病理

- **人群**：**非裔 (African American) 女性**风险倍增，20–40 岁高发。〔标准补充〕
- **病理**：**非干酪样肉芽肿 noncaseating granuloma**（与结核 TB 的**干酪样 caseating** 相对；铍中毒 berylliosis 也是非干酪样，靠职业史区分）。（来源：分科 [[完整笔记/Peixuan分科笔记/pulmonary]]）

---

## 二、肺（最常受累）

```
胸片分期 (CXR staging)
I   = 仅双侧肺门淋巴结肿大 (BHL, bilateral hilar lymphadenopathy)
II  = BHL + 肺实质浸润
III = 仅肺实质浸润（无 BHL）
IV  = 肺纤维化 fibrosis（不可逆）
```

- **标志 = 双侧肺门淋巴结肿大 (BHL)**；肺功能呈**限制性 (restrictive)**。
- **触发词**："hilar lymphadenopathy + 结节红斑 erythema nodosum" → 结节病。（来源：分科 pulmonary）

---

## 三、肺外多系统

| 系统 | 表现 | 钩子 |
|---|---|---|
| **皮肤** | **结节红斑 erythema nodosum**（早期、预后好）/ **冻疮样狼疮 lupus pernio**（鼻面紫红斑块，晚期、预后差） | EN 好 / lupus pernio 坏 |
| **眼** | **肉芽肿性葡萄膜炎 (granulomatous uveitis)** | 红眼 + 视物模糊 |
| **钙** | **高钙血症 / 高尿钙**（见 §四）| 肉芽肿活化维 D |
| **心** | **传导阻滞 / 室性心律失常 / 猝死**（cardiac sarcoidosis，少见但致命）| 不明原因房室阻滞的年轻人 |
| **神经** | 神经结节病：颅神经麻痹（**面神经最常见**）/ 下丘脑垂体受累 | （来源：分科 pulmonary 提下丘脑浸润）|

> [!tip] Löfgren 综合征 (Löfgren syndrome)〔标准补充〕
> **急性三联：双侧肺门淋巴结肿大 (BHL) + 结节红斑 + 踝关节炎/关节痛**（± 发热）。预后**最好（多自限）**，常见于年轻女性 → 多数只需 NSAIDs 观察，不急于激素。

---

## 四、高钙血症机制（分科重点，高频）

> [!danger] 肉芽肿巨噬细胞自主活化维 D
> ```
> 肉芽肿巨噬细胞表达 1α-羟化酶 (1α-hydroxylase)
>         ↓ 不受 PTH 调控（自主）
> 25-(OH)D → 1,25-(OH)₂D 活性维 D ↑↑
>         ↓
> 肠钙吸收↑ → 高钙血症 + 高尿钙（→ 肾结石/肾钙化）
> ```
> 与恶性肿瘤的 **PTHrP 机制**不同；**PTH 受抑制（低）**。（来源：分科 [[完整笔记/Peixuan分科笔记/肾脏]]）

> [!tip] 实验室 + 鉴别
> - **血 ACE (angiotensin-converting enzyme, 血管紧张素转换酶)↑**（敏感性中等、特异性差，多用于监测而非确诊）〔标准补充〕。
> - vs **原发性甲旁亢 PHPT**：PHPT = PTH↑ + 尿钙高；结节病 = PTH↓ + 1,25-D↑。
> - 治疗高钙：补液 + **糖皮质激素（抑制 1α-羟化酶）**（来源：分科 肾脏）。

---

## 五、诊断 + 治疗

> [!warning] 排除性诊断 (diagnosis of exclusion)
> 影像（CXR）+ **活检见非干酪样肉芽肿** + **排除 TB / 真菌 / 恶性 / 铍中毒**。〔标准补充：排除思路〕

> [!success] 治疗
> - **多数无症状 / Löfgren → 观察**（常自限）。
> - **有症状（肺/眼/心/神经/高钙）→ 一线糖皮质激素 prednisone 泼尼松**。
> - 激素依赖/不耐受 → **MTX 甲氨蝶呤 / 硫唑嘌呤 azathioprine / 霉酚酸酯 mycophenolate**；难治 → **TNF-α 抑制剂 infliximab 英夫利昔**。〔标准补充〕

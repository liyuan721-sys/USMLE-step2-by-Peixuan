# 胸部 CT 高频征象速查（Chest CT Signs）

> [!info] 一句话总览
> CK 影像题常给一个"形象命名的 CT 征象"当钩子 → 反射对应病。本卡是**征象 → 病 → 一句话钥匙**的总索引；其中 **晕征 / 反晕征**展开见子卡 [[完整笔记/专题笔记/pulmonary/pulmonary_晕征vs反晕征CT]]。
> 核心心法：**征象不特异，必须叠加宿主（免疫状态）+ 病程 + 伴随表现**才能锁定诊断。

> [!note]- 本篇缩写速查（首次全称中英）
> - **CT** = computed tomography 计算机断层扫描
> - **GGO** = ground-glass opacity 磨玻璃影
> - **COP** = cryptogenic organizing pneumonia 隐源性机化性肺炎
> - **IPA** = invasive pulmonary aspergillosis 侵袭性肺曲霉病
> - **ABPA** = allergic bronchopulmonary aspergillosis 变应性支气管肺曲霉病
> - **PAP** = pulmonary alveolar proteinosis 肺泡蛋白沉积症
> - **PCP** = Pneumocystis jirovecii pneumonia 耶氏肺孢子菌肺炎
> - **ARDS** = acute respiratory distress syndrome 急性呼吸窘迫综合征
> - **TB** = tuberculosis 结核；**NTM** = nontuberculous mycobacteria 非结核分枝杆菌
> - **CF** = cystic fibrosis 囊性纤维化
> - **GPA** = granulomatosis with polyangiitis 肉芽肿性多血管炎
> - **DPB** = diffuse panbronchiolitis 弥漫性泛细支气管炎
> - **DKA** = diabetic ketoacidosis 糖尿病酮症酸中毒
> - **IgE** = immunoglobulin E 免疫球蛋白 E
> - **CMV** = cytomegalovirus 巨细胞病毒
> - **HIV** = human immunodeficiency virus 人类免疫缺陷病毒

## 一、总速查表（征象 → 病 → 钥匙）

| CT 征象 | 形态 | 最经典关联病 | 一句话钥匙 |
|---|---|---|---|
| **晕征 Halo sign** | 实性结节 + 周围磨玻璃环 | **侵袭性肺曲霉病 (IPA)** | 粒缺 + 咯血 + halo → 曲霉（详见子卡） |
| **反晕征 Reverse halo / Atoll** | 中央磨玻璃 + 周围实变环 | **隐源性机化性肺炎 (COP)** | 免疫正常 + 抗生素无效 → COP；免疫抑制/DKA → 毛霉 (mucormycosis)（详见子卡） |
| **空气新月征 Air-crescent sign** | 结节/团块周围新月形气体 | **曲霉球 (aspergilloma)**；IPA 恢复期 | 粒缺恢复、中性粒回升时出现 → 预后好转信号 |
| **树芽征 Tree-in-bud** | 小叶中心结节 + 分支线（像发芽树枝） | **感染性细支气管炎**：TB 支气管内播散、NTM、误吸、DPB | 见树芽先想"小气道感染/播散"，TB 沿支气管扩散经典 |
| **铺路石征 Crazy-paving** | 磨玻璃 + 增厚小叶间隔（碎石路面） | **肺泡蛋白沉积症 (PAP)**；也见 PCP、肺泡出血、ARDS、肺水肿 | 经典 PAP，但需结合临床（PCP/水肿同样可现） |
| **印戒征 Signet-ring sign** | 扩张支气管(环) + 伴行小动脉(戒面宝石) | **支气管扩张 (bronchiectasis)** | 支气管径 > 伴行动脉 = 扩张 |
| **指套征 Finger-in-glove** | 黏液栓充填扩张支气管呈指套状 | **变应性支气管肺曲霉病 (ABPA)** | 哮喘/CF + 中心性支扩 + 黏液栓 + IgE↑ + 曲霉致敏 |
| **星系征 Galaxy sign** | 大结节周围卫星小结节簇 | **结节病 (sarcoidosis)** | 双肺门淋巴结 + 上叶为主 + 卫星结节 |
| **彗星尾征 Comet-tail sign** | 血管/支气管向胸膜下肿块卷入 | **圆形肺不张 (rounded atelectasis)** | 石棉接触 + 胸膜增厚/积液病史 |
| **滋养血管征 Feeding-vessel sign** | 血管直插结节 | **血源播散（脓毒栓 septic emboli、转移）** | 多发外周结节 + 右心心内膜炎/导管 → 脓毒栓 |

## 二、易混征象两两对照

> [!warning] 晕征 vs 反晕征（方向相反）
> - **晕征 = 中实外虚**（结节 + 磨玻璃环）→ 侵袭性肺曲霉病
> - **反晕征 = 中虚外实**（磨玻璃 + 实变环）→ COP / 毛霉
> - 完整鉴别谱见子卡 [[完整笔记/专题笔记/pulmonary/pulmonary_晕征vs反晕征CT]]

> [!warning] 空气新月征 vs 晕征（侵袭性肺曲霉病的两个时相）
> - **早期粒缺期 → 晕征 (halo)**：血管侵袭 + 周围出血
> - **恢复粒回升期 → 空气新月征 (air-crescent)**：坏死组织回缩、周围出现气体
> - 同一个病的"前后两张脸"，air-crescent 出现常提示预后转好。

> [!warning] 树芽征 vs 铺路石征（小气道 vs 肺泡）
> - **树芽征**：累及**小气道**（细支气管），多为感染播散（TB/NTM/误吸）
> - **铺路石征**：累及**肺泡 + 间隔**，经典 PAP，也见 PCP / 肺水肿 / 肺泡出血

> [!warning] 指套征 (ABPA) vs 印戒征 (单纯支扩)
> 两者都基于支气管扩张：**印戒征**是扩张支气管的横断面（环 + 伴行动脉）；**指套征**是扩张支气管被黏液栓充填（ABPA 的标志），叠加哮喘/CF + 总 IgE↑ + 曲霉致敏。

## 三、按"宿主背景"反查（CK 出题视角）

```
免疫抑制 / 粒缺
 ├─ 结节 + halo + 咯血 ──── 侵袭性肺曲霉病 (IPA)
 ├─ 反晕 + DKA/鼻眶脑 ──── 毛霉病 (mucormycosis)
 └─ 弥漫磨玻璃 + 铺路石 ── PCP（HIV/激素）

免疫正常
 ├─ 反晕 + 抗生素无效游走浸润 ── COP（激素有效）
 ├─ 树芽 + 慢性咳嗽/盗汗 ────── TB / NTM 支气管播散
 ├─ 星系征 + 双肺门淋巴结 ───── 结节病 (sarcoidosis)
 └─ 指套征 + 哮喘 + IgE↑ ────── ABPA

特殊接触/病史
 ├─ 彗星尾 + 石棉接触 ──────── 圆形肺不张
 └─ 滋养血管 + 多发外周结节 ── 脓毒栓 / 血源转移
```

## 四、复习题

> [!question]- Q1. 哮喘控制不佳患者，反复咳棕色黏液栓，CT 见中心性支气管扩张 + 指套状黏液充填，总 IgE 显著升高、曲霉特异 IgE 阳性。诊断？
> **变应性支气管肺曲霉病 (ABPA)**。指套征 (finger-in-glove) + 中心性支扩 + IgE↑ + 曲霉致敏是经典组合；治疗糖皮质激素 ± 伊曲康唑 (itraconazole)。

> [!question]- Q2. 慢性咳嗽、午后低热盗汗，CT 见小叶中心分支状"树芽"结节，集中于一肺叶。最需排除什么？
> **肺结核 (TB) 支气管内播散**（也见 NTM、误吸性细支气管炎）。树芽征 = 小气道感染/播散标志，TB 沿支气管扩散最经典。

> [!question]- Q3. 一句话区分"晕征"与"空气新月征"在侵袭性肺曲霉病中的时相意义。
> 晕征 = 早期粒缺、血管侵袭出血；空气新月征 = 恢复期粒细胞回升、坏死回缩出现气体，常提示预后转好。

> [!question]- Q4. CT 见弥漫磨玻璃叠加增厚小叶间隔（铺路石征），鉴别诊断至少列 3 个。
> 肺泡蛋白沉积症 (PAP，最经典) / 耶氏肺孢子菌肺炎 (PCP) / 心源性肺水肿 / 弥漫性肺泡出血 / ARDS —— 必须结合临床，征象本身不特异。

# 📥 图片待归位收件箱

> [!info] 怎么用
> 1. 把截图直接 **Ctrl+V 粘进下面"待归位区"**（Obsidian 会自动存图 + 生成 `![[Pasted image...]]`）。
> 2. 每张图下面那行 `>` 注释**可写可不写**——写了我归位更准；不写我读图内容自己认（UW 截图右上角有 Question Id）。
> 3. 粘完跟我说一句**"图贴了"**，我就扫描 + 归位。

> [!tip] 我归位时会做什么
> - 逐张读图识别（哪道题 / 什么主题 / 图类型）
> - 有对应错题卡 → 嵌进卡（callout 内 `> ![[名]]` 带前缀）；**无卡 → 找笔记里相应知识点贴上（不新建卡）**；一图两用 → 双地点同一 wikilink
> - 报告清单：每张图去了哪
> - 归位后清空"待归位区"（图本身留在 attachments），并在下方日志登记

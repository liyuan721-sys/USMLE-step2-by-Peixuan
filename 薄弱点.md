## 🟡 有点印象但不稳

- [2026-05-16] **抗凝逆转剂完整性 + 反向题陷阱** — 自测 STQ20260516-05 — 关联笔记 [[完整笔记/专题笔记/心内/心内_抗凝抗板_急性血栓]]
  - 漏看"INCORRECT"题干，选了正确配对（D）作答
  - LMWH-protamine 只逆转 ~60% anti-Xa（不完全；UFH 才是 100%）
  - SOP：反向题先圈"INCORRECT / NOT / EXCEPT"

- [2026-05-18] **IE + 新发 AV block → TEE 不过夜（"stable" 假象）** — 自测 Q12 — 关联笔记 [[完整笔记/专题笔记/感染/感染_感染性心内膜炎完整体系]] §5.1
  - 错选 "继续抗生素 + 观察" — 被 "hemodynamically stable" 吸住
  - 救命反射：**IE + 任何新传导异常 → TEE，不分稳定与否**（脓肿可在数小时内 1° → CHB → 心源性休克）
  - HEROIC **E**xtension = 瓣周脓肿 / 新发 AV block → 立即外科会诊

- [2026-05-18] **IE 预防 2007 后指南 — D+R only** — 自测 Q13 — 关联笔记 [[完整笔记/专题笔记/感染/感染_感染性心内膜炎完整体系]] §12.4
  - 错选 "IV ampicillin + gentamicin"（2007 前 GI/GU 老方案，已废除 18+ 年）
  - 救命反射：**"D + R only"** — IE 预防仅 **D**ental + **R**espiratory invasive，**GI/GU since 2007 = OUT**
  - 机械瓣 + 操作 ≠ 自动预防 — 必须同时满足高风险心脏 + 高风险操作

- [2026-05-18] **Janeway vs Osler 方向反向陷阱** — 自测 Q14 — 关联笔记 [[完整笔记/专题笔记/感染/感染_感染性心内膜炎完整体系]] §四
  - 字母-性质映射没咬死 — 5/17 笔记原话写过仍能选反
  - 救命口诀：**两个 J / 两个 O**
    - **J**aneway = **J**ust spots（平 / 手掌足底 / 无痛 / **V**ascular 栓塞）
    - **O**sler = **O**uch!（鼓 / 指趾 pad / 痛 / **I**mmunologic 免疫复合物）
  - 3 秒判断：Painful + finger pad → Osler→Immune；Painless + palm/sole → Janeway→Vascular

- [2026-05-18] **高钾元凶定位 — Trigger vs Permissive** — 自测 Q19 — 关联笔记 [[草稿/2026-05-17_高钾血症完整体系_v1]] §Trigger vs Permissive
  - 错选 "CKD3 单独" — 被慢性病吸走注意力
  - 救命反射：**"What changed?"** 高钾题第一问看时间线，不看 K 值大小
  - 三联 K 炸弹：**A**CEi/ARB + **M**RA + **K** supplement = 必爆
  - 隐藏升 K 药 **NTSH**：NSAIDs / TMP-SMX / Spiro / Heparin
  - **"CKD 是雷区，KCl 是按钮"** — 雷区一直在，按按钮才爆

- [2026-06-14] **PFPS 髌股疼痛综合征（治疗 + 体征）** — UW 错题 #Q12060 + #Q8886（同日 2 次错）— 关联笔记 [[完整笔记/专题笔记/骨科/骨科_HighYield速查总览]] §三
  - ⚠️ 同日连错 2 道（治疗题 Q12060 选膝固定器 / 体征题 Q8886 选"不能伸膝"）；错题卡已升 🔴 + #难记；按本表规则**再错 1 次升 🔴**
  - 救命反射：**PFPS = 练股四头肌 (quadriceps strengthening)，忌制动**；体征 = **绷股四头肌就痛 (isometric) + 压髌痛，能伸膝**
  - 干扰项黑名单：膝固定器 / 铰链护具 / 免负重拐（都是骨折·韧带·伸膝装置断裂用）；"不能伸膝" = 伸膝装置断裂 ≠ PFPS

- [2026-06-14] **晶体性 / 血清阴性脊柱关节病簇** — UW 错题 #Q4582 + #Q4581 + #Q4577（同日同簇 3 道）— 关联笔记 [[完整笔记/专题笔记/骨科/骨科_HighYield速查总览]] §七 + [[完整笔记/专题笔记/风湿/风湿_HighYield速查总览]]
  - ⚠️ 个题首错，但同一概念簇连丢 3 道 = 概念性薄弱（非偶然）
  - Q4582 假性痛风 (pseudogout)：菱形晶体 → 伴 **软骨钙化（半月板钙化 chondrocalcinosis）**，不是痛风石（那是痛风 gout）
  - Q4581 银屑病关节炎 (PsA)：腊肠指 dactylitis + 甲凹点 + DIP → next step **全面皮肤检查**（关节炎可先于皮损）
  - Q4577 反应性关节炎 (reactive arthritis)：感染后免疫病 → 一线 **NSAIDs**，抗生素不治关节炎
  - 救命反射：**晶体题=晶体形状配伴随；血清阴性关节病（PsA/反应性/AS/IBD）= dactylitis + 附着点炎 enthesitis + RF 阴性 + HLA-B27，治疗 NSAIDs 起步**

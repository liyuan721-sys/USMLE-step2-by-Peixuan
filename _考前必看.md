> [!success]- 1.1 主动脉急症决策树
> 完整：[[完整笔记/专题笔记/心内/心内_主动脉急症决策树]]
> - **AAA 急诊三分支**：稳定→CT；不稳定 + 已知 AAA→**直接 OR**；不稳定 + 无既往→**床旁 US** → OR
> - **Hard signs 阳性 = 跳影像直接 OR**（4B's of Hard：Blood spurting / Bruit / Bulging hematoma / Bloodless distal）
> - **体征 ≠ 疾病**：Grey Turner = 腹膜后出血**邮编**，多病因（AAA / 出血胰腺炎 / 抗凝血肿 / 介入 / 外伤 / 肿瘤）
> - **主动脉破裂统一机制**：高压血流穿透薄浆膜 — 升→心包填塞 / 胸降→纵隔 + 左血胸 / 腹→腹膜后 80% + 腹腔 20%
> - **Dissection**：撕裂痛位置（升→胸 / 降→背）+ asymmetric pulses + Beck 三联（心包扩散）+ Stanford A→手术 / B→药物（β-blocker 先于 nitroprusside）

> [!success]- 1.2 肾上腺皮质功能不全决策树
> 完整：[[完整笔记/专题笔记/内分泌/内分泌_肾上腺皮质功能不全]]
> - **PAI 五联指纹**：Na↓ / K↑ / HCO3↓ / Glu↓ / Eos↑
> - **危象处理**：Hydrocortisone **100 mg IV bolus** + NS 1-2L + 找诱因（**不等检验**）
> - **应激分级 4 档**：轻 25mg / 中 50mg + 25 q8h × 24h / 重 100mg + 50 q8h × 48-72h / 危象 100mg bolus + 200mg/24h infusion
> - **原发 vs 继发口诀**："**原发缺三样**（cortisol + aldo + androgens），**继发 / 三发只缺一样**（仅 cortisol）"
> - **J&L-N = 聋 + LQT + AR**（必是 LQT1 纯合）
> - **Adrenal crisis vs Anaphylaxis**：慢性 + 急性触发 vs 急性 / 五联指纹 vs 正常 / refractory vs 部分有效 / Hydrocortisone vs Epi

> [!success]- 1.3 实验室指纹大全（核心指纹一句话锁定）
> 完整：[[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全]]
> - **PAI**：Na↓ / K↑ / HCO3↓ / Glu↓ / Eos↑
> - **Cushing**：K↓ / HCO3↑ / Glu↑ / HTN + 中央型肥胖
> - **Conn**：Na 正常↑ / K↓ / HCO3↑ / HTN（年轻 / 顽固）→ ARR 筛查
> - **嗜铬**：24h 尿 metanephrines↑↑ + 三联（头痛 / 出汗 / 心悸）
> - **DKA**：Glu↑↑ / HCO3↓↓ / AG↑↑ / 酮↑ / Kussmaul
> - **HHS**：Glu↑↑↑ (>600) / 渗透压↑↑ / **无酮 / 无 AG**
> - **SIADH**：Na↓ + 血 Osm↓ + 尿 Osm↑ + euvolemic + 尿 Na↑
> - **DI**：多尿稀尿 + 血 Na↑ + USG < 1.006（中枢 vs 肾性 → 水剥夺 + DDAVP）
> - **TLS**：K↑ / Phos↑ / UA↑ / Ca↓ + AKI（化疗后 24-72h）
> - **TTP**：FAT RN 五联（Fever / Anemia 裂红 / Thrombocytopenia / Renal / Neuro）
> - **MM**：CRAB + M-spike + Bence Jones + rouleaux
> - **Bartter / Gitelman**：K↓ + HCO3↑ + **正常血压**（区别于 Conn）；Gitelman + **低 Mg**
> - **RTA**：1 高尿 pH 低 K；2 低 K + Fanconi；4 高 K + 糖尿病肾病
> - **Wilson**：ceruloplasmin↓ + 24h 尿 Cu↑ + KF 环
> - **Hemochromatosis**：Fe/TIBC > 45% + ferritin↑↑ + 六联

> [!success]- 1.6 Vaughan Williams 抗心律失常药对照
> 完整：[[完整笔记/专题笔记/心内/心内_Vaughan_Williams抗心律失常药]]
> - **框架**："**1 钠 2 β 3 钾 4 钙**"
> - **QT 效应**："1A 长，1B 短，2β 4 钙是真朋友，**3 全长（除阿米奥）**，1C 不动 QRS 宽"
> - **LQT 触发**："**游泳 1，听铃 2，睡觉 3**"
> - **LQT ECG**："1 宽 2 双 3 晚"
> - **LQT 治疗**："1 β 救命，2 补 K 防铃，3 美西律救场"
> - **QT 延长病因 5 行**（Peixuan 口诀）：IA III / 大环三环喹诺酮 / 抗精抗吐昂丹 / 氯喹噻嗪阿唑钠 / 哪位鸦片镁钾钙
> - **5 急救**："SVT 找腺苷 / 室速找胺碘 / 室颤先电击 / 扭转找镁离 / WPW 找普鲁卡因"
> - **3 禁忌组合**：LQTS + IA/III（→ TdP）/ WPW + AFib + AV 结阻滞剂（→ VF）/ SHD + IC（→ CAST 死亡↑）
> - **AF + WPW 4V 识别**：Irregular + Wide + Very fast + Variable
> - **Atropine 起效**：结上有效 / 结下（Mobitz II）无效 — 直接 pacing

> [!success]- 1.9 Biostats 6 指标决策树（救人/害人 × 绝对/相对）
> 完整：[[完整笔记/专题笔记/Biostats/Biostats_6指标决策树]]
> - **框架**：2 × 3 = 6 指标（**救人 / 害人** × **绝对差 / 相对比 / 人数**）
> - **救人**（治疗降风险）：ARR / RRR / NNT
> - **害人**（暴露增风险）：ARI 或 ARP / RRI / NNH
> - **分母法则**：救人 = Risk_ctrl；害人 = Risk_unexp（都是"大的那个"）
> - **三问 SOP**：救人还是害人？绝对还是相对？要不要倒数？
> - **半步陷阱**：算出 ARR 就停 → 但 stem 要 RRR；**R = Relative 必须除基线**
> - **算出负数** = 救人 / 害人搞反了

> [!success]- 1.12 Horner 综合征 3 级定位
> 完整：[[完整笔记/专题笔记/神经/神经_Horner综合征]]
> - **三级解剖速查**：
>   - **一级 = 中枢** → 半身 anhidrosis；病：Syringomyelia / Wallenberg / MS
>   - **二级 = 节前 / 肺尖** → 同侧面颈 anhidrosis；病：**Pancoast** / 第一肋骨折
>   - **三级 = 节后 / 颈** → 出汗常保留；病：**颈动脉夹层（急症）** / Cluster
> - **三句话保命**：
>   1. "**颈痛 + Horner = 颈动脉夹层** until proven otherwise" → CTA 颈
>   2. "长期吸烟 + 肩痛 + 手内在肌萎缩 + Horner = **Pancoast**" → CXR / CT 肺尖
>   3. "披肩痛温↓ + Chiari / whiplash + Horner = **Syringomyelia 阶段 2**" → MRI 颈髓
> - **儿童单侧 Horner + 颈 / 胸 / 腹包块** → **Neuroblastoma until proven otherwise**
> - **vs CN III palsy**：CN III pupil-involving + "**down and out**" = **PCom aneurysm 神经外科急症**（不是糖尿病单纯神经病 — 后者 pupil-sparing）

> [!success]- 1.14 老年痴呆鉴别决策树
> 完整：[[完整笔记/专题笔记/神经/神经_老年痴呆鉴别]]
> - **三步走**：
>   1. 排除非痴呆（ADL / IADL 受损 + MMSE < 24）
>   2. **看速度**：数周（CJD / Pseudo / B12 / NPH 等 8 大可治）/ 数月（VaD 阶梯）/ 数年（AD / LBD / FTD / HD / Wilson）
>   3. **看招牌**：记忆 = AD / 行为 = FTD / 视幻觉 = LBD / 步态 = NPH / 阶梯 + 局灶 = VaD / 闪电 myoclonus = CJD / concerned = Pseudo
> - **三句话保命**：
>   1. "**Pseudo 抱怨；AD 被带来**"
>   2. "**Haloperidol in LBD = NMS death**"（绝对禁忌！）
>   3. "**Wet / Wobbly / Wacky + sulci 正常 = NPH 漏诊医疗事故，VP shunt 可逆**"
> - **治疗速记**：AD = **D**onepezil → Memantine → Lecanemab；VaD = **ASA + Statin**（不是 D！）；**LBD = Rivastigmine（永远不 Haldol）**；FTD = SSRI / Trazodone（不要 AChE-i）；NPH = VP shunt；Pseudo = SSRI；HD = Tetrabenazine；Wilson = Penicillamine / Zinc
> - **B12 SCD 神经症状可早于贫血** — 别等 macrocytic 才查 B12
> - **Pseudodementia 答 SSRI，不是 Donepezil**

> **① IE + 新发任何传导异常 → TEE 不过夜**（不分稳定与否）
> - 致命陷阱：题干"hemodynamically stable" 触发"先观察"反射 → **错**
> - 脓肿可在数小时内 1° → Mobitz I → Mobitz II → CHB → 心源性休克
> - HEROIC **E**xtension = 瓣周脓肿 / 新发 AV block → 立即外科会诊
> - "Stable" 是**进展前的窗口期**，不是"病情稳定"
>
> **② IE 预防 — D + R only, GI/GU since 2007 = OUT**
> - 唯二指征：**D**ental（牙龈 / 黏膜 / 根尖周）+ **R**espiratory invasive
> - 方案：**Amox 2 g PO 30-60 min before**（PCN 过敏 → Cephalexin / Azithro / Clinda）
> - 干扰项陷阱：
>   - ❌ IV ampicillin + gentamicin（2007 前 GI/GU 老方案，已废除）
>   - ❌ IV vancomycin（不在任一预防方案）
>   - ❌ 机械瓣 + colonoscopy → 仍然**不需要**（高心脏 + 低操作 ≠ 预防）
>
> **③ Janeway vs Osler — 双字母锁定（USMLE 必出反向题）**
> - **J**aneway = **J**ust spots（平 + 手掌足底 + 无痛 + **V**ascular 栓塞）
> - **O**sler = **O**uch!（鼓 + 指趾 pad + 痛 + **I**mmunologic 免疫复合物）
> - 3 秒判断：Painful + finger pad → Osler→Immune；Painless + palm/sole → Janeway→Vascular
>
> **④ 高钾元凶定位 — "What changed?" 第一问**
> - **Trigger（急性元凶）vs Permissive（慢性底色）**：CKD 是雷区，KCl 是按钮 — 雷区一直在，按按钮才爆
> - **三联 K 炸弹**：**A**CEi/ARB + **M**RA + **K** supplement = 必爆，不需看 K 值
> - **隐藏升 K 药 NTSH**：**N**SAIDs / **T**MP-SMX / **S**piro / **H**eparin
> - **PCP 给"腿抽筋"开 KCl** = USMLE 红旗（非专科 polypharmacy 必有坑）

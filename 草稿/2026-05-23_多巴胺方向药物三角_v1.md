## 一.B、Schizophrenia 四条 DA 通路（机制深化，v1.1 新增）⭐

> [!info] 理解"为什么抗精神病药副作用是这几种"的钥匙
> 抗精神病药阻断 D2 是**全脑同时阻断**——治疗作用走 mesolimbic 通路，但 nigrostriatal / tuberoinfundibular / mesocortical 三条**误伤通路** = 三种典型副作用。**记住"哪条通路 → 哪种症状"就能反推副作用**。

| 通路 | 解剖路径 | 正常 / 病理 DA 状态 | 抗精神病药 D2 阻断后果 |
|---|---|---|---|
| **Mesolimbic 中脑边缘** | VTA → 伏隔核 / 杏仁核 | DA↑ → 精神病阳性症状（幻觉 / 妄想） | ✅ **该降——治疗作用**（↓阳性症状）|
| **Mesocortical 中脑皮质** | VTA → 前额叶皮层 | DA↓ → 阴性症状 + 认知↓ | ❌ **误伤——加重阴性症状**（淡漠 / 注意力↓ / 工作记忆↓ — SGA 比 FGA 略好但仍误伤） |
| **Nigrostriatal 黑质纹状体** | SNc → 纹状体（运动控制） | DA 平衡的运动控制 | ❌ **EPS / 帕金森样副作用**（急性 dystonia / akathisia / drug-induced parkinsonism / TD — 见 [[完整笔记/专题笔记/psych/_衍生_四大EPS诊断治疗对照]]） |
| **Tuberoinfundibular 结节漏斗** | 下丘脑 → 垂体前叶（抑制 PRL） | DA = PRL 的刹车 | ❌ **高泌乳素血症**（溢乳 / 闭经 / 男乳 / 性功能障碍 / 长期 → 骨质疏松） |

```
            抗精神病药全脑阻断 D2
                    │
   ┌─────────┬──────┼──────┬─────────┐
   ▼         ▼      ▼      ▼         ▼
mesolimbic  meso-  nigro-  tubero-   (其他)
治幻觉妄想  cortical striatal infund-
            认知 ↓  EPS     高 PRL
   ✅       ❌      ❌       ❌
治疗        副作用  副作用    副作用
```

### v1.1 临床应用（USMLE 高频）

- **D2 阻断 → 高 PRL → 溢乳 / 闭经 / 男乳**：[[复盘/2026-05-23_浩斯宾判分_v1#Q9]] Aripiprazole 是 SGA 中**最不升 PRL**的（部分激动剂 → 不完全阻断 tuberoinfundibular），其他 SGA + 所有 FGA 都升 PRL；PRL > 200 → 排垂体瘤
- **SGA 比 FGA EPS 少**：5-HT2A 阻断 → 反馈性增加 nigrostriatal 通路 DA → 抵消 D2 阻断 → EPS 风险↓（但 risperidone 高剂量、强 D2 SGA 仍可出 EPS）
- **PD psychosis 用 quetiapine / pimavanserin / clozapine**：这些药对 nigrostriatal D2 阻断弱（甚至零）→ 不加重 PD（详见 [[mistakes/uworld-mistakes#^Q12152]]）

## 一.C、DA-PRL 跷跷板（v1.1 新增）⭐

> [!info] 一句话区分本质
> **下丘脑 DA → 抑制垂体 PRL**（tuberoinfundibular 通路）。所以：
> - **DA↑ 或 DA 激动剂** → PRL↓（治高 PRL 用 cabergoline / bromocriptine）
> - **DA↓ 或 D2 拮抗药** → PRL↑（抗精神病药 / metoclopramide / prochlorperazine 副作用）

```
              下丘脑 DA
                  │
                  ▼ (抑制)
            垂体前叶 PRL
                  │
        ┌─────────┴─────────┐
        ▼                   ▼
   DA 多 / 用 DA 激动剂   D2 阻断 / DA 不足
   PRL ↓                  PRL ↑
   治疗 prolactinoma     副作用（药源）
        │                   │
   ┌────┴────┐         ┌────┴────────┐
   ▼         ▼         ▼             ▼
 cabergoline           抗精神病药     metoclopramide
 bromocriptine         (尤 risperidone / FGA)
 (DA 激动剂)           prochlorperazine
                       promethazine
                       droperidol
```

### DA-PRL 跷跷板临床应用

| 场景 | PRL 方向 | 机制 | 处理 |
|---|---|---|---|
| **Prolactinoma**（垂体瘤） | ↑↑↑（>200 多见，>500 强提示） | 垂体 lactotroph 腺瘤自主分泌 | **DA 激动剂**（cabergoline 一线 > bromocriptine） |
| **抗精神病药致高 PRL** | ↑（多 < 200，药物水平） | tuberoinfundibular D2 阻断 → PRL 刹车失灵 | 换 aripiprazole（最不升 PRL 的 SGA）/ quetiapine（低 D2）；不优先用 DA 激动剂（可能加重精神病）|
| **Metoclopramide 致高 PRL** | ↑ | 同上 | 减/停药；换 ondansetron（[[草稿/2026-05-23_GI_止吐药受体分类速查_v1#四]]） |
| **妊娠 / 哺乳生理性 PRL↑** | ↑（~10x） | 雌激素 + 哺乳反射 | 生理性，不治 |
| **原发甲减致 PRL↑** | ↑（轻） | TRH 升高刺激 PRL | 治甲减后 PRL 恢复 |
| **不宁腿综合征（RLS）** | — | 中枢 DA 功能↓ + 缺铁 | DA 激动剂（pramipexole / ropinirole）+ 补铁（**铁是 DA 合成辅因子**）|

> [!tip] PRL 升高鉴别快判
> - **PRL > 200** → 多数是 prolactinoma（药物水平不到 200）→ 查垂体 MRI
> - **PRL 50-200** → 药源性 vs 微腺瘤 → **先停药 / 换药 4 周复查**；仍高 → 影像
> - **PRL < 50** → 应激 / 生理性 / 大餐 / 性交后 / 锻炼后 → 复查
> - **PRL 升高 + 雌激素低体征**（闭经 / 不孕 / 骨质疏松）→ 高 PRL 抑制 GnRH → 性腺轴关闭，不只是溢乳问题

## 二、治疗方向对照表（核心三角 + 反向钩）

| 治疗目标 | 方向 | 药物代表 | 关键场景 | 反向钩（看到选项警惕） |
|---|---|---|---|---|
| **治 tic / chorea / DA 过多 psychosis** | ⬇ DA | VMAT2 抑制剂（tetrabenazine 系列）、D2 拮抗（risperidone / haloperidol / pimozide / quetiapine） | Tourette ↗ HD chorea ↗ 抗精神病药 psychosis | 选项给抗胆碱 = 钓饵 |
| **治 EPS（D2 阻断后 ACh 相对↑）** | ⬇ ACh | Benztropine / Diphenhydramine / Trihexyphenidyl / Amantadine | 急性 dystonic reaction / drug-induced parkinsonism | 选项给抗 DA / VMAT2 = 钓饵（会更糟） |
| **治 PD 运动症状（DA 不足）** | ⬆ DA | Levodopa / DA agonist / MAO-B / COMT / Amantadine | Parkinson 病 | 选项给抗 DA / 兴奋剂之外的 ↑DA 多选项时挑机制最直接的 |
| **治 TD（慢性 D2 阻断后）** | 停药 ± VMAT2 | 减/停元凶 / Valbenazine / Deutetrabenazine / 换 quetiapine-clozapine | 长期抗精神病药使用后舌肢异动 | 选项给抗胆碱 = ❌ 加重 TD（**TD 例外**：抗胆碱不治反加重） |
| **治 ADHD** | ⬆ DA / NE | Methylphenidate / Amphetamine（兴奋剂）/ Atomoxetine（NRI）/ α2 激动剂（clonidine/guanfacine） | ADHD 单纯 vs 共病 Tourette/tic | Tourette+ADHD 共病 → **避兴奋剂**用 α2/atomoxetine |

## 三、Tetrabenazine 双面身份 ⭐ 高价值教学点

VMAT2 抑制剂（tetrabenazine / deutetrabenazine / valbenazine）= **DA 过多疾病的真适应症** vs **DA 无关疾病的钓饵**：

| 出题 context | Tetrabenazine 身份 | 原理 |
|---|---|---|
| **Tourette / Huntington / TD 题** | ✅ **真答案** | DA 过多 → 用 VMAT2 耗竭 DA |
| **焦虑题（GAD / SAD / Panic）** | ❌ **钓饵** | 焦虑与 DA 过度无关，是 5-HT/NE 通路 |
| **抑郁题** | ❌ **钓饵** | 抑郁治疗用 SSRI/SNRI/atypical AD，不沾 DA 耗竭 |
| **PD 题** | ❌ **加重病情** | PD 本来 DA 不足，VMAT2 耗竭会让 PD 瘫 |

**经典反向对照**：
- [[mistakes/uworld-mistakes#^Q20930]]：青少年 GAD 题 + 选项有 tetrabenazine → **划掉**（钓饵）
- [[mistakes/uworld-mistakes#^Q3375]]：Tourette 题 + 抗 DA 升级 → tetrabenazine = **真答案**

**判断关键**：看"题问的是什么病"，不是"药名熟悉不熟悉"。USMLE 把同一个药出在两种 context 测你的药理学定位能力。

## 四、4 类问题的 stem 暗号

| 类型 | 暗号 stem | 例子 |
|---|---|---|
| ① DA 过多 | "**multiple motor + vocal tic**" / "**chorea + 家族史**" / "**新发幻觉**" + 加用 DA 类药后 | Tourette / HD / PD psychosis / 抗精神病药精神症状 |
| ② D2 阻断后 EPS | "**新用 / 加量抗精神病药 + 数小时 dystonia / 颈部歪扭 / oculogyric crisis**" / "**坐不住 akathisia**" / "**药后僵硬 + 震颤**" | Acute dystonia / drug-induced parkinsonism / akathisia |
| ③ DA 不足 | "**慌张步态 + 静止性震颤 + 僵硬 + 起病慢**" + 老年 | Parkinson 病 |

| ⑤ 兴奋剂场景 | "**多动 + 冲动 + 注意力差 + 跨多场所 + < 12 岁起**" | ADHD（避免兴奋剂场景：合并 Tourette / 心血管禁忌 / 物质滥用史） |

## 五、易混陷阱（普适）

> [!warning] 高频陷阱
> - **"运动症状 = 抗胆碱"反射 ❌**：[[mistakes/uworld-mistakes#^Q3375]] 用户掉的坑。抗胆碱**只治"D2 阻断后 ACh 相对↑"那一种运动障碍**（急性 EPS），不治 tic / chorea / TD / PD。看到运动症状选项**先问"是哪种 DA 问题"**再选方向。
> - **"Tetrabenazine 看起来生僻 = 答案"反射 ❌**：[[mistakes/uworld-mistakes#^Q20930]] 用户掉的坑。冷僻药在 USMLE 里**多数当干扰项 / 药理学考点**，少数当答案。**先锁诊断再看药**。
> - **TD 例外：抗胆碱不治 TD 反加重**：抗胆碱治多数 EPS（dystonia / drug-induced parkinsonism），**但 TD 例外** — TD 抗胆碱不仅无效还加重。**急性 EPS 用抗胆碱，慢性 TD 用 VMAT2/停药**，两者方向不同。
> - **PD vs PD psychosis 用药方向完全相反**：PD 运动症状 = DA 不足 → ↑DA（levodopa）；PD psychosis = DA 过多（药源性）→ ↓DA（quetiapine 低 D2）。**同一病人两个症状两个方向**。详见 [[mistakes/uworld-mistakes#^Q12152]] / [[mistakes/uworld-mistakes#^Q16164]]。
> - **Quetiapine / Clozapine / Pimavanserin 在 PD 是 ✅，但其他 SGA（Risperidone / Olanzapine）在 PD 是 ❌**：SGA 不是均质组，D2 强度不同。只有**低/零 D2** 的能用于 PD psychosis。
> - **VMAT2 双向用途**：在 DA 过多病里是真药（Tourette / HD / TD）；在 DA 无关病里是钓饵（焦虑 / 抑郁）；**在 PD 里禁用**（会加重 PD）。**看诊断决定身份**。
> - **兴奋剂双向角色**：治 ADHD（↑DA）但**加重 tic / psychosis / dyskinesia / PD 不可控感**。共病场景必须谨慎（Tourette + ADHD 优先 α2 而非兴奋剂）。
> - **Amantadine 多重身份**：① 抗 PD（增 DA 释放 + 抗 NMDA）② 治流感（M2 通道阻断）③ 治 levodopa 诱导 dyskinesia ④ 治抗精神病药引起的 parkinsonism / NMS。**在 PD psychosis 题里直接划掉**（DA 能药加重 psychosis），见 [[mistakes/uworld-mistakes#^Q12152]]。
> - **MAO-B 抑制剂（selegiline / rasagiline）双面**：抗 PD（增 DA）但**高剂量失选择性 → tyramine HTN crisis 风险 + 5-HT 综合征风险**。见 [[mistakes/uworld-mistakes#^Q4879]] MAOI HTN crisis。

## 六、Memory Hook

> [!success] 记忆挂钩
> - **一句话锁定四方向**：**"DA 多了治 tic，D2 阻了治 EPS，DA 少了治 PD，慢性阻了治 TD。"** 四个问题，四个方向，对号入座。
> - **方向三角钩**（用户原稿核心增量）：脑子 DA 油门踩太死（tic / chorea / psychosis）→ **踩刹车**（anti-DA：VMAT2 耗竭 或 D2 阻断）；DA 没油了（PD）→ **加油**（levodopa / DA agonist）；D2 阻塞导致 ACh 相对压过 DA（EPS）→ **泄掉 ACh**（抗胆碱）；慢性 D2 阻塞致使齿轮变形（TD）→ **换齿轮**（停药 + VMAT2）。
> - **抗胆碱专属阵地钩**：抗胆碱**只治急性药源性 EPS**（dystonia / drug-induced parkinsonism / akathisia 用 propranolol 优先）。**不治 tic / chorea / TD / 原发 PD**。看到"运动症状选抗胆碱"先问**是不是急性 EPS**。
> - **Tetrabenazine 双面钩**：**焦虑 / 抑郁 / PD 题里出现 = 钓饵划掉**；**Tourette / HD / TD 题里出现 = 真答案选它**。Q20930 + Q3375 双题反向串记。
> - **兴奋剂红色警铃**：看到 stem 有"Tourette / 既往 psychosis / 严重焦虑 / 既往兴奋剂滥用 / 心血管风险" + 选项有 methylphenidate / amphetamine → **划掉**，选 α2 / atomoxetine。
> - **5 类问题 5 个药篮子口诀**：**"tic 抗 DA、EPS 抗 ACh、PD 升 DA、TD 停加 VMAT2、ADHD 升 DA 但避兴奋剂遇 Tourette"**。

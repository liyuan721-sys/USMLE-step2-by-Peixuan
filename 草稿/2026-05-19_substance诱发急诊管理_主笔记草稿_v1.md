# Substance-Induced Emergencies — 管理与鉴别框架

> [!info] 笔记定位
> 本笔记替代"23 个药物 23 张表"的死记模式，用 **3 轴 + 3 一眼诊断 + 3 治疗法则** 的框架记忆。
> 框架记忆 > 条目记忆，半年后仍可还原。

---

## Part 1：核心记忆框架（先记这个 ⭐）

> [!success] **"3 + 3 + 3" 法则（拿走即用）**
>
> ### 🎯 3 个轴（定位物质）
> 1. **兴奋 / 抑制**（决定生命体征方向）
> 2. **抗胆碱 / 胆碱能**（决定"干"还是"湿"）
> 3. **意识水平**（清醒 / 解离 / 抑制）
>
> ### 👁 3 个一眼诊断（30 秒分诊）
> 1. **瞳孔**
>    - 针尖瞳 → **Opioid**（或 organophosphate）
>    - 散大 → 兴奋剂 or 致幻剂 or 抗胆碱
>    - 多方向眼震 → **PCP** ⭐
> 2. **呼吸**
>    - 抑制 → **Opioid / BZD（+酒精）/ Alcohol 中毒 / Barbiturate**
>    - 正常或↑ → 其他
> 3. **干 vs 湿**
>    - "干红热瞎疯" → **抗胆碱**（TCA、抗组胺、Datura）
>    - "出汗流涎腹泻" → **胆碱能**（organophosphate）
>
> ### 💊 3 个治疗原则
> 1. **激越 → BZD**（90% 情况：PCP / cocaine / meth / 酒精戒断 / BZD 戒断）
> 2. **抑制 → 对应解毒剂**
>    - Opioid → **Naloxone**
>    - Acetaminophen → **NAC**
>    - Organophosphate → **Atropine + Pralidoxime**
>    - 抗胆碱重症 → **Physostigmine**
>    - Methanol / 乙二醇 → **Fomepizole**
> 3. **戒断 → 中毒的镜像**（致命的只有 **Alcohol + BZD** 两个）

> [!tip] 一句话总结
> **"先看瞳孔 + 呼吸 + 干湿 → 定位物质 → 激越给 BZD，抑制给解毒剂"**

## Part 2：Axis 1 — 兴奋 vs 抑制（决定生命体征方向）

```
═══════════════════════════════════════════
        兴奋类                 抑制类
═══════════════════════════════════════════
血压      ↑                    ↓
心率      ↑                    ↓
体温      ↑                    ↓
瞳孔      ↑（散大）            ↓ 或 正常
呼吸      ↑                    ↓ ⭐ 致死点
意识      激越/幻觉            嗜睡/昏迷
═══════════════════════════════════════════
```

> [!tip] 一句话锁定
> **"兴奋全部 ↑，抑制全部 ↓，呼吸抑制最致命"**

### 兴奋类（Stimulants + Hallucinogens + Dissociatives）
- **拟交感**：Cocaine / Methamphetamine / MDMA / Amphetamine
- **致幻**：LSD / Psilocybin
- **解离**：PCP / Ketamine

### 抑制类（Depressants）
- **Opioids** ⭐（针尖瞳 + 呼吸抑制 = 最高频死亡）
- **Alcohol（中毒）**
- **Benzodiazepines**
- **Barbiturates / GHB**

## Part 3：Axis 2 — 胆碱能轴（决定"干"还是"湿"）

```
═══════════════════════════════════════════════════
   抗胆碱（"干"）           胆碱能（"湿"）
═══════════════════════════════════════════════════
皮肤   干、红、热             出汗、苍白
口腔   口干                   流涎
眼     瞳孔大、视物模糊       瞳孔小（针尖）
肠     肠鸣音消失、便秘       腹泻、肠鸣音活跃
泌尿   尿潴留                 尿失禁
═══════════════════════════════════════════════════
代表  TCA、抗组胺、Datura    Organophosphate、毒蘑菇
治疗  Physostigmine（重症）  Atropine + Pralidoxime
═══════════════════════════════════════════════════
```

> [!warning] 经典口诀
> - **抗胆碱**："**Dry as a bone, Red as a beet, Hot as Hades, Blind as a bat, Mad as a hatter**"（干红热瞎疯）
> - **胆碱能**："**DUMBBELSS**"（Diarrhea, Urination, Miosis, Bronchospasm, Bradycardia, Emesis, Lacrimation, Salivation, Sweating）

## Part 4：3 个一眼诊断（进 ED 30 秒分诊）

### 一眼诊断 ① — 瞳孔大小

```
针尖瞳 (Pinpoint, Miosis)
├─ Opioids ⭐⭐⭐ 最高频
├─ Organophosphate
├─ Clonidine 中毒
└─ 桥脑出血（非药物）

正常瞳孔
├─ Alcohol
├─ Benzodiazepines
├─ Marijuana
└─ Inhalants

散瞳 (Mydriasis)
├─ 兴奋剂全家: Cocaine, Meth, MDMA, Amphetamine
├─ 致幻剂: LSD, Psilocybin, Ketamine
└─ 抗胆碱: TCA, 抗组胺, Datura, 阿托品

特殊
└─ PCP：瞳孔大小不定，但有 **多方向眼震** ⭐
   （水平 + 垂直 + 旋转 = PCP 锁定）
```

> [!tip] 瞳孔规律
> **"瞳孔大 = 兴奋 or 抗胆碱 or 致幻；瞳孔小 = 阿片 or 胆碱能"**

### 一眼诊断 ② — 呼吸

```
呼吸抑制 (RR < 12)
├─ Opioids ⭐
├─ Alcohol 中毒（重度）
├─ BZD + Alcohol 合用
└─ Barbiturates

呼吸正常或↑
└─ 几乎所有其他物质
```

> [!danger] 致命组合
> **BZD 单用很少致死，但 BZD + Alcohol 合用 → 呼吸抑制致死**（高频考点）

### 一眼诊断 ③ — 干 vs 湿

```
"干"（抗胆碱）
├─ 皮肤干燥、潮红
├─ 高热但 NO 出汗
├─ 肠鸣音消失
└─ 代表：TCA 过量、Datura（曼陀罗）、抗组胺、阿托品

"湿"（胆碱能）
├─ 大汗淋漓
├─ 流涎、流泪
├─ 腹泻
└─ 代表：Organophosphate（农药）、毒蕈碱蘑菇
```

> [!tip] 高频陷阱
> **兴奋剂（cocaine/meth）也出汗，但是因为高热**，不是胆碱能。鉴别点：**瞳孔大（兴奋剂）vs 瞳孔小（胆碱能）**

## Part 5：Axis 3 — 戒断 = 中毒的镜像 ⭐（最优雅的规律）

```
═══════════════════════════════════════════════════════
物质         中毒时               戒断时（镜像）
═══════════════════════════════════════════════════════
Opioid       缩瞳、便秘、嗜睡      散瞳、腹泻、激越
             呼吸抑制              鸡皮疙瘩、流涕、打哈欠
             （致死）              （难受但不致死）

Alcohol      镇静、共济失调        兴奋、震颤、癫痫
             呼吸抑制              **DT、致命** ⭐
                                   6-48h: 震颤、焦虑
                                   12-48h: 癫痫
                                   48-96h: DT（震颤性谵妄）

BZD          镇静、共济失调        焦虑、震颤、**癫痫**
                                   （致命 ⭐ — 缓慢减量）

Stimulant    亢奋、不眠            **"crash"**：嗜睡、食欲↑、
             食欲↓、欣快           抑郁、自杀念头
                                   （情绪致命，生理不致命）

Nicotine     兴奋（短）            焦虑、易怒、食欲↑、注意力↓
═══════════════════════════════════════════════════════
```

> [!success] 镜像规律
> **"中毒抑制 → 戒断兴奋；中毒兴奋 → 戒断抑制"**

> [!danger] 致命戒断只有 2 个
> **Alcohol + Benzodiazepine**（同 GABA 通路）
> → 这两个必须用 **BZD 替代后缓慢减量**
> 其他戒断（opioid / stimulant）难受但不致死。

## Part 6：治疗的 "3 法则"

### 法则 ① — 默认规则（覆盖 70% 题）

> [!success] **"激越 → BZD；抑制 → 解毒剂"**

```
═══════════════════════════════════════════
激越类（兴奋剂 / 致幻剂 / PCP / 酒精戒断 / BZD 戒断）
        ↓
    第一线 BZD（lorazepam IV/IM）
    第二线 加抗精神病药（haloperidol）
═══════════════════════════════════════════
抑制类（按物质给对应解毒剂）
        ↓
    Opioid          → Naloxone
    BZD（医源性）   → Flumazenil
    Acetaminophen   → NAC
    Organophosphate → Atropine + Pralidoxime
    抗胆碱重症      → Physostigmine
    Methanol/乙二醇 → Fomepizole + 透析
═══════════════════════════════════════════
```

### 法则 ② — 5 个"必记解毒剂"

| 中毒 | 解毒剂 | 机制 / 记忆点 |
|---|---|---|
| Opioid | **Naloxone** | μ 受体拮抗；起效 1-2 分钟 |
| Acetaminophen | **N-acetylcysteine (NAC)** | 补充谷胱甘肽；**24h 内最有效** |
| Organophosphate | **Atropine + Pralidoxime** | A 治症状（毒蕈碱）、P 复活酶（烟碱） |
| Methanol / 乙二醇 | **Fomepizole + 透析** | 抑制 ADH，防止毒性代谢物 |
| 抗胆碱重症 | **Physostigmine** | 唯一能过血脑屏障的胆碱酯酶抑制剂 |

### 法则 ③ — 不要做的事（高频陷阱）

> [!danger] USMLE 陷阱清单
> - **β-blocker 用于 cocaine 高血压** ❌ → unopposed α，反而恶化（用 phentolamine 或 BZD）
> - **Flumazenil 用于慢性 BZD 使用者** ❌ → 诱发癫痫（仅医源性可用）
> - **Urinary acidification 治 PCP** ❌ → 已废弃（清除率只增 1%，且加重酸中毒）
> - **肾上腺素用于 inhalant 中毒** ❌ → 心律失常猝死（"sudden sniffing death"）
> - **Naloxone 过量** ⚠️ → 突发戒断（流涕、躁动，但不致死）

## Part 8：完整管理对照表（速查用）

| 类别 | 瞳孔 | 体温/BP | 关键 stem 线索 | First-line 治疗 |
|---|---|---|---|---|
| **PCP** | 不定 | ↑/↑ | **多方向眼震、暴力、肌强直** | **BZD** |
| **Ketamine** | ↑/正 | ↑/↑ | 解离、"K-hole"、医源性 | **BZD** + 支持 |
| **Cocaine** | **↑（散）** | ↑/↑↑ | **胸痛、癫痫**、鼻中隔穿孔 | **BZD**；HTN → phentolamine（**不用 β-blocker**） |
| **Methamphetamine** | ↑ | ↑↑/↑ | **磨牙、烂牙**、舞蹈样动作 | **BZD** ± 抗精神病药 |
| **MDMA** | ↑ | ↑↑/↑ | 派对、**低钠**（喝水+SIADH）、磨牙 | **BZD** + 注意低钠 |
| **LSD** | **↑↑** | ↑/↑ | 幻视、synesthesia、"bad trip" | **BZD** + 安静环境 |
| **Marijuana** | 正 | 正 | **结膜充血、食欲↑**、欣快 | 支持治疗 |
| **Cannabinoid hyperemesis** | 正 | 正 | **慢性大麻 + 反复呕吐 + 热水澡缓解** | 停用 + haloperidol / capsaicin |
| **Opioids** | **↓↓（针尖）** | ↓/↓ | **呼吸抑制**、意识抑制 | **Naloxone** ⭐ |
| **Alcohol（中毒）** | 正 | ↓/↓ | 共济失调、构音障碍、呼吸抑制 | 支持治疗、保护气道 |
| **Alcohol（戒断）** | ↑ | ↑/↑ | 6-48h 震颤；**48-96h DT**；癫痫 | **BZD**（lorazepam / chlordiazepoxide） |
| **BZD（中毒）** | 正 | 正 | 嗜睡、共济失调、**无呼吸抑制（除非 + 酒精）** | 支持治疗；**flumazenil 仅医源性** ⭐ |
| **BZD（戒断）** | ↑ | ↑/↑ | **癫痫**（致命）、震颤、焦虑 | 重启 BZD 后**缓慢减量** |
| **Inhalants** | 正 | 正 | **青少年**、口周皮疹、**心律失常猝死** | 支持；**不用肾上腺素** |
| **Anticholinergics** | **↑** | ↑↑/↑ | **"干红热瞎疯"**、肠鸣音消失 | 支持；重者 **physostigmine** |
| **Organophosphates** | **↓** | 正 | **"DUMBBELSS"**、肌束颤动 | **Atropine + Pralidoxime** |

## Part 9：高频混淆鉴别表 ⭐

### 表 1：眼部体征 → 中毒鉴别

| 眼部表现 | 指向 |
|---|---|
| **Multidirectional nystagmus** (H+V+rotary) | **PCP** ⭐ |
| Horizontal nystagmus only | Alcohol, BZD, phenytoin, lithium |
| **Mydriasis**（双侧散瞳） | Cocaine, meth, amphetamine, 抗胆碱 |
| **Miosis**（针尖瞳） | **Opioids** ⭐, organophosphate, clonidine |
| Ocular clonus（眼阵挛） | **Serotonin syndrome** ⭐ |
| Nystagmus + ataxia + ophthalmoplegia | **Wernicke**（给 thiamine） |

### 表 2：Hyperthermia + muscle rigidity 五大鉴别 ⭐

| 综合征 | 触发 | 关键鉴别点 | 治疗 |
|---|---|---|---|
| **PCP intoxication** | 滥用 PCP | **多方向眼震**、暴力、反复滥用史 | BZD |
| **Serotonin syndrome** | SSRI + 其他血清素药 | **反射亢进、阵挛**、ocular clonus、腹泻 | 停药 + BZD + cyproheptadine |
| **NMS** (Neuroleptic Malignant) | 抗精神病药 | **"铅管"强直**、起病慢（数日）、CK 极高 | 停药 + dantrolene / bromocriptine |
| **Malignant hyperthermia** | 吸入麻醉 / 琥珀酰胆碱 | 麻醉中急性发作、家族史 | **Dantrolene** + 停麻醉 |
| **Anticholinergic** | TCA、抗组胺、Datura | **干、红、热、瞎、疯**、肠鸣音消失 | 支持；重者 physostigmine |

### 表 3：Toxidromes 速查

| Toxidrome | 心率 | 血压 | 体温 | 瞳孔 | 皮肤 | 神志 |
|---|---|---|---|---|---|---|
| Sympathomimetic（兴奋剂） | ↑ | ↑ | ↑ | ↑ | **湿（出汗）** | 激越 |
| Anticholinergic | ↑ | ↑ | ↑ | ↑ | **干、红、热** | 谵妄 |
| Cholinergic（OP） | ↓ | ↓ | 正 | ↓ | **湿（流涎、出汗）** | 抑制 |
| Opioid | ↓ | ↓ | ↓ | ↓↓ | 正 | 抑制 |
| Sedative-hypnotic | 正 / ↓ | ↓ | ↓ | 正 | 正 | 抑制 |

## Part 10：错题积累区

### 已积累的错题
- **Q3191** — PCP 中毒首选 BZD（误选 cyproheptadine → 把 PCP 混淆为 serotonin syndrome）
  - 教训：muscle rigidity + 眼动异常 + hyperthermia **不等于** serotonin syndrome
  - 关键鉴别：用药史（滥用 vs SSRI）+ 反射（正常 vs 亢进）

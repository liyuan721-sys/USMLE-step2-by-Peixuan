# 抑郁患者家族史的临床提示意义（主笔记草稿 v1）

> [!info] 一句话定位
> 抑郁题里出现 **family history**，几乎从不是"背景信息"——它常常**改变下一步处理**。看到它先问自己：**"是 bipolar 还是 unipolar？"**

---

## 核心：三种家族史 → 三种临床含义

```
抑郁患者问家族史 → 三件事
──────────────────────────────────────────
① 家族有 BIPOLAR    → 用药前先筛躁狂；慎单用抗抑郁药（防转相）  ★最高频
② 家族有 SUICIDE    → 患者本人自杀风险 ↑
③ 家族有 DEPRESSION → 支持诊断 + 起病早易复发 + 选亲属有效的药/ECT
```

---

## ① 家族有 Bipolar（双相）→ 最高优先级红旗 ⭐

> [!danger] 核心
> 一个看似"抑郁"的患者，若**一级亲属有双相障碍** → **先筛查既往有无（轻）躁狂发作**，**不要急着单用抗抑郁药**。
> 原因：① 抬高了"这其实是 bipolar depression"的概率；② **抗抑郁药单药可诱发躁狂转相 (manic switch) 或快速循环**。

**"伪装成抑郁的双相"线索（合在一起看）：**

| 线索 | 含义 |
|---|---|
| 一级亲属 bipolar | 遗传负荷最高 |
| Early-onset（青少年/<25 岁起病）| 早发偏双相 |
| 反复发作 / 病程多变 | 单相少见如此 |
| 伴 psychotic features | 双相抑郁更常见 |
| 产后发作 (postpartum) | 双相风险窗口 |
| 抗抑郁药后**激越 / 失眠 / 转躁** | 暴露出的躁狂倾向 |

> [!tip] 下一步
> 怀疑 bipolar depression → **不用 SSRI 单药**，按双相抑郁处理（如 quetiapine、lurasidone、lamotrigine、lithium 等心境稳定剂 ± 二代抗精神病药）。

---

## ② 家族有 Suicide（自杀）→ 自杀风险因子

- **家族自杀史升高患者本人自杀风险**（中等权重因子）。
- 注意权重：仍**弱于"本人既往自杀未遂"**（后者才是单一最强预测因子，见 Q2517）。

---

## ③ 家族有 Depression（抑郁）→ 三重提示

| 提示 | 含义 |
|---|---|
| 支持诊断 | MDD 遗传度约 30–40%，一级亲属患病使风险 ↑2–3 倍 |
| 病程预测 | 家族性抑郁**起病更早、更易复发、更重** |
| **指导选药** ⭐ | 亲属对**某 SSRI / ECT 反应好** → 该患者优先选同款（药物反应有家族相关性）|

---

## Memory Hook

> [!success] 锁定
> 抑郁题见 "family history" → 先问 **"bipolar 还是 unipolar？"**
> - 这一问可能**直接改掉下一步**：从"开 SSRI"变成"先筛躁狂"。
> - 口诀：**Bipolar 防转相、Suicide 抬风险、Depression 选对药。**

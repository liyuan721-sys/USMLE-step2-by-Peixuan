# PD 用药全解（Parkinson Disease Pharmacology）

> [!info] 笔记定位
> 神经科 PD 用药主笔记草稿 **v4**。涵盖：① 多巴胺双刃剑核心逻辑 ② 用药词根速记 ③ 重点药物详解 ④ 各类药物适应人群/适应症/副作用 ⑤ 选药决策树 ⑥ PD psychosis 管理流程（含四类药诱发谵妄机制）⑦ 抗精神病药安全谱 ⑧ 高频陷阱 ⑨ Memory Hook。
> 源头错题：Q16164（PD psychosis 管理）。等神经科复习完后由 Claude Code 整合进正式主笔记。
>
> **v1 → v2 升级点**：补全各类药物的适应人群、适应症、副作用（Part 4），新增年轻 vs 老年 / 震颤为主 vs 运动迟缓为主的选药决策树（Part 5）。
> **v2 → v3 升级点**：新增缩写说明 callout（ICD 全称注解 + 与心内 ICD/疾病分类的区分）。
> **v3 → v4 升级点**：Part 6 新增"四类非 PD 精神活性药诱发 psychosis/谵妄机制"（抗胆碱能负荷 vs GABA 能镇静两条主线 + 与 PD 内源 psychosis 的区分）。

> [!info] 缩写说明
> 本笔记中 **ICD = Impulse Control Disorder（冲动控制障碍）**，指多巴胺激动剂诱发的病理性赌博/购物/性欲亢进/暴食，**不是**心内的 ICD（Implantable Cardioverter-Defibrillator 植入式除颤器），也不是 ICD-10 疾病分类编码。看到 PD / 多巴胺激动剂语境 → 一律指冲动控制障碍。

## Part 1 · 核心逻辑：多巴胺双刃剑

PD 的本质是黑质多巴胺能神经元退化 → 多巴胺不足 → 运动症状。所有 PD 用药围绕"补多巴胺信号"展开,但补多了就出精神/冲动问题。**一根轴看懂所有 PD 用药题：**

```
多巴胺双刃剑
═══════════════════════════════════════════
多巴胺 ↑ (levodopa, DA 激动剂, MAO-B/COMT 抑制剂)
    └─→ 救运动症状 ✓
        但 → psychosis(视幻觉) / 冲动控制障碍(ICD) / 异动症 ✗

多巴胺 ↓ (haloperidol, risperidone, metoclopramide)
    └─→ 抗幻觉 / 止吐 ✓
        但 → 恶化 parkinsonism(震颤、僵硬加重) ✗

           ←────────── 走钢丝 ──────────→
   多巴胺太少=瘫            多巴胺太多=疯
```

> [!tip] 一句话区分本质
> PD 用药永远在"运动症状 vs 精神症状"之间走钢丝。看到 PD 题先问:**这次是多巴胺不够(运动差)还是多巴胺太多(幻觉/冲动)?** 方向定了选项就清晰了。

## Part 2 · 用药词根速记表

| 词根 | 类别 | 代表药 | 机制 + 关键点 |
|---|---|---|---|
| **-pexole / -nirole / -gotine** | 多巴胺激动剂 (DA) | **Prami**pexole, **Ropi**nirole, **Roti**gotine | 直接激动 D2/D3；**最易诱发 psychosis + ICD** |
| **-giline** | MAO-B 抑制剂 | Sele**giline**, Rasa**giline** | 抑制多巴胺降解；联用 SSRI/哌替啶 → 5-HT 综合征 |
| **-capone** | COMT 抑制剂 | Enta**capone**, Tol**capone** | 延长 levodopa 作用；Tolcapone 肝毒 |
| **(carbidopa-)levodopa** | 多巴胺前体 | Carbidopa-**Levodopa** | 金标准,最有效；carbidopa 防外周分解;最不易诱发 psychosis |
| **Amantadine** | NMDA 拮抗 | **Amantadine** | 治异动症(dyskinesia)；副作用 livedo reticularis |
| **Benz- / Trihexy-** | 抗胆碱能 | **Benz**tropine, **Trihexy**phenidyl | 主治震颤；老人慎用(认知/抗胆碱副作用) |

```
词根速记图
═══════════════════════════════════════════
增加多巴胺信号（救运动 / 易诱发 psychosis）
┌─────────────────────────────────────────┐
│ -pexole/-nirole/-gotine → DA 激动剂       │ ← 最易 psychosis + ICD
│ -giline                 → MAO-B 抑制      │
│ -capone                 → COMT 抑制       │
│ levodopa                → 前体(金标准)     │
└─────────────────────────────────────────┘
其他机制
┌─────────────────────────────────────────┐
│ Amantadine → NMDA 拮抗 (治异动症)          │
│ Benz-/Trihexy- → 抗胆碱能 (治震颤)         │
└─────────────────────────────────────────┘
```

> [!warning] 三个 "-one" 别搞混
> - **-cap·one**（Enta**capone**）= COMT 抑制 = PD 用药
> - **-aps·one**（Dapsone）= 抗生素/麻风/PCP 预防 = **不是** PD 药
> - 差一个字母,听结尾:capone vs apsone

## Part 3 · 重点药物：Pramipexole 与 Pimavanserin

### Pramipexole（多巴胺激动剂 DA）
- **机制**:直接激动 D2/D3 受体,模拟多巴胺。
- **用途**:PD 运动症状、不宁腿综合征(RLS)。
- **三大标志副作用**（高频考点）:
  1. **冲动控制障碍 (ICD)** —— 病理性赌博、强迫购物、性欲亢进、暴食;患者常**缺乏自知**
  2. **psychosis / 视幻觉** —— DA 类是最易诱发 psychosis 的抗帕金森药
  3. **突发睡眠发作** —— 开车时睡着
- **词根家族**:prami-/ropi-/roti- = DA(Pramipexole/Ropinirole/Rotigotine)

### Pimavanserin（PD psychosis 专用抗精神病药）
- **机制**:**纯 5-HT2A 反向激动剂,零 D2 活性**。
- **独特价值**:绕开整个 D2 系统 → 抗幻觉**不恶化运动**。
- **身份**:**FDA 唯一专门批准用于 PD 相关 psychosis** 的药物。
- 词根 **-serin** 提示 5-HT2A 作用。

> [!info] 为什么 Pimavanserin 完美适配 PD
> PD 病人"D2 一动就出乱子"——拮抗 D2 抗幻觉但会瘫,激动 D2 救运动但会幻觉。Pimavanserin 走 5-HT2A 通路抗幻觉,完全不碰 D2,所以既治幻觉又不伤运动。

## Part 4 · 各类药物：适应人群 / 适应症 / 副作用（v2 新增核心）

> [!tip] 怎么用这一节
> USMLE 高频两种考法:① 给"适应人群/症状特点"问选哪类药 ② 给"典型副作用"反推是哪类药。每类药记死它的"专属副作用"和"专属适用场景"即可。

### ① Levodopa（+ Carbidopa）—— 金标准

| 维度 | 内容 |
|---|---|
| **适应人群** | **老年患者、运动症状严重者**首选(最有效);症状明显影响生活时优先 |
| **适应症** | PD 全部运动症状(尤其运动迟缓 bradykinesia、僵硬);最有效药物 |
| **副作用** | **运动并发症**(长期):① 异动症 dyskinesia ② **wearing-off(疗效末期减退)** ③ **on-off 现象(突然失效/恢复,不可预测)**;直立性低血压;恶心;较少诱发 psychosis (相对 DA) |
| **关键点** | carbidopa = 外周脱羧酶抑制剂,阻止 levodopa 在外周转化,**减少外周副作用(恶心、低血压)+ 增加入脑量** |

> [!warning] Levodopa 的"长期困境"
> 用得越久,运动并发症越多(dyskinesia + on-off 波动)。所以**年轻患者常先用 DA 激动剂"省着 levodopa"**,把 levodopa 留到后期。这是年轻 vs 老年选药差异的核心原因(见 Part 5)。

### ② 多巴胺激动剂 DA（Pramipexole / Ropinirole / Rotigotine）

| 维度 | 内容 |
|---|---|
| **适应人群** | **年轻患者(< 60-65 岁)**初始单药,延缓 levodopa 启用、减少远期运动并发症;Rotigotine 透皮贴适合吞咽困难者 |
| **适应症** | PD 早期运动症状;**不宁腿综合征(RLS)**;levodopa 的辅助/省剂量 |
| **副作用** | ① **冲动控制障碍 ICD (Impulse Control Disorder)**(赌博/购物/性欲亢进/暴食) ② **psychosis/视幻觉**(最易) ③ **突发睡眠发作** ④ 直立性低血压 ⑤ 外周水肿 ⑥ 恶心 |
| **关键点** | 比 levodopa **更易诱发精神/冲动副作用**,但**更少运动并发症**——这是与 levodopa 的核心权衡 |

### ③ MAO-B 抑制剂（Selegiline / Rasagiline）

| 维度 | 内容 |
|---|---|
| **适应人群** | **早期、症状轻**的患者(单药);或作 levodopa 辅助延长疗效 |
| **适应症** | 早期 PD 单药;辅助治疗 wearing-off |
| **副作用** | **5-羟色胺综合征(联用 SSRI/SNRI/TCA/哌替啶 meperidine/右美沙芬)**;失眠(selegiline 代谢产生苯丙胺类);轻度症状改善有限 |
| **关键点** | 最重要考点是**药物相互作用**:与各种 5-HT 能药物联用 → serotonin 综合征(高热、肌阵挛、反射亢进、自主神经亢进) |

### ④ COMT 抑制剂（Entacapone / Tolcapone）

| 维度 | 内容 |
|---|---|
| **适应人群** | 已用 levodopa 但出现 **wearing-off** 的患者(**必须与 levodopa 联用**,单用无效) |
| **适应症** | 延长 levodopa 作用时间,减少 wearing-off |
| **副作用** | 加重 levodopa 的多巴胺能副作用(异动症、恶心);尿/汗液变橙色;**Tolcapone → 肝毒性(需监测肝功)**,Entacapone 较安全更常用 |
| **关键点** | 单独无抗帕金森作用,**只能配 levodopa**;Tolcapone 肝毒是为什么 Entacapone 更常用 |

### ⑤ 抗胆碱能（Benztropine / Trihexyphenidyl）

| 维度 | 内容 |
|---|---|
| **适应人群** | **年轻 + 震颤为主(tremor-predominant)**的患者;**老年人避免** |
| **适应症** | 主治**震颤**(对运动迟缓/僵硬效果差);也治抗精神病药诱发的急性肌张力障碍 |
| **副作用** | **抗胆碱能毒性**:口干、便秘、尿潴留、视物模糊、**认知损害/谵妄(老人尤甚)**、心动过速 |
| **关键点** | "**老人别用**"是高频陷阱——抗胆碱能恶化老年认知;只在年轻、以震颤为突出症状时考虑 |

### ⑥ Amantadine（NMDA 拮抗 + 弱多巴胺能）

| 维度 | 内容 |
|---|---|
| **适应人群** | **levodopa 诱发的异动症(dyskinesia)**患者;早期轻症可短期单用 |
| **适应症** | **抗异动症(首选)**;轻度改善 PD 运动症状;曾用作抗流感病毒药 |
| **副作用** | **Livedo reticularis(网状青斑)** ⭐特征性;下肢水肿;**幻觉/意识模糊(尤其肾功能不全,经肾排泄需减量)**;失眠 |
| **关键点** | 见到 PD + livedo reticularis → 立刻想 Amantadine;治 dyskinesia 是其最高频考点 |

## Part 5 · 选药决策树（v2 新增）

```
PD 初始药物选择决策树
═══════════════════════════════════════════════
                  确诊 PD,需启动治疗
                          │
        ┌─────────────────┴─────────────────┐
        │                                   │
   症状轻微?                            症状明显影响生活
        │                                   │
        ▼                                   ▼
  可观察 / MAO-B               ┌──────────年龄?──────────┐
  抑制剂(rasagiline)            │                         │
                          年轻(<60-65)              老年(>65)/严重
                               │                         │
                               ▼                         ▼
                      DA 激动剂为主             Levodopa 为主
                  (省着 levodopa,延缓          (最有效,老人对
                   远期运动并发症)              ICD/低血压更需避开 DA)
                               │
                  ┌────────────┴────────────┐
            震颤为主?                   运动迟缓/僵硬为主?
                  │                          │
                  ▼                          ▼
        年轻可加抗胆碱能            DA 激动剂 / Levodopa
        (benztropine)              (抗胆碱能对此类无效)
═══════════════════════════════════════════════

后期并发症处理:
  • wearing-off  → 加 COMT 抑制剂(entacapone) 或 MAO-B
  • dyskinesia   → 加 Amantadine
  • psychosis    → 见 Part 6 流程(先减药 → 低 D2 抗精神病药)
```

> [!tip] 选药三句话总结
> 1. **年轻 → 先 DA 激动剂**(把 levodopa 留到后面,避免早早出现运动并发症)
> 2. **老年/重症 → 先 levodopa**(最有效;老人对 DA 的 ICD/精神副作用/低血压更敏感,避开 DA)
> 3. **震颤为主 + 年轻 → 可加抗胆碱能**(老人禁,认知风险)

## Part 6 · PD Psychosis 管理流程

PD psychosis(视幻觉为主、可有妄想/错觉)是治疗副产物 / 疾病进展,影响达 **40%** 患者。早期保留洞察力(知道"不正常"、会担心),晚期发展为妄想。

```
PD Psychosis 5 步管理流程
═══════════════════════════════════════════
1. 排除 infection / delirium / dementia
   （有 → 先治这些）
        │ 无
        ▼
2. 症状轻微？→ 仅监测 (monitor)
        │ 否
        ▼
3. 减/停【非 PD 精神活性药】
   (镇静药、抗焦虑药、TCA、抗胆碱能)
        │ 无效
        ▼
4. 减【PD 药】,从效力最低的开始逐渐减
   (顺序:抗胆碱能→DA 激动剂→...→levodopa 最后减)
        │ 仍难治
        ▼
5. 上【低/零 D2 抗精神病药】
   (Pimavanserin / Quetiapine / Clozapine)
```

> [!tip] 减药顺序口诀
> **"先减杂药,再减帕药,帕药从弱减起。"**
> 先减非 PD 精神活性药 → 再减 PD 药 → PD 药从效力最低的开始(levodopa 最有效、最不易诱发 psychosis,所以最后减)。

### 第 3 步深挖：四类"非 PD 精神活性药"为什么诱发 psychosis/谵妄

> [!info] 本质
> 这四类药诱发的更多是**药源性谵妄(drug-induced delirium)**,临床表现可与 PD 内源 psychosis 重叠(都有幻觉),所以流程第 3 步先撤这些"外来打击"。机制无非两条主线。

```
两条主线
═══════════════════════════════════
主线 A:抗胆碱能负荷 ↑ (anticholinergic burden)
   阻断中枢 M1 受体 → 乙酰胆碱(ACh)信号 ↓
   → 谵妄、幻觉、认知/记忆损害
   涉及:抗胆碱能药、TCA、第一代抗组胺

主线 B:GABA 能镇静 → 意识/认知抑制
   增强 GABA → 中枢广泛抑制 → 意识混浊
   → 谵妄(老人尤甚;用药中 + 戒断反跳均可)
   涉及:苯二氮䓬(抗焦虑/镇静)、Z-drugs
```

| 药类 | 主线 | 核心机制 | PD/老人风险点 |
|---|---|---|---|
| **抗胆碱能**(benztropine, trihexyphenidyl)| A | 阻断中枢 M1 → ACh 信号不足 → 谵妄/幻觉 | PD 本身胆碱能神经元退化(尤其 PDD/DLB),储备低,雪上加霜 |
| **TCA**(amitriptyline)| A | **强抗胆碱能活性**(与上同源)| 与抗胆碱能同主线;对比 SSRI 抗胆碱能低、低风险 |
| **抗焦虑 BZD**(lorazepam, alprazolam)| B | 增强 GABA-A → 中枢抑制 → 意识混浊型谵妄 | 用药中过度镇静 + 突然停药 GABA 反跳戒断谵妄,两头都危险 |
| **镇静催眠**(zolpidem; diphenhydramine)| A 和/或 B | Z-drug 走 GABA;diphenhydramine = 抗组胺 + 强中枢抗胆碱能(双重打击)| diphenhydramine 老人易谵妄,Beers 慎用 |

> [!tip] 把 Q16164 串起来
> 原题"减 sertraline(SSRI)收益小、减 carbidopa-levodopa 才对",底层逻辑就在这:**SSRI 抗胆碱能负荷低,不是元凶**;高负荷的是 TCA、抗胆碱能药。第 3 步要减的"精神活性药"指的是这些**高抗胆碱能/强镇静**的药,不是 SSRI。

> [!warning] 与 PD 内源 psychosis 的区别
> - **PD 内源 psychosis**(多巴胺能药 + 疾病进展)→ 多巴胺↑ 激动中脑边缘 D2,**视幻觉为主、意识清醒**(早期保留洞察)
> - **药源性谵妄**(上述四类)→ 抗胆碱能/GABA 机制,**意识波动、注意力障碍为核心**
> 两者常共存、表现交叉 → 所以先撤可逆外因(第 3 步),再动 PD 药本身(第 4 步)。

## Part 7 · PD 可用 / 禁用抗精神病药（D2 强弱谱）

核心原则:**D2 拮抗越弱 → 越安全用于 PD**。

| 药物 | D2 拮抗 | PD 能否用 | 备注 |
|---|---|---|---|
| **Pimavanserin** | 零(纯 5-HT2A) | ✅ 首选 | 完全不碰运动系统 |
| **Quetiapine** | 极弱 | ✅ | 运动影响最小 |
| **Clozapine** | 弱 | ✅ | 有效,需查血象(粒细胞缺乏) |
| **Risperidone**（-peridone）| 中高 | ❌ 避免 | 恶化运动 |
| **Haloperidol**（-peridol）| 强 | ❌ 禁用 | 严重恶化 parkinsonism |

```
安全谱（PD 最安全 → 最危险）
Pimavanserin → Quetiapine → Clozapine → ... → Risperidone → Haloperidol
   零 D2         极弱          弱                  中高          强 D2
   └──────── PD 安全区 ────────┘        └──── PD 禁区 ────┘
```

> [!danger] 三类绝对不能在 PD 用的"运动毒药"（强 D2 拮抗）
> - **Haloperidol**(-peridol)
> - **Risperidone**(-peridone)
> - **Metoclopramide**(甲氧氯普胺)—— 止吐/促动力药,但中枢 D2 拮抗,**埋雷高频**;PD 止吐改用 **ondansetron**
> 记:**-peridol / -peridone = 强 D2 = PD 别碰**

## Part 8 · High-Yield 高频陷阱总表

| 陷阱 | 错误惯性 | 正确思路 |
|---|---|---|
| PD 出现幻觉 | "幻觉就上抗精神病药" | 先减触发药;要上药只能上低/零 D2 |
| PD 出问题 | "加抗帕金森药对症" | 先分清是运动问题还是精神问题 |
| 减药对象 | "减任何精神活性药都行" | 减**最可能的元凶**(DA/抗胆碱能);SSRI 抗胆碱能低,非元凶 |
| PD 患者止吐 | "用 metoclopramide" | 它是 D2 拮抗会恶化 PD;改 ondansetron |
| -giline 联用 | 随意配抗抑郁药 | MAO-B + SSRI/哌替啶 → 5-HT 综合征 |
| capone vs apsone | 看成同类 | -capone=COMT(PD);-apsone=Dapsone(抗生素) |
| 老人用抗胆碱能 | "震颤就给 benztropine" | 老人禁(认知/谵妄);只用于年轻+震颤为主 |
| PD 患者突然狂赌博 | "新发精神病" | DA 激动剂诱发的**冲动控制障碍 ICD** |
| PD + livedo reticularis | 找血管病因 | **Amantadine** 的特征性副作用 |

## Part 9 · Memory Hook

> [!success] 一句话锁定
> **"PD 看见鬼,先减药别加药。"**
> 帕金森半夜看到 bushes 里有人 → 第一反应是把多巴胺能药往下减,而非加药/上抗精神病药。

**类比挂钩:** 多巴胺能药 = **油门**。运动症状已够用(车在跑)→ 松一点油门即可;再猛踩(加 pramipexole)就撞墙(psychosis)。

**口诀:**
- 用药选择:**PD 用药躲着 D2 走**(抗精神病药选低/零 D2)
- 减药顺序:**先减杂药,再减帕药,帕药从弱减起**
- 双刃剑:**多巴胺多了疯(psychosis/ICD),少了瘫(parkinsonism)**
- 年龄选药:**年轻先 DA,老年先 levodopa**
- 副作用反推:**赌博→DA;青斑→Amantadine;5-HT 综合征→MAO-B;肝毒→Tolcapone;老人谵妄→抗胆碱能**
- 谵妄机制:**抗胆碱能/TCA → ACh↓(主线 A);BZD/镇静 → GABA↑(主线 B);SSRI 负荷低→非元凶**

**反向记忆:** 看到 **-peridol/-peridone/metoclopramide** 在 PD 题里出现 → 立刻警觉"强 D2,陷阱选项"。

# 治疗阶梯升级 SOP（"用了一段仍恶化 → 找下一阶梯"通用框架）

> [!info] 一句话定位
> USMLE 反复考的一类题：stem 给"X 药已用 Y 时间仍恶化 / 控制不充分 / 复发" → 是**升级题**，要找**下一阶梯**（纵向升级），不是同阶梯横移、不是简单加量。本笔记把 6 大疾病的阶梯抽象成统一 SOP。

## 一、触发关键词识别（USMLE 升级题 stem 暗号）

| 暗号短语 | 含义 | 默认动作 |
|---|---|---|
| **"X has been on Y for Z months/years"** | 已经在用某药一段时间 | 准备升级题 |
| **"despite adherence to therapy"** | 服从治疗但效果不好 | 升级 |
| **"continues to have / progressive worsening"** | 持续 / 进行性恶化 | 升级 |
| **"symptoms have not improved"** | 症状没改善 | 升级 |
| **"required dose escalation"** + 仍效果不好 | 同阶梯加量已尝试 | 升级（不是再加量） |
| **"recurrent episodes / relapsed"** | 复发 | 升级 / 加二药 |
| **"functional impairment severe"** | 功能严重受损 | 跳过保守层直接上重磅 |

## 三、通用 SOP 决策树

```
看到"X 药用了 Y 时间仍恶化"
    │
    ▼
Step 1：确认是真"治疗失败"吗？
    ├─ 服药依从性？（"despite adherence" 暗号已确认）
    ├─ 剂量到位？（达到治疗剂量 + 足够疗程）
    ├─ 诊断对吗？（症状是否仍符合原诊断 / 有无新共病）
    │
    ▼  全部确认 = 真失败
    │
Step 2：先**纵向升级**还是**横向换药**？
    ├─ 同阶梯换药（横向）：当下阶梯还有其他一线（如双相一线 4 选 1）
    │   触发：器官功能否决 / 妊娠禁忌 / 不耐受副作用
    │   例子：Q2357（Cr↑ → 弃 Li 选 VPA）
    │
    └─ 纵向升级到下阶梯：当下阶梯已用尽
        触发：疗效不足 / 病程进展 / 功能严重受损
        例子：Q3375（α2 已 1 年 → 升抗 DA）/ Q12152（减药失败 → 上 quetiapine）
    │
    ▼
Step 3：升到下阶梯时，注意阶梯内细分选药
    ├─ Tourette 第 3 阶梯：Risperidone（默认）/ Tetrabenazine（无 EPS 但抑郁风险）/ Pimozide（QT 要 ECG）
    ├─ PD psychosis 第 5 阶梯：Quetiapine（默认）/ Pimavanserin（零 D2 但贵）/ Clozapine（最弱 D2 但 ANC 监测）
    └─ MDD augment：Lithium / T3 / SGA / 氯胺酮（看证据 + 患者特征）
```

> [!tip] 三种"失败"的对应处理
> - **横向失败**（这个药不适合这个病人）→ 同阶梯换药（Q2357 模式）
> - **纵向失败**（整阶梯效力不够）→ 升下一阶梯（Q3375 / Q12152 模式）
> - **诊断失败**（原诊断本身错了）→ 重新做 workup（如 MDD 治疗无效要查双相 / 甲状腺 / OCD 共病）

## 四、易混陷阱（普适）

> [!warning] 高频陷阱
> - **"剂量加量"反射**：看到"X 药效果不好"反射想"那就加量"。**错** — 多数升级题已经在治疗剂量，加量等于无效拖延。当 stem 给 "despite adequate dose" / "for Y months" → 直接升级，不是加量。
> - **"同阶梯换药 = 升级"误判**：在 Tourette 第 2 阶梯，guanfacine 失败后换 clonidine 仍是同阶梯横移（同 α2 家族）→ 不算升级。**真升级 = 跨阶梯**（α2 → 抗 DA）。
> - **横向 vs 纵向混淆**：双相维持 4 大一线（Q2357）属于同一阶梯内的横向否决；Tourette 三级阶梯（Q3375）属于纵向升级。两者**触发机制不同** — 横向触发是"病人禁忌"，纵向触发是"疗效不足"。**同时存在时**先排除横向禁忌（确保剩下的药安全），再决定要不要纵向升级。
> - **跳级陷阱**：功能严重受损 / 急性危机时**可以跳级**（如严重 MDD + psychotic features 直接 ECT 而非走 SSRI → SNRI → augment 三层）。stem 给"safety issue / acute danger / life-threatening" → 跳级是合理的。
> - **"治疗失败"≠"治疗无效"**：要确认依从性 + 足够疗程 + 治疗剂量 + 诊断准确。USMLE stem 给 "despite adherence" 这一句就是替你确认了，不要再质疑。
> - **升级 ≠ 替换**：很多升级实际是"加一药"（如 SSRI augment lithium），不是"停 SSRI 换 lithium"。看 stem 给 "add" vs "switch to" 区分。
> - **"long-term/maintenance"过滤词**：题干这个词排除"急性救命药"选项（如 haloperidol 不能维持 / SABA 不能维持 / nitroglycerin 不能维持），看到这词先把急性药划掉再选维持药。

## 五、Memory Hook

> [!success] 记忆挂钩
> - **一句话锁定**：**"X 药已用 Y 时间仍恶化" = 升级题暗号。先问"哪种失败"（横向禁忌/纵向疗效/诊断错）→ 再决定换药、升阶梯还是重新 workup。**
> - **三种失败口诀**：
>   - "**药不合人**"（横向）→ 同阶梯换（Q2357 双相 Cr↑ 弃 Li）
>   - "**阶梯不够**"（纵向）→ 升下一阶（Q3375 α2 弃用上抗 DA / Q12152 减药弃用上 quet）
>   - "**诊断不对**"（诊断错）→ 重新 workup（MDD 治疗无效查双相 / OCD / 甲状腺）
> - **不要原地踏步**：加量、换同家族药 = 原地踏步；跨阶梯 / 加二药 / 重做诊断 = 真行动。
> - **跳级允许场景**：safety / acute / severe / life-threatening → 跳过保守层。
> - **类比挂钩**：治疗阶梯像**梯子的横档** — 一档站不稳就上下一档（升级），不是在同一档原地跳；横档断了换块木板（横向换药）；如果是上错了梯子（诊断错），要回到地面重选梯子。

## 六、Tourette / 双相 / PD psychosis 阶梯对照（v1 收录的 3 道核心）

| 维度 | **Tourette** (Q3375) | **双相 I 维持** (Q2357 🔴) | **PD psychosis** (Q12152 🔴) |
|---|---|---|---|
| **触发题** | α2 用 1 年仍恶化 | 一线遇器官否决 | 减药后运动反弹无法承受 |
| **失败类型** | 纵向（α2 效力不够） | 横向（Li 遇 Cr↑） | 纵向（减药救不回运动） |
| **下一步动作** | 升下阶梯 → 抗 DA | 同阶梯换 → VPA | 升下阶梯 → 抗 DA（低 D2） |
| **首选药** | Risperidone | Valproate | Quetiapine |
| **第二选** | Tetrabenazine | Quetiapine / LTG | Pimavanserin |
| **避雷选项** | 抗胆碱（治 EPS 不治 tic）/ 兴奋剂（加重 tic） | Bupropion（诱躁狂）/ Haloperidol（仅急性） | Amantadine（DA 能 ↑ psychosis）/ Haloperidol（强 D2 PD 禁用） |
| **共同陷阱** | "运动症状 = 抗胆碱"反射 | "经典 = 最佳"反射跳过 lab | "幻觉 = 加抗精神病药"反射 |

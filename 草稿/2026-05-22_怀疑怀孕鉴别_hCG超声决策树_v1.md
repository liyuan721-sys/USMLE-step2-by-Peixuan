# 怀疑怀孕鉴别 — hCG · 超声 · 病史 决策树

> [!info] 这是什么
> 一个患者「自觉怀孕 / 怀疑怀孕」时的全鉴别框架。核心思路：**先用诊室 hCG 分三档，再用超声分叉，最后对"空子宫 + hCG 阴性"这组双胞胎用病史/hCG趋势做最后一刀。**
> 草稿 v1 — 素材来源 [[mistakes/uworld-mistakes#^Q2552]]（Pseudocyesis vs Missed abortion）。

---

## 一、主决策树（三层切）

```
患者怀疑/自觉怀孕
        │
   第一刀：诊室 hCG?  (注意：自述家测阳性 ≠ 真阳性，以诊室为准)
        │
├─ 极高 ───────────────→ Hydatidiform mole 葡萄胎
│                         超声"暴风雪/囊性团块"、子宫大于孕周
│
├─ 阳性 ┬ 超声宫内孕囊 ──────────→ Early IUP 正常早期宫内孕
│       │
│       ├ 超声宫内空 + 附件包块 ──→ Ectopic 异位妊娠
│       │                          (常伴腹痛/附件压痛)
│       │
│       └ 超声宫内无活力妊娠物 ──→ Missed abortion 稽留流产
│                                   (宫口闭, 胚胎死但未排, 症状常消失)
│
└─ 阴性 + 超声空子宫(薄内膜)
        │
   第二刀：怀过孕吗?  (查病史 + hCG 纵向趋势)
        │
   ├ 怀过：有出血+排出组织史, hCG 曾↑现↓(下降曲线)
   │        └──→ Complete abortion 完全流产 (排干净了)
   │
   └ 没怀过：无出血史, hCG 始终为零, 有强心理诱因
            └──→ Pseudocyesis 假孕 (躯体化, 非妄想)
                 └ (若有规律排卵月经 + 无怀孕信念 → PMS 经前综合征)
```

---

## 二、六大诊断速查表（按 hCG + 超声切）

| 诊断 | 诊室 hCG | 超声子宫 | 关键鉴别线索 |
|---|---|---|---|
| **Hydatidiform mole 葡萄胎** | **极高** | 暴风雪/囊性团块、大于孕周 | hCG 超高是招牌 |
| **Early IUP 正常早孕** | 阳性 | 宫内孕囊 | 正常妊娠 |
| **Ectopic 异位妊娠** | 阳性 | 宫内空 + **附件包块** | 腹痛、附件压痛 |
| **Missed abortion 稽留流产** | 阳性(下降中) | 宫内**无活力妊娠物** | 宫口闭、症状常消失 |
| **Complete abortion 完全流产** | 阴性 / 低位下降 | **空(已排空)** | 有出血+排出组织史 |
| **Pseudocyesis 假孕** | **阴性** | **空(薄内膜)** | 心理诱因、症状真实、非妄想 |
| *PMS 经前综合征(鉴别项)* | 阴性 | 正常 | 规律月经、无怀孕信念 |

## 三、流产谱系（Abortion spectrum）

> [!tip] 看两个点：宫颈口 开/闭 + 妊娠物 排没排

```
Threatened 先兆流产   宫口闭 + 出血, 胎儿仍活    → 妊娠物在
Inevitable 难免流产   宫口开 + 出血, 尚未排出    → 妊娠物在
Incomplete 不全流产   宫口开, 部分排出           → 宫内有残留
Complete 完全流产     宫口闭, 全部排出           → 子宫空 ★
Missed 稽留流产       宫口闭, 胚胎死亡但未排出   → 宫内有死胎物
```

> [!warning] Complete vs Missed 是相反两极
> - **Complete**：排干净了 → 空子宫
> - **Missed**：死了但没排 → 宫内有物
> 别把这两个搞反。

## 四、最难的一格：空子宫 + hCG 阴性 的"双胞胎"

> [!danger] Complete abortion vs Pseudocyesis 怎么分
> 两者**此刻都是空子宫 + hCG 阴性**，单次快照几乎一样。区别全在**纵向**。

| 维度 | Complete abortion 完全流产 | Pseudocyesis 假孕 |
|---|---|---|
| 是否真怀过孕 | **怀过** | **从未** |
| 关键病史 | 有**阴道出血 + 排出组织/血块**事件 | 无出血、无排出物 |
| hCG（单次） | 可能已阴性或低位下降 | 一直阴性 |
| **hCG（趋势）** | 从高 → **逐渐下降**(下行曲线) | **始终为零**，无曲线 |
| 超声子宫 | 空(已排空) | 空(薄内膜，从未孕) |
| 早孕症状 | 排出后**逐渐消退** | **持续存在**(躯体化) |
| 心理背景 | 不一定 | 长期不孕、强烈想怀孕执念 |
| 本质 | 真妊娠的自然结局 | 心理躯体化 |
| 处理 | 期待 / 随访 hCG 归零 | **精神科评估治疗** |

> [!tip] 一刀切的钥匙
> 问两件事：① 之前有没有"出血 + 排出组织"的事件？② 连续 hCG 是"下降曲线"还是"始终是零"？
> - 有出血排出史 + hCG 下行 → **Complete abortion**
> - 无出血史 + hCG 始终零 + 强心理诱因 → **Pseudocyesis**

## 五、Memory Hook

> [!success] 一句话锁定
> 「空子宫 + hCG 阴性分两种：'排干净的'(complete abortion，怀过、有出血史、hCG 下降曲线) vs '从没怀过的'(pseudocyesis，无出血、hCG 一直零、心理执念)。看病史和 hCG 趋势，不看这一刻快照。」

类比：
- **Complete abortion =「搬空的房子」**——住过人，东西搬走了(排干净)，但能查到水电记录(hCG 曾阳性)、邻居记得(出血排出史)。
- **Pseudocyesis =「从没住过的样板间」**——布置得像住了人(症状齐全)，但水电从没接通(hCG 始终零)。
- **Pseudocyesis 关键词三连**：长期不孕 + 症状真实 + 非妄想(能被客观检查说服) → 精神科。

## 2. Selection Bias — 5 大（"挑人有问题"）

> [!success] 一句话本质
> **谁进入研究**有偏 → 研究对象不能代表目标人群

### 速查表

| Bias | 中文 | Stem 关键词 | 一刀切口诀 | 修复方法 |
|---|---|---|---|---|
| **Ascertainment (Sampling)** | 抽样偏倚 | "convenience sample" / 单一来源招募 | 撒网地方不对 | 随机抽样 / 多中心 |
| **Nonresponse** | 不应答偏倚 | "response rate 30%" + 不应答者特征不同 | 网撒对了，鱼不上钩 | 提高应答率 / 比较应答 vs 不应答 |
| **Berkson** | Berkson 偏倚 | "hospital-based controls" | 在医院找对照 | 社区对照 |
| **Neyman (Prevalence)** | 患病率偏倚 | "rapidly fatal" / 暴露很早发生 | 早死/早愈漏网 | Cohort 而非 case-control / 用 incidence |
| **Attrition** | 失访偏倚 | "X% lost to follow-up" + 失访者特征不同 | 上钩又跑了 | 减少失访 / Intention-to-treat 分析 |

### 各 bias 详解

#### 2.1 Ascertainment (Sampling) bias

> [!info] 本质
> Ascertain = "确认 / 查明" → Ascertainment bias = "**确认 / 选择研究对象时**"产生的偏倚 = 抽样方法非随机 → 研究人群 ≠ 目标人群。

**生活化例子**：想研究"美国成年人手机使用时长"
- ❌ 在大学校园门口问 → 全是年轻人 → 偏高
- ❌ 在养老院问 → 全是老人 → 偏低
- ❌ LinkedIn 发问卷 → 只有上网+找工作的人答 → 不代表全美

**Stem 触发词**：
- "selected from a single clinic"
- "convenience sample"
- "study population differs from target population"
- 任何非随机抽样描述

**关键判断**：研究人群人口学特征 / 来源 ≠ 目标人群。

> [!warning] 反向暗号
> Stem 写"sample representative" / "demographics comparable" / "randomly selected" → **果断划掉 Ascertainment**（这是出题人主动排除的暗号）。

---

#### 2.2 Nonresponse bias

> [!info] 本质
> 抽样方法正确，但**不应答者与应答者系统性不同** → 应答到的样本不能代表全体被抽中者。

**经典场景**：邮寄问卷只 30% 回复 → 回复的人可能是有时间的、教育程度高的、对话题感兴趣的 → 不代表全体。

**Stem 触发词**：
- "response rate was 30%"（低应答率）
- "nonresponders differed from responders"

**关键判断**：低应答率 + 不应答者特征不同。

---

#### 2.3 Berkson bias

> [!info] 本质
> 用**住院病人**做对照 → 对照组本身不健康/已有暴露 → 病例-对照差异被稀释/扭曲。

**经典场景：吸烟 → 肺癌**
```
         普通人群吸烟率 30%
                |
         +------+------+
         |             |
     肺癌组         "健康"对照组
     吸烟 80%       理想：社区随机抽 → 30%
                   实际：医院其他科室 → 60%（医院里的人本来就多吸烟相关病）
         |             |
         +------+------+
                |
            算出来 OR
                |
         +------+------+
         真实 OR：80%/30% 很大
         Berkson 后：80%/60% 看起来小很多
         → 低估真实关联
```

**Stem 触发词**：
- "controls were selected from patients hospitalized for other conditions"
- "hospital-based case-control study"

**关键判断**：看到 "hospital-based controls" 闭眼选 Berkson。

> [!tip] Berkson vs Ascertainment 关系
> Berkson 是 Ascertainment 的**特殊形式**（专指"医院挑对照"），但 USMLE 当并列选项考。看到"医院对照"选 Berkson，更具体。

**记忆挂钩**：**B**erkson = **B**ed（住院床）。

---

#### 2.4 Neyman (Prevalence) bias

> [!info] 本质
> 暴露和发病间隔很长 → 研究开始时"早死"或"早愈"的患病者已不在研究池里 → 留下的"中等病程"患者不代表所有发病者。

**经典场景：雪铲 → MI**
```
         所有铲雪诱发 MI 的人
                |
        +-------+-------+-------+
        |               |       |
    当场猝死         慢慢康复    慢性病程
    (秒挂)          (出院走人) (还在医院)
        |               |       |
    吸烟率 90%       吸烟率 70%  吸烟率 50%
        |               |       |
        ❌              ❌      ✅
    没法问到         没法问到   能问到 ← 你的研究池
                                |
                          吸烟率 = 50%（被低估）
                                ↓
                       "铲雪与 MI 关系不大" ❌
                       真实情况：关系很强
```

**UWorld 表格原话翻译**：
> "Prevalence (Neyman) bias: Exposures that happen long before disease assessment can cause study to miss diseased patients that die early or recover"
> 
> = **"暴露的影响早就发生了 → 你现在才开始研究 → 早死的和早愈的人都已经不在了 → 你只能看到中间这批 → 他们不能代表全部"**

**Stem 触发词**：
- "case-control study of highly fatal disease"
- "exposure occurred many years before diagnosis"
- "long latency / rapidly fatal"

**关键判断**：致死性急病 + retrospective 设计 + 暴露很早 → Neyman。

**记忆挂钩**：**Neyman = "Now-man"** — 只能研究"现在还在"的人，过去死了的和已经好了的看不到。

---

#### 2.5 Attrition bias

> [!info] 本质
> 研究开始后**失访者与留下者系统性不同** → 留下的人不代表原研究人群。

**经典场景**：长期随访研究中，病情严重的脱落了 → 留下的看起来好转都很好 → 高估疗效。

**Stem 触发词**：
- "X% lost to follow-up"（高失访率）
- "those who dropped out differed from those who completed"

**修复**：Intention-to-treat 分析 — 不管脱落与否，按原分组分析（保留 randomization 的力量）。

## 3. Measurement / Observational Bias — 4 大（"测人有问题"）

> [!success] 一句话本质
> **怎么获取信息**有偏 → 同一个真实情况被记录成不同结果。

### 速查表

| Bias | 中文 | 谁有问题 | Stem 关键词 | 一刀切口诀 |
|---|---|---|---|---|
| **Recall** | 回忆偏倚 | 受访者（记忆） | "asked to recall" + 暴露很久前 | 想不起来（脑子） |
| **Reporting** | 报告偏倚 | 受访者（主动） | anonymous vs in-person / 污名话题 | 不想说（嘴） |
| **Observer (Interviewer)** | 观察者偏倚 | 研究者 | not blinded / 主观评分 / 追问更多 | 研究者手抖 |
| **Surveillance (Detection)** | 监测偏倚 | 研究者（频率） | "monitored more closely" | 查得多就发现得多 |

### 各 bias 详解

#### 3.1 Recall bias

> [!info] 本质
> 得病的人对"过去暴露"记得更清楚 / 想得更细 → 健康人想不起来 → 两组报告暴露率失真。
> 
> **关键点**：不是故意撒谎，是真的想不起来。

**经典场景**：先天畸形婴儿 vs 健康婴儿，问母亲孕期吃什么药
- 畸形婴儿妈妈：拼命回忆，"我好像吃过感冒药……可能吃过止吐药……"
- 健康婴儿妈妈：随口"没吃什么"
- → "孕期用药 → 先天畸形"关联被**夸大**

**Stem 触发词三件套**：
- Retrospective 设计（case-control）
- 严重疾病 / 不良结局
- 暴露发生很久前

**修复**：用医疗记录 / 客观档案代替自报。

---

#### 3.2 Reporting bias ⭐

> [!info] 本质
> 受访者因社会压力**故意**改答案（少报 / 多报）。
> 
> **关键点**：记得清清楚楚，但不想说真话。

**经典场景（Q12685）**：含糖饮料 + 肥胖
- 当面问 → 少报（怕被评判）
- 匿名问 → 诚实回答
- → 两次结果不同

**USMLE 偏爱的"reporting 触发组合"**：
- 含糖饮料 / 垃圾食品
- 酒精摄入量
- 毒品使用
- 性行为 / 性伴侣数
- 家暴 / 体罚孩子
- 体重 / BMI

**Stem 触发词**：
- "anonymous questionnaire" vs "in-person interview"
- 涉及社会污名话题

**修复**：匿名问卷、用客观指标（如血酒精浓度代替自报饮酒量）。

---

#### 3.3 Recall vs Reporting — USMLE 最高频混淆 ⭐⭐

| 维度 | Recall | Reporting |
|---|---|---|
| **本质** | 想不起来（记忆） | 不想说（社会压力） |
| **是否故意** | ❌ 不故意 | ✅ 故意 |
| **关键 stem** | "asked to recall" + 暴露很久前 | anonymous vs in-person / 污名话题 |
| **典型场景** | 畸形婴儿妈妈回忆孕期用药 | 含糖饮料/酒精/体重自报 |
| **典型 design** | Case-control（retrospective） | 任何设计都可 |
| **修复** | 用记录代替自报 | 匿名问卷 |

**一刀切口诀**：
- **"想不起来" = Recall**（脑子的问题）
- **"不想说" = Reporting**（嘴的问题）

---

#### 3.4 Observer (Interviewer) bias

> [!info] 本质
> 研究者**知道分组信息** → 测量 / 记录时不自觉偏向预期 → 数据失真。
> 
> **关键点**：和受访者无关，是**研究者自己**搞砸了。

**经典场景**：研究"新药 vs 安慰剂"对抑郁评分
- 研究者**知道**哪个患者吃新药 → 评分时潜意识更乐观 → 新药组评分虚高

**Interviewer bias 亚型**：访谈时用语气/措辞引导
- 对吸烟者追问更细："你确定每天就 5 支？"
- 对非吸烟者草草问过："不吸烟对吧？好"
- → 吸烟暴露率被人为抬高

**Stem 触发词**：
- "researcher knew the group assignment"
- "not blinded"
- "interviewer emphasized certain questions"
- 主观结果评估 + 无盲法

**修复**：**Blinding（盲法）**
- Single-blind：受试者不知道分组
- **Double-blind ⭐**：受试者+研究者都不知道（消除 Observer bias 金标准）
- Triple-blind：受试者+研究者+数据分析者都不知道

---

#### 3.5 Surveillance (Detection) bias

> [!info] 本质
> 暴露组被查得更频繁 → 多检出疾病 → 关联看似很强，实则只是检出率不同。
> 
> **关键点**：研究者没恶意，只是暴露组天然被多查。

**经典场景：OCP → 卵巢癌**
- 用 OCP 的女性：每年定期妇科随访（开处方要复查）
- 不用 OCP 的女性：没人盯着她查
- → OCP 组卵巢癌检出率虚高 → 关联被夸大

**其他例子**：
- DVT 史 → 反复做下肢超声 → 复发率虚高
- 糖尿病患者 → 频繁查 → 各种并发症检出率高
- 长期用药研究 → 用药组自带高密度随访

**Stem 触发词**：
- "exposed group was monitored more closely"
- "more frequent follow-up in the treatment group"

**修复**：协议规定相同监测频率。

---

#### 3.6 Observer vs Surveillance — 易混点

| 维度 | Observer | Surveillance |
|---|---|---|
| **本质** | 测/录时主观偏差 | 查的频率不均 |
| **比喻** | 同一件事记成不同结果 | 一组查 10 次，一组查 1 次 |
| **修复** | Blinding | 标准化监测协议 |

## 4. Confounding（混杂） — 不是 bias，但天天和 bias 一起考

> [!info] Confounding ≠ Bias
> - **Bias** = 研究**方法**有问题 → 用更好方法可以避免
> - **Confounding** = **第三变量**同时影响暴露和结局 → 即使方法完美也存在
> 
> USMLE 经常把 Confounding 放在 bias 选项里当干扰。

### 本质机制

```
经典 Confounding 三角：

           Confounder (C)
              /        \
             /          \
            ▼            ▼
    Exposure (E) ────► Disease (D)
       (你以为是这条线)
```

### 经典例子：咖啡 → 肺癌？

```
              吸烟
            /     \
           ▼       ▼
         咖啡  ──?──►  肺癌
```

吸烟者更爱喝咖啡（吸烟 → 咖啡）；吸烟独立导致肺癌（吸烟 → 肺癌）→ 咖啡和肺癌看起来有关，**真因是吸烟** → **吸烟是 confounder**。

### 三大判定标准

> [!warning] 一个变量成为 confounder，必须满足 3 条
> 1. **与暴露相关**（吸烟 ↔ 咖啡）
> 2. **与结局独立相关**（吸烟独立 → 肺癌，不通过咖啡）
> 3. **不在 E → D 因果通路上**（不是中介变量）

### 处理方法

| 阶段 | 方法 | 说明 |
|---|---|---|
| **设计** | Randomization ⭐ | RCT 把潜在 confounder 平均分到两组 |
| **设计** | Matching | Case-control 中按 confounder 配对 |
| **设计** | Restriction | 只研究非吸烟者 → 直接排除吸烟 confounder |
| **分析** | Stratification | 分层分别分析 |
| **分析** | Multivariate regression | 多元回归"调整"confounder |

---

## 5. Confounding vs Effect Modification ⭐

> [!danger] USMLE 高频混淆点

| 维度 | Confounding | Effect Modification (Interaction) |
|---|---|---|
| **本质** | 第三变量**掩盖/扭曲**真实关联 | 第三变量**改变**关联的方向或强度 |
| **分层后** | 各层 RR 相似，但 ≠ 总体 RR | **各层 RR 显著不同** |
| **处理** | "调整掉"（adjust away） | **报告分层结果**（不能消除） |
| **是不是问题** | ✅ 是偏倚，要消除 | ❌ 不是偏倚，是真实生物学现象 |

**例子**：
- **Confounding**：咖啡-肺癌-吸烟 → 调整吸烟后，咖啡和肺癌关联消失
- **Effect Modification**：阿司匹林预防 MI 的效果，男性比女性强 → 真实差异，分性别报告

---

## 6. RCT 的局限性 ⭐

> [!warning] RCT ≠ 万能

| 偏倚 | RCT 能消除吗 | 说明 |
|---|---|---|
| Confounding | ✅ 能 | 随机化的核心作用 |
| Selection（招募阶段） | ⚠️ 部分 | 随机化只解决分组，不解决招募 |
| **Recall / Reporting** | ❌ **不能** | 测量层面问题，RCT 救不了 |
| Observer | ⚠️ 要靠 **blinding** | 不是随机化 |
| Surveillance | ⚠️ 要靠**协议规定相同监测频率** | 不是随机化 |

**核心结论**：Randomization 只解决"分组前"的差异（confounding），测量过程的 bias 要靠 blinding + 标准化操作。

## 7. 系统性做题策略 ⭐

> [!tip] 看到 bias 题，按这个顺序问自己

```
            收到 bias / study 题
                    |
        +-----------+-----------+
        |                       |
   涉及第三变量?            方法论问题?
   (年龄/性别/吸烟           (设计/测量)
    反复出现)                    |
        |                       |
        ▼                       ▼
   +----+----+         +--------+---------+
   |         |         |                  |
分层后 RR  分层后 RR  "挑人"出问题      "测人"出问题
不同        相似       (Selection)        (Measurement)
   |         |         |                  |
   ▼         ▼         ├─ Ascertainment   ├─ Recall (脑)
Effect    Confounding  ├─ Nonresponse     ├─ Reporting (嘴)
Modification           ├─ Berkson         ├─ Observer (手抖)
(报告分层) (调整)        ├─ Neyman          └─ Surveillance (查多)
                       └─ Attrition
```

### 实战 4 步

1. **stem 里有"研究者主动操作"的描述吗？**（设计、招募、测量）→ 倾向 bias
2. **stem 里有第三个变量**（年龄、性别、吸烟）反复出现？→ 倾向 Confounding
3. **如果是 bias，问题出在"挑人"还是"测人"？**
   - 挑人 → Selection 5 选 1
   - 测人 → Measurement 4 选 1
4. **找最具体的 stem 触发词** —— USMLE 题 90% 都有一个"暗号词"指向特定 bias

## 8. 自测题（先不看答案）

> [!question]- 自测题 1
> 研究 "HRT → 子宫内膜癌"。HRT 用户因开处方需要每半年妇科复查，对照组每 5 年才查一次。结果显示 HRT 显著↑ 子宫内膜癌风险。最可能的偏倚？

> [!question]- 自测题 2
> Case-control 研究：母亲对孩子先天畸形史。研究者**知道**孩子是 case 还是 control，访谈病例组妈妈时**追问更多细节**。最可能的偏倚？

> [!question]- 自测题 3
> 研究"运动 → 长寿"。结果发现运动者寿命显著更长。但运动者**同时**饮食更健康、不吸烟、收入更高。这里存在什么问题？

> [!question]- 自测题 4
> RCT 研究新降压药。匿名邮寄问卷问"过去一周喝了几次酒"。患者承认饮酒会被医生骂 → 系统性少报。最可能的偏倚？（注意：是 RCT 哦）

> [!question]- 自测题 5
> 研究"重度饮酒 → 食管癌"，从肿瘤科找 case，社区找 control。结果关联弱于预期。最可能的偏倚？

> [!question]- 自测题 6
> ICU 找 case (septic shock)，同 ICU 其他科室找 control，发现某药物和 septic shock 无关。最可能的偏倚？

*答案见下面 callout（建议先答完再看）*

> [!success]- 答案与解析
> 1. **Surveillance bias** — 监测频率不均（半年 vs 5 年）
> 2. **Observer (Interviewer) bias** — 研究者知道分组+追问更多细节（注意区分 Recall：stem 主语是研究者，不是妈妈记不清）
> 3. **Confounding** — 多个第三变量（饮食、吸烟、收入）同时影响运动和长寿
> 4. **Reporting bias** — 故意少报（RCT 不能救 reporting）
> 5. **Neyman bias** — 重度饮酒者很多先死于肝硬化、车祸 → 进不了肿瘤科 case 组 → 关联被低估
> 6. **Berkson bias** — ICU 对照本身极不健康，可能也大量用过该药 → 关联被稀释

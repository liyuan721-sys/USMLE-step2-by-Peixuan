# 继发性高血压鉴别框架（v1 草稿）

> [!info] 笔记定位
> 这是**底层框架笔记**，目标：所有"年轻人 HTN / 抗药性 HTN / 伴特殊线索 HTN"题都能从**线索 → 病因 → 检查**三步推出来，不靠死记。
> 后续等错题积累后，由 Claude Code 整合成正式衍生笔记 `_衍生_继发性高血压鉴别决策树`。

---

## 一、为什么需要这个框架？

> [!danger] 核心原则
> **HTN 在 <30 岁人群中，"primary（原发性）"的概率显著下降，必须主动筛查继发原因。**
>
> 一般人群：~95% primary HTN，~5% secondary
> **<30 岁人群**：secondary 比例显著升高 → 必须主动查

USMLE 不会让你"忘记查继发"——只要有红旗，正确答案永远是**先查病因**，不是"start lisinopril"。

---

## 二、触发"查继发"的 5 个红旗 ⭐ 必背

```
Red Flags → 怀疑 Secondary HTN
────────────────────────────────────────
🚩 1. 发病年龄 <30 岁（无家族史）
🚩 2. Severe / malignant HTN (≥180/120)
🚩 3. Resistant HTN（3 种足量降压药仍控不住）
🚩 4. 已控制良好的 HTN 突然恶化
🚩 5. 伴有提示特定病因的症状 / 体征
       （tremor、低钾、阵发性头痛、打鼾、腹部杂音等）
```

> [!tip] 红旗五字诀
> **"幼、急、抗、变、伴"**
> - **幼**：<30 岁
> - **急**：severe / malignant（≥180/120）
> - **抗**：3 种药抗药
> - **变**：原本稳定突然恶化
> - **伴**：伴特异性症状/体征

> [!success] 一句话锁定
> 看到 **"<30 岁 + HTN + 任何额外线索"** → **不要选 lifestyle modification / start antihypertensive**，要找**继发病因**。

---

## 三、继发性 HTN 病因鉴别表 ⭐ 核心框架

按"系统分类"整理，每类有**触发线索 → 首选检查**配对：

### 🔴 药物/毒物（最常被忽视）

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **Cocaine / 安非他明 / MDMA** | 年轻、**鼻出血**、震颤、瞳孔↑、躁动、心动过速 | ⭐ **Urine drug screen** |
| **OCP**（口服避孕药） | 女性、用药史 | 停药观察 |
| **NSAIDs / 类固醇 / 减充血剂** | 用药史 | 停药试验 |
| **酒精**（慢性大量） | 饮酒史、MCV↑、AST/ALT>2:1 | 临床 + LFT |

### 🫘 肾脏

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **Fibromuscular dysplasia (FMD)** ⭐ | **年轻女性**、腹部杂音、对称性 HTN | **Renal Doppler US** / CTA / MRA |
| Atherosclerotic RAS | 老年人、广泛 ASCVD、ACEi 后 Cr↑ | Doppler US |
| 肾实质性（CKD / GN / PKD） | Cr↑、蛋白尿、血尿、家族史（PKD） | UA + Cr + 肾 US |

### 🩸 内分泌

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **Primary aldosteronism (Conn)** | **低钾**、代谢性碱中毒、肌无力、夜尿、**抗药性 HTN** | **Aldosterone/Renin ratio (ARR)** |
| **Pheochromocytoma** | **阵发性**头痛+心悸+出汗三联征、HTN paroxysms | 24h 尿 / 血浆 **metanephrines** |
| **Cushing syndrome** | 满月脸、向心性肥胖、紫纹、易瘀、肌无力 | 24h 尿游离皮质醇 / 1mg 地塞米松抑制 |
| **甲亢 / 甲减** | 甲亢：tremor、心悸、体重↓；甲减：迟钝、体重↑ | TSH |
| 原发性甲旁亢 | 高钙、肾结石、骨痛 | Ca、PTH |

### 🌬️ 睡眠

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **OSA** | **肥胖**、打鼾、白天嗜睡、晨起头痛 | **In-lab polysomnography** |

### 🫀 血管/结构

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **Coarctation of aorta** | **上肢 HTN + 下肢低血压**、股动脉搏弱、上下肢血压差 >20、肋间凹征 | Echo / CT/MR aortography |

### 🤰 妊娠

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| Preeclampsia / 妊娠 HTN | 孕 >20w、蛋白尿、水肿 | 尿蛋白 + BP 监测 |

## 四、决策树 — 年轻人 HTN 怎么查？⭐

```
年轻人新发 HTN (<30 岁)
        │
        ▼
┌─────────────────────────────────┐
│ 1. 详细询问病史                  │
│    - 药物（处方/OTC/违禁）       │
│    - 家族史                       │
│    - 饮酒 / 吸烟                  │
│    - 睡眠 / 打鼾                  │
│    - 阵发性症状                  │
│    - 月经史 / OCP                │
└─────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────┐
│ 2. 体格检查 找特异线索            │
│    - 四肢血压(→ coarctation）   │
│    - 腹部杂音（→ RAS / FMD）     │
│    - 库欣体征                    │
│    - 甲亢体征（tremer / 突眼）   │
│    - 瞳孔大小 / 鼻黏膜（→ 毒品） │
└─────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────┐
│ 3. 基础实验室                    │
│    - TSH                         │
│    - 电解质（K↓ → 醛固酮增多） │
│    - Cr / UA                     │
│    - 血糖 / 血脂                │
└─────────────────────────────────┘
        │
        ▼
   根据线索做"针对性"检查
        │
        ├─ 毒品线索 (tremor + 鼻出血 + 瞳孔↑) → ⭐ Urine drug screen
        ├─ 低钾 / 抗药 → ARR (醛固酮/肾素比)
        ├─ 阵发性三联征 → 血/尿 metanephrines
        ├─ 年轻女性 + 腹部杂音 → Renal Doppler (FMD)
        ├─ 库欣体征 → 24h 尿皮质醇 / DST
        ├─ 肥胖 + 打鼾 → Polysomnography
        └─ 上下肢血压差 → Echo / CT (coarctation)
```

## 五、USMLE 经典鉴别陷阱

### 🪤 FMD vs Cocaine（同样是年轻人 HTN）

| 线索 | 倾向 FMD | 倾向 Cocaine |
|---|---|---|
| 性别 | **女性**为主 | 任一 |
| 腹部杂音 | ✅ 有 | ❌ 无 |
| 鼻出血 / 鼻黏膜损伤 | ❌ 无 | ✅ 有 |
| Tremor / 瞳孔↑ / 躁动 | ❌ 无 | ✅ 有 |
| 首选检查 | Renal Doppler | Urine drug screen |

### 🪤 Primary aldosteronism vs Pheo（都是内分泌 HTN）

| 线索 | Conn | Pheo |
|---|---|---|
| HTN 模式 | **持续性** + 抗药 | **阵发性** + episodes |
| 低钾 | ✅ 经典 | ❌ 不典型 |
| 三联征（头痛+心悸+汗） | ❌ | ✅ |
| 首选检查 | ARR | metanephrines |

### 🪤 Atherosclerotic RAS vs FMD（同样是肾血管性）

| 线索 | Atherosclerotic RAS | FMD |
|---|---|---|
| 年龄 | **老年**（>55） | **年轻女性**（15-50） |
| 危险因素 | 广泛 ASCVD（CAD / PAD） | 通常无 |
| 病变部位 | **近端**肾动脉 / 主动脉开口 | **远端 2/3** 肾动脉（串珠样 "string of beads"） |
| ACEi 试验 | 启动后 Cr↑↑（>30%） | 同样需警惕 |

## 六、可卡因急性中毒 — 心血管急症 ⭐

可卡因不只是 HTN 病因，还是 USMLE **年轻人心血管急症**必考考点：

```
可卡因急性中毒 → 心血管急症五连
─────────────────────────────────
1. 高血压急症 / 颅内出血
2. ⭐ 急性心肌缺血 / MI（年轻人 MI 必查可卡因！）
3. 主动脉夹层
4. 致命性心律失常
5. 癫痫
```

> [!danger] 治疗陷阱 — 可卡因相关 MI / HTN
> **绝对禁用：β-blocker 单用**（α-受体无对抗 → 加重血管收缩 → unopposed α effect）
> **一线**：苯二氮䓬类 (benzodiazepines) + nitrates / CCB
> 如必须用 β-blocker → 用 **labetalol**（α/β 双阻断）

## 七、Memory Hook ⭐

> [!success] 一句话
> **"年轻人高血压，先查为什么 — 别急着开降压药"**

> [!info] 继发病因首字母口诀
> **"CHAPS + RFC"**
> - **C**ocaine / 可卡因
> - **H**yperaldosteronism (Conn)
> - **A**pnea (OSA)
> - **P**heochromocytoma
> - **S**teroids / Cushing
> + **R**enal artery（FMD / RAS）
> + **F**ibromuscular dysplasia（特别提，年轻女性）
> + **C**oarctation of aorta

> [!tip] 线索 → 检查 映射
> - **鼻出血 + tremor** → Urine drug screen
> - **低钾 / 抗药** → ARR
> - **阵发三联征** → metanephrines
> - **年轻女性 + 腹部杂音** → Renal Doppler
> - **肥胖 + 打鼾** → Polysomnography
> - **上下肢压差** → Echo / CT
> - **库欣体征** → 24h 尿皮质醇 / DST
> - **TSH 异常** → 已查

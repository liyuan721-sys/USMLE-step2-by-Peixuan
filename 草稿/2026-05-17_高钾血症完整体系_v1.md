# 高钾血症 (Hyperkalemia) 完整体系（v1 草稿）

## 核心定义
**Hyperkalemia**：血清 K+ >5.0 mEq/L
- **Mild** 5.0-5.5 / **Moderate** 5.5-6.5 / **Severe** >6.5
- **Emergency** = K+ >6.5 + ECG 改变，或 K+ >7.5 不论 ECG

## ECG 进展 5 阶段 ⭐（顺序必背）

| K+ (mEq/L) | ECG 表现 | 紧迫性 |
|---|---|---|
| 5.5-6.0 | 可能正常 | 低 |
| 6.0-6.5 | **Peaked T waves**（高、窄、尖、对称，胸前导联明显） | 中 |
| 6.5-7.5 | **P 变平 + PR 延长** | 中-高 |
| 7.5-8.0 | **P 消失 + QRS 增宽** | 高 |
| >8.0 | **Sine wave**（QRS 与 T 融合呈正弦波） | **濒死** |
| 极高 | VFib / Asystole / PEA | 心搏停 |

> **进展口诀**："**T**ent → **P**ass → **W**ide → **S**ine → **F**latline"
> 中文：T 帐篷 → P 消失 → QRS 变宽 → 正弦波 → 心搏停

## 三步治疗框架 ⭐⭐⭐（USMLE 最高频）

| Step | 目标 | 药物 | 剂量 | 起效 | 持续 |
|---|---|---|---|---|---|
| **1. Stabilize 稳膜** ⭐ | 防心律失常 | **Calcium gluconate** | 1-2 g IV 慢推 | **1-3 分钟** | 30-60 分钟 |
| **2a. Shift** | K+ 入细胞 | **Insulin 10 U IV + Dextrose 25 g** | 标准方案 | 15-30 分钟 | 4-6 小时 |
| **2b. Shift** | 协同 shift | **Albuterol nebulizer** | 10-20 mg neb | 15-30 分钟 | 2-4 小时 |
| **2c. Shift** | 仅严重酸中毒时 | **NaHCO3** | 50 mEq IV | 数分钟-小时 | 数小时 |
| **3a. Remove** | 经尿排出 | **Furosemide** | 40-80 mg IV | 30 分钟 | 数小时 |
| **3b. Remove** | 经肠道 | **Patiromer / SZC** | 口服 | 数小时 | 持续 |
| **3c. Remove** | 终极手段 | **Hemodialysis** | — | 立即 | 持续 |

> **3 S 助记**：**S**tabilize → **S**hift → **S**ink
> **核心口诀**："Calcium 稳心脏，Insulin 暂藏 K+，透析/利尿真排出"

## 关键注意点

### Calcium（Step 1）
- **ECG 一有高钾改变就给，不等血钾结果**
- **不降 K+**，只对抗心脏毒性（让阈值电位↑ → 不易激发心律失常）
- 可 5-10 分钟重复给（持续 30-60 分钟有效）
- **Calcium gluconate vs chloride**：
  - **Gluconate**（首选）：10% 10mL 含 Ca 4.65 mEq，可外周 IV
  - **Chloride**：3× 元素钙含量，**仅中心静脉**（外渗 → 组织坏死），用于心跳骤停 / 中心通路已有
- ⚠️ **Digoxin 中毒** + 高钾：传统教学是慎用 Ca（怕"stone heart"），现代证据较弱；**严重高钾 sine wave 仍可给**

### Insulin/Glucose（Step 2a — 最可靠的 shift 药）
- 10 U regular insulin IV + 25 g dextrose（如果 BG <250 必须加糖）
- **监测低血糖**（治疗后 1-2 小时高发）
- 单独 insulin 比单独 albuterol 更可靠

### Albuterol（Step 2b）
- **不可单用**（部分患者无反应）
- 与 insulin 联用协同
- **糖尿病 / 心衰 / CAD 慎用**（心动过速、缺血风险）

### NaHCO3（Step 2c）
- **仅在 pH <7.2 / 严重 metabolic acidosis 时有意义**
- 单独使用对正常 pH 患者效果差
- ESRD 患者用处更小

### Furosemide（Step 3a）
- **要有尿才有用**（少尿 / 无尿患者无效）
- 容量不足时先扩容

### Patiromer / SZC（Step 3b）
- **取代 Kayexalate**（后者已少用，肠坏死风险）
- 慢性高钾长期管理首选

### Hemodialysis（Step 3c）— **透析适应症 AEIOU**
- **A**cidosis severe (refractory)
- **E**lectrolytes (K+ refractory to medical Rx)
- **I**ngestion / Intoxication (methanol, lithium, salicylates)
- **O**verload (volume, refractory to diuretics)
- **U**remic complications (encephalopathy, pericarditis, bleeding)

## 治疗决策树

```
高钾血症
   │
   ├─── ECG 改变？(peaked T / no P / 宽 QRS / sine wave)
   │       │
   │       ├─ YES → Calcium gluconate 立即 IV
   │       │       （不等血钾，同时启动 Step 2 + 3）
   │       │
   │       └─ NO → 评估 K+ 水平
   │              │
   │              ├─ K+ >6.5 → Insulin/Glucose + 排出
   │              │
   │              └─ K+ 5.5-6.5 → 找原因 + 限钾 + 利尿
   │
   ├─── 容量状态？
   │       │
   │       ├─ 脱水 → IV fluids（同时降钾）
   │       ├─ 正常 → Furosemide 利尿排钾
   │       └─ 容量超负荷 → Furosemide / 透析
   │
   └─── 肾功能？
           │
           ├─ ESRD/严重 AKI → 早叫透析
           └─ 肾功能尚可 → 药物 + 利尿
```

## 急诊鉴别 — Brady + Hypotension 看 P 波（Q19929 触发增补）

> [!danger] 休克病人 HR 不代偿性增快 + ECG 异常 → **强制看 ECG**
> "HR 60 + BP 80/30" 看起来像 sinus brady / AV block，**真凶可能是 sine wave 高钾**。

| 诊断 | P 波 | HR 表现 | 处理 |
|---|---|---|---|
| Sinus brady / AV block | **在**（P-QRS 解离）| Brady | Atropine → Pacing |
| **Hyperkalemia** ⚠️ | **消失** | Brady 或 normal | **Calcium gluconate**（不等血钾）|
| Septic shock | 正常 ECG | **HR↑↑代偿** | Fluids → Norepi |
| Hypovolemic shock | 正常 ECG | **HR↑↑代偿** | **Fluids**（不立即升压）|

**核心洞察**：休克本该 **HR↑↑代偿**（baroreceptor reflex）→ HR "正常"或下降 = **异常信号**，必须查 ECG 找元凶。

**为什么这样**：
- 高钾压制窦房结 + 房室结传导 → 即使有低血压，HR 反应被钝化（伪正常）
- 同样的"伪正常 HR"也见于 β-blocker / CCB 中毒，但那两者 P 波在

> [!warning] Sine wave vs Torsades de pointes（看似都是异常波形，治疗完全相反）
> - **Sine wave**：规则**正弦曲线**（QRS 与 T 融合）→ **高钾濒死** → **Calcium gluconate**
> - **TdP**：**多形性 VT**（"twisting" QRS 轴扭转）→ 长 QT 综合征 → **Magnesium sulfate** + 起搏/除颤
>
> 别把这两个名词混。Sine wave = 数学曲线感；TdP = 看起来在跳舞。

## 高钾病因 5 大类（USMLE 鉴别）

| 类别 | 病因 | 备注 |
|---|---|---|
| **摄入↑** | KCl 补充、含 K+ IV 液（罕见单独导致） | 通常合并肾功能↓才高 |
| **排泄↓（肾性）** ⭐ 最常见 | **AKI / CKD / ESRD** | 尤其 GFR <20 |
| **排泄↓（醛固酮↓）** | **Addison's**、hyporeninemic hypoaldosteronism (DM)、**ACEi/ARB**、**spironolactone/eplerenone**、**trimethoprim**、**heparin**（抑制醛固酮合成）、4 型 RTA | 药物高频考点 |
| **细胞内→外转移** | **代谢性酸中毒**（非有机酸 — DKA 中 H+/K+ 互换）、**胰岛素缺乏**、**β-blocker**、**digoxin 中毒**、**琥珀胆碱**（烧伤/截瘫禁用！）、**横纹肌溶解**、**TLS**、**大量输血**（库存血 K+↑）、**剧烈运动** | 机制各异 |
| **假性高钾** | **溶血标本**、止血带过紧、**白血病/血小板增多**（采血后体外释放 K+） | 临床 + ECG 正常 → 重抽（绿管，不夹止血带） |

## Trigger vs Permissive — 急性高钾元凶定位思维框架 ⭐（2026-05-18 增补）

> [!danger] 高钾题第一问不是"K 多高"，而是 **"What changed?"**

| 维度 | **Trigger（急性元凶）** | **Permissive（慢性底色）** |
|---|---|---|
| 时间线 | 变化前后有清晰因果（小时-周） | 长期存在 |
| Stem 提示 | "新加 X 药 / 改剂量 / 漏透析 / 急性事件" | "既往 CKD / DM / HFrEF / 长期 MRA" |
| 临床决策 | **停 / 撤掉 trigger** + 急性降钾 | 调整方案 + 长期监测 |
| USMLE 责任归属 | 元凶选项 | 干扰项（"显眼但不变化"）|
| 经典例子 | **新加 KCl / TMP-SMX / ACEi 加量 / 漏透析 / 新发 AKI / 横纹肌溶解 / TLS** | CKD3、DM 4 型 RTA、HFrEF on lisinopril/spironolactone |

**判题口诀**：Stem 里的"新发 / 新加 / 新改"事件几乎一定是 trigger，**不要被慢性病吸走注意力**。

### 高钾药物完整清单 9 类（必背）

| 类别 | 代表药 | 升 K 机制 | 单用升 K 程度 |
|---|---|---|---|
| **ACEi / ARB** | lisinopril / losartan | ↓ Aldosterone → 远端集合管不排 K | 轻-中 |
| **MRA / K-sparing diuretic** | **spironolactone** / eplerenone / amiloride / triamterene | 直接阻断 ENaC / aldosterone 受体 | 轻-中 |
| **K supplement** | KCl PO/IV / 高 K 盐替代品 | 外源直接灌入 | 轻 |
| **NSAIDs** | ibuprofen / naproxen | ↓ Renin → ↓ Aldosterone + ↓ GFR | 轻 |
| **Trimethoprim** | TMP-SMX | 阻断 ENaC（类 amiloride） | 中 |
| **Heparin** | 普通肝素 / LMWH | 抑制 aldosterone 合成 | 轻 |
| **β-blocker（非选择性）** | propranolol | β2 阻断 → 阻 K 入细胞 | 极轻 |
| **Digoxin（中毒剂量）** | digoxin toxicity | 抑制 Na-K ATPase | 急性中-重 |
| **Succinylcholine** | — | K 外漏（去极化） | 急性中（烧伤/挤压/瘫痪患者放大） |

> [!warning] **三联及以上 = 必爆 K 炸弹**
> **ACEi/ARB + MRA + K supplement** — USMLE 经典致命组合，看到 list 里 lisinopril + spironolactone + KCl → **不需要看 K 值就报炸**。
>
> **隐藏升 K 药 NTSH**：**N**SAIDs / **T**MP-SMX / **S**piro / **H**eparin — stem 里这些"看似无关"的药要主动扫描。

### 高钾 stem 元凶定位 4 步 SOP

```
Step 1 — 时间线：K 是急性升 (天-周) 还是慢性高？
  急性 → 找 trigger（新药 / 漏透析 / 急性肾损伤 / 横纹肌溶解 / 大面积烧伤 / TLS）
  慢性 → 找底色（CKD / DM / 长期 MRA）+ 评估 trigger 叠加

Step 2 — 药物 list 扫描：
  扫 9 类升 K 药 — 尤其 ACEi/ARB + MRA + KCl 三联
  扫 NTSH 隐藏药

Step 3 — Acute insult 排查：
  AKI（少尿）/ 横纹肌溶解（CK ↑）/ TLS（新化疗 + 高 LDH）
  输血（库存血）/ 酸中毒 / 烧伤+琥珀胆碱

Step 4 — 假性高钾排除：
  溶血样本 / 紧握拳头 / WBC 或 PLT 极高 → 重抽 / 血浆 K
```

> [!example]- 典型 stem 分析 — HFrEF + CKD3 三联 + 新加 KCl
> "65 yo HFrEF (EF 30%) + CKD3 + lisinopril + carvedilol + spironolactone → PCP adds KCl 20 mEq/d for leg cramps → 2 weeks later K 6.8 + peaked T → 元凶？"
>
> - **Trigger**（急性元凶）= **新加 KCl** + 已有 ACEi + spironolactone 形成三联
> - **Permissive**（慢性底色）= CKD3 + HFrEF（让三联效应放大）
> - **正确归因**：三联 K 炸弹（ACEi + MRA + KCl）
> - **干扰项陷阱**：CKD3 是底色不是元凶；spironolactone 单独通常不会让稳定患者飙到 6.8
> - **临床动作**：停 KCl + 撤 spironolactone（或减量）+ 监测 + 急性降钾（Insulin/Glucose 等）
>
> 详见 [[mistakes/uworld-mistakes#^Q19-2026-05-18|Q19 自测错题]]。

## 易混陷阱

- ❌ "ECG 异常先做 12 导联确认 K+" → **错**：sine wave 看到就给 Calcium，不等抽血
- ❌ "Calcium 能降 K+" → **错**：Calcium 只稳膜，不降钾；必须配 shift + remove
- ❌ "单用 insulin 不加糖" → **错**：BG <250 必须加 25g 葡萄糖，否则严重低血糖
- ❌ "NaHCO3 对所有高钾都有用" → **错**：仅严重 metabolic acidosis (pH <7.2) 时有效
- ❌ "Furosemide 适合所有高钾" → **错**：无尿患者无效；容量不足要先扩容
- ❌ "Kayexalate 是首选 binder" → **错**：现已少用（肠坏死），首选 **Patiromer / SZC**
- ❌ "ACEi + ARB + spironolactone 联用治心衰" → **错**：高钾风险极高，**避免三联**
- ❌ "DKA 治疗前 K+ 高 = 不用补 K" → **错**：表面高钾但**总钾耗竭**，胰岛素一上必须监测并补 K（K+ <5.3 即开始补）
- ❌ "假性高钾不用管" → **正确做法**：临床 + ECG 正常 + K+↑ → 怀疑假性 → 重抽
- ❌ "高钾就起搏" → **错**：高钾的传导阻滞**对 pacing 反应差**（capture failure 类似 BB 中毒），先稳膜

## Memory Hook

- **3 S 框架**：**S**tabilize (Calcium) → **S**hift (Insulin/Glucose, Albuterol, NaHCO3) → **S**ink (Furosemide, Patiromer, Dialysis)
- **ECG 进展口诀**：**T**ent → **P**ass → **W**ide → **S**ine → **F**latline
- **触发铁律**："**ECG 一异常，Calcium 不等血钾**"
- **类比**：
  - 高钾 = "心脏正在被慢慢电死"
  - **Calcium = 电击防护盾**（不拔电源，但让你扛得住）
  - **Insulin = 把钾扫进储藏室**（暂时不见，但还在屋里）
  - **Dialysis/Furosemide/Binder = 真正把钾扔出门外**
- **时间维度**：稳膜买**分钟**，shift 买**小时**，remove 买**天**
- **"Calcium for the heart, Insulin for the wait, Lasix/Dialysis for real exit"**
- **高频药物三联警告**：ACEi/ARB + spironolactone + K+ supplement → 高钾炸弹
- **"What changed?"** — 高钾题第一问，看时间线，不看 K 值大小
- **"CKD 是雷区，KCl 是按钮"** — 雷区一直在，按按钮才爆（Trigger vs Permissive）
- **"PCP 开 KCl 治腿抽筋"** = USMLE 红旗 — 非心衰专科介入 polypharmacy 必有坑
- **隐藏升 K 药 NTSH**：**N**SAIDs / **T**MP-SMX / **S**piro / **H**eparin
- **C-BIG-K-D** 替代 SOP 口诀：**C**alcium → **B**icarb/**I**nsulin+**G**lucose → **K**ayexalate（现首选 Patiromer/SZC）→ **D**ialysis
- **"休克心不跳快 + P 波不见了 = 钾要命，先打钙"** — Brady+Hypo 鉴别核心（[[mistakes/uworld-mistakes#^Q19929]]）
- **类比补充**：
  - Sine wave = 心脏的"濒死告别信"（心电图变数学曲线）
  - P 波 = 心房的"签到打卡"，打卡消失 = 心房被钾"封号"

## 高频考点关联

- **Sine wave** → Calcium 不等血钾
- **Crush injury / 烧伤患者 + 琥珀胆碱** → 致死性高钾性心律失常（**禁用**）
- **DKA 治疗**：治疗前 K+ 假性正常或↑（其实总钾↓）→ 胰岛素治疗中**必须监测并补 K**（K+ <5.3 即开始补）
- **TLS**（肿瘤溶解综合征）：高 K + 高 P + 高 UA + 低 Ca + AKI → rasburicase + 水化 + 别 allopurinol 预防
- **ACEi + Spironolactone + ARB 联用** → 高钾经典三药警告
- **Trimethoprim** → 类似 amiloride 机制（阻断 ENaC）→ 高钾
- **Heparin** → 抑制醛固酮合成 → 高钾（即使预防剂量）

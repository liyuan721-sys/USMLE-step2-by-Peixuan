# 瓣膜病心音机制框架（v1 草稿）

> [!info] 笔记定位
> 这是**底层框架笔记**，不是知识点罗列。
> 目标：让所有"瓣膜病听到什么心音"的题都能从**机制反推**，不靠死记。
> 后续等错题积累后，由 Claude Code 整合成正式专题笔记。

---

## 一、为什么瓣膜病题这么难？

死记的常见误区：
- ❌ 背"AS = soft S2, S4"、"MS = loud S1, opening snap"…… 4 个瓣 × 狭窄/反流 × 4 个心音 = 几十个组合，背不下来
- ❌ 看到"狭窄"就以为"关闭音变大"，看到 AS 就懵

正确思路：**所有瓣膜病心音都可以从三个底层原理推出来**。掌握原理 = 不用背。

---

## 二、底层原理 1 — 瓣膜的"砰"声从哪里来？

> [!danger] 核心定义
> **瓣膜的关闭音 = 关闭瞬间血液被突然挡住产生的震动**
>
> 决定关闭音大小的**三要素**：
> 1. **瓣叶活动度**（还能不能动？钙化僵硬就不响）
> 2. **关闭瞬间瓣叶离闭合位置的距离**（离得远 → 关闭时速度快 → 响）
> 3. **关闭时的跨瓣压差**（压差大 → 关得猛 → 响）

### 心音对应表（必须熟记）

```
S1 = 房室瓣关闭
     ├─ M1 = 二尖瓣关闭音 ← MS 看这个
     └─ T1 = 三尖瓣关闭音

S2 = 半月瓣关闭
     ├─ A2 = 主动脉瓣关闭音 ← AS 看这个
     └─ P2 = 肺动脉瓣关闭音

S3 = 舒张早期，血液快速冲入扩张心室 ← 容量超负荷标志
S4 = 舒张末期，心房用力推血进僵硬心室 ← 压力超负荷标志
```

## 三、底层原理 2 — 心室对负荷的两种适应

> [!danger] 核心区分
> **左室面对"过载"有两种完全不同的反应**

```
压力超负荷 (Pressure overload)              容量超负荷 (Volume overload)
─────────────────────────────              ─────────────────────────────
病因：AS / HTN / HCM                       病因：MR / AR / DCM / HF / 妊娠 / 甲亢
机制：收缩期阻力↑                          机制：舒张期容量↑
肌节：并联增加 (parallel)                  肌节：串联增加 (series)
形态：concentric 向心性肥厚                形态：eccentric 离心性肥厚
       └─ 壁厚 ↑↑，腔不变或略小                  └─ 腔扩大 ↑↑，壁略厚
顺应性：↓↓ (僵硬)                          顺应性：相对保留
EF：通常保留 (HFpEF)                        EF：后期下降 (HFrEF)
特征心音：⭐ S4                            特征心音：⭐ S3
机制：心房推血进僵硬腔                      机制：血液冲入大腔
```

> [!tip] 一句话精髓
> **"压力 → 壁厚 → S4；容量 → 腔大 → S3"**
> 记忆：**"4 个壁，3 个空腔"**

> [!success] 类比
> - **S4** = 心房敲一面**僵硬的鼓** → "Tennessee" (Ten-nes-SEE)
> - **S3** = 血液冲进**松垮的大袋子** → "Kentucky" (Ken-TUCK-y)

## 四、底层原理 3 — 同样是"狭窄"，为什么 MS Loud S1 但 AS Soft A2？

> [!warning] 这是最容易混淆的一点

```
MS（典型病因：风湿性）              AS（典型病因：钙化退行性）
─────────────────────────────       ─────────────────────────────
病变本质：瓣叶交界处粘连             病变本质：瓣叶钙化僵硬
瓣叶活动度：✅ 仍可活动              瓣叶活动度：❌ 动不了
"狭窄"位置：瓣口变小                "狭窄"位置：瓣叶打不开
关闭前位置：✅ 被高房压顶得很开      关闭前位置：❌ 本来就开得不大
关闭速度：✅ 快                     关闭速度：❌ 慢且无震动

结果：⭐ Loud S1                    结果：⭐ Soft A2 / single S2
```

> [!danger] 通用规律
> 1. **瓣叶能动 + 关闭瞬间压差大** → 该瓣的关闭音**增强**
> 2. **瓣叶钙化 / 关不严** → 该瓣的关闭音**减弱**
> 3. **下游血管压高**（HTN / PAH）→ 关闭音**增强**

### 用同一原理看所有瓣膜病

| 病 | 病变本质 | 关闭音影响 | 机制 |
|---|---|---|---|
| **MS 早期** | 瓣叶粘连但能动 | **Loud S1 (M1↑)** | 瓣被高房压顶得远 + 关得猛 |
| **MS 晚期** | 瓣叶钙化僵硬 | **Soft S1** | 瓣动不了 |
| **MR** | 瓣叶关不严 | **Soft S1** | 关不严 → 没有"砰" |
| **AS（钙化型，老年人）** | 瓣叶钙化僵硬 | **Soft A2 / single S2** | 瓣震不起来 |
| **AS（二叶瓣，年轻人）** | 瓣叶尚可动 | A2 可正常 / **ejection click** | 瓣还能弹 |
| **AR** | 主动脉瓣关不严 | **Soft A2** | 关不严 |
| **系统性 HTN** | 瓣叶正常 | **Loud A2** ⭐ | 关闭压差大 |
| **肺动脉高压** | 瓣叶正常 | **Loud P2** ⭐ | 关闭压差大 |

## 五、整合 — 瓣膜病心音预测三步法 ⭐

> [!success] 见到瓣膜病题，强制套这个 SOP
>
> **第一步：判断负荷类型**
> - 这个病是**压力超负荷**（出口堵了 / 后负荷高）还是**容量超负荷**（反流 / 前负荷高）？
> - AS / HTN / HCM → 压力
> - MR / AR / DCM / HF → 容量
>
> **第二步：推心室适应**
> - 压力 → 向心性肥厚（壁厚腔小）→ 顺应性↓
> - 容量 → 离心性肥厚（腔大壁薄）→ 顺应性保留
>
> **第三步：映射心音**
> - 向心性肥厚 → **S4**（心房推血进僵硬腔）
> - 离心性肥厚 → **S3**（血冲入大腔）
> - 瓣叶钙化 → 该瓣关闭音**软**
> - 瓣叶能动 + 高关闭压差 → 该瓣关闭音**响**

## 六、四大瓣膜病速查表（预备框架）

> [!info] 占位
> 本表会随错题积累逐步补充。当前先放 AS 和 MS 早期作为锚点。

### AS (Aortic Stenosis) — 已建（Q4277）

```
病因：钙化退行性（老年） / 二叶瓣（年轻）/ 风湿性
病理生理：压力超负荷 → 向心性肥厚
症状三联征：Angina / Syncope / HF（一旦出现，预后差）
体征：
  ⭐ Soft / single S2（A2 弱+延迟）
  ⭐ Late-peaking crescendo-decrescendo systolic murmur (RUSB, 向颈部传导)
  ⭐ Pulsus parvus et tardus = slow-rising and delayed carotid pulse（颈动脉弱而延迟）
  ⭐ S4（非 S3）
动作影响：
  ↑ 杂音：下蹲 → 站起（前负荷↓不明显，但 HCM 杂音会↑↑，用于鉴别）
  实际上：增加左室容量的动作 → AS 杂音 ↑
鉴别：HCM（Valsalva 动作 → HCM ↑、AS ↓）
治疗：症状性 severe AS → AVR (TAVR / SAVR)
```

### MS (Mitral Stenosis) — 部分（来自 Q4277 鉴别）

```
病因：风湿性（绝大多数）
病理生理：左房压↑ → 肺淤血 → 肺动脉高压 → 右心衰
体征：
  ⭐ Loud S1（早期，瓣叶尚可动）→ 晚期变软
  ⭐ Opening snap（瓣开放音，紧跟 S2）
  ⭐ Apical mid-diastolic rumble（心尖部舒张中期低调隆隆样杂音）
  ⭐ S2 → OS 间期：OS 离 S2 越近 = 狭窄越严重
其他：
  - 左房扩大 → AF 高发
  - 严重时 P2 增强（PAH）
```

## 七、动作鉴别框架（预备）

> [!info] 占位
> Valsalva / 下蹲 / 站立 / 握拳等动作对杂音影响是 USMLE 高频考点。
> 等积累 2-3 道动作鉴别错题后建专门小节。

核心原理（先记住）：
- **↑ 前负荷 / 后负荷** → 反流杂音 ↑、AS 杂音 ↑、HCM 杂音 ↓
- **↓ 前负荷**（Valsalva 做、站立）→ HCM 杂音 ↑、MVP click 提前

# GI 止吐药受体分类速查（按受体记，不按药名记）

> [!info] 一句话定位
> 止吐药机制反复记不住，本质是"按药名死记 → 漏了机制"。本草稿按 **5 类受体**（D2 / 5-HT3 / NK1 / H1 / M1）组织：先看场景属于哪条通路 → 锁定受体 → 选药。**核心反向钩**：用止吐药后冒出运动症状（斜颈 / 舌肌痉挛 / lip smacking）→ 凶手必是 **D2 拮抗类**（metoclopramide / prochlorperazine / promethazine），不是 ondansetron。

> [!warning] 核心反射钩
> **每个受体对应一个解剖位置 + 一种止吐场景**：
> - **CTZ（化学感受器触发区）** → 药物/毒素诱导的恶心 → **D2 + 5-HT3** 受体集中
> - **肠道迷走神经末梢** → 化疗/术后 → **5-HT3**
> - **前庭核 / 内耳** → 晕动病 → **H1 + M1**
> - **大脑皮层 / 边缘系统** → 化疗延迟期 + 心因性 → **NK1（P 物质）**

## 二、5 类止吐药对照表 ⭐ 核心

| 受体 | 代表药 | 最佳场景 | 关键副作用 | USMLE 高频考点 |
|---|---|---|---|---|
| **D2 拮抗** ⚠️ | metoclopramide / prochlorperazine / promethazine | **胃轻瘫**（gastroparesis）/ **术后**（PONV 部分）/ **偏头痛伴呕吐** | **EPS / TD ⭐**（急性 dystonia / akathisia / drug-induced parkinsonism / TD） | "止吐药后冒出运动症状" = 必锁 D2 类；metoclopramide FDA **黑框 ≤12 周** |
| **5-HT3 拮抗** ✅ | **ondansetron** / granisetron / palonosetron | **化疗 ⭐ + 术后 ⭐**（一线、最安全） | **QT 延长**（特别 ondansetron）/ 便秘 / 头痛；**无 EPS** | "化疗止吐首选" = ondansetron；安全王，但 QT 长 / 低钾镁要查 ECG |
| **NK1 拮抗** | aprepitant / fosaprepitant | **化疗延迟期止吐**（第 2-5 天）— 与 ondansetron 联用 | 较少；CYP3A4 抑制 → 药物相互作用 | "化疗 24h 后仍恶心" = NK1 加进来；不能单用 |
| **H1 拮抗** | diphenhydramine / **meclizine** / promethazine（H1 + D2 双重） | **晕动病 ⭐** / 妊娠呕吐（meclizine B 类）/ 前庭神经炎 | 嗜睡 / 抗胆碱（口干 / 尿潴留 / 视力模糊） | "晕车晕船" = meclizine / scopolamine 贴片；妊娠呕吐先 doxylamine+B6 |
| **M1 拮抗** | **scopolamine 贴片** | **晕动病**（耳后贴片，长效 72h） | 抗胆碱（口干 / 视力模糊 / 老年谵妄） | "长途旅行 / 邮轮 / 晕动 + 想方便用" = scopolamine 贴片 |

## 三、Metoclopramide ⭐ 隐藏的抗多巴胺药（USMLE 高频陷阱）

> [!danger] 核心一句话
> **任何长期阻断中枢 D2 的药都能引起 EPS / TD —— Metoclopramide 是"伪装成胃药的抗多巴胺药"，跟 haloperidol 同通路。**

```
Metoclopramide 阻断 D2 受体
        │
   ┌────┴────────┐
   ▼             ▼
延髓 CTZ 的 D2  纹状体的 D2
(化学感受器     (运动控制中心)
触发区)             │
   │               ▼
止吐作用 ✓      EPS / Tardive Dyskinesia
                （穿 BBB → 中枢副作用）
```

### Metoclopramide 4 类临床场景

| 场景 | 角色 | 风险 |
|---|---|---|
| 胃轻瘫（diabetic gastroparesis） | 促胃肠动力一线 | ⚠️ **FDA 黑框：限 ≤12 周** |
| 化疗止吐 | 备选（首选 ondansetron） | EPS 风险 |
| 术后止吐（PONV） | 备选 | EPS 风险 |
| 偏头痛伴呕吐 | 急诊单次 IV | 急性 dystonia 风险（[[mistakes/uworld-mistakes_2026-05#^Q2505]] 类型） |

### Metoclopramide 高危人群（更易出 EPS / TD）
- **老年**（中枢 DA 储备低）
- **女性**（雌激素影响 D2 敏感性）
- **糖尿病**（合并自主神经病变）
- **长期 / 高剂量使用**（≥ 12 周显著升 TD）
- **合用其他 D2 拮抗药**（FGA / SGA）→ 累加效应

### 类似的"隐藏 D2 拮抗止吐药"

| 药 | 表面用途 | 隐藏机制 | 风险 |
|---|---|---|---|
| **Prochlorperazine** ⭐ | 止吐 / 偏头痛 | 吩噻嗪类，强阻 D2 | 急性肌张力障碍（[[复盘/2026-05-23_浩斯宾判分_v1#Q11]] 触发题）|
| **Promethazine** | 止吐 / 抗组胺 / 镇静 | 阻 D2 + H1 | EPS（较少）/ 嗜睡 / 老年镇静过度 |
| **Droperidol** | 术后止吐 / 急诊镇静 | 强阻 D2 | EPS + **QT 延长**（FDA 黑框）|

## 四、Ondansetron vs Metoclopramide ⭐⭐ 双星对比（最易混的一对）

| 维度 | **Ondansetron** ✅ | **Metoclopramide** ⚠️ |
|---|---|---|
| **受体** | **5-HT3 拮抗** | **D2 拮抗** |
| **解剖作用点** | CTZ + 肠道迷走神经末梢 | CTZ + 纹状体（额外） |
| **化疗止吐** | **一线 ⭐**（最强 + 最安全） | 备选 |
| **术后止吐** | **一线 ⭐** | 备选 |
| **胃轻瘫** | ❌（无促动力作用） | **一线**（促动力 + 止吐双效）|
| **EPS 风险** | **无 EPS** ✅ | **有 EPS / TD** ⚠️ |
| **QT 延长** | **有 ⚠️**（剂量依赖，低钾镁 / 老 QT 长慎用） | 较少 |
| **其他副作用** | 便秘 / 头痛 | 镇静 / 腹泻 / 高泌乳素（D2 阻断 tuberoinfundibular 通路）|
| **FDA 黑框** | 高剂量 ≥ 32 mg IV 时 QT 风险（已撤回该剂量）| **≤12 周限制**（TD 风险）|

> [!tip] 一句话区分本质
> **"昂丹司琼名字有'昂'，副作用也'安'（安全无 EPS）；胃复安阻 D2，反而'不安'（出 EPS）。"**
>
> **考题信号**：用止吐药后冒出 tic 样 / 口面部动作 / 斜颈 / oculogyric crisis → **立刻锁 D2 类**（metoclopramide / prochlorperazine / promethazine / droperidol），排除 ondansetron。

## 五、场景秒判口诀 ⭐（USMLE 高频）

| 场景 | 一线药 | 二线药 | 反向钩 |
|---|---|---|---|
| **化疗止吐**（急性期） | **Ondansetron**（5-HT3）| Dexamethasone 联用 | 化疗 + 任何选项有 ondansetron → 选它 |
| **化疗止吐**（延迟期 24-72h）| **Aprepitant + ondansetron + dex** 三联 | — | "化疗 2-3 天后仍恶心" = NK1 加进来 |
| **术后止吐（PONV）** | **Ondansetron** | Dexamethasone / Scopolamine 贴片 | 高危人群（女 / 不抽烟 / 既往晕动）术前预防贴 scopolamine |
| **晕动病** | **Scopolamine 贴片**（耳后，72h） / Meclizine | Promethazine | "邮轮 / 长途车" → 贴片（长效）；短期口服 meclizine |
| **妊娠呕吐**（NVP / HG） | **Doxylamine + B6**（Diclegis）⭐ 一线 | Ondansetron（慎用早孕，可能心脏致畸争议）/ Meclizine（B 类） | 先 B6 ± doxylamine，再升级 |
| **胃轻瘫**（糖尿病 / 术后 / 特发性） | **Metoclopramide**（≤12 周 ⚠️）| Erythromycin（短期 motilin 激动）| 长期用 + 高剂量 + 老年 → TD 风险 |
| **偏头痛伴呕吐**（急诊） | **Prochlorperazine** IV / **Metoclopramide** IV | + diphenhydramine 预防 EPS | 给完后冒出斜颈 → 急性 dystonia → benztropine ⭐ Q11 触发题 |
| **化疗预期性恶心**（条件反射式）| **Lorazepam**（BZD） | — | 心因性 / 条件反射 → BZD 不是 5-HT3 |
| **CINV 难治** | 加 **olanzapine**（5 mg）— 多受体阻断（D2 + 5-HT3 + H1 + M1） | — | "难治化疗止吐 + 选项有 olanzapine" → SGA 多受体一击多得 |

## 六、易混陷阱（普适）

> [!warning] 高频陷阱
> - **"化疗止吐 = 选最强的"反射 ❌**：化疗一线 = ondansetron（5-HT3），不是 metoclopramide。Metoclopramide 太弱 + 有 EPS 风险，**不是化疗止吐首选**。
> - **"止吐药都安全"反射 ❌**：D2 拮抗类（metoclopramide / prochlorperazine / promethazine / droperidol）会引起 EPS / TD / 高泌乳素血症。任何长期使用 + 老年 / 女性 / DM → 警惕。
> - **"Ondansetron 完全无副作用"反射 ❌**：5-HT3 拮抗类**有 QT 延长**（特别 ondansetron），低钾低镁 / 既往 QT 长 / 用其他 QT 药（macrolide / fluoroquinolone / methadone）→ 先查 ECG。
> - **"晕动用 ondansetron"反射 ❌**：5-HT3 拮抗**不治晕动**（晕动走 H1/M1 前庭通路）。晕动用 scopolamine 贴片 / meclizine / diphenhydramine。
> - **"妊娠呕吐用 ondansetron"反射 ⚠️**：早孕期 ondansetron 与心脏畸形 / 唇腭裂的相关性有争议，**一线是 doxylamine + B6**（Diclegis，FDA 批 NVP）。Ondansetron 留给一线失败 + 充分知情后用。
> - **"Aprepitant 单用治化疗"反射 ❌**：NK1 拮抗**只覆盖延迟期**，急性期还需要 ondansetron + dex 联用。Aprepitant 不能单打独斗。
> - **"Promethazine 是抗组胺所以无 EPS"反射 ❌**：Promethazine 是**吩噻嗪类**（与 chlorpromazine 同家族），有 D2 拮抗作用 → **仍会出 EPS**（虽较 metoclopramide 少）。看到 promethazine 在止吐题中要提防 EPS 风险，特别儿童（FDA 警告 <2 岁禁用）。
> - **TD 治疗"加抗胆碱"反向陷阱**：metoclopramide / prochlorperazine 引起的**急性 dystonia 用抗胆碱（benztropine）**；但**慢性 TD 例外**：抗胆碱不仅无效还加重，要**停药 + VMAT2 抑制剂（valbenazine）**。详见 [[完整笔记/专题笔记/psych/_衍生_四大EPS诊断治疗对照]]。
> - **Olanzapine 化疗用法 ≠ 精神病用法**：化疗止吐用低剂量（5 mg）olanzapine 多受体阻断；不要被 stem "olanzapine 用于化疗" 干扰认为是精神病题。

## 七、Memory Hook

> [!success] 记忆挂钩
> - **一句话锁定**：**"止吐药按受体分 5 类：D2 出 EPS、5-HT3 出 QT、NK1 配化疗延迟期、H1/M1 治晕动。看场景选受体，看副作用反推机制。"**
> - **昂丹安全 vs 胃复安不安钩**：**"昂丹司琼名字有'昂'，副作用也'安'（安全无 EPS）；胃复安（metoclopramide 中文名）阻 D2，反而'不安'（出 EPS）。"** 一字反差锁两药身份。
> - **三个场景秒判口诀**：
>   - **化疗 / 术后止吐 → ondansetron（5-HT3）安全王，首选**
>   - **止吐药后出现锥体外系症状 → 凶手必是 D2 类**（metoclopramide / prochlorperazine / promethazine / droperidol），不是 ondansetron
>   - **晕车晕船 → 抗胆碱 / 抗组胺**（scopolamine 贴片 / meclizine / diphenhydramine），前庭走 H1 / M1
> - **场景 → 受体反射**：
>   - 化疗 / 术后 → **5-HT3**（迷走 + CTZ）
>   - 晕动 → **H1 / M1**（前庭核）
>   - 化疗 24h 后 → **NK1**（中枢 P 物质）
>   - 胃轻瘫 → **D2**（兼促动力）
>   - 妊娠呕吐 → **B6 + doxylamine**（H1 + 维生素）
> - **D2 阻断三重副作用钩（穿 BBB）**：
>   - **纹状体 D2** → EPS（急性 dystonia / akathisia / parkinsonism / TD）
>   - **CTZ D2** → 止吐作用 ✓
>   - **Tuberoinfundibular D2** → 高泌乳素血症（溢乳 / 闭经 / 男乳）
>   - **三个位置一起阻** = 治疗作用 + 两个副作用一起来
> - **化疗止吐三联记忆**：**"5-HT3 + dex + NK1 = 化疗止吐铁三角"**。急性期 ondansetron + dex；延迟期加 aprepitant。难治加 olanzapine（SGA 多受体）。
> - **Metoclopramide 12 周钩**：**"胃复安 12 周封顶"** — FDA 黑框，超 12 周 TD 风险显著升。

# 继发性高血压鉴别框架（v2 草稿）

> [!info] 笔记定位
> 这是**底层框架笔记**，目标：所有"年轻人 HTN / 抗药性 HTN / 伴特殊线索 HTN"题都能从**线索 → 病因 → 检查**三步推出来，不靠死记。
> 后续等错题积累后，由 Claude Code 整合成正式衍生笔记 `_衍生_继发性高血压鉴别决策树`。
>
> **v2 更新（2026-05-16）**：整合 Q3933（做对的题）— resistant HTN 形式定义 + Atherosclerotic RAS 完整线索清单（diffuse atherosclerosis 触发 / 99% specificity continuous bruit / flash pulm edema / asymmetric kidney / ACE>30% Cr / interarm BP discrepancy → subclavian stenosis 干扰项）

---

## 一、为什么需要这个框架？

> [!danger] 核心原则
> **HTN 在 <30 岁人群中，"primary（原发性）"的概率显著下降，必须主动筛查继发原因。**
>
> 一般人群：~95% primary HTN，~5% secondary
> **<30 岁人群**：secondary 比例显著升高 → 必须主动查

USMLE 不会让你"忘记查继发"——只要有红旗，正确答案永远是**先查病因**，不是"start lisinopril"。

---

## 二、触发"查继发"的 5 个红旗 ⭐ 必背

```
Red Flags → 怀疑 Secondary HTN
────────────────────────────────────────
🚩 1. 发病年龄 <30 岁（无家族史）
🚩 2. Severe / malignant HTN (≥180/120)
🚩 3. Resistant HTN（3 种足量降压药仍控不住）
🚩 4. 已控制良好的 HTN 突然恶化
🚩 5. 伴有提示特定病因的症状 / 体征
       （tremor、低钾、阵发性头痛、打鼾、腹部杂音等）
```

> [!tip] 红旗五字诀
> **"幼、急、抗、变、伴"**
> - **幼**：<30 岁
> - **急**：severe / malignant（≥180/120）
> - **抗**：3 种药抗药
> - **变**：原本稳定突然恶化
> - **伴**：伴特异性症状/体征

> [!warning] Resistant HTN 形式定义 ⭐（必背 — Q3933 触发增量）
> **Resistant HTN = 持续高血压 despite ≥3 类不同降压药（其中至少 1 类必须是利尿剂）足量使用**
>
> 满足该定义 → **强制评估 secondary causes**（即使没有其他红旗）
>
> Q3933 stem 范例：metoprolol（β-blocker）+ valsartan（ARB）+ HCTZ（thiazide 利尿）+ amlodipine（CCB）+ atorvastatin 仍 BP 180/100 → **textbook resistant HTN** → 必查继发

> [!success] 一句话锁定
> 看到 **"<30 岁 + HTN + 任何额外线索"** → **不要选 lifestyle modification / start antihypertensive**，要找**继发病因**。
> 看到 **"≥3 类降压药 + 含利尿剂 + 仍不控"** → 同样不要加药，先**查继发**。

## 三、继发性 HTN 病因鉴别表 ⭐ 核心框架

按"系统分类"整理，每类有**触发线索 → 首选检查**配对：

### 🔴 药物/毒物（最常被忽视）

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **Cocaine / 安非他明 / MDMA** | 年轻、**鼻出血**、震颤、瞳孔↑、躁动、心动过速 | ⭐ **Urine drug screen** |
| **OCP**（口服避孕药） | 女性、用药史 | 停药观察 |
| **NSAIDs / 类固醇 / 减充血剂** | 用药史 | 停药试验 |
| **酒精**（慢性大量） | 饮酒史、MCV↑、AST/ALT>2:1 | 临床 + LFT |

### 🫘 肾脏

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **Fibromuscular dysplasia (FMD)** ⭐ | **年轻女性**、腹部杂音、对称性 HTN | **Renal Doppler US** / CTA / MRA |
| **Atherosclerotic RAS** ⭐ | **老年 + diffuse atherosclerosis**（CAD/PAD/DM/吸烟 4 件套）+ **continuous (systolic-diastolic) abdominal bruit** at 上腹/脐周/侧腹（99% 特异 ⭐）+ **recurrent flash pulmonary edema** + **asymmetric kidney size**（小的萎缩单侧肾）+ **ACE/ARB 后 Cr ↑ >30%** | Doppler US / CTA / MRA |
| 肾实质性（CKD / GN / PKD） | Cr↑、蛋白尿、血尿、家族史（PKD） | UA + Cr + 肾 US |

### 🩸 内分泌

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **Primary aldosteronism (Conn)** | **低钾**、代谢性碱中毒、肌无力、夜尿、**抗药性 HTN** | **Aldosterone/Renin ratio (ARR)** |
| **Pheochromocytoma** | **阵发性**头痛+心悸+出汗三联征、HTN paroxysms | 24h 尿 / 血浆 **metanephrines** |
| **Cushing syndrome** | 满月脸、向心性肥胖、紫纹、易瘀、肌无力 | 24h 尿游离皮质醇 / 1mg 地塞米松抑制 |
| **甲亢 / 甲减** | 甲亢：tremor、心悸、体重↓；甲减：迟钝、体重↑ | TSH |
| 原发性甲旁亢 | 高钙、肾结石、骨痛 | Ca、PTH |

### 🌬️ 睡眠

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **OSA** | **肥胖**、打鼾、白天嗜睡、晨起头痛 | **In-lab polysomnography** |

### 🫀 血管/结构

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| **Coarctation of aorta** | **上肢 HTN + 下肢低血压**、股动脉搏弱、上下肢血压差 >20、肋间凹征 | Echo / CT/MR aortography |

### 🤰 妊娠

| 病因 | 关键 Stem 线索 | 首选检查 |
|---|---|---|
| Preeclampsia / 妊娠 HTN | 孕 >20w、蛋白尿、水肿 | 尿蛋白 + BP 监测 |

## 四、决策树 — 年轻人 HTN 怎么查？⭐

```
年轻人新发 HTN (<30 岁)
        │
        ▼
┌─────────────────────────────────┐
│ 1. 详细询问病史                  │
│    - 药物（处方/OTC/违禁）       │
│    - 家族史                       │
│    - 饮酒 / 吸烟                  │
│    - 睡眠 / 打鼾                  │
│    - 阵发性症状                  │
│    - 月经史 / OCP                │
└─────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────┐
│ 2. 体格检查 找特异线索            │
│    - 四肢血压(→ coarctation）   │
│    - 腹部杂音（→ RAS / FMD）     │
│    - 库欣体征                    │
│    - 甲亢体征（tremer / 突眼）   │
│    - 瞳孔大小 / 鼻黏膜（→ 毒品） │
└─────────────────────────────────┘
        │
        ▼
┌─────────────────────────────────┐
│ 3. 基础实验室                    │
│    - TSH                         │
│    - 电解质（K↓ → 醛固酮增多） │
│    - Cr / UA                     │
│    - 血糖 / 血脂                │
└─────────────────────────────────┘
        │
        ▼
   根据线索做"针对性"检查
        │
        ├─ 毒品线索 (tremor + 鼻出血 + 瞳孔↑) → ⭐ Urine drug screen
        ├─ 低钾 / 抗药 → ARR (醛固酮/肾素比)
        ├─ 阵发性三联征 → 血/尿 metanephrines
        ├─ 年轻女性 + 腹部杂音 → Renal Doppler (FMD)
        ├─ 库欣体征 → 24h 尿皮质醇 / DST
        ├─ 肥胖 + 打鼾 → Polysomnography
        └─ 上下肢血压差 → Echo / CT (coarctation)
```

## 五、USMLE 经典鉴别陷阱

### 🪤 FMD vs Cocaine（同样是年轻人 HTN）

| 线索 | 倾向 FMD | 倾向 Cocaine |
|---|---|---|
| 性别 | **女性**为主 | 任一 |
| 腹部杂音 | ✅ 有 | ❌ 无 |
| 鼻出血 / 鼻黏膜损伤 | ❌ 无 | ✅ 有 |
| Tremor / 瞳孔↑ / 躁动 | ❌ 无 | ✅ 有 |
| 首选检查 | Renal Doppler | Urine drug screen |

### 🪤 Primary aldosteronism vs Pheo（都是内分泌 HTN）

| 线索 | Conn | Pheo |
|---|---|---|
| HTN 模式 | **持续性** + 抗药 | **阵发性** + episodes |
| 低钾 | ✅ 经典 | ❌ 不典型 |
| 三联征（头痛+心悸+汗） | ❌ | ✅ |
| 首选检查 | ARR | metanephrines |

### 🪤 Atherosclerotic RAS vs FMD（同样是肾血管性）

| 线索 | Atherosclerotic RAS | FMD |
|---|---|---|
| 年龄 | **老年**（>55） | **年轻女性**（15-50） |
| 危险因素 | 广泛 ASCVD（CAD / PAD） | 通常无 |
| 病变部位 | **近端**肾动脉 / 主动脉开口 | **远端 2/3** 肾动脉（串珠样 "string of beads"） |
| ACEi 试验 | 启动后 Cr↑↑（>30%） | 同样需警惕 |

### 🪤 Atherosclerotic RAS 完整诊断线索 + Q3933 干扰项学习 ⭐⭐（v2 增量）

> [!success] Atherosclerotic RAS 6 大诊断线索（**任意 1 个**即应启动 Doppler/CTA）
> 1. **Resistant HTN**（≥3 类降压药 + 1 利尿剂仍不控）
> 2. **Diffuse atherosclerosis 4 件套**：CAD（angina/stent）+ PAD（间歇性跛行）+ DM + 吸烟史 — **PAD 患者中 45% 有 RAS**
> 3. **Continuous (systolic-diastolic) abdominal bruit** at 上腹/脐周/侧腹 → **99% specificity** ⭐（注意：仅 systolic bruit ≠ 99% 特异，必须是**横跨收缩+舒张**的连续杂音）
> 4. **Recurrent flash pulmonary edema**（反复急性肺水肿，找不到 LV 系统原因）
> 5. **Asymmetric kidney size**（影像见单侧肾萎缩，缺血侧 < 健侧 >1.5 cm）
> 6. **ACE inhibitor / ARB 启动后 SCr 升高 >30%**（从 baseline）

> [!info] Q3933 stem 经典模式
> 65 岁男 + CAD（angioplasty + stent）+ **间歇性跛行**（右大腿疼，走路加重休息缓解 = PAD）+ DM 10y + 吸烟 1.5 PPD + HTN 20y 用 4 类降压药仍 180/100
> → 全套 diffuse atherosclerosis 触发 → **下一步发现 = continuous bruit in periumbilical area**（renovascular disease 的体格检查标志）

> [!warning] B/C/D/E 干扰项学习要点（Q3933）
> | 选项 | 错在哪 | 正确情境 |
> |---|---|---|
> | **B. Aldo/renin ratio >20:1** | 是 **primary hyperaldosteronism (Conn)** 标志；本题 diffuse atherosclerosis 更指向 RAS；且患者用 ARB（valsartan）→ ARB 升高 PRC，让 ratio 解读复杂化（但 **low PRC 仍提示 Conn**）| 低钾 + 抗药 HTN + 无 ASCVD |
> | **C. ↑ 24h 尿游离皮质醇** | Cushing — 应有满月脸 / 向心性肥胖 / 紫纹 / 肌弱 / 皮薄 / 神经精神症状（本题无）| Cushing 体征 |
> | **D. ↑ 尿 VMA** | Pheo — 应有**阵发性头痛+心悸+出汗**三联（本题持续 HTN 无阵发）| 阵发三联 |
> | **E. Rib notching on CXR** ⭐ | **CoA** — 本题双臂 BP 几乎相等（177 vs 180，差仅 3 mmHg）**不达 CoA 标准**（需上下肢差 >20 mmHg）；CoA 也通常**儿童**起病，成人 CoA 罕见 | 上下肢 BP 差 + 儿童 / 年轻成人 + headache/epistaxis |

> [!danger] 新干扰项警告：**Interarm BP discrepancy >10 mmHg → Subclavian stenosis** ⭐
> 不是 CoA（CoA 是**上下肢**差，不是**双臂间**差）
> - 双臂间 BP 差 ≤10 mmHg = **正常变异**
> - 双臂间 BP 差 **>10 mmHg** = **Subclavian artery stenosis**（动脉粥样硬化最常见）→ 与 PAD/CAD 同族
> - **CoA 的标志**：**上肢 BP > 下肢 BP**（不是双臂差），加 **brachial-femoral pulse delay** + 儿童期发现 + CXR rib notching + "3 sign"
>
> Q3933 patient 双臂 177 vs 180 = 差 3 mmHg = **nonsignificant**，所以选项 E（rib notching/CoA）排除。

## 六、可卡因急性中毒 — 心血管急症 ⭐

可卡因不只是 HTN 病因，还是 USMLE **年轻人心血管急症**必考考点：

```
可卡因急性中毒 → 心血管急症五连
─────────────────────────────────
1. 高血压急症 / 颅内出血
2. ⭐ 急性心肌缺血 / MI（年轻人 MI 必查可卡因！）
3. 主动脉夹层
4. 致命性心律失常
5. 癫痫
```

> [!danger] 治疗陷阱 — 可卡因相关 MI / HTN
> **绝对禁用：β-blocker 单用**（α-受体无对抗 → 加重血管收缩 → unopposed α effect）
> **一线**：苯二氮䓬类 (benzodiazepines) + nitrates / CCB
> 如必须用 β-blocker → 用 **labetalol**（α/β 双阻断）

## 七、Memory Hook ⭐

> [!success] 一句话
> **"年轻人高血压，先查为什么 — 别急着开降压药"**

> [!info] 继发病因首字母口诀
> **"CHAPS + RFC"**
> - **C**ocaine / 可卡因
> - **H**yperaldosteronism (Conn)
> - **A**pnea (OSA)
> - **P**heochromocytoma
> - **S**teroids / Cushing
> + **R**enal artery（FMD / RAS）
> + **F**ibromuscular dysplasia（特别提，年轻女性）
> + **C**oarctation of aorta

> [!tip] 线索 → 检查 映射
> - **鼻出血 + tremor** → Urine drug screen
> - **低钾 / 抗药** → ARR
> - **阵发三联征** → metanephrines
> - **年轻女性 + 腹部杂音** → Renal Doppler
> - **肥胖 + 打鼾** → Polysomnography
> - **上下肢压差** → Echo / CT
> - **库欣体征** → 24h 尿皮质醇 / DST
> - **TSH 异常** → 已查

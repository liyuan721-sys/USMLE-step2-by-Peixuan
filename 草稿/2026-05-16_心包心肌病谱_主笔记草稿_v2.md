## 一、心包病谱总览

```
急性 Pericarditis
    │ (病毒/自身免疫/尿毒症/post-MI/恶性)
    │ 症状：胸痛 (pleuritic, 前倾↓, 平卧↑)
    │ 体征：pericardial friction rub
    │ ECG：diffuse ST elevation + PR depression
    │ 治疗：NSAIDs + 秋水仙碱
    │
    ▼ (液体积累)
Pericardial Effusion
    │ 症状：nonspecific (dyspnea, fatigue, chest fullness)
    │ CXR：心影大 + 肺野清
    │ 首选：Echo
    │ 治疗：取决于病因 + 量 + 血流动力学
    │
    ▼ (液体↑↑ 或速度快)
Cardiac Tamponade
    │ 症状：休克前兆 (dyspnea, presyncope, fatigue)
    │ 体征：Beck triad + Pulsus paradoxus
    │ Echo：RA/RV collapse + IVC plethora
    │ 治疗：IV fluids + Pericardiocentesis
    │
    ▼ (机化 / 长期纤维化)
Constrictive Pericarditis
    │ 症状：慢性右心衰 (edema, ascites, JVD)
    │ 体征：Kussmaul + Pericardial knock
    │ 影像：心包钙化 + 增厚 (CT/MRI)
    │ 治疗：Pericardiectomy（开胸切心包）
```

## 二、各阶段疾病详解

### 1. Acute Pericarditis（急性心包炎）

**病因**：
- 病毒性（最常见，柯萨奇/腺病毒）
- 自身免疫（SLE, RA）
- 尿毒症
- Post-MI（早期：~3 天；晚期 Dressler：2-10 周）
- 恶性
- 结核
- 化脓性（细菌）

**临床三联**：
1. 胸痛（pleuritic, **前倾减轻**, 平卧加重）⭐
2. **Pericardial friction rub**（3 段, 体位敏感）
3. **ECG diffuse ST elevation + PR depression** ⭐
4. ± Pericardial effusion

**治疗**：
- 第一线：NSAIDs + 秋水仙碱（colchicine）
- 难治：泼尼松
- 化脓性：抗生素 + 引流
- 结核性：抗结核 + 长期监测（防 constrictive）

---

### 2. Pericardial Effusion（心包积液）

**关键 CXR 特征**：
- 心影增大（"water bottle" heart）
- **肺野清** ⭐⭐（核心鉴别点）

> [!danger] 心影大鉴别表 ⭐⭐
> | 病因 | 起病 | 肺野 | 啰音 | 水肿 | CXR 其他 | 首选下一步 |
> |---|---|---|---|---|---|---|
> | **CHF** | 慢性+急性 | **淤血/水肿** | 有 | 有 | Kerley B, cephalization | BNP + Echo |
> | **Pericardial effusion** | 亚急性 | **清** | 无 | 无 | Water bottle 心影 | **Echo** |
> | **Cardiac tamponade** | 急性 | **清** | 无 | ±JVD | + Beck triad | **Echo → 穿刺** |
> | **Dilated CM** | 慢性 | 可清可淤 | ± | ± | 全心扩大 | Echo |

**Pericardiocentesis 适应症决策树** ⭐：

```
发现 Pericardial Effusion
        │
        ▼
   有 tamponade?
   │            │
  YES           NO
   │            │
   ▼            ▼
 急诊穿刺    病因明确?
              │
       ┌──────┴──────┐
       ▼             ▼
   恶性/化脓/结核    病毒/特发/尿毒/AI
       │             │
       ▼             ▼
   诊断性穿刺      治原发病 + 随访
   (cytology/      (不穿刺)
    culture)
```

**禁忌穿刺的情况**：
- **主动脉夹层** → tamponade（心包血肿是止血塞子，穿刺致死）⭐
- **MI 后心室破裂**（5-14 天）→ 直接 OR
- **创伤伴血凝块**（穿刺抽不出）

---

### 3. Cardiac Tamponade（心包压塞）⭐

**核心病理生理**：
- 心包内压 > 心室舒张压 → 心室充盈受限
- preload↓ → SV↓ → CO↓ → 休克
- **急性**：100-200 mL 即可压塞（外伤/夹层/MI 破裂）
- **慢性**：1-2 L 才压塞（恶性/尿毒症，心包代偿）

**Beck Triad（必背）** ⭐：
1. **Hypotension**
2. **JVD / Distended neck veins**
3. **Muffled heart sounds**

⚠️ 完整三联仅 ~30%，不是诊断金标准

**Pulsus Paradoxus** ⭐⭐（必考）：
- 定义：吸气时 **SBP 下降 >10 mmHg**
- 机制：吸气 → 静脉回流 RV↑ → RV 被心包限制不能外扩 → **室间隔左移** → LV 充盈↓ → SBP↓↓
- **Tamponade 无 pulsus 的 4 种情况**：
  - AR（主动脉瓣关闭不全）
  - ASD（房间隔缺损）
  - 严重 LV dysfunction
  - 严重肺动脉高压

**ECG 三联**：
- Low QRS voltage
- **Electrical alternans** ⭐（心脏在液体中摇摆，QRS 振幅逐拍交替）
- Sinus tachycardia

**Echo 评估四要素** ⭐：
1. 确认 effusion
2. **RA collapse**（舒张期，最敏感）
3. **RV collapse**（舒张早期，最特异）
4. **IVC plethora**（吸气塌陷消失）

**处理三步**：
1. **支持**：IV fluids（升 preload）
   - ❌ 禁忌：Diuretics / Vasodilators / β-blockers（均降 preload，致命）
2. **解除压塞**：Pericardiocentesis（Echo 引导, 剑突下入路）
3. **防复发**：恶性 → pericardial window / 长期 catheter

**特殊情况处理**：

| 病因 | 处理 |
|---|---|
| 创伤性 | 穿刺救命 → OR 修补 |
| **主动脉夹层** | ❌ **绝不穿刺** → 直接 OR ⭐ |
| Post-MI 心室破裂 | 直接 OR（不穿刺） |
| 恶性 | 穿刺 + cytology + window |
| 化脓性 | 穿刺 + 抗生素 + 外科引流 |

---

### 4. Constrictive Pericarditis（缩窄性心包炎）

**病因**：
- 结核（发展中国家最常见）⭐
- 放疗（乳腺/淋巴瘤治疗后）⭐
- CABG 后
- 病毒性心包炎慢性化
- 自身免疫
- 特发性

**临床三联**：
1. **Kussmaul sign** ⭐（吸气 JVP 反升）
2. **Pericardial knock** ⭐⭐（最特异！）
   - 舒张早期, 高频, 膜面听
   - 机制：心室快速充盈撞硬壳
3. 慢性右心衰（edema, ascites, JVD）

**影像**：
- CXR/CT：**心包钙化 + 增厚 >4 mm** ⭐
- Echo：心包增厚 + 间隔反弹（septal bounce）
- MRI：评估心包

**心导管特征** ⭐：
- **压力均衡**：RA = RV diastolic = PCWP = LV diastolic（差 <5 mmHg）
- **Square root sign / Dip-and-Plateau**（RV 压力曲线）⭐⭐
- **Discordant ventricular pressures**（吸气时 RV↑ LV↓，反向变化）

**治疗**：**Pericardiectomy**（开胸切除心包，治愈性）

## 三、限制性心肌病（Restrictive Cardiomyopathy）

### 1. 概念
- 心**肌**本身僵硬（不是心包）
- 心包正常厚度
- **LV 受累 >> RV 受累**（壁厚不同）

### 2. 病因
| 病因 | 关键线索 |
|---|---|
| **Amyloidosis** ⭐ 最常考 | 老年男性, 舌大, MM, Congo red 苹果绿双折光 |
| **Sarcoidosis** | 年轻黑人女性, BHL, 葡萄膜炎, 室性心律失常 |
| **Hemochromatosis** | "Bronze diabetes"（铁沉积+糖尿病+肝硬化）, 铁蛋白↑↑↑ |
| **Endomyocardial fibrosis** | 热带, 嗜酸细胞↑ |
| **Löeffler endocarditis** | 嗜酸性粒细胞增多综合征 |
| **Radiation** | 同时可引起 Constrictive |
| **Glycogen storage disease** | Pompe (II 型, 儿童) |

### 3. Cardiac Amyloidosis（USMLE 高频）⭐⭐

**ECG 三联**：
1. **Low voltage** ⭐
2. **Pseudoinfarct Q 波**（V1-V3）⭐
3. **AV 阻滞**（PR 延长 / 1°-3° AVB）

**Echo**：LV 壁厚 + speckled / granular sparkling pattern

**经典反差** ⭐⭐：
> **"Echo 壁厚 + ECG low voltage"** = Amyloid
> （壁厚不是真心肌肥厚，是淀粉样物取代心肌）

**确诊**：心内膜活检 + Congo red 阳性 + 苹果绿双折光

**类型**：
- **AL Amyloid**：轻链型，与 MM/MGUS 相关
- **ATTR Amyloid**：运铁蛋白型，老年男性 / 家族性

## 四、三大鉴别终极对比表 ⭐⭐⭐

### 1. Tamponade vs Constrictive vs Restrictive

| 特征 | **Tamponade** | **Constrictive** | **Restrictive CM** |
|---|---|---|---|
| **病变** | 心包液体 | 心包纤维化 | 心肌纤维化/浸润 |
| **起病** | 急性/亚急性 | 慢性 | 慢性 |
| **病因** | 恶性/外伤/特发 | 结核/放疗/CABG | Amyloid/Sarcoid/血色 |
| **Pulsus paradoxus** | **(+)** ⭐ | (−) | (−) |
| **Kussmaul sign** | (−) | **(+)** | (+) |
| **Pericardial knock** | (−) | **(+)** ⭐ 最特异 | (−) |
| **S3 gallop** | (−) | (−) | **(+)** |
| **JVP 波形** | X 谷深, Y 消失（**单谷**） | X+Y 都深（**M/W 双谷**） | M/W 形 + 大 a |
| **RV 压力曲线** | 平滑 | **Square root** ⭐ | Dip-plateau（轻） |
| **压力均衡时机** | **整个舒张期** | **舒张中晚期** | 部分 |
| **RV-LV diastolic 差** | <5 mmHg | <5 mmHg | **>5 mmHg (LV>>RV)** ⭐ |
| **呼吸变化** | Pulsus (+) | **Discordant**（RV↑LV↓）⭐ | Concordant（同向） |
| **心包厚度（CT/MRI）** | 正常 + 积液 | **>4 mm + 钙化** ⭐ | 正常 |
| **心肌活检** | 正常 | 正常 | **异常**（Congo red+ 等） |
| **BNP** | 中度↑ | **较低 (<200)** ⭐ | **显著↑ (>800)** ⭐ |
| **首选诊断** | Echo | CT/MRI（心包） | Echo + 心肌活检 |
| **治疗** | **Pericardiocentesis** | **Pericardiectomy** | 治原发病 |

### 2. 关键鉴别记忆
- **C vs T**：起病速度 + Pulsus vs Knock + 心包钙化 + RV 曲线
- **C vs R**：心包 vs 心肌 + BNP 高低 + LV-RV diastolic 差
- **R vs T**：起病速度 + BNP + 心室壁厚度

## 五、JVP 波形完整解析（深度专题）⭐⭐⭐

> [!warning] 学习目标
> JVP 是 USMLE 心内体格检查最高频考点。本章节目标：
> 1. 理解 JVP **本质**（为什么颈静脉能反映右心房）
> 2. 推导每个波段**为什么这样**（不死记）
> 3. 看到异常波形能**立刻锁定疾病**
> 4. 床旁能正确**视诊 + 测量**

---

### 5.1 JVP 的本质 — 为什么颈静脉能反映 RA 压？

> [!info] 核心解剖
> 颈内静脉 → 头臂静脉 → 上腔静脉 → **右心房**
> **它们之间没有瓣膜！** ⭐
> 所以右心房的压力变化**直接传导**到颈内静脉
> → 颈静脉的"搏动"= 右心房压力的"图形"

```
头部
  │
颈内静脉（IJV）  ← 我们观察的位置
  │   ↑
  │   没有瓣膜（透明的"压力传导管"）
  ▼   ↓
头臂静脉
  │
上腔静脉（SVC）
  │
右心房（RA）  ← 压力源
```

**临床意义**：
- 颈静脉 = RA 压力的"实时显示屏"
- JVP 高度 = RA 压力大小
- JVP 波形 = RA 压力随心动周期的变化

**为什么不用颈外静脉（EJV）？**
- EJV 有瓣膜
- 走行有角度
- 容易扭曲压力传导
- IJV（右侧）最准确（更直接接 SVC）

---

### 5.2 JVP 5 个波段的推导（从心动周期理解）⭐⭐

**核心原则**：JVP 反映的是 **RA 内的压力变化**
RA 压力变化由 4 件事决定：
1. 心房自己收缩（推压力↑）
2. 心房舒张（压力↓）
3. 三尖瓣开/关（决定 RA 能不能排空到 RV）
4. 静脉血回流到 RA（填充压力↑）

```
心动周期时序：
P 波 → QRS → T 波 → 下一个 P 波
心房收缩 → 心室收缩 → 心室舒张 → 心房充盈
```

**5 个波段对照**：

| 波段 | ECG 时机 | 发生什么 | RA 压力变化 |
|---|---|---|---|
| **a 波** | P 波之后 | **心房收缩**推血进 RV | ↑（压力波）|
| **c 波** | QRS 之后 | 心室收缩→三尖瓣**关闭并鼓出**进 RA | ↑（小波）|
| **x 谷** | ST 段 | 心房**舒张** + 三尖瓣环下移（向心尖）| ↓（深谷）|
| **v 波** | T 波之后 | 心房**充盈**（静脉血回流，但瓣还关）| ↑（压力波）|
| **y 谷** | T 波末 | **三尖瓣开放** → RA 血涌入 RV | ↓（深谷）|

> [!tip] 一句话推导
> - **a 波**：心房收缩（推血）→ 压力↑
> - **c 波**：瓣关时鼓出（小干扰）→ 压力小↑
> - **x 谷**：心房放松 + 瓣环下移（变大 → 压力↓）→ 深↓
> - **v 波**：静脉血回流填满 RA（瓣还关）→ 压力↑
> - **y 谷**：瓣开 → 血排走 → 压力↓

### 5.3 JVP 波形完整图解

```
ECG: ── P ─── QRS ── T ──── P ──
        │     │     │       │

JVP波形:    a         v
            ╱╲       ╱╲
           ╱  ╲ c   ╱  ╲
          ╱    ╲╱╲ ╱    ╲
         ╱     ╳  X      ╲
        ╱      x谷         ╲___
                            y谷

时相含义：
  a波   = 心房收缩（P 波刚过）
  c波   = 三尖瓣关并鼓出（QRS 后）
  x谷   = 心房松弛 + 瓣环下移
  v波   = 心房充盈（T 波时）
  y谷   = 三尖瓣开放放血进 RV
```

> [!info] 顺序记忆口诀
> **A-C-X-V-Y**
> "**a**ctive contraction → **c**losing valve bulge → re**x**lation → **v**enous filling → **y**ielding valve opens"

### 5.4 各种波形异常的机制 + 对应疾病（精讲）⭐⭐⭐

#### **A. a 波异常**

##### A1. **a 波消失** ⭐
- **疾病**：**心房颤动（Afib）**
- **机制**：心房不规则颤动 → 不能有效收缩 → **没有 a 波**
- **临床场景**：老年 + 不规则脉搏 + JVP 无 a 波 → Afib
- **顺便记**：Afib 也无 S4（S4 = 心房收缩进入硬 LV，需要心房功能正常）

##### A2. **Giant a 波** ⭐
- **疾病**：
  - **Tricuspid Stenosis (TS)** — 三尖瓣狭窄
  - **Pulmonary HTN / RV 肥厚**
  - **肺动脉狭窄（PS）**
- **机制**：RA 对抗**异常高压**（瓣口窄 / RV 肥厚 / PH）→ 加倍收缩 → a 波超大
- **临床场景**：
  - TS：风湿病史 + 舒张期杂音 + Giant a + 慢 y 谷
  - PH：dyspnea + 右心衰 + S2 P2 响 + Giant a
  - **Cor pulmonale**：COPD + Giant a

##### A3. **Cannon a 波（炮弹波）** ⭐⭐ **USMLE 经典**
- **疾病**：
  - **完全性房室阻滞（3° AVB）** ⭐
  - **室性心动过速（VT）**
  - **室性早搏（PVC）**
  - **AV 结性心律**
- **机制**：
  ```
  心房和心室收缩**不同步**
        │
        ▼
  某次心房收缩时，三尖瓣恰好**关闭**（因为心室也在收缩）
        │
        ▼
  心房的血**无路可去** → 反推到颈静脉
        │
        ▼
  突然的"巨浪"= Cannon a wave
  ```
- **临床特征**：**偶发**的巨浪（不是每次心跳都有，因为只有房室同步异常时才出现）
- **USMLE 套路**：
  - **Bradycardia + 偶发 JVP 巨浪** → **3° AVB**
  - **Wide QRS tachycardia + Cannon a** → **VT**（区分 SVT 的关键体征！）

---

#### **B. x 谷异常**

##### B1. **X 谷加深** ⭐
- **疾病**：**Cardiac Tamponade** ⭐
- **机制**：
  ```
  心包液体填满 → 整个心动周期都压心脏
        │
        ▼
  唯一能"减压"的瞬间 = 心室收缩
  （心室缩小，给心包腔留出一点空间）
        │
        ▼
  此时 RA 也在松弛 + 瓣环下移
        │
        ▼
  RA 压力急剧↓ → X 谷加深 ⭐
  ```
- 配合特征：**Y 谷消失**（详见下方 C2）

##### B2. **X 谷消失** ⭐
- **疾病**：**三尖瓣反流（TR）** ⭐
- **机制**：
  ```
  RV 收缩时血液本应进肺
        │
        ▼
  但三尖瓣闭不严 → 血液**反流回 RA**
        │
        ▼
  X 谷本应是 RA 松弛压力下降的瞬间
        │
        ▼
  但被反流的血液"填平"了 → X 谷消失
  ```
- 配合：**Giant v 波**（详见下方 D1）

---

#### **C. v 波异常**

##### C1. **Giant v 波** ⭐⭐
- **疾病**：**三尖瓣反流（TR）** ⭐
- **机制**：
  ```
  RV 收缩 → 血液反流回 RA
        │
        ▼
  RA 在收缩期本应只是被动充盈（小 v 波）
        │
        ▼
  现在被反流的血液**大量灌入** → 压力飙升
        │
        ▼
  v 波夸张 → "Giant v wave" 或 "CV fusion wave"（c 和 v 融合）
  ```
- **TR 完整 JVP 特征**：**Giant v + X 谷消失**
- **TR 病因**：RV 扩大（PH / LV 衰竭）、Ebstein 畸形、心内膜炎（IVDU）

---

#### **D. y 谷异常**

##### D1. **Y 谷消失/变浅** ⭐
- **疾病**：**Cardiac Tamponade** ⭐
- **机制**：
  ```
  三尖瓣开放本应是 RA 排空到 RV 的瞬间
        │
        ▼
  但 RV 被心包液体压住 → 不能扩张接纳血液
        │
        ▼
  RA 的血"出不去" → Y 谷消失 / 变浅
  ```
- **Tamponade 完整 JVP**：**X 深 + Y 消失 = 单谷**

##### D2. **Y 谷加深 + 陡（Friedreich's sign）** ⭐⭐
- **疾病**：
  - **Constrictive Pericarditis** ⭐
  - **Restrictive Cardiomyopathy**
  - 严重 TR
- **机制**（Constrictive 经典）：
  ```
  三尖瓣开放
        │
        ▼
  心室在心包硬壳内还有"空间余量"
        │
        ▼
  血液**快速涌入** RV（rapid filling）
        │
        ▼
  RA 压力急剧下降 → Y 谷又深又陡 ⭐
        │
        ▼
  心室容积达到硬壳上限
        │
        ▼
  充盈**突然停止** → Plateau
  ```
- **Constrictive 完整 JVP**：**X 深 + Y 深 = 双谷（M / W 形）**

##### D3. **Y 谷变浅（缓慢下降）**
- **疾病**：**Tricuspid Stenosis (TS)** ⭐
- **机制**：
  ```
  三尖瓣狭窄 → 瓣口窄
        │
        ▼
  RA 向 RV 排空缓慢
        │
        ▼
  Y 谷下降缓慢、不深
  ```
- **TS 完整 JVP**：**Giant a + 浅 y**

---

#### **E. 整体 JVP 异常**

##### E1. **全波消失 + JVD**
- **疾病**：**上腔静脉综合征（SVC syndrome）**
- **机制**：SVC 被压迫 / 阻塞 → 颈静脉持续高压 + 失去搏动
- **临床**：肺癌（右上肺）+ 面颈臂水肿 + JVD 但**不搏动**
- **USMLE 套路**：肺癌 + 面部水肿 + 上肢静脉曲张 → SVC syndrome

##### E2. **JVP 高度↑（仅高度，波形正常）**
- **疾病**：
  - **CHF（右心衰）**
  - **Volume overload**（液体负荷过重）
  - **Tricuspid regurgitation**（重度）

### 5.6 床旁怎么"看"JVP？⭐

#### **Step 1：体位**
- 病人**半卧 30-45°**
- 头转向**左侧**（暴露右颈）

#### **Step 2：照光**
- **切线光照**颈部（不是正面照）
- 颈部光影柔和摆动 = 静脉搏动

#### **Step 3：找搏动点**
- 在**右侧胸锁乳突肌**内侧观察
- 找静脉柱**顶点**的搏动
- ⚠️ 看的是**柱顶**，不是某个固定波形

#### **Step 4：测量高度**
```
JVP 测量公式：
JVP (cm H₂O) = 搏动顶点到胸骨角的垂直距离 + 5 cm

胸骨角（Angle of Louis）= 第二肋骨水平
胸骨角到 RA 的垂直距离 ≈ 5 cm（半卧位）

正常 JVP < 8 cm H₂O（即顶点距胸骨角 <3 cm）
异常：JVP > 8-9 cm = 右心压力升高
```

#### **Step 5：识别波形**
- 同步触诊颈动脉作为时间标记
- **颈动脉脉搏 = S1**
- 紧贴前的波 = **a 波**
- 紧贴后的波 = **c 波** → 然后 **x 谷**
- 远端的波 = **v 波** → 然后 **y 谷**

### 5.7 颈动脉 vs 颈静脉鉴别 ⭐

| 特征 | 颈动脉（Carotid）| 颈静脉（IJV）|
|---|---|---|
| **搏动相数** | 单相（快） | **双/三相** |
| **可触诊** | **是**（可触脉搏） | **否** ⭐ |
| **位置** | **内侧**（胸锁乳突肌内） | **外侧**（胸锁乳突肌前/内）|
| **轻压锁骨上窝** | 不影响 | **波形消失** ⭐ |
| **吸气** | 不变 | **下降**（正常）/ **升高**（Kussmaul）|
| **半卧位升高** | 不变 | **变化**（随体位升降）|
| **腹压试验（HJR）** | 不变 | **升高且持续 >15 sec = 阳性**（右心衰）|

**Hepatojugular Reflux (HJR) / 肝颈静脉反流试验**：
- 按压右上腹 30 秒
- 阳性：JVP 升高 >3 cm 并持续 >15 秒
- 提示：右心衰、限制性心包炎

### 5.8 Kussmaul Sign 单独深挖 ⭐⭐

**定义**：吸气时 JVP **升高**（反常）

**正常生理（基础）**：
```
吸气 → 胸内压↓ → 静脉回流↑ → 血液被吸入胸腔
        ↓
       JVP 应该**下降**（血液被胸腔"抽走"）
```

**Kussmaul 机制**：
```
吸气 → 静脉回流↑
        ↓
RV 想接纳血液
        ↓
但被某种原因**挡住**：
  - Constrictive 心包硬壳
  - Restrictive 心肌僵硬
  - RV 衰竭/梗死
  - TS / 大量 RA 血栓
        ↓
血液**进不去 RV** → 堵在 SVC + 颈静脉
        ↓
JVP **反升** = Kussmaul (+)
```

**Kussmaul 阳性疾病**（背！）：
- **Constrictive Pericarditis** ⭐⭐（最典型）
- **Restrictive Cardiomyopathy**
- **Severe Right Heart Failure**
- **Tricuspid Stenosis**
- **Right Ventricular Infarction** ⭐（下壁 MI 累及 RV）
- **Massive Pulmonary Embolism**

**Kussmaul 通常阴性的疾病**：
- **Cardiac Tamponade** ⭐（虽然心包病，但液体允许室间隔代偿）
- 普通 CHF（左心衰）

### 5.9 USMLE JVP 答题套路 ⭐

#### **套路 1：识别 Cannon a wave**
> 70 岁老人 + 头晕 + HR 35 + 不规则节律 + **JVP 偶发巨浪**
> → 3rd degree AV block
>
> Wide QRS tachycardia 区分 SVT vs VT：
> + **Cannon a + AV dissociation** → **VT** ⭐

#### **套路 2：识别 Giant v wave**
> IVDU 病人 + 发热 + 新发心脏杂音 + 双下肢水肿 + **JVP Giant v wave**
> → IE → TR
>
> 风湿性心脏病 + LV 衰 + 双下肢水肿 + JVP Giant v
> → 继发性 TR（功能性）

#### **套路 3：识别"单谷 vs 双谷"**
> 癌症 + dyspnea + Beck triad + **JVP 单谷（X 深 Y 消失）**
> → Tamponade
>
> 结核病史 + 慢性右心衰 + Kussmaul + **JVP 双谷（M/W）**
> → Constrictive Pericarditis

#### **套路 4：Kussmaul 鉴别**
> 看到 Kussmaul + 任何病因 → 优先想：
> 1. Constrictive Pericarditis（最常考）
> 2. Restrictive CM（如 Amyloid）
> 3. RV infarction（下壁 MI）
> 4. Severe RHF / TS

#### **套路 5：SVC syndrome 陷阱**
> 肺癌（右上肺）+ **面颈臂水肿 + JVD 但不搏动**
> → SVC syndrome（不是右心衰！）

## 六、RV 压力曲线 vs JVP 对应

### 1. 不一样但对应

| | JVP 波形 | RV 压力曲线 |
|---|---|---|
| **测什么** | RA 压力（外部反映） | RV 内压（导管）|
| **波形** | a-c-x-v-y（5 段） | 收缩峰 + 舒张充盈 |
| **临床** | 床旁视诊 | 心导管 |

### 2. 关键对应 ⭐
- JVP **a 波** = RV 舒张末
- JVP **x 谷** = RV 收缩中
- JVP **v 波** = RV 收缩末
- JVP **y 谷** = **RV Dip**（舒张早期快速下降）⭐

### 3. 三种心包病的双重表现

| 病 | JVP | RV 曲线 |
|---|---|---|
| **Tamponade** | 单谷（X 深 Y 消失）| 平滑无 dip |
| **Constrictive** | 双谷 M/W（X+Y 深）| **Square root** ⭐ |
| **Restrictive** | M/W + 大 a | Dip-plateau（轻）|

### 4. 心导管数据读法

**读数据 4 步**：
1. 看收缩峰：RV >40 = PH
2. 看舒张末压：>15 = 右心衰
3. 看 4 压均衡（RA=RV=PCWP=LV）：相等 → C/T, LV>>RV → R
4. 看波形：Square root → Constrictive 或 Restrictive

**典型数据例**：

| 病 | RA | RV | PA | PCWP | LV | 特征 |
|---|---|---|---|---|---|---|
| 正常 | <8 | 25/5 | 25/10 | <12 | 120/10 | — |
| Constrictive | 18 | 35/18 | 35/18 | 20 | 120/20 | 全均衡 |
| Restrictive | 15 | 40/15 | 55/25 | 28 | 130/28 | LV>>RV |
| Tamponade | 20 | 25/20 | 25/20 | 22 | 100/22 | 全均衡+RV 压低 |
| PH | 12 | 80/12 | 80/40 | 10 | 120/10 | RV/PA 压高 |

## 七、BNP 临床应用 ⭐

### 1. 生理
- 心室肌响应**室壁张力**↑分泌
- Laplace's Law：Stress = (P×r)/(2h)
- 作用：利钠 + 扩血管 + 抑制 RAAS / 交感

### 2. 数值意义
- <100：心衰不太可能
- >400：高度提示
- >1000：严重 / 急性失代偿

### 3. 鉴别用途 ⭐
- **CHF**：↑↑
- **Restrictive CM**：↑↑↑（心肌被浸润，张力↑↑）⭐
- **Constrictive Pericarditis**：**不高 (<200)** ⭐（心肌正常 + r 小）
- **肺源性 dyspnea**：正常（鉴别 CHF vs COPD）
- **肥胖**：偏低（脂肪清除）

### 4. USMLE 套路
> 右心衰但病因不明：
> - BNP 低 → **Constrictive Pericarditis**
> - BNP 高 → **Restrictive CM**

## 八、Low Voltage ECG ⭐

### 1. 标准
- 肢导 QRS <5 mm
- 胸导 QRS <10 mm

### 2. 病因（5C5O 分类）⭐

**信号本身少（5C — Cardiac）**：
1. **C**ardiac Amyloid ⭐⭐
2. **C**ardiomyopathy (Restrictive)
3. **C**ompletely dead myocardium（大面积 MI）
4. **C**onstrictive Pericarditis（部分）
5. **C**ardiac Tamponade ⭐

**信号被衰减（5O — Outside）**：
1. **O**besity ⭐
2. **O**bstructive lung dz (COPD) ⭐
3. **O**verflow (胸腔积液 / Anasarca)
4. **O**pen air (Pneumothorax)
5. **O**hh, hypothyroid (Myxedema) ⭐

### 3. Low Voltage + 伴随线索锁定诊断 ⭐⭐

| Low V + 伴随 | 诊断 |
|---|---|
| + **Electrical alternans** | **Tamponade** ⭐ |
| + **Pseudoinfarct Q (V1-V3) + AV 阻滞 + 壁厚** | **Cardiac Amyloid** ⭐ |
| + 右轴 + P pulmonale + barrel chest | COPD |
| + Bradycardia + 怕冷 + 体重↑ | Myxedema |
| + Q 波广泛 | 大面积 MI 后 |

### 4. 经典反差 ⭐⭐⭐
> **"心室壁厚 + Low voltage = Cardiac Amyloidosis"**
> 这是 USMLE Step 2 心内最高频反差题
> 原因：心肌被淀粉样物**取代**，不是真心肌

## 九、Pulsus Paradoxus vs Kussmaul Sign ⭐

| | Pulsus Paradoxus | Kussmaul Sign |
|---|---|---|
| **看什么** | 血压（袖带听诊） | 颈静脉（JVP）|
| **吸气变化** | SBP ↓ >10 mmHg | JVP ↑（反升）|
| **代表病** | **Tamponade** ⭐ | **Constrictive** ⭐ |
| **机制** | 室间隔左移 → LV 充盈↓ | 硬壳 → RV 不能容纳 → 血堵颈静脉 |
| **病变质地** | 液体（liquid）| 固体（rigid shell）|

**助记**：
- "Pulsus 看袖带，Kussmaul 看脖子"
- "液体压塞 P，纤维缩窄 K"
- **P**ulsus + tam**P**onade
- **K**ussmaul + **K**onstrictive + **K**nock

**Pulsus Paradoxus 其他原因**：哮喘急性发作 / COPD / 张力性气胸 / 大面积 PE

**Tamponade 无 Pulsus 的情况**：AR / ASD / LV 严重坏 / 重度 PH

## 十、Memory Hook 合集 ⭐

### 1. 心包病谱
> **"液体 → Pulsus（压塞 acute）；纤维 → Kussmaul（缩窄 chronic）"**

### 2. 心影大鉴别
> **"心影大不等于心衰，先看肺野清不清"**

### 3. Tamponade 处理
> **"Fill the tank, drain the sac"**
> （灌水箱 IV fluids，排袋子 pericardiocentesis）
> **禁忌**：利尿 / 扩血管 / β-阻（三降 preload）

### 4. 穿刺禁忌
> **"创伤穿，夹层切；MI 破裂直接切"**

### 5. Tamponade vs Constrictive 物理本质
> **"水袋 vs 铁壳"**
> - 水袋（液体）：整周期传压 → Pulsus + Y 谷消失
> - 铁壳（固体）：只在容量上限起作用 → Kussmaul + Square root

### 6. Constrictive vs Restrictive
> **"外壳硬 vs 内料硬"**
> - C：心包"绑"两腔 → 压力均衡（RV=LV）
> - R：心肌本身僵 + LV 壁厚 → LV >> RV

### 7. BNP 鉴别
> **"心肌好 BNP 低（C），心肌坏 BNP 高（R）"**

### 8. JVP 异常波形速记
- a 没 → Afib
- a 大 → PH / TS
- Cannon a → 3° AVB / VT
- x 没 + v 巨 → TR
- X 深 + Y 没（单谷）→ Tamponade
- X + Y 都深（M/W 双谷）→ Constrictive

### 9. RV 曲线
> **"Square root sign = Constrictive 或 Restrictive"**
> Dip（早期 rapid filling）+ Plateau（晚期撞硬壳）

### 10. JVP 与 RV 曲线对应
> **"JVP 的 Y 谷深 = RV 的 Dip"**
> 都是"三尖瓣开放，心室快速充盈"那一刻

### 11. Low Voltage 反差
> **"壁厚 + Low voltage = Cardiac Amyloid"**

### 12. Cardiac Amyloid ECG 三联
> Low voltage + Pseudoinfarct Q (V1-V3) + AV 阻滞

## 十一、USMLE 高频套路题型

### 套路 1：MPE → Echo
> 癌症 + dyspnea + 心影大 + 肺野清 → Echo（不是 furosemide / CT / PFT）

### 套路 2：创伤 vs 夹层 tamponade
> 创伤 → 穿刺
> 夹层 → 直接 OR（穿刺致死）

### 套路 3：Tamponade 禁忌药
> 看到 tamponade + furosemide / 硝酸甘油 / β-blocker 选项 → 全错

### 套路 4：Constrictive 典型
> 结核 / 放疗 / CABG → 慢性右心衰 + Kussmaul + Knock + 心包钙化 → Pericardiectomy

### 套路 5：Amyloid 反差
> 老年男性 + 舌大 + 壁厚 + Low voltage + Pseudoinfarct Q → Cardiac Amyloid（活检 Congo red+）

### 套路 6：心导管识别
> RA=RV=PCWP=LV=20 → Constrictive 或 Tamponade
> LV (28) >> RV (15) → Restrictive

### 套路 7：Pulsus 鉴别
> 吸气 SBP ↓>10 → Tamponade（或哮喘 / COPD / 气胸 / PE）
> Tamponade 但无 Pulsus → AR / ASD / 严重 LV 坏 / PH

## 十二、HCM 完整管理 ⭐（v2.1 增量）

> [!info] 章节定位
> 落实原"十二待补充"中的"HCM vs Restrictive vs Constrictive 三方鉴别"项。
> 来源：UWorld "Management of hypertrophic cardiomyopathy" 决策图（无关联具体错题，作为高频参考整理入库）。

### 1. 诊断要点（briefly）

- **不对称室间隔肥厚** ≥ 15 mm（Echo）
- **SAM**（systolic anterior motion of MV）→ 动态 LVOT 梗阻
- **遗传**：常染色体显性，**β-myosin heavy chain (MYH7)** / myosin-binding protein C 等突变
- **杂音特征**：LLSB 收缩期 ejection-type；**Valsalva ↑ / 站立 ↑**（减 preload → 梗阻加重）/ **下蹲 ↓ / 握拳 ↓**（增 preload → 梗阻减轻）— 与 AS 反向

### 2. 管理决策树（UWorld 原图）

![[{FDAD7D49-B255-4F40-89A0-594D0A74F1E1}.png]]

### 3. 5 大干预分支速查表

| 患者状态 | 一线治疗 |
|---|---|
| **Asymptomatic** | **Routine clinical monitoring**（年度 Echo + 风险分层） |
| **Symptomatic**（dyspnea / angina）| **β-blocker** 或 **non-DHP CCB**（verapamil / diltiazem）— ↓HR + ↑LV filling time → ↓dynamic obstruction |
| **High SCD risk** | **ICD placement** |
| **AFib** | **Rate / rhythm control + 抗凝**（HCM 患者血栓栓塞风险高，不依赖 CHA₂DS₂-VASc 评分，**有 AFib 就抗凝**）|
| **Refractory + 明显 LVOT 梗阻** | **Septal ablation**（酒精消融）或 **surgical myectomy**（外科室间隔切除）|
| **Refractory + 无梗阻** | **Cardiac transplantation** |

### 4. SCD 高危标准（ICD 适应症）⭐

任一阳性 → 考虑 ICD：

- **Prior syncope**（不明原因晕厥，尤其运动相关）
- **Sustained VT**（持续性室速）
- **Family history of SCD**（1 级亲属 < 50 岁猝死）
- **LV wall thickness ≥ 30 mm**（最大室壁厚度）
- **NSVT on Holter**（非持续性室速）
- **Abnormal BP response to exercise**（运动负荷试验时 BP 不升甚至下降）

### 5. 药物机制 + 禁忌 ⭐

**用什么**（↓动态梗阻）：
- **β-blocker**（metoprolol / atenolol）— ↓HR + ↓收缩力
- **Non-DHP CCB**（verapamil / diltiazem）— ↓HR + ↓收缩力
- **Disopyramide**（IA 类，强负性变力）— 难治性梗阻可加

**禁用 / 慎用**（加重动态梗阻）：
- ❌ **Diuretics 慎用**（减 preload → ↑梗阻）
- ❌ **Nitrates 禁**（减 preload + afterload）
- ❌ **DHP CCB**（amlodipine / nifedipine — 降 afterload → ↑梗阻）
- ❌ **ACEi / ARB**（降 afterload → ↑梗阻）
- ❌ **Digoxin**（正性变力 → ↑梗阻）
- ❌ **剧烈运动**（脱水 + 心率↑↑ → ↑梗阻 + SCD 风险）

> [!tip] HCM 治疗反直觉口诀
> **"HCM 怕空腔 — 减 preload / 减 afterload / 增 contractility 都加重梗阻；治疗反向：↓HR、↑filling time、保 preload"**

### 6. HCM vs Restrictive vs Constrictive 三方鉴别（落实第十二章原待补充 item）

| 维度 | **HCM** | **Restrictive CM** | **Constrictive Pericarditis** |
|---|---|---|---|
| 心室壁 | **↑↑↑**（不对称室间隔 ≥ 15 mm）| 正常或轻度 ↑ | 正常 |
| 心室腔 | **小**（梗阻型）| 小 | 正常 |
| 心包 | 正常 | 正常 | **厚 + 钙化** ⭐ |
| **Echo SAM** | ✅ | ❌ | ❌ |
| **Septal bounce** | ❌ | ❌ | ✅ |
| Kussmaul | 罕见 | ✅ 可有 | ✅ 经典 |
| Pulsus paradoxus | 罕见 | 罕见 | 罕见 |
| 杂音 | **收缩期 LLSB + Valsalva ↑** ⭐ | 无 | 无 |
| Pericardial knock | ❌ | ❌ | ✅ |
| 心导管 | 不同 | **LV ≈ RV 舒张压** | LV ≈ RV 舒张压 + 平台征 |
| 治疗 | **BB / 非 DHP CCB → 手术** | 病因（amyloid / sarcoid）| **Pericardiectomy** |

> [!success] 三方一句话区分
> - **HCM** = 心肌**自己**变厚（基因 / 不对称室间隔）+ 杂音 + Valsalva ↑
> - **Restrictive** = 心肌**被填**（amyloid / sarcoid 浸润 / Fe / 纤维化）+ 无杂音
> - **Constrictive** = **心包**变厚（结核 / 放疗 / 心脏手术后）+ knock + septal bounce

### 7. USMLE 经典 stem 模式

> [!warning] 高频 stem
> - **年轻运动员**剧烈运动后晕厥 / 猝死 + 收缩期杂音 **Valsalva ↑** + 家族猝死史 → **HCM**
> - 30-40 岁逐渐出现劳力性 dyspnea + angina + LLSB 杂音 + Echo 不对称室间隔 → **HCM**
> - HCM 病史 + 新发 AFib → **抗凝**（不计算 CHA₂DS₂-VASc）
> - HCM + 室壁 32 mm + Holter 见 NSVT → **ICD**
> - HCM + 难治症状 + 显著 LVOT 压差 → **Septal myectomy** 或 **alcohol septal ablation**

### 8. Memory Hook

- **诊断**：**"年轻人 + 杂音 + Valsalva ↑ + 家族猝死 = HCM"**
- **治疗**：**"↓HR + ↑filling time"**（BB / 非 DHP CCB）
- **SCD 5 高危**：**"晕厥 / SVT / 家族猝 / 壁≥30 / NSVT"**
- **难治**："**有梗阻就切，无梗阻就换**"（myectomy/ablation vs transplant）
- **禁用反向**："减 preload / 减 afterload / 增 contractility 都加重 → HCM 喜静脉回流"

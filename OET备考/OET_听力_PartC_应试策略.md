# OET 听力 Part C 应试策略（注意力分配版）

> 返回 [[00_OET总览]] · 单选铁律/陷阱库见 [[OET_听力阅读_技巧]]（§Part B/C 单选方法论）· 拼写默写见 [[OET_听力_默写词库]]

> [!info] 本篇解决的问题（2026-07-06 实战反馈）
> Part C 三大困境：① 先看题细节记不住，很懵；② 一边听一边看题+选项太分心；③ 都听完再选记不住细节。
> **根因都是 working memory（工作记忆）超载** —— 解法不是"记得更多"，而是把"看"的负担在预读期消化掉，听的时候眼睛只停在一道题上。

---

## 一、先定心：Part C 容错率很高

| 板块 | 题数 | 说明 |
|---|---|---|
| Part A | 24 | 听音填空（拼对才得分）——主战场 |
| Part B | 6 | 短对话单选 |
| **Part C** | **12** | 两篇长独白/访谈（各约 5 分钟，各 6 题单选） |

- 达标线 350 (B) ≈ 全卷对 30/42。**Part C 错 4-5 个仍可过线**（前提 Part A 稳住）。
- Part C 的目标不是全对，是**别崩盘**——最怕一题没跟上就慌，连锁丢 3-4 题。

---

## 二、三个困境的诊断

| 做法 | 判定 | 原因 |
|---|---|---|
| 先看题、努力记题目细节 | 方向错 | **不需要记**——Part C 题目顺序 = 音频顺序，卷子本身就是地图，眼睛回卷子即可 |
| 边听边看全部题+选项 | 负担超载 | 每题 1 题干 + 3 选项 = 4 段文字同时处理必分心；要减负到**只看当前一道题** |
| 都听完再选 | 唯一确定错误 | Part C 考观点/态度/推断 (inference)，干扰项全是音频出现过的原词——凭事后记忆选 = 专挑陷阱踩 |

> [!danger] 铁律
> Part C 必须**边听边答**（实时跟题），但每段**听完整段再落笔**（OET 爱在后半句反转——见 [[OET_听力阅读_技巧]] 五条铁律第 5 条）。两者不矛盾：跟题是"人在哪一题"，落笔是"这一段讲完才定"。

---

## 三、90 秒预读怎么花（每篇开始前）

1. **先扫 6 个题干（不看选项）**：每个题干圈 1-2 个关键词，搞清"这题在问什么"（问原因？态度？担忧？转折点？）。这一步产出一张**话题路线图**：这篇讲座会依次经过 6 个站点。
2. **剩余时间扫选项，每个浓缩成 1-2 个词**：如选项 "the treatment carries a risk of long-term dependency" → 心里只留 "dependency（依赖）"。不精读、不硬记。
3. 若三个选项**前半雷同**（it led him / it made him / it showed him）→ 预读时就划掉前半，只留后半差异词（技巧笔记铁律第 3 条）。

---

## 四、听时操作：单题跟踪 + 路标词

- **笔尖压在当前题上**，只看这一题的题干 + 3 个浓缩关键词，其余 5 题当不存在。
- 该题内容讲完 → 选 → 笔尖滑到下一题。**选完翻篇，不回头怀疑**。
- 换题信号 = **signposts（路标词）**：
  - 独白类："Another issue is..." / "What surprised me was..." / "Moving on to..." / "The second thing I noticed..."
  - **访谈类最友好：主持人每问一个新问题 ≈ 换下一题**。

> [!warning] 丢题急救（防崩盘的关键一条）
> 发现"这段没听懂 / 走神了" → **立刻放弃该题，笔尖跳到下一题**。宁可白丢 1 分，不能因纠结丢了位置——一旦"不知道现在讲到哪题"，后面全军覆没。最多允许落后一题（one-question-behind），绝不落后两题。放弃的题最后凭大意猜一个，不留空。

## 五、选项陷阱：三个选项的词音频里都会出现

出题逻辑（与 Part A 的同义替换弱点同源）：

- **错误选项** = 用音频**原词**，但意思被扭曲（是别人的观点 / 被否定了 / 只是举例 / 是 past perfect 背景不是结论）。
- **正确选项** = 意思对但**换了说法**（音频 "patients often stop taking it once they feel better" → 选项 "poor adherence（依从性差）"）。

→ "听到哪个词选哪个" = 必错。**听意思、听态度**，尤其转折词（but / however / having said that）之后——真观点常在转折后。
→ 完整陷阱清单（相关词诱饵 / 反义陷阱 / 原词假答案 / 时态排除法）见 [[OET_听力阅读_技巧]]，不在此重复。

## 五B、陷阱全谱系：一切干扰项都在骗你「多走一步」⭐（两次实战归纳）

> [!danger] 一条总纲统摄所有陷阱
> **正确项 = 音频明确说过的意思，换个说法（同义替换）；干扰项 = 让你多走一步。**
> "多走一步" = 伸手抓原词 / 把部分夸大成全部 / 顺着一个真词往下编 / 脑补一个没讨论的维度 / 把邻近细节搬来答错题。**凡是选它需要你自己补一步推理的，几乎都是坑。**
> 数据：07-07（concussion 脑震荡 + SPRINT 血压试验）错 8 题、07-09（social prescribing 社会处方 + CF, cystic fibrosis 囊性纤维化）错 6 题，**14 题无一例外全是下面这张表里的类型**。

### 谱系表（按"你多走的那一步"分类）

| 家族 | 子类型 | 干扰长什么样 | 实战例 | 反射 |
|---|---|---|---|---|
| **A. 抓原词**（提到即选）| 原词假答案 | 干扰项用音频**原词**，意思却被扭曲 | 通用 | 听到题干原词**先缩手**，真答案几乎总是同义替换 |
| | 极性反转 | 原词方向被反（正说成负 / 否定说成肯定）| 07-07 Q42 药企资助（其实是政府资助 rather than 药企）| 听到关键词先**判极性**再动手 |
| | "过去认为"旧观点 | 旧观点抓眼词做干扰，后被否定 | 07-07 Q33（fatal 包在 It used to be thought，被 evidence is slim 推翻）| 听到 It used to be thought / previously / traditionally → 旧观点词 100% 是干扰，真答案在 "What we do know..." 之后 |
| | 提到 ≠ 正面 | 出现过的词，但语境是负面/无关 | 07-09 Q37（hospital visits 被说成 frustrating）| "出现过" ≠ "是答案"，扣它的**语境态度** |
| | 真词 + 假 claim | 抓一个真词，安上原文没有的修饰 | 07-09 Q39（complete 是真词，quickly 是编的）| 只认音频**明确说过**的动作，别顺词补意思 |
| **B. 程度/对象偏移** | 程度夸大 | 把"部分/减少"夸大成"完全/从不" | 07-09 Q32（reduce → no longer）| 听到 no longer / never / always 先警惕夸大 |
| | 主语/对象错位 | 张冠李戴：谁说的、比较谁、几个人 | 07-09 Q35（医生展望 ≠ 患者反馈；一人 ≠ 三人）；07-07 比较句主语翻转 | 盯题干限定词（all three / their feedback / 比较句先做主语翻转）|
| **C. 脑补/无中生有** | 无中生有 | 选一个音频**根本没讨论**的维度 | 07-09 Q42（选"没那么痛"，但通篇没提 pain）| 没讨论的维度绝不选，哪怕"听起来像进步" |
| | 近处细节钓鱼 | 拿邻近的真细节脑补出结论 | 07-09 Q34（膝伤/腰痛 → 脑补"怕受伤"，其实担心"活太重"）| 扣她**明说**担心什么，别顺细节脑补 |
| | 近处干扰 / 搬错位置 | 用邻近或别题的真信息答本题 | 07-09 Q31（本地花园 ≠ 理念起源）；Q38 干扰 B（社交是别题话题）| 先问"这细节回答的是**哪道题**" |
| **D. 自己改答案** | 二次怀疑改错 | 第一直觉对，被回响词带着改成错的 | 07-07 改成 C（被 doctor 回响干扰）| **没有新证据不改**（同 UWorld 改答案错 188 题教训）|

## 六、机考界面操作（highlight / strikeout 落地，2026-07-06 查证）

机考 OET（考点机考 + OET@Home）的 Listening / Reading / Writing 界面自带标注工具；**机考不发纸笔**（纸笔只在 Speaking 准备时给），"圈词"全靠界面完成。

| 功能 | 操作 | Part C 用法 |
|---|---|---|
| **Highlight** | 鼠标拖选文字 → 点弹出按钮确认；再点一次取消 | **预读 90 秒**内标题干关键词（每题 1-2 个词，标多 = 没标） |
| **Strikeout 划掉** | 右键选项 | 确定排除的选项当场划掉（前半雷同划前半 / 原词假答案）——屏幕剩的字更少、负担更小 |
| **屏幕记事本 (on-screen notepad)** | 界面自带 | 听力时基本没时间用，忽略 |

> [!warning] 时机纪律
> - 高亮动作尽量在**预读期**完成；听的过程中再去拖选高亮 = 新的分心源，得不偿失。
> - "笔尖压当前题"在机考里 = **鼠标停在当前题区域**，眼睛只锁那一题。
> - 与 [[OET_听力阅读_技巧]] "Reading Part A 不用 highlight 工具"不矛盾：工具存在，Reading Part A 节奏太赶不值得用；**Part C 恰恰相反，预读期 highlight 是核心动作**。

> [!info] 若考纸笔版 (OET on paper)
> 可用铅笔/钢笔在卷子上划线、写批注（Listening/Reading/Writing 均可），但**不允许带荧光笔**；有考生反馈个别考场不让在题卷上写字，考前和考点确认。
> 来源：[OET Support — highlights/notes in OET on computer](https://support.oet.com/s/article/Can-I-make-highlights-or-write-notes-in-OET-on-computer-at-a-venue) · [Lifestyle Training Centre 汇总](https://goltc.in/can-i-highlight-underline-or-write-notes-in-the-oet-test/)

## 七、三个针对性训练（每次练完 Part C 做）

1. **路标词训练**：拿一篇 Part C 只做一件事——每次话题切换在纸上画一道杠，不答题。练 3-5 篇，"现在讲到第几题"的方向感就出来了。
2. **同义替换对账**：对完答案打开 transcript，列出每题"音频原话 → 正确选项措辞"的替换对（deteriorated → got worse 的 Part C 版）。替换对随手发 Claude，照 Part A 生词流程回填 [[OET_听力_默写词库]]。
3. **预读限时训练**：拿新题只给 90 秒，练"题干圈词 + 选项浓缩 1-2 词"，练到 90 秒够用——预读效率上来，听时负担才真正降下来。

> [!success] 一句话总结
> **预读记"路线图"不记细节；听时笔尖压单题、段落讲完再落笔；丢了就跳、绝不回头。**

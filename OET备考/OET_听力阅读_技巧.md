# OET 听力 + 阅读 技巧

> 返回 [[00_OET总览]] · 拼写默写见 [[OET_听力_默写词库]]

> [!success] 定位
> Listening 是你**最难**的一科（拼写+打字+口音）；Reading 以你 USMLE 底子**基本白给**，保 B 即可、把时间省给听力。两科都 **≥350（B）** 才达标，容错 ≤12。

---

## 一、Listening（头号战场）

### 做题策略

- 顺序：**先做 B、C，再回头做 Part A**（A 给的检查时间多，可后补拼写）。
- 目标：Part A 错≤4 / B 错≤2 / C 错≤4 = 稳 B。

### Part A — 拼写 + 定位（决定成败）

1. **听力原词优先**：填空基本是原词，别自作聪明改同义词。
2. **缩写记音节**：methotrexate（甲氨蝶呤）→ `m-t-a`（别只记首字母，回看会忘）。
3. **听全 and/or 列举**：`a, b, c, and d` 录音常打乱顺序，要整句抓。
4. **时间空补介词**：`three years ___` → 漏 `ago` 算错。
5. **职业 vs 地点**：`employed in + 地点`，`as + 职位`。
6. **过去史**：写病名还是治疗（`underwent + 治疗手段`）。
7. **替代词**：dizziness=lightheadedness，whilst=when。
8. **单复数 / 冠词**：器官前 the；检查方法前 a；药物前不加冠词。听不清 left/right 多半是复数。
9. **动词变形**：-y 结尾（辅音+y→ied）、-e 结尾（去 e 加 ing/ed；ee/ye/oe 不去 e，如 dyeing）。
10. **数字写范围**：不写孤零零 `70`，写 `more than 70`。
11. **剂量单位**：single dose（单次）vs maximum daily dose（每日上限）。
12. 剩余时间**通读检查语法+拼写**。

### Part B / C — 逻辑题

- 先分**主旨题 vs 细节题**。细节题：主语+限定词去原文定位。
- 逻辑钥匙：**"A, but B" → B 是重点；"A, because B" → A 是重点。**
- B 部分是对话，**第一句先听清说话人身份**。
- C 部分结构固定：background（含观点）→ 疾病表现 → 治疗。每段一题，**不用通读全文**。
- 读题 90 秒：不是读选项，是**把 A/B/C 三个选项的差别翻成中文记脑子里**，听时有两把钥匙去对。

### 材料（按质量排序）

1. **必做**：5 Sample tests + OET Medicine Official Practice Book + Banfield ProMedical 9 套。
2. **YouTube**：OET Global Fighters（练 Part A，强推）/ Mission OET / JAY'S OET LISTENING。
3. **Benchmark 20 套**：偏英音，放最后练口音多样性。
4. **口音**：美/英/澳/爱尔兰都会有，实战常澳音（卷舌弱、元音短）。

## 二、Reading（保 B 即可，省时间给听力）

- **Part A（15min/20题，最赶）**：先 skim 四篇文本+脚注 → 找题干限定词 → 原文定位 → 直接复制粘贴。**不用 highlight 工具**（浪费时间，答案很直接）。数字写范围、剂量单位看清。
- **Part B/C（45min）**：先读题（主语+关键词）→ 再读文 → 最后看选项。可回看，用侧边 tab 导航。C 部分用 highlight。
- 时间分配：B 约 10min，C 约 35min。
- 逻辑同听力：`A but B → B 重要`。
- 材料：官方 Practice Book + 5 Sample tests 够用，每周 2 套保持手感。

## 🧪 酸碱 / 电解质簇 — 一页纸决策入口（考前 30 秒扫）⭐

> [!info]- 四大入口决策图（点开看 ASCII 总览 + 跳转链接）
> 把 **ABG 判读 / NAGMA→RTA / 代碱分型 / 低钠容量轴** 四条线串成一张图。每个终点 = 一个反射动作。
>
> ```
> 【入口 0：拿到 ABG 先三步法】→ 详见 SOP §一/§1.1
>   ① pH → 酸血症(<7.35) / 碱血症(>7.45)
>   ② 看 HCO3 vs PaCO2，哪个朝 pH 方向变 = 原发（"数定原发，史定病因"）
>   ③ Winter 校代偿：预期 PaCO2 = 1.5×HCO3 + 8 (±2)，偏离=叠加呼吸紊乱
>
> 【入口 1：代谢性酸中毒】先算 AG = Na −(Cl+HCO3)，正常 8–12
>   ├─ AG↑（MUDPILES）→ 高 AG 代酸，查酮体/乳酸/渗透间隙/毒物
>   └─ AG 正常（HARDASS）= NAGMA ── 低 HCO3 + 高 Cl ──┐
>                                                       ▼
>      【入口 2：NAGMA → RTA / 腹泻】先看血钾切 Type 4
>        ├─ 高钾 → 直奔 Type 4（老糖人 + 糖尿病肾病，低肾素低醛）
>        └─ 低钾 → 排除 Type 4，看代酸时尿 pH：
>             ├─ 尿 pH > 5.5 → 远端 Type 1（Sjögren/自免/amphotericin）
>             └─ 尿 pH < 5.5 → 算尿 AG = UNa+UK−UCl
>                  ├─ 负值 = 腹泻（"neGUT-ive"）
>                  └─ 正值 = 近端 Type 2（Fanconi/MM/Wilson/乙酰唑胺）
>
> 【入口 3：代谢性碱中毒】测尿 Cl 分型
>   ├─ 尿 Cl < 20 = saline-responsive → 给 NS（呕吐/NG/既往利尿剂）
>   └─ 尿 Cl > 20 = saline-resistant → 治病因（Conn/Cushing/Bartter-Gitelman/在用利尿剂）
>
> 【入口 4：低钠】低渗 + 先判容量
>   ├─ 低容量（UNa<20、高尿酸）→ 等张盐水
>   ├─ 等容量 = SIADH（Uosm>100、UNa>40、低尿酸）→ 限水/盐片/托伐普坦
>   └─ 高容量（肝硬化/CHF）→ 限水 + 螺内酯+袢；轻症肝硬化(Na>120)不特殊治疗
>        ⚠ 噻嗪 + NS 会反向加重 SIADH/高容量低钠
> ```
>
> **跳转**：ABG 三步法 + 4 病例实战 → [[完整笔记/专题笔记/肾脏/肾脏_酸碱代偿完整SOP]] ｜ RTA 四型 + NAGMA 两步 + 尿 AG → [[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全]] §2.3 ｜ 代碱分型 + 低钠容量轴 → [[完整笔记/专题笔记/肾脏/肾脏_电解质急症速查]]
>
> **5 条反射钩**：
> 1. **判酸碱原发先读数** — 看 HCO3/PaCO2 哪个朝 pH 方向变，别被病史词带偏（数定原发，史定病因）
> 2. **低钾 + 高氯 = NAGMA**，先用血钾把 Type 4（唯一高钾）切出去
> 3. **尿 AG neGUT-ive（负）= 腹泻**；正 = RTA
> 4. **代碱看尿 Cl**：<20 给盐水，>20 治病因
> 5. **SIADH vs 低容量低钠看尿酸**：低尿酸 = SIADH；高尿酸 = 脱水

## 🚽 泌尿 / 下尿路梗阻簇 — 一页纸决策入口（考前 30 秒扫）⭐ — 2026-06-05 新增

> [!info]- 排尿困难 / 尿潴留 / LUTS 决策图（点开看 ASCII 总览 + 跳转链接）
> 把 **病因定位 → 检查 → 处理** 串成一张图。每个终点 = 一个反射动作。
>
> ```
> 排尿困难 / 尿潴留 / LUTS（先排前列腺癌）
>   │
>   ├─ 急性不能排尿 + 耻骨上压痛 ─→ AUR
>   │      └─ bladder US ≥300 mL → Foley + UA（无热+WBC正常+术口净 = 反对感染，别冲血培养）
>   │
>   ├─ 慢性渐进 LUTS
>   │     ├─ 老年男 + 前列腺大 ──→ BPH → α1阻滞剂(±5-ARI)
>   │     │      ├─ 合并HTN → doxazosin/terazosin（兼降压）
>   │     │      └─ 血压正常 → tamsulosin（少降压；注意IFIS）
>   │     ├─ 年轻男 + 器械/外伤史 → 尿道狭窄 → 逆行尿道造影(RUG)
>   │     └─ 神经背景(SCI/MS/DM) → 神经源膀胱 → 尿动力+间歇导尿
>   │
>   └─ 慢性进行性 Cr↑（疑肾后梗阻）─→ 先肾US排 hydronephrosis
> ```
>
> **跳转**：决策总图 → [[完整笔记/专题笔记/肾脏/肾脏_下尿路梗阻诊疗决策]] ｜ BPH 药物详细 → [[完整笔记/专题笔记/肾脏/肾脏_BPH药物治疗]] ｜ BPH 速查 → [[完整笔记/专题笔记/肾脏/肾脏_BPH药物治疗_速查]]

> **4 条反射钩**：
> 1. **老年男术后躁动 + 耻骨上压痛 = 先查膀胱(bladder US)**，不是抽血培养（WBC正常+无热+术口净反对感染）
> 2. **慢升 Cr 的 BPH 老人 → 先肾US排积水**，别直接上药
> 3. **年轻男梗阻性排尿 + 器械/外伤史 = 尿道狭窄**，≠BPH
> 4. **tamsulosin 用药 + 白内障术前 = IFIS** → 告知术者改良技术，不停药/不取消手术；**5-ARI ↓PSA 50% 筛查要×2 + 致畸**

## 🫁 肺科 — 急性呼吸困难 / 胸痛低氧簇 — 一页纸决策入口（考前 30 秒扫）⭐ — 2026-06-12 新增

> [!info]- ① 胸腔积液：渗/漏 + 引流决策图（点开看 ASCII 总览 + 跳转链接）
> 把 **体征定位 → 定性（渗 vs 漏）→ 引流决策** 串成一张图。每个终点 = 一个反射动作。
>
> ```
> 胸膜性胸痛（吸气加重/屏气消失）+ 患侧呼吸音↓ + 叩浊
>   │
>   ├─ 先用语颤定位（区分积液 vs 实变）
>   │     ├─ 语颤↓ → 胸腔积液 / 气胸（液/气隔音）
>   │     └─ 语颤↑ → 肺实变 / 肺炎（实变传声）
>   │
>   ├─ 诊断流程：decubitus CXR（初筛自由流动）→ diagnostic thoracentesis（确诊定性）
>   │
>   ├─ 定性 = Light's criteria（≥1 高 = 渗 exudate；三条全低 = 漏 transudate）
>   │     ├─ 漏 transudate → CHF心衰 / 肝硬化 / 肾病综合征（多双侧，pH 7.4–7.55）
>   │     └─ 渗 exudate → 肺炎 / 恶性 / TB / PE / 胰腺炎 / RA（多单侧）
>   │           ├─ 血性 → 恶性 / PE / TB
>   │           ├─ 高淀粉酶 → 胰腺炎 / 食管破裂
>   │           └─ 超低糖(<30) → RA / 脓胸 / TB / 恶性
>   │
>   └─ 肺炎 + 积液 → 要不要引流？
>         ├─ pH≥7.2 + 糖≥60 + 培养阴 + 自由流动 → uncomplicated → 只抗生素
>         └─ pH<7.2 / 糖<60 / 培养阳 / loculated / 脓 → complicated/empyema
>               → 抗生素 + 胸管引流（±胸腔内 tPA/DNase；分隔重 → VATS）
> ```
>
> **跳转**：完整体系（Light's / 三阶段 / 脓胸用药 / 5 道自测）→ [[完整笔记/专题笔记/pulmonary/pulmonary_胸腔积液与脓胸]] ｜ 胸液 pH/糖/淀粉酶指纹 → [[完整笔记/专题笔记/USMLE/USMLE_实验室指纹诊断大全]] ｜ 分科主笔记 → [[完整笔记/Peixuan分科笔记/pulmonary]]
>
> **5 条反射钩**：
> 1. **呼吸音↓ + 叩浊先看语颤**：↓ = 积液/气胸；↑ = 实变/肺炎（叩诊都浊，靠语颤 + 有无浸润影分）
> 2. **PE 积液是渗出（不是漏）**：制动 + 突发 + 无浸润影 → 先想 PE → CTPA 确诊（别误成吸入性肺炎）
> 3. **CHF 是漏出**：pH 偏碱 7.4–7.55、蛋白/LDH 低；Light 方向别记反（新发未利尿别选"渗出"）
> 4. **pH < 7.2 = 引流红线**：肺炎旁积液到这条线就要胸管，不能只抗生素观察
> 5. **儿童脓胸换三代头孢还不够**：中毒貌必须加万古覆盖 MRSA + 引流（脓胸 = 肺炎链球菌 + 金葡）

> [!info]- ② 急性呼吸困难 / 低氧三联：PE / 气胸 / ARDS 鉴别图（点开看 ASCII 总览 + 跳转链接）
> 突发气短 + 低氧来 ICU/急诊，先用「**体征 + 起病 + 影像**」把三大类切开。每个终点 = 一个反射动作。
>
> ```
> 急性呼吸困难 + 低氧
>   │
>   ├─ 单侧呼吸音消失 + 叩诊【过清音】+ 气管偏健侧 + 不稳 ──→ 张力性气胸
>   │      → 立即针刺减压（临床诊断，别等 CXR）→ 再胸管
>   │      └─ 稳定/小气胸（≤2cm）→ 吸氧观察；大/有症状 → 胸管
>   │      └─ 创伤后胸管「持续大漏气」→ 大气道损伤 → 支气管镜
>   │
>   ├─ 突发 + 胸膜性胸痛 + 危险因素（制动/术后/OCP/癌）+ 双肺清/小积液无浸润 ──→ PE
>   │      ├─ 不稳定 → 床旁 TTE 看 RV → massive PE → 溶栓 tPA（无禁忌）
>   │      └─ 稳定 → Wells：≤4 查 D-dimer / >4 直接 CTPA → 抗凝（稳的先抗凝）
>   │
>   └─ 诱因后 1 周内（脓毒症/误吸/胰腺炎）+【双肺】弥漫浸润 + 排心衰(PCWP≤18) ──→ ARDS
>          → 肺保护性通气：低潮气量 6 mL/kg IBW + 平台压<30 + PEEP（复张后别撤）
>          → 「CO₂ 看 VT/RR，O₂ 看 PEEP/FiO₂」；重度→俯卧位
> ```
>
> **跳转**：PE 全体系（Wells/CTPA/抗凝阶梯）→ [[完整笔记/专题笔记/pulmonary/pulmonary_肺栓塞PE]] ｜ 气胸全体系（自发 vs 张力/管理阶梯）→ [[完整笔记/专题笔记/pulmonary/pulmonary_气胸Pneumothorax]] ｜ ARDS 全体系（Berlin/肺保护通气）→ [[完整笔记/专题笔记/pulmonary/pulmonary_ARDS]] ｜ **梗阻性休克三因速辨（都 JVD↑+低BP：填塞 vs massive PE vs 张力性气胸）→ [[完整笔记/专题笔记/心内/心内_Shock分类与next-step]] §4.0** ｜ 分科主笔记 → [[完整笔记/Peixuan分科笔记/pulmonary]]
>
> **7 条反射钩**：
> 1. **张力性气胸 = 临床诊断先治后拍片**：不稳 + 过清音 + 气管偏健侧 → 立即针刺减压
> 2. **气管方向定性质**：气胸/积液推向健侧；肺不张拉向患侧
> 3. **持续大漏气 = 大气道损伤 → 支气管镜**（不是再加胸管/胸膜固定）
> 4. **PE 先判稳不稳**：不稳床旁 TTE→溶栓；稳定走 Wells→D-dimer/CTPA→抗凝
> 5. **高 Wells 别先查 D-dimer**（只用于低概率排除）；重度肾衰抗凝用普通肝素 UFH
> 6. **ARDS = 分流**：给氧效果差靠 PEEP；低潮气量降死亡率；复张后别撤 PEEP；与心源性肺水肿靠 PCWP/BNP/奔马律分
> 7. **三因都 JVD↑+低BP**：心音遥远→填塞（穿刺）；患侧呼吸音消失+气管偏健侧→张力性气胸（针刺减压）；双肺正常→massive PE（溶栓）—— 详见 Shock §4.0 速辨表

## 💡 标签设计哲学

> [!success] 为什么只用 5 个标签？
>
> 1. **每个标签必须对应一个具体的复习动作**
>    - 模糊的标签 = 复习时不知道怎么用
>    - "高频" 标签 → 知道但不知道怎么办
>    - "薄弱点" 标签 → **每天看一遍**（具体动作）
>
> 2. **主动标注 > 被动接受**
>    - 自己标的会记住
>    - 别人贴的标签 = 视觉噪音
>
> 3. **少而精 > 多而杂**
>    - 100 个标签 = 没标签
>    - 5 个标签 = 真正的筛选工具

## ⚙️ 故障排查

> [!warning] Embedded Search 不显示？

**检查 1：阅读模式**
- 必须在阅读模式（Reading View）才显示
- `Ctrl+E` 切换

**检查 2：标签真的存在？**
- `Ctrl+Shift+F` → `tag:#薄弱点`
- 有结果 → 标签存在
- 无结果 → 还没加

**检查 3：搜索语法**
- ✅ `tag:#薄弱点`（冒号后无空格）
- ❌ `tag: #薄弱点`

**检查 4：手动刷新**
- `Ctrl+R` 重新加载 vault

# 耳鼻喉 (Otolaryngology / ENT) 笔记

> UWorld Step 2 CK · 高频考点大纲

## 1. 耳 (Ear)
- 外耳：otitis externa (Pseudomonas)、恶性 OE (糖尿病)、cerumen impaction
- 中耳：AOM (PCN 一线、TM 鼓胀)、OME、急性乳突炎、cholesteatoma
- 内耳：sensorineural hearing loss、突发性 SNHL (steroid)、noise induced、presbycusis
- 鼓膜穿孔
- 听力学：Weber / Rinne 解读
- 前庭：BPPV (Dix-Hallpike → Epley)、Ménière、vestibular neuritis、labyrinthitis

## 2. 鼻 / 鼻窦 (Nose / Sinus)
- 过敏性 / 血管运动性鼻炎
- 急性鼻窦炎：病毒 vs 细菌 (≥10d 或加重)
- 慢性鼻窦炎、鼻息肉
- 鼻出血 (Kiesselbach、posterior、HHT)
- 鼻骨折、CSF 鼻漏
- Mucormycosis (糖尿病、免疫抑制)
- 嗅觉减退 / 丧失

## 3. 咽 / 喉 (Pharynx / Larynx)
- 咽炎：viral、GAS (Centor)、mononucleosis
- 扁桃体周围 / 咽后 / 咽旁脓肿
- 急性会厌炎 (Hib，少见但急症)
- Croup (PIV) vs epiglottitis vs bacterial tracheitis vs FB
- 声音嘶哑：声带麻痹、laryngitis、声带息肉 / 结节、laryngeal cancer (吸烟)
- 阻塞性睡眠呼吸暂停 (与肺章节交叉)

## 4. 颈部肿块
- 年龄分层鉴别：先天性 (thyroglossal、branchial)、感染性 (lymphadenitis)、肿瘤 (lymphoma、metastatic SCC、salivary)
- 评估流程 (FNA 优先于 excision)

## 5. 唾液腺
- Sialolithiasis (submandibular)、sialadenitis
- Mumps 与流行性腮腺炎
- 肿瘤：pleomorphic adenoma、Warthin、mucoepidermoid、adenoid cystic

## 6. 头颈肿瘤
- 口腔 / 喉 / 下咽 SCC (烟酒)、HPV 相关 oropharyngeal SCC
- 鼻咽癌 (EBV)
- 甲状腺 (与内分泌交叉)
- 早期症状：声嘶、吞咽困难、颈部肿块、单侧鼻塞 / 耳痛

## 7. 吞咽困难 (Dysphagia)
- 口咽 vs 食管；机械 vs 神经肌肉
- Zenker 憩室、CN 异常、achalasia (与 GI 交叉)

## 8. 紧急 ENT
- 气道阻塞 (FB、过敏、感染)
- 严重鼻出血 (后部出血、HHT)
- Ludwig's angina
- 听小骨创伤、迷路损伤

## 9. 儿科 ENT
- 反复 AOM / OME → 鼓膜置管
- 腺样体 / 扁桃体肥大 → OSA
- 喉气管软化
- FB 吸入 / 鼻内异物
- 儿科颈部肿块 (thyroglossal、branchial、cystic hygroma)

## 10. 眩晕鉴别 (与神经交叉)
- 周围 (BPPV、Ménière、neuritis、labyrinthitis)
- 中枢 (脑干、小脑卒中)；HINTS

# Ethics & Communication — Step 2 CK 速查笔记

> 核心思想:CK 的 Ethics **不是哲学思辨**,是**场景套路**。
> 同一类场景,答案永远一样。
> 全部内容 = 4 大原则(背景) + 高频场景表 + Communication 万能套路。

---

## 一、4 大原则(只需了解,不靠它推理)

| 原则 | 含义 | CK 倾向 |
|---|---|---|
| **Autonomy(自主)** | 病人自己决定 | **几乎永远胜出** |
| **Beneficence(行善)** | 为病人最大利益 | 次要 |
| **Non-maleficence(不伤害)** | 不造成伤害 | 次要 |
| **Justice(公正)** | 公平分配资源 | 罕见考 |

> 记住:CK 选项里出现"尊重病人意愿"——大概率是答案。

---

## 二、Capacity vs Competence

| 概念 | 谁判断 | 含义 |
|---|---|---|
| **Capacity** | 主管医生(任何医生都可) | **临床判断**,任何医生都能评 |
| **Competence** | 法官 | **法律概念**,法庭裁定 |

### Capacity 的 4 个评估要素

1. 病人能**理解**信息
2. 能**表达**选择
3. 能**理性推理**
4. 能**理解后果**(appreciate consequences)

> CK 高频陷阱:看似"非理性"的决定不等于没 capacity。
> 例:Jehovah's Witness 拒输血、老人拒绝化疗——**只要符合 4 要素,就尊重**。

---

## 三、知情同意(Informed Consent)4 要素

1. **Capacity**(有判断能力)
2. **Disclosure**(医生告知信息)
3. **Understanding**(病人理解)
4. **Voluntariness**(自愿,无胁迫)

### 不需要知情同意的例外

- **紧急情况**(emergency,病人无意识或生命危急)
- **Therapeutic privilege**(极罕见,告知会严重伤害病人)— **CK 几乎不选这个**
- **病人 waive**(主动放弃)
- **Public health emergency**(强制隔离/治疗某些传染病)

---

## 四、代理决策(Surrogate)顺序

当病人没有 capacity 且无 advance directive 时:

1. **配偶(Spouse)**
2. **成年子女(Adult children)**
3. **父母(Parents)**
4. **成年兄弟姐妹(Adult siblings)**
5. **其他亲属**

### 决策标准(由高到低)

1. **Substituted judgment**(替代判断):病人有 capacity 时会怎么选?
2. **Best interest**(最佳利益):没有依据时,选对病人最好的

## 五、高频场景标准答案表(背就完了)

### 决策能力 & 知情同意

| 场景 | 标准答案 |
|---|---|
| 病人有 capacity,拒绝治疗 | **尊重决定**,即使会死 |
| 病人有 capacity,选择"非理性"方案 | **尊重决定** |
| 病人无 capacity,有 advance directive | **按 directive 执行** |
| 病人无 capacity,无 directive,有家属 | **按 surrogate 顺序** |
| 家属和病人意愿冲突(病人有 capacity) | **病人意愿优先** |
| 家属说"不要告诉病人诊断" | **拒绝**,病人有知情权(除非病人主动放弃) |
| 病人主动说"我不想知道" | **尊重**,可指定家属代为接收信息 |
| Capacity 评估 | **主管医生评**,精神科只在复杂时介入 |

### 未成年人(Minors)

| 场景 | 标准答案 |
|---|---|
| Minor 一般医疗 | 需要**家长同意** |
| Minor 要求避孕 | **不需要家长同意**(所有州) |
| Minor 要求 STI 检查/治疗 | **不需要家长同意** |
| Minor 怀孕相关护理 | **不需要家长同意** |
| Minor 物质成瘾治疗 | **不需要家长同意**(多数州) |
| Minor 精神健康(部分州) | 通常**不需要家长同意** |
| Minor 堕胎 | 因州而异,CK 选"按州法律" |
| **Emancipated minor**(已婚/参军/经济独立/法院裁定/有孩子) | **自己全权决定**,无需家长 |
| Minor 紧急情况 | **立即治疗**,事后通知家长 |
| 家长拒绝救命治疗(如输血给孩子) | **法院介入,治疗** |
| 家长拒绝非紧急但有益治疗(如疫苗) | **教育沟通**,通常尊重家长 |
| 14 岁少女要求避孕,母亲反对 | **给避孕药,保密** |

### 保密 & 强制上报

| 场景 | 标准答案 |
|---|---|
| 怀疑虐待儿童 | **强制上报**,不需要同意 |
| 怀疑虐待老人 | **强制上报**(多数州) |
| 怀疑家暴(成年伴侣) | **不强制**,提供资源,尊重选择(因州而异) |
| 病人有传染病(TB、某些 STI) | **上报公共卫生** |
| 病人 HIV+ 不告诉伴侣 | 1️⃣ 先**劝病人告知** → 2️⃣ 不行**通过 public health 通知** |
| 病人威胁要伤害特定他人 | **Tarasoff 原则:警告潜在受害者 + 通知警方** |
| 病人有自杀风险 | **不可让其离开**,精神科评估,必要时 involuntary hold |
| 司机有癫痫/视力问题不能开车 | **劝其停止 + 报告 DMV**(因州而异,多数选报告) |
| 同事酗酒上班 | **报告主管/医院**(impaired physician) |
| 同事偷药 | **报告** |

### 错误处理 & Disclosure

| 场景 | 标准答案 |
|---|---|
| 自己犯了医疗错误 | **主动告知病人 + 上报系统** |
| 同事犯错 | **先和同事本人谈** → 不行再上报 |
| 病人问"是不是出错了" | **诚实告知** |
| 错误被掩盖了想披露 | **披露**,无论后果 |

### 临终关怀

| 场景 | 标准答案 |
|---|---|
| 撤呼吸机 / 不上呼吸机 | **法律和伦理上等同**,都允许 |
| 病人要求"安乐死"(主动) | **大部分州非法**,即使要求也拒绝 |
| Physician-assisted suicide | 仅在合法州才考虑;**CK 默认拒绝** |
| 撤水/食(无 capacity 的植物人) | 按 advance directive 或 surrogate 决定 |
| DNR 不等于"不治疗" | DNR **只**意味着不复苏,其他治疗继续 |
| 双重效应(给吗啡缓痛可能加速死亡) | **可以做**,只要意图是缓痛 |

### Communication(沟通题)

| 场景 | 标准答案 |
|---|---|
| 病人哭/愤怒/恐惧 | **先 acknowledge 情绪**,不立刻给信息 |
| "Don't worry, it'll be fine" | **永远是错的**(空洞 reassurance) |
| 病人不会英语 | **用专业翻译**,不用家属(尤其不能用孩子) |
| 病人听不懂医学术语 | 用大白话重说,**不要让家属代翻译** |
| 病人不依从治疗 | **探究原因**,不直接换医生/责怪 |
| 病人要求做不必要的检查 | **教育解释**,不直接拒绝或满足 |
| 病人否认明显问题(如酗酒) | **共情 + 开放式问题**,不对抗 |
| 病人想换医生 | **尊重**,提供帮助 |
| Bad news 告知 | **SPIKES 法**(见下) |

### 礼物 & 边界

| 场景 | 标准答案 |
|---|---|
| 病人送小礼物(贺卡、烘焙) | **可接受** |
| 病人送贵重礼物 | **拒绝** |
| 病人邀请约会(治疗中) | **拒绝**,继续治疗 |
| 病人邀请约会(治疗结束后) | **拒绝**(精神科病人**永远**禁止) |
| 给家人朋友看病 | **避免**,除非紧急 |
| Pharma rep 请吃饭 / 送礼 | **拒绝** |
| 病人提供商业机会(投资等) | **拒绝** |

### 文化 & 宗教

| 场景 | 标准答案 |
|---|---|
| 成年 Jehovah's Witness 拒输血 | **尊重**(有 capacity) |
| 父母 JW 拒给孩子输血 | **法院介入,输血** |
| 病人坚持传统/替代医学 | **不评判,提供医学信息**,共同决策 |
| 文化原因家属要求隐瞒诊断 | 先**问病人**愿不愿意知道,按病人意愿 |
| 男医生 vs 女病人(伊斯兰文化) | **尊重偏好**,如可能换医生 |

### 研究伦理

| 场景 | 标准答案 |
|---|---|
| 研究招募 | 需要 **IRB 批准 + 知情同意** |
| 弱势群体(囚犯、儿童、孕妇)研究 | **额外保护** |
| 治疗中发现新有效药 | **告知所有受试者**,可能终止试验 |
| Placebo 控制 | 仅当**无现有有效治疗**时可用 |

## 六、Communication 万能套路:SPIKES(告知坏消息)

| 字母 | 步骤 | 做什么 |
|---|---|---|
| **S** — Setting | 设置环境 | 私密空间、坐下、关门 |
| **P** — Perception | 了解病人认知 | "您对自己情况了解多少?" |
| **I** — Invitation | 询问意愿 | "您想知道详情吗?" |
| **K** — Knowledge | 告知信息 | 用大白话,**先 warning shot** |
| **E** — Emotion | 回应情绪 | **acknowledge,不打断** |
| **S** — Strategy/Summary | 制定计划 | 总结 + 下一步 |

### "正确选项"的语言模式

CK 的 communication 题里,**正确答案通常包含**:

- "I can see this is very difficult for you..."
- "Tell me more about your concerns."
- "What worries you most?"
- "How are you coping with this?"
- "Let's talk about what this means for you."

### "错误选项"的语言模式

- "Don't worry, everything will be fine." ❌
- "You shouldn't feel that way." ❌
- "Let me refer you to..." (太早转介) ❌
- "You need to..." (命令式) ❌
- 立刻给一堆医学信息(没先回应情绪) ❌

## 七、考试题型与套路

### 题型 1:Capacity 判断

- 病人做"非理性"决定 → 检查是否符合 capacity 4 要素
- 符合就**尊重**,不符合才介入

### 题型 2:Surrogate 决策

- 按顺序找:配偶 → 成年子女 → 父母 → 兄弟姐妹

### 题型 3:Minor 同意

- 涉及避孕/STI/孕产/成瘾/精神 → 通常不需家长
- 一般医疗 → 需家长(emancipated 例外)
- 紧急 → 立即治疗

### 题型 4:Disclosure / 强制上报

- 虐待儿童/老人 → 强制
- 传染病 → 强制
- 威胁伤人(具体目标) → Tarasoff
- 自伤/自杀 → 紧急介入

### 题型 5:Communication

- 永远先 **acknowledge feeling + open-ended question**
- 永远不是 reassurance、转介、命令

### 题型 6:End-of-life

- 撤治疗 = 不上治疗(伦理等同)
- 安乐死/PAS → 默认拒绝
- 双重效应 → 可以(意图是缓痛)
- DNR ≠ 不治疗

## 十、终极心法

CK 的 Ethics:

1. **病人有 capacity → 听病人的**
2. **病人没 capacity → 找 directive,再找 surrogate**
3. **保密原则 → 除非:虐待、传染、威胁他人、自伤**
4. **沟通题 → 先承认情绪,再说话**
5. **错误 → 诚实披露**
6. **拿不准 → 选最尊重病人 + 最像"开放式对话"的选项**

> 90% 的题用上述 6 条就能选出来。

# 妊娠、分娩与产褥 (Pregnancy, Childbirth & Puerperium) 笔记

> UWorld Step 2 CK · 高频考点大纲

## 1. 产前保健
- 初次产检项目：血型 / Rh、CBC、Rubella、HBsAg、HIV、syphilis、UA / 尿培养、Pap、GC / Chlamydia、TSH (高危)
- 孕期访视频率
- 推算孕周 (LMP vs first trimester US)
- 营养：folate、iron、calcium；体重增加范围
- 疫苗：Tdap (27–36 周)、流感、COVID；禁用减毒
- 用药安全分类与常见禁忌

## 2. 产前筛查与诊断
- 非整倍体：first trimester (NT + PAPP-A + β-hCG)、quad、cell-free DNA
- 神经管：MSAFP、anatomy scan
- 基因 / 携带者筛查 (CF、SMA、地中海贫血等)
- 侵入性：CVS (10–13 周)、amniocentesis (≥15 周)
- GBS：35–37 周；GDM：24–28 周 (1h GCT → 3h)

## 3. 早孕并发症
- 异位妊娠：风险 (PID、IUD、既往 ectopic、IVF)；MTX vs 手术
- 流产分类：threatened、inevitable、incomplete、complete、missed、septic
- 葡萄胎 (complete vs partial)；GTN

## 4. 中晚孕并发症
- 子痫前期 / 子痫 / HELLP；MgSO₄；分娩时机
- 妊娠糖尿病；A1 vs A2；分娩管理
- 早产 (PTL)：tocolysis、betamethasone、MgSO₄ 神经保护
- PPROM vs PROM
- 胎盘前置、placenta accreta 谱、abruption；vasa previa
- IUGR / SGA / LGA、polyhydramnios / oligohydramnios
- 妊娠剧吐、ICP (胆酸 ↑、瘙痒)、AFLP

## 5. 围产期感染
- TORCH (toxoplasma、other = syphilis/parvo/listeria、rubella、CMV、HSV)
- HIV、HBV、HCV 母婴传播预防
- GBS：IAP 适应证
- Zika、parvo B19 (hydrops)、varicella、listeria

## 6. 分娩 (Labor & Delivery)
- 三个产程；Friedman、Zhang
- 监护：CTG 分类 (I/II/III)、胎心异常处理
- 引产 (Bishop)、催产
- 分娩异常：肩难产、umbilical cord prolapse、breech、shoulder dystocia (HELPERR)
- 镇痛麻醉
- 剖宫产指征、VBAC 与禁忌

## 7. 产后 (Puerperium)
- PPH (4T：tone、trauma、tissue、thrombin)；阶梯处理
- Retained POC、子宫复旧不全
- 产褥感染：endometritis、septic pelvic thrombophlebitis
- 乳腺：engorgement、mastitis、abscess、blocked duct
- 产后抑郁 / 焦虑 / psychosis；baby blues
- Sheehan
- 静脉血栓栓塞预防

## 8. Rh 同种免疫
- 28 周与产后 RhIG；其他指征 (流产、ectopic、出血、CVS / amnio、外伤)
- 已致敏的随访 (titer、MCA Doppler)

## 9. 多胎 / 过期
- DCDA / MCDA / MCMA；TTTS
- 过期妊娠 (≥41 周) 引产策略

## 10. 哺乳
- 益处、禁忌 (HIV 高资源国、活动性 HSV 乳房、active chemo / radio、某些药物)
- 常见困扰处理

## 11. 围产期心理 / 社会
- IPV 筛查、青少年妊娠
- 妊娠相关精神药物原则 (与精神章节交叉)

## 12. 产前 / 产后高频急症
- 子痫发作处理
- 大出血
- 羊水栓塞
- VTE、PE
- 心脏疾病合并妊娠 (peripartum cardiomyopathy)

# 生物统计与流行病学 (Biostatistics & Epidemiology) 笔记

> UWorld Step 2 CK · 高频考点大纲

## 1. 研究设计 (Study Design)
- RCT (gold standard)、crossover、cluster
- 队列研究 (前瞻 / 回顾)
- 病例对照
- 横断面
- 病例系列 / 报告
- 系统综述与 meta 分析；证据等级

## 2. 测试性能 (Test Performance)
- Sensitivity = TP / (TP + FN)
- Specificity = TN / (TN + FP)
- PPV / NPV (受患病率影响)
- LR+ = sens / (1 - spec)；LR- = (1 - sens) / spec
- ROC、AUC
- Cutoff 移动对 sens/spec 的影响

## 3. 风险测度 (Measures of Association)
- RR (cohort)、OR (case-control)、HR
- ARR、RRR、NNT、NNH
- Attributable risk、PAR

## 4. 偏倚 (Bias)
- Selection (Berkson、healthy worker、non-response)
- Information / measurement (recall、observer、Hawthorne、Pygmalion)
- Lead-time / length-time bias (筛查)
- Confounding vs effect modification

## 5. 统计推断
- 假设检验：H₀、H₁
- p 值、α、β、power = 1 - β
- Type I (α) vs Type II (β) 错误
- 95% 置信区间解读 (含 1 / 0)
- 检验选择：t 检验、ANOVA、χ²、Fisher、Pearson、Spearman、回归

## 6. 描述统计
- 均值 / 中位 / 众数；偏态分布
- 方差、SD、SE

## 7. 流行病学指标
- Incidence、prevalence (慢性病 ↑prevalence)
- Mortality (case fatality vs mortality rate)
- Outbreak investigation 步骤
- 疾病预防：一级 / 二级 / 三级

## 8. 筛查
- Wilson-Jungner 标准
- 美国推荐筛查 (USPSTF)：CRC、breast、cervical、lung、AAA、HTN、DM、HIV、HCV

## 9. 临床决策
- Bayesian 思维：pretest → posttest 概率
- 决策分析、敏感性分析

## 10. 伦理 / 研究规范
- IRB、informed consent、equipoise
- Helsinki / Belmont 原则
- 数据完整性、利益冲突

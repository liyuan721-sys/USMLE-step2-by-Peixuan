# 神经系统 (Nervous System) 笔记

> UWorld Step 2 CK · 高频考点大纲

## 1. 脑血管 (Cerebrovascular)
- 缺血性卒中：大血管 vs 腔隙；定位 (ACA / MCA / PCA / 脑干 / 小脑)
- TIA、ABCD²
- 出血：高血压性 (基底节 / 丘脑 / 脑桥 / 小脑)、脑叶 (淀粉样)
- SAH：thunderclap、CT → LP (xanthochromia)、aneurysm clip / coil；vasospasm (nimodipine)
- 颅内静脉血栓、CVD 危险因素管理
- tPA / 取栓时间窗

## 2. 头痛
- Migraine ± aura、abortive vs prophylactic
- Tension、cluster
- 二级头痛红旗征：thunderclap、>50 岁新发、神经功能缺损、晨起、姿势相关
- IIH、temporal arteritis (ESR ↑ → steroid → biopsy)
- Trigeminal neuralgia

## 3. 癫痫与意识
- 局灶 vs 全面发作；EEG / MRI
- 状态癫痫：benzo → fos/PHT/levetiracetam
- 抗癫痫药副作用
- 晕厥鉴别 (心源性 / 神经介导 / 直立)

## 4. 神经退行性
- Alzheimer、vascular dementia、Lewy body、FTD、PSP、CBD
- NPH (三联：尿失禁、步态、痴呆)
- Parkinson 病、parkinsonism、Huntington、Wilson、essential tremor、tics
- ALS、SMA

## 5. 脱髓鞘 / 自免
- MS：临床型、MRI、OCB、disease modifying
- ADEM、NMO、PML
- GBS、CIDP、MMN
- MG、LEMS (与肿瘤关联)
- Autoimmune encephalitis (NMDA-R、anti-LGI1)

## 6. 周围神经 / 神经丛
- CTS、tarsal tunnel、ulnar、radial、femoral、common peroneal
- Bell's palsy、Ramsay Hunt
- Diabetic、alcoholic、B12、CMT、vasculitic
- Brachial / lumbosacral plexopathy

## 7. 脊髓
- 急性脊髓压迫 / cauda equina
- 横贯性脊髓炎、ATM
- 前 / 后 / 中央索综合征、Brown-Séquard
- B12 (SCD)、tabes dorsalis、syringomyelia

## 8. 肌病 / NMJ
- 炎症性肌炎 (DM/PM/IBM)
- 肌营养不良 (Duchenne、Becker、myotonic)
- 周期性瘫痪、rhabdomyolysis
- MG (AChR-Ab) vs LEMS (P/Q VGCC)

## 9. 颅内压 / 创伤
- TBI、concussion、SDH、EDH、contusion、DAI
- ICP 升高处理 (head up、osmotic、hyperventilation 短期)
- 脑死亡判定

## 10. CNS 感染 / 肿瘤
- 脑膜炎 / 脑炎 (与感染章节交叉)
- 原发：glioma、meningioma、pituitary
- 转移：肺、乳腺、黑素瘤、肾、CRC

## 11. 眩晕 (Vertigo)
- 周围 (BPPV、Ménière、vestibular neuritis、迷路炎) vs 中枢 (脑干、小脑)
- HINTS 检查

## 12. 儿科神经
- 发热惊厥
- 脑瘫、神经管缺陷
- 神经皮肤综合征：NF1、NF2、TS、Sturge-Weber、VHL
- 髓母细胞瘤、室管膜瘤
- 小儿头痛、ADHD/学习障碍 (与精神交叉)

## 13. 睡眠
- OSA / CSA、narcolepsy、RBD、restless legs

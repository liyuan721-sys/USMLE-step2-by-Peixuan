# ITGB7 靶向纳米抗体（VHH）早期药物发现调研

> [!info] 本笔记结构
> Part A = PPT 调研报告要点（36 页凝练）
> Part B = 疾病发病原理（UC / GVHD / CD，USMLE 视角）
> Part C = 药物设计逻辑（从病理反推分子参数）
> Part D = 核心数据 + 竞品教训

> [!note]- 缩写全称对照（Abbreviations）
> - **VHH** = 纳米抗体 / 重链抗体可变区（Variable domain of Heavy-chain-only antibody）
> - **ITGB7** = 整合素 β7 亚基（Integrin Subunit Beta 7）
> - **UC** = 溃疡性结肠炎（Ulcerative Colitis）
> - **CD** = 克罗恩病（Crohn's Disease）
> - **GVHD** = 移植物抗宿主病（Graft-versus-Host Disease）；aGVHD = 急性（acute）；SR-GVHD = 类固醇难治性（Steroid-Refractory）
> - **allo-HSCT** = 异基因造血干细胞移植（allogeneic Hematopoietic Stem Cell Transplantation）
> - **MAdCAM-1** = 黏膜地址素细胞黏附分子-1（Mucosal Addressin Cell Adhesion Molecule-1）
> - **TIL** = 肿瘤浸润淋巴细胞（Tumor-Infiltrating Lymphocyte）
> - **Treg** = 调节性 T 细胞（Regulatory T cell）
> - **ADCC** = 抗体依赖的细胞毒性（Antibody-Dependent Cellular Cytotoxicity）
> - **FTO** = 自由实施（Freedom To Operate，专利可自由实施空间）
> - **NHP** = 非人灵长类（Non-Human Primate）
> - **PoC** = 概念验证（Proof of Concept）
> - **RO** = 受体占位（Receptor Occupancy）
> - **PML** = 进行性多灶性白质脑病（Progressive Multifocal Leukoencephalopathy）
> - **MS** = 多发性硬化（Multiple Sclerosis）

## Part A｜PPT 调研报告要点

一句话主线：**用"半衰期延长型抗 α4β7 纳米抗体（Alb-VHH，Design-2）"从 UC 切入、以 SR-GVHD 为快速通道，通过差异化表位 + 长效 PK 去改进已被维多珠单抗（vedolizumab）验证但缓解率不足的 α4β7 通路。**

### 1. 靶点生物学画像

- **ITGB7（Integrin Subunit Beta 7，整合素 β7 亚基）** 本身不单独起作用，必须与 α4 或 αE 配对成异源二聚体：
  - **α4β7** → 结合 **MAdCAM-1**，介导淋巴细胞肠道归巢（homing）
  - **αEβ7（CD103）** → 结合 **E-Cadherin（E-钙粘蛋白）**，上皮内淋巴细胞驻留
- 成药核心优势：**表达高度局限于免疫细胞**（肠道归巢 T 细胞等），不广泛分布于实质器官 → 天然规避脱靶毒性 (off-target toxicity)。
- 三种构象状态是 VHH 功能表位选择的底层逻辑：
  - 弯曲静息态 (Bent-closed) → 负别构调节，锁定非活化态
  - 伸展活化态 (Extended-closed) → 活化依赖性阻断
  - 配体结合态 (Extended-open) → 暂无干预靶点
- β-I 结构域（含 **ADMIDAS 金属离子结合位点**）= 竞争性阻断 VHH 首选靶区。

### 2. 疾病介入逻辑

ITGB7 在不同疾病中角色不同：

```
ITGB7 相关疾病谱
├── 自身免疫（UC / CD）      → α4β7+ T 细胞过度浸润肠道，慢性炎症
├── 医源性（aGVHD）          → 供体 α4β7+ T 细胞攻击受体肠道
├── 病毒感染（HIV/SIV）      → 优先靶向 α4β7+ T 细胞，建立肠道病毒储存库
└── 肿瘤（实体瘤 TIL）       → αEβ7+ TIL 高表达 = 良好预后标志物
```

> [!success] 核心推断
> **阻断型策略在 IBD 与 GVHD 领域风险收益比最优**，临床验证最充分，是 VHH 首选切入路径。维多珠单抗已验证机制，但仍有 **60–70% 患者无法缓解** → 巨大优化空间。

### 3. 竞争格局与失败教训

- **维多珠单抗 (Vedolizumab / Entyvio)**：靶向 α4β7 二聚体，高度肠道选择性，UC + CD 均获批，安全性卓越。
- **埃特利珠单抗 (Etrolizumab)**：靶向 β7 亚基（同时阻断 α4β7 和 αEβ7），UC 获欧盟批准，但 **CD III 期失败**（2024 Roche 公告）。

> [!danger] 关键教训：β7 全阻断 ≠ 双倍疗效
> 盲目扩大靶点范围可能干扰 **αEβ7+ 调节性 T 细胞的保护功能**，反而缩窄治疗窗。精准、适度的通路干预才是关键。

VHH 开发六条"避坑备忘录"：优先注射给药 / 锁定 α4β7 特异表位 / 首选 UC 为第一适应症 / Fc 沉默（VHH 天然无 Fc 是优势）/ 早做表位 Binning / 用人源化小鼠或 NHP 模型。

### 4. 纳米抗体机会窗口（7 个设计假说）

| 设计 | 构型/机制 | 战略定位 |
|---|---|---|
| **Design-2 Alb-VHH 长效阻断** | 抗 α4β7 VHH + 抗白蛋白 VHH 串联 (~30 kDa)，2–4 周皮下注射 | **第一优先**（药效 + 成药性最优解） |
| Design-3 二价串联阻断 | 两个抗 α4β7 VHH，亲合力增强 10–100× | 高价值备份 |
| Design-4 负别构调节（静息态锁定） | 单价 VHH 靶向弯曲静息态特有表位 | FTO 专利优势最强，中长期差异化 |
| Design-1 单体阻断 | 单价 VHH (~15 kDa) 竞争阻断 | 机制验证，半衰期短难直接成药 |
| Design-5 pH 敏感 | 炎症酸性微环境重获高亲和力 | 探索性 |
| Design-6 双特异性 | 抗 α4β7 × 抗 TNFα/IL-23 | 高风险高回报，待核心管线验证后启动 |
| Design-7 αEβ7 选择性 | 靶向 CD103，拓展乳糜泻 (celiac disease)/移植排斥 | 探索性 |

### 5. 早期发现实验路线图

- **抗原策略**：必须用**完整胞外域异源二聚体（α4 + β7）**保留天然构象。
- 五大陷阱：抗原截短 / 标签遮蔽表位 / 构象不均一 / 糖基化异质性 / **种属差异（致命陷阱）**——需用 α4β7 人源化小鼠或筛选能与猴交叉反应的 VHH。
- 筛选：羊驼免疫库 → 固相正筛 + 竞争洗脱 + 细胞正筛/反筛。
- 可开发性阈值：KD ≤ 5 nM、Tm ≥ 65°C、单体 ≥ 95%、PSR < 3/10、表达收率 ≥ 50 mg/L。

### 6. 适应症布局与风险

- **首选：中重度 UC（溃疡性结肠炎）** —— 致病角色确证度最高，市场空间大。
- **次选：类固醇难治性 aGVHD（SR-GVHD）** —— 孤儿药监管门槛低，vedolizumab 超说明书已显前景，可快速切入。
- 三大风险：R1 无法临床优效 / R2 动物模型预测差 / R3 免疫原性过高（各配 Go/No-Go 终止标准）。
- 里程碑 M1–M5，覆盖月 0–20（抗原制备 → 初筛 Hit → 功能验证 → 成药性 → 体内 PoC）。

### 7. 立项核心建议

- **首选分子**：Design-2（Alb-VHH 长效 α4β7 阻断），2–4 周一次皮下注射。
- **首选路径**：UC 第一适应症 + 同步 SR-GVHD 孤儿药加速审评。
- **双重差异化**：表位创新（靶向非 Entyvio 竞争的**静息态构象裂隙表位**）+ PK 创新（Alb 融合长效）。
- **生物标志物**：肠道 α4β7 受体占位 (RO) + 黏膜 T 细胞浸润减少。
- **价值主张**：**"同靶点、新深度、优安全"**。

## Part B｜疾病发病原理（USMLE 视角）

### UC（溃疡性结肠炎 Ulcerative Colitis）—— 自身免疫，慢性

1. 遗传易感 + 肠道菌群失衡 + 上皮屏障受损 → 对自身菌群/抗原耐受破坏。
2. **自身来源**的效应/记忆 T 细胞（**α4β7+**）被激活，从血液归巢到肠黏膜。
3. 穿内皮进入固有层，释放 TNF-α、IL-13、IL-23 等 → 慢性黏膜炎症、隐窝脓肿、溃疡。
4. 特点：**连续性、从直肠向近端、只累及黏膜/黏膜下层（表浅）**、bloody diarrhea（血性腹泻）、pANCA 倾向；并发 toxic megacolon（中毒性巨结肠）、结肠癌 (colorectal cancer) 风险。

### GVHD（移植物抗宿主病 Graft-versus-Host Disease）—— 医源性，急性

指报告聚焦的急性肠道 aGVHD / 类固醇难治性 SR-GVHD：

1. **异基因造血干细胞移植 (allo-HSCT)** 后，输入的是**供体 (donor)** 免疫细胞。
2. 供体成熟 T 细胞把**受体 (host)** 组织识别为异己（HLA / 次要组织相容抗原差异）。
3. 供体 **α4β7+ T 细胞**同样利用归巢通路大量涌入受体肠道。
4. 靶向攻击肠上皮 → 严重上皮损伤、水样/血性腹泻、黏膜脱落（aGVHD 高死亡率主因之一）。

> [!warning] UC vs GVHD 本质区别（但共用同一道"肠道门禁"）
> - **T 细胞来源**：UC = 自身 T 细胞攻击自身肠道；GVHD = 供体 T 细胞攻击受体肠道
> - **病程**：UC 慢性反复；aGVHD 急性、移植后数周内爆发
> - **共同点**：都靠 **α4β7–MAdCAM-1** 这道门进入肠道 → 共用靶点的基础

### 共享通路（药物靶点所在）

```
          致病 T 细胞（UC=自身 / GVHD=供体）
                    │  表面表达 α4β7 整合素
                    ▼
       肠道血管内皮的 MAdCAM-1  ←"锁"
                    │  α4β7 ="钥匙"，特异性结合
                    ▼
        T 细胞归巢 → 穿内皮 → 浸润肠黏膜
                    │
                    ▼
              释放炎症因子 → 组织损伤
```

**α4β7–MAdCAM-1 = 免疫细胞进入肠道的"门禁系统"。** 无论 T 细胞来源是自身还是供体，想进肠道就必须走这道门。

### CD vs UC 的"治疗窗"差异（为什么首选 UC）

> [!question]- 为什么 CD 的 α4β7 治疗窗比 UC 窄？
> "治疗窗"在此指**该通路能带来的临床获益空间**（管不管用、能治好多少人），不是药理学毒性区间。
>
> | | UC | CD |
> |---|---|---|
> | 累及深度 | 黏膜/黏膜下（表浅） | **透壁性 transmural（全层）** |
> | 分布 | 连续、局限结肠 | **跳跃性**、口到肛 |
> | 炎症驱动 | 相对单一依赖 α4β7 归巢 | **多通路冗余**（TNF、IL-23/Th17、纤维化） |
> | 并发 | — | 瘘管 fistula、狭窄 stricture、脓肿 |
>
> CD 是透壁 + 多通路驱动，α4β7 只管"挡新 T 细胞归巢"，挡不住深层已驻留、靠别的通路维持的炎症；瘘管/狭窄是结构性损伤更无法逆转 → vedolizumab 在 CD 起效慢、缓解率低。证据：etrolizumab CD III 期失败、V565 Phase IIb 失败。

## Part C｜药物设计逻辑（从病理反推分子参数）

核心思路：**第一性原理，从病理机制反推分子设计的硬性约束。**

| 病理事实 | 推导出的设计约束 | 具体参数 |
|---|---|---|
| 致病 T 细胞靠 α4β7 入肠 | **功能 = 竞争性阻断** | 阻断 α4β7–MAdCAM-1 结合 |
| 需持续高效占住受体 | **超高亲和力** | KD < 1 nM |
| 慢病需长期维持给药 | **长半衰期** | 2–4 周（抗白蛋白 VHH 融合 = Design-2） |
| 炎症在肠壁深处 | **强组织穿透** | 小分子量 VHH 穿透致密肠黏膜 |
| Treg 保护性细胞需保留 | **Fc 沉默/缺失**（避免 ADCC 误杀 Treg） | VHH 天然无 Fc = 天生优势 |
| β7 全阻断误伤 αEβ7+ 保护细胞 | **只锁 α4β7，不碰 αEβ7** | 靶向 α4β7 特异性表位 |

**为什么阻断"入肠"就能治病**：UC 和 GVHD 的损伤都来自 T 细胞**物理进入肠道**这一步。在"门口"挡住致病 T 细胞 → 进不了黏膜 → 无法释放炎症因子/攻击上皮 → 从上游切断炎症链。且只挡肠道归巢、不做全身免疫抑制 → 安全性远好于类固醇/系统性免疫抑制剂（这也是 GVHD 选它的理由：需要**肠道选择性**免疫抑制而非全身压制）。

## Part D｜核心数据 + 竞品对比

| 指标 | 数值 | 备注 |
|---|---|---|
| 维多珠单抗 UC 诱导缓解率 (W6) | 47.1% vs 安慰剂 25.5% | NCT00783718 (2013) |
| 维多珠单抗 UC 维持缓解率 (W52) | 41.8% vs 安慰剂 15.9% | NCT00794508 (2014) |
| 维多珠单抗全球销售额 | ~$7B | 2024 |
| IBD 生物制剂市场规模 | ~$25B | 2025 预测 |
| 抗白蛋白 VHH 专利到期 | 2026–2027 | 关键机遇窗口 |
| V565 项目 | Phase IIb 失败 | 未达主要终点 (2021) |
| Etrolizumab CD 项目 | III 期终止 | 2024 Roche 公告 |

> [!tip] USMLE Step 2 CK 高频考点：肠道选择性整合素拮抗剂
> - **Vedolizumab（抗 α4β7）**：gut-selective，用于对 anti-TNF 失败/不耐受的 IBD；**无 PML 风险**（不影响中枢归巢）。
> - **那他珠单抗 (Natalizumab，抗 α4)**：同时阻断 α4β1（影响脑归巢）→ **PML / JC 病毒 (JC virus)** 风险（高频陷阱），用于多发性硬化 (MS) 和难治性 CD。
> - **Etrolizumab（抗 β7）**：全阻断 α4β7 + αEβ7，CD III 期失败。
> - 记忆钩子：**α4 广谱 = PML 风险；α4β7 肠道选择性 = 安全**。

# 错题本（Dataview 聚合视图）

> [!info] 工作机制
> - **数据源**：`mistakes/uworld-mistakes.md`（按日期 `##` 分节，每张错题 = 一个 callout，title 行下一行有 `^Q#` 锚点）
> - **元数据字段**：`学科:: / 主题:: / 状态:: / 错因::`（沉底到 callout 末尾）
> - 本文件 **不存数据** — 只查询数据源 + 自动渲染表格
> - **表格里 Q# / 主题 都是可点击链接** — 点了直接跳到 mistakes 文件对应的卡片
> - 修改卡片 / 改字段 → 去 `mistakes/uworld-mistakes.md`
> - 字段空值规则：状态空 → 默认 🟡；错因空 → 显示 `(未填)`

> [!tip] 字段取值范围（CLAUDE.md § 七）
> - **学科**：Biostats / Ethics / 心血管 / 肺 / 肾 / Heme-Onc / 感染 / GI / 内分泌 / GYN / OB / 神经 / Psych / 儿科 / 风湿 / 皮肤 / 眼科 / 外科 / 骨科
> - **状态**：🔴 反复错 / 🟡 不熟（默认） / 🟢 已掌握
> - **错因**：知识 / pattern / 干扰项 / 粗心 / 计算

## 📊 1. 总览

```dataviewjs
const file = app.vault.getAbstractFileByPath("mistakes/uworld-mistakes.md");
const content = await app.vault.cachedRead(file);
const cards = [];
let cur = null;
for (const line of content.split("\n")) {
    const m = line.match(/^> \[!example\]- \[([\d-]+)\] (.+?) \/ (.+?) \(([^)]+)\)\s*$/);
    if (m) {
        if (cur) cards.push(cur);
        cur = { 日期: m[1], 学科t: m[2].trim(), 主题t: m[3].trim(), qid: m[4].trim(), 学科: null, 主题: null, 状态: null, 错因: null };
    } else if (cur) {
        let mm;
        if ((mm = line.match(/^> 学科:: ?(.*)$/))) cur.学科 = mm[1].trim() || cur.学科t;
        else if ((mm = line.match(/^> 主题:: ?(.*)$/))) cur.主题 = mm[1].trim() || cur.主题t;
        else if ((mm = line.match(/^> 状态:: ?(.*)$/))) cur.状态 = mm[1].trim() || "🟡";
        else if ((mm = line.match(/^> 错因:: ?(.*)$/))) cur.错因 = mm[1].trim() || "(未填)";
    }
}
if (cur) cards.push(cur);
for (const c of cards) {
    if (!c.学科) c.学科 = c.学科t;
    if (!c.主题) c.主题 = c.主题t;
    if (!c.状态) c.状态 = "🟡";
    if (!c.错因) c.错因 = "(未填)";
}

const counts = (key) => {
    const m = {};
    for (const c of cards) m[c[key]] = (m[c[key]] || 0) + 1;
    return Object.entries(m).sort((a,b) => b[1] - a[1]).map(([k,v]) => `${k} ${v}`).join(" · ");
};

dv.paragraph(`**总卡数**：${cards.length}`);
dv.paragraph(`**按状态**：${counts("状态")}`);
dv.paragraph(`**按错因**：${counts("错因")}`);
dv.paragraph(`**按学科**：${counts("学科")}`);
```

const due = [];

    dv.table(["录入日期", "已过", "状态", "学科", "主题", "Q#"], due);
}

> [!info] 维护说明
> - 视图都是 **DataviewJS** 实时查询，每次打开本文件重新扫描
> - **点击表格里 Q# 或主题** → 直接跳到 `mistakes/uworld-mistakes.md` 对应 callout（依赖每张卡 title 下的 `^Q#` 锚点）
> - 改一张卡的字段（如 `状态:: 🟡` → `状态:: 🔴`）→ 视图自动更新
> - 加一张新卡时：title 行下面必须新增 `> ^Q#` 锚点（Claude Code 入库时会自动加；手动写时记得带）
> - 锚点 Q# 规则：保留字母数字 + `-`，其他符号（空格 / `+` / `,` / `:`）替换为 `-`。例：`Q7708 + Q7709, Efrenzia` → `^Q7708-Q7709-Efrenzia`

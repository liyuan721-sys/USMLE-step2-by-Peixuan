## §一 一句话总结（强制复习清单）⭐

### 1.1 D+1 昨日新学（2026-06-11 · Poisoning & Environmental）

> 扫一遍即被动复习。昨日内容还新，§二 不出题。

**A. 毒物综合征 toxidrome + 解药**
- **TCA 过量**（Q4035/Q4487）：抗胆碱（散瞳干热肠鸣↓）+ **QRS>100 → 碳酸氢钠 sodium bicarbonate**（不是钙 calcium！钙是高钾的）
- **TCA 碳酸氢钠机制**（Q3138 ✅）：升 pH 降药效 + 钠负荷逆转钠通道；**主要治"心脏毒性"**，高热是抗胆碱（管不了）
- **有机磷/神经毒剂**（Q17900）：胆碱能 DUMBELS（缩瞳/大汗/缓脉/流涎）→ 阿托品 atropine + 解磷定 pralidoxime；**RBC 胆碱酯酶**确诊
- **阿片中毒**（Q4510 ✅）：呼吸↓最可靠 + 针尖瞳 → **纳洛酮 naloxone**（四药受体药理 + α2 破直觉见 [[完整笔记/专题笔记/USMLE/USMLE_毒物综合征toxidrome速查]] §5.1；呼吸抑制完整鉴别见 [[完整笔记/专题笔记/USMLE/USMLE_呼吸抑制低通气鉴别]]）
- **阿片戒断**（Q4509 ✗）：流涕腹泻肌痛 + 交感亢进 → **丁丙诺啡 buprenorphine/美沙酮 methadone**（激动剂）；**纳洛酮会加重戒断！**（可乐定 clonidine 压交感 = 激动 α2 刹车受体，见 §5.1）
- **β阻滞剂中毒**（Q19181 ✅）：缓脉+低压+**低血糖** → 胰高血糖素 glucagon（CCB 钙通道阻滞剂则高血糖）
- **对乙酰氨基酚 acetaminophen 过量**（Q3136 ✅）：<4h 活性炭 activated charcoal，4h 测水平上 Rumack-Matthew，超线 NAC（N-乙酰半胱氨酸 N-acetylcysteine）；无症状≠没事
- **水杨酸中毒 salicylate toxicity**（Q18925/Q19677）：高热+过度通气+耳鸣+高 AG 代酸 → 查血清水杨酸；肾衰/肺水肿 → **血透**（不是利尿/甘露醇 mannitol/乙酰唑胺 acetazolamide）
- **铁中毒 iron toxicity**（Q3827）：radiopaque + 高 AG 代酸 + 呕血 → 去铁胺 deferoxamine

**B. 环境急症**
- **低体温**（Q11417 ✅）：28–32 → 主动外部复温（暖毯+温液）；缓脉别上阿托品/起搏
- **冻伤**（Q3876 ✗）：37–39°C 温水复温；别硝苯地平 nifedipine（那是雷诺 Raynaud）
- **劳力性热射病 EHS (exertional heat stroke)**（Q3691）：核心>40 + AMS（意识改变）→ **冰水浸泡**（退热药 antipyretics 无效）
- **减压病 DCS (decompression sickness)**（Q20364 ✅）：潜水后关节痛 → 高压氧 HBOT (hyperbaric oxygen)
- **溺水**（Q20184）：低氧 + 无法保护气道 → 气管插管
- **高压电击**（Q19263）：深部损伤 > 表面 → 横纹肌溶解（查 CK）
- **环形全层烧伤**（Q3398 ✅）：肢体紧绷 + 脉搏↓ → 焦痂切开 escharotomy

**C. 咬蜇伤**
- **棕色隐居蛛**（Q20068/Q3399）：坏死性焦痂 loxoscelism → 支持创面护理
- **蜂蜇大局部反应 LLR**（Q20066）：IgE 介导，非淋巴管炎
- **毒蛇咬伤**（Q19790）：系统/进展征 → 抗蛇毒血清 antivenom
- **婴儿肉毒 infant botulism**（Q13219）：对称下行麻痹 + 便秘 → 肉毒免疫球蛋白 BIG-IV (botulism immune globulin)

**D. 重金属 + 腐蚀剂**
- **慢性砷中毒 chronic arsenic poisoning**（Q12397）：皮肤 + 手套袜套神经病 + 全血细胞↓
- **螯合剂 chelators 雷区**（CHELATE0611）：BAL（二巯基丙醇 dimercaprol）禁忌（铁/镉/硒/甲基汞）；铅脑病 BAL→EDTA（依地酸钙钠 calcium disodium edetate）顺序
- **强碱误服 alkali ingestion**（Q2377 ✗）：液化坏死 → **24h 内镜**；别活性炭 activated charcoal（挡镜）
- **腐蚀剂**（Q3856 ✅）：口咽溃疡 + 流涎 + 清醒 = 局部损伤

**E. 其他**
- **运动相关体位性低血压 EAPH (exercise-associated postural hypotension)**（Q19085）：运动**停止后即刻**晕厥（vs 心源性是运动中）

**F. 🔴 复盘追踪（昨日重做又错的感染题）**
- 原发流感肺炎 vs PCP（耶氏肺孢子菌肺炎 Pneumocystis pneumonia）（Q4418）、孕妇 Lyme（莱姆病 Lyme disease）→阿莫西林 amoxicillin（Q3257）、IVDU（静脉吸毒 IV drug use）右心 IE（感染性心内膜炎 infective endocarditis）→万古霉素 vancomycin（Q3013）、SCD（镰状细胞病 sickle cell disease）无脾脓毒症→肺炎链球菌 Streptococcus pneumoniae（Q3640）—— **见 §一 1.3 必复**

### 1.2 滚动复习

- **D+3（06-09 感染）**：落基山斑疹热 RMSF (Rocky Mountain spotted fever)（蜱+斑丘疹+板↓）→ **多西环素 doxycycline**；隐球菌脑膜炎 cryptococcal meningitis → CSF 隐球菌抗原

- **D+14 / D+30**：无错题（弹性补位）

### 1.3 必复（反复错 🔴 + 难记）

- 🔴 **原发流感肺炎 primary influenza pneumonia ≠ PCP（耶氏肺孢子菌肺炎 Pneumocystis pneumonia）**：骤起全身 + 上感 + 双肺弥漫间质（流感）；PCP 是 HIV + 亚急性 + 劳力性低氧
- 🔴 **孕妇早期 Lyme（莱姆病 Lyme disease）→ 阿莫西林 amoxicillin**（孕期禁多西环素 doxycycline；也别头孢曲松 ceftriaxone）
- 🔴 **IVDU（静脉吸毒 IV drug use）右心 IE（感染性心内膜炎 infective endocarditis）→ 经验万古霉素 vancomycin**（覆盖 MRSA 耐甲氧西林金葡菌；别头孢曲松 ceftriaxone）
- 🔴 **SCD（镰状细胞病 sickle cell disease）/无脾脓毒症 → 肺炎链球菌 Streptococcus pneumoniae**（荚膜菌）
- #难记轮换：**颈淋巴结炎抗生素失败 → Bartonella 巴尔通体（猫抓病 cat-scratch disease）**；糖尿病足骨髓炎 diabetic foot osteomyelitis → MRI+骨活检；巴贝虫病 babesiosis（蜱+溶血+疟原虫样环）

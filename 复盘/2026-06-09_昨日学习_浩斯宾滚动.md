## §一 一句话总结（被动复习 — 扫一遍）

### 1.1 昨日新学 D+1（2026-06-08，按主题归类）⭐

> [!danger] 🔴 反复错（最优先）
> - **颈淋巴结炎（Q4843/Q4353）**：急性单侧红痛波动 = **S.aureus/GAS（治 clindamycin）**；猫抓病是慢性无痛 + 猫抓丘疹。**两次都误走猫抓病 → 先问"急/慢、痛/不痛"**。
> - **脑膜炎球菌脑膜炎（Q18591×2）**：发热+瘀点+颈抵抗 → **唯一答案 = 立刻 vanc+ceftriaxone**（勿等 LP、勿降温）。

> [!info] 脑膜炎 / CNS 感染
> - **Q12240** 社区获得性细菌性脑膜炎首位 = **肺炎链球菌（~70%，±肺炎）**；Klebsiella 脑膜炎罕见。
> - **Q4265** 青少年露营+瘀点+中性 CSF = **脑膜炎球菌**（vs 无瘀点+肺炎中老年→肺炎链球菌）。
> - **Q18596** 脑膜炎球菌血症紫癜 = **DIC**（查纤维蛋白原↓）。
> - **Q19266** 急性 HIV → 无菌性脑膜炎（糖正常淋巴，别误 TB）。
> - **Q15725** 鼻窦炎+晨起呕吐/嗜睡 = **颅内并发症脑脓肿 → 头颅 CT**（↑颅压先影像后 LP）。
> - **Q3671** 12 日龄败血/脑膜 = **GBS**（晚发 >1 周水平传播，IAP 不防）。

> [!info] 肺炎 / 呼吸道
> - **Q19837** 支原体行走肺炎仍需经验抗生素（**azithromycin**；β-内酰胺无效）。
> - **Q3054** 军团菌（邮轮+腹泻+意识混乱+低钠）→ **喹诺酮/大环内酯**（克林无效）。
> - **Q4388** IVDU+空洞肺结节 = 三尖瓣 IE **S.aureus** 脓毒肺栓（非 PCP；CD4>200 排 PCP）。
> - **Q11863** 咽炎后+胸锁乳突肌颈肿+空洞肺结节 = **Lemierre（Fusobacterium）**。
> - **Q19349** 婴儿多器官弥漫粟粒+慢咳接触 = **粟粒性结核**（IGRA/TST 常假阴）。
> - **Q3830** CF 肺炎：**幼儿金葡 / 成人铜绿**（非肺炎链球菌）。
> - **Q4417** 免疫抑制+肺结节+脑脓肿+部分抗酸分枝 = **Nocardia → TMP-SMX**（Actinomyces 才用青霉素）。
> - **Q3905** 百日咳成人发病 = **疫苗免疫衰减**（5-10 年）。

> [!info] SCD / 血液
> - **Q3640** SCD 无脾**脓毒症**（无骨痛）→ **肺炎链球菌**。
> - **Q3571** SCD **骨髓炎**（有骨痛）→ **金葡 + Salmonella 双覆盖**。
> - **Q2277** 晚期 AIDS 不强化无水肿白质灶 = **PML（JC 病毒）**。

> [!info] 皮肤软组织（分层）
> - **Q3263** 隆起锐界+累及耳廓 = **丹毒（GAS，Milian 征）**；蜂窝织炎是深层平坦模糊。
> - **Q3530** 舌后上推+口底肿+颌下捻发 = **Ludwig angina（牙源性）**。
> - **Q12168** 海水伤口+肝病/血色病+出血性大疱 = **创伤弧菌 → ceftazidime+doxycycline**。
> - **Q106792** 野生动物+溃疡+化脓淋巴结 = **兔热病**（vs 孢子丝菌园艺串珠）。
> - **Q3642** 局限非大疱脓疱疮 → 外用 **mupirocin**。

> [!info] 心内膜炎 / 脓毒症 / 其他
> - **Q3012** 疑 IE 第一步 = **血培养**（先于影像/抗生素）。
> - **Q4668** IE 确诊 = 超声心动 + 血培（Osler→别查 ASO）。
> - **Q19283** 糖尿病足脓毒症 → **1h 内血培养+广谱抗生素+30mL/kg 补液**（血糖 220 不急）。
> - **Q7522** 糖尿病足 probe-to-bone⊕ 骨髓炎 → **骨活检+培养**（非截肢）。
> - **Q17223** 胆道支架后发热 RUQ+ALP↑ = **化脓性肝脓肿 → 腹部 CT**。
> - **Q16479** SSI 最强可控因素 = **吸烟**。
> - **Q17637** 轻症 Campylobacter = **仅支持治疗**（药敏是干扰）。
> - **Q18413/Q18164/Q18411** AOM：严重过敏→azithromycin / 近期用过 amox→amox-clav / 可控危险因素=被动吸烟。
> - **Q3246** 新诊断 HIV+LTBI → 治潜伏结核。
> - **Q2970** 肝包虫蛋壳钙化 / **Q9849** 疱疹性咽峡炎 / **Q3969** 猩红热砂纸疹 / **Q106829** C.diff 接触隔离 / **Q3247** 埃立克体 / **Q3104** Babesia（无脾）。

### 1.2 滚动复习

- **D+3（06-06）**：百日咳 PEP（全家大环内酯）/ PJI 时相病原 / 猫咬→amox-clav / 梅毒血清学（非螺旋体 vs 螺旋体）/ 儿童脓胸+MRSA / 念珠菌血症（拔管+抗真菌）/ 播散 MAC（CD4<50→大环内酯+乙胺丁醇）/ CMV 结肠炎（CD4<50 血性腹泻）。
- **D+14（05-26）**：QI 工具配对（fishbone/RCA/FMEA/Lean/Six Sigma/PDSA）/ Closed-loop 防口头医嘱误传 / 跌倒预防层级（人看着>机器）/ Home Health Care 适应症 / 筛查工具低 PPV。
- **D+30（05-10）**：Hill's 准则 Dose-Response / OR 方向陷阱 + 关联≠因果。

### 1.3 必复（一句话）

- 🔴 **颈淋巴结炎**：急性单侧红痛=S.aureus/GAS（clindamycin），别选猫抓病。
- 🔴 **脑膜炎球菌**：发热+瘀点+颈抵抗→立刻 vanc+ceftriaxone，勿等 LP/勿降温。
- #考前必看 **Home Health Care**：medically stable + homebound + skilled needs 三联（不是 admit/SNF）。
- #难记 **Nocardia vs Actinomyces**：Nocardia 需氧+部分抗酸+免疫抑制+脑脓肿→TMP-SMX；Actinomyces 厌氧+非抗酸+硫磺颗粒→青霉素。

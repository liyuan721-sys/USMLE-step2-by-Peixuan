## §三 4 错题深挖（精简版 — 详见配对错题卡）

### ❌ Q3 Tourette — Anchor by Letter（新错点 pattern）⚠️

**根本原因**：5-25 Q10 D = antidopaminergic 答对 → 5-27 反射选 D 但 D 已变 methylphenidate（毒药）。**字母 anchor 不可信** — USMLE 故意打乱字母位置测真懂。

**正解钉死**：α2 整类失败 → **加 risperidone / aripiprazole**（antidopaminergic）；**Methylphenidate 加重 tics（永远禁用）**；Benztropine 是 EPS 急性 dystonia 药（不治 tic）。

**入库**：[[mistakes/uworld-mistakes#^Q3-2026-05-27]] 🟡 + #难记
**衍生**：驱动建立 [[完整笔记/专题笔记/USMLE/USMLE_答案选项策略]] §二 Letter Anchor 反 pattern

---

### ❌ Q7 Swiss Cheese vs PDSA 混（升级 Q21477 🔴）⚠️

**根本原因**：没识别 "**simultaneously**" 是 SCM 暗号词 — D PDSA 是 sequential / iterative，stem 完全无此时序。

**4 大 System Design Models 钉死**：
- **SCM** ⭐ — "**simultaneously / concurrently**" + 列举 ≥ 2 不同性质 barrier
- **Hard stop** — "**physically incompatible / hard limit**"
- **Automation** — "**EHR alert / smart pump / barcode**"
- **PDSA** — "**repeated cycles / pilot then refine**"

**入库**：[[mistakes/uworld-mistakes#^Q7-2026-05-27]] 升级 [[mistakes/uworld-mistakes#^Q21477]] 🟡 → **🔴 反复错 + #难记**
**衍生**：[[完整笔记/专题笔记/USMLE/USMLE_患者安全_QI工具]] §2.7.4 加 4 大 System Design Models 钉死表

---

### ❌ Q8 RCA-FMEA 字眼陷阱第 3 次错（升级 Q22581 #考前必看 + #难记）⚠️⚠️

**根本原因**：选项 A "future + potentially cause + vulnerabilities" **三连 FMEA 句式**反射误锁 — 没识别 stem 是 retrospective（"errors HAVE occurred"）。

**钉死**：
- **RCA = 已发生过去事件 underlying causes**（过去 why）
- **FMEA = 未发生未来事件 potential vulnerabilities**（未来 what could）
- 选项 "future + potentially + vulnerabilities" 三连 → **FMEA 语言**；stem retrospective 时 = **钓饵**

**入库**：[[mistakes/uworld-mistakes#^Q8-2026-05-27]] 升级 [[mistakes/uworld-mistakes#^Q22581]] **维持 🔴 + 加 #考前必看 + #难记** ⚠️⚠️
**衍生**：[[完整笔记/专题笔记/USMLE/USMLE_患者安全_QI工具]] §13.12.8 加 RCA-FMEA 字眼陷阱反射钩子节 + stem 时间方向校验 SOP

### ❌ Q9 Home Health vs SNF / admit 反复错（升级 Q20431 🔴 + #考前必看）⚠️

**根本原因**：反射"elderly + fall + 担忧 → institution 升级" — 5-25 选 admit / 5-27 选 SNF — 都不选 home health。

**钉死决策树**：
```
Medically stable?
  ├─ ❌ → Hospitalize
  └─ ✅ Stable
       │
       ├─ Severe CI + 不能 ADL + 持续 monitoring → SNF
       │
       └─ Mild CI + 能 ADL
            └─ Homebound + 需 skilled care → **Home Health Care** ⭐
```

**入库**：[[mistakes/uworld-mistakes#^Q9-2026-05-27]] 升级 [[mistakes/uworld-mistakes#^Q20431]] 🟡 → **🔴 + #考前必看**

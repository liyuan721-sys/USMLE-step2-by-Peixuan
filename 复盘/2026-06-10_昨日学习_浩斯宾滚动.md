## §一 一句话总结（被动复习清单）

### 1.1 D+1 昨日新学（2026-06-09）— 逐条反射

**感染（皮肤/软组织 + 全身）**
- **Q2768 水痘**：不同阶段皮损（丘疹/水疱/结痂）并存 + 强痒 = 水痘；**疫苗后仍可突破感染**（更轻、皮损更少）。
- **Q10068 HSV 生殖器疱疹**：成簇浅溃疡 + **触痛**淋巴结 + 全身症状 = 原发 HSV；查 PCR/Tzanck。
- **Q3425 麻风（leprosy）**：**无痛低色素麻木斑 + 周围神经粗大** → 皮肤活检（取边缘）。
- **Q2999 芽生菌病（blastomycosis）**：疣状紫红皮损 + 痰见**宽基出芽酵母** + 中西部。
- **Q19310 传染性软疣**：中央**脐凹**肉色丘疹 → 观察安慰（**非虐待上报**）。
- **Q18459 Fight bite（拳击咬伤）**：拳头打牙 → MCP 化脓性关节炎 → **手术清创 + 氨苄西林-舒巴坦（ampicillin-sulbactam）**。

**感染（呼吸/头颈）**
- **Q4850 ABRS 最常见危险因素 = 病毒性 URI**（不是吸烟/腺样体）。
- **Q3286 儿童 ABRS 首位病原 = 不可分型流感嗜血杆菌（H. influenzae）**（PCV13 后肺炎链球菌↓）。
- **Q4418 原发流感肺炎**：骤起流感 + **双侧弥漫间质浸润**（非 PCP）。
- **Q4354 传染性单核细胞增多症（IM）**：迁延咽炎 + **全身**淋巴结 + 显著乏力 → 嗜异性抗体。
- **Q19418 老年 TB 不典型**：数月消瘦/淋巴结 + 空洞 + **CKD → TST 假阴（anergy）**。
- **Q115197 Croup 喉气管炎 = 副流感病毒（parainfluenza）**。

**感染（CNS/IE/脓肿）**
- **Q9885 隐球菌脑膜炎**：HIV + 鹅口疮 + 亚急性头痛↑颅压 → **CSF 隐球菌抗原（CrAg）**。
- **Q4163 IE 最常受累瓣膜 = 二尖瓣（MVP）**；**Q3013 右心 IE**：IVDU + 吸气增强胸骨左缘杂音（三尖瓣）→ 经验万古；**Q4714** 脾脓肿三联 + MVP = IE 脓毒栓子。
- **Q3103 CMV 食管炎**：AIDS + **大而线状**食管溃疡 + 核内外包涵体 → 更昔洛韦（HSV = 小而多发）。
- **Q3938 PCP**：**PaO₂ ≤70 / A-a ≥35** → TMP-SMX **加糖皮质激素**。

**感染（GI/肝/虫媒/人兽）**
- **Q2968 阿米巴肝脓肿**：旅行 + 腹泻史 + 肝右叶单发 → 甲硝唑 + 巴龙霉素（清肠腔）。
- **Q3789 包虫病（Echinococcus）**：肝囊肿含**子囊** + 犬/牧区（犬接触，非猪带绦虫）。
- **Q3612 蛲虫**：**夜间肛周痒** → 噻嘧啶/阿苯达唑（湖游泳是血吸虫干扰）。
- **Q12458 志贺菌肠炎**：高热 + **血黏液便** + 儿童惊厥（重做又错选 C. perfringens）。
- **Q19832 落基山斑疹热（RMSF）**：蜱区 + 四肢→躯干斑丘疹 + 板↓/转氨酶↑/低钠 → **多西环素（经验性，勿等血清学）**。
- **Q20206 猫抓病**：淋巴结炎抗生素失败 → Bartonella 血清学。
- **Q3464 CRBSI**：导管 >48h + **CoNS 4/4 瓶阳** = 导管相关血流感染。

**感染（性传播/孕期）**
- **Q3257 孕妇早期 Lyme**（游走性红斑）→ **阿莫西林**（孕妇/儿童禁多西环素）。
- **Q3256 一期梅毒 + 重度青霉素过敏（TEN）→ 多西环素**（非阿莫西林）。
- **Q3583 急性 HIV（全身/GI 型）**：亚急性 GI + 盗汗 + **一过性粉红卵圆疹** + 无痛淋巴结 → 查 HIV RNA。

**先天/新生儿/儿科**
- **Q3676 先天弓形虫病**：脑积水 + **弥漫**颅内钙化（CMV = 脑室周钙化 + 小头）。
- **Q17310 新生儿败血症**：3 日龄激惹 + 喂养差 + **白细胞减少** → 氨苄西林 + 庆大霉素（血糖 70 正常排除低糖）。
- **Q3788 新生儿破伤风（张力↑）vs 肉毒（弛缓）**。
- **Q105673 病毒性胃肠炎**：急性非血性水样泻 + 轻度脱水 → 口服补液居家。

**眼科/外科**
- **Q17172 急性细菌性结膜炎**：脓性分泌物**秒擦秒满**（成人金葡）。
- **Q3326 隔前蜂窝织炎**：眼睑红肿但**眼球运动/视力正常**（vs 眶蜂窝织炎）。
- **Q16043 术后即刻发热**（数小时）= 组织创伤 → 仅对症观察。
- **Q19256 侵袭性烧伤创面感染**：植皮坏死 + 周围红斑 + 全身表现 → **抗假单胞 β-内酰胺 + 万古**（真菌仅活检见才加两性霉素 B）。

**Biostats**
- **Q20886 效应修饰 vs 混杂**：分层后各层效应**不同** = 效应修饰（分层报告，不消除）；**OR<1 且 CI 不含 1 = 显著降低**。

**🔴 反复错追踪（今日新增）**
- **Q19349 婴儿粟粒结核**（×2，两次都误选 Listeria）；**Q4843 颈淋巴结炎**（第 3 次又选 azithromycin，应 clindamycin）；**Q7522 糖尿病足骨髓炎**（×2，应骨活检）；**Q3104 巴贝虫**（×2，没认出血涂片 Maltese cross）；**Q20430 ARP**（×2，又选 NNH 13.0）。

### 1.2 滚动复习

- **D+3（2026-06-07）/ D+7（2026-06-03）**：无错题节 → 名额转 §二 弹性补位。
- **D+14（2026-05-27）患者安全/Ethics**：RCA = 查**已发生事件的根因**（FMEA 才是前瞻找薄弱）；Swiss Cheese = 多 barrier 同时叠加；At-risk behavior → **Coaching**（不是惩罚）。
- **D+30（2026-05-11）Biostats**：ARP=(Re−Ru)/Re（%）；ARR 反推 n 必须先算该组 Rt；LR+ = sens/(1−spec)；PPV 随患病率↑而↑；Matching 治 confounding。

### 1.3 必复（🔴 + 标签）

- **🔴 ARP 阳性归因比（Q20430）**：带 % 且最大 = ARP；13 是 NNH（人数），3.2 是 RR（倍）。
- **🔴 颈淋巴结炎（Q4843/Q4353）**：急性单侧 = **S. aureus/GAS → clindamycin**；抗生素**失败后**才想猫抓病（Bartonella）。
- **🔴 脑膜炎球菌脑膜炎（Q18591）**：疑细菌性脑膜炎**先经验抗生素**，勿等腰穿/CT。
- **🔴 链球菌咽炎（Q2193）**：儿童 RADT 阴 → **咽培养**（成人不用）。
- **#考前必看（患者安全）**：RCA vs FMEA、Just Culture 三分法。
- **#难记 轮换**：巴贝虫 Maltese cross + atovaquone/azithromycin。

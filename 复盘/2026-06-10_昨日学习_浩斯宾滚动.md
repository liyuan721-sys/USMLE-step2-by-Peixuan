## §一 一句话总结（被动复习清单）

### 1.1 D+1 昨日新学（2026-06-09）— 逐条反射

**感染（皮肤/软组织 + 全身）**
- **Q2768 水痘（varicella/chickenpox）**：不同阶段皮损（丘疹/水疱/结痂）并存 + 强痒 = 水痘；**疫苗后仍可突破感染（breakthrough）**（更轻、皮损更少）。
- **Q10068 生殖器疱疹（genital herpes, HSV）**：成簇浅溃疡 + **触痛**淋巴结 + 全身症状 = 原发 HSV；查 PCR/Tzanck。
- **Q3425 麻风（leprosy, Mycobacterium leprae）**：**无痛低色素麻木斑 + 周围神经粗大** → 皮肤活检（取边缘）。
- **Q2999 芽生菌病（blastomycosis）**：疣状紫红皮损 + 痰见**宽基出芽酵母（broad-based budding yeast）** + 中西部。
- **Q19310 传染性软疣（molluscum contagiosum）**：中央**脐凹**肉色丘疹 → 观察安慰（**非虐待上报**）。
- **Q18459 Fight bite（拳击咬伤）**：拳头打牙 → MCP 化脓性关节炎（septic arthritis）→ **手术清创 + 氨苄西林-舒巴坦（ampicillin-sulbactam）**。

**感染（呼吸/头颈）**
- **Q4850 急性细菌性鼻窦炎（ABRS）最常见危险因素 = 病毒性上感（viral URI）**（不是吸烟/腺样体）。
- **Q3286 儿童 ABRS 首位病原 = 不可分型流感嗜血杆菌（nontypeable H. influenzae）**（PCV13 后肺炎链球菌 S. pneumoniae↓）。
- **Q4418 原发流感肺炎（primary influenza pneumonia）**：骤起流感 + **双侧弥漫间质浸润**（非 PCP 肺孢子菌肺炎）。
- **Q4354 传染性单核细胞增多症（infectious mononucleosis, IM）**：迁延咽炎 + **全身**淋巴结 + 显著乏力 → 嗜异性抗体（heterophile/Monospot）。
- **Q19418 老年肺结核（TB）不典型**：数月消瘦/淋巴结 + 空洞 + **慢性肾病（CKD）→ 结核菌素试验（TST）假阴（anergy）**。
- **Q115197 喉气管炎（croup）= 副流感病毒（parainfluenza virus）**。

**感染（CNS/IE/脓肿）**
- **Q9885 隐球菌脑膜炎（cryptococcal meningitis）**：HIV + 鹅口疮（oral thrush）+ 亚急性头痛↑颅压 → **CSF 隐球菌抗原（CrAg）**。
- **Q4163 感染性心内膜炎（IE）最常受累瓣膜 = 二尖瓣（mitral，MVP 相关）**；**Q3013 右心 IE**：静脉吸毒（IVDU）+ 吸气增强胸骨左缘杂音（三尖瓣 tricuspid）→ 经验万古霉素（vancomycin）；**Q4714** 脾脓肿三联 + MVP = IE 脓毒栓子。
- **Q3103 CMV 食管炎（巨细胞病毒 cytomegalovirus esophagitis）**：AIDS + **大而线状**食管溃疡 + 核内外包涵体 → 更昔洛韦（ganciclovir）（HSV 单纯疱疹 = 小而多发）。
- **Q3938 肺孢子菌肺炎（PCP）**：**PaO₂ ≤70 / A-a ≥35** → 复方磺胺甲噁唑（TMP-SMX）**加糖皮质激素（corticosteroid）**。

**感染（GI/肝/虫媒/人兽）**
- **Q2968 阿米巴肝脓肿（amebic liver abscess, Entamoeba histolytica）**：旅行 + 腹泻史 + 肝右叶单发 → 甲硝唑（metronidazole）+ 巴龙霉素（paromomycin）（清肠腔）。
- **Q3789 包虫病（echinococcosis, Echinococcus）**：肝囊肿含**子囊** + 犬/牧区（犬接触，非猪带绦虫 Taenia solium）。
- **Q3612 蛲虫病（pinworm, Enterobius vermicularis）**：**夜间肛周痒** → 噻嘧啶（pyrantel）/阿苯达唑（albendazole）（湖游泳是血吸虫 schistosomiasis 干扰）。
- **Q12458 志贺菌肠炎（shigellosis, Shigella）**：高热 + **血黏液便** + 儿童惊厥（重做又错选产气荚膜梭菌 C. perfringens）。
- **Q19832 落基山斑疹热（RMSF, Rickettsia rickettsii）**：蜱区 + 四肢→躯干斑丘疹 + 板↓/转氨酶↑/低钠 → **多西环素（doxycycline，经验性勿等血清学）**。
- **Q20206 猫抓病（cat-scratch disease, Bartonella henselae）**：淋巴结炎抗生素失败 → Bartonella 血清学。
- **Q3464 导管相关血流感染（CRBSI）**：导管 >48h + **凝固酶阴性葡萄球菌（CoNS）4/4 瓶阳** = CRBSI。

**感染（性传播/孕期）**
- **Q3257 孕妇早期莱姆病（Lyme disease）**（游走性红斑 erythema migrans）→ **阿莫西林（amoxicillin）**（孕妇/儿童禁多西环素 doxycycline）。
- **Q3256 一期梅毒（primary syphilis）+ 重度青霉素（penicillin）过敏（TEN 中毒性表皮坏死松解）→ 多西环素（doxycycline）**（非阿莫西林）。
- **Q3583 急性 HIV 感染（acute HIV，全身/GI 型）**：亚急性 GI + 盗汗 + **一过性粉红卵圆疹** + 无痛淋巴结 → 查 HIV RNA。

**先天/新生儿/儿科**
- **Q3676 先天性弓形虫病（congenital toxoplasmosis）**：脑积水 + **弥漫**颅内钙化（CMV 巨细胞病毒 = 脑室周钙化 + 小头）。
- **Q17310 新生儿败血症（neonatal sepsis）**：3 日龄激惹 + 喂养差 + **白细胞减少** → 氨苄西林（ampicillin）+ 庆大霉素（gentamicin）（血糖 70 正常排除低糖）。
- **Q3788 新生儿破伤风（neonatal tetanus，张力↑）vs 婴儿肉毒中毒（infant botulism，弛缓）**。
- **Q105673 病毒性胃肠炎（viral gastroenteritis）**：急性非血性水样泻 + 轻度脱水 → 口服补液（ORS）居家。

**眼科/外科**
- **Q17172 急性细菌性结膜炎（acute bacterial conjunctivitis）**：脓性分泌物**秒擦秒满**（成人金葡 S. aureus）。
- **Q3326 隔前蜂窝织炎（preseptal cellulitis）**：眼睑红肿但**眼球运动/视力正常**（vs 眶蜂窝织炎 orbital cellulitis）。
- **Q16043 术后即刻发热（immediate postop fever，数小时）= 组织创伤 → 仅对症观察**。
- **Q19256 侵袭性烧伤创面感染（burn wound infection, BWI）**：植皮坏死 + 周围红斑 + 全身表现 → **抗假单胞 β-内酰胺（anti-pseudomonal）+ 万古霉素（vancomycin）**（真菌仅活检见才加两性霉素 B amphotericin B）。

**Biostats**
- **Q20886 效应修饰（effect modification）vs 混杂（confounding）**：分层后各层效应**不同** = 效应修饰（分层报告，不消除）；**OR<1 且 CI 不含 1 = 显著降低**。

**🔴 反复错追踪（今日新增）**
- **Q19349 婴儿粟粒性结核（miliary TB）**（×2，两次都误选李斯特菌 Listeria）；**Q4843 颈淋巴结炎（cervical lymphadenitis）**（第 3 次又选阿奇霉素 azithromycin，应克林霉素 clindamycin）；**Q7522 糖尿病足骨髓炎（diabetic foot osteomyelitis）**（×2，应骨活检）；**Q3104 巴贝虫病（babesiosis, Babesia microti）**（×2，没认出血涂片 Maltese cross）；**Q20430 阳性归因比（ARP）**（×2，又选 NNH 13.0）。

### 1.2 滚动复习

- **D+3（2026-06-07）/ D+7（2026-06-03）**：无错题节 → 名额转 §二 弹性补位。
- **D+14（2026-05-27）患者安全/Ethics**：根因分析（RCA）= 查**已发生事件的根因**（失效模式与影响分析 FMEA 才是前瞻找薄弱）；瑞士奶酪模型（Swiss Cheese）= 多 barrier 同时叠加；At-risk behavior → **Coaching**（不是惩罚）。
- **D+30（2026-05-11）Biostats**：ARP=(Re−Ru)/Re（%）；绝对风险降低（ARR）反推 n 必须先算该组 Rt；阳性似然比 LR+ = sens/(1−spec)；阳性预测值（PPV）随患病率↑而↑；匹配（matching）治混杂（confounding）。

### 1.3 必复（🔴 + 标签）

- **🔴 阳性归因比（ARP, attributable risk percentage）（Q20430）**：带 % 且最大 = ARP；13 是需害人数（NNH，人数），3.2 是相对危险度（RR，倍）。
- **🔴 颈淋巴结炎（cervical lymphadenitis）（Q4843/Q4353）**：急性单侧 = **金黄色葡萄球菌（S. aureus）/A 族链球菌（GAS）→ 克林霉素（clindamycin）**；抗生素**失败后**才想猫抓病（cat-scratch, Bartonella）。
- **🔴 脑膜炎球菌脑膜炎（meningococcal meningitis）（Q18591）**：疑细菌性脑膜炎**先经验抗生素**，勿等腰穿/CT。
- **🔴 链球菌咽炎（strep pharyngitis, GAS）（Q2193）**：儿童快速链球菌抗原（RADT）阴 → **咽拭子培养**（成人不用）。
- **#考前必看（患者安全）**：根因分析（RCA）vs 失效模式分析（FMEA）、公正文化（Just Culture）三分法。
- **#难记 轮换**：巴贝虫病（babesiosis）血涂片 Maltese cross + 阿托伐醌（atovaquone）/阿奇霉素（azithromycin）。

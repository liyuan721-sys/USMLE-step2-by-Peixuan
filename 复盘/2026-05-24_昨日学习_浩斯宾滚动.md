## §一 一句话总结（强制被动复习清单）⭐

### 1.1 D+1 昨日（2026-05-23）新学内容 — 一句话扫一遍

**Psych 主战场（5 张 + 浩斯宾 14 张 = 19 张 psych）**

- **[[mistakes/uworld-mistakes#^Q14348]] 成人 ADHD + 物质滥用史 / 拒成瘾药** → **Atomoxetine（NRI 非成瘾）**；兴奋剂遇 SUD 史 = 红灯，遇 Tourette 共病也避（α2/atomoxetine 优先）。
- **[[mistakes/uworld-mistakes#^Q16153]] 儿童 GAD vs Social Anxiety** → GAD 是**多场所多议题担忧**（学业/家庭/健康），SA 是**特定社交场合**；儿童 GAD 一线 = **CBT → SSRI（sertraline）**。
- **[[mistakes/uworld-mistakes#^Q16490]] BED 药物** → **Topiramate / Lisdexamfetamine（FDA 批 BED 唯一兴奋剂）/ SSRI**；不是 CBT 单用。
- **[[mistakes/uworld-mistakes#^Q8923]] 儿童 OCD** → **CBT (ERP) 一线**；中重度加 **SSRI（sertraline / fluvoxamine / fluoxetine）**；clomipramine 二线。
- **[[mistakes/uworld-mistakes#^Q15088]] 🔴 MDD with Psychotic Features 治疗** → **SSRI + 二代抗精神病药**（双药），单 SSRI / 单 lithium / BZD / MAOI 都不够；重症拒食/即将自杀 → **ECT**（[[mistakes/uworld-mistakes#^Q15811]]）。
- **[[mistakes/uworld-mistakes#^Q15071]] BPD 一线 = DBT**（辩证行为疗法）；mood stabilizer / SGA 只是**辅助**靶向情感不稳/冲动，不替代 DBT。
- **[[mistakes/uworld-mistakes#^Q20930]] 儿童 GAD 一线 = SSRI（sertraline）+ Tetrabenazine 钓饵识别** → VMAT2 是 **Tourette/HD/TD** 的药，与焦虑无关，看到划掉。
- **[[mistakes/uworld-mistakes#^Q12152]] 🔴 PD psychosis 第 5 步** → 减药失败后必须上抗精神病药 = **Quetiapine（低 D2）**；禁用 haloperidol/risperidone/olanzapine（强 D2 加重 PD）。
- **[[mistakes/uworld-mistakes#^Q4879]] MAOI tyramine HTN crisis** → 食物（陈年奶酪/红酒/腌肉）+ MAOI = NE 爆发 → 血压飙升 + 剧烈头痛 → **phentolamine（α）/ nitroprusside / labetalol（α+β）**；不是 SS（SS 走 5-HT，要 cyproheptadine）。
- **[[mistakes/uworld-mistakes#^Q3375]] Tourette 药物升级** → HRT 行为 → α2（guanfacine/clonidine）失败 → **抗多巴胺药**（risperidone / tetrabenazine / haloperidol）；**不是抗胆碱**（抗胆碱只治 D2 阻断后的急性 EPS）。
- **[[mistakes/uworld-mistakes#^Q19552]] Tourette 诊断陷阱** → **vocal tic ≠ OCD verbal compulsion**；tic 是不自主刻板动作/发声，OCD 是为缓解焦虑的有意义仪式。

**Ethics 主战场（10 张连刷）**

- **[[mistakes/uworld-mistakes#^Q3625]] 公共健康 override autonomy** — 脑膜炎球菌 = 上报 + 接触者预防（rifampin/cipro/ceftriaxone）即使患者反对（强制上报传染病）。
- **[[mistakes/uworld-mistakes#^Q18702]] AMA discharge IVDU IE** → **empathic investigation**（探询原因/给戒断治疗/不威胁），不是直接放走 / 不是强留。
- **[[mistakes/uworld-mistakes#^Q105595]] 青少年 conditional confidentiality** → 性/避孕/STI/药物/精神/堕胎 **保密**；自/他害 + 虐待 = **必报**。
- **[[mistakes/uworld-mistakes#^Q3614]] HIPAA verbal consent + minimum necessary** → 给亲属 / 配偶 / 朋友的口头同意 OK，但只透露与治疗相关的**最少必要信息**。
- **[[mistakes/uworld-mistakes#^Q19048]] Decision-making capacity reassessment** → surrogate ≠ permanent，**意识恢复后必须重新评估 capacity**；若恢复 → 患者本人决定权回归。
- **[[mistakes/uworld-mistakes#^Q7283]] Pediatric chronic illness disclosure** → 慢病儿童 **assess understanding first**，不是直接告知 / 不是父母代答。
- **[[mistakes/uworld-mistakes#^Q3624]] Joint custody** → **one parent consent 即可**；只在 both refuse 危及生命治疗时 court order。
- **[[mistakes/uworld-mistakes#^Q19132]] Perinatal HIV disclosure** → **collaborative plan + timeline**（与患者一起拟告知配偶时间表），不是单方面替她告知。
- **[[mistakes/uworld-mistakes#^Q21627]] NEOT (No Escalation of Treatment)** = 不升级（不上 ICU/不插管/不 CPR）+ **继续当前治疗**；是 DNR 与 comfort care 之间的中间地带。
- **[[mistakes/uworld-mistakes#^Q21658]] Family-centered advance care planning** → 青少年慢病 → **包括家长 + 患儿 + 多学科团队**逐步建立 goals of care，不是单独父母决定。

**内分泌 / 儿科**

- **[[mistakes/uworld-mistakes#^Q20217]] 低血糖 insulin-C-peptide 鉴别** → ↑ Insulin + ↑ C-peptide = **内源性**（insulinoma / sulfonylurea / 自身免疫低血糖）；↑ Insulin + ↓ C-peptide = **外源胰岛素注射**（factitious / Munchausen）。
- **[[mistakes/uworld-mistakes#^Q21629]] Down syndrome 诊断** → 多线索整合（面容/低张/心杂音/AVSD）→ **Karyotype 确诊**，不是 FISH/超声。

**昨日 50 道浩斯宾自测 23/25（A 级）扣分点**（生成型 / 同主题反复）：

- 错点 1：**Atypical MDD 治疗阶梯到 MAOI 第 4 档没掌握** — 你选 lithium augmentation，正解 **phenelzine (MAOI)**；atypical features + "愿尝试经典药" = MAOI 钩。**默写阶梯**：SSRI/SNRI → bupropion/mirtazapine → augment（Li/T3/aripiprazole）→ **MAOI** → ECT/TMS/esketamine。

**D+7 = 2026-05-17（心内 5 张）**
- **[[mistakes/uworld-mistakes#^Q2723]] Vasospastic Angina** → CCB（amlodipine / diltiazem）+ 戒烟；**禁 β-blocker**（无对抗 α → 加重痉挛）。
- **[[mistakes/uworld-mistakes#^Q12118]] PPCM** = 妊娠末期 / 产后 5 月内新发 HFrEF，无其他病因；治疗按 HF 但**孕期禁 ACEI/ARB**（用 hydralazine + 硝酸酯 + β-blocker + 利尿）。
- **[[mistakes/uworld-mistakes#^Q2663]] β-blocker tox** → 解药 **Glucagon**（绕过 β 受体 ↑cAMP）；CCB tox → **calcium + 高剂量胰岛素**。

**D+14 = 2026-05-10（Biostats 2 张 — 与昨日错点 2 同源 🔴）**
- **[[mistakes/uworld-mistakes#^Q20227]] 🔴 OR 方向陷阱 + Case-control 不能算 risk** → "X compared to Y" → **X 是暴露，Y 是 reference**；case-control 无分母 → 算 OR 不算 risk。
- **[[mistakes/uworld-mistakes#^Q20787]] Hill's criteria — Dose-Response** → 因果推断 9 准则之一，剂量↑ → 效应↑ 增强因果可信度（非充分非必要）。

# 2026-05-18 昨日学习复盘 — 一句话重点 + 20 题自测

## Part 1 — 一句话重点（8 个主题，先扫一遍再做题）

> [!success] 1. Vasospastic Angina (Q2723)
> 年轻人 / 吸烟 / 夜间或静息胸痛 + 一过性 ST 抬高 + 冠脉正常 → **预防一线 CCB（diltiazem/amlodipine）/ 急性发作舌下 NTG / 戒烟**；**β-blocker 禁忌**（会让 α 无对抗 → 痉挛加重）。

> [!success] 2. Tachycardia-Induced Cardiomyopathy (Q4238)
> 既往心功能正常 + 持续 tachycardia（多为 AF with RVR）+ 新发 HFrEF + 二度 MR → **TIC**；治疗 = **控率 + 抗凝**（不是堆 HFrEF 标准药），心功能 6-12 月可逆。"看到 EF↓先问 3 W"：When / Why / Will reverse。

> [!success] 3. PVD 机械瓣新发杂音 (Q15831)
> 人工瓣 + new murmur（任何方向）→ **Prosthetic Valve Dysfunction until proven otherwise → 直接 echo**，不管症状、不管 INR、不管时间。机械瓣"正常听诊" = crisp metallic **click**，不能 murmur。PVD 4 机制：thrombosis (systolic, 抗凝相关) / paravalvular leak (regurg) / transvalvular regurg (生物瓣) / pannus (stenotic)。

> [!success] 4. PPCM 妊娠晚期新发心衰 (Q12118)
> 妊娠最后 1 月 ~ 产后 5 月 + 进展性呼吸困难 + S3 + 心尖部 holosystolic murmur → **PPCM → echo 第一步**，**不是 immediate delivery**。Preeclampsia 硬门槛 **BP ≥140/90**（132/85 不算）；HR 112 在心衰是**代偿性窦速**不是 fetal distress。Urgent delivery 仅 3 种：血流动力学不稳 / 难治性 HF / 胎儿窘迫。

> [!success] 5. β-blocker 中毒 → Glucagon (Q2663)
> 自杀 + bradycardia + hypotension + **wheezing + 低血糖** + IVF/atropine 无效 → **Glucagon 3-10 mg IV**（绕 β-receptor 从 Gs/AC → ↑cAMP）。三联中毒鉴别：**BB**（wheezing + 低血糖）/ **CCB**（高血糖）/ **Digoxin**（GI + 黄绿视 + 高 K）。"Atropine fail → think Glucagon"。

> [!success] 6. IE 感染性心内膜炎完整体系
> Fever + new murmur = IE until proven otherwise。**HEROIC** 手术指征：**H**F / **E**xtension（瓣周脓肿 / 新发 AV block）/ **R**esistant organism（fungi/MDR）/ **O**ngoing bacteremia ≥5-7d / **I**mmense embolism（vegetation >10mm）/ **C**onsultation early。三句秒杀：IE+new murmur → echo (TEE)；IE+new heart block → **Aortic IE + 瓣周脓肿** → TEE；IE+SOB+crackles → 急性瓣膜毁损 → emergency surgery。**Janeway = Just spots（无痛, 栓塞 / Vascular）; Osler = Ouch!（痛, 免疫 / Immunologic）**。

> [!success] 7. 心脏听诊定位 + 呼吸时相
> **AR 来源反推病因**：Left SB (Erb's point, 3-4 ICS) = **valvular AR**（瓣本身坏，含 PVD）；Right 2nd ICS = **root dilation AR**（Marfan / 夹层 / syphilis）。**RILE**：**R**ight-**I**nspiration（右心杂音吸气增强 = Carvallo's sign），**L**eft-**E**xpiration（左心杂音呼气末屏气增强）。AR 听诊全套姿势：坐位前倾 + 呼气末屏气 + 膜面 + 左胸骨旁 3-4 ICS。

> [!success] 8. 高钾血症完整体系
> ECG 进展 5 阶段：**T**ent (peaked T) → **P**ass (P 消失) → **W**ide (QRS) → **S**ine wave → **F**latline。治疗 **3 S 框架**：**S**tabilize（Calcium gluconate，1-3 min 起效，**ECG 一异常就给不等血钾**）→ **S**hift（Insulin/Glucose 最可靠 + Albuterol 协同 + NaHCO3 仅 pH<7.2）→ **S**ink（Furosemide 要有尿 / Patiromer/SZC 取代 Kayexalate / Dialysis）。透析适应症 **AEIOU**。三联高钾炸弹：ACEi/ARB + spironolactone + K supplement。

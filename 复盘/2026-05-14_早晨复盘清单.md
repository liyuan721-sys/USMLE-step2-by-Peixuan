## ⚠️ 重点关注（容易错的 6 个陷阱）

> [!warning] 这些是今天最容易错的点

1. **5 急症"第一件事"** ⭐⭐⭐
   - 不是"全部立刻 X" → 分**诊断/治疗**两步
   - 稳定 PE 高度可疑 → **经验性抗凝**（不等 CTPA）
   - 抗凝禁忌 + PE → **IVC filter**

2. **Wells 是两个评分** ⭐
   - 看腿 → DVT（9 项，≥2）
   - 看胸 → PE（7 项，>4，独有"PE 最可能" +3）

3. **MONA-BASH 现代化** ⭐
   - 真正"必须立刻"= **ASA + P2Y12 + Heparin**
   - O / N / M 都有条件
   - 下壁 MI 慎用 NTG

4. **抗凝药特殊人群** ⭐
   - 妊娠 → LMWH（不能 Warfarin / DOAC）
   - 机械瓣 → Warfarin（不能 DOAC）
   - APS → Warfarin / LMWH（DOAC 失败）
   - HIT → 不能 Heparin → Argatroban / Bivalirudin

5. **Adamkiewicz vs Watershed** ⭐
   - 一条动脉（T9-T12 左）vs 一个区域（T4-T8）
   - AAA 术后 → Adamkiewicz
   - 心脏骤停 → Watershed
   - 感觉平面定位法（脐/腹股沟 vs 乳头/剑突）

6. **依从性差 ≠ 直接换 LMWH** ⭐
   - 先教育 + 简化方案（QD DOAC + Pill box）
   - LMWH 注射依从性反而更差
   - 真治疗失败 + 严格依从仍复发 → 才换 LMWH

# 🎯 四大薄弱点 + 浓缩复习要点

## 薄弱点 1：抗凝药拮抗剂 4 对（完全空白）

> [!danger] 优先级 ⭐⭐⭐
> 完全空白 = USMLE 出血场景必考点，必须本周内背到反射弧。

| 抗凝药 | 拮抗剂 | 记忆钩 |
|---|---|---|
| **UFH** | **Protamine sulfate** | Protamine 拉硫酸根 = 与 heparin 负电荷结合 |
| **LMWH** | Protamine（只中和 60%）| 部分有效 |
| **Warfarin** + 严重出血/ICH | **4F-PCC + Vit K**（不是单独 FFP！）| PCC 几分钟起效；Vit K 需 6-24h；FFP 体积大、慢 |
| **Dabigatran**（抗 IIa）| **Idarucizumab**（Idar**a**ru = 抗 II**a**）| 单克隆抗体，5 分钟起效 |
| **Xa-DOAC**（Apix/Riva）| **Andexanet alfa**（And**ex**anet = 抗 **X**a）| 重组诱饵 Xa |
| **Fondaparinux** | 无特异拮抗 | 4F-PCC 部分有效 |

> [!tip] 三秒识别
> - "颅内出血 + 在吃 Warfarin"→ 立刻 **4F-PCC + Vit K**，不要犹豫 FFP
> - "DOAC 出血"→ 看是 II 还是 X：**Dabigatran=Idarucizumab，其他=Andexanet alfa**

## 薄弱点 2：抗凝药特殊人群选药（5 个全错）

> [!danger] 优先级 ⭐⭐⭐

| 人群 | 首选 | 禁用 | 原因 |
|---|---|---|---|
| **妊娠** | **LMWH**（Enoxaparin）| Warfarin（致畸）、DOAC（无数据）| LMWH 不过胎盘 |
| **机械瓣** | **Warfarin** INR 2.5-3.5 | DOAC（RE-ALIGN 试验失败）| 机械瓣表面 → 接触途径凝血，DOAC 抑不住 |
| **APS**（抗磷脂综合征）| **Warfarin** INR 2.5-3.5 / LMWH | DOAC（TRAPS/RAPS 试验失败）| 同上，多机制凝血 |
| **严重肾衰** CrCl<30 | **Warfarin** 首选 / **低剂量 Apixaban**（唯一允许的 DOAC）| Dabigatran/Riva/Edox（肾排）| Apixaban 仅 27% 肾排 |
| **HIT Type II** | **Argatroban / Bivalirudin**（DTI）或 **Fondaparinux** | **所有 Heparin（含 LMWH）** | 抗 PF4-Heparin 抗体 |

> [!warning] 题干识别
> - "妊娠 + DVT" = LMWH（不背就错）
> - "机械瓣"或"APS"= **Warfarin**（DOAC 是陷阱选项）
> - HIT = 血小板掉 50% + 5-10 天 + 新血栓 → **停所有 heparin** + Argatroban

## 薄弱点 3：5 急症"第一件事"（应用题全空）

> [!danger] 优先级 ⭐⭐⭐
> 这是 Step 2 CK 最高频的"分诊+决策"题型，21 道应用题里你直接全空 = 知识没固化到反应。

| 急症 | 诊断第一步 | 治疗第一步 |
|---|---|---|
| **PE 稳定 + 低/中可疑** | **D-dimer**（阴性可排）| 抗凝待结果 |
| **PE 稳定 + 高可疑**（Wells>4）| **CTPA**（不要 d-dimer 浪费时间）| **经验性抗凝**同时进行 |
| **PE 不稳定**（SBP<90）| **床旁 TTE**（看 RV strain）| **tPA 溶栓** + 抗凝（不是取栓！）|
| **PE + 抗凝禁忌**（近期大出血）| CTPA | **IVC filter** |
| **DVT** | **Duplex US** | 确诊后抗凝 |
| **MI**（STEMI）| **ECG**（10 min 内）+ Trop | **ASA + P2Y12 + Heparin** → PCI（90 min 门到球囊）|
| **Stroke** | **Non-contrast CT**（排出血）| **tPA**（4.5h 内）+ 取栓（≤24h LVO）|
| **ALI** | **临床 6 P's**（不等影像）| **立刻 IV Heparin** + 血管外科会诊 |

> [!danger] 三个高频陷阱
> 1. PE 不稳定 → **TTE** 不是 **TEE**（你写错了）；治疗是**溶栓**不是取栓
> 2. PE 高度可疑 → **不做 d-dimer**（浪费时间 + 假阴风险）
> 3. ALI → **不等影像**，临床诊断后立刻 IV Hep

## 薄弱点 4：三大束交叉时机（口诀写反了）

> [!warning] 优先级 ⭐⭐
> 概念懂、但口诀反向 → 考试焦虑下会答错 Brown-Séquard。

```
脊髓三大束交叉位置：

CST 皮质脊髓束        ┐
DCML 背柱（振动/本体）├─→ 在【延髓 medulla】交叉
                       ┘   → 脊髓损伤 = 同侧丢失

Spinothalamic 痛温觉  ─→ 在【脊髓损伤平面上 1-2 节段】交叉
                          → 脊髓损伤 = 对侧、平面下 1-2 节段丢失
```

> [!tip] 一句话记
> - **"痛温觉在脊髓里就交叉"** = Brown-Séquard 的痛温觉对侧丢、平面差 1-2 节段
> - **"CST / DCML 上去到延髓才交叉"** = 脊髓内未交叉 → 同侧征
> - 昨晚口诀里**痛温/CST 写反了**，重点纠正

# 🔧 次要错点（速修，不用专题）

- **T7 = 剑突**（不是 T6）；T4 nipple / T7 xiphoid / T10 umbilicus / L1 inguinal
- **Wells DVT ≥ 2 likely**（不是 ≥4）；**Wells PE > 4 likely**（不是 >6）
- **TBI < 0.7 = PAD**（钙化时 ABI 不可信用 TBI）
- **CLI = rest pain >2w / 慢性溃疡 / gangrene + ABI<0.4**（**claudication 不是 CLI**，只是 PAD）
- **6 P's = Pain / Pallor / Pulseless / Paresthesia / Paralysis / Poikilothermia**（"没毛" 是 **Paresthesia 感觉异常**，不是脱毛）
- **静脉性 claudication 机制 = 静脉瓣功能不全**（不是动脉瓣）
- **Neurogenic claudication 机制 = 腰椎椎管狭窄**
- **Adamkiewicz 本质**：单条动脉，T9-T12 **左**起源，供 anterior spinal artery，**AAA 修复术后**为典型 stem
- **Watershed 本质**：T4-T8 分水岭区，**心脏骤停 / 系统低灌注**为典型 stem

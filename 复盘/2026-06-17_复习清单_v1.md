## §一 🔴 反复错焊死区（最高优先 — 这些是你 2+ 次没拿稳的）

> [!danger] 今日新晋 🔴（昨天升级）
> - **ARDS (acute respiratory distress syndrome, 急性呼吸窘迫综合征) 低氧 = 分流 (shunt)**：**吸纯氧不纠正 → 治疗 = PEEP (positive end-expiratory pressure, 呼气末正压)**；不是弥散障碍、不是利尿。06-13/06-17 两次没焊死 → [[mistakes/uworld-mistakes_2026-06#^Q2303-R0617]]

> [!warning] 持续 🔴 主题（按 cluster 扫）
> - **镜像中毒三连**：CO (carbon monoxide, 一氧化碳) → 100% O₂/高压氧；氰化物 (cyanide) → 羟钴胺 (hydroxocobalamin)；抗胆碱 (anticholinergic, 苯海拉明 diphenhydramine) → 干红热散瞳。三者**别互换**（你自测反复翻转）
> - **有机磷中毒 (organophosphate poisoning)** → **阿托品 (atropine)** + 解磷定（已错 3 次，焊死）
> - **PE (pulmonary embolism, 肺栓塞) Wells > 4** → **直接 CTPA**，不先 D-二聚体 (D-dimer)（同日错 2 次）#难记
> - **CHF (congestive heart failure, 充血性心衰) 漏出液 (transudate)** → pH 7.45 方向（Light 标准别记反）
> - **PFPS (patellofemoral pain syndrome, 髌股疼痛综合征)** → 治疗 = **股四头肌强化 (quadriceps strengthening)，忌制动**（反复错）#难记
> - **坏死性外耳道炎 (malignant otitis externa)**：老年糖尿病 + 夜间剧耳痛 + 耳道肉芽 + 面瘫 → 铜绿假单胞 (Pseudomonas) → **IV 抗假单胞（环丙沙星 ciprofloxacin）**
> - **Ménière 病 (梅尼埃病)**：眩晕 1–2h + 耳鸣 + 感音聋 + 无头位诱发；限盐限咖啡因（vs BPPV, benign paroxysmal positional vertigo 良性阵发性位置性眩晕 / 外淋巴瘘 perilymphatic fistula，Valsalva 诱发）
> - **桡神经 vs 正中神经定位**（06-17 自测刚错）：桡神经看**伸肘**（三头肌在 = 螺旋沟/干中段）；正中神经**大鱼际掌侧麻 = 掌皮支 = 旋前圆肌综合征，不是 CTS** #难记

## §二 滚动复习（一句话提醒）

### D+3 = 06-14（肺 + 儿科 + 骨科·风湿，26 张）→ 详见 [[复盘/2026-06-15_昨日学习_浩斯宾滚动]] §一
- 复杂性肺炎旁积液 (complicated parapneumonic effusion, pH<7.2/糖<60/脓/菌+) → **胸管引流**；单纯小量无窘迫 → 口服抗生素门诊
- 大剂量沙丁胺醇 (albuterol) → **低钾血症 (hypokalemia)**（β₂ 驱钾入胞）+ 震颤 + 反射↓
- IPF (idiopathic pulmonary fibrosis, 特发性肺纤维化) PFT = **限制型**（↓TLC, total lung capacity 肺总量 / FEV₁-FVC 正常↑ / **↓DLCO, diffusing capacity 弥散量**）
- 股骨头骨坏死 (AVN, avascular necrosis) = **慢性酗酒 > 短程激素**；MRI 最敏感

### D+7 = 06-10（ENT 大块）
- 耳廓血肿 / 鼻中隔血肿 → **切开引流**（防菜花耳 / 鞍鼻穿孔）
- 呋塞米 (furosemide) 耳毒性：对称高频感音聋 + 大剂量袢利尿（非环丙沙星）
- 急性外耳道炎（清亮鼓膜）vs 坏死性外耳道炎（糖尿病 + 面瘫 + 肉芽）
- 血清病 (serum sickness, III 型超敏)：用 infliximab 后发热 + 荨麻疹 + 关节痛 + 蛋白尿
- CVID (common variable immunodeficiency, 普通变异型免疫缺陷)：成人反复肺炎 + 鼻窦炎 → 查免疫球蛋白

### 弹性补位 = 06-12/06-11（中毒大块，D+5/D+6）
- 甲醇中毒 (methanol)：视神经炎 + 高 AG (anion gap, 阴离子间隙) 代酸 → **甲吡唑 (fomepizole)**
- 高原脑水肿 (HACE, high-altitude cerebral edema) → **地塞米松 (dexamethasone)**（利尿剂是陷阱）
- 强碱误服 (caustic ingestion) → 24h 内镜

### D+30 = 05-18（心内 + 感染）
- IE (infective endocarditis, 感染性心内膜炎) + 新发 AV block (atrioventricular block, 房室传导阻滞) → **瓣周脓肿**
- Severe AS (aortic stenosis, 主动脉瓣狭窄) 体征识别（Q4277 第二次错，注意）
- 高钾血症 (hyperkalemia) 紧急处理三联（钙稳膜 → 转移 → 排出）
- MI 机械并发症 — VSR (ventricular septal rupture, 室间隔破裂)

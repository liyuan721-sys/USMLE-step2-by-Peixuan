长病程糖尿病患者，手掌无痛结节 + 第4、5指逐渐屈曲挛缩、伸不开，无痛无弹响 → 诊断？还有哪两个"糖尿病手"病变常同框考？	Dupuytren 掌腱膜挛缩 (Dupuytren contracture)。糖尿病手部三联 = Dupuytren + 扳机指 (trigger finger, 狭窄性腱鞘炎) + 糖尿病手关节病 (diabetic cheiroarthropathy，招牌=祈祷征 prayer sign 阳性)。共同机制 = 慢性高血糖 → 胶原非酶糖基化 (glycation) + 结缔组织僵硬。再加腕管综合征构成"糖尿病手"谱系。⚠️ Dupuytren 单独考时第一关联是酒精/肝硬化，糖尿病是第二梯队。📄 出处：骨科_腕手及周围疾病鉴别 §四
Dupuytren 掌腱膜挛缩 (Dupuytren contracture) 最经典的危险因素是什么？糖尿病排第几？	第一 = 酒精 / 肝硬化 (alcohol / cirrhosis)，stem 最爱给酗酒史。糖尿病 (DM) 是第二梯队关联。其他：北欧裔（"Viking disease"）、男性、年龄大、苯妥英 (phenytoin)、吸烟。📄 出处：骨科_腕手及周围疾病鉴别 §四
糖尿病患者双手合十时手指关节不能完全并拢/伸直、掌面留缝（祈祷征阳性）→ 什么病？机制？	糖尿病手关节病 (diabetic cheiroarthropathy / limited joint mobility, 关节活动受限)，体征 = 祈祷征 (prayer sign) 阳性。机制 = 慢性高血糖 → 胶原非酶糖基化 + 结缔组织僵硬，提示血糖控制不佳。📄 出处：内分泌_HighYield速查总览 §4.6
输 IV 钙 (calcium gluconate) 的两大逻辑分别是什么？各覆盖哪些情况？	A. 钙真低 → 补钙 (replacement)：症状性低钙血症（术后甲旁减 / 氢氟酸烧伤 / 大量输血枸橼酸 / 乙二醇中毒 / 胰腺炎-TLS）。B. 钙不一定低 → 拿钙当稳膜剂/拮抗剂：高钾 / 高镁 / 钙通道阻滞剂 (CCB) 中毒。葡萄糖酸钙外周首选（氯化钙仅中心静脉）。📄 出处：肾脏_电解质急症速查 §九
高钾和高镁都输钙 —— 两者"钙的角色"措辞有何不同？	高钾：钙稳定心肌膜、上移阈值电位 → 拉开"静息—阈值"差（不降钾，只买时间，真正降钾靠 shift + sink）。高镁：钙是镁的生理拮抗剂 (physiologic antagonist)、即刻解毒（子痫前期 MgSO4 中毒，深腱反射消失是最早预警）。两者都不降原离子。📄 出处：肾脏_电解质急症速查 §九
哪两种情况"别反射输钙"（反 pattern）？	① 重度高钙血症 (hypercalcemia) 危象绝不输钙 → 正确是 NS 水化 + 降钙素 (calcitonin) + 双膦酸盐 (bisphosphonate)。② 地高辛 (digoxin) 中毒的高钾慎/避免输钙——理论上加重强直收缩致"石头心 (stone heart)"；真正解药 = 地高辛特异抗体 Fab 片段 + 常规降钾。📄 出处：肾脏_电解质急症速查 §九
钙通道阻滞剂 (CCB, 如 verapamil/diltiazem) 过量 → 低血压 + 心动过缓，除支持外用什么？β 阻滞剂过量呢？	CCB 过量：大剂量 IV 钙（直接对抗 L 型钙通道封锁）+ 胰高血糖素 (glucagon) + 高剂量胰岛素 (high-dose insulin euglycemia)。β 受体阻滞剂过量：一线是胰高血糖素 (glucagon)，钙仅辅助。📄 出处：肾脏_电解质急症速查 §九
propofol 丙泊酚短期使用（诱导/短镇静）最常见的副作用有哪些？	低血压 (hypotension，血管扩张 + 心肌抑制，最常见) + 呼吸抑制/呼吸暂停 (respiratory depression/apnea) + 注射痛 (pain on injection，可先给利多卡因) ± 心动过缓 (bradycardia)。⚠️ 只镇静不镇痛 (no analgesia)，需止痛得另加阿片。📄 出处：外科_麻醉并发症恶性高热 §5.1
ICU 长期大剂量输 propofol 丙泊酚，出现代谢性酸中毒 + 横纹肌溶解 + 高钾 + 心衰 → 诊断？危险阈值？处理？	丙泊酚输注综合征 (Propofol Infusion Syndrome, PRIS)。阈值：>4-5 mg/kg/h 且 >48 小时（儿童尤其高危）。机制 = 线粒体脂肪酸氧化受损。五联 = 乳酸性代谢性酸中毒 + 横纹肌溶解(→高钾 + AKI) + 心衰/难治性心动过缓 + 高甘油三酯血症。处理 = 立即停丙泊酚 + 支持（纠酸/纠高钾/循环，重症透析或 ECMO）。📄 出处：外科_麻醉并发症恶性高热 §5.2
propofol 丙泊酚三个爱考的小点（绿尿 / 镇痛 / 过敏）+ 脂肪乳剂相关风险？	绿色尿 (green urine) = 酚代谢物，良性无害（干扰项）；无镇痛 (no analgesia，需另加阿片)；蛋/豆过敏 (egg/soy allergy，卵磷脂/大豆油载体，真过敏罕见)。脂肪乳剂配方 (lipid emulsion, 约 1.1 kcal/mL) → 高甘油三酯血症 + 可诱发急性胰腺炎，长期输注需计热量+监测甘油三酯。优点面：止吐 + 降 ICP/脑代谢。📄 出处：外科_麻醉并发症恶性高热 §5.3

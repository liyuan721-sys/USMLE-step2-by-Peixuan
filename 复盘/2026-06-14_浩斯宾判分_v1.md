## 二、错题详解（焊死）

### Q7 抗胆碱能 vs 拟胆碱能（汉字翻转）

苯海拉明 (diphenhydramine) → **抗胆碱能 anticholinergic**（干热红盲疯）。误选 A 拟胆碱能（=cholinergic，反向）。

> [!danger] 一字之差方向相反
> - **拟胆碱能 cholinergic**（模拟 ACh）= 有机磷类 = **湿**（DUMBELS：分泌物↑、瞳孔缩小、心动过缓）
> - **抗胆碱能 anticholinergic**（阻断）= 苯海拉明/阿托品 = **干**（口干、皮肤干红、瞳孔散大、尿潴留、肠鸣音↓、谵妄）
>
> 钥匙：看到"干"全套（dry/hot/red/blind/mad）= **抗**胆碱能。

### Q8 + Q9 CO ↔ 氰化物镜像翻车（核心）

两题故意相邻，靠实验室指纹区分；本次左右互换。

| | **Q8 → 氰化物 cyanide** | **Q9 → CO 一氧化碳** |
|---|---|---|
| 指纹 | 碳氧血红蛋白 (COHb, carboxyhemoglobin) 仅轻度↑ + **乳酸显著↑** + 对氧反应差 | **COHb 28%（明确高）** + 乳酸不高 + 无樱桃红 |
| 正解 | **B 氰化物 → 羟钴胺 hydroxocobalamin** | **A 100% 高流量氧 (O₂)** |

> [!warning] 鉴别钥匙
> 樱桃红 + 火灾两者都有，**不能用来区分**。分水岭是两个数：
> - **COHb 高 + 乳酸正常 → CO**（一线 100% O₂；高压氧 (HBO, hyperbaric oxygen) 仅重症/神经/孕妇）
> - **COHb 仅轻度高 + 乳酸 ≥10 + 对氧无反应 → 氰化物**（毒线粒体→无氧酵解→乳酸飙升；羟钴胺）
>
> 口诀："乳酸离谱、给氧没用 = 氰化物"；"COHb 明确高、乳酸还好 = CO"。
> Q9 D 选项"立即高压氧（唯一选择）"错——CO 一线是常压 100% O₂，HBO 非唯一选择。

## 四、我的追问与解析

### ① Q5 — β阻滞剂中毒 vs BZD 中毒怎么分

| | **β阻滞剂 (beta-blocker) 中毒** | **苯二氮䓬 (BZD, benzodiazepine) 中毒** |
|---|---|---|
| 神志 | 可清醒，重度才抑制 | 嗜睡/昏睡但**可唤醒** |
| 心率 | **心动过缓 (bradycardia)** ↓↓ | 基本正常 |
| 血压 | **低血压 (hypotension)** | 多正常 |
| 血糖 | **低血糖 (hypoglycemia)**（尤其小孩）| 正常 |
| 呼吸 | 多正常 | 轻度抑制（单药很少致命）|
| 瞳孔 | 正常 | 正常 |
| 解毒 | **胰高血糖素 (glucagon)** 一线 ±高胰岛素 | 氟马西尼 (flumazenil)，慎用（易诱发抽搐）|

> [!tip] Q5 钥匙
> **"生命体征基本正常 + 可唤醒 + 瞳孔正常 + 无针眼" = BZD**。
> - 与**阿片 (opioid)** 区别：阿片有**针尖瞳孔 + 明显呼吸抑制**。
> - 与 **β阻滞剂**区别：β阻滞剂有**心动过缓 + 低血压 + 低血糖**三联。
> - BZD 单药最大特点："看起来啥都正常，就是叫不太醒"。

### ② Q16 — 对，是 TCA（三环抗抑郁药）

**终末 R 波于 aVR + QRS 增宽 (>100 ms) + 抗胆碱能表现（瞳孔散大/口干/心动过速）= 三环抗抑郁药 (TCA, tricyclic antidepressant) 过量**，钠通道毒性。

> 解毒 = **碳酸氢钠 (sodium bicarbonate)**——不是单纯纠酸，是用钠负荷 + 碱化**逆转心肌钠通道阻滞**。答 A 正确。

### ③ Q22 — 激素时机不是"看神经症状"，是"压线给"

> [!warning] 纠正一个理解
> 不是"出现神经症状才加激素"——而是**怀疑细菌性脑膜炎 (bacterial meningitis) 时，地塞米松 (dexamethasone) 要在首剂抗生素之前或同时给**（给晚了无效）。
> - **目的**：减少神经并发症（尤其**听力损失**）+ 降死亡率
> - **最强证据**：**肺炎链球菌 (S. pneumoniae)** 成人脑膜炎
> - **流程**：经验性先打 → 病原回报后，肺炎链球菌→继续地塞米松；其他病原→停。
> - 正解 B："立即经验抗生素（±地塞米松）随后腰穿"——**抗生素和激素都不能等腰穿/CT**。

### ④ Q25 — 为什么是巴贝虫不是疟疾 + 无脾风险

> [!success] 为什么不是疟疾 (malaria)
> 三条决定性证据：
> 1. **马耳他十字/四联体 (Maltese cross / tetrad)** = 巴贝虫 (Babesia) **特征性**，疟疾**没有**；
> 2. 巴贝虫**无疟色素 (no hemozoin pigment)**；疟疾红细胞内有疟色素 + 香蕉形配子体（恶性疟 P. falciparum）；
> 3. **流行病学**：巴贝虫 = 美国东北部 Ixodes 蜱（与 Lyme 同一只蜱）、夏季户外；疟疾要有**疫区旅行史**——题里没有 → 排除疟疾。

> [!question]- 无脾 (SCD) 得疟疾和巴贝虫的风险一样吗？
> **不一样，分"得病风险"和"重症风险"两层：**
>
> **① 感染风险（会不会得）**：取决于**暴露**，与脾无关。被携带巴贝虫的蜱咬 → 巴贝虫；去疟疾疫区被按蚊咬 → 疟疾。美国本土 + 蜱接触 + 无旅行史 → 暴露指向**巴贝虫**。
>
> **② 重症风险（得了多重）**：脾是过滤寄生虫感染红细胞的主器官。
> - **无脾/镰状细胞病 (SCD, sickle cell disease) → 巴贝虫可暴发致命**（asplenia 是巴贝虫重症**最重要**危险因素）；
> - 无脾对疟疾也会加重，但 USMLE 把"无脾 + 美国户外 + 马耳他十字"这套**专门用来指向重症巴贝虫**。
>
> **反直觉补充**：**镰状细胞特质 (sickle cell trait, HbAS)** 反而对恶性疟有**保护**（感染的红细胞易镰变被清除）——经典进化考点；但**纯合 SCD + 功能性无脾**一旦感染巴贝虫则凶险。
>
> 一句话：**风险不对等——这名无脾患者该警惕的重症是巴贝虫，不是疟疾。**

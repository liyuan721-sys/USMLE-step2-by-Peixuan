## §二 错题逐题详解

> [!warning] Q5 溺水（你选 A BiPAP / 正确 C 插管）
> 无创通气（BiPAP, 双水平气道正压 bilevel positive airway pressure）要求患者**清醒能配合 + 能保护气道**。本题「意识逐渐模糊 + 无法配合 + 高流量面罩下仍 SpO₂ 84%」全是 BiPAP 禁忌 → 直接**气管插管 + 机械通气**。
> - 粉红泡沫痰 = 溺水后 ARDS（急性呼吸窘迫综合征 acute respiratory distress syndrome）/ 肺水肿（非容量过载，是肺泡毛细血管漏）→ 呋塞米 furosemide (D) 无效。
> - **钥匙**：意识改变 / 不配合 / 气道不保 = 直接插管，不要 BiPAP。

> [!warning] Q6 β阻滞剂过量（你选 D 氟马西尼 / 正确 A 高胰岛素正糖）
> 你记得 glucagon 是对的 —— **glucagon 确实是 β-blocker (美托洛尔 metoprolol) 过量的经典一线解毒剂**（绕过被阻断的 β 受体直接升 cAMP）。但本题选项**没给 glucagon**，选项内最佳 = **A 高剂量胰岛素正糖疗法 (HIET, high-dose insulin euglycemic therapy)**，对 β-blocker 和 CCB（钙通道阻滞剂 calcium channel blocker）过量都管用、难治性低血压首选。
> - 你选的 D 氟马西尼 flumazenil 是 BZD（苯二氮䓬 benzodiazepine）拮抗剂，与本题完全无关。
> - **对子**：β-blocker / CCB 过量 → **glucagon 或 HIET**（钙剂 calcium 主要给 CCB）。选项没 glucagon 就选 HIET。

> [!warning] Q13 水杨酸中毒（你选 B 碱化尿液 / 正确 C 血透）
> 你自己也说"忘记处理了"。轻中度水杨酸 salicylate = 碳酸氢钠 sodium bicarbonate 碱化尿液（B）；但本题有 **AKI（急性肾损伤 acute kidney injury）+ 少尿 + 重度 + 意识模糊** → 达到**血液透析 hemodialysis 指征**。
> - **血透指征（占一条就上）**：肾衰 / 容量超载 / 严重酸中毒 / 意识改变 / 肺水肿 / 血药浓度极高。

> [!warning] Q15 阿片戒断误给药（你选 D / 正确 B 纳洛酮）
> D（丁丙诺啡延迟给药）描述的是**正确做法**，不是"被误用应避免的药"。海洛因 heroin 依赖者本就在戒断中，再给**纳洛酮 naloxone** → 瞬间拮抗残留阿片 → 暴发性戒断、剧烈躁动。
> - 被误用且应避免 = **纳洛酮 B**。

> [!warning] Q17 梅毒随访（你选 A FTA-ABS / 正确 B RPR/VDRL 滴度）
> 螺旋体试验（FTA-ABS / TPPA）**治疗后终身阳性，不能看疗效**；追踪疗效/活动度用**非螺旋体试验 RPR/VDRL 滴度下降**（治愈 = 4 倍下降）。
> - **钥匙**：非螺旋体看活动度（会变），螺旋体看确诊（永远阳）。

> [!danger] Q22 「干」针尖瞳（你选 D 氟马西尼 / 正确 B 纳洛酮）⚠️ 重点升 🔴
> 本题是 Q9/Q20「湿」有机磷的**镜像对照**：**皮肤干燥温暖 + 无流涎 + 无肌束颤 + 呼吸抑制 + 针尖瞳 = 阿片 opioid → 纳洛酮 naloxone**。
> - 你三道「湿」题（Q9/Q20）全对，唯独这道「干」题翻车 → 钥匙记成了"针尖瞳→找湿征→有机磷"的**单向反射**，反过来"无湿征→阿片"没建立。
> - 氟马西尼 flumazenil 是 BZD 拮抗剂，BZD 过量瞳孔正常（非针尖瞳），完全不沾边。
> - **焊死双向钥匙**：针尖瞳先分湿干 —— **有湿征（汗/涎/颤/缓脉）= 有机磷 → 阿托品 atropine；无湿征 + 呼吸抑制 = 阿片 → 纳洛酮 naloxone**。

> [!warning] Q4 错点（"皮肤干"误判 BZD 为 TCA）

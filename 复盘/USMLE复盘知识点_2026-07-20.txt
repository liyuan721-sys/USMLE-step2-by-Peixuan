屈膝落地「pop」+ 关节积血。能不能靠「肿得快慢」区分 ACL 撕裂 vs 髌骨外侧脱位 (lateral patellar dislocation)？	不能——两个都迅速肿（数小时内关节积血 hemarthrosis，整膝绷紧）。<br>机制：ACL = 富血供韧带（膝中动脉 middle genicular artery），一撕即出血；髌骨外侧脱位 = 撕 MPFL（内侧髌股韧带 medial patellofemoral ligament）+ 常伴骨软骨骨折 osteochondral fracture = 两个出血源。<br>所以「肿得快慢」只能排掉半月板（内 2/3 无血供 → 肿得慢 + 交锁 locking），排不开这两个。<br>这一对靠「髌骨在不在滑车 + 内侧 MPFL 压痛」切：髌骨在位 + Lachman+ = ACL；髌骨外移块 + 滑车沟空凹 divot + 内侧压痛 = 脱位。<br>经典错点：听到 pop 就跳 ACL。(Q13974)<br>📄 出处：骨科_膝及周围疾病鉴别 §一-补
「pop + 快速关节积血 hemarthrosis」三个鉴别诊断（半月板 / ACL / 髌骨外侧脱位），如何两步切开？	第一刀（肿速）：排掉半月板 meniscus——它肿得慢 + 交锁 locking（内 2/3 白区无血供）。<br>第二刀（髌骨位置）：<br>· 髌骨在位 + Lachman+，内侧一般不痛 = ACL 撕裂<br>· 髌骨外移块 + 滑车沟空凹 divot + 内侧 (MPFL) 压痛 = 髌骨外侧脱位<br>关键：肿得快慢是 ACL↔半月板 的钥匙，不是 ACL↔髌骨脱位 的钥匙。(Q13974)<br>📄 出处：骨科_膝及周围疾病鉴别 §一-补 / §二 冤家对
为什么半月板撕裂「隔天才慢慢肿」，而 ACL 撕裂「几小时就积血肿满」？	血供差异：<br>· 半月板 meniscus：外 1/3 红区 (red zone) 有血供、内 2/3 白区 (white zone) 几乎无血供 → 撕裂后主要靠机械症状 (catching/locking 交锁) 提示，慢肿。<br>· ACL：富血供韧带 → 是急性创伤性关节积血 (hemarthrosis) 最常见原因（约 70%），快肿。<br>临床一刀：快肿 → 想 ACL / 髌骨脱位；慢肿 + 交锁 → 想半月板。<br>📄 出处：骨科_膝及周围疾病鉴别 §一 / §一-补

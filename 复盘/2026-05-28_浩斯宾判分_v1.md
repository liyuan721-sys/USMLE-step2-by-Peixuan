## §三 2 错题深挖

### ❌ Q17 Atypical MDD First-line ≠ MAOI ⚠️

**根本原因**：UWorld 模式记忆"**Atypical depression = phenelzine answer**"在你脑里形成强反射 → 看到 leaden paralysis + 嗜睡 + 食欲↑ → 直接选 C，没有注意选项 C 的括号 "(after SSRI trial fails)" **已经明示这不是 first-line**。

**正解钉死 — Atypical MDD 治疗 6 阶梯（Q15925 主题字段已写）**：

```
First-line:    SSRI (Sertraline / Fluoxetine) 或 SNRI (Venlafaxine)
                ↓ 失败 / intolerance
Second-line:   Bupropion 或 Mirtazapine
                ↓ partial response
Augment:       Lithium / T3 / Aripiprazole
                ↓ multi-step 失败
MAOI:          Phenelzine ⭐ atypical 经典专属（modern 时代留作"难治性首选 MAOI"）
                ↓ 极难治
ECT / TMS / Esketamine
```

**为什么 USMLE Step 2 答案 = SSRI 而非 MAOI**：
- 现代指南（2024 APA / CANMAT）：**SSRI 优于 MAOI** 因为安全性（MAOI = hypertensive crisis + 饮食限制 + 药物相互作用大）
- UWorld Step 2 2024 题库已**全面修正**为现代指南
- "Atypical 经典专属 MAOI"是 70-90 年代的教学遗留，现 Step 2 仅在 stem 写 "trial of SSRI failed + SNRI failed + augmentation failed" 时才选 phenelzine

**你的反射建立路径**：
- 5-26 Dashboard 记 "Atypical MDD MAOI (phenelzine 不是 lithium) 5-25 + 5-26 连对 2 次 phenelzine" — **但那次测的是"augment 阶段 lithium vs MAOI 比较"，不是 first-line！** 你记成了"Atypical = phenelzine 直接选" → 反射外推错。

**Memory Hook（修正版）**：
- 看到 atypical depression buzzword → **先想 SSRI/SNRI first-line**
- 看到"SSRI tried 6 weeks 无 response + bupropion tried 4 weeks 无 response + augmentation tried 失败" → 才反射 phenelzine

### ❌ Q19 DLB ψ vs PD ψ 钓饵（PD ψ Step 5 反射误锁）⚠️⚠️

**根本原因**：stem 写"worsening Parkinson disease"+"PD ψ"+"add 治疗" 触发 **5-27 5-28 刚反复训练的 PD ψ Step 5 反射 → 直接选 quetiapine**。**忽略 3 个 DLB 决定性钩子**：
1. **"started before his levodopa was initiated"** — ψ **早于** levodopa = 不是 drug-induced PD ψ
2. **"His cognition fluctuates throughout the day"** — DLB 核心诊断标准
3. **"episodes of unresponsiveness lasting minutes"** — DLB 极端 cognitive fluctuation

**DLB vs PD ψ 钉死决策树**：

```
有 parkinsonism 的 ψ：
│
├─ ψ 是 levodopa 起始后才出现 + 无 cognitive fluctuation
│      = PD ψ (drug-induced)
│      → Step 5 quetiapine / pimavanserin（不停 levodopa） ⭐ 5-27 已克服
│
├─ ψ 早于 levodopa + cognitive fluctuation + episodic unresponsiveness
│  + REM sleep behavior disorder + 严重视觉幻觉
│      = DLB (Dementia with Lewy Bodies)
│      → Rivastigmine / Donepezil 一线 ⭐
│      → 抗精神病药即使 quetiapine 30-50% 严重 sensitivity reaction（NMS-like）
│      → "1-year rule": parkinsonism ≤1 yr before cognitive decline = DLB
│                                >1 yr before = PD with dementia (PDD)
│
└─ AD/VaD/FTD 同时 parkinsonism（小概率）→ 各按主病治
```

**关键鉴别词反射钩**：
- "**ψ before levodopa**" → DLB 红旗
- "**fluctuating cognition**" → DLB 核心
- "**brief episodes of unresponsiveness**" → DLB 极端
- "**REM sleep behavior disorder**" → DLB 支持
- "**severe antipsychotic sensitivity**" → DLB 警告
- "**parkinsonism + dementia within 1 year**" → DLB（>1 yr = PDD）

**为什么 quetiapine 都不能给 DLB**：
- DLB 患者对 D2 拮抗剂极敏感 → 即使 atypical（quetiapine / clozapine）也可触发 **severe NMS-like reaction**（rigidity + 高热 + AMS + autonomic instability + 死亡率 ↑）
- 一线 = **cholinesterase inhibitor**（rivastigmine 有 DLB FDA 适应症，donepezil 也可）→ 同时治 ψ + cognitive fluctuation + parkinsonism 都有获益
- 若 ψ severe + cholinesterase 失败 → **pimavanserin**（5HT2A inverse agonist，无 D2 拮抗 → 安全）

**入库建议**：[[mistakes/uworld-mistakes#^Q19-2026-05-28]] 🔴 + **#考前必看 + #难记**（pattern 反射钓饵 + DLB 知识缺口双因）
**衍生增量**：[[完整笔记/专题笔记/psych/psych_PD_psychosis_5步阶梯]] 加 §六 **DLB vs PD ψ 决策树**（含 1-year rule + 抗精神病药相对禁忌警告 + rivastigmine 一线）

### 5.2 4 类联动 logic（Step 2 经典考点）

```
Structural （地基）          Process （执行）             Outcome （结果）
  人 / 设备 / 知识  ─────→   按 SOP 跑 ─────→   患者好不好
  ↑↓                          ↑↓                          ↑↓
  Structural ↑ 但 Outcome 没 ↑ → 找 Process 失败位 ⭐

                       Balancing （副作用）

```

**经典 vignette**：
- "**Structural 改善（加 pharmacist + CDSS）+ Outcome 不变**" → 必须**测 Process**（pharmacist 是否真的查 reconciliation? CDSS 是否真的看 alert?）
- "**Process compliance 100%** + Outcome 不变" → 重审 Structural（足够 pharmacist 吗？知识够吗？）或 reconsider intervention
- "**Process compliance ↑** + Outcome ↑ + Balancing 出现"（如压 opioid + 减 30天 readmit，但患者满意度 ↓）→ 评估 tradeoff 是否可接受

### 5.3 Q9 stem 再读 + 教学拆解

> "Hospital tracks: Structural (pharmacist/100 beds) + Balancing (患者 satisfaction with med counseling) + Outcome (preventable ADE rate). All three trending favorable. **Med reconciliation completion before discharge unmeasured**."

- 3 类有了 S + B + O，**缺 Process** = med reconciliation completion rate（执行流程指标）
- 为什么必须加：S + O 同时 ↑ 但**不知道 Process 是否真在执行** — 万一 ADE ↓ 是其他原因（药品种类变化、患者人口结构变化）而非 reconciliation 起效，缺 Process 数据则无法因果归因
- B（satisfaction）虽然涵盖患者主观体验，但**不替代 Process compliance**（满意度高 ≠ 流程执行）

### 5.4 4 类 Memory Hook ⭐

> [!tip] 4 类反射钩
> - **Structural** = "你**有**什么" → 资源 baseline（人 / 物 / 知识）
> - **Process** = "你**做**什么" → 执行率（% / completion / compliance / adherence）
> - **Outcome** = "患者**怎么了**" → rate / mortality / readmit / complication
> - **Balancing** = "**代价**是什么" → unintended / tradeoff / counterbalance

### 5.5 易混陷阱（普适）

- **Structural ≠ Outcome**：Pharmacist 数（S）不是 ADE 数（O）；ICU bed 数（S）不是 sepsis 死亡率（O）
- **Process ≠ Outcome**：Time-out 合规率（P）不是 wrong-site 错误率（O）；door-to-balloon ≤30 min（P）不是 STEMI 死亡率（O）
- **Outcome ≠ Patient Satisfaction**：30-day readmit 是 outcome；patient satisfaction 在传统 4 分类是**Outcome 子类**（patient-reported outcome），但**当用于 tradeoff 评估时**可分类为 Balancing
- **Balancing 不一定是 negative**：减 opioid + 患者疼痛 satisfaction ↓ = 负 tradeoff；但加 telehealth + 老年 access 提升 + 但 IT cost ↑ = 中性 tradeoff，仍属 Balancing

## 💡 核心知识点（一行一条，合并所有 part）

### 🧠 脊髓综合征模块（Part 1）

- **ACS 经典三要素**：前 2/3 梗死（motor + pain/temp 丧失）+ dorsal column 保留（vibration/proprioception 正常）+ Adamkiewicz 损伤
- **Adamkiewicz 动脉**：最常 T9-T12 左侧，watershed 区 = T4-T8（AAA 术后高危位置）#薄弱点
- **脊髓病变通用规则**：病变水平 = LMN signs（anterior horn）；病变以下 = UMN signs（corticospinal 中断）
- **急性脊髓损伤**：先 spinal shock（flaccid）→ 数天-周后转 spastic
- **感觉束交叉规律**：Dorsal column 走到 medulla 才交叉（同侧丧失）；Spinothalamic 进脊髓 1-2 节段内就交叉（对侧丧失）
- **Brown-Séquard 同侧 vibration / 对侧 pain-temp**：源于上述交叉时机差异
- **CCS 上肢 > 下肢机制**：Lateral CST 内部躯体定位"手内 / 脚外"，中央病变先打上肢
- **反向考点**：ACA stroke 下肢 > 上肢（皮层 homunculus 倒置）
- **Syringomyelia 三联**：cape-like 分离性感觉障碍 + 上肢 LMN 萎缩 + 多伴 Chiari I（C2-T9 最常见）
- **Syringomyelia 5 幕剧**（从内向外烂）：前白连合 → 外侧角（Horner）→ 前角（上肢 LMN）→ CST（下肢 UMN）→ Sacral sparing 最后破

### 🩸 外周血管病模块（Part 2）

- **ABI**：≤ 0.9 PAD；> 1.3 钙化（DM/CKD）→ 改 **TBI**（趾肱）< 0.7
- **TBI 缩写歧义**：Toe-Brachial Index（PAD）vs Traumatic Brain Injury（Neuro）
- **CLI**：rest pain > 2 周 + ABI < 0.4 / 溃疡 / 坏疽 → **必须 revasc**
- **CLI 招牌钩子**：**"Dangles leg over bed at night"**（重力辅助灌注缓解疼痛）
- **ALI 病因校准**：~30% 栓塞 + **~70% 原位血栓**（不全是栓子）
- **ALI 6 P's + 6 h 窗口**：Pain/Pallor/Pulselessness/Paresthesia/Poikilothermia/Paralysis
- **Wells 是两个评分**：DVT（9 项，≥ 2 likely → US）vs PE（7 项，> 4 likely → CTPA，独有"PE 是最可能 +3"主观项）
- **PE 三分层**：Low-risk / Submassive（RV strain）/ Massive（休克）
- **Massive PE 招牌组合**：**新发 RBBB + 呼吸困难 + 低血压** → 0.5 秒锁定 → tPA 准备
- **Submassive PE 5 项 RV 标志**：回声 RV dilation/hypokinesis / CTPA RV/LV > 0.9 / BNP↑ / Troponin↑ / ECG 新 RBBB
- **McConnell sign**：RV free wall 不动，apex 仍动 = 急性 PE 特异
- **Phlegmasia Cerulea Dolens**：严重髂股 DVT → **静脉淤血继发动脉缺血** → 治疗 = IV Heparin + CDT/取栓
- **Phlegmasia Alba vs Cerulea**：Alba 早期（白色，反射性动脉痉挛）→ Cerulea 晚期（青紫，继发动脉缺血）
- **动脉 vs 静脉病治疗 5 大对比**：抬高 / 压迫 / 抗板 / 抗凝 / 运动 / 溶栓 — 全部反着
- **动脉病禁忌 compression**（加重缺血）；静脉病 compression 前必查 ABI
- **白血栓 vs 红血栓**：动脉 = 血小板主导 → 抗板；静脉 = 纤维蛋白主导 → 抗凝
- **DVT 妊娠只能 LMWH**（不能 DOAC/Warfarin）；癌症 DOAC > LMWH > Warfarin

### 🚨 5 大急性血栓事件治疗哲学（Part 2 核心产出）

- **PE 不稳定** → tPA；**DVT** → DOAC
- **MI** → DAPT + PCI（90 min）；Heparin 辅助
- **Stroke** → tPA（4.5h）/ 取栓（24h） → **急性禁用抗凝/抗板**，24h 后才 ASA
- **ALI** → IV Heparin + 紧急手术（6h 窗口）
- **核心反直觉**：脑子最娇贵，急性不抗凝不抗板（出血转化致命）
- **时间窗口口诀**："90-4.5-6-24"（MI PCI / Stroke tPA / ALI / Stroke 取栓）
- **同栓子不同脏器治疗反差**：AFib 栓塞 → Stroke 用 tPA / ALI 用 Heparin
- **MONA-BASH 现代化**：真正必须立刻 = ASA + P2Y12 + Heparin；O 和 N 都有条件；M 仅按需

腕管综合征 (CTS, carpal tunnel syndrome) 压迫正中神经时，三条分支——指皮支 (palmar digital) / 掌皮支 (palmar cutaneous) / 返支 (recurrent motor)——各自受累还是保留? 判据是什么?	判据 = 分支相对屈肌支持带 (transverse carpal ligament) 的走行位置。<br>① 指皮支 palmar digital (图紫): ❌ 受累(穿腕管) → 桡侧 3½ 指 + 桡侧掌面麻"紫全灭"。<br>② 掌皮支 palmar cutaneous (图绿): ✅ 保留(腕管近端分出、走支持带浅面) → 大鱼际皮肤感觉正常"绿还亮"。<br>③ 返支 recurrent motor: ❌ 受累(纤维穿腕管) → 大鱼际肌(thenar: 拇短展肌 abductor pollicis brevis APB / 拇对掌肌 opponens pollicis / 拇短屈肌浅头 flexor pollicis brevis FPB)无力+萎缩。<br>鉴别: 大鱼际"皮肤感觉正常但肌肉萎缩"= CTS 特征; 若皮肤感觉也丢 → 更近端病变(前臂旋前圆肌 pronator teres 综合征)。<br>(神经_周围神经卡压治疗阶梯 §四)
足下垂 (foot drop) 三个来源——腓总神经 / L5 神经根 / 坐骨神经——如何一击区分? 关键三点是什么?	关键三点 = 足内翻保不保 + 有无背痛 + 感觉范围。<br>① 腓总神经(腓骨头): 足内翻 ✅ 保留(内翻归胫神经) + 无背痛 + 感觉仅小腿外侧/足背。钩子=跷腿/久蹲/腓骨颈骨折。<br>② L5 神经根: 足内翻 ❌ 也弱 + 背痛放射过膝 + SLR(直腿抬高)+ + L5 皮节(含踇趾蹼)。钩子=L4-L5 椎间盘突出。<br>③ 坐骨神经(近端): 足内翻 ❌ 弱 + 屈膝也弱 + 膝下大片麻(除内侧=隐神经) + 无背痛。钩子=髋后脱位/臀注射/骨盆骨折。<br>钥匙: 足内翻保留=腓总; 内翻弱+背痛=L5 根; 内翻弱+无背痛+屈膝弱+膝下大片麻=坐骨。<br>(神经_下肢神经定位速查 §二)
下肢口诀 PED / TIP——腓总神经 (peroneal) 与胫神经 (tibial) 各管什么动作? 损伤各是什么表现?	PED = Peroneal Everts and Dorsiflexes（腓总 = 外翻 + 背屈）→ 损伤 = foot dro-PED 足下垂（静息呈内翻+跖屈、失外翻/背屈、steppage gait 跨阈步态）。<br>TIP = Tibial Inverts and Plantarflexes（胫 = 内翻 + 跖屈）→ 损伤 = 站不了 TIP-toe（踮脚尖）、屈趾/足底感觉丢。<br>互补: 腓总管"抬"、胫管"踩"。→ foot drop 时足内翻保留 = 腓总（内翻归胫）。<br>(神经_下肢神经定位速查 §二)
手内在肌口诀——哪些归正中神经 (median)、哪些归尺神经 (ulnar)? LOAF / DAB / PAD 各指什么?	正中(返支)=大鱼际 LOAF: Lumbricals 桡侧 1/2 蚓状肌 + Opponens pollicis 拇对掌肌 + Abductor pollicis brevis 拇短展肌 + Flexor pollicis brevis 拇短屈肌浅头（功能 OAF = Oppose/Abduct/Flex）。<br>尺神经 = 其余手内在肌: 骨间肌 + 3/4 蚓状肌 + 小鱼际三肌 + 拇内收。骨间肌 DAB = Dorsals ABduct（背侧外展）/ PAD = Palmars ADduct（掌侧内收）。<br>钥匙: 拇精细外展/对掌 = 正中; 手指开合 = 尺。大鱼际萎缩 → 正中; 小鱼际萎缩 → 尺。<br>(神经_上肢神经定位速查 §一)
四种手畸形——尺爪手 / 握不成拳 / 正中爪手 / 誓言手——各对应哪条神经的近端还是远端? 规律是什么?	规律: 远端损伤 = 伸指/静息就爪（残留长屈肌还在拉）; 近端损伤 = 试握拳才现（长屈肌也瘫、静息反不爪 = 尺神经悖论）。<br>① Ulnar claw 尺爪手 = 远端尺(腕/Guyon)，4/5 指，静息。<br>② Trouble making a fist 握不成拳 = 近端尺(肘)，4/5 指，握拳时。<br>③ Median claw 正中爪手 = 远端正中(腕管/掌裂伤)，2/3 指，静息。<br>④ Hand of benediction 誓言手 = 近端正中(髁上/前臂)，2/3 指，握拳时。<br>尺累及 4/5 指、正中累及 2/3 指; 大鱼际萎缩 → 正中, 小鱼际萎缩 → 尺。<br>(神经_上肢神经定位速查 §二)
上肢针刺觉在这几处消失各是哪条神经? ① 虎口背侧 ② 食指掌侧指尖 ③ 小指 ④ 肩外侧军徽 ⑤ 前臂外侧	① 虎口背侧 = 桡 Radial（管背不管尖）<br>② 食指掌侧指尖 = 正中 Median<br>③ 小指 = 尺 Ulnar<br>④ 肩外侧军徽 (regimental badge) = 腋 Axillary<br>⑤ 前臂外侧 = 肌皮 Musculocutaneous<br>钥匙: 桡神经绝不给指腹 → wrist drop 的人指尖仍有感觉（排除正中的坑）。<br>(神经_感觉招牌点速记 §一)
下肢针刺觉在这几处消失各是哪条神经? ① 足底 ② 足背大部 ③ 第一趾蹼背 ④ 足外侧缘 ⑤ 小腿内侧/足内侧缘	① 足底 = 胫 Tibial（踩地面）<br>② 足背大部 = 浅腓 superficial peroneal<br>③ 第一趾蹼背(拇趾与2趾之间) = 深腓 deep peroneal<br>④ 足外侧缘 = 腓肠 sural<br>⑤ 小腿内侧/足内侧缘 = 隐神经 saphenous（股神经 femoral 的分支）<br>钥匙: 坐骨神经损伤时小腿内侧感觉保留（那块归股/隐，不归坐骨）。<br>(神经_感觉招牌点速记 §二)
上下肢感觉"镜像"规律——哪对神经管"背面+蹼+drop"、哪对管"抓握/踩地面"?	伸肌/背面神经（都会 drop）: 上肢 桡 Radial（手背+虎口背, wrist drop）↔ 下肢 腓总 peroneal（足背+第一趾蹼背, foot drop）。<br>屈肌/接触面神经: 上肢 正中 Median（手掌桡侧=抓握面）↔ 下肢 胫 Tibial（足底=踩地面）。<br>虎口背 ↔ 第一趾蹼背 完全镜像（都是伸肌神经给的背侧小蹼）。<br>(神经_感觉招牌点速记 §三)

### Q9. DIP 处理：减元凶优先，不是加抗胆碱 ❌

> [!danger] 错点
> 你选 C. Add benztropine；正解 = **B. Reduce or discontinue risperidone; consider switching to quetiapine or clozapine**。

**为什么 B 是首选？**

DIP（drug-induced parkinsonism）本质 = **抗精神病药阻断 nigrostriatal D2 → 类似 PD 临床表现**。因为 **元凶在体外（药）**，首选 = **撤药 / 减量 / 换低 D2 药**（quetiapine / clozapine / pimavanserin）。Benztropine（抗胆碱）只是**症状性辅助**，不解决根本——而且老年（70F）抗胆碱副作用风险高（谵妄 / 尿潴留 / 视力模糊 / 加重痴呆）。

**EPS 谱处理对照（关键）**：

| EPS 类型 | 触发 | 起病 | 首选治疗 | 抗胆碱角色 |
|---|---|---|---|---|
| **Acute dystonia** | 抗精神病药起始 / 加量 | 数小时-数天 | **抗胆碱 IM**（benztropine / diphenhydramine） | ⭐ 一线 |
| **Akathisia** | 抗精神病药起始 / 加量 | 数天-数周 | **propranolol**（β 阻断） | ❌ 无效 |
| **DIP（drug-induced parkinsonism）** | 长期抗精神病药 | 数周-数月 | **减/停元凶 + 换低 D2** | 二线辅助（老年慎用） |
| **TD（tardive dyskinesia）** | 慢性抗精神病药 | 数月-数年 | **停 / 换 clozapine + VMAT2** | ❌ **反加重** |

> [!tip] Memory Hook
> **"急性 EPS（dystonia/akathisia）= 救症状（benztropine/propranolol）；慢性 EPS（DIP/TD）= 救通路（减元凶/换低 D2/VMAT2）。"**
>
> 看 stem 给"长期用药 X 月" + 类 PD 三联 → 立刻锁 DIP → **减元凶**，不要被"抗胆碱治 EPS"反射拐走。

**5 个干扰项各踩什么坑**（USMLE 经典三连陷阱）：

| 选项 | 内容 | 坑 |
|---|---|---|
| A | "risk 3.07 倍" | ❌ **case-control 无分母 → 算 OR 不算 risk**（你这次踩这坑）|
| B | NHW odds = Black 3.07 倍 | ❌ **方向反**（"X compared to Y" → X 是暴露 / Y 是 reference）|
| C ⭐ | Black odds = NHW 3.07 倍 | ✅ |
| D | "race 是 causal factor" | ❌ **关联 ≠ 因果**（需 Bradford Hill 9 准则）|
| E | "CI 不含 1 → causality" | ❌ **CI 不含 1 = 统计显著关联，不是因果** |

**钉死 4 条铁律**：

```
① "X compared to Y" → X 是暴露，Y 是 reference
② Case-control 只能算 OR 不能算 risk（无分母 = total exposed）
③ CI 不含 1 = 统计显著关联（非偶然）≠ 因果
④ 因果需 Bradford Hill 9 准则 + 排除偏倚 + 排除混杂 + 排除反向因果
```

### Q21. PD psychosis 第 5 步：quetiapine 不是 ECT — 同主题第 3 次反复错 🔴 ❌

> [!danger] 错点（🔴 升级）
> 你选 E. ECT；正解 = **C. Quetiapine（或 pimavanserin / clozapine）**。

**同主题第 3 次反复错家族**：

| 次数 | 日期 | 题号 | 错点 |

| 1 | 早期 | Q12152 | 减药失败 → 你选其他，正解 quetiapine 🔴 |

| **3** | **2026-05-24 (今日)** | **Q21** | **第 5 步 你选 ECT，正解 quetiapine** |

**为什么 C 是首选，E 不是？**

PD psychosis 阶梯（钉死）：

```
① 排 infection/delirium/电解质（UTI / 脱水 / 低钠 最常见诱因）
       │ 排除
       ▼
② 排非 PD 精神活性药（BZD / 抗胆碱 / 阿片 / 镇静催眠）
       │ 排除
       ▼
③ 减 adjunctive DA 能药（顺序：amantadine → MAO-B → DA agonist 如 pramipexole）
       │ 仍幻觉
       ▼
④ 减 levodopa（注意运动症状代价）
       │ 运动失代偿 + 仍幻觉
       ▼
⑤ 上低/零 D2 抗精神病药 ⭐
   - Quetiapine（首选，便宜 + 经验多）
   - Pimavanserin（零 D2，最干净，贵）
   - Clozapine（最弱 D2 但要 ANC 监测）
   ⛔ 禁：haloperidol / risperidone / olanzapine（强 D2 → 加重 PD）
   ⛔ ECT 不是标准（last-resort，运动症状改善但精神病非主流路径）
```

**E（ECT）为什么是钓饵？**
- ECT 在 PD 中**对运动症状有帮助**（DA 释放↑）→ 但**对 PD psychosis 不是一线**
- 适应症窄：**严重抑郁伴 psychotic features / 紧张症** + PD → 才考虑 ECT
- 本题 stem 没给抑郁 / 紧张症 / 自杀 / 拒食 → 不进 ECT 适应症

|---|---|---|
| **Duloxetine（SNRI）** | + 抗抑郁双效 | 肝病 / 严重肾病 |
| **Pregabalin** | 起效快 / 镇静辅助睡眠 | 嗜睡 / 体重↑ / 老年跌倒 |
| **Gabapentin** | 便宜 / 安全谱广 | 同上 + 起效慢 |
| **TCA（amitriptyline / nortriptyline）** | 历史悠久 / 失眠双效 | **老年抗胆碱副作用**（避免老 ≥65 / BPH / 闭角青光眼 / 心律失常 / 直立低血压）|

**为什么 D 最完整**：USMLE 这种"D = any of the above"型选项存在时，**通常 D 就是答案**（出题人意图就是测你是否知道**多个 first-line 平等**而不是死记单一首选）。

**C 也不算大错**：duloxetine 是公认 first-line 之一，临床上确实经常首选（双效抗抑郁 / 一日一次依从性）。但 USMLE 答案优先选**最完整 / 最准确**的选项。

> [!tip] Memory Hook
> **"DPN 4 选 1 — 谁都能 first-line，选 D（all of above）。"** 看到"any of the above acceptable"型选项 → 优先考虑 D。

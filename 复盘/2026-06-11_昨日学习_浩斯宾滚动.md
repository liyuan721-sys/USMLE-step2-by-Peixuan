## §一 一句话总结（强制复习清单）

### 1.1 D+1 昨日新学（2026-06-10 · ENT/头颈/外科/免疫）

**错题（22）：**

- **Q2828 🔴 坏死性外耳道炎 (Necrotizing/Malignant Otitis Externa, NOE)**：老年糖尿病 + 夜间剧耳痛 + 耳道**肉芽** + 面瘫 = NOE，病原 **铜绿假单胞菌 (Pseudomonas aeruginosa)**（非曲霉 Aspergillus）。延伸：面瘫 = 颅底骨髓炎 (skull base osteomyelitis) 扩散红旗。
- **Q4282 🔴 NOE 治疗**：确诊后 = **IV 抗假单胞**（环丙沙星 ciprofloxacin / 头孢吡肟 cefepime），氨苄西林-舒巴坦 (ampicillin-sulbactam) 不覆盖铜绿。延伸：NOE 两次错——一次病原一次治疗，主线"NOE=铜绿"。
- **Q4599 腮腺肿物 (parotid mass) + 面瘫**：渐大无痛腮腺质硬肿物 + **面神经麻痹 (CN VII palsy)** = 恶性 (malignancy，神经侵犯)。延伸：腮腺良性肿物**不犯面神经**，面瘫=恶性红旗。
- **Q3449 🔴 梅尼埃病 (Ménière disease)**：眩晕 1–2h + 耳鸣 + **感音神经性聋 (SNHL)** + 无头位诱发 = 梅尼埃（内淋巴积水 endolymphatic hydrops），非良性阵发性位置性眩晕 (BPPV)。延伸：BPPV 听力正常、头位诱发、秒级。
- **Q108098 梅尼埃病 (Ménière) 生活方式**：确诊后一线 = **限盐 + 利尿剂 (diuretic，如 hydrochlorothiazide)**。延伸：今日同病第 2 道，反复把"短暂眩晕"误选 BPPV。
- **Q16433 外淋巴瘘 (perilymph fistula)**：**压力诱发**眩晕（Valsalva/举重/喷嚏）+ 外伤史 + **进行性听力下降**。延伸：诱因看性质——头位=BPPV，压力=瘘。
- **Q20314 耳廓血肿 (auricular hematoma)**：必须**先切开引流**再加压（预防菜花耳 cauliflower ear）。延伸：软骨无血供，血肿隔断软骨膜营养扩散 → 软骨坏死。
- **Q16961 鼻中隔血肿 (septal hematoma)**：软、波动 = 血肿 → **先切排**再填塞（同耳廓血肿法则）。延伸：硬=偏曲/骨折；今日"软骨无血供"盲点犯两次。
- **Q16127 牵涉性耳痛 → 头颈鳞癌 (HNSCC)**：耳镜正常的耳痛 + 烟酒 + 颈淋巴结 = 头颈鳞状细胞癌（牵涉痛），非中耳病。延伸：耳正常的耳痛要想到迷走/舌咽牵涉。
- **Q16201 头颈鳞癌 (HNSCC)**：扁桃体溃疡 + **固定无痛淋巴结** = HNSCC（非放线菌病 actinomycosis）。延伸：烟酒型 vs HPV 型（年轻、多性伴、预后较好）。
- **Q4544 急性化脓性腮腺炎 (acute bacterial/suppurative parotitis)**：术后脱水 + 口腔卫生差 → 唾液淤滞 → **金黄色葡萄球菌 (S. aureus) 逆行**；发热+WBC↑+红肿剧痛。延伸：预防 = 补液 + 口腔护理（治源头）。
- **Q11906 涎腺肿大 (sialadenosis)**：**双侧无痛**腮腺肿大 + 慢性酒精/糖尿病/暴食 = sialadenosis（非炎非瘤）。延伸：无痛+双侧 vs 化脓性的痛+发热。
- **Q18467 双膦酸盐相关颌骨坏死 (medication-related osteonecrosis of the jaw, MRONJ)**：长期双膦酸盐 (bisphosphonate) + 拔牙后**颌骨暴露不愈**。延伸：口腔无痛/隐匿不愈病变红旗谱。
- **Q2844 口腔白斑 (oral leukoplakia)**：不可刮除白斑最大危险因素 = **烟草 (tobacco)**（非糖尿病），癌前病变 → 活检。
- **Q15993 急性外耳道炎 (acute otitis externa, OE)**：游泳/挖耳 + 耳屏牵拉痛 = OE，局部抗生素滴耳。延伸：与 NOE 对照（OE 无骨侵犯无系统病）。
- **Q2838 呋塞米 (furosemide) 耳毒性**：对称高频**感音聋** + 反复心衰大剂量**袢利尿剂 (loop diuretic)** = 呋塞米耳毒（非环丙沙星 ciprofloxacin）。延伸：袢利尿剂/氨基糖苷 (aminoglycoside)/顺铂 (cisplatin) = 耳毒三巨头。
- **Q4201 非变应性鼻炎 (nonallergic rhinitis)**：刺激物触发 + **无眼痒** + 抗组胺 (antihistamine) 无效 → 鼻用激素 (intranasal corticosteroid)。延伸：有眼症状/季节性=变应性鼻炎 (allergic rhinitis)。
- **Q3428 鼻中隔穿孔 (septal perforation)**：鼻整形后呼吸**哨声**。
- **Q16302 分化型甲状腺癌 (differentiated thyroid carcinoma) 复发**：全切 + 放射碘 (RAI) 后 **甲状腺球蛋白 (Tg) 升高** = 复发标志物。延伸：Tg 是分化癌术后随访肿瘤标志物。
- **Q20700 血清病 (serum sickness)**：克罗恩病 (Crohn disease) 用英夫利昔单抗 (infliximab) 后**发热 + 荨麻疹 (urticaria) + 关节痛 + 蛋白尿**（5–14 天）= 血清病（III 型免疫复合物）。
- **Q4301 普通变异型免疫缺陷 (common variable immunodeficiency, CVID)（错选哮喘激发）**：成人反复鼻窦肺感染 → 查**定量 Ig**，非甲基胆碱 (methacholine) 激发。延伸：与下方 Q19993 做对形成"反射已建立"闭环。
- **Q3980 钙调磷酸酶抑制剂 (calcineurin inhibitor, CNI) 副作用** + **Q7494 移植后感染时间轴**：CYP3A4 抑制剂（地尔硫䓬 diltiazem/唑类抗真菌 azoles/红霉素 erythromycin/葡萄柚）→ 他克莫司 (tacrolimus) 血药↑ → 急性肾毒。

**做对巩固（14）：** 鳃裂囊肿 (branchial cleft cyst, Q16058)·颞下颌关节紊乱 (TMD, Q2194)·成人会厌炎 (epiglottitis)→颈侧位片(Q15961)·耳硬化症 (otosclerosis, Q16535)·甲状腺术后扩张性颈血肿→立即切开(Q16292)·气压伤鼓膜破裂 (barotrauma TM perforation, Q18418)·腭隆突 (torus palatinus, Q3427)·会厌炎插管失败→环甲膜切开 (cricothyrotomy, Q15962)·鼻咽癌 (nasopharyngeal carcinoma, Q12450)·X 连锁无丙球血症 (XLA, Q3195)·重症联合免疫缺陷 (SCID, Q4762)·普通变异型免疫缺陷 (CVID, Q19993)·过敏性休克 (anaphylaxis, Q17055)·慢性肉芽肿病 (CGD, Q22334)。
> 免疫缺陷四联对照已建：**XLA (X 连锁无丙球血症)**（婴儿·B↓·Ig↓全）/ **CVID (普通变异型免疫缺陷)**（成人·B 正常·Ig↓）/ **SCID (重症联合免疫缺陷)**（婴儿·T+B·FTT）/ **CGD (慢性肉芽肿病)**（吞噬细胞·过氧化氢酶阳性菌空洞+淋巴结炎）。⚠️ Q2842 阿司匹林加重性呼吸道疾病 (AERD / Samter 三联：哮喘+鼻息肉+NSAID 诱发) 标记"做对不确定"——要扎实。

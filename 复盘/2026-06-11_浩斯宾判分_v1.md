## 错题详解（4 道，全部反复错×2 → 已升 🔴）

### ✗ Q4（Q4418）原发流感肺炎，非 PCP

- 你选 **A 耶氏肺孢子菌肺炎（PCP）**（还自问"这人 HIV？"）；正确 **B 原发流感病毒性肺炎（primary influenza pneumonia）**。
- **你的疑问恰是钥匙**：stem 写"**既往健康** + 无 HIV"。**PCP 的前提是免疫抑制（HIV CD4<200 等）** —— 没有免疫缺陷背景就直接排除 PCP。
- **正解逻辑**：健康宿主 + 流感样前驱 3 天 + **骤起**高热肌痛干咳 + 双侧弥漫间质浸润 + 重度低氧（SpO2 88%）= 原发流感肺炎。区别**继发细菌性肺炎**（流感好转后再恶化、叶段实变、痰脓）。
- **反射**：**"双侧间质浸润" ≠ 一定 PCP**；先问"有没有免疫缺陷"，没有 → 先想病毒不想机会感染。

### ✗ Q6（Q3257）孕妇早期 Lyme → 阿莫西林

- 你选 **C 头孢曲松（ceftriaxone）**（带?）；正确 **B 阿莫西林（amoxicillin）**。
- **逻辑**：早期局限性 Lyme（游走性红斑 erythema migrans，无神经/心脏受累）一线本是**多西环素（doxycycline）**，但**孕妇 doxy 禁用（致畸）→ 换阿莫西林**。
- **ceftriaxone 的陷阱**：它**只用于晚期/播散性 Lyme**（神经 Lyme、Lyme 心脏炎致高度 AVB、关节炎口服失败）。本题无神经心脏受累，不上 IV。
- **反射**：**早期 Lyme → 口服（孕妇=阿莫西林）；ceftriaxone 留给"脑子和心脏"出事的人。**

### ✗ Q8（Q3013）IVDU 右心 IE → 经验万古霉素

- 你选 **C 头孢曲松（ceftriaxone）**；正确 **B 万古霉素（vancomycin）**。
- **逻辑**：静脉吸毒者（IVDU）+ **胸骨左缘收缩期杂音吸气增强（三尖瓣反流，Carvallo 征）** + 肺多发空洞结节（脓毒性肺栓子）= 右心 IE。IVDU 首位病原 **金黄色葡萄球菌（S. aureus），须覆盖 MRSA → 经验首选万古霉素**。
- **ceftriaxone 的陷阱**：**不覆盖 MRSA**（只覆链球菌/MSSA/HACEK），用于非 IVDU 自体瓣亚急性 IE（草绿色链球菌）。
- **专题笔记更全**：[[完整笔记/专题笔记/感染/感染_抗生素高频20场景秒杀]] 第 12 行 = **Vanc + Cefepime**（cefepime 加覆铜绿）；UWorld 单选最佳答案是万古，记**万古覆 MRSA 是核心**。
- **反射**：**针 + 吸气增强杂音 + 肺空洞 = 三尖瓣金葡，经验上万古。**

### ✗ Q25（Q3640）SCD 无脾脓毒症 → 肺炎链球菌

- 你选 **A 沙门菌（Salmonella）**（后改 B）；正确 **B 肺炎链球菌（S. pneumoniae）**。
- **逻辑**：镰状细胞病功能性无脾 + 脓毒症 → **无脾 = 荚膜菌脓毒症，首位肺炎链球菌**（其次 Hib、脑膜炎球菌）。
- **Salmonella 的陷阱**：它是 **SCD 骨髓炎（有骨痛时）的特征病原**。**同一个 SCD，问法不同病原不同**：问"脓毒症"=肺炎链球菌；问"骨髓炎"=金葡 + Salmonella 双覆盖。你被"SCD=Salmonella"口诀诱导，漏读了问的是脓毒症。
- **专题笔记**：[[完整笔记/专题笔记/血液/血液_镰刀细胞病SCD]] §零「同病三问法」已建完整对照。
- **反射**：**无脾发热 = 荚膜菌 = 肺炎链球菌；Salmonella 要等"骨头痛"才登场。**

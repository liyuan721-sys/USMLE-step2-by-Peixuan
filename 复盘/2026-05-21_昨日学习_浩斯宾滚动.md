## §一 一句话总结（强制复习清单）

### 1.1 D+1 昨日新学（2026-05-20，19 张卡）

- **Q15843 Bipolar I vs Brief Psychotic Disorder** —— 看到 delusion 先数 DIGFAST，凑齐 manic episode 就是 Bipolar I，psychotic 只是 specifier；诊断优先级 mood > 原发精神病。延伸：BPD 要求 psychosis 独立于 mood episode 且 <1 月。
- **Q11815 Synthetic Cathinone（bath salts）** —— 极端 combative + 自主神经飙 + 尿毒检阴性 + 精神症状拖数天–1 周 = 浴盐；vs MDMA（共情欣快 + 高热 + 短病程）。
- **Q16049 类固醇诱导精神病** —— 高剂量激素 + 急性偏执 + 认知完整 → 药源性精神病；first step = 减停致病药，不是加抗精神病药。
- **Q13573 UDS 假阳性** —— 意外阳性（尤其 PCP）先做 GC-MS 确证再行动；NSAID/DXM/抗组胺 → PCP 假阳性；医生 ≠ 执法者（保密义务）。
- **Q15791 Catatonia 紧张症** —— 凝视不动 + 不语 + 越推越顶（negativism）+ 学舌（echolalia）→ Catatonia；lorazepam challenge 诊断+治疗合一；避开抗精神病药。
- **Q15096 咖啡因中毒** —— 交感兴奋 + 明确咖啡因暴露 → substance-induced；焦虑诊断优先级：物质① → 躯体病② → 原发（排除性诊断）③。
- **Q4215 大麻中毒** —— 结膜充血（红眼）+ 口干 + 瞳孔正常 = 大麻；瞳孔决策树：针尖→阿片，散瞳→拟交感/致幻，正常+红眼→大麻。
- **Q11936 MI / Precontemplation** —— 不在乎的患者：用 MI 制造矛盾、把行为连到他在乎的价值；别讲后果 / 贴标签 / 站队 / 跳着开方案。
- **Q21169 MI / Evoking change talk** —— readiness ruler 打分后向下问"为什么不是 0"，别向上问"怎么到 5"；MI 永远挖理由，不讲后果。
- **Q4670 AAS（合成代谢类固醇）滥用** —— gynecomastia + 不育/睾丸萎缩 + 脱发痤疮 + roid rage = AAS；社会史里的诱饵物质 ≠ 答案。
- **Q20497 Venlafaxine 剂量依赖性高血压** —— venlafaxine 高剂量 → NE 再摄取抑制 → 血压↑ → 监测 BP；属 SNRI 类效应。
- **Q15098 Lamotrigine 皮疹** —— lamotrigine（双相抑郁相用药）致皮疹，见疹即停（可能是 SJS/TEN 前兆）；合用 valproate / 加量过快放大风险。
- **Q16066 TD & EPS 四兄弟** —— 抗精神病药异常运动看时间线：dystonia（时-天）→抗胆碱；akathisia（天-周）→β阻滞；parkinsonism（周-月）→减量；TD（月-年）→停药/VMAT2。benztropine 会加重 TD。
- **Q15145 MS 最常见精神并发症** —— 年轻女性 + 神经症状时空多发 + MRI 胼胝体 → MS；最常见精神并发症 = depression（~2/3），不是 dementia。
- **Q19477 老年认知下降先查听力** —— 老人健忘+退缩+说话大声+反复问 + MoCA 正常 → 先查听力（Presbycusis）；老年认知/功能下降先排可逆病因（听视抑药甲 B12）。
- **Q2517 自杀最强风险因子** —— 既往自杀未遂 = 单一最强预测因子（past behavior predicts future）；NSSI（cutting）≠ suicide attempt。
- **Q2360 自杀处置 assess before act** —— 抑郁伴 SI 但无 plan/intent → 先做自杀风险评估，再决定处置；住院门槛 = ideation + plan + intent 三者齐全。
- **Q11903 神经痛共病抑郁** —— DPN 共病抑郁 → SNRI（duloxetine）一药两治，关键在 NE 成分；SSRI 缺 NE 治不了神经痛。
- **Q15849 双相抑郁用药** —— 双相抑郁相一线 = 二代抗精神病药（quetiapine/lurasidone 等）；选药看"当前相"非既往史；抗抑郁药禁单用（诱发转相）。

### 1.2 滚动复习

- **D+3（2026-05-18）· Q4469 AF 抗凝指征** —— 非瓣膜性房颤抗凝看 CHA₂DS₂-VASc 评分；瓣膜性 AF（中重度 MS / 机械瓣）直接抗凝、不算分。
- **D+7（2026-05-14）· Q17169 Adrenal Crisis vs Anaphylaxis** —— 低血压对补液/升压药反应差 + 激素停用/应激史 → adrenal crisis，立即 hydrocortisone；anaphylaxis 有荨麻疹/喘鸣、给 epinephrine。
- **D+14（2026-05-07）/ D+30（2026-04-21）** —— 无对应日期节，名额转入弹性补位。

### 1.3 必复

**🔴 反复错（扫全本，按主题归并）：**

- **AS 体征 / S3 vs S4 机制（Q4277，反复错 ≥3 次）** —— 压力超负荷→S4，容量超负荷→S3；severe AS：迟弱脉、杂音晚峰、S2 减弱/单一。
- **TOF / 紫绀型先心病（Q2429）** —— Tet spell：膝胸位、给氧、吗啡、补液、phenylephrine（α1 激动剂↑SVR）——不是 α1 拮抗剂。
- **Vaughan-Williams 抗心律失常药分类** —— 4 大类机制 + ECG 效应双向背（药↔ECG 改变）。
- **CAST 试验 / Class IC 禁忌** —— 结构性心脏病 / MI 后禁用 IC（致心律失常、死亡率↑）。
- **ADHF 初始管理（Q6945）** —— 干湿冷暖 4 象限；湿暖 → 利尿 + 扩血管；缺氧/呼吸窘迫上 NIPPV。
- **Mobitz I vs II（Q3766）** —— I 型 PR 渐长后脱落（良性）；II 型 PR 固定突然脱落（→ 起搏器）。
- **胸痛鉴别 / Costochondritis（Q4650）** —— 可复制的胸壁压痛 → costochondritis（排除心源性后）。
- **新生儿先心病 / 杂音鉴别** —— innocent（Still）murmur vs pathologic 的时序与特征。
- **MDMA 中毒 + Serotonin Syndrome（Q11853）** —— 病因要答 substance（MDMA），不能只答 SS；高热 + 反射亢进 + 肌阵挛 + 低钠。
- **围产期 GAD（Q21090）** —— 孕期焦虑治疗 CBT 一线，避 paroxetine / BZD。
- **MI 沟通"讲后果"反射（Q11936 / Q21169）** —— MI 题见"讲后果 / 威胁"选项直接排除。

## §三 错题深挖

### ❌ Q2 DPN First-Line — 反复错（5-24 Q19 同题再错）⚠️

> [!danger] 反复错升级
> **5-24 Q19** 同主题已错过一次 → **今天再错** = 第 2 次同主题错 → **升 🔴 反复错**

**根本原因**：**pattern 反射**选最熟悉的单一选项（C duloxetine 临床常首选）— 没读 D 选项就锁定。这是**典型"USMLE 答案选项策略"陷阱**。

**普适规则 — DPN first-line 4 选 1 平等**（ADA/AAN 2024）：

| 药 | 类 | 慎用 |
|---|---|---|
| **Duloxetine** | SNRI | 肝病 / 严重肾病 |
| **Pregabalin** | gabapentinoid | 嗜睡 / 体重↑ / 老年跌倒 |
| **Gabapentin** | gabapentinoid | 起效慢需滴定 |
| **TCA (amitriptyline / nortriptyline)** | TCA | **老年抗胆碱副作用** ⚠️ |

**USMLE 选项策略钉死**：

| 选项形态 | 应对 |
|---|---|
| **"D = Any of the above"** ⭐ | **存在时优先考虑 D**，除非题干给特定患者特征指向单一首选 |
| "All of the above" | 通常错（USMLE 很少给 all of above）|
| "None of the above" | 通常错 |
| 同质 3-4 个 + 1 异类 | 异类常是答案 |
| 同义重复 2 个 | 两个都不是答案 |

**Memory Hook**：**"DPN 4 选 1 平等：duloxetine / pregabalin / gabapentin / TCA — 谁都能 first-line；见 'D = any of above' 不要急选熟悉单选"**

**类比**：菜单上"汤"+"沙拉"+"前菜"+"任一前述"——服务员问你要哪个 = 测你知不知道四个都算前菜（D），不是测你最熟哪个单品。

**反射钩升级**：见到"first-line treatment for [chronic neuropathic pain syndrome]" + 选项 4 个具体药 + 一个 "any of the above" → **0.5 秒选 D**，不读单选。

**状态升级**：[[mistakes/uworld-mistakes_2026-05#^Q19-2026-05-24]] 🟡 → **🔴 反复错** ⚠️

### ❌ Q12 SMART Goal Premature — 新错点

**根本原因**：**误以为"daily tracking log = 好习惯 = 答案"** → 没识别 tracking 是 **SMART goal 建立后的 implementation step**，不是 first step。

**普适规则 — Lifestyle Modification 时序**：

```
First step = Set SMART goal via Motivational Interviewing ⭐
        │
        ▼
② Implementation tools（在 SMART goal 建立后才合理）：
   • Daily tracking log（C 选项 — 实际是这一步）
   • Pill organizer（药物 adherence）
   • Automated reminders
   • Schedule integration
        │
        ▼
③ Confidence rating（MI — E 选项，在 SMART goal 后才用）
        │
        ▼
④ Specific dietary / exercise prescription（D 选项 — 在 SMART goal 后才合理）
        │
        ▼
⑤ Reassess at 1 month → 升级或调整
```

**反射钩**：见到"motivated patient + vague goals / overly ambitious plan + first counseling approach" → **0.5 秒选 set SMART goal**（不是 testing / 不是 tracking / 不是 specific Rx / 不是 confidence rating）

**Memory Hook**：**"Lifestyle counseling first step = set SMART goal；tracking / confidence rating / specific Rx 都是 SMART goal 后的事"**

**类比**：减肥 app 不会让你"第一天就每天记体重"（C），而是先"我们定个本月小目标"（B SMART goal）— 没目标的 tracking 是无锚漂流。

**状态**：新 🟡（首次错）；下次再错 → 升 🔴 + #难记

### ❌ Q20 PD ψ Step 5 — 5-24 三巨头错点漂移 ⚠️

> [!danger] 5-24 三巨头反复错追踪
> - **5-23**：选 E ECT → 错
> - **5-25 浩斯宾 self-test**：答 C quetiapine ✓（第 1 次连对）
> - **今天 5-26**：选 **A discontinue carbidopa-levodopa** → **错（错点漂移）** ⚠️
>
> 计数重置 → **维持 🔴**（未连对 2 次）

**根本原因**：**误以为"levodopa 是 DA 能药 → 也属于可减的 PD 药"** → 没识别 levodopa 是 PD foundation drug，**不能轻易停**（motor 严重恶化 + NMS-like 风险）。

**普适规则 — PD ψ 5 步阶梯钉死**（"哪个药要减"的顺序）：

```
PD psychosis 出现 → next step?
        │
        ▼
① 排除谵妄 / 感染 / metabolic（先排可逆病因）
        │
        ▼
② 减 anticholinergic（如 trihexyphenidyl / benztropine）
        │ 仍 ψ
        ▼
③ 减 amantadine
        │ 仍 ψ
        ▼
④ 减 DA agonist（pramipexole / ropinirole）
        │ 仍 ψ
        ▼
⑤ Levodopa 是 foundation — **不减 / 保留**；加 **quetiapine 或 pimavanserin**（低 D2 SGA）⭐
```

**关键反射**：**"Levodopa 是 PD 唯一不能随便减的药"** — 减 levodopa 会引发：
- 严重 motor deterioration（rigidity / akinesia）
- **NMS-like syndrome**（抗精神病药综合征样的高热 + 肌强直）
- Patient 生活质量崩塌

**禁忌反射**：
- **不**用 **haloperidol** / typical antipsychotic（高 D2 → 加重 PD motor）
- **不**用 **risperidone**（高 D2，同上加重 motor）
- **不**停 levodopa（NMS-like + motor 崩）
- **不**反射 ECT（不是 PD ψ 一线）

**正确的 SGA 4 兄弟（PD ψ 用）**：
- **Quetiapine** ⭐（最低 D2 + 评估首选）
- **Pimavanserin** ⭐（5-HT2A 拮抗剂 + FDA 批 PD ψ + 完全不碰 D2）
- **Clozapine**（次选，但需 ANC 监测）
- **Olanzapine** ❌（中等 D2，加重 PD motor）

**Memory Hook**：**"PD ψ Step 5 = quetiapine / pimavanserin；levodopa 是地基不能拆，haloperidol/risperidone 是毒药"**

**类比**：盖房子拆装修可以（anticholinergic / amantadine / DA agonist 都是装饰层），地基（levodopa）拆了房子塌（motor 崩 + NMS-like）。墙裂了（ψ）→ 加梁（quetiapine），不是拆地基。

**入库**：[[mistakes/uworld-mistakes_2026-05#^Q12152]] 维持 🔴 反复错；**错点漂移**（5-23 选 ECT / 5-26 选停 levodopa — 两端钓饵都踩，更印证 quetiapine 是唯一答）；状态升级 = 计数清零 + 标 #考前必看

**4 大高热综合征解药矩阵（钉死参考）**：

| 综合征 | 解药 | 一句话机制 |
|---|---|---|

### ❌ Q23 DOAC + tPA — 知识缺口（你标 A? 自觉不确定）✓ 元认知正确

**根本原因**：**DOAC + acute stroke 处理流程**不熟。你标 A?（带问号）说明已经意识到 unsure — 元认知 OK，但反射停在 A 不对。

**普适规则 — Acute Stroke + DOAC 用户 SOP**（AHA/ASA 2019 + 2024 update）：

```
Acute ischemic stroke + last DOAC dose < 48 hr
        │
        ▼
① CT 排除 hemorrhage（OK if 无）
        │
        ▼
② 评估 tPA candidacy
        │
        ├─ Direct Xa inhibitor (rivaroxaban / apixaban / edoxaban)
        │   ├─ Last dose < 48 hr + normal renal → **tPA 禁忌** ⚠️
        │   └─ 特殊情况：normal aPTT + PT + ecarin clotting time + thrombin time + 特殊 anti-Xa level < 检测下限 → 极个别中心考虑
        │
        ├─ Direct thrombin inhibitor (dabigatran)
        │   ├─ Last dose < 48 hr → tPA 禁忌
        │   └─ **Idarucizumab reversal** → 后可考虑 tPA（dabigatran 唯一有 FDA 批的逆转通路）
        │
        ▼
③ Last DOAC < 48 hr 且 tPA 禁忌 → **首选 Mechanical Thrombectomy if LVO**（large vessel occlusion）⭐
        │
        ▼
④ No LVO + tPA 禁忌 → 仅 supportive + 抗血小板 + 二级预防（非急性溶栓）
```

**关键禁忌钉死**：

| Anticoagulant | < 48h 末次剂量后 tPA |
|---|---|
| Warfarin INR > 1.7 | ❌ 禁忌 |
| **Direct Xa (rivaroxaban / apixaban / edoxaban)** | ❌ **禁忌**（无可靠 reversal 给 tPA） |
| Dabigatran | ❌ 禁忌（除非 idarucizumab 已 reverse） |
| LMWH 治疗剂量 < 24h | ❌ 禁忌 |
| Heparin（PTT 正常） | ✓ OK |

**反射钩**：
- 见到 "rivaroxaban / apixaban + 急性 stroke + last dose < 48 hr" → **0.5 秒喊"tPA 禁忌；走 thrombectomy if LVO"**
- 不反射"tPA 是急性 stroke 第一线" — 这是**有 anticoagulant 时的反禁忌**

**陷阱钩**：
- B "Andexanet alfa reversal → tPA" ❌ — **不推荐**这条路径（andexanet 增加血栓 paradox + ASA/AHA 没批此用法）
- E "PCC then tPA" ❌ — 同样无证据，反而增加血栓 risk
- D "Aspirin + observe" ❌ — under-treatment for acute LVO stroke
- **C ⭐**：Withhold tPA + mechanical thrombectomy if LVO 是 DOAC 用户 acute stroke 的**正解**

**Memory Hook**：**"DOAC < 48 hr + acute stroke = tPA 禁忌 → 走 thrombectomy if LVO；Andexanet/PCC 不开溶栓的路"**

**类比**：rivaroxaban 像在血里撒了"防凝剂残留"— 你再给 tPA = 在油桶旁点火（出血风险爆增）。绕开 = 用机械工具捞掉血栓（thrombectomy），不点火。

**状态**：新 🟡（首次接触此考点 — 你之前没做过这个题）；建议 1 周后重做

## §一 一句话总结（强制被动复习清单）⭐

> [!tip] 用法

### 1.1 D+1 昨日新学（2026-05-30 肾脏/酸碱/电解质）

**做错的（重点）：**
- **GU 外伤影像顺序**（Q3349/Q3358）：骨盆骨折 + 尿道损伤 → **逆行尿道造影(RUG)先于导尿**；钝性肾损伤(flank+血尿) → **CT abdomen/pelvis**。
- **钙草酸结石防复发**（Q3895）：首选**降钠饮食**（不是降钙）—— 限钠减尿钙排泄。
- **代酸肾代偿主力**（Q13842）：肾排 H⁺ 主要靠**尿铵(NH4⁺)↑**（+ titratable acid + HCO3 重吸收）。
- **ABG 判读**（Q2786）：肺炎脓毒症 pH7.49 + PaCO2↓ + HCO3 正常 = **未代偿呼吸性碱中毒**（HCO3 还没来得及降）。
- **Type 1 远端 RTA**（Q18881/Q4828）：NAGMA + 低 K + **尿 pH>5.5** = 远端泌 H⁺ 缺陷（Sjögren 成人 / 婴儿 FTT）。
- **肝硬化低钠**（Q106381）：Na130 非重度 → 不需特殊治疗；**噻嗪会加重低钠**（陷阱）。
- **肝硬化袢利尿**（Q2826）：→ 代谢性碱中毒（HCO3↑ + 低 K 低 Cl）。

**做对但不确定（要扎实）：**
- **HRS 诊断下一步**（Q2219）：肝硬化 AKI → 先**测尿钠**（<10 提示 HRS / 肾前性）。
- **代碱分型**（Q4866）：呕吐 + 低 K 低 Cl + **尿 Cl<20 = saline-responsive → 补 NS**；>20 = resistant(Conn 等)。
- **药物性 SIADH**（Q2180）：等容量低钠 + 低尿酸 + SSRI。
- **VUR 分级**（Q106084）：钙盏变钝 = severe(III-IV) → 预防性抗生素。
- **advanced CKD 虚弱老人**（Q14465）：体重↓ → watchful waiting + 营养支持（非激进透析）。

**自信做对（反射已建，扫过即可）：** 氨基糖苷肾毒性=ATN(Q4379) / 酸碱金标准=pH+PaCO2(Q2787) / Calciphylaxis=高磷(Q15372) / 老男吸烟无痛血尿→膀胱镜(Q16045) / 慢性高钾先 review 用药(Q2167) / 高钾急症已给钙→insulin+glucose(Q4422)。

### 1.2 滚动复习

- **D+3 (5-28)**：肾脏大节 —— Transplant RAS（ACEI 后 Cr↑指纹）/ HRS / Lithium-NDI（缺陷在集合管不是下丘脑）/ Analgesic nephropathy / AIN（嗜酸+WBC casts）/ Mixed acid-base（Winters）/ 婴儿低容量高钠先 NS。
- **D+7 (5-24)**：Ethics 沟通 —— **DNR ≠ Do Not Treat**（DNR 仅适用心肺骤停，不阻止溶栓等 QoL 干预）；IPV 筛查 / 跨文化先了解 patient beliefs。
- **D+14 (5-17)**：心内 —— **PPCM**（妊娠晚期/产后新发心衰）；机械瓣新发杂音→PVD；β-blocker 中毒→**Glucagon**。
- **D+30 (5-01)**：无日期节。

### 1.3 必复（🔴 + #考前必看 + #难记 轮换）

- 🔴 **Fall 防跌**（Q107153/4）：循证二级预防 = **监督下力量+平衡训练 / 优化骨密度**；**call/bed alarm 非 evidence-based**（钓饵）。
- 🔴 **DLB ψ Step 5**（Q12152）：幻觉/认知波动**早于 levodopa** = DLB → **rivastigmine 一线**，抗精神病药相对禁忌。〔今日 #难记 轮值〕
- 🔴 **急性激越**（Q15856）：ED/acute/immediate/IM = **olanzapine/haloperidol ± lorazepam IM**；心境稳定剂/抗抑郁全是干扰（起效慢）。
- 🔴 **DPN 一线**（Q2/Q19）：**duloxetine / pregabalin**（"any of the above"选项暗号）。
- #考前必看 **RCA vs FMEA**（Q22581）：RCA = **既往**事件根本原因（回顾）；FMEA = **上线前前瞻**识别失效模式（severity-likelihood-detectability）。

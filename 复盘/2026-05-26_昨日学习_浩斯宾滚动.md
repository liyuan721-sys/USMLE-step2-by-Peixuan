### 1.1 D+1 昨日新学（2026-05-25）— 12 张卡

**浩斯宾自测错点（3 张）**：

- [ ] **Q3 BED 一线用药** — BED 一线 = **lisdexamfetamine** 或 **topiramate**（FDA 批 BED）；fluoxetine 是 **BN** 一线；AN 没药（CBT + refeeding + 监护）。看 BED + 选项有 fluoxetine vs lisdexamfetamine → **选 lisdexamfetamine**。
- [ ] **Q11 Down syndrome 确诊** — FISH 24-48 hr 快但**不能区分 free trisomy vs translocation**；**Karyotype 唯一能 confirm**（识别 t(14;21) → 父母 carrier 时复发风险 ~10-15%）；Down 3 机制：free trisomy 95% / Robertsonian 3-4% / mosaic 1-2%。
- [ ] **Q10 Tourette α2 失败下一步** — Tourette 阶梯：CBIT → α2-agonist（**clonidine = guanfacine 同类**）→ antidopaminergic（risperidone / aripiprazole / VMAT2）→ DBS。Stem "failed guanfacine" = α2 整类失败 → **跳到 antidopaminergic**，不换同类 clonidine。

**Ethics 系统学习（9 张）**：

- [ ] **Q21141 Hospice eligibility** — Hospice = life expectancy ≤ 6 mo + 放弃 curative，**可保留 palliative**（symptom control 仍做）；非"什么都不做"。
- [ ] **Q19553 FMEA (Failure Mode and Effects Analysis)** — **Prospective** 工具，**新流程上线前**用，识别 future vulnerabilities；vs RCA = retrospective（已发生 sentinel event 后查 underlying causes）。
- [ ] **Q106239 Surgical time-out** — WHO Universal Protocol，开刀前全队暂停核对 patient/site/procedure → 减少 **retained surgical objects** + wrong-site surgery；属系统层 forcing function。
- [ ] **Q21477 Swiss Cheese Model** — 多层独立 barriers 重叠（教育 + 流程 + 技术 + 监督）才挡住 error；单一 layer 都有 holes，**overlapping multiple barriers** 是核心 patient safety 理念。
- [ ] **Q22581 RCA (Root Cause Analysis)** — **Retrospective** 工具，已发生 sentinel/near-miss event 后 identify **underlying causes of past events**（不是 future vulnerabilities — 那是 FMEA）；重点是 system 层而非 blame individual。
- [ ] **Q20431 Home Health Care 适应症** — Medicare 报销 3 条件：**medically stable + homebound + skilled needs**（skilled nursing / PT / OT / speech / wound care）；vs SNF = post-acute rehab；vs hospice = terminal。
- [ ] **Q107185 Geriatric Prescribing — 减 ADE** — 老年 polypharmacy 防 ADE 核心 = **limit number of prescribers**（single coordinating physician 看全部药）；Beers criteria + 定期 deprescribing review。
- [ ] **Q20229 Hospital Readmission Reduction** — 减 30-day readmission：**telephone follow-up within 48-72 hr**（address loss to follow-up）+ med reconciliation + scheduled PCP visit within 7 d。
- [ ] **Q22061 减 Unnecessary Imaging Request** — Specialist 要 imaging：**Specific exam + clinical commentary 指明 specific concern**（"r/o cholecystitis" 不是"abdominal pain workup"）→ radiologist 给针对性 read，减不必要 imaging。

### 1.2 滚动复习（D+3 / D+7 / D+14）

- [ ] **D+3 (5-23) HIPAA verbal consent (Q3614)** — 转诊 / release info：**verbal consent + minimum necessary disclosure** 即可（不需 written 在 routine clinical context）；Sensitive 信息（mental health / SUD / HIV / abortion / genetic）default 不释放 — 需 specific authorization。
- [ ] **D+7 (5-19) Seasonal Affective Disorder (Q15074)** — 秋冬 onset + atypical features（hypersomnia / hyperphagia / 体重↑）+ 春夏 remit；一线 = **bright light therapy 10,000 lux 30 min 每早**；二线 SSRI；预防 = 秋初 prophylactic light therapy。
- [ ] **D+14 (5-12) ROC 最佳 cutoff (Q107800)** — ROC = Y 轴 sensitivity vs X 轴 (1 − specificity)；**最佳 cutoff = 距左上角最近的点**（max sens + spec sum）；AUC 越大测试越优（0.5 = 无价值 / 1.0 = perfect）。

### 1.3 必复（🔴 + #考前必看 + #难记 轮值）

**5-24 三巨头 🔴 反复错**（5-25 浩斯宾各连对 1 次，**再 1 次可降 🟡**）：

- [ ] **🔴 OR/CI/因果三连陷阱** — Adjusted OR with CI **excluding 1** = statistically significant association after adjusting confounders；**residual confounding** 仍可能 ≠ 真因果；"reverse causation" 需考虑（outcome 是否反推因子）。
- [ ] **🔴 PD psychosis Step 5 用药** — PD ψ 5 步阶梯：① 排谵妄/感染 → ② 减 anticholinergic → ③ 减 amantadine → ④ 减 DA agonist → ⑤ 仍幻觉 → **quetiapine 或 pimavanserin**（**不是 ECT 不是 haloperidol 不是 risperidone**——haloperidol 会加重 PD motor）。
- [ ] **🔴 Atypical MDD MAOI** — Atypical MDD = mood reactivity + hyperphagia + hypersomnia + leaden paralysis + rejection sensitivity；**首选 MAOI（phenelzine）或 SSRI**；**不是 lithium**（lithium 是 BP / TRD augmentation）。

**#难记 轮值**：

- [ ] **Q11853 MDMA + Serotonin Syndrome** — MDMA (ecstasy) → 释放 5-HT + DA → hyperthermia + clonus + autonomic instability + altered mental status；**SSRI 加 MDMA = SS 暴发风险**；Tx = **cooling + benzodiazepines + cyproheptadine（5-HT2 antagonist）**，**不是 dantrolene**（dantrolene 是 NMS / 恶性高热）。

## §一 一句话总结（被动复习清单——扫一遍即可）

### 1.1 D+1 昨日新学（2026-06-04，6 张）

1. **[[mistakes/uworld-mistakes_2026-06#^Q15829]] 婴儿 UTI 经验药** → 首选**第三代头孢（cefixime）**；**儿童禁用 fluoroquinolone（软骨损伤）**，cipro 只留给 Pseudomonas/留置导尿。
2. **[[mistakes/uworld-mistakes_2026-06#^Q4615]] 术后急性尿潴留（AUR）** → 老年男 + BPH + 术后 + 耻骨上压痛 + 躁动 → 先 **bladder US（≥300 mL）**；**WBC 10,000 正常 + 无热 + 术口清洁 = 反对感染**，别冲血培养。
3. **[[mistakes/uworld-mistakes_2026-06#^Q4196]] 成人 UTI nitrites 判读** → **nitrites 阳性特异指向肠杆菌（E. coli）**；STI 不产 nitrites 且有 CMT + 黏液脓性分泌物（本例清亮、无 CMT → 排除）。
4. **[[mistakes/uworld-mistakes_2026-06#^Q4749]] 梗阻性尿路病 + 梗阻后利尿** → 腰痛 + **尿少↔偶发大量尿交替** = 梗阻 + 梗阻后利尿（失钾→乏力）；**独肾**让单侧梗阻就 Cr↑；"lisinopril + Cr↑"是 RAS 烟雾弹（RAS 不痛、不改尿量）。
5. **[[mistakes/uworld-mistakes_2026-06#^Q4683]] 横纹肌溶解 → AKI** → found down + 花斑 + **K↑ PO4↑ Ca↓ + AST>ALT** = 横纹肌；myoglobin → AKI；**Na 正常可直接排除 ODS**；治疗大量补液。
6. **[[mistakes/uworld-mistakes_2026-06#^Q11109]] 输尿管结石 MET** → **≤5 mm 保守 / 5–10 mm tamsulosin(α1) / >10 mm 外科**；只有 α1 阻滞剂促排石。

### 1.3 必复（强制接触）

- **🔴 Swiss Cheese Model（Q21477）**：多个 barrier **同时叠加（simultaneously）**拦截错误 = 瑞士奶酪模型；区别于 PDSA（迭代）/ Hard stop / Automation。
- **🔴 Home Health Care vs SNF（Q20431）**：能在家 + 需间断技术性护理 → **Home Health**；需 24h 技术性护理/康复 → SNF。
- **#考前必看 肾脏_酸碱实战速判卡**：ABG 判读三步——① 看 pH 定酸碱 → ② 看原发（PaCO2/HCO3）→ ③ 算代偿（Winter's: 预期 PaCO2 = 1.5×HCO3 + 8±2），偏离即 mixed。
- **#难记 USMLE_实验室指纹诊断大全（§6.4 横纹肌）**：横纹肌四联 **K↑ / PO4↑ / Ca↓ / AST>ALT** + CK↑↑↑ + 尿潜血阳性镜下无 RBC（昨日 Q4683 已挂钩）。

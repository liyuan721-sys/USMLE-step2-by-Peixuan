## §一 一句话总结（被动复习强制清单）

### 1.1 D+1 昨日（2026-05-22）新学 — 20 张全覆盖

1. **Q3380 Selective Mutism** — 焦虑障碍家族；**特定社交场合（典型=学校）持续 ≥1 月不说话**，舒适场合言语正常；一线 = CBT，严重再 SSRI（fluoxetine）。
2. **Q12203 NMS** — 抗精神病药 + **高热 / 铅管样僵直 / 意识改变 / 自主神经不稳** 四联；机制 = 中枢 D2 阻断；白细胞↑/CK↑/肌红蛋白尿 = NMS **诊断拼图**，不是另诊断证据。
3. **Q15856 急性躁动药物管理** — Safety risk + verbal de-escalation 失败 → **IM lorazepam ± haloperidol**（分钟级控制行为）；冷静后再 workup + 长期治疗。🔴 反复错。
4. **Q4895 急性精神病初始抗精神病药** — **配合患者 → PO 治本（二代 SGA）**；LAI 真正适应症 = chronic nonadherence；住院仅一次 ≠ nonadherence。
5. **Q17716 Brief Psychotic Disorder** — 应激后 **1d–<1 月** 精神病性症状 ≥1 项，最终完全回基线；可标 "with marked stressor"。
6. **Q16064 心理治疗五选** — Chronic schizophrenia + 家人同住 → **Family therapy** 降 high expressed emotion → 减复发；治疗选择 = "改什么 + 治什么病"，不是"哪个最高级"。
7. **Q15017 新发精神病 workup 顺序** — 年轻 + 急性 + 无显著病史 → **urine tox screen 性价比最高第一步**（揪 cocaine/amphetamine 等 stimulant-induced psychosis）；New-onset psychosis 是排除性诊断。
8. **Q12468 Valproate 肝毒性** — VPA 头 **6 个月** 内药物性肝损伤；前驱 = malaise/乏力/嗜睡/食欲↓/恶心；定期 LFT；其它招牌副作用 = 震颤/血小板↓/脱发/体重↑/胰腺炎/高氨血症脑病/神经管缺陷。
9. **Q2518 Tardive Dyskinesia 处理** — 长期 D2 阻断 → 高动力性运动障碍口面部典型；**处理 = 换 SGA + VMAT2 抑制剂 (valbenazine/deutetrabenazine)**；**抗胆碱能对 TD 无效甚至加重**（与 acute dystonia 相反）。
10. **Park1 Drug-Induced Parkinsonism 处理** — 抗精神病药 D2 阻断致 DIP（数周-数月、对称、有用药史）；阶梯 = **减量 → 换低 D2 SGA（quetiapine/clozapine）→ amantadine/benztropine**；**levodopa 在 DIP 中禁**（恶化精神病、突破不了阻断）。
11. **Park2 IPD vs DIP** — IPD = **不对称 + 缓慢 + 无用药史 + levodopa 反应良好**；TRAP 四联 = Tremor at rest / Rigidity / Akinesia-bradykinesia / Postural instability。
12. **Q2353 AN 住院 + Euthyroid Sick Syndrome** — AN 触住院指征即收（**HR<40 / BP<80/60 / 体温<35 / 电解质异常 / 极低 BMI**），优先于纠正单项异常；低 T3/T4 + TSH 轻升 = ESS，喂饱即愈，**补 levothyroxine 有害**。
13. **Q14990 bvFTD vs DLB** — bvFTD = 额颞退化（Pick/tau）→ **行为/人格首发，早期记忆相对保留**；DLB = 视幻觉 + parkinsonism + REM 行为障碍 + 波动性认知。
14. **Q21259 Adjustment Disorder** — 可识别应激源后 **3 个月内**起病情绪/行为症状，痛苦/功能受损但**不满足任何完整 DSM-5 诊断**（兜底）；本题 < 2 周 + 症状不达 MDD 数 = AD。
15. **Q2552 Pseudocyesis 假孕** — 长期不孕 + 主观早孕症状齐 + **诊室 hCG- + 超声空子宫** = 非妊娠者**非妄想性**确信怀孕（躯体化）；处理 = 精神科评估。
16. **Q3719 Conversion vs Somatic Symptom Disorder** — Conversion = **真实神经缺失 + 与神经解剖矛盾**（能站起来却抬不起腿）+ 紧跟应激；Somatic SS = 真实躯体症状 + 过度焦虑/重视；看"症状本身讲不讲得通"。
17. **Q15047 Reactive Attachment Disorder** — 5 岁前 insufficient care → 依恋破坏 → **情感/社交退缩 + 不寻求或不响应安抚 + 持续负性**；与 DSED 同源镜像（RAD 退缩 / DSED 过度亲近）；治疗 = 稳定响应性养育环境 + 心理服务，**非药物**。
18. **Q16164 PD psychosis 管理** — 治疗副产物，多巴胺↑激动中脑边缘 D2 → 视幻觉为主；**5 步流程 = 排除 infection/delirium/dementia → 监测 → 减非 PD 精神活性药 → 减 PD 药 → 低/零 D2 抗精神病药**（pimavanserin/quetiapine）；运动症状轻 = 有减药余地。
19. **Q16310 Cluster B 人格障碍鉴别** — **HPD = Attention 戏剧热 / NPD = Admiration 自大冷 + 缺共情 / BPD = avoid Abandonment 自残 splitting 空虚 / ASPD = 漠视权利 + ≥18 岁 + 15 岁前 conduct disorder**；HPD vs NPD 看情绪温度。
20. **Q107408 PMT 适应症 + ADHD 跨场所标准** — **行为问题局限单一场所（仅家中）≠ ADHD**；ADHD 硬门槛 = 跨多场所（家+学校+诊室）；PMT = **A-B-C 三段框架**（Antecedents 沟通 / Behaviors 演练 / Consequences 后果）；ADHD 药物三档：兴奋剂 → atomoxetine → α2 激动剂。

#### 🔴 反复错（17 张，每张 1 句被动过一遍）

- **Q4469 AF 抗凝（5-18 #^Q4469）** — CHA₂DS₂-VASc ≥2（女≥3）= **必抗凝**；nonvalvular AF 一线 DOAC（apixaban/rivaroxaban），机械瓣/中重度 MS 仅 warfarin。
- **Q2429 Tetralogy of Fallot（5-16 升级）** — Tet spell 急救 = **knee-to-chest + O₂ + 镇静 + 容量 + phenylephrine（α1 激动剂↑SVR）**；不是 α1 拮抗剂。
- **Q4454 Digoxin 中毒（5-16 升级）** — 诱因是 **低钾 / 低镁 / 肾功能不全 / 老年女性体小**；血钾**高低提示严重程度**（高钾 ≥ 5.5 = 重度），但**诱因是低钾**。
- **Q15096 咖啡因中毒（5-20 升级）** — 焦虑 panic 表现 + 物质暴露史 → **优先 substance-induced，原发焦虑诊断让位**；同样适用 stimulant/EtOH/THC。
- **Q19929 高钾血症（5-15）** — ECG 改变（T 高尖 → QRS 宽 → 正弦波）→ 立刻 **calcium gluconate 稳膜** → **insulin+D50/β2 转细胞内** → 排（lasix/树脂/透析）。
- **Q21169 MI 后果反射连错（5-20 升级）** — 看到 "lifestyle change"题不要选 "讲后果(scare tactic)"；选 **共情 + 开放式提问 + readiness ruler 向下问**。
- **Q11815 Bath Salts 合成卡西酮（5-20 升级）** — 暴力 + 精神病 + 高热 + 兴奋 + 毒检阴性 → bath salts (synthetic cathinone)；不是 MDMA / 冰毒。
- **Q15843 BPD vs Bipolar I with psychotic features（5-20）** — 完整 manic syndrome（DIGFAST）带 psychosis → 优先 **Bipolar I + psychotic features**，BPD 让位；BPD 要求 psychosis 是"主菜"且独立于 mood/物质。
- **Q11903 慢性疼痛+抑郁（5-20 升级）** — 神经病理性疼痛 + MDD → **SNRI（duloxetine）一线**，杀两鸟；不是 TCA（老人抗胆碱毒）也不是 SSRI（疼痛弱）。
- **Q2355 / Q15856 急性即刻处理 ≠ 长期维持（5-22 升级）** — Mania **急性** = lorazepam ± haloperidol IM 镇静；**长期** = lithium / valproate。两题连续误选长期药。
- **Q2517 自杀最强单一风险因子（5-20）** — **既往自杀未遂**；不是抑郁/物质/家族史/慢性病。
- **Q2360 自杀意念 next step（5-20）** — 抑郁 + SI → 先 **suicide risk assessment**，再决定门诊/住院/药；不是直接开 SSRI。

#### #考前必看（轮值 1 句）

- **[[完整笔记/Peixuan分科笔记/Biostats_Master]] § OR / RR / HR 区分** — **OR**（case-control 用，无法算 risk）/ **RR**（cohort + RCT）/ **HR**（生存分析，含时间）；OR > 1 = 暴露组 odds 高于 ref 组；"X compared to Y" → X 是暴露。

#### #难记（按日期 mod N 轮值 — 今日轮 AF 工具）

- **[[完整笔记/专题笔记/心内/心内_AF复发与心律失常工具边界]]** — AF 抗凝 ≠ 心律转复 ≠ 复律；**长期 control = rate (β-blocker/CCB)** 通常优于 rhythm；**新发 AF + 不稳血流动力学 = 同步电复律**；rhythm control 适应症 = young/symptomatic/first episode/HF。

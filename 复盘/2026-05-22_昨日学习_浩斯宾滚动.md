## §一 一句话总结（强制复习清单）

### 1.1 D+1 昨日新学（2026-05-21，14 张卡）

- **Q15925 非典型抑郁（Atypical MDD）** —— 「能对好事高兴（mood reactivity）+ 食欲↑/嗜睡/铅样麻痹/拒绝敏感」≥2 项 = atypical；mood reactivity 是**诊断要件**不是排除依据，物质在场 ≠ 致病（须验剂量）。对照：melancholic 反着来（瘦+早醒+晨重暮轻+全面快感缺失）。
- **Q4877 青少年自杀未遂·非自愿住院** —— 急性自害风险 → 必须住院（可非自愿）；**未成年精神科非自愿住院可越过家长同意**；免责书不解除 duty to protect。「留观过夜」只适用中毒消退后再评估。
- **Q15054 抗抑郁药诱发转躁 first-step** —— 抗抑郁药数天内情绪飞涨 + 睡眠需求↓ = 病理性转躁（真起效需 4-6 周）；first step = **停诱因药**，不是加 mood stabilizer。通则：drug-induced 急性事件先撤药。
- **Q2355 急性躁狂激越即刻处理** —— 砸门扔杯 / 拒口服 / 安全风险的急性 mania → 第一步 **IM antipsychotic ± BZD**；Li/Valproate 是长期维持类别、口服需数天滴定，远水不救近火。
- **Q4816 PDD vs 人格障碍** —— 慢性低落 ≥2 年 + ≥2 附加症状 = PDD；PDD vs 人格障碍**不靠时间**（都长），靠「有无僵化行为模式 + ego-dystonic vs syntonic」；自卑是症状非诊断；仍主动约会 = 无回避 = 排除 avoidant PD。
- **Q7289 抗抑郁药黑框警告沟通** —— FDA 黑框：**<25 岁、自杀念头/行为（非 completed）、小幅、全类药 → 继续治疗 + 密切监测**；这是「沟通措辞题」非「治疗决策题」，答案居两极端之中。
- **Q15088 精神病性抑郁联合用药** —— 重度抑郁 + 妄想（罪恶/腐烂/虚无）= MDD with psychotic features → **抗抑郁药 + 抗精神病药**（缺 AP 开不了妄想锁）；Lithium 是增效剂、治不了 psychosis。
- **Q15811 精神病性抑郁 → ECT** —— 妄想 + 急迫（拒食脱水 / 即将自杀 / 妊娠）→ **ECT**（快不及·不能吃·不能用药，无绝对禁忌）；「原药有效→重启」是最强诱饵。
- **Q4848 高泌乳素鉴别** —— prolactin **≥100 ng/mL → 垂体 MRI** 排 prolactinoma；aripiprazole（部分激动剂）几乎不升泌乳素；用 stem 数值做鉴别钥匙。一代抗精神病药/risperidone 升、二代多不升。
- **Q11967 儿童急性精神病 → SLE** —— 儿童急性精神病先排可逆器质病因；精神病 + 关节 + 血（血小板↓）+ 肾（蛋白尿/血尿）= 多系统拼图 → SLE，**先筛 ANA**；毒检阴性 + 多系统否定药物归因。
- **Q2505 急性肌张力障碍（Acute Dystonia）** —— D2 拮抗药（止吐药/抗精神病药）数小时内头颈持续收缩 + 神志清醒 = dystonia；急救 **抗胆碱能**（diphenhydramine/benztropine）；读题先看用药史，别误当低钙抽搐。
- **Q4885 二代抗精神病药代谢监测** —— Olanzapine 招牌 = 代谢综合征 → **查空腹血糖 + 血脂**（baseline/3 月/每年）；Olanzapine ≠ Clozapine（后者招牌 = 粒缺/癫痫/心肌炎 → 查 CBC/ANC）。
- **Q16115 Clozapine 致癫痫** —— 抗精神病药降癫痫阈值，**clozapine 最高**（剂量依赖）；「治疗抵抗 + group home + 反复住院」画像反推 clozapine；精神科致癫痫二人组 = Clozapine + Bupropion。
- **Q11898 静坐不能（Akathisia）** —— 换药/加量后坐立不能 + 加量更糟 = akathisia（最常见 EPS、剂量依赖）；处理 = **减量 + propranolol（首选）**；换药往低 EPS 换，haloperidol 治 EPS 永远错。

### 1.2 滚动复习

- **D+3（2026-05-19）· Q2501 NMS vs EPS** —— 抗精神病药 + 高热 + 铅管样僵直 + 意识改变 + CK↑ = NMS；处理 = 停药 + dantrolene/bromocriptine；与 5-HT 综合征（下肢 clonus、反射亢进）鉴别。
- **D+7（2026-05-15）· Q3069 WPW + Preexcited AF** —— 预激 + 房颤 = 不规则宽 QRS 心动过速 → procainamide/ibutilide；**禁用 AV 结阻滞药**（adenosine/CCB/BB/digoxin）—— 会经旁路加速诱发 VF。
- **D+14（2026-05-08）/ D+30（2026-04-22）** —— 无对应日期节，名额转入弹性补位。

- **Q11815 浴盐 vs MDMA** —— 极端暴力 + 自主神经飙 + **体温正常** + 病程拖 ~2 天 = 合成卡西酮（浴盐）；尿检都阴性不是鉴别点，靠**体温 + 病程 + 暴力度**。
- **Q4454 地高辛中毒诱因** —— 慢性地高辛中毒（恶心 + 黄视 + 心律失常）经典诱因 = **低钾 / 低镁 / 高钙** + 利尿剂 / 胺碘酮·维拉帕米 + 肾衰；高钾是急性中毒的**结果**、不是诱因。

## §一 一句话总结（强制复习清单 — 扫一遍 = 被动复习全覆盖）

### 1.1 D+1 昨日新学（2026-05-19 · 21 张错题卡）

#### A. 物质滥用 / 戒断 / 成瘾药物

- **Q12541 AUD 一线药**：still drinking + cravings 强 → **Naltrexone**（μ-阿片拮抗，边喝边启动）。延伸：成瘾科 4 兄弟 — Naltrexone 管酒+阿片 / Acamprosate 只管酒（已戒断+肝病选它，肾排）/ Disulfiram 吓唬已戒酒者 / Bupropion-Varenicline 戒烟专员。
- **Q15104 AUD 药物治疗**：戒酒 2 周 + cravings + 怕复发 + 无肝病 → first-line **Acamprosate / Naltrexone**；**Disulfiram 是 second-line**，三必备（前线失败 + motivated + supervised）缺一不选。延伸：Disulfiram 不降 craving，靠"喝了难受"威慑；有 craving 的人要的是 anti-craving 药。
- **Q3187 ✅ AWS withdrawal seizure**：术后 30h + 长期酗酒 + tonic-clonic → **IV BZD**；**Phenytoin 对 AWS seizure 无效**（Na 通道阻断，不补 GABA）。延伸：AWS 时间窗 "6 颤 / 12 抽 / 12 幻 / 48 谵"。
- **Q3762 ✅ Alprazolam withdrawal seizure**：短效 BZD + 漏药 2 天 + 新发 seizure → Alprazolam。延伸：BZD 半衰期越短 withdrawal 越爆烈；GABA 类（BZD+酒）停药会 seizure 可致命，SSRI/SNRI 停药是 FINISH 综合征**不 seizure**。
- **Q3189 Heroin（阿片）戒断**：吐/泻/痛/失眠四联 + 出汗 + **vital signs 正常** → opioid withdrawal。延伸：戒断 vital signs 口诀 "GABA 飙 / 阿片平 / 兴奋剂沉"；阿片是"反向中毒"（中毒针尖瞳→戒断散瞳 / 中毒便秘→戒断腹泻）。
- **Q3191 PCP 中毒**：多方向眼震 + 暴力 + 肌强直 → PCP；任何 substance-induced agitation **首选 BZD（lorazepam）**。延伸："NOT BZD"只有 4 个 — opioid(naloxone)/APAP(NAC)/有机磷(atropine+2-PAM)/抗胆碱重症(physostigmine)。

#### B. 焦虑 / 情感障碍

- **Q3382 Adjustment Disorder**：明确应激源 + 起病 ≤3 月 + SIGECAPS <5 + 求医 → AD，一线 **psychotherapy**（不是 zolpidem 对症）。延伸：MDD 优先于 AD（SIGECAPS ≥5 不管有无 stressor 都是 MDD）。
- **Q8909 焦虑障碍鉴别**：怕"被负面评价" = **SAD**（public speaking 归 SAD performance-only，不是 specific phobia）。延伸：怕什么定诊断 — 物本身=phobia / 逃不掉=agoraphobia(TOECC ≥2类) / 怕下次发作=panic disorder。
- **Q21090 围产期 GAD**：worry 一整年 + muscle ache 孕前已有 + 安抚短效 + 辐射多领域 → GAD，一线 **CBT**（孕期避 paroxetine + 避 BZD）。延伸：触发器 ≠ 议题 — 看 worry 辐射几个领域，不看起点；"It's natural to worry…" 类选项 80% 是错的。
- **Q15074 SAD 季节性抑郁**：秋冬抑郁 + atypical features + SSRI **partial response** → **augment 加 light therapy**。延伸：抗抑郁药 "Partial = Augment，Zero = Switch"；家族史亲属用某药有效是干扰项。
- **Q2496 Panic Disorder**：正在 ED 发作中 → **BZD（lorazepam）救急**；维持期才用 SSRI/SNRI+CBT。延伸：panic 痛苦在中枢恐惧（GABA），不是外周心跳 → 不选 propranolol（那是 performance anxiety 事前用）。
- **Q15841 双相维持 Stable ≠ Stop**：bipolar 稳定 2 年仍 **终身维持**（BPS 三兄弟 = Bipolar/Psychosis/Schizophrenia 终身）；MDD 首发才能 6-12 月后减停。延伸：沟通 SOP = Empathy → Educate → Maintain（共情但不让步）。
- **Q15103 ✅ Bipolar 孕期维持**：valproate 致畸 → 换 **Lamotrigine**（孕期双向保护 + 低致畸）。延伸：精神科致畸口诀"丙卡奥致管缺，锂帕致心"。
- **Q2357 Bipolar I 维持 4 选 1**：Cr↑ / PKD 家族史 → 避 **Lithium**（100% 肾排）→ 选 Valproate（肝代谢）。延伸："Lithium 走肾，Valproate 走肝，哪条路坏避哪条"；本题错因是粗心（漏看 stem 实验室+家族史）。
- **Q15895 儿童抑郁 + 被动 SI**："消失就好了" = passive SI + ≥1 高危 → **急诊精神评估**优先于一切治疗。延伸：精神科 5 步 SOP（Safety → Stabilization → Diagnosis → Treatment → Follow-up）；儿童 SSRI 有 black box（增 SI）。

#### C. 精神药理 / 急诊毒理

- **Q2499 SGA 机制**：quetiapine 等 SGA = **5-HT2A + D2 双拮抗**（loose D2，少 EPS 多代谢）。延伸：抗精神病主受体永远是 D2（不是 D1）；α1/M1/H1 是副作用受体。
- **Q2501 NMS vs EPS**：haloperidol <2 周 + 高热 + AMS + 自主紊乱 + 铅管样强直 → NMS；refractory 用 **Bromocriptine**（"NMS BAD" = Bromocriptine/Amantadine/Dantrolene）。延伸：见 haloperidol 副作用先问 3 连问（发热/AMS/自主乱），任一"是"即排 EPS。
- **Q11853 MDMA + Serotonin Syndrome**：派对 + SSRI 用户 + 高热 + 肌阵挛 + 反射亢进 + **低钠 122** → MDMA。延伸：4 大高热综合征 — SS"湿热抖" / 抗胆碱"干热不抖" / NMS"湿热僵" / MH 麻醉中；SS 解药 cyproheptadine。
- **Q3126 TCA 过量**："best predictor of complications" 问预后 → **QRS 时限**（>100 ms 给 NaHCO₃，>160 ms 抽搐+室速）。延伸：诊断指标 ≠ 预后指标；抗胆碱"干"（Dry）vs 拟交感"湿"（Wet）。
- **Q3546 儿科 Still murmur**：无症状 + grade≤2 + vibratory + 站立↓ → innocent murmur → **reassurance**，不查 echo。延伸：HCM 站立杂音↑（与 Still 反向）；家族史红旗是 SCD <50 岁，不是 65 岁 MI。
- **Q16309 ✅ Paranoid Personality Disorder**：弥漫性不信任 + 无妄想/幻觉 + 终身 pattern → PPD（Cluster A）。延伸：PPD"世界对我不安好心"（弥漫）vs Delusional Disorder"邻居 John 监视我"（聚焦固定妄想）。

### 1.2 滚动复习（D+3 + 弹性补位历史节）

- **D+3 · 2026-05-17 · Q2663 BB 中毒**：自杀 + 心动过缓 + 低血压 + **wheezing + 低血糖** + atropine 无效 → β-blocker 中毒 → **Glucagon**（绕 β 受体激活 Gs/AC ↑cAMP）。三 B 鉴别：BB→支气管痉挛+低血糖 / CCB→高血糖 / Digoxin→GI+视觉+高钾。
- **弹性 · 05-15 · 主动脉夹层降压**：先 **β-blocker（控 HR + dP/dt）再 vasodilator**；顺序反了 vasodilator 单用 → 反射性心动过速 → 剪切力↑加重撕裂。
- **弹性 · 05-15 · Acute Pericarditis**：胸痛随体位/呼吸变化 + 心包摩擦音 + 弥漫 ST 抬高 + PR 压低 → 首选 **NSAID + colchicine**。
- **弹性 · 05-14 · PE 不稳定第一步**：血流动力学不稳定（低血压/休克）的 PE → **溶栓（tPA）**，不是先 CT；不稳定时影像让位于救命。
- **弹性 · 05-16 · Statin 不耐受 / ASCVD**：statin 肌痛 → 先停药排查 → 换低剂量/隔日 statin → 仍不耐受加 **ezetimibe ± PCSK9**；LDL 决定治疗强度。
- **弹性 · 05-15 · Orthostatic Hypotension**：站立 SBP↓≥20 / DBP↓≥10 — 容量型（HR 代偿↑）vs 神经源型（HR 不变）；晕厥 4 类 = 心源性 / 反射性（vasovagal）/ 体位性 / 神经源。

### 1.3 必复（🔴 反复错 + #考前必看 + #难记 轮换）

- **🔴 Vaughan-Williams 抗心律失常药分类**（错题本 **3 次** 反复错 — 顽固级）：IA 奎尼丁-普鲁卡因胺-双异丙吡胺 / IB 利多卡因-美西律（缩短 QT）/ IC 氟卡尼-普罗帕酮（**CAST：心梗后/结构性心脏病禁用**）/ II β-blocker / III 胺碘酮-索他洛尔（延长 QT）/ IV 非 DHP CCB。**ECG 双向锁死**：见 QT 短想 IB，见 QRS 宽想 IC。
- **🔴 AS 体征 / S3 vs S4 机制**（错题本 **2 次** 反复错 — 05-18 同题再错升级）：**S4 = 心房用力射血进僵硬心室**（压力超负荷 — AS / HCM / HTN）；**S3 = 容量超负荷快速充盈**（HFrEF / 重度 MR / 高输出）。AS = 迟峰收缩期杂音 + S4 + 脉搏 parvus et tardus。
- **#考前必看 · Biostats**：2×2 表是命根 — Sn/Sp 是检验固有属性（不随患病率变）；**PPV/NPV 随患病率变**（患病率↑→PPV↑）；LR+ = Sn/(1−Sp)，LR+>10 显著提高 post-test probability。
- **#难记 轮换 · ToF Tet spell 急性处理（KOMI-PB-bicarb）**：**K**nee-to-chest 屈膝抱胸（↑SVR）/ **O**xygen / **M**orphine（降耗氧+镇静）/ **I**V fluids（↑preload）/ **P**henylephrine（↑SVR）/ **B**eta-blocker（解漏斗部痉挛）/ bicarbonate（纠酸）。核心 = ↑SVR + ↑preload 把右向左分流推回去。

## 二、错题详解（4 道）

### ❌ Q15 — XLA，不是 SCID（免疫缺陷镜像混了）

**钥匙：T 细胞正常 + B (CD19) 细胞缺失 + 全免疫球蛋白 immunoglobulin↓ = X 连锁无丙种球蛋白血症 (X-linked agammaglobulinemia, XLA / Bruton)。**

- 重症联合免疫缺陷 (severe combined immunodeficiency, SCID) = **T + B 都垮**，特征 = **淋巴细胞减少 + CD3 T 细胞缺失**。
- 本题明说"**T 细胞正常**"→ 直接排除 SCID。只 B 系崩 = XLA。
- ⚠️ 你和 Q18（答对 = SCID，"CD3 T 细胞缺失"）做成镜像了：**看 T 细胞在不在，就分得开 SCID（T 缺）vs XLA（T 在、只缺 B）**。

### ❌ Q17 — 尿酸结石靠"碱化尿"，别嘌醇不是首选

**尿酸结石 (uric acid stone) 核心病因 = 尿太酸 (pH<5.5)**，尿酸在酸性尿析出。

- **一线预防 = 枸橼酸钾 potassium citrate 碱化尿到 pH 6–6.5 + 大量水化** → 还能**溶解**已成石（其他石做不到）。
- 别嘌醇 allopurinol 仅在高尿酸血症 / 碱化无效时加，**非单药首选**。
- 你选 D 打了问号 = 直觉不稳。记：**酸性尿的石头 → 把尿调碱。**

### ❌🔴 Q21 — 孕妇 Lyme = 阿莫西林（反复错 ×3！）

**孕妇 / <8 岁儿童早期 Lyme (莱姆病 Lyme disease) → 阿莫西林 amoxicillin。**

- **孕期禁多西环素 doxycycline**（胎儿牙着色 / 骨发育）；你选 A = 多西环素 = 孕期的雷。
- 非孕成人才 doxycycline；神经 / 心脏 Lyme 才头孢曲松 ceftriaxone。
- 🔴 对应真题 [[mistakes/uworld-mistakes_2026-06#^Q3257]]，**已第 3 次错**（6-09 → 6-11 头孢曲松 → 6-12 多西环素，干扰项每次不同 = 正解 amoxicillin 没锁死）。
- 已建当日追踪卡 [[mistakes/uworld-mistakes_2026-06#^Q3257-R0612]]。
- **焊死口诀："孕妇/小孩 Lyme → amox；成人 → doxy；脑/心 → ceftriaxone。"**

### ❌ Q24 — 他克莫司（钙调磷酸酶抑制剂）毒性

**移植后 震颤 + 高血糖 + 高血压 + 高钾 (+ 肾毒) = 钙调磷酸酶抑制剂 (calcineurin inhibitor, CNI：他克莫司 tacrolimus / 环孢素 cyclosporine) 经典毒性谱。**

- 他克莫司尤其：**神经毒（震颤 / 头痛）+ 致新发糖尿病**最突出。
- 你选 D = 硫唑嘌呤 azathioprine → 它的毒性是**骨髓抑制**，不是这套代谢 + 神经谱。
- 小鉴别：他克莫司 = 神经毒 + 糖尿病；环孢素 = 多毛 + 牙龈增生。

## 三、Q8 强化：袢利尿剂耳毒性（你点名复习）

> [!info] 袢利尿剂耳毒性 (loop diuretic ototoxicity)
> - **机制**：袢利尿剂（呋塞米 furosemide / 布美他尼 bumetanide）抑制内耳**血管纹 Na-K-2Cl 转运体**（与肾小管同靶）→ 内淋巴电解质紊乱 → **双侧对称高频感音神经性聋 sensorineural hearing loss ± 耳鸣**。
> - **风险↑**：大剂量 / 快速静推 / 肾衰 / 合用其他耳毒药（氨基糖苷 aminoglycoside、顺铂 cisplatin）。
> - **可逆性**：袢利尿剂**多可逆**（停药 / 降量恢复）——与**氨基糖苷（不可逆）**的关键分水岭。
> - 对应真题 [[mistakes/uworld-mistakes_2026-06#^Q2838]]；完整耳毒药谱见 [[完整笔记/专题笔记/神经/神经_耳鸣鉴别谱]] §四。

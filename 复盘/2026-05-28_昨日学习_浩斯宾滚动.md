### 1.1 D+1 昨日新学（2026-05-27）— 32 张卡精华一句话

**浩斯宾 25 题自测错点（4 张，全在 D+2 5-25 重测）**：

- [ ] **Q3 Tourette "Anchor by Letter" pattern 新错点 🟡 + #难记** ⚠️ — α2 (guanfacine) 失败 → 加 **risperidone / aripiprazole**（antidopaminergic）；**Methylphenidate 永远禁用（加重 tics）**；Benztropine 是 EPS 急性 dystonia 药（不治 tic）。**5-25 D = antidopaminergic 答对 → 5-27 反射选 D 但 D 已变 methylphenidate 毒药**；**USMLE 故意打乱字母 anchor 测真懂**。
- [ ] **Q7 Swiss Cheese vs PDSA 混 — 升 🔴 + #难记**（5-25 + 5-27 反复错）⚠️ — **4 大 System Design Models 钉死**：① **SCM** = "**simultaneously / concurrently**" + 列举 ≥ 2 不同性质 barrier ② **Hard stop** = "physically incompatible" ③ **Automation** = "EHR alert / smart pump / barcode" ④ **PDSA** = "repeated cycles / pilot then refine"。本题 5 种措施 simultaneously implement → SCM。
- [ ] **Q8 RCA primary purpose — 第 3 次同型错 🔴 + #考前必看 + #难记** ⚠️⚠️ — **RCA = 已发生过去事件 underlying causes**（"past events"）；**FMEA = 未发生未来事件 potential vulnerabilities**（"future" + "potentially cause" + "vulnerabilities"）；选项三连"future + potentially + vulnerabilities" → **FMEA 语言钓饵**。stem retrospective ("errors HAVE occurred") → RCA。
- [ ] **Q9 Home Health vs SNF / admit — 升 🔴 + #考前必看**（5-25 admit / 5-27 SNF 都不选 home health）⚠️ — **Disposition 决策树**：Medically stable? → ❌ Hospitalize / ✅ Stable → Severe CI + 不能 ADL + 持续 monitoring → **SNF** / Mild CI + 能 ADL + Homebound + 需 skilled care → **Home Health Care**。78 岁 mild CI + 能 ADL → home health。

**5-27 上午做题：Ethics / Patient Safety / 老年 fall / Research / Communication 大爆发（28 张）**：

- [ ] **Q105919 OUD 家属教育 — Buprenorphine "trading one drug" misconception** → 直接答 **long-term outcomes**（不是 "ease withdrawal" / 不是 partial agonist 术语 / 不是 "only way"）。家属沟通核心 = OUD = 慢病，buprenorphine 不是 substitution 而是 evidence-based maintenance（减 mortality / 减 relapse / 增 retention）。
- [ ] **Q107154 老年 Fall 二级预防 — Bone density 优化（FRAX + DEXA + bisphosphonate）** 减 serious fall **injury**（不是 call alarm / 不是 stop anticoag / 不是 hip protector — hip protector 依从性差）。区分 "fall prevention" vs "fall injury prevention" — 后者重点在骨密度 + 抗骨折药。
- [ ] **Q106048 诊断错误 root cause = Cognitive Overload** — busy urgent care + multitasking → **availability bias 40% + confirmation bias 40% = 80% 诊断错误**。Cognitive overload 是 diagnostic error 之**首因**（不是 system bandwidth 不是 fatigue）。
- [ ] **Q106049 CDSS 减 cognitive overload + bias**（linked Q106048 Item 2）— 电子辅助决策工具 (electronic decision support) 提供 differential 列表 + alert → 减 availability / confirmation bias；不是 RCA / 不是 simulation / 不是 care coordination / 不是 lab automation。
- [ ] **Q22186 Surgical Time-out — 患者参与 site verification** — 患者**清醒 / 自愿 / 同意**时**亲自标记 site**（不是 brief / 不是 ≥2 experienced / 不是 mark nonoperative / 不是 enter OR 前完成 final time-out — final time-out 在 OR 内 anesthesia 前做）。
- [ ] **Q19813 CLABSI Prevention — Hand hygiene 最 fundamental 最 efficacious intervention** ✓ — CLABSI bundle 多措施，但 **hand hygiene 单一最 efficacious**（减 risk 最多）。不是 catheter dressing change / 不是 chlorhexidine bath / 不是 daily review of necessity。
- [ ] **Q106620 Psychological Safety promotes RRT activation** ✓ — Psych safety = "speak up 不被 penalize" 文化 → 鼓励 nurse / junior 早 activate Rapid Response Team；不是 closed-loop / 不是 independent decision / 不是 cognitive bias / 不是 standardize transitions。
- [ ] **Q107171 老年 Fall — Visual Acuity Testing** ✓ — Multifactorial fall risk assessment 包含 **sensory disturbances**（视/听）；reliance on visual input **↑** when proprioception **↓**（DM neuropathy + vibratory sensation 减 → 视觉补偿 → 视力差则 fall ↑）。
- [ ] **Q19788 Teach-back 防 sound-alike drug error** ✓ — **Glucovance vs Glucophage** 同形不同药 → Teach-back（让患者复述 name + dose + indication）；不是 "ask if questions" 被动 / 不是 avoid combinations / 不是 family / 不是 printed summary。
- [ ] **Q106819 HFE — Brightly colored visual signs at point of care** ✓ — Hand hygiene 提示视觉 cue **point of care**（直接在洗手位置）> training / patient ask / supply at workstations；**HFE 优于 education**（人记不住 / 疲劳 / 跳过）。
- [ ] **Q106664 Cognitive Bias 误归因 team collaboration 差异 ⚠️** (linked Item 1 of 2) — 表面"team collaboration 差"实际是 **Differences in understanding of team goals**（看似偏见误归因，实际为 shared mental model 缺失）。**Q106665 Item 2 →** **Nurse-led Debriefings** 解决 ✓（不是 Lean / 不是 checklists / 不是 handoff templates / 不是 surgeon-only training）。
- [ ] **Q107153 老年 Fall Prevention — Supervised Strength Training** ✓ — chronic disease + 起立 push with arms = gait/muscle imbalance 信号 → 监督下力量训练；不是 vitamin D empiric（缺 25-OH-D 数据时不空开）/ 不是 ACE inhibitor 停 / 不是 handout / 不是 alcohol 减（无相关史时）。
- [ ] **Q106640 ERS 有效性 = Structured 调查 + 响应机制 + 反馈** ✓ — Error Reporting System 不是单单 submit 就完事，**关键 = 结构化调查 + 反馈 loop + 接 RCA**；不是 individual judgment / 不是只识别最严重 hazards / 不是限定 nurse-MD。
- [ ] **Q126013 Just Culture: At-risk Behavior 处理 = Coaching** ⚠️ — Just Culture 3 层处理 ① **Human Error** = Consoling（系统问题）② **At-risk Behavior** = **Coaching**（教育 + 帮校准 risk perception）③ **Reckless Behavior** = Sanctioning（罚）；不是 Consoling for human error / 不是 Retraining for knowledge / 不是 Sanctioning for reckless / 不是 No action。
- [ ] **Q19909 Transfer of Care — Primary Care Follow-up Counseling** 防 loss to follow-up — 出院前明确 PCP 随访时间 / 内容 / 联系方式；不是 "monitor pending results for all patients"（系统责任）/ 不是 inform family / 不是 postpone discharge。
- [ ] **Q19785 诊断错误 — Metacognition（理解自身 cognitive patterns）** ⚠️ = 防诊断错首选；不是 CDSS / 不是 intuitive thinking（System 1 反加重 bias）/ 不是 past experiences（availability bias）/ 不是 prioritize past diagnoses（anchoring）。Metacognition = "thinking about thinking" — 是 most effective single intervention。
- [ ] **Q107484 Research / Therapeutic Misconception + PI 双重角色** — 参加 trial 误以为获得 **superior care**（实际 trial 不保证 individual benefit）→ Next action = **解释 trial 性质 + 可能不获益**（informed consent core）；不是 disclose PI conflict + seek care elsewhere（PI 双重角色不要求"另觅他医"，要求充分 disclosure + assent）。
- [ ] **Q4066 Angry Patient 应对 SOP** = **acknowledge + nondefensive + open-ended question**（"我看到你很生气，能多告诉我一些吗？"）；不是 transfer to attending / 不是 premature interpret cause / 不是 cite bill of rights / 不是 reassure 自己受过训练。
- [ ] **Q21751 P4P + ACO = 双向优化 cost + quality** — Pay-for-Performance (奖励 quality metrics) + Accountable Care Organization (集体担 cost) 是**唯一两端兼顾**的 model；不是限制住院长度（降本但 quality ↓）/ 不是 mandate generic trial（短期降本 long-term quality ↓）/ 不是 prior authorization gate（officially cost control 实质 access 障碍）。
- [ ] **Q21966 PCC = 综合 patient values（含 cost / affordability）+ shared decision-making** — Patient-Centered Care 不只是 "尊重 autonomy"，而是 **integrate patient values + preferences + cost concerns** 制定方案；不是 biweekly 增 copay / 不是 NRT precontemplative（precontemplative 不愿戒不用 NRT）/ 不是 减重 first（priority 由 patient 决定）/ 不是死守 first-line guideline 不顾 affordability。
- [ ] **Q18785 Clinical Uncertainty Counseling 3 节 SOP** ✓ — ① **Introduction**（"今天我想和你谈下检查结果，可能不确定"）② **Data review**（中性数据 + 不暗示走向）③ **Plan of care**（具体下一步 + timeline）；不 minimize（"routine"）/ 不 vague（"not sure what's wrong"）/ 不 alarming blunt（"I suspect cancer"）/ 不 falsely reassuring（"no matter what"）。
- [ ] **Q106618 P4P + Equity / Health Disparities** ⚠️ ✓ — 大型国家 P4P 必须按 **patient demographic characteristics**（insurance type / income / shortage area status）调整 incentive 分配，否则 **preferentially reward affluent regions** → 加剧 health disparities → 反而**降低 overall quality**。不是 teamwork / QI / experience / burnout。
- [ ] **Q105782 Professional Boundaries — Treating Friends/Family** ✓ — 关系本身**不禁止**处方但 **standards of care 必须保持**；患者 decline 必要 evaluation + 要求 controlled substance → 不 obligated 提供 inappropriate intervention；明确 standards 边界 + 谈判 + transfer of care if 不 abide（不是 refuse based on friendship / 不是 terminate / 不是 accept risk acknowledgment）。
- [ ] **Q21728 Medical Futility 钉死** ⚠️ ✓ — Physician **no ethical obligation** 给 futile treatment（即使 AD 写 "everything possible" 即使 surrogate 要求）；二分：**Physiologic futility** = 不能改善生理 / **Qualitative futility** = marginal benefits substantially outweighed by 痛苦；**High-risk-some-benefit ≠ futile**（informed competent request 时不能 refuse）；Resource stewardship + prognosis impact **不是** sufficient standard for futility。
- [ ] **Q126839 Managed Care 反向激励 + Physician COI** ✓ — Managed care 用 bonus 与 utilization metrics 挂钩**本身不 inherently unethical**（cost-conscious care 合理）；**但 financial incentive → withholding medically necessary referral → impermissible**（violates beneficence）；医师必须 place patient welfare above personal financial gain；与 P4P 镜像（P4P 奖励质量 / Managed care 反向奖励 reduced utilization）。
- [ ] **Q19787 Preventable ADE 3 层 Risk Factors（Patient / Provider / Medication）** ✓ — Plain-language teaching 击中 vague Rx + low health literacy 双因素 root cause；不是 follow-up / 不是 illegible / 不是 CDSS / 不是 drug abuse screening — 都不 address initial misinterpretation。
- [ ] **Q107485 Research — Minors (<18) 入组 Assent** ✓ — **Assent**（agreement，不是 legal consent）+ parental/guardian informed consent；Assent oral script (7-11) vs written (≥12)；Exception: study **directly benefits** minor → assent 不 required；Assent ≠ medical treatment of minors（治疗不需要 minor agreement）。
- [ ] **Q107384 Interprofessional Communication — Imaging Discrepancy** ✓ → **直接联系 radiologist** discuss + amend report（不是另请 second opinion / 不是 bypass radiologist 自己 order CT / 不是 repeat x-ray now/later 跳过 collaborative dialogue）。

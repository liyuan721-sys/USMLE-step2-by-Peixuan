## 二、错题详解（6 题）

### Q3 — Nodal block + atropine 有效 → 观察

> 下壁 STEMI 24h + 窄 QRS escape 50/min + atropine 后 HR ↑80 + 传导恢复 1:1

**正解 B**（TCP 备用 + 观察）。Nodal block + atropine 有效 = **良性自限**（数小时-数天恢复），**不需要永久起搏器**。

**错因分析**：把 nodal 和 infranodal 处理搞反了：
- Nodal（窄 QRS + atropine 有效）→ **观察**
- Infranodal（宽 QRS + atropine 无效）→ **才**装 PPM

→ 衍生：[[完整笔记/专题笔记/心内_AF复发与心律失常工具边界#Part 3. AV Block 定位 — Atropine 反应判断]]

### Q4 — Infranodal → TCP → PPM

> 前壁 STEMI + 宽 QRS escape + atropine 无反应

**正解 C**（TCP → permanent pacemaker）。LAD 缺血侵蚀 His 束 → infranodal → 必须 pacing。即使后续恢复也装 PPM（His 受损隐患极高）。

**错因分析**：选 D（再 angiography）可能想 stent 再闭塞，但题目未给"新 ST 抬高"这个 stent thrombosis 标志。**T 波倒置 = MI 演变期自然演变**（reperfusion T），不是再闭塞 → 见 [[完整笔记/专题笔记/心内_MI并发症时间轴#7.C MI 后 ECG 演变 4 阶段]]。

⚠️ **Q3 + Q4 连续错** = **AV block 定位决策** 薄弱（详见 §三薄弱点）。

### Q12 — 后内侧乳头肌 = RCA 单一供血

> 下壁 MI + 急性 papillary muscle rupture + 暴风式肺水肿 + apex soft systolic murmur

**正解 C (RCA)**。后内侧乳头肌（PMP）= **RCA 单一供血**（PDA 分支），下壁 MI 时最易缺血断裂。前外侧乳头肌（ALP）有 LAD + LCx 双重供血，相对安全。

**错因分析**：选 E（Left main）— Left main 闭塞会**立即大面积梗死死亡**，不会出现 day 4 才发生的迟发 papillary rupture。下壁 MI ≡ RCA。

**记忆挂钩**：**"后内侧乳头肌只吃 RCA 一口饭，所以最容易饿死"**

→ 衍生：[[完整笔记/专题笔记/心内_MI并发症时间轴#3.3 Papillary Muscle Rupture → 急性 MR]]

### Q15 — Williams = 高钙

> 4 月龄 FTT + 精灵脸 + 星状虹膜 + SVAS

**正解 B (Hypercalcemia)**。**Williams 三联：FTT + 高钙 + SVAS**（机制：elastin gene 邻近 Ca 调节区缺失）。

**错因分析**：把"Williams 高钙"和"DiGeorge 低钙"弄混。

**钙方向对应**（必须刻死）：
- **Williams** → **↑ Ca**（精灵脸 + SVAS）
- **DiGeorge** → **↓ Ca**（CATCH-22 中 H = Hypocalcemia 甲状旁腺发育不全）

→ 衍生：[[完整笔记/专题笔记/儿科_染色体病多系统对照表#📋 快速锁定：Stem 关键词 → 综合征]]

### Q16 — DiGeorge = 22q11.2 deletion

> 新生儿强直阵挛（低钙抽搐）+ 鹅口疮 / 肺孢子菌（T 细胞缺陷）+ truncus arteriosus

**正解 C (22q11.2 deletion)**。**CATCH-22**：**C**ardiac（truncus/ToF/IAA）+ **A**bnormal facies + **T**hymic aplasia（T 细胞缺陷 → 病毒/真菌）+ **C**left palate + **H**ypocalcemia（甲状旁腺发育不全）+ **22**q11。

**错因分析**：选 D（5p-）= Cri-du-chat = 猫叫样哭声 + 小头 + 智障，跟免疫缺陷 / 心脏畸形无关。

⚠️ **Q15 + Q16 连续错** = **染色体病-多系统对应** 薄弱（详见 §三薄弱点）。

### Q17 — Tet spell 禁忌 = β-agonist

> ToF Tet spell **禁忌**药

**正解 D (Isoproterenol)**。β-agonist → ↑ 心肌收缩 → ↑ RVOT 漏斗痉挛 → 紫绀加重。

Tet spell 三大禁忌：
- ❌ **β-agonist** (isoproterenol)
- ❌ 扩血管药（nitrates / ACEi）— ↓ SVR → R→L 分流↑
- ❌ 利尿剂 — ↓ 前负荷

**错因分析**：选 E（Propranolol）**方向反了**。Propranolol 是 β-blocker → **缓解 RVOT 漏斗痉挛 = 急救一线药之一**。

**急救三联目标**：↑ SVR + ↓ RVOT 痉挛 + ↑ 前负荷
- Knee-to-chest（↑ SVR）/ Phenylephrine（↑ SVR）→ 满足 #1
- Propranolol（缓解痉挛）→ 满足 #2
- IV fluids（↑ 前负荷）→ 满足 #3
- Morphine（镇静 + 降低儿茶酚胺）→ 间接降痉挛
- O₂

→ 衍生：[[完整笔记/专题笔记/心内_CHD_ToF法洛四联症#六、Tet Spell 急救 ⭐⭐⭐]]

## §一 一句话总结（强制复习清单 — 扫一遍即被动复习）

### 1.1 D+1 昨日新学（2026-06-14，肺 + 儿科 + 骨科·风湿 26 张）

> [!info] 昨日 4 大反射（最该焊死）
> ① **复杂性肺炎旁积液（pH<7.2 / 糖低 / 脓 / 细菌＋）→ 胸管引流**；单纯小量无窘迫→口服抗生素门诊随访。② **大剂量沙丁胺醇 (albuterol) → 低钾血症 (hypokalemia)**（β₂ 驱钾入胞）＋震颤＋反射↓→复查电解质。③ **IPF (idiopathic pulmonary fibrosis, 特发性肺纤维化) PFT = 限制型**（↓TLC / FEV₁-FVC 正常或↑ / **↓DLCO**）。④ **股骨头骨坏死 (AVN) = 慢性酗酒 > 短程激素**（长期大剂量激素才是经典诱因）。

- **Q117999 血管环 Vascular ring**：婴儿喘鸣/吞咽困难（双主动脉弓压迫气管食管）→ CT/MR 血管造影确诊。
- **Q4831 新生儿 RDS (respiratory distress syndrome, 新生儿呼吸窘迫综合征)**：母糖尿病→胎儿高胰岛素延迟表面活性物质 (surfactant)→气促/呻吟/毛玻璃→外源性表面活性物质。
- **Q19417 原发性肺结核**：叶浸润 + 同侧肺门淋巴结肿大 = Ghon complex（✅做对）。
- **Q3731 创伤性膈肌破裂**：钝伤后腹腔脏器疝入胸（左侧多，胃泡/肠管）→ 胸腹 CT；胸片可见胃管尖端在胸腔。
- **Q20107 / Q12421 肺炎旁积液**：复杂性（pH<7.2、糖<60、脓、细菌＋）或脓胸→**胸管引流**；单纯小量（<1 cm 侧卧位）无窘迫→**口服抗生素门诊**随访。
- **Q4297 肺心病 Cor pulmonale**：COPD → 孤立性**右心衰**（颈静脉怒张/肝大/下肢水肿）**无左心衰证据**（✅做对）。
- **Q3026 沙丁胺醇低钾**：大剂量 β₂ 激动→K⁺ 入胞→肌无力 + 腱反射↓ + 震颤 + QT/U 波→复查电解质（不是新发神经病）。
- **Q12459 IPF PFT**：限制型——↓TLC、FEV₁/FVC 正常或升高、**↓DLCO**（弥散障碍是关键）。
- **Q19607 CAP 危险因素**：老年 + 非典型抗精神病药 **喹硫平 (quetiapine)** 镇静→误吸→社区获得性肺炎。
- **Q4632 右主支气管插管**：插管后**右侧**呼吸音正常、左侧减低（右主支气管更陡直）→ 退管 2 cm + 复查。
- **Q3553 毛细支气管炎并发症**：RSV，**<2 月龄**最危险并发症 = **呼吸暂停 apnea**（先于喘息出现）。
- **Q19811 HHT 遗传性出血性毛细血管扩张症**（Osler-Weber-Rendu）：复发鼻出血 + 皮肤黏膜毛细血管扩张 + **肺 AVM 咯血/反常栓塞** + 家族史。
- **Q20411 早期 ILD (interstitial lung disease, 间质性肺病)**：双下肺 **Velcro 爆裂音** + 杵状，胸片可正常→**HRCT (high-resolution CT, 高分辨率 CT)**（早期最敏感）。
- **Q16267 异物吸入 FB**：幼儿突发**单侧**喘息 + 沙丁胺醇无效 + 呼气相过度充气→支气管镜（诊断＋治疗）。
- **Q4770 肺炎低氧**：肺内**分流 (shunt)**——受体位影响（健侧卧位氧合改善），吸纯氧纠正差；vs 死腔（PE）。
- **Q4582 假性痛风 CPPD (calcium pyrophosphate deposition, 焦磷酸钙沉积病)**：急性单关节炎（膝/腕）+ 影像**半月板/关节软骨钙化 chondrocalcinosis**；菱形正双折光晶体。
- **Q4636 腕管综合征**：轻-中症首选**夜间腕夹板**（孕期尤其，多产后自行缓解）；重/肌萎缩才手术松解。
- **Q21345 股骨头骨坏死 AVN (avascular necrosis)**：慢性**酗酒** > 短程激素；髋/腹股沟痛、MRI 最敏感（早期 X 线正常）。
- **Q4581 银屑病关节炎**：next step = **全面皮肤 + 指甲检查**（找银屑病斑块/甲凹陷/油滴征）确立诊断。
- **Q16328 Colles 骨折 + 正中神经**：远端桡骨背侧移位骨折后桡侧三指麻木 = 急性腕管/正中神经牵拉。
- **Q15468 川崎病心衰**：急性期**淋巴细胞性心肌炎**致心功能下降（不是冠脉瘤晚期）→ IVIG + 阿司匹林。
- **Q4577 反应性关节炎**：一线 = **NSAIDs**（不是抗生素，除非活动性衣原体尿道炎才治原发感染）。
- **Q15848 儿童化脓性关节炎**：经验万古覆盖 MRSA 仍无效→加**头孢曲松**覆盖 **Kingella / 革兰阴性**。
- **Q3575 网球肘 vs 桡管综合征**：压痛点定位——肱骨**外上髁**（网球肘）vs 外上髁远端 3-4 cm 旋后肌处（桡管）。
- **Q19907 肩脱位后神经/肌腱**：复位后看**三角肌区感觉**——感觉缺失→**腋神经**损伤；感觉完整 + 抬肩无力→肩袖撕裂。

### 1.2 滚动复习（D+3 / D+7 / D+30 各 1 条；D+14 无节）

- **D+3 (06-12) 气管支气管损伤**：钝胸伤后持续大量漏气 / 胸管引流后肺仍不复张 / 纵隔气肿→**支气管镜**确诊定位。
- **D+7 (06-08) 脑膜炎球菌血症 (meningococcemia)**：青少年瘀点/紫癜 + 休克→**DIC (disseminated intravascular coagulation, 弥散性血管内凝血)**（查纤维蛋白原↓、血小板↓、PT/PTT↑、D-二聚体↑）；疑诊先经验抗生素勿等。
- **D+30 (05-16) 地高辛中毒 (digoxin toxicity)**：恶心 + **黄绿视** + 心律失常（房速伴阻滞 / 室早）+ 急性中毒**高钾**→**地高辛特异性抗体 Fab (digoxin immune Fab)**；低钾/低镁/肾衰加重毒性。

### 1.3 必复（🔴 + #考前必看 + #难记）

- **🔴 移植肾动脉狭窄 (transplant RAS, renal artery stenosis)**：移植后 1-2 年难治性高血压 (resistant HTN) + 用 **ACEI/ARB（血管紧张素转换酶抑制剂/受体阻滞剂）后 Cr 急升** = 经典指纹→多普勒/MRA，停 ACEI（勿盲目加量赖诺普利 lisinopril）。
- **🔴 Bipolar I 维持治疗**：首选**锂盐 (lithium)** 或丙戊酸 (valproate)；**抗抑郁药单用禁忌**（诱发转躁/快速循环）。
- **#考前必看 Ethics 老年安置**：患者有能力 + 居家有支持→**居家护理 (home health care) 优先于 SNF / 入院**（尊重自主 + 最少限制原则）。
- **#难记 Biostats Cohort vs Case-Control**：比较"**发病率 / 相对风险 / 前瞻随访**"→**队列 cohort**；**从已患病/罕见病倒추暴露**→**病例对照 case-control**（算 **OR**）。

### 1.1 D+1 昨日新学（2026-05-26）— 24 张卡精华一句话

**浩斯宾自测错点（4 张）**：

- [ ] **Q2 DPN first-line "any of the above" 反复错升 🔴** — DPN first-line 4 选 1 平等（duloxetine / pregabalin / gabapentin / TCA — ADA/AAN 2024）；见 **"D = any of the above"** 暗号优先 D，不要急选熟悉单选 duloxetine。**5-24 Q19 + 5-26 浩斯宾 Q2 同题再错 = 升 🔴**。
- [ ] **Q12 Lifestyle counseling first step = SMART goal via MI**（不是 daily tracking log / 不是 confidence rating / 不是 specific Rx）；tracking 是 SMART goal 建立后的 implementation step。5 步时序：SMART → implementation tools → confidence rating → specific Rx → reassess。
- [ ] **Q20 PD ψ Step 5 错点漂移：Levodopa 是 foundation 不能减** ⚠️ — PD ψ 5 步阶梯：① 排谵妄 → ② 减 anticholinergic → ③ 减 amantadine → ④ 减 DA agonist → ⑤ **加 quetiapine/pimavanserin**（不停 levodopa — 减 = NMS-like + motor 崩）。**Haloperidol/risperidone 是毒药**。Q12152 维持 🔴 + 新加 #考前必看。
- [ ] **Q23 DOAC + acute stroke：tPA 禁忌，走 thrombectomy if LVO** — Rivaroxaban / apixaban / edoxaban / dabigatran 末次 < 48h → **tPA 禁忌**（无可靠 reversal-tPA 通路）；**Idarucizumab reverse dabigatran 是唯一 FDA 批的 reversal-tPA 例外**；Andexanet alfa / PCC ❌ 不开溶栓的路。

**Ethics / Patient Safety / QI 工具大整理（11 张 — 多数 ✓ 做对但知识深化）**：

- [ ] **Q21970 Fishbone diagram (Ishikawa)** — Organize **多 category + causes**（RCA 团队讨论视觉工具）；3 大组件：problem (鱼头) + categories (鱼脊主刺) + specific causes (鱼侧支)。**不是 Lean**（Lean = 减 extraneous steps）。
- [ ] **Q19684 Closed-loop communication** — 防 **verbal order miscommunication**（如 "20 of K" → 误录 vitamin K IV 而非 oral potassium 20 mEq）；read-back 3 步：sender → receiver 复述（full name + dose + route + time）→ sender 确认。**Sign-out 是 shift change handoff（防 omission）≠ closed-loop 实时纠错**。
- [ ] **Q106765 Defensive medicine → Clinical pathways** — Fear-driven overuse → 遵循 guideline = standard of care = malpractice shield；不是 cognitive bias 训练（cognitive bias 适 diagnostic error，"医生知道不必要还做 = defensive medicine vs 医生不知道做错 = cognitive bias"）。**Linked vignette 警示：Item 1 ≠ Item 2 主题**。
- [ ] **Q106764 Framing bias** — 受 **chart documentation / prior info / triage label**（别人的 interpretation）过度影响形成 incomplete differential；vs **Anchoring** = 自己被 raw fact 锚（crushing chest pain → ACS）。⚠️ **5-26 暴露根本卡点**："chart = interpretation 不是 raw data"反射未建立 → 状态降 🟡，错因 — → 知识。
- [ ] **Q106042 Just culture 时序原则** ✓ — Systems first（已 sustained + comprehensive 仍 suboptimal）→ Individual accountability（build into evaluation + 阶梯式 disciplinary + real-time peer feedback）；不是 culture of fear（先罚后修）也不是 no-blame culture（不问责）。**"先修路，再问司机"**。Hand hygiene 4 大 noncompliance 原因 + solutions 必背。
- [ ] **Q23111 Systemic quality concerns → Team-based + committee** ✓ — Excessive variations + 无 adverse events → 内部 team-based first（meet director + offer to participate on committee + frame as shared goal），外部 accreditation board 是 last resort。**Decisive vs 锦上添花原则**：leverage（leadership + community trust）= 锦上添花（justify "participate" 措辞），但**方向**由 no-adverse-event 决定。
- [ ] **Q21753 Lean methodology** ✓ — 减 **waste / inefficiency / extraneous steps**（streamline scheduling → 减 wait time / no-show / transportation）；vs **Six Sigma** = 减 defects 到 6σ / **PDSA** = iterative cycles / **Change management** = 早期采纳者 / **Swiss Cheese** = mental model（不是 methodology）。
- [ ] **Q21366 Implicit bias 3 大管理** ✓ — ① **Assess** (IAT 标准化工具) ② **Group debriefing + collaborative discussion** ③ **Train metacognition**；**避免 punitive measures**（distress + 失 reporting + 损 trust）；anonymous portal 适 explicit 而非 implicit 偏见。
- [ ] **Q23112 Effective Team Functioning 6 元素** ✓ — Safety culture / **Collaboration (shared goals ⭐ + distinct roles avoiding steep hierarchy)** / Psychologic safety / Responsiveness / Situational awareness / Transparency；**Distinct roles ≠ steep hierarchy**（角色清楚但等级 flat → 鼓励发声）；role overlap = harmful duplication（多 insulin admin 例）。
- [ ] **Q18751 WPE → Interprofessional rounds** ✓ — **Sentinel event 定义**：patient safety incident unrelated to underlying condition → death / severe temporary / permanent harm；**RCA 3 原则**：① fundamental factors ② focus on systems NOT individual (no blame) ③ action plan organizational improvements；Wrong Patient Error CPOE → pharmacist + nurse on rounds = collaborative oversight 早发现 incorrect order。
- [ ] **Q22359 Quality Indicators 4 类** ✓ — **Structural**（人 / 设备 / 知识）/ **Process**（compliance / 执行率）/ **Outcome**（健康结局）/ **Balancing**（counterbalancing tradeoffs）；本题 stem 给了 Structural + Balancing + Outcome 缺 **Process**（checklist 完成率 — 若 low 解释 outcome 不变）。
- [ ] **Q21520 Verbal Handoff Best Practices** ✓ — Focus on **sickest patients first**（time-conscious + priority-based）；不是 questions at end / 不 discourage note-taking / 不 exhaustive details。

**其他科散点（5 张）**：

- [ ] **Q19812 Catheter-related VTE in CVC manipulation**（儿科 / 肺）— 突发 hypoxia + hypotension after CVC flush → 已存在 catheter-tip clot 被 flush dislodge → PE；vs heparin error（出血不是 PE）/ vs anaphylaxis（urticaria + bronchospasm）/ vs acute pulm edema（rales + JVD）。
- [ ] **Q107149 Inpatient Fall Prevention — Direct line of sight supervision** — Delirium + agitation 患者首选 **constant observation by sitter**（直接视线 + 即时 redirection）；不是 bed alarm（only signals fall in progress）/ 不是 restraints（rule-out 后才考虑且不防 fall）/ 不是 bright lighting at night（破坏 sleep-wake cycle 反加重 delirium）。
- [ ] **Q107151 + Q107152 Fall screening + Tailored Prevention** ✓ + 学 — Screening tool **low PPV** 当 80% classify as high-risk vs actual 11.5/1000 events（high pretest probability 导致 false positives 多）；后续 prevention 必须**基于 individual risk factors tailored**（不是 universal bed alarm / 不是 bed rest 反 deconditioning / 不是 wristband 当 80% 都贴时失去 signal value）。
- [ ] **Q105800 Shoulder Dystocia: Simulation-Based Training (SBT)** — Unpredictable emergency 不能 risk stratification 防；SBT > checklist > debriefing > RCA；team-based maneuvers (McRoberts / suprapubic pressure / Woods screw) 要 fluid execution，**靠肌肉记忆 + 团队默契**。
- [ ] **Q105825 HFE Standardization：metric-only units** ✓ — tPA 160 lb 误入 160 kg → 过量；Human Factors Engineering 优先**消除歧义在系统层**（metric only）> policy（人记不住）/ visual cues（疲劳忽略）/ 2-person verify（still ambiguous）。
- [ ] **Q106949 Airborne Precautions for primary VZV (chickenpox)** ✓ — 颗粒 **< 5 微米** → 长时间空中悬浮 + 远距离传播；3 大 airborne：**M**easles + **T**B + **V**aricella (zoster disseminated 或 primary chickenpox)；带 **N95 + negative pressure room**。
- [ ] **Q107227 Time-out 三大核心要素** ✓ — **Patient ID + Procedure + Site**（不是 indication / 不是 meds / 不是 team names / 不是 complications）；WHO Universal Protocol 减 wrong-site / wrong-patient / wrong-procedure。
- [ ] **Q4320 Suspected Child Abuse → 先 interview the patient about abuse** ✓ — Direct + age-appropriate + open-ended + alone（无父母在场）；不 family therapy（让 abuser 在场）/ 不 CBT（治疗在 dx 后）/ 不 reassurance（漏 abuse）。
- [ ] **Q13465 Elder Abuse 识别** ✓ — 异常 fracture + 财务 exploitation + medication nonadherence 恶化 + 与家人同住 = 红 flag；不是 alcohol / 不是 dementia / 不是 depression alone。

### 1.2 滚动复习（D+3 (5-24) / D+7 (5-20)）

- [ ] **D+3 (5-24) Q22639 Autonomy DEMAND vs REFUSE 不对称** — **拒治端绝对**（competent patient 拒任何 tx 绝对受尊重，即使危及生命）；**请求端 conditional**（要求 tx 受 physician medical judgment + beneficence + nonmaleficence 制约 — 不开 medically inappropriate tx，即使 patient demand）。本题 patient 想 lung cancer screening CT 无 indication → 不批。
- [ ] **D+7 (5-20) Q16066 TD & EPS 四兄弟鉴别**：① **Acute dystonia**（数小时-数天起；颈/眼/喉痉挛；Rx benztropine / diphenhydramine） ② **Akathisia**（数天-数周；坐立不安；Rx 减药 + propranolol） ③ **Drug-induced Parkinsonism (DIP)**（数周-数月；rigidity + bradykinesia + tremor；Rx 换 quetiapine/clozapine，**不加 levodopa**） ④ **Tardive Dyskinesia (TD)**（数月-数年；orobuccal + 不自主舞蹈；Rx VMAT2 抑制剂 valbenazine / deutetrabenazine + 减元凶）。**5-24 浩斯宾 Q9 DIP 就考这个**。

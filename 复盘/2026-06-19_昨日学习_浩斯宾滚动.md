## §一 一句话总结（被动复习 — 扫一遍即接触全部昨日内容）⭐

### 1.1 NBME11A 真错（20 道）——「我为什么错」一句话钥匙

**Block 1**

- **Q001 外科 / 运动性热衰竭** — 运动+高热+虚脱+**可唤醒** = 热衰竭（<40℃）；T≥40+昏迷/抽搐才是热射病。误选 vasovagal（晕厥无持续高热）。
- **Q003 Biostats / PPV** — "我这阳性可信吗" = **PPV**；低患病率→PPV 塌。似然比只是中间量，别当 patient-level 概率。
- **Q004 心内 / 婴儿 SVT 处理** — 窄 QRS+HR>220 已由 ECG 确诊 → **迷走→腺苷**（稳）/电复律（不稳）；echo 留控制后，别先查结构。
- **Q008 GI / 外伤性胰腺炎** — 上腹被撞+**淀粉酶↑** = 外伤性胰腺炎；**脂肪酶正常≠排除**。误选肠挫伤。
- **Q013 肺 / PE 院内死亡预后** — 死亡因素 = 休克+**高龄**；下肢体征只回答"是不是 DVT"，不预测死亡。诊断 vs 预后要分流。
- **Q014 肾 / 幼儿继发 HTN** — 小孩 HTN = 继发 until proven → 先查肾（**肾动脉多普勒**）；四肢血压一致排缩窄；肾功能正常≠肾动脉正常。误选 TFT。
- **Q045 内分泌 / 心因性多饮** — **发作性+心理应激+化验正常** = 心因性多饮；DI 是**持续**多尿+高钠脱水（禁水可浓缩 vs 不浓缩）。误选 DI。
- **Q047 GYN / 妊娠宫颈癌** — 孕期无痛出血+**宫颈脆性溃疡** = 宫颈癌（胎盘后底排前置、宫颈口闭排流产）→ 活检别回避。误选流产。
- **Q048 儿科 / 新生儿化脓性关节炎** — 化脓性关节炎**各年龄首位都是金葡**；<3 月加 GBS+G−。**Listeria 管脑膜炎不管关节**。误选 Listeria。
- **Q049 Psych / 急性肌张力障碍** — 眼上翻+颈强直+**清醒无热 WBC 正常** = 急性肌张力障碍 → 抗胆碱；NMS 必有高热+AMS+CK↑。误选 NMS。

**Block 2**

- **Q061 Biostats / case-control 选指标** — "先有病人再找对照" = case-control = **只能 OR**；risk 家族（AR/RR/绝对/降低）需发病率分母，全划掉。误选 AR。（Biostats OR 家族第 4 次失误，再错升 🔴）
- **Q062 心内 / 稳定房颤** — 房颤先看"**快不快、塌不塌**"；HR 74 不快+BP 稳+脱水诱因 → 补液+**观察**。起搏器治"慢"不治"乱"。误选起搏器。
- **Q067 感染 / HUS-STEC** — 血便后**血小板↓+LDH↑↑+haptoglobin↓** = HUS → 病原几乎只剩 **STEC（E. coli）**；先认综合征再回推病原，**避抗生素**。误选伤寒沙门菌。
- **Q070 肺 / 肺 MAC** — 瘦老不吸烟女+支扩结节+消瘦夜汗 = MAC（Lady Windermere）→ **先 AFB 痰涂片+培养**（无创病原学），血丝痰非大咯血。漏答。
- **Q063 感染 / 无菌性脑膜炎 CSF** — **颈强直+意识清 = 脑膜炎**；意识乱/抽搐/局灶才是脑炎。CSF 糖正常+单核+蛋白正常 = 病毒性。dormitory 是钓饵，**脑膜炎 vs 脑炎看精神状态不看 CSF**。误选病毒性脑炎。

**Block 3**

- **Q136 神经 / LEMS** — Lung cancer + **口干便秘(autonomic)** + **无反射** = LEMS（突触前 VGCC）→ **重复神经刺激递增**确诊；MG 反射正常/疲劳加重/递减。误选肌活检。
- **Q137 Ethics / 终末期拒绝治疗** — 能力完好+**正常悲伤**+拒绝延命 → 尊重自主 + **先问患者是否想找人讨论**；悲伤≠抑郁（不反射精神科）、家属未知≠必开家庭会议。误选精神科评估。
- **Q143 Heme-Onc / 治疗相关 AML** — 既往**化疗**多年后+外周 myeloblasts = t-AML；filgrastim 是刻意单列的钓饵（仅弱关联），原癌复发不以 myeloblasts 表现。误选 filgrastim。
- **Q149 儿科 / 乳突炎** — AOM + **耳后压痛+耳廓外移** = 乳突炎升级 → **颞骨 CT + IV 抗生素**；前颈可动无痛淋巴结是反应性干扰。误选口服阿莫西林。
- **Q145 感染 / 肾盂肾炎选药** — 选药 = 药敏 × **部位穿透** × **患者毒性/过敏**；呋喃妥因仅下尿路、听力↓避氨基糖苷、磺胺过敏避 TMP-SMX → **环丙沙星**。误选呋喃妥因。

### 1.2 NBME11A 做对但标记（7 道）——巩固反射，别下次翻车

- **Q007 外科 / 张力气胸** — 创伤+单侧低氧+锁上/颈膨隆+患侧**鼓音**/呼吸音消失 = 张力气胸 → 针刺减压（别等片）。vs 血胸浊音 / 填塞 Beck 三联。
- **Q054 内分泌 / celiac 骨软化** — 维 D↓↓+PTH↑+ALP↑+**钙不高** = 维 D 缺乏型骨软化（继发甲旁亢）；原发甲旁亢是**高钙+低磷**。
- **Q058 儿科 / 男婴发热 UTI** — 婴幼儿发热性 UTI 影像**首选 renal US**（无辐射）；VCUG 确诊反流分级；**nitrite 阴性不排除**小儿 UTI。
- **Q066 神经 / VHL** — 小脑**血管母细胞瘤**+家族史 = VHL → 腹部 CT 找肾癌/胰/嗜铬；别反射结肠镜（那是遗传性结直肠癌谱）。
- **Q131 儿科 / 婴儿 SVT 识别** — 婴儿不诉心悸只表现喂养差/烦躁/呼吸快；**250+规则窄 QRS = SVT**；Hb 9+肝脾是干扰项。（与 Q004 处理闭环）
- **Q138 肾 / 儿童 CKD 限磷** — CKD：高磷→继发甲旁亢→肾性骨病 → **限磷**；限钾、限液（水肿别多喝水）、**儿童不限热量**（保生长）、贫血补 EPO 非 B12。
- **Q146 Psych / 丙戊酸致血小板减少** — 双相药+鼻出血 → 丙戊酸血小板减少 → **先查血小板计数**（不急停药）；治疗范围内也可血液毒性；轻度偏高的锂是干扰项。

### 1.3 必复（🔴 反复错 + 考前必看，与昨日内容呼应）

- 🔴 **脑膜炎球菌脑膜炎（Q18591，反复错×2）** — 高热+颈强直+**瘀点**+低压 → **立即经验抗生素 ± 地塞米松，勿等 LP**（呼应昨日 Q063：细菌型 CSF 不站这边，但临床高危先打针）。
- 🔴 **Study Design 选择（→cohort）** — 比较"**未来发病风险**"= 前瞻性队列；与昨日 Q061 "case-control 只能 OR" 互为正反面。
- 🔴 **急性化脓性颈淋巴结炎（Q4843/Q4353，反复错×3）** — 急性单侧红肿痛淋巴结 = S.aureus/GAS → **克林霉素**；别反射猫抓病/azithromycin。
- 🔴 **糖尿病足骨髓炎（Q7522，反复错×2）** — 确诊 = **骨活检+培养**，不是截肢、不是表面拭子。
- #考前必看 **有机磷中毒** — 流涎/流泪/针尖瞳/支气管分泌+肌束颤 → **阿托品 ± 解磷定**；针尖瞳≠阿片。

## 错题详解

### ✗ Q11（Q3640）镰刀贫脓毒症首位病原 = 肺炎链球菌

- 你选 **C 金黄色葡萄球菌（S. aureus）**；正确 **B 肺炎链球菌（Streptococcus pneumoniae）**。
- **逻辑**：镰状细胞病（SCD）→ **功能性无脾（autosplenectomy）→ 包膜菌（encapsulated）易感** → **脓毒症/脑膜炎首位 = 肺炎链球菌**（其次 Hib、脑膜炎球菌、沙门菌）。
- **S. aureus 的陷阱**：SCD 里金葡确实重要，但它是**骨髓炎**的病原（与**沙门菌 Salmonella** 并列），**不是脓毒症首位**。别把"骨髓炎病原"错套到"脓毒症病原"。
- **反射**：**无脾 + 发热 → 先想肺炎链球菌**（赶紧经验性抗生素 + 该打 PCV/PPSV23 + 青霉素预防）。

### ✗ Q20（Q20430）ARP —— 🔴 反复错第 3 次（又选 NNH 13.0）

- 你选 **D 13.0**；正确 **E 68.8**。**这是今天第二次、累计第三次错在同一陷阱**（5-11 选 B 3.5%、今天追踪卡刚建、自测又选 D）。
- **D 13.0 = NNH（需害人数 = 1/ARI = 1/0.077 ≈ 13，单位"人数"）** —— 又被它骗。
- **ARP = (Re−Ru)/Re = (0.112−0.035)/0.112 = 68.75% ≈ 68.8%**（带 %，问 "percentage"）。

> [!danger] 🔴 ARP 反射还没建立 —— 强化口诀
> 题问 "**percentage** attributed to" + 给两组 risk → **要 %**：
> - **3.2** = RR（相对危险度，**倍数**，连 % 都没有）
> - **3.5 / 11.2** = 两组裸 risk（**%** 但不是答案，是公式零件）
> - **13.0** = NNH（**人数**，1/ARI；看到"两位带小数无 %"立刻警惕）← 你两次的坑
> - **68.8** = **ARP**（带 %、最大、= 暴露组里"多出来"的占比）← 答案
>
> **一眼锁**：选项里**带 % 且最大的那个 = ARP**。13.0 没 %、是人数，先排除。

## 你标注的疑问解答

### Q8「HIV 急性期还可以导致脑膜炎！」✓ 完全正确
急性 HIV（acute retroviral syndrome）**约 25% 可致无菌性（淋巴细胞性）脑膜炎**。本题铁证：**CSF 淋巴为主 + 糖正常 + 肠病毒 PCR 阴 + 全身淋巴结 + 斑丘疹 + 性活跃** → 急性 HIV。诊断查 **HIV RNA 病毒载量**（早期抗体可阴）。你抓得很准。

### Q10「A（D-二聚体单项）为什么不对？」
- **D-二聚体敏感但极不特异**：VTE、术后/创伤、炎症/感染、妊娠、肿瘤、老年、肝病都升高。**脓毒症患者 D-二聚体几乎一定高**，单凭它**区分不出 DIC**。
- **确认 DIC 要看"消耗"证据**：**纤维蛋白原（fibrinogen）被消耗 → 低**，配合血小板↓ + PT/PTT 延长。本例已有板↓ + PT/PTT 延长，**再加低纤维蛋白原就锁定消耗性凝血病（DIC）**。
- 所以 **B 纤维蛋白原**是最具确认价值的单项；A 单查 D-二聚体在脓毒症里"必高但不特异"，不能确认。

### Q14「B?」✓ 对
军团菌（Legionella）肺炎 = **呼吸喹诺酮（左氧氟沙星 levofloxacin）或大环内酯（阿奇霉素 azithromycin）**。β-内酰胺无效（胞内菌）。"社区肺炎 + 腹泻 + 低钠 + 相对缓脉 + 尿抗原" = Legionella 招牌。

### Q24「B?」✓ 对
HR = 0.70（95% CI 0.55–0.88）：
- **HR 是"瞬时风险比（hazard）"** → B 表述准确："任一时刻新药组瞬时事件风险约为对照 0.70 倍"。
- **CI 0.55–0.88 不含 1 → 统计显著**。
- 排除：A"降低 70%"错（是降 30%，HR=0.70 → 1−0.70）；C 错（CI 不含 1，显著）；D"中位生存延长 30%"错（HR≠中位生存比）；E 把 HR 等同 RRR 不严谨。

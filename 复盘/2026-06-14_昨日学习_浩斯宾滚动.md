## §一 一句话总结（强制复习清单 — 扫一遍即被动复习）

### 1.1 D+1 昨日新学（2026-06-13，肺大块 20 张）

> [!info] 昨日 4 大反射（最该焊死）
> ① **呼吸音增强 = 肺实变**（积液/气胸/不张都减弱）；② **疑 PE 先算 Wells：≤4→D-二聚体，>4→直接 CTPA**（你昨天栽 2 次）；③ **CHF=漏出液 pH 偏碱 7.4–7.55**（连错 2 次）；④ **capnography 平线 = 零 CO₂**（食管/脱管/骤停，有生命体征则=食管）。

- **Q20145 Capnography**：平线 = 食管插管/脱管/骤停；有生命体征→管移入食管；右主支气管仍有 CO₂（不平线）。
- **Q2636 SIADH（SCLC）**：轻症一线=限水<800 mL/d；地美环素=二线（肾毒）；NS 反致钠更低。
- **Q4145 / Q4537 肺挫伤**：钝胸伤 + <24h + 外周非肺叶斑片浸润 + 输液后加重；CT 最敏感；vs ARDS 双侧 24–48h、脂栓 24–72h+神经+瘀点。
- **Q3049 GPA**：上呼吸道（鼻窦/鞍鼻）+ 肺空洞 + GN + 气管狭窄溃疡 + c-ANCA/PR3；石工硅尘是干扰钩子。
- **Q4053 CHF 漏出液**（🔴）：pH 7.4–7.55、蛋白/LDH 低；正常胸液 pH≈7.60（间皮泵 HCO₃⁻）。
- **Q10982 慢性咳嗽**：夜间加重+抗组胺无效=哮喘→PFT（必要时甲胆碱激发/ICS 试验）。
- **Q3859 / Q16488 PE 诊断路径**（🔴 #难记）：术后/癌症+心动过速+低氧+胸片正常=Wells>4→直接 CTPA；预防剂量抗凝不排 PE。
- **Q4540 失血性休克**：颈静脉平+冷+小左室腔+EF↑高动力（代偿）；镜像心包填塞（怒张+充盈压↑）。
- **Q4387 肺实变体征**：叩浊+**呼吸音增强**+语颤增强+egophony=consolidation（唯一呼吸音增强者）。
- **Q16261 新生儿张力气胸**：早产 RDS+正压+突发单侧呼吸音↓+透光增亮+不稳→立即针刺胸腔造口（不停 nCPAP、不等片）。
- **Q2997 侵袭性肺曲霉病**：粒缺/移植+发热+胸膜痛+咯血+halo 征→GM/BAL→伏立康唑±棘白菌素。
- **Q2303 ARDS 低氧机制**：弥漫分流→吸氧不纠正+低顺应性+高 A-a；靠 PEEP 不靠猛加 FiO₂。
- **Q2298 HOA**：杵状+痛性关节病→先胸片找肺腺癌（不吸烟者首位肺癌）。
- **Q2817 原发肾上腺功能不全**：低钠+高钾+低糖+嗜酸↑+体位性低血压；粟粒 TB 累双肾上腺；高钾=原发（继发钾正常）。
- **Q4562 机械通气氧合**：氧合=PEEP+FiO₂（ARDS 先加 PEEP）；CO₂/pH 正常不动 RR/VT。
- **Q4566 急性支气管炎**：URI 后咳>5 天+脓/血丝痰+无发热+胸片清→仅对症；脓痰≠细菌，不查痰。
- **Q19704 妊娠期呼吸困难**：孕酮过度通气→分钟通气↑+FRC↓+VC/FEV1 正常=正常生理；肥胖限制=VC↓+分钟通气↓（相反）。
- **Q6942 早期脓毒性休克**：暖期=SVR↓+心指数↑+脉压宽→洪脉+暖肢；晚期才冷。

### 1.2 滚动复习（D+3 / D+14 / D+30 各 1 条）

- **D+3 (06-11) TCA 过量**：QRS>100 ms → 碳酸氢钠（治心脏钠通道毒性，不是纠酸而已）。
- **D+14 (05-31) 产后尿潴留**：滴尿=充溢性 overflow → 导尿减压（≠观察、≠拟胆碱）。
- **D+30 (05-15) WPW + 预激房颤**：避免**所有** AV 结阻滞剂 = **ABCD**（**A**denosine 腺苷 / **B**eta-blocker β阻滞剂 / **C**CB 维拉帕米·地尔硫卓 / **D**igoxin 地高辛）→ 压房结逼冲动走旁路→室颤 (VF, ventricular fibrillation)；用普鲁卡因胺 (procainamide)/电复律。

### 1.3 必复（🔴 + #考前必看 + #难记）

- **🔴 PE 诊断路径**（昨日栽 2 次）：Wells>4 → 直接 CTPA，别先 D-二聚体。
- **🔴 有机磷中毒**（第 3 次）：胆碱能危象 → **阿托品**（+ 解磷定）；去污优先。
- **#考前必看 高钙危象**：急症先 **生理盐水 (NS, normal saline) 大量补容量**（补水的同时钠负荷抑制近端小管钙重吸收 = **利钙尿 calciuresis**，NS 本身一举两得），再 降钙素 (calcitonin, 快但 2-3 天快速耐受 tachyphylaxis 失效) / 双膦酸盐 (bisphosphonate, 慢 2-4 天但持久治本)，别先纠病因。⚠️ **呋塞米 (furosemide) 不再常规用**——容量没补够就利尿→脱水→血钙反更高，"立即呋塞米"是干扰项。
- **#难记 脑膜炎球菌脑膜炎**：疑诊 → **先经验抗生素（± 地塞米松）勿等腰穿/CT**。

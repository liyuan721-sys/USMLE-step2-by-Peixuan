## §一 一句话总结（被动复习清单）

### 1.1 D+1 昨日新学（06-12 · 30 张卡）

> [!tip] 中毒/环境急症 — 钥匙串
> - **针尖瞳 (pinpoint pupil) 三鉴别（焊死）**：**湿**（大汗+流涎+肌束颤+缓脉）= 有机磷 organophosphate → **阿托品 atropine + 解磷定 pralidoxime**；**干**（无湿征+呼吸抑制）= 阿片 opioid → **纳洛酮 naloxone**；瘫+热+突发 = 脑桥出血 → CT。**别一见针尖瞳就反射阿片，先找湿征。**（有机磷第 3 次错 Q17312，误选丁丙诺啡 buprenorphine）
> - **有机磷处理顺序**：① ABC 稳定（插管）→ ② **去污（脱衣 + 冲洗皮肤/眼，阻止经皮继续吸收）** → ③ 解毒（阿托品 → 解磷定）。去污**优先于解毒**（Q3134）。确诊 = **RBC 乙酰胆碱酯酶 RBC AChE**。
> - **抗胆碱中毒 anticholinergic（苯海拉明 diphenhydramine）= 干红热散瞳狂 + 尿潴留**（mad/blind/hot/dry/red + full as a flask）；与有机磷"湿"相反，与 5-HT 综合征"湿 + 阵挛"相反。
> - **BZD（苯二氮䓬 benzodiazepine）过量 vs 阿片**：两者都嗜睡 + 共济失调 + 含糊语；**BZD = 瞳孔正常有反应 + 生命体征常正常**；**阿片 = 针尖瞳 + 呼吸抑制**。

> [!tip] 毒醇 + 火灾双中毒 — 高 AG 代酸钥匙
> - **甲醇 methanol 毒眼**（视物模糊 / 视盘充血 optic disc hyperemia / 失明）+ 高 AG（阴离子间隙 anion gap）代谢性酸中毒 → **甲吡唑 fomepizole**；**乙二醇 ethylene glycol 毒肾**（草酸钙结晶 calcium oxalate / 胁痛 / AKI 急性肾损伤 acute kidney injury / 低钙）→ **甲吡唑**。**视觉症状 = 区分两醇的钥匙**；水杨酸 salicylate 也致高 AG 代酸但**不毒眼**。
> - **氰化物 cyanide（火灾塑料燃烧）= 樱桃红 + 乳酸爆表(>10) + SpO₂ 不掉** → **羟钴胺 hydroxocobalamin**（火灾首选，不造高铁血红蛋白；亚硝酸钠火灾禁）。
> - **一氧化碳 CO（火灾烟雾）= SpO₂ 假性正常 + 神经症状（躁动/抽搐）** → **100% 氧非重吸面罩**；诊断靠动脉血气测 COHb（碳氧血红蛋白 carboxyhemoglobin），不看 SpO₂。**火灾 = CO + 氰化物双中毒**。
> - **尼古丁中毒 nicotine（幼儿误服烟头/电子烟液）= 双相**：早期刺激相（HTN 高血压 hypertension + 心动过速 + 肌阵挛 myoclonus）+ 毒蕈碱样（流涎/呕吐腹泻/喘鸣）→ 后期抑制相。

> [!tip] 环境急症 — 降温/复温/下降钥匙
> - **高原病三药**：乙酰唑胺 acetazolamide「管高」（AMS 急性高山病 acute mountain sickness）/ **地塞米松 dexamethasone「管脑」（HACE 高原脑水肿 high-altitude cerebral edema）** / 硝苯地平 nifedipine「管肺」（HAPE 高原肺水肿 high-altitude pulmonary edema）；**任何重型 → 下降永远第一位**。HACE/HAPE 水肿是**血管漏（非容量过载）→ 利尿剂 furosemide 无效甚至有害**（患者反而容量不足）。
> - **热射病**：经典型（NHS 非劳力性热射病 nonexertional heat stroke；老人 + 高湿桑拿 + DIC 弥散性血管内凝血 disseminated intravascular coagulation / 肺水肿 / 血液浓缩）→ **蒸发降温 evaporative**（老人首选）；劳力型（EHS 劳力性热射病 exertional heat stroke；年轻 + 运动）→ **冰水浸泡 ice water immersion**。退热药 / 丹曲林 dantrolene 无效。
> - **冻伤 frostbite**：37–39°C 温水快速复温 + 镇痛；**复温后仍持续缺血（灰白/感觉缺失）→ 血管造影 angiography / 锝扫描查血栓 → 24h 内溶栓 tPA 救肢**；**截肢延迟至数周后组织分界 demarcation**（急性期高估坏死范围）。

> [!tip] 肺科 + 创伤 — 06-12 下半场
> - **术后肺不张 atelectasis**：早期 ABG（动脉血气 arterial blood gas）= **原发呼吸性碱中毒**（疼痛/焦虑过度通气）；预防 = **深呼吸 + 激励性肺量计 incentive spirometry**（戒烟需术前 >4–8 周才有效）。
> - **胸腔积液 Light 标准**：PE（肺栓塞 pulmonary embolism）致**渗出性血性积液**（制动 + 突发 + 无浸润影）；CHF（充血性心力衰竭 congestive heart failure）致**漏出液**（pH 偏碱 ~7.45）。
> - **气管支气管损伤**（钝/穿透伤后**持续大量漏气** + 胸管后肺不复张）→ **支气管镜 bronchoscopy** 确诊。**大量血胸 massive hemothorax**（>1500mL 或持续 >200mL/h）→ **急诊开胸 thoracotomy**。**下肋骨折（9–12 肋）→ 警惕腹腔实质器官（肝/脾）损伤 → 腹部 CT**。
> - **孤立肺结节（SPN solitary pulmonary nodule）高危大结节** → 组织诊断/手术切除；**移植后 CMV（巨细胞病毒 cytomegalovirus）肺炎**（已吃 TMP-SMX 复方磺胺甲噁唑 trimethoprim-sulfamethoxazole 预防 → 排除 PCP 肺孢子菌肺炎 Pneumocystis pneumonia）；**平台压 plateau pressure = 呼吸系统顺应性**（吸气暂停测）；**URI（上呼吸道感染 upper respiratory infection）后亚急性咳嗽（UACS 上气道咳嗽综合征 upper airway cough syndrome）→ 一代抗组胺**；**稳定期 COPD（慢性阻塞性肺病 chronic obstructive pulmonary disease）初始 = LABA（长效 β₂ 激动剂 long-acting beta-agonist）+ LAMA（长效抗胆碱能药 long-acting muscarinic antagonist）**。
> - **儿科气道**：先天性膈疝（CDH congenital diaphragmatic hernia）→ **气管插管 + 胃管减压**（禁 bag-mask 面罩加压通气 / 胸管）；会厌炎 epiglottitis → **先气管插管**（手术气道仅在插管失败时）；哮吼 croup = 喉 + 近端气管水肿（✅做对）。
> - **Heme-Onc（血液肿瘤）**：肾衰患者用**吗啡 morphine → 活性代谢物（吗啡-6-葡糖苷酸 morphine-6-glucuronide）经肾蓄积 → 阿片中毒、AMS（急性意识改变 altered mental status）**；换**芬太尼 fentanyl / 美沙酮 methadone**（不靠肾），避吗啡 / 哌替啶 meperidine / 可待因 codeine / 曲马多 tramadol。

### 1.2 滚动节点（每节 1 条提醒）

- **D+3（06-10 头颈/免疫）**：🔴 **坏死性外耳道炎 malignant otitis externa**（老年糖尿病 + 夜间剧耳痛 + 耳道肉芽 + 面瘫 = 铜绿假单胞菌 Pseudomonas → 静脉抗假单胞 环丙沙星 ciprofloxacin）；**梅尼埃病 Ménière disease** = 眩晕 1–2h + 耳鸣 + 感音聋 + 无头位诱发（≠ BPPV 良性阵发性位置性眩晕 benign paroxysmal positional vertigo），生活方式**限盐**。
- **D+7（06-06 感染）**：**梅毒 syphilis 血清学** = 非螺旋体试验（RPR 快速血浆反应素 rapid plasma reagin / VDRL 性病研究实验室试验 venereal disease research laboratory，查活动度/随访）+ 螺旋体试验（FTA-ABS 荧光螺旋体抗体吸收试验 fluorescent treponemal antibody absorption，确诊终身阳）；**猫咬 → 阿莫西林-克拉维酸 amoxicillin-clavulanate 预防**（多杀巴斯德菌 Pasteurella multocida）。
- **D+14（05-30 肾/酸碱）**：**钙草酸盐结石 calcium oxalate stone 防复发首选 = 限钠**（高钠促尿钙排泄）**不是限钙**；1 型远端 RTA（肾小管酸中毒 renal tubular acidosis）= 低 K + 正常 AG（阴离子间隙 anion gap）代酸 + 尿 pH>5.5。
- **D+30（05-14 心内/创伤）**：**先天性 LQTS（长 QT 综合征 long QT syndrome；Jervell & Lange-Nielsen 型 = 长 QT + 先天性耳聋）**；CVC（中心静脉导管 central venous catheter）置入后必做 **CXR（胸片 chest X-ray）** 排气胸/定位。

### 1.3 必复（🔴 + 标签轮换）

- 🔴 **有机磷（反复错 ×3）#考前必看**：针尖瞳别反射阿片 → **先找湿征（大汗/流涎/肌束颤/缓脉）= 有机磷 → 阿托品**。
- 🔴 **孕妇/小孩 Lyme（反复错 ×3）**：游走性红斑 + 孕 → **阿莫西林 amoxicillin**（doxycycline 孕期/<8 岁禁；ceftriaxone 留给神经/心脏 Lyme）。**一看到"孕"字立即划掉 doxy。**
- #难记 **急性单侧颈淋巴结炎（×3 猫抓病陷阱）**：经验治疗 = **克林霉素 clindamycin**（覆盖 S.aureus / GAS），不是猫抓病的阿奇霉素。
- 🔴 **镰刀贫功能性无脾脓毒症 → 肺炎链球菌 S.pneumoniae**（非沙门菌）；**糖尿病足骨髓炎确诊 = 骨活检培养 bone biopsy**（非截肢）。

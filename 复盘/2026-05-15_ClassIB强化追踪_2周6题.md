A 19-year-old man with LQT3 (SCN5A mutation) has recurrent syncope. He is started on mexiletine. Which ECG change is **most expected** with mexiletine therapy?

- [ ] A. PR prolongation
- [ ] B. QRS widening
- [ ] C. QT shortening
- [ ] D. QT prolongation
- [ ] E. ST elevation

### Q2.

A drug used for ventricular arrhythmias in **post-MI ischemic** patients has the following properties: rapid binding/unbinding to Na channels, preferentially binds inactivated/depolarized channels, and **shortens** action potential duration. Which drug class does this describe?

- [ ] A. Class IA (Quinidine, Procainamide)
- [ ] B. Class IB (Lidocaine, Mexiletine)
- [ ] C. Class IC (Flecainide, Propafenone)
- [ ] D. Class III (Amiodarone, Sotalol)
- [ ] E. Class IV (Verapamil, Diltiazem)
- [ ] ? 不确定（蒙的 / 二选难定）

### Q3.

A 55-year-old man **2 years post-MI** with reduced EF presents with palpitations from paroxysmal AFib (no WPW). The intern suggests flecainide for "pill-in-the-pocket" cardioversion. The attending **immediately stops** this plan. What is the underlying principle?

- [ ] A. Flecainide is contraindicated in any AFib
- [ ] B. Flecainide causes profound QT prolongation and TdP
- [ ] C. CAST trial — Class IC antiarrhythmics **increase mortality** in patients with structural heart disease / post-MI
- [ ] D. Flecainide is only effective in atrial flutter, not AFib
- [ ] E. Flecainide causes irreversible pulmonary fibrosis

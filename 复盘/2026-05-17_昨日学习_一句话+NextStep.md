## Part A · 一句话知识点（25 条）

### AV Block

1. **1°**：PR > 0.20s 恒定、无漏 → 观察。
2. **Mobitz I (Wenckebach)**：PR 渐长 → 漏 → 重置；阻滞在 AV 结、grouped beating，**无症状只观察**。
3. **Mobitz II**：PR 恒定 → 突然漏；阻滞在 His-Purkinje，**直接 PPM**（不要等症状）。
4. **3° AV block**：P-QRS 完全分离、宽 QRS 逸搏；先 transcutaneous → PPM。
5. **Symptomatic bradycardia 第一步**永远是 **Atropine 0.5 mg IV**，无效才上 pacing / epi。

### Vaughan Williams

6. **IA (Quinidine / Procainamide / Disopyramide)**：↑QT、torsades；procainamide 长期用 → **drug-induced lupus**。
7. **IB (Lidocaine / Mexiletine)**：对**缺血心肌**和**室性心律失常**最有效，缩短 QT。
8. **IC (Flecainide / Propafenone)**：禁用于**结构性心脏病 / CAD**（CAST trial 显示 ↑死亡率）。
9. **II (β-blockers)**：↓SA、↑PR，post-MI / HF 死亡率获益。
10. **III (Amiodarone / Sotalol / Dofetilide / Ibutilide)**：↑QT；**Amiodarone** = 肺纤维化 / 甲状腺 / 肝 / 角膜；**Sotalol** 兼 β。
11. **IV (Verapamil / Diltiazem)**：禁忌 = HFrEF、WPW、2° / 3° block。
12. **Digoxin toxicity**：黄绿视、N/V、scooped ST、any arrhythmia 都可能（最经典 = **atrial tach + AV block**）；解药 **Digoxin-specific Fab**。

### 晕厥

13. **Vasovagal**：前驱出汗 / 恶心 / 隧道视，倾斜试验 +，**无需治疗**。
14. **Cardiac syncope**：劳力性 + 无前驱 + 心脏病史 → 优先 echo（HCM / AS）。
15. **HCM**：劳力性晕厥 + 收缩期杂音 **Valsalva ↑ / squat ↓**；治疗 β-blocker、避免脱水 / 硝酸甘油 / 利尿；猝死风险 → ICD。
16. **AS**：晚期三联征 angina / syncope / HF；杂音 RUSB → 颈，**handgrip ↓** 杂音、**squat → stand ↑**；重度 → **AVR**。

### 新生儿先心病

17. **TOF**："**boot-shaped heart**" + tet spell → 蹲位、O₂、β-blocker、morphine；婴儿期 PGE1 维持 PDA。
18. **导管依赖型**（HLHS、严重 COA、TGA、Pulm atresia）→ **PGE1** 立即开放 PDA。
19. **TGA**：cyanosis 出生即重，"egg on string"；需 PGE1 + balloon septostomy。
20. **ASD**：固定分裂 S2 + RUSB systolic murmur；多无症状到成年。

### 抗凝 / 抗板 + 实验室指纹

21. **UFH 过量** → **Protamine** 全逆转；**LMWH 过量** → Protamine **仅逆转 ~60%**（部分）；DOAC 各有专属逆转剂（Idarucizumab → dabigatran；Andexanet → Xa）。
22. **HIT**：肝素后 5-10 天 ↓血小板 > 50% + 血栓 → 停所有肝素（含 LMWH）、改 **argatroban / bivalirudin**。
23. **指纹**：↓Na + ↑K + ↑BUN/Cr + 色素沉着 = **Addison**；↑Ca + ↓PO₄ + ↑PTH + **Cl/PO₄ > 33** = **1° HPT**；↑Ca + ↓PTH + 肺门淋巴 = **sarcoid**。

### 降脂 / ASCVD

24. **High-intensity statin** 适应：临床 ASCVD、LDL ≥ 190、糖尿病 40-75 岁 + 10y 风险 ≥ 20%。
25. **Statin 不耐受（myalgia）**：先查 CK + 排除甲减 / 药物相互作用，再 **同药低剂量 / 隔日 / 换 pravastatin / rosuvastatin**；横纹肌溶解才停药。

## Part B · 诊断后 Next Step 速查表

> [!tip] 使用场景
> 疾病能诊断、但"下一步做什么"卡壳时直查。按系统分组，每行 = **触发条件 → 立刻做 → 易错点**。

### B1. 心律失常 / Brady

| 诊断 | 触发 | **下一步（立刻）** | 易错点 |
|---|---|---|---|
| **Sinus brady + 症状** | HR < 50 + 头晕 / 低血压 | **Atropine 0.5 mg IV** → pacing if 无效 | 无症状只观察；**不要跳过 atropine** |
| **Mobitz I 有症状** | PR 渐长 → 漏 + 头晕 | **Atropine** 可试 → 无效 pacing | 多数无症状只观察 |
| **Mobitz II** | PR 恒定 → 突漏 | **跳过 atropine → Transcutaneous pacing → PPM** | atropine 对 infranodal 无效 |
| **3° AV block** | P-QRS 完全分离 | **跳过 atropine → Transcutaneous pacing → transvenous → PPM** | **atropine 可能反致 asystole**（↑窦率但 AV 不通） |
| **High-grade block + unstable** | hypotension / altered MS | **Transcutaneous pacing 立即** + epi / dopa drip | 别等 atropine 起效 |
| **Digoxin tox 稳定** | N/V + 黄视 + K/Mg 低 | **停药 + 补 K/Mg** | 不急着上 Fab |
| **Digoxin tox 不稳定** | K > 5.5 / 致命心律失常 | **Digoxin Fab 立即** | 不要给 Ca（worsen） |

### B2. 瓣膜 / 心肌

| 诊断 | 触发 | **下一步** | 易错点 |
|---|---|---|---|
| **AS（疑似）** | 劳力 angina / syncope / HF + RUSB 杂音 | **Echo 确诊** → 重度有症状 = **AVR** | 不是先抗血小板 |
| **HCM 高危** | 晕厥 + septal ≥ 30 / 家族 SCD | **ICD + β-blocker** | 不用 nitro / 利尿 |
| **HCM 无高危** | 杂音 + 轻度症状 | **β-blocker**（或 verapamil） | 避免脱水 |
| **AIC（酒精性 CM）** | 长期酗酒 + EF ↓ | **戒酒**（唯一可逆） | β-B / ACEi 是辅助 |
| **ADHF 急性肺水肿** | 呼吸困难 + SpO₂ ↓ | **NIPPV + IV furosemide + IV nitro**（SBP > 110） | β-B / ACEi 急性期暂停；morphine 已淡出 |
| **Post-MI 3-5 天 + Beck 三联** | LV free wall rupture / tamponade | **紧急 pericardiocentesis + 急诊手术** | 不是 thrombolytics |
| **Malignant pericardial effusion** | 肿瘤史 + tamponade physio | **Pericardiocentesis + 病理 + pericardial window** | |

### B3. 先心病（婴儿）

| 诊断 | 触发 | **下一步** | 易错点 |
|---|---|---|---|
| **TOF tet spell** | 蹲位缓解 + boot-shaped | **蹲 / knee-chest + O₂ + morphine + β-B + 补液** | **禁 inotropes**（加重 RVOT 梗阻） |
| **导管依赖型**（TGA / HLHS / 严重 COA / Pulm atresia） | 生后 cyanosis / 休克 | **PGE1 立即维持 PDA** + 转 ICU | TGA 加 balloon septostomy |
| **TGA "egg on string"** | 出生即重度发绀 | **PGE1 + balloon septostomy → 手术** | 别等手术，先 PGE1 |
| **ASD 固定分裂 S2** | 多无症状 | **Echo 评估 + 择期闭合**（Qp:Qs > 1.5 或症状） | 不急 |

### B4. 抗凝 / 出血

| 诊断 | 触发 | **下一步** | 易错点 |
|---|---|---|---|
| **UFH 过量出血** | aPTT 长 | **Protamine 完全逆转** | 1 mg / 100 U UFH |
| **LMWH 过量出血** | enoxaparin + 出血 | **Protamine（部分 ~60%）** | 不要指望全逆转 |
| **HIT** | 肝素 5-10 天 + PLT ↓ 50% + 血栓 | **停所有肝素 → argatroban / bivalirudin** | 禁单独 warfarin（皮肤坏死） |
| **Dabigatran 出血** | | **Idarucizumab** | |
| **Xa 抑制剂出血**（apixaban / rivaroxaban） | | **Andexanet alfa**（或 4F-PCC） | |
| **Warfarin INR > 10 / 大出血** | | **4F-PCC + IV vit K** | FFP 是次选 |

### B5. 内分泌 / 代谢指纹

| 诊断                                   | 触发                                  | **下一步**                                                               | 易错点                                        |
| ------------------------------------ | ----------------------------------- | --------------------------------------------------------------------- | ------------------------------------------ |
| **Addison 疑似**                       | ↓Na + ↑K + 色素 + 低血糖                 | **Cosyntropin stim test** 确诊 → **glucocorticoid + mineralocorticoid** | Adrenal crisis 时**先打 hydrocortisone**，别等结果 |
| **1° HPT** 原发性甲旁亢                    | ↑Ca + ↑PTH + ↓PO₄ + <font color="#ff0000">**Cl/PO₄ > 33** | **24h 尿 Ca**（排 FHH）+ DXA +</font> 肾超声 → **parathyroidectomy** 指征             | 不是先 bisphosphonate                         |
| **Malignancy hypercalcemia (PTHrP)** | ↑Ca + ↓PTH + Cl/PO₄ < 33            | **IV NS + calcitonin + bisphosphonate**                               | calcitonin 起效快但快速失效                        |

> [!tip] Cl/PO₄ ratio 鉴别意义
> **Cl/PO₄ > 33** 是区分 **1° HPT vs 其他高钙病因（如恶性肿瘤高钙）** 的经典指标。
> - **1° HPT**：PTH 致 ↑Cl 重吸收（轻度高氯性代酸）+ ↓PO₄ 重吸收 → 比值 **> 33**
> - **恶性肿瘤高钙（PTHrP）**：PTH 被抑制，PTHrP 不引起高氯性酸中毒 → 比值 **< 33**

### B6. 降脂 / 预防

| 诊断 | 触发 | **下一步** | 易错点 |
|---|---|---|---|
| **High-intensity statin 指征** | ASCVD / LDL ≥ 190 / DM 40-75y + 10y ≥ 20% | **Atorvastatin 40-80 或 rosuvastatin 20-40** | |
| **Statin myalgia (CK 正常)** | 大腿酸 + CK 正常 | **同药低剂量 / 隔日 / 换 prava-rosuva** | 不立即永久停 |
| **横纹肌溶解 (CK > 10×)** | CK 急升 + 棕尿 | **立即停 statin + IV 水化** | |

### B7. Syncope

| 诊断 | 触发 | **下一步** | 易错点 |
|---|---|---|---|
| **Vasovagal** | 前驱出汗 / 恶心 + 站立 | **Reassurance + 避诱因 + counterpressure** | 不上药 / 起搏 |
| **Cardiac syncope（疑）** | 劳力 + 无前驱 + 心病史 | **Echo + ECG**，必要 Holter / EPS | 不能 reassure |
| **Orthostatic** | 立位 SBP ↓ 20 / DBP ↓ 10 | **停诱因药 + 补液 + 渐起 → fludrocortisone / midodrine** | |

---

> [!tip] 6 条记忆 Hook（USMLE 高频陷阱）
> 1. **Brady 路径**：Atropine → Pacing → PPM（永远按顺序，别跳）。
> 2. **Tamponade physiology** = 不管病因，先 **pericardiocentesis**。
> 3. **婴儿 cyanosis 不缓解 O₂** = **PGE1**（无脑反射）。
> 4. **HIT** = 停肝素 + argatroban；不要 warfarin / 不要 platelet transfusion。
> 5. **Adrenal crisis** = **先打 hydrocortisone，再确诊**（diagnosis can wait, steroids can't）。
> 6. **ADHF 肺水肿** = **NIPPV + Lasix + Nitro**（morphine 已淡出，β-B / ACEi 急性期暂停）。

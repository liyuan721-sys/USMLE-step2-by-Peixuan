## §一 一句话总结（被动复习 — 扫一遍即接触全部昨日内容）⭐

### 1.1 D+1 昨日新学（2026-06-21）

**血液 · 出凝血四项判读（核心新建）**

- **四维反射** — 拿到 PT / PTT / BT（或血小板）"正常 vs 延长"组合先锁病因：PT=外源（华法林）｜ PTT=内源（肝素）｜ BT/板=初级止血。
- **vWD vs 血友病**（最经典对）— 两者 **PT 正常 + PTT↑ + 板数正常**完全一样，**只看 BT + 出血部位**：黏膜出血 + BT↑ = vWD；关节/深部出血 + BT 正常 = 血友病（X 连锁男童）。
- **肝病 vs DIC**（最像对）— 都 PT↑PTT↑板↓；钥匙 = **因子 VIII**（肝病正常/↑、DIC↓，因 VIII 来自内皮非肝）+ **D-二聚体**（DIC 飙、肝病不高）。
- **肝病 vs 华法林** — 都 PT↑；钥匙 = **因子 V**（肝病↓、华法林正常）+ **维 K 试验**（肝病不纠正、华法林可纠正）。因子 VIII 在这对**不区分**（都正常）。
- **混合试验** — 1:1 混合**纠正 = 缺因子**（血友病/vWD/肝病）；**不纠正 = 抑制物**。抑制物再分：**即刻不纠正 = 狼疮抗凝物（→ APS 血栓/流产）**；**孵育后才不纠正 = 获得性因子 VIII 抑制物（老年/产后突发大出血）**。
- **华法林两陷阱** — ① INR 早达标是 **VII 先掉的假象**，真抗栓靠 **II（~60h）慢掉 → 桥接肝素 ≥5 天**；② **蛋白 C 也早掉 → 起始短暂促凝 → 华法林皮肤坏死**。
- **板↓ 但 PT/PTT/纤维蛋白原正常 + 裂红细胞 = TTP/HUS**，别和 DIC 混。

**血液 · 其他笔记** — [[完整笔记/专题笔记/血液/血液_血制品输注速查]]（判读定病因后选 FFP / 冷沉淀 / PCC / 血小板）｜ [[完整笔记/专题笔记/血液/血液_反复鼻出血鉴别]]｜ [[完整笔记/专题笔记/血液/血液_HighYield速查总览]]。

**Biostats · 研究设计识别（🔴 反复错第 4 次）**

- NBME11 Q017：焦点小组"态度"研究 = **qualitative（定性）**，误选 matched case-control。反射钩：**先问"数据是叙事(themes) 还是数值(结局)"** → 再看修饰词；`matched`/`randomly selected` 是噪音；random **selection**(抽样) ≠ random**ization**(随机分配)。

## §一 今日所学知识点（07-11 神经大 block）

### 1.1 今日错题 — 新卡（🟡 答对≠掌握，全是"定位钥匙"）

- **AD 影像 (Q3058)**：老年 + **突出记忆丧失（找词/走失）+ 隐匿渐进 + 无局灶/无阶梯 = AD** → MRI 招牌 = **颞叶（海马/内侧颞叶）萎缩（晚期，早期常正常）**。多发腔隙梗死 + 阶梯样才是血管性痴呆。⏱️ 读痴呆 stem 找**最早认知起点**（记忆已数年）别只看最近行为恶化。[[mistakes/uworld-mistakes_2026-07#^Q3058]]
- **痴呆初筛下一步 (Q16082)**：疑痴呆**常规实验室 = CBC + B12 + TSH + CMP + 影像**（人人做，找可逆病因）；**梅毒/叶酸/维D/CSF/EEG = 选择性**（有风险因素才做）。B12 是常规、梅毒是选择性。[[mistakes/uworld-mistakes_2026-07#^Q16082]]
- **蜱麻痹 (Q2289)**：**上行瘫 + 反射消失 + 感觉正常 + CSF 正常 + 徒步/林区 = tick paralysis**（毒素，蜱吸血 4–7 天）→ **全身搜蜱移除，1 h 见效**。GBS 才有 CSF 蛋白高（albuminocytologic）+ 自主神经紊乱。[[mistakes/uworld-mistakes_2026-07#^Q2289]]
- **Todd 麻痹 (Q4155)**：儿童跌倒失意识 → **醒后 confused（postictal）** → 一过性偏瘫 → **影像正常 → <36 h 恢复 = 发作后 Todd 麻痹**。偏瘫型偏头痛有头痛、**不致意识丧失**（干扰项）。[[mistakes/uworld-mistakes_2026-07#^Q4155]]
- **颈动脉夹层 (Q16169)**：青年 + 轻创伤后**单侧头颈痛 + 部分 Horner（ptosis+miosis，出汗保留）+ TIA/卒中 = 颈内动脉夹层** → CT/MR 血管造影 + 抗栓。**部分 Horner 无汗保留** = 交感随颈内、出汗随颈外。DAI 需重型外伤 + 意识障碍。[[mistakes/uworld-mistakes_2026-07#^Q16169]]
- **ALS 球部起病 (Q19242)**：球部无力 + **UMN（jaw jerk 亢进/Babinski）+ LMN（舌萎缩 + 肌束颤）+ 感觉正常 = ALS** → **EMG 确诊**。**一个 jaw jerk 阳性即一票排除** MG / 肉毒 / 肌病（都无 UMN 征）。[[mistakes/uworld-mistakes_2026-07#^Q19242]]
- **钝性创伤颈椎影像 (Q18658)**：**NEXUS 5 项全阴 + 无高能机制才可不影像**；任一阳性（脊柱压痛/神经缺损/意识改变/中毒/**distracting injury**）→ **颈椎 CT（>平片，术前完成）**。开放长骨骨折 = distracting injury。[[mistakes/uworld-mistakes_2026-07#^Q18658]]
- **单椎骨折→全脊柱 (Q18659)**：钝性创伤**发现 1 处椎骨骨折 → 影像全脊柱**（第二处非连续骨折 ~20%）→ 胸腰椎 CT。C5-C6 半脱位 = **C6 根病**（腕伸无力 + 拇指麻），已被骨折解释，别去查颈动脉。[[mistakes/uworld-mistakes_2026-07#^Q18659]]
- **帕金森静止性震颤 (Q4400)**：**静止显、随意运动减轻/消失、不对称、搓丸、分心（心算）加重、不受酒精影响 = PD 静止性震颤（基底节）**。小脑性相反（越近目标越抖 + 共济失调）；原发性震颤 = 动作抖 + 饮酒改善。[[mistakes/uworld-mistakes_2026-07#^Q4400]]

### 1.2 今日错题 — 反复错二/三刷（🔴 今天又栽，必须背死主线）

- **🔴 头颅血肿 cephalohematoma (第2次·Q2472)**：产钳后**质硬 + 界清 + 不跨骨缝 = cephalohematoma（骨膜下，受骨缝限制）**。判据永远**先看跨不跨骨缝**：跨缝 = caput（软/凹/出生即有）或 subgaleal（软/波动/可致命）。今天被"caput 常见/软"带偏。[[mistakes/uworld-mistakes_2026-07#^Q2472-R0711]]
- **🔴 儿童偏头痛影像 (第2次·Q3672)**：反复搏动头痛 + **先兆一过性可逆（<1h）+ 查体正常 + 无 red flag = 偏头痛（临床诊断）→ 不做任何检查，直接 NSAID**。别见"视觉症状"就找眼压/头 CT。[[mistakes/uworld-mistakes_2026-07#^Q3672-R0711]]
- **🔴 尺神经肘 vs 腕 (做对后又错·Q4170)**：**屈腕无力（FCU）+ 手背内侧麻（背皮支）+ 握力↓ = 肘部（近端）**；腕部（Guyon 管）= 屈腕正常 + 手背感觉保留。本例明写"屈腕无力"→ **肘**。[[mistakes/uworld-mistakes_2026-07#^Q4170-R0711]]
- **🔴 HIV 远端对称多神经病 (第2次·Q18492)**：痛性远端多神经病**一线镇痛 = gabapentinoid（加巴喷丁）/ SNRI（度洛西汀）/ TCA / 辣椒素**（不分病因）。镁**仅用铂类化疗神经病预防**，HIV 不用；也别选激素。[[mistakes/uworld-mistakes_2026-07#^Q18492-R0711]]
- **🔴 病理性小头畸形 (第2次·Q18791)**：**小头（HC<3rd）+ 神经异常/跨线下降 = 病理性 → 首查脑 MRI**（结构/围产缺氧/先天感染）。小头本身 = 脑生长异常 → **影像优先**，别被肌张力↓带向 EMG。[[mistakes/uworld-mistakes_2026-07#^Q18791-R0711]]
- **🔴 首次无诱因癫痫→脑转移 (第3次·对后又错·Q110558)**：成人**首次无诱因癫痫 + red flag（局灶起源 + 继发全面化 + 恶性史 RCC 肾细胞癌 + 持续局灶无力）= 脑 MRI 找结构灶**。**Na 132 仅轻度、血糖 70 正常** = 代谢数值都不是病因，别被数字钩去补高渗盐/糖。[[mistakes/uworld-mistakes_2026-07#^Q110558-R0711]]
- **🔴 Broca 失语定位 (第2次·Q3738)**：**非流利 + 理解保留 + 复述差 = Broca = 优势额叶**（常伴右侧肢体无力，MCA 大脑中动脉上分支）。**B**roca = **B**roken = 额（前=输出）。优势颞叶 = Wernicke（流利胡说 + 理解差）。定位三步：流利度→理解→复述。[[mistakes/uworld-mistakes_2026-07#^Q3738-R0711]]

## §二 之前高危知识点必复

### 2.1 神经 🔴 反复错（今日同科优先 — 这些反射还在崩，背死）

- **🔴 MS 痉挛对症 (Q19797)**：MS 肌痉挛 = **baclofen / tizanidine**；别选 pramipexole（那是 PD/RLS）。（07-08 又错）
- **🔴 华法林相关 ICH 逆转 (Q3737)**：华法林 ICH = **PCC（凝血酶原复合物）+ 维生素 K**，不是阿司匹林/FFP 首选。（07-08 又错）
- **🔴 亨廷顿病 (Q3954)**：**舞蹈样动作 + 精神症状 + 认知↓ + 家族史（AD 遗传）= Huntington**，别混 FTD。（07-08 又错）
- **🔴 GCSE 治疗后不应答→EEG (Q19890)**：全面惊厥性癫痫持续状态用药后仍不醒 → **查非惊厥性 SE = EEG**，别选 flumazenil。（07-08 又错）
- **🔴 糖皮质激素性肌病 (Q4443)**：GCA（giant cell arteritis，巨细胞动脉炎）长期激素后近端无力 + **CK（肌酸激酶）正常 + ESR（红细胞沉降率）正常/不高 = 类固醇肌病**（减激素），别当 PMR（polymyalgia rheumatica，风湿性多肌痛）复发加量。（07-08 又错）

### 2.2 感染 🔴 经典反射轮换（六月最大反复错簇，你屡栽）

- **🔴 孕妇早期 Lyme（游走性红斑）→ 阿莫西林 (Q3257)**：孕期禁多西环素/避头孢曲松 → **amoxicillin**。（反复错×3）
- **🔴 流脑先经验抗生素勿等腰穿 (Q18591)**：疑脑膜炎球菌脑膜炎 → **立即经验抗生素 + 地塞米松，勿等 LP/CT**。#考前必看（反复错×3）
- **🔴 巴贝虫 babesiosis (Q3104)**：无脾/免疫低 + 血管内溶血 + 蜱区 → 血涂片（环形体/Maltese cross）。（反复错×3）
- **🔴 急性单侧颈淋巴结炎经验治疗 = clindamycin (Q4843)**：病原 S.aureus/GAS；别被"猫抓病"钓去 azithromycin。（反复错×3）
- **🔴 迁延咽炎 + 全身淋巴结 + 显著乏力 = 传单 → 嗜异性抗体 (Q4354)**。（反复错×2）
- **🔴 IVDU + 吸气增强胸骨杂音 = 右心 IE → 经验万古霉素 (Q3013)**（覆盖 MRSA），别头孢曲松。（反复错×2）

### 2.3 跨科经典反射 🔴 轮换（中毒/骨科/风湿/Biostats）

- **🔴 CO ↔ 氰化物镜像别互换 (Q3223 / Q107283)**：火灾烟雾 + **樱桃红 + 乳酸爆表 = 氰化物 → 羟钴胺**；单纯 CO 中毒 → **100% 氧（重则高压氧）**。
- **🔴 骨科腕管综合征 (Q4636)**：轻中症**首选夜间夹板**（孕期尤其保守），别一上来注射/手术。（反复错×2）
- **🔴 PFPS 髌股疼痛 (Q12060/Q8886)**：治疗 = **股四头肌强化（忌制动）**；体征 = 股四头肌等长收缩/髌骨研磨痛。（反复错×2）
- **🔴 反应性关节炎一线 = NSAIDs (Q4577)**：非抗生素（即便有前驱 GU/GI 感染史）。（反复错×2 同选抗生素）
- **🔴 Biostats ARP vs NNH (Q20430)**：**归因危险度百分比 (ARP) = (RR−1)/RR ×100%**，别填成 NNH（number needed to harm，需要伤害人数）的 13.0。#考前必看（反复错×3）
